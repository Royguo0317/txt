《日经新闻》3月9日报道，日本外相茂木敏光(Toshimitsu Motegi)和防卫大臣岸信雄(Nobuo Kishi)将在下周会见美国国务卿布林肯(Antony Blinken)和国防部长奥斯汀(Lloyd Austin)。这将是拜登今年1月就任以来美国内阁官员首次访问日本。

上一次“2 + 2”会议是2019年4月在华盛顿举行的。此次“2+2”会议日本和美国的外交和国防部长预计将重点关注中共多次对尖阁列岛周围海域的入侵，两国计划共同对抗中共，加强对中共的威慑。四位部长将重申《美日安保条约》(U.S.-Japan security treaty)第5条，条约规定了美国对日本的防御义务，涵盖了在日本被称为尖阁列岛，中共称之为钓鱼岛的无人居住的岛屿。他们还将对中共今年2月颁布的《海警法》表示关注。该法律将中共海警定位为准军事组织。

中共新的《海警法》允许海警在外国船只在中共国声称拥有主权的水域从事非法活动时使用武器。而中共声称，位于东中国海(East China Sea，中共国称东海)、由日本政府管辖的尖阁诸岛(Senkaku Islands，中共国称钓鱼岛)及其附属岛屿是中共国领土的一部分，这项立法可能针对在这些无人居住的岛屿周围航行的日本船只。此举引发了东京方面的抗议。

日本政府近期多次抗议中共国海警进入尖阁诸岛附近海域并驱赶日本渔船。日方担心，日本渔民和日本海上保安厅的巡视船可能都会成为中共国《海警法》的袭击对象。茂木敏充称日本将“冷静和坚决地”回应任何可能的中共国海岸警卫队保护其领土的行动。2月23日，五角大楼发言人约翰·柯比(John Kirby)批评了中共海岸警卫队入侵尖阁列岛周围领海的行为 “可能导致误判和潜在的物理和物质伤害”。

日美2+2对话是安全政策方面的最高双边协商会议。因一直以来，日本和美国的供应链都依赖中共国，双方一直在努力避免紧张局势蔓延到经济领域，因此在以往会议上并没有明确提出对抗中共的海上扩张。2017年对话的联合声明只提到日本和美国将为“东海的和平与稳定”共同努力。2019年对话的联合声明表达了“对改变东海现状的单方面胁迫企图的严重关切和强烈反对”，仅仅间接对中共海上扩张表示关切和反对。

此次2+2会议的联合声明将见到日本和美国对中共策略的转向。预计日美在此次联合声明中将向中共发出明确信号。3月3日，拜登政府制定了一份国家安全战略纲要，将中共列为在经济和技术等各个方面挑战国际秩序的唯一竞争对手。声明强调，美国将努力塑造新的国际规范和协议。拜登政府还宣布了一项政策，对美国依赖中共的所有半导体、电池和稀土产品的供应链进行审查。

日美两国政府计划对香港局势和新疆维吾尔族人权问题表示关切。他们还将讨论朝鲜的核项目和导弹项目，以及太空和网络安全等新领域的防务合作与双方的疫苗合作，以共同应对中共的“疫苗外交”。日本和美国还会与澳大利亚、印度进行拜登政府后四方安全对话的首次峰会，四方联盟将就安全合作问题进行商谈。
