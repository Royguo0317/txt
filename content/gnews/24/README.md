![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/03120738/unknown-3-1.png)

中共国医生石正丽是 2014 年与美国医生联合开展冠状病毒项目的团队的一员，但该项目因风险太大而被国土安全部关闭。在美国的研究项目被关闭后，石博士在中共国武汉继续她的冠状病毒研究。

来自中共国的石正丽医生是包括拉尔夫·S·巴里克 （Ralph S. Baric ）医生在内的团队的一员，该团队在 2015 年的《自然医学》杂志上发表了一篇文章。

在文章中，他们讨论了显示出人类出现潜力的蝙蝠冠状病毒。 这篇文章发表于 2015 年。

2014 年，HHS 致函北卡罗来纳大学教堂山分校，宣布他们将取消对该项目的资助。该信确认出自拉尔夫博士。

在美国停止研究后，中共国继续推进该项目，并在武汉病毒学中心进行研发。从石正丽的论文和简历可以看出，他们在实验室成功分离出病毒，正在积极尝试物种传播。

同样重要的是要注意，早在 2017 年，我们就掌握了关于中共国一家高度安全的病毒学研发中心病毒泄漏的可靠情报，该病毒泄漏导致 SARS 病毒外泄并导致人员死亡。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/03121951/unknown-4-1.png)

这一信息提供了一个基础，该理论与 COVID-19 是一种变体的理论相矛盾，该变体只是在野外蝙蝠中神奇地发生了变异，然后在人类吃了一碗美味的蝙蝠汤时跳到了人类身上。

从 2014 年以来与该项目相关的所有已发表研究论文中，很明显 COVID-19 已经在实验室存在。

很难判断病毒泄漏是有意还是无意。

2017年，中共国也发生过类似的病毒释放事件，导致8人患病，1人死亡。它始于中共国疾控中心实验室的两名工作人员独立分离和体外/体内实验一种 SARS 病毒。在其中一次会议中，科学家们采用了一种以前未知的 SARS 病毒变种，并将其从 BSL-4 高防护设施移到了两人正在工作的低安全性的腹泻研究实验室。显然，病毒灭活过程没有正常进行，两人都在实验室被感染，然后继续感染实验室外的其他人。

昨天我们注意到，位于中共国湖北省武汉市的武汉病毒研究所正在招聘人员，于 2019 年 11 月进行蝙蝠迁徙和病毒传播的生态研究。 我们现在从石博士的简历和论文中知道，她当时仍在研究冠状病毒：

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/03122604/unknown-5.png)

根据国家评论获悉：

2019年12月24日，武汉病毒所发布了第二个招聘启事。 翻译的该帖子包括声明，“对携带重要病毒的蝙蝠病原生物学的长期研究，证实了SARS、SADS等重大新的人畜传染病的蝙蝠来源，发现并鉴定了大量新的蝙蝠和啮齿动物新病毒。”

直到一月份发生的事情仍然是个谜。 但是，现在我们有更多的证据表明中共国多年来一直在武汉的一个实验室研究冠状病毒。我们也有 2017 年实验室冠状病毒造成死亡的证据。
