5月1日郭文贵先生盖特爆料，国内银行对私人贷款基本停滞，社会融资借款利息高达30%比比皆是。仅3月份1000多亿，4月份600多亿资金逃到加拿大。大额资金流入虚拟货币包括喜马拉雅币中。中共高层、大公司的大额资金逃往日本。其他中共中层、企业家大量逃往加拿大、欧洲国家。有钱者资金疯狂外跑，权贵者想尽办法藏钱。这是任何一个王朝崩溃前的一个绝对征征兆。民生银行多个股东大额贷款者全部限制出境。仅过去三周对民生银行边控4000多人，直接在边境拦下200多人。

与文贵先生爆料相呼应，自由亚洲电台4月30日报道，中国人民银行、银保监会、证监会、外汇局等金融管理部门，在4月29日，对在国内具有很大影响力的13家网络平台企业实际控制人或代表进行了约谈。

这是继约谈蚂蚁集团之后，对服务于中小企业客户的民间金融企业的又一次大的动作。被约谈的企业有腾讯、度小满金融、京东金融、字节跳动、美团金融、滴滴金融、陆金所、天星数科、360数科、新浪金融、苏宁金融、国美金融、携程金融等。

根据官方统计，中国微信用户已超过12亿，淘宝超过8亿，QQ接近6亿。互联网的巨大影响力已经远远超过传统媒体，成为中共国民众获取新闻资讯的主要途径。因此中共当局一直担心某一突发性事件，或负面舆论引起民意沸腾，产生难以预料的后果。

浙江大学的社会学学者陆镇宁说：“在经济领域国进民退的趋势更加明显，在加速进行。作为互联网企业，也是经济比较活跃的领域，北京政府肯定会不遗余力的加以控制，加以限制。下一步很有可能将其国有化。国有资本更多的参股，甚至控股。”看来，为了控制一切，中共当局正为这些大型民企国有化作准备。

虽然，曾经完全国有化的中共国和现在的朝鲜就是一面”历史的镜子”，但”制度的自卑、信仰的自卑、心灵的扭曲、精神的分裂”，使中共产生了一种病态的控制欲和占有欲。为此，不惜”颠倒日月”、”反转历史车轮”，但这能得逞吗？
