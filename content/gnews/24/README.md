# 中共国对十名逃离香港的人判处刑期

[!\[\]()!\[\](https://spark.adobe.com/page/r4bHg2Bpv9A9z/images/86b0d46f-f4ba-42d5-bc06-9eb3625f71bb.png?asset_id=fe637f81-1e0f-44ac-9dc2-6a386d397253&amp;img_etag=%22b4a2fcdb758db01bb668f808e8f94a4c%22&amp;size=1024)](https://spark.adobe.com/page/r4bHg2Bpv9A9z/images/86b0d46f-f4ba-42d5-bc06-9eb3625f71bb.png?asset_id=fe637f81-1e0f-44ac-9dc2-6a386d397253&amp;img_etag=%22b4a2fcdb758db01bb668f808e8f94a4c%22&amp;size=1024)

香港（美联社）-周三，中共国一家法院判处了十名曾试图搭乘快艇逃离香港的亲民主和抗议人士七个月到三年的徒刑，此案对香港具有重大的政治意味。

南部城市深圳市盐田区法院对两名被控组织者给予了最严厉的刑罚，由于他们在命运不佳的8月23日企图达到自治的台湾。亲属说，所有被告均认罪，此举显然是为了获得较轻的判决。

据信，被告担心他们会因支持香港民主运动而被起诉。香港媒体报道说，至少有一个人可能因六月份北京对这个半自治领土实施的一部严厉的新国家安全法而被捕。

组织者被判处两到三年徒刑，而其他八名参与者则被判入狱七个月。

当他们的船被中共国当局拦下时，船上共有十二人。法院说，已经为两名未成年人举行了不公开的听证，虽然这两人承认了非法越境的罪行，但不会受到起诉。

周三，内地当局将这两名最年轻的，年龄分别只有17和18岁被拘者移交给了香港警方。

警方在香港的新闻发布会上说，他们可能因潜逃而在香港面临额外的指控。

商业罪案调查科署理总警司郑丽琪表示：“我们将向法院申请继续羁押这两人，明天将这一问题提交法院。”

警方说，二人有机会通过电话与家人交谈，亲属们能够在周三探望他们。

被判处七个月徒刑的黄伟贤和李子贤的家属在一段录音中说，他们希望大陆政府就探视权做出明确的指引。录音由正在协助这些家庭的团体“12港人关注组”发布。

根据录音，被判处七个月刑罚的郑子豪的父亲说，“每天都是不公正的”。

被告的亲戚说，他们被禁止雇用自己的律师，而且这些指控是出于政治动机。被告可能会因越境而被判处最高一年监禁，而组织偷渡则可能会被判处七年监禁。

他们未经许可越过海上边境进入中共国大陆水域被接走。虽然香港是中国的一部分，但旅客往返于从陆地和海洋都环绕着香港的大陆时仍必须通过移民局入境检查。

这些判决似乎是警告持不同政见的活动家们不要企图逃避国家安全法的执行。

“中国是法治国家，有法必依，违法必究。中国司法机关严格依法办案。“中共国外交部发言人汪文斌在例行新闻发布会上说。

香港已经冻结了数名已逃往国外的反对派的资产，并发出逮捕令。其中有逃向英国，该国曾统治这片领土直到1997年移交给中共国。

香港被承诺将在移交后的50年内维持其独立的政治、经济和社会体系，包括比中共国大陆更大的言论与抗议自由。批评人士说，中共国的做所作为，包括实施国家安全法、广泛逮捕批评家和取消立法会选举，几乎使这一承诺无效。

美国和几个欧洲国家呼吁释放这十二个人返回香港，称对他们的审判是不公平的。欧盟在一份声明中说，除了不能自己选择法律代理人外，他们与政府任命的律师的接触也受到严格限制。
