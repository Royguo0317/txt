# 重要事件

- ** 03.17 大纪元网 [中共侵犯人权 欧盟同意制裁四官员一实体](https://www.epochtimes.com/gb/21/3/17/n12817506.htm)**


两名欧盟外交官表示，欧盟大使周三（3月17日）同意将四名侵犯人权的中共官员和一个实体列入黑名单。这是自1989年天门广场镇压事件后欧盟对北京首次实施[制裁](https://www.epochtimes.com/gb/tag/%E5%88%B6%E8%A3%81.html)。

欧盟27国大使周三同意以侵犯人权为由，对和中共相关的个人祭出新一轮[制裁](https://www.epochtimes.com/gb/tag/%E5%88%B6%E8%A3%81.html)。对包括四名中共官员和一个中国实体实施旅行禁令和资产冻结。预计欧盟外长将会在下周一正式批准。在此之前，被制裁者的身份不会被公开。

“对于严重侵犯和践踏人权的行为，将采取限制性措施。”一位欧盟外交官说，欧盟大使批准的这份制裁名单中除了四名中共官员，还包括来自俄罗斯、利比亚、南苏丹和朝鲜等国官员，共计11人。

- **03.17 GNEWS [日美高官在内阁级国事访问中强调对中国的担忧](https://gnews.org/983479/)**


在本周早些时候举行的日美2+2联合记者会上，美国国务卿安东尼-布林肯（Antony Blinken）和国防部长劳埃德-奥斯汀（Lloyd Austin）与日本同行讨论了共产党中国在印太地区构成的特殊威胁，反复强调了自由开放的印太地区的愿景。日本是自美国总统拜登（Joe Biden）1月上任以来，两位官员在首轮内阁级亲自国事访问中访问的第一个国家。

“我们一致认识到，中国的行为在与现有国际秩序不一致的情况下，对[日美]同盟和国际社会构成各种挑战。我们仍然反对任何试图改变现状的单边行动，包括在东海和南海。我们对中国海岸警卫法也有严重关切。”日本外相本木敏光表示。

在中共的指挥下，中国海岸警卫队不断在日本尖阁诸岛海域巡逻。这些行动是违反国际法的。

防卫大臣岸信夫说：”我重申了用一切手段保护日本领土的决心，美国提到《日美安保条约》第五条适用于尖阁诸岛，我们能够确认对日本防卫的承诺”。

美国国防部长奥斯汀称，日美同盟是美国支持自由开放的印太地区战略的基石。

“我知道日本和我们一样，对中国破坏稳定的行动感到担忧，正如我之前所说，中国是国防部的步履挑战。而且我们知道，在当今不断变化的全球动态中进行竞争，只能通过团队精神和合作来完成，而这正是我们与日本联盟的标志。”奥斯汀先生说。

美国国务卿布林肯还提到了中共侵蚀香港的自治和台湾的民主，以及在新疆和西藏侵犯人权的问题，以及日美同盟与韩国在保卫印度洋-太平洋地区和确保其人民福祉方面的作用。

两国还谈到了朝鲜无核化的重要性。

美日两国对中共政权的态度听起来令人鼓舞。不过，他们对中共的态度似乎没有美国国务卿蓬佩奥和川普政府那么强硬。

- **03.17 CNN [美国在布林肯与北京会晤前制裁24名中港官员](https://www.cnn.com/2021/03/17/asia/us-hong-kong-china-sanctions-intl-hnk/index.html)**


在美国国务卿安东尼-布林肯（Antony Blinken）本周晚些时候在阿拉斯加与中国高层外交官会面之前，华盛顿已经对24名中国和香港官员进行了制裁，原因是北京正在对香港进行镇压。

这些制裁是根据《香港自治法》出台的，该法是美国去年通过的，以回应北京在香港实施的禁止分裂、颠覆和勾结外国势力的新国家安全法。被制裁者包括中国最高决策机构之一的25人政治局委员王晨，以及起草国家安全法的委员会中唯一的香港人谭耀宗。

国务院周三公布的名单上的许多人物此前已被川普政府禁止与家人一起前往美国。根据HKAA的指定会带来金融制裁，这可能包括限制银行业务、从美国金融机构贷款、购买财产或与美国实体交易。

Blinken在一份声明中说，此举是为了回应北京对香港民主出台的新限制，这将进一步限制香港人的选举权。布林肯表示，新的制裁措施凸显了 “我们对全国人民代表大会3月11日单方面破坏香港选举制度的决定深表关切。这一行动进一步破坏了向香港人承诺的高度自治，剥夺了香港人在自身治理中的发言权，英国已宣布此举违反了《中英联合声明》。”

- **03.17 BreitBart [北京因香港抗议电影提名而禁播奥斯卡奖](https://www.breitbart.com/entertainment/2021/03/17/report-beijing-bans-oscars-broadcast-over-hong-kong-protest-film-nomination/)**


据报道，中国专制政府周三下令国家电视台不得转播下个月的奥斯卡颁奖典礼，因为其中一部被提名奥斯卡的影片是关于香港抗议运动的。

脸皮薄的中共还对最佳影片提名者《游牧民族》的导演赵克柔怀恨在心，因为她几年前批评了中共。

据报道，让奥斯卡被中国电视台禁播的影片是由挪威的安德斯-哈默（Anders Hammer）执导的一部关于2019年香港抗议活动的短纪录片《不要分裂》。这部35分钟的专题片追溯了抗议活动从起源于对一项有争议的引渡法的反击，到去年夏天中国对香港实施暴虐的 “国家安全法”，有效地将政治异见定为犯罪，并粉碎了香港有限的自治权。

Hammer周三告诉香港自由媒体，他的目标是 “捕捉香港街头的激烈气氛，因为年轻的抗议者正冒着一切风险为自己的未来而战。”
