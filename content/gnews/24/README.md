近期，中共国出口到世界各国的海运费涨至十多年来的新高，从而引起全世界海运费暴涨。上周开始深圳盐田港至阿尔赫西斯拉的运费达一万美金的新高位，这周至苏丹的运费也进入万元美金时代。除此之外，宁波到英国菲利克是托的海运费业已涨超一万美金。

另一方面，各国港口大量集装箱堆积，一些国家港口拒绝空柜入港，另有港口开始收取拥堵费，大部分的海运公司缩短免箱期。各种花式做法，层出不穷。为何突然间出现了这种情形？

在此谨做几点原因分析：

1. 因为疫情封城导致上半年中共国出口到各国的货物停滞发货，解封后，圣诞旺季各国都增加订货量，短时间集中发货所带来的影响。
2. 中美贸易战加上中澳关系恶化，最近中共国陆续在进口冷链食品检测出病毒，直接导致中共国进口物品的急剧下跌，进口货物减少。笔者在中共海运公司的朋友那里得知，现在的情形基本是出去3.5个货柜，只能回运1个货柜。
3. 因为集装箱进出港的比例悬殊，集装箱工厂生产订单爆满，产量完全供应不了市场。再者由于暂停从澳洲进口铁矿石，导致钢铁涨价，空集装箱出厂价也出现新高，形成恶性循环的新高度！
4. 由于这几个月疫情缘故，还是有很多国家的港口没有全负荷工作，导致港口的集装箱积压，出港货柜不及时、港口堵塞，空柜无法及时装船；还有很多航运公司的海员相继有曝出检测出病毒阳性而隔离治疗等等因素导致船只回航无法准时装满货柜起航。


大部分海内外华人对中国从世界进口产品所知不详，仅知道中国是世界工厂，出口量世界第一。可事实上中国出口到世界的产品大部分都是技术含量低的轻工业产品，比如服装鞋帽、床单被套、箱包、小五金、玩具、小家电之类的。但是中国从海外进口的产品就完全不同了，除了我们熟知的高端芯片，医疗设备，还有各种原矿，木材，大部分的农产品，肉类、海鲜等等，同时中国的种猪都是从海外进口。此类多为民生产品。

在中美贸易战以前，一箱难求、海运费暴涨3-6倍、空柜生产供不应求的情况从未有过。大家或许可以从进出口的巨大差异中看出端倪，人民币升值到6.5时代，但是中共国的物价并没有下跌，相反的还在大幅上涨。

中共国的小农种植，注定农产品的产量没有美国这种大规模农场机械化种植产量高。中共国几千年来以种植为主，土地已经贫瘠，加上近几十年的化肥农药的滥用，种植技术的落后，导致中共国的土地污染严重，农产品不但产量不高，质量也很难提高。进口一批种猪只能培育供应市场3-4年的猪，然后又要重新进口。正如中共国官方所言，土猪目前称比大熊猫可能都稀少，土猪的生长周期长，耗费饲料多，如果中方停止进口种猪，土猪饲养将增大饲料消耗。黄豆、玉米基本都靠进口，加上贸易战引起的进口急剧减少，国际海运费的暴涨，直接引起大米、黄豆、玉米等大宗农产品的价格飞涨。玉米和黄豆是食用油的主要来源，二级产品是饲料的主要来源。最近中共国境内的这些农产品的价格也开始报道涨价，这一系列的生态链断裂呼之欲出。

这一切与每个人息息相关，中共国的粮食危机早已开始，只是普通百姓在生态链的末端，暂未感知到而已。由于计划生育导致目前中共国主流家庭一般都是3口之家，居住面积小，工作压力大节奏繁忙很多人习惯外卖的生活方式，在家里囤积物资的习惯基本没有。无论从哪一方面来说，目前中共国的这种困局都会继续下去，且会持续恶化，短时间之内看不到转机。等最末端的大部分老百姓意识到了粮食等民生产品买不起买不到了，那将为时已晚，定会引起一系列的次生灾难与人道危机。这是所有人都不想看到的。爆料革命路德社一直在呼吁屯粮，就是为了让大多数人远离次生灾难。 爆料革命新中国联邦，一直在最前沿为中华民族遮风挡雨，为建设没有中共的新中国而负重前行。为了唤醒更多的同胞，我们需要行动、行动、行动！
