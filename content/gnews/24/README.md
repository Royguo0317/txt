2012年9月11日班加西當地時間晚間，四名美國公民在一起針對美國駐班加西領事館和附近另一建築設施的襲擊中被殺害，其中包括時任美國駐利比亞大使克里斯托弗∙史蒂文斯。時任美國國務卿是本屆美國川普總統在2 016年總統大選中的競選對手，民主黨人希拉里·克林頓；時任美國副總統是即將到來的2020年美國總統大選中川普總統的競選對手，民主黨人喬·拜登。

讓我們通過相關郵件解密和美國政府的文件回顧一下班加西事件的整個經過。

在2012年9月11日當地時間凌晨4點至凌晨6點左右，兩名美國駐班加西領事館的安全警衛發現一名身著利比亞警察制服的可疑男子正從領事館附近一幢興建中的別墅高層內以手機拍攝領事館的照片。安全警衛因而拘留了他一小段時間，但很快便予以釋放。該名男子隨後駕駛警車離去，並向利比亞警察局投訴。已經逝去的領事館負責信息情報事務的通訊官肖恩·史密斯注意到了這名男子的行為，認為他可能在勘察地形，因而於中午左右傳送了一封訊息給他的朋友，表示“希望我們能活過今晚。我們剛剛發現一名負責維護領事館周邊治安的當地警察用手機拍攝領事館的照片“。正如他所預料的，當晚該領事館即遭到了一夥約由125至150名穆斯林極端恐怖組織武裝分子的襲擊，駐防於領事館外圍負責安全保護的利比亞當地民兵組織烈士旅在襲擊開始後便全部逃離，該夥武裝分子隨後迅速攻入領事館。當時負責保衛大使的美國特勤人員將大使、信息情報官員、一名資訊管理官員領入緊急避難室，而其他人員則來到另一棟建築物取出戰術防衛裝備。由於遭遇大批武裝分子的強力火力阻擊，他們試圖營救大使的嘗試失敗且只能暫時撤退並尋求其他支援。這時，駐美大使及通訊官仍被困在緊急避難室，且恐怖分子開始點火焚燒建築物。事後，克里斯托弗·史蒂文斯大使被發現躺在一個只有窗戶可以進入的房間內，被當地民眾救出並送往醫院，但因昏迷不治逝世。通訊官肖恩·史密斯的屍體則在當晚由特勤人員在返回領事館時被找到。

在襲擊領事館之後，該夥武裝分子隨後又襲擊了距離領事館1.6英里的一處美國中情局附屬基地，企圖攻擊駐地的幾十名美國情報人員。駐地特勤人員只能依靠自身火力保衛，直到一組來自利比亞首都的黎波里的增援小分隊自費支付30,000美元徵用了一架噴射飛機到達班加西，並彙合利比亞政府軍趕到後才得以擊潰武裝分子。此時距離襲擊發生已經過去將近12小時，且兩名外聘特勤人員前海豹突擊隊官兵葛蘭·多赫提和泰隆·伍茲在戰鬥中犧牲。

現在，讓我們來看一下班加西事件前後，美國時任奧巴馬、拜登、希拉里政府在班加西事件前後是如何履行了自己的職責:

1. 時間撥回到2012年6月份，班加西當地已經發生過一起針對英國大使的小型襲擊。當時駐防的民兵組織全部逃離而沒有履行他們的職責。雖然英國隨後從班加西撤離，但這也從側面支持了一種觀點，即必須有可以信賴的、本國的強有力軍事力量來保護各國在利比亞的存在，而不能過多依賴本地軍事組織。據一位拒絕透露姓名的美國官員稱，美國相關部門可能沒有意識到負責保衛駐美使館的當地民兵組織烈士旅背後的伊斯蘭極端主義背景。根據時任美國國務卿希拉里·克林頓的口述，她在該事件引起人們對利比亞美國基地安全問題的關注之前，完全沒有註意到駐利比亞大使史蒂文斯發給國務院的外交電報。因此，她根本就沒有了解到史蒂文斯大使在電報中提到的對班加西當地日益惡化的安全形勢及領事館自身薄弱的安全防衛的擔憂。正因為如此，美國是幾個西方國家中僅有的依賴當地的民兵來駐防外圍防線的。如果一位國務卿連自身職責範圍內的外交電報都沒有閱讀，連本國駐外人員的安全都無法保護，可想而知她怎麼可以在總統的位置上盡職服務這個國家的公民。

2. 根據已經披露的信息，來自利比亞首都的黎波里的美軍增援小分隊是自費支付30,000美元租用飛機到達的班加西，而距離班加西最近的美軍駐意大利西西里島的錫戈內拉(Sigonella)基地到班加西的飛行直線距離與的黎波里相仿，且當天即有一支美國陸軍突擊隊入住。要知道，駐紮在班加西領事館的外交安全勤務局的干員以及區域安全官已經於攻擊事件爆發的第一時間便知會了位於華盛頓特區的總部。當時，利比亞當地時間約為晚間9:40分，而美東時間則為下午3:40分左右。當時班加西領事館人員以“恐怖攻擊”一詞形容該事件。美國時任最高軍事長官和執掌外交和情報的官員在得到信息後，僅僅命令美國非洲司令部派出”掠食者”無人機採集信息而不排除任何正式的支援軍事力量。他們認為依靠領事館附近的中央情報局工作站的軍事能力就可以控制當時的局勢，這是如何草率和失職啊？殊不知，上百名訓練有素的武裝分子對美國領事館和大使的襲擊，等同於攻擊美國本土，尤其是在本·拉登在2011年被擊斃後。國務卿希拉里確認她在9月11日美國東部晚些時候曾經和奧巴馬總統通過電話，雖然通話內容尚未解密，但我們可以看到的結果就是沒有實質性的軍事幫助被提供，最終導致四名美國公民犧牲，其中包括時任美國駐利比亞大使。

3. 時任美國總統奧巴馬及副總統拜登是在9.11這樣一個特殊的日子，在美國駐外領事館遭到恐怖襲擊，大使下落不明的情況下完全沒有把保護美國國家利益和公民人身安全放到首要位置。

4. 根據官方公開的報導，基地組織頭目本·拉登於2011年5月1日被擊斃。自2010年12月底爆發的“阿拉伯之春“運動在短短幾年內推翻了突尼斯、埃及、利比亞、阿爾及利亞、也門、蘇丹等六國政權，很多北非和中東地區的國家至今仍然處於動蕩之中，敘利亞就是其中之一。根據聯合國西亞經濟社會委員會發布的報告，“阿拉伯之春”運動僅自2011年至2016年已為一些地區帶來高達6,140億美元的經濟損失。班加西事件恰恰就發生在阿拉伯之春進行得如火如荼之時。作為全球第一大國的執政內閣，奧巴馬、拜登、希拉里內閣理應將更多力量投入到地區和平和穩定上來，而不是一再美化自身正義使者的形象，但事實上卻不斷撤離或削弱美國在衝突地區的存在。這將不可避免的導致專制力量被推翻後，而取而代之的卻是軍閥割據及人道危機。

綜上所述，我們可以看到，即使是世界第一大國的領導人，當他們沉迷於政治圈宣傳成為口炮黨，並喪失價值公知和基本的決斷力時，價值觀和國家安全也會受到嚴重威脅。據社交網絡最新披露的信息，艾倫·哈羅·帕羅特，來自CIA的吹哨人，告訴查爾斯·伍茲（在班加西遇害的泰隆·伍茲之父），奧巴馬和拜登政府向伊朗派發了1,520億美元，以掩蓋海豹突擊隊在中東行動中被襲擊和其他有關基地恐怖組織的內幕。艾倫承諾很多相關資料將在隨後陸續公佈於眾。

筆者認為屆時所有人將會對奧巴馬、拜登、希拉里政府進行一次重新的評價，是否有黑金交易？是否有掩蓋事實？一切有待揭曉。
