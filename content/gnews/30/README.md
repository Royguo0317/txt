1.美国彻底看清“王杨”补牢，为时晚矣

美国之音于3月19日报道：美中两国本星期五结束了在阿拉斯加举行的拜登政府执政以来的首次面对面高层会谈。尽管这次会议无果而终，但美国国务卿布林肯在会后表示，美方达到了举行这次对话的两大目的：即向中方明确表明美国及其盟友与伙伴对中国的行为保持严重关切；同时清楚地表明了美国的政策、优先事项和世界观。美国国家安全顾问沙利文指出，美方会前与会后都保持着清醒的认识，将基于这次会谈的情况，在与盟友和伙伴国磋商之后决定下一步行动。

美方原本并不打算与中共国举行双方会谈，所以并没有为到访的中共国代表团安排晚宴，这让浸淫华盛顿政坛数十年的杨娘娘颜面扫地。

2.中美外交陷入历史冰点

路德社在昨日节目中说布林肯和沙利文只作为美国代表前来向中共国传达美国政策和优先世界观，并不是来跟中共国进行现场辩论的，无论中共国如何表演都没有意义。双方既没有召开联合记者会，也没有发表联合声明，美国此前与日本、韩国结束会谈时都发表了联合声明，这意味着中美此次会晤彻底谈崩，不可能再有任何进展，中美外交陷入历史冰点。

3.杨洁篪和王毅谩骂式的开场是在向美方宣战

中共国很清楚因为中共病毒问题，注定美国后续要采取切实行动。杨洁篪和王毅参加这次会谈就是在演戏，演给中国老百姓看。他们谩骂式的开场也是在向美方宣战，明确中共国将奉陪到底。

从中共内部政治角度来看，杨洁篪一直是中共安插在美国最重要的外交官。在去年“120”之后，杨洁篪肯定被中共委以重任，要通过运作平息中共病毒问题。时至今日，美国对中共国采取的一系列外交政策验证了，杨洁篪在美国耗费巨资的运作彻底失败，国务卿布林肯上任后首次出访竟然直接跳过中共国访问日本。习在看清目前中美僵局后开始不信任杨洁篪，所以习特派王毅随同前往监督杨娘娘的工作，杨很有可能面临被习清算的结局。

4. 中共妄图重新建立国际秩序

中共国之所以始终强调要回到以联合国为核心的国际秩序，就是因为中共国在联合国收买了众多亚非拉穷兄弟。中共妄图以此重新建立国际秩序，习总加速师现在不仅想在中共国当皇帝，还想在全世界当皇帝，但是，他忽略了中共国之所以有今天的经济成就是得益于美国主导的现有国际秩序，一旦北约组织与亚太联盟共同对中共国发起行动，中共国经济及中共政权很快就会土崩瓦解。

5. 馬來西亞與北韓斷交

朝鮮官媒—朝中社於3月19日週五報導，朝鮮外務省當天發表聲明稱，朝鮮決定斷絕與馬來西亞的外交關係。 原因在於馬來西亞法院判決將一名涉嫌洗錢犯罪的北韓公民引渡到美國。 從北韓政府的過激舉動可以判斷這位北韓公民的身份極為特殊，對美國有非常重要的情報價值。 在這個關鍵節時刻馬來西亞向美國拋出橄欖枝，可見美國加強亞太地區掌控權的戰略措施已經初見成效。

6. 楊潔篪王毅謾駡式開場白內容解讀

3月18日，中外辦主任楊潔篪、外交部長王毅同美國務卿布林肯、安全顧問沙利文在安克雷奇舉行中美高層戰略對話。 在美方發表開場白後，楊潔篪首先發言，中方的謾罵式開場白震驚在場所有記者。

記者本應該在雙方會談開始時撤離，但是布林肯國務卿主動邀請記者現場參會，中共國謾駡式的開場白讓所有在場記者極為震驚。 即使在川普政府時期，中共外交官也不敢如此叫囂美國政府。 中共國外交部網站謊稱本次會談是中方應美國邀請參加兩國會談，而實際情況是中共特地追到美國懇請與美方進行雙方會談。

從楊潔篪的發言內容可見，中共很清楚美國已經看清中共完美犯罪的陰謀，所以中共有意採取強硬態度批評美國，指責美國沒有給中共國定罪的資格，應該交由聯合國來定奪。 中共通過收買亞非拉國家來獲得聯合國選票優勢從而逃脫被定罪，這是中共完美犯罪計劃的一部分。 中共外交部發言人趙立堅在發言中說開場白只是開胃菜，後面正餐才是重頭戲，這所謂的正餐就是閉門會議要討論的中共病毒問題，趙立堅在兩國會談前強調中方專家已經收到聯合溯源研究報告，中共不斷強調病毒溯源問題要交給科學家解決，要以WHO的調查報告為準，美國的情報資訊不客觀。 中共很清楚美國想用當年對付薩達姆的方式對付中共。

雖然中共國表面上非常強硬，有非常嫻熟的談判技巧，但是美方在原則性問題，尤其是在中共病毒溯源問題上絕不可能有任何讓步。 中共的今天是美國華爾街一手扶持的結果。 中共國近幾十年積累的國民財富全被幾個盜國賊家族佔有，這讓美國非常容易瞄準把心，只要消滅這幾個家族就可以分贓，這也是中共盜國賊家族作繭自縛的結果。
