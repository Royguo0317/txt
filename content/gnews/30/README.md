# **霍利参议员承诺反对1月6日的选举认证**

[!\[\]()!\[\](https://spark.adobe.com/page/W6TKiVnhfPlKB/images/c1baf9ed-b88c-412b-b31a-dd1b1791e809.png?asset_id=c8d7d9ee-f0be-4d98-a733-106b34dff67e&amp;img_etag=%2295ca486da07e071eec08ad54674340b0%22&amp;size=2560)](https://spark.adobe.com/page/W6TKiVnhfPlKB/images/c1baf9ed-b88c-412b-b31a-dd1b1791e809.png?asset_id=c8d7d9ee-f0be-4d98-a733-106b34dff67e&amp;img_etag=%2295ca486da07e071eec08ad54674340b0%22&amp;size=1024)

参议员乔希·霍利（Josh Hawley）透露，他将在1月6日反对选举人团结果的认证。

霍利指出，他的决定是对“包括脸书和推特在内的超级公司为支持乔·拜登而干预本次选举的空前努力”以及“选民舞弊指控”的回应。

他强调，“在2004年和2016年选举之后，国会中的民主党人在认证选举票时提出反对意见。”

“由于这些原因，我将遵循民主党国会议员多年来的同样做法，并在1月6日的认证过程中提出反对意见，以提出这些关键问题，”他的声明总结道。

[!\[\]()!\[\](https://spark.adobe.com/page/W6TKiVnhfPlKB/images/280f09c4-3f54-488c-8578-20ddba50fa84.jpg?asset_id=bedc73ab-aaee-40a8-a305-7c9f707260c6&amp;img_etag=%229c2d9ea8262504759c8f7a29afc7612c%22&amp;size=2560)](https://spark.adobe.com/page/W6TKiVnhfPlKB/images/280f09c4-3f54-488c-8578-20ddba50fa84.jpg?asset_id=bedc73ab-aaee-40a8-a305-7c9f707260c6&amp;img_etag=%229c2d9ea8262504759c8f7a29afc7612c%22&amp;size=1024)

乔希·霍利@HawleyMO 数千万关注选举诚信的选民应该被倾听。我将在1月6日代表他们提出反对意见。 霍利关于在1月6日选举团认证过程中提出反对的声明

“在2004年和2016年选举之后，国会中的民主党人在认证选举票时提出反对意见以引起对选举诚信的关注。当他们这样做时，受到民主党领导人和媒体的赞扬。并且他们有权这样做。但现在，我们这些关注此次选举诚信的人们也有权做同样的事情。

“我不能在没有提出一些州，尤其是宾夕法尼亚州，未遵守本州选举法的事实的情况下，投票认证1月6日的选举团结果。并且在没有指出包括脸书和推特在内的超级公司为支持乔·拜登而干预本次选举的空前努力的情况下，我不能投票认证（1月6日的选举团结果）。至少，国会应该调查对选民舞弊的指控，并采取措施确保我们选举的诚信。但国会至今未采取行动。

“由于这些原因，我将遵循民主党国会议员多年来的同样做法，并在1月6日的认证过程中提出反对意见，以提出这些关键问题。”
