# 国家脉动：“支持拜登的共和党人”代表团，由中共资助访问中共国

《国家脉动》今天得以透露，一个由前美国国会代表组成的代表团，其中还包括了“支持拜登的共和党人”社团的筹款人，访问了一个中共领导的智囊团，该智囊团被美国政府标记为进行“秘密情报搜集”行动。

这次访问是中美交流基金会（CUSEF）活动的一部分，旨在“传播有关中共国的积极信息”和“有利报道”。

该代表团于2012年4月访问了中共国首都北京和湖南省，代表团成员包括美国众议院的五名前议员：康斯坦斯·莫雷拉，芭芭拉·肯内利，菲利普·英吉利，约翰·坦纳和马丁·弗罗斯特。

莫雷利亚（Morella）是五个人代表团中的两名共和党成员之一，她在2016年和2020年分别支持希拉里·克林顿和乔·拜登，并且代表拜登为“拜登的共和党支持者”组织募捐活动。她最近还与中共在国会山的代表一起参加了支持中共的政策的活动。

莫雷拉（Morella）曾经说过，“在诚信方面，我并不对川普有的崇高敬意，”但是她这次却访问了世界上最大的侵犯人权行为的国家：中共国，并发表了讲话表示支持。

根据2011年《外国代理人注册法》（FARA），游说公司BLJ Worldwide与其客户，此次旅行团的组织者中美交流基金会（CUSEF）都进行了注册。CUSEF是中国共产党资助的一项由董建华发起的基金，该基金经常针对美国的大学提供政策研究、高层对话和交流计划的资金。

中美交流基金会的主席董建华还担任中国人民政治协商会议（CPPCC）的副主席，该会议被美中安全与经济审查委员会确定为中共统一战线工作的重要组成部分，其主要目的是“采取行动消除潜在的、反对中共的政策和权威的根源。”

该委员会的报告补充说：“统战的战略是采用多种方法来影响海外华人社区、外国政府和其他组织，促使他们采取行动来支持对北京有利的政策和立场。” 监督“统一战线系统”，即“军事和文职官员协同实施在外国的影响力”。

E.美国前议员项目

美国的政府官员，例如众议院和参议院的议员，经常在全国和本地都非常活跃地参与政治活动。他们在美国政府运作方面的专业知识使他们在日常工作中独树一帜，可以直面我们的政客面临的问题和挑战。因此，他们经常与当前的政客保持密切关系，并担任华盛顿特区及其当地国家的顶级企业的顾问。

BLJ提议在2009年12月的试点计划中，邀请前美国国会议员到中国进行访问。前议员可以访问中国现任和前任政治人物，大学教授和学生，智囊团和商人，以加深对中国和平发展的了解。此外，文化活动和游览古迹也可以为中国的发展历史提供进一步的背景了解。这样的访问还将帮助中国官员和思想家了解美国对当前与中国有关的内政和外交事务的看法。

在访问之后，BLJ将与前国会议员合作，编写他们的旅行报告，并将这些报告分发给美国的政客、专家以及其他相关个人和组织。BLJ还将确保访问期间的媒体报道，这是CUSEF争取在西方媒体中获得“有利报道”和“传播正面信息”的更广泛目标的一部分。

时间安排：BLJ建议在2010年安排1-2次旅行，根据参议员在国会期间对美中关系的忠诚度以及对中国的开放态度来选择参加者。我们建议在2010年春季与4-6位前众议院代表进行一次旅行，然后在2010年秋季与4-6位前参议员再进行一次旅行。

注意：提议的前国会议员中国之行的实施以获得CUSEF批准为条件。

2011年备案文件显示：“在访问之后，BLJ将与前国会议员合作，编写他们的旅行报告，并将这些报告分发给美国的政客、专家以及其他相关个人和组织。BLJ还将确保访问期间的媒体报道”，这是CUSEF争取在西方媒体中获得“有利报道”和“传播正面信息”的更广泛目标的一部分。

他们参加了在钓鱼台国宾馆举行的中共国际友好联络会（CAIFC）会议，该国宾馆是中共用于正式国事访问的外交机构。
