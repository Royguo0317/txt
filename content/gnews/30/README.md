## 川普签署法案，违规中概股将被踢出美国证交所
` LONG JING` [轉載自GNews](https://gnews.org/zh-hans/664445/)

```
据白宫消息，川普总统于12月18日（周五）签署了一项法案，将禁止中共国公司在美国证券交易所上市，除非它们遵守美国的审计制度。
```



```
川普总统签署的法案名叫《外国公司控股法》。该法案规定，如果外国公司的证券连续三年未遵守美国公共会计监督委员会的审核规定，则该外国公司的证券将被禁止在美国的任何证券交易所上市。
```



```
此外，该法案还将要求上市公司提供相关信息，证明它们是否由外国政府拥有或控制。


尽管该法案适用于任何国家的公司，但毫无疑问，针对目标明显是在美国上市的中共国公司，例如电子商务巨头阿里巴巴、科技公司拼多多和石油巨头中石油等。
```



```
长期以来，中共一直以国家安全为借口，不允许海外监管机构检查当地会计师事务所，从而让中共国公司制造假账瞒天过海，在美国及其他国家大肆敛财。《外国公司控股法》的出台，宣告中共在美国铺开的财路行将终结。
```
