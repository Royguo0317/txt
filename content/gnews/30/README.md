布赖特巴特新闻(Breitbart News)报道，周三中共发布了一份指责美国人权的年度报告，中共将活跃而自由的政治辩论和美国的联邦制度列为侵犯人权的行为。

中共国务院新闻办公室是中共的正式宣传机构，每年都会发布一份类似的报告，通常是针对美国国务院的年度全球人权报告。此前有报道称，美国喜剧演员吉米·坎摩尔的电视节目侵犯了美国人民的人权。

今年，为了回应美国对中共在新疆维吾尔族和少数民族进行种族灭绝的担忧，中共急于发布此报告进行反击。该报告主要关注两项主张:一是美国对中共病毒疫情的应对侵犯了人权；二是美国对待其少数民族和种族的态度有着天生的偏执。

美国联邦制度允许各州独立于华盛顿决定自己的大部分命运，中共抨击美联邦制度，坚称真正的人权罪犯是美国政府。

报告在前言中宣称，“美国一直认为自己是一个例外和优越的国家，却发现自己的疫情失去了控制，并伴随着政治混乱、种族冲突和社会分裂。”“由于政府的鲁莽反应，疫情失去了控制，变成了一场人类悲剧。”报告中指责美国联邦制称，“没有国家指导方针，也没有有组织的努力来重新开放这个国家，除了（复制）各国已经采取的措施。”

报告强调，在美国大流行性流感预防控制方面，公共卫生官员，州长和美国国家领导人行为和态度不一致，甚至共和党总统和民主党，联邦政府和民主党州间矛盾频生。

在中共版“人权”报告中说到，美民主党和共和党之间的矛盾分歧越来越明显，已逐渐从政策分歧转变为身份之争，报告还将美国人购买枪支的合法权利描述为侵犯人权，并声称美国正在对其原住民进行“种族灭绝”。

这里提到的“矛盾”是美国政治体系的一部分，美国政治体系不会监禁公开反对执政党核心信仰的团体和个人。而在中共，公众的分歧通常会导致异见者受到刑事指控，被冠以“寻衅滋事”等定义不明确的“罪”。除了中共以外，没有任何政党在行政、立法或司法部门拥有权力。中共的各省完全归属听命于中央。极权专制国家指责民主国家因为“民主”而侵犯人权，中共的无耻真是超出想象。
