## 中共在阿富汗悬赏塔利班袭击美国军人

![]()![](https://gnews.org/wp-content/uploads/2020/12/capture-177.jpg)图片来源：美国之音 （图右为塔利班副酋长穆拉阿卜杜勒·加尼·巴拉达尔）
根据AXIOS近日报道，美国川普政府两位高级官员向AXIOS透露称，川普政府正在解密一未经证实的情报，该情报表明中共国出钱雇佣在阿富汗的非国家行为体（non-state actor，即那些具有重大政治影响力但与任何特定国家无关的个人或组织）来袭击美国军人。

**情报透露在敏感时期**

报道称，值得注意的是该情报的披露时间正好在总统第一任期结束前的21天，众所周知，在他表示要加大对中共国施压后，几个月来一直盛传是俄罗斯秘密买通塔利班武装分子在阿富汗执行杀害美军的计划。据悉，目前中共国驻华盛顿大使馆对此未做回应，而川普总统也没就此事与中共领导人进行讨论，而拜登方面是否知情也不是很清楚。

该情报包含在12月17日的总统简报中，国家安全顾问罗伯特·奥布赖恩（Robert O’Brien）也向川普总统口头通报了此事。多个机构的行政官员目前正在努力确认此情报的准确性。如果这一情报得到证实，这将代表对中共国的重大战略转变，并且将导致中美之间的紧张局势急剧升级；如果该情报不能被证实，则会引起人们对此情报来源的动机以及对其进行解密的决定的质疑。

曾参与最近关于中共国讨论的一位高级官员向AXIOS表示，和对待其他第一手情报一样，美国对首份报告都会持谨慎态度，但该官员也强调，所有与军队安全有关的情报都会被认真严肃处理。据另一位了解内部讨论情况的高级官员称，在12月22日，美官员召开政策协调委员会（Policy Coordinating Committee,简称 PCC）会讨论此事，一是确定与国际法院进行磋商来核实初始报告，二是就驻留在阿富汗的美军部队的安全保障形势与情报和国防部门进行商讨。没有官员就情报来源和细节做任何具体说明。

**中共一直向阿富汗非法运送武器企图破坏美国在阿的和平努力**

报道揭露称，中共国长期以来一直左右阿富汗的外交，之前曾以讨论阿富汗和平计划为由公开官方邀请塔利班领导人到北京访问，但与此同时，来自中共的资金和武器装备还是源源不断地输往阿富汗冲突地区，这与中共四处宣扬的“和平”努力很不一致。据德国马歇尔基金会，中国-阿富汗事务高级研究员安德鲁·斯莫尔（Andrew Small）称，中美间有意愿共同合作的重要领域已经少之又少了，谋求阿富汗和平还算是其中之一，而且中方知道美国正在逐步撤回驻阿美军，没有理由在此时惹事端。

有美官员称，美国有证据显示中共国试图通过提供激励资金或悬赏金的方式鼓动阿富汗“非国家行为体”对美国军人进行袭击，美国国家安全委员会(National Security Council)正在组织全面调查，但该官员并未特指这些“非国家行为体”是塔利班还是其他超出“非国家行为者”定义范围的人。同时，情报透露中共提出悬赏是在今年2月下旬美国和塔利班达成协议后，但具体什么时间并未提供细节。该官员还指出，此美阿协议后，没有美国士兵在阿富汗战斗中死亡。美国政府已获得较早有关中共国非法输送武器到阿富汗的情报。

英美两国政府此前曾谴责过塔利班使用中共国制造的武器。而中共为何如此关注阿富汗，据分析是部分源于中共想阻止阿富汗成为其所称的“穆斯林分裂主义团体”的基地。根据《印度时报》12月25日的报道，阿富汗安全官员称，近日在阿富汗，一专门针对维吾尔人的中共间谍集团被暴露。

**评**：正如路德先生在节目中所说，该篇Axios的报道比较“委婉”，说白了就是中共花钱给塔利班来攻击美国士兵。这再次验证了路德节目早在2019年说的，塔利班的背后就是中共，塔利班之所以能活得这么久就是因为中共作为金主在不断给钱，来制造各种恐怖活动。中共通过花钱形式来攻击第三国的驻军的是完完全全的恐怖主义行为。

川普政府正在解密该情报的消息被媒体爆出，其意义非常重大。根据路德社Bo博士讲解，情报界的事情一般都属于保密范围，都会保密处理。但是川普政府选择将这一消息在目前敏感时期来“解密”，它的性质和意义就会完全不同。结合近日美国商务部、外交部的行动，相信此消息将进一步加速对中共的“跨国恐怖组织”的定性。如果情报被证实正确，则意味着中共对美军的直接攻击，到时美国采取什么样的军事行动也不为过，热战真的一触即发。
