摘要：川普就职第一年的离职率达到了34%，每一次人事变动，都意味着施政的反复、推行政策的艰难。弗林案的目的不在于查弗林，而是希望弗林将军配合，通过伪证把通俄门的谎言圆通。班农则另外开辟了公共舆论的战场，并和郭文贵先生结盟。

2016年大选硝烟未散，更激烈的斗争已经开始。川普就职当天，哥伦比亚特区、纽约和美国许多大城市发起了反川普的游行。许多白左如丧考妣，为川普的就职痛苦不已。相当多天真的美国人和美国的阴谋家接受不了川普执掌白宫的事实。

作为手握联邦行政、联邦司法系统、美联储等几乎所有非选举官员提名权的总统，川普作为政治素人连整理名单的能力都不具有。作为总统，他要开始高度依赖工作团队，特别是现有官僚机器的运转。这些人是敌是友，在后面的任期中会协助川普成就大业，还是阴谋破坏，甚至关键时候反目，都不得而知。川普就职第一年的离职率达到了34%，可见第一个任期整合团队的难度。

川普内阁的人事任免包括：

- 国务卿：雷克斯•蒂勒森(Rex Tillerson)，熟悉俄罗斯事务，和建制派的外交理念相符，和班农的立场相左；继任者迈克•蓬佩奥(Mike Pompeo)，有深厚的军事和情报背景；
- 财政部长：史蒂芬•莫努钦(Steven Mnuchin)，前高盛高管、达沃斯人的代表，已卷入拜登家族丑闻；
- 国防部长：詹姆斯•马蒂斯(James Mattis)、马克•埃斯伯(Mark Esper)，均公开反川；
- 司法部长：杰夫•塞森斯(Jeff Sessions)，因在司法调查中回避，被迫辞职；继任者比尔•巴尔(William Barr)；([世纪大案 11：司法部长](https://gnews.org/zh-hans/409277/))
- 交通部长：赵小兰(Elaine Chao)，参议院多数党领袖麦康奈尔夫人，前小布什政府劳工部长，赵家和CCP江家有着几代人的深厚交情；
- 国土安全部长：约翰•凯利(John Kelly)，卷入非法游说1MDB和遣返郭文贵案；
- 幕僚长：莱因斯•普里巴斯(Reince Priebus)，是川普就任前的共和党全国委员会主席，后被国土安全部长约翰•凯利取代；凯利的问题暴露后，由马克•梅多斯(Mark Meadows)继任；
- 国家情报总监：丹•科茨(Dan Coats)，因丑闻离职；继任者为前众议院情报委员会成员约翰•拉特克利夫(John Ratcliffe)；([世纪大案 16：众议院情报委员会](https://gnews.org/zh-hans/438204/) )
- 中情局长：蓬佩奥任国务卿后，由吉娜•哈斯伯(Gina Haspel)继任，卷入阴谋推翻川普内阁的丑闻；([世纪大案 17：贼喊捉贼](https://gnews.org/zh-hans/441412))


最后，也少不了班农本人被排挤出白宫。川普内阁的每一次人事变动，都意味着施政的反复、推行政策的艰难。更不用说内阁之外的人事任免或提名了。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08020447/200213160743-flynn-trump-barr-split-exlarge-169.jpg)弗林、川普、巴尔 图片来源：CNN
川普排干沼泽还没开始，就遇到了弗林案。川普提名的国家安全顾问麦克•弗林(Michael Flynn)上任不到一个月，便被设计下台，成为奥巴马-拜登留守政府暗中打击川普的第一位受害者。弗林案的目的不在于查弗林，而是希望弗林将军配合，通过伪证把通俄门的谎言圆通。前奥巴马政府的许多高层官员均参与了弗林案，这使其成为比水门事件更恶劣的美国政治丑闻。2020年，弗林被平反，川普称：“这是美国历史上最大的政治犯罪。”

通俄门如同体制的肿瘤，贯穿川普第一个任期([世纪大案 15：FBI往事 世纪大案](https://gnews.org/zh-hans/432315/)，[16：众议院情报委员会](https://gnews.org/zh-hans/438204/)，[世纪大案 23：FISA的前世今生](https://gnews.org/zh-hant/494113/) )。众议院情报委员会公开了通俄门调查笔录，内容显示：奥巴马政府的高级官员承认，找不到任何有关2016年美国总统大选期间川普与俄罗斯勾结的证据。

2020年5月7日，美国司法部撤消了对弗林的指控，司法部长巴尔公开为弗林辩护。时任代理国家情报总监理查德•格伦内尔(Richard Grenell)解密了一份当时提出揭露要求的奥巴马官员的清单。5月13日有参议员公布了这份名单，时任副总统拜登、FBI局长科梅、CIA局长约翰•布伦南(John Brennan)、国家情报总监詹姆斯•克拉珀(James Clapper)、白宫幕僚长丹尼斯•麦克唐纳(Denis McDonough)赫然在列。

弗林案已经如此艰难，除了弗林案，还有司法部公布的尼基•戴维斯(Nickie Davis)、艾立•布罗迪(Elliot Broidy)案、罗杰•斯通(Roger Stone)、约翰•博尔顿(John Bolton)案 … …， 每个人的故事都说来话长。川普一边试图排干沼泽，一边发现自己提名的官员、任用的助手，乃至过去的朋友，许多人原来就是沼泽的一部分。

当白宫首席策略师班农被沼泽生物排挤出川普的核心团队后，川普继续在白宫孤军奋战。而班农则另外开辟了公共舆论的战场，并和郭文贵先生结盟。排干沼泽不再只针对美国，而是排干全人类的沼泽。
