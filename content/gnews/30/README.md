## **「黨替你說話」：中國共產黨的統戰制度和外國干涉**
![This image has an empty alt attribute; its file name is c4e67699-bf01-449c-b6d7-6e4fb0cbf541.jpg](https://spark.adobe.com/page/9NcnBpeLWGyCh/images/c4e67699-bf01-449c-b6d7-6e4fb0cbf541.jpg?asset_id=096ac78a-de3f-417d-a786-5c6b7d648bef&amp;img_etag=%22cfbda424b292f9b94d11da24b8cab9b4%22&amp;size=1024)
「統一戰線…是鞏固黨的執政地位的重要法寶…也是實現中華民族偉大復興中國夢的重要法寶。」

——[中國國家主席（中共總書記）習近平，於2015年中央統戰工](https://web.archive.org/web/20190826053157/http:/www.tibet.cn/cn/news/yc/201712/t20171222_5282108.html)作會議

中國共產黨通過選派少數民族、宗教運動人士以及商業、科學、政治團體的代表，加強其在國內外的影響力。它聲稱有權代表這些團體發言，並利用他們聲稱其合法性。這些工作由統一戰線系統（下簡稱「統戰」）來推進。統戰系統是一個負責影響黨外團體的黨和國家機構網絡，特別是那些聲稱代表民間社會的團體。它管理並擴大了統一戰線，一個致力於實現黨的目標的實體聯盟。中共開展這些活動，稱之為統戰工作，但其作用往往是隱蔽的或欺騙性的。

近年來，與統戰有往來的團體和個人因其參與政治干預、從事經濟間諜及對大學校園施加影響而受到前所未有的審查。在澳大利亞，與統戰工作部（UFWD）關係密切的商業人士成員被指控出於中共國利益而干涉澳大利亞內政。在美國，至少有兩名統戰科學家組織的高級成員因涉嫌偷盜技術而出庭受審。孔子學院在UFWD的大力滲透下，由於其對學術自由和校園的影響而引發了十多年的爭議。許多中國學生及學者協會是中國留學生的團結陣線，他們參與壓制學術自由，動員學生在澳大利亞、加拿大、英國和美國從事民族主義活動。

Covid-19中共病毒大流行也凸顯了海外統戰網絡。在澳大利亞、加拿大、英國、美國、阿根廷、日本和捷克共和國，一些團體被[動員](https://archive.vn/Rgr5v)起來，從世界各地搜刮稀缺的醫療用品，然後運往中國。這些行動似乎與統戰機構中華全國歸國華僑聯合會的指示[有關](https://archive.vn/w2rtr)。黨中央把該聯合會[描述](https://archive.vn/Rgr5v)為”黨和政府聯繫海外同胞的橋梁和紐帶”。病毒在全球傳播後，統戰組織開始與中共合作，向世界各國捐贈物資，宣傳中共關於這一流行病毒的統一口徑。

![This image has an empty alt attribute; its file name is 4e768253-c8bb-4c65-a686-f0a2c0b67d99.jpg](https://spark.adobe.com/page/9NcnBpeLWGyCh/images/4e768253-c8bb-4c65-a686-f0a2c0b67d99.jpg?asset_id=3b1b195d-77c2-4a72-87d6-be8923355c8e&amp;img_etag=%22778089c0ebc80bd8e8dffb0706928839%22&amp;size=2560)

無論這些活動是否有損防控病毒的努力，它們似乎都讓各國政府大吃一驚，並向其顯示了統戰工作的高效率。中國共產黨試圖干涉僑胞社區，影響政治制度，秘密地獲取有價值的敏感技術等行為，只會隨著中共國與世界各國之間愈發緊張的關係而加劇。隨著各國政府開始面對中共的海外干涉和間諜活動，理解統戰制度將是至關重要的。

我今天發表在澳大利亞戰略政策學院（[Australia Strategic Policy Institute）的文章](https://www.aspi.org.au/report/party-speaks-you)，剖析了中國共產黨統戰制度及其在對外干涉中的作用。本文描述了統戰系統的一系列機構和目標，並說明瞭該制度的結構、運作方式及其力求實現的目標。文中還揭示了在統戰系統內的數十個機構，在技術轉讓、大外宣、干預政治制度甚至影響跨國公司高管等方面起到的作用。

統戰制度幾乎一直是中國共產黨的核心制度。 在其歷史上大部分時間都由政治局常務委員會的一名成員領導，政治局常委是該黨的最高領導機構。然而，習近平總書記比前幾位領導人更加強調統戰工作，自2015年以來，他通過提升統戰的地位，使其工作更接近於共黨革命時代的重要性地位。

統戰工作的範圍在不斷演變，反映了中國共產黨的全球野心、對自身安全面臨的內部威脅的評估以及中共國社會觀。

統戰是一種政治模式，是共產黨控制政治代表權的一種方式——通過統戰工作所針對的團體發聲——意味著其對外擴張正是對外輸出中共政治制度。總的來說，海外統戰工作將促使中共對外國政治制度中的政治表述及代表權方面產生不當影響。
![This image has an empty alt attribute; its file name is 903e89f0-fe99-44d9-a68b-25874dd283ae.jpg](https://spark.adobe.com/page/9NcnBpeLWGyCh/images/903e89f0-fe99-44d9-a68b-25874dd283ae.jpg?asset_id=d8efa78e-4378-441f-b26f-961c84323631&amp;img_etag=%22ec0108b0c7879441d320580c6308e0fb%22&amp;size=1024)
習近平重振這一體系，突顯出各國很有必要對中共在全球範圍內的技術轉讓操作，作出更有力回應。然而，各國政府仍未能對其進行有效管理，同時也鮮見對統戰系統的分析。這種信息的缺失可能導致西方觀察家低估統戰制度的重要性，並將其體系中的各種手段歸為常規操作。例如，外交官可能認為統戰的工作是”公共外交”或”外宣”，但他們意識不到相關秘密行動的嚴重程度。安全部官員可能對犯罪活動或間諜活動有所警惕，但卻低估了一些公開活動能促成這些行為的重要性。分析師[可能會忽視](https://sinopsis.cz/wp-content/uploads/2019/11/ep.pdf)中共影響力中互相關聯的因素，正是這些因素相結合才真正起作用。

各國政府應該摧毀中共統戰網絡，防止其利用某些人物和機構作為秘密影響及技術轉讓的工具。政府應首先著力培養對外國干涉的分析能力。在此基礎上，他們應發表宣言性政策聲明，制定框架對抗它。對策應包括政府相關領域的執法、立法改革、威懾力和各項能力建設。各國政府應通過參與和審慎用詞，來遏制統戰工作對僑胞社區可能產生的分裂影響。

執法雖然至關重要，但不應是全部或主要的解決方案。外國干涉通常發生在一個無法通過執法行動來解決的灰色區域。對統戰工作的反制必須讓民間群體、媒體和華人社區參與進來。他們應設法將針對干涉機構的懲罰性措施與支持和接觸受統戰工作影響的社區同時放到計劃中來。有效打擊對外國干涉，是保障中國裔公民真正參與政治的關鍵。
