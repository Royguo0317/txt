2020年6月4日7點37分，新中國聯邦成立！

三年來，在郭文貴先生和斯蒂芬班農先生爲代表的“爆料革命”吸引並凝聚了來自牆內牆外、世界各地擁有信仰、正義、追求民主自由的億萬戰友。並由戰友們自願成立了喜馬拉雅監督機構，這成爲捍衛人們民主自由，保護財富安全以及與世界各地有志之士進行溝通交流的橋梁。伴隨著文貴先生宣布新中國聯邦成立，“爆料革命”書寫了曆史性的新篇章。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/05221610/1-40.jpg)

開幕式中數架直升機盤旋空中，懸挂著新中國聯邦國旗，海面周圍以LadyMay爲首數艘戰艦海中護航，紐約市中心大屏及市出租車隨處可見喜馬拉雅宣傳短片，場面聲勢浩大，氣勢如虹。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/05221652/2-33.jpg)

新中國聯邦國旗又稱49宇宙星旗或喜馬拉雅星旗，大海之藍，生命淵源，象征永恒、寬廣、自然、神秘、浪漫與夢幻，太陽之金，象征財富、平安、民心、權利、珍貴與喜悅。長寬比3:2，由49顆金色喜馬拉雅星雙橢圓環繞呈中字型圍合而成，以天地互通之勢，顯陰陽平衡之象，共同構成對稱、穩定的宇宙能量空間，每一顆星是生命的象征，天賦人權，衆生平等；中間星代表信仰，也稱信仰之星，凝聚民心，捍衛正道主義，彙聚天下與世界各國世代友好相處。整體造型形似宇宙星空，又似喜馬拉雅之巅。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/05221722/3-14.png)

開幕式中，前亞洲國腳郝海東先生與斯蒂芬班農先生宣讀了中文版與英文版的新中國聯邦宣言，郭文貴先生咬破手指，歃血爲證，以示滅共決心。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/05221806/4-14.jpg)

新中國聯邦宣言中主要包含新中國聯邦的願景與喜馬拉雅監督機構的承諾，在願景中提出參照西方民主法治體系，在相應國際法及國際機構與喜馬拉雅監督機構的監督下，制定憲法、建立三權分立正體，通過一人一票的方式産生新政府，選舉與彈劾制度並存。新憲法提到了對人權、法治、信仰自由、言論自由及私有財産的保護，同時也涉及了教育、養老、醫療、環境保護等民生基本需求保障問題，同時還對香港、澳門、台灣、西藏等地區有美好的展望以及對中共盜國賊集團掠奪的財産進行追繳並還富于民的迫切需求。
