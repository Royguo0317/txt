**西班牙20202年12月23日电/西喜社——**

白宫备忘录的起草者认为，联邦选举认证由担任参议院议长的副总统迈克·彭斯（Mike Pence）左右。 此外，彭斯拥有决定是否拒绝问题州选票的唯一权力。 但是，法律要求他必须在12月的第四个星期三（今年12月23日）这样做。
据Axios报道，白宫消息人士透露，川普总统对他的许多所谓的“盟友”尤其愤怒，这些盟友没有为捍卫选举公平而进行积极的斗争，最著名的便是彭斯副总统。
据称，总统从反川普集团的“林肯项目”发布的广告中摘出一则声称彭斯正在“离他而去“的广告。 消息人士称，川普认为彭斯“没有足够的战斗力” —就副总统而言，他自大选以来没有就选举诚信发表任何公开评论。
林肯项目（Lincoln Project）是美国政治行动委员会，由多个共和党人和前共和党人于2019年末成立。 在2020年总统大选期间，该组织旨在防止唐纳德·川普连任，并在美国参议院击败所有争取竞选连任的共和党人。 2020年4月，委员会宣布了对民主党总统候选人乔·拜登（Joe Biden）的支持。
彭斯除不关注选举外，一直专注于其他任何事情，包括花时间在乔治亚州进行竞选活动，着重于政府的反堕胎政策，并作为中共病毒工作队的负责人，致力于生产中共病毒疫苗。
副总统还计划访问以色列，巴林和波兰，出发日期定在1月6日，即他打算主持国会联席会议并制表和核实选举团票数的同一天。 一位彭斯的盟友告诉Politico，这次旅行的时机“不可能是巧合。”
如果在选举诚信上不做任何事情，彭斯将是认证拜登为美国第46任总统的人。 彭斯在选举人团投票认证中将扮演的角色显然在总统看来“已开始扩大”。 Axios指出：“川普可能认为彭斯……认证选举结果而最终背叛。”
但是，《国家档案》（National File）获得的白宫备忘录草案说，尽管副总统可能是在川普棺材上钉下最后一颗钉子的人，但他也可能掌握着生存的关键。 在备忘录中，川普的盟友透露，彭斯不仅有权拒绝来自违宪州的选举人团投票，而且有责任“这样做”。
仅凭彭斯就可以拒绝涉嫌欺诈性选举人团认证，川普总统的选举命运被摆在彭斯的肩膀上。
