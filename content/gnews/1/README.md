**西班牙2021年3月16日电/西喜社——**据美联社3月16日报道，瑞典周二成为最新的暂停使用阿斯利康中共病毒疫苗的国家，欧洲监管机构正在根据一些该疫苗接种者的危险血栓报告审查安全数据。

欧洲药品管理局计划周四召开会议，审查专家对疫苗的调查结果，并决定是否需要采取行动。

瑞典公共卫生局周二表示，将暂停使用阿斯利康疫苗，等待欧洲药品管理局会议结果。德国、法国、意大利和西班牙周一暂停使用该疫苗。

同样是在周二，德国政府表示将推迟到欧洲药品管理局会议之后召开一个虚拟的州长峰会，以审查该国的疫苗接种工作。该峰会原定于周三举行。

阿斯利康公司的疫苗是获准在欧洲大陆使用的三种疫苗之一。但对该疫苗的担忧不断升级，给欧盟的疫苗接种行动带来了又一次挫折。

阿斯利康表示，没有证据表明该疫苗会增加血栓的风险。该公司表示，在整个欧盟和英国接受疫苗的1700多万人中，已经有37份血栓报告。

印度尼西亚周一暂停使用该疫苗，称正在等待世界卫生组织关于可能的副作用的全面报告。
