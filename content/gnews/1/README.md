**一. 先分享四个业界动向：**

**1. 美国司法部没收比特币**。

黑客通过黑客软件导致美国东海岸燃气管道关闭，勒索比特币。但是遭美国政府没收。美国政府并不是破解了比特币钱包，而是控制了黑客使用的云服务器。虽然没有透露技术细节，但是对于数字货币安全和合法化是一个利好。

比特币的地址和私钥有一定联系，但需要超级计算机的计算能力才能破解，量子电脑在未来可能可以通过一个比特币地址破解（倒推）其私钥，从而盗走数字货币资产。

香港也在策划港币数字货币EHKD，但是和中国人行合作，实际上还是中心化的货币。香港是自由贸易港，美元通用，是推广加密货币的良好土壤，香港是一个不可忽视的市场。

**2. 全球监管机构制定全球加密货币资本的最严格规则——巴塞尔银行委员会。**

为了规避洗钱、恐怖主义等风险，假币货币资本需要应用最严格的规则。很多机构和个人消费者也有兴趣开立数字货币账户。

**3. StateStreet（道富）银行设立了数字部门**。

这家总部在波士顿的老牌银行管理高达40.3万亿美元，决定建立数字部门和其他银行竞争。

**4. 加密货币纳入美国养老金**。

Coinbase与退休基金公司结为合作伙伴，将加密货币纳入美国401k养老金计划，资产规模约1.8万亿美元。

**喜马拉雅交易所的技术架构解析：**

- 中心数据库，中心化管理
- 撮合（引擎）交易，处理和保证交易安全性、稳定性
- 前端（网站登录，用户体验，产品经理）和后端（客服体系）
- 钱包系统——银行的保险库，冷钱包，热钱包
- 安全的加密，防止黑客攻击


**二. 德国开放机构投资加密货币基金**

德国法律允许金融机构20%资本投资到加密资产。

加密资产和数字资产有所不同，加密货币资产可分为Security token（股份）和utility token（众筹）。

区块链技术的演进：

加密货币阶段(1.0) &gt; 智能合约smart contract (2.0)阶段，3.0阶段则是去中心化应用，超过货币、融资和证券市场。未来将是industry 4.0技术，支持实业应用。

Web 1时代 信息技术 (Web,读) &gt; Web 2时代平台技术 (Web, App,读和写) &gt; Web 3时代代币经济 (token economy，读和写，端到端，没有中间环节)。代币将是Web 3时代的核心和“大杀器“App。

未来将是基于加密技术的经济体系，G系列喜币将是爆料革命战友的诺亚方舟。

区块链也可以理解为一个大的数据库，去中心化数据库下沉到区块链中，用区块链的存储方法进行存储。点对点数据传输模式，没有中间环节，不可串改。
