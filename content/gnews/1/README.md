说到中国经济发展反而让中共更加集权专制，我们不能不提及经济学家杨小凯先生，他在中国经济发展一片大好的情况下，坚定地认为：中国经济发展（注定）会进入“后发劣势”。

“后发劣势”是指经济、技术、制度都落后的国家，有极大空间追赶先进国家，如果不先学习先进国家制度，直接照抄其经济模式和技术，虽然会获得快速发展，但会进入抄袭技术和商业模式的路径依赖，丧失自我创新的动力，也没有创新所需要的自由环境，失去改革制度的能力（政府会有更大经济实力和技术能力加强专制），直至受困于制度无法继续发展，并产生更大问题。中共国的今天已经完全验证杨小凯先生的判断。

杨小凯先生从青少年起就真正忧国忧民，发现中共的专制危害性后，以原名杨曦光发表文章《中国向何处去》，引发全国震动，尤其极大地触动了四人帮，他们无法相信这篇文章出于19岁青年之手，把杨小凯和家人及他们怀疑的所谓“背后人”都抓进监狱。

杨小凯在10年牢狱中，向知识分子狱友学习，出狱后直接考入大学，后进入美国普林斯顿大学获取了经济学博士学位。杨小凯是中国人中尤其经济学领域难得的有自己原创理论的学者，对现代世界经济学有着重要贡献，并赢得世界经济学届极高的声誉和尊敬，他后来成为澳大利亚莫纳什大学首席经济学教授，哈佛大学客座教授，澳大利亚社会科学院院士。

茅于轼先生曾经建议中共国邀请杨小凯先生回国讲授发展经济学，但被中共政府拒绝。杨小凯先生一生未改忧国忧民本色，不畏中共强权，在89年六四远动后，率先联合海外学者为受害学生签名发声并捐款。

热爱思考、敢于说真话的他，利用回国讲座的机会，向北大等高等院校学生和学者阐述后发劣势的担忧，和中国应该先有“自由，共和”然后再考虑“民主和科学”的思想。令人痛惜的是杨先生因为10年牢狱之苦和出狱后夜以继日地学习和研究，不幸患病，在2004年英年早逝，享年才55岁。

而杨小凯先生深深敬仰的人就是哈耶克。杨小凯先生认为哈耶克的著作极大地震撼了他，让他有了深入骨髓的变化，并认为哈耶克对人类的影响，会与孔子对中国人的影响一样，深远而且无孔不入，只是影响的方向会截然相反。

哈耶克从以学术发展，让世界更加美好（普通人也能生活的自由幸福）的角度出发，努力研究植物学、人类学、法学、政治学、经济学、神经科学等多学科，进而深刻地理解人性、法律、政治与经济的关系，不仅分别获得法学和政治学博士学位，并因为经济周期理论获得1974年的诺贝尔经济学奖，更成为20世纪最伟大的政治思想家。

20世纪古典自由主义一代宗师哈耶克（Friedrich August von Hayek，1899-1992）

撒切尔夫人和里根总统都因为遵从哈耶克的自由市场经济和政治理念，分别使英国和美国摆脱通货膨胀，大量失业的经济危机状况，并创造两国多年不见的经济长期高速发展奇迹。

中国也是在文革后无以为继，实行有限的自由市场经济后，人民有机会通过自己的勤劳和才智获得更好生活，从而实现了中国经济发展奇迹。

回到“为什么经济发展让中国更加专制的问题”，可以结合这次美国大选，为什么全世界自由民主灯塔之地的美国会发生这么大规模舞弊，控制言论自由的乱相，面临全面左转的问题？也可以结合为什么苏联解体后，不再由苏联共产党领导的俄罗斯又进入普京的集权专政状态，这些问题哈耶克在几十年前就给了我们答案。

哈耶克早在1944年发表的著作《通往奴役之路》中指出：不论共产主义、社会主义、法西斯主义、纳粹主义（极端民族社会主义）都是不同名称下的计划经济的社会主义，与其他极权政体一样本质上都是奴隶制社会。那些打着为人民创造天堂般生活的人往往会行集权专政之实，让人民成为地狱里的奴隶。

马云和刘强东曾经说人工智能大数据可以真正实现计划经济，结果被该体制直接打脸并“消失”。

中国明清等不同朝代，都曾有过那时的繁华和强大，因为都是专制统治，本质上也都是奴隶制社会，比如《红楼梦》里富贵荣华，不可一世的的贾家，其身份是“包衣奴”，在现代中共国，就是“白手套”，名字不同，实质相同，都是今天富可敌国，荣耀加身，明天就可以因为统治者一句话，身首两处，家破人亡。

集权专制政府必然会与对立的自由民主制度天然为敌，英国作家乔治奥威尔的《1984》，通过小说的方式，生动深刻地描绘出集权专制社会用封闭、谎言、技术和暴力，颠覆伦理，钳制人性，不断为民众树立假想敌，制造恐惧，转移民众视线以维持其专制统治。《1984》中所讲的统治手段适用于任何专制集权社会，尤其是今天“老大哥时时监控“（人脸识别无处不在），“真理部”（宣传部）负责不断制造谎言，对真正自由民主国家喊打喊杀的战狼外交的中共国。

这次美国大选是自由民主力量和世界极权力量，有史以来在美国战场上的最大战役，自由民主力量以让人瞠目的方式失败，但也惊醒警示了更多人—自由民主不是免费的，更需要精心呵护、勇敢捍卫。

哈耶克说：那些以为牺牲自由就可以获得（生命或财产）保障的人，最后会发现，既没有保障，也没有自由。

今天不论集权专制国家里闷声发大财的白手套们，官员们，还是全世界普通民众，都要承受专制者使用国家力量的任意碾压，希特勒发动摧毁世界的二战后，美国大选，香港和席卷全球的CCP病毒都是最新的明证。

2019年高评分黑马电影《饥饿站台》中，代表社会最高层的人用布满食物的餐台（代表社会资源）给处于各层级封闭不自由环境中的人分发资源。接近顶层的上层注定可以方便得到更多资源，然后把剩下的资源和肆意凌辱向下传递，下面阶层必然因为得不到足够资源，或者完全得不到资源而饿死。不论顶层怀着多么美好的分配意愿（共产主义原则：各尽所能，按需分配），在不自由的封闭社会，统治者永远要做那个分配者，靠近他们的上层注定要保住自己的阶层地位—固化阶层，即使偶有阶层流动，越往下层，越注定备受欺凌，越会发生弱肉强食，互相残杀，尸横满地…….

哈耶克解释了计划经济会必然导致集权专制的原因：因为没有任何个人和团体能掌握市场运转所需的所有知识和信息，因此“计划体制”国家必然会照成巨大浪费，灭杀知识分工，导致真理终结的集权专制，在集权专制国家，优秀的人没法发展或生存，只有那些没有才能、依赖赏赐、阿谀奉承、毫无道德底线的人才可能获得统治者信任，注定社会道德沦丧，经济崩溃，民不聊生。

不论苏联，中国，北朝鲜，还是委内瑞拉，无不一一验证哈耶克的论断。

哈耶克对人类的另一重大贡献就是他发现了“导致人民不断被奴役”的重大秘密，即“货币发行的被垄断”，并提出“货币非国家化（去中心化）”的理论。他认为金钱是人类发明的最伟大的“自由”工具，只有金钱会向穷人开放，权力永远不会。但他知道货币去中心化这个理念对现代人来说过于激进，哈耶克特意从货币的历史、货币的功能、货币的本质通过《货币非国家化》这本著作进行阐述。

货币因为人类社会分工进行交易而产生，货币的产生远远早于国家的建立。历史上人类曾经以贝壳、烟草等物品作为货币，所以货币是用于标价、交易，是可用于存储财富，可以延期支付的商品。货币本质上是因为信任（信仰）而被大众接受、使用和追逐的商品。

直到金属货币出现后，人们担心金属货币的含金属量是否足值，所以国家背书的货币成为最受信任的货币，国家也在为货币背书的过程中，发现了发行货币的巨大利益，逐渐通过立法权和暴力工具，把货币发行权垄断起来，也让人们逐渐建立了国家发行货币天经地义的观念。

然而正是货币发行被垄断后，才周而复始地出现通货膨胀，即国家增发货币，导致物价上涨，比如原来1元钱一个的包子，货币增发后需要10元才能买到1个，国家也通过这种手段把百姓的90%财富合法秘密地偷走。统治者的统治也离不开利益集团的支持，通过货币增发输送给利益集团利益，也是其维护统治的重要方法。百姓没有选择的权力，无论法币贬值得多么厉害，也只能看着自己的财富被稀释，被掠夺，直到通货膨胀引发经济萧条，民不聊生。

哈耶克因此告诉我们：人类自由和私人财富最大的威胁是不受制约的公权力，虽然人类为此通过制度、法律对公权力进行了限制，但是对保护私有财产的一个重要的手段被长期忽略了—-即铸币权被垄断。

铸币权被垄断就绝不可能抑制住超发货币的冲动，这些货币就是以国家暴力为后盾，强制流通的“法币”。

**哈耶克提出：**货币作为交换中介商品，可以像其他商品一样由各个私人银行（私人企业）发行，接受市场考验和竞争，真正实现破除铸币权的垄断，即货币的非国家化（去中心化）。

因为市场的本质是自愿、开放、竞争，人们在市场的博弈是长期的，反复的，因此私人企业发行的货币会在市场竞争中优胜劣汰，公民可以根据该企业的信誉，币值的稳定程度来自愿选择，像其他商品一样，一家企业发行的货币，如果失去信用，丧失购买力，人们会毫不客气地抛弃它，改用其他私企发行的货币。

汉朝文景之治时就曾经由私人铸币，吴王刘濞和邓通是两大铸币商，因为互相竞争，那时的货币是历史上成色最足的货币。

Paypal，支付宝和微信钱包本质上也是与法币和其信用体系挂钩的私企数字货币，依托ebay，淘宝和微信的用户网络协同基础获得的信用。（中国政府只要是继续集权专政，如果使用数字货币，会因为计算机技术实现对百姓更加便利的财富剥削，人身控制。）

**根据哈耶克的阐述，它们分别可以理解如下：**

自由：只服从法律，不由他人专断意愿强制的，自发自愿的表达和行为。自由也意味着多样性，就像大自然因万物自由生发，才最有生命力。相信真正有共同信念的战友们只要有可发展的平台空间，就能自生自发出更好的生态秩序。

从苏联到美国的作家安兰德以她的方式—小说《阿特拉斯耸耸肩》告诉世界：用谎言、黑暗手段和暴力妄图控制世界的人一直存在；也告诉我们自由的可贵，对用谎言和暴力妄图控制世界的人绝不妥协的精神的可贵。安兰德那句“我可以为所爱的人死，但不会只为他们而活”的话，激励了无数热爱自由、有才华、责任感和奋斗精神的人。这次美国大选，我们也真切看到这些人：川普总统、班农先生、朱利安尼先生、 蓬佩奥国务卿，纳瓦罗先生 。。。。。。当然还有我们的郭先生，Dr Yan，路德先生，博士军团，郝海东先生，叶钊颖女士和其他一直在付出不言放弃的战友们。

**法律：**能够保护个人自由（言论自由，人身自由，财产自由，发展自由……），适用于所有人的事前规章。使得任何个人能十分肯定地预见当局在某种情况下会使用其强制力，并根据预见规划自己的个人事物；对政府的一切行为有明确约束的规章制度。

是否破坏“自由”是区分好坏法律的一个重要原则，比如大公司游说政府立法建立保护其行业垄断地位，就破坏了其他人想要进入该行业互相竞争的发展自由，就不是好法律。比如CCP为维护专制，制定香港国安法，限制和破坏了香港人的言论自由、人身自由，就是恶法。

**法治：**任何个人能十分肯定地根据法律预见当局在某种情况下会使用其强制力，并根据预见规划自己的个人事物；政府的一切行为受到法律约束的国家治理形式。

**共和：**是指人民除了认真负责地选举出的政府代表,代表自己行使保障国家安全，人民自由和公共利益的权力，更要重视政治、学习、讨论和积极参与政治。知道政府权力是公有物，国家的治理是所有公民的共同事业。

联邦制：是由两个或两个以上的政治实体（共和国、联邦州、省等等）结合而成的一种国家结构形式。各联邦只是让渡部分权力给联邦政府，联邦政府对外代表国家主权，但与各联邦分享国家主权。因为每个地区都有其特定的气候地理历史人文，该地区的人因更了解和对自己更负责，更能因地制宜地解决问题，影响地方政策。联邦制有利于维持一个国家多元社会的差异与多样性。

**民主：**民主的基石包括集会、结社及言论自由、平等的公民权、选举权和生命权。现代民主包括四个关键因素：（1）通过自由公正的选举产生政府 （2）作为公民积极参与政治和公民生活 （3）尊重和保护所有公民的人权 （4）法律和程序同样适用于所有公民。民主在自由中寻求平等，但民主本质上只是一种手段，在一定程度上可以让政府对其言行负责，但民主不足以保护自由。
