**撰稿：脐橙君**

哇，习一尊又开始长篇大论，指点江山，为国把脉啦。

根据路透北京5月28日报道，习一尊周五发表重要讲话，内容围绕“必须保持强烈的忧患意识，科技攻关要奔着最紧急、最紧迫的问题去，在石油天然气、基础原材料、高端芯片等关键核心技术上全力攻坚，并加快突破一批药品、医疗器械、医用设备、疫苗等领域关键核心技术。”

之后，新华社又引述了他在两院院士大会、中国科协第十次全国代表大会上发表，想要作为进一步的补充，内容称“创新链产业链融合，关键是要确立企业创新主体地位；要发挥企业出题者作用，加快构建龙头企业牵头、高校院所支撑、各创新主体相互协同的创新联合体，提高科技成果转移转化成效。”

说到这里，你们有没有感觉，一尊的讲话内容每次都差不多，似乎放在什么背景，什么主题下，都是能适用的“万精油”文章，帮他写文章的人真是好省力，就像找到致命的“舟山蝙蝠病毒骨架”一样，“骨架在手，什么都有”，不怕你们跟不上病毒节奏，就怕你们感叹怎么变异那么迅速。一尊也是同样的道理，一篇万精油文章的架构搭好，拿到世界会议，就强调中共的合作发展姿态，拿到国内会议，就强调落实创新，发展卡脖子技术，至于，你把脉之后的药方呢？哪有把完脉之后，就说一句“多喝水”当成药方的道理啊？那不是变成“骗子郎中”了嘛。

一尊说“科技创新成为国际战略博弈的主要战场，围绕科技制高点的竞争空前激烈。我们必须保持强烈的忧患意识，做好充分的思想准备和工作准备。”科技创新重要，幼儿园的娃娃都知道，要不怎么问他们：你们长大想做什么呀？十有八九都是“科学家”，那么对于幼儿园娃娃都知道重要的问题，是否应该解答如何使科技创新强大呢？如何在竞争中取得胜利呢？什么思想准备和工作准备需要做呢？一尊，你倒是把话说清楚呀。

一尊说：“科技攻关要坚持问题导向，奔着最紧急、最紧迫的问题去。要从国家急迫需要和长远需求出发，在石油天然气、基础原材料、高端芯片、工业软件、农作物种子、科学试验用仪器设备、化学制剂等方面关键核心技术上全力攻坚。”这是不是变相承认了中共在这些领域有很大的障碍，发展70年，改革开放的红利没有提升这些领域的竞争力，相反在耗尽红利的同时，又要重新在那么多领域奋起直追，我可否问一下，这些红利究竟跑进了谁的口袋？70年的执政，你们到底发展了什么技术？

一尊说：“要在事关发展全局和国家安全的基础核心领域，瞄准人工智能、量子信息、集成电路、先进制造、生命健康、脑科学、生物育种、空天科技、深地深海等前沿领域，前瞻部署一批战略性、储备性技术研发项目，瞄准未来科技和产业发展的制高点。”的确，这些都是西方国家正在布局和远远领先你的领域，你不服输可以，但是又要搞类似“芯片大跃进”的笑话，就真的把大家当傻子了。一个连走路没练好的人，能指望他成为跑步冠军吗？这么简单的逻辑，总不会没人懂吧。

接下去，重点来了，一尊说“要优化财政科技投入，重点投向战略性、关键性领域；并加大基础研究财政投入力度、优化支出结构，对企业基础研究投入实行税收优惠，鼓励社会以捐赠和建立基金等方式多渠道投入，形成持续稳定的投入机制。”这段赤裸裸地表达了下一步红色家族贪腐的方向，就是既要挖空现有财政，又要掏尽民间资本，两手都要抓，都不能有任何手软。

习一尊对方法论做了重要指示： “当今世界的竞争说到底是人才竞争、教育竞争。要更加重视人才自主培养，努力造就一批具有世界影响力的顶尖科技人才，稳定支持一批创新团队，培养更多高素质技术技能人才、能工巧匠、大国工匠。”既然准备闭关自主培养人才了，那么两耳不听窗外事，怎能造就有世界影响力的人才呢？

我转念一想：没关系，一尊到时说你是世界级的，你就是世界级的。难道你还敢有半点怀疑吗？就感恩戴德地享受习皇的恩赐吧。
