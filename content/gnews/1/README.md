## 【世事解评】习近平对全国乡村强化网格化集权统治

【关键词】中共中央，乡村第一书记，乡村工作队，脱贫

【摘要】本文通过中共最高层针对其最基层的政权单位——乡村的最新政策指示和实际行动意见，揭示了中共不得不加强乡村政权管理，进一步巩固中央集权统治的本质目的。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18102006/%E4%B9%A0%E8%BF%91%E5%B9%B3%E5%AF%B9%E5%85%A8%E5%9B%BD%E4%B9%A1%E6%9D%91%E5%BC%BA%E5%8C%96%E7%BD%91%E6%A0%BC%E5%8C%96%E9%9B%86%E6%9D%83%E7%BB%9F%E6%B2%BB-1.png)图片来源：https://www.resdc.cn/data.aspx?DATAID=287

据中共喉舌新华社电，2021年5月11日，中共中央办公厅，印发了《关于向重点乡村持续选派驻村第一书记和工作队的意见》（以下简称《意见》），“并发出通知，要求各地区各部门结合实际认真贯彻落实”。这次的《意见》，面向的是中共社会最基层的建制单位——乡村，表面上的脱贫致富帮扶，实则是为了加强中央集权对社会最小管理单位的直接掌控。

该篇《意见》虽然没有给出具体的选派何时开始，但是，从《意见》的发布机关和传播出口，可以看出是“最高指示”。

我们来看看这个“最高指示”究竟意味着什么？

一、“下派爪牙”，下乡暴露高层政体不稳

中共一贯喜欢派遣特派员，建立健全党支部，对全体中共党员和普通民众进行思想意识形态的禁锢洗脑和强化教育。党领导一切，党指挥一切，体现在中共的管理体系上，共产党的第一书记，是各级各部门组织单位的最大官员，高于任何其他的业务官职。

中共的集权统治，是把整个国家“网格化”，从省（或自治区），到市、县，最小到乡镇。乡和村是最小的政体单位。中共把整个国家分割成一小块一小块网格。通过从中央选派的特派员（“看护队伍”），把集权的长手伸向社会组织形态的最底层。换言之，中央集权的爪牙遍布全国每一个角落。

如果靠制度和良好的政策激励，能够激发基层工作的积极性，创造生产力，难道还需要靠集权统治的爪牙，下到地方去指导工作（“抽皮鞭”）吗？

如果上下级信息通畅，从下级逐级向上级汇报工作，不就可以免去大规模的自上而下的特派员进驻了吗？看来是自下而上的汇报体系出了问题，比如虚报和瞒报各种统计数据，就是中共国一贯的官场和政府职能部门的劣迹习惯。

当自下而上的正常的信息传递、汇报体系出现问题时，就不得不自上而下，选派信息员、特派员，还必须给予这些被派遣的干部们，一个本地最大官职的头衔——乡村第一书记。所以，这次中共在全国范围内，大张旗鼓地向中共政治管理体系中最小单位的乡村，选派第一书记，就是中共治国体系出了大问题的表现，尤其是基层信息情况不明，出现下面的实际情况传递不上去，上面的最高指示传递落实不下去的政治体系危机。

让我们具体看看都有哪些乡村被纳入这些网格：根据《意见》规定，被纳入网格化管理的乡村包括如下四个方面：

1. 重点关注真穷的：脱贫村、易地扶贫搬迁安置村（社区），加大选派力度；
2. 巩固脱贫任务较轻的村，可缩减选派人数；
3. 不听话的，调皮捣蛋的（对共产党党组织软弱涣散的）：常态化、长效化、全覆盖；
4. 想覆盖你没商量（其他根据需要选派）。


从上面四类被选派对象的特点，表面上有重大有侧重，实际上是全覆盖，不会有任何一个乡村被漏网。而且选派下去的人是书记挂帅，讲政治大过治理贫困。根据被选派乡村的具体特点，可以看出，只要说你需要被派遣，就必须得接受。接纳被选派的第一书记或者工作组，是一项基层必须服从的政治命令。

二、为新一轮的全民“上山下乡”做准备

这次选派工作队，是常态化的。虽然每一期的派驻只有2年的时间。被派驻的对象，包括了过去因为年龄问题“上不去”的剩余干部。这样，也解决了臃肿的干部队伍人员过剩的问题。中共官场体系庞大，人浮于事。剩余干部不在少数。

未来，其他的过剩劳动力，比如，因制造业外流和国际大环境封锁压力下，国内经济下滑所带来的大规模的失业大军，预计都将被赶到乡村去。等着这批书记和工作队把各地（重点）乡村给“理顺了”，理“听话了”，接下来，就可以派遣大规模的各类各级别的支扶帮衬队伍了。新一轮的“上山下乡”运动就要到来了。

三、不抓经济抓政治，墙国经济堪忧

习近平被爆料革命和新中国联邦冠以“习神”的荣誉称号后，似乎心里很受用，知道解决中共国的农村问题才是振兴经济、实现其强国梦的根基。但是，习神只知道抓政治，不懂经济，如何强国？

书记挂帅，各村为保乌纱帽都去搞思想建设了，谁还去真正做事，真抓实干搞经济？

试想，农村都派遣钦差大臣下乡挂帅了，各地村官都去搞思想建设了，意识形态取代实际工作一定会成为主抓农村经济的重中之重，粮食作物真的能成为主抓工作吗？会否重演中共国1956年的“大锅饭”造成大饥荒？笔者将严重关切。而且，未来各地区书记，将迎来一轮被考核掉、被轮换掉的风潮。基层原班人事变动不可避免，现有正常农村建设、管理体系不可避免地再次被外行领导内行所取代。

后果：下派党羽爪牙，人人自危

脱贫攻坚，靠的是正确导向，而不是高压的统治，更不是高喊几句习神语录。脱贫攻坚，更需要好的激励机制，鼓励生产和科技的创新，发挥劳动人民的积极性。而不是通过下派党羽爪牙，令人人自危。总之，希望习神领导的中共举国集权体制在临灭亡前不要继续使用高压、集权的“东厂”、“西厂”那一套封建社会的老套路来鱼肉百姓，想想自己下一步该如何面对政权的全面破产，而不是继续试图玩农村包围城市、上山下乡来巩固自己屁股底下的所谓“龙椅”。
