#### **可以看出，未知的犯罪者目前正在攻击医院所依赖的计算机系统，这是特别危险的。或者称其为犯罪。**

黑客攻击爱尔兰卫生机构，健康服务管理机构（HSE）。 *HSE*老板保罗·里德（*Paul Reid*）说，为了保护它们免受特洛伊木马的大规模攻击，已关闭了整个IT系统。 “这是非常复杂的攻击，而不仅仅是标准攻击。” 病毒疫苗接种程序不受影响。 针对病毒的疫苗接种将按计划进行。

诊所取消所有预约，孕妇和急诊患者仍将得到治疗。 黑客攻击的主要目标是IT系统中的信息。

黑客攻击爱尔兰卫生机构，健康服务管理机构（HSE）解释说，现在将与安全部门一起评估这种情况。 对于由此给患者和公众带来的不便，我们深表歉意。”

根据最初的发现，一周前，与美国最大输油管道上的黑客攻击一样，类似的勒索软件也被用于网络攻击。 据美国联邦警察称，这次袭击是由犯罪集团*“ DarkSide”*实施的，并导致了在美国东海岸加油站的抢购。

同时，日本技术集团东芝公司宣布，它已成为欧洲勒索性黑客攻击的受害者。 支付系统和复印机制造商的子公司东芝Tec于5月初遭到*“ DarkSide”*的入侵，该公司的法国代表说。 正如该公司进一步解释的那样，仅丢失了少量的工作数据。
