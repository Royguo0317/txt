# **红衣主教帕罗林在新的采访中谈及金融丑闻、中共国和伊拉克问题**

**这位身为梵蒂冈国务秘书的红衣主教说，世界正在经历大规模的危机，这些危机凸显了团结和对话的必要性。**

[!\[\]()!\[\](https://spark.adobe.com/page/HWr0zL9B1yoGU/images/e5a6af28-37d5-4fe5-9ed5-8bc5260087d0.jpg?asset_id=dbc97225-de78-4dcf-9d91-31a80826c185&amp;img_etag=%22e2cbc7f2a28f93a2fd2475cbf5e3db78%22&amp;size=2560)](https://spark.adobe.com/page/HWr0zL9B1yoGU/images/e5a6af28-37d5-4fe5-9ed5-8bc5260087d0.jpg?asset_id=dbc97225-de78-4dcf-9d91-31a80826c185&amp;img_etag=%22e2cbc7f2a28f93a2fd2475cbf5e3db78%22&amp;size=1024)红衣主教帕罗林（Pietro Parolin）近日接受法国天主教电视网KTO采访。视频截图来自KTO

梵蒂冈城（宗教新闻社） – 梵蒂冈第二高官、红衣主教帕罗林在最近的一次采访中谈到了该机构最近的财务丑闻、教皇即将进行的伊拉克之行以及梵蒂冈与中共国的争议性交易。

帕罗林(Parolin)说，尽管发生了震撼天主教会的“令人悲伤和痛苦”的财务丑闻，但组成罗马教廷的办公室和部门的改革“已经实现了”。他还谈到了教皇即将进行的伊拉克之行所带来的安全问题，以及天主教会在国际上所拥有的“道德说服力”。

这位身为梵蒂冈国务秘书的红衣主教还谈到了最近与中共国达成的一项关于册封主教的协议，该协议遭到了那些批评中共国缺乏宗教自由的人的反驳。

帕罗林表示，他“深刻尊重那些有不同意见的人和批评教廷对华政策的人”，但他说，他希望这项协议能“结出果实”，同时强调在他所说的循序渐进的外交努力中需要“耐心”。

***KTOTV推特：****改革#curie，#梵蒂冈的丑闻，#伊斯兰教，#中共国，但也有教会的国际行动，#天主教徒在社会中的作用……教宗的第一部长Pietro #帕罗林(Parolin)红衣主教在独家#采访中回答了KTO。2021年1月29日上午11：03*

教宗方济各在2013年的使徒训令《福音书》（EvangeliiGaudium，或称福音之乐）中概述了他对梵蒂冈众多的、有时是多余的办公室进行改革的计划，该训令敦促教会 “开启福音传扬的新篇章”。

帕罗林周五（1月29日）在接受法国天主教网络KTO采访时表示，方济各提出的许多改革措施已经颁布。他指出，金融改革设立了三个新的部门——经济委员会、经济秘书处和总审查长办公室，以解决梵蒂冈的经济问题。

在有报道称，梵蒂冈国务秘书处在帕罗林的领导下，在伦敦投资了大约2亿美元的高风险房地产交易，给梵蒂冈带来了相当大的损失——也让梵蒂冈陷入了尴尬的境地之后，金融改革变得更加迫切。

帕罗林说:“教皇希望直接解决出现的这些问题，也是为了让罗马教廷尽可能地透明化，这样它就可以真正地行使它的天职，这是对彼得的继承者的服务，但尤其是对福音的服务。”

方济各还改革了梵蒂冈的其他重要部门，帕罗林补充道：“ 大部分工作已经完成，现在还要采取其他步骤。”这些步骤中包括发布期待已久的使徒宪法，该宪法将正式确定最近的改革活动。

这位红衣主教说，这部宪法的临时标题为“Predicate Evangelii”，即“传福音”，很可能会在“今年年底前”发布。

在接受法国梵蒂冈记者Philippine de Saint-Pierre (内容)宽泛的采访时，帕罗林还谈到了教宗方济各3月5日至8日对伊拉克的教宗访问。他说：“教皇对这次伊拉克之行确实很感兴趣，尽管此行可能会面临挑战。”

1月21日，巴格达发生恐怖袭击事件，造成35名平民死亡，80人受伤，在这个暂时平静但曾经历过此类暴力行为的国家引发了不安。伊斯兰国组织宣称对这次袭击负责，这让教皇访问的安全措施受到质疑。

帕罗林说：“教皇想去伊拉克，我相信，特别是为了鼓励基督徒。”他提到伊拉克基督徒因战争和财政破产的后果而逃离家园。

他补充说，宗教间对话也将是一个关键方面，因为伊拉克是一个穆斯林占多数的国家，教皇方济各可能会促进政治稳定和改革。2019年2月，教皇与爱资哈尔大伊玛目谢赫•艾哈迈德•塔耶布共同签署了一份关于人类博爱的文件，帕罗林说，这份文件代表了“基督徒和穆斯林之间关系的重大进步”。

该文件促进了不同信仰的人之间的非暴力和容忍，同时加强了所有个人作为上帝的子女拥有人类尊严的共同原则。

“教会必须照亮和重新唤醒人们的良知，宣讲和宣布原则不是一件小事，因为这些原则随后会影响具体的行动，”红衣主教说，“想象一下，如果我们都相信普遍的博爱。肯定世界会改变，改变很多。”

帕罗林说，包括天主教会在内的所有宗教在当今社会中都可以“发挥作用”，因为许多公民认为世界是由他们的信仰所决定的。他补充说，特别是在政治往往“挣扎地”应对现代挑战的今天，“给宗教一个公共空间，让他们能够发出自己的声音，是重要的。”

红衣主教说，对于天主教会来说，推广其信息的首选方式是通过在国际组织层面参加条约和协议。他继续说，有时这意味着采取逆潮流而动的立场，就像天主教会在一些地方对移民和环境的立场一样。

“教廷和教会，不能辜负他们的使命，他们的使命有时会与霸权发生冲突。”帕罗林说。

梵蒂冈不仅与一些天主教信徒，而且与美国上届政府发生冲突的一个方面，是与中华人民共和国就主教任命问题达成的一项有争议的协议。这项协议一直保密，据说可以让中共国政府给教皇一份可能的主教候选人名单。

虽然在天主教会的历史上，双边协议对选择有权势的教会代表进行规范并不罕见，但反对中梵协议的人认为，该协议没有考虑到中共国的人权侵犯和对其地下天主教会的迫害。

帕罗林在采访中承认“情况极其复杂”，但他敦促：“要记住，这份协议不会、不可能成为解决中国教会不得不面对的所有问题的协议，但这只是试图改善教会状况的一小步。”

他补充说，世界正在经历大规模的危机，从环境危机到政治危机再到社会危机，但用教皇方济各的话说，“这也是一个机会”。他继续说，目前新冠肺炎的流行强调了团结和对话的必要性。

帕罗林说，教皇方济各在1月初接受了第一剂疫苗，同时鼓励其他人以他为榜样。“我们知道现有的所有反对意见，但最终我们必须相信，这种（疫苗）将帮助我们摆脱危机。”他说。
