耐克《行为准则》和《行为准则领导标准》禁止任何类型的监狱、强迫、抵押或契约劳动，包括详细的行动自由条款和禁止任何基于种族背景或宗教的歧视。我们将继续定期与我们的所有供应商接触，以评估是否符合耐克的《行为准则》和《行为准则领导标准》。

我们一直在与中国的供应商进行持续的尽职调查以识别和评估，与中国其他地区雇佣维吾尔族或新疆其他少数民族有关的、潜在强迫劳动风险。根据不断变化的信息，我们加强了审计协议，以识别与潜在劳动力转移项目有关的新风险。我们正在进行的调查没有发现，在我们供应链的其他地方，存在雇佣（被强迫劳动的）维吾尔族人或新疆其他少数民族的证据。

耐克与浩源鹏服装制造、青岛吉发集团、昌吉溢达纺织及其在新疆维吾尔自治区的任何其他设施没有关系。澳大利亚战略政策研究所的有关报道是不准确的。与泰光集团相关的是，当2019年有关新疆维吾尔自治区情况的报道开始浮出水面时，泰光集团已停止在新疆维吾尔自治区为其青岛工厂招聘新员工，而独立的第三方审计也证实该工厂已没有任何来自新疆维吾尔自治区的员工。我们正在进行的调查没有发现我们在中国供应链的其他地方，存在着雇佣（被强迫劳动的）维吾尔族或其他来自新疆的少数民族的证据。
