**原文翻译：**

一个由科学家和医学专家组成的研究冠状病毒来源的跨机构小组，在川普政府执政的尾声中，就如何报告他们的结论陷入僵局。

据熟悉专案组内部争执的人士透露，70%的专家希望公开报告，疫情背后的病毒是从武汉病毒研究所的高安全性医学研究实验室意外泄露的。尽管中国科学家拒绝合作研究病毒的基因构成，拒绝解释病毒的来源，但这一结论还是得到了支持。

中国政府还发布虚假信息，包括谎称美军将病毒带到中国。

这场幕后辩论已经进行了三周，包括一些直言不讳的反川普的科学家。

争论的焦点是，美国政府是否可以正式指定中共国为COVID-19爆发的原因，并追究北京的责任。中共国在全球范围内发起了一场公关活动，以转移对其处理和欺骗相关疫情的批评。

特别工作组的分歧反映了这个问题的极端政治化，许多人认为这是一个紧急的国家安全优先事项，值得两党支持。

反对实验室逃逸结论的是一个团体，他们坚持认为病毒是自然发生的人畜共患事件，从蝙蝠开始到中间动物宿主，然后到人类。约有15%的专案组成员主张这个结论。

第三种观点，也是得到情报界和军控界越来越多支持的观点，他们希望专案组的最终报告能够断言，该病毒（正式名称为SARS-CoV-2）是由中共国军方制造的，作为一种工程生物武器。估计有7%的专案组成员同意这一观点。

这一立场最早是由流亡的中国病毒学家闫丽梦披露的，她曾在中国实验室工作，并表示她确信冠状病毒是在中国实验室制造的。闫女士发表了一份广泛的科学研究报告来支持她的说法。

其余8%的专案组专家未作决定，他们希望得到进一步的数据，而中共国拒绝提供这些数据。

据有关人士透露，专案组包括一些反川普的专家和科学怀疑论者。

该小组内部的僵局意味着，其调查结果不太可能在川普总统1月20日离任前公布。这意味着，预计即将上任的拜登政府将对中共采取更加和解的政策，将不愿意最终公布专案组的结论。

专案组的工作因中共国政府的阻挠而受阻。北京官员拒绝合作研究病毒的来源，并继续坚持认为病毒始于武汉的一个 “不受监管的海鲜市场”—一年多前，该市场已被消毒，并清除了所有可能证明疾病爆发地点的证据。

疫情发生后进行的中国研究称，2019年12月在武汉首次发现病毒，已经排除了海鲜市场作为原因的可能性。试图报告海鲜市场来源异常的中国科学家，都被政府打压。

中共国政府也开始将疫情的责任从市场转移到质疑的说法上，即病毒起源于南欧，由海外工人传播到中国。

世界卫生组织正在对病毒的中心起点进行调查。北京起初阻止考察组入境，但中共国政府本周宣布，考察组已获准进入中国，并于周四前往武汉。

实验室泄漏起源理论，直到最近才由白宫副国家安全顾问马特·波廷格（Matt Pottinger）首次披露。

“有越来越多的证据表明，实验室很可能是最可信的病毒来源，”他最近告诉英国立法者。

卫生与公众服务部发言人将问题转给了国务院。

国务院的一位官员说。”国务院最近召集了一个外部专家小组，讨论冠状病毒的可能来源。虽然我们不透露私人谈话的内容，但讨论加强了我们的观点，即中共国需要透明化，并对武汉发生的事情进行全面彻底的说明。”

一位消息人士说，预计中共国将继续在病毒来源问题上进行搪塞，希望新政府放弃进一步调查的必要。

如果该病毒是一种生物武器，这是一个对国家安全有重大影响的结果，联邦政府一直无法确定，也无法确定来源和发起人。消息人士说，因此，美国很容易受到未来生物武器攻击的影响，因为美国无法发展必要的基础设施，进行必要的情报工作，以确定武器的类型和来源。
