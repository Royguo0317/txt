七哥谈沼泽的那期节目，让我们更加清晰这个世界的游戏规则。翟东升不过是个前台的表演者，对游戏规则却是无知。

中共认为他们是中国人中的精英，智力超群，对历史权谋的熟稔在他们的每个毛孔里都散发着傲骄的臭味。七哥揭穿了这种精神意淫，在沼泽面前，中共可以用，但不可信。

中共盗取政权后，沼泽为什么会给他们机会，发给他们管理许可证呢?

1. 信仰上，沼泽是绝对有神的热爱和信仰指针的。从二战希特勒第三帝国崛起被灭，铲除纳粹，建立以色列国，二战后民族解放独立。而中共是无神论，沼泽不可能喜欢他们。2000多年前商鞅和法家对我们民族思想的束缚，导致我们与其他民族相比，信仰的韧度不够，容易导致政治优先于信仰。由于信仰的深度不够，我们需要更多时间，需要七哥和爆料革命战友们站出来，沼泽才愿意看看我们。真是惭愧，我们的文化宏大，可是在信仰上却太少。

2. 组织上，中共以组织严密和情资收集的优势战胜了国民党，对人的思想把握超过一切组织。沼泽喜欢稳定，稳定才有钱，所以中共成了一个选择，但现在不是唯一的选择。

3. 从信用上，一个傻子知道自己傻，不装聪明人，长期有了傻子这个信用，那沼泽也相信你是真傻子。而中共吹自己是聪明人，比所有人都聪明，所有人跟他绑一起叫人类命运共同体。沼泽看中共真是呵呵拉。沼泽可以容忍你骗自个，但不能让你长期骗他们。

4. 从行事风格上，沼泽不喜欢张扬，而中共就爱万国来朝，普天同庆。这本没有什么冲突，但你放病毒，搞别人的选举，张口闭口核威摄，自以为是大撒币蓝金黄。核武器是大杀器，你扔别人家，自己也会受到波及，那些地方住着你们的大老婆，二老婆，三老婆……这种思维说服不了任何人。
