# **中共国正在推动军事霸权：北京高科技武器库和可怕的导弹几乎不可能被阻止**

- 中共国正斥巨资制造超级超音速导弹
- 它已经建立了一支无人驾驶飞机部队，并且拥有350枚核弹头
- 大学在国家军事力量方面发挥着核心作用


**疯狂**

中共国正花费巨资制造超音速导弹，其速度如此之快（高达声速的二十倍），以至于军事专家们认为没有任何方法可以进行拦截。

确实，一些分析人士担心人类对这种致命武器的反应能力不足，而针对这些致命武器进行防卫的唯一方法将是依靠人工智能和计算机系统。

超级超音速导弹速度高达每秒钟几英里，发射后几分钟内即可对目标进行令人惊讶的攻击，因此被描述为战争的“改变游戏的武器”。

美国战略司令部司令约翰·海顿将军三年前对参议院委员会说：“尽管美国也正在开发这种‘星球大战’式的武器，但我们不能否认，我们没有任何方法来阻止他人来部署这种武器对抗我们。”

这种能够携带核弹头的导弹可以对人员、车辆和建筑物进行精确的攻击。

为了测试这种武器，北京政府说，三年前，它正在建造一个风洞，可以模拟高达25倍音速的环境。一家承包商说，该风洞已经对高超音速导弹进行了六分钟的试飞。

这种使用了复杂的传感器、制导系统和创新的推进方法的超级超音速技术，其复杂性已经可以与制造原子弹相提并论。

**石墨烯**

这是一种革命性的材料，具有巨大的军用和制造潜力。厚度只有一个原子的大小，是人类所知道的最薄和最轻的一种材料，它能传导热量，吸收光，可延展并且比钢结实200倍。

它是由曼彻斯特大学的研究人员于2004年发明的，中共国家主席习近平对该实验室进行了正式访问。

它在军事上的应用包括弹道导弹的涂层，暴露于高温下的高超音速飞行器的布线，车辆的伪装以及部队的防弹衣。

中共国的报道表明，Z-10攻击直升机是波音公司“阿帕奇”的竞争对手，已经装备了北京航空材料研究所研制的石墨烯装甲。该研究所与英国的三所大学有联系，并且和这三所大学所属的两个研究所合作，这两家研究中心专门研究航空航天业中石墨烯的使用。

中共国媒体还报道了中共在南中国海人工岛上的军事设施上使用石墨烯涂层的计划。中共国政府在南海部署了装备有核导弹的晋级弹道导弹潜艇，引发了争议。

**监控科技**

最近以来，中共最恶化的趋势之一就是建立了一个监视国家，中共试图通过不断监视其公民的活动、思想和言语来控制14亿公民。

中共通过庞大的街头网络摄像头，面部识别技术，生物特征数据，官方记录，人工智能以及对在线活动的监视来实施跟踪，这些日常活动包括购物和外卖食品订购习惯等日常活动。

最极端的例子是在西部省份新疆，维吾尔人和其他穆斯林少数民族受到全天候监视。

大部分监控网络是由国有的中国电子技术集团公司开发，该公司由四所中共国大学提供支持，并与七所英国大学建立了联系。

**中共国的大学**

作为习近平争取中共全球霸权的一部分，他采用了所谓的“军民融合”战略；其中，大学在在最大化国家军事力量方面发挥核心作用。

中共国宪法还规定，即使是私人企业开发的新技术，也必须依法与人民解放军共享。

一个重要的研究机构是位于湖南的国防科学技术大学，该大学由军方控制，专门研究高超音速、无人机、超级计算机、雷达和导航系统。

它与八所英国大学建立了联系，包括与一个世界著名学府的正式合作。

其他八所英国大学也与北京航空航天大学有联系，后者将其研究预算的60%用于国防活动。

另一个重要的中心是哈尔滨工业大学。它与美国领先的弹道导弹制造商建立了联合研究实验室，并与三所英国大学建立了联系。

**无人机编队**

北京政府正在开发成群的“自杀式”无人驾驶飞机，可以在目标定位时悬停在空中，同时相互交流并协调行动，而无需任何人工干预。

这标志着机器人战争的下一个时代，自主武器会取代被预先编程或远程控制的无人机。

美国和以色列也在研究这种技术，而英国也在上个月与坎布里亚郡皇家空军斯佩达丹姆进行了20架无人机的测试。

这项先进技术使用计算机算法（通常以昆虫和鱼类的生物学研究为模型）来创建可自我导航的无人机编队。

**核武器**

据估计，中共国总共拥有350枚核弹头，其中204枚是从陆基发射器发射的远程导弹。如图所示，超音速导弹于2020年3月19日从夏威夷考艾岛的太平洋导弹靶场设施发射。

五角大楼最近的一份报告警告说，为赶上俄罗斯和美国，北京计划在未来十年内将其核武库数量翻一倍，这是习近平争取全球主导地位的一部分。

其中许多武器是由中国航天科技集团公司开发的。该公司是一家大型国有企业，与至少五所英国大学有联系。
