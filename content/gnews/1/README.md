**西班牙2021年6月11日电/西喜社——**

6月2日，委内瑞拉的Monitoreamos.com发表了一条新闻，标题为：教廷大使在告别委内瑞拉时拒绝接受马杜罗的奖励。

教廷大使—乔尔达诺（Aldo Giordano）先生在被教皇弗朗西斯（Francisco）调任为梵蒂冈驻欧盟代表后，于星期三在布鲁塞尔告别了委内瑞拉。
