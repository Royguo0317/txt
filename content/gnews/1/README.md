# 中共国的数字丝绸之路随着中东5G一起发展

**尽管美国做出了努力，华为和其它中共国公司仍继续在海湾国家中迅速进军。**

[!\[\]()!\[\](https://spark.adobe.com/page/MN39rt0M5jYl6/images/b6215977-9492-4cc9-a571-2365d0c2d5be.jpg?asset_id=f1caf5bc-8c5a-49b9-a713-48f0b77e5406&amp;img_etag=%22d4809c823363c56d06227a1a2c9575f3%22&amp;size=2560)](https://spark.adobe.com/page/MN39rt0M5jYl6/images/b6215977-9492-4cc9-a571-2365d0c2d5be.jpg?asset_id=f1caf5bc-8c5a-49b9-a713-48f0b77e5406&amp;img_etag=%22d4809c823363c56d06227a1a2c9575f3%22&amp;size=1024) 中国的数字丝绸之路

“共同的新价值”是第40届年度迪拜国际电脑及网络信息（GITEX）技术展览周的口号，迪拜国际电脑及网络信息技术展览周是12月6日至10日在迪拜世界贸易中心（Dubai World Trade Center ）举行的2020年唯一的面对面全球技术展览。其组织者是享誉整个中东地区，尤其是在 海湾合作委员会（GCC）国家中，作为该地区5G和云计算的领导者： 华为。

2020年，这家全球电信供应商和手机制造商因其在不断发展的全球5G运动中所扮演的角色而成为头条新闻——最近的新闻是在东南亚地区，该公司签署的一系列协议垄断了印度尼西亚5G网络。去年，包括阿联酋、沙特阿拉伯、巴林、科威特和阿曼在内的海湾合作委员会国家（GCC）的11家电信公司与华为签署了大规模的5G合同。

到2025年，海湾合作委员会国家将容纳全球不断增长的5G用户中的大部分。华为已跃上潮流，以利用他们认为充满希望的中东5G的未来。自2019年9月以来，该公司的领导层一直看好中东地区。该地区每年1640亿美元的信息和通信技术（ICT）产品市场是华为满腔热情的一个很好的理由。去年，海湾地区的公共管理云服务激增，价值近30亿美元。由海湾政府主导的这一努力已向华为确认，海湾合作委员会（GCC）国家支持5G的云计算是一个利润日益丰厚的市场。

但是，最近5G的增长对海湾合作委员会国家中的5400万人的日常生活产生了什么影响？并不多。就本地而言，5G大多是在智能手机下载速度更快、模拟通信速度更快的范围内广泛实施的。从12月1日起，华为的新款P40 PRO 5G手机已经在阿联酋面向消费者销售。但分析师仍对5G在海湾合作委员会国家中的多个行业产生积极影响的潜力持乐观态度，特别是能源使用优化、云计算、超高速宽带和物联网（IoT）创新，包括无人驾驶汽车、运输和工厂设备。

[!\[\]()!\[\](https://spark.adobe.com/page/MN39rt0M5jYl6/images/92f74c84-f3c6-437c-9bd4-d1b262596712.jpg?asset_id=64052045-e9ce-49aa-86bf-02d1fb662f1a&amp;img_etag=%22939ee97fb8a2eef73781a933abe298c5%22&amp;size=1024)](https://spark.adobe.com/page/MN39rt0M5jYl6/images/92f74c84-f3c6-437c-9bd4-d1b262596712.jpg?asset_id=64052045-e9ce-49aa-86bf-02d1fb662f1a&amp;img_etag=%22939ee97fb8a2eef73781a933abe298c5%22&amp;size=1024)

作为回应，尽管西方为遏制其全球扩张做出了努力，但华为仍在不懈地将其数字足迹扩展到整个中东。美国一再对华为在国际市场上的存在表示安全担忧。2018年以来，美国决策者一直在积极尝试阻碍该公司及其中方竞争对手中兴的海外扩张，特别是在欧盟内部。川普政府对华为的压力，特别是最近采取的遏制向中共国公司出售关键半导体技术的措施，引发了全球范围内关于国际上依赖华为工具包的数据隐私及安全相关弊端的讨论。

尽管美国政府阻止华为扩张的努力在美国国内和欧洲部分地区取得了成功，但海湾市场却毫发无损。这也许是由于该地区在地缘政治中的高度敏感地位。鉴于严重依赖中共国的石油市场以及美国重要的军事和外交影响力，海湾经济体没有理由成为世界上两个超级大国的代理战区。

但是冠状病毒使海湾国家的经济越来越依赖中共国，因为中共国的经济从全球衰退中复苏的速度超过了美国。由于中共国解除了冠状病毒限制，该国的决策者已经采取重大步骤来发展他们称为全球电信运营商的“工业互联”以提供行业中远比他们的西方竞争者更便宜的价格。沙特电信公司（STC）报告说，由于远程教育的激增，带宽跃升了1000％以上。他们目前正努力确保必要的网络基础设施，以适应沙特雄心勃勃的2030愿景（Vision2030）战略的需求。同样地，卡塔尔正准备希望在2022年举办第一届5G世界杯。此次活动依赖英国电信巨头沃达丰（Vodafone）与华为之间的关键合作关系，可能会因英国最近对华为产品的禁令而受到打击，这将使该地区在未来几年内进行关于5G的战斗变得更加重要。

华为在上述提到的全球5G增长中占据了2020年有关中共国公司的大部分对话，其中包括中共国的数字丝绸之路（DSR）倡议。数字丝绸之路一词最初在2015年的政府白皮书中发布，是对中共国一带一路倡议（BRI）（中共国主导的基础设施发展的全球战略）的扩展。现在，数字丝绸之路已成为中共国在海外数据和电信活动的一个广泛使用的绰号。根据复旦大学的说法，数字丝绸之路政策本身包括五个关键方面：以基础设施为核心，数字丝绸之路依靠扩大对贸易、金融和“民心”的影响，所有这些有利于形成海外政策。其中包括发展中国家的网络安全、数字治理和数据共享实践。

实际上，一带一路倡议和数字丝绸之路都是中共国政府行为者和企业做出的经济和政策决策的统称。数字丝绸之路品牌模糊不清，使许多中共国公司（可能各有不同程度的中共国政府参与）的政策可能迥然不同。即使在这些公司内部，中共国国有和私营企业的混合结构也使高层决策者的动机使外界难以理顺。

没有哪家公司或政府机构可以完全代表中共国的数字丝绸之路，甚至包括华为。但是美国5G运营仍在中东地区落后于亚洲。日本经济研究中心（Japan Center for Economic Research）预测，鉴于在冠状病毒条件下的表现，中共国经济将在2030年之前超过美国。随着美国5G的推出落后于亚洲，华为及其同行公司目前向中东市场进军可能会为亚洲在该地区未来的主导地位指明道路。

跨国协调工作，例如，去年欧盟与日本之间建立的数字基础设施合作伙伴关系，是美国政府在5G竞争中保持重要性的积极战略。美国参议院外交关系委员会（Foreign Relations Committee）共和党主席吉姆·里施（Jim Risch）最近发布了一份报告，希望在欧盟和美国之间建立跨大西洋伙伴关系，理由是对中共国控制的公司支持数据存储的巨大能力感到担忧。就像华为在中东的“共同的新价值”的口号一样，合作战略可能有助于西方保持相关性，并为中东信息和通信技术市场带来越来越有竞争力的价值。
