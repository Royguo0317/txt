#### 全国各地虚涨注册选民数量

最令人大开眼界的，是整个国家注册的选民人数的统计情况。选民册中充斥各样人等，从死去的人到1992年以来从来没有投过票的人，各种生日，有1800年1月1日的，还有1900年1月1日的，等等，有已经离开本州的人，还有更了名的人，以及重复登记的人。这就是为什么会有如此之高的选民登记比例。

也就是说，他们耍了很多小花招来保持一个虚涨的选民名单。在任何地方查询登记选民的百分比，你都会发现这个比例是基于该州整体人口的百分比，而不是18岁或以上的有资格成为选民的人口的百分比。这种算法，使选民登记的百分比率降低而貌似更为真实。

例如在计算16岁女孩中入学某一特定高中的百分比时，你不会把男孩也算进去。然而，在计算有资格选民中的登记选民的百分比时，他们却包括了那些18岁以下没有选举资格的人口。非常狡诈。

即使这样，各州的选民登记率依然都大幅上涨。事实上，有些州的选民登记率甚至超过其有资格人口的100％，例如密歇根州(Michigan)的选民登记率为103％。**在宾夕法尼亚州，这一比例为****89.30****％，但是当深入审视各县情况时，你会发现这些数字急剧上升，例如在雄鹿县****(Bucks County),选民登记率高达****97.35****％。**
