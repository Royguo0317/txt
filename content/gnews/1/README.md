# **《关于继续实行严重侵犯人权和腐败问题国家紧急状态的通知》**

## **法律与司法**

**发布时间 2020年12月16日**

2017年12月20日，总统通过第13818号行政命令，针对世界各地严重的侵犯人权和腐败现象，宣布全国进入紧急状态，并根据《国际紧急经济权力法》（50 U.S.C. 1701等）采取相关措施，应对美国国家安全、外交政策和经济所面临的不寻常和特殊威胁。

侵犯人权和腐败行为的普遍性和严重性，其全部或大部分来源在美国境外，其继续对美国的国家安全、外交政策和经济构成不同寻常的威胁。 为此，2017年12月20日宣布的国家紧急状态必须在2020年12月20日之后继续有效。 因此，根据《国家紧急状态法》第202(d)条(50 U.S.C. 1622(d))，我将第13818号行政命令中宣布的关于严重侵犯人权和腐败的国家紧急状态持续一年。

本通知将在《联邦公报》上公布，并转交国会。

唐纳德·川普

白宫

2020年12月16日
