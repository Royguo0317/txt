中共大外宣《环球时报》 近日发文鼓吹，中共国年轻人已经“沉迷”智能手机应用程序游戏来了解中国共产党的历史 。

据报道称，一名22岁的中共国大学毕业生周颖 说：“从未想过有一天会沉迷于”手机游戏了解中共的历史，但“却享受了探索的过程”。该游戏的玩法是，“在游戏中寻找隐藏在1910年代上海绘画中的五个共产文物，比如马克思的头像和五四运动的海报” 。每当玩家成功找到一个物品时，游戏就会简要介绍与该物品有关的人或事件。

1919年的“五四运动”始于中国北京，以学生为主导，抗议《凡尔赛条约》将中国山东省割让给日本。来自北京各地的大约3000名学生抗议者聚集在天安门广场，以谴责当时中华民国政府的决定。示威活动最终演变成暴力，痛殴张宗祥，火烧赵家楼。五四运动后，陈独秀和李大钊等在苏维埃的扶持下于1921年7月成立了中共。

中共国将在今年7月1日庆祝建党100周年，为了鼓励年轻一代更多地了解中国共产党的丰功伟绩，中共官员已经举办了很多宣传活动，这款视频游戏只是一个例子。

周颖所玩的游戏叫做“上海的革命火花”，吸引了许多年轻人。这是最近数十种手机游戏之一，集中于介绍中国共产党的历史和发展，并针对大陆的九零后。”

中共历史学习电子游戏，脱口秀和桌牌游戏等其他形式的出现，迎合了中共国年轻人对新事物新科技的热情，但最终这些宣传手段都是吹嘘该党永远伟大，光荣，正确，中共苦心积虑地将洗脑充斥在年轻人生活的方方面面。
