**西班牙2021年5月24日电/西喜社——**

一位台湾官员指责中共国在台湾岛上传播有关Covid-19疫情的虚假消息，称这就是为什么政府要发布信息和驳斥在线散布的虚假信息。

台湾在控制大流行数月后，现在正在应对国内感染的激增，整个岛处于高度戒备状态，人们被要求留在家中，许多场所被关闭。

台湾一再警告说，中国正试图利用“认知战”来破坏对政府的信任及其对大流行的反应。台湾声明是民主自治的政府。

内政部副部长陈宗仁周六在对记者发表讲话时说，他们“明显感觉到”中共国对台湾的宣传和错误信息所具有的危险。

他补充说，“这就是我们为什么不断向所有人解释虚假信息的内容的原因，是要引起人们的注意。 我们必须立即予以制止，不要让认知战争影响台湾社会。”

陈（副部长）列举了一些网上流传的假新闻的例子，包括台湾总统蔡英文被感染并掩盖这个信息。

他说：“我想对所有人说，这确实是卑鄙的假新闻。”

在确认蔡住所的一名工人被感染后，蔡本人在本周检测为阴性。

一位监视中共国在台湾活动的安全官员说，本周台北政府认为，北京正在从事认知战以“制造混乱”，并破坏公众对（政府）如何处理这种流行病的信任。

中共国台办周四表示，台湾的指控是“虚构的”，是（台湾）政府试图将注意力从实际问题上转移开。 它补充说，台湾应该“停止玩政治游戏，并采取切实措施尽快控制病毒流行”。

台湾表示，周六和周日对于打破病毒传播链至关重要，并敦促人们留在家里。

卫生部发布了社交媒体狗吉祥物，一种叫做宗柴的柴犬，还推荐一些歌曲，供独处的人们可以在家唱歌以保持娱乐，例如台湾摇滚歌手伍佰的热门单曲《孤树》。

外交部说：“在周末，除非绝对必要，否则请不要外出。”还在麦克风前展示了戴着眼镜的宗柴狗。
