知乎上[有完全遵守劳动法的公司吗？](https://www.zhihu.com/question/444864540)这个话题让人看后百味杂陈，现将部分内容转贴如下——

［外企刚过来的时候不了解情况，开始高工资不加班，现在没有了。但是有一天，招进来一个中国的HR，说了一句话：“中国的员工不需要那么高的工资！”老外用了中国的HR，重新给员工设置了薪酬，发现给老板生了一大笔钱。于是，在国内，中国的HR把外企的HR都挤掉了。“中国的员工都特别喜欢奋斗，让他们正常下班会不高兴的！”于是老板又发现，招收中国的管理层，可以让员工一天工作十二个小时，于是中国的管理层又把部分老外管理层挤走了。有一个中国的员工不喜欢加班，但是工作完成的都很好，老板正准备给他评优秀涨工资。这个时候有聪明的人开始提醒，不但不能给涨薪，还要增加工作量，直到他跟其他人都需要加班完成为止，好不然其他人都不加班了。于是老板也听了，果然大家晚上9点之前都没有人走了。在一个毫无规矩和底线的世界里，于是每个人像是养殖场上叠的高高的笼子里拥挤的动物，谁也别想呼吸道一口新鲜空气。昨天看到一个公司的招聘信息，有个公司的自我介绍最后一条快笑死了：公司遵守劳动法。。。］

在时下中国日夜拼搏的白领们遭遇996已经不足为奇，而中国白领们也并不是唯一的受害者，那些在生产线上披星戴月被企业管理者们以极低廉价格收割血汗的打工者所遭遇的更甚于白领们。而使这种罪恶遍及中共国企业的也不仅仅是HR，与HR共同作恶的还有这些公司的法务工作者。

同为打工者的HR和法务工作者们为什么一心要参与这种让打工者无偿加班、缩短或者削减节假日的犯罪呢？这与中国人的人种无关，这与时下中共国的大环境有关，因为上有好者，所以这些HR和法务工作者们必会下有所甚焉，其实不管是外企还是民企都是要以赢利为目的，在法律是儿戏的情况下，在工会是摆设的情况下，当犯法的成本几乎为零而因犯法所赚取的利润远远超出常规的时候，资本家和企业家大多是不会在乎赚来的钱里哪些是干净的哪些是压榨出来的血汗的。

一个靠压榨血汗生存的公司是不会有创造力的，一个被榨干血汗的员工是不会有工作热情的，一个把法律变为儿戏把工会变成摆设的国度是注定不会有未来的，在这个群体作恶的社会里，违法者的成本可以归零，而造成这一切无法无天困境的正是那个无法无天的、靠鼓动工运和学运起家的中国共产党，它的存在是这一切罪恶的根源，但是甘于做奴才的HR和法务工作者却并不会随着中共的倒台而绝迹，所以从这一点来说，灭共之后的重建依然任重道远，而这种重建就包括让法律不再成为儿戏、让工会不再成为摆设。
