在过去一周内，在中共国内连续发生恶性杀人事件：

- 大连劳动公园


5月22日12时，大连劳动公园门前突发一起严重交通事故，一车猛烈撞击多人，致多人倒地受伤，至少死亡5人以上，场面惨不忍睹。警方给出解释凶手报复社会。

- 武汉保利砍人事件


5月28日武汉保利发生一起报复社会的砍伤几名无辜路人，但是在31号搜索事件详细经过的时候，已经全部被屏蔽删除，只能在百度的网络记忆上有一点点信息。

- 广东清远英德


2021年5月28日，英德市公安局发布警情通报。22岁女生被残忍杀害。

- 河北任丘砍人：


2021年5月24日，河北沧州市任丘一中学内发生一起砍人事件，一名男子持刀在校园内行凶，砍伤多人，当地警方迅速赶赴，并第一时间控制了犯罪嫌疑人，警方给出解释凶手为精神病患者。

- 南昌红谷滩砍人


5月24日，江西南昌市，3名年轻女子在红谷滩新区凤凰中大道逛街，突然遭到男子行凶，其中一24岁女子遇害，有知情者表示这个凶手原因为，找不到老婆，所以就讨厌女性，自己想轻生，导致事件发生，惨不忍睹。

- 义乌砍人事件


5月28日15时36分许，公安局接到群众报警称，稠城街道化工路一名男子持刀捅伤一名女子及男子。经初步查明，犯罪嫌疑人陈某（男，41岁，辽宁人）因家庭纠纷持刀刺伤李某（女）、陈某某（男）。陈某与李某系夫妻关系。目前，案件正全力侦办中

- 湖南郴州砍人


5月28日中午，湖南郴州白露塘综合大市场附近发生一起令人震惊的持刀伤人事件，一名男子持刀砍伤5名走在上学路上的小学生，根据最新消息显示，其中一名小学生已不幸遇难，场面让人不忍直视，幼小的生命就这么被结束。

- 南京新街口砍人


5月29日晚，南京一男子驾驶租来的白色轿车将前妻撞倒在地，现场热心市民阻拦无果后，吉某某再次碾压该女子后逃离现场。其后用刀刺伤6名围观群众，自杀未遂被警方控制

首先，我们被上面的一件件触目惊心的杀人案件和中共国宣传的目前中国是全球最安全的国家形成了鲜明的对比，百姓的生命安全随时被威胁到，不知道什么时候就会有飞来横祸，结束生命。大家可以对上述的案件进行深入分析，新闻都是来自墙内被中国控制的媒体加工后发布的。但是在这些事件背后是什么让底层的百姓如此的残忍呢？原因无非是在目前的墙内的环境已经让人的生存到了窒息的地步，大批量的经济实体倒闭关门，金融诈骗遍地暴雷，同时压在百姓身上的教育，医疗，养老三座大山无法呼吸。被逼的底层民众前后夹击，对生活和未来失去了信心，但是邪恶的中共为了他们的统治地位，根本不允许躺平抗争，还要在他管控的信用体系中，给你征信打分，就给你各种无底线管控。在这种强压下难免会被压爆，开始类似报复社会的行为，这个是一个多米诺骨牌开始，希望没有被波及的民众不用庆幸，每个人都会在中共的残酷统治中变成受害者。只是时间问题。

其次，为何在中共媒体管控这么苛刻的情况下，如此多的暴力事件会被一次次播报出来，还能在媒体和网络上进行广泛传播，对内无非就是对百姓的进行恐惧威慑，让大家知道现在的社会是不安定的，只有CCP的集权统治才可以管理好这个国家。中共一直想把14亿百姓紧紧的抱在自己身边来当肉盾，延续他的统治时间。因此郭先生的远见战略布局已经早早的打破了邪恶中共的计划。在几年前就开始宣传中共不等于中国人。
