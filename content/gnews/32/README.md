据《内幕报》（Insider Paper）2月18日专题报道，造成这次全球疫情大流行的中共冠状病毒源自武汉的一个实验室，这是德国汉堡大学享有全球盛名的物理学家及纳米科学家罗兰·维森丹格（Roland Wiesendanger）在一份新的研究报告中指出的。

**中共病毒源自武汉实验室**

自中共病毒大流行发端以来，到现在有两种关于病毒起源的观点。其一是由蝙蝠传染人类的说法，另一个就是从武汉（病毒研究所）实验室释放出来的观点。

**中共病毒大流行起源的研究**

报道称，维森丹格教授在报告中展示了他从2019年至2020年搜集到的证据，并非常肯定地指出：“病毒本身所具备的信息，大量专业期刊杂志的公开报道，以及社交媒体的信息都表明，这次冠状病毒大流行是源自武汉实验室的事故。”维森丹格教授对这次病毒溯源的研究做了一个说明，“对科学的好奇心驱使我这么干的，同时也是责任感的原因。作为一名科学家，我必须询问由SARS-CoV-2病毒(即本次大流行的CCP冠状病毒)造成这一切灾难的起源是什么。”

**研究报告阐明了什么观点？**

维森丹格教授表示，“与以前爆发的冠状病毒，如非典（SARS）和中东呼吸综合症（MERS）不同的是，尚未发现导致这次冠状病毒大流行的中间宿主。而这个中间宿主有可能就是把SARS-CoV-2病毒从蝙蝠传染给人类的必要途径。”“因此，用人畜共患传染病理论（Zoonosis Theory）来解释本次病毒大流行，是没有足够令人信服的科学依据的。”

报道指出，维森丹格教授的论证表明，“这间位于武汉的病毒研究所是目前世界上收集蝙蝠病原体最多的机构之一，而这些蝙蝠病原体却发源于中国南方遥远的洞穴。因此这些蝙蝠极不可能以自然的方式迁徙2000多公里到达武汉，然后在紧靠病毒研究所的地方引发了全世界范围的病毒大流行。”
