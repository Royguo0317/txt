## 中共超限战灭港行动（十六）让爱与和平占领中环前夕
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1261582/)

#### 搜集/撰文：岁月如歌 / 封面合成：文粤
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/21034410/5211.jpg)
2013年1月香港大学法律系副教授戴耀廷表示，若香港民众再不进一步行动，2017年普选的目标将泡汤。他在1月16号在《信报》发表《公民抗命的最大杀伤力武器》，鼓励香港市民以及民间领袖要以有限度的、非暴力的公民抗命形式表达港人自决的权利。他认为过去港人各种争取政治权力的真正方式如游行示威、反酷刑、五区公投、占领政府总部和绝食来的压力都不足以令中共 改变 立场 和 让步 。他 提议 发起 占领 中环 行动 并 论述 及 划 定期 于 2013年 前半年 运动 ，时间 预计 在 2013年 7月 到 2014年 初夏 希望 通过 这次 运动 争取 香港 的 特首 普选。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/21034426/5212.jpg)图片：Reuters/达志影像
戴耀廷教授经过对过去的几个运动的观察，特别是2013年元旦大游行，有示威者占据中环马路，最后只有长毛（梁国雄）一人被拘控，受到这个启发，他发现事情的演变已经不再是一项政治行动，已成为一种精神，有着一套信念价值。而这个价值就是对民主制度的坚持，民主不单是目的也是手段。他是一种能确保个人享有平等权力的制度，是能够尊重个人的尊严。在法律的角度看，不是不问法律内容就要求人们盲目的去遵从法律，人们所追求的法治是要法律能“权下”和“达意”。若法律未能实现公益应公民的基本权利未能得到充分及有效保障，人们可以用有限度的、非暴力的违法行为为进行公民抗命去促使不公义的法律改变过来。这是戴教授发起占领中环的意义所在，他认为这种“占中”精神的价值包含了民主、人权、平等、尊严、法治、公平和工艺。他它也不只是一些崇高的价值，涉及到公民参与抗命的行动去争取香港落实真普选是要承担罪责的。也就是说，香港人用这种方式去抗命争取普选权，中共是不会轻易放过的一定会问责定罪，因此参与抗命的人不只是光动嘴巴而是要实际行动，要付出个人代价的。戴教授发起占领中环运动采取的是公民自主的、有责任的主动去参加，希望全民用各种方式参与或支持运动。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/21034437/5213.jpg)图片：削发仪式在九龙慈云山的一家教堂内举行。戴耀庭称，削发是代表他们已经「退无可退」
占中运动正在酝酿的时候，中共于2014年8月31日全国人大常委会正式通过决议为2017年特首普选立下框架，也就是“831落闸”。中共的决定在香港就像扔下一颗炸弹，震惊了香港各界。媒体纷纷发表评论表达对831决定的不满和怀疑。什么是落闸？ 831决定2017年香港特首选举的方法是：一，提名委员会人数1200人按原然有四大界别选出；二，特首参选人必须获得半数提名委员会提名；三，委员会只能选出二到三个候选人。并且特首只限于中共所要求的“爱国爱港人士”。这就是“831三落闸”。表面上看是中共允许2017年特首普选，但是仔细看他的条文和候选人都是按照中共人大的决定，选民可以投票的是经过提委会选择出来的候选人。提委会都是亲共派（也就是前面文章说的中共统战胜利品），他们帮中共严格把关特首人选，泛民主派想参选特首，在筛选环节可能连提名的机会都没有。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/21034449/5214.jpg)图片：2014年7月1日游行 Reuters/Tyrone Siu
香港实行普选，是中共在《中华人民共和国香港特别行政区基本法》里写下的法律承诺。 《基本法》阐明，1997年回归后，香港将按循序渐进的方式实行普选改革。当然中共在立这条法律的时候就埋下伏笔，具体时间具体办法需要循序渐进到什么程度才可以实行双普选设下陷阱。才让中共有借口有空子去拖延实行双普选。香港人要求双普选的决心迫使中共不得不承诺在2017年给香港人普选行政长官，只是行政长官，立法会选举还要押后。但是在接下来的政制改革公众意见咨询不久，中共对自己的承诺不但食言还拐弯抹角找各种理由阻止。

针对中共“831决定”戴耀廷教授形容，香港人与中共的对话已经走到了尽头，各种对抗一触即发，占中运动将会联合香港各界展开抗争。也有评论认为中共全国人大常委会以170票全票通过“831决定”，连一张反对票弃权票都没有，其目的是想告诉香港人，中共政府决意坚定，香港人休想做出任何修改。这也是香港统一战线的表现。
