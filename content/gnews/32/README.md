**二、法国总统马克龙新冠检测呈阳性**

- 翻译、评论：gogogo4
- 编辑、校对：文园
- 信息来源：Die Nachrichten


原文标题：[Präsident Macron positiv auf das Coronavirus getestet](https://www.deutschlandfunk.de/frankreich-praesident-macron-positiv-auf-das-coronavirus.1939.de.html?drn:news_id=1206472)

欧盟峰会一周后，法国总统马克龙的新冠检测呈阳性。爱丽舍宫表示，他有新冠症状，但仍然可以工作。西班牙和葡萄牙政府首脑——桑切斯和科斯塔，以及欧盟理事会主席米歇尔作为密切接触者被隔离。

据马克龙办公室透露，他将自我隔离一周，并进行远程办公。他在法国首都表示，包括前往黎巴嫩的所有计划的行程都已取消。

上周末，马克龙曾在布鲁塞尔会见了欧盟国家领导人和政府首脑。两天前，他再次接待了葡萄牙总理科斯塔，并在巴黎主持了一次内阁会议。因此，法国总理卡斯特克斯（Carex）必须被隔离。而同样需要隔离的西班牙总理桑切斯，则是因为在周一与马克龙在欧洲安全与合作组织在巴黎的一次活动中有过接触。

而政府发言人在柏林表示，默克尔总理已经进行了新冠测试，结果为阴性。

在今年夏天，因感染新冠而病重的英国首相约翰逊，则通过推特祝福马克龙早日康复。

**评：各国政要负责为本国国民制定抗“疫”规则，但如果你连自己都保护不好，国民是否也有理由质疑你定的规则呢？政要的安保措施级别应该也很高，即便这样仍然不能幸免。恐怕也是没听过中国那句老话：千防万防，家贼难防。**

德国生物科技公司（Biontech）和辉瑞公司的新冠疫苗继续在更多国家获批。

厄瓜多尔卫生部长Zevallos宣布，将于（明年）1月份开始（疫苗接种）活动。年初的5万剂疫苗已经准备好。同样，智利也准备使用疫苗。根据总统Piñera的说法，该国将在本月接受两万剂疫苗。
