1月16日，英国外交部网站宣布，英国、法国和德国政府谴责伊朗当局为开发可用于核反应堆和核武器的金属铀做准备，法国、德国和英国政府对伊朗准备生产金属铀的声明深表关切。

根据三国的声明，这些计划与《联合全面行动计划》相矛盾，根据该计划，伊朗承诺在15年内不参与铀冶金的开发和研究。声明称，金属铀的生产可能会产生严重的军事后果。

早些时候，美联社援引国际原子能机构（IAEA）的话说，伊朗领导人已经开始安装生产金属铀的设备。伊朗方面曾表示，这是“与开发新型发动机相关的研究”。

1月初，伊朗开始生产高达20%的浓缩铀，这大大超过了2015年核协议商定的4%的极限，伊朗建设在福尔多的一个核设施已启动。美国称增产至20%是“核勒索 ”，并表示希望国际原子能机构继续监控伊朗核计划。

近日，美国要求彻查武汉病毒研究所和中共病毒来源，美国国务院发布1000多家被中共军方控制的中共国企业名单，所有美国资本和个人都不允许投资这些中共国企业，这些都意味着认定中共病毒为中共军方制造的生物武器在稳步推进，一旦实锤敲定，那么不仅美国，北约联盟也必须参战消灭中共，同时美国与中共的金融脱钩已经全面开始。

中共每逢危机的时候，都会推动伊朗这个中共一直扶持的独裁国家利用制造核武器对世界挑起事端，意图转移美国以及北约盟国对其正在进行的打击，但是无论如何，都改变不了全球灭共的步伐。
