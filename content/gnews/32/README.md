# **美国对中共国的依赖是一场蓄势待发的灾难**

**对美国采矿业的忽视及矿产和金属的外包使美国生产基础备受压力，造成毁灭性后果**

[!\[\]()!\[\](https://spark.adobe.com/page/4bIZ0525SSgs1/images/a68483c4-d51f-4145-82ee-dd28b1bcfb47.jpg?asset_id=4bf8c58f-5984-47fa-b881-54b24bdc97d5&amp;img_etag=%22a6d09d51243d0c99a405fddd55bc1989%22&amp;size=2560)](https://spark.adobe.com/page/4bIZ0525SSgs1/images/a68483c4-d51f-4145-82ee-dd28b1bcfb47.jpg?asset_id=4bf8c58f-5984-47fa-b881-54b24bdc97d5&amp;img_etag=%22a6d09d51243d0c99a405fddd55bc1989%22&amp;size=1024)中共国和美国 – 鱼吞汽车插图，格雷格•格罗伊施（Greg Groesch）/《华盛顿时报》

如果不尽快采取务实行动以制止美国对进口矿产和金属日益增长的依赖，尤其是来自中共国的进口，我们的经济将受到严重损害。

拜登政府和国会必须应对这个非常现实的危险，而不仅仅局限于拥有全球供应链的行业。

矿产是现代经济所不可或缺的。安全的供应对于生产一切产品都至关重要，从智能手机和电动汽车到支撑我们国家安全的先进武器系统。然而，尽管我国地下资源丰富，但在美国开辟新矿或扩大现有矿场却愈发困难。无论采矿项目多么有价值，以及它将创造多少就业机会，当激进的持不同政见者聚集时，即使最有希望和必要的项目也可能脱轨。

这种心态的根源可追溯至20世纪90年代，当时美国从矿产和金属的小规模进口国变成主要进口国。对美国采矿业及矿产和金属外包的忽视，是我国生产基础空心化的表现，并且是毁灭性的后果。美国对矿产进口的依赖达到历史最高水平，在过去20年里翻了一番。

由于清洁能源技术和基础设施是矿产密集型产业，因此扭转这种对进口的依赖和重振国内生产的需要成为当务之急。修建数千英里新高压电线以提供更多可再生能源，加快电动汽车（EVs）和充电站的生产，以及制造诸如先进电池的高科技产品，所有这些都依赖于矿产和金属的基础，而这些矿产和金属的范围涵盖整个周期表。

[!\[\]()!\[\](https://spark.adobe.com/page/4bIZ0525SSgs1/images/4567b120-fe8d-4e97-99b9-67a9ce8bb0b7.jpg?asset_id=d17340cd-bd96-42a0-a10d-35e2cd092ccf&amp;img_etag=%2286f0b427ed3a0f2bb481d580202d31a6%22&amp;size=2560)](https://spark.adobe.com/page/4bIZ0525SSgs1/images/4567b120-fe8d-4e97-99b9-67a9ce8bb0b7.jpg?asset_id=d17340cd-bd96-42a0-a10d-35e2cd092ccf&amp;img_etag=%2286f0b427ed3a0f2bb481d580202d31a6%22&amp;size=1024)

当前最大问题莫过于电动汽车的生产部分。预计到2030年，全球电动汽车市场将从2019年的320万辆汽车——轿车、SUV、皮卡和卡车——达到2690万辆。生产电动汽车电池尤其是创造就业和经济增长的引擎。

目前，一辆电动汽车的电池组占汽车总成本的四分之一，成为销售价格中最重要的因素。但在一个关键方面，该系统已经变得越来越脆弱：在至关重要的电池金属方面越来越依赖中共国——镍、锂、铜、石墨和钴。

[!\[\]()!\[\](https://spark.adobe.com/page/4bIZ0525SSgs1/images/552459c0-5727-4d0c-bf35-512cb00a28e9.png?asset_id=778bb1f4-5eaa-480a-b135-bcbb7e528c27&amp;img_etag=%229438750033c2c2f0b1701cb0f90e0f7e%22&amp;size=2560)](https://spark.adobe.com/page/4bIZ0525SSgs1/images/552459c0-5727-4d0c-bf35-512cb00a28e9.png?asset_id=778bb1f4-5eaa-480a-b135-bcbb7e528c27&amp;img_etag=%229438750033c2c2f0b1701cb0f90e0f7e%22&amp;size=1024)

此外，还有稀土矿产。但是，虽然中共国未出现电池金属和稀土的短缺，但也不会出现造成严重问题物资的短缺。比如，像导致供应链中断和造成价格飙升的对美国禁运，将危害美国的汽车工业。

中共国在矿产开发方面的主导地位对美国来说是严重的战略问题。首先，作为目前世界上最大的电池生产国，中共国为本国电池制造商提供电池金属的优惠准入措施，并正在利用其对金属供应链的控制来重新定位全球汽车工业。

这种情况不仅威胁到美国汽车制造业的未来，还造成类似于美国在20世纪70年代严重依赖中东石油所经历的安全问题的可能性。

现在就需要解决该漏洞。以锂为例，锂离子电池的关键成分。尽管有预测显示需求量大增，但美国只有一个正在运营的锂矿，并且国内也没有加工厂。而另一方面，中共国已经垄断了市场。用OPEC的奇思妙想换取中共国的阴谋诡计是地缘政治的升级吗？这是一个令人深感不安的局面。

正是因为危害如此严重且不可预测，我们需要使我们的对手更难以将矿产作为武器使用，并在矿产武器被使用时采取措施保护自己。这意味着建立清洁能源经济所需的国内矿产供应链。回流美国的工业基础必须从回流矿产开发和加工开始。对美国采矿业令人震惊的忽视必须停止。现在是时候重新投资和重建这个国家所需要的可持续、安全和绝对必要的采矿业了。
