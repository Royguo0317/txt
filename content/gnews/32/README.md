**西班牙2021年3月27日电/西喜社——**
![]()![](https://gnews.org/wp-content/uploads/2021/03/ff7c4404600b96e0.jpg)
什么是契约精神，西方契约精神最早追溯到古希腊，一种自由，平等，守信的精神。西方人注重契约精神，客观因素上由于地缘经济的影响，小规模的城邦隔着海洋，加上贫瘠的土地，海上贸易成了必要的经济来源，只有遵守契约精神才能够生存。而中国大部分都接壤陆地，地理环境相对封闭，人与人之间对于契约的需求弱很多，更多地是讲究人情和道德。

“何言乎王正月？大一统也 。”自古以来大一统的思想一直深入人心，经过秦汉之后可以说是确立和巩固了该思想，大一统思想只要存在，崇拜强权人物，极权统治就不可避免，极权统治带来的必然就是一个人治的社会，而当一个社会实行人治，契约精神就很难存在。

到了如今在CCP统治下的中国，更是如此。对外来看，20年前有CCP耍手段加入WTO，签订协议后便不予理睬，享受着优惠却不执行条款。现有CCP外交部表示《中英联合声明》已经是不具任何现实意义的“历史文件”，令多少人大跌眼镜。

对内来看，从当初“只生一个好，政府来养老”，到如今“赡养老人是义务，推给政府很可耻”。再从生活中来看，强拆就是个最好的例子，数十年前签订的合法契约到今天变成了废纸一张。

守信作为契约精神的基本要素之一，CCP显然不具备。引用七叔3月25日盖特中的一段话，任何看CCP历史上，没有一个政策是连续性的，没有一个CCP对老百姓的承诺是兑现的，全都是骗，假，黑。

当一个政府没有契约精神的时候，指望百姓拥有，那是不现实的。处在被极权统治下的百姓，面对这样一个人治型的非契约社会，自然就开始了弱肉强食的丛林法则，利用和背叛是家常便饭，遵守契约和规则意味着你会失去自身利益。这些年来CCP的所作所为，利用了人性的黑暗面，暗示人们去上行下效。
![]()![](https://gnews.org/wp-content/uploads/2021/03/dd73f6f342393501.jpg)
政府强拆了民用房子，就算你反抗，你能有任何补偿吗？公司强制不遵守已签订的劳工合同时制，以各种理由要求员工加班，公然违反劳动法，你敢提意见么，你会辞职么。看到没，只要对我方有利，不守契约，没有关系。 趋利避害是人的天性，当在一个社会中违背契约所带来的收益远大于违约需要付出的损失和代价，那违约会是一个很不错的选择。而这一切正是由于政府的默认甚至是鼓励。

契约精神包含自由，平等，守信。这些内容都是建立在法治的基础之上，我一直认为人都是善恶共同体，优良的体制激发人的良知，恶劣的制度催生人的邪恶。

而CCP正是属于第二种情况，CCP的邪恶体制引导百姓们成为畸形的心理，只有铲除CCP，砍掉压在人民身上的这只大手，建立以法治为基础的社会制度，以中华民族勤劳，和平，智慧的特性，学习法治，建立契约精神不是什么难事。方向和道路走对了，到达目的地只是早晚的问题，而现在有新中国联邦的指引，我相信时间也不用太久。
