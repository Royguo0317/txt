## 【独立观点】 北约秘书长抨击中共不认同欧美价值观

![]()![](https://gnews.org/wp-content/uploads/2021/03/WhatsApp_Image_2020-10-24_at_04.33.05-9.jpeg)



路透华盛顿6月7日，北大西洋公约组织（NATO）秘书长斯托滕伯格在结束与美国总统拜登的会晤后，在白宫外接受记者采访：他谴责中国对台湾、维吾尔人和香港抗议者的“强制”政策。还说，很显然中共政府的领导人“不认同我们的价值观”。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/10043652/%E5%9B%BE%E7%89%8727-1.png)图片来源： DW

斯托滕伯格还说：北约同意拜登总统对俄罗斯的态度，包括他与俄罗斯总统普京在日内瓦将进行的对话。

斯托滕伯格怎么知道中共政府的领导人不认同欧美国家的价值观呢？很显然，是拜登总统告诉他的。回想5月28日，拜登在位于汉普顿市的兰利—尤斯蒂斯联合基地（Joint Base Langley-Eustis）对部队说：“我们正在打一场民主与独裁的战斗。”在这次演讲中，拜登还明确表示中共政府的领导人是独裁者。

独裁者怎么可能认同自由民主世界的价值观呢？中共在新疆对维吾尔人的种族灭绝罪行，对香港民主运动的血腥镇压，对台湾的武力威胁，最重要的是对全世界的病毒生物战，无一不是对西方民主自由价值观的践踏。

北约组织是1949年在美国华盛顿签署协定成立成立，是欧洲和北美国家联合防御，当时的主要战略目标是防范（冷战时期）华沙条约组织的大规模军事入侵，总部在布鲁塞尔。冷战结束后，华沙条约组织解散，现在北约的目标已经转换成联合维和、维持区域稳定、解决各种世界性问题，还有仍旧低调地防范俄罗斯。

所以，斯托滕伯格有资格说，北约同意拜登总统对俄罗斯的态度，和谈话内容。这表示北约在俄罗斯的问题上，和美国的立场达成一致。

显然，这是北约组织和拜登政府一样，认为目前最重要的敌人已经由俄罗斯转变为中共。而且，北约组织已经达成共识，将会联合防范中共，在必要的时候，比如说台海发生战争时，北约组织会联合维和。

由此可见，卡在中共脖子上的绞索，越来越紧了。
