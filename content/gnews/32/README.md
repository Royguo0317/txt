遭到华盛顿邮报报导点名台湾拒用敌国产的疫苗。台湾卫福部长、中央流行疫情指挥中心指挥官陈时中2月4日表示，此报导有些形容词可讨论，但整体内容是实在的。另外同样拒绝使用中共疫苗的国家还有乌克兰、伊朗国家。

台湾人民不选择使用中共疫苗，除了台湾就目前新冠状病毒疫情的控制尚属平稳，无迫切使用疫苗的需要性，政府也对中共疫苗是否有效持观望态度，因为中共长期对台湾的文攻武吓，造成台湾人对中共的疫苗是否造成次生灾难存在疑问。
![]()![](https://gnews.org/wp-content/uploads/2021/02/TWchan-338x225.jpg)
而乌克兰为何不选择使用中共疫苗，1991年苏联解体后乌克兰重新独立，作为独立国协发起与创始国之一。同年8月24日，乌克兰政府发表的国家独立宣言，正式宣布脱离苏联独立，改国名为乌克兰共和国，标志着乌克兰结束了337年受俄罗斯的统治历史。被共产党统治三百多年的乌克兰深切的认知了共产党的谎言是极度不可信的，他们宁可选择其他国家所研发的疫苗，也不会选择中共疫苗。

最不可思议的是与中共称兄道弟的伊朗居然也拒绝使用中共疫苗，伊朗经济以石油开采业为主，美国自伊斯兰革命后就从未进口伊朗石油，而欧盟也已对伊朗实施制裁，禁止成员国与伊朗续签石油进口协议。2005年末中国与伊朗签署了一项价值1000亿美元的石油合作协议，成为该国重要的石油贸易伙伴。

2019新型冠状病毒令伊朗确诊人数已超过一百万，其中54000多人因此而丧生。（截止至2020年12月）以伊朗的感染人数来看，应该有施打疫苗的迫切性，按照中共与伊朗长期以来的密切合作交流来看，不选择使用中共疫苗实在令人怀疑，是否伊朗对中共疫苗效用存有疑虑而选择不使用，还是有不为人知的内情。本文仅代表个人立场
