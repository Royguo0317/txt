《门道专家》发表文章，报道了威斯康星州议会的决定，题目“刚刚好：共和党控制的威斯康星州议会授权调查2020年总统大选”。【3】

文章说，共和党控制的威斯康星州议会周二通过决议，授权对2020年总统选举进行调查。该决议以58票赞成、35票反对的党派路线获得通过。委员会现在起，将有强制作证和收集文件的传唤权利。

美联社报道：共和党控制的威斯康星州议会星期二通过一项决议，授权对拜登在该州险胜的2020年总统选举进行调查。

众议员乔·桑菲利波（Joe Sanfelippo）说，这项被民主党人反对的决议，如果委员会决定发出传票，（进行）强制作证和收集文件，需要给予委员会授权。他（桑菲利波）是负责调查的议会选举和竞选委员会副主席。

授权调查的决议以58票对35票的党派路线投票通过，所有共和党人支持，所有民主党人反对。上个月共和党人下令对选举结果进行审计后，该法案获得通过。

共和党人说，他们希望收集更多证据，看看是否违反了法律，但民主党人说，他们试图在政治上得分，破坏公众对选举的信心，侮辱投票办事员、投票站工作者和其他选举管理人员。

川普总统在威斯康星州选举之夜取得了相当大的领先优势，当时民主党的投票管理人员便停止了计票。到了第二天早上，拜登以某种方式领先了。
