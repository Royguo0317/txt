据路透社报道，12月13日某政府背后支持的网络黑客攻击了美国财政部以及美国商务部下设的国家信息与通讯管理这个机构。该机构负责制定互联网及通讯政策。此次骇客攻击非常严重，直接被上报到了周六举行的国家安全委员会会议。

据报道称，骇客使用了位于美国德克萨斯州奥斯丁一家名为太阳风（SolarWinds）的软件公司的软件产品，兼容了微软的云办公平台，对于美国政府以上部门内部的电子邮件实施了数月的监控。这个某国到底是哪一国？在过去的一周时间，各方势力都在进行激烈的角力。

由中共背后支持、利用大规模选举舞弊获得虚假胜选结果的乔·拜登，在12月14日（周一），由选举人团认定在美国2020年总统大选中胜选。12月15日，俄罗斯普京总统给美国乔·拜登发去贺电。北大西洋公约组织秘书长早在11月23日就致电拜登表示了祝贺，而普京的这番表态是三周以后的事情。正如普京12月17日在俄罗斯年度记者会上表示的，俄罗斯的这番对乔·拜登的示好，无非是希望拜登能解决美俄关系正常化问题。

值得注意的是，普京在17日的年度记者会上也同时表示，俄中两国在许多领域有着共同利益，两国领导人保持着高度互信和务实的良好关系。从中共外交部汪文斌12月18日的答记者问的互动和反馈中可以确定，中共国跟俄罗斯之间确实有很深的合作。这些不仅仅是俄罗斯向中共国出口大量农产品、中共国是俄罗斯第一大贸易伙伴、两国跨境电商合作密切，更重要的是中共同俄罗斯有国家级的合作。那么，会不会是在互联网骇客攻击美国方面有相关的合作呢？

美国国务卿麦克·蓬佩奥在12月18日（周五）接受马克·莱文秀的采访时表示，这周发生的网络攻击是美国历史上最为严重的一次，非常清楚，这是俄罗斯干的。而川普总统在12月19日（周六）表示，中共国才是大规模网络攻击美国政府系统的罪魁祸首。对于这次网络袭击，俄罗斯总统普京否认了俄罗斯政府参与了此事。正如17日他在年度记者会上所声明的，俄罗斯也并没有干预美国总统大选。
