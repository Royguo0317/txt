**西班牙2021年5月31日电/西喜社——**

安东尼·福奇博士在《澳大利亚人报》2012年曝光的一篇论文中，似乎捍卫了功能增强的”重要性”研究，尽管它有大流行病的 “风险”。《国家脉动》又揭示了国家过敏和传染病研究所所长在2012年12月17日的会议上，有一次专注于有争议的生物武器研究形式的类似评论。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/31110004/unknown-47.png)

由美国国立卫生研究院（NIH）主办的12月活动，[“ HPAI H5N1病毒的功能获得研究”](https://translate.google.com/website?sl=auto&amp;tl=zh-CN&amp;ajax=1&amp;prev=search&amp;u=https://www.youtube.com/watch?v%3DBACqqgRpktA%26t%3D2085s%26ab_channel%3DNationalInstitutesofHealth%2528NIH%2529), 福奇在介绍会上成为演讲嘉宾。

美国国家过敏和传染病研究所所长（NIAID）似乎捍卫了病毒功能获得性研究，该研究将传染病的毒力和致死性提高定义成“不可缺少”。他还将自己的研究机构描述为，“从事研究工作的大多数人（但不是所有人）的主要资助者”。

福奇承认功能增强研究的潜在弊端，并指出研发生物武器存在“担忧”。即这些实验所产生的“产品或信息”，可能会被他人以不小心、不规范的方式、被没有经验的人、甚至被故意滥用的方式使用，从而危害社会”。

这些评论是随着越来越多的对NIAID主任福奇的审查而出土，还有福奇通过他的人，把美国纳税人的钱馈赠给武汉病毒学研究所之间的关系。

2012年10月，福奇在为美国微生物学会撰写的文章中还提出，这种有风险的研究形式可能会引发 “一场大流行病”：

“在一个不太可能但可以想象的事件中，如果那个科学家感染了病毒，导致病毒爆发并最终引发大流行病，那该怎么办？”

“在冠状病毒领域工作的科学家可能会说—事实上我已经说过了–生物武器实验和由此产生的知识的好处超过了风险，”他补充说。在《澳大利亚人》的莎莉·马克森（Sharri Markson）最先报道的论文中指称，大流行病 “更有可能 “自然进化发生。

福奇补充说：“我们不能指望那些有这些顾虑的人，简单地相信我们这些科学界人士的话：认为生物武器实验的好处大于风险；我们也不能忽视他们要求提高透明度的呼吁，他们对利益冲突的关切以及不满，以及他们努力要对话是否应该进行这样的实验？”

他在论文中重申：“我们这些相信这项工作好处的科学界人士，有责任认真和尊重地解决上述这些问题。”
