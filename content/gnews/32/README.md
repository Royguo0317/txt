**1.经济**

在解释喜联储和HCoin之前，必须先要了解一些相关的经济学基本原理。

首先，人在生活中有各种需求，需求分为两类：商品需求和服务需求。例如：食物、工具都是商品。服务员、理发师都是服务。

什么是“经济”？“经济”就是生活需求的交换。例如：农民种植苹果，苹果卖掉后换取理发服务。这样的交换过程就是经济。

有需求交换就有经济，没有需求交换就没有经济。交换的速度越快，交换的规模越大，经济就越好；反之，就是经济变差。

**2.货币，金融产品**

**货币**：对不同商品的价值进行量化，便于需求交换，这样就产生了货币。货币就是需求交换的媒介。

货币通常都会锚定一个稳定存在的属性。例如，黄金作为货币，作为硬通货，是因为黄金锚定了可以长期稳定存在的金属物理属性。

法币锚定的是国家信誉、国家税收体系。

在监狱里，会用香烟和罐头作为货币；在南太平洋的一个岛上，用石头作为货币；夏威夷的黑帮用SPAM午餐肉作为货币。稳定存在，便于流通，都可以作为货币。

不利于流通的东西不适宜用作货币，或者说，越有利于流通的东西越有利于作为货币。

无论是纸币替代硬币、电子货币替代纸币，还是比特币的出现，每一次的货币升级换代都证明了这一点。

七哥说比特币是没有锚定的，**我的观点是**：比特币锚定的是全球电脑网络。由于加密技术将比特币信息分散在全球各地的电脑上，导致其货币信息可以稳定存在，不会轻易消失。除非全球电脑同时断电且不能再上线。

**金融产品：**当生产物资过多时，就可以换到过多的货币，这些过多的货币就衍生出新的商品：金融产品。

金融产品通常不会直接用来兑换商品，主要目的是理财。例如，黄金、房产、证券。

**货币和金融产品的异同：**

（1）货币便于流通：便于交换。金融产品没有此要求。

（2）货币稳定性：固定数量的货币购买固定数量的商品（物价稳定）。快速的升值、贬值都不利于货币的使用，会造成社会混乱（引发频繁且大幅度的通胀和通缩变化）。金融产品没有此要求。

（3）可保存性：不容易损坏，不可被剥夺。例如，保险箱保存货币；战乱时购买黄金。

（4）合法性：来源可追踪，不可被民间轻易复制。例如，纸币有独特编码，有防伪措施。铸币都是有刻字的。

（5）货币可以转换为金融产品。例如，黄金、法币（外汇期货）、比特币。

当一种货币转变为金融产品的时候，一定是有另一种货币取代了它的货币地位。例如，美元作为金融产品的时候，你手中使用的货币是人民币。比特币或黄金作为金融产品的时候，你手中使用的是各国法币。

（6）货币不能同时具备货币和金融产品双属性。货币就是用来交换商品的，不可能大量储存法币等它升值。

生活中的常识可以得知：商品卖的越来越便宜，所以你就不花钱了？你还是会继续买东西。

马斯克推广狗狗币的目的，是把狗狗币从金融产品变成货币。但是金融产品不可能同时具备货币属性，所以这种方法永远都不会成功。

**喜联储的做法**：HCoin兑换法币（喜美元、喜日元），再通过法币购买商品。它以加密货币作为转账的中转工具，再以法币作为支付系统。这样的双币系统同时具备了金融产品和货币双特性。

**3.交易模式**

在经济交易的过程中，存在两种交易模式：

（1）普通交易模式：并行多笔一口价成交。

单向交易。买卖双方的角色不能互换。商品总数有上限，卖完即止。

不需要竞价，不同成交之间没有互相关联性，成交价格不会互相影响，价格比较稳定。

例如：菜市场，商场。

（2）交易所模式：竞价单笔成交。

双向交易。买方购买后立即转变为卖方。金融产品可循环销售，无止境，销售总量没有上限。

每笔都要竞价，每一次成交都会影响下一次成交的价格，价格波动较大，没有上限。

例如：股市，各种金融市场。

**4.比特币**

比特币打造初期是货币，是可以直接交换消费品的，当引入交易所之后，变成了金融产品。

[!\[\]()!\[\](https://spark.adobe.com/page/BSYml7PsTnMFy/images/d078008d-b799-4817-8d11-b7e97446310f.png?asset_id=07b3a439-8f12-46d5-a79b-56ebab84387d&amp;img_etag=%22368915392b7ed6de5378fc16a0d48c1f%22&amp;size=1024)](https://spark.adobe.com/page/BSYml7PsTnMFy/images/d078008d-b799-4817-8d11-b7e97446310f.png?asset_id=07b3a439-8f12-46d5-a79b-56ebab84387d&amp;img_etag=%22368915392b7ed6de5378fc16a0d48c1f%22&amp;size=1024)

**比特币作为货币的优点：**资金流向无法追踪。

**比特币作为货币时的缺点：**交易难度过大（交易过程过度依赖双方熟识度）。资产虽然可以随意转移，但是受限于社交关系。你不认识持有比特币的熟人，你就换不到比特币。导致其只适合大规模的暗网交易，普通百姓难以使用，无法实现全商品跨境流通。

比特币只在黑市交易时具备货币属性，在正规市场只能具备金融产品的属性，这些货币的交易是可以追踪的。

注意，只有比特币有货币属性，其它任何比特币衍生出的加密货币都只有金融产品属性。

和普通货币一样，交易所里的加密货币同样受政府控制、监管。比如，和外汇一样，国内的普通老百姓一样难以买到比特币。

**HCoin和其它货币之间的不同**：

（1）官方货币（法币）：完全中心化的，相当于集权体制。

（2）比特币：完全去中心化的，相当于无政府状态。

（3）HCoin+喜联储：部分中心化的，相当于民主化小政府。大部分时间不管，只在关键时刻出来管一下。

**4.货币增发与通货膨胀**

**通货膨胀受三个因素影响**：人口总量，货币总量，商品总量。

（1）即便手中有很多钱，如果商品不缺货，价格也不会上涨。

全球人口持续增加，商品的需求也持续增加。

科技一直在发展，也就是生产力一直在提升，商品总数一直在大幅度增加。通常不会出现断货的现象，即便货币超发，普通商品的价格也不会上涨，也就是不会出现通货膨胀。

（2）货币增加只会导致金融产品产生通胀。

财富聚集现象：会赚钱的人会一直赚钱，那么多印发的货币必然会被富人持续的赚走，普通百姓手里的资金量几乎是不变的。富人不会把钱都花在日常消费上，而是会购买大量的金融产品来保值、增值。

这就是房价、股市、黄金、加密货币全都一起大幅上涨的原因。

由货币超发导致的消费品价格上涨，速度较慢。而且一定是超发了非常非常多的货币，才会如此。

（3）只有缺货，供应不足的时候，引发竞价购买，此时才会引发消费品的快速通胀。有钱人买得起，穷人买不起。

现在的情况是，由于疫情原因，导致全球生产力整体下降，商品供应不足，必然会引发消费品的通货膨胀。

此时就需要更多的流动资金，那么金融产品上涨之后就会被变卖，以增加资金流动性。金融产品的价格就会开始下跌。也就是经济危机。

**比特币是如何冲击美元体系的？**

所谓的美元霸权，依赖的是SWIFT系统，比特币完全规避了SWIFT系统，使得美元体系无法惩罚违法的人，对抗美元体系的人就可以肆意妄为而无所顾忌。

可以推论得出，美元体系完全无法和比特币体系对接，因为比特币体系要么就是带交易所的模式，笔笔都可以追踪，那就和现有的银行体系没有任何区别。要么就是没有交易所体系，完全无法追踪。

喜联储引入违法转账可追踪的系统，以弥补比特币的优点所带来的负面问题，完全可以和美元体系进行对接。

当加密货币被定义为洗钱工具的时候，加密货币的交易就变成非法了，此时，交易所就会被强制关闭。历史上有加密货币交易所被查封的事件。

比特币不会消失，但是会沉沦。受到打击后被打回原形，彻底转入地下交易，失去金融产品的属性，重新变为单纯的洗黑钱的工具。HCoin的市场前景就会把比特币的投资资金吸引过来。就像电视的出现严重打击了收音机，互联网冲击了电视。

喜联储不会对各国的法币形成冲击，因为法币是交易过程中的一环。

喜联储冲击的是货币存管系统和交易系统，例如，银行、贝宝等电子交易系统。

喜联储会成为全球最大的银行和交易系统，一定程度上取代SWIFT。

**6.HCoin的升值潜力和操作**

七哥在初次发行的时候限制购买，可以保留大量场外资金。发行后，这些场外资金流入，就会让HCoin升值。

初期能涨多少，完全看流通量和进场资金的比例。如果大家都持有HCoin不卖，那么就会导致流通的HCoin减少，抛售减少有利于HCoin的上涨。HCoin上涨之后，就会吸引更多的金融投资，进一步加剧HCoin的价格上涨。

当资金池的资金量过大，每天的交易额变得很庞大的时候，抛售G币就变成了水池中的一滴水。此时抛售就变得毫无波澜了。战友们耐心等待这一时刻的来临。
