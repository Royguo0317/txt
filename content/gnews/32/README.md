## 法国《费列罗》12/21刊文：中欧投资保护协议即将达成
` 喜马拉雅巴黎七星农场` [轉載自GNews](https://gnews.org/zh-hans/682495/)

作者：LEONOR HUBAUT  翻译：Lorraine  校对：Ikonic

经过七年的谈判，欧盟与中国政府之间将在未来几天内宣布一项投资保护协议。该协议将扩大欧洲公司在中国的业务范围。 据泄露的信息显示，北京应该会做出真正的让步，同意向生物技术，汽车行业，以及医疗和其他服务等领域的欧洲公司敞开大门。作为回报，欧洲将允许中国投资者进入可再生能源领域。

协议的第二回合，即投资保护，中国的让步似乎比较少。北京不愿在中国像在欧洲那样，结束对公营企业的巨额补贴政策。对知识产权的尊重，以及按欧洲要求停止对技术的强行转让，仍然存在很多疑问。协议草案发至27个首都。 担任欧盟轮值主席的柏林，极力推动该条约在年底前缔结。 法国也表示非常支持。 但是对可再生能源很重视的丹麦和荷兰，似乎并不热情。其他国家也加入其中，特别是因为北京拒绝批准国际劳工组织的某些公约，包括关于禁止苦力的公约，维吾尔族的困境和香港的局势。 对此，审批该协定的欧洲议会很难说服众国。然而，有一点需要特别说明。因为不确定因素太多，中国会做的比以往任何时候都要更加灵活，他们会极力在拜登入驻白宫之前就让双方达成协议。
