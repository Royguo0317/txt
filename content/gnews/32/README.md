## 荷兰议会成为一周内第二个指控中共在新疆犯下种族灭绝的国家机构

据《CNN》援引《路透社》 2月26日报道：

荷兰议会周四通过了一项不具约束力的动议，称中共国维吾尔穆斯林少数民族的待遇等同于被种族灭绝。这是欧洲国家首次采取这样的行动。

活动人士和联合国人权专家说，至少有100万穆斯林被关押在偏远的新疆西部地区的难民营里。活动人士和一些西方政客指责中共国使用酷刑、强迫劳动和绝育。

中共国否认在新疆有任何侵犯人权的行为，并称其营地提供职业培训，是打击极端主义的需要。

荷兰动议说：“中共国正在施行对维吾尔少数民族的种族灭绝。”荷兰动议没有直接说中国政府要对此负责。

中共国驻海牙大使馆周四表示，任何关于新疆发生种族灭绝的说法都是 “彻头彻尾的谎言”。荷兰议会 “故意抹黑中国，并严重干涉中国内政”。

![hague]()![hague](https://gnews.org/wp-content/uploads/2021/02/Screen-Shot-2021-02-28-at-18.34.16.png)*位于海牙的荷兰议会大楼 – 图片来源于网络*

加拿大本周早些时候通过了一项不具约束力的决议，将中共国对待维吾尔人的行为标为种族灭绝。

荷兰议案指出，中共国政府的 “实施绝育措施 ”和 “设立劳改营 ”等行为符合联合国第260号决议，即通常所指的种族灭绝公约。

首相马克·鲁特（Mark Rutte）所属的保守派VVD党对该决议投了反对票。

**‘极大关注’**

外交部长史蒂夫·布洛克（Stef Blok），政府不想使用种族灭绝一词，因为联合国或国际法庭还没有公开宣布这种情况。

布洛克在动议通过后对记者表示：“维吾尔人的情况令人非常担忧,” 并补充说，荷兰希望在这个问题上与其它国家合作。

该动议的提出者、中左翼D-66党的立法者斯约德·斯约德斯马（Sjoerd Sjoerdsma）分别提议游说国际奥委会将2022年冬奥会撤出北京。

他在回答问题的电子邮件中告诉路透社；“认识到在中共国发生的针对维吾尔人的暴行是什么，即种族灭绝，可以防止世界舆论被误导，并迫使我们采取行动。”

中共国驻海牙大使馆在其网站上的一份声明中说，近年来新疆维吾尔族人口不断增加，享有较高的生活水平，预期寿命也更长。

声明中说：“你怎么能说这是种族灭绝？与新疆有关的问题从来都不是人权、民族或宗教的问题，而是打击暴力恐怖主义和分裂的问题。”

中共国驻日内瓦联合国大使周三指责西方列强利用维吾尔问题干涉中国内政。
