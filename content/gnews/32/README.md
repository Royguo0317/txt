针对中共病毒来源和传播真相，总部位于巴黎的法国最知名主流媒体，影响力最大的法文报刊之一VALEURS ACTUELLES 于1月3日发表文章如下：

近日，美国川普政府的一位前高级顾问官员确认，对于一年前始于中（共）国武汉的冠状病毒大流行，“武汉生化实验室泄漏”理论应该是该病毒最“可能”的来源。

之后，也是在一年前，首批冠状病毒病例随即在欧洲大陆被确诊。时至今日，社会各界对这种肆虐全球的瘟疫大流行的根源仍未查明。尽管考虑了所有可能性，美国政府的这位前高级官员仍然笃定“武汉实验室的泄漏”乃是目前最“可靠”的病毒来源说。与此同时，《每日邮报》（Daily Mail）的时政专栏作家，博明（Matthew Pottinger）也撰文表示对该理论的赞同并声称：现在，即使是中共国领导人也不得不公开承认他们先前关于这种病毒来自武汉动物海鲜市场的说法是错误的。

据称是病毒学家操作和改造了这一来自于蝙蝠的冠状病毒!

根据这位川普总统前国家安全副顾问的说辞，以目前所能收集到的最新情报显示，该病毒应该是从距离武汉海鲜市场18公里外的武汉病毒学研究所的实验室里泄漏出来的，而该研究所正是病毒学家们改造该源于蝙蝠体内冠状病毒的专门机构。该前白宫官员补充确认道：“越来越多的证据表明，该生化实验室可能是最可靠的病毒来源地。”

与以上病毒来源于武汉实验室的理论相呼应，《每日邮报》（Daily Mail）专栏作家Matthew Pottinger也指出：该病毒原体极有可能是由于“泄漏或事故”而意外从实验室逃脱的。

有鉴于此，来自世卫组织的一队专家们正准备前往武汉，试图对疫情大流行的起因展开调查。但对此，有极端批评者却担心认为：考虑到中共国长期对世卫组织施加的巨大影响，该病毒来源真相调查只不过是作秀而已。

必须一提的是，来自北京的宣传机构现在断言，没有证据表明Sars-CoV-2是来自大城市武汉的华南海鲜市场，那里甚至并无蝙蝠类动物出售。更何况这些蝙蝠动物的栖息繁殖地位于武汉市数百公里之外的地方，且人类记录的首例这种冠状病毒疾病从未在此被确认过。
