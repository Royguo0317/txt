摩根家族的发展史实际上就是美国的金融发展史，同时也是世界的金融发展史。历史学家将摩根家族及其合作伙伴网络称为美国巨大银行业帝国的一部分——“摩根公司”。

摩根家族一度发展成美国的金融大鳄，几乎控制了美国经济。虽然他三次挽救了美国经济，但是由于摩根家族对国家金融的影响力巨大，后来美国总统罗斯福，提出法案《格拉斯-斯蒂格尔》，该法案于1933年获得通过，法案制定了投资银行和商业银行的并购规则。

自此，摩根家族资产被迫分成五个分支，其中J.P.摩根公司成为一家商业银行，摩根士丹利成为投资银行。 一九三六年摩根家族的总资产超过七百八十亿，占了美国金融业的二分之一；分拆以后它的影响力至今还深深的影响着华尔街的一切，包括规则和战略。现在仍然是全世界最大的金融机构，同时也是对全球经济影响最大的一个家族。

那么他跟我们爆料革命为什么有这么紧密的联系，并且这次会作为嘉宾出现？因为这其中合作最大的一部分是Hcoin。摩根家族（摩根大通）跟郭先生合作Hcoin是最大的合作。

摩根家族涉及虚拟币早在二零一五年就开始了。大家可以上网去查一个叫R3联盟，这是二零一五年一家互联网金融科技公司叫R3 CEV，汇集了全球四十二家知名银行搞了一个内测：主要是银行票据（银行发行的商业商兑票据）；还有商兑票据的二次交易及赎回业务。准确的翻译过来讲，这应该叫银行商业票据数码化的一个尝试。在二零一五年运行了一年多，其中高盛、摩根斯坦利和摩根大通以及桑坦德银行都参加了，运行一段时间以后成果并不好，所以摩根斯坦利和摩根大通都分批从R3联盟中退出来了。

之后在二零一六年，摩根大通和花旗银行就单独在区块链技术方面投入了一百亿美元，去发展区块链的底层技术和应用场景开发。现在市场上实际使用的过程叫应用场景，它的底层技术过去是没有的。前段时间一直说这次是“三千年不遇的金融变革”就跟这个技术有关。

当时银行叫商业票据二次交易和赎回，银行金融业仅解决票据和二次交易是不够的，他们还需要解决供应链金融、金融票据、信用证、加密货币和发债，并且应用范围需要提高到能够适应数字化金融。

二零一六年，他们投资百亿做了很多的深化研究，尤其是在区块链底层技术开发，效果非常好，而且应用场景开发的也非常好。花旗在这方面的研究最早，而且试发行过数码货币，由于合规问题和信用透明度问题无法解决，但是后来放弃了，摩根大通试图尝试发行JPM数码币，也没有成功。JP摩根在二零一六年走的路跟花旗几乎是相同的。
