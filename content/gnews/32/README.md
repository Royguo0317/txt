**西班牙2021年3月25日电/西喜社——**据美联社3月25日报道，匈牙利周四创下了日死亡人数的新纪录，达到252人，在这个不到一千万人口的国家里，匈牙利的死亡人数达到了近1.9万人。

3月8日推出的额外封锁措施尚未产生效果，因为新感染和住院人数继续使之前12月创下的高峰相形见绌，医院报告说，由于重症监护室爆满，出现了接收能力问题。

匈牙利当局可能会将小学生和幼儿园学生的数字教育延长到目前4月7日的最后期限之后，以减缓最新的感染病例的强劲增长。

一名政府官员周四在接受采访时表示，鉴于创纪录的新病例和死亡人数，过去两周，匈牙利每100万居民的死亡率是世界上最高的，因此可能不得不延长截止日期。
