今天是5月16日。我对“五一六”这个数字比较敏感，它源于一段我的文革经历。

文革初期，停课闹革命也波及到了小学。作为小学生的我，对于不上课、没人管的状况很开心，整天跟在大孩子后面到处游荡看热闹。一天，我跟在大孩子后面跑去看抓到的反革命。在一个大院中央，里三层外三层围着很多人，只听见从人群中传来阵阵木棍敲闷鼓的声音——其实这是木棍打在人身上发出的响声——还听到微弱的哀嚎声。围观的人群高喊：“打死‘五一六’分子！”、“打得好！”我那时对周围发生的事情懵懵懂懂，但不能忍受那个打人的环境，便离开了。没过多久，听说那几个“五一六分子”被打死了。又过了几天，听父母说，被打死的那几个人不是什么“五一六分子”，而是邻省逃荒要饭的农民。这消息让我震惊，也让我感到害怕——被打死的“五一六”分子竟然是可怜的农民！
