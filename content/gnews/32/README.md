巴里克在2014年游说，反对暂停功能增强研究时断言：“功能增强是公共卫生准备和应对的关键工具之一。”（上面图片来自该视频：[https://rumble.com/](https://rumble.com/vihi61-wuhan-lab-partner-defends-gain-of-function-research..html) ）

“我想给你一个好消息。在危机中总有赢家的机遇，对吗？所以，是否你想做好准备，并在下一次大流行中赚钱；是否你想要这样做，那就买危险物质防护服制造商和防护服公司的股票，或者购买针对特定流行病生产抗病毒药物的公司股票。”他指出。

如上图所示，“有些产品做得很好。1918年包括口罩，今天也是如此，”他建议说。
