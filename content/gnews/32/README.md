**西班牙2021年4月17日电/西喜社——**

![]()![](https://gnews.org/wp-content/uploads/2021/04/161864878272916_P8181800.jpg)图片来源：und.com

美国投行摩根士丹利承认，对冲基金Archegos的倒闭使其损失近10亿美元。

这家华尔街银行在第一季度财报中表示，9.11亿美元(6.6亿英镑)的费用与 “单一优质经纪客户” 的损失有关，该行后来确认该客户为Archegos。

尽管受到Archegos的冲击，但得益于交易的繁荣，摩根士丹利在此期间的利润增长了150%。

净利润为39.8亿美元，高于去年同期的15.9亿美元。

净收入飙升至156.7亿美元，因为该行从一系列商业并购和公司上市中获取了更多的费用。

**Archegos****发生了什么？**

对冲基金通过买卖股票来赚钱。有人认为，Archegos在某些公司进行了一些大额投资，但这些投资开始出现问题。

它的支持者随后坚持要求它赶紧筹集资金，即所谓的追加保证金，促使一些异常大的股票销售。

催收保证金是指，如果用借来的钱进行的部分交易急剧下降，银行要求客户投入更多资金。

如果他们无力承担，贷款人就会出售股票，试图收回对银行的欠款。

截至3月底，Archegos欠摩根士丹利6.44亿美元，而该投行因卖出与对冲基金交易相关的股票，又损失了2.67亿美元。

首席执行官詹姆斯·戈尔曼（James Gorman）表示，注销其与客户Archegos交易的决定是 “必要的，钱花得很值”。

摩根士丹利是上个月被迫快速亏损卖出股票以回收资金时，对Archegos有风险敞口的六家大银行之一。

瑞士瑞信此前警告称，由于与现已倒闭的对冲基金进行交易，它将不得不承受超过42亿美元的损失。
