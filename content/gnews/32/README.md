![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/09130850/%E5%9B%BE%E7%89%8723-1.png)德国电影《帝国的毁灭》海报

1945年4月18日是历史的转折点，就在这一天，恶魔希特勒走进位于柏林总理府附近的地下室，开启自我毁灭倒计时模式，在地图上意淫如何反击盟军。

地面上，那个曾经举办奥运会、号称散发着新古典主义（来自阿尔伯特·施佩尔设计）建筑艺术的柏林城，已被苏军大炮炸成了残垣断壁。

苏军兵临城下，德国败局已定，还是有人坚信元首是在诱敌深入。希特勒临死之前依旧不忘愚弄人民，那些被纳粹思想洗脑的少年自愿当元首的保卫者，朝圣般地接受希特勒加持勋章，无知无畏地充当纳粹的炮灰。

此时的希特勒是个病夫，帕金森患者，身躯佝偻，头发散乱，眼神呆滞，右手不停地颤抖。当他得知德军再也无力与苏军抗衡后，大骂将领没有认真执行他的战略战术。

有好事者将电影《帝国的毁灭》希特勒发飙那段剪辑出来重新配上字幕，取名为《元首的愤怒》与时俱进编成网络段子，令人莞尔。

特勒恼怒之下签发命令，解除他最信任的空军元帅戈林一切职务。这还不算，希特勒在自杀前立下遗嘱，戈林不再是他的接班人，可见对戈林的憎恨有多深。

元首和元帅断舍离的戏码中国人民很熟悉，皆因毛魔头和他的亲密战友林彪元帅狗尾续貂一回，谁说历史不能重演？

除了帕金森，希特勒在心理和生理上都是病态的，瘾君子，畸恋亲侄女，狂躁、精神分裂、歇斯底里、强迫症、梅毒恐惧症和自大狂。

希特勒还是个懦夫，导致苏军围城的局面源于他的疯狂，闻到了战败气息，却迁怒于将军们无能。明明害怕大审判，偏说自己宁愿自杀也不受辱。

希特勒自杀看似壮举，实则为懦夫行径找借口。真正的战士不会选择自杀，而是马革裹尸战死沙场。跟我上、给我上，一字之差，凸显英雄与懦夫云泥之别。

独裁者都是一个怂样子，蛊惑人民为他去卖命，自己恨不能找个地缝钻进去藏身。如萨达姆，前脚举起金子打造的冲锋枪狂秀英雄领袖身段，后脚躲入地沟，直到被美国大兵揪出来为止。

那些希特勒的追随者，在元首穷途末路之际表现如何呢？

在纳粹宣传部长戈培尔看来，元首是战无不胜的纳粹之鹰，将战败的因果归罪于德国人民，正如戈培尔说言：“我们从来没有强迫人民，是他们选择了我们。现在，该是他们为自己的选择献身的时候了。”

戈培尔这句话赤裸裸地暴露出纳粹的无耻嘴脸，经济腾飞、奥运会、高速路，从战败国一举变成横扫欧亚大陆的第三帝国，希特勒给人民打鸡血，人民为纳粹买单，苏军攻陷柏林城，大肆强奸妇女，这就是被洗脑的代价。

希特勒的情妇艾娃，在地下室和希特勒举办婚礼，然后跟随希特勒一起开枪自杀。戈培尔全家人为希特勒殉葬。影片最令人唏嘘的是戈培尔七个孩子，被母亲连哄带骗服下氰化物。

戈培尔灭子理由是：“绝不让孩子生活在没有帝国的天空下。”

戈培尔太太给孩子们喂完药出来关上房门，有气无力地蹲下，没有眼泪，站在卧室门外的戈培尔面无表情。可怜的七个孩子，生错了时间，生错了家庭。对于纳粹，不能用人的思维和逻辑来解释。在戈培尔看来，跟随元首而去，死的光荣。

有影评家批评本片导演，过于渲染希特勒的末日凄凉，令观众产生同情。影评家多虑了，除了纳粹崇拜者，没人同情困于地下室的希特勒，导演恰恰抓住了恶魔末日的真实面貌，那个狂傲不可一世的希特勒，不过是个胆小如鼠的病夫、懦夫而已。
