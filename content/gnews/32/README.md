## 我的母亲
` 喜马拉雅巴黎七星农场` [轉載自GNews](https://gnews.org/zh-hans/692457/)

作者：阿月  校对：Pv0

我的母亲解放前出生在北方一个小山村里，她的父母早早地过世了，解放后只会写自己名字的母亲被招进工厂成为一名普通小工人，在工厂里认识了我的父亲，可能是嫁鸡随鸡嫁狗随狗的缘故，两人结婚后没多久就在那个动荡的时代，从北方一路颠簸来到南方的一个小城郊区工厂里安了家。母亲有一双单眼皮的细小眼睛，大方型脸满脸的雀斑，而且前门牙爆出，上下嘴唇被这排爆牙齿分了家。人长得丑点没关系，可是她特别喜欢吃大蒜伴黄豆酱，一日三餐也少不了的是酒一杯，饭后的烟一只，最让我不能忍受的是那张嘴里发出来的口臭味，心里就暗暗地发了誓：长大后决不吸烟喝酒！每当老师要求写我的母亲这样命题的作文的时候，我总是拿着笔发呆。过后大家在讨论时，都是单位里的孩子，彼此之间都是知根知底的，有同学就开玩笑地说我的母亲是地主婆。当时南方的女人没几个会吸烟的，而且会吸烟的女人给人的印象都不会是好人，羞愧的我真想找个地缝钻进去….

为了传宗接代，母亲再接再厉生到第五个孩子，终于等来了儿子，可惜已经穷得叮当响，只能喝着米粥长大的弟弟至今都耿耿于怀。老大是母亲的得力助手，老二是父亲培养的养老对象，老三的我照顾妹妹和弟弟，不知什么缘故笨手笨脚的老二总是成为了母亲的出气筒……无论是学校里的事情还是家里的事情，孩子们和固执的父亲争辩的时候，母亲却从来不说话，不放声，继续喝着自己的小酒….

时光如流，从前的丫头片子们都长大了，每次回家看望母亲时都会带上各种礼物，里面都少不了的是酒和烟，临走时也会留下些零花钱。省吃俭用的母亲用这些钱买了一套单位的商品房，终于住进了自己的房子。去年的某一天，母亲最后一颗爆牙掉了，嘴唇终于可以团聚了，可脸盘也变得瘦小了，眼睛成了一条线，身体也一天不如一天，不见棺材不掉泪的她依然没有改变的是一日三餐的酒一杯，烟一只。有点家产就是不一样了，在单位里腰杆挺直了嗓门也大了。一天弟弟来找我说，母亲想去公证处把她的房子留给弟弟，怕我不同意先来探试一下我的口气，面对做一天和尚敲一天钟的弟弟，没文化又没内涵一生走来不容易的母亲，我知道母亲心里装着满满的都是她唯一的宝贝儿子。到了这把年纪我还能改变什么，只要她开心做她自己想做的事情就好。留给母亲的日子越来越少，我不在乎那房产价值的多少，可还是因为母亲的重男轻女让我伤心了一些日子……

武汉病毒的爆发和蔓延，考验着世界各地人们的人生观价值观，渐渐的把世界清晰地分成了两个水岭，我自然而然地成为了少数，发小群同学群都容不下我，更夸张的是在我的家族群，我最亲的在日本生活了几十年的大姐也在无理由反驳的情况下把我踢了出去。一天无事不登三宝殿的妹妹微信找我，告诉我母亲有话跟我说，有种太阳从西边出来的感觉。从我19岁走上社会，经过了风风雨雨迎来了人生的50个年头，这是我记忆中第一次母亲主动找我，可母亲却用严厉的口吻说要听大姐的话，不要关心政治，你什么都管不了，管好你自己就好了……我哑言失笑了，这就是从来只会看新闻联播，只喜欢看各种虚构夸张打日本鬼子的电视剧的母亲，自己的一个孩子却嫁给了日本人而因此在单位里终于可以扬眉吐气的母亲，看着她满脸的皱纹，听着她连吸带喘沙哑的声音，我无言以对….

这就是我的母亲，她不仅长得丑而且还有那么多的恶习，但也含辛茹苦地把我们抚养长大。做事不公没有原则的她依然是我的母亲，我依然关心她，感谢她带给我的一切。

子不嫌母丑、家丑不可外扬这些流传已久的古言，也是在争辩中听到最多的几句，这丑字用在我母亲身上再合适不过了。只是时代不同了，丑字的内容已经今非昔比。试想一下如果我的母亲是一个小偷，是一个拉皮条者，是一个吸毒者，是一个捞地沟油做小零食的奸商，是一个拐骗别人家孩子的人贩子……这还能用丑字来形容衡量吗？这已经超越了古人认知的道德底线，已经构成了犯罪行为……时代的变迁，许多古言都已经无法再形容当今的乱象，在这恶魔当道的世界里，我们不仅要练就一身火眼金睛识别真善恶的本事，而且还要跳出红色教育的框框条条，认清世界的本质。作为母亲的我不希望如同我的母亲一样，只要把孩子喂饱养大就完成了任务，更不希望被完全洗脑，失去了辨别是非能力，只能在昏沉中度日。

再看看我曾经热爱的现在也依然热爱着的被洗脑后称为母亲的祖国，49年它的成立到70年大庆的这段岁月里，一路走来就是用鲜血和谎言写就的历史，而那些鲜血背后的故事不但惨绝人寰，而且鲜为人知。从一开始的 “镇反”和“土改”屠杀了农村乡镇的地主阶级，然后是“三反五反”屠杀了城市中的资产阶级，然后是政治运动“大跃进”造成的大饥荒，湖南广西上演了平民百姓人吃人的骇人听闻的人间惨剧。接着是“十年浩劫”，然后是对西藏人民的镇压，改革开放的八九六四血腥屠杀青年学生。之后对法轮功、异议人士、民主人士和新疆少数民族的迫害，到今天东方之珠香港学生遭受的“被跳楼”“被浮尸”“被性侵”等迫害……几千万无辜的生命被掠杀在所谓的祖国母亲的怀抱中，更不要说那些夫妻被强迫离婚的、父母子女被迫划清界限的，将人致残、逼疯的，从肉体上精神上残酷折磨至死的等等数不尽的人间悲剧，造就了多少痛苦的家庭，多少支离破碎的家庭….虎毒不食子，面对这一切残酷的历史和现实，试问一下有谁的母亲会这样残害毒杀自己的孩子？这哪里是丑，连禽兽都不如，简直就是恶魔！
![]()![](https://gnews.org/wp-content/uploads/2020/12/IMG_5151.jpg)
但是请记住我们的母亲只是一位：和我们有着血缘关系的女人！也请不要冤枉了我们的祖国，祖国是祖先开辟的生存之地，人们崇拜、爱惜和捍卫这片生生不息世代相传的土地。今天我们的祖国是我们脚踏着的有960多万平方公里的中华大地，它也经历了坎坎坷坷几千年，它也是百孔千疮，历尽磨难；在这片土地上有着妖魔用一切邪恶手段妖惑着人间，不要被她美丽狡诈的笑容、华丽虚伪的外表所蒙蔽，应该勇敢地站起来揭露她邪恶的真面目，传播真相，唤醒世人，赶走这妖魔，找回那敬畏天地神、有人性有诚信的故土我的祖国…….
