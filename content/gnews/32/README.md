中国在很大程度上遏制了新冠疫情，因此在过去几个月中对自己政治制度的信心大大提高。中国国家主席习近平说，这次抗疫斗争有力彰显了中国国家制度和国家治理体系的优越性。在社会、经济以及香港治理问题等方面，今年的人大会议将作出具有深远意义的政治决定。

新的五年规划将在本次两会期间通过。该文件充满了自信，涉及的时间段实际上已经远远超出了2025年。在迄今发布的公报中，“2035年”一再出现。届时中国应该基本实现社会主义现代化。

***简评：**这标题真是讽刺和荒唐。两会一年一度的政治小丑表演秀时间，竟然会增强中国政治制度信心。天天喊道路自信、理论自信、制度自信、文化自信，却通过一个所谓的外媒来为自己脸上贴金。2035年中国应基本实现社会主义现代化，就像脱贫一样。党让你贫，就扶着你贫，党不让你贫，就全国都脱贫。这边宣布脱贫，就立即下文，贫困人的补助没有了，而且还要缴纳比以前几倍的所谓保险资金。*
