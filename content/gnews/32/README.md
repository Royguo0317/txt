斯皮尔伯格执导过一部电影叫《间谍之桥》，讲的是冷战时期美国政府抓捕、起诉苏联间谍阿贝尔的故事。由汤姆·汉克斯扮演的多诺万律师第一次探访阿贝尔，劝他与美方合作时被阿贝尔断然拒绝。多诺万吃惊于眼前的这位干瘦老头，他那副冰冷的表情，看不出一丝胆怯。多诺万问阿贝尔：“你为什么不害怕？”

阿贝尔冷冷地回答：“Would it help?” （这样有用吗？）

这句话阿贝尔一共说了三次。第二次在初审开庭前，多诺万告知阿贝尔，庭审对他这个间谍来说极为不利：苏联方面卸磨杀驴不承认他是苏联公民。多诺万问他是否担心自身安危？阿贝尔反问道：“Would it help?”第三次是美苏双方在一座桥上交换间谍，即将回到苏联的阿贝尔告诉多诺万：“我的人民会枪决我。”多诺万再次问他，你害怕吗？阿贝尔依旧面无表情地说：“Would it help?”

拜登当选总统，一些战友灰心失望溢于言表，令笔者想起阿贝尔这句话。文贵先生发起的爆料革命，本质上是一场实力悬殊的生死对决，可谓一郭战一国。郭先生一直强调，以美灭共、以共灭共的策略。就像郭先生说的那样，谁灭共我们就支持谁。当郭先生、路德先生说出川普百分百是赢家，并非夸大其词，而是根据时下各方情况做出的预判。班农先生、朱利安尼先生、鲍威尔律师和林伍德律师都曾断言川普会赢得大选，包括川普本人，在投票当日宣布自己胜出。你能说他们都是忽悠吗？

若从中国人非黑即白的单向思维角度讲，川普没有连任，最失望最有资格骂他无能的应该是7500万选民。而事实恰恰相反，我们看到川普车队到达佛州时，街道两旁站满了欢迎川普的人们。

1.19，是注定载入史册的伟大之日。2020年这天，闫丽梦博士通过路德社爆料，向全世界揭露中共病毒真相，挽救无数人的生命。2021年这天，国务卿彭佩奥先生发表声明，将中共定为反人类罪和种族灭绝罪。遇神亵神，遇佛杀佛的恶魔中共终于被套上了绞索。

从郭先生最新透露来看，如果川普能连任，敌对国和两罪不会这么快发布：中共早就号准川普的脉搏，通过“买买买”，能拖一天是一天。川普是美国总统，做什么首先考虑的是美国利益最大化，这么做外人无可厚非。

郭先生常说爆料革命有天助——对中共定罪就是天助，是上帝的安排。1.19之前，灭共在美国是“非法”，毕竟中共还是美国承认的合法执政者。1.19后，灭共是合法，和中共勾兑是非法。今天推特火速关闭中共驻美大使馆账号就是开端例证。风向变了，谁沾中共谁完蛋，中共就是那个人人喊打的过街老鼠。

四年来每逢出现挫折，郭先生都能从中看出新的机遇，成为射向中共的子弹——这是爆料革命不被中共摧毁的关键所在，亦是诸多战友跟随郭先生加入新中国联邦的原因。世界上没有一帆风顺的事：唐僧取经历经了九九八十一难，犹太人磨难中不放弃信仰才有了今天的以色列国。

孟姜女哭倒长城是美丽的传说，也是弱者的意淫——长城不相信眼泪。新中国联邦不是失望出来的，中共两罪更与抱怨无关。失望与抱怨，从来都是弱者的代名词，这和访民们盼望有个包青天出来为自己做主是一样的心理。失望与抱怨，只会让敌人更加肆无忌惮和强大，对个人来说，正如阿贝尔所言：“这样有用吗？”
