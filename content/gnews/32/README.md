共同社3月4日报道，中日邦交正常化50周年即将于明年来临，面对图谋改善关系的中共，日本无意迎合，一年前就被日方推迟的习近平作为国宾访日议程已被无限期搁置。日本外务省干部断言“在工作层面完全没有进行协调或相关讨论”。中共外交部发言人汪文斌在本周一的记者会上说“中日双方未就有关重大议程进行探讨”。

由于日中在尖阁诸岛(钓鱼岛)的对立及中共在香港的恶行等问题，导致日本民众对中共持悲观及反感态度。日本内阁府2月公布的民调结果显示，认为当前日中关系“并不良好”的占81.8%，对中共国“没有亲近感”的达到77.3%。

日本媒体的新闻报道通常比较谨慎。中共和日本在钓鱼岛上的争端由来已久，中共在香港最残暴的镇压是在去年，但当今日本的民调结果显示日本民众对中共的恶感出现了前所未见的窜升，其主要原因应该是日本人已普遍接受中共发动超限生物战的事实。如果日本政府不顾及激愤的民意，继续运作习近平访日，肯定招致朝野的一致反对。

据共同社今日另外一篇报道，本周四日本厚生省公布，目前在全球迅速蔓延的英国、南非、巴西三种传染性极强的变异CCP病毒，已扩大至日本47个都府中的19个。日本专家称“比现状更急速蔓延的风险很高”，有可能导致“第四波”疫情；日本国立感染症研究所透露，国内病例中93%的人不曾前往海外。

随着日本疫情的反复加重，日本人民的愤怒情绪将对政府形成巨大的压力，有可能促成日本政府参与到新中国联邦以毒灭共的正义行动中——这一天必定到来！
