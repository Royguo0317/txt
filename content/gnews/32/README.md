“据《印度快报》11月25日报道，根据印度内政部网络犯罪协调中心的综合报告，印度政府根据《信息技术法》第69条A款条文**禁止了****43****款中国****APP**。这些APP当中包括阿里巴巴旗下的支付宝、速卖通（AliExpress）、钉钉、淘宝直播，腾讯旗下的腾讯视频海外版（WeTV），以及在中国市场知名的芒果TV、快手短视频等。印度政府称，‘采取这项行动的依据是，这些应用程序参与了不利于印度主权、国防、国家安全和公共秩序的活动。’ 印度政府已于今年6月29日**封杀了****59****款中国****APP****，****9****月****2****日又封杀了****118****款中国****APP**。已经封杀的APP中包括TikTok、微信、小米浏览器、美拍等知名产品。截至目前，印度封杀的中国APP数量已达220款。”
