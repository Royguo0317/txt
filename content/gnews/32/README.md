今天看墙内新闻报导一条消息杨洁篪应约同美国国务卿布林肯通电话，不出所料该新闻又是通稿。新闻中提到了中方各种表态，强调，指出，有关台湾问题，世界秩序药理联合国宪章为基准以及在病毒问题上向美国泼污，对于布林肯表示了些啥，说了啥这篇通稿只是一笔带过。

我发现，现在墙内的这种新闻通稿基本这样，开头就是中方又臭又长的一大篇如何如何，结尾外方说些啥都是一句带过，当老百姓的谁也不知道外方代表到底说了啥。我就纳闷这算哪门子国际新闻？国际新闻不就是让国内老百姓知道外国啥情况么，这倒好外国人说啥咱不知道，就知道所谓的中方代表说些啥。我就纳闷了这种没有前因没有结果的报道让人怎么看。指着这些没头没尾，没前没后的东西能洗脑老百姓？

从这篇通稿我至少看到了以下三点：第一，提台湾问题说明前段时间的美国军用运输机C-17全球霸主到访台湾真的是让感到ccp很害怕，它很害怕它才会反复强调台湾是中国不可分割的一部分如何如何。说白了就是怕美军常驻台湾，怕美军耽误自己在南海和台湾大计划。

第二，提了所谓的一个体系，一个秩序。就是现在美国和英国重新签了大西洋宪章。又让ccp感到害怕了，怕自己被踢出这个自己“只占便宜，不吃亏”的现行WTO经济贸易体系，说白了就是怕世界文明国家和自己脱钩。口口声声的以联合国宪章为基本准则。殊不知联合国宪章都是根据大西洋宪章改编而来的。现今的二战后文明秩序能够持续100多年都是各个国家遵守大西洋宪章的结果。还拿美国人权问题说事呢。自己家49中事件，甘肃马拉松事件以及江苏职业本科事件等等你尊重人权了么？各种维权被打，维权者社交媒体账号被封杀这种事实在太多了吧。这么些年听ccp的新闻说美国人权问题都耳朵都起茧子了，有没有具体说说哪些侵犯人权的事啊。一提美国人权问题怎么老是翻来覆去就那几件事？
