#### 作者：RU WEI

2021年2月26日，国产宫颈癌二价疫苗在北京启动第一针接种。在民航总医院体检康复中心，积水潭医院妇产科副主任医师赵芳接种了北京首针疫苗。这位女士在打完第一针疫苗后1个月、6个月再次接种，即可完成接种程序，能有效远离84.5%的宫颈癌发病风险。这意味着，我国摆脱了宫颈癌疫苗完全依赖进口的状况。目前，在我国上市的宫颈癌二价疫苗分进口和国产两种，市民选择哪一种更合适呢？据了解，该款国产宫颈癌疫苗一针价格为329元，加上注射费，三针接种完毕花费仅需1000元左右，是进口二价疫苗价格的一半。除了价格优势之外，对于低年龄组，国产二价疫苗还有一个优点就是接种疗程短。
