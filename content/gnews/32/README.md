新德里——美国国务卿彭佩奥（Mike Pompeo）和国防部长马克•埃斯珀（Mark Esper ）在这里与印度同行进行了为期两天的会谈，旨在面对中共国日益增长的侵略，加强安保联系。

所谓的“2+2”外长和国防部长对话的最近一界会议包括签署一项新的协议，以加强军事和情报合作。

所谓的“2+2”外长和国防部长对话的最近一界会议包括签署一项新的协议，以加强军事和情报合作。

埃斯珀先生与印度国防部长拉杰纳特•辛格举行了会谈，彭佩奥先生会见了外交部部长苏伯拉曼尼亚•贾尚卡。两国还希望增加国防装备的共同生产和共同开发。

埃斯珀先生说，会谈在加强美印关系方面取得了实质性进展。

埃斯珀在会谈结束时的新闻发布会上说：“基于我们共同的价值观和共同利益，我们肩并肩支持为所有人建立一个自由开放的印太地区，特别是鉴于中共国侵略的加剧和破坏稳定活动的增加。”

彭佩奥——川普政府推行“对抗中共国”新政策的动力——称此次会谈是调整“我们民主国家”对抗中共国威权主义侵犯的重要一步。

[!\[\]()!\[\](https://spark.adobe.com/page/6fxVeK4IYbwnP/images/22999d0a-3d5b-4c3c-946f-75fb85035b16.jpg?asset_id=20d9c4a4-fdfe-48d3-8967-e65e1580d0a8&amp;img_etag=%223536d0bdc74517ec6e9bb16946dbe1f1%22&amp;size=1024)](https://spark.adobe.com/page/6fxVeK4IYbwnP/images/22999d0a-3d5b-4c3c-946f-75fb85035b16.jpg?asset_id=20d9c4a4-fdfe-48d3-8967-e65e1580d0a8&amp;img_etag=%223536d0bdc74517ec6e9bb16946dbe1f1%22&amp;size=1024)图片来自网络

彭佩奥说，关于（新冠）疫情的讨论使得美国“卫生与公共服务部”与印度卫生当局之间加强了合作。

彭佩奥说：“战胜来自武汉大瘟疫的挑战也引发了关于“中国共产党”的激烈讨论。我们的领导人和我们的公民越来越清楚地看到，中共没有对民主、法治、透明或航行自由以及建立一个自由、开放和繁荣的印太地区的承诺。”

但在新闻发布会上，印度领导人在讲话中没有提及中共国，这与新德里处理北京问题的低调态度是一致的。

国防部长辛格先生说，只有两国所面临的重大挑战的议题得到了深入讨论，美印两军之间的合作“进展顺利”。

最重要的协议是美国国家地理空间情报局（National Geospatial Intelligence Agency）、美国摄影间谍局（photographic spy Agency）与印度国防部（Indian Defense Ministry）之间签署的《基本交流与合作协定》（BECA）。该协议将加强情报共享，以监控中共国在印度洋的海军活动，包括广泛的潜艇部署。

双方还达成了海上信息共享技术协议。

最近的会议是此类会议的第三次，是美国在亚洲发展更紧密的联盟以对抗中共国而作出的重大外交努力的一部分，以应对该地区在陆地和海洋争端方面的加强活动。

中共国国有媒体《环球时报》周二在报道这次峰会时说，这些协议可能会提高印度的军事能力和武器精准度，但中共国分析人士得出的结论是：“印度绝对没有机会扭转中印军事力量差距的现实，更不用说在可能的军事冲突中打败中共国。”

周二早些时候，彭佩奥和埃斯珀在印度国家战争纪念馆献上花圈，提醒人们最近与中共国发生的致命冲突。今年6月，中共国军队与印度军队在有争议的高海拔边界发生冲突。这场战斗造成20名印度士兵死亡，中共国军队死亡人数不详。

美国和印度官员说，新协议是增加后勤支持和海上合作（主要是联合海军合作）的一部分。最近，美国海军尼米兹号航空母舰在印度洋与印度海军进行了一次“过关”演习。去年，两国军队举行了首次由海军、海军陆战队、空军和陆军参与的三军演习。

澳大利亚最近第一次同意在11月份与来自美国、印度和日本的军舰一起参加印度的马拉巴尔海军演习—所谓的“四国联盟”（Quad-of-States），旨在反击中共国在亚洲日益加剧的地缘政治侵犯。

退役海军上尉吉姆•法内尔（Jim Fanell ）说，《基本交流与合作协定》（BECA) 将加强海上作战情报共享，并向北京发出强烈信号。

法内尔在最近的一篇评论文章中说：“这次会议是在中共国（PRC）不断扩张、越来越咄咄逼人和暴力行为的背景下举行的，特别是它涉及到确保航行自由和自由进入公海（市场）。”

法内尔上尉说，美印军方所做的不仅仅是谈话，而且还共同行动。本月初，一架海军P-8海上巡逻机降落在印度安达曼和尼科巴群岛的布莱尔港。

由于担心印度作为所谓的不结盟国家集团（nonaligned nations grouping）领导人的传统角色，印度一直不愿与美国加入正式的国防与安全联盟。然而，印度政府因为对中共国侵略性的日益担忧而改变了看法。
