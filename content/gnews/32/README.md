美俄领导人即将于6月16日在日内瓦举行峰会，英国广播公司(BBC)近日对此峰会做了特别的前瞻性报导，并指出，拜登(Joe Biden)和普京(Vladimir Putin)的见面不会是一场友好的会晤。

**目前美俄关係处于低谷**

报导认为，拜登政府上台后，与俄罗斯的对抗加剧。俄罗斯也将美国列入其官方“不友好国家”(Unfriendly States)名单，双方目前都没有相互派遣大使，两国都形容双边关係处于历史低谷。

较早前，一些俄罗斯高级官员因俄国吞併克里米亚、涉嫌干预选举等事情受到美国的制裁，两名前美国海军陆战队员现在还被俄罗斯关押在监狱中，其中一名被指控从事间谍活动而被判处高达16年的刑期。最重要的是，拜登在三月份时赞同了媒体称普京是一个杀手(Killer)的观点。

**白宫表示需要和俄罗斯建立可预测的关係**

报导称，拜登这次欧洲之行是他入主白宫后的第一次外访，日程安排紧凑，包括周一在布鲁塞尔北约(NATO)总部的会议，然后将在日内瓦（Geneva）与俄罗斯的普京举行一对一会谈，此举给了克里姆林宫很大的面子，况且这是一次完整的峰会，而不是走廊式外交的短暂会谈。白宫方面表示，美国需要和俄罗斯建立“稳定”(Stable)和“可预测”(Predictable)的关係。俄罗斯自2014年起，使用“小绿人”(Little Green Men)方式，即由俄国装备的秘密武装人员，进入并吞併克里米亚，让世界，特别是西方各国对普京的外交政策保持警惕。

普京最近特意强调，他对西方的看法是敌对关係。在圣彼得堡举行的经济论坛上，普京再次声称，美国想要“遏制”(Contain)俄罗斯的发展。几天前，他曾威胁要击碎任何外国侵略者的牙齿，认为世界需要明白俄罗斯复甦后的地位和实力。更扬言“能够解决这些问题的机制才不致使经济峰会不会白费功夫”。

**俄罗斯智库：普京需要透过美国获取政治及经济好处**

报导指出，莫斯科的智库“俄国国际关係委员会”（RIAC）总监科图诺夫(Andrei Kortunov)认为，拜登和普京以总统身份会面本身就已是一项成就，峰会的象徵意义很重要，它使俄罗斯达到与美国同一联盟的高度，普京也在意这一点。科图诺夫续说：“普京总统认定美国不希望俄罗斯大展拳脚，这一观念应该不会改变。普京作为一个理性的政治家，在目前的敌对关係下，降低相关的成本、风险和张力也是件好事。”在俄罗斯民意方面，科图诺夫表示：“俄罗斯国民对外交政策的胜利并不感兴趣，而更著眼于国内紧迫的社会和经济问题。很明显，普京已经无法在国内施展魔法获得政治或经济好处。”

政治评论员舍夫佐娃（Lilia Shevtsova)对这次会议有另一种看法，她认为，这是俄美关係下滑的开始，普京从心底里绝对希望与美国总统平等，在其任期内得到重视，还更想展示一下雄风，成为世界主要人物的一员。所以俄美两国会先测试彼此的红线，然后在最坏的低谷寻找对话点，如果事态发展到双方不能进行对话的僵局，那麽俄罗斯将变得更加难以预测。对即将进行的美俄峰会，舍夫佐娃作出了如下预测：“作为这次会议的餐前小菜，拜登会打出有关反对派人士纳瓦尔尼(Alexei Navalny) 的人权牌，同时普京会反击美国也在做著同样的事情；而后的主菜就是：降低紧张局势。”

另一波俄罗斯评论家认为，“外交战争”有可能会稍微缓一缓。由于近年来美国已驱逐了数十名俄罗斯外交官，驻俄使团也禁止僱用当地人，这导致相关的签证服务大幅削减。美国的举动也会改变俄罗斯囚犯之命运，包括2018 年被捕并否认间谍罪的惠兰 (Paul Whelan)。俄罗斯也可能会往前迈一步，允许其大使返回华盛顿。最近俄罗斯再次推动交换囚犯，但普京提出的条款不太可能满足美国的要求。

报导认为，在会谈中，普京最不想被问及的话题应该是人权问题，因为那必定包括被投毒后关进监狱的反对派人物纳瓦尔尼。莫斯科法院刚刚认定纳瓦尔尼的政治办公室和其反腐败组织为“极端份子”，但这一裁决很容易将推迟到峰会之后。普京似乎更想发出这样一个信息：俄罗斯内政和美国无关。
