## 【世事解评】中国“人口红利”慢慢消失，产业转移进行时！

香草山金融部 维度



![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/12160942/logo-3.jpeg)



根据维基百科定义，在经济学上，“人口红利（英语：demographic dividend）是指因为劳动人口在总人口中的比例上升，所伴随的经济成效应。它通常发生在人口转型时期晚期，此时因为生育率下降，使得受抚养的幼年人口减少。当人口负担系数小于或等于50%，此时称为人口机会视窗期。”

我对“人口红利”的理解就是整个社会中，年轻、能工作、能劳动、能积累财富的人较多；年迈、体弱多病、年幼、没劳动能力还花钱的人占比较少。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18201659/16x9-feature-photos-MOS-logos-2-14.png)
中国的人口周期的周期是被国家机器强制干预的。如果是按照自然的人口变化周期，中国的人口趋势不应该是现在的周期。但是中共国作为集权国家，权利过于庞大，国家干预力度过于野蛮和强悍。这些都导致了中国的人口变化周期被改变了。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18202008/2-38.png)
中共国“计划生育”背景下，人口趋势开始改变。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18201836/3-46.png)
根据维基百科收录，1978年宪法第五十三条规定“国家提倡和推行计划生育”。现行1982年《中华人民共和国宪法》对计划生育有多项规定：

- 第25条规定“国家推行计划生育，使人口的增同经济和社会发展计划相适应”
- 第49条规定“夫妻双方有实行计划生育的义务”
- 第89条规定“国务院领导和管理全国计划生育工作”
- 第107条规定“县级以上地方各级人政府管理本行政区域的计划生育行政工作”


在中共国不断的人为干预下，原本的人口变化周期的节奏被打断了。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18201635/4-19.png)
上图是中共预估的劳动人口分布比例趋势图：粉红色的是劳动年龄人口，黄色的是老年化人口。从图中我们可以看出，CCP 预估的是2045年人口红利趋势开始逆转。可事实上，人口红利趋势可能更早就开始逆转。

英国媒体《金融时报》(Financial Times)上周三披露，中国政府的最新人口普查结果显示该国人口出现近六十年来首次下降，缩减到十四亿以下。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18201534/5-22.png)
随后官方给出的数据和英国媒体报道的有比较大的出入。这里面的真假辨别，我想各位读者都有自己的判断。除了印度，东南亚和一些南半球的国家也会陆续进入人口红利周期。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18201431/6-9.png)
上图是世界银行，2016-2016对于全球人口的监测报告，图中也可以看出：中国和俄罗斯，都是属于人口红利周期的末期。东南亚，印度等国是人口红利的早期阶段，也是后续最有利于经济发展的地区之一。

那么“人口红利”对于全球的产业分工有没有什么影响呢？
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18201356/7-10.png)
- 根据上图历史大数据，我们可以看出人口红利前期，也就是红利还不明显的时候，国家都是以售卖资源、低端或者高端制造业为主要经济来源。红利末期和后期是以高技能制造业为主要经济来源，日本，韩国就是例子。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18201318/8-9.png)
处在早期的人口红利国家绝大多数在中低收入国家；高收入的国家，现在基本上都处在人口红利的后期，正常情况下人口周期是在不断的循环的。

上面世界银行的数据说明，未来人口红利的国家，不仅是很多，而且人口数量也足够承接产业转移。而且这些未来人口红利的国家都是低收入的国家。这对于中共统治下的中国来说，绝对不是什么好消息。因为世界的产业分工是很明确的，如果不能成功的产业升级，就会陷入前不着村，后不着店的尴尬境地。

在中国进入“人口红利末期”的以后，下一步会进入“后人口红利时期”。在这样的大周期下，国家在世界经济的分工中主要以高端的制造业和高端的服务业为主。中共治理国家的策略是“一民政策”。国家只需一种民：耕战之民，即可以创汇和打仗的人民。那些既不能给国家创汇，也不能为国牺牲的人，在中共的眼里就是废物，是被清除的对象。

在“一民政策”的基础下，中共又实行“愚民教育”。国人没有独立的思考能力，没有责任心，“假大空”，浮躁和急功近利。因而中共国不可能有高精尖的产业。当产业转移后，中共国会迎来大面积的失业。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18201244/9-11.png)
尽管，中国“人口红利”还有几年窗口期，但是作为投资者，一定是看预期，也就是在人口红利真正的转折点之前，大批的企业或是投资者一定是提前布局，转移其产业链。加上中共的愚蠢,傲慢和嚣张，满世界放毒，必然导致其更快的与全世界经济脱钩。

这方面的信息和行动越来越多，还会越来越快。在我们可见的未来几十年，中国的经济会异常的艰难。解决这个困局的办法只有一个，那就是消灭中共。
