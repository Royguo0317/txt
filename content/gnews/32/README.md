**西班牙2021年4月21日电/西喜社——**

中共国内一家媒体援引一位专家的话说，如果政府不尽快取消其限制家庭生育两个孩子的政策，未来五年中国的出生人数可能会减少到每年1000万以下。

广东省人口发展研究院所长董育正对中国金融新闻媒体壹财表示，未来几年中国的总人口也可能下降。

根据国家统计局的数据，2019年中国出生的婴儿数量下降了580,000，降至1,465万，出生率为10.48％，这是自1949年开始采用目前的数据采集方法以来的最低水平。

尽管通常统计局会在2月底发布此类数据，但它目前尚未透露去年的数据。

预计该国出生率的下降和社会快速老龄化将考验其支付养老金和照料老年人的能力。

尽管中国于2016年取消了长达几十年的独生子女政策，但由于医疗保健，教育和住房成本上涨，多数夫妻不愿拥有成员多的家庭。 COVID-19大流行带来的经济不确定性进一步影响了生育孩子的决定。

来自公安部的单独数据显示，去年的出生人数从2019年的1179万下降了15％至1003.5万。

南方城市深圳的劳工专家刘开明说，农村地区的一些儿童可能没有统计在内。他预计到2020年新生儿的数量将在1000万至1400万之间。

刘说：“明年的出生人数可能会低于一千万。”

此前中共表示将在四月初公布十年一次的人口普查结果，至今尚未结果。
