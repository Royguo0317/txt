## 爱尔兰在遭受黑客攻击后关闭了医疗保健网络技术系统
` 喜马拉雅意大利罗马达芬奇农场` [轉載自GNews](https://gnews.org/zh-hans/1247637/)

信息来源：德国世界报网络版 
发布时间：2021年5月14日
翻译 /整理/ 简评： Shuizhuyu

在美国输油管道遭到网络攻击一周后，爱尔兰卫生部门成为了类似攻击的目标。因此，公共卫生服务（HSE）必须在周五关闭其整个计算机系统。在线服务推特的权威人士说：“公共卫生服务的网络技术系统受到了严重的勒索软件攻击。”作为预防措施，所有医疗保健网络技术系统都已关闭。根据初步发现，黑客攻击被归咎于追捕健康数据的国际罪犯。

据公共卫生服务称，袭击不会危及患者的安全。 “对于给患者和公众带来的不便，我们深表歉意。”爱尔兰政府还明确指出，疫苗接种的过程不受该过程的影响。疫苗接种“按计划进行”，救护车服务和急诊室也不受影响。

根据最初的发现，一周前，在针对美国最大输油管道的黑客攻击中，类似的勒索软件被用于网络攻击。根据美国联邦警察的说法，这次袭击是由犯罪集团 “黑暗面” (Darkside)进行的，并导致在美国东海岸的加油站发生恐慌。

神秘的黑客组织 “黑暗面” 于去年出现。黑客特别针对大公司，据媒体报道，勒索勒索的费用高达数十万甚至数百万美元。该组织称自己是非政治的，只对金钱感兴趣。没有与任何政府的联系。但是，美国指控肇事者在俄罗斯，并且该恶意软件起源于俄罗斯。

在勒索软件攻击中，黑客试图锁定或加密计算机系统，并向用户勒索钱财以释放数据。 2017年，美国和英国将朝鲜大规模的恶意软件攻击归咎于朝鲜，该攻击影响了150个国家/地区的约30万台计算机。在“ WannaCry”攻击中，英国三分之一的医院以及西班牙电信公司（Telefónica）和美国物流公司联邦快递（FedEx）受到攻击。

简评： 爱尔兰的医疗保健系统遭到了黑客的攻击， 黑客的主要目的是勒索。 现在已经给这个黑客组织定性为犯罪集团。 美国指控肇事者在俄罗斯。 那么回想起来， 在美国大选的时候，有些黑客行为据说就是来自俄罗斯，在俄罗斯的背后就有中共的影子。 那么这次对美国和爱尔兰的黑客袭击有没有可能背后也是中共呢？我们不得而知，但是， 就现在的灭共形式来看， 中共的疯狂可以导致它干任何事情。 看看最近一周以来， 世界各地的不太平，至少我们知道，哈马斯对以色列的攻击是有伊朗的支持， 而伊朗的后面一定是中共。 黑客袭击又是中共常用的伎俩。 相信美国的技术能有一天找到幕后黑手。 也相信下周美国和俄罗斯两国外长的会面也会谈到这个问题。
