美国民主党总统候选人拜登之子亨特与中国和乌克兰商界的不当交易，引发极大争议。事实上，美国华人自媒体「路德社」上月底（九月24日）就披露，不满总书记兼国家主席习近平的中共「江泽民派系」人马，早将3枚电脑硬盘交给美国政界，内容提及亨特与习近平高达45亿美元的秘密协议、亨特的色情影片，以及中共的生物武器计谋等。

**川普已拿到副本**
美国犹太媒体「犹太和平之声」（Jewish Voice）19日指出，「路德社」在上月24日的节目中踢爆，由于中共内部发生内鬨，效忠江泽民、曾庆红、孟建柱等「江派」主要人物的3名党员，已将3枚电脑硬盘提供给美国司法部和民主党籍众议院议长裴洛西，总统川普也得到一份拷贝。

据指出，第1枚硬盘是关于亨特及前纽约市长、「彭博新闻社」创办人同时也是拜登大金主的彭博（Michael Bloomberg），在中国被引诱及偷拍的性爱和虐童影片，这是中共高层惯用勒索外国政商人物的手段。不仅如此，第1枚硬盘的内容还涵盖亨特与习近平及国家副主席王岐山之间，涉及45亿美元利益输送的秘密协议。

**另有习近平、王岐山海外资产清单**
值得注意的是，「路德社」的爆料与「华盛顿时报」本月15日的报导似有关联，该报导掌握亨特2017年8月一封电子邮件，指他在父亲担任美国副总统期间，曾与中国华信能源公司董事长叶简明搭上线，要求支付1000万美元顾问费等。至于第2枚硬盘的内容，是关于习近平和王岐山等中共官员的海外资产清单和私生子女名单。第3枚硬盘则是关于中共製造及散布武汉肺炎病毒的真相。

**拜登对子不法情事「高度妥协」**
报导指出，「路德社」的影片含有亨特不法情事的清晰照片，以及亨特与乌克兰布里斯玛（Burisma）天然气公司董事会成员波札斯基（Vadym Pozharski）的电子邮件，与「纽约邮报」、「华盛顿时报」稍后的爆料内容高度吻合。「路德社」暗指，拜登对儿子在中国涉及的不法情事「高度妥协」，而中共握有这些证据，因此才希望拜登拿下美国总统大选胜利，进而加以操控。
