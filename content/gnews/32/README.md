又是年下，

想家……

不敢回去，

怕万一。

万一，

是否还是原来的几率？

万一，

是否依旧事不关己？

万一，

已成为必然，比如隔离21天。

万一，

在我的故乡已不再顾名思义。

万一，

共匪突然炸毁三峡……

万一，

共匪突然引爆核电站……

万一，

共匪突然抓我强打疫苗……

万一，

进去容易出来难……

我怕万一！

可亲人需要我时，

无论多怕，我都想回去……

故乡

我一定要回去的家。

一定，

炊烟从白雪堆里袅袅升起，

一定，

孩子们欢快的笑声洒满大地，

一定，

把自由当做回家的礼物相赠，

一定，

在我们心里，故乡从未远离。
