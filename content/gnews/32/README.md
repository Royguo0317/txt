2月23日中新网：大赦国际”报告近日称，中共国将年幼的维吾尔族儿童送往孤儿院，对维吾尔族家庭强制实施“骨肉分离”政策。中共国外交部发言人华春莹23日在记者会上说，“强制实施所谓的‘骨肉分离’，这不是发生在中国的新疆，而是发生在美国。”

“学生是否住宿，完全是由学生本人和家长自愿选择。。。这些学校与中国其他地区以及世界其他国家目前开办的寄宿学校，没有本质的区别。”

“近年来大家也都知道美国政府是如何对待移民儿童的，比如，他们实施的零容忍移民政策，就导致大量的儿童被迫与父母兄弟姐妹分离。”

都知道不能和流氓讲道理，但是就怕流氓和你讲道理。美国的移民政策是法律方面的讨论，是美国社会的全民参与。而中共国对新疆人民是独裁政权的暴行。对于是否有强迫新疆的父母与孩子分开这件事，中共只需要向外界完全开放这些这些自称的“寄宿学校”就可以，而不是让新疆人在中共的镜头前“载歌载舞”。谁知道镜头后面有没有一支枪呢？
