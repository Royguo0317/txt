翟东升最近因为一段讲话如“我们在美国上面有人”， “拜登儿子的基金会是谁帮他建立的”等等一下子成了网络红人，中共长期以来欺骗老百姓，以假治国是其治国理念之一。这次在对中共内部中高层的讲话中，不小心说了真话，这个真话却让大家看到了中共渗透美国深层政府的真实情况。

翟东升是何许人？他是中国人民大学国际关系学博士，曾在欧洲和美国做过访问学者，现任中国人民大学国际关系学院副院长。据说他是中共某常委的女婿，他的主攻方向是研究货币与金融的国际政治经济学，中国对外经济关系，美国经济学。我们知道，中国人民大学是培养间谍，为中共高层提供统治人民的理论基础，如何渗透外国势力，很多中共高官就是从该校毕业的，是彻头彻尾的中共统治机器之一。

翟东升最近于12月5日在他的微博转发了他在中共的国际货币网发表的一篇文章，合作者有沃伦·寇弛，题目是《为什么世界需要一个与硬通货锚定的储备财富》，文章认为，目前用美元汇率进行交易的成本已经超过了用美元汇率进行交易所获得的好处，建议改用根据几个主要国际货币的汇率基础上采用真实的特殊划定权利。简单来说，就是美国应主动放弃美元的全球储备地位，让人民币在国际上有话语权。说到底，是要搞乱国际金融体系，可以让人民币避开美元体系的监管，那些盗国贼们的钱可以有安身之处。

另外，与翟东升共同署名的沃伦·寇弛（Warren Coats)曾服务于国际货币基金会，2003年成为开曼群岛货币权利中心主任，总统经济顾问，最近还是阿富汗中央银行与国际货币基金会相沟连的指导者，也是哈撒克斯坦政府中央银行的指导者。开曼群岛大家很熟悉，很多中共高官家人都在开曼群岛存有大量账户，这人现在又去阿富汗和哈撒克斯坦指导当地货币与国际货币相连通，是不是让中共高官的钱也可以有一部份存在那里？我们拭目以待。
