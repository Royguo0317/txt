爆料革命以来，郭文贵先生多次提出平民式的民主，世界要大翻盘；班农先生更是平民主义运动的倡导者。川普总统是美国几百年来民选出来的，唯一没有任何从政经历的个体商人。加上美国政府的多年政策，让美国及老百姓利益上的损害，民众的意愿沉默暗涌。奥巴马政府的腐败无能，拜登家族与中共的勾兑，以及被揭露出来的拜登家族以及其他政客、商人、媒体等表现出来的极左倾向，使我们对“精英政治”不得不加以思考。笔者尝试对“精英政治”的起源、发展及现状做一浅显的勾勒，对研究美式民主及对新中国联邦的未来民主发展方向，作一铺垫性的展望。

美国政治学者托马斯·戴伊，在《民主的反讽——美国精英政治是如何运作的》一书中指出，美国式民主政治的现实状况是“精英民主”，即治理美国的是精英，而不是民众。这些少数精英已经牢牢地控制住了美国政治，他们实行的是寡头政治。

咱们就先探讨一下，什么是精英主义。

简单地说，精英主义(elitism）是西方政治文化传统中贵族气质的一种表现，是对西方贵族政治传统的继承，主张精英政治，怀疑平民政治或大众政治。elitism 最早在1850年被使用于评价尼采与卡莱尔的思想是一种精英主义思潮，在19世纪末20世纪初由莫斯卡、帕累托、米歇尔斯三位形成了其理论体系，先后又经历古典精英主义、民主精英主义和现代精英主义三个发展阶段。其三个阶段也是异彩纷呈，莫衷一是。从产生的那一天起就饱受争议、非议与辩驳。它的发展的内部逻辑是从强调精英政治和统治性、精英与大众的对立性，演化为强调精英阶级的治理性和技术性，及精英与大众的融合性。

1 古典精英主义

古典精英主义就是在19世纪末20世纪初，由意大利的莫斯卡、帕累托和德国的米歇尔斯确立的理论体系，法国的勒庞和西班牙的奥尔加也可归为古典精英主义。在西方政治思想中一提到“精英主义”，必提前三位政治理论家的观点。所以对古典精英主义的研究，在整个精英主义研究中起着举足轻重的作用。

1.1 产生背景

19世纪末20世纪初，欧洲尤其西欧的经济发展模式与社会结构的变迁，使社会中精英的产生方式发生了变化，政治体制的变革也使政治精英，以与以往不同的机制被选拔出来。

这一时期的欧洲，在工业革命的推动下迸发出来的能量，令人称奇，被称为“世界的发电站”。技术、经济、文化等都有巨大的变化。机械装置成为欧洲的主要标志，机车、煤气厂、发电厂广泛普及。历史学家诺曼戴维斯指出，此时欧洲人切身感受到各种不寻常的“力量”：电流和炸药、人口增长、新型社会力量、新的商业和工业、军事带动的人力和机械、文化运动、政治势力，在各种力量的冲击下，精英产生的条件也多样化了——-经济、政治和社会诸领域，都出现了新的多样化的精英选拔机制。比如更细的社会分工，使技术性分工成为全新的精英产生方式，与传统的精英产生方式相互交织。

1.1.1 市场机制产生经济生活精英

资产阶级革命带来了西方市场经济制度的长足发展，生产资料市场、金融市场、产权市场等各类市场纷纷登场，产品增加，供给超过需求而又出现大萧条。科技发展又使市场更加细分，不同消费群涌出。激烈的市场竞争，促使一批优秀分子脱颖而出。

1.1.2 社会阶级结构变化产生新的精英阶层

市场和社会分工细化，使资本主义发展初期形成的两大阶级——资产阶级与无产阶级——的对立，此时发生了界限模糊的状况，出现了中间阶层，社会阶层结构变得更复杂。比如熟练工人或职员、技术人员、男女专业人员等大量出现，是最重大的社会变化，对家庭日常生活及对普及教育、技术训练等的需要大大提高，大部分实际权力从资本家手中转移到经理和管理人员手里，白领阶层壮大。这种变化改变了政治权力集团中的社会构成，来自中产阶级、知识分子和经理阶层的人逐渐进入了政治权力的核心。

同时，生产力的发展和工业化也使农村人口逐渐走出农村，形成大量自由劳动力，满足工业化要求，城市人口大量增加。劳工组织、工会和社会主义政党纷纷建立，为各种社会改革运动、精英的产生，提供了肥沃的土壤。

1.1.3 民主政治的扩张产生政治精英

1828～1926年，西方社会处于政治民主化浪潮的第一步。17、18世纪的资产阶级革命只是推翻了封建统治，政治民主化的进展却十分缓慢。从19世纪60年代开始的政治改革浪潮席卷欧洲，扩大了政治参与的范围，政治民主化的进程大大加快了。议会制改革、普选等加强了大众组织的政治性。政治活动渐成专业性的职业，大量的政治活动法律、法规的出现使一般人难以掌握这方面的知识，从而促使政治精英不断涌现，政党本身的不断发展和竞争，也使政治精英锻炼成长，逐步形成了民主制度下的政治精英选拔机制。

1.2 古典精英主义观点集锦

1.2.1 精英的定义

古典精英主义对精英一词的定义，一直存在争议，原因是当使用精英这词时，所指代的人群往往不尽相同。可以从两方面理解，一方面是在一般意义上定义什么是精英；一方面是在不同的社会阶段，精英指代的是哪些人，哪些阶级阶层，或者哪些职业群体、社会集团。

帕累托的定义为，在人们自己活动的领域内能力，水平较高的人就是精英。他提出打分制。这是一种纯技术标准，排除了任何道德评判。

米歇尔认为，精英就是组织中的领导者，这是技术性的分工标准，将治理与统治混淆了。但这一定义为后来的精英主义发展提供了思想源泉。

其实这些定义所形成的两条标准，即素质能力和权力地位，两方面的侧重和发挥，成为后来精英主义演变的一个重要标志。

1.2.2 精英与大众的关系

这一点在古典精英主义中是明确的，他们都明确指出精英与大众是存在一定距离的，不仅在地位上，在文化、知识修养、语言、习惯和家庭习俗等方面都存在差异。这些差异造成了统治阶级与被统治阶级的鲜明界限，有时是不可逾越的。这种区分是生硬的、绝对的，但为后来的理论发展奠定了框架。问题是，社会被分出掌握权力的精英群体和没有权力的大众，政治资源的分配出现了不平等。与此同时，统治与治理相分离，使精英阶级逐渐转为技术性治理活动。

1.2.3 对民主政治的批判

古典精英主义对民主政治的批判极为深刻，以至于引起很多误会，又让以为它是反民主思潮的。但其实它是指出了当时西方民主的弊端和虚伪性。指出绝对的平等是不存在的，政权从来没有也不会建立在大多数人的明确首肯上面。即组织处处意味着寡头统治，民主必须是有组织的，而组织总是导致寡头化和官僚化，即精英政治，民主就是精英式民主。其实古典精英主义者不是否定当时的自由民主体制，而是反对无政府主义，主张精英与民主结合，这也为后来的精英主义民主化奠定了基础。

2 民主精英主义

民主精英主义产生于20世纪10～30年代。主要代表人物是德国的马克斯·韦伯和美国的经济学界熊彼特，他们提出了民主意义上的“精英民主论”，认为民主政治的实际运作为少数精英所主导，政治活动是不同精英团体的相互竞争，争取选民的支持，承认民主的“竞赛规则”，亦忠于民主的价值。这是一个承上启下的阶段，在这一阶段，古典精英主义完成了由古典向现代的转变。

2.1 理论观点集锦

2.1.1 统治与治理分离

韦伯科层制理论的层级制、连续性、非人格化和专业化，就是将政治或行政活动进行技术性分解，并以此进行技术性分工。他不再论述关于政治的原则性问题，转而关注政治统治如何实现，如何最有效率地实现政治目标，出现了政治与行政的分离。理论上出现了美国伍德罗·威尔逊《行政学研究》一文，提出政治与行政分离，行政学应该成为一门独立的学科。美国另一名学者出版《政治与行政》，发展了行政应与政治相分离的理论。

2.1.2 民主的工具性

熊彼特对民主的定义，就是将民主视为一种挑选政治精英的制度、程序、技术和方法。提出“民主方法就是那种为作出政治决定而实行的制度安排，在这种安排中，某些人通过争取人民选票取得做决定的权力”。这一定义成为民主精英主义的代表性观点。古典民主理论中的核心问题——人民的意志——到了精英民主理论转换成了“努力赢得人民的赞同”，只强调意识形态和选举仪式的作用。

2.1.3 行为主义倾向

后来，特别是二战后，行为主义的研究兴起，使精英研究更加偏重实证，出现了行为主义的精英研究。这种理论已经不再对民主制度的基本原则进行分析，而将重点放在了具有操作性的政治过程方面。

2.1.4 精英民主与参与式民主

参与式民主提出了对精英式民主的批评，提出民主必须有广泛的参与，不仅在政治生活领域，社会生活的所有领域，都要有广泛参与。没有这样的参与，民主就很难存在。因此，一个民主政体如果要存在的话，就必须相应地存在一个参与社会，即社会所有领域的政治体系，通过参与过程得到民主化和社会化。因为缺乏参与，精英与大众的分野本身，就是将大众排斥在政治体系之外，而这与民主精神背道而驰。

3 现代精英主义

这个阶段的精英主义成为激进的精英主义，主要发展在美国，表现得“左”倾。主要代表是美国的詹姆斯·伯纳姆，查尔斯·赖特·米尔斯，丹尼尔·贝尔。

3.1 精英主义的趋势

3.1 经理阶层与权力精英

伯纳姆的观点表现出，现代精英主义更多强调“精英政治”具体是哪些人在统治，哪些人实际掌握权力。现代社会中的新的领导阶级——经理、工程师、工艺学家、工业化学家、人事管理以及许多方面的专家、行政人员等，使现代国家的职能越来越复杂，仅有政治家或“人民代表”来统治的简单制度就不够了。他预言，不仅议会政府要转到行政管理层面，而且民主政治本身将最终消失，由资本社会转向管理者社会，管理者将在事实上取得对社会的统治，成为社会的统治阶级。

米尔斯则认为，“权力精英”（power elite)将成为“统治阶级”“政治阶级”。在美国社会、经济、政治、军事领域中的领导者、掌权者，构成了美国的权力精英。这些人集权力、财富、声望于一身，决定着美国政治、经济的走向。三个领域的决策者们控制着美国主要的“历史性决策”。但权力精英会通过各种手段，欺骗大众，并使公民社会保持沉默，权力精英会为了本身利益无须对公众负责，必然导致腐败。权力精英的活动会压制普通公民参与的愿望与活动，大众社会的到来，并没有改变权力精英掌握政治权力的基本结构。

3.2 技术性专家

精英阶层从事的就是技术性管理工作，即知识阶层，从事着后工业社会的治理活动。丹尼尔指出，后工业时代，关键问题不仅是新的权力基础，从财产标准或政治标准转移到知识方面，而且知识本身的性质也发生了变化，比如计算机的运用，使经济系统模型、社会行为模型和决策模型，都有可能进行大规模的“有控制的实验”，反过来，这些实验又使人可以选择不同的未来，及对我们生存的影响。技术精英的产生对权力精英垄断格局的影响，使精英的产生更加多元化，权力分散向下的力量更加严重。

3.2.3 技术层面的深入

自韦伯提出科层制，精英阶层就从政治过程中的统治，逐步后退到治理，职业政治家，官僚和职业经理人是精英阶层的主要组成部分。当精英们的活动主要集中在技术性领域时，新的精英群体会在更深、更大层面寻找精英治理的环境。比如技术精英、知识精英等，知识精英与职业政治家，官僚的结合、形成了新型的精英治理模式。

最后总结，精英主义的局限

精英主义没有否定大众式民主，但也没有解决大众民主适用的范围。表面上大众式民主不断扩张，但各技术环节仍由精英控制。大众式民主下仍然是精英式民主在活动，只是由直接变成间接，由显性变为隐形，由表面形势变为实际内容。

此次总统大选与揭露拜登家族的腐败，多数大媒体与自媒体进行各种拦截、控制，删减真相的信息传播。民主化的浪潮势不可挡，信息化时代的来临，势必引起网络式组织的涌现，新的自媒体兴起。比如路德社、GTV、Gnews、Gab等。

另外，精英主义缺少价值判断。精英主义对政治活动缺少基本的价值判断，一个政府的好坏，不在于是否是精英治理。简单地说，精英主义在推崇精英治理时，对精英人员好与坏不做道德评判。拜登当政几十年，自己腐败，儿子淫秽，家族贪腐，这些都从未成为他升迁的障碍，2020年还被推举为民主党总统候选人，这岂不是精英政治的一大笑话。

精英主义也过于强调精英集团的作用，忽略大众特别是中产阶级在社会发展中的作用。按照精英主义对社会成员的划分，精英与大众属于两大社会集团。精英群体的作用如得不到大众响应，其行动也十分有限，尤其在社会变革时期。此次总统大选就是一个十分鲜明的例子。川普总统从上次开始到此次竞选，都是走的群众路线，各州奔走，与选民同呼，“使美国更伟大”。反观拜登，当儿子的淫秽录像、家族贪腐的录像充斥媒体时，躲在地下室不见媒体，不回答大众的责问，只等着政党为自己安排一系列的阴谋，来蒙蔽老百姓视线，蒙混过关，绕弯赢取大选。

此刻的美国大众，已经醒悟。这种精英式的政治把戏已经结束。班农、文贵、川普等带领的这一场平民式的民主运动，将带领美国民主走向新的未来，为新中国联邦及世界的民主，带领前进的方向。
