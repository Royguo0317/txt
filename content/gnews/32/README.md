《路德时评》2021年4月21日早间节目主要内容：

一、日本即将加入五眼联盟

大纪元4月21日报道[\[1\]](https://www.epochtimes.com/gb/21/4/21/n12894573.htm)：日本驻澳洲大使山上信吾近日表示，日本有望在短期内加入五眼联盟（Five Eyes）这个情报共享组织。澳洲国防安全专家表示，日本将是五眼联盟吸纳新成员的最佳选择。五眼联盟是由五个英语国家组成的情报共享联盟，成员国包括澳洲、美国、新西兰、英国和加拿大。
![]()![](https://gnews.org/wp-content/uploads/2021/04/五眼联盟.jpg)[图片地址](https://i.epochtimes.com/assets/uploads/2021/03/GettyImages-483143735-600x400.jpg)
路德社评论：

1、五眼联盟是由五个英语国家所组成的情报联盟，在英美协定下组成的国际情报分享团体，包括澳大利亚、加拿大、新西兰、英国和美国。五眼联盟的历史最早可以追溯到第二次世界大战，同盟国所发布的大西洋宪章。冷战期间，ECHELON监视系统被五眼联盟所开发，用于监视苏联及其东欧盟友。[\[2\]](https://zh.wikipedia.org/wiki/%E4%BA%94%E7%9C%BC%E8%81%AF%E7%9B%9F)

2、五眼联盟是一个庞大的秘密网络，斯诺登曾爆料五眼联盟是人类历史上最高级别的超越国界的情报机构，这就相当于北约组织中的情报部门。日本以其七十年遵守契约的优异表现被五眼联盟认可，日本加入五眼联盟后将加强西方国家处理汉语情报的能力，也进一步制衡中共国。

3、情报系统绝对超越各个国家的法律，但是五眼联盟彼此信守承诺，这几十年并没有利用情报作恶，日本也是如此。未来的新中国如果要融入西方社会，一定要有契约精神，否则将再次被世界所淘汰。

4、日本之所以即将加入五眼联盟也离不开习总加速师的“帮助”，五眼联盟原属于英系国家，他们对于汉语情报信息的处理和分析能力远落后于日本，除了语言，日本还非常了解中共国的思维方式，对于中共官话、套话和秘密对话只有日本情报部门可以提炼分析，所以，日本加入五眼联盟最大的目标就是分析和处理中共国情报。

5、日本加入五眼联盟后将进入快车道，最重要的目的就是灭共，日本早已经做好和中共国脱钩的准备，安倍下台就是在与中共做内部脱钩，好让菅义伟上台后可以推动灭共，日本很有可能早在几年前就依据情报得知中共国将成为全世界的敌人，并提前为灭共做准备。

二、中共外交部开批谭德塞

中共外交部回应谭德塞称新冠溯源报告评估不充分，汪文斌强调说，世卫组织于3月30日发布了中国—世卫组织联合专家组溯源研究报告，明确了病毒由实验室泄漏极不可能的结论，不存在所谓的获取原始数据困难。各方都应当尊重科学，尊重科学家的意见和结论。在这方面世卫组织尤其应该发挥表率作用。[\[3\]](https://finance.sina.com.cn/tech/2021-04-21/doc-ikmxzfmk8182005.shtml)
![]()![](https://gnews.org/wp-content/uploads/2021/04/谭德塞-1.jpg)[谭德塞](https://static.dw.com/image/53496385_401.jpg)
路德社评论：

1、谭德塞两星期前说“病毒由实验室泄露极不可能，这一结论的评估还不够充分，需进一步研究，并准备派相关领域专家赴华。”中共经过两星期的努力仍旧无法搞定谭德塞，所以中共外交部才站出来发言批评谭德塞的言论，相信不用多久，中共将开始批评科学界。

2、中共原计划等WHO发布中共病毒报告后能为中共彻底洗白，但是谭德塞在报告发布后立即反水，这打乱了中共原先的计划。谭德塞反水意味着世卫组织接下来将代表全世界向中共发表声明，要求对中共病毒溯源展开进一步调查，到时中共国所有媒体都会瞄准WHO开炮。无论中共耍什么样的伎俩，在绝对实力面前都无济于事。

3、中共自以为将中国老百姓训练成奴隶和炮灰就是强大的象征，这是极其愚昧的想法，真正的实0力来源于每位拥有自由意志的个体，这才有创新能力，才能发展出经得起验证的科学，中共假骗偷的体制绝对不可能有科技创新。

三、中共军事专家呼吁警惕“应对中共国危险委员会”

中共军事专家杨承军4月20日撰文称[\[4\]](//mil.ifeng.com/c/85b2L1JF1aF)，要高度关注并警惕美国“应对中共国危险委员会”。美国自2019年3月25日成立“应对中共国危险委员会”以来，对于这个具有重要咨询地位和辅助决策的机构，国内外媒体报道极少，对其关注度也远远不够。对此应引起我们的高度关注和警惕，特别是对它的使命任务、人员组成和运作方式，决不可等闲视之。

该委员会的成员组成，都是重量级人物和极端保守派、对华强硬派，包括前中央情报局局长伍尔西、前教育部长贝内特、前众议院议长金里奇、前白宫首席策略师班农、对华援助协会主席傅希秋等。这次的委员会有40多名前政府官员、国会议员、智库成员组成。
![]()![](https://gnews.org/wp-content/uploads/2021/04/当前危机委员会.jpg)[图片地址](https://x0.ifengimg.com/ucms/2021_17/F13382BEA777E2B73E470AC1063F0B73DC3EDBE_size31_w729_h452.jpg)
路德社评论：

1、杨承军绝对是爆料革命的忠实粉丝，文贵先生在2019年就提及并参与筹建“应对中共国危险委员会”，杨承军这篇文章就透露出中共已彻底失去了理论自信，现今的人民币就跟前苏联时期的卢布一样大规模超发，最终一定会被美国做空而疯狂贬值。

2、杨承军教授也意识到反中共不是某些政党所为，而是历史潮流，他承认120之后进入两党联合灭共时代，无论这些军事专家如何叫唤，也不能改变中共必亡的结局。
