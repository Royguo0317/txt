新闻一

国防时报-国防新闻网官方微博3月22日消息[1]，当日，美空军1架RC-135U电子侦察机（绰号“战斗派遣”），经由巴士海峡进入南海，对中国华南沿海地区开展侦察，期间一度抵近至领海基线外25.33海里处，后掉头折返，刷新了公开已知的美对华抵近侦察最近距离。通常，美军对华侦察都保持在海岸线外约50-70海里左右的位置，25.33海里可谓一反常态，基本已经逼近到了毗连区的边缘（24海里），可见美军的侦察强度有了进一步的提升，这也必将导致潜在军事风险的走高。

2、RC-135U电子侦察机之所以要如此近距离地靠经中共国领空，就是想逼中共国打开雷达系统并记录这些雷达系统的参数，再进行情报收集。路德社此前分析说，从日本到台湾整条防线已经被美日台联军彻底封锁，从台湾到巴士海峡由美军亲自管理，目前还没有看到中共采取任何的反制行动，这次也展示了中共国真正的国防实力。

二、美英欧盟宣布联合制裁中共国官员

美国之音 3月23 日报道[3]：美国、欧洲联盟、英国和加拿大对几名中国官员实施制裁，原因是他们在中国的新疆地区践踏维吾尔穆斯林少数民族的人权。美国国务卿与英国和加拿大外长以及澳大利亚和新西兰外长同日还分别发表联合声明，谴责中国政府在新疆践踏人权。美国国务卿布林肯说，美国与盟友团结在一起而采取了制裁措施。欧盟和英国也制裁了这两名官员以及其他两名官员—中共新疆党委常委王明山和前新疆党委副书记兼政法委书记朱海仑。在有关制裁的第一波消息传出后，中国外交部立即做出回应，谴责这些制裁是“基于谎言和虚假信息”。中国紧接着宣布对10名个人和四个实体实施制裁，称他们“恶意传播谎言和虚假信息”。被中国制裁的包括欧洲议会的五名议员。

这是1989年北京暴力镇压天安门广场民主抗议运动以来欧盟首次对中国实施经济制裁。英国和欧盟还制裁了新疆生产建设兵团公安局。所有27个欧盟成员国政府都同意实施制裁。
![]()![](https://gnews.org/wp-content/uploads/2021/03/xinjiang-2.jpg)[图片地址](https://gdb.voanews.com/cdcc3a08-82c7-43eb-86f0-ef6af47f58e9_w650_r1_s.jpg)
路德社评论：

1、这是西方文明国家第一次联合起来对中共国在新疆犯下的种族灭绝罪和反人类罪实施制裁，这说明西方已经认定中共此项罪行，已经不可能有任何回转的余地。这全面拉开了西方世界制裁中共官员的序幕，此次制裁行动的关键不是那些中共国官员，而是英美欧盟的全面联合行动。

2、美国国务院正式制裁文件中明确写道：中共继续在新疆犯下种族灭绝罪和反人类罪。这是拜登政府上台后首次以书面形式明确中共继续犯下此项罪行，已经成为一套法律。更重要的是，国务院文件中说加拿大、英国还有欧盟将一起行动，这说明这些国家都已经认定中共的种族灭绝罪行。同时，美国第二次呼吁中共国释放被关押的新疆少数民族人士，如果中共仍然无动于衷，欧美国家将会继续加码。

3、中共立即采取反制措施，宣布对欧盟和英国相关人员及实体实施制裁，中共此举就是明确表态不会释放任何新疆被关押的少数民族人士，并且还不承认这件事。按照习总加速师的风格，西方国家很快将继续加码制裁中共官员。

4、被制裁像王君正、陈明国这样的中共官员都是习派势力刚提拔起来的，这些人的资产和私生子女早都已经转到海外，这些官员很清楚欧美制裁令对他们的打击力度。这将在党内引起寒蝉效应，有越来越多的资产流向海外，比特币还将继续疯涨。

5、中共的一系列战狼外交已使得反共灭共成为美国的政治正确，随着欧洲国家跟进制裁中共，反共灭共也将成为欧盟国家的政治正确，当反共灭共的政治正确大势形成之后，任何一位西方政客被中共制裁都将成为他们的荣誉，成为他们未来的政治资本。中共和西方你来我往的制裁与反制裁行动将进一步推动中共国与西方世界脱钩，中共国白热化的内部斗争逼着习放弃外部的国际形象，专注于解决内部矛盾来保证其执政的稳定性。
