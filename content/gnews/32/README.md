《自然新闻》泽连科规约的发现者弗拉基米尔·泽连科博士今天称儿童疫苗授权是”强制性的人体实验”，他呼吁对此类政策负责的人因”危害人类罪”受到审判。

泽连科告诉美国前线医生（AFLDS）：”根据疾病预防控制中心的数据，18岁或18岁以下的健康儿童在没有任何治疗的情况下，从中共病毒中恢复的比率为99.998%。 “在医疗上来说不需要任何疫苗。特别是一种实验性和未经批准的mRNA注射，已证明有许多危险的副作用。

他接着说：任何强迫或命令儿童接受这种实验性疫苗注射的政府或个人都直接违反了日内瓦公约禁止强迫人类实验的规定。这些罪犯是最高等级的罪犯，必须根据危害人类罪被绳之以法。

泽连科博士指责以色列国政府”强迫本国公民进行人体实验”，敦促停止这一运动。他说：”以色列政府用胁迫来强迫本国公民进行人体实验。 “绿色护照是一种歧视手段，对无辜的人们施加了不合情理的心理压力。

知情同意被无视，医疗必要性未予考虑。因此，年轻健康的人和那些已经有抗体的人仍然被迫进行他们不需要的实验性医疗干预。

他继续说：”以色列政府——停止向无辜和未经同意的公民的身体发射微型导弹。谨慎行事，停止人体实验，并在使用新的和未经批准的技术之前收集更多的安全性和有效性数据”。

泽连科博士因其在应对中共冠状病毒大流行方面所起的作用而被列入诺贝尔和平奖提名的一组医生中。

他在使用硫酸羟氯喹和锌治疗中共病毒患者方面取得了举世瞩目的成就，发现使用这两种物质的死亡率下降了8倍。他说在头5天内用硫酸羟氯喹和锌治疗可使死亡率降低85%。
