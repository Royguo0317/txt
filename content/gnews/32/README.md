**1. 香港国安处处长蔡展鹏被揭光顾无牌按摩店**

[!\[\]()!\[\](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/820aaeaa-2c14-435a-b641-3ff3357efcdc.jpg?asset_id=b2facd66-3319-4f5c-a67d-0d08c50584aa&amp;img_etag=%2281f97868f2f2defd1a152a7439bb5bb6%22&amp;size=2560)](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/820aaeaa-2c14-435a-b641-3ff3357efcdc.jpg?asset_id=b2facd66-3319-4f5c-a67d-0d08c50584aa&amp;img_etag=%2281f97868f2f2defd1a152a7439bb5bb6%22&amp;size=1024)

香港国安处处长蔡展鹏被视为香港下一届警务局长的热门人选，最近被爆，光顾无牌按摩店，现已被下令休假，接受调查。蔡展鹏自1995年加入香港警队，任职督查，主要负责保安情报和行动工作。其于2009至2020年连升五级，由督查升至高级助理处长，2009年升为警司，2014年晋升为高级警司，2017年6月晋升为总警司，出任保安部副主管，2019年1月擢升为助理处长，掌管保安部，2020年7月擢升为警务处高级助理处长，掌管国家安全处，2020年获颁授香港警察卓越奖章。2021年5月12日，香港多个传媒报道，身为国安处处长的蔡展鹏在一个月前光顾无牌按摩院。由于蔡展鹏的职位敏感，而且事件有损警队形象，因此警方高层得悉事件后，警方一直没有披露事件，在事发一个月后才被传媒曝光。《南华早报》在5月12日报道蔡展鹏因为早前被发现光顾无牌按摩院，现正休假一个月及接受调查。警务处处长邓炳强在傍晚开记者会时表示，已经指派有组织罪案及三合会调查科调查此事。这次事件是否是中共利用完蔡展鹏之后的卸磨杀驴，还是香港内部的派系斗争，我们拭目以待。

**2. 香港大纪元记者梁珍遭暴力袭击**

[!\[\]()!\[\](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/16451c75-ba73-4bd1-a3f9-acea78f25743.jpg?asset_id=59fc08e1-3bc8-42a9-b0d0-ea11030eb362&amp;img_etag=%2263104560bc4bb3fc0095172438894294%22&amp;size=2560)](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/16451c75-ba73-4bd1-a3f9-acea78f25743.jpg?asset_id=59fc08e1-3bc8-42a9-b0d0-ea11030eb362&amp;img_etag=%2263104560bc4bb3fc0095172438894294%22&amp;size=1024)

5月11日中午，香港大纪元记者梁珍在住所楼下遭到凶徒持棍棒袭击十多下。凶徒袭击梁珍约一分钟，后立即驾车逃离现场。梁珍已经就事件报警，并到伊丽莎白医院验伤，腿部被凶手打伤并出现多处瘀伤。 目击证人李先生说，当时他吃过饭，在路边走过时，见到斜对面有个肥硕的男子手持球棍从一辆车出来，然后听到有位女士呼喊救命， 并有一辆面包车挡住了外人视线。李先生走过去看时，见到一名男子手持球棍掉转头返回车上，急忙开车逃走，这是一个车牌号为TV3851的奔驰车。李先生怀疑这是一个团伙蓄谋的暴力攻击事件，并表示愿意出来作证。梁珍表示，5月8日在同一地点等车时，有一名男子突然向她跑来，该男子在大热天用外套包裹全身，突然从衣服里掉出了一根黑色棍子。那名男子好像感觉被发现了，行事败露， 有点不知所措地捡起了棍子。4 月24日，有另一名可疑男子到梁珍家敲门滋扰，梁珍说她觉得中共是在恐吓她，她已经被他们盯上了而且知道她的住址。梁珍本人身兼香港法轮大法佛学会会长，她指出从4月2日至9日，香港法轮功真相点连续遭中共暴徒袭击，承印香港大纪元报纸的大纪元新时代印刷厂4月12日清晨遭4名暴徒持刀、锤等凶器砸毁印刷机，并抢走电脑，致使香港《大纪元》被迫停刊数日。香港现在的环境对媒体工作者充满了危险，已经从白色恐怖变成真实的人身攻击。

**3. 粤港公仆互掛， 爱国者治港被要求宣誓效忠基本法， 多名议员辞职**

[![](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/f24d1150-436b-40e3-974d-febaf535a2c0?asset_id=17033698-08a4-423b-8325-fbde5edfa3d8&amp;img_etag=%22e51d36a325e7ba18f2ed8ed5366cefcc%22&amp;size=1024)](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/f24d1150-436b-40e3-974d-febaf535a2c0?asset_id=17033698-08a4-423b-8325-fbde5edfa3d8&amp;img_etag=%22e51d36a325e7ba18f2ed8ed5366cefcc%22&amp;size=1024)

2021年5月10日，香港特别行政区行政长官林郑月娥接受中共央视【新闻1+1】主持人白岩松访问时表示，会将工作重点放在粤港澳大湾区，早前已经和北京、上海等地政府签订公务员互换挂职交流协议，也将与深圳市，广东省政府等签订相关类似协议，希望香港公务员借此机会深入了解大湾区内地城市的工作和运作方式，同时欢迎中国当地公务员到香港交流。这样的安排是有深层的目的，首先影响的是尚未入职，将来会入职的准公务员。要成为香港公务员，必须要宣誓效忠中共，就这样还是不足以取信于中共，还需要到中共内地政府机关中进行考核，经内地政府认证，才能正式取得香港公务员资格。以后香港新进的公务员要经过层层验证以保证忠诚，实现爱国者治港，建立一支中共直接指挥的公务员队伍，确保中共统治下的一国一制。 5月12日，香港立法会以40票赞成，1票反对通过《公职条例草案》，要求区议员宣誓拥护《基本法》。 然而有许多人对宣誓不满，上年度有1800 多名公务员离职。 早前中共和香港当局修改选举机制，把区议会视作“非政府组织”，立法会和选特首的选举委员会再没有位置给大部分由人民直选的区议员。此次要求区议员宣誓被外界视为当权者进一步收窄民主派参政的重要举措。另外多名卷入示威案件或涉及民主派初选的《国安法》案件的区议员亦相继宣布辞职，个别人士因被判囚而被取消议员资格。

**4. 首名外佣打疫苗后离世。塞舌尔疫情反弹国药疫苗有效性受质疑**

[!\[\]()!\[\](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/8d666a5c-25af-44a2-88b9-55eb200a64d1.png?asset_id=14f8ac46-1ff9-4e10-bf9c-31aa89c4c361&amp;img_etag=%22a9ff7e73f3b5cae7deee199df3f21c71%22&amp;size=1024)](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/8d666a5c-25af-44a2-88b9-55eb200a64d1.png?asset_id=14f8ac46-1ff9-4e10-bf9c-31aa89c4c361&amp;img_etag=%22a9ff7e73f3b5cae7deee199df3f21c71%22&amp;size=1024)

香港卫生署近日公布一名44岁外佣在本港接种復必泰18日后离世，成为香港首宗外佣接种疫苗后死亡个案。该名外佣于4月21日在社区疫苗接种中心注射復必泰后5月9日离世，经解剖后证实死因是主动脉夹层动脉瘤破裂，而卫生署表示目前无临床证据显示事件由疫苗引起。

[!\[\]()!\[\](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/4704a550-388f-406c-918e-fc3cd496fdbf.jpg?asset_id=f73a4ad9-e452-48ad-a9d4-149127fc653a&amp;img_etag=%22f422730b3ad439f3064aad77219d4c8b%22&amp;size=1024)](https://spark.adobe.com/page/K19NyNqkdb7Bo/images/4704a550-388f-406c-918e-fc3cd496fdbf.jpg?asset_id=f73a4ad9-e452-48ad-a9d4-149127fc653a&amp;img_etag=%22f422730b3ad439f3064aad77219d4c8b%22&amp;size=1024)

塞舌尔是一个人口刚刚超过10万的印度洋小岛国，位于马达加斯加东北。现在，首都维多利亚正在与中共病毒感染的激增作斗争，不得不重新采取限制行动的措施。 塞舌尔接种了两剂疫苗的人群中，57%打的是国药疫苗，43%打的是阿斯利康（AstraZeneca)疫苗。而塞舌尔的初步报告显示国药疫苗的有效率为50%而不是国药集团宣称的78.1%。 现在，世界上中共病毒疫苗接种率最高的国家塞舌尔出现了感染病例激增，尽管大部分人口已经接种了国药疫苗。

在塞舌尔一家电话服务中心工作的玛丽•内芝（Marie Neige)已在今年3月打了中共国药集团的疫苗，正在期待着几周后的疫苗将提供的充足保护。但在周日，玛丽的中共病毒检测呈阳性，她表示已失去嗅觉和味觉，而且喉咙有点刺痛。玛丽认为疫苗本应该保护她和接种了疫苗的人们，不只是保护他们不被病毒感染而是保护他们不受这些症状的困扰。爆料革命路德社和闫丽梦博士早在2020年1月就说过病毒来自中共武汉实验室，并非来自自然，所谓的疫苗根本就防御不了变异的病毒。 只有灭共才是唯一的出路！
