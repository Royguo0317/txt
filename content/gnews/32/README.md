路德社评论如下：

1、该救济法案多大五千多页，众多议员在极短的时间内就进行投票，基本没有阅读全部内容，民主党就想通过这种方式赢得川普总统批示；

2、虽然这份法案为新冠救济法案，但其中暗藏大量与新冠救济不相关的条款，各利益集团就希望川普总统签署该法案后，依据这些条款进行牟利；

3、该份文件第5893页暗藏一句重要条款『让总统使用防暴法变得无效』，这就是政治家的邪恶，利用美国制度的漏洞牟取利益；

4、国会接下来可以否决川普总统对该法案的“否决”，然后川普总统还可以再次否决，最后进入最高法进行裁决；

5、川普总统在废除《国防预算法案》前就要求加入『废除230法案』，但最终没有成功，因此《国防预算法案》被川普总统废除；

6、川普总统此举就是要废除主流媒体的话语权，推特此前已明确表态，一旦川普总统卸任，推特将立即封锁他的账号，川普总统所有行动的核心就是为争取言论自由，一定要废除『230法案』；

7、如果最终进入最高法裁决，川普总统律师团队可以从该法案中找到违宪条款促成最高法废除该法案，即使该法案最终通过，川普总统依然能够拖延该法案的执行；

8、美国的伟大就是三权分立的体系的伟大，可能某一时刻三权同时腐败，但不可能三权永远同时腐败，而中共国一党专政，可能某一刻诞生明君，但这体制必定催生昏君。
