#### 【G童话】

从前有一个小男孩，不小心打碎了妈妈心爱的水晶杯。

妈妈很生气，这套水晶杯是妈妈从法国买回来的，一共有八个，外面镶嵌着金丝，雕刻着花纹，非常华贵美丽。

妈妈非常珍爱这套水晶杯，只有过圣诞节的时候才舍得拿出来用，平时根本不让小男孩碰。

现在，水晶杯被小男孩打碎了一只。

小男孩很难过，也很后悔。他很喜欢水晶杯，也不想让妈妈生气难过。

小男孩来到湖边，那里住着一位女巫。小男孩恳求女巫把他变成水晶杯，这样妈妈就不会再为碎掉的水晶杯难过了。

女巫问：万一妈妈想你了怎么办？

小男孩想了想，说：那就把我的影子印在水晶杯上吧。

小男孩于是变成了一只水晶杯，妈妈的八只水晶杯终于凑齐了……

#### 【作者解读】

这是本人在2021年3月写的一篇童话，写完之后，心里有种淡淡的忧伤。

小男孩代表的是人类与生俱来的纯洁的灵魂，水晶杯代表欲望，是后天形成的。当灵魂试图打碎欲望时，出于人性的贪婪与执着，人类最终选择把灵魂献祭给了欲望……

人类文明的发展，其实已偏离了上帝最初的设想，在物欲的泥沼中越陷越深，忽略了人作为一个上帝杰作，应当经历与体验的生命的奇妙与美好。

我们所追求的物质上的享受，对权力欲望的满足，最后都会上升到精神方面的愉悦，体验到一种被他人尊重、认可的愉悦感。然而，生命中的很多精神方面的极致的愉悦，并不需要金钱与权力，比如：亲情、爱情、友情；朋友之间的信任、真诚、理解和支持；内心的安宁、祥和、满足等等，而这，恰恰被很多人忽略了。

在物质世界里，人们伤害亲情、友情、爱情，去攫取金钱；出卖朋友，以满足权力；牺牲有限的生命，去追求无限的财富；忙碌、仇恨、行尸走肉……

当我们忙碌到没有时间陪一陪孩子，听听孩子们清脆的笑声，亲吻爱人，留意花园里的玫瑰，湛蓝的天空中飞鸟的痕迹，有规律的阅读丰富自己的内心，听音乐……就算我们拥有权力与财富，又有什么意义呢？

每个人的心里都住着一个纯洁的小男孩，代表着上帝的美好祝福。身体只是灵魂的容器，削足适履的蠢事，相信不会有人去干，同样，我们也不要为了满足“容器”的索求，去伤害自己的“灵魂”。不要把灵魂献祭给欲望，这是本文想要表达的思想。
