路透社，北京5月11日 – 随着出生人数下降，中国的人口增长是1950年代以来最慢的，因为出生人数下降，这使人们怀疑北京是否有能力为经济提供动力，因为它陷于同样困扰着像日本这样的发达国家的老龄化趋势。

自从1970年代末实行独生子女政策以来，中共国人口增长开始退潮，周二发布的2020年进行的十年一次的人口普查的结果显示十年来中共国大陆的人口增长了5.38％，达到14.1亿。 这是自1953年开始进行现代人口普查以来增长率最低的一次。

数据显示，2020年每个妇女的生育率为1.3个孩子，与日本和意大利这样的老龄化社会相当。对于中共国的决策者来说，令人震惊的警报是，世界第二大经济体可能在没有首先积累起G7国家的家庭财富的情况下已经进入不可逆转的人口下降之中。

这个数字意味着中共国还差一点实现其在2016年设定的目标，即以生育率约为1.8，到2020年将其人口增加到约14.2亿。2016年，中共国以二胎为限取代了其最初旨在遏制当时的人口爆炸的一胎政策。

人口结构的急剧恶化将给中共施加压力，使其加大对夫妇生育更多孩子的激励措施-到目前为止，这些激励措施远未抵消职业选择压力和生活成本挑战的影响，夫妇们说这些压力阻止了他们生更多孩子养育大家庭的想法。

分析人士说，鉴于人口的严重老龄化已经出现，这次人口普查数据也将给决策者们提供弹药，他们主张尽快提高该国的退休年龄。

国家统计局局长宁吉哲在发布人口普查结果后说：“从近年来的人口发展趋势来看，未来人口增速将继续放缓。”

宁说：“中国的人口今后会达到峰值，但具体时间还不确定。据估计，未来一段时间中国的人口总量会保持在14亿人以上。”

2030年达到峰值？

近几个月来，中共官方媒体对人口前景的看法越来越沮丧，称未来几年人口可能开始减少。 联合国预测，生活在中共国大陆的人数将在2030年达到顶峰，然后开始下降。

但在4月下旬，《金融时报》援引身份不明的知情人士称，该国人口实际上在2020年比前一年下降了。

去年2月发布的较小规模的官方调查显示，2020年的数字实际上仅略高于2019年的14.005亿人。

数据中的一个亮点是人口中年轻人的比例出乎意料地增加了-2020年，人口的17.95％为14岁及以下，而2010年为16.6％。

从2016年到2019年，除2016年外，年出生率大多下降。宁说到，中国的去年出生人口为1200万，大大低于2019年的1465万。

总部位于北京的智囊机构“中国与全球化中心”的人口学专家黄文正说：“并没有使用已公布的人口普查数据来确定中国正面临着出生率的大幅下降。”

该专家说，即便中国的人口在2020年没有下降，它也将在2021年或2022年，或很快出现。

**“生活成本太高了”**

城市夫妇，尤其是1990年以后出生的夫妇，对家庭的独立性和职业的重视程度高于抚养孩子，尽管他们的父母对他们施压生育子女。

中共国大城市人口众多，是新生儿的巨大来源，但生活成本的不断飙升也阻止了夫妻生孩子的愿望。

据一家官方智囊机构2005年的一份报告，在中共国，一个普通家庭抚养一个孩子的成本为人民币49万元（合74,838美元）。当地媒体报道说，到2020年抚养费用已高达199万元，是2005年的四倍。

去年4月结婚的26岁的上海保险专业人士张安妮（Annie Zhang）说：“有一个孩子对我这个年龄的女性而言，是对职业发展的毁灭性打击，”

“其次，抚养一个孩子的成本高的离谱（在上海）。” 她在2020年人口普查结果发布之前评论说。 “你生完孩子立即告别自由。”

不断减少的可工作成年人口数量，不仅给中共国的适龄工作人口增加了压力，也给社会生产力造成了压力，这也将考验中共国为老龄化社会偿付和照顾的能力。

普查数据显示，2020年年龄在65岁及以上的公民占总人口的13.5％，远远高于2010年的8.87％。
