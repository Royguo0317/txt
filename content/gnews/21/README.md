## 袁隆平逝世预示着共产党的粮食将尽

袁隆平于北京时间2021年5月22日逝世于湖南湘雅医院，在墙内媒体上可谓备极哀荣，各路媒体纷纷表示哀悼，待遇和2015年死亡的乔石万里相当，比之张万年慰健行等人在媒体上的待遇犹高一筹。网络上对他的争议不绝于耳，我对这些专业不擅长，所以无可置喙。但是从他能在2018年公开说如果外国不卖粮食，中国人肯定要饿肚子，说明他还是有一点良心，所以我对他的离去以逝世字眼，以表示尊重。而且从他在2019年接受勋章的时候，面对习近平仍然显得非常不卑不亢，说明这个人是有底气的，要么有本事，要么有与他明面上湖南政协副主席这个位置不匹配的实力。

那么他的逝世为什么意味着共产党的粮食将尽呢？众所周知，中国在共产党的统治下，从49年起，中国开始走向大规模饥荒贫困，期间还有大规模的饿殍遍地，易子相食的情况发生。直至20世纪九十年代末世界经济发展欣欣向荣之际，中国人民仍然处于贫困饥荒的境地。在这期间，只要是农村出身的人该知道，只有年节才有沾荤腥补充营养的时候，而且从乞讨角度来看，在21世纪之前，乞讨的人大多数是求一碗饭，我出身农村，即使在99年，仍然能看到来自安徽等省份的可怜人，穿州跨省，单人或者结群，或背着棉被托着破碗，或结群以表演马戏的方式乞讨。所求唯有果腹米饭足以，若是米饭加一条腌炸鱼干，乞讨者则痛哭流涕感谢。我祖母目不识丁，面貌严肃但心地善良，看到表演马戏的少年被折断手臂后，托碗在各家各户乞讨，祖母在此少年碗里的米饭压实填满，还加了正在吃的几条腌炸鱼干，少年痛哭流涕，鱼干又咸又涩并不好吃，但在那时却是荤腥的替代品。而自共产党加入世贸以后，则以讨钱者为多，并且除了收鸡毛收头发的义乌人外，很少有外省人再去农村了，这样说明从这时期确实达到了温饱社会，胡温也在此之后取消农业税是证明。姑且不论几十年前共产党造的孽，单说加入世贸后达到温饱社会是共产党领导的功劳吗？我相信有脑子的人都清楚怎么回事。而且据公开资料查询得知，袁隆平的杂交水稻技术早在这之前就已经出现，但似乎并未解决吃饭问题，因为在加入世贸以前，中国农村青壮劳力务农人员比例相当高，而且这一时期房地产并未出现，大规模房地产占领耕地的情况还没发生，否则袁隆平的杂交水稻技术真这么厉害，没道理在加入世贸之前会出现农村人风调雨顺的情况下仍然一年难见荤腥的情况。

共产党自1989年北京屠城后，唯一的一点合法性就此荡然无存。所有政权的合法性要么来自传承，要么来自大众选举。传承方面如中国古代的五德说，今朝会不同程度承认前朝，阐述自己和前朝同出一源，表示我朝是继承前朝的。西方的更别提，都说自己来自希腊罗马，例如历史上那么多自称罗马帝国的。大众选举方面就是历史非常短暂的国家,如推翻军政府后的韩国,新加坡等等。而共产党这两者都不沾，靠阴谋上位，靠武力镇压，合法性相对前两者本来就少，再加上和平时代种种杀戮，战争，饥荒，靠洗脑靠谎言维持一时，在北京屠城前几乎都能被它自圆其说，把被它造成的灾害怪在苏联，西方，天灾，即使文化大革命快瞒不住了，也推在四人帮头上。但是1989年，和平示威的市民，学生没有任何犯罪行为，连小偷都罢偷表示不给运动抹黑。而共产党这种情况下发动屠城，就再也难以自圆其说。所以当马屁精何新给中共元老王震拍马屁，说他在这场运动中指挥若定，王震冷冷说道，死人终究是不好的。是王震这个老匹夫心地仁慈见不得死人？其实他们就是害怕合法性消失，到了那个地步的人都明白没有合法性连山寨里的山大王都不如，没有人心凝聚，灭亡只在片刻之间。这也是共产党能在教科书里说镇反杀人杀得好，发动朝鲜战争干得好，三年大饥荒是苏联干的，是天灾。即使是文化大革命，他们都能找到角度来洗一下。但就是八九六四，是不允许大家讨论，因为八九六四是真正伤害到他们的统治根基。

这种情况下共产党就要慢慢构筑新的合法性，合法性怎么来？得造神，表示共产党是天命所归。从这以后，其实不会选政治人物造神，否则又给他凌驾其他派系之上的机会。而是找各行各业的人造神，例如做好事不留名只会写日记拍照片的雷锋，因为研究火箭被李双江戴绿帽的钱学森等等，而袁隆平也在此期间也逐渐被封神，我记得在九十年代，在教科书上袁隆平出现的频率开始高起来了，教科书经常出现的一句话就是中(共)国以世界百分之七的土地养活百分之二十的人口，下面就是袁隆平的照片。傻子也知道共产党在给中国人洗脑。上面说过即使在99年，中国广大农村仍然是时节或农忙时才可能沾荤腥，其他时候均是吃素，导致营养不良的人居多，从乞讨方面也是这个时候以求一碗饭者居多。中国进入温饱社会最有力的表现就是农村青壮劳力务工人数增多，这同样是在加入世贸组织后才发生的事，因为加入世贸组织后，粮食进口的价格居然比中国本土百姓种植的成本还低，而且加入世贸组织后，外资大举进入中国开办企业，青壮劳力一算，种地一年累死累活还没有进工厂赚得多，而且守在枯燥乏味的农村，一到晚上就黑灯瞎火，哪有灯光璀璨的城市有趣呢？袁隆平的杂交水稻技术早在这之前都非常成熟，若是有用，为什么还要等到加入世贸组织后才基本解决中国人民的温饱社会呢？当然事情的发展也不会按照袁隆平的意思去进行，几十年如一日的帮他封神造神，从之前的杂交水稻之父到被封为当代神农，德鲁伊，成了共产党粮食的图腾。共产党希望籍袁隆平这面旗帜对中国百姓洗脑，通过袁隆平表达，如果不是共产党，你们能吃饱饭？这样可信度增加，而且这确实能蒙蔽不少的中国百姓，借此熄灭百姓反抗之心。不知道袁隆平的想法是怎样，但相信他是有羞愧之心的，否则在2018年接受采访时不至于说不从外国进口粮食，中国人要饿肚子这番话。

郭先生说过，马明哲，沈南鹏，任正非等人明面上是企业老板，其实背后相当于共产党的国家领导人，因为他们是共产党对外冲锋的尖刀。而从袁隆平在2019年接受共产党功勋表彰大会，面对习近平仍然是不卑不亢来看，恐怕袁隆平背后的影响力不比这三人低，甚至略高一筹。

2019年的共产党，有几个人见了习近平是不谄媚至极？马明哲等人对于共产党的统治来说还只是工具，是术的层次，而袁隆平对于共产党的统治来说，是隐隐有旗帜的地位，几乎到了道的层次。

两者作用就不一样，这同样也侧面证明为什么袁隆平死后共产党的各路媒体一窝蜂给袁隆平哀悼，顺带搭上一个肩上带星的丘八肝脏专家（十有八九都是沾满鲜血的刽子手）。待遇堪比2015年死去的乔石万里。试问如果是马明哲等人死掉，共产党各路媒体会这样吗？例子可以参考进入中南坑都穿牛仔裤准备帮共产党攻克量子技术的张首晟。

袁隆平今年已有九十岁，新闻上说是因为摔跤住院然后导致现在的逝世。从公开的新闻得知袁隆平一向精神矍铄，身体硬朗，居然还去迪拜研究如何在沙漠里种水稻。再加上能匹配共产党领导人的医疗条件，应该说想死都难。

我没有证据就不阴谋论，但是共产党放出生化武器给共产党自己造出了铁桶一般的国际绝境，外汇必然锐减，而且现在作为粮食出口大国的印度本身被生化武器伤害得哀鸿遍野，粮食价格必然涨高。外汇锐减和粮价上涨就对共产党是双倍的打击。

房地产大开发前中（共）国土地产出的粮食只能让人民饿不死，而现在是工厂和房地产侵占耕地粮田以及企业排污污染，田地至少在这基础之上减少百分之三十左右，而且今年长江又进入汛期，多地发布防汛预警，对于田地的影响也是至少增加百分之五左右，再加上外汇的锐减以及国际铁桶一般的绝境，粮食所需的化肥和种子也必然会受到影响，记得在加入世贸组织之前，化肥用的不多，一亩田产水稻顶多800斤左右，这只能让人民饿不死。后来化肥大举应用和种子质量的提高，亩产也就是1600斤左右。而且田地长期使用化肥，造成土壤板结，来年必须更多用化肥才能解决，而且人有人力，田有肥力。因为化肥的大量使用，让田很少休耕，当化肥和种子都出问题，可能连原始的800斤都难以达到。而且即使人口普查显示中国人口减少，但基数还是比之前多。这一连串的问题，很难相信共产党的粮食还能打出牌来。再结合郭先生早就说过共产党粮食大概在六月份消耗殆尽，加上袁隆平—共产党的粮食图腾倒了。这很难说不是上天对我们发出共产党粮食要出大问题的预兆！
