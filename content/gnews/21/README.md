3.1 月6日晚间，中芯国际发布公告称，获OTCQX市场的营运者OTC Markets Group的通知，根据行政命令和相关监管指引，公司将从1月6日（星期三）交易结束时被撤出OTCQX市场，公司的证券将不再具资格在OTC Link ATS 上报价或交易。OTCQX是美国场外证券的金融市场，中芯国际在2019年从纽交所退市后，公司美国预托证券股份在OTCQX 市场进行交易，其被撤出是受到川普总统行政命令限制。

7.2021年1月5日，河北省新增20例本地中共肺炎确诊病例，其中石家庄市报告19例。由于疫情防控形势严峻，石家庄接连出台多项措施严格人员管理。6日，石家庄客运总站发布消息，暂时停运，河北境内多条高速禁止上道、劝返车辆。自6日至8日24时，石家庄、邯郸等地需进京车次被限制进京，车票暂停发售。虽官方否认，但石家庄已经事实“封城”。

二、世界新闻

1.美东时间1月6日下午1点，美国国会两院举行联席会议，对各州提交的总统大选结果进行认证。会上，参、众两院各有共和党议员对亚利桑那州提交的选举人团投票结果提出反对意见，从而触发参、众两院各自辩论。但在辩论的同时，有民众冲入国会大楼，与国会警察爆发暴力冲突，导致审议工作暂停。川普总统随后发推，呼吁抗议民众支持国会警察和执法人员，维持和平抗争。其所发内容连遭推特公司删除，并被封锁账号12个小时。华盛顿特区市长宣布，将在1月6日晚上6点实施宵禁。

不幸的是，在国会发生的骚乱中，超过50人被捕，14名警察受伤，4人死亡。

2. 美东时间1月7日凌晨，副总统迈克·彭斯宣布，国会已确认选举团的选举结果。白宫通讯副幕僚长兼社交媒体总监 Dan Scavino 随即代表川普总统发表声明：“尽管我完全不同意选举结果，而且事实也证明了我的观点，但是不管怎样，1月20日将会有一个有序的过渡。我一直说，我们会继续努力奋战，确保只有合法的选票才能予以计票。虽然这个结果代表了总统历史上最伟大的第一任期的结束，但这只是我们让美国再次伟大的斗争的开始！”

3.美国南方佐治亚州两名联邦参议员的决选竞争激烈，但不幸的是此次选举是大选舞弊的翻版。民主党参议院候选人华诺克和奥索夫都在数据上赢得胜利。

4.第一夫人梅拉尼娅·特朗普的两名高级助手于周三（1月6日）辞职。有消息称，还有部分白宫高级官员正在考虑辞职，包括国家安全顾问罗伯特·奥布赖恩（Robert O’Brien）和他的副手马修·博明（Matthew Pottinger）。
