# 香港：又有11人因企图乘坐船只逃往台湾一案被国家安全部门逮捕

**年龄在18至72岁之间的八名男子和三名女子，因涉嫌去年帮助12名民主活动人士而被拘留**。

[!\[\]()!\[\](https://spark.adobe.com/page/iaPF72Zy2wH5G/images/f1f15be6-adba-4415-ac42-d4748d0f657b.png?asset_id=f94153d2-d229-46dd-af36-c4c822b8f7e5&amp;img_etag=%22f748392f0fdebd12f23adb99b16ea1bf%22&amp;size=2560)](https://spark.adobe.com/page/iaPF72Zy2wH5G/images/f1f15be6-adba-4415-ac42-d4748d0f657b.png?asset_id=f94153d2-d229-46dd-af36-c4c822b8f7e5&amp;img_etag=%22f748392f0fdebd12f23adb99b16ea1bf%22&amp;size=1024)周四，香港律师黄国栋在办公室外被警察押走 。图片：Vincent Yu/AP

据当地媒体和活动人士报道，香港警察根据《国家安全法》逮捕了11人，原因是他们涉嫌帮助去年被指控企图乘船逃往台湾的12名亲民主人士。

据《南华早报》援引未具名消息人士的话称，警方以 “协助犯罪者 “的罪名逮捕了年龄从18岁到72岁之间的8名男子和3名女子。

试图帮助去年8月在中国大陆被拘留的12人的律师黄国栋（Daniel Wong Kwok-tung）也在被捕之列。

黄在脸书上写道，警方星期四凌晨6点到达他的公寓。

黄是香港民主党的一员，以向在2019年香港反政府抗议活动中被捕的数百名抗议者提供法律援助而闻名。他说，他不知道将被带往哪个警察局。

警方告诉《路透社》，他们仍在收集信息，无法立即发表评论。

当地媒体说，被捕者涉嫌协助12名香港居民去年的逃离行动。这12名香港居民面临与2019年在中共国统治下的香港的反政府抗议活动有关的指控。一行12人在试图乘快艇逃往台湾时，在海上被捕。

去年12月下旬，中共国一家法院以非法越境罪判处他们中的10人监禁7个月到3年的徒刑，此案引起了国际社会对这些活动人士所受待遇的关注和担忧。

在被捕时尚未成年的两人，现已返回香港。

被拘留者的家人说，他们被剥夺了与独立律师接触的机会，并怀疑香港当局在中共国的逮捕行动中提供了帮助。

自北京于2020年6月颁布了被批评者认为旨在消除异议人士和遏制自由的《国家安全法》以来，自治的台湾已成为香港亲民主活动人士的热门目的地。

根据国安法，已有100多人被捕。

电信运营商香港宽带网（Hong Kong Broadband Network）周四承认，为了遵守国安法，它已经封锁了一个反政府网站。据信，这是警察首次在不涉及法院的情况下使用国安法审查在线内容。

在当地媒体周末报道后，确认警方已下令供应商限制访问“香港编年史”（HKChronicles）网站。“香港编年史”网站是一个亲民主的平台，该平台收集并发布在2019年大规模抗议活动期间的信息，包括警察官员的身份。
