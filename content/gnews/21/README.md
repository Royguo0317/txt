“我们要高举中国特色社会主义伟大旗帜，牢固马克思列宁主义、毛泽东思想、邓小平理论、坚持三个代表重要思想，深入贯彻落实科学发展观……”这是当年大陆考生政治科目考点，厚颜无耻一如既往，弥天大谎亘古无双，伟大光荣正确得中人欲呕。

但是，多数人并未发现这背后隐藏的潜台词，说来也简单——去掉各种哄骗墙国傻白甜的“中国特色”理论，只留动词——“高举”、“牢固”、“坚持” 、“深入”、“贯彻”、“落实”；再加上“一个中心，两个基本点”，答案呼之欲出：三点式，别号比基尼。 “高举”、“牢固”、“坚持”胯下某物；“深入”、“贯彻”、“落实”床上玩物——全部是进行某种运动的过程。主要对付三点，上面两点“挡膻中”，下面一点“挡中央”。名唤党中央，实为“裆中央”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/07003036/%E4%BA%91%E4%B8%8D%E5%99%A82.jpg)
原以为走狗文人们都是“只盼坟前有屏幕，看奥运，同欢呼”的亡灵派诗人拥趸，万万没想到文宣部笔杆子不知是精虫上脑还是良心发现，我手写我心，以隐晦笔法，揭淫秽真相。奈何凡夫俗子只看表象，让你勉强饿不死就为之击节赞叹，真真是普天同庆，呜呼哀哉。



事实上，某党对于生殖器的把控，乃几千年来亘古之未有。甚至刻意宣扬的“封建礼教”多半也是穿凿附会来制造刻板印象，以造成“当今比以前自由”的假象。传统意义上的礼教大防，其实远没有这七十年来得严重。下面，且让笔者翻一翻故纸堆。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/07003101/%E4%BA%91%E4%B8%8D%E5%99%A83.jpg)


“（叔梁）纥与颜氏女野合而生孔子。祷于尼丘得孔子。” 这段《史记•孔子世家》的记载明确说明，孔圣人虽贵为万世师表，也不过“玉米地兴之所致来一发”的意外产物，当然那时候中国还没有玉米。事实上，春秋战国乃至后世千年，华夏大地野合“胡天胡地”蔚然成风，周代礼乐尽管等级森严，还是宽限了一天作为“野合日”——农历的三月初三。当代人往往只记得三月三冻得把眼翻，但当时有个雅称，上巳，甚至有祈雨踏青等诸多活动。 “暮春者，春服既成。冠者五六人，童子六七人。浴乎沂，风乎舞雩，咏而归。” 《论语》除了之乎者也，更不乏此等风雅事，当然过程中你要是和谁看对眼，脱离队伍钻个草丛，并不算违背礼数。 “浴乎沂”就是在沂水洗澡（山东临沂），而当时的烂漫风流文昌地，两千年后竟成了网瘾少年十万伏特电刑院，今昔对比，不禁令人困惑是否错乱了时代？
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/07003122/%E4%BA%91%E4%B8%8D%E5%99%A84.jpg)


上巳节祈雨踏青顺道相亲，露水情人们成双成对去钻茅草堆庄稼地小树林天雷勾地火，这种习俗直到唐代还有，不信请看杜甫《丽人行》——“三月三日天气新，长安水边多丽人。态浓意远淑且真，肌理细腻骨肉匀。绣罗衣裳照暮春，蹙金孔雀银麒麟。”当然还有更风雅所在，文人诗家来往章台柳巷，俊才荟萃曲江池平康坊。绝非今日大保健盯着钟点到时赶人，而是真正笔墨相亲诗歌酬和。男人狎伎，女子亦然——但凡有点小财的，养几个男宠乃寻常事尔，当时称作面首。最有名莫过高阳公主，私生活之混乱放到当今也足以为之瞠目。另外就是武则天，媚娘老矣，尚宣怀义矗二堂——可怜木构建筑史奇迹，遭后者付之一炬，惜哉。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/07003140/%E4%BA%91%E4%B8%8D%E5%99%A85.jpg)
论中国历史，必言及汉唐。但按照《红楼》说法，强汉盛唐实乃“臭汉脏唐”——刘家皇帝男女同吃，比如断袖之癖就出自汉哀帝。上有所好，下必甚焉，后来魏晋南北朝也喜好男风。唐代这点不明显，但是就老婆这事儿，也能“父死子继兄终弟及”，或者抢儿子老婆“扒灰”——杨贵妃就是唐明皇“扒灰”所得，为了掩人耳目还让她假称出道，道号太真。遂有了《长恨歌》里的那句“中有一人字太真，雪肤花貌参差是”，不得不说，贵圈真乱。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/07003209/%E4%BA%91%E4%B8%8D%E5%99%A86.jpg)
纵然贵圈真乱，但并不妨碍汉唐成为中国最伟大的两个朝代。正如当今同样“贵圈真乱”的美国成为世界灯塔一样。自从宋、辽、夏三朝鼎立，胡汉恩仇，礼教大防，中华便由开放包容渐渐转为内敛自噬，但有宋一代，没了上巳，还有上元——“东风夜放花千树，更吹落，星如雨。宝马雕车香满路，凤箫声动，玉壶光转，一夜鱼龙舞。娥儿雪柳黄金缕，笑语盈盈暗香去。众里寻他千百度，蓦然回首，那人却在，灯火阑珊处。”而所谓七夕“中国情人节”，不过是乞巧，一群女生自娱自乐罢了，哪像商家吹嘘，什么节日，最后都变成买买买。最终结局男友为买买买头悬梁锥刺股九九六ICU，女友日日花呗债台高筑——割韭菜，好一手。

真正“礼教大防”，是在连续挨了蒙元、满清两记重锤之后。中间还有个工作狂朱八八推波助澜，妄图管天管地管空气——比如全国一年只放三天假。洗澡麻烦，所以之前朝代都五日十日一“休沐”，休息沐浴。自此卫生堪忧，明末人口又达到从未有过的一亿五千万，于是和中世纪一样鼠疫横行；另外建造户籍“黄册”，放在南京后湖（玄武湖），力求登记到每个人头，工匠军户更是勒令子子孙孙不许转业，明后期部分管辖渐渐松懈，奈何清军入关，于是变本加厉三百年。汉唐妇女随便离婚改嫁，甚至私奔——比如著名的红拂夜奔，大兴城（长安）司空杨素甚至都懒得去管；哪像明清贞节牌坊，閙到最后出现无数个祥林嫂献门槛。两宋虽有缠足，但只如绑腿一般用于修形，不会像清代硬生生勒断脚掌——谁TM喜欢畸形？除非是变态，不想让人逃跑！秦汉曾严刑峻法，之后各种酷刑渐渐绝迹。唐代就算谋反最多也就枭首腰斩杖杀，从辽朝开始，单凌迟就有脔杀、支解、铁梳、投崖、投高崖、五车毂杀、生瘗（活埋）、射鬼箭、炮掷、践骑、钉割、沉河、焚、剖心诸法。因为孤立系统熵恒增，不对外开拓的帝国便会越来越趋向窝里横——比如元明往后越发酷烈的刑罚，直到满清十大酷刑，如今登峰造极的大数据控制，活摘器官，采生折割，放毒杀人——绝对是文明倒退。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/07003238/%E4%BA%91%E4%B8%8D%E5%99%A88.jpg)
“它管天管地管空气，中间还管着你的生殖器”，其实某党管理生殖器，还要细分三种，第一种是明管，一会儿英雄母亲，一会儿计划生育，一会儿开放二胎；节育环和强制流产曾造成多少人伦惨剧；譬如自家媳妇一尸两命，导致某位特种兵恼羞成怒，遂有了九四建国门枪击事件。第二种是暗管，明税暗税、民脂民膏，林林总总比当年的秦帝国还高，普罗大众疲于奔命，结果都成了官僚家蓝金黄筹码。这还不够，只听天空一声屁响，熏得人如饮琼浆如毒断肠，“小目标赚他一个亿”、败家娘儿们的最爱“马爸爸”、以及全民老公“王思聪”闪亮登场，人们心中注满消费主义——感官享乐为人生目的，为此还不加班加点累死活该？正如当年的阿萨辛派山中老人，将少年们麻醉，让他们在流着奶与蜜，有圣处女七十二侍奉的“应许地”醒来，享乐几天后再度麻醉，送回原地。自此他们便对死后天堂深信不疑，为了那个骗局甚至可以粉身碎骨肝脑涂地。而始作俑者们，坐高堂，食高粱，全不计及他人丧。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/07003300/%E4%BA%91%E4%B8%8D%E5%99%A89.jpg)
而最可恶的，莫过于第三种，“运动式执法”。回看这七十年，其实就是全国范围内永无宁日的瞎折腾——除四害、三反五反、上山下乡、大炼钢铁、文化大革命。明面上大声疾呼人人平等，工农阶级领导一切；事实单生殖器而言，就分出无数等级。文革期间，国家领导共妻乱伦，是游龙戏凤；中层干部作风不检，是感情冲动；底层男女，稍有情愫，便是流氓成性。加之样板戏洗脑，八个完美英雄，只聊革命不谈感情。就算党支部没察觉，周围一旦揭发，立刻秒变阶下囚，扫厕所戴鉄帽剃阴阳坐飞机脸上刺大字都是意料中事。八十年代初终于放开，压抑已久的还未释放，又迎来了“严打”风潮，牵个手都能被当流氓一枪爆头，各种冤假错案株连何止亿万！至于罪魁祸首们往往都是三部曲——拍脑袋决策，拍胸脯保证，拍屁股走人，哪怕身后洪水滔天，“胜似闲庭信步”。哪怕到如今，还会时不时来个扫黄打非。

事实上，一个缺德的社会，才会到处强调道德；一个没有法治的社会，才会各种运动式执法——共党高层要夜夜笙歌，所以养刁妇女消费欲，让无数吊丝舔狗为五斗米累断老腰；结果心上人还变成二奶小三；要残害幼苗，所以天天教育底层不要早恋，专心学习文化知识，然后青梅竹马突然失踪，报完警就石沉大海再无音讯。甚至东莞被取缔，据说就因为复刻了高层的变态花样玩法——好一个只许州官放火，不许百姓点灯！更兼之，颇有生殖器意向的“中国特色”“建筑艺术”，不单海外红色基因开枝散叶，还有各地真正的奇葩建筑——譬如酷似大裤衩的中央电视台，一柱擎天式广西新媒体中心，以及女阴状（或者全知之眼）海航标志——对天来一发，就以为奸了全世界；阳物埋土里，就觉得日了整地球。上帝欲人灭亡，必先欲人疯狂；正如当年泰坦尼克号，就算上帝亲自来，它也弄不沉这艘船。
