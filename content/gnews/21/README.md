随着CCP病毒疫苗接种工作在全球多个国家逐步展开，围绕疫苗所产生的问题也日益增多。最新一项民调发现，只有大约一半的美国人表示愿意接种疫苗。许多人出于对疫苗安全性的担忧，表示会等到更多人接受注射后，确认安全再接种。在中国，目前疫苗接种率仅为3.56%，可见人们对在不到一年时间里“快马加鞭”研发出来的疫苗的安全有效性持严重怀疑态度。

那么战友们如何避免被接种？以下是医疗部战友为您答疑解惑的部分记录：

**问：请问有什么办法可以逃脱打疫苗呢？**

答：可以说自己有严重的过敏史。有过敏症状的人是不适合注射新冠疫苗的，因为注射后会加重过敏现象，最严重时可导致死亡。过敏体质者会因各种原因引起过敏，所以如果您实在是非常的想要躲避打疫苗，可以称自己有过敏史。生物界中有几千种过敏源可导致人体过敏，而且过敏现象也是随情况变化反复，有时候通过检测也不一定查得出来。

问：太好了！有过敏史就可以不用打了，太给力了！那么正在吃羟氯喹是否也和疫苗有冲突吗？如果是的话就更好了，谢谢！

**答：羟氯喹是免疫抑制剂，和疫苗有冲突。**

**问：请问如果人在美国，因为工作单位要求员工必须要注射疫苗后才能上班。我看CDC说要有过敏证明才能不用打，我不知道怎么才能拿过敏证明？我没有明显过敏。**

答：很多病人在看过敏专科医生的时候并不一定有明显的症状，医生大多是根据病人的描述来判断其是否有过敏史。因为很多过敏症是具有季节性的，突发性的。有些可能会要通过验血来寻找过敏原（病人可以拒绝验血），但既便是验血也有很大的可能找不到过敏原，这很常见。一般过敏的常见症状是皮肤瘙痒，眼痒，喉咙痒，流泪，打喷嚏，流鼻涕，严重时会出现呼吸困难。

**问：那我写对哪种物质过敏比较好?**

答：阳光，冷空气，灰尘，桃子，西瓜，奇异果，苹果等多种水果，海鲜，咖啡，牛奶，鸡蛋，麦麸，坚果，动物毛发，螨虫….. 都有可能是引起过敏的过敏原。

**问：請問美國的疫苗安全嗎？**

答：建议战友可以去闫博士的推特了解一下详情，目前世界上还没有安全的疫苗对付中共的生化病毒的。保护好自己与家人。

**问：请问过敏性体质的是不是更不考虑打疫苗了?**

答：因为新冠疫苗研发上巿的时间太短，因此对它的长期副作用还并不了解，这也是为什么很多人都不进行接种的最大原因。至于过敏性体质人群接种后的短期副作用是明显的，就是导致过敏反应严重。
