在《自然》杂志上发表的一项最新研究中，Wang等人研究了COVID-19 mRNA疫苗引发的抗体的中和活性以及SARS-CoV-2对SARS-CoV-2变体的自然感染。

持续进行的COVID-19大流行对公共卫生，经济和社会造成的破坏性影响使疫苗的开发成为全球卫生的重中之重。 因此，作为对COVID-19的紧急响应，疫苗的开发正以前所未有的速度发展。

当前，COVID-19疫苗有四种主要类型：核酸（mRNA和DNA），病毒载体，蛋白亚基和灭活病毒。 两种COVID-19 mRNA疫苗（Pfizer-BioNTech开发的BNT162b2和Moderna开发的mRNA-1273）已获得美国食品药品管理局（FDA）和欧洲药品管理局（EMA）的授权。 此外，Ad26.COV2.S（Johnson＆Johnson / Janssen）已获得FDA和EMA的批准，ChAdOx1 nCoV-19（AstraZeneca）已获得EMA的批准，二者均为病毒载体疫苗。

BNT162b2和mRNA-1273是脂质纳米颗粒配制的，核苷修饰的RNA疫苗，编码SARS-CoV-2的融合前加标糖蛋白。 他们都在3期临床试验中显示出良好的疫苗效力（94–95％），可预防COVID-19。

但是，SARS-CoV-2的新兴变体及其全球扩展引起了人们对当前COVID-19疫苗对潜在变体（VOC）的潜在保护降低的担忧。 在英国（B.1.1.7），南非（B.1.351）和巴西（P.1）中出现了在刺突蛋白中带有多个突变的显着变体。  B.1.1.7变体（20I / 501Y.V1）是全球分布最广泛的VOC，在受体结合域（RBD）中具有N501Y取代，在N末端域中具有H69 / V70缺失，并且与P681H突变相邻 刺突蛋白中的弗林蛋白酶切割位点。 该变体与增加的透射率相关。  B.1.351变体（20H / 501Y.V2）包含几个突变，包括K417N，E484K和N501Y。  P.1变体（B.1.1.28.1）在穗蛋白的RBD中具有K417T，E484K和N501Y取代。 这些挥发性有机化合物还具有D614G突变，可增强病毒快速传播的能力。

Wang等人测试了来自疫苗接种者血浆（BNT162b2，n = 6； mRNA-1273，n = 14）对具有K417N，E484K，N501Y以及这三种RBD突变组合的假病毒的中和活性（B  .1.351变体）。 研究显示  https://doi.org/10.1038/s41392-021-00623-2

对E484K，N501Y和K417N：E484K：N501Y组合的中和活性降低1-3倍（分别为p = 0.0033，p = 0.0002和p​​ &lt;0.0001），但对野生动植物的中和活性没有显着差异 型和K417N突变。 该结果表明，COVID-19 mRNA疫苗引发的中和抗体对新兴的具有RBD突变的SARS-CoV-2 VOC无效。 此外，SARS-CoV-2感染6个月后获得的恢复性血浆在中和K417N：E484K：N501Y组合方面的效力降低了0.5到20.2倍（p &lt;0.0001）。

在随后的分析中，该研究扩展到了SARS-CoV-2 RBD特异性记忆B细胞。  mRNA疫苗引起了类似于自然感染的强烈的SARS-CoV-2 RBD特异性记忆B细胞反应。 单克隆抗体（mAb）由SARS-CoV-2 RBD特异性记忆B细胞表达，对SARS-CoV-2伪病毒具有强大的中和活性。 但是，在17种最有效的mAb中，有14种在中和K417N，E484K或N501Y突变方面表现出降低或取消的活性。 还检测了mAbs的选择压力。 当在疫苗诱导的单克隆抗体的存在下培养重组水泡性口腔炎病毒/ SARS-CoV-2 S时，这些突变就出现了。

同样，Chen等[2]报告说，接种BNT162b2的个体的血清对新兴的SARS-CoV-2变体的中和活性降低。 他们观察到针对B.1.1.7分离株（2倍），E484K / N501Y / D614G重组变异体（4倍）和两种编码B.1.351的嵌合SARS-CoV-2菌株，疫苗中血清的中和力显着降低。 与Vero-hACE2-TMPRSS2细胞中的D614G变体相比，峰值（10倍）和P.1峰值（2.2倍）。  K417N / D614G变体未显示中和活性显着降低，表明来自BNT162b2疫苗接受者的血清对含E484K和N501Y病毒的中和能力较低。

此外，Wang等[3]证实了在恢复血浆中B.1.351菌株的中和活性大大丧失（9.4倍），而接受过mRNA疫苗的疫苗的血清则丧失了中和活性（10.3-12.4倍）。  E484K似乎是抗中和性的主要因素。 而且，针对B.1.351变体，mAb治疗剂在临床使用中或在临床研究中的中和能力降低。

2个SARS-CoV-2突变，疫苗和免疫力：表示其变体  COVID-19疫苗引发SARS-CoV-2特异性CD4 +和CD8 + T细胞反应以及中和抗体。 即使当前COVID-19 mRNA疫苗引发的抗体显示出针对SARS-CoV-2变体的中和活性减弱，但T细胞反应可能对宿主抵抗SARS-CoV-2变体具有一定作用。

综上所述，最新研究的结果表明，抗药性SARS-CoV-2变体的出现可能会使当前COVID-19疫苗的作用无效。 但是，COVID-19疫苗不仅可以引发中和抗体，还可以引发SARS-CoV-2特异性CD4 +和CD8 + T细胞反应。 已经证明，使用多种疫苗平台（包括mRNA和病毒载体）进行疫苗接种可引发SARS-CoV-2特异性CD4 +和CD8 + T细胞反应（图1）。 原则上，逃避T细胞应答比中和抗体应答更困难，因为多个T细胞表位分散在整个病毒蛋白中，而中和抗体靶向病毒蛋白中的狭窄区域。 尽管已经报道了消除与主要组织相容性复合物结合的SARS-CoV-2突变，[4] Tarke等人[5]最近报道了SARS-CoV-2变体对COVID-19恢复期的CD4 +和CD8 + T细胞反应的影响不大。 和COVID-19 mRNA疫苗的接受者。  T细胞对变种B.1.1.7，B.1.351，P.1和CAL.20C（在南加利福尼亚州出现）的反应与对SARS-CoV-2祖传菌株的反应没有差异。 尽管变异，但大多数SARS-CoV-2 T细胞表位均被保留。

当前COVID-19大流行的未来是无法预测的。 不断需要对变种的出现进行仔细的监视，并对其变态对公共卫生的影响进行彻底的调查。  SARS-CoV-2 VOC的不断出现对更新当前的COVID-19疫苗以及开发提供更广泛保护的疫苗都具有影响。 即使SARS-CoV-2变体逃脱了当前COVID-19疫苗引发的中和抗体，T细胞免疫也可能通过减轻疾病的严重程度和降低死亡率来帮助减轻COVID-19的疾病负担。 应在不久的将来评估COVID-19疫苗在现实世界中的有效性，尤其是在预防住院，并发症和死亡方面。 最重要的是，应执行公共卫生政策以确保对SARS-CoV-2变体的担忧以及疫苗对VOC的功效降低不会导致COVID-19疫苗接种率下降。
