# **情报分析师隐瞒中共国干预选举的信息以支持拜登**

[!\[\]()!\[\](https://spark.adobe.com/page/gYLT8bdEtgR8J/images/3b478780-9ca4-483b-9b02-578076ca2ee5.png?asset_id=9676f157-14d0-44b1-9aad-4ce4c768ee3d&amp;img_etag=%22b84385225d9e658c424513e23a572ef2%22&amp;size=2560)](https://spark.adobe.com/page/gYLT8bdEtgR8J/images/3b478780-9ca4-483b-9b02-578076ca2ee5.png?asset_id=9676f157-14d0-44b1-9aad-4ce4c768ee3d&amp;img_etag=%22b84385225d9e658c424513e23a572ef2%22&amp;size=1024)

国家情报总监办公室的一份新报告显示，官员们淡化了中共对2020年大选的干预”以避免支持川普的政策”，因为很多人 “不同意”政府的对华政策。

这一揭露来自《华盛顿观察家》获得的一份14页的报告，该报告由国家情报总监办公室解决方案部主任巴里·祖拉夫（Barry Zulauf）撰写。这份报告于周四送交参议院情报委员会，揭示了该机构对2020年大选中外国影响的评估是在 “两极分化”的氛围中进行的，并由那些 “不同意”川普政府对中共国态度的官员完成。

“鉴于俄罗斯和中共国分析人员对考察目标的分析方式不同，中共国分析人员似乎不愿意将中共国的行动评估为不当影响或干涉。分析师们似乎不愿意让他们对中共国的分析被提出来，因为他们往往不同意政府的政策，言下之意，我不希望我们的情报被用来支持这些政策。”祖拉夫介绍说。

国家情报局局长约翰·拉特克利夫（John Ratcliffe）也写了一封信，呼应了祖拉夫的观点。

“以我作为掌握了所有美国政府关于中共国的最敏感情报的个人的独特优势来看，我不相信情报界分析师所表达的多数观点能全面、准确地反映中共国政府影响2020年美国联邦选举的努力程度。”他解释说。

拉特克利夫还指出，在祖拉夫的报告中发现了 “令人担忧的揭示，（包括）关于中共国选举影响报告的政治化，以及对根据情报提出不同观点的分析人员施加不当压力”。
