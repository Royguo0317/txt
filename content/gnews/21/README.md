## 【香港要聞】港警修改《媒體代表條例》國際記者聯盟『IFJ』發聲明譴責【跟踪報導】
`2020-09-26 04:51 GM42` [轉載自GNews](https://gnews.org/zh-hant/383535/)

製作單位：戰友之家-粵語組
編撰：點蒼山
覆核：艾文

**據【立場新聞】訊**：北京時間9月23日，港警宣佈修改《警察通例》中的《傳媒代表條例》不再承認記協和攝記協記者證持有者為傳媒代表。通過查詢警方網頁上供公眾查閱的通例第39章已做出修改。員警公共關係科總警司郭嘉銓在商台電視節目承認，修改《通例》是為了方便前線警員工作，又說此舉不會影響記者和校園記者在公眾地方採訪。
     香港七家大學和學院新聞系24日發出聯合聲明，強烈敦促警方撤銷該修改內容，並呼籲港府考慮警方單方面的決定對香港新聞自由的損害，並停止修訂。聲明強調《公民權利和政治權利國際公約》和《基本法》新聞自由是香港公民的基本權利。
     此外多名議員聯合記者會撐網媒，斥政府打壓新聞自由，議員毛孟靜、梁國雄、許智峰、盧俊宇、呂文光、陳嘉朗等紛紛發言，呼籲傳媒工會提出司法覆核，主流媒體覆蓋不到的新聞現場更需要網媒，這樣才令公眾知情權更完整，不能剝奪其採訪權。議員還指出真正危險的不是記者而是員警，記者在採訪中被警方強行帶離，打壓港人知情權和採訪權，同時警方不斷攻擊學生記者，未來所有學生媒體將被定為非法媒體，打壓學生追求真相的權利，修例目的就是要控制傳媒。
     國際記者聯盟（IFJ）於今天發表聲明批評，聲明指當局修改條例目的在於把持香港傳媒工作者，以試圖控制傳媒的重要報導，警方舉動反映政權在剝奪新聞自由，並認為警方沒有諮詢業界就修例是沒有基礎的，要求警方撤回修改。 香港外國記者會（FCC）也發聲明強烈反對港警作法，質疑會損害本港新聞自由。

戰友點評：
港警昨天做出的強制修例，引起了軒然，各界做出了強烈反應，毫無懸念全都直指港共政府打壓香港新聞自由，用管理大陸那套來整治香港，圖謀將香港新聞傳媒和港府變成奴才和主子的關係 ; 另外外界看的很清楚，港府的這一作法目的也在於遏制新生一代學生記者和媒體的成長，掐滅香港未來的新聞自由之光，其心歹毒。 利用修例未來可以不再受制隨便抓捕前線的民間記者，讓恐怖暴行無法傳播出來。
中共在毀滅的路上飛奔，受到的批評越來越多，反對聲音也越來越大，各界並不畏懼中共的威脅和壓制。 中共帶給世界的災難讓所有熱愛民主、自由、法治的人們越來越清醒和團結，現在的中共在國際上日趨孤立，表演越來越拙劣，讓人厭惡，針對港警修例，如果媒體工會發起司法覆核，中共會再如何表演？ 我們拭目以待，無論如何制裁中共的走狗港府官員，港警官員的日子不遠了，香港人的血淚不會白流。
