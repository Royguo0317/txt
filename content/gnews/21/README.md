## 民主体制虽不完美，但独裁体制天理难容！
` 为子孙爱七哥农场` [轉載自GNews](https://gnews.org/zh-hans/537984/)

作者：阿娜
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08105514/%E7%8B%AC%E8%A3%81.jpg)
民主体制是法治的制度，独裁体制是人治的制度。独裁体制的国家有中国、北朝鲜、委内瑞拉、伊朗等，这些国家的独裁者或者独裁集团只会利用手中的权力为自己服务，人民没有一点自由，丝毫没有安全感和幸福感。下面，我从一个普通人的视角来分析一下独裁体制的邪恶性。

**一、独裁国家给人民带来的只有不安全感，建立国家军队不是为了保护本国的人民，而是为了压迫人民，维护利益集团和独裁者的统治。**有些独裁国家的百姓还在饿肚子，而当权者却花费大量金钱和人力去搞军备竞赛、军事建设。等到老百姓反抗时候，这些军队和军人就成了镇压人民的武器。

**二、在独裁国家，法律制度只是书面上的东西，真正落实起来还是那些掌握权力和占据政府高位的人说了算。**老百姓想要跟政府要公平、要依法办事那简直是比登天还难。中国人常常会说“哎，当官的就是这个样儿，怎么办呢，还得活着啊！”这就是老百姓的普遍心态，他们已经被强权洗脑、压迫得失去了对法治的任何需求和信心。

**三、独裁国家的人民没有自由说话的权力，连最起码的人权都没有。**看看有哪个独裁国家允许老百姓对政府有批评或者不同意见呢？没有。1989年64天安门大屠杀事件，到了21世纪的今天，竟然在香港再次上演。中国“网络长城”越建越高，大陆老百姓在中国局域网看不到谷歌，看不了油管，看不到脸书，用不了一切国外用的社交媒体软件，只能使用言论被严格控制的微信、微博。听到的看到的新闻都是被审查过的，中央电视台是共产党的官方频道，传达的全是党想要传达的新闻和声音——“一切都是社会主义好，一切都要听党的，中国共产党救了全中国，毛主席是最伟大的领袖“等等。老百姓经过洗脑，当然不知道毛泽东曾经是中国最大的独裁领导人，几次运动曾让千万的中国人民死去。这就是独裁体制给人民带来的灾难。如今。中共独裁集团给新疆建“集中营”起名叫“再教育营”，强迫新疆人放下自己家里的老婆、老人、孩子不管，去接受什么“再教育”。多么惨无人道啊，听起来像是只有在二战德国纳粹时期才发生的事情。除此以外，更不用说违背人性的计划生育政策和限制人口流动的户籍制度；更别提什么“上访拦截制度”、 “拆迁制度”，一切的一切目的都是维护独裁者的统治。公民的集会权利、选举的权利、自由迁徙的权利、言论自由的权利在中国、北朝鲜根本不存在，老百姓也被洗脑得渐渐忘掉了这些词汇的存在了。

**四、独裁国家给老百姓带来的只有贫穷，没有任何一个独裁国家的经济发展目的是让全体国民富强，更谈不上让整个国家强大和持续发展。**委内瑞拉拥有全世界几乎最肥沃的土地和丰富的石油储备，可委内瑞拉的大部分老百姓却在为温饱问题发愁，货币体系也一度崩塌。中国看似高楼大厦很多，GDP据说是全世界第二，可是中国的GDP增长的要素之一就是大修基建。中国的房子越盖越多，老百姓却买房越来越难。房子可以说盖的比总人口都要多了，可并不是人人买得起房子住。当然民主国家美国发生过房产泡沫，日本也发生过，按着规律中国也会发生。但美国、日本发生后你可以看到老百姓生活发生了哪些改变。等到中国这个独裁国家房产泡沫发生时，相信大家一定会对独裁体制有个更深刻的影响。

**五、再就是信仰自由在独裁国家根本不存在是让人民失去信仰，这是独裁者和独裁集团管控百姓很重要的手段。**宗教信仰是传播爱的，独裁者却在想法设法的制造分裂、制造矛盾，这样的事情正在中国发生着，现实生活中处处存在着制造汉族和少数民族间的矛盾，制造中国人和美国人之间的矛盾。其它独裁国家也是一样，独裁者是惧怕老百姓心中有爱、有信仰的，害怕大部分人团结起来推翻这些邪恶的独裁统治。历史上就是一再上演这样的事情，只是现在的独裁国家更加高明罢了，他们运用互联网来分散人民的注意力，来转嫁矛盾，用金钱至上的洗脑宣传来让人民失去对信仰的追求，失去对信仰的信心。而人心的普遍丧失，对于一个社会而言是最可怕的。

以上，就是我对独裁体制邪恶性的一些粗浅分析，下次我将谈一谈我对民主制度的看法。
