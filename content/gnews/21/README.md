日经新闻(Nikkei)报道，日本首相菅义伟(Yoshihide Suga)下月访问华盛顿，将与美国总统乔·拜登(Joe Biden)举行峰会时确认台湾海峡稳定的重要性。

双方正在讨论在峰会后的联合声明中加入对台湾问题的关切。同时，两国将采取统一立场反对中共赋予海警权力的新法律。

这样的声明将标志着美国和日本领导人罕见地公开表达对中共威胁的关切。这种新的担忧反映了盟国对东亚力量平衡变化的紧迫感。美国国防部长劳埃德·奥斯汀(Lloyd Austin) 3月16日在东京访问日本防卫省时，日本国防大臣岸信夫(Nobuo Kishi)指出，中共的国防开支现在已经增长到台湾的16倍。随后，在美日的 “2 + 2”会议上，双方“强调了台湾海峡和平与稳定的重要性”，同时发表联合声明。

3月初，美国印太司令部司令戴维森(Philip Davidson)在国会听证会上说，他认为中共可能会在未来年6年内采取军事行动控制台湾。这位海军上将针对中共的常规威慑削弱的担忧在日本防务界引发了回响。

中共拥有1250架战斗机，是美国在该地区军力的5倍。据美国印太司令部估计，到2025年，这一差距将扩大到8倍。至于战舰，差距预计将从目前的5倍扩大到9倍。

中共威胁成为印太地区各个国家的防范重点，美日联合声明代表该区域的抵共联盟已经成型。
