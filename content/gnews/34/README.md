**西班牙2021年1月11日电/西喜社——**圣地亚哥动物园野生动物园的几只大猩猩已经检测出中共病毒阳性，据信是此类灵长类动物在人工饲养中的首例。

丽莎·彼得森，公园的执行主任周一称，在公园的八只大猩猩被认为有病毒，并且有几个已经咳嗽。

感染似乎来自公园野生动物护理团队的一名成员，该成员的病毒检测也呈阳性，但一直没有症状。兽医正在密切监测这些大猩猩，它们都在圣地亚哥北部的公园栖息地里一起生活。
