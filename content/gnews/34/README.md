03月26日，《费加罗报》援引法国总统埃马纽埃尔·马克龙（Emmanuel Macron）的话说，在中共冠状病毒大流行的情况下，欧洲正面临一场新型的世界大战，包括由于俄罗斯和中共国在疫苗领域的行动。

法总统说，欧洲“正面临着新型的世界大战，面临着攻击”，包括因为“俄罗斯和中共国想用疫苗影响世界的愿望”。

马克龙认为，在这种情况下，欧洲需要在中共冠状病毒疫苗的生产上实现自给自足。“我们已经建立了生产（疫苗）的能力。我们正在建设生产第二代疫苗的能力。” 马克龙指出，中共病毒会继续传播和变异，所以欧洲必须为这种情况的发展做好准备。

马克龙强调，欧洲是唯一有真正的“中共冠状病毒”疫苗接种战略的国家。欧盟已经制定了全面的产业政策，使其能够在今年下半年达到每年约20-30亿剂的生产能力。“我们将能够满足我们的需求，并大量出口。”马克龙对此充满信心，他表示到夏季欧洲将成为生产“中共冠状病毒”疫苗剂量最多的大陆。

俄罗斯的“人造卫星五型”疫苗目前正在接受欧洲监管机构的审批程序。欧洲药品管理局(EMA)于3月初开始进行专家审查，应该很快会揭示俄罗斯药品是否符合欧洲标准。

俄罗斯外交部表示，俄药面临“欧洲机构的误导、歧视和示威游行运动”。副部长阿列克桑德·古列斯科（Aleksandr Grushko）对此表示，欧盟在疫苗问题上发表了政治化言论。

此前，法国国内市场事务专员蒂埃里·布雷顿（Thierry Breton）称，欧盟对四种获批疫苗感到满意，“我们完全不需要人造卫星五型疫苗。”
