随着美国当前法律意义上的总统和副总统的宣誓就职，美国新一届政府开始就位，它是美国对立政治派系之间妥协的结果，这开启了美国政治历史上的新篇章，对于世界政治与爆料革命运动进程已经、正在并即将产生重要的影响。

笔者曾经在GNews文章[《正义与邪恶 光明与黑暗》](https://gnews.org/zh-hans/695187/)中阐述了正义与光明、邪恶与黑暗的含义。从这个含义中可以理解到世界各国各地区都存在这样的两极，更何况有着三亿多人口的美国呢？同时从文章[《对第三次世界大战的基本认知》](https://gnews.org/zh-hans/651266/)中可以理解到本次世界大战本质是CCP与世界各国部分暗黑势力勾结一起试图控制和奴役全世界人民。因此，CCP才是轴心国，是所有邪恶势力的核心，只有打掉了这个核心势力，世界人民才会得到拯救，免于被奴役，免于被“大重启”计划锁上枷锁的命运。

目前为止，爆料革命已经取得了重要的阶段性成果：美国国务院正式公布认定CCP的种族灭绝罪和反人类罪，这是终极的裁定。这个结果的出现，是灭共运动中的里程碑事件，相信每一个知晓其重要性的战友都会欢欣鼓舞，斗志昂扬。它不仅表明爆料革命运动的正义性，同时也表明美国内部的两极力量妥协的结果，是共同灭共的决定，这是至关重要的。

川普临时离开了白宫，顺利交付了总统位置。有人理解为这是爆料革命的一大挫折，是的，在正义与黑暗的较量中，这是一个大挫折，对于快速灭共是一个挫折。但是，从整个爆料革命运动的世界性范围和阶段性来看，却是一大胜利。

从世界性范围角度看，许许多多正义的人士不再奢望单独一个美国总统能够怎样，而是必须要组建更多的正义力量联盟来与邪恶力量做斗争；不再以为靠单纯的念经、祈祷就能将邪恶力量打败，而是需要我们每一个人作为上帝的一份子去与暗黑势力做斗争，展示各自的人生风采；不再将胜利的希望仅仅寄托于某一个特定的政治人物，而是通过越来越多的人们聚集在爆料革命的周围，形成更为强大的力量与邪恶势力抗衡。

之前路德社节目中提到投票拜登的票数也是几千万，而彻底灭共需要民意的支持，并且必须是大比例的支持。郭先生也提到爆料革命不支持任何一个美国党派，不参与美国政治，避免法律风险。但只要是与CCP勾兑的就一定是我们的敌人，我们会遵照美国的法律规则进行揭发，而凡是灭共的就是我们的朋友。从这点来看，又何愁投票支持拜登的几千万普通美国人不会很快加入到爆料革命运动中来呢？

从阶段性来看，爆料革命绝不仅仅是灭掉CCP就完成了历史使命的，因为目前人类社会所处的阶段是从三级世界往四级世界的转换（关于世界级别的界定，请参阅《太傻天书》），而信仰的重大改变和进步是现在和后续爆料革命运动的主要内容之一。

人世间存在着正义与邪恶，黑暗与光明，是因为每个人从灵魂层次选定的进化道路不同导致的，并且通过每一个人的自我意识以及遗忘的帷幕（指让绝大多数人忘掉前世的机制）让人们自己在关系的交往中体验彼此和进化自身灵魂。美国的大选，尤其是最后的关头，所有涉入其中的每一个人都在做着自己的决定：背叛、反背叛、碟中谍等等精彩大戏粉墨登场。但是这里的关键问题，正如郭文贵先生指出的，你是真的有信仰还是假信仰，真的信上帝还是假的信上帝。我们也需要进一步看到，真的有人是信上帝并且也喜欢做暗黑势力的，因为这些人中也有开悟的人，只不过他知晓自己灵魂层次选择的是服务自己的进化道路，而这类人，才是暗黑势力里面的可怕力量。关于进化道路和进化标准，大家可以参阅GNews文章[《从信仰的角度理解“唯善不败”》](https://gnews.org/zh-hans/716498/)和书籍《太傻天书》。

之所以提到这点，这与近三十年来西方社会灵性运动的发展有关，自1993年开始，《与神对话1》开始发行，后续连续127周位列《纽约时报》的销售排行榜，其中的论述涵盖了宇宙的结构、生与死、婚姻与性、精神与宗教、工作与事业、罪与罚，教育、经济、政治、社会、二元、阴阳等等各个方面，对美国和世界各国已有的宗教信仰教义以及社会和生活各个方面产生了极大的冲击。但问题是很少有人能够将系列书籍读完整以及理解较为完整，往往断章取义自认为理解了。比如关于性，认为自己可以随意与他人性交而不计后果；关于善恶，认为自己即使做了恶事，回到灵魂的世界回看只不过是一段尘世而已，也没有所谓的惩罚人类灵魂的地狱。但他们没有深切理解里面所阐述的因果规律。比如因果规律的本质是：你促使别人经历的，你也会早晚要经历到，这样你的进化才会是完整的（该部分请参考《与神对话3》）；但许许多多在此系列书中获得收益的人更没有看到此系列书籍没有提到进化的道路和进化的标准，从而许许多多的人就忘乎所以、得意忘形的以为自己可以随意做任何事情，认为只要对自己有益的，就随心随性而为，认为放任他人做坏事是真正的爱他们，认为这是被伤害和被杀的人所谓的命运。

于是我们就明白民主党的一些人物标榜爱、神、上帝等高大上词汇的同时，他们却在实际的行动中背弃着自己灵魂所选择的进化道路，而许许多多投票拜登投票的几千万的民众，因为看不到事实的真相，在盲目的信仰上帝的同时却实际上做着南辕北辙的事情。在香港运动中许许多多的人被CCP所折磨杀害的事实面前，在面对新疆种族灭绝屠杀的事实面前，许多人选择了袖手旁观，选择了侧目而视，选择了掩耳盗铃。这是多么讽刺的画面！他们完全忽视了一个基本的事实：人生的意义在于选择，关键在于你选择是为自己还是为他人。

如果我们真的相信神，就会相信神（上帝、佛祖、道、老天爷、万佛万神）知道每一个人的所思所想、所说和所为。真正有信仰的人肯定知晓这点，但为何世界是这个样子呢？那是因为我们都是神的一份子，我们通过彼此的关系来体验二元，经历阴阳，来进化自己的灵魂。

所以，我们看到一场新信仰的运动在兴起，一方面西方社会原有的宗教三十年来面临着巨大的挑战和信任危机，加上目前的第三次世界大战，人们更加认识到现有宗教的局限性、不足，或者说叫不完整性。同时，在这场世界性大战中，每个人参与其中，做着各自的选择。即使那些所谓不选边站的人，也在做着自己的选择：选择了对自己灵魂进化而言消极的立场。

虽然我们面临着川普暂时离开白宫这样的现实，但它只不过是一个小阶段而已，与人类社会进化相比，这是一个必经的阶段。目前的战局，对于整个人类社会的进化却又是极其完美的一件事情。而后续爆料革命运动的快速进展，也将会超出大家的想象。在这个过程中，我们已经看到许许多多的战友落伍、离开和背叛，但这又何尝不是好事呢？一些意志不坚定者的主动离开，会让我们的队伍更加凝聚；背叛者的显形，锻炼了战友们的战斗力，彰显了真战友的身份；同时，越来越多走服务他人灵魂进化道路的人们会加入到爆料革命中来，并且还将是潮水般涌入。对于那些离开和背叛的战友，感谢他们一路来的陪伴，尊重他们的人生选择。如《明日之神》书中所讲，这个世界，大约1-2%，也就是大约一亿人的灵魂来自于更高层级的社会，7-8%的是刚刚进化到这个层次的投生于这个级别社会才几辈子的可爱灵魂们。而只有当这1-2%的人们觉醒并起来行动，才能够推动整个人类社会的进化，这就是所谓的关键多数。作为凝聚这些灵魂的爆料革命运动，郭先生以无私无我之心、之言、之行，带领和号召着这些灵魂的加入，这是多么壮美的人类历史画卷！！！

灭共是一项世界级的运动，想要一帆风顺，一蹴而成，怎么可能呢？有此类想法的，想想都觉得可笑。在接下来的灭共运动以及灭共后的短暂历史中，我们还将看到世界各国，尤其是中国大陆的人民，也将经历集体意识对抗所带来的经验，无论这个经验是快乐的还是痛苦的，短暂的还是长期的，但所有人的灵魂都是快乐的，因为每个人的灵魂都在渴望得到人生的际遇，让他们投身于快速进化自身灵魂的历史运动中。

郭先生发起和引领的爆料革命，在深刻影响和改变着越来越多人的心灵，触及灵魂的深处，在深刻改变着世界各国的政治结构版图。作为爆料革命的一员，战友们各自在做着点滴的贡献，践行着服务他人进化道路的选择，世界的明天由我们亿万战友手中创造，人类社会的进步与进化由我们积极推动，还有什么理由不为此欢呼呢？
