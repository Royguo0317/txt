“习的自信和世界经济的惨状形成了鲜明对比”，《日经亚洲评论》近期报道。这种“在年度庆典上鼓吹‘成功 ’的需要将成为世界经济的炸药桶”。

2019年，虽然对冠状病毒对即将爆发的国家和全球经济的破坏性影响几乎一无所知，习仍然使用这些话：“经济下行压力”…然而现在却沉默了:

一年后，即使中共国和世界经济经历了一场历史性的经济动荡，但是习没有在声明中提到任何“经济下行压力”。确实，中共国的经济在恢复，但似乎有一些事情不对劲。

在中共国，对抗疫情和中共成功阻止疫情对经济造成严重影响，已经被中共国视为历史上值得骄傲的事件。武汉甚至还举办了一个以病毒为主题的展览。
