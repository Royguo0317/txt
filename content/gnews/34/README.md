中共中央第六生态环境保护督察组（以下简称“督察组”）近期在一份报告中公开批评国家能源局存在的诸多问题，报告严厉批评了该局没有听从中共领导人习近平关于环境问题的指示，并责令能源局在30个工作日内将整改方案报送党中央、国务院。

该报告用罕见的尖锐措辞批评国家能源局，指责能源局的工作离习近平生态文明思想和重要指示的要求以及党中央、国务院的决策部署还存在差距。督察组此举释放出一个信号：环境监管部门已明确得到党中央、国务院的授权，获得了对抗中共树大根深的煤炭电力行业的权力。

督察组报告指出，国家能源局一些同志认为，能源领域最重要的是保供，生态环境要求过高会增加企业生产成本；一些同志认为落实“管行业，必须管环保”尚不具备条件等等，由于思想认识存在这些偏差，能源局没有将生态环境保护摆上应有高度。对于多年的环保督察发现并反馈的能源领域的生态环境问题，能源局存在“与己无关”的心态。归根结底，是国家能源局恶化的政治生态对落实生态环境保护的要求造成了严重影响。

中国是全球碳排放最严重的国家之一，能源供应严重依赖煤电行业是其重要原因。根据国家统计局数据，2019年，煤炭在中国能源结构中占比58%。习近平在去年9月份的联合国大会上承诺，为顺应《巴黎气候协定》代表的全球绿色低碳的大方向，中国将在2060年前实现碳中和，即二氧化碳净零排放，且排放量将在2030年前达到峰值。值得注意的是，习近平对外在全球气候变化议题上的积极主张，与拜登就任首日让美国重回《巴黎气候协议》的举动不谋而合，而中央监察组此番强硬表态，则显示出习近平对内已着手制衡被其他派系把控的能源系统，为独占在党内和全球大重启行动中的话语权，他势必会孤注一掷将权力斗争进行到底。
