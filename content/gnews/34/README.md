1月16日，根据俄罗斯防疫指挥部数据，在过去24小时内，俄罗斯共有590名感染中共病毒死亡病例，死亡人数创2021年初以来的新高。今年的最高纪录是1月14日（570人）。

每天死亡人数最多的是莫斯科–79人，俄罗斯首都共有1.24万名中共病毒感染者死亡。过去24小时，圣彼得堡有67例死亡病例（共8 700人），莫斯科地区有32例死亡病例（共3 200人）。在克拉斯诺亚尔斯克边疆区州，过去24小时内有25人死亡，自疫情在该州开始以来，那里的受害者人数已超过1900人。

据俄罗斯防疫指挥部统计，俄罗斯死亡总人数达到65085人。俄罗斯国家统计局给出了另一个关于俄罗斯中共病毒死亡的统计数字。根据后者的数据，4月至11月有116030人死亡。该机构考虑到所有感染后的死亡，包括那些中共病毒不是主要死因，是感染中共病毒后导致其他致命性疾病的并发症。

中共病毒在全俄传播期间，共发现3544623例感染病例，每天增加24092例。

根据约翰·霍普金斯大学的统计，全球中共病毒的受害者人数超过200万人。按绝对死亡人数计算，美国（389000）、巴西（207000）和印度（151000）居首位。俄罗斯排在第八位。该国每100万居民中中共病毒死亡人数排名第56位–446人（世界统计网站数据）。
