**中共的“千人计划”**

2008年12月23日，中共中央办公厅转发《中央人才工作协调小组关于实施海外高层次人才引进计划的意见》。海外高层次人才引进计划（简称“千人计划”），主要是围绕国家发展战略目标，千人计划的组织领导海外高层次人才引进工作小组负责千人计划的组织领导和统筹协调。其实，这是以国家的名义实施CCP的全球“挖角盗窃计划”。

在“千人计划”中，会聘请西方学者与中共国机构合作，进行可能会使隐性合同偏向中共的交易。例如，一些教授通过秘密从中共国大学领钱使他们的薪水翻了一番。 这样，即使他们的研究大部分是由北美纳税人资助的，中共也能够从这些学者那里廉价地获得研究成果和专利。千人计划更着力于那些生在中共国、有海外经验的人，因为他们被认为是在西方收集信息的潜在资产。

**康希诺生物与加中生物医药科技发展协会**

曾在麦吉尔大学受训的宇学峰和朱涛离开了加拿大的生物制药业，于2009年在天津创立了康希诺生物。在回国前，他们曾是与中共政府合作的加中生物医药科技发展协会成员，该协会2003年7月6日在多伦多市议会大厦宣布成立。

该协会网站上2011年1月的报告显示朱涛是该协会的成员，他于2010年12月被选为中国的“千人计划”。2011年的报告称，康希诺生物在 “获得了国家，天津市和其他各级政府的财政支持” 之后，已经部分完成了在天津的研发中心。

南开大学生物学学士，微生物学硕士，加拿大麦吉尔大学微生物学博士。1988至1991年，在南开大学微生物系任讲师。1996年，蒙特利尔IBEX Technologies Inc.公司任科学家，从事酶制剂药物的研究和开发。1998年加盟Sanofi Pasteur。历任科学家，细菌疫苗开发部加拿大分部总监，细菌疫苗开发全球总监， 北美研发中心生物工艺研发总监。他成功地领导了十余个基因重组蛋白质疫苗的研发及两个GMP生产车间的设计和认证，具有多年的生物医药工业研究，开发及生物医药制品GMP生产经验，熟知北美，欧洲和ICH 药物质量管理规范，具有丰富的项目管理经验和优秀的管理技能。入选京津冀生物医药产业化示范区创业领军人才及天津市 “千人计划” 高层次引进人才，并获得天津市特聘专家称号。2010年4月全职回国担任康希诺生物技术有限公司董事总经理。曾任加中生物医药协会会长，现任加中生物医药协会中国分会会长。

据加拿大华人生物医学与制药协会网站上发布的会议记录，于2011年2月的一次会议上，宇雪峰发表了题为“华侨回国创业的经验”。 其中揭示了中共在多伦多的人才招聘网络与康希诺生物的创业公司围绕其从中共政府获得资金的谈判之间的联系。 幻灯片还显示，他建议华裔加拿大研究人员与中共政府开展科学业务可能会涉及“文化冲击”。

**加中共研中共病毒疫苗 ****最终落得被中共耍**

康希诺和NRC的关系可以追溯到当年开发埃博拉疫苗，该疫苗已于2017年获中共国批准使用。埃博拉疫苗和新冠候选疫苗都用HEK293生产。HEK293是人类胚胎肾脏细胞株，它最初是在1973年从流产胎儿的肾脏细胞取得的，这种细胞不仅容易在实验室培养，还能够轻易接受外援的DNA。大约十年前NRC将此细胞用来大量生产疫苗。

NRC于2014年首次与康希诺生物合作，签署授予康希诺使用该细胞株的非独家许可。为康希诺生物以及背后的中共军方提供了细胞株来生产埃博拉疫苗。

NRC与加拿大总理特鲁多（Justin Trudeau）2020年3月签署了一项协议，与中共国分享了加拿大用于疫苗开发的专有细胞系。作为回报，加拿大将对康希诺生物产生的疫苗进行测试，并在NRC实验室大规模生产该疫苗。但在8月，NRC宣布与康希诺生物的协议并未执行，因为中共官员未能将其疫苗送交加拿大进行测试。与此同时，据11月中共官媒报道，自6月以来，许多中共军方人员已经接种了康希诺生物疫苗。

NRC与中共合作已经被耍好几次了，在重大间谍和黑客案中已失去了知识产权，而且还不吸取教训。例如，2014年，NRC成为一次中共国网络攻击的目标，据估该网络攻击给加拿大造成了数亿加元的损失，并丢失包括NRC员工个人信息的大量数据。

**其他**

康希诺生物的天津实验室于2016年启动了一项中美人才招聘计划，该计划由中共国统战部下属的“海外中国事务”办公室提供支持。

最近议会委员会关于加拿大与中共国的往来听证会上，有官员呼吁所有政府部门为了国家安全应审查涉及中共国公司的潜在合同。
