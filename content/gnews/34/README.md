**西班牙2021年4月19日电/西喜社——**

医生说，痒痒的皮疹可能是接受莫德纳疫苗的副作用。

根据周三发表在《新英格兰医学杂志》上的一封医生的信，在一项3万人的试验中，大约有12名受试者在接受第一剂量后,一周左右的时间看到他们的手臂变红，疼痛，瘙痒或肿胀。

信中说，其中5名患者的注射部位有直径超过10厘米（4英寸）的大面积皮损。

两人在其他地方也出现了皮疹，如手肘附近和手掌。

“所有12名患者都被鼓励接受第二剂，并完成了他们的[Moderna]疫苗接种过程，”信中说。

![]()![](https://gnews.org/wp-content/uploads/2021/04/image0-16.jpg)

报道称，第一次注射后平均约八天出现皮疹，但大约四五天后，所有病例都自行痊愈。

其中3名患者在第二次用药后出现了同样的反应，而另外3名患者的皮肤问题则没有那么严重。

医生表示，这些反应看似无害，但可能被误认为是更严重的情况。
