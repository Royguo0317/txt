# 白宫沦陷，民主制度隳颓

北京时间10月26日，正义与邪恶决战十三天之第三天。

随着拜登家族硬盘门持续发酵，一直占据各大社交媒体的话题榜首。由于硬盘的内容繁杂，牵涉甚深，正当大家的脑海还在萦绕着亨特•拜登乱伦视频的时候，今天的主角轮到了美国前任总统奥巴马。全网热议的G-TV再次爆出一张照片，在白宫椭圆办公桌前面，亨特•拜登与奥巴马同框出现，映入大众眼帘。

根据路德先生的爆料，窥拍奥巴马的正是其令尊乔•拜登，而亨特•拜登自己偷偷也复制了一份，高潮不断，剧情跌宕起伏。

其谈论的焦点，是默许中共在南海军事化，通过中间人输送10亿美元达成交易，奥巴马能在当中敛财多少，我们不得而知。但是竟然可以在白宫高谈阔论敏感话题，作为美国总统属于严重的渎职，这是典型的“通共门”！严格来讲，这算是来自中共的密令，奥巴马充其量就是一个政治扯线木偶罢了。

中共治国无道，却弄权有术，在乔•拜登身上耕耘数十载，助推他扶摇直上。乔•拜登可以称得上是美利坚合众国名副其实的地下沙皇。竟可随心所欲地直接指挥总统，干涉行政，美国民主政体的弊端在此昭显无遗。白宫亦沦为权钱交易、寻租的公寓。此番举动，再次佐证了美国引以为傲的民主制度，在蓝金黄面前，不堪一击，其脆弱暴露得淋漓尽致。

民主与法治的余额丧尽之际，便是人类文明黑暗到来之日。

难道人类真的会堕入万劫不复深渊吗？不是的。每每关乎到美国的国运，都会有定海神针的正义力量站出来。例如像先贤林肯、里根这些人，都是在最危急的时刻站出来，如今有班农先生以及朱利安尼先生勇敢地站出来。

再看川普总统，寸步难行，身边幕僚是敌是友，难以捉摸。尽管败象连连，但仍开辟一线生机，纵然面对火海，无所惧哉！左右护法班农先生与朱利安尼先生，日以继夜，千辛万苦地工作，用尽洪荒之力帮助川普总统排干沼泽。加上有爆料革命的推波助澜，扭转乾坤，让美国再次伟大，指日可待。

随着越来越多的人顿悟，正义联盟的力量日益骤增，不再像以前，爆料革命单打独斗。在GTV其中某个影像出现了这么一段话，大致内容就是江志成，吴征等关键人物已倒戈转做证人。可见其德或露，固有善言，希冀他们用暂未泯灭的人性余光，囊萤正义的灯塔。

肉身凡胎，人之秉性，有权必揽。此番大战过后，全美必痛定思定，举国反省，填补漏洞，修正草案，与时俱进，直至找出合适的制度。
