美国时间1月6日，随着彭斯副总统正式发出声明他不会驳回争议州的选举人票，抗议者进入国会大厦，在冲突中一位女性被国会警察子弹打中死亡。所有的主流媒体立刻称这些川普支持者是暴徒，但有证据表明进入国会的暴力骚乱者很有可能是乔装打扮成川普支持者的安提法成员。

由于本次总统选举舞弊导致的巨大争议，数十万的美国民众涌向华盛顿，华盛顿的警方已经采取了各种积极的措施来应对，而国会大厦又是本次华盛顿抗议活动的中心，警方更是防备森严，然而抗议者们居然轻松闯入了国会大厦，不得不让人生疑。更凑巧的是就在昨天，川普总统刚刚宣布要求将安提法列入恐怖组织。这一幕则让我想起了希特勒当年利用所谓的国会大厦纵火案最后实现了他在德国进行独裁统治的历史。

1933年1月30日，希特勒由多个右翼政党支持被选为德国总理，他领导的纳粹党在议会584席中占有196个席位。希特勒希望国会可以通过魏玛共和国宪法规定的《授权法》，这样他就可以不通过议会，自行制订规章以代替法律。可是修宪需要国会三分之二的多数议员通过才能生效，当时左翼的社会民主党是议会中第二大党，占有20%席位（121席），第三大党德国共产党占有17%席位（100席），这两个党派都坚决反对通过该《授权法》。

希特勒为了达到目的，于1933年2月27日制造了国会大厦纵火案，并于现场抓获了纵火犯——荷兰共产党人范·德·卢贝。事件发生后，他立即发动媒体攻势，宣称这是共产党发动革命的信号，大力宣传只有通过《授权法》才能制止共产党发动革命，否则德国就会处于共产党的恐怖统治中。随后希特勒于3月1日选举前将德国共产党列为非法。3月23日国会复会后，《授权法》顺利通过，德国的立法权、宪法修改权以及同外国缔约的权力全部交到了希特勒政府的手上，魏玛宪法名存实亡。随着1934年8月2日兴登堡总统的离世，希特勒成为德国的唯一领导人，一个大独裁者。

1939年9月1日，纳粹德国入侵波兰，挑起了第二次世界大战。在这场人类历史上规模最大的战争之后的70多年里，中共依靠欺骗和蓝金黄的手段慢慢地渗透进美国政治和经济的每一个细胞，控制了美国华尔街，控制了硅谷以及所谓的主流媒体和社交媒体。

就在国会大厦骚乱发生后的第一时间，美国所有的主流媒体立刻开动宣传机器，大肆渲染那些少数冲进国会的人是川普支持者，并将“暴徒”的帽子扣在了和平抗议的川普支持者头上。而就在同一时间，川普总统的推特和脸书账号被封，被禁止发声。对比这些主流媒体和社交媒体在安提法和黑命贵的打砸抢活动中默不作声只作壁上观的表现，我们有理由怀疑这次所谓的“国会大厦骚乱事件”只是中共策划的一场阴谋。

在骚乱之后继续的国会辩论和投票中，支持川普总统的议员们对亚利桑那州选举结果的挑战，在参众两院最后都以失败告终，而这一结果与发生在国会大厦的骚乱有着必然的联系，这正是中共所需要的结果。
