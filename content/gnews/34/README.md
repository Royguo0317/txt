爆料革命已掌握中共生物武器的重磅证据，包括习亲笔签名的文件及相关影音资料，中共不可能逃脱正义的审判。

Ⅶ.参议员科鲁兹最新福克斯采访，引用美国1876年大选案例，并提议成立“选举委员会”。川普总统很可能通过彭斯副总统1月6日的判断取得胜选，而不必使用“选举委员会”策略。

2.1月4日是香港及中共国2021年股市开盘第一天，受美国纽约证券交易所摘牌影响，中国移动等三大运营商股票大跌。此前一天，中共国证监会表示“即便摘牌，对公司发展和市场运行的直接影响相当有限”，疑稳定市场信心。

3.日本《产经新闻》1月3日报道，首相菅义伟接受专访时表示，中共希望参加《跨太平洋伙伴全面进步协定》（CPTPP），但以中共国现行的政治、经济体制来说，要加入很难。他称，CPTPP规定要求成员国必须具有非常高水准的市场开放程度，“以中共国现行的政治和经济体制很难参加。”但菅义伟表示期待印度加入。
