# 中共国驻英大使刘晓明离职

**批评英国禁止华为5G参与在英国建设的刘晓明被中共国外交部副部长代替。**

[!\[\]()!\[\](https://spark.adobe.com/page/wwlAiBmFPvJZ7/images/3858b06f-9530-4d6a-bd2c-c4e3b6d574e5.jpg?asset_id=921b3fa8-4de7-408a-8abf-4e321a85c961&amp;img_etag=%228267363aa80f9a94123f73232208ebeb%22&amp;size=2560)](https://spark.adobe.com/page/wwlAiBmFPvJZ7/images/3858b06f-9530-4d6a-bd2c-c4e3b6d574e5.jpg?asset_id=921b3fa8-4de7-408a-8abf-4e321a85c961&amp;img_etag=%228267363aa80f9a94123f73232208ebeb%22&amp;size=1024)刘晓明不断地否认维吾尔族穆斯林被强迫关入新疆集中营，摄影: Simon Dawson/路透社

中共国长期担任驻英大使刘晓明是英中经济更紧密关系和在香港实施新安全法的坚定捍卫者，他的卸任标志着两国关系在2015年达到高峰，但此后明显恶化的时代的结束。

刘晓明将由中共国外交部副部长郑泽光接替，郑泽光曾是卡迪夫大学法学院的学生，曾有传言将成为中共国驻美大使，并且仍然被视为几年后该职位的候选人。

刘晓明今年64岁，从2010年开始担任大使。在大多数大使任期为四年的情况下，他的是个马拉松式的任期。
[!\[\]()!\[\](https://spark.adobe.com/page/wwlAiBmFPvJZ7/images/18a821da-ec89-4664-98d5-c746bf384b93.jpg?asset_id=f51cc08a-6314-415e-87a9-4fcc797797d4&amp;img_etag=%225fe6cc7939ba9517fd457b9ff67a7f6f%22&amp;size=1024)](https://spark.adobe.com/page/wwlAiBmFPvJZ7/images/18a821da-ec89-4664-98d5-c746bf384b93.jpg?asset_id=f51cc08a-6314-415e-87a9-4fcc797797d4&amp;img_etag=%225fe6cc7939ba9517fd457b9ff67a7f6f%22&amp;size=1024)2010年5月26日，英国女王伊丽莎白二世在伦敦白金汉宫私下会见中国大使刘晓明。（Picture Source/图片来源 CNN）
他对英国禁止中共国公司华为使用5G网络的决定一直是一个激烈的批评者，还多次否认维吾尔族穆斯林被强行关进位于新疆维吾尔自治区的集中营。

中共国今年六月实施新的香港主權法之前，中共国和英国之间的关系可以说已经有所下滑，但是被英国视为违反了《中英联合声明》的新《香港国安法》，为两国关系的重大变化铺平了道路。

在中国6月对香港实施新的主权法之前，中英关系可以说已经开始下滑，但被英国视为违反中英联合声明的新法，为中英关系的重大变化铺平了道路。

刘晓明在BBC第一频道(BBC One)的《安德鲁·马尔秀》(The Andrew Marr Show)上受到了更广泛的关注，当看到维吾尔族穆斯林被蒙上眼睛并跪在地上的视频时，他声称这些图像是“伪造的”。

在华为风波中，刘晓明表示，英国可能会 “承担将中国视为敌对国家的后果”；”英国能否为其他国家的公司提供一个开放、公平和非歧视性的商业环境已经成为问题。”

刘晓明同时指控英国政府发表“对香港不负责言论”，在一系列咄咄逼人的新闻发布会上坚称，香港的抗议游行和自由无关，而是由不负责任的外国人挑起的。

在第三届中英经贸论坛上，他在最后一次定调演讲中表示，政治关系的下降并没有出现经济关系的放缓。刘晓明说：“2019年，中英贸易额达到了历史新高862.72亿美元（约合644亿英镑）。在今年的前十个月，中英货物往来同比增长了2.8%，超过同期中共国对外贸易总体增速”。2018年，刘晓明获得了伦敦金融城(City of London)的自由权，这标志着英国金融与中共国之间的密切联系。

刘晓明职业生涯的顶点应该是2015年，当时习近平总书记对英国进行了一次“超级国事访问”，宣布中英关系进入“黄金时代”。

英国明年将在格拉斯哥主办第26届联合国气候变化大会(Cop26)，为实现联合国在11月会议上制定的更雄心勃勃的目标，英国迫切需要中共国与世界各国合作，这位新大使将对中英两国就气候紧急状况进行的讨论至关重要。

英国从1月31日起向香港持有英国海外公民护照(British National Overseas, BNO)提供最多5年的工作和学习权利，新的大使同时要处理这一后果。BNO持有者可以六年后申请定居，然后获得英国公民身份。英国提供的权利可以由BNO持有者的成年子嗣、配偶以及年轻的孩子享有。英国同时在今年7月底终止了与香港的引渡协议，抗议北京强行实施香港国安法。但是，英国还没有对任何中共打压香港的官员实行任何制裁。

郑泽光的任命首先由《南华早报》曝光，但没有被英国官方宣布，中共在驻英大使的位置上放了一个研究近期中美对抗的专家。他从1986年开始外交生涯，1990年起专注于美国问题二十多年，2012年底但任外交部部长助理，2015年但任副部长。

郑泽光是中共回击川普政府对中共处理冠状病毒的行动上的前沿官员，他曾说一少部分美国官员让美国转向背对中共国。

他指控美国是香港民主运动背后“最大的黑手”，他最近说：“美国根本不在乎香港的民主、人权与自治；美国只想搞乱香港和破坏中共国的平稳发展”。他认为美国制裁中共侵犯香港人权的官员是野蛮的”。



中共国预计英国在脱欧后会受到冲击，而且很可能会处于美国政府的庇护之下，因此，郑泽光对美国政治的了解，将有助于他应对鲍里斯•约翰逊寻求扮演“全球英国”新角色的问题。
