## 【香港要闻】团体反对港府强制检测 质疑借此进一步侵犯人权 前线护士担忧损医患互信

整撰：文燕

覆核：文非
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/16034158/image001-32.png)多名区议员及公民团体人士、护士等召开记者会，反对政府强制检测。 （图片来源：大纪元）
据【大纪元】11月15日讯，政府昨日（14日）订立《预防及控制疾病（对若干人士强制检测）规例》（第599J章），于今日（15日）生效。公民中心联通多名区议员及医护界人士召开记者会，反对强制检测，担心政府借此进一步侵犯人权。有抗疫前线护士表示，强制检测损害医生与患者之间的互信，或令部份轻症病人抗拒看医生。

政府昨日刊宪的强制检测规例，授权政府要求某些人群接受中共病毒（武汉肺炎）测试，也授权医生出具书面指示要求某人检测。违反强制检测者将被定额罚款2,000元，并收到强制检测令，如果再不遵从，可被罚款25,000元及监禁6个月。

公民中心成员欧赞年宣读声明指，政府应该为市民接受病毒测试提供方便条件，而不应该订立刑罚，强迫市民检测，侵犯人权。他也批评政府做此决定前没有公众咨询，甚至没有咨询受影响最大的医学界的意见，就将医生拉进政府的人身管控流程。

曾在亚博馆抗疫的香港专职医疗人员及护士协会干事刘凯文批评，政府强制检测对医患关系「百害而无一利」。他担心，有轻微症状的病人不想面临强制检测及罚款、监禁，可能因此拒绝求医，反而不利防疫。

潘朗聪还指，政府推出「安心出行」APP，可以记录人的行踪，「是迈向健康码、全民监控的开端」。虽然政府声称这个APP是自愿安装。潘朗聪说，在中国大陆，健康码也是所谓「自愿安装」，但是没有健康码就不能进出很多公共场所，失去出行的自由。

外界也担心政府推行强制检测，为全民检测，以至于建立DNA资料库铺路，加强监控市民。

西贡区议员范国威说：「现在这是第一步，在特定群组、特定处所某类人士被政府定义为需要强制检测。观乎政府过去的行事模式，我们担心，未来可能全民都要强制检测。」
