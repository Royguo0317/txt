![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/24223314/logo-3-1.jpg)



作者：香草山写作组 叮咚猫

据《我们爱川普》报道，感恩节前夕，美国律师林‧伍德向佐治亚州第十一巡回上诉法庭提出紧急动议，要求对其诉讼进行快速审查。

动议称，由于在2020年大选中存在大量的非法邮寄选票，佐治亚州的选举过程存在缺陷，偏离宪法。林‧伍德希望该诉讼在12月14日，即认证选举结果的那一天之前得到审理。

美国第十一巡回上诉法庭刚刚发布了该案件的日程安排：

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/26125755/%E3%80%90%E7%83%AD%E7%82%B9%E6%92%AD%E6%8A%A5%E3%80%91%E4%B8%8A%E8%AF%89%E6%B3%95%E5%BA%AD%E6%89%B9%E5%87%86%E6%9E%97%E4%BC%8D%E5%BE%B7%E5%AF%B9%E5%85%B6%E8%AF%89%E8%AE%BC%E8%BF%9B%E8%A1%8C%E5%BF%AB%E9%80%9F%E5%AE%A1%E6%9F%A5%E7%9A%84%E7%B4%A7%E6%80%A5%E5%8A%A8%E8%AE%AE.png)

被上诉人必须在2020年12月1日星期二下午5点前对上诉人的案情摘要做出回应。

上诉人的答辩状（如有）必须在2020年12月3日星期四下午5点前提交。

这使得川普总统竞选团队反转大选结果的进程又前行了一大步。
