**西班牙2021年1月15日电/西喜社——**加拿大采购部长表示，欧洲的生产问题将暂时降低辉瑞公司交付疫苗的能力。

部长阿尼塔·阿南德说，由于欧洲生产线的问题，这家美国制药商暂时减少交付量。她补充说，虽然该公司表示仍将在3月底前交付400万剂疫苗，但这已无法保证。

加拿大只收到了38万剂疫苗，本来本月应该再收到40万剂。它预计2月份会有近200万剂疫苗。加拿大希望在9月前为所有想接种的人接种疫苗。
