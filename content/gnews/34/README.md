**西班牙2020年12月12日电/西喜社——**

德克萨斯州共和党主席艾伦·韦斯特(Allen West)说，最高法院驳回德州选举诉讼案“后，守法州可能希望形成自己的“联盟”。

最高法院驳回选举案的理由是，德克萨斯州“在另一州进行选举的方式上没有表现出司法上可识别的利益。”

韦斯特说:“最高法院驳回由17个州和106名美国国会议员加入的德克萨斯州诉讼时，已裁定一个州可以采取违宪行动并违反自己的选举法，从而对遵守该州的其他州造成破坏性影响，且罪恶州不会承担任何后果。”

韦斯特继续说：“这一决定建立了一个先例，州可以违反美国宪法，且不受追责。”  “这一决定将对我们的宪法共和国的未来产生深远的影响。”

他补充说：“也许守法国家应团结在一起，组成一个遵守宪法的州联盟。”
