![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/09190948/GNEW-GTV-MOS-LOGO-1.jpeg)



摘要：从小在意大利黑手党长大的朱利安尼率团前往费城调查选举舞弊，不久传言费城黑帮老大梅利诺愿意和过去切割：如果川普总统愿意特赦他，他将出席国会听证会，证明他为拜登竞选团队伪造了30万张假选票。

2020年大选，11月3日的计票当日，群魔乱舞，摇摆州纷纷停止计票。4日，身为川普私人律师的前司法部次长、前纽约市长鲁迪·朱利安尼(Rudy Giuliani)表示，由于宾州的费城出现了大规模选票欺诈，律师团队将紧急赶往费城办案。

朱利安尼生于纽约布鲁克林的意大利裔社区，在典型的罗马天主教家庭长大。朱利安尼的父母都是意大利移民的子女，他父亲哈罗德·朱利安尼(Harold Giuliani)做管道工和酒保，没有什么正当职业，因为被控骚扰和抢劫的重罪，曾入纽约著名的新新惩教所。出狱后，他给亲戚的黑帮打工，经营高利贷、博彩等行业。从小在意大利黑社会耳濡目染的朱利安尼，日后读了纽约大学法学院，成为司法部次长，又在纽约市长任内成为彻底打败意大利黑手党，成为迫使纽约黑帮从良的传奇人物。

宾夕法尼亚(Pennsylvania)，坐拥20张选举人票的长期摇摆州；费城(Philadelphia)，意为友爱，是美国民主的发源地之一，美国独立战争时起草独立宣言和联邦宪法的地方。宾州的摇摆取决于人烟稀少的农业郡选民和费城、匹兹堡这样的大城市的左派选民。而这次大选的舞弊中心就是费城。

乔伊·梅利诺(Joey Merlino)，外号瘦乔伊(Skinny Joey)，意大利裔。在维基百科上，他的职业是费城黑帮老大(crime boss)。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/17030140/4BA4957400000578-5665337-image-a-1_1524922356900.jpg)费城黑帮老大梅利诺 图片来源：AP
意大利裔的超级检察官朱利安尼来到费城，费城的意大利裔黑帮老大老乡作何感想？2020年11月14日，网络上开始流传来自美国黑手党内部的传闻，瘦乔伊将去国会作证，揭露拜登在大选中舞弊的内幕。

有消息称，宾州的政治团体和瘦乔伊合作，以10美元一张的成本做了30万张投给拜登的选票，全部用现金支付。瘦乔伊的黑帮组织以每小时1000美元以上的高薪雇人干了这些脏活，以每小时几千张选票的速度赶工了60多个小时。还把这些选票用记号笔做好标记，装在没有标记的纸板箱里，放到费城会议中心的外面。

据称，瘦乔伊已经放话出来，如果川普愿意全面赦免他的罪行和相关记录，他将在国会面前提供令人震惊的证词，来说明拜登是如何窃取选举的。川普律师团队中的名人、脱口秀主持人和畅销书作者约当·色库洛(Jordan Sekulow)发推确认此事，并上Newsmax节目谈了宾州选举舞弊的情况。

这件事最初由The Buffalo Chronicle独家披露，并由色库洛转载。据了解他的人说：

黑帮人物也渴望正常人的生活。除了良心正义的觉醒，他也有个人的梦想。他想要干净的记录，想在联邦土地上钓鱼狩猎，想在国家公园管理局工作。他需要一个干净的记录才能得到这些东西。

他开玩笑说他并不介意因此成名，如果他可以挺身而出为这场惊天弊选大案出庭作证，他可能会获得书和电影的版权。他非常有魅力，并且知道如何讲一个神奇的故事。他可以成为超级明星和一个真正的名人。他可能会以此机会开发和推出一系列节目，这将可能会为他带来不菲的收入。

如果以上消息全部可靠，这件事意味着美国庞大的黑帮组织开始认清形势，转入倒拜登阵营，和自己不干净的过去切割，正视更好的未来。抽干沼泽的历史潮流，连真·黑社会也开始认真对待了。
