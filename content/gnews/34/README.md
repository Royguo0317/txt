朱利安尼的法律行动会有效果吗？在某些西方国家领导祝贺拜登当选后，一些共和党人也公开表态倒川。与此同时，我们也看到了一些有意思信息，如郭先生9日的盖特：

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/10225224/Screen-Shot-2020-11-11-at-11.52.00-AM.png)
SCIF是什么？路德时评昨天做了解说，那是敏感间隔信息设施的监控室，是美国军事国家安全、国防和情报方面的，发生最重要情况时会在那里监测、监控。它是建筑物内的封闭区，可以永久或暂时设置，可以在官方的政治建筑物或在船上、在官员私邸或酒店客房等必要的地方，紧急时甚至可以快速设立便携式的SCIF。艾森豪威尔大厦里头有一个，小布什总统曾经在那里处理对恐怖分子的作战。路德根据郭先生引用的那条推文说，SCIF对此次大选过程所有的作假做了实时监控。监控计票所用的叫Scorecard和Hammer的两个软件（软件现在被推特屏蔽了）。联邦警卫队至少在16个州全天候执行任务，诱捕参与选举作弊的人员，所有涉及欺诈的摇摆州都在列。

谁在执行之一行动呢？FBI、CIA当然不配合川普总统，另有一个由朱利安尼负责执行的FISA法案，原来就建有100条监控线。比如它可以监控突然多出的13万选票，其中有多少万张是假选票全都可以统计出来。问题是这样的监控合不合法？其程序是否有合法性？这个监控能不能用于在法庭作证？视频、录音是否合法获得？其重要前提就是有没有外国组织参与。如果背后涉及外国势力，有外国资金来源，FISA法案执行人员所获资讯和情报的合法性就不成问题了。
