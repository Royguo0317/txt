PPI和CPI均一路快速上行至通胀区间。（需要引起注意的是：CPI环比连续4个月为负值，显著是受到了猪肉价格一路下行的影响。后续需要密切关注PPI的上涨是否能成功传递到CPI上。CPI环比为负值，则意味着之前的PPI一路上行并没有十分成功地传递到CPI上，因为PPI上涨及上游生产成本的传递导致部分商品价格的上涨因素并没有有效抵消掉猪价的下行因素。）

这是自5月5日的五天内通报的第二个新增感染群
另，卫生部昨天通报，“62451感染群”新增3人，原有40人，一共43人。10起本土病例当中至少4起初步判定为印度变异病毒感染
在昨天通报的10起本土病例当中，至少4起初步判定为B.1.617病毒株（印度变异）检测阳性，正在复核：
1. 51岁中国籍妇女（第62971例），樟宜机场T3清洁工
2. 57岁新加坡籍男子（第62972例），樟宜机场T3辅警
3. 38岁新加坡籍男子（第62969例），Grab私召车司机
4. 40岁新加坡籍女子（第62998例），地产公司风险评估员
上述机场清洁工和机场辅警都属于“62873感染群”（机场人员），该感染群其他6人是否感染B.1.617，尚未说明；除此，日前通报的如“61822感染群”（港口工人），最早的确诊病例也是B.1.617病毒株（印度变异）感染。因此，实际上感染B.1.617的人数肯定远远大于此。
