## 中共超限战灭港行动（卅八）伞后组织 扩阔政治光谱

#### 搜集/撰文：岁月如歌 / 封面合成：文粤


![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/21041103/6213.jpg)
雨伞革命后，香港社会纷纷涌现各种独立团体，这些团体多达几十个，分成两大系列：金钟系和本土系。这些团体都围绕雨伞革命精神传播下去。港府政改是与否已经不重要，人们对真普选的追求一刻也没有停止。雨伞革命催生了香港新一代年青人从政热情，在社会不公面前他们认为有必要站出来担当，和老一辈民主人士共同负起责任。

这些团体组织有不少强调“重返社区，深耕细作”、继续争取民主，宣扬真普选理念。更有的团体就是冲着2015年区议会选举而立的。在金钟系和本土系都有参与2015年区议会选举和不参与之分。金钟系参与竞选的团体有11个，而本土系多达20个。不管参与还是不参与选举的团体其目的都一样，就是对抗建制派。大家都知道议会的重要性。掌握议会话语权非常重要。此次选举将会选出香港十八区区议会共431个民选议席，之后，新一届区议会将不再设有委任议员。连同27个当然议员，合共458个议席。当选的区议会议员任期为四年，由2016年1月1日至2019年12月31日。对于泛民主派来说是个机会，从临时立法会到现在都是建制派议员占绝大多数，有绝对的优势为中共效命。也就是说，立法会一直是被中共掌控。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/21041117/6214.jpg)图片来自大纪元
为了改变这种状况雨伞革命人士纷纷收起雨伞，落区参政。 「不少雨伞人走入社区深耕细作，众多在伞后成立的新团体开设街站，来街头伞聚。由专业人士组成的杏林觉醒、法政汇思拒绝离地；维修香港组织社区服务队，借免费维修向基层推广民主；鸡蛋同盟、湾仔好日志、港民领域等凝聚同路人建设自主社区；民主种子遍地开花」。民主同路人以不同的方式落区深耕，有的在地区小店或者小商铺摆放政治议题文宣单张；有的在宣传雨伞革命视角文化；有的年轻人利用下班时间集中在一起，探讨民主道路；更有的团体都在利用业余时间落去普及法律知识和选举制度，给市民讲解分析功能界别等立法会知识……年青人或在自己熟悉的社区默默耕耘，或“空降”在自己不熟识的选区也不介意，只因不甘心建制派自动当选而临时空降出战做“雷霆伞兵”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/21041126/6215.jpg)图片来自voa
伞兵（伞后人士）的出现也是应时代产物，受雨伞运动启发，在雨伞运动期间或之后参政，延续雨伞运动的精神，他们大部分认为香港议会已是个摆设。伞兵有别于传统政客，认为主要抗争战线放在街头。 2015年区议会选举前，香港社会可谓「百家争鸣」。当然，面对伞后组织如雨后春笋，百家争鸣，共匪及其走狗建制派们也一刻不闲着，他们也在想办法对抗这些新生派。了解中共特性的人都知道中共对付个人就是造谣抹黑攻击，对付社会动乱最厉害的一招就是掺沙子，百年了一直都在用这一招，而且屡试不爽，成本低效果好。这次小小的选举，对手采用的招数就是掺沙子。

2015年8月政团青年新政召集人梁颂恒到廉政公署备案，称有一位自称是网台主持人名叫Anthony Cheng郑永健的男子，主动接触他们称有财团资助他们出选区议会，挑战冯检基、尹兆坚等主要是泛民任议员的选区，每参选一区可获资助15万港元。香港有线新闻卧底走访该男子得出同样信息，该男子称不需要你赢只需要150到200票能起决定作用就行。钱花光也不用还给有大中华、中港台资金背景的公司。给的选区也以泛民主派选区为主，并称是为了防止境外势力影响香港政治。一个月后该男子被廉政公署检控。

由于伞后团体、伞兵太多，一般市民难以认识，对手鱼目混珠，把水搞浊，把事情搞乱，这些招数一看就知道是中共劣质的手法。区议会选举报名就出现了近10名“假伞兵”，他们打着黄丝带旗帜，无政纲，部分人用同一个手机号码，他们只在泛民与建制选情咬紧的港岛选区出击，被市民批评“搞搞阵”（捣乱，鱼目混珠），泛民质疑假伞兵旨在分薄泛民票源。这些现象也只有中共这帮没素质的流氓才能做得出来，长期做坏事不打草稿，总露出马脚，不管过去的双十暴动，六七暴动，还是现在的参选造假，后来的反送中运动更是层出不穷的掺沙子做恶，这些都能看到中共无处不在的魔影及卑劣的手段和无耻的嘴脸。
