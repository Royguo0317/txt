保守党领袖艾林·奥图尔（Erin O’Toole）12月14日在接受加拿大知名独立媒体TRUE NORTH 采访时指出，中共政权正在从特鲁多总理“软弱”的外交政策和“天真”的亲共态度中大受其益。

他在年底的这次媒体专访中表示，“中共政权一直利用加拿大的羸弱猖獗横行。而事实上，特鲁多他关于共产主义政权真正本质的观念可能太过偏颇。”

保守党领袖的此番抨击言论是在反政府独立媒体 REBEL NEWS REPORT 日前报道了特鲁多政府官员试图阻止加拿大军方领导层决定终止与中共党卫军原计划在安大略省军事基地进行冬季协同战斗训练后的几天后所发出的。

随着该报道的迅速发酵，国防部长哈吉特·萨杰扬（Harjit Sajjan）不得不迅速公开确认，加拿大不会再计划与中共国进行任何相关军事训练。但奥图尔（O’Toole）仍重申，与中共党卫军的军事合作训练从来都不应该放在加拿大政府的议事日程之内。

他为此解释说，“在我们两位公民无辜被中共政权逮捕入狱，在中共国大肆筑建维吾尔人集中拘留营，在有着30万加拿大公民居住的香港实施中共警察管制的种种恶劣情况下，允许在加拿大的领土上训练中共国军队的决定是非常恐怖且危险的，此举无疑也是特鲁多政府最致命的天真。”

加拿大全球事务部的一份秘密备忘录警告说，中共国有可能视此冬季军训取消之举是针对孟晚舟羁押案有关的报复性举措。加拿大此前应川普政府的要求逮捕了华为首席财务官孟晚舟并试图将其引渡至美国受审。

在奥图尔（O’Toole）提出该话题的同时，自由党副总理克里斯蒂亚·弗里兰德（Chrystia Freeland）也对外笼统地回应了关于要求中共遣返加拿大人迈克尔·科夫里格（Michael Kovrig）和迈克尔·斯帕沃尔（Michael Spavor）的重要性，他们因孟晚舟一案而遭中共政府报复性逮捕羁押，至今仍被拒绝进行公正审判，也不允许加国领事合理探访。

True North 的记者安德鲁·劳顿（Andrew Lawton）对此表示，特鲁多政府对中共国的一贯方针就是“维持原状”。

奥图尔（O’Toole）就此强调，自由党的绥靖治国方针并不能把加拿大带回正常发展的轨道，他认为必须拥有强大的实力来与中共霸权针锋相对。

他对劳顿说，“我并不是说立即开始对中共国亮剑示强，但我们必须勇敢站起来，去直面中共政权的威胁并向其证明加拿大的核心价值不可贱卖。对中共国而言是很大的贸易机会，我其实更主张‘加拿大制造’，我希望在加拿大这里做事，少依赖一个无任何诚信可言的邪恶政权。”

在接受采访时，奥图尔（O’Toole）还提及了特鲁多（Trudeau）在2013年所发表的歌颂中共国“基本专政”的那次臭名昭著的发言。

他为其保守党在对中共国一贯所秉持的强硬立场而感到自豪，包括最近在下议院获得通过的针对“华为”的一项动议。该动议给予加国政府30天的期限来决定是否允许“华为”这一中共党属企业在加拿大销售和提供其5G通讯设备及相关服务。
