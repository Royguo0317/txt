## 在朝鲜发出警告后，国防部长拜登警告美国军队准备“今晚作战”

国防部长劳埃德·奥斯汀周四警告朝鲜，在独裁政权谴责韩国正在进行的军事演习后，美国军队已经准备好“今晚作战”。

奥斯汀在星期三的新闻发布会上讨论了美军对“朝鲜半岛无核化”和“保持军事准备的重要性”的承诺。

国防部长说:“我们的军队仍然准备‘今晚作战’，我们继续朝着最终将战时作战指挥权移交给(大韩民国)指挥的未来联合部队指挥的方向取得进展。”他接着说:“尽管满足这一过渡的所有条件需要更多时间，但我相信这一进程将加强我们的联盟。”

本周早些时候，朝鲜领导人金正恩的强大妹妹警告美国“不要制造麻烦”，同时批评美韩军演，她的政府认为美韩军演是入侵演习。

朝鲜在世界人民的眼里，一直都是茅厕里的石头，又臭又硬。但是在川普总统在任上期间，特别是在任期的最后一两年，在苏莱曼尼被无人机斩首后，朝鲜的导弹也不发射了，叫骂声也停止了，甚至金三胖很久都没有出现在电视机的面前。金家三代统治朝鲜数十年，靠的不是“又臭又硬”，而是需要有这样一个强硬的态度来维护金家的统治，当真正的威胁出现后，他也只能偃旗息鼓装孙子了。可是拜登上台后，朝鲜像是吃了补肾药一般，又硬起来了，朝鲜到底怕的是谁，一目了然。
