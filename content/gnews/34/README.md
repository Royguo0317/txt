# 重要事件

- **[05.08 Zero Hedge: 中共军方在2015年讨论将COVID武器化，以使敌人的医疗系统崩溃](https://www.zerohedge.com/covid-19/chinese-military-discussed-weaponizing-covid-2015-cause-enemys-medical-system-collapse)**


2015年，中共军方科学家讨论了如何将SARS冠状病毒武器化，这比COVID-19大流行病在中共国武汉出现的时间早了五年，中共科学家与一个美国资助的非政府组织合作进行所谓的 “功能增益 “研究，使蝙蝠冠状病毒更容易感染人类。

据《周末澳大利亚人报》（新闻集团的子公司）报道，在一份由中共人民解放军科学家和中共国公共卫生高级官员撰写、美国国务院在调查COVID-19起源时获得的长达263页的文件中，解放军科学家指出，在生物武器攻击期间，需要住院治疗的病人突然激增，”可能导致敌人的医疗系统崩溃。

它认为SARS冠状病毒可能预示着一个 “基因武器的新时代”，并指出它们可以 “被人工操纵成一种新出现的人类疾病病毒，然后以一种前所未有的方式被武器化和释放出来”。

英国和澳大利亚外交事务和情报委员会的主席汤姆-图根哈特和詹姆斯-帕特森说，这份文件引起了人们对中国在COVID-19的来源方面缺乏透明度的重大关切。

这份题为《非典型肺炎的非自然起源和作为遗传生物武器的人造病毒新物种》的中文文件概述了中国在生物战研究领域的进展。

“继其他科学领域的发展之后，生物制剂的投放也取得了重大进展，”报告指出。

“例如，新发现的冻干微生物的能力使储存生物制剂和在攻击时将其气溶胶化成为可能。”

根据澳大利亚战略政策研究所的国防大学追踪报告，其中10名作者是隶属于西安空军医科大学的科学家和武器专家，该大学因其国防研究水平而被列为 “非常高风险”，包括其在医学和心理科学方面的工作。

空军医科大学也被称为第四医科大学，在2017年习近平主席的军事改革中被置于解放军的指挥之下。该报主编徐德忠在2003年非典疫情期间向中国军委和卫生部的最高领导层报告，向他们通报了24次情况并编写了三份报告，根据他的在线传记。-《澳大利亚人报》

“我们能够验证它的真实性，它是由特定的解放军研究人员和科学家撰写的文件，”根据报告，数字取证专家罗伯特-波特说，他曾为美国、澳大利亚和加拿大政府工作–并且以前曾分析过泄露的中国政府文件。”我们能够在中国互联网上找到它的起源。”

美国前国务卿迈克-蓬佩奥和他的首席中国顾问余万里在《华尔街日报》2月的一篇专栏文章中提到了这份文件，写道：”2015年解放军的一项研究将2003年SARS冠状病毒的爆发视为外国势力发动的’当代基因武器’。”

而据澳大利亚战略政策研究所执行主任彼得-詹宁斯称，”研究能力没有明确的区分，因为它是用于进攻还是防御，并不是这些科学家会做出的决定，”他补充说，”如果你表面上是为了保护你的军队免受生物攻击而培养技能，你同时也是在给你的军队提供进攻性使用这些武器的能力。你不能把这两者分开。”

该研究还研究了释放生物武器的最佳条件。”生物武器攻击最好在黎明、黄昏、夜晚或阴天进行，因为强烈的阳光会破坏病原体，”它说。”生物制剂应在干燥的天气中释放。雨或雪会导致气溶胶颗粒沉淀。

“最好有一个稳定的风向，以便气溶胶能够漂浮到目标区域。”

军方科学家最离奇的说法之一是他们的理论，即造成2003年SARS疫情的SARS-CoV-1病毒是一种人造生物武器，由 “恐怖分子 “故意释放到中国。-澳大利亚人报

据《大纪元时报》报道，该文件的消息是在5月3日报道武汉病毒研究所与中共国政府合作的一个团队，该团队由五名军事和民事专家组成，”他们在武汉病毒研究所实验室、军事实验室和其他民事实验室进行研究，导致 “在野生动物中发现动物病原体[导致疾病的生物制剂]。

正如我们在3月份所指出的，由安东尼-福奇博士领导的美国国家卫生研究院（NIH）”资助了一些涉及WIV科学家的项目，包括武汉实验室在蝙蝠冠状病毒方面的大部分工作。”

据《每日电讯报》报道，2017年，福奇的机构恢复了对一项有争议的资助，在未经政府监督机构批准的情况下，在中国武汉对蝙蝠冠状病毒进行基因改造。作为背景，在2014年，奥巴马政府暂时停止了对蝙蝠冠状病毒的功能增益研究的联邦资助。在该决定的四个月前，美国国立卫生研究院通过向彼得-达斯扎克领导的非营利组织生态健康联盟拨款，有效地将这项研究转移到武汉病毒研究所（WIV）。

2014年6月，NIH在 “了解蝙蝠冠状病毒出现的风险 “项目下，支付了EcoHealth 370万美元赠款中的第一笔666,442美元，并在2019年5月之前每年支付类似款项。

值得注意的是，据《华盛顿邮报》的乔希-罗金（Josh Rogin）报道，多年来，在施’蝙蝠女’郑莉博士的领导下，世界疫苗组织 “公开参与了与美国大学和机构合作的功能增益研究”。

因此，现在我们有一份来自中共国军方的2015年文件，描述了将COVID作为生物武器使用–在COVID-19大流行病爆发的四年前，距离一个致力于使蝙蝠COVID更容易传播给人类的中国实验室只有几英里远，如果你认为它们可能有关，你就是一个兜售 “被揭穿谎言 “的阴谋理论家。

对于那些说 “COVID-19不可能是人造的，因为实验室制造的病毒会有被操纵的明显迹象 “的人来说，恰恰相反。正如尼古拉斯-韦德三天前在《原子能科学家公报》中指出的那样，”较新的方法，即所谓的 “无籽 “或 “无缝 “方法，没有留下任何定义的痕迹。其他操纵病毒的方法也是如此，如连续通过，将病毒从一种细胞培养物重复转移到另一种。如果一种病毒被操纵了，不管是用无缝隙方法还是用连续通过的方法，都没有办法知道是这样的。”

仿佛痛苦而明显的答案就在我们面前，只是被对中共国友好的政客、大型科技公司和新闻机构的宣传所掩盖，这应该是地球上最简单的连线游戏。幸运的是，一年前还是禁忌的事情很快就会暴露在世界面前，这要感谢《原子科学家公报》，它在本周早些时候敢于打开武汉病毒的 “潘多拉盒子”。

- [**05.08 The National Pulse: 美国顶级生物防御实验室得出结论，COVID-19可能已从实验室泄漏，披露了机密研究。**](https://thenationalpulse.com/breaking/bio-lab-concludes-covid-could-have-leaked-from-lab/)


美国一个顶级生物防御设施对COVID-19的起源进行的一项机密研究得出结论，该病毒可能已经从中共国的一个实验室泄漏。

辛克莱尔首先报道的这份先前未披露的报告是由能源部首要的生物防御研究机构–劳伦斯-利弗莫尔国家实验室的科学家执行。利弗莫尔的 “Z部门”（该实验室的情报部门）的研究人员于2020年5月27日发布了该报告，被列为 “最高机密”。

该报告的一个关键收获是，COVID-19可能源自中国的一个实验室，强调该理论值得进一步调查。

在给辛克莱尔的一封电子邮件中，利弗莫尔的一位发言人证实了该报告的存在，但拒绝提供更多信息。公共事务主任Lynda Seaver写道：”因为你提到的报告是保密的，””我们的实验室不适合讨论这个问题。”

“消息人士说，Z部门的报告在发布时只得到了有限的分发，促使特朗普政府的一些高级美国官员询问为什么去年5月没有提请他们注意。据说能源部和国务院的高级官员进行了干预，以阻止美国低级官员的努力，这些官员对Z部门的报告很感兴趣，与撰写报告的利弗莫尔科学家联系，以寻求更多信息，”该报告补充说。

- [** 05.07 Reuters: 美国参议院中国法案将设立 “首席制造 “职位以促进研究**](https://www.reuters.com/technology/us-senate-draft-bill-would-provide-110-billion-tech-research-2021-05-07/)


根据路透社周五看到的一份131页的立法草案副本，美国参议院委员会领导人已经起草了一份1100亿美元的妥协措施，用于五年内的基础和先进技术研究和科学，并设立一个白宫首席制造官，以应对来自中国不断上升的竞争压力。

参议院商务委员会主席玛丽亚-坎特韦尔和该委员会首席共和党人罗杰-威克的修订法案将于周三由委员会进行辩论。

两党提出的 “无尽的边界 “法案将在五年内授权大部分资金，即950亿美元，用于投资关键技术领域的基础和高级研究、商业化以及教育和培训计划，包括人工智能、半导体、量子计算、先进通信、生物技术和先进能源。

这项由参议院多数党领袖查克-舒默（Chuck Schumer）、民主党人、共和党参议员托德-杨（Todd Young）等人提出的措施还将授权另外100亿美元，以指定至少10个区域技术中心，并创建一个供应链危机应对计划，以解决半导体芯片短缺损害汽车生产等问题。

修订后的版本还将创建一个经参议院确认的新的首席制造官，他将在总统的行政办公室工作，并将领导一个新的制造和工业创新政策办公室。

它还将指示商务部建立 “供应链弹性和危机应对计划”，包括 “供应链在面对冲击时抵御和恢复的能力，包括流行病和生物威胁、网络攻击、极端天气事件、恐怖主义和地缘政治攻击、大国冲突和其他威胁。”

该法案还寻求促进基础研究，以加快创新，推进关键矿产开采战略和技术，消除 “国家对供应中断的矿产和矿物材料的依赖”。

该法案草案还将阻止中国公司在没有豁免的情况下参与美国制造计划。该计划是一项由政府和公司主导的努力，旨在建立工业竞争力，减少能源使用，并加强美国国家安全。

- [**05.09 Reuters: 中共国火箭碎片将在未来数小时内重返大气层**](https://www.reuters.com/technology/us-senate-draft-bill-would-provide-110-billion-tech-research-2021-05-07/)


欧洲和美国的追踪中心周六说，中共国上周发射的最大火箭的残余物预计将在未来几个小时内重新穿过大气层。

虽然对火箭的落点仍有不同的估计，但似乎越来越有可能不会击中美国。

中共国外交部周五表示，大多数碎片将在再入大气层时燃烧，极不可能造成任何伤害，此前美国军方表示，美国太空司令部正在跟踪它所谓的不受控制的再入。

美国太空司令部估计再入将在格林尼治标准时间周日0211发生，正负一小时，而美国联邦资助的以太空为重点的研究和发展中心–航空航天公司的轨道再入和碎片研究中心（CORDS）将其预测更新为格林尼治标准时间0302两侧的两个小时，火箭将在太平洋上空再入。

欧盟空间监视和跟踪（EU SST）说，它对长征5B火箭体再入时间的最新预测是格林尼治标准时间周日0232两侧的139分钟。

欧盟SST说，在人口密集地区发生地面撞击的统计概率 “很低”，但指出该物体的不可控性使得任何预测都不确定。

报告美国太空司令部收集的数据的Space-Track估计，该碎片将在地中海盆地上空重新进入。

哈佛大学的天体物理学家乔纳森-麦克道尔在推特上说，人们相信美国在潜在的撞击中是安全的，但最近的预测仍然在追踪它从哥斯达黎加一直到澳大利亚和新西兰。

以每秒约4.8英里的速度飞行，重返大气层的时间仅相差一分钟，在地面上转化为数百英里的差异。

“这很难预测，也不是一个精确的测量，”Space-Track在Twitter上写道。

长征5B–包括一个核心级和四个助推器–于4月29日带着无人驾驶的天河舱从中国海南岛升空，天河舱内有将成为中国永久性空间站的生活区。这枚火箭之后还将有10次任务，以完成空间站。

长征五号火箭一直是中共国近期太空雄心的组成部分–从运送其计划中的空间站的模块和人员到向月球甚至火星发射探索性探测器。

上周发射的长征是5B变体自去年5月首飞以来的第二次部署。

麦克道尔之前告诉路透社，火箭碎片有可能落在陆地上，也许是在人口密集的地区，就像2020年5月那样，当时第一枚长征5B的碎片落在象牙海岸，损坏了几座建筑物。没有人员伤亡的报道。

中共国火箭发射的碎片在中国国内并不罕见。4月下旬，湖北省十堰市当局向周边县市的人们发出通知，要求他们准备撤离，因为预计碎片会在该地区降落。

“长征五号B的再入大气层是不寻常的，因为在发射过程中，火箭的第一级达到了轨道速度，而不是像通常的做法那样落入射程，”航空航天公司在一篇博文中说。

“空的火箭体现在处于围绕地球的椭圆轨道上，它被拖向一个不受控制的再入。”

自上周以来，空的核心级一直在失去高度，但由于不可预测的大气变量，其轨道衰减的速度仍然不确定。

它是返回地球的最大空间碎片之一，专家估计其干质量约为18至22吨。

去年返回地球的第一个长征5B号的核心级重达近20吨，仅次于2003年哥伦比亚号航天飞机的碎片，1991年苏联的礼炮7号空间站和1979年美国宇航局的太空实验室。
