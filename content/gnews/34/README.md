首先，我M仔在这感谢每一位旁听师和送车师，当自己亲身被困在囚车里，才体会到原来旁听师和送车师对我如此重要。还记得我第一次上庭，当裁判官拒绝了我的保释申请，那一刻我无法接受，情绪差点崩溃。当我极其绝望的时候，当囚车驶离法庭之际，我看到了很多手足，又听到大家喊「撑住啊！手足」、「坚持住啊」、「我们不会忘记你们的！」甚至有手足从法院追到了医院(手足们，你们也要注意自己安全呀)……我在这多谢你们每一位！当时我的泪水夹杂着鼻涕立刻把口罩弄湿了。回到收押所，那个画面依然挥之不去。也正因为你们的支持，才让我可以继续支撑。看守所里的生活虽然没有想象中那么难过，但每天重复着同样的节奏，而另一方面是对未来的茫然，刚开始时我的情绪非常反复，都是靠大家的支持和鼓励做为支撑下去的动力。在第二次提堂离开时，法庭外有大量警员巡逻，但仍然有手足冒着随时被捕的风险送车，非常多谢每一位…但请…请注意自己的个人安全，我绝不愿意看到你们傻傻地受伤和被抓。
