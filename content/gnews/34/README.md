- 在英国脱欧的非常关健时刻，英国政治顾问谋略首席特别顾问卡明斯11月13日辞职离开唐宁街10号，所以英国脱欧非常危险。

- 美国对中共国31家军企的制裁，威力持续发酵。美股中共国企业陆金所单日跌幅很大，3月份软银抛售100多亿的股份，阿里巴巴是大头。
- 各大国企，煤炭、银行、大科技公司接连违约。中共国债市出问题，断血式效应。
