## 传奇球星马拉多纳去世，生前曾被中共坑取利益

**作者：京都富士会 秋末的落叶**

11月25日，传奇球星马拉多纳经抢救无效去世，享年60岁。这个令人心碎的消息让整个球迷世界陷入了无限的悲痛之中。阿根廷政府宣布全国三日默哀，为他通往星河赛场的道路送去了祝福。

在马拉多纳的职业生涯中，有着与中国慈善事业千丝万缕的联系。例如，2010年的中国之旅，马拉多纳举办慈善晚宴、慈善比赛等，为中国贫困癌症患者募款百万元。2012年参加百万慈善活动，并将100万元捐给中国儿童慈善机构。

但是，在马拉多纳与中国的渊源中，名叫唐清慧的中共政协港澳台侨工作顾问榨取了马拉多纳的利益。北京时间2017年6月，马拉多纳曾经在社交媒体中声称，“唐清慧对我撒谎，欺骗了我，中国现有的所谓马拉多纳足校，和自己没有任何关系。我也将与唐清慧面对面交流，让她知道自己是个多大的骗子。”。

原来，唐清慧以马拉多纳的名字注册了一家叫做马拉多纳足球学校的公司。这一举动，使得老百姓抱着有望被马拉多纳指教的幻想将孩子送去学校，从而受骗。这种欺骗，也让马拉多纳的名誉饱受争议。而唐清慧给予的反应竟是“马老师，发生肾么事了？”随后事情不了了之。而到目前为止，唐清慧依然打着马拉多纳的名义办学，并且一切媒体舆论无一敢说道说道事情的来龙去脉。

一边是将足球视为国家命脉的阿根廷，一边是以唐清慧为代表的中共骗钱委员会，两者有着本质的不同。然而，两个天壤之别的体制，为何能让他们的足球变为生意并走到一起？中间是谁牵的线？难道中共足协主席陈戌源不知道吗？而陈戌源在任职前，竟然是上海国际港务股份董事长，一个弄港务的，他会踢球吗？是谁给他的权力？恐怕这背后都是上海港务集团背后的上海证券给予的权力。

所以，从一开始，足球坑取马拉多纳利益只是开头，而随后中共借此名义的足球渗透，才是背后的核心力量，好一个“足球外交”。所以在2017年同年，也就是中共明目张胆的对马拉多纳的发声指责置之不理的同年，中共宣布，2017年双边贸易额为138.09亿美元，其中中方出口90.67亿美元，进口47.42亿美元，同比分别增长12.1%、25.9%、–7.4%。目前，阿是中共在拉美第五大贸易伙伴，中共是阿全球第三大贸易伙伴的声明。

那么，球迷朋友们，你们希望中共活几天呢？
