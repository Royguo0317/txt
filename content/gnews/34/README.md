12月13日，美联社报道，美国司法部已要求调查被美国主流媒体宣布美国2020总统大选所谓获胜者乔·拜登的儿子—-亨特·拜登（Hunter Biden）的财务信息。

司法部要求提供与亨特·拜登相关的20多个机构的信息，包括乌克兰天然气布利斯玛股份有限公司（Burisma Holdings），亨特·拜登曾于2014年4月至2019年4月期间担任布利斯玛公司董事会成员。

司法部表示，提出这一要求是对亨特·拜登的收入来源进行调查的一部分，目前尚不清楚他在乌克兰的活动是否是本案的中心。除了有关布里斯玛的信息外，司法部还要求提供有关亨特·拜登与中共国的金融关系的信息。

亨特·拜登的律师拒绝就此事发表评论。

12月10日，亨特·拜登宣布特拉华州检察官办公室正在对他进行税务调查。总统候选人之子表示相信调查人员的专业性和客观性将确认其案件的合法性。

媒体指出，调查人员主要对亨特·拜登参与的外国交易感兴趣，特别是在中共国的业务以及可能涉及的洗钱和逃税行为。

美联社报道说，此案在乔·拜登提名大选之前的2018年开始审理。但是，几个月前，司法部决定暂停调查，因为这可能会影响11月3日选举的投票结果。现在已恢复调查。

现任美国川普总统曾多次指责亨特·拜登和他的父亲参与腐败及金融犯罪。在选举前两周的10月，他要求联邦调查局对拜登家庭进行调查。在此之前，川普总统呼吁，“我们必须把拜登家族关进监狱。把乔∙拜登关进监狱，把希拉里关进监狱。拜登家族组织了一个‘犯罪企业’，就像吸尘器一样，吸入了数百万美元”。

川普总统指责乔·拜登利用他的职位来帮助其儿子亨特·拜登从事商业活动，亨特∙拜登曾于2014年至2019年担任乌克兰天然气公司布利斯玛股份有限公司（Burisma Holdings）董事会成员，当时的副总统乔·拜登正是美乌事务的主管。此外，这位前副总统还被指控向乌克兰当局施加压力，要求解雇当时正在调查布利斯玛的前乌克兰总检察长维克托·肖金（Viktor Shokin）。乔∙拜登本人否认了所有指控。

川普总统指责乔·拜登说谎，“这些电子邮件表明，乔·拜登一再声称从未与亨特·拜登谈论过生意是完全错误的。现在有无可辩驳的证据，他试图掩盖丑闻。美国从来没有一个政府比奥巴马–拜登政府更腐败。乔·拜登是一个腐败的政治家。”

美国2020总统大选于11月3日结束，美国官方要12月才会正式公布大选结果。

中共及其美国代理人乔·拜登利用控制的美国媒体，单方“意淫式”宣布在美国2020总统大选中获胜。但是川普总统及其团队依旧坚持一切走美国的法律程序，一切遵守宪法、联邦法律和州法律规定，在法律框架下揭露中共及其代理人在美国2020总统大选中的虚假选票事实。川普总统一定会赢得美国2020总统大选的胜利。

在美国2020总统大选前，爆料革命向全世界全面公布了亨特·拜登“硬盘门”事件以及与中共深度勾兑的实锤证据。目前，美国司法部全面介入，此事继续发酵中。
