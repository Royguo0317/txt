青岛市卫生健康委员会（以下简称青岛卫健委）于2021年4月11日发布了[关于紧急寻找同车厢密切接触人员的公告](http://wsjkw.qingdao.gov.cn/n28356065/n32563060/n32563061/210411192515590502.html)：

2021年4月11日，市北区疾控中心报告由南京入境集中隔离期满返青的一名人员核酸检测结果为阳性。该人员3月26日由南京入境，4月9日解除集中隔离，当日15：45由南京南站乘坐G282次列车（5车厢11F，途中19：00左右去过一次5车厢和6车厢之间的卫生间）于21:38到达青岛站。该人员抵青后，即由专车点对点接送至市北区集中隔离点进行医学观察，全程实行闭环管理，严格落实各项防控措施。

现紧急寻找上述时间段乘坐该次列车与该人员同车厢以及共用过卫生间的乘客。请相关乘客务必做好个人防护，尽快与当地疫情防控指挥部办公室或疾控中心取得联系，配合落实相关防控措施。

请广大市民不必恐慌，时刻增强个人防护意识，关注自身健康状况，做好个人防护，规范佩戴口罩，保持安全社交距离。如出现发热、咳嗽、腹泻、乏力等症状，应在做好个人防护的前提下，立即前往就近发热门诊进行筛查诊治，就医过程中尽量避免乘坐公共交通工具。

青岛市市北区疾病预防控制中心联系电话：0532-82817891（工作日8:30-17:00），0532-82817955（24小时值班电话）。

2、如果这位2021年4月11日检出的阳性患者在此前一直体内没有病毒，那就意味着核酸检测结果没问题，再考虑到人体内是不可能自己产生中共病毒的，所以这位阳性确诊者就必然是在他2021年4月9日离开南京隔离点回青岛的路上感染中共病毒的，那么自南京隔离点至抵达青岛的这一路上，他所乘坐的交通工具、他所经历的公共场到底是谁把病毒传染给了他？

在说完了这两种可能性之后，我们要再来谈谈G282次列车的整辆列车，我们要知道哪怕所有的密切接触者都无一遗漏地被找到并且都按规定进行了医学隔离，存在于G282列车的中共病毒会因为这位阳性确诊者和所有密切接触者的医学隔离而在整辆G282车厢内绝迹吗？

依上例同样的逻辑，即使这位阳性确诊者在沿路搭乘的所有交通工具、经过的所有公共场所内的密切接触者无一遗漏地被找到并且进行了医学隔离，那么这位阳性确诊者和所有的密切接触者在沿路使用的所有交通工具、经过的所有公共场所内都不会再遗留一个病毒吗？

在分析了以上种种可能之后，我们再来谈一下青岛卫健委的这则公告意味着啥？它除了具有麻痹作用之外，对于中共病毒的疫情防范不会有任何意义，再推而广之所有中共国的这种确诊阳性之后寻找密切接触者进行医学观察和检测的把戏都是闹剧，因为所有的卫健委都故意先入为主地认定某一批次检测人员中的阳性确诊者就一定是最初的潜在传染源，而从来不会怀疑会不会有漏检人员，从来不会怀疑某一批次检测阳性确诊者是由其他人感染的，因为这样一来寻找真正的传染源和密切接触者将变得更加不可能！
