自2021年5月11日中共国家统计局发布第七次人口普查数据以来，其数据的严重不合理和明显造假，就已被广泛质疑和打脸。随后，5月31日，习近平又亲自主持中共中央政治局专题会议，听取“十四五”时期应对人口老龄化的重大政策举措汇报，审议《关于优化生育政策促进人口长期均衡发展的决定》。基于对中共的认知，我们可以得出中共国人口现状的一个基本结论——中共国人口既结构老龄化又数量大幅减少。这个基本结论与文贵先生从中共统计部门获得的内部情报也是一致的。

鉴于前期针对中共第七次人口普查的数据分析的文章和结论很多，在本文笔者就不对中共造假的数据本身作过多解读，而是将重点放在中共此次数据造假的动机及背后的深层次原因上，同时也借这个时机对中共制造的人口灾难做一个简单梳理。

**一、为什么中共第七次人口普查数据广受关注**

俗话讲，有人就有一切。对于一个国家或者地区而言，制订社会福利、文教卫生、城市发展等各种政策，以及社会、经济和产业等规划都必须以人口的现状和发展趋势作为基本依据。中共国长期号称人口数量世界第一、经济总量世界第二，以及劳动力和消费市场巨大。改革开放以来，中共依靠大量低廉的劳动力，获得了经济的快速发展，同时也奠定了中共国在世界贸易、经济和供应链当中的地位。因此，中共国人口的现状和变化，就不仅仅是中共自己内部的事情，就成了关乎世界经济、金融、贸易、供应链等诸多领域发展和布局的问题，这也就是中共国的人口数据历来广受关注的主要原因。自中共病毒肆虐以来，对全球经济造成重大打击，与此同时，中共国内部的政治、社会、经济和民生等整体形势也在加速急转直下。美欧日韩等外资外企纷纷撤离，经济和产业链不断崩塌。**值此世界形势和格局面临重大变化之际，特别是全球“以毒灭共”已大势兴起、不可阻挡，中共国的人口问题，已不仅是世界各国和各种势力综合评估中共政权及其发展的重要依据，而且还成了关乎当下如何抉择，如何选边站等迫切问题的重要参考**。如对于一些外资外企而言，是继续加大投资还是缩减投资，抑或是痛下决心一撤了之；再如，对于一些力量而言，是继续与中共勾兑还是选择撇清切割，抑或是选择大势对中共反手一击，等等。可以说，中共第七次人口普查数据，因在当前这个特殊的历史大背景下，被赋予了更多的更深层次的意义和关注。

**二、中共第七次人口普查数据造假背后的主要原因**

众所周知，中共造假历来不外乎两个目的，一是粉饰太平，二是掩盖真相。中共国人口老龄化，是世界共知的事情，无法掩盖，对于这个问题中共也选择不回避，这点也可以从5月31日，习近平主持召开的中共政治局专题会议所释放出来的信息得出这个结论。由此可见，**中共第七次人口普查数据的造假的核心目的，就是为了掩盖人口大幅减少这个基本事实，进一步来讲，是为了掩盖中共国生育率低迷和中共病毒死亡人数这两个基本事实**。概括起来，中共造假背后的动机至少有以下几方面：

**（一）*****掩盖中共的无能和执政之恶*****。中共的无能和执政之恶，可以说是造成中共国生育率低的罪魁祸首，也就是说，当前的人口问题是中共一段时间或一个时期执政恶果的具体表现**。由于中共的洗脑宣传，使得这一点具有很大迷惑性，因此，笔者在这部分稍微着墨多一点。纵观中国历史，可以说，人口明显增长的时期，就是政治比较清明和宽松的时期，反之就是统治者和政治比较黑暗昏聩的时期（除去战争和战乱）。中共改革开放40多年，经济快速发展，中共也大力营造“厉害了我的国”的虚假盛景，但生育率却在不断下降，究其原因，主要有二，**一是“割韭菜”的各种政策让老百姓不堪重负、疲于奔命**。经济的发展，并没有改变中共对老百姓的“驭民五术”，反而让各种压榨老百姓的政策和手段，变得更加变本加厉，且更具隐蔽性。特别是所谓的房地产、教育和医疗，这三个涉及广大民生的行业的“产业化”，给广大的老百姓压上了新的“三座大山”，其中又以房地产为最，房价越炒越高，让整个社会充满焦虑，让老百姓疲于奔命。虽然自2013年起，中共就开始逐步放开生育管制政策，到2015年全面放开，同时出台了一些具体的鼓励生育政策，如将原法定产假98天延长了30天，将陪产假延长了5天达到15天，增加了生育补贴等，但这些鼓励生育的政策与老百姓身上的压力和负担相比，就是微不足道的。 **二是社会问题和矛盾不断积累和加剧**。中共国经济的发展，未能真正惠及老百姓，加之中共在涉及广大普通百姓利益方面的事情，中共从来没有真正下决心解决。如教育、食品安全、财富分配、社会福利、阶层固化等问题，使得中共国的社会问题，没有随着经济发展而得到化解，反而不断积聚和加剧，这也让广大普通百姓不敢生。以中共国的教育为例，户口或住房与择校入学进行绑定的问题，教育资源高度聚集、不均衡的问题，校外辅导和培训的问题，可以说，在当下的中共国“没钱、没关系、没文化、没精力”就无法养育孩子。可见，中共国老百姓且自顾不暇，生育就是再给自己制造新麻烦、增加新负担。

***（二）掩盖民生恶化的事实***。中共国的经济不断恶化，让广大百姓生育意愿不断下降，也就是说广大老百姓不愿生。当下，中共国内普通百姓能直观感受的就是，企业倒闭和失业越来越多，就业越来越难，钱越来越难挣，并且钱越来越不值钱，生活的艰难和对未来的迷茫，让越来越多人选择躺平和“随薪锁欲”。对于中共国当下民生的艰难，以美团骑手为例，2020年美团发布的《骑手就业报告》显示，截止2020年上半年（6月30日）美团拥有的295万外卖骑手中，四分之一的骑手学历在大专以上，其中本科生17万，研究生近6万人，高学历骑手的比例还在不断上升；据2021年5月的公开信息，美团平台的骑手规模达1000万人，但美团将这些骑手，外包给了第三方劳务派遣公司，目的是压低保险和其他各种劳务成本，最大程度规避各种风险。美团骑手的情况变化，就是中共国内失业规模和就业的艰难，以及民众保障的缺失的一个缩影。老百姓不愿生，生育率低，就是中共国内民生艰难的直接反映。

***（三）掩盖病毒死亡的真实数据***。这是中共需要掩盖的重要问题。中共释放病毒祸害世界的同时，也让中国人民遭受巨大灾难。但中共却通过隐瞒死亡人数，来宣传制造所谓的“大国抗疫”成果假象，来忽悠世界和国人。根据文贵先生爆料，约百万国人死于中共病毒，这与中共对外公布的5000多人的数据相比差距很大。因此，开展于2020年11月1日至2020年12月10日的第七次人口普查，就肩负了处理国人病毒死亡真实人数的任务。

**三、对中共制造中国人口灾难总体情况的简要梳理**

纵观中共的历史，就是诉诸暴力，杀自己人、杀老百姓，以及祸害人民，不断制造人口灾难的历史。由于有关历史资料可查，笔者就不做具体展开，仅对中共建政后的有关情况简单梳理如下。

***（一）中共的内斗和政治运动直接造成人口大面积死亡***。如，1950年至1951年镇反运动，造成无辜死伤72万左右；1955年至1957年肃反运动死伤29万；1957年至1958年整风反右运动上万人消失；1958年至1962年“大跃进”死亡达3700万至4300万。根据杨尚昆组织的公安部内部调查结果，实际死亡人数可能高达9600万，该数字相当于中共给自己的百姓扔了245颗广岛级别的原子弹；1966年至1976年文革期间，受各种政治批斗迫害而死亡的人数，达6000万至1亿左右，从北京的“红八月”起，诸多地区发生了大屠杀，包括广西文革屠杀（及大规模吃人事件）、内蒙古内人党事件、广东文革屠杀、云南文革屠杀、湖南文革屠杀等，亦爆发了二十世纪世界上最重大的，人为技术灾难“河南‘75·8’水库溃坝”事件。此外，中共十八大以来，因所谓的“反腐”至少造成数百万人入狱或者非正常死亡。

***（二）恶法恶政导致的人口灾难***。如，强行推广计划生育政策，36年来最少有4亿胎儿被屠杀。再如，强拆百姓房产，非法占有百姓私产，造成的死亡无法统计，等等。此外，毒疫苗、毒奶粉，造成的婴儿死伤，高达几十万。具体人数因为当局隐瞒而无法统计，还有就是因食品安全，空气、土地和水质等环境污染，对广大百姓引发的慢性和恶性病例不计其数。

***（三）对人民的直接屠杀和镇压造成的人口灾难***。如，1989年天安门大屠杀，使用坦克机枪残杀中国青年学生，死亡10454人；1999年至今对法轮功的迫害，死亡人数无法统计。再如，中共在新疆，关押了最多达300万新疆人民，并对西藏同胞进行残酷镇压，死亡人数无法统计。2019年至今对香港的大屠杀，上万人死亡，数万人失踪。

***（四）释放中共病毒造成的人口灾难***。2019年中共释放中共病毒以来，造成百万中国人民死亡。

**四、几点简要结论**

***（一）中国人民是中共的最大受害者***。中共对中国人民的杀害、戕害是中国历史之最，中国历史上没有任何一个朝代杀害如此多的人民（包括外敌入侵）。

***（二）中共对中华民族和中国犯下深重罪孽***。由于人口发展的周期性和长期性，当前中共国的人口问题将不断显现和加剧，其对中国和中华民族的影响将达数十年乃至百年。

人口是一个国家和民族发展最基础的要素，中共对中国和中华民族的犯下的罪孽之深重，不可饶恕。可见，对中国人民而言，唯有灭共才能真正解救国家、民族和人民。
