# **美国和盟友将打造“无中共国”的科技供应链**

**独家新闻：拜登将签署总统令，支持芯片、电池和稀土产业发展**

[!\[\]()!\[\](https://spark.adobe.com/page/IZPDrmn6Rcjq4/images/447add29-3b08-426e-8667-9226c1d4d390.png?asset_id=08484fd5-5336-4345-a86e-835689472750&amp;img_etag=%229ac34395ce2e904f6ce78f82ff3d9175%22&amp;size=2560)](https://spark.adobe.com/page/IZPDrmn6Rcjq4/images/447add29-3b08-426e-8667-9226c1d4d390.png?asset_id=08484fd5-5336-4345-a86e-835689472750&amp;img_etag=%229ac34395ce2e904f6ce78f82ff3d9175%22&amp;size=1024)由于今年芯片短缺，科技供应链的问题变得更加紧迫，这对汽车制造商的打击尤为严重。©路透社

华盛顿/台北——美国总统乔·拜登最早将于本月签署一项行政命令，与台湾、日本、韩国等国家合作，加快建立对中共国依赖程度较低的、包括芯片和其他具有战略意义产品的供应链。

该文件将下令制定国家供应链战略，战略预计将对供应网络提出建议，使其更不容易受到灾害和不友好国家制裁等干扰。日经社获得的草案显示，措施将集中于半导体、电动汽车电池、稀土金属和医疗产品之上。

该命令指出，“与盟友合作可以带来强大、有弹性的供应链”。这表明国际关系将是这项计划的核心。华盛顿有望寻求与台湾、日本、韩国在芯片生产方面的合作关系，以及与包括澳大利亚在内的亚太经济体在稀土方面的合作关系。

美国计划与盟友共享重要产品的供应网络信息，并将寻求利用互补性生产。它将考虑建立在紧急情况下快速共享这些物料的框架，并讨论如何确保库存和备用制造能力。参与计划的各国可能会被要求减少与中共国的业务往来。

随着今年芯片短缺，汽车制造商遭遇的打击尤为严重，这使问题变得更加紧迫。

波士顿咨询公司的数据显示，过去几十年来，美国在全球半导体制造能力中的份额直线下降。1990年为37%，现在已降至12%。

虽然美国已要求台湾（份额为22%列居榜首）提高产量，但那里的工厂已在满负荷运转，短期内提高供应量的选择不多。

同时，波士顿咨询公司预测，在1000亿美元政府补贴的帮助下，2030年中共国将以24%的市场份额领跑全球。

[!\[\]()!\[\](https://spark.adobe.com/page/IZPDrmn6Rcjq4/images/f2891973-e583-4c45-9a8f-5d53a5bb7eab.jpg?asset_id=74531414-026f-4375-ae47-994474bf7c24&amp;img_etag=%227a1c07bcfd72d3e621b89063f11fe7cf%22&amp;size=2560)](https://spark.adobe.com/page/IZPDrmn6Rcjq4/images/f2891973-e583-4c45-9a8f-5d53a5bb7eab.jpg?asset_id=74531414-026f-4375-ae47-994474bf7c24&amp;img_etag=%227a1c07bcfd72d3e621b89063f11fe7cf%22&amp;size=1024)

重要产品过于依赖中共国会带来安全风险。北京曾利用法规向贸易伙伴施压。例如在2010年日本控制的尖阁诸岛（中共国宣称是钓鱼岛）局势紧张的情况下，中共国对日本的稀土出口实施了禁运。

美国约80%的稀土从中共国进口，而一些医疗产品90%依赖中共国。

重组供应链可能需要相当长的时间，尤其在半导体领域。由于世界顶级芯片制造商的数量有限，这些公司拥有决定是否遵循美国领导的影响力。所以，重组供应链需要其他国家政府的理解和配合。

一位日本政府消息人士说：“据我目前所知，美国将对其供应链进行密集审查，以厘清半导体和稀土行业在多大程度上依赖于哪些国家。之后，美国将与盟国一起商讨对策。”

华盛顿已经开始奠基工作。自去年秋天以来，在与北京紧张关系不断加剧的情况下，华盛顿呼吁，与台湾、日本和澳大利亚等拥有宝贵技术或资源的经济体一起，脱离来自中共国的供应链。

台北的反应尤其迅速。美国和台湾高级官员在11月签署了一份谅解备忘录，意图促进七个领域的技术合作，包括半导体、第五代无线技术、以及“安全、可靠和可依赖的供应链”。

全球顶级芯片代工企业台积电去年春天同意在亚利桑那州建设制造厂，这很可能成为这种双边关系的象征。这家芯片制造商将为该工厂投资120亿美元，该工厂将为军方生产半导体。该厂计划将于2024年投入生产。美国政府正在为该项目提供补贴。

自去年以来，日本经济产业省一直努力吸引台积电进入日本。这不仅将建立更加牢固的三向供应网络，还将向日本提供可靠的未来尖端芯片来源。日本政府已计划2000亿日元的预算（合19亿美元）为这家代工厂铺设红毯，以期与日本企业进行可能的合作。

这似乎正在取得成果。日经社本月获悉，台积电正在计划于日本建立一个耗资200亿日元的研发中心。

在稀土领域，美国正在与澳大利亚合作，绕开中共国的主导地位。澳大利亚稀土矿商Lynas在美国国防部的财政支持下，正在德克萨斯州建立加工厂。

[!\[\]()!\[\](https://spark.adobe.com/page/IZPDrmn6Rcjq4/images/c41ceafd-7fa0-4f24-ba13-a12abd622bc4.jpg?asset_id=da6b2ba1-8092-402c-810c-6e245d55e0f4&amp;img_etag=%2227dca40f80ad41c9e5b28037c5f0fbf0%22&amp;size=2560)](https://spark.adobe.com/page/IZPDrmn6Rcjq4/images/c41ceafd-7fa0-4f24-ba13-a12abd622bc4.jpg?asset_id=da6b2ba1-8092-402c-810c-6e245d55e0f4&amp;img_etag=%2227dca40f80ad41c9e5b28037c5f0fbf0%22&amp;size=1024)

电动汽车电池是另一个需要采取行动的领域。因为松下和韩国LG化学公司的市场份额正在被中共国的竞争对手夺走。

但在其他领域（例如5G）中，新的供应链对于美国和日本公司而言可能会非常昂贵，因为这意味着将失去与具有成本竞争力的中共国供应商（如华为公司）的合作机会。
