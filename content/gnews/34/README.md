获得川普总统背书后的参议员马可·卢比奥对（Marco Rubio）【1】对《Newsmax》电视台说，2024年共和党没有人能打败川普总统。

《Newsmax》发表文章说【2】，尽管2016年共和党初选与川普总统展开竞争（如果没有争议的话），但佛罗里达州参议员卢比奥在争取明年连任时获得了川普总统的支持。

事实上，过去的竞争对手现在相互尊重。

的确是，卢比奥说，如果川普总统在2024年再次竞选总统，他不会挑战川普总统，“美国最受欢迎和最有影响力的共和党人”。

“如果他决定竞选总统和提名，他将成为该党的提名人。” 卢比奥周三对Newsmax电视台的《格雷格·凯利（Greg Kelly）报道》说，“我就是这么看的。”

“他还没有做出那个决定，这条路还很长，但如果唐纳德·川普决定在2024年再次参选，他将成为共和党候选人，我想几乎所有与我交谈过的人都会同意这一点。”

卢比奥并不是共和党总统候选人提名中第一个暗示他会听从川普总统这位潜在的对手，曾经的大使尼基·海利（Nikki Haley）本周表示，如果川普总统参选，她不会与他竞争，而且会支持他的总统竞选。

至于卢比奥在竞选连任时得到川普总统的支持，他告诉凯利，川普总统的名字在佛罗里达有分量，也有选票，长期居住在纽约的川普总统最近选择将佛罗里达列为自己的家乡。

“我们与川普政府合作得非常好。” 卢比奥对主持人格雷格·凯利说，我们一起做了很多积极的事情，很有成效的时光和很好的工作关系。

“另外，我们注意到的事情之一，这是一个额外的特别支持，因为他是佛罗里达州的居民和选民，所以，我不仅需要他的支持，我还需要他的选票和他所有搬到佛罗里达的家人的选票。”

至于现任总统乔·拜登，以及他本周宣布的在阿富汗长期冲突后将撤军的消息，卢比奥对此表示担忧。他担心从阿富汗撤军可能会让基地组织恐怖组织在阿富汗“重组”，并再次威胁美国人的安全。
