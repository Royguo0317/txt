**西班牙2021年5月5日电/西喜社——**

一家日本医生工会的负责人周四警告说，今年夏天在东京举行奥运会，世界各地成千上万的人聚集在一起，这可能导致新的冠状病毒“奥林匹克”菌株的产生。

尽管日本在奥运会推迟一年后多次承诺将在东京举行“安全有保障的” 2020年奥运会，但日本仍在努力遏制第四波大流行，并防备扩大到覆盖全国大部分地区的紧急状态。

尽管有严格的病毒预防措施，日本官员，奥运会组织者和国际奥委会（IOC）都誓言奥运会将继续进行。外国观众已经被禁止入境，下个月将对国内观众做出决定。 但是，即使采取了这些措施，对于运动员和官员涌入日本的担忧仍然存在。日本的疫苗接种过程如同冰川上行走般一样缓慢，只有5％以上的人口已经接种了疫苗。

日本医生工会负责人上山直人(Naoto Ueyama)说，由于有来自200多个国家和地区的人们即将抵达东京，因此在7月举办奥运会将很危险。

他在新闻发布会上说：“存在于不同地区的所有不同的病毒突变株都将在东京集中并聚集。我们不能否认奥运会后甚至可能出现新的病毒株的可能性。”

东京奥运会的官方合作伙伴《朝日新闻》周三发表社论，敦促取消奥运会。但前国际奥委会副主席迪克·庞德当天晚些时候表示，体育盛会应该，并且将继续进行。

官员们表示，政府目前正准备把原定于5月31日取消的全国大部分地区紧急状态延长，极有可能延至6月，距离奥运会7月23日开幕的前几周。

但国际奥委会委员约翰·科茨（John Coates）表示，即使在紧急状态下，奥运会也可能举行。上山直人对此观点大为光火。

上山说：“日本人民确实对这些声明感到非常愤怒，对于医疗保健和医疗专业人员更是如此。” 日本的医疗系统目前正承受着巨大的压力，某些地区的官员担心，奥运会带来的潜在额外压力。例如，在遭受重创的大阪市，上周为严重病毒患者预留的348张病床中，有96％已被使用。

本周早些时候，美国建议不要前往日本，但奥运会组织者表示，这不会影响奥运会。白宫周三表示，日本政府已保证将就对奥运会的担忧事项保持密切关注。
