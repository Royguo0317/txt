据韩联社和朝中社18日报道，在17日朝鲜劳动党八届三中全会上，金正恩仔细分析了美国新政府的对朝政策方向，并提出今后将在对美关系上坚持的战略、战术应对和行动方案。

报道还称，金正恩评估了最近国际政治舞台上出现的变化等外部环境，要求提高朝鲜的战略地位，进一步发挥主动性，并为此营造有利的外部环境。

这是金正恩在美国新政府上台后第一次结合外部环境，发表了自己的立场，金正恩在会上强调，为维护国家的尊严和自主发展所带来的利益，保障和平环境和国家安全，对话和对决两方面都要做好准备，尤其在对决方面要做好万无一失的准备。

这段话其实透露了金正恩对外界释放的积极信号。金正恩松口“对话”机会，其实已经在朝中和朝美关系上，做出了摇摆。疫情对全世界各个国家经济的冲击都是十分突然和巨大的，作为朝鲜这个综合国力不强的国家，疫情对经济的影响，以及对人民生活水平的影响一定是十分严重的，之前，朝鲜靠着中共这个靠山，必要时哼唧一声，为中共喊喊话，当一下小弟，换来的会是中共对其经济和贸易方面的扶持。但中共现在自己的经济也自身难保，前一阶段，朝鲜的粮食危机再一次被爆出，那么对于独裁政府来说，如果百姓的温饱也出了问题的话，对其政权的威胁将是巨大的。因此在这个危机关头，金正恩重新理一下手中的牌，并分头释放信号后再谈判，就是迫在眉睫需要解决的事。

一方面与中共谈判，如果需要朝鲜的继续支持，就要看中共给的利益是不是大，而且能不能持久。另一方面与韩国甚至美国谈判，就意味朝鲜判断中共死期不远，急于变换立场不至于殃及金家皇权，也取得一个“审时度势”的口碑。无论是哪种原因，金正恩在此时的发声，其实对中共来说是一个很大的威胁，因为已经孤立无援的一尊能否留住小兄弟的心，已经不是一点钱能够解决的了。
