今年11月对于中共国债券市场来说并不平静。

11月10日，河南省国企永煤集团公告称，一笔10亿元人民币的企业债券到期无法偿还本息，构成实质性违约。11月16日，华晨汽车集团控股有限公司公告称，已构成债务违约金额合计65亿元，逾期利息金额合计1.44亿元。11月18日，清华控股有限公司公告称，控股子公司紫光集团有限公司发行的“17紫光PPN005”未能按期偿付本息，已构成实质性违约。

接二连三的违约事件备受瞩目的原因在于它不仅是国有企业违约，而且是AAA高评级国有企业违约。这些违约事件不仅爆出近年来高杠杆带来高风险的现象，而且“国企不违约”的铁律被打破。

我们再来比较2014年和截止2020年11月的相关数据
![]()![](https://gnews.org/wp-content/uploads/2020/12/compare.png)图片来源搜虎
在中共国，国企背后有着政府影子，政府不会放任企业破产，一定会出手相救，之前也是一直这么做的。如今政府袖手旁观并不意外。早在2008年金融危机之后，中共伪政府推出四万亿计划，大规模放松信贷时就埋下恶果。之后中共一直实行量化宽松政策，货币放水刺激经济，如今无论是铁路、城市地铁、房地产，早就都是产能过剩。随着中美贸易战，疫情冲击经济活动的停摆以及随后的全球需求下滑，中共国经济可谓内需疲软，外需走弱。出口、消费、投资拉动经济发展的“三驾马车”一蹶不振。从2018年提出“六稳”方针（稳就业、稳金融、稳外贸、稳外资、稳投资、稳投资），到今年4月又提出“六保” （保居民就业、保基本民生、保市场主体、保粮食能源安全、保产业链供应链稳定、保基层运转）。这些信息无不透露出整体经济严重恶化，钱袋子干枯。中共伪政府不但已经失去救助违约国企的能力，而且在死亡的泥潭中越陷越深。
