最近，美国的CCP病毒疫情稍为缓和，总统首席医疗顾问安东尼．福奇就频频出现在公众视野。本周四，白宫冠状病毒工作组成员、现任美国国家过敏和传染病研究所所长福奇和美国第一夫人吉尔.拜登，一起参观了疫苗接种点。这很正常，因为抗疫本来就是福奇当前最主要的工作，但随后发生的事情令人作呕。

在众多摄像机镜头前及现场众目睽睽之下，第一夫人突然手握福奇的胳膊问在场的人们：“你在这里见过我们的美国英雄（福奇）吗？”，当人群以掌声回应时，福奇以微笑欣然接受了“荣耀”。在记者提问环节，第一夫人意犹未尽，再一次强调“福奇博士就在这里，你知道谁不爱福奇博士？”，然后她自问自答：“我想每个美国人都喜欢福奇博士”。福奇则以微微鞠躬来欣然接受第一夫人的赞美。

令人费解的是，第一夫人没有片言只语说明一下福奇为什么是美国英雄，而欣然接受英雄封号的福奇甚至礼节性的谦逊都省略了。人们不禁会问，地位比一般的卫生部长重要的福奇，到底为什么是美国英雄，是反对川普总统釆信闫丽梦博士和路德社的警告封禁美中航班得力？是资助中共武毒所和香港P3实验室的病毒功能增益研究有功？是使美国成为全球染疫、死亡率最高的国家作用显著？是在禁止使用安全有效的羟氯喹上有苦劳？是在国会听证会上回答资助武毒所质询时敢于撒谎？还是他就任医疗一把手以来的30年，将美国的医药费推到几乎全世界最高？

公道自在人心，如果暂且抛开以上质疑，位高权重的福奇就职多个重要部门，领着几份纳税人支付的高薪，为美国人工作本来就是他的份内事，难道仅仅履行本职工作就是美国英雄？美国人民如果确实找不到答案，就应该追问第一夫人吉尔．拜登，为什么安东尼．福奇是美国英雄？她凭什么授予福奇“美国英雄”这个神圣无比的称号？
