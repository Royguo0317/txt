德国之音报道，缅甸军政府自2月1日罢免民选领导人昂山素季以来，每天都面临抗议活动。缅甸军方对此进行了残酷镇压，迄今已造成500多平民死亡。联合国驻缅甸特使警告称，在这个亚洲国家，“流血事件”迫在眉睫，缅甸正处于内战的边缘。美国敦促中共向政变领导人施压，利用其影响力，让缅甸军方领导层对动乱负责。

周三，美国表示，将继续呼吁中共追责那些对政变负责的人。美方称缅甸军政府的所作所为不符合美国的利益，也不符合美方的合作伙伴和盟友的利益，同时指出也不符合北京的利益。早在本月初，美国国务院便呼吁中共利用其在缅甸的影响力，帮助恢复该国的民主政府。自2月1日发生政变以来，美国和中共官员多次谈到缅甸的局势。美国国务院发言人内德·普赖斯(Ned Price) 称，“我们已敦促中方发挥建设性作用，利用他们对缅甸军方的影响力，结束这场政变。”

与强烈谴责军事行动的西方国家不同，中共以谨慎的态度回应政变，并强调稳定的重要性。利用联合国常任理事国的身份，中共国否决了联合国意图对缅甸军方采取的制裁行动。中共驻联合国使团的一名外交官表示，任何回应都应避免“加剧紧张局势或使局势进一步复杂化”。

周三，联合国缅甸问题特使伯格纳(Christine Schraner Burgener)在联合国安理会闭门会议上警告说，缅甸可能面临“规模空前”的内战。在致命镇压后，缅甸的抗议活动仍在继续，联合国特使呼吁，“如果我们只等他们准备好谈判，地区局势只会恶化。缅甸将发生流血事件。” 她呼吁国际社会考虑采取“具有重大意义的行动”来恢复缅甸民主。

美国贸易代表戴琪周一表示，根据2013年的贸易和投资协定，美国将立即暂停与缅甸的所有接触，直到民主选举产生的政府回归。
