**译评：英国为了替换华为的4G和5G设备将耗资至少7亿美元，这是自食其果。何止英国如此，任何与中国共产党政府合作的国家和公司实体，都将为自身和自己所在的社区、民族与国家带来不可估量的损失，看似节省了费用，但却步入了深渊，看似占了便宜，但却失去了老本。正如郭先生所讲“你看重利息，而共产党看重你的本金”。**

(彭博社) — 戴着硬壳帽、穿着荧光夹克的电信工程师花了十年时间翻越建筑物，在英国各地安装华为技术有限公司的设备。现在，由于美国领导的反对中共国公司的运动，从赫尔的穆塞尔法院塔楼开始，他们开始了一项耗时七年、耗资7亿美元的任务，将把这些设备全部拆除。

赫尔市是英国最大的网络公司BT Group Plc（英国电信集团股份有限公司）将铲除华为的第一个英国城市。这家总部设在伦敦的公司表示，它将在7月前淘汰华为在那里的所有设备，用诺基亚公司的部件代替。

英国电信首席技术官霍华德-沃森在接受彭博社采访时说：“我们相当热衷于选择一个城市地区，并对其进行整体检查，以确保我们能够真正检查我们是否对客户服务产生了不利影响。到目前为止，这方面的迹象真的很好。”

在全国范围内，工程师将不得不在英国电信的18000个移动信号塔、屋顶和其他地点中的12000个重复这一手术。沃森说，到目前为止，大约有130个已经完成，因为寻求地主许可、关闭道路以及最终在任何时候调动数百人的过程非常耗时。

这也是一项昂贵的工作。英国电信预留了5亿英镑（7.01亿美元），用于支付诺基亚和爱立信公司的替换套件。英国文化部长奥利弗-道登说，这项禁令将使英国的5G推广推迟三年之久，并使整个行业损失20亿英镑。

英国电信在赫尔的工作提供了对美国启动的改造和替换任务的现实的早期一览，美国指称总部设在深圳的华为是一个不可接受的安全威胁，因为它接近北京政府。华为一再否认其设备会带来任何风险。

不过，英国还是命令这家中国电信巨头在2027年之前完全撤出5G基础设施。 不仅仅是华为最近安装的5G设备必须被移除，还有其更为普遍的4G基站。这是因为第一代5G与4G密不可分，而华为的设备无法与竞争对手的同等设备互通。

但是，尽管在华盛顿和威斯敏斯特的大厅里做出的决定，意味着数百名电信工程师，现在面临着拆除他们刚刚完成安装的天线和小工具的任务，但也有一线生机。在英国脱欧和冠状病毒之后的金融动荡中，这为他们赢得了多年的额外有偿工作。
