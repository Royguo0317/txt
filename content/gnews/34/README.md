## 中共国海警船创纪录连续112天航入钓鱼岛
` 日本东京方舟农场` [轉載自GNews](https://gnews.org/zh-hans/1300443/)

【日本东京方舟农场】 撰稿：青衣      校对：MIMI

据《法国国际广播电台》6月4日报道，日本海上保安厅巡逻船4日发现4艘中共国海警局船只在尖阁诸岛（中共国称钓鱼岛）附近领海外侧的毗连区航行。这是连续112天在附近发现中共国公务船，刷新了2012年9月尖阁诸岛国有化后的最长连续天数纪录。

据日本第11管区海上保安本部（那霸）称，4日发现的4艘船中有1艘搭载了类似机关炮的装置。海上保安厅巡逻船对其发出了勿靠近日本领海的警告。据悉，中共国公务船从2月13日起一直持续在尖阁诸岛附近海域航行。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/05112547/8BD29A93-92C7-4438-908E-BB6BAA96ED6A.jpeg)

《共同社》今天报道称，据日本海上保安厅透露，今年以来总计有147天发现有中共国公务船在该海域航行，其中有20天驶入日本领海，而去年全年仅有29天驶入领海。根据目前的情况看，今年的节奏已超过去年，在领海内试图靠近日本渔船的情况也较为多见。

日本官房长官加藤胜信4日在记者会上表示：「我们认为这是极其严重的事态。将继续冷静、坚决地应对。」 日本防卫相岸信夫则指出：「绝不能容许以实力为背景单方面改变现状的尝试。」

据《共同社》报道，尖阁诸岛是位于冲绳本岛以西约400公里处的无人岛群，附近海域被指可能埋藏石油资源。中共国从上世纪70年代开始，主张对其拥有主权，并且中共国今年2月还施行了允许海警局使用武器的《海警法》。

近期，在美日外长和防长「2+2」会议联合公告，以及日本首相访美后发布的美日联合公报中，都表明了美日共同捍卫尖阁诸岛的决心。此外，根据日本NHK 5月29日报道，为牵制中共国海警船于尖阁诸岛（钓鱼岛）海域的活动，日本维新会和国民民主党确定方针，将共同向国会提交自卫队法等法律修正案，这是两党首次联合在众议院提交修正案。该修正案明确规定，自卫队须承担对中共国海警局船只的警戒监视和情报收集任务，并允许队员在不超出自卫范围的情况下使用武器。

在如此背景之下，中共国海警船还变本加厉，一再进行挑衅行为，使得该地区擦枪走火的可能性进一步加剧。真不知以中共国的军事实力，尤其是海军的实力，究竟有何底气胆敢面对美日联军的还击。
