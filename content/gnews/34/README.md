瑞典议会表达了对首相斯特凡·勒文（Löfven）的不信任。在斯德哥尔摩的 349 名国会议员中，181 人中的大多数投票反对自 2014 年以来一直在任的总理。这使得勒文成为第一位在这种不信任投票中落败的瑞典首相。他现在有两个选择：要么向政府辞职，要么在一周内召开新的选举。他将如何决定仍然是开放的。他宣布了周一召开新闻发布会。

勒文和他的社会民主党迄今为止在绿党的少数政府中领导瑞典。中央党和自由党容忍了这个红绿色政府，勒文于 2019 年初与他们达成了政治协议。经过艰难的谈判，有关各方同意了一项罕见的跨越传统党派边界的合作。

简评： 瑞典是一个民主自由法治的国家， 一个国家的政府总理可以通过议会在执政两年后再做评估， 可以让你继续任职， 也可以罢免你。 瑞典的总理就遇到了这样的问题， 大多数议员的法对票， 导致总理被罢免。 要么重新选举要么辞职。 这和中共国的终身制， 一党制，是多么大的差距呀。 爆料革命新中国联邦就是要向这个方向去努力， 达到发达国家已经拥有的制度。
