尤其是一个国家领导人的人品，如果只有智商和冲劲没有人品，那这个国家就惨了！整个世界也不安全了！为什么？

1. 食品安全问题。国内蔬菜农药超标，种菜的人给自己吃的东西和要卖给别人吃的东西分开来种，只给自己吃的不放农药。这一点众所周知！可是，中国相关部门以及国家领导人不重视，上梁不正下梁歪！

2. 中共有特供食品。明知国民吃的食品有安全隐患，可是中共先是没有监督机制，后又开始了专门给他们自己的特供，连榨菜都吃特供。这么没人品的国家领导人无视人民的利益，无疑是人民最大的灾难！

3. 世界大工厂的消失。中国曾经被称作“世界大工厂”，在世界各地，从各种便宜货到相当昂贵的奢侈品，都可见Made in China的标签。当年家喻户晓的陆剧【外来妹】中，聪明能干的中国打工仔们离开农村离开东北，纷纷聚集经济相对发达的南方，开始没日没夜的工作，给中国创造了可观的经济收入！可惜结果呢？国进民反倒退了，贫富差距越来越大！一场病毒，外商开始陆续离开大陆，打工的老百姓随之失业！

4. 中国数字脱贫。大量工人失业了，大学甚至研究生毕业以后也很难找到工作，他们很多不得不当起了快递小哥。这个时候，中国居然宣布实现了小康社会！简直是自欺欺人玩自嗨！

5. 中共病毒。为了制造这场病毒，中共派相关的科学家们到国外学习、请教国外专家，经过其畸形努力造成了全世界大流行，破坏了全世界的健康环境，全世界无数家庭家破人亡！

中共有智慧有劲头儿，缺的就是人品（人性），其危害之大可以毁灭地球！
