作为在部队服役的士兵，我相信他们都希望自己的吃穿住行很好。况且，包括伙食在内的一切安排并不由自己决定。伙食好了作为士兵当然很高兴，伙食差了可以被ccp解释为考验战士们的意志，也可以被解释为后勤供给极其困难，是身不由己的事。

2，上海市黄浦区政府食堂菜价的意外曝光。
![]()![](https://gnews.org/wp-content/uploads/2021/04/文章3-4.jpg)
显然这种特权福利在中国公务员体系内是普遍现象。除此以外，还存在烟酒、各类商品、度假专区、养老医院等公务员特权专享的待遇。确实，我们都痛恨这种现象的存在。

然而，作为在体制内工作的人员，吃着价廉质优的饭菜当然很高兴，每个月再有大量的津贴会更加高兴，并且得到的好处是多多益善。谁会去拒绝给予自己的额外利益呢？这是人性。

那么，ccp的这些现象对我们新中国联邦有什么警示作用？

在联盟亦或是农场主给予不属于或者远超我们应得的利益时，战友们会拒绝吗？

文贵先生提倡的让战友们赚钱，有钱后要学会感恩、奉献、帮助同胞、帮助新中国、大爱、无我。作为新中国联邦人，我们需要正视并且摒弃的首先就是贪恋。比如，此次分配gcoin的过程中，联盟奖励每位农场主100万的gcoin配额以及奖励100万的gcoin配额给各个农场管理层。难能可贵的是小皮匠和安红率先将奖励给自己的配额转赠给了秘密翻译组。

每位农场主都为新中国联邦做了大量的工作，每个农场的管理层也为自己的农场的发展倾注了心血，有些人可能还冒着生命的危险在做着工作。当然，主观上讲，奖励的配额合理甚至不能用利益来衡量。但是，问题是，如果我们的利益分配是建立在主观而非客观的审视上，在新中国联邦内部可能形成利益阶层。农场主可以感性的认为某位战友贡献了很多，从而给予更多的利益，比如文贵先生直播中提到的两位战友分了100万个gcoin配额。

因此，我们的新中国联邦在分配利益的过程中，应最大可能的将战友们的工作进行量化。如翻译工作，可以参照市场价将每位战友的翻译工作的总量进行计价。尽可能的避免ccp式的嫌麻烦的豪气的大笔一挥的现象。

在利益分配之初就将各种问题考虑到，在制度上做到最大限度的公平，禁得起推敲。
