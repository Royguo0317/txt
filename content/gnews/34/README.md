## 香港教科书将“携枪的米老鼠”描述为美国侵略者

[!\[\]()!\[\](https://spark.adobe.com/page/jLhhsTWOrvKcQ/images/4e28ad03-978e-4169-af4d-854aac38f27f.jpg?asset_id=4b517980-7eec-4d55-91be-87611161ffb5&amp;img_etag=%22729c1b4b33940c2d17e547ae8278fa90%22&amp;size=2560)](https://spark.adobe.com/page/jLhhsTWOrvKcQ/images/4e28ad03-978e-4169-af4d-854aac38f27f.jpg?asset_id=4b517980-7eec-4d55-91be-87611161ffb5&amp;img_etag=%22729c1b4b33940c2d17e547ae8278fa90%22&amp;size=1024)

Screenshot via Twitter（图片来源： 推特截图）

**据《南华早报》周四报道，香港公立学校一本新的教科书使用了一张描绘着“携枪的米老鼠”的图像来说明跨国公司如何进入发展中国家并“侵略当地文化”。**

据香港总部的《南华早报》报道：“在香港学校的新教学材料中，有一张图片描绘了穿着军装很像米老鼠和高飞的角色、投放汉堡而非炸弹的战斗机，以及麦当劳角色罗纳德·麦当劳和奶昔大哥的邪恶版本。”

《南华早报》在审查了这本高中教科书的预览后报道：“通过学习该指南，学生可以学习到，跨国公司是如何进入发展中国家的本地市场，并入侵当地文化的。传达的信息在很大程度上是一种反对文化的帝国主义。

据《南华早报》报道，该书的卡通章节“鼓励学生讨论西方对全球价值的影响，以及是否文化同一性或多样性是由此产生的结果”。

该报报道：“这幅描绘了美国跨国公司‘文化入侵’的政治漫画，出现在有争议的对通识教育科目的计划性改革的第一组教材中。”

教科书的出版商马歇尔·卡文迪许教育（Marshall Cavendish Education）最近发布了针对教师的初版通识科目教材。这幅反美漫画出现在其中一本书中，本周在香港引发了公众批评。

马歇尔·卡文迪许教育（Marshall Cavendish Education）的发言人于4月29日回应了公众反对，他告诉《南华早报》，这些书用于“过渡性教材，主要供教师参考，并且在咨询教育工作者后可能会更新”。

香港教育局于11月下旬宣布，将全面审查从2009年起公立高中开设的必修通识教育科目的课程。香港的亲共派领导人批评了该课程对2019年夏开始的民主运动（目前已逐渐减弱）起到了促进作用。该课程讲授公民可以拒绝服从等概念。

香港电台（RTHK）在当时报道，香港教育局在11月26日表示，将“重修”通识教育科目课程，以强调中共国的“发展、宪法、基本法（香港实际上的宪法）和法治”。

香港教育局宣布此消息之前五个月，中共国于2020年6月下旬对香港强制实施了国家安全法，该法在很大程度上镇压了香港的民主运动。该民主运动要求中共国在香港减少影响。从英国后殖民统治返回中共国后，香港获得了有限的自由。在2019年6月高峰时期，香港的民主运动鼓舞了超过50万人走上这座城市的街道。

中共对香港强制实施了国家安全法，目的就是消灭民主运动，该运动自开始以来已经持续了一年多。国家安全法在香港设立了新的罪行，例如恐怖主义罪、分裂国家罪、与外国政府勾结罪、颠覆国家政权罪等，（这些罪行）被设计成一个模糊的概括。

在香港，任何被判犯有国家安全罪的人都将面临至少十年的监禁。在最近几周和几个月里，香港政府当局利用新法律逮捕了亲民主运动的著名领导人士，并判处他们长期监禁。
