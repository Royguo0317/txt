该报将这名维吾尔族妇科医生简单地称为居尔金，47岁，她在2011年移民到土耳其伊斯坦布尔。

“我明白手术的意义，但后来我才知道，这是维吾尔族的政策，”居尔金说，”我也被绝育了。”

居尔金估计，强制绝育手术大约始于1980年，比中共为维吾尔族和其他穆斯林少数民族人民修建集中营的消息曝光早了几十年。

人权组织指责中共至少从2017年开始，在新疆的强迫劳动营中强行关押了1-3万维吾尔族和其他少数民族。中共官方否认了这一指控，并称其在新疆经营着以马克思主义教育课程为特色的职业中心，作为将少数民族融入中国劳动力和社会的努力的一部分。新疆看守所的幸存者和前雇员作证说，他们遭受或目睹了奴隶劳动条件、肉体和性折磨，以及强迫绝育。

英国广播公司2月3日发表了一篇报道，指称新疆拘留营的 “几名前被拘留者和一名看守 “告诉该广播公司，”他们经历或看到了有组织的大规模强奸、性虐待和酷刑系统的证据”。

作为CUSEF项目的一部分，CAP每年都会派代表团前往中共国。这个与中共政府有联系的组织的宣传册显示，CAP代表团讨论了 “广泛的问题，包括地缘政治合作、军方对军方的合作、气候变化、能源安全、贸易平衡和全球金融危机。

201l年的图片显示，CAP代表团在中共的 “人民大会堂 “与高层掌门人会面。中共国驻外使馆的一份报告叙述了中共国官员如何 “敦促美国妥善处理涉台问题，重视中共国的关切”。

继2013年代表团之后，CAP和CUSEF共同撰写了一份长达110页的中美关系未来报告，敦促加强经济联系和军事协作。
