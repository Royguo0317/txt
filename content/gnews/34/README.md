美国当地时间2021年1月6日开始，参议员约什·霍利连发三条推文。

![]()![](https://gnews.org/wp-content/uploads/2021/01/图片2-1.png)

第一条推文是在安提法成员对他家人进行威胁之后，他表示：“如果卑鄙的安提法可以恐吓我的家人，那么他们就可以对美国的任何家庭这样做。在家里人们应该是安全的，我们绝不会对此坐视不管。”

第二条推文在国会会议之前，他说：“今天，我有机会也有义务代表我的选民发言，并在选举人团认证期间提出反对。我期待辩论。”

虽然随后发生的一切显然出乎所有人意料，参议员并未因此乱了方寸。他在第三条推文中表示：“感谢那些挽救生命的勇敢的执法人员。暴力必须停止，袭击警察和违反法律的人必须受到起诉，国会必须恢复工作并完成任务。”在谴责暴力的同时，他并未忘记自己这一天来到国会的初衷。在现在这个文贵先生说的滑稽达到顶点的时候，不断有人变节，有人退缩，而霍利议员并未忘记自己的使命，仍在难能可贵地坚持。
