**弗吉尼亚大学规定秋季开学前为学生接种冠状病毒疫苗**

弗吉尼亚大学（UVA）正命令其学生接受针对中共冠状病毒的疫苗接种，以便在秋季学期回到校园内亲自上课。

“从7月1日起，所有在下一学年准备亲自在大学生活、学习或工作的学生必须在返回学校之前全面接种疫苗，”大学校长吉姆▪︎瑞安在学校网站上发布的公告中说。

该大学校长补充说，如果学生想在秋季回到校园内亲自上课，“必须在7月1日之前提供疫苗接种证明”。

“没有接种疫苗且不符合例外条件的学生在7月1日之后将被禁止到场，”学校警告说。

该大学接着说，“学生可以寻求医疗或宗教豁免接种要求，”但表示，如果这种豁免被批准，学生将每周接受冠状病毒检测，“并可能接受其他公共卫生检测”。
至于大学员工，瑞安说，大学希望“没有医疗或宗教豁免需求的教职员工尽快接种疫苗”。 “我们的员工接种疫苗对于防止COVID-19的传播和恢复正常的亲身体验至关重要，”该大学补充说。任何未接种疫苗的弗吉尼亚大学员工将接受强制性的流行病学检测，该检测将于今年夏天开始。

瑞安补充说，大学管理人员“将监测员工的疫苗接种率，并与公共卫生专家协商，考虑是否要求为所有员工接种疫苗”。

弗吉尼亚大学并不是唯一一个强制要求其学生在返回现场上课前接种冠状病毒疫苗。3月，罗格斯大学成为美国第一所强制要求学生接种疫苗的主要大学，它宣布如果学生想在秋季停止虚拟学习并返回校园，就必须接种疫苗。

此后，哥伦比亚大学宣布，学生必须接种疫苗才能在秋季回到教室上课。

同样，密歇根大学迪尔伯恩分校也规定其学生要么接种冠状病毒疫苗，要么每周接受测试，以便在秋季学期返回教室上课。
