14. Smartmatic的选举技术被称为”选举管理系统”，它是这个领域的计算系统的先驱。他们的系统通过互联网将投票数据传输到一个中央计算中心。投票机带有一个数字显示器，有指纹识别功能来识别选民的身份并打印选民的选票。选民的拇指指纹与电脑中的身份记录相连。Smartmatic创建并运营了整个体系。

15. 查韦斯坚持要求Smartmatic在设计系统时，要做到系统可以改变每个选民的投票而不会被检测到。他希望软件本身的功能能够达到以下效果：如果选民将自己的拇指印或指纹放在扫描仪上，那么指纹就会与该选民的姓名和身份记录联系在一起，标记为已投票，但该选民不会追踪到被改变的记录。他明确表示，该系统的设置必须不留下任何痕迹，没有任何证据表明某一特定选民的投票发生了变化，也没有任何证据显示一个名字或者指纹连结的是一个被改变的投票记录。Smartmatic同意建立这样一个系统，并制作了软件和硬件，为查韦斯总统实现了想要的结果。

16. 在Smartmatic选举管理系统投入使用后，我仔细观察了几次利用Smartmatic软件操纵的选举。其中一次选举是在2006年12月，查韦斯与罗萨莱斯竞选。查韦斯以压倒性优势获胜。他的得票率接近600万，罗萨莱斯得票率为370万。

17. 2013年4月14日，我目睹了委内瑞拉的另一次全国大选。在这次大选中，Smartmatic选举管理系统被用来操纵和改变结果，以保证乌戈-查韦斯的继任者能够胜选。

21. 我想指出的是，多米尼克公司和其他公司的电子选举系统的软件及基础设计依赖于Sartmatic选举管理系统的派生软件。简而言之，Smartmatic软件是每间选票计算公司的軟件和系統的基因。

22. Dominion公司是美国三家主要的选票统计公司之一, 该公司使用相同的方法和基本相同的软件设计来存储、传输和计算选民身份数据和投票数据。Dominion公司和Smartmatic公司是一起做生意的。这些软件、硬件和系统都有相同的基本缺陷，这使得操纵者有多种机会破坏数据，并掩盖整个过程，使一般人无法发现任何异常。投票机显示出选民想要的投票结果，然后打印出相同结果的纸质选票。但这一点并不重要，重要的是软件，是计票软件进行数字化投票并报告结果。软件把电子信息改变成计票系统的操作者想要产生的结果。就是这样完成的。所以，是软件本身配置了投票和投票结果，改变了投票人的选择。无论投票人投什么票，软件都会自行决定结果。

23. 所有计算机控制的投票过程都是在封闭的环境中进行的，因此，选民和任何观察者都无法察觉到正在发生的事情，除非发生故障或其他事件，导致观察者质疑这一过程。我在委内瑞拉加拉加斯的秘密计票中心亲眼看到，操纵和改变选票的过程可以实时进行。对我来说，这是非常令人惊讶和不安的事情。我很震惊，因为我从来没有想到会在现场看到它发生，然而我看到了。所以，我亲身体会到，选民的决定或纸质选票的内容并不重要。是软件操作者和软件决定投票结果而不是选民。Smartmatic公司拥有所有选票和投票以及在委内瑞拉存储的投票信息。他（？）保证Smartmatic公司实施或使用的投票系统是完全安全的，不会被入侵，也不会被改变。

25. 但是后来，在2017年的时候，马杜罗在委内瑞拉参加议员选举，而Smartmatic打破了他们和委内瑞拉政府的保密协议。他（？）通过媒体发布了一个公开的公告，说那些选举中使用的所有Smartmatic投票机完全是被操纵的，是被委内瑞拉的选举委员会操纵的。说所有的尼古拉斯-马杜罗和其他竞选议员的人的选票都被操纵了，他们实际上已经输了。因此，我认为这是最大的证据，证明舞弊可以实行，但软件公司会否认。该公司公开承认smartmatic公司曾创造、使用并仍在使用可被操纵或更改的计票软件。

26. 我对2020年美国总统选举期间在众目睽睽之下发生的事情感到震惊。这些情况和事件令人不禁想起2013年委内瑞拉总统选举中发生的Smartmatic软件电子改票事件。美国所发生的事情是，使用Dominion软件的五个州突然停止了计票。在停止计票的时刻，唐纳德-特朗普的票数大幅领先。然后，在凌晨没有发生投票且计票报告离线的时候，情况发生了重大变化。第二天早上，当计票报告恢复后，有一个非常明显的变化：支持票数倒向对方候选人乔-拜登。
