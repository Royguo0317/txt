中共官媒《环球时报》周四在一篇文章中称，中国人正在对“情绪识别技术”这个能让政府跟踪人们情绪的人工智能的使用越来越习以为常，它已经成为人们每天生活的一部分。该项技术将被应用于 “包括健康，反恐和城市安全等多个领域”。在示例中，警察使用情绪识别技术发现了一辆车中司机紧张程度超过普通人，从而搜查这辆车，后发现该车藏有毒品，并逮捕了车中的乘客。《环球时报》虽然没有具体地提及到“情绪识别技术”应用于新疆的统治，但却提到了把监狱的囚犯当做实验物件。因此，很有可能该地区的集中营网络使用到该项技术。有报纸宣称，至少有6所监狱正公开地对囚犯使用这项技术来预测那些囚犯的暴力值。《环球时报》提到，在中共国之外，如果政府使用该项技术来监测人们的情绪，公众可能会表现出对隐私和安全的担忧。虽然如此，该报称，这项技术已经在包括美国在内的多个国家使用，并坚称“不是邪恶的’读心术’。
![]()![](https://www.gnews.org/wp-content/uploads/2021/03/image003-28.jpg)
解放报文章指出，中共国政府不断地以高科技将其监控系统现代化，一项最新的计划是建立一超级DNA数据库，北京当局将DNA档案数字与实时监控工具结合，通过该超级DNA数据库对公民进行全面的监控，从而建成一个全球“数字集权国家”。
![]()![](https://www.gnews.org/wp-content/uploads/2021/03/image005-7.png)
中共国不仅在一步步加强对14亿民众的监控，还通过各种技术手段开始全球布局。近年来，中共透过脸部识别、无人机与全球定位系统，大规模监控本国人民、维护政权统治，而这套“数字集权”模式也逐渐在向全球扩张。
![]()![](https://www.gnews.org/wp-content/uploads/2021/03/image007-7.png)
美国大选期间，脸书、推特、谷歌等公司使用数字技术手段，进行言论审查，变相引导舆论导向，严重破坏言论自由。

近期缅甸军方为镇压行动，使用监控无人机、iPhone破解设备等“数字武器库”，这引起外界忧心世界正往“数字集权时代”一步步靠近。

香港人权律师桑普表示，从近年中共对内施行数字集权、美国总统大选的科技巨头封锁特定言论，再到近期缅甸军方使用数字武器库，可以看到数字集权主义正在泛滥，如果世界不实时纠正的话，可能真的会出现乔治·奥韦尔所著《一九八四》中提到“老大哥永远看着你”的局面。

解放报的社评也提到，，1984年作家奥威尔在他的作品中曾经想象过“数字集权国家的世界，但谁都没有想到，如今，中共国的领导人“老大哥”习近平，通过无处不在的电子摄影机，脸部辨识技术以及网络监控等高科技，把它变成现实。

无论是中共国的情绪识别技术、DNA数据库还是在全球逐渐推广的人脸识别技术、数字控制、定位系统，中共国正在全球建立数字集权国家。
