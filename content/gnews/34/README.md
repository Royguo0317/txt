印度正受到中共病毒的袭击，使新冠状疫情迅速恶化，造成2150万人确诊、23.5万人死亡，联邦卫生部5月7日公布前一天仅24小时，新确诊病例达414188人，连续2天超过40万水平，印度专家认为确诊人数和死亡人数严重被低估，特别是来自贫穷邦的患者，多数来不急到医院前就因缺乏氧气呼吸器而死亡，甚至待在家中等待奇迹出现。

印度目前以英国变异病毒株B.1.1.7和印度本土变异病毒株B.1.617两种病毒肆虐，在双重突变病毒后现在还出三重突变病毒B.1.618，印度科学家发现三重突变病毒的传染力更强，特征是除了刺突蛋白中的 E484K 和 D614G 突变，可逃避、削弱人体对病毒免疫反应，英国变异病毒株B.1.1.7大多数集中在新德里，而印度本土变异病毒株B.1.617则集中在受灾最严重的西部马哈拉施特拉邦（Maharashtra）。

印度政府首席科学顾问维杰·拉科万（K Vijay Raghavan）5日在卫生部新闻发布会上承认，莫迪政府未预期受新冠状病毒的感染人数会爆量增加，目前印度医疗氧气系统濒临崩溃，氧气物资严重缺乏，造成患者家属争抢氧气罐的状况，医院超出医疗负荷被迫拒收病患，导致大量病人只能等待死亡，爆量增加的尸体导致焚化炉超出负荷，中共病毒肆虐核弹般的效应已重创印度的政治、经济、医疗、民生和人心。

中共声称这是印度挑战中共国所付出的代价。2019年底文贵先生警告过，中共将对世界放毒，2020年119路得社、博士军团、闫博士表示用羟氯喹加锌片就能防御病毒感染，只要每盒不到8美元的羟氯喹就能挽救无数个生命和家庭，中共病毒危机对印度造成的伤害，远远大于每人发放羟氯喹，再加上只要研制出疫苗就会出现变种病毒。印度之前用羟氯喹解决了病毒危机，随后就遭到了变种病毒的攻击。中共在有解药的情况下，发动生化病毒引发全人类的灾难与恐慌，一方面放毒一方面搞疫苗经济，只有加速中共交出解药，才能挽回更多无辜的生命。本文仅代表个人立场
