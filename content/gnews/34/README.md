**一、蓬佩奥说对抗中共是跨党派共识**

5月5日，前国务卿蓬佩奥接受Pat Robertson采访的发言内容如下：

最大的敌人永远在美国内部，海外最大的敌对势力就是中国共产党；中共与前苏联有所不同，中共有14亿老百姓、庞大军队和巨大的经济体量；印太盟友的团结，以及美国对台湾的支持能威慑习神，让习不敢轻易对台湾动手；过去20年，美国对抗恐怖主义的政策是正确的，但现在，美国面对的是具有超规模军事实力的中共。美国建立印太司令部就是为对抗中共，这是跨党派的共识而不关两党政治

美国官员的发言常从系统的角度出发，蓬佩奥上任后就开始着手建立印太司令部，经过多年的努力才建立成功，这就是为对抗中共国而准备的，美国为消灭中共一直在做各种布局。爆料革命之所以要提前站出来发声，就是要提醒墙内的老百姓尽早做好准备。

美国成立印太司令部就是要把前二十年美国最精英的反恐力量转移到灭共战线上，Sellin博士就是最典型的代表，美国参与的所有现代战争拼的就是后勤能力，靠的是体系思维。从前国务卿蓬佩奥的发言可以看到，美国从川普政府时期就开始着手为灭共做全盘布局，这次布局牵扯到美国各党派、政府机构和军队的调换等等一系列工作，美国至少已经准了三年以上的时间。这艘庞大的航母已经调转好方向，剩下的就是勇往直前。

**二、路德社反驳汪公公称美国制定霸权规则**

5月6日，针对布林肯近日称美国的目的不是试图遏制或者是压制中国，而是努力维护以规则为基础的国际秩序，汪文斌6日称美国制定霸权规则。

中共国一会儿说要跟美国脱钩，一会儿说要跟美国合作，一会儿又说要跟美国对抗到底，中共国的对美政策始终飘忽不定，这才是真正的心口不一。当年制定联合国宪章时，中华民国才是中国的代表，中共当时还未与世界接轨。

美国如果真如汪公公所说是一个霸权国家，美国无论到哪儿都应该受到盟友的抵制，但事实并非如此，美国不论到哪儿都有盟友的支持，反观中共国，连一带一路的合作国家都开始站出来指责中共。中共国如此急于给美国扣上霸权国家的帽子，就是因为他们意识到美国正在与其他国家结盟对付中共国，中共国为此非常恐惧。拜登政府上台后不仅夺回了WHO的控制权，美国在非洲、东南亚、美洲等多个地区的结盟运动进展得都非常顺利，未来，联合国各成员都会开始反对中共，支持中共国的国家将变得越来越少。

**一、习神将再次掀起一场“文革”运动**

自由亚洲4月30日报道：习近平上台后，中共的宣传口径开始转变，网信办主任庄荣文提出了一套全新历史论述，对前三十年作出了充分认可。但是，网易新闻网晒出刘鹤于2008年发表的一篇题为《刘鹤：没有对文革灾难的反思，就不可能有今天中国的经济增长》。

习神现在想给文革翻案，全面回到毛时代，现在，习神就是想把中共国拉回到文革，再次掀起一场党内斗争，将所有敌对势力打下台，党内其他势力也已经感受到了危机，他们试图利用各种渠道发声。路德先生在此提醒所有党内高层，习神接下来发起的党内斗争首先就是针对你们这些当权派，军队就是习神发起内斗的背后力量，他的目的就是为夺权。

**二、巴西总统公开宣称病毒就是中共国的生物武器**

DailyMail新闻网5月6日报道称：巴西总统博尔索纳罗(Jair Bolsonaro)对中共国提出了强烈批评，指责中共国制造了这场疫情，以引发一场“生物战争”。

巴西总统是在非常正式的场合公开宣称病毒就是中共国生物战的一部分，他作为巴西总统敢如此说话，背后肯定有情报依据，否则将给中巴两国造成巨大的外交压力，反对党也会借此批评巴西总统。

G7外长会议刚结束，Sellin博士就发表公开声明，WHO顾问梅茨先生也站出来接受采访，随后巴西总统就做出此番表态，彻底捅破了这层窗户纸，足以看出世界各国已经开始联合揭露中共病毒真相，这就是文贵先生在直播中提及这两周要发生的“病毒小风暴”。

巴西总统在发言中称，军方已经掌握中共病毒真相的情报，这在全球媒体引起强烈的反响，况且巴西不仅是南美最具影响力的国家，还是“金砖五国”成员，金砖五国之上就是G7联盟。此前，中共国与巴西关系还非常铁，现在巴西彻底反水。巴西总统号称巴西的“川普总统”，他其中一个儿子曾是班农团队成员，巴西总统第一时间就能从班农先生那了解中共病毒真相。路德社爆料，过几天，巴西将全网播放闫博士揭露病毒真相的视频，巴西总统此番言论就是在为接下来巴西将全国播放有关闫博士的采访进行舆论造势。

中共很清楚西方国家早晚会发现病毒真相，各国军情系统早已获得中共病毒的情报，但是中共却一直在作死的路上越走越快。美国在对中共国行动前，首先要做到预防再次发生病毒疫情，之后才会进行反击。巴西总统的公开表态意味着美国已经制定好防御方案。
