## 中共超限战灭港行动（十三）2012双普选被否和遁序渐进民主政制的骗局

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/18014509/5181.jpg)
2007、08年双普选被中共人大常委会否决之后，亲建制派政党民进党在2005年5月中提出希望香港特别行政区政府在2012年实行双普选。这个提议获得大部分立法会议员支持。泛民主派对200708年实行双普选达成了共识，但是被中共人大否决了，他们寄望2012年双普选同样达成共识，包括亲共政党民建联。大家都一致认识到，促成2012年双普选所有政党都责无旁贷。民建联的政纲曾一度改为「致力创造条件，争取2012年落实行政长官及立法会全部议席由普选产生」，自由党政纲中也出现致力于香港民主进程的积极语句「推动政改，全面普选可期待」，「争取最早在2013年普选特首及其后落实全面普选立法会」，可以看出这两个在政改上不够坚定的政党在当时也都很期待。当然这两个政党在得势后都变脸了。

2007年12月29日全国人大常委会发表草案指出香港可以在2017年实行行政长官普选， 2020年立法会实行全部议员也普选。 2012年的双普选方案则否决：

1. 2012年的行政长官选举方式不实行全面普选，
2. 2012年的立法会选举方式不实行全面普选，
3. 2012年的行政长官选举及立法会选举方式可以根据基本法的原则修改，
4. 2012年的立法会组成方式由地区直选及功能组别各占一半，
5. 2017年行政长官选举可以实行与普选产生，
6. 当行政长官以普选产生后立法会全数议席由普选产生。

这里我们可以看到中共又在耍拖延计，一面强力阻止民主选举，一面又画个大饼给香港人以希望，还是老套路，抽你两耳光给一颗糖。否定2012年双普选的依是什么？根据政制及内地事务局发表的《香港特别行政区向全国人民代表大会常务委员会提交关于香港特别行政区政制发展咨询情况及2012年行政长官和立法会产生办法是否需要修改的报告》里面是这样说的，「根据2004年4月6日全国人民代表大会常务委员会的释法（也就是中共人大否定2007、08年双普选的决定）⋯⋯《基本法》第45条和68条，并通过《基本法》附件一和附件二规定的行政长官和立法会的产生办法，并进一步规定根据香港特别行政区的实际情况和顿据渐进的原则，最终达致行政长官由一个广泛的代表的提名委员会按照民主程序提名后普选产生和立法会全部成员普选产生的目标」。看清楚了吗？依然是《基本法》45条48条以及附件，依然是遁序渐进，这个理由可以无休止阻止香港双普选，事实也证明到目前为止，香港被中共利用“遁序渐进”，遁没了原本大英时代留下的相对民主的东西，利用“遁序渐进”的手段，遁来了红色政权坐稳香港。

香港人争取民主政制从1988年的“八八直选”就已经开始了，到1991年立法局开始有的地方出现地区直选议席。但是双普选屡推屡败，一直被遁序渐进拖着敷衍着过渡时期拖到回归后，又拖到0708年，中共依然是拖和敷衍，最后否决，市民把双普选寄望于2012年，中共知道无法拖了就直接否定，然后给希望香港人2017年”可以“实行行政长官普选，2017不行又到下一届⋯⋯狗日的！太可恶了！就是这样中共用拖延战术来牵住住香港人欺骗香港人。中共根本没有诚心让香港食相民主政治，但是又要做出一个好人的样子来欺骗世界，玩弄香港。从1988年到2021年这二三十年的历史中，香港人的一直在争取民主进程这是无庸置疑的，但是到今时今日，民主越来越远，只剩下独裁。

港中矛盾越来越严重，累积到一定的程度都会爆发。 2007年12月29日人大常委发表草案否决2012年香港特别行政区行政长官和立法会选举方式后的当天香港人终于忍不住怒火，泛民主派发起了“反对人大否决普选大游行“，近7000人参加。民主之父李柱铭先生绝食抗议。 2008年1月13日香港泛民主派举行主题为「争取2012年双普选，不要2017假民主」游行。呼吁港人团结一致争取真普选。这次的游行有22000人，警方则称只有6800人。香港警方早就得中共撒谎真传，2019年反送中运动200万人游行，港共极权政府亦可以瞪眼说假话只有几十万人。
