墨博士：在大选前在幕后的正义力量需要对大选舞弊进行打击，那就要使黑暗的力量浮出水面，这个布局和诱饵就非常重要，什么诱饵会是这些黑暗的力量明知道是陷阱还不得不参与进来呢？什么时间点可以让他们猖狂的发动黑暗力量呢？那会不会就是这次大选，这次大选的正义力量到底是什么布局呢？

美国大选的法律战

路德说，现在大选已经进入了第二个阶段：法律战。朱利安尼先生带领他的律师团队已经去往费城起诉。还有很多川普团队在投票点进入法律诉讼。路德先生在咨询班农先生相关信息：大选接下来的诉讼战会从地方法院开始。如果裁决川普获胜，那么多出来的选票就不算。如果拜登团队不服判决可上诉。如上诉后还有问题，那第二个法庭裁决完后基本就是终审。如果认为终审裁决有违宪法，那么就到最高法庭。双方都准备好了诉讼战，这场大选最终会到最高法院。

对于大选，川普会上诉拜登舞弊。但如果川普赢了，拜登也会上诉川普有舞弊。中共在选票造假中不仅仅把假票投给了拜登，同时也投川普。我们要知道中共不是支持拜登那么简单，中共真正的目的是搞乱美国。在中共19大中，习提出的四个自信中所谓的制度自信，其实就是他们吃透了美国的政策和体制，中共自信地用他们狡诈的手段背后操纵美国的政治，让其进入国家混乱状态。对于美国人来说，可能比较难以理解大面积造假的行为。然而对于中共来讲大面积造假是司空见惯的日常。比如在前几个月美国海关查获了上万张来自中共的假美国驾照。如果中共的假驾照都可以以假乱真，那么伪造选票是轻而易举。当选票被制造出来后，中共要做的就是找人进行大量的投寄。这是低成本扰乱美国政局的行为。这就最终会造成民主党和共和党同时被假选票问题缠身，以致最后双方的支持者都不相信对方。中共造假从来都是粗制滥造，他们只要把局面搞得够乱，谁都不相信谁就足够了。当中共为双方制造假选票的时候，无论谁先发起诉讼，都会出现另一方不承认的局面。虽然事实是各自有假选票，但却可以相互说假选票来自对方的诬陷。中共甚至可以制造假的视频证据，让事先找好的人拿着假选票投川普，然后把大量假视频公布出来指控川普大选舞弊，从而构陷川普。

中共坐庄通吃 选举扰乱美国
