**香港的金色招牌已经变色、变酸，ICAC ****廉政公署这杯咖啡已经变了味。连廉政公署也没了，香港还是香港吗？**

如果说，港英年代的香港政府相信：文明就是要为敢于揭露黑暗的人提供保护；那么今天，港共年代的香港政府奉行：极权就是要对敢于揭露黑暗的人打击报复。
![]()![](https://gnews.org/wp-content/uploads/2021/01/一杯变了味的咖啡-（五）image003.jpg)
2020年12月28日，清晨7点，民主党前立法会议员林卓廷被廉政公署（ICAC）上门拘捕，罪名是：于2019年12月至2020年7月期间，向公众披露一位正接受廉政公署调查的人员身份，涉嫌违反《防止贿赂条例》第30条。
![]()![](https://gnews.org/wp-content/uploads/2021/01/一杯变了味的咖啡-（五）image005.jpg)
订立《防止贿赂条例》的目的是基于贪污案件一般比较隐密，而廉署在调查期间一旦走漏风声，很容易打草惊蛇，让涉案人员毁灭证据。并且正是由于此罪行不易为外人所察觉，通常需要依靠知情者举报，而保护好知情人的隐私才能赢取公众信任。因此廉署想要肃清贪腐，保密制度非常重要。而这第30条的存在，就是保证案件在调查阶段能够保密不被曝光，令举报人涉险以及让疑犯有机会破坏证据或者阻碍调查。维护好公众对投诉机制的信任，有利于反贪工作的长期开展。
![]()![](https://gnews.org/wp-content/uploads/2021/01/一杯变了味的咖啡-（五）image007.jpg)（图片截取ICAC影视作品）
根据《防止贿赂条例》第30条：任何明知或怀疑正有调查就任何被指称或怀疑已犯的第II部所订罪行而进行的人如无合法权限或合理辩解，而向公众、部分公众或任何特定人士披露该受调查人的身分或该受调查人正受调查的事实或该项调查的任何细节，即属犯罪。

简单来说，就是林卓廷向公众披露了某人因涉嫌触犯《防止贿赂条例》的罪行而被廉政公署调查，这个行为违反了 《防止贿赂条例》第30条。

这里有个很关键的部分，就是《防止贿赂条例》下的罪行具体指哪些？归纳来说指的是向公职人员提供利益而作出或不作出行为的贿赂罪等的成文法罪行。

回到林卓廷这单案件，他被控告的事由是去年12月31日，就元朗721事件的定性，他曾经反驳警察公共关系科高级警司江永祥的说法时，提及时任元朗警区助理指挥官游乃强因公职人员行为失当罪正被廉署调查。
![]()![](https://gnews.org/wp-content/uploads/2021/01/一杯变了味的咖啡-（五）image009.jpg)（图片截取自林卓廷youtube）
但是，公职人员行为失当罪，根本不是《防止贿赂条例》下的罪行，而是一项普通法的控罪。

2020年6月2日在立法会保安事务委员会上，廉政专员在回应邵家臻议员质询时，亦曾清楚表明过这点，公职人员行为失当罪不是《防止贿赂条例》指明的罪行。
![]()![](https://gnews.org/wp-content/uploads/2021/01/一杯变了味的咖啡-（五）image011.jpg)
那么，基于披露游乃强因公职人员行为失当罪正被廉署调查，而控告林卓廷违反《防止贿赂条例》第30条，从逻辑上、法理上都扭曲了法例原意，为了堆砌罪行而强行造罪，简直荒天下之大谬。

曾在2007年服务于廉政公署担任过调查主任的林卓廷在被拘捕时对廉署调查员说：「你我都知道，第30条指的哪些罪行，今时今日，廉署居然自甘沦为极权打压的工具」。其后，他特意换上ICAC的T恤到廉署总部接受审问，希望唤起昔日同袍的初心，不要滥用防贿条例，参与政治检控。
![]()![](https://gnews.org/wp-content/uploads/2021/01/一杯变了味的咖啡-（五）image013.jpg)（图片截取自林卓廷youtube）
反送中运动前，林卓廷因为持续追究梁振英的UGL事件，与党友前立法会议员尹兆坚成立名为「天下为公」的众筹活动，集资彻查前特首梁振英的严重舞弊案件。
![]()![](https://gnews.org/wp-content/uploads/2021/01/一杯变了味的咖啡-（五）image015.jpg)
在去年反送中运动721事件爆发时，林卓廷第一时间赶赴现场，成为了7.21元朗恐袭事件的受害者之一。过去一年他一直追究此事，多次公开拆穿警方谎言，甚至多番对警务处处长邓炳强开火，要求邓炳强解释警暴问题。在7.21事件一个月之后，他还联同当日的七名伤者向警方提民事索偿，并入禀控告邓炳强涉嫌违反《警察条例》，同时，成立两条投诉警暴热线，供市民提供证据，并要求廉署介入调查。
![]()![](https://gnews.org/wp-content/uploads/2021/01/一杯变了味的咖啡-（五）image017.jpg)
为了还真相给港人，为了还公道给死者，如此无畏无惧，自然成为中共眼中钉，遭受政治迫害。

公民党前立法会议员谭文豪批评廉署有违遏制政权，制止腐败的初衷，成为政治打手。社民连副主席「长毛」梁国雄声援林卓廷时亦怒斥「原告变被告，廉署变东厂」。

2017年林郑月娥上台后，开启三权合作，狼狈为奸的新纪元。中共为林郑铲除异己，铺平道路；林郑投桃报李唯『中共』马首是瞻，出卖香港，拱手让『假、贪、骗』无恶不作的中共所管控。廉政公署也彻底抛弃初衷，沦为中共东厂。这颗向往民主法治、拥护廉洁奉公的明珠也从此蒙尘。
