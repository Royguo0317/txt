据中共凤凰网财经频道11月11日报道，河南省新乡延津县石婆固镇新规定下达，对“一宅一户”超过167平方米标准的住宅按阶梯收费。该财经频道解读，每户宅基地住宅使用面积超过167平方米的，不管是以往建好留下来的楼房，还是新建的房屋，都需要缴纳“有偿使用费”。

笔者随即在中共党媒“今日头条”发现有关“宅基地超面积缴费通知单”的照片。该通知单显示其法律依据是“封政文2020年63号文件”。通知单规定宅基地住宅按照167平方米为标准，超过部分100平方米之内的按照每年每平方米5元收费，超过100-200平方米的按照每年每平方米10元收费，每增加100平方米就按照每年每平方米5元增加收费，以此类推。该通知单显示，一户五百平米宅基地（包含前后院）每年将被收费约2455元。

从这个新闻中，我们发现有几个看点。其一，在中共国“党要领导一切”的自上而下权力分发体系中，公开发布“宅基地收费”必定有中央级权力的暗示和默许；这样大面积官方收费却没有权威政策的发布，以便于随时抵赖。其二，城市房地产税讨论多年并试点了几个城市，尚未全面落地。而针对宅基地收费却悄悄地已经在实施了。最后，读者需要了解的是在中原河南农村宅基地，半亩地包含院子真的不算很大。这意味着这个收费标准把半数村民纳入了收费范围，看来中共是真的钱荒了。

按照李克强公开的收入人民币1000元/月以下的中共国劳动者有6亿人，再均摊老年人和儿童的未就业人数，估计收入在人民币400元/月/人，也就是说包含外出打工和耕地收入一家三至四口，年收入大约5000-9000元人民币。如果每年再缴纳2000多元的宅基地收费，必将对生活产生巨大的影响。

中共中央财政几乎把所有税源都变成了国税，例如把北京的地铁票收入都归了国税；而把就业、医疗、教育、文化等负担都抛给了各省地方财政支出，数十万老兵维权就是甩负担的典型事件；中共口口声声消灭贫困实际上就是变相取消了贫困县的中央财政补贴。

随着2020年各地方政府、各基层土地财政收入骤减，又缺少中央财政拨款，基层公务员发不出工资的消息层出不穷。中共基层国家机器钱荒告急，“针对宅基地的收费”即是在中央的默许和纵容下，各地暴力机器如警察、城管、辅警等公务员实现“自收自支”的尝试。
