![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/15191639/2021-06-15-161429-1.png)
“**跨大西洋联盟是建立我们的集体安全和共同繁荣的坚实基础。 在我看来，欧洲和美国之间的伙伴关系现在是而且必须继续是我们希望在21世纪完成的所有任务的基石，就像我们在20世纪所做的那样。美国完全致力于我们的北约联盟，我们将保持对第五条约的信心。”
—— 拜登总统，2021年2月19日**

拜登总统将于6月14日参加在布鲁塞尔举行的北约首脑会议，该会议汇集了所有30个盟国的领导人。在峰会期间，总统将重申通过北约建立持久的跨大西洋纽带，并强调美国对第五条约的坚定承诺——任何对一个国家的攻击就是对所有国家的攻击，将受到集体回应。 盟国领导人将发起一套雄心勃勃的倡议，以确保北约在2030年及以后继续为我们的公民提供安全。

跨大西洋关系是建立在共同的民主价值观的基础上的。北约的力量不仅来自于它的军事力量，也来自于它建立在尊重民主、个人自由和法治基础上的团结和共同目标，正如《华盛顿公约》所规定的那样。

现在，北约已进入第73个年头，它是历史上最强大和最成功的联盟。北约曾在冷战中降服共产主义集团，今天它为欧洲和北美的约10亿人提供安全保障。随着北约在近20年内逐渐减少其在阿富汗的军事存在，美国和我们的北约盟国和伙伴将继续通过民间活动和安全援助支持阿富汗人民。美国还将继续北约在伊拉克的非战斗咨询任务中与我们的盟友和伙伴并肩作战。

北约在协调COVID-19的民事反应方面发挥了关键作用，我们在世界各地空运了数百吨关键物资，并建立了近100家野战医院，同时保持其威慑和防御态势。这种援助正在拯救生命，并证明国防投资和能力建设提高了我们国家在面对各种危机时的恢复能力，而不仅仅是针对军事威胁。

**峰会的主要成果：**

一个新的战略概念：盟国将同意修订北约的战略概念。这一框架将指导联盟应对不断变化的战略环境，其中包括俄罗斯的侵略性政策和行动；中华人民共和国对我们的集体安全、繁荣和价值观构成的挑战；以及恐怖主义、网络威胁和气候变化等跨国威胁。新的战略概念将准备在2022年的北约峰会上通过。

更新网络防御系统：领导人将批准一项新的北约网络防御政策。该政策将加强联盟的协调，以确保联盟能够抵御我们面临的由国家和非国家行为者实施的恶意网络活动的日益频繁和严重威胁，包括对关键基础设施的破坏性勒索软件攻击。这项更新的政策还将为北约的政治、军事和技术网络努力提供战略指导，以阻止、抵御和应对全方位的网络威胁。领导人还将确认捍卫我们的网络和确保盟国依靠值得信赖的供应商来提供下一代电信网络的重要性。

保持我们的技术优势：领导人将申明，北约确保我们共同防御的能力依赖于保持我们的技术优势。盟国将启动一个国防创新加速器，以促进他们的技术合作，并加快采用将加强联盟的防御和安全的新兴技术。

应对气候变化：领导人将同意一项气候安全行动计划，并确定北约成为理解和适应气候变化对安全影响的主要国际组织的目标。他们将同意根据《巴黎协定》中的国家承诺，减少军事活动和设施产生的温室气体，并同意启动一个定期的全球气候和安全高级别对话。

强化威慑和防御：盟国将致力于实施新的军事概念和战略，加强北约的威慑和防御态势，以应对来自俄罗斯和其他地区的威胁。北约还将继续监测俄罗斯在乌克兰及其周边地区的部署。

更多地分担责任：自2014年奥巴马、拜登政府期间通过《威尔士国防投资承诺》以来，非美国的国防开支已经连续7年上升了。盟国领导人将再次承诺履行威尔士承诺的全部内容，并向北约提供现金、能力和现役部队的贡献。

对北约进行投资：盟国还将承诺确保北约在领导、人员和资源方面达到必要的水平，以落实峰会上作出的决定。领导人将同意确定额外的资源，包括通过北约的共同筹资，以加强北约应对当前和未来安全挑战的能力。

加强协商和凝聚力：盟国将承诺在北约加强政治协调，处理与他们各自和集体安全有关的所有事务。各国领导人还将重申他们对共同价值观的承诺，包括个人自由、人权、民主和法治。

更强大的社会：认识到我们的安全所面临的日益复杂的威胁，盟国领导人将申明，国家和集体的复原力对于可靠的威慑和防御是至关重要的，对于保障我们的社会、公民和共同价值观也是至关重要的。盟国领导人将发布《加强复原能力承诺》，概述未来的优先事项，包括供应链、关键基础设施和能源网络的安全，以及对大流行病和自然灾害的准备。

更深入的伙伴关系：盟国将通过加强与北约伙伴的对话和实际合作，提高北约加强基于规则的国际秩序的能力，包括欧盟和印度太平洋地区的伙伴（澳大利亚、日本、新西兰和大韩民国）。领导人将再次承诺北约的开放政策，该政策为任何认同我们的价值观并履行必要的责任和义务的欧洲国家提供了加入的途径。
