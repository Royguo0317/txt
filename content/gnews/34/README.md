在2020年的病毒危机中，德国政府的支出大大超过了收入。  联邦，州，地方和社会保障组织的赤字为1396亿欧元-低于先前的预期。

由于2020年底惊人的强劲经济增长，病毒危机给国家债务留下的缺口比以前预计的小。正如联邦统计局周三在柏林宣布的那样，联邦，州，地方和社会保障部门的支出总额比收入增加了1,396亿欧元。一月份的初步估计为负1582亿欧元。

统计学家解释说：“这是自2011年以来的第一次赤字，是德国统一以来第二高的赤字，仅次于1995年创纪录的赤字，当时信托债务被纳入了国家预算。”

进行修正的原因之一：欧洲最大的经济体到2020年底的增长快于先前的假设。尽管受到了限制，但在10月至12月的国内生产总值与上一季度相比增长了0.3％，此前1月份的初步估计仅显示了0.1％的小幅增长。结果，2020年的整体下滑幅度为4.9％，略低于此前的5.0％。尽管如此，这是自2009年金融危机以来的最大跌幅。
