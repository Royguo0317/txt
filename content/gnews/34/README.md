**一、新中国联邦新闻**

1.“路德时评”本日摘要（[早间](https://www.youtube.com/watch?v=oauysenfE28)、[晚间](https://www.youtube.com/watch?v=PN31Ao4YmWY)）:

Ⅰ. 纳瓦罗5月28日接受班农作战室（Warroom）采访时透露，川普政府时期已经在着手建立类似911调查委员会的总统委员会，以调查病毒起源、评估疫情损失，但被姆努钦、库德洛阻止。

Ⅱ. 美国总统乔·拜登已下令政府的主要研究实验室加入寻找中共病毒的真正起源，筛选大量以前未被审查的数据。其5月26日下令由能源部管理的17个国家实验室用先进的超级计算机处理大量的数据，协助情报界进行90天的冲刺，检查病毒是否来自中共国实验室。

2. 英国前内阁部长戴维斯5月29日将中共为阻碍全面调查中共病毒大流行是否因中共国实验室的泄漏而开始的努力比作切尔诺贝利核灾难。

3. 前国务卿迈克·蓬佩奥5月29日在对中共病毒疫情产生于武汉病毒研究所（WIV）这一说法发表评论时表示，WIV在进行民用研究的同时还从事军事活动。“我可以肯定的是，我们知道他们在该实验室内从事与中共解放军相关的工作，在他们声称只是良好的民用研究的同时，还在进行军事活动。”

4. 新西兰将支持澳大利亚与北京就其对大麦进口征收重大关税而正在进行的贸易争端，这是在应对中共经济胁迫方面加强区域团结的关键一步。

5. 《麻省理工学院技术评论》5月27日曝光，使用普通话的网络黑客冒充联合国（UN）代表，对中共国最西部的新疆境内的突厥少数民族维吾尔族人发动网络攻击。

6. 在5月28日的声明中，美国海关和边境保护局CBP表示已经对大连远洋渔业有限公司发出了扣留放行令，该公司位于中共国与朝鲜边境附近的港口城市大连。CBP发现该公司的船队至少有11项强迫劳动的指标，“包括身体暴力、扣留工资和虐待性的工作和生活条件“。这些指控包括对许多印度尼西亚工人的虐待。

7. 广州的官员已经颁布了有针对性的关闭措施，以努力控制新的中共病毒疫情爆发。

8. 印度尼西亚的目标扩大其潜艇舰队到目前的三倍，达到12艘，以应对此中共对其水域的多次入侵。雅加达还将加强其护卫舰舰队。

9. 中共国银保监会副主席梁涛表示，当前全球货币政策在宽松的基调下分化加速，西方发达经济体政策宽松幅度前所未有，决策的独立性明显下降，手段工具突破传统底线，不但导致财政可持续风险，而且产生巨大外溢效应，推高全球债务规模，推升金融市场泡沫和通胀预期。

10. 5月28日，在美国总统乔·拜登给情报机构90天时间提交一份新的报告之后，印度支持对中共病毒起源进行进一步调查的呼吁。

11. 今年7月1日前后，北京将大肆举办中共建党百年相关活动。不过，中共官媒新华社5月28日报道说，只邀请部分在京的港澳台记者和外国记者现场采访庆祝活动，不邀请境外记者临时来京采访。

12. 中共二十大召开前夕，重庆市公安局展开人事大洗牌，多名官员被调整。中共重庆市委书记陈敏尔也被盛传将调离山城，进京高升。稍早前有传闻指具有文宣背景的陈敏尔二十大入常，或接替王沪宁。

**二、世界新闻**

1. 美国国务院表示，拜登政府5月27日通知俄罗斯，美国将不会重新加入《开放天空条约》，即使双方准备在下个月举行领导人峰会。官员们强调拜登政府愿意在共同关心的问题上与俄罗斯合作。

2. 美国国务院5月26日发表声明，将古巴保留在未与美国反恐努力 “充分合作 “的国家名单上。该名单还包括叙利亚、委内瑞拉、朝鲜和伊朗。

3. 澳大利亚斯科特·莫里森和新西兰的总理杰辛达·阿德恩将在5月30日进行15个月以来的首次面对面会谈，此前有迹象表明这两个邻国在如何处理与中共国的关系上存在分歧。预计人权问题和中共病毒疫情将进入相关议题。

4. 针对白俄罗斯卢卡申科政权5月23日出动战斗机迫降民航航班，以逮捕独立记者和其同伴事件，由美英德等21个国家组成的媒体自由联盟（Media Freedom Coalition）29日发表声明予以最强烈的谴责。国际民航组织理事会27日宣布对该事件表示强烈关切，并决定开启实况调查。

5. 据法国公共预算部长奥利维尔·杜索普5月29日报告，法国国家预算赤字在2021年将上升至“约2200亿欧元”。
