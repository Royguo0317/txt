路透社消息，台湾国防部罕见地承认，欧洲国家正在为台湾的国产潜艇项目提供帮助，这一敏感项目并不仅仅从美国获得援助。美国政府在2018年为美国制造商参与该计划开了绿灯，外界普遍认为此举有助于台湾获取主要零部件，但尚不清楚哪些美国公司参与了该计划。

台湾多年来一直致力于改造其潜艇力量，其中一些潜艇可以追溯到二战时期。因中共声称台湾是其领土并咄咄逼人地宣称不放弃武力统一台湾。台湾现有舰队无法与中共抗衡，因此一直寻求方法增加其军事实力。政府支持的台湾CSBC公司(CSBC Corporation Taiwan)去年开始建造新潜艇，计划在2025年交付八艘潜艇中的第一艘。

由于担心激怒中共，欧洲国家通常就允许对台军售持谨慎态度，尽管台湾在2018年表示，它正在与一家总部位于英国的直布罗陀公司就新潜艇舰队的设计进行谈判。台湾的四艘现役潜艇中有两艘是上世纪八十年代在荷兰建造的，荷兰后来拒绝向台湾出售更多的潜艇。

法国向台湾出售了护卫舰和战斗机。台湾去年表示，正寻求从法国购买设备，以升级“天安号”的导弹干扰系统。

台湾的国防部长上个月说，美国已经批准向台湾出口装备舰队的敏感技术。上周三，台湾更是宣称决定购买美国洛克希德·马丁公司的爱国者(Patriot)地对空导弹的升级版，以提升空防能力，应对来自中共日益增长的威胁。

在周五晚些时候的一份声明中，台湾国防部否认了美国出版物《国家利益》(National Interest)称朝鲜曾讨论帮助台湾建造潜艇的报道。台湾政府表示，“在我们潜艇的发展过程中，我们从前没有、现在没有、将来也不会与朝鲜有任何接触，援助都是由欧洲和美国等国家提供的。”
