## **爆料革命揭露****CCP****病毒超限战**

2020年1月19日，路德社节目成为全球第一个发布关于CCP病毒真相的媒体，传递了病毒骨架为中共军方拥有，病毒已经人传人、强变异，疫情会大爆发等重要信息。信息源自香港大学公共卫生学院传染病研究中心病毒和免疫学专家闫丽梦（Li-Meng Yan）博士。当时，中共所有大外宣，大内宣均矢口否认病毒人传人。美国主流媒体及世界卫生组织（WHO）都声称，这只是流感。[【1】](https://www.youtube.com/watch?v=CLTjg03CPEs) 2020年1月29日，路德社再次第一个告诉全世界，羟氯喹是治疗CCP病毒的有效药物。[【2】](https://www.youtube.com/watch?v=m3PQss3oMkg&amp;t=113s) 据闫丽梦博士透露，羟氯喹的使用已有60年历史。羟氯喹可以通过减弱病毒的S蛋白与人类受体ACE2的结合，降低细胞因子风暴的产生，减少病毒给人体带来的致命伤害。在谨遵医嘱，严格按照安全剂量服用的前提下，现阶段，羟氯喹可以有效低起到预防作业，并用于早期治疗。据她披露，国内某些级别的高官都知晓羟氯喹对新冠病毒的治疗作用，他们及一些部队医院的医生可能自己也在服用。当中共国大量出口羟氯喹至俄罗斯等国时，大多数中国人并不知情，面临高风险的前线医护人员也不知情。为何中共不公布这些信息？闫博士谈到：“中共不公布病毒来源的真实信息、原始毒株的信息，不是因为中共可以找到疫苗，而是中共不想让世界知道病毒的真相，企图延缓科学界对病毒的反应时间、拖延疫苗和治疗药物研究的时间。”

闫博士稍后于2020年4月28日逃离香港，平安抵达美国。2020年7月10日，闫丽梦博士接受美国福克斯电视台专访，揭露中国政府隐瞒新冠疫情，威胁其家人，试图阻止她公布真相。闫博士表示，她决定前往美国并且向全世界讲出真相的原因并非源于政治目的，而是为了让人类可以免受CCP病毒带来的更大灾难。她了解中国政府将如何对待爆料人，以及中国政府与世卫组织之间的腐败关系。如果试图在大陆或香港讲出真相，她将面临被消失的命运。

2020年9月14日，闫博士公布了第一份有关CCP病毒来源的调查报告并指出：中共政府“有意制造并释放”Covid-19病毒。病毒并非来源自然，而是基于中共军方拥有的舟山蝙蝠病毒骨架，经过实验室改造，人工制造的。[【3】](https://zenodo.org/record/4028830#.X_3l2dhKiM_) 2020年10月8日，闫博士公布了第二份调查报告，直言：疫情的全球大流行是中共军方“超限生物战”的结果。她还在报告中指出，CCP病毒符合中共军方所定义的生物武器标准，而且影响力远远超过常规意义上的生物武器。因此，她在报告中将CCP病毒定义为一种“超限生物武器”。[【4】](https://zenodo.org/record/4073131#.X_3mlNhKiM8)

早在2020年3月，芝加哥大学的国际法教授弗朗西斯·博伊尔（Dr. Francis Boyle）博士就曾明确表示CCP病毒是一种进攻性生物武器。[【5】](https://www.youtube.com/watch?v=hhvDqPoONjc)[【6】](https://www.youtube.com/watch?v=H0fEZr2psMQ) 在老布什总统执政期间的1989年，博伊尔教授曾为美国生物武器反恐怖主义法案的起草人，该法案经参众两院一致通过。
