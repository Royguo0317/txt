**西班牙2021年4月21日电/西喜社——**

![]()![](https://gnews.org/wp-content/uploads/2021/04/GettyImages-1311497702-1200x800-1.jpg)图片来源：theepochtimes.com


佛罗里达州的共和党州长罗恩·德桑蒂斯（Ron DeSantis）表示，中共（CCP）应对其掩盖和不当行为负责，这加剧了CCP病毒的流行。

“至少，我认为这可能是那个（武汉）实验室逃出来的病毒。他们自己也知道，所以他们掩盖了事实。很长时间他们都对此闭口不谈。他们没有提供进入调查许可。”

“他们有意做出了这样的决定，我认为在此过程中可能已经付出了很多生命。因此，绝对应该追究他们的责任。”

中共官员知道武汉爆发病毒性疾病至少有六个星期，然后才向世界发出传染病的警告。同时，该政权逮捕了举报人，并审查了有关疫情爆发的在线讨论。中共将武汉与中国其他地区隔离开来，但同时却继续允许国际航班进出该巨型城市。

在4月1日接受《大纪元》采访时，迪桑蒂斯指责中共将其严厉的封锁模式推向其他国家，作为一项必要的缓解措施，导致像意大利这样的国家经济被摧毁。 迪桑蒂斯还谴责了包括安东尼奥·福奇（Anthony Fauci）在内的美国一些高级卫生官员，他们盲目地采用了共产主义政权的封锁模式。

“我认为像福奇这样的人，都在依靠中共来做这些事情。你不能相信来自中共的任何东西，”德桑蒂斯说。 “我认为，就他们的所作所为，赋予他们合法性就是一个巨大的错误。”

在福奇的领导下，国家过敏和传染病研究所（NIAID）花了纳税人的钱在中国研究蝙蝠冠状病毒，其中包括有争议的功能增强研究，研究冠状病毒如何从动物发展到人类的跳跃能力。在新闻报道显示一些参与研究的人与武汉病毒学实验室有联系之后，川普政府削减了对这些项目的资金，该实验室就位于报道的第一例CCP病毒病例出现的附近。

迪桑蒂斯说：“像福奇这样的人以及他们与中共的关系，在早期，他们根本不介意让中共国负责。” “他们一直在接受来自这个的所有宣传。这显然带来了负面影响，因为我认为这有助于让人们接受封锁。”

这位称自己是“天生反共产主义者”的州长认为，中共提倡封锁植根于共产主义政权的手段，该计划旨在使其他国家用不明智的限制来损害其经济。

迪桑蒂斯说：“西方由于采取了其中一些政策，从而对其自身造成了很大损害，事实证明，这些政策没有阻止病毒的扩散，但却对经济具有毁灭性。”

“我确实认为这与信息运作层面有关，他们相信，如果可以让其他国家封锁，他们就会自愿接受一路的宣传，特别是在欧洲，这最终将对中国有所帮助。我认为的确已经帮了中共。”

川普在2020年9月22日的讲话中说：“中共政府和实际上由其控制的世卫组织，错误地宣布没有人与人之间传播的证据。错误地说没有症状的人不会传播这种疾病。”

迪桑蒂斯不是第一次谴责共产主义政权。去年5月，他呼吁中共停止囤积个人防护装备，以“坑害世界其他国家”。

迪桑蒂斯当时说：“中共在国内和国际事务中影响力的不断增长，是对美国安全与繁荣无处不在的威胁之一。”
