## 接种疫苗的人捐献的血液被扔掉了
` 喜马拉雅意大利罗马达芬奇农场` [轉載自GNews](https://gnews.org/zh-hans/1338717/)

消息来源：今日意大利（today.it）
发布时间：2021年06月21日
翻译整理：萧亦

- ![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/13172013/23-9.jpg)


几周前在罗马举行的No Vax（不接种疫苗）示威活动中的意大利罗马喜剧演员恩里科·蒙特萨诺 (Enrico Montesano)在前线参加游行后，现在通过他的 Facebook 个人账户宣布：“接种中共病毒 疫苗的人捐献的血液将不被使用并丢弃”。他解释说，这些信息来自一个未公开的来源，但遗憾的是不久之后意大利血液志愿者协会（Avis）迅速否认了这一消息。

根据这位罗马演员的说法，一个神秘的消息来源告诉他，接种疫苗的捐献者的血液会凝固，因此袋子会被扔掉。因此，他的论点是，有必要让人们决定自愿选择是否接种疫苗。

意大利血液志愿者协会主席 Gianpietro Biola却表示这位喜剧演员的言论可以被定义为“危害公共健康并损害协会和捐助者承诺的危险言论”。他补充说：“接受疫苗后献血对献血者本人或接受输血的患者都没有任何风险。”

简评：在面对中共病毒的问题上，试图传递真相的言论似乎都被冠以“阴谋论”。很简单的道理，谁敢承认疫苗接种者的血液如果不可用那就是变相承认疫苗有问题。
