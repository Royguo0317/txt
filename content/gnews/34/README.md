# 彭佩奥：现在超过50个国家反对中共国的电信活动

[!\[\]()!\[\](https://spark.adobe.com/page/GiDjRI6RM3iES/images/94ca95fc-65fd-428e-b174-106e0824a7f4.jpg?asset_id=9c769c46-b6ce-4747-befd-723b825cebb6&amp;img_etag=%226b7edb82fc03911b00753602e8416a41%22&amp;size=1024)](https://spark.adobe.com/page/GiDjRI6RM3iES/images/94ca95fc-65fd-428e-b174-106e0824a7f4.jpg?asset_id=9c769c46-b6ce-4747-befd-723b825cebb6&amp;img_etag=%226b7edb82fc03911b00753602e8416a41%22&amp;size=1024)2020年11月10日星期二，华盛顿国务院，国务卿迈克-彭佩奥在媒体发布会上讲话。美联社图片/雅克林-马丁(Jacquelyn Martin）


美国国务卿迈克尔-彭佩奥（Michael Pompeo）周四表示，因担心电子间谍活动，已有50个国家和美国一起避开使用中共国电信设备。

彭佩奥先生还表示，财政部正准备对中共国实施更多的制裁，国务院将继续积极努力应对中共带来的威胁。

“我不想在总统会做出决定前捷足先登，但不要误解，我们为应对来自中共的威胁所做的工作还在继续”，他在接受电台主持人休-休伊特采访时说。”还有相当多的工作要做。国务院还有工作要做。财政部还有一些事情要做。”

彭佩奥先生说，美国转变对华政策的运动旨在扭转过去的政策，他称之为 “屈膝、讨好中共国，让他们游离于每一个国际规则之外”。

批评者说，前几届政府对北京的绥靖政策使美国的就业和繁荣受到威胁。川普的政策一开始以贸易为重点，后来转向保护美国的电信和基础设施不受华为技术公司和中兴通讯等中共国竞争对手的影响，而这两家公司都与该国有广泛的联系。

![]()![](https://spark.adobe.com/page/GiDjRI6RM3iES/images/90fb5222-0896-4c53-a0b9-65878bff4d60.jpg?asset_id=26748cbc-54d0-4293-9ab0-d983ef2eb31b&amp;img_etag=%225e2022a60a48da27ab93157861223247%22&amp;size=1024)

他说，”中心思想 “是，”如果美国信息要在一些电信基础设施上流动，我们应该知道谁在接触它，接触它的人不能是中共国军事机器的一部分，”彭佩奥先生说。”我们把这称为’清洁网络’。”

国务卿说，有50个国家加入了美国的努力—-这是对那些说川普政府没有与盟友和伙伴合作对抗中共国的批评者的反驳。他说，所有这些国家都同意，他们不希望中共国的电信设备或其他电子元件进入其关键基础设施内。

清洁网络计划于8月启动，目的是保护国内和国际通信网络和用户的个人信息不受中共国大规模数据挖掘的影响。其设想是说服各国政府对数字信息和基础设施采用国际公认的安全标准。

这包括使电信运营商、应用程序和提供这些应用程序的商店、云计算和有线电视网络免受中共国硬件和软件的影响。根据国务院的一份情况介绍，该方案旨在建立一种全面的方法，”保护国家资产，包括公民的隐私和公司最敏感的信息，使其免受中共等恶意行为者的侵略性入侵”。

参与该计划的有英国、日本、巴西的政府和电信供应商。在欧洲，27个北约盟国和26个欧盟成员国都是该计划的一部分。

在美国，Verizon、AT&T、T-Mobile、Oracle和Cisco(这些公司)也是参与者。

在另一个问题上，彭佩奥先生说，面对中共国的压力，继续支持民主统治的台湾是(美国)两党的共识。

“我认为核心的理解是，这是一个民主的模式，生活在台湾的人民应该得到尊重，让中共履行他们所做的承诺。我想这是双方都能同意的。”他说。

美国最近宣布向台湾出售MQ-9 “死神 “攻击无人机，并在早些时候批准出售价值约50亿美元的喷气式飞机和其他武器，作为加强该岛防御的努力的一部分。

[!\[\]()!\[\](https://spark.adobe.com/page/GiDjRI6RM3iES/images/d0c2d4a5-9f05-4c0d-b79e-5931178cc420.jpg?asset_id=23b460ab-04d0-4201-b5b7-4b4d073369e1&amp;img_etag=%22d1db90ad898d7e5696bde585f223aedd%22&amp;size=1024)](https://spark.adobe.com/page/GiDjRI6RM3iES/images/d0c2d4a5-9f05-4c0d-b79e-5931178cc420.jpg?asset_id=23b460ab-04d0-4201-b5b7-4b4d073369e1&amp;img_etag=%22d1db90ad898d7e5696bde585f223aedd%22&amp;size=1024)图片来自网络

彭佩奥先生还表示，川普政府还在继续揭露中共国在美国的影响力行动，包括向机构和游说者支付政府款项。

反影响的工作包括限制在大学校园内开办的与中共国政府有联系的孔子学院，以及中共在美国进行渗透和秘密影响行动的部门—-统战部的工作。

“我们已经踢走了在休斯敦活动的间谍，我们已经让他们的宣传部门在没有在美国注册为外国特工的情况下更难以开展活动，这些都是非常重要的事情”，彭佩奥先生说。

他补充说，整体政策的转变对外交政策的建立产生了重大影响，但怀疑者仍然存在。

彭佩奥先生说：”仍有怀疑论者，仍有一些人想用一些小生意、一些认识到的市场，来换取美国的国家安全”。

但这种情况正在改变。”长期以来，自上世纪70年代初，世界一直拒绝展示中共事实上的真实面目。”他说。”我想我们现在在欧洲可以看到，在非洲可以看到，在东南亚当然也可以看到。这些国家已经认识到了中共对他们国家的威胁。”

彭佩奥先生最近帮助组建的被称作印度-太平洋”四方”大国联盟, 包括美国、日本、澳大利亚和印度, 以对抗中共国的经济和军事扩张主义。

“我们都已经认识到，如果我们在南海共同努力，我们在印度洋共同努力，我们以四个伟大的民主国家可以继续工作的方式工作，我们将建立出一套共识，为该地区的旅行、航行，所有美国非常关心的事情提供更多的安全”，他说。
