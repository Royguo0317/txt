《九香》是一部讲述中国母亲的电影，可能看过的人并不多，但这可能是我看到过中国电影里最深刻的一部。电影的主人公叫九香，是20世纪五六十年代，一个生活在东北农村里最普通的农家妇女, 19岁嫁人,朴实能干,育有四子一女, 可一场雪暴压垮了她家的草房，等她从废墟里救出了五个年幼的孩子后， 又接到丈夫被倒塌的砖窑压死的噩耗。在那个窗户只能用纸糊，全家睡在一张炕上的年代，一个女人拉扯五个孩子何其辛苦和艰难。为了孩子能多吃点，九香经常不吃饭，等孩子睡觉后去舔孩子饭碗里的残渣。

丈夫生前好友老关是个离了婚的单身男人，也是一个特别朴实的老好人，看到九香的艰难，也一心喜欢漂亮朴实的九香，对她一家关照有加。一开始帮九香修缮房子，后来经常送粮食接济九香一家，真心实意地对五个孩子好，一来一回两人互生爱意。可是在那个偏执的年代和环境，没有爸爸的孩子们经常受到村里其他孩子的欺负，又因为老关经常照顾九香一家，村里的闲言闲语不断，老大和老二因为孩子们骂他们是野孩子大打出手，让九香心里非常难受。后来老大偏执地认为老关是败坏母亲名声的坏人，经常给老关使绊子，还踩坏了老关送给母亲的亲手做的发饰，即便这样九香也从来没有打骂过孩子。 老关和九香就这样隐忍着孩子们的不理解和村里人的诋毁，互相扶持艰难地生活着。直到后来有一次九香带女儿去镇上买东西，因为九香被老关拉去买布料，而女儿凑热闹看杀猪被意外撞到骨折流血不止，吓得九香抱到医院给女儿输血才抢救了回来。事后九香自责认为全是自己的过错，女儿才遭受了这么大的罪，于是坚决地和老关分手。

之后的日子，九香过得更加艰难，曾一度被镇上面铺店老板说服把最小的儿子天星过继给他们，而换回粮食。九香以为天星在面铺店可以过得好些，偷偷去看，却发现天星经常在店铺里帮忙刷碗。九香实在受不了又去跪着求人把孩子要了回来，从此再苦全家人也没有分开。在九香和老关分手后，其实老关从来没有停止对九香全家的帮助，总是默默地帮她度过难关。几年后终于把五个孩子慢慢拉扯大，老大老二即将考上大学。九香觉得这个时候对孩子没有亏欠了，准备去找老关，而老关在这个时候因为母亲病重离开了农村，回到了城镇去照顾母亲。而当九香得到消息去火车站追老关的时候，看到的是火车快速驶去的一幕，九香最终错过了老关，心里也满含对老关的歉疚。

后来孩子们都成家立业，九香也积劳成疾，身体每况愈下，终于在一次去城里探望孩子的时候鼓起勇气去找了老关，结果老关几年前就因病先走了，支持着九香撑到现在的那股精神一下就没有了，很快九香随老关走了，带走一辈子的遗憾。

看过电影，不知不觉泪流满面，电影里的一幕幕和文贵先生描述过的他成长的环境何其相似。20世纪五六十年代的中国，经历着大跃进人为的饥饿和无休止的斗争，中华传统文化被毁于一旦，文贵先生的母亲和九香一样，代表着无数个中国母亲，即使在经历那种物质最匮乏、精神最压抑、黑白最颠倒的年代，她们依然传承着善良、勤劳、宽容和牺牲，给孩子们在黑夜里点亮那盏灯，在风雪中撑起那道墙。中国母亲的伟大，就是这些最弱小、最平凡、最朴实的女人用自己的生命孕育并传承着中华文化里的真和善！为了我们每一个人的母亲不再像九香那样压抑自己，可以自由地选择，为自己真正幸福地活着， 让我们站起来，一起干掉中国共产党！！！
