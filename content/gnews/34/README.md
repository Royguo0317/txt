在中美脱钩大局已定的局势下，维系与欧盟的关系成了中共国的一根救命稻草。然而，历经七年谈判的《中欧全面投资协定》去年年底达成协议，今年5月20日却被欧洲议会投票中止，类似永久冻结。虽然中共急忙邀请塞尔维亚、匈牙利、爱尔兰、波兰四国外长访问，力图挽回局面，但是就连亲共的彭博社也慨叹，“自从冠状病毒爆发引发一系列外交争端以来，几个欧洲国家对中国的态度越来越不友好。”

《中欧全面投资协定》中止绝非偶然，经过郭先生的解读，我们才知道背后有那么多故事，有新中国联邦战友的贡献与付出。这次新中国联邦一周年庆典中，天使闫丽梦博士被班农先生誉为拯救全人类的英雄，美国摩根家族灵魂人物的加入，多位美国政治人物的现场发表演讲，都在表达对新中国联邦的支持。今天中欧协议的冻结，关键是世界知道了中共的邪恶，必须追溯中共病毒真相。今天中共国失去了欧洲，明天必将失去全世界。

**译文：**

近期，在一次具有政治和象征意义的投票中欧洲议会中止了《中欧全面投资协定》，实际该协议处于深度冻结状态。中共国的情绪广泛报道为“震惊和失望”。为签署该协议双方已经历了七年的马拉松式的谈判，直到去年12月才结束。近期日媒分析，习近平的“强硬外交”显然适得其反，导致中共“失去了欧洲”。

此前，布鲁塞尔（欧盟）曾提出停止谈判，直到中共国取消对个别欧盟立法者和学者的制裁。中共现在正展开外交攻势以挽救该协议，最近举行了一系列高层会议，从波兰到匈牙利，还有塞尔维亚、法国和爱尔兰，旨在影响欧盟国家并维持对《中欧全面投资协定》（CAI）的支持，

中共国对欧盟官员的制裁是对西方社会批评中共国的报复。西方谴责中共国残暴的人权记录，尤其是以新疆维吾尔族穆斯林遭受的苦难，美国和英国致力于长期推动，欧盟也积极参与，国家社会积极呼吁并作出惩罚。尽管中欧经过七年的谈判，在去年12月乔-拜登入主白宫前，双方终于实现了一项协议。许多人当时批评说，“匆忙”允许中共国签署该协议，从长远来看，这将损害欧洲的安全和主权。

然而，正如这些媒体观察到的那样，“在中欧和东欧的大部分地区，对中国的热情已经降温。因为人权问题和停滞不前的投资，这些失望的伙伴都倒向了美国，令北京感到震惊。”

仅仅过了几个月，最终由华盛顿主导，中共国的“胜利”（签署协议）就在与西方的针锋相对的敌对行动中瓦解了。5月20日欧洲议会的投票，是一个压倒性的、具有巨大象征意义的欧盟集体对中共国作为欧洲的“伙伴”表示不满：599票赞成冻结批准，30票反对（58票弃权）。

《彭博社》也对习近平在欧盟的深入推进尤其是东欧和巴尔干地区，最终失去欧洲的这一主题进行了分析：“投资协定曾被视为欧洲独立于美国，以及中国有能力与采取更温和方式的美国盟友合作的证明。”周末的报道还强调了这一点：“自从冠状病毒爆发引发一系列外交争端以来，几个欧洲国家对中共的态度越来越不友好。”

在过去的一周里，中共外交部长王毅认为，一个具有里程碑意义的七年投资协议不应该与中共认为的“单独问题”（新疆种族灭绝）联系起来。

“投资协议不是单方面的恩赐，与新疆有关的问题关系到中共的主权和安全”，王毅周二在慕尼黑安全会议的远程电话中说：“欧盟中的一些人试图将不同性质的问题联系起来，将贸易问题变成政治问题，这是不可接受的，将一事无成，”这似乎是中共国在处理与西方各国关系和贸易恶化时一贯强调的重点。
