## 中共超限战灭港行动（卅五）雨伞革命的影响（上）
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1328561/)

#### 搜集/撰文：岁月如歌 / 封面合成：文粤


![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/17014605/6171.jpg)
铜锣湾最后一块占领地失陷，标志着“让爱与和平占领中环”运动从最初的和理非演变成雨伞革命的所有努力争取在警方暴力清场下暂时告一段落。港共并没有半步退让，依然是“枪杆子里出政权”的强硬态度。梁振英依然是中共的应声虫、传声筒，指出2017年特首选举必须按《基本法》和831决定，市民追求民主必须依法若只讲民主不讲法治，是无政府状态，并希望非法诉求者遵从《基本法》 。

有头发边个（谁）想做瘌痢？中共修订《基本法》想以此来愚弄和箍死香港人。对港人玩弄文字游戏，可惜港人不傻，法律精英无数，都知道基本法玩的猫腻。邓小平的「宜粗不宜细」有意不明确界定，基本法的“循序渐进”实现民主、普选就能遁序几千年，把香港人玩残。基本法是港人头上的紧箍咒，港人有任何诉求，港共政府从不听取，宁敢与民为敌，亦要听从中共主旨。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/17014629/6172.jpg)图片来自维基百科
学民思潮领袖黄之峰表示，香港政府未解决社会撕裂问题、不愿对话，因此，明年“必然”会再有占领行动。学联秘书长周永康亦认为，可以预见未来的游行，若警方不放行，市民即会就地坐下来，发生占领。周永康还表示，这次雨伞运动“未胜不败”，已令部分人觉醒了，不再只顾家庭和事业，他预期未来会有更多人投身民主事业。中共喉舌《人民日报》海外版发文章指责“占中”运动给香港造成数以万亿的经济损失，破坏香港国际形象，破坏香港和谐，更破坏香港法治和商业环境。并称占中结束，「但香港的政改之争未必会随之结束，香港反对派政党近日扬言，会抵制第二轮政改咨询，并否决政改方案」。是谁不按当初的承诺在破坏香港的法治？香港损失这么大中共有感到心痛吗？

历时79天的雨伞革命对香港造成的影响是绝对的，但是没有和平占中与雨伞革命，中共潜移默化、温水煮青蛙式的独裁统治对香港的影响更是可怕，那是永远的代价。运动期间对交通、旅游、商业零售等是立竿见影的，金融方面在金钟、中环港府一带在占领期间，就有17间银行及29个营业网点暂停服务。零售业也因示威人数太多影响员工正常上下班，食肆不能正常营业，金店更是惧怕安全混乱问题大部分关门。学生也暂停上课，交通方面是最大的，270条路线巴士路线受影响，150万人乘车受影响。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/17014645/6173.jpg)图片来自:BBC中文（一批市民拒绝离开被警方拘捕。）
政治方面的影响是主要的，中共通过这次占中运动，知道香港人不好骗不好对付了，会想出更强硬的办法来对付港人。基本法委员会主任李飞认为，从法理角度出发，公民提名不符合《基本法》，为什么不符合？中共压根儿就不打算让公民提名，也不愿实行西方的民主政治，要走挂着遮羞布的有中国特色的香港民主法治，也就是由中共钦点，再由市民投票。因此根本就没有这一条法例，当然不符合啦。所以中共和港府开口闭口都要以《基本法》作为政治法律基础，由它框定。中共这两年在国际追究CCP病毒真相上的态度，也和利用《基本法》控制香港人一样。一直要以它控制着的世界卫生组织或联合国为标准做决定来对付各国质疑，欲以此逃脱追责。谭书记现在反水了，看它怎么狡辩，若中共控制的联合国喽啰们也反水了，它还能不能这么坚定以这两个国际组织做盾牌呢？

2015年6月李飞出席记者会表示将今次政改视为试金石，港人及港官员是否拥护「一国两制」一试便知。李飞毕竟是中共思想教育出来的满脑子“霸王”的官员，用他的马列主义思维衡量被大英帝国自由法治沐浴了155年的香港人，认为“皇命不可违”，“圣旨不可抗”必须百分百顺从，抗旨必暴民，无法接受。他说「为什么一定要拼命反对8.31决定，那是不尊重人大常委会的决定……你反对这一决定，你到底把一国放在什么位置上？」他以为香港人是大陆奴隶。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/17014655/6174.jpg)图片来自twitter
李飞警告泛民议员，政改方案是体现是否支持“一国两制”和《基本法》的试金石，「是转向合作共赢正道，还是继续走对抗俱损歪道的分水岭」，恐怕也是中共剔除对其“不忠”的立法会议员借口吧？

李飞强调「831决定是全国人大常委会决定，具有最高法律效力，不存在还未实施便改变的可能。该决定不止限于一次选举，而是长期有效，2017年后的普选都要按照831决定，惟实施后可能就实际问题作出修改」。
