## 美国各州表示，Facebook、Twitter必须做更多的工作来阻止COVID-19反病毒者

据《路透社》作者：[Jonathan Stempel](https://www.reuters.com/journalists/jonathan-stempel)，2021年3月25日报道：

（路透社）–美国12个州的总检察长周三指责Facebook公司和Twitter公司在阻止人们利用其平台传播冠状病毒疫苗不安全的虚假信息方面做得太少。
![]()![](https://gnews.org/wp-content/uploads/2021/03/1-341.jpg)
文件照片。2019年11月26日拍摄的这张插图中，Twitter和Facebook的标志以及二进制网络代码。REUTERS/Dado Ruvic/插图

民主党检察长在给Facebook首席执行官马克-扎克伯格（Mark Zuckerberg）和Twitter首席执行官杰克-多西（Jack Dorsey）的信中说，缺乏医学专业知识、往往受经济利益驱使的”反疫苗接种者”利用这些平台淡化COVID-19的危险，夸大疫苗接种的风险。

他们呼吁两家公司执行自己的社区准则，删除或标记疫苗错误信息。

信中说，反疫苗者控制了Facebook、Instagram和Twitter上65%的公开反疫苗内容，并在这些平台和谷歌的YouTube上拥有超过5900万粉丝。

它还说，一些错误信息针对黑人和其他接种率落后的有色人种社区。

信中说，”鉴于反疫苗者对你们平台的依赖，你们在防止有关冠状病毒疫苗的错误信息传播方面具有独特的优势，这些信息对我们各州数百万美国人的健康和安全构成了直接威胁，这将延长我们的经济复苏之路。”

Facebook发言人Dani Lever表示，该公司已经删除了数百万条COVID-19和疫苗的错误信息，并试图通过定期引导用户访问卫生当局的可靠信息来打击”疫苗犹豫症”。

Twitter表示，它已经删除了超过22400条与COVID-19帖子相关的推文，并优先删除可能造成”现实世界”伤害的内容。

周三的信件由康涅狄格州、特拉华州、爱荷华州、马萨诸塞州、密歇根州、明尼苏达州、纽约州、北卡罗来纳州、俄勒冈州、宾夕法尼亚州、罗德岛州和弗吉尼亚州的总检察长签署。

扎克伯格，多西和谷歌母公司Alphabet Inc的首席执行官桑达尔-皮查伊将于周四在众议院的两个小组委员会面前就打击在线虚假信息作证。

冠状病毒大流行已使全球超过1.24亿人患病，并造成270多万人死亡。
