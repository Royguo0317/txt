## 全球征签活动——支持《美国制止强摘器官》法案

#### 作者：xinyuan 审核：Pv0 / Peace Wind 封面：昆仑竹

近日，“终止中（共）国滥用器官移植国际联盟（ETAC）”发起一项网上征签， 呼吁全球公众签名，支持美国国会参议院和众议院通过《制止强摘器官法案》。

《制止强摘器官法案》具有以下五重意义：

1. 授权美国政府拒绝或者撤销购买非法人体器官者的护照。

2. 要求美国国务院每年就外国强摘人体器官的状况发布报告。该报告将负责辨识强摘器官的责任人和实体机构。

3. 要求向参与了强摘人体器官的外国机构提供器官移植外科医生培训的美国机构的情况，提供年度报告。

4. 禁止将器官移植手术设备出口给参与强摘人体器官的实体机构。

5. 对参与或支持强摘人体器官的外国官员和实体机构实施制裁。

中共这些年来，实行了针对犯人、法轮功学员及部分其他宗教和少数民族团体成员的器官活摘罪行。这些人士被关押在劳教所或监狱等场所，他们的器官供商业移植谋取利益。主要参与部门是政法委和军委两大部门。

针对中共的恶魔行径，联合国禁止酷刑委员会2015年12月9日要求对中华人民共和国政府强摘包括法轮功学员器官的指控进行独立调查，该指控内容连续多年被列入联合国及美国国务院和美国国会等机构的人权报告书。美国众议院也于2016年通过343号决议案再度谴责并要求美国国务院彻查并采取行动。加拿大国会国际人权委员会（2014年12月）陆续通过决议谴责中共政府强摘法轮功修炼者等宗教人士的器官。欧盟通过《欧洲理事会反对人体器官贩运公约》在以色列、西班牙、台湾陆续完成立法，禁止到中共国进行器官移植旅游。

一直以来，很多国人都觉得自己活得还算不错，除了食品，空气质量问题，我们是总体来说是安全的。一切都岁月静好。我们哪里知道，在这平静的湖面下隐藏着多少罪恶，埋葬着多少破碎的家庭，又有多少冤魂在暗夜里悲鸣不止。

两次世界大战期间，世界在混乱的叫嚣与哀嚎声中，带走了上千万人，同时也留下了几千万残障士兵和平民。这一切，虽然邪恶，但都发生在阳光下。甚至被拍成影视作品，以警醒世人。

而在中共执政期间，在这看似和平的几十年间，竟然非正常死亡了近一亿人，更有4亿未出生的胎儿被以各种残暴的方式结束生命。你可能感到震惊，也可能不敢相信和承认这些血淋淋的事实。因为这一切完美到除了当事人家庭，其他人都浑然不觉。而很多当事人家庭都被封口，被威胁到无法说出真相。一切罪恶就这么静悄悄地进行着。到现在为止，不但没有停止的意思，反而更变本加厉。

很多国人希望死后保持完整的身体，可多少同胞因为与中共政见不同或信仰问题，被活摘器官，残忍杀害。更不必提很多人正常死亡之后被发现身体的器官不翼而飞，连人类最基本的愿望都无法达成。器官捐献本应是某个个体发自内心令人尊敬的选择，但在这片土地上，却被中共滥用，成为了人类历史上最大的罪恶。演绎了又一桩突破人类底线的集体事件。

作为国人，如果我们不做些什么，我们该如何面对这些含冤逝去的生命？ 我们还有几个5亿可杀？还有多少国人的器官可以被非法移植？谁知道里面会不会有我们的朋友，我们的家人，甚至我们自己？今天，美国网站发起了全球征签活动，希望同胞们都能积极参与，动动手指，尽自己的一份力量，做我们能做的。尽早终结中共的无底线罪恶。
