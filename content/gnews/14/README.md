# **推特已暂停川普总统的帐户，威胁将完全删除**

[!\[\]()!\[\](https://spark.adobe.com/page/QS4vXx8Bslwlq/images/fcb6da4d-0af7-422e-87c8-41c0068b39a4.png?asset_id=0e3679a6-1b93-4c20-a068-80669e9b893c&amp;img_etag=%22e27644074e0c2691246cab3ef2b9212d%22&amp;size=2560)](https://spark.adobe.com/page/QS4vXx8Bslwlq/images/fcb6da4d-0af7-422e-87c8-41c0068b39a4.png?asset_id=0e3679a6-1b93-4c20-a068-80669e9b893c&amp;img_etag=%22e27644074e0c2691246cab3ef2b9212d%22&amp;size=1024)

**曾经压制了有关亨特·拜登硬盘和中共病毒信息的社交网络服务平台，现又暂停了美国总统的帐户。**

在美国国会大厦发生了混乱场面后，由杰克·多尔西（Jack Dorsey）运营的推特于周三晚上宣布：

*“由于华盛顿特区前所未有的持续暴力局势，我们要求删除@realDonaldTrump今天早些时候发布的三条推文，因为这些推文屡次严重违反我们的公民诚信政策。这意味着在删除这些推文后，@realDonaldTrump的帐户将被锁定12个小时。如果推文不删除，该帐户将继续保持锁定状态。”*

[!\[\]()!\[\](https://spark.adobe.com/page/QS4vXx8Bslwlq/images/c367444f-0c84-4711-bec9-eb1d75548874.jpg?asset_id=2507c7aa-af3e-48d6-8508-b943e7733fdf&amp;img_etag=%22043fb35dd721a5bba5cf75e7654f2adc%22&amp;size=2560)](https://spark.adobe.com/page/QS4vXx8Bslwlq/images/c367444f-0c84-4711-bec9-eb1d75548874.jpg?asset_id=2507c7aa-af3e-48d6-8508-b943e7733fdf&amp;img_etag=%22043fb35dd721a5bba5cf75e7654f2adc%22&amp;size=1024)

在抗议者闯入美国国会大厦并且一名抗议者被警察谋杀后，川普总统在推特上发布了一段呼吁和平的视频，片刻之后，推特就公布了这一消息。

推特随后威胁要暂停这个有8900万关注者的帐户：

*“若未来违反推特规则，包括我们的公民诚信或暴力威胁政策，将导致@realDonaldTrump帐户永久停用。”*

与许多居民和政府雇员报告称，无法在华盛顿特区市区内使用他们的手机一样，《国家脉动》也已发现，目前在白宫附近，手机无法正常使用。

长达五年以来，川普总统及其基础选民一直呼吁共和党人遏制大型科技公司的力量。大多数人拒绝了，而且许多保守组织继续从大型科技公司那里收取大量金钱，从而在华盛顿特区为他们宣教。
