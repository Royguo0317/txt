## 德國未來政治領導人的反共表態

**簡評：**

德國一直以來都是中共藍金黃的重災區，從德國車企在中共國的優厚待遇，中共國與德國的貿易占到中共國與歐洲貿易總量的三分之一，默克爾多次到訪與中共國多位領導人親密接觸，到德國前總理飛利浦·羅斯勒與王岐山家族和海航間說不清道不明的關係……

而現如今，我們卻看到德國毅然決然站在了歐洲反共滅共的最前沿，“戰汪”王毅出訪德國本希望能得到外交支持，卻碰了一鼻子灰：德國外長呼籲撤回《港版國安法》，多位議員在人權問題上強勢發聲，德國媒體也警告稱“中共國正在歐洲散播極權理念”。應該說，作為共產主義邪靈的誕生地，德國人對於馬克思主義共產政權是個什麼玩意再清楚不過了。更何況中共病毒已經散播到全世界，德國既沒有客觀條件也沒有任何理由，再去為了利益而與中共沆瀣一氣。

在默克爾已經明確表態不再尋求2021年繼續連任的情況下，德國最大政治勢力基督教民主聯盟候選人的表態，就變得尤為重要。可以預見的是，不論是誰接任默克爾，都將一改德國十幾年來親共的國家戰略。從德國各界的呼聲來看，未來在5G、一代一路、貿易、人權等等問題上對中共施壓已成定局。雖然傳統的建制派政治正確依然具有很大慣性，川普及平民主義勢力得到老牌政治集團的認可難度很大，但僅從“滅共”這一立場上來說，雙方是一致的。而這，對於任何一位嚮往法治、民主、自由的中國人來說，就已經足夠了。

**問答：德國領導人候選人諾伯特·羅特根**

**德國聯邦議院(Bundestag)負責中共國、5G和跨大西洋關係的外交事務負責人**

[!\[\]()!\[\](https://spark.adobe.com/page/Q97rn89oLLfyH/images/f56a84e2-d3e4-4e15-bf35-d3f58e381b5f.jpg?asset_id=538da26c-664f-4c1d-b4a3-75596f29b572&amp;img_etag=%22f3174ee10495fc7aa0f006d256a28b0f%22&amp;size=1024)](https://spark.adobe.com/page/Q97rn89oLLfyH/images/f56a84e2-d3e4-4e15-bf35-d3f58e381b5f.jpg?asset_id=538da26c-664f-4c1d-b4a3-75596f29b572&amp;img_etag=%22f3174ee10495fc7aa0f006d256a28b0f%22&amp;size=1024)諾伯特| Maja Hitij 攝影/ 蓋蒂圖片

作為有關接任安格拉·默克爾(Angela Merkel)的下一任德國總理大選系列報導的一部分， POLITICO首席歐洲記者卡尼辛內基（Matthew Karnitschnig）對話諾伯特·羅特根，德國聯邦議院外交事務委員會主席。羅特根是角逐右翼執政黨基督教民主聯盟(CDU)領導權的三位候選人之一。

以下是採訪文字記錄，為篇幅和明晰稍作編輯。

**China關於中共國**

[!\[\]()!\[\](https://spark.adobe.com/page/Q97rn89oLLfyH/images/0cc34aeb-f6dc-412c-9768-18e62217ba11.jpg?asset_id=3702e5a8-c8b4-40d4-932f-046457141635&amp;img_etag=%2265f2f81ad4f31d5d4c781c3d51e53479%22&amp;size=2560)](https://spark.adobe.com/page/Q97rn89oLLfyH/images/0cc34aeb-f6dc-412c-9768-18e62217ba11.jpg?asset_id=3702e5a8-c8b4-40d4-932f-046457141635&amp;img_etag=%2265f2f81ad4f31d5d4c781c3d51e53479%22&amp;size=1024)

POLITICO: 您如何看待歐洲與中共國的關係?我們在近期內是否可能達成正在談判的投資協議？

羅特根: 歐盟在這方面立場很明確:我們堅持，中共國對歐洲企業的開放程度應與中共國企業在歐盟受到的待遇相當。這不僅意味著公平的市場准入，還意味著在不被迫轉讓技術，或被迫與中共國企業合作的情況下自由地做出商業決策。中共國更樂意維持現狀而不想在一些方面做出讓步，這就是談判在過去7年未能成功的原因。

POLITICO: 這是否意味著你支持歐洲當前的路線。

羅特根:是的，歐洲要求互惠的立場是正確的。我們可以由此看到在一些領域歐盟可以真正團結起來，它同樣擁有影響力和全球實力。在貿易政策方面，我們是一股世界力量。

POLITICO: 然而，與中共國的關係仍然困難重重…

羅特根:總的來說，與中共國的關係處於變化之中。越來越明顯的是，事情的發展方向將取決於這一事實:中共國不僅是個大市場。在習近平的治理下，這個國家已經變成了一個非常重要的地緣政治參與者。必須對此進行重新評估，而在德國，評估尚未完成。這是對中共國政治的根本變化和中共國在習近平治下新的自信作出的反應。

POLITICO: 你不覺得我們西方人對中共國有些天真嗎?在過去20年大部分時間裡，歐洲行業一直對知識產權盜竊發出警告，並抱怨中共國的商業環境。

[!\[\]()!\[\](https://spark.adobe.com/page/Q97rn89oLLfyH/images/79bf6590-e69b-4b5e-b38d-20461731d6e8.jpg?asset_id=fa984488-f40e-4089-95ce-ffbe070482ad&amp;img_etag=%224a03fc462390a459f6e22e2e1532b3a7%22&amp;size=1024)](https://spark.adobe.com/page/Q97rn89oLLfyH/images/79bf6590-e69b-4b5e-b38d-20461731d6e8.jpg?asset_id=fa984488-f40e-4089-95ce-ffbe070482ad&amp;img_etag=%224a03fc462390a459f6e22e2e1532b3a7%22&amp;size=1024)

羅特根:這是事實，但在過去的一兩年中，一種新的現實主義出現了。這是我們必須繼續努力的方向。現在還不晚。

POLITICO:什麼發生了變化？

羅特根:今天的情況在兩個方面有所不同。長期以來，德國工業界的看法是:無論我們受到了多麼差的對待，我們都必須留在中共國，因為市場太大了。雖然沒有人大聲說出來，但中共知道情況就是這樣，這對任何談判立場都會產生可以預見的負面影響。

POLITICO: 第二個變化呢？

羅特根: 20年前，中共國對自己的認知與現在大不相同。它曾認為自己是一個發展中國家。他們關注的是自身發展，沒有意識到他們需要為國際政策投入資源。這在習近平的領導下也發生了變化。中共國現在正在奉行一種自信的、擴張性的外交政策。

POLITICO: 以哪種方式？

羅特根:中共國在“一帶一路”倡議、在南中國海的領土主張，以及最近在疫情大流行期間的行為，都表現出新的信心。中共國明確希望其他國家接受中共國關于冠狀病毒的說法，並公開對中共國的支持表示感謝。

POLITICO: 現在歐洲該怎麼做？

羅特根:關鍵的是，現在對這些變化做出應對還不算太晚。但窗口正在關閉。這就是為什麼我們歐洲人必須在中共國問題上達成共識。我們的做法應該是現實主義的，而不是敵意的。

POLITICO: 談到南中國海，柏林方面一直在考慮是否向南中國海派遣德國海軍，以發出一個信號。你支持這一觀點嗎？

羅特根: 海洋自由是國際法的一項基本原則，這對我們這個出口國具有重大意義。我們應該準備為此做點什麼。與此同時，我認為從德國的角度來看，考慮到我們目前的能力，其他地方有更迫切的需求。我們的首要任務應該是製定全面的對中共國的歐洲戰略，德國在這方面發揮著關鍵作用。

POLITICO: 人們認為你是個跨大西洋主義者。難道你不想在中共國問題上和美國更緊密地合作嗎？

羅特根:一旦華盛頓恢復理性政策，包括對中共國的，屆時我也會支持跨大西洋對中共國政策。

**5G和華為**

![]()![](https://spark.adobe.com/page/Q97rn89oLLfyH/images/06fd64b1-07a6-46b9-83a5-3485f8f0008a.jpg?asset_id=5f08ee5e-4236-4796-bfc1-8fc9ca343d71&amp;img_etag=%22a63717eabea857f1ce674a07fb0e4a79%22&amp;size=2560)

POLITICO: 這對於當前圍繞5G和華為的爭論又意味著什麼？

羅特根:在關於5G的辯論中，我們需要盡快做出決定。我的立場很明確:擁有一項我們可以用於工業的歐洲技術既符合我們的利益，同時也保證我們的數字主權和國家安全。我們決不能允許自己變得對外依賴，而容易受制於人。在我看來，有關5G的決定具有至關重要的戰略意義，非常適合作為歐洲對中共國共同政策的切入點。

POLITICO: 您如何看待德國政府的做法？

羅特根:在過去的一年裡，我們在德國進行了激烈的辯論，最終我們沒有做出任何錯誤決定。現在，人們希望盡快做出正確的決定，這是由想要擴大德國和歐洲數字主權的目標所推動的。我對此是樂觀的。

POLITICO: 美國很早就做出了決定。為什麼德國需要這麼長時間？

羅特根:因為德國有不同的利益考量。但在過去的一年裡，此處爭論的焦點有所變化。一開始，德國政府毫不懷疑中共國企業應該參與其中。現如今既然一家公司的參與已經變成信任問題。這就將取決於外國政府對該公司影響的程度。

POLITICO: 川普對德國的態度似乎使跨大西洋兩岸的關係複雜化了。如果喬·拜登獲勝，你對他有什麼期待？

羅特根:不幸的是，美國政府在所有這些問題上的立場—國防支出，北溪(天然氣管道)，華為等等方面，一直是適得其反。伊朗是另一個例子。如果拜登獲勝，我預計他的政府將回到基於理性思考與合作的伙伴關係上來。

**跨大西洋關係**

![]()![](https://spark.adobe.com/page/Q97rn89oLLfyH/images/9f4d7b61-cbea-4533-926b-c8e856a7dd6f.jpg?asset_id=51ae39f3-bbdf-4ec2-b234-f6268126a2ff&amp;img_etag=%228a12ba761f2899a9fc6f5ddfa6b2370c%22&amp;size=1024)

POLITICO:你如何定義“夥伴關係”？

羅特根:我預計，我們歐洲人將被要求做更多的事情。如果白宮發生變化，我們需要準備好通過做出實質性貢獻，塑造大西洋兩岸的關係。在美國方面，我期待回歸理性、合作和多邊主義。也許我們會接收到新的“具有領導力的伙伴關係”的邀請。這一次，我們應該接受它。

POLITICO: 所以你期待著美國外交政策的徹底逆轉？

羅特根:不，完全不是。外交政策的重點將放在中共國，更籠統地說，尤其是在印太地區。結論將是，與歐洲人合作比以往任何時候都更為緊迫。因此，新一屆美國政府將明白，目標不應該是將美國的立場強加給德國和歐洲人，而是找到一種新的分工合作方式。雙方都需要做出貢獻。

POLITICO: 如果川普再次當選，北約還有未來嗎？

[!\[\]()!\[\](https://spark.adobe.com/page/Q97rn89oLLfyH/images/0f4ae441-0889-4254-8e55-f11a7478d875.jpg?asset_id=dd98f480-75db-4fc3-bdc1-4d93a8278de1&amp;img_etag=%2249cd45b94e82d0bc2a01d086287bb6f3%22&amp;size=2560)](https://spark.adobe.com/page/Q97rn89oLLfyH/images/0f4ae441-0889-4254-8e55-f11a7478d875.jpg?asset_id=dd98f480-75db-4fc3-bdc1-4d93a8278de1&amp;img_etag=%2249cd45b94e82d0bc2a01d086287bb6f3%22&amp;size=1024)

羅特根:如果他再次當選，我們將看到我們在過去四年經歷的加速。首先最重要的是，對美國自身而言其內部的分裂和好鬥情緒是真實的。其結果是，美國將看到自己發揮國際領導作用的能力進一步削弱。一個內部分裂、充滿敵意的國家，終有一天會失去處理外交事務的能力。因此，我們看到美國繼續退出國際政治舞台，留下一個其他國家將非常樂意填補的真空。

POLITICO: 冷戰結束30年後，美國為什麼還要繼續保障歐洲的安全？歐洲已經足夠富裕和強大了，對嗎？

羅特根:從根本上說，這是事實，尤其是從美國人的角度來看。但像這樣的事情需要時間。塑造國際秩序的能力需要的絕對不只是金錢。但作為歐洲人，我們不能永遠迴避對自己安全負有的責任。

POLITICO: 你理解美國對德國未能承擔更多責任的失望嗎?或者這是一種誇大其詞？

羅特根:巴拉克•奧巴馬也有這種挫敗感。但是現在的美國政府被一種每當別人不遵從時就施以懲罰的邏輯所驅動。在這種基礎上建立夥伴關係是不可能的。更重要的是，這種做法會帶來破壞性的後果，對美國的影響大於對德國的影響。在那種不合理的情況下工作很困難。

POLITICO: 你們的政黨被認為是偏保守的。即便如此，你和你們許多基督教民主聯盟的人在美國總統選舉中支持一位民主黨人。為什麼？

羅特根:傳統上，基督教民主聯盟與共和黨的關係更為密切。從歷史上看，共和黨人在戰後時期更多地參與了國際事務。德國統一期間，和老布什的關係達到了頂點。我們最近被迫見證了川普帶著敵意接管了共和黨。

POLITICO: 難道你不認為，如果只能與美國兩個主要政黨中的一個合作，從長遠來看對德國來說會很艱難嗎？

羅特根:如果白宮發生變化，那共和黨就必須對過去四年進行反思。共和黨人將不得不重新定位自己。在未來，我們將看到不同的面孔。

採訪於9月15日在柏林進行。
