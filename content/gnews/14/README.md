面对邪共的猖獗，越来越多已经觉醒的美国人正在唤醒着更多美国人。参议员汤姆·科顿（Tom Cotton）发表了一份长达80页的报告——“击败中共：有针对性的脱钩和经济持久战争”，该报告与其他关键文件一起，可以作为美国人思考如何与中共国竞争的指南。

《布雷特巴特新闻》（breitbart）2月27日报道了科顿参议员的的报告【1】。

文章说，科顿的报告设想了一场针对中共的冷战，也就是说是一场耐心的经济、战略和意识形态的斗争，不打一枪就能击败中共，而不是（通过）真正的热战。毕竟，我们用50年赢了对前苏联的冷战，今天，阿肯色州的立法者相信，我们可以在第二次冷战中再赢一次。

为了打这场新冷战，科顿指出，我们需要国内的结构性变革。目前的美国联邦政府组织在挫败中共国（可动用）全社会（力量）进行经济持久战方面并没有优势，这反映了美国制度性方面的严重差距，即使考虑到过去几年（我们推行了）积极的改革和措施，这些机构的表现仍远远低于它们的实力。

在这最后一句话中，我们可以看到科顿参议员对川普总统政府的致敬——“过去几年的积极改革和措施”——尽管他的文件保持着对未来的紧迫感：还有很多事情要做。

科顿也牢牢记住了历史，他指出，1947年的《国家安全法》（National Security Act）是第一次冷战的基础性改革之一，是宝贵的行动框架，正是这项立法重组了五角大楼，建立了一支新的独立的美国空军，并成立了中央情报局。

在调查目前的情况中，科顿列出了一长串的问题和关切，强调了制定一项关于贸易、投资和知识产权（IP）的全面国家战略的必要性。例如，他指出，在过去十年中，有记录在案的中共国知识产权盗窃案在美国激增了1300%；给美国造成的知识产权损失总计数千亿美元，甚至数万亿美元。

此外，科顿将重点放在中共瞄准的领域，包括美国高等教育，通过贿赂和暗箱操作（等手段），然后（科顿又）将重点放在（那些中共国）尚未获得满足的需求上，涉及关键稀土元素的供应、5G宽带建设和人工智能的安全发展。

至于解决办法，他把重点放在了该在的地方：运用哈密顿（Hamiltonian）的重点强调保护和促进我们本国生产重要商品的能力。对于那些想深入研究这些主题的人，科顿提供了不少于326条注释。

然而，在这里我们可以停下来观察一下，这位参议员对有些问题很重视，对另一些却不够。例如，他28次提到“好莱坞”，关注的焦点是美国娱乐业为了追求中共国市场和中共国的投资，在全球范围内自我审查，这是很不厚道的。然而与此同时，“宗教自由”却没有被提及，中共国受压迫的基督徒，或者更受压迫的穆斯林维吾尔人、法轮功的不幸追随者都没有被涉及。

没有人会认为科顿不是坚定的捍卫宗教自由者，哪怕是共产主义政权的拥护者也不会，然而将宗教信仰作为一个关切点纳入其中，会使这份文件更有力量。

毫无疑问，科顿未来会有更多关于中共国和自由，包括宗教自由的报告出来。与此同时，这份报告——老式的说法是白皮书——以一个有力的注解来结束：把国家安全问题放在首位来考虑与中共国的经济竞争，加强我们的工业基础和供应链，更好地协调和保护我们国家的科研企业，美国可以建立必要的政府基础设施，以正确利用其巨大的能力，并在长期的经济战争中取得胜利。。

对美国来说，幸运的是，并非只有科顿一人提出对中共国的担忧，呼吁对这个极权的共产资本主义政权采取更强有力的政策。例如，两年前，这位作者注意到另一份有关中共国威胁的强有力的报告，那份报告是由参议员卢比奥（Marco Rubio）撰写的。

事实上，对中共国的担忧正在广泛地形成共识，就在2月24日，《亚特兰大宪法》（The Atlanta Journal Constitution）杂志报道了四名佐治亚州国会议员，劝说卡特中心停止和放弃与中共国政府“前线组织”合作的努力，卡特中心当然是由前总统吉米·卡特创建的非营利组织，可以肯定的是，卡特中心在中共国资金的诱惑方面并不是独一无二的，与“大熊猫”的这种合作在美国各地都有发生，虽然这可能是合法的，但肯定是错误的。

有趣的是，佐治亚州的四位议员巴迪·卡特、德鲁·弗格森、乔迪·希奇和奥斯汀·斯科特都是众议院共和党研究委员会的成员，在印第安纳州众议员吉姆·班克斯的积极领导下，他们在中共国问题上表明了雄心勃勃的鹰派立场。

因此，我们可以看到一个新的运动正在进行中，因为“美国优先”成为“保卫美国”，希望与世界各地志同道合的盟友联合起来。

没有人想和中共国开战，也没有人会认为中国人民是敌人，然而，习近平领导的中共是一个威胁，因此我们必须清醒地认识到这一威胁。今天，中共正把它肮脏的触角伸向世界各地，就像苏联共产党80年前所做的那样。

目前，拜登政府和民主党在华盛顿拥有权力，因此我们必须看看他们如何选择应对来自中共的威胁。

希望现任民主党领导人能够坚强地站起来，然而，如果不是这样，那么出于国家安全的需要，火炬将传递给新一代的共和党领导人。退役军人汤姆·科顿（Tom Cotton）在2月26日的保守党政治行动（CPAC）会议上，就国内外的安全问题发出了强烈的信号，他站在了新卫队的前列。（全文完）
