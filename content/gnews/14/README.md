**原文翻译：**

据日本《每日新闻》 12月21日报道，日本政府周一批准了创纪录的5.34万亿日元（合516亿美元）的2021年度国防预算，作为应对“中共国带来的安全挑战”工作的一部分。

新近扩大的军事预算，比2020年增加了0.5％，划拨3.23亿美元为日本引进“能够从对方射程之外攻击敌方船只”的远程导弹。

东京在12月18日表示，将开发新的能够命中西南冲绳群岛周围更远距离军舰的“防区外”反舰导弹，包括北京也声称拥有主权的有争议的东海尖阁诸岛附近区域。

日本防卫大臣岸信夫（Kishi Nobuo）在上周五的新闻发布会上说：“我们西南诸岛周围的安全环境已变得严峻。我们必须以适当的方式对此作出回应。”

当时路透社指出，东京发展更远程导弹计划的公告标志着自9月当选以来，日本首相菅义伟的第一个重大防务政策决定。随着日本军方继续购买“可用于打击朝鲜导弹发射场的空射导弹”，并考虑向炮兵部队中增加“可能到达中共国地面目标的打击型武器，例如巡航导弹”，该导弹计划和支持该计划的预算将在周一发布。

近几个月来，日本对中共国在东海的活动增加渐感担忧，其中包括屡次入侵有争议的尖阁诸岛周围水域。

岸信夫于12月14日表示，他“同意中共国国防部长魏凤和寻求在双方官员之间尽早建立热线的请求”，以调解未来关于尖阁诸岛的争端（中共国称其为钓鱼岛）。两位国防部长已通过视频会议讨论此事。

共同社称，在视频会议之后，岸信夫对记者说：“我强烈要求不要采取任何可能导致局势升级的行动。”岸信夫还补充说：“他敦促中共国不要试图凭一己之力单方面改变现状。”

共同社称，东京和北京“在2018年启动了防务部门之间的沟通机制，以避免在海上和空中发生意外冲突”，但自那以后，这两个亚洲邻国在建立热线方面”进展甚微”。

据《每日新闻》报道，日本周一的新国防预算创下历史新高，“这是连续第七年，国家也增强了应对朝鲜导弹和核威胁的能力”。

2021年的军事预算将划拨7.06亿美元用于制造新型战斗机，新型战机“预计将耗资约400亿美元，并于2030年代投入使用。”路透社周一报道，在美国洛克希德•马丁公司（Amerian Lockheed Martin Corp.）的协助下，三菱重工有限公司将领导该飞机的研制，这是日本近30年以来的第一架新型战斗机。
