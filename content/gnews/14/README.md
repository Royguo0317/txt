## 开悟的代价——浅谈社会制度改革

七哥在11月30日的直播中说:“任何认知和开悟都是要付出代价的，……就这么大的代价，连我们身边的很多人都没有唤醒，……悲剧是你(自己)造成的。”“共产党可怕，被共产党洗了脑的这些僵尸粉们更可怕。”

洗脑是从欺骗开始的，要想欺骗就得控制舆论，控制了舆论才能控制人们的思想，控制了思想，就达到了洗脑的目的。这个现象在国内是普遍存在的：看看身边的同学同事，家里的长辈亲属兄弟姊妹，哪一个没有上过当受过骗；再看看媒体上报道的，上当受骗的多如牛毛。当中有上小当的、有上大当的、有被骗的倾家荡产的，有被骗了一时的，也有被骗一世的。就这样的代价，绝大部分的中国人还是相信共产党，他们的理由是：中央的政策是好的，是被下边执行歪了。我身边听到王岐山贪腐的人有八成不相信，这就是七哥说的“可怕的僵尸粉们”。可悲的是，不仅中国人被洗脑，世界上很多老牌资本主义国家的人也被洗脑，如英、法、德、意、罗、西、加等，尤其是美国人，中共病毒杀死几十万人都没有开悟，直到大选舞弊。

全球被洗脑控制的人大体可被分成两种类型，其中普通人占被洗脑的大多数，这是“僵尸粉”的主力军。另一类则是世界主要经济体的富商、政要、军事、科技等被引诱和中共勾兑利益而被控制的人。这类人不乏有一些是上了贼船骑虎难下，最终相信他们一定是后悔莫及的。“不碰南墙不回头，不见棺材不落泪”可以被看作是这些人的一大共性。七哥说到这些事时非常的痛苦。确实啊，这个代价太大了，中共病毒害死了不计其数的世界人民，全球经济损失无法估量，甚至可能要倒退几十年——这是全人类的代价！

为什么中共要对全世界人民洗脑？为什么全世界会有这么多人上当受骗被洗脑控制仍不开悟？难道是这帮邪灵恶魔们智商太高、聪明绝顶、有巫术？还是被洗脑的人智商太低、太愚笨、太贪婪？ NO！都不是！“共产主义者”们的目的不言自明，但“僵尸粉”们的问题就要综合分析了。这其中既不能片面的责怪“僵尸粉”们，也不能全部归罪于“共产主义者”们。若要想避免这种代价的再发生，我认为必须大力改革社会制度。

我很早就看过一篇有关心理学的文章，大意是：对人的智商做了调查，发现超常智商（天才）的人约占3%，低智商（弱智）的人约占3%，其余的94%都是正常智商的人。在现代文明社会（不包括专制独裁国家），一个国家的公民大致可以分为三大群体：

（一）社会的管理者。基本上是由少数的智商较高、能力较强、社会经验丰富、精力旺盛又有管理欲望的人来做的。这些人不一定都是超常智商的人，但没有弱智人。他们中一部分是专职政客，掌握着符合其职务的权力；一部分是为社会创造了大量的财富，同时也拥有大量的财富和大量的资源并且都有较高的权力的人。

（二）支撑这个社会运转的主力，就是中产群体。他们依靠自己的劳动付出，创造了大量的物质财富和剩余价值（缴税），同时也在消耗着大量的社会财富。这些人数量庞大，占总人口数的三分之二以上（财富和人数不是对等的），对会的稳定平衡起着决定性的作用。他们之中绝大多数既不知道政治与生活的紧密关系，一般情况下也不是太关注社会管理层（政治）的问题——但他们却关心与自身利益密切相关的问题（美国各行业、各领域被中共严重渗透却不被察觉就是明显的例子）。

（三）没有劳动能力或不能完全自给，靠财政发放生活补贴的人，只占少数人。

通过以上的分析，再结合近几十年来世界各国发展的状况，基本上能确定——第一和第二类人群对社会制度的运行都起着非常重要的作用。当中他们各自的分工不同：第一类主导管理，第二类主导监督。社会能否正常运行，就看他们能否相互制衡。这种制衡就涉及到一个国家制度的设计、法律的制定、对权力的监督、对公民的约束以及对大众文化的引导教育等一系列问题。这些内容庞大复杂，以后专文讨论。我只根据自己对中国过去的了解和认知，对未来的新中国联邦提出几点建议：

1. 制度和法律的制定要详细全面，涉及到所有方面；
2. 政府权力不能太大太集中，要大量分散，包括各企业各专业领域，不能有任何绝对权力的存在；
3. 监管要严苛、透明（除了国家军事），还要有对监管人员的再监管，要设立严格的辟嫌条款；
4. 法律条款要和每个人的责任权利紧密挂钩，有明确且易执行的奖惩条款，使四分之三以上的公民自觉地参与社会秩序的管理——既不能形成多数人的暴政，又不能忽视每一个人的合法权利。


最后强调，制定法律条款一定不能忽略人性的弱点。
