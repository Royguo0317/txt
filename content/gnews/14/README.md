- [Gnews： 微信、抖音等59款中共国应用在印度被永久禁止](https://gnews.org/zh-hans/820351/)


据《印度时报》1月26日报道，自去年6月底以来，印度政府对中共国的267个应用程序(不同批次)采取了前所未有的行动，已永久禁止59款应用, 包括抖音、百度、微信、阿里巴巴的UC浏览器、购物应用俱乐部工厂、小米视频通话,和BIGO Live等。印度政府最初根据《信息技术法》第69A条对中共国的应用程序提起诉讼，指控它们从事有损印度主权和完整、国防、国家安全和公共秩序的活动。此前，印度IT部官员对这些中共国公司的回应表示不满，包括它们的数据收集和数据处理方法，以及有关数据安全和隐私的问题。其他被认为是永久禁止的包括SHAREit、 like、微博和小米社区。鉴于两国之间日益紧张的局势，政府已经开始挤压中共应用程序在印度的影响力，特别是在它们被指控滥用印度公民和企业数据、监视和从事反印度活动之际。前几轮禁令清单中的中共国公司已收到了印度IT部的通知，并被要求提供有关其在印度的业务、订阅者、数据收集和信息处理等细节。长长的问题清单还要求了解有关中共企业“未经授权访问数据”的细节，说明这些企业的安全功能薄弱，可能通过未经授权访问数据导致间谍活动或监视。官方消息人士称，上周这些中共公司单独收到了有关永久禁令的通知。

### 2020年工业增速逐季回升 信息通信业平稳较快发展

## 

今天（1月26日），国新办举行新闻发布会，工业和信息化部表示2020年我国工业生产增速逐季回升，全年规模以上工业增加值同比增长2.8%。新兴产业发展加快培育壮大，态势良好，高技术制造业同比增长达到7.1%，高于整体工业4.3个百分点。工业和信息化部表示，2020年信息通信业保持平稳较快发展。全年电信业务总量按上年不变价计算同比增长20.6%，新建开通5G基站超过60万个，终端连接数突破2亿，实现全国所有地级以上城市覆盖。2021年将在实现地级以上城市深度覆盖的基础上，加速向有条件的县镇延伸，同时按需做好工业、能源、交通等重点领域的网络建设。
