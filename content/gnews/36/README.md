# **与武汉实验室有紧密联系的皮特•达扎克表示，新冠的起源将在“几年内”被发现**

2014年，总部位于纽约的非营利组织健康生态联盟（ EcoHealth Alliance）的主席皮特•达扎克（Peter Daszak）获得了安东尼•福奇（AnthonyFauci）博士的美国国立卫生研究院（NIH）的资助，与武汉病毒研究所（WIV）等合作，研究蝙蝠冠状病毒如何“进化并跳入人类群体”。

[!\[\]()!\[\](https://spark.adobe.com/page/OKxZ7vg7SvIHK/images/7705bc12-4193-4f73-9a56-3264c8d6e93f.jpg?asset_id=f484d7a3-2c30-42cd-bcda-a72b5ef828a1&amp;img_etag=%22ac5201a2e023ce0aa0271e0bdf62a642%22&amp;size=2560)](https://spark.adobe.com/page/OKxZ7vg7SvIHK/images/7705bc12-4193-4f73-9a56-3264c8d6e93f.jpg?asset_id=f484d7a3-2c30-42cd-bcda-a72b5ef828a1&amp;img_etag=%22ac5201a2e023ce0aa0271e0bdf62a642%22&amp;size=1024)皮特•达扎克（Peter Daszak），生态健康联盟（EcoHealth Alliance）主席

该项资助的初始资金为666442美元，从2014年6月开始，2019年5月结束，以“理解蝙蝠冠状病毒出现的风险”项目名义每年支付370万美元。值得注意的是，奥巴马政府在2014年10月，也就是皮特•达扎克的合同开始4个月后，削减了对“功能增强”研究的资助，而据《华盛顿邮报》的Josh Rogin报道，**武汉病毒研究所**在“蝙蝠女”石正丽博士的领导下，多年来 **“公开与美国大学和机构合作，参与功能增强研究”**。

Rogin去年4月曝光了外交电报，表达了对武汉病毒研究所安全的严重担忧后，他说（在一本新书中）：“许多为实验室辩护的科学家都是石正丽博士的研究伙伴和资助者，**比如全球公共卫生非营利组织“生态健康联盟”的负责人皮特•达扎克；他们的研究与她的研究息息相关。如果武汉实验室被牵连到这场大流行中，他们将不得不回答很多棘手的问题。”**

[!\[\]()!\[\](https://spark.adobe.com/page/OKxZ7vg7SvIHK/images/43a6af65-c88a-40e5-af7d-53185a0da679.jpg?asset_id=7127da93-5219-475c-8b2b-5b68180d9821&amp;img_etag=%221e23ac285c772a8a90bfad0a41201f3c%22&amp;size=2560)](https://spark.adobe.com/page/OKxZ7vg7SvIHK/images/43a6af65-c88a-40e5-af7d-53185a0da679.jpg?asset_id=7127da93-5219-475c-8b2b-5b68180d9821&amp;img_etag=%221e23ac285c772a8a90bfad0a41201f3c%22&amp;size=1024)石正丽（中间），皮特•达扎克（最右）/ 图片来源：新兴病毒组（Emerging Viruses Group）

简而言之，皮特•达扎克一直坚持“实验室泄露”理论是不可能的，而通过中间宿主的随机自然起源是唯一的答案——存在着**巨大的利益冲突。**

据报道，去年8月，美国国立卫生研究院在COVID-19大流行的情况下削减了对皮特•达扎克的资助，直到健康生态联盟满足了几个要求——**包括外部团队对武汉病毒研究所的亲自检查——才扭转了终止资助的做法，同时暂停资助。**

事实证明，达扎克（他又坚称SARS-CoV-2不可能来自他工作过的实验室）是世界卫生组织（WHO）对武汉病毒所进行超级合法“视察”的最主要成员，并说现在臭名昭著的病毒学研究所与他合作多年。巧合的是，**这次旅行恰巧满足了达扎克继续接受NIH资助的要求**。

不出所料，据《华尔街日报》报道，达扎克说，世卫组织团队在武汉之行中了解到，“携带与新冠病毒同属一家族的冠状病毒的动物的肉在华南市场上出售”。主要的假说是：“携带祖先病毒或密切相关病毒的蝙蝠或其他野生动物物种感染了农场动物或个人，然后由他们携带到华南市场。”

荷兰病毒学家Marion Koopmans回应了达扎克的立场，重申该团队认为病毒可能从实验室中泄露出来的可能性 “极小”——因为他们没有证据表明病毒是这样做的（这与缺乏自然起源证据有某种程度的不同）。

达扎克称：“有一条从武汉到华南各省的通道，在那里的蝙蝠中发现了与SARS-CoV-2最接近的亲缘病毒。”他补充说：“而这是一条非常重要的线索。”

所以现在——有了这一点背景——你会原谅达**扎克在周三说“可能要过几年**”我们才会发现COVID-19的确切来源。

他在伦敦智库Chatham House组织的一次网络研讨会上说：“我相信我们很快就会找到答案”。 他把“很快”定义为：“**在未来几年内**”，当“**我们将有真正重要的数据，关于它从哪里来，以及它如何演变**” [ZH：而且，在一个难以想象的巧合中，它最开始出现在武汉病毒研究所的门口——在那里，他们正在实验如何使蝙蝠冠状病毒更容易感染人类]。

达扎克还指出，他并没有向武汉病毒研究所的研究人员询问一个神秘失踪的蝙蝠病毒数据库。
