尽管之前拜登家族已经与万向集团、叶简明的华信集团等开始合作，但是当王歧山遇上乔·拜登，他们的双修乱性、腐败和权欲本性使得彼此臭味相投、一见如故。渤海产业基金是渤海华美的第一投资方，紧紧抓住王歧山的渤海系就能揭开中共的钱是如何披着合法的外衣转向拜登家族的。王歧山从2008年担任中共国副总理，2012年进入中央政治局常委，开始了声势浩大的“反腐运动”。反腐运动的实质是借反腐名义打击政治对手，吞并和转移大量的公司产权和账户。这时候的王歧山一手遮天，已成为真正的中国金融沙皇，以至于布隆伯格称王岐山是世界上最有影响力的人。 在2011年5月9号，第三轮中美战略与经济对话在华盛顿开幕，时任副总理的王岐山作为国家主席胡锦涛特别代表与作为副总统的拜登开始亲密接触。

**渤海华美的主要投资方渤海产业基金：被隐藏的身份秘密**

2006年12月28号总规模达200亿元人民币的渤海产业投资基金诞生，集万千荣光于一身。它被镀上了中国第一家政府批准的人民币产业基金的金字招牌，承载着监管层对本土人民币产业基金试点的殷切期望。然而，鲜为人知的是，数年来，渤海产业基金一直刻意隐藏着一个见不得光的秘密。尽管最终出资人都是国有机构，**但渤海产业基金管理有限公司在法律结构上是一家名副其实的中外合资企业**。

商务部亦曾明确规定由渤海基金管理公司管理运作的渤海产业基金需按照外资投资的规定报批。知情人士称，渤海产业基金在07年就对内下达了“封口令”，禁止员工向外透露外资身份的信息。在渤海产业基金的官方网站上，一段介绍该基金的话异常醒目：“渤海产业投资基金的设立，打破了中国私募股权投资基金领域由外资基金主导的局面，推动中国本土股权投资的专业化发展。”

渤海产业基金突出阐明了其本土基金的身份，明确地澄清它的非“外资”属性。清一色的国有出资人背景无疑也让外界很自然地认同其本土基金的说法。早在2010年就有网商记者胡中彬对渤海产业基金管理有限公司写过详细的调查报道，详细跟踪了该公司故意隐匿外资身份的过程。

2006年底，在天津市政府的大力推动下，负责产业基金审批的国家发改委于06年12月15日正式批准渤海产业基金及其管理公司的设立。首期基金规模60.8亿元，基金的管理公司渤海产业投资基金管理有限公司（下称渤海基金管理公司）也正式成立。公司注册资本金为2亿元人民币，股东包括中银国际控股有限公司（下称中银国际）、天津泰达投资控股有限公司和渤海产业基金的各出资人。而渤海产业基金方面介绍首期基金的出资人情况是：“渤海基金管理公司出资0.8亿元，全国社会保障基金理事会、国家开发银行、国家邮政局邮政储汇局、中银集团投资有限公司（下称中银投资）及天津市津能投资公司五家公司各出资10亿元”。

但真相却并非如此。很少有人将注意力停留在中银投资身上，事实上，其为中国银行股份有限公司（下称中国银行）在香港注册成立的全资附属机构，尽管最终出资人是国有性质的中国银行，但由于注册地在香港，外资身份非常明确。而蹊跷之处正是在此。由于中银投资的特殊身份也使得渤海产业基金的身份性质也发生了巨大的变化。

从法律架构上，渤海产业基金因包含境外发起人而属于外资身份明确无疑。但这些，一直都被渤海产业基金作为秘密而未向外界披露，即便是被其投资的企业也并不知晓。**了解内情的人士称，渤海产业基金在投资时从不会向对方透露公司外资的身份。**在一份批号为“商资批（2006）2483号”商务部批复文件中，商务部对渤海产业基金管理公司的设立进行了批复，而**这份文件不仅再次确认了渤海产业基金管理公司的外资身份，**还明确规定了渤海产业基金相关投资行为应按外商投资的相关规定报批。这意味着，渤海产业基金实际上是与凯雷投资集团、黑石集团(Black Stone)、KKR等众多国际私募股权基金一样，都属于外资性质，在投资时将受到产业政策的限制，同时还需进行相关的行政审批。

由于外资基金在进行投资时面临产业限制政策，渤海产业基金一直以基金最终出资人和股东都是国有机构，资金来源本身属于国内为依据，希望能够获得政策绿灯。但这未能得到商务部的认可。知情人士称，商务部并不同意的原因是，国内有许多大型企业都在香港设立了窗口公司，再进行返程投资，它们享受了外商投资的优惠政策，但如果当享受优惠的时认定为外资，当为了规避限制的时就认定为内资，这逻辑并不可行。

尽管受到发改委、商务部、外汇管理局、天津市政府等政府监管机构的密切关注和监督，发改委在当时的批复文件中也明确要求了该基金“须主动接受主管机关的监督检查，定期向我委报送年中和年末经营报告，随时报告公司运营中的重大事项”。商务部也明确要求投资项目报批。外管局照理也应该根据商务部审批手续再批准办理结汇事宜，但是渤海产业基金仍然可以置所有国家监管部门和法律法规于不顾，投资项目完全不受外资的限制政策，享受外资企业的优惠政策，不交税，而且可以将投资收益往境外合法汇出。这是怎么做到的呢？因为这家所有的渤海系就是和海航一样，是王歧山的家族企业，分别担任渤海产业基金董事长和总经理的李礼辉和李祥生都是王歧山的马仔。
