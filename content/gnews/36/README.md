## 英国自2021年9月起禁止安装新的华为5G设备

![]()![](https://spark.adobe.com/page/whlbWmOCmuUax/images/23db594c-aabc-43da-9045-382149c2dfd9.jpg?asset_id=a232ec4f-974d-47f9-aa45-570f4e56c866&amp;img_etag=%22aa7252e2c851707f10effce290955fc6%22&amp;size=1024)资料图片：在该插图中，在PC主板上看到英国国旗和带有华为和5G网络标识的智能手机
资料图片：2020年1月29日拍摄该插图照片，在PC主板上看到英国国旗和带有华为和5G网络标识的智能手机。路透社（REUTERS）/达多·鲁维克（Dado Ruvic）/插图/资料图片

伦敦路透社（Reuters）- 英国政府周一表示，英国电信公司在2021年9月之后不得安装新的华为5G设备，这是从高速移动网络清除中共国公司设备计划的一部分。

英国已经下令在2027年底之前将所有华为设备从其5G网络中移除，与包括美国在内的情报盟友保持一致，美国称该公司存在安全风险。

电信公司也曾被禁止在年底后购买新的华为5G设备。一些立法者曾担心，这些公司将会贮存设备，以在2027年的最后期限之前进行部署。

然而，运营商表示，他们将会在明年推出替代设备，使得2021年的截止变得“可控”。

[!\[\]()!\[\](https://spark.adobe.com/page/whlbWmOCmuUax/images/0507a24d-638c-450d-9615-0258452f6b4b.jpg?asset_id=ff5a19f8-fdbe-4f9c-a74b-8f69f3613760&amp;img_etag=%22d9558061142ad3c18927c02af19f64c2%22&amp;size=2560)](https://spark.adobe.com/page/whlbWmOCmuUax/images/0507a24d-638c-450d-9615-0258452f6b4b.jpg?asset_id=ff5a19f8-fdbe-4f9c-a74b-8f69f3613760&amp;img_etag=%22d9558061142ad3c18927c02af19f64c2%22&amp;size=1024)

中共国对此项举措进行了批评，而华为在上周表示，在新法律公布后，英国希望将其排除5G推广之外，如果这些公司违反禁令，可能会被罚款10万英镑（约133,140美元）,这一举措令华为感到失望。

周一的公告是在议会就新的电信立法进行讨论之前宣布的，并具体化了设备移除的时间表。

数字化大臣奥利弗·道登（Oliver Dowden）在一份声明中表示，“我正在制定一条明确的途径，将高风险供应商从我们的5G网络中完全移除”。

“这将通过新的和前所未有的权力来实现，以确认并禁止对我们的国家安全造成威胁的电信设备。”

政府还宣布了一项实现5G供应链多元化的新策略，包括一笔2.5亿英镑的初始投资，与日本公司NEC合作进行试验，以及建立新的研究设施。

英国移动运营商BT、沃达丰（Vodafone）和3（Three）都推出了5G，在其无线接入网络的部分使用华为。

他们已经转向其他供应商，比如爱立信（Ericsson）和诺基亚（Nokia）。

政府还对已经宣布的华为设备在光纤到户和其他千兆网络的35%上限，规定了2023年1月28日的最后期限。

英国政府表示，其7月份的决定与担心美国对芯片技术的制裁可能影响供应链有关。当时华为表示，该决定令人失望，并且是关于美国的贸易政策，而不是安全问题。
