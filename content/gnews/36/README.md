福奇想“跳船”为自己脱罪？5月11日，福奇在美国参议院听证会上接受参议员兰德·保罗质询时表示，他对武汉病毒研究所所做的事情一无所知，并否认NIH曾资助武毒所进行“功能增强”实验，但他表示支持彻查病毒来源。

显然，福奇是在为自己开脱，把自己装扮成一无所知。那么他的话能被人相信吗？不能。不过从他的话里倒是可以听出，他也承认了武汉病毒研究所进行“功能增强”实验这一事实，而这一点至关重要。

我们先来看一下质询时的部分对话内容：

保罗：“福奇医生，你是否仍支持用美国卫生研究院(NIH)资金去资助武汉实验室？”

福奇：“你这样说是完全不正确的。NHI从来就没有——现在也没有资助武汉病毒研究所的病毒功能增强研究。”

保罗：“难道你从不认为武毒所的蝙蝠病毒刺突蛋白插入SARS病毒的过程就是功能增强吗？”

福奇：“不是的，目前持这种观点的人不在少数，因为至少已经有200名剑桥工作组的科学家签署了一份声明，认为这就是功能增强。（我们资助的项目）不是功能增强，如果你去查看相关的拨款，看看进度报告，便知那并不是功能增强。”

保罗：“那么你是否仍然支持对武汉病毒研究所投入资金？”

福奇：“我们现在没有向武汉病毒研究所投入资金。”

保罗：“那么你是否曾支持过向其投入资金呢？我们过去在你的监督下，就是这样做的。”

福奇：“我们确实通过EcoHealth给了武毒所钱，EcoHealth是二级代理，二级基金。”

保罗：“你是否曾支持NIH向武汉病毒研究所提供资金？”

福奇：“让我向你解释一下为什么这样做。因为SARS-CoV-1源于中国的蝙蝠，那么如果我们不去调查蝙蝠病毒和相关的血清学，不去看看在中国谁可能被感染过，那对我们来说是不负责任的行为。”

保罗：“送钱给无法让人信任的中（共）国政府，让他们拥有相关知识，让他们研究这种极度危险的病毒恰恰是不负责任的行为。像你这样的支持功能增强研究的政府科学家…”

福奇：（打断保罗的话）“我不支持功能增强研究，你说得不对。”

保罗：“像你这样的为功能增强研究辩护的政府工作人员认为COVID-19（中共病毒）的突变是随机的，不是人为设计的。你是否可以在这个小组面前明确地阐明COVID-19（中共病毒）不可能源于实验室中的系列病原体？”

福奇：“我对中国人（中共）所做的事情一无所知，我完全赞成对在中（共）国发生的事情进一步调查。但是，我再次重复，NIH绝对没有资助在武汉病毒研究所进行的功能增强研究。”
