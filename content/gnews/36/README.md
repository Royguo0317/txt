## 美国共和党州长表示结束疫情强制性规定的时刻已经到来
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1150911/)

- 作者：坐看云起时
- 制图：透明的遮羞布

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/04/29134505/virus11-1.jpg)
据TheWestJournal 4月28日报道，美国的两位共和党州长–得克萨斯州州长格雷格-阿伯特和阿拉巴马州州长凯-艾维本周告诉福克斯新闻，明确表示，他们希望他们的州能够迈过中共病毒强制规定的时代，认为已不再需要各种与大流行病有关的各种限制。

阿拉巴马州州长凯-艾维认为，进入大流行病已有一年多的时间，世界必须要向前看，无休止的政府授权并不是解决问题的答案。

德州州长阿伯特指出，严格的防疫准则对德克萨斯州的企业和社区产生了影响，同时也承认许多人仍然对这种大流行病感到担忧。但根据德州目前的疫情状况，他认为德州已经可以准备恢复正常生活，允许德克萨斯人和企业做出自己的决定，释放出该州经济的全部力量。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/04/29134530/virus12.jpg)
阿拉巴马州州长凯-艾维也发出通告，全州的强制口罩令将于周五到期，新的健康令将生效。

与此同时，美国疾病控制和预防中心周二刚刚宣布，只有完全接种疫苗的美国人才可以不戴口罩外出，美国总统乔-拜登则将7月4日定义为他的目标，指出希望到那天人们能够与朋友与家人们一起庆祝独立日。并指责取消强制口罩令的各州是 “尼安德特人的思维”，显示了国家和一些州级层面针对疫情处置方针战略之间的明显对立。

由此可见中共病毒大流行在如何看待政府的控制和指导方面进一步分裂了美国人和他们的领导人，而这些分歧将在未来几个月变得更加清晰。

笔者评：

**一年多的病毒大流行大大地激化了世界原有的各种矛盾，如何在控制疫情的同时保持经济则更是全世界同时面对的两难抉择。**

**由于世界对于病毒真相的无知或不愿及不敢直面，大部分人都将恢复正常生活的所有希望寄托在疫苗之上。当前印度正在经历的病毒大流行向世人敲响了警钟，希望世界能早日醒来，真正地意识到只有解决中共才是解决病毒大流行的关键之所在。**
