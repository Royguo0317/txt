# 乔什·霍利: 有谷歌、脸书和推特协同审查的证据

[![](https://spark.adobe.com/page/oTREpMr8jKUdv/images/73a73f63-25a3-4540-bcf2-c77fd087dea2.jpg?asset_id=b20583ad-7273-40d3-89e6-07d77581a3f6&amp;img_etag=%22dd11f3f4953ea7ff0360150caaf78ff6%22&amp;size=2560)](https://spark.adobe.com/page/oTREpMr8jKUdv/images/73a73f63-25a3-4540-bcf2-c77fd087dea2.jpg?asset_id=b20583ad-7273-40d3-89e6-07d77581a3f6&amp;img_etag=%22dd11f3f4953ea7ff0360150caaf78ff6%22&amp;size=1024)Drew Angerer/Getty Images

**参议员乔什-霍利(Josh Hawley)表示，脸书的一名爆料人已经与他进行了接触，指控脸书、谷歌和推特进行协调审查，以压制和审查它们的平台。**

霍利（Hawley）通过其推特账户宣布，爆料人已经向他提供了涉嫌协作的证据，他将在参议院司法委员会的听证会上就此事向推特的杰克-多西（Jack Dorsey）和脸书的马克-扎克伯格（Mark Zuckerberg）提问，两位CEO将在听证会上作证。

密苏里州参议员霍利（Hawley）是大科技公司的激烈批评者，他说，爆料人还指认脸书有一个内部平台来管理协调审查(客户言论)。

[![](https://spark.adobe.com/page/oTREpMr8jKUdv/images/9c9b5c65-e080-4f00-9378-34f01c919741.jpg?asset_id=fb891084-d5bd-4eb5-b37a-393808ce2a8d&amp;img_etag=%225952c668dba1f01b2ba1ad8d63b1a187%22&amp;size=2560)](https://spark.adobe.com/page/oTREpMr8jKUdv/images/9c9b5c65-e080-4f00-9378-34f01c919741.jpg?asset_id=fb891084-d5bd-4eb5-b37a-393808ce2a8d&amp;img_etag=%225952c668dba1f01b2ba1ad8d63b1a187%22&amp;size=1024)

“我已经从脸书的爆料人那里听说，他透露脸书和谷歌和推特协调审查(客户言论)，”霍利说。”脸书有一个内部平台来管理它。我将在明天的听证会上向马克-扎克伯格和杰克-多西质询此事。”

正如布赖特巴特新闻的卢卡斯-诺兰（Lucas Nolan）今天早些时候报道的那样，大科技公司对川普总统(在社交媒体言论)的审查继续升级：

布赖特巴特新闻广泛报道了社交网站在2020年总统大选前加大审查力度的情况。推特上周表示，从10月27日到11月11日，它给30万个违反选举错误信息相关规则的用户帖子贴上标签。推特还修改了转发功能，要求用户在转发帖子前加入自己的评论。

从大选日到上周五，推特将川普总统约34%的推文和转发标记为 “有争议”。布赖特巴特新闻对此进行了大量报道，最近注意到，川普总统在推特和脸书上的25个帖子在24小时内被贴上标签或争议。

参议院司法委员会的听证会将于东部时间周二上午10点举行。

[![](https://spark.adobe.com/page/oTREpMr8jKUdv/images/fe735944-d17d-44c3-9a83-c5ca849725bb.jpg?asset_id=8f78daa7-0c9a-42c6-a8a9-bd87bd455e5d&amp;img_etag=%22c5c13ccd308f615de29f96f37f29332e%22&amp;size=1024)](https://spark.adobe.com/page/oTREpMr8jKUdv/images/fe735944-d17d-44c3-9a83-c5ca849725bb.jpg?asset_id=8f78daa7-0c9a-42c6-a8a9-bd87bd455e5d&amp;img_etag=%22c5c13ccd308f615de29f96f37f29332e%22&amp;size=1024)

此次听证会的主题聚焦于2020年大选日之前的数周和数月内发生的对新闻和政治言论的广泛审查。听证会的标题是 “爆料新闻:审查，压制，和2020年的选举。”

阿鲁姆·博卡里(Allum Bokhari)是布赖特巴特(Breitbart)新闻的高级技术记者。他的新书《#删除: 大科技公司的战争——消除川普运动和偷窃大选》，书中收录了对谷歌、脸书和其他科技公司内部消息人士的独家采访，目前已上市。
