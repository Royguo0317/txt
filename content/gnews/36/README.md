众议院共和党的《中共国特别工作组》（China Task Force）【1】发布了众议院共和党委员会领导给众议院议长等联名公开信，敦促两党合作，加快灭共立法，采取行动，应对中共对美国和世界带来的威胁。【2】

文章题目，“麦考尔（McCaul）与众议院共和党委员会领导人一道呼吁，要严肃审议任何的与中共国竞争的立法”。

全文如下：

众议院外交事务委员会主席、共和党人迈克尔·麦考尔（Michael McCaul）和另外几位共和党众议院委员会领导加入了科学、太空、技术委员会高级会员弗兰克·卢卡斯（Frank Lucas）主导的、写给众议院议长南希·佩洛西（Nancy Pelosi）以及多数党领袖斯坦尼·霍耶（Steny Hoyer）的联名信，敦促他们优先通过关于美国竞争力的立法。成员们强调，需要采取两党两院的共同行动，以应对中国共产党对美国技术竞争力、经济增长和国家安全日益增长的威胁。

这些成员写道：“解决中国共产党（CCP）所构成的世代威胁，不是共和党的问题，也不是民主党的问题，而是美国的问题。”在国会审议与中共国战略竞争的行动时，我们写信敦促你们在正常程序下，通过众议院的任何此类立法，我们不要浪费这个机会，通过协调来制定强有力和深思熟虑的政策。”

尊敬的佩洛西议长和多数党领袖霍耶，

在国会考虑与中共国进行战略竞争的行动时，我们写信敦促你们在正常程序下通过众议院的任何此类立法，我们不要浪费这个机会，通过协调来制定强有力和深思熟虑的政策。作为参与这项工作的委员会的高级成员，我们认为，任何此类立法都必须在两党、两院、公开和透明的进程中制定。

解决中国共产党（CCP）带来的世代威胁不是共和党的问题，也不是民主党的问题，而是美国的问题。我们随时准备与你们和我们的委员会主席合作，让中共对自己的行动负责，同时也加强我们自己在创新方面的全球领导地位，包括投资于美国的研发、支撑国内供应链、改善与战略盟友的协调以及解决其他安全问题。

我们高兴地看到，参议院正在努力应对中共对美国技术竞争力、经济增长和国家安全之间的关键问题的威胁，虽然我们可能无法就参议院目前审议的所有政策达成一致，我们赞赏人们普遍认识到了中国共产党领导的、使我们面临的危险，并愿意为此采取行动。

我们现在有机会让我们的委员会共同协商两党立法，通过众议院，并与参议院讨论。

正如你们所知，去年，众议院共和党会议成立了我们自己的《中共国特别工作组》（CTF），以解决中共在全球和军事上的主导地位。在六个月的时间里，反恐委员会与政府官员、私营部门领导人以及两党的外部专家举行了60多场简报会，以制定长期的、两党合作的解决方案，最好地应对中共的有害战略。我们对民主党选择不参加特别工作组感到遗憾，但我们认为，我们提出了强有力的两党解决方案，反映了我们各自委员会的意见。我们去年9月发布的这份报告包括了400多条关于美国如何更好地应对中共威胁的政策建议，其中三分之二是两党合作的。我们相信，这些两党的建议中，有许多可以帮助制定一套应对中共战略竞争的立法方案。

现在是采取行动的时候了，我们的国家和经济安全取决于未来数月和数年的情况，我们敦促你们不要利用预算协调或其他立法伎俩，使这成为一个党派程序。相反，众议院应该利用相关委员会的专业知识，确保我们的应对措施能够应对来自中共的各种威胁，来自两党的成员都有良好和成熟的想法，应该被邀请到谈判桌上来，这一问题值得深思熟虑的讨论和辩论，（明确）授权立法的规定与通过正常程序拨款（的关系）。

最后，除了加大投入美国的创新，以保持美国在经济和技术上的竞争力外，我们还需要讨论监管和许可改革、知识产权保护、国家安全和国防等开支。

我们的许多委员会已经在为解决与中共国的战略竞争而制定两党法案而努力，我们希望我们能够在公开透明的过程中尽快推进任何立法。

我们期待着在接下来的几周和几个月里与众议院领导层和委员会领导人就这些重要问题展开合作。

（全文完）

答案有了，两党运作的、在外交委员会高票通过的、针对中共的《2021战略竞争法案》1个多月不见动静，卡在了巫婆佩洛西那里！民主党人还不参加众议院共和党会议成立的《中共国特别工作组》，难见民主党人对中共病毒溯源追责的呼吁；在中共夸大报道美国一系列的弊端中，几乎没有看到对美国大选欺诈舞弊的报道。一切迹象表明，在民主党高层、拜登政府有很大的与中共勾兑的力量，或者说，难以摆脱中共影响的力量，导致了美国灭共行动的缓慢。也许，这是他们对邪共通过舞弊欺诈的大选，助民主党人篡位的回报。希望这是笔者开脑洞式的猜想！
