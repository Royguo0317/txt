据[王毅同索马里外长阿瓦德通电话](http://world.people.com.cn/n1/2020/1112/c1002-31927701.html)（2020年11月12日02:18  来源：[新华网](http://www.xinhuanet.com/2020-11/12/c_1126728216.htm)）：

［王毅表示，中国是最早承认索马里独立的国家之一，索马里是第一个同新中国建交的东部非洲国家。长期以来，中索相互尊重、平等相待，**在涉及彼此核心利益问题上相互理解、相互支持。索马里是中非合作论坛、中阿合作论坛和共建“一带一路”的重要一员，是中方值得信赖的好朋友、好兄弟。中方赞赏索方恪守一个中国原则，坚定支持索马里维护国家主权、尊严和领土完整的努力。**中方愿以纪念中索建交60周年为契机，同索方落实好两国领导人共识，推动中索关系迈上新台阶。

王毅表示，新冠疫情暴发以来，中方在自身防疫压力还很大的情况下，克服困难，向索方提供了紧急抗疫物资，同索马里人民共渡难关。中方愿同索方一道，优先实施好索和平重建急需的物资援助和人员培训等项目，探讨在农业、渔业、卫生等领域开展更多合作。中方愿继续鼓励中国企业赴索投资兴业，助力索马里经济社会重建。］

据[王毅同布隆迪外长欣吉罗通电话](http://world.people.com.cn/n1/2020/1112/c1002-31927702.html)（2020年11月12日02:18  来源：[新华网](http://www.xinhuanet.com/2020-11/12/c_1126728217.htm)）：［王毅表示，中布关系保持高水平发展，双方构建了牢固的政治互信，各领域合作成果丰富。布隆迪成为中国的“全天候”朋友、“全方位”伙伴。**中方将一如既往支持布方维护国家主权、反对外来干涉。相信布方会继续在涉及中国核心利益问题上同中方站在一起。**中布要在国际和地区事务中密切沟通协调，坚定捍卫多边主义，维护两国和发展中国家的共同权益。

王毅说，中布务实合作经受住疫情考验稳步推进。**中方愿同布方加强沟通协调，将共建“一带一路”、中非合作论坛北京峰会成果落实同布方发展农牧业、支持青年事业、实施良政等经济社会发展“六大任务”紧密对接，不断深化两国互利合作。**中方坚定支持布方抗疫努力，愿继续向布方提供力所能及的支持和帮助，助力布方早日彻底战胜疫情。］

什么是中国的核心利益？据[百度百科——中国核心利益](https://baike.baidu.com/item/%E4%B8%AD%E5%9B%BD%E6%A0%B8%E5%BF%83%E5%88%A9%E7%9B%8A/13236425?fr=aladdin)：［“中国核心利益”包括：国家主权，国家安全，领土完整，国家统一，中国宪法确立的国家政治制度和社会大局稳定，经济社会可持续发展的基本保障。 [1]］
