从七哥与约翰叔叔的对话中，我们知道共产党对我们思想洗脑导致了我们的“电路被切断了”。七哥的意思是我们从小的思维进度和逻辑导图被活生生阉割了，我们的思想之树上长不出芽，结不了果，或者果实都被虫蛀了。而一个没有思维能力和建立在信仰基石上的逻辑判断的人，能干出令人无法想像的事情。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/05230803/2221.jpg)图片来源：环球实报
我们从娘胎里出来，没有任何缘由地相信自已的家庭和亲族，中国人对家庭的依恋是建立自己人格的基础，但没想到我们的父母和家庭也是被洗脑对象，思想中毒之深不知不觉中每个人成了共产党手中的枪，战友们的文章都阐述地很直中要害，我想谈谈自已的体会。

1. 问题背后的心理定势

西方人发问是基于这东西是什么，而我们文化中倾向于这东西是为了什么，一个是本质，另一个是用途，

世界上好的东西不一定是为了有用，而创造出好东西的人未必知道所有的本质和预后，像霍金的黑洞理论能听懂的人不多，他也未必知道黑洞所有奥秘，他这个理论给人类提供了可能性和参考。一个对本质都摸不着的人就先下判断，先把用途限定死，也就挡住了对本质的理解。

2. 不鼓励激烈地讨论

我们刚到西方社会很不适应媒体和公众教育领域中人与人激烈但不失尊严，不带人身攻击地发问和智力对抗，而这是锻炼一个孩子明辨是非最有效的方式。我记得有一个犹太富人家族有六，七个兄弟，他们有家族企业，在家中孩子面前，饭桌上，周末休闲中不停地讨论，争吵生意，每个人面红耳赤，每个人以更高声音分贝来阐述自己想法。说服不了别人很正常，但每个人都说出自己观点，没有自己观点一直保持沉默的人被视为想隐藏什么的人，也就是不坦率的人。这种家庭的训练让每个人在社会上有极大优势。

3. 推敲和细节检查

很多被P2P骗的人在生活中都是精明、有一定经济基础、老于世故的人，这些人也是小有成就的人，那为什么共产党骗完草根和蚁民，还能骗得了他们呢？第一是盲目信任，轻信家人，轻信有钱有权的朋友，轻信三人画虎，脑子被人说三遍不干某事就是LOSER，盲目从众成了一群乌合之众。第二是不检查每个细节，实际上骗子的每一个细节都经不起推敲，每一个漏洞的回答都可以反转，当你问自己二十多个问题仍没有把握，当你问骗子二到五个问题对方就勃然大怒说你是骗子时，信心弱的人就上当了。第三其实细节检查的难度没有想象中那么高的操作性，就像很多婚姻欺骗，女方对怀疑的男方每句话都不信，但付出了肉体、感情、金钱、岁月的代价后，仍活在每天男方会出轨的提心吊胆中，其实可以雇用一个合法侦探去调查清楚真相，这个费用很小，最少得到真相就是胜利。

4. 注意我们交流的用词和态度

处于现在这种墙内的至暗时刻，我们如何向家人和朋友传播真相和避险是需要一定能力的。不是你有好心就足够，是要真正感同身受。我在出来之前尽量做到不说假话，少说违心的话，我已经有了一定觉醒但仍处于被严重洗脑中，很多狂妄无知仍让我惭愧，但我知道自己身处黑暗，心向光明。亲人从海外归来，我们觉得兴奋，对文明国家无限憧憬，可亲人一开口就是“我们每天为你们祈祷，这里空气有煤烟子味”，正常人听后的感觉是我们活在地狱里，吸着毒空气，已经救无可赎。这种话除了让人更绝望没有一点用处，因为你高高在上，帮助人落不到细节上。半年前我把物资清单发给亲朋好友，一家子不可能都信你的话，我就挑跟自己同龄或叔父辈的人，跟他们细细地讲，他们的认知不可能一下子达到战友们这个级别，我就说大疫后有大饥饿，很多国家不出口粮食这种事实和历史现象，他们一下子就懂了。大部分我们的家人朋友是善良的人，也懂真相，耐心真实平等都可以让他们识破谎言。
