以下是简要翻译拉希姆·卡萨姆2020年11月12日在国家脉动新闻的节目“阻止盗票”（Stopping the Steal）：

1. 有一位法官裁决说不要计算在宾州区分出来的选票。说宾州州务卿违反了选举的法律常规。

2. 川普团队需要有个系统，每天由一位领头人向公众宣布诉讼进程。

3. 周六成千上万的川普支持者要在华盛顿游行，据称是“百万人美国伟大游行”。

4. 采访欧盟议员Jerome Riviere。世界大媒体都不报道美国大选有舞弊。法国媒体在掩盖，只是讲拜登“赢了”。

Jerome说欧盟议会的安全委员会不允许讨论美国大选有舞弊，欧盟议会说提到大选舞弊的新闻都是假新闻，压制言论自由。

拉希姆说法国的玛丽安·勒庞是西方唯一的主流政治家不承认拜登当选，因为过程还没结束。

Jerome说欧盟标榜自己依法治国，但并不是这样的。欧盟支持拜登，因为他的政策会接近奥巴马。拜登和欧盟都支持因疫情封城，这会破坏经济。大选显示川普有7300万支持者，他的动力很强。

拉希姆问法国有没有邮寄选票。

Jerome说因为存在大量舞弊法国在1975年废除了邮寄选票。然而，亲近马克龙的法国参议员建议这回要使用邮寄选票。全球主义者不相信老百姓。在美国买酒都要出示身份卡，保证购买者年龄在法定21岁以上。那凭什么邮寄选票不需要出示身份卡。

5. 娜塔莉·温特斯说Dominion公司（生产验票机）充满民主党高层人员，包括佩罗西的前幕僚长以及奥巴马选举的人员。Dominion的员工的政治捐款中，96%都是给民主党捐款。

6. 嘉宾Molly McCann说宾州问题比较大，拜登目前领先5万票，但有争议的选票有60万张。最高法院可能会裁决，也可能送回宾州州议会，关注州议会。

密西根州有私人非盈利组织诉讼要求选票审计。

川普团队也在密西根州诉讼要求禁制令，禁止证明选票，因为有些选票可能有造假。

7. 福克斯收视率下降。
