**【熟鸡蛋如何孵化的？】**

我们有听说凤凰涅槃的故事，鸡蛋返⽣，孵化⼩鸡，乃中共治下独有的惊世创作。

⽂章称：“鸡蛋返⽣，顾名思义，由熟鸡蛋变成⽣鸡    蛋。这是⼀个难以想象的，甚⾄是不可能的事。但这样奇特的现象，确实在郑州春霖职业培训学校发⽣了。⼀群特别培训的学⽣，在郭萍⽼师指导下，正在进⾏⼀个奇特实验，即熟鸡蛋重新变成⽣鸡蛋，并将返⽣后的⽣鸡蛋孵化成雏鸡，并且已经成功返⽣了40多枚。”

⽂章介绍，实验的过程分为五步：取蛋、煮蛋、装蛋、返⽣、孵化。其中返⽣步骤为：“学⽣开始利⽤超⼼理意识能量⽅法，开始鸡蛋返⽣，20分钟后鸡蛋返⽣成功。 由观察员再次进⾏透光试验，鸡蛋透光，卵⽩、卵⻩分明可⻅，并当场破壳⼀枚，证实鸡蛋返⽣呈液态，返⽣成 功。”

在后续《“熟鸡蛋鸡蛋返⽣孵化雏鸡”实验报告(孵化阶段)》⽂中报告，熟鸡蛋返⽣已经实验成功。返⽣后的鸡蛋是否具有⽣命⼒，在郭萍⽼师、郭店鸡场郭太安⽼师及其他爱好者，密切关注孵化进程。最终成功孵化出新的雏鸡，确认熟鸡蛋返⽣后的鸡蛋同样具有⽣命⼒。

报告说⽤的是五步返⽣孵化，⽤时20分钟，利⽤超⼼理意识能量⽅法。⾄此才明⽩，不是要把熟鸡蛋返⽣，⽬的是熟鸡蛋孵化⼩鸡是超⼼理意识能量法的⼀种验证。如果不是⼤家关注到这篇⽂章，估计下⼀篇应该是⽤超⼼理意识能量法直接能让⼈起死回⽣了。
