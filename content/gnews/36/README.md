## 驳斥中共驻澳大使的无耻谎言 究竟谁在经济胁迫与挑衅？

中共驻澳大使成竞业在4月29日对澳洲工商界人士讲话中指责澳洲对中国进行经济胁迫和“挑衅”。
其实我们可以盘点一下中澳之间贸易方面的摩擦。2020年5月18日中共宣布对澳洲的大麦征收80%的惩罚性关税。然后又对澳洲的4家肉类出口商取消出口到中国的资格。
在此之后，中共又是动作不断，从葡萄酒到木材，再到干草，一路走来，到底谁在进行经济胁迫和“挑衅”？大家应该一目了然。
有人说，中共的这一系列动作是因为澳洲政府挑衅中国在前。他们所举的例子是澳洲禁止华为的5G以及要求追查新冠病毒的来源。
但正是从这两件事上，我们更可以看清中共无赖和心虚的本质。
澳洲禁止华为5G通讯设备在澳洲的使用，这是事实。但这是澳洲合乎WTO贸易原则的正当行为，因为他们已经觉察中共正在全方位向澳洲渗透，进行情报收集，而华为的5G设备正是中共的后门之一。
在这一点上，禁止华为设备已经无关贸易，而已经是关乎国家安全的事情。
在WTO国际贸易规则体系中，有一些“国家安全例外”（National Security Exceptions）规则，指允许当事方以维护国家安全为由终止某种特定贸易。具体包括GATT（《关税与贸易总协定》）第21条，GATS（《服务贸易总协定》）第14条第二款和TRIPS（《与贸易有关的知识产权协定》）第73条。
GATT第21条规定：“本协定的任何规定不得解释为：（a）要求任何缔约方提供其认为如披露则会违背其基本安全利益（Essential Security Interests）的任何信息；或（b）阻止任何缔约方采取其认为对保护其基本国家安全利益所必需的任何行动。”
WTO《技术性贸易壁垒协定》（The Agreement on Technical Barriers to Trade，简称TBT）也包含“国家安全例外”条款。其第2条第2项规定：“成员方应保证技术法规的制定、采用或实施在目的或效果上均不对国际贸易造成不必要的障碍。为此目的，技术法规对贸易的限制不得超过为实现合法目标所必须的限度，同时考虑合法目标未能实现可能造成的风险。此类合法目标特别包括：国家安全需要；防止欺诈行为；保护人类健康或安全、保护动植物生命或健康及保护环境。”该条款规定了贸易限制的程度，强调不应实施超过实现合法目标所必需的贸易限制，认定以国家安全为由实施特定程度的贸易限制是合法的。
WTO《政府采购协议》（2012年版）第三条规定了有关国家安全例外：“本协议不得解释为阻止任何参加方，在涉及武器、弹药或战争物资采购，或者涉及为国家安全或国防目的所须的采购方面，在其认为保护根本安全利益的必要情形下，采取任何行动或者不披露任何信息。”同时，协议规定了安全例外适用的情形，包括为保护公共道德、秩序或安全所必须的措施、为保护人类、动植物生命或健康所必须的措施以及为保护知识产权所必须的措施。
对于这一些规则，澳洲可以使用，中共也可以使用，作为这些保护国家安全的举措，绝不能视之为贸易挑衅。
而澳洲提出追查新冠病毒的来源，作为世界上也是遭受新冠病毒侵扰的国家，提出追查病毒来源和真相，这是正当的权利，毕竟全世界付出了这么多人命和经济的代价。不但是澳洲，世界上任何一个国家都有这样的权利。以这一点来打击澳洲，只能体现中共的流氓和心虚。
中共为了打击澳大利亚，故意置WTO条款和追查病毒真相的正义呼声不顾，而蓄意地把舆论歪曲和引向澳洲政府的经济胁迫和挑衅，其用心极为险恶。1、把中澳交恶的责任推向澳洲，2、误导不清楚事实真相的中国民众。
中共所使用的手段才真正称得上“经济胁迫”和“挑衅”。除了上述两个险恶用心以外，中共另有一个最为深层的盘算是：以澳洲挑衅的名义打击澳洲，使其服软，借此警告其它国家不要步其后尘。
中共本来以为通过连环经济打击和舆论胁迫，澳洲会感受压力，从而产生内讧，最终认怂。但中共万万没有想到澳洲“不吃这一套”，反而愈战愈勇，两党之间、朝野之间同仇敌忾，让中共不但计策失效，而且已经无计可施，它现在的贼喊捉贼正是这一现状的体现。同时，其它国家也看到中共色厉内荏，不过是外强中干的绣花枕头而已。
中共的脚被砸了一次又一次，别人砸的不算，最重要的在于其不断自作孽，天怒人怨，我们且看其能嚣张到何时！
