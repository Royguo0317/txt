从周日开始，英国将再次被视为病毒危险区域。印度突变体的出现令人担忧。联邦政府也在重新评估其他国家。

尽管感染人数很少，联邦政府仍将从周日起再次将英国归类为病毒危险区域。原因是罗伯特·科赫研究所（*Robert Koch Institute*）周五在互联网上宣布的，该病毒变种“至少在有限的情况下”在印度首次发现。 B.1.617.2变种在印度引起了巨大数量的感染。邻国斯里兰卡和尼泊尔也将在周日接受联邦政府的升级：斯里兰卡是一个危险地区，尼泊尔是一个病毒变异地区，这是最高的危险等级，对此有广泛的旅行限制。

但是，周日还将有9个国家和地区的病毒风险等级降级，从高发到危险地区：塞尔维亚、波斯尼亚和黑塞哥维那、科索沃、北马其顿、爱沙尼亚、乌克兰，约旦、黎巴嫩和巴勒斯坦自治领地。从那里进入的任何人都不必再被隔离，而可以在进入时显示阴性测试。

与加那利群岛一起，德国人另一个受欢迎的度假胜地也将被完全从病毒危险区域中删除。但是，这对游客不再有任何实际影响，因为随着新的入境法规的生效，假日游客从那里返回的隔离义务已在星期四减少。回德国的飞行测试义务继续适用。
