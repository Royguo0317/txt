## 公民自由高于一切——由比利时政府遭勒令解除疫情限制措施所思
` 喜马拉雅德国慕尼黑感恩农场` [轉載自GNews](https://gnews.org/zh-hans/1050539/)

作者：文渊

根据比利时晚报3月31日的报导：布鲁塞尔上诉法院判决比利时政府须在30天之内解除以抗击Covid-19疫情为由而实行的所有限制措施。人权联盟代表所有比利时公民，将比利时政府告上布鲁塞尔上诉法院。法院认为，比利时政府实行的疫情限制措施似乎缺乏充分的法律依据。若比利时政府逾期仍未解除限制措施，将被处以每日5000欧元的罚款。而比利时政府亦将对此判决提出上诉。

在中共病毒肆虐的欧洲，几乎所有国家皆有实行限制措施（限制多人聚集、宵禁、封禁等）以期控制疫情的扩散。作为受疫情侵害比较严重的国家之一，比利时实行限制措施在我们看来亦属于正常之举。而比利时人却认为自由与人权高于一切，政府无权以抗疫为由剥夺公民的权利。他们以实际行动抗争，对政府说不，最终通过法律途径拿回了自己的权益。

公民自由与抗击疫情，孰轻孰重？这是该案件所呈现的现实问题。也许我们认为：当然抗击疫情更为重要。比利时人不顾抗疫大局，不愿为此而牺牲自己暂时的权益，实在不可理喻。可是，这是我们华人的思维。在西方人的眼中，自由往往高于一切。“若为自由故，生命皆可抛”。我们可以不认同西方人的某些思想观念，但必须承认，不同地域之间确实存在着巨大的文化差异。面对不同的价值观，我们应当予以充分的尊重。

该案件揭示出的深层意义更值得我们思考。在西方民主国家，法律制度健全，规则井然有序，无论政府或人民皆是依法办事（比利时政府依照法规条文而颁布限制措施，民众亦有权利通过法律途径来推翻政府的决定）。这种平等理念、法治精神不正是我们所缺少的吗？不正是未来新中国所要追求的吗？郭先生多次说道：“我们千万不要笑话西方国家因民主体制而带来的一些乱象。因为我们从来没有过民主自由，我们没资格对人家评头论足”。如果未来的中国也能成为法治社会；如果未来的中国人也能通过法律来捍卫自己的权益，郭先生与战友们的中国梦也就实现了。 我们敬佩比利时人的勇敢与不屈的精神。不过，唯一让我们捏把汗的是：一旦限制措施在一个月后全面解除，面对如此严重的疫情，比利时的抗疫形势会恶化吗？毕竟，多数欧洲人仍不知晓病毒疫情真相，在日常生活中疏于防范，饱受疫情所害日久。祝福比利时民众，愿上天嘉奖他们的精神，愿病毒远离他们。
