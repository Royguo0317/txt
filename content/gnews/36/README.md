2020年*2*月*26*日，世卫组织公布《*WHO-*中国*2019*冠状病毒联合调查报告》，高度赞扬中共的抗疫成果

在中国进行9天调查之后，世界卫生组织于2月26日公布了《[WHO-中国2019冠状病毒联合调查报告](https://www.who.int/docs/default-source/coronaviruse/who-china-joint-mission-on-covid-19-final-report.pdf)》。

报告中提到，联合调查组访问了北京、成都、武汉、广州、深圳等城市的中心和社区医院、交通枢纽、地方疾病防控中心以及一个海鲜市场。

关于病毒的来源，报告中明确指出：“COVID-19是一个由动物感染人(zoonotic)的病毒”（第8页）。报告中提到，虽然具体中间宿主还没有查明，但是中共国在多方面的研究结果，让调查组得出这个结论。

报告中还高度赞扬了中共的抗疫行动：“面对未知病毒，中国推行了有史以来最重大、灵活、和彻底的疾病控制措施。

…

中国控制呼吸传染病疫情的重大举措，改变了这场正在迅速升级的疫情的发展进程。

…

中国坚持不懈采用非药物方式控制病毒传播，这种做法值得世界学习。”
