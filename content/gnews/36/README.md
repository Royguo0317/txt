3.继1月10日新增126例中共病毒感染者之后，1月11日石家庄市再新增21例感染病例，不过内部人士透露，实际感染数字远远高出这个数字。当日石家庄再有9地升为中风险地区。此外，邢台南宫疫情严重，正在紧急修建可以容纳五千人的方舱医院。

4.新任香港终审法院首席法官张举能周一（1月11日）称，“司法独立对于香港法治和行政公正具有根本的意义，对不论是地方或海外的公众和商界对我们司法制度的信心以及香港的国际声誉也同样重要。” 他指出，司法独立意味着法院“绝不能屈服于不当外部压力或影响”。

6.1月8日，中共国第三方支付公司国付宝信息科技有限公司发生工商变更，该公司第二大股东国富通信息技术发展有限公司正式退出，美银宝在国付宝信息科技有限公司的持股比例由70%增至100%，PayPal正式全资控股国付宝支付。这也是迄今为止首家外资全资控股的中共国第三方支付公司。Paypal和支付服务商Stripe刚刚高调切断川普总统竞选团队接受捐款的渠道，显然它们对中共的许诺更感兴趣。

7.新当选的爱荷华州共和党众议员阿什利·欣森周一（1月11日）证实，她已经要求佩洛西行使权力，阻止将中共报纸送交国会。欣森表示，“办公室收到不请自来的《中国日报》，让我感到震惊。” 《中国日报》是由中共控制的宣传工具。据报道，尽管早有一些议员提出抗议，但此刊物在美国国会已派发多年。

二、世界新闻

1.周一（1月11日）美国第一夫人梅拉尼娅·川普发表“我们的前进道路”的公开声明，她对1月6日的国会暴力事件感到“失望和沮丧”，并呼吁美国人“以文明方式治愈”。

2.白宫1月11日的新闻稿公告，川普总统宣布华盛顿特区从1月11日到1月24日进入紧急状态，时间跨越到拜登1月20日就职典礼之后四天。紧急状态授权国土安全部和联邦紧急事务管理局在国家首都协调救灾工作。这个消息是在川普总统和副总统彭斯会面后传出的消息。

3.1月11日，美国参谋长联席会议主席马克·米利（Mark Milley）将军和俄罗斯联邦武装部队总参谋长、陆军将军瓦列里·格拉西莫夫Valery Gerasimov）举行电话会议，双方讨论了共同关心的安全问题。

4.美国众议院周一（1月11日）再度启动对川普总统的弹劾程序，而大部分共和党议员并不支持佩洛西的主张。即使最终民主党挟众议院多数表决通过，这份决议也没有强制力。彭斯此前已经表态反对援引第25条修正案罢免总统。

5.尽管民主党和建制派媒体在上周三（1月7日）的国会入侵事件后开展了针对川普总统的非法化的运动，但国防部今日已表示，川普总统仍然是政府中唯一有全力直接下令采取军事行动的人，也是唯一拥有使用国家核武库酌处权的人。国防部将避免参与任何 “军事政变”。

6.美国驻联合国常任代表克拉夫特将于台湾时间1月13至15日访台。台湾总统府1月11日宣布，蔡英文总统预计于1月14日上午在总统府内接见克拉夫特。外交部长吴钊燮则表示，克拉夫特这次访台主题是与台湾讨论如何促进台湾在国际组织中的参与，议题非常重要。

7.美国驻荷兰大使霍克斯特拉星期一（1月11日）宣布在大使馆正式会见了台湾驻荷兰代表陈欣新，成为自国务卿蓬佩奥上星期六宣布解除美国与台湾关系中自我限制之后第一位在美国政府场所公开会见台湾官员的第一位美国官员。

8.位于爱达荷州北部的互联网提供商“Your T1 WIFI”证实，由于Facebook和推特对川普总统实施封杀后，大量使用该公司互联网服务的人投诉，表示不再愿意访问Facebook和推特，故该公司将封锁两家大科技公司向其客户提供服务。

9.拜登1月8日借图森枪击案10周年之际，承诺要击败全美步枪协会（NRA）及“终结枪械暴力”。 NRA周日（1月11日）回应指出，拜登企图废除宪法第2修正案赋予美国人民拥有枪械的权利，呼吁持枪人士保持警惕，并参与政治及立法程序。路德社此前已指出，拜登如成功当选将终结美国人民拥抢权，以便实现精英主义统治。

10.美国国务卿蓬佩奥在1月11日的一份声明中说，再次将古巴列入支持恐怖主义的国家名单，因为古巴一再为国际恐怖主义行为提供支持，为恐怖分子提供避风港。他说，“通过这一行动，我们将再次追究古巴政府的责任，并发出一个明确的信息，即卡斯特罗政权必须停止支持国际恐怖主义和颠覆美国司法。”

11.1月11日，印尼食品和药物监督管理局批准紧急使用中共科兴疫苗。这种疫苗正在印尼万隆进行第三期试验，有1600名志愿者参与了试验。根据印尼生物制药公司的负责人潘妮在雅加达所透露，疫苗的有效性只有65.3%。而根据路透社提供的消息，科兴生物在巴西三期试验取得的有效性低于60%。

12.据葡萄牙日报消息，葡萄牙总统马塞洛·雷贝洛·德索萨中共病毒检测呈阳性。
