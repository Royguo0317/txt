今天看了一部超级震撼的纪录片，The  End of The World as We Know 。内容可能很多战友们早已经看过了，但我是第一次看，真的震撼了。

里面讲到科学家们用含有汞的材料来制作食物，还有生活用品比如牙膏，还有药品。甚至水在被净化的过程中，都有可能被添加有害物质。也谈到迪士尼动画片里的一些暗示，也谈到强制坠胎其实是杀人，还有好莱坞名人用婴儿血来美容。不只是性侵，还有杀害。也谈到艾滋病，原来是有申请专利的，这意味着只有人造的东西才可以申请专利。那么，我们是不是可以有一个疑问，那埃博拉病毒和艾滋病毒其实是人造的病毒呢？还有我们使用的手机和平板电脑，其实都有间谍软件。那么，是谁，在监视我们呢？

再来看看全世界主流媒体。很多新闻，你以为是新闻，其实都是摆拍的类似肥皂电视剧。这部纪录片里还描述了加拿大的特鲁多总统有很多朋友，其实是有恋童癖的犯罪者。前总统奥巴马和希拉里，都有很多恋童的犯罪者朋友。他们能相谈甚欢真是奇迹，让人们联想这些人的价值观是否一致。疯狂的左媒努力把邪恶的理念正当化，来误导普通民众。甚至有的疯狂媒体，说儿童超过八岁的性行为已经很晚了。多个国家利用作弊，把有特殊嗜好的，外表好看的傀儡推上政治舞台，欺骗民众，让大家误以为是民众的正当选举结果。也谈到很多欧洲皇室其实是纳粹后代，其中英国皇室也有一些见不得人的勾当。很多道貌岸然的政治家、慈善家，其实背后都在做邪恶的事情。 纪录片的结尾说到了Q的标志。Q是代表这个世界的正义力量，像一颗种子，种下人性的光辉，将要成长为新的世界秩序。现在，我们爆料革命战友，新中国联邦人也是正义的力量，全世界人民无论种族都在参与这场正义与邪恶的较量，新旧世界正在交替。一个和平、自由、平等、和谐的新时代即将来临。每个人的发声、行动、立场都很重要。铲除邪恶势力，重建美好未来。
