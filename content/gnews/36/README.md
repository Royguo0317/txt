6月11-13日召开了G-7峰会**，**关注之后欧洲媒体对峰会的报道，可以使我们对当今世界局势有更深刻的体会。G-7七国集团会议的声明中，宣布了一些共同协议，宣告了一个新的时代的正式开始：

结束全球化，回归以国家为基本力量的新的经济体系，美国必须占据主导地位。

美国拜登政府以及处于领导地位的其他西方民主国家，在这次会议上定义和中共之间的关系为：战争。并明确表示，行动已经开始。

为了实现这个目标，美国面临挑战，将会采取一系列的行动：

1.抛弃自由主义思想，回归保守主义。即国家的利益重于个人，国家的力量将更多的介入到社会、经济、文化的各个方面。这包括在许多领域恢复保护主义，增加关税，市场单一化。（这和川普的想法很接近，区别是美国会和盟国一起干）。

2. 抛弃全球化，重新以国家为单位进入地缘政治竞争。比如成立亚约，注重意识形态的区别，明确敌对意识，对敌对国家进行包围和孤立，甚至对抗，有点像冷战。这将使中共很难再从政治和经济方面把力量投送到西方。

3. 驯服华尔街的金融力量，使之位于国家利益之下。

4. 收拢科技巨头，使之服从国家。7国会议宣布征收15%的全球性公司税，就是针对脸书、谷歌等跨国企业的，现状它们在美国之外的很多地区是不缴税的，每年税金超千亿美金。这只是第一步，后续将彻底把这些科技巨头收服。

5. 增加就业，提高工资，产业链回归，重新打造中产阶级。国民生活水平的上升是提高凝聚力的关键。一些东欧国家投向中共，也是因为本国的经济需要。美国将重新担负起这个责任。强调本地产业链的重要性，并提供更多的就业机会和改善收入条件，真正让利给工人阶级，重新打造稳定收入的中产阶级。并将以美国为龙头，带动盟友的发展。

6. 种族和解，推行新文化思想，忽视出身、身份，从文化、经济、社会各个方面增加社会凝聚力，从而提高美国的政治优势。这也是拜登解释其再三对美国之前的种族主义道歉的原因。

7. 改变欧洲的自由主义政治体系，回归保守主义。

8. 创建新的，不但包容而又各自独立的全球经济体系。

美国的转向和中共对全球化的依赖，将发生剧烈的冲突。美国面临巨大的挑战：全球化的既得利益者，被“蓝金黄”的力量，已经空心化的产业结构，疫情的影响，超发货币可能引发的通胀，通胀可能引发的强烈的经济危机，中共可能联合的其他力量的反扑，都将使美国、中共、全球都处于一场生死危机之中，没有退路，鹿死谁手？
