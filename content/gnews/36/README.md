***前按***

人民币汇率变化及未来走势，是爆料革命战友茶余饭后热衷讨论的话题之一。然而很多战友聊及人民币话题时，八分冲动二分理性，易宏观化、感性化、情绪化，并想当然地做出宏观判断，落入CCP做庄陷阱。感性化与情绪化，是一切庄家最大的朋友。本文仅从汇率管理和技术角度，剖析人民币汇率形成机制，以及CCP央行做庄技巧，分享一些汇率操控及背后的细节，协助感兴趣的战友，更好地理解人民币汇率波动，做出理性判断及决策。

***共产主义土鳖进城***

人性本善。当老布什政府用共赢心态及绥靖政策，让六四轻松翻篇；当克林顿政府用最善的揣摩，让共产党加入世贸，放生进化，彼时，中共党内决策层，还是清一色的土包子，从未见过什么是真正的货币，甚至从未理解什么是真正的商业。入世之后，伴随廉价低保劳动力逐步融入全球生产链条，中国大陆的贸易体量与制造能力皆扶摇直上，然而中共并未遵循经济规律，按承诺开放资本项目流通、放弃核心行业垄断、完善国民人权保护，且未露半点与全球商业环境接轨的意向，还利于民。中共拒绝将人民币汇率彻底市场化，却反其道而行之，利用汇率操控与全球偷窃，打造竞争优势，用非自然孕育的方式快捷制造垄断，对内持续压榨底层劳工，对外试图称霸全球所有主流行业，完全打破赐予它第二次生命的全球自由派预期。（在贸易丰盛的同时，允许本国汇率按货物与服务贸易供需而自然波动，同时鼓励全球生产要素、技术、工业、资本等的自由流通，让汇率波动反映它们的交换，彼此自然迁移，是西方自由派，即当年纯正无共产污染的西方左派，以入世放生中共的最原始动机。）

***土鳖变流氓***

2005年后迫于国际尤其美国压力（代表人：当年一身正气、如今一身猥琐的民主党参议员舒默），中国共产党被迫启动第一次大规模汇率改革，不再单一紧盯美元，对外宣称参考一揽子主流货币，实行有管理的浮动汇率制度。人民币汇率因此被迫启动升值，并于2008年金融危机之前，较汇改前升值接近20%。为应对08年全球金融危机，各主要经济体央行大规模印钱扩表，货币贬值，自顾不暇，有效缓解了中共进一步改革汇率的外部压力。然而，没了外部鞭子盯在身后一抽一动着催促，看似生存环境舒服的中共，却丧失了汇率制度深化改革、融入全球的最好时机。怠慢与延迟，代价惨重。

2008年之后，中共趁全世界钱荒及资产价格贬值，对外，变本加厉地偷窃/买断全球技术，延展军民融合战场至西方学术界，媒体及大企业寡头；对内，借08危机之名，理所当然地扩张以国际偷蒙拐与低人权优势为基的生产体系和贸易体系，拒绝大规模汇率改革，纵容贸易顺差，积攒巨额外汇储备，武器化使用。每一毛钱的货币进出交换，并未反映全球自由派预期，即：需同时携带技术，资本，人性人文的全面交流，彼此互换，替换，自然分配财富，工业格局，不同细分的商业生态于中国人民及全球人民。恰恰相反，以外汇为铆印刷的人民币，只做到了截富于民，还富于贼，货币大量聚集在非法竞争打造的垄断工业体系及其链条企业，以及无大众流通性的不动产资产之上，并反之充当寡头资本/盗国集团抵押物，套空金融机构，无限循环，直至逼近明斯基时刻，即资产回报率不足以偿还债务利息。

***变本加厉***

2015年之后，在延迟了汇率改革7年、占了中国/全球人民便宜7年之后，中共统治家族对内压榨已经接近瓶颈，垄断已垄无可垄，民间些许可获自由的行业油水所剩无几，不动产囤积大量货币，蓄水度已近饱和。出口创汇效率增长接近停滞，普通民众，走至榨无可榨之临界，明斯基时刻呼之欲出。然而对内榨无可榨，只是对外故技重施的开始。中共央行试图在经济‘奇迹’童话暴露真容之前，进一步绑定全球投资者接盘国内资产。汇率政策因而进一步变动。同年11月，第二次大型人民币汇率机制改革宣部实施，史称811汇改。811汇改为央行发布的汇率中间价，引入了一个新的参考值，即前一日收盘汇率，试图以此增强汇率定价中的真实市场供求；同年11月，进一步将人民币中间价 ‘配方’ 改为：前一日收盘汇率+一揽子货币汇率变化，以规避汇率定价中对单一美元的依赖，以更好引入市场化因素定价。

然而，汇改之后人民币汇率一路下行，中共未曾料到，引入‘市场化因素‘背景下，国内外市场玩家用脚投票人民币前程，企业倒闭潮与移民潮2015之后双双密集爆发，市场产生卖空惯性，单边贬值延续接近两年之久，‘卖空共产党’情绪达致历史之最。2017年四月，中共央行推出外汇指导委员会 + 全国外汇自律机制，该机制纳入了几乎所有大型商业银行，政策性银行及大型政策性进出口买卖汇企业；同年五月，该自律机制核心成员，正式推出操控市场情绪的杀手锏，逆周期因子，并给它起了一个漂亮无比的自扯蛋英文名字，Counter-cyclical Factor，以作全世界推广，操控卖盘情绪。这意味着，2017年五月始，中共央行的人民币汇率中间价的形成机制公式里，又加入了第三要素，变为：前一日收盘汇率 + 一揽子货币汇率变化 + 逆周期因子，并对外宣称：外汇自律机制成员（涵盖所有外汇参考价报价银行）会根据经济基本面，自行设定该因子。一言以蔽之，中共试图以此工具，管理外汇交易市场情绪，随时借市场之名、行干预之实逆转任何不利趋势。Counter-cyclical Factor，人如其言，共产党生性认为，人的自主，不配掌控趋势，自由进化，只能进化成猪，我才是你永远的爹。

***逆周期因子详解***

逆周期因子对人民币汇率中间价的影响，无法直接计算，央行及外汇自律机制组织，也并未公布任何计算公式，属于黑箱操作，市场各方只可以根据历史数据自行倒推。据市场各方普遍观察，逆周期因子在应对人民币贬值时动用最多。例如，人民币汇率被市场力量连续做空之后，若只引入”收盘汇率+一篮子货币汇率变化”生成下一个交易日的汇率参考中间价，必会形成做空预期及惯性，导致汇价连续跳空，市场形成单边卖方情绪，此时点后再试图挽回并逆转趋势，成本将消耗极大，必将损耗大量外汇储备为代价。在外贸增量放缓及外资流出的宏观大背景下，汇率操控成本效率至关重要。

而引入逆周期因子之后，单边预期会被跳跃式逆转：做空情绪高涨之后引入‘逆周期因子‘以暗箱计算中间价，使隔日中间价跳涨，从而有效展示央行意图，引导做空力量主动反转，打断悲观力量聚集，扼杀长期贬值压力于襁褓。长周期上，亦可高效降低外汇储备直接干预汇率的消耗，因依靠反向情绪引导系统，用市场抑制市场，自我残杀，打击做空力量。

逆周期因子‘红极一时，与内部资本流出管制，外汇储备入市干预，榨干离岸人民币交易市场流动性（推高利率，即提高拆借人民币以做空的资金成本），调控离岸市场远期外汇存款准备金，形成五大最重要的汇率干预工具。然而，作为最黑箱，过去三年使用最隐秘，以“各报价行按市场化自主决定”之名的外衣伪装最好，宏观影响最悄无声息的工具，逆周期因子，是其中重中之重。

***逆周期因子之殇***

逆周期因子的做庄艺术在于，借外汇自律机制及报价行成员行之口，自称以人民币供需长期基本面为基干预市场，用市场伪装威权，隐秘树立中共护党动机为标靶，并可警告全球做空力量，谁想在一个交易日内制造短期波动，助长长期空方情绪，外汇自律机制成员机构及报价行（几乎所有大中型银行及进出口力量）会辣手摧花，隔日可立即反向调整中间价，无视市场情绪与前日市场汇率收盘价。并美其名曰：市场自律，报价行各有宏观立场，与央行干预无关，形成人民币定价‘导师‘力量。

2017年5月，中共央行首次启动“逆周期因子”逆转长达两年的人民币贬值趋势；2018年8月于川普政府大规模加税之后第二次启用，延续又长达两年，直至病毒战改善顺差环境、人民币需求推出人造高峰、自认美国大选铁板钉钉之后，于2020年10月提出“淡出使用”。最长的一次逆周期因子“附体”，也同时暴露出应用局限，即：逆周期因子，只能短期释放干预信号，比如卖空情绪高涨时，引导人民币短期升值，临时性影响市场情绪，长期只能降低做庄成本（长期直接入市干预的外汇消耗成本），并不能颠覆市场预期，逆转任何经济基本面。在长期情绪、宏观预期明确的情况下（此类预期如，贸易制裁 – 长期贬值，如病毒战制造顺差 – 长期升值），只能持续滴灌应用，仅作降低外储入市干预成本的工具，对改变人民币供需总体趋势，束手无策，功效螳臂挡车。

***人民币战略概览***

人民币过去两年的牌价，因此并未加权太多市场化要素，即并非由自由买卖市场供需定价，这包含自由货物贸易，服务贸易，金融/资本交易。操控自由市场情绪，以市场力量吃市场，可有效降低维系汇价点位的外储消耗成本（由前章逆周期因子即可管中窥豹），让每一个报价点位都为宏观战略服务。这些战略，包含但不局限于：1.维系外贸管理的盈亏平衡点，按战略需求，实现外汇顺/逆差的预算控制。2. 人民币定点国际化，摆脱美元长臂管辖，利用DCEP，对冲SWIFT脱钩影响及香港大陆化的影响（中共60%-70%美元现金流来自香港）。定点国际化，即打造世界贸易支付体系第二极，另立山头，临时维系强势幻像，制造吸引力，向一带一路深度勾兑国家，强推大宗贸易货币，即储备货币，吸收国内烂账资产的印钞，摆脱内部危机，间接借外部力量脱离内部明斯基时刻。然而储备货币地位的前提，是大宗商品交易定价布局，比如布雷顿森林瓦解之后的美国与中东谈判，协议石油美元定价。因此，遵循类似过程，大宗商品战区，石油（伊朗委内瑞拉），农产品（巴西等），金融批发市场（伦敦等），皆为中共执行人民币全球战略主战场，深度蓝金黄甚至病毒外交重灾区。重中之重。
