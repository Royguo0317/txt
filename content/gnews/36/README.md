由于爆料革命的不懈努力，中共操控美国大选和制造投放病毒的证据基本坐实并被大多数美国人民所接受，美国和北约联军对中共实施军事打击的可能性越来越大。如果开打中国将可能形成军阀割据的局面，真的为墙内的民众担心。不过我觉得这种军阀割据的局面会很短暂，本文在此做个推演。

研究一下辛亥革命的历史对我们会有启发。1911年10月10日武昌起义后，由于革命军领袖均不在武昌，当时担任湖北新军协统（相当于今制上校军衔）的黎元洪被推举为武昌革命军政府的都督。随后各省宣布独立，原满清政府领导人成为新政府都督。黎元洪召集各独立省的都督成立“各省都督府代表联合会”，这是一个为筹组新的中央政府的临时性过渡机构。不到3个月，1912年1月1日亚洲第一个共和国中华民国临时政府宣告成立，1912年2月12日清廷在袁世凯和各地独立政府的逼迫下宣布退位，而黎元洪因为首义有功被推举为民国临时政府的副总统，后来还两度成为短暂的总统。彼时的中国人民很幸运没有经受太多的战乱和生灵涂炭，这一方面是形势所迫清廷不得不如此，另一方面也是清廷和袁世凯的智慧，用赎买的方式实现了政权的和平过渡。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/07203845/re-f.png)
我们设想一下辛亥革命的历史会不会重演或者部分重演。如果开战，联军将在短时间内摧毁中共的主要军事目标，比如北斗导航卫星、导弹发射基地、机场、铁路、高速公路、军港、通讯设施、弹药库等，据郭文贵先生透露有3700个。此时习近平为了防止出现反对派，肯定会以安全保护的名义将中共要员控制在自己手中，但是他不可能将各地诸侯均集中到身边控制。联军进攻后，肯定会控制几个地处沿海远离北京的南方城市比如广州甚至上海等。**这个时候只要有某个省的军政首长与联军配合，像黎元洪那样的人出现，就可以在联军保护下成立第一个反对派政府**。由于此时中共的主要军事设施都被摧毁，习近平对各省失去控制，很多省的军政首长很可能都会宣布独立。届时反对派政府就可以联合各省独立政府筹组新的临时中央政府，**那些各省独立政府在联军的威慑下，应该都会顺应时势加入到临时中央政府，否则会遭到联军的毁灭性定点打击**，最后孤立习近平的中共末代朝廷。

由于习近平等人制造和投放中共病毒祸害全球，属于犯下制造生化武器的反人类罪，和平赎买是不可能了。一种可能是中共内部出现袁世凯式的人物，击毙或者拿下习近平等顽固分子向联军投降。一种可能就是在联军的打击下，习近平等顽固分子要么被击毙，要么被俘虏。等待习近平们的就是萨达姆卡扎菲或者米洛舍维奇的下场。当然如果习近平良心发现被迫主动投降，则善莫大焉，可以减轻一些罪责。不过对独裁者来说这种可能性非常小。

**笔者预计，即使最坏的情况出现，最多半年即可结束战斗**。我们相信黎元洪式的人物肯定会出现，我们期待袁世凯式的人物也会出现，郭先生说过中国肯定会有袁世凯。当然袁世凯的出现越早越好，中国少些战乱，百姓少些苦难。**不过郭先生说中国会出现华盛顿式的人物，我觉得这个人物非郭先生本人莫属。**

民国成立不久即陷入军阀割据的局面达十余年，这是因为两个原因，一个是因为孙中山有独裁野心，屡次违约以暴力手段破坏中央政府的运行，甚至为了获得外国资金支持不惜引狼入室和苏俄勾结。另一个原因是当时处于一战之后，没有一个主要的外国力量支持袁世凯的北洋政府，袁也不愿意为了外国支持而牺牲国家利益。袁复辟称帝在某种程度上也是为了避免中国军阀割据四分五裂。**由于有新中国联邦这个国际监督机构的存在，军阀割据的局面最多只是短暂的存在**，有新中国联邦在美国的影响力以及本身的财力，任何独裁者或割据者必将遭到定点打击。从中也可以看出成立新中国联邦的重要作用。

新的临时政府成立后，首先就是颁布法令在全国永久取缔中共，永久禁止社会主义和共产主义。其次就是稳定局面，各地原政府人员守土有责，必须维护当地社会稳定，维持治安，保证百姓的衣食住行通讯等正常生活，尽最大可能恢复生产经营，以使临时政府能在一个稳定的环境下平稳过渡，减少动荡，减轻百姓损失，对原政府人员的安抚就显得非常重要。**新中国联邦宣言中明确宣布要对99.99%的中共党员实行特赦，就是为了安抚人心实现平稳过渡，这是一个历史性的高瞻远瞩之举。同时为避免出现人道灾难，新中国联邦会连同国际社会保证中国人民的食品、水、医疗、能源的基本供应，由此可以看出郭先生经营G系列的高明之处，一切得靠实力说话。**

总之，跟随郭先生三年多的经验告诉我们，只要有郭先生在，总会有办法让中国百姓少受痛苦少受损失，这也是郭先生一片赤诚之心苦心孤诣孜孜以求的。我们在此奉劝中共高层人员，认清世界大势顺应文明潮流，**识时务者为俊杰识正义者为豪杰**，尽早站出来和中共决裂，同盗国贼切割，这是你们立功赎罪为中国人民做件好事的唯一机会，而中国人们也不会忘记你们，历史也会记住你们。当初黎元洪和袁世凯都是清廷体制内人物，黎元洪后来做了副总统和总统，武昌首义永载史册。而袁世凯如果不称帝谁也没法取代他和平演变、结束专制、开创共和的历史地位。千万不要成为中共的殉葬品。**我们也告慰墙内的民众，黎明前的黑暗不会太久，战乱之时好好保护自己和家人，新中国的曙光就在眼前，没有中共的好日子就要开始了！**
