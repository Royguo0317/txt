![]()![](https://gnews.org/wp-content/uploads/2021/03/Screen-Shot-2021-03-16-at-03.48.51.png)*国会大厦 – 图片来源：Olivier Douliery/AFP via Getty Images*

众议院周三通过了拜登总统提出的1.9万亿美元COVID-19 新冠病毒救济方案的更新版本。

投票结果为220-211，除了缅因州民主党众议员贾里德-戈登(Jared Golden)和所有共和党人投反对票外，所有民主党人都投了赞成票。

众议院上月底通过了该法案的一个版本，随后参议院于3月4日通过。参议院取消了联邦最低工资上调 $15美元的规定，并修改了其他条款，如将每周补助失业救济金减少了$100美元至$300美元。两次投票中都没有共和党人支持该方案。

据白宫透露，拜登计划在周五签署该法案，并在近期“上路”，试图让美国人相信这个方案是个好主意。新闻秘书詹-普萨基在华盛顿的新闻发布会上称其为 “美国历史上最进步的法案”。

民主党人利用预算程序，在没有两党支持的情况下，在国会强行通过该法案，引起了共和党的批评。

密苏里州共和党众议员安-瓦格纳（Ann Wagner）周三在众议院大厅告诉同事们，“我起来反对今天摆在我们面前的带有党派色彩的 $1.9万亿美元支出法案。可耻的是，民主党人无视他们为美国人民提供真正的新冠疫情救济的义务，而是试图利用这一程序来塞满党派议程项目。这个法案没有针对性，不及时，也没有与新冠疫情挂钩。我们需要专注于解决当前的关键问题：将疫苗送到美国人手中，为我们当地的企业、餐馆和娱乐场所提供救济，并支持那些受到这场疫情严重影响的人。在这 $1.9万亿美元的庞大一揽子计划中，只有9%用于对抗COVID-19 新冠疫情。而在刺激付款之外，今年甚至有近一半的资金不会被花掉”。

但共和党人过去也做过同样的事情，民主党人指出他们没有违反规则。

众议院预算委员会主席肯塔基州民主党众议员约翰·雅姆特（John Yarmuth） 说，“我们已经采取了应对这场疫情所要求的紧迫性行动，同时遵循预算调节方案所需的每项众议院规则和适当程序”。

他还说，“美国救援计划毫无疑问是积极的，但研究人员和卫生专业人员告诉我们，这是扩大测试规模、加快公平分配疫苗所需要的。他们说，如果我们想拯救生命，一劳永逸地打败这场大流行病，就需要这样做。经济学家们还明确表示需要怎样才能实现强有力的、包容性的经济复苏。我们也再次倾听了（他们的建议）”。

其他人也表示该法案将提供急需的救济。

加利福尼亚民主党众议员萨鲁德·卡尔巴哈（Salud Carbajal）在众议院大厅说，“百年难遇的大流行病需要大胆的投资，这就是美国救援计划。这项法案投资  $200亿美元在手臂上打针，并确保每个人都知道在哪里、何时、如何接种疫苗。许多学生已经回到学校或即将回去。加利福尼亚州将获得 $150亿美元来安全地重新开放（社会），并保持学校开放，以弥补损失的时间”。

“我们还投资了我们的经济支柱–小企业。事实上，经济学家认为，这项法案可以让我们在一年左右的时间里恢复到接近充分就业的状态。为了确保地方政府能够继续提供必要的服务，该法案仅在我所在的地区就向各市县提供了超过 $2.59亿美元的资金。最后，这个方案对美国人民进行投资。直接支付、失业救济、税收抵免和医疗补贴将使数百万人摆脱贫困。”

共和党人宣称，该法案包括了一些与抗击疫情无关的措施。肯塔基州共和党众议员安迪-巴尔(Andy Barr)认为，经济已经做好了全面复苏的准备，同时指责民主党人“利用这场大流行病”来通过一些党派项目，比如给州政府和地方政府送钱，给囚犯送支票。

立法者上一次通过刺激措施是在2020年12月，当时前总统唐纳德-川普还在任。那笔$9000亿美元的方案包括一轮经济刺激支票、为“薪酬保护计划”提供额外资金以及补充失业援助。
