## 【香港要聞】林鄭接受深圳衛視專訪  心懷鬼胎勾結中共 不惜犧牲香港利益  愧對港人

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/14032648/Untitled-1-07_H1Rt2_1200x0.png)圖片：立場新聞
據立場新聞10月13日訊，林鄭月娥12日接受《深圳衛視》專訪，表示香港近年過分強調本土文化，有少部分人排斥內地，將透過教育讓年輕一代更加掌握“一國兩制”，讓他們嘗試進入內地了解發展機遇，減少抗拒感。她還指，香港與深圳之間未必有直接競爭，不介意香港的本地生產總值被深圳超越，期望港深兩地有新的合作平台。

林鄭月娥又指，在香港做事困難，她每次到深圳市，市領導都很關心她，甚至很同情她，因為在內地做事就做事，就能做到，在香港要用迎難而上、排除萬難的精神才能做到一件事，但只要為香港及市民好，特別是為香港的下一代好，這些工作都是值得做，會繼續聽社會的聲音。

戰友點評：

林鄭的這番言論：

第一：挑起大陸人和港人之間的矛盾，割裂中華文化，事實是港人從過去到現在一直救濟內地，改革開放是港人最先進入內地投資，大災大難是港人捐助最多，三十多年來每到六四，港人堅持紀念為內地自由民主呼籲等等。香港文化是勤奮努力，拼搏自強的文化，港人用智慧和雙手建設一個享譽全球的金融中心、物流中心，香港文化同樣是中華文化。

第二：為中共進一步在國內洗腦國人添油加醋，她說香港目前發展有阻力，因土地有很大製約，政治環境比較複雜。事實是香港經濟出現問題是因為歷屆港府被中共控制，將香港的實業轉移到內地，形成香港經濟空心化，勾結地產商和內地富豪大肆洗錢並推高房地產價格，加大港人生活負擔，奴役港人。香港土地有嚴格的規劃和產權，中共無法通過搶劫和強拆奪得更多的土地，抹黑香港經濟發展問題，洗腦大陸。

第三：明目張膽強調對香港年輕一代的教育控制，要香港年輕人接受中共近乎裸奔的“一國一制”替代一國兩制，意圖將港人教育成邏輯混亂，不明是非的奴隸。

林鄭的這番言論真實記錄了她的罪惡，在光復香港後她將接受法律的審判。
