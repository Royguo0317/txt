#### 參賽選手陣容

1號選手 熊119

2017年左右的時候看YouTube，他以為郭先生是個跑到國外去的有錢煤老闆，追劇般的看，直到有一期郭先生的爆料和跟他老家有關係，和他從鄉鄰們知道的事實完全一致，才對七哥有了信任並開始仔細聽爆料，逐漸地成為了一名戰友。之前在國內種地賠的底朝天，p2p被割韭菜十分徹底，心灰意冷去了韓國，但郭先生的勇敢鼓舞了他，滅共不只是郭先生，更是所有戰友的願望。他向朋友們同事們傳播爆料革命傳播真相，見縫插針，希望更多人加入爆料革命成為戰友。

2號選手 美麗善良

來自日本櫻花團的她，雖然之前不間斷的聽過幾次文貴先生的爆料，但沒有持續關注接觸。去年6月份香港反送中活動，200萬香港人的上街遊行的畫面，深深的震撼了她，開始在推特上轉推支持，但是因為家人的反對，被摔過手機，警告離婚後關閉了推特，因為如果離婚只能帶著孩子回國，所以做出了妥協。今年一月份，共產黨病毒爆發之後，憤怒到胃疼的不說，哮喘也有發作。之後的幾天，回想起我自己的經歷和中共體制下的受害者的父母在這邪惡黨所經歷的苦難，下定決心給法治基金捐款，支持文貴先生引領的爆料革命。她說，雖然我是小小小螞蟻，但是推翻中共真的會差我一個，遇見爆料革命是我生命中最重要的故事。

3號選手 文依

深深感受過共產黨的種種惡行的她第一次聽到郭先生爆料就被深深吸引，她相信在中共治下家家戶戶都有這樣的血淚史，很多人選擇了忘記，她卻沒有也不能忘記，沒有共產黨不能幹的，只有你想不到。共產黨禍害中國70年，文化沒有了、歷史的傳承沒有了、人心沒有了、道德沒有了、良知沒有了，擔憂中國人何去何從。現在做聽寫義工，組長派任務就放下手邊的事情馬上做，她說“我越快交戰友們就能越快看到視頻，越快傳播，我知道自己好重要。”群組裡投票、舉報偽類、為戰友的視頻點贊，都積極去做，因為缺我一個不行。

4號選手 fizz

他為路德節目被youtube停止直播而憤怒，他說，這種氣憤是我很久很久沒有感受過的，這種氣憤是我小學時弟弟妹妹被欺負時，那種氣憤，不是為了自己而是為了親人。兩三年來，點點滴滴積累的和爆料革命精神上面的聯繫和共鳴讓他如此憤怒，他才意識到爆料革命已經在精神世界里扎下了根，在大片大片的沙漠當中，那是一片綠地，有小溪，有果樹，有小鳥飛來飛去。如果爆料革命輸了，文貴路德被幹掉了，那片綠地也就灰飛煙滅，下一個綠地也許一輩子也找不到。這是他無法承受的結果。所以參加了727，加入了農場，爭取乾點有意義的事情，為了不讓最壞的結果不出現而努力。

5號選手 海之嵐

她認識和跟隨爆料革命的過程和很多戰友一樣，自從觀看了郭先生的視頻之後，日常就是搜索和觀看爆料革命的視頻，點點滴滴影響身邊的家人、朋友。一邊做飯一邊放放郭先生、路德、 面具先生等爆料戰友的視頻，把聲音開大，讓家人們慢慢接受。勸服不了國內的親人屯糧，就幫他們在網上訂好，不怕被誤會被責備，直到現在中共中央都透出糧食短缺的消息後，家人們逐漸理解。她說，我們不能抱怨親朋好友聽不進勸說，因為他們在國內收到的信息不對等，又被中共洗腦和仇恨教育了多年，所以要給他們時間。她相信當防火牆被打開以後，牆內的平民有機會知道真相，就會了解到中國、世界真正發生了什麼。滅共不只是文貴先生和爆料革命大咖的事，它是我們每一個人的事。

6號選手 傳福音的人

工作之餘聽郭先生的視頻和去教會，這兩件事情是他在非洲工作生活的重要組成部分。在一個香港人帶領的教會裡，向教會的大部分弟兄姊妹分享郭先生的爆料，讓他們去YOUTUBE 觀看，三年來的堅持，讓很多人從開始的不以為然，到後面有的能真正接受，大家也常常為香港守望禱告。今天面對的不僅僅是滅共這麼簡單，更重要的還有滅共後重新恢復，重新塑造價值觀的龐大工程！共產黨種下的毒，摧毀的不只一代人的脊梁，更在方方面面擊垮了這個民族！他說，我們是在重塑自己，重塑新的中國，把生活在黑暗裡的生命都帶到光明中，我們在為下一代預備他們的未來，這是我們存在的價值！

7號選手 山上的城

她是北美的一名小螞蟻戰友，在海外生活了很多年，最早是從基督教媒體了解到CCP對北京守望教會的逼迫打壓，後來慢慢發現海外教會，海外社會也被CCP嚴重滲透，噩夢似乎更近了。作為一個小螞蟻，除了對郭先生敬仰佩服，自己似乎做不了什麼。香港運動讓她對CCP邪惡的認知又一次被刷新，香港這些勇敢的抗爭者們也讓她看到了華人的希望。她相信，神賜給我們每個小螞蟻能力可以移山，因為我們選擇站在正義一邊，站在神這一邊。她因此而創作了一首原創歌曲，並會在本期節目中首次唱響。一點微光是弱小的，但無數個弱小的聲音，無數個微弱的小光合在一起就能震撼天地，照亮黑暗。正義終將戰勝邪惡，天亮了！

8號選手 Nick

他在2017年5月和朋友吃火鍋，被傳福音的朋友傳了郭文貴先生和爆料革命，從此便開始跟隨爆料革命並在國內傳播，在自己的朋友圈相對隱晦地傳播爆料革命的信息，關於大外宣，關於中美貿易戰，關於王健，關於香港，關於美金和糧食等等，每次回國內出差過海關的時候都很擔心，因為微信裡的言論是有監控的。他逢人都會跟他們說CCP病毒的真相，堅定相信爆料革命並且盡力將爆料革命的大福音傳播給身邊的人。用七哥的名字給兒子取名，幸運的參與了GTV投資獲得了一把椅子。他希望能繼續影響國內的家人朋友，讓他們及時防範病毒，幫他們轉移資產，參與到G系列中來。

9號選手 文甜

她和她的先生都是爆料革命的堅定支持者，當看到威廉王發布的招募64露臉合唱的時候，第一反應就要不要報名，她先生特別支持，商量到深夜，考慮到最壞的結果。她傾聽了內心最真實的聲音——“此刻不站出來還待何時？難道我要把這事交給我的孩子去做嗎？不能，我要我的孩子安全、要她能生活在自由裡、要她能去追求自己的理想；不想讓她去呼吸霧霾的空氣、不想讓她去吃含有農藥的食品、不想讓她生活在危險中……”第二天就報名了，錄製視頻，等視頻發出去後，心裡突然就不緊張了，很自豪很驕傲，這是活了幾十年做的最有勇氣最有意義的一件事。當天晚上，她看到先生在收拾東西，把證件、藥品等收拾到了一個包裡，就問他怎麼了，他說以防萬一可以拿著包隨時離開，當時就淚崩了……

10號選手 勺子星ady

60後的他，經歷了文化大革命的尾巴，姐姐是和姜昆同級的老三屆，哥哥是下鄉知青，8964是大學生，當時在學校裡參加學生運動，正計劃著去天安門廣場支援運動，差一點就去了，後來很早移民海外。他全家都支持爆料革命，他帶著太太和兒子一起去韓國喜馬拉雅農場，全家總動員義無反顧的參加了7.27全球滅共大聯動。他不滿七歲的兒子因此得到了一頂新中國聯邦的帽子，之後每週因ccp病毒只上的一天學，兒子卻堅決必須戴新中國聯邦的帽子去學校，兒子說這帽子最帥！為了可愛的孩子能夠平安健康的成長，不再遭受CCP毒害，他說，我們必須勇敢的站出來，用行動支持郭文貴先生，用行動滅共！

11號 漁九傾

她從2017年關注爆料革命，緊跟爆料革命步伐，從微信到土豆，在網上在現實裡，家人和朋友，一直不斷的傳播爆料革命。在現實生活和工作中，她不斷地向身邊的同學朋友同事講ccp的邪惡，讓他們屯糧換美金，剛開始收效甚微，但是她抓住每一次機會進行傳播。 2019年11月參加了糧食危機救援群，屯了糧食和物品，為戰友們做準備，感到份外的驕傲和自豪，終於能從一個吃瓜觀眾，參與到爆料革命的實際行動中來了。 2020年5月在戰友的引領下，做了粵語組的文宣義工，進行聽寫和推廣工作，在GTV在推特進行爆料革命的宣傳。在閒暇時間，她改寫一些歌曲，加入爆料革命的內容，給群裡的戰友大家傳唱，還製作成音頻視頻在群裡在推特再GTV進行傳播，利用爆料歌曲互相鼓舞互相攜持著前行，共同唱出戰友之情唱出大家的心聲。

12號 Estie

從小在國內教育環境下長大，沒有在國內見過什麼黑暗的陰暗面，自己也沒有受過打壓， 所以工作剛出國時也是小粉紅，有人批評共產黨，就真的感覺就是在罵自己，特別難受，然後忍不住站起來跟人辯論。後來受先生的影響開始對爆料革命好奇，他告訴我， 文貴先生說的事都是事實，每一件事都應驗了，我們正在經歷歷史時刻， 我不希望我以後老了後悔沒有傳播真相。所以也開始看文貴先生和路德社的節目，這一下就一發不可收拾。真的像聖經裡說的：萬有相互效力讓愛神的人得益處！讓我們都團結起來傳播爆料革命，傳播ccp病毒真相！
