## 西非國家馬利发生叛乱
`2020-08-18 22:16 GM37` [轉載自GNews](https://gnews.org/zh-hant/302430/)

作者：GM40

西非國家馬利的叛亂士兵領袖今天表示，馬利總統凱塔（Ibrahim Boubacar Keita）和總理席斯（Boubou Cisse）已遭「他們」逮捕。

非洲聯盟（African Union）嚴詞譴責叛亂士兵拘捕馬利總統和總理等官員。 俄羅斯新聞社（RIA）也引述俄國外長鮑格達諾夫（Mikhail Bogdanov）的話說，俄國已收到馬利總統和總理被捕的消息。

同時，兩名安全消息人士告訴路透社，凱塔在首都巴馬科（Bamako）遭叛亂士兵逮捕。

巴馬科外約15公里的卡蒂（Kati）軍營叛亂士兵，稍早逮捕若干高階文官與軍官，並將他們帶回軍營。

法國總統馬克宏（Emmanuel Macron）與西非國家領袖討論馬利情勢，並敦促西非經濟共同體（ECOWAS）調解。 美國、法國和西非經濟共同體均譴責這項叛亂。

席斯稍早發布簡短聲明，呼籲叛亂士兵停止叛亂活動，並以對話解決當前情勢。 法國外長勒德里安（Jean-Yves Le Drian）稍早在聲明中表示，法國譴責這項叛亂，並敦促叛亂士兵返回營區。

卡蒂軍營2012年的叛亂導致軍事政變，當時總統托赫（Amadou Toumani Toure）被推翻，加速馬利北部落入聖戰士手中。聖戰士持續在馬利北部和中部運作。

凱塔被控貪汙以及國內治安日益惡化，自6月一直面對大規模示威，示威群眾要求他下台。
