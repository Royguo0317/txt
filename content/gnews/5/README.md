**据《大纪元》援引路透社，2021年5月5日报道：**

美国贸易代表凯瑟琳·泰（Katherine Tai）（Bill O’Leary / Pool通过路透社）

华盛顿——美国贸易代表凯瑟琳·泰（Katherine Tai）周三表示，她希望与中共国官员“在短期内接触”，以评估两国之间“第一阶段”贸易协议的执行情况，其结果将影响华盛顿对北京惩罚性关税措施的命运。

泰在《金融时报》线上活动上表示，她尊重美中贸易政策连续性的必要性，包括川普政府去年实施的为期两年的贸易协议。

泰说：“这（贸易协议）是我们达成的协议，它将是我们工作的起点，它也是我们建设（美中贸易关系）的基础。”

她补充说：“我非常期待与中共国同行正式会面，评估他们的表现并衡量他们说的话，以此来表达我们的兴趣并为前进道路打下基础。”

泰说，她尚未与中共国国务院副总理刘鹤——第一阶段贸易协议谈判的北京代表——或任何其他中共国高层官员建立正式联系，但她希望“在短期内”取得联系。

当被问及美国前总统唐纳德·川普（Donald Trump）每年对价值约3700亿美元的中共国商品加征的关税是否会在近期被乔·拜登（Joe Biden）的政府取消时，泰说，这很大程度上取决于与中共国官员的对话以及第一阶段贸易协议的“有效性”。

拜登政府对特朗普政府关税的态度，还取决于“我们在推动这一特殊而且非常重要的关系方面所形成的愿景。”

**贸易审查**

泰和其他拜登政府高级官员正在对美中贸易政策进行从上到下的审查。川普于2018年在世界两个最大的经济体之间发起了一场关税战。华盛顿和北京于2020年1月签署了第一阶段贸易协议。

该协议于2021年底到期，协议要求中共国在2年内将对美国出口的购买额增加2000亿美元，而北京远没有达到这一目标。中共国政权还承诺对美国公司的知识产权保护、农业生物技术和金融服务准入进行监管改革。

泰说，与中共国合作领域的最大希望之一就是应对气候变化，这是拜登政府的主要政策推动力。她说，许多经济体之间需要合作来解决全球变暖问题，但气候问题为发展新产业和基于创新的更公平竞争提供了机会。

当被问及美国是否将与台湾进行自由贸易谈判时，凯瑟琳说，与台湾的任何贸易和经济接触都必须与拜登政府在国内外的其他政策目标协调一致。她淡化了与台湾达成全面自由贸易协定的前景。

“我想打破这种观念，即我们采取积极的贸易参与的唯一方法只是通过自由贸易协议。让我们考虑一下灵活性，考虑我们拥有的所有工具，让我们随机应变。” 泰说。
