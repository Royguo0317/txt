美国时间12月12日，《门道专家》独家报道了多米尼投票机的软件具有自动生成选票的功能。基于这个功能，操作人员可以轻松地生成“成堆的假选票”。文章链接如下：



> [BREAKING EXCLUSIVE: Dominion Voting Machines Have the Ability to Create Ballots!](https://www.thegatewaypundit.com/2020/12/breaking-exclusive-dominion-voting-machines-ability-create-ballots/)



这个发现来自于独立调查记者凯尔·贝克（Kyle Becker），他引用了一位名为“DataDude316”推特用户的推文。“DataDude316”是一位根据公开信息和情报进行研究的人士，同时也是一位有证书的数码操作员、测试员和分析师，他在推文中写到，

“多米尼可以被黑客入侵吗？是的。

可以给多米尼输入一堆只标记了拜登的“测试性选票”吗？是的。

这些“测试性选票”可以被多米尼视为合法选票吗？是的。

可以整批清除川普的选票吗？是的。

那我是怎么知道的？因为它写在了多米尼的说明书手册中。”
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/13025202/Picture30.png)图片来源：推特截图
在多米尼公司的官方网站上（[https://www.dominionvoting.com/optional-solutions/](https://www.dominionvoting.com/optional-solutions/))，关于多米尼投票系统“可供选择的解决方案”，有以下一段描述：

“自动测试平台生成器（The Automated Test Deck Generator）可创建全面的测试，进行高效、便捷的逻辑性和准确性测试。使用选举项目的数据库，可以随机生成一堆带有标记的测试性选票，来最大程度地保证系统的准确性。进行扫描时，这些自动生成的测试性选票可以和实际制表的结果进行对比，从而验证打印出来的选票的质量以及每个制表器的完全准确性。”

并且，在多米尼投票系统说明书的第30页中也介绍了此功能：

“如果您所在的辖区允许的话，请使用自动测试平台的应用。多米尼投票系统的这个应用，可以自动生成适当的测试平台（选票）和希望的结果总数。”

而非常凑巧的是，乔治亚州购买了这个“自动测试平台”的可选性解决方案。根据乔治亚州的采购文件的第59页显示：

“自动测试平台的创建 – 多米尼提供了一项可选服务，来创建自动化的全面测试平台，以协助客户进行逻辑性和准确性的测试。通过使用选举日的数据库，可以生成一系列基于计算机算法预先标记的选票，设计这些算法的目的是提供最高的系统准确性保障。这些测试平台（生成的选票）一旦被扫描，就可以创建一个已知的结果，用来和制表的结果进行比较。这可以消除由于手工标记而导致的错误，从而提高对测试结果的信心。”

来自于乔治亚州富尔顿郡（Fulton County）的投票站经理苏珊·沃伊尔斯（Susan Voyles），在听证会上就作证她看到过成批的具有相同标记的选票，其中标记拜登的选票高达95％以上。

并且来自宾夕法尼亚州的证据表明，费城也从多米尼公司购买了这一选项，请参阅费城官方投票网站上公布的多米尼系统文件的第89页：
