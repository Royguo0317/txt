**李家超：研**** 23 ****条立法填补国安法不足　料本届政府难完成**

保安局局长李家超接受《星岛日报》访问时表示，基本法23 条立法循叛国罪、颠覆罪、分裂国家罪、煽动叛乱罪、窃取国家机密罪、处理外国政治团体在港活动等七个方面去做，国安法则主要针对4 个层面，故政府应尽法立23 条以填补不足，但他承认立法程序复杂，料本届政府任期内难以完成。

李家超指，国安法针对四个范畴，即分裂国家、颠覆国家政权、勾结外国及境外势力、组织实施恐怖活动，其实并未完全涵盖基本法 23 条的范围。他说，政府持续研究 23 条立法，方向将包括「已涵盖范围需不需要强化？是否有其他方面都要涵盖？」

他又以社团条例作例子，认为要研究优化社团条例以处理「严峻行为」，另外，回归前订立针对叛国罪、刑事罪行等条例，也要研究是否要强化或修订。同时，有关泄露国家机密，也要研判现行的官方保密法是否足够。 23 条是采用普通法立法，李家超表示，会研究外国相关法例，亦会看 2003 年提出的 23 条立法草案建议内容是否仍然适用。

李家超又表示，《国安法》实施后，本港的罪案，包括纵火案、刑事毁坏案等数字大幅下跌，认为《国安法》「明显产生正面效果」。在未计及《苹果日报》案前，截至上星期，国安处处理的案件有 11 宗，拘捕 109 人，检控 62 人。

她又指，大众经常将限聚令聚焦于游行，但其实条例是关于「聚会」，而室外聚会不一定是指示威游行，也包括周末的聚会。她重申，这些「难以规管、行为本身有风险、社会上没有强烈营业需要或市民社交生活要求」的聚会，需要政府在衡量后作出判断，而有关判断并非针对某一项特定的活动。

**周四起放宽食肆聚会等人数****研恢复港澳通行**

本港近日疫情缓和，特首林郑月娥今日（21日）下午宣布，自24日（星期四）凌晨起，放宽部份社交距离措施，包括食肆营运限制及已接种疫苗人士抵港后强制检疫安排等，为期两周，至7月7日。

食卫局局长陈肇始公布，实施的新一轮社交距离措施内容，会维持4人「限聚令」。餐饮业4类的运作模式中，A类及B类食肆限制维持不变，提升C类D类人数上限。宴会人数上限放宽至180人。

宗教聚会人数上限放宽至50%，若2/3出席者已接种首剂疫苗可达至100%

酒吧、夜店、夜总会每台人数上限放宽至4人。卡啦OK、派对房间每间房或每台人数上限增至8人，麻雀馆人数上限亦将放宽。

政府亦研究放宽已注射疫苗人士抵港的检疫期，若接种两剂疫苗满14天，则强制检疫期可缩短至7天。

对于本港疫苗接种率，林郑称情况不理想，已接种第一剂疫苗市民人数，仅占合资格人口的28.5%，料距离建立屏障仍有较长一段时间。她强调，疫苗并非本港制造，不论市民对政府有何看法，都恳请市民打针。

与大陆通关方面，林郑指，因上月广州出现感染个案，「来港易」计划仍未推出。待当局密切监察情况稳定后，会再推行「来港易」。同时，港府将与中央商讨较大型的人员往来政策，研究如何让香港及大陆人员恢复减低检疫要求的同行。
