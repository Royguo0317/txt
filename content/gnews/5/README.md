读德教启蒙第十期有感：从小我们在学校读书的时候学习达尔文的进化论，殊不知这个理论对我们有深深的危害。读完这期德教启蒙，我才了解达尔文主义的来龙去脉，不然还以为我们就是猴子变的，就是比较高级的动物而已。

翻开历史，达尔文从小家里就让他上神学院，当神父，但他不愿意，只喜欢研究植物，生物学。曾被邀请跟随英国海军猎犬号对南美洲进行长达5年的考察。在这期间他研究各种不同的生物，比如发现不同种类的蝴蝶，慢慢认为动物都是有关联的，只是演化进程不同而已。于1859年写了本书基于自然选择的《物种起源》，阐述各种生物都是为了适应环境而不断地竞争与变异。达尔文信撒旦教，反对基督，所以进化论也称为魔鬼的圣经。但这只是在生物上的学说，直到英国托马斯·赫胥黎（Thomas Henry Huxley）将达尔文主义发扬光大，还有美国的共产党人理查德·霍夫斯塔特（Richard Hofstadter）把达尔文主义引入社会学，认为人类应该像动物一样，弱肉强食。这也就给所有的帝国主义，殖民地，战争找到的藉口。因为弱就要被欺负，把人变成动物，这才是罪魁祸首！

我们再看看达尔文主义是如何引入到中国的。1895年，也就是《物种起源》发表36年后，北洋水师学堂的严复歪曲翻译赫胥黎的《进化论与伦理学》。一般来说，翻译尽量是跟原文一致，但是严复却只强调前面的进化论，不谈伦理学，翻译为《天演论》，认为自然界的进化规律完全适用于人类社会，还自创“物竞天择，适者生存“。并不是社会进步就要杀死弱小的。所谓落后就要挨打，世界上落后的国家多了去了也没见被欺负，是欠揍才要挨打，不作不死！

当时正是西方列强打开了闭关锁国的中国。特别在经过甲午海战后，知识分子都想救国。我们前几篇有提到过，一直打败仗，一直反思，从改器物到改制度，最后认为文化不行要改思想，就有了五四运动及新文化运动。当时主流的两种思想就是《天演论》与《民约论》。《民约论》即社会契约论，讲天赋人权，人生而平等，反对专制。而严复歪曲翻译的《天演论》则是中国版的进化论，把斗争哲学引进到人类社会，因为生产竞争，这就很容易被坏人充分利用。如果按照进化论的优胜劣汰，当时中国很落后，我们是不是应该都被淘汰呢？为什么说近现代无大师？因为当时无论保皇派还是革命派都接受进化论，也就是种族主义思想。

当时的康有为虽然追求仁爱平等，但是在世界种族问题上则认为不平等，认为白种黄种人是最优秀的，对黑种人充满鄙视。而革命派的孙中山，邹容则是国内的种族主义，认为汉族受满族严重迫害和奴化。还有梁启超，崇尚强权及武力，认为实力一样就要讲道理，实力不一样就不用讲道理。这不是强盗吗？也就是民国的各位思想家都接受进化论，彻底改变国人的思维模式，中华传统是讲究道，遵循道的规律，天道不变，道义不变。而达尔文主义的弱肉强食，丛林法则的动物思想，给马克思思想的传播奠定了基础。

马克思还曾给达尔文写过信，这两人都给人类带来了灾难。马克思讲唯物辩证主义，不信神，那进化论就是反对有神论的天然盟友。马克思、恩格斯、毛贼东都是推崇进化论的，把动物这套引进到人类社会，把斗争学说植入人的思想。相信进化论的也就很容易相信唯物史观。一个是生存斗争，一个是阶级斗争，都是弱肉强食。如果人就是动物变的，就是高级动物，那每天就食色性也，每天搞物质跟交配就行了。

按照进化论的理论，如果猴子在进化，那猪也在进化呀，我们有没有看过直立走路的猪啊？那是猪八戒，妖怪变的。达尔文看到的各种不同的蝴蝶，认为生物都有关联，其实只是生物的多样性而已。在2009年即达尔文诞生200周年的时候，社会各界对达尔文主义，社会达尔文进行了各种批判。在美国就有超过500多名不同学科的科学家联名发对进化论。所谓人上人，就是踩着别人爬上去，再去吃别人。达尔文主义跟马克思主义对人类产生深远危害。首先人类自身认识大倒退，从人变成动物，跟猪一样只知道吃跟交配。再到马克思的斗争哲学，把人从动物再倒退成魔。所以共产党说党员都是特殊材料做的。因为他们都不是人，是魔。达尔文主义使原本充满温情的人类社会堕落到动物世界，把人的各种高贵品德都隐没了，只知道物质与交配，弱肉强食甚至给社会上各种不公、战争提供了理论依据，我比你强就要去抢你。

达尔文摧毁基督教，不相信上帝，把人堕落到动物，竞争意识深入人心，所以尔虞我诈，欺负别人是合理的，殖民地也是合理的，抢也是合理的。再向外延申到物质主义，欲望泛滥，除了吃就是交配，只要自己舒服，同性恋，甚至动物都可以。所以找毛病就是要先找到历史渊源，才能对症下药。

我们再看看中华传统文化怎么讲的。从《礼运大同篇》提到的“大道之行也，天下为公”，也就是天下是公天下，是一切民众的，而不是家天下。中国历史进入夏朝后（夏启篡位）就从公天下变成家天下了，到秦始皇的大一统，连封建（分封建国）都没了。所以我们要反对大一统，反对专制极权，而不是反封建。“故人不独亲其亲，不独其子其子”，不仅要爱自己的父母和孩子，也要爱别人的父母和孩子。“使老有所终，壮有所用，幼有所长，矜、寡、孤、独、废疾者，皆有所养”。老人能安享晚年，年轻的能有所作为，孩子能健康成长；失去爱人的，没有父母，没有孩子的都能得到很好的照顾。“男有分，女有归”。这就是总结几千年前三皇五帝的宝贵经验，当时就能有这么美好的大同世界，最终还是人心的问题。

改变人心，唯有靠德行的教化，战友们，让我们一起努力吧。
