## 郝海東怒斥中國足球假騙偷
`2020-07-23 09:17 喜馬拉雅戰鷹團 Himalaya Hawk Squad` [轉載自GNews](https://gnews.org/zh-hant/273674/)

聽寫：Janny

編輯：Giselle

審核：海闊天空

Page：拱卒

本文根據《路德訪談》2020年7月11日視頻20:56- 23:07 內容整理



郝海東：有句老話說“千好萬好不如青島，千難萬難不離嶗山”。青島最早被德國殖民，（是德、日在華的商業中樞）。青島的足球也很好，還有跑馬場、青島啤酒、嶗山可樂。嶗山可樂最好喝！裡面有中藥成分、大棗等原料。那是我小時候最深的記憶，開運動會當時買一瓶嶗山可樂……

葉釗穎：還有餅乾。

郝海東：鈣奶餅乾，哈哈！青島有海有山，青島人骨子裡性格堅韌、堅強，當然也有一些愚忠的人，畢竟是山東嘛！有一些“君君臣臣”這些東西，但是也出了很多有天賦的。這種內心性格，有它的特質，骨子裡有一種不服輸！講個笑話，我爺爺70多歲還跟人打架，把手打傷了。所以老天爺最後給了爆料革命和新中國聯邦最好的禮物，就是小閻（英雄博士閆麗夢）！確實特別特別激動！昨天小葉也忍不住掉眼淚，我也哽咽了，和文貴先生通話，確實太激動了！

郝海東：剛才墨博士提了“對行業的尊重”，中共對行業沒有尊重，它們不知道什麼叫行業規律。它們就是拍腦門，誰官大誰來做決定。我們都是一線球員，我10歲起就一直在足球行業裡。有人在課堂上學幾年後，就叫所謂的“足球博士”，這種人他能跟我講什麼呢？他跟我講郝海東足球該這麼踢、那麼踢，那不對是吧？你對行業不尊重，讓我們還踢啥啊？你來踢不就得了？你不要老講對吧，你也上去看看這個前鋒、前衛，後衛、邊後衛、守門員、中衛、前腰、邊鋒、中鋒……他們總以為自己很懂，為什麼？他們看了十幾、二十幾年球，能看不懂嗎？他們認為自己能看懂。所以他們對行業不尊重，對真話恐懼，他們不願意聽到真話。

比如他們搞的這種身體素質測驗，說你能跑多遠？多快？但這不是他們最終的目的，他們打著旗號說讓你練成多快、多遠，但是最重要的就是他們可以通過這些測試獲得他們想要的利益！裁判員受賄賂、他們受賄賂，完了再去操縱你、管理你！ “這麼嚴格”的測驗、測試，怎麼能弄虛作假呢？他們都是弄虛作假！只要你想通過測試，給裁判錢就能弄虛作假。像我這種到了國家隊的，他們為了能讓你過、能參加聯賽、能讓你比賽，他們就會弄虛作假。我的測驗早上6點鐘找倆裁判到香河，弄一喇叭、慢悠悠往那一放，我和裁判聊著天，他按完嘟嘟嘟，那種磁帶伴奏完，我們聊了半個小時，最後說:“哦好！郝海東過了”。這就是他們的假、騙、偷！

葉釗穎: 你今天在這是爆料了！之前你是說，但是沒說那麼詳細。你之前一直說:“我肯定能過，他們永遠都會給，但是不說這麼詳細，別人也不會知道。為什麼別人每次測驗不合格，怎麼你每次又給過了”！

郝海東:他們對所有行業的褻瀆就是這樣。雖然我得益了，但是我從內心裡很厭惡！我可以不批評他們，就像他們說的：郝海東你別說話就行了，跟你也沒啥關係對吧！你也能過，幹嘛抨擊國家政策呢？

葉釗穎: 很多人覺得，每次你是過不了才抨擊。你不願意……

郝海東: 對！不願意讓人家有這個測驗！

他們一直胡說什麼前鋒不需要耐力，你還得防守呢，還需要滿場跑呢，還得搶球呢！他們混淆概念。球員有各自的特點、各種位置的不同，不需要誰來規定我能不能上場，我踢得好壞，由俱樂部、教練員、市場來決定。但是他們不是，他們要來決定。因為他們要控制你，你只要不聽我的話、不跟我勾兌，我就不讓你過！我就可以利用這些東西不讓你踢，或讓你踢的不舒服。

我就不高興！我就一直抨擊他們！他們有時候對我也很不滿，但是又沒有辦法。有時候禁賽我一年半年，也不想再招我到國家隊了，但是沒辦法！我水平擺在這！郝海東可不是踢了一屆4年、5年國家隊，3年、8年國家隊。郝海東踢了十幾年！我從1988年到奧林匹克隊，1987年國家2隊，到1992年國家隊，一直到2004年。經歷的這些外籍教練都是施拉普奈、霍頓、米盧、阿里漢！這不是我郝海東牛不牛B，是市場以及這個行業的選擇。他們選擇我進國家隊，我沒給過他們（足協、教練員）一分錢。如果我郝海東但凡有一次這樣的行為，你想想，以我這種性格罵了這幫當官的，還有我一天的好嗎？沒有一天能這麼呆著，早進監獄了對吧。

徐明當時控制了3個隊，大連實德、四川冠城、遼寧。他後來還讓趙本山當過一段時間的俱樂部的董事長，四川冠城也是他跟張海（深圳健力寶）控制。我打完比賽了，張海都給我打電話：“海東啊，踢得好！”因為我們客場贏了山東魯能，我當時是主教練兼隊員啊！我當時踢球還當主教練。徐明說，海東今天好好打，今年保你冠軍！然後我們打完大連實德和四川冠城的比賽，徐明給我打電話:“海東不錯啊！”我們客場剛贏了2 : 1！ “我們給你打個電話感謝一下”！你想想半年以後我就不干了，我說你們愛找誰當這個主教練就當去，我就接著踢球！

2004年我（離開球隊）去了英國。 （中國足球行業）已經爛到根了！操縱遍布整個比賽體系，已經沒有真實了。你只要想真實的、想要把這個事情、行業真實，就成了另類，最後就要倒霉！

最後他們控制球員、控制教練，有時候連球員的錢也不發。我們很多球員踢1年，白條寫著欠你40萬，就這樣！當然他們沒欠過我一分錢，我從來沒有合同，口頭上講幾百萬，他們就得給！

葉釗穎：今天爆這個料，估計人家球員都不知道！你都沒講過！

郝海東：郝海東在中國踢了一輩子足球，沒有見過一份合同，沒簽過一個字！
