**内部消息**

消息人士内部得到消息，进入2021年6月以来，墙内3家银行机构（含分行）发布了四期个贷不良转让公告。参与批量转让个贷不良开始加速。

**时间轴**

2021年

- 6月4日、6月6日：
工商银行陆续发布了2021年第四期、第五期个人不良贷款（个人经营信用贷款）转让公告。
- 6月8日：
农业银行重庆分行发布了2021年第一期个人消费类不良贷款转让公告&gt;
- 6月10日：
民生银行广州分行发布了2021年第一期个人经营类信用不良贷款转让公告。


在此之前：

- 2021年2月：
工商银行、平安银行公布转让首批个贷不良资产包公告&gt;
- 2021年4月：
交通银行发布了2021年第一期个人消费信用类不良贷款（北京地区）转让公告。
- 2021年5月：
又有三家银行发布了三期个贷不良转让公告。


银保监会正式印发《关于开展不良贷款转让试点工作的通知》5个月后，参与批量转让个贷不良的银行机构逐步增多。

截至6月11日，已有包括140家国有大行（含分行）、143家股份行（含分行）开立了不良贷款转让账户。

**个贷不良资产包金额逐渐变大**

值得注意的是，随着批量转让个贷不良试点逐步实施，个贷不良资产包的金额也逐渐变大，除“几百万”的小包外，一些“千万”以上“大包”也逐渐浮现。

举例：

- 民生银行6月15日拟转让的个贷资产包本息合计1862.68万元
- 平安银行6月15日公开转让的个贷资产包本息合计达5967.98万元


**消息人士指出**

消息人士提示诸位战友近日满屏的平安集团市值蒸发超4000亿，似乎里面有着某种必然的联系。

**分析认为**

随着个贷不良资产包不断增大，各家资产管理公司将不能再以“试水”的心态选择“拿包”，后续资产管理公司该以怎样的价格接手以及是否实现盈利成为资产管理公司关注的重点。

笔者认为此番个家银行纷纷急于出手个贷不良资产包，预示着金融经济崩塌已经开始，虽避而不谈房地产类个人贷款，但是一定隐含着墙内房地产崩盘前的风险转移。
