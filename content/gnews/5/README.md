上一篇中我们谈了人性与信仰在人类的早、中期的变化，本篇则谈谈其在后期及当代的变化。

四、专制社会

随着时代的进步，人们发明了很多先进的生产工具。特别是近一百多年，自动化的生产工具雨后春笋般地涌出。这些工具便捷了各行业的劳动者，创造了大量的财富，也产生了大量的剩余价值。在这种条件下，人类的某些区域进入了专制（集权）社会，致使原来的君主、国王、酋长及新生的家族集团变为专制的主体（统治者）。当中还有一些国家的统治者是幕后的，这些国家表面上看是有一个公开的政府机构，实际国家却是由背后一些真正有权力的人（集团）控制着。

在专制社会，大量的财富是由少数人控制的：他们借着权力霸占了劳动人民创造的大部分财富，享受着国家所有的免费福利，过着奢侈又荒淫无度的生活。然而，大部分的劳动者只能维持着温饱生活，而且不能享受免费的教育、医疗、养老的福利，甚至仍然有相当多的人民过着贫穷的生活。更重要的是，这种社会里的人民没有言论自由、没有人身自由、没有财产安全、没有生命安全，统治者（各级政府）可以随意剥夺任何人的财产和生命。

这种社会没有公德与法制，只有权力和财富。同时，社会中人性的善良、同情、爱心已被压在心灵仓库的最底层，相反，其中的自私、贪婪、丑恶、残暴被极度地放大，赤裸裸、毫不掩示甚至“荣耀”地被展现出来。另外，这里的人们在信仰上别无选择，无论信仰崇拜任何宗教、任何神，最后都落到了权力和金钱上。正如中国百姓说的那样，“穷的只剩下钱了！” 而专制社会之所以会沦丧到如此地步，其罪魁祸首就是统治者及他们制定的法规——中共国就是典型的例子。

五、民主社会

据说当代国际社会中有一百多个民主国家，但真正意义上的民主社会我至今还没看到（也许是我孤陋寡闻）。即使是以美国为首的民主国家，他们虽然是民选总统、民选议员，但选举活动结束后，民主就走完了过场。胜选者将竞选时对人民的承诺转眼就忘光，只是完全按照出资人的要求行事。同时，全国民众完成选举任务后，基本上也没有什么权力能够过问政府的事情，甚至对于有些直接关系到民众基本生活的事情，他们也无能为力。所以，当今世界上民主国家的选举制度基本上是个面子工程，其真正控制政府权力的都是那些躲在沼泽地里的超级资本家——美国2020年的大选就是例证。反观美国历任总统，从上世纪七十年代的尼克松到今天的拜登，基本上罕见能兑现竞选时承诺的总统（川普可能是最不错的）。

从历史的发展过程看，人性与信仰即是与利益紧密联系的，又是与社会规则（法律）和管理机构息息相关的。我认为，人性是通过利益展现出来的。另外，人的信仰首先也是为实现利益而作出选择的，其后信仰又受到社会规则（法律）的约束，但其实规则（法律）又是由管理者（统治者）制定、掌控和的执行的。这就是说，人的信仰是由个人的利益和统治者规则决定的，同时也是随着社会制度的变化而改变的。这使得我们可以最终得出一条定律：社会的规则（法律）是由掌控社会的权力者决定的——谁掌控了社会的权力，谁就是社会制度的主宰者。那么，如果人民大众掌握了社会的权力，所有的规则（法律）就是由人民大众来制定，而人民作主的社会也就是真正的民主社会了。

我认为，真正的民主社会应该是人民大众掌权：对于各级政府的人员，应该像《新中国联邦宣言》中描述的那样，由一人一票选举出来的；所有的法律法规都要经过人民大众的讨论，而后再一人一票的通过。另外，各级政府官员上任后首先需公布个人财产和工资收入情况，且需要把竞选时承诺的目标项目长期公布在政府网站，由人民大众组成的机构进行监督。为此，他们需要每年向人民作年终汇报，公布年初承诺的各项目标的完成情况。同时，社会需把民众参与社会监督管理的条款写进宪法，也要写在实体法律和程序法律中。社会还要把民众参与社会工作的责、权、利的关系也写在法律中。这样就能使每一个公民都知道，参与社会工作是必须的责任，是和每个人的利益紧密相关的。相对的，社会要充分利用先进的科学技术，利用人性中积极向上的优点，调动一切能参与社会工作的公民，使尽量多的公民参加到社会管理中。在这样的社会中，那些想侵占公共利益、侵害他人权益的行为就难以得逞。在这种制度下，社会发经济发展的速度可能会慢一些，但当社会上大幅度地缩小贫富差别后，所有劳动者的福利待遇都会有大幅度的提高。此时，每一个劳动者都能获得自己合理的收入，所有无劳动能力的人都能得到应有的福利。这里的人民没有恐惧、没有威胁、没有生活顾虑、没有邪灵恶魔的残害，大家都生活在自由、平等、善良、互助、和谐、无污染的环境中——那是多么令人向往的社会啊!

我认为，做到这一步并不难，但当然要有一个过程。因此，倘若能合理使用法制和宣传（教育）这两个重要且高效的工具，相信经过几年时间就能有明显的成效。那么，只要通过几代人的努力，我们一定能实现这个美好的愿望！
