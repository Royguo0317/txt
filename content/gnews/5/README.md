## 包子脑瘤手术，现中共最后垂死挣扎
` 喜马拉雅 日本京都富士会` [轉載自GNews](https://gnews.org/zh-hans/716088/)

![]()![](https://gnews.org/wp-content/uploads/2021/01/77fccbcb37f8b4f9.jpg)
作者：招财娘娘 (京都富士会）

最近几天，有关习包子进医院做脑瘤手术的爆料，经《路德社》首发后，广受全球关注，包括日本、香港，台湾在内的媒体纷纷跟进。在手术前和手术后，习包子心虚的发了很多“大招”，包括送钱换取与普京通电话等。1月4日，他更是发布“ 向全军发布开训动员令”以及“公布新党员条例”来巩固“一尊”地位，种种迹象就像垂死的人想要抓住最后的救命稻草。

1月4日，习近平签署所谓“中央军委2021年1号命令”，向全军发布开训动员令。另外，时隔16年再次修订《中国共产党党员权利保障条例》。新修订的《条例》多了两千多字，从2004年版本的五章38条，扩写为五章52条。以上两个动作，都表明患上脑瘤的习包子，自知气数已尽，时日无多，用尽最后力气揽住岌岌可危的军权与党心，想说“死也不放手”。

在社交媒体上标签为“曾经的中共中央党校教授、现与中共党彻底脱钩”的蔡霞，北京时间1月5日发文表示：“ 中共公布新党员条例，是加强9000万党员围着一个人转”。

在墙内的媒体表述中，“新条例”令中共党员享有更多权利。 但作为曾参与中共制度与理念的建构的前中共中央党校教授，蔡霞非常清楚新条例隐藏的意思。她直言，这次颁布的新修条例，是中国共产党的“全面倒退”。“任何制度的制定，都有基本制度理念，就是一种价值取向、价值选择。但这次的新修条例，基本理念是控制全党，而不是真正为了党员行使民主权利。”

笔者咨询了一些专家，从医学的角度，脑瘤手术属于“高风险”手术之一，那么最近习包子的手术“成功”，能否代表他就一定能康复？实际上，一个人一旦患上脑瘤，那么依据肿瘤生产部位不同，可出现视力、视野、嗅觉或听觉障碍及肢体运动障碍，也如同一个政党出了问题，方方面面都会导致恶性循环一样。所以医生选择手术方案时，往往有一定的局限性，对患者的自身情况有一定要求，特别是年龄大的。简单来说，原则上，手术治疗脑膜瘤应争取完全切除，手术切除要把周边的脑膜一同切掉才可以，但即使为全切术，在原发部位仍会有肿瘤残存，所以很难根治，复发率很高。

目前，尚不得知习包子是肿瘤全切除，还是半切除。但可以确定的是，存在习包子身体内的脑瘤细胞，不可能移除干净。而习包子的脑瘤，与统治中国长达70年的中共何其相似！无论怎么改头换面，都只是局部止痛，并不能除根，因为根的本质是邪恶，邪恶的细胞早已渗透到方方面面，无论华佗在世还是扁鹊降临，都无法妙手回春。唯一的方法是摧枯拉朽，从身体到灵魂，彻底的推翻换掉。
