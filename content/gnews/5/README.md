《每日邮报》报道，鲍里斯·约翰逊(Boris Johnson)将于今天宣布，即将对英国的国防，安全和外交政策进行重大评估和调整。该战略将定义英国作为世界大国的未来，表明英国将在安全和威慑方面采取更加强硬的立场。同时，英重大评估报告警告称，俄罗斯和中共是“最大的威胁”，到2030年将可能出现生物武器攻击。

鲍里斯·约翰逊(Boris Johnson)将于3月16日公布一份庞大的国防、安全和外交政策评估报告。这份报告被标榜为自冷战结束以来对英国在世界上的地位进行的最激进的重新评估。报告长达100页，名为《竞争时代的全球英国》(Global Britain in a Competitive Age)，报告建议英国对核弹头数量的限制从180枚增加到260枚。安全和国防的重点从常规力量转向尖端战争，投资于太空和激光武器。它认为，增加核弹头上限是“对不断演变的安全环境的认识”以及“不断发展的技术和理论威胁的范围”。为了保障英国及其盟友的安全，分配给北约的最低限度、可信的、独立的核威慑力量仍然至关重要。

报告称，英国将“毫不犹豫”捍卫其价值观，并将加强对关键基础设施的保护，如国家电网、交通和供水。报告支持建立新的最先进的反恐行动中心，以在发生袭击事件时简化警方和情报机构的反应。令人担忧的是，报告指出，在未来十年里，另一场大流行病“现实的可能性”很大。它强调普京领导下的俄罗斯是“积极威胁”，同时强调中共对繁荣构成了“系统性挑战”。到2030年，成功的化学、生物、放射性或核(CBRN)攻击被认为是“可能的”。

鲍里斯·约翰逊(Boris Johnson)将在安全与威慑问题上采取更强硬的立场，这可能会让军队更定期地、更长时间地派驻海外。约翰逊称，“我对英国在世界上的地位和我们抓住机遇的能力非常乐观。”

预计约翰逊随后将在下议院表示，英国需要使用“我们所掌握的一切工具”，“在一些国家试图破坏开放和自由的国际秩序时，确保一个民主仍能蓬勃发展的世界。”

然而，军方领导人明确表示，对新技术的投资将意味着“工业时代”能力的削减。
