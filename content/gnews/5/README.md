## 中共超限战灭港行动（廿八）雨伞运动 之 928催泪弹驱散行动
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1304855/)

#### 搜集/撰文：岁月如歌 / 封面合成：文粤


![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/07053356/6071.jpg)
9月28日下午5点半过警方发表新闻公告，将会使用更高的武力。随后警方开始提升武力，从特别“战术小队”的背包式、更大计量的胡椒喷雾，到催泪弹，实施大规模的驱散行动。 5时58分，防暴警察在夏悫道近添美道举起「警告催泪弹」黑旗后不到十秒射出4枚催泪弹，导致数十名示威者受伤，惨叫声不断，志愿者急救站也被警方催泪弹击中和摧毁，受伤者和志愿者急转入海富中心内进行急救。

示威者无法抵挡催泪弹发出的气体，被迫退到添马公园，200多人因吸入催泪弹气体感到身体不适，倒在地上休息，最后26名人士需要送院。身处几百米之外的市民戴着眼罩都感到眼睛刺痛，不停的流出泪水，十分难受。警察射催泪弹惹众怒，待催泪弹气体威力散去后数百名示威者走到太古广场对出的金钟道，占据东行三条行车线，也有大批群众带烟雾散去后再次返回原位，更多的市民奔赴现场支援，添美道方面，所有人即拿起毛巾掩盖口鼻保护避免吸入过多气体，，大会主持在广播怒骂「警察可耻」。防暴警察见示威者重新聚集，于下午6:02再次举起黑旗，并于03分到05分时间内再度发射近10枚催泪弹。至此防暴警察在10分钟内发射了三轮催泪弹对付示威者，平均2分钟一次。有一名戴着口罩的男子因不惧催泪气体，双手举着雨伞不退避，被媒体称为“雨伞人”，成为雨伞革命的象征。

图片来自维基百科

下午6时50几分，一批防暴警察到来增援，他们全套防爆装备，在道中推进，举起两个标语包括「催泪弹」和「速离，否则开枪」，7点左右发射了一枚催泪弹，2分钟后接着有10多枚催泪弹驱散人群。香港有线新闻视频播出此画面。示威群众越来越多，超过10万人，他们被催泪弹气体逼得四散，纷纷跑到中环的德辅道，干诺道中和香港大会堂对出的地方，占据有关交通主干道，与警察展开“攻防战”。港铁收到命令要配合警方，所有线路不得停金钟站，并且用来承载警察增援，对付市民。警方曾一度举起橙色旗欲开抢，示威者指警方配备AR15半径自动步枪及橡胶子弹，但被问到都拒绝承认。据协助伤者的社工透露，警察晚上表示所有的留下的人都被当作「暴徒」，警察已配备实弹长枪。

图片来自维基百科

事态不断升级，晚上10点20分左右学联收到消息，指警方已经用橡胶子弹来驱散示威者，便紧急呼吁参与者撤离，「必须保持实力，择日再会」，并要求警方让示威者安全离开。同时重申若特首梁振英在凌晨12点前仍未回应罢课时提出的四大诉求就会发起无限期罢课。凌晨过后，警方再度发射催泪弹，大批示威者被迫从夏悫道、干诺道逃避至金钟湾仔方向。有示威者不适需要送院，但被警方故意阻碍救援。特首梁振英凌晨发表讲话，否认没有派出解放军，也没有警察开枪，呼吁市民不要相信谣言，和平散去。说什么「香港是我们的家，香港稳定发展有赖和平守法，尊重法律。我们不想香港乱，也不希望大家正常生活受到影响，政府希望和社会各界共同努力，尽快回复社会秩序」。好话说尽坏事做绝，就是不能兑现双普选，就是变着法子敷衍香港人。并坚定中共中央不会作出让步。事态已经发展到撕裂火力程度，港府依然是宁愿做中共听话的狗（特首梁振英是党的好地下党员不算狗，是爪牙）。不能、不敢、也不会为香港人（包括自己）做半点努力争取。

图片来自 苹果日报

多家海外媒体引用中共外宣喉舌『博讯』的消息透露，三低特首（低票、低民望、低凝聚力）梁振英和中联办主任张晓明、国务院副总理刘延东、警务处处长曾伟雄、中央港澳办工作小组组长张德江，五人于28日晚上10点向中共报告，建议用武力镇压清场。被习近平否决，才有了梁振英凌晨发表谈话。说真假难辨，不过大家都知道博讯是什么背景什么职能。梁振英等五人绝不是什么好鸟，习近平就更不用说了，全世界都尝到了他独裁残暴的后果。绝不是博讯说的习近平生气怒批他们，说香港事情香港人民自己做主的角色。维基百科关于这方面的链接也失效，如果有过查维基百科资料的经验的人都知道，非常多的关键的档案或资料，总会被适当地“链接失效”，或者相关内容消失，这是中共的流氓常态，对其不利的就让它消失。在墙内最常见的就是无处不在的摄像头，在关键时候坏了，不见了或者刚好没电。
