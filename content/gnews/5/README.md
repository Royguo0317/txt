## 疫苗是福是祸？ 韩国当地感染暴发 驻韩美军大规模接种疫苗
` 首尔喜韩农场` [轉載自GNews](https://gnews.org/zh-hans/698572/)

**新闻翻译：喜马拉雅**
![]()![](https://gnews.org/wp-content/uploads/2020/12/Screen-Shot-2020-12-30-at-3.55.50-PM.png)
据驻韩美军(USFK)称，五角大楼对驻外美军的首次大规模疫苗接种工作**集中在目前驻韩部队**。这里的驻兵每天都盯着有核武器的朝鲜。

据法新社报道，USFK的一份官方声明称，对在该国医疗单位工作的军方和文职医护人员、急救队伍和指挥人员进行了初始剂量的莫德纳Moderna疫苗接种，**而此时此刻，东道国韩国正在经历第三波毁灭性的冠状病毒感染。**

USFK的医疗卫生专业人员和急救队伍今天接受了COVID-19疫苗第一轮接种。我们的工作人员将继续戴口罩，保持物理距离，洗手，遵循公共卫生指南。

**目前约28.500名驻韩美军**仍采取自愿接种疫苗。美军司令罗伯特-艾布拉姆斯Robert Abrams在一份声明中说：“我希望你能为你和你的家人做出关于疫苗的知情决定”。

艾布拉姆斯将军早前被拍到亲自接受疫苗，鼓励所有人员效仿。据报道，根据FDA一周半前发出的紧急授权，美国在海外的四个地点正在进行莫德纳疫苗的接种。

同时，韩国周二公布了单日死亡人数创下纪录。

**韩国周二公布因COVID-19而死亡的人数达到创纪录的40人**，每天感染冠状病毒的病例再次上升到1000多例，政府采取行动采购更多的疫苗。

这是自该国1月底报告首例冠状病毒以来的单日死亡最高纪录–是在第三波顽固的感染潮中出现的。迫使当局收紧社会隔离措施，韩联社将其称为 **“凸显了医疗系统的危机**”。

文在寅总统和Moderna首席执行官Stephane Bancel周一在视频通话中答应为2000万韩国人采购足够的疫苗剂量，预计于2021年第二季度交付。

加上另外的疫苗采购协议，韩国预计最终将有足够的疫苗覆盖5000多万韩国人。

根据韩国疾病预防控制机构（KDCA）的数据，周二该国又公布了1046个病例，使确诊感染人数达到58,725人。
