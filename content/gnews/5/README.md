- [**Gnews：华为波兰前销售总监因被指控从事间谍活动而受审**](https://gnews.org/zh-hans/1289891/)


美联社6月1日报道：华为公司在波兰的前销售总监和一名波兰网络安全专家在波兰开庭受审，罪名是从事间谍活动，但是两人均不认罪。

一名检察官要求秘密进行审判，因为一些证据具有保密性质。辩护律师提出反对意见，称指控的性质要求程序透明。王某和杜尔巴伊洛都用流利的波兰语说，他们希望进行公开审判。但在短暂的休庭后，三位法官组成的合议庭以国家利益为由宣布诉讼程序将以闭门方式进行，记者被告知离开。

王某和杜尔巴伊洛于2019年1月被波兰当局逮捕，并被指控在为中国技术公司华为寻求商业交易的掩护下为中国从事间谍活动。王某自被捕以来一直被拘留。杜尔巴伊洛曾是包括国内安全局在内的政府机构的网络安全专家，在被拘留6个月后被保释出来。两人都面临参加外国情报行动和威胁波兰利益的刑事指控。如果罪名成立，他们将面临至少三年的监禁。

据《Gazeta Wyborcza日报》报道，起诉书有部分内容是保密的，但其公开部分指控王某是一名中国情报人员，从2011年至2019年为华为公司寻求高级合同，使该中国公司能够深入了解并影响波兰的国家和地方政府数据系统。

审判预计将持续数月，目前还没有最终判决的日期。与此同时，波兰已经签署了一项由美国主导的 “清洁网络 “协议，旨在排除中共的硬件供应商。



- [**Gnews：欧洲反疫苗科学家述说疫苗真相**](https://gnews.org/zh-hans/1289837/)


欧洲药品管理局EMA于5月25日在其网站上发布了一封信函，关于该局在最近几个月里授权在我国进行的所有疫苗。一份六页的文件，Ema用它来回应 “Doctors for Covid ethics”协会在去年4月1日和20日的两封信中进行的严厉指责。批评直接针对该机构的主任Emer Cooke，但也针对欧盟委员会主席Ursula von der Leyen，以及欧洲理事会主席Charles Michel。

这些提出批评的医生来自世界各地的一百多名科学家和大学教授，从美国到英国，从德国到法国到西班牙。其中许多人是流行病学家、传染病专家、微生物学家、化学家和生物学家。他们已经在2月28日写了第一封信给欧洲药品管理局EMA。得到答复的医生们认为，所得到的保证并不令人满意。于是他们又回到了攻击EMA中。在4月20日的最新信件中，他们写道：”我们对基于基因的疫苗的担忧不仅限于阿斯利康的，还有辉瑞、摩德纳和强生公司。我们最担心的是，你给人的印象是脑静脉窦血栓形成（CVST）是一个非常罕见的不良事件。事实上，情况可能恰恰相反。CVST的主要症状在不良反应列表中占主导地位。” 以下是不良列表的内容：”剧烈的头痛，恶心和呕吐，意识改变，语言改变，视力改变，听力问题，不同部位不同程度的瘫痪，以及运动控制的丧失”。

反疫苗的医生继续说：“我们相信，你们公开承认的因基因疫苗而死亡的人数，只是这些疫苗实际死亡人数的一小部分”。所有这些指责都围绕着一个关键点，即迄今为止批准的疫苗，按照这些医生的说法，应被视为 “实验性药物”，因此应予以处理。事实上，在4月1日的信中，反疫苗医生写道：”我们的担忧来自多个证据来源，包括SARS-CoV-2的’尖峰蛋白’不是一个被动的对接蛋白，但它的产生可能通过多种机制启动血液凝固。不幸的是，你在3月23日的答复没有说服力，也不能接受。这种对疫苗安全不屑一顾的做法给人留下了非常糟糕的印象，即欧洲药品管理局Ema是在为制药公司的利益服务，而实际上EMA的职责应该是评估这些公司的产品”。
