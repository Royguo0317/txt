2020年11月6日美国国务院发文祝贺新西兰总理雅辛达·阿登（Jacinda Ardern）和工党在21天前（10月17日）的全国议会选举中获胜。文章赞扬新西兰人民展示了参与性民主的力量。 文章称，新西兰是我们最亲密的朋友之一。 我们祝贺新任外交大臣纳尼亚·马哈塔（Nanaia Mahuta），并期待与她和新西兰政府其他部门合作，以我们共同的历史和民主价值观为基础，进一步加深我们持久的伙伴关系。

第40任新西兰总理工党亲共党魁雅辛达·阿登的连任，在大选日取得了压倒性胜利。

当时前任副总理、外交部长反共领袖温斯顿·彼得斯（Winston Peters）在35年间第一次失去议会所有席位，他已卸任所有职务，并告别他的政治生涯。
