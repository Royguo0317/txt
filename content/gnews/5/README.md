二、美韩或将废除《韩美导弹指南》

法新社5月21日报道[\[2\]](https://www.rfi.fr/cn/%E5%9B%BD%E9%99%85/20210521-%E6%96%87%E5%9C%A8%E5%AF%85%E4%B8%8E%E6%8B%9C%E7%99%BB%E6%88%96%E8%B0%88%E5%BA%9F%E9%99%A4-%E9%9F%A9%E7%BE%8E%E5%AF%BC%E5%BC%B9%E6%8C%87%E5%8D%97-%E6%9D%BE%E7%BB%91%E9%9F%A9%E5%9B%BD)：韩国总统文在寅定于当地时间今天5月21日在白宫与美国总统拜登会面。文在寅与拜登或将讨论废除《韩美导弹指南》。《韩美导弹指南》距今已有42年。当时韩方为获得导弹技术，决定在美方的控制下，引进导弹技术，如今该指南反而成为限制韩国军事发展的枷锁。文在寅就任总统以来，如何确保导弹主权成为重要的施政课题。目前韩国研制的导弹射程被该指南限定在800公里之内。

三、美将举行横跨17时区大型军演威慑中共国

Military新闻网5月21日报道[\[3\]](https://www.military.com/daily-news/2021/05/21/biggest-navy-exercise-generation-will-include-25000-personnel-across-17-time-zones.html)：美国将举行有史以来最大型的海上军事演习，数以万计的水手和海军陆战队员将参与，并测试这些军种在准备与中共国或俄罗斯发生冲突时的远距离协同作战实力。航空母舰，潜艇，飞机，无人船和约25,000名人员都将参加有史以来最大规模的海上军事演习，此次军演将跨越17个时区，美国，非洲，欧洲，以及太平洋地区的水手和海军陆战队将加入其中。

路德社评论：

1、美国军演与中共军演有天壤之别，美国军演完全是实弹射击，太空卫星、空军、无人机、航母舰队等所有作战系统在此次军演中都要进行测试，单美国一家就能举行跨越17个时区的军演。
