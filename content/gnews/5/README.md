魯迪·朱利安尼（Rudy Giuliani）正在揭開近期迅速擴散的亨特·拜登醜聞的真相，他週二晚告訴Newsmax電視台的格雷格·凱利（Greg Kelly），“本週我會見了特拉華州警方，原因是在據稱亨特·拜登的硬盤裡發現了令人震驚的材料。”

朱利安尼說，他的發現遠遠超出了他指控前副總統喬·拜登所從事的向外國勢力的權力尋租，以及所謂的回扣。

“這是一個非常、非常敏感的問題，”朱利安尼告訴格雷格·凱利，並聲明這件事發生在亨特·拜登和他的父親喬·拜登之間的一次短信對話中。

“裡面有令人厭惡的性侵和墮落。”他說。

朱利安尼稱，短信和照片涉及一名不知名的14歲女孩。

這位前紐約市市長向Newsmax電視台提供了一張帶有短信的照片。他說，這條短信是亨特·拜登出示給他父親的信息，談論了一些他在一名年輕女性身邊不當的行為。

亨特·拜登短信裡說：“她（指亨特·拜登的嫂子，且是亨特·拜登的情人）跟我的心理醫生說我性行為不當，因為我跟某某（14歲女孩）Facetime通話時不穿衣服。她說她不想看見我，是因為我會一邊吸毒，一邊跟女孩Facetime通話……說我給孩子造成了一個很不安全的環境。”

朱利安尼還聲稱，亨特·拜登的硬盤裡有許多未成年女孩的照片。

朱利安尼說，他和紐約前警察局長伯納德·克里克在星期一會見了特拉華州警察局，並將證據交給了他們。

Newsmax沒有評論據稱是亨特·拜登的硬盤、筆記本電腦或相關材料，Newsmax沒有對證據的真實性或指控的準確性作出任何結論。

朱利安尼說：“我們對這些未成年女孩得不到保護感到非常不安。

“如果你看了這個硬盤上的照片，卻沒有向警方報案……如果你是一名執法人員，你就會被判重罪。

“如果你只是一個普通公民，你將是一個可怕的人。”這位前美國紐約南區檢察官說。

“這裡有很多女孩的照片，我的意思是，她們不應該出現在那裡。我可以告訴你。如果特拉華州警方對此不採取任何行動，那將是一件非常可悲的事情。”

這位川普總統的私人律師稱，亨特·拜登筆記本電腦曝光事件“可能是我見過的最嚴重的醜聞”。

他補充道，“裡面的內容是一種駭人聽聞的腐敗，這就是我為什麼向特拉華州警方報案的原因。”

他聲稱，喬·拜登更有興趣利用他的兒子亨特·拜登作“收銀袋”，為他向外國勢力兜售影響力和接受賄賂，這些興趣都超越了他保護自己兒子和其他人的職責。

“這封信是寄給副總統的，”朱利安尼指著一份打印出來的關於與這位14歲孩子來往的文本說：“他都做了什麼？”

他說，主流媒體一直迴避這一報導，因為這將對喬·拜登的競選構成致命打擊。

“對這件事有一道鐵幕式的審查制度……因為這對作為候選人的喬·拜登來說是致命的。作為一個人，這對他來說應該也是致命的。我的意思不是說消滅他的生命，但他應該被起訴。”
