日经新闻报道，就在欧盟(European Union)努力阐明其对印度洋-太平洋地区的首个正式战略之际，日本最高外交官开始了对中欧和东欧的访问，以争取对中共海上扩张主义采取更强硬路线的支持。

日本外相茂木敏充星期五会见了斯洛维尼亚外相安泽·洛加尔。洛加尔在随后举行的联合新闻发布会上敦促与日本加强双边合作，并强调在印度洋-太平洋地区必须尊重国际法。

茂木敏充表示，日本将与斯洛维尼亚就欧盟的印度洋-太平洋战略“密切合作”。

波斯尼亚-黑塞哥维那是他的下一站。他还将于本周四前往波兰，与维谢格拉德集团(Visegrad Group, V4)的领导人会晤。维谢格拉德集团是欧洲四国集团，还包括捷克共和国、斯洛伐克和匈牙利。

茂木敏充此行正值欧盟希望最早于9月敲定印度洋-太平洋地区的详细战略之际。欧盟成员国在欧盟的经济和外交决策中拥有发言权，这意味着这些作为中共“一带一路”基础设施建设倡议的参与者的中东欧国家，可能会影响最终战略。

茂木敏充还将讨论日本推动“自由开放的印太”，并强调遵守国际法和维护航行自由的重要性。作为访问的一部分，茂木敏充将参加星期一开始在伦敦举行的七国集团外长会议。他将概述他在中东欧地区的外交努力，并敦促欧盟作为一个集团制定对华战略。

日本和美国一样，对中共的海洋扩张主义采取了坚定的立场。日本政府正密切关注欧盟是否会效仿美国的做法，以及欧盟如何解决与中共相关的经济和安全问题。

例如，中共和欧盟在地理上相距遥远，这意味着中共不会像对日本那样构成直接威胁。有人担心欧盟可能会对中共采取比日本和美国更温和的立场，尤其是考虑到中共国的经济影响力。

为了欧盟自身的经济安全，美国最近一直在敦促欧盟与中共国保持距离，香港和新疆地区的人权问题也受到了国际社会的抵制。但欧盟内部的许多国家无法忽视它们与中共之间的经济联系，在与中共对抗方面犹豫不决。

与此同时，中共多年来一直在争取中东欧国家的支持，这是中共推动“一带一路”进入欧洲的一部分。2008年国际金融危机后，中共向该地区提供了资金援助，并于2012年启动了与巴尔干半岛国家和中东欧国家的“16+1合作框架”。该框架后来扩展为“17+1”。

其中一些国家已经对中共介入该地区感到失望。黑山共和国(Montenegro)一直难以向中共偿还一项公路建设项目的资金，这是“债务陷阱”风险令中共的倡议陷入困境的一个例子。

今年2月，中共主席习近平出席了17+1峰会，但波罗的海三国领导人没有出席。波兰也对对华贸易不平衡表示担忧。

日本将这些进展视为一个潜在的契机，试图争取欧盟一起“从印度洋-太平洋地区到西巴尔干半岛和非洲，建立可持续、全面和基于规则的互联互通。”日本外相在这个时候访问这些国家，意义重大。日本在争取欧盟对中共国海洋扩张主义采取更强硬立场的支持的同时，日本开始开启东亚领导人的角色的努力。
