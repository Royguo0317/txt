## 技术贴:如何组建太阳能发电系统（一）

最近的停电火了，作为半个行家，我想给大家介绍一下我的用电经验，为各位提供参考。一般家用电器功率最大的就是热水壶，电磁炉，基本都在2000瓦左右，照明也就几十瓦，电视，电脑300瓦左右，笔记本150瓦左右。这些都是通用值，了解了这些之后，接下来就给大家介绍常用发电系统了。大家一般想到的就是发电机，但是战时发电机的燃料供应是无法保证的，考虑到战时基本提供小功率电子设备的充电，所以重点推荐太阳能发电系统，好处是投资少，方便好用，不被限制！

太阳能系统，为什么叫系统呢？因为我们搭建的设备多，所以就叫做系统了！主要设备有，电池板，充电控制器，电池组，逆变器，电缆组成。这里的设备型号很多，但基本就是12/24/48/64伏电压的，所以你的选择一定要电压匹配，不然你是做不了这套系统的，连上了也没用，用错了有可能会出现问题，甚至损坏你的设备！所以很重要！

首先教你如何选择逆变器：
如果你用电量不大，那么就选择功率小，投资少的，比如说你就充手机，笔记本电脑，也就150瓦以内的，那么就可以选择1000瓦修正正弦波逆变器，以及与其电压匹配的电池组。为啥要大那么多呢？因为你知道，国产的东西就喜欢吹牛，虚标很严重，国内良心价格也就是150左右！但总功率不可以超过它标称的一半！如果不差钱，买个好的，那就买正弦波的，基本价格差不多翻十倍，那可是真的，标多少就多少！同样1000瓦功率的正弦波逆变器价格在一千以上。

重点介绍电池：
假如你搭建的是12伏系统，那就方便多了，因为电池的选择范围更广，汽车电池，电动两轮车的，大汽车的电池基本都是12伏的，所以可利用的电池很多！唯一要注意的就是容量！单块电池那不存在问题，并联起来的时候一定要容量相同，电压相同，不然后果很严重！虽然没有锂电池那么危险，但也很吓人！所以四块，五块电池的，最好在同一种设备上取下来用，这样相对来说好一点，如果没有专业知识，坚决不要把汽车电瓶和小电瓶混用！！！汽车电瓶单独使用就可以了！如果不差钱，可以上锂电池，容量大，重量轻，买的时候一定咨询卖家，支持太阳能充电不，不然容易出问题！我用的是锂电池，12伏系统，200安时的，负荷不大，也就150瓦左右最大300瓦负荷，全年不用管，天天有电！字数关系，就介绍到这里！
