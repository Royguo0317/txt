## 为了加强思想控制，北京等地严查校外培训机构复课，未打疫苗老师或面临被辞退
` 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hans/970274/)

**文章内容**：

近日，北京有关部门加强了对校外培训机构的监管力度，培训机构被全面排查。据了解，出现这种情况的原因与教育界代表人士呼吁“严查校外培训机构”“彻底取缔校外培训机构”直接有关。

**两会期间，多位人大代表表示教育行业需规范严查**

据报导，3月8日，中共全国政协委员、上海市教育委员会副主任倪闽景，北京师范大学教授、创新发展研究院院长关成华等委员出席新京报两会直播节目，对近日大众关注的“彻底取缔校外培训机构”等话题进行了深入谈论。

全国人大代表、华中师范大学教授周洪宇表示：针对眼下大量在线培训平台“野蛮生长”的问题，建议加强对在线教育平台以及各种教育APP的监管，规范校外在线线下的培训，对诸如超标超前、应试导向、虚假宣传、制造焦虑等违法违规行为予以严肃查处，同时建立在线教育机构从其从业人员负面清单。

**沪、京两地早已开展动作，一刀切举施引发不满**

早在1月23日，北京市教委要求全市暂停一切线下培训和集体活动。

近日，北京多区出台红头文件，要求对校外培训机构进行排查，并且暂停培训机构线下上课。

3月5日，海淀区教委发布档，要求对各类校外培训机构进行联合检查。

3月8日，昌平区校外培训机构专职小组通报，开展对学科类/语言类/中考高复读培训机构暂停培训实地检查。

据了解，检查部门要求，注射CCP病毒疫苗作为上岗必要条件，未打疫苗者包括不愿打，身体条件不允许打疫苗的老师，或面临被辞退。根据网上统计CCP病毒疫苗有数十种副作用之多，尤其是中共自产疫苗副作用更达到73种之多，被外界喻为最不安全的疫苗，中共享强制的手段让民众当小白鼠，疫苗经济、疫苗政治，疫苗外交，其恶罄竹难书。

这种情况，不只发生在北京，在上海经营私人学校的张女士对自由亚洲电台表示，近期，北京，上海乃至全国各地，几乎所有的校外培训机构被勒令停业。

**为安抚舆论，封杀报导，教委紧急辟谣，欲盖弥彰**

自北京等地出现有关暂停校外培训机构的措施和新闻报导后，遭到舆论哗然，相关文章也被封杀，为了安抚舆论，平息众多家长投诉，3月12号，北京市教委紧急出来辟谣，欲盖弥彰。

**措施背后，意在掩饰中共正加强对学生的思想控制，排除西方思想价值观对学生影响**

中国问题研究学者薛驰表示：大陆培训行业问题很多，但像这样一刀切的关停排查，比较罕见，中共借着整顿名义，很可能想借此达到对学生洗脑的完全化。中共从2020年已经暂停了中国大陆考点开放举行美国历史、世界历史、欧洲历史、人文地理等四门课目的考试，要求对小学教材进行排查，禁止用境外教材和学校自己研发的课程。可见，教育作为中共的思想战略阵地，中共的洗脑开始从娃娃抓起，并且不遗余力的消除西方自由思想和价值观对学生的影响。
