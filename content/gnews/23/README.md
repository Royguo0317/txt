## 史蒂夫·班農的庭審將於2021年5月開始
`2020-09-02 15:41 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/330702/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/02153611/1599075355640.jpg)圖片來自：cnn。 com 
根據美國CNBC記者丹·曼甘（Dan Mangan）的報導， 聯邦法官阿娜麗莎·托雷斯（Analisa Torres）星期一宣布，曾擔任唐納德·川普總統（Donald Trump）的前最高助手史蒂夫·班農（Steve Bannon）和其他三名被告，將於2021年5月24日開庭受審。他們被指控的罪名是，騙取用來修建南部邊境牆的數十萬美元私人捐款。

報導稱，曾經是川普首席戰略家的班農和三名同事蒂莫西·謝伊(Timothy Shea)，布萊恩·科爾法奇( Brian Kolfage)和安德魯·巴多拉托(Andrew Badolato)被指控欺騙捐助者，因為他們籌集了“超過2500萬美元，用於在美國南部邊界修建隔離牆”。週一，托雷斯讓科爾法奇以500,000美元交保。謝伊和巴多拉託的交保金額分別都是25萬美元。

報導指出，班農於8月20日被捕，但他並沒有認罪，最後以500萬美元交保獲釋，並於本週四以175萬美元的現金或財產作抵押。當時， 班農正在康涅狄格州（Connecticut）沿海的一艘150英尺的遊艇上。該遊艇由中國億萬富豪郭文貴所擁有，他是班農集團的合夥人。在他們兩週前被起訴後，代理美國檢察官奧黛麗·斯特勞斯（Audrey Strauss）在一份聲明中說：“被告利用虛假藉口，欺詐了幾十萬名捐助者。他們為自己的利益，以修建邊境牆為名籌集了上千萬美元，申稱所有這些錢都將花在建牆上。”

聯邦檢察官說，雖然被告一再向捐助者保證，“我們建牆”（We Build the Wall）計劃的發起人科爾法奇“將不會獲得一分錢，但被告秘密地將數十萬美元轉給了科爾法奇，而這筆錢被他用來過奢侈的生活。 ”聯邦檢察官還說，班農從“我們建牆”項目獲得了100萬美元的資金，為了挪用這筆錢，他創建了一個獨立的非營利組織，其目的是“促進經濟民族主義和美國主權”。所有四名男子均被控以一項共謀電信欺詐罪和一項共謀洗錢罪。

報導指出，37歲的三重截肢者科爾法奇是在2018年底在GoFundMe網站上發起了眾籌活動以建造邊境牆。起訴書稱，科爾法奇“反复虛假地向公眾保證，他不會“拿一分錢的薪水”。檢察官說，但他實際上從該集團所獲資金超過了35萬美元。根據起訴書，班農曾在一份公開聲明中說“我們是一個志願者組織”。班農被捕後對記者說：“整件事情都是為了阻止想要修建邊境牆的大眾。 ”
