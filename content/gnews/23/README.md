**写，不需要理由**

写诗

哪怕你是那条大雨里的流浪狗

哪怕你是那个被凌辱多年的妓女

哪怕人们早无话可说

哪怕这世道黑暗得要割掉你的喉咙

哪怕洪水猛兽就在身边

哪怕天塌了下来

写吧，不要什么理由

也不要用任何字

就用那颗依旧滚烫的心

**辩解**

并非不善言辞

只是不愿辩解

秋天的雨有点冷

而他们的话却从未温暖过

想想天上的乌云和天下的乌鸦

他们是同样的黑吧

远处，泥泞里的公交车满载摇晃的人群

正驶向同一片乌云

**伞**

它遮不住人间的烟火

它让人猜你另一半的样子

它隐藏真实，尤其是在阳光下

它遮蔽痛苦，特别是在大雨里

人们匆匆而过，比头顶的乌云还忙

而在某处，它是盾牌，阻挡着现实的子弹

无论你是否愿意，总会有一双勇敢的手伸过来

和你站在一起，把头顶那片天举得更稳
