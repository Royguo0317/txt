## 彭斯：欢迎参众议员6日提反对意见和呈交证据

![]()![](https://gnews.org/wp-content/uploads/2021/01/1.5-2.png)图为2020年10月8日，美国内华达州博尔德城（Boulder City），副总统彭斯在竞选集会上发表讲话。(RONDA CHURCHILL/AFP via Getty Images)

美国副总统麦克‧彭斯（Mike Pence）周六（1月2日）表示，他欢迎参、众两院议员在[1月6日](https://www.epochtimes.com/gb/tag/1%E6%9C%886%E6%97%A5.html)提出反对意见以及呈交证据。

副总统办公室主任马克‧肖特（Marc Short）周六对媒体发出声明称：“副总统彭斯与数百万美国人一样，对这次选举中的选民舞弊和违规行为感到担忧。”

“副总统欢迎参、众两院议员利用法律赋予的权力，在[1月6日](https://www.epochtimes.com/gb/tag/1%E6%9C%886%E6%97%A5.html)向国会和美国人民提出反对意见和呈交证据。”声明说。

美国国会11名共和党参议员周六联合发表声明，要求国会遵从先例成立选举委员会、对存在的争议性选举结果进行为期10天的紧急审计。

这11位参议员们表示，国会[1月6日](https://www.epochtimes.com/gb/tag/1%e6%9c%886%e6%97%a5.html)的联席会议投票是仅剩下的唯一宪法权力，能考虑和解决对严重选民舞弊的多项指控。他们将反对[选举人团](https://www.epochtimes.com/gb/tag/%e9%81%b8%e8%88%89%e4%ba%ba%e5%9c%98.html)投票，直到审计结束。

共和党参议员们认为，2020年选举中的舞弊和违规行为指控都超过了他们有生之年以来的任何一次，而且美国人普遍存在这种担忧。

根据路透社/益普索（Reuters/lpsos）的民调显示，39%的美国人相信“选举被操纵”。持这种看法的有共和党人（67%）、民主党人（17%）和独立人士（31%）。

而原本以为法院会听取证据并解决这些严重选举舞弊的指控；但最高法院却两次拒绝接收案件。

要求国会设立独立选举委员会并非共和党首创。

最早的先例出现在1877年，海斯对蒂尔登（Haye-Tilden）总统选举中出现了严重的舞弊和非法行为的指控。当时，有三个州——佛罗里达州、路易斯安那州和南卡罗来纳州的选举被指控存在非法行为。

国会和媒体都相当关注这些指控，并任命了一个选举委员会——由五名参议员、五名众议员和五名最高院大法官组成——来审议和解决有争议的问题。

而民主党国会议员也曾于1969年、2001年、2005年和2017年在国会挑战总统选举结果。而且，在1969年和2005年，都有一位民主党参议员加入民主党众议员，在两院要求投票，决定要不要接受被挑战的总统选举人票。

到截稿时，国会已经有12名共和党参议员表示，将在1月6日挑战选举结果；此外，还有至少四十多位共和党众议员表示，将加入1月6日的挑战。

根据国会流程，如果各自一名参议员和众议员在6日的[国会联席会议](https://www.epochtimes.com/gb/tag/%e5%9c%8b%e6%9c%83%e8%81%af%e5%b8%ad%e6%9c%83%e8%ad%b0.html)期间对关键州的选举人票提出反对意见，将引发每个争议州分别两小时的辩论。

随后两院议员撤回到各自议院并针对反对意见进行辩论和表决。如果反对意见最终获得两院多数票，则成立，被挑战州的[选举人团](https://www.epochtimes.com/gb/tag/%E9%80%89%E4%B8%BE%E4%BA%BA%E5%9B%A2.html)票将作废。

根据《宪法》和法律，副总统主持6日的国会联席会议，但对有争议的[选举人团](https://www.epochtimes.com/gb/tag/%E9%80%89%E4%B8%BE%E4%BA%BA%E5%9B%A2.html)票有决策权的仍是国会两院的多数派。

白宫备忘录的起草者认为，联邦选举认证由担任参议院议长的副总统迈克·彭斯（Mike Pence）左右。 此外，彭斯拥有决定是否拒绝问题州选票的唯一权力。1月6日副总统彭斯在国会联席会议中的举动是比较重要的一环，但相信川普团队会有不同的预案和应对。

1月3日，爆料革命路德社透露彭斯副总统有重要的视频被中共所要挟，同时彭斯副总统非常注重个人声誉和多年来维护的保守形象，这也印证了彭斯总统自从哈德逊反共演讲之后对中共罕有批判的声音和举动。

希望彭斯能够放下“小我”，在这个历史的关头，站在正义的一方成就“大我”。
