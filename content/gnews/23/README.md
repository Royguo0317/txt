**中共早前趁着美国总统大宣政局动荡不稳，无暇顾及他国事务，乘机再对香港施加辣手，以【港版国安法】颠覆国家政权罪名，滥捕53****名民主派初选的参选人及组织者。路透社今日（12****日）引述北京消息人士指，抓捕行动在中共指使下进行。中共谋求一劳永逸解决民主派问题，并修改立法会选举制度，香港民主选举可能成为绝响！**

**中共谋求一劳永逸解决民主派**

【大纪元香港】引述路透社的报道称，有两名不具名的中共官员，自称对香港事务有深厚认识，并且代表中共利益。其中一名中共官员证实，近日民主派初选大抓捕行动，是中共消灭民主派声音的一连串行动之一，旨在「确保香港不会重返18个月前的社会运动境况」，又指行动陆续有来，预料未来至少一年将会出现更多强硬措施，宣称「中共已忍耐很久了，需要一劳永逸解决问题」。

**或有可能再次押后选举**

该名中共官员指出，即使已经前所未有地对民主派进行大规模抓捕，中共仍然忧虑他们在今年举行的立法会选举中取得过半数的议席，故此现正研究改变选举制度，声称为了彻底解决香港政治制度「缺陷」，或有可能再次押后选举，惟他亦称有关计划仍未落实。

**中共正研究改变特首选举委员会**

路透社亦引述另一中共消息指出，中共正研究改变特首选举委员会。现时，特首选委会由1,200名选委所组成，当中大部份受中共控制。消息形容，有关改动将会「动摇整个政治根基」。

特首选委会将于明年选出下一任特首，早前有传闻指中共「人大」或会在会议上决定，取消以民主派占大多数的区议员选委资格，不过最终未有出现。

特首办回应立场新闻查询时称，自去年强推「国安法」之后，「街头暴力」已被遏止，声称港人合法权利、自由将会得到保障。发言人又表示，立法会换届选举将会于今年9月5日举行，官员会做准备工作，确保将是一场公开、公平、诚实选举。

**战友观点：**

2020年9月，香港本来要举办立法会选举，后来因为疫情的原因，政府押后一年，也就是2021年9月举行。

今天传来消息可能2021年的9月可能也不会有立法会选举。香港真正的、透明的、公开的、民主的选举可能成为绝响！

2021年1月6日的「35+ 初选」大抓捕已经重创了民主派和泛民的骨干力量，甚至可以说是一网打尽。一旦罪名成了，绝对不能参与立法会选举。但北京政府依然担心，还是没有安全感。今天路透社的报道直接戳破中共意图： 近日民主派初选大抓捕行动，是中共消灭民主派声音的一连串行动之一，旨在「确保香港不会重返18个月前的社会运动境况」；中共官员又指「即使已经前所未有地对民主派进行大规模抓捕，中共仍然忧虑他们在今年举行的立法会选举中取得过半数的议席，故此现正研究改变选举制度。」

资深媒体人吴志森认为，也许有人会有疑问，民主派的人士都快被瓦解了，北京还怕什么？ 2021年1月6日的大抓捕使虽然民主派人士几乎被抓光，能参选立法会选举的民主派大减，本土派也不多，但是「一批倒下一批上来」正是北京所不愿看到的。

北京对上次的区议会选举结果是相当震惊的，原因是经过一年的返送中运动，无数的抗争者被抓捕，但香港市民不但没被吓倒，只要有投票的机会，都会义无反顾地投出宣誓他们政治立场的一票。

港人处事一向灵活变通，Plan A 不行就会有Plan B 和Plan C 。今天北京政府所担心的正是香港人那种打不死的抗争精神！中共担心的是民心，只要民心不死，港人总会想到更好的办法继续抗争！
