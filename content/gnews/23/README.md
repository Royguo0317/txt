## 《路德社》精选新闻简讯：澳驻中共国大使说中共的贸易行为是“斗气”

出品：《路德社》 编辑：《路德社》精选新闻专栏组
![]()![](https://www.gnews.org/wp-content/uploads/2021/03/123-7.png)
澳洲新闻网3月26日报导：经外交与贸易部官员证实，澳大利亚与中共国的贸易在几乎所有行业都是直线下降的，总体数字在很大程度上是中共国对铁矿石的强劲需求支撑的。在过去的九个月中，中共国政府瞄准了澳大利亚的几大产业，包括大麦，煤炭，木材和龙虾，以试图迫使堪培拉在一系列纠纷中立足。 澳大利亚驻北京大使已将中共国对澳大利亚的经济惩罚运动标记为“斗气”，因为两国之间的外交关系仍然陷于混乱。

评论：

中共国之所以要采取贸易战方式对付澳大利亚，就是想阻止澳大利亚继续要求中共国开放实验室接受调查，阻止澳洲与美国联合对付中共。但这并未阻止澳洲联合美国采取一系列的针对中共的行动，可见中共试图用市场为政治筹码来换取澳洲妥协的阴谋彻底失败。中共国因抵制澳洲产品也付出了巨大的代价，国内粮食价格飙升，多个省份因供电不足迫使工厂停工，经济生产停滞不前，这完全是中共自食其果。
