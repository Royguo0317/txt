**信和黄志祥帮政府谷针****拨出凯滙****1,080****万元单位作大抽奖****平面图曝光**

港府一直呼吁市民打肺炎疫苗，然而市民反应不佳。信和置业（083）与黄廷方慈善基金有限公司及华人置业（127）联手推出“打疫苗送楼”，得奖者可以抽奖获得观塘凯滙一个449方呎的一房住宅单位。

信和指，持有有效的香港永久居民身分证香港居民，必须年满18岁并已于香港完成接种两剂新冠疫苗，较早前已在香港接种了新冠疫苗的居民也可参加。大抽奖参加者须于2021年9月1日或之前登记，即有机会参与抽奖赢取终极大奖，即观塘凯滙一个449方呎一房住宅单位，市值1,080万元。

黄廷方慈善基金董事黄永光指，希望透过推出大抽奖活动，合力提高香港的疫苗接种率。

翻查资料，中央驻港官方机构中联办2019年初斥资约2.47亿元、以“一约多伙”形式连购凯滙20个单位，并豁免超过7,410万元辣税款项。意味今次获奖人士，可以和中联办职员做邻居。
