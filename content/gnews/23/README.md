2021.1.16（早间）路德时评 路安墨唐谈

**一、习近平依然没有露脸**

从路德社爆料习近平头部手术之后到现在已经超过16天没有现身，到底是习的身体状况不允许，还是说习担心美国的行动，又或是他对自己身边的人不信任不敢出来。最近路德社一直连续关注习的动向，反而很少讨论川普总统的行踪。原因是美国对中共的行动已经排山倒海地过来，而习近平持续的隐身，说明了中共内部已经意识到自己的危险，各大家族在想办法怎么自保。

**二、美国国务院再发重磅病毒溯源声明**

15号美国国务院针对武汉病毒研究所发布了病毒溯源的声明，WHO日前去到中国调查新冠病毒，但其行程并不包括武汉病毒研究所。美国发布这份声明目的也就是直指武汉病毒研究所，意思是武汉病毒研究所才是真正该关注和调查的。声明中提到一直以来中共阻止对新冠病毒起源的透明和彻底的调查，而选择投入大量资源进行欺骗和虚假宣传，美国将竭尽全力支持可信和彻底的调查。

**三、蓬佩奥国务卿连续发推关于新冠病毒和中共的责任**

近日美国国务卿蓬佩奥多次在媒体以及演讲中提到病毒的溯源和追究中共责任的态度。并批判中共在新冠病毒中的角色及对全世界造成的大灾难。而且最新蓬佩奥国务卿的推特中表示：“中国共产党和中国人民不同，并支持中国人民摆脱中共专制统治的愿望”。说明美国政府内部已经彻底掌握了充分的证据证明新冠病毒的来源和中共故意隐瞒疫情真相的邪恶。接下来美国的行动会让中共在新冠病毒的作为付出应有的代价。

**四、美国再发布1000多家中共军方企业**

美国政府再发1108家企业的名单，并禁止美国公司与之有任何商业关系，实际上就是经济上的脱钩。随着美国对病毒掌握的真相和理解到中共对全世界的危害。川普政府在美国总统选举和权力交接的最后时刻，仍然对中共做出一系列制裁等强硬措施。而美国能够做出这些决定和变化，和爆料革命在背后做出的努力和贡献分不开。到最后美国一定会宣布中共制造生化武器病毒的真相，消灭中共指日可待。
