据《网关专家》（The Gateway Pundit）新闻网乔·霍夫特（Joe Hoft）2021年3月1日报道：

特约撰稿人：Lawrence Sellin博士。

**国际金融届和全球科学机构从不希望透露****COVID-19****病毒的真正来源，尽管事实上在大流行开始的早期就已有人质疑它的实验室起源。**

现在有证据表明，美国政府内部人员与美国科学界的特定成员互相配合，又在无疑是来自国际商业和媒体的加持下，保护中共国免于承担责任，并以此来保护自己的利益。

早在2020年1月13日，网络上已经有评论称，COVID-19病毒是在中共国的实验室制造的，美国科学家可能帮助中共国科学家获得了制造病毒的知识，确切的说是在武汉病毒研究所石正丽博士的实验室里，并在北卡罗来纳大学拉尔夫·巴里奇（Ralph Baric）博士的协助下制造的。

2020年1月26日，受人尊敬的调查记者和国防专家比尔·格茨（Bill Gertz）在《华盛顿时报》的文章中写道：

“在全球范围内传播的致命性动物传染的冠状病毒可能起源于武汉市的一个实验室，与中国秘密研究的生物武器计划有关”

2020年2月3日，共和党建制派人士、时任白宫科技政策办公室（OSTP）主任的凯尔文·德雷格梅尔（Kelvin Droegemeier）致信美国国家科学院院长玛西亚·麦克努特（Marcia McNutt）博士，要求：

“美国国家科学、工程和医学科学院(NASEM)应迅速审查信息并确定数据要求，以帮助确定2019-nCoV的起源，特别是从进化/结构生物学的角度。”

奇怪的是，他的担忧是由一篇来自印度的研究文章“2019-nCoV刺突蛋白中独特的插入物与HIV-1 gp120和Hag的相似性 ”引发的。该文晦涩难懂、未经同行评议且已被撤回。

现在回想起来，那篇文章似乎是一个科学领地里的稻草人，暗示着涉及政府的其它动机，但它们是什么仍然不清楚。

尽管如此，根据被征召来制定回应OSTP 要求的科学家名单，德雷格梅尔请求的调查结果已经注定，即完全免除中共国的责任。

这些科学家是克里斯蒂安·安德森（Kristian G.Andersen）（斯克里普斯研究所）、拉尔夫·巴里奇（Ralph Baric)（北卡罗莱纳大学公共卫生学院）、特雷弗·贝德福德（Trevor Bedford）（Fred Hutchinson癌症研究所）、阿文达·查克拉博蒂（Arvinda Chakraborty）（纽约大学医学院）、彼得·达萨克（Peter Daszak）（生态健康联盟）、吉吉·格隆瓦尔（Gigi K.Gronvall）（约翰-霍普金斯大学彭博公共卫生学院）、汤姆·英格斯比 (Tom Inglesby)（约翰-霍普金斯大学健康安全中心)和斯坦利·珀尔曼（Stanley Perlman）（爱荷华大学）。

国家科学院在给OSTP的信中引用了武汉病毒研究所的一篇科学文章，称该病的爆发 “始于当地的海鲜市场”，明确支持中共政府的论点，即COVID-19大流行是动物向人类的自然传播。

此外，美国国家科学、工程和医学科学院的三位院长建议美国继续与中共国，特别是中国科学院和武汉病毒研究所密切合作。

这些回应国家科学院咨询要求的科学家中大多数都有与中共国有长期合作的历史，和/或后来参与大力支持中共国和COVID-19起源于自然发生的理论。

在一些重要方面，国家科学院2020年2月6日获得的最终版本与2020年2月4日的原始草案几乎没有相似之处。(参见 “OSTP NAS电子邮件2 ”PDF)

根据国家科学院与顾问科学家的邮件往来，科学家们似乎明显在努力支持自然发生的理论，并消除或淡化任何关于COVID-19病毒是经过工程设计的、具有独特的与人类结合的特性、可能向更高的感染力突变、并拥有任何相关冠状病毒都不具备的福林酶切位点的提法。

随后，回应国家科学院咨询要求的科学家们甚至开始了自己的公关活动，支持中共国和COVID-19病毒是由动物自然传播给人类的理论。

2020年3月7日，彼得·达萨克（Peter Daszak）和斯坦利·珀尔曼（Stanley Perlman）作为作者联合在英国医学杂志《柳叶刀》上发表文章，称 “我们共同强烈谴责暗示COVID-19不是自然起源的阴谋论”。

2020年3月17日，克里斯蒂安·安德森（Kristian G.Andersen）作为资深作者出现在 “SARS-CoV-2的近端起源 ”一文中，该文称COVID-19病毒 “不是一种被刻意操纵的病毒”。

2020年9月21日，吉吉·格隆瓦尔（Gigi K.Gronvall）作为约翰-霍普金斯大学团队的成员之一，该团队写了一篇批评中国举报人闫丽梦博士的文章，她提出了COVID-19病毒来自实验室的证据。

众所周知，拉尔夫·巴里奇（Ralph Baric）还与武汉病毒研究所的 “蝙蝠女 ”石正丽长期合作，包括进行涉及高危险性的 “增强功能 ”的研究实验。

正如尼尔·帕特尔（Neil Patel）最近在《每日信号》（Daily Signal）中所描述的那样，世界卫生组织调查COVID-19病毒来源的团队中唯一的美国代表彼得·达萨克（Peter Daszak）也是石正丽的密友，是将美国纳税人资金投入武汉病毒研究所的关键性人物。

达萨克甚至组织了一场公关活动，在没有进行任何彻底调查之前，就把实验室泄漏假说描绘成 “阴谋”。他的发言人后来说，目标是保护实验室的科学家，但最终，也许是有意的，受益者是中共国、国际金融利益集团和全球科学机构。
