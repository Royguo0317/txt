## 香港買房，其實又是一個坑！

### **你以為這不關你事，但你早已在其中！**

*作者：玄天生*

*校閱：Julia Win*

**引言：在香港擁有一個屬於自己的居所、屬於自己的家，是每個人的夢想與所望，但在一個赤化的香港城市中，買房其實是進入另一個惡夢，或許萬劫不復也未可知。**

借用香港昨晚在觀塘業主大會新聞，「無視業主要求，宣佈流會，引曉麗街坊不滿」，來解說現今置業後的實情。([新聞鏈接](https://www.inmediahk.net/node/1073917))

置業對每個人來說是重要的，不單是能有個落腳地方，更重要的是能安居樂業。看似買了樓房，人生就已經安穩，但在一個邪魔中共和港共政府的管治下，人民買了樓房只是掉入更深的深淵而已。就像路德先生節目中說的，它們是非常差的物業管理，更是聲名狼藉的管理人，不但可以超支，還要納稅人買單；不單官商合作，更是吸血鬼、我們身處在末期的癌細胞的體制下。

中共80年代開始已經佈局白手套，進軍香港地產業。香港地產商開始佔據各地皮，再配合它操控的港共政府，配合地產商家的利益交換，地皮高價賣出，地產商開發，利益再分出給建築商。導致樓價一年比一年高，近幾年，紅色地產商也相繼進場，瓜分市場，最終成為世界上最高樓價的城市。(補充，[中聯辦在港擁逾 700 物業，兩成傘運(2003年)後購入](https://www.thestandnews.com/politics/%E7%9C%BE%E5%BF%97%E8%AA%BF%E6%9F%A5-%E4%B8%AD%E8%81%AF%E8%BE%A6%E5%9C%A8%E6%B8%AF%E6%93%81%E9%80%BE-700-%E7%89%A9%E6%A5%AD-%E5%85%A9%E6%88%90%E5%82%98%E9%81%8B%E5%BE%8C%E8%B3%BC%E5%85%A5-%E9%BB%83%E4%B9%8B%E9%8B%92-%E5%8F%8D%E6%98%A0%E5%8B%A2%E5%8A%9B%E6%AD%A3%E6%93%B4%E5%BC%B5/))

當然這只是剛剛開始，買新樓房當然是好，新樓住下來30年差不多吧！但是次工的建房、建完賣出還要多次補修，沒有找個專業驗樓師幫手，分分鐘第一個坑。其次不實際的實用面積、當你500呎的建築面積的樓房，有6成實用面積那已經很好了，有良心的地產商可以達到7或是8成，但少至又少。不實際的實用面積後，還要面對奇奇怪怪的建築房間，鑽石形的、四尖八角的……，不僅不實用，還不能用盡面積，其後的納米房，你明啦。這時候，你就要靠專業的設計師來幫你修房擺設，又要花費一筆血汗錢。
[!\[\](https://spark.adobe.com/page/AIjKmh1T4fYd0/images/d7d9858d-f071-4325-bdbc-c1826dc302c2.jpg?asset_id=b75a6ba1-697a-47b6-9587-c51e62e81e1d&amp;img_etag=%221d1ef5f8a4dcfad80e181da52834eed4%22&amp;size=1024)](https://spark.adobe.com/page/AIjKmh1T4fYd0/images/d7d9858d-f071-4325-bdbc-c1826dc302c2.jpg?asset_id=b75a6ba1-697a-47b6-9587-c51e62e81e1d&amp;img_etag=%221d1ef5f8a4dcfad80e181da52834eed4%22&amp;size=1024)
面對新樓，這還好辦。但新樓造價高，賣得更貴。高樓價，8成貸款，一買就立即成為樓奴，除非你是有錢人。所以買二手樓就是大部份中產、或是死慳死抵儲起的首期星斗市民，但坑也就來了。

二手樓，基本30年以上樓齡。樓齡愈長，當然就相對價低。但香港的高房價，樓齡50年的，在重要地區，如名校區、交通方便、地鐵站口、城市中心等，都能增加房價，680萬也有可能(50年住多10年，其實住唔住到都成問題，一般買的就是放租、或是等收購)。再就是買了房就要找設計師或是有信用的裝修師父，不然分分鐘就會有爛尾裝修工程、黑社會找上門，沒有好好裝修的、把你家破壞的……不專業、不守誠、態度差等等，這是第一關。若是弄上法庭，快的就幾個月，慢的就好幾年，為了自己房間，不但花錢、花時間還有可能對方無賴、破產，得不償失。

再就是，樓齡愈高，就意味需要大維修。正常來說，大廈的「石屎」一般都只有約60年壽命，當然要保持到60年以上，就要做好有效維修。看一下國外的超百年的古房，維修費可不便宜。所以理論上到了60年樓房，其實應該要開始考慮收購方案。但是以香港「畸形的樓價」，各個業主都希望收購方高價收購，愛財無可厚非，但不合理的發展就是一個惡性循環，貪字得個貧。最重要的是，港共政府不會去解決相關問題，差劣的體制造成的就是「飯桶」高層與高官，而且配合利益輸送。超過30年的樓房就要抽籤，中的就要「進行大維修」，修好後10年過了，再放入抽籤池，中了再修，治標不治本，這也衍生另一個相關問題，維修工程的壟斷，所謂低價招標，來的就是差劣、偷工維修。

[!\[\](https://spark.adobe.com/page/AIjKmh1T4fYd0/images/ac36a0cf-c355-4061-a929-f0bedbfc5305.jpg?asset_id=a1d36740-b494-4c47-868e-519c6ec0513f&amp;img_etag=%2204aaeabf9c65b8392fe21b69c6362e33%22&amp;size=2560)](https://spark.adobe.com/page/AIjKmh1T4fYd0/images/ac36a0cf-c355-4061-a929-f0bedbfc5305.jpg?asset_id=a1d36740-b494-4c47-868e-519c6ec0513f&amp;img_etag=%2204aaeabf9c65b8392fe21b69c6362e33%22&amp;size=1024)

其次，就是業主立案法團，由大廈組織，或是有能力的人來組成法團，負責管理大廈。當然自己有能力也可以自己管理，但是大家都是上班族，上年紀的也不一定有此專業或是能力，所以物業管理公司就很重要。這就是又一重點了吧。在中共治下，紅色、藍色圈佔據大部份，好的管理公司當然可以給你建議，但是要吸血嗎，做樓房維修的工程公司，再配合地區工作的立法議員的作用。想想，大家都是普通人，沒開過公司又如何會懂管理呢﹖所以不會的就聽從建議，維修壟斷工程。紅色、藍色或是差的工程公司，都有多家子母公司，這家名字爛了換個名字再騙，要不就是找另一家，其實就是之前的母公司，結果就是把整個樓房弄得破敗不堪。再加上法團，維修一下，這就又是一個利益之門。如果沒有堅強信念，有信仰的人，慢慢的也就成為一個小圈子。觀塘事件其實只是冰山一角。香港的手足，一定有聽過天價維修工程吧！沒有用心去管理大廈，最終就是大家住的地方，不斷用次質量維修、被偷工減料，要不就是貪下工程費用，給出的就是沒必要的工程。本可以使用60年的樓房，經這一折騰，40年就像一個老弱傷殘的老人一樣，漏水、缺沙等等，再次維修就是個天文數字，因為不補好，少補少修也根本治不了根。

除了樓房問題，樓上樓下，左鄰右里也是個問題。碰上不講理的，你天花板漏水；樓上不負責任；法團、管理公司又沒能力；政府低效工作，一套下來，人不瘋了，也會被樓房給煩死了。所以買到樓房，就是另一個坑的到來。當然如果你是比較遠的地方，一橦橦自己的，可以減少鄰里問題，但是這些地方就會有黑社會、地區老大等等……又陷入另一個問題。

在邪惡、極權、獨裁、紅龍赤共治下，一個不負責任的港共政府，再加上小圈子選舉的社會，人民最終還是受害者。看看現在、國安法、惡歌法、23條，人民只有反抗！堅持、小心，但不會放棄。接下來我們來看國際出手吧。
