据大纪元5月28日报道，中共非但没有遵守防止核扩散的承诺，而且加大了与朝鲜的贸易交易，并且庇护朝鲜黑客。而美国为了应对中朝威胁，则全面放开了对韩国导弹研发的限制。

美国国会智库表示，中共未遵守防核扩散承诺，中共军方不仅大力加强自身的核能力，而且向伊朗和朝鲜出口与核和导弹相关的货物。此外，中共还加大了对朝鲜的燃料供应。今年4月和5月，在朝鲜海域的运输船只从几只增加到50多只。不仅如此，中共还隐匿朝鲜黑客，帮助其逃避法律追责。而这些黑客在从事某些犯罪行动时，都会寻求中共的帮助。

为了应对中朝日益严重的威胁，5月21日，韩国总统文在寅在白宫与美国总统拜登会谈后，发表了取消韩国导弹研发剩余限制的联合声明。此项声明将有助于韩国提升其进攻和防御能力，并对美国牵制中共也起到了重要作用。

70年前，中共依靠人海战术，用几十万年轻人的尸骨堆起了金家王朝。历史的伤痕虽已褪去，但新的对抗已经形成，中共国的“舵手”和“隔壁三哥”依旧联手，共同对抗世界文明。“金太阳”的一再挑衅，可能会让朝鲜回到石器时代。而中共国最好的结局恐怕就是“过上朝鲜人的日子”。
