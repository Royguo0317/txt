## 印度的警醒：領導世界的應是美國而非中共
`2020-07-06 08:53 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/255590/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/06084400/1-20.png) (圖片來自theweek.in 印度周刊） 
喬治·亞伯拉罕(George Abraham) 7月4日在印度國家先驅報(National Hearld India)就最近發生的中印衝突，和中共病毒爆發後世界對中共的覺醒，發表評論文章。

**中共蓄謀中印衝突**

文章引用芝加哥前市長拉姆·伊曼紐爾（Rahm Emmanuel）的言論“永遠不能錯過危機所帶來的機遇”，他認為，中共或故意操弄病毒，或處理病毒嚴重失職，它顯然想藉此中共病毒危機為契機，迫使全世界屈服在他們腳下。

目前已經有超過121,000的美國人死於中共病毒。而印度不得不封閉整個國家，以對抗這個致命的中共病毒。就在這樣的情形下，更讓印度感到屈辱的是，中共在拉達克（Ladakh）入侵了的印度領土，導致了二十多名印度士兵喪生。

看看中共士兵是如何玩弄“不許攜帶武器控制線”這個所謂的基本協議的，就知道他們有多麼的卑鄙狡詐。據印度消息稱，最初雙方意圖是拉開距離、脫離接觸，但突然轉向，爆發了混戰。中共士兵帶著石頭和狼牙棒衝下來襲擊印度士兵，其中一些士兵被推下懸崖掉進加爾萬(Galwan)河，據說許多人在跌入下面的冰河後喪生。

筆者回顧了1962年的中印戰爭，那場戰爭導致3250名印度士兵死亡，失去了相當於一個瑞士國土面積的阿克賽欽（Aksai-Chin）地區， 從此和平感蕩然無存，因為中共國一直不斷地偷偷摸摸地採取小規模入侵行動，而印度從未採取過任何戰術升級應對。另外，印度過於顧忌中共國的敏感性，經常對中共國在南中國海的霸凌行為保持中立。

文章指出，為了討好中共，印度與達賴喇嘛停止接觸，與中共一年一度雙邊峰會，莫迪總理的綏靖政策也沒有成效。由於印度的注意力集中在巴基斯坦，加上印度與尼泊爾的關係不佳，印度鄰國對印度的支持也不足，這都讓中共藉機成功地用武力慢慢地改變了領土現狀。

就在整個世界陷入中共病毒危機的時刻，他認為中共在中印邊界的軍事行動不是偶然的，是趁機奪取印度領土的蓄謀。他列舉了中共近幾週對周邊地區的行為：在南中國海擊沉了一艘越南漁船、圍攻馬來西亞海上鑽井平台、威脅台灣並加緊了對香港的控制。

**中共借病毒危機尋利**

文章還認為中共操弄了世界衛生組織，讓其否認中共病毒的危險性，誤導了全世界。中共還在世界各地瘋狂採購防禦病毒的醫療防護設備，之後又將這些用品出口給那些國家以謀取骯髒的利潤。

那些不屑美國在世界擔當領導地位的人，思考一下中共國日益增大的實力吧： 中共的極權根本不尊重人權，鄙視並且經常嚴厲懲罰持不同政見者。在武漢的中共病毒危機的高峰期，發生過病人在家中被外面反鎖釘死，還有，居民聽到附近火葬場的慘叫聲。

看看中共如何處理中共病毒疫情危機，就知道這個專制是怎樣卑鄙無信，怎樣掩蓋真相傳播虛假信息來損害世界。據美國情報機構的報導，中共掩蓋了中共病毒爆發力度，低報病例人數和死亡人數。源於2019年底中國湖北省的中共病毒疫情，僅報告了86,000個病例，死亡4600人。

他引用內布拉斯加州（Nebraska）共和黨參議員本·薩斯（Ben Sasse）說法：“聲稱美國死于冠狀病毒的人數比中共國多是虛謬。” “我不想評論機密信息，但我們不能不面對的是：中國共產黨為保住它的政權，已經撒了謊，還正在撒謊，並將繼續撒關于冠狀病毒的謊。”

筆者認為，毫無疑問，中共的政治性決策使得這種致命病毒在全世界擴散，中共要負責任。中共的網絡虛假信息、腐蝕世界衛生組織（WHO）的政治操作以及缺乏道德操守的表現，導致這場健康和經濟危機仍在持續，各國政府還在竭盡全力應對中共病毒的蔓延。

三十年來，中共國經濟的高速發展，和中共管理政治化的不可預測性，給全世界帶來相當大的焦慮和深切關注。在此刻，中共病毒大流行之時，中共還在南中國海挑釁，儘管該地區的國家有所抵制，但他們並沒有做到阻止中共的放肆。儘管美國常來南中國海秀肌肉，遏制中共海軍日益增大的威脅，但目前該地區的領導人已經被疫情大流行所纏身，不得不依賴中共國的投資和醫療援助，因此根本無法抵抗中共。

此時此刻，中共極權的鐵拳再次揮舞，不僅威脅著世界人民的福祉，而且實實在在地威脅著全球的和平與安全。
