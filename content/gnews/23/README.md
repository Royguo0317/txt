**美国安顾问对中共发布严厉警告声明**

美国国家安全顾问奥布莱恩发表声明：美国将会「考虑用更多手段」来回应中共「不负责任以及有危害性的做法」。川普总统哪怕只剩下10天也依旧有3个「72小时」来对付中共。国家安全顾问不同于负责外部事物的国务卿，他作为川普内阁成员具有调动各种力量的执行权力。

**奥布莱恩声明中的「更多手段」绝不是简单「文斗」**

代表着情报与行动的白宫联络人奥布莱恩声明中的「更多手段」绝不限于财政、经济脱钩等层面，不是简简单单的文斗，美国八千万人不可能就这么被糊弄过去，这就是「天线宝宝习」去哪里了，连声明都不敢发，只能躲在地洞的原因所在。

**朝鲜突然深夜阅兵示威美韩**

韩国JCS称军方发现朝鲜周日举行深夜阅兵的迹象，美韩情报部门对此密切监视。朝鲜深夜阅兵的目的，第一威胁韩国不要轻举妄动，因为地理位置濒临，火炮也能直接打过去。发射导弹的话可以有雷达提前监测，但火炮攻击就纯粹靠抛物线原理没有办法预先知道。借此北朝鲜拿韩国做人质对美国示威显示可以与其玉石俱焚。第二，在晚上没有电的情况下演练军事指挥系统能否正常运行。

结合美国安顾问对中共发出警告，北朝鲜深夜阅兵以及「天线宝宝习到底在哪里」这些国际新闻，只有对习活捉才能制止北朝鲜的耍流氓手段，如果斩首北朝鲜发动火炮会造成大量死亡等人道灾难。

**习近平最高指令系统失灵**

最高级别的指令系统就是口谕，让人找不到证据。跟习近平对接的人就只有几个亲信，但就在这样的情况下，爆料革命都知道习近平下达的指令与行动，导致习处于一种“到处找屋子哪里透风”的崩溃状态。而当习的指令系统做不了的时候，爆料革命内部战友就可以利用他的这套指令系统指挥国家机器做正确的事情，同时习近平又是集权力独裁于一身，这样中共的结束其实是很快的。
