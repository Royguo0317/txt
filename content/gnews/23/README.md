在下曾谏近平先生以天下为重，以家人为重，以己身安危为重，弃权力而投美利坚。如此为之，可医脑患，可免己家之祸患，亦可造福苍生。奈何先生固执，不可为之。在下早已言明，先生周边尽皆嗜权之人，全然不会顾及先生安危。今先生造祸如此之大，能存活至今日，实乃先生为罪魁祸首，体制众人，皆待众怒全攻先生而去，先生何以如此麻木不仁？先生之祸，源于体制，而体制依然望众祸归于先生一人，先生当真不知？

昔日泽东在世，体制内蠢蠢欲动，欲取而代之。泽东但亡，其妻下狱，其党被诛。而后，对其尸身开膛破肚，填塞不腐之物，置放于会堂之中，供众人瞻仰。《礼记·祭义》曰：“众生必死，死必归土，此谓之鬼。骨肉毙于下，阴为野土。《韩诗外传》曰：“人死曰鬼，鬼者归也。精气归于天，肉归于土”。华夏文化，讲求入土为安。而众体制人，全然不虑泽东遗言，令其暴尸于堂所之中，成众人休闲嬉戏观赏为用。此等行径，如东汉末年，董卓被诛，暴卓尸于市。然对泽东更甚，令其永不入土，永不为安，令天下永无泽东墓，仅剩赏尸堂。小平观后，心中惊慌，感其生平亦罪孽深重，恐死后有人糟蹋其身，遂告继位者，若其亡，火化尸骨，余灰洒海。先生若有不测，庙堂该如何待先生后人？该如何待先生之身？

先生如此贪恋庙堂权力，宁得罪天下人也要恋栈帝位，忘生死而对百官，如此为之，所为何故？佛家五毒，先生全占。但先生可知生老病死，人生循环。毛泽东言：“与天斗，与地斗，乐无穷，定胜天。”然泽东年不过百岁，龄八十三故之，终究难敌天道。先生更应知，因果报应，轮回循环。泽东难入土，先生以为报应否？

先生患疾已久，切疾情反复，若此疾再次复发，先生以为如何？此病症，因何发，发几时，先生早已知晓。先生强压之下，头疾复发已成循环之势，且势不可挡。如此境况，先生何再一意孤行，抓握权柄。如此权力场之阴谋诡计，如此帝王座下不问人情，如此病危下之虎视眈眈，先生不觉人生苦楚么？先生此时此景，刻刻煎熬，时无间，空无间，受苦无间，人虽活，心已入阿鼻地狱矣。

先生纵然不为苍生念，亦当为己念。人非长生，匆匆百年，何必心入阿鼻狱，日日担惊，夜夜受怕，皱纹横生，苦熬白发。纵然可熬过此劫，先生亦是油尽灯枯，行将朽木。一朝身死，难保不是亲人下狱，泽东第二。若身后交权，平生所为皆他人嫁衣，何苦来哉。

在下实望先生康健，安不安否，便不得而知了。爆料革命，以法而行。先生康健，便可行庭审。如远东审东条英机，如纽伦堡审纳粹。如此审判，为天下立命，为百姓述怨。华夏百姓当知何为“天子犯法与庶民同罪”，当知这古训如何行，更当知犯此古训者得何果，不仅是一口愚民空谈。如此，在下望先生去疾成功。

在下二次谏言，望先生明我心意。虽在下有庭审先生之心，亦望先生“放下屠刀，立地成佛”。
