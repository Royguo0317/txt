**红色高棉大屠杀或称红色高棉种族灭绝、柬埔寨大屠殺或柬埔寨种族灭绝，是指1975年至1979年初，柬埔寨共产党总书记波尔布特领导的红色高棉****共产主义政权在柬埔寨进行的大规模杀戮事件。据估计，在红色高棉统治时期，柬埔寨全国范围内共有150万至300万人非正常死亡，约占当时柬埔寨总人口的四分之一。**

我非常好奇一件事情，为什么在共产主义国家，总是会发生这类的耸人听闻的大屠杀事件。即使如此，今天的中国共产党依然在恬不知耻的歌颂党的伟大。共产党俨然是邪教无疑，为什么今天还会有人乐此不疲的追求着共产主义呢？

我们先说说邪教的行为特征

**一，行为控制**

**●强硬的清规且有责罚制度**

**●居住环境，家人、情人、同居者、甚至往来者受到规范**

**●衣着打扮等受到规范**

**●食物受到规范**

**●睡眠受到规范**

**●无自由休闲的空间**

**●教义的授课及组织活动（如念经、礼拜、冥想等）占去生活中最大部份的时间**

**●主要决定不能自作，必先取得批准**

**●须汇报思想，将自己感受与行动报告给上层干部**

**●集体思维高于个人**

**二，资讯控制**

**●使用谎言，包括控制消息，歪曲资讯**

**●减低教外信息的散布或不鼓励接触这些，包括对教批评的说话，和使成员过度忙碌以至无暇接触外界**

**●形成成教内和教外对垒的局面，由教主决定谁可知什么**

**●鼓励教徒互相举报，成立相互监督制度**

**●大量并广泛使用教义和由教派传播的信息，并大肆宣传**

**三，思维控制**

**●须全然依赖并服从教主或讲师**

**●将教义定为真理，宣扬非黑即白的二元思维论**

**●不能批评教主或教义**

**●使用特殊语言代替常规的思想表达**

**●压制思考，只容许“好”及“正确”的想法，否定批判、理性想法以及建设性的批评**

**●长期冥想，咏唱咒语和祷告**

**●不准容纳别的信仰**

**四，情绪控制**

**●使人的感受受到控制**

**●相信所有问题都是自己的错，而教主或干部是不会错的**

**●过度夸大罪恶感**

**●过度夸大恐惧，包括怕自由意志，怕外界，怕敌人，怕失去救赎，怕​​离开教派，怕遭否定**

**●使用过大的情绪波动来控制人**

**●强迫人公开悔罪**

**●出现非理性的恐惧，比如害怕领导人，害怕离开，不会反驳领导，如离开即对前途无信心**

**●教外没有快乐可言，故不能离去**

**●对离开教派者杯葛**

**●离开教派即属堕落**

**●离开教派即有灾祸**

为什么总是无法和中国人交流，为什么让一个中国人改变心意是这么的难。首先，你要明白，虽然说，中华人民共和国是一个国家，它拥有一个完整的行政体制。但是，它实际的运行方式，是一个邪教。从邪教的特征来衡量，共产党几乎每条都占上了。公众在不明真相的情况下，不由自主的的加入了一个邪教。更悲哀的是，很多中国人就出生在一个邪教之中。而在中国，每个人都清楚，一切都要听党的。

这是一个民族的悲哀，中华民族亲自建立了一个邪教，而后又有人加入了这个邪教，然后他们的后人诞生于这个邪教，还要为这个邪教摇旗呐喊。更可怕的是，一个行政机构的运行方式，权力的伸展，都是基于邪教的壮大和发展。

自从秦始皇统一中国，自从中央集权制的建立，中国人就对权力充满无限的渴望和幻想。可是随着时间的推移，世界的变迁，尤其是1840年鸦片战争后清廷的腐败，到民国的建立，人们渐渐看明白，皇帝是可有可无的。再用中央集权这一制度禁锢中国人，对于民国时期的人们来说，已经是不可能的了。中华民族可谓命途多舛，民国时期，苏维埃成立，共产主义思潮开始流入中国。从这时开始，共产主义被中国独裁者捧为圣经。因为，独裁者终于找到了新的统治方法和纲领。对于独裁者来说，他们从来没有信仰，有的只是赤裸裸的野心，和寻求奴役别人的权力。

人世险恶，人心不古，《共产党宣言》中提出“消灭私有制”的主张，让众多心怀不轨的人热血喷张。而把人划分为两大阶级，即无产阶级和资产阶级，让社会上的多数无知穷人对还能拥有财富的人充满了憎恨，让有阴谋的人可以驱使更多无知的人。最后再用阶级斗争的方式，对有产者进行抢劫。这便是共产主义所谓的进行合理的分配。这样的一场毫无道理的集体抢劫，通过伟大的共产主义理想进行包装，变得冠冕堂皇。

没错，共产主义的实行，就是极恶的人煽动一群人在仇富的情绪下进行犯罪。当这一切被共产主义理想包装过后，人会有了一种心态；即这种行为不是犯罪或这种犯罪是合理的。而当出现有良知的人阻止的时候，就被打成反革命。有对立面出现的时候，也被打成反革命。就这样，在别有用心的人驱使下，共产主义成为了犯罪者的“免罪金牌”，这种情况下，犯罪的人越来越邪恶，越来越无所顾忌，最后导致当权者都不安全。

这样才能解释，为什么会出现红色高棉这种大屠杀。也只有这样才能解释，为什么中国会出现所谓“三年自然灾害”，让数千万人死亡。而这种暴政情况下，毛泽东还对红色高棉领导人波尔波特进行了支持。这不是独裁者对于一个独裁者的支持，这是一个犯罪者对另一个犯罪者的支持。更是一个邪教头领对另一个邪教头领的支持。

可悲的是，累累白骨的历史，在共产邪教的洗脑下，变成了街头巷尾的一种杂谈。好像，一切都只是发生在别人身上。这种无知的可悲，迎来了共产邪教新教主。而这位新教主，开始了对国内外毫无罪恶感的大屠杀。
