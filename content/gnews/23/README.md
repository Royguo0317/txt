**美國倡導一個自由和開放的印太地區**

今天，我們正在制定一個美國在該地區重要的、有爭議的部分——南中國海政策。 我們明確表示：北京對南海大部分地區近海資源的主張是完全非法的，其控制這些資源的霸淩行為也是非法的。

在南海問題上，我們尋求維護和平與穩定，以符合國際法的方式維護海洋自由，維護商業航行暢通無阻，反對任何使用脅迫或武力方式解決爭端的企圖。 我們與許多盟友夥伴有著共同的重要利益，他們長期以來一直支持基於規則的國際秩序，但這些共同利益受到了中華人民共和國（中國）前所未有的威脅。北京使用恐嚇手段破壞東南亞沿海國家在南海的主權權利，逼迫它們放棄近海資源，堅持單方面的主導權。

**國際法庭仲裁中共在南中國海為非法行為**

2016年7月12日，根據1982年《海洋法公約》——中國是該公約的締約國——組建的仲裁庭在一項一致的裁決中，駁回了中國的海洋主張，認為其沒有國際法依據。仲裁庭幾乎在所有訴求上站在了提起仲裁案的菲律賓一邊。

今天，我們將在美國對中國在《海洋法公約》中的海事索賠的立場與仲裁庭的裁決保持一致。 具體來說中共不能在法庭認定的菲律賓專屬經濟區或其大陸架上，對菲律賓提出任何海洋主張——包括任何源自斯卡伯勒礁（黃巖島）和南沙群島的專屬經濟區主張，這兩個地方完全屬於菲律賓的主權權利和管轄權範圍，北京對這些島礁沒有任何領土或海洋主張權利。

**美國將與東南亞聯盟國家站在一起維護南海穩定發展**

世界不會允許北京將南中國海視為其海上帝國。美國與我們的東南亞盟友站在一起，根據國際法規定的權利和義務，保護他們對近海資源的主權權利。我們將與國際社會一道捍衛海洋自由和尊重主權，反對中共在南海或更廣泛的地區強行推行 “強權即正義”的欺淩行為。

**編者觀點：**

中共國在南海主張的九段線完全是非法行為，2016年國際法庭判定中共在南海的主張為非法。現在，美國堅持站在菲律賓以及東南亞聯盟國家來對抗中共，這也正是美國在用行動告訴中共不要在南海亂來。美國2019年在南海巡航的時間比之前40年的總和還要多。自2012年以來，美國在南海首次舉行了罕見的雙航母編隊聯合軍演，同時最近也在印太地區和印度、日澳也進行了聯合軍演。美軍在印太地區部署三艘航母，意味著印太地區的南海、臺海的緊張局勢。隨著中共進一步瘋狂，美軍也一定會在南海上動手，讓我們拭目以待！
