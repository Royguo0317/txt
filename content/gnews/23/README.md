这是美国总统拜登入主白宫后首次在国外。他与英国首相就一个主题进行了交谈：建立一个新的联盟。

在英国康沃尔举行的 G7 峰会之前，美国总统乔拜登呼吁世界各地的民主国家联合起来反对独裁者。 “我们正处于世界历史的转折点，”拜登周三在英格兰东部米尔登霍尔空军基地的美国士兵面前说，这是他上任以来首次出国旅行。

“你比任何人都清楚民主不是偶然发生的。我们必须捍卫它。我们需要赋予他们权力。”他补充说，“我们需要摒弃那些认为民主时代已经结束的人。”

美国、德国、英国、加拿大、法国、意大利和日本的国家元首和政府首脑以及欧盟代表将于周五至周日在康沃尔西南部举行峰会。

重要工业化国家的会议旨在展示世界主要民主国家之间的团结。因此，东道主英国也邀请了韩国、南非、澳大利亚和印度。中心议题包括气候保护和冠状病毒大流行。
