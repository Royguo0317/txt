## 斯伯丁將軍：這是我見過的最愚蠢的事
`2020-05-30 04:24 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/217525/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/30042012/1-131-67.jpg)
5月29日，據彭博新聞報道：美中關係惡化將導致華爾街可能在中共國市場損失數十億美金。美國戰略家斯伯丁將軍(Robert Spalding)據此發推說：「抱歉，這是我見過的最愚蠢的事。」

在經歷了多年的貿易戰動蕩之後，美國的政策制定者現在開始將矛頭對準金融業，因為越來越多的人懷疑美國公司將資金投入中共國——這個被視為地緣政治大敵的專制獨裁國家。政策制定者和立法者正在考慮限制美國對中共國公司的養老基金投資，並限制中共國公司在美國融資的能力。

然而，某些公司認為，作為世界第二大經濟體的中共國是最大的市場。 例如，通用汽車在中共國的汽車銷量多年來一直超過美國，中共利益集團購買的寶馬超過美國和德國的總和。

高盛集團、摩根士丹利、摩根大通、花旗銀行和美國銀行等華爾街巨頭均在中國有數百億美元的股份，2019年，美國五家大型銀行在中共國的敞口總額為708億美元。儘管他們在中共國的資產相對較小，但他們未來在華的大型擴張計劃可能在兩國間針鋒相對的鬥爭中破滅。

高盛估計，到2026年，中共國經紀行業的利潤可能達到470億美元，而外國公司的目標會佔據相當大的一塊。據估計，商業銀行的利潤為80億美元，預計總資產將達到80億美元，黑石和先鋒集團等基金巨頭也在追逐這一目標。這些美國金融服務公司為阿里巴巴集團等中國公司提供諮詢服務，政治關係的緊張將使他們的增長計劃將變的虛無縹渺，而且還會威脅到他們多年來的在華收益。

前銀行家詹姆斯•斯坦特(James Stent)表示：「如果你是一家美國金融機構，而且你已經獲得批准一項進軍中國的計劃，那麼你將試圖在美國政府允許的範圍內繼續這一計劃，因為你看到了未來巨大的利潤。」他曾在兩家中共國銀行的董事會工作了十多年。「美中冷戰對你在中國建立業務的計劃是不利的。」

一個為美國國會提供諮詢的機構本周對華爾街向美國政壇的施壓提出了質疑，稱立法者需要「評估美國在金融市場中擴大參與的可取性，因為這個市場被一個戰略競爭對手以政治優先的借口扭曲。」川普的高級經濟顧問表示，北京應對美國負責，將釆取措施最大縮水目前美國銀行在中共國的投資計劃。

針對中共鎮壓香港的行動和《香港國安法》，美國將升級對中共國的制裁，包括金融制裁。唐納德•川普(Donald Trump)總統表示對中共不高興，周五，川普在白宮發表美國對中共國關係新的國家政策，並將釆取措施以」保護美國的安全和繁榮「，他提到與中共國的經濟往來造成美國」每年好幾千億美元的損失」。
原駐華軍事參贊斯伯丁將軍今天高度讚揚川普在白宮的講話，他發推說：」美國重歸自由！我們再也不會犧牲援助和原則來(幫助)中共殘酷壓榨中國人民。」網友跟貼說，多年來」中共捏住了華爾街老闆的睾丸，他們無恥地與獨裁者勾兌以獲利……這些貪婪的全球化企業與中共勾結，將我們賣了30年……該給他們一個教訓了」。

還有網友評論：銀行對於一個共產主義國家來說太幼稚，共產黨許諾他們未來每年賺取15％利息的計劃，而當你們所有人參加時，CCP將毫無風險地沒收你們的錢。在中共國，法律可以隨時隨地改變，而且中共領導人喜歡改變。中共政府控制著中共國的所有企業、銀行、投資以及市場。把數十億美元的客戶資金交給中共政府是非常愚蠢的，就像把汽油交給縱火犯一樣。現在是時候阻止大公司以幾美元的價格將美國和我們的自由賣掉了。
