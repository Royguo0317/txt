《路德时评》2021年月日间节目主要内容：

一、捷克红衣大主教声称中共病毒是’“生物武器”

环球新闻网2月18日报道[1]：据捷克《布拉格晨报》17日消息，布拉格总教区总主教杜卡日前把新冠病毒称为“中国病毒”并表示病毒是泄漏的“生物武器”。对此，捷克军方明确表示不同意此观点，捷克知名免疫学家瓦茨拉夫· 霍热希直言杜卡的说法是“可耻的”。中国驻捷克使馆17日也敦促杜卡总主教立即纠正错误，切实消除恶劣影响。

报道称，杜卡此前宣称，世界上“所有军方的专家”都相信他的观点，他们只是“不敢说”或者“不能说”。一名杜卡的发言人还声称，该声明得到了捷克军方成员的支持。不过这位发言人并未点明是具体是军方哪一位成员支持这一说法。

路德社评论：

1、多米尼克雅罗斯拉夫·杜卡（生于1943年4月26日）是捷克枢机主教，是第36届布拉格的大主教和罗马天主教枢机主教，他是苏联共产主义的反对者，曾任多米尼加共和国省长，捷克共和国上议院主席，以及欧洲上议院联盟副总裁一职，与捷克现任总统关系甚密。

2、捷克大主教之所以敢在讲道时公开声称中共病毒就是中共释放的“生物武器”，背后一定也有某种势力的支持。历史上每次遇到重大事件都是宗教界人士首先站出来表态，最后都被一一验证是正确的，天主教之所以伟大，可以延续上千年，很重要的就是他们总能在最关键的时刻站在正义一方。他在发言中声称所有军方专家都知道中共病毒就是生物武器，只是他们不敢表态，杜卡很可能就是从军方人士获得情报得知和认定中共病毒就是生物武器，以杜卡作为红衣大主教的身份和公信力，他绝对不会轻易发表如此重磅言论。

3、捷克杜卡红衣主教此番发言将会影响捷克政府以及周边国家的一系列行动，中共不可能把北约三十个成员国都成功收买，尤其是这些国家中的宗教力量，红衣主教主管某个教区的具体事务，其在地方的实际影响力远大于罗马教廷。

4、捷克是一个位在于中欧的内陆国家，但在历史上也被分类成东欧国家。捷克在古代被称为波希米亚王国，是神圣罗马帝国的首都和三十年宗教战争的发源地，在近代是奥地利帝国和奥匈帝国的工业中心；在冷战时代被称为捷克斯洛伐克，是世界七大工业国之一，它曾是首个站出来反对纳粹，反对苏联共产主义的国家。

5、中共认识到杜卡大主教的背景和极大的影响力后，他们不得不用双语在其媒体平台严厉抨击杜卡的言论，这足以看出中共出现了极度恐慌。中共发文明确要求杜卡纠正错误，这一方面是在向捷克发起挑战，纳粹在二战时期也是如此威胁捷克，虽然英法两国出卖了捷克，捷克依然不畏惧当年纳粹的强权，誓死抵抗。另一方面，中共此举也是向天主教势力发起挑战，以捷克人民的一贯作风绝对不可能就此妥协。

6、只要捷克政府或者北约组织任何一个国家认定中共病毒就是“生物武器”，北约组织和美国就必须联合对中共国发动战争。闫博士的报告将中共病毒定义为“超限生物武器”，这是中共在和平时期发起的5.0版本战争，这与传统的生物武器定义有巨大区别，这也导致西方武器专家认为中共病毒不是生物武器。所有战友们一定要将超限生物武器和战争5.0版本的概念宣传到西方世界，更新西方军政商各界人士对中共病毒的认识。
