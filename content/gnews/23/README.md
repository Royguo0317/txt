## 加拿大溫哥華地區CCP病毒實際感染人數是目前報導確診病例的8倍
`2020-07-16 11:33 gnewscanada` [轉載自GNews](https://gnews.org/zh-hant/266847/)

翻譯報導：喜馬拉雅聯盟加拿大農場 陳廷青

編輯校對： HUH

評論：

從卑詩省的這項研究不難看出，研究者們已經慢慢認識到被新型冠狀病毒，也就是CCP病毒感染的人數遠遠超出已經檢測確診的人數，他們對CCP病毒的新的認識可以警醒溫哥華乃至其他加拿大地區的公民對CCP病毒提高警惕和防範意識。但值得提醒的是，該報導中研究者有提到：“低水平的傳播，也意味著很少人擁有抗體來應對未來可能出現的疫情高峰”，這種說法是很不科學的。希望研究者，科學家和媒體們不要誤導人們去期待利用群體免疫來抵抗CCP病毒，應該提醒大家注意防護，追溯病毒來源，尋求根本上的解決方法，嚴肅的對待這場戰爭。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/16112646/image-59.png)圖片來源：58溫尼伯
2020年7月16日，Vancouver Sun媒體報導，加拿大卑詩省衛生部長阿德里安·迪克斯（Adrian Dix）對卑詩省的抗疫情進步感到滿意， 同時披露研究表明，溫哥華地區被CCP病毒感染的人數是報告病例數的8倍。這項由卑詩省卑詩大學疾病控制中心、生命實驗室和公共衛生科學家做出的聯合研究已經被發表在健康研究網站medRxiv上，有關此發現的新聞發布會定於週四上午舉行。 

研究者們說，這些發現表明，卑詩省總體感染率大概不到百分之一，社區傳播成功得到抑制。生命實驗室收集的血液樣本， 3月份數據發現869個樣本中有2個呈陽性，患病率為0.28％，5月份的樣本發現患病率為0.55％。衛生部部長阿德里安·迪克斯（Adrian Dix）將這個被估計的感染率描述為“非常低”，並認可卑詩省的公共衛生措施和公眾合作的有​​效性。迪克斯說：“這進一步證明了我們走在正確的道路上。”“當然，該研究認為這事情有兩面性，即低水平的傳播，也意味著很少人會擁有抗體來應對未來可能出現的疫情高峰。” 

如果將這項研究的感染率應用於所有省份人口，以溫哥華區為例，則可能意味著約有2.8萬人已被CCP病毒感染，而迄今記錄在案的確診病例僅是3149例。 

 “卑詩省的總體感染水平偏低，這是由於於卑詩省檢測範圍造成的。也就是說，我們已經明確地表示，感染COVID-19的人多於測試呈陽性的人，尤其是在三月和四月期間，我們把測試計劃重點放在了特定人群上，比如醫療保健工作者”，迪克斯說。這是在加拿大的首次研究。 

迪克斯說，這項研究很重要，因為它涉及到沒有自行選擇進行CCP病毒測試的人，這些人可能不相信自己感染了這種病毒。該研究由省衛生官員邦妮·亨利（B​​​​onnie Henry）博士授權，並由大學臨床研究倫理委員會批准。尚未經過同行評審。 

卑詩省週三新增21例，無新增死亡病例。政府在新聞稿中說，目前有207活躍病例，而有2,753例呈陽性的人已經康復。
