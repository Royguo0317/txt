## 中共神藥【連花清瘟】最後的瘋狂！
`2020-08-25 08:33 GM30` [轉載自GNews](https://gnews.org/zh-hant/316199/)

**作者：小妍說**

這兩天在國內有壹篇關於連花清瘟膠囊的文章被傳的沸沸揚揚的，文章內容我用翻譯軟件翻譯了壹下，大概就是說：美國國家生物技術信息中心表示，“整體”用中藥連花清瘟治療14天可導致“癥狀恢復率顯著提高，癥狀恢復時間縮短”。根據美國國立衛生研究院分支機構美國國家醫學圖書館的NCIB的資料，將284名COVID-19患者隨機分配為單獨接受常規治療或與連花清瘟合用，並服用四粒膠囊每天3次，共14天。NCIB說：“主要終點是癥狀（發燒，疲勞，咳嗽）的恢復率。”它報告說，“總體”是對接受常規治療的142例患者使用142名患者使用中草藥進行的14天治療，“結果是，與對照組相比，其癥狀恢復率顯著提高，並且癥狀恢復時間更短”。研究表明： “治療組的發燒，疲勞和咳嗽的恢復率也更高。” “連花清瘟膠囊對（COVID-19）的治療具有良好的安全性。”NCIB說：“據我們所知，這是第壹個證明連花清瘟膠囊安全性和有效性的多中心隨機臨床試驗。”

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/25082121/%E5%9B%BE%E7%89%871-18.png)

看來這連花清瘟又出幺蛾子了，前幾天中共國媒體剛剛發出說疫苗12月份就要上市了，這都沒被國外媒體出來推薦壹下，怎麽這個連花清瘟都這麽久了還能如此厲害呢？而在中共國生活的人壹說到連花清瘟估計沒有不佩服的，佩服的不是它的療效，佩服的是從甲流到乙流，從禽流感到現在的中共病毒，過去十幾年裏所有病毒性事件中，每壹次都有連花清瘟的身影，每壹次連花清瘟都被中共國家層面診療方案推薦。我就納悶了，世界上那麽多抗病毒藥都做不到“獲得不同病毒診療指南的認可”，連花清瘟卻做到了。它不僅做到了，並且上市16年，藥品說明書中關於不良反應和禁忌的描述依然是：尚不明確。尤其是現在這種情形下，美國政府又是封抖音，又是封微信的時刻，連花清瘟竟然奇跡般的獲得了“美國國家生物技術信息中心”的推薦，這完全超出了普通人能理解的政治範疇了。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/25082655/%E5%9B%BE%E7%89%872-115.jpg)

不過這個美國國家生物技術信息中心（NCBI）我估計有很多人把它當成了壹個純粹的生物醫學研究機構。不！所謂“美國國家生物技術信息中心”就類似於中共國的知乎和國外的品蔥壹樣，正常人都可以在上面隨便發表內容。更可笑的為什麽幾個中共國人要跑到外國的知乎上去宣傳中藥 ，是國內不能發，還是他們知道發了也沒用？而且就這麽網站竟然被中共的自媒體吹成了“美國推薦使用連花清瘟抗新冠，效果好到令人發指”。這是哪個不長眼發出來的呢？我看這就是中共國內疫苗12月份要上市，導致壹大批人坐不住，想趁最後的紅利期再撈壹筆吧。
