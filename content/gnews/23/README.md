**西班牙2021年1月26日电/西喜社——**一名中共病毒受害者的亲属要求会见来访的世界卫生组织专家团队，称该团队应该与受影响的家庭对话，他们声称自己被中共国政府捂住了嘴巴。

张海的父亲于2020年2月死于中共病毒。他一直在组织受害者亲属要求向官员问责。张海说，他担心世卫组织可能被用来为所谓的中共在疫情爆发初期的失误提供掩护。

世卫组织说，这次访问是调查病毒起源的科学任务，而不是问责。世卫组织团队预计将在本周晚些时候开始实地工作。

香港大学公共卫生专家、前世卫组织官员福田敬治说：”这一切都要看团队能接触到什么。他们真的能够提出他们想问的问题吗？”
