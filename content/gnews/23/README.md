## 特朗普政府正以最快速度從中國撤出供應鏈
`2020-05-05 11:39 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/194689/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/05113321/%E7%89%B9%E6%9C%97%E6%99%AE%E6%94%BF%E5%BA%9C%E6%AD%A3%E4%BB%A5%E6%9C%80%E5%BF%AB%E9%80%9F%E5%BA%A6%E4%BB%8E%E4%B8%AD%E5%9B%BD%E6%92%A4%E5%87%BA%E4%BE%9B%E5%BA%94%E9%93%BE.jpg)
中國美國商會和上海美國商會在今年四月進行的一項調查中，發現五分之一的美國公司表示中共病毒將加速他們的企業與中共國的脫鉤，而超過一半的公司（52％）表示，制定長期供應鏈的戰略現在還為時過早。

與此同時，特朗普政府正在緊鑼密鼓，準備在中共冠狀病毒疫情過後，減少對中國製造業的依賴，並對公司轉移供應鏈實施新的激勵措施，例如稅收優惠和補貼。

美國務院經濟發展、能源與環境部副部長，基斯·卡拉奇（Keith Krach）告訴路透社，在過去的幾年裡，美國一直在積極推動產業鏈脫鉤，現在美國要全力以赴干這件事，重點是放在關鍵和極其重要的領域。

美商務部，州政府及其它聯邦機構正在探索措施，以確定哪些供應鏈是必不可少的，以及如何將它們與中國脫鉤。 消息人士稱，一項這樣的提議涉及建立一個不包括中國在內的“可信賴的夥伴”貿易網絡，稱為“經濟繁榮網絡”。

美國務卿龐培奧（Mike Pompeo）上周表示，美國正在與澳大利亞，印度，日本，新西蘭，韓國和越南等國家合作，為推動全球經濟向前發展，提出了包括如何重組產品供應鏈，以防止類似情況從此不再發生。

國土安全部在一份最新報告中發現，中共為了囤積醫療用品，有意掩飾了最初的中共冠狀病毒爆發的嚴重性。

一位美國高級官員告訴路透社， “這場疫情風暴後，人們更增加了對與中共國做生意的種種憂慮。” “如今人們的經濟損失，要遠遠超過以前與中國達成交易時賺到的全部利潤。”
