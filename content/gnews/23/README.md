中共的防火墙于1998年设立至今有20几年，其主要作用是监控所有经过国际网关的通讯，对认为不符合中共官方要求的传输内容，进行干扰、阻断、屏蔽，从而到达舆论控制，洗脑百姓的目的。而不是互联网审查的定义说的是指透过立法规管和官方倡议等手段，来控制或禁止人们于互联网之上所观看、存取、发表的内容从而达到自我审查。

中共的防火墙就是一个大监狱。中共建立防火墙把中国14亿百姓圈养在一个网络监狱里面，无法获得世界外面的真实信息，尤其是那些不利于中共对14亿老百姓统治的信息。国外对重要事件的报道，墙内的网民是没有办法了解到真相的。比如1989年6月4日的天安门事件，2019年6月开始的香港人反对《逃犯条例修订草案》的和平上街游行，再比如对爆料革命的郭文贵先生的各种抹黑事件等等。有了防火墙，就可以黑白颠倒、歪曲事件。对自己统治不利的事件进行拦截、屏蔽或者干扰。墙内网民只能在在墙内的网站阅读相关新闻，给你画个圈就地为牢地阅读新闻。

中共的防火墙是为了愚昧墙内的14亿中国人。在没有新闻自由的中国，媒体就是姓党。对国外只报忧不报喜，让中国人觉得外国人生活在水深火热之中；对国内只报喜不报忧，让中国人觉得在中共的统治下的中国很幸福。然后墙内的中国人就慢慢地就失去了对外界政治的关心，慢慢地就失去了自己对重要政治事件的思考能力和独立分析能力，甚至是基本的逻辑思考能力。再加上中共这些年对西方传统媒体和重要媒体机构的蓝金黄，即使墙内的网民能够出墙来所阅读到的有关中共的新闻也是经过加工的新闻。墙内的网民就更不可能知道中共的真面目。长此以往14亿中国老百姓就被中共洗脑成无知愚昧，党国不分，甚至被中共利用绑架，成为中共要挟西方的工具。中共的战狼外交，14亿老百姓成为肉票却浑然不知，被中共一同绑上战车，走向地狱。
