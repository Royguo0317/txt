6月15日，据英国《每日电讯报》报道，英国首相约翰逊在宣布英国与澳大利亚达成了一个自由贸易的协议后，把矛头指向中共，抨击了中共对澳大利亚采取的咄咄逼人的行为，并誓言，英国将与“我们的朋友并肩站在一起”，以此对澳大利亚总理莫里森表示了坚定的支持。

![]()![](https://gdb.voanews.com/A2885F66-AF05-4B7B-A4A1-6F03D3D703B0_w1023_r1_s.jpg)图片来源：Daily Telegraph

同时，法国总统马克龙也不点名地批评了中共对澳大利亚进行的经济胁迫行为，称这“明显违反了国际法”。

目前，由于中共对澳大利亚实行了经济胁迫，导致澳大利亚的出口产品在中共国受到不公平打压。虽然，因铁矿石和煤炭价格的暴涨，澳大利亚的出口总额比去年略有增长，导致中共对澳大利亚的经济胁迫并没有对澳大利亚政府产生太大的经济压力。但是，中共这一胁迫对于澳大利亚的农产品出口还是有不小的影响。因此，澳大利亚的农民敦促莫里森政府缓和与中共的关系，以解决农产品出售的问题。

目前，英澳这项协议是英国在脱欧之后的第一个双边自由贸易协议，将有助于澳大利亚的农产品出口到英国，也极大地缓解了莫里森政府所受到的压力。这也是继5月份布林肯在新闻发布会上表示，“面对中共国的经济胁迫，美国不会让澳大利亚独自在战场上面对”之后，又有两个主要的发达国家站出来，对澳大利亚表示支持。同时，这也是一个信号——向受到中共经济胁迫的国家表明，抛弃中共，加入到以美国、英国为首的反共阵营，就会获得比中共更具实力的盟友，和更好的经济前景。因此，这使得这项协议具有更重要的政治和象征意义。

在中共不可协商的强势挑战之下，世界各国都面临着前所未有的经济与政治压力。为此，各方联合一起反制中共，是必然的选择。随着G7首脑峰会的结束，各界对中共的经济反制行动有了实质性的推进。看来，随着经济联盟和军事联盟的确立，面对不可预测及强势扩张的北京（中共）当局，西方民主国家对中共野心的容忍度越来越低了。
