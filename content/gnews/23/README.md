## 香港醫生梁子超：港全民自願檢測假陽性或高達3% 隔離者有被感染風險
`2020-08-23 08:45 GM01` [轉載自GNews](https://gnews.org/zh-hant/313400/)

原標題：【武漢肺炎】梁子超：全民自願檢測假陽性或高達3% 隔離者有被感染風險
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/23084229/skjd-1.jpg)新聞來自：蘋果日報
據蘋果日報2020年8月23日報道，香港呼吸系統專科醫生梁子超日前指出，根據政府早前為多個屋邨的居民進行篩查，平均3萬個測試有1宗陽性個案推算，假設假陽性比率為1%，及全民自願檢測經混樣、獨立及衞生署覆檢3次測試，最終假陽性比率或高達3%，情況不理想，也有可能令假陽性病人於隔離期間反有被感染風險。
