# **解密的战略文件揭示了美国加强了针对中共国的间谍活动**

[!\[\]()!\[\](https://spark.adobe.com/page/KSLuuAMOvqQPS/images/5a15fe6a-ff05-4b10-968f-ce160ee81de0.jpg?asset_id=158eb0dd-0085-4e6a-ab36-aa120a66b0a6&amp;img_etag=%2214c8a96dbfa31230a8da656baf23bde5%22&amp;size=2560)](https://spark.adobe.com/page/KSLuuAMOvqQPS/images/5a15fe6a-ff05-4b10-968f-ce160ee81de0.jpg?asset_id=158eb0dd-0085-4e6a-ab36-aa120a66b0a6&amp;img_etag=%2214c8a96dbfa31230a8da656baf23bde5%22&amp;size=1024)

美国总统竞选期间，由美国驻北京大使馆悬于酒店外的美国和中共国旗帜（美联社照片/AndyWong）

曾属机密的白宫战略文件揭示，美国已加强情报活动，打击中共国在美国和世界范围内广泛的间谍活动。

川普总统于2018年2月批准了该文件。文件呼吁保持对中共国的情报优势，以及“使美国、盟国和合作伙伴免受中共国情报活动影响。”

这份长达10页的文件题目为“美国印太地区战略框架”。其第一次公开确认了，美国中央情报局和其他机构一直在加紧针对北京当局的间谍活动。

这份战略框架文件称，美国还应向盟国提供装备“以展开行动应对中共国，并反制中共国在其本国的秘密活动。”

国家安全委员会(National Security Council)文件指出：“扩大并优先考虑美国情报和执法行动，以对抗中共国行为的影响力。”

该文件还呼吁美国情报机构“加强公共和私营部门的防御和进攻性反情报职能，以抵消中共国日益增长的情报优势；扩大与他国政府的情报外交和执法合作，以增进对中共国意图和能力的理解。”

与过去通过接触和贸易寻求改变中共国共产主义体系的政策不同，川普政府的框架呼吁保持美国的战略主导地位，并阻止中共国“建立新的、由其完全控制的势力范围。”

该框架规定了川普政府对中共国和北朝鲜的强硬方针。

[!\[\]()!\[\](https://spark.adobe.com/page/KSLuuAMOvqQPS/images/e11204c9-dcac-4e4e-9d48-95cc29bf05af.jpg?asset_id=d20e0e0a-fee8-496f-9c52-2e0f1476c3e1&amp;img_etag=%2245f1de7b0be9a6c60c62384c7e87c9e3%22&amp;size=2560)](https://spark.adobe.com/page/KSLuuAMOvqQPS/images/e11204c9-dcac-4e4e-9d48-95cc29bf05af.jpg?asset_id=d20e0e0a-fee8-496f-9c52-2e0f1476c3e1&amp;img_etag=%2245f1de7b0be9a6c60c62384c7e87c9e3%22&amp;size=1024)

一张组合照片显示，5名中国军官被美国指控从事网络间谍活动已于2015年被FBI逮捕。

川普政府执政期间，联邦调查局和司法部开展了强有力的高调行动，打击国内的中共国间谍活动，并逮捕和起诉了数十名在美中共国特工。

针对海外，中央情报局在白宫的指导下，协助盟国和合作伙伴发展强大的实力，开展反情报和反武器扩散行动，以及实现更好的网络安全、工业安全和机密信息管理。

从2010年开始，中央情报局在中共国遭遇了一次重大情报失误。

据对此次重大失败有所了解的情报人士称，由于中央情报局的变节分子以及联络被识破，该机构在中共国损失了超过二十名招募的特工。

中共国国家安全部和军事情报机构一直在从事大规模的人员间谍活动以及网络和经济间谍活动。

[!\[\]()!\[\](https://spark.adobe.com/page/KSLuuAMOvqQPS/images/545bb71b-923e-41ad-a69e-8cc1dd7e609e.jpg?asset_id=85b6cc21-0f08-4d94-8223-adbda3f7a35c&amp;img_etag=%224ccece740dc320839cf80e8542712272%22&amp;size=2560)](https://spark.adobe.com/page/KSLuuAMOvqQPS/images/545bb71b-923e-41ad-a69e-8cc1dd7e609e.jpg?asset_id=85b6cc21-0f08-4d94-8223-adbda3f7a35c&amp;img_etag=%224ccece740dc320839cf80e8542712272%22&amp;size=1024)

纽约市警察因帮助中国从事间谍活动而被捕。

这些间谍活动取得了令人瞩目的成功，它们窃取了美国的军事和私人部门机密，并将其应用于中共国军事和民用部门现代化。

军事方面，这份战略报告指出，美国将阻止中共国使用武力，并“发展能力和战术体系，在各种冲突中挫败中共国的行动。”

其中一条建议是要求军方“增强具有可靠战力的美军在印太地区的军事存在和姿态，在各种冲突的行动中挫败中共国。”

该报告要求实施一项防御战略，否认中共国在所谓的第一岛链（从日本向南延伸到南中国海的一连串岛屿）内的领海和领空控制权。

该文件还呼吁捍卫台湾等第一岛链国家，并对第一岛链外所有领域的冲突保持“控制力”。

对于台湾而言，政府的计划要求通过“有效的非对称防御战略和能力”，增强其作为自治岛屿的防御能力。这将使台湾军方“在与中共国接触中保持主动”。

美国政府已提议向台湾出售超过120亿美元的美国武器，其中包括能够跨越100英里的台湾海峡、打击中共国大陆目标的导弹。

对于北韩，白宫将平壤描述为“当前的急迫危险”，并集中力量防止北韩威胁美国及盟国。

该框架称，美国的目标是让金正恩政权明白，放弃其核武器库是其生存的唯一途径。

为了实现这一目标，美国政府立誓通过外交、军事、执法、情报和信息工具对平壤当局施以“最大的压力”，“阻碍北韩的大规模杀伤性武器计划，抑制货币流动，削弱政权，并以此施压北韩进行扭转其核武器和导弹计划的谈判。”

关于韩国，该战略要求提供更先进的武器装备，并致力于帮助首尔和东京当局更好地合作。

《印度时报》2020年9月报道逮捕3名为中共国进行间谍活动的人士

关于印度，该文件呼吁加速印度的崛起和提供区域安全的能力。

与川普总统对贸易和经济的强调相一致，这份美国国家安全委员会(NSC)的文件呼吁巩固全球经济领导地位，以及促进自由和公平贸易。

这份长达10页，未注明日期的文件以前由标记为“机密”，由疑为其作者的国家安全委员会亚洲总监博明（MattPottinger，音译马特·波廷格）加密。

博明先生后来成为了国家安全副顾问，并在上周亲川普示威者袭击美国国会大厦后宣布辞职。

国家安全顾问罗伯特·奥布莱恩(Robert C. O’Brien)说，该文件于2018年获得总统批准，并在过去的三年中被用作战略指导，以支持2017年的国家安全战略。

奥布莱恩在一份声明中说：“北京当局正在增加其施向印太地区国家的压力，要求其将自由与主权置于中国共产党设想的‘命运共同体’之下。”

他说：“美国的行为是不同的。我们力求确保我们的盟友和合作伙伴（在珍视且致力于维持印太地区自由开放方面看法一致的国家）能够维护和保护其主权。”
