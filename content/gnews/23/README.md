## “戰狼”變“獨狼”，中共正被世界拋棄
`2020-07-11 07:22 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/260138/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/11060248/Picture-1-66.png)圖片改於中共電影《戰狼》海報
近日，澳大利亞《悉尼先驅晨報》發布了一篇由英國《每日電訊報》國際商業事務編輯安布羅斯·埃文斯-普里查德（Ambrose Evans-Pritchard）撰寫的一篇評論文章，闡述了全世界選邊站隊結果表明，中共沒有具有全球經濟實力或信譽的盟友，正日益成為一隻顧影自憐的“獨狼”。

文章指出，儘管在聯合國人權理事會支持中共對香港實施“國安法”的國家有53個之多，但這些基本都是與中共合作“一帶一路”的專制小國，象古巴、巴基斯坦、緬甸、委內瑞拉、喀麥隆這樣的國家，他們的GDP總和只佔全球的4%。而澳大利亞，德國、法國、英國、日本、瑞士等27國則在對立的一面，反對中共這個損害香港自由、違反一國兩制承諾的法案。

憑著4萬億的外匯儲備，和對取代美國領導地位的渴望，中共從十多年前就開始了“戰狼”掠奪型外交策略。 *（注：例如在西方發達國家，中共使用“千人計劃”、“藍金黃”計劃等方式，盜取他國的商業秘密、知識產權，甚至盜取國家機密，來大幅提升中共在科技領域、軍事領域的水平和地位；對不發達國家，中共實施“一帶一路”計劃，通過向這些國家投入巨額的貸款修建基礎設施，換取他們的自然資源，乃至獲得有利的全球性戰略資源，並由於這些小國難於償還所借的巨款，中共還能進一步挾制他們聽命於自己，本次聯名支持中共香港國安法的國家就是一個例子。）*

以美國為首的西方國家開始逐漸意識到，中共的所有對外活動，無論是學術活動，還是民營企業的商業活動，都包含著中共不可告人的野心和目的。這次中共在香港“國安法”第三十八條把域外管轄權擴大到全世界任何地區的任何人後，向世界展示了中共稱霸世界後的可能景象。

所幸的是，中共還沒強大到世界無法對其製約。控制著國際經濟體系的西方國家和亞洲的富裕民主國家，開始形成一個統一戰線，在經濟、科技、甚至軍事等領域共同對抗中共不斷擴張的野心。

一方面，中共依靠進口技術、低成本勞動力形成的世界工廠模式已達極限，若想再進一步，必須要進入更高科技的技術前沿地帶，而這部分技術都掌握在發達國家手中，當中共的技術偷竊和強制技術轉讓的方式讓西方世界震驚後，他們只能就此止步，華為公司就是一個典型的例子。

雖然華為公司擁有不少5G移動通信方面的專利技術，和足夠低廉的設備成本，但它並未掌握製造芯片所需的微觀材料技術，也缺乏維持其在5G移動通信和即將到來的”物聯網”中佔據全球主導地位所需的關鍵原材料。當美國在5月份彈指一揮時，台灣台積電等芯片公司瞬間就切斷了對華為的芯片供應，讓華為稱霸全球5G市場的夢想頃刻破滅。

另一方面，中共國不斷擴大的產能需要不受限制的全球准入，但因為它那咄咄逼人的“戰狼”外交策略，讓它開始被一個又一個的國家拒之門外。印度上週以安全的名義禁止了TikTok和其他58個中國應用。美國則撤銷了中國移動、中國電信等多家運營商在美國的運營許可。英國全面禁止華為設備參與5G網絡建設。

越來越多的西方國家開始覺醒，即使被中共拉攏的小國也開始遠離中共，菲律賓、馬來西亞、泰國、科威特等多個國家宣布退出中共主導的“亞投行”，馬來西亞、巴基斯坦、尼泊爾、緬甸等國也陸續宣布取消或擱置“一帶一路”的項目，原因是評估這些項目的代價太大。

當全世界的民主國家都團結起來，聯手對付中共時，貪婪的“戰狼”只能變為一隻形單影隻的“獨狼”。
