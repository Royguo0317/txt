## 美國海軍陸戰隊副司令,四星上將加里·湯瑪斯確診CCP病毒
`2020-10-07 21:11 GM77` [轉載自GNews](https://gnews.org/zh-hant/409560/)

編譯:WKN
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/07210627/%E6%88%AA%E5%B1%8F2020-10-08-%E4%B8%8A%E5%8D%889.05.20-1.png)
「2020年當地時間10月7日」**美國海軍陸戰隊副司令加里·湯瑪斯將軍周三晚間宣佈,他感染了COVID-19病毒。**湯瑪斯是周二開始自我隔離的至少九位高級領導人之一,此前海岸警衛隊副司令查理斯-雷上將(Adm.Charles Ray)本周的疾病檢測呈陽性。

**根據美國海軍陸戰隊的聲明,湯瑪斯有輕微的癥狀,但在其他方面感覺良好。 他將繼續在家中隔離。** “海軍陸戰隊正在按照[疾病控制和預防中心]的指導方針,遵循COVID的既定政策,包括隔離和接觸者追蹤,”發布聲明說。 “根據CDC指南,任何與將軍密切接觸的海軍陸戰隊人員也將被隔離。”

**五角大樓首席發言人喬納森-霍夫曼周三晚間表示,他知道托馬斯的檢測結果呈陽性,但表示目前 「我們沒有其他高級領導人的陽性檢測結果可供報告」。**“我們將繼續遵循疾控中心的指導,進行自我檢疫和聯繫追蹤,”霍夫曼在一份聲明中說。

**海岸警衛隊副司令查理斯-雷上將(Adm.Charles Ray)本周檢測呈陽性後,湯瑪斯和其他8名高級軍事領導人進入隔離區。**這包括聯席會議主席馬克-米爾利將軍、副主席約翰-海滕將軍、海軍作戰司令邁克-吉爾德伊上將、陸軍參謀長詹姆斯-麥康維爾將軍、空軍參謀長查理斯-“CQ”-布朗將軍、空軍太空作戰司令約翰-“傑伊”-雷蒙德將軍、國民警衛局局長丹尼爾-霍坎森將軍和美國網路司令部保羅負責人-納卡松將軍。

湯瑪斯上周出席了五角大樓與查理斯-雷和其他高級領導人的會議。 據軍方發佈的消息,海軍陸戰隊司令官大衛-伯傑將軍正在英國弗蘭伯勒海岸的英國伊莉莎白女王號航母上參觀。 **海岸警衛隊官員說,目前還不清楚查理斯-雷在哪裡感染了COVID-19。**
