据《零对冲》(zero Hedge) 1月7日报道，安提法伪装成川普支持者“强行闯入”了美国国会大厦，整个国家都“吓坏了”。与经常在世界范围内的不合规国家策划的恐怖政变和暴力起义不同，这次的“暴力起义”夺取政府控制权的机会完全为零。华盛顿特区尽管知道抗议活动是有计划的，但还是选择不增加警力准备抗议活动，而且有镜头显示是警察主动让他们通过警察路障。

可想而知，这种事件使左派的评论家在社交媒体上对川普大喊大叫。同样可以预见的是，这也让他们呼吁对社交媒体进行审查。

《纽约时报》发表了两篇新文章，标题分别为《国会山暴动是在社交媒体上组织起来的》和《国会山暴力事件是社交媒体的报应日》，这两篇文章都认为硅谷科技巨头应该对言论进行更多的重拳限制。

在前者中，《纽约时报》的谢拉-弗伦克尔写道：”周三的暴力事件是在社交媒体网络中运作的在线运动的结果，人们相信选民欺诈和选举被窃取的说法。”他引用了 “塔西-加巴德是俄罗斯资产 “有名的智库旋转师雷尼-迪雷斯塔的专家分析。在这起丑闻中，阿拉巴马州参议员选举上演了俄罗斯干预的 “虚假指控”。

这种传播互联网上不受管制的交流将导致暴力起义的说法出自弗兰克尔之口。弗兰克尔说”对于世界各地的反政府示威者来说，社交媒体和消息应用程序已经变得至关重要，作为组织和向其他公民传递信息的手段，限制对此类技术的访问已经成为政府镇压的重要手段，就像警察的实际存在一样”。在他与其他人合写的一篇文中写道：“随着亲川普的抗议者周三冲进国会大厦，并停止了选举团投票的认证，Twitter、Facebook和YouTube等社交媒体公司在传播错误信息和成为川普先生的传声筒方面所扮演的角色再次受到批评。”文章还报道了美国总统因涉嫌用帖子煽动暴力而被暂时停止社交媒体特权，然后讨论了川普讨论区中流传的各种虚假信息和暴力思想。

《纽约时报》称“那些另类的社交媒体网站在周三充斥着川普支持者的组织和交流。在Parler上，一个热门的标签是#stormthecapitol(冲击国会大厦）。”还称网站上的许多川普支持者似乎还相信一个虚假的谣言，即左翼运动安提法（Antifa）对在抗议活动中实施暴力负责。

反诽谤联盟主任乔纳森-格林布拉特对《纽约时报》说，”我们知道社交媒体公司在阻止极端主义在其平台上发展方面充其量只是乏善可陈”。”言论自由不是煽动暴力的自由。这不属于受保护的言论。”

在未来几天，我们可能会看到更多这样的文章，主张加强对网络传播的监管，以防止未来发生类似今天的事件。就其本身而言，这对普通公民来说，听起来不会有太大的影响。采取措施防止人们在社交媒体上策划暴力和恐怖主义没有错吧？

但你如何预测哪些抗议活动会是 “暴力 “的？你如何决定哪些抗议活动和哪些政治异议需要被审查，哪些应该被允许自由交流？你是让硅谷巨头来做主吗？还是让他们像一直以来那样与政府协商？你会信任这些机构中的哪一个，来监管哪些抗议活动可以被允许在网上组织？

美国的实际权力结构似乎只对审查互联网完全消除政治异议感兴趣。

2017年，来自Facebook，Twitter和Google的高级官员被带到参议院司法委员会，并被告诫要提出一些政策，以“防止在美国煽动不和”。世界社会主义网站在2017年报道了以下内容。

夏威夷民主党参议员Mazie Hirono要求这些公司通过一份 “任务声明”，表达他们 “防止煽动不和 “的承诺。证词最实质性的部分发生在听证会的第二部分，期间大部分参议员已经离开，美国情报机构的两名代表在一屋子大部分空椅子前作证。

前美国陆军军官、前联邦调查局特工、”保障民主联盟 “成员克林特-沃茨发表了以下世界末日的宣言。*“内战不是从枪声开始的，而是从语言开始的。美国与自己的战争已经开始了。我们现在都必须在社交媒体战场上行动起来，平息信息叛乱，因为这些叛乱会迅速导致暴力对抗，很容易变成分裂的国家。他还说：”只有当那些传播虚假故事的渠道被压制后，才能阻止社交媒体用户听到虚假信息。让枪声发不出声来，战火就会结束。”*

这听起来很可怕，像是政府官员和特工在告诉社交媒体公司，审查可能会助长任何形式的动乱的通讯信息是他们的工作。您是否相信这些垄断性的大型公司决定人们的异议言论是否可以接受？我不相信。朱利安·阿桑奇（Julian Assange）被判处以错误的监禁，而大众媒体也加大了对信息帝国的控制，我们现在正在为自己的主权而战。

现在的主流媒体都是左派的天下，言论审查的受害者无数，包括川普总统，1月6日他被推特和Facebook限制不能发声。川普总统让和平抗议者保持秩序回家的视频被删除并贴上“煽动暴力”的标签。用230条款为保护伞，由垄断型媒体巨头按照他们的政治立场来决定压制不同声音后果严重到可以颠覆国家，我们要一起站出来反抗，取消他们滥用的230条款，保护民主和言论自由。
