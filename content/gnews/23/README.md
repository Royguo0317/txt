《布雷巴特新闻》1月21日报道，新上台的拜登政府暂停了联邦石油、天然气许可证，旨在审查上届政府的传统能源政策对气候、野生动物保护区所造成的影响。拜登政府命令各机构审查川普政府的其他100多项环境指令，包括取消了利润丰厚的加拿大到墨西哥湾的Keystone XL输油管道。

该管道长2,735公里，每天可从加拿大艾伯塔省运送大约80万桶石油到德州的墨西哥湾。拜登的行政命令称，“保持 Keystone XL 管道许可证不符合我国政府的经济和气候要求”。美国石油管线协会首席执行官安迪·布莱克表达了不同的看法，他说，拜登的新命令无异于“扼杀(建设管线)1万个工作岗位，从工人口袋里拿走了22亿美元的工资，而且这不是美国人现在需要或想要的（政策）”。加拿大总理特鲁多也表示，“我们感到失望”。

川普在4年前参选总统时的竞选口号是“把产业带回家”。前苏联解体后开启的全球化，使美国的大量资本及中低端产业离开美国，主要原因除了美国的资本主动寻找和欠发达地区的低廉劳动力结合，还有美国的能源不能自足，需要消耗大量能源的中低端产业被迫外迁。产业空心化的后果是美国劳动阶层大量失业，贫富分化程度快速达到50年以来新高，美国国家财政下降，国债快速上升。

川普4年前之所以转败为胜，就是因为他的竞选口号切中美国的大问题，因而得到劳动阶层的选票。川普上任后信守竞选承诺，立即进行了战略重构，其中的能源自足是战略重构的基石。川普放开奥巴马对传统能源产业的束缚，在勘探开釆、管路建设、石化产业等方面釆取扶持和鼓励政策，并大力引导资本进入，仅用短短的3年，美国的石油产量快速上升到1300万桶(2019年)，实现了能源自足，从而吸引了大量产业回归，至2019年底止，美国的失业率创下50年来新低。但这触动了利用美国的低关税，在世界各地尤其中共国使用奴隶劳工生产廉价商品，再倾销到美国的既得利益集团的利益。

在中共病毒肆虐，全球经济处于崩溃边缘，美国的传统能源产业一片哀嚎的情况下，拜登政府为了保护野生动物，为了远未达到成熟应用的清洁能源产业，为了重回肮脏的巴黎气候协定，竟然重手打击为美国经济托底的传统能源产业，确实令人扼腕和费解。
