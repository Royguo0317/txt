黎智英：我認為現在是最好的時機，整個世界，尤其是美國都在對抗中共，這是改變中國的時刻，世界也將被改變，中共與普世價值背道而馳，如果中國不改變，我們將不會有和平。中國需要擁有自由世界的價值觀。

主持人：但是黎先生，如何能改變中共呢？

黎智英：制裁他們，在美國和全球範圍凍結（中共）貪官貪污的財產和帳號，制裁中共的技術產品或停止向中國輸送技術產品，美國可以通過多種方式向中國施壓，使其改變，中共在自己的國家裡為所欲為，但是在國際上，他們不得不守規則。

主持人：您會被捕嗎？

黎智英：會的。
