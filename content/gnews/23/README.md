04月01日，克里姆林宫新闻处报道，俄罗斯总统普京与沙特王储穆罕默德·本·萨勒曼·沙特（Mohammed bin Salman Al Saud）通了电话，并讨论了气候变化和绿色能源的发展。

声明称双方“特别关注了在全球气候变化和环境保护问题上的互动前景，包括绿色能源。”

王储通报了发起的环境倡议，这些倡议旨在减少该地区的温室气体排放，绿化各领土并增加可再生能源的比例。

这次谈话是在沙特方面的倡议下进行的。俄罗斯领导人指出了这些举措的意义，并谈到了俄罗斯当局在环境领域所作的努力。

普京与王储的另一次电话交谈发生在2月份。他们讨论了以OPEC+形式达成的协议的执行情况，以及向王国供应俄罗斯疫苗的可能性。

《消息报》3月份援引消息人士的报道，俄罗斯政府同意在2025年至2035年期间减少对可再生能源（RES）发展的支持力度。根据该报报道，减少的幅度将达到22%，从4000亿卢布减少到3130亿卢布。
