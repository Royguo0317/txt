## 上访者的悲歌——请停止向魔鬼的献祭

#### 作者：xinyuan 审核：pv0 / Peace Wind
![]()![](https://gnews.org/wp-content/uploads/2021/03/ebd7aac82511dd75.jpg)
今天看到大纪元一则报道，是上海一位90岁的访民刘淑珍，因房屋被强拆导致权益受损、至今已持续16年未获解决，3月3日上午，在市政府领导接待日当天，她手举诉状，在市政府上访，遭遇到十余名警察粗暴对待，其中一个警察竟然冲过来掐住老人脖子，抢走诉状，最后还把老人和其女儿崔福芳同时带走，现场一位拍照者也被同时带走。在警车上，老人与之理论，警察却无理地谩骂老人，还逼迫崔福芳说出手机密码，进入其手机，把其拍照的视频删除。

看到这篇文章时，真的是让人很愤怒又无助。愤怒的是警察的蛮不讲理。无助的是我只能眼睁睁看着这样的事情一而再，再而三地发生，却无能为力。

在中共治理下的中（共）国，类似的剧本真是层出不穷，多如牛毛。记得多年前，在一个夏日的街头，大概8点的样子，很多人围着一个坐轮椅的老人。他残疾得很厉害，即使坐在轮椅里，背也是高高拱起的。当时的他正在哭诉自己的家事，周围的人你一言我一语地劝说着，主要意思是劝他赶紧回家，不要再折腾了。可是他固执地坚持要继续下去，还天真地认为自己还是没找到该找的那个人，否则自己的冤案一定会被解决的。

最后，周围的人都无奈地摇摇头，散去了。我默默地看着这一切，我无法想象他一个行动不便的老人要怎么走完这上访之路。老人的毅力虽然让我佩服，但他决绝的固执却让所有人无能为力。人们只能担忧地看着他，万一他摔倒了怎么办？万一轮椅坏了怎么办？他的亲人会不会担心地无法入眠？这一切，他都不考虑吗？明知这会要了自己的老命，也明知这会让亲人寝食难安？

老人还是要继续他的执着。那意思真是要死磕到底了？可他拿什么拼？对于那个一无所有的老人来说，他只有残破的轮椅，和一具随时会倒下的残躯。常年的上访之路，风餐露宿的各种折磨，巨大的心理压力，是他看起来就像风中随时熄灭的蜡烛。至今想起那位老人，我都会感觉无比的伤感。

这类悲剧实在是太多太多了。从某种意义来说，他们不畏艰险，无惧强权，好像都是英雄般的存在。可是，从另一方面来讲，他们却都有一个根本性的误判，那就是他们完全低估了中共政府的邪恶！也完全不相信自己是永远也不会得到想要的结果的。这种误判让他们的上访之路也变成了献祭之路。祭出了自己的青春和快乐，也祭出了自己的幸福和人生，外加搭祭上亲人的安宁！这最终成就了亲者痛仇者快的可悲事实，这是何等的悲剧人生！

忽然想起了文贵先生讲到他在山东济南信访办的一段讲话，内容如下：我看到老乡们，有的人为一件事上访达十年。先不管对错，可以算下帐，生命里有多少个十年，18岁前不懂事，是未成年人。45岁以后人逐渐走下坡路。不管男女，到65岁基本是半废调整状态，所以45到65岁之间也是犯罪的高峰期，因为要抓住生命的尾巴。从18到45岁是20多年，从45到65也是20多年。这20年7000多天，2个7000多天是15000天。这15000天没病没灾还有饭吃的情况下，是人生最好的时间。可你一上访3年，5年，10年，浪费掉1000天或者1万天，这个世界上受的最大惩罚就是你，不管对与错，你把生命耗在了路上，绝不证明你执着，也不证明你坚强拿5年去干这种事，因为你上访的地方，你不是在跟正义打交道，也不是跟别人打架，你是在跟自己打架，你在跟完全没影子的鬼打架，在流氓共产党那里能找到真相吗？那是不可能的，与其花3或5年干这种事情，不如把更多的时间用在生活上，让自己强大。当自己有机会的时候，因为你有能力了，会站的更高，实现你的梦想。世界上最可怕的事就是浪费自己的生命。假如有一天，我惩罚敌人时，我一定耗死他，如果有能力把敌人送进监狱，我一定让他活着，让他看到我强大。过的好。从爆料革命第一天我就告诉过大家，让敌人最痛苦的事情就是你活的比他好。你比他强大！

在中共即将灭亡之际，我把文贵先生的那些话献给所有正在上访和即将上访的朋友们。希望能让那些疲惫的灵魂停下脚步，多一些思考，早日回家与亲人团聚，等待中共灭亡之日的到来。也让准备上访的人们停止无谓的牺牲，保护好自己和家人，当中共灭亡之日，我们再和它们算大帐。

毫无悬念地通知各位上访的朋友，中共即将灭亡，明天无限美好，请珍重！
