根據**【Now新聞】**報導，美國國務卿蓬佩奧在10月5日接受日本放送協會專訪時提到香港問題。他再次批評中共拒絕履行香港“一國兩制”和“50年不變”的承諾。

美國務卿稱，中國共產黨拒絕履行承諾——對香港人的基本承諾，在50年內允許一國兩制，（讓香港人）享有一點點自由。中國共產黨撕破這一點，這是關乎自由或暴政。

**戰友點評：【文悅】**
     美國國務卿蓬佩奧近日出訪亞洲，與日本、美國、澳大利亞和印度外長在東京舉行四方安全對話。國務卿蓬佩奧在會上就南海、東海等問題譴責中共，呼籲與會各方攜手應對中共威脅。世界各地正義力量已對中共國逐漸形成合圍之勢。而且我們很欣慰地看到，美國務卿已經把“中國共產黨”和“中國14億人民”分割開來，這恰恰是爆料革命的巨大成果！
