## Delta 病毒变种感染者的病程更艰难

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/09051543/taylor-brandon-jYI7L4zLFkM-unsplash-scaled.jpg)
迄今为止，冠状病毒的 delta 变种在德国几乎没有引起大面积感染。来自英国和印度的数据表明，这种突变肯定在增加：它增加了住院风险，并且比其他变种更具传染性。

根据英国卫生当局的初步调查结果，最初在印度发现的冠状病毒的 delta 变种可能会更频繁地导致更严重的 Covid-19 疾病。 “英格兰和苏格兰的初步调查结果表明，与 alpha 变种相比，入院风险可能更高，”英国公共卫生部最近的一项风险评估显示。变种 B.1.1.7 最早在英国发现，被称为 alpha 变种。据当局称，一些地区的住院人数有所增加，但尚无法全国趋势估计。

然而，已经有更明确的证据表明 delta 变种可能比 alpha 变种更具传染性。根据各种分析，英格兰公共卫生局假设“增长率大幅增加”的可能性很高。来自印度的一项预印本研究也涉及 delta 变种。来自新德里国家疾病控制中心的研究人员等假设可转移性比 alpha 变种高 50%。因此，delta 变种可能会导致非常迅速的爆发，这也会感染已接种疫苗的人。

简评： 全世界都把希望寄托在疫苗上了， 现在新的病毒变种Delta来了， 它的传染性更高， 感染者的病程更长， 这就意味着死亡率更高。 并且还会传染给接种疫苗的人。 那么就是疫苗根本就没有保护。 更别提疫苗那些未知的副作用。 人类遇到的这次挑战是谁引起的， 人类引起的这次灾难该不该追责？
