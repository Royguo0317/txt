## 天賦人權，自由之花需要我們一起來捍衛！
`2020-06-13 03:10 GM30` [轉載自GNews](https://gnews.org/zh-hant/232472/)

**作者：Michael Gabriel**

共産主義幽靈在産生之時就在歐洲遊蕩，但歐洲沒有給予這種幽靈落地生根的土壤，“共産共妻”信念是共産幽靈的靈魂，滋養這個惡魔的土壤是一個社會必須有廣大的窮人，即所謂的無産階級，“共産共妻”富人的財富和女兒對這些一窮二白的窮人來說有巨大的誘惑性，貧窮的俄國和中國爲滋養共産主義惡魔提供了天然的土壤。共産幽靈來到中國催生了CCP惡魔，CCP不僅繼承了“共産共妻”的邪惡信念，而且傳承了中國曆史上的皇權馭民邪術，集古今中外邪惡信念及手法爲一身的CCP惡魔奴隸了14億中國老百姓，其野心並不滿足于統治中國，CCP要在全球建立人類共同體，推行其“共産共妻”信念和馭民邪術。

共産主義披著民主的外衣，以“共産共妻”爲旗幟欺騙忽悠了社會的底層人民跟著鬧革命，等通過流氓手段奪去政權之後，整個國家的“財富共有化，權力私有化”，老百姓失去了所謂的民主，被代表並成爲了CCP的奴隸，權力集中于CCP幾大紅色家族，幾大紅色家族成爲了奴隸主，他們掌控了國家的財富，並且享有各種特權尤其是“刑不上大夫”。爲了保持CCP血旗不倒，幾大紅色家族的奴隸統治千秋萬代傳承下去，CCP必須對外推廣其“共産共妻”的國際共産主義，使全球紅色一片，人類民主自由湮滅。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/13030537/image0-101.jpg)

CCP邪惡無恥，人神共憤，其血旗不倒，在國內是有一定策略的：一是從肉體上消滅民主自由，一場場的政治運動和迫害屠殺，包括對外戰爭和8964，從肉體上消滅有自由民主追求的人們；二是從精神上閹割民主自由，從對宗教人士的迫害到對各種教義的修改，從文化大革命到全國教育要把思政教育放在首要位置來重視，從莫談國事到網絡禁言，從媒體宣傳有紀律到影視播放有傾向，在所有的精神文化領域CCP徹底消滅了國內民主自由的種子。三是消滅一切追求民主自由的民間組織，CCP靠組織一群窮人起家，非常迷信組織的力量，在CCP統治的國度不准任何民間組織的存在，即使個別的存在也是CCP黨組織的一個馬甲，用法律來剝奪老百姓示威遊行的權利，因爲CCP非常清楚自己的邪惡高壓統治，只要有民間組織興起，出來振臂一呼，CCP就土崩瓦解。四是杜絕民主自由之光的外部輸入，建立現代網絡長城，CCP對國內的愚民洗腦教育，使得全體國民都認同了這個體制，但現代互聯網技術的發展致使全球信息互通有無，CCP築起了人類曆史上最臭名昭著的防火牆工程，把14億老百姓用防火牆圈在奴隸牧場中。

CCP泯滅人性，但以往在國際上招致的反對聲音不多，也是有一定策略的：一是用中國老百姓的血汗錢來收買亞非拉的小國窮國，讓他們爲CCP搖旗呐喊，進而獲得國際社會的認同，爲其竊取政權的合法性來粉飾。二是控制國際組織機構，通過各種藍金黃手段收買控制國際組織機構的工作人員，讓這些人員爲CCP賣命，通過國際組織機構改變國際規則和秩序或是讓現有的國際規則實效，讓有利于CCP在國際上推廣國際共産主義。三是藍金黃西方發達國家的政客，讓其對CCP的惡行視而不見，不僅如此這些政客還要支持CCP的國際共産主義，幫助共産國際在其國家落地生根。目前來看，一帶一路以來，許多亞非拉的窮國已經跟隨了邪惡的CCP，CCP對這些國家的傀儡控制已經邁出了實質性的一步，在其國土建立軍事基地、修建鐵路，讓這些國家欠下巨額債務，其官員成爲中共的傀儡。而西方的一些民主國家也淪爲中共的幫凶，一些國家的政客想在中美之間玩政治平衡，對香港問題態度不明，對antifa騷亂火上澆油，對全球的滅共號召置之不理。

美國是民主自由的典範，是二戰後國際規則制定和推動的領頭羊，是人類自由之花的守護者，由于其強大的經濟、軍事、科技、文化等實力的存在，一直是CCP推行國際共産主義的威脅，也是CCP極權存在的威脅，爲了應對美國，CCP有單獨的全方位策略：一是挖空美國企業，以各種優惠條件招商引資，讓美國企業在中國可以賺取超額利潤爲誘餌，誘騙很多企業搬遷至中國，主要目的是讓美國失業人數增加，壯大無産階級的人群，美國橄榄型社會變成金字塔型社會，中産階級人數比例下降，這爲國際共産主義幽靈落地生根創造了土壤。二是輸入毒品和假美鈔，CCP通過各種途徑給美國的窮人帶來毒品和假美鈔，使他們具有了對CCP的人身依附性，通過了各種類似antifa的組織收編了美國墮落潦倒的窮人，以備策動政治騷亂做准備，這些人和平時期聽從CCP的指揮用投票選舉有利于CCP的政客上台，非常時期以暴動手段推廣共産主義奪取政權。三是藍金黃手段收買政客和財閥，收買或者培養有共産信仰的政客，讓這些政客弱化對華制裁政策，同時爲國際共産運動網開一面，當美國底層窮人反政府的時候，這些政客將會支持暴亂。至于對華爾街財閥的收買則是讓其投資中國奴隸工廠，賺取超額利潤，而且還不時通過各種換器官、雙修幼女等反人類的行爲對其個別者進行腐蝕拉攏。四是偷竊美國科研技術，美國硬實力的強大在于技術創新，CCP通過學校交流、引進千人計劃、留學生、科研投資等手段全方位地竊取美國技術，像華爲這樣的公司都是執行機構，把偷竊來的技術包裝上市搶占市場先機和市場份額，以低廉優惠産品幹掉美國的同行企業，讓美國依賴中國的産品。五是監控美國，用類似華爲生産的通訊産品來監控美國，用技術後門和黑客盜取美國所有的通訊技術，用大數據分析來監控美國的政治和經濟動態。六是進行意識形態顛覆美國的民主自由理念，在美國推廣孔子學院，推廣三綱五常極權思想，通過大外宣和收買的美國媒體來進行國際共産思想宣傳，當然宣傳有策略不是直白的，美國的言論自由媒體自由被邪惡的CCP有組織的利用來宣傳煽動共産主義意識形態，尤其在社交媒體CCP用許多僵屍賬號挑撥美國政治內鬥，以便形成有利于CCP的輿論。
