关于正在进行中的阿拉斯加的中美外交谈判，郭文贵先生在3月19号的盖特中解读了中共如何玩弄种族矛盾并提出应对的建议。

**中共根本没资格指责美国的种族问题**

在3月18号的郭先生的盖特中，郭先生一语道破中共最害怕最担心的就是美国把它定为反人类罪、种族大屠杀，中共其实想通过阿拉斯加谈判来解套。

所以王毅在谈判中上来就强硬批评美国的黑人种族问题。以一种美国和世界没资格管我中共正在进行的“种族灭绝”的强盗流氓逻辑在阿拉斯加撒泼打滚。郭先生说，“中共这回打的招儿叫种族歧视、种族大屠杀，就是美国你杀黑人。”

郭先生接下来列数了中共在中国国内犯下的种族灭绝罪行。在西藏杀的人，新疆杀的人，还有杀的汉人、香港人，还有50几个其他民族，包括云南那几个小民族。历史上像当年在大理下关水傣和汉傣，那杀了多少人！郭先生激愤地说，“我说他（中共）杀的任何一次零头，都是你美国历史上这200多年的总和，他（中共）没有任何资格指责你（美国）所谓的种族问题！”

而且，郭先生强调，“中共正在扼杀，正在屠杀。而且此时此刻正在屠杀，这就是我说你们美国人要明白，共产党跟你玩的是内部搞的种族大屠杀！”就是挑动美国内部种族矛盾，让美国不同种族之间进入仇恨和相互战争状态。

**中共的目的是搞种族大革命输出，挑起世界对白人的种族仇恨**

郭先生建议好好看一看印度的电影《白虎》，这是共产党真实的，向世界搞文化大革命，文化大革命输出。他说，“搞种族大革命输出，（挑起）种族仇恨，然后最重要的把世界上所有的问题，历史的全归你白人，我说打你们的朱利安尼，打你川普，打你班农，打你美国所有的右派和极右派，打你共和党，所有的一招儿，你（右派）就是种族歧视。”

**正在施行种族灭绝的中共没有资格指责世界存在的种族歧视**

然而，郭先生点醒美国人，中共在国内历史上和现在干下的种族灭绝，和美国的种族歧视实在差太远了，有着本质的区别。在郭先生的提醒后，美国人也醒悟过来，一个搞种族灭绝的中共怎能有资格来指责美国社会或是全世界少数地区存在的种族歧视？

郭先生提醒美国人，“我说是你们该反击的时候了，到底种族歧视存不存在？到底种族大屠杀这俩之间是什么关系？哪个重要？关键问题你（中共）现在还在干，我说这就让全世界来调查调查吧，让把全世界来评评理吧，我说你干嘛不利用这次机会啊？”

**中共内部斗争需要一场强硬的外交表演**

除此之外，中共的外交表演，有中共国的内部政治需求。郭先生说，“我说这个杨洁篪还有王毅，他俩都想争下一个常委，都想当管外交的常委，杨洁篪要退，王毅要上，他俩拼命的表现胳膊硬，擀面杖子硬，是吧？”

郭先生一针见血指出，“（中共的外交）出发点全是内部政治斗争，是上海帮和习大神的斗争，你去看这两个坐在那的时候，看上去执行的都是你中央的命令，事实上都想的是明年谁来继续干，杨洁篪继续干，还是王毅上来，当所谓的管外交的委员，共产党一撅啥屁股，就知道它拉什么屎。”

中共不但杀少数民族，也杀汉人，中共根本就是灭整个华人民族。“为了中共的统治，中国人死一百万，一千万，一个亿，对他来讲，那简直是太小事了。” 郭先生特别强调香港，他说，“更重要的事情在香港现在发生的事情，他（中共）在大屠杀，汉人是不是少数民族？汉人还是大族民族，杀少数民族你当然是大罪，杀汉族难道就不是罪吗？杀人都是罪。”
