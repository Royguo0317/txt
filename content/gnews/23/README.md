## 《路德社》精选新闻简讯：欧盟同意制裁中共官员八九六四后首次
` 路德社精选` [轉載自GNews](https://gnews.org/zh-hans/989014/)

出品：《路德社》 编辑：《路德社》精选新闻专栏组

《法广》3月17日援引《路透社》发表题为＂欧盟同意就新疆人权问题制裁中国官员及实体＂报导，

![]()![](https://www.gnews.org/wp-content/uploads/2021/03/1-90.png)

欧盟外交官同意就新疆人权问题制裁4名中共官员及一个实体，这是自1989年六四事件后欧盟对华实施武器禁运以来，对中方的首次制裁。

欧盟外交官告诉路透社记者，中国官员被指控侵犯维吾尔族穆斯林少数民族的人权，此举反映了欧盟，美国和加拿大对维吾尔人状况的深切关注。

人权活动家和联合国人权专员说至少一百万新疆穆斯林被关在了集中营，这些人在集中营中遭受酷刑，强制劳动及绝育！

据悉，这次被制裁的对象总共有11个，除了中共，还包括俄罗斯，利比亚，南苏丹，北韩，所有名单将会在3月22日正式批准后公布，德国外交部长马斯（Heiko Maas）3月15日指出：通过人权制裁，我们创造了惩罚侵犯人权行为的机会，我们显然想利用这一机制。欧盟去年年底通过了欧洲版《马格尼茨基人权问责法》，今年二月首度制裁俄罗斯官员！

中共国驻欧盟大使张明则在推特上叫嚣：制裁是对抗行为，并说：“我们希望对话，不希望对抗，请欧方三思而行，若有人持意搞对抗，我们为了＂人民＂，唯有奉陪到底＂。

欧盟对于中共官员的制裁，标志着对华政策的一大转变！
