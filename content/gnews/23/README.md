**西班牙2021年3月4日电/西喜社——**据独立报3月4日报道，中国的一项重要研究显示，中共国政府的目标是“降低维吾尔族的人口密度”，并通过给新疆成千上万维吾尔族和其他少数民族提供新的工作机会—通常是远离家乡的工作机会—来同化少数民族文化。

中共国政府否定了任何这种改变该地区人口结构的目标，并强调工作岗位转移的目的是提高他们的收入，使少数民族摆脱贫困。

然而，根据研究报告，政府的想法是通过改变维吾尔族人的生活方式和思维方式，将他们同化到大多数汉族文化中，BBC新闻在研究了这份研究报告后报道说，这份研究报告于2019年12月被中共国政府意外地放在网上，但几个月后被撤下。

例如，2017年的一个视频报道，从政策看到密集实施的时候，显示了当地实际发生的事情。英国广播公司新闻称，中国共产党开办的新闻频道长达4年之久的视频报道，直到现在都没有在国际新闻中出现。

视频显示了新疆地区一个村庄的政府官员如何在两天多没有一个人上班的情况下，不得不挨家挨户地走访。

视频显示，官员们向一位父亲施压，要求他送19岁的女儿去参加工作。当这对父女拒绝时，官员们继续向他们施压，直到她最终让步，说 “如果别人去的话 “她就去。报道的最后，这位年轻女孩和许多像她一样的人告别了家人。

这篇新闻报道是在北京因新疆地区少数民族的状况和他们在集中营的状况而面临国际社会的巨大压力之际发表的。中共国一再否认这些指控，尽管根据目击者的描述，有报告显示在北京声称是职业培训中心的营地里发生了酷刑、殴打和强奸事件。

3月2日，中国外交部发言人汪文斌说：”新疆现在社会稳定，发展势头良好，当地人民过着安全幸福的生活。”

“新疆所有居民充分享有生存权和发展权等权利。新疆各少数民族的语言、传统文化和风俗习惯得到了很好的保护和继承。60多年来，新疆经济增长了200多倍，人均GDP增长了近40倍，居民预期寿命从30岁提高到72岁。”汪文斌称。

然而，由南开大学一批学者撰写的中国研究报告指出，劳动力转移是 “影响、融合和同化维吾尔少数民族的重要方法”。他们强调，他们搬迁到中国其他地区，实际上 “降低了维吾尔族的人口密度”。

中共国外交部驳斥了这份研究报告，称报告反映的 “只是作者的个人观点，很多内容与事实不符”。
