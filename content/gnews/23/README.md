## 从《论法的精神》谈为什么中共无法颠覆现有国际清算体系

撰稿：无王无帝定乾坤
![]()![](https://gnews.org/wp-content/uploads/2021/02/图片.png)    图片来源与网络
近期，环球同业银行金融电讯协会（SWIFT）联手中共央行清算总中心、数字货币研究所等成立合资公司——金融网关信息服务有限公司。其目的是为推动中共人民币与央行数字货币国际化，绕开美元结算体系。去年中国四家大型国有银行已经在深圳等城市对央行数字货币进行大规模测试。

此前，中共伪政府喉舌新华社曾发文：港交所总裁李小加表示，要加快人民币国际化巩固香港国际金融地位。在去年中共利用大外宣在西班牙《真理报》宣传央行数字货币将动摇美元在国际结算体系里的霸权地位。我借用孟德斯鸠的思想谈谈为何以上言论为毫无根据的无稽之谈。

孟德斯鸠著作《论法的精神》中谈到：“英国大宪章规定，当债务人的动产或个人物品足以抵押而且愿意抵押时，债权人不得扣留债务人的地产或收入。这表明英国人的所有财产都等于货币。当法律偏袒债务人时，他的物品就不代表货币，也不代表货币的符号。专制体制下，由于不信任，所有人都把包银（货币）深埋于地下，物品因而就不代表包银（货币）。”——可见货币与其代表的物品价值相等，因而可以任意拥有其中的一种，在这种情况下，国家必定兴旺发达。反观CCP伪政权的人民币，1994年货币改革后，发行人民币是以外汇储备为锚，即以美元储备为锚。

近年来因中共滥发货币，对内暴政统治、债务爆雷、侵略香港，对外贸易战、被美国制裁、生物武器袭击全球等诸多因素导致中共国经济下滑，创汇能力急遽下降，但中共仍然维持人民币对外升值。此外还伴随着人民币在墙内的购买力逐渐下降、物价上涨，还伴随着老百姓收入下降、债务压力增加。近期出现了北京燕郊房价暴跌，业主送房既不要钱也不要房的奇葩现象。中共国内由于缺乏赚钱效应，导致货币流通速度变慢，物品商品变现能力下降，再加上中共对人民币取现钞以及自由兑换外币的限制，更加使人民币价值下降。

人民币并非在法治的环境下发行与流通，在限制流动与兑换中，中共可以随意操纵货币，并剥夺人们的私有财产。所以由于墙内觉醒的人已对中共主权信用失去信心，也对人民币失去信任，才会出现大量抛售资产变卖房产与企业，变现后兑换美元等外币将资金转移至海外的情况。货币对内贬值对外升值是有违正常经济规律的，人民币真实价值早已与官方的虚假汇率脱钩。所以在中共国内，人民币货币已经不等于其代表的物品的价值，物品也不代表人民币，也不代表货币的符号，这中形势下的政权也将走向衰亡。

从人民币国际结算地位来说，孟德斯鸠提出：“为了从源头上消除弊端，所有希望发展贸易的国家最好制定一项法律，规定只需使用真实货币，禁止任何会使真实货币变为虚拟货币的作法。一切物品共同的计价手段最不应该发生变化。贸易本身具有很大的不确定性，在事物性质所决定的这种不确定性上面，如果再加上另一种不确定性，那就很不好。”

美元最为国际贸易的结算货币有长期的历史，这不是强制形成的，而是凭借美国主权信用的保障，由市场选择了美元。用无信用基础的“虚拟货币”改变原有的国际结算货币，会造成国际经济与金融体系的风险，这是危险的做法。

在我看来，孟德斯鸠先生所谈的虚拟货币，并非仅包括我们今天所认识的数字虚拟货币，还包括一切没有法律与制度保障、不能真实反应经济现状与商品物价以及随意超发的货币。人民币不但具备以上特点，而且已沦为中共统治的工具。中共伪政权在对墙内以及对国际逐渐失去信用的背景下，加之大额人民币不能自由取现兑换外汇。从这些角度讲，人民币已经属于“虚拟货币”，以虚拟货币（人民币）为锚的虚拟货币（央行数字货币）会更弱于真实货币。2020年3月美国与九国央行或金管局建立了货币互换协议，旨在共同维护货币市场的稳定，而其中并无人民币，可见闻名世界国家对于人民币的不信任。这样的虚拟货币更不可能国际化替代真实货币进行贸易结算。

前期中共谈到数字货币不能兑换黄金和外汇，可见中共的数字货币根本不具备包括自由流通、自由兑换、储存等货币应有的功能，这正是孟德斯鸠笔下的“虚拟货币”。当墙内一旦对它失去信心，就会大量出现应数字货币换取商品储存的现象，导致数字货币价值巨贬、物价上涨与恶行通货膨胀。如果商品价格出现出现数字货币价格高于人民币价格，那么人们会选择纸币现金交易甚至在极端情况下出现外币交易或以物易物。

人民币已沦为无信用的“虚拟货币”，以及以“虚拟货币”为锚的央行数字货币来挑战国际结算体系中美元的地位，相当于用“虚拟货币”来挑战真实货币，这是在给国际结算体系增加不确定性，ccp试图搞乱并控制各国家结算体系，这是在挑战世界秩序。最终不但不能使人民币国际化，反而将加速与文明世界货币体系的脱钩。
