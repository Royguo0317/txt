日本福岛核电站因**10**年前的地震伴随海啸发生了最高级别的**7**级核事故，反应堆堆芯内所有核燃料全部熔毁并泄漏，这起事故造成了严重的次生灾害，受灾区域面积达到**337**平公里。4月13日日本政府宣布，从现在开始计划在两年内向大海排放超过100万吨已经过处理的放射性废水，立即遭到了中共战狼**+**泼妇的围攻。

核事故后的**10**年以来，日本没有偷偷排放废水，而是在国际原子能机构的监督下进行净化。日本首相菅义伟于本周二郑重其事地表示：在排放之前，水箱中的水将充分稀释，使氚的浓度大大低于符合国际标准的日本国家监管标准。水中氚释放微弱辐射，但对健康的风险“非常低”。其他日本官员表示，排放是在国际原子能机构（IAEA）监督下进行的，以前已经有类似的排放，国际原子能机构认可不会对环境造成不利影响。

海洋环境的好坏事关全人类的福祉，任何国家都有权利和义务维护海洋的安全，并对日本核废水排放的安全性进行验证，合理途径是要求国际原子能机构或日本政府递交报告，提供一切便利通道和化验资料进行核查监督，甚至派出独立机构到现场取样化验核实，一旦发现问题立即叫停并依据国际法追加相应处罚。

但是我们所看到的中共，不是依法去挖崛事实，然后就事论事，而是釆取一贯的流氓手段进行无端攻击。中共官媒称福岛核废水排放是日本“可耻的决定”，并指责国际原子能机构“没有经验”使其有资格批准日本的废水排放计划。言下之意是国际原子能机构不受中共操控，不是自己人，所以经验不足，没有资格，中共就是这个逻辑。

尤其是华春淫这个泼妇的骂街令笔者忍俊不住，乐了一把。华春淫在本周二的记者会上耍泼称**:**日美声称核废水经处理已经符合国际安全标准，甚至可以饮用，那么他们为什么不为自己保留水呢？或者把它运到美国？笔者查阅了相关报道，至少没有发现日本方面表示计划排放的核废水可以饮用，也没有找到美国方面的相关表态。

《环球时报》就废水排放对日美进行了夸张的长篇谩骂后终于漏出了狼尾巴，称“美国纵容日本邪恶的地缘政治表现”，实际上是报复日本在台湾、钓鱼岛上对中共的强硬态度，对即将举行的美日首脑会谈施加压力，掩耳盗铃的中共应该明白，美日加快反共灭共的步伐，是遵从民意和天道。所以，不管中共如何费尽心机，拼命挣扎，都改变不了行将灭亡的结局。
