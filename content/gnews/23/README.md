美東時間10月23號，美國主流媒體福克斯公開了一條新聞，標題為著名的民主黨人Harris（賀錦麗）被列為拜登家族企業風險投資專案的關鍵連絡人。賀錦麗為民主黨副總統競選人，參與了和現任副總統彭斯的同台演講，其丈夫任德龍為歐華律師事務所合夥人。

提供證據的是美國海軍前中尉博布林斯基（Bobulinski），作為亨特·拜登與現已破產的華信中國能源公司的首席執行長官，他收到了來自吉姆的電子郵件，該郵件明確了以上關鍵連絡人名單。博布林斯基將作為拜登家族作奸犯科的重要證人之一。
