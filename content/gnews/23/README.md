**西班牙2021年3月19日电/西喜社——**

周三多数自治区累计发病率上升，造成全国平均发病率水平停止下降。卫生部今天的报告是过去14天每10万居民发病率是127.91例，而昨天公布的这个数据是127.80例。这是自今年1月下旬以来从未出现过的小幅上涨。

虽然几乎所有的增长都是轻微的，但值得注意的是，阿拉贡自治区和纳瓦拉自治区的发生率分别上升了7和6个百分点。埃斯特雷马杜拉、马德里和巴斯克地区则增加了两个百分点，安达卢西亚、加那利群岛、卡斯蒂利亚-拉曼恰、卡斯蒂利亚和莱昂以及加泰罗尼亚地区则增加了几十分之一或最多一个百分点。同样值得关注的是梅利利亚自治市，发病率增加了22个百分点，休达的发病率也增加了两个百分点。

西班牙各地的新发感染数据也呈现出同样的趋势，并且比前一天有所上升。周二有4962个新感染病例，本周三有6092个。自疫情开始以来，西班牙的感染总数现已达到3,206,116人。

至于死者，卫生部报告说，在过去24小时内有228人死亡，而周二的数字是141人。 据该部说，过去一周有405人因中共病毒而丧生，自疫情爆发以来已有72793人丧生。

周二，住院的中共病毒患者比例是6.67%，周三这个比率是6.45%。

其中，因中共病毒而入住重症监护室ICU的病人：周二为20.24%，而周三是19.93%。略有下降。
