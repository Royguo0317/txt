## 中共煽动美左派又出劣招鼓动“ 撤消军费”
`2020-09-14 15:43 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/356023/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14153911/1600109986700-1.jpg)图片来源：THEHILL




美媒体《国家脉动》的资深记者娜塔莉·温特斯（Natalie Winters）于9月13日发表了一篇文章，揭露了中共党媒中国环球电视网（CGTN）挪用美国左派 “撤销警察经费 “（defund the police）的口号而强加在“美国军费”上，企图再度鼓动左派挑起事端。

文章中指出，中国环球电视网在这篇名为《撤销军费是美国气候核算的必要步骤》的文章中，无中生有地坚称美国的军队是 “地球上最大的污染者之一”，并以此来证明撤销军费说法的合理性。然而众所周知，实际上，中共在空气和水的污染方面才是压倒性的世界第一。中国环球电视网在这篇800字的文章中还妄言，美政府撤销对军队和警察的经费都是 “合情合理 “的，也是“对美国完全没有气候计划和狂热的军事开支的失败的双重解决方案”。 这家作为中共喉舌的媒体继续描述了美国政府”如果能将这笔钱的一小部分用于帮助发展国家的生产力，使其脱离化石燃料，就能带来严肃而持久的积极变化”。

这一荒谬的呼吁是在中共因制造和传播中共冠状病毒，并对美实施了几十年的经济和信息战而导致美中的紧张关系持续上升之际发出，让人无可避免的怀疑其居心所在。

译评：气候变化问题、反战以及人种歧视等等，这些争论议题无一不是美左派人士的敏感热点，而在大选前的敏感时刻，中共喉舌这篇把这三个议题强词夺理地串联在一起的文章，正是中共卑劣居心的明证。穷途末日的中共明知自己的“环境污染”恶名，还能堂而皇之地扯美军是”地球上最大的污染者之一”的谎，实在令人喷饭。这也再度证实了文贵先生于2017年所披露的中共对美国的3F计划 （[https://gnews.org/zh-hans/154701/](https://gnews.org/zh-hans/154701/)）。希望美国民众，特别是左派们能睁开雪亮的双眼，不要再受中共的欺骗和煽动了。
