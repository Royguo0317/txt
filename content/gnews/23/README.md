##  **司法部官員稱：中共國休斯敦領事館長期以來一直受到聯邦調查局的關注** 


[!\[\](https://spark.adobe.com/page/PTVzPDa741uPK/images/77ad20b9-78dd-45c1-9367-500139c846bc.jpg?asset_id=c65f5575-a4aa-4132-8695-2b021289aa9e&amp;img_etag=%22569afd443ffd2493aab884e6c59f961a%22&amp;size=1024)](https://spark.adobe.com/page/PTVzPDa741uPK/images/77ad20b9-78dd-45c1-9367-500139c846bc.jpg?asset_id=c65f5575-a4aa-4132-8695-2b021289aa9e&amp;img_etag=%22569afd443ffd2493aab884e6c59f961a%22&amp;size=1024)照片： 美國得克薩斯州休斯頓，中國僱員離開中共國領事館後，便衣美國安全官員包圍了領事館入口，2020年7月24日， 路透社/ Adrees Latif 
華盛頓（路透社）——美國司法部高級國家安全官員周三表示，美國聯邦調查局(FBI)早就注意到中共國駐休斯頓領事館是中共竊取美國知識產權的基地。

在一次華盛頓某智庫的討論中，約翰·德默斯（John Demers），美國司法部負責國家安全的部長助理，說川普政府關閉休士頓領館的行動不是“隨機選擇的”。

德默斯（Demers）告訴戰略與國際研究中心，美國政府關閉領事館的舉動是為了(徹底)打亂“我們一段時間以來一直在跟踪的(犯罪)情況”。據他所知，最近在美國30個不同城市有50起類似的為中共國謀取工業和知識產權利益的間諜活動被曝光。他說，休斯頓只是“冰山一角”。

德默斯（Demers）說，美國政府之所以在最近提出限制使用中共國應用程序抖音，一個主要的擔憂就是美國人在使用該應用程序時允許它訪問敏感的個人數據，有時甚至包括聯繫人列表和位置信息。

德默斯（Demers）說，如果美國在5G數據和電話網絡的開發和建設中使用中共國通訊巨頭華為的技術，那將會使美國人的數據更大程度地暴露在中共的面前，這也是為什麼川普政府付出“巨大努力”來說服歐洲和亞洲盟友應謹慎行事，不要放任中共國製造的設備進入他們的網絡中。

他補充道，預計2020年期間，美國將對更多中共國參與的電腦黑客活動提出刑事指控。他說，司法部正在重新指派檢察官來推進與中共國相關的案件。
