**西班牙2021年1月12日电/西喜社——**在一所小学爆发中共病毒疫情后，荷兰一个城市正计划对其所有两岁及以上的居民进行检测，其中包括在英国首次发现的传染性更强的变种病例。

兰辛格兰市政府决定对其6.2万名居民几乎全部进行检测，这标志着荷兰自疫情开始以来首次发生这样的大规模检测。

当地卫生当局上周表示，与Bergschenhoek镇一所学校有关的约75个病毒病例群包括30个新变种的阳性测试。

大规模检测的目的是研究新变种的传播情况，并平息其在靠近港口城市鹿特丹的市镇的蔓延。

作为严格的五周封锁的一部分，荷兰的所有小学都被关闭，普遍预计本周将延长封锁时间。
