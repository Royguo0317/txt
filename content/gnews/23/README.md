04月22日，美国驻俄罗斯大使约翰·沙利文已经离开外交使团，并且已经在“前往华盛顿的路上”。这是俄新社援引大使馆的话报道的。

据塔斯社报道，沙利文抵达谢列梅捷沃机场。据该报称，他可以乘坐在伦敦转机的航班离开俄罗斯，该航班将于莫斯科时间10:00从谢列梅捷沃起飞。

在华盛顿，预计沙利文将与乔·拜登政府的成员举行磋商。俄罗斯外交部于4月16日建议他离开莫斯科，即华盛顿对俄罗斯实施新制裁的第二天。 然而，这位外交官在4月20日才确认离开。他说，他将在美国呆上几个星期，但没有给出确切的返回日期。

4月20日，总统发言人德米特里·佩斯科夫（Dmitry Peskov）在回答关于美国大使何时可能返回莫斯科的问题时说，“大使在协商后返回是出于权宜之计。”他说，两国之间的关系现在处于“低点”。

俄罗斯驻美国大使阿纳托利·安东诺夫（Anatoly Antonov）于3月中旬离开华盛顿。4月7日，外交部副部长谢尔盖·里亚布科夫说，他回国“不是未来几天的事”。副部长说：“时间的确定将取决于华盛顿在双边轨道上将采取什么措施。”一周后，美国推出了针对俄罗斯的一揽子新制裁措施。
