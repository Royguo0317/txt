《紐約郵報》報導，中國病毒學家，香港公共衛生學院前研究員閆麗夢博士在接受一次採訪後不久就被推特封停，在該採訪中，她聲稱在中國武漢的一所實驗室製造了新型冠狀病毒。

閆博士在與福克斯新聞主持人塔克·卡爾森交談時稱，中共國正在掩蓋該病毒來自一所武漢實驗室的證據。閆博士告訴卡爾森：“他們不希望人們知道這一真相。另外，這也是我被停職、封口的原因，我是中共想要消失的目標。”

針對閆博士的聲明，卡爾森回應：“由於您對病毒來源問題提出質疑，所以我可以認為您不是反華種族主義者。但是我們目前尚不清楚為什麼推特將您禁言，以及為何您不被美國其他媒體所重視。”

布萊特巴特新聞向推特尋求回應，但該社交媒體網站的一名代表稱，推特對這一問題不予置評。閆博士稱，她於週一在開放數據網站Zenodo上發表了一篇論文能夠為她的聲明提供佐證，該論文詳細說明瞭如何用六個月的時間在實驗室環境中“方便地製造” 新冠病毒。
