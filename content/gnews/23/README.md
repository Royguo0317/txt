**西班牙2020年12月17日电/西喜社——**川普竞选总统高级法律顾问兼律师詹娜·埃利斯（Jenna Ellis）表示，总统大选是“被从普总统手中偷走的”，必须纠正并防止这种情况再次发生在美国。

在《约翰·所罗门报道播客》的一次采访中，艾利斯说：“我们越继续深入了解”，这个国家越看到大量污染了选举的“腐败和完全的无法无天”，人们呼吁 “答案”和“问责制”。

埃利斯在接受采访时说，无论2020年大选的最终结果如何，“我们都必须确保50个州中的每个州都非常重视选举的公正性……他们将作出符合宪法的合理的保障措施，以确保美国人民的意志不被剥夺。”

川普总统尚未在总统选举中向拜登认输。 他和其他人声称选举被欺诈污染了。 埃利斯指出，她希望“各州立法机关将认真履行其宪法授权的工作，以确保其选民不被剥夺选举权，并将正确的代表送往选举人团。”

本周早些时候，在选举人团投票期间，乔·拜登获得了赢得总统选举所需的270张选举人票。投票清点将在1月6日由副总统迈克·彭斯（Mike Pence）主持的国会联席会议上进行。

某些州的共和党选民对川普总统投了有条件的选票，尽管这并不影响正式的选举人团的投票：“我们采取程序性选票来保留可能提出的任何法律主张，” 宾夕法尼亚州的川普竞选团队的主席伯尼·康福特说过: “这绝不是为了篡夺或反对宾夕法尼亚州选民的意愿。”

关于某些州的共和党选民问题，埃利斯说：“他们仍然是川普领导下的合适代表，他们绝对可以投下他们的选票，因为选举认证是虚假的。”
