文章最后总结，“…中国并不缺主粮，进口部分主粮主要出于民众想要吃优质粮、满足世贸组织分配的进口粮食份额”。这更是胡说八道，中共国什么时候考虑过民生、民意。根据中国农业信息网对2021年1月农产品供需形势分析中多处提到，“…玉米进口量比上年度增加300万吨”、“…国内大豆进口量9800万吨，比上月调增300万吨”、“…北半球大豆已收获完毕，美豆产量增幅不及需求增幅，俄罗斯大豆产量下降；南半球大豆播种已接近尾声，后期生长和产量受干旱影响的情况仍有待观察”。从中共农业的内部预测报告可以看出中共内外媒体的报道互相矛盾，中共粮食进口量持续增加，加之全球粮食价格将持续维持供不应求的形势，中共国的粮食危机将难以避免。
