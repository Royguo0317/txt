**西班牙2021年3月23日电/西喜社——**据路透社3月23日报道，法国外交部消息人士称，法国周二召见了中共国大使，强调针对法国立法者和一名研究人员的侮辱和威胁，以及北京决定制裁一些欧洲官员是不可接受的。

驻法国大使卢沙野去年4月已经被外交部传唤，原因是使馆发帖和推特为北京应对中共病毒疫情进行辩护，并批评西方国家的处理方式。

中共国大使馆上周警告说，不要让法国议员在即将访问自治的台湾期间会见官员，引来法国的回绝。

此后，它一直在推特上与总部位于巴黎的战略研究基金会的中国问题专家安托万-邦达兹（Antoine Bondaz）发生争吵，使馆将其描述为 “小暴徒 “和 “疯狂的鬣狗”。

“这仍然是不可接受的，已经超越了一个外国使馆的极限。”这位法国官员在卢沙野受到外交部亚洲司负责人的召见后说。

这位不愿透露姓名的官员说，卢沙野的行为正在为改善中法关系制造障碍。

美国、欧盟、英国和加拿大周一因中共国官员在新疆侵犯人权而对中共国官员实施制裁，这是美国新总统拜登领导下的西方国家首次对北京采取此类协调行动。

作为报复，中共国外交部制裁了几名欧洲公民，包括法国欧洲议会议员拉斐尔-格鲁克斯曼。

法国官员说，这位特使已被告知法国不赞成这一决定，并补充说，卢沙野”对他被告知的极其直接的内容明显感到震惊”，并试图改变谈话内容，讨论台湾问题。
