2月26日，国内网上出现一位器官移植专家的讣告——青岛大学医疗集团副院长、青岛大学医学部器官捐献与移植研究院院长、青岛大学附属医院器官移植中心主任、肝脏病中心主任、著名器官移植专家臧运金教授，于2021年2月26日凌晨不幸辞世。讣告没有说明死因，国内媒体《医学界》27日称臧运金系跳楼自杀身亡。其死因据说与最近被反腐调查，压力太大有关，莫非又是被以自杀为由灭口？

![]()![](https://gnews.org/wp-content/uploads/2021/03/图片-2-1.png)
