## 新西兰今年接种流感疫苗有点麻烦
` 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hans/1054444/)

编译：铭心

新西兰免疫专家指出今天接种流感疫苗比较复杂，因为新冠疫苗。

新西兰卫生部长已经确认现在确保有240多万针疫苗，这比去年多了30多万针这已超出了去年供应记录。因为生产问题，疫苗到达时间会延迟。卫生部表示流感疫苗对65岁及以上的老人开放接种大概从4月14日开始。65岁以下的人暂时还不行要等待5月。

免疫咨询中心负责人Nikki Turner表示五月份提供流感疫苗有些困难因为今年人们要开始接种新冠疫苗。她相信今年打流感疫苗的人会少一些因为一些人像一线的工作人员会先打新冠疫苗。两针新冠疫苗必须最少要间隔三周时间。有些地方新冠疫苗和流感疫苗会同时上市。流感疫苗要在冬季流感季节来临之前打。如果流感疫苗和新冠疫苗同时期打会比较困难。

建议间隔两周时间

Turner医生建议在流感疫苗和新冠疫苗之间相隔两周时间。因为新冠疫苗还没有过临床检验的安全时间，如果打了其他疫苗会降低效果。如果必要的话大家可以两个疫苗都打，这只是一般的时间线，不是绝对的规定。对于高风险的人群打预防流感的疫苗还是很重要的。

系统内的压力

新西兰皇家全科医生学院的负责人Bryan Betty表示管理疫苗的能力比较让人担心。他说两针新冠疫苗和流感疫苗大概有一千两百万个疫苗在接下来的六个月时间，这个强度和压力贯穿着整个系统。这个问题严重程度取决于接种疫苗的时间和疫苗之间的交叉。

南方医疗的负责人Stephen Graham表示疫苗接种的时间在老人院已经构成了问题。他说在考虑到大量的老年人是新冠病毒的高危人群即将接种疫苗。他不确定如果流感疫苗四月份就可以对65岁以上的老人开放，那打新冠疫苗的时间就要延迟，因为要有两周的间隔。

Turner医生表示她不认为今年会有很多流感的发生，因为边境关着，有效的使用社交距离。但是也可能会比去年差一点因为今年新西兰没有完全封城。不管怎样，她还表示情况不可能说会有多差，只希望我们不会有很糟糕的流感传染季但是这也很难预测。
