**簡評：**

美中經濟與安全審查委員會的一份最新的報告指出：為了增強其軍事實力，中共國系統性的網羅在美獲得“科學技術成果”的中共國學生和學者，為他們竊取美國的科學技術研究，並通過地下渠道發送給與中共國軍隊有關的公司和機構。美國國務院在八月份致函美國高校董事會警告中共國對美國高等教育界的威脅。

**新報告稱，中共國利用海外留學生竊取美國技術**

一份新的國會報告指出，中國共產黨已經建立了一個複雜的網絡，用來脅迫海外留學生幫助其竊取美國的技術。

[!\[\]()!\[\](https://spark.adobe.com/page/duWWwjSzK3c0F/images/5593776b-7501-43ea-99a7-9dad9b683b94.jpg?asset_id=db32d0b9-11db-4462-99b3-6c320b03c222&amp;img_etag=%229b82b9a4841679203f11adfd98f53328%22&amp;size=1024)](https://spark.adobe.com/page/duWWwjSzK3c0F/images/5593776b-7501-43ea-99a7-9dad9b683b94.jpg?asset_id=db32d0b9-11db-4462-99b3-6c320b03c222&amp;img_etag=%229b82b9a4841679203f11adfd98f53328%22&amp;size=1024)

美中（共）經濟與安全審查委員會周三報導，多年來，中共國一直試圖通過竊取美國的科學技術研究來增強其軍事實力，並且“建立了龐大的生態系統，包括組織機構，程序和激勵機制，用以吸收和利用在國外獲得“科學技術成果”的中共國學生和學者。

自2006年以來，在美國的中共國學生數量增加了五倍，其中一小部分顯著的群體，他們主要是從事科學和數學領域研究的學生，因涉嫌偷竊美國的科研成果而對美國的國家安全構成了很大的威脅。報告說，中共國經常要求這些留學生從事有利於中共國共產黨的活動，這是他們能出國留學的一個條件。大部分這些學生竊取的技術都是通過地下渠道發送給與中共國軍隊有關的公司和機構，從而為中共國軍隊提供了創新的優勢。

報告說：“實際上，美國的大學正在培訓將在一系列與美國國家安全利益相對立的組織中工作的科學家和工程師，這些對立面的組織就包括“中國人民解放軍（PLA）”。

每年進入美國的中共國學生數量巨大，這給對他們進行審查是一種挑戰。該報告稱，由於案件積壓和人員不足，美國聯邦當局僅拒絕了不到5％的、來自中共國的“被標記過的”的簽證申請者。

中共國目前是全球最大的盜竊知識產權的國家，他們使用複雜的孔子學院的網絡，在美國的校園，領事館和政府中實施他們的“千人計劃”來進行前所未有的數據和技術盜竊。

川普政府已將在幾乎所有的經濟和學術領域中擺脫中共國影響力作為其主要的優先事項。在八月份的一封信中，國務院警告了美國高校董事會關於中共國對美國高等教育的威脅。

信中寫道：“中共的行動對學術自由和人格尊嚴構成了巨大威脅。儘管美國人在許多問題上可能存在分歧，但這種威脅將會使我們團結在一起。”
