**西班牙2021年4月21日电/西喜社——**

在西班牙，研究方面的人才总是多于资金，在2008年的大衰退之后尤其如此。在这个全球化和全球经济向东方转移的21世纪，不仅最卑微和准备不足的人转移，甚至重要科学部门的医生也在转移，这是一种令人担忧的人才流失。

这就是佩德罗-拉博达（Pedro Laborda）不得不做的事情，这位34岁的阿拉贡人在萨拉戈萨大学学习有机化学，在2008年最严重的经济危机袭击西班牙时攻读博士学位。在完成学位后，他经历过10个月的失业期，在欧洲一直没有找到工作。最后，在《自然》杂志上找到了一份来自中共国的工作机会。”

2015年1月2日，他来到了南京市（Nanjing），在农业大学做了两年的博士，然后又在江苏农业大学做了两年的博士后。”他现在在南通市生命科学大学主持一个实验室，他的妻子和他们的一个女儿，也来到南通。

他不是唯一的人。自去年年底以来，拉博达担任西班牙中国研究人员网络（RICE）的主席，该网络成立于2016年，接纳了一百多名在这个国家工作的科学家和学者。其中有一些著名人物，如塞维利亚医生费尔南多-阿雷萨纳（Fernando Arenzana），上海巴斯德研究所的联合主任；卡洛斯-纳瓦雷特（Carlos Navarrete），一位在上海交通大学使用超级计算机工作的理论物理学家，以及马里奥-兰扎（Mario Lanza），他在苏州研究纳米材料，现已前往阿联酋。在人文方面，拉斐尔-马丁-罗德里格斯（Rafael Martín Rodríguez）是一位来自马德里的47岁人士，他已经在中共国生活了十年。他拥有马德里自治大学的地理和历史学位，以及语言学和当代史的博士学位。

拉博达是位移居国外的西班牙研究人员在全世界创建rice网络的17人之一。”这种人才外流始于2008年危机之后，是一个影响整个欧洲的现象，不仅有西班牙科学家，还有意大利人、法国人和德国人“。

“中共当局非常热衷于吸引外国人才，但不是所有的名额都能填满，因为每年有500个研究人员和科学家的职位。例如，在南通大学，我们正在寻找化学、生物和材料方面的博士后，”拉博达说。尽管他澄清说 “基本工资不高，有时甚至低于西班牙”，但他强调，”我们对生产能力、出版物和专利有奖励。如果你努力工作，发表很多文章，你可以得到相当多的钱，但竞争非常激烈。而且在中共国最诱人的地方是他们为你提供一个稳定的职位和你自己的研究团队，就像我的团队，有30个人。如果我留在西班牙可能无法得到这样的东西。
