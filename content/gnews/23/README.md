當前世界，每次科技的發展變化，都會帶來新的機會，也會造成新的危機。就如電的出現，不但帶動了工業的發展，提高了人們各方面的生活質量，但也讓一部分人失業。同理，網絡的出現給商務、社交、娛樂、通訊帶來新的發展機會，也同樣讓實體店、傳統通訊工作人員失去了發展機遇和工作機會。

在未來的10-30年的發展變化中，會延續過去的發展路徑，通過互聯網發展，進入物聯網、雲計算、大數據、以及人工智能時代。那麼在未來，同樣會帶來危機和新的機遇，會讓傳統企業倒閉、人員失業，也會帶來新的就業和新業態。

在此，筆者根據自身的從業經驗，從“未來什麼樣的人會下崗”、“實體店大量消失的原因”、“平台是未來最大的房東”三方面解析G系列平台的潛在優勢。

**一、90%的人將失業**

為什麼說未來90%的人都將面臨下崗呢？這就如有了電話、網絡以後，郵差會下崗的原因一樣，因為一些特定的崗位和職業，是不同時代下產生的。

在未來10-30年，人工智能的發展將解放人們的雙手和勞動力，工廠裡重複勞動的工人，全部會有人工智能機器人代替，人類要做的工作就是控制機器。就是說在過去一個車間需要1000人，現在只需要1個，甚至全智能運作，只需要在後端控制機器人，輸入操作指令和維護機器人。

當然除了工廠，所有重複勞動的崗位，危險的職業，也都會有人工智能代替。最常見的就是司機、保安、服務員、保洁以及銀行櫃檯人員。

從近兩年各地無人駕駛汽車的落地就可以看出，未來根本不需要司機，甚至不需要私家車，可以通過出行需求去匹配出行工具。另外，銀行的一部分工作人員也會被機器代替，從移動支付到數字貨幣就可以看出，未來支付交易、存款不需要去銀行網點辦理業務。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/26082956/2-64.png)
未來不但勞動力會被機器代替，靠腦力勞動的人群也有被淘汰的風險，因為現在進入了數字化時代，大數據時代。

如果35歲以上的人，還停留在傳統的經營模式，運營用傳統的營銷、傳統的銷售、傳統的管理，經營中會被淘汰。從近兩年各大網絡公司大量淘汰35歲以上的人群，就可以看出，未來進入了一個新時代，不改變經營思維，會拖垮企業發展。

所以在未來不想被淘汰，必須要進入數字化經營邏輯中，一切工作流程、信息、內容都要數字化，要用數字統計，區塊化記賬和分配，同時通過雲計算，獲取經營中的各種大數據，用數據說話，才是未來的方向。

那麼被淘汰的人該何去何從呢？要么快速的自我變革，適應新的市場，要么去從事機器無法代替的行業。

**二、50%的企業將消失**

數字化、人工智能、大數據的到來，不但給個人就業和發展帶來影響，也會給企業帶來危機和新機遇。

因為目前大量傳統企業的困境，在於供需無法匹配，想要解決這個問題，需要優化供給方。這也是數字化轉型的必要之處，因為只有把經營環節中的貨、人、場景、流程、物料等信息、內容通過數據中台打通，變成可以統一管理的數字，集合成大數據，才能判斷市場的真正需求。

有了數字和數據以後，可以根據用戶的需求做生產、營銷和銷售，才能實現共享精準匹配，降低庫存風險，提升整體的產能。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/26083151/4-43.png)
正是因為有了數字化和數據化以後，大的企業才能更快的發展，因為大企業有更多的財力和人才，這就會導致市場出現兩極分化。

大企業可以通過數據判斷輸出的需求，根據數據和數字去生產，在滿足市場需求的同時，可以提升品牌的集合度，提升品牌的溢價能力。

大企業未來要做的是通過前端做入口，獲取用戶的信息數據，通過線上APP、物聯網等多個場景，與消費者建立連接，實現复購、提升用戶的價值。

同時企業有了用戶資源和大數據以後，可以根據需求，整合商業的供應鏈，通過規模效應碾壓市場中的傳統企業，小微企業。

那麼這個時期，靠生產和流通為核心的傳統企業，就無生產之地了，因為單靠付費推廣或者實體銷售，雖說短時間可以獲得客戶，但未來用戶都會被已經做數字化轉型的企業佔有。

傳統企業沒有用戶和數據，就無法判斷未來的市場需求，更沒有供應鏈優勢，所以靠生產和流通生存的企業在未來10年會面臨大量倒閉。

**三、80%的人靠平台生存**

那麼大量的人失業，大量的中小微企業倒閉以後，他們該如何生存呢？在未來10年，想要生存和發展的個人，都離不開平台，平台將是未來最大的房東。

就以近兩年來說，大量的高校學生，找不到合適的工作，就直接去送外賣，白領失業去跑共享汽車去了。他們做的工作都是依附在平台上生存的，就是直接靠平台接單，平台收取扣點，這不就是新的房東嗎！

對於各種實體店來說，未來想要生存必須要依附於平台，通過平台賦能做數字化轉型，適應未來的市場。但想要通過平台展現商品，通過平台去獲客，跟在線下店租門店的邏輯類似，因為需要給平台繳費，要租用平台的空間，通過平台去管理用戶和信息。

就如現在的餐飲店一樣，離開平台都無法生存，平台就是商業的未來，從各大巨頭搶占本地服務市場就可以看出這一點。另外在未來當大量工作被機器代替以後，人們的工作時間會越來越多，那麼就會有更多以精神需求為主的產業出現。就如現在的短視頻、直播的新個體一樣，需要通過內容滿足人們精神的需求，從中獲取利益。這一樣也離不開各種社交媒體平台，需要通過平台連接內容的供給需求。
