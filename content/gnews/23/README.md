## 中共国宣传漫画警告西方“最后的晚餐”

- 中共国官方媒体喉舌在 G7 峰会后发布恶搞漫画
- 西方民主国家会晤痛斥中共国侵略
- 北京谴责该组织并称中共国正在被西方诽谤
- 该漫画针对澳大利亚并称该国为“两面派”


翻译：Boxinglady ( 文拳）

在 G7 峰会之后，中共国将矛头对准西方民主国家，并在一幅关于西方民主衰落的怪诞漫画警告中称澳大利亚是一个“两面派”国家。

共产党喉舌《环球时报》在一篇文章中对列奥纳多·达·芬奇的《最后的晚餐》进行了卡通模仿，将美国、英国、意大利、加拿大、日本、德国、法国、印度和澳大利亚描绘成他们的国兽。

在艺术家’Bantonglaoatang’在微博上发布的插图中，动物们围坐在一张桌子旁，桌上放着一个中共国地图形状的蛋糕，图片的标题是：“通过这个我们仍然可以统治世界”。

澳大利亚的特点是一只袋鼠，桌子中央的美国代表耶稣，呈白头鹰的形状，而英国是狮子，法国是公鸡，印度是大象，加拿大是海狸，意大利是狼，日本是狗，德国是鹰。

文章说：“坐在狗旁边的是一只袋鼠，它左手伸向美国印刷的钞票，右手抓着一个袋子。”

“袋鼠象征着双面澳大利亚，它积极与美国合作遏制中共国，但也渴望从其最大贸易伙伴中共国赚钱。”

上周末在英国康沃尔举行的 G7 峰会之后，来自澳大利亚、韩国和南非的嘉宾国家领导人与来自 G7 国家加拿大、法国、德国、意大利、日本的领导人会面。 、英国和美国，联合声明高度批评中共国，并呼吁这个不透明的政权对新冠病毒大流行的起源要更加透明，同时还抨击北京糟糕的人权记录。

中共国反击称G7是“诽谤”中国。中共国驻伦敦使馆表示，对提及新疆、香港、台湾等歪曲事实，暴露美国等少数国家的险恶用心的做法表示强烈不满和坚决反对。

白宫国家安全顾问 Jake Sullivan 表示，G7 周日的声明是该集团向前迈出的重要一步，因为领导人齐心协力，需要在从捍卫民主到技术竞赛等各种挑战上与中国“对抗和竞争”。
