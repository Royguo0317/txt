## 美台關係氣瘋中共 胡錫進：隨時開戰 讓台獨灰飛煙滅

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/18214856/F17D5B95-57F6-4DB0-A7D0-8FD5D95E95A7.jpeg)
18日，胡錫進連發三篇博文，大肆威脅美台關係。美國副國務卿克拉奇17日訪問台灣，觸動了中共敏感的神經。次日，解放軍組織台海實戰演練，戰機從四個方向逼近台灣空域，台灣22次廣播驅離。胡錫進稱，這次是攻台預演，美台關係再進一步，隨時準備開戰。

胡錫進發文表示，通過這次演習，也通過之前的系列演習，解放軍不斷積累攻台經驗，掌握台灣防禦體系的系列關鍵數據。它們就是攻台的實操性預演，只需一個政治理由把它們從演習版激發成實戰版，「台獨」的一切就將灰飛煙滅。

中共對此特別點名了台灣當局「民進黨」。中國國防部新聞發言人任國強警吿稱，「美國和民進黨當局加緊勾連，頻繁製造事端，無論是以台制華還是挾洋自重，都是痴心妄想，注定是死路一條，玩火者必自焚。」看來中共對「以台制華」相當恐懼，承認台灣為獨立國家，會給中共致命打擊。

胡錫進在接受BBC採訪時說，他首先要糾正該名記者稱「很多國家都把台灣視為一個主權獨立的國家」說法，他強調，台灣不是聯合國成員國，包括英國也與台灣沒有外交關係，台灣領導人不能去英國訪問，在英國的土地上不能升起台灣的青天白日旗，這些都證明瞭台灣不是一個主權獨立的國家。

就在本週三，美台之間的另一次會面，美方表示願意幫助台灣重返聯合國。美國駐聯合國大使克拉夫特與駐紐約台北經濟文化辦事處處長李光章共進午餐，稱之為「歷史性」會面。川普政府進一步強化美台關係，雙方討論了如何更好地幫助台灣參與聯合國的不同方法。克拉夫特期待未來能與台灣總統蔡英文會面。中共駐聯合國大使耿爽提出強烈抗議，這違反一中原則和中美三個聯合公報規定以及聯大第2758號決議。

胡錫進指出，對於台灣，中國當然不能放棄使用武力的選項。他相信中國不會輕易那麼做，但是內地有《反分裂國家法》，如果台灣當局觸犯該法，宣佈法理「台獨」，或者與美國那樣的國家，建立起只有國家之間才有的實質性關係，那麼我相信解放軍一定會採取軍事行動，解放台灣。

他還威脅道：「這還是客氣的。美國高官來一次，解放軍的戰機就要逼近台灣一步。如果美國國務卿、防長來台灣，解放軍戰機就應飛越台灣島，直接去台島上空演習。我們試射的導彈更應該飛越台灣島，直至飛越台所謂「總統府」上空。台當局不想過了，我們就來成全它。」

美國對台是日拱一卒，中共的噩夢就是台灣和美國建交。在戰後，中國被承認的合法政府是中華民國政府，也是聯合國成員國。中共和美建交之後，美國拋棄了台灣，簽署了中美聯合公報，中共政府成了代表中國的政府。如今三十年河東，三十年河西，若中美聯合公報作廢，中共政府在國際上就失去了合法性，也會被踢出聯合國。

胡錫進最後一條微博寫道，「不僅沒有「海峽中線」，也沒有台灣的所謂「領空」。以為美台關係就是保險箱了，瞧把民進黨當局狂的。你們不是叫囂為了捍衛「獨立主權」要「戰至最後一兵一卒」嗎？好的，成全你們，大陸的軍機今天從四個方向飛過來了，今後會飛得更近，飛到台灣島上空，準備戰鬥吧。」

胡主編替中共喊出了心聲，「不想過了，就都別過了。」美日又一次在台灣相聚，讓中共感到恐懼。19日，在台訪問的克拉奇和日本前首相森喜朗將出席李登輝告別儀式。森喜朗給蔡英文帶了口信，新首相菅義偉期盼與其通話。這將是日台斷交後48年來首次通話。

面對中共的恐嚇和美日拋出的橄欖枝，小英總統能否克服壓力，讓台灣站起來？
