福克斯新闻报道，拜登总统在接受电视采访时表示，他同意俄罗斯总统普京是一个“杀手”，并威胁要对俄罗斯实施进一步制裁，随后俄罗斯于周三召回了驻美大使。

据路透社报道，俄罗斯大使Anatoly Antonov将返回莫斯科。俄罗斯官员说，这一决定是为了确保两国外交关系不会进一步受损，可留出空间来讨论两国未来的关系。

美国国务院和俄罗斯驻华盛顿的美国大使馆没有立即回应置评请求。

美国高级情报官员披露了俄罗斯在普京的指示下试图干预2020年总统大选的调查结果后，拜登发出了警告。美国国家情报总监艾薇儿·海恩斯(Avril Haines)的办公室得出结论，俄罗斯试图通过“诋毁”拜登来影响选举结果，使之有利于前川普总统。

拜登正在审查对俄政策并在接受电视采访时说，如果事实证明普京干预选举，他将付出代价。

拜登说，“我们谈了很久，他和我，当我们——我比较了解他。谈话开始时，我说，‘我认识你，你也认识我。如果我证实这件事发生了，那就做好准备。’”

拜登强调，公众“很快看到”他的政府将会对俄罗斯涉嫌干预选举采取什么样的行动。

拜登政府最近就俄罗斯毒杀并监禁反对派领导人阿列克谢·纳瓦尔尼(Alexey Navalny)一事对俄罗斯进行了制裁。据称，在国家情报局长发布报告后，拜登政府正在考虑进一步惩罚俄罗斯。当被问及拜登是否认为普京是一个杀手时，拜登说，“我认为是”。
