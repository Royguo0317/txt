**西班牙精神病学家谈疫情对社会心理的影响**

**西班牙2021年3月16日电/西喜社——**

据ABC报3月14日记者Javier Ansorena的报道：西裔美国精神病学家、研究员和教授。路易斯-罗哈斯-马科斯说：”我们还需要两三年时间来恢复”。这位精神科医生经历和救援过受纽约911事件重创的伤者，他说与这次大疫情的痛苦是不同的。他警告说，它的伤害是慢性的，需要时间来恢复。

路易斯-罗哈斯-马尔克斯（LUIS-ROJAS-MARCOS）是一位世界著名的精神病学家，在纽约工作了几十年。他曾负责纽约的公立医院系统，现在指挥着拥有3500多名医生的PAGNY组织。曾经参加过9-11恐怖袭击后的救援活动。日前，接受记者采访，谈到了疫情对精神健康的作用。

**问：我们是否有**** “****中共病毒疲劳症****“****，是否疲惫不堪，不知所措？**

**答：**是的，这是一个巨大的创伤因素。我们不仅不知道明天要做什么，也不知道半年或一年后要做什么，这就打开了焦虑、恐惧、无助入侵我们的思想。这些都是不知道要持续多久的慢性感觉，对人们的生活质量有非常大的负面影响。

**问：如此种种，是否存在心理健康危机？**

**答：**是的，确实如此。长期的焦虑、无助、不自主的禁锢、身体的疏离，都是心理健康的障碍。而当患者为了避免感染而不敢去医院，情绪低落，又没有治疗方法时，事情就更复杂了。结果，自杀率、吸毒率、酗酒率都上升了……这是毁灭性的。

**问：疫情已被政治化，科学被怀疑，专家被怀疑****….****。**

答：这是另一个问题。因为我们都需要领导者。尤其是在暴风雨中，告诉我们如何脱身，该如何解决这种情况。信任是人类稳定的基本要素。你要在一定程度上相信自己，相信别人会带领你，相信人类的团结。当失去了信任，就会增加一个压力和无奈、无望的因素。而你不能没有希望。失去希望是自杀的基础，当一个人不再看到生命的意义时，就会自杀。

**问：如今的领导者们是否尽到了自己的责任？**

答：答案是，在大家这么困难的时候，我们还没有好的领导。在国家层面，一直存在着很多混乱，缺乏明确性和可靠性。

**问：****“****我们会变得更强大 ****“****这句老生常谈的话语有什么现实意义吗？**

**答：**会有一部分人，会变得更加强大，这就是我们所说的创伤后成长。我相信，在这一进程中，在克服这一流行病的斗争中，有些人将找到一种复原力、灵活性和组织自己的能力，当然，也有很多人不会。

**问：对于像西班牙人这样身体接触、亲吻和拥抱文化较多的社会，病毒对心理健康的影响是否更大？**

答：这是毋庸置疑的。如果身体接触在你的生活中已经常态化了，突然间，它被禁止了，你就必须适应。

**问：当我们走出疫情的时候，会不会出现情绪爆炸？**** ****情绪崩盘等风险？**

答：这发生在911事件上，当时看起来没有更多的飞机会坠落。对我们所经历的恐怖事件有了了解，但事情已经过去了，也就容易释怀了。有一天，疫情过去了，我们把门窗打开，出来相拥，就像地震后一样。

**问：你建议人们如何面对大流行的余波和可能还会出现的波折？**

答：第一件事：认为自己可以做一些事情来帮助自己。你要有条不紊地去做事情，相信自己的能力：参加课程，锻炼，什么都可以。第二件事：事实证明，在这些情况下照顾他人的人，能更好地应对这些逆境。另外，我也明白，这对每个人来说都不容易，幽默感。以幽默的方式看待疫情的困境。

**还有一点：大脑不能很好地应对缺乏解释的情况。寻找关于疫情的解释有助于增加一点希望，希望有一天不会再发生这样的事情。**

笔者简评：

从采访中可以看出，欧洲，包括西班牙的民众，因为疫情，人们的心理产生了巨大的负面影响，无助、失望、抑郁、愤怒等，**这也是中共超限战希望达到的一个效果：摧毁民众的信心，活在恐惧和未知中。**

我们更注意到记者的问话：“**疫情已被政治化，科学被怀疑，专家被怀疑****….”**

我们不禁想问：

- 如果疫情是“自然起源”的，那么闫丽梦博士的两篇科学报告指出病毒来自武汉实验室，由中国特有的舟山蝙蝠病毒为骨架研制而成。而这两篇科学报告至今无人能够反驳，又如何解释？闫博士可是接受世界上任何人关于此报告的一对一公开辩论的。
- 所有有关疫情“自然起源”的论据都来自中共国，但至今为止，他们既没有上传最关键的证据—最早的原始毒株样本（按照世卫组织规定是必须上传的），也不允许外国机构进入武汉进行独立调查。他们想隐瞒什么？
- 现在我们来谈谈科学，科学是需要反复论证的，既然闫博士提出了病毒“非自然起源”的论点，科学家们可以用事实去说明她是错的啊，而不是谩骂和政治指控。我们需要相信的是科学依据，而不是科学家。请拿出证据，专家！
- 再回到现实的角度，本着负责和职业的态度，难道我们不是应该调查所有的可能性，一个一个的排除，最后得出真相吗？我们不是应该支持一个由世界各地的科学家组成的，能独立、不受限制、详细和专业的调查吗？而不是之前由世卫组织所组织的，完全受中共国控制的那种调查。
- 在真实的、独立的科学报告没有出来之前，为什么只有坚信“自然起源”才跟政治无关，所有怀疑“自然起源“ 的科学家和专家就是被政治化，阴谋化？而为什么“自然起源”不敢接受对质？这才是我们所需要知道的真正解释。


在文章的最后，科学家隐晦的提到了应该追查病毒真相，“**寻找关于疫情的解释有助于增加一点希望，希望有一天不会再发生这样的事情。“。**
