![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/02104531/logo.jpg)



路德访谈在2020年12月3日晚对郭文贵先生的直播“中国人的灾难真正开始了”进行了解读。上周北约、俄军方已经评估中国的病毒超限战、疫苗战、核战对人类的危害，星球大战计划重启，全球进入铲除中共的大潮流中。

节目中提到了灭共的几个关键词“外交手段”、“摧毁经济”、“斩首”、“军事打击”、“联军进驻”。其中，“摧毁经济”是指靠脱钩、制裁等手段摧毁整个中共国的经济体系！在全球摧毁中共国经济的重锤砸下后，究竟中共国会发生什么？人民币将何去何从？

首先谈谈制裁。2020年12月14号，大选的权变选举结果确定后，川普总统就能腾出手收拾中共国的大批军民融合企业（华为、腾讯、中兴等）以及中国各商业银行。针对这些企业和银行的新一轮制裁是要彻底将这些单位击垮，一旦出手，中共国内将会有大批银行面临倒闭，大量债务变成坏账，你在这些银行的一切储蓄、投资可能一夜之间就荡然无存。

这一切债务的最终负责人是央行，摆在央行面前的选择只有两个，要么大量印钞（大面额钞票）救银行，要么银行破产。

如果央行选择前者，超发货币将引发巨大的灾难——人民币恶性通胀，购买力急剧下降，具体参见委内瑞拉。如果央行如是应对，未来“房价白菜”和“白菜价的房”就不是开玩笑的梗了，中共国也将经历一次“苏联解体”。届时，富人也会无比艰辛，而穷人将体会肉体和精神的双重折磨。苏联解体后，一个女孩为了一个罐头可以一个晚上任人蹂躏，未来的中共国，如是没有尊严的事情也会上演，堪比人间地狱。

如果央行选择后者，在短时间内，虽然人民币依然会大幅贬值，但至少现钞（纸钞）还能保持一定的购买力。但银行破产的后果就是所有债务成为坏账。你在银行的所有资产，包括储蓄、债券、金融产品等等，都将荡然无存！

根据基本的金融知识，基础货币是远远少于货币供给的，再加上国内巨大的金融杠杆，一旦危机到来，就只有基础货币能够短时间内还保有一定购买力。基础货币是指现金（硬币和公众持有的央行印发的钞票）和银行体系中的准备金，而我们在银行体系中的非现金资产都属于广义的货币。也就是说，应对即将到来的危机的唯一方法就是——把所有资产全部换成现钞！即便无法阻止恶性通胀带来的购买力下跌，但至少能够逃避银行破产的坏账，也能在小段时间内保持一定购买力。

事已至此，仍然在中共国金融体系内的资产已经是不日蒸发，爆料革命已经不只一次提醒各位要把资产转移，也提供了一次又一次的投资机会。时至今日，后悔已无用，要当机立断，把资产都换成纸钞和物资（粮食、饮用水、口罩、药品等）。这是最后的一根救命稻草！

其次，谈谈脱钩。一旦美方宣布美元与人民币脱钩，就无疑是宣布开战了。脱钩，意味着人民币成为了人冥币。如果说制裁使得中共国经济体系分崩离析，脱钩就好比是将整个经济体系“连根拔起”了。

脱钩的第一个影响就是将中共国排除在了世界经济体系之外，俨然成为一座经济孤岛，这对于没有完整供应链、无法完成经济内循环的中共国来说几乎是灭顶之灾。美元脱钩，人民币在国际的信用归零，届时中共国大量的国际贸易停止，也无法进口石油等战略物资。

美元脱钩是“杀敌一千，自损八百”的狠招，脱钩也美国带来很大的损失。美元脱钩必然导致中共国内人民币的购买力几乎无限制下跌。

综上所述，制裁和脱钩这套组合拳引起的连锁反应将会带来中共国的经济雪崩，墙内百姓唯一的自救方案就是，把能够换成现钞的资产全换成现钞，同时准备好生存物资（粮食、饮用水、口罩、药品等）。
