在偽代表們“代表”十四億人民通過新版香港國安法的時候，白岩松先生也代表偽代表們發聲，並從偽代表們熱烈的掌聲中讀出了現場的掌聲代表了14億人民的心聲。白岩松先生的本領在於，他真能把偽代表們放出的屁解讀成讚歌，把偽代表們的掌聲解讀為喝詩班唱給上帝的頌歌，而且這頌歌還能代表普天之下的14億草根。由此可見，白岩松先生祖上是不姓白的，本來姓黑，黑白顛倒之後，就姓白了。

那麼這熱烈的掌聲到底是什麼呢？這就需要到對“極權政權”的末日瘋狂進行把把脈了，【在消滅了“尖銳”批評之後，“溫和”的批評就成了最刺耳的，也會被消滅掉； 在批評被消滅後，調侃又會無法容忍； 在聲音都被消滅之後，沉默被視為無聲的反抗； 接著就會要求讚美，然後再消滅那些讚美得不起勁的； 最後掌聲無法停下來，因為先停止鼓掌的就會先被消滅掉。】，現在這掌聲已經是最熱烈的階段了，還沒有到無法停下來的階段，接下來就應該是無法停下來的階段了，因為這個極權和它的奴才們都瘋了，它們已經聽不到為它們敲響的喪鐘了。
