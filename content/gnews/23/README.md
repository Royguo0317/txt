美國懷恩明州代表利茲·切尼在周二接受福克斯的訪問時指出，中國共產黨故意容許病毒傳播，導致全球經濟損失慘重，必須為此負責。

毋庸置疑，中共在得知病毒即將人傳人大規模擴散的當下，損失必將務必慘重，而與其讓它在本土內肆虐，不如讓它擴散至全球。因此他們暫停了武漢至全國的航線，卻對國際航線置之不理，任其發展。美國不但要迅速轉移供應鏈，而且要禁止中共繼續偽裝成文明國家中的一員參與其中，更要向公眾普及中共多年來為了擊垮美國所作出的一系列惡行。

美國必須不但能夠自保，還要繼續引領自由世界，世界必須在秉持自由原則的國家所制定的規則下運行，而非中共。
