**激烈回应联合公报**

次日，新京报快讯报道，中（共）国驻欧盟使团对公报进行了回应[1]，批评北约是对中（共）国的诋毁，对国际形势和自身角色的误判，是冷战思维延续和集团政治心理作祟。BBC也报道了[2]《北京要求停止炒作中国威胁》，在中共和北约两方中间“和稀泥”，为中共传递“中共追求国防和军事现代化是正当、合理、公开和透明的”的信息，同时引用北约主席的话，“我们不会进入新的冷战，中（共）国不是我们的对手，也不是我们的敌人”。

**偷换概念，答非所问**

**一面绑架老百姓，一面挑拨离间**

环球时报发表社评[4] 社评：G7公报造势，中国人不吃这一套，混淆政府与人民，中共与中国的概念，企图绑架中国人民。此外宣传口开始对七国集团挑拨离间，发表评论[5] 崔洪建：渲染“中国挑战”治不了北约顽疾，侮辱北约是脑死亡；挑拨欧洲与美国关系；夸大俄罗斯为假想敌，从而企图分裂北约。
