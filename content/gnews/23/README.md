蓬佩奧在接受福克斯新聞採訪時表示，中共國是美國最大的威脅，而對付來自中共的威脅，美國已經準備好了。中共對美國類似“武漢病毒”的攻擊再也不會得逞。

美國國務卿蓬佩奧9月3日接受福克斯新聞採訪，就記者提出的“中共國和俄羅斯哪個對美國更具威脅”的問題作了答复。他明確指出，中共國是美國最大的威脅，沒有之一！中共滲透美國的方式是俄羅斯遠不能企及的。

蓬佩奧指出，中共對美國的威脅來自各方面，包括對知識產權和商業機密的盜竊，剝奪美國數10萬個就業機會等，他們還將影響美國大選，但更深遠更重大的威脅是，中國正在擴軍備戰。他說：“川普總統不會聽之任之。”今天實施的對中共國外交人員的限制行動，就是“告訴中共的外交官們，他們在美國本土大搞間諜活動的時代該畫上句號了！”

蓬佩奧還再次抨擊了中共散播病毒對美國及世界造成的深重災難和巨大損失。指出，川普總統會確保中共對美國的攻擊不會再次得逞。他說：“這就是專制獨裁政權的所作所為”，“這是自由與專制兩大意識形態之間的戰鬥。”“在川普總統領導下，我很有信心，我們會贏！”

蓬佩奧還指出，在過去兩年裡，美國在軍費上的投入超過了以往幾十年中的任何一年。川普總統將會建立一支強大的軍隊，包括海軍、空軍和海軍陸戰隊。他說，對付中共，“我們已經準備好了！”
