## 疫苗接种护照——荒谬的自由通行证
` 喜马拉雅德国慕尼黑感恩农场` [轉載自GNews](https://gnews.org/zh-hans/939654/)

撰文作者: 文渊

***摘要: 欧盟委员会二十七个成员国拟最早今夏推出统一的疫苗接种护照，以期挽救被中共病毒摧毁的旅游经济，某些国家对此态度仍有所保留***。

已持续一年之久的中共病毒重创了欧洲各国经济，特别是伤害了各国的旅游业。许多国家都实行了长期的封禁措施和旅行限制，令百姓苦不堪言。如今随着疫苗的接种，完全解除封禁、人员自由旅行似乎有望，往昔的美好生活似乎即将重归。一些政客认为: 疫苗是保卫国民生命、挽救本国经济的救命稻草，接种疫苗是走出当前困局的唯一良策。

世界上只有爆料革命传播中共病毒的真相，也仅有爆料革命提供解决危机的办法。郭文贵先生、闫丽梦博士多次指出: “盲目接种疫苗不仅无法解决疫情，反而会导致次生灾难的发生”。尤其是中共国生产的疫苗，如同毒药，源源不断的输送至世界各地。中共国所谓的”疫苗经济”在牟取暴利的同时，也在戕害着世界人民的生命。

对欧洲各国而言，如果尚未了解中共病毒作为超限生物武器的本质、不了解中共疫苗的危险性、而仅仅为了追逐眼前利益而盲目提倡注射疫苗或者推行疫苗护照，将会把本国人民的生命置于危险之中，也会把本来因为中共病毒已经岌岌可危的本国经济进一步推向悬崖的边缘。 如果将接种疫苗比作注射毒药，疫苗护照则如同为每一位接种者颁发一份服毒证书。何其滑稽，何其讽刺。接种者非但不知自己注射的是毒药，却以为自己得到了免疫，还获得了一份自由通行证，可以无需隔离、不受限制的自由旅行。这难道不会引发更大的危机吗?

以笔者个人观点，此事尚未成定局，或仍存变数。若您身处欧洲，且不必过于忧虑。虽然欧盟已原则上同意推行疫苗护照，但若要在欧盟全面使用，难度依然不小。首先，各成员国在此问题上仍有分歧。连法国总统、德国总理迄今仍不敢对疫苗护照大开绿灯，由此可见一斑。再者，欧洲民众普遍持有疫苗怀疑论，大多对接种疫苗持反对态度。即使某些政客主观希望本国民众接种疫苗、推行疫苗护照，却也必须顾及民意。如果大多数民众拒绝接种疫苗，何来疫苗护照? 在此情况下，若强推疫苗护照一定会碰壁，并招致民众的激烈反抗。这一幕是任何民主国家的政客所不愿看到的。

退一步讲，即使疫苗护照在欧盟以及世界各地开始使用，也并非末日。至少爆料革命的战友们懂得如何保护自己，即拥有比普通民众更高的安全机率。我们知道，接种疫苗不能解决问题，颁发疫苗护照更是滑稽可笑。只有结束中共、迫使中共向世界交出解药，才是人类自救的唯一办法。
