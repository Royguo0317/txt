今次美中会谈就好似当年1960年苏联领导人赫鲁晓夫在联合国公然，”拿鞋拍枱”一样，极为相似。 共产党必快将灭亡，重蹈历史。

共产党的两个崩溃去分析「以钱灭共」。

![]()![](https://gnews.org/wp-content/uploads/2021/03/032802.jpg)

第一个崩溃。 中国在2020年，人口红利消失变成负担。 2016年到现在的经济崩溃还未发生，是因为大陆有好强的行政措施。 可以操作到暂时不会崩溃。 也可以从技术上去解释为什么。 首先，传统上的崩溃系指：例如：
1）失业，好多人供不起房子; 
2）好多做生意收不到钱; 债券违约赖账; 
3）在传统的自由经济的银行体系出现问题上，人民便会挤提银行，由于银行之间业务是有关联的，一间银行出现问题就会和其他银行来往而也会出现问题，连锁反应倒下去，这叫系统性风险，于是整个经济向下跳水，因为经济不能够运行。 这就是所谓的经济崩溃。
另一种经济崩溃就好像当年国民党时期的金圆券，政府乱印钞票，造成超级通货膨胀，这是就会破坏了整个社会经济肌理，经济会开始停摆，并且会有两个情况会出现：
1）超级通胀，是怎样呢？ 举例：如生产一碗面，是需要面粉，但制造面粉的人将不会愿意供货给面店的，是因为供给了面店之后，再买不回来生产面粉的小麦价格，他只能一点点卖给他，每一级供货环节只会一点点供给。 经济活动便会停顿。 当其时国民党的经济就崩溃了。 阿根廷嘅经济就是这样崩溃。 我们谈论经济崩溃需要明白经济是怎样崩溃。
2）因为通缩而崩溃，物价越来越便宜，导致人民储存资金不用，钱在手中等待迟点购买，因为预期物价会更便宜，导致经济停顿，20 世纪中期最危险的经济崩溃就系通缩，令到经济破坏不堪，十九世纪末期整个通缩持续20几年，终于有人开始拿点钱消费。 于是物价开始不跌从而恢复过来。 但是在现今中国比较难发生。 因为中共国发明了一招对付通缩。 这招就叫印银纸，只要不断印银纸就不会发生通缩，不断量化宽松不可能发生通缩，导致你不可能不用钞票，为什么？ 因为你不用钱，它会印更多，中共国是不怕印钞票的，原因是：

1）它不怕通胀，因为如果发生通胀，它可以实施配给物资，每个人都有实名登记，配给让经济可以继续运行一段时间;

2）信用崩溃也难以短期发生，因为人民币在国际上并不是一个自由兑换的货币，如果出现挤提的话，政府可以多印钞票给人民银行就可以了。 你挤提不垮它的。 所以不会有系统性风险。 国内债务危机怎样严重都好，只会引起资产泡沫，到了最后现在200多万亿的M2、现在400多万亿的债项，都不紧要，银行销掉二百几万亿债务就完事，人民银行撑住它就可以啦。 谁问银行拿钱，它不让你取钱就可以，因为行政措施限制一切。

现在唯一令到我国经济有真正风险，用最强的行政措施唯一不可控制的一件事：就是外汇！ 当共产党没有外汇的那一刻就死定了。 因为共产党没办法生产外汇，因为超出了国家管理的范畴。 要共产党产生外汇危机还需要点时间的。 到目前为止最多会引起经济放缓。 新中国联邦的G系列，就是能帮共产党放血的超级大动脉！ 这亦是以钱灭共一个方式。 相信美国和其他同盟国，又会清楚，在金融市场上积极帮共产党放血的。 （待续）
