## 辟谣：“中国粮食自给自足” 事实：粮食危机迫在眉睫
` 喜马拉雅-华盛顿DC农场` [轉載自GNews](https://gnews.org/zh-hans/709595/)

**作者： 依靠农民**
![]()![](https://gnews.org/wp-content/uploads/2021/01/1-4.jpg)图片来源：www.takepart.com






粮食问题，是困扰中国两千年的头号问题，自汉代以来，两千年中国所有朝代的更迭都和土地、粮食有关，应了一句老话：衣食足、百事兴，吃不饱、光扯筋！到了CCP执政的这七十年，粮食依然是头号安全问题，号称伟光正、代表了人类最先进生产力的CCP，一直用谎言洗脑中国人，厚颜无耻的说，是CCP养活了14亿中国人。

回到1400年前的唐朝，在开元盛世时期，中国的人均粮食产量达到了800斤，此后，包括CCP建政后，中国人一直处于吃不饱的状态，一直到1996年，改革开放近20年后，中国的人均粮食产量才第一次超过唐朝水平。难道说，过去的2000年，是汉皇朝刘氏家族、盛唐李氏家族、五代十国鲜卑族、宋朝赵氏家族、元朝蒙古人、明朝朱家人、满清爱新觉罗、民国国民党养活了中国人？这些朝代的执政者都没那么的不要脸吧！

明朝，通过海上贸易，从美洲经过欧洲，辗转引进了土豆、玉米和红薯，粮食产量顿时大增，导致人口增加，这个趋势一直持续到清朝后期，期间包括传教士带来的山东的烟台苹果、大白菜等。20世纪头三十年，日本给当时满洲带去了现代的耕作技术和优良的水稻品种，拓荒了大量地的黑土地，今天，我们吃到的所有优质东北大米，全部是源自日本的月光米。1972年，中日恢复外交关系后，日本向中国提供了大量的化肥，中国的小麦、水稻产量才突然增产了70%。改革开放后，随着对外贸易和引进西方化工技术，能够在全国范围内大规模施用化肥、引种先进高产量品种，中国的粮食才在1996年达到完全自给，这已经是中国大陆在CCP手里沦陷37年以后的事情了。

自1996年后，在过度施用化肥和过度开垦的基础上，中国迎来了粮食连续9年的产量增长，但是，这样破坏生态、不可持续的化学农业模式，立即开始报复中国人——从2012年开始到现在，粮食产量不增反降，中国的粮食自给率已经达不到80%，如果不是每年花费700亿美元进口1.04亿吨大豆、2000万吨玉米小麦、几百万吨大米，中国人将陷入严重的饥荒。
![]()![](https://gnews.org/wp-content/uploads/2021/01/image0-7.png)![]()![](https://gnews.org/wp-content/uploads/2021/01/image1-1.jpg)图片来源：[https://cj.sina.cn/articles](https://cj.sina.cn/articles/view/6927412434/19ce7ecd202000hu3c?from=finance&amp;vt=4&amp;autocallup=no&amp;isfromsina=no)






按照CCP的统计口径，中国粮食自给率是95%，但是，这没有计算中国落后的农业生产和食品流通体系造成了可怕的损耗——从田间到餐桌，因为不合理的加工、储存、物流和消费方式，中国的粮食损耗高达惊人的14%，加上CCP承认的5%的缺口，就已经是19%了。

事实上，农业食品经济Agri-Food经济循环，是世界上仅次于大气和洋流循环的最复杂的体系，需要的是可持续的自组织运行，而在中国，CCP对农业农村实施确实2000前的商君书独裁控制，每28各农民就要养活一个CCP官员，从朱元璋那里学来的包产到户的小农弱民政策，导致中国农民早已彻底失去了种田的积极性，高昂的隐形税负和畸高的土地成本，让所有人离开了补贴就无法挣钱和生存。几十年来，水源污染、过渡施用化肥，已经严重毒化了所有的农田，中国南方所有大米几乎都重金属超标，上亿亩耕地严重板结退化无法耕种，还有几千万亩耕地处于长期撂荒状态，中共的粮食产量长期处于虚高瞒报状态，习近平也梦在鼓里自嗨。

所有城市，为了土地财政，把大量的生态指标硬性覆盖到农村的耕地上，北京因此近120万亩耕地被重上了树不能产粮，上海这个数字是30万亩。仅仅内蒙古巴彦淖尔就有1300万亩耕地人为盐碱化，全国加总后是个天文数字。

遥感卫星测定的中国耕地总面积远不止18亿亩，农业部在内部会议上承认的就有21.4亿亩，但是沙化、毒化、盐碱化、人为撂荒是在太严重了，超过30%，中国越来越养活不了自给了！
