**西班牙2021年2月20日电/西喜社——**据每日电讯网2月19日报道，最近有报道称，被关押在中共国新疆集中营的宗教少数群体遭到暴力轮奸。

被迫在集中营内教书的克尔比努尔-西迪克本周在美国有线电视新闻网（CNN）发表的一篇采访中提出了这些指控。CNN的报道完全依靠目击者的描述。

西迪克的故事一开始就指出，一名女警察很早就告诉她，她被派去调查有关该设施内发生强奸和酷刑的报告。西迪克说，这名女警察向她描述了营地的男看守如何经常在喝酒时吹嘘他们如何 “强奸和折磨女孩”。

西迪克说，她第一次接触到的新被拘留者大约有100名男女，他们的手脚都戴着镣铐。她说，即使是那些身体和精神都很强壮的人进来，最终也被中共在集中营中设置的残酷制度所击垮，估计有近200万宗教少数群体被关押在那里。她说，她在集中营工作时目睹了 “可怕的悲剧”，那里的妇女经常 “大声哭泣”。

另一位目击者图尔·齐亚乌敦(Tursunay Ziyawudun)告诉CNN，她被送进集中营时没有犯罪。在难民营呆了一个月后，当局释放了齐亚乌敦，但不久后又逮捕了她，经历了9个月的 “噩梦”。现在身在美国的齐亚乌敦告诉CNN，大量女性经常被迫挤在狭小的空间里，只允许她们有极少的时间使用厕所等基本设施，并且因为使用设施时间过长等小事而受到残酷的惩罚。

在一次审讯中，齐亚乌敦说，她被毒打得昏了过去。还有一次，看守将她强行推到一张桌子上，在她体内 “插上电击棒”，”扭动和冲击 “她，直到她因疼痛而 “昏迷”。几周后，她被几名男子轮奸，她说这种经历在集中营里发生过多次。

“他们非常虐待狂，通过殴打和将我的头打在墙上，造成疼痛和对身体的伤害，”齐亚乌敦说，”这是他们惩罚我们的方式。”

还有许多其他的报道出现，对中共国集中营中发生的事情提出了类似的指控。

中共新疆宣传部门发言人徐贵祥本月早些时候说，宗教少数群体在新疆没有被围捕，并在没有证据的情况下声称，中共国只是在打击 “令人发指的顽固派领袖和极端组织的骨干”。

中共国外交部在给CNN的一份声明中，没有谈及与CNN通话的几位女性的指控，但表示她们的说法是假的。
