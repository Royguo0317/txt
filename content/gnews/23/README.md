5月20日，据塔斯社报道，日本政府打算将参加东京奥运会和残奥会的外国代表团总数比最初计划减少一半以上。共同社周四援引消息人士的话报道，人数将不超过7.8万人，这不包括运动员。

去年，在决定推迟奥运会之前，预计将有大约18万人的外国代表团参加。

昨天，该机构还报道，日本政府打算禁止外国领导人和部长在东京奥运会上与本国运动员会面。采取这种措施是为了防止中共病毒的传播。此外，奥运会的组织者希望在国家领导人或五位随行部长抵达的情况下，将官方代表团的规模限制在12人。

将建议外国领导人的代表在日本停留的时间尽可能短，不要使用公共交通工具，不要离开东京。如果他们同意遵守这些要求，并且中共病毒检测呈阴性，那么他们抵达日本后的两周检疫期将被取消。

东京奥运会将于2020年7月24日至8月9日举行。由于疫情的原因，比赛被推迟了一年，现在将于2021年7月23日至8月8日举行。3月20日，组织者最终决定在没有外国粉丝的情况下举行奥运会。关于接纳日本观众的决定将在6月作出，这取决于该国中共病毒疫情发展情况。
