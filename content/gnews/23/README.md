据路德社4月10日报道，美国民主党和共和党将共同推出一项重要法案，即“2021年战略竞争法案”。周三，参议院外交委员会将对该法案进行讨论和表决。外交关系委员会主席梅嫩德斯（Bob Menendez）相信该法案将会以压倒性的优势获得批准通过。该法案是参议院反共广泛努力的一部分。参议院多数党领袖、民主党人查克‧舒默（Chuck Schumer）此前[宣布](https://www.epochtimes.com/gb/21/4/8/n12867315.htm)，将通过推出法案的“快捷通道”，制约中共。该法案是两党联手、快速通过法案的一个典范，是彻底结束中共的重要法案！

“2021年战略竞争法案”授权通过外交和战略转变，来反制北京的扩张。[建议将印太地区](https://www.voachinese.com/a/senate-introduced-comprehensive-china-bill-20210408/5846046.html)的安全援助列为优先要务，加强美国的外交努力，在西半球，欧洲，亚洲，非洲，北极和大洋洲处理中共国构成的挑战；建议加强与盟友在军控领域的协调合作，共同面对中共国的军事现代化和扩张。

据路德社分析，该法案的核心要点之一，是要求每一个联邦部门和机构都指定一名高级官员负责协调与中共战略竞争方面的政策问题。届时，美国在联邦层面、州层面、和政府部门层面，通过法律的方式，将中共彻底踢出。
