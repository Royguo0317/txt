## 印度與中共一戰是印度投向西方的“投名狀”
`2020-09-04 02:34 GM08` [轉載自GNews](https://gnews.org/zh-hant/333286/)

作者：小惠

正如40年前的1979年2月17日至3月16日期間，中共為代表中華人民共和國和越南社會主義共和國之間的那場中越戰爭（中共國大陸稱“ 對越自衛反擊戰”，越南稱“反中國擴張主義戰爭”）。實際是中共拿越共“祭旗”，向西方民主世界表明立場與蘇共決裂的“投名狀”。也正是這場戰爭換來了共產黨實則為了“續命”而實施的“改革開放”，讓共產黨多活了40年。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/04022441/159919799250809_P8316614.jpg)
今天的印度早已看出了中共國即將走向末路，而中共國的崩潰留出的產能缺口與今天印度人口結構和環境完美的契合；也正是因此，印度必須經歷一場與共產黨的戰爭來向歐洲交出一份“投名狀”，保證完美的接過中共倒台之後的這個“製造業大國”稱號的接力棒；同時這場戰爭還可以洗去60年代中印戰爭的恥辱，這次的印度和曾經今非昔比，一旦戰爭爆發，我有利於相信中共未必是印度的對手。

目前可以毫不誇張的說印度已經做好了開戰的準備，印度已經向拉達克地區增兵36000人，目前又增兵75000，印軍真正密密麻麻的在邊境線集結。印度此前就向邊境地區運送了眾多T-90坦克,隨後又不斷在前線地區修建機場，向邊境地區運送大量空軍裝備。其中包括米格-29蘇-30等眾多印度主力戰機,此外印度國產的LCH直升機也在抵達了對峙前線,不久前,印度從法國購買的首批陣風式戰鬥機也交付部署到了前線。印度官方所稱印度已經加強了在達拉克附近的力量部署，約20萬人。不論是軍隊數量，還是軍事裝備，印度已經取得地區性絕對優勢。

除了軍力的優勢之外，外交方面印度同時也和主要西方國家站在了一起：

首先美國參議院軍事委員會通過2021國防授權法案，提出建立“太平洋威懾倡議基金”；同時美國三個航母戰鬥群現身印太海域；美國、澳大利亞、日本和印度組成了，作為抗衡中共的“印度洋-太平洋戰略”四方安全對話。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/04022532/65A429151E7526334B2E13E9A952DA6AE6250A8C_w1080_h719.jpg)
其次就是共產的的“親爹”俄羅斯。根據俄羅斯衛星通訊社報導，俄羅斯與印度打算就相互派遣部隊、軍艦和軍機的流程達成共識。根據這份協議，雙方可以向另外一方派遣，5艘以上軍艦，10架以上戰機和3000人以上部隊。同時，還規定了部隊入境，安全保障、提供住所、醫療援助以及維護設備的流程。普京對於中印兩國的態度已經非常清晰，絕對的支持印度。前些日子，在紐約聯合國總部，經過投票，印度終於如願以償，成功當選為聯合國非常任理事國，任期從 2021年1月開始，期限為兩年。據悉，在6月17日聯大第一輪投票中，印度便以184票的高票成績，當選為聯合國非常任理事國。這其中與俄羅斯從中斡旋有絕對關係，而且對印度提出的擴大聯合國常任理事國的倡議也沒有表示明確反對。據印度媒體報導，五個聯合國常任理事國里唯一投反對票的是中國，由此可見，中共在印度問題上是徹底被孤立了。

在軍事裝備上正如俄羅斯前國防部長伊万諾夫曾所說，“俄羅斯向印度提供的最新式裝備和先進軍事技術中，許多甚至在國際上都是獨一無二的。俄羅斯是世界上唯一與印度在敏感軍事技術領域合作時肯將自己的秘密相售的國家。”如印度採購的俄製蘇-30MKI戰機性能要遠遠強於中共國蘇-30MKK；俄羅斯最新式的T-14″阿瑪塔”主戰坦克；除了T-14坦克，印度還準備購買俄羅斯的”鎧甲-S1″防空系統，還有已經達成協議的s400防空系統等等各種先進武器裝備。

面對如今外部環境，軍事裝備的決對優勢，印度完全會為了自己可以看到的“美好明天”，打出一場關乎民族未來和尊嚴的生死一戰，這就是他走向西方世界的“投名狀”。
