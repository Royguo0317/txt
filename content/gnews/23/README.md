## 夏季出現諾如病毒？西北民族大學通報265名學生不適，確定為諾如病毒感染
`2020-09-05 11:08 GM01` [轉載自GNews](https://gnews.org/zh-hant/335498/)

內病毒專題:文文

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/05110537/image1-1-4.jpg)

9月4日晚間，西北民族大學官方微博發布“關於校內部分同學發生腹瀉等不適癥狀情況的通報”稱，截至9月4日，學校共計265名學生出現腹瀉、嘔吐、發熱等癥狀。從9月2日開始，該校陸續有學生出癥狀，經多部門實地調查、取樣檢測，確定為諾如病毒感染性腹瀉等疊加引發。

由於正處在CCP病毒疫情期間，各大中小學校又在恢復開學，這樣的聚集性感染，不得不讓人繃緊神經，懷疑是否與CCP病毒感染相關。

通報中稱經對27名有發熱癥狀的學生進行了新冠病毒檢測，均為陰性。而在7名有癥狀的學生中發現了4人有諾如病毒，3人沒有。

是否真是諾如病毒引起的，我們不得而知，但是，夏季出現諾如病毒感染是非常少見的。諾如病毒具有明顯的季節性，人們常把它稱為“冬季嘔吐病”。

中共國內的各種不透明，可以讓中共隨意掩蓋事實真相，隨意甩鍋疫情來源。唯有消滅疫情的根本源頭中共，建立起受人民監督的透明政府，民眾才能得到真實可靠的信息，才能真正控制好疫情。之前中共報道的其他群體性感染事件：

1. 8月23日，安徽淮南市壽縣保義鎮493人發熱腹瀉，中共稱誌賀氏菌感染。
2. 3月26日，貴州錦屏縣209名高三學生集體發熱腹瀉，中共稱食物中毒。
