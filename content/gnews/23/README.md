## 【世事解评】今天你被“诈”了吗？
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/1085631/)

作者：纽约香草山 猪逻辑

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3.jpg)



近日，中共强制要求国人下载一款“国家反诈中心”手机软件APP，号称这是国家层面为了防诈而推出的，可有效防范诈骗，守护群众的“钱袋子”。有网友曝光称，如不安装不让进办公大楼，不让进入政府部门办理业务，无法入住酒店、小区，无法乘坐地铁等公共交通，不让上班，不让上学，等等。对此有网友感慨，“我真没有想到有天让我感受到强权”。

![]()![](https://gnews.org/wp-content/uploads/2021/04/0d645c074c5bed63d5e0b6765957f2de.jpg)

楚门的世界，在真实世界中上演。总有一双隐形的眼盯着你的手机隐私，监控你的行为行踪，甚至监控你大脑里的想法。这就是中共国。事实上中共通过手机针对公民的监控时来已久。墙内很多手机蹊跷地自动生成空文件夹可能叫着新浪、百度、腾讯的名字，删除后又自动生成，令人生疑。如今强制安装“国家反诈骗中心”手机软件，一方面为防止百姓翻墙收看到真实的信息，可能另一方面是为了阻止后期的G系列数字货币等大规模在墙内通行，进而造成资金自由流出的局面，这正是中共最为担心的。

以反诈骗的名义“强装软件”，听上去很美很动听，又是防止电信诈骗、网络贷款，又是防止网络刷单、“杀猪盘”，等等。但中共国鼠警一窝根本不是啥秘密，这些诈骗或涉黑背后多有公安或监管人员的包庇纵容甚至串通勾连分赃。中共所有的恶，都打个一个冠冕堂皇的理由，让你脱掉内裤，还要以检查忠烈的的名义，要强奸你，还要代表着人民和政府合法而为之，要共你的财产，就打着奉献的名义，要你永远相信主人的领导和猪圈的幸福安全，就要告诉你猪圈外面有多危险。而事实上，中共就是制造诈骗最多的祖师爷，电视里天天大言不惭宣传假新闻，各类砖家叫兽站台推销假药，近年来泛亚、易租宝等各类P2P，令多少家庭倾家荡产有的甚至家破人亡至今未能解决。

![]()![](https://gnews.org/wp-content/uploads/2021/04/id12871692-IMG_7469-600x400-1.jpg)图片来源：大纪元

中共不解决人民提出的问题，只解决提出问题的人民。打着“反诈骗”的目的强制或诱导安装软件，以及建立大规模监控监听机制的最主要目的在于定点清除、打击和阻止反对者的声音，将其扼杀于摇篮。醉翁之意不在酒。不然，大街上到处监控头，为何需要它们发挥作用找到被拐儿童，失踪大学生，以及一些重大事故的关键时候，总能“及时地出错”呢？
