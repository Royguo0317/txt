1月26日，陈薇受命率军事医学专家组紧急赶赴武汉，开展新冠病毒疫苗研发攻关工作。

也就是说，在墙国百姓还不知道发生了这么危险的事情的情况下，陈薇院士已经开始了疫苗研发工作。可至今，CCP还在强调没有隐瞒疫情！

3月16日，陈薇带领科研团队研制的新冠病毒疫苗，成为墙国第一个获批正式进入临床试验的疫苗。

也就是说，仅仅1个半月，陈薇的疫苗不仅研制出来了，还获批进行临床试验了。针对至今仍在困扰着全球的新冠，那些发达国家都束手无策，而我们，连马桶盖都要千里迢迢背回国去的国民的一员之陈薇女士，您这是吃了什么样的仙丹？而且，临床试验审批部门也是异常的给力，平时老百姓办个户口、改个名都要盖好几个章，要证明你爸是你爸吗，这怎么突然就一步登天了？别怪我这个“人民”太笨，实在是难以理解！

资料确认中，笔者看到这样一段陈薇院士的慷慨陈词（括弧中的内容是笔者的读后感）。

原话是：我们的团队研发能走到今天，过程中首要的是团队的定力，还有我们对自己技术的信心（这几句铺垫是墙国发言的必备功夫词，可以忽略不计），还有对国内外法规的了解（这句纯属忽悠墙国老百姓，你们懂什么国外法则，这一句就可以给你打懵，让你无语）。我们把几十年积累的十八般武艺（这句更是忽悠百姓，几十年积累了什么？仅就诺奖，墙国远远落后于世界，拿出过什么真正傲人的科技成果？），应该说能想到的、能做到的，都用上了！（最后这两句我信！什么猫腻都敢用，这我真的相信！）。

另外，陈薇院士的另一段话，笔者很难理解。

原话是：为尽快破解病毒密码，研制出疫苗，陈薇院士带领的科研团队与后方科研基地联合作战，以军人的使命刻苦攻关，经过成百上千次试验，团队不断取得突破。

难解之一，病毒传播本是生物医学类研究领域的问题，为什么扯上了军人？难不成这是不打自招了的生化武器研究吗？
