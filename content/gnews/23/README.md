这片土地的下面是几千年的文化。

文化，一个多么尴尬的词，

无墨的笔碰上无底的根。

一个人和身边的山川河流悄悄对话。

百无一用是书生，崖山之后还是中华。

文人，一个多么耻辱和光耀的名词。

与墨同哭，黑字的背后灼烧千斤灵魂，

在惨白的尘世里铺开白纸吧，天就快亮了。

天亮了：人们就忘记了。

书写：为自己点灯。
