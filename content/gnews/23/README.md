COVID-19病毒在全球已导致370多万人死亡，过去六个月未在动物和人类身上进行测试的大规模实验性疫苗产生的副作用数量和频率是前所未有的。接种疫苗仅4个月后，欧洲药品管理局(EMA)针对疑似药物反应病例的EudraVigilance系统已经无法应对大量报告的副作用。由于EudraVigilance数据库系统的存储限制为199,801行，数据库目前处于容量状态，在添加新数据之前必须备份旧文件。美国疫苗不良事件报告系统(VAERS)也面临同样的问题，从5月开始将不再更新所有疫苗不良事件数据，这意味着在5月之后发生不良事件或死亡的受害者可能不会被计入数据库中。

本章将重点介绍在欧洲和美国发生的COVID-19疫苗的不良反应数据，在开始讨论之前，我想提出一个即使7岁孩子也能理解的风险/效益分析:

1. 为什么各国政府都禁止了HCQ等有效的预防和治疗药物，而只推广没有临床证据证明有效的疫苗?
2. 为什么主流媒体没有报道疫苗引起的不良事件?
3. 疫苗是否已经在人体上进行了安全测试?疫苗是否获得FDA/EMA或其他机构的批准?你知道疫苗的成分吗?
4. 你知道接种疫苗造成的伤亡数字吗?
5. 你知道疫苗的长期副作用吗，比如生育能力?疫苗生产商是否对造成的任何损害或伤害负责?
6. 你知道在哪里可以找到准确的不良事件信息吗?


截至2021年6月6日，欧洲共有13867人死亡，1487353人发生其他不良事件。欧洲目前使用的四种主要实验性疫苗如下:

– COVID-19 mRNA vaccine Moderna (CX-024414)

-辉瑞生物科技公司的covid-19 mRNA疫苗-阿斯利康COVID-19疫苗(Chadox1 ncov-19)

-杨森(AD26.COV2.S) COVID-19疫苗

对这些数据的比较表明，死亡数字正在迅速增加，目前由于接种疫苗，平均每天有110多人死亡。
