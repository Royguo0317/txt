## 从烟院士到酒院士（5）

先不说”烟院士”能选上，也不说’酒院士”能不能选上，看看什么人选不上吧。以下两位科学家的经历大家肯定非常熟悉。

屠呦呦（1930年12出生）， [强国中医科学院](https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%9B%BD%E4%B8%AD%E5%8C%BB%E7%A7%91%E5%AD%A6%E9%99%A2)终身研究员兼首席研究员，强国中医科学院中药研究所青蒿素研究中心主任，博士研究生导师，抗疟药[青蒿素](https://zh.wikipedia.org/wiki/%E9%9D%92%E8%92%BF%E7%B4%A0)和[双氢青蒿素](https://zh.wikipedia.org/wiki/%E5%8F%8C%E6%B0%A2%E9%9D%92%E8%92%BF%E7%B4%A0)的发现者。这些[抗疟药](https://zh.wikipedia.org/wiki/%E6%8A%97%E7%96%9F%E8%8D%AF)，为南亚、非洲和南美等热带的发展中国家的人改善了健康状况，被认为是20世纪[热带医学](https://zh.wikipedia.org/wiki/%E7%86%B1%E5%B8%B6%E9%86%AB%E5%AD%B8)的显著突破。

2011年9月，屠呦呦获得当年世界临床医学界最高奖项“[拉斯克临床医学研究奖](https://zh.wikipedia.org/wiki/%E6%8B%89%E6%96%AF%E5%85%8B%E4%B8%B4%E5%BA%8A%E5%8C%BB%E5%AD%A6%E7%A0%94%E7%A9%B6%E5%A5%96)”，成为首位问鼎该奖的强国人。

2015年10月5日，屠呦呦因在研制[青蒿素](https://zh.wikipedia.org/wiki/%E9%9D%92%E8%92%BF%E7%B4%A0)等抗疟药方面的卓越贡献，与美国生物学家[威廉·C·坎贝尔](https://zh.wikipedia.org/wiki/%E5%A8%81%E5%BB%89%C2%B7C%C2%B7%E5%9D%8E%E8%B4%9D%E5%B0%94)、日本科学家[大村智](https://zh.wikipedia.org/wiki/%E5%A4%A7%E6%9D%91%E6%99%BA)，共同被诺奖委员会授予该年度[诺贝尔生理学和医学奖](https://zh.wikipedia.org/wiki/%E8%AF%BA%E8%B4%9D%E5%B0%94%E7%94%9F%E7%90%86%E5%AD%A6%E6%88%96%E5%8C%BB%E5%AD%A6%E5%A5%96)，以表彰三人研究出针对一些最具毁灭性的寄生虫疾病具有革命性作用的疗法”。屠呦呦独享其中一半奖金。

屠呦呦是历史上第三位因[疟疾](https://zh.wikipedia.org/wiki/%E7%98%A7%E7%96%BE)研究获奖的人，首位[华人](https://zh.wikipedia.org/wiki/%E8%8F%AF%E4%BA%BA)[诺贝尔生理学或医学奖](https://zh.wikipedia.org/wiki/%E8%AB%BE%E8%B2%9D%E7%88%BE%E7%94%9F%E7%90%86%E5%AD%B8%E6%88%96%E9%86%AB%E5%AD%B8%E7%8D%8E)获奖者，亚洲第二位及首位[华人](https://zh.wikipedia.org/wiki/%E8%8F%AF%E4%BA%BA)女性自然科学类诺贝尔奖得主，也是首位接受[强国高等教育](https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%8D%8E%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD%E9%AB%98%E7%AD%89%E6%95%99%E8%82%B2)，且在强国大陆进行研究工作的自然科学类诺贝尔奖得主，同时是第11位[拉斯克临床医学奖](https://zh.wikipedia.org/wiki/%E6%8B%89%E6%96%AF%E5%85%8B%E5%A5%96)和[诺贝尔生理学或医学奖](https://zh.wikipedia.org/wiki/%E8%AF%BA%E8%B4%9D%E5%B0%94%E7%94%9F%E7%90%86%E5%AD%A6%E6%88%96%E5%8C%BB%E5%AD%A6%E5%A5%96)双奖得主。

2017年1月9日，[强国国务院](https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%9B%BD%E5%9B%BD%E5%8A%A1%E9%99%A2)将2016年度强国科学界最高荣誉“[国家最高科学技术奖](https://zh.wikipedia.org/wiki/%E5%9B%BD%E5%AE%B6%E6%9C%80%E9%AB%98%E7%A7%91%E5%AD%A6%E6%8A%80%E6%9C%AF%E5%A5%96)”授予屠呦呦。屠呦呦由此成为第一位获得国家最高科学技术奖的女性获奖者和非[两院院士](https://zh.wikipedia.org/wiki/%E4%B8%A4%E9%99%A2%E9%99%A2%E5%A3%AB)获奖者。

然而，如此卓越的世界顶尖科学家，在当年强国院士评选中，落选了。

颜宁，1977年出生于山东章丘。1996年进入清华大学中学习，2000年获得清华大学学士学位。从清华大学毕业之后，到美国普林斯顿大学分子生物学系学习。2004年，毕业后颜宁继续在普林斯顿大学从事博士后研究。2007年，清华大学医学院聘请颜宁担任教授。

在清华任职期间，颜宁和其研究小组在《自然》、《科学》和《细胞》这世界三大顶尖科研杂志上一次次研究成果公布，其中一项研究成果还被《科学》杂志评选为“2009年科学十大进展”。颜宁和团队们取得的成就还不仅如此，早在留学期间，颜宁就在研究膜蛋白相关内容，回国组建研究团队后，又继续该项领域的研究。仅用6个月的时间，颜宁团队就解析出人体14种葡萄糖转运蛋白中GLUT1的结构，将膜蛋白研究推上崭新的高度。这项研究成果可是世界结构生物学家们花费近20年的时间都没能攻破的难题。

在这些丰厚的成果下，颜宁先后获得国际蛋白质学会青年科学家奖和赛克勒国际生物物理奖。不久之后，颜宁又获得“中国科学之星”和“影响中国2017年度科学人物”的称号。

有如此众多研究成果的顶尖科学家颜宁，在2017年强国院士评选中，也落选了。

同年，颜宁受普林斯顿大学邀请，离开清华大学前往美国任教，两年后的2019年，颜宁当选为美国国家科学院外籍院士。

院士不应该是一国最顶尖科学家的代表吗？为何在强国选不上的院士，却能当选美国国家科学院外籍院士？是美国的院士比强国低级吗？是强国科技强大到连诺贝尔奖获得者都不值钱了？显然不是的！
