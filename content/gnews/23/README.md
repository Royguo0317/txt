![]()![](https://gnews.org/wp-content/uploads/2021/04/2021-04-27-coverphoto.jpg)（图片来自网络）
4月13日，中共军方门户网站发文《马云湖畔大学被勒令关停！巨大阴谋浮出水面》。文章直言马云的湖畔大学，是比财阀本身更恐怖、更恶性的“党阀”，冠以大学之名，用树立朝圣形象的做法铸起山头，这比柳传志的“泰山会”走得还要远。文中把湖畔大学比作东林书院、黄埔军校，马云想要成为蒋介石一样有巨大政治成就的校长；暗指湖畔大学已经不再是一个民营企业学校，而是要钱有钱，要人有人，试图推翻CCP的一个越来越庞大的组织。

联想到墙国媒体在之前报道过的一则新闻，2021年1月21日，成立了28年的富豪圈子“泰山会”被解散了，解散前共有会员16人：联想柳传志、四通段永基、万通冯仑、泛海卢志强、复星郭广昌、远大张跃、信远控股林荣强、巨人史玉柱、百度李彦宏和段永平、科海陈庆振、科瑞郑跃文、思达汪远思、横店集团徐文荣、和光商务吴力、华谊兄弟王中军。马云也曾经是该组织成员。

现在，诸如此类批判马云的文章，在墙国媒体上不少，这是对彻底消灭马云或者彻底收割所谓民营企业家的资产做舆论铺垫吗？笔者在此作一个肤浅的解读。

湖畔大学，被中共媒体比作是比财阀本身更恐怖、更恶性的“党阀”，因为它是堡垒式、摇篮式的。去年年底，占地两万亩的湖畔大学云南分校，被勒令暂停。

下面简单归纳一下墙国媒体对马云批判和定罪的主要焦点：

一、马云被比作胡雪岩，形成独立王国。

湖畔大学绝对不是一般意义上的传道、解惑的学校，而是一个圈子，是一个牛叉商人的圈子。 “当资本壮大到一定规模时，容易妄自尊大、忘乎所以到僭权越位，到觊觎公权力，进而对社会造成危害”。前有吕不韦，后有胡雪岩。

湖畔大学这种招收精英、培育精英的模式，被称为是变相否定中共国的人才选拔机制，是典型的脱离群众，撕裂中（共）国人才群体，客观上形成你们和我们！这被CCP认为是非常危险的信号。
![]()![](https://gnews.org/wp-content/uploads/2021/04/2021-04-27-figure1.png)
二、马云、柳传志、冯仑、郭广昌、史玉柱、沈国军、钱颖一、蔡洪滨、邵晓锋等通过创办湖畔大学紧密联系到了一起。马云成了商界领袖。“……创业经验、三十名以上员工、纳税三年，必须有三位保荐人”，被CCP认为是马云笼络人才的手段，为CCP所忌惮。

三、以讲学的名义聚集势力……最后形成了官商大合流，成为一个巨大毒瘤。马云的野心，被比作东林书院、帝国危机之源。东林党，敲响了明帝国亡国的丧钟！马云的湖畔大学被中共认为是颠覆CCP的隐患。

“这些精英前台人物，不甘于被操控的命运，要想独立自主而建立的新组织。商、官合流，形成新的生态；这个新生态系统的后面，必然有着更大利益诉求。”通过湖畔大学，资本巨头们有着合流的趋势。

墙国媒体甚至说，蒋介石就是通过黄浦军校，实现了政治抱负。“这个学校不是一个民营企业学校，也不和国营企业去竞争……，那么更大的诉求是什么？”

其实马云以及最近被墙国媒体大肆批判的湖畔大学、泰山会，简单来说，成员都是中共的白手套。在帮助中共维稳过程中，他们是帮助中共吸干百姓血液的帮凶，而且也得到了极大的好处。但在和CCP一起作恶的过程中，他们对CCP的残暴和黑帮作派应该有切身的认识，试图彻底消灭另一派，应该是完全可能的。

究其原因，CCP目前有了消灭马云帮的想法，首先是缘于CCP内部黑帮之间的火拼，这与政治抱负、是否建立普世价值和民主、是否有益于全国普罗大众的新中国没有任何关系。马云帮不过是站错了队，跟错了人而已。或者说，马云帮和王健，吴晓辉等人一样，在CCP中南坑老杂毛的眼中，不过是一张手纸而已，现在，到了该扔掉的时候了。
