## 【世事解评】公益诉讼数量上涨，难掩中共以假治国、以黑治国之实
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/1177481/)

作者：香草山写作组 鹰(文言)



![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/04221555/s2-2.jpg)



环球网5月5日转载新华网报道，中共最高检察院日前公布2021年主要办案数据，一季度公益诉讼数量32914件，同比上升99.5%。

参考2020年2月份中共最高检察院网站文章“自爆”数据，2019年1月至11月，公益诉讼案件立案113848起（平均每月约10349起），其中提起诉讼的4019件，占比仅有3.5%。结合上述21年数据，一季度32914的立案数量仅比19年上涨6%，至于99.5%的同比上涨原因在于2020年初的CCP病毒大爆发，全国大封城下居民都被“拘禁在家”，又何谈立案和诉讼了。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/05211633/Screen-Shot-2021-05-05-at-9.15.29-PM.png)
公益诉讼重点在于清理环境治理和资源保护领域，以及查办破坏生态、督促恢复耕地、林地、湿地、草原之类。因而，其中民事立案占比较少。2021年一季度的民事公益诉讼案件占立案总数10.3%；生态环境资源领域立案占比54.5%；国有财产保护领域占比3.9%；食品药品安全领域占比19.1%；新领域公益损害问题占比21.9%。以上直接关切民生的也仅有食品药品安全而已。公益诉讼检察依旧是保护国家利益及权贵阶层的“权力游戏”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/05211613/Screen-Shot-2021-05-05-at-9.15.33-PM.png)
郭文贵先生的被发红通、资产无故被查封、员工被抓被虐待；闫丽梦博士的被构陷，郝海东先生、叶钊颖女士因讲真话和支持爆料革命被国内网络“移除”；再有武汉吹哨人因“寻衅滋事”而电视“大游行”，再到被谋杀或被消失……实名举报山西疫苗造假的记者被打击报复；举报违规排污、黑煤矿的个人信息被有意曝光，以及每年数不尽的上诉无门、上访被拦截被盯梢……中共的种种枉法行径以及恶意依法抢钱圈人的行为，都映衬着其毫无法制观念、党规大于一切的黑社会传统。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/05211558/Screen-Shot-2021-05-05-at-9.15.39-PM.png)
再多的立案根源也只是国有财产的再分配和对社会财富的掠夺，根本无关百姓利益的维护。执法者和立法者，都成了国家机器的提线木偶，法律也沦为现代版的“江湖规矩”。中共体制下立案诉讼、检察司法再多的作秀都是强拉硬扯的“立牌坊”。欺世盗民的目的和本质不会更改，以黑治国、以假治国这一中共独裁政权的特性最终只能随着政权的崩塌而消散于世。
