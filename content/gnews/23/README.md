**西班牙2021年1月16日电/西喜社——**墨西哥周五公布的中共病毒病例创下了历史性的高峰，新确诊的感染病例达21,366例，约为一周前日增数的两倍。该国还录得1,106例死亡病例。

目前还不清楚是否由于英国病毒变种的存在而出现的飙升，其中到目前为止只有一例被证实在一名来访的英国公民身上。

该国现在已经看到了近161万的总感染，并已看到记录超过13.9万死亡。

该国极低的检测率意味着这是个低估的数字，官方估计实际死亡人数接近19.5万。由于检测工作做得太少，最近几周，所有接受检测的人中有8%后来死亡；通常，只有症状严重的人才会接受检测。
