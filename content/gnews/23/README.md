9月11日，中共外交部发言人赵立坚在例行记者会上应询介绍，农融已被任命为新任中国驻巴基斯坦大使，并将适时赴任。值得注意的是，与大多数外交官不同，农融现任广西壮族自治区民族宗教事务委员会主任、党组书记，自治区党委统战部副部长。在他的简历中，并没有外交系统的工作经验，而是直接从中国边疆省区到“巴铁”国。

农融出生于1967年9月，壮族，广西马山县人。1991年7月，农融从北京大学技术物理系原子核物理学专业毕业后回到广西工作。此后直到这次任命之前，他的工作经历都没有离开过广西。他曾担任过广西壮族自治区对外贸易经济合作厅办公室副主任、合作处副处长、处长，自治区商务厅东盟国家及港澳台地区处处长，广西国际博览事务局综合协调处处长、党组成员、副局长，中国-东盟博览会秘书处秘书长助理，防城港市委常委、副市长、市委副书记，贵港市委副书记、市长。2019年12月，农融任广西壮族自治区民宗委党组书记、自治区少数民族语言文字工作委员会主任、自治区党委统战部副部长(兼)，2020年1月任自治区民宗委主任。

笔者寻遍网络，总算找到农融最大的荣誉是第十三届全国人大代表，最新最重要讲话是今年5月，农融在接受采访时表示，广西以铸牢中华民族共同体意识为主线，持续推进民族团结进步创建工作，形成各族人民亲如一家，民族关系十分融洽的大好局面，赢得并始终保持“堪称我国民族团结进步的典范”的崇高荣誉。

可惜，一个北京大学技术物理系原子核物理学专业的毕业生，本可以成为科学家，却被中共一直流放边疆；可叹，在中共即将终结的路上，农融依然感恩戴德去上任，走向可想而知的命运；可笑，党国已难找外交人才，居然将“巴铁”交给并没有外交系统的工作经验的地方官员。连中共报纸都在讥笑这一神任命，或者“巴铁”对中共已没有利用价值？
