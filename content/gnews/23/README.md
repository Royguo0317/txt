## 奥斯汀签署内部指令以统一国防部的中共国行动

国防部长劳埃德·J·奥斯汀三世（ Lloyd J. Austin III）今天发布了一项内部指令，将国防部的工作重点放在应对中共国作为这一国家头号步伐挑战上。

[!\[\]()!\[\](https://spark.adobe.com/page/SPgn9aUsu3huq/images/758f1d85-cd30-46fa-b489-19c4e9444c5d.jpg?asset_id=9be0e7f0-e15b-4290-bd15-d3cd20860f2b&amp;img_etag=%22a7da408c9f66121a701228e1e3c558a2%22&amp;size=1024)](https://spark.adobe.com/page/SPgn9aUsu3huq/images/758f1d85-cd30-46fa-b489-19c4e9444c5d.jpg?asset_id=9be0e7f0-e15b-4290-bd15-d3cd20860f2b&amp;img_etag=%22a7da408c9f66121a701228e1e3c558a2%22&amp;size=1024)图片：海军陆战队上校帕特里克·克罗斯利

奥斯汀接受了他在2月份成立的中共国问题特别工作组的建议。该文件是保密的。

奥斯汀在一份关于该指令的书面声明中说：“我今天提出的倡议嵌套大的美国政府对华方针之中，并将有助于为我们正在制定的国防战略提供信息。”

国防部长感谢了特别工作组的成员，并说现在是采取行动的时候，以确保国防部的行动与面临的挑战相匹配。

**“我今天部署的行动将提高我们军队的能力，以振兴我们的盟友和合作伙伴网络，增强威慑力，并加速发展新的作战概念、新兴能力、塑造未来的部队立场和现代化的文职和军事人员队伍。”**

**—— 国防部长劳埃德·J·奥斯汀三世**

国防部高级官员在谈到背景时说，该指令将为应对中共国带来的挑战给予更大的关注和统一的行动。

这些倡议呼吁国防部投资美国独一无二的盟友和合作伙伴网络。一位高级官员说，他们还规划了在所有战争领域加强威慑力的必要性，并寻求加速发展新的作战概念。

他说，这些举措“目的在于简化和加强与美国盟友和伙伴的合作，特别是在印度——太平洋地区。”

这位官员说，虽然特别工作组的建议集中在该部门，但它们仍然“嵌套在《临时国家安全战略指导》中，并补充了各部门、机构和白宫在中共国政策方面的多方面工作”。在未来，这些结果将被用于制定国防战略和其他审查。

内部指令的编写有明确的目标，有实施的最后期限，以及监督和问责的机制。该官员说：“在可能的范围内，我们设计的这些行动贯穿了该部门的命脉，利用现有机构，只在必要时建议采用新的流程。”

[!\[\]()!\[\](https://spark.adobe.com/page/SPgn9aUsu3huq/images/eec1aebd-7422-465c-88cc-8c429ff14e8b.jpg?asset_id=fcfbc0f6-65b0-40d1-8d93-ee64be2572f5&amp;img_etag=%22663eb904bd50071d229b0806812c9590%22&amp;size=1024)](https://spark.adobe.com/page/SPgn9aUsu3huq/images/eec1aebd-7422-465c-88cc-8c429ff14e8b.jpg?asset_id=fcfbc0f6-65b0-40d1-8d93-ee64be2572f5&amp;img_etag=%22663eb904bd50071d229b0806812c9590%22&amp;size=1024)图片：海军陆战队军士长莎拉·纳多

该指令是保密的，但该官员可以分享行动范围的一个例子。他说：“为确保该部门拥有我们有效竞争所需的人员，部长已责成……负责人事和战备的国防部副部长更新专业军事教育和文职专业发展，以使该部门与针对中共国的优先事项保持一致。

他说，国防部长还将通过实验和原型设计阶段加速联合作战的概念。

对军人来说，最引人注目的行动可能是与盟友和伙伴的联系。军方之间的关系是这种外联的一部分，将有更多与盟友和伙伴的军事演习。

特别工作组的工作已经完成，它将会被解散。
