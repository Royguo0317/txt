美國總統川普8月6日發佈行政命令,依據《國際緊急狀態經濟權力法》(IEEPA)4545天內禁止任何與微信相關嘅交易緊,[美國微信用戶聯合會(](https://uswua.org/)[U.S. WeChat Users Alliance,USWUA](https://uswua.org/)[，](https://uswua.org/)[USWUA](https://uswua.org/)[)](https://uswua.org/)隨後喺8月27日提告,控訴該命令中嘅「交易」定義不明,若通訊都算「交易」,則違反憲法保障的言論自由權利。

**USWUA****是非營利組織美國微信用戶聯合會嘅簡稱。**   **由**美國紐約同新州執業律師曹英,加州和俄勒岡執業律師朱可亮,華盛頓和紐約執業律師倪非,加州、紐約同新澤西執業律師袁钢同紐約執業律師吴圣洋五位華裔律師於8月8日共同成立,目的係代表喺美國境內嘅微信用戶喺聯邦法院發起訴訟,並請求法院宣佈總統令無效。

普遍認為:該團體由清一色嘅華人律師組成, 佢哋雖聲稱代表喺美微信用戶,中國嘅背景不言而喻。  此次對於美國政府禁微信令出台,中國方面如果毫無反應反而極唔正常,對犀利国嘅國民無法交代。  美國政治三權分立嘅生態正好係中國明裏暗裏嘅下手機會。

同時對於**USWUA**告到嘅理由係微信禁令違反美國憲法第一修正案,限制言論自由、媒體自由以及組織和集會自由云云。  其實理由係企唔住腳嘅,因為正是微信作為社交平台,佢受到中國政府的審查和禁止不同言論嘅做法,違反咗美國憲法嘅第一修正案。  係微信被禁嘅主因。  同時微信支付喺美企業廣泛應用,逃離了人民幣美元結算機制,都為偷稅漏稅創造了條件。  都係美國取締微信嘅主因之一。
