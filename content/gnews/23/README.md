“人口红利逐步向人才红利转变”，5月11日，国家统计局在新闻发布会上高调宣布。

可以说，改革开放三十年的巨大变化，凝聚了农民工的巨大付出。但三十年后，这些人已经到了知天命的年纪，可他们仍在辛苦打拼，他们得到了什么？

随着农民工的老龄化，劳动年龄人口减少，中共又发现新的“富矿”——大学毕业生。

教育部5月13日表示，2021届全国普通高校毕业生总规模909万，同比增加35万。人才红利真的来了。

当老百姓用血汗钱把子女养大，孩子大学毕业后，他们才发现，孩子还是农民工。

在如今这个就业环境下，对于缺乏贵族气质的广大毕业生来讲，想找一份称心如意的工作，真的太难。工作不好找只能接替农民工的岗位。

7万研究生送外卖，硕士转行当月嫂，北大的卖猪肉，大学生修鞋等等，这样的“励志”新闻比比皆是。外卖、月嫂、修鞋、卖肉之前不都是农民工的专属职业工种吗？

中共真是把老百姓当韭菜，割了一茬又一次茬。

都说知识改变命运，在中共国要看人了。比如，小学文化的习近平同志。对于没有红色基因的大学毕业生而言，十年寒窗，大学毕业找不到工作，只能接过农民工手里的铁锨和扫把，当农民工的接班人。

我这话有人肯定不服气，李克强不是普通家庭出生吗？是的，但李克强在习皇帝的眼睛不过是个打工仔，郭先生说过，宁肯当小三，也不做那个窝囊的总理。再看看王公公、杨娘娘，这些思想阉割的人物，和过去的太监有何不同。这些人只不过换个马甲，依然是打工仔。
