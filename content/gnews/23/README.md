## 【香港要闻】海外港人发起「香港约章」吁全球联手抗共
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/977596/)

搜集/编撰：天灭中共

流亡海外的前立法会议员罗冠聪、许智峰，以及张昆阳、梁继平、邝颂晴、黄台仰、梁颂恒昨日（14日）发起「2021香港约章」。希望透过《约章》来凝聚国际战线，集结全球力量，对抗中共霸权及压逼，争取港人的民主自由。

同时，亦期盼同所有中国同胞一齐，结束中共一党专政，呼吁国际社会对抗中共威权扩张，共同守护民主自由的核心价值。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3159.jpg)（图片来源：立场新闻）
**冀用《约章》记载时代信念**

《2021 香港约章》（全文详见文末连结）的开篇写道：「自1980 年代起，港人一直争取民主制度，然而中共一党专政本质不变，经过屡次破坏《中英联合声明》后，一国两制已名存实亡。」

在昨日网上举行的《约章》发布会上，8 位发起人提到，过去有许多民主运动也有发起《约章》，如诺贝尔和平奖得主刘晓波发起的中国《零八宪章》，天鹅绒革命当中的《七七宪章》，故希望以《约章》手法，让属于这个时代的声音得以流传，信念得以记载。

**《约章》的主要内容分为三个篇章。**

**「香港篇：港人自主」**

第一部份是「香港篇」，指香港人是一个独特的共同体，拥有自己的文化、历史、经历以及价值观。这份身分认同应受到尊重，以传承这些珍贵的特质。
因此，香港人应有权决定香港的前途和事务，包括民生丶政制丶宪制等改革。同时，香港人亦应享有所有国际公约内列明的自由以及权利。
​另外，香港应实行三权分立，司法丶行政及立法机关互不统属，法治制度应以限制公权力、彰显公义为最高原则，法官判决应合乎人权及法治的标准，而本地最高法院应享有宪法解释权，不受干预。

**「中国篇：中港一起反抗，结束一党专政，共行自治之路」**

第二部份的「中国篇」开门见山，正视立场，提出「香港的民主运动针对中共一党专政的政治暴力，而非中共政体辖下的中国国民。」

中国共产党垄断所有权力，实行一党专政，丝毫没有民主自治、分权制衡、法治平等、 人权尊严的理念。

中共不断透过各种国家计划将意识形态输出，削弱民主制度正当性、巩固独裁专制，用国家机器打压、逼害异议人士、人权律师，以大数据监控人民思想，侵犯隐私，军事威吓台湾，以及对少数民族的文化清洗和种族灭绝。

​因此，呼吁致力推动中国政体转型的中国国民一起反抗暴政，共行自治之路。唯有结束一党专政、建立民主制度，才可在中国境内革新旧有体制，实践民主和自由的价值。

**「国际篇：全球连结，制衡中共」**

最后一个部分的「国际篇」提出应以跨越党派的信念进行国际倡议工作，希望连结更多支持香港的盟友，争取国际社会，民主政府、跨国企业、民间组织等共同合作，监察及制衡中共政权的扩张，以及其对全球民主自由体制的政治、经济、法律、军事及科技的威胁。另外，与各地反对极权，争取民主、自由的抗争者相互连结，共同促进、实现自由、自主、平等、多元、公平和民主共治的普世价值。最后呼吁国际社会不应再受惠于中共的人权侵害，香港作为自由世界的一部分，与民主国家共享相同价值，国际社会也应共同守护被威权挑战的自由价值。

**中国「零八宪章」**
2008年12月9日，由张祖桦负责起草、刘晓波等人修改，并由303位中国各界异见人士及维权律师，包括鲍彤、丁子霖、刘军宁、戴晴、于浩成、浦志强、张祖桦、茅于轼、冉云飞等签署，就促进中国民主化进程、改善人权状况，建立中华联邦共和国解决两岸问题及各民族问题倡议宣言，呼吁中国政府进行修宪，实行改革。
《零八宪章》以捷克的《七七宪章》为蓝本，主要针对自由、人权、平等、共和、民主、宪政这6个概念，提出修改宪法、分权制衡、司法独立、宗教自由、言论自由、公民教育、财产保护等19个具体主张。

这份《宪章》是经历长期的人权灾难和艰难曲折的抗争历程之后，觉醒的中国公民认识到，自由、平等、人权是人类共同的普世价值；民主、共和、宪政是现代政治的基本制度架构。

**评论：「中共不灭，抗争不止」**

这条争取民主自由的坎坷之路，从六四运动至今，中国人走了32年，死伤无数；从九七回归至今，香港人走了23年，损失惨重。​

2008年刘晓波发起「零八宪章」、2020年「爆料革命」发起「新中国联邦宣言」、2021年香港人发起「香港约章」。

纵然民主自由的追寻之路漫长而艰难，但我们奋斗的脚步永不停下。
