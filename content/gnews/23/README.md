## 文錦說電影——此時上映《八佰》，妳怎麽看？
`2020-09-07 08:42 GM77` [轉載自GNews](https://gnews.org/zh-hant/339068/)

近日，壹部講敘國民黨八百壯士堅守四行倉庫的影片《八佰》上映，獲得墻內幾乎所有門戶網站壹致的高度贊譽，稱其燃血程度堪比《戰狼2》。

這部由《老炮兒》的導演管虎執導的影片。在去年7月15日以“技術原因”為由曾被禁止上映，外界壹度猜疑其被叫停的根本原因，是反映國民黨抗日的主題及片中多次出現的青天白日旗。

然而，如此美化國民黨的影片，為什麽會在此時被允許重新上線？為什麽中共黨姓媒體壹致將其放在頭版頭條大肆宣傳？

**《八佰》所處時代背景**

《八佰》講敘國民黨抗日時期背景下，1932年1月28日日軍突然進攻上海，因危及英法美意在上海的租地，在多國使館調解下中日雙方簽訂了《淞滬停戰協定》，日軍以不再進攻上海為條件，駐軍在上海自立防區，英法美意各租地保持武裝中立，並規定不允許正規國軍駐軍上海市區內。

日本口頭上表示避免戰爭，暗地裏卻大規模增兵，為應對日軍再次突襲上海，八十八師等國軍部隊在上海外圍秘密備戰。1937年8月13日淞滬會戰打響，國軍人員傷亡慘重，防線全線松動，主力西撤迫在眉睫。

同年10月27日，八十八師二六二旅五二四團附參謀謝晉元臨危受命，率五二四團壹營四百余人，留守閘北駐紮四行倉庫掩護大部隊撤退，向國際聯盟表達未放棄上海的抗戰決心，以爭取國際支援。部隊為壯聲勢對外宣稱人數“八百”。

四行倉庫前面是壹條蘇州河，河對面是英租界，日軍在倉庫後方和兩側，前無去路後有追兵。苦鬥四日後，部隊奉命撤到對岸的英租界。八百壯士的浴血奮戰打破了日軍三天攻陷上海的計劃。11月12日夜，上海最終淪陷，歷時三個月的淞滬戰役告終。

**《八佰壯士》VS****《八佰》**

同樣是四行倉庫守衛戰，臺灣1975年就曾拍過電影，名為《八佰壯士》。影片重點刻畫兩名核心人物謝晉元和女童子軍楊惠敏，從上之下展現謝晉元率領“八百壯士”誓死掩護大部隊撤離，和楊惠敏孤身壹人潛入四行倉庫送達國旗並帶出四百多壯士的名單，以示軍民齊心保衛國家的大無畏精神。

管虎的《八佰》則以淞滬戰役大潰敗開篇，在壹片混亂中組編“八百壯士”。編導至下而上著重於描敘平凡的個人，對人性的怯懦多處著墨，再三描繪害怕殺人、屢屢想逃走的軍人。《八佰》展現蘇州河對岸很多民眾的自顧行樂和坐在飛船裏高高在上看“現場直播”的“國際觀察團”， 讓“天堂般的租界vs.地獄般的四行倉庫” 形成鮮明的對比， 突出人性的自私冷漠和國際力量不可信賴。

當“八百壯士”誓死保護青天白日旗飄揚在倉頂上，最終喚醒了對岸民眾的熱血，影片意在告訴觀眾，並非人人都是天生的英雄，但所有小人物集合起來，定能成就壹個撼動人心的故事。

**《八佰》展開的聯想**

跟隨爆料革命的戰友們如果看《八佰》，很容易從四行倉庫和八百壯士身上，聯想到去年反送中以來的香港和冒死抗爭在壹線的香港人。中共遠比日本侵略軍更陰險更殘暴，中共在全世界人的眼皮底下以換防為由派武警部隊進駐香港，強行實施國安法，以黑治港對待壹脈傳承的同胞。

中共用築起的高墻蒙住14億人的眼睛，用謊言掩蓋壹切歷史真相洗腦老百姓，從抗日戰爭到非法執政，從大躍進到大饑荒，從文革到天安全大屠殺，從鎮壓維族人、藏人到以黑治港、強施蒙古漢化教育，從釋放病毒至今中共妄圖掩蓋自己數不勝數的罪行。

中共作惡多端天理不容！近日，美國司法部門公開了以習近平為首的中共盜國賊與美國政客勾結，企圖遣返郭文貴先生的壹連串證據文件，很顯然美國掌握著更多能致這群盜國賊於死地的犯罪證據。

**《八佰》近期上映的險惡用心**

隨著閆麗夢博士揭露病毒真相的層層深入，和以美國為首的全球自由聯盟國家的步步緊逼，習近平用“五個絕不答應”綁架十四億人民對外予以還擊，對內向十四億民眾伸出魔爪。中共擇機上映《八佰》並大肆宣傳，中共將謝晉元之子謝繼民推到舞臺前發聲：“八百壯士謝晉元樹了壹個榜樣，每壹個軍人，都能夠盡自己的責任。勇敢不怕死，中國就能夠跟日本人打，中國就不會亡。”

其險惡用心之意定是：傳遞國家已到了危難時刻的信號，妄圖借此片點燃不明真相的十四億中國老百姓的民族仇恨，將十四億人綁在自己這架即將走向滅亡的戰車上，讓其以八百壯士為榜樣犧牲自我壹起對抗“西方列強“。

《八佰》中那象征著“日落旗不落天長國運長”的青天白日旗，在影片中完全不見迎風飄揚的白日光芒，僅露出大片血色和羞答答的壹抹青天。但無論中共怎麽用技術去處理，卻永遠掩蓋不了歷史的真相，掩蓋不了中共所犯下的滔天罪行，防火墻終將被推到，全世界正義的力量將會與中國人民站在壹起，消滅中共！
