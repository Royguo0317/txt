## 中共超限战灭港计划(二十九)六七暴动之福建帮
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1064591/)

搜集/撰文：岁月如歌 / 封面合成：文粤

根据《自由亚洲中文网》采访苏婆婆得知，六七暴动在香港北角是个主要阵地，华丰国货是大本营幕后指使。华丰国货指挥一群黑帮人到处去搞破坏，在路上埋炸弹，对人伤害很大。这些黑帮人都是些什么人呢？自从清朝开始一直到中共国改革开放，在福建省福清县很多人偷渡到国外。早期偷渡到日本的福清人由于生活的艰难他们就把原来只接收福清人帮会改为也接收其他省来的华人一起组成同乡会，慢慢地这个同乡会就变成了黑社会帮派并扩张到其他地方。在东京他们有地下赌场作为收入，这个帮派与其他国家的黑社会均有合作，比如日本的山口组、美国的黑手党以及前面文章提到的旧中国社会就存在了的三合会都有来往，一起合作在东亚地区贩毒开非法的地下赌场。福建帮又称福清帮或者福龙帮，在台湾称简义帮。福建帮发展到今天在很多国家都是不受欢迎的，不管是公开的移民还是偷渡的蛇头都不愿意接受福建人。福建帮在香港是有名的黑帮，从过去到现在都是。福建帮是靠中共发达起来的，自然听从共产党指挥。北角原来是上海派的地盘，董建华和唐英年就是上海派的，谭耀宗则和苏婆婆同一个工会。后来上海派搬走慢慢地北角成了是福建帮的聚集地，六七暴动福建帮是主力。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4081.jpg)图片来自苹果日报
独有偶，2019年反送中运动中也出现了福建帮，在福建帮老巢港岛北角，炮台山一带袭击游行队伍，支持警察，这难道是巧合吗？反送中运动进入到第99天上街运动已非常激烈，黑警手段也非常残忍，并发出游行禁令，但是都阻止不了港人上街的决心。黑警对上街人士发催泪弹，用高压水车喷射示威人群导致发生中冲突，在追打过程中突然出现了一大批手持棍棒刀械的白衣人，袭击勇武派，对游行的人乱砍乱打，攻击过程中还大喊大叫“支持警察”，“福建人都过来”，“来打蟑螂”等，白衣人不但对上街的人乱打还对路边的人泼洒汽油。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4082.jpg)图片来自自由时报
在福建帮去香港所谓的支持警察行动之前，国内的微信圈等平台很早就传得沸沸扬扬，说有人高价雇请福建帮将到香港去支持警察，对抗港独分子。说福建帮看不下去了，自告奋勇组团到香港打蟑螂，网络上晒出很多福建帮的照片，微博曾一度热搜福建帮。南华早报、苹果日报等多个媒体都相继报导了这个事情。炒得非常热闹，这很符合毛泽东倡导的并被中共视为战争胜利法宝的笔杆子革命——舆论战，“车马未动，舆论先行”。 2019年的反送中运动中发生的721元朗暴力事件， 811北角暴力事件，施暴人员都称自己是中共国人福建人。而针对香港721、811这两个暴力事件，中共媒体和中共喉舌《人民日报》、《环球时报》等都大加赞赏福建人的爱国爱港英勇行为，更呼吁煽动更多的爱国爱港人氏上街备战。当时中联办主任王志民就是福建仙游人，他跟福建帮黑社会有密切联系。 2020年1月王志民被免职换上了浙江义乌人骆会宁。中共说这是福建人自发去香港支援的爱国爱港义举你相信吗？郭文贵先生在爆料中曾经说过中共就是世界上最大的黑帮。这两件大事都有福建帮出现，可以看出中共确实像郭文贵先生说的那样就是个大黑帮，黑帮头头。能够指使黑帮为他卖命的当然是黑帮头头大黑帮了。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4083.jpg)图片来自自由时报
五六十年代的香港工业正蓬勃发展，但是工人很辛苦工资低一个月最多一两百块钱，而且缺少劳工保障，遇到劳资等问题只能靠工会帮忙，而支持他们的是当时的树胶塑胶业工会，这工会是左派控制的的。六七暴动的直接导火索就是，新蒲岗人造花厂厂方降低工价，计件工价每磅假花9毛6分，减到6毛多，工资降低大概三成，工人向厂方交涉无果。厂方后来把啤机部停产，解雇了数百名工人里面有一个叫萧剑辉的。树胶塑胶业工会出面帮他们解决劳资纠纷，当时萧剑辉他们的感觉非常好，他说像大人哄小孩一样，有人支持感觉到不孤单。结果工人们被工会牵着鼻子走，工会要他们去贴大字报，他们感到很无奈但是又不得不做。萧健辉说没办法，我们需要他们帮忙只能听他们的话。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4084.jpg)图片来自立场新闻
工会和工人采取阻止厂方出货的方式来要求厂方赔偿损失，导致双方冲突。警方武力制止抓捕了21人包括萧剑辉和工会主席。左派组织工人前往声援，围观的人越来越多有人借机会发泄对社会的不满。左派媒体《大公报》等趁机把事情扩大化报导。当时的做主编罗孚回忆说，要这样报到是上面的指示，不但要扩大化，还要清算港英100年来的统治罪恶。 《大公报》的报导对当时的影响是很大的，为此几十年之后一直心里不安的《大公报》主编罗孚（已故，93岁）为自己曾经做过的事情道歉。他后来被认为是美国间谍，在北京被软禁10年后返回香港，于1993年3月15日在《当代月刊》发表题为《为香港式文革致歉》的文章自我反省。另外一个为六七暴动所做过的事道歉的人是前“学友社”主席梁慕娴女士。中共多次发声谴责港英政府镇压，香港左派以为大陆要收回香港，便在港九升级反抗，把事情扩大化。
