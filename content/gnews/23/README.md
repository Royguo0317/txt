## 洪災未過、蝗災頻發、糧倉開始“被”著火， 再不囤糧，接下來拿什麽來拯救家人？
`2020-07-04 08:39 GM30` [轉載自GNews](https://gnews.org/zh-hant/253712/)

**作者：麓山之蜂**

自2020年6月以來，中共國已有數省相繼遭遇持續強降雨天氣，城市內澇、泥石流頻發，可以洪水所到之處，完全是慘不忍睹、不忍直視。對于受災受難的老百姓來說，可謂損失慘重，許多老百姓的房屋被衝毀，親人在洪災、泥石流中失去了寶貴的生命。7月2日，長江三峽水庫入流量達到50000立方米每秒，長江2020年第1號洪水發生，長江水域上的三峽大壩、葛洲壩等大壩全力泄洪，這對于長江流域的老百姓來說，無疑又將是一場慘痛的災難。

2020年7月4日，中共天氣網由發布：西南地區至長江中下遊雨勢增強 江南華南繼續“焖蒸”的氣象預警。

今年的強降雨天氣來的比往年早，有些地方在5月底就開始，多數省份則在6月份相繼暴發洪災、泥石流。往年的強降雨則是集中在7、8月份，但是今年的持續性強降雨來的特別早、特別凶、持續時間估計會是特別久。而在水、火面前，一切都是渺小的。所以，許多地方的各種莊稼、糧食作物可能會顆粒無收。對于人口大國、糧食進口大國，這無疑是一個讓人無法面對，但是又不得不接受的事實。所以，大家一定要有緊迫感、危機感，提前、持續性地做好應對接下來更加惡劣的自然災難與國際環境。

除了中共冠狀病毒、南方水災，而中共國的吉林、湖南、廣西相繼爆出蝗災的消息，但中共官方則是輕描淡寫、一筆帶過式報道。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/04082726/%E5%9B%BE1-1.jpg)

2020年6月5日，吉林市官方微博發文[農業部門緊急通知！權力做好事蝗蟲檢測與防控]，據蟲情調查，蛟河市、桦甸市、永吉縣、龍潭區、昌邑區、船營區等地的部分荒地、林地、崗地等處已發生蝗蟲危害，呈點片發生。截至6月4日，全市蝗蟲發生面積約13.4公頃，平均發生密度10-20頭/平方米，密度高的可達50頭/平方米，危害部位大都爲闊葉草本植物的嫩葉，危害狀爲缺刻狀，個別導致整片葉子被吃光。全市農田尚未發生蝗蟲危害，目前蝗蟲處于蝗蝻階段。

同月，據澎湃新聞報道：6月18日從甯遠縣委宣傳部獲悉，大約在6月9日，柏家坪鎮禮仕灣社區茄子園村和棉花坪瑤族鄉關塘村出現大量蝗蟲。10日，當地農業部門利用植保無人機等對村莊周邊的樹木及農作物進行噴藥除蟲，農技人員對河灘邊的草叢進行噴藥。

從數位網友發布出的視頻與爆料得知：廣西省桂林市的農業大縣全州縣遭遇蝗災入侵。視頻中，密密麻麻的蝗蟲落在地面上、莊稼上，連人身上都是密密麻麻的螞蚱(蝗蟲)。農作物的葉子、樹枝上的葉子都被咬殘破了。另外全州縣的安和、石塘、廟頭等村鎮都出現蝗蟲蔓延的迹象，其中尤以紹水鎮爲甚。僅紹水河邊柳樹的受災面積據估計就達100畝以上，農作物也差不多被吃完了。而中共黨媒對此全部是閉口不談，而官方網站與微博上都查不到任何消息。如果此次全州的蝗蟲得到有效控制，那中共黨媒、官方網站與微博絕對是大幅報道。這次的沈默是不是表示蝗災已經到了無法控制的地步呢？

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/04083112/%E5%9B%BE4-1.jpg)

2020年7月1日18時許，位于都勻市沙包堡街道辦事處的都勻市糧食儲備庫發生火災。中共官方稱大火于當日19時30分全部撲滅，過火面積約200平方米，起火原因還在進一步調查中。黔南州應急管理局官方消息稱：都勻一糧食儲備庫失火 無人員傷亡！此文中結尾的“火災無情人有情，希望廣大網友不信謠、不傳謠，積極傳播正能量”，大家有沒有品味出此地無銀三百兩的深意呢？還記得2018年全國糧倉大清查時，許多大糧倉相繼被大火嗎？
