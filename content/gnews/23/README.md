作者：香草山写作组 欧先生 校对/发稿：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/29231938/logo-3-1-1.jpg)



郭文贵先生最近传递的信息提到深层次沼泽会跟新中国联邦合作。结合各种其他信息，可以认为阻碍正道主义的沼泽即将被排干。所以郭先生说川普总统以前以“年”为时间计的战斗过程，目前将踩满油门，七十二小时雷霆行动之下，将在大选结束前结束这场反击战。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/20072839/22-11.jpg)
如果你一直跟随爆料革命就会明白这种好消息和信心的传递，将一扫这段时间大选的荒唐以及目前最高法的决定给战友们带来的心霾。

所以沼泽被排干之后，接下来就是川普总统为代表的雷霆反击战。我们一定要相信郭先生说的，川普总统肯定会连任。

那么接下来的重点就来到了以共灭共。这一点对国内战友尤其重要。这个问题可大可小的程度可以决定国内的过渡形势是和平还是战争。谈到这个问题前，我们可以再把郭先生前几天关于国内形势的那几期节目，特别是让很多人看完心情沉重的那一期视频复习一下。

看完之后再联系到深层沼泽即将脱离跟CCP的合作，所以国内的形势其实已经在好转了。因为我们都知道，我们心情沉重的原因正是基于对CCP疯狂的了解和预期，这群亡命徒在最后时刻如果有更多的外部力量加持，势必将更加疯狂。“中国人那么多，死几个亿完全可以接受”……这样的言论将很有可能成为现实。遥想新中国成立之后为了便于统治，几十场运动、自然灾害、文革……害死了多少人？而如今到了这个恶魔生死时刻，更加不可想象。

所以在这个节骨眼上，作为CCP重要合作力量的深层沼泽的选择，应该可以到让CCP傻眼的程度。而明面上，所谓的CCP盟友俄罗斯、朝鲜、伊朗、委内瑞拉、巴基斯坦……，哪一个都不敢站出来力挺CCP。所以明暗两面的协助力量都被遏制住之后，CCP在面对接下来的正义力量雷霆反击战的时候，只能孤身应对。

而他们怎么应对的呢？据称他们已经躲到山西的老山上去了，他们的子女、财富已经转移到国外。很容易猜测，他们此时在老山里正在商量如何绑架国内的十四亿人民作最后的反扑。让自己手无寸铁的人民乌泱泱地冲向加特林不是不可能，哪怕几十万人能堵住一个加特林的枪口。这是非常疯狂的，但如果你对CCP几十年的历史有深刻了解，你会知道这是CCP的本性使然。更有病毒和核泄漏的威胁等。

我们当然不希望发生战争，尤其是国内的战友以及亲戚朋友在国内的战友。我们接下来的关注点势必在于中共国内以共灭共的走势。郭先生在直播里早前分析过这个形势，除了让战友们学会自保跟农场保持联系之外，更重要的是积极推动国内袁世凯的出现。9000万党员出来一些反戈的力量，在内部形成灭工力量，这个即使对于国内战友们来说也是最安全的。因为同一个党派内部厮杀，是很难动员党员和人民的。而郭先生一直让大家在安全前提下传播爆料革命，也是为了在这一天到来的时候，更多的人能保全自己。因为这场正邪之战的力量对比到今天已经越来越悬殊。

而关于接下来以共灭共的形势，我们必须紧跟郭先生的消息。个人认为以现在美国的科技和军事实力，热战躲过去并不难，因为到时候危险的地方很容易知道。但国内的战友和亲朋好友将面临的真正考验是疫情和这场灭共战带来的次生灾难。

这场灾难是几十年以来滋生出中共这个恶魔的中华大地必然要经历的。这一点相信特别是在国内的真正战友理解更深，当因为坚持信仰或者在传播爆料革命正道主义的时候面对几乎所有人的指责、嘲讽、被孤立……的时候，你会认为CCP被灭之后那些声音就会自动消失吗？

而作为爆料革命坚定的支持者，我相信郭先生说的新中国联邦和农场会给战友们以支持和保护。战友们一定要保重！
