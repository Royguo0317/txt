## 【快讯】加强政策监管，长租公寓前途渺茫
` 喜联盟编辑部 Himalaya Alliance Editorial Team` [轉載自GNews](https://gnews.org/zh-hans/882392/)

**喜联盟内新闻、素材**｜鹰(文言)
**校对**｜老蚂蚁666 **发布**｜wenwu

新浪网2月2日转载北京日报消息，北京发布《关于规范本市住房租赁企业经营活动的通知》，对长租公寓经营模式和资金收取等做了明确限定，并于3月1日正式实施。

1、《通知》要求租金贷仅能拨付给个人。住房租赁企业向租客预收的租金原则上不得超过3个月，且收、付租金周期必须一致；收取的押金数额不得超1个月，且需专用账户监管。

鉴于18年以来过百家长租公寓企业相继爆雷，几个月前的蛋壳公寓资金链断裂更是将“租金贷”这一强加租客的“以贷养债”的“违规”操作公之于众，于是就有了上述的“租金贷”不得拨付住房租赁企业。

虽然规定比较明确，但专用账户的押金所受监管机构变为北京房地产中介行业协会，而非政府机构，其次住房租赁企业可以通过“先划拨个人，再由个人上缴”的方式将 “租金贷”资金纳入囊中，这种上有政策下有对策的“灰色监管”仍显得毫无意义。

2、《通知》要求住房租赁企业所用合同需遵照本市统一示范文本，并将合同信息在北京市住房租赁监管平台备案；企业需在房屋产权人书面同意后再进行装修，解除合同时应当依照合同规定履行各自责任，不得强收装修费。此举虽有助于住房企业和房屋产权人之间的责任划分，但仍衍生出一系列的问题。

例如装修质量和费用的规范如何界定？虽然统一文本可以一定程度上规避“霸王条款”，但装修成本、质量等一系列“标准”仍缺乏统一界定，住房租赁企业仍可利用“以次充好”手段“牟利”，相应的又需要地方政府对此出台规定或成立监察部门，无形之中增加了政府的“掌控”权力，更难避免由此衍生的“官商勾结”现象。

3、《通知》称北京将继续鼓励支持专业化、规模化住房租赁企业，并给予一定的税收、金融等优惠；针对违法违规企业采取风险警示或停业整顿。鉴于大力发展长租公寓的基调不变，同时政府对该类企业的监管力度加大（程序、资金往来、资格认定等），不可避免的会滋生企业以利益换取商业优待问题。与以往不同的是，“整改”后的监管机构和监管领导的变化，但实质仍是中共惯用的以权力压榨企业进而掠夺人民财富的伎俩。

长租公寓的爆雷是由于“长收短付”这类“以新钱还旧债”的庞氏骗局的崩盘。而中共对长租公寓市场的整治依旧停留在加大政府职能上，并没有从根本上切断企业养债的陋习。

同时所谓“加大监管、处罚力度”也只是将庞氏骗局的获利方从企业转向到政府机构。原有的企业一家公开“行骗”和“孝敬”领导，现在变成更直接的“孝敬”政府机构。这一变相的“公私合营”收割社会大众财富的阳谋，更验证了中共“一切向钱看”，极端推崇个人享有社会财富的野心无限放大。
