据《国家脉搏》1月22日报道，美国社会对媒体的信任度已降至历史最低点，许多新闻专业人士决心为此做出努力以改善这种危险的情况。

这种对媒体的超低信任度是非常危险的。其原因在于，对社会重要机构的信任，尤其是对政府和媒体的信任，是社会凝聚在一起的粘合剂。然而，这种信任关系在十年前就已经开始消融；如今，对于亿万美国人来说，这种对社会公共机构（特别是传统媒体）的信任已不复存在。

根据调查机构爱德曼（Edelman）与新闻网站Axios独家共享的年度信任度调查报告的数据，有史以来第一次，只有不到一半的美国人信任传统媒体。而对社交媒体的信任度则达到27％的历史最低点。

56％的美国人认为，新闻记者故意掩盖事实，报道假消息，或严重夸大其词，蓄意误导公众。

58％的美国人认为，大多数新闻机构在报道中刻意支持特定意识形态或政治立场，而不是以报道事实为准则。

爱德曼在2020年总统大选之后再次对美国社会进行的调查发现，这些数字在进一步恶化中：有57%的民主党人相信媒体；而在共和党中，这个数字只有可怜的18%。

评：
在民主社会，新闻媒体被称为在立法权、司法权、行政权之后的“第四大权力”。它肩负着向社会传递事实真相，和在此基础上凝聚社会大众的作用。它是立法、司法、行政能够得到有效执行的催化剂，是推动社会文明进步的重要力量。

可是随着资本对媒体，特别是对渗透力更强的社交媒体的控制，以及这些资本力量与政客的勾结，新闻媒体距离它们向社会传递真相的初衷越来越远，而为了支持特定意识形态和政治立场无所顾忌地报道各种有利它们自己的假消息。

特别是在之前的四年与2020年总统选举中，绝大多数传统主流媒体和掌握话语权的各大社交媒体，凭借230条款的“庇佑”，大肆传播假消息，或者忽略有利于与其勾结政客的政治对手的好消息。最终，亿万大众对大选结果的质疑，没有得到严肃调查。我想，这就是本文所描述的大多数美国对媒体信任度降到历史最低的原因。
