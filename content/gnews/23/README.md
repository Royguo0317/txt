郭先生在爆料之初就指出了正道主义是人类唯一的未来。何为正道？许多人是说不清楚的，其实我们古人对正道主义早有定义。正道主义就是：以道中天下，可以制礼；以道和天下，可以作乐；以道公天下，可以理财；以道周天下，可以御辱；以道因天下，可以立法；以道观天下，可以制器；圣人不以一己治天下，而以天下治天下。

天下皆为自然，道法自然，无为而治。这就是最高境界的政治治理，在上位的领导人和在下位的黎民百姓都要达到圣人的境界，如何达到这种境界，只有通过法治与德教并举。

法治是为了惩治人们行为上的恶，是底线；而德教则是通过家庭教育、学校教育、社会教育、心灵教育来引导人们断恶从善，改变人心。其中的社会教育主要就是靠媒体来实现。

我们的GTV、GNEWS正是应运而生的新的社交媒体平台，在这个历史的变革中，我们高高竖起正义的旗帜。这个“正”字是由“一”和“止”组成的，上面的“一”代表天道，这个字的意思是止于一，止于天道，就是要合道。所有的行为、思想、语言都符合道，这才叫正道，佛法里面就有八正道，一切都不能偏离道。在正道的基础上，我们再追求民主、自由、荣誉、爱情才有意义，否则都是梦一场。
