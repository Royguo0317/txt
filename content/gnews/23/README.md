美国海军研究所网站5月13日报道，美国、日本、法国和澳大利亚本周开始在印度洋-太平洋地区进行联合训练。

ARC-21联合演习将于5月11日至16日在日本各地举行，其中包括法国、日本、美国和澳大利亚。日本防卫大臣岸信夫在演习前表示，东京寻求深化防务合作，不仅包括美国盟友，也包括“志同道合的国家”。

根据日本海上自卫队的一份新闻稿，日本防卫相岸信夫周二（5月11日）在记者会上表示“希望进一步深化共享自由开放的印度洋-太平洋这一愿景的四国合作关系”。 ARC-21将加强四国之间的防务合作，以实现“自由开放的印度洋-太平洋”。演习的目的是为了提高海军的岛屿防御战术技能。

美国第七舰队发布的一份新闻稿称，在海上阶段，每个国家的舰艇将计划两栖登陆和空袭，同时还将与其他参与演习的舰艇进行编队演习、防空演习和通信演习。

围绕尖阁诸岛（钓鱼岛）和台湾局势，日美加强了对中共制衡姿态。欧洲各国和澳大利亚也提高警惕，加入了同盟行动共同遏制中共。
