## 从民主党人朋友坚决投票给共和党川普总统看美国的伟大
` 喜马拉雅-华盛顿DC农场` [轉載自GNews](https://gnews.org/zh-hans/640480/)

**作者：Faller**




![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/11210936/4-6.jpg)图片来源：secretchina.com


笔者是一个在美国留学生，四年前川普希拉里大选前到来到美国。困惑于学校的教授在川普赢得大选后落寞伤痛好几个月的情绪。四年后，再次经历这个陌生国度里牵动世界眼球的大选，也在最近半年听一个美国当地朋友说她的投票的过程，就想分享一点这个朋友对这次大选前后的一些看法和行动，应该可以代表一部分美国人最真实想法。

首先介绍这位朋友，她住在加州，是一位华裔女性，丈夫是白人，全职太太，一儿一女家庭幸福收入好。作为家庭花销的管理者，她说川普总统的税法会让她的家庭每年多支出几千美元的税。身在深蓝州的她是一位十几年党龄的资深民主党人，加入民主党是为了有投票权，参与社区的一些提案的投票，这些提案会影响到孩子的上学医疗，家庭的保险税收等方方面面。

在大选前的10月初，有一次她和我聊天，说到要投票给川普。我非常纳闷问她：你是民主党员，可以投票给川普总统吗？她说当然可以，投票给谁是她的权利。她觉得民主党现在有很多问题，她甚至想要退党。她的话让我这个从小长在红旗下的人大吃一惊，退党是不是和叛变一样？她解释说她刚加入民主党时，民主党是理性的群体，不是今天的样子。这十几年下来，民主党通过一些外来人口和非法滞留者改变了加州的居住人群的结构，从而改变了选民的结构，当有一些不同或反对的声音出现时，会有一群暴力的人们出现，甚至去打砸抢普通的商铺，这是违法的并且伤害了普通民众的人身和财务的安全。并且她的女儿最近需要看医生，因为超过了13岁，她这做妈妈的竟然无权看女儿的资料，但是医生有权利。说这是最近民主党通过的一项决议。这些让她这个民主党人不能忍受。

她接着告诉我，共和党人面对有不同的或反对声音时，更多会以文明守法的方式进行解释或反对，面对问题他们更多地去找当地政府理论，而不会伤害普通民族。她的邻居里大部分是共和党人，这几年住下来互相尊重彼此党派观点，邻里关心问候一起聚会玩游戏，更没有因为不同的党派仇视彼此，让人敬重。她的邻居听到她支持川普总统都很高兴，还和她开玩笑说她的提案很“共和党”

十几天后，她告诉我，，这次大选，她一定投票川普总统，并且她说服了先生和有投票权的一个孩子都投票给川普，哪怕川普总统的税收政策会让她的家庭多交几千美元的税。几天后她家里的三张选票都投给了川普，亲自送到投票站。并且她说不退出民主党了，因为她要用自己的声音去影响民主党，退了党 就没有这个权利了。

在11月2号大选前一天，这个朋友跟我说了她的担心，如果川普总统当选，很大概率会有动乱。她不希望看到动乱。她最敬佩的是川普总统提名了很多法官，哪怕川普输了，都会有法律保障美国大选。

这位朋友的投票过程给我这个外国人上了一堂生动的美国大选课，让我看到这个国家的伟大在于总统的竞选是按有法律程序进行的；让我看到这个国家的伟大还在于一个普通公民在发现问题时的冷静理智和对法律的信任，用自己的行动，哪怕多付出金钱，也为心中的文明和民主而努力。现在大选正在进行法律诉讼，祝愿这位朋友心愿得以实现！
