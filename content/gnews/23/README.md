## 彭佩奥要求“说谎”的中共国释放因报道病毒而被捕的武汉公民记者

现年37岁的前律师张展（ZhangZhan）基本上是实时在博客上发表广泛流传的博客帖子，内容描述了在冠状病毒于武汉传播初期，中国当局在应对病毒时的混乱和管理不善，她周一被上海市法院以“寻衅滋事”罪**判处四年有期徒刑**。
