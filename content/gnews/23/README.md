6月11日，罗马尼亚总统克劳斯-约翰尼斯（Klaus Lohannis）签署了一项得到华盛顿支持的法案。该法案出于安全担忧，禁止中共和华为参与该国5G电信网络的开发。这项法案源于2019年的一份美国-罗马尼亚备忘录，两国政府在备忘录中表示：“作为基于风险的安全措施的一部分，对5G供应商进行仔细和全面的评估是必要的”，应排除那些由外国政府控制并且缺乏透明所有权结构的供应商。
