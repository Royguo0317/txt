爆料革命军团自2017年开始集结，三年以来已经自发组成了一个强大的喜马拉雅矩阵作战形式。灭共的战役是人类历史上难度系数最高，也最残酷的战争，虽然大家看不到硝烟，但是硝烟和流血牺牲每时每刻都在媒体领域、信息平台、金融行业、政治组织等都多场合多领域发生着，这场没有硝烟的战争比有史以来的热战更激烈，更血腥。

2017年，紧随着文贵先生的爆料，各地战友分别在各个国家和地区自发组成了矩阵的雏形，并由站在第一线的勇士向敌人发起首轮攻击，这种攻击首先以诱敌深入的方式展开。随着战友集结数量的增加，矩阵队形开始由外而内，由内而外双向组成了有核心、有互助、有互相掩护、有声东击西、有似虚似实、有进有退的联合作战方阵。矩阵的中心充溢着一种流动的作战招数，招数的变化防不胜防，如同矩阵组成的汉字中的“回”字，是方，是矩，是坚不可摧的防护，也是亦柔亦刚的能量运行，更是天圆地方的宇宙力量。随着针距的移动，中心即是外围，外围也是中心，战略战术变幻莫测，层层诱敌深入。当敌人攻击的时候，每位战友只需一个小小的动作，即可筑起矩阵坚不可摧的防御模式，让敌人无懈可击。

2020年10月，战役进入了最后的浴血奋战阶段，我们要以爆料革命的力量助推国际正义的力量来排干沼泽。据路德先生10月20日的直播中提到，文贵先生在三四月份的爆料中说到“我们在撒网”，如今看来这张大网早已将这些大鱼围拢了起来，从华尔街，到政坛，再到大众媒体和科技公司，我们即将进入一步步收网的阶段。 而在这个时候，这一只只大鱼还在不断地咬钩。

“没有从天而降的英雄，只有挺身而出的凡人。”  最后13天的决战，比起那些为爆料革命牺牲了亲人、自由甚至是生命的无名英雄们和那些抛头露面抗战在一线的战友，我们能做的非常微不足道。但是每一份努力，哪怕就是动动手指，移动鼠标的力量就能为敌人的灭亡加上一根稻草，就能推动这个战斗的矩阵发出威力。
