9月28日，宜賓市敘州區人民政府官方微博發布“關於宜賓伊力科技有限公司發射藥車間發生爆炸事故的情況通報”，通報稱，2020年9月28日16點30分左右，宜賓伊力科技有限公司發射藥車間發生爆炸事故，“當時傳來巨大的爆炸聲，我們的門窗嚴重受損。”村民們介紹。記者在爆炸現場外面的碧水街、宜屏快速通道看到，民房窗戶幾乎全部被震壞，一家汽車銷售公司三台新車玻璃被震碎，損失巨大。爆炸產生的巨大威力甚至將幾百米外的玻璃震碎。當地村民收集到三段灰色金屬物，手感沉重，已經完全扭曲變形。村民告訴記者，該金屬物就是從爆炸現場飛出來的。
截止17時30分，事故造成1人失聯，1人重傷，1人輕傷，事故發生後，宜賓市政府副市長張琦率市應急局和相關專家，敘州區區長瞿進、常務副區長宋慶伍、副區長袁峰率公安、消防、應急、衛健、生態環境、高場鎮人員趕赴現場處置，搶救傷員，搜尋失聯人員，疏散人群，做好善後工作。
*簡評：以現場村民的描述，爆炸的威力巨大，對爆出的人員傷亡數字表示疑問。 “發射藥”又是發射什麼的？在只有一個聲音的國度，從字面報導上我們無從得知事情真相。*
