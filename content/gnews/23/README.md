**西班牙2021年5月28日电/西喜社——**

每年的6月4日，香港市民支持爱国民主运动联合会（简称“支联会”）都会在维多利亚公园举办大型烛光晚会，悼念“六四事件”中的死难者，三十年来风雨无阻。而每一年参与的人数也会随着当时社会和政府变化情况而“潮起潮落”，出席人数也会反映出时下香港市民对港府政策的不满情绪。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/28181144/unknown-1-7.png)

2019年为“六四”事件纪念三十周年，正逢反送中条例运动爆发前夕，参加烛光晚会的人数为三十年来最多的一次，大概参与人数达到18万人。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/28181338/unknown-2-6.png)

在2020年“六四”事件三十一周年， 警方以新冠肺炎（COVID-19）疫情为由，首次禁止支联会在维多利亚公园举办“六四”烛光晚会。当晚的维多利亚公园，沒有舞台，沒有节目预备，也沒有出席人数统计，只有香港支联会人士及泛民主派人士和大批市民冒险走进维多利亚公园，高举烛光。很快，自发人群差不多填满了六個足球場。事后，警方以“未经批准集结”、“煽惑他人参与未经批准集结”等罪名拘捕了支联会及民主人士24人。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/28181427/unknown-4-2.png)

今年是“六四”32周年，也是中共“全面管治香港”、“香港被全面政治打压”的一年。过去的大抓捕和全面禁制集会自由，显示了香港人不再享有《基本法》和《人权法》保障的自由，而当前的香港法律仅仅是当权者“排除异己”的镇压工具而已。

国安法生效后，今年的维多利亚公园是否还能延续布满烛光的和平悼念活动吗？

今年维多利亚公园的“六四”纪念活动有可能被指违反“限聚令”，还会有民众前往参加吗？而且历届“六四”晚会中最常出现的口号： “释放民运人士、平反八九民运、追究屠城责任、结束一党专政、建设民主中共国”等，自从港版国安法出台后，“结束一党专政”这句的口号便涉嫌触犯有“煽动颠复国家政权”的罪名。今年还会被再次喊出来吗？

香港行政长官林郑月娥回应此事时表示，要“依法办事”， 还表明说“今年是共产党的百年庆典，大家要看到在共产党领导下，这几十年的国家腾飞和为人民带来美好的生活，我们的立场是支持国家的执政党”。

中共方面也表示：将以前所未有的强硬手段处理参加“六四维多利亚公园的烛光纪念晚会”的民众，把所有出席悼念活动的人士，一律定性为不符“爱国者”的标准，日后无法参选区议会和立法会，而现任区议员更会被直接剥夺资格。

香港警方也预计会从严执法，若当晚的悼念活动被指构成非法集结，活动带头者及政界人士很有可能会被逮捕，其他在场参与者也可能同遭拘捕及起诉。
