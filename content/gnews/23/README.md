**1****、南希：墙内战友的安全要放在第一位——**

墙内战友和我们在国外的战友们真的是不一样的，他们是冒着生命的危险在支持我们的爆料革命，在这里我们向加入到我们银河系的墙内战友致以最高的敬意，我们也强烈建议墙内战友们一定要继续保护好自己，安全绝对是你们第一位的。

**2****、请蒙古小哥分享《马背英雄》不一样的感受——**

**蒙古小哥：**《马背英雄》制作出来发表之前，确实我们跟威廉王是有一些沟通的，最后还有其他很多战友的共同付出，这首歌对我的震撼还是非常大的，它首先不只是把一些蒙古族的元素表现得非常好，还融入了非常浓烈的爆料革命的元素还有灭共的元素在里面。

比如说刚开始是很悠扬的一些呼麦、说唱，但是它非常陡峭的就转变到了——是谁灭掉了我们的草原、是谁掐灭了孩子的希望这样的说唱里面，就好像平静的生活突然被邪恶的共产党统治得中断了一样，这首歌也是给我们带来了非常大的勇气和希望。

**3****、文俊战友对于《马背英雄》的分享——**

**文俊：**我第一次听到这首歌的时候是在七哥发出来的，我听到后我就感觉这首歌特别的有力量，虽然我对音乐也不懂，看到七哥的MV之后就感觉很帅、很酷，适合我们年轻人，这就是我对这首歌的感受。

**4****、南希：蒙古小哥当初有没有预想过马背英雄会呈现出这样惊艳的一个结果？**

**蒙古小哥：**当时我知道威廉王和唐平姐是非常用心地在制作这首歌曲，因为当时我们在群里面也有一些互动嘛，他们也会发一些歌词，然后让我们其他战友去把它翻译出来，然后让威廉王他们去感受一下读音，让他们有个灵感去创作，当时也没有想到做出来的效果是这样的，反正是远远超出了能想象到的范围，总之震撼是很大的。

**5****、南希：《马背英雄》这首歌其实编曲非常复杂，把蒙古族的这个三大音乐遗产：呼麦、长调和马头琴都融入到里面，请蒙古小哥给我们简单的介绍一下蒙古的三大音乐遗产——**

**蒙古小哥：**刚听这首歌的时候，我只是从整体上感觉这首歌唱得非常好、非常感动，然后仔细我再听才发现里面有非常多的一些蒙古族的一些音乐的元素，虽然我也不是很懂蒙古族的音乐，我也不是很懂音乐，但是我说说自己的感受吧，它有一些呼麦、马头琴、长调之类的，然后结合着歌曲的内容，比如说，一些说唱还有一些组歌，疾驰在马背上。

**6****、南希：在《马背英雄》的前期的筹备工作中有哪些事比较难忘的吗？**

**蒙古小哥：**刚开始威廉王的想法是想多一些蒙古语的歌词，其他战友在群里也会帮威廉王翻译，威廉王有时候他那个读音我们给他纠正，读来读去，反正读很多遍，他才能把自己的读音纠正过来，这个印象还是挺深的。

但是威廉王最后歌曲出来的时候，据他自己说只要了最有力的一句话就是欧哈姆呐姆Gi欧哈斯噶呀——消灭共产党，然后整个歌曲也非常符合七哥唱出来的那个气场、那个气势，总之是非常适合七哥的一首歌。

7、**东游记：因为七哥这首歌里头写得疾驰在马背上，我就很好奇，因为我们是内陆地区的人，从来也没有见过马、也没有见过这些真正的这种生灵，我就想问真的是蒙古人现在还可以骑马吗？**

**蒙古小哥：**当然可以骑呀，骑马是有现实的生产意义的，比如说在一些草原上或者是那些农村里的人，他们骑马是有现实意义的、是有用处的，不是说骑马就是出去潇洒一下的。然后在内蒙古也有一些纳达木的比赛，那些就是为了比赛的赛马类似的比赛，如果你想骑马的话，欢迎来内蒙古来转转。

**8****、东游记：其实这首歌曲创作的话，我相信一定有那种有感而发的感觉，然后真的是觉得共产党做的这些事情是集世界上的集恶之大成，那您是什么时候感觉出来共产党对于蒙古有那种真的是有作恶的方面产生的，你能说一下吗？**

**蒙古小哥：**共产党对蒙古作恶的方面，其实从我们学生时代能接触到爆料革命的战友或多或少都会有一些感觉不对劲的那种地方，但是真正感受到的是当时我在中学的时候发生过那种牧民的牧场被他们霸占这种事情，就像歌词里面唱的一模一样的情况，他们就是用一些手段抢走了牧民的牧场的使用权，然后在牧场上开矿什么的，然后用就是各种非法的手段和各种非法的途径霸占牧场。

后来牧民也有维权的情况，也发生过一些冲突，然后牧民也有失去生命的这种情况发生，当时在我们当地轰动还是挺大的，这个可能就是比较非常明显的一些对共产党的一些恶的一些感受吧。

**9****、南希：请蒙古小哥分享一下挺郭的经历，郭先生说的哪一点是最打动您让您加入到爆料革命当中来？**

**蒙古小哥：**郭先生的每一句话、每一场直播、每一个视频、都能触动我的想法，都能给我激发出很大的力量，比如说在墙内之前可能感受到共产党或多或少的不好，然后在了解爆料革命之前比如说消灭共产党、共产党会从这个世界上消失这个事情是想都不敢想的一件事情。

但是认识了七哥之后，七哥直播里的每一句话都非常有力量。就是七哥说的那句——共产党你完了！就是听到这一句从那一刻开始就已经不是不敢想共产党会消灭、会消失了，就是坚信共产党一定会从这个世界上消失了。

**10****、南希：请躺哥分享在什么情况下了解到爆料革命追随郭先生的**

**躺哥：**我是在去年的5月份马上临近六四的时候，在一个偶然的机会在YouTube上看到路德的一个节目，当时的时候他频道下面还有一个VOG连接，就是战友之家，我当时从那加进去以后，从这个地方跟战友们在一起聊天，然后我接触爆料革命、关注郭先生。

从郭先生在直播说了一些事情以后对我的触动，主要是我从年轻的时候对这个政府、对共产党就没有好感，他们的一些事情还有社会当中这些事情我是非常反感的，但是自己也想不通，也找不到病根在哪，只是心里面就是认为共产党作恶多端，他们所有做的事情包括这个什么银行、金融，还有包括对底层的小贩，都是在疯狂的收割。

通过爆料革命关注郭先生以后，就开始翻看郭先生的视频，从他所有的视频里面，我看到了自己想得到的答案——CCP作恶，国家逐步走向衰落，还有民生以及我们家乡的环境。只是他们在宣传说空气不好、天气、水资源什么的，我小的时候呢，我们家乡附近那个河里面还有水，一直到现在基本上北方的河流是河就干。还有工业或者是城市的污水排放的臭水沟，对这些事情自己也很心痛，最后从郭先生爆料中间找到了病根儿，原来如此。

就前面这几个战友说的郭先生的这首歌的创作过程、创造经历，感觉这就是肯定是非常的艰难也非常的有乐趣，大家在共同努力做一件事情，但是我看到的不仅是唱得好或者是歌词写得好，大家都用心了。

最主要的是我是看到了郭先生在爆料革命期间对全世界尤其是墙内的自己的同胞，包括新疆、西藏、香港还有蒙古，在蒙古族同胞遭受了一个教育就被洗脑然后强制取消母语教育的事件的时候，好多人都是从网上看一些视频，但是敢于帮助蒙古族发声的、系统性地传播这件事情的，只有我们爆料革命。

从这里我看到了郭先生他这种大爱，不是说光拯救墙内的战友或者说光拯救汉族，包括所有的少数民族——在CCP所说的少数民族，强化分类的。我们就是属于政治性地被分类了，有的我们可能不是，这个自己可以去查。在这个方面我认为这首歌是替蒙古族同胞兄弟们发声，郭先生心里面惦记着这些同胞，这是一个我认为是体现了郭先生的真正的是一个也不落下，所有的人想办法唤醒，把他们的苦难想办法传播，我看到的是这样的含义。

**11****、南希：请文俊分享一下挺郭经历**

**文俊：**我在2018年的7月份因为工作的原因了解到文贵先生，当时是在YouTube上，说实话我当时了解到文贵先生的还是都是娱乐性质地看待文贵先生，因为当时文贵先生还没有明确提出来灭共，是在明确的爆料，什么爆肛芳、什么这冰冰那冰冰、什么1808，我也是被这个吸引了我也是很好奇，七哥也老是说要放冰冰的视频，一直也没放，等到现在也放，所以我就被他这个娱乐一直吸引到、一直走到现在，我相信很多战友也是跟我一样。

之后18年文贵看春晚，喊了10句”灭掉共产党”的时候，我真的我流泪了，因为在18年之前，我从来都是不敢想象的，连想也不敢想，我脑海里就没有这个词，当文贵先生喊出来的时候，我说实话我就是心中有一团怒火，因为你都知道七哥是给人一种很有力量的感觉，我就被他那种力量带动了，从那以后我就想去跟坚定地跟七哥一起走下去吧灭掉共产党，虽然我当时我也不知道他能不能灭掉共产党，说实话的。

但是我就是被七哥的人格魅力给吸引了，我就愿意跟他，说实话18年的时候我想不到今天会是这样的，我真的想不到，18年的时候我也不知道未来会是怎么样，我就愿意跟他一起、关注他、一起走下去，所以就这样，从18年文贵看春晚的时候之后就到19年的6月份，香港反送中的时候，那时候我也在香港。

那时候整个香港反送中的经历之后我是彻底醒了，其实香港反送中运动几个月前就开始酝酿了。到6月份这种爆发，它首先是刚开始是80万的大游行，之后是90~100万的大游行，之后到最大规模的200万的大游行，之后我看到了香港媒体、香港政府那种不要脸的那个声明，他说这200万人都是支持我们，但是我当时在香港我确实感觉到这200万人都是不支持的，都是No的。

所以之后我看到香港的电视台里放的都是这些学生都是暴徒、都是危险分子，我看到他什么什么丢砖头、丢那个蓝气瓶等等，之后我用手机打开我的YouTube，因为YouTube很多人游行的现场直播，我在YouTube上看到了很多警察暴打那些学生，跟我在电视上看的完全是不一样的，之后我就带着疑问跑去现场看过一次，因为我没有敢很靠近，我看到的是警察确实是有在追赶学生，所以我是比较倾向所有YouTube网上的那些视频，之后就是香港那些孩子被自杀，太多了，陈彦霖、周梓乐以及网传的陈彦霖的妈妈也是跳楼自杀，跳楼的人掉到地上是没有血的，地面上一滴血都是没有的，尸体也是好像被水泡过的，用当时那个网络媒体上说的这些人就是被放干血的。

在看到这些之后，从2018年的7月份带着娱乐性质、抱着好奇心的心态看到文贵先生之后到18年文贵看春晚喊出了十句消灭共产党之后，我那时候就决定要跟七哥一起走下去，之后到那个反送中之后我是彻底地醒了，醒了之后一直走到现在，这就是我的整个经历。

**12****、请东游记战友分享一下感受**

**东游记：**挺郭经历每个人都不一样，说到香港事件我也有香港的朋友，他就跟我说香港的大学，我估计是港大，他说校长也换了，然后总共1700多个学生毕业有1300个学生是拿不到毕业证书的，很多都是他的朋友的孩子都是没有毕业证书就毕业了，就是很气愤嘛，其实我们看到的都是年轻人，其实很多的香港的有钱人、有产者其实都是对这个事情都是有自己的看法的，只不过是因为自己的利益或者是害怕，只能是敢怒不敢言吧，但是其实你说香港的这些老百姓有钱人他们不知道中共干的这个事情是对法治的一种亵渎、对老百姓的一种损害吗？他们其实心里都知道的。

从现在香港出去的人数上面来说，还是从香港还有大陆出逃的钱，尤其是广东那边出的钱来说的话，都是非常大量的，可能我们战友没有感觉到，但是我们是知道一些消息就是这些出逃的钱的量是非常非常巨大的，其实每个人都在用脚投票，尤其是有钱人是用脚投票，我们战友的话是要用声音来投票，我们一定要跟世界上所有人说，共产党不代表中国、中国人也和共产党不一样。

**13****、南希：请蒙古小哥分享有没有尝试向您身边的人来传递爆料革命、传递真相**

**蒙古小哥：**我自己是18年接触的爆料革命，然后一路这样走过来的，后面也会给我身边的人去分享一下，比如跟我的父亲或者大学的同学、过去的一些朋友，就是给他们讲讲文贵先生的视频还有路德社的一些东西，虽然说让身边的朋友突然去接受或者是了解这样一些事情，尤其是在墙内翻墙也不是很方便的情况下，确实不是那么容易的。

但是说他们在接触我给他们的这些信息之后，他们还是情绪上可能会有一些变化的，也许不会直接地表现出来，身边的人在我分享给他们一些事情之后，说实话还是多少会有一些隔阂和矛盾存在的，但是有些事情就是慢慢地去努力、去尝试嘛，在以后我也会继续地去做一些尝试。

**14****、南希：请躺哥分享向身边的人传递爆料革命真相的经历和分享的效果**

**躺哥：**传播真相的事情每天都在做，我认为这是自己应尽的责任，因为我经常说咱们爆料革命的战友，我们这一辈人都是肩负消灭CCP的责任，我们都是被上天所赋予了这个使命。

在墙内传播确实是非常困难，在咱们所说的这些事情没有发生之前，每一个人或多或少对自己的家人还有自己的朋友、有亲情关系的这些人，基本上推进爆料革命现在非常困难，主要是CCP洗脑还有这个防火墙的原因，包括一些常年可以说上千年以来中国人思想上受那种禁锢的教育、没有自己的思想、没有自己的思考能力，这是墙内普遍存在的现象。

我们这些草根的战友们的话语权基本上是没有的情况下，在没有事实可以佐证的情况下，每天经受的嘲笑还有说是传销或者说被什么什么势力洗脑了，这都是天天我们所面临的。墙内的传播没有大家想象得那么容易，墙外的战友们可以做一些小卡片、宣传册或者说去上街做一些宣传活动，或者说是露脸直播都行。这些在墙外战友看来非常简单、顺手就能做成的事，但是在墙内战友身上现在是绝对不可能发生的。
