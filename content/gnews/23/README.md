## 訪韓之旅！楊娘娘出現的地方，就肯定沒有好事！
`2020-08-19 22:05 GM30` [轉載自GNews](https://gnews.org/zh-hant/303462/)

翻譯：喜馬拉雅韓國農場summerlee
校對：李小甜KR

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/19220254/yang_jiechi_afp-1.jpg)

中共外交部證實，負責中共外交的政治局委員楊潔篪將從本月21日，開始兩天一夜的訪韓國之旅。 據中共外交部19日報道，外交部發言人趙立堅當天在例行通報會上說：“楊潔篪將應韓國國家安保室長徐勳的邀請，于21日至22日訪問韓國釜山，並與韓方進行討論。” 趙立堅說，在訪問韓國之前，楊潔篪將在19日和20日訪問新加坡。 楊潔篪于2018年3月曾作爲特別代表訪問了韓國，在與文在寅總統的會談中討論了中朝首腦會議的結果以及朝鮮半島無核化的解決方案。 韓國青瓦台發言人姜敏時在19日上午的簡報中說：“應韓國國家安保室長徐勳的邀請，楊潔篪先生將從本月21日至22日訪問釜山。” 徐勳市長將在22日上午與楊潔篪進行會談，並在午餐中就應對新型冠狀病毒19的合作，高層交流等雙邊關系以及朝鮮半島和國際局勢等雙方關心的問題交換意見。” 早些時候，楊潔篪曾以特使的身份于2018年3月訪問韓國，探訪文在寅總統，並分享了中朝峰會的結果。 此後，據報道，楊潔篪還曾在同年7月對韓國進行非公開訪問，與當時的韓國國家安保市長鄭益龍討論了取消對因韓國部署高空導彈防禦系統（THAAD，THAAD）而實施的經濟報複，並對韓國，北韓，美國，中共國四國的終戰宣言進行了討論。

評論：郭先生說過“楊娘娘出現的地方，就肯定沒有好事！”正值韓國疫情第二波爆發之時，楊娘娘此次勾兌訪問，讓韓國民衆對韓國政府的外交做法感到非常憤怒。8月21號（本周五）韓國農場戰友們在首爾“反對楊潔篪訪韓”的示威活動。
