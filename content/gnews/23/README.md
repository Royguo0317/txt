**2.印度从欧佩克石油进口的份额骤降至二十年来的最低点**

[从工业和贸易来源获得的数据显示，在截至3月底的一年中，石油输出国组织欧佩克在印度石油进口中的份额降至至少20年来的最低点，因为这个亚洲第三大经济体的总体采购量降至六年来的最低点。印度以牺牲来自非洲和中东的石油为代价，购买了更多美国和加拿大的石油，将从石油输出国组织成员的购买量减少到每天286万桶左右，将该组织的进口份额从之前的80%左右挤压到72%。](https://www.reuters.com/world/india/opecs-share-indian-oil-imports-plunges-2-decade-low-trade-2021-05-03/)

**6.4月中共国制造业PMI为51.1% 低于上月0.8个百分点**

[据中共国统计局服务业调查中心, 4月份，中共国制造业采购经理指数(PMI)为51.1%，低于上月0.8个百分点，仍高于临界点，制造业继续保持扩张态势。从企业规模看，大、中型企业PMI分别为51.7%和50.3%，低于上月1.0和1.3个百分点，小型企业PMI为50.8%，高于上月0.4个百分点。从分类指数看，在构成制造业PMI的5个分类指数中，生产指数和新订单指数均高于临界点，原材料库存指数、从业人员指数和供应商配送时间指数均低于临界点。4月份，非制造业商务活动指数为54.9%，较上月回落1.4个百分点，非制造业扩张步伐有所放缓。](http://economy.caijing.com.cn/20210430/4761673.shtml)

[评论：国际市场上芯片短缺、国际物流不畅、集装箱紧缺、运价上涨等问题依然严峻。高技术制造业供应商配送时间指数连续三个月低于44.0%，企业原材料采购周期持续延长，正常生产活动受到较大影响，所有这些都会使PMI指数往下行。](http://economy.caijing.com.cn/20210430/4761673.shtml)
