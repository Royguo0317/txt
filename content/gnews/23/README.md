04月18日，英国政府网站称，在俄罗斯外交官被驱逐出境的情况下，英国支持捷克当局。

“英国完全支持我们的捷克盟友，他们已经表明了格鲁吉亚革命卫队在企图进行危险和恶意行动方面走得多远，并注意到索尔兹伯里袭击事件后令人不安的行为方式。”声明说。

4月17日，捷克外长扬·加马切克宣布驱逐18名俄罗斯外交官。他指出，他所提到的人是格鲁吉亚（现为俄罗斯武装部队总参谋部总司局）和外国情报局（SVR）的成员。

总理安德烈·巴比什后来解释说，捷克当局怀疑这些外交官参与了2014年摩拉维亚（Vrbetica）村弹药库的爆炸事件。

捷克当局后来将被指控的国防部总参谋部雇员亚历山大·彼得罗夫（Alexander Petrov）和鲁斯兰·波希罗夫（Ruslan Boshirov）列入通缉名单。英国当局认为他们犯了2018年3月在索尔兹伯里毒杀前俄罗斯情报官员谢尔盖·斯克里帕尔及其女儿尤利娅的罪行。

4月19日，捷克当局将在欧盟外长会议期间讨论弹药库爆炸的情况。俄罗斯外交部发言人玛丽亚·扎哈罗娃说：“布拉格很清楚这种伎俩之后会有什么后果”。
