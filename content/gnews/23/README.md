## 中共公安部部长入疆强调治疆精神 强硬对抗涉疆“种族灭绝”制裁
` 喜联盟编辑部 Himalaya Alliance Editorial Team` [轉載自GNews](https://gnews.org/zh-hans/1020810/)

**作者：墨尔本雅典娜农场 人民公敌**

![]()![](https://gnews.org/wp-content/uploads/2021/03/Picture1-38.png)

(图片来自网络)

人民日报3月24日发布消息称[1]，中共国务委员、公安部部长赵克志于3月19日至24日在新疆调研时，重申习近平在第三次中央新疆工作座谈会上的讲话精神，强调全疆要“完整准确贯彻新时代党的治疆方略”，形成机制长期深入开展“去极端化”工作，“坚决挫败‘以疆制华’‘以恐遏华’的图谋”。

中共第三次中央新疆工作座谈会于2020年9月25日至26日在北京召开[2]。习近平在会上表示，新疆取得社会稳定、长治久安的发展局面，归功于中国共产党的领导，并强调要完整准确贯彻中共的治疆方略，坚持“依法治疆、团结稳疆、文化润疆、富民兴疆、长期建疆”。

然而，在这些冠冕堂皇的辞藻背后，却是新疆上百万异见人士被集中关押。被关押者受到非人虐待和折磨，他们在饥饿的驱使下被迫背诵习近平语录，部分妇女甚至遭受轮暴……[3]。但中共却堂而皇之地将这样的“集中营”称为“职业技能教培机构”，对异见人士的洗脑行为称之为“去极端化”。这就是中共声称的“依法治疆”，而实质却是：顺我者昌逆我者亡。

此外，中共通过同化政策和同化教育，对新疆文化进行侵蚀。如利诱维汉通婚[4]，利用内地新疆班教育同化维族精英，大量拆除新疆境内的清真寺等[5]。所谓的“文化润疆”不过是中共摧毁穆斯林文化的一个完美谎言。

事实上，媒体曝光出来的内容只是中共在新疆地区所犯罪行之冰山一角，美国政府此前根据情报信息对中共作出涉疆“种族灭绝”和“反人类罪”的认定名副其实。

据俄罗斯卫星通讯社3月22日消息[6]，本周一（3月22日），美国财政部外国资金管制办公室宣布制裁两名中共官员——王君正和陈明国。这是继去年美国财政部对包括中共政治局委员、新疆自治区党委书记陈全国在内的涉疆官员制裁之后的再次制裁。就在同一天，欧盟也宣布制裁四名中共官员和新疆建设兵团。

对于美国和欧盟的这一制裁行动，间谍遍布全球的中共不可能不提前得知情报。在制裁宣布之前，中共派出公安部部长去新疆调研，这难道是一个巧合吗？在“调研”期间，赵克志重申、强调习近平及其党派以“反恐”和“去极端化”之名镇压、同化穆斯林的治疆方略。这明显是在对内恐吓、强稳人心，对外则强硬对抗国际社会制裁。

如此看来，中共在“种族灭绝”和“反人类罪”这条道上要一路走到黑了。
