校对：待命（文晓）（京都富士会）

排版：山頂之風（京都富士会）

如果说之前乔拜登秘密找共和党大佬麦康奈尔“谈条件”，请求川普连任后能够赦免他和他的家族，是一次主动“跳船”的行为，那么这老头的“跳船”显然没有成功。因为在他想“跳船”的那一瞬间，中共高层不早不晚发去了“恭喜拜登当选”的贺电，这消息无异于一把从背后顶上去的枪，警告他别想跑路。

而最近，包括纽约时报、CNN在内的左媒，就像约好了似的，铺天盖地统一报道美国联邦部门因税务问题调查拜登儿子、亨特·拜登的消息。12月17日，社交媒体上还出现了“拜登感染中共病毒”的传闻。的确在最近一次公开露面中，有人记录拜登“70多秒咳嗽13次”，尽管拜登称自己“有点感冒”，但福奇还是公开建议：“快去打疫苗”。网络上有人发起投票，认为拜登“感染了”占55％，认为“没感染”占45％。

在最新的一次采访中，拜登称对亨特的税务调查是“不正当行为”，他对此并不担心。这是自媒体报道浮出水面以来，这位当选总统首次就针对他儿子的商业交易的指控做出的回应。不过这个回应同样非常虚弱，因为除了他自己的回应之外，左媒和拜登同盟们就这件事站出来为他撑腰的几乎没有。

拜登的所谓“胜选演讲”，更像自说自话的单口相声。演讲视频被福克斯转播7小时后，这位获取85000000选票的白先生获得观众是：14337人/次；点赞205；点踩1691。而四年之前，川普的胜选演讲，观看次数达到了6,890,230次。网友评论：“福克斯等于告诉大家，85000000选票是个笑话！实际上，操作本次大选舞弊的黑暗势力，如果要继续为拜登撑场面，实际上并不难，只是到了这个败势的尾声，同盟们已经没有必要再去撑他，甚至已经有放弃他的种种迹象。

所以从目前的局势来看，拜登越来越像一盏快要熄灭的“败灯”，作为被推倒台前的一个棋子，同僚们等不到他自己熄灭。我们观察到，棋盘上的各方势力想把他“弄下去”的爪子已经悄悄伸出。各种对他不利的消息层出不穷，他也正渐渐走入一定会溺身的深渊中心——因为只要调查亨特拜登，只要一切证据公之于众，“大老板”拜登是这条绳上浮现的最大的蚂蚱，完全不可能隐身。谁会让一个贪污犯、卖国者、叛国者当总统，这可能吗？

12月17日晚，拜登发的最新一条推说：“让我明确一点，美国一直为世界和平过度权树立榜样，我们将再次这样做”。这个呼吁，和他最近絮絮叨叨关于“气候”“戴口罩”的倡议一样虚弱无力。他当然希望“和平过渡”，但没有正义哪来和平？这一段时间，这个78岁的老人，谁也不知道他是否曾真的扭伤了脚，而现在康复了没有，谁也不知道他是否被下了中共病毒，不停的咳嗽和这个病毒是否有关，要知道下毒可是CCP的绝活。要知道和他家族进行利益勾兑的CCP的代理人叶简明，目前生死不明，拜登步叶简明的后尘，也并不会令人惊讶。

距“败灯“从”高空坠落”还有多久？是在他完成铺路，带着魔鬼式笑声的贺锦丽浮上来的时候吗？还是在他败选之后、面临的司法大审判？

美国的总统不是他说了算，也不是CCP 说了算，美国总统，最后一定是由美国人民说了算的，是由宪法赋予总统保卫美国的权力和职责说了算的。越战越勇的川普总统以及美国正义力量正在勇猛攀登，拜登在他带着咳嗽的“胜选演讲”时，实际上已经是来到最危险的高空，接下来将表演给我们的就是高空落体。
