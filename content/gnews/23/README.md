10月22日，《華盛頓郵報》報導，美國川普總統最近與他的顧問們就辭退聯邦調查局（FBI）局長克里斯托弗·雷（Christopher Wray）進行了反复討論。

據知情人士表示，總統對雷感到失望，因為聯邦調查局沒有在總統選舉中對川普總統的競爭對手民主黨總統候選人喬·拜登及其兒子亨特·拜登的罪行展開調查。

川普總統指責喬·拜登利用他的職位來幫助其兒子亨特·拜登從事商業活動，亨特∙拜登曾於2014年至2019年擔任烏克蘭天然氣公司布利斯瑪股份有限公司（Burisma Holdings）董事會成員，當時的副總統喬·拜登正是美烏事務的主管。此外，這位前副總統還被指控向前烏克蘭總統維克多·F·亞努科維奇(Viktor F. Yanukovych)施加壓力，要求解僱當時正在調查布利斯瑪的前烏克蘭總檢察長維克托·肖金（Viktor Shokin）。喬∙拜登本人否認了所有指控。

熟悉白宮問題的消息人士稱，川普希望聯邦調查局局長在《紐約郵報》爆出拜登家族醜聞的背景下發表正式聲明，指證喬·拜登之子亨特·拜登與布利斯瑪高層瓦迪姆·波扎爾斯基（Vadim Pozharsky）的往來電子郵件。

本週早些時候，美國國家情報總監約翰·拉特克利夫（John Ratcliffe）表示，亨特·拜登往來信件的洩漏與外界所說的俄羅斯虛假宣傳沒有任何關係。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/22075202/200429-john-ratcliffe-ap-773.jpg)圖片來源：美聯社
有消息人士透露，川普總統認為對雷的任命是他最糟糕的人員安排決定之一。他們指出，雷的行為受到白宮辦公廳主任馬克·梅多斯（Mark Meadows）和白宮社交媒體總監丹·斯卡維諾（Dan Scavino）的私下譴責。

川普總統表示，如果他在即將到來的11月3日的總統選舉中獲勝，雷將被免職。同時，川普的這個意見可能會根據投票結果，以及共和黨是否仍是美國參議院的多數黨而改變。

10月17日，美國川普總統在喬治亞州的競選支持者集會上稱，我們必須把拜登家族關進監獄。把喬∙拜登關進監獄。把希拉里關進監獄。拜登家族組織了一個“犯罪企業”，就像吸塵器一樣，吸入了數百萬美元。

在10月19日，美國川普總統在亞利桑那州圖森向選民發表講話時稱，美國聯邦調查局（FBI）應該調查美國總統候選人喬·拜登的案件。他說，他不知道聯邦調查局目前是否在進行調查，還建議記者就喬·拜登事件是否已進行調查聯繫FBI局長克里斯托弗·雷（Christopher Wray），讓其解釋這件事。

2017年，川普解雇了美國聯邦調查局（FBI）前局長詹姆斯·科米（James Comey）。在對前總統候選人希拉里·克林頓（Hillary Clinton）“郵件門”進行調查的過程中，總統對他採取的決定不滿意，希拉里的行為可能導致國家機密信息的洩露。聯邦調查局在2016年總統大選之前就開始就希拉里的“郵件門”事件進行了調查，但沒有發現提起訴訟的理由。但是，希拉里本人表示，正是聯邦調查局局長的行為導致她在選舉中失敗。
