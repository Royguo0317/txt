## 《花木蘭》遭抵制潮大陸首映日票房慘淡
`2020-09-13 07:59 GM36` [轉載自GNews](https://gnews.org/zh-hant/352245/)

編撰：JAY、zhong、文非、文肯尼； 審稿：文橙
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/13075549/913_8.jpg)
**簡訊：**

迪士尼耗費2億美元巨資拍攝的影片《花木蘭》引起的風波不斷，被全世界所抵制。 **中國大陸週五（** **9** **月** **11** **日）上映，但首日票房慘淡，只有** **5381** **萬元人民幣。**

《花木蘭》爭議不斷是因為其“過度改編、更換了劇本只為了使劉亦菲主演花木蘭”。 **參與演出花木蘭的演員** **也是** **爭議不斷：劉亦菲在香港反送中抗爭運動期間，發文撐港警鎮壓港人，致港人發起** **“** **抵制木蘭** **”** **（** **#boycottMulan** **）行動。此外，香港演員甄子丹，也在中共強行實施《港版國安法》之際，聲稱：慶祝香港回歸祖國。**

《花木蘭》正式上映之後，再次成為輿論焦點。 **片中部分內容是在大陸新疆拍攝，而且在片尾鳴謝名單中，出現了** **8** **家新疆政府機構** ，包括“吐魯番公安局”、“新疆自治區黨委宣傳部”等，實質上為迫害維吾爾人的中共機構，涉及非法抓捕、監禁、強制洗腦等侵犯人權的罪行。

據了解，2018年8月13日迪士尼宣布《花木蘭》開拍，那時候美國政府和非政府組織已經發布了大量有關維吾爾人等被關進新疆“集中營”迫害的報告。 **但迪士尼還是在新疆取景、並受到中共機構資助。美國多位政要對此公開譴責：中共迫害人權，迪士尼同流合污，有辱美國價值。**

**評論** **：**

美國政府早在2018年初就發出新疆人民受到共產黨非法拘禁、政治迫害、種族迫害等嚴重人權爭議，且新疆人民的遭遇很早就為世界所熟知。但中共仍然利用從人民手中非法獲得的資金，挹注於電影娛樂產業作為大外宣的政治統戰手法。

**身為美國知名、老牌，且在世界範圍具有深遠影響力的動畫、電影公司迪士尼，卻與中共同流合污，美國政要對迪士尼評論為** **“** **有損美國價值** **”** **十分到位。**

美國建國不過僅僅三百年，但是在法律、民主、人權方面卻已經是世界的榜樣，美國企業與中共同流合污，著實玷污了美國民主憲政價值，美國政府應積極採取相關行動，配合世界人民的共同抵制來抵禦共產黨邪惡力量。
