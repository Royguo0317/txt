## 《香港自治法案》導致香港官員被美國銀行銷戶
`2020-07-28 07:58 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/278254/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/05105757/66d3745d-abb0-40d3-8043-f34888cda6d2.jpg)圖片來源：澳喜農場
據《金融時報》最新報導，外國銀行應按美國《香港自治法案》規定拒絕給部分香港官員開戶，並應對政治敏感人物的客戶資料審查增加。根據國際反洗錢指南，政治敏感人物的客戶資料審查本來就應該更加嚴格，不知道香港的外國銀行以前有沒有嚴格遵守反洗錢指南。

一名香港的高級官員發現，受中共國安法和美國《香港自治法案》兩頭擠壓，在香港外國銀行開戶越來越麻煩。陳智思（Bernard Chan）是給香港行政長官林鄭月娥做諮詢的行政委召集人，也擔任一家名為亞洲金融的保險公司總裁。亞洲金融是否為中共官員洗過錢呢，這很難說。他說一家美國銀行通知他已經給他銷戶並歸還賬戶餘額，但沒有點出這家銀行的名字，說其他官員也有類似情況。他的銀行賬戶銷戶時間在國安法施行以前，但是類似情況現在越來越普遍了。這家美國銀行是良心銀行？還是有熟悉國際反洗錢規定的戰友在裡面嚴格執行國際反洗錢指南？他向《金融時報》哭訴道：“現在只能在本地銀行開戶了。如果去一家美國銀行開戶，他們都不想和政治敏感人物有任何瓜葛，匯豐銀行也不想被拖下水”。

美國以[《香港自治法案》](https://gnews.org/zh-hans/254715/)對香港國安法暴政進行了報復，對損害香港自由與法治的中共官員和中共香港市委進行製裁，並將與這些犯罪分子有關聯的銀行從美國金融系統踢出去。陳先生從與銀行的交談中了解到，銀行要同時準備香港國安法暴政的加害人名單和被加害人名單，做加害人名單是為了避免美國法律傷害到自己，做被加害人名單是為了避免中共暴政傷害到自己，兩面討好。從《香港自治法案》的立法目的、內容等方面與普通法司法制度的判決原則來看，香港金融機構製作香港國安法的被害人名單以屏蔽被害人的銀行開戶資格，屬於協助損害香港自由與法治，同樣難以保證不會使這些唯利是圖的銀行被排斥在美國金融系統之外。

匯豐銀行最近質詢了香港自由鬥士黃之峰的稿費收入來源，不過他的銀行賬戶仍然是好好的。匯豐銀行拒絕評論，並辯護說自己先前關閉的用以資助反共活動的賬戶是出於反洗錢的目的，難道匯豐銀行沒有替中共官員洗錢？

渣打和花旗也拒絕評論。一個知情人說花旗銀行沒有關閉上述港共官員的賬戶。

在香港經營的全球銀行和基金正讓香港澄清關閉支持香港國安法暴政的官員賬戶是否違反國安法。中共擺出了耍流氓的一貫習性，說這個很難分清黑白，具體情況取決於司法部門和法庭。

評：

從國安法規定和[《金融時報》](https://www.ft.com/content/4e8ad550-0b99-4634-aa93-01ac1a0eb5f2)報導的事實來看，香港已經沒有法治，不可能有正常的資本市場，市場環境早已經改變， [PESTEL](https://www.business-to-you.com/scanning-the-environment-pestel-analysis/#:~:text=PESTEL%20is%20an%20acronym%20that,as%20STEEPLED%2C%20DESTEP%20and%20SLEPIT.)分析中第一位的就是政治環境，全球在港金融機構的管理層應該做完整的PESTEL分析並果斷作出抉擇，除非甘願被共產。
