## 【时事快评】从苏伊士运河航道堵塞到霍尔木兹海峡能源­危机——郭先生一语成谶

引起各大媒体关注的 “长赐号”货轮在苏伊士运河对角搁浅，完全阻塞航道近一周后, 在船尾船首处挖出约3万立方米的沙子，并动用了11艘拖船和两艘强大的海上拖船才将其拖出搁浅地方，重新浮起使航道开通。
![]()![](https://gnews.org/wp-content/uploads/2021/04/image-17.png)图片来自网络
“长赐号” （EVER GIVEN）货轮建于2018年，船长400米，宽近60米，是一艘22.4万吨级的集装箱船，也是世界上最长的集装箱巨轮之一。船东为日本今治造船子公司正荣汽船株式会社。台湾长荣海运公司以湿租（至少一名船员随船出租）方式租为旗下的长荣黄金级货柜船，负责运输长荣货柜。3月23日清晨，该货轮在苏伊士运河搁浅，阻塞航道，3月29日得以解困。

被称之为欧亚大动脉的苏伊士运河贯通埃及，连接红海和地中海，运河全长193公里，交通十分繁忙，地理位置极具战略意义，是连接欧亚的最短海运航线，也是全球少数可以通行大型商船的无船闸运河。

全球海运物流中，约15%的货船要经过苏伊士运河。根据国际航运业刊物《劳合船舶日报》（Lloyd’s List）数据显示，每天有近200万桶石油经过该运河。如果运河阻塞一天，就会导致96亿美元货物停运，相当于每小时4亿美元/330万吨货物，每分钟670万美元。如果绕道非洲的好望角需要多航行10-12天2 ，货运成本将会大幅增加。
![]()![](https://gnews.org/wp-content/uploads/2021/04/image-18.png)图片来自网络
世界保险业巨头德国安联保险集团（Allianz）预计，这次运河堵塞对全球贸易造成每周约60亿至100亿美元损失，年增长率减少0.2 – 0.4个百分点。《华尔街日报》援引海运中介公司Braemar ACM 的数据显示，事故发生后，亚洲到中东海运航线的货轮租金大涨47%3。由此可见苏伊士运河在世界海运中的重要地位。但此次巨轮搁浅事故，让全球意识到它原来是如此的脆弱。

关于这次事故的起因，目前多数的媒体报道都是一般广为流传的已知版本，即所谓 “长赐号”货轮于 3月23日从中国前往荷兰鹿特丹港途中，在通过苏伊士运河时遭遇强风、沙尘暴，船身偏离航道，触底搁浅，横卡在北向河道。然而，埃及苏伊士运河管理当局主管拉比（Général RABIE）将军对最初的沙尘暴导致事故之说提出了质疑，他认为，原因更可能是“技术故障”和 “人为失误”。

有海上、运河、内河航行知识的人都知道，轮船在航行过程中遇到困难复杂航道，恶劣天气，设备故障等情况，船长必须亲自在驾驶室指挥。现代轮船都装备有助航仪器，如雷达、卫星导航、无线电定位仪、经纬仪以及其它船舶安全航行的设备，夜间也能保障正常航行。俗话说，大船掉头难，像“长赐号”这样的巨轮发生“意外”掉头，自然因素确实难以自圆其说。

就在苏伊士运河阻塞发生之际的3月25日，伊朗与中国签署了一项为期25年的协议，该协议包括中国对伊朗能源和重要基础建设等关键领域的投资5。 伊朗驻俄罗斯大使吹捧说，该地区新的南北贸易走廊可能成为苏伊士运河战略航道的”替代方案“。此协议的签署当天已在全球头条新闻中引起关注。

“长赐号”卡住苏伊士运河的事件，怎么就如此凑巧发生在上述中伊协议的签署期的节骨眼上？不能不让人怀疑此起事件的背后是否是某种预谋？

说到这里，笔者不禁联想到郭先生多次在视频中提到中共惯于搅混水摸鱼。郭先生在2018年10月17日参加《路德访谈》时提及另外一个号称全球石油运输的 “总阀门”，连接波斯湾与印度洋的重要通道霍尔木兹海峡。
![]()![](https://gnews.org/wp-content/uploads/2021/04/image-19.png)图片来自网络
2019 年1月25日，郭先生在直播视频里又提到：“委内瑞拉、安哥拉、还有中东，占中国80%的石油、气的供应量，都是要通过马六甲海峡和霍尔木兹海峡”。

2019年5月12日，紧邻霍尔木兹海峡的阿联酋富查伊拉港发生爆炸，造成7艘到10艘油轮严重火灾，阿联酋官方证实是人为的“蓄意破坏”。 这又是谁在霍尔木兹海峡制造危机呢?

郭先生在2019年6月20日的直播视频里再次谈到：“我2017爆料就说过，霍尔木兹海峡是个大火药桶，也是帝国的坟墓。某天某日霍尔木兹海峡，伊朗一旦出现问题，我当时就说伊朗的协议是扯蛋的事，伊美协议不会进行下去。某一天共产党和伊朗和某些人就会合作，把那个油船给炸了。大家知道一个油船是多少吨呢？想想数量多少吨？最窄的地方大家看一下霍尔木兹海峡，阿联酋与伊朗接壤的地方，其中有两三个小岛，有多么的浅，多么窄，一两个船就挡住了，你想清理掉要6-8个月。6-8个月将使能源停止供给60% 到80%，全世界的资源会是什么样子？整个世界经济格局是什么样了？共产党最信奉的一条就是：浑水才能摸鱼！你家的灾难就是我的机会。今天的共产党，已经成为了全人类认为的威胁。那个时候的共产党，或者说那个时候的中国，全人类都忽视了你，或者当你不存在，没人认为你是威胁。现在完全不同了，你想再浑水摸鱼，你就是黑手！”
![]()![](https://gnews.org/wp-content/uploads/2021/04/image-20.png)图片来自网络
无论是苏伊士运河、霍尔木兹海峡还是马六甲海峡、巴哈马通道，中共通过香港的长江和记实业集团已控制了数十个潜在的经济咽喉，包括全球52个港口的287个泊位6。在被美国国防部标记为“美国生命线和过境地区” 的八个扼流点国际地区中，和记就控制了6个港口，这对美国和世界安全构成的潜在威胁是难以低估的。

因此，笔者认为，苏伊士运河航道堵塞事件之后，未来在霍尔木兹海峡仍然还将大概率发生新的破坏性事件。请保持密切关注。
