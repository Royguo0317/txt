在与中共病毒大流行作斗争的过程中，全世界至少有11.5万医生和卫生工作者丧生。世界卫生组织总干事谭德赛今天向他们表示敬意。

谭德赛在第74届世界卫生大会开幕式上说：“在将近18个月的时间里，世界各地的卫生保健和社会护理工作者在生死之间工作，挽救了无数生命，并为其他人奋斗，尽管他们尽了最大的努力却没有成功。许多人自己感染了病毒，尽管鲜有报道，我们估计至少有115,000名医务工作者在为其他人的服务中付出了生命的代价。卫生和社会卫生工作者做着英雄事，但他们不是超级英雄。他们和我们其他人一样都是普通人。他们也会发笑，哭泣，恐惧和希望。其中许多人为此感到沮丧，无助和无保护，并且无法获得个人防护设备，疫苗和工具”。

谭德赛继续表示：“我们正在与病毒交战，这需要战争的逻辑和增加武器的能力。如果我们现在不采取行动，而该病毒将继续造成深重的苦难，在最贫穷的国家中传播和改变。”
