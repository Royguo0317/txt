《纽约邮报》11月21日报道称，肖恩·乔伊斯（Joyce）被聘为Airbnb公司的第一任首席信息安全官，以帮助保护平台上用户的信息安全，但在工作仅6个月后，他就突然辞去了行政职务，“出于对大规模租赁平台如何共享数据的担忧”。 [1]

Airbnb的前首席信息安全官肖恩·乔伊斯（Sean Joyce）也是联邦调查局（FBI）的前副局长。消息人士称，“乔伊斯在任职期间越来越感到震惊，因为该公司对其与执政的中共政府共享的数据并不完全透明，包括在该国旅行的美国人所共享的数据”，“他还担心Airbnb有意愿考虑来自中共国的更多数据请求。”

新闻背景：

1. 2009年4月，该公司从红杉资本获得了种子资金。 2010年11月，他们在Greylock Partners和红杉资本（Sequoia Capital）的A轮融资中筹集了资金。 [2]

2. 2014年4月，该公司完成了TPG Capital 4.5亿美元的投资，该公司的价值估计约为100亿美元。 [3]

3. 自2007年推出以来至2013年，Airbnb其累计住宿人数将突破1000万。在2013年有超过600万的访客在该平台下住宿订单，比前一年增加了一倍多。到2019年10月，每晚有200万人住在Airbnb。 [4]

4. 为继续为中共国提供更多的个人信息，Airbnb联合创始人内森·布莱卡奇克（Nathan Blecharczyk）对负责该平台的高管说：“我们不是在这里提倡美国价值观。” [5]

新闻分析：

肖恩·乔伊斯（Sean Joyce）作为联邦调查局（FBI）的前副局长，上任该公司信息安全的主管职务，对公司现状应有预计的，但依然被信息暗箱泄露的程度所震惊，并在短期内迅速辞职，只能说明这其中涉及极其严重的违法行为！

红杉资本（Sequoia Capital）作为众所周知的中共国代言人，竟然在2009年就参与Airbnb天使轮投资，其在Airbnb董事会拥有极大的决定权，除了坐收几十亿美元的投资收益，同时也建立起中共国收集美国公民个人信息的重要管道。

“到2019年10月，每晚有200万人住Airbnb”的数字看，推测中共国通过红杉资本（Sequoia Capital）控制的Airbnb，获取了数千万美国公民的个人资料，其中极有可能包含信用卡账号、安全码和社会安全号等重要隐私信息。
