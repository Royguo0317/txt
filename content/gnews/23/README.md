**译文：**

1. **迈克尔****·****伊登博士揭露关于冠状病毒的八大谎言，谎言的目的是强制推行全球数字身份证和疫苗护照**


成为举报人的前辉瑞（Pfizer）公司副总裁迈克尔·伊登（Michael Yeadon）加入了【战斗室】，揭露政府误导公众接受疫苗护照的八大谎言。

伊登在制药业工作了32年，是10年前离开的辉瑞制药。他说自己必须要说出真相，因为政府在病毒、封城和疫苗等各个方面都在对公众散布着“巧妙的撒谎”。

伊登博士说：“政府告诉我们的关于病毒、治疗和疫苗的一切信息都是错误的。为各国政府提供建议的人都很精明，但他们说的却是谎言。这样做的目的是为了恐吓人们接受那些基于基因的非同寻常的疫苗。”

他说：“我认为他们的终极目标是迫使我们每个人都拥有一个电子疫苗护照，也就是数字身份证。”

“总而言之，这就是我必须发声的原因。”

**冠状病毒的八大谎言：它是一种全新的病毒（不，它与四种不同的感冒病毒相似）；人体完全没有免疫力（有）；病毒致死率超高（不是）；没有可治疗的方法（有）；口罩有效（没有）；封城有效（没有），有无症状传播现象（没有）以及疫苗都是普通疫苗（不，有基因疗法疫苗）。**

##### 2. 前辉瑞公司副总裁伊登博士：变异的冠状病毒“并不可怕”

前辉瑞公司副总裁迈克尔·伊登（Michael Yeadon）博士说，对新冠变种病毒的恐惧是政府助推的谎言，并没有科学依据。

他周三告诉【战斗室】：“这种病毒会变异，变异的产物我们称之为变种病毒。但从基因学上来说，它们从属于同一个大的病毒族类。”

伊登博士解释说，过去的18个月里这种病毒在其最初的武汉病毒序列基础上只变异了0.8%，这意味着它仍然与原始病毒株几乎保持着相同的性态。

伊登博士补充说：“事实证明，它实际上并没有超出人体的免疫抵抗范围。因而人们对这些病毒变种不必过度担心。然而，政府却在告诉你，你可能需要接种不同的疫苗。”

这位大药厂的前科学家如今变成了爆料人，他说世界各地的政府散布了关于病毒、封城和疫苗的八大谎言，目的就是推动全球数字身份证。

##### 3. 揭秘：福奇、达萨克和政府资助的科学家们在2020年初是如何掩盖新冠病毒来自于武汉实验室的

劳伦斯•塞林（Lawrence Sellin）博士是研究中共病毒事务的专家、美国陆军预备役上校，他揭露了托尼•福奇如何在2020年初帮助掩盖新冠病毒的武汉实验室起源。

塞林在周三的《战斗室》节目中展示了福奇、彼得•达萨克（Peter Daszak）和其他政府资助的科学家之间的电子邮件和Zoom通话记录，证明他们合谋压制了涉及武汉实验室的信息。

塞林说：“很明显，他们尽一切可能压制了关于新冠病毒源自实验室的任何讨论。”

在他们在Zoom上进行了通话商议之后， 由政府资助的研究员克里斯蒂安·安德森(Kristian Andersen)发表了一篇流传很广的论文，声称该病毒起源于自然。论文被引用了成千上万次。

“福奇在2020年4月再次现身，并且得到一封来自达萨克的电子邮件，达萨克表达了对福奇支持他推广新冠病毒自然起源说的感谢，说福奇的帮助很大。”

塞林说：“我认为所有的这一切都是在故意掩盖新冠病毒的起源，使我们在搞清病毒的源头和到底是谁把它弄出来的问题上，至少延迟了一年时间。”

塞林博士还将中共对美国科学界的渗透程度评为 “10级”。

当被问及如果用1到10的数值来描述中共国对美国的渗透程度，他会如何评级时。他说：“显然是10级，这是毫无疑问的。”

塞林博士说：“这种情况已经持续了几十年。一开始还只是凤毛麟角，到现在已经有成千上万的中共科学家在美国接受了训练，有的学成后回了国，有的则继续留在了这里。”

他说：“在美国的大学里，政府部门，甚至是国防部，到处都有他们这些人。”

##### 4. 福奇的邮件门：新冠病毒最可恨的谎言就来自于这个福奇博士

拉希姆•卡萨姆（Raheem Kassam）披露了福奇博士最具爆炸性的电子邮件，这些邮件证明福奇对疫情大流行撒了谎，并对和他有关的病毒功能增强性研究要被曝光深感恐慌。

福奇不仅在私下里说过口罩不起作用，也说过不存在无症状传播的现象。他本来知道病毒很可能是基因工程的人造产物，却全力向公众推广虚假的病毒自然起源理论。

福奇向他在国际信息销毁协会（NAID）的副主任发送了由武汉的“蝙蝠女”石正丽博士和北卡罗来纳大学教堂山分校（UNC）的拉尔夫·*S*·巴里克合作撰写的关于冠状病毒的功能增强性研究的论文。福奇资助了这项研究，但在国会宣誓后的证词中却矢口否认这项研究的存在。

此外，一位政府资助的科学家克里斯蒂安·安德森(Kristian Andersen，就是后来撰写了病毒来自自然的那篇论文的人–编者注。)在2020年1月31日的一封电子邮件中告诉福奇说 “病毒的一些特征看起来像是经过人工编辑的。”

卡萨姆说：“往轻了说，这一证据明确证明了病毒人造说根本就不是什么阴谋论，福奇在美国国立卫生研究院和自己的专家们老早就在议论这个话题了。”

“往重了说，这就是一次全世界前所未闻的真相大掩盖。”
