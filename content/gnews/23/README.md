**班农先生**：川普总统在11月3日赢得的大规模胜利在被邪恶的人盗取，我们必须在一切能够利用的战场进行反击。

如果选举学院在选举被盗问题解决之前就开会裁决，那么 2021年1月20日宣誓就职的将是乔·拜登。我们现在需要解决这个问题，我们现在在浪费时间。

**杰克·波索比茨**解释了昨晚午夜时分受到的重击：这是肖恩·康奈尔与议员迈克·凯利一起提起的诉讼。帕特里夏·麦卡洛是宾夕法尼亚州联邦法院的法官。法官最不喜欢的是对他们撒谎，尤其向他们提供的不实信息让他们做出错误的裁决，她之前的裁决就是听了认证已经完成，认证结束了，一切都彻底完成了。现在她听到这种新观点，认为相当紧急：不对啊，你在发布新闻稿，但委员会已经提出选举人尚未裁定，许多选票尚未完成认证，那你在电话会议上说的是怎么回事？她对这项裁定非常不满。

但是更大的问题是，这又与宾夕法尼亚州的规定有关，即所谓的无借口的缺席选民投票，即有四个类别，每个类别都需要缺席者满足特定的条件。这是宪法规定，几乎与威斯康星州类似，因为州宪法已经将这些规定定下来了，你不能通过立法行为改变宪法的规定。

这是一个完全独立的案例。尽管宾夕法尼亚州的州最高法院由绝对多数的民主党控制，但最终裁决的不是州最高法院，而是联邦立法机构。

马斯特里亚诺上校所做的是改变战略。他和朱利安尼举行的听证会是立法委员会会议。那是第一次听证会，就是一步高棋，显示出对比的力量。他在向整个共和党展示，如同跟随他三十一年的士兵跟随他取得胜利那样，要么跟着他一起战斗，要么就可能会重新组建一个党派。

**前纽约市警察局局长伯纳德·凯里克**：我们做的与宾夕法尼亚州一样，那就是教育立法者，让他们确切地了解发生了什么，因为如果你上网看，或看主流媒体，或是听许多政客们，没人在说真话，没有人谈论投票机和软件等问题的真实情况。

目前最大的问题是立法机关，还有总检察官们都缺乏勇气。

他还爆料：星期一或星期二，我们可能有足够证据缴获多台投票机。如果他们在这之前尝试抹去机器信息，我们的审查员中有法学科学家能够将其恢复。还有更重要的，现在已经有在选举委员工作的人挺身而出，他们会告诉我们投票机发生了什么，查看哪里，捕取投票的地方，这会非常有帮助。

**拉希姆·卡萨姆：**川普总统应该解雇身边那些告诉他拖时间的所有顾问。我们现在需要50个朱利安尼，珍娜律师这样的人们推促他前进，这样才能在12月14日之前按计划时间实现目标。

由于真正的媒体无能为力，以及川普团队沟通策略不完美，因此，很难看到川普在各个战场州取得胜利的信息。

**我们同时在打两场战争：信息战和正义之战**。
