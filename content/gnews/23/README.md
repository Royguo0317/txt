**西班牙2021年1月29日电/西喜社——**斯里兰卡周五开始为前线卫生工作者、军队和警察接种中共病毒疫苗，因为有人警告说，由于卫生人员感染中共病毒，医疗部门面临崩溃。

斯里兰卡周四收到了50万剂疫苗，作为邻国印度的捐赠。牛津-阿斯利康疫苗又称COVISHIELD，由印度血清研究所生产。

政府表示，15万名卫生工作者和11.5万名军警将首先获得该疫苗。

斯里兰卡自10月以来一直在经历新的疾病爆发，出现了两个集群—一个以服装厂为中心，另一个以鱼市场为中心。

斯里兰卡共报告了61,585个病例，其中297人死亡。
