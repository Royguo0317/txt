## 八角棒槌：奔湧吧！滅共第一浪！
`2020-05-06 20:33 GM09` [轉載自GNews](https://gnews.org/zh-hant/195839/)

可能很多人都已經看過，5月4日那天，央視跟bilibili合作，搞了個叫《後浪》的演講。因為有B站參與，八方知名UP主便有幸——他們自認為——獲得賣力的資格。至於對廣大年輕觀眾會產生什麼影響，那不是該考慮的事，只要能跟中央，跟官方搭上關係，就能一舉奠定更加出名的根基。因此，為短片的混剪奉上素材才是當務之急。老實說，我只是個屌絲，並且看問題也很不積極，好在已年逾不惑，有些事一旦看明白，就覺得很沒意思。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/06202932/Screen-Shot-2020-05-07-at-9.28.44-AM.png)
一部近四分鐘的短片，演講者那因年齡和世俗賦予他的老沉持重，混合中共不斷叮囑要展現出的慷慨激昂，一隻盛滿新鮮雞血的精緻大碗端上檯面，呈現在熱情單純的年輕人面前。對我個人而言，這一畫面既視感十足，頓時讓我想起了我們遙遠的過去。在那時，皇子謀逆一旦敗露，帝王往往也會派人端來一隻碗，只不過盛的不是雞血，而是羊肉湯豬骨湯什麼的。但這是在外人看來，只有帝王和皇子才最清楚，那是一碗鴆毒。

如果我說《後浪》裡有毒，恐怕不等中共動手，眾UP主的唾沫都能把我淹死。不過我也能理解，畢竟除死亡本身，最為人所恐懼的，該當是自身已凝固的價值觀被徹底顛覆。因此，面對無形中已全然接受的錯誤認知，我並不奢望每個人都擁有敢於否定自己的勇氣。並且我也相信，一旦面對無數殘忍的真相，即便有勇氣，也會在勇氣之後因創傷而迷失。不覺得疼痛，是因為不知有傷，後來知道了，卻對未來徹底絕望。

在這條道上，海明威曾是過來人，一本《太陽照常升起》堪稱迷失的最佳註腳。有點不同的是，當年那一代的迷失，歸根結底是起於對人類戰爭的重新認識，而正等待著我們的那份迷失，理由卻是因上當受騙隨時會死在自己主子手上。“六四”之雷被中共埋了三十年。三十年，中共飄在鮮花密布的雷區上空，左手高舉國旗，藏在背後的右手捏著刀柄，不斷號召腳下的年輕人前進！前進！於是迎著學費、996、婚姻、房貸車貸、苛捐雜稅的槍林彈雨，無數青年邊匍匐前進，邊扯紅脖子上的青筋，不住地高喊祖國萬歲！

如此陣仗雖不比當年戰場上的血肉模糊，屍體橫陳，可論其滑稽程度，相信早已遠超當年的架勢。想返校好好學習？對不起，一場病毒下來，宿舍變成了方艙，貴重物品不翼而飛的同時，課本行李扔得遍地都是；想玩玩遊戲呢？也對不起，別說《如龍》和《戰地4》，就連和諧有愛的《動物森友會》都被禁。對不公稍稍有牢騷，新浪知乎賬號就保不住；敢公然反抗的，一百人就叫你消失，一百萬人則不惜葬花的代價，再次啟動雷區重演“六四”。在年輕人身上，中共犯下的罪罄竹難書，面對這樣一個主子，你說它的《後浪》有沒有毒？

中共的字典裡沒有有趣，只有有意義。至於什麼才算有意義，全得依黨的號召來定。我已年逾不惑，毫不謙虛的說，中共衡量“有意義”的標準我看得很清。簡單概括有兩條，第一條叫仇恨，目前看，這條成效已遠超中共的預期，否則一邊用蘋果看好萊塢和HBO的劇，一邊肆無忌憚唾罵美帝的怪象你就不能解釋。

另一條叫恐懼，出於對民眾的恐懼，中共一定會控制民眾，讓他們貧窮而不得閒暇。古代帝王靠的是大興土木，中共的絕技當屬蠱惑人心。經歷了大躍進、大饑荒、文化大革命，人們餓著肚子，不是用鍋片子煉鋼，就是互相批鬥，上山下鄉。轉眼七十年，作為恐懼的一體兩面，年輕人也好，中年人也罷，中共腳下的民眾早已被恐懼懾得服服帖帖。毒奶粉毒疫苗得忍著，P2P爆雷得忍著，物價上漲繼續忍著，油價不降反升？笑話！連冠狀病毒都能忍，區區油價算個啥？

今天回想起來，如果沒有爆料革命，恐怕這恐懼的皮球，將再也沒機會一腳踢回給中共。全世界和爆料革命，只要美國一出手，定會形成內外夾擊，叫中共立馬隔兒屁。中共的結局如何，中共自己也看得一清二楚，不然它就不會拼命躲在十四億民眾背後，發瘋叫囂著要用兵馬俑幹倒自由女神。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/06203012/Screen-Shot-2020-05-07-at-9.29.13-AM.png)
在中共的《後浪》短片中，演講者提到要把民族的變成世界的。記得有種說法叫民族的才是世界的，假如此話當真，那兵馬俑就該是世界的。聽起來似乎沒問題，可再仔細一想，倘若把工藝技術和美感，以及時間賦予它的歷史意義等文化成分剝離開，想必兵馬俑就只剩是專制集權的象徵。試問這時它還是世界的嗎？我說不是，所以我貓在Gnews寫東西，而中共說是，因此一切中華文化全被它砸毀，最後只捧著個烏托邦的象徵，向全世界進軍。

當然，我不否認民族的可以是世界的，但前提得撇開阻礙文化的一切象徵。年輕人的確可以是後浪，但要說後浪就該被前浪麻痺和統御，我看還不如做前浪。不禁要做前浪，還要學習香港青年，堂堂正正做第一浪！在爆料革命的影響下，全球滅共勢不可擋。年輕人們想要有未來，只能做第一浪，否則年輕時浪再大，不再年輕時只剩一個泡，直到死去後連泡都不剩。因此我也要跟年輕人一道，像水冰月那樣堅定地立誓：我要代表自己，消滅中共！
