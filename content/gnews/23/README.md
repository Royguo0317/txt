## 五眼情報聯盟報告指出中共蓄意掩蓋事實
`2020-05-03 22:26 GM10` [轉載自GNews](https://gnews.org/zh-hant/193367/)

**作者：Dogman**

澳洲每日電訊（Daily Telegraph）獲得一份由五眼情報聯盟撰寫的研究報告， 指出這份15頁的報告針對中共對病毒控制不力提供了證據。

五眼情報聯盟是由美國、加拿大、英國、澳大利亞和新西蘭的情報機構組成。

這份卷宗的內容涉及了爆料革命以及一些媒體一直宣傳的內容，其中包括早期中共蓄意虛假宣傳病毒不會人傳人，噤聲和讓吹哨人消失，毀滅實驗室的病毒證明，在未獲得病毒源頭的情況下對武漢華南海鮮市場進行消毒清理以及拒絕為國際科學家提供活性病毒樣本等。

“雖然科學界之前普遍表示病毒來自海鮮市場”，每日電報說，“美國的情報委員會首次開始調查CCP病毒是否來自美國的實驗室”。

針對中國政府對病毒進行掩蓋的事實行為，這份卷宗提到中共“壓制並銷毀了證據”，並指出 “中共命令在實驗室中銷毀病毒，對海鮮市場進行消毒清理，不對外公布病毒測序序列，發布文章需要提前通過審核才能刊登等”。通過每日電報的描述，讀者能看到病毒擴散後，中共對消息的封鎖，對證據的毀滅，以及對國際社會需要病毒活性要求的各種不配合。

文中還提到了對於整個CCP病毒流行以及研制疫苗極為關鍵的“零號病人黃燕玲”。黃此前在武漢病毒研究所工作，據南華早報報道，她被認為是第一個染上CCP病毒的 人，但是現在黃的個人信息在武漢病毒研究所的官網上已經找不到。

在媒體關於美國NIH對武漢實驗室的資助進行報道之後，這份卷宗中揭示了，在明知病毒可能對通過蝙蝠直接傳染人並且沒有解藥的情況下，“澳洲政府負責培訓並資助了中國的科學家團隊”，並且“這項合作一致延續至今”。

結合2020年5月2日郭文貴先生的直播透露出的信息，與武漢病毒研究所息息相關的科學家已經逃離中共的控制，目前正在與美國政府合作，相信在未來不久，CCP病毒的源頭，以及背後各種支持相關研究工作的機構都將會公之於眾。
