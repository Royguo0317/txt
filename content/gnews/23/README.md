香港原是一个小渔村，1842年8月29日，清政府在第二次鸦片战争中战败，《南京条约》清政府将香港岛割让给英国，英国开始管制这个港口小村。1843年建立港英政府，这个小港口由英国优良的资本社会体制进行各方面的商业运作，吸引华人商业中高层入住香港经商品。港府既然是广九铁路和启德机场。在有秩序的发展中，这个期间，大批中层高层华人带资本到港口去参加建設与商业业务，小魚港成为初具规模的商业重镇。

1941年日本在第二次世界大战侵略国占领香港，香港建设一度遭到停滞。直到1945年日军向英美联盟双手举投战败书后撤离香港。二战结束之后，港英政府开始注重港口小镇的有序的建设，首先在工业外贸的发展，順带金融产业，房地产等行业的跟进建设。慢慢演变成为一个港口吞吐量逐年递增递增。成为亚洲远东一个重要的自由贸易港口，也同时成为一个工业化城市，而且多元经济与世界各族文化同步发展成为多元文化互相包容的大都巿。经过四十年左右的发展，19世紀80年代，香港已经成为一个世界的重要城市，仅次于伦敦，纽约之后，成为一个东方的世界金融名城。

香港从一个半封闭狀況的小魚村港島，慢慢地逐年提提升到层次高的开明与民主的生活中。香港政府引领的西方文明民主制度在香港的成功转型，居民生活素质逐漸得到提高。民众的思想层面，从底层次上升到一个更高的文明社會境界，接受西方民主制度的优良体制— 西方文明社会基督信仰的契约精神。作为东亚的一个重要港口城市，香港吸引全球各地大批貿易与銀行公司入驻，各界精英逐年而来到香港長驻，经过几十年的发展，香港名副其实成为东方明珠。

《东方之珠》，这首曲子演绎了香港本土民众亲身经历香港由一个小港口发展到东方名城的一种浓厚的刻骨铭心之眷恋情怀。其中之一词＂每一滴泪珠仿佛都说出你的尊严，让海潮伴我来保佑你＂。这一句是最能演绎着香港如何由一个小渔村成为一个世界的金融都市，这都是香港政府在英国用西方資本与管理体制来指导香港政府是如何将其建设成为一个著名的金融工业贸易大都市。

香港的文明自由制度是英国西方文明社会在东方结出了普世价值的人权价值的体现。与共产联盟制度的财产共有极权统治有天地之别。因为共产联盟是没有信仰的极权专制制度，故中共国极力地打压有宗教信仰的百姓，各地全力拆掉毁坏教堂寺庙，唯拜共产恶魔，从从幼儿园开始进行＂别亲娘亲不如党亲＂的邪恶腦控。在有信仰自由的香港文明社會，共产联盟用无神论脑控人民如奴隶不能有异议与思考。而受西方民主的信仰自由进入骨髓的香港民众。必将影响到共产主义联盟的极权专制统治。

在1997年7月1日英国将香港主权有条件的前提下，将香港归还给中共国管理执行一国两制五十年制度不变。在2019年5月中共国拟将通过法律将香港的民主司法制度终结。将饭犯罪押送到中共国审判。这违反了中英國对香港的一国两制制度，所以香港民众几百万人上街抗议游行知道今天。中共国派驻黑警与港警配合，暗中指挥林邓港府，武力鎮压香港的法律与人生自由，八九六四在香港重演，赤匪紅魔并控制了香港，遭到全世界西方文明国家的制裁。
