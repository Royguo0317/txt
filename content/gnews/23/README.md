## 亚利桑那州马里科帕县在就职仪式1小时后才提交选举证据
` 喜马拉雅-华盛顿DC农场` [轉載自GNews](https://gnews.org/zh-hans/780629/)

**作者： CPA Jim**

OANN记者克里斯蒂娜·鲍勃（Christina Bobb）2021年1月20日发推说难以想到的是亚利桑那州马里科帕县（Maricopa）选举监督委员会同意在拜登就职仪式后一个小时才发布选举证据。所附附件内容是经过艰苦的斗争，亚利桑那州议会得到了马里科帕县选举监督委员会的同意，提交亚利桑那州议会所发传票中要求的所有选举相关信息以开展选票审计。

从这里可以看出在1月6日确认选举结果前，由于审计证据尚未取得，选票审计都还没有完成。按照国际审计准则，审计证据都没有，无法实施替代审计程序，如果又要求出审计报告，只能出无法表示意见的审计报告。但是为何要拖到现在，才肯提交选举相关证据呢？
