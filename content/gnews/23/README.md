近日香港全城哄动，最热话题之一莫过于接种新冠疫苗可参加抽奖。发展商信和及华置昨日公布，9月1日或之前完成2剂疫苗的满18岁香港永久性居民，可能获地价值1,080万元观塘凯滙一个面积449平方一房全新单位，且发展商会代缴印花税或双倍印花税。

对于信和及华置送出一个凯滙单位作为奖励，诱引广大市民接种新冠疫苗，地产建设商会主席兼会德丰地产主席梁志坚5月28日表示已知悉有关活动，相信各行家随后会跟随。梁称，每间发展商都有其途径配合政府抗疫。发展商送楼吸引市民打针这一消息随即便引起社会讨论。据港府最新数据，截至28日晚8时的过去24小时，有约2.56万市民在网上预约打针，较前日2.02万人，急增逾5,000多人。但政府疫苗预约网站则显示两款疫苗接种中心，6月可预约日子仍有甚多名额，暂未见因「有楼抽奖」而爆满。

根据《经济日报》资料，有关楼盘价单，凯滙3座47楼D室价单定价为1277.4万元。另外据售楼说明书所示，该单位面向西北，单位设有一个露台、工作平台，设有「梗厨」，浴室设有浴缸，客饭厅为长方形间隔。

据报商界还有多个机构加入诱引市民接种新冠度苗的行列，行政会议成员、经民联立法会议员林健锋表示，现时本港疫苗接种率不高，正与香港总商会商讨，分2阶段推出疫苗奖赏计划，赠送不同「衣食住行」的奖赏，鼓励市民接种疫苗。当中更劲爆的有婚姻配对服务，已接种第1针的市民可以递交相关资料，接种第2针后便可安排与对象见面，亦设有现金奖，详情会再公布。

林健锋5月30号出席电视节目时指，早前与多个机构商订资助提供奖赏，疫苗奖赏计划中有不同商户、机构送出如酒店住宿、机票、戏票、饮食券、抗疫产品、验身、验眼的医疗券等。希望有打针的市民亦可参与，计划每两星期抽1次细奖，并在9月再定日子去抽大奖。
