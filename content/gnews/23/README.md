## 眾議院共和黨公佈應對中共威脅報告
`2020-10-01 11:25 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/395539/)

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/01110819/101-3.jpg)圖片出處：abcnews.go.com
據《政治》（Politico）9月30日報導，一個由共和黨領導的特別工作組週三公佈了一份應對中共威脅報告。此前，該工作組就CCP病毒爆發的根源、醫療供應鏈的缺陷等問題對中國共產黨進行了長達數月的調查。

該小組由15名眾議院共和黨人組成，由得克薩斯州眾議員邁克爾·麥考爾（Michael McCaul）擔任主席。該小組於5月在全球CCP病毒大流行期間正式成立，共和黨對中共未能披露有關該病毒早期傳播的關鍵信息表示震驚，並呼籲美國應當減少對中共國的醫療用品依賴。不過，從更廣泛的意義上講，共和黨人認為，中共的侵略行為太長時間未受制止，現在是美國改變其以往失敗戰略的時候了。

眾議院少數黨領袖凱文·麥卡錫(Kevin McCarthy)在周三的新聞發布會上說，近八個月來，他一直在努力組建一個兩黨工作小組，以應對來自中共國的經濟、技術、軍事和政治威脅。但麥卡錫說，民主黨人最終在該計劃宣布的前一天晚上退出了該計劃。唐納德·川普（Donald Trump）總統和共和黨（GOP）以抨擊中共政府作為2020年競選主題，同時描繪民主黨總統候選人喬·拜登（Joe Biden）為對抗中共的軟弱派，民主黨人當然予以否認，並抨擊川普在疫情爆發伊始曾吹捧中共應對病毒的方式。麥卡錫表示，如果拜登入主白宮，他擔心這位前副總統不會像川普政府那樣對中共政府採取強硬態度，從而導致這個計劃最終流產。

據麥卡錫介紹，共和黨的報告列出了83項主要調查結果，提出了430項政策建議，其中三分之二是跨黨派的。自成立以來，工作組已與125人會面，其中包括政策專家、商業領袖、議員以及現任和前任政府官員。麥卡錫對記者說：“這不僅是眾議院歷史上關於中國的最全面的報告，而且是大膽、可行、跨黨派的。它不僅列出了挑戰，還列出了解決方案。” 該報告呼籲確保醫療和國家安全供應鏈的安全，與台灣達成雙邊自由貿易協定，並限制節儉儲蓄計劃資金投資於某些中國公司。該報告還呼籲評估中共在新疆的反人類罪是否構成種族滅絕，並為香港難民提供安全庇護所。特別工作組的成員希望其中許多可以成為法律，麥卡錫還強調稱，如果共和黨贏回眾議院，實施這些建議將是他們的首要任務。

另《美國之音》有更詳細的報告部分內容：這份眾議院外委會少數黨有關新冠病毒起源和向中共及世衛問責的報告得出結論認為，中共在武漢疫情爆發初期隱瞞真相，打壓披露真相的醫療人員，未能履行《國際衛生條例》中有關在第一時間向世衛組織通報重大公共衛生事件的規定，這種瀆職行為給全球造成重大生命和經濟損失。報告認為，中共高層不但沒有及時向世界發出警告，反而在海外秘密購買大量醫療物資並將國內的供應鍊和產能國有化，而當有其他國家要求對中共問責時，中共則以經濟和貿易手段加以脅迫，要求保持沉默。

報告還認為，世界衛生組織迎合中共宣傳，助長了中共的掩蓋行為。這份眾議院外委會共和黨人的報告反對美國退出世衛組織，但提出四項建議：要求更換世衛組織領導層，總幹事譚德塞引咎辭職；恢復台灣在世衛組織的觀察員身份；美國與志同道合的世衛組織成員國和台灣對中共在疫情早期的掩蓋行為以及世衛組織的失誤展開國際調查；改革《國際衛生條例》。最終報告提到，在中期報告發布之後，世衛組織和中共政府都修改了他們關於新冠疫情的公開聲明：世衛組織發布了新的、“更新”的時間線；中共暫時收回了他們在2019年12 月31 日通知了世衛組織的說法。
