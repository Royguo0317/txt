03月11日，据塔斯社报道，俄罗斯国家杜马议长维亚切斯拉夫·沃洛金（Vyacheslav Volodin）对记者说，俄罗斯正在与其他国家的结算中使用本国货币，以不断减少对美元的依赖。

他以欧亚经济联盟（EAEU）为例，沃洛金说，欧亚经济联盟以本国货币结算的比例已经增加到74%。“我们系统性地增加与中共国、土耳其的本国货币贸易。”

此外，2月底，俄罗斯将日元和人民币纳入国家福利基金（NWF）的货币结构，同时将美元和欧元的份额从45%降至35%，国家杜马发言人提醒。

3月初，联邦委员会议长瓦伦蒂娜·马特维延科（Valentina Matvienko）表示，俄罗斯应减少经济上对美元的依赖。同时，美国的货币将继续流通。俄罗斯议会上院负责人表示，由于近年来央行和政府代表总统推行的去美元化政策，俄罗斯的外汇储备对美元的依赖性降低，俄罗斯对美国政府债务的投资在过去十年中下降了30倍。

此前，外交部副部长谢尔盖·里亚布科夫（Sergei Ryabkov）表示，应采取紧急措施减少以美元结算，认为乔·拜登（Joe Biden）领导的美国新政府正准备加强对俄罗斯的制裁。
