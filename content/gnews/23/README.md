## 【軍事快訊】美前高官建議為台灣提供F-35隱形戰鬥機支持，以應對中共威脅
`2020-10-10 10:50 CHANGDAO` [轉載自GNews](https://gnews.org/zh-hant/415909/)

作者：美東香草山農場滅共霹靂組  人間世



中共戰機近期頻頻擾台，挑釁行為不斷升級，致使台海局勢驟然緊張。美國前國防部副長佈萊恩（Stephen Bryen）認為，台灣仍需美國更大實質承諾。他提出，可以讓美國代訓台灣空軍飛行員，並以租賃形式提供台灣F-35隱形戰鬥機編隊，以協防颱灣。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/10103859/114-%E3%80%90%E5%86%9B%E4%BA%8B%E5%BF%AB%E8%AE%AF%E3%80%91%E7%BE%8E%E5%89%8D%E9%AB%98%E5%AE%98%E5%BB%BA%E8%AE%AE%E4%B8%BA%E5%8F%B0%E6%B9%BE%E6%8F%90%E4%BE%9BF-35%E9%9A%90%E5%BD%A2%E6%88%98%E6%96%97%E6%9C%BA%E6%94%AF%E6%8C%81%EF%BC%8C%E4%BB%A5%E5%BA%94%E5%AF%B9%E4%B8%AD%E5%85%B1%E5%A8%81%E8%83%81.png)圖為美國空軍F-35戰機。 （美國空軍/資料照）


美國前國防部副部長，國防技術安全管理局創始人布萊恩近期在《新聞周刊》(Newsweek)刊文指出，面對中共對台灣的軍事威脅，美國不能袖手旁觀。目前美國限制對台出售先進戰機，主要基於兩個原因，一是擔心台灣空軍飛行員能力不夠，二是擔心中共國的反應。因此他建議，可由美軍培訓台灣飛行員，以解決第一個問題，同時，在美建立一支可供台灣調遣或租賃的F-35後備機隊，以解決第二個問題。

布萊恩還指出，台灣目前沒有隱形飛機，而中共國去年中開始全面量產J-20隱形戰機。同時，台灣需要一架可以在公路和簡易機場上運行的短距離起降飛機，F-35B是世界上第一架超音速短距起飛且垂直著陸的隱形飛機，非常適合台灣，並有足夠保護台灣的作戰範圍。

布萊恩強調，為台灣提供一支F-35戰鬥機編隊是一項重要的政治決斷，有助於穩定東亞安全局勢，而能否做出這一決斷，關鍵在於美國和台灣是否擁有足夠勇氣。
