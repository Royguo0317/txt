## 藏族士兵奮勇參戰欲奪回家園，中共必為挑釁印度付出沉重代價

校對 坐看雲起時

據9月23日Brahma Chellaney新德里報導，中共外交部長王毅無恥地宣稱中華民族沒有侵略和擴張主義的“基因”。的確，侵略和擴張主義的確不屬於“遺傳基因”，中華民族也沒有這種基因。始終有著逐步滲透乃至侵略和擴張主義的是中共自己，與中華民族無關。中共一直試圖建立現代版的共產帝國，以此將其他國家變成其準附庸國，如古代帝國一般只有屈服於皇帝才可獲得所謂“和平公平”地與帝國貿易的好處。

中共企圖利用中共病毒肆虐全球之際，達到其迅速擴張掌控世界野心。因此在4月和5月，中共黨衛軍（PLA）偷襲印度拉達克地區寒冷邊境地區並在那裡建立了堅固的設防營地。

事實上這一舉動沒有達到預期目的，反而激起印太國家的猛烈回擊，強化了他們之間安全合作，其中包括中共國最強大的競爭對手美國。中美雙邊在技術，經濟，外交和軍事等各方面戰略對抗全面升級。如今中共國面臨與世界脫鉤和供應中斷的危機，不得不大量購買囤積礦產資源和農產品。

中共這次真正誤判了其喜馬拉雅邊境鄰國印度，此舉迫使印度放棄以往對中共國的綏靖政策。中共打著要求“鞏固邊境防禦”並“確保喜馬拉雅地區的邊境安全”的旗號，實則乾著對其鄰國的非法入侵行為。

但是這次印度絕不妥協而準備戰鬥。 6月在中共黨衛軍伏擊並殺害了在拉達克加勒萬河谷巡邏的印度士兵之後，一場徒手直接對抗導致許多中共國士兵喪生。這是中共黨衛軍近40年來除了在聯合國維和之外的行動中首批遇難者。這一結果讓中共非常尷尬。以致於在印度紀念20名陣亡的烈士時，中共卻不敢承認中方確切死亡人數。

事實是中共國沒有能力在軍事對抗中戰勝印度。為了應對挑釁，印度在喜馬拉雅邊境部署了相匹配的軍事設施，並運輸並保障了在即將來臨的嚴冬中維持部隊和裝備所需的物資。與此同時，印度特種部隊最近佔領了戰略性制高點，可以俯瞰中共國在班公湖南側的主要部署，這是對中共黨衛軍的又一打擊。不同於中共一向的偷偷摸摸，這一切均是在中共的鼻子底下完成的。

如果這些行動還不足以使中共國蒙羞，印度公開宣布先遣特種邊防部隊（SFF）包括藏族士兵，印度方面為一名在這次行動中犧牲的藏族士兵舉行了隆重的軍事葬禮。

位於中印兩國之間的西藏在1951年被中共政府吞併，（譯者評論：在此後的幾十年裡，中共政府不僅違背對西藏人民高度自治的承諾，相反進行了數次大規模的血腥鎮壓），因此西藏人民一直把中共國視為一個殘暴的佔領國，那些渴望自由西藏人踴躍加入SFF部隊與中共作戰。印度這次行動的信息非常明確：中共對西藏的主張並沒有像他們聲稱的那樣堅不可摧。

事實證明，中共任何企圖控制不屬於它們的領土的努力都是事與願違，正如中共在南中國海的行動所表明的那樣。中共偏愛不對稱戰或超限戰，它將常規戰與非常規的心理戰，媒體信息戰，法律戰和外交戰相結合。雖然中共不費一槍一彈以超限戰改變了南中國海的地緣政治版圖，但顯然他的這一企圖在中印喜馬拉雅邊界地區行不通。相反中共的做法將對地區穩定至關重要的中印關係置於危險境地。中共不想退縮也不想發動公開戰爭，因為開戰也不可能取得決定性勝利而挽回其顏面。

中共可能擁有世界上最龐大的現役軍人，但印度軍隊也很龐大。更重要的是，印度的頑強軍隊具有在高海拔地區進行了低強度衝突方面的經驗；相比之下，中共黨衛軍自1979年災難性入侵越南以後就再沒有戰鬥經驗。鑑於此，喜馬拉雅山脈的中印戰爭可能會陷入僵局，雙方都將遭受重大損失。

中共以為能輕易搞垮印度。由於日益嚴重的中共病毒危機導致印度經濟出現了有史以來最嚴重的萎縮，中共迫使印度將越來越多的資源轉移到國防領域。同時中共巴鐵盟友違反停火協定的挑釁行為也升級至歷史新高，這導致印度雙面受敵。正如一些中共國軍事分析家所建議的那樣，中共的如意算盤是利用美國總統大選期間可能不會尋求發動戰爭的情況下對印度進行速戰速決的局部打擊。

但是，印度似乎具有抵抗中共各種壓力的能力，經濟萎縮的可能性比中共預期的要低。中共恣意挑起的這場喜馬拉雅山脈的遭遇戰，反而喚醒了對手的高度警覺及強力反擊，由此必將把中共自己推向絕境。
