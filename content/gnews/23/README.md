据《国家脉搏》报道，莉兹·切尼议员的丈夫的律师事务所代表几个与中共有联系的客户–包括与中共政权军队有联系的公司–并雇用党的前官员来巩固其大规模的中共国业务。

菲利普·佩里是众议院共和党会议主席的丈夫，他似乎因对自己政党及其领导人唐纳德·川普的古怪批评而下台。他担任“莱瑟姆和沃特金斯”律师事务所的合伙人，该公司在上海和北京设有办事处，与美国国务院标记为中国共产党“工具”的公司合作，是解放军的合作者。

佩里曾在乔治·W.布什政府中任职。他成为政府官员,游说团体和法律公司之间的“旋转门”

“旋转门”一词意味着人们进出政府以获得金钱收益。如果人们进入政府的原因是为了为国家服务，将其描述为“旋转门”是不合适的。佩里之前声称。

**共产主义的客户**

当佩里声称进入政府是为了为国家服务，但他的同事们正在为一个充满敌意的外国需求而服务。

作为其“大中国区业务”的一部分，“莱瑟姆和沃特金斯”称自己为与中共有关联的科技公司腾讯投资一家美国公司提供“咨询”。

国务院国际安全与防扩散局将腾讯描述为“中国政府的工具”，并指出，如果（中共）官员们决定寻求他们的援助，该公司“没有真正的能力向中共说‘不’。”国务院报告补充说，腾讯为中共大肆推动 “符合其威权主义模式的世界”运动，提供了“技术的监视和社会控制的基础”。

就连左翼的国际特赦组织，也将腾讯的数据加密能力评为零分（满分100分）。指出

腾讯没有“公开表示他们不会批准政府的后门请求。”
