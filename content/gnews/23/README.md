## **报道：在英国顶尖大学的“中共国间谍”有望在数周内被逮捕**

[!\[\]()!\[\](https://spark.adobe.com/page/mFvjX67Fg7YJM/images/93229417-4562-4335-a6b0-2112f85d0d92.png?asset_id=d138187d-a73d-4be9-b025-c48f90c9d5b9&amp;img_etag=%2273bb2ec84b5888c68f5d66681218582f%22&amp;size=2560)](https://spark.adobe.com/page/mFvjX67Fg7YJM/images/93229417-4562-4335-a6b0-2112f85d0d92.png?asset_id=d138187d-a73d-4be9-b025-c48f90c9d5b9&amp;img_etag=%2273bb2ec84b5888c68f5d66681218582f%22&amp;size=1024)

**在英国政府高层对英国学术界进行的一个调查中，预计将会有中共国“间谍”被捕，据信这些“间谍”已将军事技术秘密传递给了北京的共产主义政权。**

据报道，外交部、特别处和英国税务与海关总署（HMRC）的调查人员草拟了一份英国学者名单，这些学者被认为向中国共产党提供了机密技术的信息，而中国共产党利用这些技术来迫害少数民族和政治异议人士。

据《星期日邮报》周日报道，该调查预计会导致许多学术人员在数周之内被捕。

报告继续揭示，调查人员已经在那些高度依赖中共国学生学费的英国大学，和学术人员所做的可疑技术转让之间“找到了关联信息”。

根据《星期日邮报》的调查，受调查的主要大学包括26%的收入来自中共国学生的曼彻斯特大学和伦敦帝国理工学院、28%的收入来自中共国学生的利物浦大学和谢菲尔德大学、以及约10%的收入来自中共国学生的牛津大学和剑桥大学。

该文件确实指出，目前还没有迹象表明大学本身卷入了任何不当行为，或者没有意识到这些学者被指控的非法行径。

据信，这些未被列出名字的英国学者可能违反了2008年《出口管制令》，违反该命令的最高刑罚为监禁十年。

据报道，2月，英国最大的外国情报机构MI6开始调查一些“该国一些最负盛名的大学”是否在与中共国武器研发机构的联合项目中违反了国家安全法。

该机构随后揭露，据称有200名英国学者因与共产主义政权共享技术而受到调查。

智库奇维塔斯（Civitas）本月发表了一份名为《武装中共国？中共国军事综合体及其在英国大学中的科学研究潜能》的报告，该报告详细地介绍了帝国理工学院与中共国军方的联系，并称英国前24名大学中有14所与中共国军事武器研发企业有联系。

伦敦帝国理工学院去年与和中共有关联的科技巨头华为签署了500万英镑的协议。据透露，该学院至少拥有四个“由中共国主要武器供应商赞助的研究中心”，其中进行的研究还包括了为中共国空军下一代隐身技术的开发。

奇维塔斯的报告还声称，剑桥大学已经与(中国)国防科学技术大学合作，(中国)国防科学技术大学是与中共国军方有联系的机构，已受到了美国的制裁。

剑桥大学声称与中共国的合作是重要的，他说：“该大学的所有研究都受道德规范和出口管制法规的约束。”

曼彻斯特大学被指控“通过英国纳税人资助的研究中心向中共国主要核导弹集团”提供帮助。

在公众的强烈反对下，曼彻斯特大学取消了与中国电子科技公司的协议，此前有消息称该军事公司向新疆集中营地区提供了技术。

该大学声称尚未意识到该公司与新疆穆斯林维吾尔族的种族灭绝之间的联系。

奇维塔斯报告的作者拉多米尔•泰勒科特（Radomir Tylecote）说：“英国纳税人可能正在无意中为帮助中共国军方迅速获得潜在统治地位的研究买单，”他补充说：“这在战略上是不一致的，尤其是当英国自身军事科研经费还是如此的贫乏。”
