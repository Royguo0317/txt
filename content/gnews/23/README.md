## 特斯拉暂停使用比特币支付

2021年5月12日，特斯拉CEO埃隆·马斯克在社交媒体上表示，出于比特币开采大量耗能可能会给环境带来负担的担忧，特斯拉已经暂停使用比特币购买特斯拉汽车。消息传出后，比特币大跌，一度跌破5万美元，COINBASE交易所显示当天数字货币整体下跌12.4%。为何在此时此刻马斯克宣布暂停使用比特币购买汽车，该事件是否与中共有关，本文进行重点讨论。

记得今年2月特斯拉曾宣布购入价值15亿美元的比特币。马斯克也在推特上表示，公司不会出售任何比特币。那马斯克为何暂停使用比特币购买特斯拉汽车？他解释，“比特币开采和交易中需要大量化石燃料，尤其是煤炭，这给环境带来承重负担”。

谈到这里，让我们先了解下比特币是如何开采的，为何与煤炭有关？本文先介绍下比特币的开采过程。由于挖矿中需要进行特定的算法，因此需要有专业的挖矿[芯片](https://baike.baidu.com/item/%E8%8A%AF%E7%89%87/32249)，而[芯片](https://baike.baidu.com/item/%E8%8A%AF%E7%89%87/32249)的耗电量极大，有报道过，几年前开采1枚比特币的成本就达到近2000美元。中共西部拥有丰富的煤炭能源，集中分布在内蒙古、陕西、宁夏、新疆等省区。因此很多挖矿厂都开设在那里，以将挖矿成本最低化。七哥在盖特中也提到，中共国的军方是拥有比特币的大庄家。

比特币大跌对中共的杀伤力最大。那中共近期什么举动引起马斯克对比特币360度的大转弯。还记得近期中共自演自导的特斯拉汽车刹车门、自燃门嘛？对，就是中共的逼宫，要求特斯拉交出核心行车数据、打压特斯拉在中共国的市场、特斯拉股票近期大跌等事件。这些事件促使马斯克以一己之力封杀比特币，以反击中共！还有一点信息可以印证这一点，七哥盖特中说，苹果公司正在严肃讨论将生产链撤出中共，原因与中共国对特斯拉的打压有关。
