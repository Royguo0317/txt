迟浩田曾亲口对郭先生说，入党才可以提干，才可以得到更多资源，从而形成特权阶层，共产党就靠这个控制了社会的精英，把社会精英与共产党这架战车捆绑在一起，让国家力量100%控制在共产党手里，社会财富和资源也100%控制在共产党手里，有了这两点，中国任何人都不会造反。在这种模式下，在中国只有一个选择——与共产党为伍，但是与共产党为伍，不一定与共产党一起作恶，这是另一个概念。郭先生回忆参与审判他的法官以及在青丰看守所的许多警察，都是有人性有良知的人。中共有9000万党员，连同家属，大概有3亿多人，这3亿多人，都应该被惩罚吗？如果这3亿多人都被杀光，中国还存在吗？中共要的就是这个结果，它绑架了全中国的精英，绑架了对这个国家民族有贡献有能力的人，你想灭掉中共，就先灭了这个族群这个国家，这是共产党种下的毒。
