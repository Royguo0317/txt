## 因恐惧疫情，德国不得不第二次取消慕尼黑啤酒节
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1168307/)

- 作者：云复

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/03160235/bar-406884_1920.jpg)
由于担心中共病毒的蔓延，巴伐利亚州官员连续第二年取消了慕尼黑啤酒节的庆祝活动，该官员称在全球大流行期间举办庆祝活动有太多风险，而这些庆祝活动吸引来自世界各地的游客。

巴伐利亚州州长马库斯·索德（Markus Soeder）表示，他们决定以“沉重的心情”决定取消该州举世闻名的啤酒节，但是由于中共冠状病毒的数量仍然居高不下，而且德国医院已经在为此苦苦挣扎，所以必须这样做。

他保证说：“慕尼黑啤酒节将再次举行，将再次伟大。”

德国正处在中共冠状病毒封锁之中，其中包括禁止大型聚会，每10万居民每周新增146.9例新感染。

据该州疾病控制中心罗伯特·科赫研究所（Robert Koch Institute）称，巴伐利亚州略低于全国平均水平，每周每10万人有145.4例新感染。

慕尼黑啤酒节通常定于9月18日至10月3日举行，吸引来自世界各地的600万游客。

Soede说，到处都是硕大的帐篷和大量酒精的消耗，再加上可能从境内外带入这种病毒的变异变种，会使得继续这个节庆特别的危险。

去年慕尼黑啤酒节取消后，德国南部城市约有50个啤酒棚和其他场所在严格的指导下举办了小型聚会。迪特·雷特（Dieter Reiter）市长说，希望是，在一定的限制下，露天啤酒花园和露台将再次开放是有可能的。

他说，尽管取消慕尼黑啤酒节的决定非常困难，但如果城市等待太久并在准备工作开始后不得不取消，那将变得更糟。

他说：“对我个人而言，这不是一个容易做出的决定，因为对于市长来说，这是一个重要的日子。” “更重要的是，这对全世界数百万的粉丝来说是一个莫大的耻辱。”

根据德国上个月通过的国家“紧急刹车”立法，措施限制了个人接触，关闭了休闲和体育设施，以及关闭或限制进入许多商店的通道，这些地区连续三天每10万居民每周有100例以上的新病例。限制还包括了晚上10点至凌晨5点的宵禁。
