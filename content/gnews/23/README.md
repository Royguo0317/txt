![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/08134856/image-53.png)图片来源： [blazingcatfur](https://www.blazingcatfur.ca/2020/05/17/goldstein-taiwan-unlike-canada-has-defeated-covid-19/)
據加拿大多倫多當地媒體10月7日報導，Wei Wei 是多倫多地區的一位房地產開發商。 Wei Wei作為中國政府的代表曾在籌款活動期間兩次與特魯多會面，現在他被指控經營一家非法賭場。 7月份，約克區警方搗毀了一座被用於非法賭博的2萬平方英尺豪宅，據稱這是由身為中共代理人的Wei Wei 擁有。他曾經在2016年自由黨的籌款活動期間兩次與特魯多總理會面。約克警方在一份新聞稿中說，警方已提出指控，並扣押了價值1000萬加元的財產，其中包括”11支槍和100多萬美元的現金”。據National Post報導，在Wei Wei 的臥室裡還繳獲了一支突擊步槍。

眾所周知，Wei Wei 曾多次與特魯多總理會面，包括在2016年備受爭議的有多位華裔名流參與的籌款活動。特魯多曾經表示這是為了吸引來自中國的投資。據當時報導，每位參加活動的客人被收取1，500元費用，這引發了對自由黨的指責，自由黨被指責在兜售與首相接觸的機會。客人中就有一位來自中國的張姓大亨。張也是中國文化產業協會會長，負責提升”中國文化軟實力”。

Wei Wei 還參加了同年與特魯多會晤的中共四人代表團。會面活動在中國商會主席Benson Wong的家中舉行，單單是使用的餐具價格就高達1500加元。幾週後，張姓大亨和另一位四人代表團的成員向特魯多基金會捐贈了100萬美元，一部分用於資助蒙特利爾大學法學院的學生， 另一部分是用來為已故的皮埃爾·特魯多樹立雕像。就在不久前， Wei Wei還作為中共的代表頻繁出現在高端的商業和政治場合。

警方表示，Wei Wei涉嫌開設非法賭場只是對其有組織犯罪的大規模調查的一部分。 “雖然這次查獲的贓物總額巨大，但這還只是開始，” 警長Jim MacSween在一份新聞稿中說。 “通過地下賭場獲得的巨額資金用於從事其他犯罪活動， 包括賣淫和販毒等。高額的賭注還會導致槍支暴力、武裝搶劫、綁架、敲詐勒索和其他嚴重的暴力犯罪。” 我們將繼續以約克地區的有組織犯罪為目標，利用我們掌握的一切工具，包括財務審查，確保追究這些犯罪分子的責任。

這次行動中共有45人被捕，其中33人被控多達70多項刑事犯罪。調查仍在進行中。在45人中，Wei Wei、他的妻子和25歲的女兒被控經營非法賭場和其他罪行。消息人士還告訴《National Post》說，在這間所謂的”麥肯齊5號俱樂部”曾經舉辦了一場”只有受邀請者才能參加的雞尾酒會”，包括當地政界人士在內的數百人參加了酒會。 Wei Wei曾公開的發放印刷精美的請柬， 因此這並不是什麼秘密。

在最近的國會質詢中， 保守黨議員質疑道， 兩位涉及這家非法賭場的人士和另外一名BC省人士都曾經支持過香港黑警對港人的鎮壓， 而他們給自由黨捐贈了數以百萬計的經費。這如何讓加拿大人相信自由黨政府在與中共打交道的過程中是清白的呢？特魯多則回應：“自由黨對籌款活動有嚴格而透明的規定， 並允許媒體跟踪報導”， 但他並沒有就賭場事件以及涉及到的自由黨捐款者直接回應， 而是強調目前應給更關注疫情。
