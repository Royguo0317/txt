10月11日，《澎湃新聞》消息，教育部近日就「關於盡快全面實施殘疾學生15年免費教育」的建議答復稱，當前我國面臨經濟下行壓力、財政收支矛盾突出等新形勢，不宜從中央層面制定貧困家庭幼兒、殘疾幼兒免費入讀公辦園政策。

也就是說，在國家財政層面殘疾幼兒免費入讀公辦園的提議被中共否掉了。

根據全國第二次殘疾人抽樣報告調查（2007年），全國0—6歲殘疾兒童總數139.5萬。如果按每人5000元入園就讀費用，每年這些費用也僅僅70億元，相對於習近平對非洲600億美元的支持實在顯得微不足道。

大概在2008年，聯合國人權調查員托馬謝夫斯基對中國在教育領域進行了近二個星期的考察。調查的結果顯示，中國幾乎是世界上對公共教育投入最少的國家，其教育投入只佔其國內生產總值的2%，遠遠低於聯合國教育投入6%的標準。

根據《中國統計年鑒2009》及國家統計局公佈的數據顯示，2000年到2009年10年間，國家財政性教育經費支出10年累計「欠賬」已達16843億元。

回看前段時間貴州大方縣拖欠超4.7億元教師工資的事件也就不足為奇了（9月6日《央視新聞》消息）

中共對國內的孩子教育如此「吝嗇」，但是對其它國家的教育卻大方的很。

《環球網》2016年10月24日消息，中國將在2017年向菲律賓提供160億美元援助，菲政府將優先照顧教育、農業及衛生方面（作為世界第二大經濟體，中國醫療衛生投入佔GDP比重在183個國家和地區中排倒數28位）。

來自中國兒童少年基金會數據顯示，中國有近300萬貧困兒童上不起學，而300萬這些孩子1年的基礎教育經費只需投入10多億元。

這大概就是中共謀求的」大國風範「。
