## 《路德社》精选新闻简讯：拜登6月起首度出访 出席G7与欧美峰会
` 路德社精选` [轉載自GNews](https://gnews.org/zh-hans/1133954/)

出品：《路德社》 编辑：《路德社》精选新闻专栏组

![]()![](https://gnews.org/wp-content/uploads/2021/04/042405.png)

白宫周五宣布，美国总统拜登6月中起开启上任后首度出访外国之行，将前往英国、比利时，参加G7与北约峰会，与欧盟领导人会面。与前任特朗普首度出访选择沙特阿拉伯不同，拜登上任伊始，就显出要重振大西洋两岸关系的愿望以及恢复欧洲盟国对美国信任的决心。白宫宣布，拜登此行将参加在英国西南部6月11-13日举行的G7峰会，将与英国首相约翰逊以及多国领袖会晤。接下来拜登将前往布鲁塞尔参加北约峰会，确认美国对北约、跨大西洋安全与集体防御承诺。在布鲁塞尔，拜登将与欧盟领袖米歇尔以及欧盟委员会主席冯德莱恩一道出席欧盟-美国峰会，凸显基于共同利益与价值观的跨大西洋伙伴关系稳如磐石。

欧洲已经在开始转向，与美国站在一起。共同的价值观让双方彼此站在一起共同应对中共的威胁。中共利用所谓的“一带一路”要控制中东，控制能源，制造地区的不稳定来转移视线的目的已经失败。
