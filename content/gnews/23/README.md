2月间，中共国海关总署指，多次从台湾输入中共国的凤梨中截获“检疫性有害生物”，单方面宣布自3月1日起暂停对台湾凤梨的进口。传言说是中共没钱了、台湾和美国走得太近……，无论怎么说，都不会是中共所声称的原因。

![]()![](https://gnews.org/wp-content/uploads/2021/03/WhatsApp-Image-2021-03-03-at-10.45.27.jpeg)图片来源：美国之音中文网

台湾人民及政府面对此事件，也以全民买凤梨之举，展示了团结一致的行动。与此同时，也督促了台湾政府，面对即将倒塌的中共经济，必须再加紧脚步，降低对出口中共国的依赖性。

由于中共一向霸道专政、好施淫威，耍各种政治流氓手段，找各种借口和理由威逼利诱世界各国就范，背后沾沾自喜。这次的禁止凤梨进口行动，无论什么原因，台湾凤梨都成了中共对内对外政治斗争手段的又一个牺牲品。中共如果有一丁点儿爱民、照顾之心，就不会对层出不穷的金融诈骗事件毫无改革究责之心，坐视不理，也不会满世界故意投放生化病毒，不会防民之口甚于防川、大费财力人力建防火墙，更不会不守承诺、背信中英香港条约，残害杀戮香港百姓少年…… 。在此，身为一位台湾人，我为我们台湾人民及政府感到骄傲，因为我们心中有爱、有尊严。
