## 中共綁架康明凱、邁克爾和中糧拒買加拿大大豆能不能換回孟晚舟？
`2020-07-16 08:19 Diago` [轉載自GNews](https://gnews.org/zh-hant/266664/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/16081403/%E5%B0%81%E9%9D%A2-23.jpg)
作者：Diago

英雄科學家閆麗夢拯救全人類的壯舉讓人淚目，我相信中共再怎麼想滅她的口，都已經是鞭長莫及了，在為英雄科學家自豪和驕傲的同時，也衷心祝願英雄科學家一切安好！在推特上看到相關內容，英雄科學家的姓本來就是一個門裡邊有個三，莫非就是冥冥之中，上天派下來開啟滅共第三道大門的天使？

對於閆博士中共已經鞭長莫及，對於正在加拿大等待引渡美國的孟晚舟則應該是中共另一塊最大的一塊心病，據環球網2020年6月19日21:25發佈的[加拿大人康明凱、邁克爾是否有可能被判死刑？趙立堅回應](https://baijiahao.baidu.com/s?id=1669933961142840439&amp;wfr=spider&amp;for=pc)——【關於康明凱案，經依法審查，2020年6月19日，北京市人民檢察院第二分院以為境外刺探國家秘密、情報罪，將加拿大籍被告人康明凱起訴至北京市第二中級人民法院。

起訴書指控被告人康明凱為境外組織刺探我國家秘密、情報，情節特別嚴重，其行為觸犯了《中華人民共和國刑法》第一百一十一條之規定，犯罪事實清楚，證據確實、充分，應當以為境外刺探國家秘密、情報罪追究其刑事責任。

關於邁克爾案，經依法審查，6月19日，遼寧省丹東市人民檢察院以為境外刺探、非法提供國家秘密罪，將加拿大籍被告人邁克爾·斯帕弗起訴至丹東市中級人民法院。

起訴書指控被告人邁克爾·斯帕弗在華期間為境外刺探、非法提供我國家秘密，情節特別嚴重，其行為觸犯了《中華人民共和國刑法》第一百一十一條之規定，犯罪事實清楚，證據確實、充分，應當以為境外刺探、非法提供國家秘密罪追究其刑事責任。】

這件事與中共勢欲奪回孟晚舟有沒有關係？明面上看不到，但是我看到了中共媒體上鋪天蓋地轉發的另一條消息——

搜狐網2020年6月29日10:14轉自觀察者網[康明凱妻子再發聲：對特魯多拒絕釋放孟晚舟感到失望](https://www.baidu.com/link?url=Nn5mw_l0TocblMsiTDTeMtEahGk-Bxv25sIAf-QZuF9KyztdzWS_GLmikM4enhu2XGjDFufmlwRTUA3UfIdKQMOOapM6XZhYkUZMH2qKoBq&amp;wd=&amp;eqid=efedf8d000030b7d000000065f103084)：【6月19日，兩名加拿大公民康明凱和邁克爾，因涉嫌為境外刺探國家秘密罪在中國被提起公訴。相比加拿大警方在美國施壓下無端拘捕孟晚舟，康明凱和邁克爾的犯罪事實清楚，證據確鑿。這兩個案件性質不同，卻被部分西方媒體、政客相提並論。6月25日，加拿大19名前政要還聯名致信特魯多，要求釋放孟晚舟女士，遭總理特魯多一口回絕。6月28日，康明凱的妻子納吉布拉，在首次公開露面不久後，又一次接受加媒採訪，對特魯多拒絕釋放孟晚舟感到失望。值得一提，納吉布拉指出，加拿大在逮捕孟晚舟的決定上存在公平方面的問題，“我們不應該屈服來自美國的欺淩行為”。加拿大CTV新聞6月28日報導，納吉布拉在當天的採訪報導中指出，特魯多已強烈否決19名前政所提訴求，拒絕釋放孟晚舟女士，也拒絕考慮以此換回因涉嫌從事危害中國國家安全活動在中國被捕的康明凱和邁克爾。】

這些事背後有沒有中共操作的影子？肯定有！那些致信的政要有沒有被中共藍金黃？在此我們不妄下結論，但是從這些桌子底下的較量，可以看出中共為了奪回孟晚舟已經無所不用其極，在綁架並以死刑威脅兩名加拿大公民康明凱和邁克爾無果後，最近中共又來了一招——

據網易2020年7月16日15:04發佈的[中糧集團霸氣警告加拿大，停止進口大豆，直到釋放孟晚舟再考慮](https://3g.163.com/news/article/FHLRD3R70511V6CK.html)：【近日，中糧集團出面支持華為，其中中糧集團的高管戰略總監肖陽直接表示：“考慮到目前加拿大依舊無理由的扣押孟晚舟，中糧集團將停止進口加拿大的大豆產品，通過進口其他國家的大豆產品直接替代加拿大的大豆產品”。中糧集團的舉動也會隨著加拿大的態度而改變，加拿大什麼時候釋放孟晚舟，中糧集團才會再次考慮進口加拿大的大豆產品，也是很明確的告訴加拿大儘早釋放孟晚舟，不然吃虧的都是加拿大。】

這一次是中企出面了，中糧集團這一次的出面透露了兩個資訊：

1、中糧集團與作為中共軍方企業的華為是一夥兒的，雖然中企都是黨企在爆料革命的揭露之下已經鐵板釘釘，但這一次中糧集團的表演更為這個事實添加了新證據；

2、中共為奪回孟晚舟已經紅了眼，就差派中共党衛軍去把孟晚舟綁回來了；

可是中共這幫以馬列主義武裝起來的流寇，即使竊據政權七十載，依然不懂什麼是法治，什麼是三權分立，當然至於什麼是人權和道義這樣的詞彙就更是這幫流寇所無法理解的。

當一個政權妄圖用下三濫的手段通過偷、騙、搶建立世界霸權的時候，它就變成了世界公敵；

當一個政權用製造並在全球播灑生化武器試圖打垮全人類建立世界霸權的時候，它就變成了宇宙公敵；

因為強權壓不垮正義，理性須臾不能缺席。在這個人神共憤的宇宙公敵即將走向滅亡的時候，那些以法律的名義綁架他國人質的下三濫，那些以金錢的誘惑試圖讓他國跪下的下三濫，除了徒增笑料之外，唯一的作用就在於這個政權每多存在一秒都會萬分增強人類對它的厭惡和必須儘早滅掉它的意志。

中共怕自己死得不夠快，現在都已經自己給自己的棺材上釘釘子了。
