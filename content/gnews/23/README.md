国家安全顾问杰克·沙利文声明

2021年2月13日 声明发布

世界卫生组织（WHO）的使命从未像现在这样重要，我们对其专家和他们每天为防治CCP病毒和促进全球卫生和健康安全所做的工作深表敬意。这就是为什么拜登总统拒绝并推翻了川普政府脱离世卫组织的决定。但重新参与世卫组织，也意味着要以最高标准来要求它。在这个关键时刻，保护世卫组织的信誉是重中之重。我们对通报CCP病毒调查早期结果的方式深感关切，并对得出这些结果的过程提出质疑。这份报告必须是独立的，专家的调查结果不受中共国政府干预或改变。为了更好地了解这次疫情并为下一次疫情做好准备，中共必须提供疫情爆发初期的数据。展望未来，包括中共国在内的所有国家都应参与到预防和应对突发卫生事件的透明和有力的过程中来，世界才能尽快掌握到尽可能多的信息。
