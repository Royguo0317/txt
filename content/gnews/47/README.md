# **预计拜登将禁止使用“中国病毒”一词，以打击与新冠病毒有关的种族主义**

**这些措施试图使联邦机构避免川普政府时期的敏感言论**

[!\[\]()!\[\](https://spark.adobe.com/page/nQbALWqsEgbii/images/988ce92a-2684-4078-82e1-0d34c42e66fa.jpg?asset_id=1a06e335-2b21-4411-a71b-9e4e11605a51&amp;img_etag=%228f8c605b7a3e3905a660fcb27eb1e014%22&amp;size=2560)](https://spark.adobe.com/page/nQbALWqsEgbii/images/988ce92a-2684-4078-82e1-0d34c42e66fa.jpg?asset_id=1a06e335-2b21-4411-a71b-9e4e11605a51&amp;img_etag=%228f8c605b7a3e3905a660fcb27eb1e014%22&amp;size=1024)

乔·拜登（Joe Biden）有望采取行政行动，以消除因新冠大流行而针对亚裔美国人的种族主义和仇外心理。

哥伦比亚广播公司新闻报道称，据熟悉该计划的人士表示，这些指令既可以是行政命令，也可以是总统备忘录，并将成为注重平等的一系列措施的一部分。

预计将采取行动指导联邦机构检查是否有“中国病毒”之类的仇外说法出现在现有政策、指令或政府网站上。

唐纳德·川普（Donald Trump）反复使用诸如“中国病毒”和“功夫病毒”之类的敏感性词语提及新冠病毒。

自从新冠病毒大流行以来，针对亚裔美国人的骚扰和暴力事件已大大增加。

预计将要求司法部门从更好的数据收集开始，采取措施解决举报事件中的这种上升趋势。

即使拜登政府采取的是象征性行动，也将被视为使联邦政府远离前总统言论的积极举措。

（拜登政府）还将要求卫生和公共服务部开展有关新冠病毒的语言和文化培训，以及有关消除种族间误解的更广泛指导。

哥伦比亚广播公司报道，预计还将宣布其他与部落政府、公平住房和私营监狱有关的平等行动。此外还正在准备有关投票权的其他措施。

本周将看到为实现一系列竞选承诺而推出的一套执行行动计划。

周一，我们看到总统签署了命令，推翻了对跨性别军事人员的禁令，并承诺联邦采购美国制造的商品。

根据哥伦比亚广播公司看到的文件，周二的重点是种族平等；周三将看到针对气候变化的行动；周四将重点关注医疗保健；周五将以移民措施为中心。
