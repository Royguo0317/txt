![]()![](https://gnews.org/wp-content/uploads/2021/04/五月花评1大学.jpg)
4月22日，CCP人民网发表题为《一流大学建设要坚持社会主义办学方向》的文章。文中开篇写道：“教育兴则国家兴，教育强则国家强。在国家发展进程中办好高等教育，办出世界一流大学，首先要在体现中国特色上下功夫。“

这句“教育兴则国家兴，教育强则国家强”，效仿1900年2月10日发表的梁启超《少年中国说》中的名句：“少年智则国智，少年富则国富，少年强则国强”。结尾落点为“体现在中国特色上下功夫”。我们是否可以认为，其开篇就在教育国民尽学抄袭之能，此即为中国特色？

第二段内容为4月19日习近平到清华考察时讲话，言语间要办一流大学，要培养一流人才。请问世界一流大学的标准是什么？中国特色社会主义就是一流大学的标准吗？

文中接下来谈到清华大学的诞生，原文说：“清华大学诞生于国家和民族危难之际，成长于国家和民族奋进之中，发展于国家和民族振兴之时”。而百度百科上说：“学校前身清华学堂始建于1911年，校名‘清华’源于校址‘清华园’地名，是清政府设立的留美预备学校，其建校的资金源于1908年美国退还的部分庚子赔款。1912年更名为清华学校。1928年更名为国立清华大学。“

清华是CCP口中万恶的美帝国主义提供资金所建，清华的学生是清政府准备送到美国留学的预备生。瞬间一地鸡毛！

大家熟知的清华校训：“自强不息、厚德载物”，是1914年11月，梁启超在清华演讲时，引用《易经》中的两个卦象描述而来。为何信仰马克思主义无神论者的CCP却仍以此为校训，延续至今？

文中还提到：“马克思主义是我们立党立国的根本指导思想，也是我国大学最鲜亮的底色。”那么意味着大学里设置的各类学科，文史哲经、理工农医，都要刷上红颜色，都只能在政治禁锢之下，开展教育教学。就像让骏马带上龙套，只能在磨坊里转圈拉磨一样。CCP的大学不是在培养千里马，是在培养听话驴！

文章结尾喊出：“朝着中国特色世界一流大学的方向大跨步前进”，尽显大跃进时代气息和无脑自信。

在独裁统治下，墙国的各个领域都被CCP进行洗脑和渗透，大学也不例外。言论和思想自由，是科学创新的土壤，只有推翻CCP，中国才有可能建设真正的世界一流大学。
