《每日电讯报》（Daily Wire）1月12 日报道，世界首富兼特斯拉和Space X首席执行官埃隆·马斯克（Elon Musk）在推特 (Twitter) 上回应一个讽刺大技术审查的故事时说：“许多人对西海岸高科技充当自由言论仲裁人的事实超级不满意。”

马斯克的推文是回应近期发生的社交媒体事件，包括亚马逊（Amazon），苹果（Apple）和谷歌（Google）在内的平台删除了Parler，并清除了许多右翼用户社交媒体平台账号。

一段时间以来，马斯克提出了一些保守的观点。在2020年5月，他表示自己对当时的家乡加利福尼亚州的左派统治感到厌倦，显然他在推特上敦促追随者加入美国的保守派，同时他在推特上发推文简洁地说：“服用红色药丸”，暗示他们不要在支持左派。

2020年7月，他发推用德文抨击了共产主义之父卡尔·马克思（Karl Marx）：“简而言之，达斯·卡皮塔尔（Das Kapital）” ，德语意思就是：“免费给我放屁”。
![]()![](https://gnews.org/wp-content/uploads/2021/01/屏幕截图411.png)
2021年1月7日，脸书（Facebook）宣布将无限期禁止川普总统的帐户。 推特借口“有进一步煽动暴力的危险”，在1月8日无限期禁止了川普总统的帐户，然后在1月9日永久禁止了他的帐户。

推特同时也删除了很多川普总统的支持者的账户。就在人们陆续将自己的社交平台转移到Parler的时候，苹果和谷歌周六从其应用商店中删除了Parler应用；周日晚上，亚马逊从其云托管服务 (Amazon Web Services) 中删除了Parler。

路透社报道：“据亚马逊网络服务团队（Amazon Web Services）（AWS）给Parler的电子邮件，因Parler未能有效处理暴力内容的增加而违反了AWS的服务条款，暂停了Parler对其服务器的使用权。路透社亚马逊发言人证实这封信是真实的。”

周一，Parler向亚马逊提起诉讼，该诉讼首先断言：“上个月，被告Amazon Web Services，Inc.（’AWS’）与流行的社交媒体平台推特签署了一项多年协议，以便AWS可以支持每日交付数百万条推文。 AWS目前向Parler提供了相同的服务，Parler是保守的微博替代产品，也是推特的竞争对手。”

该诉讼指出：“不过，周五晚上，推特上最受欢迎的推文之一是“吊死麥克彭斯”（’Hang Mike Pence’）。但是，AWS没有计划，也没有威胁要暂停推特的帐户。”

该诉讼声称“ AWS有效终止Parler帐户的决定显然是出于政治上的动机。它显然也旨在减少微博服务市场的竞争，从而使Twitter受益。因此，AWS与被告推特一起违反了《谢尔曼反托拉斯法》第1条。 AWS还违反了与Parler的合同，该合同要求AWS在终止服务之前向Parler提供30天的通知，而不是AWS实际提供的不到30小时的通知。最后，鉴于预期在不久的将来有数百万用户注册，AWS故意干预潜在的经济优势。”
