## 【军事快讯】10/27美国国务院正式通知台湾外交部最新的一批对台军售，总价逾二十三亿美元
` 纽约香草山农场` [轉載自GNews](https://gnews.org/zh-hans/473997/)

作者：香草山写作组 雨倩 校对/审核：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/25214337/WhatsApp-Image-2020-10-24-at-08.09.25-1.jpeg)



据《新闻看点》披露：10月26日美国国防安全合作局通知国会，国务院已决定批准向台湾出售最多达到100套“鱼叉”防御系统以及相关设备，这是在7天当中第四笔对台湾的重大军售方案。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/28040357/1-247.jpg)
几天前还对台湾批准了另外的三笔对台军售案。

台湾外交部对此表示由衷感谢，并在声明中说：“这是川普总统任职期间第九度，也是今年第四度的重大军售。这充分体现美国政府对台湾国防需求的高度重视，并已逐步落实对台军售常态化机制。”

美台商业协会会长韩儒博表示：这些陆基移动岸防火力配备“鱼叉”导弹，增加了台湾现有的反监防御，并大为增强了赫阻能力。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/28040039/2-62.jpg)
美台这边的密切互动，让中共的小心脏受不了了。中共外交部表示要制裁对台湾军售的美国企业，包括洛克希德·马丁公司（Lockheed martin)、雷神公司（Raytheon）、播音公司（Boeing）等。
