日本福岛因地震引起的大海啸重创核能发电厂，造成出于意料的重大核灾事故，至今已满十年，但是一切仍无法回复正常，复原之路遥遥无期。

台湾有一群支持核能发电的人，打着核能是干净能源的口号，老想着要再重启核能发电的政策。核能发电在台湾真得有他们所说的那么好，好像没有什么风险吗？只要稍微关心台湾核能发展及了解台湾地理环境人口分布的民众都会对台湾的核能电厂从心里打起寒颤，祈祷在未停机前绝对不要出意外。

所有的发电厂都有各自的优缺点，是否采用的衡量标准就是使用以后利益要大于潜在的损害及风险。目前支持核能发电的群体只打着核能发电对空气污染的影响最少，所以要采用核能发电来减少火力发电产生的空气污染。单从此点来比较核能电厂与火力发电厂对空气品质的影响，的确核能电厂的空气污染较少。但是核能电厂本身产生的污染及风险却故意被忽视不提。

核能电厂最大的成本及问题是发电后的核废料如何处理。目前台湾没有再处理提链核废料的能力，也不被允许发展此技术，所以核废料只能找个地方做掩埋，或是送到国外作初步处理再送回台湾，最终都是要留在台湾。问题是每个人都怕核电厂及核废料在自家隔壁，谁也不愿意与核物质牵扯上亲蜜关系。最终这些核废料会成为台湾无法解决的大问题。

台湾的核电厂分别建在南北两端，有些还是建在断层带上，万一发生操作意外、大地震、或是战争引起的人为故意破坏，以台湾全长也才394公里，核电厂事故后最少要有20公里的撤退安全区，则台北一半的居民都要撤离，这还没有将核物质飘到上空在大气中扩散的范围完全计算在内。这也只是在平时的核灾意外状况下的想定状况。万一遇到战争时被故意破坏，那就是无法预测的核能灾难，台湾的人口只能都往中部撤退，完全没有安全的逃生空间。

台湾是个小岛，适于人居的空地非常有限，不像现在许多使用核电厂的大陆国家，他们的领土广大，有足够的纵深可以在核灾发生时做撤退的空间。即使是如此，德国也坚持执行既定的废除核能发电的政策。

台湾有独特的地理环境及特殊的战略考量，在中共随时有攻击台湾的风险下，每座核能电厂都像是潜藏的核弹一样威胁着台湾居民的生命安全。在衡量核灾风险优先于对空气污染的考量下，核能电厂所制造的核废料及万一核能灾变后不可收拾的后果，都必须使核能发电被排除在台湾发电厂的选项之外。
