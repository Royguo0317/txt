## 中共野蛮粗暴销毁电动代步车实际是眼红背后的利益
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1034654/)

视频&撰稿：必有回响

根据网上报道3月27日，中共相关部门在北京顺义区华新凯业物资再生有限公司对查扣的1569辆违规电动三四轮车进行了集中销毁。根据报道称从去年12月底，中共在北京就开始所谓的整治“非法”电动三轮四轮车查获了5000多辆三轮四轮

中共给电动三轮四轮定了“5宗罪”：

1. 违反道路交通信号灯通行；
2. 在道路上逆向行驶；
3. 违法进入高（快）速路及环路主路；
4. 占压消防通道违法停放；
5. 在人行道上行驶等乱行乱停重点交通违法行为。


确实在中共国不遵守交通法则是有那么一些人的存在，就算是小汽车也有不遵守交通规则的，对于汽车不遵守规则就罚款扣分，那为什么不对电动车进行一样的处理呢？而且是否所有的电动三轮四轮车都有犯了“5宗罪”，答案肯定是否定的。这正如中共夺取政权后一贯使用的帽子罪手段，一旦中共最高领袖们觉得有什么人或者事物妨碍了他们的利益，他们就会给这些人或事物编制一箩筐的罪名大肆宣传，然后利用被洗脑的民众站出来批斗，然后中共在以“为了人民的利益着想”把这些被扣帽子的人或物拿下甚至消灭，历史上土改，三反五反，文革等等皆是如此。

其实电动三轮四轮一开始就是合法生产合法销售的，电动三轮四轮甚至是电动摩托普通人道路上使用也是将近有十来年历史了，一直允许经销商卖给普通百姓了，也没有做过普法宣传说电动车上路是违法的。笔者身边也有开电动三轮或者电动摩托的朋友，他们开到城市道路上全完没有交警管。既然如此说明中共一开始是允许的，如果说电动车上路是违法的在这十来年的时间里，为什么政府基本没有监督管理呢？中共一方面默认上路不管，一方面不正式承认电动车的合法性，也不给老百姓发牌照，这正是中共给老百姓埋下的伏笔，挖的坑。

正如当年毛让知识分子给中共提意见，美其名曰“百花齐放”，殊不知当知识分子们提出的意见直接真实的指向中共邪恶的时候，毛就认为这些知识分子阻碍了中共的利益影响了中共的江山，所以起了杀心，后来的反右运动来了个一网打进。其实这个电动车的事也是类似的，当初中共允许电动车销售和道路上跑也是因为电动车可以带动一些地方的经济和就业，这个产业可以养活一批人，很多老百姓自己也能靠电动车做做小生意，所以这十来年都相安无事。

但是时过境迁现在不一样了，因为现在中共的国企也开始搞电动车产业了，众所周知中共国内的汽车企业不是国企就是打着私人名义的白手套企业，如吉利汽车实际背后的受益人就是习的家族。试想一下国内电动车市场是多么巨大的一块蛋糕，所以一切阻碍了中共吞下全部蛋糕的人都得被拿下。根据公开资料显示中共国内有2亿电动车所谓的涉及“违法驾驶”。也就是说如果中共如果拿下这2亿所谓的“违法车”，那将再创造多少亿人民币的利益。首先这些所谓的“违法车”如果被中共收缴根本不会给你补偿，就是白抢老百姓的财产。就目录被收缴的电动车一小部拿来表演给大家看是销毁了，那更多肯定是被拿去卖二手配件了，如果全国2亿车都被收缴的话那这也是上百亿人民币的财富。

还有一种情况就是2亿辆车没有全部被收缴，但是这2亿辆车就再也不能上街了，不能上街老百姓为了生活也得买车啊，那只能去买中共党企的生产的电动车了，想想这个市场价值能上千亿万亿人民币。

中共向来是朝令夕改，一切以统治集团的利益为准，各级政府吃、拿、卡、要，不消灭中共人民就没有好日子过。
