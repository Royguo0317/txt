据世界一体新闻网站（WIONEWS.COM）2月22日报道，因质疑中共官方关于去年6月15日加尔万河谷与印度军队冲突的说辞， 中共当局拘留了3名微博主。中共官媒《环球时报》称“因抹黑在加尔万河谷解放军烈士，在两名网络用户被捕后，又有位于西南的四川省的一名博客主因侮辱烈士而’认罪’，已被逮捕。”

上周五，中共承认于2020年6月，在加尔万与印度士兵的冲突中方有伤亡，宣布了为冲突中牺牲的4名士兵和受伤的1名团级指挥官颁发军功章。这些逮捕行动是在中共公开承认解放军伤亡后发生的。仇子明是被捕的三人之一，他是前《经济观察报》调查记者，他曾质疑官方对加尔万河谷冲突的说法。
![]()![](https://gnews.org/wp-content/uploads/2021/02/网络.jpg)*图片来源：网络*
他在微博上表示，伤亡人数应该高于官方报道，因为在救助伤员时，双方还有士兵死亡。子明还质疑了为何中共在8个月后才宣布伤亡情况，相比之下，印方在事发后立即就公布了伤亡官兵。子明说，“在印方看来，他们赢了，而且付出了较少的代价。”在他被捕后，南京警方发布了一则消息，称“他是因为在对待印度军方非法入侵问题上，散播虚假信息，抹黑4名牺牲的英雄和1名受伤的英雄，而被捕的”。 被捕后，仇子明的微博账号立即被暂停了。
![]()![](https://gnews.org/wp-content/uploads/2021/02/网络1.jpg)*图片来源:网络*
另一位陈姓博主也在被捕三人之列，他因在一次微信群聊中发表评论而在北京被拘留。据警方称，他们接到投诉，称一名28岁的年轻人在微信上发表侮辱解放军士兵的言论。据《环球时报》称，此番言论令群情激愤。北京警方采取了行动，并表示按照陈某的说法，发表这些言论是 “为了宣泄情绪”。

第三位是25岁的杨姓博主。党媒《环球时报》称，杨因发布 “抹黑 “中共国军队的帖子而被网民举报，在社会压力下，杨某于第二天向警方坦白。按照警方的说法，他已经被处以7天拘留。
