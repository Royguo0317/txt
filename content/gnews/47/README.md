据台湾《自由时报》4月1日报道，智利大规模接种中共国科兴疫苗后，病毒感染人数不减反增。4月1日，智利新增7830例确诊，创自去年3月疫情爆发以来的单日最高数字。智利政府被迫宣布，从4月5日开始，智利本地和外国居民将被禁止出入该国，以减缓疫情散播和防止变种病毒入侵。

![智利近來確診病例數不斷上升，政府週四（4/1）宣布將關閉邊界。（美聯社）]()![智利近來確診病例數不斷上升，政府週四（4/1）宣布將關閉邊界。（美聯社）](https://img.ltn.com.tw/Upload/news/600/2021/04/02/phpwsOe7O.jpg)图片来源：美联社

除了智利以外，土耳其、巴西等国家在接种了中共国疫苗后，也出现了病毒确诊人数不降反升的情况。值得一提的是，巴基斯坦总理、总统和国防部长，都是在接种中共国疫苗后又感染病毒的。更有甚者，在香港，因为接种中共国疫苗已经造成数人死亡。

自从有了冠状病毒疫苗以来，疫苗的安全性一直备受关注，尤其是中共国生产的疫苗。这其中有两个重要原因：第一，中共国是目前能生产疫苗的国家中唯一生产灭活疫苗的国家；第二，中共国研发疫苗的过程和数据不透明。由于这两个原因，国际社会很难对其安全性和有效性进行评估。如今，中共病毒已经造成全世界一亿多人感染，两百多万人死亡，而中共国生产的疫苗正在再次伤害世界人民。越来越多的人已经觉醒：只有消灭了CCP，病毒才能真正消失。
