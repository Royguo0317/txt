## 万恶的户口本
` 东京樱花团-Tokyo Sakura Group` [轉載自GNews](https://gnews.org/zh-hans/713542/)

HTS-02C-202012-018

作者：欲静（喜马拉雅东京樱花团）

- ![]()![](https://gnews.org/wp-content/uploads/2021/01/写真-1.jpg)


中国的户籍制度五花八门，身为中国人的我，到了后期才弄明白还有这么多种户口形式。比如说，农业户口，城市户口，集体户口，集体户口里面还有企业集体户口，学生集体户口。

看着琳琅满目的户口名称，不仅让人进入了 混乱状态，也会让人感到好奇，但这些只是表面，重点是隐藏在户籍后面的黑手，从一开始就伸向了我们中国老百姓。

接下来，我们来简单地述说户口后面的心酸与眼泪吧。

1. 黑户。在自己的国家出生的孩子不一定都是有身份的，这就是中共国。首先，如果超生了孩子，因家庭条件的关系，缴纳不起罚款的，不仅孩子没有户口，上学，就医的保障也是没有的。其次，早期的集体户口出生的孩子，因户口属性不是家庭户口，所以，也是没有户口的，同样，这些孩子也是没有任何保障的。
2. 退伍待遇。光荣参军，摸爬滚打几年退伍后，城市户口是定向分配工作的，而农村户口则什么也没有，这时就能看出户口体现的资源管控了。
3. 城市户口和农村户口的医疗，养老待遇的差距是不能用一点儿来形容的。可以说是天壤之别。
4. 跨区就学。从政策上是有义务教育的，可是因为当地经济无法满足全家生活而搬迁到其它省市地区的孩子们，上学是要交纳跨区就学费的，只有缴纳这笔款项后，才能正常上课，反之，困难家庭的孩子依然上学困难，所谓的义务教育在哪里？
5. 户口定性。从字面上解读的话，出生在城市就应该有城市户口，出生在农村就应该有农村户口，但实际上，孩子的户口属性是完全跟着父母走的，也就是强制定性，想要改变户口属性时的手续以及金钱是难以想象的。


种种因户口而衍生出的事件不计其数，也造成了无数的人间悲剧，也是中共国的国民自出生以来就不得不背负的阶级差别和一道无形的枷锁。究其根源，就是资源分配，金钱交易和人员管制。这就是违背人性，违背自由最典型的产物—-中共国的《户口》制度。
