最近一场滑稽的选举，使得很多美国人开始意识到，真相是如此的珍贵。搭着这趟寻求真相的民意顺风车，在接下来的几个月中，唯真不破，坚持发声将是一项比拼耐力和毅力的工作。在这个关键的节点上，请不要相信什么预测，不要窝在沙发里等待好事发生，但一定要相信唯有我们每个人的努力才能使那个美好的预言成真。

使用海外手机号

我们知道，注册推特等社交网站或paypal等国际支付平台都需要海外手机号。目前，使用国内网购的实体电话卡在卖家的指导下基本都可以正常使用。如果不方便自行购买礼品卡，收到电话卡后别忘了让店家帮忙充值电话费。

有些战友会担心卖家知道手机号，而我们会用这个手机号注册推特发表言论。如果推特和卖家平台交换数据进行撞库，那岂不是暴露了自己的身份。虽然尚未听说由于这个原因被喝茶的普通战友，但这个担心确实有一定道理。有这个担心的战友可以使用购买来的实体手机卡去注册虚拟手机号，然后使用虚拟手机号注册各种社交平台。

同时，需要注意实体电话卡绑定的电子邮箱、备用电话号码、地址等其他信息都由卖家持有。在电话卡激活且有足够的余额后，一定记得修改，但不要输入任何墙内使用、或与自己的真实身份有关联的信息。

我们不是通信工程的专家，但容易理解，当手机卡插到手机里的那一刻，通过基站或GPS的定位信息，手机设备的编码、手机号、基站或者GPS获取的地理位置信息就已经不是秘密了。如果不巧安装了墙内的APP，那么手机通讯录、云盘等其他信息也不再安全。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08121058/1-22.png)
因此，我鼓励无线WIFI路由器不选择国内厂商，使用单独的翻墙手机，只安装与爆料革命有关的软件。更进一步的，为了减少以上信息被窃取的风险，平时将手机SIM卡拔掉，关闭不使用的APP，并尽量使手机工作在省电模式。如果一段时间不使用手机，则将其设置为飞行模式或干脆关机。

收听节目

相信在墙内的很多战友都有类似恐怖的经历：朋友刚给我介绍一位名为上官才子的伙伴，马云和刘强东爸爸们马上就推荐殡葬用品。墙内手机的软件偷盗隐私信息已经不是新闻。他们利用手机强大的处理器，实时解析我们的言语，然后把这些数据上传。

确实有很多战友在收听节目时候会采用外放，互相交流的时候也不可避免的说起这些敏感信息。如果在某些特定的场合担心被手机程序窃听，最简单的方法是同时播放一些背景噪音。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08121116/2-33.png)
传递墙内信息

前段时间看到墙内战友发图片表达对CCP的愤怒，却无意间暴露了自己的身份，令人非常担心。请注意墙国的监控无所不在，你用摄像头拍风景，风景里的摄像头也在拍你。目前比较流行的视频解析算法中，车牌号识别是非常古老且成熟的。为了检查违章，特定车牌号的出行路径非常容易被追踪。因此，在照片中首先应该避免的，就是人脸，车牌号等这些信息，发出这些图片前，一定要仔细检查，必要的时候用绘图工具涂抹后再发布。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08121126/3-33.png)
如果还是不确定图片是否会被反向定位，还可以采用打时间差的方法：比如今天拍了一张有价值图片，存起来等到几天至几周以后再发布，并仅标注大致拍摄时间段。这样做有两个好处，一是发布之前还有机会再次检查内容，二来经过一段时间后，公安或交通系统已经积累了大量的监控数据，回去找就像大海捞针非常不容易，反过来增加了自己的安全系数。

规避媒体审查

现在的推特越来越像微博。相信很多西方的小伙伴对于如何对抗言论审查没有经验。这一点反而成为了我们墙国人民的一大优势。有关中文内容，可以使用火星文。我们在战友之家可以直接生成火星文，非常方便。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08121141/4-11.png)
当然，英文也是妥妥的好使。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08121136/5-1-1.png)
说到图片，常用的处理手法就是把图片下载下来，稍稍在图片上用画笔划拉两下再发布。此方法虽然简单，但在实践中是性价比最高的。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08121240/6-12.png)
另外，对一些后台自动识别文字的软件还可以将文字图片扭曲，使得后台文字识别程序不能正常工作：
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08121417/%E5%9B%BE%E7%89%873-6.jpg)
对于一些图片，我们可以对一些部分进行模糊或者马赛克处理，避免违法某些法律。

有的时候，我们想让图片以原貌发布而不进行遮盖，同时尽量避免程序自动识别面部等关键特征，那么可以采取添加噪声以及用荧光笔在关键部位划拉几笔的方法：
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08121439/8-5.png)
从上到下越说越复杂。大部分情况下，最简单的方法就很有效。如果面对比较严格的审查，那么需要叠加使用更麻烦一点的手段。

最后，如果审查者不得已采取大量人工审查，那么就标志着媒体大战进入了最后的肉搏阶段：我们快赢了。
