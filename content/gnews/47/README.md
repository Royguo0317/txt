**新闻1 ****： ****【疫苗】中共国疫苗在东南亚遭遇信任危机**

菲律宾是东南亚中共病毒疫情比较严重国家之一， 截至目前，全国总共由60.7万人感染中共病毒, 其中死亡12,608人。进入3月份，菲律宾每天新增中共病毒病例超过2800人。

菲律宾从3月1号开始在全国接种疫苗，目前接种的疫苗只有来自中共国的科兴疫苗。  菲律宾政府计划在2021年内为全国半数人口即7000万完成中共病毒疫苗接种。 但是根据OCTA研究中心的一份调查显示，只有18%的菲律宾成年人愿意接种疫苗。对于各国疫苗信任度的调查中， 41%受访者信任美国疫苗， 25%的受访者信任英国疫苗，只有13%的受访者信任中共国疫苗。

近期，菲律宾综合医院医生协会要求在对医护人员接种之前，对中共国科兴疫苗的有效性进行复审。该协会指出，95%的协会会员反对使用中共国科兴疫苗。尽管菲律宾食品与药品局已批准中共国科兴疫苗的紧急使用，但由于它只有50.4%有效率，因此不建议给医护人员接种。

菲律宾总统杜特尔特虽然信誓旦旦向民众保证中共国疫苗有效，但他的医生以中共国科兴疫苗不适用于年龄大于60岁以上人为由，没有批准给杜特尔特接种中共国科兴疫苗。

其他东南亚国家政府在中共国疫苗外交的威逼利诱下，也不得不表态支持中共国疫苗。 马来西亚政府前首相和现任首相都接种了美国疫苗，但是为了让民众放心接种中共国科兴疫苗，马国的两位部长“自告奋勇”愿意接种中共国科兴疫苗。 印尼海事和投资统筹部长卢胡特多次在公开场合表达对中共国为印尼提供疫苗表示感谢。

尽管中共国通过蓝金黄外交，诱使一些东南亚国家继续为中共国疫苗站台，但是本周泰国和香港接连爆出接种中共国科兴疫苗后，多人出现严重副作用甚至死亡的事件。 这使得各国民众更加担忧中共国疫苗的安全性。

**新闻2 ****： ****【疫苗护照】 ****新加坡航空公司率先使用国际航协旅游通行证，践行“疫苗护照”**

3月8日，新航宣布将推出一个旅行通行证应用程序（IATA Travel Pass App）。新航将成为全球首个以移动应用程序以进行数字健康验证的航空公司。并在3月15日至28日期间，由新加坡飞往伦敦航班的乘客将开始试行。IATA Travel Pass APP 管理在Covid-19疫情间乘搭飞机时所需提供的乘客个人资料，以向有关当局证实乘客的出境资格。这些资料主要包括冠病检测结果和接种疫苗程序状态。

与此同时， 中共国开始为本国公民提供类似的“国际旅行健康证明”。用户可以在中共国社交媒体“微信”上下载这款数码式健康证明，证明书内记录着用户接种疫苗以及中共病毒测验的数据。中共国外交部长表示，启用这一健康证书是为了世界经济复苏及跨境人员往来便利化提供“切实可行的中国方案”。不过，这一国际旅行健康证明的使用目前仍基于自愿原则，而且也只面向中国公民。除数码版之外，这款国际旅行证书也提供纸质版，它是目前全球范围内，第一款已知的“疫苗护照”。
![]()![](https://gnews.org/wp-content/uploads/2021/03/Picture2-12.png)***中共国“疫苗护照”***

新加坡总理李显龙接受BBC专访时表示，中美之间的冲突比五年前更可能发生。但是，他坚持认为双方发生军事冲突的可能性“还不大”。李显龙指出中共国的政治方向，确实引发了与世界大小国家的紧张关系。他认为中共国现在走的路，并不符合中国的利益。

新加坡是美国在东南亚重要的安全合作伙伴，同时美国也是新加坡最大的投资国家。保持与美国良好的合作关系在新加坡外交战略中起着举足轻重的作用。 中共国是新加坡最大的贸易出口国， 中共国的市场对于新加坡的经贸出口至关重要，因此新加坡与中共国保持着非常密切的经贸关系。新加坡的整体外交策略就是在美国和中共国之间寻求平衡，力求对新加坡利益的最大化。

未来一旦美国政府对中共的战略确定，尤其是全球以毒灭共和以反人类罪灭共的大战略确定后，新加坡政府肯定会调整外交战略，届时新加坡政府在美中关系上，就必须要选边站了。

**新闻4****：** 【**蓝金黄】印尼政府部长卢胡特被指责为中共国“代言人”**

印尼海事与投资统筹部长卢胡特多次在公开场合表示，中共国对于印尼政府的要求非常慷慨，全部同意印尼政府的条件，并且与我们分享技术，这个对印尼经济在疫情之后的复苏，非常重要。但是他的这番言论被印尼的网友指责其为中共国的“代言人”。

印尼是东南亚地区中共病毒最为严重的国家，截至目前，印尼全国有141万人感染中共病毒，其中死亡38,229人。 中共国在疫苗接种给与印尼政府强有力的支持， 目前印尼接种的疫苗中绝大部分是来自中共国。 为了表示对中共国疫苗的支持， 印尼总统佐科在1月13日接种了中共国的科兴疫苗。 截至目前，印尼是东南亚国家接种疫苗人口最多国家。 同时中共国还与印尼政府达成一致， 由中共国提供疫苗技术，与印尼当地医药机构联合研制开发疫苗，以满足印尼当地的疫苗接种需求。

中共国成功利用疫苗外交，让印尼政府的高级官员能够为中共国背书。1月份印尼已经开始大量使用中共国的科兴疫苗接种，但是中共国还没有对外公开过科兴疫苗研制的所有科研数据。这让很多印尼的老百姓对于中共国科兴疫苗的有效性和安全性保持怀疑。 2月份印尼爪哇岛万丹省一所医院的20多名医护人员感染Covid-19病毒，其中多名医护人员已经接种了中共国的科兴疫苗。  这个事件让印尼老百姓再次质疑中共国的疫苗的有效性。

坦白地说， 中共国在印尼蓝金黄策略非常奏效，尽管民间对于中共国的疫苗颇有微词，但是印尼政府目前来看是力挺中共国疫苗。
