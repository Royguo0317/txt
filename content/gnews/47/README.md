张海所谓的“主张”有以下三点：

1. 张海父亲去年因感染COVID-19而病故，他向法院提告，追究当地政府和医院有关人员责任并索赔，但法院拒绝受理； 
2. 张海之后又要求湖北、武汉两级政府公开疫情资讯，也石沉大海； 
3. 武汉市地方政府一直不承认自己的瞒报犯罪行为，敢做不敢为，事后还百般狡辩。

而事实上这三点的主要核心是中共隐瞒了武汉疫情严重程度，并导致感染者未得到及时、有效救助。而这点已经在国际社会上形成了事实，即使中共承认，对其伤痕也是不是根本性的。

而全世界对疫情真相关注的核心应是以下两点： 
1. SARS- COVID-2是否是武汉实验室制造； 
2. 病毒是否是中共军方生物武器，并故意散播到全世界。

想要查清以上两点，WHO（或其他国际组织）到武汉应该做的事情就不仅仅是见受害者家属这么简单的事情，而是像班农先生在节目中所说，进入到武汉病毒研究所： 
1. 查看所有的实验记录 
2. 查看所有仪器设备原始数据 
3. 调查实验试剂/动物购买及使用情况
