在2月21号比特币每一枚价格达到了58000美金，这个价格是自比特币发行以来的最高价。比特币在2010年初始发布时价格低至每个不足1美分。比特币总量2100万枚，交易价格跌宕起伏，让投资者感受到了巨大的风险。突发事件和市场情绪的意外变化都会导致价格的剧烈波动。比特币的价值迅速下降数百甚至数千美元的情况并不少见。

比特币的特征：

-它们不受监管：作为一种去中心化的数字货币，比特币的开发目前不受任何政府和中央银行的监管，而是由对等互联网协议自行管理。

-它们易受错误和黑客攻击：作为数字货币，加密货币易受技术故障，人为错误或黑客攻击的影响。

-它们可能会受到分叉或停产的影响：比特币货币交易会带来其他风险，例如硬分叉或停产。当发生硬分叉时，事件周围价格可能会大幅波动。

而Gcoin作为新中国联邦发行的虚拟货币，它的潜力和价值是不可低估的。Gcoin有比特币没有的特性，

-受监管，每年发行10 亿

Gcoin不会超发，在美国证监会监管下，每年批准的数量是限定的，一共可以发行100年。

-与Gdollar自由兑换，低风险

Gdollar将会成为和美元在布雷森体系以前一样，有信用的货币。Gdollar和黄金挂钩，货币价值更高。 同样，Gcoin不会像比特币会大起大落，是一种低风险投资选择。

-新的一种中心化虚拟货币，反洗钱

Gcoin的购买要通过KYC认证，它会对每个投资人进行严格认证，防止不合法的资金流入成为操控市场价格的投机行为。Gdollar成为新中国联邦的一个主权货币，CCP被灭了以后，Gcoin会更有投资潜力。

-可以有更广泛的消费平台

Gcoin 可以在 GMall 和GFashion平台上购物消费和结算。G系列推广到各个国家，它可以被当作新的支付系统用于消费。

因此，CCP的灭亡，对每个爆料革命的战友来说，拥有Gcoin和Gdollar将会有更美好的未来。
