04月18日，美国总统拜登从阿富汗撤军的决定与美国最高军事指挥部的建议相矛盾。《华尔街日报》援引消息人士的话报道了这一情况。

据该报报道，驻中东美军司令弗兰克·麦肯齐将军、北约驻阿富汗部队负责人斯科特·米勒将军和美国武装部队参谋长马克·米利将军建议拜登将2500人的特遣队留在阿富汗。

该报消息称，美国国防部长劳埃德·奥斯汀警告总统，从阿富汗撤军实际上意味着结束对该国已经建立的脆弱稳定的支持。

有消息称，拜登仔细权衡了军方的意见，但决心结束军方参与美国历史上最长的战争。

《华尔街日报》写道，军方高层在4月6日得知拜登的决定，这与将军们的建议相悖。正如该报所言，总统坚持认为，撤军最有助于避免阿富汗重新爆发敌对行动。

4月14日乔·拜登承诺，将在2001年9月11日纽约和华盛顿袭击事件周年纪念日之前撤出所有美军。部队将于5月1日开始逐步撤离。美国宣布后，北约接着宣布从阿富汗撤军。

美军自2001年起进驻阿富汗。2001年9月11日，武装分子劫持了四架客机，军队进入该国是为了应对袭击。其中有两架撞向纽约市的世贸中心大楼，另一架飞机的目标是华盛顿特区的五角大楼，第四架飞机坠入宾夕法尼亚州的田野。共有2977人遇难，在俄罗斯被禁止的“基地” 恐怖组织宣称对此次袭击负责。

在美军驻扎阿富汗期间，有2000多名美国军人被杀。自2011年起，美国开始逐步减少其特遣队，并在2014年撤出主要作战部队。在该国领土上还有北约部队执行非军事任务。
