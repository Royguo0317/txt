二、世界新闻

1. 美国海军在一份声明中说：”海军陆战队的平型”马金岛”号（USS Makin Island）是一艘两栖攻击舰，配备了数千名海军陆战队员、直升机和倾斜旋翼运输机，周五加入了美国”西奥多·罗斯福”号航母的综合作战。这支远征军打击小组充分表明，我们保持着一支具有战斗力的部队，有能力应对任何突发事件，遏制侵略，并确保区域安全与稳定，以支持一个自由和开放的印太地区”。

2. 以色列国防部长本尼·甘茨周日对他的美国同行劳埃德·奥斯汀表示，以色列将与华盛顿合作，确保任何有关伊朗核计划的”新协议”都能保障地区安全。发表上述言论之际，奥斯汀对以色列进行了自恢复2015年伊朗核协议谈判以来的首次美国高层访问，犹太国家强烈反对这一协议的重启。

3. 据周日晚间的报道，美国国家安全局前副局长英格利斯（johnc.Inglis）将被拜登总统任命为国土安全部网络安全局局长。美国政府正努力应对两起重大网络事件：一是俄罗斯的网络间谍活动“太阳风”（SolarWinds），危害了9个联邦机构和大约100家私营公司；另一个是中共国对Microsoft Exchange服务器的黑客攻击，这对私营部门造成了打击。

4. 乌克兰军方4月11日表示，俄罗斯支持的分离主义叛军对乌方发动了炮火攻击，造成一名乌克兰军人死亡及一名军人重伤。与此同时，克里姆林宫却声称，俄罗斯不会与乌克兰发生战争。
