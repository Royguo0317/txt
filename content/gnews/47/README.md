笔者在国内和美国都有过经营初创企业的一线商业经验，即有些B2B的经验。但像这次买房的C2C正规交易的经验还是第一次。本人是爆料革命长期战友，通过文贵先生的爆料，自己也基本知道中美商业和经济领域将要发生什么。尽管自己不是商界大款，但从最基本的经济盘算出发，买房比租房划算。给别人的租金基本可以还月供贷款，所以决定买个价格便宜且性价比高的房子。

买房的过程中，想起国内相关的商业环境，就有一些感触。最近爆料革命越来越多好消息，今天相中和确定买下了一个房子，心情大好。结果自己和自己喝起酒来，不知不觉进入酒酣意浓之境。有感而发，写下这篇小文，和大家分享。

### **我经历的美国买房过程**

第一步，个人先通过银行拿到贷款额度，这一过程称为prequalify 或 preapproval。

第二步，通过Zillo（一个卖房软件）和Agent（代理）获取实时新房上市信息。美国什么事似乎都离不开代理。

第三步，中意的房子按自己想出的价格下单，俗称offer。只简单强调一下，这个offer其实是个正式的合同。下offer时买房人可以说一个时间，比如早上10点下的offer，你可以在offer里说当天晚上10点一定要给我回复。但是有时可能你给的时间太紧或者卖家有事，卖家可能会说晚几个小时或者一天内给回复，基本也就在一天范围内会给答复。

第四步，卖家有三个选项：拒绝买房给的offer，交易结束；同意offer，直接签约；或回复新的价格让你考虑。这样就进入讨价还价，一般不会超过3次。大部分情况第二次就会达成交易，比如回复，增加100块，能接受就接受，不接受就算了。

第五步，确定价格后，买家找专业的检测公司对房屋进行检查，并提供检查报告。如果房子检查出来什么问题，买家要求卖家修复或者降价。过程中如果双方达不成共识，比如检测有一个问题，买家觉得卖家一定要修，但是卖家觉得是小问题，属于正常情况坚决不修也不降价。这样谈不拢的话，双方不必承担任何责任的毁约，从头再来。

第六步，贷款下发，交房拿钥匙，入住。

### **几点感触**

买家下offer时可以要求某个确切的回复时间。比如早上10点上市的房子，11点买家去看了，各方面还比较满意。标价100块，买方出100块或者98块或者103块，并且要求晚上10点前给回复。卖家一般都会在约定时间前回复，并且很大可能直接答应，也就是当天挂出去当天卖掉。卖家不会想挂一段时间看看能不能卖更高价钱。

整个交易不用担心意外被坑的情况。就算卖家挂出去一个小时后接受了买家的offer，他也会把房子从网上撤掉，改变状态到pending（交易中），不再售卖。作为中共毒区出来的人有点不可思议。几乎所有国内的人都会想我多挂几天，骑驴找马，多接受点报价，看能不能多卖一点。我眼睁睁看着很多房子在一天之内消失，当然也有原因目前美国利率低，房子好卖，但是整个交易能在一天之内完成也是很震撼的。

买家加价买东西，这点在中国也是不可思议的。毒区出来的有一个潜意识，我一定要出比你标价更低的价格买到相应的产品。哪怕低1毛钱也行，要不会觉得很亏，特别是房子这样的大额商品。但是美国买房子，如果你看到满意的房子，多少加个价太正常不过了。这个在美国生活一段时间的人都觉得没什么，但是仔细想想也是很震撼的，至少和国内比较的话是不可思议的。因为你不管想什么办法，找什么关系，还是要些礼品，都想降一下价钱。这里请注意，这个是买家个人主动加的，和国内明确加价提车完全不是一个概念。

另外一点，我本人在国内和美国有过一线（其实就是最底层）的商业经验，还有一个最最最明显的区别在于，和美国企业签的项目从来不用催款！！！合约时间到的时候，你连电话和邮件都不用检查，只用查账户上的钱是否在账上即可。国内要账的情况相信有过经验的人都明白我说的什么意思。我们整个商业的底线原则、诚信环境、守约思维与合规的逻辑方面有太长的路要走。期望灭共后我们也能尽快建立起这么文明的商业基础。
