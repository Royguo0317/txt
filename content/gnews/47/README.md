## 中共国品牌在推动抵制西方服装公司之际吹嘘使用新疆棉花

**作者：**REBELNEWS March 26，2021

![]()![](https://gnews.org/wp-content/uploads/2021/03/41.png)图片来源：REBELNEWS原文
中共国正在报复西方对其在新疆侵犯人权的制裁，将矛头对准了依赖其在中国供应链的行业：服装。H&M公司被中共国置于菜板上，该公司半年前发表的言论受到中共国审查，当时该公司与疏远了中共国，因为其在新疆强迫犯人劳动。

数十个其他品牌也因拒绝使用来自动荡地区的产品和资源而面临来自愤怒的中共国的制裁，西方领导人认为中共在新疆搞种族灭绝，有200多万维吾尔族和其他群体的成员被拘留。

中共国运动服装品牌利用共产主义、民族主义的浪潮，自豪地夸耀自己使用新疆奴隶棉。这一努力正导致对西方服装品牌的全面抵制，因为中共国名人公开谴责这些公司，并呼吁公民支持中共国用新疆奴隶劳动制造的产品。

NPR报道说，中共国所有主要的在线零售商，包括淘宝网，都把瑞典品牌从店面撤下，因为公众呼吁伤害批评这个共产主义国家的西方公司。中国移动公司、华为、维沃、小米和腾讯也禁止H&M进入其应用商店。此外，百度、天平和AutoNavi等中共国地图提供商从地图中删除了H&M的实体店位置。

随着反对H&M的言论愈演愈，中共国官方媒体纷纷呼吁抵制其他品牌，包括耐克、汤米·希尔菲格、卡尔文·克莱因、彪马、康弗斯、优衣库、阿迪达斯、新平衡和巴宝莉，因为他们拒绝使用新疆棉花。

2020年，澳大利亚战略政策研究所的一份报告将这些品牌与正在进行的暴行牵连在一起，数十家公司公开谴责中共国的奴隶劳动行为。

“纯正而完美无瑕的新疆棉花不能允许任何势力玷污或抹黑它，”一位中共国商务部发言人周四以久经考验的共产主义方式表示。

耐克中共国品牌大使王一波宣布，他的工作室将断绝与耐克的联系，他说：”我们公司和王坚决抵制任何玷污中共国的言论和行为，不能侵犯国家的尊严，我们坚决维护祖国的利益。

他的民族主义言论得到了其他中共国名人的赞同和耐克的赞助。

与阿迪达斯和优衣库有联系的中共国名人公开谴责这些品牌，并呼吁支持使用新疆棉花的中共国品牌。

雅虎财经称，西方公司的抵制导致中共国国内运动服装公司在中共国国家宣传和民族主义言论的推动下，股价飙升。许多中共国公司自豪地承认他们使用新疆棉花。

在香港联交所，李宁上涨10.74%，至50港元：安踏上涨8.4%，至121.30港元;Xtep国际上涨2.72%，至4.53港元。微博上的热门标签一直称赞这些团体专门使用新疆棉花，尤其是李宁在其服装标签上列出了新疆棉花。

一些西方公司试图通过支持中共使用奴隶棉来摆脱抵制，包括日本零售商Muji和雨果Boss，他们在其搜索词清单中加入了”新疆棉花”，并公开吹嘘自己采购这些有争议的材料。

老板雨果表示，”坚持生产和制造高质量的产品，使用来自世界各地的面料，包括来自中共国主要产区的原材料，”包括新疆棉花。

“多年来，我们始终坚持一个中国原则，坚决维护国家主权和领土完整。我们已与许多优秀的中国企业建立了长期合作，并将继续这样做。新疆长棉是世界上最好的棉花之一，我们相信优质原料一定会实现价值。我们将继续收购和支持新疆棉花。
