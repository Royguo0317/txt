政客们还在做着春秋美梦之际，随着各国各地不断出现的疫苗事故，疫苗经济已然成了疫苗灾难，再次验证郭先生说的：相信共产党走进火葬场；包括国家包括个人。这些相信了共产党与其同流合污的政客与商人们结局是羞耻的无法收台,必然葬送在自己的无知与贪婪之上。4月10日郭先生盖特说到，摩根先生本来也要打疫苗，因为关注爆料革命的信息，坚决不打疫苗现在已经得到他私人医生的肯定，倍感欣慰。

4月11日中共CCTV报道，强生疫苗推广接种后，美国国内多地发生了接种该款疫苗后出现头晕、呼吸急促等不良反应状况。从4月9日开始美国佐治亚州、科罗拉多州以及北卡罗来纳州的部分地区已宣布暂停接种强生新冠疫苗。报道中指出仅4月7日当天，该地曾有425人接种了强生新冠疫苗，其中有8人在接种后出现了不良反应情况，其中有一人被送往医院。同样在7日，美国科罗拉多州的一处新冠疫苗接种点也被紧急关闭。当天在这里接种强生新冠疫苗时，有11人出现了不良反应，其中2人因副作用反应强烈被送往医院。这一疫苗接种点的负责人表示，出于对市民负责的态度，该疫苗接种点决定终止当天余下的接种工作，未接种强生疫苗的600多名市民被重新安排到另一处接种其他疫苗。

无独有偶强生公司的疫苗在欧洲又被发现其中含有凝块的问题，目前被兼管机构调查中. 欧洲药品管理局（EMA）正在审查强生公司的COVID-19疫苗与血凝块之间的可能联系。

欧洲的药物风险委员会的报告指出有四例在接种疫苗后出现了严重的血凝块。目前尚不清楚血块是否与疫苗有关或与其他医学问题有关。但是强生公司曾发布一份正式声明中说，该公司“意识到”“所有COVID-19疫苗都报告了血栓栓塞事件（血凝块）”。其症状可能包括呼吸急促，胸痛，腿部肿胀，持续的腹部疼痛，神经系统症状，过度或容易瘀伤或注射部位以外的皮肤下有细小血斑。

不管是辉瑞,阿斯利康还是强生都出现了各种各样的问题,更不用说中共国的疫苗，由于信息封锁的原因，国内觉醒的民众也是越来越多，体制内真实打疫苗的情况不断50%,现在中共也在不择手段的想忽悠民众打疫苗，前段时间的免费打疫苗的口号有点行不通了，中国人不吃这一套。近期又出台打疫苗送鸡蛋,打疫苗送百元现钞等促销福利,相信很快会出台各种的强制手段逼迫人民打疫苗,如出入小区自由\工作自由等.我们只要记住一点,相信共产党,走进火葬场。凡动刀的必死在刀下，共产党既然打开了潘多拉盒子向全世界投送了病毒,它也必然死在病毒的事件上。已毒灭共，全球灭共浪潮已经势不可挡。
