**西班牙2021年2月26日电/西喜社——**据国家报2月25日报道，欧洲联盟希望不惜一切代价避免在残酷的冬季浪潮之后面临新的感染升级。由于制药业，特别是阿斯利康公司正在努力加紧生产中共病毒疫苗，欧盟领导人呼吁 “紧急加快 “其许可、生产和销售进程。此外，他们还敦促各公司 “尊重与布鲁塞尔达成的合同交付期限”。

欧洲的疫苗接种活动开局很不顺利。据欧洲疾病预防控制中心（ECDC）统计，在短短两个月内，只有2.5%的欧洲人口注射了这两种药物。欧盟在为其最脆弱的公民进行免疫接种方面还有一段路要走，因为到目前为止，80岁以上的人中只有10.5%的人已经完全接种了疫苗。在这些数字下，每一次延迟交付都是一次打击，尤其是疫苗接种是实现欧洲医疗系统解堵的关键。莫德纳和阿斯利康的新延迟使27个国家处于戒备状态，它们仍在等待从第二季度开始恢复到正常速度，届时它也将开始接受杨森的订单，并拥有3.8亿剂量的处置权。

在通过视频会议解决健康危机的第二次峰会上，欧盟各国元首和政府首脑向冯德莱恩提出了众多问题。据欧盟消息人士透露，包括马里奥-德拉吉在内的欧盟领导人首次将他们几乎从竞选开始以来就面对的挑战，如疫苗生产的延误、欧洲制造的困难、需要有一个可预测的交付时间表、制药集团在生产和出口疫苗方面有更大透明度，都摆在桌面上。欧盟27国也担心，如果有一种能降低疫苗效力的新变种的出现，可能会放大所有这些问题。

冯德莱恩在随后的新闻发布会上表示，布鲁塞尔仍在努力 “解决生产和供应链的瓶颈问题”，并补充说，这些努力已经取得了部分成效，现在有45家工厂在生产疫苗。欧盟也抱怨交货延迟，这甚至引发了布鲁塞尔和英国阿斯利康公司之间的争吵。”企业必须确保其疫苗生产的可预测性，并尊重合同规定的交货期限。”领导们还呼吁 “透明”。因此，欧盟领导人呼吁布鲁塞尔对未能履行承诺的企业，在禁止疫苗出口方面要更加强硬。
