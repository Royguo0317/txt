## 【香港风云】24: 黑鹰直升机

![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-106.jpg)



摘要：2019年底香港反送中、台湾大选、中俄伊军演同时进行。台湾黑鹰直升机坠毁，参谋总长等台军要员身亡，次日伊朗发生了一系列的无人机暗杀、军事报复、逃亡、击落客机。郭文贵先生透露中共的远程牵引技术，以及共产党对台湾军政两届的渗透。

2019年底，香港反送中运动残阳如血，台湾选战如火如荼，中共在香港、台湾兴风作浪的同时，在中东也动作不断。12月27日至30日，中国、伊朗、俄罗斯开始在印度洋北部的阿曼湾举行40年来首次联合军事演习。波斯湾通过霍尔木兹海峡进入阿曼湾的印度洋外海，是全球原油输送的战略要道，是郭文贵先生反复提及的地缘政治要冲。

2019年12月30日，郭先生在直播中说：“香港的运动刚刚还没有结束呢，走向另一个阶段的时候，中俄伊的军演具战略意义！”

郭先生谈到伊朗革命卫队和中共盗国贼家族、昆仑银行等方面的幕后交易，中、俄、伊、朝鲜、土耳其邪恶集团对基督教西方文明世界的威胁。“这就是为什么中俄培养的革命卫队一定会遭受美国的打击”。

“但是大家要记住，我2017年爆料的时候就说，霍尔木兹海峡有一天会有人去炸几艘船，会有人出事儿，甚至偷袭美国军舰，在过去两三年都发生了。

“这和一带一路、2025、2035、2049都是一样的，目的只有一个，3F美国。这是为什么我在2017年新闻招待会上，最后一分钟，我告诉翻译，告诉所有台下的记者，美国人准备好了没有，黑暗已经来临！”([生化危机 1: 遥远的警告](https://gnews.org/zh-hans/614371/) )

![]()![](http://5b0988e595225.cdn.sohucs.com/images/20190802/1b7dd884ba4f44c19abccab8bc73dc53.jpeg)

伊朗风波未平，2020年1月2日，在离台湾大选不足十天的紧张时刻，发生了一场黑幕重重的军事政变。当日上午7时50分，一架隶属台军的黑鹰UH-60M直升机计划从台北松山区飞往宜兰东澳岭雷达站，其间迫降在新北市乌来山区。事故造成同机13人中的8人遇难，其中包括台军参谋总长沈一鸣上将与两位少将。

在中华民国军队国家化的体制下，国防部为行政院下属的军事主管部门，国防部长为政务职文官。国防部设参谋本部，是军令系统的最高单位，首长为参谋总长，为现役军职最高职务。罹难的参谋总长沈一鸣将军，曾任空军司令和法国幻影战斗机试飞员。郭先生评价他“可不是一般的将军，是深懂这个世界上最先进的飞机航空、军事、情报技术的”。

![]()![](https://ichef.bbci.co.uk/news/640/cpsprodpb/16843/production/_110372229_3cc22b86-83ac-42ec-a988-05f8b9d96a0a.jpg)罹难者中的参谋总长沈一鸣 图片来源：AFP

黑鹰直升机事件惊魂未定，中东风云又起。2019年12月30日郭先生曾说“中共和俄罗斯把危难之时的伊朗的革命卫队当成了一个借来的刀来杀美国，美国难道不明白嘛”。

“头两天，我在洛杉矶的时候，和中东的一位王子，我跟他通电话，我说我向你保证，美国人一定会有行动。他本来要开新闻发布会的，我们欢迎俄军，欢迎中共，包括伊朗虽然是我们的仇敌，但是这个演习是有必要的，维护中东安全利益。我说你疯了，你想自杀啊。我说你记住，如果你敢把这个新闻发布会发出去，我说美国人的导弹真有一天会像炸南斯拉夫大使馆那样炸你家去了，我说你最好别干。最后他放弃了，今天跟我说，Miles我感谢你。”

郭先生在中俄伊军演时期谈到的美国导弹，绝非耸人听闻。黑鹰直升机事件次日1月3日，伊斯兰革命卫队头目卡西姆·苏莱曼尼(Qasem Soleimani)的车队遇袭。一架美国MQ-9收割者无人机，在伊拉克巴格达国际机场附近，发射导弹袭击了这位伊朗二号人物。中东情报之王、中共的座上宾、伊朗极权政权的接班人，在北京、上海、香港、德黑兰、巴格达等地呼风唤雨的伊斯兰革命卫队少将、圣城军指挥官、伊朗的海外军事情报头目苏莱曼尼在自己的车上被大卸八块。

美军对苏莱曼尼的突然行动引发了一系列事件。数日后的1月8日，伊朗伊斯兰革命卫队发起“烈士苏莱曼尼行动”，向伊拉克的阿萨德空军基地和库尔德斯坦阿尔贝拉空军基地发射了22枚弹道导弹。这次复仇行动没有造成美军伤亡，似乎是一次为了安抚内政的象征性报复行动，从另一方面也暴露了伊朗当局对川普政府的忌惮。

苏莱曼尼之死引起伊朗当局内部巨变。1月8日当日早晨，乌克兰国际航空公司(UIA)的PS752号班机，由波音737-800客机执飞，从伊朗德黑兰伊玛目霍梅尼国际机场起飞飞往乌克兰基辅，许多敏感人士乘坐PS752班机逃离伊朗。伊朗当局为了阻拦他们不择手段。班机起飞不久，客机连续被两枚伊朗军方发射的托尔-M1导弹击中。着火的飞机在试图返航的过程中爆炸坠毁，造成176名人员罹难。

![]()![](https://s4.freebeacon.com/up/2017/09/controlfailure.jpg)Lady May游艇被黑时，船用计算机发出的警报

早在2017年7月18日，郭先生乘坐技术先进的Lady May游艇在纽约哈德逊河上航行，遭遇失控转圈，险些和300英尺外的大型液化天然气(LNG)船相撞。此后29日、31日，船行至哈德逊河熊山大桥(Bear Mountain Bridge)附近时，被四旋翼无人机近距离跟踪。事后证实船用计算机控制系统被中共的黑客攻击。

FBI和CIA对这样的攻击手段感到震惊。当年9月8日，法治基金董事、五角大楼的喉舌、美国顶级的军事记者比尔·格茨(Bill Gertz)先生在华盛顿自由灯塔的国家安全专栏发文详谈此事。

Lady May事件后，郭先生透露中共是目前世界上拥有远程牵引技术的少数国家之一，马航MH370事件也和远程牵引技术有关。2020年1月3日，黑鹰直升机事件和苏莱曼尼事件发生后，郭先生上直播分析局势：

“中共能把你美国所有拥有电脑的现代化技术，所有的只要是有电脑的，有芯片的飞机、汽车、游艇、大小船，包括家里你的智能家居，都可以控制。”

郭先生表示，黑鹰直升机并非外界猜测被地面导弹击落，而是通过远程牵引技术坠毁的。黑鹰直升机事件背后有巨大的政治、军事、情报斗争背景。

中共对台军的渗透远超常人想象，台湾台军内部绝大多数被收编了。中共在大陆有专门的人员和设施用于接待台军将领。“在台湾，我告诉大家：共军的渗透远远超过如此。基本上说，在台湾的军队想干掉谁，干掉某个人那是说干掉就干掉，比他到秦城监狱杀个人还简单。”

黑鹰直升机的失事，不仅显示出中共对台军高层的染指，还印证了Lady May被黑客攻击时，郭先生对美国相关部门的警告：“我曾经给美国的某些部门都说过，我说共军获得你们的军事核心技术，其中一个渠道就是台湾，再一个就是巴基斯坦，然后再就以色列，还有一些中东某国。”

2020年1月4日，郭先生又谈到：

“现在叫非常香港，另外一个非常台湾，台湾的黑盒子已经解密了。共军派出了N个人，现在在福建有对台的三个班子，就是一定要把大选拿下来。在厦门，还有在其他的某个县，专门进行通信监听，指挥台湾作战，务必拿下，务必要拿下台湾大选，韩国瑜，一定要拿下。绝对不能让蔡英文连任。军办、军委七办、还包括总政联络部全面出击，N个人已到达台湾，除了原来工作的，现在很多人在台湾。

“亲爱的兄弟姐妹们，你们看看最近澳门，澳门有些诡异，在澳门有一个对台的指挥中心，对港的指挥中心，就在澳门。未来会有人爆出来的啊，记住我今天说过的话，未来会有人爆出来的。在澳门对台、对港的指挥中心，包括情报收集中心，现在美国也了解了，但是台湾的黑盒子引起了美国高度重视。所以美国高层说，必须把这件事情以最坏的角度去想。”

2019年底到2020年初，香港反送中、台湾大选、伊朗军事行动，还有美国的川普弹劾案，和中美贸易协议([生化危机 10: 疫情前的美国](https://gnews.org/zh-hans/700427/) )，就这样千丝万缕地交织在一起。然而，2020真正的主题——CCP冠状病毒疫情才刚刚酝酿成型([生化危机 7: 一切都听党，走向火葬场](https://gnews.org/zh-hans/696481/) )。这一切世界风云莫测的背后都是同一张大手——CCP。
