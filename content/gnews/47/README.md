《纽约时报》和《ProPublica》联合发表的新证据证实，从疫情一开始，中共政府就一直试图对世界其他地方隐瞒CCP病毒的信息。

2月7日，爆料CCP病毒的医生李文亮死于他向世界发出警告的疾病。他在中国湖北省武汉市中心医院工作时，看到了新的严重急性呼吸道综合症（又称作SARS）。SARS就曾于2002年起源于中共国。

随着这位34岁医生英年早逝的消息传开，悲痛在微博、微信等社交媒体上传播开来–北京却着手掩盖真相。

《纽约时报》和《ProPublica》的联合团队报道说，”他们命令新闻网站不要发布推送通知，提醒读者[李]的死讯。他们告诉社交平台逐步从热门话题页面上删除他的名字。他们还启用了大量虚假的网络评论员，在社交网站上散播转移视线的言论，”。北京的互联网监管机构–中国网络管理局杭州办事处在其COVID审查行动中总共发布了3200多份指令和1800份备忘录–所有这些都是由黑客组织C.C.P. Unmasked（赤裸的中共）泄露的。

“当数字媒体正在加深西方民主国家的社会分歧时，”调查团队警告说，”中共正在操纵网络言论来执行共产党的共识。”

1月，在冠状病毒还没有被明确识别之前，中共就开始加班加点地在真相上误导世界，以保护党的形象，当感染率开始飙升之时—甚至让疾病看起来不那么严重。结果，世界失去了预防全球流行病的最佳机会。

这是两个左派新闻组织的揭露，令人震惊。现在，左派、右派和中间派都应该团结起来，让中共为保护自身形象而使整个世界付出巨大和致命代价的恶行负责。
