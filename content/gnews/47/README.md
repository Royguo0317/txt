自计算机互联网时代以来，信息的传播效率可谓空前提升。任何一个人都可以在某个社交平台通过写文章、评论新闻等方式发表自己的意见，并且只需要几秒钟就可以传遍全世界。

在这种情况下，专制国家往往需要对“言论自由”进行较大限制，原因很简单：专制国家需要霸占朝纲、彻底排除政治上的竞争对手，实现一个统治集团能够控制民众为己用，为自身利益服务。

当民众对这个集团不满意、有大范围的厌恶的时候，他们将可能失去对普通民众人身乃至精神的控制。在控制了言论后，当有民众发表对统治集团不利的言论时，统治阶级通常会以一个看似合理的理由去删除这些言论，甚至惩罚发言者。

而对于普通百姓的老百姓来说，他们往往很容易相信这些理由，进而造成思考能力的禁锢。

以CCP和小粉红作为一个例子，他们通常会将追求民主自由的言论贴上标签，包括“崇洋媚外”、 “西方反华势力”、 “公知”、“汉奸卖国贼”等等。

我们可以看出，当统治阶级或者其他具有权力的人士，给原本对美好的理想化事物的追求，贴上一个反派的标签，这可以轻易地激起普通百姓的愤怒，煽动百姓对他们的仇恨心理。

这样一旦这些对统治阶级不利的言论出现，就会有许多人去攻击和污蔑。 即使统治阶级对他们实施惩罚乃至酷刑，也不会引起普通百姓的反感，甚至会让普通百姓将这些惩罚、酷刑视为理所当然。

言论自由是一把揭露真相的利剑，它是民主社会的基础，我们不能因为一个人主张糟糕就封锁他的言论，这是不对的。他的言论也有个好处，那就是引发人们的思考和讨论，这对于反思历史是大有好处的。

对于一个真正为老百姓谋福利、对老百姓好的执政者，是从来不会害怕言论自由的，因为民众可以通过对比不同信息来判断到底谁对谁错。

如果你没有撒谎骗人，老百姓真的对你很满意的话，那你又为何看到不同意见就要乱扣帽子、乱咬人、把对方踩三脚又堵住嘴巴呢？这么做了，那只能说明你心虚、恐惧、心里有鬼，害怕自己正在做的见不得人的事情被暴露，这是极其可耻的。

因此，我们应当明白，无论多么糟糕的主张或者言论（除了太极端的言论、对他人进行歧视和诽谤的言论外），都不具有实际意义上的危害性。任何言论都杀不死人，只有实践了才知道是否有危害。我们没有理由限制言论自由，发表言论也是天赋人权之一。
