**《国家脉冲》独家报道：亨特·拜登（Hunter Biden）的电子邮件帐户似乎表明，他的父亲前副总统乔·拜登与他的家人进行电子邮件交易时,使用了化名“彼得·亨德森”（PETER HENDERSON）。**

尽管拜登（或“大佬”）的真实姓名在其私人帐户中已显示了一段时间，但其真名却在2016年初发生了变化，在亨特·拜登电子邮件中，从“约瑟夫·拜登”变为“彼得·亨德森”。

目前尚不清楚这种变化是来自亨特·拜登还是乔·拜登的设备，但是彼得·亨德森的电子邮件中包含了诸如“保持联系，爱你的爸爸”之类的消息和共享链接，此类内容通常是讨论的是希拉里·克林顿的信息。

在汤姆·克兰西（Tom Clancy）的《杰克·瑞安》（Jack Ryan）系列电影中，彼得·亨德森（Peter Henderson）是一个渗入美国政府的克格勃的名字。
