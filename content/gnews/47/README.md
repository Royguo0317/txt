- **[01.29 Breitbart: 拜登冠状病毒顾问就戴双口罩质疑福齐：它 “实际上可能适得其反”](https://www.breitbart.com/politics/2021/01/29/biden-coronavirus-adviser-questions-fauci-double-masks-it-may-actually-be-counterproductive/)**
拜登总统的冠状病毒顾问迈克尔-奥斯特霍姆博士质疑安东尼-福齐博士提议的双层口罩，或同时戴上两个口罩。周四在MSNBC上露面时暗示，这种做法 “实际上可能适得其反”。
MSNBC的Chuck Todd说：”有很多人说，双重遮盖将成为[疾病控制和预防中心（CDC）]的官方建议。”他问Osterholm在这个问题上的立场。
与福西不同，他在最近的采访中并没有否定双重遮盖的想法，奥斯特霍姆警告说，这可能会 “适得其反”。
“在使用面巾遮盖物方面加倍，实际上我们需要非常小心，”他告诉托德。”这实际上可能会适得其反，没有帮助。”
“我们知道，我不是一个空气生物学家，但我们知道，事实上，它们可能实际上阻碍了空气的进出，导致空气实际上在布和你的脸之间的大缝隙中逸出。这是一个更大的问题，而不是一个更小的问题。”他继续警告说，”太厚的布 “会阻碍空气的有效流动，造成窒息感。
“而空气就会从缝隙中进进出出。所以，这就是为什么我们必须非常小心人们现在提出关于双重遮盖等建议，而没有真正的数据支持，”Osterholm补充道。
在拜登总统的新型冠状病毒首席医学顾问福西出来支持双重遮盖的几天后，，他对TODAY的Savannah Guthrie提出了不同的观点：它 “只是从常理上讲，它可能会更有效”。
当被问及这种做法是否会带来变化时，他说：”你知道，很可能是这样的，我的意思是，这是一个物理覆盖，以防止飞沫和病毒进入。
“所以，如果你有一层物理覆盖物，你把另一层放在上面，这只是一个常识，它很可能会更有效，这就是为什么你看到人们要么是双层口罩，要么是做一个N95的版本，”他补充道。
在周二出席《美国新闻室》节目时，福齐再次没有排除双重口罩的可能性，即使是那些已经接种过疫苗的人。
Fauci目前对口罩和双重口罩的立场与他在2020年3月所持的观点形成了鲜明的对比，他认为口罩提供的心理安慰多于实际。
现在，人们不应该出去走路，没有理由带着口罩到处走。”Fauci当时说。
“当你处于爆发期时，戴上口罩可能会让人们感觉好一点，甚至可能会挡住一个飞沫，但它并没有提供给人们认为的完美保护。” 他也提醒可能会有 “意想不到的后果”。
“人们总是摆弄口罩，他们也经常摸自己的脸，”他补充道。
最近几天，关于双重口罩的争论不断升温，特别是随着病毒变种的不断扩散。
据《华盛顿邮报》周四报道，”戴口罩正在变成只是戴口罩”。
“美国发现了高传播性的冠状病毒变种，公共卫生专家敦促美国人升级简单的布质口罩，这些口罩已经成为大流行期间的主力盾牌。”该媒体继续补充说，这种改变可能 “就像在你已经戴的口罩上套上第二个口罩一样简单，或者更好的是在外科口罩上戴上一个织物口罩。”
- **[01.29 Daily Caller ：强生公司的冠状病毒疫苗有效率为72%，但对新病毒株的保护作用较弱](https://dailycaller.com/2021/01/29/johnson-johnsons-coronavirus-vaccine-72-effective/)**

强生公司周五宣布，其冠状病毒疫苗对防治COVID-19的有效性为72%，但对南非一种新型病毒株的有效性仅为57%。
虽然效果略逊于辉瑞和Moderna的疫苗，这两种疫苗在12月被FDA批准，但强生的疫苗提供了显著的分销优势，这可能是国家对抗病毒的关键。与辉瑞和摩德纳不同的是，强生的疫苗只是一针，可以储存在冰箱而不是冰柜中。
尽管该疫苗对被认为传染性更强的南非毒株的效率减弱，但强生公司的官员在宣布结果的声明中表示，该疫苗对遏制疫情至关重要。
该公司首席科学官Paul Stoffels博士说：”这是可以用一剂疫苗就能起到作用的大流行疫苗”。
尽管目前还不清楚这两种获批的冠状病毒疫苗对新菌株的效果如何，但早期研究表明，变异体只对疫苗产生病毒抗体的方式有轻微影响。
- **[01.29 Breitbart CBS报道：BGI（生物技术领域里的华为）试图利用冠状病毒检测收集美国DNA数据](https://www.breitbart.com/national-security/2021/01/29/report-huawei-of-genomics-tried-to-use-coronavirus-tests-to-collect-u-s-dna-data/)**


据美国情报官员透露，在疫情初期，中国提出在美国建立检测设施，但基于他们对安全风险的警告而被拒绝。
CBS报道的重点是BGI集团（早先称为北京基因组学研究所），这是一家中国生物技术公司，提出要在华盛顿、纽约、加州和至少其他三个州建立冠状病毒检测实验室。该公司提出了一些规模较小的协助冠状病毒检测的提议。
BGI生产了数以百万计的冠状病毒检测试剂盒在世界各地销售，并获得数百万美元的报酬，在许多国家建立检测实验室，包括以色列和沙特阿拉伯等美国的重要盟友。
美国政府对中国检测技术的大举推进心存疑虑，早在2020年春季就警告说，中国公司可能正在寻求从急需提高冠状病毒检测能力的外国客户那里获取敏感数据。
一位美国官员在2020年5月告诉彭博新闻社，BGI是 “基因组学界的华为”，指的是中国电信巨头，由于担心中国情报部门利用该公司的技术进行监控和间谍活动，该公司已被一些西方网络屏蔽。与华为一样，BGI坚称自己不属于中国政府，也不受中国政府控制，并向客户承诺其个人数据将得到保障。
60分钟节目采访了美国国家反间谍和安全中心（NCSC）前主任威廉-埃瓦尼纳（William Evanina），他说，他的办公室对中国利用冠状病毒检测收集外国公民数据发出的警告是合理的。埃瓦尼纳说，他很高兴能够说服美国各州长不要让BGI建立测试实验室，他坚持认为，尽管该公司否认，但它与中共和人民解放军有关。
“我们不仅向每个美国人发出了咨询，而且向医院、协会和诊所发出了咨询。知道BGI是一家中国公司，我们是否了解这些数据的去向？”他说。
埃瓦尼纳和其他安全专家特别警告说，如果中国能够收集大量的美国人的医疗数据，包括他们的DNA，那么中国就有可能垄断治疗所需的诊断设备和药品的生产，从而获得对美国民众、政府和美国工业越来越大的影响力。
据报道，中国政府正在疯狂地收集本国公民的DNA，却没有对为什么需要如此广泛的数据做出任何合理的解释。这项工作就像今天中国的许多监控技术一样，开始于新疆省受压迫的维吾尔族穆斯林中，但它迅速蔓延到全国各地，重点是采集中国男性的DNA。
“与官方家庭记录、监控录像或警方报告中的证人陈述相匹配，这些样本将成为中国当局追踪一个男人或男孩及其他们亲属的有力工具，无论他们认为合适的理由是什么，”医学伦理学家詹姆斯-莱博德（James Leibold）在7月为《纽约时报》撰写的一篇专栏文章中警告说。

### **[01.29 新浪网 中国华融资产管理股份有限公司原董事长赖小民被执行死刑](https://news.sina.com.cn/c/2021-01-29/doc-ikftssap1750504.shtml)**

经最高人民法院核准，2021年1月29日上午，天津市第二中级人民法院依照法定程序对赖小民执行了死刑。
　　2021年1月5日，天津市第二中级人民法院以受贿罪、贪污罪和重婚罪，判处赖小民死刑，剥夺政治权利终身，并处没收个人全部财产。赖小民上诉后，天津市高级人民法院裁定驳回上诉，维持原判，并报请最高人民法院核准。

　　天津市第二中级人民法院收到最高人民法院的刑事裁定书和执行死刑命令后，依法对罪犯赖小民宣判并执行死刑。临刑前，赖小民与其近亲属进行了会见。
