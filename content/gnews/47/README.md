## 卑劣的“小蓝人”

日前，中共国渔船大举入侵南海牛轭礁，引发南海情势紧张。外媒揭露，中共有一支政府始终不承认的海上民兵“小蓝人”（Little Blue Men）部队。这些民兵的船只坚固、装载自动化的武器，还比世界上90%的渔船都还要快，而且，这个CCP不愿承认的部队正日益扩张，规模已经达到令人忧心的程度。

根据CNN报导，中共国220艘重型渔船日前大举入侵南海牛轭礁，引来菲律宾严厉抗议，并指称船上人员为中共国海上民兵。对此，中共外交部发言人华春莹表示当时仅为一些渔船躲在牛轭礁附近避风，是正常的行为；中共驻菲律宾大使馆更直接否认，声称“中国根本没有海上民兵组织”。

此次事件，让笔者想起了抗日战争中的中国敌后战场，中共所依仗、并在日后大肆吹捧的“武装群众”战略。据“中共中央党史研究院”《党史文汇》2020年第9期刊载的“中国共产党在抗日洪流中的使命与担当”一文中描述：“中国共产党一贯重视发动群众、依靠群众、组织群众，把人民群众作为生命之基、力量之源。在中国抗日战争中，中国共产党更是把组织和动员人民群众开展人民战争，作为打败日本侵略者的根本保证。”“国民党当局却不愿意要人民群众参加抗战而坚持单纯政府和军队抗战。”简而言之，中共对群众参与战争持肯定意见，并在抗日战争中普遍采用这种战略，而当时作为代表中国的国民政府却对此战略并不推崇。

凡事都有逻辑，为什么中共和国民政府对这种战略的态度会存在如此大的差异呢？根据1907年《陆战法规和惯例公约》（《海牙第四公约》）附件《陆战法规和惯例章程》第一章第一条：“在民兵或志愿军构成军队或军队的一部分的国家中，民兵和志愿军应包括在’军队’一词之内。”在这种情形下，作为该公约缔约国的日本所属部队，对占领区疑似存在游击队的村庄、城市，进行大肆劫掠和侵袭，存在着一定的法理依据。而在笔者看来，这也是国民政府不推崇这种战略的一大原因。（在此声明一点，笔者并不是要为日本侵华的残忍行径寻找合理性，只是相对客观地对当时的战争现实进行分析。）

在抗日战争尝到甜头之后，中共在二战结束后的国共对抗过程中，同样多次采用了类似的所谓“人海战术”。 中华民国陆军一级上将胡琏将军在回忆中说道：“当年我在沂蒙山区与共军作战，亲眼看见他们驱使老百姓带两手榴弹来冲锋；我守军用机枪扫射，眼见死的都是老百姓，自然不忍打下去，这时共军正规军就上来了。” “我知道人海战术，但我们能用吗？我们宁可认输。”

现如今，“小蓝人”依旧是这种战略的体现。同时，CCP还大肆开展“军民融合”，其目的就是用平民作为恶党的挡箭牌，一方面令对手投鼠忌器，另一方面，又可以通过道德谴责作为对抗的策略。

“战狼”赵立坚口中“中国人民就是中国共产党的铜墙铁壁”，被作为工具利用的中共国老百姓们，你们心甘情愿成为炮灰吗？

也许，那句常被戏谑为带路党的口号，才是正确的选择: “Sir, this way, please!”
