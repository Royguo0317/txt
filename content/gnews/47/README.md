据美国内华达保守的新闻评论网站DJHJ Media 11月27日报道，美国佛蒙特州共和党州长菲尔·斯科特（Phil Scott）要求学校向学生“询问”有关他们家庭的感恩节活动。

报道称，教育系统中的一些人和政界人士得出的结论是，当父母把孩子放到当地的学校时，这个州就接管了孩子父母的权力。而公立学校则是由工会控制的马克思主义门户，每年迫使数百万的孩子通过该系统获得高中毕业文凭，但并不准备让他们成为有生产力的美国公民。

而现在，佛蒙特州州长菲尔·斯科特则以与中共病毒大流行作斗争的姿态，指示学校就学生在感恩节所做的事情询问学生，因为他想知道他们是否举行了大型的家庭聚会。

斯科特周二在推特上宣布，那些承认违反该州假日旅行和聚会规定的学生或父母将被迫参加两个星期的在线学习。如果有状况的学生做中共病毒测试，处罚将减少到一周。斯科特在周二的新闻发布会上威胁说，他“工具箱中有很多工具”，如果病毒继续在他的州传播，他可以使用它们。

报道还提到了共和党州长俄勒冈州州长凯特·布朗，认为她是个极左派的全球主义者。因为她鼓励公民对违反她最近的行政命令的邻居们报警，其中包括在家庭聚会中最多容纳两个家庭六个人的限制。任何在感恩节违反布朗命令的人都可能面临最高30天监禁的轻罪处罚，最高罚款1,250美元，或两者并罚。
