## 埃隆-马斯克的SpaceX卫星点亮新西兰北岛的天空
` 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hans/983179/)

编译：万壑千山

近期，生活在新西兰北岛的人们将可以看到天空中明亮的卫星链。埃隆-马斯克的SpaceX Starlink卫星公司计划创建一个由小型宽带卫星组成的大规模星座，为全球提供互联网服务。

该公司于2019年5月发射了第一批60颗卫星，并希望在2027年完成多达42000颗卫星的组网。

航天科学家Duncan Steel表示，3月14日晚上8点02分到8点13分，该公司发射60颗卫星。

Palmerston North、Taupō、Raglan、Hamilton、Tauranga、Auckland和Whangārei等地区只要天空晴朗都能看到卫星。

惠灵顿和新普利茅斯视角不佳。

Steel说：”行动从晚上8点02分开始，当第一颗卫星出现在西南部的低空时，每颗星都会用5分钟左右的时间在天空中缓慢上升，经过观测者的南面，然后又在东面和东北面之间的低空中渐渐消失。链中最后一个可见的星体将在晚上8点08分出现，8点13分再次消失。卫星在轨道上的速度约为每秒7.5公里，在大约325公里的高度经过新西兰，星链已多次在新西兰上空旅行。这并不稀奇，但人们确实觉得这些卫星在天空中移动相当惊人。未来会看到更多，它们在一两周内都会保持这样的链状，因为它们正在逐渐扩散开来。”

科学家警告说，SpaceX Starlink卫星可能会造成太空垃圾的噩梦。
