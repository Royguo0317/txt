## 美防長：美計劃建造“改變遊戲規則”的未來艦隊為對抗中共海上力量
`2020-09-18 13:46 vog_dongdong_vog` [轉載自GNews](https://gnews.org/zh-hant/366993/)

美國國防部長埃斯珀（Mark Esper）週三（2020年9月16日）宣布了一項雄心勃勃的計劃：為了應對來自中共的海上挑戰，美國將增加預算，擴充海軍，增配由無人戰艦、潛艇、戰機組成的無人海軍系統。 這項名為”邁向未來”（Future Forward）的”遊戲規則改變”計劃將美國海上艦隊從目前的293艘擴大到355艘以上。

未來美國海軍是355艘以上的載人和無人艦船組成的均衡力量，從空中、海上和海底都具備致命打擊能力，由越來越少地面戰鬥人員組成，增配無人車輛、各類型無人艦載戰機，更強大潛艇部隊，此外還有訓練有素的軍人和現代化的造船廠和高技能的工人。

今年早些時候，海軍批准了7.95億美元的合同，用於購買新型導彈導彈護衛艦的第一艘船，並可以選擇再購買9艘，總價值近56億美元。這是海軍十多年來尋求的第一個新的大型造船計劃。

本月初，海上獵人原型機與羅素號航空母艦完成了作戰，展示了有人和無人編隊的各個方面。無人反潛，獵殺潛艇，具備高達70天/1萬海浬的驚人續航力！
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/18133547/1-1-9.jpg)
埃斯珀在周三在加利福尼亞蘭德公司的演講中說：“第二次世界大戰後，美國，我們的盟友，我們的合作夥伴領導了建立以國際規則為基礎的秩序，這種秩序植根於我們的共同價值觀，在七十多年來支持了世界各地的穩定與繁榮。然而，這種自由和開放的秩序現在正處於迫害之中。”

```
根據美國國防部於2018年制定了《國防戰略》（NDS）致力於三方面： 
第一，增強部隊的殺傷力和戰備力； 
第二，加強聯盟，建立夥伴關係。 
第三，改革部門，使我們的最高資源（我們的資源與最高優先事項）保持一致。
```

埃斯佩說，“所有這三個方面的基礎，也是我們許多努力的基礎，將部門專注於中共。” “中國是美國最大的安全威脅，印度太平洋地區是美國軍方優先戰區。”

埃斯珀明確指出：“北京正發展長距離、自主和無人潛艇，北京認為發展潛艇應對美國海軍是划算。就海軍實力而言，中國無法與美國匹敵。即使我們停止建造新船，在公海能力方面，中國也要花費數年的時間才能縮小差距。”
