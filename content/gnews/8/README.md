•北京中共政府單方面決定在香港實行新《國安法》，這只是破壞香港自治和自由的一系列單方面決定中的一項。違背了國際條約《中英聯合聲明》中對香港人的承諾。

•美方本希望自由繁榮的香港為威權主義的中共提供榜樣，但很顯然中共正在讓香港效仿它自己。隨著香港人民繼續反對中國共產黨，美國將和香港人民站在一起。
