## 拜登政府将全面实施对中共科技威胁的清除工作

#### 翻译：sisyphus（天元）| 责编：人间四月
![]()![](https://gnews.org/wp-content/uploads/2021/03/capture-7.jpg)图片来源：breitbart
据《华尔街日报》近日评论文章称，川普政府曾发布禁止与多家中国科技巨头做生意的禁令，其中包括华为科技有限公司，据来自美国企业界的知情人士透露，拜登政府计划允许川普时代的一项规则，即旨在打击来自中共科技的威胁并将力排众议在下月在生效。

该规则最初是在去年11月提出的，它使商务部能够禁止与中共进行被认定为对国家安全构成威胁的技术相关的商业交易，这也是保障美国供应链安全所做的努力的一部分。电信、金融和其他行业的公司表示该规则可能会扼杀行业创新和竞争力，并曾期望该规则会随着美国政府对中共科技的政策进行广泛审查的同时被推迟。该知情人士还透露：拜登政府正在计划遵守这项规则，而且政府官员担心弱化该规则可能会传达关于新政府对中共态度的错误信息，导致潜在地助长有关政府对中共态度软弱的批评声音。

有消息称现任政府官员已经向商界暗示他们不会积极执行该规则。这样可能会软化其影响，尽管企业代表表示该规则仍将使规模较小的企业产生大量新的合规成本和不确定性。但也有消息否定上述猜测。

由亚马逊公司和花旗集团等大公司CEO组成的”商业圆桌会议”表示，该规则“以目前的形式对美国企业来说是不可行的，在没有重大修改的情况下，不应该考虑让其最终公布”。而沃尔玛公司和国际商业机器（IBM）则表示：该规则的书面内容”过于宽泛”，将损害经济，同时无法起到捍卫美国国家安全的目的。

商务部会继续接受公众对该规则的意见，直到3月22日，并补充说，届时该规则将最终敲定。商务部还说“值得信赖的信息和通信技术及服务对我们的国家和经济安全至关重要，仍然是拜登/哈里斯政府的首要任务”。根据商务部的估计，该规则可能会影响多达450万家各种规模的美国企业，会要求他们在与该法规所称的 “外国对手”进行涉及尖端技术的采购和交易时需先获得政府许可，否则可能会面临交易的强制取消。

根据规则草案内容，新的政府对美国企业的监督将涉及美国关键基础设施、网络和卫星业务、大型数据托管业务、广泛使用的互联网连接软件以及先进计算、无人机、自主系统或先进机器人技术等相关交易。它可能会影响相关企业的销售或在某些情况下影响技术的使用。

1月初，时任美国总统川普已签署了一项行政命令，禁止美国企业与包括支付宝在内的8家中共软件公司进行商业交易。

电信和金融服务行业被认为特别受到该规则的影响，因为它们拥有信息技术服务的大量用户，并处理敏感的消费者数据，但许多其他面向消费者的企业也有很多风险。

作为拜登政府对中共政策方向的先导，这一规则的命运正受到密切关注。华盛顿对于中共科技设备制造商和互联网平台所带来的安全和经济风险，已经形成了固化的共识。国会中的共和党人已经就对中共采取强硬立场的问题质询了相关拜登内阁提名人。

中共外交部发言人回应拜登总统下令审查关键物资供应链安全的言论。该发言人说，”用政治力量改变经济法是不现实的做法，既不能解决国内问题，也不利于全球供应链”。

中共此前曾指责华盛顿不公平地歧视中共企业，并试图以能否顺利进入中国市场做筹码，向外国企业施压，使其无视并游说对抗美国政府的限制。如果拜登政府允许该规则继续实施可能意味着美国企业会遇到更多麻烦，这些企业发现自己越来越多地陷入华盛顿在经济政策上努力与中共对抗和北京对美国政府行动报复的拉锯战之中。

川普政府曾发布好几个与中共几家科技巨头做生意的禁令，包括从电信制造商华为技术有限公司到微信等平台，尽管其中一些措施已被法院阻止。川普政府还试图迫使短视频应用软件抖音（TikTok）的中共所有者将其美国业务出售给美国公司，不过这一努力陷入了僵局，目前已被搁置。

而即将出台的商务部规则在某些方面是川普政府针对中共科技企业的行动中影响最深远的。它将赋予该部门广泛的权力，要求其为广泛的技术交易提供许可证，或者直接禁止这些交易。

华为的业务一直被美国之前的限制措施压制，可能会再次受到新规则的冲击，华为在评论中质疑该法规的合法性，并敦促拜登政府采取”整体风险管理”的方式，而不是 “对特定参与者设限”的做法。

许多商业领袖承认来自中共和其他敌对国家的技术所带来的风险，以及应对这些风险的必要性。这些风险包括窃取知识产权、健康数据和个人财务信息，以及从美国境内追踪美国人的位置和进行企业间谍活动，根据《美国商业周刊》的一份草案显示，这些风险包括窃取知识产权、健康数据和个人财务信息。

许多企业领导人担心，新规则将降低风险的责任过多地交给了企业，同时也带来了潜在的巨大成本和不确定性。比如一些企业担心他们将被要求更换已经在使用的设备，合规总成本可能在第一年达到520亿美元之多。根据商务部的估计，每年的实施费用高达200亿美元。

包括几家领先的科技集团在内的数十家美国商业团体已经提交了反馈，敦促拜登政府缩减或推迟该规则。该团体认为，拟议的规则含糊不清，问题很大，因为按照书面规定，该规则将赋予该部门近乎无限的权力，可以干预几乎任何美国公司与外国公司之间牵扯到技术的商业交易，并且几乎没有适当的程序、问责制、透明度，或者说是没有技术含量和缺乏与其他政府方案的协调，他们认为这些建议的目的也是为了保护国家安全。
