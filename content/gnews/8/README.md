[人类自由网站](https://humansarefree.com/2021/06/european-database-reactions-following-covid-19-shots.html)报道，据欧洲的疑似药物反应报告数据库[EudraVigilance](https://www.adrreports.eu/en/background.html)的统计，截至2021年6月5日，欧洲在接种四种实验性COVID-19疫苗后记录了13,867例死亡和1,354,336例受伤病例。

从记录的伤害总数来看，报告了683,688起**严重伤害**，相当于超过了总数的50%。如果有关的可疑不良后果：对应于导致死亡的医疗事故，危及生命，需要住院治疗，导致另一种重要的医疗状况，或延长现有的住院治疗，导致持续或严重的残疾或丧失能力，或属于先天性异常，出生缺陷，则可归类为“严重伤害”。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/08162521/136-1-European-Database-of-Adverse-Drug-Reactions-for-COVID-19-Shots-13867-DEAD-and-1354336-Injuries.jpg)图片来源：[人类自由网站](https://humansarefree.com/2021/06/european-database-reactions-following-covid-19-shots.html)

欧洲药物警戒系統 ([EudraVigilance](https://www.adrreports.eu/en/background.html)) 是欧洲数据处理网络和管理系统，用于报告和评估在欧洲经济区的临床试验中已批准或正在研究的药品的可疑不良反应。欧洲药品管理局代表欧盟药品监管网络运行该系统。

该网站由欧洲药品管理局于2012年推出，为公众提供药物可疑副作用（也称为可疑药物不良反应）的报告。这些报告由国家药品监管部门和持有药品上市授权（许可）的制药公司以电子方式提交给[EudraVigilance](https://www.adrreports.eu/en/background.html)。这些报告用于评估药品开发过程中的益处和风险，并在药品在欧洲经济区（EEA）获得授权后监测其安全性。

欧洲在接种四种实验性COVID-19疫苗后的伤害记录如下：

- *BioNTech/Pfizer公司的实验性mRNA疫苗Tozinameran（代码BNT162b2,Comirnaty）的总反应：截至2021年6月5日，有6,732人死亡，502,162人受伤*
- *Moderna公司的实验性mRNA疫苗mRNA-1273 (CX-024414) 的总反应：截至2021年6月5日，有3,821人死亡，101,767人受伤*
- *强生公司的实验性COVID-19疫苗JANSSEN（AD26.COV2.S）的总反应：截至2021年6月5日，有466人死亡，25,950人受伤*
- *牛津大学/阿斯利康公司的实验性疫苗AZD1222/VAXZEVRIA（CHADOX1 NCOV-19）的总反应：截至2021年6月5日，共有2,848人死亡，724,457人受伤*


![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/08162529/136-2-Summary-June-05.jpg)图片来源：[人类自由网站](https://humansarefree.com/2021/06/european-database-reactions-following-covid-19-shots.html)

这些总数是根据提交给[EudraVigilance](https://www.adrreports.eu/en/background.html)的报告估算的。如果根据所报告的不良反应百分比，总数可能要高得多。
