## 美國之音9/25/2020：人權組織和國際社會譴責中共對藏人的強迫勞動政策
`2020-09-29 01:21 CHANGDAO` [轉載自GNews](https://gnews.org/zh-hant/389755/)

香草山綜合報導

中文稿：
 1.習近平3月6日說，今年農村人口全部脫貧的目標，“必須如期實現，沒有任何退路和彈性”。
 2.中共以脫貧為由在西藏實施農牧民“轉移就業”的強迫勞動政策，系統性地對西藏農牧民大規模進行集中、封閉、半軍事化的培訓，然後強迫到外地從事低收入工作，將藏人與他們的社區，宗教和文化關係割裂開來，以更受制於政府的控制。截止7月，超過西藏總人口1/3的人已被轉移就業。這與新疆“再教育營”的政策如出一轍。
 3.由16個國家60多名議員組成的“對華政策跨國議會聯盟”對此發表聲明，敦促本國政府在全球馬格尼茨基人權問責法案，企業供應鏈風險管理和國際獨立調查方面採取措施。
 4.自由西藏組織週三發表聲明譴責這一政策，稱這將進一步撕裂西藏的社會結構和生活方式，是在踐踏藏人的人權。
