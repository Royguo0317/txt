**西班牙2021年3月16日电/西喜社——**据美联社3月16日报道，中共国已经批准了另一种中共病毒疫苗的紧急使用。

这一消息来自中共国科学院微生物研究所。该疫苗已于3月1日获准在乌兹别克斯坦使用。临床试验的最后一个阶段正在进行。目前还没有关于该疫苗的安全性和有效性的同行评审数据公开。

该公司发言人说，这是一针三剂，每针之间间隔一个月。与中共国开发的其他疫苗一样，它可以在正常的冷藏温度下储存。

根据中共政府官员周一在北京举行的新闻发布会上的最新数字，已经注射了6490万剂疫苗。这些疫苗主要是给医护人员、在边境或海关工作的人以及特定行业的人注射的。

中共漠视百姓的生命和健康，为了自身利益不择手段地“大干快上”疫苗经济。疫情爆发短短一年多时间，已经批准了五个中共病毒疫苗，要知道在这之前，七十年来国产疫苗的记录是零。
