传说，有一种神兽「龙」，虽天性温驯，但在喉咙处的有一鳞片，其生长方向与其他鳞片是相反的，称之为「逆鳞」。逆鳞相当敏感，即便轻微触碰，龙都会无比疼痛，触发强烈反击，故此，有了「龙之逆鳞，触之必怒」的典故。

**「韭菜」的反击**

最近几日，中（共）国事故频出，先是在大连有一名男子因投资失利而报复社会，驾驶宝马车，踩足油门，高速撞向正过马路的人群，酿成5死5伤的惨剧。

几日后，阜新市政府门口又有推着电瓶车的男子，在一辆黑色私家车驶出政府大门时，迅速靠近该车，几秒钟内发生剧烈爆炸，现场浓烟形成直径约12米的蘑菇云 ，造成1死5伤。 虽然官方报称是电瓶车突然爆炸，亦对黑色私家车内的伤者身份三缄其口，但不少网民回看监控片段，猜测事故另有内情，怀疑是肇事者刻意报复官员。

表面上看，两名肇事者可能是无力解决问题，心有不忿，转而委过于人，用极端的方式报复社会，报复权力阶层。 但实际上，在中共治下，这种悲剧的发生有其必然性，今天报导上的这两个男人，明天就可能出现在我们身边。

这样的报复行为是弱势群体情绪崩溃后，唯一能发出的一点「鱼死网破式」的怒吼，犹如周杰伦在《逆鳞》中所唱：「用力的还击，发出声音，让他们安静；他们畏惧，睁大眼睛，他们躲避，然后放弃，专心聆听，我的声音。」

**逆鳞的报复，痛之深，恨之切**

《韩非子·说难》中以「龙之逆鳞」作比喻，「夫龙之为虫也，柔可狎而骑职，然其喉下有逆鳞径尺，若有撄之者则必杀人」，借此奉劝说客不要触及君主的底线，否则恐遭杀身之祸。

古代皇帝固然是「真龙天子」，君威不容挑战，但中国人同样是「龙的传人」。触碰君之逆鳞尚且引爆震怒，何况几乎被中共刮除逆鳞的普通民众，即使再懦弱的顺民 ，只要是人，就有不能被践踏的红线，否则将难以估计他会作出怎样的极端抵抗，以上发生的两宗案件就是实例。

值得关注的是，随着中共邪恶被不断揭露，国际社会要同中共划清界线的动作无疑越来越具杀伤力，这势必令本就千疮百孔的中共经济濒临崩溃边缘，而受生计重压的人群将随之扩张至各行各业，各个阶层。届时，基层民众会首当其冲，成为最大受害者，一旦蔓延，伤害将难以想像。

又如，近期发生的甘肃马拉松悲剧，如果不是顾忌奖金，遇难的21个人可能在天气骤变之初，就选择中途放弃，实在没必要​​硬着头皮撑下去，正因为家庭生计之忧，加上制度上的缺失，才令他们最终跑上这条不归路。

试想下，这样的创伤对他们的家属来说，如果得不到妥善安抚，会不会触发又一宗报复惨剧呢？无人可以预测。唯一肯定的是，唯有消灭中共才能终止悲剧重演，否则，民不聊生之下，极端的报复行为会更加频繁，更加随机，甚至更加凶残，谁都无法幸免。
