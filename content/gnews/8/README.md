## F1 与 MOTOGP的全球顶级赛事上为何难见中共国车手

由FIA（国际汽车联合会）与FIM（国际摩托车赛车协会）举办的一系列赛事中，F1（一级方程式赛车）和MotoGP（世界摩托车锦标赛）赛事的2021赛季也均在三月的最后一个周末成功开赛了。这两项全世界最顶级的赛事，为何从未出现过来自中共国的车手？

[!\[\]()!\[\](https://spark.adobe.com/page/R8ElvL8E3Vnzh/images/31a89f6e-d903-4cb1-9ad8-47a96226490a.jpg?asset_id=b6c68e53-d763-405a-8d2d-9d9377f1119e&amp;img_etag=%22be1d3369e49349905feb36b879fa3944%22&amp;size=1024)](https://spark.adobe.com/page/R8ElvL8E3Vnzh/images/31a89f6e-d903-4cb1-9ad8-47a96226490a.jpg?asset_id=b6c68e53-d763-405a-8d2d-9d9377f1119e&amp;img_etag=%22be1d3369e49349905feb36b879fa3944%22&amp;size=1024)

在中共国所谓的“特色”社会主义国家里，运动员是非常特殊的群体。众所周知，赛车运动需要的经济投入是远超普通人能够负担的。我们不妨先从这些经济投入较小的奥运会运动员们说起，看一看他们的生活状况如何。

在中共国，奥运会选手一向采用举国选拔制度，从小开始物色，在超负荷高强度的状态下进行训练。中共国确实在许多奥运项目中获得了一定的成绩，却是以一代又一代运动员们的青春年华和身体健康为代价的。这样过度的训练给运动员们带来的各种后遗症，以及形同虚设的社会保障系统，让这些为国家放弃自己一切的运动员们，命运如同一次性筷子般悲凉，没用了便惨遭丢弃。有些读者会说，我们可以看到一些运动员似乎在退役之后依然过着不错的生活，但事实上，这仅仅是金字塔塔尖的那一小部分。在中共举国选拔体制下，无数运动员参与其中，却因为成绩不够顶尖，一生默默无闻，他们同样有着过度训练带来的各种后遗症，可他们的退役生活又是怎样的呢？我想中共党媒《人民日报》的专题网页对运动员退休后艰难生活的轻描淡写，只是揭露了中共选拔制度之残酷的冰山一角而已。

[!\[\]()!\[\](https://spark.adobe.com/page/R8ElvL8E3Vnzh/images/ae4609ab-5f67-4446-8e55-1e5423ef8898.jpg?asset_id=aa9acae7-c8ff-46a2-98a6-8d90353c23c0&amp;img_etag=%2223497f98064b65152dd5c46946b63bbe%22&amp;size=1024)](https://spark.adobe.com/page/R8ElvL8E3Vnzh/images/ae4609ab-5f67-4446-8e55-1e5423ef8898.jpg?asset_id=aa9acae7-c8ff-46a2-98a6-8d90353c23c0&amp;img_etag=%2223497f98064b65152dd5c46946b63bbe%22&amp;size=1024)图片来源：[人民网](http://sports.people.com.cn/BIG5/31928/357261/)

这样的劳民伤财不惜一切代价也要获取奖牌的中共国，试问有多少人明白这样的行为是完全违背了奥林匹克精神？对金牌近乎疯狂的追求已经完全失去了参与奥运会的意义。中共治下的体育界到底有多少不为人知的肮脏的秘密？纳税人的钱有多少是真正的被投入到运动员的健康发展当中？又有多少安置退役运动员的专用款项被真正落到实处？中共国治下的运动员的悲惨命运，从郝海东先生与叶钊颖女士对体育界腐败的揭露中便可见一斑。正如中共党媒自爆的那样，连曾经为国奉献的奥运会运动员们退役后都是这般光景，那么在国际赛车竞技场上难觅中国人的原因便也不言而喻了。

我想首当其冲的原因便是中共并不愿意为中国人花钱。奥运项目因其赛事特点，也为了让更多的国家参与，在项目的选择上更倾向于设备造价低廉、更多考验运动员体能的项目。相比奥运会的项目，赛车运动不仅需要大量的资金投入与研发，同时也对高精度的工业生产有着极高的要求。参加赛车运动，不仅投入的资金大，更不符合中共一切为钱服务的目的，甚至更加暴露出中共国精密制造和机械研发实力的巨大问题。

其次便是因为中共国的１４亿朴实的老百姓，由于长期被灌输只要一心爱国、为国奉献便能够善始善终的思想，导致对事情的是非对错缺乏判断力。很多运动员相信了中共的谎言，以为这样至少可以拥有一技之长养活自己，傻傻的以牺牲自己的健康与青春为代价，“被成为”了中共国的运动员。谁曾想到即使成为金字塔顶端的那些奥运冠军们，也不一定能够健康的正常退役，拥有幸福的人生。中共用美好的职业前景来诱骗百姓，用为国争光的虚名来绑架人民。被洗脑的中国人民，有多少人能够具备足够的思考能力，明白中共花重金参加奥运会的真实目的呢？笔者想起当时看到郝海东先生与叶钊颖女士，挺身而出为新中国联邦发声时的感动，更相信如果不是这样的举国选拔体制，而是健康的运动员职业化体制，一定会有更多的中国人站起来，为自己为正义发声。

细数从中国人开始参加汽车、摩托车赛事至今，能够在国际赛场上留下名字的不足十位。而登入F1赛场的唯一一位中国车手马青骅也仅在2021-13成为车队后备车手，参加练习赛，为车队提供测试数据。

笔者作为一个中国人，其实和每一位爱赛车的中国人一样，希望看到更多的中国车手能够出现在国际赛场上。可面对贪婪无耻的中共，笔者只看到纳税人的钱被无尽的挥霍，车市被控制，无法保障的运动员退役生活，没有对人的最基本的尊重。我想我们每一个人都不应该允许中共在中华大地上继续撒野，唯有参加爆料革命，加入新中国联邦，唯有灭共，才能让我们每一个中国人真正的活出体面与尊严。
