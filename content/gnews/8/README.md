## 美國國務院第三高官或將於9月下旬訪台
`2020-09-03 12:33 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/332652/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/03122928/cropped_image1.png)圖片來源: keithkrach.com 
根據台灣媒體《自由時報》(Liberty Times ) 近期報導，美國副國務卿基斯·克拉赫(Keith Krach)可能將於9月底前往台灣與台灣政府舉行商務討論。

近期，台灣決定從2021年1月1日開始，結束一項禁止進口含有瘦肉性藥物萊克多巴胺的美國豬肉政策。而克拉赫一直負責經濟增長，能源和環境相關議題。這次克拉赫的訪問，將會是自1979年兩國斷交後，美國訪問台灣的最高級國務院官員。

在此之前，美國東亞和太平洋事務局助理國務卿大衛.蒂維爾(David R Stillwell)和台灣經濟部長王美花，台灣駐華盛頓代表蕭美琴，將在8月31日討論美台經貿議題的視訊會議中會討論到克拉赫訪台事項。

報導表示，被《自由時報》稱為國務院“第三把手”的克拉赫表示有興趣親自領導台灣的這一輪會談。據《自由時報》報導，台灣外交部（MOFA）正在和美國國務院商討籌備有關的議題會談，並正在迅速推進，使克拉赫最快有可能在9月下旬訪問。討論主題範圍可能包括供應鏈的變化、5G和半導體領域，但是經濟部長王美花說，台灣不會預設任何先決條件，並表示台灣非常希望與美國達成雙邊貿易協議，但不會有時間表的壓力，因為還有其他的議題需要更多的討論。
