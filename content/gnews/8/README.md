### **Guo Media App 事件的起源**

一款郭文贵先生推出的应用程序-Guo Media App成功上架后，中共国官员联系苹果要求解释。苹果应用程序程序审查主管在凌晨时分以电邮通知同事，表示郭文贵的任何应用程序程序都不能出现在中共国苹苹果应用商店中。苹果接着调查此事，解聘了核准那款应用程序上架的审查人员。

[!\[\]()!\[\](https://spark.adobe.com/page/oSsIX1IyMNzIA/images/63750bb2-5d36-43bb-b6a8-adc1a43e2ada.png?asset_id=2baa02bd-b565-43a9-a7a2-b37c3ff11eac&amp;img_etag=%2227dd98a33679e564812f04821ec6a2e3%22&amp;size=1024)](https://spark.adobe.com/page/oSsIX1IyMNzIA/images/63750bb2-5d36-43bb-b6a8-adc1a43e2ada.png?asset_id=2baa02bd-b565-43a9-a7a2-b37c3ff11eac&amp;img_etag=%2227dd98a33679e564812f04821ec6a2e3%22&amp;size=1024)

1. –2018年初， Guo Media 向苹果 app store 提交上架申请，不久后中共要求苹果拒绝上架Guo Media 相关应用程序，尚不清楚中共是如何知道 Guo Media App提出上架申请的。
2. –2018年 2 月 4 日苹果的经理用电子邮件向同事发了一个问题：“郭文贵是否在苹果的&lt;中共敏感名单上&gt;？”Guo Media App上架问题被提交给苹果的“执行审查委员会”。
3. –2018年 2 月 19 日，执行审查委员会决定，郭文贵属于中共的所谓黑名单，同时将他的名字加到了苹果公司内部的：“中共国app store 删除维基页”，同时还对审查软件程序中添加了会自动生成黑名单标记的功能。
4. –2018年 8月 ，Guo Media 再次提交上架申请，并做了一些修改。应用程序评审员范昭表示，没有发现任何违反苹果规定的内容， 8 月 2 日他批准了Guo Media App在中共国上架
5. –2018年 9月，苹果的调查报告称：Guo Media App 上架，是因为没有遵循“中共国隐蔽步骤”；苹果管理员回答说：这款应用批评了中共政府，这足以拒绝其上架。
6. –6个月后，苹果解雇了范昭。


### **苹果向北京低头由来已久**

苹果为配合CCP 2017年施行的“网络安全法”，同意将中国用户数据数据資料存进CCP国营企业管理的服务器。

位于贵州省贵阳市的资料中心预计6月完工，苹果也在内蒙古自治区建造新的资料中心。据报道，苹果在中国的资料中心实际上完全由CCP政府人员控制。苹果同意将能解锁中国用户個人数据資料的数字密钥存在这些资料中心，并在中国禁用后放弃苹果在其他资料中心使用的加密技术。

[!\[\]()!\[\](https://spark.adobe.com/page/oSsIX1IyMNzIA/images/cca5c05d-549a-47e6-a7f9-8c24d3981d55.png?asset_id=c7d78587-016b-48b0-9076-90140fd1e9dd&amp;img_etag=%22d29352925df0d6aa6b9253fc647b7122%22&amp;size=1024)](https://spark.adobe.com/page/oSsIX1IyMNzIA/images/cca5c05d-549a-47e6-a7f9-8c24d3981d55.png?asset_id=c7d78587-016b-48b0-9076-90140fd1e9dd&amp;img_etag=%22d29352925df0d6aa6b9253fc647b7122%22&amp;size=1024)

1. 2017年苹果同意icloud 转移到中共国（中共完全控制服务器硬件），但是对密钥的存放位置，是放在中共国服务上还是放在美国的服务器上，产生了分歧。
2. 2017年 6 月苹果与中共官员达成协议，在密钥放在哪里的问题上故意模糊处理。
3. 2018年2月，8个月后 icloud 密钥送被送到中共国，这让苹果至少两名参与谈判的苹果高管感到惊讶。
4. 苹果icloud 交由“云上贵州”运营，规避了美国法律的风险。苹果员工称，苹果在中共国使用的任何加密技术都必须得到中共国的批准。两名苹果工程师说这个措施（隔离中共国的icloud 服务器）是为了防止中共国的安全漏洞蔓延到苹果的其他数据中心。
