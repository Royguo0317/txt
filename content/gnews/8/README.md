## 中国警告要与美国进行“核对决”
` XINYUAN` [轉載自GNews](https://gnews.org/zh-hans/1318879/)

作者：Ben Graham

编译：澳喜农场—XINYUAN
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/13031222/image3-2.jpeg)
中国官方报纸《环球时报》的编辑胡锡进表示，目前，加强中国的核进程对于中国对美国的“战略威慑”至关重要。

在美国总统乔拜登呼吁对冠状病毒大流行的起源进行进一步调查之后不久，他发表了上述评论。

胡锡进在《环球时报》的一篇专栏文章中写道。 ““我们必须为中美之间的激烈对决做好准备，中国核弹头的数量必须达到让美国精英不寒而栗。以迫使他们考虑是否与应该与中国进行军事对抗”。

胡先生曾称澳大利亚为“粘在中国鞋上的口香糖”，他在微博社交媒体平台上的一系列帖子中加倍强调自己的战争口号。

“鉴于美国对中国的战略遏制力度不断加大，我想再次提醒，我们有许多迫在眉睫的任务要完成，但其中最重要的一项是继续快速增加核弹头和战略导弹的数量，例如像东风41那种具有超长范围和较高的生存能力的导弹。胡先生在一篇被中国人权活动家詹妮弗翻译的帖子中曾写道：“这是中国对美战略能力的基石。”

“我们必须为美中之间的高强度对决做好准备，届时大量的DF-41和JL-2和JL-3将成为支撑我们战略意愿的支柱。 “我们的核导弹数量一定要多到让美国精英一想到要与中国发生军事冲突的同时就会颤抖。”

“在这样的基础上，我们可以冷静、积极地处理与美国的分歧，并且避免各种擦枪走火。”

“随着美国对中国的抵制继续恶化。他们如果冒险，我们需要运用我们的实力，，迫使他们保持冷静，否则将面临无法承受的风险。”

几天前，美国总统拜登命令美国情报界调查 Covid-19 病毒是否首次出现在中国是来源于动物还是实验室泄漏？上述调查激起了中国的勃然大怒。

这一举动暗示，在等待调查世卫组织对这场大流行病（在全球造成超过350万人的死亡）是如何开始的这个问题上，人们越来越失去耐心了。

在世卫成员国正在进行的会议上，欧盟国家和其他一系列国家也强烈要求，下一步该组织要澄清并努力解开这个谜团，这看起来对避免未来的大流行是至关重要的。

2019 年底，在 Covid-19 首次在武汉爆发一年多之后，世卫组织才终于在 1 月向武汉派遣了一个独立的国际专家团队，以帮助调查大流行的起源。

但在 3 月下旬才发布的拖延已久的报告中，国际团队和中国同行没有得出确切的结论，而只是根据他们认为的可能性对一些假设进行了排名。
