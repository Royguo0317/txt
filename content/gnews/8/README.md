7月9日，日本東京新增224例中共病毒感染者，創造東京單日最高紀錄。從7月2日開始，東京的中共病毒感染人數幾乎每天超100人。東京現在的情況已經比上次實施緊急狀態前嚴重，引起了很多民眾的擔憂，但是日本內閣秘書長菅義偉卻說，目前沒有必要再次實施緊急狀態。

為遏止冠病疫情蔓延，日本政府在4月初宣布東京、大阪、神奈川、埼玉和千葉等幾個都道府縣進入為期一個月的緊急狀態，並於4月中把緊急狀態擴大到全國，直到5月中下旬疫情改善後才解除。日本網友也有說，雖然恢復經濟對大家很重要，但比起經濟大家更在乎健康。

**編者觀點**

中共病毒爆發初期，很多國家意識到了問題的嚴重性，於是紛紛縮減航班，關閉口岸。後來形勢進一步惡化，便開始始封城封國，對待病毒的態度可以說是謹小慎微，嚴防死守。但是很多國家並沒意識到，這次的病毒可不是普通的自然病毒，它並沒有隨著時間的推移而有絲毫弱化的趨勢。

封鎖了一段時間之後，經濟壓力越來越大，中小企業大批量倒閉，失業人口飆升，選民抱怨聲高……而政府為了扶持企業和救濟失業人口，投入大量資金，甚至動用國家戰略儲備金。隨著時間的繼續延長，政府意識到再這樣下去經濟會徹底崩潰，也會完全失去選民的信任，於是便出現了很多國家冒著病毒擴散的風險解封的現象。如果正好碰上大選，該國的執政黨還得營造抗疫成功的假象，全面解封，讓民眾情緒舒緩，以贏得選票。
