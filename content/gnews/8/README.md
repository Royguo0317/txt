**作者：左媛**

上文说到迟浩田对德国的不屑：历史没安排德国做“地球之王”，德国杀人不够，德国不如“我们民族优越”……他还会红口白牙说些什么黑话呢？今天咱们继续探讨。

“我们老祖宗给我们留下了两个最根本的传家宝，就是无神论和大一统。我们中华文化的奠基人孔老夫子就是创始人。”

“我们所以与德国不同，是因为我们是彻底的无神论……我们中华社会所以历来崇拜圣人，就是因为我们不崇拜任何神……为什么近来我们发决定，要进一步宣传无神论。如果让西方的有神论到中国来把我们掏空，如果让中国的老百姓都去听神的话、跟神走，还会有谁能老老实实地听我们话跟我们走？”

本文单说迟浩田鼓吹的“无神论”。

“无神论”是中华老祖宗留下的传家宝吗？这就是谎言！中华文化崇尚的不仅不是无神论，而且还是“多神论”。首先，盘古女娲就要打脸这个老共匪。“盘古开天地”“女娲造人创婚姻”是几千年口口相传的神话，直到今天还在民间流传，并且在官方的重要场合频频引用。中共也屡屡称呼“我们是炎黄子孙”，炎黄二帝不是神又是什么？且不说历代君王有浩浩荡荡的祭祀大典，中共官方也为炎帝黄帝“建档立案”，也为二帝举行盛大的祭拜仪式，为民族树立祖宗。夸父逐日，仓颉造字，尧舜禅让，大禹治水……无一不是中华文化的美好神话，这些不都是古老祖先创造的各种神的故事吗？笔者恨不得拿一部《山海经》啪啪啪扇中共的猪脸！中华文化流传的这些“神”还有很多很多：河神，山神，灶神……中共不知道这些神吗？当然知道，但他们向民众鼓吹“无神论”有他们的险恶目的。此为后话。

孔老夫子一句“子不语怪力乱神”，就可以把中华历史上数不胜数的各种“神”抹杀干净吗？何况孔夫子是不说“乱神”，而不是不说“神”。中共今天抬出孔夫子为它洗地，甚至满世界开遍“孔子学院”为它抢夺话语权输出专制文化，可是50年前，也是它中共兴师动众掘了孔夫子的祖坟，号召全民“打倒孔老二”，身体上把他挫骨扬灰，名誉上把他“批倒批臭”。孔夫子不过是历史上一个读书人教书人，一个有政治理想最后却不得不自嘲为“丧家犬”的失意人，自己也没有一纸半页留下来。笔者想，孔子奔波一生，不留只言片语于后世，大概也是知道自己经历坎坷思想复杂，那就毁誉由人吧，就像后世武则天的“无字碑”一样。那部无数人评说的《论语》，不过是他的几个门徒凭记忆记录下自己先生的一些言行教训故事而已，里面有多少增删失误甚至篡改，到目前为止实在是无从考证。就这么一个客观的人这么一部群写的书，中共却由自己的阶段性利益而对他时毁时誉，可见孔夫子多么有先见之明！

迟浩田把孔夫子捧为中华文化的奠基人，为什么？所有的历史神话都是中华文化的基石，诗经离骚诸子百家，它们都是中华文化的基石，所有为它们付出代价的有名无名的祖先都是中华文化的奠基人，而不是只有孔老夫子。中共单捧孔子是有它的目的的，此是后话。

迟说，我们所以历来崇拜“圣人”，是因为我们不崇拜任何神。事实上，我们中国人既崇拜“圣人”也崇拜各种“神”，孔夫子就被后世誉为“文圣”。中共为什么要睁眼说瞎话又如此厚此薄彼呢？因为它有一个最终目标：打造“共产党”这个史无前例的“神”！不允许中国人崇拜任何神，只允许崇拜共产党！听听它们为自己创作的逼迫全民唱了几十年的红歌：“东方红太阳升，中国出了个毛泽东。他为人民谋幸福，他是人民的大救星。……共产党像太阳，照到哪里哪里亮。哪里有了共产党，哪里人民得解放。”延安窑洞毛成功从党内对手夺权，于是就产生了这首改编于陕北情歌的最著名红歌《东方红》，从此，毛泽东就被歌颂成了中国人的“圣人”（大救星），共产党就被成功设计成了中国人的“神”（太阳）。哪怕斗死几十万杀死几百万饿死几千万，也要唱“毛泽东是大救星共产党像红太阳”。毛死后，大部分中国人能吃饱穿暖，更不断有人能走出国门，看到外面美丽新世界和共产党宣传的有天壤之别，与毛时代相比更有云泥之别。党的神圣地位在铁的现实面前逐渐被动摇了。共产党意识到了这个巨大的压力和挑战，于是要进一步宣传“无神论”，进一步控制人心。他话说得毫不掩饰：大家都听神的话跟神走，还会有谁老老实实听我们话跟我们走？他们宣传无神论，打压基督教，焚烧十字架，抓捕基督徒……都不过是为了赤裸裸地抢救“共产党”这个江河日下的骗子，强迫十四亿中国人只能追随它。所以，今天中国大陆的大街小巷到处张贴着“听党话跟党走感党恩”的彩色宣传标语，就像一个垂死的病人在ICU里拼命打强心针输氧气，吊着那一口气不闭眼，纯粹是浪费人力物力财力，当然，即使华佗再世也给它续命不了几天。

世界有一神论宗教，那是信奉“上帝”；中共有一神论宗教，那是信奉“中共”。这就是世界和中共的本质区别。今天更有新红歌欺骗国人：打天下坐江山，一心为了老百姓的苦乐酸甜。这是习大神的夫人彭丽媛女士亲自演唱的红歌，不过是新版的《东方红》。中共宣讲的“无神论”是欺骗是谎言，目的就是要树立“共产党”这个唯一的“神”。当今，中共极力把习近平这个“初中博士”捧上神坛，集党政军三权于一身还不满足，还兼任各种各样的“组长”，还不断给世界指明各种方向，白皮《红宝书》发放到世界多国，自吹自擂被多国领导捧为圣旨……自己歌颂自己 + 强迫百姓歌颂自己 + 吹嘘外国人歌颂自己，简直是刷新古今中外不要脸记录。但这就是中共要的“一神论”效果！

当大家都看清中共的这个本质，也就不再上当。当欧美看清中共不仅要挑战欧美的人，还要取代欧美人信奉的“上帝”的时候，他们联合起来对抗中共的日子也就近了！当中国人一天天认识中共谎言➕暴力的性质时，中国人灭共的力量就无比强大！
