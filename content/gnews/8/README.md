## 甲骨文確認與TikTok 擁有者「字節跳動」達成協定
`2020-09-16 02:44 GM77` [轉載自GNews](https://gnews.org/zh-hant/360060/)

編者:Jordan Novet、文佑(荊棘不鳥)、文非、文肯尼
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/16024237/%E6%88%AA%E5%B1%8F2020-09-16-%E4%B8%8B%E5%8D%882.37.17-3.png)
**關鍵點**

**1)****這筆交易可以使甲骨文具有更強的推廣力量****,****並使其與年輕消費者更加相關。**

**2)****該交易仍需美國政府批准。**

甲骨文週一證實,在川普政府下令在美國剝離TikTok的幾周後,甲骨文已與熱門影片分享應用TikTok的中國擁有者「字節跳動」達成協定。

**該交易仍需美國政府批准**

“甲骨文公司確認了姆努欽部長的聲明,該聲明是週末「字節跳動」向財政部提交的提案的一部分,甲骨文公司將作為可信賴的技術供應商「甲骨文公司週一表示。

早些時候,姆努欽在CNBC上表示,美國政府計劃在本周審查這項交易。 我只想**說****,****從我們的立場來看****,****我們需要確保代碼****,****首先它是安全的****,****美國人的數據是安全的****,****手機是安全的****,****我們將在未來幾天與甲骨文的技術團隊進行討論****,****“**他在 “Squawk Box “上說。

白宮以擔心美國公民的個人數據會被分享給中國政府為由,曾給字節跳動規定了9月20日的最後期限,要求字節跳動在9月29日前宣佈在美國出售TikTok的計畫,否則將被禁止。 該交易必須在11月12日之前完成。  TikTok曾表示,中國政府沒有獲得用戶數據。

“我們可以確認,我們已經向財政部提交了一份提案,我們相信該提案將解決政府當局的安全問題,”TikTok在一份聲明中說。  「這項提案將使我們能夠繼續支持我們在美國的1億人的社區,他們喜歡TikTok的互動和娛樂,還有數十萬小企業主和創作者,他們依靠TikTok來發展他們的生活並建立有意義的職業。 “

**根據這筆交易的結果****,****它可能會促進甲骨文的雲業務。它可能會讓甲骨文在年輕消費者中更加引人注目****,****並使其在這個著名的區間有廣告控制力。**

在雲端運算和資料庫軟體方面與甲骨文競爭的微軟也對TikTok的美國業務感興趣。 但微軟的競標被拒絕,該公司周日宣佈退出這一進程。 微軟8月2日表示,一直在就收購TikTok的美國業務進行談判,沃爾瑪也曾因TikTok的電商潛力而對其感興趣,並在上周與軟銀和谷歌母公司Alphabet的聯盟關係破裂後,選擇與微軟聯手。 周日,沃爾瑪表示仍想投資TikTok。

據《紐約時報》8月報導,TikTok曾表示,它在美國擁有超過1億的月活躍使用者,其中幾百萬使用者都是十幾歲的孩子。 根據產業研究公司Comscore的數據,12月份英國TikTok使用者中最大的年齡段是18-24歲。

作為一款娛樂應用,TikTok在甲骨文的其他應用中脫穎而出,因為甲骨文只是向大型企業、學校和政府機構銷售軟體。 不過,甲骨文和TikTok在行銷方面有著共同的興趣。  TikTok通過向使用者展示廣告獲得收入,而甲骨文則提供軟體,行銷人員可以用它來管理臉書和其他管道的廣告發佈。  TikTok的使用者增加很快,而甲骨文在最近的8個季度中,有4個季度的收入都在下降。

作為TikTok的技術供應商,可能意味著甲骨文將提供其雲端基礎設施供該應用使用。 這將給甲骨文帶來另一個他們高管可以吹噓的熱門屬性,作為雲客戶,它可以吸引更多的雲業務。 上周在甲骨文與分析師的財報電話會議上,董事長拉裡-埃裡森(Larry Ellison)多次提到視頻通話服務商Zoom對甲骨文公共雲的使用。 根據技術研究公司Gartner的數據,甲骨文並不在2019年按收入計算的前五大雲基礎設施供應商之列。

收購説明甲骨文擴大了其產品群組。 2010年收購「Sun」使甲骨文成為數據中心伺服器的著名銷售商,2014年收購Micros後,甲骨文進入了POS機設備市場。 甲骨文在2016年以91億美元收購雲軟體公司NetSuite,是其43年歷史上最大的交易。

**甲骨文與美國政府保持著聯繫。**首席執行官薩夫拉-卡茨是川普過渡團隊的成員,埃裡森今年早些時候在加州的一處住宅為川普的競選活動舉辦了籌款活動。 它還與民主黨人有聯繫:中情局前局長、奧巴馬總統時期的國防部部長萊昂· 。 帕內塔(Leon Panetta)是甲骨文董事會成員。

字節跳動成立於2012年,當年就推出了新聞應用「今日頭條」。 它在2017年推出了針對中國以外使用者的TikTok,此前,它還為中國人帶來了一款類似的應用,名為「抖音」。 同樣是在2017年,字節跳動收購了一款單獨的影片分享應用Musical.ly,這款應用在美國已經有了大量粉絲。 去年,美國外國投資委員會就Musical.ly的交易聯繫了字節跳動,因為擔心中國政府獲取用戶數據的能力。

字節跳動表示,它將遵守中國政府更新的出口限制,其中涉及”基於數據分析的個人化資訊服務推薦”等內容。  TikTok依靠推薦系統來決定應該向使用者展示哪些視頻。

**編者觀點****:**

字節跳動是中共擴展其權力的關鍵白手套之一,其按照中共的要求肆意搜集消費者的個人資訊,並且利用特有演演演算法推送視頻,其中不乏激化社會矛盾和虛假資訊。 TikTok是字節跳動旗下重要的一環,相當於國外版本的抖音,其作惡多端,是中共對美超限戰的一部分,因此美國勢必要對其發動攻勢,在最短時間內要求其關閉或出售,及時止損。
