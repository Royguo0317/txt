澎湃网11月9日报道，暴风集团公告称公司股票将于11月10日摘牌退市，截至9日收盘，股价0.28元/股，市值9226.69万元，这个曾经最高市值达400多亿元的妖股终于落下帷幕。

暴风集团股份有限公司成立于2007年1月，注册资本3.24亿元，主要经营互联网信息服务业务。15年在创业板上市，借A股市场疯涨大势，及暴风集团自带“互联网基因”，上市后前41个交易日收获37个涨停（其中一字涨停28个），股价最高达到327.01元，市值最高达400多亿元，股票市盈率过千倍 。

2016年2月，暴风集团进军体育产业，与光大资本分别出资2亿、6000万元成立浸鑫基金，并募资52.03亿元用于收购MP&Silva Holding S.A.股权65%股份（MPS曾拥有意甲、英超、法网等多项热门体育赛事版权，但因经营不善于2018年10月破产清算）。2019年7月，公司实际控制人冯鑫因在上述融资过程中涉嫌行贿被上海检察院批捕。

暴风集团在迅速扩张的同时，债务累积、资金运转困难，在2018年上半年净利润已负（-1.06亿元），2019年上半年净利润更是降至-2.64亿元。2019年前三季度营收仅0.936亿元，同比下滑90.95%，净资产为-6.5亿元，同比下滑184.50%，若不能将年末净资产转正则会被暂停上市。

受冯鑫被批捕、高管辞职影响，暴风集团进入“无人掌舵”境地，业务几近停滞，员工薪酬支付愈发困难。19年11月因拖欠机房服务器托管费用，合作方终止提供服务，暴风网站和手机APP均无法运营。2020年5月，证监会以涉嫌信息披露违法违规对暴风集团立案侦查。暴风集团最新市值跌破1亿，在A股市场市值垫底，累计跌幅92.8%，堪称A股史上最惨跌幅。

截至当前暴风集团被法院列为被执行人16次，总额高达1075万元，法律风险破千（含169条限制高消费、20条股权冻结等）。在经历30个交易日的退市整理期后，公司股票终止上市，并将转入全国中小企业股份转让系统进行股份转让。

暴风集团从400亿元市值神话到黯然退市，虽有创始人冯鑫入狱、公司净资产无法转正、业务停止等因素，但归结起来，最主要的原因还是以下3点：

1、暴风集团上市后虽市盈率极好，股票市值等增长迅猛，但由于积淀较浅，无论是公司规模和运营范围均存在根基不稳的现象，但却在1年后仓促大额募集资本进军体育行业，其后更是由于MPS的没落导致投资失利，债务飙升，公司法人被逮捕，其余高管被迫离职，公司彻底停摆。

2、暴风集团战略布局过大，在PC互联网市场形成竞争力后，一下子涉足互联网电视（暴风TV）、VR（暴风魔镜）、体育（暴风体育）等多个领域，不但偏离了公司主营业务，更由于投资过多、资金负担过重造成资金周转日益困难，债务过快累积，最终导致资不抵债。

3、暴风集团的成功始于暴风影音这一PC端播放器的普及，但由于其核心技术缺乏创新，虽在PC互联网市场份额逐步加大，但由于缺乏核心驱动，终究只能被迭代换新的市场摒弃。

暴风集团的成功沾了A股疯涨的福利，但也正是由于中共虚拟经济的虚假臃肿、份额过大造成经济失衡，实体经济严重受挫，导致上市参股企业频频受挫。加上暴风集团本身根基虚浮，不但市场竞争力日渐低下，更因圈钱套现导致债务高筑、资不抵债，最终只能破产清算。暴风集团的黯然退市离场也是中共经济的一个缩影：利用偷盗技术、压榨人力换来的虚假繁荣，在技术受牵制、外贸遭打压、人民负担加重越发反抗之下，只能崩溃瓦解。
