第十九天对纽约SEC抗议，我们聚集在大楼的侧面。又是一个大阴天，微风裹着阵阵乌云从左手边的哈德逊河方向飘来，也给我们带来了点点细雨。

早晨的纽约交通繁忙，车流如潮水，比起依靠信号灯，往往需要靠人工指挥交通。为了方便需要停车的抗议者，我们一般会早些回合——在纽约，九点之前停车会便宜一点。而在我们的回合地，几乎每天都停着警车和往来巡逻的警察。不过，他们大多数情况并不和我们交谈，只是确认我们这里的安全性。

今早参与工作的战友是招财猫和金牛大哥：他们两位在十字路口发传单以及清理周围环境中的垃圾。

大多数情况下，周遭的行人都会接受战友们发送的传单。纽约的人蛮open的，他们喜欢探求。尤其在他们身边、眼皮底下有这样一群人，他们会想了解这群人在做什么，有什么样的诉求，为什么要站在这样一个繁华的金融中心，甚至政府部门的楼下。毕竟，我们的横幅里都包含着SEC或者CCP——CCP可能有些美国人不一定知道，但SEC他们肯定不陌生。据我观察，每十个人中几乎有一半会接受我们的传单。这个数字其实很大，因为每个人的背后还有无数的人际关系网络。而这样一个关系网意味着信息会传得很快，那我们给行人发传单的意义就很大了。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/14164047/20-6.jpg)

昨天也提到过，我们目前没有看到任何其他媒体到这里采访、报道，但我们GTV平台有自己的新闻和评论来揭露真相——这是我们GTV的宗旨，也是我们GTV的责任、追求和信仰：报道最新发生的事件，还原事件真实的情况。更何况，这次抗议是我们GTV投资者亲身经历的事情，我们责无旁贷。

前来抗议的GTV投资者们日复一日地坚守，却不为享受什么，亦不与利益挂钩——绝对没有——他们就是为了正义，为了信念。他们也并不仅仅为了自己，为了这几个人，或是这一个小群体，他们的心声代表了其他GTV投资者的心声，也代表了万千同胞共同的诉求。哪怕很多战友无法亲临现场，但他们的心声和诉求我们都知道，所以我们的口号、我们的标语、我们的诉求也是他们的心声。为此，我们来到这里的战友放下了家庭、工作、生意，甚至把身体的辛苦、疲惫、不适都放下了，靠着自己的意志，经受着这次的考验：每天朝九晚五，除了吃饭休息，一直坚守在各种恶略天气中。更不要说，很多战友非但在这里坚守了一个月，还要兼顾其他义工工作。

作为GTV的投资者，作为站在前线的战友，他们就是抱着这么一个信念：如果需要我们，我们就来。不论是需要我们站在前面，还是喊口号，或是做任何义工，我们都是义不容辞的。这些天的抗议过程中，接受采访，担任主持或摄影，处理现场突发事件，甚至有时需要与警察沟通，每一位战友都为此做好了准备。看似我们只是单纯的站在那里，但每一个人都在用自己的眼睛、耳朵、大脑，观察、记录、判断着周遭可能发生的一切，并随时准备应对——这是对我们在现场的每一位战友的考验，也是我们的责任。我们不只代表着我们自己或家庭，我们还代表着千千万万的投资者，更代表着无数的同胞。我们站在这里，带着考验与责任，更充满着信念和荣誉。为此，我们遵守着每一条现场抗议的规定，严格执行着警察提出的要求，我们保持着自己的素养、态度和形象，我们传递着我们一直追求的真相、诉求和信仰。
