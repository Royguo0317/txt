《路德时评》2021年4月26日晚间节目主要内容：

一、习声称要不惜血战坚持到底

法新社4月26日报道[\[1\]](https://www.rfi.fr/cn/%E4%B8%AD%E5%9B%BD/20210426-%E4%B9%A0%E8%BF%91%E5%B9%B3%E7%AA%81%E8%AE%BF%E7%BA%A2%E5%86%9B%E7%BA%AA%E5%BF%B5%E5%9B%AD-%E6%8F%90-%E4%B8%8D%E6%83%9C%E8%A1%80%E6%88%98-%E5%9D%9A%E6%8C%81%E4%B8%8B%E5%8E%BB)：昨日，习近平突访广西桂林全州县才湾镇红军湘江战役纪念园时表示，困难再大，“就想想红军长征，想想湘江血战”，透露出“不惜血战一场”摆脱困境的决心，和对取得“奇迹般胜利”的憧憬。外界观察习近平的困难，除美国压力外，近期对他的最大挑战，应该是今秋可能举行的中共19届7中全会。
![]()![](https://gnews.org/wp-content/uploads/2021/04/红军纪念园.jpg)[图片地址](https://s.rfi.fr/media/display/d60cbdb4-a6a8-11eb-a1b3-005056bff430/w:900/p:16x9/XiJinpingComemorerArm%C3%A9e%20rouge.webp)
路德社评论：

1、中共曾在湘江战役中死伤惨重，十万军队最后只剩下三万，当他们逃窜到遵义后，毛贼东通过党内斗争谋权上位成功。中共之所以在湘江战役中死伤惨重的最重要原因就是党内斗争激烈，当时党内其他势力对李德和博古的领导极度不满，中共领导人宁可让军队当炮灰也要拿下军队的控制权。

2、按照中共惯例，习前往广西肯定有其特殊目的，肯定要起到某种宣传作用，习近期面对来自美国和党内敌对势力的巨大压力，习的战狼外交和主动出击策略现在看来是失败的，明年他又面对权力交接的压力，可谓是内忧外患。

2、习之所以面临内忧外患，都是他推崇的战狼外交和主动出击策略带来的后果，中共独裁体制决定只有习一人说了算，等他下台后肯定得面临清算。对习神来说，现在最重要的就是延续他的独裁统治，党内各派势力对习在位以来的所作所为极度不满，在习神的领导下，中共国现在面临改革开放以来最严峻的国际形势，他又向世界发起了病毒战，未来肯定避免不了全世界向中共追责。

4、习神此时提及湘江血战，一方面他已经承认中共国现在面临的危机局势，另一方面也是想将责任推卸给前任领导人。现在的中共党内斗争就像当年的湘江战役，习神最终是成为下台的博古，还是成为夺权上位的毛贼东，仍未见分晓。

二、欧美内部报告对习独裁表示担忧

新闻一

大纪元4月27日讯[\[2\]](https://www.epochtimes.com/gb/21/4/26/n12907092.htm)：美国媒体“政治”（Politico）的欧洲网站本周一刊文说，欧盟高层内部报告显示，中共总书记习近平“转向独裁”令欧盟担忧能否分别看待欧中之间的商业利益和政治关切，欧盟对此已越来越感到悲观。
![]()![](https://gnews.org/wp-content/uploads/2021/04/欧盟议会大楼.jpg)[欧盟议会大楼](https://i.epochtimes.com/assets/uploads/2021/04/id12907158-1103280840022208_1-600x400.jpg)
新闻二

大纪元4月25日讯[\[3\]](https://www.epochtimes.com/gb/21/4/24/n12903045.htm)：中共近年来在国内与国际上变得专制和咄咄逼人，西方社会逐渐把其源头聚焦在中共党魁习近平身上。美智库报告分析说，习不指定接班人将出现四种后果：中共接班人惯例被习近平打破；习2022年交权或留任；习遭遇挑战或政变；习意外死亡或丧失行动能力。
![]()![](https://gnews.org/wp-content/uploads/2021/04/习结伴.jpeg)[图片地址](https://i.epochtimes.com/assets/uploads/2021/04/id12882934-6ad5e9301feaf8ff7e3e6deaa75cc169-600x400.jpeg)
路德社评论：

1、美国与欧洲都开始关注习神在接下来的十九届七中全会是否会交出政权。若习神不想交班，他是否能够压制住党内其他势力，如果习神交班，他的铁杆就会立马转向，新领导人上台后肯定要对习进行清算。

2、习神在位十年倒行逆施，得罪了大量党内官员，导致党内官员离心离德，让中共国陷入极为恶劣的环境。习身边的铁杆拍马屁水平虽然高，但是理政能力极差，他们也非常担心习神下台后会面临清算。所以，今年秋天是习神最后的机会，由于习的赌徒心态和底线思维模式，不排除会使用极端手段力保自己的皇位。

3、在中共的绞肉机体制下，领导人退位后如果没有实力必然要面临被清算的风险。习神在位十年以防腐为名拿下了一百多万党内官员，王岐山曾说他本人只是“报幕员”，一切都是习主席的领导。习神至今没有取得任何丰功伟绩，还将中美关系带入历史冰点，他身边也没有多少可信任的官员。习神虽然拿下了香港，却遭致西方的各种制裁，他试图拿下台湾，美日承诺将保卫台湾安全，这又让习神望而却步。习神现在只能从内部想办法，所以他就想通过“扶贫”和“血统江山”拿下政治制高点，为了让党内敌对势力屈服，习神必须采用非常强硬的手段，近期发生的阿里巴巴和美团事件就非常明显，习神已经开始从经济和政治两方面来强力清扫党内敌对势力。

4、十九大是党内斗争最激烈的时刻，也是党内其他家族势力生死存亡的时刻，所以各派一定会拼死一搏来延续自己的政治生命。
