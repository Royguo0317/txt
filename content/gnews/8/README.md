## 【文革 2.0】将席卷香港教育界 梁振英纠集「红卫兵」 批斗教育工作者
` GM42` [轉載自GNews](https://gnews.org/zh-hans/569223/)

编撰：文悦

覆核：文白
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/18002417/Untitled-1-23_ukePv_1200x0.png)图片：立场新闻
据立场新闻十一月十七日讯，前行政长官梁振英接受香港电台《视点31》直播访问，提到逾千名18岁以下青少年因反修例运动被捕，「15、16 岁青少年，知道多少反修例的来龙去脉？他们连投票权都没有，为何要拎汽油弹上街呢？」他认为相关行为「其中一个好可能的源头，就是学校内某些老师」。并称803 基金已接获不少家长报料，部分「耸人听闻」，「老师关上门，家长又不在，老师讲一些仇恨社会、反对国家的言论，好可怕」，称咨询法律意见后，不排除与家长合力控告相关学校。

梁振英表示，自己在任特首5年间，「没出现过去一年那么耸人听闻的事」，称自己的班子在很多范畴都迎难而上，而在占中事件中「看不到大量教师、倒有大量18 岁以下青少年被捕」，当时学生都是静坐，没有暴力行为。相反，他指出现时有中三、中四学生，被搜出可以做爆炸品的物料，社会应该问，为何少年人被「激进化」。

梁续指，学校老师很大机会是青少年「激进化」的源头之一，「我们一定要将源头找出来，不然不能正本清源」，强调教育并非为教师提供就业机会，问责对象应该是青少年及家长。他又要求全面检讨投诉老师的聆讯程序，指因循是错，「要看世界各地怎样处理老师失德，尤其在散播仇恨方面，是很严格的没有人好像香港这么做」。

教育界立法会议员兼教协副会长叶建源于脸书发文回应梁言论，称梁当年任命的的教育局局长吴克俭，「做足五年，是非常热衷旅游的局长，但在教育方面可谓一事无成，请问梁振英曾否负起疏忽监督的责任？」又指梁曾在任内以特首身分，阻挠商人捐款予本地大学，导致大学筹款困难。他续批评，梁振英轻忽和敌视教育的态度并非自今日开始，称「看不到迎难而上，只看到落井下石」

梁振英成立的「803 基金」已入禀司法覆核，要求教育局公开被裁定专业失当的老师和任教学校名单，梁振英说，要是教师被教育局查明专业失当，因而被谴责、被警告甚至被钉牌，教育局有责任告知所有家长及社会，强调「英、美等国都是这么做」。

他又提到近日本港学校的退学潮，称803基金一直收到不少家长报料，自己亦接见过家长群组，称当中有人将子女送到英国读书，插班，原因是去年开始的反修例风波，形容相关家长「不是因为子女离开就不关心」，向基金提供了很多「耸人听闻」的情报。

梁振英说，在适当时机取得法律意见后，不排除相关家长「已无后顾之忧」，届时基金将跟家长一起采取法律行动来控告学校，又强调情报来自「并非一个，是一群家长」。

提到大部分香港高官将子女送到外国读书、包括梁振英本人，是否不信任本港教育，梁称「有好多原因」，包括希望培养子女独立，正如有外国家长都会将子女送到寄宿学校，香港亦会收取很多外地来的学生，声称送走子女并不等于批判香港教育。

被问到现时社会撕裂、有意见认为比梁班子在任时更严重，尤其警民关系撕裂，梁振英反驳指「有什么撕裂得过消费都要分颜色？」批评有人在「暴动」期间，走去（个别商店）打砸抢放火，有没有人是建制（派）这边或政府这边的，去宣扬这些讯息？ 」并点名批评《苹果动新闻》宣扬激进化主义。

至于近日突然高调关心香港教育等议题，梁振英说在去年年底之前，自己主要在大湾区、内地各省市工作，惟眼见香港发生如此「十恶不赦」的事，作为香港人不能坐视不理；是否有机会再参选特首？梁称「目前没有」

战友点评:

不甘寂寞的「老领导」前特首梁振英，在不遗余力利用其「803」基金对「钉牌」教师和涉事学校穷追猛打后，还嫌自己搅和的不够，现在干脆炮轰所有十八岁以下参加「反送中」抗议的青少年是受他们学校老师的唆使蛊惑才上街，并作出激烈行为的。在梁的眼中老师不珍惜「工作机会」，煽动学生仇视国家社会。他还扬言要和那些「爆耸人听闻大料」的家长们一起状告相关学校。
