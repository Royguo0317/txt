## 布林肯呼吁世卫不惧中共 允许台湾加入世界卫生大会
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1185011/)

- 编译：坐看云起时

![]()![](https://gnews.org/wp-content/uploads/2021/05/taiwan-2132642_1920.jpg)
《国会山报》5月7日消息，美国国务卿安东尼-布林肯（Antony Blinken）周五呼吁世界卫生大会（WHA）允许台湾作为观察员参加本月晚些时候的年度会议，而这一倡议必定会激怒中共，因为中共一向认为台湾在其管辖之下。

布林肯在声明中说：”没有任何合理的理由将台湾继续排除在这个论坛之外，”他呼吁世界卫生组织（WHO）总干事谭德塞（Tedros Adhanom Ghebreyesus）邀请台湾参加会议，不应顾忌中共的抗议。

声明中指出， “全球卫生和全球卫生安全挑战与边界和政治争端无关。台湾从其处理这些问题的方法中提供了宝贵的贡献和经验教训，世卫组织领导层和所有负责任的国家应该认识到，在WHA会议上排除2400万人民的利益只会危及而不是推进我们共同的全球健康目标。”

布林肯支持台湾作为观察员参加WHA的声明是建立在国际呼吁和参众两院立法者的共识之上的，WHO的决策机构将在会议期间确定其年度全球卫生议程。

此前七国集团（G7）外长发表的公报便包含了支持台湾参与WHA。 美国参与的七国集团会议结束后于周三发表的公报中指出：”我们支持台湾有意义地参与世界卫生组织的论坛和世界卫生大会。国际社会应该能够从所有伙伴的经验中受益，包括台湾对解决COVID-19大流行病的成功贡献。”

中共一向反对台湾以观察员身份参加世卫组织或世界卫生大会，认为这违反了中共将其视为中共领土的 “一中原则“。中共外交部汪文彬于周四声称台湾参与的所有国际组织活动，均要按照中共所谓的“一中原则”来处理。

文章中指出，台湾被认为是最早对中共病毒大流行发出有意义警报的国家之一。台湾于2019年12月31日写信给世卫组织，称 “武汉至少有7例非典型肺炎”，并点名了首次报告疫情的中共国城市武汉。而中共当局的拖延报告疫情，从最轻的角度上来说，被视为未能满足国际上对疾病威胁的报告要求，从重的角度来说，则是在努力掩盖中共病毒的威胁。
