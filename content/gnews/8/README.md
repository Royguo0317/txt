DJHJ MEDIA 11月16日报道，代表川普总统进行集体诉讼的律师西德尼∙鲍威尔（Sidney Powell），在拉什∙林堡节目（Rush Limbaugh show ）中对主持人马克∙斯泰恩（ Mark Steyn）谈到了Dominion投票系统调查的震惊发现，同时告诉大家美国的选举的制表计数工作是如何在外国和外国人手中完成的。

西德尼∙鲍威尔（Sidney Powell）说：”我对这一切感到愤怒。我对腐败感到愤怒，中央情报局和联邦调查局对他们收到了投诉却没有采取任何行动。这一切让我更想知道谁得到了什么报酬，谁对这一切负责，而且得到了特殊的好处。”

鲍威尔（Powell）在推特上表达了她揭露Dominion投票系统在此次选举舞弊中所扮演角色的决心。

她在11月15日发表的推文中说道：

金钱和权力腐蚀了无数弱小或邪恶的人。

现在是所有善良的公民站起来的时候了，来帮助这个伟大的国家和我们所爱的自由.

#共产主义者沉迷于对共产主义国家的破坏，但他们在这里不会成功。

她在11月16日的推文中说：

选票实际上被送到了西班牙巴塞罗那（Barcelona, Spain）和德国法兰克福（ Frankfurt, Germany ），在那里他们可以被进一步修改操纵，然后再送回美联社和纽约时报进行报道。

在林堡节目中，鲍威尔（Powell）告诉马克-斯泰恩（Mark Steyn)：

“存在太多的问题。马克，现在很难列举出所有的问题。该系统（Dominion)是用委内瑞拉的金钱和利益专门创建和设计的，目的是用来操纵乌戈∙查韦斯（Hugo Chavez）和马杜罗（Maduro）的选举。据我所知，它被输出到国际上，用以操纵阿根廷的选举，并且它被用于改变（我们的）选举，使人们看起来投票给拜登先生，但川普以压倒性优势获胜。我开始收集证据时，证据像通过消防水喉般地涌出，而现在更像海啸般地涌来。诚实、爱国的美国公民都会站出来告诉我们到底发生了什么。我今天刚得到消息 100名Dominion公司的员工已经从他们的链接账户中删除了与Dominion公司的联系，而Dominion正在疯狂地清除人们的名字。”

“我的数学专家告诉我，算法正在被用于改换投票。它自己的操作手册就告诉你，他们能这么做。算法改成67%投拜登，选票以数十万相同的数量和相同比例多次注入这个数字，比如在威斯康星州注入三次，在密歇根州注入两次，或每相隔20分钟或采用其它手法。这很荒唐，那些说没有舞弊证据的人就是为了个人利益想掩盖舞弊事件的人。”

“现在问题是谁为他们的州买了这些Dominion系统而得到了特殊好处，我的意思是这种腐败的水平，美国人民很长一段时间前已经觉察到了，现在才有人站出来，他们意识到，我在这里，我会为之奋斗，直到我们把一切公布出来。”

“这些家伙，我不知道他们在谁的工资单上，都需要被解雇。”

“艾米∙克洛布查（Amy Klobuchar）和伊丽莎白∙沃伦(Elizabeth Warren)曾经抱怨过这件事，几年前卡罗琳∙马罗尼（Carolyn Maloney）也写过一封信，他们甚至在网上清除掉了他们引用的文章。我们去找链接时，它们已经不见了。在这点上我不怀疑涉及到科技公司和硅谷，虽然他们也在试图压制我们在所有这些问题上的言论自由，以掩盖他们自己的丑事。”

“我认为证据将是压倒性的。我要警告任何想要认证此次选举的州， 要认真地重新考虑它，因为他们认证的是他们自己的欺诈行为和他们自己在欺诈中的合谋行为。我以后甚至可能会提起集体诉讼，起诉他们参与其中。 这实在是太荒唐了。各州的立法机构现在需要控制并拒绝认证，特别是那些摇摆州，他们受到了数十万张选票变化的严重影响，智能技术手册中告诉你，他们可以做到这一点，他们可以改变任何他们想要改变的选票，并将任何ID从一张选票重新分配到另一张选票。他们可以成批次的选取选票，并销毁它们。投川普选票可以添加到拜登名下，他们能以任何方式操纵他们想操纵的一切，因为他们有VBN行动和访问权限。。Raheem Kassam昨天在推特上发布了一张照片，Glenn Simpson在Dominion投票机的屏幕后面看着一切。这台机器是为了委内瑞拉选举作弊而制造的。”

“它被设计成这么做，计数被送到西班牙巴塞罗那和德国法兰克福，他们可以在那里进一步操纵改动，然后送回美联社和纽约时报和其它报道。这次会被抓到的原因，是因为川普的领先优势太大，它计算的那个算法不够高明，所以他们才不得不在8点停止计数。”

“你可以称它们为委内瑞拉的机器，虽然它使用了几个不同的公司（包装）。我们有委内瑞拉共产党员在计票，古巴共产党员在计票，并决定我们的选举结果。这绝对是一种暴行。应该由我们最高层的情报调查人员进行调查，最好是军方，因为这是国家安全的威胁，这正是他们这么做的原因。别忘了中共对这一切的影响，这里面有来自中共国的资金。”

“机器改变算法的方法有很多。他们可以增减选票，改变选票的数量，如果他们仍不满意，他们还可以在巴塞罗那改换。他们现在在佐治亚州进行的手工计票，试图用它来说明Dominion投票软件舞弊是一个骗局，这是胡说八道。他们在那里做了各种不同的事情，包括关闭共和党人的账户，办理临时投票然后投票人消失无踪。我们必须进入机器本身，获得软件并检查它。但我们知道佐治亚州的软件故障，而且随时都在发生这样的故障这可能是因为软件在上传，因为他们可以通过互联网改变数据，而他们在此之前从未连接过互联网。补丁是在选举前一天放进去的，从来没有被认证过。在选举前，机器是不应该被触动的。这本身就是一种违法行为。”

”有人告诉我对佐治亚州州长有特别的顾虑，而州务卿也因为急于购买Dominion机器进入该州而获得了一些个人利益。这些机器很容易被黑客入侵，15岁的孩子都能做到。”

“如果他（上议院首席大法官约翰∙罗伯茨）不接受（Dominion案），他应该被弹劾。我觉得非常乐观，真相会被揭露出来。地球上的每个人都在压制这个故事。我将释放大杀招。”

Gov.Tech.com 2019年7月30日的文章中，报道了佐治亚州将1.07亿美元的投票机合同授予Dominion。文中说道：”佐治亚州官员周一将3万台新投票机的合同授予了Dominion Voting公司，取消了该州17年的电子投票设备，取而代之的是打印纸质选票的触摸屏。”
