川普总统12月2日在白宫发表了长达46分钟的演讲，宣布这可能是他最重要的一次演讲，阐述了选举舞弊的猖獗以及他维护宪法及人民投票权利的坚定决心。

DJHJ媒体 12月2日报道了此演讲节选，川普总统在演讲中表示：

“我们需要对选票进行系统分析，这关系到选举和美国选民的权利。我们需要统计所有合法选票。在美国人民的帮助下，将恢复选举制度的完整性。”

“我们本来赢了很多，但是他们灌注选票，我们落后了一点。” 川普总统展示了不同摇摆州在不正常时间段里出现大量灌注选票的统计图形。

“我们不被允许匹配签名，有很多票数不正常，对我不利。在重新计票时，需要匹配签名。我们需要比对过去选举的签名，会发现有很多非法签名。他们利用疫情作掩护，邮寄出了太多选票，而这些选票都寄给了死人。没有人比中共国看到这种情况更高兴了。结果一定要推翻，有人说太苛刻了。我们必须把选举结果翻盘，这是一场灾难。我们会把它展示出来–希望法院也能看到，并做正确的事情。我们不能再忍受这种选举了。看看过去吧 也许会重新投票，但我不认为那是合适的。我们不是要向你们展示一小部分选票，而是要给你们看几十万张票，比法律规定的还要多。媒体不会报道，因为他们知道结果，甚至我说的这个会被贬低，但我代表的是7400万人民，代表的是全国人民。邮寄选票是腐败的。为了重新上台，他们什么都会做，什么都会说。”

“我们把美国放在第一位，但他们只想为自己争取权力，只想赚钱。这就是为什么他们不想要我，因为我正在把权力还给你们。（他们用）弹劾的骗局和百万美元和大规模调查，想把我推翻，根本就没有（他们所编造的）勾结。我听说，那些在华盛顿做了失败努力的人现在正试图在纽约陷害我。纽约检察长创立了一个平台，一直在试图通过调查拉倒我。他们想打倒的不是我，他们想打倒的是我们。”

“这个政府中的一些人，幸运的是，并不是所有的人都被打倒了、被贬低了，他们只是消失了。人们失踪了，被追捕和骚扰。我们抓住了科米（Comey ）和麦凯布（McCabe）， 我们正在等着一个我从来没有见过的，名叫达勒姆（Durham）的人递交的报告。他说过，他将提交一份报告， 但我们还没有看到。对我来说最难的就是对做事的人们解释为什么任何事情都没有做。证据是压倒性的，但什么都没有做。”

川普总统同时也在推文中发布了他的演讲视频。在演讲中他坚定地宣告，作为一名总统，他的最高职责就是维护宪法的完整性。[【1】](https://twitter.com/realDonaldTrump/status/1334240039639937026?s=20)
