## G系列正在搭上数字货币进入千家万户的顺风车
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1013429/)

翻译&评论: Jony(8 Mile)
![]()![](https://gnews.org/wp-content/uploads/2021/03/屏幕截图-2021-03-24-221742-2.png)图片来自网络截图
自从比特币上市已有12余载，这12年来比特币既承载了开山鼻祖的光环，也面临着一系列的漏洞，包括缺乏安全性、涉嫌偷税洗钱、涉嫌非法交易等等。然而，这12年里，与万众期待相反的是，数字货币并没有在安全和技术上有实质性的创新和突破，区块链应用也并没有想象中的在商业上百花齐放、落地生根。

步入2021年，这一切似乎有了新的转机，文贵先生已经多次在直播中透露美国当局对数字货币发展和监管的态度和规划。在美国、在澳洲已经越来越常见标记着可以通过比特币进行提现或汇兑的提款机。那么，什么样的数字货币能在这一轮重新洗牌中一跃而起呢？通过文贵先生对G系列数字货币的规划，我们已经找到了答案。

首先，是安全性，必须具备银行级别的安全性，包括资金安全和信息安全。其次，是交易量，100年1000亿的总量满足了数字货币作为交易货币所需要的数量和可得性。第三，是消费支付的生态圈，即可以同时实现线上的虚拟支付和线下的实物支付。第四，背后20%的黄金储备保证了币值的相对稳定。第五，公母币的设计，其中稳定币可以吸纳存款、开展银行金融服务，增强用户粘度。第六，严格的KYC，杜绝洗钱等非法交易。

以上六点是笔者认为G系列数字货币现阶段最为独特的优势。在不远的将来，当G系列数字货币被市场认可，能够被纳入一揽子数字货币，那么，G系列“以钱灭共”所造成的蝴蝶效应必然会摧垮中共的虚假泡沫和腐烂政权。

看看这篇题为《比特币ATM机正在登陆人们身边的加油站熟食店和便利店》文章为我们打开了一个全新的视野。

在蒙大拿州、卡罗莱纳州和纽约市等地的一些熟食店和加油站，比特币ATM机不断增多的趋势已经开始显现。据路透社报道，CoinFlip和Coin Cloud等公司已经在全国各地安装了 “数千台” 自动取款机。Coin Cloud在美国各地已经拥有1470台取款机，其目标是到2021年底达到1万台。

Quad Coin的创始人Mark Shoiket说：”我只是假设有这样的需求，假设人们希望比特币无处不在。” 他最近刚刚飞往蒙大拿州，找到了7个安装比特币ATM机的新地址，其中包括当地的一家烟具店。

目前美国有28185台比特币ATM机。报告指出，在过去5个月里，就约有1万台刚刚安装。有些人使用自动取款机是因为他们有时觉得与实体机器互动更舒服，而其他人则是出于匿名的目的。取款机费用从总交易额的6%到20%不等。

现在，除了阿拉斯加之外，每个州都有比特币ATM机。

51岁的匹兹堡居民丽贝卡-怀特（Rebecca White）说。”假如我们买完菜后，还剩60美元，我会在比特币ATM机那里停下来。”

加密货币合规公司CipherTrace的金融调查和教育总监Pamela Clegg说：”ATM市场的增长 — 这甚至不是一个温和的增长，它几乎是45%的增长。增长幅度相当惊人。”

但监管机构一直在密切关注这些新出现的机器。例如，新泽西州调查委员会最近发表了一份名为 “加密货币站点的诈骗、可疑交易和可疑做法” 的报告。

Coin Cloud 首席执行官Chris McAlary表示：”我们预计最坏的情况是冠状病毒的冲击，但财政刺激出来了，这帮了不少忙。一些人拿着刺激补助，然后购买了数字货币。”

CoinFlip首席执行官Daniel Polotsky说：”有些人没有银行账户或者不喜欢使用银行账户。”

来自亚特兰大的Bitcoin Depot在过去的一年里，其ATM机的数量也从500台增加到1800多台。客户大多是 “25-40岁” 的人。比特币ATM机制造商General Bytes表示，需求 “飙升”，公司去年的库存已经用完。他们去年卖出了3000台机器。报道称，其中90%的机器流向了北美。
