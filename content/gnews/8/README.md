## 【澳媒观点】中共国“超速运转”的疫苗接种计划

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/25035217/20210525-3-Cover.jpg)（来自网络截图）
澳洲媒体《悉尼晨报》引用新加坡媒体数据，中共国计划向全国提供“前所未有规模”的中共病毒疫苗，每天将为“超过2000万人”接种疫苗。澳洲媒体称，这个数字等于“每天向全澳大利亚所有合格疫苗人群提供一剂疫苗”。按照这个速度，中共国会很快“赶上欧美”。澳洲的媒体观点认为，中共国当局将有“更大的灵活性”，促进其可能将数亿剂疫苗分发到其他发展中国家。

文章引用牛津大学汇编的数据显示，中共国的中共病毒疫苗接种“加速增长始于3月26日”。而同时期，中共国和世界其他地区“并驾齐驱”，即“每100人中有6.77人接种了疫苗”。在那之后，中共国的中共病毒疫苗接种“进入超速档”。最近的数据显示，中共国已经达到“每100人接种了34.55疫苗”，而世界其他地区为每100人的接种覆盖为“21.6”。也就是说，中共国“在两个月内增加了415％”。

文章继续指出，中共国曾经出现接种在同一天达到“创纪录的1700万剂”。可是，即使按此超常规的操作，预计到6月份，中共国仍然“无法实现其14亿人口的40％接种疫苗的目标”。文章称，在美国，英国和欧盟的人均数量上，三者“均受到去年疫情的打击”，接种上也“仍然保持领先”。可是，中共国“正在迅速赶上”，原因是中共国敦促公众“为国家的利益而接种疫苗”。同时，在中共国内不断出现的“小规模的地方性疾病暴发”使得很多墙国人“摆脱了疫苗的犹豫”。

文章提到，中共国认为，中共致力于早日打开大门，而“目前国外的大流行情况仍然非常严重”。因此，中国需要“尽快建立群免疫力”。为达到这个目标，很多地方政府以“食品和代金券”，“精巧的广告活动”等作为中共病毒疫苗接种的诱惑。各种宣传手段都为了强调和保证“疫苗的安全性”；同时，在舆论上不断上演和强调世界各地都在“争抢中共国疫苗”的紧迫形势。

文章指出，中共国一直“热衷于在获得世界卫生组织紧急批准之前和之后向国内外观众推广其疫苗的功效”。中共国现在已经“生产了超过6亿剂”相关的中共病毒疫苗；同时，中共国已经“在亚洲，南美和中东运送了超过2.65亿剂”。而与之相对照，科技和研发疫苗都领先的美国，迄今为止，才“已向世界其他地区承诺投放8000万剂”。

文章谈到，虽然国药集团“已获得世界卫生组织的批准”，可是“仍在接受紧急使用审查”。而在智利，在其所得到的中共国的“1050万剂Sinovac”，4月份时发现该疫苗的“有效率为65％”。

当然，文章引述中共方面的报道和相关人士称，中共国的中共病毒疫苗会继续提升其有效率。
