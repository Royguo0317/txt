**西班牙2021年3月13日电/西喜社——**据美联社3月13日报道，美国卫生官员正在警告卫生专业人员，有关中共病毒和流感的，一种广泛使用的实验室测试的假阳性结果风险。

食品和药物管理局周五因卫生设施使用罗氏的中共病毒和季节性流感的cobas测试发出警报。该机构警告说，该检测的处理管出现问题，可能会导致没有实际感染的人出现误诊。

罗氏的测试系统被广泛用于在医院和实验室筛选大批量的患者样本。

食品和药物管理局建议卫生工作者对样本进行多次检测，以帮助保证准确性。该机构表示，如果测试提供的结果相互矛盾，则可能表明存在问题，应停止使用。
