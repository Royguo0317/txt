9月9日，美国总统特朗普被挪威议员提名诺贝尔和平奖。此消息一出，立即成为了全球主流媒体讨论的焦点。与当年奥巴马被提名时美国无论是左派媒体还是右派媒体铺天盖地的讽刺和挖苦不同，此次特朗普总统被提名可谓是实至名归。那么，二者到底有何不同呢？笔者接下来将做一个简单的分析。

首先，我们从获奖时间上分析。奥巴马被提名诺贝尔和平奖的时间是2009年10月9日，此时奥巴马刚刚当选美国总统，在没有任何政绩的情况下就被提名诺贝尔和平奖，不经让人质疑到底是什么原因呢？最后诺贝尔奖委员会给出的官方理由是：奥巴马有推动人类和平的良好愿望！

亲爱的读者朋友们，当您知道是这个原因时有没有觉得很滑稽呢？只是有一个愿望而无任何实际成果的情况下就能获得诺贝尔和平奖，如果是这样的话笔者也能获得诺贝尔和平奖，我们每个人都可以获得诺贝尔和平奖，甚至就连习总加速师都能获得诺贝尔和平奖，毕竟人家至少嘴巴上天天挂着“人类命运共同体”。可想而知，奥巴马获得诺贝尔和平奖的真正原因绝对不是这个，而是纯粹的意识形态问题。奥巴马还没有上任总统时诺贝尔奖委员会就深知其骨子里就是一个反资本主义的社会主义分子，奥巴马的意识形态深受严重左倾的诺奖委员会的青睐，所以他们迫不及待地给了奥巴马这个奖。因为如果晚两年再给的话，奥巴马会凭借其极度无能的执政表现让诺贝尔奖委员会再也找不到任何塞给他这个“和平奖”的理由。

事实上也正如我们刚刚谈到的，奥巴马执政八年美国的经济一直不景气，失业率居高不下，国债翻了将近一倍。由于美国在中东地区的不作为导致无论是伊拉克还是叙利亚的局势长期处于极度混乱的状态。而美国派往伊拉克的驻军不减反增，每年凭空消耗美国几百亿美元的军费。同时，奥巴马政府和伊朗签署的“伊朗核协议”不仅没有遏制伊朗的核武器计划，反而更加变本加厉。此外，奥巴马在8年执政期间放任中共国在南中国海填海造陆，军事化各种岛礁，对印太地区的和平与稳定造成了极大的安全隐患。

对于美国国内，奥巴马这个社会主义分子强力推动美国“禁枪”令，企图剥夺美国人民捍卫自由和私有财产的权力。如果真的如他所愿，那么美国人民将饱受各种形式的“按提法”和“黑命贵”运动的摧残，当前正在美国上演的这些暴乱将提前几年发生。像奥巴马这样一个极左的无能的总统何德何能可以获得诺贝尔和平奖！估计诺贝尔先生要是听到这个消息得气得从棺材板里蹦出来！

反观特朗普总统，被提名诺贝尔和平奖的时间是其美国总统第一届任期即将结束的时候。特朗普总统在第一届任期的四年里先是大力发展了美国经济，同时解决了美国南部边境的难民危机，在提高了美国人民收入的同时又保护了美国人民的生命财产安全。

在国际上，特朗普总统先是通过贸易战极大地破坏了中国共产党企图控制美国经济乃至世界经济的企图，让中国共产党称霸世界的野心胎死腹中。同时，特朗普总统利用其高超的外交手段在不费一兵一卒的情况下就让以色列和阿联酋这两个世仇国家正式建交，二者的建交对整个中东地区的和平演变将起到极为深远的积极影响。仅凭这一点特朗普总统就能够获得诺贝尔和平奖！这是肉眼可见的成果而不是奥巴马的狗屁良好愿望，甩了奥巴马八十条街不止。

此外，在面对中国共产党制造并释放的生物武器“中共病毒”的攻击时，特朗普总统并没有逃避，而是亲力亲为地推荐能够有效预防并控制该病毒的“硫酸羟氯喹”药物，试图拯救更多的美国人民。在疫情追责方面也是极为坚定地要追求中共隐瞒并故意扩散疫情的责任。不仅如此，当前美国政府正在一步一步坐实中共制造生化武器的反人类罪行，一旦正式宣布罪行成立那么等待中共的将是全世界的围剿，中共必然会彻底被消灭！消灭中国共产党对于人类和平的贡献可以说在整个人类历史上也是无出其右的，这样的人才有资格获得诺贝尔和平奖！

最后笔者简单总结一下，特朗普总统执政四年为美国人民创造了财富，守护了美国人民的生命财产安全，推动了中东局势的和平演变，更是在坚定地与人类历史上最邪恶的中国共产党进行你死我活的战斗！他理应获得这个“诺贝尔和平奖”！而奥巴马执政八年，给美国带来的是翻倍的国债，居高不下的失业率，给世界留下的是混乱的中东局势，伊朗的核武器开发，俄罗斯吞并克里米亚，芬太尼大肆杀害美国青年等等，我们应该授予奥巴马一个“撒旦战争奖”！
