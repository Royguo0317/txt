## 《路德社》精选主题专栏：真正的科学家不会向强权低头
` 路德社精选` [轉載自GNews](https://gnews.org/zh-hans/1079267/)

《路德社》精选主题专栏组出品

![]()![](https://gnews.org/wp-content/uploads/2021/04/041201.png)

4月8日，中共外交部在新闻会上，点名美国前白宫国家安全委员会委员[杰米·梅茨](https://translate.google.com/website?sl=en&amp;tl=zh-CN&amp;u=https://jamiemetzl.com/)（[Jamie Metzl](https://translate.google.com/website?sl=en&amp;tl=zh-CN&amp;u=https://jamiemetzl.com/)），称他联合24名科学家，发表质疑世卫病毒溯源报告的公开信，是抹黑中共，将科学调查政治化的行为。可是，中共发言人故意忽略梅茨先生世卫咨询委员会成员身份，而强调其任职白宫的历史，本身就已将科学调查政治化了。

这24名科学家来自世界各地，其中有澳大利亚的流行病学教授，法国的病毒学家，动物学家，新西兰的数据科学家，印度的微生物学家，德国的遗传学家，及法国的生物信息学教授。他们曾在今年3月4日，世卫病毒溯源小组去中共国调查前，发表过第一篇公开信，质疑世界卫生组织（WHO）赋予中共否决权，让他们对入选调查小组的科学家具有否决权。而且最终敲定专家在技能上仅专注于公共卫生和人畜共患病，而缺乏[调查实验室或相关事故](https://translate.google.com/website?sl=en&amp;tl=zh-CN&amp;u=https://www.nature.com/articles/d41586-021-00375-7)的[法医技能](https://translate.google.com/website?sl=en&amp;tl=zh-CN&amp;u=https://www.nature.com/articles/d41586-021-00375-7)。最重要的是，世卫调查组没有获得授权，对所有病毒相关信息进行独立，全面，不受限制的调查，他们无法[访问详尽的实验室记录，](https://translate.google.com/website?sl=en&amp;tl=zh-CN&amp;u=https://www.wsj.com/articles/china-refuses-to-give-who-raw-data-on-early-covid-19-cases-11613150580)数据和人员情况，因此无法充分地评估各种起源假设。

他们在4月7日发表的第二封公开信里表示，“在阅读了世界卫生组织关于病毒起源的报告后，我们很遗憾地得出结论，我们之前的担忧是完全有道理的。”公开信指出，世卫报告对于为什么应该将“实验室相关事故”视为“极不可能”；或者为什么应将“通过未知动物宿主的自然溢出”视为“极有可能”，没有提供充分的依据。公开信指出报告“未能达到可靠的分析和评估的一些最基本的标准，指出其中12项不正确，有争议，自相矛盾或不准确的陈述”。

正是这样一批忠于科学事实，不畏强权，敢于提出质疑，关注人类生存危机的良心学者，被中共发言人污蔑为，对中共进行“有罪推定”的“同一批人“。那么与中共实验室合作16年，在调查开始前，就坚称该病毒并非来自实验室的达扎克们，他们是不是应该被称为，为中共进行“无罪推定”的“同一批人”？

科学推动着人类的进步，改变着人类文明进程。但是必须是真科学，而不是伪科学，就像英雄科学家闫丽梦博士，作为世卫组织参考实验室里，第一个受委派与中共北京疾控所联系的科学家，2020年1月19日通过路德社节目，爆料中共生物武器，从而打破了中共的全盘计划，被迫宣布中共病毒人传人，让世界各国政府有所准备，挽救了无数人的生命。天使科学家闫丽梦博士用自己的科学报告，改变了人类历史。

联合目录新闻网站4月9日，对梅茨等科学家的第二封公开信进行了报道，新闻说，中共政府并非公开透明，而是妄图全面控制所有关于大流行起源的研究。公开信呼吁对病毒起源重新调查，同时取消中共国或任何其他政府的否决权。这封信还要求保护举报人，以确保中共国科学家可以“共享相关信息而不必担心受到报复”。

对中共发言人的攻击，梅茨先生在推特回应，这并不会阻止我们，反而激励我们提出更尖锐的问题，这样才能避免下一次病毒的全球大流行。他表示，中共没有回答实质性的问题，而是选择“混淆和转移”来自公开信中提出的批评和担忧，这是“非常不幸的”。

24位科学家的公开信呼应了早前14个民主国家质疑世卫调查的声明，美国下周将出炉的2021战略竞争法案，也把彻查中共病毒起源列为重要一项。美日印澳，日韩同盟正在形成对中共的包围之势，中共这回是真的搬起石头砸了自己的脑袋了，一些还在观望，尚未选边站的国家，也将顺应民意，加入到这场以毒灭共的正义之战。
