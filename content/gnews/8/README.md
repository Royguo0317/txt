## 【世事解評】“環球”社評中俄關系，走夜路吹口哨

作者：紐約香草山健身部 Jack Li文陽

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/24215852/GNEW-GTV-MOS-LOGO-2-2-1-768x103-1.jpeg)



6月16日將舉行的美俄元首會晤，令中共惶恐不安。楊潔篪搶先一步，于5月25日前往莫斯科與俄羅斯安全會議秘書帕特魯舍夫會晤。《環球時報》急忙發社論，高調盛贊了中共與俄羅斯的關系，企圖對抗美歐聯盟，爲習神壯膽。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/26025526/%E5%9B%BE%E7%89%871-138.png)

環球說，中俄新時代全面戰略協作夥伴關系高度穩定和確定，“沒有止境、沒有禁區、沒有上限”，不僅處于中俄兩國關系的最好時期，而且達到了當今世界大國平等戰略合作的最高水平，沒有任何力量的和任何計謀可以削弱今天的中俄關系。

中共所謂的新時代，即病毒大流行的時代，也是全球正義力量聯合滅共的時代。這樣的新時代，俄羅斯和普京能看不清楚嗎？北約亞約重新集結，中共心焦如焚，環顧四周，最具實力的莫過于俄羅斯了。但俄羅斯本身也深受中共病毒的禍害，以俄羅斯的情報能力，病毒從何而來，普京心知肚明。全球的左中右三派，一致認定病毒非自然，力推追查病毒真相。只要證明病毒是中共的實驗室出來的，中共必死無疑。普京在下一盤大棋，在中共滅亡前，賺取最大的利益。對此，俄羅斯才是無止境、無禁區、無上限、無底線，這才是俄羅斯與習近平戰略合作的最高水平。普京暗自盤算，俄羅斯民族的偉大複興，就靠老習了。

環球自評中俄戰略關系，是美國和主要盟友擠壓的結果，但是中俄全面戰略協作遠遠超越了那樣的臨時地緣政治動力，經20幾年的不斷積累夯實已轉化成兩國各自國家戰略的基石，雙方背靠背的戰略倚重和利益交織成爲不斷升值的資産，這個世界根本就不存在可以交換它的等價物。

環球說得很正確，美國及其盟友的合圍圈越來越小，壓力越來越大。中共有何自信“背靠背”的俄羅斯不會從後面來一刀？這背後一刀，無論砍頭插腰，一定是滅共最快最狠的一刀。習近平的四個自信能無視這後顧之憂嗎？

環球說，奉行霸權主義的美國視中國和俄羅斯爲最大障礙。此言純屬胡言亂語，顛倒黑白。其真相是企圖奪取世界霸權的中共，被重新認清其面目的西方正義聯盟圍追堵截，中共拼命拉上俄羅斯，與西方爲敵，負隅頑抗。

中共說中俄不會結盟，此言不虛。中共倒是很希望與俄羅斯結盟，但俄羅斯絕不會與中共簽任何同盟協定，否則，全球滅共時將無法脫身。反觀美日、美韓都簽訂了針對中共的聯合聲明，中共所謂的與俄羅斯的全面戰略合作關系顯得多麽的蒼白無力，中共叫得再響，還不如美日美韓一紙聯合聲明實在。

環球顛倒黑白：美國和西方的同盟關系是美國用實力主導維持的，一旦美國的實力減弱，它們之間的這種關系就可能松動。在全球化讓國家關系遠遠超越地緣政治單一維度的時代，從長遠看，美西方將難維持“鐵板一塊”。中俄關系的最大價值是平等互利，結伴不結盟創造的不僅是彈性，更是韌性。這樣的價值和原則將被時間證明比霸權主義盟友體系更具生命力。

不知中共是糊塗還是無知，西方和美國的關系，是建立在相同的信仰和價值觀基礎上的。而中共和俄羅斯的關系，才是建立在利益上的。一旦沒有利益，俄羅斯立馬變臉，每次請俄羅斯出場站台，不知都要付出多少民脂民膏？中共居然還有臉說中共和俄羅斯的關系有彈性韌性生命力。試問中共，哪國占中國最多的土地，文革期間，大打出手，差點動用核彈，害得老毛要“深挖洞、廣積糧”的又是誰？

環球社論說：中俄站在一起，任何霸權主義都立刻顯出某種心虛。上周，美國國務卿與俄羅斯外交部長拉夫羅夫在冰島首都雷克雅未克舉行了會晤，緊接著，6月16日拜登將與普京在日內瓦會晤。這美俄最近來往也太頻繁了吧，誰最害怕，誰最心虛，不言而喻。楊潔篪慌忙趕到莫斯科，生怕美俄又簽一個聲明協議。現在，美日印澳韓，再加上歐盟，這些含金量最高的國家和美國一道，圍堵中共。隨著滅共的深入，俄羅斯如果無意加入美歐正義聯盟的話，普京和外交部長就沒必要與美國首腦見面了，只是想最後把自己賣個好價碼，中共不會看不到吧？看來，中共快到慌不擇路的時候了。
