进入新年以来，在中国出镜率最高的“明星”——中共党魁习近平突然在媒体上不再露脸，对此，海外自媒体路德社进行了解读。路德社还认为，美国大选存在两党妥协的可能性。

**习近平很可能已做手术**

“往年的元旦期间，往往是习近平露脸最频繁的时候。”路德说。

资料显示，2019年1月2日，《告台湾同胞书》发表40周年纪念会在人民大会堂举行，习近平出席并讲话。2020年1月2日，习近平同厄瓜多尔共和国总统莫雷诺互致贺电，庆祝两国建交40周年。今年的1月2日，中央电视台播道说，习近平“日前”签署命令，发布新修订的《军队装备条例》。路德表示，“日前”这个措辞非常的模糊，恰恰是想掩盖习近平的真实活动信息。

“根据可靠情报，习近平很可能已经做了手术。他最后一次露面是2020年12月31日的全国政协茶话会，他的手术时间不会太长，但手术需要全身麻醉，这样需要的恢复时间就比较长了。”路德分析，习近平不可能去301医院做手术，因为那样不利于保密，他很可能是在一个秘密的地方，由熟悉他病情的香港医生负责治疗。据习近平“身边人”透露，他前一阵子拍了“医学片子”，他的脑部血管瘤已经发展到“非常危险”的程度，手术不得不做。

路德指出，最近北京政坛风声鹤唳，习近平指定的“危机办”作用不大。“危机办里面没有彭丽媛。一方面他利用彭丽媛监督危机办成员，另一方面，他对彭丽媛也不放心，彭和阎维文的暧昧往事也让习心存芥蒂。”路德说。习近平的弟弟习远平则是“危机办”的重要成员。习远平在北京号称“二皇帝”，出任危机办成员说明，习近平笃信“上阵亲兄弟，打仗父子兵”这句老话。

“现在最盼望习近平死掉的是王岐山和江家。习近平的钱袋子中兴公司被美国制裁的时候，习近平亲自打电话给川普总统求情，救了中兴一命。纽交所强制中国三大通信运营商退市后，习近平政府并没有激烈反应，仅仅让商务部发表了一个不疼不痒的声明。因为打的是江家的钱袋子，江家现在肯定恨死习了。况且，中共一向有二把手觊觎大位的传统，历史上当过二把手的刘少奇、林彪都盼着毛泽东赶紧去世，今天的王岐山也不例外。“路德说。

路德社嘉宾墨博士认为，习近平很可能已经在这几天内做了一次手术，但是否还得继续做手术要看手术效果。有分析人士查询央视“领导人视频集”发现，中共中央政治局委员的活动大多止步于2020年12月31日的全国政协茶话会，以及稍早之前的中央农村工作会议。李克强活动稍微多一些，也仅仅是在12月31日和俄罗斯总理米舒斯京互致新年贺电。墨博士怀疑，政治局的同事们很可能在习手术期间失去了自由和外界通信渠道。“习近平对他们最不放心，只有严密监视政治局的人，习才能安心手术。这也是我高度怀疑习近平已经做了手术的原因。”墨博士说。

**鲁哈尼为何认定川普几天内将被杀**

1月2日，伊朗总统鲁哈尼对川普总统发出死亡威胁。他说：“川普将在几天内死亡。这名罪犯的生命将终结，他将进入历史的垃圾堆，对此我们感到非常高兴。”

就在几天前，路德社曾报道，习近平内部命令，1月2日，最迟不超过1月6日要让川普总统“人间消失”，习近平下命令的音频也已到了川普手中。对于鲁哈尼的声明，路德认为，这不仅验证了路德社信息的准确性，还说明暗杀川普的计划不是伊朗实施的。“这可能是川普总统提前回到白宫的原因。川普总统就是要让敌人出手，他好作出回应，可以说是胆大技更高。”路德说。

墨博士认为，鲁哈尼的讲话口气有点像犯罪集团中的“从犯”，以为主谋有能力干掉川普，才会对外这么高调。“目前能暗杀川普的国家只有俄罗斯和中共国。俄罗斯普京总统不可能有这个意愿，只有中共国习近平才会想暗杀川普总统。”墨博士认为，其实现在最危险的是习近平本人。国际正义力量要活捉习，政治对手们想让习近平赶紧死掉，习近平身边现在肯定有好几拨力量在角逐，习近平的压力“前所未有”的大。

**大****选面临妥协充满变数**

最近的美国总统大选已经进入了最后的高潮，路德分析认为，这场大选和1876年的美国大选有相似之处，可能面临1877年两党妥协的局面。

美国内战结束后，共和党掌控了联邦政府，开始在南方推行政治重建和北方模式的经济发展，遭到强烈抵触。在这种背景下，民主党总统候选人蒂尔顿1876年参选时轻松赢得多数选民票，并声称自己在选举人团中也获胜。但是，当时民主、共和两党都声称本党候选人在佛罗里达、路易斯安那和南卡罗来纳州胜出，因此共和党拒绝接受三个州的选举人团选票，俄勒冈州选举人团成员的人选出现争议，南方和北方州的矛盾也空前激烈。

因此，国会不得已成立了一个两党共同参与的联邦选举委员会，成员包括众议员、参议员和最高法院的法官。结果共和党参选人海斯以185票对184票淘汰蒂尔顿，当选总统。不少历史学家认为，南方的政治领袖们私下跟海斯的支持者达成协议，推举他任总统，作为交换，海斯同意将联邦军队撤离南方，南部各州恢复“自治”。这个结果被称为“1877年妥协”，这笔政治交易一方面避免了第二次内战爆发，另一方面宣告“南方重建”终结，而获得自由不久的黑人境遇再度恶化。

“我们有充足的情报证实，2020年的大选情况和1876年总统大选惊人的相似，两党也面临妥协。”路德表示。

近日，美国共和党11名参议员联合发表声明，要求国会成立联邦选举委员会。路德认为，如果选举委员会能够成立，按照法律走完辩护程序，那么大选作弊的大量证据将被公布，美国民众将会彻底了解这次大选的真相，对于唤醒民意和川普总统连任将非常有利。

“1876年的美国大选最终以妥协告终，牺牲了黑人的利益。1882年的《排华法案》，损害了华人的合法权益。2020年的大选两党之间肯定要有妥协，但我们认为中共不等于中国，两党妥协也只能牺牲中共的利益，不能牺牲中国人的利益，因为有爆料革命和新中国联邦的存在和实力。”路德呼吁，海外华人应该坚定自己的政治立场，并更多的发声，捍卫中国人的正当权益。
