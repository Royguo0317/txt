**简评：**

辉瑞公司宣布疫苗对中共病毒预防的有效程度超过90%的消息一出来，金融市场巨震，股市狂涨，道琼斯指数、标普指数突破历史新高，彷佛中共病毒的影响已然失效。利用媒体释放的消息，这背后的资本在市场上赚得盆满钵满。

胡锡进在此时翻墙发推，说瑞辉疫苗的新闻出现的时机有点怪，中共的疫苗试验被巴西叫停，其实就是搅混水。胡刁盘暗指中共国疫苗因政治原因被其它国家排挤，顺便挖苦川普总统，言下之意就是疫苗研发和川普政府前期的政策无关。

关于瑞辉疫苗的事情，文贵先生在2020年11月10日发布的盖特视频中说：“兄弟姐妹们千万记住，我郭文贵说的，疫苗不管用，这就是整个黑暗势力的控制。千万不要染上病，千万不要相信那个疫苗。“

中共政权还存在，谈论疫苗为时过早，连原始毒株都没有，疫苗如何能解决ADE效应和病毒的强变异性？只有灭共才是疫苗研发的基础。

# **中共国媒体评论辉瑞疫苗新闻:“时机有点奇怪”**

[![](https://spark.adobe.com/page/LsCR24v7xgjvt/images/0216bb4a-ca08-4fd8-b769-892566d76030.jpg?asset_id=282c7af6-bf7f-46b8-83ce-c9430c54fbc1&amp;img_etag=%2286aae039c0df749a18123c9ac0ec8bb9%22&amp;size=2560)](https://spark.adobe.com/page/LsCR24v7xgjvt/images/0216bb4a-ca08-4fd8-b769-892566d76030.jpg?asset_id=282c7af6-bf7f-46b8-83ce-c9430c54fbc1&amp;img_etag=%2286aae039c0df749a18123c9ac0ec8bb9%22&amp;size=1024)

中共国官方宣传媒体《环球时报》总编胡锡进周二表示，辉瑞(Pfizer)制药公司宣布中共冠病毒疫苗研发取得突破的消息“很奇怪”，并指出时机是在美国大选之后。

胡锡进质疑辉瑞公司(Pfizer)的宣布时间，既发生在美国总统大选之后，紧接着巴西政府就停止了一种中共国候选疫苗的临床试验，声称该疫苗的一名参与者发生了“严重不良事件”。巴西总统博尔索纳罗(Jair Bolsonaro)表示，他不会允许巴西政府从中共国购买、发放疫苗，巴西民众公开表示不相信中共国。

上周，巴西最大城市圣保罗的民众走上街头，高呼“我的身体，我的选择”，反对一种强制的由中共国研发的疫苗，而该疫苗得到了博尔索纳罗(Bolsonaro)的竞争对手、州长乔•多里亚(Joao Doria)的支持。

[![](https://spark.adobe.com/page/LsCR24v7xgjvt/images/405953a5-2415-42cd-8046-767bc31c2530.jpg?asset_id=4f2964f5-a3e1-49dc-8f8f-c77b30a26842&amp;img_etag=%22316e8ec25c6d84b65c083a4acb20c1f2%22&amp;size=2560)](https://spark.adobe.com/page/LsCR24v7xgjvt/images/405953a5-2415-42cd-8046-767bc31c2530.jpg?asset_id=4f2964f5-a3e1-49dc-8f8f-c77b30a26842&amp;img_etag=%22316e8ec25c6d84b65c083a4acb20c1f2%22&amp;size=1024)

截至11月3日，世卫组织已经确认了47种正在研制的中共国冠状病毒候选疫苗。其中11个项目正在中共国研发，要么由政府，要么由政府控制的公司研发。胡锡进对时机的担忧似乎与辉瑞公司(Pfizer)的声明(如果该声明发生在大选前的话)对美国川普总统的不利影响没有直接关系，而是与中共国候选疫苗的成功有关。胡锡进表示担心其他竞争者可能会出于“政治”原因将中共国疫苗挤出全球市场。

中共国一直以研制危险、无效和在其他方面毫无用处的疫苗而闻名。据报道，中共国公民对自己国家的疫苗极度不信任，面临丑闻时会在全国范围内抗议。例如，在2018年的事件中，官方发现长春的长生生物科技公司(一家与政府有关联的制药公司)售出了近100万份过期或有问题的疫苗，这意味着有近100万儿童在不知情的情况下丧失免疫功能。

[![](https://spark.adobe.com/page/LsCR24v7xgjvt/images/9a1d082b-465f-4e55-86c3-d15c95e8f908.jpg?asset_id=eff8a07a-2121-4368-b4ba-331eb0ae773a&amp;img_etag=%228ca476a9e34f41286456cbb2ea08a72c%22&amp;size=2560)](https://spark.adobe.com/page/LsCR24v7xgjvt/images/9a1d082b-465f-4e55-86c3-d15c95e8f908.jpg?asset_id=eff8a07a-2121-4368-b4ba-331eb0ae773a&amp;img_etag=%228ca476a9e34f41286456cbb2ea08a72c%22&amp;size=1024)

胡锡进在他的宣传报纸上写道：“就在美国总统选举几天后，辉瑞公司（Pfizer）宣布在开发新冠疫苗方面取得了重大进展。这是好消息，但时机有点奇怪。随后，巴西暂停了中共国新冠(中共冠状病毒)疫苗的三期临床试验，理由是‘出现了严重的不良反应’。这就更奇怪了。”

胡锡进补充道：“我非常担心这些关于疫苗的公告过度涉及政治和过分追求利润。”

胡锡进声称，在中共国目前研发的候选疫苗中，只有巴西的“冠状病毒”候选疫苗出现了突然暂停测试的情况。胡锡进补充说：“由于目前还没有足够的信息，我们需要等待进一步的信息发布。”他称关于候选疫苗的公告“太突然了”。

他说：“我个人对中共国的疫苗研发和处理疫情的整体严肃性有信心。中共国的科研机构是第一个分离出冠状病毒毒株的，也是第一个投资于疫苗开发的，所有这些都处于国家级的严格控制之下。”

中共国是中共冠状病毒的起源地，该病毒于2019年底在中部城市武汉首次被发现。中共国官员审查并拘留了医生和其他卫生官员，他们曾警告武汉有传染病蔓延。北京方面还向世卫组织提供了虚假信息，世卫组织随后称没有证据表明中共冠状病毒会人传人。直到今天，中共国仍在继续推广应对冠状病毒感染的无把握的“传统中药”疗法。

[![](https://spark.adobe.com/page/LsCR24v7xgjvt/images/60879d01-f07e-4bb3-89b1-dc48bea6e7f7.jpg?asset_id=11011650-99f9-470c-908b-b201e29cfb85&amp;img_etag=%22da63fa559596ebd7ffd588901f039da5%22&amp;size=1024)](https://spark.adobe.com/page/LsCR24v7xgjvt/images/60879d01-f07e-4bb3-89b1-dc48bea6e7f7.jpg?asset_id=11011650-99f9-470c-908b-b201e29cfb85&amp;img_etag=%22da63fa559596ebd7ffd588901f039da5%22&amp;size=1024)

胡锡进在推特上重申了他对辉瑞公司（Pfizer）的怀疑。

[![](https://spark.adobe.com/page/LsCR24v7xgjvt/images/879cdb6e-6582-4949-881f-dc7ac3dd4a4d.png?asset_id=71b75b39-997e-4295-a7ab-7df3177a7fd7&amp;img_etag=%22280f7b3453414c1a1e4548e7c81939a2%22&amp;size=2560)](https://spark.adobe.com/page/LsCR24v7xgjvt/images/879cdb6e-6582-4949-881f-dc7ac3dd4a4d.png?asset_id=71b75b39-997e-4295-a7ab-7df3177a7fd7&amp;img_etag=%22280f7b3453414c1a1e4548e7c81939a2%22&amp;size=1024)

辉瑞公司（Pfizer）周一宣布，对其冠状病毒候选疫苗的初步分析发现，该疫苗对预防中共国冠状病毒感染的有效程度可能超过90%，且没有明显副作用。该疫苗是与德国制药公司BioNTech联合开发的。辉瑞的发言人还坚称，该疫苗不是“曲速行动”的一部分。“曲速行动”是川普计划与美国制药公司合作，尽快研发一种针对冠状病毒的疫苗。

[![](https://spark.adobe.com/page/LsCR24v7xgjvt/images/ca0e0310-3056-4e65-90eb-950c0b114a02.jpg?asset_id=50cb41d2-264f-4a14-bb9f-3d57fb673bb8&amp;img_etag=%225defaa5aa33d97a2692d1f0d6dbc52cc%22&amp;size=1024)](https://spark.adobe.com/page/LsCR24v7xgjvt/images/ca0e0310-3056-4e65-90eb-950c0b114a02.jpg?asset_id=50cb41d2-264f-4a14-bb9f-3d57fb673bb8&amp;img_etag=%225defaa5aa33d97a2692d1f0d6dbc52cc%22&amp;size=1024)

今年7月，辉瑞公司（Pfizer）宣布与川普政府签署了一项19.5亿美元的协议，作为研发和发放冠状病毒疫苗的“曲速行动”的一部分。

法新社(AFP)当时报道说：“这是迄今为止在曲速行动下最大的一笔交易，旨在加速冠状病毒疫苗、治疗和诊断的研发、生产和发放。”

中共官员已采取行动，尽可能快地向数万人注射实验性候选疫苗。10月，中共国科技部证实，有6万名“志愿者”接种了多种疫苗，而且共产党没有记录任何“严重”副作用。据中共国官媒报道，来自北京科兴生物制品有限公司的一种候选疫苗已经将人体试验扩展到了巴西、印度尼西亚和土耳其。

在明确参与实验的“志愿者”数量之前，中共国国家卫健委(NHC)医学科学技术发展中心的负责人在8月份表示，他们已经开始分发候选疫苗，“以备紧急使用”。中共国疾控中心主任高福称，他在7月份接种了一种候选疫苗。

在中共国宣称在研制冠状病毒疫苗方面取得重大胜利之际，国内出现了一些报道，称共产党难以说服公民接种流感疫苗。几十年来，流感疫苗一直是常规免疫周期的一部分。1960年，美国卫生局局长首次建议每年向呼吸系统疾病高危人群接种流感疫苗。

9月，全球卫生外交关系委员会高级研究员黄严忠告诉《南华早报》(SCMP) ：“虽然中共国有能力增加流感疫苗供应，公共需求始终较低，因为缺乏信任、缺乏公众意识、缺少渠道以及成本问题。”据报道，中共国官员将这种不信任主要归咎于长春长生生物技术公司的疫苗丑闻。

当家长们得知自己的孩子没有有效地接种疫苗时，全国爆发了抗议活动。2019年1月，一群愤怒的家长在公众场合暴打一名共产党卫生官员。



这一事件引发了作为家长的活动人士新一轮反对这个政权。持不同政见者何芳梅(音) 在10月份失踪，可能是因行动被捕，她的女儿在接种了有问题的疫苗后患病。

在兰州，一家疫苗工厂去年爆发了布鲁氏菌病(一种传染性细菌疾病)，原因是工人使用过期的、因此而无效的卫生产品来清洁设备。上周，政府官员证实，到目前为止，兰州已有超过6000人在这场持续爆发的疫情中被感染。
