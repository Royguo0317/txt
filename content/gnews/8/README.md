![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/03045249/LOGO3_1-1.jpg)



用完就扔——这是很多人对待一次性物品的态度。当然还有一部分人主张要回收所有的物品，争取有再生循环的机会——这是一个地球治理的概念，也是未来新能源的一个领域。事实上，我们对待物品，有一个分类、处理、循环的过程，亦有一个重新发现其价值的意识——这反映了人类是如何对待事物以及如何对待自己的。

七哥在13日的直播中讲述了中共是如何利用人的，他们利用完就将其杀死，而且不仅是肉身的消失，还要祸灭其九族。普通国人对这些手段一点都不陌生，因为杀戮每一秒都在墙内发生，区别只是今天死、还是明天晚上死，是发生在熟人身上，还是在市委书记身上。但不会有人为了市委书记被抓而难过流泪，因为大家都知道他是大大小小魔鬼中的一个。就像人不会为魔鬼落泪，因为魔鬼喝人血的时间太长了。由于中共霸占了一切生存资源和优势，因此只要树起“利益”这个牌子，就会有无数人纷涌而至 ，都想在其中分一杯羹。这些人巴不得被中共“驯化”、“乳化”。当然，你得有价值才有人想驯化，普通人是没有资格被驯化的。

![]()![](https://media.discordapp.net/attachments/798315419826913331/854133608172355604/image0.jpg?width=1083&amp;height=609)

在中共心中，够资格被驯化的是下面几类人：

1. 极度聪明者。无论是王歧山、郭沫若、乔冠华，还是董宏、车峰、二马等，他们都是聪明人。这些人在民主国家里同样可以飞黄腾达，但不幸生在中共体系中，只能效忠这唯一的体系，最后再被这个体系吞噬。其中，王歧山摸透了中国历史，但他的才干都用来对付老百姓了；郭大诗人有才不假，可晚景凄惨，孤家寡人；乔冠华被利用了一辈子，但得不到信任；至于董宏、车峰、叶简明，他们哪一个不是百姓口中的“别人家的孩子”，从小到大都是佼佼者，但至死才发现，聪明没有用、关系没有用、生殖器没有用，死后都要被双规；二马的演讲还历历在目，青年榜样的称号还没掉，自己的钱却消失了。相比之下，我们老百姓忙着、累着，有点小钱，还能入土为安；这些才俊们为中共“鞠躬尽瘁”，到头来发现连家人的安全都守不住。
2. 极度狠毒者。孙立军、傅政华、王立军、孟建柱都是一个套路——狠毒、投机且胆大包天。在古代，他们通常被称为“酷吏”。而今，他们效仿德国的盖世太保——掌控无处不在，监控亦无处不在。他们不一定都学过法律，但他们谁都想管，把一切人都当成敌人。
3. 生殖器关系网中之人。虽然中共的生殖器关系网庞杂，但一个人想用生孩子来保平安、保钱是不可能的。肖建华有那么多个孩子，当中有一个出来为他奔走的吗？没有。孩子既不能阻止父亲的被捕，也不能用不了父亲的钱。对于中共内部人员，孩子数量的多寡跟炫耀有关，与江山无关。细数独裁者的后代，古往今来有几个过得愉快的？波尔布特有后代，但其后代会感到骄傲吗？这些人只会终身生活在“你父亲是某某”的阴影之下，害怕有人来找他们算账。这部分可以展开细述很多，我们下次再谈。


所有臭名昭著的邪恶组织中，离我们最近的是ISIS。想必大家看到ISIS斩首视频时，都会感到极度不适。人们约定俗成的是，没有被公义裁定过的举动是不合法的。因此，ISIS的斩首是非法的，自杀式袭击也是非法的，对平民的杀戮更是恐怖主义。即使凶残如ISIS——他们不放过任何拒绝执行他们命令的人——也没听过他们会连坐家人。与之相比，中共更恐怖，其始终用家人来拿捏手下的棋子，等棋子被用够了要扔掉时，他们的家人也会被连坐，一起被消失。但中共认为，所有人在死之前都得给我干活。所以，中共不会让你早早的死了，而是要多多地奴役你，用来供养他们自己，也就是割韭菜。等中共把人用老了、用够了，或是它觉得不安全了，那么，奴隶们就去死吧，反正还有大把的人等着被“驯化”呢。但中共忘记了一点：他们自己也是人，也是要死的。

玩人者，终被人玩；杀人者，葬于人手。说到底，中共想玩世界，现在却被G7给玩了。
