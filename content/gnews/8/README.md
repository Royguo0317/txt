## 美國聯邦調查局逮捕並指控疑似中共間諜人員
`2020-07-22 02:22 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/272162/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/22021914/1-101.png)圖片來源：澳洲新聞
據《法院新聞服務》（Courthouse News Service）20日報導稱，舊金山聯邦調查局逮捕了一名38歲中共國女性- 宋辰（Chen Song）。聯邦調查局指控她在斯坦福大學進行研究時，刻意隱瞞了與中共軍方之間的關係，同時起訴她簽證刑事欺詐，於週一公開了這位科學家在獲得美國簽證後隱瞞了她與北京某研究院的隸屬關係以及自己為解放軍工作的事實。她在斯坦福大學主要研究神經系統疾病，重點研究重症肌無力。這種疾病會導致骨骼肌無力，從而影響人體運動和身體機能。

聯邦調查局特工喬伊斯•布拉洛克（Joyce Blalock）在周一發表的證詞中說：“我認為，宋某故意誤導信息，她聲稱與中共解放軍空軍沒有任何联系，但她無法提供具體明確的理由來對此進行解釋。”根據證詞描述，當宋某於2018年首次抵達美國時，她說自己以前曾在北京釣魚台醫院做過研究員。聯邦調查局（FBI）找到了宋某的材料顯示，她的軍人身份與她在申請簽證時填寫的信息相矛盾，與她在2011年從第四軍醫大學畢業後，就與中共軍方脫離關係的簽證申請表內容背道而馳。證詞說，她發表的在中共國期間的研究，證明她在中共空軍下屬的一個研究機構工作。據布拉洛克說，當宋某看到她的照片資料時，她表示她將選擇保持沉默權。

聯邦調查局發出逮捕令後，他們發現宋某試圖從她的計算機中刪除一封郵件，這封郵件能證明她一直在釣魚台醫院使用虛假履歷表作為她為中共軍方成員在美國進行研究的幌子。如果被判犯有簽證欺詐罪，她將面臨最高10年的監禁和最高25萬美元的罰款。她和她的女兒都住在美國，但最近她談到要讓女兒回到中共國接受教育。

此案發生時，也是中共和美國之間國際關係緊張局勢繼續加劇之時。川普總統上週簽署了一項行政命令，終止了美國與香港之間特殊地位的關係。美國政府還通過了一項法案，譴責中共政府對維吾爾族穆斯林的不人道殘忍做法。同時在南中國海領土主張的爭端方面，美國也拒絕承認130萬平方英里的大片海域中共的主權主張。

中共在美國研究型大學的嚴重間諜滲透也引起人們越來越多的關注，宋某隻是因涉嫌簽證欺詐或直接為中共從事間諜活動中的其中一個案例而已。目前，宋某還未被指控從事間諜活動刑事罪，但2018年，她提交的簽證申請材料中卻對她自己的工作經歷進行了隱瞞。
