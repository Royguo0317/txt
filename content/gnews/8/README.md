## 【热点播报】福奇妻子掌管NIH生物伦理学部门会有利益冲突吗？

作者：纽约香草山媒体部 7bends

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/01162447/gnews-logo.jpeg)





据[Life Site News](https://www.lifesitenews.com/mobile/news/conflict-of-interest-faucis-wife-runs-bioethics-department-at-nih?__twitter_impression=true) 6月4日报导，福奇博士的妻子Christine Grady博士已经悄然成为美国医学研究和医学伦理学的关键人物。她是美国国立卫生研究院（NIH）生物伦理学部门的主管，同时也是人类受试者研究部门的头。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/10215833/161-1-fauci_and_wife__810_500_75_s_c1.jpg)*安东尼·福西博士和他的妻子克里斯蒂娜·格雷迪博士，于2016年在奥巴马白宫的一次活动‎中 Alex Wong / Getty Images*

在NIH的27个研究所和中心里，福奇长期担任所长的国家过敏和传染病研究所（NIAID）拥有独特的使命来应对新出现的公共卫生威胁。NIAID以NIH的道德规范为指导。这就是说Grady关于伦理的决定与他丈夫关于疫苗研发的决定密切相关，并为他丈夫提供信息。NIH参与的最具争议的项目之一就是创造人源化小鼠，将流产胎儿的组织移植到小鼠体内。而NIAID则参与了Moderna疫苗的开发和试验。

尽管美国政府疫苗不良事件报告系统的官方数据显示,从去年12月开始注射Moderna疫苗以来共有122712起与该疫苗相关的不良反应，其中2077名患者在注射疫苗后死亡，约占美国接种疫苗后死亡总数的一半，同时有8822例严重不良反应。然而，对于投资者而言，疫苗是成功的。NIH主任柯林斯说，Moderna因疫苗已得到美国政府25亿美元的资金，并将从疫苗的销售中获得185亿美元，而NIH则拥有该产品的部分知识产权。

当福奇博士领导的NIAID的科学家与医药公司共同开发Covid-19疫苗时，遵循着由福奇妻子Grady所领导研究的道德规范，这其中难道不存在利益冲突吗？
