据True Pundit 12月5日报道，在11月大选日后，拜登自我宣布当选新一届总统。紧接着拜登提名安东尼-布林肯(Antony Blinken)为新国务卿。而布林肯是华盛顿特区咨询公司WestExec Advisors的创办者之一，该咨询公司网站上与中共国有关的业务细节迅速被删除，因为该公司的业务在拜登宣布胜选后受到了审查。

WestExec Advisors咨询公司成立于2017年，美国研究型大学是他们的客户之一。该咨询公司与大学合作，从事扩大与外国研究合作、接受外国捐赠、欢迎外国学生参加关键的STEM项目。

该公司夸口他们与政府之间有深厚的关系，可以代表客户与政府打交道，许诺帮助解决客户的问题，为客户提供一手资料和政府渠道。该公司在其网站上表示，它帮助客户驾驭 “战略竞争时代与中共国有关的风险”，并一度强烈建议其负责人将依靠他们的政府关系，帮助寻求中共国资金的美国大学 “与国防部一起驾驭这些问题”。他们迄今拒绝透露他们的客户或详细说明其工作的确切性质。

该公司在7月26日至8月2日期间，即拜登在8月底接受民主党全国代表大会提名前几周，从其网站上删除了对此类工作的介绍。

该咨询公司的工作之所以受到关注，是因为拜登已经或正在考虑在他的新一届政府中启用该公司的几位负责人和顾问，包括任命WestExec创始人之一布林肯为国务卿，前五角大楼官员米歇尔-弗卢诺伊(Michèle Flournoy)为潜在的国防部长，提名艾薇儿-海因斯为情报主管，任命詹-普萨基为情报秘书。

WestExec Advisors 吹嘘的“与政府有深厚关系”，并以此为资本在美国研究型大学招揽生意，声称他们可以为这些大学找到捐款，这意味着什么？首先所谓与政府有深厚的关系，当然是指的拜登曾做副总统，可以用他在政府里的权利和影响为客户提供方便。其次为大学找到捐款，但代价是允许捐款人送研究员或者学生加入被捐款的大学里的重要科研项目。这就是中共捐款的目的——从美国偷技术和科研成果。几十年来，中共从美国用这个方法偷走了不计其数的科研成果。拜登伙同他的美国卖国贼伙伴为了金钱，勾结中共、出卖美国。如今又与中共合伙欺诈2020美国总统大选，迫不及待地宣布胜选，并开始任命新政府要职。美国卖国贼一一被暴露，中共在美国所犯下的累累罪行已难以遮盖。排干沼泽，法责卖国贼，消灭中共成为当前美国最重要的任务。
