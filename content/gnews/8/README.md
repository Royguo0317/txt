## 【文艺灭共】 隔絕文明的愚民之術 ——朝鮮紀錄片《太陽之下》
`2020-09-22 03:51 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/376234/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/22034806/Picture45.png)*图片来源：朝鮮紀錄片《太陽之下》海报*[http://fims.kofic.or.kr](http://fims.kofic.or.kr/kobis/mobile/mast/mvie/searchMovieDtl.do?movieCd=20161842)
一個俄羅斯劇組要求進入平壤拍攝紀錄片。朝鮮政府同意了，但條件極為苛刻：朝鮮政府派出的指導人員負責安排拍攝內容，只能拍攝政府挑選出來的拍攝對象，這些人的言行都處於嚴格監督下。而且每天拍攝的素材都要經過檢查，不符合政府要求的內容會被刪去。

這樣拍攝出的紀錄片顯然是不真實的，但導演曼斯基想出了一個辦法。他們始終讓攝影機出於開機狀態，記錄下了官方人員如何指導人們配合拍攝。每個場景都要反復拍攝多次，直到這些監督人員認為滿意為止。曼斯基把這些排演、干預和多次拍攝的場景都剪輯到紀錄片當中，這些內容在最後完成的紀錄片中超過一半。所以觀眾看到的是一個真實的紀錄片，所有人的日常起居、言談舉止、表情情緒都處在嚴格的監視和控制中。

影片一開始，小主人公珍美對著鏡頭說：“朝鮮是東半球最美麗的國家。”這也是朝鮮政府力圖通過影片傳達的資訊。負責監督影片拍攝的人必須讓觀眾看到朝鮮國民多麼熱愛他們的政府和領袖，在他們的統治下過得多麼幸福。

接著，我們看到，無論是在教室、工廠、地鐵還是街道，到處都是兩位領袖的畫像和雕像，此外沒有任何裝飾。人們從早上睜開眼，觸目所及只有偉大領袖。每一天的生活中也只能聽到偉大領袖。在課堂上，老師反復講的只有一個內容：金日成將軍用石頭把日本人砸跑了。掛著一身獎章的將軍給孩子們講的是領袖帶領他們用步槍打美國飛機。學生操練喊的是跟隨金大將軍的腳步，學校的活動發言是祝偉大領袖安康長壽，參加群眾舞蹈的人們唱的是心中只有偉大領袖，舞蹈演出是為獻給金日成大元帥。

在這個密不透風的地方，人們唯一能用來填充大腦、抒發感情、公開談論的只有偉大領袖。這就是主人公一家呈現出來的生活。對生活在現代文明世界的觀眾而言，這種單調、乏味、枯燥的狀態毫無人生樂趣可言，顯然和“幸福歡樂”相距甚遠。

但參與拍攝的人們對此卻有著截然不同的理解。他們努力顯現出對自己國家的自豪感，甚至希望所有人都羡慕他們的美好生活。為了達到這個目標，參與拍攝的每個人確實都非常盡力。在首都平壤，人們的物質生活也極為艱難。然而面對為了拍攝擺出的一整桌子飯菜，珍美一家都絕對服從安排，他們反復演習、表演，直到這個場景拍攝結束，盤子裏的飯菜還是滿滿的。我們可以看出，室內的供暖明顯不足，但為了拍攝，人們穿上漂亮的服裝，尤其是女人和孩子，甚至露著小腿，只在以為已經停機的時候才搓手取暖。從始至終，他們都在努力呈現他們最好的狀態，在他們的意識中，這就是理想的世界、理想的生活。

不过珍美和她的同學畢竟都是七八歲的孩子，反復拍攝顯然讓她們的體力透支。她們被要求在臉上時刻顯現出陽光、歡樂和飽滿的情緒，但在極度疲憊時，表情會顯露出真實的狀態。鏡頭一直在捕捉孩子們的臉。被要求聽將軍講戰鬥故事時，一個小女孩已經無法克制住睡意，但她拼命讓自己清醒過來。珍美在長時間練習舞蹈時努力面帶微笑，但因為疲勞，她的表情僵硬，顯得很古怪不自然。即便是導演按照要求，沒有把干預拍攝的內容放到影片中，這些令人心酸的細節也让观众無法認同主人公一家的“幸福”。

在影片結尾，小珍美突然忘記臺詞。她一下子失去了控制，大顆淚珠滾滾落下。沒人理會她的傷心。只有不允許和演員交談的影片導演忍不住想安慰她，讓她談談高興的事，小珍美很吃驚，回答說自己想不出來。在導演和翻譯的鼓勵下，她終於想起一首“高興”的詩：“偉大領袖……”

為什麼片中的人和我們對生活的理解有這樣大的差距？當一個孩子的全部快樂就是歌頌領袖，她未來的人生之路將會是怎樣的？除了心甘情願為統治者服務，她還能找到其他生活目標嗎？

毋庸置疑，這當然是統治者最滿意的。“從娃娃抓起”，讓人們在孩童時代就習慣於被灌輸、被操縱、被控制。片中這些無辜的小孩子，已經被商鞅五術中的“愚民術”洗腦成了宣傳工具，稚嫩的臉上沒有了正常人童年的快樂。而這對我們其實也是再熟悉不過了。

七十年來的中國是這樣，當今的朝鮮也是如此。正所謂“愚民有術”，人民越愚昧越有利於極權統治。而這樣做最行之有效的手段就是與文明隔絕。文明世界是人類幾千年積累起來的一種複雜多樣的生態系統。每個在其中生活的人，從外部接收到的信號越豐富多樣，對大腦產生的刺激越複雜，人的思維和情感也就越豐富。就是為什麼，在多元的社會中，各種觀念的融合交匯，往往會撞擊出高度的文明。從古至今所有高度發達的社會系統都不是封閉的。而一旦一個系統開始閉合，必然意味著衰落。

不过對極權者而言，只要能穩固統治，根本不會在乎文明與否。

走向極權的統治者，首先便要封閉社會系統，讓個體失去活力。眼界狹窄、思維僵化更容易被洗腦。就像在紀錄片《太陽之下》中看到的那樣，在這個極度單一化的社會，人們被封鎖在由“領袖”的標語口號和畫像組成的洗腦系統中，無法接受任何真實資訊。僅僅幾十年，幾代人便被徹底改造，他們的價值觀已經完全不同於外面文明社會中的人。

但人類社會的發展是有規律可循的。文明越衰落越有利於統治，如此惡性循環，導致極權社會最終必然崩潰。所有的極權體系莫不如此。可憐的是片中那些被捆綁其中而不自知的人們，被宰割被踐踏時依然高唱頌歌。
