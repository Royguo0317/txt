1. 中国篮球界第一人，现任篮球协会主席姚明，在50周年纪念活动中给中共站台，说”我觉得交流是唯一的办法。在没有交流之前，误解和偏见永远存在，只有通过交流去化解。”共产党奉行一朝天子一朝臣，内部斗争非常激烈，局外人不知道里面的政治博弈政治勾兑的复杂。毛贼东时代的第一运动是乒乓球，当时乒乓球第一人容国团，因为他是刘少奇的人，所以在文革的时候就被斗死了。文革时期很牛的乒乓外交第一人庄则栋，在文革结束后的邓小平时代，被整得很惨。体育界反映了中共几个政治家族的争斗。一个运动员要想当上体育界的主席局长副局长，后面如果没有政治家族的势力支持，是没有任何可能的。中国老百姓在电视上看见的为中共站台的那些光辉形象，没有一个人是真正光辉的，一定是跟中共同流合污的人。中共国运作姚明到美国NBA打球，在NBA得到很多出场率，被NBA和火箭队力捧，背后是江家势力在运作。中共是在利用体育，每推动一项体育运动都是有特定政治意义的，是为了跟别的国家进行民间交流，获取各个国家人民的好感。
2. 4月25日清华大学举行的110周年校庆日升旗仪式，成了党卫军歌颂党的表演，学生发言假大空，”要随时接受祖国召唤的准备，勇于担负起国家民族伟大复兴的责任”，但却不提校训”自强不息，厚德载物”。叶钊颖是清华经管学院2001级本科生，表示清华人对体育的传承是非常好的，但庆祝仪式上那些人如同夜总会小姐般的穿着打扮，和精神面貌颠覆了她对母校的美好印象。清华校训的”自强不息”中的”自”，首先得有自我，得有自由，得有独立思考和自由言论才能强，强了以后才能延绵不断。但中共在偷换概念，恶意曲解，实际上是在剥夺我们的自信摧毁我们的人格，在割我们的根。
3. 赛林博士推文揭福奇博士与Malik的隐密关系。推文中赛林博士质疑福奇博士、FDA资助Mailk进行病毒增强实验，并爆出关键的NIH合同号码，显示美国军情部门已经掌握相当多的资料，朝着揭开最终答案迈进了一大步，众所周知Mailk就是知晓中共病毒最关键的人物，他领导的港大实验室在全球冠状病毒研究、分离领域处于领先地位，中共国实验室的新冠病毒分离都需要港大协助。路德先生新冠病毒是港大实验室首先分离出来的，操作人就是闫博士的老公，但是Malik故意不然他对外界透露，防止引起外界的注意，现在随着美国军情部门调查的深入进行，NIH的机密合同都被挖掘出来，并且直接指向福奇博士，未来真正参与中共病毒的人如Malik、曹务春、陈薇、还有很多目前没有列出名字的人都会爆光出来。
4. 美国参议员保罗接受福克斯采访质疑福奇资助中共进行病毒功能增强实验。这意味着美国已经开始在立法层面探讨如何堵住科学界进行病毒增强实验的法律漏洞，同时把科学界的伦理问题引入公众的讨论，为下一步立法监管做舆论的铺垫，路德解读NIH资金审核委员会最初否决了资助“病毒功能增强”研究，但是后来福奇领导的NIH踢出了这些投反对派的委员，最后重新支持“病毒功能增强”研究，这背后的动机和计划未来都需要调查清楚，这些调查行动的背后爆料革命、闫博士扮演了重要的推动作用，路德表示美国推动关于科学伦理的讨论，未来将会针对这些违反科学伦理的人追责。
