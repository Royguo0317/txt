## 联合国统计2020年因疫情超2.55亿人失业

印度媒体报道，联合国周一公布的数据显示中共病毒大流行对全球就业影响巨大，2020年全球超过2.5亿人因疫情失业。

联合国国际劳工组织(ILO)发现，与2019年第四季度相比，2020年全球工作时间减少了8.8%，这相当于2.55亿个全职工作岗位。报告指出“大约是2009年全球金融危机期间丧失的全职工作岗位数量的4倍”。

自从2019年12月中共病毒疫情出现已经导致210多万人死亡，感染了超过1亿人，重创全球经济。国际劳工组织表示这是1930年代大萧条以来最严重的就业危机。

联合国劳工机构解释说，大约一半的工作时间损失是由那些仍在工作的人减少的工作时间计算出来的。去年全球经历了“前所未有的就业流失水平”。

2020年，全球官方公布的失业率为6.5%，上升了1.1%，增加了3300万人，达到2.2亿人。报告中显示还有8100万人没有登记失业，而是“直接退出了劳动力市场”。

国际劳工组织说，去年工作时间的损失使全球劳动力收入减少了整整8.3%。

这相当于减少约3.7万亿美元，约占全球GDP的4.4%。

尽管近期推出了中共病毒疫苗，让人们对世界将很快能够控制疫情产生了希望。但国际劳工组织警告称，今年全球劳动力市场复苏的前景“缓慢、不平衡和不确定”。

失业危机对全球工人的影响不均衡，对女性和年轻工人的影响远远超过其他行业。

从全球来看，去年女性的就业率下降了5%，而男性的降幅为3.9%。女性更有可能受到更严重的打击也承担了更多的负担，例如照顾被迫留在家里不上学的孩子。

年轻员工失业的可能性也增大。全球范围内，15至24岁的年轻人失业率为8.7%，而年龄较大的员工失业率为3.7%。

报告还强调了对不同行业的影响不均衡，其中住宿和食品服务受到的影响最大，就业人数下降了20%以上。相比之下，信息和通信领域以及金融和保险领域的就业人数激增。

联合国报告中还包含对2021年全球就业的展望，然而不从根本上控制中共病毒，一切都是空谈。中共病毒源自中共军方实验室，是中共制造的生物武器超限战。目前全球所有国家最迫切的需要是追查病毒来源，揭开病毒真相，从根本上灭绝中共病毒。中共是世界最大的毒瘤，造成这一切灾难的罪魁祸首。只能消灭中共，才能解决这一场中共生物病毒带来的世界危机。
