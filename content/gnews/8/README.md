![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/28025951/Politico.jpg)圖片來源：Politico
11月27日，伊朗首席核科學家被殺的消息引發了世界的關註，DJHJ媒體對此進行了報道，並將本事件與比利時正在進行中的對伊朗外交官在內的國際恐怖主義刑事指控聯系在壹起，對當年奧巴馬政權與伊朗訂立的漏洞百出的核協議提出了質疑。

報道中指出，隨著周五伊朗最高核科學家死亡的消息以及伊朗領導人受審的消息開始在美國出現，關於前總統巴拉克-奧巴馬（Barack Obama）政府如何參與資助伊朗政權試圖在中東獲取核主導權的問題依舊是人們關註的重點。

據Axios報道：“周五，伊朗國防部發表聲明，確認伊朗科學家、伊朗軍事核項目策劃人莫塞-法赫裏紮德（Mohse Fahrizade）Mohsen Fakhrizadhe已被暗殺。”

被暗殺的法赫裏紮德曾是伊朗國防部Amad項目的負責人，該項目直到2003年都致力於核彈的研發。2018年，以色列曝光了摩薩德在德黑蘭捕獲的伊朗核檔案後，以色列總理內塔尼亞胡(Benjamin Netanyahu)公開披露了法赫裏紮德，稱他為“伊朗核彈之父”。

而目前在比利時開庭的國際恐怖主義案件則是伊朗外交官阿薩多拉拉·阿薩迪（Asadollah Assadi）夥同其它三名伊朗人試圖於2018年在法國巴黎，計劃炸毀壹個流亡的伊朗反對派團體的會議。該事件引發了人們對於伊朗在國際上以外交官員身份行使恐怖主義的擔憂和譴責。[【1】](https://twitter.com/HaddadEffat/status/1332361593057906688?s=20)

伊朗不但在國際上開展恐怖活動，還壹直致力於鎮壓國內的民粹活動和爭取自由活動。伊朗壹位爭取自由的活動人士稱，他們希望“伊朗政權領導人因在世界各地從事恐怖活動而受到起訴”。

那麽奧巴馬為什麽要跟這樣壹個臭名遠揚的政權簽訂壹個所謂的“歷史性”的核協議呢？

根據推特大V推主，記者Heshmat Alavi的長篇推文指出，“前伊朗官員，親德黑蘭遊說者數十次在奧巴馬的白宮主持會議。長期被控隱藏與伊朗政府關系的親伊朗支持者特裏塔·帕爾西，與奧巴馬政府官員會面了33次。”[【2】](https://twitter.com/threadreaderapp/status/1332363959165444099?s=20)

左派壹直堅稱奧巴馬與伊朗的協議對美國來說是壹件好事。民主黨人喬·拜登是重申奧巴馬政策的合法繼承人。然而，奧巴馬的政策是用美國納稅人的錢在世界各地資助大規模致命恐怖主義。

推主John Calvert在他的推文中指出，法赫裏紮德的死亡徹底斷送了拜登將來與伊朗重回核協議的所有可能。[【3】](https://twitter.com/JCalvert_music/status/1332380966598418436?s=20)

川普對伊朗的強硬態度壹直受到民主黨及主流媒體的攻擊，左派主流媒體在電視上攻擊川普並同情支持那些伊朗民眾指認的恐怖分子們。

伊朗作為壹個在國際上肆虐的國家恐怖主義，奧巴馬與拜登政權與伊朗的核協議無疑助長了這種恐怖行為，人們不禁有理由懷疑，拜登可能是全球恐怖主義和反人類罪行的幫兇。

評論：奧巴馬和拜登政權與伊朗的核協議壹直受到保守派批評。亨特-拜登硬盤門出現前，曾有中情局內部爆料人出面指認，所謂的伊朗核協議只是為了掩蓋伊朗配合他們出演虛假的“獵殺本-拉登”，並滅口執行任務的海豹突擊隊而付的1520億美元的封口費，而其中的20億，則以現金的方式運回國內用以買通政客們。[【4】](https://gnews.org/zh-hans/422704/)此事曾引發了巨大的關註，但因亨特硬盤門的曝光而尚未繼續引爆。該爆料是否會因當前事件從回人們的視線中？讓我們拭目以待。
