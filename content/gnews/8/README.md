五月，变种病毒构成的疫情对台湾造成了巨大的危机。先是狮子王事件，然后是万华阿公店爆出问题。中华民国疾管署在处理疫情上的不足，造成了中共病毒危害一次性的在五月份集中爆发。这一波的疫情中华民国的中央政府具有巨大的责任。首先，台湾的负压隔离病房总署在过去一年几乎减半，台湾的重症患者医疗量大幅缩减。台北市市政府在万华区阿公店问题上的不作为导致了万华区茶女黑户巨大。阿公店茶艺女实际数量要高达上千人，涉及客户从北台湾到南台湾；政府宣称追踪到全部八十名外劳身份茶艺女，和实际情况差距巨大，黑户无法控制，必然导致小区感染难以控制。

不仅仅是疫情，台湾的电力供应也出现了巨大的短缺问题，台中，台南都收到了不同程度的波折，而且这种情况在台电投入新的发电机组之前无法改变，这一过程可能会持续数月。台湾的供水问题也出现了非常严峻的情况，已经对台湾的芯片产业和民生造成了巨大的影响。

民间对中华民国政府的批评非常频繁。生产和民生问题导致了中华民国政府的压力非常巨大，总统蔡英文的支持率达到了历史新低，不支持率也创下了历史新高。目前的状况对民进党政府来说已经构成了非常严重的执政危机。

目前中华民国国军的系统并没有爆出有疫情的问题，但是不排除国军中有和阿公店有交集的人员。即便没有直接接触到第一波感染源，被隐瞒的社群感染可能已经波及到了国军的安全，有可能对国军的战斗能力造成了一定的影响。

对于制和执政党等进行攻击。中共一贯的作法就是采取大外宣攻击，而这种攻击在民心脆弱的时候最容易起效果。所以台湾人民一定要格外小心，不要被中共的宣传所迷惑，任何在台湾内部宣传中共大陆政府的都是中共政府在台湾的代理人。中共诡计多端，但是只要台湾人民对中华民国的中流砥柱，也就这个国家的国体，主权，民主政治有信心，对台湾无数奋斗前线的医护人员有信心，对勇于牺牲的国军有信心，对视台湾核心利益的美国及其盟友有信心的话，台湾人一定能够战胜这一波中共病毒危机。
