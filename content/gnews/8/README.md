05月22日，据《卫报》报道，德国公共卫生研究所将英国列入中共病毒潜在危险菌株流行的国家名单之后，从5月23日起，德国当局将禁止从英国到德国旅行。

只有拥有德国公民身份的人将被允许从英国和北爱尔兰进入德国。此外，德国公民的近亲也可以进入德国，只要他们一起旅行。豁免也将适用于因严重原因（如失去亲人）而留下的人。

在英国本身，于5月17日开始了与中共病毒有关的第三阶段限制措施的解除。据首相鲍里斯·约翰逊说，接种疫苗的人数一直在稳步增加，所使用的疫苗显示出很高的有效性，住院人数也逐渐减少。

同时，约翰逊指出，新毒株目前并没有改变在国内的威胁情况。

根据现有的计划，英国当局还打算在6月21日取消大部分剩余的限制。

截至5月21日，英国有2160多万人接种了疫苗。全国共有近450万人感染了中共病毒。
