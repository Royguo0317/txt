## 9月6日，歷史上又多了一件不可忘記的事件
`2020-09-09 19:51 喜馬拉雅戰鷹團 Himalaya Hawk Squad` [轉載自GNews](https://gnews.org/zh-hant/344168/)

*作者、PAGE：玄天生｜校閱：Julia Win｜審核：海闊天空；*

**引言：歷史是由人創造的，而每一個重大事件都會被以不同形式記錄下來。 2020年9月6日，對於香港來說又是一個不同尋常的日子，不僅會成為被記錄的一天，還將成為令香港人永久紀念的日子。**

9月6日，本應該是香港人投票選出立法會成員的日子，但因為中共放出生化武器襲擊全世界、隱瞞中共病毒的真相和真實情況。今天，香港人再次走上街頭，再次反抗這個被中共利用及操縱的港共政府，反抗中共這個世界上獨裁血性兇殘的邪魔。可悲的是，警察們已經完全走火入魔，已經成為港共政府及中共用完即棄的工具。

圖：書生百用(文宣谷) 9月6日發生了什麼事！

從上圖可以看到，每一次遊行到最後，魔警就像是著了魔一樣，只要看見你是年青人；只要看到你穿黑衣服；只要你說了「光復香港，時代革命」，那你就是有罪，就是犯了《國安法》！面對極權，聲音就是我們手中的武器，行動就是我們對中共說「不」的表達。

革命道路千辛萬苦，每一次外出必定是槍林彈雨，危機重重。就算安全回家，你也要提心吊膽、不安及害怕，害怕的是你非常了解的中共的邪惡；害怕的是你會不會和「被消失」的手足一樣結果。但是恐懼是不能阻擋我們向中共說「不」的意志，也不能讓我們下一代活在一個以警治國、以黑治國、以邪治國的極權統治下。所以香港每一天發生的事件，正是我們的記憶、我們的歷史、我們的見證。

**入魔的黑警察**

![](https://spark.adobe.com/page/0LmHsy6oTzEEB/images/cb680ba1-703f-4f9b-af4c-afb08127a875.jpg?asset_id=323a4d28-20f5-4617-9522-facb10f67c92&amp;img_etag=%222ab7c26e33c2bb501549d2bc90e64274%22&amp;size=1024)

由中共控制的香港警務處和鳳凰衛視聯合製作的《暗夜星辰》，是一檔替黑警在元朗事件中犯下的罪惡洗白的節目，在節目中黑警說：“你們真的那麼恨我們嗎？”

[!\[\](https://spark.adobe.com/page/0LmHsy6oTzEEB/images/e4b290be-a4a1-4370-8fe4-21f7c8085a5e.jpg?asset_id=4bfbdf46-1cdb-4fbd-b384-9f5a04fa9ec8&amp;img_etag=%22dee99715a3db52d9b6a436b4eabfbc29%22&amp;size=1024)](https://spark.adobe.com/page/0LmHsy6oTzEEB/images/e4b290be-a4a1-4370-8fe4-21f7c8085a5e.jpg?asset_id=4bfbdf46-1cdb-4fbd-b384-9f5a04fa9ec8&amp;img_etag=%22dee99715a3db52d9b6a436b4eabfbc29%22&amp;size=1024)

就在節目之後的不到一星期，惡魔黑警就殘暴的對待懷孕的女子及12歲的小女孩（如圖所示）。看到這劃面，你還能有選擇嗎？你還能容忍嗎？你能不憎恨嗎？

當然，中共一直以來就在進行仇恨教育。讓人們對立；讓富人之間對立；讓當權者對立。只要人民永遠不找當權者，當權者又有權力控制人民時，人民永遠成為弱勢。只要人民不團結，只要他們互相指責，中共就達到了分而治之的目的。而同時，中共利用警察來轉移憤怒和視線，掩蓋中共接下來想做的一切，也好把責任推給港共政府，讓我們去恨那些自甘墮落、自甘出賣自己靈魂、走入魔道的工具-港共黑警。

現在的香港，只能容下中共的聲音

黑警知法犯法就不用說了，連香港沒有三權分立也已開口了。一國兩制就是一個笑話，一個騙取所有西方信任的一個荒謬承諾。可惜西方的好意卻養出一個中共魔鬼，一個草菅人命的邪魔。沒有法律的監控，沒有任何規條的部隊還能成為警察嗎﹖他們不過就是任意藐視生命、藐視法律，任意發洩的一群喪心病狂的魔警擺了。

魔警看少林足球看多了

其實自97回歸以後，香港這顆東方明珠就己被中共裹挾，漸漸失去了它的光芒，像中共國內地一樣，成了一個以警治國的城市，也演變成了一個惡警無理由隨意抓人的危險地區。真相被掩蓋，消失的人無法尋覓，被自殺的人還在沉冤待雪。一個只會造假、做惡、沒誠信的政府，法治早己被人治取代的城市，你還相信它有公正，有信用嗎？

不滅中共，世界無淨土。全世界感染中共病毒達到2千7百萬，死亡率也已接近90萬。世界就這樣不去查找真相，就這麼相信病毒來源於自然﹖多少科學家被中共腐蝕，甘願做中共的幫兇，甘願埋沒良心！你們要替中共背負這逝去的90萬生命之債嗎？

我們再回看9月6日。這一天，在香港的歷史時間軸上，是個重要的日子，本應該是香港人投票選出立法會成員的日子，它標誌著獨立於政權之外的法制體系進一步完善的日子，告訴世人香港是三權分立的，是有法治的。然而，2020年9月6日，中共治下的政府，以種種藉口，再次取消了標誌著香港法治的選舉，再次向香港人民施以中共政府的獨裁專制，其實也是再次向世界宣告了，香港法治已死！

對於香港人，這個日子意義的重大不言而喻。作為香港人，我們不僅僅會紀念，還會用我們的行動去抗爭，跟隨郭文貴先生以及引領的爆料革命，與世界正義力量一起，消滅中共！期待著香港的9月6日早日回歸！
