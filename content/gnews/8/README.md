## 真相還是謊言，是每人內心必須做出的抉擇
`2020-10-11 02:35 秘密翻译组G-Translators` [轉載自GNews](https://gnews.org/zh-hant/417076/)

![wengui_pray.png]()![wengui_pray.png](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/11022855/1-70.png)*圖片來源：路德社新聞簡報*
當閆博士的兩份病毒報告呈現給世人的時候，全球已經有三千六百多萬人被CCP病毒感染，因中毒死亡人數已經超過了一百萬人。這不單單是一個表面的數字，每一個數字的背後都是一個或者幾個家庭的支離破碎，對死者家屬來講都是一場人生中最不願意經歷的災難。當兩篇科學報告已經從正反兩方面證實CCP病毒就是超限生物武器的時候，你還選擇相信它是來自自然嗎？在你的人生經歷中有過一次自然變異的病毒會造成像當下這種全世界醫療體系癱瘓、經濟崩潰、人人自危的局面嗎？

當前的很多精英媒體泯滅良心為CCP站台攻擊閆博士，用盡洪荒之力讓大家相信病毒來源於自然。但是閆博士的報告已經得到相關權威部門及專業人士的充分肯定，所以現如今所有媒體的負面宣傳只能是希望讓真相被認清得再慢一點的掩人耳目的策略。但是所有的讀者和聽眾都應該具備一個基本常識就是，此時此刻，CCP必須用證據證明自己沒有製造生物武器，沒有發動生化戰爭。否則只能證明閆博士的報告是確鑿無疑的事實與真相，其它任何形式的攻擊、宣傳和所謂科學精英的站台都已經無濟於事。

讓我們重拾記憶回到2020年1月19日，當路德通過節目告訴大家這個病毒來自實驗室的時候，CCP是怎麼宣傳的，WHO又是怎麼宣傳的？我們看到、聽到了蝙蝠、穿山甲、蛇、三文魚，甚至還有宇宙其它星球參與進來，各種各樣離奇的推斷和結論。而爆料革命從那一天起到現在從未改變過病毒來自實驗室的說法。但凡有一點點辨別是非能力的讀者和聽眾都應該有一個判斷，知道誰在撒謊，因為撒謊的特點就是當謊言被揭穿的時候必須要用另一個謊言掩蓋原來的謊言，然後謊上加謊、越撒越謊，直至漏洞百出、惱羞成怒、蠻橫打壓。從看似精明而實在矛盾的謊言痕跡可以清晰地判斷，從一開始就是一個彌天大謊。

那麼基於此，有獨立思考的讀者和聽眾就會去分析為什麼我們會聽到一個又一個來自CCP的謊言，而沒有真相；為什麼真正的真相被曝光之後卻總是被惡意攻擊，但又拿不出有力證據證明被曝光的真相不是真相。只能有一種可能性就是CCP病毒從一開始即是生化武器。但是這個真相又不能讓大家知道，所以用盡其他邪惡手段打壓和詆毀閆博士。中共知道，真相一旦大白於天下，他們的邪惡計劃將落空，並受到全世界的司法審判和追責。

其實這是一個特別簡單的道理，為什麼還有那麼多的人願意相信病毒來源於自然呢？因為很多人已經喪失了獨立思考的能力，並且因為慾望名利而泯滅了是非善惡的判斷能力。在中共國的老百姓每天疲於奔命、養房養車、忙於治病，加上洗腦的虛假信息宣傳和極權體制，根本就沒有時間和基本的法治條件去思考什麼是真相。當他們自己的命運被統治政權奴役的時候，獨立思考的能力就已經萎縮了，統治者的思想通過國家機器的暴力壓制和宣傳欺騙就變成他們的思想。當西方的各個精英階層為了名利而配合中共掩蓋病毒真相的時候，他們也喪失了獨立的是非與善惡判斷能力，淪為中共的幫兇和共犯。

魯迅說：“當渾濁成為常態時，清醒便成了一種罪“。

梭羅說：“如今世界，謊言與幻覺被捧為至高的真理，現實反倒變得無比的荒唐”。

但是，現在閆麗夢英雄已經站在了舞台中央，她在拯救全人類。每一位內心尚存良善的人，你不再孤單無助，你只需要做出一個選擇：選擇加入真相的隊伍改變自己的命運，還是選擇膜拜權力的愚化繼續苟且任人擺佈。正義與邪惡的終極較量已經開始，我們每一個人都沒有第三個選擇！
