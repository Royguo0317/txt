布赖特巴特新闻网 (Breitbart News)5月13日报道，有报告称七家苹果供应商可能在中共国使用了强迫维吾尔族劳工。一名苹果发言人表示，“我们将继续尽一切努力保护员工，确保他们得到有尊严和受尊重的待遇。”

根据《资讯》最近的一份报告，几家苹果供应商据称在中共国使用强迫劳工。该情报组织与两个人权组织合作，确定了七家为苹果提供产品并据称支持强迫劳动项目的公司，其中包括针对维吾尔族穆斯林少数民族的侵犯项目。

据称，七家供应商中有六家参与了中共政府运营的工作项目。《资讯》指出，根据人权组织的说法，工人可能因拒绝参加工作计划而被监禁，而参加工作计划的人往往被迫运送到离家很远的地方。

苹果公司的一位发言人称，“在我们运营的任何地方都没有发现强迫劳动的证据。”苹果表示，它在进行的“每一次评估”中都会考虑强制劳动问题，并没有与参与此类项目的供应商合作。

苹果发言人表示，“我们将继续尽一切努力保护员工，确保他们得到有尊严和受尊重的待遇。”强制劳动的使用已经成为科技界的一个主要问题，很多零部件和产品都是在中共生产的。

今年1月，川普政府表示，中国共“对新疆以穆斯林为主的维吾尔族和其他少数民族和宗教群体犯下了种族灭绝和反人类罪”，引发世界范围内对其强迫劳动侵犯人权行为的关注和谴责。这不是苹果公司第一次被指控使用强迫劳工，去年12月就有报道称该公司与参与这些项目的供应商有过交易。
