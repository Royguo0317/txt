我們看看高福在視頻中都說了什麼：

【（視頻旁白略）溯源問題呢，中國政府、中國科學家一直在努力，我們在做著這方面的工作，我們基於跨種傳播的冠狀病毒，我們推測，這個病毒肯定最早是從（這是人類對冠狀病毒認知、知識積累推導出來的），它肯定是從蝙蝠來的，但是推導的呢，不應該是從蝙蝠直接感染的，它中間有個中間宿主，我們大家一直在找，但是中間宿主到目前我們還沒有找到，（記者問話略），新冠病毒現在推翻了我們好多認知，對我們好多知識的積累走不動了，也就是這個規律不按這個規律來做了。（記者旁白略），可能最早呢，我們推測的是在海鮮市場，可能有，但是現在看來海鮮市場本身也可能是受害單位，在這之前，這個病毒已經存在了。（記者旁白略），我們並沒有在這些動物裡邊發現新冠病毒，但確確實實在一些環境裡面發現了病毒，就象目前好多國家報導說，在他們的一些sewage一些下水道找到了一些病毒，就是我們呢，當時也在一些環境的廢水裡邊當時也分離出了病毒，所以呢，等於這個病毒呢，環境裡邊看來它持續的時間，比我們想像得要長。（記者旁白略）】

高福透露了以下幾點：

1、新冠病毒是跨種傳播；

2、新冠病毒來源於蝙蝠，但人類不是從蝙蝠直接感染的，是通過中間宿主感染的，但是中間宿主到目前還沒找到；

3、武漢海鮮市場本身可能是受害單位，原因就是海鮮市場的動物裡邊並沒有發現新冠病毒，而在海鮮市場的環境廢水裡分離出了病毒。

基於這三點，我來說一下我的分析，高福已經很明確的說明了海鮮市場可能是受害單位，也就是說它是新冠病毒的始發的，但不是發源地，關於發源地高福先生沒說，但是說在環境廢水裡分離出了病毒，什麼是環境廢水呢？以筆者不專業的角度來解釋就是，水分為上水和下水，也就是說從我們的生產、生活方面，上水是通過上水道輸送的，下水是通過下水道輸送的，而下水道輸送的下水分類有：雨水、污水和廢水，為什麼要重點說下水的概念呢？因為下水分為污水和廢水，而污水分為工業污水和生洗污水，而生活污水又分為廚房污水和廁所污水；那麼廢水是什麼呢？廢水包括工業廢水和生活廢水，**對於海鮮市場來說，它的廢水就是地面清掃廢水和海鮮販賣過程的冰水，以及冷凍物品融化後的廢水，請注意：它不是生活污水，不是糞便或者其他什麼下水，它是海鮮市場的環境廢水！作為生物研究方面的科學家，高福對於環境廢水的這個用詞是相當準確的！**
