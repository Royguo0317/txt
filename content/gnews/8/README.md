《紐約郵報》最新發布的電子郵件再次引起人們對喬∙拜登兒子亨特∙拜登的烏克蘭交易事件的關注，此事引發了民主黨在今年早些時候對特朗普總統的彈劾，但以失敗告終。

上個月，共和黨領導的參議院國土安全和財政委員會發布了一份為期一個月的聯合調查報告

(https://www.hsgac.senate.gov/imo/media/doc/HSGAC\_Finance\_Report\_FINAL.pdf)，

調查亨特·拜登在烏克蘭天然氣公司Burisma Holdings董事會中的角色，但委員會成員表示，他們的調查“面臨許多來自委員會民主黨人的障礙”，執行機構“未能按照文件要求進行”。

但在《紐約郵報》報導了一個據稱屬於亨特·拜登的硬盤，據稱其中包含喬·拜登和Burisma會面的電子郵件後，參議院國土安全委員會又展開了調查。

以下是亨特·拜登電子郵件報告的時間表：

2019年4月

位於喬·拜登家鄉特拉華州（Delaware）的電腦維修店店主John Paul Mac Isaac聲稱，待修筆記本電腦所有者是亨特·拜登（Hunter Biden），他（認為是亨特）將三部被水損壞的MacBook放到維修店，在其中一台筆記本電腦上發現了色情圖片。

Issac是一名註冊共和黨人，他告訴《紐約郵報》，他截取了一段12分鐘的視頻，顯示亨特∙拜登吸食非法藥物，並與一名身份不明的女性發生性行為。 Fox News沒有去證實這些說法。

Issac認為亨特·拜登是筆記本電腦的所有者，他在該設備上發現”Beau拜登基金會”的貼紙。他說，他無法核實電腦主人的真實身份，因為他在法律上是盲人，而且兩英尺以外的情況他看不清。

Issac說，鑑於俄羅斯對烏克蘭天然氣公司（Burisma）的黑客攻擊，他不認為自己是被一個邪惡的外國演員陷害的。

4月份，亨特·拜登還辭去了他在烏克蘭天然氣公司Burisma Holdings的董事會席位，同一個月，他的父親宣布競選總統。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/17042724/job-sbu.org_.png)圖片來源：job-sbu.org
2019年8月

90天過去了，筆記本電腦的主人還沒有付款，也沒有來取回，Issac說他無法聯繫到主人。與此同時，6月和7月彈劾（總統）鬧劇一直在加緊進行，因此Issac在電子郵件中對拜登與Burisma的交易進行了關鍵詞搜索。

據稱，Issac發現了據稱披露亨特·拜登向Burisma公司一名高管介紹其父親、時任副總統喬·拜登的電子郵件，而不到一年，喬·拜登就向烏克蘭政府官員施壓，要求解僱調查該公司的檢控官Victor Shokin。然而，美國和歐洲國家正在向烏克蘭施壓，要求解僱Shokin，原因是他缺乏控制國內腐敗猖獗的能力。

“親愛的亨特，謝謝你邀請我來華盛頓，給我一個機會去見你父親，並在一起度過了一段時間。這真是一種榮譽和快樂”，這封據稱來自Burisma顧問Vadym Pozharski的電子郵件寫的，日期是2015年4月17日，大約一年前，小拜登擔任董事會工作，月薪5萬美元。

一封據稱是Pozharski在2014年5月發來的電子郵件，要求亨特代表公司“就如何利用自己的影響力提供建議”。

喬·拜登否認與兒子的海外商業往來有任何牽連。

2019年9月

Isaac稱，就這些電子郵件的情況，他聯繫了一位中介，但他拒絕透露其身份。中介隨後聯繫了聯邦調查局。 Isaac說，中間人是他認識了幾十年的美國公民。

2019年11月

聯邦調查局來到Isaac的家中，複製了一份筆記本電腦的內容，但留下了設備原件。

2019年12月

聯邦調查局回來取走了筆記本電腦，但不是在Isaac拷貝硬盤內容之前。一名特工告訴Isaac，如果亨特·拜登來找筆記本電腦，告訴他筆記本電腦在儲藏室裡，通過短信通知聯邦調查局，他們會把筆記本送回來。

Isaac隨後聲稱，聯邦調查局的調查沒有下文了。

2020年初

當他聽不到來自聯邦調查局的消息時，Isaac說他聯繫了幾位國會議員，他們沒有回應。這時，他的中間人聯繫上了魯迪·朱利安尼的律師Robert Costello。

2020年9月

史蒂夫·班農向《紐約郵報》透露了硬盤的存在，魯迪·朱利安尼得到了一份硬盤的副本，他在星期天向《紐約郵報》提供了副本。

2020年10月

《紐約郵報》發布了關於這些電子郵件的報導。拜登陣營否認當時的副總統拜登和Burisma高管之間有任何此類會面。他們還指出，這篇涉及朱利安尼聲稱證據的報導是不真實的。

“紐約郵報從來沒有向拜登陣營問詢這個故事的關鍵要素，”貝茨繼續說，“他們當然從來沒有提到魯迪·朱利安尼，聲稱擁有此類材料。朱利安尼無根據的陰謀論，以及與俄羅斯情報部門有關人員結盟被廣泛報導。”貝茨補充道：“此外，我們已經復審了喬·拜登的官方日程安排，從未有像《紐約郵報》所說的那樣的任何會面。”

Twitter和Facebook也嚴格限制了這篇報導的傳播，Twitter援引了其黑客政策和私人信息政策，Facebook表示，這篇報導需要經過獨立事實核查機構的核實。

共和黨參議員特德·克魯茲、林賽·格雷厄姆和喬希·霍利週四呼籲Twitter和Facebook的負責人作證，並表示傳票正在進行中，因為右翼的批評者聲稱，社交媒體平台一直在審查批評民主黨的報導。

參議院司法委員會領導人宣布，他們將於週二投票通過一份傳票，要求推特首席執行官傑克·多西（Jack Dorsey）於10月23日（星期五）在委員會作證。霍利說，他希望委員會也能投票傳喚Facebook首席執行官馬克·扎克伯格。

面對來自各方的壓力，Twitter公司於16日發表聲明，將改變發推的限制政策。
