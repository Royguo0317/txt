**战友观点：**

中共通常借用媒体宣传为其统治阶级涂粉抹脂，尤其宣传其功绩卓著方面不余遗力。染指各种评选活动就更不足为奇了。

香港时下最热门的就是举着赤旗以民主自由维护治安的名义对异见人士的滥告滥捕。这些铺天盖地的事情吸引全球关注，忽略了一些看似不起眼的小事。

港台英文节目Backhat 像往年一样举办“年度人物”评选，这是港台年年都有的活动，很平常。和美国总统大选比较，简直就是一个天上一个地下，根本不可同日而语。这个活动是通过民意选举，网上投票方式得出结果，对胜出者为香港做出的贡献肯定，纯粹电视台的活动，但是选择归选择，到公布结果日却宣布取消了。香港已经不平常，难道这小活动也不平常了吗？
![]()![](https://gnews.org/wp-content/uploads/2021/01/港台2020年度人物评选中被取消17975274_401.jpg)图片：民调显示，香港人的“不快乐”程度创1992年以来新低
我们看看候选人都有哪些：港台记者利君雅，苹果日报，前线医护人员，清洁工，卫生防护中心传染病处主任张竹君，行政长官林郑月娥，退伍军人联合会会长蔡彼得，流行病学家高本恩，独立股评人David Webb，粉岭高尔夫球场。这些候选人里竟然有港共第一傀儡林郑月娥，恐怕想平常也不容易了。

有选举就有拉票，很正常的现象。特首拉票也是易如反掌的事。香港人最大的社区，从脸书上的Lihkg吹水台，时事台的几万条评论看出，某些人想要的结果并没有众望所归，而港台记者利君雅却成了“众矢之的” ，被亲共媒体攻击为“公器私用”

很多市民在查询选举结果《立场新闻》也向港台查询都无果。选举结果不得而知，只有从庞大的评论里可窥见端倪。据法广最新公布的林郑月娥8月26日民调显示：评分26.8分，民望净值负64%，其中53%的人给特首0分。德国之声今日（2021.1.4）公布特首民意民意，民望评分创1992年以来最低。评分29.7分，民望净值负53%，其中44%给0分，特首支持率只有18%，反对率却高达69%。从这些数据我们不难得到答案，另外，是谁有这么大的本事叫港台取消活动，这个大家也心知肚明。
