**西班牙2021年1月9日电/西喜社——**乔·拜登本周发誓要在任内击败“美国步枪协会”，这是他对自己打算推行的枪支政策做出的最直白的表态之一。

拜登是在2011年图森枪击案周年纪念日向美国众议员加比·吉福德斯发表讲话时提出这一主张的，吉福德斯在该案中头部中弹，另有6人最终丧生。

作为其枪支控制政策的一部分，拜登承诺将制定诸如对半自动武器的广泛监管，以及废除保护枪支制造商在其产品被非法使用时被起诉的立法。
