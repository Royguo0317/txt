**西班牙2020年12月4日电/西喜社/——**总统的法律小组，联邦调查局（FBI）已调查了资深技术工作者收集的选民数据。

一位曾任川普竞选工作人员的资深数据专家于周四向《新闻报》透露了广泛，昂贵和深入的选民欺诈调查，自11月3日有争议的选举以来，他和一组工人全天候沉浸在其中。

选民诚信项目负责人马特·布雷纳德（Matt Braynard）接受John Solomon Reports的播客采访说：“选举后，我有一些想法，我对如何解决选民欺诈或非法投票有确定的方式。”“因为，你知道，在大选后的第一天或第二天，会有各种各样的疯狂理论，我会考虑是传闻或猜测。”

“我以为，你知道，我在这里有一些技巧，”布雷纳德告诉记者:“我在这方面有很强的背景。”

自1990年代以来，川普2016年竞选活动的战略总监布雷纳德就一直在选民数据领域工作。他的履历包括在共和党全国委员会的政治分析部门以及选举数据服务公司（Election Data Services）的工作经历，布雷纳德将该公司称为“美国首屈一指的选举管理和重新划分公司”。

他说：“我对法案的产生方式不了解，但是我对投票的方式有非常深刻的了解，包括选民的登记方式，不同的投票方式以及点票和记票方式。这是我25年来一直非常熟悉的东西。”

据布兰纳德说，他在最近的选举中为探索选民欺诈所做的初步努力使他在GoFundMe网站上进行了一次大规模的众筹活动，筹集了超过20万美元的资金。 但是网站随后以涉嫌虚假选举信息为由，取消了众筹活动。

“我们从来没有说过我们到这里是来证明存在选民欺诈或证明这一点，或者我们将通过这次调查来推翻这次选举。”布雷纳德说。“我们一直在关注，’好吧，看，这里有一些技术可能发现问题。’”

布雷纳德说，随后在基督教众筹网站GiveSendGo上的筹款活动筹集了数十万美元。

**“你必须捡起石头”**

布兰纳德说，他和他的团队在拥有可观的预算的情况下，成立了一个主要的召集行动，以向认为自己选票可疑的选民们伸出援手。

他说：“我想要名字，还有电话号码，地址，签署的声明以及大量的宣誓书，因为这才是唯一真正重要的事情，”考虑到选举中获胜的几率。

布莱纳德说，他们的调查显示了大量可疑的选民数据。例如，他的团队与选民联系，“他们要求缺席选票，但根据该州的数据未返还这些选票”。

他说：“在没有退还缺席选票的人中，有很多人从未提出要求。我认为我们发现最多的是亚利桑那州。这些人中有44%从未要求缺席投票。”

他说：“我们还发现那些确实提出要求但没有退还的人当中，实际上有很多人说，’是的，我确实退还了它。’”

布赖纳德说，利用国家地址变更数据库和从一个地方移居到另一个地方的公民选民登记记录等数据源，他的团队发现了“相当大数量”的选民可能在他们不居住的州进行了投票。

布雷纳德说，他的团队发现了许多可疑安排的例子，包括“宾夕法尼亚州，乔治亚州，密歇根州的数千张选票”，据说选民将邮政设施列为他们的住所，这是违法的。

他还列举了威斯康星州的“无明确限制”选举政策，由于中共病毒大流行，今年这一政策得到了大大扩展，该政策允许选民提交不符合选民身份证要求的选票。

他说：“基本上，我们随机抽样了那些选民。而且我们发现，在我们可以识别社交媒体资料的人中，有44％的人并非无明确限制的。”

布雷纳德说，他的团队一直在“共享数据，并与一些总统的律师，他们的法律团队以及一些并行或独立工作的法律团体进行对话”，例如《阿米斯塔德计划》。

布雷纳德还指出，联邦调查局已与他的团队联系以审查他的数据。他说：“我要说的是我们将绝对合作。”

数据专家承认，他不确定在调查中投入的所有时间和金钱是否会产生过去几周出现的那种令人信服的证据。

他说：“你只有花完钱才知道。你永远不知道沼泽里的那块岩石下面是否有一堆令人毛骨悚然的爬虫，直到你把它捡起来。”
