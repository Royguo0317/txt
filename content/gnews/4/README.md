1921年，在浙江嘉兴南湖上的一艘小船中，中国共产党召开了第一次全国代表大会。当时参会的十几位成员中谁也不会想到，中共会在28年后成功夺取了中国的政权。笔者总结，中共的成功主要有以下四个因素：

1. 中共获得苏联的全面支持。中共的成立及其发展扩大，都是苏联在背后一手策划的。当时，苏联想把中共扶植为效忠苏联的工具，从而达到侵略中国的目的。为此，苏联为中共提供了各种经济、物质资源、技术及相关人员，中共也派出了大批高级人物去苏联进行学习、培训等。
2. 日本的侵华战争给了中共成长扩大的机会。1931年爆发的“九一八事变”开启了日本的全面侵华战争。这场长达14年的战争，使得本已濒临死亡的中共得以喘息，并从中茁壮成长。中共打着“抗日”的宣传旗号，暗中扩大自己的实力，抗日战争十四年间，中共就由两万人扩展到120万人。
3. 中共利用谎言取得民心。“自古得天下者，必先得民心”——中国共产党虽然成立时间不长，但他们深知民心的重要性。当时的中国民众饱尝艰辛，他们迫切希望过上自由、民主、富足的生活。于是，中共利用民众对于“国家富强”、“民主自由”、“生活改善”等强烈要求，提出了一个又一个响亮的宣传口号，成功欺骗了广大人民群众，最终达成了其掌权的目的。
4. 国民政府的腐败堕落及国民党内斗。1945年二战结束后，国共两党都试图一统中国。当中，国民党依赖美国以及城市有文化的中高层菁英，而共产党依赖苏联和广大农村的贫苦百姓。由于国民党长期以来的贪腐已经到了病入膏肓的境地，导致国民政府全然瘫痪，加之经济动荡、制造恐怖氛围等等因素，最终导致中共成功掌权。如果国民党能彻底贯彻“三民主义”，那么中共肯定不会取得胜利。


中共掌权的70余载，中国人民不但没有过上自由、民主、富足的生活，反而遭受了空前的灾难。当下中共官僚贪腐、经济衰退、言论管控，种种高压政策导致民不聊生，人心思变。这些迹象都意味着这个时代的衰落。遵循历史发展规律，一个时代的没落必将会被另一个时代所取代。

当今的二十世纪，人类民主自由是这个时代必然的要求。中共宣传的“民主”从来都是是谎言，从未实现过一天。中共如今的集权专政是有违时代发展潮流的，只有去除中共这个“毒瘤”，中国才能向着正确的方向发展。人民只有获得民主自由，才能过上幸福美好的生活。
