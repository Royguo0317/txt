**西班牙2021年3月2日电/西喜社——**据澳大利亚卫报3月1日报道，在加拿大和荷兰通过类似的议会动议后，一名澳大利亚参议员将寻求上议院其他议员的支持，承认中共国对维吾尔穆斯林少数民族的待遇是种族灭绝。

南澳州独立参议员帕特里克（Rex Patrick）表示，这项拟议的动议—放在参议院3月15日的通知文件上—在澳大利亚应该与国际社会一起采取立场的时候，这对主要政党是一个考验。

虽然莫里森政府迄今拒绝将中国新疆地区的情况描述为种族灭绝，但外交部长玛丽斯-佩恩却对 “一些非常可怕的报道，特别是关于强迫劳动、关于再教育营、关于系统性酷刑和虐待妇女的指控 “表示关注。

帕特里克的动议草案指向加拿大下议院上周采取的步骤，以及美国政府自己早先的认定—在拜登政府下仍然有效—中华人民共和国（PRC）对新疆维吾尔人实施了种族灭绝。

提案称，参议院 “同意中共国对新疆维吾尔人的待遇构成种族灭绝罪”。

它还呼吁中共国 “立即停止拘留中心的酷刑和虐待；废除大规模拘留营、软禁和强迫劳动的制度；停止所有强制性的人口控制措施；停止在新疆和中国其他地方对维吾尔人和其他宗教和少数民族的迫害”。

主要党派有可能试图从形式上拒绝帕特里克的动议，或者以动议涉及复杂的外交政策问题为由拒绝他提出动议。但是，如果这种情况最终发生，这位参议员计划寻求暂停执行议事规则来提出动议，他认为这将显示出大家在这件事上的立场。

帕特里克把他的动议的处理看作是对整个参议院的考验，同时也是对所谓的 “金刚狼”（Wolverines）的考验—这个由中国鹰派组成的团体主要由联盟议员组成，但也包括一些工党代表。

“我们现在看到整个国际范围内的国家都站出来，宣布新疆发生的事情是种族灭绝，澳大利亚需要在其中扮演一个角色，”帕特里克告诉澳大利亚卫报。

“底线是，有一系列国际公认的事实，导致其他国家政府以这样的方式做出回应。我们与那些站出来说些什么的国家有着共同的价值观。由于中共国的外交和经济实力，我们需要一个多边的回应。”

佩恩上周没有回应澳大利亚卫报关于澳大利亚是否考虑将中共国的行为贴上种族灭绝的标签的问题，但她在周一告诉天空新闻。”我们对这一转折的说法有稍微不同的态度。我的意思不是说这是一种学究式或语义式的方式，但英国和加拿大都有不同的机制来做出这样的声明，事实上美国也是如此。但这是我们正在仔细研究的事情。”

佩恩还重申，澳大利亚呼吁中共国允许联合国人权事务高级专员米歇尔-巴切莱特 “适当进入新疆”。

这与英国外交大臣多米尼克-拉布（Dominic Raab）的呼吁类似，后者周一表示，针对维吾尔族穆斯林的人道主义犯罪正在中共国以工业化的规模进行。中共国当局和巴切莱特的团队仍在就这样一个事实调查团的条件进行谈判。

中共国政府多次表示，在新疆发生的事情 “没有任何下作”，外国政府和媒体应该 “停止在新疆相关问题上对中共国的无端诋毁”。

在参议院复会时通过拟议的动议，将有可能重新引发与北京的紧张关系。

但人权观察组织的澳大利亚主任皮尔逊（Elaine Pearson）表示，最近英国、荷兰和加拿大的强烈声明 “应该鼓励澳大利亚政府考虑可以采取哪些进一步的行动来解决发生在新疆的一连串严重侵权事件”。

“例如，澳大利亚应该支持通过联合国人权理事会建立一个新的对中共国的监督和报告机制，”皮尔逊说。

参议院就这一问题进行投票的前景，正值莫里森政府面临着后座议员要求与台湾谈判自由贸易协定的呼声—这是对北京高度敏感的另一个问题。

昆士兰自由国民党参议员马特-卡纳万（Matt Canavan）表示，新西兰已经与台湾达成了贸易协议，如果澳大利亚也追求一个贸易协议，将是 “非常有意义的”。

卡纳文—上周与台北驻澳大利亚经济文化办事处代表常以立(Elliott Charng)会面—他说，鉴于中共国最近几周对这个民主管理的岛屿升级了军事压力，他为台湾感到难过。

“我们总是需要阻止其他国家采取侵略性的行动，而台湾是我们国家的好朋友，”卡纳文告诉澳大利亚卫报。

“虽然我们面对的是来自中共国的一系列大不相同的威胁，但我们也受到了中共国政府无理和非法贸易制裁的影响。”

新南威尔士州自由党参议员Concetta Fierravanti-Wells此前曾呼吁澳大利亚采取进一步措施，正式承认台湾是一个国家，她说:”毋庸置疑，我非常乐意支持与台湾达成自由贸易协定。”

维多利亚州工党参议员基钦（Kimberley Kitching）2018年曾随安东尼-阿尔巴内塞（Anthony Albanese）率领的议会代表团前往台湾，他说台湾是 “一个繁荣的民主国家，一个优秀的全球公民，也是该地区战后经济成功的光辉典范”。

担任参议院外交、防务和贸易参考委员会主席的基钦说，贸易多元化 “应该不仅仅是寻找我们可以增加澳台之间双向贸易的方法，还应该将台湾纳入其他多边论坛和非正式协会”。
