# **12家日企将终止涉维吾尔族强迫劳工的业务**

日本共同社周日的一项调查显示，鉴于新疆维吾尔少数民族被强制劳役的事实，12家日本大企业已推出政策，若发现有中共国公司从强迫劳工中获利，将立刻终止与其所有业务往来。

美国和英国对原产于新疆自治区的棉花和其他产品先后实施了进口限制，日本企业面临巨大压力，必须对供应链中的这种侵犯人权的行径采取行动。

[!\[\]()!\[\](https://spark.adobe.com/page/80YfwaRj3Oijv/images/38a8023b-b38f-41cb-8e97-1a5c68d52232.png?asset_id=0d5c8232-3693-496a-8489-d4698c0c929b&amp;img_etag=%229935cfa3b3fbdbb570e4926d01e3ed45%22&amp;size=2560)](https://spark.adobe.com/page/80YfwaRj3Oijv/images/38a8023b-b38f-41cb-8e97-1a5c68d52232.png?asset_id=0d5c8232-3693-496a-8489-d4698c0c929b&amp;img_etag=%229935cfa3b3fbdbb570e4926d01e3ed45%22&amp;size=1024)2019年12月22日，民众在香港参加谴责中共国涉嫌侵犯维吾尔人人权的集会（共同社）

日本政府因迟迟不实施类似制裁而受到批评，但由于担心激怒中国，日本政府在处理这一问题上一直处于被动。

澳大利亚智库“战略政策研究所”在去年的一份报告中指出：**“80多家全球公司通过虐待性的劳动力转移项目，直接或间接受益于新疆以外地区维吾尔族工人的使用。”共同社最近询问了该智库提到的14家日本公司，它们打算如何应对。**

除松下公司拒绝置评外，其他企业要么否认与涉嫌受益于强迫劳动的企业有直接业务往来，要么表示无法核实对其供应商的指控。

谈到今后的政策，有12家公司回应称，如发现其商业伙伴使用强迫劳动，将终止或考虑终止与其的业务往来。

与一家涉嫌使用强迫劳动的公司签订了许可合同的东芝公司表示，无法确认这些说法，但已决定在年底前终止与该公司的业务往来。

这12家企业还包括休闲服装品牌运营商优衣库、索尼公司和日立公司。

对人权情况的了解，很多外资企业依赖于中共国企业的自身报告，而对直接商业伙伴以外公司的人权情况如何掌握，对外资企业又是一大挑战。

三家日本企业与被美国政府禁运的中共国企业或这些企业的母公司有业务往来，其中之一是无印良品连锁店运营商Ryohin Keikaku（良品计画）公司。

“我们所有的棉花和纱线都已经通过了第三方的国际有机认证。”该公司说。

这家零售商的官方网站曾上架标有“新疆棉”的产品，但在经历共同社调查后，这些产品被删除。
