2021年2月14日郭先生在Gclub 会员的第一次在线直播中，回答了会员提出的一个关于新中国联邦护照进展的问题。郭先生回复说目前还在确定购买岛屿的阶段。新中国联邦的成立和购买岛屿什么关系？对于一个新政权的成立，现行国际法的规定又是什么？笔者对一个新国家政权的成立所需要的流程和规定进行了简单的梳理。

**一）**** ****联合国是否对新国家政权有决定权？**

首先，联合国对于会员资格的规定链接如下：https://www.un.org/zh/sections/member-states/about-un-membership/index.html

《联合国宪章》规定：“凡爱好和平的国家，接受《联合国宪章》所载义务，经本组织认为确能履行这些义务，都可以成为联合国会员国。”申请加入联合国的国家须经安全理事会推荐，再由联合国大会决定，才能获得联合国会员国资格。

**联合国既不是国家，也不是政府，因此无权认可一个新的国家或政府。**只有其它现有的国家或政府才拥有承认一个新的国家或政府的权力，承认与否通常表明这些国家或政府是否愿意与其建立外交关系。但是，联合国作为一个独立国家构成的组织，它可以决定是否接纳一个新国家成为其会员国，或者接受一个新政府全权代表的证书。

那么一个新的国家或政府如何获得联合国的承认？申请程序大致如下：

1）申请国向联合国秘书长提交一份申请，同时提交一封正式声明接受《联合国宪章》所载义务的信函。

2）安全理事会审议申请。安理会15个理事国中至少有9个投赞成票，并且5个常任理事国（中共国、法国、俄罗斯联邦、大不列颠及北爱尔兰联合王国和美利坚合众国）没有任何一个投反对票，才能通过任何接纳新会员的建议。

3）如果安理会建议接纳该国，则该建议将递交联合国大会审议。但必须有三分之二多数会员国投赞成票，联合国大会才能接纳一个新会员国。

二）**联合国关于观察员的规定**

链接：https://www.un.org/zh/ga/about/index.shtml

从以上内容我们可以看到，联合国并没有权力认可一个新的国家或者政府，但是可以对非会员国、实体和组织发出邀请作为联合国的观察员。然而在现行的规定下，通过正常的法律途径新中国联邦是无法在联合国获得注册会员国和观察员身份的，因为窃取了常任理事国的中共肯定会使用一票否决权，否决新中国联邦的会员申请，并拒绝邀请新中国联邦成为联合国的观察员。

**三）有关新国家政权成立的国际法**

目前关于新国家政权成立的国际法只有一个，就是联合国注册文件3802 号的《蒙得维的亚公约》。公约链接如下：https://treaties.un.org/pages/showdetails.aspx?objid=0800000280166aef

公约中规定一个新的国家政权成立的必备条件:

1. 固定的人口
2. 清楚的领土边界
3. 政府
4. 与其它国家建立关系的能力


从以上内容我们可以确定，国际法规定新的政权成立必须要有自己的领土。

**四）新中国联邦在国际法下的发展**

首先，联合国并没有资格承认或不承认新中国联邦，只有其它国家政权才有资格。

其次，只要中共国继续窃取中国人在联合国的安理会席位，新中国联邦就无法作为一个独立的国家或者政治实体参与联合国事务。

再次，按照《蒙得维的亚公约》，新中国联邦必须要有自己的领土，无论领土的大小，这也是郭先生一直在买岛的原因。现在我们拥有共同信仰的战友或者Gclub 会员，已经具备了成立一个新国家的固定人口的条件。农场联盟委员会通过发展后也非常容易成为一个政府组织。

目前的难点在于与其它国家建立外交关系的能力，台湾就因此而处于一个尴尬的国际政治角色。但是国际关系的本质就是利益，郭先生创建的G系列的金融能力、媒体的话语权、多年的国际关系以及对情报的获得和分析能力，已经具备了和其它国家建立平等关系的能力。郭先生在美国大选结束后也说到，今年的发展重点是GTV 、Gcoin和G系列。

新中国联邦从国际法的法理上，一旦拥有自己的领土就具备成立新国家政权的条件。此外，病毒的肆虐和危害也会促进新中国联邦在国际上被认可的进程。相比1989年的六四事件，目前新中国联邦成立已经具备天时、地利和人和。
