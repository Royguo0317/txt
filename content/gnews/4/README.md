# **美国称北京对香港的举动是对自治的“直接攻击”**

上周五（3月5日），美国谴责中共国提出的对香港选择立法会议员的新否决权是对香港自治的“直接攻击”，并要求北京改弦易辙。

美国国务院发言人内德•普莱斯（Ned Price）对记者说：提议的措施“是对香港的自治、香港的自由和民主进程的直接攻击”。
