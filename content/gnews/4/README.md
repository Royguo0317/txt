# **美航母战斗群在中台紧张局势期间进入南中国海区域**

[!\[\]()!\[\](https://spark.adobe.com/page/9MV0XQFWECf9w/images/16d99ebd-e91e-43d4-b4e5-f69e97e418d7.png?asset_id=b8886529-fda6-416d-9ca0-a67ee0c02b5d&amp;img_etag=%22e5b1730a2353d613cae5bb4d781bcce2%22&amp;size=2560)](https://spark.adobe.com/page/9MV0XQFWECf9w/images/16d99ebd-e91e-43d4-b4e5-f69e97e418d7.png?asset_id=b8886529-fda6-416d-9ca0-a67ee0c02b5d&amp;img_etag=%22e5b1730a2353d613cae5bb4d781bcce2%22&amp;size=1024)

根据美国军方信息，在中共国和台湾紧张局势升级之际，一个美国航母战斗群周六进入南中国海区域，意图促进“航海自由”。

根据美国印太作战司令部的声明，罗斯福号航空母舰正带领由邦克山号驱逐舰 (USS Bunker Hill)、拉塞尔号驱逐舰 (USS Russell)和约翰•费恩号驱逐舰 (USS John Finn)组成的战斗群。

“在我30年的生涯中驶过诸多海域之后，回到南中国海感觉很好，进行例行演练，促进自由航行，并且向盟友和伙伴提供保障。” 海军少将道格•维里西莫(Doug Verissimo) 在声明中说。

“全世界有三分之二的货物要通过这个非常重要的地区，至关重要的是，我们要保持我们的存在，并继续促进基于规则的秩序，正是这种秩序让我们所有人都繁荣起来。” 他补充道。

美联社报道，航母战斗群是在台湾报道中共国轰炸机和战斗机进入台防空识别区的同一日进入了南中国海。

美联社报道，台湾为应对中共国的行动紧急起飞了战斗机，并且广播了警告，并且以 “部署防空导弹监视飞行活动” 作为反击。

美联社指出，中共国的飞行是向台湾总统蔡英文施压，迫使其承认北京对该岛的主权的更大行动的一部分。

在美联社获得的一份声明中，国务院发言人爱德华•普赖斯 (NedPrice) 说：“我们敦促北京停止其军事、外交和经济对台施压，转而和台湾民选政府进行有意义的对话。”

彭佩奥在担任国务卿期间的最后举措之一是取消了美国和台湾官员之间的会晤限制，打破了数十年来自行施加的限制，并可能引发北京方面的愤怒。

拜登总统的国务卿候选人托尼•布林肯 (Antony Blinken) 在参议员听证会上称，中共国对美国造成“最严峻挑战”。

布林肯也认同彭佩奥对中共对维吾尔穆斯林少数民族进行“种族灭绝”的认定，并且呼吁美国重新加入国际组织以对抗北京的强势威胁。他还发出信号，支持投资美军以遏制北京的威胁。
