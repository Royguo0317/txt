5月28日（周五），拜登在位于汉普顿市的兰利—尤斯蒂斯联合基地（Joint Base Langley-Eustis）对部队说：“我们正在打一场民主与独裁的战斗。”

“我跟中共国主席习近平相处的时间比世界上任何其它领导人都多——我跟他在一起进行了24小时的私人会晤，期间只有一名翻译在场；我还跟他在中国和美国旅行了17000英里。”拜登接着说，“他坚定地相信，中共在2030年或2035年之前，将吃掉美国，因为独裁者可以快速做出决定。”拜登没有详细解释“吃掉美国”是什么意思。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/01211645/Untitled-design-70.png)
拜登自担任美国总统以来，在面对中共国的问题上，拜登一直都是打太极。这是拜登首次清晰明确地表示对习近平的评价——习近平就是一个独裁者，而且，这个独裁者企图或者正在蚕食美国。同时，拜登不再回避他与习近平的关系，他坦承世界上没有任何一个国家领导人比他更了解习近平，这证明了他对习近平评价的正确性和真实性。这是一个明确的信号—拜登与习近平割席的信号。

如之前小编所说，美国国会两党对反制中共国的广泛共识，必定会对拜登政府产生重大的影响。与习近平割席和反制中共，就是拜登的选择。相比国会山的忙于制定反制中共的法律，拜登把和中共的较量，直接定性为“一场民主与独裁的战斗”，美国将对中共国采取更强硬的战略。

拜登代表美国政府所宣示的：美国对中共国的战斗，是为保护自己而战，是为保卫民主而战，美国是为正义而战。美国必须为赢得这场战斗而作好准备。
