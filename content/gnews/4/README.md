## 歐洲開始展開對邪惡軸心國的製裁
`2020-07-31 15:21 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/281578/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/31151541/1596222854240.jpg)圖片來自：twitter 
據布魯塞爾（路透社）週四報導，歐盟對涉嫌參與全球主要操縱網絡攻擊的中共國，俄羅斯軍事情報部門和朝鮮的企業實施旅行和金融方面的製裁。歐盟的一份聲明中表示，在首次針對網絡犯罪的製裁中，歐盟將俄羅斯軍事情報特殊技術部門作為主要目標，該部門被稱為俄羅斯聯邦武裝部隊總參謀部。

歐盟譴責俄羅斯軍方在2017年6月操控了兩次網絡攻擊，對歐洲的幾家公司造成了巨大的經濟損失，同時俄軍方還被指控在2015年和2016年發動了針對烏克蘭電網的兩次網絡攻擊。

為俄羅斯軍事情報局工作的人員也因涉嫌參與2018年4月對荷蘭的“禁止化學武器組織”（OPCW）的網絡攻擊而遭到製裁。

朝鮮企業朝鮮博覽會（Chosun Expo）也因涉嫌支持黑客組織拉撒路集團（Lazarus Group）而遭到製裁。拉撒路集團被認為要對全球一系列重大襲擊事件負主要責任，其中包括2016年針對孟加拉國銀行在紐約聯邦儲備銀行賬戶的8,100萬美元搶劫案，該案是世界上最大的網絡詐騙案。據稱，朝鮮博覽會還與對好萊塢電影製片廠索尼影業的黑客攻擊有關，其攻擊主要是為了阻止2014年上映有關朝鮮領導人金正恩的諷刺電影（電影名：《The interview》）。美國財政部去年就對拉撒路集團和其他兩個朝鮮黑客組織實施了製裁，理由是它們涉嫌參與對索尼影業和孟加拉國中央銀行的黑客攻擊。報告稱，朝鮮的主要情報部門是那次黑客攻擊的幕後黑手。但朝鮮否認參與任何網絡襲擊。

歐盟的製裁也打擊了中共國的公司海泰科技發展有限公司（Haitai Technology Development）。該公司被指控參與了竊取全球跨國公司商業敏感數據的名為“雲漏斗行動” （Operation Cloud Hopper）的黑客襲擊。兩名涉嫌參與行動的中共國員工也同時遭到了製裁。制裁措施包括實施旅行禁令和資產凍結，禁止歐盟個人、公司以及其他的實體向被列入黑名單的人提供任何資金支持。

中共國駐歐盟外交使團在周五的一份聲明中辯稱，中共是網絡安全的堅定捍衛者，也是黑客攻擊的最大受害者之一。希望通過對話與合作而不是通過單方面製裁來維護全球網絡空間安全。
