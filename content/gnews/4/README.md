To我爱的xxx：

我想问问大家对CIC青山湾入境事务处中心有多少了解呢？这是由入境处管辖，用作羁留没有香港合法身份的人，之前没有人关注过CIC，直到2019年的抗争运动中，有一位报道抗争的印尼女记者yuli因「违反居留条件」而被拘押在CIC才引起外界关注，这里面的环境只能说比监房更严峻和更恶劣。在CIC呆过的人都知道我们墙内手足的事，最近我收到来自CIC的信件，小弟我想和大家分享一下。

其中一位朋友已经在香港的监房被关押了一段教长日子，所以分享了五点：

1. 善待身边人，和谐共处

2. 要尝试找到令自己开心的事，接受不能改变的事，多和身边人联系，多沟通，多分享，千万不要轻易和人起争执

3. 尝试找个运动锻炼头脑和筋骨，避免冲突

4. 相信明日一定会更好，所有眼前的困难都会有结束的一天

5. 面对苦难，要保持正面，要勇敢，有信心，要坚强，神会为自己开路，神总会在自己身边
![]()![](https://gnews.org/wp-content/uploads/2021/04/image002-47.jpg)
非常感谢在CIC呆过的朋友，尽管面对严峻的环境，都坚持写信鼓励我们，在监房呆过的人都清楚心里肯定有很多的不开心。其实以上五点，在日常生活之中，都是通用的，我们都知道（如果你没有宗教信仰，第五条就当作是你自己的信仰就好了）但往往知易行难，就是一个讲出来非常浅显的道理往往做起来时你会发现困难重重。以上这五条，以我入册的经验，都只是停留「知道」，但却「不明白」(即知其然不知其所以然)。但知道这些方法可以帮助你面对困难，就算不明白，在你面对困难时都有助从中学到面对。

身在监牢的日子时间的确不好打发，但我每一次一想到在内地和在CIC关押的手足同胞，他们面对的是更加严峻恶劣的环境，对比他们我自己真的没资格去讲「难挨」这个词。就好像之前的那位手足所说，如果自己被关押在大陆，他都不敢保证自己是否还有勇气去为民主奋斗，老实讲，我也不敢保证。所以今天当我收到CIC朋友的来信，真的非常惊喜。他们在如此困境中还在关心我们，但相反自问，我们又有没有去关心他们……就好像我们把坐监做借口冷待他们，他们习惯了冷漠…其实除了CIC，小弟我也收过来自cat A(刑期长的狱友)的支持信。他们身处cat A，被关押了廿十几年，但依然非常关心社会，也心系墙内手足，觉得自己什么都没做过。

另外一位CIC的笔友提醒我，「你在那里只是停留一会儿，因为那里并不是你的家」。这一句话小弟一直紧记心中，提醒自己不要迷失，眼前困难只是考验，不要自怨自艾太多，不要太执着。始终，总有回家的一日。
