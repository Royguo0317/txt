## 香港首次報導冠狀病毒再感染病例 該名男子去西班牙旅行後正返回香港
`2020-08-24 23:26 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hant/315495/)

編譯:玫瑰天空

據福克斯新聞於當地時間8月24日報導,香港發生第一例[COVID-19](https://www.foxnews.com/category/health/infectious-disease/coronavirus)病患再感染病例。 據[多份報導稱](https://nypost.com/2020/08/24/covid-19-researchers-record-first-case-of-reinfection/),一名33歲的男子在4月份從冠狀病毒中康復後,於本月再次染上了[新型冠狀病毒](https://www.foxnews.com/health/coronavirus-happens-less-children-adults-study)[。](https://nypost.com/2020/08/24/covid-19-researchers-record-first-case-of-reinfection/)

據《[日本時報》報導](https://www.japantimes.co.jp/news/2020/08/24/asia-pacific/science-health-asia-pacific/hong-kong-first-coronavirus-reinfection/),這名男子是在去西班牙旅行後返回香港的。 當時,香港大學的研究人員說,他在機場進行的一次篩查中檢測出該病毒呈陽性。研究人員告訴《日本時報》,這個再感染病例標誌著「世界上第一例從COVID-19中康復的患者,此後再次發生了COVID-19感染的實例證明”。據報導,使用基因組序列分析,研究人員能夠檢測到該患者感染了兩種不同的冠狀病毒。 根據《日本時報》,患者沒有出現第二次感染的癥狀,這表明「隨後的感染可能較輕」。

香港大學的研究人員在一份聲明中說:”在這份報告之前,許多人認為康復的COVID-19患者具有抵抗再感染的免疫力,但是,有證據表明,某些患者在幾個月後抗體水準逐漸下降。  “

該大學的臨床微生物學家凱爾文· 。 凱旺· 。 杜博士(Kelvin Kai-Wang To)告訴[《紐約時報](https://www.nytimes.com/2020/08/24/world/covid-19-coronavirus.html)》:”我們的結果證明,他的第二次感染是由他最近獲得的一種新病毒引起的,而不是此前的老病毒 。  “

據《日本時報》報導,世界衛生組織加大了這項研究的力度,該報告指出,世衛組織的技術負責人瑪麗亞· 。 凡凡凡 。 科霍夫告訴記者,記錄此類案件很重要。但是,她警告說「不要作出任何結論」,並指出,隨著時間的流逝,需要對病例進行更多的研究,才能更好地瞭解患者對新型冠狀病毒Sars-Cov-2的抗體反應。傳染病專家告訴福克斯新聞(Fox News),有傳聞說可能會再次感染,但這是第一份涉及科學證明病例的報導。

加利福尼亞州傳染病專家Ravina  Kullar博士說,該報告令人擔憂。庫拉爾說:「先前已有關於再感染的報導,但沒有經過嚴格測試證實的情況。  “

美國傳染病學會(IDSA)成員的庫勒(Kullar)解釋說,從新型冠狀病毒中康復的患者需要數周才能擺脫病毒遺留的碎片痕跡,即使該病毒不再存活,其檢測結果仍為陽性。 但是庫拉爾指出,這名33歲男子的案子是另一回事。”該報告從兩輪感染中對病毒進行了測序,發現兩組病毒之間存在差異,表明該患者可能第二次被感染。 這可能支援以下發現:某些感染COVID-19的患者在短短幾個月後抗體水平逐漸下降。  “

紐約長島的流行病學家亞倫· 。 格拉特(Aaron  Glatt)博士也發表了類似的聲明,對福克斯新聞說這起案件很重要,因為該病毒的再感染是「另一種COVID-19毒株,因此不太可能仍然可以檢測到第一次感染疾病時的可致死病毒。  “也是IDSA成員的Glatt告誡要保持調查結果的正確性。

“雖然這清楚地表明瞭我們懷疑-COVID-19免疫力可能不是終身的-但必須意識到,這是繼全球超過20,000,000例COVID-19病例後的第一例再次感染。 因此,實現免疫至關重要,也許至少在短期內。 即使這種免疫不會永遠持續下去,但免疫實際上也是相當好的。  “
