#### 1、**埃瑞克·格雷滕斯：2020，以色列的遭遇，我们选举的后果**

嘉宾发言由美国参议院候选人埃瑞克·格雷滕斯（*Eric Greitens*）提供文本。

作为以色列的美国朋友，我们现在看到了民主党人执政带来的严酷后果。不久前我们还看着客机从不同的阿拉伯国家起飞，首航以色列。那是川普政府的和平协议–《亚伯拉罕协定》的成果。随着哈马斯的导弹向以色列的倾泻而下，事态再清楚不过地表明，今后我们需要推选出的领导者应该是意志坚强的斗士，能和我们的盟友坚定地站在一起。

作为川普促成的和平协议的直接结果，以色列和阿拉伯世界之间广泛的关系正常化的重要步骤具有历史意义。川普总统、他的高级顾问库什纳（Jared Kushner）和国际谈判特别代表阿维•贝尔科维茨（Avi Berkowitz）因在实现中东和平方面所做的努力，足以被提名诺贝尔和平奖。

可悲的是，仅仅几个月之后我们惊恐地看到，曾经熟悉的悲剧更悲惨地降临到了以色列，那就是针对平民的大规模导弹袭击。拜登政府上台后立即戏剧性地将重心从以色列移开，以一种自上一次民主党软弱执政以来见所未见的方式，重新点燃了以和巴之间的冲突。

川普总统在奉行他的“美国优先 ”策略时考虑周全，正如他自己所言：“美国优先并不意味着只有美国”。他通过将美国驻以色列大使馆迁往耶路撒冷并承认耶路撒冷为以色列首都的历史性行动，向世界发出了一个明确的信息——美国与以色列站在一起。

川普总统在支持以色列和挑战固有正统观念之外还更进一步，他播下了和平的种子。与其他绝大多数西方国家领导人不同，那些领导人喜欢对中东冲突采取 “退后”或“背后引导”的方式，而川普总统则表现出毫不含糊的强势。他切断了美国对巴勒斯坦权力机构的无条件财政支持，要求美国的资金援助必须换来善行与和平。他还确认以色列的军事反应不是施暴，而是对恐怖组织和极端分子有节制的报复，因为那些人公开宣称要将犹太民族和犹太人从地球上抹去。

虽然川普总统坚定地支持以色列，但这并不妨碍他也以尊重的态度对待它的阿拉伯邻国。在此过程中，川普政府能够寻找共同点，巧妙利用 “美国优先 ”的力量来实现历史性的中东和平协议。

可悲的是，美国2020年大选的后果迅速波及全球，我们的伟大盟友以色列现在正处于被围攻的态势之中。拜登政府不仅放弃了孵化和平和确保该地区安全的政策和姿态，乔•拜登还反过来迅速开始资助巴勒斯坦权力机构。

往轻了说，拜登政府现在是在助力针对我们伟大盟友以色列的恐吓行为；往重了说，他们就是在直接资助那些恐怖活动。

拜登团队还开始对伊朗的恐怖政权打擦边球，表明他们想重回灾难性的伊朗核协议。哈马斯身后站着的巴勒斯坦权力机构和伊朗，考虑到拜登政府最近对这两者的妥协行为，这些恐怖袭击不可否认是拜登政府软弱政策的直接后果。

这就是为什么我们需要把拥有该地区实际经验的斗士送进国会中，就是那些服务于犹太社区，并且有勇气和力量反对左派的反犹主义的人。我们不能再支持更多鹦鹉学舌的建制派共和党人。美国需要的是有勇气的领导团队。在2022年我们有机会遏制拜登政府，重申美国对我们盟友的承诺，并回击任何阻碍和平的邪恶力量。

埃瑞克•格雷滕斯是曾在中东地区服役的海豹突击队队员，前密苏里州州长，现任美国参议员候选人。

#### 2. 约翰•弗雷德里克斯：百分之八十的共和党人知道总统大选被窃取

电台主持人约翰•弗雷德里克斯（John Fredericks）说，80%的共和党人 “知道总统大选被窃取了”。”

弗雷德里克斯在杰基尔岛（Jekyll Island）的美国共和党大会上进行了现场报道，他周五告诉【战斗室】，普通民众已经在全力以赴。他说：“在我们彻底弄清2020年11月3日总统大选真相之前，我们这边没有人要略过它。在我所接触的所有共和党人中有80%的人，哪怕是温和派，也都想要大选真相的答案。”弗雷德里克斯说：“无论在情感上还是在理性判断上，他们都明白此次总统选举是被窃取的，乔•拜登并没有得到那8000万张选票。”

弗雷德里克斯还介绍了富尔顿县选举腐败案的最新进展，这次是共和党人试图阻止MAGA候选人苏珊·欧普拉瑟斯（Susan Opraseuth）当选党主席。弗雷德里克斯说，大卫•谢弗（David Shafer）在选举连任佐治亚州共和党主席的道路上势头正劲，此前他赞同在该州针对选举结果进行全面的合规审计。

#### 3. 班农：普通民众和老百姓“正在改变历史的轨迹”

【战斗室】回顾了周四晚上在纽约市举行的令人难以置信的“新中国联邦成立一周年”的活动。

班农说中国的底层民众和老百姓，即遭受中共蹂躏的草根们，在“改变历史轨迹 ”上具有共同的意志。

他说：“这个专制政权视中国人民如草芥，这一点已经传递给了美国的普通听众，而且冲击巨大。”班农说【战斗室】观众的力量是 “不可阻挡的”。他说：“是观众们力量正在改变历史的轨迹，让它走向真理和正义。这才仅仅是开始！福奇即将成为一只替罪羊，但这还远远不够，我们还要扒得更深。”

在此次“周四自由之旅”的周年庆活动中，迈克•林德尔（Mike Lindell）、迈克尔•弗林(Michael Flynn)、劳伦斯•塞林博士(Dr. Lawrence Sellin)、闫丽梦博士（Dr.Li-Meng Yan）、娜塔莉•温特斯（Natalie Winters）、彼得•纳瓦罗（Peter Navarro）、鲁迪•朱利安尼（Rudy Giuliani）、杰克•波索比克（Jack Posobiec）、比尔•格茨（Bill Gertz）等人都先后发表了演讲。

卡萨姆说：“今天的事情意味深长。”

#### 4. 班农：弗朗西斯•柯林斯辩称武汉实验室不是中共的工具，他应该立刻被解雇

班农说，弗朗西斯•柯林斯（Francis Collins）应该立即被解雇，因为他还在为中国共产党当“托”。

这位美国国立卫生研究院院长在接受休•休伊特（Hugh Hewitt）节目采访时说，武汉病毒研究所正在被 “妖魔化 ”，说它并不能“简单地被当作中国共产党的工具。”

“这是一个彻头彻尾的谎言，他应该立即被终止职务。”班农周五在【战斗室】节目中说。“武汉病毒研究所百分之百是由解放军控制的。”

柯林斯在为该实验室辩护，称它拥有一些 “世界上最好的科学家。” 不要忘了正是柯林斯所领导的机构资助了那个实验室的将病毒武器化的功能增强性研究。

拉希姆•卡萨姆（Raheem Kassam）说：“1939年的时候，柏林的那些科学家也很出色，但这种出色与他们是否向邪恶政权妥协并成为其利用工具有什么关系么？”

卡萨姆还指出，武汉实验室在研究所内举办中共“精神 ”研讨会，还标榜要落实共产党的目标“不打折 ”。

“你柯林斯博士却跳出来，口口声声说那个实验室不是中共的工具。很好！仅凭这样的言论，你就该被炒鱿鱼了。”卡萨姆说。
