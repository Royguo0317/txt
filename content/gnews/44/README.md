**西班牙2021年5月2日电/西喜社——**

根据美国国家疫苗不良事件报告系统中的数据，科罗拉多州的一名15岁男孩在注射有争议的辉瑞COVID-19疫苗后仅两天就死于心脏病，尽管此前没有过敏反应史。

该案例在数据库中以VAERS ID 1242573列出，表明该15岁男孩于2021年4月18日“接受了辉瑞/百安泰的疫苗接种”。他于2021年4月19日开始对该疫苗产生不良反应，在接种疫苗2天后于2021／04/20去世。” VAERS数据库还显示，他没有其他疾病，没有既往疾病，没有已知的过敏，没有先天缺陷，也没有永久性残疾。他只是在接种有争议的辉瑞疫苗的两天后死于“心脏衰竭”。

该报告还指出，一名妇女在接种了与15岁死亡之前所用的相同的辉瑞疫苗后，经历了全身瘫痪和剧烈疼痛。田纳西州妇女布兰迪·帕克·麦克法登（Brandy Parker McFadden）在注射疫苗后因灼痛而惊醒，不久就意识到她无法动弹，并被送到医院。

尽管有与辉瑞疫苗有关的令人恐惧的死亡和致残报告，但该公司仍坚称其疫苗是安全的。事故发生后，辉瑞公司发给WKRN新闻2的声明部分写道：“迄今为止，全球已有超过2亿人接种了我们的疫苗。
