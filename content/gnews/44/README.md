414是何方神圣来的？不说不知道，一说吓一跳。 414是清华大学井冈山兵团四一四总部分裂自清华大学井冈山兵团，简称四派或四一四派。领导人沉如槐、孙怒涛等。清华大学井冈山兵团又是什么东西来的，？原来清华大学井冈山兵团也叫井冈山战斗队，成立于1966年12月19日一直到1968年全面领导清华大学的所有文革工作。是文革期间清华大学生建立的庞大的革命造反组织，其头目叫蒯大富。该组织联合北京的其他四所高校（北京大学、北京航空学院「后改名北京航空航天大学」、北京师范大学、和北京地质学院「今中国地质大学」），引领全国高校的文革浪潮。是文革时期造反派的主力骨干，是文革发动者的主要帮手。井冈山战斗队发展迅速，因观点不一致，成立四个月后就分裂为兵团总部（简称团派总部或者团派自认为是正统井冈山兵团）、414串连联会（也就是414派）。 1967年4月14日清华大学井冈山兵团四一四(革命)串联队成立，5月29日正式成立清华大学井冈山兵团四一四总部。于是与团派之间展开激烈的辩论战和冲突，并不断升级，对人力，财力损失非常巨大，影响到中央高层，这就是有名的“清华大学百日大武斗”。
