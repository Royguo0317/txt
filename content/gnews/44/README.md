登場人物のご紹介：

Burisma（ブリスマ）

Burisma Holdings Limitedは、エネルギー探査・生産会社グループの持株会社である。ウクライナのキエフにあるが、キプロスのリマソールに登録されている。Burisma Holdingsは2002年からウクライナのガス市場で事業を展開していて、ウクライナの最大の民間ガス会社の一つである。

資料：[https://en.wikipedia.org/wiki/Burisma](https://en.wikipedia.org/wiki/Burisma)

ハンター・バイデン

ロバート・ハンター・バイデン、別名ロバート・H・バイデン、元アメリカ副大統領ジョセフ・バイデンの息子である。2014年4月にウクライナのガス会社Burismaの取締役として採用された。麻薬乱用で米海軍を追放され、石油とガス分野の経験がないにも関わらず、父親の関係でBurismaの取締役になり、月に5万ドル以上の収入を得ることができた。

参考資料：[https://conservapedia.com/Hunter\_Biden](https://conservapedia.com/Hunter_Biden)

ヴァディム・ポザルスキー（VADYM POZHARSKYI）

ヴァディム・ポザルスキーはブリスマ（Brurisma）グループの取締役会の顧問として、グローバル・プロジェクトや国際組織、公共・政府関係の協力を担当していた。国際協力や欧州政治の問題でウクライナ閣僚の顧問を務めた。

2014年に公務員を退職し、国連環境計画（UNEP）にコンサルタントとして参加。また、ウクライナの国連工業開発機関（UNIDO）のプロジェクトでは国家コーディネーターを務めた。

2016年以降、ヴァディム・ポザルスキーは国際エネルギー安全保障フォーラム組織委員会の委員長を務めている。

参考資料：[http://energysecurityforum.org/esf-council/92-vadym-pozharskyi](http://energysecurityforum.org/esf-council/92-vadym-pozharskyi)

H.バイデンの元ビジネス・パートナーである。デボン・アーチャーはハンター・バイデンとクリストファー・ハインツのRosemount/Seneca Partners, LLCでのパートナーである。ハンター・バイデンはジョー・バイデン元米国副大統領の息子である。クリストファー・ハインツはジョン・ケリー元米国務長官の連れ子である。
