真正的强者是不停的战斗，不被恐惧所征服，不为邪恶和利益所收买！这是郭先生在他1月7日的直播中说出的！ 而现实生活中，被中共洗脑的很多人，会认为这样的精神是不识时务，甚至是傻瓜。这种人在现今的中国社会是不被认可，不被重用，或者说是被歧视被鄙视的。中共治下的人伦颠倒是非颠倒已经达到极致！

曾经和一位国内30几岁小有科技进步工程师，在一个设计院里供职，但苦于做设计工作，每天设计画图，一方面十分枯燥另一方面极为贫瘠，苦于奔命，又要养家糊口，不得已要走上层路线，美其名曰要让家人和自己得到更好的生活！殊不知，经过多轮考核各种走关系的结果，终于达到了他自己认为可以给家人和孩子更好生活的所谓领导岗位。上任之初要熟悉官场，各种阿谀奉承，走马灯似的各种酒场交锋，对于一个理工男已经是筋疲力尽。还要迎合上级做各种不符合实际的财务报告。半年下来苦不堪言，还是没有达到想达到的目标，轻则受到各种批评，重则责骂！

我以为，尝到了这样的苦头，知道自己不能驾驭这官场，他可以打退堂鼓，换个地方再重拾技术这门手艺，回归自然！没成想，这位总结了半年的经验，结果是和上级混的还不够近，马屁拍的还不够，所以才招致如此的多批评和指责，最后决定不但要和上级混入各种酒场，还要混入各种肮脏的交易中，还要一起进入色场。

后果大家可想而知！上级的错误他承担，拿到点小钱，这样周而复始，需要承担的就越来越大。这就是中共治下的老百姓的悲哀，出发点是想给家人更好的生活，结果是把自己弄成混迹于酒色官场，官没升多高，最后跌下来的时候还是要粉身碎骨。虽然不知道他现在变成什么样了，但是这样的卑微，能有什么好结果呢！最后一定是众叛亲离！醒醒吧！我的同胞！
