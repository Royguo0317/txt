## 梦断肯尼迪机场：刘彦平v.s.爆料革命

作者：纽约香草山写作组 兰兰、文徬

![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-13.jpg)

2017年5月26日，在纽约肯尼迪机场，中共国航的接机人员和候机乘客目睹了一幕精彩的谍战片：一群荷枪实弹的美国FBI特工把准备登机的几位VIP乘客按倒在地，收缴了手机与电脑，之后经过审讯和训斥，一行人员被驱逐出境。中间那位其貌不扬的大叔竟然是时任中共国安部纪委书记、中央纪委、国家监委驻国安部纪检监察组组长—-刘彦平。这个在中共国内颐指气使、不可一世的特务头子竟然被美国六亲不认的执法人员用枪顶着并按倒在地，肯尼迪机场成为了刘彦平大人的魂断地，其政治生命也在此刻终结。

![]()![](https://gnews.org/wp-content/uploads/2021/01/Screenshot-2021-01-18-174221.jpg)

这是中共国有史以来的第一次，有副部级、负责情报和执法的高官被美方取消签证。对文贵先生一向极不友好的《华尔街日报》，在2017年10月26日以“中国国安人员赴美劝返郭文贵， 各方交锋堪比谍战片”为题加以报道，对其中情节描写绘声绘色。2017年5月24日，在文贵先生的同意之下，刘彦平一行4人持过境签证，登门造访了其位于纽约第五大道荷兰雪梨酒店18楼的寓所。在超过一个小时的会谈中，刘彦平等中共国官员敦促文贵先生回国，承诺将会解冻其资产，并放过文贵先生的亲戚们一马。由于刘彦平一行4人持商务签证入境美国，依据美国法律并不能从事官方事务，但刘彦平以中共国家安全部纪委书记名义进行公务活动，早已引起美国联邦调查局FBI的关注。当天下午的下班晚高峰时间5点钟左右，FBI人员在纽约宾州火车站当场向刘彦平等4人查问。刘彦平等人先谎称是文化事务外交官员，后不得不承认是中共国安官员。FBI认为，刘彦平等人涉嫌跨境执法，从事非法活动，要求他们尽快离开美国，不能再与文贵先生交谈。刘彦平等人之后登上火车返回华盛顿。FBI以为他们将会在24小时内离开美国，但两天之后，即5月26日，刘彦平等人在当天傍晚飞返中国之前，再次来到文贵先生的寓所。

在华盛顿，为了是否逮捕刘彦平等人一事，白宫官员与美国国务院、美国国防部以及美国情报机关等有关官员召开了一次电话会议。美国国务院官员担心事件将导致在中国的美国官员受到报复，对其采取行动表示有所保留。会议期间有人提出另一建议：再次检查这一行中国官员，目的是使他们赶不上飞机，以便做进一步的扣留。但由于美国官员们始终没有达成共识，FBI调查人员只能扣下刘彦平等人的手机，然后才准许其登上飞机，回到中国。

中共派刘彦平持过境签证来美遣返文贵先生，这里面又有什么勾兑呢？实际上，就在刘延平造访前半小时，文贵先生收到了来自中共国内神秘战友的来信，告知其有三波队伍前来：第一波，刘彦平专程来纽约劝降；第二波，原中共国国安部副部长孙立军去华盛顿勾兑，还有吴征等人收买Elliott Broidy，刘特左买通美国司法部官员，Steven Wnn出面游说川普总统遣返文贵先生。而实际上，中共特务郑祺和几个福建人黑帮要下黑手杀害文贵先生。这三波人马，为了独占利益，互相拆台。持公务签证的孙力军在华盛顿被FBI禁止去纽约与文贵先生见面，孙力军于是多番与文贵先生通电话，请其出面交涉无果后，悻悻而去。刘彦平一波则顺利地与文贵先生会面，以为能够拔得头筹。

![]()![](https://gnews.org/wp-content/uploads/2021/01/Screenshot-2021-01-18-174311.jpg)

2017年5月24日傍晚，刘彦平在纽约曼哈顿文贵先生家中的谈判录音，堪称世界顶级谈判技巧的经典范例。中共特务头子刘彦平安排对与文贵先生的会谈进行录音，以做要挟他人和自证的依据，但他万万没有想到，自己的录音会被美国执法机构收缴并泄漏。刘彦平对文贵先生说，他临行前一天，曾面见中共政治局委员、中央政法委书记孟建柱。刘彦平在谈话中证实，文贵先生没有任何刑事犯罪，并威逼利诱要与其配合，来维护中共国家利益，更不要爆中共国领导人王岐山、孟建柱的料。并具体到6月16日，哪些不能爆；7月1日前哪些话不能说；8月全球媒体见面会前，要先将内容通报刘，爆料方向要调整等。刘彦平的话里藏话，称“在法律框架内有些东西可加可减”等，他亦透露了不少关于中南海里的秘密，比如令计划是如何窃取习（近平）办文件的，如何窃听党内高层的，如何玩弄空姐的，也提及了中共高官薄熙来、及其夫人谷开来的丑闻。

在刘彦平与文贵先生谈判之前，中共已答应作出两大让步：一是承诺将文贵先生的妻子、女儿带到美国来；二是将被关押的文贵先生的四哥和六哥释放。在2017年5月24日的具体谈判中，刘彦平亲口保证：文贵先生的妻儿可自由出入中共国；只要文贵先生愿意配合，还可以保释其侄女郭丽杰。并提出，文贵先生的问题，不一定通过刑事处罚来解决，可以进行民事协商。中共作为人类有史以来最大的土匪黑社会集团，视法律如儿戏，对内对外行事各种下三滥手段，无所不用其极，但遇见文贵先生的正道主义一再败下阵来，偷鸡不成反噬一把米。

刘彦平原本准备退休后携夫人到美国安享晚年。2017年5月发生在肯尼迪机场的一幕让他知道，这辈子他都不可能再踏入美国半步了，实际上他也明白，自己的政治生命也就此完结。2017年10月在中共公布的十九届中纪委委员名单中其被除名，2018年退居二线，担任第十三届全国政协委员的虚职。

而政治根基不如刘彦平的孙力军，政治生命和肉体生命则一起受到了审判。2020年4月19日，中共中央纪委国家监委网站宣布，孙力军涉嫌严重违纪违法，接受中央纪委国家监委纪律审查和监察调查。距离2017年5月，仅仅3年时间，孙力军的下场，不幸被文贵先生言中了。昨天还是座上宾，今天已成阶下囚。孙力军被抓的消息公布的第二天，4月20日，傅政华不再兼任中共司法部党组副书记，4月29日，不再担任司法部长。有报道称，傅政华已被软禁。紧接着，又有报道称，5月3日，孟建柱被抓捕。孟建柱的后台，是前中共政治局常委、国家副主席曾庆红。曾庆红的后台是前中共独裁者江泽民。中共恶党邪灵内部土崩瓦解，气数将尽！
