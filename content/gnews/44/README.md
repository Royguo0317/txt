计划生育时造成的人间惨剧想来大家也听说了不少，当事人及家庭都受到不同程度的伤害，我今天想说说一个阻止这些悲剧发生的故事。

有这么一个人，因为一次偶然在农村里游玩，发现河边钓鱼的钓上来一具婴儿的尸体。他非常震惊，问当地人到底怎么回事，当地人说这经常发生的，因为只能生一个孩子，很多家里面又重男轻女，所以怀孕的话就去做胎儿性别鉴定，发现是女娃娃就引产丢掉。这件事给他留下非常深刻的印象，觉得特别残忍，强烈想要去阻止这种悲剧。

他思考的结果是因为有非法做胎儿性别鉴定和非法引产机构的存在，才导致这么多未出生的女娃娃被非法引产。他想了办法去查询了相关的法律政策、文件等，后面发现国家有出台相关打击这些行为的文件，并且根据举报查实情况给予举报人不同的奖励，各市县也下达了任务指标要打击这些行为。他太兴奋了！觉得又能做好事，阻止这些悲剧的发生，同时还能拿到奖励，不用自己贴钱顺带维持生活，说干就干！

从哪里入手呢？小卡片，小传单。他开始四处收集街上散发的明写的暗示写可以做人流手术的名片。跟着地址去查找这些黑诊所黑机构。

可是哪有那么容易呢，如果只看到做非法引产器械或者材料，举报了最多只是罚点款，对这些机构并没有太大影响。要抓现场就必须要有孕妇，他根据这些关键要素去进行寻找，和确定就是要做非法引产的孕妇打好招呼，包括给予引产营养费等，来换取孕妇与他的配合。孕妇在里面做手术做到一半，就发信息给他，他提供信息给抓捕小组。期间也遇到非常狡猾的黑机构黑医生，抓捕的时候发现房间不对，根本就没在那个地方。还有做引产前要求孕妇把衣服和手机全脱了放起来，根本没办法及时传递信息。

这可难不倒他，他就是喜欢琢磨，原来就想去学刑侦，但因为视力问题也没成功。他喜欢捣鼓一些电子设备，还有很强的侦查能力和反侦察能力。列好计划雇人进行盯梢，24小时地进行，还自制了一些无线电来进行监听和传递信号，终于摸排了几个进行非法引产的地点，包括一些出租屋，小民房，旅馆酒店等。靠着他掌握的这些信息，相关部门现场多次进行成功抓捕，他也获得了文件上所说的奖励。

但他发现摸排工作越来越难完成下去，越早能抓到的都是一些没什么背景或者保护的黑诊所，有很多在抓捕过程中提前被人泄露信息医生跑掉，明明就是非常少的人只有高层知道的消息却还是跑掉，或者他开始被人用不同的方式警告，他知道这些保护伞的级别越来越高。

所以他开始有些担心，经常他今天一举报或者领取奖励，第二天就在别的地方听别人传了他的名字，哪里哪里人，多高多壮等身材样貌个人信息等。他个人信息被严重泄露，害怕被人打击报复，他想办法造了一张假身份证，因为领取奖励需要打在银行卡里，他又去购买了同一个名字的银行卡，用化名继续完成这个事情。

再后来，他就因为被查以假身份证的名义来举报骗取奖励金而被拘留。

不知道在派出所里，他能不能思考清楚到底是自己保护隐私的错？保护伞的错？黑机构的错？引产孕妇的错？还是未来得及呼吸这个世界新鲜空气那一个个女婴的错？
