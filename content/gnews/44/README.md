《印度快报》当地时间2月9日报道，印度总理莫迪和美国总统拜登在周一进行了第一次会谈，对一系列印美合作的问题进行了讨论。双方称他们将致力于基于规则的秩序的制定, 期待进一步巩固两国双边战略伙伴关系以及共同维护印太地区的和平与安全。

莫迪在推特上说，他向美国总统转达了他的良好祝愿，两国领导人讨论了地区问题以及共同的优先事项。他说，“拜登总统和我致力于建立基于规则的秩序。我们期待巩固我们的战略伙伴关系，以促进印度洋-太平洋地区和其他地区的和平与安全。”同时双方还同意进一步合作以应对气候变化。

莫迪指出，印美伙伴关系是基于共同的价值观，他说，两国有一个实质性和多方面的双边议程，有不断增长的经济合作和人民间充满活力的关联互动。

拜登自上世纪70年代担任参议员以来，一直被认为是加强印美关系的坚定支持者。在奥巴马(Barack Obama)政府时期，印美战略和防务关系出现了重大进展，拜登时任副总统在其中发挥了关键作用。

此次通话似乎并没有涉及到具体的合作细节，包括双方如何应对中共的挑战和威胁。印度正在努力发展自身经济以期可取代中共在亚洲的地位，近期的印中边界纠纷，印度试图与中共抗衡的疫苗外交及抑制中共全力发展5G技术等，都让印中矛盾成为不可忽视的议题。希望印美可以达成共识，在美国全力打造结盟政策下，合力抵抗中共。
