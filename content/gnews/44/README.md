**西班牙2021年1月19日电/西喜社——**美国加利福尼亚州的流行病学家敦促停止超过 300,000 剂Moderna疫苗，因为一些接种它的人，因可能的严重过敏反应需要接受医疗。

埃里克 S.潘博士建议疫苗供应商停止使用一个批次的Moderna疫苗，等待完成调查。她说，在一个疫苗接种点接种的人中，只有不到10人需要就医。

但她也表示，接种疫苗的严重反应极为罕见。

该病毒已经在加州夺走了超过3.3万人的生命。
