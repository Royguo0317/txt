## 【医疗前沿世界】CDC公告中共病毒可以经由气溶胶方式传播

作者：　纽约香草山医疗部　云豹

![]()![](https://gnews.org/wp-content/uploads/2021/03/WhatsApp_Image_2020-10-24_at_04.33.05-8.jpeg)



美国疾病管制中心（CDC）1於2021年5月7日公告，中共病毒SARS-CoV-2可以经由气溶胶颗粒的方式传播。传染性气溶胶未小於或等於5μm的颗粒，病毒依附在空气中的“气溶胶”上，当我们吸入这些比飞沫更轻的颗粒时如果病毒浓度够高，就可能被感染中共病毒。以前在评估飞沫传染时认为室内的社交距离只要有1.8公尺就是安全的，现在CDC证实气溶胶是传染中共病毒SARS-CoV-2的一种途径，那么即使社交距离超过1.8公尺，只要空气中的病毒浓度足够，就有可能被传染。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/17022000/Unknown-1-5.jpg)

《美国国家科学院院刊》（PNAS）在4月27日刊登麻省理工学院的研究论文2，显示配戴口罩加上良好的室内通风系统，才有办法避免病毒传染。另外一篇刊登在《国际感染病期刊》(IJID)上的论文3显示，从距离患者2～4.8米处收集的空气样本中可培养出SARS-CoV-2病毒。所以即使保持4.8公尺的社交距离，在密闭空间中只要病毒量足够，还是没有办法避免病毒传染。综合多篇论文的研究内容后显示，在密闭空间内，如果在病毒与空气混合完美的条件下，病毒能均匀地分布在各处，此时保持社交距离60英尺（18公尺）并没有比6英尺（1.8公尺）更安全。此点显示室内空气是否能完整地与外界空气交换流通来稀释室内的病毒才是重点。

CDC表示，携带SARS-CoV-2病毒的呼吸道液体的传染性途径有以下三种主要方式：

1. **吸入了**携带病毒的非常细小液滴或是气溶胶颗粒。在距离传播来源3～6英尺的距离内，这些携带病毒的极细小液滴和颗粒的浓度最大，传播的风险也最大。
2. 附著在呼出的小滴或气溶胶颗粒上的病毒被**沉积**到暴露的粘膜上（即“飞溅和喷溅”，例如咳嗽）。当距离越接近带著高浓度病毒传染源的呼出液滴和气溶胶颗粒时，传播的风险也越大。
3. **用手接触已**被病毒污染的粘膜，或者接触被病毒污染的无生命体表面而弄脏自己的手。


在有病毒传染源的室内，停留的时间只要超过15分钟，即使保持6英尺的社交距离，仍有可能被传染。在某些情况下，在感染者已经离开后不久穿过该空间的人也可能被感染。

根据已发布的报告，在下述情况下会增加SARS-CoV-2的感染风险：

1. 处在通风不良或空气换气不足的封闭空间中 ，此时呼出的呼吸液，尤其是非常细小的液滴和气溶胶颗粒，会在空气空间中积聚。

2. 如果感染者进行体育训练或提高声音（例如训练、大喊大叫、唱歌），则呼出呼吸液的呼气量会增加。

3. 长时间暴露於这些条件下超过15分钟。

预防被感染的方式建议如下：

1. 保持安全社交距离，身体的距离尽量离开越远越好。

2. 在社区使用合适的口罩（例如防护面罩、工作或手术用口罩）。已被感染者配戴口罩的效果更明确，可以避免将自己具传染性的呼吸分泌物传递给别人。

3. 室内要维持充分的通风并与室外充分换气，可以採用热交换机来更换室内的空气。避免停留在拥挤的室内空间。

这些方法可以减少病毒吸入的传播，也可以减少病毒在暴露的粘膜上的沉积。 注意自己手部的卫生清洁方式及良好的环境清洁，可以减少被传染的可能。
