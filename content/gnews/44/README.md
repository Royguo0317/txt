2月9日，挪威国防部长弗兰克-贝克-延森（Frank Bakke-Jensen）公开表明：”中国正日益成为挪威安全利益的挑战。”他指出，“近年来，中、俄情报安全部门的专业性和机动性都有所提高。挪威的国防、外交和安全政策、大北方[北极]、斯瓦尔巴群岛以及卫生、能源和高科技领域都是外国情报部门非常感兴趣的领域。”

就在几天前，荷兰军事情报和安全局局长 简-斯温楞斯（Jan Swillens）将军曾警告说：”我们看到越来越多的俄罗斯和中国的间谍活动出现在荷兰的主要部门和知识机构。军工部的调查表明，荷兰大学有数十名中国学生获得了相关军事技术的博士学位，结果发现这些人与中国军队或中国国防工业有直接关系。“

斯温楞斯解释说：”中国已经公开宣称要在2049年成为世界上最强大的国家，为了实现这一目标，中国正在用各种方法获得高科技技术。俄罗斯也是。”同时，研究也表明，中国、俄罗斯和伊朗等国家都有，针对包括荷兰在内的西方国家的进攻性网络计划。中国和俄罗斯的网络能力、知识和经验非常丰富，成功的几率很大，他们可以对某些地方进行数字化渗透。”

几天后，芬兰情报部门负责人安蒂-佩尔塔里（Antti Pelttari）在接受国家广播公司YLE采访时表达了上述担忧：”危险在于，一个专制国家，如俄罗斯或中国，可能会使交通、电信网络、电力分配、医院运营区块、支付系统等瘫痪。换句话说，几乎是一个社会的所有活动。”佩尔塔里称，”专制国家已经在试图接管芬兰的重要基础设施。”

法国、捷克、波兰、罗马尼亚、爱沙尼亚和拉脱维亚等国已经效仿英国，采取措施否决将中国元器件纳入其5G网络，瑞典也明确禁止华为和中兴参与。最大的疑点，也是绝对可能向中国倾斜的是德国的立场，德国刚刚通过了一项新的网络安全法，尽管布鲁塞尔发出了警告，但目前来看，德国可能允许华为成为其5G基础设施的一部分。
