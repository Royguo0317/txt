4月19日，美国宇航局NASA见证了21世纪的莱特兄弟时刻，其发射的“机智”号无人直升机在火星表面完成了39秒的处女飞行，这也是人类首次在另一个行星上完成了航空器的动力控制飞行。这一壮举无疑是人类从地球文明迈向星际文明的又一里程碑，让人热血沸腾，让我们对未来的星际旅行充满了无限遐想。

在美国等西方国家不遗余力推动人类文明进程的时候，反观中共，还在乐此不疲的窝里斗，利用商鞅五术治国，裹挟洗脑14亿百姓。据笔者体制内朋友透露，中共国的公职人员大部分时间都被迫花在毫无意义的会议上，其中至少一半跟党建有关。每个人定期或不定期要参加各类政治教育活动，还要像学生一样，上党课、练习题、考党史、写思想报告等。在各单位的年终综合绩效考核中，党建工作占比多达50%，专项业务工作占比却少得可怜，群众满意度却几乎不纳入考核范围。再看看中共国的办公大楼，上至人民大会堂、中央银行，下至村委会、信用社，哪一个政府和金融机构的办公楼不比老百姓的居所好上千万倍的。由此看来，为人民服务只是噱头，为党服务才是重心。

在恶党成立百年之际，墙国上下掀起一片学党史、学“习”之风，中共要求各级各单位组织开展主题展览、文艺汇演、知识竞赛及红色旅游等各色活动，对民众进行红色教育。大肆宣扬党的丰功伟绩，为自己歌功颂德，逼着老百姓“学党史、听党话、感党恩、跟党走”。为了控制舆论并愚弄百姓，除职业五毛水军外，中共还密文要求各级各单位公职人员充当网络评论员，按时按量的对人民网、新华网等中共官媒发表的文章进行正面点评或舆论引导。
