2、文贵现在接受美国之音采访时，火鸡龚一直在问一些刁难性问题，文贵先生所爆料的内容都是基于事实、真实的经历和情报，也有完整的逻辑性，爆料内容最后都被一一验证。“419断播”事件是文贵先生身先士卒，他将灭共的战场拉到海外。

4、“419断播”事件之前还没有一位有如此影响力、并且深知中共邪恶本性的中国人站出来挑战共产党，这次断播事件也是美国之音几十年以来首次出现，随后美国之音数位官员被政府解雇，这也让美国人彻底明白即使是崇尚自由的美国也已经失去言论自由，只有站出来消灭中共才能拥有真正的自由。

5、“419断播”事件后，美国正式意识到中共在美国的影响力，这也促使美国重新审视中共对美国的威胁，重新审核美国获得有关中共在美国犯罪的各个情报。文贵先生引领的爆料革命从2017年开始后，不断受到中共的打压、威胁和伤害，爆料革命新中国联邦的影响力不但没有减小，反而越来越强大，现在能直接影响美国最高决策层。

二、美将出台新规限制中共国450万家企业

《日经亚洲评论》周五（4月16日）报导[\[1\]](https://www.epochtimes.com/b5/21/4/17/n12885983.htm)，美国政府最早将在5月要求公司在使用来自中国或其他被视为敌对国家的信息技术设备和服务时必须持有许可，此举可能会影响多达450万家企业。这些规则不仅仅影响与政府合作的企业，也同样影响在美经营的私营企业。据商务部估计，在美国约600万家公司中，有四分之三的公司使用外国技术，包括海外企业的美国分公司。
![]()![](https://gnews.org/wp-content/uploads/2021/04/美国商务部大厦.jpg)[美国商务部大楼](https://i.epochtimes.com/assets/uploads/2020/09/Department_of_Commerce_Samira-Bouaou-600x400.jpg)
路德社评论：

1、站在今天回看2017年以前的中美关系，没有人会想到如今的中共已经成为美国最大的敌对国，成为全世界最大的威胁，这一切都是爆料革命经过数年努力才让美国彻底看清了中共的邪恶本性。

2、这项新规是近几个月以来对中共国经济影响最大的一条规定，美国的所有电子设备、通讯设备、遥控设备等等只要使用来自中共国电子元器件都必须上报美国政府批准，准备审批材料、审批时间、以及是否获批，这些都是不确定因素，美国企业为规避风险自然就会用其他国家的产品替代，这对中共国电子元器件企业来说是灭顶之灾。

3、中共墙内媒体报道：该规则不仅影响与政府合作的公司，还影响在美国运营的所有私营公司。美国商务部估计，在美国大约600万家公司中，有四分之三使用外国技术，其中包括海外企业在美国的分支机构。该规则适用于广泛的技术，包括关键基础设施和电信网络中使用的硬件和软件，以及人工智能和量子计算技术。该列表还涵盖了处理个人信息的服务，以及监视设备，例如可联网的监控摄像头，传感器和无人机。根据定义，在内部网络中使用中文路由器，工厂中使用中文摄像头或中文云服务来处理客户数据的企业可能会受到审查[\[2\]](https://www.163.com/dy/article/G7V820A30511RIVP.html)。因为中共作恶导致国内产业链受到巨大冲击，如果这些企业再不行动，那未来就只能被荒废。

4、路德社早在2018年就说中美将全面脱钩，“120大选”之后就爆料两党将联合灭共，3月1日保守派大会后，全美一定全面加速灭共。中共本以为去年签订“115贸易协议”后迎来了巨大的发展机会，以为让拜登政府上台就赢得了终极之战的胜利，中共每次都越赌越大，最后押上了身家性命，其实这些都是美国给中共挖下的大坑。

5、这些规则是根据前总统唐纳德·川普于2019年签署的第13873号行政令（《确保信息和通信技术与服务（ICTS）供应链》）执行的。拜登政府曾表示，尽管美国企业反对，但针对被视为对美国构成威胁的中共国科技公司，他将继续执行川普时代的规则[1]。

6、拜登政府依然延续川普政府政策，使用中共国技术不上报的企业将面临处罚。拜登政府上台后真敢干，由于川普总统是商人并非政客，在政治手段上没有拜登政府狠，况且美国大多数电子行业都是拜登政府的支持者。如果中共国企业家早听爆料革命的意见，提前将企业和资本转移到国外，就能避免巨大的损失。

7、这项政策是从美国国家战略安全的角度颁布的，美国现在的敌对国只有中共国、俄罗斯、朝鲜、古巴、伊朗和委内瑞拉，里面只有中共国的IT产业最发达，中共国为全世界供应各种电子产品的元器件。中共国利用从海外偷来的技术在本国生产，又低价卖给西方国家来绑架他们的摄像头、云服务等软硬件系统，以此来窃取他们的商业信息和数据信息。美国将中共国定为敌对国后，接下来颁布的所有政策都会配合跟进，这也是向中共内部施压，另一方面，美国牵头建立印太联盟后，他们正在将中共国的产业转移到印太联盟国家。

8、这项禁令一颁布后，苹果手机不可能再让中共国企业代工生产，欧盟和日本政府肯定会继续跟进，这对中共国高端制造企业是灭顶之灾，中共国内将有无数人因此失业。
