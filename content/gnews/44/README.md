美东时间2020年12月21日，美国国土安全部部长沃尔夫（Chad Wolf）发表重要演讲，主要内容如下：

今天，我们国家的和平繁荣的威胁主要来自于中国，需要明确的是，我说的中国威胁是指北京的领导层而不是中国人民，中国人民正受到他们政府的打压和极权统治，中共将共产主义作为指导思想使其与前苏联的意识形态一起在中国国内盛行。中国把自由开放的经济和民主世界视为政治和意识形态的威胁，他们根据自己的理念重塑世界秩序，集权和专制统治，构建一个中共决定谁输谁赢的世界，所有的决策和目的都是维护中共政权而不是帮助中国人民，这种意识的思维模式驱使着中共的行为，他们正在我们的国土上威胁着我们的经济和人民以及我们的生活。更严重的是，他们用卑劣的手段颠覆西方国家的稳定。中共国不仅不公平竞争还在破坏国际秩序，其不在传统战场与我们对抗，而是开展秘密的虚假信息战，从内部瓦解我们，其后果足以引起每一个美国人的担忧。中共的极权思想与中国恶意攻击是有关联的，他们把这个理念作为前提，中共相信为达目的不择手段，更不在乎谁受到伤害，不管是全世界正面临着新冠病毒威胁的亿万民众，还是被打压的无法自由生育、不能信仰的、不能自由发表言论的以及被政府非人道破坏的中国人民。而这种意识形态在实际行动中也决定着中共国对美国的战略。用直白的话讲，这是一个极权国家，中共垄断了的政治、科技、经济、资源、工业，无视人权与尊严以及被政府的迫害的恐惧的人民，这些结合起来让中国政权成为美国和全世界的严重威胁，这些也让中共有能力破坏包括自由、平等、诚实、透明的自由民主的价值观。中共政府已经发起了对我国国土和人民的政治、经济和文化的全面进攻，是时候认清这些以及中共的本质并还击他们的时候了。

这是历史上美国官方第一次把中共和中国人民分开，也是美国官方第一次表态威胁来自中共。相信美国会有更进一步的表态和行动，用郭先生的话来总结：“滚天雷真的来了”。
