## 中共信息战打手推特即将限制病毒疫苗信息

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/17162506/capture-120.jpg)图片来源：[techcrunch.com](https://techcrunch.com/2020/03/05/twitter-bans-hate-speech-around-age-disability-and-in-the-wake-of-the-coronavirus-outbreak-disease/)
据《国会山报》（The Hill)12月16日报道，推特于本周表示，随着全美为一线人员疫苗接种工作的开始，接下来的几周，推特将对有关疫苗的虚假信息打上标签并删除。

推特称，平台将扩大目前针对病毒错误信息帖子的政策，即将包含有关疫苗是”有害、虚假或误导描述“的推文包括在内。如果推文提及疫苗是用来“故意制造伤害”， 或散布接种疫苗产生不良影响的说法，就会贴上标签，推特同时会要求用户删除帖子。此更新的政策将在下周一开始执行，并在接下来的几周内全面推广。

人们注意到，自中共病毒爆发以来，大量关于病毒特性、起源及其传播方式的推文被推特公司限制、贴标签或要求用户删除，直至封杀用户，甚至关于预防措施药物功效的推文也被贴上虚假或误导性信息的标签，比如有关硫酸羟氯喹的推文。

推特宣称他们将与世界各地的公共卫生部门密切协作来保证这项新的言论管控政策得以持续和透明的执行。另外，脸书在本月早些时候也宣称要对疫苗不实信息进行删除行动。

另悉，辉瑞公司（Pfizer）和BioNTech公司生产的第一批疫苗于上周五获得美国食品药品监督管理局（FDA）的批准，将会在本周向全美的医护人员进行发放，而来自Moderna公司的第二种疫苗可能会在本周获得紧急授权。

简评：自中共病毒爆发以来，推特公司对有关揭露病毒真相信息的封杀，使其充当中共言论自由打手的角色暴露无遗。现在，人们普遍对疫苗功效和副作用抱持怀疑态度，并对辉瑞公司（Pfizer）的资金来源和中共背景产生疑虑。而推特此时宣布将对质疑疫苗的推文实施严格审查，这清楚的表明了推特公司做为利益集团马前卒的身份。随着疫苗的推广，其副作用如何，甚至更多不可告人的秘密都将慢慢浮出水面，我们拭目以待推特公司将如何应对必将汹涌而至的挑战。希望230法案能尽快废除，从而结束推特和脸书这些社交媒体对自由言论和真相的非法审查和限制。
