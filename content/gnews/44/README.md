然而，在所有被中共“热心”送了疫苗的国家里，疫苗被送得次数最多的国家就是墨西哥。2021年2月19日，中共国的第一批疫苗到达墨西哥，据墨西哥政府称，这批疫苗将优先提供给[教师](https://www.chinanews.com/gj/2021/02-20/9415381.shtml)等人群接种。2021年2月27日，第二批科兴疫苗抵达墨西哥城，据新华社墨西哥城[报道](http://www.xinhuanet.com/politics/2021-02/28/c_1127148484.htm)，墨西哥从去年，即2020年12月24日，就已经启动了疫苗接种，优先为医护人员、老年人注射。2021年3月13日上午，[第三批](http://www.chinanews.com/gn/2021/03-14/9431637.shtml)科兴疫苗运抵墨西哥城。
