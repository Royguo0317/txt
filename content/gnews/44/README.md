新浪网12月7日转载环球时报报道，近期日本欧洲媒体频繁报道，英国“伊丽莎白女王”号航母编队明年将在西太平洋地区长期驻扎；法国将在2021年5月同日美在东海联合军演。军事灭共已初见苗头。

日本媒体报道，英国“伊丽莎白女王”航母打击群最快明年年初将部署亚太，辐射范围包含冲绳等在内的西南诸岛，后续将与日本自卫队、美军开展联合军演。

英国“放风”的同时，法国也在积极参与行动，将在21年5月参与美日的陆海联合作战，出动含舰艇、陆军部队、飞机等在内的众多大型军备，规模和战力上完全碾压中共“假演习”，其海军参谋长更是明确指出此举针对中共。

从郭文贵先生和路德时评把北约将采取的军事行动（含日、澳、印等美国在亚洲的同盟国）披露出来后，中共内部乱作一团，习近平不得不“突击”搞政治教育继续对军队内部高压维稳，侧面印证爆料革命情报的可靠性。

而英国航母的进驻，以及法国参与美日军演等消息的流出，也为接下来更多大规模军事行动做铺垫，北约的灭共军事准备已经正式开启。

历年来的军费开支虚报、冒领、克扣拿要造成中共军队的武器装备换装缓慢、老旧失准，而实弹射击稀少和形式化更加剧了中共军力的“假、大、空”、虚有其表。

无论是多个航母打击群的围猎，北约以及美国同盟国的大型军备调度和参与，还是美国“太空武器”的定点打击，中共只能是“待宰羔羊”，顷刻间土崩瓦解。
