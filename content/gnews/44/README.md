**西班牙2021年1月21日电/西喜社——**

根据医院官网介绍：上海复旦附属肿瘤医院设有徐汇院区、浦东院区两个院区，截至2019年底，医院实际开放床位2000余张，设有26个临床医技科室。医院职员工2,561人。

2019年医院门诊量157.29万人次，住院10.07万人次，手术5.14万人次；这样的一家对百姓至关重要的医院的徐汇院区，因为疫情封闭了。

因为在今天中共通报里说上海有3例疑似的案例，其中一例就在上海复旦附属肿瘤医院的徐汇院区外包后勤保障人员。现在整件事情已经持续发酵。
