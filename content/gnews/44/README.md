战争所要具备的三个要素：信仰、经济、政见（意识形态）这和我们中华文化的 （三观）极其相似异曲同工。如果三者同时出现，那么战争则无可避免，分歧越大、战争越是临近，分歧越大、战争的烈度也会相应的提高。远观近看、今天的现实世界这三种因素都已经同时出现，战争即将到来！

优胜劣汰是自然法则。既然已无法避免发生，我们所要做的只有选择立场。战争爆发、狼烟四起灾难降临时、次生灾害接踵而至！所以的人都要面对一个现实、就是如何生存，健康的生存。作为CCP政府部门在应对灾难，如何来救助贫民百姓的问题就摆在每一个人面前。

先让我们看看四川地震时的救助情况吧，交通中断，电力、信息中断，在最开始最关键的72小时之内，政府没有拿出一个有效的救助方案。军队行动迟缓，甚至还没有民间自发组织救援队伍的速度快。当看到这时，我想说一句话：“如果那里是战场，敌人正在残杀我们的百姓同胞。你们政府机关和你们手里掌握的军队是这样子对待的吗？”但墙内环境里却没有一个人（敢）站出来对政府问一句：为什么？这是为什么？！我们所得到的只是一个概念回答：鞭长莫及，爱莫能助！？

再来看看年初武汉疫情爆发时，中共政府对平民百姓的救助，所有官方发布的信息掩盖事实，混乱不堪，含糊其辞，朝令夕改，让民众无所适从，这本应该就是他们职能职权范围内应该做到的事情，但事实却让我们看到了无所事事，无能为力、无可奈何！这些个酒囊饭袋，愚蠢至极！更有甚者，利用自己手中的权力，去和民众争抢仅有的一点医疗资源。这时候没有人（敢）站出来说一句：为什么？（你）凭什么？在这里请允许我抱一句粗口“一群蠢材，一群混蛋！”。这些都是刚刚发生过的事，当灾害发生，贫民百姓得不到及时有效的救助，而逝去活生生的生命，这些都是我们的同胞亲人！难道都忘了吗？

以上这些还都是发生在和平时期，如果战争爆发呢？再加上接踵而至的次生灾害呢？就这样的政府职能部门，还能有多少能力来救助我们的平民百姓亲人？战争对平民造成的灾难远远大与和平时期，甚至是百倍，千倍！聚焦到个人看，当灾难来临时，要抛开一切、什么钱财、资产都不重要了，生存下去最重要！退一万步讲，即使你有什么信仰、什么主义政治观点，但首要的是先要活下去！失去了生命一切都是0 ！尽早做准备来应对灾难才是最佳选择。人的生命只有一次，一旦失去即使万能的神也不可能再造复制一个你回来！

望看到这个信息的人，珍惜生命的人，深爱亲人的人们，尽最大的努力去唤醒沉睡的亲人，及早做准备。在此时此刻祈祷天下苍生远离战场远离战争！愿万佛众神保佑天下苍生、心怀善良正义的人们平安！尽早清除邪魔恶妖，还世界一片光明美好的净土！
