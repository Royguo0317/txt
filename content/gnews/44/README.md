[美日2+2外交与国防部长会谈](https://china.kyodonews.net/news/2021/03/633ad1c96b19.html?phrase=%E7%BE%8E%E5%9B%BD%E5%8A%A1%E5%8D%BF%E7%A7%B0%E5%B0%86%E5%90%91&amp;words=%E7%BE%8E,%E5%9B%BD%E5%8A%A1%E5%8D%BF,%E7%A7%B0,%E5%B0%86,%E5%90%91)于3月16日在日本举行。日本首相菅义伟在官邸会见正在访日的美国国务卿布林肯和国防部长奥斯汀。菅义伟在美日安全保障协议委员会上称，双方在面向强化日美同盟的威慑力和应对力进行了充分的讨论。

会后双方发表共同声明，点名批评中共国不守国际秩序，与国际秩序不一致，使美日同盟与国际社会面临挑战。

菅义伟表示：“希望近期与拜登总统会谈，确认日美同盟的纽带。”

3月17日美国国务卿布林肯在记者会上表示，将在本周与杨洁篪、王毅的会谈中，向中共传达日美对中国海警船驶入钓鱼岛周边海域，违反现存国际秩序等举动的也表达严重疑虑。他重申，《日美安全保障条约》第五条也适用于钓鱼岛，坚决反对有损于日本对钓鱼岛施政的任何行动。另外对于中共国加强军事施压的台湾海峡相关局势，他表示日本在台海局势有共同的利害关系”。

日美两国政府就中共国建造人工岛推进南中国海的军事基地化，对中共国的不法海洋权益相关主张和活动表示了反对。日美双方还对香港局势和新疆人权问题进行了讨论。

美国国务卿和国防部长首先到日本访问，显示出东亚和南海台海的重要程度。钓鱼岛朝鲜核问题和台湾海峡等热点，背后都有中共的鬼影。近期中共国利用超限病毒武器传播全世界，其单方面改变世界格局的意图的风险日益加剧。西方各国虽然深受蓝金黄的肆虐，但中共称霸全世界的野心已经使西方不得不重视起来，开始建立联盟，近期的美日印澳联手对抗中共国的意图明显，相信四国联盟建立后，西方各国会陆续加入。
