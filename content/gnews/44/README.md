## 欧洲五国吁以色列放弃在约旦河西岸建定居点

5月6日，法国、英国、德国、西班牙和意大利呼吁以色列放弃在约旦河西岸建造定居点。这几个国家的外交部官员周四在一份声明中说。

公报说，“我们敦促以色列政府改变其决定，在被占领的西岸哈尔霍马定居点开始建造540座住宅建筑物，并结束在被占领的巴勒斯坦领土上扩大定居点的政策。”两国在一份联合声明中说，“根据国际法，定居点是非法的，对和平解决巴以冲突的前景构成了威胁。”

根据这五个国家外交部的说法，“如果扩大霍马山定居点的决定得到执行，将破坏以耶路撒冷为以色列首都和巴勒斯坦国的可行的前景。”

公报中还说，“我们呼吁各方不要采取任何单边举措，恢复可信的对话，以便走向两国解决方案并结束冲突。”

以色列在约旦河西岸的建设被认为是解决长达数十年的巴以对峙的一个主要障碍。2016年12月，联合国安理会通过第2334号决议，要求以色列立即停止在巴勒斯坦被占领土上的定居点活动。以色列随后宣布，它不会遵守联合国安理会这一文件的规定。
