一、新中国联邦新闻

1. “路德时评”本日摘要（[早间](https://www.youtube.com/watch?v=dQxVBRgQzAA)、[晚间](https://www.youtube.com/watch?v=XCwxGnzvzd8)）:

Ⅰ. 美国务院正式认定中共政府犯种族灭绝与反人类罪，2002年以来有8位国家元首被判为反人类罪，其中包括萨达姆和卡扎菲，无一逃脱国际审判。国务院发布的这份文件将中共放在人类的对立面，并明确习近平作为中共领导人犯下的种种罪行。彭佩奥国务卿颁布此文件是消灭中共非常重要的一步，将为中美关系重新标定风向。

Ⅱ. 路德社在2020年1月19日早间节目中爆料中共病毒真相，回首这一年，现实远比想象还严重，现在全球将近一亿人确诊感染中共病毒。在这一年的时间里，中共销毁了病毒相关的资料，并持续不断的掩盖真相。闫博士和爆料革命还在继续推进普通人对中共病毒来自实验室的认知，通过揭露中共的邪恶和假、骗、偷让人们知道病毒来自中共。同时闫博士也在研究病毒的特点和中共病毒与舟山蝙蝠病毒10%的差异究竟会带来哪些影响，寻找中共病毒的治疗模式，提升防疫宣传以及研究如何评估疫苗有效性和安全性。

2. 川普政府1月19日宣布，中共在新疆对维吾尔人犯下了种族灭绝和反人类罪。美国国务卿蓬佩奥在推特上说：“我决定，中华人民共和国在新疆针对维吾尔穆斯林和其他宗教和少数族裔犯下了种族灭绝和反人类罪。这些行为是对中国人民和世界各地所有文明国家的侮辱。中华人民共和国和中共必须被问责。”

3. 1月19日，美国新政府的国务卿人选布林肯表示，川普总统对中共采取强硬的态度是正确的，并支持蓬佩奥国务卿认定中共在新疆犯下“种族灭绝”的决定。布林肯的说法预示了即将上任的新政府对中共国的态度可能将维持现有的应对基调。

4. 1月19日，美国参议院军事委员会和国土安全委员，分别举行了对当选总统拜登的国防部长和国土安全部长提名人的确认听证会。美国防部长提名人奥斯汀表示，中共国是美国头号威胁。

5. 1月19日，德国工商大会发布的2020年度评估报告显示，中共国已经取代法国，成为德国的第二大出口市场。默克尔此前被曝光以德国电信进入中国市场为条件，促成中欧投资协议。1月19日又有消息称，默克尔很可能帮助中共刚刚与欧盟达成了一项新协议，如果欧盟议会通过，将导致大量来自中共的现金流入欧洲共同市场。

6. 据媒体报道，美中贸易协议一周年，中方采购承诺未达标。按照协议要求，中共同意在2020年底前购买约1590亿美元的美国商品。彼得森国际经济研究所进行的分析显示，截至2020年11月，中共实际自美采购了约820亿美元的商品，只完成了上述目标的52%左右。

7. 世界上最受尊敬的对冲基金之一艾略特（Elliott）将关闭其香港办事处，成为首批在香港关闭业务的大型金融机构之一。港府不断的打压所谓颠覆国家政权或外国 “干涉 “行为，这引发了市场对香港作为全球金融中心的未来的担忧。

8. 日媒1月19日报道称，中共和巴基斯坦已陷入他们在“一带一路”倡议上最严重的分歧，导致中巴经济走廊（CPEC）年度双边峰会推迟。最初造成推迟的原因是中共肺炎疫情，但后来，中巴两国在1号干线铁路项目和经济特区问题上的分歧成为主要原因。

二、世界新闻

1. 美国国会参议院共和党领袖麦康奈尔1月19日声称，川普总统“挑动”数以千计的支持者于1月6日冲击国会，企图但未能阻止国会认证民主党人乔·拜登在11月的选举中取胜。麦康奈尔的讲话令人震惊，其言论可能会被视为共和党核心小组的明确信号，几个小时后有消息称，川普总统和“熟悉的人”正在讨论组建新政党。

2. 涉嫌掩盖2020中共干涉大选实情的吉娜·哈斯佩尔，于1月19日辞去了中情局局长职务。哈斯佩尔还亲自阻止过通俄门文件的解密。

3. 美国驻柏林大使馆1月18日正式照会德国联邦政府，宣布将从19日起对俄罗斯籍”命运女神号”铺管船执行制裁措施。美国驻柏林大使馆发表声明称，美方将继续”坚定地采取一切必要且适当的措施，以阻止对美国及其欧洲盟友的重大国家利益构成威胁的’北溪2号’工程。

4. 世界卫生组织第148届执行委员会1月18日起至1月26日以全视讯会议方式举行，美国在1月18日第一天的会议上呼吁中共交出冠状病毒资讯，在1月19日会上则进一步公开肯定台湾防疫经验及贡献，强调台湾应成为观察员参与世卫组织。

5. 川普总统在离开白宫的最后几个小时作出决定特赦班农先生，他表示这是他任内最后一项作为之一。另据消息称，川普总统在最后一刻放弃了赦免自己和家人的想法。

6. 美国网络安全企业 Malwarebytes 表示，该公司于去年遭受了针对 SolarWinds 的同一黑客组织（UNC2452 / Dark Halo）的入侵。这是第四家遭受SolarWinds黑客入侵的安全公司。川普总统于1月19日发布行政令，旨在限制外国干预势力使用美国网络基础设施即服务解决方案。

7. 川普总统于1月19日晚间不顾FBI的反对，正式解密了一大批通俄门“交火飓风”文件，预计相关文件将于1月20日公布。
