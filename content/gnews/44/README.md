## 中共的反制裁终于碰到铁板——欧洲议会临时取消审议中欧投资协定
` JPHIMALAYA` [轉載自GNews](https://gnews.org/zh-hans/1007887/)

【日本东京方舟农场】作者：Maarago 素材整理：武汉老温 校对：文小律

据美国之音2021年3月23日 15:04发布的[不满中国反制裁 欧洲议会临时取消审议中欧全面投资协定](https://www.voachinese.com/a/impact-china-eu-sanctions-20210323/5824981.html?utm_source=twitter&amp;utm_medium=social&amp;utm_campaign=dlvr.it)：

［星期一（2021年3月22日），欧盟与美国、英国和加拿大协调行动，罕见地在同一天宣布对涉嫌在中国新疆践踏穆斯林维吾尔少数民族人权的中国官员实施制裁。

之后，中国当局迅速做出回应，宣布对欧洲10名个人和4个实体实行制裁，包括多名欧洲议会的议员、欧洲一些知名智库和学者。他们都曾就中国的人权问题积极发声。］

［来自法国的欧洲议会议员吕克兹曼（Raphael Glucksmann）在中共的制裁名单上榜上有名。他在推特上说，“我在中国的制裁名单上，不能进入中国（还包括我的家人！），不能与中国官员或公司接触，这一切都是因为我为维吾尔族人发声、为人权发声。让我们明确指出：这些制裁是我的荣誉勋章。抗争将继续下去！”

与此同时，中共对欧盟的制裁可能影响到欧盟与中国的经贸关系和欧洲议会对一项关键的投资协定的核准。］

［欧洲议会国际贸易委员会副主席居拉(Winkler Gyula)星期一在一份电邮中说，“鉴于欧盟与中国关系今天的最新发展，尤其是（中方）令人无法接受的制裁”，欧洲议会决定取消原定星期二就签署中欧全面投资协定CAI举行的一个审议会。他还在推文中说，“欧盟是讲价值观和原则的，无论是在欧盟内部，还是在全球范围内。”］

蓬佩奥国务卿在其离任前一天的119给中共送上的种族大屠杀的告别礼余波荡漾至今，从中共一开始要全力摘掉种族大屠杀的帽子而未果，再到美国、英国、加拿大及欧盟成员国等国家分别对中共种族大屠杀定性的确认，中共与所有这些确认中共种族大屠杀的国家间一直采取口炮和谩骂的方式应对，再到这些欧盟成员国家对负有种族大屠杀责任的中共内部人士制裁，再到中共采取对等所谓制裁回应，现在欧盟国家被制裁者纷纷对被中共制裁感到荣耀，并且终于关于中共种族大屠杀的国际角力进入了实质阶段——这一回欧洲各国要动钱袋子了，要对准与中共国的投协定了。

一直以来中共绑架下的十四亿奴隶和其所构成的市场就是中共的经济核武器，中共以市场准入为武器顺共者昌逆共者以经济核武器狂轰乱炸，在软硬兼施之下攻城掠地成功的让西方的政客和金融资本家跪在中共脚下，但是这一次看来中共终于踢到铁板上了。

关于中共与欧盟之间关于种族大屠杀的制裁与反制裁大戏，中共的看门狗胡锡进也在[2021年3月23日14:33](https://weibo.com/1989660417/K7waahqAj?from=page_1035051989660417_profile&amp;wvr=6&amp;mod=weibotime)发布的新浪微博中进行了煞有介事的评论——

［美英加三国与欧盟第一次联动制裁中国，内容是象征性的，就是制裁新疆的4个人和建设兵团的公安局，欧盟还有意没有把新疆主要负责人纳入其中，但他们很享受“西方团结起来制裁中国”的这一声势。大家都看到了，中国迅速反制裁欧盟的10个人和4个机构，对美英加肯定也会追加制裁。

请注意，这一次的相互制裁都没有动经贸，因此还是属于打嘴仗，摆姿势。事已至此，老胡主张既来之则安之，就你来我去地过招呗。最重要的是，这样的“骂战”和姿势斗狠过程会很漫长，而且不排除会逐渐加码，中国社会一定要始终高度清醒，处变不惊，阵脚不乱。］

［中国与西方关系的实质是生意，那才是实实在在的利益。欧洲会很心疼，咱们也会心疼，现在只有美国咬牙从奶酪上抠下了一点碎末，欧洲还没舍得抠，我认为中国也不要轻易把意识形态冲突往经贸方向引。咱们就彼此闹着，继续发展着，中国当下发展的本事可是比西方强一大块，这样搞着搞着中西骂战的大环境就逐渐变了。中国越发展越好，新疆也越来越好，他们骂着骂着就自己提不起气来了，这场前所未有的中西舆论战最终要以这样的方式落下帷幕才对。］

按照胡叼盘的预测在这场围绕对中共新疆种族大屠杀的角斗中西方的政客们骂着骂着就提不起气来了，再顺着胡叼盘的分析延伸一下就是接下来就像当年的六四一样，人也杀了、骂也骂了，中共这个流氓还可以继续人五人六的与西方的政客们勾兑了，可是这一次，看来真不行了，接下来呢？如果欧盟断绝与中共的经济合作、如果美国断绝与中共的经济合作、如果西方断绝与中共的经济合作，那么中共连跪的机会都没有了。也许在中共灭亡前还会有勾兑，也许西方的政客们还会有反复，但是灭共大势已经不可逆转，当西方看透中共的本质，当中共的“经济核大棒”最终被证明是假擀面杖子的时候，中共的丧钟已经敲响。
