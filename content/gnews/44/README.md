美国总统唐纳德·川普周三呼吁在国会联席会议期间聚集在美国国会大厦附近的抗议者回家，同时声称选举是被偷走的。

川普在视频讲话中说：”你们现在必须回家。我们必须要有和平。我们必须要有法律和秩序，要有伟大的人在执法。”

川普断言选举是 “欺诈 “的，但主张国会大厦的抗议者 “不能被这些人玩弄于股掌之中……我们必须有和平”。

“我们爱你们，你们很特别，”他说，”我知道你们的感受。”

川普随后呼吁他的支持者回家，“和平回家”。

**[pic.twitter.com/Pm2PKV0Fp3](https://pic.twitter.com/Pm2PKV0Fp3)**

- 唐纳德**-J-**川普（**@realDonaldTrump**）**2021**年**1**月**6**日


此前，国会联席会议期间，参众两院正在就是否认证各州选举结果进行辩论，大批民众冲进国会大厦。

在川普发布视频前，他呼吁示威者保持和平，不要攻击美国国会警察。

根据现场照片，一些人能够进入众议院和参议院大楼—甚至坐在议员的椅子上。

大多数国会议员–无论共和党和民主党–都谴责这一事件，并呼吁民众离开。众议院议长南希·佩洛西（Nancy Pelosi）、参议院少数党领袖查克·舒默（Chuck Schumer）和乔·拜登（Joe Biden）呼吁川普向支持者讲话，让他们离开。

拜登在下午4点左右的电视讲话中说，川普应该 “现在 “就上电视向抗议者讲话，并让他们回家，他把这次活动称为 “叛乱”，”近乎于煽动”。  佩洛西(Pelosi)说，她和舒默 “呼吁川普总统要求所有抗议者立即离开美国国会大厦和国会场地”。

正在宣读选举人证书的副总统迈克·彭斯也在推特上谴责了这一暴力事件。

“发生在美国国会大厦的暴力和破坏必须停止，而且必须立即停止，”彭斯写道。”任何参与的人都必须尊重执法人员，并立即离开大楼……和平抗议是每个美国人的权利，但对我们国会大厦的这种攻击是不能容忍的，参与的人将受到法律的最大程度的起诉。”

官员说，国民警卫队正在前往国会大厦的途中。

白宫新闻秘书凯利·麦克纳尼在一份声明中说：“在川普的指示下，国民警卫队正与其他联邦保护部门一起上路。”

她补充说：”我们重申川普总统的呼吁，反对暴力，保持和平。”
