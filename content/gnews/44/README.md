## 在中共国的西方资本恐将被消失
` 秘密翻译组G-Translators` [轉載自GNews](https://gnews.org/zh-hans/713371/)

![]()![](https://gnews.org/wp-content/uploads/2021/01/capture-2.jpg)图片来源:天下杂志
新年伊始，美国《国家评论》杂志在其网站上发表了一篇评论文章，对中国共产党的“专制资本主义”从其发展到如今对西方政府和公司的渗透和影响做了一个比较全面的刨析，并警醒西方社会要对中共模式的资本主义即将产生的巨大负面影响做好准备。

**中共实施的所谓自由经济并未真正意义上的资本主义**

文章开始就明确地指出，虽然专制的中国共产党与资本主义格格不入，但中共深刻了解一个事实，就是富裕和安全的资本家是专制政权的天敌。在中共的专制和资本主义混合模式下，资本主义只是其创造财富的手段，而权力才是其最终的目标。追求资本主义成功的企业家本能的要求中共可以保护其个人和财产权利不受专制法令的影响。企业家手中的资本是一种政治资源，会对中共实施其集权计划形成一种威胁。

意识到这一点，中共开始通过派驻党员到私人公司以及国企投资私企等方法实现其对私人企业的控制。在基本公民权利和司法独立缺失的状况下，私人企业和公司无法在中共国获得真正的公司独立经营权，异议和对公民权的要求对于中共政权都是威胁，必定遭到中共的镇压。中共国从鼓励外部投资和内部竞争向仇视资本主义的转变，在某种程度上可以说是步前苏联的后尘。1928年前的苏联实行的是经济自由化政策，从而挽救当时濒临崩溃的经济，但1928年后，斯大林突然改变了方向，实施集体农业化清算富裕农民，自由经济嘎然而止。中共国自己的经济自由化实验始于1981年的邓小平时代，他在确保中共对权力的绝对把握的同时，开始去中心化，将私有元素注入经济活动中。随着经济的自由化，国际企业被邀请进入中共国，但代价却十分高昂，因为中共政权要求他们与当地公司合作并对其进行培训，这种安排导致知识产权被广泛盗窃，而且国内竞争对手常常在政府补贴的帮助下，迅速取代了这些国际竞争对手在国内市场的地位。中共政权所支持的公司利用其国内竞争优势进入国际市场，削弱国际竞争对手。同时，中共国对国际“合作伙伴”实行不平等的监管，将他们排除在中共国市场外，例如Uber等数不胜数的例子。

**在中共国的西方投资正在面临被消失的风险**

西方已经开始意识到这是一场不公平的游戏，正在放缓流入中共国的资本，但这并不表示中共的新商业主义形式的资本主义就要完结，因为它已根深蒂固，无法迅速连根拔起。然而，中共并未放松对经济的政治控制，私企和高管的决策行动自由被大幅度削减。例如在11月时，中共以商业模式与党的目标不符为由，出乎意料地阻止了蚂蚁金服的首次公开发行股票。

在中共国有大量投资的国际企业必须要做最坏的打算，他们会被迫以“优惠”方式出售其在中共国的设施和业务。同时，鉴于中共对资本输出的管制，中共邓时代的资本主义“实验”结束后，许多西方在中共国的投资很有可能被没收，国际市场上的西方竞争者会最终意识到，来自中共国的对手其实既受中共摆布，又受中共国国家政权工具的支持。

西方社会一直想当然地认为，尽管中共国是一党专政，但只要其拥有自由市场经济体系，就应该被给予自由市场中的贸易伙伴的待遇。无论过去中共国政经之间如何貌似的无关联，但在共产党实际掌控了名义上的独立公司后，政经之间所谓的距离就消失的无影无踪了。许多有中共背景的公司，包括一些具有战略意义的行业中的公司，已开始不履行其债务义务了。国际债权人会被允许收回资产吗？当公司股权持有人为中共或中共国的地区和地方政府时，它们会被清除吗？如果政府救济这些公司，是否会同样平等对待国内外债务人？或是当这些公司换了所有权甚至以一个新的公司品牌继续运营的情况下，外国债权人会发现他们的资产实际已经名存实亡了吗？可以肯定的是，外债将会被显性或隐性地废除。从前的商业债务目前面临着通常主权债务才有的风险，即政府命令可以废除债务。简而言之，西方企业在中共国的资产账簿上正在迎来一个减记的浪潮。

文章最后指出，在中共眼里，西方企业并不是自由市场意义上的竞争者，而是被视为影响其主权利益的敌手，因此中共会尽其所有来对付他们。不久的将来，在中共国的西方资产极有可能会被大量没收，西方企业高管们最好对此做好准备。在此之前，美国政府应通过立法允许西方公司就被中共国没收的财产向美国法院提出赔偿要求。同时，因为中共宣称控制所有中共国公司，基于诉讼和立法的目的，所有这些公司都应被视为一个独立政府控制实体的一部分。西方要准备好为参与中共资本主义买单。

**点评:**  在所谓社会主义的共产中国，中共借用资本主义名义壮大了其权贵阶层，无论国营民营，其根本目的是为权贵资本服务，没有任何真正意义上的自由经济。中共国没有真正的资本家，即使像马云这些身家上亿的所谓企业家，在中共国不过就是上层权贵的白手套，都是权贵的奴隶。中共国是一个混合怪胎型经济体，其本质就是中共权贵用来实现一党专政的工具。许多中共官员同时也是某某公司的负责人，中共还将党支部建立在包括外资企业在内的所有公司实体中，以确保所有公司遵守党的法律，党的眼目能监视所有一切。西方对中共国几十年来绥靖政策现在终于走到了尽头。正如文贵先生所说，正是以美国为主的西方社会不断对中共的误判和纵容造成了现在这种局面。亡羊补牢，希望为时未晚。
