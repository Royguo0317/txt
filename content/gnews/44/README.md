![]()![](https://gnews.org/wp-content/uploads/2021/01/15379279793144o51q09716-e1609767566399.jpg)1943年，南泥湾，359旅开荒生产。周从初 拍摄 
1941年，王震带领八路军120师359旅在陝西省延安城东南45公里一个叫南泥湾的地方，开始了毛泽东提出来的军民大生产运动，以满足根据地的物资供给，史称“军垦”运动。从当年3月到11月警卫营开到，359旅採用刀耕火种的原始烧荒方式，焚毁森林，烧木肥土，使南泥湾的森林消失殆尽。
![]()![](https://gnews.org/wp-content/uploads/2021/01/39b0695009cb43eebf47a21fb8ee186e_th.png.jpeg)图为音乐家马可&马海星在读父亲写的信 图片来源: sohu网站
为配合中共宣传“军民大生产运动”，1943年由贺敬之作词，马克作曲的《南泥湾》应势而出，歌词中“陝北的好江南”即是指南泥湾。歌中唱到：

花篮的花儿香 听我来唱一唱，唱一呀唱。
来到了南泥湾，南泥湾好地方，好地呀方。

当年的南泥湾，到处呀是荒山，没呀人烟。
如今的南泥湾，与往年不一般，不一呀般。
如啊今的南泥湾， 与呀往年不一般，
再不是旧模样，是陝北的好江南。
![]()![](https://gnews.org/wp-content/uploads/2021/01/1537928151403ps4snq992p-e1609768262853.jpg)图为1943年，毛泽东的题词：“自己动手”、“丰衣足食”
那么南泥湾果真是“一把锄头一支枪，生产自给保中央”的农垦运动的先进代表么？各类文件中大生产运动种植的粮食和“特货”究竟是什么？
![]()![](https://gnews.org/wp-content/uploads/2021/01/6357704603280585409924289.jpg)图片来源: 民初思韵网站
1990年中共历史专家台湾学者陈永发发表了《红太阳下的罂粟花》，提到了中共大生产运动中种植的农作物中有粮食和特货（鸦片）。种植罂粟的原因是因为当时中国境内的鸦片被广泛当做硬通货使用，陝甘宁边区物产贫瘠，种植鸦片出口用赚取外汇。

![]()![](https://gnews.org/wp-content/uploads/2021/01/Unknown-4.jpeg)毛泽东贴身警卫张思德，在南泥湾烧鸦片时被砸死在窑洞裡，毛为此还亲笔题词悼念。那题词就是在北京中南海南门影背上那五个大字：爲人民服务。张思德究竟何许人也？他在南泥湾干了啥？（大纪元合成）

而大陆学者张耀杰的研究，毛泽东亲笔题词的中共中央警卫团战士张思德并非死于烧木炭，而是烧製鸦片的过程中意外身亡。

关于南泥湾的鸦片种植事实的隻言片语，在维基百科中有如下资料：

1942年特务团、九团、四支队来南泥湾「皆曾种地，但中心在种特产。（《南泥湾调查》，《中共中央西北局文件汇集》一九四三年（一），第270页，中央档案馆/陝西省档案馆，1994年11月出版）

在1942年各单位中底面积调查表中，八团种植1000亩，收烟300两，特务团收烟100两，警卫团收烟800两，炮兵团种植六百亩只出八十亩，收烟几十两。「第一，贪多；第二缺乏经验，未好好的採访老百姓的意见，如炮兵团下种时，老百姓说太早，种了不出来，我们不听，结果六百亩只出了八十亩。其他是施肥割烟皆不熟悉……」。(《南泥湾调查》，《中共中央西北局文件汇集》一九四三年（一），第272页，中央档案馆/陝西省档案馆，1994年11月出版)

时任三五九旅九团九连连长，后长期担任南泥湾生产大队长的刘宝斋所述，所在部队在南泥湾孟酒沟种大烟，用多馀的粮食酿酒，菸酒贩到国统区卖钱，「为这事常同机关的打架。连队要挣钱，机关要收税。一次往河东贩卖烟土，货上了船，船帮上一圈端刺刀的战士，看谁敢挡……」。（高红十. 哭刘老. 延安一片月. 武汉大学出版社. 2012年1月: 第182页.）

1947年胡宗南进攻延安，从陇东佯攻，主力从宜川北上延安，走多年未走小路奇袭。南泥湾的金盆湾是其中路线之一，国军参谋人员化妆为鸦片走私商人进入边区，「沿路贩卖西药和布料，归途带鸦片回来」，确认了路线通畅。（张朋园. 攻延安的部署. 盛文先生口述历史. 中研院近代史研究所口述历史系列. 北京: 九州出版社. 2013年: 第66页）

![]()![](https://gnews.org/wp-content/uploads/2021/01/v2-702dab5650c6921613af82aafcaeee7f_hd.jpg)图片来源: getit01.com

实际上中共红军时期在部分苏区就已经开始收鸦片税，零星开始鸦片贸易了。到抗日战争时期，由中共官方主导鸦片种植与鸦片贸易，解决财政危机。国共内战时期，鸦片已经作为重要经费来源。根据《山东解放区海关史料纵览》，国共内战后期，中共已将鸦片通过山东烟台等地销往香港。

中共1949年建政以后，继续通过香港向世界出售鸦片与鸦片製品，获得外汇。1952年全面禁毒后，在黑龙江生产建设兵团、新疆生产建设兵团的国营农场以「药用」为名，种植罂粟，约到20世纪80年代初才终止大面积种植（一三○团史志编纂委员会. 一三○团志. 北京: 中华书局. 2002年: 第124页），中国药用罂粟集中到甘肃农垦集团种植。

鸦片贸易当时解决了中共财政困难，对中共夺取中国政权有著决定性影响。如1949年年初西北财经办报告：「由于西北贫瘠，特货（代指鸦片）七、八年来向为西北支持财政，稳定金融，周转贸易的槓杆。一九四二年以来，凭藉著这种特种经营解决了财政经费的百分之七十上下，换入入口物资历年来供给了党政军百分之六十至八十以上，供给市场百分之四十至二十，银行外汇之绝大部分过去也依靠了它的出口换取，这些都相对地减轻了人民负担和支援了革命事业。在一九四八年，除华北援助之外，也还主要地是依靠它使我们渡过了财政困难的一年，如果现在忽尔停止了这种经营，则今后将用何种力量来周转贸易和支持金融，实为一很大困难问题」。

共产主义政权製毒贩毒问题并非中共一家，朝鲜劳动党、越南共产党、缅甸共产党、哥伦比亚革命武装力量等共产主义政权或武装，都在遇到财政困难时，将目光瞄准了邪恶的毒品。这也许就是共产国际无耻邪恶本质的体现。

中共鸦片的大量种植和贩卖始于延安城东南的小小南泥湾，一首红遍大江南北、看似歌唱自力更生的《南泥湾》背后竟然是贩卖鸦片的罪恶。在整个中共专制压制下的中华大地，究竟还有多少披著美丽光辉的邪恶，荼毒著被蒙蔽的民众？
