## 什么是SCIF?

今天郭先生发布了一条盖特，大概内容是 “在选举之夜，如果白宫观看结果，特朗普及其竞选团队将聚集在西侧的艾森豪威尔大厦的**SCIF**。”

什么是SCIF？SCIF的功能是什么？通过在网络搜索相关资料得知，SCIF全称为Sensitive Compartmented Information Facility，汉语译为：国防/国家安全及敏感情报信息隔离设施。SCIF特指在建筑物内用于处理敏感信息和机密情报的封闭区域，通俗讲就是我们经常听到的“安全室”。

美国政府和军队高级官员会在SCIF内商议与国家安全相关的机密信息。通常会在建筑物内修建永久性SCIF设施，最著名就是白宫的“安全室”。 如果总统外出期间，遇到紧急情况，也可以迅速搭建便携式SCIF设施。搭建地点既可以在酒店房间、也可以在舰船或私人住宅等处。在临时性SCIF设施内部，可以进行敏感文件处理或召开秘密会谈的需要，也可以通过加密线路联络SCIF外部人员。

SCIF主要防范外国情报部门或未经授权的人员，在未经授权的情况下，侵入固定或临时设施获取机密信息。SCIF最大特点是防范电子监控，防止敏感安全和军事信息和数据泄露。因为所有敏感信息必须在SCIF的封闭空间进行处理，存储，使用或讨论，同时必须遵循一系列统一的安全标准，包括物理和技术要求。SCIF周围通常有多层保护措施，会执行最严格安保措施。所有获准进入SCIF的人员，必须经过多重安全检查和信息验证，并在使用过程中受到严格监控。

为什么特朗普竞选团队要在最关键的11月3日选举夜聚集在SCIF？他们到底在防范什么，同时又通过世界最先进的信息监控和处理系统了解什么？

笔者大胆推测，2020年美国大选前，特朗普总统已经获得中共协助拜登阵营进行选举舞弊的准确情报，以及幕后中共操纵选举，进行选票造假的相关计划。特朗普竞选团队可能已经在投票之前布下了收集选举舞弊证据的“天罗地网”，因此在11月3日晚聚集在SCIF等待获得相关证据。

我们已经知道，中共在过去的三十年间，通过蓝金黄等手段对美国进行了全方位的渗透，白宫也在某种程度上被攻陷。通常情况下，在大选夜总统候选人都会在竞选总部，和自己的竞选团队以及支持者们分享选举的最新进展，等待选举结果的揭晓。但此次美国总统大选非比寻常，特朗普及其竞选团队可能已经预计到拜登阵营可能的选举舞弊手段，因此才会来到SCIF，通过这里的绝密信息监控和联络渠道，在及时获取计票现场一手信息的同时，根据情况进行证据获取和策略调整。正因为SCIF的最高保密级别，在这里的信息才不会被中共及拜登阵营获取，才可能使得他们“丑行”可以有机会大白于天下。我曾经在GTV上看到过一段视频，有一位女性计票员旁若如人的篡改着一摞选票，而旁边执勤的警察来回走动，居然对这位计票员的举动视若无睹。该视频的拍摄角度，是在计票员的头顶上方。我不知道这段视频流出的渠道，但我相信大选之夜在SCIF的特朗普竞选团队成员，应该在很多摇摆州看到了类似的选举舞弊行径。

相信特朗普总统及其竞选团队在准确情报的支持下，早已进行了“取证”布局，同时也应该掌握了问题选票的分布区域及相关数据。相信随着选票造假信息和视频，以及相关证人证词的相继曝光，将会逐步收集到所有在SCIF获取的线索的实锤证据，将会成为重要的法律证据。这些证据不仅将破灭拜登窃取选举成果的企图，也将会彻底粉碎中共妄图操纵美国大选，称霸世界的野心。
