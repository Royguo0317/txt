摘要：面对严峻的国际形势和国内经济的极端困境，中共大力推出各种示范区的目的，除了提振中国社会信心以外，极有可能是在世界各国跟中共全面脱钩之后，为收紧中国社会空间、号召人民自力更生做出的一系列准备。

**什么是“共同富裕示范区”**
“共同富裕示范区”是继中共千年大计的「雄安新区」之后，共产党在浙江省设立的一个以国家资源、政策倾斜扶持的一个试点区域，目标在2035年把浙江打造成全面富裕的例子，然后批量复制推广到全国，让中国人实现共同富裕。

「全民小康」还没有完成，“全民富裕”先来了，共产党深深的知道「全民小康」这个词已经骗不了中国老百姓了，所以只能画一个更直白的大饼“全民富裕”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/15054310/%E3%80%82.jpg)图片来源：网络
“共同富裕示范区”还没有让中国人开始富裕呢，中共的吹鼓手学者，人民大学副校长刘元春就开始迫不及待的为世界开起了中国药方：“设计富裕示范区要为全人类所面临的收入分配两极化提供中国的方案、中国的思想”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/15054808/0-5.png)图片来源：网络
除了绕过大部分贫困的中国地区，选择了一个较为富裕的地方去建设“共同富裕示范区”以外，中共这个“共同富裕区中”的政策、指示还是一如既往的假大空骗，所有的政策措施都流于表面，根源性的问题从来不提及和解决。

**共同富裕真的能实现吗？**
我理解中的共同富裕就是一个国家的生活成本不高，收入差距不大，政治清廉，社会贫富差距小，国家有各种稳健的政策和社会保障体系。

欧美资本主义国家就实现了我理解中的共同富裕，完善的社会保障，低廉的物价，均衡的税收结构，合理的收入分配制度，干净的生存环境，和有底线的人文环境。
这不就是共产党口中的“共同富裕示范区”吗？

反观中共，推行的所有政策都是缺乏基础的空中楼阁，设立的各种“示范区”也只是为了政治的需要和政治口号的需要，这种虚假的政治宣传口号和政治游戏共产党在中国大地操作了几十年，至于老百姓，领导说富裕了老百姓就算是富裕了。

最终，中共治下的这些“示范区”唯一能发挥的作用只能是推高当地的房价，推高当地百姓的生活成本，让“共同富裕”的目标变得离老百姓更远，让当地官僚集团实现“共同富裕”的目标变得更近。
