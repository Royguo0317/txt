## 分析：川普总统通话其实揭示了一个深刻的细节，即共和党建制派们抗拒，不愿意，也拒绝透明

*“**这个我不知道，我面前没有，我们正在调查……**”*

这不是美国总统在本周日被《华盛顿邮报》（最近从中共那里拿了几百万美元）泄露的电话中说的含糊不清、不置可否的话。它们是建制派共和党人在电话中所说。州务卿布拉德·拉芬斯伯格、瑞安·日耳曼（州务卿法律顾问）和副州务卿乔丹·福克斯在1月2日的通话中所说。

尽管《华盛顿邮报》进行了党派诬陷—认为这次通话在某种程度上反映了川普总统向共和党同僚提出了选票要求，但实际上总统并没有这样做。在整个通话过程中，总统明确表示，他的呼吁是为了选举透明、全面透明的审计和公众参与。总统从来没有像建制派媒体所描绘的那样，暗示他希望选票被捏造或伪造。 他甚至提出要回避部分谈话，并在最后要求 “真相…… 就是这么简单。”

您可以在本文下方阅读完整通话记录并收听整个通话内容。

其实总统在通话一开始就撕破了具体的细节，而这些细节是他们的对手，建制派共和党人在通话中从未提及的。川普总统表示：

“我们至少有2或3—25万到30万张选票被神秘地丢进了选民名单 其中大部分与富尔顿县有关，该县还没有被检查。我们认为，如果你检查签名—真正检查富尔顿县的签名—你会发现至少有几十万人的签名是伪造的。”

他继续说：”我们有，我相信有4,502名选民参加了投票 但他们没有在选民登记名单上，所以有4,502名选民参加了投票， 但他们没有在选民登记名单上，而他们必须（是在登记名单上的）。你们有18,325个空置地址的选民。地址是空的 他们不被允许被计算在内。这可是18,325人。”

总统继续说 “你们有州外的选民” “他们在佐治亚州投票，但他们来自州外，有4925人。你有缺席选票寄到了空置的地址 他们是寄到空置地址的缺席选票。他们身上没有任何关于地址的信息，那是2326张。”这种细化程度其实很了不起，尤其是对于一个媒体不断指称不关心细节的总统来说。

而拉芬斯伯格团队甚至没有特别有力的回击（这些问题），这大概也是他们泄露通话内容而非继续对细节进行讨论和要求的原因。通话中有几个关键点反映了佐治亚州共和党建制派领导人的不诚信：

1. 富尔顿县

总统和他的团队一再提出富尔顿县涉嫌违规，特别是指控拉芬斯伯格和团队在没有解决人口更密集的富尔顿县问题的情况下就转向科布县。

川普总统在这里说 “你把我们的人派去了我们并不想去的科布县。而且你说会公之于众。所以我们派去了我们的专家，但他们不允许进入房间。但我们不要科布县，我们想要富尔顿县，而你不给我们。现在，我们为什么不做签名（检查）—为什么不能向公众开放？”

对此，律师莱恩·日耳曼简单地回答道。“我们选择了科布县，因为那是唯一一个有证据表明 签名验证没有正确进行的县。”

不仅仅是应承认该州的错误行为，还应承认有选择性地采用严格的标准，并拒绝让总统团队进入该州人口最多的县。

2. “同意令”

这与共和党和民主党在大选前签署的协议有关，也与为什么川普总统说极左的佐治亚州活动家斯泰西·艾布拉姆斯（Stacey Abrams）“玩弄 ”了共和党人，让他们签署了一份违宪协议有关。协议本身就是一个文字沙拉（自己看），关于什么是选票上的签名匹配。如果没有一方意图欺骗选举，人们可能会说，就不需要这样的 “协议 “了。

拉芬斯伯格和日耳曼没有直面这个问题，而是开始挑剔 “同意令 “这个词，这个词在国家、州和地方媒体上被反复使用来描述这个协议。甚至连反对川普总统的律师也使用了这个词。

拉芬斯伯格在整个交流过程中，只是简单地表示，。”这是一个和解协议。” 他的同事莱恩·日耳曼也绕开了问题的关键。”我不知道，我不知道。我不。我不相信[这是一份同意协议]，但它不在我面前。”

3. 佐治亚州律师拒绝进一步透明

在电话结束时，听到幕僚长马克·梅多斯说：“听起来我们已经得到了两个不同方面的一致同意即我们可以对该些地区进行调查，假设我们可以在接下来的24至48小时内，继续进行核对，以便我们可以查看两项声明同时确保我们可访问州务卿不论是声明有效还是无效的数据 是这样吗？”

日耳曼立刻反驳道。“不，我不是这么说的”并试图阻止进一步调查佐治亚州选票总数的想法。

当时总统的一位律师库尔特·希尔伯特建议：”州务卿可以代表总统的律师，这样我们就可以在你没有任何违规行为的情况下，获取这些信息和私人信息？”总统理当即说：”我并不想知道是谁。你们可以很保密地进行。你们可以签一份保密协议。这没关系。我不需要知道名字。”

4. 鲁比·弗里曼。

或许不是拉芬斯伯格团队的过错，但《华盛顿邮报》的音讯最初在提及佐治亚州选民统计和左翼活动家鲁比·弗里曼和她的女儿沙耶·莫斯时留下了。固然《国家脉动》并没有发表声明，但《华盛顿邮报》会删改这些名字是令人费解的，特别还提及她们在互联网上是如何的知名。

这些名字一共被《华盛顿邮报》编改了18次。

5. ‘我们只想知道真相’

在通话结束时，川普总统如他一直在反复重申的：“我们只是想知道真””

这远不是《华盛顿邮报》报道的那样充满 “压迫 ”的通话，总统在62分钟的时间里，遍问细节和障碍，不断确定他要的是准确、透明和公开。

律师克莱塔·米歇尔（Cleta Mitchell）或许是在电话中说过的最好的，她说：“……你可以不断地告诉我们，并公开声明，你调查了这件事，这里没有什么可看的。但我们不知道这些。我们只知道你告诉我们的事情。我不明白的是，为什么不符合每个人的最佳利益的方式进行彻底（检查），比较如你所说的这些数字，因为… 试图能够得到真相，并且我们没有任何方式能确认你所告诉我们的。你说在州立农场体育馆进行了调查。我没有任何报告 我从未见过调查报告 我不知道那是什么 我一直在参与其中，但我不清楚。而这只是25个类别中的一个。”

对于像亚历山大 奥卡西奥·科蒂兹（Alexandria Ocasio·Cortez）这样的民主党人来说，为了这通电话就呼吁另一次弹劾的努力是可以预见的，也是令人厌恶的。很有可能A·O·C本人 在回应媒体的评论请求之前，根本都没有听过这个电话。

但这就是当今政治和媒体的现状—一边处在不断的狂热削弱宪法规范的状态中，一边将矛头指向另一方。

任何真正听过电话的人都会听到川普总统所说详尽且直率。川普总统的沮丧可以理解，当然他采用了一些纽约商人特有的虚张声势的方式。但重要的是：他采用的是一个期望公开、透明的人的说话方式。而对方根本没有。

这就说明了我们需要知道的一切。
