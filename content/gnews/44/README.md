**西班牙2020年12月5日电/西喜社/**

川普竞选团队已在乔治亚州的一家法院提起诉讼，试图在举行“新的总统选举”之前，停止对该州总统选举结果的认证。

周五在富尔顿县高等法院提起的诉讼称，选举结果中计有成千上万张非法选票，从而导致“对有争议选举的结果产生了重大怀疑”。

该诉讼要求：“应禁止，撤消与之相关的任何认证，”并立即下令进行符合乔治亚州法律的新总统选举，或采取符合乔治亚州宪法的公平的救济方式”。

该诉讼进一步辩称: “批准紧急禁令救济是符合公共利益的，将使乔治亚州选民对2021年1月5日的参议院选举符合《选举法》抱有信心。”
