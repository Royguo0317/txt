1月22日，俄罗斯商业咨询报道，俄罗斯赞成保留《削减战略武器条约》 (START-3)，并欢迎美国决定将其延长5年。

俄罗斯总统新闻秘书德米特里·佩斯科夫（Dmitry Peskov）指出，俄罗斯欢迎有延长该文件的政治意愿这一事实。“当然，我们只能欢迎政治意愿，但一切都要看这个提案的细节，当然，现在我还不准备做任何评论，他们还有待研究”。

佩斯科夫认为，在这种情况下，考虑到彼此关切的问题，“考虑到俄罗斯的”立场是非常重要的。

在美国总统拜登就职当天，俄罗斯外交部发出延长《第三阶段裁武条约》的提案，称美国前总统川普政府试图在延长条约的提案中侵犯俄罗斯利益。

两天后，白宫新闻秘书詹·普萨基表示，华盛顿向莫斯科提出将《第三阶段裁武条约》最长延长5年。五角大楼支持白宫的倡议，称这符合国家安全利益，也给两国提供了探索新的军控机制的机会。

联邦委员会在接受俄罗斯广播公司的访谈中积极评价了将条约延长至2026年的建议。同时，联邦委员会国际事务委员会副主席安德烈·克里莫夫（Andrei Klimov）承认，美国并没有全部接受俄方提议，提出了美方的建议。他说：“我们和美国的谈判人员要在这里找到一个解决方案。”。

评论：美新政府上任后，俄罗斯积极抛出橄榄枝。俄罗斯提交了《第三阶段裁武条约》，美国新一届政府已同意最长延长5年，具体实施措施将在双方谈判代表签下方案后才能知晓。俄罗斯总统新闻秘书佩斯科夫表示，俄总统普京和美国总统拜登会谈问题暂未提上日程。在12月普京发给拜登的贺电中，普京表示，“对全球安全和稳定负有特殊责任的俄罗斯和美国尽管存在分歧，但能切实促进解决世界现在面对的许多问题和挑战”。普京表达了“俄美合作应符合两国人民乃至整个国际社会的利益”，他为两国合作的进一步接触已经做好了准备。
