## 美国印度太平洋司令部寻求270亿美元来对付中共

随着美国外交政策越来越侧重于亚洲以应对岛链危机，美国军方在该地区的司令部正在寻找更多资金。在3月1日提交给国会的一份新报告中，美国印度太平洋司令部（INDOPACOM）呼吁在2022年至2027年之间增加约270亿美元的支出，以便更好地与中共对抗。

该报告由印度国防工业发展局局长菲利普·戴维森（Philip Davidson）将军编写，尽管有一些重要变化，但与该司令部去年发布的建议相似。去年的报告是被《太平洋威慑倡议》（PDI）所要求的，该计划被纳入《2020年国防授权法》（NDAA）。

《太平洋威慑倡议》也包含在2021年《国防授权法》中。它被插入到军费开支法案中，以确保印度国防军司令部拥有对抗中共和实现2018年国防战略所需的力量，该战略概述了美国军方从中东的反恐转向俄罗斯和中共国，即所谓“大国竞争”。

印度国防工业厅呼吁美国在关岛建设一个价值16亿美元的360度持久和综合防空能力，其中一个是在普劳建造的价值2亿美元的高频雷达系统，另一个是价值23亿美元的天基雷达群。该报告中的另一项值得注意的提议包括一个33亿美元的地面远程导弹系统，以构成“第一岛链沿线高度可生存，精确打击的网络。” 第一岛链环岛包括日本附近的群岛、台湾、菲律宾和印度尼西亚。

五角大楼正在审查其在亚洲的态势以及军方对华总体政策时，发表了美国印度太平洋司令部报告。此次审查由对中共国鹰派的伊利·艾瑟（Ely Rather）领导，他被任命为国防部长劳埃德·奥斯丁（Lloyd Austin）就亚洲事务提供咨询。自拜登总统就职以来，多艘美国军舰驶入南中国海，并穿越台湾海峡，加剧美中的紧张关系。

国防部评估说，如果中共国对台湾使用军事力量，它可能会通过干扰和网络攻击来补充其空中和海上封锁行动，以及在控制冲突的国际叙事时隔离台湾当局和民众的信息行动。

拜登政府已表示将对中共国采取强硬态度。国防部长劳埃德·奥斯丁在上个月的参议院确认听证会上，承认中共国威胁五角大楼的步调，并称《太平洋威慑倡议》是“一种有用的工具”，他将与国会合作实施。

奥斯汀在确认之前说：“毫无疑问，我们需要在印度太平洋地区采取更具弹性和分布力的姿态，以应对中共国在新作战概念的支持下的反干预能力和方法。” “如果得到确认，我将回顾我们在印度太平洋地区的情况，包括我们的表现、能力、后勤、演习、基础设施以及能力建设和与盟国及伙伴的合作。”
