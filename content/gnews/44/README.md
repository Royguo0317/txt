## 【香港要闻】港共政府白票废票的恐惧 称集体投白票或涉操纵破坏选举
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1092458/)

蒐集\編撰：文燕 / 封面合成：文粵

政府公布修改选举制度细节，修例订明选举期间公开煽惑他人不投票、投白票或废票，或故意妨碍或阻止他人投票均属违法，律政司司长郑若骅在记者上透露，透过手势、或是在衣物上传递有关资讯，均被视为是「公开活动」。政制及内地事务局局长曾国卫重申，条例针对「煽惑」行为（背后的动机）而非投白票本身，强调煽惑投白票会「破坏」选举公信力。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4149.jpg)
林郑月娥昨日出席行政会议前被追问投白票问题。 【立场新闻】指，林称如果有人在选举期间，煽动选民集体去做一些行为（投白票）可能触及破坏、操纵选举。林郑月娥再以去年民主派初选打比喻，指有人提议「揽炒 35+」，是有人有计划、有意图，组织煽动进入立法会，反对所有法案，继而令行政长官落台。

**违例煽惑不投票　最多判囚 3 年**

政府新公布的条例草案中，建议订明任何人在选举期间，借公开活动煽惑他人不投票、投白票或废票，即属违法；任何人故意妨碍或阻止另一人在选举中投票，亦属干犯舞弊行为，违反相关条例最高刑罚监禁3 年。

律政司司长郑若骅解释，无论该人是否身处公众地方，亦可作「公开活动」，包括向公众作通讯，例如谈话、播片、发电邮；而由公众观察到的行径、例如动作、手势、衣物，或是在窗外展示信息；向公众分发、传播材料，都算是公开活动。

**林郑月娥：仅限制「选举期间」呼吁**

有记者问及，如现在有人呼吁投白票，是否会被视为违法。林郑月娥称，条文十分清楚，是限制市民不要在「选举期间」作出相关呼吁，不是「任何时候」。翻查现行《选举(舞弊及非法行为)条例》，「选举期间」即该选举提名日起，至选举投票日止。

**战友观点：**

政府对呼吁投白票、废票的「规管」建议一出，网民已纷纷在讨论区列举不同情境，如「举行提醒人『煽动人投白票是犯法』的活动」、「投票日着白色衫」是否违法；又质疑何为「投白票手势」，声称「不想误做手势犯法」。亦有人在社交平台发文反讽，「各位千万千万千万不要投白票」，又反问为何投白票合法，但呼吁别人做合法的事是犯法。

记者会上，有记者多次问及有关「公开煽惑」的定义细节，但港府官员都没有清晰回覆。

**1）「公开活动」及「选举期」的定义**

如于私人聚会、家庭聚会、公司聚会内呼吁，是否属「闭门」；而「公开」及「闭门」的概念，是否也适用于网络及社交媒体；若网上私人发布的讯息，被他人用于在其他群组发布，是否亦有刑责。

**2）针对「煽惑、呼吁」情景**

【立场】记者向政府查问，假如员是分析「投白票」、「投废票」或「不投票」的效果、或夸张地呼吁「记得不要投白票」、又或声言没有候选人合适当选、或者表示自己已经投白票，或不会投票、转发他人呼吁投白票、废票的资讯；或是呼吁「凭良心投票」、「做正确的决定」、「心照」、在海外呼吁投白票等等，是否需付上刑责。

官字两个口，党同伐异，都是他们说了算。政府所公布的选举细节定义非常不清晰，可以随便伸延定义。到底为何港府这么惧怕港人投白票？因为他们自知这次由中央「完善过」的选举，根本就不是真实的的选举。
