## 多少老去的父母根本领不到中共承诺的独生子女费

![]()![](https://gnews.org/wp-content/uploads/2020/12/屏幕截图313.png)
1978年，中共将计划生育政策正式写入宪法， “国家推行计划生育，使人口的增长同经济和社会发展计划相适应” 。在1980年中央五次人口座谈会后确定了“提倡一对夫妇生育一个子女”。盗国贼们偷偷地到处播种，但残酷的计划生育就这样在中华大地开始了。

从1982年陆续开始，中共中央、国务院当年下发的《关于进一步做好计划生育工作的指示》中第三十七条第一款写明“对于城镇居民中的独生子女父母，男性满60周岁，女性满55周岁时，按一定标准发放计划生育奖励金”。

几十年过去了，响应党的号召，膝下仅有一个子女的父母渐渐老去，到了退休的年纪，想起党当年的承诺，纷纷去申领独生子女费，可结果如何呢？
![]()![](https://gnews.org/wp-content/uploads/2020/12/c7f5438ce9ba481093aeed01d4b92be4.jpg)
在网上搜索“ 不给独生子女费”，可以看到一些信息。比如有的单位以效益不好没钱为由；有的单位以工作调动、年限不够为由；有的单位推社区、社区推单位等等，就是不给发放。网上能发出声的毕竟是少数，还有多少年迈的父母忍气吞声，吃了哑巴亏呢？

有人咨询法律顾问，答复是：未发放独生子女费肯定是违法的。但现有的司法体制，对单位未发放独生子女费的行为如何处罚、由何部门处罚均未规定。而且劳动仲裁和法院对此也不受理，所以想要讨个说法有难度。

老去的独生子女父母退休时，无权也无势了，中共就摆人一道，说不给就不给，在官官相护、只解决提出问题的人不解决问题的恶劣环境中，老百姓还能怎么办？

网易新闻《然并卵的独生子女费》中显示，高达83.1%的人根本没领过独生子女费。
![]()![](https://gnews.org/wp-content/uploads/2020/12/屏幕截图312.png)
有多少父母年轻时听信中共的承诺，孩子只生了一个。计划生育政策关系到家家户户，给国内造成了严重社会问题，包括大批处境悲惨的失独老人、男女比例失衡、人口老龄化等等。退休时领不到党当初承诺的独生子女费，老百姓得多么堵心？但此时已经没有了抗争的精力和体力。中共真是狠啊！
