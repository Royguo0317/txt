2018年8月10日，华盛顿邮报在评论费恩斯坦参议员的间谍司机一案时，写道：想象一下，如果发现众议院或参议院情报委员会的共和党主席有一名俄罗斯间谍正在为其工作，绝对会引起政治大风暴吧？好吧，这个月我们知道，一个中国间谍（Russell Lowe）（在薪资记录中被列为“办公室主任”），作为加州民主党参议员戴安娜·费恩斯坦（D-anne Feinstein）的司机工作了20年，而且这名司机一直通过中国旧金山领事馆向中国国家安全部报告。主流媒体对此有何反应？勉强一瞥而已。

不得不说这篇两年前的评论眼光独到，2020年12月20日，专家门户网站报道，福克斯电视台主持人玛丽亚‧巴托罗姆(Maria Bartoromo)在周日的访谈节目中说，她看到一张照片，中共女特务方芳（Christina Fang）2011年就和这名间谍司机站在一起合影。这张照片登在加州中国学生联合会的账号下，而方芳当时担任该联合会的学生会主席。

据美媒Axios报导，方芳2011年至2015年期间，在加州的政界活跃异常，专门搜罗湾区乃至全美有前途的政界新秀。在此期间，方与中共驻旧金山领事馆保持着异常密切的联系。旧金山湾区是美国高科技产业区硅谷的所在地，中领馆在当地从事间谍活动当然不弱。

据分析，中共努力渗透美国政界，有的时候甚至不惜花费数年甚至数十年的时间才能培养出“成果”，可谓放长线钓大鱼。因为“中共知道今天的市长和市议会议员，可能成为明天的州长和国会议员”比如年轻的加州众议员埃里克‧斯沃韦尔（Eric Swalwell）。方2011年认识他的时候，他才31岁还是一个名不见经传的市议员，正是缺钱缺人支持的时候，方芳出现了，多次给他筹款助选，帮助他迅速崛起，后来斯沃韦尔成为美国国会议员，被任命为众议院情报委员会委员，并担任中央情报局（CIA）监督小组委员会的首席民主党人，掌握着美国的高级机密。

方在2015年被FBI调查暴露前，还自愿参加了加利福尼亚州众议员Ro Khanna 2014年 （当时38岁）的众议院竞标，并为夏威夷州众议员Tulsi Gabbard于2013年 （当时32岁）筹款。尽管两者都说与方没有进一步的联系，但是凭借中共间谍放长线的功力，谁能说他们不是下一个斯沃韦尔。
