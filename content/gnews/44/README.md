6月9日，日本《统一日报》发表了一篇题为[【新中国連邦は中共消滅闘争の灯台】](http://news.onekoreanews.net/detail.php?number=88843&amp;thread=04)的文章，题目大意是：新中国联邦是灭共战的灯塔。这是笔者看到的第二篇日本媒体报道新中国联邦的文章。

文章说：在美中文明大战之际，旨在消灭中国共产党的新中国联邦迎来了成立一周年的日子。这是解放黑暗的中国大陆之灭共革命的灯塔。世界因病毒大流行和文化战争而陷入混乱，但为真理和生命的尊严而战的勇士们的出现，必将迎来结束100年极权共产主义的时代。

在1989年6月四日天安门广场抗议事件发生32年后的6月4日，世界各地都举行了追悼会。在香港，民间团体每年都会举行集会，吸引数以万计的人参加。但自从中共政府背弃“一国两制”政策后，对示威活动的镇压变得更加严酷。与此同时，新中国联邦在日本东京举行了纪念成立一周年庆典。去年，在天安门广场抗议活动31周年之际，郭文贵、史蒂夫·班农等人在美国成立了新中国联邦，旨在消灭中国共产党和中国人民解放军。

纪念庆典以美国为主会场，共四个国家举行——日本、英国和新西兰同时现场直播。 开场曲 《光荣的香港》与香港的示威录像一起播放，表达了新中国联邦对香港民主运动的支持。曾担任川普前总统法律顾问的纽约市前市长朱利安尼，以及曾担任总统助理、以调查2020年美国大选舞弊的《纳瓦罗报告》而闻名的彼得·纳瓦罗，在美国主会场直播演讲，严厉指责了中共政权对中共病毒的责任。

新中国联邦宣言中将中共定义为恐怖组织。详情包括“无视人权、破坏人性、违反民主、背弃依法从政、违反承诺、杀戮香港同胞、屠杀藏人、输出腐败、危害世界”。这场革命的创始人郭文贵先生，自2017年以来一直在推动“爆料革命”（揭露中共的腐败和政治丑闻的运动）。

以美国政治制度为蓝本的新中国联邦，将使各省和自治区充分行使自治权。香港、澳门、西藏和其他地区将实行特殊的独立自治制度。台湾将维持现状，通过贸易促进寻求稳定发展和共同繁荣。“特別独立自治”是指“将允许每个自治区拥有自己的司法和军事力量，就像美国的联邦制度一样，并授予其除国防和外交事务以外的所有政府管辖权。”生活在中共国的绝对大多数人，如果理解了新中国联邦的愿景，他们将别无选择，定会支持。

新中国联邦宣布，将永远废除社会主义制度和经济共享制度，中共掠夺百姓的财产将被收回。 新宪法和法律将由公民投票决定，借鉴西方国家的法律体系，包括英国的《普通法》和美国的 《判例法》。新中国联邦与迄今为止的反共运动的不同之处在于，它几乎完全在网络上运作。就目前而言，新中国联邦的日本分会正急于建立自己的媒体平台和金融基础设施，包括SNS和新闻媒体。通过GTV、Discord等平台主要针对中国大陆，用中文进行传播。郭文贵先生全力支持该运动，会员们（战友）也纷纷捐款。“法治基金”是一个独特的基金，几乎所有成员都向其捐款。

喜马拉雅监督机构指导下的新中国联邦在世界各地的分支机构不设办公室或其它实际地点。因此，会员的居住国不一定与本人所属分会的所在国一致。当然，必须非常小心地排除中共的监视和干预，所有成员都使用昵称而不是他们的真实姓名进行交流。

一位参加一周年庆典的成员说：“我们想告诉大家，中国人民和中国共产党是两回事，我们的宗旨是唯真不破。我们只讲真话，让所有正义力量成为我们的盟友，我们一定会赢。”庆典上，众议院议员原田义明也发来了视频讲话。

新中国联邦是照亮黑暗中共大陆的一座灯塔。韩国的自由主义右派也在与中共作斗争，中共毫不掩饰其以解放“金日成邪教极权主义”的朝鲜为目标，欲主宰朝鲜半岛的野心。”战友”们有一个共同的目标就是消灭集权中共，一个巨大的文明联盟正在孕育之中。
