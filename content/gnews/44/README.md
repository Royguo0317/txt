《华盛顿时报》2月8日报道，由85名众议院共和党人提出了重新授权Keystone XL管道的立法，他们共同发起的Keystone XL管道建设和就业保护法案将允许建造和运营该管道。他们谴责拜登总统上任第一天就取消该项目的命令是“对美国工人和家庭的灾难”，同时宣布撒销拜登的行政令，(该项目的重新授权及运营)不再需要拜登。

众议院少数党领袖凯文·麦卡锡表示，“拜登总统停止建造该管道的决定对美国工人和家庭来说是灾难性的。它对全球气候变化的影响微不足道，但它对工人、家庭和能源独立的影响绝对是负面的”。如果没有民主党的共同提案，该法案很难在众议院取得进展，除非它能赢得石油和天然气州民主党人的支持。

外界普遍认为，拜登上台后不会取消一个已经投入巨资并且可以给美国带来巨大利益的在建项目。Keystone XL已于2008年动工，建成后每天将至少83万桶廉价阿尔伯塔焦油砂油输送到德克萨斯州墨西哥湾沿岸的炼油厂。拜登取消该项目势必引起各相关方的强烈反弹。

拜登上台后颁发的行政令被挑战，包括暂停边境墙建设、停止对非法移民遣返、在美国住满8年可获绿卡等等都受抵触，德克萨斯州等多个南部边境州更是公开反对或拒绝执行。德克萨斯州甚至发起独立公投，以示对美国联邦三权系统的失望。

目前由民主党把持的参众两院，不太可能支持作为少数党的共和党废除拜登的的行政令，但美国各个州可以依法拒不执行总统的行政令，原因是美国继承的是英国的法律体系，社区是高度自治的。美国的权力系统是由下(选民)向上逐级授权而构成，总统需要获得大多数选票及各州的授权才有正当权力。

美国全名叫美利坚合众国，就是由具备高度独立性的名个州联合而成的联邦，也可以理解为由多个小国家自愿组成的联邦。因此，联邦总统的合法性是必须获得选民的授权，当美国总统选举出现舞弊现象时，总统的合法性遭受质疑，当然政令不通，国家甚至有分裂和内战的风险。
