在中国神话中第一次谈大洪水，是共工和祝融大战。共工兵败，就一头撞向不周山，谁知不周山是撑天的柱子，经共工一撞便断了，于是半边天塌下来，天上露出大洞，地上一片汪洋，造人的女娲，便炼五色石用来补天，天地才算又奠定了，谨慎善良的人得以存活。
