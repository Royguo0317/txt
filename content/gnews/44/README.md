加拿大魁北克省法语主媒《魁北克日报》（LE JOURNAL DE QUÉBEC）于5月2日刊登题为《灾难性跨省住房危机》的社情新闻。文中对当前魁省自去年初“中共病毒”疫情爆发至今，日趋严峻的住房供给危机再次进行了详细地报道。显而易见，我们早就预料到的疫情次生之祸已开始严重影响着当地的民生和民情，特别是对社会中底层弱势群体的直接碾压则更为其甚，逃无可避！

但世人皆知新冠病毒之险害，而未觉投毒者中共之邪恶，这才是当今世界真正的灾难和悲哀。“中共不灭，病毒难止！”的警世箴言，现在也正被炼狱煎熬，生灵涂炭的印度彻底验证。

全文摘要编译如下：

在魁北克省，从Rimouski到Gatineau，从Sherbrooke甚至是Drummondville到Saint-Jérôme（魁省的各个中小城市名），（中共病毒）COVID-19疫情所导致的住房危机正在深度波及该省的每个角落，并引起了各界特别是房客保护组织与日俱增的担忧和困扰。

为此，魁北克住房委员会和房客权益协会的发言人无奈道：“住房危机着实很严重，并且魁北克省所有地区的房地产价格仍在不停地飞涨，而由此对社会带来的危害性将是永久性的！”

现在的蒙特利尔，所有人都在议论着找到合适住宿房屋的巨大困难性。但这种情况远非只是蒙特利尔这样的大都市所特有。在魁省的其他地区，可租住房源的紧张短缺状况也是空前“灾难性的”。

根据加拿大抵押和住房公司（CMHC）的最新统计数据，魁北克目前主要地区的住房平均空置率约为0.6％。而CMHC认为，要使租房市场保持供需平衡，空置率必须在3％以上。可想而知，现在魁省住房的稀缺性不言而喻。

同时，也正是租房市场的状态是如此偏向供方，而导致了全省范围内住房租金不断急剧飙升的恶劣情况。以蒙特利尔一室一厅的租房价格为例，其2021年的平均价格相较2019年疫情流行之前上涨了近一倍。

许多居民也因高昂的房租价格而长期无法找到合适的住房，过着全家颠沛流离的动荡生活。（全文完）
