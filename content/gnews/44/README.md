**西班牙2021年1月2日电/西喜社—**

据美联社1月2日报道，印度已经在全国范围内测试了其中共病毒疫苗输送系统，因为它准备推出一个接种计划，以阻止中共病毒大流行。

周六的演习包括将数据输入一个在线平台，用于监测疫苗交付，以及测试疫苗的冷藏和运输安排。

这次大规模的演习是在政府指定的专家小组召开会议审查潜在候选疫苗的申请后一天进行的，其中包括由牛津大学和英国制药商阿斯利康开发的前锋Covishield。

政府计划在疫苗接种计划的第一阶段为3亿人接种疫苗，其中包括医疗保健和一线工作人员、警察和军队以及50岁以上有潜在疾病的人。

印度已经确认了超过1030万例中共病毒病例，在世界范围内仅次于美国，超过14.9万人在印度死亡，仅次于美国（34.7万人）和巴西（19.5万人），位居第三。
