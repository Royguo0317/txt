怎样在家中训练我们的胸大肌

基础篇  跪姿俯卧撑

在全球被CCP病毒肆虐的情况下，很多人都被迫待在家里，不能出行进行户外锻炼或者到健身房去锻炼身体。如果一个人长时间待在家里，会引发各种不适，尤其是我们身体肌肉的萎缩。在这这个情况下，我们更加要注意我们的身体状况。

今天小编教练就给大家介绍一种即简单、又实用，老人、大人、小孩都可以做的跪姿俯卧撑动作，该动作可以训练到我们的胸大肌。

跪姿势俯卧撑动作讲解

首先双脚併拢，双膝着地，双臂伸直略宽於肩部。双手放在胸部两侧下方，两个手掌於身体向内形成45度夹角，双脚搭在一起。保持腿部和上半身还有头部处在同一水平面上，不要弓背还有撅屁股。这是该动作的起始图。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/09022223/%E5%9B%BE%E7%89%8744-1.png)

然后以双膝盖为支点弯曲肘关节直到胸部与地面有一个网球大小的距离，如下图。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/09022430/%E5%9B%BE%E7%89%8745-1.png)

然后停顿1到2秒后将自己的身体推到初始位置。注意我们此时此刻要把注意力集中在我们的胸大肌上，不要让我们的手臂肌肉过多的参与进去(做的时候就想像是我们在用胸大肌这推大地往前走的感觉)。

初级标准 1组10个。

中级标准 2组 15个。

高级标准 3组25个。

如果你不能完成标准的跪姿俯卧撑的话，可以减少动作幅度，不要让胸部降到地面一个网球的距离，而是把动作幅度调整到你能舒服完成的距离，同时也可以增加次数。你也要不断训练并逐步增加该动作的幅度，直到可以完成一个标准的跪姿的俯卧撑。
