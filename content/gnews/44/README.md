**西班牙2021年1月12日电/西喜社——**西班牙自上周五卫生部更新和确认401例死亡，已报告超过61,000例新的中共病毒感染，急剧增加，该国的顶级中共病毒专家归咎于圣诞聚会，而不是在英国确定的新病毒变种。

领导西班牙卫生应急协调中心的费尔南多·西蒙说，该国已发现70例新变种。

“在我们的情况下，我们看到的演变，”他说，”是我们在过去几天的行为问题。”

西班牙的14天累计发病率，被传染病专家认为是一个很好的指标，周一的数据飙升至每10万居民435例。

病毒患者让该国14%的医院病床忙得不可开交，这一比例在重症监护病房几乎翻了一番，达到26%。

自疫情开始以来，该国已经记录了210万例感染和至少5.2万例中共病毒的死亡。
