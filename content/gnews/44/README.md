最初“深喉”一词是指水门事件中向新闻界披露尼克松政府内幕的神秘人物，后被证实是时任的联邦调查局副局长。此处借用“深喉”来指位高权重，靠近川普本人，不仅仅是外漏消息，而且是危害国家安全的一些人。他们深耕政坛，被沼泽地培植扶持，他们可以是任何党派、可以是任何背景。

说川普有信仰、有善心可以，说川普是政治小白的要么天真，要么别有用心。任何商业、学术等江湖中玩弄权术，互相捅刀是常态。在一个充斥着人精和人渣的地方，对于一位久经沙场盖楼发财的地产大佬来说不可能会幼稚。然而即使他已做好趟浑水的心理准备，但面临的事实比他原来估计的还要恶劣。

在进入白宫前，川普最好的政治助手也许就是班农先生，入驻白宫后，他身边多了各个利益集团的人，从好的方面讲可以是助力，坏的方面，是川普被这些集团束缚了手脚，耗费精力，很多良策难以出白宫。随后，班农先生不得不离开白宫，全球组织动员，推进平民运动，唤醒被社会主义蒙骗的人。接着川普炒了科米，炒了一打办事不利、名声不佳的所谓高级幕僚……

川普总统上任至今，身边出没诸多“深喉”。对于川普身边的“深喉”，他们做的事有目共睹。他们伤害了川普，辜负了总统的信任，危害了国家安全。对待“深喉”没必要马上翻脸，如果辞退后又来个新的“深喉”不是好办法，川普可以将计就计，先留着他们……

川普从不亲切称呼“深喉”他们，但对彼得纳瓦罗总是称为“My Peter，……”。而往往是纳瓦罗先给媒体吹风，随后总统拍板。川普中了CCP刻意投放的病毒后，护卫他人身安全的不是“深喉”们；川普被弹劾打得很猛时，捍卫他的不是“深喉”们；川普打算脱钩倒是让“深喉”们急疯了。在班农先生、朱利安尼先生……最主要是郭文贵先生引领的爆料革命地协助下，川普明白“深喉”也是个小卒子，关键是“深喉”背后的推手。

川普不选择军管戒严也许是有其它办法，也是要维护美国人对公权力的信任，以防全美大混乱。川普的抉择不是出于私利，而是为了“让美国再次伟大”。如果川普向“深喉”妥协，丢掉自己的信念和信仰，那他对付不了中共。
