## 海外版Switch及游戏卡大量下架 中共在害怕什么

3月30日，有网友发布微博称：海外版Switch游戏机及游戏卡在网购平台被大批量下架。

对于海外版Switch、PS、Xbox游戏机一夜之间下架的现象，有业内人士表示，或许是因为一些海外版游戏涉嫌走私等违法行为：“不少海外版游戏机都是通过分装各部件，以各种各样的方式运到境内，然后再组装出售。”

但记者发现，国行版以及保税仓发货的Switch游戏机仍可正常购买。据称，因为有大店老板因涉嫌走私被抓，这个是否跟抵制西方浪潮有关，有待观察。

针对上述的问题，我们大家可以开脑洞思考一下，中共为何把海外版本突然批量查封。首先，中共这次也是对进出口贸易商的一次割韭菜行径，借机把批量的商户和老板的原始资本直接归为国有，其实就是邪恶中共几大家族的搜刮民膏，压榨百姓的另外一种行径而已。

再次，目前国际形势一边倒的对中共进行了制裁打击，这个也是中共的所谓自残式反击而已。一来可以让国内小粉红们再次看到伟大国的强硬态度，继续对国内进行洗脑教育。

最后，我对海外版switch游戏种类和内容也进行了大致的了解。海外的游戏第一个有对中共病毒和专制统治等等邪恶的做法有类似的描述场景，这个恰恰也是中共害怕的地方，他们担心墙内的人被这些真相唤醒，必须要进行封杀。

因为玩游戏的都是年轻人，年轻人觉悟会让他们更加害怕，前面有香港的抗争，缅甸的战斗，都是年轻人的坚持不懈的追求自由和法制的斗争。如果国内的年轻一代被这些游戏唤醒开智，那将是不可控的后果。但是这个封杀更加激发了年轻人好奇心，这样虽然不能继续玩游戏，但是他们游戏的小团体，小群体，都会对次事件进行讨论热议。本来大家美元感觉的事情，被中共这么一加速，让大家更加想了解真相了。
