［第四十条　售出的产品有下列情形之一的，销售者应当负责修理、更换、退货；给购买产品的消费者造成损失的，销售者应当赔偿损失：

(一)不具备产品应当具备的使用性能而事先未作说明的；

［第五十二条　销售失效、变质的产品的，责令停止销售，没收违法销售的产品，并处违法销售产品货值金额二倍以下的罚款；有违法所得的，并处没收违法所得；情节严重的，吊销营业执照；构成犯罪的，依法追究刑事责任。］

那么啥是敲诈勒索罪呢？据[百度百科——敲诈勒索罪](https://baike.baidu.com/item/%E6%95%B2%E8%AF%88%E5%8B%92%E7%B4%A2%E7%BD%AA)：［敲诈勒索罪是《[中华人民共和国刑法](https://baike.baidu.com/item/%E4%B8%AD%E5%8D%8E%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD%E5%88%91%E6%B3%95/721359)》第二百七十四条规定的罪名，是指以非法占有为目的，对[被害人](https://baike.baidu.com/item/%E8%A2%AB%E5%AE%B3%E4%BA%BA/7025012)使用恐吓、威胁或要挟的方法，非法占用被害人公私财物的行为。敲诈勒索罪是一种重要的侵犯财产罪，其犯罪对象是公私财物。《刑法》规定，敲诈勒索公私财物，数额较大或者多次敲诈勒索的，处三年以下[有期徒刑](https://baike.baidu.com/item/%E6%9C%89%E6%9C%9F%E5%BE%92%E5%88%91/1388509)、[拘役](https://baike.baidu.com/item/%E6%8B%98%E5%BD%B9/584117)或者[管制](https://baike.baidu.com/item/%E7%AE%A1%E5%88%B6/40681)，并处或者单处罚金；数额巨大或者有其他严重情节的，处三年以上十年以下有期徒刑，并处罚金；数额特别巨大或者有其他特别严重情节的，处十年以上有期徒刑，并处罚金。 ］

在了解了相关法律规定之后，我们可以对那位购买过期商品索赔的顾客的行为作出判断，首先对于公民来说法无禁止即可为，他故意采购过期商品然后向商家索赔并无不妥，因为这样的行为并不被法律所禁止，而且在[中华人民共和国产品质量法](https://baike.baidu.com/item/%E4%B8%AD%E5%8D%8E%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD%E4%BA%A7%E5%93%81%E8%B4%A8%E9%87%8F%E6%B3%95/5028597?fr=aladdin)第四十条明文规定的消费者可以索赔的条文之下，这位故意采购过期商品并向销售者索赔的公民所谓涉嫌敲诈勒索则纯属无稽之谈。
