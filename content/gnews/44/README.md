## 《路德社》精选新闻简讯：欧盟议员誓言阻止欧中投资协定落实
` 路德社精选` [轉載自GNews](https://gnews.org/zh-hans/1176165/)

出品：《路德社》 编辑：《路德社》精选新闻专栏组
![]()![](https://www.gnews.org/wp-content/uploads/2021/05/image001-5.png)
据《法广》4月30日报道，一些欧洲议会议员不满中共国早前宣布制裁部分欧洲议会成员，誓言将会阻止欧中投资协定落实。在欧洲议会28日举行的会议上，超过30名议员发言，他们抨击中共的制裁行动，并警告欧中投资协定已被“搁置”。

这份欧中投资协定是在去年底，欧盟与中共国领导人在视讯会晤中宣布原则上达成全面投资协定的。月前，中欧关系因为新疆人权问题恶化，欧盟宣布制裁4名中共国官员，指他们侵犯新疆维吾尔族人权。中共随即反制，宣布制裁10名欧洲议会议员及相关机构等。如今，议会内多数主要政党已正式宣布，如果中共的制裁仍然存在，他们将不会支持欧中投资协定。在中欧互相制裁后，外界对双方能否落实中欧投资协定感到悲观。在此，小编提醒各位看官，早在今年一月份，美国国务卿蓬佩奥就已经宣布，在CCP指导和控制下的中共国犯下了种族灭绝罪。
