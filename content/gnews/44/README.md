大选后近两个月来，宾夕法尼亚州各种关于选票认证错误的消息不断。据DJHJ Media报道，12月28日宾夕法尼亚州众议院公布了在该州认证的总统选举结果的选票误差，竟然有20多万张，这是川普总统和拜登之间选票差距的两倍还多。

共和党州代表拉斯·戴蒙德（Russ Diamond）周一发表报告指出：宾夕法尼亚州的总统竞选结果情况有多糟糕？计入的总选票数比投票的人数多出20多万。([点击此处阅读报告全文](http://www.repdiamond.com/News/18754/Latest-News/PA-Lawmakers-Numbers-Don%E2%80%99t-Add-Up,-Certification-of-Presidential-Results-Premature-and-In-Error))

“哈里斯堡（HARRISBURG）：部分州议员通过大量的数据分析发现，计入的选票数量和2020年的大选中的投票人总数出现了很大的的差异，这样的结果是如何被州务卿凯西 布克瓦（Kathy Boockvar）和州长汤姆 沃尔夫（Tom Wolf）认证的？”这些发现是对宾夕法尼亚州最高法院、州务卿和其他影响选举的人的质疑之外又一个新证据。

根据美国国务院记录的官方县级选举结果与2020年11月3日投票的选民总数的比较表明，据报总投票数为6,962,607，而DoS / SURE系统记录显示，只有6,760,230的选民实际投票。在总共6,962,607张选票中，在总统竞选中总共计6,931,060张选票，包括选票上的所有三名候选人和手写候选人。

计入的选票比投票人数多出202,377张选票，加上总统竞选中31547张多票或少票，结果是惊人的170,830张选票差异，这是两位主要总统候选人在全州范围内的差距的两倍多。2020年11月24日，布克瓦认证了选举结果，沃尔夫发布了总统选举人的确认证书，称拜登比川普多获得80,555张选票。”

“当我们关注差异的细节，引发了有关选举结果违规行为的更多的问题。这些发现使人们质疑SURE系统的准确性，各县之间采用《宾夕法尼亚州选举法》的一致性以及负责我们选举监督的人的能力。”
