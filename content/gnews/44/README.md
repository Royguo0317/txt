**西班牙2021年4月21日电/西喜社——**

人权观察网4月19日（纽约）消息，人权观察在今天发布的一份报告中说，中国政府正在对新疆西北部地区的维吾尔人和其他突厥穆斯林犯下反人类罪。中共及其帮凶们应为实施对维吾尔族人的大规模拘留、酷刑和文化迫害等广泛而系统的政策措施而负责，且需国际方面联合行动起来一同制裁此政策执行责任人，推动问责制，并迫使中共政府改变其政策方向。

这份长达53页的报告：“断其源，断其根——中共对维吾尔族和其他突厥族穆斯林犯下的反人类罪”，是在斯坦福大学法学院“人权与冲突解决”的协助下撰写的，该报告借鉴了中共国的政府文件、人权团体协会、媒体和学者提供的新资料，在国际法律框架内评估中国政府在新疆的行动。报告指出了一系列针对突厥穆斯林的侵犯人权行为，即对该种族人口的广泛和系统的攻击行为：大规模任意拘留、酷刑、强迫、失踪、大规模监视、文化和宗教抹杀、家庭分离、强迫遣送到中共国、强迫劳动以及性暴力和包括绝育手术在内的强迫节育等严重侵犯。

“中共当局系统性地摧毁突厥穆斯林人的生活、宗教、和他们的文化”，人权观察中国主任索菲-理查森说：“中共解释说正在提供’职业培训’和’去激进化’，但这种说辞不能掩盖反人类罪的严峻现实”。

反人类罪在国际法下被认为是最严重的侵犯人权行为之一。中共政府对突厥穆斯林的压迫并不是一个新事件，但在最近几年达到了前所未有的程度。除了大规模的拘留和对伊斯兰教信仰普遍限制外，还有越来越多的证据表明，中共对其实施了强迫劳动、广泛监视和非法将儿童与家人分开等。

斯坦福大学人权和国际司法中心教员贝丝-范-沙克说：“现在越来越多的可靠证据表明中共政府对新疆突厥穆斯林人的政策和做法符合国际刑法规定的反人类罪标准。该政府不会阻止这些罪行，更不会去惩罚那些实施者和责任人，这表明需要相关国家的政府采取协调一致的强有力行动，以加强问责。”

人权观察和斯坦福人权诊所敦促联合国人权理事会通过成立一个调查委员会的决议，负责调查对反人类罪的指控，确定对虐待行为负责的官员，并追究他们的责任。联合国人权事务高级专员也应监测和报告新疆的人权状况，并定期向人权理事会通报。

相关国家的政府应实施针对性制裁，诸如，禁发签证和经济限制等，并对有犯罪行为的当局实施有针对性的个人制裁。他们应在普遍管辖权的概念下，以刑事案件来追究这类行为。而且应该采取贸易限制和其他措施，以结束中共国强迫劳动的现象。

理查森说：”越来越清楚的是，需要全球联合来应对和结束中共国对突厥穆斯林的反人类罪行。中国是一个强大的国家，从而让它为其无情的虐待行为负责也变得额外重要”。
