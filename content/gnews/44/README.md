# **数据：密歇根州分析认为缺席选票“被电脑操纵”, 并显示有数十万张选票**之多

**对密歇根州选民数据的新分析显示，数十万张邮寄选票被标记为可能存在有欺诈行为，导致数据分析师断定存在“实锤证据”指向“电脑算法操控”。**

#### **疯狂的邮寄选票**

该报告还对密歇根州选举官员在邮寄投票方面的行为提出了关注。在申请的3507129张选票中，该报告指出数十万张选票存在诸如“重复选票ID”，“缺少投票地址”和“出生在1921年以前”这样的质量问题。

#### **计算机**干预

在该报告的第五节，”不合理的密歇根州缺席选票调查结果 “还假设，有 “非常实锤的证据表明，密歇根州的缺席投票计数被计算机算法操纵”。

该报告继续写道：“从表面上看，似乎每个选区的制表设备都经过了编程，将一定比例的缺席选票从川普转向了拜登。”

这些指控的前提是，密歇根州各县的缺席投票趋势偏离了全国范围的正常情况，而且“在每个选区，民主党缺席选民的百分比都超过了共和党的”。
