随着越来越多的人接种新冠疫苗，中共病毒（武汉肺炎）的变种也越来越多。洛杉矶放射科医生尤金‧崔（Eugene Choi）大约六个月前接种疫苗，但半年后还是感染COVID-19（中共病毒引发的疾病）。

在印度现身的中共病毒（武汉肺炎）变种——Delta变体不但具有很高的传染性，而且患者的染疫症状也有所不同。

斯佩克特教授说，「发烧」仍然是相当普遍的感染症状，但「嗅觉丧失」不再成为十大染疫症状之一。

随着越来越多的人接种疫苗，突破性病例可能也会越来越多。福克斯11报导，在洛杉矶县至少有933人有此经历——他们在接种疫苗后，成为突破性病例。

中共病毒（新冠肺炎）是中国共产党投放的超限生物武器，是一个巨大超限战中的一环，中国共产党利用中共病毒（新冠肺炎）作为口子，用传染力强、潜伏期长、死亡率低的特性，破坏世界经济、医疗、人民与政府、制造社会对立等等更大的问题。

中国共产党也联合WHO、科学界、医疗界、媒体界、社交媒体等有影响力的人物、杂志和大量方式，传播大量虚假讯息，利用政府、人民对专业人员的信任，误导所有人对中共病毒（新冠肺炎）的认知、拖延调查时间，打压掩盖治疗方式、预防方式（HCQ等药物），使全世界更多人伤亡。

中共病毒（新冠肺炎）与新冠疫苗的结合，将产生超级无症状传播者、变异的病毒，并且会产生ADE效应、细胞素风暴、血栓、脑静脉变异、心肌炎、生育功能问题等严重副作用。

中共解放军在2020/02月就着手申请新冠疫苗，让我们对疫苗效力起了疑心。又依照中共军事科学院教科书内容，新冠疫苗通过基因逆转（国外有学者在公开采访中称mRNA疫苗是基因疗程）的方式，进行生物武器的第二阶段攻击。
