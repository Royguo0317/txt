## 澳大利亚干草种植者准备迎接中共国的打击
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1012707/)

撰稿：Shirley

发稿：MG4
![]()![](https://gnews.org/wp-content/uploads/2021/03/澳洲干草.png)
网络图片

据澳大利亚广播公司报道，澳大利亚对中国1.6亿澳元的干草出口市场可能成为中澳持续贸易紧张局势的最新牺牲品。
因为贸易紧张，导致中国未能续签几十张干草出口许可证。中国是澳大利亚第二大饲料出口市场，干草种植者面临改变本季种植计划的压力。
自上月底以来，澳大利亚有28家向中国出口干草的企业没有续签到期的许可证。澳大利亚120万吨干草出口中，西澳种植的干草占一半，南澳和维多利亚州也是主要干草种植州。
干草出口的主要市场包括日本、韩国和台湾，但由于中国乳牛群的增长，近年来中国已成为澳大利亚饲料的重要买家，每年购买量约为35万吨。
澳大利亚最大的干草出口商Gilmac在三个州经营着5个基地，全澳有150名员工。总经理帕切特 (Munro Patchett) 先生表示，大约半年前向中国海关总署申请续签5年出口许可证。
帕切特先生回避了干草可能是中澳贸易紧张局势中最新陷入的商品的说法。”有三家公司仍然能够出口，因为他们的到期时间是2023年， 没有像牛肉和葡萄酒那样对干草征收关税。”他说。
帕切特先生表示，干草行业希望尽快解决此问题，如果不能，农民将改种莜麦或其他作物。
一些西澳的农民也表示，在即将开始的播种中，会减少干草的种植量，而改种其他谷物或油菜籽。
自2020年，中共国就开始陆续对澳大利亚的产品进行惩罚，包括对煤炭、大麦、铜矿石、木材、牛肉、糖、葡萄酒和龙虾等加征关税或禁止进口，现在轮到干草也要遭受类似的命运，看来中共国准备要放弃来自澳大利亚的一切了。中南坑真的是吃了秤砣，铁了心，准备和西方联盟彻底脱钩了。
