## 民主家常谈：民主就在你身旁
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1039950/)

撰稿：文娜

朋友们，好！今天第一次节目，开篇：讲民主。民：即人。以个体为单位：一人；一个人O二个人称：俩。三个人以上称：众。

一个人，在精卵结合形成胚胎时，上天就授予了他生命。他就有生存的权力。这个权力是人人平等的。没有那个人是多余的，天生该灭。所以坠胎是不对的，没有谁有权剥夺人的生命！中国有句话叫做：天生我材必有用！有一本天书上说：我未将你造在腹中，我已晓得你 耶利米书1:5。这就是，每个人的出生，都不是偶然的。而且，上天给人类生活在地球上，其实也留下了一本说明书。就像你买一台电视机，就跟着一本使用说明书，好叫你正确使用，不出意外。这本人类说明书，其实很多人都听说过，不过上天有祂美好的旨意，祂正向我们走来，在我们这个民主乐园平台，我会开始带大家去了解。现在继续民主话题：一个孩子，成长到法定成年前，有法律监护人：或者父母，或者指定某人，某机构。什么意思？就是在未成年前，你的民主权力弱小，必须有人帮你看着，这是对你的保护。成年了，各国都有自己的法定。到了这日，你的民主权，回到你自己手里。这时，你与任何人，包括监护人，都是平等关系。这样，在法律面前，人人平等了！
![]()![](https://gnews.org/wp-content/uploads/2021/03/16171769061.png)网络截图
在法律面前，人人平等。中国的宪法也是这样说的，但多年的事实显明，实际还是家长制和独裁专制。这个就是无法无天，就是没有民主！国人没有民主意识的具体表现很多，我会从小家的婆媳大战，兄弟反目，妒忌朋友等讲起，再到中共的假骗偷，在这里先给你们说几个耳熟能详的例子，你就会明白，古往今来，中国人的传统和现今的社会制度，是有缺陷的！首先，中国人在小家里，最难处理的关系，要数：婆媳关系。我会讲二个千年迷思的话题：第一个，有一个母亲问儿子：我和你媳妇掉水里，你先救谁？这时儿子当然说先救母亲。媳妇问先生：我和你娘掉水里了，你先救谁？我看了很多答案，比较好的说：我娘会游泳。哈。其实，民主观念健全，人人平等。这时母亲不是上级，媳妇不是下级。去掉依附条件，只看客观环境因素，谁近先救谁！这有什么好为难的呢？为难就是因为没有民主意识！第二个话题。”娶了媳妇忘了娘”。娘养大了儿子，特别是寡母好不容易养大了儿子。不仅是古时，就是现代，儿子结婚了，母亲要睡在儿子和儿媳妇中间，这样的事情也不时能听见。根源就是：没有民主。若是娘能认识到：儿子与媳妇成家，就是离开父母，与媳妇结成一体。与父母家是平等的二个家庭，父母没有对儿辈的控制权。儿子对父母，有的是尊重，和孝敬，没有了服从的义务。这样才叫民主！从古至今，相信没有人没听说过：公说公有理，婆说婆有理；清官难断家常事！为什么？就是因为没有民主！民主就是清官断案的原则，有了人人平等，就再没有断不了的混乱。现在，大家能明白一些，我们传统的缺陷了吗？所以，我们要建立民主意识，让民主常在你我的身旁！

前面讲了民主在每个人的生活中间。有生命了就有自主权，即自己的话语权！也就是所谓的民主。通常讲民主，多加自由二字。那么民主自由好不好呢？其实，民主自由要有度才好！你说自由，我有权自由杀个人，那是我的自由，这明显是不对的。所以必须给自由立个度，这个度就是法律。没有了这个法律，就没有了标准，也没有了限制，民主就变成了某些人放大自己的民主权力，而形成了不民主的专制。所以要真正民主，就要有一个原则。这个原则，我叫祂：爱真理爱人，爱人如己。真理，是做一切”正确事情”的指弓I。爱真理，就是用真理引导我们处理事情。同时爱人，怎么个爱法呢？爱人如己！对待别人如同对待自己。古语说：己所不欲勿施于人！我们简单点，就是：爱人如己。很好记：爱真理爱A,爱人如己。大家都尊从这个原则，人才有岀路，社会才会和谐！你会说，中国也有法律啊！但我现在问你：中国为什么还出这么多问题？

接下来，我讲几个社会问题，大家就会更清楚，中国的社会，是不是需要“民主”，中国人的问题出在什么地方？。首先讲毒奶粉的事件。一瓶毒奶粉，把孩子毁了。带给这个小家庭，再不是快乐，而是望不到头的不知所措。有没有想过，这也和民主有关！首先，这对夫妇，为什么买平宜奶粉呢？李克强讲全国四亿人口人均收入一千，六亿人口人均收入二千。这对妇夫的收入仅够为生，只好省了又省，却不知后果如此严重。一个号称世界第二大经济体，国民的收入却微薄，国家也没有福利制度。这能体现人人平等吗？其次，为什么有毒奶粉呢？还是和钱有关，归根结底，造毒奶粉的人，有一点点爱人如己的心，不那么向钱看，能做这事吗？所以，一个国家机器的败坏，导致同胞们没有爱心没有一点民主意识，追求一切向钱看。

第二件讲杨改兰事件。杨改兰这些底下层的平民，看不到出路，只好杀儿自杀。习领导，不是扶贫了多少年，扶贫资金的被贪污，就不多说了。投诉报案的被收监，被秋后算账的有多少，大家都知道。

第三件事。讲讲奴隶制。奴役制国人都知道，但可能觉得很遥远，古时候才有奴隶制。其实，现在中国就是奴隶制。奴隶的主人是中共。中共，有户口制，有出国限制，有发声限制。领导层有特供，有特权，高高在上。小老百姓卖个菜，有城管。听听爆料革命，要被喝茶。相安无事的战战兢兢，你能够发言说民主，已经不错。更不要想老百姓可以拥有枪枝。这是什么？这就是国家专制，怕人民起来反抗，造反。这就是强权管理，主人说了算，别人只有听的份，人们什么时候有民主自主权了？法律这么多年为国人做了什么保护？我只看到对高层的保护。这不就是妥妥的现代奴隶制，是什么？

所以，毒奶粉事件，杨改兰事件，还有城管的霸道，医院的红包，其实，城管也好，医院也罢，不是单方面就造成这样的景况，首先是因为中共的自私和盗国，上梁不正下梁歪。导致同胞们没有爱心没有一点民主意识，追求一切向钱看。其头从有中共以来，中国人就被带领走入一条与天头与地斗与人斗，其乐无穷的歧途，良心眠灭，不相信头顶三寸有神明，从来不考虑后果，这才是现在中国变成这样的根本原因。中共自己是神，所以反对所有宗教，反对的目的：就是害怕人认识真神，离他而去。常言道，不怕不识货，就怕货比货。如果是真货，怕对比吗？所以，你现在明白中共的真实意图了？

所以，我们是否要回自己从有生命时就自带的民主权力，我们已经成年，每个人都是平等的。只有职务的不同，没有等级的不同。这个就是民主。我们要民主社会！我们要民主，我们要争取。讲到这里，在这个第一讲里面，我归纳总结我所看到的中国人的现象，给国人一点启发。从有天地万物以来，有了人，上天也给了我们生活在地球这个世界上的一本说明书。这说明书，很多人都听说过，祂叫《圣经》。祂是民主国家美国的开国之本，是民主自由的根基。要真民主自由，我们得从从前中共带领的歧途归正！所以，我们要虚心放下成见，欢迎各种信息，首先不能禁言，这个是民主的前提，从而探得正途！今天的节目，希望大家喜欢，更有耳目一新的收获？就是这样，真的东西是没有人不喜欢的！听讲过后，你认为一线曙光照进来？对！老天爷没有瞎眼，老天爷爱全人类，老天爷派来了郭文贵先生。郭先生说了，他不会做总统什么的。这可以说爆料革命，不是要替代中共，做中共第二。所以，现在大家学习擦亮眼睛！尽量的去寻找真理，和学习真理，并运用真理这个武器。只有持守真理，才能最后胜利！

以后每周的讲台，会针对每周时间里发生的事件，作民主概念和圣经真理方面的介绍，让大家更了解认识民主和真理。做个有智慧的人！我爱你们！耶稣爱你们！神爱世人！好。下周见！
