**香港网台D100主持人杰斯因曾在网上发起众筹，去年11月遭国安处以「洗黑钱」及「资助分裂国家」罪名拘捕后获准保释。杰斯2月7日因节目中言论，再被控4项「作出具煽动意图的作为」罪，案件昨日（10日）在西九龙裁判法院提堂。裁判官苏惠德以无充足理由相信被告不会再犯为由，不准杰斯保释，还押至5月10日再讯。**

**以黎智英案判词 拒杰斯保释申请**

控方引用终审法院2月9日颁下对黎智英保释案的判词，根据「港版国安法」42条（2）列明，除非有充足理由相信被告不会再犯，否则不得予以保释。控方又指，警方仍需时调查大量涉及被告的片段、言论以及财务资料，要求押后案件。

港版国安法指定法官、总裁判官苏惠德，拒绝杰斯的保释申请，并将案件押后至5月10日再讯。辩方表示不会于8日内申请保释覆核。

**杰斯4日网上言论被控罪**

关于被告杰斯（原名尹耀升，52岁）涉嫌违反《刑事诉讼程序条例》第10（1）（a），被控4项「作出具煽动意图的行为」罪。

包括：

1. 去年8月8日，即有外国向北京及香港实施制裁翌日，杰斯曾说希望特首林郑月娥「死全家」，又指制裁只是小小惩戒，片段有不少观看人数。

2. 去年8月15日，杰斯表示希望香港人借镜乌克兰革命，属于露骨地暗示他人使用武力达到非法目的。

3. 去年9月5日，杰斯公开支持民主派35+「揽炒十步曲」，并号召他人参与第二日反对「港版国安法」及押后立法会选举的游行，以及于节目内表示希望港人及内蒙古独立为族群，又对共产党作出不当批评，显然仇视国家和港府。

4. 去年10月10日，以负面言词形容五星红旗，并且以不当及扭曲的言词批评国家政权。

此外，指控还包括40条网台片段，指杰斯煽动他人企图推翻共产党，鼓吹他人使用武力例如投掷汽油弹，又指他支持公民抗命、支持香港独立及台湾自决权、有实则行为及组织让犯案者潜逃至台湾等。

**战友观点：**

杰斯案件坐实了「港版国安法」是没有保释的。杰斯为D100 Radio节目〈北京饭店〉、〈三点维权〉、〈上纲上线〉及〈长Q直进〉的主持人，在去年2月发起「千个爸妈台湾助学」计划，资助受社会运动影响的学生到台湾就学。

国安法生效，言论进一步收紧，政治气候高压，网台节目主持因节目言论及内容，被控以作出具煽动意图的作为。由去年至今，陆续有评论人如萧若元、桑普等离开香港，身处外地畅所欲言；但亦有人仍然留港，顶着压力针砭时弊，杰斯是其中之一。

香港时事评论员曾志豪坦言：以往时事评论有机会触犯的罪行如诽谤罪，「可能还有条线，我不人身攻击，讲事实就没事。」国安法出现后，和以往最大的分别，是没有人知道红线在哪里。对他们来说，简直就是一步一惊心，随时踩到国安法红线。
