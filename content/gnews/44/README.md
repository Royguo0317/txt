布赖特巴特新闻网 (Breitbart News) 6月17日报道，印度与中共军队在两国有争议的喜玛拉雅边界发生致命冲突一周年之际，印度国防部长拉杰纳特·辛格警告称，印度“有能力回应侵略”。

印度国防部长辛格6月17日在启动印度边境公路新系统时对记者说，“我们寻求世界和平(但是)如果有人向我们显示出侵略性的态度，我们将做出回应。”

印度国防部长提到，2020年6月15日至16日晚，中印两国边防部队在喜马拉雅山脉西部拉达克北部的加尔万山谷发生了致命的边境冲突。这场混战造成20名印度士兵和大约40名中共解放军(PLA)士兵死亡，尽管北京从未正式证实超过两名的解放军在冲突中死亡。士兵们徒手搏斗，传统的印中军事规范禁止士兵在没有标记的边界上开枪。这次冲突是印度和中共国45年来最严重的边境冲突。

加尔万山谷的冲突引发了印度和中共国之间持续的对峙，两国在对峙后的几周和几个月里都在喜马拉雅边界部署了更多的军队和军事装备。

印度国防部长辛格6月17日在介绍印度最新的边境公路系统时表示，“印度边境公路组织(BRO)尽管面临挑战，要有在崎岖多山的地形中建设世界级公路的能力，凸显了阿特玛尼尔巴巴拉特(Atma Nirbhar Bharat)的理念。”Atma Nirbhar Bharat在印地语中的意思是“自力更生的印度”，指的是印度总理纳伦德拉·莫迪发起的一项倡议，旨在帮助新德里减少对外国供应链的依赖，特别是减少对以中共国为基地的制造业中心的依赖。辛格说，新的BRO项目是通过印度的“东进政策”实现的，这也是莫迪的设想。
