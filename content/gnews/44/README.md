## 从德国电信看中欧投资协议
` 康州盘古喜马拉雅农场` [轉載自GNews](https://gnews.org/zh-hans/752643/)

作者：康州盘古农场-残夜听雨声
审核：康州盘古农场-V

本周德国杂志《*Wirtschafts Woche*》公布了一份重磅报告，其中揭示了默克尔为何不惜一切代价要推动中欧投资协议的达成：

中共愿意为德国电信提供手机的准入牌照。而5年后，德国电信甚至能在中共进行通讯网络的基础建设，而作为回报，德国的电信市场将向中共彻底打开！

中欧投资协议由习近平、冯德莱恩、默克尔和法国总统埃马纽埃尔·马克龙在一次视频会议中敲定，但至今尚未获得欧洲议会通过，在中欧投资协议条约中显示中共将放宽在华运营欧洲企业的限制，但要求与中方合作伙伴由合资的方式运营，并且要与中共分享敏感技术。而欧洲需要付出的代价是允许中共国的企业加入欧洲国家的基础能源建设。该协议的主推政客表示该协议将为了欧洲企业带来一次新的前所未有的机会，但随着这份重磅报告的显露，更多深层的利益勾兑显露出来，

德国联邦政府间接和直接持有德国电信将近31.9%的股份，可以说德国联邦政府拥有德国电信的最大话语权以及是其最大的利益获得者。2019年在全球抵制华为的背景下，德国电信依旧与华为加强了战略合作计划，并且在2019年中期达成了合作协议，该协议称华为将采取措施避免美国制裁而导致的供应链中断，并承担因此造成的损失和延误的成本，

而根据strand报告称，在德国本土市场，德国电信将依靠华为提供65%的基站和天线。可以说在整个欧洲战场，德国电信是华为最有力的支持者之一，而毫无疑问，德国电信的背后就是德国政府。尽管华为的中共PLA背景在国家安全层面已经昭然若揭，但德国政府依旧要将华为5G引入德国的基础通讯设施，置国家安全于不顾，不得不让人深思默克尔与中共有多少的利益勾兑。

而如今中欧投资协议的重磅报告又显示出默克尔已经完全将德国的国家安全拱手相让中共。在21世纪，通讯和能源是一个国家安全的核心要素，一旦中共的企业获得准入德国通讯和能源市场的机会，短期内德国的企业可能会获得低廉的生产成本以及更加广阔的市场，但随之付出的代价，是整个德国都要沦陷于中共的红色爪牙之中，这不仅让人想起二战时忙于追逐利益的犹太人以及德国纳粹。如今的德国成为了追逐利益的犹太人，而中共的邪恶比起纳粹有过之而无不及。最终的结果一定是新的血腥轮回。

而对于默克尔而言，不知她的后悔多一点，还是绝望多一点呢？

当笔者看到该协议条约时，不由又联想到20年前中共加入世界贸易组织时的协定，当时中共依旧承诺了一大堆的协议：市场开放、知识产权保护、市场透明、人权保护。但是一条都没有兑现，最后反而滋生了一带一路，3F干掉美国计划，以及生化病毒战争等等计划的实施。

人类的天真以及对利益的追逐最终付出了代价。如今这个中欧协议又是代表着欧洲政客再次打算陶醉于中共编制的谎言以及承诺的利益。但是人类还能承受再次相信中共所付出的代价吗？中共引起的三次世界大战已经开始！当德国纳粹肆虐欧洲时欧洲人沉默，最后几十年的一蹶不振。而如今中共再次引发了三战，假如再次为了利益沉默，最终付出的代价可能是整个人类的被奴役，人类还有退路吗？灭共才有生路！
