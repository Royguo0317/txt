## 【香港专题】得奖节目【新闻刺针】被灭 之 魔爪为何「拔刺」（二）
` GM42` [轉載自GNews](https://gnews.org/zh-hans/615786/)

整撰：文燕

覆核：文粤

【香港有线宽频】12月1日以疫情严峻，以节省资源为由裁减旗下香港有线电视台新闻各部门约40名员工。专门负责调查报道的《新闻刺针》全组编采人员被「拔刺」，引发【有线电视中国组】「总辞」。中央在加速换血，《新闻刺针》裁员更标志着「第四权」「一个时代的终结」。

【有线宽频】发表声明，指今次人事调动涉及约100名员工，包括职务调动及离职。声明又指，在新冠肺炎疫情下，公司以「开源节流」、「觅求新机遇」和「控制成本」的方式维持竞争力，因此需要全面检视及调整各部门架构及人力资源。据了解，被解雇包括《新闻刺针》全体成员、有线中国组助理采访主任黄丽萍、至少两名港闻记者、剪片部主管、多名摄影师共40人。

遭解雇的有线中国组助理采访主任黄丽萍对香港媒体表示，中国组人手不足，解雇任何人都难以运作，公司仍削减人手令员工忍无可忍。她表示，有线新闻部员工薪水非常低，一早已减去奖金和安排员工放无薪假，但仍然坚守岗位，在疫情期间无论多危险也留守武汉。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/04033742/image001-1.png)《新闻刺针》记者收拾物品离开有线大楼。
有媒体翻查资料，【有线宽频】集团旗下多家公司在两期保就业计划均获批补贴。当中香港有线新闻有限公司获批约2,124万，承诺受薪人数373人；有线财经媒体有限公司获批约207万，承诺受薪人数39人；i-CABLE Network Operations Limited亦获794万，承诺受薪人数152人；香港有线新闻速递有限公司获约111万，承诺受薪人数19人。笔者粗略测算，有线宽频集团属下公司，两期保就业获逾3,000万元补贴。因此，有线新闻部声称公司在新冠肺炎疫情下需要「开源节流」、「控制成本」的说法不成立，就算是也不致于急迫大量裁员。

**裁员带有政治目的**

香港记者协会表示高度关注此次裁员事件，并指出《新闻刺针》过去一年时有针对警方或政权的报道，不难令人联想是以节流为名，实际目的却是减少或不作敏感的报道。有媒体认为有线电视这次的裁员有两点不寻常的现象，而这些不寻常的解雇员工方式，令人不得不相信这次裁员是有政治目的。

1）针对有能力、有贡献的一班

例如【新闻刺针】在行内都是有口皆碑，多年来为有线电视带来国际和本地奖项无数。但这次【新闻刺针】 全组被灭，为什么？记者、编辑、工程部，被公认干得出色的全都被裁退。正常的裁员都是裁退庸员、高薪的，但这次裁员特点非常诡异，令人不得不联想到与政治有关。

2）「4人帮」似乎带有政治任务

今年8月起，有线新闻发生多次高层人事变动。新闻部主管突然被切换，换来4位来自无线电视『TVB』 和前亚洲电视的高层入主有线电视。原新闻部负责人丶有线电视执行董事冯德雄转任顾问，由谢燕娜丶陈兴昌丶李臻及许方辉4人共同主管新闻部，外界质疑新的主管们立场倾向建制，「较听从政府的话 」。

据内部消息称，这4人平时与同事零沟通，却私底下开「闭门会议」。内部消息指，事前并无风声，突然在昨天早晨开编采会，「4人帮 」中来自亚洲电视的女高层谢燕娜突然宣布裁员100人，当中包括新闻部40人。这次的旋风式的裁员，事前没有经过商量，而被裁员工名单也是空降的，各部门主管都非常愕然。当同事追问裁员原因时，「4人帮」表示不知情，称是由人事部决定，令人难以置信。

**坚守新闻岗位 为港人知情权奋斗**

为什么要对付【新闻刺针】？因为当权者不允许新闻界把事实公诸于世，不让港人知道真相。资深记者、媒体人李惠玲指出，这次【新闻刺针】事件的发生，不单止属于新闻界，也属于全港市民，因为他们要夺取新闻界的采访和报道权，骨子里就是要夺取香港人的知情权。他们不让大众知道他们不想你知道的真相，他只想你知道那些歌功颂德的事，使香港新闻通通变成【中央电视台】。

有线电视这几年一直以经济不好为理由对员工减薪、冻薪或削减福利，现在是每月放两天无薪假。这些同事薪水虽微薄，但因为共同的新闻理想，专业的角度，无论多艰辛都坚持下去。武汉爆发中共病毒期间，中国版的记者，深入武汉灾区第一线采访，为了向市民报道真相，冒着生命危险，以生命换取第一手资料。如果没有足够的勇气和使命感，绝对做不到。而当时，有线新闻部的记者做到了。
