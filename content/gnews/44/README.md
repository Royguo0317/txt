## 班农《战斗室》爆宾州选举舞弊听证会引发联合决议

![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/28170045/The-Washington-Pundit.jpg)图片来源：The Washington Pundit
据DJHJ Media 11月27日报道，史蒂夫·班农在收视率高、信息量大的《战斗室（War Room）》中采访了共和党州参议员道格·马斯特里亚诺（Doug Mastriano)。他们在节目中爆料，宾州政府将努力达成一项联合决议，取消宾夕法尼亚州州务卿凯西·博克瓦尔(Kathy Boockvar) 在任命总统选举人方面的权力，对2020年总统选举结果进行认证，马斯特里亚诺认为需要根据选举舞弊的指控进行调查。

马斯特里亚诺在周三（11月25日）组织了一场听证会，让川普的法律团队提出他们的选举舞弊证据，这些证据故意被大多数民主党同盟媒体所忽略，但《战斗室》和其它独立媒体，包括《门户专家（Gateway Pundit）》报道了这次采访。根据马斯特里亚诺和班农的说法，这些证据引发了看到听证会的人的广泛担忧，认为不应该把投票认证给民主党人乔·拜登。川普律师鲁迪·朱利安尼和珍娜·埃利斯亲自出席了听证会。川普通过埃利斯的手机在听证会上发表了约十分钟的讲话。几位证人出示了欺诈的指控和证据。

《战斗室》的联合主持人拉希姆·卡萨姆在推特上发文：“马斯特里亚诺参议员在直播中告诉《战斗室》 如果有必要的话，会一直打到最高法院。还透露他打算通过一项联合决议案，将宾夕法尼亚州选举人的选择权交还给立法机构”。马斯特里亚诺说，决议的动力源于参议院共和党人周三在葛底斯堡举行的“爆炸性听证会” ，证据显示，大规模的欺诈行为使民主党提名的乔·拜登的选票超过了川普总统。

马斯特里亚诺说：“听证会摆出了证人提出的事实，媒体应该关注，这是一个令人信服的案子；人们非常震惊，我们有那么多的证据，而且还有更多的证据。我们现在要通过联合决议，夺回我们的宪法权力”。

“1938年，议员们决定把选择代表的工作交给国务卿。因此，谁赢得了州里的民选票，如果选举是公平的，代表权自然就会归他所有，也是合理的。如果不公平，那为什么要把这个权力交给那些作弊的人。绝对不行！所以感谢上帝，美国有联邦宪法，说我们要行使宪法权利和权力，所以我们要解决众议院的问题，希望今天就能解决。”

另一位观众写道：“该决议将有多个部分，包括但不限于收回宾州立法机构根据美国宪法第二条规定的选举人席位的权力。立即安排宾州立法机构下届会议的席位并于12月1日开幕。马斯特里诺在《战斗室》上说他们有投票权，他们正在努力从现在到周一结束之间要完成这项联合决议。 ”

这次历史性的大规模选举舞弊暴露了美国被中共的深度渗透，包括媒体界、科学界、司法界等，但是也令人学习到了美国法制的健全、三权分立的重要意义。拥有健全严谨的系统，虽然会有被侵蚀的情况，但可以很快自我修复，重塑再次伟大的美国。
