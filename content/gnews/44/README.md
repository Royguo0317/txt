## 中共“强烈反对”美国敦促台湾参加世卫会议提议
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1202531/)

#### 编译：坐看云起时
![]()![](https://gnews.org/wp-content/uploads/2021/05/taiwan-2132642_1920.jpg)
5月7日，美国国务卿布林肯发表声明呼吁世卫组织邀请台湾以观察员身份参与世卫大会。此前，七国集团（G7）外长会也发表联合公报，表示支持台湾有意义地参加世界卫生组织有关论坛和世界卫生大会。

以往一踩就跳的中共外交部近来反应迟钝。5月10日才于记者招待会上表示了老生常谈的”强烈不满和坚决反对”。称美国国务卿布林肯的有关声明严重违背了一中原则和中美三个联合公报规定，严重干涉中国内政。

《国会山报》（The Hill)5月10日文章报道，中共外交部发言人周一表示，美国国务卿布林肯的声明 “不利于未来全球应对公共卫生危机”，并 “严重违反了’一中原则'”，称坚决谴责并拒绝这种做法。

近几个月来，台湾已经成为美中关系的一个主要问题，因为西方国家和台湾政府都警告说，尽管台湾是一个事实上的独立存在，但中共似乎越来越有可能对这个它声称拥有主权的岛国进行军事控制。

中共政府一再警告美国不要干涉台湾或对台湾政府表示支持，而美国国务院则批评中共试图恐吓台湾和该地区的其他 “邻国”。

美国国务院发言人在3月告诉国会， 美国将继续向中共正在进行的试图恐吓包括台湾在内的该地区邻国的令人不安的模式表达严重关切。”我们对台湾的支持是坚如磐石的。我们将与朋友和盟友站在一起，推进我们共同的繁荣、安全和价值观，以及一个自由和开放的印度洋-太平洋地区-这包括深化我们与民主台湾的非官方关系。”
