**西班牙2021年1月12日电/西喜社——**美国卫生官员建立了一个网站，帮助人们找到在哪里可以获得中共病毒的抗体药物，这些药物如果在感染发生后足够早地使用，可能有助于预防严重疾病和住院。

其中两种药物—来自礼来公司和Regeneron制药公司—已获准在美国紧急使用，但繁文缛节、医护人员短缺和其他问题使许多病人和医生无法获得这些药物。

卫生和人类服务部官员周一表示，截至上周，分发给各州和地方卫生点的641,000多个治疗疗程中，只有25%的疗程被使用。

一个大问题是找到一个有药物的地方。该网站包括一个工具，人们可以在50英里内找到管理治疗的地点。医生将确定患者是否符合标准。治疗必须在第一次出现症状的10天内开始。

这些药物是免费的，尽管人们可能会被收取静脉输液的费用，这是一种一次性治疗，大约需要一个小时。
