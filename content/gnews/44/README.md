## **美国联邦通信委员会对在美运营的两家中共国电信公司采取行动**

华盛顿（路透社）——周三，联邦通信委员会说已经开始行动以取消中国联通（美洲）、太平洋网络及其全资子公司信通电话向美国提供通信服务的授权。

[!\[\]()!\[\](https://spark.adobe.com/page/9u0qCuUkeTI2f/images/e8fd217a-8bb7-4ea1-bc55-19ae8a651e92.png?asset_id=f82e5edc-b0aa-4b34-b159-3ea6cf49d927&amp;img_etag=%22030a926d0205b8f6c8703a85d99a0e8f%22&amp;size=1024)](https://spark.adobe.com/page/9u0qCuUkeTI2f/images/e8fd217a-8bb7-4ea1-bc55-19ae8a651e92.png?asset_id=f82e5edc-b0aa-4b34-b159-3ea6cf49d927&amp;img_etag=%22030a926d0205b8f6c8703a85d99a0e8f%22&amp;size=1024)文件图片：在美国华盛顿特区的联邦通信委员会总部，可以看到（张贴的）警示标志，2020年8月29日。路透社/安德鲁·凯利

（2020年）4月，美国通信监管机构发布命令警告称，可能吊销3家中共国国有电信公司的运营牌照——其中2家周三已经提到过，另一家是中国电信。中国联通（美洲）有一个为期20年的旧许可，可以提供美国国际通信服务。

中共国的电信公司没有立即回应置评请求。

联邦通信委员会周三说，这些公司在现阶段未能消除人们对它们在美国授权的严重担忧。

监管机构在去年12月份启动了类似的程序，开始撤销对中国电信的授权。中国电信是中共国最大的通信公司，拥有美国运营许可将近20年。

联邦通信委员会专员杰弗里·史塔克斯（Geoffrey Starks）指出，许多中共国电信公司“还拥有在美国境内运行的数据中心。”他说联邦通信委员会目前缺少权限来“处理这个潜在的国家安全威胁。”

2019年5月，联邦通信委员会经投票一致否决了中国移动——另一家中共国国有通信公司——向美国提供服务的权利，理由是中共国政府可能会利用这一许可对美国政府进行间谍活动。

联邦通信委员会对中共国的电信公司持强硬立场。

周五，联邦通信委员会依据2019年的一条旨在保护美国通信网络的法律，认定5家中共国公司对国家安全构成了威胁。

联邦通信委员会点名的公司：华为技术有限公司、中兴通讯公司、海能达通讯公司、杭州海康威视数字技术有限公司和大华技术有限公司。

（去年）4月，联邦通信委员会批准了Alphabet的子公司谷歌使用部分美国-亚洲海底电缆的请求，但不能连接香港。

上周，脸书撤回了一项使用电缆在美国和香港之间传输互联网流量的申请，理由是“来自美国政府的对通信直连的持续关切”。

（去年）9月，脸书、亚马逊和中国移动撤销了他们连接旧金山和香港——海湾高速电缆系统的一部分——的申请。
