## 赞美新疆的“法国记者”原来是中共央视前主播
` JPHIMALAYA` [轉載自GNews](https://gnews.org/zh-hans/1060610/)

【日本东京方舟农场】 作者：青衣   校对：文小律

中共国环球电视网(CGTN) 3月28日刊登了一篇署名为“波孟” (Laurène Beaumond)的法国独立记者的文章，题目为《我的新疆：停止假新闻专横》，谴责外界对中共国新疆政策的指控。文章的观点及内容与中共国官方的宣传完全一致，中共国据此大做文章，在全球掀起舆论宣传战，企图为其在新疆的残暴镇压行为洗白。

然而这位所谓的“法国独立记者”日前被质疑并不存在。《法国世界报》（ Le Monde ）上周揭露该文的作者——“独立记者波孟”并不存在。该报称，已向发放记者证的法国职业记者证委员会数据库查证，发现并没有这个“曾在巴黎多个报社工作”的记者的资料。此外“波孟”的社交媒体账号也是今年3月份刚刚注册，账号头像是来自网络图片。

此事在西方引起一片哗然。《费加洛报》（Le Figaro）记者根据线索对此进行追踪，并于4月3日刊登报道称，“波孟”确有其人，并已对她进行了独家专访。

据《费加洛报》的报道称，该报近日找到了文章作者本人，她是央视法语台前主播。

报道中指出，这位一度被质疑为机器人的“记者”确实存在，是住在法国西部萨特省（Sarthe）的一位40 岁出头的法国女性。她在电话专访中指责《法国世界报》怀疑她存在的真实性，不过她也同时暗示CGTN编造了她“法国独立记者”的身份，以帮助传播中共的有关论调。《费加洛报》报道分析，引用西方人士的所谓“个人见闻”，是中共国新的宣传手法。

这位拒绝透露真实身份的撰稿人表示：“那篇文章从头到尾都是我写的，然后向CGTN投稿。 我很担心我的安全和对我迎面而来的攻击，以及说我不存在的言论，让我很震惊。 ”她向《费加洛报》坦言，她曾经是中共国中央电视台（CCTV）法语台主播，2011到2017年住在北京。 她表示使用化名是为了保护身份。

报道指出，这位化名“波孟”的前央视主播强调，她的确如同CGTN所写，曾在巴黎第四大学修读艺术史与新闻，并且也曾在一些报社“做过实习”，但她不愿透露媒体名称。“波孟”表示，这篇引发争议的文章是自己主动提笔写的，并在3月24日投稿，最后于3月29日刊出。 《费加洛报》对此评论称：很凑巧，刚好就是欧盟和美国等多个西方国家对中共国涉新疆镇压官员发出制裁的这一天。

据称，这位女士当时的丈夫来自乌鲁木齐。 她承认并没有进行新闻调查，只是描写自己的“所见所闻”。然而这个“所见所闻”就被CGTN拿来当作回应西方指控的报道，并且波孟于2011至2016间，前往新疆的数次“观光行程”，是在习近平2017年以打击伊斯兰恐怖主义与分离主义为名，下令展开集中营政策之前。

《费加洛报》评论指：“她的文章展示了中共政权在全球推广宣传战役中的新影响力策略。透过这些自称是西方‘专业记者’或‘资深研究员’的‘见证’，企图抵抗国际对新疆教化政策的指控，而这个被美国国务院谴责为‘种族灭绝’的政策，已导致上百万穆斯林遭监禁。 ”

面对这段时间以来，西方对中共在新疆犯下“反人类罪”和“种族灭绝罪”所采取的接踵而来的历史性制裁，中共国官媒开始展开大量宣传，透过突然冒出来的数量暴增的所谓“西方人士”、西方“独立记者”、西方“评论员”，以这些人支持中共国的所谓“研究”和“见证”，对抗西方媒体对其人权批评的所谓“假新闻”，企图混淆视听，颠倒黑白。

中共国外长王毅之前曾在新闻发布会上引用“法国知名作家”维瓦斯（Maxime Vivas）的著作《维吾尔：以终结假新闻》，作为新疆繁荣且稳定的证据。 然而事实上，退休的工程学家维瓦斯都曾表示自己“不是中国专家”。那么，他的文章又何具代表性？又有何说服力？

如今，CGTN“特约作家”“波孟”深陷媒体风暴中，她向《费加洛报》坦承“其立场站不住脚”，并要求匿名，怕被人视为“被中（共）国收买的可怜女孩”。因担心家人安危，“波孟”称将不再使用这个化名发表文章，但并不表示将中止与中共国官媒的合作关系。

显然，这个自称“可怜女孩”的记者骗子毫无悔改之意，并且还舍不得中共撒给她的一把狗粮，可耻又可悲。中共对新疆所犯下的“反人类罪”和“种族灭绝罪”已成为铁的事实，为魔鬼洗地唱赞歌，将成为她一生的耻辱，最终毁掉的是她的声誉和未来。
