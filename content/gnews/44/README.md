近日，一篇“奇闻”在墙内朋友圈传播，内容尺度之大，点评之犀利，实属近10年来罕见。

该文作者按时间线简述了各国目前对中共国的一系列从政治，军事，政策等方面的围追堵截，打脸华为所谓的“占领全球市场”真相，里面引用的诸多新闻内容，多是墙内新闻媒体没有报道过，或者只是隐晦提到的。

行文一改过去自媒体公众号小心翼翼，遮遮掩掩的做派，打出“共产党”、“内外交困”、“众叛亲离”等词语。同时在结束语中，也问了个非常好的问题，留给读者思索——中共国不是宣传自己地大物博，遭人嫉妒吗？那事实是，其他同样物产丰富的国家，为什么都没有遭受过这样的“围堵”？一个人同时被周围所有人讨厌，难道不是这个人该反省下自己的所作所为吗？

当然，我们也应该清醒地看到，这篇文章能在朋友圈发表，事件背后完全有可能是江派或者其他势力的推动，试图给“习神”的“加速”行为定罪，同时洗白自己的派系，否则文中为何对中共整个集团过去的罪恶只字不提？即便如此，我们也乐意看到墙内派系斗争的白热化，毕竟他们对自己的揭露，要比普通人来得更狠，更有力量。
