## 亚洲各国面对中共军事霸权何去何从？

《日经亚洲》（Nikkei Asian）2月14日刊登了一篇名为《中共已经赢得了亚洲的军事竞赛》的文章，作者威廉·布拉顿（William Bratton）曾担任汇丰银行亚太股票研究主管，著有《中国崛起，亚洲衰落》一书， 他在文中呼吁亚洲各国要增加军事建设以应对中共威胁。

文章指出，近年来，中共军费开支飙升引发了亚洲军备竞赛，很多亚洲国家都增加了国防预算和新装备计划。但亚洲地缘政治的一个奇怪之处在于，中共的邻国虽然害怕中共不断增长的硬实力，但它们几乎没有紧迫感，中共军队变得越发先进精悍，许多邻国的军队却停滞不前。如果说这是一场地区性军备竞赛，参与者很少，以至于中共国在发令枪响之前就赢得了竞赛。

中共的地区军事领导地位是其在过去30年中取得经济巨大进步的结果。中共不断增长的国防预算使得中共军队在各个方面都进行了现代化改造，中共对新技术和装备的投入令人印象深刻。这些投资使得中共的区域军事力量提升显著，这一点在中共海军及其新航母、驱逐舰、攻击舰和潜艇上尤其明显。

与解放军的进步形成对比的是，亚洲各地的许多军队都缺乏资金，面临着相对落后的局面。即时各地区的国防预算正在扩大，但与其国内生产总值(GDP)相比，它们往往规模较小，而且增长速度低于基础经济体。事实上，在过去10年里，中共许多邻国的军费开支占GDP的比例都有所下降。

这一趋势在东南亚最为明显，那里的大多数国家似乎都根深蒂固地不情愿去建立可靠的防御能力。只有新加坡在致力维持一支先进、装备全面的军队，相比之下，印尼、马来西亚和菲律宾的国防开支都只占GDP的1%左右，他们的军队长期资金不足，装备老化且常常无法使用。

同样的情况也发生在台湾。1999年至2019年期间，台湾持续削减国防开支占GDP的比例，从3%以上降至2%以下，尽管来自台湾海峡对岸的威胁越来越大。面对中共军事威胁，台湾在2020年和2021年增加了国防预算(部分是为了从美国购买新设备)，但目前尚不清楚，相对于面临的威胁这是否足够，或者能否抵消此前投资不足的影响。

日本也没有大幅增加国防开支占GDP的比重。它的国防预算可能位列亚洲第三，仅次于中共和印度，尽管自2013年以来国防预算有所增加，该国的军事开支仍仅占GDP的1%，这一比例远低于新加坡、韩国和澳大利亚。因此，考虑到日本的规模、地区地位和该国面临的风险，日本军队的能力不足。

对于这种军事上的忽视，有三种可能的解释。

首先，与其他国内优先事项相比，许多亚洲国家政府根本不认为国防支出重要。例如，随着人口老龄化，日本、韩国和台湾都面临着快速增长的医疗费用。同样，许多东南亚国家在政府预算面临压力时，往往会优先更紧迫的考虑。

第二，亚洲国家可能已经认定，鉴于该地区的军事力量平衡已向中共转移，任何回应都将是徒劳的。这个超级大国的军事开支已经超过了亚洲其他国家的总和。中共的军费支出分别是印度的4倍和日本的5倍，是东南亚国家总军费的6倍多。即使加大支出，如此量级的差异可能也无法解决。

第三种可能的解释是，该区域的大部分地区国家仍然依赖美国的军事保护，但这是一个有风险的策略。这里存在一定程度的不确定性。寻求外部支持以弥补其军事缺陷的亚洲国家可能会发现，如果美国或任何其他亚洲以外的国家认为这种伙伴关系的成本和风险太高，这种外部支持就会变得脆弱。

如果中共的邻国将其军事进步视为一个长期问题，那么他们现在就需要采取行动。他们必须在国防上加大投入，并且建立集体的地区安全结构来支持他们各自的努力。如果不这样做，就会发出一个明确的信号，即他们对该地区不断变化的地缘政治动态缺乏紧迫性。如果它们都缺少应对中共日益增长的威胁的紧迫感，那么世界其他国家为什么要这样做呢?如果亚洲国家愿意接受中共的军事霸权，那么世界上更遥远的其他大国也可以接受。
