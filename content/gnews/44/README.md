## 川普提名布莱恩·布鲁克斯为货币监理局主管
` 秘密翻译组G-Translators` [轉載自GNews](https://gnews.org/zh-hans/602287/)

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/29103354/capture-157.jpg)图片来源：Bloomberg
据Newsmax网11月27日报道，川普总统正式提名布莱恩·布鲁克斯（Brian Brooks） 接管关键银行监管机构，即货币监理局局长一职。布鲁克斯自五月起担任该局代理局长。白宫在周五的简报中称，提名已送交参议院，参议院银行委员会主席迈克·克拉波（Mike Crapo）也已表示，参议院将会对布鲁克斯这一任期为五年的货监局局长提名举行确认听证会。

报道称，对于那些一直抨击川普总统拒绝接受大选结果的民主党立法委员来说，川普总统在本届政府末期安插这样一个关键行业监管角色，此举高度反常，当然遭到这些民主党人的炮轰。假如乔·拜登（Joe Biden）能入主白宫，在银行委员会主席、爱达荷州共和党人克拉波迅速确认布鲁克斯的任职后，拜登将被迫在1月20日就职典礼后决定是否立即解雇布鲁克斯以拨出这个眼中钉。虽然法律允许拜登罢免布鲁克斯，但此项权力从未有人用过。假使拜登在参议院不能获准行使该权力，他的代理财政部长也能立刻开除布鲁克斯。

美国银行的主要法规都是由美联储、联邦存款保险公司和货币监理局共同制定和批准。因为其它两个机构还将继续由川普总统任命的人主管一段时间，所以货币监理局被认为是拜登实施其银行计划的关键部门。布鲁克斯此刻正紧锣密鼓地完成一项法规的出台，即禁止银行拒绝向具有政治风险的企业提供贷款，例如与石油勘探、枪支和私人监狱有关的借款人等。该提案的公众意见征询截止日是1月4日，如果川普总统的法律诉讼战未成功，布鲁克斯就只有几天的时间在拜登就职典礼前来最终确定此项法规。
