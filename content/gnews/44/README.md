法律和宪法可以使川普在（看似）未赢得选票的情况下继续任职。

总统选举制度很复杂，因为本来这个设计不是直接“民主”的。《宪法》要求各州选出“选举人”，然后选举人再投票选举总统。逐渐地各州都已通过法律，以确保该州总统选举的民众投票决定“选举人”，但这只是法律，而非宪法义务。

现在想像一下选举周期间，川普在11月3日选举夜领先。但在之后的几天里，乔·拜登（看似）领先，同时共和党人对成千上万张邮寄选票提出异议。民主党人提起反诉。考虑到这种混乱，立法机构决定自己认命选举人。

这是九个摇摆州情况：8个州的立法是由共和党代表。如果这其中一个或多个州认为选票混乱不合规定而无效，他们可能会任命自己的”选举人”，那就是共和党赢。在某些州，民主党人可以提出异议并提起诉讼。民主党州长或州务卿可将自己的“选举人”名单推荐到国会，这会加剧混乱，但这很可能也是共和党计划的一部分，因为你会看到，国会于(2021年)1月6日召集选举人投票时，某些选举人的合法性将被挑战。

另一个可能是，共和党国会议员可以决定有争议的州的选票不应被简单地计算在内。例如，如果密西根州的选票无效，这将确保两个候选人都不会获得270张选举人票。那时，《宪法》明确指出：众议院投票决定总统选举，但每个州只有一票。如果目前的数字保持不变，那就是共和党26州占多数、民主党23州、还有一州持平。因此，选举结果将是川普连任。
