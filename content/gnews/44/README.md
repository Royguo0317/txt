首份针对新疆种族灭绝指控的独立报告来自于本周二华盛顿特区纽兰战略与政策研究所。美国有线电视新闻网(CNN)独家看到了这份报告的预览版。

报告称，中共政府“对正在进行的针对维吾尔族人的种族灭绝负有国家责任，这违反了(联合国)种族灭绝公约”。这是第一次有非政府组织对新疆种族灭绝的指控进行独立的法律分析，包括北京可能对被指控的罪行承担什么责任。香港(CNN)根据50多位国际法、种族灭绝和中国地区的全球专家的独立报告，中共政府在新疆被指控的行为违反了联合国种族灭绝公约的每一项条款。

Newline特别行动主管、这份新报告的合著者阿泽姆·易卜拉欣(Azeem Ibrahim)表示，有“压倒性的”证据支持该公司关于种族灭绝的指控，这是一个全球大国，其领导人是种族灭绝的策划者。中共针对该地区维吾尔族人的政策和做法必须从整体上看待，这相当于意图将维吾尔族人作为一个整体或部分摧毁。

**种族灭绝定义：**
*1948年12月，联合国大会批准了长达四页的“联合国种族灭绝公约”，对什么是“种族灭绝”有了明确的定义。中国和其他151个国家都是该公约的签字国。公约第二条规定，种族灭绝是企图实施“意图全部或部分摧毁一个民族、民族、种族或宗教团体”的行为。根据公约，种族灭绝有五种方式：杀害该团体的成员；对该团体的成员造成严重的身体或精神伤害；故意施加旨在造成全部或部分肉体破坏的生活条件；实施旨在防止在该团体内生育的措施；或强行将该团体的儿童转移到另一个团体。自1948年该公约出台以来，大多数种族灭绝的定罪都发生在联合国设立的国际刑事法庭，比如卢旺达和南斯拉夫的法庭，或者是其国家法院。2006年，前独裁者萨达姆·侯赛因在伊拉克一家法院被判犯有种族灭绝罪。然而，任何国际刑事法庭的设立都需要得到联合国安理会(UN Security Council)的批准，中国是安理会常任理事国，拥有否决权，因此不太可能就新疆种族灭绝指控举行任何听证会。*

**事实证据清晰且有说服力**

据美国国务院称，据信有多达200万维吾尔人和其他穆斯林少数民族被安置在该地区庞大的拘留中心网络中。前被拘留者声称，他们在那里受到灌输、性虐待，甚至被强制绝育。中共否认有关侵犯人权的指控，称这些中心对于防止宗教极端主义和恐怖主义是必要的。

2月8日，由世界维吾尔大会(World Uyghur Congress)和维吾尔人权项目(Uyghur Human Rights Project)委托伦敦埃塞克斯法院(Essex Court Chambers)发布的另一份报告得出了类似的结论，即有一个针对中共政府种族灭绝的“可信案件”。考虑的证据包括数千名维吾尔族流亡者的目击者证词和中共政府的官方文件。自2014年以来，据称有100万至200万人被中共政府拘留在新疆各地多达1400个法外拘留营中，当时中共政府发起了一场表面上针对伊斯兰极端主义的运动。这份报告详细描述了营地内的性侵、心理折磨、文化洗脑和死亡人数不详的指控，包括拘留营里的维吾尔族囚犯……被剥夺了基本的人类需求，受到严重羞辱和不人道的待遇或惩罚，长期被单独监禁，没有食物。

自杀非常普遍，以至于在押人员必须穿着‘自杀安全’的制服，不能接触到容易造成自我伤害的材料。

另外该地区维吾尔族出生率的急剧下降-在2017年至2018年间下降了约33％-归因于据称中国实施了官方的绝育，堕胎和节育计划，在某些情况下是强迫妇女未经她们的同意。

虽然违反种族灭绝公约中的一项行为将构成种族灭绝的裁决，但Newline的报告声称，中共政府在新疆的行动符合所有标准。这是一份由法律专家、地区专家和中国民族专家完成的报告，帕克表示，在世界第二大经济体中，这种“严重违反国际秩序”的行为引发了人们对全球治理的质疑。“如果这还不足以引发某种行动，甚至不足以采取立场，那到底需要什么呢？” 根据公约，其他151个签字国有责任采取行动。1月19日，即将离任的川普政府宣布中国政府在新疆实施种族灭绝。一个月后，荷兰和加拿大议会不顾各自领导人的反对，通过了类似的动议。
