**西班牙2021年5月1日电/西喜社——**

本周一，西班牙旅游国务秘书费尔南多-巴尔德斯向世界旅游理事会（WTTC）年度峰会保证，西班牙将在6月实施欧盟的 “数字绿色证书”（即所谓的CVD），并在5月进行试点测试。

就在本周三，欧洲议会在全体会议上以540票赞成、119票反对和31票弃权的巨大优势为CVD项目开了绿灯，但其想法与欧盟理事会（国家）提出的想法有冲突。欧洲议会提议，所谓的疫苗接种护照是类似于 “快速通道 “的东西，这种快速通道可以避免其持有者在从一个欧盟国家到另一个国家旅行时经历额外的限制。安理会则在本月14日的一项决议中否决了 “自由通行 “的想法，并指出每个国家都有权力施加它认为合适的额外限制，哪怕旅行者拥有疫苗护照。

目前的状况是技术上也非常混乱，像法国或丹麦这样的国家已经在单方面测试两种类型的发展，并根据自己的喜好制定措施、法规和解决方案，而西班牙则以欧盟之前提出的建议为框架，试图迅速推进，并寻求制定区域标准。

在西班牙，将由各自治区负责开发CVD（数字绿色证书）并与自治区卫生系统整合，这些自治区必须在6月7日之前提供他们的解决方案，中央行政部门将只负责在数据库中收集由各自治区发送的证明，同时将与其他国家共享该基础，以便目的地国能够核实。

政府的想法是，在5月，所有这些都将准备好进行试点测试，在6月初将有一个明确的解决方案，这将决定性地促进旅游业，甚至是来自欧洲以外的旅游业。

最基本的问题是什么？如果所有欧盟国家没有承诺在一个共同的平台上工作，那么这个系统就无法发挥作用。此外，还有数据保护，隐私保护等问题。

最重要的问题是什么？

它的作用相当有限。只适用于所谓的 “自由通行 “进入一个国家，因为不可能禁止没有携带它的人入境，它不会影响（或似乎不会影响）进入封闭场所、音乐会或体育赛事，甚至不清楚使用未经欧洲药品管理局批准的解决方案（如俄罗斯和中国疫苗）接种疫苗的用户会发生什么，尽管一切都表明他们将被排除在CVD之外。
