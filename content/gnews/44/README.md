剩下的1940年代以及后的这些港台籍中共党员，他们究竟是脚踩两岸三地两条船，同时拥有两岸三地双重身份的中共党员，还是由于他们的父母是港台籍，所以他们虽在大陆出生但仍列为台湾籍呢？这一点看来只有港台当局对照名单进行比对了。
