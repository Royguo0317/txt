#### 2021.01.24 环球时报：有“一中”框架，台海才能避免误判

大陆军机23日、24日连续在台湾西南空域演习，而且两天分别出动战机13架和15架，很多都是解放军主战飞机。台当局备感压力，美国务院则发声明，指责大陆对台恐吓，表示美对台承诺“坚如磐石”。不过美方声明中将中美三个联合公报、“与台湾关系法”和对台“六项保证”并列，这与特朗普政府后期拒提三个联合公报的态度有所区别。

解放军为何在拜登政府上台之初派战机大规模近台演习？美台分析界的解读都未必靠谱。事实上，解放军战机绕台飞行、跨越所谓“海峡中线”已是常态，而台海局势中的美台路线也在惯性延续，现在尚无美台进行调整的明显迹象，整个局势都处于之前业已形成的氛围和节奏中。

人们期待拜登政府带来一些新气象，但是支持这种期待的信息在台海地区鲜能看到。分析界可以依据解读具体行动的参照系相当混乱，充满不确定性，因此很难保证解读与实际情况不会南辕北辙。

其实中美之间已经失去了互相了解的基本框架，双方有越来越多读不懂对方的地方，彼此战略误判的几率越来越高。比如中方会想美方派“罗斯福号”航母近日来南海活动是什么意思？美方会想解放军战机此时近台演习是什么意思？中方又会想美方这么快发声明是什么意思？解放军战机24日又去了台西南空域，美方又会怎么想并作何反应呢？

台当局死抱美国对抗大陆的态度没有要变的样子，而对台海局势主导力提升最快的无疑是大陆方面。如果蔡英文之流不思克制，继续把赌注押在美方不断强化的支持上，而且拜登政府真的顺着蓬佩奥后期的极限操作往前走了，那么台海局势进一步恶化将是注定的。解放军十几架战机去台西南空域可能是最轻的，那些战机出现在台湾岛上空将是迟早的事。

台海局势要想缓和，必须解决局势恶化的根源问题，那就是民进党当局拒绝承认“九二共识”，反对“一个中国”。台海局势的所有紧张都源自民进党当局的这一倒行逆施。而这一变化与美国建制派对台海局势的把控大方向亦有矛盾，与中美三个联合公报水火不相容。美方如果将对华博弈的一系列政策建立在民进党当局这一有明显“台独”指向的轻狂冒进之上，其战略风险将是长期并且累加的。
