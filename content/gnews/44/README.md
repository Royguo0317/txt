据美联社消息，英国撤销中共官方媒体在该国的广播牌照，此前一项调查发现，牌照持有人缺乏编辑监督，并与中共有联系。

英国通信监管机构2月4日表示，撤销了中国环球电视网（China Global Television Network，简称CGTN）的英国牌照，这是一个国际英语卫星新闻频道。CGTN曾在英国的免费和付费电视上播放，但目前还未收到CGTN对此事的评论。

监管机构对该频道进行的调查，始于几位受害人提出的投诉。投诉称，该频道违反了公平性和准确性的规定，向公众播放了他们所遭受的逼供。其中一位是来自前英国驻香港领事馆的雇员，他因提供有关抗议者的信息而被中共警方拘留并遭受酷刑。另一位是来自英国公司的调查员，在中共国被囚禁期间被迫招供。

监管机构表示，发现拥有该频道许可证的实体Star China Media Limited对CGTN的输出未承担编辑责任，而该责任是维持许可的要求之一。

英国通信监管机构表示，由于宣称“重要信息”缺失，针对CGTN许可证的转让申请被其拒绝。 “我们认为CGTN应该被撤销执照资格，因为一个最终由中共控制的机构把持着它”， CGTN最终还是付出了代价。

该监督机构同时表示，此前给予CGTN “大量时间 “来遵守规定，但这些耐心 “现在已经用尽”。英国通信监管机构说，”经过慎重考虑所有事实，以及广播公司和观众的言论自由权，我们认为撤销CGTN在英国的广播执照是合适的” 。
