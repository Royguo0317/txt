## 【战友心声】民主的必备条件（一）道德伦理信仰

作者：纽约香草山医疗部 云豹



![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-2-1.jpg)



民主就是由人民做主人：人民拥有管理政府的权力，政府的所有成员都是由人民选举出来的公仆。因此，政府成员的组成可以看做是人民的缩影：有什么水准的人民就会选出什么水准的政府成员。这是民主政治的宿命。

如何能拥有高水准的政府成员呢？最根本的解决之道，是从人民的道德、伦理、信仰等精神素质层面地培养着手。

人之所以与动物不同，在于人懂得惭愧、怜悯、维护家庭伦理、敬畏自然或神祇。由于这些精神力量的支持及约束，让人类能克制自己的欲望，能与家族成员及他人分享自己的利益，能在对来生的期许中让自己身处困境仍能维持奋斗的勇气，能在顺境中约束自己，不因无限扩张的欲望而升起害人的恶念。

在中华文化的传统教育中，镌刻着敬畏天、地、鬼、神的印记。“举头三尺有神明”，这句话一直是警示着人们不能随意逾矩的文化传承；建立在伦理规范下，坚定维护家庭亲族体系和谐茁壮的信念，也是人们心中难以撼动的牵绊。

这些正是孕育民主体制的肥沃土壤。敬畏天地、深信因果、守护家族和谐及利益、期许美好的来生，这些足以让人们自动约束因无限扩张的欲望而升起的造恶心念。

不幸的是，中共的唯物无神论故意破坏了这些约束民众向善的主要力量。无神论让民众只相信眼前的一切才是唯一能拥有的，于是民众不信因果、不守伦理道德、胡作非为，只图眼前的利益，不计任何手段或代价只求达到目的。

这种心态下的人民所“选举”出来的政府成员，也一定是以这种心态来做事的。那政府又怎么可能为民众谋福利，他们的所作所为一定是贪赃枉法、图谋私利的。更何况，中共何时给过人民去选举的权利？

在未来新中国联邦的建立过程中，必须先重新建立起人们对于天地、自然、因果法则的敬畏、对来生的期许、对家庭亲族成员的敬重、对身边他人的分享以及对贫困残弱的守护观念。这些观念能启发人们思考：人为何生、如何活及怎样才能活得光彩炫丽，而不是只一昧地追求物质的满足。

要达到此目的，媒体必须担负起重要的传播力量。媒体可经由不同方式及角度的启发、探讨，在互动式的潜移默化中，逐渐重塑民众的精神力量。必须有精神食粮富足的民众，才会有精神文明富足的国家，也才可能期待有正常运作的民主体制。如何来达成此目标，将是GNEWS、GTV等媒体平台的重大使命及责任。
