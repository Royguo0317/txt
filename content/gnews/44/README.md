据德国之声中文网报道，香港警务处处长邓炳强近日的“假新闻”与国家安全有直接关系的言论，受到香港外国记者会主席的质疑。

邓炳强表示，不少外部势力利用在香港的代理人，用“假新闻”“假消息”的方式，在香港煽动仇恨、分化社会、制造矛盾等。他强调，对于尝试用“假新闻”危害香港安全的人，警方会全面调查，一有证据就会进行拘捕及检控。

![]()![](https://gnews.org/wp-content/uploads/2021/04/图片24-4.png)

针对邓的此番言论，香港外国记者会（FCC）主席瑞凯德（Keith Richburg）本周致函邓炳强，“迫切”要求其澄清以下三大疑问：香港尚无“假新闻”法，如何具体定义“假新闻”；什么形式的报道（例如文字、图片、影片、社交媒体帖文）会构成“假新闻”，有何法律基础进行调查和检控；如何保证在调查“假新闻”的同时，兼顾政府曾许下的保证香港新闻自由的承诺。

这封在香港外国记者网站发表的公开信还指出，该会对此事深切关注。瑞凯德质疑，“假新闻”法规被许多国家用于打压新闻及言论自由。

警方回应道：“没有人可以凌驾于法律之上。我们会果断采取行动，就违法行为展开调查和搜证。”警方回应并未就香港外国记者协会提出的三大疑问做出回答。
