摘要：中共病毒已经肆虐全球1年多，截至4月22北京时间8点全球感染人数超过1.43亿人次，死亡人数超过300万人次。各个国家从去年开展注射疫苗，但疫情都没有得到控制，而且病毒还在不停地变异。爆料革命的引领人郭文贵先生在4月21日的盖特里称“中共病毒是生化武器，永远不可能有疫苗，所有的疫苗都是假的。根据科学家和国内的情报，在欧洲的那位朋友明确的告诉我们共产党有解药，去跟共产党要解药”。郭文贵先生的盖特再次让深受中共病毒祸害的世界各国人民带来了希望。

回顾一年前，爆料革命的战友闫丽梦博士于2020年1月19日通过路德社爆料中共病毒会人传人的真相。由于在此日之前中共掩盖了人传人的真相导致病毒肆虐全球。各国人民对闫丽梦博士的爆料将信将疑，因而错过了采取防疫措施的最重要的时间点而导致各个国家的人民无辜感染和死亡。导致今天这样的局面主要从三个方面分析。
![]()![](https://gnews.org/wp-content/uploads/2021/04/病毒肆虐6.png)
首先，中共掩盖了病毒是生化武器并会人传人的真相。虽然李文亮医生第一时间向上级报告了人传人的真相，但最后被中共封锁。结果就是李文亮医生自己感染最后未能救治而死亡。中共明白它自己的所作所为当然不会轻易让世人知道真相。紧接着2020年的1月19日闫丽梦博士通过路德社告诉世界中共病毒是会人传人的真相。过了12天，美国在2月1日才开始宣布禁止中国公民前往美国，接着各国才陆续断航。此时中共病毒已经扩散，断航已经没有办法阻止病毒的传播。
![]()![](https://gnews.org/wp-content/uploads/2021/04/病毒肆虐5.png)
其次，科学家闫丽梦博士关于中共病毒是实验室制造并会人传人的三份报告，不仅没有引起世界各国的科学家的重视反而受到了各种无情攻击。她一再指出“在没有查清楚SARS-CoV-2的真相之前研发的疫苗对病毒都是无效果”。以美英德为主的国家医疗机构还是继续研发疫苗。俗话说”知己知彼才能百战不胜”，在没有查清楚中共病毒结构之前就研发疫苗那就是等于浪费时间和金钱。也同时证明了疫苗研究部门为了金钱可以全然不顾纳税人的生命安全。闫博士提供的预防中共病毒的羟氯喹也受到了无情的打压。因为这个药太便宜了，而且非常容易购买到。它阻塞了利益相关人通过疫苗谋取巨额利益的管道。

![]()![](https://gnews.org/wp-content/uploads/2021/04/病毒真相09.png)
