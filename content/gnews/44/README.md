紧随外交部，《华盛顿邮报》、《路透社》、BBC、CBC等跟进造势，大张旗鼓地进行了配合宣传。

《路透社》文章题目为，“世卫组织说，该小组将在武汉访问实验室、市场和医院”。【3】

文章说，一个由世界卫生组织领导的研究COVID-19大流行起源的小组将于周五会见中共国科学家，并计划参观武汉的实验室、市场和医院。

该小组周四离开位于武汉被隔离的酒店，开始实地工作，两周前（小组成员）抵达2019年底这个出现病毒的中共国城市。

调查团一直受到拖延、对访问（区域）的进入，以及中美之间争辩的困扰，美国指责中共国隐瞒了最初疫情的程度，并批评了（对参观访问设定的）条款，中共国专家正是在这些条款下进行了第一阶段的研究。

调查团队计划参观医院、实验室和市场。世界卫生组织在微博上说，“实地考察将包括武汉病毒学研究所、华南市场、武汉市疾控中心（疾病预防控制中心）实验室。”

据报道，独立专家小组将在中共国再待两周，他们还将在武汉与首批中共病毒患者进行会谈。

“所有的假设都摆在桌面上，因为研究小组在他们的工作中遵循科学。（世卫）他们应该得到他们需要的支持、访问和数据。”

丹麦团队成员西娅·菲舍尔（Thea Fischer）说，参观华南海鲜批发市场（病毒最初被认为是在这里传播的）将有助于深入了解该市场是疫情的中心，还是只是病毒的放大器。

菲舍尔在武汉通过电话告诉路透社记者，“现在实际的实地工作可以开始了，我希望在这部分任务中，我们能够不受阻碍地进入所要求的目的地和人员。”

“但重要的是要记住，这项任务的成功和来源追踪是100%取决于对相关来源的访问，不管我们有多能干，我们有多努力，只有在中共的支持下，这才可能实现。”

在离开他们的隔离酒店后，没有和记者说话，队员们登上一辆巴士前往一家湖边酒店，那里的部分建筑和地面被封锁。

在世界卫生组织宣布他们计划访问的几个小时前，世界卫生组织总干事特德罗斯·阿达诺姆·盖布雷耶苏斯（Tedros Adhanom Ghebreyesus）在推特上说，“感谢中共国卫生部长马晓伟就#COVID19病毒起源调查任务进行了坦率的讨论。”

世卫组织一直在努力实践着人们的期望。世卫组织紧急情况主管迈克·瑞安本月对记者说，“没有任何答案的保证。”

病毒起源被高度政治化。调查组原定于1月初抵达武汉，中共国推迟访问引起世卫组织负责人罕见的公开批评，美国前总统川普指责世卫组织在疫情爆发初期“以中共国为中心”。

中共国通过官方媒体引述进口冷冻食品包装上存在病毒，以及科学论文称2019年病毒已在欧洲传播的说法，（开始）推动在武汉发现该病毒之前国外已经存在的概念。中共国外交部还暗示，2019年7月马里兰州德特里克堡美军实验室突然关闭与疫情有关。
