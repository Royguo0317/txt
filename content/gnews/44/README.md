雷军在谈小米未来的发展策略时说道，“小米将会用互联网赋能制造业在继续和代工厂真诚合作的基础上，我们会深度参与制造业，小米将成为未来生活方式的引领者，小米将成为中国制造业不可忽视的新兴力量”。从他的豪言壮语中可见小米集团的目标就是要逐渐渗透进制造业以及老百姓的生活中，把互联网带入人民生活的方方面面。中共通过操作这类互联网公司对老百姓的生活进行全方位的监控，对各行各业的企业进行渗透、控制，更方便地榨取企业利润。
