## 国际特赦组织：中共迫害新疆维族 构成反人类罪
` HoustonLitstar` [轉載自GNews](https://gnews.org/zh-hans/1320358/)

作者：休斯顿星光农场写作部 文顺


![This image has an empty alt attribute; its file name is Screenshot-2021-06-01-215655-2.jpg]()![This image has an empty alt attribute; its file name is Screenshot-2021-06-01-215655-2.jpg](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/08042348/Screenshot-2021-06-01-215655-2.jpg)
**6月10日，人权团体国际特赦组织（Amnesty International）公布调查报告，指出中共镇压数十万新疆维吾尔族与其他少数民族，已构成“反人类罪”（Crimes Against Humanity）。**

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/13174814/19-7.jpg)图片来源：网路

据法广网报导，国际特赦组织在这份长达160页报告中详细描述中共“有系统性地、国家组织型大规模羁押、酷刑和迫害，已构成反人类罪”。

报告中记录了中共国当局自2017年以来，对维族人和其他突厥裔少数民族采取的“极端措施”，也包含曾被关押在中国新疆地区集中营人士的证词。

国际特赦组织采访50多名曾被拘留在新疆集中营的人全都表示，他们因持有宗教图片或与国外的人士联系而遭拘留。许多曾被关押的人士描述，他们会先被带到警察局，被手铐和脚镣铐在钢椅上接受讯问。

他们表示，被殴打、剥夺睡眠和狱中人满为患非常常见，在“极度有纪律”的拘留营中，他们没有隐私或自主，且可能会受到严厉的惩罚。
报告说：「除了将数十万人送进监狱外，还有数十万人、甚至可能是百万人以上被送往拘留营。」

国际特赦组织秘书长卡拉马尔德（Agn?s Callamard）表示，这些受虐情事“震撼人类良知”，并呼吁联合国依据国际法展开调查。

她说：“穆斯林少数民族面临反人类罪和其他严重侵犯人权的行为。” “中共国必须立即拆除拘留营，释放任意被拘留在营中或监狱的人。”
