据疫情新闻（Pandemic News)1月22日报道，一位批评中共病毒可能从武汉实验室意外逃逸或泄露理论的美国科学家承认，他是为了保护中共国科学家而谴责这种说法。

位于纽约的非政府组织 “生态健康联盟”(EcoHealth Alliance)主席彼得-达斯扎克博士(Peter Daszak)在2020年2月领导了一项工作，以平息所有关于中共病毒可能从中共国国家支持的研究和学术机构——武汉病毒学研究所意外逃脱的怀疑。著名医学杂志《柳叶刀》也在全球大流行的早期阶段发表声明，谴责 “暗示中共病毒没有天然来源的阴谋论”。当时还没有对病毒的起源进行任何形式的严格调查。达斯扎克在2020年2月6日发表的声明中进一步重申了对中共国的支持，称他与其他科学家站在一起，强烈谴责那些认为中共病毒并非自然起源的“阴谋论”。声明称科学证据压倒性地表明，这种病毒像许多其他新兴疾病一样起源于野生动物。6月，达斯扎克还为《卫报》写了一篇题为：”忽略阴谋论：科学家们知道中共病毒不是在实验室里产生的 “的评论文章。

但在2021年1月15日星期五，达斯扎克的发言人告诉《华尔街日报》，他发表声明是为了保护中国科学家不受批评，他说：”《柳叶刀》这封信是在中国科学家收到死亡威胁的时候写的，这封信是为了表示对他们的支持，因为他们夹在试图阻止疫情的重要工作和网络骚扰的挤压之间。”但声明却被用来压制对中共病毒起源有不同意见的人。

世卫组织团队前往武汉探寻病毒来源。根据世卫组织的说法，该团队此次到中共国的的官方任务是确定病毒如何、在哪里以及何时从动物身上传播到人类。达斯扎克医生是世卫组织派往武汉查明中共病毒起源的15人国际专家小组的成员。目前他和他的团队已到达中国，正在强制隔离的第四天。每天的时间都被排满了虚拟会议。团队领队、世卫组织食品安全和动物疾病专家彼得-本-恩巴雷克（Peter Ben Embarek）说，一旦他们完成强制检疫期的工作，将获准 “四处走动，亲自与中共国同行会面，并去我们想去的不同地点”。恩巴雷克表示，他希望访问武汉华南海鲜批发市场，试图确定在首批确诊病例之前的几周内，那里的 “一切进进出出”。那里经常出售许多野生动物，据（中共）说可能是所谓的中共病毒 “动物到人的跳跃 “环境，或者是加速这种跳跃的地方。”我们知道病毒在某个时间点起源于蝙蝠，然后我们知道2019年12月在武汉出现了人类病例。”恩巴雷克说。”但中间发生了什么，中间有多少其他动物物种参与，以及在哪里，还有待更详细的发现。…我们并不知道在那段时间里发生了什么，这也是我们正在寻找的。”

关于该研究所在最初的疫情爆发中的作用，问题仍将存在；如果世卫组织团队不进行调查，这些问题将继续挥之不去。

时任国务卿迈克-蓬佩奥（Mike Pompeo）甚至表示，政府有理由相信，为该研究所工作的几名研究人员 “在2019年秋季，也就是在第一例已确定的疫情发生之前就生病了，其症状与中共病毒和常见的季节性疾病一致”。蓬佩奥说，这让人质疑该研究所蝙蝠相关病毒高级研究员石正丽的说法，即该研究所的学生和工作人员中 “没有出现过[中共病毒或SARS相关的]感染”。蓬佩奥指出，在实验室里 “意外感染 “的可能性比人们想象的要大，特别是考虑到中国已经发生过这样的事件。2004年，北京爆发了一场SARS疫情，导致9人感染，1人死亡，事件起源于一个研究机构。

**评：**

做为世卫组织成员的达斯扎克医生，曾在中共病毒疫情之初的去年二月份，领导了一批专家，在中共拒绝世卫组织到武汉P4实验室进行调查的情况下，制造舆论，谴责“中共病毒并非来源于自然”的说法，这显然是在为中共掩盖病毒真相而撒谎。

冠状病毒专家闫丽梦博士去年先后发表过两篇报告，证实中共病毒就是中共军方在实验室制造的生物武器。驳斥了之前多名科学家在毫无根据地前提下，声称中共病毒来源于大自然，这种为中共洗地的说法。到目前为止，还没有人敢对闫博士的论文提出挑战，用切实的证据来证明阎博士的指证是站不住脚的。

此次世卫组织被允许进入中共国调查中共病毒来源，只是提出去中共所谓的病毒来源地——华南海鲜市场，那只能是浪费时间，或是走过场。到底他们会不会去病毒真正的发源地——武汉P4实验室呢？笔者对此不抱太大希望。

为什么达斯扎克在这次去中共国调查病毒来源时突然改口，或者说口气变软，一改往昔“病毒像许多其他新兴疾病一样起源于野生动物”的论调，而变成了“为了保护中国科学家不受批评”。他是怕对“中共病毒来源于实验室”的谴责，压制了病毒来源的不同声音？还是想对“病毒不是来源于大自然” 的说法委婉地表达同意？亦或是一年后，疫情发展到今天，全世界已有近一亿人感染，超两百万人死亡（不含中共国的真实数字），并且还有更多人将因病毒感染、死亡的情况下，有些许良心不安？
