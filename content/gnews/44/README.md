**2.美国天然气价格预计有上行压力**

[天然气价格在过去几个月中飞涨，随着夏季的临近，价格可能会保持在这些水平或仅略微上升。传统基金会（Heritage Foundation）的经济学家尼克·洛里斯（Nick Loris）称，“随着美国和世界各地越来越多的经济体解禁，随着燃料需求的持续增长，这将给油价带来更大的上行压力。”洛里斯补充说，拜登对近期的油价上涨没有影响，但政府已决定禁止在联邦土地上进行石油开发的新租赁，从长远来看，这可能会影响石油的供应，进而影响油价。](https://www.washingtonexaminer.com/news/gas-prices-not-expected-to-fall-anytime-soon-as-us-emerges-from-pandemic)

**3.英国预算赤字达到二战结束以来的最高水平**

[4月23日官方数据显示，英国在冠状病毒大流行期间的预算赤字上升至第二次世界大战结束后一年以来的最高水平。国家统计局表示，截至3月底的财政年度，公共部门净借款（政府支出与税收之间差额的官方衡量标准）达到3031亿英镑（4200亿美元）。这相当于该国年度国内生产总值的14.5％，是自1946年赤字占GDP的15.2％以来的最高水平。在战后时代，赤字在2008年金融危机后达到顶峰，达到GDP的10％左右。自1970年以来的平均赤字占GDP的3.4％。其主要原因是中共病毒大流行一年多以来，政府已经投入了数十亿英镑试图支撑经济和就业。](https://www.breitbart.com/europe/2021/04/23/uk-budget-deficit-at-highest-level-since-end-of-second-world-war/)

**4.特斯拉正面临中共政府的强烈反对**

**5.强生股东批准首席执行官近3000万美元的薪酬计划**

[强生公司的股东在4月22的医疗保健公司年度股东大会上批准了首席执行官亚历克斯·戈尔斯基（Alex Gorsky）的2020年2960万美元薪酬计划。强生的薪酬计划受到了一些投资者的审查，因为强生的薪酬计划并未计算出对高管人员的奖励，其中包括与成千上万起诉讼有关的约90亿美元的成本，诉讼称该公司助长美国阿片类药物危机等。戈尔斯基还回答了投资者的问题，即强生公司的中共病毒疫苗与非常罕见但严重的血液凝块状况（可能导致在美国暂停使用）之间的可能联系。他称该公司期待着美国监管机构的审查](https://www.oann.com/jj-shareholders-approve-ceo-gorskys-near-30-million-pay-plan/)

**6.28家企业被纳入电信业务经营不良名单, 移动、电信分公司在列**

[中共国工业和信息化部官方公众号“工信微报”公布了2021年第一季度28家电信业务经营不良企业名单，其中包括中国移动5家分公司，中国联通5家分公司，中国电信4家分公司，中国有线电视1家分公司。 公告显示，依据《电信条例》《电信业务经营许可管理办法》等相关规定，28家企业因违规接入未备案网站、APP违法违规收集用户信息、违反电话实名制、不正当竞争、无证经营电信业务等违法违规行为，受到电信管理机构行政处罚并被纳入电信业务经营不良名单。](https://finance.sina.cn/2021-04-23/detail-ikmyaawc1306680.d.html?from=wap)
