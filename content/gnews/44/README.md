国家情报总监约翰·拉特克利夫（John Ratcliffe）于2020年12月4日在《华尔街日报》上写到，中共国已经在测试超级士兵技术。 中共寻求使用CRISPR基因编辑工具来编辑人类胚胎，以培养战斗力更强、更有耐力、更有效率的士兵。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/08105640/Captain_America_The_First_Avenger_poster.jpg)《美国队长》海报 图片来源：维基百科
什么是CRISPR？ 这是一项可以在所需位置切割生物体基因的技术，可以删除现有的基因和/或添加新的基因。 该技术的发展获得了2020年诺贝尔化学奖。

中共国是CRISPR应用领域的领头羊。 中国研究员贺建奎于2018年11月利用CRISPR技术在中国深圳创造了人类第一个基因编辑婴儿，而他的技术是在斯坦福大学学习时获得的。 如果中国平民能够创造出基因编辑婴儿，那么中国人民解放军当然可以制造出经过基因“改良”的超级士兵。

中共会受到道德约束吗？ 当然不会。 中共已经示范过了如何把人变成武器，通过新冠病毒“增强”的中国人把病毒传播到世界各地。 在爆发初期，中共禁止国内航班，但允许国际航班离开武汉。 截至2020年12月7日，全球新冠病例数和死亡人数分别达到6400万和150万。根据闫丽梦博士的报告，新冠病毒是中共采用“限制酶切位点手法”基因改造后的生物武器。

虽然大多数人运用科学来挽救生命和改善生活，但中共却在利用科学制造反人类的超级士兵和超级病毒。 中共没有道德底线，因为中共信奉无神论。 只有永久消除中共，才能解除它的威胁。
