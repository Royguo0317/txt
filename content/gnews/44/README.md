纽约香草山记者站的Kathy文艺战友，在去庆典活动之前，在酒店大堂，对Allen美发师战友和他的美甲师太太，进行了面对面的采访。采访内容如下，供您欣赏。

（文艺记者）我面前站着的这位帅哥，就是昨天为我们做头发的发型师Allen战友，在傍边的，是帮助大家做指甲的，是他美丽的太太。我刚刚下楼，正好碰上他们。Allen战友设计的发型，真的做到了妙手回春。当我做好发型，非常满意主动地送上小费，他回绝说：“给小费就是伪类”。我尴尬地收回了，也收获了这份战友之情。首先由发型师分享这两天来的感受。谢谢！

（Allen战友) 新中国联邦成立一周年，全球战友都在看着我们的形象，我们不是代表自己，现在我们是代表新中国联邦来出席这个盛大的庆典。我本人是美发师，现在是用到我的专业的时候了。我一定要给力，把我们战友的形象设计好，让全世界看到我们新中国联邦人全新的面貌。我帮助大家把这种面貌体现出来，并传送出去。我尽自己的所能。一个人的能力有限，尽力而为吧。谢谢大家！

（文艺记者）您即是美容师、美发师，也是心灵的美化师，我看你胸前除了我们新中国联邦国旗的胸针外，还有另外两个胸针，请问，这有什么特殊意义吗？

（Allen战友）哦，这是买西装时自带的，配套的，这太特别了！

（文艺记者）这几天我一直观察Allen战友忙忙碌碌地，一个接一个地为战友设计发型，同时让我安排没有整理发型的战友做好准备，这比世界上任何专业的，收费的态度要好多少倍。这就是战友之间的情意！

（文艺记者）站在傍边的这位女士，就是Allen先生的太太。她一直在为战友们做美甲。战友们排着队等。这对美丽的夫妻一直在为战友服务。让太太为我们讲两句？

（Allen太太）我们尽所会的技能，让我们新中国联邦人，站在全世界的舞台上更漂亮、更帅气、更美丽、更得体……

（文艺记者）讲得太好了！就是让我们每一位新中国联邦成员，都要从里到外，从外到内都是美的。二位的辛勤劳动，做出这样的牺牲和奉献，我们会永远记得。尤其在今天这特殊的日子，我们每一位战友，走在街上的时候，看上都是美美的、帅帅的。谢谢你们辛勤的劳动，谢谢你们的付出。感谢。一会儿将在会场相见！
