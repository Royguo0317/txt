美东时间11月2号，美国总统大选前24小时，川普总统和彭佩奥国务卿分别于白宫及国务院官网发布讨共宣言。

川普总统在白宫官网表示中共不再像毛泽东领导下那样只影响中国人民，中共包括释放Covid-19（中共利用超限生物武器向全球发动超限生物战争）、破坏海洋环境、知识产权盗取、渗透美国政治及相关行业等行为影响了整个世界。川普总统强调美国不再对中华人民共和国的行为视而不见，美国正在行动。

彭佩奥国务卿在国务院官网上指出中共在香港逮捕八名泛民主主义政治人物，其中包括立法会议员五名，其行为是出于政治目的滥用执法权。彭佩奥认为中共试图破坏香港应有的自治和对人权的尊重，美国强烈谴责中共恶行并于香港人民站在一起。

美国大选前川普总统和彭佩奥国务卿的“讨共宣言”，证明本届政府在爆料革命的持续曝光下，已充分认识到中共企图称霸世界奴役人类的野心，本次宣言也表明了川普政府的态度，大选之后将快速开启以法灭共进程。
