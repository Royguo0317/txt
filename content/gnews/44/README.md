## 论法媒“Covid-19有助于抑制全球气候暖化”之怪谈
` 喜马拉雅德国慕尼黑感恩农场` [轉載自GNews](https://gnews.org/zh-hans/1042734/)

#### 撰文作者：文渊

3月底“气候与持续”法案于法国国民议会进行讨论。该法案旨在通过一系列强制性法规，降低法国温室气体排放量，以延缓所谓全球气候暖化。近日，众多法国民众与法国媒体展开了造势活动，以期该法案获得通过。3月28日，法媒“世界报”竟有分析[报导](https://www.youtube.com/watch?v=mfrbLNBpanE)，暗示Covid-19疫情有助于抑制全球气候变暖。
![]()![](https://gnews.org/wp-content/uploads/2021/04/bear.png)
该报导称：因受Covid-19的影响，2020年全球能源消耗下降，使全年温室气体排放量相较前一年降低5%。然而却分析道，2020年由Covid-19制造的减排效果对抑制全球气候变暖之贡献极其有限。若要在21世纪末将全球气温升幅控制在2 °C以内，全球必须在今后每年（至少持续10年）皆达到5%的减排指标，即今后每年皆应复制2020年因Covid-19带来的减排成就。

观看后，我感到十分不适。在左倾思想盛行的欧美地区，媒体报导都讲政治正确。环保议题（对抗全球气候暖化、发展节能减排）属于政治正确，因此经常被拿来反复炒作、大书特书。而持相反观点的议题属于非政治正确，很难被允许发表。所以在西方国家，诸如此类的报导多如牛毛，我早已对其见怪不怪。我之所以对这篇报导感到反胃，实在是因为其中某些观点突破了我的认知底线。

该报导不仅歪曲了事实真相，更向听众传递了不健康的价值观。以下就事实、价值观两方面分别说明。

（一）全球气候暖化——是真相还是谎言？

在学校时，书本中说全球变暖是事实，我们不曾怀疑。在媒体上，科学家说全球变暖是事实，我们更不会质疑。全球变暖，好一个毋庸置疑的“事实”，就此在我们心中打下了深深的烙印。直到追随爆料革命之后，终于恍然大悟、如梦初醒。所谓的全球暖化理论，其实是中共等暗势力编造的世纪谎言。

根据路德社观点：全球气候暖化是中共等暗势力为实施“全球大重启”计划而炮制的虚假理论，旨在垄断资源、剥夺民权、加强专制等。该理论的制造者们，他们会允许普通民众了解百年来地球的真实气温变化吗？他们会以身作则、率先做到节能减排吗？可惜全球尚有如此之多的普通民众被蒙在鼓里，并自愿为对抗所谓全球气候暖化而牺牲自己的权益（如自愿吃素、限制出行等）。他们的善良与伟大，反衬出谎言制造者的阴险与卑鄙。
![]()![](https://gnews.org/wp-content/uploads/2021/04/Madness.png)
（二）以环保之名——您愿意与病毒长期为伴吗？

如果媒体向听众传播了错误的信息，这或许不全是媒体的错。例如，在左派势力治下的欧洲，指望一家媒体敢于质疑全球气候暖化，这是极不现实的。不过，如果媒体向听众输出了不健康的价值观，则可以罪加一等。

该篇报导将Covid-19与气候问题关联在一起，含沙射影的指出疫情对节能减排以及对抗全球气候变暖带来的积极作用。言下之意，如果全球今后每年都处于疫情之中，将有助于达到减排指标、将有利于对抗全球气候暖化。当我理解到这层涵义后，顿时汗毛竖起、脊背发凉。我宁愿相信这是一个误会：是由于法国媒体记者的天真浪漫，使他们做出了夸张的比喻和表述。但无论如何，任何媒体皆不应该为疫情渲染正面意义（哪怕环保议题在西方高于一切）。

结束语：

这个主题使我再次想到了瑞典那个“环保女孩”。在西方人眼中，她的形象是正面的。但战友们知道，她只不过是一个被人利用的棋子。西方青少年在接受左派媒体的轮番宣传后，不知又有多少人会成为下一个“环保女孩”呢？

西方民众生活在自由民主的环境，享有相对的新闻与言论自由。与中国人相比，他们是幸福的。但是在如今这个肮脏污秽的世界，西方人与中国人所接受的信息并没有本质的不同。即使是西方人，也无法逃脱媒体的洗脑宣传。

唯有结束中共，才能使全球媒体界重回正轨，才能结束这个遍地谎言的疯狂时代，才能还真相于全球人民。爆料革命、GTV、G-News重任在肩，舍我其谁！
