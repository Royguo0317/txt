# **何謂加密貨幣？**


**定義**

以密碼學原理確保交易安全，並控制貨幣發行的數字貨幣。又稱「虛擬幣、數字貨幣」。
註：加密貨幣與數字貨幣仍有區別。數字貨幣並沒有利用密碼學確保交易安全、控制發行量，未進行加密的數字貨幣發行量可被發行商所更改。

**比特幣：開創扭轉法定貨幣機制違背市場機制的可能**

- 本質：世界上第一個脫離現有國家貨幣金融體系與政府監管體系的分佈式公開帳本。（獨立運作機制、不存在中心數據機構）
- 誕生背景：2009年，全球金融危機。各國政府在金融危機當中，利用納稅錢積極給予金融機構、銀行業者補助。目前，比特幣被廣泛用於暗網交易，甚至被中共作為繞過美元監管體系、顛覆美元的途徑。
- 意義：法律與權力可以規管加密貨幣的使用，但無法規管加密技術的發展與應用。（加密技術可應用在許多領域當中）


政清表示，由比特幣白皮書內容、出現時點（全球金融危機），可以認為比特幣的出現意義是「無政府主義、貨幣自由化」理念的實踐。
金融危機之下，私人所有的金融機構面對財務困難，本應走向破產、清算的結果。但因為法定貨幣體系，使政府祭出大量紓困方案，並動用人民納稅錢給予金融機構補貼。

**貨幣自由化意義：避免行政壟斷、法律干涉**

行政壟斷與商業地位壟斷應予以區分。行政壟斷，是指政權利用權力、法律限制特定技術、商業活動或金融發展；商業地位壟斷，是指在合乎法律規定、市場自由競爭前提之下，透過自身技術能力或商業模式吸引大量用戶，建立自身不可動搖的市場壟斷地位。
兩者差別在於，在市場自由競爭的前提下取得市場獨佔、寡占地位，不會限制其他競爭的商品服務出現。
註：加密貨幣不會完全取代現有的法定貨幣系統，加密貨幣將與法定貨幣共存。

**區塊鏈：分佈式的數據庫系統**

區塊鏈技術出現在比特幣之後，是按照比特幣白皮書及比特幣實際運作進行總結，以比特幣為基礎建構了區塊鏈技術。




- 分佈式：
區塊鏈技術以節點記錄完整的數據。節點又區分為「全節點、輕節點」，所有全節點均有權利對全網交易進行廣播與驗證，但每個區塊僅需一個全節點負責交易廣播與驗證。


全節點：均存有完整的區塊鏈數據。(全部比特幣的交易數據)
輕節點：擁有與自己相關的交易數據。(個人比特幣的交易數據)


- 數據庫系統：「區塊鏈」=「區塊」+「鏈」
每個區塊存儲固定時間內系統的所有交易數據，「區塊」以時間標記的先後順序組成「鏈」。

以比特幣為例，比特幣區塊鏈系統，以10分鐘為單位儲存全球比特幣帳戶交易總數據。但實際在運作時，比特幣區塊鏈儲存單位可能會轉為5分鐘或者超出10分鐘為單位記錄、儲存交易數據。

# **共識機制**


區塊鏈儲存數據的方式有全節點、輕節點兩種，全節點記錄有特定時間單位的所有比特幣交易數據，並且負責交易廣播與驗證。（所有的全節點都記錄有全部的交易數據，但交易廣播與驗證工作只需要一個全節點負責）

單一個全節點負責交易數據的廣播與驗證，數據廣播是為了同步更新其他節點儲存的交易數據。而該負責數據廣播與驗證的全節點，就是俗稱的「礦工」。而一次只能有一個全節點負責廣播、驗證交易數據，因此各希望取得負責廣播、驗證交易數去權利者，必須經過高速運算來取得擔任廣播、驗證數據的全節點資格。

以比特幣為例，比特幣的區塊鏈系統會讓礦工執行雜湊函式的運算，該雜湊函式的運算是無意義的。但因為雜湊演算法的技術特性，利用雜湊演算法所計算出來的雜湊值（Hash Value），無法逆向演算回原本數值的特性，因此達到不可回溯性。

希望成為礦工的候選者，必須利用設備高速運算取得擔任礦工的資格。政清舉例，雜湊演算就好比拋擲硬幣，欲取得擔任礦工資格必須連續21次擲出硬幣正面，第一個連續擲出硬幣正面的候選者就會被系統判取得礦工資格，負責該10分鐘內的所有交易數據廣播、驗證。礦工負責交易數據的廣播、驗證，通過共識機制會取得對應的獎勵。
