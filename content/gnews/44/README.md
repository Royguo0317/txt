## 中共国养老基金巨额亏空 中共再下狠招 【延迟法定退休年龄】白嫖百姓养老金
` shangshang` [轉載自GNews](https://gnews.org/zh-hans/556401/)

作者：小妍说

11月3日，新华社受权发布《中共中央关于制定国民经济和社会发展第十四个五年规划和二〇三五年远景目标的建议》。《建议》提出，实施渐进式延迟法定退休年龄。目前中共国的退休制度，男女退休的年龄并不一样。男性年满60岁，女干部年满55岁、女工人年满50岁方可退休。从事井下、高温、高空、特别繁重体力劳动或其他有害身体健康工作的男性，退休年龄在55周岁。这个退休年龄框架，始于1951年2月23日颁布的《中华人民共和国劳动保险条例》，当时中国的人均寿命还不足50岁。1978年《国务院关于颁发〈国务院关于安置老弱病残干部暂行办法〉和〈国务院关于工人退休、退职的暂行办法〉的通知》再次予以确认，一直沿用至今。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/14034905/B3C37ACF-D31C-462D-B37C-D4F8327CAB1D.png)
之所以要推行延迟退休，根源中共官方给出的答案是由于中共国日益加剧的老龄化状况。中共国家统计局公布数据显示，截至2019年末，全国60岁及以上人口为25388万人，占总人口18.1%。其中65周岁及以上人口17603万人，占总人口的12.6%。这一数据跟2018年年底相比，老年人口增加约439万。2018年末，我国60周岁及以上人口24949万人，占总人口的17.9%，其中65周岁及以上人口16658万人，占总人口的11.9%。但由于种种原因，八年多以来，延迟退休方案迟迟未定。目前，业内讨论较多的主要有两种方案：第一种是先将男女退休年龄统一到60岁，把女性两档退休年龄统一到一档，再同时延长男女退休年龄；另一种方案是男女各自延长，女性延长步伐比男性快，最终统一男女性退休年龄。中国人民大学中国社会保障研究中心副主任杨立雄赞同第一种方案。他认为，部分女性50岁就退休，年龄太早，当务之急是应该将男女退休年龄先统一起来。“第一步，女性可以先统一到55岁，统一之后再往60岁一步一步延长；男性在女性统一到55岁之后，也开始起步延长，当女性的退休年龄延迟到60岁，男性应该延迟到了62、63岁，然后再一起统一延迟到65岁。在保持‘小步慢走、渐进到位’的基础上，女性可以稍快一些，男性慢一些。”
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/14035051/DE21668D-FADA-4CA2-B185-F3183F095ED8.png)
真不知道这叫什么操作？是猪油蒙了心还是怕农奴翻了身？退休年龄还和发达国家去比，发达国家的工作强度、福利待遇和中共国一样吗？我们社会福利比的上发达国家吗？人均可支配收入比得上发达国家吗？发达国家失业不担心，看病不担心，住房不担心，生孩子养孩子教育孩子不担心， 我们却活的步步惊心的。这根本就没有可比性，中共这就是没有发达国家的命，却生了发达国家的病。就算你要延迟退休也要先保证就业啊，你能保证所有人到60岁都有工作吗？没有工作怎么保证生活，都去领低保吗？还是要百姓自己掏存款坐吃山空等退休？一边要做着996,35岁后随时下岗找不到工作的社畜；一边是65岁退休；一边还缴纳大量可能都用不到的五险一金，这不就是想白嫖百姓的养老保险吗？好家伙，这一进一出就省了10年的养老金。在我看来，这个政策就是中共体制内对体制外的剥削。体制内，50岁以后，非领导职务，基本就喝茶看报了。上班对他们来说就是福利。延长退休就是延长福利。体制外，35岁以后，非领导职务，基本就下岗了，要不就是到处打零工。并且现在很多中共企业，包括银行招保洁阿姨都不要50岁以上的了，你说一个60岁的老头能干嘛？社区看大门都不愿意要，并且沉重的体力工作已经在压榨他们的生命力了，还让他们延期退休，这不就是“钝刀子割肉”劫贫济富嘛。中共国人平均寿命70岁，实际上能有多少人活到70岁呢，一个人25岁参加工作到65岁退休交40年养老保险，然后能领10年都算运气不错，毕竟在中共国这种重度污染的环境下平均寿命75岁也算寿比南山了。在中共国养老金到底养得是什么人，大家心里都是非常清楚的，韭菜肯定不需要养，因为给点水它就发芽，整点肥料它就长，等着别人收割，割了几茬后，让你开花、结果、留种。前段时间刚刚说到医保单位缴纳的30%不再划入个人账户，现在又来个延迟退休，这真是不打算让百姓活了。从过去的“只生一个好，政府来养老”，到“只生一个好，政府帮养老”，再到“养老不能靠政府”，再到今天的“延迟退休好，自己来养老”，再到最后的“以房养老”。中共国人的养老，到底该靠谁？所以说在中共国靠养老金养老就是一个笑话。
