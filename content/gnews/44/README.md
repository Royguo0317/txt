# **即使中共国停电，仍有超过50艘澳大利亚运煤船在中国沿海搁浅**

[!\[\]()!\[\](https://spark.adobe.com/page/wyEC8K5CCFcyr/images/dbaa681c-a2a8-4874-91e5-8432bb96c1b1.jpg?asset_id=de1c0042-9f83-4a74-b7fe-0194f1164b0f&amp;img_etag=%22d19f6ae27d6f348531e0be3aa4c7b11b%22&amp;size=2560)](https://spark.adobe.com/page/wyEC8K5CCFcyr/images/dbaa681c-a2a8-4874-91e5-8432bb96c1b1.jpg?asset_id=de1c0042-9f83-4a74-b7fe-0194f1164b0f&amp;img_etag=%22d19f6ae27d6f348531e0be3aa4c7b11b%22&amp;size=1024)

有53艘船在海岸边等待了超过4周，而来自其他国家的运煤船都已经交货了。

中共国拒绝承认今年十月对澳大利亚煤炭禁运导致了当前中共国内的煤炭短缺。摄影：Wang Kai/美联社

尽管中共国今年面临煤炭短缺和近几年来最严重的停电，但仍有超过50艘澳大利亚运煤船因中共国政府的进口禁令搁浅在中国沿海。

中共国拒绝承认今年十月对澳大利亚煤炭的禁令导致了当前中共国内的煤炭短缺，但这一禁令与国内煤炭价格上涨有关。分析人士指出，现在的情况下，任何煤炭进口都会缓解压力。

数亿中国人民被勒令严格限制用电，包括限制工厂生产、禁止高层公寓使用电梯、不允许在温度高于3摄氏度时使用暖气。

据官方媒体报道，煤炭买家在煤场外面排长队装车，铁路公司也被要求优先运输煤炭。海关官员已经取消了一些港口对澳大利亚以外国家煤炭的进口配额。十一月，中共国政府批准了4个在新疆西部的新煤矿。

但是根据《卫报》从能源市场情报公司开普勒(Kpler)获取的数据，在中共国政府因贸易争端加剧于10月份突然禁止进口后，满载数亿吨澳大利亚煤炭的船只——有一些煤炭已经提前付款了——仍搁浅在海岸周围。

开普勒数据显示，53艘船只现在仍然在海上等待，已经超过4周，而所有从其他国家运煤的船只，除了一艘以外，都已交付货物并离开。

“12月，3艘在堵塞名单上的澳大利亚运煤船离开了中共国，前往其他港口，”开普勒称，“其中两艘从8月就开始等待，另一艘从10月开始等待”。

尽管堵塞现象在过去一个月得到改善，煤炭在交货前的平均等待时间仍然是2019和2018年同期的两倍左右。

《卫报》制图，来源开普勒

为了解释电力短缺，中共政府将矛头指向疫情后的工业用电增长、异常低温现象，并指出中国南方没有集中供暖的能源效率。中共国领导人习近平去年9月承诺，到2060年实现碳中和，此后中共国也在努力实现节能目标。

自12月初以来，浙江、湖南、江西和内蒙古自治区都实施了限电措施，因为这三省的用电量均同比增长。

直到年底，浙江省义乌市所有的路灯和广告牌都被关闭了。义乌被认为是世界上最大的批发市场，当地政府表示，义乌的企业被限制推迟开工时间。

义乌的企业向《卫报》证实，他们告诉客户，限电会造成延误。

一个义务商户说，他们希望客户理解订单延期。“通常许多物品都是机械包装的，现在义乌一直在断电”，商家表示说。在微信上，一家工厂的合伙人告诉批发商，他们已经买了手电筒，这样他们就可以继续发订单了。

据新华社报道，国家发改委表示，浙江不存在电力供应短缺，可以保证浙江省的需求，但是同时表示，一些地区已经采取措施，“以促进节能减排”。

为了达到减排目标，显然需要关停路灯和工厂用电，但当地居民对此表示怀疑。一位居民在微博上表示：“今年做生意本来就很困难，现在政府指标把老百姓压得喘不过气来。”
