香港媒体20201212讯，经丹麦到英国的香港前立法会议员许智峰接受采访表示，正在考虑转到其他五眼联盟国家，一同到达英国的有其父母、太太和女儿，他在与流亡到英国的罗冠聪会面后认为英国已有罗冠聪，为了把战线再拓宽，他未必会留在欧洲，而是会考虑前往其他五眼联盟国家，他做好用自己双手在国外谋生的身心准备，哪怕开UBer，送外卖，只要有手有脚就不怕饿死。同时许智峰认为香港人已经被《国安法》噤声，流亡海外的港人应为港人发声，他也计划在留英期间团结已移民英国的港人，定期聚会商讨，如何在海外支援香港民主运动，他呼吁大批已移民的港人，在海外为香港做事。
