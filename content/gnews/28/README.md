共和党控制的参议院周五投票推翻了川普总统对7400亿美元国防法案的否决。

川普总统上周三否决了7400亿美元的国防法案，因为该法案将以邦联军命名的基地重新命名，并且不允许将我们的军队从其他国家撤走。

川普还想增加一项条款，终止对科技巨头的第230条保护法案。

众议院周一晚间以压倒性的投票结果推翻了川普总统对国防法案的否决。

投票结果是322-87。

在罕见的元月一日会议上，参议院投票推翻了川普的否决。

投票结果为81-13。

这是美国历史上第112次否决权被推翻。
