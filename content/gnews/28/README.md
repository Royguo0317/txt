## 全球二次爆發，中國風景獨好？
`2020-10-01 07:00 Ronald` [轉載自GNews](https://gnews.org/zh-hant/394795/)

內新聞作者/素材：夢落欣生      校對：文跡～見證神蹟

據中國青年報報導，在世衛組織於9月28日舉行的新聞發布會上，世衛組織緊急情況計劃執行主任邁克·瑞安28日表示，ccp病毒導致的真實的感染和死亡數字很可能被低估了。

7月發表在《JAMA內科雜誌》上的一項研究發現，美國因新冠病毒死亡的人數可能被低估了28%。白宮曾經引用過的一個來自衛生計量與評估研究所的模型顯示，從現在預計到明年1月1日，將有371500名美國人死於新冠病毒。

在新聞發布會上，世衛組織總幹事譚德塞宣布達成一項協議，向全球發送1.2億個新冠病毒快速檢測（試劑），尤其是在中低收入國家。 9月29日，對於民眾所關心的第二波疫情，張文宏表示，秋冬季疫情反彈高峰已經在路上，只不過這是全球範圍內的反彈，不代表是在中國。張文宏認為本地疫情基本為零，即使世界疫情第二波爆發，中國基於一貫的防控策略，有能力在極短時間內控制少量的輸入病例。
