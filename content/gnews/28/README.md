## 無錫中級法院的幺蛾子
`2020-06-13 02:37 GM30` [轉載自GNews](https://gnews.org/zh-hant/232436/)

**作者：玉米地大姐**

無錫法院這幾天火了，網上公布16位法官親屬名單，鄭重做出承諾：在其擔任員額制法官及院領導期間，其配偶、子女及父母一律不得在無錫範圍內從事有償法律服務，如有違反從嚴查處，追究相應責任。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/13021552/2-99.jpg)

現代快報報道，公示的16名法官既有無錫中院副庭長，也有來自區法院的黨組成員、副院長、執行局長、庭長、審判員等。他們的配偶、子女從事執業律師、實習律師、兼職律師、基層法律服務工作者等情況。

6月10日下午，無錫中院煞有介事地就此公告發布情況說明。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/13021948/3-52.jpg)

說明寫道：落實法官任職回避制度，有利于促進司法廉潔，防止人情案、關系案、金錢案的發生，有利于推動人民法院反腐倡廉建設，維護司法公正，提升司法公信力。

今年4月最高人民法院出台《關于對配偶父母子女從事律師職業的法院領導幹部和審判執行人員實行任職回避的規定》，于5月6日起施行。

這一方面有利于促進法官更加自律、嚴格遵守職業紀律；另一方面也希望通過社會監督發現問題線索，對確有違反任職回避制度行爲的法官依法嚴厲查處，從而實現法院內部監督和社會外部監督的有機統一，推動法院領導幹部和審判執行人員任職回避制度得以真正貫徹執行，落地生根。

下一步，無錫法院將進一步加強內部監督，通過加強信息化建設，建立回避提示和預警系統，構建起有效的廉潔司法” 防火牆 “，實現違紀違法行爲的” 硬隔離 “；

另一方面，更加歡迎和誠心接受外部監督，認真受理信訪舉報，依法嚴肅查處違紀違法行爲，堅決懲治司法腐敗行爲，全力營造風清氣正的司法生態環境，努力讓人民群衆在每一個司法案件中感受到公平正義。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/13022011/4-36.jpg)

筆者初看新聞標題以爲無錫地區老百姓趕上這波好法官了，個個都敢亮“家底”。看罷內容原來不是主動發自內心的，是配合最高法5月6日出台的《關于對配偶父母子女從事律師職業的法院領導幹部和審判執行人員實行任職回避的規定》做做樣子而已。

網上那些贊美之聲，相信有一半是配合輿論宣傳的五毛在鼓噪點贊，另一半是不明真相的群衆，把16位法官們集體意淫成包青天了。

亮出家屬名單就能司法公正純屬天方夜譚忽悠二傻子，憲法還寫著言論自由呢，誰當真誰是棒槌。還有這個公告怎麽看怎麽有給法官及其家屬打廣告拉生意的嫌疑。有職務有電話，知道找誰好使管用，免去有錢找不到門路的麻煩。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/13021759/5-16.jpg)

舉個例子：法官的律師老婆不接本地案子，老婆的初戀男友說我這有個上億元的案子，官司打贏了，同桌的你拿六，我拿四。你啥都不幹，和你老公通融一下就OK。你說法官的老婆幹不幹？

不幹是傻子，就算這個法官的老婆不幹，誰敢保證其他法官家屬、親戚不接這個大單子？因爲明面上沒有違反承諾對不對？

再說，16位法官除了家屬、親戚，誰敢擔保沒有朋友、同學、情人、二奶私生子啥的？除非給每個法官身上裝上電子監控，24小時無死角地盯著他們的一舉一動，否則防不勝防啊！

退一萬步講，就算16法官及其家屬都能遵守承諾，如果主管16位法官的上級領導有什麽案子打個招呼關照一下，法官們怎麽辦？是遵守承諾啊還是領會領導的意圖啊？

一個司法不獨立的國度，大案講政治，小案靠關系，領導的想法就是法。16位法官吃喝拉撒都受制于他人，不尊重領導想法的法官，就“違法”了，還奢談什麽公平、公正？

最高法院官微公開聲稱：“始終把黨的政治建設擺在首位。”說白了這不就是黨大于法嘛！無錫中級法院鼓搗出的這個幺蛾子純屬自娛自樂，順便忽悠一下包青天的粉絲們。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/13022124/6-17.jpg)

中共盜國賊以黑治國、以假治國，以警治國，以貪反貪，無論他們鼓搗出什麽新鮮花樣來，萬變不離其中。共産黨不滅亡，中國沒有司法公平、公正可言。

最後應個景，配個老梗，和無錫中級法院公告比，這個才叫真實：

大蓋帽啊兩頭翹
吃完原告吃被告
原告被告都吃完
