川普向弗林和他的家人表示祝贺。“我知道你们现在将会有一个真正的美妙的感恩节！”

此前，据多篇报道称，川普告诉与之亲近者，他计划在离任前赦免弗林。

弗林是第二位在通俄调查中被定罪而后获得总统宽大处理的川普助手。川普在长期密友罗杰·斯通（Roger Stone）即将入狱报到的前几天，对他进行了减刑。这是（他）广泛尝试的一部分，以撤销多年来一直笼罩着对他（这届）政府的调查和对六七名合作人提出的刑事指控。

弗林在2017年初担任川普国家安全顾问23天，他两次承认向FBI探员撒谎，称他与俄罗斯前大使谢尔盖·基斯利亚克就奥巴马政府实施的与俄罗斯干预2016年大选有关的制裁进行了对话。

在花了三年时间起诉弗林的案件，并在1月建议将他判处6个月监禁后，司法部在总检察长威廉·巴尔（William Barr）的指示下，于5月突然宣布，“在对本案的所有事实和情况，包括新发现和披露的信息进行认真审查后”，计划撤销此案。

司法部得出的结论是，FBI对弗林的约谈是“与FBI对弗林先生的反间谍调查无关的，也不合理的”，而且约谈 “在没有任何合法调查依据下进行。”

此举引发了司法部可能受到政治影响的质疑。联邦法院法官埃米特·沙利文正在决定是否批准撤诉请求。

正当联邦法官在心存疑虑的权衡，是否批准司法部撤销起诉的请求，尽管弗林本人已承认向FBI撒谎与俄罗斯接触的罪名。此时的赦免令使得对弗林的刑事起诉失效。

在川普结束他单一任期之际，此举很可能会激发支持者们的热情，他们将此案作为一个热门话题，并聚集在这位退役陆军中将周围，坚称他是一起不公正起诉的受害者。川普本人曾多次热情地谈论弗林，尽管特别顾问罗伯特·穆勒（Robert Mueller）检察官对俄罗斯与2016年川普竞选活动之间的关联进行了调查，也曾经称赞弗林是一位模范合作者。

赦免是这一一波三折的案件的终了。去年司法部突然动议撤诉，坚持认为弗林本来就不应该接受FBI的询问，却被美国地区大法官埃米特·沙利文拒绝了这一请求，并任命一名前法官对联邦政府的立场进行反驳。

在此后的几个月里，一个由三名法官组成的小组命令沙利文撤销此案的决定被上诉法院全体法官推翻，此案回到沙利文处。在9月的一次听证会上，弗林律师西德尼·鲍威尔告诉法官，她曾与川普讨论过弗林案，但也表示不希望被特赦—大概是因为她希望他在法庭上得到平反。

即将离任的总统传统上在其任期的最后几天给予赦免和减刑。

川普此前表示，他在担任总统数周后解雇弗林是因为他对联邦调查局和副总统迈克·彭斯撒谎，但此后他又描绘弗林是一个无辜的人，被奥巴马政府官员陷害，决心 “扳倒一个总统”。弗林也被许多川普支持者和保守派媒体人士视为政治报复的受害者。

川普4月30日在推特上写道。“发生在战争英雄迈克尔·弗林将军身上的事情，绝不允许再发生在美国公民身上！”

弗林的律师是西德尼·鲍威尔（Sidney Powell），她曾是川普竞选活动法律团队的成员，指控广泛的欺诈行为让总统失去了连任的机会。鲍威尔在公开场合提出一些野心勃勃的说法后，竞选律师鲁迪·朱利安尼和珍娜·埃利斯周日发表声明，不赞成鲍威尔。

鲍威尔9月向沙利文承认，她和川普曾讨论过可能的特赦，但她说她要求总统不要介入。

如果沙利文最终决定拒绝司法部的撤诉请求，弗林有可能被判处任监禁。这一撤诉请求是在5月由总检察长威廉·巴尔特别任命的一名来自圣路易斯的联邦检察官对该案进行审查后提出的。而赦免使弗林免于任何可能性会被判入狱。
