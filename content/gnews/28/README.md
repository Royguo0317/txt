## 澳大利亚周一批准临时使用辉瑞疫苗

**西班牙2021年1月25日电/西喜社——**澳大利亚医疗监管机构已批准使用其首支中共病毒疫苗，为下月开始接种铺平了道路。

治疗商品管理局周一临时批准16岁及以上的人使用辉瑞和生物技术开发的疫苗。老年护理机构的居民和工作人员、前线医护人员和检疫人员是优先接种首批疫苗的群体。

澳大利亚总理斯科特-莫里森对这一进展表示欢迎。他说，澳大利亚是首批完成全面程序正式批准疫苗的国家之一，而不仅仅是给予紧急批准。

澳大利亚与辉瑞公司签订了1000万份两剂疫苗的协议，如果供应允许，还可以选择购买更多的疫苗。它还订购了5380万剂由阿斯利康公司和牛津大学生产的疫苗，条件是获得监管部门的批准，其中5000万剂将与墨尔本的生物制药公司CSL合作，在澳大利亚生产。

澳大利亚的目标是在10月前完成接种工作。这个拥有2600万人口的国家报告的病毒病例不到3万例，死亡人数略高于900人。
