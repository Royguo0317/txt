## 中共疫苗既不保证安全，也不确保有效，意义何在？
` wenwu` [轉載自GNews](https://gnews.org/zh-hans/717145/)

喜马拉雅-国内新闻组/素材：小黑姐（文雅） 封面：麦田67号 校对：圣斗士(沙加)

新华社北京1月4日电，截至4日10时，全国共有45个地区调整为中风险地区。北京市顺义区、河北省石家庄市进入“战时状态”，大连出现超级传播现象。国务院联防联控机制综合组最新发布《关于做好2021年元旦和春节期间新冠肺炎疫情防控工作的通知》，提倡家庭私人聚会聚餐控制在10人以下，50人以上的活动应当制定防控方案，落实重点人群在春节前完成新冠肺炎疫苗接种任务……

近日中共党媒加紧对新冠疫苗接种进行宣传造势，许多不明就里的民众跃跃欲试，认为打了疫苗就能远离新冠的毒害和困扰，但事实真是这样吗？

一、接种人群限制

党媒央视新闻在1月3日发布的『新冠病毒疫苗接种须知』中明确指出适合接种和不适合接种的人群，目前疫苗接种的年龄限制为18至59岁，且孕妇、哺乳期妇女、药物不可控的高血压、糖尿病并发症人员不适合接种。此外，除了以上限制外，不适合接种疫苗的人群还包括正处在发烧、感染等疾病急性期、患免疫缺陷或免疫紊乱的人群以及严重的肝肾疾病、恶性肿瘤患者等等。

根据新华社2020年1月17日的一篇报道显示，16至59周岁的劳动年龄人口有89640万人，占总人口比重64%。这意味着会有36%的人口被排除在接种计划之外，而老年人又是病毒高发人群。且不说中共的疫苗是否安全有效，但如此设定已是匪夷所思。

二、接种后有效性无法保障

根据党媒报道，由于新冠病毒发现的时间不到一年，无法保证这个疫苗能保护多长时间，按目前证据提示保半年没问题，但目前还没有可靠的证据表明可以保半年，并且还声明就算少数人接种后仍然可能没有免疫力或者发病，仍然要佩戴口罩、洗手、保持社交距离等。
![]()![](https://gnews.org/wp-content/uploads/2021/01/image0-1-2.png)
三、 疫苗的安全性无法保证

2020年11月初，巴西志愿者因注射中共国疫苗死亡，巴西暂停了中国疫苗实验。12月12日，秘鲁宣布，出于谨慎理由，暂停中国疫苗临床实验。由于党媒和各大主流媒体隐瞒真相，普通民众无法获知准确信息，中共国国产疫苗因为没有完成第三期实验，加之没有公布完整数据，一致引起外界对其疫苗安全性和有效性的质疑。

这样一个既不能保证安全，也无法确保有效的疫苗，接种的意义何在？
