10.天文學家在早期宇宙中發現了我們銀河系星系的首次相似。

檔案照片:在這張圖片中,NASA / ESA哈勃太空望遠鏡瞥見了無數臂狀結構,這些結構圍繞著這個被禁止的旋渦星系(NGC 2608)掃過。 看上去像是我們銀河系的略微拉伸,較小的版本。

(歐空局/哈勃/美國宇航局，,到|。裡斯 \*)   。

1920年代初,著名的天文學家埃德溫· 。 哈勃(Edwin Hubble)發現的我們的銀河系不是宇宙中唯一的銀河系。 在隨後的幾十年中,我們意識到實際上我們的宇宙中有無數的星系。 現在,天文學家們第一次發現了和我們銀河系十分相似的星系。

據估計,新發現的星系SPT0418-47距我們的星系約120億光年,這意味著到目前為止,它的光需要近120億年才能傳播並到達我們。 新發現的星系是在宇宙僅佔其當前年齡的10%(約14億年)時形成的。

這個星系包括氣體,塵埃和數十億顆恆星,所有這些恆星都通過重力聚集在一起。 銀河系擁有135.1億年的歷史,它擁有整個太陽系和一些太陽系,並且呈螺旋狀出現。

迄今為止,天文學家一直認為,在宇宙的早期,形成的大多數星系都是不穩定的。 與此信念相反,這個新發現的星系「令人驚訝地穩定」。 馬克斯· 。 普朗克研究所的弗朗西斯卡· 。 里佐(Francesca Rizzo)說:「這一結果代表了銀河系形成領域的一項突破,表明在我們附近的旋渦星系和銀河系中觀察到的結構已經存在了120億年前。  ”
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/16102042/image-97.png)
SPT0418-47星系在天空中以近乎完美的光環出現。

(阿爾瑪 (ESO/ NAOJ / NRAO)裡佐 。)

根據聲明,遙遠的星系SPT0418-47在被捕獲時,表現為附近星系周圍近乎完美的光環。

遙遠星系的更詳細資訊和更多特徵尚待確定,因為它位於宇宙很遠的地方。 因此,即使是最強大的望遠鏡也無法識別星系,因為它看起來很小且很微弱。

12.科學家稱冠狀病毒至少與1918年流感大流行一樣致命。

研究人員在醫學雜誌《 JAMA Network Open》週四發表的一項研究中警告說,冠狀病毒至少與1918年的流感大流行一樣致命,如果世界領導人和公共衛生官員未能適當控制它,死亡人數甚至可能更加嚴重。

主要作者傑里米· 。 浮士德博士在接受採訪時說:「我們希望人們知道這具有1918年的威力。  “他補充說,紐約的爆發至少比1918年的爆發更嚴重,當時醫生沒有使用呼吸機或其他先進設備來挽救生命,就像今天一樣。  「這不是像流感那樣可以輕鬆而過的東西。  ”

美國紅十字會的成員在**1918**年將流感受害者轉移。

研究人員比較了1918年大流行高峰期間紐約市與Covid-19爆發前幾個月的死亡人數。 他們使用了疾病控制與預防中心,紐約市衛生和心理衛生部門以及美國人口普查局的公開數據進行分析。

研究人員發現,在1918年流感大流行期間,死亡人數的增加總體上較高,但與紐約市冠狀病毒爆發的頭兩個月所觀察到的情況相當。 研究人員寫道,考慮到衛生,現代醫學和公共衛生方面的改善,冠狀病毒早期爆發期間的增長”大大大於”  1918年大流行高峰時期的增長。

美國在周三記錄了由Covid-19造成的1500多人死亡,這是該國自5月底以來最致命的一天。

7月1日發表在 《JAMA Internal Medicine》上的另一項研究 發現,確診的美國冠狀病毒死亡人數大大低於真實人數。

這些研究人員發現,超過正常水準的死亡人數也超過了歸因於Covid-19的死亡人數,這使他們得出結論,其中許多死亡可能是由冠狀病毒引起的,但尚未得到證實。 他們說,州報告的差異和大流行中美國死亡人數的急劇增加表明,Covid-19死亡人數被低估了。

世界衛生組織說,這種病毒沒有「靈丹妙藥」,醫護人員可能需要採取一系列治療措施來説明患者抵抗疾病。 當前,美國許多醫院都在使用抗病毒藥物瑞德西韋,該藥物已被證明可以幫助縮短部分住院患者的康復時間。 據世衛組織稱,也有許多疫苗正在研發中,至少有26種已經在人體試驗中。

CDC的數據顯示,公共衛生官員和傳染病專家經常將Covid-19與1918年的流感相提並論。 據估計,從1918年到1919年,流感在全世界殺死了5000萬人,其中包括67.5萬人。 相比之下,第一次世界大戰中有超過2000萬人死亡。
