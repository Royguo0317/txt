## 加拿大运动员入籍中共国出赛东京奥运会

加拿大《国家邮报》4月1日报道加拿大运动员尼娜·舒尔茨（Nina Schultz）已入籍中共国，并将代表中共参加东京奥运会。
![]()![](https://gnews.org/wp-content/uploads/2021/04/glaciermedia.ca-2.jpg)*图片来源：glaciermedia.ca*
译文如下：

当尼娜·舒尔茨（Nina Schultz）只有19岁时，全世界就已经意识到她在七项全能项目上的竞争力了。

这位出生在加拿大卑诗省新威斯敏斯特的后起之秀为加拿大获得了上届英联邦运动会的银牌，仅落后于英国的卡塔琳娜·约翰逊·汤普森（Katarina Johnson-Thompson），后者现已成为该项目世界排名第一。

三年后的今天，舒尔茨已经不是加拿大田径界的代表着。在中加两国之间关系紧张之际，她已成为中共国公民，并希望在2021年的东京夏季奥运会上为这个刚刚接纳她的国家出场比赛。至于这种身份的转变，舒尔茨曾说过，主要出于个人原因，这似乎早于当前的中加关系动荡。尽管如此，仍有一些对北京政权的批评者质疑她的选择。

她的外婆曾是中共国的跳高世界纪录保持着，但由于当时中共国正在进行抵制奥运会而无法参赛。她的外公也是中共国本国的跳高纪录保持者。他们的外孙女早在2017年就表示希望实现外祖母当时无法实现的目标。

加拿大田径队主教练格伦罗伊·吉尔伯特（Glenroy Gilbert）说，由于种种原因，舒尔茨在过去三年中几乎没有参加比赛，但她肯定足够的潜力赢得奥运会奖牌。
“显然，我们将她视为七项全能项目的后起之秀”，奥运金牌得主吉尔伯特说，“我们非常希望也尽了一切努力让她留在加拿大……但她对自己的目标非常明确。”

话虽如此，加拿大田径队的运动员和这位现中共国籍运动员之间并没有任何不适的感觉。无论是在卑诗省新威斯敏斯特出生的舒尔茨本人还是她的家人，都拒绝对此置评。她的母亲是上世纪90年代从中共国移民加拿大的，而他的父亲则出生于加拿大。

上个月，当她出现在中共国山东省的一次田径比赛中时，中共国媒体报道了她对中共国的忠诚。 中国共产党经营的《环球时报》援引她的话说：“我希望能在东京奥运会上获得七项全能奖牌。”

世界田径网站（World Athletics）网站确认，她的申请在经过理事机构要求的静默期后已经改变，她可以从2021年4月开始代表中共国参加国际比赛。

鉴于她的外祖父母都是中共国体育界的杰出代表，舒尔茨很可能是有意或无意间成为了中共国统一战线部门（UFWD）的目标，该部门一直致力于影响生活在其他国家的华人。加拿大香港友协的Ivy Li说：“拥有像她这样的家庭背景……她毫无疑问会受到中共国统战系统的监控。”李最近组织了一份请愿书，要求抵制即将在北京举行的2022年冬季奥运会。 “如果她的祖父母仍生活在中共国，那就说明了更多。毫无疑问，这个加拿大家庭会受到密切关注。”

李认为这些年轻的精英运动员一般都会高度关注自己的运动，所以舒尔茨可能没有时间或兴趣来了解中共国的侵犯人权行为。

但是她补充说：“如果这位年轻女士有机会阅读这些报道，尤其是有关新疆维吾尔族妇女遭受的系统性强奸和酷刑的报道，我想知道，她仍然想代表中共国吗？”

在连续打破了加拿大青少年记录后，舒尔茨获得了堪萨斯州立大学的体育奖学金。 在她的第一次NCAA比赛中，她赢得了七项全能比赛，并创造了该项目加拿大20岁以下新纪录，该成绩在当年七项全能项目上名列世界第三。

随后，她在2018年赢得了英联邦运动会银牌。那一年，来自奥林匹克频道网站的一篇文章将舒尔茨列为当年田径赛场上“十大最值得关注的运动员”之一。但是即使在那个时候，舒尔茨已经表明她可能不会在未来效力加拿大。

不同寻常的是，她早在2017年即以华人华侨身份参加了中共国全运会，并用“流利的普通话”告诉当地记者，她想代表中共国参加奥运会。

中共国新华社引述她的话说：“我来这里并不是出于一时冲动，而是出于我一直想实现我外祖母的梦想。”

郑凤荣是中共国第一位创造世界纪录的女子运动员，她在1957年以1.77米的高度打破了当时的世界纪录。但是，由于中共国在当年因与台湾当局的政治纷争而抵制奥运会，郑凤荣便再也没有机会在奥运赛场争夺奖牌。

舒尔茨的决定似乎早于加拿大于2018年12月逮捕华为高管孟晚舟，该事件引发了两国之间的冲突。据世界田径运动会的记录，在2019年6月，舒尔茨的注册转移已经被正式批准。

目前尚不清楚舒尔茨是否还保留她的加拿大护照，但中共国法律通常不允许双重国籍。前任加拿大驻北京外交官查尔斯·伯顿（Charles Burton）表示，舒尔茨此时为中共国效力的决定“令人不安”。麦克唐纳德·劳里尔学院研究员说，自幼成长在在卑诗省的舒尔茨似乎并没有与中共国的牢固而持久的联系”。

无论如何，在将近三年没有参加正式比赛的情况下，舒尔茨首先应该努力获得夏季奥运会的参赛资格。这位七项全能运动员于2019年从堪萨斯州立大学转到佐治亚大学并有一年时间没有参加任何比赛。由于新冠疫情，2020年的 NCAA赛季也被取消了。

根据世界田径协会规则，改变效力国家的运动员必须在三年内退出国际比赛。舒尔茨实际上是一年多前离开佐治亚大学的，该校的首席田径教练佩特罗斯·基普里亚努说。

他在电子邮件中说：“她只是想离开NCAA并成为一名职业运动员。” “这是我知道的全部。”
