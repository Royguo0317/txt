## 菜比肉贵，沉重的菜篮子

香草山写作组  文驹



近来，墙内居民在受到新冠疫情肆虐的痛苦后，迎来新一轮民生难题，全国范围内出现了蔬菜价格暴涨的局面，甚至出现了菜比肉贵的现象。对绝大多数的老百姓而言，这沉重的菜篮子，给本来艰辛的生活更添沉重的压力。

我们从中国地图上选择几个不同地域的城市，看看菜价到底涨到几何。

以疫情严重的华北石家庄为例，河北官媒1月8日曾报导，“疫情防控期间，北国超市、永辉超市、苏宁家乐福超市等70家连锁超市门店共同承诺，从1月8日起至1月底，大白菜、土豆、洋葱三种耐储蔬菜实行惠民价格，设立专柜，并且在价格不涨的基础上，同时做到供应充足。” 1月10日举行的河北省疫情防控发布会上，河北省市场监督管理局局长金洪钧还宣称，主要生活必需品价格平稳。

但是市民感受到的却是另一种现实。大纪元报道，1月10日，网曝石家庄同祥城小区居民称，2斤韭菜、2斤黄瓜，1袋饼丝、1袋面条要卖105元。
![]()![](https://gnews.org/wp-content/uploads/2021/01/2101194.png)
网友花燃山…：河北我们这里青辣椒12一斤，杭椒15….生菜都卖到了十块一斤！香菜15一斤！生姜18一斤！大葱五块钱一斤，鸡蛋五块五，蒜黄什么的更吃不起。

“刚迪cp”也质疑当局宣称的“生活必需品价格平稳”说法，斥“蔬菜、蛋都涨了很多，为什么还在说平稳呢？在欺骗谁？ ”  还有人说：“召开发布会说这个话“菜价不涨”的人没用，说菜价不变的没有用，这话不是老百姓说的，你看看底下的评论，所有人都在评论菜价涨了，这就不是一个个例了。这就是明显的吹牛呢，或者是说这个话的已经被假象蒙住了双眼 。”

我们再看看首都北京，天子脚下的菜价几何。一颗白菜12.39元，几颗青椒不到一斤14.21元，一颗一斤半的萝卜3.32元。 一位青岛市农产品批发市场商户透露，尖椒平时一斤3元多，现在一斤9元多。而大葱的批发价一个月前是2-3元，现在涨到了6-6.5元。
![]()![](https://gnews.org/wp-content/uploads/2021/01/2101195.png)![]()![](https://gnews.org/wp-content/uploads/2021/01/2101196-1.png)![]()![](https://gnews.org/wp-content/uploads/2021/01/2101197.png)
视线在转向东北，据头条新闻报导， 黑龙江齐齐哈尔杨女士1月12日在超市买菜，还没买完菜，老板就换标签涨价，油菜3块多钱立马涨到4块多钱一斤。

沈阳网友‘小荷漫纸鸢V’16日刚发帖称，‘物价已经涨的越来越过分了，大白菜超一块五一斤，土豆一块三，萝卜三块，鸡蛋快六块了 ，真的快要活不下去了…’ 一天后17日再发帖称：‘昨天的物价，今天已全涨，大白菜2块， 土豆2块，逼着死吗？ ’
![]()![](https://gnews.org/wp-content/uploads/2021/01/2101198.png)
接下来看看中部江苏连云港的情况，据中国江苏网报道，记者在市区连续走访了多家农贸市场和连锁超市，普通的青椒都卖到12元/斤，小米椒更是卖到28元/斤，价格直逼猪肉，让很多市民大呼吃不起。通灌路一家连锁超市里，最便宜的黄彩椒零售价7.98元/斤，青尖椒10.98元/斤，青线椒11.80元/斤，小红米椒28元/斤。相比之下，干辣椒相对便宜一点，其中辣椒丝32元/斤，小红米椒干36元/斤。
![]()![](https://gnews.org/wp-content/uploads/2021/01/2101199-2.png)
再看看南方的厦门，据东南网报道称，厦门气温一再刷新人们想象的同时，厦门菜价却是一片热火劲头。家住五缘湾的郑阿姨，因为两颗辣椒和菜贩理论起来。“两颗手指大小的红辣椒，居然2.4元，以前一两颗辣椒根本不过秤，都是送的呀。”菜贩也很委屈，“真是没法送啊,现在小米椒30元一斤、大黄椒25元一斤，最普通的青辣椒也要七八块一斤”。

与菜价起飞的，还有“八戒”。郑阿姨说：“之前排骨已经降回到40元以内了，但这几天好像又回来了，你看我买这么一点就要60多元。” 思明区营平市场上，肉价连续两个月走高，精瘦肉每斤价格从24元飙涨至27元，排骨则从38元涨到42元，涨幅均超过一成；而在松柏市场，肋条肉从26.9元涨到30.9元。

我们再看看物产丰富的天府之国四川，成都蒜苔价格上涨，曾最高达到50元/斤，单价堪比牛肉，超过猪肉。蒜苔是四川人民餐桌的家常蔬菜，蒜苔回锅肉是四川非常有名的特色菜。而今大量普遍种植的蒜苔价格如同坐上了火箭，真实令人望而生畏。蒜苔的上涨是蔬菜整体上涨的一个缩影，其他蔬菜也普票上涨。成都网友“-ChiaMo…”说：“成都已经算农产品丰富自产自销的地方了吧，就瞅了一眼家附近菜场：鸡蛋8元/斤；菠菜16元/斤；鸡胸肉20元/斤；大白菜22元/颗；但最震撼的还是豌豆尖，25元/斤！”
![]()![](https://gnews.org/wp-content/uploads/2021/01/21011910-2.png)
据内媒报道，成都某蔬菜公司领导宋国明就蒜苔涨价称，“其实蒜苔涨价对于成都市民来说影响不大，很多人一年也吃不了几次。”任伟说，综合来看，每到年底蒜苔涨价是正常的市场行为，若消费者确有需要，可选择购买冻蒜苔或其他替代品。

可见为中共洗地的言论是多么荒谬与无耻。就本轮全国范围内的蔬菜涨价，还有更为无耻的国家级言论。央行副行长陈雨露1月15日在国新办新闻发布会上表示，总体来看，2021年我国的物价水平大概率会保持温和上涨

网友“雨晨漫路”斥：别闹了，别玩文字游戏了。老百姓的日子已经够艰难了。大白菜2.5/斤，大葱8元/斤，菠菜4元/斤，还提肉类吗？

“粉红-DSB”：温和是多少？别他妈跟老百姓玩文字游戏，上涨就上涨，你们就抓紧时间印钱吧，让老百姓手里的钱贬值了目的就达到了！

“鲁鲁之家2013”：发炎人，先温和地煽自己俩嘴巴，一月以来向钱葱领军，鸡蛋已经涨到鸡屁股疼，温和？

网友“落日西凉”对于官方的说法作出解读：‘失业叫下岗，通货膨胀叫温和上涨，金融危机叫经济下行………老配方老味道，专业修辞你懂的！’

“拾了一条大鱼”呼吁：给条活路吧！这两年的经济状况， 还涨呢！封建社会吸血都没这么狠？

通过梳理全国范围东西南北中不同城市的蔬菜暴涨的情况，清晰感受到老百姓菜篮子的沉重，民生的艰难。在大陆企业倒闭、病毒横行、物价飞涨的情况下，中共官方依旧用官话、套话、谎言欺瞒民众，老百姓真是水深火热之中苟延残喘。看着蔬菜价格上涨如此凶猛，老百姓连菜都快买不起的情况下，居然有所谓的专家还厚着脸皮解释是天气变冷的原因，疫情运输原因云云，真是令人出奇的愤怒。笔者不是“专家”，但是笔者知道，没有哪年冬天是不冷的，没有哪年菜价肉价涨得这么不靠谱的。笔者还知道，这蔬菜涨价折射出来的是中国的经济正在崩溃的问题，人民币快成废纸的问题，中共马上要完蛋的问题。
