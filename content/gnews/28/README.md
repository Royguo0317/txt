**西班牙2021年4月7日电/西喜社——**

据ABC新闻4月7日报道，欧洲药品管理局(EMA)将在 “周三晚些时候 “就阿斯利康公司的中共病毒疫苗与部分疫苗接种者的血栓形成之间的联系发表新的意见。

欧盟卫生部长将于本周三下午6点紧急召开会议，分析阿斯利康公司研发的中共病毒疫苗，以便欧洲药品管理局(EMA)发布新的报告，来说明这种疫苗与血栓出现的关系。

周二，EMA疫苗接种策略负责人马可-卡瓦莱里（Marco Cavaleri）证实，阿斯利康的疫苗与一些疫苗接种者正在经历的血栓事件之间存在 “关联”，主要是50岁以下血小板缺乏的女性。

下午，欧洲药品管理局得出结论，应将血栓作为阿斯利康疫苗的一种非常罕见的副作用。

欧洲药品管理局确认，疫苗的效益风险比仍为正值。即接种疫苗的好处大于它带来的风险。 EMA负责人埃默-库克（Emer Cook）在新闻发布会上表示，疫苗的接种显然是“有用的“。而EMA风险评估委员会负责人萨宾-施特劳斯在新闻发布会上表示，该疫苗 “对中共病毒有效”，风险非常罕见，约为十万分之一。

同时，英国药品和保健品监管局（MHRA）表示，将为30岁以下的人提供阿斯利康疫苗的替代方案，但是阿斯利康疫苗的好处仍然大于风险。

欧洲药品管理局在接受记者提问时还表示，除了俄罗斯的 “卫星五型 “外，还有几种疫苗正在进行评估，并表示不知道哪种疫苗会最先被批准。
