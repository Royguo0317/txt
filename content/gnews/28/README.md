- **05.02. Zerohedge [SpaceX太空舱成功将4名宇航员从国际空间站送回，这是1968年以来的首次夜间溅落 SpaceX Capsule Successfully Returns 4 Astronauts From ISS In First Nighttime Splashdown Since 1968](https://www.zerohedge.com/markets/spacex-capsule-successfully-returns-4-astronauts-iss-first-nighttime-splashdown-1968)**
四名宇航员周日清晨从国际空间站成功返回地球，他们在墨西哥湾进行了 “溅落”。
据《华尔街日报》报道，这些宇航员乘坐SpaceX的太空舱返回，标志着NASA “几十年来 “首次在夜间返回地球。
Crew Dragon Resilience在周日凌晨2:57进行了 “软溅落”，然后被吊入佛罗里达州巴拿马城附近的一艘回收船。宇航员在太空中停留了168天，完成了SpaceX的首次 “业务往返任务”。
乘员组由美国国家航空航天局宇航员迈克尔-霍普金斯、维克多-格洛弗和香农-沃克以及日本宇宙航空研究开发机构的野口宗一组成，去年被带入轨道，然后与国际空间站对接。
据报道，美国国家航空航天局和SpaceX公司正在相互进入一种操作 “节奏”，四名宇航员的返回标志着这两个机构一系列里程碑中的最新事件。据《日报》报道，这两个机构 “定期安排人类穿梭于太空，使用该公司的商业化火箭和太空舱”。
另一个四人小组也在国际空间站，开始他们的任务，在上个月被SpaceX的火箭发射到轨道上。
报道称，SpaceX打算在未来15个月内为美国国家航空航天局发射7个太空舱，包括3个货运舱。
阿波罗8号是最后一次夜间溅落，它发生在1968年任务绕月之后。

- **05.02. Newsmax [亨利-基辛格警告说，美中两国将出现大决战 Henry Kissinger Warns of US-China Armageddon](https://www.newsmax.com/newsfront/henrykissinger-nuclear-weapons-world/2021/05/02/id/1019817/)**
前国务卿亨利-基辛格曾经领导了将中国带入世界经济以可能传播民主影响的努力，现在他警告说，这两个世界大国之间可能发生核战争和高科技战争。
“在人类历史上第一次，人类有能力在有限的时间内消灭自己，”97岁的基辛格周五在麦凯恩研究所2021年塞多纳论坛上对前参议员乔-利伯曼（Joe Lieberman）说。”我们已经开发了一种力量的技术，甚至超出了70年前任何人的想象。
“而现在，在核问题上又加上了高科技问题，在人工智能领域，其本质是基于人成为机器的伙伴，机器可以发展自己的判断力。”所以在高科技大国之间的军事冲突中，它具有巨大的意义”。
在川普政府就中国的人权和贸易侵权行为向其施压后，拜登政府寻求在推动民主与与中国紧张关系升级的威胁之间取得平衡，基辛格认为中国是美国和这个词的 “最大问题”。
“基辛格说：”如果我们不能解决这个问题，那么风险是全世界都会在中国和美国之间形成一种冷战。
与中国相比，美国和苏联之间的冷战差异就显得微不足道了，因为后者拥有经济实力。”苏联没有经济能力，”他说。”他们有军事技术能力。”[他们]没有像中国那样的发展性技术能力。中国除了是一个重要的军事大国之外，还是一个巨大的经济大国。”
基辛格说，美国的对华政策必须采取双管齐下的办法：坚守美国的原则，要求中国尊重，同时保持不断的对话，寻找合作的领域。他说：”我并不是说外交手段总是会带来有益的结果。”这是我们的复杂任务，”他继续说，并承认，”没有人完全成功地做到这一点。”
