**西班牙2021年3月4日电/西喜社——**
![]()![](https://gnews.org/wp-content/uploads/2021/03/妇女节图片.png)
据国家报3月4日报道，2020年的 “3月8日妇女节 “大游行，在整个大疫之年，一直备受争议。就在封城前一周举行，当时意大利已经爆发并开始局部封城，西班牙也已经有300多例，却举行了超过10万人的游行，之后两天，感染病毒人数直接翻了2倍，并一路直上。这次游行也掀起了一场政治和司法风暴。马德里政府因在CCP病毒可能有传播风险之前，没有禁止当时的三八妇女大游行而受到法官的调查。此案已由马德里法院立案。

今年，马德里大区政府代表处，在周四决定禁止所有将于3月7日（星期日）和3月8日（星期一）国际妇女节举行的游行活动。代表何塞-曼努埃尔-佛朗哥在 “详细研究 “了每一项请求后，以 “公共卫生原因 “为由提出了禁令。”如果预期的所有与会者都参与游行，届时将有超过6万人，在马德里的不同街道上移动。”该代表在新闻发布会上解释说，此决定是基于马德里地区的14天内累计发病率为每10万居民261人，仍然处于极端风险的第四级。

何塞指出，马德里大区疫情数据属于整个西班牙最糟糕的几个大区之一，住院率超过14%，重症监护病房（ICU）的情况比10月份第二波大流行期间的最高值还要高，入住率超过40%。此外，至今为止，一些“三八妇女节平台“ （一些为争取女权的组织建立的平台，以追求女权为口号，已成为西班牙一支重要的政治力量）下的工会和协会，已经在马德里地区举行了104次游行活动，作为替代方案， “他们的正义愿望和要求 “已经得到了满足。他还强调，这一决定是对警方和医疗组织的报告作出的回应。

之前，政府的立场是“断然拒绝“ 类似去年那样的大规模游行，但允许少于500人的团体上街游行，并保持必要的安全距离。到目前为止，已经收到了45个游行的请求，大部分在马德里市区内。

作为卫生部的立场，部长卡洛琳娜-达里亚斯指出：”由于我们仍处于极端危险的情况下，没有任何商量的空间”。
