2020年4月，新浪微博上出现了投资大师沃伦·巴菲特（Warren Buffett）的一幅照片，他戴着比亚迪口罩，照片中写着“我的BYD口罩正在帮助我实现自己的目标”，身着的广告衫写着“我想长命百岁，目前还不错”。没有搞错？巴菲特在为一家诞生仅仅两个月的公司大打广告！
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/28090043/Picture28-3.png)照片来源：新浪微博
这家公司就是比亚迪生产医护用品分公司，根据中国企业记录，该公司2月7日在中国政府注册，开始“生产和销售医疗设备和工业防护设备”，没有生产个人防护设备的历史，但是在获得FDA批准几天后，公司就获得了一笔10亿美元的交易，向加利福尼亚供应口罩，加州州长加文·纽瑟姆（Gavin Newsom）签了这份口罩大单，汇款后但从未收到口罩，记者怀疑州官员以购买“疫情用品”为掩护，向中国大量汇款，然后中国给予财务回扣。（Vice News   12/04/2020报道）

迄今为止，加州政府尚未作出公开回复，比亚迪公司开始起诉Vice News，吃瓜群众甚是惊奇，小小口罩缘何有魔力，居然令如此多的大人物牵涉其中？
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/28090124/Picture29-3.png)加州州长纽瑟姆   (图片来源：Los Angels Times)
第一主角无疑是加州州长纽瑟姆，宣布这笔交易仅两周后，纽瑟姆就具体细节仍然保持沉默。加州民主党的顾问拒绝[了《洛杉矶时报》](https://www.latimes.com/california/story/2020-04-20/gavin-newsom-n95-masks-byd-chinese-company-california-legislature)对该协议的置评请求；据《旧金山纪事报》报道，加州参议院和众议院也一直对这笔交易讳莫如深；纽瑟姆政府官员上周四在萨克拉曼多举行的预算监督听证会上拒绝向州参议员提供合同副本，尽管该州已经支付了一半费用；该州紧急事务办公室首席副主任通过电视会议告诉参议员，提供该交易的所有细节可能会破坏该州口罩的供应线。（纽约邮报报道）

该州长匆忙与比亚迪下属一家新公司签下大单，顶着压力不回应、不公开合同细节，有什么不可告人的内幕？

第二主角无疑是比亚迪医护用品公司。据Vice新闻报道，这家2月匆忙注册成立的新公司，通过中间人花了2万美元拿到了FDA许可，4月份便接了加州口罩大单。比亚迪不同凡响，需要什么样的人脉资源才能搞得定腐朽没落的美国地方官僚？或许比亚迪美国公司发挥了作用。

Vice新闻继续报道，比亚迪兰开斯特工厂（生产电动车美国公司）建成后，企业一直在州政府运作，陆续花了100多万美元，后来又捐了5万美元，促使加州2018年通过了汽车燃料消费税法案，汽油税从每加仑34.7美分增加到每加仑47.7美分，柴油税从每加仑34美分增加到67美分。

如果仅仅是比亚迪美国公司游说运作，让加州政府出台有利于其行业的政策可以理解，然而让政府掏出真金白银购买一种通货（口罩），而且是大单给一家新公司就存疑了。我们不是阴谋论者，但是从常理出发，如果没有特殊的关系或者利益，比亚迪口罩工厂是拿不到订单的。

此时关键人物——股神巴菲特出场了，这位坊间流传的世界顶级投资大师，中国的很多新贵都排着队，期待和他共进一顿几百万美元的午餐。巴菲特怎么会给比亚迪口罩免费打广告？ 10月30日在cnTech Post一篇文章 “巴菲特为什么要投资比亚迪而不是特斯拉？” 可以为我们带来一些启迪。

2008年巴菲特的合伙人查理·芒格建议巴菲特与王传福会晤，同年9月，巴菲特以8港元的价格收购了2.25亿股比亚迪股份，耗资18亿港元。此文重要提示，巴菲特合伙人“芒格并不只是看重比亚迪在2008年的前景，而是芒格了解比亚迪的布局以及中国政府推动新能源产业前进的决心”。2009年1月，中国财政部发布了《关于实施节能与新能源汽车示范推广的通知》，新能源汽车补贴开始实施，比亚迪的股价开始飙升。王传福一度成为《胡润中国百富榜》上 “中国首富”。而巴菲特投资的股权目前的市值高达352亿港元，巴菲特在12年内赚了330亿港元，是原始投资的18.6倍。

相信大家都看懂了，所谓投资大师巴菲特，也是代表华尔街为红色资本抬轿子事先做局，随即党妈配合出台政策利好，故双方一拍即合，政策出台后大家一起分赃割韭菜。当前比亚迪十大股东里，有了港交所、中央汇金、香港中央结算公司等，比亚迪是一家党企（王传福只是白手套）。资料显示，虽然特斯拉总回报远远超过比亚迪，因为有中国政府背书（比亚迪一直是电动汽车行业中政府补贴最高的中国公司），比亚迪是稳赚不赔，巴菲特当然选择比亚迪（以下是两家公司股票走势对比图）。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/28090512/Screen-Shot-2020-11-29-at-12.04.55-am.png)图片来源：证券网站
巴菲特与中共权贵走得很近，是否还是“中国人民的老朋友”尚不得而知。2010年5月11日他的合伙人查理·芒格（Charlie Munger）接受Money.CNN.Com采访，认为中国的共产主义对商业有利；5月的中国日报也刊登文章，“巴菲特，芒格赞扬中国的制度，领导能力”。现在可以确认了，巴菲特就是华尔街投资中共国的一位代表。

再来聊一下巴菲特的政治倾向，左媒CNBC报道，“作为来自内布拉斯加州的保守派共和党议员的儿子，他最终确实在政治上与爸爸决裂。巴菲特长期以来一直是自由派事业和候选人的支持者，至少可以追溯到民权时代，包括对美国总统奥巴马和希拉里-克林顿竞选活动的公开和财政支持”。左媒的观点直接证明了巴菲特是民主党的铁粉，而加州是民主党的长期经营的票仓，巴菲特下属公司在加州有巨额的投资。

2013年起沃伦•巴菲特的中美能源控股公司以20亿至25亿美元的价格收购了加利福尼亚州的两个SunPower太阳能光伏电站项目；2016年5月，伯克希尔·哈撒韦公司（Berkshire Hathaway）开始在苹果公司建立一个股份持有人，如今该股份持有量已超过2.5亿股，约占苹果流通股的5.8％。巴菲特在加州还有其它投资项目。
现任加州州长纽瑟姆是民主党成员，自2019年1月起担任加利福尼亚州第40名州长，2011年至2019年担任副州长。在州长岗位时间较长，诸多方面与巴菲特有交集，媒体记载了近两年发生的两件事情：

2019年10月“加州州长希望伯克希尔哈撒韦竞购破产的PG＆E”，其中PG&E是一家濒于破产的电力企业；2020年7月加州州长“呼吁沃伦·巴菲特支持拆除水坝“，以拯救已濒于灭绝的鲑鱼种群。最终巴菲特没有收购PG&E，同意拆除水坝（E＆E新闻记2020年11月18日报道，” 历史性交易刷新了大坝拆除计划“）。如此看来，巴菲特的配合，加州很多政绩工程无法推进，所以州长对比亚迪网开一面也在情理之中了。

维基百科记载，2014年纽瑟姆批准加州第47号提案，将毒品和财产犯罪等某些非暴力犯罪重新归类为轻罪;2015年7月，纽瑟姆支持64号提案，使21岁以上的加州居民的大麻使用和种植合法化；纽瑟姆于2020年5月份正式认可乔·拜登（Joe Biden）担任总统；拜登的竞选搭档卡马拉.哈里斯，来之加州参议院，也是纽瑟姆的密友，可以说深蓝加州与拜登团队很有渊源。

分析到此，读者可以作出判断，在加州民主党深耕的地盘上（连续两任民主党州长），巴菲特有的大额资本投入，而且与民主党的政治理念一致。今年正值2020美国大选波谲云诡之时，拜登团队竞选资金充裕，多米尼投票系统被外国势力远程操控修改数据，中共国深度界入美国大选。那么比亚迪口罩大单背后的棋局开始清晰：
