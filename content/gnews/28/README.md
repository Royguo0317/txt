## 法国周日起要求从其他欧盟国家入境旅客提供阴性检测证明

**西班牙2021年1月23日电/西喜社——**法国将从周日开始要求从其他欧盟国家入境的旅客提供中共病毒阴性检测证明。

强制性核算检测必须在出发前72小时内进行。政府表示将在机场和港口进行具体控制。卡车司机和跨境工人不受该措施的影响。

此举是在周四的视频峰会之后，欧盟领导人在峰会期间同意边境应保持开放，并评估更多的措施来对抗中共病毒变种的传播。

那些从欧盟以外前往法国的人已经需要出示阴性测试证明，一旦进入该国，就会面临七天的隔离。它正在执行全国范围内的宵禁，从晚上6点到早上6点，以减缓病毒的传播。

法国已报告超过7.1万人死于该病毒。
