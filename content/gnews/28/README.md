## 為什麼成龍在香港成了臭狗屎？
`2020-08-20 16:39 GM36` [轉載自GNews](https://gnews.org/zh-hant/308248/)

收集：臺灣荔枝

翻譯：文盲2020 審稿：文橙

香港昔日最著名的對外招牌之一成龍，如今成了臭狗屎！
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/20163617/820_7-2.png)
在2019年10月25日由美國航空公司和捷豹路虎在加利福尼亞州比佛利山莊的比佛利山希爾頓酒店舉辦的2019年英國學院英國獎頒獎典禮上，成龍拿了ALBERT R. BROCCOLI BRITANNIA獎，以表彰他對舞台上娛樂事業的全球貢獻。照片:KEVIN WINTER / GETTY IMAGES NORTH AMERICA / GETTY IMAGES VIA AFP。

對西方世界來說，成龍是一個武術片熒幕英雄，也是一個受歡迎的動作片明星。然而在他的出生地香港，成龍相當不受歡迎，尤其是在香港的民主運動中。

“西方人讚美成龍，但他們不了解他，”一個支持香港民主運動的名為”香港世界城”的熱門推特賬號告訴VICE新聞。 “他並不是自己所塑造的受歡迎的吉祥物。”

在電影界以雜技格鬥風格和滑稽喜劇著稱的成，中文名成龍，很容易成為亞洲最具知名度和影響力的明星之一。他在20世紀70年代從特技演員做起，一路走來，在與喜劇演員克里斯-塔克（Chris Tucker）一起出演熱門動作喜劇系列片《匆匆那年》（Rush Hour）後，他成為國際巨星。

迄今為止，成龍已經出演了200多部電影和節目，仍然是香港娛樂圈最知名的人物之一，估計淨資產為4億美元。

但成龍直言不諱地支持中國執政的共產黨，這在香港引起了爭議–他甚至被招入該黨的政治諮詢機構效力。

成龍在二○○四年時，因一句「三一九槍擊案是天大的笑話」，在台灣引發軒然大波

2009年，成龍質疑新聞自由的想法，並表示中國的控制是一件正面的事情。

成龍表示，「有自由好還是沒有自由好，我現在真的，已經混亂了，太自由了，就變成像香港現在這個樣子，很亂，而且變成台灣這個樣子，也很亂。」他結論， 「我慢慢覺得，我們中國人是需要管的。」因為若不管，就會「胡作非為」。

“我不確定擁有自由是否是好事，”他在2009年的一次會議上說。 “我逐漸開始覺得，我們中國人需要被控制。如果我們不被控制，我們就會為所欲為。”

香港世界城在最近的一條微博中提到了陳水扁的言論，解釋了為什麼香港人不喜歡成龍。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/20163702/820_8-1.png)
推特內容如下：為什麼香港人如此討厭成龍？以下其中一個原因.

成龍公開表示：我逐漸開始感覺到，我們中國人需要被管教起來，如果我們不被管起來，那麼我們就可以為所欲為！

“在香港，支持他的粉絲很少，因為他所持的信念很有爭議–這里和台灣的人都屬於中國大陸，因此，不配享有自由，”推特頁面告訴VICE新聞。 “他的言論是不可理解和不可饒恕的。”

成龍在2019年國家電視台的電視採訪中批評香港再次出現民主抗議活動，重申他的親中立場，並表示希望香港”回歸和平”。

“最近在香港發生的事件讓人感到難過和壓抑。”成龍說。 “香港和中國是我的出生地，也是我的家。中國是我的國家，我愛我的國家。”

最近，成龍表示支持中共對香港實施的有爭議的國家安全法。該法禁止一切形式的分裂、顛覆、恐怖主義和勾結外國的行為，並威脅最高可判處終身監禁。

成龍在一份團體聲明上加了簽名，聲明中寫道。 “我們充分理解維護國家安全對香港的重要性，支持中共代表大會關於香港國家安全法的決定。”

對**反骨仔，跟「為權力撐腰」的舉動** ，香港和台灣網民迅速給他貼上了”兩面派卑鄙小人”和”離經叛道者”的標籤。

“成龍沒有做好榜樣，沒有維護香港的安全和核心價值，而是藉助他在中國大陸的政治地位，”香港民主黨副主席羅健熙告訴VICE新聞。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/20163822/820_9-1.png)
“在上世紀80年代和90年代，他作為一個演員可能很受人喜愛和尊重，但現在情況已經發生了變化。現在香港人對他有很多憎惡，他已經沒有任何資格準確代表我們的城市和人民。當個人的人品有瑕疵的話,要如何代表香港?天大的笑話,過街老鼠在演英雄誰還想看呢?”

推特內容如下：

我在國外經常遇到的情況是，當人們知道我來自香港時，就會拿出成龍來模仿我。

拜託！不要再這樣做了，這是令人反感的，成龍曾經是很受歡迎，但他現在是臭狗屎，而且他在臭了很多年！

成龍也是華裔澳大利亞人氣藝術家Badiucao的政治藝術活動的主題。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/20163848/820_10-1.png)
推特內容如下：

嘿！對面的成龍和中共的狗腿子們，當你愛人和國旗同時掉海裡，先救哪個？

“當你和中共政府站在一起時，你就和暴力、審查、集中營和種族清洗站在一起，”這位政治漫畫家告訴VICE新聞。

“成龍是香港最大的明星之一。他有社會責任為家鄉發生的事情發聲，但他卻為北京支持的暴力和警察暴行辯護。成龍可能是很多中國人的偶像，但他濫用自己的名氣和影響力，與共產黨結盟，不僅背叛香港，也背叛自己。”

藝術家還說：”民主幫助藝術家，給我們創作的自由。成龍是民主的敵人。”

中國大陸市場是一個利潤豐厚的市場，對好萊塢擁有權力和影響力。華盛頓哈德遜研究所高級研究員、澳大利亞悉尼美國研究中心兼職教授約翰-李（John Lee）此前對VICE新聞表示，激怒中共將意味著用自己的職業生涯來付出代價。

“名人和娛樂業高管如果發表被認為是批評中國和共產黨的敏感言論，可能會帶來嚴重的商業後果，”李說。

“中國的票房收入價值近100億美元，使其成為僅次於美國的世界第二大票房收入，有估計這個數字甚至會翻倍，”他補充說。 “中國內地觀眾也是一個極具影響力的市場，美國和其他地區的電影業都在尋求拓展這個市場。”

多年來轟動香港的反政府抗議活動動盪不安，也使香港的娛樂業出現分裂。

像成龍和葉問明星甄子丹這樣的明星人物，在即將上映的迪士尼真人版《花木蘭》中，因忠於宣傳中國和共產黨而受到獎勵。但同為香港老牌明星，如演員周潤發、Cantopop歌手何韻詩等，他們在香港民主運動背後投下了支持的一票，卻付出了事業的代價。

“有一種背叛和虛偽的感覺，因為成龍出生在香港，大概享受了香港提供的自由，”翰-李（John Lee）告訴VICE新聞。

“這對他的事業大有裨益，”翰-李（John Lee）補充道。 “但今天，他在高度敏感和政治問題上表達了與中共政府相同的宣傳信息，強調愛國主義和穩定的重要性，而不是自由和民主。”

成龍的經紀人沒有立即回應VICE新聞的這一報導。
