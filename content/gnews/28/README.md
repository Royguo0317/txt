据《华盛顿观察家》2月25日报道， 川普在11月被拜登总统击败，但在众多共和党选民心中，川普仍是赢家；他们很希望看到他第三次入主白宫。

川普将于周日在佛罗里达州中部的保守政治行动会议(CPAC)上发表演讲，这是共和党内部人士和基层保守派年度聚会中最令人期待的演讲。

共和党政界的大人物们都会在保守政治行动会议(CPAC)的主舞台上发言。 有些可能是总统竞选人，包括阿肯色州参议员汤姆-科顿、德克萨斯州参议员特德-克鲁兹、佛罗里达州州长罗恩-德桑蒂斯、密苏里州参议员乔希-霍利、南达科他州州长克里斯蒂-诺姆、前国务卿迈克-蓬佩奥、佛罗里达州参议员马可-卢比奥 以及佛罗里达州参议员里克-斯科特。 全国共和党参议员委员会主席 观察名单上的另一位人物小唐纳德-川普也将发表讲话。

值得注意的是，前副总统迈克-彭斯（Mike Pence）和美国前驻联合国大使尼基-海利（Nikki Haley）缺席。

川普的演讲将是他1月20日退出白宫后的首次演讲。他计划明确表示，他不会从政治中退休，而是打算作为共和党的实际负责人继续活跃在2022年中期选举之前。

这次聚会的大主题是 “美国取消”。共和党人认为，民主党全面控制下的联邦政府，以及文化和企业力量，都有意压制保守派。例如，川普被永久地从推特上赶走。但今年保守政治行动会议(CPAC)的更大目的是激励保守派活动家重新参与政治，对美国制度有信心。

施拉普说，在共和党失去白宫和参议院控制权后，他们已经灰心丧气。和川普一样，他们也相信2020年的选举是被偷走的。”会议的一部分是让每个人都意识到我们必须摆脱这种沮丧的模式，”他解释说。”很多都是政治辅导。”

2000年代末，倾向于自由主义的共和党人罗恩-保罗（Ron Paul）往往是保守政治行动会议(CPAC)上最受欢迎的人物，这表明一些保守派人士渴望摆脱罗纳德-里根的外交和国内政策，而这些政策已经定义了一代人的保守主义。

2011年，川普首次出现在保守政治行动会议(CPAC)的主舞台上。五年后，他以自己的形象重塑了共和党的基础，走上了赢得总统职位的道路。

“这是一个很好的地方，可以看到党在各种问题上的位置，”共和党策略师安迪-苏拉比安说。”这是一个更好的地方，可以发现可能需要一些时间才能展开的趋势。”
