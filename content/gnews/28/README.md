**教育的目的是為社會主義培養接班人,絕不允許培養共產主義制度掘墓人。**

新書中收錄的習近平2019年3月18日的講話中,習在學校思想政治理論課教師座談會上強調,教育的目的是為社會主義培養接班人,絕不允許培養共產主義制度掘墓人! 如果培養了半天,培養出來的是吃裡扒外、吃哪家飯砸哪家鍋的人,甚至是我們這個制度的掘墓人,那辦教育幹什麼? 那將是教育制度的失敗。

習近平提出5點警告城市人口激增可能帶來更高的社會動蕩風險,由於2014年李克強在其年度政府工作報告中承諾,中國將促進1億人口向城市遷移,習近平的上述講話是在駁斥李克強的路線。

習稱,所謂的境外敵對勢力加大滲透和激化力度,境內一些組織和個人不斷變幻手法,製造思想混亂與我爭奪人心。 一些單位黨政官員政治敏感性,責任心不強,在重大意識形態問題上含含糊糊遮遮掩掩,助長了錯誤思潮的擴散。

**在重大問題上絕不允許和中央唱反調,絕不允許吃黨的飯砸黨的鍋,對於搞亂黨性的人要敢抓敢管。**

習強調,中共意識形態、關乎體制,關乎道路,關乎國家,關乎政治安全,各級黨委和思想宣傳部門、教育部門,要加強領導和管理,黨報、黨刊、黨網,以及各大院校要強化政治意識,在重大問題上和中央保持高度一致。

絕不允許和中央唱反調,絕不允許吃黨的飯砸黨的鍋,要高度重視苗頭性、傾向性問題,打好主動仗,防患於未然。

習稱,如果聽任這些言論「大行其道,指鹿為馬,三人成虎」,勢必搞亂黨心民心,危及黨的領導和社會主義國家政權安全,在事關堅持還是否定四項基本原則的大是大非和政治原則問題上,必須增強主動性、掌握主動權、打好主動仗。 習近平還要求中共各級官員,要敢抓敢管,敢於亮劍等。

**習近平對政法系統進行”延安整風”運動。**

中國領導人習近平希望加強國內的政治紀律,中國政法機構的官員們已收到命令要「刀刃向內、刮骨療毒」,放下私人忠誠,揭露有問題的同事。 “一組組組組組讀”從您應該以”延安整風”為這次”教育表示”行動的樣板,延安整風是毛澤東在20世紀40年代發起的一場運動,運動鞏固了他在根據地延安對中共的統治地位。 “清除害群之馬,”教育行動陳一新在上月的動員會上說。 清查對黨不忠誠不老實的’兩面人’。”

這場延安整風運動是習近平清洗對手派系的努力,行動涉及範圍之廣表明習近平想徹底調整政法系統。

今年早些時候,中共還成立了一個名為”平安中國建設協調小組”的治安政策委員會,以加大打擊動蕩和犯罪的努力。
