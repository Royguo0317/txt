现在这场会谈已经正在进行中，而在这场会谈之前中共驻美国大使馆于2021年3月18日发布了[崔天凯大使在安克雷奇接受媒体采访（实录）](http://www.china-embassy.org/chn/zmgx/t1862127.htm)，崔天凯这一次没有脱下西装换上战装，他已经到达了安克雷奇，在这次访谈中透露出浓浓的悲观情绪——

1、对于会谈结果的预盼——［我们并不指望一次对话就能解决中美之间所有问题，所以对这次对话也没有过高的期待或者幻想。我希望，这能够成为一个开端。双方能够开启一个坦诚、建设性、理性对话和沟通的过程。如能做到这一点，我认为这次对话就是成功的。］

2、关于美国对日韩的2+2会谈及与印澳的视频峰会——［美国和其他国家要发展双边关系，那是他们的事情。我们只是希望，任何这种双边活动不要以第三国为目标，不要损害第三国利益。可能有人认为，在跟中方见面之前先找一些别的国家谈一谈，发点声音、造点声势。其实这个做法没太大必要，也不见得有多大用处。就像有的人一个人走夜路会唱歌给自己壮胆，其实没有多大用处。有什么问题要跟中方谈，尽管可以面对面摊开来谈。］

原来中共国已经需要美国走夜路唱歌来壮胆了，原来［美国国务院高官就职后首次出访选择日韩盟国，美日印澳也举行了视频峰会］。是美国为了与中共会谈时发点声音、造点声势，原来中共国已经如此地令美国仰其鼻息了。

3、对于与美方的会谈中可能涉及的涉港、涉疆与台湾等问题——［你刚才提到的几个问题都涉及到中国的主权、领土完整、国家统一。在这些核心利益上，中方没有任何妥协退让的余地，这也是我们将在这次会谈中清楚表明的态度。如果认为单方面或者找了几个别的国家给中方施加压力，中方就会妥协、就会让步；如果认为我们为了这次对话所谓“有成果”，就会答应任何一方的单方面要求——我劝人们最好放弃这种幻想，这种态度只会把对话引向死胡同。］

涉港、涉疆与台湾等问题是中共的核心利益意味着什么？它意味着中共祸害香港的目的和举动不会停止，这意味着中共对新疆的种族屠杀不会停止，它意味着中共对台湾的打压不会停止，在这一系列的核心利益之下，中共自己不仅仅是走进了死胡同，而是已经走在了去停尸房的路上，所谓中共不会在此次会谈中就核心利益做出妥协和让步，这一点是没错的，中共永远不会改变其邪恶目标，但区别就在于目前包括美国在内的国际社会对涉港、涉疆与台湾等问题的定性是不需要中共来妥协和退让的，因为国际社会对涉港、涉疆与台湾等问题的定性就是在为中共的棺材板上钉钉子。

既然国际社会在给中共的棺材上钉钉子，那么这次会谈它能起到什么作用呢？这次会议或许能让中共晚死几天，但它绝对不会让中共免死，也许这才是中共的杨娘娘、王太监和崔大屎要低三下四的在阿拉斯加与美方官员举行他们所称的高级别会谈的唯一目的吧！
