## 中共间谍科学家被抓为欧洲情报界敲响警钟

![]()![](https://gnews.org/wp-content/uploads/2021/03/capture-14-35.jpg)图片来源：每日怪兽
据《商业内幕》（Business Insider）近日报道，一位杰出的科学家在波罗的海国家替中共从事间谍活动而被指控，该事件给全欧洲的情报官员敲响了警钟。知情者告诉《商业内幕》说，这个案例与中共企图渗透全世界科研机构的长远邪恶计划有着密不可分的关系，在某些方面中共想超越俄罗斯成为西方最大的敌人。

报道称，年长57岁的爱沙尼亚副海军上将塔莫·库特斯（Tarmo Kõuts）被判三年监禁。该海军上将曾密切参与过一些敏感项目，现已经和爱沙尼亚军方切断关系。检察官说，塔莫·库特斯于2018年开始为中共从事间谍活动，去年9月被秘密逮捕。丹麦政府官员向媒体提供的账户显示，他获得了约2万美元和几次豪华旅行。

《商业内幕》与三名欧洲情报官员进行了交谈，其中两名来自西欧，一名来自波罗的海某国。他们都要求匿名，不对媒体公开信息，但他们的身份为《商业内幕》所熟知。这些情报官描述了如此一种情形，即中共大肆渗透欧洲各国的间谍活动构成了欧洲反情报界的最大威胁。波罗的海官员说：“虽然关注俄罗斯的情报活动仍是我们的首要目标，但监视中国所投入的资源也每年都在增加。我们与俄罗斯间谍的交往经验自然相对于中国要多一些，但一年来我们一直在警告中国。”

报道指出，立陶宛2月份宣布禁止使用中共国的安全软件承包商。大约在同一时间，爱沙尼亚官员警告说，中共国和俄罗斯的承包商也有着类似的威胁。比利时一位官员告诉《商业内幕》说：中共在欧洲的渗透已经有五年了，“我们首先看到布鲁塞尔的间谍活动急剧增加，中国驻欧盟的外交使团为间谍活动作掩护，这太明显了。在2019年，中国一共向欧盟及其学术界派遣了多达250名情报官员。”欧盟外交政策官员在2019年2月的一份报告中显示，至少有250名中共国间谍和来自其他国家的300名情报人员在布鲁塞尔活动。比利时官员说：“那时候我们意识到中国不仅有最多的间谍，而且大多数间谍有可能都是中国人。”

2020年1月，媒体报道显示，德国检察官正在对德国一名欧盟高级特使以及两名德国游说者进行间谍调查，该案件正在德国以隐私案进行审判。四月份，欧洲的大学被警告说中共国正在以学术交流为幌子进行情报收集活动。

来自北约一个国家的第三情报官员说，爱沙尼亚案件是中共国招募间谍的典型案例。爱沙尼亚案中，中共机构利用会议差旅和学术交流为幌子来招募，培训，并奖励像塔莫·库特斯这类人做间谍。

报道称，这位不愿透露姓名的官员说：“他们喜欢偷窃学术方面的东西。” “中国非常重视知识产权，科学研究和工业技术的收集，而俄罗斯人则更加注重传统的硬性政府情报收集。“这意味着间谍活动通常是在比较放松的会议环境中进行的，这样间谍也更好招募。而且中国给他们的奖赏也很容易掩盖……奢侈的旅行伪装成学术会议，美酒佳肴或妓女服务拿上了合法的会议上，这样就使招募间谍变得非常容易了。”
