## 中共掠奪人民財富的幾種方式

中共就是吸附在14億中國老百姓，包括九千萬普通黨員身上的寄生蟲，不創造價值，只知道壓榨、索取、欺騙、消滅，對於中共，他們只喜歡鬥爭，與天鬥、與地鬥、與人鬥，下一步他們恐怕要與宇宙來鬥一斗。中共滿足邪惡政權慾望最有效的方式就是對人民財富的直接掠奪。

1、資本控制：貨幣超發引發的通貨膨脹問題對於持有貨幣的中國百姓來說就是在無形間掠奪了他們的財富，削弱了他們的購買力。假設市場上100億的貨幣對應等值的產品，那麼你帳戶上10萬元理應可以購買10萬的產品，這時候中共增發一倍貨幣，市場貨幣量變200億，但是產品還是100億，產品標價實際由1元變為2元，那你帳戶裡的10萬隻能買到之前一半的產品。前些年的房地產價格不斷上漲，理論上抗擊了通脹風險，但在當今有價無市的狀況下，嚴重缺乏流動性的房產實際上已經沒有任何投資價值了。

2、壟斷資源：毛澤東土改實際上是對全中國人民的掠奪，在他選定的基礎條件下，中國經歷了各種拆除重建，每一次所謂的規劃都是對人們財富的又一次掠奪。土地還是那塊土地，但中共就能做到賣你好多次。無恆產則無恆心，中國人現在賺快錢的心態，也是源於固產一次次被掠奪所帶來的不安全感。

3、生產要素壟斷：改革開放的最大成就之一就是勞動力解放，人民收入開始跟隨市場定價。但時至今日，隨著經濟發展和物價上漲，人民收入水平明顯趨於落後，低收入人群佔中共國絕大多數。其中原因，一定有中共的打壓和限制，如不存在獨立的工會，強行徵收五險等。將人們生活維持在溫飽水平，是更有利於中共統治的。

4、稅收、社醫保及相關行政性收費：中共稅收可以說是全世界最複雜的稅收，作為相關從業者，也經常被搞暈，稅目多、要求多，還有徵稅方式的隱蔽，導致絕大部分人不知道自己到底被中共掠奪了多少錢。

- 2011年福布斯推出的全球稅負痛苦指數的排名，其中中共國位居第二，僅次於法國。 （該指數由公司稅收、個人所得稅率、富人稅率、消費稅率/增值稅率以及雇主和僱員的社會保障貢獻等計算而得，考慮比較全面，可參考價值高。）雖然法國高於中共國，但法國社會福利有400多項包括幼兒園至大學學費全免，醫療保險全民免費享有，社會養老金，各類家庭補助金等。中共國的高稅負並沒有帶來高福利，如教育仍為九年義務教育，不含蓋高中、大學費用；退休年齡一再推遲，近期江蘇退出新政策，個體人員（即無企業聘用的）推遲退休5年，即使順利退休，800-900元的退休金無法滿足生活需求；醫保報銷條件繁多，大量藥物無法報銷，近期推出政策調整，將每年返還醫保卡金額降低至20%，其餘全部統籌。


- 稅收徵收隱蔽，例如，消費者所購買的產品，其實際價格中包含了大比例稅收，例如一件標價100元的衣服，有12.71元為增值稅及附加稅；如果該產品為消費稅徵收稅目，如標價100元的高檔化妝品，則還需要繳納11.38元的消費稅，總計納稅23.09元。再比如社醫保繳納也是典型的披著福利外衣的資產掠奪。首先個人需繳納工資的11%，企業為32.5%，人們認為企業繳納部分與自己無關，但這間接導致了企業付給個人的工資縮水，最終損害的還是個人的收入。同時就中共社醫保的質量而言，那是一塌糊塗，其實際受益程度遠沒有付出的多。如中共國男性退休年齡60周歲，假設一個人工資3000元，25歲開始繳納養老保險，35年共繳納365400元，60歲後開始領取養老金，一個月1000元，則需要30.5年才能回本，還不包括這麼多年的利息，這簡直就是一種綁架。


- 企業在稅負及各類行政管理費用掠奪下運行困難，例如增值稅名義上對增值額增稅，最終稅負由消費者承擔，但對於下游企業來說，產品無論是否賣出，增值稅已經繳納，這導致了企業承擔很大風險。同時企業需要在覆蓋產品成本及中共稅負的前提下為產品定價，平衡銷量和價格問題，易讓企業陷入困境。另外地方性政府隨意要求繳納各種費用，如前期環保安全話題熱門，各地區政府要求企業繳納金額不等的環保費或安全費等。
