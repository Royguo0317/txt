據魁北克新聞網報導，公眾對9月29日舉行的首輪美國總統辯論期望甚高，由於空前的萬眾矚目程度以致於有媒體把這場在川普總統和民主黨候選人前副總統拜登之間的對決認為是該國歷史上最重要的辯論之一。

在目前中共瘟疫大流行、種族緊張局勢和可能動蕩的投票過程等惡劣形勢背景籠罩之下，川普和拜登之間的直接衝突代表了美利堅合眾國兩種截然不同的強烈願景。但公眾必須意識到，因為兩名候選人身後都各有一個穩固的黨派集團，所以辯論不可能改變這種境況。由於拜登明顯缺乏應有的精力而經常被其對手稱為“瞌睡喬”，但他也許有機會能證明自己並擊敗強大的川普，以期擴大其日後的選舉基礎。而這次辯論就是其一個不可多得的機會。

然而在辯論當晚的許多時刻，前副總統表現出了特別嚴重的認知功能障礙。有時面對問題他會遲疑徘徊片刻，無法保持其思維和言論繼續流暢地表達。不管是忘記當下的問題，還是結結巴巴地混淆邏輯順序，“瞌睡喬”的形象始終都恰如其分，實至名歸地適合他。在其軟肋遭到對手致命攻擊的壓力下，拜登會立即詛咒總統以捍衛自己，就像是一頭因強烈求生本能而欲逃脫陷阱的困獸。但如果川普同樣這麼做，所有左翼媒體都將會毫無意外地把總統釘在道德的十字架上。而川普面對拜登所採取的姿態，就如同對抗魔鬼時必須具有的戰鬥態勢。

必須指出在首輪的辯論中，川普並沒有悲憫饒恕其對手，即使這意味著不得不打破繁文縟節的禮儀。面對當晚主持人華萊士時時刻刻處心積慮地對拜登不惜一切代價的幫助和偏袒，總統依然一如既往地表現出了其難以置信的強大戰鬥力和旺盛的精力。

此外，本次拜登的表現頗為捉摸不定而且看上去似乎沒有以往那般好鬥，前副總統全程幾乎沒敢直視過川普並始終逃避回答所有針對他的敏感問題，而是選擇直接與空氣和觀眾交談。拜登對其選民的暴君式態度使所有其可憐的支持者們感到無比絕望。

毋庸置疑，“法律和秩序”一直都是川普辯論的中心主題。而總統也正是用這一重點徹底擊敗其對手。而拜登無疑被美國當下所有最暴力的城市都由民主黨政府主導的這一事實所完全擊潰，並在辯論中屢屢毫無基本概念和常識地將反法西斯民兵、社會共產主義和反美主義與其他最激進組織如ANTIFA等混為一談！

川普在辯論中自信表示，所有法律和秩序的力量都將選民以壓倒性多數轉向擁護現任總統。而拜登的選情恰恰相反，他在不停地收復和聚集邪惡的反法治力量陣營！即使該前副總統聲稱支持對執法部門的融資，但事實仍然是支持他的選民和那些激進的民主黨人，一定會繼續推動這種目無法紀的致命意識形態，而將美國各界帶入萬劫不復的深淵。

總統辯論的最後部分以“如何確定總統選舉中的投票方式？”這一充滿緊張氣氛的問題而結束。儘管拜登明確表示將遵從一切選舉結果，但川普卻非常堅定地宣稱絕不容忍任何選票欺詐行為，否則他將很有可能對投票結果提出法令質疑。總統還強烈呼籲其支持者謹慎提防任何形式的選舉欺詐惡行，他甚至還懷疑最高法院能否最終掌握所有合法的選舉文件。

與此同時，如果Amy Coney Barrett最終被成功任命為最高法院的大法官，那麼該結果將使保守派法官的權力平衡構成達到6比3！考慮到在當前這種不平衡黨派權利的情況下進行的投票選舉，這也許將會無可避免地開啟美國歷史上又一個特別動蕩的政治時期。
