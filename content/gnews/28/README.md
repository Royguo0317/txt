## 美國參議員提出法案保護台灣免受中共國侵略
`2020-09-20 10:16 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/371159/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/20100938/1-98.png)

圖片來源：https://www.oneplusnews.com/news/5a8de750ef8690731a6ccbf4 
據《聚焦台灣》報導，美國華盛頓特區9月19日訊，美國共和黨參議員里克·斯科特（Rick Scott）提出了一項法案，以阻止中國對台灣使用武力。

發表在斯科特網頁上的一份聲明說：擬議中的《台灣防衛法》旨在建立“有限度授權（美國）總統有權使用軍事力量，特別用於保護台灣的安全，和免受武裝侵略。”該法案將允許美國通過增強台灣抵禦中共侵略行為的能力，來加強其對台灣的長期政策。

斯科特在其網站上明確指出：“中國共產黨持續威脅我們的重要盟友，不僅威脅台灣人民，而且威脅美國和我們在全球的盟友。”“我們必須竭盡所能來阻止中共對和平的民主力量使用武力，《台灣防衛法》體現了我們捍衛台灣和維護自由民主制度的承諾。”

該法案是由共和黨眾議員泰德·尤霍（Ted Yoho）在美國眾議院提出的，該法案還鼓勵美國總統或國務卿在台灣的土地上與台灣總統會晤。

該法案還力求在美國，台灣以及其他志同道合的伙伴之間建立安全對話機制，並舉行聯合軍事演習，以應對一旦中共入侵台灣，即可採取協調一致的軍事行動。該法案同時敦促中共放棄使用，或威脅使用武力犯台。

該法案進一步闡明：“美國的這一政策是基於台灣在非和平情況下，如何採取一切努力以確保台灣的未來。這些非和平情況包括：抵製或禁運，對西太平洋地區和平與安全的威脅以及其他美國嚴重關切的問題。”

此外，法案還要求美國貿易代表在該法案實施後的180天之內，與台灣方面就雙邊貿易協定進行談判。

簡評：

首先，此法案提出的背景值得關注，此時正值美國副國務卿克拉奇（Keith Krach）率團訪問台灣之際，當然也是中共叫囂並派戰機騷擾之時。美國共和黨參議員里克·斯科特（Rick Scott）在此時提出此項法案，進一步表明了美國希望加強與台灣關係的決心，同時也是對中共軍事威脅的蔑視。

其次，多年來，中共始終把台灣作為與美國在國際政治博弈中的一枚棋子，同時對台灣實行威脅與滲透的雙重手段，使台灣一方面籠罩在中共飛彈的陰影之下，另一方面，親共勢力甚囂塵上。近年來，特別是中共病毒爆發之後，中共的野心與邪惡暴露無疑，中美關係已經到了關鍵節點，而台灣無疑是美中之間，即正義與邪惡之間較量過程中，雙方必爭的製高點。台灣將何去何從？這個華人世界唯一的民主自由之地，她的未來已成為全世界的焦點。

此時《台灣防衛法》的提案，是向台灣發出的一個明確的信號，美國政府一改以往在台灣問題上小心翼翼的做法，並強調該法案旨在：“捍衛台灣和維護自由民主制度的承諾。”這無疑使台灣政府更加有信心對抗中共的威脅。

另外，在參議員里克·斯科特對該法案的十一條說明中我們可以看到，該法案的主旨是加強美台全方位軍事合作，以抵禦中共武力犯台，同時也敦促美台達成貿易協議，加強經濟合作，而更為引人注目的是鼓勵美國總統或國務卿盡快踏上台灣的土地，並歡迎台灣總統到美國國會發表演講。毫無疑問，這已經是美台建交的前奏了。

最後，我們注意到，蔡英文總統在歡迎美國副國務卿克拉奇的致辭中說：“台灣有決心踏出關鍵的一步，透過彼此面對面的意見交換，將會為台美更深化的合作開啟新的里程碑。”那麼，是什麼因素使“台灣有決心踏出關鍵的一步”呢？當然，這些因素是多方面的，其中毋庸置疑的是，美國在立法層面所推進的，旨在維護台灣自由民主制度和領土安全的法案是至關重要的一環。
