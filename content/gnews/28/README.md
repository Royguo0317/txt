據人民網2020的 27日9:52來源新華社的[習近平將在亞洲基礎設施投資銀行第五屆理事會年會視訊會議開幕式上致辭](http://world.people.com.cn/n1/2020/0727/c1002-31799147.html)：

【外交部發言人華春瑩27日宣佈：亞洲基礎設施投資銀行第五屆理事會年會視訊會議將於7月 28日舉行。國家主席習近平將在開幕式上致辭。】

參加這界理事會的有多少國家？我們知道郭文貴先生在“**郭先生****2020****年****7****月****12****日****GTV****直播**”中提到——

【你看什麼亞洲基礎設施銀行、亞投行，還有什麼亞洲東盟合作機制，這都是挑戰西方的，所謂的要新型兩國關係，都挑戰美國的。我說這都是作死的路子，都是作死的。然後頭幾天，這些國家跟我開會的時候，我說你們還愚蠢地現在在這個亞投行還投資股份，你向他借錢啊？看他借不借給你啊？他說每次我們借錢，都囉嗦得不得了。我說這就是中共和西方的差距，西方再壞，他說了事算事，他有合約精神。我成立了亞投行，我給你了股東體制，我要借給你錢，我就借給你錢。我說共產黨這王八蛋不是這個，全是相反的，我說你去試試去。頭一段時間，中共威脅每個人，王毅挨個打電話，一天打個四五次，騷擾人家國家領導人，我有緊急電話，兩國關係電話，大國元首電話，大國元首熱線。他們分三個層次，叫做新型戰略大型國關係，這是一個關係；什麼全天候兄弟夥伴關係，實際上就是把過去不結盟變成了結盟關係；然後兩國戰略性的關係。他不同的國家有不同的熱線，就拿著熱線騷擾人家。騷擾人家幹嘛？你必須站出來支持《香港安全法》。有的國家就不想支持。你不支持？亞投行的貸款沒了。你不支持？我跟你的合約不認了。】

【就像那欺民賊一樣，它就是在那威脅，瞪眼撒謊，他撒謊的時候很嚴肅很認真，這傢伙很搞笑，人家這受不了這些國家。啥時候了，還威脅我支持香港法，我不支持香港法，你威脅我。所以頭兩天跟我聯繫。我說你們現在最簡單的，別威脅我了，老子走，我退，我不在你亞投行幹，我不參與這個能咋地？剛才這個會議當中這些人說，我們都退了。大家知道多少退出亞投行嗎，36家，36家。我說它不叫亞洲投資銀行。你們一定要記住，我從幾年前就說，每個國家跟我建立關係，我說叫亞洲詐騙銀行，或者叫亞偷銀行，亞洲的偷東西的銀行，我說你找死啊，那是恐怖組織。現在大家都理解了。】

在明天就要召開的亞投行中已經有多少國家退出來了呢？為了查出到底哪些國家退出了亞投行，我查閱了[亞投行成員國名單(亞投行有多少個成員國加入？亞投行成員國2020名單最新)](http://www.3sww.com/article-detail-84300-1.html)（時間 : 2020-05-03 15:02:55 作者：[北京裝修網](http://www.3sww.com/)）裡邊列明的80個成員國，再與[亞投行100個成員國名單匯總一覽](https://www.mrcjcn.com/finance/yinhang/302206.html)（2019-7-15 11:12:45來源：每日財經網責任編輯：斑馬）列明的100個成員國進行比對，經比對，截止2020年5月3日：

以下21個國家退出了亞投行——

貝寧、吉布地、盧旺達、以色列、瓦努阿圖、白俄羅斯、幾內亞、科克群島、黎巴嫩、巴布亞新磯內亞、阿爾及利亞、象牙海岸、厄瓜多爾、加納、肯雅、利比亞、摩洛哥、塞爾維亞、多哥、突尼斯、烏拉圭；

以下3個國家新加入了亞役行——

吉爾吉斯斯坦、伊朗、俄羅斯。

在2020年5月3日至今另外哪15個國家退出了亞投行呢？等我們看到亞投行的最新成員國名單以後再揭一揭名為亞投行實為亞偷行的瘡疤。

明天敬愛的習近平總書記就要在亞偷行第五屆理事會視訊會議上致辭了，相信這一屆理事會也是最後一屆理事會，祝亞偷行的這最後一屆理事會在敬愛的習近平總書記致辭之下能夠圓滿閉幕！

另外兲朝經常批評川普總統動輒退群，對這些退出亞偷行棄暗投明的國家，中共咬部、日人民報和新華社這些騙子部門和中共喉舌冷嘲熱諷了呢？我很奇怪！
