大西洋地区法院法官蒂莫西·c·巴顿(Timothy C. Batten)昨晚深夜公布的判令中有一条被埋没的线索，它描述了佐治亚州的律师——表面上是通过地方政府代表公众——捍卫多米尼投票系统的“商业秘密”。

“被告的律师还辩称，允许这种取证检查将给被告带来巨大的安全和专有/商业秘密风险。”
