中共軍方自2020年9月18日至24日期間，派遣多架次軍機，越過“台海中線”，對台防空識別區進行侵擾！

18日首批派出以，轟-6，殲-10/11/16等4型戰機，共18架所組成的混合編隊，對台進行越線侵擾。

19日則比前日多增配了1架運-8反潛機，共計19架所組成的混合編隊，繼續對台進行越線侵擾。

20日不明原因未有任何前出動作！
 21/22/23/24日，均只派有2架運-8反潛機，在台海周邊區域作短暫飛行後隨即脫離！

中共外交部發言人王文斌在9月21日，舉行的例行記者會上，剛剛“霸氣”宣稱：從來就不存在所謂的“海峽中線”一說！

中共當日的對台強硬軍機侵擾行動就“偃旗息鼓”！
僅僅派出2架運-8反潛機。
 “故作姿態”的短暫前出後，旋即馬上脫離回航。

在隨後的幾日，無一例外的均派出了2架運-8反潛機，在台海周邊短暫巡飛後立即脫離返航。

這與之前中共所叫囂的“武統台灣”，隨時搶灘奪島的“強硬”口氣，形成了鮮明的極大反差！

中共公然否認“台海中線”存在的客觀既定事實，就等同於其單方面撕毀了原有的“台海交戰守則”！
這種平衡一旦被打破，將意味著雙方都會面臨著急劇攀升的“擦槍走火”事件的發生。

中共自知，如若在此種近乎於“自由裁量交戰”的規則框架下，其空軍現有實際戰力很難獲取空優！

對於中共而言，“武力侵台”只有在發起突襲的第一波次，遠程火力覆蓋階段內，中共或許才有可能達到其預定攻擊效果！
