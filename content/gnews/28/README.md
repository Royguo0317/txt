## 共党向世界输出独裁和动乱

Sherryok 水云间

继澳大利亚4月21日宣布取消与中国签署的“一带一路”协议后，5月20日，又一个太平洋岛国-萨摩亚，宣布将搁置[中共](https://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1.html)“[一带一路](https://www.epochtimes.com/gb/tag/%E4%B8%80%E5%B8%A6%E4%B8%80%E8%B7%AF.html)”倡议下的巨额港口开发案。但是中共岂能咽下这口气，昨日，新当选的萨摩亚第一任女总理马塔法（Fiame Naomi Mataafa）及她的信仰统一党（FAST Party），被锁在众议院门外，不得不在草坪上举行宣誓仪式。

4月9日萨摩亚大选后，FAST 党以26席对25席战胜执政39年之久的人权保护党（HRPP），获得多数执政，但是HRPP以女性议员比例少于10%的理由，要求增加一名议员名额，此举在5月17日被法院裁定违法。因此马塔法当之无愧成为新任总理。5月20日，马塔法接受媒体采访，表明上任后将首先搁置与中共签署的一带一路项目，她说：“萨摩亚是一个小国，我们的海港和机场已可满足我们的需求。”“很难想像，当政府需要优先处理更紧迫的项目时，我们需要如此规模的开发案。”她的发言也标志着萨摩亚对华政策的转变。

中共一带一路不久前才被澳大利亚叫停，现在又被太平洋岛国打脸，习总加速师的脸面往哪搁。 果然，5月22日晚，没有任何理由和解释，萨摩亚元首宣布取消24日周一的国会宣誓。按照萨摩亚的法律，24日是大选后宣誓的最后期限，尽管法院再次紧急裁定这一取消会议的决定非法，但是众议院仍然在周一闭门，将前来国会宣誓的新总理等拒之门外。形势如此严峻，就连军队都出动包围了议会，但是坚强的女总理没有退缩，她在众议院门外的帐篷里宣誓就职！

一次民主选举理应带来的平稳政权交替，因为掺进了中共一带一路的利益冲突和势力较量，导致这个常驻人口仅20万的岛国，政局如此动荡。现在下台的HRPP 指责FAST党政变，声称将实施反击，足见这一岛国的动荡还将持续下去。

值得注意的是，中共号称给全球华人提供免费疫苗，“撑起保护伞”的春苗行动，已经在4月30日落地萨摩亚，在这个萨摩亚政局微妙复杂的形势下，中共会不会故伎重演，再一次使出恶毒的“生物超限战”，陷萨摩亚于紧急状态，以左右政局，是绝对要引起太平洋岛国严重关注的。

5月20日中共发言人回应萨摩亚将取消一带一路新闻时，仍不忘提自己“一贯秉持人类命运共同体的“真实亲诚”理念”，开展同萨摩亚等太平洋岛国的互利合作，目的是改善萨摩亚基础设施状况，促进当地经济发展。那么如此真实亲诚的天朝大善人，在主人下了逐客令后，是不是该乖乖的离开，而不是私底下一溜小动作，强奸他国民意，妄图给天朝增加一个附属国？萨摩亚人民应该庆幸在掉进陷阱前醒了，但是请一定不要低估中共的邪恶，保护家园，远离中共！
