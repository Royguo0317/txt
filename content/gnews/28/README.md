从一个政治正确走向另一个政治正确，用这句话来形容这两年关于奥斯卡的种种新闻非常合适。先不管奥斯卡的选择究竟如何，就从这几天中共的反应来看已经很能说明问题了。不仅是全网噤声，甚至也有微信群因为讨论奥斯卡而遭到封禁，如此表现让我都替中共觉得丢人。今年奥斯卡的入围片质量如何目前还无法评价，不过这样的构成让我想起了2018年的金马奖，不同时间不同地点但故事背后都有中共的从中作梗。

先来看看2018年金马奖的这些入围影片，《我不是药神》、《地球最后的夜晚》、《风中有朵雨做的云》、《谁先爱上他的》、《范保德》等等，不管什么地方出品质量都有一定保证，就算是张艺谋的《影》即使内核不够，外在的皮囊也还撑得住场面。不过当年的最佳剧情长片以及最佳改编剧本却给了一部主流视野里最名不见经传的影片，胡波导演的《大象席地而坐》。当时胡波妈妈上台领奖，大屏幕上放出胡波遗像的场景至今我都不会忘记。为什么这部电影能获奖？为什么导演会选择自杀？这一幕也让我开始去了解《大象席地而坐》，以及电影之外的种种故事。

**满洲里的动物园有一只大象，它他妈整天就坐那，可能有人老拿叉子扎它，也可能它就喜欢坐那。很多人跑过去抱着栏杆看，有人要扔什么吃的过去，它也不理。**

**——胡波《大象席地而坐》**

这是一个在电影中贯穿始终的传闻，而在经历了相似且无望的一天之后为朋友出头的韦布、为弟弟报仇的于城、陷入与教导主任不伦恋谜团的黄玲以及被家人赶去老人院的王金都选择了离开，前往满洲里看大象。整部影片都笼罩在一个灰黑色的气氛之下，天气是灰蒙蒙的，镜头是背光的，你很少能看到人们的神情，因为对于没有期望的明天不可能有任何表情。

**“你根本就不知道活着是怎么回事。我哪有精力去给你营造你喜欢的东西，我自己还一团糟搞不清楚呢。”**

**“我过得太惨了。”**

**“这都不是问题，一直这样，我也这样，你也一样，就是这样。”**

**——胡波《大象席地而坐》**

电影中压抑到让人绝望的气氛也正映射了残酷的社会现实。中共治下的这片土地为什么人与人之间根本没有对话，因为中共的剥削让你为了活着连悲伤愤怒的时间都不能拥有，能看到的不就是一张张面无表情的脸么。中共所至之处寸草不生，对于影片拍摄也是一样。本来在中共国要拍摄出灰暗朦胧的画面效果不是很难，然而在开拍前当地的工厂全部被关闭，雾霾全部消失，天天晴空万里。所以只能在天濛濛亮和天黑之前这两段时间拍摄，用章宇的话说，每一个镜头都是赌出来的。

**完成这部电影用了一整年时间，而最终，没有一帧画面属于我，我也无法保护它。它被外力消解掉了。**

**——胡波《牛蛙》后记**

除此之外，电影本身只有60万人民币的预算，主演的演出费都非常低，而工作人员甚至是无薪工作。而在电影时长上胡波和投资公司冬春影业有着不一样的意见，随着分歧越来越严重，冬春影业通过抬高价格的方式将版权把持在自己手中，胡波也因为这一系列状况上吊自缢。

一部时长达到4小时的影片能有票房么？就算是后来有金马奖加持，有这种疑问也是很正常的事情。但不同的是中共国的投资公司选择了否定，而台湾则有人选择了相信。《大象席地而坐》在台湾的发行权竞标竟然出现了各方争抢的局面，最后繁盛映画4万美元的成交价比一般大陆来的影片成交价足足高出了4倍之多。而根据台湾一般的票房分成方式，《大象席地而坐》要卖到约248万新台币以上片商才能回本。结果呢，上映仅三周票房就突破了600万新台币，这样叫好又叫座对于中共国无疑是一次狠狠的打脸，这个片子真的没法删减。

**从2013年开始，我去除了语言的修饰，又剥离了美化和塑造，将写作看作直面生活最有力的方式。于是从其中得到某种力量，以对抗世界的灰暗。**

**——胡波《大裂》后记**

胡波本人既是一名导演，也在写作小说。《大象席地而坐》这部电影就是改编自他自己的短篇小说，电影在台湾热映的同时小说也有热卖。并且胡波是金马电影学院的第六届学员，在金马电影学院的收获也构成他电影精神的一部分。一名既写小说又拍电影的导演，一名在金马电影学院学习过的导演，这样有才华的人最后却被逼得上吊自杀。甚至如果没有这个悲情英雄式的故事知道这部作品的人会少很多，金马奖鼓励他成为了一位真正的创作者，也用奖项对他表示最高的肯定，而中共国做的事情只会让人寒心。

没拿奖项的时候对于奖项十分看重，拿了奖项后又觉得获奖者不符合自己的要求，正常人对于中共的思维绝对理解不了。胡波的故事虽然很具有悲情英雄特色，但对于一名导演来说，他更希望的是自己的作品可以正常在电影院上映，哪怕没有票房起码也完成了作为导演的探索与尝试。然而中共能让你留下像这样悲情的结局已经不错了，更多的情况是人间蒸发或者泯然众人，虽然电影没说，但小说描写了大象席地而坐的原因，因为它的后腿断了。
