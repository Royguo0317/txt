## CCP滑向深淵——法國著手剔除華為5G
`2020-07-12 11:25 GM77` [轉載自GNews](https://gnews.org/zh-hant/262954/)

编撰：文秀、Pearl 珍珠、映兰轩、台灣荔枝、文肯尼

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/12110727/Huawei-696x392-1.jpg)

继英国宣布最快今年底停止使用华为5G之后，法国对华为的态度也开始有所转变。近来，一位多数党议员表示，法国决定仅对华为5G移动设备给予临时安全批准，这表明政府打算逐步淘汰华为公司产品。

法国网络安全机构负责人安西（Anssi）本周表示，将为使用中国公司产品的无线运营商给予5G限时豁免。该项措施的出台将是一个质的转变，因为直到最近，法国还试图站在中间立场，使华为能够保持重要的供应商地位，只是将其排除在无线基础设施的某些领域之外。

法国四家移动运营商中的其中两家–Bouygues电信和Altice-SFR在其4G网络设施中部分使用了华为设备，比例约占全国无线基础设施的22%，并计划在5G建设中再次使用华为。而据一位消息人士透露，政府打算将这一比例降至13%。

法国国民议会网络安全委员会成员埃里克-博托雷尔(Eric Bothorel)表示，这项措施将阻止法国供应商选择华为。他在电话采访中表示，”这将是对华为产品的逐步淘汰，但不会是完全淘汰，未来网络设施中使用华为设备的比例将大幅度下降。”安西（Anssi）总干事杰里奥姆 波巴赫德（Guillaume Poupard）周一在《回音报》的采访中也表达了同样的观点，即减少华为产品在法国通信网络设施中的使用比例是 “顺势而为”。

西方安全官员担心，华为的系统可能被黑客或敌对国家渗透，这给欧洲国家带来了两难的局面：在处理快速增长的数据流量和新的工业应用方面，这些国家严重依赖华为5G网络，如果中途更换供应商，损失巨大。但根据新的政策，安西对华为的限时授权只有三到八年。这对Bouygues和SFR等运营商构成了一定的技术挑战，因为如果将华为5G设备置于4G系统之上，三到八年后需要推倒重来，这样做是不现实的。这将逼迫他们不得不考虑成本更高的解决方案。还有一些运营商必须要拆除华为设备，但政府尚未表示是否打算对他们造成的损失进行补偿。

由于针对华为态度的改变，法国部署5G的时间进程将受到影响，而这一进程也已因COVID-19的持续流行而推迟。法国政府期望最快在今年9月举行5G设备招标，以跟上其他国家网络升级换代的步伐。

这次临时安全批准的决定反映出法国对全球最大的移动网络设备制造商华为的立场已趋于强硬。
