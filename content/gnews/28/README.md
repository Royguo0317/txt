## 拜登就中共国威胁向北约施压
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1315086/)

#### 编译：Boxinglady ( 文拳）
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/11115315/Screenshot-2021-06-11-8.52.38-AM.png)图片截自网络
布鲁塞尔——在周一与北约盟国会晤期间，拜登总统预计将敦促北约采取更多行动，以应对来自中共国不断上升的威胁，同时仍能威慑来自俄罗斯的持续威胁。

但他会遇到一些盟友的怀疑，他们质疑北大西洋公约组织可以对中共国发挥什么作用—中共国被视为在北大西洋地区几乎没有直接军事威胁——以及这种努力是否会分散该联盟的主要威慑目标俄罗斯。

北约于 2019 年开始正式解决中共国的崛起问题，并在联盟领导人会议后的一份声明中指出“我们需要共同应对的机遇和挑战”。

熟悉其准备工作的外交官表示，下周峰会后的公报将包括有关挑战和北约回应的更多细节。中共国将是该联盟计划的新战略概念的焦点。

在周五的新闻发布会上，北约秘书长延斯·斯托尔滕贝格指出了北约面临的挑战之一：“俄罗斯和中共国反对以规则为基础的国际秩序。”

周三，美国副国务卿温迪·谢尔曼 (Wendy Sherman) 提到了对中共国知识产权盗窃、网络黑客以及北京涉及台湾、香港和新疆的行动的共同担忧。她在智库美国德国马歇尔基金会的一次活动中表示，“这些问题是欧洲盟国关心的问题，显然也是北约关心的问题。”

但北约内部在如何接近中共国方面存在分歧。

美国希望与盟友结盟，向北京施加广泛压力。欧洲各国政府不想陷入对抗，并渴望保护贸易关系。匈牙利与中共国特别亲近，最近阻止了欧盟就香港新国安法批评北京的声明。

2019年以来，北约加强了对中共国的分析和信息共享。它加强了与澳大利亚、新西兰和日本的伙伴关系，尽管欧洲盟友希望排除这些关系针对北京的任何表现。该联盟在中共国控制基础设施的战略影响上温和地推动了成员。
