## 中國將失去WTO的“永久正常貿易”類別
`2020-09-23 09:26 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/379892/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/23092012/1-146.png)圖片來自：inventiva.co.in
《華盛頓郵報》（Washington Post）9月19日報導，隨著越來越多的人認識到中共侵犯人權和對CCP病毒傳播的虛假宣傳和故意隱瞞行為，中共政府正飽受來自國際社會的輿論攻擊。現在，阿肯色州共和黨參議員湯姆·科頓(Tom Cotton)提出的一項新法案將幫助美國製止北京猖獗的濫用侵權行為。該法案提出的時機已經成熟。

美國對中共國的“永久正常貿易關係”（PNTR），在原則上是中共國商品與我們一些最親密盟友的商品得到同樣的待遇。這種進入美國市場的方式推動了中共國對美國出口的迅速增長，這種增長為消費者提供了許多低廉的商品，同時也造成了美國製造業的崩潰。它使中共政府快速增加財富，並為軍事建設提供了大量資金，為其對外擴張和內部鎮壓提供了補貼。

科頓參議員的提案將採用2001年之前實行的年度審查制度來取代中共國的永久貿易地位。新的議案要求，總統必須每年裁定中共國的貿易地位，然後得到美國國會的批准。總統還必須審查中共國在一系列領域的行為，包括人權、使用奴工以及是否從事或縱容工業間諜和知識產權盜竊。因此，不論總統批准，或者不批准，都要經過國會，這樣美國對中共國的不當行為就有了反制的機會，以防止北京做出出格的行為。

這並不是第一次提出這樣的法案。 2007年，包括林賽·格雷厄姆（RSC）(南卡羅來納州共和黨)和謝羅德·布朗（Serrod Brown）(俄亥俄州民主黨)在內的兩黨參議員提出了一個類似的法案。儘管科頓的法案還沒有任何共同發起人，但不難想像，一旦總統競選的黨派壓力消失，這種情況將會改變。自上次提案到現在，13年來，北京的強勢和實力都在不斷增強。

中共國政府一直想要取代美國成為世界上最強大的國家。但是，中共國主導的世界會尊重國際上的法律和規範嗎? 中共國已經在利用自己的經濟實力威逼其他國家或組織跟隨自己的意志。如果中共國有足夠的實力，它不會承認世貿組織或其他西方建立的全球國際組織對自己不利的決定。相反，中共國正在利用西方規範來獲得最終取代這些規範的力量。美國及其盟國不能讓這種情況發生。我們現在有能力反擊。科頓的法案與中共國建立了相互制約的關係。如果它想要從與美國的貿易中獲得經濟利益，進而擴展到與西方的貿易，它就必須在某種程度上遵守西方的規範。如果是這樣，對美國的威脅就會減少。

根據北京的所作所為來審批最惠國待遇，最終也會惠及中共國的老百姓。台灣、新加坡和香港已經成為富有的國家或城市，它們的巨大成功表明，如果中共國積極參與，也會取得成功。這應該是普通中國人想要的: 繁榮、穩定和獨特的文化。讓羅納德·裡根(Ronald Reagan)出名的演講名為《選擇的時刻》(A Time for choice)。裡根認為，美國人面臨著“命運的交匯”: 他們可以選擇冒著所有風險的自由，也可以選擇最終走向共產主義暴政的更容易的道路。在對待共產主義中國時，我們面臨再次選擇。

美國仍然是世界上最強大的國家。科頓的法案將讓我們永遠使用這種權力，同時讓中國政府做出艱難選擇，是發展經濟還是堅持共產主義？我們應該再次選擇自由，選擇科頓的法案。

原文作者Henry Olsen是《華盛頓郵報》專欄作家，也是道德與公共政策中心的高級研究員。

評論：

中共國自從加入WTO以來的表現，有目共睹，已經對WTO 的規則造成了根本性的破壞。中共國一直不承擔維持國際經濟秩序的責任。 2010年中共國成為世界第二大貿易體，但是，在參與全球貿易的過程中，說得好聽一點，一直起消極作用，說得不好聽一點，一直起破壞作用。這對世界貿易體制提出了挑戰。科頓應對中共國的做法提出了新法案，以約束中共國在WTO中的行為準則，這一法案直接取消了中共國的“永久正常貿易關係”，也就是最惠國待遇。這對中共是一大打擊，但逼迫中共遵守WTO的規則，對國際貿易的公平競爭無疑是有利的。
