## 中共存在瑞士銀行的資產將被過渡到新中國聯邦
`2020-09-02 07:15 GM31` [轉載自GNews](https://gnews.org/zh-hant/329879/)

郭文貴先生在2020年8月29日與慕尼克戰友聯線時接受戰友採訪。有戰友問：瑞士十一月公投會涉及瑞士的企業和侵犯人權的國家和個人的關係，存在瑞士銀行的盜國賊的他們的財產未來會怎麼處理呢？

郭文貴先生透露：“我會告訴你百分之八十以上的可能，所有在瑞士存的錢都會過渡給我們新中國聯邦，來作為未來的新中國的代持。”
