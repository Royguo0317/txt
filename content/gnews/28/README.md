自11月3日美国大选以来，几乎每天都有各种大选欺诈舞弊的报导。这可能是人类历史上最大的盗窃案了，但这次盗的不是金银珠宝，而是美国接下来的4年总统的宝座。更为滑稽与不解的是，这种事情居然发生在伟大，民主，自由，一直被称作世界警察的美利坚共和国。我想，就算是包青在世，对此案也可能无能为力，只好说“元芳，你怎么看？ ”也就罢了。

出身于知名大学知名经济教授的纳瓦罗，被川普总统4年前直接挖到了美国白宫，做了白宫经济顾问。在华盛顿这个什么都缺唯独不缺政治精英的沼泽地带，川普总统和纳瓦罗可以说是华盛顿外来人口，没有政治资本，只能依法实干。面对严重违规的大选，12月17日，纳瓦罗发挥作为专家教授的特长，以非白宫人员身份发布了一份长达36页的分析报告（与其说是报告，不如说是学术论文）。此文先在班农先生的War Room发布，接着川普总统也转推。该报告搜集和分析大量实锤证据，包括超过50份诉讼与司法裁定，成千上万的宣誓书，声明和证词，智库和律师事务所的公开分析，视频与照片，和公众评论等，无懈可击地论证了这是一场对六个摇摆州涉及到六大关键层面选举违规的’完美欺诈’。相信该报告对唤醒普通大众了解大选真相有重大的推动作用。

将纳瓦罗的这份报告结合文贵先生与路德社的多次爆料，和美国情报总监所透露的情报来看，这是一场由中共，美国黑暗力量以及其它外国势力策划已久，手段之多，利用资源之广，行动之猖獗的选举欺诈，目的就是不让川普总统连任，或者至少要搞乱美国。

作为天选之子的川普总统自从2016年上任美国总统开始，在文贵先生领导的爆料革命，班农先生主导的平民主义，和其他正义力量的支持下，开始了对抗中共的艰难路程。先是以中美贸易协议出手，接着对华为等中共背景的企业制裁，触到中共的痛点。一贯邪恶的中共为继续稳固江山，实现野心、掩盖罪行，在香港反送中事件和没有得逞的通俄门弹劾案后，中共开始向世界尤其是美国释放了冠状病毒，目的当然是想搞乱美国，逼川普总统下台。但是，上天派了闫丽梦博士来到爆料革命，向美国向世界揭露病毒来源的真相。世界正义力量对中共和美国沼泽等黑暗力量步步为营，再加上亨特·拜登的硬盘门。除了地狱之门，中共已无路可走。于是乎，中共与其长期渗透勾结的美国沼泽力量开始了策划已久的”终极之战”, 用尽一切手段与力量来干预美国大选。心想：”在过去4年没有把你川普总统搞下台，再怎样也不能让你川普连任下面4年了，不然我们怎么活？！”，”就算你川普总统可能继续连任，至少我可以把你美国搞乱搞死，这样我们也可以多活几年，等待满血复活后再战。”

本以为这次的大选欺诈堪称“完美”，实在没想到川普总统的民意如此强大。这就导致他们的行动变得非常公开，非常猖獗，很容易的就被抓到了大量实锤的证据。再加上情报，川普总统以及文贵先生对他们的犯罪动机和手段早已掌握在手，只等引狼入室，抓个正着了。我想中共和美国沼泽力量对这种形势与结果也是早有预料，只是迫于时间的紧迫，他们没有其它更好的办法，只能不顾及颜面，先来个”既成事实”再说，然后再上演一场”皇帝的新装”。证据摆在那，只要没看见，就是不存在。主流媒体没有看见，反而帮”皇帝”喊口号；几大自媒体也没有看见，反而还对想说实话的小孩不让发声；无论是民主当还是共和党，当事的政府官员没有看见，反而帮助”皇帝”穿新衣；立法和司法机构没有看见，反而对前来诉冤的挡之门外⋯就这样，看似一场”皇帝的新装”一直可以演下去，一直演到拜登当选下任总统。

然而，幻想终究不能成为现实，”皇帝的新装”终究被喊出来。随着正义力量的不断努力，越来越多的老百姓开始明白真相。纳瓦罗这篇36页的报告，将对选举违规真相的披露上升至一个新的台阶，对激起有利于川普总统的民意大有帮助。然后等美国情报总监发布其大选欺诈与干预的报告后，在政府，国会，军队，以及司法层面将中共和美国沼泽力量干预大选而一锤定音。再加上生物武器，接下来由川普总统主导的行动将如天雷滚滚般砸向中共和美国沼泽力量。

此前美国国务院几条重磅推特，把中共和中国人分开，直指中共奴役中国人绑架美国利益，资本渗透，偷窃科技最后全面破坏美国政治体制，信仰和价值观，输入中共意识形态。国务院这种连批乃历史罕见。接着，川普总统签署外国公司审计法，与中共的金融开始脱钩；商务部制裁中芯国际等中共实体企业或大学，科技开始脱钩；对中共60家公司国际清算银行制裁。再接着，美国国防部宣布暂停总统过渡。

最新消息报导，手工重新计数显示，川普在密歇根州安特里姆县获胜！重新计票显示，川普以9,759击败拜登5959票。选举夜川普以约3,000票的差距惜败。本周初，对安特里姆郡的Dominion投票机进行了法医审计。该报告称，Dominion投票系统是“故意和有目的地设计，带有固有的错误，从而造成系统性的欺诈并影响选举结果。”

面对这人类的终极之战，以川普为首的正义力量绝不会妥协。军队不认作弊上来的“总统”，军队守住底线，其他战线继续进行。最终就看是用和平手段瓦解还是强力手段挽救这场被欺诈被干预的大选，解决这场名副其实的政变。
