04月15日，由于美国的制裁，以及乌克兰的局势，波兰外交部已经召见了俄罗斯大使。这是由当地新闻机构PAP报道的。外交使团向塔斯社证实，谢尔盖·安德烈耶夫大使被传唤到外交部。 “大使今天确实被传唤到了波兰外交部。现阶段不会发表任何评论。”使馆发言人弗拉基米尔·亚历山德罗夫说：

此外，据路透社报道，英国外交部还就俄罗斯的“敌对活动”传唤了俄罗斯驻伦敦大使。

外交部在其网站上表示，英国与美国一样，对“俄罗斯持续的恶意活动”感到担忧。它声称，俄罗斯对外情报局（SVR）据称是一系列网络攻击的幕后黑手，其中包括全球几百个组织以及多个美国政府机构受到太阳风（SolarWinds）软件黑客事件影响。

“俄罗斯在世界各地的恶意行为模式–无论是在网络空间的（行动）、干预选举还是其情报部门的侵略行动–都表明，俄罗斯仍然是英国国家和集体安全的最严重威胁。”该部门网站指出。

英国外交大臣多米尼克·拉布表示，英国将与合作伙伴一起继续查明非法活动。“我们看到了俄罗斯破坏我们的民主制度的行为。英美指出俄罗斯的恶意行为，是为了让我们的国际伙伴和国内的公司能够更好地保护自己，并为这种行动做好准备。英国将继续与盟友合作，查明俄罗斯的恶意行为。”拉布说。

此前，据白宫新闻处报道，美国总统拜登签署了一项法令，实施新的反俄制裁。推出新制裁的解释是，希望“让俄罗斯为其政府和情报部门侵犯美国主权和利益的行为付出代价”。

根据该行政命令所述，俄罗斯应负责：

·企图破坏美国和盟国的民主选举；

·参与对美国和盟国基础设施的黑客攻击；

·跨国腐败以影响外国政府；

·企图在俄罗斯境外伤害持不同政见者和记者；

·损害对美国国家安全有重要意义的国家和地区的安全，包括侵犯其他国家的领土完整。

根据美国总统令，财政部外国资产管制办公室禁止国内银行和其他金融机构在2021年6月14日后在一级市场购买俄罗斯银行、国家主权基金或俄罗斯财政部发行的以卢布等货币计价的债券，并禁止向后者提供贷款。此外，白宫表示，如有必要，可能扩大对俄罗斯国债的制裁。

俄罗斯外交部发言人玛丽亚·扎哈罗娃则表示，莫斯科打算对华盛顿的行动给予“果断回击”。“对制裁的反应将是不可避免的。”她说。
