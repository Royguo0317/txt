**二、新中国联邦的未来规划**

**1、使命和目标**

结束中共专制集权统治，剥夺其政权合法性，重新构建一个以正道主义为核心信仰，三权分立、民主法治的国度，重新塑造中国人在国际上的形象，重新以崭新面貌融入国际大家庭，愿意与世界人民和平共处，共同发展经济，维护和平、民主、法治、自由的国际秩序，共同实现人类的和谐、高效、可持续的生存环境和发展环境。

**2、选举和监督**

新中国联邦权力在民，选举是权利的前提。新中国联邦任何的权力，可以经由网上实名投票的合法公民一人一票选举产生，选举由新中国联邦国家宪法规定和保障。新中国联邦立法、行政、司法权三权分立，分别由议会、总统和最高法院独立行使相应权力并相互制约，司法完全独立并实行陪审团制度，立法机关对行政和司法机关依据宪法独立行使弹劾权。联邦权力和地方权力非等级制关系，地方权力不是联邦权力的绝对执行机构，而是根据事项确定不同的权力分属、实施和担责。

新闻媒体不隶属于任何政党，独立行使公民知情权从而达到监督作用，新闻媒体对发布信息的真实性和公正性负责。

**3、宪政、人权和财产权**

新中国联邦宪法由公民一人一票的形式确定。一经确定，任何个人、团体和党派都必须无条件遵守。

新中国联邦公民不分城乡、地域和身份平等地享有宪法规定和保护的包括但不限于生命权、发展权、允许拥枪绝对自卫权、选举与被选举权等在内的所有人权；和包括但不限于言论、集会、宗教信仰自由在内的基本自由。

新中国联邦保护各类私人合法财产永久不受侵犯和剥夺。

**4、教育、医疗、就业、养老等社会保障**

新中国联邦公民人人享有受教育的权力。联邦政府自学龄前开始普及双语教学，并有义务提供自小学入学始至大学入学前的免费教育。参照有关国际惯例，大学教育由国家建立教育补助基金或提供无息贷款方式解决。学前教育由国家根据家庭收入情况适当补助解决。

政府应通过建立基本医疗保险和大病保险等复合医疗保险体系，逐步实现新中国联邦公民人人免费医疗。

政府应建立失业保险和临时应急救助计划等方式对失业人员进行救助。

政府应通过建立基本养老保障金、强制退休金计划和鼓励退休金计划在内的多重保障计划，实现新中国联邦公民人人老有所养。

政府应同时加大对儿童的社会保障力度,并对教师、医护人员、退伍军人等特殊群体的社会保障予以倾斜。

**5、军队和国防**

新中国联邦军队属于国家，不隶属于任何党派。总统拥有受宪法规定可行使的最高军权。致力于建设一支自卫和国际维护和平的力量。联邦负责国防事务，大幅度削减国防开支，国防开支绝对透明化。不生产、不首先使用核、生、化等大规模杀伤性武器。新中国联邦愿意与国际社会一起打造人类千年和平计划，同世界人民一道开创人类永续和平发展的新纪元！
