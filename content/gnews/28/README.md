据零对冲(ZeroHerge)11月5日报道，川普总统竞选团队称，“他们在内华达州提起关于全州共有1万多张选票由不住在州内的选民投出的诉讼。”

大纪元的杰克∙菲利普斯(Jack Phillips)报道，在周四的新闻发布会上，民主党总统竞选选手乔∙拜登(Joe Biden)在内华达州获得了超过本州居民数的大约8000张选票。内华达州拥有影响总统竞选结果的6张关键选举人票。

川普内华达州竞选主席阿旦姆∙拉克萨尔特（Adam Laxalt ）说，“这个山区在垃圾桶中发现邮寄选票，有人在邮箱中发现18张选票，这是不合规则的投票证据。我们尚未有机会检查这些选票签名，也没有特意检查几十万张邮寄选票。” 他补充说，“甚至有些计算的选票来自已经去世的选民。”与此同时，他声称，“数千名被纳入选票统计的人已经在病毒流行期间搬出克拉克县。” 克拉克县包括该州最大的城市拉斯维加斯。因此，他表示川普的团队正在寻求紧急方案，并正在阻止所有这些县的投票。

川普前国家情报代理局长里克∙格伦内尔（Ric Grenell）在会议上说：“如果你住在内华达州没达到30天就投票是非法的，我们因此提出法律诉讼，旨在保护合法选民。” 克拉克县对此尚未有任何答复。

川普竞选经理比尔∙斯特派恩（Bill Stepien）告诉记者，川普在宾州赢了约20万选票。杰森∙米勒(Jason Miller)表示，最早在星期五晚些时候将会明确川普将再连任四年总统。米勒补充说，“他期望额外的法律行动能够确保透明地计算先前的选票。”米勒还声称，“川普应该是赢得亚利桑那州的。”内特∙西尔弗（Nate Silver)说过:“福克斯新闻和美联社应该撤销对拜登的呼应。” 杰森∙米勒还透露内华达州的其它合法行动可能在今天晚些时候发生。

共和党观察员将能够在宾夕法尼亚州观察计票，这得益于该州上诉法院的裁决。乔治亚州州长拉芬斯堡（Raffensburger）正在主持一场乔治亚州的计票简报。

川普团队11月5日在费城举行新闻发布会，概述提出诉讼并停止计票的计划。川普总统在11月5日的推特上也宣布选举日后的投票应该不予计算。

川普普竞选团队星期三在宾夕法尼亚州、密歇根州和佐治亚洲提出了法律挑战，民主党嘲笑此举。 共和党要求竞选观察员更容易进入正在处理和计算选票的地点，并妥善解决对缺席选票的担忧。

费城记者招待会不久后重点将转移到内华达州，这个州目前是民主党总统提名人乔∙拜登暂时领先。内华达州政府官员准备在美东时间周四中午发布选举结果，川普政府计划在11点30分举行一场新闻发布会，可能会宣布指控民主党人欺诈和渎职的诉讼。拉斯维加斯的新闻界人士预计将从11:30分开始接待各种人士，包括前情报官员Richard Grenell，前内华达州检察长Adam Laxalt，美国保守党主席Matt Schlapp和内华达共和党主席Michael McDonald。

里克∙格雷内尔(Ric Grenell)计划周四上午在拉斯维加斯举行记者招待会宣布川普竞选团队将提出一项计算每张合法选票的诉讼。川普普竞选团队说，“至少有1万人不在内华达州里生活却在州里投票。”

宪法专家乔纳森∙图利（Jonathan Turley）最近在福克斯新闻报道上解释：川普政府的战略是在宾夕法尼亚、密歇根和佐治亚等州提起诉讼，因为那里存在着选票欺诈的行为。
