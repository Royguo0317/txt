## 英议员呼吁清洁公司供应链打击新疆强迫劳动

路透社3月17日报道，英国议员周三表示，英国应考虑打击中共在新疆地区对维吾尔族的强迫劳动，将无法展现清洁供应链的公司列入黑名单，并对违反《现代奴隶制法》的公司施以严厉惩罚。

商业、能源和工业战略(BEIS)审查委员会表示，一些公司在供应链中“故意无视”滥用劳动力的风险，而政府未能正确执行人权标准。委员会成员努斯拉特·加尼(Nusrat Ghani)在一份声明中表示，“在有令人信服的虐待证据的情况下，令人遗憾的是，英国政府没有出台新措施，禁止英国企业从新疆和中共其他地区维吾尔人的强迫劳动中获利。”他指出，“向数百万英国客户销售产品的公司不能保证它们的供应链不受强迫劳动的影响，这令人深感担忧。”

中共驻欧盟大使张明周二表示，“仇恨中共者”正在为政治利益散布谎言。他对欧盟计划对其侵犯人权的行为实施制裁提出警告，称如果布鲁塞尔干涉中共内政，北京不会让步。

中共国80%以上的棉花来自新疆。反奴隶制活动人士警告称，尽管多数品牌表示，它们与新疆的工厂没有关系，但它们的供应链很可能受到维族人采摘的棉花的污染，这些棉花出口后被其他供应商使用。

议员们表示，许多品牌报告称它们采取了强有力的程序来防止侵犯人权，但未能进行“必要和基本的尽职调查”，以确保它们的供应链没有牵涉到强迫劳动。与此同时，政府表示2015年的《现代奴隶制法案》(Modern Slavery Act)现在已经过时，不能有效地对付违反规则的人，而且BEIS审查委员会表示，目前政府尚“没有迹象”执行对英国商业与强迫劳动之间联系的调查。

该委员会表示，在新疆运营的公司必须证明，它们没有违反《现代奴隶制法》(Modern Slavery Act)。它要求政府发布一份证明自己没有违规公司的白名单，并将那些未能提供证据或回答问题的公司列入黑名单。同时提议政府应加强企业的供应链透明度义务，并对不合规行为实施“严厉罚款”。

BEIS审查委员会正在推进计划，将《现代奴隶制法》(Modern Slavery Act)的适用范围扩大到一些公共机构和大公司，并对不遵守该法案的组织实施财务处罚。声明中指出， “强迫劳动是世界上最卑鄙的行为之一，政府不会容忍这种剥削，无论这种剥削发生在英国还是国外。”
