##  **葉望輝: 2020年—美國毛主義文化大革命的發動？** 



對於大多數美國人而言，我們當前這場“不滿之夏”的危機猶如一場晴天霹靂。它的開始是由於一些憤怒的暴民在城市裡發起的暴力攻擊行為，進而引發成為一場暴亂事件，他們呼籲推翻所謂的不公正的舊秩序。他們使用的方法從大規模遊行示威到破壞政治歷史標誌，再到面對面的和在線的“批鬥會”，在當今的行話中通常被稱為“封殺文化”。

表面上看，當前這場運動的催化劑是發生在5月25日的喬治·弗洛伊德（George Floyd）的窒息死亡所引發的國際事件。前警官德里克·喬文（Derek Chauvin）跪在受害者脖子上的可怕照片和事實，這在遭到普遍譴責和聲討的同時，也引發了強烈的政治和媒體反應。這也推動了一場關於抗議系統性不公正的大規模抗議運動，而其宣稱的目標卻遠遠超出了明尼阿波利斯（Minneapolis）悲劇的核心，即種族和警察行為失當問題。

當他們夾雜著憤怒，同情著發生在明尼阿波利斯的不公平事件，以及要求得到系統性的回應的同時，許多美國人仍有理由擔心，當前的群眾運動不是在呼籲改革或恢復公民權。這是對美國自身基礎的有組織的攻擊。

對個人，機構和財產的有針對性大規模襲擊與21世紀的美國經歷是格格不入的。修正主義者對各種公共紀念物的襲擊令人困惑。這些在暴民面前沒有屈服並贖罪的公眾人物和公司遭到的羞辱和折磨是著實令人震驚的。

因為某種原因，我們目前的經歷讓我們感到既陌生又不怎麼美國。這其實是半個世紀前中國共產黨文化大革命的迴聲。

毛澤東害怕失去對自己在1949年成立的“新中國”的控制，於1966年5月發動了無產階級文化大革命，以“除四舊”（舊觀念，舊習俗，舊習慣， 和舊文化）的方式淨化國家。

《衛報》 2016年的一篇文章很好地總結了文化大革命的使命和策略：

- “我們的目標是打倒和粉碎那些走資本主義道路的當權者……以便促進社會主義制度的鞏固和發展。”
- 隨著對“封建”傳統攻擊的開始，學校和大學關閉，教堂，神社，圖書館，商店和私人住宅遭到洗劫或破壞。
- 戴著紅色袖章和穿著軍服的青少年團伙在北京和上海等城市的街道上游盪，攻擊那些穿著“資產階級”的衣服或梳著反動髮型的人。 “帝國主義”路牌被拆除。
- 黨的官員，老師和知識分子也發現自己是被瞄準的對象：他們在邪惡的“批斗大會”之後被公開羞辱，毆打，在某些情況下被謀殺或自殺。


陷入馬克思主義的激進青年，釋放了攻擊和推翻一個國家的傳統經濟，政治和社會基礎而支持專制社會主義的能力。聽起來有點耳熟？

批鬥會的現代化和日趨完善也是引人注目的。

“批鬥會的目的是塑造民意，侮辱，迫害或處決政治對手和那些被視為階級敵人的人。 總的來說，鬥爭的受害者被迫接受一大群人在語言上地謾罵和肢體上地攻擊直到他或她對之前所承認的各種罪行供認不諱。批鬥會通常在被告的工作場所舉行，但有時會在運動場館進行，如果目標眾所周知，那麼運動場會聚集很多人。”

恭喜21世紀被喚醒的暴民們，他們將互聯網，社交媒體和“智能”電話視頻的功能添加到了這個20世紀版本的古羅馬競技場的體驗中……毛澤東的紅衛兵都會欽佩你對這場封殺文化的表演。

儘管許多主流媒體都迴避了當前這場有組織的，激進的和有革命性因素的抗議浪潮，但有理由質疑這樣一種觀點，即我們正在目睹的一切僅僅是對系統性種族主義積累事件或是對不公正的一種本能的和自然的反應。

去掉紅袖章和1966年共產黨的指示，這樣一個資金雄厚，訓練有素，而媒體可以作為新的保駕護航方式被操控的組織，似乎正試圖在2020年發動美國毛主義文化大革命。這些美國革命者的目標雖然不能與中共國毀滅性十年的規模相提並論，但其目標同樣大膽。

美國主流黨及其民選代表早就應該承認、組織和反對這場激進的革命了。
