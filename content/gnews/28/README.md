**以下翻译引自《大流行病造富：新冠病毒疫苗利润造就了****9****位新晋亿万富翁》**

当生产新冠病毒疫苗公司的股价急剧飙升后，至少9位亿万富翁从此诞生。

在这份新晋亿万富翁名单中，名列前茅的当属Moderna首席执行官Stéphane Bancel和BioNTech（和辉瑞合作生产疫苗的公司）的首席执行官Ugur Sahin。他们二位的身价都在40亿美元左右。

伴随股价飙升，中共国康希诺生物制品公司的高管和Moderna公司的早期投资者成为账面上的亿万富翁。其中部分原因是对新冠病毒疫苗的利润预期，以及对公司未来良好前景的预期。

2020年2月以来，Moderna的股价已经上涨了超过700%，BioNTech则飙升了600%。康希诺的股价同一时期上涨了约440%。康希诺的单剂量新冠病毒疫苗已于2月批准在中共国使用。

一些活跃人士说，这种财富的产生凸显了大流行病中残酷的不平等。这9位新晋亿万富翁总身价为193亿美元，足够为低收入国家7.8亿人全面接种所谓的疫苗。

慈善组织乐施会健康政策经理万豪（Anne Marriott）在一份声明中说：“这些亿万富翁所体现的人性是制药公司从他们对这些疫苗的垄断中获得了巨额的利润。”她还说到：“这些疫苗是由公共资金资助的，首先应该是全球的公众产品，而不是私人的谋利机会。”

本周五正好恰逢G20集团全球卫生峰会，世界各国领导人将在会上讨论放弃疫苗的知识产权保护。

拜登支持这种举措，支持者认为这样有助于扩大全球供应，缩小各国间的疫苗接种差距。而反对者，诸如德国，认为保护知识产权对创新至关重要，即使取消专利，因生产能力有限和疫苗成分不足，对提升供应的意义不大。

从德国政府获得3.25亿欧元用于开发疫苗的BioNTech公司表示，它承诺向低收入国家提供疫苗。在一份与CNN商业版共享的声明中，该公司表示，复杂的制造工艺和建设新工厂是挡在增加全球疫苗供应面前的主要障碍之一。专利反而不是限制性因素。

BioNTech在今年前三个月获得了11亿欧元的净利润，主要归功于新冠病毒疫苗的销售额，而去年同期该公司则亏损5340万欧元。

Moderna公司本月早些时候报告说，今年前三个月的新冠病毒疫苗销售额达17亿美元，是有史以来第一个盈利的季度。高盛预计Moderna 2021年新冠病毒疫苗收入将达132亿美元。该公司是从美国政府获得的数十亿美元资金来用于开发疫苗。

康希诺生物制药和Moderna公司未对此事做出回应。
