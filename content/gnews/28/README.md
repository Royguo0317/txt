中共国推迟了人口普查数据的公布。它仍然是世界上人口最多的国家。但是，在饥荒过后出生的孩子最少的一次普查结果。 对于北京的大国计划来说，这是个坏消息。

中国的统计学家对此感到兴奋。实际上，他们想在1月份发布最新的人口普查结果。病毒流行原因导致统计结果推迟到四月。

人口普查对北京来说是一件大事。中共国每十年统计一次人口。结果表明，未来世界人口最多的国家和世界第二大经济体将如何发展。但是早在1月份，区域统计数据就表明北京存在问题。最近，有报道甚至暗示中国人口在过去十年中有所减少。

中国统计局长宁吉喆在周二介绍了统计数字。他是按照党的通常要求这样做的。 新闻稿说，第七次全国人口普查已经完成了这一计算。这表明人口统计学是重中之重。在中国总统的待办事项清单上排名很高。

这就是数字所代表的意思：根据国家统计局的数据，目前有14.1亿人居住在中共国。这比上次人口普查的2010年增加了约7200万。或相对而言：增长约5.4％。

从欧洲的角度来看，这听起来可能令人印象深刻。毕竟，这意味着过去十年来中国的人口增长超过法国。相比之下：德国目前有大约8300万居民。自1990年代初以来，人口仅略有变化。自1992年以来已超过8000万。

但是从中国的角度来看，7200万的增长幅度很小。这是自1953年进行第一次人口普查以来的最小增幅。这些数字驳斥了有关中国人口减少的报道。但是，即使通常被粉饰的党派媒体现在也认为该国在未来一两年内人口将会减少。
