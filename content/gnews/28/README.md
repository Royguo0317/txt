拜登总统发布病毒调查令后，5月26日，参议院一致通过了一项“新冠病毒起源2021年法案”，该法案要求拜登政府和国家情报局局长，将有关中共武汉病毒研究所与病毒起源关系的情报解密。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/28025738/%E5%9B%BE%E7%89%875-31.png)

共和党参议员乔什·霍利参与了发起这份议案，他在法案通过前在参议院发言说：“美国人民应该了解COVID-19的起源，他们应该知道这场席卷全球的可怕流行病是如何开始，如何蔓延的，以及中共国在此过程中扮演了什么样的角色。”这项法案是由共和党参议员乔什·霍利和迈克‧布劳恩上个月提出的。该法案要求国家情报局局长在90天内解密有关中共国实验室与病毒之间联系的“所有信息”。该法案还要求DNI“尽可能多地”公开提供有关病毒起源的信息，以便“尽快”识别该起源。布劳恩参议员在参议院发言中说：“现在很明显，实验室泄漏理论是完全有可能的，美国人要真正的答案，我们不会从世界卫生组织的拙劣调查中得到这些答案，也不会从中国共产党政权的掩盖中得到这些答案。”

这项两党一致通过的法案在拜登总统发布情报界彻查病毒来源的命令一天后通过，实属罕见。如果该措施下一步能通过众议院并由拜登总统签署为法律，则国家情报局局长办公室将在90天内，要对中共病毒研究所和新冠病毒起源之间的所有情报进行解密。

美国媒体界、军方、政界和科学界纷纷公开表态要彻查中共掩盖疫情真相，直指中共病毒就是来自中共国实验室，并在各自的领域上推动，纷纷表示要向中共国追责。最近很短的时间里，追查中共病毒成了美国的政治正确。一些与中共有牵扯关系的人，都纷纷弃中共而去，文贵先生说：“在福奇讲话之后，实际上美国向全球发出了信号，要向中共追究病毒真相，全球各个国家都必须马上选边站。” 在病毒彻查真相问题上，美国的左中右派全面合作，形成合力，将会越走越快，然后带动西方盟国和其他国家，共同灭共。
