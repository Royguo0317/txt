## 陰陽難測的中共病毒：山東菏澤通報壹例復陽無癥狀感染者
`2020-09-15 10:08 GM64` [轉載自GNews](https://gnews.org/zh-hant/357825/)

內病毒專題：文文 校對：天藍色獨角色
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/15095624/screenshot_2020-09-15-21-13-43-572_discord.jpg)
山東菏澤市曹縣9月15日通報了壹例奇怪的感染病例，其奇怪在於“復陽”“無癥狀”，更怪在中共馬上宣布“傳染性較小”，也不知道中共怎麽確認的傳染性較小。文貴先生在疫情爆發初期就強調過病毒的壹大特點就是“陰陽難測”，這個病例生動的詮釋了邪惡病毒的這壹特點，可見文貴先生情報之準確。

這壹病例的診斷和治療經歷了如下過程：
無癥狀感染者➡️ 治療痊愈➡️ 隔離14天➡️ 兩次核酸陰性+壹次抗體陰性➡️ 復檢核酸陰性➡️ 核酸及抗體陽性

馬某，男，菏澤市曹縣人，2020年8月15日從迪拜抵達廣州，8月16日診斷為無癥狀感染者，在廣州市接受治療，愈後於8月21日至9月5日集中隔離，9月5日乘坐飛機抵達鄭州。9月6日、9月7日兩次核酸和壹次抗體檢測均為陰性。9月12日復檢核酸檢測為陰性。9月14日再次復核核酸及抗體呈陽性，為復陽無癥狀感染者，中共經拍腦袋以後，確定傳染性較小。
