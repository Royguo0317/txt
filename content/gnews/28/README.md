## **“黑命貴”運動也令美國從中共設下的陷阱中醒來**

[!\[\]()!\[\](https://spark.adobe.com/page/Pdbqd4JISF2u2/images/683ca7c2-a0ae-4089-9f7c-a673c60c2c34.jpg?asset_id=3a7e224e-a290-4ae7-9a22-1409a27692de&amp;img_etag=%225d249690590ca2e4ebdf8c6cabc22d00%22&amp;size=1024)](https://spark.adobe.com/page/Pdbqd4JISF2u2/images/683ca7c2-a0ae-4089-9f7c-a673c60c2c34.jpg?asset_id=3a7e224e-a290-4ae7-9a22-1409a27692de&amp;img_etag=%225d249690590ca2e4ebdf8c6cabc22d00%22&amp;size=1024)

近日，隨著調查的深入，越來越多的中共是席捲美國“黑命貴”運動幕後力量的證據被起底，中共妄圖通過挑撥、撕裂美國各個階層、破壞美國社會秩序來影響美國大選，搞弱美國、搞亂美國、搞死美國的陰謀浮出水面。

9月16日，美國一家媒體發表了一篇名為《黑名貴聯合創始人和中共在美組織合作》的報導。報導說美國傳統基金會的岡薩雷斯調查發現，“黑命貴”創始人艾麗西亞.加爾薩開辦的企業，竟然接受來自中共在美國代理機構“中華進步會”的資金捐助。該企業還在公司的網站上堂而皇之地標明自己是“中華進步會”的資金贊助項目，赤裸裸地表明了自己的資金來源。 “中華進步會”是中共在美國間諜滲透的重要組織，曾經成功讓中共國的國旗在波士頓市政廳上空飄揚，是中共國官方喉舌媒體《中國日報》一貫表揚的組織。

無獨有偶，另一家美國媒體發現，中共通過各種方式支持美國極左勢力的1619項目。 1619項目是極左翼黑人漢娜·瓊斯於2019年在《紐約時報》開發的課題項目。該課題項目刻意渲染黑人在美國歷史上的苦難和地位，甚至要把黑人的歷史地位凌駕於美國歷史敘事之上。該項目核心觀點認為美國的建國年是1619年，即奴隸第一次到來的那一年，而不是通常公認的1776年的《獨立宣言》年。明眼人一看就知道這是個撕裂美國種族，製造種族仇恨的研究課題，中共自然不會放過這個絕佳的機會。中共國官方媒體中共國環球電視台（CGTN）在美一直著力推動最左翼的修正主義歷史倡議“1619項目”，對該組織對歷史的敘述給予充分肯定。還專門撰寫顛倒黑白的專欄文章《巴爾與特朗普試圖改寫奴隸歷史，但現實卻失敗了》，文中充斥著對川普總統和司法部長比爾·巴爾的嘲諷和人身攻擊，為美國國內種族分裂行動推波助瀾。

美國媒體對中共暗地煽動“黑名貴”運動的深入挖掘，進一步驗證了郭文貴先生幾年來向全世界揭露的中共的藍金黃和3F計劃。中共在正面對抗美國沒有勝算的情況下，發動各種形式的超限戰，搞弱美國、搞亂美國、搞死美國。這次黑命貴運動讓美國上上下下親眼見證了中共宣揚矛盾，製造矛盾，利用矛盾的高超本領，也讓美國民眾在社會秩序被破壞殆盡的恐懼中徹底醒來。中共支持的“黑命貴”運動，披著自由、民主、平等的華麗外衣，卻包藏動搖美國社會安定，煽動各種族之間仇恨，讓美國人鬥美國人的險惡用心。

翻遍中共的活動史就是一部無事生非、挑撥離間的暗黑史，無論是處於弱勢的國共時期，還是處於權力巔峰的竊國之後，中共外交和統治的厚黑權謀手段就從來沒有停止過。抗戰時期，國民黨在正面抗擊日本侵略者、保家衛國，中共卻躲在後方坐山觀虎鬥，在國民黨統治區，暗地煽動工人學生不斷製造麻煩，讓國民黨疲於奔命。甚至還暗中勾結日本人，和日本簽訂互不侵犯協議。 1949竊國後，中共更是把挑撥離間的手段玩到爐火純青，歷次政治運動讓群眾鬥群眾，讓中共國人恨中共國人，沒有信仰的中共國人被拆得四分五裂，中共無恥集權政府卻穩坐釣魚台。

越來越多的美國媒體和民眾看穿了中共通過“黑命貴”組織宣揚馬克思主義思想，製造社會階層分裂的把戲。 9月22日，一家媒體發現“黑命貴”組織竟然一直在其官方網站上宣揚，應該摧毀美國以家庭為核心的傳統價值觀，子女應該交給國家統一撫養。要知道公共撫養子女是馬克思主義的價值觀，這是共產黨方便給下一代集體洗腦的做法。在“黑命貴”的真面目曝光後支持率直線下降的情況下，“黑命貴”組織偷偷地把這些宣揚馬克思主義思想的網頁刪除了。最近甚至有黑人勇敢地站出來揭發中共的詭計。前美國國家橄欖球聯盟（NFL）名將赫舍爾·沃克（Herschel Walker）於9月21日透露，與中共有聯繫的資金正在資助美國“黑命貴”運動。

現在美國從上到下，無論是政府要員，還是平民百姓都已經意識到引起美國騷亂的不是執法事件，而是中共通過在美國各地的代理人伸出的黑手。要從根源上阻止極端暴力思想的傳播，要徹底掐斷騷亂運動的資金來源，唯一的釜底抽薪的辦法就是消滅中國共產黨！
