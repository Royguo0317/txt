## 中共一手调控楼市一手带风向让韭菜炒房

2021年4月17日,网易报道一则新闻标题为:惊呆！男子偷挪公款近7000万：炒股炒期货“血亏”1000多万，疯狂买房却增值50%，还买了辆“土豪金”宝马……

具体内容是这样的，据都市快报报道，今年年初，浙江台州临海市某电子信息科技公司在对账时发现巨大的财务漏洞，**有近****7000****万元资金被员工挪作私用了。企业账户在不知不觉间被掏了这么大一个****“****窟窿****”****，这让企业主惊恐不已，赶紧向临海市公安局报了案。接到报警后，很快锁定了犯罪嫌疑人****——****这家公司财务人员小李，将其抓捕归案。**

笔者认为中共国的新闻写手造假水平也太低级了，说谎也不先打一下草稿，毫无逻辑可言。据2018年12月25日，人民网就报道，2018年6月份，央行发布了《中国人民银行关于非银行支付机构开展大额交易报告工作有关要求的通知》(银发〔2018〕163号)。根据该通知要求，2019年1月1日起，包括微信支付，支付宝等在内的非银行第三方支付机构以下交易情况必须上报：其中第三条就规定自然人客户支付账户与其他账户发生当日单笔或者累计交易人民币50万元以上(含50万元)、外币等值10万美元以上(含10万美元)的境内款项划转。这一条就意味着个人账户的大额交易及流水异常将接受央行监控管理。

对这则新闻笔者有很多疑惑：

第一、这位财务人员小李有什么通天本领？身为普通老百姓，个人帐户三年有7000万的资金流动，央行数据中心没有掌控到他帐户的异常？难道央行数据中心的鼠标就在他手里控制着吗？

第二、请问这位记者，这位财务人员小李是如何把公款挪到个人帐户上，取现呢，还是通过转帐？在中共国有哪个省的税务局允许企业可以取现或现金进出7000万，或是允许从公帐上转出7000万资金到个人帐户上？这种私人企业早就被税务局以偷税漏税查到破产了，难道这当地税务局的局长就是小李他爹？

第三、难道这家某电子信息科技公司是当地的大国企？三年能被挪用7000万还没有发现资金问题，允许出现这种问题也只有国企才有这种可能性，私人企业别说三年7000万，就是被挪用7万都能马上被发现。

因此笔者认为此则新闻挪用公款是假，带风向让韭菜们赶快去炒房是真，中共国的房地产是彻底要崩盘了，根本卖不动，只能胡编乱造吸引眼球让韭菜们再被割一次。
