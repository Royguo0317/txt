举报人蒙哥马利曝光了奥马巴总统、前CIA局长约翰布伦南（John Brennan）和前国家情报局局长詹姆斯克拉珀（James Clapper）不仅监视了奥巴马的政治敌人唐纳德川普（Donald Trump）和迈克尔福林（Michael  Flynn）等包括100多名美国政府官员，还将监视遍及国会、联邦法官、美国企业和高管以及数百万美国人。
