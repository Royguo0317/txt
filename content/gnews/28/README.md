中共黨員如欲申請移民美國將遭拒絕入境,即使人已在美國,可能無法通過綠卡申請。

蓬佩奧出席四方會談 強化美日同盟。

蓬佩奧說,印太地區盟友一起合作、對抗中共的”剝削、脅迫和腐敗”,這比以往任何時候都更加關鍵。
