##  **白宮冠狀病毒案例凸顯病毒持續威脅** 


[!\[\](https://spark.adobe.com/page/yBrzeRDUiDRkg/images/81853366-de65-4c2f-b36a-fa7b6bac40e0.jpg?asset_id=2182dc7f-66dd-473a-abbc-ad7615c4f6ad&amp;img_etag=%22515357d21bd71f970968c4c9533ddc10%22&amp;size=1024)](https://spark.adobe.com/page/yBrzeRDUiDRkg/images/81853366-de65-4c2f-b36a-fa7b6bac40e0.jpg?asset_id=2182dc7f-66dd-473a-abbc-ad7615c4f6ad&amp;img_etag=%22515357d21bd71f970968c4c9533ddc10%22&amp;size=1024)
川普總統的國家安全顧問的冠狀病毒(COVID-19)檢測呈陽性，這凸顯了隨著美國病例增加，冠狀病毒的威脅——甚至對白宮的威脅，還在持續中。

白宮國家安全顧問奧布萊恩是迄今為止感染冠狀病毒級別最高的官員。直到上週，川普一直在努力將這種病毒（的影響）最小化。在周一彭博社首次報導這一情況之前，有關奧布萊恩檢測呈陽性的細節在白宮被嚴格保密。

這一情況一直困擾著川普週一去北卡羅來納州莫里斯維爾的行程，他在那裡強調了通過政府的“曲速行動”在冠狀病毒疫苗研究方面取得的進展。 (曲速行動計劃是聯邦政府為了研發冠狀病毒疫苗而實施的耗資至少100億美元的計劃) 。

川普週一下午離開北卡羅萊納之前在白宮對記者說，他“最近”沒有見到奧布萊恩，他完全不知道奧布萊恩是什麼時候被檢測出陽性的。川普還表示，他計劃當天晚些時候給奧布萊恩打電話。

從上週開始，川普在討論冠狀病毒大瘟疫時被迫採取更加現實和清醒的語調。起初他對最近案件激增的情況不以為然，後來承認情況“在好轉之前會變得更糟”。在強烈要求親自舉行在佛羅里達州，傑克遜維爾舉行的共和黨大會後，他突然取消了這個活動。

奧布萊恩檢測呈陽性對總統而言是他必鬚麵對的另一個壞消息。

白宮在向記者發表的一份簡短聲明中證實了這一消息，說奧布萊恩“症狀輕微，一直在自我隔離，並在安全地點工作”。白宮表示，川普或彭斯副總統都不存在被感染的風險。據報導，奧布萊恩在與家人休息了幾天后檢測呈陽性，據說是在女兒生病後感染的。

其他幾個在白宮工作或在其他地方與總統關係密切的人也被檢測出冠狀病毒呈陽性，不過沒有一個人職位比奧布萊恩高。由於與川普接觸，奧布萊恩經常接受檢測。上週，艾森豪威爾行政辦公樓的一名自助餐廳員工和一名被分配到總統直升機中隊的美國海軍陸戰隊員也被檢測出冠狀病毒呈陽性。

“這提醒我們，如果不採取預防措施，沒有人是安全的，無論他們在哪里工作或生活，”兩黨政策中心的首席醫療顧問阿南德·帕爾克說，“美國每天仍有近6萬例冠狀病毒確診病例。”
[!\[\](https://spark.adobe.com/page/yBrzeRDUiDRkg/images/d6b95298-2036-41cf-9ddc-f4bbbcd4498f.jpg?asset_id=3e9b2742-4bf2-4a3b-8456-a3632221d23b&amp;img_etag=%2263959b8b145123264c785adb8c0bd56f%22&amp;size=2560)](https://spark.adobe.com/page/yBrzeRDUiDRkg/images/d6b95298-2036-41cf-9ddc-f4bbbcd4498f.jpg?asset_id=3e9b2742-4bf2-4a3b-8456-a3632221d23b&amp;img_etag=%2263959b8b145123264c785adb8c0bd56f%22&amp;size=1024)美國總統川普，佐治亞州州長佈賴恩-坎普，衛生和人類服務部長亞歷克斯-阿扎爾和CDC副主任史蒂夫-門羅在佐治亞州亞特蘭大的疾病控制和預防中心，2020年3月Tom Brenner /路透社。 
公眾對奧布萊恩事件的關注與對5月初彭斯的新聞秘書凱蒂·米勒的冠狀病毒檢測呈陽性的關注相似。米勒與彭斯和白宮冠狀病毒工作組的其他成員保持著密切接觸，她被檢測的消息讓彭斯不得不與川普保持距離，出於謹慎，彭斯讓工作組的成員進行了自我隔離。

此後不久，白宮要求(包括總統橢圓形辦公室在內的)西廂辦公室的助手們佩戴口罩，除非是在他們的辦公桌旁。然而，這一政策在6月份變成“推薦”而不是“要求”使用口罩。根據哥倫比亞特區重新開放的規定，白宮在6月底也停止了對所有進入大樓的遊客和工作人員進行溫度檢測。

被問及白宮是否計劃在奧布萊恩檢測呈陽性後改變其規定，白宮發言人朱迪·迪爾在一份聲明中表示，總統的醫生和白宮部門會”密切”合作，結合目前疾病控制和預防中心的指導方針和最佳做法，採取預防措施來確保總統、第一家庭和整個白宮大樓始終安全、健康。”

迪爾說:“除了保持社交距離、隨時可用的洗手液、定期對所有工作空間進行深度清潔和麵部遮蓋之外，繼續對那些接近總統的人進行冠狀病毒檢測，以確保最大程度地限制接觸。 ”

帕爾克曾擔任美國衛生與公眾服務部副助理部長，他說，個人在任何公共場合都應該戴布製口罩，並建議白宮規定，在白宮西翼的公共區域必須戴口罩。

奧布萊恩是川普最看重的顧問之一，經常向川普匯報國家安全方面的情況。 7月10日他最後一次隨總統前往位於佛羅里達州多拉的美國南方司令部，聽取有關打擊販毒工作的簡報。有時可以在白宮的車道上看到他在電視採訪後和一小群記者講話。

白宮最近的事態發展凸顯出及時獲得檢測對防止未被檢測的冠狀病毒傳播的重要性。隨著病例的增加，美國各州和地方出現了檢測延遲，這使得迅速隔離病例和進行接觸者追踪變得更加困難。佛羅里達、加利福尼亞和德克薩斯等州經歷了最令人擔憂的病例激增。

雖然在白宮內並不是必須戴口罩，但一些官員用自身行動來鼓勵在公共場合戴口罩，這包括了副總統和總統的女兒兼高級顧問伊万卡·川普。

奧布萊恩是這些戴口罩的官員之一，一位政府消息人士說，國家安全顧問一直非常認真地對待冠狀病毒。在今年3月的一次演講中，奧布萊恩敦促公眾遵循美國疾病控制與預防中心的指導意見，比如洗手、生病時呆在家里以及保持一定的社交距離。

不過，其他官員還沒有在公開場合戴過口罩。川普只是在最近向沃爾特里德醫療中心展示了對戴口罩的強烈支持，他戴了一個有總統印章的口罩，後來又懇求美國人戴口罩，此前他已經拒絕在公開場合戴口罩好幾個月了。

週一下午，川普在參觀莫里斯維爾的生物技術設施時再次公開戴上了口罩。

白宮經濟顧問拉里·庫德洛週一在白宮外接受電視採訪時，首次戴著口罩對記者說話。

當被問及他的口罩時，庫德洛說:“我強調戴口罩已經有兩三週了。我們不會保持經濟開放，我們不會實現V型複蘇，我們不會讓孩子們重返校園，除非我們遵循簡單的準則——戴口罩、保持距離、接受檢測和（保持）個人衛生。”
