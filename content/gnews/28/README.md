## 路德社重磅爆料：FBI局長雷即將被解雇
`2020-10-20 11:06 GM06` [轉載自GNews](https://gnews.org/zh-hant/436134/)

文章撰寫：【重生】

《路德時評》2020年10月20日早間節目再爆出重磅：美國FBI局長克裏斯托弗·雷即將被政府解雇。

隨著三個硬盤事件不斷發酵，有越來越多信息被陸續爆出，硬盤中存有大量涉及拜登家族腐敗資料，還有亨特拜登虐童等嚴重違法行為的照片視頻信息，這些信息讓全美國人震驚。班農先生在節目中曾提及：FBI在2019年就已獲得該硬盤。按照慣例，FBI早在2019年獲得該硬盤時就該對亨特拜登嚴重違法事件展開詳細調查，但最後被拜登勢力壓制。很顯然，FBI局長克裏斯托弗·雷肯定知曉此事，他也參與隱瞞該事件。

路德社爆料現任FBI局長克裏斯托弗·雷與郭文貴先生曾有過合作：當年雷所在的律師事務所承接郭文貴先生的案子，作為該律師事務所合夥人的雷負責郭先生案件。隨後，由於海航以每年500萬美金聘請該律師事務所為法律服務商，雷最後因利益沖突停止參與文貴先生案件。隨後不久，雷就被提名為川普政府FBI局長，很快經投票通過後奇偶走馬上任。

路德社在節目中重磅爆料：美國FBI局長克裏斯托弗·雷很可能今天就將被解雇。接替他上任局長的極大可能是FBI副局長大衛·鮑迪奇（David Bowdich），爆料革命戰友對該副局長名字較為陌生，但是他對爆料革命起到至關重要的作用。不久前，司法部正式發文提及郭文貴先生遣返案件，他就是核心負責調查此案件的官員，同時，他也全程負責調查閆麗夢博士事件。壹旦他上任，對爆料革命將是重大利好。

在路德社揭露美國財政部長被習收買後，又揭露壹大佬即將離職。川普政府排幹沼澤行動徹底進入攻堅階段，邪惡力量將被徹底清剿，正義力量正在集結！
