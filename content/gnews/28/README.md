上期小教练讲解了一个核心力量的训练动作。大家经过一段时间训练，核心力量应该已经得到了一定的训练，这期继续再来一个比较难的动作。

**V字型仰卧起坐**

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/10223818/62-1.jpg)图片来源：见水印

1、平躺在垫子上，两腿并拢自然伸直，两臂放在头后，也是自然伸直。

2、吸气，双腿保持伸直状，缓慢向上抬举，与此同时，上身躯干也缓慢同步抬起。

3、以胯部为轴，使身体形成对折，两臂伸直向上伸展，与腿部接触。

4、呼气，然后恢复起始原点。

这个动作10到20个为一组。动作注意要点：呼吸一定要配合，不要刻意的吸气或者憋气；四肢始终保持自然伸直，膝盖不要弯曲；动作不要快而猛，缓慢一点，用肌肉的力量来抬举和放下，不要利用惯性。

这个训练的动作，主要锻炼了腹直肌、腹内斜肌、腹外斜肌、腹横肌、下背肌、竖脊肌以及髂腰肌等等。但该动作要求练习者具有一定的身体协调性，所以协调性比较差的练习者，可以相对减少对折程度，或者手脚分开进行。等练习多了，熟悉了以后，就可以轻松完成啦。长期坚持这一练习，可以练出漂亮的腹肌、腰部曲线、腿部线条，还可以改善肩部、颈部以及背部的肌肉由于长期紧张而造成的疼痛。
