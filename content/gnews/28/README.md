230法案功能：
1.保护互联网公司不被第三方发布的假新闻或不良内容遭到指控，也即对第三方发布的内容有豁免权。
2.互联网公司如果认为用户发布内容涉嫌违法，可以取消用户的帐号或删除用户发布的内容。

美国1996年颁布的这项法案，旨在促进互联网的发展。如今发展到谷歌、脸书、推特、油管几家巨头垄断了全世界的互联网，使它们可以肆无忌惮地撒谎和造假新闻，对不符合他们意识形态和他们当前宣传口径的言论进行审查和删除，包括删除帐号。
