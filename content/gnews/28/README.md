一、最新威視合同

上週，加拿大《國家郵報》報導，有爭議的中共國有安全企業威視（Nuctech）贏得了680萬美元的競標，以在全球170個加拿大大使館、領事館和英聯邦高級專員公署安裝設備。但這僅僅是威視對加拿大安全基礎設施的最新的滲透。據加拿大《全球新聞》報導，自2017年以來，威視已經贏得了加拿大四份邊境安全和海關合同。

最新的威視合同不僅僅是涉及使館和領事館的X射線設備或軟件程序這麼簡單，這等於讓中共為加拿大所有外交駐地提供“保護”。威視的設備將掃描執行加拿大外國任務人員的所有物品，這有可能為中共情報機構提供原始數據。這份合同有可能使加拿大的駐外機構成為中共政權的監聽站。

威視以最低的價格擊敗了北美的公司贏得合同。在與中共國有企業參與的競標中，只以價格為競標依據是愚蠢的。威視應被取消資格，它可以拿到中共政府直接和間接的補貼，這是不公平的競爭。加拿大政府有沒有對威視展開過盡職調查？

二、其他威視合同

同樣，加拿大也應撤銷先前授予威視公司的邊境安全和海關合同，並應對這些合同是如何被批准的展開調查。我們還需要弄清楚這些合同是否允許中共獲得過境所有人員和貨物的身份信息和金融交易信息。這些信息代表了更主要的安全漏洞，可能會影響加拿大的貿易夥伴和盟友。

毫無疑問，威視已經像在許多歐洲國家一樣，希望獲得對加拿大機場安全系統的控制權。一位德國部長說，威視正在“使用大規模的低價定價策略”來削弱競爭對手，而歐盟委員會“必須認真分析（機場安全合同）並確保我們航空旅客的高度敏感的個人數據受到足夠保護。”

三、政府態度

上週，《全球新聞》向加拿大外交部長弗朗索瓦·菲利普·香檳（François-Philippe Champagne）詢問了與威視公司達成的交易。他承諾將審查“與安全性或安全有關的任何可能問題”，並表示“將採取一切適當行動以確保我們在世界各地執行任務的安全。”

但是，這很難令人安慰。弗朗索瓦一直與中共相處融洽，而且幾年前還從“中國銀行”借貸120萬美元在英國倫敦購買了兩套公寓。 《環球郵報》於今年6月報導了此事，之後他將貸款轉至一家加拿大銀行。但是部長仍未回答問題：為什麼首先要從“中國銀行”借款？他所獲得的貸款條件是否比加拿大和英國的銀行所提供的條件更優惠？

四、加拿大情報機構

加拿大情報與安全局（CSIS）應該調查所有威視、華為和其他觸角已觸及並參與加拿大基礎設施建設的中共實體或個人。 6月，CSIS發出警告，稱加拿大被中共情報機構視為“有吸引力和寬容的目標”。當時，加拿大前駐華大使戴維·穆羅尼（David Mulroney）坦率地警告：“加拿大是一個困倦而又不為人知的目標。我們現在沒有像澳大利亞和新西蘭那樣的警覺性。最好有所改變。”

五、短評

加拿大是一個被中共多年來全方位滲透的西方國家，這些滲透包括對政府、黨派、電信科技、生物科技、石油能源公司、礦業、大型工程企業、等等等等。近年來僅個別針對加拿大大型企業的併購案被以涉及國家安全為由阻止，總體上還是難以抵擋中共的全方位超限戰。

CCP病毒爆發以來，一架接一架海航、東航航班毫無限制的抵達加拿大，邊境也幾乎沒有任何檢測、防疫、隔離措施。在加拿大疫情全面爆發前，16噸個人防護物資被直接捐助給中共紅十字會，這些救命物資是否全部送到了國內一線醫護人員手裡不得而知。
