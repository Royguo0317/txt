健身是一辈子的事，有一个健康的身体才会有一颗强大的心脏。尤其在面对CCP这个魔鬼时，我们更应该有一个强大而智慧的体魄来勇于面对邪恶。因此，本篇将为大家介绍“高等篇——钻石俯卧撑”。

在上两期中笔者介绍了跪姿俯卧撑和标准俯卧撑，不知道大家有没有在这两周多的时间里坚持训练呢？别忘了，健身可是一辈子的事！相信经过这两周多的时间，你已经是一名中级水平的健身初学者了。

今天，笔者再介绍一个新的俯卧撑动作——“钻石俯卧撑”。这种钻石俯卧撑对我们的臂力（肱三头肌）还有核心的要求是比较高的，假如战友们在这两周的时间里认真完成了以上两期的训练，相信大家的臂力还有核心已经可以支撑钻石俯卧撑的一组（包含1-5个动作）了。

**钻石俯卧撑动作讲解**

首先跪在地板上，然后双手食指和大拇指相连成钻石状放在胸部中间下方，最后双腿并拢向后蹬，双臂伸直，保持脚后跟和上半身处在同一水平面上，不要弓背，不要撅屁股。下图为该动作的起始图：

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/25235047/48-6.jpg)

然后，弯曲肘关节直到胸部于地面有一个网球大小的距离。注意，向下时用肱三头肌和胸大肌中间部分的肌肉在控制身体重量慢慢向下，不要撅屁股。如下图：

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/25234928/47-9.jpg)

随后，停顿一到两秒，再将自己的身体推到初始位置。向上时一样用肱三头肌和胸大肌中间部分的肌肉控制身体重量慢慢向上抬起，抬起肘关节时不要锁死关节。这里的的判断标准是：

初级标准 1组5个

中级标准 2组10个

高级标准 4组15个

如果你不能完成标准钻石俯卧撑的话，可以减少动作幅度，不要让胸部降到距地面一个网球的距离，而是把动作幅度调整到你能舒服完成的距离，同时，也可以增加次数。这里，大家要不断地训练肱三头肌的肌肉部位，并逐步的增加该动作的幅度，直到可以完成初级水平的五个标准钻石俯卧撑。
