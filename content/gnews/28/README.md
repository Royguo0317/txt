费罗斯（Fellows）网站上的清单写道：“该手表是1989年北京天安门事件后，中共北京市委和北京市政府送给部队的。”

一块授予中共国人民解放军（PLA）纪念1989年天安门大屠杀的手表已经在英国奢侈品拍卖行费罗斯出售。

表面刻有中文 “89.6. 纪念平息爆乱纪念”，并根据拍卖清单显示，该表的图案是一名解放军士兵的形象。
![]()![](https://gnews.org/wp-content/uploads/2021/04/watch8964.jpg)照片由费罗斯拍卖行提供
“费罗斯网站上的清单说：“这块手表是1989年北京天安门抗议活动后由中共北京市委和北京市政府送给部队的。”

清单继续说：“北京天安门民主抗议活动是1989年6月由学生领导的。军方下令镇压抗议活动，中共国人民解放军被派往占领北京市中心城区的部分地区，该事件造成许多人丧生，尽管具体数字经常有争议。”

根据历史学家估计，1989年6月4日，中共国人民解放军出动镇压北京的民主抗议者时，有数百人，也许数千人死亡。北京官方的立场是，这次行动中只有23人死亡。

这块手表估计价值2500英镑至3500英镑，是由“一位不愿透露姓名的主人”送到拍卖行的。据信，这块奖牌手表是送给解放军选定成员的几百只中的一只。清单上写道：“这很可能是英国仅有的一款此类手表进行拍卖。”

*镇压后。 照片：**HRIC档案，由盖尔·巴特勒（Gail Butler）提供，利比·施马尔茨（Libby Schmalz）*

澳大利亚华人艺术家巴迪乌曹（Badiucao）周四在推特上指责拍卖行 “拍卖天安门大屠杀的证据”。

一位1989年流亡美国的学生运动的组织者刘秀告诉《香港自由报》：“这是我第一次看到这种纪念品手表，作为1989年民主运动的学生领袖，我感到愤慨。中共国共产党鼓励和奖励血腥屠杀无辜学生和公民的士兵，对六四抗议者来说是巨大的耻辱。”

他补充说：“这类拍卖应该被禁止，就像希特勒在纳粹德国发布的奖牌禁令一样。”

在欧洲一些地区，出售纳粹纪念品是非法的，尽管没有任何规则禁止拍卖与天安门大屠杀有关的物品。

费罗斯是一家家族经营的英国豪华拍卖行，成立于1876年。

香港自由报社已联系费罗斯置评。
