## 被美國制裁 林鄭月娥自爆：信用卡不能用了
`2020-08-17 23:59 GM10` [轉載自GNews](https://gnews.org/zh-hant/301061/)

**文/GM40**

彭博資訊報導，香港特首林鄭月娥接受大陸官媒訪問時透露，她被美國制裁後，信用卡不能使用了。

林鄭月娥自嘲：美國的制裁造成自己個人一丁點不便，因為我們必須用到一些金融服務，我們不知道這些服務是否會與一家與美國做生意的機構有關，而且使用信用卡會有阻礙。不過，對我而言，這些事沒多大意義。」

美國川普政府日前宣布制裁包括林鄭月娥在內的11名大陸、香港官員，懲罰他們在箝制香港政治自由方面扮演的角色。
