澳大利亚维多利亚州自由党议员大卫-戴维斯 （David Davis）说，安德鲁斯政府在”没有适当的检查和权衡下”，与一家与维吾尔族劳工有联系的中共国企业签署了合同。”这就是一个丑闻”。

安德鲁斯政府为一个价值24亿美元的地铁项目从与维吾尔族工人有关的中共国供应商那里购买了火车零件。它选择继续与该公司合作是因为寻找新的供应商会造成延误，而且成本太高。

他说：”去年贾辛塔**·**艾伦（Jacinta Allan，维多利亚州交通基础设施部长和郊区铁路环线部长）被追问此事时，……她否认，并说 “一切都很好，没什么可查的，我已经听取了部门的汇报，一切都很好。”

“现在我们去调查并要求让我们看看这些文件……（发现）该部门所做的仅仅是，他们写信给火车供应商，询问他们是否使用奴隶制劳工，他们毫不意外地回答说没有。

“这就像要求你的孩子批改他们自己的作业。很明显，政府已经接受了这些中共公司的保证。这真是天真到了极点”。
