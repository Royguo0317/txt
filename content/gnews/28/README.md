在众多重磅新闻中有一条不起眼的新闻：。只要占用耕地就要收税。个人观点如下：

1. 缺钱了，要广开税种。甚至要榨干最底层农民最后的一点血。

2. 缺粮了，粮食问题已经非常严重，而且没有多余的耕地可用。18亿亩的耕地红线根本也是假的，实际远远没有达标。农地被占用作建厂建房的，挪作他用的比比皆是。所以要对大量被占用的农用耕地收税，让这些被占用作建房，建厂的都退回去种地。有点像之前的退耕还林。要么种地，要么收钱。

3. 其实很多土地早已经没有肥力，就算现在马上被逼种粮食，但由于整体污染严重，根本没有预期产出。所以最后本质上还是多收钱。

4. 农民都进城打工，耕地荒废。现在税收政策出来以后，逼着农民回去继续种地，减轻城市里的就业压力。反正占了农地就要交税。

5. 如果农民觉得不想种，不值得，那就转让给村里。土地虽然不能买卖，但是可以转让啊，转着转着就转到所谓的集体手里去了，最后都控制在国家手里了。农民彻底无产阶级了。
