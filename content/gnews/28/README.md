## 日本东京、关西 新冠疫情持续扩大中

根据11月11日NHK新闻，继日本东京首都圈二度发布紧急事态宣言后，应大阪、兵库和京都三个关西县的要求，根据《特别措施法》发布紧急事态宣言，作为应对新冠病毒的对策。由于疫情持续扩大中，菅义伟首相、首相官邸田村宪久厚生労働大臣、梶山弘志経済产业大臣、加藤胜信官房长官、西村康稔经济振兴大臣进行讨论，决议考虑调整医疗服务系统，来因应新冠病毒疫情发展。
![]()![](https://gnews.org/wp-content/uploads/2021/01/japanvirus-400x225.jpg)
官方证实日本境内出现不同于英国SARS-CoV-2 VOC变种病毒与南非的SARS-CoV-2变种病毒(501Y.V2)的全新变种病毒，这个变种病毒是由4位从巴西入境的日籍旅客所携带。另一方面爱知县和岐阜县两位县长已要求政府在12日发表新冠病毒紧急声明。栃木県宇都宫的済生会宇都宫医院急诊和重症监护中心主任小仓隆史表示：新冠病毒正向老年人快速蔓延中，许多人因此引起重病住院，请菅义伟首相认真应对并采取适当措施。本文仅代表个人立场
