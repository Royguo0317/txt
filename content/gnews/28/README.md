拉吉政府刚刚发布的开放法令，概述了从4月26日星期一开始意大利重新开始的路线图：通过逐步重新开放再达到恢复正常状态，尽管重新开放，但是行政长官还是不想取消一项最无形的措施，即从晚上22：00到凌晨5：00宵禁，对于政府来说，这仍然不能被动摇。

最早反对宵禁的是Lega领导人马特奥·萨尔维尼（Matteo Salvini），中右翼的另一位代表意大利兄弟会的领导人乔治·梅洛尼（Giorgia Meloni）持有与萨尔维尼（Salvini）相同的观点：对于政府而言，确定是否以及何时可以离开家，不是政府的特权，我们之所以能够容忍，是因为发生了紧急情况，发生了大流行。因为我们有责任感，因为我们是认真的人。但是超过一年后，如果继续将人关在室内，尤其是如果没有更充分的理由的话，新冠病毒与宵禁“无关”。像我们这样的国家，在一年半之后，对基本自由的限制已经不再是可以容忍的了，这已经足够了。卫生部副部长皮埃尔保罗⋅西莱里（Pierpaolo Sileri）在接受《La Stampa》报纸采访时解释称，因为数字仍然不足以打破限制，现在取消宵禁还为时过早。

#### **简评：**

**虽然在意大利绝大多数人都听从政府的安排，但是反对宵禁的声音一直没有停止。局部地区的示威游行也时有发生，自从中共病毒爆发以来，人们的生活方式发生了巨大的变化。行动受到了极大的限制，外出就餐和酒吧聚会都是意大利人很重要的社交方式，当容忍度到达一定程度的时候就需要发泄，政府也需要顶着巨大的压力面对民众的不满。意大利的主流媒体并没有关于病毒真相的报道，人们只能盲目的等待疫情的结束。在被各种限制的煎熬下，人们希望这个病毒能有消逝的一天，一切回到从前，但是更残酷的疫苗灾难会再次浇灭他们渺茫的希望**！
