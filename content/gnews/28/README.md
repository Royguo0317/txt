## 香草播报：【热点播报】塞内加尔花370万美元购买20万剂中国国药疫苗

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3.jpg)



摘要：来自达卡（路透社）的消息称，塞内加尔财政部发言人周五表示，塞内加尔已支付了20多亿非洲法郎（372万美元）购买了中国国药公司开发的20万剂新冠病毒疫苗，平均每针近19美元。

![]()![](https://gnews.org/wp-content/uploads/2021/02/image0-82.jpg)

非洲国家正在全球范围内努力采购疫苗，迄今为止，通过非洲联盟（非盟）和世界卫生组织（世卫组织）支持的项目获得的覆盖范围远远不能满足非洲大陆的需求。

财政部发言人巴勒·普雷拉确认了注射成本，该成本仅涵盖其1600万人口的一小部分，但没有提供进一步的细节。

作为一个中低收入国家，塞内加尔有资格在世卫组织COVAX计划的第一波支付中免费获得约130万剂阿斯利康疫苗，并可在全球范围内额外购买其他所需疫苗。

另外，非盟表示，它已为非洲大陆准备了大约6.7亿剂疫苗，根据该计划，塞内加尔有资格以略低于2300万美元的价格获得340万剂疫苗。

这些包括超过62万剂辉瑞公司的两剂疫苗，每剂6.75美元，超过120万支阿斯利康的两剂疫苗，每剂3美元，以及近150万份强生的单发疫苗，每剂10美元。

塞内加尔总统麦基·萨尔本周表示，他的目标是在2月底之前开始疫苗接种运动。这个西非国家记录了大约3万例新冠病毒病例和735例死亡。

塞内加尔卫生部一名高级官员周五在论坛上表示，到2021年底，塞内加尔的目标是为350万人中的约90%接种疫苗，其中包括卫生工作者和19至60岁的高风险个人。

尽管与西方疫苗生产商相比，中国起步相对较晚，但中国正在加紧努力在非洲和其他地方分销疫苗。

（1美元=537.5万非洲法郎）
