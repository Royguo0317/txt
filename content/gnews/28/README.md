**当这个世界知道共产党的病毒威胁每个人的生命和安全、和家人安全、财产安全的时候，没有哪个傻子还会非要跟共产党一起走进火葬场**。这个世界上最现实的，最最现实的就是金钱、实力、还有威胁着个人的安全和利益的时候，没有人、没有任何理由还会跟着共产党！

**共产党打开这个潘多拉盒子、还有武汉实验室——那个所谓进门的碑上写着“这是个潘多拉的盒子”那天起，共产党就注定了灭亡**。郭文贵先生说：“你们会看到一波又一波的、一浪又一浪的反共国际浪潮，以法灭共、以毒灭共、以真相灭共，跟着爆料革命、新中国联邦真正的拯救这个世界。”

“而且我相信，**包括美国政府和很多国家政府很快会宣布，不打疫苗或不能强迫打疫苗，这是个基本这逻辑**。”郭文贵先生说。

另外一个对中共国内影响较大的，就是前几天郭文贵先生说的“躺平币”。“躺平币”是郭文贵先生针对中共国内流行的“躺平”一词，提出的感念。

“**既然你不能站着，你也不能跪着，你也不要坐那儿被剥削，你就躺平吧。**”郭先生说，只要国内年轻人“躺在那儿”，就可以挣“躺平币”，并且计划要拿出几千亿美金，并且要保证与20%的黄金挂钩。

这个感念一经提出，便引起了中共国内大量人士的关注。郭先生在5月27日的第二次盖特视频中说，这个“躺平币”的概念对共产党的影响来源于什么？来源于当初共产党对中国人所有的杀地主分土地、杀土豪分土地造成的严重的后果。

“今天大家看上去没关系，事实上中国人开始在思考，**我们这么辛苦的劳动，到底我们的日子能不能过得更好？为什么我们不能像西方国家一样，拥有自己的土地？**为什么不能像西方一样，我们可以选择政府？为什么不能像西方一样，我们想信什么教，就信什么教？**为什么我们那么辛苦，却没有办法拥有西方一样体面的生活、和基本的房产和车辆？**”

中国人对幸福的追求真的很简单，在农村只要打打麻将、炒点肉吃、能吃饱馒头，每天撅着腚干10小时，都是幸福的！但就是对这点儿生活底线的追求，很多人都无法实现。**土地、水、粮食被污染，吃着有毒的食品还欠人家钱；孩子娶媳妇，全家破产；得一场病，死不了、活不了；死了埋不起，埋了，坟在那儿搁不起；在家呆着，都要承受人造病毒感染的风险**！

中国人太不容易了！郭先生说：“躺平币不仅对躺平的人要给币，未来还要让中国很多阶层的人有躺平币。我希望未来把“躺平币”发展成一个向共产党讨债的工具，共产党必须把这些年欺骗老百姓的钱还给中国人，包括当年所谓的杀富豪、杀地主、抢走的土地，必须还给人民。”

郭先生指出，“躺平币”概念就是他几年来说的：**必须把权力还给中国人民，一人一票选举的政府；中国人必须有中国的土地，中国人必须要平分中国所有的地下、地上的资源；中国人必须要有拥有自己的，所有各地、各县、各市的所有的资源，不能被外来者剥夺，包括台湾、香港**。郭先生说，“躺平币”会唤醒大家，让更多的阶层、更多的领域去躺平。

郭先生说，这个“躺平币”需要一段时间。“现在我们要把团队建起来，有大量的前期投资、大量的前期准备，而且要有绝对的最高端的人员，包括政治、经济，金融系统。”

就像一年前他提出的喜币概念一样，“现在喜币和喜Dollar发展的程度是超出你们任何人想象的，未来在市场公开交易的时候，你们都会明白了。”郭先生说。
