6月11日五角大楼发言人约翰.柯比（John Curby）表示，美国将向乌克兰提供额外1.5亿美元的军事援助，这是自3月份宣布的1.25美元军事援助的补充。此批军事物资包括雷达系统，无人机和军事通讯设备。

有意思的是，五天后美国的拜登总统和俄罗斯的普京总统将在日内瓦举行让世界很多人纠结的会面。很显然那些高度关注这件事的国家和个别组织，怕自己成为交易的棋子，但是通过柯比先生的发言，乌克兰的命运已经清楚的显露出来。

柯比讲到：幫助烏克蘭軍隊維護國家領土完整並提高與北約的兼容性。國防部繼續鼓勵烏克蘭根據北約原則和標准進行改革，以推進其在歐洲-大西洋的努力。

很明显乌克兰进入北约的大门已经打开，半只脚已经迈进了北约的大门。美国国防部科比先生的最后一段话才是重点:“提高乌克兰军队與北約的兼容性。國防部繼續鼓勵烏克蘭根據北約原則和標准進行改革”。军援的雷达系统，无人机和军事通讯设备只是婚礼上的香槟或者鲜花，婚礼仪式（美国北约同意接收乌克兰）才是重要的。乌克兰已经得到了最想要的——摆脱俄罗斯彻底加入西方阵营，先是北约而后是欧盟。

俄罗斯的普京会同意吗？答案是肯定的，路德先生也在节目中谈到：美国和俄罗斯私底下已经达成了一揽子的共识，相互都有可以放弃的妥协和必须坚持的底线，最终彼此比较满意之后才有了日内瓦的会面。

我以前的文章已经谈过俄罗斯普京最想要的依次是北溪2号，克里米亚，维持白俄罗斯卢卡申科的政权，乌克兰东部的顿巴斯地区两个州的部分领土。个人依然认为普京已经达到了大部分的目地或者是全部，今天不再赘述。

为何说乌克兰已经赢了，已经梦想成真？打开乌克兰的地图就可以看到，失去克里米亚乌克兰依然拥有很长的黑海海岸线，同时最大的沿海重工业城市敖德萨完好无缺。而恰恰由于历史原因克里米亚岛上居民大部分是俄罗斯族裔，是百分百亲俄罗斯的。这就是为什么在2014年公投时克里米亚选择了俄罗斯而不是乌克兰，假如继续留在乌克兰无疑也是身在乌克兰心在俄罗斯，克里米亚出事或者倒戈是早晚和必然。

在2014年乌克兰摆脱俄罗斯的事件中，历时七年多的时间，经历了同俄罗斯的战争和内部的政治力量的不稳定。终于迎来今天普京和拜登的日内瓦会面事实。乌克兰的精英们终于可以轻松的坐下来喝一杯咖啡休憩，更或者可以开香槟庆祝。

乌克兰的亲西方的势力大部分在乌克兰的首都和乌克兰西部，这非常好理解。一是历史原因，乌克兰的西部保留着波兰人，立陶宛人和德国人的生活习惯以及不同于俄罗斯的文化，宗教信仰。这才是我们看不见的种子，只要种子不死就有破土而出的可能！

二是长时间近距离和欧盟国家接触，贸易往来，文化交流，肯定会发现好与坏，真与假，善与恶。试问谁不想过舒心快乐的生活？这就是看得见的种子，是种子就有机会发芽。

乌克兰想要什么？简单地说就是：自由，民主和安全，稳定的生活。不要再发生类似上世纪三十年代的饥荒夺取乌克兰几百万人的生命，要知道乌克兰有“世界粮仓”之称，是世界第三产粮国。只有共产党集权残暴统治下才能发生产粮国会饿死人的创世纪的悲剧。

就这一件历史的痛苦记忆就足以促使乌克兰尽快的进入北约。今天美国五角大楼的声明和真实的军事物资的援助及美国发出援助的愿望无一不在表明一切都成为事实，即乌克兰马上要进入北约，北约基本上是美国说了算。这不需要像进入欧盟那么的艰难，因为欧盟国家里有几个亲俄罗斯的国家，同时也有亲中国的国家，这几个国家在重要时候是俄罗斯或者中共的马前卒和打手帮凶。

另外CCP当然不希望乌克兰加入北约，因为他再也骗不到，买不到乌克兰先进的武器技术，比如连俄罗斯都觊觎的制造航母的技术人员和设备（前苏联的航母制造基地在乌克兰），大型重型运输机的航空发动机技术。但即将到来的是——CCP不需要，也不会购买任何东西，他马上要死翘翘了。而俄罗斯的普京比中国人聪明和理性的多，如果再坚持和想控制乌克兰，那结果可能是普京会失去俄罗斯。

从乌克兰的历史中可以看出，乌克兰向往西方的自由，民主几乎用了一百年或者更长的时间，他们的坚持，流血，拼搏和战斗之 后迎来的这美好的未来无不告诉和警示着我，战友们和新中国联邦人：郭先生及爆料革命这四年多的时间（郭先生已经准备近三十年），我自己都感觉好漫长好漫长。可是和乌克兰以及波兰，波罗的海三国及其他的前社会主义国家相比来说，我们的付出和执行力还差的很多很多，这也是到今天我们中国这片土地还被中国共产党蹂躏着，糟蹋着，霸占着。依旧献上郭先生借用美国政治大佬的一句话：民主之花是用鲜血不停的浇灌的！
