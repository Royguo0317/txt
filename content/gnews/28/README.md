据2021年3月28日CNN报道，美国国务卿安东尼·布林肯（Antony Blinken）在布鲁塞尔访问期间，接受采访时表示，不建议对中共处理疫情爆发的行为而采取惩罚性措施，更应该重视的是，对中共病毒的应对计划和防止类似疫情未来再次大流行。

布林肯此番言论回避中共病毒追责意图明显，基本违背了其前任迈克·蓬佩奥（Mike Pompeo）和川普政府的主张。川普政府此前一直呼吁对中共病毒追责。国务卿布林肯采访中表示，需要“对过去负责”，要求中共对2019年病毒的来源问题保持透明，但另一方面他指出，重点应放在建立一个更强大的体系，并表示世界卫生组织WHO是体系的一部分。

他认为，应对未来潜在大爆发的准备工作，很大一部分是“确保系统就位“，包括具有透明度，信息共享和国际专家协作的国际卫生组织机构。布林肯对WHO的评价也违背了川普政府的执政方向，川普政府曾宣布退出世界卫生组织。另一方面，布林肯也表示对即将发布的世界卫生组织关于病毒起源报告背后的“方法论和调查过程”表示担忧，其中包括中共政府显然在协助WHO撰写报告这一事实。前往中共病毒起源地武汉的世界卫生组织实地调查小组表示，他们并没有获取病毒相关原始数据的权限。

布林肯在采访中还指出中美关系的复杂性，同时具有对抗性，竞争性和合作性，并表示会与同盟国合作应对中共国所面临挑战的重要性。
