## 颱風調查：公開信息分析顯示亨特·拜登是美國國家安全隱患
`2020-10-19 10:37 gnewscanada` [轉載自GNews](https://gnews.org/zh-hant/434171/)

喜馬拉雅聯盟加拿大農場文永校對坐看雲起時上傳WJ

最近浮出水面的關於亨特·拜登的貪腐醜聞，讓美國民眾大為震驚，而實際上這和中共與拜登的勾兌相比，卻只能算是小菜。 9月29日，Revolver News公佈了一份關於亨特·拜登和中共國商務往來的詳細調查報告，而其中列出的潛在國家安全隱患，引人關注。
該報告由“Typhoon Investigations（颱風調查）”組織編寫。該組織是一家宗旨為保護本國企業，預防國外公司威脅的調查公司。該報告很多信息來源於公開信息，很容易驗證，整個報告的可信度極高。

該報告的一個重要結論，就是指出亨特·拜登能夠成為中共國國有企業渤海華美（BHR）的控股人之一，就是因為他父親喬·拜登的政治影響力。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/19102436/Ekqj6pLWAAQ2CCA-1.jpg)圖片來源：Twitter
報告中說：“亨特·拜登能夠成為渤海華美股東，沒有其他的合理解釋，只可能是為了政治收買。幾乎所有的（渤海華美）投資人都是國家政策部門，國有企業，或者是僅小部分私有的國企。資金的投資範圍也主要局限於政府支持的項目或公司。從我們獲得的具體合同條款來看，條款也與正常公司經營不同，只是為了把資產和經費轉移，只讓某一方實體受益。這其中能看到明顯的中共國政府官員影響美國個人，或者TA的親屬的意圖。該合同讓中共國政府官員可以與亨特拜登有見面機會，並安排用公用資金，比如社保基金，地方國企資金，以及國有銀行來給該公司輸血。儘管該投資合作從2013年才正式成立，主要的投資人關係以及和公司關鍵人見面等事宜，在很早以前就已經完成了。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/19103404/20190927104352964.jpg)圖片來源：阿波羅
中共國在影響國外個人和機構方面投入了大量資金和政治資本。考慮到該公司涉及到大量中共政府高層官員，這樣的合作在正常的市場體制內，不受政府影響能夠達成的可能性為零。尤其值得注意的是，該項目涉及的關鍵人和中共國影響的機構有密不可分的聯繫。渤海華美主要投資方渤海控股是中國銀行的子公司。渤海華美的其他合夥人都是國有企業，他們把（其他地方獲得的）資產和收益轉到渤海華美。 ”

該報告還引出另外一個台灣人：林俊良（Michael Lin），指出他在亨特拜登與渤海華美之間起了重要的橋樑作用，這也說明林俊良很可能是由中共控制的。另外也有情報指出，美國和另外某國正在調查林俊良和中共國的關係。 “林俊良是（拜登在）渤海華美投資，以及中共國外交部影響的組織之間合作的中間人。根據B和C（分別來源於兩個國家的情報機構）的情報來源證實，林俊良因為給中共國做事而上了重點關註名單。”
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/19102350/%E6%9E%97%E4%BF%8A%E8%89%AF.jpg)林俊良圖片來源：百度百科
報告還指出外交部長楊潔篪是亨特拜登相關的項目的批准人。在渤海華美項目中，亨特·拜登擁有10%的股份，價值5000萬美金。如果拜登當選，楊潔篪有可能會以亨特·拜登在中共國的相關利益作為談判的籌碼來影響美國政治。這無疑對美國國家安全是一個挑戰。 “喬·拜登與中共政府的這個合作由外交部長楊潔篪經手，楊潔篪與拜登最初交往開始於楊擔任駐華盛頓大使期間。亨特·拜登2013年入股的渤海華美是由中共國外交部建立，目的就是為了在楊潔篪在職期間收買外國領導人的政治影響力。”該報告從側面驗證了川普總統提到的，亨特·拜登的商業關係存在國家安全問題。

評：颱風調查僅憑公開信息即可得出亨特·拜登的商業活動情況不正常，楊潔篪與拜登的合作關係歷史由來以久，對於國家安全是隱患的結論。但長久以來，竟完全沒人對亨特·拜登和拜登整個犯罪家族的不法行為提出任何質疑，拜登本人還一直被媒體包裝為一名完人，因他大兒子的去世還被渲染成為美國精神的悲情英雄，多麼的諷刺和可悲。朱利安尼先生和班農先生手持實錘鐵證，仍需要從司法和媒體封鎖中殺出一條血路。由此可見，美國的司法媒體和政府部門被中共和中共代理人滲透把持之深，也更讓世人看清了中共不滅則世無寧日這一事實。
