## 【鹰眼墙国】中共新建公司专司宽带卫星群

作者：纽约香草山健身部 文真

![]()![](https://gnews.org/wp-content/uploads/2021/03/GNEW-GTV-MOS-LOGO-2-3.jpg)



中共国有资产监督管理委员会4月29日发布新闻称成立中国卫星网络集团有限公司，负责建立和运营由13000枚卫星组成的宽带卫星群。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/28045647/%E5%9B%BE%E7%89%878-42.png)

中共2021～2026第14个五年计划和“2035长期目标”要求建立一个通信、地球观测和导航卫星的综合网络。中共国国家发改委在2020年4月将“卫星互联网”列入“新基础设施”清单。9月，一家名为“国网”的中国公司向国际电信联盟提交频谱分配文件，计划申请两个命名为GW-A59和GW-2的低轨道卫星群，由位于海拔500～1145千米的12992颗卫星组成。目前，建造该星群的时间表、目标最终用户以及如何访问和销售该系统尚未可知。

“国网”工程大肆宣称与Starlink等卫星网展开太空竞赛，给粉红们注入了一剂兴奋剂，继续洗脑民众编造厉害国的假象。Starlink星链卫星网的实现运行将对中共国的网络防火墙带来致命打击。为能继续阻止中国人自由获取信息，中共利用“国网”工程，企图将现有防火墙延伸至太空，组成一张控制言论自由的“天网”。

此前，中共宣布低地轨道星座计划时指出，该计划为“一带一路”国家和农村地区提供通信。新成立的中共国卫星网络集团将独立运作，目前没有公布参与该卫星网的承包商，这表明其他国家企业和商业部门有可能参与到该卫星网的建设中。“国网”工程就像其她“一带一路”的工程一样，打着帮这些国家建设基础设施的谎言，利用贷款项目渗透和控制他国的经济。

中共国有两大太空承包商——中国航天科技集团和中国航天科工集团，分别拥有“鸿雁系统”和“虹云系统”两大卫星网系统。新成立的中国卫星网络集团独立运作，成为与前两者平行的国有集团，由此不难看出中共内部派系之间的较量。卫星导航是军事部署和通信的关键。习近平牢牢掌握军队的控制权，势必要控制太空军事部署和通信。通过组建新的公司建立卫星网，他可以部署亲信控制太空的“话语权”，从而制衡甚至取替前两个卫星网计划。

建造超大型卫星网的关键在于火箭运载能力和可回收技术。SpaceX正是在这两方面取得技术突破的，从而降低成本，并在短时间内实现了Starlink的可靠运营。中共不久前因火箭无法准确回收沦为各国谴责的对象。除了制造太空垃圾，中共的太空科技实力根本撑不起它称霸太空的野心。
