## 美國參議院立法《外國公司問責法》”絞殺”中概股？中共離滅亡更近了一步！
`2020-05-24 00:32 GM30` [轉載自GNews](https://gnews.org/zh-hant/210922/)

**作者：滅共日記**

美國國會參院20日通過，將強化對中共國及其他外國企業的監管，讓劣等的中共國和外國公司從華爾街股市下市。接下來還須在衆議院通過，並由特朗普總統簽署才能成爲法律。法案規定，任何一家外國公司連續三年未能遵守美國上市公司會計監督委員會的審計要求，將禁止該公司的證券在美國證券交易所上市。該法案還將要求上市公司披露它們是否爲外國政府所有或控制。該法案將迫使中共國國企和不遵守美國法律的中共國公司下市。也就是說許多中共國公司將禁止在美國證券交易所上市，這也切斷中共國企業向美國投資者融資的道路。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/24002315/1-131-22.jpg)

《外國公司問責法》的主旨是要求在美上市公司財務審計透明，並且不受外國政府擁有或控制。通常在美國公開募集資金的公司必須經過美國證券交易委員會認證的會計公司的審計，這些審計工作還將受到上市公司會計監督委員會的監督。但是在美上市的中共國公司經常以‘國家安全’爲由，阻止美國監管機構檢查文件和記錄。一旦這項法案落地，在美國上市的中概股將徹底完蛋。但是如果中概股遵守這項法案，這就和中共國自身的法案産生了衝突。中共國法律要求，按照《保密規定》，在國外上市的中共國公司檔案管理和跨境監管等事情應該由中共國監管機構檢查，而且《新證券法》也明確說了，國外的證券監管機構不能在中共國直接調查，任何單位也不能擅自向國外機構提供相關信息。因而造成中共國企業雖然在美國上市，卻不需要完全遵守美國證券法。也導致“中概股”的信用記錄一直欠佳。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/24002338/2-98-15.jpg)

法案裏主要有兩個痛點， 一個是美國機構可以介入審計， 一個是有政府持股需要披露。在中共國很多老百姓因爲經常看到媒體上關于互聯網巨頭發家創業史的洗腦新聞，所以很多人還相信國內互聯網巨頭都是靠能力起家的，老百姓真是太天真了，就比如淘寶，爲什麽國外最先進駐中國的電商平台沒有成功，而淘寶就成功了？你以爲真的是因爲馬雲厲害嗎？完全不是，因爲他能減稅，能壟斷，能賣假貨，三者缺一也沒有今天的阿裏巴巴， 而這三個都是中共政府給的。百度騰訊京東其實也都一樣，國內大公司實際控制人基本都是中南坑那些家族的人，只不過人家有能力不被披露而已。而美國在這個時候出台這個法案，我認爲這個法案主要針對就是中共國的中概股，一旦通過後影響最大的一定是中共國的國企，央企，因爲很多中共國國企，央企根本沒有辦法證明自己不受中共政府控制，這也意味著美國開始在資本市場進行中美脫鈎的正式行動 ，第一步就是讓中共國的企業離開美國，因爲美國上市的中概股是中國獲得外彙的一個重要渠道，美國現在掐死中概股，讓中國外彙短缺更進一步，意味著中共國整個的創投行業會萎縮下來，無以爲繼，這也相當于給中共國的高科技産業斷糧，把中共國融資的道路徹底堵上。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/24002410/3-94-18.jpg)

美國拿中概股開刀，切入點是會計制度。既符合美國的脫鈎戰略，又符合現有的美國監管規則，又符合美國投資者的利益。這樣不可調和的矛盾導致的結果就是，今後想到美國上市的中共國企業會越來越少、越來越難，在美國上市的中概股將面臨越來越多的麻煩，中共國獲得美國資本的所有渠道將被徹底切斷，更加意味著中共向滅亡又邁進了一大步。
