“计划生育”是一项使中华民族断子绝孙的屠杀政策。

中共国计划生育政策带来的“杀人运动”，不仅在中华五千年历史上找不到先例的，放眼全世界也是没有先例的。这是中国人生命的一场人为大浩劫。

![]()![](https://gnews.org/wp-content/uploads/2021/02/image-114.png)图源网络

古今中外，从来没有哪个组织、政府或国家，会系统的使用暴力、丧心病狂的屠杀胎儿，并冠以国策的借口来破坏社会的人口自然结构。

然而，这些罪恶在中共国却已存在了三十六年之久。数亿的胎儿死于非命，一个个幼小的亡灵凄惨无边，这是一个何等惊人的杀戮！害人一命，都得偿还，8.28亿多胎儿婴儿的血债，那是何等规模的罪业。

用杀人来解决社会问题，这是中共一贯的传统。计划生育，实质上就是有计划、有组织、长期、持续地对婴孩和胎儿的大屠杀。这是对中华民族的一种斩草除根式的杀戮，是彻底的反人类行为。

可怜的中国人在自己的土地上，只是为了再生一个孩子，就像罪犯般东躲西藏，逃避中共的抓捕，他们连亡国奴的待遇都不如。无数的母亲眼睁睁地看着自己的孩子被那些计划生育恶人们杀死，欲哭无泪。

为了赢得这场战争，中共自上而下成立了一个庞大的计划生育系统：从人员编制到培训，从医院配备到警察配合，再加上打手招募、文宣洗脑、媒体欺骗、各种计划生育器具的生产等等，是一个完备的杀人集团。中共用这些手段来对付一群手无寸铁的平民百姓。甚至，他们还把计划生育作为考核地方官员政绩的一个指标。其用心险恶，计划细致，做好充分的准备屠杀胎儿、婴儿，结果何其惨烈。

计划生育政策最一开始，先是欺骗。当欺骗无效时，暴力就开始登场了：恐吓、绑架、株连、拘禁、罚款、酷刑等等，恐怖手段无所不用。

计划生育干部经常带着一群打手在晚上出动，抓捕一些适龄夫妻，做计划生育手术。对于那些抵抗或逃匿者，他们的手段是：扒房牵牛，抢粮抢财，绑架其亲友（甚至邻居）后非法拘禁、用刑，最终达到抓捕当事人的目的。他们甚至还把一些未成家的女孩拉去当替罪羊，做绝育手术。那些计划生育恶人们如狼似虎，根本不容你辩说。有些手术医生，甚至不把孕妇当人看，致使有些孕妇在草率手术后死亡。
