無癥狀，顧名思義，沒有癥狀的病毒攜帶者，如果不是被排查出來，他們的身體狀況與正常人相同。而重型肺炎患者，往往出現呼吸困難、多器官衰竭、呼吸衰竭等臨床癥狀，其中多器官衰竭以肺部衰竭最為顯著，這將使患者的身體受到不同程度的損害。

病毒感染及癥狀發展需要過程，為什麽最開始只是無癥狀感染，而後卻出現了重癥患者。在6名確診患者中，又是50%（3名）重癥率。這樣的重癥率是非常不正常的，很有可能來自於感染患者數量的瞞報或漏檢，從而增加了重癥患者比例。

閆麗夢博士第二篇病毒論文發表後，她的家鄉青島壹夜間成了疫情重災區，各地紛紛排查與青島人員的往來，儼然壹派最初排查武漢的架勢。中共想做什麽，全國哪個城市沒有病毒在感染和傳播，是全國的疫情無法掩蓋拿青島開刀？還是因為郝海東先生、閆麗夢博士的家鄉都是青島。或者兩者皆有，或者有更多因素。中共會不會在青島用此借口，對郝海東先生和閆麗夢博士親朋下毒手？中共的邪惡無底線已經70多年，他們做出什麽都不足為奇。只是為青島同胞祈禱，祝福他們平安。期待全世界正義聯盟快速斬除邪惡中共政權，還世界和平安寧。
