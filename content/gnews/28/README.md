如果没有了互联网，想看《路德时评》和 GTV，怎么办？

当节目通过卫星直播后，我们可以直接用卫星天线（俗称锅）和卫星电视接收机（机顶盒）收看（此方式已有很多战友介绍过，路德节目中也提到过）。

除了用锅收看，还有几个途径也可以收看电视直播：DTMB数字地面波电视；模拟电视，用电视机天线接收；还有最常见的有线电视；曾经存在过一段时间的移动电视。本文主要介绍DTMB方式。

DTMB，这是一种用地面波传递电视节目的方式。也叫数字地面电视。简单讲就是将锅接收到的卫星电视信号通过地面的发射塔传输到用户。

这种方式的好处是，几乎无障碍就可收看境外电视。例如，广东就可直接收看来自香港的地面波电视。同理，只要你周边有地面波信号，你就可以看电视。不过，能收看哪些节目取决于发射塔。

最早的时候，电视机都是接收无线信号，也就是模拟电视，信号通过电视机自带的天线接收。现在也有，只是非常少了。今年底，中共国境内的模拟电视信号将全部关停。明年，中共国就没有模拟电视了。换句话，现有的设备还可以发射模拟电视信号。模拟电视的缺点就是图像清晰度差。

目前墙内，城市大多都以有线电视为主，就是通过有线电视专用机顶盒插卡收看。卡是用来收费的，由当地有线电视台分档次收取。偏远地区则主要是数字地面波电视，基本免费。

若开战，只要在第一时间占领当地电视台，就可以及时准确地接收并传递信息，同时也防备CCP相同目的的反扑和破坏。每个电视台、广播电台都有卫星天线（建在楼顶等高处，非常巨大、显眼），用这些现成的锅就可接收卫星信号。再占领发射天线，就可向覆盖范围传送电视信号，视具体情况，可以是模拟电视，也可以是地面波电视。如果占领当地有线电视台，就可通过原有线网络传输节目。

中共国有个规定，2014年以后发售的32寸屏以上电视要具备DTMB功能。就是说大于等于32寸的平板电视机不需要机顶盒，只要连接天线，就可收看地面波电视。

大约3年前，地面波电视就几乎覆盖了中共国全境，绝大部分地区现在都有地面波电视信号。有少数地区在2018年后就没信号了，原因不明。本人推测，在这些地区的设备和发射天线应依然可正常工作。

家用的地面波电视接收天线，多是常见的鱼骨天线（八木天线）。天线与接收机用同轴电缆连接。同轴电缆就是有线电视入户用的那种圆柱形电线。卫星接收机同样也是用这种电线。

突发状况发生后可采取：在中共国边境地区接收到境外地面波电视后，通过发射塔接力的方式将电视信号传入内陆。一个发射塔大约覆盖90公里的范围。一个接一个，首先将信号传入大城市。接收天线由空运或就地组织制造，并向市民免费发放，也可教授天线制作方法，自行制作。相比锅而言较容易在短时间内满足收视需求。

或者直接由内陆，从已占领的发射塔向周围发射信号。

再或者，用热气球作为发射塔，接力传输。

又或者美国已有更先进的技术，可以超远距离传输地面波，微波通信等。

以上种种的前提是有正常的电力。

若战时，电力系统被毁，或是电网故障。那么，应对的方法，最简单可行的还是空投传单。图画加文字，用无人机、热气球散发，肯定会有很好的效果。

最后一个，短波广播，是一个永远不过时的信息传递方式。

相信七哥最终可能使用一种出人意料的新方式来传递信息，美国也会向世人展示其在通讯领域的尖端科技。
