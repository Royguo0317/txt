**人性的弱点总会暴露**

这就是人，不管你多伟大，在平静的生活中，在身体状态良好的状态情况下，在没有受到挑战的情况下，人性是不会暴露出来的；当身体条件非常差或者不舒服，周围的环境又很差，例如雾霾天气，或遇上一系列倒霉的事，或恰好在男女更年期，身体非常弱的这种情况下，能控制自己不出现赌博心理，不做出愚蠢决定的唯一办法就是：信仰，这样才不会对爆料革命将信将疑。一个人对事情是真信还是不信，只有自己知道。

**与“忠诚于事”的人打交道，和“诚实待人”的人合作**

我在清风看守所里看到过无数人当被法官判决完死刑、死缓、10年、20年、15年、5年、8年，当中如果牵扯到了家人朋友的，就能反映出一个人人性的最后底线。当人性在阳光下的时候，就能看到有些人，会在恐惧前面失去自己，由于想自救就丢掉了人性。这种情况下，很多人希望所谓涉案人还要忠于自己，替你保守秘密，替你担责，但是基本上，我可以告诉大家，任何经历过刑事案子、经历过大狱和失去自由的人，要求你的亲爹亲妈、你的另外一半能经受住这种所谓的忠诚几乎不可能，这就是人性。

所以在经历了清风看守所以后，我更加冷静地与人相处，这就是为什么我希望用诚实的人，我喜欢、我特别愿意和忠于事的人打交道。**与“忠于事，诚实待人”的人打交道，和“诚实待人的人”可以合作**，**因为这样的人会守法，这样的人让你知道你有没有安全性。**“忠于事”就是你这个人对事情是忠诚的，忠诚是忠于某件事情，执着地、认真地而且“说和行”是一致的，因为**“忠于事之人可用，诚于实的人可交”**，所以我说忠诚和诚实两回事儿。

**爆料革命战友要“诚实待人、忠诚事物”，而不是“忠诚于人”**

在我们爆料革命当中，我能看到我们很多战友，“忠”，不是忠于事而是忠诚于人。忠诚于人就是用于人，叫“用以忠，用以诚”，实际上成了“为他人所用”。很多人都希望别人忠于自己、诚于自己、忠诚于自己，甚至要忘掉他自己。举例那位战友，说 “我被骗了，火币的谁谁谁答应我什么什么，跟我说什么什么”。我要告诉战友，你千万不要把你的判断而导致的失败怪罪于别人，你所有的判断是基于让别人忠于你、诚于你，忘掉他自己，这本身就是不合乎逻辑的。

我们在爆料革命当中永远不要要求战友们要“忠于自己、诚于自己而忘掉他自己”，这是共产党的流氓逻辑。我们首先要让所有人诚实，“诚实待人、忠于事物”，这样的人可交、可用、可来往，这要分清楚。

有的人他忠，忠得愚钝，此人更加危险。此人诚，很诚，诚得傻，这人也危险。诚实不愚钝，都是天下的稀有之人。能“诚于事的人”是你最需要的人，而不是忠诚于人，凡是忠诚于人的都是工具。叫人“忠诚于人”的都是天真、都是别有目的。

“真人真事”节目展露了怎么看待“诚实待人、忠诚事物”。爆料革命、新中国联邦需要你的忠诚、灭共需要你的忠诚，但郭文贵不需要你的忠诚。人与人之间需要诚实待人、互相对待。你忠诚于你的爆料革命和新中国联邦，没让你忠诚于所谓老班长、长岛哥、喜马拉雅和郭文贵。你忠诚你的信仰，这样的情况下我们诚实待人，具体事上以诚实待人就可以了，互相尊重。

忠诚灭共，就不会有任何这样那样的想法。诚实待人，不能只看到付出的，要看到你得到的。就如刚刚这个爆料上海和台湾的战友，如果你相信唯真不破、相信灭共，你相信比特币、火币（的真相），你就不会去赌博，在你的怀疑和犹豫间就将付出了代价。

我们忠诚于事物、忠诚于灭共的事业、忠诚于我们追求的新中国联邦、我们忠诚于爆料革命、我们忠诚于金融规则、我们忠诚于对战友的一切的承诺，但是不忠诚于任何一个人。
