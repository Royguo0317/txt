## 蓬佩奧倡議世界各國回歸傳統價值觀——呼籲梵蒂岡拒絕與中共續簽主教任命協議
`2020-09-25 19:53 GM101` [轉載自GNews](https://gnews.org/zh-hant/382888/)

美東時間9月23號，美國國務卿蓬佩奧先生在聯大視頻演講倡議世界各國回歸傳統價值觀，以勇氣和信念來維護人權。蓬佩奧指出國際人權項目處於危機之中，而《世界人權宣言》的核心是讓人們回歸傳統價值觀，重拾道德、哲學和宗教遺產和傳承來鞏固所有人固有的權利。

蓬佩奧認為中共與信仰為敵，給美國及全球帶來巨大的威脅。蓬佩奧呼籲梵蒂岡拒絕與中共續簽兩年前達成的主教任命協定，該協定將讓中共變本加厲的迫害教徒，並有損梵蒂岡的道德權威。

美國國務院于2019年成立了“不可剝奪絕密委員會”部門機構，該委員會的報告指出美國應在其外交政策中大力捍衛人權，使之與美國的建國理念更緊密的契合，並堅持宗教自由是美國最基本的價值。
