![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/17030848/1.jpg.png-1.jpg)图片来源：每日邮报（Daily Mail）
根据阿鲁姆·博哈里（Allum Bokhari）在布赖特巴特新闻网 (Breitbart News)的报道，美国参议员罗杰·威克（Roger Wicker）上周提出一项新的法案：“言论自由法案“（PRO-SPEECH Act）。

威克参议员曾明确指出大型社交媒体正在滥用其市场权利，这些公司带政治倾向地审查用户言论，刻意的压制某些观点。对此威克参议员出台新法案的目的是保护消费者在不干扰平台功能或对其他用户构成数据隐私风险的前题下访问或使用合法的内容，应用程序，服务或设备的权力。禁止互联网平台基于种族、性别、宗教、党派或民族原因对用户采取任何行动。禁止大型网络平台从事反竞争活动，例如通过封锁或带有歧视性质的行为反击竞争对手的网络平台。互联网平台应基于平台管理和商业服务条款向公众披露其所有的应用商店、云计算服务、操作系统、搜索引擎或社交媒体网络。

简单来说，该法案主要针对科技巨头例如脸书，推特，谷歌，苹果等公司，新法案要求这些平台遵守一系列有关访问服务的规则，而且禁止这些平台从事限制或控制消费者言论的不公平，反竞争行为，不仅如此该法案还要求禁止通过欺骗性的手段误导消费者的言行。

威克参议员表示新法案将维护互联网自由，促进竞争，并保护消费者免受这些公然的偏见之害。互联网应该是一个自由表达不同言论的开放平台。
