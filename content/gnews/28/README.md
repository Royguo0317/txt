**西班牙2021年1月8日电/西喜社—**

据美联社1月8日报道，中国北方的一个城市正在为任何报告未进行中共病毒检测的居民提供500元（77美元）的奖励。

南宫市政府的这一提议是在该市及周边的河北省数百万人正在接受检测，作为控制中国最近最严重的感染爆发的努力的一部分。

悬赏提供政治或社会上不守规矩者的信息—告密，在中国有着悠久的历史。

中国通过使用一些被认为是极端和高度侵犯性的措施，控制病毒在当地的传播，这些措施包括封锁整个城市，对人们的行动进行严密的电子监控，以及禁止进出中共部分地区。
