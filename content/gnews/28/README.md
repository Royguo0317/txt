## 路德社快訊——習近平出賣金正恩家族基因密碼交換遣返郭文貴
`2020-08-24 05:08 GM62` [轉載自GNews](https://gnews.org/zh-hant/314487/)

編輯記者：川渝小哥G    校對編輯：文宠

8月17號，美國司法部發起關於中共核心領導人習近平和王岐山，曾計劃將爆料革命贊助者郭文貴先生從美國遣返回中國的起訴檔。文件中明確了中共願意以北韓金家的全基因密碼作為交換條件。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/24021208/1-117.png)![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/24021348/2.jpeg)
8月24號，路德社指出中共通過生殖器治國，創造了大量私生子女，再將其安置於世界各地。同時採用家族基金的方式在海外合法儲藏了巨量財富，其私生子女可通過一張無署名紙條隨時提取。路德明確獨裁政權的核心是其藏匿於世界的財富地圖和基因地圖，二者緊密聯繫，找到了基因地圖就能挖掘藏匿的巨量財富。習近平不惜出賣北韓金正恩家族來交換遣返郭文貴，以保全習近平及中共官員在海外的基因地圖與財富地圖。
