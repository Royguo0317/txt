根据《法国真理报》等当地媒体报道，法国总统马克龙谈到对G7峰会的看法，他认为尽管所有G7国家对中共国在维吾尔人和香港遭受的压迫等问题持反对立场，但G7峰会“不是一个敌视中国的俱乐部”。

马克龙在康沃尔首脑会议后的新闻发布会上说，七国峰会是一个 “民主国家的峰会”，尽管有分歧，但希望 “在所有全球问题上与中共国合作”。

法国、意大利、英国、加拿大、美国、德国和日本的国家元首和政府首脑在卡比斯湾（英格兰西南部）举行的峰会的最后公报中宣布 : “我们打算推广我们的价值观，包括呼吁中共国，要尊重人权和新疆地区的基本权利、自由，以及香港地区的高度自治”，

点评 : 众所周知，欧洲是个被中共渗透的重灾区，尤其是法国总统马克龙，在G7峰会后回法国总统府发表这样的措辞能看出马克龙总统非常害怕中共，在安慰中共并向中共表明他的立场。

现在全世界都在加紧彻查中共向全世界输送共产主义的邪恶行径，用蓝金黄收买世界政要，用一带一路掠夺世界财富，用孔子学院偷窃世界先进技术。而马克龙总统居然还是继续向中共国伸出橄榄枝。

G7公报指责中共侵犯人权，并公开表示支持台湾参与世卫。

公报中写道：“我们重视台湾海峡和平及稳定的重要性，并鼓励各方和平解决两岸问题。”

但是法国总统马克龙周日肯定地表示，G7 是一个“民主国家的峰会”，尽管存在分歧，但仍希望“在所有全球问题上与中国合作”。

难道台湾参与世界卫生组织对马克龙是不能接受的吗? 难道公报写的“我们重视台湾海峡和平及稳定的重要性，并鼓励各方和平解决两岸问题”，这不能引起法国总统的重视吗 ?

台湾的民主，台湾的人权是世界的榜样，台湾的科技也是世界领先之一，这些都是事实，而且被全世界认可的。那么为什么法国总统却视而不见 ? 避而不谈 ?

据德国之声报道，虽然拜登政府上台后不断在经济和政治上对中共国采取强硬的立场，但部分G7的欧洲成员国则更为谨慎。
