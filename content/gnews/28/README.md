## 【生化危机】6: 逃离香港
` 纽约香草山农场` [轉載自GNews](https://gnews.org/zh-hans/618646/)

作者：香草山写作组 Tiffany 的早餐 校对/发稿：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/02104531/logo.jpg)



摘要：闫博士在危急中决意逃离香港，中共对她的监控和加害也步步相逼。路德和郭文贵先生安排战友一路保护，闫博士死里逃生前往美国。

本节内容参考郭文贵先生2020年5月2日、7月12日和路德2020年7月12日谈闫丽梦博士出逃经过。

2020年4月28日，香港大学公共卫生学院的研究人员、临床医学博士(MD)及哲学博士(PhD)、病毒学家闫丽梦博士前往香港机场。为了摆脱可能通过技术手段跟踪她的人，她没有携带日常使用的电话。她像一名普通的旅客一样登上国泰航空飞往洛杉矶的班机。她清楚，淹没在巨大的旅客数据中的她，只要有一个环节惊动了监控她的人，对方将动用一切手段，在机场将她绑架，甚至召回已经起飞的航班。一旦她落在他们手上，她极有可能变成香港海面众多的浮尸之一。

闫博士冒险和路德联系，公开武汉疫情内幕后，她的人生安全受到巨大威胁([生化危机 5: 闫博士的发现](https://gnews.org/zh-hans/618522/))。郭文贵先生透露，在闫博士打算逃离香港的前一段时间，就被暗中监视，面临过许多生命威胁，那些人是一定要做掉她的。在中共的黑手伸向闫博士的同时，爆料革命那边的情报战也启动了，其情报之细致甚至包括准备对闫博士动手的具体实施方案。路德警告闫博士，不可以吃别人准备的食物，不可以单独去约会，每五分钟和路德沟通一次。闫博士同时安抚她的丈夫，没有再关注路德的节目。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/05031108/1600825601548558.png.jpeg)“大郎，乖，快把这毒药喝了！”
离开香港前几天，闫博士发现他丈夫神秘地接到裴伟士(Malik Peiris)的电话，唯唯诺诺。之后他给闫博士做了早餐，然后一整天在电脑前，又无事可做，行迹紧张而可疑。事先得知情报的路德告诉闫博士不可以吃。闫博士为了不引起怀疑，只吃了一点，推说身体不适。

就像潘金莲对武大郎说：“大郎，该喝药了。”闫博士的先生中午对她说“把早餐吃了”，到下午又说“把早餐吃了”。就闫博士吃的一点点早餐，就让她昏睡，心脏极为不适，心率加快。实际上早餐里有一种药，可以引发心脏病猝死，但是代谢完后无法查出。

为了把戏做足，让闫博士之死自然而然，不仅如此，裴伟士还给史蒂夫·班农(Steve Bannon)发了一份邮件，说闫博士心脏有问题。闫博士吃完早餐后感觉心脏不适，之后自行去医院检查，已有明显的心脏病迹象。她丈夫下午还要带她去陌生的海边，疑为对自己的妻子下杀手。路德和闫博士在高度紧张的几天中安排好了逃离方案。
![]()![](https://i.ytimg.com/vi/n2E0raFVsrg/maxresdefault.jpg)
情报战是虚虚实实的过程。路德透露他们故意露出破绽让中共误判。闫博士同意逃离香港后，第一时间在香港买了新手机和号码。她用旧手机说将搭乘私人飞机离港，降落到美军的航空母舰。另一边，她用新手机安排好了买机票的事宜。同时，郭文贵先生公开掩护，说专家已经到了欧洲。

事过多日后，路德谈到私人飞机的问题。在香港，私人飞机的乘客是情报机构重点监控对象。民航客流巨大，反而目标小，安全。路德曾在访谈节目中放话：“我们可以用私人飞机把她接出来，送到美军的航空母舰上。”当监控闫博士的人半信半疑地严查私人飞机的可疑旅客，研究私人飞机在航空母舰上降落的技术可行性时，闫博士的民航航班已经安排好。

路德透露，为了营救闫博士，在安排订票时，为了防止查到闫博士的信息，故意在订票的时候写错了一个字母。而按照航空公司的惯例，一个字母是在容错范围之内，不影响登机。然而，正是这个写错的字母，让查询她的难度陡然增加——这是郭先生熟知的情报系统的惯例。订票时订头等舱，可以享用经济舱不具有的一些临近起飞前的改签权限。最初订票没有用闫博士的原名，只是在临近出票时把名字改回来，避免前面被查到。

闫博士在临走的一段时间天天去P3实验室。实验室内部不能带手机，联系不上是正常的。4月28日，她把手机留在香港大学，没有引起怀疑。当晚爆料革命的几十个战友暗中保护，冒着生命危险把她送上飞机。许多关键的岗位上，都有中港两边的战友在保护她。送闫博士去机场的两位战友，待到闫博士上飞机后就被抓了。

闫博士安全到达美国东部之前，路德还和往常一样早晚两集主持路德时评节目。细心的观众可以注意到，路德做节目时不时地看手机，并且在嘉宾发言时走开。这期间一直在和闫博士联系，并安排她的行程。在一万多公里的航程中，她不敢吃任何东西，怕中毒或被下药。她也知道对方的手段，一旦暴露，对方完全可以动用国家的力量，在太平洋上空强令飞机返航。

当中共确认闫博士已经到达美国并处于最高规格的安保中后，知道大势已去。一个月后的2020年6月，裴伟士迅速退休。身为香港大学最久负盛名的教授之一，连送别的晚会都没有参加，扔掉他刚刚从林郑月娥手上拿到的巨额研究经费，直接逃回了他的贼窝斯里兰卡([生化危机 3: 楞伽山之妖](https://gnews.org/zh-hans/615608/))。
