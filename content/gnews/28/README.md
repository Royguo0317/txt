## 快讯！中共国北方沙尘暴又来了，超15省市将受影响
` 喜马拉雅德国慕尼黑感恩农场` [轉載自GNews](https://gnews.org/zh-hans/1027740/)

收集：Jay ；编撰：WLQF

【Gnews 北京时间2021年3月28日】受东移蒙古气旋的影响，中国北方地区将出现大范围沙尘天气。此次沙尘天气过程将影响约15个省份。

![]()![](https://gnews.org/wp-content/uploads/2021/03/截屏2021-03-28-上午8.02.52.png)图片来自网络

其中，北京受沙尘影响时间将超12小时。今早至白天有沙尘暴，最低能见度1000m左右。北京市气象台已经发布沙尘暴黄色预警信号以及大风蓝色预警信号。据北京市气象局消息，今天北京风力将增强，有4到5级偏北风，阵风7到8级，山区可达8到10级。

根据中共中央气象台发布的消息称，3月28日至29日，新疆、内蒙古、甘肃、青海、宁夏、陕西、山西、北京、天津、河北、山东、河南、辽宁、吉林、黑龙江等地的部分地区有扬沙或浮尘天气，其中，河北、北京、天津、辽宁、内蒙古等地区有沙尘暴。

此次北京遭遇强沙尘暴已经是今年的第二次了。第一次就发生在北京两会期间的3月15日，被网友称为北京近十年来见到的最严重的一次沙尘天气过程。针对一次又一次的沙尘天气，中共网友都在质问，为何三北防护林没有“拦住”沙尘？
