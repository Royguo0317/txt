02月21日，美国总统拜登准备与德黑兰就限制伊朗核计划进行谈判，美国家安全顾问杰克·沙利文在CBS频道这样说。

“他愿意与伊朗坐到谈判桌前，讨论我们如何重新回到对伊朗核计划的限制。这个提议是有效的。”沙利文说，拜登打算克制伊朗获得核武器，并认为最好的方法是“勤奋、精明的外交”。

此前，美国国务院发言人斯内德·普莱斯（Ned Price）也表示，美国愿意应欧盟的邀请，以 “五加一 ”的形式（俄罗斯、英国、中共国、法国和德国）参加与伊朗和其他参与者的会谈，讨论伊朗的核计划。

2015年与伊朗达成的核协议，即《联合全面行动计划》（JCPOA）。2018年，美国退出该协议，并对德黑兰实施经济制裁。美国川普政府后来多次扩大制裁范围。

今年2月，路透社报道称，拜登政府已通知联合国将川普的请愿书撤回，该请愿书要求对伊朗重新实施制裁。
