## 歐洲三駕馬車如何應對中共
`2020-08-18 07:33 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/301567/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/18072921/1-88.png)圖片來源：https://www.eurasiareview.com/28062019-france-germany-britain-to-provide-instex-with-credit-line-by-friday/ 
哈佛研究員菲利普·勒科爾（Philippe Le Corre）和約翰·弗格森（John Ferguson）8月14日在《外交家》雜誌上發表評論文章：主題是隨著英國的領導和法國的跟進，世界正在等待德國對華為和香港等的決定。

文章指出，在美中關係似乎每天都跌至新低點的時候，兩國針鋒相對互關領事館，川普總統全面施壓中共科技公司的當下，雖然歐洲還未現對中共政策的拐點，不過這場新冷戰在歐洲戰場的態勢也正在逐步升級。

文章也列舉了英法如何應對中共的一系列行動，近幾週，英國已採取重大舉措來改變對華政策：1、邀請香港英國海外護照持有人申請永久居留權；2、從2027年起將華為排除在其5G網絡之外。法國在這兩個方面都效仿英國。

英國政府禁止英國電信運營商購買華為設備，以及在2027年之前拆除現有華為5G設備的政策，在歐洲各國掀起了波瀾。此前，由於英國堅稱其可以充分管理來自華為的風險，數月以來歐盟國家則一直藏在後面未做表態。然而，英國突然宣布此禁令之後，整個歐洲國家發現自己無處可藏。 7月22日，法國也決定在2028年之前事實上禁止華為5G設備。

英國外交大臣多米尼克·拉布（Dominic Raab）在倫敦的一系列活動引起了中共駐英國大使劉曉明的強烈反應，激怒了中共。比如在香港不斷擴大的衝突導致英國中止引渡條約，為英國國民海外護照持有人開放公民身份，以及對敏感警察裝備的貿易限制。更重要的是，英國最新型航母不久將開始在南海巡邏，並可能在文萊或新加坡駐紮。

文章表示，而德國是E3集團（譯者註：E3集團，即歐洲聯盟三巨頭包括英國、法國和德國）裡最後一個表態者，也可能是決定歐洲走向的最終決策者。因此，德國已成為歐洲5G的最後重要戰場。雖然非五眼聯盟的一員，德國自認其比美國盎格魯撒克遜同盟有更大的自治權。在2019年的中共之行，訪問了華為、騰訊等中共巨頭的高科技實驗室之後，默克爾總理（Angela Merkel）對中共的定位可能已經明顯加強。她訪問歸來後便制定了“新國家計劃”（Neustaat Plan），作為後默克爾時代的藍圖，它通過將傳統的德國工程技術和製造能力轉變為以科技為動力的數字時代來避免經濟停滯。

作者表示，該計劃特別批評了德國“過於官僚，複雜和緩慢，無法應對新的數字世界秩序的挑戰”的問題。默克爾將與中共的合作夥伴關係視為確保德國高科技未來的唯一可行選擇，從而來解決這些問題。迄今為止，在科技領域之外，默克爾在許多爭議性問題的激烈爭論中最為模棱兩可。但最終，她的政府不得不跟隨其他西方民主國家對香港新的國家安全法實施制裁。

最後作者評論道，中共很有可能認為與歐盟最終達成投資協議的機會正在迅速縮小，在默克爾退出後成功的機會可能大大下降。中共也可能繼續在貿易投資上進行拖延，直到默克爾的任期結束。無論哪種方式，毫無疑問，E3集團的決策對中共都非常重要，因此，儘管英、法、德在做法上可能還未步調一致，但若三國在中共外交政策上達成罕見一致的話，則會引發中共的憂慮並對其戰略部署造成壓力。
