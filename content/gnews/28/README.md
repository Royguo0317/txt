## 【鹰眼墙国】百公里马拉松12.2%的“致死率”，谁之过？

环球网5月23日转载央视新闻客户端消息，22日甘肃省白银市景泰县举办的马拉松赛事中，参加百公里越野赛172名人员，遇难者21人，另有8人轻伤住院救治。

白银市景泰县山地马拉松大赛于2018年开始举办，而今是第四届。今年的比赛共有近万人参加比赛和健康跑，而参与百公里马拉松的人员达172人。22日上午9时开跑，赛至中午时，赛段20公里至31公里处突遇冷空气并伴随冰雹、冻雨、大风气象，气温的骤降导致参赛人员出现失温症状（人体核心温度从37度～35度为轻度失温，35～33.88度为中度失温，在33.88～30度为重度失温达到“冰人”状态）。重度失温持续时间较长或施救方法错误都容易增大致死率。截至目前，因气温骤降死亡的参赛者已有21人，另有8人轻伤住院救治。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/23094305/Screen-Shot-2021-05-23-at-9.35.50-AM.png)
白银市景泰县连续三年承办马拉松越野项目，无论是赛事承办，还是相关准备活动都应较为熟练，然而在甘肃省气象局提出21-22日有“大风沙尘、降温降水”预警信息的前提下，作为提供比赛现场气象服务的景泰县气象局在气象通报中却只提到最低最高空气、风向等数据却只字未提冷空气过境信息。是县气象局的天气预报出现差错？还是在已知冷空气来袭却选择性的忽略？
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/23094246/Screen-Shot-2021-05-23-at-9.35.54-AM.png)
马拉松百公里赛事由于里程长、耗时长，所以需要沿途提供如移动厕所、卫生清洁、医疗救护、弃赛收容等服务。引导车、补给车、救护车、收容车也应随时待命，提供必安全保障。同时，马拉松赛场的救护车一般配有氧气、便携式呼吸机、心脏起搏器等救助设备。AED（自动体外除颤器）也成为马拉松赛道急救的标配。但是在上述事故中，赛事承办方失责，不但赛道遭遇冷空气这一紧急情况只是来于参赛人员在工作群的求助，更未见上述必备的安全保障车辆随行。22日的天气情况本就属于低温症的高发区间，再加上突发的冰雹、冻雨恶劣天气，低温症是172名参赛人员必然要遭遇的险境。赛事承办方已有三年承办经验，却只派20了人参与救援行动。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/23094222/Screen-Shot-2021-05-23-at-9.36.00-AM.png)
赛事公司在调取每个参赛人员的GPS定位情况并增派救援力量，但泥石流天气加剧了地面救援难度：12点17分得到求助信息，但直到14点救援队伍才到达高难山地越野赛道上山点。救助力量的严重不足，时效性的低下，救援的不及时，最终酿成21名重度失温人员的遇难的悲剧。

山地马拉松大赛作为特色宣传的体育赛事，不但缺乏必要的天气实时通报，更缺乏安全救援保障体系。172人参赛21人致死的惨痛代价能否唤醒当地政府的“人命优先”的意识，我们无从得知。但可以肯定的是奴役、鱼肉人民的中共体制不被推翻，各地政府和组织的官僚主义、形式主义作风也不会有丝毫改变。
