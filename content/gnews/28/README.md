## 蓬佩奧明確中國媒體被中國共產黨所控制，並由習近平將軍親自導演
`2020-10-10 23:33 GM101` [轉載自GNews](https://gnews.org/zh-hant/416866/)

美東時間10月9號，美國國務卿蓬佩奧先生接受了Newsmax的採訪，就中共國媒體對其評價等問題發表了自己的看法。1998年成立的Newsmax是美國最具影響力的保守派新聞和網站，每月約有400萬獨立訪問者。

針對中共國國企媒體對其“有史以來最差的美國國務卿”的評價，蓬佩奧指出中國媒體被中國共產黨所控制，並由習近平將軍和他的同志們導演，因為本屆政府限制了中共在全世界的惡性活動，所以導致了中共國媒體的低等評價。

針對中共超限生化武器“新冠病毒”對美國造成的影響，蓬佩奧表示中共專制政權向世界隱瞞病毒資訊，讓美國和世界人民為此付出了代價，世界人民已經瞭解中共的威脅，越來越多的國家選擇與美國結為盟友，而中國共產黨已經開始以多種方式付出代價。

蓬佩奧指出本屆政府的區別在於拒絕向中國共產黨屈膝和妥協，川普總統將做出重要決定，而自己不能在川普總統宣佈決定前透露相關資訊。
