## 12港人被設局送中？港人家屬抗議促飛行服務隊公開記錄
`2020-10-08 11:17 shangshang` [轉載自GNews](https://gnews.org/zh-hant/411375/)

作者：vog尚
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/08110749/E2A33B8D-A4E4-4291-A52B-DACA21C76A88.jpeg)
新聞來源：頭條日報

12名因參與反修例示港人，仍被深圳鹽田區看守所扣留，有報道指飛行服務隊曾在當日出動並於空中進行監視，12名港人部份家屬今午到赤鱲角的政府飛行服務隊總部抗議。警員指眾人違反「限聚令」，拉封鎖線調查。

12名港人部份家屬今日下午到赤鱲角政府飛行服務隊總部抗議，要求公開相關的出勤記錄，質疑港府預先設局，並通報內地進行拘捕行動。

早前有傳媒聲稱，飛行服務隊曾於當日派出定翼機「B-LVB」及獵豹直升機「B-LVH」協助警方追蹤12名港人乘坐的快艇，又指航機曾在西貢布袋澳上空盤旋，並在快艇附近上空飛行，質疑港府通報內地追截。
