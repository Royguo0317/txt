## 【微尘时评2期】《2021战略竞争法》——美国的灭共“战略路线图”

作者：喜马拉雅东京樱花团 / 喜马拉雅的微尘

美东时间4月22日，美国参议院外委会外交关系委员会以近乎全票（以21票赞成、1票反对）通过了《2021年战略竞争法案》（Strategic Competition Act），由于该法案直接剑指中共，且具有跨党派、层次高、涉及面广、杀招实招多、可执行性强等特点，虽然该法案尚未最终通过，后续还将经历参众两院审议投票的程序，但已经引起了全世界密切关注，当然同时也引起了中共的激烈反应。

该法案近300页，共分为投资美国竞争力、投资盟友与伙伴、投资美国价值观、投资经济及确保战略安全5大章节，该法案是在延续川普政府对中共政策和策略的基础上，经由美国两党对中共和中共国战略和政策所做的一次全面总结、整合、更新与强化，可以说，该法案是美国对中共和中共国的战略和政策全面转向的里程碑式法案，甚至堪称美国灭共的“战略路线图”。

**一、法案总体简析**

**（一）法案制订的主要背景**

**一是中共合法性的“遮羞布”被爆料革命扒掉，让全世界彻底认清了中共的邪恶本质**。四年多来，爆料革命广大战友坚持不懈地揭露和传播中共的真相和本质及其对世界的危害，随着事实不断地被验证，全世界对中共的认识愈发深刻，特别是香港“反送中”事件、中共病毒的真相更是将中共的邪恶生动地呈现在世人面前。通过爆料革命的长期运作以及与美国政、军和情报等各界的长期合作，蓬佩奥国务卿于2021年1月19日宣布“中共在新疆对维吾尔人犯下了种族灭族和反人类罪”，至此中共及其政权合法性已经荡然无存。随着美国新任拜登政府对中共“种族灭族和反人类罪”的再次确认，以及后续英国、加拿大、澳大利亚等国家的跟进认定，都已标志着中共的邪恶本质已经为全世界所看清。可以说，此法案具有政治、价值观和道义的正确性。

**二是中共的问题和危害已到非解决不可的地步，迫切需要美国牵头制订系统性的“灭共”战略**。中共对美国再在其本土长期开展以3F为目的，以政治、经济、司法、网络信息、媒体宣传甚至生物病毒等为手段的超限战，在国际上开展以削弱美国影响力、破坏美国国际秩序为目的，以控制国际组织和“国际会议”、结盟邪恶势力国家、推行“一带一路”战略等为手段的领导地位抢夺战，中共近年变本加厉的作为已经让美国寄望于“和平改变中共”的想法愈发渺茫。及至3月18、19日在“中美阿拉斯加会谈”上，中共表示“不吃美国这一套”，以及4月14日至17日美国气候特使克里（John Kerry）代表拜登政府访华“传达最后通牒”时，中共表示“拒不接受”，美国对中共的仁至义尽换来的是中共的死不悔改，至此中共耗尽了美国给的最后机会和耐心。由于灭共是全球大事，作为全球领导者的美国就有责任也有必要牵头制订相应的灭共法案，因此，此法案既能调动美国自身的力量和资源，同时又可作为其盟国行动的参照和指引。

**三是中共病毒对世界的危害和影响已不容久拖，亟待揭秘真相以彻底解决病毒问题**。截至目前，中共病毒已造成全球约1.5亿人感染、三百多万人死亡，但全球病毒蔓延尚未得到有效控制，死亡人数不断增加，同时变异毒株还不断出现，导致全球未来充满不确定性。中共病毒对资本市场、贸易、产业链、跨国投资以及餐饮酒店、交通等行业影响巨大，而导致全球出现企业倒闭潮、失业潮，使得全球经济陷入二战以来最严重的衰退。由于中共病毒是超限生物武器，病毒强变异和免疫逃逸的特性就决定了其不可能有疫苗，但中共及世界知名生物医药企业却大搞“疫苗经济”，还在制造中共病毒的二次灾难。更令人失望的是，在1月22日WHO发布的“武汉病毒考察报告”中，WHO仍与中共沆瀣一气，罔顾病毒真相。可见，中共不除中共病毒就无解，而中共病毒不除，全世界将永无宁日。可见，此法案也是解决中共病毒现实危害的必须。

**（二）法案的主要特点**

**一是靶向准，是灭共专用法案**。至少可从以下两点来看：第一，法案开宗明义剑指中共。在该法案的第一页就写明“致力于解决涉及中共国（PRC）的问题”字样，可见该法案的目的和对象十分明确，就是针对中共解决中共。第二，法案对中共和中共国的称谓严谨讲究。该法案通篇使用“中共”（CCP）和“中共国”（PRC）这样标准严谨的称谓，未使用“中国”（China）这样广义的称谓，体现了美国两党认同“中国人不等于中共、中国不等于中共”，从称谓就可以看出该法案只针对中共和中共国（中共政权），而非中国和中国人民。

**二是民意强，是跨党派法案**。该法案由民主党参议员梅内德斯（Robert Menendez）及共和党议员里施（Jim Risch）等共同提出。美国两党政治长期存在“为反对而反对”的常态，因此跨党派法案极为罕见且一般针对国内事务，针对一个外国政党和政权这种国外事务的跨党派法案就更为罕见了，由此可见，该法案所代表了最广大的美国民众对中共和中共国的态度和民意。

**三是层次高，是战略性法案**。该法案定位为战略层面的方案，是美国国家意志的集中体现，“对抗中共、消灭中共”将成为美国当前及今后一段时期的大事，在外部美国要通过军事、经济和科技等一系列支持来强化盟友的能力和与盟友的关系，并与全球盟友携手抗击中共，在内部美国从国家机器到民间组织、私人企业等各个领域都要被发动起来，集中整个国家的资源和力量来与中共对决，胜负不分将誓不罢休。

**四是涉及面广，是综合性法案**。该法案内容覆盖面非常广，可以说，除直接动用武力以外无所不包，该法案基本涵盖了政治、经济、军事、外交、科技、意识形态和价值观等方方面面。可见该法案是美国全系统、多维度与中共全面对决的综合性法案。

**五是可执行性强，是务实型法案**。法案的关键在于落实。该法案对于“要干什么”、“为什么干”、“要达到的目标和效果”、“谁负责干”、“如何干”（包括各部门的协调配合等）、“资源保障”、“资金安排”和时间节点等等都给予了明确定义，使得该法案易于执行和落地。此外，还有2点值得特别注意，首先，对法案执行落实要求总统负总体责任，在该法案的正文前的说明部分的第5节（sec.5）“国会意见”（Sensed of Congress）就对在法案落实过程中总统的职责进行了规定，即要求总统在落实法案中负责整个行政系统各部门的领导和指挥，以及确保分配足够的资源来保证法案落实。第二，负责法案具体落实的官员级别高，法案要求联邦政府各个部门和机构的负责人应指定副部长或以上级别的高级官员来协调和推进法案落实。

**六是杀招实招多，是强力方案**。该法案虽然站位高、涉及面广，但其内容绝非“花架子”和“面子货”，而是有不少直击中共七寸的狠招甚至杀招。如“撤离在中共国的美企并将中共国踢出美国供应链”、“打击中共的信息战和虚假宣传”、“加强美国与台湾关系并给予台湾与他国政府同等待遇”、“抵制2022北京冬奥会”、“调查中共病毒真相”等等。由于后续还将对部分条款做具体简析，在此不作展开。

**二、值得关注的重点条款及简析**

中共能够对外渗透、影响甚至控制全球，对内成功绑架14亿中国人，主要有**“钱袋子”、维稳能力、宣传造假能力和外部生存空间**四方面依仗。在此，值得关注的重点条款是指，法案中对中共打击直接有效的主要相关条款。下面，笔者就基于4月22日参议院外交委员会审议版法案做简要的梳理和分析：

**（一）与打掉中共“钱袋子”相关的主要条款**

**一是“拨款助科技供应链撤出中国”**，法案授权行政部门在2022年至2027的六年期间，每年动用1500万美元的预算, 供美国驻外使馆聘请外部专家，协助美国企业或个人撤出中国市场、将部分生产设备移出，抑或是将供应链多元分散至中国以外地区。

**二是“在资本市场清理清查中共企业”**，法案要求国务卿会同国家情报总监、财政部长应在法案生效的180天内向国会有关委员会提交一份关于中国企业对美国资本市场的威胁和风险评估报告，内容包括是否存在影响美国国家安全、严重侵犯人权，以及重大财务风险等问题。

中共的“钱袋子”，是指改革开放以来积攒的财富，这给了中共在国际上开展“蓝金黄”、“撒币外交”和“装大尾巴狼”的底气和硬气，以及对内强化政权合法性和维稳能力的依托。该法案要求美企撤离中共国，同时将中共国从产业链剔除，由此将导致中共国的经济和产业塌方甚至崩盘；在美资本市场清理中共控制企业，一方面将切断中共通过美国资本市场向全球吸血的管道，另一方面还将加剧外资从中共国撤离，由此将给中共金融、经济造成重大打击。可以说，没钱，中共对内政权的合法性就难以为继，同时对外也没有“蓝金黄”的资本。

**（二）与打掉中共维稳能力相关的主要条款**

**一是“打击中共的‘数字威权主义’”**，法案提出美国主导关键数字技术的国际标准制定，扩大信息和通信技术（ICT）产品和服务的市场份额，压减对中共国信息和通信技术产品的供应需求。与欧盟、日本、台湾和“五眼联盟”组织成员国签订数字商品和技术贸易协议。

**二是“加强出口管制和投资筛选制度”**，法案规定对中共国在人工智能与机器学习、5G电信和其他先进的无线联网技术、半导体制造、生物技术、量子计算、监视技术（包括面部识别技术和检查软件）等方面实施出口管制；同时还加大对中共借道香港规避美国出口管制和资本投资的审查力度。

中共的维稳能力，主要依靠“军警”和“监控技术”，近年来中共在军警队伍的人数不断扩大、装备水平不断提升的基础上，还不断加大运用信息通信、大数据、人工智能等技术对人民进行全面密切监控，强化维稳的预判和预警能力，打造了一个“高科技奴隶社会”。由于中共在半导体、集成电路、人工智能以及相关监视技术方面缺乏自主研发能力，该法案必将重创中共数字维稳能力。

**（三）与打掉中共宣传造假能力相关的主要条款**

**一是“打击中共全球的恶意影响力”**，法案将在2022至2026间，每年拨3亿美元到“对抗中国影响力基金”，以对抗中共在全球的“恶意影响”（malign influence）。面对中共舆论战，法案授权于2022年至2026年财政年度期间，每年拨款1亿美元给美国国际媒体署（USAGM），资助支持驻外媒体、建立独立媒体、在中国境内与境外对抗虚假信息及投资能规避言论审查的技术。法案也授权在同一期间，每年动用1亿7000万美元经费，资助支持媒体自由、训练与保护记者的计划。

**二是针对“一带一路”推动印太基础建设、军事援助和外交**，因应中国“一带一路”计划，法案预计拨款7500万美元，授权美国国务卿建立“基础建设交易与协助网络”计划，在印太区推动发展具永续性、透明与高品质的基础建设；拨款20亿美元，用于专项援助印太区域的国家和地区；拨款1250亿美元，用于印太外交。还将为印太地区提供6.55亿美元的对外军事援助资金，以及为印太海上安全倡议（Indo-Pacific Maritime Security Initiative）提供4.5亿美元。

宣传造假能力，这是中共的“以假治国、以骗治国”的重要基础和手段，对内宣传洗脑营造“伟光正”虚假形象，对外欺骗世界制造“繁荣强大”的虚假形象，通过夸大政治、经济、军事和科技等方面成就制造大量虚假宣传素材，以及利用“防火墙”、电视报纸等传统媒体、互联网和移动社交媒体等手段制造虚假的宣传信息。在此基础上，再通过在国际上大搞能源开发、基础设施建设（包括港口，电网，5G电信网络和海底电缆）输出影响力和控制力。法案的以上条款，必将打掉中共通过各种宣传机器营造的“假光环”和假形象。

**（四）与打掉中共外部生存空间相关的主要条款**

**一是“开展中共病毒真相溯源”**，法案要求国家情报总监与其他美国政府部门合作，在法案通过后180天内搜集整理并且提交国会一份有关中共病毒来源的报告“新冠疫情大流行报告”，该报告要包括中共病毒最有可能的来源，美国政府拥有的详细信息，以及武汉病毒研究所或者其他相关机构有关来源，传染力，泄漏等的相关信息，指明一种最有可能的病毒来源，并且展示如何得到这个结论，等等。

**二是“禁止美国政府代表团参加2022北京冬奥会”**，针对中共种族灭绝罪，法案要求禁止美国派政府代表团参加2022年北京冬奥会，对其实施外交抵制。

**三是“强化台美关系”**，法案承诺松绑美台官方交流限制，提升美台关系和交流层次，加强对台安保，强化推动台湾参与联合国、世界卫生大会（WHA）、国际民航组织（ICAO）、国际刑警组织（INTERPOL）及其他适宜国际组织，促进台湾与美国及其盟国的国际合作。

中共的外部生存空间，也就是世界对中共合法性的认可，由于一个国家政权的合法性，除掌控国家机器之外还要为外部世界所承认，这对中共政权而言至关重要。由于中共靠假骗盗取政权，中共一直对政权合法性的充满担忧和焦虑，加之其执政能力低下，导致国内矛盾社会问题重重，获取所谓的外部认可和国际形象就显得至关重要，由此可以达到以外打内、以外压内的效果来压制中共内斗和舒缓民众质疑。由于中共释放病毒为祸世界，加之其“种族大屠杀”和“种族灭绝”罪名已经做实，这两件事情的真相是中共无法承受之重。再加之，习近平上台后一直在备战“收复台湾”，该法案也让中共的台湾梦破灭。

**三、简要结论**

中共为祸世界，世界苦其久矣。及至《2021年战略竞争法》推出，美国以国家战略的形式，集举国资源合盟国之力，全面抗击中共，如本文标题概括，该法案就是美国灭共的“战略路线图”。
