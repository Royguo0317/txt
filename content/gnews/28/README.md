台湾的蔡英文总统於5月10日以视讯会议的方式在第四届“哥本哈根民主高峰会”中表示，威权国家势力的扩张使民主价值遭受威胁，全球民主伙伴必须团结合作。中共国外交部发言人华春莹11日针对此次会议回应说，这场峰会是一场“不折不扣的政治闹剧”，并自己吹捧中共国才是“真正的民主”。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/12085542/5eecb044d0f21-3.png)

台湾已经连续第五年没有收到世卫组织的邀请，被拒绝参加世卫组织的年会。台湾的外交部长吴釗燮发文表示，中共阻挠台湾参加世卫年会，并呼吁世卫组织做正确的决定，让台湾参加。对此中共国外交部发言人华春莹说“没有人比中国更关心台湾同胞健康”。台湾外交部长吴釗燮痛批中共说谎。

中共外交部说中共国才是真正的民主，其意思是说中共是中国人民的真正主人，中国人民都是奴隶。这就是中共所说的“民主”真义。

台湾从国民党的威权统治下经过数十年民众的抗争牺牲，才争来现在的民主自由，民众在法律保障的范围内可自由发表自己的任何观点言论，政治人物可被公开评论甚至提起诉讼，这些都是在中共统治下的中共国不可能发生的事。民主在台湾已经是现在进行式，而在中共国只是催眠式。期待在未来新中国联邦的中国有真正的民主。

台湾在中共的阻挠下持续被排除在世卫组织外，在此次中共病毒的疫情中，反倒是幸运的事。就由于台湾不能参加世卫组织，一切要靠自己，所以在此次中共病毒疫情初始时，就凭著以往对中共说谎成性的经验判断，中共所说的不会人传人一定有问题，当机立断管控中共国人入境，避开了初始传入病毒的危机。中共故意製造病毒，还隐瞒疫情，想要将中共病毒传到世界各地。竟然还公然说“没有人比中国更关心台湾同胞健康”。只能说中共是黄鼠狼给鸡拜年——没安好心。

在中共国，不被中共当人民的主人，才会有真正的民主；不被中共关心民眾的财產何时可收割，才是最好的关心。期望真正的民主及关心早日在中国实现。
