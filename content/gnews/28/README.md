**译评：**

该篇关于美军士兵拒绝接种中共病毒疫苗的报道，显示了向来重视情报工作的海军官兵对疫苗的担忧。左媒CNN的报道一如既往，文章里有很多关于中共病毒疫苗不正确的说法，比如“接种中共病毒疫苗是战胜该流行病的最佳手段”，就根本没有足够的证据以支持该说法，而且也缺乏理论依据。该篇报道还提到了美国食品和药品管理局（FDA）给予某疫苗的许可，其实并非正式许可，仅仅是“紧急许可”。

即使不考虑中共病毒是来源于解放军的生物武器，对如此仓促推出的疫苗抱有安全疑虑也是正常现象，更何况对于海军士兵来说，他们对于中共病毒来自解放军实验室的怀疑，距离他们本身的专业领域其实并不遥远，所以这样的结果也毫不奇怪。我们注意到，更精锐的海军陆战队勒琼营（Lejeune）的疫苗怀疑者比例更高，那是因为他们更专业，并且有更重视情报的传统。

在该篇报道中提到，有些民主党议员竟然向白宫呼吁，希望在军队强制推进中共病毒的疫苗接种。这篇CNN报道的推文下也有很多读者有类似的呼吁，甚至有人提出对拒绝免疫接种的军人取消其感染中共病毒的医疗保险。其实关于疫苗副作用的报道已经越来越多了，近期澳洲的医生已经开始公开质疑，注射中共病毒疫苗的医疗行为将因严重的血栓副作用而产生法律风险。

全世界范围内，这种极为草率和鲁莽的大规模接种中共病毒疫苗的行为，所造成的危害将会继续显露，并对我们的社会产生深远的影响。这些疫苗所带来的切身危害，必然唤醒大批左翼的民众，让他们从对主流媒体、“专家”和政府的盲信盲从中醒悟过来，向倡导限制政府权力的保守派和平民主义靠拢，从而导致“大觉醒”的到来。

**译文**：

美国有线电视新闻网（CNN）华盛顿消息：根据美国海军陆战队周五向CNN提供的数据，该队近四成人员拒绝接种中共病毒Covid-19疫苗。美国海军陆战队是第一个披露全军接受接种和拒绝接种数字的部门。

截至周四，约有75500名海军陆战队员接种了疫苗，包括完全接种和部分接种的男女服役人员。约有48000名海军陆战队员选择了不接种疫苗。不接种率为38.9%。CNN已与其它部队联系，以了解接受接种率和拒绝接种率。

海军陆战队员对疫苗接种的接受率——61.1%与军方早先估计的三分之二，即约66%相差无几。另有102000名海军陆战队员尚未获得疫苗。海军陆战队的总人数包括现役、预备役和个人动员增援的海军陆战队员。

根据CNN收到的另一组数据，在著名的海军陆战队基地之一北卡罗来纳州的勒琼（Lejeune）营，拒绝接种率远高于此，达到57%。在26400名已获得疫苗的海军陆战队员中，有15100人选择不接种疫苗。这个数字既包括海军陆战队第二远征军，也包括海军陆战队东部设施勒琼（Lejeune）营。另有11500名现役海军陆战队员将按计划获得疫苗。

海军陆战队发言人凯丽·弗拉绍尔（Kelly Frushour）上校在一份声明中告诉CNN：“我们完全了解，广泛接受Covid-19疫苗接种是战胜该流行病的最佳手段。而应对该流行病的关键是建立对疫苗的信心”。弗拉绍尔称，海军陆战队员选择不接种疫苗的潜在原因有很多，包括让别人先接种、等到强制接种、通过其它渠道接种或对疫苗过敏，“今天拒绝接种的服役人员可能改变主意，在下次有机会时接种疫苗。”

CNN上个月曾报道，服役人员拒绝接种疫苗的比率可能接近50%。这个数字明显高于国防官员公开使用的33%的数字。

军方现在不能强制接种疫苗，因为他们只有食品和药物管理局的紧急使用授权。这意味着需要接受其它一系列疫苗接种的服役人员，可以选择拒绝注射中共病毒的疫苗。

官员们表示，大部分人犹豫是否接种疫苗都是由于担心疫苗的研发速度及其长期的影响。

国防部约有220万名服役人员在全球各地执行任务。接受率每下降10个百分点，就意味着有22万人选择不接种疫苗。这个数字可能大到足以影响部队备战。去年，军方经历了几次Covid疫情，受到高度关注。其中一次疫情发生在部署于太平洋地区的一艘航空母舰上。

上个月，一群民主党议员呼吁总统乔·拜登发布“知情同意豁免令”，强制所有美军服役人员接种中共病毒Covid-19疫苗。他们在信中写道，“虚假信息和疫苗怀疑论”正在影响服役人员选择不接种疫苗。
