雖然由郭文貴先生領導的爆料革命，在三年多前就已經不停的警告、吶喊關於中共的邪惡及其極獨霸世界的野心，西方卻充耳不聞。一直到中共武漢疫情爆發後，將中共一群群可能感染的人們送到世界，引起大量人口感染及死亡(現在全球已經超過50萬人死亡)後，才把這個自由世界在美夢中慢慢醒來，使自由世界真正開始認清了中共本質與威脅，開始面對這已經幾乎被白蟻式侵蝕簍空的西方文化與價值觀。

這是一篇訪問稿，由戈德科恩訪問斯伯丁將軍關於新冠病毒對中美關係的影響。文中可以看出戈德科恩不是川普總統的支持者，也擔心中美關係的惡化對美國人本身的傷害。但斯伯丁將軍去直接了當地指明中國有兩部分，黨與人民。中國共產黨並不代表中國人，他們是中共政權下的最大受害者。斯伯丁將軍並指出現在對付中共集權的挑戰，除了要從製藥，財經，產業鍊等等多頭徑行來健全美國本身國家安全政策並應與盟國團結一起對抗這些挑戰。最後並指出，中美脫鉤是必要的，不然美國人將繼續受到傷害。

原本美國天真的以為對中共開敞大門，中共會經由學習西方而改變，但對這與歐美自由分享世界基本價值觀完全不同的中共是個癡人說夢。現在的美國就像一位糖尿病患者面對必須切除那已經壞疽的腳，脫鉤是長痛不如短痛，必須的求生之路。

##  **新冠病毒將如何重塑美中關係？** 


> Genesis 創世紀

[!\[\](https://spark.adobe.com/page/1ZvAfaYpuSVE6/images/4e19fecd-aac1-4a93-8ab3-47b0907355e1.jpg?asset_id=d9693be1-4d36-4aac-a582-213999606521&amp;img_etag=%228316fa2f30fb350de969b1f0c97f30fd%22&amp;size=2560)](https://spark.adobe.com/page/1ZvAfaYpuSVE6/images/4e19fecd-aac1-4a93-8ab3-47b0907355e1.jpg?asset_id=d9693be1-4d36-4aac-a582-213999606521&amp;img_etag=%228316fa2f30fb350de969b1f0c97f30fd%22&amp;size=1024)
(戈德科恩)目前，美國在兩黨之間所達成絕無僅有的共識是需要對中共國採取強硬態度。在新冠病毒大瘟疫之前，中美關係已經處於低谷。大瘟疫只會使此更加惡化。

即使川普政府沒有試圖分散對新冠病毒的疏忽處理的注意，這說法也是正確的。但是白宮目前的言論（儘管總統表示對習近平有濃厚的感情）使川普的反對者幾乎沒有改變路線的餘地。

為什麼在新冠病毒疫情之前的關係就已經如此糟糕？美國對中共國有許多合理的不滿，以下僅列出一些：

- 不公平的商業行為和產業政策，包括國家補貼和與中共國“冠軍”公司的模糊關係。
- 南中國海的軍事化
- 駭客和間諜活動。
- 從投資到記者待遇，在各個領域都沒做到互惠的安排。
- 一張關於地獄的侵犯人權的目錄，包括新疆的現代集中營，該集中營已經拘留了超過一百萬的維吾爾族人，拘留了律師，記者，激進活動家，並在中國不斷破壞著公民社會。


中共國對新冠病毒的缺乏透明度，以及其激進的宣傳和外交努力，只會使局勢惡化。我不相信斯伯丁將軍在這方面與我持不同意見。我們對於該如何處理這是可能會有不同。

我擔心的是，皮尤（Pew）最近的一項調查顯示，華盛頓特區的反華情緒現在正在全國蔓延，這使我們忽視未來的危險。我們不能一夜之間就將經濟與中共國脫鉤，即使那是最好的方法。有人真的想戰爭發生嗎?甚至是個金融戰爭？

我相信我們需要對關係中出了什麼問題以及如何改變它進行更多的思考。我還認為，我們需要考慮我們的行動如何影響中國共產黨及其領導人習近平的選擇。需要製定策略，而不是互喊。但是，現在，一致責備中共國的輿論淹沒了所有其他意見，這不利於我們的決策。

最後，川普政府如此大力推廣“武漢實驗室”理論，以至於包括澳大利亞和英國在內的美國盟國都站不住腳，這嚴重損害了它僅存的信譽。這也使得政府對華為危險性的特殊了解的主張變得不那麼可信。所有這些使中共國在向世界宣傳方面具有優勢。這也使中國政府在未來的貿易談判或任何其他形式的談判中，甚至不太可能再相信美國政府。

因此，用一句話來說：新冠病毒將使美中關係更加惡化。這將使美國在任何改變中共國行為的行動中要想集結其朋友和盟友將變得更加困難。


> Response 應對措施

[!\[\](https://spark.adobe.com/page/1ZvAfaYpuSVE6/images/2bfde597-6756-486d-a93d-980eb73da019.jpg?asset_id=86be73c0-4971-49c3-9b86-73e9b839cf94&amp;img_etag=%22f4e1fbe4b752df3c80aa813b6aad0c66%22&amp;size=2560)](https://spark.adobe.com/page/1ZvAfaYpuSVE6/images/2bfde597-6756-486d-a93d-980eb73da019.jpg?asset_id=86be73c0-4971-49c3-9b86-73e9b839cf94&amp;img_etag=%22f4e1fbe4b752df3c80aa813b6aad0c66%22&amp;size=1024)
(斯伯丁將軍)文字的力量之大令人難以置信。在談論美中關係時，使用正確的詞語是至關重要。我們正在與中國共產黨而不是中國作鬥爭。公平地說，我知道傑里米知道這一點的，並不是反對中國人民，而是有時候用“中國”一詞簡單點。但是對我來說，至關重要的是，所有人都應該知道有兩個中國：人民和(中國共產)黨。當談到諸如政策，大外宣和大瘟疫等問題時，我們應歸咎於”黨”。

儘管我完全地接受傑里米（Jeremy）對中共缺乏透明度，他們的道德，道德和商業違法行為以及他們的宣傳的評估，但我不同意川普政府的“歸究中共的一致說詞”會對中共產生影響。我們的總統和我們的政治溫度是對中共戰略巨著的重點。

即使不是數十年，該(中共)策略已進行有年了，。它始於天安門之後，並將在2020年大選後持續進行。

最重要的是，中共將我們的民主，我們的人權法案和我們在憲法中的自由視為對他們統治的威脅。

(中共)9號文件最恰當地描述了這一點：“促進西方憲政民主：[是]試圖破壞當前的領導和具有中共國特色的社會主義的治理。”

因此，中共征服中國人民是不夠的。它還必須積極破壞國外的憲政民主，並防止其感染中國人民的思想。

中共接著說，中國人民必須接種以預防這種思想病毒。 “各級黨員和政府必須。 。 。將思想領域的工作做為日程表上的重點，定期分析和研究思想領域的新發展，迅速有效地作出反應，並搶先解決思想領域中的所有問題。”

其他相關文件包括：CCP憲法章程；天安門文件；和新疆文件等。

這些文件可以一眼看出中共的意圖。學者們不得不通過各種採訪和軼事的洩露或發現來推測中共的真實意圖和其本性。未能講述中共和中國人民的真實故事，使中共得以捕捉和控制說詞。那些被允許在大防火牆之外“很好地”講述中共國故事的，並且重視大陸簽證獎勵重於揭發中共真相的人也是同謀。我們對政權的性質了解得越多，所有民主國家就越安全。


> Penultimate 倒數第二次警告

[!\[\](https://spark.adobe.com/page/1ZvAfaYpuSVE6/images/de3d0200-76d3-4c10-89c3-876ec4afcc40.jpg?asset_id=1a477d88-9f1f-49fc-b90c-94136ae81de2&amp;img_etag=%22f053d106a2bf335f3985646ba4b36906%22&amp;size=2560)](https://spark.adobe.com/page/1ZvAfaYpuSVE6/images/de3d0200-76d3-4c10-89c3-876ec4afcc40.jpg?asset_id=1a477d88-9f1f-49fc-b90c-94136ae81de2&amp;img_etag=%22f053d106a2bf335f3985646ba4b36906%22&amp;size=1024)
(戈德科恩)我同意中國和中國人民與共產黨不同，在美中關係中大多數問題應歸咎於該黨。

但是，當前兩國之間的緊張局勢已經在傷害著美國和中國的人民和企業。無論您是將中美關係惡化的責任歸咎於中共，川普，還是與中共國政府合作過緊密的美國公司，最終結果都是一樣的。

“脫鉤”和“新冷戰”是中美關係的兩個最常見的字眼，在新冠狀病毒之後，我們將在美國媒體和政界人士口中更常聽到這些話。這將給在兩國之間以及與兩國有著深厚情感，財務和個人關係的公司，家庭以及許多美國和中國非黨派民眾帶來更大的痛苦。

因此，我覺得有必要重複我在這次交流的“開場白”部分中的問題。我不相信斯伯丁將軍在

這方面會與我有異議。我們可能不同之處在於該如何處理。

我相信，現任政府的戰略是完全零戰略，只是隨興的敵對行為，這種行為是不分共產黨或是中國人民的，也不考慮會對美國人民和企業造成的傷害。


> Finale 結束語

[!\[\](https://spark.adobe.com/page/1ZvAfaYpuSVE6/images/09302abd-5df3-4f80-a464-9e905c7703d8.jpg?asset_id=479fd588-15ca-4d6b-b9b4-57e97fd958f3&amp;img_etag=%22b8cbec81eb93e07cbff81dacc53c7356%22&amp;size=2560)](https://spark.adobe.com/page/1ZvAfaYpuSVE6/images/09302abd-5df3-4f80-a464-9e905c7703d8.jpg?asset_id=479fd588-15ca-4d6b-b9b4-57e97fd958f3&amp;img_etag=%22b8cbec81eb93e07cbff81dacc53c7356%22&amp;size=1024)
(斯伯丁將軍)要回答最初的問題，”冠狀病毒將如何重塑美中關係”，我得談談傑里米關於缺乏策略的評論。

2017年12月，我們的《國家安全戰略》（NSS）確定了我們應回答所有問題的確切方式。問題從一開始就是華盛頓那些資深政客無法理解NSS。

NSS中的重要部分已被完全忽略。不是被只負責執行的聯邦官僚機構，而是被大多數的華盛頓政客和媒體所忽略。因此，世界其他地區未能理解NSS的行動號召。

這國家安全戰略是基於“對美國利益的清晰評估的美國原則”。這意味著我們必須確保我們一直促進的言論和宗教自由，法治和自由貿易。它也承認我們已經停止向國外推廣這些原則，並已將教育和提倡(這些原則)的工作交給了21世紀如中共的專制主義者。

國家安全戰略意識到競爭不僅涉及炸彈和子彈，還涉及電腦科技，經濟金融。威權國家“決心讓經濟變得不自由不公平，發展其軍事力量，並控制信息和數據以壓制其社會並擴大其影響力。”因此，經濟和數據是該戰略中所定義的競爭的關鍵新領域。

這意味著美國必須採取不同的競爭方式，因為我們自1989年以來對中共的戰略極其幼稚。我們的政策“基於以下假設：與競爭對手的接觸及將其納入國際機構和全球商業中，將使他們成為良性行為者和可信賴的伙伴”。

為此，美國“必須與志同道合的盟國和夥伴合作，以確保我們的原則佔上風，並執行規則，以使我們的經濟繁榮。”這就是為什麼在國家安全戰略發布後，國家安全委員會和國務院開始與盟國和夥伴進行雙邊接觸，以倡導對中共領導的中共國等威權政權採取新的方法。我們的盟國正在回應。越來越多的美國盟友選擇應對中共提出的挑戰，對華為說不，並審視中共國的投資。在我們的后冠狀病毒世界中，美國及其盟國正無時無刻地目睹著獨裁政權對信息的控制和缺乏透明度所導致的亂象。

美國人之所以受到傷害，不是因為缺乏戰略，而是因為在製造業，製藥業和電信業等關鍵行業對敵對的獨裁統治太過依賴。 NSS就是為應對這一問題而編寫的，儘管還有很多工作要做，但在實行三年後該政策開始生效了。

(我們)在認清挑戰上正不斷地增強。像《國防生產法》這樣封存的權威法被除塵並投入使用。其他諸如外國投資委員會已被改頭換面。國家安全戰略提供促進政策和激勵措施將關鍵的國家安全產業帶回美國本土。 ”它尋求通過支持“聯邦和州級的科技工程數學領域的努力”來培養本地人才。它繼續說：“美國將優先考慮對經濟增長和安全至關重要的新興技術，例如數據科學，加密，自主技術，基因編輯，新材料，納米技術，先進的計算技術和人工智能。 ”

因此，是的，傑里米（Jeremy），是有一個策略，而且正在奏效。由於中共不斷掠奪性的經濟侵略，美國人受到了傷害，導致數百萬的工作機會流失。在中共允許下製藥廠生產芬太尼並將其運往美國，導致大量毒品在社區氾濫，殺害成千上萬人。當然，全球冠狀病毒大瘟疫，造成成千上萬人喪生，創造了創紀錄的失業率，並為數百萬人帶來了巨大的艱辛。

脫鉤很重要，因為美國人一直受到傷害並繼續受到傷害。有一個深思熟慮的策略是試圖為中共提供一條新的前進道路。目前看來，他們決定不接受。他們可能永遠不會決定接受，那也沒問題，因為我們已經有一條路讓美國恢復到她原本面貌–一個民主的港灣和抵抗專制風暴的庇護所。
