## 關於思想的隨筆
`2020-05-31 17:12 GM06` [轉載自GNews](https://gnews.org/zh-hant/218912/)

作者:南侠

很早就想寫一篇關於思想的文章，因為思想的力量大而無窮，有人的地方就會有思想。

哈耶克在《通往奴役之路》中有一句名言：“在社會的演化過程中，沒有什麼東西是不可避免的，使其成為不可避免的是思想。”可見，思想控制著一切，如果你能控制人們的思想，你就能控制他們的行為。

當然，思想有好有壞。好的思想讓人們走向民主、法治的自由之路，就像美國，宣導獨立之精神，自由之思想；壞的思想讓人們走向獨裁、專制的奴役之路，就像中共，党國不分，党領導一切，爹親娘親不如党親。

中共自他誕生之初，就已經意識到思想的重要性，一開始就打著馬克思、列寧主義的旗號，然後每一屆領導人上臺都會提出一套有代表性的思想體系，比如：毛澤東思想、鄧小平理論、三個代表、習近平新時代中國特色社會主義思想等。因為中共深知，誰控制了思想，誰就控制了一切。為了控制思想，中共建立防火牆，雇傭大量水軍，蒙蔽人們的眼睛，捂住人們的耳朵。天天宣揚：美帝亡我之心不死、反共即反華，國外水深火熱、國內安定平和。可憐的強國人一直生活在謊言中。

再看看中南坑的正門——新華門旁邊的那副類似對聯的標語，戰無不勝的毛澤東思想萬歲與偉大的中國共產黨萬歲是並列的。換句話說，毛思想在，中共在，毛思想亡，中共亡。

如今，爆料革命，橫空出世，為中國人在思想上點亮了一盞明燈，他的核心就是唯真不破。在中共國隨處可見的假大空社會裡，爆料革命就是消滅毛思想的那股中流砥柱！
