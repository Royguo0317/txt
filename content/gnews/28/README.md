## 首尔将给宠物猫狗免费检测中共病毒

**西班牙2021年2月8日电/西喜社——**韩国首都表示，如果宠物猫狗接触到受感染的人类并出现症状，将给它们免费进行中共病毒检测。

首尔官员朴有美(Park Yoo-mi)在网上发布会上表示，被发现感染病毒的宠物必须在家中或市属设施内隔离14天。

上周中央政府发布了对宠物进行病毒检测的指导方针，此前东南部城市金州的一只猫咪成为该国第一只被证实感染中共病毒的动物。这只猫属于一对母女，她们是与金州一家宗教设施相关的数十名确诊患者之一。

朴槿惠说，官员准备从周一开始对宠物进行免费检测。

首尔官员说，当局将对宠物进行测试，而不是所有动物，因为它们与人类密切接触。其他地方政府也计划对宠物展开类似的测试。

韩国新确诊的中共病毒病例总数周一跌破300例，为两个多月来首次，同时当局稍微放宽了物理距离的规定。官员开始允许人口密集的首尔地区以外的餐馆、咖啡店、室内健身房和其他设施延长营业时间一小时。
