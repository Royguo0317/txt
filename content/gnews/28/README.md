中共國將探索開發婚前輔導課程，民政部、全國婦聯近日聯合印發了《關於加強新時代婚姻家庭輔導教育工作的指導意見》，其中強調，探索開展婚前輔導，開發婚前輔導課程，幫助當事人做好進入婚姻狀態的準備，努力從源頭上減少婚姻家庭糾紛的產生。探索將頒證儀式引入結婚登記流程並實現頒證常態化，通過引導婚姻當事人宣讀結婚誓言、領取結婚證，在莊重神聖的儀式中宣告婚姻締結，讓當事人感悟銘記婚姻家庭蘊含的責任擔當。

良好的兩性關系，核心是建立在兩個健康的成年人基礎上。而中共的傳統教育，沒有人文教育，所有的孩子從小經歷的都是：考試和政治洗腦，只知道提升考試技巧。根本不會和人相處。

不知道如何和同學、室友、朋友、父母、親人，身邊所有人建立不同的親密關系。家庭中，家人也不會合理引導，出了問題要麽說：孩子還小不懂事，要麽說：學習第壹別的不用管。從小，沒有學習如何承認錯誤和道歉，只會找借口。

如果不知道如何與人相處，就不擅長和人建立親密關系。在伴侶和婚姻關系中出問題，不知道如何解決。失敗的教育所造培養的不健康的成年人，沒有正常的解決方法，不會成熟地解決問題。勢必造成婚姻悲劇，尤其是被“毒雞湯”誤導，很多人更不會解決矛盾。

簡而言之，婚前輔導課程就是整個中國教育失敗的體現。

管天管地管空氣的共產黨，最終無法為社會培養正常的“人”。很多人，成年之後，都在吐掉中共教育的“毒奶”之中，成長為正常、健康的成年人。
