10月4日，溫哥華天氣陰冷，溫哥華戰友仍然按照加農的安排，繼續到黃河邊家示威抗議。老江組織安排戰友行動，並和前兩天一樣，在黃河邊家門口歷數黃河邊的罪惡醜態。

這應該是溫哥華戰友持續第18天的打偽活動，主要偽類對象就是黃河邊。黃河邊打著自媒體人士旗號，行假騙之事，在報導中一直惡意歪曲和編造，對爆料革命和文貴先生極盡誹謗之能事，迫害海外追求正義的戰友。

今天的打偽活動自上午10點持續至下午5點，戰友們根據個人情況自願參加。北京人在北美戰友也在前台以飽滿的激情，幽默地揭示黃河邊的惡行，讓戰友們忍俊不禁，成為快樂滅偽、快樂滅共的好例子。

此前，戰友們給黃河邊的鄰居發放了宣傳單，鄰居們都已了解黃河邊一直勾結中共、出賣戰友並歪曲事實報導的醜惡，戰友們的示威抗議也得到了鄰居們的支持和理解，同時小區的治安警察也給予了支持。

溫哥華打偽行動，是爆料革命環球打偽行動的一部分，要讓世界知道這些偽類、欺民賊的惡行，這些偽類就是人類中的垃圾，中共的走狗。

打偽黃河邊的行動一直會進行下去。
