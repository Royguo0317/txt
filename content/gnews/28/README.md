03月18日RBC报道，3月17日公布了伊朗专家的最终报告，报告并没有让基辅满意。乌克兰外长德米特里·库列巴说，基辅认为，伊朗对2020年1月乌克兰波音飞机在德黑兰附近坠毁事件进行的调查，存在诸多违规行为。他称，伊朗当局就是这样试图 “玩世不恭地隐瞒”坠机的真实原因。

“我们今天在公布的报告中看到的，不过是企图狡猾掩盖飞机失事真正的原因。我们不会允许伊朗隐瞒真相，我们不会允许它逃避对这一罪行的责任。无论付出多少努力和时间，正义都会胜利。”库列巴在乌克兰外交部在其脸书上发表的一份声明中说。

据他说，伊朗对这次灾难的情况进行了技术调查，发现了许多违反国际标准的行为。“调查有失偏颇，提出的证据有选择性，结论有欺骗性。文件没有提供所有的情况，没有揭示悲剧的根本原因和导致悲剧发生的一连串行动。这不是一份报告，这是一个操纵的集合，其目的不是为了找出真相，而是为伊朗伊斯兰共和国辩护。”库列巴说。

除乌克兰外，加拿大也对该报告的可信度表示怀疑，因为加拿大的公民也在被击落的喷气客机上。“我们仍然没有答案，为什么伊朗没有采取必要措施，如完全关闭其领空，以防止这种悲剧的发生。我们重申，加拿大致力于寻求答案、伸张正义、追究伊朗的责任。”据CBC报道，外交部长马克·加诺（Marc Garneau）和交通部长奥马尔·阿尔加布拉（Omar Algabra）在一份联合声明中说，“并确保…充分补偿PS752航班遇难者的家属和亲人。”

2020年1月8日晚，乌克兰国际航空公司的波音737客机在德黑兰机场起飞时坠毁。机上有176人，分别是伊朗、加拿大、英国、德国、瑞典、阿富汗和乌克兰的公民。所有的人都死了。坠机事件发生在伊朗军队对伊拉克美军基地发动导弹袭击的当晚，是对卡西姆·苏莱曼尼将军遇刺事件的回应。

伊朗起初否认与这起事故有任何牵连。然而，后来出现的录像和其他证据证实了该客机受到地面导弹攻击的说法。1月11日，悲剧发生3天后，伊朗当局承认乌克兰国际航空公司的波音737飞机被误击落。伊斯兰革命卫队军事航天部队指挥官阿米尔·阿里·哈吉扎德（Amir Ali Hajizadeh）解释说，防空系统运营商将从德黑兰机场飞出的飞机识别为一枚巡航导弹。由于通讯问题，操作员没有及时向指挥部报告，独立决定向客机发射导弹。伊朗总统哈桑·鲁哈尼（Hassan Rouhani）亲自为这一事件向基辅道歉。2020年12月，伊朗政府批准向每个在空难中丧生的家庭支付15万美元。伊朗于2021年3月17日公布了关于该事件的最终报告。在报告中，专家已经证实“由于防空系统操作人员的失误，德黑兰附近的飞机被确定为敌方目标”，之后向其发射了两枚导弹。
