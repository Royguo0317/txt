澳喜农场堪培拉街头灭共宣传有感

在5月30日澳喜农场刚刚结束的又一轮全澳灭共大游行的活动中，六个城市共同线上线下联合行动，声势浩大地成功形成了又一次的爆料革命真相宣传和灭共潮流推动的积极效应。作为一个在首都地区参与堪培拉街头灭共宣传的战友，同时也是一个海外定居的华人，笔者有如下的观察和感想。

首先，从我们的现场战友综合起来的共同认识，我们这次通过又一次的勇敢亮相和非常明确的灭共宣言，充分地体现出来以“真善狠准”为出发的海外华人新形象。为什么这么说呢？先以一个小小的战友现场反馈来说明。堪培拉街头阵地宣传中，一宜战友在向路人和当地居民派发关于中共病毒真相小册子的时候，有人首先就问，你们是台湾人吗？当一宜战友回答，不，我们来自大陆。对方十分惊讶。他们知道之后也非常佩服。

从这个简短的交流我们可以知道，从大陆来的人給西方人留下的印象就是：胆小、畏缩、不敢当、没有正义感、根本不敢旗帜鲜明反共，更不要说进行灭共的实际行动了。可是，我们这样一群人，大方朴素地站在澳洲国家首都的繁华街区，人来人往的中心步行街，向来去的人流宣传爆料革命、揭示中共的邪恶、分享中共病毒真相、控诉中共的反人类罪行、高喊灭共的口号、为新中国联邦的未来疾呼，这一切体现出极为本质的海外华人形象改观。

“真善狠准”是爆料革命倡导人和发起者郭文贵先生提出的灭共关键点，也是新中国联邦新一代公民的基本要求。真就是基于一切的真相，所有的事实，只说和真相事实有关的资讯，分享基于数据和事实的所有确乎存在的发生。善就是关怀，就是无私，就是奉献，就是为了他人的公益之心。狠就是勇气、是无畏、是可以牺牲的力量、是对邪恶立场绝不相容的明确。准就是针对性强、就是抓住斗争的要点、就是分清主次。

多少年来，海外华人被很多打着民主旗号的人士代表，这些人吃着中共的伙食，被中共的经费喂养，出卖有民主诉求的民众，甚至与中共合作伤害海内外的真正向往民主的人们。这些人不仅给所有的中国人以扭曲和伤害，还带来了国际社会和西方人民对华人形象的误会和隔阂。很多生活在海外华人，受到蒙蔽，受到自身认识的局限，受到中共暴力机器的压迫，也不得不湮没了自己的自由权利和独立意志。他们就算在海外生活，也是图个平安顺畅，不想参与政治，不想积极为自己为他人发声。他们就是身处西方民主社会，也还是一直都是沉默、无声的人群。

这一次，从爆料革命现身于世开始，从郭文贵先生现身于国际灭共之路开始，平民革命的旗帜高扬，无数海外华人在文贵先生的勇气感染下和真相的感召下自发聚集，海外华人前所未有地出现一个全新的面貌。从此之后，大陆民众在海外的华人走向了一个国际化的民主追求之路。从这里，海外华人第一次全面公开地展开了自己与过去旧形象旧意识脱钩的重建之路。从这里，海外华人可以第一次为自己立言，为他人发声，为所在国的价值观响亮地维护，为所在地区的人民发出明确的反共宣言和做出有力的灭共行动。

“真善狠准”不仅是让海外华人得到一次洗涤和重塑，在此过程中，海外华人所取得的任何一点一滴进步和完善，都在无时不刻地感染和焕发墙内的同胞。海外华人在爆料革命中的坚持与努力，让我们墙内高压下的同胞深受鼓舞，激发了他们更加充满信心地遥相呼应，彼此协作。同时，海外华人在各个大洲各地区的努力，也相互鼓励了彼此的战斗。这些落实在行动上的灭共努力共同推进到了今天，我们可以感受到，可以看得到，可以体会到的西方社会与国际层面对爆料革命的不断改变的认识。他们看得到，爆料革命的旗帜下，所集结的是富有正义感，富有责任心，富有原则和立场，富有人的良知和行动力的华人群体。我们希望世界都能看到中共不能代表中国人，也不能代表中国，我们已经让世界看到，这就是成功。我们要在过去的努力基础上，继续秉承“真善狠准”的精神，做好海外华人的本份和应有的积极作用。华人要积极地参与这次改变世界、拯救人类的爆料革命，拯救他人也是救赎自己。继续努力，继续朝向更好的华夏人的明天。
