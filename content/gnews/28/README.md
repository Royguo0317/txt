我从未见过推进如此快的事情，从它发生至今原本被压制了这么久的时间。玛莎-布莱克本和我昨天就这个问题举行了新闻发布会，乔什-霍利和我提出的法案得到了美国参议院的一致通过，就是公布我们自己情报部门持有的信息。

这对拜登总统、民主党人施加很大压力。布林肯国务卿，几天前我采访过他，关于为什么不向每个美国公众媒体发布相关信息，从而搞清楚这一问题，他真的没有一个很好的答案。

**主持人：**有意思。在泄密的情报中，是否有某个点，真正让你觉得我们需要知道更多，美国人民需要了解这些信息？

他们之所以这样做，意味着他们不希望当这件事（新冠病毒来自实验室，是生化武器）完全曝光的时候自己被追责。这是我在美国参议院以来见过的推动得最快的议案。所以，一定有一些结果。如果没有，它至少会形成一种趋势，也许我们会让中共国来承担责任。

看看本届政府和我们欧洲的伙伴（他们对中国的依赖性更强）将如何推动，一定很有趣，这是我们需要关注的。

**布莱恩：**对于福奇，我们唯一想说的是，这件事他完全脱不了干系。他从新冠病毒大爆发的最开始就误导大家。但是两个星期前，福奇和柯林斯开始说整件事需要透明度。那么他们是否会建议总统发布这些情报机构中信息呢？就像几天前的布林肯，基本上他们所说不建议这么做，但现在又说如果他们知道其中的一些内容，不希望此事被压制。

**主持人：**有意思。你正在推动这一进程，以解密关于大流行病来源的情报，你已经告诉我们你为什这么做。听起来你相信你会成功地实现这一目标，情况是这样吗？
