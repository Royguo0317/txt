medRxiv发表的观察研究发现，如果以更高剂量分发给患有严重疾病的呼吸机的患者，抗疟疾药物羟氯喹和锌可以提高新冠病毒患者存活率近200%。

研究的结论指出：“我们发现，当HCQ和AZM这两种药物的累计剂量高于一定水平时，患者的存活率是其他患者的2.9倍。”

该研究补充说：“通过使用因果分析和考虑体重调整累积剂量，我们证明，&gt;3克HCQ和&gt;1克AZM的联合疗法大大提高了新冠病毒患者的IMV存活率，HCQ累积剂量&gt;80毫克/kg的工作效果要好得多。这些数据尚未适用于不在IMV上的住院患者。由于高剂量HCQ的人的AZM剂量较高，我们不能将因果效应完全归因于HCQ/AZM联合疗法。然而，AZM可能确实对存活率的提高做出了重大贡献。”

这个结果是由新泽西州的圣巴纳巴斯医疗中心对255名患者进行研究后得出的。
