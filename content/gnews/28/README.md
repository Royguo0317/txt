## 國際銳評丨重磅到疲憊，不曾偷王毅突提不應另起爐灶

作者：喜馬拉雅聯盟加拿大農場 亞倫

據路透7月9日報導，中共國務委員兼外長王毅稱，中美關係正面臨建交以來最嚴重的挑戰，中美關係不應另起爐灶、不能強行脫鉤（怕了、要跪），應繼往開來與時俱進（屁話、相信你就又瘸了）。

**小知識補充：**

*另起爐灶是中華人民共和國建國初期的三大外交政策之一（其他兩個是一邊倒和打掃乾淨屋子再請客），指的是不承認原有中華民國時期建立的一切外交關係，而是在新的基礎上與其他國家建立新的平等的外交關係。 （資料來源維基百科）*

中共國務委員王毅在中美智庫媒體視頻論壇所提到的“中美關係”指的是1972年基辛格訪華之後，基辛格主義與中共元老勾兌的骯髒爐灶，是兩國壟斷嗜血權貴地下交易平台。既不代表中國普通人民的心聲，也不符合美國人民的利益。

那麼問題來了，什麼叫不應另起爐灶，美國準備不用中共這個假大空騙的灶，準備用哪個真善美的灶。一語點破，即是說美國不和中共交朋友了，但準備和可以代替中共政權的新國家交朋友。

而目前的華人世界中，唯一有資格和美國掛鉤的、代替中共政權的、又能真正代表人民利益的、能為中國人民主持公義的只有剛剛在封港紐約自由女神前建國的新中國聯邦，王毅知道，美國是要棄掉“七宗罪”的中共邪魔犯罪組織政權，要和新中國聯邦掛鉤、起爐灶了。

之前不管是蘋果店裡賣斷卡（禮品充值卡）的Gcoin還是網頁版被大規模退款的Gdollar，表面上看著是投資避險，其實是人心所向，是被文貴先生一次次用真實爆料以及驗證所砸醒的正確選擇。並且之前文貴先生也在視頻中說過，新中國聯邦有能力接盤50%-70%的美國國債，說明新中國聯邦不僅有大量可靠情報的信譽，也有雄厚的資金實力，不僅僅是口炮黨，還是走在最前沿的行動派。

關於這個脫鉤美國已經說的很清楚了，不是跟中國人脫鉤，而是跟中共國脫鉤！剛剛美國參眾兩院無異議通過的代表美國民意《香港自治法案》，就是全面與中共脫鉤的法案。 （詳細請參考《國際銳評：雷霆之鎚終砸下, 眾議院全票通過制裁和中共關聯銀行，港幣金融的脫鉤大幕》）

可笑的是，也就是在一年多以前，委員王毅在一次新聞中心舉行記者會上大言不慚的說，“與中國脫鉤，就是與機遇脫鉤，就是與未來脫鉤，某種意義上也是與世界脫鉤。” 現在王毅又著急的溫柔中帶了點委屈的出來說千萬不能和中共國脫鉤，因為據《路德社》爆料透露，川普總統有可能在《香港自治法案》正式生效前，出台更為嚴格的製裁行政命令，為了顯示自己更反共，因為《香港自治法案》是民主黨眾議院搞出來的。

再次聲明，美國政府從未說過要和中國人民脫鉤，不和中國人民合作來往，而是不和中共有關係的人做生意和有經濟金融的來往。中國人民很勤勞又刻苦，新中國的未來一定前途光明，中國市場必然成為真正的巨大市場。

新中國聯邦的經濟實力、國際信譽真正獲得了西方認可，而王毅現在拙劣地放低姿態地假服軟、真欺騙，可見美國金融脫鉤的打擊是致命的。如今加拿大、澳大利亞已經正確的選擇迅速終止和香港的引渡條例，這是司法上的脫鉤，一切都在一步一步的叫殺中。
