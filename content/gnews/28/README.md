## 歷史回顧｜臺美斷交帶來的庶民情感
`2020-10-09 10:09 公民之聲` [轉載自GNews](https://gnews.org/zh-hant/413226/)

內新聞：Y.M.O 素材：霹靂年/Y.M.O 校對：雅典娜的聖鬥士(沙加)
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/09090647/09a8e88e-0a2e-4ca0-bf9e-7098a314b9f9_cx0_cy13_cw0_w1023_r1_s.jpg)
臺灣國民黨立法院黨團10月6日在立法院提出“美臺復交”與“政府應請美國協助抵抗中共”兩個討論案，全院無異議通過。這應該是壹條讓很多人都有點吃驚的新聞，至少胡編的標題是這樣表達的——《“臺美復交”“協防臺灣”，國民黨提的！？》。這感覺就像國民黨背叛了中共壹樣，甚至綠營方面多少也會有些吃驚。當然國民黨提出這兩個議案的動機我們不作討論，筆者最感興趣的是蘇貞昌院長在去立法院做施政報告並接受質詢前接受媒體聯訪時說的話。

蘇貞昌表示，臺灣會跟美國從有外交關系到沒有外交關系，都是國民黨搞的，而現在國民黨終於良心發現，也是好事壹樁，但這要好好努力，也希望對外大家團結壹致，也希望國民黨能壹改過去沒有用臺灣最大利益來考量。

蘇貞昌院長的壹句話就把臺美之間這幾十年的關系給說清楚了。臺美復交的進度如何，這個筆者無法預測。但這句話讓筆者想起了當年臺美斷交前後島內的庶民情感以及現在看來很像是詛咒的壹段歷史。

70年代的臺灣發生了壹連串劇烈動蕩事件，被迫退出聯合國、邦交國相繼離棄、蔣介石逝世、美麗島事件，當然包括臺美斷交。這些變動壹個接壹個震撼著島內的民眾，上層怎麽想的我們不知道，但對於庶民來說會產生兩種情感。第壹是他們夾在中美之間，盡管雙方都說自己是正義的，但受傷的還是這片島嶼，所以會對雙方都不信任。這也影響到了現在發生的壹些事情，比如對於CCP病毒防控的成功，還有在美國肉類制品進口問題上的爭論都有這樣的庶民情感因素在裏面。第二也因為雙方都不能信任，所以可以相信的只有自己。當時的青年們只要稍微有些意識都不可能置身於這樣的氣氛之外，他們漸漸走出了60年代的虛無，開始關註腳下的土地，掀起了尋根的熱潮，臺灣的文化藝術在華語世界壹直處於領先地位的源頭也就在這個年代。

當時美國決定和北京建交，與臺灣斷交。應該說有很多民眾感覺是灰暗的，心裏很不舒服，甚至會有壹定的自我懷疑。不同領域的人在這樣的背景下會拿出屬於他們的回應，侯德健在這樣的情緒下就寫出了《龍的傳人》這首歌。如果聽過侯德健在中廣錄音室自己唱的demo，其實它本來是壹首偏哀怨的小調，也因為是受到這樣強烈的情感刺激，所以這首歌和侯德健其它作品相比完成度其實沒有那麽高。大家熟悉的李建復《龍的傳人》中的版本是陳誌遠的編曲把它變出了壹種輝煌的感覺，這是讓侯德健比較意外的，當然這是音樂上的部分。

非常諷刺的是，《龍的傳人》是因為臺美斷交的怨氣所寫，而現在卻被CCP經常拿來作為在各大晚會上歌頌所謂民族自豪感的曲目，當然詞曲作者侯德健的名字是不會打出來的，讓小花朵們知道就糟糕了。侯德健後來出走大陸，被臺灣官方稱為“叛逃”，在六四事件之後又被共產黨驅逐出境，在每壹個歷史關口都做了被當局譴責的選擇，不由得表示嘆息，龍的傳人更多地成為了對侯德健的壹種詛咒。

這裏讓我們站在歷史看未來的話，就像滅共之於終結CCP病毒，臺美的斷交如果是壹個循環的開始，臺美的復交會不會成為這個循環的終結？還是借蘇貞昌院長的話來說吧，“美國是在世界上最能為臺灣發聲，也最有能力為臺灣打開國際空間做努力協助的朋友，我們會繼續努力，增強跟美國的關系，關系改善也要壹步壹步來。”
