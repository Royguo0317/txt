**西班牙2021年2月2日电/西喜社——**西班牙政府表示，在检测到来自巴西和南非的中共病毒变种后，正在加强对这些国家航班的限制。

政府发言人玛丽亚·赫苏斯·蒙特罗说，从周三开始的未来两周内，只有搭载西班牙国民或在西班牙的外国居民的航班才被允许降落。这项措施还允许位于西班牙和法国之间的小公国安道尔的国民或居民入境。

前往欧洲免护照申根区以外国家的过境旅客将被允许在西班牙停留最多24小时，但不得离开机场过境区。货物、医疗或人道主义航班也将被允许。

西班牙已经与英国有类似的空中和海上旅行限制。
