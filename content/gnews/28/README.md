**西班牙2021年5月31日电/西喜社——**
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/31193846/image0-1-128.jpg)
中国周一宣布，将允许夫妻生下三个孩子（目前的上限是两个孩子），以扭转新生婴儿数量的迅速下降趋势。

以下是学者，经济学家和公众的回应：

张新宇，30岁的旅游者，来自郑州的一个孩子的母亲

“在大多数情况下，主要由女性承担抚养子女的责任。这个社会并没有为妇女提供太多支持。如果男性可以在抚养孩子方面承担更多义务，或者家人可以更多地关心刚有了孩子的女性，实际上很多女人都能生第二个孩子，因为如果孩子有兄弟姐妹会更好。但是考虑到全局，实际上，我不想生第二个孩子。生三孩更不可能。”

甘玉阳，三十岁的旅游者，一个孩子的父母

“如今，年轻人不得不买房。这种压力已经很大。然后，您必须考虑（孩子的）教育费用，等等。我认为这项政策将难以实施。”

JEAN-PIERRE CABESTAN，香港浸信大学教授

“除非政府实行真正的激励措施，比如，为生三个孩子的夫妇提供特殊津贴，减免交通费用和其他激励措施，我认为未来几年中国夫妇不会生更多孩子。”

张志伟，定点资产管理首席经济学家

“直接影响可能是积极的，但在宏观层面上影响不大。长期影响取决于政府能否成功降低抚养儿童的成本，特别是教育和住房费用。”

周皓，商业银行高级经济学家

“如果放宽生育政策是有效的，那么目前的两胎政策也应被证明是有效的。但是，谁想生三个孩子呢？年轻人最多可以生两个孩子。根本问题是生活成本太高，生活压力太大。”

丁双，标准特许首席经济学家

“毫无疑问，这是朝正确方向迈出的一步，但仍然有些怯懦。政府可以全面放开生育政策。即使实行了全面放开政策，出生率也可能不会大幅提高，因此没有必要继续采取限制政策。完全放开的生育政策应该至少在五年前实施，但现在为时已晚，尽管迟到总比不到好。

李怡斐，上海纽约大学，社会学家

“我觉得该提议未能认识到生育率下降的原因……人们并不受制于两个孩子的限制，而是因为当今中国抚养孩子的高昂费用而受到阻碍。住房、课外活动、食物、旅行和其他所有东西费用迅速上涨。有效的政策本应提供更多的社会支持和福利；在我看来，提高育儿限额本身不太可能以有意义的方式让人们产生心理倾斜。

“挑战是多方面的，因此需要在多个政策领域采取认真协调的行动来重建人们对未来的信心……期望公民如此机械地响应政策变化是不明智的。”

刘烨，伦敦国王学院 社会学家

“在我看来，这是一场数字游戏。实际上，有三个主要障碍影响家庭（尤其是妇女）无法生育更多孩子），针对这些障碍仍然缺乏具体的政策建议。

“三大障碍是育儿成本，对育龄和育儿妇女的就业歧视以及各个行业和私人托儿服务提供者都缺乏保障儿童福利的能力。”

徐红才，中国政策科学学会经济政策委员会副主任

“这是迈出的好的和重要的一步，但可能难以扭转出生率下降的趋势。我们应该制定其他政策来鼓励生育。生育率仅为1.3，这太低了，这是一个危险的信号……我们坚持渐进改革，在保持稳定的同时不断进步。

“每个家庭可以生三个孩子是可以的，这是可以接受的。有些农村家庭可能希望生更多的孩子（如果我们完全取消控制的话）：那也可能带来问题。”

SIMONE TAGLIAPIETRA，驻布鲁塞尔的布鲁格尔智库研究员

Tagliapietra说，中国温室气体排放有四个主要驱动因素：人口规模，人均GDP，GDP的能源强度和能源消耗的排放强度。如果增加一个驱动因素，则需要更大幅度地减少其他驱动因素，以控制排放。

Tagliapietra说:“随着中国放松其一胎化政策，需要通过降低某些或全部其他条款来实现减排。”

“核心问题是第三和第四因素是否可以以足够快的速度下降，以使第一和第二因素保持不变或者甚至增长。”
