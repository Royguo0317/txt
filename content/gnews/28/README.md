## 【鹰眼墙国】康复医疗“再”推进，初始阶段困局难解

环球网6月17日转载央视新闻客户端消息，中共卫健委等八部门印发《关于加快推进康复医疗工作发展的意见》，大力推动康复医疗中心和康复医师队伍建设。

2020国内65岁及以上人口数量达1.9亿人，按照《意见》中每10万人配备康复医师6人的标准，仅老人群体就需要11400名康复医师，14亿人口基数需要8.4万人，而2019年执业（助力）医师数量14661人，其中执业医师12657人，不及需求的二成。《关于加强北京市康复医疗服务体系建设的指导意见》指出2020年每千常住人口0.5张康复护理床位。而2019年康复护理床位数为271695张（同比18年增长10.3%），但每千常住人口0.194张，缺口依旧巨大。

2010年中共卫生部出台《关于将部分医疗康复项目纳入基本医疗保障范围的通知》陆续将医疗康复项目纳入医保。但直到2016年，康复项目数量才从9个增至29项，且主要以治疗各种残疾为主（如肢体残疾、言语残疾、听力残疾）。2021年，山西省出台政策，将医养机构纳入医疗保险范围，并逐步增加医疗康复项目；广东省将脑瘫、自闭症、心理疾病等28个康复项目纳入医保范围。但是，康复医院依旧是以盈利性质的私营为主（2019年私营康复医院占比48.87%，国营占比22.81%），医保报销的比例较低，仍需要病人支付高价药品和高昂的住院费用。因此，康复医疗依旧停留在初始阶段。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/17223152/Copy-of-16x9-feature-photos-MOS-logos-19-1.png)图片来源：网络
到2023年，国内康复医疗产业规模有望破千亿，年复合增长率高于18%。但是，2018年后，康复医院一直呈现收不抵支的现状：2019年国内康复医院总收入134.68医院，总支出134.83医院，净利润亏损-1.9%。再者，国内康复器具行业虽然种类众多，但缺乏创新能力：2017年高端康复器具市场占有率不足10%，大部分康复器具是山寨或仿制国外过期专利。康复器具自主创新性差，市场集中率较低，工艺质量水平不高。即便屡屡推进康复医疗进程，但无创新、硬件设施匮乏、人力物力缺口巨大，发展也只能寥寥。
