## 【世事解评】商道：CCP特色画皮下的商业监管体制
` 纽约香草山农场` [轉載自GNews](https://gnews.org/zh-hans/689924/)

作者：香草山写作组   如水 校对/发稿：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/29231938/logo-3-1-1.jpg)



商业是一种有组织的提供顾客所需的商品与服务的行为，现在已成为一种经济活动，以货币为交换媒介来实现商品流通。商品流通是指商品或服务从生产领域向消费领域转移的过程，它的本质是交换，是基于人们对价值的理解的等效交换。
![]()![](https://gnews.org/wp-content/uploads/2020/12/12-11.jpg)
也就是说一个商品或服务，生产出来要经过一定的流通环节，最后到消费者手中，然后产品或服务被消费者消费，而源头厂家和各个环节都得到了货币回报，这算是一个完整的过程，正常的流通体系。在这个流通过程中，产品的包装、定价、质量和渠道是否畅通至关重要。一个健康的流通体系，是为消费者所认可接受的包装，合理的定价，合格的质量及正常顺畅的销售渠道。当然也包括流通中的各个促销手段，促销是促进销售的一种模式，根据产品、厂家、商家不同而形式各异。

在这个体系里，质量可以说是重中之重，“质量是企业之本，质量是企业是生命”，质量是决定产品能否长久立足于市场的关键。同时信誉也很重要，信誉是建立企业形象的关键。所以企业对质量和信誉的态度或者文化也就很重要了。要用严谨、认真、科学的态度来对待每一个环节，形成一个规范的市场经济秩序。

但是独裁专制的CCP完全破坏了这个经济秩序，金钱至上、物质至上、假大空治国，造就了一个浮夸、虚伪、权力至上、关系万能的商业体系。有权有势的统治阶级，投机取巧，空手套白狼，人为操纵房市、股市，控制企业命脉，肆无忌惮地收割百姓血汗。他们利用职权无底限地将土地、银行等各种国有资产轻易占有、随意侵蚀。

这些统治者为了他们的既得利益，并不去真正关心产品质量、国民生计健康等问题。整个社会就出现了这种假大空、泡沫满天飞的虚假经济，带着华丽包装的虚假产品充斥着社会的每一个角落。

CCP的畸形经济模式，导致整个社会没有真正的监督管理体制。众所周知的各类质量体系认证、产品检验检疫报告、信用等级认证以及各类单位及产品荣誉认证证书等就是这种监督管理体制的表现形式。

但是在CCP的监督管理体制下，企业可以不需要经过真正检测，便能拿到所有的合格证书，拿到优质产品顶尖技术奖。各类信用证书、各类企业荣誉、各类品牌建设、各类诚信资质等，大都是官商勾结的衍生物，目的只是为了欺骗合作者，蒙蔽消费者，为了盘剥利益，收割民众资产。

这种体制严重地伤害了消费者。整个中国，雾霾、水霾、土壤霾、有毒产品遍地都是。盗版侵权、传销和各种类型的商业欺诈屡禁不止。虚假广告满天飞，制售假冒伪劣和有毒有害产品的案件频繁发生。毒奶粉、假疫苗、地沟油、苏丹红等各种有毒有害食品从来没有中断过。

曾经风靡一时的三鹿奶粉，其企业通过了ISO9001、ISO14001认证、GMP审核和HACCP认证，获国家实验室认可证书、国家认定企业技术中心称号。“三鹿”商标被认定为“中国驰名商标”；产品一举夺得国家科学技术进步奖。中国中央电视台新闻频道的《每周质量报告》节目及多个国家机关联合为其产品质量做背书。在这耀眼的光环下，是他们肆无忌惮的在奶粉中添加三聚氰胺这种低廉的化工原料，致使几十万婴幼儿出现严重的泌尿系统问题、脑部积水问题、痴傻问题，身体严重衰弱，甚至有死亡事件。

这一切的根源都是这个独裁的专制导致的，只有废除这个独裁专制的体制，消灭万恶的CCP，才能改变。新中国联邦任重而道远，商业体系、市场经济秩序的整顿和规范，关系到人民群众的切身利益，影响着经济是否健康运行。改变CCP特色的邪恶体制，需要标本兼治，努力推动市场经济秩序持续好转，促进公平竞争，净化消费环境，还国人一个真正健康有序的商业体系市场运营机制。
