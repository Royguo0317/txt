*中国石化营销有限公司交易概述*

*公司背景*

*▪ 中石化是中共国最大的综合性能源和化工公司之一，在中共国设有上游，中游和下游的业务。*

*▪ 中国石化的主要业务包括：油气勘探与生产，提炼，销售和分销， 以及石化和化学产品。*

*▪ 中国石化营销有限公司（以下简称“ SMC”）是中国石化的全资子公司，目前拥有30,351个加油站，并拥有超过20,000家快捷便利店。 SMC的加油站主要分布在沿海和内陆各大城市的重要位置。*

*▪ 2013年，SMC的销售收入为人民币14,986亿元，净利润为人民币251亿元， 总资产3314亿元，净资产558亿元。 2013年，成品油销量达到1.78亿吨，国内市场份额约为60％。 SMC在成品油销售的市场份额方面排名第一，具有高利润率和强劲的现金流。*

*中石化注资的理由*

*▪ 中石化于2014年6月30日宣布，SMC重组后，将通过注资将外部的社会和私人资本引入SMC。*

*▪ 国有企业传统上从石油业务中获得了很大的收益，此次注资为中国石化营销公司（SMC）提供了机会，使其业务多元化，并将其业务转向非石油业务。*

*▪ 中国石化营销公司（SMC）能够利用其广泛的销售网络，再借助混合所有制，这会使它的非石油业务有更广阔的快速发展机会，这将带来更高的利润率。*

*中国石化营销有限公司交易概述*

*招标过程*

*▪ 增资过程包括多轮招标和竞争性谈判。*

*▪ 第一轮的重点是投资者的选择和鉴别，中国石化和中国石化营销公司（SMC）会根据投资资格和背景确定潜在的投资者。 在这一轮中，有超过100家机构投资者参与了招标过程，申购承诺总额超过3000亿元人民币。 经过独立评估委员会的审查和筛选后，中国石化和SMC选中了37家感兴趣的投资者。 而一些知名的全球私募股权基金，例如科尔伯格•克拉维斯•罗伯茨（KKR）、德州太平洋集团（TPG）、新加坡政府投资公司（GIC），未能通过本轮融资。*

*▪ 第二轮的重点是谈判和完成交易。 中石化和SMC与有兴趣的投资者进行了有条件的交易谈判。 交易各方完成了与注资有关的内部批准程序和相关的外部批准程序。 在最后一轮中，成功地吸引了37个投资者中的25个。*

*▪ 确定投资者和他们的投资额度的方法要权衡一些因素，包括潜在投资者的要约价格和他们的提议投资额、潜在投资者与SMC的互补优势、是否有可能成为SMC的未来业务合作伙伴、潜在投资者的注册地点（即国内或海外）、潜在投资者给中国公众带来的利益、 潜在投资者的行业地位，品牌形象和声誉以及财务实力、潜在投资者对SMC股权的预期持有期、潜在投资者的现有业务或重点投资领域不得与中国石化的主营业务发生重大冲突等。*

*重要事实和数据*

*▪ 2014年9月12日，BHR和其他投资者一起与SMC达成注资协议，总认购额为1,070.94亿元人民币，以收购SMC的29.99％的股权。 根据协议，BHR以人民币60亿元的价格收购了1.68％的股权。 该认购金额在SMC所有投资者中排名第七。*

*▪ SMC的成品油销售增长保持在3-7％左右，与此同时，2013年市盈率（P/E）的预期估值为9.96倍。 假设2014年净利润增长15％，2014年的货币后市盈率是12.37倍。*

*▪ 该项目的预期内部收益率约为11％-18％。*

*▪ SMC将保证不低于50％的派息率，并且根据投资估值，股息收益率将达到4％。 管理层预计SMC将在3年内在香港证券交易所上市。*
