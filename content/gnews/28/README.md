《路德时评》2021年3月15日早间节目主要内容：

一、中共对亚太联盟成立感到恐慌

《环球网》3月15日发表一篇题为《社评：美国很想把日韩进一步筹码化》的文章[1]，文中说美国国务卿布林肯和国防部长奥斯汀今天抵达日本，随后还要去韩国，分别与两国举行2+2面对面会谈。美方非常高调，强调这两场会谈对强化美国盟友体系的重要意义，奥斯汀声称，美国的目标是要确保拥有能力及行动计划，“对中国或任何想与美国较量者做出可信的威慑”。

路德社评论：

1、中共对社论向来都非常重视，这需要最高级别单位审核通过后才可以在官方媒体平台上发表。该文称美日和美韩会谈是美中会谈的预备会议，这完全是中共在自我意淫，中美会谈的真相是中方强烈要求与美方展开谈话，所以才特意追到阿拉斯加，可见中共已经对中美未来脱钩感到恐慌。

2、杨洁篪与蓬佩奥在2020年6月会见时就谈及有关病毒真相问题，中共至今没有就此问题做出官方回应，所以此次会谈一定还会谈及中共病毒真相。虽然中共利用世界生物界大佬为病毒自然来源学说站台，但是在爆料革命的影响下，美国政府并没有轻易相信病毒自然来源学说，他们对病毒真相仍旧不依不饶，中共很清楚病毒来源问题已经捂不住了。

3、中共看到美国即将联合日韩两国后感到极度不安，中共国对外扩张的势力被极度压缩，美国从来都没有将日韩当作筹码，这三个国家是盟友关系。美国并不是非常重视此次中美对话，与中共国脱钩的战略方针早都已经定好且不可能更改。中共国当年与苏联断交，苏联计划灭掉中共政权，但是中共输送各种利益保住政权，后来才有中共国与美国建交联合拖垮前苏联。

4、这篇文章提到美国推动建立亚太联盟就是要让中共国的邻邦国家形成联盟，与中共国脱钩，这再次验证爆料革命路德社情报来源的准确，中共国也已经获得情报。中美一旦正式脱钩，中共国周边国家将承接美国等西方发达国家的生产订单，亚太联盟国绝对都会赚得盆满钵满，这才是中共国恐惧的原因。

5、中共国此时还天真地以为通过交易就能扭转现在的国际局势，西方国家需要的是建立共同秩序，在同一个规则下互相合作。中共无论与前苏联联合，还是加入WTO始终都是规则的破坏者。中共当年在前苏联的支持下打赢了国民党，夺走了中国统治权，后来又忽悠前苏联帮助中共国发展经济经济建设，后来又忽悠美国，自从爆料革命横空出世后，中共已经不可能再继续忽悠西方国家。

二、潘石屹之子被北京警方通缉

中共极左媒体西陆网2月25日发文《潘石屹儿子诋毁英烈至今无人管！背后到底有什》[2]，随后3月15日，《人民日报》直接发文《北京警方:网民潘某造谣诋毁英雄烈士，对其刑拘追逃》[3]，文中说道，一新浪微博用户曾于2020年6月23日在他人微博评论区发表造谣诋毁英雄烈士的言论，造成恶劣社会影响。经警方调查核实，该微博用户潘某（男，30岁）于2020年2月2日离京出境后，一直在境外。目前，海淀公安分局已依法对潘某刑事拘留，并对其开展追逃。在此，警方敦促其尽快回国接受调查处理。

路德社评论：

1、潘石屹之子在去年6月23日发布诋毁英雄烈士的微博，9个月后由《人民日报》撰文对潘石屹之子进行追捕。潘石屹肯定在期间通过各种运作无果后，才安排他儿子在二月份离开大陆，中共此举明摆着就是对潘石屹及其背后的王岐山下手。
