## **无题**

山亦悠悠，
水亦悠悠，
俯瞰今世阅风流。
谁人于命叙春秋？
正道无休，
灭共无休，
唯真不破一声吼。
问君天下谁风流？

## 元旦

泪洗庚子满春秋，
血染山河志未酬。
北雁南飞醒望眼，
大千未来寄辛丑。

## 彼岸花

大江东去浪淘沙，
风雨掠过看戎花。
历尽艰辛寻益处，
流亡半世走天涯。
