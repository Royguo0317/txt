## 西班牙执行“疫苗护照”，这无异于再次走在悬崖的钢丝上
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1306249/)

**撰稿：脐橙君**
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/07195927/27-3.jpg)
自从比利时今日宣布启用疫苗护照之后，西班牙却用实际行动执行了。

西班牙是世界第二大旅游目的地。经历了严峻的新冠疫情之后，西班牙从6月7日起向全球已接种新冠病毒疫苗的人士开放边界，希望借此振兴受疫情重创的旅游业。 西班牙2020年的游客量相较于2019年，跌了77%。

根据法新社报道，不仅对于已经接种疫苗的欧洲人，西班牙大开国境，甚至那些还未接种疫苗的欧洲人，只要通过快速抗原检测，也能入境西班牙。

据悉，西班牙全国各地的酒店和餐馆已经陆续重开，航空公司也恢复疫情高峰时期停飞的航线。同时，西班牙也会从6月7日开始，允许游轮停靠港口。但尽管如此，离当地政府之前定下的年内吸引4500万名旅客的目标还相距甚远，截至4月底，当地的旅客人次只达到180万。

而且针对疫情重灾区的英国，西班牙方面也宣布为接种过疫苗的英国游客开辟“绿色走廊”。这意味着已接种疫苗的英国游客无须等待核酸检测结果就可自由入境。

文贵先生一直说，病毒的黑暗远远没有到达顶峰。而且闫博士也一直重申：拒绝疫苗护照。病毒的黑暗没有到达顶峰，是因为中共还逍遥法外，越是被国际压力挤压得透不过气，越是为了自保，要去攻击别国，只有他国乱，才能换来他们苟且的一线生机。因此，中共绝不会停止向世界继续投毒的计划，如果西班牙大开国境，正好为中共投毒创造了一个完美的掩护。之前印度的疫情爆发，中共外宣就把疫情和当地宗教和个人卫生联系在一起，混淆他们故意投毒的事实。

而闫博士说拒绝疫苗护照，并非拒绝有效的疫苗。而是，作为深谙这次病毒真相的她来说，如果不取得最原始的毒株，就无法开发出有效的疫苗，因为这场病毒不是自然来源，而是一起生物战，疫苗又怎能应对一种武器呢？

西班牙的做法不仅对国民不负责，而且会引起他国的效仿，毕竟疫情之下，所有国家的经济都走到崩溃边缘，一边是病毒威胁，一边是经济崩溃，怎么选都不会有好结果，而唯一的正解就像闫博士所说，除掉那个共产主义政权，因为中共才是这场病毒的始作俑者。
