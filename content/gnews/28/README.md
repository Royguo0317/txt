## 中共國漁船再度到南美侵略和破壞當地海洋資源
`2020-07-28 19:20 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/278575/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/28191736/Picture-1-179.png)
圖片來源: [The Maritime Executive](https://www.maritime-executive.com/article/chinese-fishing-vessel-rams-korean-coast-guard-boat%29)

據衛報(The Guardian)7月27日報導，位於南美洲的厄瓜多爾海軍在距加拉帕戈斯群島(Galápagos Islands)約200英里的群島上發現了一支龐大的插滿中共國五星旗的捕魚船隊。

**加拉帕戈斯群島蘊藏著豐富的漁業和旅游資源**

加拉帕戈斯群島於1978年被宣佈為聯合國教科文組織世界遺產，它不僅以啟發了查爾斯·達爾文(Charles Darwin)的進化論而聞名，也是以全球鯊魚種類最豐富之處的鯊魚天堂而聞名，其海洋保護區擁有世界上最多的鯊魚種類，其中包括瀕臨滅絕的鯨魚和鎚頭魚。厄瓜多爾總統列寧·莫雷諾(Lenin Moreno)稱該群島為“最富饒的捕魚區之一，是整個星球的生命之床”。加拉帕戈斯群島也以其獨特的植物和野生動植物而聞名。聯合國教科文組織將這個每年吸引25萬遊客的群島描述為“活生生的博物館和進化論的展示台”。

**大規模中共國捕魚船出現在南美海域引發厄瓜多爾擔心**

厄瓜多爾前環境部長約蘭達·卡卡巴德斯（Yolanda Kakabadse）表示，目前約有多達260艘船出現在離該島周圍188英里寬專屬經濟區外的國際水域中。非政府組織藍色星球協會（Blue Planet Society）的發言人約翰·休斯頓（Joh Hourston）說：“我們感到非常震驚，龐大的中共國工業捕魚船隊現在正在加拉帕戈斯群島附近。”龐大的捕魚船隊使人們對更擔心脆弱的海洋生態系統會被嚴重破壞。卡卡巴德斯告訴《衛報》：“大規模的捕魚船隊和對海洋物種的掠奪會對加拉帕戈斯群島的物種平衡構成了巨大威脅。”據悉中共國大規模捕魚船隊每年都會駛入加拉帕戈斯群島附近的海洋，而今年的船隊規模是近些年來最大之一。

**厄瓜多爾密切監視中共船隊行動，在必要時採取措施保衛海洋權益**

卡卡巴德斯和基多市前市長羅克·塞維利亞（Roque Sevilla）週一被任命為位於南美大陸以西563英里的諸島來設計“保護戰略”。塞維利亞說，我們將作出外交努力，要求中共國漁船撤出該區域。他說：“中共國在保護區邊緣的無節制的捕魚活動已經嚴重破壞了厄瓜多爾在保護加拉帕戈斯群島海洋生物的所有努力。”他繼續補充說，該小組還將尋求執行保護遷徙物種的國際協議。

卡卡巴德斯還表示，政府將繼續努力把專屬經濟區擴大到各島周圍350英里，與厄瓜多爾大陸的經濟區結合起來，從而封閉中共國船隊目前在兩者之間的國際水域走廊。厄瓜多爾還試圖在太平洋的鄰國哥斯達黎加、巴拿馬和哥倫比亞之間建立一個海洋保護區走廊來把這片重要的海洋多樣性地區封鎖保護起來。

厄瓜多爾國防部長奧斯瓦爾多·哈林(Oswaldo Jarrín),表示，自上週發現該船隊以來，厄瓜多爾海軍一直對其進行監視。他說：“我們處於戒備狀態，正在進行巡邏，絕對要避免發生諸如2017年那樣的事件。” 2017年事件，厄瓜多爾海軍在加拉帕戈斯群島海洋保護區內扣押了一艘中共國船隻- -福遠漁冷999(Fu Yuan Yu Leng 999)號，該船上被發現攜帶300噸海洋野生生物，其中大部分是鯊魚。

**譯評** :

中共不僅用中共冠狀病毒殘害世界人民，還為了滿足高級官員宴客的魚翅料理，派了260多艘的漁船來南美洲加拉帕戈斯群島海洋保護區意圖盜捕幾千條以上的鯊魚，造成海洋生態圈的浩劫。在中共的統治下，不僅國內人民沒有幸福、自由，全球人民的生命、自由受到威脅，全球自然生態更是收到嚴重破壞。萬惡的根源就是中共，唯有消滅中共，全球各國的自由、人權才能受到保障，自然生態發展的永續經營才有可能。
