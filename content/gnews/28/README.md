## 閆麗夢博士受訪英國著名脫口秀將公佈證明病毒來源的科學證據

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/12115121/Snipaste_2020-09-12_23-50-28.png)图片来源： Loose Women
來自世界頂級傳染病研究中心的冠狀病毒專家閆麗夢博士於2020年9月11日接受英國著名脫口秀節目《無拘無束的女人》（Loose Women）採訪。

閆博士告訴主持人：

- 她“是一名醫生和博士”,是最早研究冠狀病毒的科學家之一，“與世界各地的許多頂尖專家一起工作，擁有中國大陸的兩個學位”，於2019年12月“被指派對武漢新肺炎進行秘密調查”；
- 通過她在中國醫學界和科學界的人脈，她發現了病毒人傳人的證據,看清中共正在史詩級地掩蓋真相並歪曲病毒來源；
- 她所在的香港大學實驗室負責人要求她對真相保持沉默，同時她受到人身威脅；
- 隨著死亡人數的上升，她“看到沒有人對此做出回應…人們害怕政府…這是一種危險的病毒…人類和全球健康都處於（危險中）”，她覺得有道義和科學責任將真相公佈於世，她聯繫了“一位在美國的著名的中國油管節目主持人”，然後逃到美國;
- 在她逃離香港前，中共已從政府數據庫裡“把她所有的信息抹去”，僱傭網軍“散佈有關她的謠言”，稱她是“撒謊者…啥也不懂…在實驗室裡養倉鼠的”,“他們（中共）試圖控制”她的“家人和朋友”，她“突然不復存在…在中共統治下，對中國人而言，這（中共手段）是常識”
- 她正在和在美國的頂級科學家團隊準備報告，將於幾天內發布，用科學證據證明：“這個病毒不源於自然界”,病毒基於中共軍方擁有的舟山蝙蝠病毒，“它(病毒)來自武漢實驗室,他們為什麼製造它”，來源海鮮市場的說法只是個“煙幕彈”;
- “病毒基因組序列就像人類的指紋…基於它你可以識別這些東西（病毒）”
- “任何人，即使你沒有生物學知識，也能讀懂它（證據)，並自己檢查，識別和驗證”;
- “了解它(病毒來源）至關重要，我們無法克服它(病毒)，這將危及每個人的生命”。


《無拘無束的女人》（Loose Women）是與BBC齊名的英國ITV出台的收視率極高的脫口秀節目，由四位家喻戶曉的女主持人組成一個採訪小組，成員包括著名記者Jane Moore和英國足球前國腳蘭帕德（Lampard）妻子，專訪名流,主題包羅萬象涵蓋政治，娛樂，八卦，日常生活等，至今已有3000多集。此次專輯邀請全世界21位風雲女性，閆博士為第一位嘉賓震撼登場，被主持人稱為“頂級科學家”，“世界勵志女性”, “在國際上已掀起了巨浪”。閆博士受邀採訪說明了她的身份和可信度已被世界主流媒體認可，病毒真相成為普通民眾關注的話題，歐洲滅共的第一槍已在英倫打響。繼《無拘無束的女人》（Loose Women）採訪後，世界各大媒體爭相報導，包括美國的紐約郵報（New York Post）和英國的每日郵報（Daily Mail），世界正在被喚醒。病毒真相公佈之日，將是全球點燃滅共導火索之時。
