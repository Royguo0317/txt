**作者：Ansel Lewis**

牆內高考臨近，日前山東濟甯女子苟晶發帖稱兩次被頂替上大學事件發酵，這挑戰人類良心底線高考冒名頂替在微信圈持續火爆，對于這種事件CCP宣傳策略不是壓制而是進行輿論引導，讓輿論矛頭對准高考冒名頂替的違法操作人員，而不是社會主義體制，不是CCP，起到輿論引導的主要是各類官媒，如人民日報惺惺作態發表評論：嚴查“冒名頂替”真相，告慰“被偷走的人生”。文中要把CCP政府的權威體現出來，“隨著有關部門介入，事件距離水落石出又近了一步。”要老百姓知道CCP政府是好的，而且還要恐慌一下挑戰體制的聲音：“如果有人借機炒作，也應承擔法律責任。”
