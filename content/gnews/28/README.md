## 【独立观点】拜登禁止美国实体投资59家中共国企业

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/01040128/s2-7.jpeg)



6月3日（周四），拜登总统签署了一项行政令，禁止美国实体投资59家据称与国防或监视技术行业有关联的中共国企业。拜登政府表示，此举扩大了川普时期出台的一项有法律缺陷的行政令涵盖的范围。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/06044924/%E5%9B%BE%E7%89%871-9.png)图片来源： RFI

中共国外交部周五回应称，中方敦促美方尊重市场规律和原则，撤销打压中共国企业的各种所谓“清单”。但据路透社引述美国一位官员表示，“我们完全预计，在未来几个月……，我们将把更多中共国企业纳入新行政命令的限制清单中”。

显而易见，拜登政府仍然是延续川普时期的反制中共国战略，不仅如此，拜登政府还在继续推进。中共国和习近平原来预计，拜登当选总统后，美国与中共国的紧张局势会很快得到缓和，还会放松对中共国的制裁。现在看来，中共国和习近平的希望已经落空了。

并且，从中共国外交部汪文斌的发言来分析，中共国对美国政府的反制战略，没有什么实质性的解决办法，只能口头抗议一下。

总统选举期间，拜登与中共关系匪浅的传言甚广。拜登为了证明自己，在反共的事情上，必须表明立场。尤其是现在的政坛，反制中共已经成为两党的共识，以及国会山与五角大楼的主流，拜登不敢也不能放松对中共的反制。可以说，拜登为了自己的政治安全，对中共只有割席这条路可走。

如白宫国安会印太协调员坎贝尔（Kurt Campbell）上月所说：与中共国接触的时期已经结束，今后双边关系的主导模式将是竞争。美国政府，从五角大楼到国会山再到白宫，已经达成广泛共识，在全面调整对中共国的战略。对中共的绞索，已经开始收紧了。
