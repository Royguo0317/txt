## 浅谈简化汉字之恶

撰稿：澳喜反抗军

汉字，是世界上最古老的文字之一，是中华民族的灵魂与象征，也是炎黄子孙之间相互联系的桥梁和纽带。它由象形文字演变成兼表音义的意音文字，具有集形象、声音和辞义于一体的特征。因此，它在各类文字中是独一无二的，对世界文化的发展影响极其深远。

关于汉字的起源，有结绳说、八卦说、刻契说等等。而民间流传最广的，则是仓颉造字说，即所谓“神农尝草，仓颉造字”是也。关于该造字者的历史，又有许多种说法：汉代的《说文解字》与《淮南子》等书中记载，仓颉是黄帝时期主管造字的左史官；而同时期的《河图玉版》与《禅通记》则记载，仓颉曾经自立为仓帝，又曾于洛汭之水拜受洛书。传说中，仓颉是双瞳四目，也就是共有四只眼睛，每只眼睛各有两个瞳孔。而中国历史上所记载的重瞳者，总共则只有八人。

据《淮南子.本经训》记载：“昔者仓颉作书而天雨粟，鬼夜哭”。意思是说，仓颉见灵龟负图，上书丹甲青文，于是他上仰天文、下俯地理，根据大地山川和鸟羽兽印而创造出了文字。所以天上竟然下粟如雨，而夜里却听见野鬼在不停的嚎哭。

为什么会这样呢？就是因为仓颉造出的文字，可以让刚刚从蒙昧走向开化的古人们用来传达心意和记载事情。因此人们都欣喜若狂的竞相学习，这自然非常值得庆贺。但是此时的天帝却担心人们只顾习字而忘了种田，一旦误了农时，田园就会荒芜，于是下了一场谷子雨来提醒人们，这就叫做“天雨粟”；而为什么又会有“鬼夜哭”呢？这就是因为仓颉所创的文字含有造化天机，里面蕴藏着大量的能量与信息。如果人民一旦掌握了文字，于是民智就会被开启，那些鬼怪和世间的坏人们假若再想着继续愚弄老百姓就有没那么容易了。所以它们纷纷感到惊恐不安，只好在夜里鬼哭狼嚎了。

根据现有的考古发现证明，神州大地的先祖们早在七、八千年以前就开始在龟甲上刻画符号了。河南舞阳贾湖遗址中出土的带有契刻符号的龟甲经碳14检测，年代距今为7762年（±128年）；公元前5000年至前3000年的仰韶文化遗址、以及公元前3500年至前2240年的山东泰安大汶口文化遗址中，发现在陶器上刻画的符号已经达数十种之多。

由此可以看出，汉字的起源应当是原始的图画，而自古以来中国就有“书画同源”一说。唐代张彦远在他的《历代名画记.叙画之源流》中说道：“颉有四目，仰观垂象。因俪鸟龟之迹，遂定书字之形，造化不能藏其秘，故天雨粟；灵怪不能遁其形，故鬼夜哭。是时也，书画同体而未分，象制肇始而犹略。无以传其意，故有书；无以见其形，故有画。”他的著作，就对汉字书画同源做出了合情合理的解释。

到了公元前16世纪的商朝，华夏文明已经发展到了相当高的水平，其主要特征之一就是甲骨文的出现；之后随着青铜器的大量铸造，因为先秦时期称铜为“金”，所以这些刻在青铜器上的文字就被称做“金文”或者“钟鼎文”。换句话说，甲骨文是商代书写的俗体，而金文才是官方使用的正体。此时，汉字就开始显示出了“正体多繁、俗体趋简”的印记。

到了西周后期，金文演变为“大篆”。就是把金文中粗细不匀的线条变得均匀柔和，字形结构也趋向整齐，由此而奠定了方块字的基础；之后秦朝的丞相李斯又对当时的文字进行收集整理，删繁就简美化加工而形成了“小篆”。此时的民间继而把小篆体变圆为方、改曲为直，于是又出现了“秦隶”。

汉代由于社会文明的进一步发展，汉字在民间出现了蚕头燕尾的波折之笔，书写起来轻松自如，于是就形成了非常优美的“汉隶”；之后草写的隶书演变出了“章草”，到了汉末又演变出了笔画连绵回绕、书写简约方便、字与字之间有连缀的“今草”；而后到了唐朝，从今草继而再演变出了能抒发书者胸臆、寄情于笔端的“狂草”。

汉代以降，三国时期由魏国的著名政治家和书法家钟繇以及晋代的王羲之、王献之父子，又糅合了隶书和草书而创造出了“真书”。后人因为以这种字体作为学习书法的楷模，所以改称真书为“楷书”；而介于楷书和草书之间的字体就是“行书”。因此，汉字的演变基本上就是从甲骨文到金文、大篆、小篆、隶书、草书、楷书，再到行书这样一个过程。

因此可以看出，汉字是集形、声、义于一体的相统一的文字符号。迄今为止，它是世界上唯一没有中断的表意文字，也是世界上使用时间最久的文字。汉字是一种文化、是一类科学，也是一门艺术，它的出现对世界文明的推动和发展起到了难以估量的作用。

但不幸的是，自中共于1949年篡权以后，就开始不遗余力的对中华传统文化进行灭绝性的打击。1951年春，匪首毛泽东就明确提出了“文字必须改革，要走世界文字共同的拼音方向”的主张，以便于向苏联看齐。1956年，在大陆讨论文字改革的时候，毛又称“汉字是一种落后的字体，必须要改革成像拉丁文那样”。当时中共的主子苏联也认为汉字是落后的语言文字、是残余的象形文字，而拼音文字才是科学先进的文字。
