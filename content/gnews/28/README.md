制造生化武器荼毒全人類，處心積慮偷襲台灣，隨著病毒真相的揭開和3F美國計劃的暴露，全球滅共力量已經齊聚，RICO制裁中共法案將實施，死神無人機已瞄准中南坑，四面楚歌之下迷信風水的中共在國內加速了西部大開發步伐，妄想西部是其逃生之門，殊不知西部是其下地獄之門。

中共國的政策制定都冠冕堂皇地說是爲了民生爲了國家複興，政策的真實原因也許從維護中共血旗不倒和對外共産擴張角度能更好地解釋。像1994 年的分稅制改革表面是中央財政集權，真實目的是便利幾大惡魔家族從財政上節制地方掌控整個中共國；2003年國務院政策文件提出房地産業成爲國民經濟的支柱産業表面是發展房地産經濟，真實目的是推高房價“以房弱民以房貧民”；2015年抛出的“中共制造2025”計劃和2018年的“中共標准2035”表面上是爲了國家崛起，真實目的是取代美國稱霸全球；等等。2020年5月中共國又推出《關于新時代推進西部大開發形成新格局的指導意見》聲稱，西部大開發工作對促進區域協調發展，決勝全面建成小康社會，開啓國家新征程具有重要意義。其實，早在2000年西部地區開發領導小組就成立並運作，當時說是爲了提高西部地區經濟和社會發展水平，鞏固國防。20年了西部也沒發展起來，如今又重啓西部大開發，中共的真實目的是什麽？

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/30084543/2017-2.jpg)

在奇門遁甲中中共政權生門的方位在西部，加上中共竊國之前在陝甘甯一帶苟延殘喘獲得生機，迷信風水的中共惡魔對西部情有獨鍾，因爲在他們的觀念中西部是保持他們血旗不倒的地方，是其生門所在，西部大開發表面上是爲了協調東西部發展全面建成小康社會，真實目的是爲了保持中共血旗不倒。不僅如此，就連中共國的軍事門戶網站也起名“西陸網”，在世界文化中中國是東方國家而中共國的軍事門戶網站起名爲“西陸網”令很多人感覺到滑稽，其真正原因就在于中共惡魔迷信西部是其生門所在，加上一幫江湖術士推波助瀾，而中共獲得血旗不倒的法寶是槍杆子，因此黨衛軍網站必須冠以“西”字開頭。當然宣傳上要冠冕堂皇地引用幾句古詩來诠釋“西陸”的來曆。

奇門遁甲作爲前人類文明留下的天文地理學博大精深，其顯示的中共政權的“生門”方位在“西”，這裏的“西”其實是指西方國家，並不是指中共國領土範圍內的西部。1949年中共竊取政權是因爲西方國家對蘇聯妥協的結果，8964之後中共苟延殘喘也是西方國家對中共惡魔綏靖的結果。改革開放之後中共國成爲制造業大國對內極權加重對外擴張推行共産國際，其技術資金都來自西方發達國家，防火牆的技術來自美歐，國內監控技術也來自美歐，軍事技術幾乎都是通過軍民融合剽竊美國技術，而且中共惡魔壯大力量的資金很大一部分來自華爾街，是西方的技術和資金滋養了中共惡魔，西方世界是中共惡魔生存的決定因素，中共政權的合法性取決于西方世界的認同，其生門在西方國家。

中共惡魔釋放病毒荼毒西方世界，在經濟外循環快被切斷的同時啓動西部大開發想借助西部大開發續命，認爲國內西部是其生門所在，妄想把西部打造成堅固的堡壘就可以抗衡整個世界，就可以保持血旗不倒並傳承千秋萬代。自5月份中共提出推進西部大開發以來，國內的政策重點轉移到了西部，各種資源也加快了從東部沿海向西部轉移。東部沿海面臨著外企大舉撤出，又面臨著國內制造業向西部轉移，中共設想的是東部升級打造創新高端制造業，西部承接中低端制造業，殊不知外企的撤出把高端制造業技術都帶走了，再加上美國對偷竊技術的中共國企業的制裁，中共國東部沿海企業還將徘徊在中低端制造業。如今，中共國要以計劃之手把中低端制造業轉移到西部以穩固血旗，其結果肯定是西部大興土木破壞河山，東部沿海人去樓空，再加上人口銳減，城市墓群死寂一片，整個國民經濟陷入大蕭條。
