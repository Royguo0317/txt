近日，笔者的一位战友叙述说，某银行电话专员通知她该行信用卡已逾期，请尽快还款。这位战友辩驳说：“我每月都还最低还款额呀，利息你们也收了，为啥让我全额还款？现在疫情期间，放假在家，单位开资都困难，哪有钱全额还款呀”。进过几轮交流，对方就一句话：“您的信用卡已逾期，请尽快还款”。

用过墙内银行信用卡的朋友都知道最低还款额这个政策。信用卡一共有三种还款方式，分别为全额还款，分期还款和最低还款。最低还款是指持卡人到了还款日之后无法还清全部欠款，从而选择先还清部分欠款的行为，最低还款额一般为消费金额的10%以及其它各类应付款项，具体计算方式为：最低还款额=信用额度内消费款的10%+预借现金交易款的100%+前期最低还款额未还部分的100%+超过信用额度消费款的100%+费用和利息的100%。

信用卡最低还款额的方式虽然可以避免发生逾期，但选择偿还最低还款额没有免息期，账单内的所有消费款项都会从入账日开始计算利息，按照0.05%的日[利率](https://jin.baidu.com/article/3593835.html)来计算复利，有些银行卡用户以省下的90%钱继续可以买买买，而且银行还贴心提示您，按最低还款额还款不会留下个人信用不良记录，也不会影响未来的房贷，这些用户以为可以薅到银行的羊毛，殊不知用信用卡最低还款额的方式虽然可以避免发生逾期，但选择偿还最低还款额没有免息期，账单内的所有消费款项都会从入账日开始计算利息，按照0.05%的日利率来计算复利，所以选择最低还款额的方式是需要一定成本的。这种成本其实是非常高的，仔细算算相当于每年18%的利息，几乎是住房贷款的4倍。这也是从墙内银行卡诞生以来一直伴生的怪胎（刮骨钢刀），激发韭菜欲望的同时收割韭菜的利益没商量。

那么新年伊始，为啥墙内有银行会放弃这么大的利益出尔反尔废弃该项最低还款额政策，催促客户全额还款呢？我想战友们都应该知道爆料革命已进入关键时刻，墙内CCP统治下的经济、金融崩溃在即，不管不顾利息收入，以急急收回本金为重中之重。
