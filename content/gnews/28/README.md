据《华盛顿观察家报》引用NBC新闻网报道，川普政府国家安全顾问罗伯特·奥布莱恩在上周举行的全球安全论坛的访问中讲到如果拜登获胜，国家安全委员会将会努力使政权的交接平稳过渡。他同时强调乔·拜登和贺锦丽“很显然”是赢得了大选。

他还指出 “即使是最具争议的时期我们都可以和平顺利的移交权力，前总统乔治·布什与戈尔角逐总统时，亦是到12月中旬才开始权力交接，最后也如期完成。如果川普不能连任，或是另外的结果，我相信白宫都会希望看到一个专业的政权交接。”

奥布莱恩的此番言论与竞选团队和白宫的立场截然不同。尽管左媒一边倒地“预测”拜登胜选，但川普律师团队一直在致力于揭露选票舞弊，并在多个州获得了不可胜数的确凿证据，因而，竞选结果还没有最后出来。而这次的大选闹剧，不仅仅在于明目张胆手法多样的选票投票造假，还在于左媒步调一致的造势，包括大选结果还未出来之时，迫不及待地宣布（预测）拜登胜选，面对这些明显违背司法程序的举动，作为国家安全顾问的奥布莱恩却没有提出质疑，反而推波助澜，在媒体公开谈论“政权交接”话题，算不算知法犯法呢？

去年九月奥布莱恩接替博尔顿成为川普政府的第四位国家安全顾问，管理五角大楼，国务院和美国情报机构的数百名专家，负责为总统搜集情报并提出建议。众所周知，奥布莱恩还是律师出身，他曾任加州国家律师事务所Arent Fox LLP执行合伙人，也是洛杉矶律师事务所Larson O ‘Brien LLP 的创始人之一。

他曾经指出中共国的意识形态对美国的民主是巨大威胁，也支持川普总统对新冠病毒来源的调查，而他本人也在几个月前感染新冠病毒。今年8月，他也曾表示，为了干预美国2020总统大选，与中共国政府有关联的黑客已将攻击目标瞄准美国选举基础设施。他一直被认为是川普坚定的支持者。

在此次46分钟的访谈中，除了中东问题，奥布莱恩也再次谴责了中共政权在香港问题上对法律和人权的破坏以及华为5G网络对国土安全的威胁。访谈最后两分钟他提到了以上关于权力交接的言论。但无论是川普本人，还是国务卿蓬佩奥和白宫贸易顾问纳瓦罗均表示，对舞弊得出的竞选结果拒不承认。副总统彭斯更是表示，川普政府已经做好进入第二任期的准备。
