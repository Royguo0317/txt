根據該視頻——2020年8月27日15時許上海半木軒公司就要求哺乳期員工手寫心得發表公開致歉信，公開信稱，郵件內容屬實，確實是由公司人事部人員發出，公司稱當事人朱女士最終決定與公司解除勞動合同，公司將後續的補償金、賠償金和其他訴求委派專人與她商量、安排，信中稱，已與上海市勞動保障監察機構、工會組織婦聯等機構進行了溝通，將提交相應材料。新京報記者從朱女士同事處獲悉，事情起因是公司單方面降薪且克扣生育津貼，同事稱朱女士與公司協商未果後，申請了勞動仲裁，於是發生了後續，同事稱，朱女士是單位工作6年的老員工，此前擔任銷售部文職工作，8月26日晚當事人朱女士首次針對此事發文回應——“感謝大家的關心和關注，大家的關懷是支持我的最大力量。很多朋友關心，後續情況，給大家一個回復，就不一一作答了：25日確實撤銷了一份仲裁，共涉及兩項訴求：1、補齊單向降薪工資差額；2、被齊生育津貼差額。公司已足額發放了降薪工資差額及生育津貼差額，故撤銷了相關仲裁。從6月產假結束回歸工作後，不是特別順利，也遇到了很多困難，特別是心得郵件收到之後非常痛苦，精神壓力很大，目前狀態也不是很好，今後在公司繼續工作已不現實，所以今天已和公司就解約問題進行溝通，目前在家休息。”

整個事件從24日半木軒發出所謂“申明”——
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/24211413/95eef01f3a292df561ed8df6691f716735a873d8.jpeg)
再到8月25日半木軒宣傳部門表示朱女士已經回原崗位，再到8月27日下午半木軒的公開道歉，其實驗證了前文[上海半木軒傢俱的人力資源部為何甘心做狗把哺乳期員工逼到發瘋](https://gnews.org/zh-hans/315306/)的種種猜測，人力資源部用最惡毒的方式基本把哺乳期的朱女士逼到半瘋，想要把人逼走結果變成了網路熱點，在形成公共危機的情況下，半木軒在疲於應對之下，到了現在的結局：半木軒把克扣、拖欠以及單向降薪差額補齊，同時根據勞動法等相關法律協商賠償額，半木軒也出了公開道歉信。

這個結局算不算圓滿？半木軒的“申明”和“道歉”改變不了其血汗工廠的形象，因為法律和合同一直都在，視法律和合同為兒戲的作法一直都在，只是之前沒有踢到鐵板，所以人力資源部這一次依然會象瘋狗一樣對提出異議的哺乳期員工設置種種難以想像的刁難，但這一次踢到鐵板了，於是作為資方的半木軒軟了下來，半木軒還能不能走出這場公共危機？誰也無法回答，但是因這一起事件的負面影響是永遠抹不掉的。

相信這份要求手寫心得體會的郵件也會成為這個時代的一個重要標記——在中共國的白領們曾經被同為打工者的人力資源部同事如此惡毒的欺壓和虐待。
