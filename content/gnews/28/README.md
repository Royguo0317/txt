## 报告：脸书吹哨人泄漏文件揭露该公司审查“疫苗犹豫”内容

[!\[\]()!\[\](https://spark.adobe.com/page/pPZyvPWeVNWkQ/images/f0bde7db-4696-4564-8a16-172fab93e86f.jpg?asset_id=a921aad4-1102-4c0a-b568-bb53a34ab9a9&amp;img_etag=%22c35b904a2b3a49e7246dd3f8a2c7c5cd%22&amp;size=1024)](https://spark.adobe.com/page/pPZyvPWeVNWkQ/images/f0bde7db-4696-4564-8a16-172fab93e86f.jpg?asset_id=a921aad4-1102-4c0a-b568-bb53a34ab9a9&amp;img_etag=%22c35b904a2b3a49e7246dd3f8a2c7c5cd%22&amp;size=1024)

**两名脸书“内部人士”向《真相工程》（Project Veritas）敲响了警钟**

来自脸书的吹哨人称，这家科技巨头正在审查其平台上发表疫苗犹豫的内容。

来自《真相工程》的新报告显示了内部泄漏的备忘录，**其中解释了“疫苗犹豫内容降级”，这表明该公司的“目标”是“极大减少用户接触疫苗犹豫的内容”。**

另一个泄露的文件解释了“边界疫苗框架”，该框架深度分析了如何将该内容与另一个“目标”分类。另一个“目标”是“确定并分层可能在某些情况下不利于疫苗接种从而导致疫苗犹豫或拒绝接种的非违规内容的类别。”该文件补充称，“我们已经根据潜在危害以及评估危害所需的内容大小对它们进行了分层。”

一名脸书“内部人士”告诉《真相工程》创始人詹姆斯·奥基夫（James O’Keefe）：“脸书在其算法中使用分类器来确定某些内容……他们把这些内容叫“疫苗犹豫”。而且，在用户不知情的情况下，他们把这些内容进行了评分，即‘ VH分数’，‘疫苗犹豫分数’。然后根据该分数将降低或保留该内容，这取决于里面的具体内容。”

奥基夫描述此内部人士为脸书的“数据中心技术员”。此人透露，这家科技巨头正在对其38亿用户中的1.5％进行“测试”，重点关注“权威健康页面”的评论部分。

此人告诉奥基夫：“他们甚至试图在这些内容出现在你的界面之前，在你还没有看到之前就控制它。”
