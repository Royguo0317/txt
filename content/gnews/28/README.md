【想嫁兵哥哥的援鄂女護士身份造假】近日，關於“援鄂護士”于鑫慧訂婚喜訊的多篇報導在各大平臺刷屏。在這些報導中，新聞主角于鑫慧為一位江蘇南通的“95後”護士，在疫情最嚴重時，她瞞著父母，獨自一人從南通輾轉前往武漢，成為了一位醫務志願者，在康復隔離點駐守了55天。然而，就在網友們紛紛送上祝福時，卻有人質疑稱“於鑫慧並非醫療工作者”，當初她在媒體上自稱“護士”的身份疑似造假。對此，南通市衛生健康委員會宣傳處負責人告訴紅星新聞記者：“她不是醫療工作者，也不是醫療衛生系統的。她不是公立醫院的(工作者)，也不是民營醫院的。她也沒有護士資格證。”

【報導中的於鑫慧】曾在採訪中自稱護士 獲“南通五四青年獎章”等獎勵

據媒體報導，于鑫慧於2月19日或20日抵達武漢，隨後在武漢某康復驛站工作，是該康復驛站第一位前來報到的“醫護人員”。對於其身份，有媒體表述為“南通一家醫院泌尿外科的護士”，也有媒體表述為“南通一家民營醫院的護士”。 報導稱，於鑫慧被安排在武昌區里安隔離點醫療組進行志願服務，負責火神山、雷神山兩家醫院出院患者14天隔離期的護理工作。她的工作日常是早中晚定點6次查房，量體溫，詢問和登記病人的情況，並配合醫生採集核酸檢測的咽拭子。在援助湖北期間，“除了用自己的醫療專業知識來幫助康復點病人外，她還從南通帶去了一百張獎狀準備‘頒發’給患者，她說這是為了安撫康復者的情緒。”

▲媒體相關報導。截圖自網路

媒體稱，4月15日，馳援武漢56天的於鑫慧回到了江蘇南通，南通洋口鎮疫情防控指揮部的工作人員帶著鮮花來迎接她，於鑫慧告訴媒體記者：“作為一名醫護人員，我做了我想做的事，我用盡了我的全力，用到了我全部的經驗和所學，我沒有任何遺憾。”

同時，她告訴前來採訪的媒體：“因為我在醫院上過班嘛，知道住院病人的心理其實是很壓抑的。而且當時覺得患者多是中老年人，他們本身體力就不如年輕人，出來之後還要去隔離點住上近半個月，心理肯定會特別壓抑，頒發獎狀就是為了讓他們開心。” 紅星新聞記者看到，網路上有許多於鑫慧援助湖北期間身穿防護服在康復點工作的照片。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/18010416/3-21.jpg)![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/18010439/4-14.jpg)
▲網傳於鑫慧疫情期間在武漢工作時的照片。圖據網路

據媒體報導，此後，於鑫慧獲得了許多榮譽：2019年如東縣優秀志願者;2020年3月入選江蘇好人榜;2020年南通五四青年獎章獲得者;2020年江蘇最美人物江蘇最美青年抗疫先鋒獲得者;2020年7月，被評為“中國網事·感動2020”二季度網路感動人物;2020年7月，被評為2019年度宣傳推選學雷鋒志願服務“四個100”先進典型暨百名疫情防控最美志願者。

【南通市衛健委回應】於鑫慧沒有護士資格證 也沒有在衛生系統工作過 在於鑫慧的婚訊發佈後，有一些人對其醫護人員的身份及個人作風等問題提出了質疑。同時，在中國裁判文書網上，紅星新聞記者檢索到，2019年12月26日，於鑫慧因民間借貸糾紛，被江蘇省如皋市人民法院納入失信被執行人名單並限制其高消費。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/18010504/5-16.jpg)
▲相關民間借貸糾紛執行裁定書。截圖自中國裁判文書網

一位元認識於鑫慧的網友告訴紅星新聞，她從未聽於鑫慧在過去提起過自己護士的身份，“以前她一直說自己是哲學系的，要不是因為這次疫情，我根本不知道她的護士身份”。

這名網友向紅星新聞記者提供了一張疑似於鑫慧抖音頁面的截圖，畫面為曬出一本復旦大學的學生證。該網友說，在媒體的報導中，於鑫慧的學歷是南通衛校畢業，“我覺得這與她曾經表述的學習經歷不一致，於是就去問她。”

根據該網友提供的疑似兩人的對話顯示，於鑫慧給出的解釋為：2014年南通衛校畢業，2018年大學畢業，因為大學不是學的醫學專業，所以在媒體報導中沒有寫最高學歷。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/18010811/6-14.jpg)
▲該網友與疑似於鑫慧的對話。受訪人供圖

紅星新聞與於鑫慧本人取得了聯繫，對於目前網路上的質疑以及衛健委官方部門對其醫護人員身份的否認，於鑫慧以短信回復：“公道自在人心，清者自清，網上誇大的那些事情，有侵權，有誹謗，有造謠，對我造成的影響，對比(此)我必定會追究其法律責任!”隨後，對於記者的進一步追問，於鑫慧不再回應。

對於于鑫慧的醫護人員身份，南通市衛生健康委員會宣傳處負責人告訴紅星新聞記者：“她不是醫療工作者，也不是醫療衛生系統的。”紅星新聞記者追問，于鑫慧“不是醫療衛生系統”工作者，那是否有可能在私立民營醫院工作，該負責人再次進行了否認：“她不是公立醫院的(工作者)，也不是民營醫院的，據我所知，她也沒有護士資格證。”

隨後，江蘇南通如東縣衛健委辦公室一位工作人員也向紅星新聞記者表示：“她不是我們體制內的工作者，她也沒有在如東工作，關於她的事情都是在她去了武漢之後我們才知道的。”
