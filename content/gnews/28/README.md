2020年美国的总统大选，以其在美国历史上最为严重的舞弊行为震惊了世界，其深度、广度和疯狂的程度可谓世界之最。舞弊的手段各式各样。首先是发现选票数据出现了异常波动，由正常的平缓的增长突然变成跳跃的增长；进而发现了各种非法选票：死人选票、超时选票、多次重复的选票、冒名顶替的选票、电脑修改的选票……。随着调查的深入，更发现点票机是由中共和其他国家提供的机器，这种机器经过专门设计，使操作者可随时修改程序以作弊；在点票场地还出现了明目张胆的干涉，甚至暴力阻拦合法监票人员进入现场的行为。舞弊现象涉及多州，参与部门主要是民主党控制的政府机构、邮政系统、高科技企业和金融企业等，参与人员主要也是民主党成员，还有个别政府要员也深陷其中；美国的主流媒体也积极配合掩盖丑闻打压真相。川普团队的律师朱利安尼在调查后确认，这是一次有预谋、有组织、系统性的且协调一致的破坏大选的行动。目前大选还未结束，川普团队正在奋力争取，我们喜马拉雅的战友们也与美国爱国者们一起，正在积极努力地揭露事实真相和中共的野心，以期唤醒美国民众。我相信正义必将胜利，川普必将赢得大选，正如七哥所言，拯救美国这场大选危机的必将是爆料革命。

在一直为热爱民主法治自由的人们所羡慕，并被视为人类的理想之地的美国，民主法治自由却被如此大规模地破坏和践踏，这还是第一次。事情发展到今天的程度并非偶然。从现在往回看，才知道从四十年前美中建交的那一刻起，就已经预示了今天的这个局面。从基辛格们到拜登们，四十年来美国政客已经从对中共的谨慎试探发展到同舟共济，然而美国社会却没有一点反应。他们怎么能隐藏这么多年呢？这四十多年里就没有引起过别人的注意吗？不！不是没人注意，而是因为他们已经形成了多个政府部门、多行业、多领域、甚至连续几届总统都参与其中的与共产国际相勾结的庞大的卖国集团。

中共从毛泽东开始就定下了一个野心勃勃的目标——复兴中华（欺骗中国人民），当世界领袖。虽然已经换了几代头目，但目标没有改变，计划一直在执行中。从中国对世界上其他主要国家和联合国重要部门的“蓝金黄”程度看，就剩下美国还没有被中共完全掌控，而这次对美国总统大选的干涉、控制就是最后的决战。如果不是七哥发起了爆料革命，美国就将变成中共的附庸国，全世界人民就都将变成中共的奴隶。

中共在世界各国进行“蓝金黄”的时间久、程度深、面积广，且屡屡得手，尤其是美国都几近沦陷。到底是哪里出了问题，根子在哪里？这个问题引人深思。

我认为有两方面的原因：一是人性中普遍存在自私、贪婪、懒惰、恐惧的弱点，而从历史的经验来看，这些弱点单靠个人的力量是无法克服的（能做到的极少）；二是制度的设计，即法律法规的设计，这里是可以多动脑筋进行大力改革的领域。美国的先贤们已经意识到人性弱点的存在，所以设计了三权分立、相互制衡监督的制度，并在两百多年的实践中不断地修正完善，已经是当今最不坏的制度了，但仍然存在很多漏洞，经不住厚黑这一恶魔的攻击。虽然在法律法规方面已经有了非常详细严格的条款，在执行和监督方面却有一定的裁量度，而这个裁量度又是由具体的人来掌控的，这样一来监督过程就难免受到个人观念情感的带有倾向性的影响，所以对权力的监督就是最最重要的问题。当今世界各国都设有监督机构，但是监督机构的人员多少都和这个国家的经济利益有着不可分割的关联，甚至监督人员本身就是握有重权者，而在一个经济利益相互交错、相互依赖的团体内，要使掌握权力的人完全不腐败，做到真正的公平公正是很难或根本不可能的。

如果跳出这个圈子另辟蹊径，使监督者和被监督者完全脱离经济利益关系，或许会柳暗花明。这方面，七哥已经有了新的构想。在今年6月4日宣布的《新中国联邦宣言》（以下简称《宣言》）中就勾画出了大体的轮廓。

《宣言》开篇就明确定义了喜马拉雅是一个监督机构，这是一个颠覆了我们以往所有观念的全新的监督机构。下面摘录几段关于喜马拉雅监督机构的描述：

《宣言》序言：“在新中国联邦成立之际，喜马拉雅监督机构特此庄严地承诺：……本机构将为新政府的建立做好一切准备和外联工作……”

“喜马拉雅监督机构将会一直同国际合法监督机构一起监督新政府依法运行，同时遵守国际法和新政府法律，接受国际相关合法机构的严格监督。”

《宣言》附件（1）——“该机构建立于法制基金和法治社会基础之上，是一个由正式人员组成、受国际法保护的民间团体。它将永远作为第三方，架设一个没有中共的新中国联邦与世界实现永续和平、公开透明、合法守约与开放合作的平台和桥梁。”

“该机构作为奠基于海外的公益组织，接受由中国人民选出的新中国政府和美国政府共同达成的、包含惩罚机制在内的严格的法律约束，以保证其作为第三方民间机构对新中国的民主、自由和法治提供长久有效、透明、公正的监督。组织成员如违反规定，将受到美国、新政府、本机构和国际监督机构四方法律程序带来的最严厉的惩罚。”

从上面的描述可以清楚地看到，这个监督机构完全打破了世界上民主法治国家现有制度中的规则，具有几个明显的特点：（一）该监督机构是民间的有组织的第三方机构，而不是官方的监督机构；（二）是在国外注册的；（三）监督机构同时接受新中国联邦和美国政府以及国际相关法律的约束；（四）机构成员不只是中国人，还有外国人；（五）机构成员不参与新中国联邦的任何职务，也不从新中国联邦获得报酬；（六）从成立喜马拉雅以后，七哥就开始构建经济网络并快速实施了部分基础项目。从七哥描述的宏大蓝图和目前的运行情况看，未来喜马拉雅监督机构完全是一个独立完整的经济实体，不受制于任何一个国家和组织。这最后一点是最最重要的。

这样的监督机构可能还不被人们理解或接受，但是这样的设计断开了监督者与被监督者之间在权力方面的联系，即使监督机构中个人腐败，也不会扰乱被监督国家的秩序，也不会给被监督国家造成什么损失。

喜马拉雅监督机构的具体章程还没有公开，以上只是我自己的理解和一些粗浅的想法。相信在七哥的推动下，在战友们和未来新中国联邦公民的努力下，中国一定会是一个更先进、更文明、更富有、更美丽的国家。

（2020年11月20日）
