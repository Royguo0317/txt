## 談談5G標准與華爲、台積電的關系
`2020-05-24 21:40 GM30` [轉載自GNews](https://gnews.org/zh-hant/211837/)

**作者：文同**

路德有期節目談到有關美國商務部出台新政策，允許美國公司在美國以外的國家參與5G標准的制定，被定義爲是與華爲競爭有關5G標准的制定，似乎這樣的定義有商榷之處，主要鑒與以下幾點理由：

1、華爲通過低價競爭幾乎主導了世界5G標准的制定，所有接受華爲參與5G建設的國家也必須接受華爲主導的5G標准，如果允許美國公司到這些國家參與5G建設只有合作、沒有競爭。畢竟35%的價格優惠不可抗拒，不管美國政府如何勸戒，甚至以不分享情報信息、不賣F-35來表示強烈不滿，但始終不能警醒那些歐盟國家、包刮老牌同盟英國，他們的選擇還是要錢不要命。在此情況下美國公司是很難在這些國家參與競爭的；

2、深層次問題是華爲爲什麽能在世界上攻城略地，除了低價傾銷外，關鍵背後還有一個用高技術支撐著它的後台—台積電。5G技術缺不了10奈米以下半導體晶片，華爲5奈米、7奈米晶片的唯一來源就是台基電。缺了台積電的晶片華爲不可能制造5G基站設備和終端手機，有何能力去制定5G標准；

3、在美國公司被禁售晶片到華爲的一年多時間裏，台基電好幾次開記者會高調宣布：對華爲的供貨不受影響。那麽台積電如此囂張的後台又是誰？美國”deep state “那些長期與中共勾兌利益分享的、無恥、貪婪、毫無低線的暗黑力量。美國商務部一次又一次的延誤戰機，一開始是含美國技術25%禁售，後來是20%，到3月份開始死人了還10%呢，一直到5月份已死了超過8萬人了、被逼到牆腳沒有退路了，才宣布只要含美國技術就禁售。事實上美國商務部是可以一劍封喉的，因美國掌握了台積電3項關鍵性技術，缺了任何一項台積電就癱瘓。爲什麽讓華爲可以有一年多的空窗期瘋狂的購貨，屯積了台積電30%以上200億一300億美元的貨。讓我們扒開台積電的面紗看看它的內核裏都有什麽；

鳍式場效電晶體(FinFET – Fin Filed- effect transistor)

FinFET 是一種立體的場效應管，屬于多閘極電晶體。凡是制作10乃米以下(3nm、5nm、10nm)靠的就是這個鰭式場效電晶體。這個技術由美國加州伯克利分校胡正明教授領導的團隊和美國另外一家科技公司合作研發的。

**ASML半導體晶片光刻機**

ASMl光刻機由荷蘭廠商制造，全世界絕大多數半導體晶圓廠的光刻機都出于這家荷蘭廠商，包括台積電10乃米以下的和中國10乃米以上的晶圓廠光刻機。制造ASML光刻機的關鍵材料和技術也都在美國手中。這就是爲什麽中芯國際幾年前用2億美金預定的光刻機，即將交貨時被取消了。可見美國政府是有足夠的掌控力的；

EDA（電晶體設計自動化）

EDA是IC設計産業鏈必備工具。台積電要把幾億—幾百億個晶體電路集成在一小塊晶片上必須要用到美國的EDA技術，全球EDA主要由3家美企Synopsys, Cadence, 及Mento Graphics把持，3家合計在全球市場占有率超過60%，在中國的市場則集中度更高，高達95%以上。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/24212600/4-31-12.jpg)

以上三項美國技術只要斷供一項台積電就癱瘓，那怕只占台積電技術的1%。

台積電的股權結構

再扒一下台積電股權結構竟然發現台積電實質上是家美國公司。其第一大股董是—華旗託管台積電存託憑證專戶（美國的退休基金），占了台積電總股權的20.54%。第二大股董—台灣行政院國家發展基金管理會，只占股6.38%。剩下的小股董也有不少美國公司，算起來外資占了台積電70%以上的股權，美國絕對控股。面對美國大老板台積電應該是惟命是從的，爲什麽敢不聽話呢？顯而易見，必有妖孽躲在陰暗的角落相互配合。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/24212657/5-31-10.jpg)

現在中國所以能制造10乃米以上的半導體晶片，台積電也起了關鍵性作用，可以說爲中國的從無到有作出了重大貢獻。

“中國半導體之父”張汝京

張汝京原本就是台積電的股董兼副總經理。因不甘居董事長張忠謀之下，在中共重賞之下的張汝京又策反了一批台積電技術骨幹，並竊取了大量台積電秘密專利高科技知識産權，在上海高調創辦了中國第一個半導體晶圓廠“中芯國際”。上海是大盜國賊江家的地盤，在通過大量黑資本的支持下，“中芯國際“的晶片很快有了量産，並搶走了很多台積電的客戶。張忠謀氣不過有人動了他的奶酪，一紙訴狀把“中芯國際”告上了美國加州法院，一輪、二輪都判“中芯國際”敗訴而導至張汝京辭職、並判罰“中芯國際”10億美元賠償。讓人大跌眼鏡是台積電並沒有拿走這筆罰款，而是轉而交換了“中芯國際”的10%股份。人們這才明白台積電真正的目的是要進入中國14億奴隸市場，加入中共割韭菜權貴俱樂部。從此大量華爲海獅的晶片也有了來源—“中芯國際。

台積電現任董事長：劉德音

比較張汝京、現任台積電董事長劉德音更是有過之而無不及。2016年台積電不滿意僅有在“中芯國際的10%股份，幹脆高調與南京市政府簽約，在南京浦口區獨資建造台積電南京廠。這個投資30億美金，占地面積超過670畝，僅僅用了14個月就完成了建築工程，用了半年的時間完成了進機上線，2018年台積電南京廠就開始了量産。在量産典禮上，現場權貴冠蓋雲集(見圖)，劉德音在典禮上說：“從興建廠區，進機生産、再到量産等，持續創造奇迹。打破了3項台積電建廠記錄：第一建廠最快，第二上線最快，第三該廠爲最美、也最有特色的台積電廠區”。劉德音還表示：“中國的半導體業從曾經的落後到追趕，時至今日，相關企業已位列世界一流”。真可謂台積電爲中共的半導體晶圓産業立下了汗馬功勞和豐功偉績。

相對去年美國川普政府從國家安全考慮要求台積電去美國建廠，劉德音以成本太高爲由扭扭捏捏、推三阻四就是不肯去。爲此中共的大外宣發表文章：贊揚劉德音敢爲華爲兩肋插刀 ，頂住壓力繼續供貨並不去美國設廠，堪稱台商典範(見：我愛研發網)。這種赤裸裸的選擇與共匪合作，拿著美國人的錢、美國人的技術到中國去創造建廠奇迹，到美國建廠就有困難的虛僞說辭，只能說明他中了CCP 病毒：錢—就是唯一價值觀、錢—就是唯一領導，錢—就是靈魂的歸宿，錢—就是我的上帝，他中的毒已深入骨髓。

盡管現在台積電迫于不可抵抗的壓力，已宣布要去美國建廠並願意停止向華爲供貨，但還是繼續接受了7億美元的華爲定貨，因對華爲還有3個月的寬限期。實在是想不明白，爲什麽美國商務部一而再、再而三地給寬限期、沒完沒了？難道還嫌美國CCP 病毒感染的人數不夠多、死的人數不夠多嗎？
