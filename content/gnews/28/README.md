**概要：中共非正常权力交接有三种方式，一种是和平选举或者逼宫，一种是军事政变，一种是暗杀。由于中共内部和军队严密的监控体系，正常情况下发生的概率很小。但是中共目前面临内忧外患，反叛的人越来越多，非正常权力交接的可能性越来越大，以共灭共的脚步越来越近。**

最近几天路德社一直在分析习近平的病情和行踪，调动了整个新华社，引发了一张照片的血案，情报加分析，调侃加吓唬，着实让人感觉到了快乐灭共的乐趣，以共灭共的脚步越来越近了。

以共灭共的最好方式就是习近平先生自我革命，这种可能性现在看来几乎为零。其他无非三种方式，一种是和平选举或者逼宫，废除习近平这个独裁者，选出愿意解体中共、还权于民、与西方社会和平相处的领导人，一种就是军事政变，还有就是暗杀。

在中共篡政后的历史上有过一次最高领导人非正常和平选举过渡，就是华国锋被选下去由邓小平实际掌权。但是这次和平过渡最主要的原因是邓小平实际掌握军权，军队起了震慑作用。另外有一次逼宫未遂，就是文革前夕毛刘对决，刘要求召开中央全会想在组织上把毛选下去，但是作为中央全会法定召集人的总书记邓小平拒绝召集，致使刘的计划没有成功，其背后的原因也是因为此时毛已经指使林彪在北京周边部署军队，加上周恩来情报系统的配合，邓不敢轻举妄动。胡耀邦和赵紫阳的和平下台，其原因是他们只是名义上的最高领导人但实际最高领导人是掌握军权的邓小平。中共是靠枪杆子起家的，所以谁掌握和控制枪杆子谁就是实际最高领导人，所以我们需要分析一下中共对军队的控制方式。

中共宪法中虽然规定国家军事委员会领导全国武装力量，但宪法同时规定党领导一切。实际上中共还存在一个党的军事委员会，中共的武装力量由党领导，用中共的话语说就是**“党指挥枪”。**

早在1927年9月的三湾改编，中共就模仿苏联的军队体制，“**支部建在连上”**，在连和连以上的军事单位内建立党组织。连以上的各级军事单位除了军事主官外，还设立同样级别的党组织领导人，通常称为政治委员。政委是该军事组织中党组织的书记，而军事主官一般为副书记。据脱离中共的中共早期著名领导人张国焘在其回忆录中所说，中共军队的政委有否决军事主官决定的权力，任何军事命令如果没有政委的副署是无效的，军队中的人事任免主要由党委决定。中共通过在军队中设立党组织并拥有重要权力，实行军事主官和政治主官双首长制，实际控制了军队。

中共控制军队的另一个重要手段就是其鲜为人知的警卫系统。中共军队主要领导人的警卫甚至司机都是上一级统一安排的，最早的机构叫中央保卫局，后来是中央警卫局和总参保卫局。据张国焘的回忆录称，中共军队的重要命令除通过军事主官和政治主官两条线下达外，还秘密通过警卫系统下达，警卫员如果发觉主官违反了上级命令，有权对主官缴械并限制其自由。

中共建政后延续了战争时期党对军队的管控方式，并把党对军队的管控复制到武装警察、公安、国安和情治系统，各级武装警察和公安都设立党组织和政治委员，通过同样的方式控制警察力量。这样中共控制了全部武装力量。

中共历史上唯一成功的军事政变就是1976年毛去世后不久，华国锋联手叶剑英和汪东兴抓捕了以毛的遗孀江青为首的“四人帮”，其重要原因首先是汪东兴掌控中央警卫局，其次是叶剑英在军队中有广泛影响，而“四人帮”的成员都没有军队经验。**从此谁掌控军队和中央警卫局，谁才是真正的老大**。

1989年江泽民踏着六四的鲜血上位后最初的几年并没有真正掌权，江名义上是军委主席，实际军权掌握在邓和杨尚昆杨白冰兄弟手中。1992年中共十四大前夕，江的军师、红二代的核心人物曾庆红联手同为红二代的刘京、俞振声，和邓朴方建立了亲密联系，利用于永波的告密，以六四追责这个让邓小平非常敏感的事由，成功离间了邓和杨氏兄弟的关系，换上了邓的另一个亲信刘华清掌管军队。其后不久，江又利用远华案抓捕了刘华清的女儿刘超英，逼迫刘华清就范。当时邓年事已高，不久后去世，从此江实际掌管了军权，成为实际上的最高领导人。

2004年胡锦涛接任军委主席后，由于军队中的几名副主席都是江提拔的干将，中央警卫局则由江的亲信由喜贵掌控，所以胡当政十年没有掌握过军权，始终是傀儡。

2012年习近平登上大位，他利用拜登和美国的配合，先是抓捕了周永康、郭伯雄和徐才厚（病故），后又抓捕了房峰辉逼死了张阳，通过改组中央军委机构、军种和战区，提拔了自己的亲信；将武装警察部队从中央政法委指挥转变为中央军委指挥；任命自己的亲信为中央警卫局局长。十九大又将与自己家族有世交的张又侠提升为军委副主席，任命从福建一路跟随他的亲信王小洪为公安部常务副部长兼特勤局局长，掌控公安力量。习成功掌控了中共的军队、武装警察和公安等全部武装力量，成了新的独裁者。

从目前的情况看，军队中应该有很多将领是不满习的独裁的，江曾时期提拔的许多将领还有相当部分留在军队里。**但是由于中共军队的双首长制，还有警卫系统的暗线控制，此外肯定还有高科技情报特务系统的暗中监控，而且特务和情报控制有多条线**，房峰辉就是由于电话被窃听而事情败露被捕，军事将领稍有异动即可能遭撤换甚至被捕。中共军委有个规定，成建制调防一个排及以上的军队需要军委主席签字批准，调动军队很难不被发现。自1979年的越战之后中共军队多年未打仗，军事将领都是靠裙带关系和卖官鬻爵而提升，少有战场上打出来的将领的英勇豪气，几乎都是明哲保身之辈，虽不排除军事政变的发生，但正常情况下可能性不大。

中共历史上也有暗杀的事情发生。文革时期林彪的儿子林立果就曾经策划暗杀毛的“571工程”，但并未成功。林彪元帅的坠机据说也是因为飞机上安装了定时炸弹爆炸而坠机身亡，这是中共历史上最著名的暗杀疑云。林彪去世后邓小平复出，接邓进京的飞机也被查出安装有炸弹。皮定钧将军因为飞机失事去世，据说也是暗杀。朱德元帅的死亡很蹊跷，有传说是毛死前怕朱德不服江青上位而用药物将其暗杀（中共领导人的医疗保健部门也是很关键的，一般掌握于党魁手中，这也是江泽民目前常住上海的原因之一）。由于中共的历史不透明，众多疑云无法证实，虽然暗杀是有可能发生的，但由于现代高科技监控手段越来越发达，象荆轲刺秦王那样暗杀的难度越来越大。

以上分析的是在正常情况下的情形。但是目前中共面临以美国为首的西方国家的围剿，同时面临国内经济的崩塌，**内忧外患并存**，中共众多权贵家族利益受损，加上习王多年的选择性反腐树敌太多，中共各大佬家族对习不满情绪越来越大。最近又传习做脑部开瓢手术恢复不顺，多日不曾露面，局面诡异。如果想反叛的人越来越多，那么小概率事件连续多次不发生的概率就会很小，也就是发生的概率很大，概率论上称之为无限重复情形下的“**小概率事件必然发生**”。习防范100次，有一次没防范好就全完了。**天下苦共久焉，中共苦习久焉**，不管什么方式，**识时务者为俊杰，识正义者为豪杰**，我们和郭文贵先生一样希望中共内部能有堂堂七尺男儿登高一举实现和平过渡，顺应世界大势以共灭共，这样中国人民能少受很多痛苦，全球经济也避免大幅度衰退。相信万佛万神自有安排，郭文贵先生每次直播时和爆料革命战友的虔诚祈福必有回响，愿上天保佑我苦难的中国和中国人民，大家做好准备迎接没有中共的新中国。
