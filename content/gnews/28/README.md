## 【熱點播報】 白宮哈弗才女發言人CCP病毒檢測陽性，白宮持續陷落
`2020-10-05 22:26 CHANGDAO` [轉載自GNews](https://gnews.org/zh-hant/405111/)

作者：香草上農場寫作滅共霹靂組 macncheese



美國總統新聞秘書，即白宮發言人凱莉·麥肯尼（Kayleigh McEnany）在CCP生物武器超限戰攻擊下，成為又一位被感染的白宮官員。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/05220257/98-%E3%80%90%E7%83%AD%E7%82%B9%E6%92%AD%E6%8A%A5%E3%80%91%EF%BC%9A%E7%99%BD%E5%AE%AB%E5%93%88%E5%BC%97%E6%89%8D%E5%A5%B3%E5%8F%91%E8%A8%80%E4%BA%BACCP%E7%97%85%E6%AF%92%E6%A3%80%E6%B5%8B%E9%98%B3%E6%80%A7%EF%BC%8C%E7%99%BD%E5%AE%AB%E6%8C%81%E7%BB%AD%E9%99%B7%E8%90%BD-1.png)


2020年10月5日，週一上午11時許，美國總統新聞秘書、白宮發言人凱莉·麥肯尼（Kayleigh McEnany）在推特上宣布她對新型冠狀病毒的測試結果呈陽性，這一信息顯示白宮確診的冠狀病毒病例進一步擴大。根據多份報導，兩名新聞秘書的助理也在早前確診。白宮內部現已有多位工作人員測試結果呈陽性，包括上週四確診的川普總統。目前為止，凱莉·麥肯尼女士是最新一位確認感染病毒的白宮官員。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/05220206/98-%E3%80%90%E7%83%AD%E7%82%B9%E6%92%AD%E6%8A%A5%E3%80%91%EF%BC%9A%E7%99%BD%E5%AE%AB%E5%93%88%E5%BC%97%E6%89%8D%E5%A5%B3%E5%8F%91%E8%A8%80%E4%BA%BACCP%E7%97%85%E6%AF%92%E6%A3%80%E6%B5%8B%E9%98%B3%E6%80%A7%EF%BC%8C%E7%99%BD%E5%AE%AB%E6%8C%81%E7%BB%AD%E9%99%B7%E8%90%BD-2.png)


凱莉·麥肯尼女士在自己推文中說，“自我連續病毒測試均呈陰性後，包括上週四之後每一天測試，週一早上，我的新型冠狀病毒測試結果呈陽性，目前為止沒有出現任何相關的症狀。”此外，她還表示自己“即將開始進入隔離階段，並仍繼續代表美國人民進行遠程工作”。
