历史真的是一面镜子。想起文革的起因：毛泽东由于一意孤行发起了“三面红旗”运动，造成了三年大饥荒中数千万人的死亡，而被迫退居二线。为了重新夺回失去的权利，毛以“反苏修、反美帝”的革命名义攻击所谓的“走资派”，开始了一场与“文化”无关的大革命运动。这场文革运动在两年之内就达到了高潮，造反派大规模的夺权、建立“革委会”以取代党政机关，还在全国范围内上演大规模武斗并导致全面内乱，但整个社会对毛泽东的个人崇拜却达到了前所未有的顶峰。

再观习近平几年来的所为，比毛泽东的“大跃进”、“文革”有过之而无不及：在国内，大权独揽、破坏任期制；美化文革；不定接班人选；挖空民营经济，使大部分民营企业家惶惶不可终日，导致民企倒闭、资金跑路；宣传空洞无物的习思想，大搞一言堂，排除异己到达了登峰造极。在海外，强推“一带一路”，实行全球大撒币，非但没有得到各国的尊重，反而因为对当地的自然、社会环境的破坏而引起全球的反感。在走头无路时，竟然启用极端的恐怖行为，用生化武器向国内、外定点传播，造成人类历史上最邪恶的灭种族大屠杀。现在世界各国政府已经清醒地认识到中共对人类和地球的威胁，纷纷要求追查CCP病毒真相，清算CCP罪行！共产党已如丧家之犬！
