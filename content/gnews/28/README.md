她從銀行行長降為員工，辭職4年後落馬了

龍江銀行股份有限公司原黨委副書記、行長關喜華涉嫌嚴重違紀違法，目前正接受紀律審查和監察調查。
9月25日，據黑龍江省紀委監委消息，龍江銀行股份有限公司原黨委副書記、行長關喜華涉嫌嚴重違紀違法，目前正接受紀律審查和監察調查。

新京報記者註意到，關喜華曾任龍江銀行股份有限公司行長，2015年降為龍江銀行員工，2016年辭職。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/26071522/cxc.png)
據官方簡歷，關喜華生於1963年5月，1987年畢業於東北農業大學，1989年進入交通銀行大慶分行工作，歷任開發區支行行長、市場營銷處處長、分行副行長等職，2008年任大慶市商業銀行股份有限公司黨委書記、董事長。

2009年，關喜華調任龍江銀行股份有限公司副行長，次年任公司黨委副書記、副行長，2011年出任龍江銀行股份有限公司行長，任職4年。

2015年，關喜華卸任龍江銀行股份有限公司行長職務，任公司員工，壹年後，於2016年辭職。

新京報記者註意到，2015年前後，龍江銀行有多名高管落馬。

2014年9月，時任龍江銀行股份有限公司黨委副書記、監事長楊進先（正廳級）被查。據媒體報道，此前，壹個自稱其近親屬的人士曾在網上實名發帖舉報其11項違法行為。

2015年7月，龍江銀行副行長王貴彬(副廳級)落馬。

據官網介紹，龍江銀行經中國銀行業監督管理委員會批準，於2009年12月25日正式對外掛牌營業，是在原齊齊哈爾市商業銀行、牡丹江市商業銀行、大慶市商業銀行和七臺河市城市信用社基礎上合並重組而設立的省級城市商業銀行在黑龍江省內13個地市均設有分行，轄屬營業網點222個，在黑龍江、湖北、安徽、福建、寧夏等5省區發起設立村鎮銀行11家。
