## 真善忍與真善狠
`2020-10-17 13:01 CHANGDAO` [轉載自GNews](https://gnews.org/zh-hant/430684/)

作者：美東香草山教育組春草


![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/17125640/131-%E7%9C%9F%E5%96%84%E5%BF%8D%E4%B8%8E%E7%9C%9F%E5%96%84%E7%8B%A0-1.jpg)


“真善忍”這三個字不知何時在中國流行起來，在傳統社會文化中很少提到“真”，“善”和“忍”字出現在很多方面；而“真善狠”是七哥提出來用在爆料革命中的。本文僅就“忍”和“狠”說說自己的看法。

“忍”在這裡的意思，是表述人控制情感、情緒、慾望的，如：忍耐、忍受、忍讓、忍痛、容忍、忍辱負重、忍痛不言、忍氣吞聲、忍辱求全、忍垢偷生……；從以上的詞組可以看出，“忍”主要是一種心理活動，凡是“忍”者都要經受心理上的折磨、精神上的痛苦壓力，甚至損害健康和生命，故有“忍字心上一把刀”之說。

在中國把能否能“忍”作為個人修養素質的標準，如面對等級高的人不要爭辯，即便被誤解不要解釋，被侮辱了不要反抗，在社會來往中利益受損不要追責等等，這謂之“有涵養”，很多文人還把“難得糊塗”作為社會行為的座右銘。總之就是對社會上的邪惡人事不要批評，對惡魔流氓的欺辱不要反抗……。這是統治者極力倡導的社會文化，為他們統治壓榨殘害人民製造輿論，尋找理論依據。告訴民眾中國自古以來就是這樣，你若不忍，就讓你輕者損失財物、皮肉痛苦，重者家破人亡，人間消失。在這種環境下，平民百姓以及權勢低下的人和團體經常遭到欺凌，財產被掠奪，女妻被凌辱，卻上告無門，中國的老百姓一直生活在沒有平等沒有自由，暗無天日的水深火熱之中。久而久之，民眾就習慣了這個制度，故民間流傳著“胳膊擰不過大腿”，“在人屋簷下不得不低頭”等”名言”，寧願忍受著精神上的痛苦、折磨，也不作聲， “寧死不打官司”在民間廣為流傳。

但是民眾的忍讓卻使統治者更為殘暴，使他們認為理所當然對老百姓就是要兇惡、狠毒，就是要殘暴鎮壓，所以王岐山說“中國人一百年也開不了智、吃草也能過三年”，根本不把老百姓當人看！從社會的角度看，正是民眾的忍讓助長了統治者的邪性惡性，使他們無所顧忌地掠奪老百姓的財產，肆無忌憚地殘害著老百姓的生命，這種“忍”實際就是助紂為孽，從法律的角度看也是一種犯罪。

“狠”在這裡的意思，是表述堅決、果斷、盡力、不達目的不罷休……。

針對中共邪惡的統治、對老百姓殘暴的毫無人性的手段，以及中共的爪牙走狗，七哥提出了“真善狠”，用“狠”取代了“忍”，顛覆了中國兩千多年的觀念。從古今中外的歷史看，很少有專制獨裁的統治者，會自行退出歷史舞台，即使民主社會的政府一不留神，也會退回到集權獨裁的社會。因為人性中自私貪婪的弱點，很多人一旦身居高位、大權在握時，他們的私慾就會膨脹，只要環境條件合適就會發展演變成專制獨裁殘暴的統治者，並出現一批追隨者和廣大的為了生存不得不低頭認命的老百姓。

面對這種現狀，用“真善忍”不但不能解決問題，更助長了他們的囂張氣焰，必須用“狠”才能徹底消滅邪惡的統治者及其走狗，“真善忍”只能用於廣大的老百姓。從古至今沒有用“真善忍”能消滅或推翻殘暴統治者及走狗的先例。

我認為這個“狠”就是對於社會中出現的惡人惡事不忍讓、不遷就、不講情面，哪怕是微不足道的惡行也不妥協、不放過，要用智慧和行動，堅決地把惡行消滅在萌芽中（要盡量保護自己的生命安全）；但對於位高權重的殘暴的統治者及其爪牙走狗就要講究方法策略，要有長遠的計劃步驟，不蠻幹，不盲目行動，根據情況變化隨時調整戰略戰術，一旦時機成熟，堅決果斷出手，不給他們以還手的機會，這就猶如一場你死我活戰爭。在這方面七哥以他過人的才智和行動能力為我們展現了傲人的豐富的案例，為我們做出了絕世的榜樣。當然我們是沒法和七哥相比的，只能在我們的能力之內積極地思考行動。

比如這次全球依法懲賊的行動，前線的戰友們做的太好了，尤其是德州和到德州的戰友，做的非常出色，有計劃有步驟、有紀律，有理有據，不卑不亢，不妥協、不放棄，不懼黑警的威脅，不論遇到什麼困難一直堅持到今天——第十三天，不達目的決不罷休，真正做到了“狠”字，從理論到行動踐行著“真善狠”。相信在戰友們的努力堅持下，一定會把中共的爪牙、走狗、偽類、欺民賊——傅希秋等，揪出來暴露在光天化日之下，受到法律的嚴懲！

七哥提出的“真善狠”豐富了中國的傳統文化，彌補了社會文化的缺失，增添了中華民族為人處世的新觀念、新標準，對人類社會的進步、文明的發展將會起到積極的促進作用，將會融入未來的新中國聯邦的社會主流文化中。
