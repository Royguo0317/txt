加拿大主要法语媒体《魁北克日报》于5月8日刊登新闻，就现今“中共病毒”疫情波及破坏下的本国航空业之萧条惨状进行了及时报道。全文编译如下：

目前，严峻的疫情形势继续无情地打击着加拿大航空公司（加航）。该公司上周五宣布，2021年其首季度资产净亏损高达13.04亿加元。该不利情况比去年更严重，彼时加航报告的同期净亏损额为10.49亿加元。仅仅一年后，这个“漏洞”就加深了2.55亿加元。在2020年，加航每股摊薄亏损为4.00加元，而今年为3.90加元。

此外，截至3月31日，加航的收入为7.29亿加元，比2020年同期的37.22亿加元大幅急剧下降了29.93亿加元。在2021年第一季度，加航的旅客运送量仅为11.24万人次，该数字远远低于去年同期三个月的992.7万人次（-88.7％）。

据报道，加航在（中共病毒）疫情的第一年就解雇了约20,000名员工，而在2021年初又辞退了近2000人。

值得一提的是，该公司在上个月从联邦政府获得了近60亿加元的援助计划，用以逐渐偿还因（中共病毒）疫情泛滥而无法出行旅客的损失，且最终向这些客户退还了12亿加元的票款，并以此换取联邦政府的上述资金援助。

加航总裁兼首席执行官迈克尔·卢梭忧心指出：“加拿大航空业仍然受到持续不断恶化的（中共病毒）COVID-19大流行的侵害，加航第一季度的营销业绩无疑证明了目前该行业的恶况。现在看来，政府应该宣布并实施一项重新开放我们国家的计划显得尤为重要，并以此取代目前所谓的以科学为基础，针对所有旅客的（中共病毒）筛查措施以及各种强制性旅行检疫规定。”

最后该总裁强调道：“随着疫苗接种计划的持续执行，加拿大应该可以安全地重新对外开放，并同步放宽此前的各项旅行限制。”（全文完）
