川普总统的私人律师鲁迪·朱利安尼（Rudy Giuliani）带来了几位非常可靠的证人出席了听证会。 据一位证人的陈述，他看到数十个USB卡的内容被上传到投票机，从而在短时间内为拜登赢得了五万张选票。而当另一位证人指出在宾夕法尼亚州拜登蹊跷地获得了60万票，而川普只获得了3200票时，全场一片哗然。

在听证会上，朱利安尼（Rudy Giuliani）指出，宾夕法尼州的选票有两个统计缺口， 即有672,770张选票没有被任何人检查就被秘密的放入投票箱；另外，总共寄出了180万份邮寄选票，而最终被计数的选票是250万份。

在本次总统大选中，宾夕法尼州涉及了多项选举舞弊，诸如：宾州选举官网删除了投票数据库链接；费城的”选票单接收箱”(dropbox)发生有人重复投递选票；邮寄选票延期的“违宪”行为；宾州最高法院越权，插手地方选举等。对此，宾州的民众在议会大楼前抗议大选舞弊，捍卫自由和价值，在一系列正义力量的驱使下，宾州的情况出现了逆转，宾州联邦法院法官终于宣布叫停宾州的选举认证，这是美国人民的胜利。此次听证会是美国历史上最重要的听证会，尽管主流媒体拒绝报道，但是真相是不会被掩盖的，让我们对接下来的宾州选举舞弊事件的最终结果拭目以待。
