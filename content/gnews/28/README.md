## 聚焦中共“两会”之一：高度关注“卡脖子”问题

3月5日，腾讯网转载新京报一篇题为“高度关注的‘卡脖子’问题，今年将从这方面着手”的文章。中共国国务院总理李克强在3月5日上午的政府工作报告中提出，要提升科技创新能力。就此问题文中给出了进一步解释，“关键领域创新能力不强”，之前被形象地概括为“卡脖子”问题。近两年，在美国芯片断供的压力下，一长串“卡脖子”清单被列出，造成创新能力差的根本原因专家分析认为是“基础研究不足、企业发挥的作用不够”。为此政府工作报告给出了两点建议，对基础研究“大幅增加投入”，按“揭榜挂帅”要求解决“卡脖子”问题。

“两会”的政府工作报告将提升科技创新能力放在了重要位置，也就是变相承认由于美国的制裁，中共国被卡住了脖子，如果问题得不到解决，就是死路一条。其实早在2020年9月《科技日报》就给出了“卡脖子”清单包括：高端光刻机、芯片制造、个人电脑和手机操作系统以及相关工程软件、核心工业软件、数据库管理系统、机器人核心算、航空发动机，医学影像设备元器件等等25项技术。仔细观察不难发现，一旦这些问题得不到解决，电脑、手机变板砖，一切与现代社会接轨的计算机技术、网络、航天、工业都无法运行，中共国立刻回到石器时代！

众所周知，中共国的科技之所以落后，本质就是中共国的制度问题，教育制度的落后、各项管理制度的混乱、腐败，造成人才流失和科研滞后是一种必然。中共国的教育体制犹如一个有着统一标准的教育机器，使用统一的教材、进行统一的考试，有所谓的统一答案，限制了学生独立思考和创新的能力，在这种统一的教学体系下，很难培养出有创新能力的人才。

即使拥有创新的人才，在中共国的体制下科研人员也无法安心完成科学研究。一名科研工作者面临着科研经费、晋升职称、提拔干部等许许多多的现实问题。在中共国，项目经费被层层克扣无法真正落实到科研人员手中，他们还要面临提干、评职称等等问题，但是这些并不是根据研究成果评定的，是靠资历、靠关系、靠腐败才能达到，这也打击了科研人员的积极性。再加上整个社会只看重车子、房子、票子，而真正的科学家需要静下心来，踏踏实实从基础做起，也许一百次的实验才能成功一次，也许一个科研工作者一个创新要研究许多年，也许一生都无法成功，其实成功恰恰是由这些失败积累而成的。整个社会都急功近利如何让科学家独善其身？！于是中共不去研究科学，不去创新，只能靠窃取其他国家的先进技术，还美其名曰“弯道超车”。

如今美国及西方民主国家的制裁在一步步紧逼，中共的脖子被一点一点“卡”紧，中共自己也意识到偷窃不行，花钱买也不行，其科技将死，现在再如何加大投资力度、如何让企业承担责任也无法“起死回生”。只有早点消灭中共，让生活在防火墙内仇恨美国制裁、拥护中共制度的被愚弄的老百姓早点醒来，彻底改变“劣币驱逐良币”的体制，才能让科技事业真正发展起来。
