## 病毒：諾如病毒感染，成都錦江區幼兒園多名幼兒嘔吐
`2020-09-26 10:21 GM06` [轉載自GNews](https://gnews.org/zh-hant/384277/)

編輯：雅典娜之矛

近日，墻國內多地出現新冠病毒無癥狀感染者。此外，各地也出現不同類型其他傳染疾病。

今日，成都錦江區通報幼兒園多名幼兒出現嘔吐：符合諾如病毒感染特征。
截止目前，出現癥狀的幼兒情況穩定，身體正逐漸好轉，均居家休養。
新京報快訊 針對媒體報道成都壹幼兒園多名孩童出現嘔吐、腹瀉癥狀壹事，9月26日，成都市錦江區教育局官方微博發布通報，2020年9月24-25日，錦江區貝多彩幼稚園小（1）班部分幼兒相繼出現嘔吐或腹瀉癥狀，截止目前，出現癥狀的幼兒情況穩定，身體正逐漸好轉，均居家休養。

9月26日下午，區疾控中心檢測出幼兒排泄物中諾如病毒呈陽性，結合臨床癥狀研判，符合諾如病毒感染特征。
