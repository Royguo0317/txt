【[#*西安*报告*1**例本土确诊病例*#](https://s.weibo.com/weibo?q=%23%E8%A5%BF%E5%AE%89%E6%8A%A5%E5%91%8A1%E4%BE%8B%E6%9C%AC%E5%9C%9F%E7%A1%AE%E8%AF%8A%E7%97%85%E4%BE%8B%23)：系医院封闭隔离病区检验师，意外暴露造成偶发感染】据陕西省卫健委，3月18日凌晨，西安市报告1例封闭隔离病区本土确诊病例。接到报告后，省卫生健康委立即指派专家连夜赶赴西安市指导处置工作。[#西安新增1例本土确诊病例#](https://s.weibo.com/weibo?q=%23%E8%A5%BF%E5%AE%89%E6%96%B0%E5%A2%9E1%E4%BE%8B%E6%9C%AC%E5%9C%9F%E7%A1%AE%E8%AF%8A%E7%97%85%E4%BE%8B%23)

确诊病例刘某，西安市第八医院封闭隔离病区检验师，主要负责隔离病区内患者的核酸采集、实验室检验工作。该病例3月3日核酸检测阴性，3月4日进入封闭隔离病区，一直在封闭隔离状态下工作和生活，与外界没有接触。3月11日，每周例行核酸检测阴性。截至3月16日，每日监测体温均正常。3月17日8时35分，出现头痛，体温37.9℃，胸部CT双肺可见云絮影，核酸检测结果可疑。当日省、市两级疾控中心再次核酸检测，结果均为阳性。3月18日凌晨，经省市联合专家组会诊，诊断为中共病毒确诊病例，目前在定点医疗机构隔离治疗。

![]()![](https://gnews.org/wp-content/uploads/2021/03/33-7.jpg)

根据人民日报所报道的——［经省市专家组初步研判，该病例系在封闭隔离病区内意外暴露造成偶发感染。该医院为收治中共病毒感染者的定点医院，现隔离治疗和医学观察境外输入确诊病例8例、无症状感染者5例。隔离病区实行全封闭管理，同批次工作人员3月4日以来均在封闭隔离状态下工作和生活，目前核酸检测结果均为阴性，已全部落实隔离医学观察措施，对其工作、生活场所已落实环境采样、封闭隔离、终末消毒等防控措施。］，这一例确诊病例一直在封闭隔离状态下工作和生活，那么[-初禾](https://weibo.com/u/2532504284?refer_flag=1005055013_)所在的小区的7号楼和9号楼出现的隔离是因西安新增1例本土确诊病例导致的疫情扩散呢，还是在西安公布的1例确诊病例之外的未上报病例呢？无论是哪一种情况，都只能证明一点——西安的疫情绝对不是官方所公布的那样！既然西安是这样，那么整个中国的数据会是真的吗？既然中共病毒是生物武器，既然这是针对全人类的超限生物战，我们除了严阵以待切实做好自我保护之外，我们到底还能相信任何中共国的疫情披露信息吗？
