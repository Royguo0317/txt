主要内容

一、从张恩华的突然去世看中共恶酒文化

4月29日，大连实德队的前队长，国足冲世界杯时的主力后卫，48岁的张恩华因为前一晚上生日宴上饮酒过量去世。 喝酒肯定是诱发心脏猝死的重要原因，尤其是运动员退役后更容易出现这个问题，因为不运动，喝酒，喝假酒，喝混酒，就会诱发心梗。张恩华去世算晚的，48岁才去世，很多中共的运动员很早就喝酒死掉了。中共的体制和酒文化祸害的不仅仅是足球队员，篮球队，羽毛球队，乒乓球队里的运动员喝酒喝死的比足球队的只多不少。在中共国，不喝就办不成事，不仅在工作中需要勾兑，在工作之外也一样。中共拿酒拿酒桌文化来衡量一个人是否忠诚听话，能不能使唤，结果就是身体完了家完了人也喝没了。在这种酒囊饭袋的体制里，不可能培养出好的球员来。

二、中共大力宣传习近平的三轮摩托草帽毛巾

人民网发表了”三轮摩托、新华字典、草帽毛巾背后的青年习近平”。文中说1969年不满16岁的习近平主动申请到陕北农村插队，在梁家河大队带领乡亲们修筑淤地坝，尽心尽力帮助村民，还把奖励的三轮摩托车换成了农用工具给村民用。习实在没有什么东西可宣传的，虽然在福建浙江上海干过多年，但没有政绩，找不到料，没法服众，只能把摩托换农用工具的事都当成了丰功伟绩，写在党史党建上，这红色江山快要完蛋了。真正有能力的人，是会有一些能拿得出手的政绩的。薄熙来就能拿得出来很多政绩，比如在重庆给老百姓的房屋优惠政策，城市绿化政策，建立了交警快速回复机制等。而靠红色基因靠喝酒上位的工农兵学员的习近平，在党内没法服众，只能通过党报把自己捧成伟人。但最近党报拍习的马屁拍过了。大家都知道习的错别字非常多，党报却宣传习十几年前就把新华字典都背下来了，这简直是在打习的脸。

三、习近平下令与美争夺社交媒体话语权

大纪元获得的一份档泄露。在四年前，习近平下令制定互联网规则，争夺话语权，包括互联网技术话语权，建立独立根伺服器，摆脱美国限制。此外，强调互联网空间话语权，包括国内和国外话语权。国内方面，中共通过对微信、微博等新媒体平台管制，培育更多的红色大V，自干五，导致平台活力尽失，没有异议的声音，完全听从指挥。对于海外媒体话语权，深入进行布局，提升影响力。这个档是2017年，来自辽宁省网信部门，时任省委书记是李希。但是为什么关于互联网话语权的事，习近平没有去找马云、马化腾等互联网大佬去谈，而是找了李希，因为李希是习近平的心腹。而这里强调的海外媒体话语权对手是江泽民派系控制的海外话语权，是习近平要从江派手里夺取海外的媒体话语权。此外，也要争夺中共在推特、facebook等大社交平台的话语权。一切从2017年开始就在按计划布局。但是中共错估了人性，认为人性都是邪恶的，导致计划必将失败。这个世界还是有很多人内心有良知和正义的，因为爆料革命，郝海东、叶昭颖、闫丽梦等战友的挺身而出，打乱了中共在争夺海外媒体话语权的如意算盘。中共强调做大做强数字人民币，其用心美国早已看穿，就是威胁美元霸权。美国也强调中共第一阶段的知识产权保护没有做到301调查要继续开展。这都表明中共争夺海外媒体话语权的计划失败了
