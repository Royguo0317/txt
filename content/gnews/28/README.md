03月20日，乌克兰内务部长阿尔森·阿瓦科夫表示，美国总统拜登对俄罗斯总统普京说的话可能导致西方和俄罗斯之间的新冷战。乌克兰内政部网站称，他是在电视节目《Svoboda Slova 》播出时说的。

“在普京看来，这可能是西方文明与俄罗斯帝国冷战的开始。但我担心这不会变成乌克兰的热战，因为俄罗斯总统可能会对我们再玩一次冒险。”阿瓦科夫说。

他补充说，对乌克兰来说，甚至不存在普京是否是凶手的问题，因为乌克兰早在2014年就已经得到了答案。

此前，拜登对记者提出的是否认为普京是凶手的问题作出了肯定的回答。美国领导人还承诺，俄罗斯将为干预美国大选付出代价，拜登说他在电话中警告普京。

克里姆林宫表示，拜登的言论可被视为不愿修补与俄罗斯的关系。“美国总统的这些言论非常糟糕。他显然不想与我国建立关系，我们将以此为出发点。”佩斯科夫说。

普京本人也用一句话回应了拜登的话。“我记得小时候，我们在院子里互相争论的时候，我们常说这样一句话：‘谁叫自己叫那个名字’。而且这不是偶然，也不是单纯的玩笑，其心理含义非常深刻。我们总是在别人身上看到自己的特点，认为他和我们一样。我们根据这一点来评估他们的行动。”总统说。他邀请拜登不要“缺席挑衅对方”，而是在直播中进行“开诚布公”的辩论。
