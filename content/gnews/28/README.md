2020年11月4日，美国总统大选，威斯康辛选票数据造假，登记的投票人数3,129,000，而选票加总却是3,239,920，多出110,920张选票。

假设110,920张选票均是乔拜登候选人造假选票，实际得票应为1,519,414张，48.56%。

川普总统实际得票1,609,586张，51.44%。
