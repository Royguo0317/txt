232-197的投票是历史性的。它使特朗普成为美国历史上第一位被弹劾两次的总统。

10名共和党人和每一位投票的民主党人一起批准了单项弹劾条款，该条款指责特朗普煽动暴力对抗他所领导的同一个联邦政府。

监督这两项弹劾工作的议长南希-佩洛西(Nancy Pelosi)说，特朗普拒绝承认选举失败–以及他随后召集支持者到国会大厦推翻选举结果的努力–相当于煽动叛乱。她说，总统让国会别无选择。

除了切尼之外，其他9名投票弹劾特朗普的共和党人是国土安全委员会的顶级共和党人约翰-卡科(John Katko)、众议员。Fred Upton(密歇根州)和Jamie Herrera Beutler(华盛顿州)，他们与Katko是中间派周二集团的共同主席；Reps. Adam Kinzinger（伊利诺伊州）和Peter Meijer（密歇根州），他们都是战争老兵；众议员Anthony Gonzalez（密歇根州）。

“我们如何与40%仍然相信总统的人找到任何共同点？”众议员罗-卡纳（D-Calif）问道。”你可以把他从Twitter上踢出去。你可以弹劾他。你可以追究他的责任。但这不会突破一个巨大的人口……他们相信特朗普所创造的现实。”

- 01.13 TheGatewayPundit [总统特朗普在白宫的新视频信息中谴责政治暴力，抨击大科技审查制度。](https://www.thegatewaypundit.com/2021/01/president-trump-condemns-political-violence-slams-big-tech-censorship-new-video-message-white-house/)

“我想说得很清楚：我明确谴责我们上周看到的暴力事件。暴力和破坏行为在我们国家绝对没有地位，”特朗普说。

“我的任何一个真正的支持者都不可能赞同政治暴力。没有一个真正的支持者可以不尊重执法部门或我们伟大的美国国旗，”他说。

这是参议院多数党领袖米奇-麦康奈尔的发言人说的。该发言人称，这位肯塔基州共和党人的助手已经告诉参议院少数党领袖查克-舒默的幕僚，麦康奈尔不会同意援引权力要求参议员召开紧急会议。

因此，备忘录继续说，弹劾审判可能要到1月20日下午1点拜登宣誓就职后才能开始。

- 01.13 TheGatewayPundit [密歇根州法官命令民主党州务卿公布与多米尼克、Facebook、苹果、亚马逊和谷歌的所有通信。](http://密歇根州法官命令民主党州务卿公布与多米尼克、Facebook、苹果、亚马逊和谷歌的所有通信。)


大选结束后的第二天早上，在密歇根州北部一个几乎每个前院都能看到特朗普横幅、旗帜和院子里的标语的红色县，选民们一觉醒来，发现乔-拜登在11月3日的大选中壁咚了特朗普总统。经过进一步检查，发现多米尼克投票机统计的6000张选票从唐纳德-J-特朗普翻到了乔-拜登。媒体称这是一个 “小故障”，并将其归咎于人为错误–美国公众感到震惊，并要求立法者调查这一事件以及在多个关键摇摆州出现的其他选民违规行为。

除了数以千计的选票从特朗普总统手中转到民主党候选人乔-拜登手中，密歇根州中央湖（位于安特里姆县）居民威廉-贝利（William Bailey）担心，在一项投票倡议中，允许在中央湖村内设立大麻场所的选票262-262平后，选票通过多米尼克制表机重新运行。

当选票被放入机器时，有3张选票被销毁，没有被放入制表机。在计票机重新计票结束后，在少了3张票的情况下，结果是262-261，倡议书获得通过。当然，这个结果只有在3张选票被销毁后才有可能出现。

美国英雄和宪法律师马修-德佩诺勇敢地同意接受贝利的案件。

12月4日星期五下午5:30，第13巡回法庭法官Kevin A. Elsenheimer批准William Bailey及其IT专家团队对16台Dominion投票机、制表机、拇指驱动器、相关软件和书记员的 “主制表机 “进行取证研究。

2020年12月14日，密歇根州第13巡回法庭法官Kevin A Elsenheimer同意允许DePerno先生的委托人William Bailey及其技术精湛的IT专家团队对密歇根州安特里姆县的16台Dominion投票机进行取证检查。周一，不顾密歇根州州务卿本森的反对 埃尔森海默法官同意将法医检查的结果公布于众。结果是令人震惊的。

在对密歇根州安特里姆市的16台多米尼克投票机进行取证检查后，联合安全运营集团已经对这些机器进行了取证。MI，Allied Security Operations Group得出结论，Dominion Voting机器的错误率为68.05%。德佩诺解释说，当选票通过机器时，高达68.05%的错误率意味着68.05%的选票被送往批量裁决，这意味着他们将选票收集在一个文件夹中。”选票被送到某个地方，在另一个地方的人可以改变投票，”德佩诺解释说。联邦选举委员会准则规定的允许选举错误率是25万张选票中的1张，即0.0008%。

12月15日，密歇根州参议院监督委员会就11月大选中密歇根州使用的Dominion Voting机器向Dominion公司CEO John Poulos提问。

密歇根州参议员皮特-卢西多(Pete Lucido)问普洛斯先生：”普洛斯先生，作为一个选民，我怎么知道我没有任何软件被操纵的违规行为？我怎么能确定呢？” 多米尼克首席执行官回应说，他告诉卢西多参议员：”在一天结束的时候，你要记住，在一天开始的时候，投票箱是安全的，是空的。”他还说：”而在一天结束的时候，每个选民都有一些选票存入那个密封和上锁的投票箱。” 普洛斯补充说：”这些选票没有去任何地方。”他向委员会保证：”它们一直在投票站工作人员的安全控制之下。”

德佩诺先生联系了《百分百联邦》，解释说多米尼克公司的首席执行官约翰-普洛斯在没有任何证据证明他的说法的情况下，做出了关于票箱被安全控制的这一声明。

代表William Bailey起诉安特里姆县的宪法律师解释说，2020年11月27日，他和Bailey先生以及作为其团队成员的五名IT专家 “得到了四个选区的许可，对他们的制表机和制表机打印关闭卷进行检查”。德佩诺先生解释说，在他们检查的4台Dominion机器中，至少有2台机器侧面的封条破损。其中一个破损的密封条是在Mancelona Twp.的Dominion Voting机器上发现的，第二个破损的密封条是在密歇根州的Central Lake发现的；此外，Central Lake的Dominion Voting机器的侧面没有安全锁。

# 墙内咨讯

#### [“牛”皮吹爆了 P2P平台小牛资本涉非法吸存遭警方立案侦查](https://finance.sina.com.cn/roll/2021-01-14/doc-ikftpnnx6752431.shtml)

证券时报记者 谢忠翔

规模千亿元级别的P2P平台小牛资本集团被深圳警方正式立案侦查。

1月13日，深圳南山公安分局发布通报称，小牛资本集团有限公司（下称“小牛资本”）涉嫌构成非法吸收公众存款罪，为保护投资人权益，惩治犯罪，深圳南山警方于2021年1月8日依法对小牛资本（“小牛在线”、“小牛私募”平台）立案侦查。

通报显示，经全面收集证据，1月13日，深圳南山警方对小牛资本涉案嫌疑人依法采取刑事强制措施。

**仍有过百亿元未兑付**

据小牛资本官网，该公司成立于2012年，旗下平台包括小牛普惠、小牛在线、小牛新财富、小牛投资和牛鼎丰科技；涉及普惠金融、财富管理和投资管理业务。

2013年6月，小牛在线平台上线运营。截至目前，累计成交金额1171.74亿元，累计出借1578.31万笔，累计注册人数608.15万，累计出借人数85.94万。

2020年5月9日，小牛在线在其官网发布《平台网贷业务良性退出公告》，称自公告发布之日起，平台将停止发布任何新标及停止计息，并关闭出借人开户、充值、投标等功能；同时，自公告发布之日起，暂缓兑付，待兑付方案经全体出借人公开投票表决通过以后，按照兑付方案启动兑付工作。

在公告中，小牛资本解释称，近两年网贷行业经营环境恶化，出借人投资风险及平台经营风险增大，疫情极大增加了未来的不确定性，各大网贷平台陆续退出，小牛在线决定退出网贷行业，逐步结清存量网贷业务。小牛在线承诺，良性退出期间，平台股东及高管“不跑路、不失联、不撤资”。

同年5月11日，深圳警方在官方微博发文称将督导风险处置，要求小牛在线一定严格按照监管要求、落实清退主体责任，加强与出借人的有效沟通，保障出借人合法权益。同时还提到，平台全体借款人必须依约履行还款义务，以免对自身信用造成严重影响，或被追究法律甚至刑事责任。

借款数据方面，目前小牛在线官网已无法查询。不过，据证券时报记者2020年5月报道，当时小牛在线平台借款人总数为116.71万，借贷余额为104.20亿。其中，1万至5万的借款金额占比最高，达37.5%；5万至10万的借款金额占比为24.22%；10万以上的借款金额占比为22.96%；1万以下的借款金额占比为15.33%。从数据上来看，5万以上的借款占比近半。

值得注意是，其未偿本金总额不止104.2亿元。根据小牛在线于去年8月7日公布的《监委会投票结果及成立临时监委会公告》显示，截至2020年8月6日18时，已完成确权的总人数为10.03万人，代表的未偿还本金为95.73亿元，占平台未偿还本金总额的84.39%，据此推算，当时仍有113.44亿元尚未偿还。

小牛在线官网显示，自2020年5月宣布“良性”退出以来，该平台已启动十期的兑付工作。7月30日，第一期共兑付5000万元，累计至今共兑付2.38亿元。据此推算，目前该平台尚未兑付的本金总额仍达111.56亿元。

***简评：**官商勾结，割韭菜一波未平，一波又起。*

#### 参考消息：中国多地鼓励民众就地过年

据拉美社1月11日报道，中国政府出台各种措施，利用免费退票、严格隔离和薪资奖励等劝阻民众不要在传统的农历新年外出旅行，而是留在居住地过节，以遏制新冠疫情的反弹。

报道称，北京、上海、广东、河南、湖北等地都向农民工发出通知，鼓励他们留在就业地，除不可抗力原因外，不要返回家乡过春节。

例如，武汉市要求，全国低风险地区返汉返乡过春节的人员，全部要持7天内核酸检测阴性证明，不能提供核酸检测阴性结果的，街道为其安排核酸检测并进行14天的社区健康管理。

报道还称，在中国的一些企业中，春节期间被安排加班的员工不仅有3倍工资，每天还有额外奖金。铁路公司和许多航空公司已经提供免费退票服务。

报道指出，国务院应对新冠肺炎疫情联防联控机制综合组将负责监督全国的情况，因为在春运期间，总会有一定数量的人员在全国各地流动。今年春运预计将从1月28日持续至3月初。

另据埃菲社1月11日报道，由于一些地区暴发小规模疫情，为了避免疫情扩散，中国多地已宣布取消春节庆祝活动。春节是这个亚洲国家最重要的传统节日。
