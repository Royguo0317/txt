## 卷土重来的日本疫情
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1146057/)

**撰稿人：东南**

据日本广播协会电视台统计，截至当地时间26日18时（北京时间17时），日本当天新增新冠死亡病例35例，累计死亡超过1万例，达10012例。大阪府已连续5日感染人数过千人，日本政府前几日向东京、京都、大阪等地再次颁布紧急事态宣言，酒类餐饮与卡拉ok场所全部停业，因疫情原因东京奥组委也宣布，5月9日的田径测试赛将不开放观众入场。
![]()![](https://gnews.org/wp-content/uploads/2021/04/16195832741.png)**（图片来源自由时报）**
记得日本政府去年底宣布的暂停来自所有国家和地区的新入境。在英国发现的变异新冠病毒基因序列一致的病毒。相关的5名感染者从英国返回，日本国内机场检疫所当即确认了这5人的感染情况。日本的之前从时间上可以明显看出事态的紧急程度，从发布到执行的时间都不足48小时。而这一次的印度变种病毒的爆发远超之前，并且目前已完全没有源头踪迹可寻。要知道日本首相菅义伟在3月18日召开的新冠病毒肺炎疫情对策总部会议上刚宣布，3月21日日本首都圈四地如期解除紧急状态。但从解封到这次停业间隔时间只有短短的1个多月。怎样的经济体能承受一年数次如此反复的沉痛打击？

在我看来中共这一年多的时间里已经成功的把世界搅的一团乱，**日本去年年初病毒刚开始在本土出现时，为保奥运召开拉动GDP不断的淡化疫情宣传，这种扭扭捏捏的政策直接导致日本从疫情初期至今共计56.9万人感染，10012人死亡的惨痛代价，另外要重点说的是半数的死亡病例还都是发生在近3个月内，再和现在世界上大多数的政府一样，永远政治正确的话，疫情怎么可能得到彻底的解决，谁敢保证这是最后一次爆发？**

但幸好在钱和命哪个更重要的问题上，日本还是鼓起勇气地站了出来，不再扭扭捏捏对共模糊政策。日本驻澳大使近期透露：对日本加入“五眼联盟”持非常乐观的态度，并且日本正在为加入“五眼联盟”做准备，届时日本对于中共的之前各项情报的了解深度，一定会为“五眼联盟”的情报系统做一个有效的补充。**同时我们爆料革命战友的使命也是要和世界正义力量联合，也希望能有更多有正义感的国家跟进，要知道CCP只能为少部分“贪婪”政客带来一时的利益，但为此损害的确是成千上万的普通民众，让我们携手一起揭穿中共虚伪的面具**，使其在这个地球上的丑态无所遁形。
