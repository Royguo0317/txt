## 路德社重磅：习安插在美国间谍大佬——张博
` GM06` [轉載自GNews](https://gnews.org/zh-hans/503325/)

文章撰写：【重生】

《路德时评》2020年10月30日晚间节目继续爆料拜登家族与中共勾兑腐败证据，最新公布的一封邮件揭示出拜登家族与习近平派系有密切的关系。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/30075412/%E9%87%8D%E7%94%9F.png)
该封邮件是Francis Person于2015年7月28日发送给亨特拜登，主要内容如下：

Francis邀请亨特·拜登一同携带家属前往中国旅游，他们抵达中国后行程将由张博及其家人接待，他们从美国将飞往北京，让后前往沈阳，最后再飞往度假胜地海南。

Francis提及张博家族在中国享有非常高的地位，夸奖张博非常有智慧，他将接任家族王朝事业，未来他将成就伟大事业。他正在和NBA湖人队魔术师约翰逊[1]和勒布朗詹姆斯[2]商讨2016年在辽宁举办一场NBA比赛。

路德社爆料Francis与拜登家族合作多年，曾任乔·拜登竞选副总统时的竞选助理，同时在2009-2014年任美国总统总统助理和副总统特别顾问，现在是华锐娱乐公司（Harves Entertaiment）联合创始人及其董事会成员。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/30075511/%E9%87%8D%E7%94%9F2-1.png)
路德社也在节目中揭开张博家族的神秘面纱，张博父母在沈阳拥有华锐世纪集团[3]，该集团是以房地产开发、项目投资、商业运营为核心业务的大型现代企业集团，该集团的董事会主席就是张博的生母邵景兰女士[4]。

邮件中还提及张博的岳父，现任海南省省委书记刘赐贵[5]，从刘赐贵履历可以看到他从福建起家，习在福建任职期间，刘赐贵就是习的部下。2014年被调往海南省任职，从2017年5月份开始任职海南省委书记，省人大常委会主任、党组书记，省军区党委第一书记。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/30075653/%E5%88%98.png)
刘赐贵于2015年1月4日接受《中国新闻网》采访时“亮家底”：“我的妻子、儿子都是公务员。我哥哥、姐姐早就退休了，他们的子女也都没有经商办企业。请大家心中有数、共同监督，任何打着我的旗号办事的都应该拒绝。”[6]他在此次采访中刻意隐瞒他有一个女儿，而张博就是他女儿的丈夫，张博家族在中国拥有巨额财富，该家族非常低调。同时，路德社还帮我们回顾：刘赐贵是刘呈杰的养父，郭文贵先生曾在直播中多次提及刘呈杰，未来他将揭秘谁是刘呈杰的亲生父亲，想必长期关注爆料革命的战友应该猜到刘成杰的亲生父亲是谁了。

路德社在节目中爆料刘赐贵隐瞒其女儿的重要原因之一就是掩护张博，因为张博是习在美国最大的间谍之一。接下来，我们就来扒一扒张博。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/30075721/%E5%BC%A0%E5%8D%9A.png)
张博，现居住在美国哥伦比亚华盛顿地区，他在美国大学完成学业，主修财务与国际商务专业，是美国华瑞娱乐（Harves Entertaiment）的创始人兼CEO。通过网络搜索他的信息后发现其公司与英国曼联队有深度合作[7]，张博作为华锐世纪CEO与曼联高层洽谈合作，在中国建立曼联主题娱乐体验中心。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/30075801/222-4.png)
为何张博四十岁不到就可以有如此大的成就，可以和曼联，NBA湖人有紧密合作？路德社彻底曝光张博的全部信息：

1. 张博岳父刘赐贵为海南省省委书记，刘赐贵是习的部下，也是刘呈杰的养父；
2. 张博母亲邵景兰身居高位，为华锐世纪集团董事长，来头非常大；
3. 东北人和商业集团董事长戴永革，国务院秘书长王忠禹，海航王健和陈峰在张博面前就像狗一样卑躬屈膝；
4. 张博还是方正集团的隐名股东（文贵先生也是方正集团股东）；
5. 习近平女儿习明泽在哈佛读书期间，张博为其提供全程安保服务，并支付习明泽在美全部费用；
6. 张博家族在曼哈顿拥有大量房产，他就相当于东北的地下银行，权力极大；
7. 张博是习在美国最重要的间谍，其公司就是间谍机构，主要任务就是渗透共和党，渗透川普家人；
8. 张博接到习特派间谍任务：深度渗透美国高层，影响2024和2028年美国大选，进而影响美国政治走向；
9. 张博在美国成立的华锐娱乐公司就是帮习在海外开展“蓝金黄”行动，渗透美国NBA，渗透美国好莱坞，这些都是美国文化的象征，拥有巨大观众流量，中共就想通过渗透NBA和好莱坞影响民意，进而操控美国。
