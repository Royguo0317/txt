**川普總統簽署《香港自治法案》並簽署行政令，取消香港最惠國待遇**

隨著華盛頓和北京之間的緊張局勢持續升溫，川普總統周二宣布了白宮對中國采取的兩項行動。

14日在白宮玫瑰園舉行的新聞發布會上，川普總統宣布已簽署《香港自治法案》，並簽署了一項行政命令，以終止美國對香港的優惠待遇。川普總統強調，他比任何一屆總統在針對中共問題上都表現的更為強硬。

川普總統簽署的《香港自治法案》在7月2日美國參眾兩院全票通過。該法案將主要針對那些幫助中共侵犯香港自治權的實體以及與它們有業務往來的金融機構實施制裁，是對中共通過“ 港版國安法 ”嚴重違背香港高度自治地位的強烈回應。中共“ 港版國安法 ”收緊了北京對香港的全面控制，破壞了一國兩制的承諾，為應對中共對港威脅，5月27日美國國務卿蓬佩奧宣布取消香港自貿區地位。

**港版國安法是北京入侵香港，肆意破壞香港法治，嚴重違背香港高度自治地位的惡法**

港版國安法的通過使得北京具備以分裂主義、顛覆或恐怖活動，或外國幹預香港事務等為由肆意破壞香港法治，該法案的通過被視為中共對香港民眾抗議活動要求民主的直接回應。根據該法案，香港警方將擁有超級廣泛的權力，包括在沒有搜查令的情況下可以任意搜查，有權利命令互聯網服務提供商或平臺刪除被認為是違反港版國安法的相關信息。這些毫無法律根據的法條嚴重破壞了香港的民主制度，徹底違背了中共對香港許諾的“一國兩制”政策。

隨著川普行政命令的簽署，來自香港的出口商品將受到與來自中國大陸商品征收關稅的同等待遇–這標誌著將徹底切段內地利用香港逃避關稅的漏洞。該法案將制裁上到中共政治局常委下到香港警察等所有參與起草、制定、表決並執行香港國安法的所有個體與機構。這個法案所涵蓋的範圍以及未來的執法力度在中美歷史上是前所未有的。

**在南海問題上，川普政府也將采取反制措施遏制中國在南海日益強硬的侵略擴張。美國國務卿蓬佩奧****7****月****13****日表示，中國在南海的領土主張“完全不合法”。這是美國政府首次在南海領土爭議問題上表明立場。**
