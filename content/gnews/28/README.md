## 河南濮阳市突发暴雨变“水乡泽国”

中共国6月13日晚间，河南濮阳市突发特大暴雨，降雨量达277至322毫米，最大1小时降雨量达到120至132.8毫米。濮阳市区内积水严重，根据流出来的视频显示，街道上的积水已经超过膝盖，有的地方甚至已经超过了人腰部的位置，有行人甚至在街道上游泳。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/14064744/20210614_task5_Cover.jpg)网络图片
根据6月13日12时18分，河南濮阳市气象局发布的雷电黄色预警显示：预计未来6小时内濮阳大部分地区将有雷电活动，局地伴有短时强降水、短时大风等强对流天气；当日20点又发布暴雨黄色预警信号：预计未来6小时内，我市市区、清丰县、南乐县、范县降水量将达50毫米以上，并伴有雷电和短时强降水；

**【笔者评】中共国内城市的地下排水设施偷工减料，每当夏季雨水多的季节，中共国各地都会一派“水乡泽国”的独特风景，老百姓生活受到了重大的影响，财产受到严重损失，年复一年，从来没有任何改变。**
