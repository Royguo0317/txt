据华盛顿自由灯塔报道，众议院共和党人针对中共掩盖冠状病毒大流行的起源正在进行调查。众议员罗伯-威特曼和安-瓦格纳周二提出法案，命令对中共官员和卫生当局实施全球马格尼茨基制裁。这些制裁与用于惩罚世界各地侵犯人权者的制裁相同。

众议员威特曼提出的《世界应该知道法案》将调查可能在疫情中发挥作用的美国国家卫生研究院的研究资金，对找到冠状病毒的来源施加压力。该措施还提议禁止美国高校与可能在冠状病毒爆发早期阶段受到牵连的中共国研究机构之间的未来合作。

威特曼说，我们不能允许中共国阻碍对COVID-19的起源进行自由和公正的调查，我们也不能容忍中共国压制信息，这些信息可能被证明对预防未来的大流行病至关重要。《世界应该知道法案》实施的制裁为揭开COVID-19起源的真相铺平了道路。

众议员瓦格纳提出的《赔偿美国人法案》将加倍呼吁制裁中国共产党的某些成员，并为冠状病毒的美国受害者家属建立一个基金。该法案还将剥夺中共国在世界贸易组织中的“发展中国家”的地位，并禁止美国养老金在中共国投资，以及实施其他经济惩罚措施。

瓦格纳说，她的立法将向中共国发出一个信息，美国不会容忍这种可怕的行为。中共国采取了蓄意的、不计后果的行动，使冠状病毒得以传播，在全世界范围内杀害了数百万人。这不是一个可以辩论的问题，中共必须被追究责任。《赔偿美国人法案》将确保中共国为其腐败、不道德和危险的行为付出代价。

美国卫生与公众服务部监察长办公室周二宣布，它将对纳税人资助美国与武汉病毒研究所的合作进行调查，包括给生态健康联盟的税款。该组织因与武汉病毒实验室的合作而受到共和党人的指责。

中共制造和散布的中共病毒致使60万美国人，全球380万人失去了宝贵的生命，至今这一悲剧仍在继续。希望全世界协同一致，尽早确认病毒起源，向中共追责。
