1. “路德时评”本日摘要（[早间](https://www.youtube.com/watch?v=Qj3mz-Ap30Y)、[晚间](https://www.youtube.com/watch?v=IWAlu6SdJiM)）:

Ⅰ. 北京时间1月25日习近平出席世界经济论坛“达沃斯议程”对话会，并发表特别致辞。通过对比习之前参加世界经济论坛的影像资料，发现在照片日期、身材比例、动作表情上都与真实的习近平存在出入。结合此前习手术后身体并未恢复的实际情报，此次讲话视频实为造假，视频应为替身代拍。习本人术后仍未露面。

Ⅱ. 习近平领导的“文革”运动已经开始。近日中共官媒平台开始大量发布“大字报”，信息内容宣传习领导的反腐主题整风运动。从新华社发文来看，习的整治对象不仅包含军队，还将包括中纪委。习、江、胡、王四派势力正展开厮杀。

Ⅲ. 中共新华每日电讯微信公号刊发题为《吉林通化部分地方“断粮断药”：物资配送承诺别落而不实 》的评论。吉林通化市东昌区实行按区管控后，有市民反映政府通告的“必需品免费配送”政策没有及时落实，部分家庭出现“断粮断药”危机，有的家庭“一个苹果俩孩子吃3天”。爆料革命战友提前囤粮，救己救人。 中共此后将更加严格地管控人民言论，全面封锁消息。

Ⅳ. 路透罗马1月26日消息，意大利政府消息人士称，意总理孔特周二向内阁确认，他将很快向国家元首递交辞呈。孔特代表的中左翼联盟势力盘踞政坛长达16个月，此次换届曾经支持过闫博士的右翼联盟马泰奥·萨尔维尼有望接任。意大利政坛与梵蒂冈教廷关系密切。

Ⅴ. MSCI Inc表示将在1月27日起将5家中共公司从其旗舰全球股票指数中删除，五家公司为：中广核电力有限公司、中国化工、中国核电、中国船舶工业和浪潮国际。中美经济脱钩已经成为美国的基本国策，也是利益集团的要求，拜登政府将坚持脱钩策略。

2. 1月25日，日媒报道，为避免中共”千人计划”对日本科学人才的渗透，文部科学省要求从2021年度起，向政府申请科研经费的人员有义务申报曾接受的外国资金。

3. 印度媒体报道，中共和印度军人在北部锡金边界地区的纳库拉发生冲突。20名解放军及4名印度士兵受伤，但目前情况已受控。报道提及，事发时解放军尝试越过锡金的实际控制线，印度军人阻止他们。

4. 美国印度太平洋司令部表示，美国海军西奥多·罗斯福号航母编队于周六进入南中国海地区。美空军一架U2S侦察机于周一经台湾以东空域，执行南中国海侦察任务，最近距离台湾海岸线仅22.55海里。中共外交部表示，美国频繁派舰机进入南中国海活动，炫耀武力，不利于地区局势的和平与稳定。

5. 此前广深地区部分银行房贷停贷的现象已经扩展到长三角地区，上海多家银行总行及分支机构1月房贷额度吃紧，也有大行出现本月停贷的现象。

6. 前香港立法会议员许智峯表示，他已经就汇丰银行冻结户口一事，详细向英国外交事务委员会提供证词，而汇丰银行的行政总裁CEO祈耀年以及银行首席合规官Colin Bell两人在1月26日将就此事面对外交委会的质询。

二、世界新闻

1. 美国人权组织全球自由联盟1月24日发表了15个古巴天主教神父的联名信件《我已看见我的子民所受的苦难》（[文件](https://www.globallibertyalliance.org/global-voices-for-liberty/15-catholic-priests-in-cuba-write-open-letter-to-the-people-of-cuba-live-in-truth-and-freedom-reveals-our-real-power-as-citizens)）。信件号召古巴人民共同参与反抗共产主义的束缚，因为“选择在真理和自由中生活是我们作为公民的真正权力”。

2. 美国拜登总统1月25日宣布“购买美国货”行政命令（[文件](https://www.whitehouse.gov/briefing-room/statements-releases/2021/01/25/president-biden-to-sign-executive-order-strengthening-buy-american-provisions-ensuring-future-of-america-is-made-in-america-by-all-of-americas-workers/)），确保美国的未来由美国所有工人制造。行政令填补了美国联邦政府在“购买美国货”政策中的漏洞。拜登政府的这一举措封堵了中共公司进入美国政府采购清单的企图，此前中共认为拜登政府的上台会支持其这一主张。

3. 以色列本周宣布，在阿以两国签署由川普政府促成的历史性正常化协议四个月后，已在阿拉伯联合酋长国（UAE）开设了大使馆。同一天，阿联酋批准了在特拉维夫的使馆开幕。阿联酋是在《亚伯拉罕协定》下使以色列实现邦交正常化的四个阿拉伯国家中的第一个，随后是巴林、苏丹和摩洛哥。

4. 一个自称 “MAGA爱国者党 “的组织已在美国得克萨斯州正式成立。联邦选举委员会1月25日公布了该党全国委员会的组织通知。目前还不清楚该组织是否与川普或他的任何家庭成员或核心成员有联系。

5. 众议院弹劾管理人员1月25日向参议院递交的川普弹劾案。从而启动由100名参议员组成的参议院的审理程序。

6. 一枚火箭能单次搭载多少人造卫星登上太空的世界纪录再度被刷新，在美国佛罗里达州，143枚形状不同，大小不一的卫星，搭乘SpaceX猎鹰火箭发射升空，进入预定轨道。一台卫星的发射单价也已降为100万美元，为后续围绕空间卫星的商业活动铺开道路。

7. 1月25日，美国拜登总统正式恢复对来自巴西、爱尔兰、英国和其他26个允许跨越开放边界旅行的欧洲国家的非美国游客实施旅行限制。这项决定是新政府对抗中共病毒疫情计划的一部分。美国因疫情死亡的总人数已接近42万人。
