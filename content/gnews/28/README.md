11月7日，美国总统大选民主党候选人拜登预测自己胜出，在联邦选举委员会没有官方宣布选举结果的情况下，台湾总统蔡英文迫不及待地祝贺拜登当选美国总统，其所作所为令人瞠目结舌。

台湾一直以来都被看作是中国大陆民主、自由的样板，在防治中共病毒取得卓越成绩后更被看作是世界的楷模，让世人对蔡英文当局刮目相看。今年5月份蔡英文连任台湾总统之日，川普政府三位高官对其连任表示公开祝贺，蔡英文主政台湾以来在自由、民主、文化等方面，都得到了川普总统的鼎力支持，美国政府更是为了保持台海的和平稳定，使其有能力不惧中共的淫威，加大了对台湾军售。去年在美国政府的邀请下蔡英文作为台湾总统第一次以官方身份访问美国，同时今年美国也派出了副国务卿首次防台。种种迹象表明，川普政府对于蔡英文政府和台湾人民的支持达到了前所未有的高度，照此发展下去在不久的将来台湾即可得到美国等世界大多数国家的认可成为独立的国家,实现真正的独立。

本次美国总统选举，在官方并未宣布选举结果的情况下，蔡英文竟然抛弃恩人川普总统，迫不及待地向拜登表示祝贺，上演了现实版的农夫与蛇，真是令人大跌眼镜。爆料革命三年以来在文贵先生的引领下，在无我信念的感召下，正义的老百姓逐渐开智，认识到世界一切的恶源就是中共。硬盘门事件彻底揭露了拜登和中共国勾兑之深，涉及面之广，涉及金钱数额之大，所作所为之邪恶，已经远远超出了人类的底线，对此类信息蔡英文当局不可能不知道。在此背景下，蔡英文仍然执迷不悟，对自己及台湾人民恩重如山的川普总统恩将仇报，对邪恶势力代表人物的拜登跪舔，其丑态令人厌恶、其所作所为令人不齿。

但是蔡英文当局不能代表全部台湾人民，这次事件也让台湾人民和世界人民看清楚了蔡英文当局本来面目：要不就是隐藏很深的九层妖塔的第九层，要不就是所谓的墙头草两边倒，其最多只能算作一名政客，一个跳梁小丑。无论是哪一种情况，其必将遭到正义力量的抛弃、惩罚。
