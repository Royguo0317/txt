今天我来讲讲我的所见所闻，村干部暴力拆迁给老百姓带来的血与泪，家败与人亡。

旧城改造、城中村拆迁、新城规划，说的冠冕堂皇，不过都是他们敛财的幌子。市政府拍卖土地给房地产开发商，房地产开发商就要面临拆迁的问题，怎么办？

一定是找村干部摸排测量拆迁，村干部就会招揽村里的恶霸、地痞、流氓帮他们干这些活，许诺给这些恶霸的头目钱财、房产。

去你家里测量房子的面积，你不同意他们测量拆迁，提出抗议，那些村干部的爪牙就会对你家进行断电断水、辱骂恶言相向；

你与那些村干部爪牙发生冲突，就会被他们打伤打残，你去报案，村干部就去协调公安对你的案件进行冷处理，陪些医疗费了事，公安收了村干部的黑钱，更偏向村干部一方，本该是刑事案件也变成了一般治安案件，冷处理或者不给予处理，最后不了了之。

这些都是村干部背后指使的，给村干部和这些爪牙的钱和贿赂公安的钱，是房地产开放商出的。

每一家每一户，村干部最了解基本情况，家里男丁少、老实软弱的他们就用恶霸、地痞、流氓恐吓殴打，家里有上班人员他们就用你所在的单位给你施压，给你穿小鞋、恐吓你、开除你，直到你就范。

这时候的你如果还不同意拆迁，每个市里都会有拆迁办公室组织的综合执法大队，这些人都是从公安、城管、城建里抽掉的人，和村干部及村干部的爪牙们，对你进行强制拆迁，对你家进行包围，破门而入，用强行架走，殴打绑架的手段对你全家进行带离。

等放你回来的时候家里已经被这些土匪给夷为平地了，你去告状上访一点用都没有，官官相互，对你进行维稳，告你妨碍公务拘捕判刑，搞的你一点脾气都没有，最后只能忍受。

别以为这些就完了，前期房地产发商拿到地可以贷款，政府支持村干部配合，拆迁完毕后，所谓的给你几个月的租房子钱，对于一家三口还可以租到房子，家里有老人孩子的只能分开去租房子住，最后搞的亲情也淡了。

有的回迁房烂尾了，一等十年八年的比比皆是，老人等不到新房子气死了，最后家里是不欢而散，离婚、家破人亡，这些都是共产党欠下的血债。

可是那些市里的领导、村干部、恶霸、地痞、流氓已经收到了房地产开放商给予的金钱、房产、美女利益分成，老百姓在他们眼里就是待宰的羔羊。

投诉无门，碰到烂尾的回迁楼盘，围攻市政府、扯布条、拦路下跪喊冤，只能被公安维稳抓捕判刑。底层太苦了，强拆带来的血泪苦难，太多了。写着写着不由的就掉下了眼泪，新中国联邦让老百姓草根看到了希望，这些共产党欠下的血债，都要加倍偿还。
