## 中共發起所謂《全球數據安全倡議》，網民稱“賊喊捉賊”、”做了婊子想立牌坊”
`2020-09-11 11:15 pmzy` [轉載自GNews](https://gnews.org/zh-hant/348572/)

內新聞：瓦者

- ![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/11111452/image1-2.png)


2020年9月8日，中共發起《全球數據安全倡議》，提出全球數字治理應遵循的“秉持多邊主義、兼顧安全發展、堅守公平正義”三原則，為全球性問題提供解決方案。提出“各方應在相互尊重基礎上，加強溝通交流，深化對話與合作，共同構建和平、安全、開放、合作、有序的網絡空間命運共同體”，並且“呼籲各國支持並通過雙邊或地區協議等形式確認上述承諾，呼籲國際社會在普遍參與的基礎上就此達成國際協議”。

中共發出上述倡議，我們看看它自己是怎麽做的。中共對內壹直實行高壓維穩，進行大規模網絡監控和嚴格的網絡審查，設立防火墻，既沒有言論自由，也屏蔽世界的真相。同時，根據中共國的情報法，中共政府可以要求仼何中國公司提供任何數據信息，無論它在何處經營，該法律要求任何中國公司“支持，協助和配合國家情報工作。”它不僅輕松掌握本國國民的數據和隱私，也利用植根於海外的應用程序木馬，竊取全世界的數據和隱私，用於不可告人的目的。有證據表明，它有組織地對海外目標進行大規模的黑客攻擊。

面對中共的惡行，川普政府在行動。蓬佩奧國務卿宣布擴大清潔網絡，推出五項新行動方針。1.清潔運營商：撤銷和終止中共國電信及其他三家公司美國國內外的電信服務授權。2.清潔應用商店：讓不可信的中共國應用從美國應用商店下架，如抖音、微信等，它們充當內容審核的工具，並對美國國民個人數據構成重大威脅。3.清潔應用：阻止華為和其他不可信應用的預裝，不希望各公司在華為侵犯人權及中共的監視機構中成為同謀。4.清潔雲服務：限制中共國服務提供商在美國收集、儲存、處理大量數據和敏感信息，保護國民個人信息，企業的知識產權，包括CCP病毒的疫苗研究等。5.清潔電纜：努力確保美國和他國的海底電纜不被中共惡意利用，並呼籲所有熱愛自由的國家和公民加入清潔網絡計劃。

中共強烈指責美國的清潔網絡計劃是企圖在國際互聯網平臺復制霸權戰略，以“清潔”之名行“清洗”之實，打著“安全”的幌子圍獵他國領先企業。也指責美國政府的做法違背了互聯網全球互聯、開放、去中心化的本質，違背了技術架構的公平普及性。

中共這樣的言論其實就是賊喊捉賊。比如TikTok，該應用在美國的下載量已超過1.75億次，並且像其他應用程序壹樣，公司後臺可以訪問大量敏感的個人數據，包括上網和瀏覽活動，位置數據和搜索記錄。8月31日班農作戰室提到，TikTok主動收集手機裏關於“抖音”和”冠狀病毒“的信息，微軟能看到服務器上由TikTok發出的請求指令。到底誰在破壞全球互聯網的開放與安全？到底誰在用非法的手段危害他國公民的隱私和安全？

印度也已對中共國移動應用實施了三次封殺措施。第壹次， 6月29日中印部隊在加勒萬河谷發生流血沖突後，有59款中共國應用程序被列入封殺名單。包括抖音（國際版）和微信等火爆產品。第二次，7月份，有47款中共國生產的應用程序下架。這些產品大都是前壹批被封殺的產品精簡版或變體版。第三次，9月2日再次封禁118款產品。在封殺前兩天，中印雙方再次發生沖突。封禁中共國應用程序是壹個針對性的措施，目的是確保印度的網絡安全。

中共在網絡空間上的行為，必將遭到自由法治國家和社會的聲討和反擊。隨著真相的進壹步揭露，中共提出的所謂“網絡空間命運共同體”，將成為歷史的笑話，讓我們拭目以待。
