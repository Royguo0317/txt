# **海军陆战队在日本本土附近进行军事训练以对抗中共国**

鉴于目前持续紧张的中美局势，特别是在南海中国海地区，美国海军陆战队越来越多地回归其最初的两栖作战任务，让人想起二战期间在太平洋的岛屿战役。

据本周报道，这正是美国海军陆战队在日本本土岛屿上进行的训练类型。《华尔街日报》报道称：“海军陆战队正在日本加强训练，以应对西太平洋的岛屿冲突，使日本处于美国应对中共国军事挑战的前沿核心位置。”

[!\[\]()!\[\](https://spark.adobe.com/page/kSgcTqt3fA17k/images/66261890-9ee4-44ae-a31b-6854a7482c88.png?asset_id=951123ae-259f-459f-87e6-320797f50467&amp;img_etag=%2293ddabf87762fd209dcdc18bf90921e7%22&amp;size=2560)](https://spark.adobe.com/page/kSgcTqt3fA17k/images/66261890-9ee4-44ae-a31b-6854a7482c88.png?asset_id=951123ae-259f-459f-87e6-320797f50467&amp;img_etag=%2293ddabf87762fd209dcdc18bf90921e7%22&amp;size=1024)

演示图片，美国海军陆战队训练，Getty图片

报道详细指出，对小规模、快速部署的新重点将使美国军队和阵地更难定位。这与美国911后的“反恐战争”环境非常不同，反恐战争在阿富汗和伊拉克出现大规模的部署和占领。

报道引用驻北卡罗来纳州勒琼基地的海军陆战队第八团第三营的指挥官尼尔·贝里(Neil Berry)中校的话说：“我们正在远离营帐、电脑屏幕，因为1.)它位置固定，2.)它有很大的电磁信号。”

《华尔街日报》进一步详细描述了岛屿训练的实际情况：

在最近一系列的训练中，数十名海军陆战队员乘坐两架CH-47支努干直升机降落草丛中，随后便消失，紧接着日本士兵乘坐两架鱼鹰倾转旋翼飞机抵达。他们的模拟任务：避免被发现，重新占领一个在敌人导弹和火炮覆盖范围内的港口。

报道还说：“此次训练反映了新的重点关注于小规模分散部队和指挥中心，这样的目的是让敌人更难定位和摧毁。”

因此，除了模拟类似南海争议岛礁的小规模军事化岛屿环境（中共解放军在南海制造了一系列这样的岛礁），海军陆战队正在准备避免中共国先进的雷达和技术能力的探测。

特别是在去年，美国加强了空中训练和海军航行，在该区域（南海），同时在有争议的台湾海峡附近，以“自由航行”的名义演习，向北京发出警告。
