## 中共国企业20倍价差恐慌性大举囤集芯片

【日本东京方舟农场】撰稿：青衣     校对：文小律

![]()![](https://gnews.org/wp-content/uploads/2021/04/1EDBD3BD-F00C-46D1-BB9E-9CAD97BDD971.jpeg)

据《大纪元》4月22日消息，面对全球芯片短缺，中共国企业正开始大举囤货，甚至出现以高于市场20倍的价格买入芯片。

另据中共国媒体《第一财经》4月21日报道，三星电机的一名前负责人琼（June）透露，中共国半导体行业已出现恐慌性备货，“中（共）国的部分企业在全球市场上正在以最高20倍的价差收购芯片。”

然而因为芯片短缺，安全库存已经探底，目前的采购即使不考虑价格，持续下单，也无法确保能买到所需数量的芯片。

微控制单元（MCU）芯片去年每个价格为8美元，目前涨至50美元，价格是去年的6倍以上。芯片行业人士马凯表示：“相比之下，50美元的特殊MCU还是比较少，但通用MCU从几毛钱涨到几块钱的比较多，现在的行情普遍是涨价8到10倍，大家压力都比较大。”

而现在即使肯付高额价格，也未必能抢到芯片。《芯谋研究》首席分析师顾文军表示：“最近有大陆设计公司长期支持的境外一家代工厂也被砍单，被迫以比之前超高的价格购买，且只争取到300片。”

从目前的情况来看，“恐慌性”备货不仅仅发生在芯片的上游环节，在下游终端产品上，一些厂商也开始大规模囤货，包括手机厂。realme中国区总裁徐起表示，之前手机供应链备货一般大概够8个月使用，但他们已提前备了整年的量。IDC分析师王希认为，先进制程芯片的供应波动明显正影响手机厂的产品和采购计划。

而据《证券时报》的消息显示，有业界人士认为芯片短缺至少要持续到2022年。根据中共国海关的数据显示，去年中共国的半导体进口增加了15%，今年3月份更是达到创纪录的359亿美元。在业内看来，供应链的失衡已经进入了一种“常态”。

目前，芯片涨价再加上部分原料价格上涨已经造成中共国电子产品价格上调，而有的汽车公司已因芯片短缺被迫将产量下调。

自遭到美国芯片制裁后，中共国就陷入了“芯片荒”，芯片“大跃进”也以失败告终。因为缺芯，华为养猪去了；因为缺芯，很多加工厂歇业。中共国不仅因为被制裁失去了高端芯片，现在由于全球芯片短缺，连低端芯片的供应都已面临危机，而靠大举囤货又能坚持多久？缺了“芯”的中共国正逐步走向“石器时代”。
