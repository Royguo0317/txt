## 脑动脉瘤是常见病吗？

撰稿人 ：人文综合部 jhajha      审稿：Yumi

脑动脉瘤是一种严重的常见病，是导致蛛网膜下腔出血的主要原因，起病急而凶险，不及时救治易造成死亡，且治疗后并发症较为常见，影响生存质量，因此了解脑动脉瘤的发病机制，降低发病风险尤为重要！

脑动脉瘤是肿瘤吗？不是肿瘤，它是由于局部血管壁薄弱，长期血流冲击下和血管腔内压力增加，形成凸起膨出或囊性突出，形状上像是一块肿瘤，因而称之为脑动脉瘤。

一、脑动脉瘤形成的原因：1、血管壁局部的先天性缺陷；2、随年龄增长的动脉粥样硬化；3、曾经发生过的颅内感染；4、曾经发生的脑创伤；5、其他原因，如颅内肿瘤。这些因素与血管腔内长期压力增加的基础上引起囊性膨出。脑动脉瘤是造成蛛网膜下腔出血的首要原因。

二、脑动脉瘤与血管瘤的区别：

1、血管是由动脉、静脉和毛细血管组成，动脉和静脉的血管壁的构成基本相同，从内向外依次分为内膜、中膜和外膜，脑动脉瘤就是脑动脉血管壁的中膜缺失，血管壁薄弱，在长期血流压力冲击下，容易形成膨出或凸起。

2、血管瘤分为：毛细血管瘤、静脉血管瘤和动脉瘤。

动脉瘤是动脉血管壁的病变或者损伤，形成动脉壁局限性或者弥漫性扩张膨出，可以发生在任何部位，多以肢体主干动脉、主动脉、颈动脉常见。

3、脑动脉瘤是发生在颅内的动脉瘤，好发于颅底动脉环的大动脉分支处。部分由于先天异常因素造成，部分是由于后天长期血流冲击压力下造成的。随着年龄增长，动脉壁弹性降低，在血流长期冲击压力下形成。

三、脑动脉瘤有哪些症状：

脑动脉瘤患者在情绪激动，血压突然升高，或者剧烈运动，用力排便，咳嗽等诱因，突然发病，以数秒数分钟发生头痛最为常见。

（一）出血前兆症状：表现为轻微头痛、眼眶痛，动眼神经麻痹，此时请警惕随之而来的蛛网膜下腔出血。

（二）出血发生后症状：

1、由于局部血管破裂，血液流入蛛网膜下腔，在脑脊液中扩散，刺激脑膜，引起剧烈头痛、颈强直等脑膜刺激征。

2、血液流入蛛网膜下腔，增加了颅内内容物，颅压增高，继发脑血管痉挛、脑梗塞而出现偏瘫，运动性或感觉性失语，大动脉瘤会压迫视神经，出现视野偏盲。癫痫发作。

3、血液粘稠，影响脑脊液回流吸收，可发生急性交通性脑积水或蛛网膜粘连，使颅压急剧增加，进一步减少脑血流量，加重脑水肿，导致脑疝。

4、出血还可以刺激下丘脑，产生血糖升高、发热等内分泌失调和大汗淋漓等自主神经功能紊乱的症状。

5、颅内神经压迫症状：单侧眼睑下垂、单侧脸麻木，瞳孔放大和畏光，视网膜出血，眼球不能上下移动和内收。

四、分类：国际采用的Hunt五级分类方法，根据症状轻重，将病情分类，评价疗效，选择血管造影和手术时机。

一级  无症状，或有轻微头痛和颈强直。

二级  头痛较重，颈强直，除动眼神经麻痹外，无其它颅神经症状。

三级  轻度意识障碍、躁动不安和轻度脑压迫症状。

四级  半昏迷、偏瘫、早期去脑强直和植物神经紊乱。

五级  深昏迷、去脑强直、濒危状态。

五、治疗：

1、非手术治疗：主要目的是防止再出血和控制脑动脉痉挛。

2、手术治疗：发生了蛛网膜下腔出血，应及时手术，手术分为开颅夹闭术和血管内介入栓塞治疗，

3、特殊类型动脉瘤的治疗：多发脑动脉瘤治疗和巨型脑动脉瘤治疗，

六、并发症：1、再出血：出血后24小时内再次出血的几率最大，发病一个月内再出血几率较高，高龄、昏迷、血压大于170mmHg再次出血风险较大。

2、继发脑血管痉挛，表现为意识改变，局部神经功能损害（如偏瘫、失语）。

3、脑积水，精神障碍，步态异常、二便异常。

七、预后：与年龄、血压、术前有无合并其它疾病、脑动脉瘤的大小、位置、有无血管痉挛等因素有关。

脑动脉瘤是常见病，由于病因不明确，所以没有办法预防，但是保持情绪稳定和保持血压正常很重要，切记不要情绪波动太大，继而影响血压升高，造成脑血管破裂蛛网膜下腔出血！
