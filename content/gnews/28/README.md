中共官媒央视新闻网的《国际锐评》栏目2月４日发表题为《新疆好不好？眼见为实》的文章。文章说，一份有关新疆棉纺织行业的调研报告在国内外受到广泛关注。报告指出，参与调研的93.2%的员工是通过现场招聘会、亲友推荐等方式自愿选择进入棉纺织企业务工。同时，**100%的员工说可以在上班期间自由喝水和如厕**。

这篇粉饰中共的文章实在是用力过猛。100%的员工上班期间可以自由喝水和如厕，这是人类最基本的生理和隐私需求，竟被党媒作为人权功绩来宣传。又如“在劳动合同签订时，企业会为少数民族员工提供汉语与维吾尔文字的双语合同，劳动者的权利依法得到充分保护，何来美西方政客所污蔑的‘强迫劳动’？”。文章此地无银三百两的说明让人第一反应觉得可笑之后不由自主地反思：中共究竟以什么样的人权标准对待当地人？他们的劳动合同是否在本人自愿的条件下签定，在劳动过程中能享受到基本人权就是自由喝水和上厕所？

文章还写道，当地一家饲料厂某员工托现场记者给美国前国务卿蓬佩奥带个话，说“新疆所有人都很讨厌他，请他不要再撒谎骗人了”。新疆所有的人怎么可能都知道蓬佩奥先生呢？这是典型的党媒意识形态宣传谎言！
