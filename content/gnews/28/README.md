**西班牙2021年5月6日电/西喜社——**

近几天，中共国大陆有个大学生“为了爱国，要杀光6亿穷人”的“公开信”遭到众多网民“炮轰”。人们对这种“恶魔”，既感到可怕又可恨！只有去年沈阳那个叫赵盛烨的青年教授要让整个人类与中国“同归于尽”可与其“媲美”。

其实这并非“发生”在今年的事。你上网一搜就知道。这是去年10月7日，一名自称“广西艺术学院20届学生”石可，发“给国家和社会的一封公开信”，标题是《我愿意对穷人开杀戒，包括我的父母》。当时这名学生所在的学校就出来“辟谣”了，说有人盗用他们学校学生信息，发表极端言论。然而，两百天过去，一直没有下文，这样，也就成了“此地无银三百两”。果真如那所学校辟谣所言，“被盗用信息”的当事者石可为什么不出来坚决否认？

看来只能认定是“好汉做事好汉当”。从这点而言，石可像个男子汉，尽管像个恶魔一样的男子汉；而那学校出来“辟谣”，就显得特虚伪，很可能就是为了“息事”而扯谎。

现在事情已过去大半年，不知怎么回事，这几天又被网友扒出来“炮轰”。很多我想说的话，网友们在文章中都已说过，自己只想说几句网友们没有注意到或没有说的。

这个大学生要“杀”的六亿人都是穷人，且是月收入不足一千元人民币的极端穷人。好在尽管这则短文的作者也是个穷人，可毕竟每月退休金远高于这个数，应该不在石可所“杀”之列。这样，本人也就可以继续苟延残喘，做点小文章，而不用害怕这个大学生来杀俺了。

那么这六亿穷人又都是些什么人呢？我这个普通民众当然不可能去调查，但可以做合理猜测。我自己认为这六亿人应该多是乡下老人（未成年人肯定不能计算在内），而这些乡下老人，他们绝大多数年富力强时，即使“吃糠咽菜”，勒紧裤腰带，也是卖力地辛勤劳作。遗憾的是，由于说不清道不明的原因， 这些人一直没能“富起来”。既然年轻时就是穷人，老了之后，既不能像台湾那样与城里退休者享受同等退休金待遇，也不像大陆城里人，只要在一个即使是很一般的企业（事业单位更不用说了）干到退休，每月退休金也不止一千元人民币。这样，即使说不上富，至少不会被这个叫石可的学生列入“杀光”的“名单”中吧。

这样一说，也就清楚了：这被因“爱国”厌恶到要“杀光”的六亿穷人，其实，或者，很可能也辛勤劳动大半生，只是由于这样或那样的原因，没能摆脱贫困。这原本已经让这些人感到很悲哀了，可他们万没想到，就因为穷，现在竟然连小命都难保。真是悲哀透顶！

然而，这六亿人的“穷”真的就完全是他们自己的责任吗？说到这里，不知道这个叫石可的孩子在大学里学的什么“艺术”，平时是否也读点课外书，而所读书中有没有三百多年前一个叫洛克的英国人出版的《政府论》，明白什么叫国家？什么叫政府？什么叫人民？这三者又是一种怎样的关系？特别是结合现实，咱很想知道：是六亿穷人拖累了国家，还是国家对不起这六亿穷人？如果是后者，这个叫石可而学艺术的大学生又“该当何罪”！

而那所艺术学校呢，尽管可以“为艺术而艺术”，但做为一个现代高等学府，除了教学生学习艺术，是不是也还应该让他们懂得最基本的道理？什么是最基本的道理？除了孩子不能对父母“开杀戒”外，还应该明白本人上面所讲的国家、政府和人民的关系。即使一个学艺术的大学生，若连这些最基本的道理都不懂，那么将来即使艺术学得再好，谁也不敢保证他们什么时候会犯起傻来，犯下反人类或反人民的滔天大罪。

我这样说，大家也不要大惊小怪，不是没有“先例”：去年一个已经三十五岁的沈阳年轻教授赵盛烨如果大权在握，又把他惹生气了，他会不会在“一气之下”，真的要整个人类与中国“同归于尽”？在这种人心里，是不存在“反人类罪”的，更不懂什么叫“反人民”。

说到“反人民”，在新政权统治下，其实一直有这种“罪名”。几十年前，这个国家专政机关，在提到某些坏人包括许多受到冤枉或被诬蔑的“坏人”时，最喜欢用的一个短句就是：某人“反党反人民”。几十年前自己不懂，只能姑且信之。后来，尤其是到了现在，知道动不动说一个人“反人民”，完全是胡说八道。如果说有些所谓的“坏人”“反党”，我倒相信，因为他们对中共有意见而又不许说出来，久而久之，憋的时间长了，自然会生出“反”的思想。这种现象，按人性来讲，很正常。但说这种人也“反人民”，就很难说得通：他为什么要“反人民”？他“反人民”干什么？他反的“人民”又是什么样的人？

不过，自从去年出了赵盛烨，而现在又出了个要“杀光六亿穷人”的大学生，觉得“反人民”的人确实存在，比如，如果说这个叫石可的大学生“反人民”，你会觉得他冤枉吗？他要“杀光”的六亿穷人，100%是“人民”，既如此，这不是“反人民”又是什么？

在今天的中共国大陆，你还能找到像那六亿穷人一样的“穷官”吗？可见，统治者一直在说谎，一直在欺骗，并且还收到了很好的“效果”！要说悲哀，这才是这个国家政权的最大的悲哀！
