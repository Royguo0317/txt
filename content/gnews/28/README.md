## 【热点播报】由于临床实验出现不明原因疾病，强生公司Covid-19疫苗研究紧急叫停！

作者:香草山写作组    正道主义联盟

审核：浪迹天涯
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/12234350/10122-1.png)
STAT媒体证实了该研究暂停的消息，说这是由于“一名研究参与者不明原因的疾病”。该公司拒绝提供更多细节。

强生强调，所谓的不良事件——疾病、事故和其他不良医疗结果——是临床研究的预期部分，并强调了研究暂停和临床暂停之间的区别，临床暂停是一种正式的监管行动，可以持续更长的时间。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/12234409/10121-1.png)
媒体指出，虽然临床研究中出现暂停并非罕见，但疫情期间的疫苗研究关注度明显更高。而且，强生公司的此次研究规模达到了6万人，研究的一位消息人士称，在这种规模下，即使这次的问题已解决，也可能出现新的状况。

9月8日，阿斯利康公司(AstraZeneca)和牛津大学(Oxford University)正在开发的另一种Covid-19疫苗的大规模研究被搁置，原因是英国一名患者疑似出现不良反应。据信病人患有脊髓横向脊髓炎。疫苗的研究在英国暂停大约一周后又重新开始，此后其他国家也重新开始。

美国强生公司曾在9月23日高调宣布，该公司研发的新冠候选疫苗进入三期临床试验阶段。在获得试验批准后，强生成为第四家获得美国政府Operation Warp Speed支持的三期临床试验制药商。另外三家分别是Moderna，辉瑞和阿斯利康。

不过阿斯利康也有一则坏消息，据路透社援引美国卫生部长的消息称，阿斯利康的新冠疫苗在美国的试验仍被搁置，美国食品和药物管理局（FDA）正在调查导致该全球试验终止的患者疾病。
