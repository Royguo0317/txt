**布莱特巴特：北京宣布抗疫胜利后，面临又一轮冠状病毒爆发**

![]()![](https://spark.adobe.com/page/YgsZ9ud4RbbMK/images/03acfd4c-3e44-4e54-a51e-1ce3f04c07c0.png?asset_id=ebc609dc-45d4-45c8-8695-fedb8b8cb890&amp;img_etag=%22b5b4d774f2e9fc9b92b6d4e219d7bf06%22&amp;size=2560)2020年12月27日星期日，在北京街头支起的帐篷前，群众排起长队等待接受冠状病毒检测。政府敦促居民在2月的春节假期期间不要离开市区。北京在上一周确认了数个病例之后，开始实施了新的限制措施和大规模检测。

据报道，周六在北京国际首都机场附近的顺义北部，发现了五例新的冠状病毒病例，北京市立刻进入了“紧急应对模式”。

可能还有第六例，但尚未得到官方确认。

《南华早报》援引北京市政府发言人周日的讲话说：“有许多零星的爆发。疫情控制形势十分严峻。各区、各机构和单位都必须进入应急响应和战斗状态…….加大防控力度，制定更加严格的防控规则来阻止病毒的传播。

据称，这些应急措施包括更严格的口罩要求、对中共病毒的大规模检测、旅行限制以及感染区域的隔离。据说仅在星期六，就有超过100万居民接受了检测。一些公共景点已被关闭或被指示减少运营，并且会持续到元旦以后，同时学校也可能会关闭。

“谁组织旅游，谁就负责管理和执行疫情控制规定。旅行社需要控制出入北京的旅行团规模，进行风险评估并起草应各项急预案，”市文化旅游局的官员说。

路透社指出，周日是北京连续第四天报告有新的病毒感染确诊案例，其中至少有一人去过另一个城市并在当地被确诊。

“中国基本上控制住了冠状病毒，但在少数城市中，零星的病例依旧再次出现。当局计划从2月11日起，在为期一周的农历新年假期之前，为5000万高危人群接种疫苗，”路透社援引中共国国家媒体的话写道。

![]()![](https://spark.adobe.com/page/YgsZ9ud4RbbMK/images/4f3bd464-865a-4a27-9e6b-236e19ca7678.jpg?asset_id=1accd791-cec0-4582-955c-84c887ef6409&amp;img_etag=%223f7db0e52cf41277eb5f7921eec12b82%22&amp;size=1024)（网络图片）

北京官员还禁止了居民旅行和公共聚会，并且一直会持续到春节之后。春节是中国最重要的节日，通常会有大量的旅行度假和家庭团聚。

《南华早报》还报道说，北京的市民，特别是那些不方便在家工作的市民对“紧急应对模式”的限制感到非常不满。大瘟疫使北京本来已经难以忍受的通勤状况变得更加糟糕，同时也增加了在这个生活成本高昂的城市挣钱的难度。许多不满的居民指出，最近一轮冠状病毒感染的人群中，包括了很多不但要辛苦的加班工作，而且还要在上下班花费很长时间的人们。

中共庞大的网络检查人员迅速在网上删除了这些抱怨和不满。《南华早报》指出，在中共国版的Twitter——微博上，一个以前很受欢迎的、抱怨北京现状的话题标签，现在已经被清零。

尽管中共国在六个月前就宣称消灭了中共冠状病毒，但北京和其它大城市此前都陆续报告了病毒案例“激增”。上一次北京启动同一级别的应急响应和“战时”措施是在6月份，那一次大约有100个病例。
