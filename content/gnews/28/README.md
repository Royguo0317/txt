## 【拆台新闻】银行贷款结构优化的谎言
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1290119/)

**撰稿：脐橙君**
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/01193142/21.jpg)
来自央视网的消息，银保监会今天（6月1日）公布，截至4月末，金融机构对实体经济发放的人民币贷款余额为180.79万亿元，同比增长12.7%。其中，制造业今年新增贷款1.1万亿元，科技创新、绿色发展等领域信贷增长也较快。4月末，银行业房地产贷款同比增长10.5%，增速创8年来新低，贷款结构更加优化。

何为“优化”？就是将原来和现在的贷款结构比例分布做比较，同时明确贷款风险可控的比例比先前要高，而且违约贷款的比例降低，这叫“优化”。

但在中共国，何为“风险可控”的贷款，本就是一个变量。银行贷款中的所谓风险，就是根据当下的经济环境，对不同行业做出的评价，这是一个动态的标准，因为每个行业都是有周期的，而且在中共国，最大的周期来自于政府的行政指导。例如，曾经一度光伏行业受国家之力支持的时候，太阳能这个领域就成为政府追捧的领域，那么对于银行等金融机构来说，这个领域的企业自然就处于贷款发放中的绿色通道中，相对审核比较容易通过，那么自然对于光伏领域的贷款比例就会相对提高。但之后，对于该领域的打击是180度的大转弯，一下从天堂掉入地狱，银行看到政策转向后，贷款指南也立即发生变化，降低该行业的贷款规模，甚至对于一些小的企业甚至停贷或者抽贷，这无疑对有贷款的光伏企业来说，是雪上加霜，政府的打压加上银行的无情，让许多光伏行业的中小企业死去，那么造成的该领域的坏账率上升也是必然的事。因此，从统计的口径来看，因为对该领域的放贷规模降低了，就可以美其名曰地说结构优化了，但坏账率增加了多少，这部分坏账对整个贷款结构有什么影响，却从来不会出现在银行的对外公布数据中。

同理，从今天的这段新闻中我们可以看出这样一个逻辑，因为对于房地产贷款的规模增速所有降低，就得出“贷款结构优化”的结论，也同样可笑。增速降低但还是在增，这是一个基本的常识。而且在中共经济濒临崩溃的时候，房地产作为经济的支柱行业，像续命针一样苦苦支撑着中共的经济，一边知道不能对房地产政策进行松绑，因为会加速经济崩溃，另一边却不得不继续支持着房地产企业融资，因为突然停贷会抽贷，无疑就是经济自杀。

我们从中共监管部门对外的报告中，看不到一丝一毫令人信服的数据，反而漏洞百出，逻辑混乱。中共也知道自己的经济千疮百孔，但不得不说谎话来麻痹无知的百姓，不然当百姓成群结对地觉醒过来，那么中共的谎言就会像多米诺骨牌一样一个个倒下，停都停不了，那种溃败就是致命的。
