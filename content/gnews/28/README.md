法新社3月25日报导：200多艘中国渔船集结在菲律宾专属经济区内的牛轭礁周边，据信船员是中国海上民兵，菲律宾对这项大规模且具威胁性涌入提出外交抗议。罗克说，杜特尔特再度向中国驻菲律宾大使黄溪连强调，2016年菲律宾在著名的南海仲裁案取得胜利。当时荷兰海牙常设仲裁法院在2016年时，就菲律宾控诉中国的九段线主张违反联合国海洋法公约所做的宣判，裁定菲律宾胜诉。美国、日本和加拿大也对中国的船只表示担忧。

评论：

每年经由南海的贸易额高达3.4兆美元，中共深知这是巨大的蛋糕并想据为己有，所以中共国近几年大力发展海军想夺走南海的控制权。《国际法》不允许国家武装力量对渔船开火，中共就利用此法律漏洞将民兵安插在渔船上，不断干扰南海周边国家渔民们的正常生活。这就是典型的中共流氓行径，中共正在用挑衅的方式逼这些东南亚国家加入印太联盟，未来吃亏的依然是中共自己。
