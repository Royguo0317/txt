新闻媒体是在每个人生活周边消息来源的重要取向，及讯息来源，当有了正确的讯息你就会往正确的道路走，当一直获得假讯息及错误的讯息你也会一直走在错误的道路之上，讯息会改变的思考及你的判断，只有你要出他们所引导的事物，去实施验证你才能拿到真相，做出正确的选择。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/27220753/%E5%9C%96%E7%89%87-1-87.png)
新闻媒体在播报的当下或之前之后，不论是总统、某部长、国台办、商业大老等相关来告知或解释新闻的，你报导出来的结果，文章的内容，报导的内容，都是影响着你的观看者的判断及思绪，你在这个当下你以做出了了选择，走向正义或是走向邪恶，你是否摸着良心来看待过这件事情，你是否知道你的文章报导播出后将会造成什么影响，会持续的影响，你无法抹灭你造成的伤害，你无法抹灭你曾经的恶，除非你将你的良心找回来，走回正确的道路上，不然这些你想解释的，都只是一个无意义的文章、报导，因为你用人民对你的信任”媒体第四权”，来破坏人民的信任，这才是最可恶的。

传播真相，是现在最重要的事情，要将这些我们曾经信任的”媒体第四权”导正，让他们醒悟，了解你们也是因为拥有了人民的信任才有这一切，真相才是你我的需要，也是媒体的基础。
