## 我们需要有探求真相的精神和勇气
` 国内新闻组` [轉載自GNews](https://gnews.org/zh-hans/772497/)

喜马拉雅-国内新闻组 伍哥 图片：山口君 编校：老顽童2017

2021年刚刚开始，我便有幸成为一个准感染病毒的人，被居家隔离14+7天。

都说境由心生，但有时还是不能尽如心愿。本来已经足够小心防范的，生怕被感染尽可能地减少外出，岂料人算不如天算，确诊人员一小时前到过的柜台，我也去这个柜台交费了，他们根据大数据就这样被隔离了。

隔离，就意味着我存在感染病毒和携带病毒的危险，名义上是为了我的健康，其实本质上是为了不去传染别人，是为了保护他人而剥夺限制了我的自由。并且由我来承担一切不利后果，唯一的福利就是免费测一次两次的核酸。相比那些被集中隔离全部自费的人来说，我还是很幸福的，不用每天交住宿费、餐费、检测费，好像自己赚了大便宜，从经济上看，被隔离者不仅增加了超预算开支，同时也减少了收入，算是落难群体了，可是对我们这样的困难群体怎么没见墙国政府给予点儿补贴，相反看到墙国向国外四处捐款援助免债，动辄上亿美元，每天教育我们爹亲娘亲不如党亲的党妈，让我如何说服自己来爱你？

没有人想也没有人愿意被隔离，一旦被隔离了，往往第一时间的反应就是对阳性病毒携带者的抱怨，埋怨他为什么哪都去？是不是他们不应该携带病毒出入哪里呢？他们哪里知道自己核酸呈阳性，何况还有无症状之说，谁闲着没事儿去花钱做核酸检测，并且检测费用相对很多人来说那是够自己和家人吃几顿饭的，怎么可能花钱找病。勤俭节约的国人即便有病也不舍得花钱看医生，再说了核酸检测也并不是哪个医院都能做的，签约的家庭医生都生活在统计局的数据库里，根本指望不上。他们不知道自己是病毒携带者，他们去哪里那是他们的自由，我们没有任何权力去责怪他们，他们也是无缘无故被感染的人。

思来想去，结果发现大家都是无辜的，我们都是受害者，那么谁应该对此负责呢？如果这个病毒是有人公然违法国际法偷偷研制出来的，如果这个病毒是有人故意或过失泄露的，如果有人故意隐瞒或掩盖事实真相，那么这个人（或者团体）应该对我们包括全世界的受害者负责，追究他们的反人类罪行！唯一能够做的是—我们需要有探求真相的精神和勇气！
