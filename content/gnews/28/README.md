## 重磅：FBI和司法部陷落 “通俄门”相关文件公之于众

美国参议院司法委员会主席林赛·格雷厄姆（Lindsey ·Graham）周五（1月15日）公布了有关“交火飓风调查”的一系列文件和文字记录。“交火飓风”调查行动是联邦调查局2016年至2017年就川普竞选团队与俄罗斯方面关系（简称“通俄门”）所作的早期调查。格雷厄姆称，这个与“通俄门”相关的早期调查，是FBI和司法部历史上最无能和最腐败的调查之一。

格雷厄姆在周五的一份声明中说：“我认为这个‘交叉飓风’（Crossfire Hurricane）的调查是该部门高级领导层的一次系统性大规模腐败，但这些不能否认联邦调查局和司法部在保护我们国家时的敬业，也不能否认他们是勤奋的爱国者。”他说，他的委员会已经发布了“尽可能多的材料”。但他同时指出，“一些机密材料仍未公布。”

格雷厄姆说：“我认为‘交火飓风’是联邦调查局和司法部历史上最无能、最腐败的调查之一。”他还指出，他“感谢所有参与宣誓作证的人和他们的坦率。他们为我们绘制了一条进行体制改革的道路”。

格雷厄姆指出：“《外国情报监听法》（Foreign Intelligence Surveillance Act，简称FISA）法庭被欺骗了。关于被调查者的无罪信息被扣压。相关调查人员，除了少数的例外，都有着令人难以置信的偏见，他们为了政治目的而使用了手中的执法权力。”“被调查对象的生活完全被搅乱了。我希望，此类反间谍调查能够得到控制，这种情况在美国绝不应再次发生。”

格雷厄姆猛烈抨击了前局长詹姆斯·科米（James Comey）和前副局长安德鲁·麦凯布（Andrew McCabe）领导下的联邦调查局，称其如果“不是非常无能”，就是“他们在故意纵容巨大的不当行为”。

他还表示：“除了川普竞选团队与外国势力勾结这个结论之外，他们对任何的其它解释都视而不见”。

格雷厄姆还抨击《外国情报监视法案》针对前川普竞选助手卡特·佩奇（Carter Page）的监视行动授权是“一种拙劣的歪曲”。他指出，签署这些授权令的前司法部官员，比如前司法部副部长罗德·罗森斯坦（Rod Rosenstein）和前代理司法部长萨利·耶茨（Sally Yates）“已经承认，如果他们早知如此，他们就不会签署”。

格雷厄姆说：“调查在应该停止的时候被推动了。唯一合乎逻辑的解释是，调查人员希望得到自己的结果，因为他们存在偏见。”

格雷厄姆强调，他将继续推进反间谍调查及并批准申请方面的改革。他希望“民主党和共和党的同僚能够在这些问题上找到共同点”，同时敦促联邦调查局局长雷“继续他已经启动的改革”。

他说：“很难相信像‘交火飓风行动’这样的事情会发生在美国。”“底线是今后在涉及政治调查时，我们必须有更多的制衡。我们必须在监视许可的申请上指定更多的有意义的签署。我们需要恢复美国人民对这个体系的信任。”

**背景资料：**

联邦调查局于2016年7月启动“交火飓风”调查行动，该调查是联邦调查局就2016年至2017年期间川普竞选团队与俄罗斯方面关系所作的早期调查，调查结果交给主持“通俄门”调查的米勒。2017年5月，司法部委派米勒就川普竞选团队在2016年大选期间涉嫌“通俄”以及干预司法展开调查，简称“通俄门”。米勒于2019年3月完成调查，认为川普竞选团队没有在选举期间“通俄”，但在川普涉嫌“妨碍司法”问题上未得出明确结论。2019年5月，米勒宣布离开司法部，并关闭特别检察官办公室。俄罗斯政府多次否认干预2016年美国总统选举。
