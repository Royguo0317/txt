## 九大糧食產區廢了八個半 還不備糧？
`2020-07-31 08:54 GM77` [轉載自GNews](https://gnews.org/zh-hant/281330/)

作者：文小明
審核：Lori文噠

眾所周知，地大物博的中共只有九大商品糧食基地，它們分別是：太湖平原 、鄱陽湖平原 、洞庭湖平原、江漢平原、江淮地區、成都平原 、松嫩平原 、三江平原、珠江三角洲。

目前由於南方長江流域一帶罕見的暴雨水災，不但讓5000多萬人流離失所，更是讓整個中國人民的七大糧食產區顆粒無收。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/31084809/6-12.png)

中國南方多省發生嚴重洪澇災害的同時，位於東北的[遼寧](https://www.aboluowang.com/tag/%E8%BE%BD%E5%AE%81_1.html)省遭遇69年來最旱夏天。作為糧食基地的三江平原受到了巨大的威脅！

關於黑龍江的松嫩平原我在“東北糧食基地出大事了”（ [https://gnews.org/zh-hans/205626/](https://gnews.org/zh-hans/205626/)）一文中指出，**黑龍江地區受疫情影響明顯，大量的百姓不願也不敢隨便出門，種植成本明顯提升，供應鏈的停滯，造成主要集中在化肥、農藥價格大幅上漲、人工成本明顯增長兩個方面，像人工施肥從上一年的150-200元/畝費用上漲至當下的300-350元/畝，農藥的價格同比也有15-20%的上浮。截至目前黑龍江大部分地區種子銷售進度與去年同期相比，僅有50-60%左右，拋開天氣等導致春播延後的因素，單從種子銷售這個進度數據來看，黑龍江地區玉米整體的播種面積較上一年度大幅調減已經是不爭的事實。**

中共國內正“享受”著水火兩重天的“酸爽”的感覺，目前有新聞報道阜新和錦州等地農作物大面積絕收，樹木也因缺水雕零，滿地黃葉如同提前進入秋天。

但是中共水利部7月29日消息，從6月1日到7月27日，遼寧省平均降水量108.8毫米，較常年同期偏少53.1%，較去年同期偏少20.6%，是1951年以來同期降水最少的一年。

截止7月28日的統計，全省作物受旱面積1,792萬畝，旱情主要集中在西部的阜新、錦州、營口、葫蘆島、朝陽、盤錦，以及[沈陽](https://www.aboluowang.com/tag/%E6%B2%88%E9%98%B3_1.html)和鐵嶺等地區。

遼寧西部的春夏多幹旱天氣，因此抗旱的玉米是當地種植面積最大的農作物之一。七月底本應是玉米抽穗拔節的時候，但由於近兩個月沒有有效降雨，大面積玉米沒熬過抽穗期就已經面臨絕收。

阜新馮家鎮林家村雖在7月29日下了二十多分鐘雨，但雨量僅能打濕地面，大量玉米旱死。距離林家村44公裏的滿堂紅鄉情況相同，29日的雨只打濕地表3厘米的泥土，3厘米以下全是幹土，當地有農場種植的玉米三分之二面臨絕收。

遼寧多地的大面積玉米沒熬過抽穗期就已經面臨絕收。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/31084831/7-12.png)

錦州因旱情造成的受災人口近70萬。當地村民表示最起碼有百分之五六十的農作物將顆粒無收。 “現在下雨也不行了，現在這種秧，牛羊都不會吃，這種是有毒的。”

目前大連大量家庭的飲水剛好夠用，澆菜園子的水需要從上面的屯子拉，玉米地則根本無法澆，再拉水也不劃算了。

此外，當地河水枯竭，楊樹因缺水開始枯黃掉葉，滿地的落葉顯出秋天的景象來。

另據中共[央視](https://www.aboluowang.com/tag/%E5%A4%AE%E8%A7%86_1.html)網報導，阜新梅力營子村有近14,000畝耕地，由於入夏後幾乎沒有降雨，全村耕地重旱面積超7,500畝，絕收面積超6,300畝。

有當地民眾在網上說，“真的遼寧幹旱太嚴重了，單位自來水只有一樓有，再往上都沒水，水壓不夠，天還熱，南方分點雨水給北方多好，哎，太難”。

“南方的雨挪到我家這下下吧，一直都沒下雨，我們這的莊稼都黃葉了，今年又收不到糧食了，我們飲水都困難了”。

“真的快旱不行了，村裏飲用水好久沒來了，沒有井的住戶太難了啊！有井的也抽不上來水，水線太低了，玉米都絕收……老百姓太難了太難了”。

遼寧是我國重要的糧食產區，其土壤肥沃、晝夜溫差大、生長期等因素使得這裏的作物品質優良，遼寧地區主要糧食作物有水稻、玉米、高粱、小麥、谷子、大豆等，2019年遼寧糧食產量達486億斤，占全國40%。由此可見，遼寧旱災對於糧食作物的影響深刻的關切著“中國飯碗”。

尤其是今年玉米價格本就持續上漲，大家都在等待新豐收的玉米可以沖擊市場，如果遼寧的玉米被嚴重影響，下半年玉米的價格恐怕就更加不容樂觀了。除了玉米，小麥和大豆也是今年糧食價格的波動區，小麥先漲後跌，大豆也受到進口沖擊，遼寧的小麥、大豆產區如果產量不理想，想必也會帶動小麥和大豆價格的一波上漲。
