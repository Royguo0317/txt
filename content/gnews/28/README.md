## 谈感恩和感谢

**作者：京都富士会 华夏的文丽**

日本人在吃饭之前，会双手合十，说一声itadakimasu ，然后才开始吃饭。这一句话中文的意思是说我要开始吃饭喽，我要接受大自然的馈赠咯。这句话表达了对两方面的感谢，第一是对植物和动物这些食物本身的感谢，因为人们享用了动物和植物的身体。第二是对制作食物的人的感谢。这个民族虽然政治上也很黑暗，但老百姓却拥有感恩的心。

同时，在日本的一些公共场所，尤其像图书馆之类的橱窗里，总是会贴满各种各样的招募义工的海报。比如清扫树叶义工，比如伴聊义工，比如防灾演习训练科普义工，比如给小朋友读小人书的小老师义工。总之，就是任何人都可以做义工。只要在自己的时间和身体状况允许的情况下，都可以做力所能及的事情，来温暖和回馈这个社会。

谢谢，只是一句很平常的话，很多人都会在幼儿园时期或者入园之前就知道这句礼貌用语。不管别人对自己是否有帮助，当别人提出建议或者意见的时候，说一句感谢，首先是礼貌，然后是修养和心态的表现。但是，现在很多孩子对父母已经不说感谢了，得到的宠爱和关怀好像是理所当然，还要求更多。初中生高中生开始，拥有智能手机，同学之间互相攀比，还要穿戴名牌服饰，索求更多更好的物质生活，而忘记对父母说声谢谢。在网络上，还看到很多年轻的孩子，对自己的父母有超过父母经济承受能力以外的奢侈要求。当父母无法满足的时候，孩子对父母恼羞成怒。看过感到痛心！攀比之风非常恶劣，是从什么时候开始这些代表华夏名族未来的孩子们，变得这么陌生，这么可怕。谁在引导一代人的价值观，消费观，世界观？ 将来一定让孩子们知道评判一个人是否有魅力，不要看其外表有多么完美，而要看对方是不是有一颗感恩之心。有感恩之心的人，一定会和颜悦色地对待周围的人和事。常常微笑，相貌自然不会差。值得去爱的人，首先要对父母有感恩之心，其次去回报这个社会。当然这个社会环境要做到人们相互友好、相互信任、齐心合力，需要一个很漫长的过程。在ccp的恶意引导下，年轻一代变得很浮躁，对任何人都充满猜疑，没有安全感。新中国联邦将要通过每一位的努力，把感恩、感谢、信任、合作，关怀……这些原本美好的气氛带回中华大地。
