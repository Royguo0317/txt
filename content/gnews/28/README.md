**西班牙2021年1月29日电/西喜社——**根据官方周五公布的初步统计，中共病毒大流行拖累西班牙2020年经济产出较上年下降11%。

年末，这个使用欧元货币国家中的第四大经济体在连续增长六年后首次出现萎缩。与2018年相比，2019年的产出增长了2%。

在4月至6月季度增长16.4%后，2020年最后三个月经济增长微不足道的0.4%，主要受内部消费和投资驱动。10月至12月的微弱增长让一些预测即将回归衰退的分析师感到惊讶。

西班牙统计机构INE也表示，2021年1月通胀率较2020年同期增长0.6%，这是自中共病毒冲击西班牙以来首次出现物价上涨。
