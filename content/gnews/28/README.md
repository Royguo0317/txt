**西班牙2021年1月24日电/西喜社——**夏威夷州报告称，其12月的酒店入住率与2019年同期相比下降了一半以上。

夏威夷旅游局数据显示，受中共病毒大流行影响，上月该州23.9%的酒店客房爆满，与2019年12月相比下降了56个百分点。

据《檀香山星报》报道，12月夏威夷各主要岛屿的入住率都在20%上下，唯独考爱岛降至13.4%。

在美国，只有华盛顿特区的酒店入住率低于夏威夷。

总部位于田纳西州的STR公司负责住宿观察的高级副总裁扬·弗赖塔格(Jan Freitag)表示，夏威夷严格的中共病毒旅行限制对一些人来说是一种阻碍，但对其他人来说也是一种吸引力。

他说，人们越早感到舒适地回到飞机上，行业就会越早恢复。
