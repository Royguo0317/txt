中共病毒一定會二次爆發。次爆發死人會驚人。共產黨控制不住自己製造的病毒了。北京可能是一兩個病例嗎？全世界疫情都嚴重。
……
我在咱們5月31號那天，我跟一個咱們美國重要朋友，我跟他通話。他說Miles，你爲什麼說這個病毒一定會復發？因爲我知道它是100%的共產黨的化學武器、生化武器，它一定會爆發。而且他能造，他不見得能控制得住。就我跟你說的，你把蠍子、你把蒼蠅、你把蚊子你把它收起來能做到，放在一個房間裏，只咬你自己，咬你屁股、咬你腚、咬你自己臉，那都可以。但你把門打開、你放出去了，說我再收回來。不可能！就這麼簡單的道理。
