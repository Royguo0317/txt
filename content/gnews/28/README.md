**今天的大事是提交选举人。**

压力，压力，人们持续施加压力才改变了这些地方的历史！

**约翰·弗雷德里克**：在史蒂夫·班农作战室的猛烈攻击下，以及在作战室听众的巨大压力下，我终于第一次看到五个星期以来，佐治亚州的首次突破。

为什么这是一个突破？很简单，这不仅仅是签名必须匹配，也必须审核邮戳，是否是已经注册的人，是否18岁以上，有地址吗，不是邮政信箱，是公民，等等。如果拒绝率达到5％或6％的范围，则乔治亚不可能获得认证，因为这意味着，500万个选票中的5%，推断出来就是100万个选票的1/4。因此他们无法证明。

乔治亚还发生了第二件事，12月7日终止的1月5日选举登记中，11月3日没有投票的另外80,000人登记要投票。州议员劳弗勒和珀杜要求透明，并要求公布这80,000个人名单。

今天，弗吉尼亚共和党选举了16位赞成川普的选举人，将其提交给州长肯普。如果最高法院否决，那么全体会议将立即对这16名选民进行认证。这是非常聪明和关键的举措。

**杰克·波索比克**：本周末星期六在华盛顿的游行，看到很多人，感谢杰克一直在报道选举之后的各种法律调查和听众会。

威斯康星州有两种诉讼：
第一，威斯康星州最高法院拒绝了川普总统的诉讼，但法官说：威斯康星州最高法院有解释法律的责任，不是为了特定诉讼人的利益，而是为了公民的利益。我们被选举出来为威斯康星州人的正义服务，确保威斯康星州的选举完整性。该法院多数成员无视威斯康星州人民的权利，否认了他们的正义。
第二，威斯康星州共和党执行董事马克·杰斐逊，做出了有利于总统的裁决：说威斯康星州错误地宣称选民可以因为COVID-19而获得不到场投票的权力，这违反了威斯康星州的法律，即一个人只能由于年龄或身体疾病才能够获得该权力，而不是其他人的疾病来限定。因此，除非你确实受到限制，否则不能声称自己身体虚弱。那么补救措施是，每一张选票都有审查。因此，主流媒体没有报告这第二个诉讼。

**鲍里斯·埃普什泰恩**：毫无疑问，威斯康星州的大法官们在异议中提出了有221万张非法选票。

今天川普选举人的原因是，正如您在威斯康星州和佐治亚州所提到的那样，比赛还在进行， 必须了解他们如何盗取真相，大多数人没有看到真相。法庭说你应该早点诉讼，这没有任何意义，是法庭上一群保守派法官不愿解决这个问题，问题是威斯康星州举行了非法选举。

提交川普选举人的策略，可以追溯到1960年夏威夷第一次涉嫌分歧的大选。关键性是，如果两个选举人都没有送出，则被推翻也没有关系。只有递交了选举人才能计算在内。因此，今天这么做，是川普总统对其爱国选民履行的义务，维护了川普总统的权利。无论这场竞赛发生了什么，我们都有信心，乔治亚州，宾夕法尼亚州，威斯康星州，密歇根州，内华达州，新墨西哥州，川普总统将最终无论是通过法律，还是通过立法都占上风，因为我们相信，当只计算所有合法选票时，他已经赢得了那些州。

**布莱恩·肯尼迪**：这显然是一个信号，表明我们需要对所有使用多猫腻软件的城镇进行重新统计，就像在安特里姆县那样，观察到的错误率是68％。 出现了错误，一旦投票进入投票系统中的决定堆，人们就必须介入并决定将票投在何处，这为 人工操作选票创造了机会。
