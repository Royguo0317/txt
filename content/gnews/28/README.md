##  **英國通信辦公室Ofcom裁定，中共國家電視台對於英國公民的強迫電視認罪違反了英國法律規定** 



英國通信辦公室(Ofcom)已經通知中國環球電視網（CGTN），他將在終審裁定後對CGTN進行處罰，其中可能包括罰款或者吊銷執照，而且對此不得上訴。

在國會議員敦促首相鮑里斯·約翰遜（Boris Johnson）重新考慮雙邊關係時，監管機構的這一舉動可能會加劇英中之間的外交緊張關係。約翰遜預計，出於安全考慮，開始縮減中國電信商華為的設備在英國5G網絡中的使用規模，這將是倫敦和北京之間的另一場的博弈。

Ofcom於2019年5月啟動的調查重點是，CGTN播出了中共國當局從英國人彼得·漢弗萊（Peter Humphrey）摘錄的“供詞” ，他和他的美國妻子一起被拘留了近兩年。

Ofcom認為相關的電視播報違反了關於“不公正和不公平待遇”的規則，並且無理侵犯了隱私，因為它們“有極大可能嚴重影響了觀眾的是非判斷。”

播報的內容會使觀眾相信漢弗萊先生“對罪行的坦白和陳述是真誠的而且自願的”。對他的評論“則是事實陳述，而非指控”。 Ofcom說，CGTN並沒有給他提供機會對不法行為的指控做出辯護。

判決書中寫道：“漢弗萊先生在庭審前就承認了罪行，而且是在那些將他關押的人也在現場的情況下，這足以引起人們對其認罪是否發自內心和是否他對認罪知情產生重大懷疑。”

“我很高興Ofcom就中共國家電視台的濫用行為做出了正確的決定。這些虐待行為非常野蠻。”漢弗萊先生對《電訊報》表示。

“不幸的是，口供摘要和電視強迫供認的做法仍在繼續，但希望我的案子能使北京注意到這種做法必須結束。”

Ofcom還發現CGTN並未就如何使用錄像獲得漢弗萊（Humphrey）先生在知情情況下的授權。

然而，CGTN辯稱，漢弗萊先生以前曾擔任記者的角色，這足以表明他了解情況。它還提供了知情授權的證據– 一張來自中國公安部的說明，其中邀請了CGTN和其他國家媒體採訪漢弗萊。

這張據稱是來自漢弗萊先生的說明是這樣寫的：“我同意與中國記者見面……公安局告訴我，本次採訪的目的是了解我們對案件的處理結果將是有利和寬容的。”

一家人權非盈利組織“保障捍衛者”的負責人彼得·達林說：“ CGTN擁有機密的警察文件，這無疑顯示了這個中共國的廣播公司與該國警察之間的勾結。”

曾在2016年在中共國被強迫電視認罪的瑞典公民達林（Dahlin） 。他說：“CGTN知道漢弗萊所謂認罪是被強迫的，但還是讓他上電視陳述並且播出了電視認罪”。

漢弗萊先生和他的妻子均於2013年因一起涉嫌與製藥巨頭葛蘭素史克（GlaxoSmithKline）在中國的賄賂案而被拘留。在這起案件的處理過程中，他稱曾被囚禁在骯髒的環境中，被下藥，並被戴上手銬鎖在鐵籠中的鐵椅上，被強迫認罪。

他繼續遭受癌症和創傷後壓力綜合症（PTSD）的折磨，這是“在被囚禁的兩年中，我遭到中國當局的虐待的直接後果，當時他們拒絕我接受治療，以此迫使我簽署虛假的供詞。”

他說，他們（中共警方）並沒有“涉嫌犯罪的真實證據”，而是由“操縱法律的人把我和我的妻子送進監獄的”。

目前尚不清楚中共國將如何應對監管機構的公告。過去一年，在華盛頓對CGTN和其他幾家中共國官方媒體在美實施限制之後，北京已經驅逐了包括《紐約時報》，《華盛頓郵報》和《華爾街日報》等美國媒體的18名記者。

Ofcom正在對CGTN播出的，來自瑞典公民桂明海的電視認罪展開調查， 桂明海出版了有關中共國領導人以及前英國駐香港領事館工作人員鄭民傑（Guim Min-kit）的書籍。

桂先生五年前在泰國度假時失踪，然後在中共國露面。他在一家國家廣播電台“承認”了2003年的一次致命的酒後駕車事故和走私非法書籍。

2月，他（桂明海）因“向境外組織提供情報” 而在秘密審判中被判處有期徒刑十年。他在英國劍橋大學學習的女兒安吉拉·桂（Angela Gui）於2018年11月向Ofcom投訴。

鄭成基在去年被迫“招供”嫖娼後提出了申訴。這個“罪行”後來被用來證明他於2019年8月失踪15天的合法性。他說，在此期間，他受到中共國秘密警察的酷刑，毆打和訊問，稱其以英國公民的身份參與了席捲香港的民主抗議活動。

此後，香港公民鄭先生已在英國獲得政治庇護。

今年5月，Ofcom發現CGTN對去年席捲香港的民主抗議活進行的報導“嚴重失衡”。監管機構表示正在考慮制裁。

代表中共國的CGTN在全球範圍播報，在英國以及天空衛視(Sky)和Freesat上，它都擁有一個自己獨立的頻道。
