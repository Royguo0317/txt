## 【香港要闻】封杀自由意志，港府研立法禁市民投白票抗议「爱党者治港」
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1051361/)

搜集：P0 / 编撰：天灭中共

中共人大修改《基本法》附件一、二之后，为「爱党者」铺设了从政的康庄大道，全面封堵民主派的参选机会，亦令民意再没有机会出在议事堂上。

坊间有声音呼吁港人，虽无力抵抗中共极权，但要始终保持愤怒。更提出用「投白票」的方式，杯葛新选举制度，向中共式民主说「不」。

**律政司：投白票会阻碍选举**

可能担心强大的民意会再掀国际关注，令自己受中共责难，港府官员相继表示考虑立法，进一步市民收窄权利。政制及内地事务局长曾国卫、律政司司长郑若骅先后在公开场合表明，正考虑研究法例，规管市民「投白票」行为。

郑若骅指出，「投票是公民责任，投白票会令选举不能进行。」但她未就白票如何阻碍选举提出具体理据。
![]()![](https://gnews.org/wp-content/uploads/2021/04/40595.jpg)
**法律界陈文敏：与其担心白票，不如思考为何有白票**

名誉资深大律师陈文敏认为，选民投白票是表达意见的常见现象，他形容人大修改后的选举制度，尤如铜墙铁壁，安全系数极高，认受性极低，他估计当局预料到，会出现投票率创新低或废票率创新高的现象，因此想靠立法规管相关行为。

陈文敏大律师强调，将宣传或鼓吹投白票列为违法行为，易有灰色地带，根本无补于事，如果选民不投白票，而是投所有候选人一票，同样是制造废票。 「政府与其担心白票。不如花时间想想为何会发生这种事。」

**民主党：选民冀用白票表达不满**

民主党主席罗健熙在港台节目上斥责政府的想法十分无谓，「若连白票都要规管，明显是『离开文明』」，更反问到「支持政府是否已属香港人的唯一选择？」

民主党前主席李永达表示，未见过任何一个国家或地方以法律禁止投白票，直言不理解为何投白票会阻碍选举举行，「当一个选民投票，看到3个候选人时，一个都不想投，但他没有撕毁，扯烂、毁坏选票（犯法行为），只是投了白票，这并不犯法，这是选民的自由意志。」

他更质疑如何执行规管，「难道每次投完票举起来给你看吗？除非你再改制度。」

**建制派：毋须过份规管投白票行为**

就连建制派亦对规管做法有保留，身为律师的立法会议员谢伟俊直言，香港没有法律要求市民必须投票，投票只是公民责任，不受法例规管，而且选民投白票只会影响投票率，并不影响选举结果，认为现阶段毋须规管投白票问题。

**「笔者评论」**

笔者记得，2016年特首选举前夕，选委会界别分组选举时，民主派名单取得326席，无奈始终特首选举是小圈子选举，入闸参选的门槛很高，因此，当时专业议政召集人郭荣铿多次强调，「如果到时所有候选人都很不堪，我们只能投白票表达对这个制度的不满，这是我们最后、最无奈的选择。」

换句话说，「投白票」是港人别无选择下的自由意志，废票也好，白票也好，都是对极权统治的反对票。
威权高压之下，人心向背应是预料中事，中共再强大，再霸道，即使能够禁锢全世界人的肉身，永远不能禁制人的思想。
