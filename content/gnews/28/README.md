## 美議員湯姆克頓重磅講話——川普總統強勢追責中共CCP病毒
`2020-08-28 07:32 GM62` [轉載自GNews](https://gnews.org/zh-hant/321366/)

編輯記者：川渝小哥G 校對編輯：文劍 文香 文宠
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/28072234/%E5%B0%81%E9%9D%A2-3-1.jpg)
美國政治家湯姆克頓於1977年5月13日出生於美國阿肯色州的達達尼爾,現年43歲,是共和黨人士。 2005年年至2010年期間湯姆克頓在美國陸軍服役,獲得了包括銅星勳章在內的多項榮譽。 2014年在擊敗了兩屆民主黨現任議員馬克普賴爾后,湯姆克頓當選參議院議員。

為期四天的美國共和黨全國代表大會於美國東部時間8月24日開始,當天下午共和黨正式提名川普總統為2020年共和黨總統候選人,他將與民主黨總統候選人拜登展開角逐。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/28071942/%E5%9B%BE%E4%BA%8C.jpg)
美國東部時間8月27號晚,共和黨全國代表大會臨近結束時,湯姆克頓上台發表了壓軸演講。 他認為二戰時期,美國依靠自身強大的國力維護了世界和平,並捍衛了民主自由的價值觀。 川普總統上任後加大了國防投入,消滅了包括伊朗蘇萊曼尼在內的多個恐怖頭目,打擊了非常危險的伊朗核條約,讓中東的阿拉伯和以色列達成了和平協定,維護了全球的安全。

湯姆克頓認為中共釋放的CCP病毒給美國及全球帶來了巨大傷害,面對中共製造的威脅和挑戰,川普總統採取措施打擊了中共的欺騙行為,並會繼續讓中共承擔CCP病毒帶來的全部損失。 湯姆克頓表示美國人希望和平,但和平的前提是自身的強大和堅定,美國人需要一個願為美國而戰的總統,而川普總統具備以上條件,他會和人民站在一起讓美國再次強大。
