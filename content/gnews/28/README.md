## 至暗穹顶（最终篇）——OPEN YOUR MIND
` 喜马拉雅意大利罗马达芬奇农场` [轉載自GNews](https://gnews.org/zh-hans/796219/)

```
作者: Tito
校对/发稿人: Ting Guo
```


![]()![](https://gnews.org/wp-content/uploads/2021/01/pexels-photo-2440530-9.jpeg)图片作者: Arthur Brognoli

为啥要写个最终回呢？因为美国大选后，一段故事已经落幕了。这段简短的写作时间中，我写了我自己是如何磕磕绊绊走到爆料革命中来的，我写了我自己是如何思考这个世界的，我写了我自己是如何看待这个社会的方方面面，我写了我对历史的疑惑，我写了我对经济的费解，我写了我对家乡的怀念，等等等等。

我喜欢思考很多事情。在写作的过程中，通过对不同资料的搜索，我发现，其实真相一直在我们身边。爆料革命告诉我们的，是一个看世界的角度，是一种怀疑世界的眼光，尤其是对于真相和真理，要不断用怀疑的态度去面对，因为真理和真相也是需要更新的。

前段时间，联盟在GNEWS征文，和希望听到广大战友的意见。我觉得，有意见的人不会很多。因为很多人的思想已经是属于封闭状态了。在混迹GNEWS期间，我就发现这一情况十分严重。广大战友都太正式的面对文章这样东西了。GNEWS的麻烦，一个是板块儿问题，一个是它本身的定位就限制了它的发展。

GNEWS的灭共属性应该更多是在新闻类文章里面的。可是目前的情况，恰恰观点类文章的灭共属性更强烈。这就限制了自己，观点应该是自由的，是奔放的，是打开心扉的，是有追求的，是有分析的，是有思想的。在国内长时间浏览新闻的过程中，我就发现，新闻就是编，从来没有真。观点就是爱党和正能量，从来不自由。所以当有了GNEWS这个平台之后，估计很多人都没有从过去的的封闭思维中跳出来。比如说，我就是爱写习近平和AV女优掉水里，你的本性想先救哪个？我就是喜欢这么写，这是国内媒体绝对不能讲的。但是现在可以讲了，为什么还要端着呢？

打开自己的思想，真的是很重要。

我估计，不会有人想我这么不正经的去述说事情。但是我就是喜欢这样，我参加爆料革命是为了什么呢？不就是为了这难得的自由么！比如说，郭先生说燕子和蛇的问题。我就会非常猥琐写出来我的想法。我希望我会像黄继光堵枪眼儿一样，大胆的冲上前去，一边冲一边喊：“战友们，为了新联邦，我去缠住诱惑的燕子，蛇你们就看着办吧！”当然，就是想想，估计我的级别，别说燕子，就算是小鸡都轮不上我。当然，缠住燕子后，我会经常和她们谈心，争取感化她们。真的就是简单的谈心。

这一心态，就是人性，每个人都要面对。当你面对那一刻，思维就会变得灵动了，自不过，人生总有选择，做好取舍。郭先生身边的诱惑从来不少，他做的不只是坚持，更多的是取舍。我明白，人和人的取舍，终会不同。我只说，每位战友要记得，在爆料革命中，才有取舍的机会，和CCP打交道，你永远没有选择。

所以，打开自己的思维，面对自己，做好自己的选择。而后，任思想自由翱翔。这对于写东西的人是很重要的。

未来，我还想继续写，只不过，一个《至暗穹顶》已经无法承载我的思维了。这个系列，是我很早想要揭露中共水军的套路和目的而推出的。后来又加入了对国内教育金融历史等各方面的看法。这个系列应该结束了，爆料革命有了新的开始，我应该也是。

以后，我就多写写我自己的猜想吧，TITO猜想。
