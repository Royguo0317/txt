据《华尔街日报》12月20日报道，中国亿万富翁马云在11月2日的一次不同寻常的会议上，向中共国监管机构官员表示，他准备做该国需要他做的一切，并主动提出将其金融技术巨头蚂蚁金服集团的一部分移交给中共政府。分析指出，马云此举是试图挽救他与监管机构的关系， 而此次会议的时间正是在蚂蚁金服集团公开上市的几天前举行的。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/25221653/image2.tif)P2: https://bit.ly/3pnYojw
值得注意的是，在今年10月习近平主席发表了关于控制金融风险的讲话后，马云竟然胆大包天发表评论，称其扼杀了企业创新。现在，事隔不到一个月，监管机构便找上门来了。 尽管马云竭尽谄媚之能事，但仍然于事无补，其IPO计划惨遭叫停。据知情人士透露，习近平亲自下令中共国监管机构调查蚂蚁金服造成的风险，并关闭了其上市通道。

然而，蚂蚁金服被迫搁置上市计划仅仅是个开始，随后，中共监管机构针对所谓的“平台经济”和大型科技公司所引导的互联网经济采取了一系列行动。

对马云而言，这也许仅仅是噩梦的开始，据悉，目前中共国金融监管机构正在考虑对蚂蚁金服施加更严格的资本和杠杆法规。在这种情况下，国有银行或其他类型的国有投资者将收购蚂蚁金服，以弥补因收紧政策而导致的资本短缺。

而分析人士指出，中共政府其实早已将将蚂蚁金服的某些金融服务产品收归国有了，例如将网联（NetsUnion）的银行间支付系统划归中央银行控制，负责清算银行与第三方支付提供商之间的交易。

近年来，习近平领导下的中共政府已经屡屡向私人企业集团祭出屠刀，例如，房地产大亨王健林的大连万达集团被迫出售资产，缩减业务并偿还银行贷款。另一个是安邦保险集团，目前已被国家接管，其创始人吴小辉于2018年因欺诈和挪用公款罪被判处18年徒刑。此外，航空和酒店业集团，海航集团（HNA Group）也不得不撤回在海外的疯狂收购并出售资产。

与此同时，马云一直凭借其良好的政治关系而安然无恙，然而，自其10月24日发表演讲以来，便销声匿迹了。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/25221619/image5.tif)P5: https://bit.ly/2Kwuwms
多年来，由马云控制的蚂蚁金服和电子商务巨头阿里巴巴集团控股有限公司（BABA -1.68％）以及互联网巨头腾讯控股（Tencent Holdings）基本上都很少受到政府监督，并已经取得了巨大的成功，以至于包括总理李克强在内的中共国领导人经常表示，互联网和大数据的使用对推动未来经济增长至关重要。

但是，中共领导层也显示出对这些公司积累的财富和影响力越来越感到不安，以及对其监管不严所带来的风险，中共领导层特别担心像马云这样飞速发展的企业在吸引资金的同时，也使国家金融体系面临更大的风险。于是，中共国的所谓反托拉斯监管机构开始以反垄断的名义对阿里巴巴和腾讯子公司采取了收购，兼并，罚款等一系列措施。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/25221609/image6.tif)P6: https://bit.ly/3nLakf6
滑稽的是，中共国的国有垄断企业，如中国电信，金融服务，航空，能源等领域，从未见过被以反垄断的名义进行过任何制裁或拆分。现在强调所谓“反垄断”，无非是习近平将这些本属于其他政治势力集团的白手套们收归己用而已。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/25221554/image7-2.jpeg)P7: https://bit.ly/3hdhKoX
