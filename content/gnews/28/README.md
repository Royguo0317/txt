中国象棋里的“卒”，一次只能走一步，它不像“车”、“马”等一次可以走多步。“日拱一卒”是指时间长了，再小的“卒”的每一次进步都会产生价值，意思是只要勤奋努力、坚持不懈，每天像个卒子一样前进一点点，终会有所成就。“功不唐捐终入海”中的“唐”意为“白白的，徒然的”，唐捐，虚耗、废弃的意思。“功不唐捐”是佛家语，意思是努力地去做某件事，所下的功夫不会白白地浪费，世界上的所有功德与努力，都是不会白白付出，必有回报。

在2021年1月19日，我们爆料革命的战友和新中国联邦的公民也应该说是“功不唐捐”。今天，路德先生终于戴上了红领带，因为美国国务院宣布了一个重要的决定——宣布中共在新疆犯下种族灭绝罪和危害人类罪。

以下是节选的美国国务卿对中共在新疆实行暴行的决定书：

1. 在仔细审查现有事实后，我确定至少自2017年3月以来，中华人民共和国(中共国)在中国共产党(中共)的指挥和控制下，对新疆以穆斯林为主的维吾尔族和其他少数民族和宗教团体成员犯下了危害人类罪。这些罪行仍在继续，包括：任意监禁或以其他方式严重剥夺100多万平民的人身自由，强迫绝育，对大量被任意拘留的人实施酷刑，强迫劳动，以及对宗教或信仰自由、言论自由和行动自由施加严厉限制。第二次世界大战结束后，纽伦堡法庭对纳粹分子的危害人类罪进行了起诉，同样的罪行正在新疆发生。

2. 此外，在仔细研究了现有的事实之后，我已经确定，在中国共产党的指挥和控制下，中共国对新疆以穆斯林为主的维吾尔族和其他少数民族和宗教群体进行了种族灭绝。我相信这种种族灭绝正在进行，我们正在见证中共国有系统地企图消灭维吾尔人。这个世界上经济、军事和政治实力第二强国的执政当局已经明确表示，他们正在强行同化并最终消灭一个脆弱的少数民族和宗教群体，同时他们还宣称自己的国家是全球领导者，并试图按照自己的形象重塑国际体系。
