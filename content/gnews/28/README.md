校对：康州农场-Richard

梅拉尼娅·特朗普是美国第一夫人，她于周二在宾夕法尼亚州为丈夫唐纳德·特朗普举行的竞选集会上发表讲话。 她在致开幕词中，感谢美国人民在她和家人在从CCP病毒康复期间他们的大量支持，她赞扬总统对这一病毒灾难的处理。 在谈到总统对病毒的抗疫时，她说：“我们不会关门，藏在恐惧中。 我们将努力工作找到真正持久的解决方案。”

她向那些因病毒失去亲人和与病毒抗争的人进行鼓励和表示同情。 梅拉尼亚甚至对特朗普总统的推文发表评论，称她可能并不总是同意他所说的话，但她深爱并认为，美国人能够“每天通过社交媒体平台及时和直接了解总统”是非常重要的。

特朗普夫人提醒美国人，特朗普在椭圆办公室工作的四年里，这个国家取得了很多成功。并鼓励到如果他连任四年，美国的繁荣将会持续。 她说：

“作为美国人，我为我们有一位将国家放在首位的总统感到自豪。 记住，我们还是一个强大的国家。 在唐纳德·特朗普（Donald Trump）担任总统的情况下，当我们击败COVID-19时，我们将变得更加强大。 这次选举的影响不仅仅是明年，而是四年甚至更久远。 这将继续使美国迈向真正的繁荣与成功。 我们不能也不应倒退。”

她呼吁美国人注意乔·拜登危险的社会主义议程，并警告说如果乔当选总统将对美国产生的影响是：他将推翻特朗普在其第一任期中建立的一切。 “乔·拜登的政策和社会主义议程只会毁掉美国以及过去四年中建立的一切。 我们必须将唐纳德留在白宫，以便他完成已经开始，并将使我们的国家继续繁荣的事业。”

梅拉尼娅·特朗普是对的。 乔·拜登（Joe Biden）和支持他的民主党人竭尽全力摧毁这个国家，并将我们的创国原则摒弃。 他们对美国以及这个国家的建立基石不了解也不关心。 他们也不在乎或不想看到它繁荣发展，也不想让美国梦在公民生活中占上风。 民主党人一直在试图占领这个国家，但美国人知道发生了什么，我们不会允许。唐纳德·特朗普也知道，这就是为什么他当选。 正如他所说：“他们不光是盯着我，也在盯着你们，我刚好挡住了他们。”
