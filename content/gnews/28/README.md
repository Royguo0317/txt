**2020年1月14日，武汉卫健委称“尚未发现明确的人传人证据，不能排除有限人传人的可能，但持续人传人的风险较低”**

*1*月*14*日，武汉卫健委官网在其公布的新冠疫情知识问答中称，尚未发现能够证明存在人传人现象的证据

武汉卫健委官网在当天公布的《新型冠状病例感染的肺炎疫情知识问答》中称：“现有的调查结果表明，尚未发现明确的人传人证据，不能排除有限人传人的可能，但持续人传人的风险较低。目前，正结合临床和流行病学资料开展进一步研究。”
