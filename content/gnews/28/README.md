## 【快讯】中共政企疯狂发债，“冥币”时代加速逼近
` XMLYG777` [轉載自GNews](https://gnews.org/zh-hans/623787/)

内新闻/素材：鹰(文言) 校对：文迹～见证神迹

新浪网12月6日转载央视报道，根据Wind数据显示，截至11月底，国内的债券承销规模达9.11万亿元，超去年同期33.38%。

2020年11月底债券承销规模已经超过19年全年的7.62亿元；今年的债券承销收入预计300亿元，超19年203.2亿元约47.6%。短短2年时间，债券承销收入接近翻倍（18年承销收入149.2亿元），即中美贸易战之后，中共的债券承销规模和收入反而“迅猛增长”。

惊人的债券承销市场中，有21家承销商年内承销金额超1000亿元，其中最大的为中信证券和中信建设，承销金额分别为1.12万亿和1.02万亿，分别占比12.33%和11.23%。在全球经济萎靡和中共假擀面杖子经济日渐萧条的背景下，中共利用国有企业和资产以债券的名义大肆敛夺社会和个人财富。

除公司债占比加大外，今年以来券商承销地方债规模也迅速增大，达到1.05万亿元，超去年同期134.89%。面对各地的“疯狂发债”，中共财政部不得不发布《关于进一步做好地方政府债券发行工作的意见》，加强对地方债的监管和约束。

从今年以来的债券承销规模和占比来看，各大型国有企业和地方政府都面临短期债务的偿还压力大、流动资金减少的窘境，却继续以债券的名义搜刮社会个人财富，用借新钱还旧钱的“传销方式”继续蒙骗大众，以维持中共经济的庞氏骗局。

中共政府通过发行债券将财富集中后，用于还旧债或继续贷款等，势必造成整个社会流动资金的大幅减少，加上自19年开始的通胀通缩并存，人民币的购买力持续下降，底层民众面临房贷车贷及其利息累积，物价飞涨（粮食越发紧缺、肉类价格上涨），生产生活越发雪上加霜。
