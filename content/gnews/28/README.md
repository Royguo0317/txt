摘要：6月12日的G7峰会举行特别会议后发表了《卡比斯湾宣言》（Carbis Bay Declaration），是指七国集团领导人认可并签署的一份共同试图阻止新冠病毒(CCPvirus)再度大流行的协议文件。

卡比斯湾位于英国康沃尔郡，是海滨度假胜地，也是本次七国集团（G7）举行年度峰会的会场。
G7领导人在周六签署了《卡比斯湾宣言》。其目的是为了防止类似COVID-19疫情(CCPvirus)再次卷土重来，对世界造成二次重大灾难和经济破坏。
据美国约翰·霍普金斯大学统计，自疫情爆发以来，全球已有超过1.75亿人感染，超过370万人死于中共病毒。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/14011247/%E5%8D%A1%E6%AF%94%E6%96%AF%E4%BC%9A%E8%AE%AE.jpeg)图片来源：网络
约翰逊指出「要真正战胜冠状病毒(CCPvirus)并恢复健康，我们需防止这样的大流行再次发生。这意味着要吸取过去18个月来的教训，并在下次危机发生时采取不同做法」。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/14011437/%E5%8D%A1%E6%AF%94%E6%96%AF%E9%A2%86%E5%AF%BC%E4%BA%BA.jpeg)图片来源：网络
卡比斯湾宣言阐明的一些承诺：
1.将未来任何疾病的疫苗开发、治疗和诊断的许可时间削减到100天内。
2.加强全球监控网络和基因组测序能力。
3.支持改革和加强世界卫生组织（世卫组织)。

预计该宣言将纳入来自企业、政府和科研机构的国际专家组报告中的建议。
联合国秘书长安东尼奥·古特雷斯和世卫组织主任泰德罗斯·盖布雷耶苏斯博士也将参加周六的会议。
世界卫生组织总干事谭德塞强调 “ 世界需要一个更强大的全球监测系统，以检测新的疫情和大流行风险 ” 。

作者点评
