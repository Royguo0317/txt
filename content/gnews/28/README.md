## 美印太司令指中共欲在5年内夺取印太军事主导权

作者：军迷Wilson
![]()![](https://gnews.org/wp-content/uploads/2021/03/33333333.jpg)
据共同社华盛顿3月5日报道，在本周四美国智库举办的活动中，美军印度太平洋司令部司令戴维森表示，中共国打算在2026年之前，在印度太平洋地区占据军事优势地位。如果中共通过军事扩张，在事实上占领或控制尖阁诸岛（中国称钓鱼岛）和南海等，戴维森认为，一旦现状被中共改变，那将成为永久性的(即很难再夺回来，因为这些区域都是中共导弹的压制范围)。因此，戴维森呼吁当务之急是与盟国一道加强对中共的威慑力。

戴维森直言，当前最大的危机是美国及盟国在印太对中共的威慑力在衰退，如果我们不能使中共明白，它在该地区釆取军事行动将代价过高，那么后果不堪设想。
戴维森认为，围堵中共西太洋第一岛链中的南段，即日本冲绳县至菲律宾之间将是最薄弱、最容易被中共突破的(因为中共已经在该海域的众多岛屿上布置战略性武器)。

戴维森肯定了日、澳、英法等国已经认识到，在印太维护航行自由的重要性及表现出对抗中共的决心。他认为关岛是当前西太平洋反制中共的战略心脏，因此加强美属关岛的导弹作战能力是当务之急。
