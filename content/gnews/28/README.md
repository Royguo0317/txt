## 网传习近平即将进行颅内动脉瘤手术？真真切切!
` Snow` [轉載自GNews](https://gnews.org/zh-hans/698059/)

编撰：peacelv 复核：WENJUN

近日路德时评介绍说习近平的脑动脉瘤要全麻开颅，刚得到最新消息习的手术以前已经做了几次了，都是用介入方法，靠X光影像定位用穿刺针沿着微细导管通过血管进入患病部位处理，不是开颅，但这也是全麻手术。血管瘤不能根除，对于生命来说这是个定时炸弹，随时会破裂，岁数越大，压力越大风险越大。

![]()![](https://gnews.org/wp-content/uploads/2020/12/截屏2020-12-30-上午10.18.41.png)

这几年来关于习近平健康问题一直不绝于报纸，比如南洋报2019/03/29的报道。外国媒体发现，与法国马克龙会面时，以往昂首阔步，自诩肩扛200斤走山路不换肩的习近平，此次出访似乎脚有问题，步幅失衡、坐时艰难。网传怀疑他患痛风。习近平25日在巴黎与法国总统马克龙检阅仪仗队时，步姿大异于昔日，走得相当缓慢，且步幅不平衡，似乎右脚较弱，动作亦相当拘谨。习在沙发椅坐下时，要双手扶椅借力缓缓坐下，其间多次向下望自己双腿，三度调整自己坐沙发的位置，坐得相当小心。但港医认为应不是痛风导致的。

又如法广2019年5月5日报道习近平访问欧洲，在检阅军礼的时候被拍到步伐僵硬、走路蹒跚，是不是双脚有些问题？

又如自由时报2019/06/09的报道，2019/06/07习近平访问俄罗斯，寻求中俄结盟来对抗美国贸易战攻势时，从圣彼得堡国际经济论坛离场时发生一段小插曲，他走到台前与台下多名嘉宾握手之际，却突然失去平冲向前倾倒，差一点摔跤跌倒，港媒揶揄这是要下台的节奏。

![]()![](https://gnews.org/wp-content/uploads/2020/12/截屏2020-12-30-上午10.22.20.png)

其实早在2015年就有论坛爆出患帕金森综合症，习近平阅兵时，头颅一直向左偏移，目光直视空旷的广场，一脸愁容，一点都不关注受阅的官兵，见万维网论坛。

综上所述，先天的血管瘤，再加上每天处心积虑的与人斗与天斗，精神一直难以有放松的状态，最后导致不得不开颅治血管瘤，看来这一消息空穴来风。独裁的政权权力过于集中，独裁者身体健康的变化会引起多种潜在的政权变化。习无论是否手术，手术成功与否，中共体制的覆亡是一个必然，因为美国和北约对CCP病毒（生化武器）以及大选干涉的追责让中共不再有任何腾挪的空间以及时间换空间的机会。
