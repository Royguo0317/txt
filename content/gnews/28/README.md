美联储主席杰罗姆.鲍威尔在本周三的记者会上声称，中共国近来快速推行的数字化版本人民币，对美联储研究和推行自己的数字货币计划不会产生任何影响，拥有发达支付系统的美元，作为世界货币的地位非常稳固。鲍威尔强调:中国的数字货币在美国行不通，它真的允许政府实时查看每一笔付款。

国际货币基金组织的数据显示，去年底美元占世界官方外汇储备的近60%，而中共的份额仅为2.25%。由于人民币不能自由兑换，为了方便贸易结算，这点可怜的份额还应该包括中共和不少国家的货币互换额度。

货币是否被更多的人接受和使用，是衡量其信用的标准，人民币缺乏信用是因为中共的极权体制是反科学、反市场化的，而且是腐败的温床，人民币服务于中共高成本的残暴统治，是中共富可敌国的盗国们肆意盘剥老百姓的工具。

人民币号称世界第二经济体使用货币，由于中共国的经济状况继续恶化，如果允许人民币自由兑换，将立即引起人民币的抛售潮、汇率的爆跌及中共财政的崩塌，因些，即使换了数字人民币的马甲，其缺乏信用的本质丝毫不变，已经不可能成为国际自由兑换货币。中共不知道吃错了什么药，竟然带上这样的垃圾满世界张扬。
