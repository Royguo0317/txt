据12月1日加拿大知名独立媒体Blacklock’s Reporter报道，加拿大联邦参议院目前最大的，其组织成员主要由总理特鲁多亲自任命的“独立参议员核心小组”领导人，马来西亚华裔联邦参议员胡元豹近日公然发表言论指出：加拿大政府应该大力支持中共国在成为全面发展的经济强国进程中所采取的一系列强权措施！

胡元豹，这位来自BC省并由总理钦点的独立参议员组织负责人，其评论发表于加中友好协会（Canada-China Friendship Society）这一致力于所谓“帮助加拿大人更好地了解中共国过去所取得的成就，当下所面临的挑战以及未来将致的前景”而为中共政权在加拿大利益服务的亲共组织！

加中友好协会曾于9月接待了公然恐吓加拿大人的中共国驻加拿大大使丛培武。该组织在其网站的“资源”部分上播放了一段中共国驻美国大使的视频，该视频出现在声名狼藉的CNN上，以此企图否认中共政权正在对维吾尔等少数民族所实施的众多种族灭绝罪行。不仅如此，该网站还大肆宣传中共国电影“第一次告别”，该影片旨在美化中共当局在维族民众居住地区强制推行普通话这一威胁当地少数民族语言使用和存在极端文化政策。

胡元豹认为，“我们应向中共领导人明确表示加拿大政府衷心希望中共国能在经济上取得全面成功！虽然在我看来，鉴于目前加中相互冲突的双边政治背景下，加拿大当局似乎很难直接这么做，但在不同场合和不同时间里向中共卑躬示好，这在其外交层面上却并非难事。”

该独立参议员领袖表示：“加拿大的个体企业必须自行决定是否要在违反其价值观和良知的地方开拓业务，而且我获悉许多加拿大企业不明智地拒绝在中共国发展它们的经营。”由于目前对加拿大市场上由维吾尔族奴隶劳工所生产商品的日益关注，最近几个月来，该国政府一直在努力加强对中共国经营的加拿大企业的监督。例如，全面调查庞巴迪（Bombardier）等加拿大公司在中共国涉嫌使用奴隶劳工的真实情况。

很显然，胡元豹在中加友好协会的此番挺共言论，其意在提倡加拿大当局应该对中共政权采取更加放任和纵容的态度，并认为必须放弃“试图去改变中共国是加拿大政治使命的一部分”这一错误理论。他还进一步建议“我们应该对实施不同的外交策略是否会产生可以更好取悦中共政权的结果采取包容的态度”。

他继续强调，“加拿大政府应明确对外表示，支持中共国到2049年成为全面发展经济体的宏伟目标。除此之外，我们希望加拿大成为中共国在全面发展经济并和其分享该国取得成功道路上的一部分。”

“我的观点很明确，对于整个加拿大而言，应该与中共国或美国这样的经济体大国建立这种命运利益共同体的联系，而大国之间的联系要点是如此之多，以致于我们不能简单地提出某个问题，而该问题会影响到双边密切关系的所有其他部分。” 但针对以上所谓的大国关系论调，胡并未同时提供与其陈述相对应的事实依据，也未阐明中共所采取的诸如种族灭绝罪行是否为目前加中双方外交关系恶化的基本根源。

自2018年应美国政府的要求在温哥华逮捕华为的孟晚舟以来，中加之间的关系看起来一直很紧张，美国政府正以欺诈等罪名起诉孟晚舟。而作为对此举的报复，中共当局随意拘禁了两名加拿大公民迈克尔·斯帕沃尔和迈克尔·科夫里格，也就是通常被外界简称为的“双迈克尔”事件。作为参议员，胡元豹此前曾全力呼吁加拿大政府代表孟晚舟的利益介入和干预该引渡案。

评论：

通过以上信息我们不难看出，目前身居加拿大政坛和亚洲经济学术高位的胡元豹，自其被总理特鲁多正式任命为加拿大联邦独立参议员核心小组的话事人以来，就始终充当着中共政权在加国社会政治高层的传声筒和发言人，他不仅积极出席各种由当地中共使领馆主办的各种外事活动，精心按中共国的布局计划，以其在加拿大对亚洲经济政策首席顾问的学术身份，不断向加拿大政府提供各种有利于中共国在加肆意牟取各种非法利益的各项决策建议，而且倾力协助中共政权在加拿大政治和经济领域实施其各种蓝金黄手段而最终控制了该国几乎所有高层的方方面面。
