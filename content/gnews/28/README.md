## 南非国防军情部被指控采购未经批准的中共病毒疫苗

比勒陀利亚媒体报道，国防和军事退伍军人大臣Nosiviwe Mapisa-Nqakula已对南非国防军情报部门从古巴购买有争议的中共病毒药物的指控进行了独立调查。

该任务组将由南非前情报总检查官佐拉·恩卡卡尼（Zola Ngcakani）担任主席，并与另外两名官员一起调查去年年底对购买中共病毒药品的指控。据称，南非国防军购买了用于治疗中共病毒的药物，该药物未经南非健康产品监管局批准。也有人指控，该药可能是去年被派往南非协助抗击中共病毒大流行的古巴医生带上飞机走私到南非的。但是，南非国防军捍卫了对此药物的采购决定，称该药是在大流行爆发后紧急购买的。南非健康产品监管局试图在比勒陀利亚的南非军事健康服务站没收该药，被拒之门外。

这种中共病毒治疗药物使用干扰素alfa-2b作为免疫调节剂来治疗测试呈阳性的患者，且称对与呈阳性患者保持接触的人有防护作用。南非国防军认为使用此药物可以降低中共病毒死亡率。国防军称，“客观数据表明，古巴，巴基斯坦，伊朗，乌克兰，巴西和其他地区的8000多名受试者已从中受益。”

但是，一些健康部和国防专家驳斥了这一论点，他们说，正如研究表明的那样，该药物实际上对中共病毒患者没有益处。而对该药物的采购，运输和储存等问题引起了军事监察员的注意和担忧。

自中共病毒席卷世界后，疫苗已经成为新兴的经济产业。新中国联邦和闫丽梦博士一直致力揭发中共病毒是中共实验室制造的生物武器，从病毒本源上消灭病毒是唯一的方法，迄今为止并无有效疫苗。而消灭病毒的根本就是推翻中共政权，不然新的迭代变异病毒会层出不穷。南非疫苗问题仅是世界各国的缩影，贫穷国家无力支付西方高额的疫苗费用，又必须维护国家安全、满足国内防疫药物的供应，令各国陷入中共病毒疫苗困境。
