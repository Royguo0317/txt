## 【战云长天时评29期】中共是“完美犯罪”学理论践行者——案例十四：隐形国情下的农业改革

作者：喜马拉雅东京樱花团 / 捆绑CCP一千年

前言：中共20大“振兴乡村”计划是一个彻头彻尾，带有明显文革化的新农村政策，在披着乡村振兴的美丽外衣下，掩盖其举国犯罪体制的完美犯罪心理，其新型农村关系则是新型大国关系的概念的复制，他将利用乡村振兴计划吞吐大量城市知识青年，即为了解决新生代大学毕业生的就业率和出路问题公然提出“上山下乡”计划。一场全面新型文革运动似乎山雨欲来。笔者认为，习近平的高举人民的口号实乃高举人民的头颅大开杀戒也。呜呼哀哉我中华。国何以再度被纳粹集团党卫军劫持于无形之中，唯我人民仍在昏昏欲睡？为了挽救我中华儿女，笔者毅然决然地以笔代枪，参加郭文贵先生领导的这场灭共大战，实乃吾辈之幸也。作为农民的儿子，笔者深知父亲仍是坚定的毛思想，正因为如此，揭露中共完美犯罪的事实，乃笔者良心的鼓舞所致。仅仅作为一个鲜活的人，你没有理由装睡不醒，同胞们，醒来吧，中共邪恶计划再次企图实行农村包围城市计划一旦得逞，你我生于斯者还有完肤吗？

据新华社北京时间3月6日“20大两会”报道，一篇题为《（两会受权发布）李克强参加广西代表团审议》的报道提到了习近平“坚持绿水青山就是金山银山理念”的青山论。而报道标题“两会受权”的“受”并非错字，乃是国家立法机关接受中国共产党习近平政府旨意叫［两会受权］。彻底党国权利倒置，这也意味着习近平承认劫持了国家。尽管如此，但这位曾经勇敢地在“上一届两会”答记者问时勇敢爆料的李克强总理称，中国还有6亿贫困人口月收入在1000元人民币以内。这发生在习近平数字脱贫的关键时刻，而这位看似勇敢的中共国国务院总理李克强选择两会期间来到广西代表团审议报告是不是向广西人表达了一丝爱莫能助的无奈呢？尽管李克强泛泛地提到要坚持“绿水青山就是金山银山理念，加强污染治理和生态保护”的话，但仍旧看得出，在习的威权下，“众位爱卿”心里战兢恐惧的程度可谓达到了大清帝国的高度。比如，“加强污染治理”在极度贫困的广西省可谓是一个伪命题，这甚至在中共农业现代化高度落后状况下提倡生态保护和农村发展、甚至工业发展都是相当矛盾的问题，这里包含着气候变化的完美欺骗行为。

为何完美犯罪骗局在农村？比如，农民创收，养猪和养鸡在胡锦涛时代都比较赚钱，但习近平政府以环保为由不让农民养猪，而空缺的猪肉市场需求可以挪去满足美国等西方市场。这是在中美贸易战之前的事，于是，美国制裁后造成全国性高肉价，使人们在更加微薄的收入面前加剧家庭开支紧缩现象。李克强继续说道，在关于“落实巩固拓展脱贫攻坚成果同乡村振兴有效衔接的要求，加强农村特色产业技术研发攻关，增强农业竞争力和抗风险能力，促进农民增收。提高基层尤其是县乡师资水平……”这里提到农村仍需要在“巩固、拓展、脱贫攻坚”同“乡村振兴”基础上开展“农村特色产业技术研发攻关”。从这几句话中我们窥视出怎样的农村？这公开表明农村脱贫是假，为农民创收面临如何在突破西方技术封锁的前提下发展现代农业是真。而农业研发技术攻关显然是句空话。因为中共在加速退出历史舞台的过程中，它不可能回到过去。而“乡村振兴”这个概念可以理解为农村绣花枕头和面子工程。

据笔者所知，乡村振兴局取代过去的国家扶贫办。就此问题，笔者前两天从一位国家扶贫办退休官员口中了解到的情况是这样，乡村振兴办的编制仍然是国家扶贫办原班人马。工作内容由扶贫改为防止返贫。笔者戏称，那不还是一个肥差吗，继续巧立名目向上面申请财政拨款，拿出一小部分钱在农村建个花园和乡村公路什么的，或将农民土砖房粉饰掩盖一下丑态。春节期间再买点小礼物选几个代表性家庭慰问一下。其他结余资金的三分之二的公款拿来中饱私囊。他会心地笑了，说；“你懂政治……”。这不就是完美犯罪吗？已经消声的人们如何给他们定罪？中共在几乎所有领域都是利用这种完美犯罪心理来玩弄人民的。因为中共高举人民的目的就是要把人民很很地摔到地上。

一位广西长大的女孩在推特（twitter）中这样描述中共扶贫工作，“扶贫政策作用不大，流于形式。干部下乡扶贫就是帮孤寡老人安装彩色电视机、电冰箱和空调。而农村老人在家务农，根本就没有闲钱去使用这些高耗电的电器，也没钱交一个月26元的数码电视收视费。扶贫干部只是为了满足拍照写报告，向上级汇报完成任务。”这样的扶贫工作不但没有给农民增加收入，反而变本加厉赚取农民的微薄血汗钱。而电视的诱惑在于了解外面世界，实际上是迫使穷人在洗脑教育和娱乐化状态下还要“谢主隆恩”。而另一位网友说，“真正的赤贫是集体失语”，说21世纪的中国贫困是结构性的，可以和肿瘤类比。相关新闻被报道出来，也是相关部门被允许的情况下操纵的舆论，在赚取人民眼泪的情况下衬托扶贫工作的重要性，将消极面转化为积极面的完美犯罪方式。这对中共的执政合法性没有半点威胁。可谓生于威权下，死于无声哀叹中。

对于广西农村“赤贫”现象的描述，2019年11月1日，BBC中文网一篇《中国：贵州女生日开销两元折射的贫困人口现状》的报道讲述了这位名叫吴花燕的故事；“据中国媒体报道，本月较早前，这名年轻女生因为呼吸困难而被送进医院。她的身高只有1.35米（4尺5寸），体重仅20公斤多一点。医生发现，她的心脏和肾脏都有问题，这是五年来一直缺少食物造成的。吴花燕说，她需要省下钱来照顾患有精神病的弟弟。吴花燕的母亲在她四岁时就去世了，后来父亲也在她上学的时候离世。她和弟弟之后就跟奶奶生活，后来又由伯父一家抚养，但是伯父每月只能给他们300元的生活费。这笔钱当中的大部分都花在为弟弟治病上。于是，吴花燕自己每天只有两元的生活费。据媒体报道，她从来不吃早饭，一天只能吃两块钱的米饭，且曾有五年多的时间，大部分时候都靠吃白饭拌糟辣椒过活。”（见《BBC中文》链接）而这个故事距今2年时间，这位女孩有没有脱贫呢？而贵州、广西、全国有多少个吴花燕的故事呢？然而，更大的危机——大饥荒也许即将到来。

习近平说，尽管中国粮食生产连年丰收，对粮食安全还是始终要有危机意识，称中国的餐饮浪费现象“触目惊心”。官方新华社评论称，餐饮浪费现象，事关国家粮食安全的大问题，“再厚的家底也经不起挥霍”。为应对美国与中国经济脱钩的多项措施，中国提出了“内循环”的主张。观察人士认为，中国如今的经济“内循环”与多年来每逢世界经济走低中国“刺激内需”的政策最大的不同在于如今与美国关系对立。（见《BBC中文（粮食）》）实际上，习近平的“触目惊心”掩盖了一场可能发生的全国性的大饥荒。刽子手习近平竟然有勇气杀人，竟以隐晦的方式承认了卖国的事实，竟然是以中共国“再厚的家底也经不起挥霍”一语彻底暴露了自己。而经济陷入“内循环”就是承认被迫闭关锁国的现实，全球统治计划要破产？为何不可告人的2035、2045计划却仍在信誓坦坦保证实施呢？

中共尽管一步一步地陷入僵局，甚至政体即将脆断，而完美犯罪的定义对于中共恶政，至今仍然有效，因为中共举国体制犯罪，在强大的国家机器面前，人民就是没办法将其绳之以法。除了新疆种族灭绝罪构成反人类罪，被美前国务卿蓬佩奥定性后，至于更大的恶，即和平时代利用超限生物基因武器攻击全人类这样的案例至今也无法定罪，这才是灭共的症结所在。然而，中共所谓改革开放40年来，将社会两极分化距离简直拉到了天际空间去了。如果你将广西壮族自治区一个失去双亲家庭的收入和盗国贼王岐山等人的收入相比，你能想象那是亿万太空的距离吗？

笔者为何认为习近平对农村《走中国特色社会主义乡村振兴道路》的政策不仅仅是一种对农民的杀戮行为，更是消灭新生代知识分子、大学生的行为？习近平说，“要建立以工补农、以城带乡的新型城乡关系。长期以来，……农村‘造血’不足，加上‘失血’过多，导致出现严重‘贫血’。要实现乡村振兴，必须招才引智，促进各路人才‘上山下乡’。”（见《人民网》链接）这是2017年12月28日习近平在召开农村工作会议上讲话的一部分。这样赤裸裸的文革思维是要再次吹响“上山下乡”运动的集结号。因为当城市大量新增大学毕业生无法就业时，大量闲置的知识青年自然给政府造成巨大的压力，在国门被迫关闭状态下，就业率正如外交政策一样，再也无法继续演戏了，因此这就需要发动文革。而乡村振兴计划则需要给农民一点甜头尝尝，并将有大量城市知识女青年输入农村，这对无法支付巨额彩礼的农村青年，无异于送来了春梦一般的大礼。笔者已经无法想象，农民一定会像大陆伪装的港警一样，在无约束状态下奸淫香港女孩后那种高呼“习主席万岁”的事情一定会在农村重演，因为这大大利好农村青年。即便在娱乐中挥霍青春，甚至死去，都是心甘情愿。因为习近平亲口说的，农村人“失血”过多，需要增补农村青年贫血现象。而乡村振兴工作还隐藏了习近平发动农民造反的企图。只要习近平“新型大国关系”玩不转了，就一定要玩“新型城乡关系”。这也许是习近平青山绿水、金山银山背后真正的振兴计划。这也一定是为了给自己打鸡血准备的一条后路，但新中国联邦绝对不会容许他有葬身之地。我们正在四面追讨他的一切反人类犯罪行为。笔者相信解放中国14亿人口的的日子不远了！
