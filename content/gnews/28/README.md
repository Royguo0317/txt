卫报报道，联合国秘书长安东尼奥·古特雷斯 (António Guterres)表示，联合国已开始与北京方面就“无限制”访问新疆进行谈判，以了解维吾尔少数民族受到的待遇。据称此举得到了北京方面的支持。

联合国人权事务高级专员巴切莱特(Michelle Bachelet)今年2月表示，有关新疆任意拘留、虐待、性暴力和强迫劳动的报道需要进行彻底和独立的评估。巴切莱特表示，有关组织访问的谈判已经开始，但尚未达成任何协议。

古特雷斯周日对加拿大CBC电视台表示，“目前，联合国人权专员办公室和中共当局正在进行一场严肃的谈判。”“我希望他们能尽快达成协议”，允许“无限制地”访问新疆。

古特雷斯表示，中方已多次向他确认，“他们希望完成这一使命”。活动人士对不受限制地进入中共进行一次有意义的访问的前景表示怀疑。

上周一，欧盟率先对四名中共国官员和一家实体实施制裁，英国和加拿大随后也做出了类似决定。加拿大和英国的外交部长与布林肯发表联合声明说，三国一致要求北京结束其在新疆的“镇压行为”。他们表示，他们掌握了“压倒性”证据，“包括卫星图像、目击者证词和中共政府自己的文件”。

中共立刻进行反击报复，对欧盟采取了范围更广的惩罚性措施进行回击。制裁范围包括欧洲议员、外交官、机构和家庭，并禁止欧盟企业与中共国进行贸易。上周六，北京方面又宣布了对两名美国宗教人士、一名加拿大议员和一个人权倡导机构实施制裁，因为他们对中共在新疆维吾尔族的种族灭绝罪行进行了批评。

借鉴世界卫生组织在中共国进行疫情溯源的调查结果，中共只会让调查组看到中共允许他们看到的情况，因此，对于所谓“无限制”的调查，调查组是否能够在中共高压控制下见到真正的新疆，国际社会普遍表示怀疑。
