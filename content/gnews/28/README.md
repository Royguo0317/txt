## 中共国人口普查数据中是否造假
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1244550/)

作者: 菜园子112～尕虎
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/15050134/002-2-6.jpg)网络图片
5月11日，中共国国务院公布了人口普查数据，2020年年底的总人口数为141178万人，多么吉利的一个数字，较2019年的140005万人，新增人口1173万人。注意这里请大家注意，2019年新增人口是467万人，一年之内人口暴增至1173万人，增长率为151.2%，这样的增幅是不是很诡异，有没有人为编造的痕迹，不急我们先不要下结论。

接下来我们对比一下出生率和死亡率，2020年的出生率和死亡率目前还没有公布，我们根据以往的数据进行对比，2020年的人口结构0—14岁的总人口数25338万人，从2006年至2019年人口合计总数22738万人，假设一下这些孩子都健健康康没有夭折，那么就可以得出2020年的出生人口总数2600万人，一个漂亮的数据。新增人口数有了，出生人口数也有了，倒推一下：2600万人-1173万人=1427万人，由此我们可以得出2020年死亡人口数为1427万人。

从中共国家统计局统计的往年的死亡统计中可以看到，1990年-2019年每年的死亡人数维持在763万人-998万人这个区间，但是公布的2020年人口普查的数据可以得出当年的死亡人数达到了1427万人，死亡人数暴增了四百多万人，死亡率由2019年的7.14%暴涨到2020年的10.11%，恐惧不？

现在我们有了2020年中共国的死亡总人数，大家发现了没有，这多出来的死亡四百多万人是怎么死亡的。2020年发生了什么我想大家都知道-中共病毒，那么这四百多万人的死亡和中共病毒有没有关系，我想大家心中已有答案了，中共政府一直宣称他们把疫情控制的很好，目前公布的疫情死亡人数也就几千人，但是他们自己公布的人口普查数据把他们的底给漏了。湖北省2019年总人口5927万人，2020年总人口5851万人，人口没有增加反而减少了76万人，看到了没有，这减少的人口数就是死亡人数中的一部分，因为还没有包括2020年的出生人口，2019年湖北省的死亡率7.8%。

从以上数据我们可以判断出中共国公布的人口普查数据有造假的嫌疑。目前中共国经济下滑，普通人收入降低，高房价，高学费使老百姓生活负担很重，年轻人没有组建家庭的意愿。虽然中共政府鼓励生育二胎，可是经济收入的降低导致老百姓的生育意愿下降，大部分省份的出生率都在降低，甚至一部分省份出生率出现了负增长。而中共国现在公布的人口普查新增人口1173万人是哪里冒出来的，只有他们心里清楚，他们到底想掩盖什么？我有一个大胆的推测，他们想掩盖2020年的人口死亡数据。全世界各个国家在中共病毒的肆虐下，感染人数和死亡人数都高居不下的情况下，中共国疫情竟然没有死亡多少人，这个从逻辑上也解释不通，因此可以判断，中共国在2020年年初的病毒爆发初期死亡人数肯定有几百万人，由于他们掩盖真实数据，我们只能在他们公布的数据中寻找答案，以此来验证我们的判断是否准确。
