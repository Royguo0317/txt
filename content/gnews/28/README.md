## 中共超限战灭港计划（三十四）六七暴动 之 青年乐园和“针”
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1085211/)

搜集/撰文：岁月如歌 / 封面合成：文粤

根据前学友社主席梁慕娴理女士《我所知道的“青年乐园”》一书中披露《青年乐园》简称《青乐》是和「学友社」并排的地下组织。 1956年4月创刊，六七暴动发生后因为宣扬极左文章为主被港英政府以煽动罪叫停，存在时间只有11年半。别看它这么短命但是他的影响力可不一般，那些曾经的作者、编辑后来都功成名就：如陈坤耀、黄玉山、徐碧美、曾钰成、梁锦松、小思、陈毓祥、李国强、这些人都非常厉害，在香港回归后更有影响力，有的在回归后的特区政府里面工作。

梁慕娴女士称青年乐园为“战友组织”。它由中共地下党创办和领导，工作地位和学友社一样，是官立的、拿津贴、拿补助的私立（简称官津补私）组织。以学校学生为发展对象，进行爱国爱党洗脑教育。青乐以文学写作为主来吸引学生，学友社前面介绍过，是利用舞蹈，戏剧，音乐等艺术活动来发展学生。他们的任务目的相同，都是为了发展青年学生，培养党员干部。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4131.jpg)图片来自消失的档案
《明报》于2017年3月25日发表了吴康民的一篇回忆文章《青年乐园忆昔》：吴康民非常坦白了说明当时的情况，「我当年虽任教培侨中学，但年轻力壮，总想在香港青年工作中有所作为。眼看美国人插手香港青年活动，颇有与他们一争短长的雄心壮志。所以毅然负起一个对抗《中国学生周报》（由美国资助的国民党报）责任，纯是初生之犊不怕虎的一股勇气，并不是受到什么指示而为的」由于经费紧张，吴康民借岳父的家作为办公地点，小姨黄穗华担任督印人，陈乐群、陈序臻做编辑，社长兼总编辑汪澄（原名汪潜），是汉华中学美术教师、画家。吴康民说「另有友人介绍一位青年人洪新前来帮忙」和「并不是受到什么指示而为的」是说《青乐》由地下党提议，他主动承担起这个责任呢，还是创办《青乐》是他个人因为有雄心壮志自己创办呢？石中英在引述吴康民的口述时说洪新是吴的襟弟，他为什么不说真实话呢？是忘记了吗？但对汪澄他能说出是他的学姊诸兆庚的丈夫。吴康民后来渐渐淡出了青乐，汪澄59年也离开了青乐移居夏威夷。梁慕娴说，通过这些关系，刊物初期与吴康民及其亲友关系十分密切，《读者谈青年乐园》作者则称吴为左派“大老“。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4132.jpg)图片来自852邮报，图为吴康民
吴康民评价后期的《青乐》，说港英政府早已知道《青年乐园》的背景，以及吴与刊物的关系，但是都没有动手收拾他们。如果不是「反英抗暴」时受左的领导层把这些“灰线”单位孤注一掷投入斗争，损失也许不至于这么大。并引述邓小平的名言：“左的东西在我们党的历史上可怕呀！”、“中国要警惕右，但主要是防止左。」历史上这样的教训比比皆是。但是”左“的错误总是一犯再犯，原因是一些不知就里的群众，总以为”左“的就是进步的、革命的。吴康民之所以有这一番的评论主要是67暴动是失败的，中共企图借六七暴动壮大他们在香港的影响力和实际控制香港，但最终被港英政府摆平，把一些原来隐藏得很好的地下组织或者是伪装得很好的地下组织都暴露了。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4133.jpg)图片来自消失的档案
香港地下党组织是怎样的方式和配合工作的呢？这些组织都是有港澳工委统一领导，下面设多个战线，如工运展现、教育占线、工商展现、新闻占线等各种战线。教育战线又分为红线和灰线。红线直接由港澳工委创立及领导的红校，例如香岛中学、培道中学和华书院的这些学校。灰线有青年乐园、学友社以及那官津补私的，在学校内成立的秘密读书会，这些读书会在后来的六七暴动转变成斗委会。这些战线是最基层的单位，发展新党员，完成上级指示，特别六七期间各种游行示威，贴大字报等都有这些单位的地下党员组织和发动。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4134.jpg)图片来自搜狗百科
梁慕娴说要完成这些工作就离不开地下党员。要找到这些党员也不容易他们都隐身于各个单位，有可能是书店经理、工会工人、记者或者银行职工等。曾子美是曾钰成和曾德成的妹妹，她在庇理罗士中学读书，中三时候认识另一个住在西环的桃李台中六学姊，这位学姊经常来找她玩，经常借书给她看，如《家》、《春》、《秋》，后来又有《青春之歌》、《红岩》借给她看。曾子美在这些书和学姊的影响下不知不觉关心时事，了解中国共产党，后来的暑假有同学介绍她去青年乐园做暑期工担任二手书出让工作。庇理罗氏中学的另一位学生韩雪也在被采访的时候透露他经常被学姐带去参加学校开办的中文学会、辩论会、戏剧表演等一些节目，还经常带他们去爬山什么的。前面文章说到的苏婆婆在被采访时也说过，他们在读书的时候就被中共地下党的文学刊物、唱歌、跳舞等这些文艺活动所吸引和洗脑，她也曾经被所谓的学姊或者朋友们带去看过《红色娘子军》。石中英（原名杨宇杰）也是在学校的读书会认识同学并被介绍去青年乐园做送报员的。中共地下党就是这样不声不响地把这些涉世未深的学生，通过同学关系朋友关系等培养感情取得他们信任，再利用红色刊物红色文艺表演等慢慢地感染他们的思维，逐渐的把他们洗脑成为“自己人”。这位中六学姊就是地下党的“针”
![]()![](https://gnews.org/wp-content/uploads/2021/04/4135.jpg)图片来自消失的档案
1967年庇理罗氏中学的一个余姓的学生因为写了一首打油诗发表在青乐，讽刺校长被取消奖学金，另一位同学叫劳惠琼发起募捐帮助余姓同学被开除，这位中六学姊就召集韩雪，曾子美等十多人到湾仔一家补习社开会，（该补习社于1968 年成为《新晚报》副刊〈学生乐园〉的读者活动中心）商量如何帮助劳惠琼，会上学姐主张回校上课。劳惠校当天得到同13位学的声援，并高呼口号抗议，最后全体被捕。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4136.jpg)图片来自维基百科
前面文章提到过的叶国华（原名叶允鸿，改名叶宇腾）1962年因犯错被派到学友社锻炼。大姐告诉时任学友社主席的梁慕娴说是自己人给他安排工作。经过一段时间的观察和试探叶国华的工作能力非常出色，得到信任。人造花厂事件发展成群众暴动，学友社很多人都吓跑了。各界都成立斗委会，上级命令这些“官津补私灰色的组织必须成立斗委会，梁慕娴和地下党员着手把「留下的积极分子按学校单位分别组成斗委会及官津补私文艺战斗队组成斗委会」。叶国华则在皇仁书院直接领导四个只有十五，六岁的学生，组成皇仁书院斗委会。 「虽然只是四人起家，但几个月间便一层层地发展为约十多人的组织。他们在皇仁书院挂标语“粉碎奴化教育”、“粉粹反华课本言论”。在在校内散发“吿皇仁同学，工友，老师书”，张贴个是反迫害标语」，有同学被抓，有的被开除，有的反而上了好的学校，升级了。在地下党内，党员干部能发展多少属下党员，能抓住多少群众就是向党邀功的本钱。皇仁书院是名校，影响大，叶国华得到这张皇牌，是“六七暴动”的大赢家。叶国华升官发财是毫无疑问的，在地上，他当上红顶商人和特首董建华的特别顾问（现为香港政策研究所主席）。在地下，也晋升为灰线领导人人之一。而曾经跟他打拼的兄弟们李继番、何安顿、伍镇环、蔡文田、宋树材（他一人发展15个下线）等都成为他升官发财的“绊脚石”。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4137.jpg)图片来自壹周刊
梁慕娴在2018年说，「中国共产党总书记习近了扬言，“要牢牢掌握中央对香港的全面实实管治权”。这样十拿九稳的宣布靠的是什么？就是地下党」。 2020年美国总统大选期间，美国情报部门截获密电，1月6日前干掉川普，这么确定时间内完成，靠的估计也是中共的地下党员。川普和很多共和党人，在这么严密的安保下，每天筛查也被准确无误的，高出正常被感染数据几十倍的感染CCP病毒，这难道不是中共地下党所谓吗？ 「作为香港地下党总指挥的香港工委，经过这二十年的运作，己经技巧娴熟，得心应手。破壤民主运动的步伐节节推进，势如破竹。香港民主派未能遏止地下蟒蛇的进攻，要吹响危险的的信号了」。她的预言非常准确。
