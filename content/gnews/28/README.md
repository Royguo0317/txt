《路德时评》2021年3月6日早间节目主要内容：

·美国对中共结束香港体制居然只是谴责

新闻一：

网易新闻网3月5日报道[1]：十三届全国人大四次会议4日晚举行预备会议，通过了大会议程，其中包括审议全国人民代表大会常务委员会关于提请审议《全国人民代表大会关于完善香港特别行政区选举制度的决定（草案）》的议案，至此，外界高度关注的如何落实“爱国者治港”，全面准确贯彻“一国两制”，将从完善香港选举制度开启。专家表示，倘《决定（草案）》此次经由人大表决通过，全国人大常委会或将在未来一到两个月内作出处理，制定改革方案。全国港澳研究会副会长刘兆佳认为，**这意味着中央已下定决心，从根本上改变香港长期以来的政治“游戏规则”，即让政权牢牢掌握在爱国者手中，让“反中乱港”分子彻底出局**。

新闻二：

美国之音3月6日报道[2]：美国政府星期五谴责中国修改香港选举制度的举措是对香港自治与民主程序的“直接攻击”，并表示美国正努力在国际社会“激励集体行动”来应对中国在新疆、香港等地的镇压。拜登政府已表示，美国必须做好准备，向北京为在新疆、香港的行动及其对台湾的威胁施加代价。
![]()![](https://gnews.org/wp-content/uploads/2021/03/香港李.png)[图片地址](https://www.voachinese.com/a/us-called-chinese-move-to-change-hong-kong-electoral-system-a-direct-attack-on-its-authonomy-20210305/5804011.html)
路德社评论：

1、《香港基本法》保障的就是英国在香港留下的三权分立制度，该制度让香港老百姓拥有真正的言论自由和结社自由，三权分立制度中的三条根就是独立的行政权、立法权和司法权。中共早已全面掌控香港行政权，现在基本控制立法权，但是中共还未全面控制香港司法权，因为香港最高法的12名大法官中有许多是英国国籍大法官，这也是西方相信香港法制的原因之一。

2、中共已经决定要从根本上改变香港的游戏规则，文贵先生在直播中说3月5日对新中国联邦有非常重大的意义，这重大的意义只有文贵先生和路德社才能真正明白。美国之所以至今没有对中共在香港犯下的滔天大罪采取行动，即使《香港人权与民主法案》通过后也是如此，就是因为香港大法官中还英籍大法官，但是现在一切都已经被中共改变。那接下来英美肯定会结束和香港的关系，所有海外资本都将从香港撤离。

3、中共一旦控制所有香港大法官，西方国家和资本都不会再相信香港拥有真正的法制，所以西方国家不可能再承认香港的国际地位，也不可能承认港币的合法性，就像文贵先生所说的，香港彻底结束了。

4、铁幕演说，是1946年3月5日，前英国首相温斯顿·丘吉尔在美国富尔顿城威斯敏斯特学院发表的反苏联、反共产主义的演说，运用“铁幕”一词之意攻击苏联和东欧社会主义国家“用铁幕笼罩起来”，因此此演说被称为“铁幕演说”。铁幕演说也被认为是正式拉开了美苏冷战的序幕。1947年3月12日美国“杜鲁门主义”出台，标志着以美国为主的资本主义阵营，与苏联为主的社会主义阵营之间的冷战正式开始[3]。丘吉尔在1946年3月5日的铁幕演说中对共产主义的形容非常适合现在的中共，中共继承了苏共和共产主义国家的所有劣行。

5、希特勒曾在1936年3月5日主动撕毁凡尔赛条约跨过莱茵河，试探英、法及周边国家的反应，但是德国只有10万军队，最后英法只对德国主动跨过莱茵河的行为进行谴责并没有采取实际行动，德国随后立即在国内进行战争动员，军队人数激增上百万。文贵先生去年就爆料中共早就想结束香港“一国两制”制度，中共利用香港制度的漏洞逐渐控制香港立法委，经过二十年的渗透推举出林正月娥成为中共的走狗，进而控制香港。

6、美国只有等香港大法官离任，香港彻底被中共控制后才选择动手，香港至今已有无数人献出了宝贵的生命，这是非常可悲的。文贵先生肩负着上天赋予的使命，如果不是他在2017年站出来揭露中共将控制香港，很多人还误以为香港仍然是民主法制社会，听了文贵先生的话提前离开香港的战友现在一定会觉得很庆幸。

7、历史总是在无情地重演，爆料革命现在所做的就是尽早让美国介入，拯救更多的香港人，尽力拖延中共的行动时间，争取让更多香港人离开香港，美国现在的举动就像当年英法两国，只谴责中共却没有行动，他们对中共的绥靖政策助长了中共的疯狂。

8、文贵先生最重要的品格就是尊重生命，美国之所以伟大就因为它虽然有先进的武器，但是美国可以做到放马归山，放枪入库。文贵先生虽然拥有巨额的资产，拥有巨大的政治影响力，但是他依然坚守做人的底线，持守信用，尊重生命，尊重人，这是他难能可贵之处。

9、中共的“完美犯罪计划”最关键的就是精心策划，他们早在几十年前就在精心策划今天的行动，中共后续肯定还会放出连环招，爆料革命就是要揭露中共精心策划的阴谋，当这些阴谋都被一一验证后才能让西方国家意识到中共的邪恶，才能彻底消灭中共。

10、从中共精心策划的角度分析，中共早在上世纪八十年代签署《中英联合声明》时就在欺骗全世界，从香港回归，到加入WTO，当中共国有资本后，再通过蓝金黄手段从全世界盗窃技术。爆料革命在此希望西方社会尽早醒悟，尽早意识到中共绝不可能收手，绝不可能自行改变，因为中共就是一群神经病的集合，习就是这群神经病中的神。

11、末代港督彭定康说[4]，“中共采取迄今为止最大的一步来抹杀香港在法制框架下实现自由和更大民主的愿望”。彭定康对中共的描述太过于肤浅，香港早已失去了民主与自由，中共接下来一定会把香港变成新疆，香港将迎来一场血雨腥风。
