![]()![](https://gnews.org/wp-content/uploads/2021/04/05B42C49-1DE8-4501-ADDB-FFF271CDB8FF_cx0_cy1_cw0_w1080_h608_s.jpg)

参考消息网4月15日一篇题为《境外媒体:克里访华为中美气候合作”探路”》的报道称，中共国生态环境部14日发布，应中方邀请，美国总统气候问题特使约翰·克里于4月14日至17日访华。期间，中共国气候变化事务特使解振华将同克里在上海举行会谈。

报道宣称，美国特使克里前往上海向中共国示好，是为确保两国领导人能在气候峰会中取得成果所做的高层准备。在气候变化问题上，如同在其他重大问题上一样，双方需要彼此。

事实真如中共所宣传的—克里访华是为了向中共北京示好，为了改善两国关系么？恰恰相反，笔者认为克里此行代表美国向中共发出最后通牒。

首先，克里的身份是前国务卿，即并非在职官员。至少看出以下两点：

一、 美国对于此次访问中共国并不重视，派出的只是半官方性质的代表，因此本质上谈不上是国事访问，也体现了拜登政府不愿和习政权过多纠葛，且两国间的正式对话渠道依然关闭；

二、 也正因为克里的半官方身份，很多正式渠道不方便说，不方便放的话，都可以由他来实现。反正说的是个人意见，表面上谁也代表不了，听不听是你们的事。

其次，访问的理由是交流气候问题这一无关痛痒的，不牵涉实质议题或内容的项目。表明美方根本无意与北京进行实质性的、建设性的会谈，而仅仅借一个由头而已。

第三，出访的时机正值美国跨党派和军情部门达成共识—中共国是美国最大的挑战和竞争对手—之际。一方面美国正式以官方形式提出中共不代表中国人和海外华人或者华裔美国人的概念；另一方面，美国认定中共对新疆维吾尔族等少数穆斯林族裔犯下种族灭绝和反人类罪；更为关键的是，从最近一段时间美国左中右各个主流媒体以及跨党派官员、学者等对于中共病毒真相与追责咬住不放的的背景下，拜登政府即便有与习政权勾兑之心，也万无勾兑之胆。况且，拜登政府也已明白，灭共是解决美国国内矛盾和实现其利益最大化的必须。

接着，让我们分析一下克里此行的目的：

一、 警告北京不得单方面改变台海关系现状甚至武力犯台，否则美国将联合盟友坚决捍卫台湾。我们看到，近期拜登政府不断提升与民主典范台湾的交往层级和热度。国务卿布林肯也放话必保台湾。

二、 奉劝北京在香港问题上回归到《中英联合声明》的轨道，恢复香港的民主、法治、自由。美国对于中共侵蚀香港自治地位的谴责和制裁一刻并未放松。

三、 要求北京释放所有被非法关押的维吾尔族等少数民族，停止种族灭绝和反人类罪行。

四、 最后也是最重要的，要求北京公布所有中共病毒原始数据，开放实验室给美国和国际社会进行调查，明确病毒真相和追责。

克里的目的就是向中共发出最后通牒：放弃一切幻想和抵抗。否则，用通俗的话讲就是：你(中共)想怎么死？是你自己动手还是我来帮你？
