**新闻主要内容：**

美东时间1月4日，纽约证券交易所发布最新公告，将不再继续针对中共国三大电信运营商实施摘牌行动，三家上市公司将可以继续在纽交所上市和交易。但纽交所表示将继续评估13959号行政命令对这些发行人的适用性及其持续的上市状态。消息公布后，在香港上市的三大运营商涨幅超过5%。

伦敦时间1月4日，指数提供商富时罗素表示，根据美国川普政府禁止美国投资被指有军方背景中资企业的行政令的指引，将从其全球股票指数中进一步删除三家中共国公司的股票。自周四起，富时全球股票指数和富时中国A股指数将删除中国联通、中芯国际和南京熊猫。

**简评：**

1、2020年12月31日，纽交所宣布将对中共国三大电信运营商启动摘牌程序。摘牌的依据是2020年11月12日美国总统签署的13959号行政命令，要求从2021年1月11日起，禁止美国人投资任何和中共军事相关的证券，或基于该证券的衍生品。纽交所刚给中共送完新年大礼，没过几天就发生180度大反转，停止摘牌行动，令人大跌眼镜。这说明中共有能力干预美国内政，能让川普总统“政令不出白宫”。美国国内和中共勾兑的暗势力仍在押宝拜登当选，妄图将来推翻川普总统的行政令，继续在资本市场给中共“输血”，和中共相向而行，走“共同富裕”的道路。

2、1月5日港股收盘后，中国电信上涨3.35%，中国移动上涨5.13%，中国联通上涨8.5%。以涨幅最高的中国联通为例（如下图），1月4日，受纽交所摘牌的消息影响，股价跳空低开，不过并没有低开低走，反而有一股神奇的力量托住股价，震荡吸筹，下午迅速拉升，当日回补缺口。截止收盘，中国联通上涨0.45%，当日成交2.39亿股，远超过12月以来日均5000万股左右的成交量。从1月4日的股价走势来看，并没有显示出“泰山压顶”之势，反而有很多资金“看好”中国联通，大幅买进，让股价在日线级别的支撑位附近收了一根阳线，好像川普总统的行政令失效了一样。1月5日，中国联通跳空低开2.5%，让一些多单止损后，股价开始上涨。受纽交所停止摘牌的消息影响，联通股价加速上涨，盘中最高涨幅10.74%，当日成交2.92亿股。

3、从股价走势来看，中共早已成竹在胸，通过暗箱操作，让川普总统的行政令失效，在股票大跌之时进场布局，等第二天纽交所发布消息配合股价暴涨，既秀了自己的“实力”，“打脸”川普总统，又让自己赚得盆满钵满。

4、纽交所首鼠两端、配合中共行动的同时，指数提供商富时罗素继续除名中概股，坚决执行川普总统的行政令。被踢出富时指数的股票包括中国联通。中美双方在金融领域的争斗已经在全世界范围内展开，接下来还会有各大机构、公司在灭共力量和中共之间做选择。

5、中美双方到了角逐的关键阶段，互有胜负都是很正常的事，就像拔河比赛，两边势均力敌，此消彼长。整个灭共过程就像一部精彩绝伦的电影，主角历经磨难，到了最后才将对手一击毙命。所以，不必为了中共表面上占了上风而沮丧。谁笑到最后，谁才笑得最好。三大运营商被暂时免除摘牌，反而会让更多人看清中共的邪恶，加速民意的觉醒，让灭共的队伍更加强大。中概股最终的宿命不仅仅是被摘牌，还会面临巨额官司。
