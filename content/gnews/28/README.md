一些人谈到川普总统的时候，通常会说：

“他不是总统候选人”

“他很粗鲁，很不仁慈。”

“他不是一个好人。”

所有这些事情及更糟糕的是，他个性特点完全掩盖了他的优秀记录，就好像把一个坦克放在了储水池中给他洗冷水澡。

但真相真是这样的吗？

这是不是真正的问题？我们看看在过去的四年中，他是不是一直带领这个国家在朝着正确的促进经济发展方向发展？我们可以把这些记录和这个男人分开吗？ 我们应该为数以百万计的美国人因为总统的不拘礼节的举止和性格（他的粗鲁，吹牛，和虚张声势的事实所困扰）感到羞耻。

这些都是没有道理的。总统的行为举止往往会削弱总统任期，和总统职位，大多数美国人只想要一位让他们感到骄傲的总统。

我们也更喜欢像温斯顿·丘吉尔、德怀特·艾森豪威尔、富兰克林·罗斯福、罗纳德·里根或巴拉克·奥巴马（本报在 2008 年和 2012 年热情支持）的一流气质和举止。但他们今年都没有参加选举。

让我们看看川普总统的记录：

在发生中共病毒前，美国在唐纳德·川普总统的领导下，经济蓬勃发展、繁荣，是自20 世纪 50 年代以来从未有过的。看一下过去三年中你的 401(k)退休福利计划就知道了。

并且没有任何一任总统能让美国黑人的失业率低于以往任何时候。

在川普总统先生的领导下，我们的贸易关系有了很大改善，我们的贸易协议也被重写了。多亏了他，中美洲再次出现在地图上，阿巴拉契亚人和小时工都有了希望。

川普总统为这些受苦的同胞做的还不够好吗？不。但他了解他们，也许他不太清楚，但他知道到他们的痛苦。

从来没有人问过美国人民或“低空飞行”国家的人民是否想把工作送到国外——直到川普总统。是他提出让完全不受约束的自由贸易转向管理和公平贸易。正如他所说的那样，他把美国放在第一位。

他还履行了任命美国最高法院法官承诺。他第三次任命的是艾米·康尼·巴雷特，这是最好的——法学家，她的思想、性格和学术都是一流的。我们希望她反对司法和行政过度行为。

最后，让我们谈谈本地区最重要的关切之一———能源。在川普总统先生的领导下，美国在我们大多数人的有生之年首次实现了能源独立。如果没有壳牌石化综合体（”裂解工厂”），还不知道宾夕法尼亚州西部在哪里？

诚然，唐纳德·川普不是丘吉尔，但他是真实的，并且他完成了他应该做的事情。

他不是统一者。他经常表现得像个人的总统，而不是整个国家。他没有采取任何措施来减少我们的分歧，事实上，他经常加深分歧。所有美国人的信念和智慧应该受到所有美国人，特别是总统的尊重。

川普总统完美地处理了疫情？不。但没人能控制流行病。总统过去是对的，我们不能害怕疾病，我们必须保持美国的开放和工作。

他没有听那些以为可以帮助他的人。他也没有听政府官员的，只做自己应该做的。

但拜登-哈里斯是让我们提供了更高的税收和做保姆国家，他们向恶霸独裁和被唤醒要摧毁历史的人下跪，而不是从历史中学习和建设国家。

他们结束了加州的分裂和让加州实现经济繁荣梦想并给最需要工作的人一份工作。“高薪绿色工作”可能不是匹兹堡、克利夫兰、托莱多或扬斯敦的工作。

他们对中共很柔和，但是川普总统非常明白中共是我们的敌人。

对于这个工作来说拜登太老了，而且很虚弱。他很有可能无法完成这个任期。川普先生也老了，但看起来很健壮。但如果需要，川普总统有一位副总统迈克·彭斯先生他是一个非常安全的助手。

而卡马拉·哈里斯参议院没有提供准备成为总统的证据。

自 1972 年以来，这家报纸一直不支持共和党人竞选总统。但我们相信川普总统尽管有缺点，但今年没有更好的选择。我们尊重并理解那些有不同感觉的人。我们希望我们能更有热情，我们希望总统能变得更有尊严和政治家风度。每个美国人都必须下定决心，做他或她认为对社区和共和国最有利的事。用你的良心投票，不管发生什么，都要相信，信任这个国家。
