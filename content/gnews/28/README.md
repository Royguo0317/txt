現任副總統邁克·彭斯大家都比較熟悉，他原為印第安納州州長，2016年經川普提名成為副總統。彭斯副總統是虔誠的天主教徒，堅定嚴謹，邏輯嚴密，有條不紊，深為川普總統和同僚讚賞，是川普總統的得力助手。
