在1983年的“邪恶帝国”演讲中，里根总统将苏联称为“现代世界的邪恶轴心”。

美国第40任总统里根拒绝接受这样的观点，即世界上两大超级强国应该对冷战和两国间正在进行的核武器军备竞赛负有同等责任。相反，他将这场冲突定义为正义与邪恶之间的战斗。

38年后，在里根研究所最近的一次线上会议中，美国阿肯色州参议员汤姆·科顿再次使用了 “邪恶帝国 “标签，但此次针对的是第三大超级强国— 中共。

参议员汤姆·科顿在一份长达84页的报告中提到：”我们必须打败中共这个邪恶帝国，把它扫进历史的垃圾堆。” 他称美中争夺全球主导权的斗争是一场决定世界命运的较量，是一场旷日持久的终极之战。

美国新任总统拜登上任后的首次中美会晤于上周在阿拉斯加的安克雷奇举行，双方敌意明显。尽管一位美国政府高级官员表示，会谈是 “实质、严肃和直接的”，但南京大学国际关系学院教授朱锋认为，本次会晤一开始就有 “浓浓的火药味”，而这并不出人意料。

美国国务卿安东尼·布林肯（Antony Blinken）指责北京破坏了全球稳定，而中方代表杨洁篪则表示，美国因为其自身的内部问题，没有资格居高临下同中(共)国说话”。

布林肯的前任、前国务卿迈克·彭佩奥（Mike Pompeo）在川普政府期间采取了强硬的态度，他甚至将当前的国际事务状况与1936年纳粹德国崛起时的世界相提并论。

彭佩奥表示，允许中(共)国举办明年的冬奥会，将给北京带来类似希特勒帝国在1936年柏林夏季奥运会后所享有的信誉提升。

彭佩奥还提到，”我们了解这段历史，我们中的那些曾与以色列的朋友们合作过的以及那些致力于反犹太主义的人们都曾说过，绝不允许上世纪30年代发生的事情再次发生”。

彭佩奥认为，国际奥委会有责任确保这些奥运会东道主应是当之无愧的，而他坚信中共国政权不配。

彭佩奥表示，不反对与中国人进行对话，但反对将对话或言语误认为是行动。川普政府的强硬政策包括通过对违反基本贸易准则的中(共)国产品征收关税来惩罚这个共产主义国家。

美国田纳西州参议员玛莎·布莱克本一直站在体育争议的前沿来讨论美中关系。近一年来，布莱克本参议员一直在质疑为什么美国国家篮球协会（NBA）与一家受中共摆布的电视公司达成协议，而与此同时，由于涉及到新冠病毒的起源，对中(共)国颠覆美国的行为的调查仍在继续。

去年夏天，这位田纳西州参议员致信美国国家篮球协会，质疑该联盟与中(共)国之间的关系。她提出，国家篮球协会与中国央视的新协议是在有报道称这个国家囤积防护设备、禁言医生、并谎报中共新冠病毒人传人风险的情况下做出的自私之举。

布莱克本表示，”虽然国家篮球协会一直在努力提高国内对社会问题的认识，但人们担心联盟对在国外犯下的侵犯人权行为视而不见，去年甚至因中共施压而低头。”她还提到，”国家篮球协会和一些球员的行为给人的印象是，该联盟将利益置于原则之上。”

美国印太司令部司令菲利普·戴维森（Adm. Philip Davidson）上将最近在参议院军事委员会成员面前作证时，称中(共)国是 “21世纪对安全的最大长期战略安全威胁”，并称中共可能会在这十年内试图控制台湾。

戴维森上将的不祥预言与彭佩奥等其他高级官员的警告相呼应。彭佩奥曾说，中(共)国确实对我们国家构成了最大的生存威胁，拜登政府需要运用川普政府的强硬政策，以确保美国的自由，保护美国的勤劳公民。
