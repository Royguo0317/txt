据硬盘资料显示，上面描述的照片场景摄于2016年4月4日，亨特拜登正在跟奥巴马谈论，关于中共要在南海建立军事基地，要求奥巴马不要表态以默许，这样中共会给好处费10亿美元。看那画面奥巴马显然没有反对的意思。据说在谈话后的25小时后亨特就收到中共10亿美元的汇款。而2016年4月1日习近平及其团队刚到华盛顿开核武论坛会议，之后中共一直坚称是美国默许了他们在南海的军事化.

贵为一国之君，奥巴马对亨特应该是俯视之态，但是相反照片里的奥巴马低着头一副欣然接受的姿态。可以想象只有在巨大利益和诱惑面前，他才能有如此态度。利益和贪婪显然已经打垮了奥巴马。10亿美元就出卖了国家利益！10亿对于一个国家来说数额并大，但对奥巴马私人而言却是相当大一笔钱。据说奥巴马上台之前家产只有40万美金，卸任时竟高达四千万美金。这钱都从何而来，不言而喻。他肯定高兴无比，在离任前还能发一笔横财，尽管那钱只是10亿的零头，大头都给中间人拜登父子分了，而零头也是通过其他所谓做顾问、演讲等表面合法的方式分批拿到的。

据说这个视频是亨特拜登拍的，他发给了中共，这样既可以作为证据，又可以向主子邀功。同时乔拜登也有一份视频，一来他怕儿子独自吞掉这笔钱，有了此证据就可以控制奥巴马，即使奥巴马离任以后也要为拜登家族服务。怪不得前不久奥巴马还厚着脸皮在为拜登竞选总统演讲背书。

细思极恐，这个视频太可怕了。白宫总统办公室居然被拍了视频，如此安保严密的地方，说明整个白宫国会及参众两院相当大部分官员的黑材料，可能都掌握在拜登手上，怪不得川普说沼泽很恶很深。

习总是脸上挂着笑，看来是有原因的。中共在美国渗透如此之深，布置如此严密，本以为胜券在握，拜登当选世界就成了他们的天下。万万没想到半路杀出个程咬金川普，一下打乱了中共的计划。习和拜登以为硬盘的内容没有别人知道，就算有人泄露，新闻媒体也都被蓝金黄了,都拿所谓“政治正确”当借口而拒绝谈论甚至封杀硬盘门事件，主流媒体特别是一些左媒已经彻底沦为中共的工具。哪知道2017年郭文贵先生在自媒体亮相开启了爆料革命，他们的内幕早被文贵先生所掌握，而且用自己打造的新闻媒体平台爆出硬盘内容，与美国川普及正义人士遥相呼应，随之而来的终极之战一定会以正义一方告终。

文贵先生早就说美国大选会以滑稽之势收场。以共灭共，“江习之争”，以美灭共如今都成了事实。文贵先生说自己肩负着上天的使命，毫不为过。硬盘门事件在美国联邦调查局已经压了10个月之久，如果没有文贵先生的爆料，拜登父子与中共勾结及其卖国叛国的犯罪事实，恐怕美国人民永远都不知情。拜登父子会永远以好人善人的政治精英形象展示在人民面前。说不定拜登这次如果当选了总统，那么他和中共狼狈为奸，世界就将彻底被魔鬼控制。中国人民乃至世界人民都会被带入地狱的深渊，人类文明甚至会从此万劫不复。

我们真的感谢上天赐予我们文贵先生，除了他谁有这个实力和智慧。更重要的是，文贵先生站在了正义的一方。因为文贵先生的爆料，硬盘门事件将持续发酵，未来更多美国CIA，FBI，国会，华尔街，媒体界等等各条大鳄在沼泽慢慢抽干后而再也无处遁形。

我们的终极之战一定会以正义之师的胜利而告终。
