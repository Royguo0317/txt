赵婷说：“我记得很清楚的是《三字经》，第一句就是‘人之初，性本善’。这六个字对小时候的我影响极大，至今仍深信不疑，哪怕有时候现实好像相反，不过我总是能在世界各地遇到的人中发现良善。”

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/01054324/117336656_hi065503971.jpg)图片来源：BBC

我们都知道，中共独裁政府从建国初期就利用阶级划分让民众自我牵制，利用对人才价值的划分让民众为其效力，利用地域划分控制人口流动，利用宣传手段控制民众正常思维⋯⋯，在没有自由民主的环境下，让无数有才能的仁人志士竟折腰！

郝海东夫妇在路德4月25日的访谈节目中提到姚明时说：“给你一些政治待遇，包括一官半职，让你在不知不觉当中成为了它的附属。”他还说：“你想当个什麼主席，当个体育局什么副局长局长，如果后面没有根，没有你的家族势力，没有这些常委的支持，那是没有任何可能的。由此可见，中共历来好用顺我者昌！”

郝海东先生也提到荣国团——我的同乡和儿时偶像，第一个获得男子单打世界冠军的乒乓球运动员。当年他的一句名言“人生能有几回搏”，令他怀着报效祖国的梦想踏上了一条不归路。但因為与贺龙有交情，十年浩劫中树倒猢猻散，他在经历了无法抗拒的肉体和精神双重折磨，以及心理上难以接受的威慑后，最终自杀身亡，英年早逝！这就是中共惯用的除异己的手段，逆我者亡！
