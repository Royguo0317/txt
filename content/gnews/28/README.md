## 中共的欺騙加劇人們對種族清理生物武器試驗的恐懼
`2020-05-15 13:37 GM10` [轉載自GNews](https://gnews.org/zh-hant/204189/)

**作者：Dogman**

此文為比爾格茨於05/14/2020在《華盛頓時報》文章的譯文。

原文請查：https://www.washingtontimes.com/news/2020/may/14/china-deception-fuels-fears-biological-weapons-eth/

根據前任和現任的美國政府官員透露，中國政府在CCP病毒爆發以來的欺瞞行徑加劇了人們針對北京開發生物武器的擔心，而這項針對特定人群的生物武器研究有能力定點打擊某些特定的種族。

一位川普總統政府高級官員告訴《華盛頓時報》稱，已經知道中國政府實施了包含定點打擊特定族群的秘密生物武器等項目。
這位官員不具名透露，“我們正在觀察這些潛在的針對少數民族的生物實驗”。

這些關於生物武器研究的細節來自於跟中國這些項目直接相關的工作人員，而且這些信息都被囊括在情報報告之中，可能在未來會對外解密。

這位官員說，中國政府針對CCP病毒的反應，提高了人們對於其秘密生物武器研制的擔心。北京掩蓋了此病毒的高傳染性，並且推遲了對外借的預警。

“我們對於中國政府能夠遵守BWC和其國際職責表示擔憂，”這位官員說，他所引用的BWC是《生物和毒素武器公約》，這是一項用於制止研發和生產生物制劑的國際公約。“如果我們只從這次病毒危機中學到一件事，那就是我們不能相信中國政府會做正確的事情。”

中國與1984年簽署了《生物和毒素武器公約》，隨後超過100個國家加入這項努力之中。其中公約的第五章中要求，簽署國需要“咨詢並且配合其他國家，來解決任何與生物威脅相關的問題”。

自從12月份病毒在中國開始爆發以來，中國政府一直拒絕國際專家對其針對其武漢試驗中所進行的蝙蝠冠狀病毒研究的調查要求。
中國駐聯合國代表團發言人對於尋求評論的所有電話和郵件均無回復。

國家安全局的針對國際生物武器安全條例方面的多位官員已經擔心多年。

與《時報》聯系的情報人員支出，中國政府在2011年曾向聯合國表達過，對於科技的迅速發展所導致的針對某些特定種族的生物武器和其他病毒武器的研制活動，北京當局也表示非常擔憂。

中國政府在文檔中沒有說他們自己也有正在進行中的生物武器項目，但是美國的情報官員和其他的外國事務專家說，北京存在並持續存在此類項目。

一些人說，在2011年中國政府官員在聯合國發表的指南中提到允許外國政府參與其科研活動，但是當時北京當局並未對外公布。
在這份題目為《阻止生物威脅：我們能做什麽》的文檔中，為了“與病毒鬥爭和改善健康”的科學創新可能會被用於有效的病毒攻擊，中共官員對此表示非常擔憂。他們還引用了“靶向藥物輸送科技導致病毒更容易傳染”，並且“特定族群的基因標記”和“人造病毒”。

這份聯合國指南是參與了《生物和毒素武器公約》的12個國家聯合編撰的。一個附錄提到，“同慣例相關的新科學和科技創新”包括各國提交資料的背景信息。

多位參與人員揭示，目前持續中的研究可被同時用於進攻型和防守型的生物武器應用的研發。

美國透露了與基因物質和微生物操控，日益升級的溝通，建立信任和科學規範，生物安全的輸出控制和改善等方面公的最新進展。
一位現任和兩位前任的官員說，中國政府發展針對特定族群的生物武器的可能性令人不寒而栗。

寶拉狄薩特（Paula DeSutter）是一位負責審核，遵守和執行的前任助理國務卿，她說中國政府在2011年的文檔中“已經承認在進行相關工作，而這並不讓人驚訝，因為有傾向表明他們在研究包括針對特定種族的生物武器。”

“令人非常害怕的是，CCP病毒可能是在武漢針對這項生物武器進行的測試，而且這項測試可能從2011就開始了”，狄薩特跟《華盛頓時報》說。這個說法看來被中國軍方的信息不斷加強和證實。已退休的張仕波將軍在2017年出版的《戰爭新高地》一書中說，生物科技的發展，不斷加大進攻型生物武器的可能性，包括針對“特定族群攻擊”的威脅。

美國官員在一次近期的國務院年度武器控制報告的執行摘要中表示，他們對於中國政府的生物武器項目表示非常擔憂。報告中說，“在報告進行期間，中國政府參與了雙向功能生物武器的活動，其中一些加劇了對其是否能夠遵守BWC的擔憂”。第一章內容表示，BWC的簽署國答應永遠不會研制，生產或存儲生物和毒素武器，或者以生產武器為導向的研制武器和設備。美國國務院2005年，2010年和2020年的報告指出，中國政府持有一些用於進攻型生物武器的物質違反了BWC協議。2020年的報告中還指出，中國政府自己發表的建立信任機制的報告中沒有說明北京當局過去進行的生物武器項目和保留至今的這些武器。

一位川普內閣的前高級官員不具名表明對於中國政府一直進行的針對特定族群和其他的相關生物武器研究非常擔憂。這種擔憂引起國務院的2019年度武器控制報告的註意。這位前官員還說，在此之前，國務院和情報部門極大的忽視了這些情況。盡管中國政府在2011年遞交給聯合國的指南中沒有明確提供北京當局試圖發展種族清除生物武器的直接證明，但是這位前官員說，中國軍方發表的側重生物戰爭的文獻引發嚴重的擔憂。“這是一項有雙重功能的研究，是中共幽靈具備進攻能力，”這位縣官員說。“中國政府無法證明這項針對種族的研究有著和平的應用。”中國政府沒有對外界公開這份2011年提交給聯合國的指南。但是，中國駐聯合國大使何亞非當時寫，密切關註生物科學和技術的發展是維持BWC有效性的必要條件。“在條例的框架下，中國政府支持加強關註和評估生物科技創新的影響，以預防生物科技被用於恐怖行為，並且使其服務與人類”，何先生當時寫道。

中國政府曾經說，他們正在研究合成性生物學，專家相信這項研究可以被用於編輯基因序列，以達到制造可以感染特定人群的病毒。一份2016提交給生物安全回顧會議的報告中指出，生物和毒素武器威脅並非抽象，也不僅僅是存在與理論中。
“科技的發展和日益廣泛傳播的關鍵原料，儀器和知識導致此類武器觸手可及”，這份報告中指明。
“過去使用的生物武器表現出令人恐懼的效果，現在有清晰的證據表明，恐怖組織，一些個人和國家持續不斷的謀求這種可惡的武器。”

北京當局參與鎮壓了多個上述民族族裔，包括新疆人和中國西部的其他穆斯林。目前估計有1百萬穆斯林人被關押的集中營中，而中共官員將其稱為再教育中心。中國政府還針對西藏人民進行了鎮壓。中國軍隊吞並了西南區域，而很多西藏人保持對達賴喇嘛的忠誠，並且一直尋求獨立。中國政府2011年提交的關於BWC的信息在一份基於日內瓦的BWC網站上的報告中可以查詢到。
英國和加拿大政府資助了這項活動，而報告中包括相關專家的有關發現。
