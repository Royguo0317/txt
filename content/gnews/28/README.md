**去年统筹铜芯罩生产卷利益输送疑云HKRITA获政府拨412万研发可再造纺织物**

去年中卷入政府亿元「铜芯口罩」利益输送订单疑云的香港纺织及成衣研发中心（HKRITA），同年底获创科基金批出一新项目研发可循环再造纺织物，「用于生产（含肥料）纤维素超吸水性聚合物的中试规模系统之开发」平台研发项目, 当中业界赞助106万元、政府资助412万元。

**斗地主| 港府粉岭塘坑村强行收地**

粉岭塘坑村一间运输公司昨天突遭地政总署收地。运输公司东主指称在仍未收取收地赔偿及特惠津贴情况下，昨天突有数十名地政总署人员到来收地及拆屋，双方一度对峙，工人稍后更拆除写字楼的顶层货柜，东主父子今日透露，与地政总署商讨后，限时三天要他们搬走及清理私人物品，但迄今仍未倾妥赔偿事宜。
