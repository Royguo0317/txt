**二、本期专题   加拿大3-5岁孩子心理发展，情绪控制，学校教育情况**

加拿大学校及幼儿园充分体系了一切以孩子为本的宗旨。语言不通的孩子刚来到加拿大上学，听不懂英语，学校校长主动找到家长说可以请老师专门来教孩子语言，让孩子继续留在本校学习，而不需要上专门的语言班。这是Farmsister大姐家孩子的例子。

老江则主要讲了3到5岁的孩子情绪控制问题。中共国盛行攀比，大人的歧视偏见等对这个年龄段的孩子造成了不良影响。中共国的孩子大部分情绪控制能力差，反观加拿大的家长非常重视培养孩子自我调节和控制情绪能力。大部分学前班的孩子都能养成良好的情绪习惯。

Jackie大姐讲了加拿大对弱势儿童的包容、尊重和特殊照顾。这里没有中共国对先天发育不良(唐氏综合症)的孩子不尊重的称呼，例如残疾、傻子、痴呆等。在加拿大，全社会尤其是学校对弱势孩子极其尊重和包容。老师会细心观察问题儿童，进而建议家长带孩子去相关机构做鉴定。政府对于特殊孩子有额外照顾，例如自闭症儿童可以得到相关的活动费用报销，特殊孩子进入社会后可以享受终身免税政策。

老江补充到加拿大是一个有爱心的国家。亲人之间经常相互表达我爱你，家庭亲人的关系比较和睦。他自己也是身体力行，经常对家人表达喜爱、欣赏之情。他认为我们不应该吝啬赞美的语言，要勇于对自己最亲近的人说出感性的话。

Jackie大姐回想起刚来加拿大遇到的一个例子，一位银行工作的爸爸休产假的事情。这位有着良好工作的爸爸对Jackie大姐说他要休一年半的产假时，着实震惊了她，在中共国从来没有听过父亲休产假的。仅产假这一方面，足以说明加拿大的福利在世界上处于都是领先地位。
