美国大选惊心动魄，牵动着全世界人民的心，也让世界深入了解着美国民主选举制度的程序。拜登领先但涉嫌造假，川普落后已开展法律行动。本年度大选，正在重演2000年由法律裁决总统的一幕大戏。

法治、民主和自由是美国的代名词。美国以基督信仰立国，在一定程度上代表着当今世界上最高的民主体制。尽管随着全球化的进程，一些负面价值观、政治制度实际上给美国带来很大的负面影响，但美国有神和法律的佑护，终究会回归有利于美国人民的固有价值观。

法律和秩序是社会稳定的基石，民主和自由应该在法治基础上才可能真正实现，大选也不例外。针对本次总统大选的川普法律行动，刚刚重新当选的参议员、参议院司法委员会主席林赛·格雷厄姆（Lindsey Graham）宣布捐款50万美元支持川普法律诉讼，这就是对美国法律的巨大信心的体现。

共和党参议院多数党领袖麦康奈尔11月6日发推，也表达了对大选选票合法性的关切。他这样说：“这是在美国这个伟大国家必须采取的措施：应该对每一次合法的投票都进行计数。 不得非法提交任何选票。 各方都必须观察这一过程。 法院在这里适用法律并解决纠纷。这就是美国人投票决定结果的方式。”
