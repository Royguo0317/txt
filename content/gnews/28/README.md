自拜登总统上任以来的首次面对面会议中，美中在阿拉斯加结束了为期两天有争议的会谈。两国关系在西藏，香港，新疆和台湾从贸易到人权，以及中共在南海和中共病毒疫情大流行上的态度等一系列问题上都存在分歧。

国务卿布林肯说：“我们的盟友和合作伙伴对中共令人担忧的行为表示重大关切。中共的行为会威胁到维护全球基于规则的稳定秩序。这不仅是中共的内部事务，我们今天有义务在这里提出这些问题。我们非常明确地列出我们自己的政策，优先事项和世界观。”国家安全顾问杰克·沙利文称中共已对“基本价值观进行了攻击”。中方代表杨洁篪激怒地回应：“美国境内的许多人对美国的民主制度缺乏信心，美国应改变自己的形象并停止在世界其他地区推进自己的民主。中共国不会接受美方的无端指责，挑战中共是死路一条。”

在香港人权问题，新疆种族大屠杀和实验室制造生物武器的事实面前，中共战狼外交部再次向全世界展现了他们毫无底线的流氓嘴脸。全球联合灭共已是大势所趋，看中共还能强硬多久。
