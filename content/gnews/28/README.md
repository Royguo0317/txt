## 華為對英國電信行業的滲透已經深入骨髓
`2020-07-11 08:08 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/260188/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/11080152/1-37.png)圖片選自： [https://www.telegraph.co.uk/technology/2019/04/30/vodafone-admits-finding-hidden-back-doors-huawei-equipment/](https://www.telegraph.co.uk/technology/2019/04/30/vodafone-admits-finding-hidden-back-doors-huawei-equipment/) 
據衛報（theguardian.com）報導：英國電信（British Telecom）高管表示，如果拆除設備，英國的5G，4G和2G客戶將面臨很大的問題。

現在人們認為鮑里斯·約翰遜（Boris Johnson）正準備向保守黨妥協，他們反對5G網絡中有華為設備，並草擬計劃截至2023年要將中國公司的設備佔比降到零。

儘管華為一直在試圖證明自己是一家完全獨立的員工擁有的組織，但華為與中共政府的聯繫使其受到了嚴格的審查。英國政府1月宣布，儘管存在安全擔憂，仍會允許將華為設備用於英國的5G網絡的建設，只不過要求華為設備份額不超過總網絡的35％。該決定對沃達丰和英國電信是個福音，因為兩家公司目前各自的網絡中華為基站約佔60％。

英國電信一位高管告訴國會議員，如果要求截至2023年刪除所有華為設備，英國將面臨手機“停電” 。英國電信首席技術和信息官霍華德·沃森（Howard Watson）在向下議院科學技術委員會發表講話時說，三年內將中國公司從英國的手機基礎設施中撤離的時間表是無法實現的。他說：“ 三年內達到零將意味著全國4G和2G以及5G客戶的服務間斷，在短時間完成這麼多的站點操作是不可能的。“

沃達丰（Vodafone）英國網絡負責人安德里亞·唐納（AndreaDonà）證實了沃森的警告，他說，這些工作會造成通訊服務的間斷，“有時會持續好幾天，取決於改造所需要的工作量” 。

華為的英國首席執行官傑里米·湯普森（Jeremy Thompson）在被主席格雷格·克拉克問及英國華為員工是否可以自由發表意見時，湯普森回答：“是的，我們在英國擁有一個管理團隊，就像其他任何英國組織一樣，我們可以自由發表意見，是的。”克拉克問：“那麼您對香港的新安全法有何看法？” 湯普森拒絕回答。
