## 【观点】中共的心灵鸡汤，是愚民的毒
` 墨尔本雅典娜农场` [轉載自GNews](https://gnews.org/zh-hans/894852/)

作者：文纸鹤
编辑：翼族
![]()![](https://gnews.org/wp-content/uploads/2021/02/Picture2-3.png)图片来源：http://zqb.cyol.com/
生活中常常听到一个词——心灵鸡汤。看一看心灵鸡汤的文章或故事，往往会让人的心灵感到十分“温暖”，具有精神安慰作用，以及动机（激励或颓废）强化作用。但是中共的心灵鸡汤，就是愚民的毒了。我的青少年时期就经常被这样的心灵鸡汤感染，比如我最初知道的《读者》、《知音》、《青年文摘》……

《读者》1981年由甘肃人民出版社丛刊编辑部创刊，原名《读者文摘》，截至2019年8月《读者》杂志的累计发行量超过20亿册。《知音》杂志社1985年靠湖北省妇联拨付的3万元开办经费起家，这本情感类杂志以悲惨曲折的爱情故事以及名人轶事等内容，多年稳居国内期刊发行量第一、世界第五的位置。而《青年文摘》则由共青团中央主管、中国青年出版社主办，创刊于1981年1月，是一本以青少年为核心读者群的文摘类综合刊物，刊物集萃来自报纸、期刊、图书等大众媒体，是中国发行量最大的青年杂志，单期发行145-150万册。

在中国一切媒体都听党的，所有的报刊杂志自然也不例外，这几本畅销杂志的发行量之大，毒害之深，不可轻视。我刚开始阅读这些杂志里的一些文章时也常常觉得感动，比如，故事A里的主人公太悲惨了，相比之下自己已经是很幸运了，知足吧；故事B里的主人公好幸运呀，只要听话，继续努力，是金子就会发光的，只要默默付出最终也会幸运的……但隐隐之中，我也有点好奇：这些故事也太巧了吧！慢慢我明白过来，这些心灵鸡汤都在有目的的侵蚀人的思想，心灵鸡汤好喝，只是因为放了很多味精。这种愚民之术，让人们在感动、感恩的时候不知不觉就中了毒，把感谢党、感谢政府、感谢领导也融入在内了。

后来有了电视，又有了互联网，接触到的信息越来越多，但是这类心灵鸡汤不但没有减少，反而更加充斥着人们的眼与耳。长期以往，一个人就患上了“斯德哥尔摩综合症”，给他解放他都不敢接受，就像某某歌词唱的那样“就算是生活给我无尽的痛苦与折磨，我还是觉得幸福更多…… ”听的我毛骨悚然。

身在中国，却在中共治下，每个人的身心都双重受毒：空气污染、水污染、食品污染，还有要命的CCP冠状病毒；而另一方面，中共思想之毒更是可怕，它侵蚀着每个人正义的灵魂！爆料革命四年来我学会了听其言观其行，我向长辈求证他们的亲身经历，因为他们用时间验证了真假有分，善恶有分！

《新中国联邦宣言》，给出了闪亮的信仰指引之星。灭共，需要觉醒，需要实力，需要智慧。只有戒掉了中共的思想之毒，跟随七哥和爆料革命，才能真正觉醒，才能增强实力，才能增长智慧！
