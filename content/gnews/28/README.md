## 【小象评奇文】六一节党给孩子送来了红色的毒礼包

六一节来了，中共号召全国小朋友“看动漫，学党史，让我们随着动画片中的主人公一起，做一个听党话跟党走的好孩纸吧！”小象很想问一下，党要带我去哪里？党又不是人，不会说话，不会带路。小象看到，中共的宣传部，换了手法， 对互联网里长大的小朋友用了动漫的方式，更直观更容易小孩子理解。以前用唱歌，升旗，跳舞的方式，有点老套了。还考虑到小孩子注意力没有那么长时间的集中，分成了很多小短片。

小象觉得这个话题都没有什么好说的了，小孩子肯定不会主动去看，幼儿园和小学学校就会不断地强行灌输，这种扭曲的有毒的洗脑手法，长期折磨着每一个在中共国生长的孩子，最可怕的是这个组织绑架者老百姓的一生，你的学习，你的工作，你的家庭，你的收入，你的去世。这才是很多大人成为中共教育的帮凶，他们总是教育孩子：大家都是这样，不要试图去改变社会，你要适应社会。文贵先生昨天在谈及一个国内人士终于投奔新联邦的时候说,评价这位富有位高的人士，“他最终是怕共产党的。站到新联邦这一边，不是他的初衷，是他中和评估得失的结果”是的， 文贵先生的勇气，一直鼓励着我们。为何中国人如此害怕中共呢？从小的老百姓的人生，一开始从点点滴滴，方方面面都被中共所控制，包括生与死，整个中国大陆就是一个大监狱，老百姓就是生产资料。

我们的勇气从何而来呢，只有跳出这个中共的思想怪圈，孩子远离中共的文化宣传，文化洗脑，培养一个自然的，健康的我们的未来。
