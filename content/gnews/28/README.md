**西班牙2021年5月28日电/西喜社——**

香港周五表示，其台北代表处被停止的原因是,台北对香港内政的“严重”干涉，包括提供援助“暴力”示威者的提议，台湾对此予以驳斥。

自2019年在香港爆发支持民主的抗议活动以来，香港政府支持的政府与民主统治的台湾之间的紧张关系有所加剧，中国对此采取了广泛的国家安全法，以促使许多激进分子离开，其中一些人是台湾人。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/28075648/QQ-2.jpg)
香港周二表示，已终止香港经济贸易及文化处的业务，并补充说，这一决定与台湾最近出现的冠状病毒病例增加无关。

该声明特别指出了台湾去年在台北开设办事处的举动，以帮助实施安全法后可能想离开香港的人们。

他说，该办公室向“暴力示威者和试图破坏香港的繁荣与稳定的人”提供了援助，并将台湾的行动称为“挑衅性的”。

声明说，该市台湾代表处的所有工作人员已返回家园。

台湾中共国政策制定大陆事务委员会在回应中说，中共国已将安全法强加于香港人民，给香港的民主和人权带来了“灾难”。

他补充说，台湾与其他民主国家一样，在“争取民主与自由的斗争”中支持香港人民，并且政府向合法来到台湾的香港人提供人道主义帮助和必要的服务。

在发生了2019香港的反送中运动后，大量追求民主与自由的香港人逃往台湾避难。例如2020年8月，12名港人从香港搭船拟偷渡前往台湾，途中被广东海警截获，之后扣押在深圳盐田。其中部分人士曾因涉及2019年在香港参与“反送中”运动，而被拘捕并起诉。去年12月，其中8人被以“偷越边境罪”判处有期徒刑7个月，今年3月22日刑满出狱。

去年，有超过10,800名香港公民移居台湾，这一数字几乎是2019年的两倍。林郑在6月份通过了严厉的港区国安法之后：北京对民主运动进行了镇压。

台湾在政治和地缘政治上对大陆也是至关重要的，拿下台湾，等于将石油通道与马六甲海峡彻底打通，完全将第二岛链放入掌中，目前深受大陆经济威胁的日本到时就只能被迫臣服。在亚洲内，勾结伊朗，巴基斯坦，暗结以色列，阿联酋，沙特，整个亚洲就在掌握之中了。再者，大陆的国内政党矛盾，经济危机，大规模失业，金融危机，来自国外病毒追责的压力，都可以攻打台湾来得到缓解。
