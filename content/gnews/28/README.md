## 【鹰眼墙国】民办教育促进法实施条例先行，难遏民办学校“营利性”实质

环球网6月16日转载法制日报消息，近期颁布的《中华人民共和国民办教育促进法实施条例》将于2021年9月1日起施行。条例中明确民办教育应当以公益办学为初衷，并明确规定地方政府不得利用国有企业、公办资源举办或参与举报民办学校。

根据中共教育部数据，2020年国内民办学校18.67万所，占全国学校总数的1/3，在校人数5564.45万人，占比近1/5。2019年民办普通小学数量为6228所，占全国普通小学总数15.8万所的3.9%，在校人数944.91万人，占比8.95%；2019年民办初中数量5793所，占比全国初中总数的11.05%，在校生数量687.40万人，占比约为13.99%；民办义务教育学校占总数量21.08万所的比例约为5.7%。义务教育学校仍是由公立学校占大多数。2019年民办高中数量3427所，占比24.54%，在校生359.68万人，占比约14.4%。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/16121158/Screen-Shot-2021-06-16-at-12.07.00-PM.png)
虽然民办义务教育学校的占比较小，但近年来其高价收费等现象却此起彼伏。虽然中共教育部严禁通过笔试、面试等方式提前招生，但基层民办学校“掐尖招生”现象却屡见不鲜。招生过程的“择优性”决定了民办高校的成绩领先，因而民众选择民办学校的热情不减。再者，虽然各地政府出台教育划片等措施，但公立中小学等依旧沿用户口“分配”制度。对于户籍在划片“属外”的家长而言，就近的民办学校只能是他们的选择。

近年来各种挂牌学校兴起，冠名的“伪名校”和地产开发商两两结合，办学之中不但赚取高昂的管理费和学费，更推升了当地房价、带动经济效益，为当地政府带来可观的政绩。名校的出卖名声，房地产开发商的大肆宣传，底层民众在被各种洗脑鼓吹下，用高价为孩子的义务教育买单。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/16121143/Screen-Shot-2021-06-16-at-12.07.03-PM.png)
民办学校的收入主要取决于学生学生人数、住宿费、餐饮费、学费等，民办教师的薪酬水平也来自于学校的收支。由于民办教师和公办教师退休待遇上的差异（民办教师退休待遇由其个人缴纳的职工基本养老保险提供，而公办教师退休则有事业单位退休保障制度以及财政经费支持），民办学校收费标准普遍高于公立学校。即便有非盈利的号召，若教师薪金来源以及学校正常运转的资金不得到解决，民办学校的“盈利性”本质也就无法改变。更何况近年来万科、恒大、碧桂园等房地产巨头进驻教育领域，民办学校已经彻底沦为教育商业化、牟利化的工具。
