## 中共春晚太烂屏蔽“春晚无聊”，表彰会不敢提李文亮
` Himalaya Australia` [轉載自GNews](https://gnews.org/zh-hans/910062/)

作者：必有回响

中共春晚节目有跟不上时代滑稽可笑的《唱支山歌给党听》，更有荒诞无耻编造的展现抗击疫情的小品《阳台》，笔者认为中共用编造的小品来宣传也是有点跟不上时代，扒开底裤还是毛时代的配方，比如毛时代那些宣传片舞台剧样板戏，可以说笔者这样不属于毛时代从小接触西方电影的人完全半点都没有认同感，但是如果是毛时代那时候这样的洗脑可是相当成功的，因为那个时候整个社会都是封闭的，没有电视电脑网络，大家从小能看到的就是那样垃圾一般的歌颂党伟大的节目。现在是信息爆炸的社会这样的洗脑显得非常老土，虽然中共春晚整体布置也用上了信息社会所应该的高科技，但是看到一些让人笑掉大牙的节目也是反映了中共中南坑那群老头确实思想上太落后时代，以至于出现这样的洋不洋土不土的结合。

《阳台》里面的内容让人笑掉大牙，里面也有个所谓的老领导天天嚷着要去中共自己制造病毒的抗疫前线去作志愿者，可实现谁看到有一个局级以上的干部敢跑到现场的？都是那些基层的小人物或者普通老百姓去支援的，大点的领导早就跑到没影了，今年以来的疫情大家都能从流传的视频里看到都是一刀切的封城没人管没人问，领导在哪呢？所以这开头就是纯粹的笑话。

老领导的隔壁邻居一个大妈说是年轻的时候插队和老领导是一起的，还要安慰老领导给他天天唱歌解解闷，你说这出真是笑死我了，谁不知道中共领导那些破事儿，这让我很容易想到老领导年轻的时候估计是小领导，这大妈年轻的时候是不是过去给他当个暖床的啊？他们年轻的时候就是一个勘探队的，后来又一起去了新疆，大庆等地。虽然这个小品纯属胡编，不过这剧情却实在是让人笑掉大牙！

当然我们新中国联绑人能品出里面那些荒诞的味道，不过现实要好得多，正如笔者所说的，中共春晚这种洋不洋土不土的形式就连国内天天被洗脑的中国人都觉得无聊，这不，中共国内连“春晚无聊”的关键字都给屏蔽了，你看中共是多么不自信啊！我想应该是很多人这样吐槽。不过这还不是笔者的发现，这是推上BBC记者，曾任美国广播公司北京局局长9年的@StephenMcDonell的发现。

从微博，贴吧和知乎被屏蔽来看，笔者认为应该是很多年轻人在这上面吐槽中共春晚无聊，因为这几个社交媒体都是年轻人为主的，是非常大社交媒体，类似美国的推特之类的。

另外就是，中共春晚抗疫表彰个人，居然表彰了少将陈薇，但是中共却完全不提中共国内的吹哨人李文亮医生，因为这个病毒就是中共一手制造的，而李文亮医生的出现其实是突然有点扰乱了中共的计划差点揭穿他们的阴谋，所以他们在中共春晚上完全不提李文亮就是做贼心虚。当然不管李文亮是有意还是无意他都做了英雄，可惜他这个英雄被中共害死成了悲情英雄，不过老天保佑！我们爆料革命新中国联绑闫博士不会再成为悲情英雄，她的揭露影响了世界，笔者坚定的认为我们新中国联绑闫博士路德已经创造了历史，中共的报应只是还没有显现，不是不报，时候未到！
