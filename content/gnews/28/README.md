这是德国信义宗牧师马丁·尼莫拉流着羞愧与悔恨的泪水，以德文写成的一篇忏悔诗，叫做《我没有说话》。它真实地记载了1933年到1937年间，纳粹德国党卫军头目希姆莱在希特勒的授意下，带领盖世太保警探在大街上到处逮捕犹太人和成千上万的异议人士及敢于反抗的基督徒，并且迫害不愿妥协的罗马教会成员。很多人都未经法律程序被盖世太保投入集中营，其中大多数遭到拷问、虐待，或是直接处决。他们以「捍卫国家安全」的名义，动辄深夜破门而入闯进犹太人的家庭，或者从孩子身边夺走天主教徒的父亲。有些人遭受酷刑，被迫害至死；有些人从此骨肉分离，阴阳两隔。

当时的德国知识分子与牧师们迫于希特勒的淫威，屈服于纳粹党势力，沉默地坐视纳粹肃清一群又一群的无辜者。导致了德国人民永久地刻上了耻辱与悔恨的心理烙印。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/30100800/D469E0A5-BFAA-4488-8ACA-EE56648A6C0D.jpeg)

无独有偶，在地球的另一端，近日「习特勒」治理下的中共国河北省，全副武装的中共警方也用暴行给人民上演了惊人相似的一幕。

5月20日至21日，中共河北警方对新乡民间天主教「代牧区」采取行动，野蛮地逮捕了几乎所有教会人员，包括被教廷祝圣的63岁主教张维柱。该主教牧养的新乡代牧区有10万信徒。

据罗马亚洲新闻（asianews.it）网站5月22日消息，5月20日下午，至少有100名全副武装的警察（来自沧州、河间和沙河桥）包围了沙河桥一处被用作教区修道院的建筑。这是一名天主教徒拥有的小工厂。警察不由分说冲进大楼，强行铐走了四名神父、教授、三名从事牧民工作的神父以及与他们一起在工厂上课的10名修道生。

据此报道，5月21日，张维柱主教、七名神父和十名修生被警方押走。现年63岁的张维柱于1991年被祝圣为主教，曾数度被中共逮捕入狱。这次「盖世太保模式」的突然袭击之后，中共警察还强行没收了神父们和修道生的所有个人财物。信仰传播者在中共国一旦入狱，多数会遭受非人道的折磨，不脱一层皮通常是出不来的。因此逮捕神职人员的野蛮行径不由得令人发指。

该消息称，此次行动是中共针对民间天主教的迫害。由于警方部署规模庞大，行动指挥来自高层，所以人们认为，这次中共的暴行应该是提前经过精心策划及部署的。而且，当局按照一贯的、打击异议人士的「传统作风」，下令警察沿街沿巷、挨家挨户地搜寻逃亡的修士，授意这些中共的「盖世太保」们要「除草务尽，不落一人」。一旦发现十字架、雕像、圣人画像、教宗照片等天主教信仰标记物件，立即没收和销毁，并对屋主或拥有者处以罚款追责和行政拘留。

今天，人所共知的是：中共专制集权政府惧怕思想与信仰的传播，特别是在「大觉醒」力量的促进下，梵蒂冈教廷开始抵制中共的渗透，不再向中南海低头和妥协。而中共政府也不再承认梵蒂冈天主教，不再承认教廷认可的河北新乡代牧区。因此，那里的神父、修道院和修道生的活动都被当局划定为「非法」。按照中共政府的《宗教事务条例》，这个被用作修道院的工厂就此难逃厄运，他们的负责人就面临着被逮捕、被逼迫、被囚禁的命运。

中共从无任何诚信可言，2018年中共与梵蒂冈教廷签署的《临时协议》已经成了废纸一文。多数信徒认为，从主教被囚禁，神父和修道生被逐离，以及神父们被处以巨额罚款的情况来看，这个中梵《协议》已被出卖。

从纳粹德国到如今的中共国，从莱茵河畔到河北新乡，历史在不断上演着轮回，纳粹「盖世太保」清洗和迫害天主教与基督徒的旧戏还不断地在今天上演。可是人们已经不再沉默，人世间的各样的暴行已经不能再掩盖于地下，世界已经变了，沧海已成桑田。

也许大多数的人们还没有意识到，有件事情已经截然不同：我们有了爆料革命。
