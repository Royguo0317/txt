## 【香港要聞】警修例縮窄採訪空間 記協質疑警方如何判斷記者影響行動效率
`2020-09-26 03:54 GM42` [轉載自GNews](https://gnews.org/zh-hant/383507/)

製作單位：戰友之家-粵語組
編撰：文燕
覆核：文錦

【**香港電台**】9月24日報導，香港記者協會等8個傳媒工會召開聯合記者會，抗議警方修改《警察通例》中「傳媒代表」的定義。
     記協主席楊健興批評警方過去一段時間濫用封鎖區，劃定相當大的封鎖範圍，目的就是要限制採訪。對於警方在警察通例中配合傳媒採訪的安排上，加入“在不影響行動效率”作為前提，他質疑警方如何判斷記者影響行動效率，擔心有關安排會將阻撓記者，甚至要求記者離開的做法合理化。
     警方表示修改警察通例目的是協助傳媒採訪，楊健興質疑這是不符事實，希望同行看清楚政府及警方的真正意圖，共同反對今次修改通例。
     攝影記者協會主席陳奕釗說，警方不承認攝記協的證件，最受影響是一批自由工作者，因為他們沒有特定的記者證，身上會持有採訪當日代表機構發出的工作證明檔，警方截查時可能需要較多時間核實檔，大大增加採訪的不便。他強調，申請攝記協記者證的條件嚴格，學生記者不會獲發相關證件。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/26034703/image001-1.jpg)圖片：香港電台
【Now新聞台】9月24日報導，中大新聞與傳播學院院長李立峯認為，警方修訂《警察通例》、“傳媒代表“的定義，縮窄了傳媒採訪空間。
      李立峯指出，“原則上你給一個官方機構，用他們的準則其實不一定是對的，不一定適合現在的實際情況。第二，網路世界轉變迅速，新興網媒的興起速度可以很快，如果用一個很死板的系統去規管誰是或誰不是傳媒，會很容易追不上轉變。”
     “很多時候的爭議不是討論記者是真還是假，反而是，的確會有情況出現前線的新聞工作人員可能是學生媒體，可能是公民記者，是否他們的行動給予人認為不夠專業。”

**戰友點評：【文燕】**
      警修例縮窄採訪空間，其實是向市民確認了港府要把大陸那一套監控新聞的做法移植到香港。記協主席楊健興批評警方過去一段時間濫用封鎖區，劃定相當大的封鎖範圍，表面上說生怕記者影響（警方）行動效率，實際目的就是要限制採訪，進而限制香港市民的知情權，不想市民知道真相，不想醜行暴露而已。現在只是講不出道理，也拿不出依據，就把這些條規用法律手段，粗暴地限制記者。
      警察有先進的武備，以執行工作為名，可以合法動用武力，任何人只要反抗，就是襲警，權力十分不對等。例如最近12歲女童與哥哥路過旺角買顏料時，突遭男防暴警迎面撞飛，女孩受驚逃跑，最後卻遭多名警員騎身和膝壓制服。
      又早前在旺角一名路過孕婦被被防暴拉扯及箍頸後倒地，再被噴胡椒噴霧。防暴警察在執行任務時情緒失控，對手無縛雞之力的市民使用不必要的武力，他們這些行為必須要受到社會的監察，必須向社會交代，而傳媒扮演的傳播新聞真相和監督角色便十分重要。
      實際上，全世界都有越來越多以獨立身份作報導的自由記者及個體戶式的記者，這些自由媒體甚至學生媒體對事態發展、社會事件、及對互聯網上的反應更敏銳的網媒。
     這些自媒體補充和豐富了記者的報導，正如李立峯所說的，“網路世界轉變迅速，新興網媒的興起速度可以很快，如果用一個很死板的系統去規管誰是或誰不是傳媒，會很容易追不上轉變。“
