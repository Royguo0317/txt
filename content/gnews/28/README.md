中共國是造成10萬多美國人死亡的中共冠狀病毒的製造者，也是很多有關川普總統和美國的負面新聞的幕後主使。

中國共產黨創建了一個名為“統一戰線”的實體，旨在確保美國媒體對中國拍馬屁，制定對中共國有利的政策。統一戰線還支持有利於中共國的民主黨政策，並推動這些政策的推廣。我們已經多次報導過中共國經濟是如何停滯不前甚至倒退的。有許多衡量標準支持這一立場，但你從未在任何自由世界的媒體上聽到過這一點。

我們在2019年8月發布了川普前首席策略師史蒂夫·班農(Steve Bannon)關於中共國的演講。他討論了中共國是如何與美國展開經濟戰爭的。他說，在川普總統到來之前，美國一直在輸。接著他又說:好吧，遊戲是這樣的，現在我們匯聚在一點上，他們能理解。我們可以把它拿下來。整個建築都是建在沙子上的…如果他們(中共國)讓貨幣貶值，就會有更多資金外流。他們有3萬億美元的外匯儲備，相信我，在一秒鐘內，這些錢會在一秒鐘內湧出來。這就是他們自己的人民對經濟的看法。我們任由這些傢伙擺佈。我們允許這些傢伙佔領南海…這場貿易戰將以勝利告終，你將看到的是整個供應鏈轉出中共國……但是，美國MSM從未對此進行過報導。這在一定程度上是因為中共國對美國的MSM有重大影響。

前中央情報局(CIA)行動官查爾斯•“薩姆”•法迪斯(Charles“Sam”Faddis)發表了一篇文章，概述了當今中共國試圖統治美國媒體的努力。以下是他發表在andmagazine.com上的文章片段:”幾週後，我們陷入了一場相當於全國范圍內的暴動。紀念碑被損毀，雕像被推倒，我們大城市的商業中心被摧毀。然而，在整個過程中，我們大多數的大眾媒體都把規模空前的暴力和無法無天描述為“和平抗議”，破壞的範圍被掩蓋了。在美國橫行的無政府主義者和馬克思主義者的要求似乎被系統地錯誤地簡單地描述為對正義和改革的要求。”

“這是怎麼回事？在一個植根於法治和尊重民主進程的國家中，我們怎麼會走到這樣一個地步:暴力和違法行為不僅得到寬恕，而且得到支持?為什麼憲法第一修正案的護衛者們似乎與致力於破壞現有社會秩序的團體結盟?毫無疑問，部分原因在於，我們的主流媒體對唐納德·川普(Donald Trump)的仇恨是盲目的，它會支持任何似乎反對他的東西。然而，還有其他因素在起作用，其中一些正在發生的事情，實際上可能是由於一個明確的反美議程，以及某個外國角色的願望，不僅要破壞這個偉大的國家，而且要推廣它自己的特殊意識形態。”

中國共產黨(CCP)使用宣傳和影響行動作為一種投射其力量和削弱其敵人的手段。美國是這類行動的主要目標之一。這些操作不是在某種特殊的基礎上運行的。他們是由中國共產黨的統戰部在全國范圍內協調和指導的。

中國共產黨的統一戰線系統動員黨的“朋友”打擊黨的敵人。這一制度存在了很長時間，但近年來在習近平的領導下，它得到了極大的激勵和擴展。它在外國政黨、僑民社區、大學和公司內部運作，所有這些都以促進黨的利益為目標。統一戰線製度的明確目標包括破壞社會凝聚力，加劇種族緊張，影響政治。然後，Faddis確定了MSM媒體和CCP之間的諸多聯繫:大多數主要的美國媒體都是由母公司所有的。這些公司在中共國都有重要的商業利益。他們依賴於中共國的善意和中共國政府的批准來維護這些利益。

紐約時報
墨西哥億萬富翁卡洛斯•斯利姆(Carlos Slim)擁有該公司17.4%的股份，並擁有董事會三分之一的投票權。斯利姆與中國共產黨有聯繫的公司有大量商業往來。在這些關係中，華為目前正因間諜活動和損害美國國家安全的行為而受到美國的製裁。
華盛頓郵報
2013年，亞馬遜CEO、億萬富翁傑夫·貝佐斯以2.5億美元收購了《華盛頓郵報》。貝佐斯與中共國市場有著直接而巨大的聯繫，中共國市場是由中國共產黨控制的。亞馬遜最受歡迎的產品，如Echo和Kindle，幾乎都是在中共國工廠生產的。當你訂閱《華盛頓郵報》時，會附送一份名為《中國觀察》的廣告增刊。本增刊由中共國官方媒體《中國日報》提供。
CNN
該公司由華納傳媒(WarnerMedia)所有並運營，後者與中國共產黨有著重要的財務和組織聯繫。華納傳媒(WarnerMedia)投資了中共國媒體投資基金(CMC)，這是一家由中國共產黨監管的媒體公司。 CNN通過其總裁傑夫·朱克(Jeff Zuckers)參與了NBA在中共國的廣播，並努力擴大NBA在中共國的節目。
MSNBC和NBC
MSNBC和NBC新聞都由NBC環球運營，這家公司與中國共產黨有著巨大的經濟聯繫。 NBC是與中共國官方媒體合作的。新華社被美國國務院認定為一個“外交使團”，這意味著它完全是中國共產黨的一個分支，並不是一個獨立的新聞機構。 NBC Universal旗下的東方夢工廠(Oriental DreamWorks)由一家中共國投資集團全資擁有。
美國廣播公司
華特迪士尼和ESPN都屬於美國廣播公司。兩家公司都對中共國經濟進行了大量投資。 2019年，當NBA拒絕譴責中共國在香港的行為時，ESPN的高級新聞總監指示ESPN的工作人員避免討論中共國政治或與香港有關的事情。

彭博社
前紐約市市長邁克爾•布隆伯格(Michael Bloomberg)和他的公司彭博資訊(Bloomberg LP)在中共國都有大量投資。彭博在中共國市場向其網站銷售終端，並通過向中共國債券市場輸送數十億美元美國投資者資金，為中共國企業提供融資。彭博資訊支持364家中共國企業。其中159家屬於中國共產黨。 2014年，彭博(Bloomberg)叫停了一項針對中共精英的財務調查。當這篇報導準備出版時，主編馬修·溫克勒粉碎了它，他說:“這肯定會，你知道，邀請共產黨，你知道，徹底關閉我們，把我們踢出這個國家。”所以，我認為這個故事是不合理的。 ”

簡而言之，除了那些更保守或更自由的媒體外，基本上所有主要的美國媒體都受制於中共國強大的影響力。他們所報導的正是中國共產黨希望他們報導的。他們說的就是共產黨想讓他們說的。
