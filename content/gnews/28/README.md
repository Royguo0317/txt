## 中新聞：再現供銷社——廣東省供銷集團有限公司揭牌
`2020-09-14 09:55 GM36` [轉載自GNews](https://gnews.org/zh-hant/354888/)

編撰：JAY、zhong、文非；審核：文肯尼
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14095426/914_10.jpg)
![]()

廣東省供銷集團有限公司9月11日在廣州舉行揭牌儀式，並宣布組建廣東省供銷集團，堅持為農服務宗旨和合作經濟屬性、深化綜合改革的重要舉措。

廣東省供銷集團宣稱，將堅持實踐為農服務、做強主業，優化層級、強化管理，加強合作、協同發展，黨建引領、完善治理的原則為建設廣東供銷公共型農業社會化服務體係發揮支撐作用。

**編者評論：**

CCP此時加強供銷社模式的推廣與建設，其主要目的是應對日益惡化的全國性糧食短缺並迎合內循環經濟，作準備。廣東省作為中國的產糧大省，此時成立供銷集團更是此地無銀三百兩，預示著中國即將很快步入糧食危機。
