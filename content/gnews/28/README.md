每次在推特或海外媒体上看到中国老百姓因言获罪或维权行动中被暴力残酷对待的新闻总是怒火中烧，恨不得明天就能出现超自然力量把中南坑给炸平了，然而平息怒火后却是无奈和无能为力后的伤心欲绝，这种心理起伏状态在过去十几年里已太多次了。当今天看到大纪元的一篇报道《直言中共病毒  北京退休教授被控刑两年半》，笔者愤怒至极，实在见不得邪恶的中共当局对曾经的高等教育工作者又是老者实施的侵犯人权的行为，一个正常的社会应该是敬重知识分子的，尤其是有良知的学者。CCP的打手们：你们积点德吧，见好就收，免得将来被依法清算的时候画面不太好看！

现今69岁的陈兆志是北京科技大学的退休教授，而且还是发明家，去年在网上发表言论“武汉病毒不是中国病毒，是中共病毒”并批评中共当局隐瞒疫情以及处置疫情不当，随后被当局打击报复，陈兆志被北京市海淀区东升派出所以所谓“编造、故意传播虚假信息”罪名拘留，后被逮捕，以“寻衅滋事”罪羁押了一年。据了解，陈兆志患有脑退化症、脑梗、高血压等多种疾病，健康状况很不好，为此家人曾多次尝试申请取保候审，但都遭到警方拒绝。陈教授患有老年痴呆，这一点就不该让他接受任何司法审判，然而检察院竟然建议量刑两年六个月！这完全就是政治迫害！完全蔑视了宪法赋予公民的自由言论权，尽是展现了中共国的司法缺乏人性、以黑治国的一面！

中共当局对宪法的践踏，对人民的迫害在任何时代都一直在进行，可能在不同城市地区，形式和方法尺度略不同，但始终从未停息过。而当局每次对群众的迫害打压，都应该是民众开智的“好教材”，都是活生生例子啊，可惜的是，在长期洗脑和压迫下，人民已经逐渐失去了反思能力，似乎不愿意去接受各种残酷的现实，更多的人是选择逃避，也就是“不敢开智”。

自从2017年1月郭文贵先生横空出世在社交媒体开始对中共高层的贪腐进行爆料以来已有4年时间，七哥根据当时实际国内国际政治环境，深入浅出地、战略性地进行着揭露，在这4年时间里，在七哥的不间断爆料的“协助下”至少有好几千万甚至上亿墙国民众被唤醒，当然是在经历了长时间的痛苦的自我反洗脑后渐渐清醒的，笔者相信大多数战友会有相同的经历，即当我们对郭先生深信不疑、坚信已初步看清CCP本质后会满怀希望甚至略带自信地去尝试给身边的请朋好友传递一些事情的真相，可每当满怀期待能得到对方共鸣之时，换来的却总是出乎意料的为CCP辩驳开拓，这种信息严重不对称导致双方各执一词，立场完全不同，最终落个不欢而散。笔者真希望周围能有更多的人通过中共对百姓的种种侵犯人权的恶行来进行反思，从而看清中共的邪恶本质！
