## 中印邊界衝突迅速升級，中共將會如何收場？
`2020-06-16 21:58 GM30` [轉載自GNews](https://gnews.org/zh-hant/236178/)

**作者： 麓山之蜂**

2020年5月9日，中共國與印度兩國部隊于錫金北部地區的納庫拉發生對峙，後演變爲肢體衝突，在衝突中雙方共有11人受傷。隨後，中共國與印度都宣稱是對方侵入了自己本國領土，事後雙方都向該地增添士兵。2020年 6月 2日，BBC中文網，中印對峙進一步升級 兩國大幅增兵加勒萬河谷。而中共國與印度的此次邊界衝突事件迅速升級，2020年6月16日，中共國與印度雙方在加勒萬河谷爆發軍事衝突：印度有三人死亡，中共國則是5死11傷。

![](https://gnews.org/wp-content/uploads/2020/06/1-242.jpg)

中共國與印度邊界衝突史：

1951年，中共國通過軍事行動和談判，入駐西藏。印度駐華使館向中共政府發出照會中說：「中共軍隊之侵入西藏不得不認爲是可悲的。」1951年2月2日，印度軍隊乘中共國參與韓戰之際，派兵占領了西藏噶廈政府管轄的達旺地區。

1959年藏區騷亂後，十四世達賴喇嘛流亡印度，成立西藏流亡政府，主張西藏獨立。1960年中共國務院總理周恩來訪問印度，同尼赫魯總理會談，提出談判解決邊境問題，未得印度方面的接受。1960年後，中蘇開始交惡，而中美關系因韓戰的原因也處于僵局。而印度與正在進行冷戰的蘇聯和美國關系都不錯。

1962年中共國與印度發生邊境戰爭，戰爭在中印邊境東段和西段進行，從1962年10月20日開始，至11月21日基本結束，曆時1個月，經曆了兩個作戰階段：第一階段自10月20日至28日，第二階段自11月16日至11月21日。1962年11月21日，中工國政府聲明主動停火、主動後撤和設立民政檢查站等三項措施：自11月22日北京時間零時起中共軍隊單方面全線停火；開始全線後撤至1959年11月7日中印雙方實際控制線北20公裏之外，脫離雙方軍事接觸；「爲了保證中印邊界地區人民的正常往來，防止破壞分子的活動和維護邊境的秩序，中國將在實際控制線本側的若幹地點設立檢查站，在每一個檢查站配備一定數量的民警」；另外還單方面將繳獲的武器、軍車和軍用物資交還印方。

1987年中國與印度曆史上發生的第三次邊境衝突。該事件雖未造成直接的軍事衝突，但是兩國之間緊張的形勢已經使得中印再次走到了開戰的邊緣。由于美國的調解，隨後和解處理，但是印度卻蠶食了不少領土。最終在1993年，兩國簽訂了一項「以確保實際控制線兩側的和平」的協議，這場邊境危機由此告終。

2017年6月18日，中共國與印度的邊防部隊在洞朗地區進行的軍事對峙。2017年8月28日，中印雙方結束對峙，但沒有對外說明各自具體接受了對方的什麽條件。

而2020年6月16日發生的軍事衝突傷亡事件則是自1962年10月20日至11月21日，以來的再次人員傷亡衝突事件。

而中共在這次軍事衝突後會如何抉擇呢？如果選擇與印度打一場局部戰爭，中共小摩擦叫的響，中共敢嗎？民心、黨心還會存在嗎？一個對外只懂得阿谀奉承，對自己的百姓確實拳腳相向的政黨，還會有人跟隨嗎？
