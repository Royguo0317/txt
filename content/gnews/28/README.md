**1.****美国输油管道遭黑客攻击**

[美国最大燃油运输管道运营商日前受到网络黑客攻击，被迫关闭整个供应网络，引起恐慌。发起攻击的黑客团队声称他们无意制造混乱，纯粹只是为了钱。根据 FBI 的报告，黑客团队名叫”黑暗面 （DarkSide）“，在Colonial Pipeline遭到勒索软件网络攻击后，美国于周日发布了紧急立法。这条输油管道每天输送250万桶原油，相当于东海岸供应的柴油、汽油和航空燃料的45%。网络攻击发生后，该运营商于周五下线,恢复服务的工作仍在继续。受管道中断影响，最近一周，美国加油站汽油价格跃升0.06美元，可能很快就会升至2014年以来最高。](https://www.bbc.com/news/business-57050690)
