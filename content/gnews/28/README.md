## 日本决定将第三次紧急状态延长至5月31日，L452R突变对日本人更具威胁性
` JPHIMALAYA` [轉載自GNews](https://gnews.org/zh-hans/1183363/)

【日本东京方舟农场】 作者：小李在大阪　校对：文小律

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/07044612/91A6FA4F-9EA8-4C7D-8FE3-191102AF5EFF.jpeg)

由于变异中共病毒肆虐，日本政府决定将4月25日至5月11日的第三次紧急状态延长至5月31日，并且在东京、京都、大阪、兵库的基础上，将爱知及福冈也增加为紧急状态的对象地区。

目前，日本大阪地区疫情最为严重，医疗几近崩溃，用于收治重症患者的病床已全部满床，很多重症患者根本无法接受相应的治疗。在家中等待床位的轻症及中症患者因病情突然加剧而去世的情况也频频发生。

另一方面，变异中共病毒的感染范围急剧扩大，东京5月5日的新增感染者中有62%的人感染了​N501Y变异毒株，而兵库的变异毒株感染者更是高达90.9%。

目前尚不清楚中共是否针对特定族群设计并制造了特定的变异毒株，但研究表明特定的变异毒株对日本人更具威胁性。东京大学医科学研究所佐藤佳（副教授）的研究团队针对细胞性免疫进行了调查，发现L452R突变能够避开“HLA-A24”白血球，而60%的日本人都具有这种白血球，也就是说对日本人而言，携带L452R突变的毒株传染性更强且更易造成症状加重。另外，在印度发现的变异中共病毒同时携带E484Q和L452R突变，特定突变对某个族群的特异性威胁或许是中共病毒在印度造成大量患者死亡的原因之一。
