## 【军事快讯】日本首相菅义伟明确要求俄罗斯归还北方四岛
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/887249/)

作者：Wilson

![]()![](https://gnews.org/wp-content/uploads/2021/02/MOS1192-【军事快讯】日本首相菅义伟明确要求俄罗斯归还北方四岛.jpg)图片来源：美联社

[共同社](https://china.kyodonews.net/news/2021/02/4b4f89cecc39.html)2月7日报道，日本首相菅义伟刚刚在国会答辩中强调对北方四岛的主权，明确展现要求俄罗斯归还的立场。引起俄方的强烈反响，俄方认为，日方从安倍时期要求收回齿舞群岛和色丹岛的“两岛解决”方案，升级为收回全部北方四岛，立场和态度已经趋于强硬。

早在1月20日，菅义伟在众院全体会议上就表示：“北方领土是我国拥有主权的岛屿，和平条约谈判的对象是四岛的归属问题，这是我国一贯的立场。”他强调将基于写明以包括国后岛和择捉岛在内的四岛全部归属为谈判对象的1993年《东京宣言》开展谈判。

但俄罗斯要求日本承认，“作为第二次世界大战的结果，南千岛群岛（北方四岛）处于俄罗斯主权之下”。莫斯科国际关系学院教授斯特列利佐夫分析，日本从安倍政府追求的两岛解决方案提升到了要求归还全部四岛的原有立场。日俄双方立场顽固，谈判预计难有进展。”

在俄罗斯宣布已修宪中止争议领土谈判后仅几天，菅义伟首相就在国会发表对俄罗斯比安倍更加强硬的立场。目前，日俄外交谈判渠道已因俄罗斯修宪而堵塞，接下来双方的较量或将暴力化。之前笔者曾就此预言，俄罗斯的修宪或已埋下战争隐患。日本和俄罗斯在北方四岛归属问题上似乎难免兵戎相见，这是由大和民族和俄罗斯民族的民族性决定的，是宿命。
