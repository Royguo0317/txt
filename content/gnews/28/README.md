最近，墙内出现贵州茅台酒曲加工是由成年女性光脚踩压的视频，且工作环境恶劣、卫生条件堪忧，引起社会广泛热议。

![]()![](https://gnews.org/wp-content/uploads/2021/03/image-23.png)视频截图

大名鼎鼎的国酒茅台是经过大妈的光脚踩成的啊，中共的权贵知道吗？世卫“皮特打杂客”知道吗？

墙内媒体网易站出来洗地了，称“贵州茅台总工入选中（共）国工程院院士候选人之后，有关茅台这个‘酱香科技’的段子也越来越多。这不，茅台少女踩曲这一‘核心科技’又在网上热传。实际上，少女踩曲是茅台酒多年来引以为豪的独特工艺。在制作酒曲的过程中，有一个在外人看来十分神秘的环节——由年轻姑娘来踩曲。”

通过这段报道我们了解到，敢情少女踩曲是秘密科技啊。先放下科技含量一说，所谓的少女是啥梗？视频截图中的是少女吗？

该文章对少女踩曲解释到：这一古老的踩曲工序主要由女工完成的原因是，男人力气大，踩曲踩得实——而“曲块中的间隙少了，反倒不利于微生物发酵”。女工更注重细节，可以精细操作，而且女性的脚相对出汗较好，不会对酒曲造成影响。少女的纤纤玉足分泌物少，霉菌少，流汗少，更能够确保酒曲的酸碱度不会产生比较大的变化，在多种酿酒微生物平衡快速生长繁殖的同时，还能保证微生物种类的多样性。

这段话给出的理由是：男性力气大，女性流汗少。可我在意的是，这酒里是不是有脚汗、脚气，跟我们扯什么力气大，脚汗少啊，这明明讲的是脚汗是酿造茅台的必要配方啊。这下想通了，为什么配方是秘密的。它能不秘密嘛！如果公开了，就意味着消费者花高价，一桌子人集体舔大妈的脚丫子嘛！还酱香型白酒，干脆改成脚汗味白酒更贴切。怪不得中共将它定为国酒，真的是那帮孙子好这一口啊！况且这酒可不便宜，大妈脚汗酒的价格在人民币1000以上，市场上还紧俏得很，供不应求，假酒泛滥。

该文章洗地到这里，发现不能自圆其说，于是开始耍无赖了。文中称，“茅台历来都是这么酿造的，这是工艺的独特性之一。没有这个踩曲的环节，就没有茅台酒，没有少女踩曲，就没有酱香酒的优雅细腻。”

文章还称，踩曲女工得是“211”或“双一流”高校大学生！笔者想说，踩酒曲跟大学生有个毛关系！还名牌大学！教育资源这样分配合理吗？但想想山东大学的女大生都可以被校方用作三陪女，让女学生去踩个曲算是特别优待了吧？何况从视频截图来看，压根不是中共口中的女大学生，估计这样的“肥差”还轮不到穷人家的闺女吧。

该文章的配图中，远处有架着摄像机摆拍的所谓记者，车间的卫生环境令人咂舌：穿鞋的人随意在满是酒曲的地面走动、站立。这是喂牲口的环境吗？中共权贵也喝从这里踩出来的酒啊。想想文贵先生说过，西方的猪和狗都比中共过得幸福，看看这景象，简直太符合了。
