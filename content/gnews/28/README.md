11月13日，美国川普总统在Twitter上宣布，在宾夕法尼亚州，他的支持者不被允许查看费城和匹兹堡的70万张选票，这意味着他在这个地区的胜利。

川普总统说，”无法查看70万张选票，所以根据我们伟大的宪法，我们赢得了宾夕法尼亚州。”

早些时候，宾夕法尼亚州的联邦上诉法院驳回了共和党的要求，共和党要求在投票日结束后所邮寄的9,300张选票无效。

据福克斯新闻报道，拜登在宾夕法尼亚州获得49.9％的选票，而川普总统获得49％的选票。

与此同时，另一个州法院支持现任总统，并批准了川普总统提出的诉讼，要求不考虑将邮寄选票计算在内，如果这些选票无法确认他们的身份的话。

美国2020总统大选于11月3日举行。四天后，一些美国主流媒体主观向外界宣布所谓的乔∙拜登在选举中获胜，因为他获得了必要的270张选举人票。

然而，川普总统强调，这样的结论还为时过早。官方还没有公布最终的选举结果，要等到12月才正式公布。
