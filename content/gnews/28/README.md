## 被歪曲了的忠孝二字——对中国文化的深思

儒家文化是中国文化中的重要部分之一，对于人的道德标准通常简化为“忠孝信悌礼义廉耻”，是儒家倡导的做人之根本，出自于《论语·学而》，简称“人生八德”。

其中概念比较明确的是“信悌廉耻”，信是指信用，诚信，言而有信，言出必行，表里如一；悌指的是在弟兄姐妹之间或朋友之间的友爱帮助，有兄恭弟友之说；而廉耻指的是廉洁的情操和羞耻之心。

至于礼义，礼为礼节或礼仪，表示得体，而义可以表示仁义、道义、正义和情义等，解释起来较为复杂，这里不作详述。

本文主要分析忠孝两字。经过历史变迁，忠孝两字已被统治者提升到首要地位，那么什么是忠？忠什么？什么是孝？

这里孝相对比较好解释，就是孝敬父母长辈，所谓身体发肤受之父母，父母辛苦养育儿女，孝敬父母是儿女自然产生的感情，无可厚非。但长期以来，人们已把孝演化为孝顺，所谓只有犯错的孩子，而天下无不是的父母，这样的孝顺概念就成了一种滥用，我们在二十四孝典故中，可以看到一些不可理喻的故事，什么“埋儿奉母”之类的人间惨剧，是犯罪，却被美化成行孝的美德，这完全是统治者自比父母，愚弄百姓的邪恶教条。圣经中父母和子女的关系讲得很清楚，《以弗所书》“6：1-4 你们作儿女的，要在主里听从父母，这是理所当然的。 ‘孝敬父母，使你得福，在世长寿。’是第一条带应许的诫命。你们作父亲的，不要惹儿女的气，只要照主的教训和警戒养育他们。”  所以说，无论是父母，还是孩子，都应该符合天道。

接下来谈谈，什么是“忠”？  “忠”这个字在《论语》中出现过17次，可见孔子十分重视“忠”这个道德规范，“忠”的基本意思是尽心竭力、公而无私之谓。《说文》：“忠，敬也。尽心曰忠”。《广韵》：“忠，无私也”。朱熹说：“尽己之谓忠”。杨倞《荀子·礼论注》：“忠，诚也”。“诚”、“实”同义，即“忠”是诚实无欺的意思。这里的“忠”有“敬”“尽心”“诚”的含义，有“忠于心”的含义，所谓“心”，就是我们的良心，即忠于我们的良心。良心是提醒人行善而不作恶的本能，当人们做了不道德的事时良心会谴责。但千百年来，这个“忠”字，已经被邪恶的人歪曲成了“忠君爱国”，似乎不忠君就是不爱国，就是不忠不孝，把“忠于心”替换成了“忠于人”，由此延伸，才会有“忠于君王”，“忠于毛”和“忠于党”，殊不知，一旦把“忠于心”变成“忠于人”，人类就进入了万劫不复的境地，因为那个“人”极有可能就是魔鬼，我们怎么能忠于魔鬼呢？事实也早已证明，毛泽东也罢，共产党也罢，都是魔鬼。我们再也不能让魔鬼扭曲了“忠孝”二字。

所以说，我们对忠孝的定义一定要很清楚，不能让魔鬼混淆了概念。
