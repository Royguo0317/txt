在最初新冠疫情爆发的时候，文贵先生、路德和英雄科学家闫丽梦博士都曾经说过新冠病毒就是中共军方研制的，是发动“生物战”的武器。时至今日，世界终于发现中共国拥有一个广泛的研究网络来支持生物战，它早已存在于中国共产党的“十三五”规划中，并详细描述了如何将军事研究中心和民用研究中心融合在一起，研究生化武器。有些科学家就是为实现中共的“生物战”计划，专门从事生物武器研究的军方科研人员。有证据表明，中共国军方在2019年11月就完全知道新冠病毒的存在，来自军方研究人畜共患病的病毒学家，金宁一演讲的幻灯片揭示了新型冠状病毒候选疫苗研发项目进展的时间轴：从2019年12月1日到2019年12月10日，在10天时间内完成了阳性病例采集和病毒序列分析；从2020年1月27日到2020年4月20日，在短短3个月的时间里，完成了从基因合成和疫苗筛选到免疫构建，再到动物免疫实验的过程。显然中共军方更早地知道了新冠病毒的详细情况，而紧急警报是在几个月后才出现，这几乎可以作为中共对世界发动“生物战”的实锤证据。
