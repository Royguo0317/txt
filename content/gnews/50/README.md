在一天之内，疫苗注射超过460万的剂量。在单日内完成超过460万笔的用量，创下了历史最高的记录。

一天之内打疫苗的剂量，打破了上周六的记录，超过500,000。今天，有超过四分之一的成人已经接种疫苗。

例如，在密歇根州，现在CCP病毒的传播速度，每月传播速度比以前更快。

密歇根州政府制定了严格的地方政策，比如，戴口罩，容量限制，待在家里办公。

我们鼓励他们考虑向州提供疫苗，考虑向密歇根州接种疫苗。如果我们不带来免疫力，病毒将继续传播，病毒还会在美国和密歇根州突变。

美国和密歇根州将不会对疫苗负责，但白宫正在努力制定计划， 正在考虑其分配疫苗的计划。

根据人口分配疫苗的数量，至少有50％的美国成年人，至少接受了一次剂量。

来自推特的消息，一个名叫贝克西，她注射第二针疫苗之后，仅一周，她就离世死了。这个消息，让人非常的难受。

如果，她没有注射假的疫苗，没有感染病毒，那么，她现在应该还是活着的。

我们想知道导致密歇根州CCP病毒传播的原因是什么？

一些人认为，似乎来自许多年轻人。但是，不仅在密歇根州，在佛罗里达州也一样。在英国，病毒传播性更强，病毒还在变种，并且正在蔓延。

冠状病毒来自哪里？怎么控制病毒的传播呢？你又能做些什么呢？

不要打疫苗，因为，没有人会对你负责。
