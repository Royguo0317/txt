这些天，德国基督教联盟党党内发生了很多事情。 口罩腐败事件发生后，此党必须证明自己的国会议员们的作为是透明和道德的。具有讽刺意味的是，党的重要成员，卫生部长詹斯·斯潘（Jens Spahn）的工作得到了一些指责。。

在柏林郊区的一座贵族庄园中，有一栋1920年代的灰色建筑师的别墅，在它前面有两个瓦砾容器和一个建筑围栏，正在整修中。根据临时在房屋广告网 Immobilienscout24上刊登的，这所房子有7个房间，居住空间为285平方米。

自去年十月以来，它已经属于联邦卫生部长詹斯·斯潘（Jens Spahn）。他和他的丈夫丹尼尔·冯克（Daniel Funke）以略高于400万欧元的价格买下了别墅。詹斯·斯潘必须忍受严厉的批评，这也充满了嫉妒和怨恨。

这个房子还给德国基督教联盟党政客带来了实质的的批评。
