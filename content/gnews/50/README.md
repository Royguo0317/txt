郭先生2021年2月23日发到盖特的视频中说：”江家这次去新加坡，两个结果。说明江家这次斗争，江家占了劣势，没把习干掉。习也做出承诺，说明习一定是给江家或者讲述了，像当年似的，你们家出去发展弄钱去吧，别在里边搞政治了。离我远点。去哪儿？去新加坡了。

江家败走新加坡，习近平在国内就会走向集权巅峰，想做什么畅通无阻。按照习近平的性格，赌徒心态，想做千古一帝，未来很可能主动引发冲突。继缅甸之后台湾很可能是中共下手的目标。台湾在2月19日更换国安高层人员时，中共派出8架军机扰台。以及在2月20日中共在南海军演时派出11架军机扰台。

中共国在2021年面对着前所未有的国内压力，习近平会不会对台湾下手，谁也说不准。但最好别有不会发生的乐观想法。但在习的认知里，中共才是世界最强的，就像慈喜对当时对世界局势的认知一样。

拜登政府准备与日本韩国台湾澳洲合作，摆脱芯片供应链对中共国的依赖。可见台湾在芯片产业链中的地位。一旦台湾芯片停止供应世界，所有使用台湾产芯片的公司都将大面积停工，产生更多的失业，无产品可卖导致财务报表急剧恶化，纳斯达克崩盘都有可能。如果被中共掌控了台湾的产业链，让美国弯腰都有可能。如此巨大的利益与千古一帝的名声面前，赌徒心态的习近平极有可能对台湾出手。

台湾处在中美冲突的核心位置，在面对国家安全的问题上不能以可能性来做出决策，因为一旦发生是无法挽回的。必须做好必然发生的应对措施，提前准备。投降的结果只有一个，台湾亲共的势力灭掉台湾反共的势力。
