**中共新反腐运动直指军内权利大扫荡**

习近平依然没有露面，墙内利用国家机器全网开启“反腐运动“，覆盖至边远地区。目标直指军队权利，从各媒体上看到强力的反腐新闻，显示出新一轮反腐败运动在内部被掀起，好似当年的文革再现，却比当年更来势凶猛和血雨腥风。这次由习亲自牵头的全党运动，是大面积大批量，上到国家领导下至基层人员，以政治为主的“从严治党”，想利用这场运动彻底清理军队内部势力和政府中的反对势力，表明出了习对任何人都不信任以及防止其他派系联合反习。

**以共灭工的大时代已经开始**

现在处于信息互联网时代，爆料革命唤醒了众多民心，传递给大家真实的信息。美国的总统换届产生了一剂催化作用，拜登当选后成为了中共党内斗争的导火索，拜登没有因硬盘门事件而推翻否决前政府定下的反人类罪，诱因是因为中共内部已经被分解和瓦解并引起了党内人士对习的逼宫，加上他的身体状况和脑瘤，权利带来的神经质，一点风吹草动，将已处于万丈绝境中的习近平推向他更加疯狂并大力反扑。这就激发了人们的愤恨并形成了千钧一发状态，开启了以共灭共大时代运动。

**从严治党**

军队内部人心涣散，之前中共的目标是针对着江，王岐山的改革和自己清廉形象，作为第三方反腐运动，帮习夺回权力，习站在一个貌似客观公正的角度上，各派势力还需要习来做主，说个公道话，都在拉拢他，现在习的地位已经转变为独裁的角色，拜登的就职后没有跟随习，加之脑袋长瘤子，选择集体目标转移到了针对着习的方向，习的如意算盘打空并发出来“从严治党”的号召，就是告诉外界他已经产生的恐惧的信号，发自自己内心的信号就是产生崩溃的起点。

**中共被定义为反人类罪全面发酵**

关于中国，拜登有另一个选择，「美国国际宗教自由委员会」委员鲍尔接受「美国思想领袖」采访时说：中共被认定犯下「种族灭绝」罪恶，这与当年纳粹对犹太人的大屠杀等同！而且美国认定后，全世界的国家和组织都会跟进承认这一认定。由于历史原因，在二战时期，中华民国接纳了大量流亡的犹太人，犹太民族对中国抱有感恩之心，中共借机与以色列建立了良好关系，并获得了大量帮助，但犹太人对反人类罪和种族大屠杀罪是恨之入骨，将会引发犹太势力与中共划清界限，由于犹太人在国际上是以经济利益为导向，政客都是以政治利益为导向，川普政府已经彻底扭转了美国对华的政策，拜登作为新政府要想在政坛上留下政绩就必须要灭共，犹太人因利益差点被希特勒灭族，现在绝对不会对中共问题上重蹈覆辙。这也是习和拜登的最后较量。

**拜登支持川普将美大使馆留在耶路撒冷的意义**

拜登将美国大使馆留在耶律撒冷，不太可能扭转特朗普的以色列政策。这个意义非常重要，耶路撒冷是基督教和伊斯兰教的圣地，是各宗势力的必争之地。明示了欧美人对耶路撒冷的态度，圣地在欧美人心中的位置，这表明了拜登政府对川普政府期间的蓬佩奥国务卿取得的外交政绩的认可，更是表明了不太可能扭转美国对华的政策。明示了美国政府是一致对外的。
