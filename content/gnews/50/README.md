#### 作者：约翰婶婶
![]()![](https://gnews.org/wp-content/uploads/2021/03/a16da256-3692-40a3-94cd-e6f99596db67-e1616750313776.jpeg)图为H＆M上海专卖店。 封面来源：瑞典国家新闻社（TT）
2020年10月21日，良好棉花发展协会发表抵制新疆棉花的声明，直到近日被国内各家媒体报道，这与近期国际社会对新疆人权问题的追责密切相关。中共借此事件为攻打台湾和接下来的闭关锁国作舆论准备。

作为中共大外宣的喉舌胡锡进，近日在微博中对此事件发表了一系列言论。他说，H&M今天冲上热搜是“因为这两天中西围绕新疆问题的意识形态交锋出现白热化，带动了两边的社会情绪。”针对事件的发展，他说，“围绕新疆的对立升级很可能刚开个头。”在挑动民族情绪方面，胡锡进声称，西方最怕“中国民间声音。”“我们在这场斗争中决非如一些人想象的那样无力和被动。”“让我们调动各种力量，发挥他们各自之所长，充满创意地打一场捍卫主权与尊严的战争。”

中共多家官方机构和媒体都在借此挑动民族情绪。外交部、商务部、中消协、共青团等声称，这伤害“中国人”的民族感情、人格尊严。众明星也表态终止合作。3月25日，《环球时报记者探访京沪H&M实体店，有店员强调“我们也爱国”》写到，想赚“中国人”的钱，就更应该尊重“中国人民”的感情。

当中共假骗偷伎俩在国际上无法表演下去的时候，煽动民族情绪就是最后一招。中共现在大规模制造舆论，本质就是想颠倒黑白、让国人误认为，国际社会对新疆人权问题的追责是抹黑中共形象。

中共在民族情绪的把控上向来是游刃有余，屡试不爽。中共经常以激发民族主义巩固自身的政治合法性，同时利用民族主义情绪爆发推进中共的外交目标。在当前内外交困的大环境下，配合“中国人不吃这一套”的战狼外交，中共为转移国内外矛盾进行积极舆论铺垫，但中共这套把戏恐怕不是玩火自焚，就是搬起石头砸自己的脚！
