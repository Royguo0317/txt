## 【香港要闻】夏宝龙曲解邓小平的原意 篡改《基本法》的立法精神
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/945239/)

搜集\编撰：西西 \封面：文粤

**2月22日港澳办主任夏宝龙发表讲话，洋洋洒洒近5500字的，据【众新闻】报道，熟悉中共政情的资深媒体人程翔撰文表示，夏宝龙讲话的内容实质就是篡改《基本法》的立法精神和立法原意，改变该法所规定的「双普选」制度，从而达到两个目的：一是彻底清除香港的非亲共政治力量；二是腰斩香港的民主化进程。**
![]()![](https://gnews.org/wp-content/uploads/2021/03/image001-2.png)
程翔认为第一个目的是：夏宝龙引用邓小平关于「爱国者治港」的原则，只是断章取义地强调这五个字，但对邓小平这五个字以外的其他内容则避而不谈。我们不妨看看他是如何割裂邓小平的原意。 （见附表一）
![]()![](https://gnews.org/wp-content/uploads/2021/03/image003-2.png)资料来源：众新闻
从上表可以看到，邓小平关于「爱国者治港」的论述，其实包涵三个内容：爱国者标准、对治港者的政治要求、以及特区政府人员的构成，是一个比较完整的体系。可以看出，邓小平主张的是一种很宽松、包容、非排他性的原则。但是夏宝龙只执其「爱国者治港」这五个字，却完全删掉他对其他两部分的文字和精神。即使在「爱国者标准」这个根本的问题上，夏宝龙也是塞进私货，把邓小平的三条标准硬推演成「爱国=爱社会主义=爱党」的荒谬结论。

夏宝龙这篇讲话的第二个目的，就是通过修改香港的选举法，达到篡改《基本法》有关香港政治运作的规定，实质性地中断香港的民主化进程。
![]()![](https://gnews.org/wp-content/uploads/2021/03/image005.png)图片来源：苹果日报
在夏宝龙讲话后，香港政府马上按照他的意思出台了界定「爱国」行为的「正面清单」和「负面清单」。 （见附表二）
![]()![](https://gnews.org/wp-content/uploads/2021/03/image007.png)资料来源：众新闻
**根据夏宝龙介定的「不爱国」行为比负面清单还多，资深媒体人程翔撰是这样总结：**

• 从事危害国家主权安全的活动、

• 利用各种手段歇斯底里地攻击中央政府

• 公开宣扬「港独」主张

• 在国际上「唱衰」国家和香港

• 乞求外国对华对港制裁施压

• 触犯《香港国安法》

• 挑战中国国家根本制度、拒不接受或刻意扭曲香港宪制秩序

• 「揽炒派」——不惜把香港毁掉，以此来裹挟民众，胁迫中央

• 反对中国共产党——声称拥护「一国两制」，却反对「一国两制」的创立者和领导者

• 程翔总结夏宝龙讲话的要旨以及香港政府的正负面清单，目的就是要扼杀香港整个社会的反对派，不但不让他们涉足任何公职，甚至可能随时以《国安法》来镇压他们，使香港政坛出现一个集体噤声的局面。

**战友观点：**

2015年12月30日，习近平在十八届中央政治局第二十九次集体学习时的讲话，见《习近平谈爱国》，中共中央党史和文献研究院编辑。习近平曾这么说：「坚持爱国和爱党、爱社会主义相统一是当代中国爱国主义的本质……只有坚持爱国和爱党、爱社会主义相统一，爱国主义才是鲜活的、真实的，这是当代中国爱国主义精神最重要的体现」。

夏宝龙的讲话，实际就是将「爱国=爱社会主义=爱党」的歪理以法律形式强行压给香港人。香港回归之后，左派就一直想确立这个论述，现在就由夏宝龙直接用行政命令强加给每一个香港人。夏宝龙说：「『爱国者治港』必须落实在制度上。…… 确保特别行政区行政、立法、司法机构的组成人员以及重要法定机构的负责人等，都由真正的爱国者担任。重要岗位在任何情况下都不能让反中乱港分子占据」，这就意味着将来特区全部重要职位（官方及半官方）都只能由亲共人士担任。

除此以外，中共有意撤回「双普选」的承诺，或者彻底改变「普选」的定义，实行「有中共特色的普选制度」。

**联合国《公民权利和政治权利国际公约》第25条规定：每位公民应有下列权利和机会，……和不受不合理的限制：**

（甲）直接或通过自由选择的代表参与公共事务；

（乙）在真正的定期的选举中选举和被选举，这种选举应是普遍的和平等的并以无记名投票方式进行，以保证选举人的意志的自由表达；

（丙）在一般的平等的条件下，参加本国公务。
