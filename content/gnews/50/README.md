**西班牙2021年3月3日电/西喜社——**一则在周四公布的视频显示，弗罗里达大学的学生对拜登最近有关中共种族灭绝的态度表示震惊。

在CNN的市政厅活动中，拜登对于种族灭绝的话题发言讲话表示：每一个文明中的领袖有不同的价值观和行动准则去遵守执行。不过华盛顿邮报替白宫指出，拜登同样表示中共会为他们的行为付出代价。

《学院改革》记者奥菲莉·雅克布森（Ophelie Jacobson）到弗罗里达大学采访时询问了学生对于拜登发言的看法，并询问了拜登的发言是否令他们感到不安，美国政府是否应该对中共更强硬。

大部分的学生还以为这是川普的表态（译者注：在美国大学里左派思想站上风，大部分的自由党和左派人士认为川普是一个愿意于独裁者打交道做交易的人。这主要是由于川普和金正恩之间的来往。）

采访记录：

雅克布森：大家好我是来自于《学院改革》的奥菲莉·雅克布森。我们这次要讨论的是中共维吾尔族种族灭绝事件。上周拜登总统表示种族灭绝是中国文化的一部分。我相信你们已经听说过，从2014年开始，中共政府就把维吾尔族人关进集中营，你们对此有何反应？

学生1：我认为我们应该让所有的人都知晓正在发生的事情，以确保我们能够采取行动阻止这些事情的发生。

学生2：很明显这种事简直是糟透了。

学生3：仅仅作为维吾尔族就被杀害实在是太悲惨了。

学生4：让人费解的是在国家层面我们什么行动都没有。

雅克布森：我这里有一段某个总统的发言，请大家猜猜是谁的：“我对习近平在香港的行为，习近平对台湾和维吾尔族使用武力的行为没有看法，这是因为每一个文明中的领袖有不同的价值观和行动准则去遵守执行”。

学生5：我先猜他是川普。

学生3：川普。

学生6：我大概也会猜川普。

学生7：川普或者克林顿。

学生8：奥巴马。

学生9：我觉得是川普说过这话。

雅克布森：那么如果我告诉你们是拜登在市政厅活动上说的这话呢？

学生5：我很惊讶，非常惊讶。

学生8：你会希望这种恶劣的事情会引起美国总统更强烈的反应，尤其是在一个已经算是竞争对手的国家。

学生6：我觉得吧，这种话从拜登嘴里说出来实在是不敢相信。

学生4：大概吧。
