## 【战云长天时评25期】中共是“完美犯罪”学理论践行者——案例十：隐形国情下的反向思维
` 日本东京樱花团` [轉載自GNews](https://gnews.org/zh-hans/946532/)

作者：喜马拉雅东京樱花团 / 捆绑CCP一千年

中共国一年一届的全国人民代表大会和中国人民政治协商会议——“两会”即将在北京召开。而这种劳民伤财的政治秀只不过是中共国内部确权的游戏而已。笔者认为，这就是所谓的中共国国情之一！

每当中共在一些做法上受到西方媒体和政府的质疑批评时，总会冠以“国情”论来搪塞。这显然是一个绑架十四亿中国人的伪命题。笔者相信每个国家都有不同的国情，但世界上除了中共国外，有哪一个民主国家总在外交上以国情论来狡辩、推卸自己责任的？既然是伪命题，为何中共国内人们相信这种混账说辞呢？笔者将从中共一贯使用的反作用思维来谈中共国国情如何完美欺骗了中国人和世界？

中共大外宣新华社3月3日发表一篇题为《钟南山对话福奇：疫情防控需要全球合作》的报道，强调了中美在疫苗上的全球合作，“钟南山认为，新冠病毒是全人类的敌人，假如新冠病毒还在个别国家蔓延，新冠肺炎就不可能在全世界得到控制。”而福奇认为，“至于如何找到经济和防疫之间的平衡，需要多方努力，包括社会科学、政治科学界联合起来，找到真正有效的解决办法。”有意思的是福奇倡议，只有政治科学界联合才能解决与中共全球合作的可能。如果说，中共惯用的伎俩就是反作用逆向思维，人们要读懂中共高大上的文章和新闻报道就一定要“反向”地来看，不然你怎么读得懂中美两大邪恶之子福奇、钟南山的连线会议？企图指望拜登政府放开对中共的政治对话，营造环境，再大谈特谈疫苗下的“群体免疫”这样美丽的谎言来掩盖其完美的犯罪企图？

为什么说“群体免疫”是一个完美欺骗？闫丽梦博士早在2020年多次警告过世界，可惜是无用功。首先，如果在自然界滋生自发的流行病毒环境下，在知道病原体前提下，确定了可信的疫苗，再谈群体免疫是完全正确的。问题是，中共封锁原始毒株，世界尚未在法律上认定中共是超限生物基因战，如果你还相信群体免疫、还相信疫苗是安全的，岂不是自欺欺人？根据中共国自己的生物基因战经典著作中就可以得知，中共国不仅发动了超限战，还在藉着疫苗发动超限生物基因战。笔者摘录一段文字表明他们自己证明自己做了这些：

根据“**基因治疗，是以更换或更换缺陷的基因，使人的遗传组成发生永久性变化**，以治疗某些疾病，有益于人类。然而，生物武器制造者**却可以反向使用，插入致病基因**。……逆转录病毒（HV可代表）、痘苗病毒等可永久性整合进人类染色体。……最终可能以类似的载体**将有害基因插入到某一不知情的人群**”。（见《非典非自然起源和人制人新种病毒基因武器》91—92页）笔者想请问各位，难道美国的福奇不知道他们在做什么？难道钟南山不知道这些？看看科兴疫苗和辉瑞疫苗在全球致病的巨大副作用便知，根据这个描述，他们己制造了杀人的生物基因武器，正在持续杀死了地球亿万人生命，并且仍在不遗余力地制造第二轮杀戮武器——疫苗，人们高度信赖的帮助免疫阻隔流行病的疫苗。可谓前罪未被定罪，便以更大更隐蔽的恶来掩盖之前所做的恶，即通过注射疫苗，用反向使用原理，插入致病基因，使人的遗传组成发生永久性变化”。问题是两者都使他们赚取了巨额财富，笔者对他们的完美犯罪计划感到万分震惊。

震惊！震怒之余，另一件正在发生的事情就是中共在即将召开的两会前发表了《毛泽东关于对美斗争的六则论断及其启示》的文章。据中共党史署名作者戚义明于3月3日发表的文章看来是赤裸裸地宣布即将再次发动文化大革命的野心，并再次将美国作为首要敌人。

这不仅仅是典型的文革思维，更是政党层面的犯罪升级。文革中，毛泽东成功炮制了“美帝国主义是纸老虎”这个概念，达到了激发民族主义者“仇美”（仇恨美国）的名族仇恨。这一仇美的宣传已经成为中共思想病毒的一种，他仍然隐藏在很多中共国人们的心里，一旦中共政权遇到危机，他就随时点燃这种仇美火花，激活早已种植在人性里的“毛思想病毒”——隐形病毒（Sealh virusce），在病毒学界指的是一种“特殊感染途径和过程的病毒”。这个特殊感染途径就是漫长的立体化的中共思想洗脑过程。中共建党100周年，执政长达70多年，这种从教育体系到各行业的生产过程中全面劫持了中国人的思维方式和思考方式，甚至无一幸免。尽管如此，但笔者总是强调，上帝仍然为中国存留亿万万没有向中共下跪的义人，如郭文贵先生领导的爆料革命战友们。

如果习近平政权文革式的做法是继续高举人民仇视美国那就意味着承认对美作战的全面失败，或由进攻转为退守（如果无法继续投毒）。习近平在篡夺政权上采取了借尸还魂的反向做法。全面文革化是一种伤害自己本民族的犯罪行为，目的是自保王位。笔者曾论述过借尸还魂是《孙子兵法》第十四计，即习近平现在必须要借毛泽东的尸体，又借助毛泽东的理论（思想灵魂）形式在习近平的肉体中复活的做法。笔者援引《毛泽东关于对美斗争的六则论断及其启示》该文作者原话如下：

“毛泽东关于‘美帝国主义是纸老虎’的论断启示我们：虽然美国现在还是头号资本主义强国。……但从长远看，和平发展仍然是时代主题。世界格局正在发生深刻变化，广大发展中国家和新兴市场国家的不断壮大势不可挡，中华民族伟大复兴的坚定步伐势不可挡，‘时’和‘势’均有利于我。”

毋庸置疑，再次把“美帝国主义是纸老虎”的文革口号搬出来就已经是吹响文革号角，吹响从西方战场撤退的声音，尽管如此，这段话的意思仍然是“煮熟了的鸭子嘴硬”（民间谚语）。中共仿佛在说，这个“纸老虎”是最强大的国家，虽然从硬军事实力上打不过“纸老虎”，那中共就用战争**反向使用**的思维，乘着和平时期人们对“疫苗基因武器”和“人制人新种超限生物基因武器“认知上的空白之际，中共悍然在和平年代发动了全人类有史以来的第三次世界大战。值得关注的是，这似乎和彭佩奥等共和党人——美国保守力量大会形成鲜明对比，这次大会被视为美国吹响了全面灭共集结号。（见3月2日美国时间（路博艾冠谈节目））而洗脚习近平全面文革化就是退守动作。

文革后，毛泽东钦定了华国锋为接班人。而本届（20大）两会的内幕就要释放这些腐朽的尸臭味。正如郭文贵先生说，这次两会的主要目的就是要全面文革化，习近平要指派陈敏尔为接班人，废除中共国领导人由全国人大伪民主选举产生的“党内民主”制度。尽管中共人大选举的是国家主席，但党国一体化政治可被视为一个意思。中共大外宣新华社头条证实了这一点。（见《新华社头条》链接）

中共就如何钦定习近平的忠实“传人”？需要拿出传人培育的修炼“秘诀”。而陈敏尔隐藏可谓很深，记得《路德社》在曝光习远平的情人景甜时也曝光过陈敏尔。习近平此时明确提出对传人的钦定过程要做到在“特定时间、特定地点、特定对象上帮助中青年干部成长成才”。显然这是公然的“三双休制度”啊！三个“特定”修炼“秘诀”是不是有点类似于藏传佛教的活佛转世确权制度呢？即笔者在《极权下的宗教与人权》一文里提到的“国内寻访”活佛的“受封”“定制”“掣签”这三特定呢？想必一定会炮制出一些神话来，表明自己是“真命天子”。如果说这也是中（共）国国情，那一定是中共皇权思维使然。笔者缺乏想象，究竟在接班人上如何对其人选进行修炼？秘诀一定要过“圣人”关，必先在72妃子的密室中修炼才能无色无情无我吧。想想景甜为何屡次出演大戏而不红火？或者练就陈峰王岐山等杀人不会眨眼之功？总之，必须修炼出金刚之体，不然美国蜻蜓无人机一咬就毙命。

文革思维是不是中共病毒？笔者在就相关话题和中共官员讨论时发现，他们都直接或间接地认为（文革）那是在当时中（共）国国情下必须要这么做的事情，笔者曾就文革问题再次询问一位中共国前国家扶贫办官员，通过与其对话确认他骨子里的毛泽东思维根深蒂固，他说：“如果不搞文革，美国和西方国家就要侵略我们”。这位官员还认为温家宝对“文革是个错误”的评价是不对的。而另一位中科院某部党支部书记则一身清廉，对于中共国是否犯下种族灭绝罪和发动了不受控制的生物战这样的现实，他们是坚决不相信这种逆天的犯罪事实是他们伟光正的党做的。如果一个政党的生存能力总是满足于给自己的人民制造一种虚幻的光明景象，违心地树立一个假想敌，以此制造国家仇恨，迫使人民自觉隔绝进步思想，使人们彻底臣服在中共极权下，甘做奴隶的这一套威权做法本身就是极其反人类的做法。这些都是中共思想病毒使然，笔者相信大多数共产党员都是被利用的还有良知的中国人。

长期以来，中国人实际处于世界最大的监狱中慢性自杀，消耗了青春、被迫改变了基因、葬送了灵魂、赔上了自己子孙的前途，而中共高官子女却生活在他一手炮制的人民仇恨的“美帝国主义纸老虎”中享受荣华富贵，过着王子与公主般的生活。他们没有带领自己同胞走向“共同富裕”，而是朝着相反的方向前进，如文革时期，把“人民”高举起来，扔进人间炼狱，这也许是笔者对中共“高举人民”最好的诠释吧。

西方世界似乎都明白习近平高举人民的险恶用心，BBC中文早在2017年6月的一篇报道中就这样断定习近平的伟大复兴梦是一场噩梦，“他认为这种民族复兴的理念有着令人不安的历史前例，特别是在20世纪欧洲早期的扩张时期。……中（共）国伟大复兴还有着复仇的意味在里面”。（见 BBC中文）

习近平的这种拙劣骗术竟然是借助互联网防火墙和“毛思想的墙”就轻而易举地达到了完美犯罪结果。一个现实版的《悲惨世界》不能再继续让他们在中国延续他们的完美犯罪过程。毛泽东这种“与天斗与地斗与人斗其乐无穷”的撕裂社会的黑帮文化病毒一定要彻底从中国大陆和全世界范围内移除，被他欺骗的人民总有一天要起来审判他们。因为中共政权既得罪了“天”，又得罪了“地”，他们诱导、激动人与人之间的猜忌，破坏性地创造了互斗互害的社会范式，抵制了人类一切进步的思想。中共习近平及其政党把毛思想包裹着习近平的“新时代”思想病毒合成马克思致癌基因的DNA片段，以隐晦的方式施放这种“隐形病毒”，将人类作为感染靶人群，使其“无形”地传播于世界任何一个可以渗透的地区、国家。如果笔者这种诠释有半点夸张，你就是在否认中共病毒是中共超限生物武器，它正在成为“全人类的敌人”。
