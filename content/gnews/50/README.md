澳大利亚战略咨询总监亚瑟·切伦科夫（Arthur Chrenkoff）3月5日在《澳大利亚观察家报》撰文指出中共国驻澳大利亚副大使王晰宁上周在澳大利亚中国工商业委员会在堪培拉举办的晚宴上就“如今在澳大利亚做中国的朋友是多么困难”这一话题发表了讲话，切伦科夫就此进行了抨击。

王晰宁在讲话中说道：“现在我们的澳大利亚朋友们处境艰难，但你们的中国朋友和你们站在一起。历史会证明，你们选择中国作为朋友，是明智的且具有远见的。你们的子孙会因为你们的选择而骄傲、而受益，你们的中国朋友和他们的孩子也为有你们这样的朋友感到鼓舞和自豪。为了美好的明天，大多数人选择多交朋友，但总有那么一小撮人选择多树敌人。但那些出于一己私利、蓄意污蔑诋毁中国，破坏中澳友谊、损害两国人民福祉的败类会遭到世人唾弃，他们的后代会羞于提及他们在历史中的反面作用。”

“目前，中共国的朋友被冠以“拥抱熊猫”的绰号。它曾经是一个表示赞赏的词，但现在却带有贬义。如今对中共国友好，成为其朋友在澳大利亚是一种罪过和错误。而对中国人采取严厉的态度则是爱国的澳大利亚人的立法立场。只有这样，澳大利亚的政治家才能赢得澳大利亚人民的心。今天，在澳大利亚很难成为中共国的朋友。”

New.com.au网站对这篇演讲的中文原文进行核实时指出，讲话的英译文中少了一个词，即王晰宁指责那些诋毁中共国的人为“败类”或“人渣”。

按照中共的说法，切伦科夫很骄傲自己是中共所谴责的“败类”中的一员。

切伦科夫指出：中国共产党有个妙招，说自己是中国或中国人的同义词。之所故意混淆是因为：首先，它错误地把中国描绘成一个整体，国家、人民和政府是统一的；其次，因为通过模糊三者之间的界限，它可以把任何批评描绘成对国家和人民的攻击。

当然，事实并非如此。没有人把中国当作一个国家或整个中国人民来诋毁。问题在于中国共产党，它虽然放弃了共产主义作为一种经济主张的承诺，却坚定地将共产主义作为一党专政的政府体制。这是一个害怕世界和害怕自己人民的政府/政党，否则它会欢迎人民通过自由民主选举来判断和问责。中共不像过去那样疯狂杀人了(在1949年到1976年间，大约有6000万人被谋杀、饿死或累死，几乎相当于两个半澳大利亚人口)，但中共政府仍不承认其国民的任何人权，继续压迫少数民族和异议人士，将100万维吾尔人关在集中营里，并在海外有扩张野心，不仅是觊觎邻国领土，还想成为超级大国和全球领袖。指出这些事情并不是要诋毁中国，而是要指出中共政府的本质。

毫不奇怪，本届中共政府禁止讨论“七大禁忌”，包括普世价值、言论自由、公民社会、公民权利、司法独立、党的历史错误和裙带资本主义。

作者自豪被中共称为“败类”。认为出生在共产主义政权下的自己，被另一个更狡猾、更诡诈因而对世界更加危险的政权称作“败类”是一种荣誉勋章。

切伦科夫指出他不会参加中共肮脏的游戏，通过商业交易赚钱，或因追随中共的路线而得到更直接的补偿，。他认为中共所指责反共人士所谓的“自私”，只是希望生活在一个更多的人享受和平和自由的世界里。他希望自己能亲眼看到这个独裁专制的政权的末日到来而不是与其同流合污。
