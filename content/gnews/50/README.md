《路德时评》2021年4月26日早间节目主要内容：

·赵婷从曾经的中共骄傲到全网封杀

BBC新闻网4月26日报道[\[1\]](https://www.bbc.com/zhongwen/simp/chinese-news-56885536)：39岁的华人女导演赵婷凭电影《游牧人生》（Nomadland，《无依之地》/《浪迹天地》）成功获得2021年奥斯卡最佳导演奖，她成为首个获此殊荣的非白人女性。中共对这一消息采取了封禁措施，相关网络帖子被大面积移除。

在摘得奥斯卡最佳导演奖项前，赵婷已在今年2月底获得金球奖电影类最佳导演奖，那时她一度受到中国官媒的褒奖和网友的热议。但好景不长，一些中共国网友挖出她此前所谓的“辱华”言论。在2013年一段接受美国电影杂志《电影人》（Filmmaker）的采访中，赵婷回顾自己的童年时表示，当时的中共国是“一个遍地谎言的地方”。
![]()![](https://gnews.org/wp-content/uploads/2021/04/id12906373-GettyImages-1314472671-450x300-1.jpg)[图片地址](https://i.epochtimes.com/assets/uploads/2021/04/id12906373-GettyImages-1314472671-450x300.jpg)
路德社评论：

1、《游牧人生》这部文艺片在美国获得左右两派人士认可，在左派人士眼里，这是一部追求自由的电影；在右派人士眼里，这是一部反映08年金融危机之后美国经济的真实写照，凸显出美国急需一场重大变革来扭转现状。华裔在这两年将成为全球焦点，在全球围剿中共大势形成之际，中共试图打种族歧视牌进行反击，用种族歧视挑起中华民族与西方社会的民族矛盾。

2、当赵婷拿下金球奖时，《环球时报》还报道说这是中国的骄傲，当时网易爆料赵婷疑似“辱华言论”，说赵婷在接受采访时称自己是美国人，中共官媒随后就彻底封杀赵婷，再次显露出中共的丑恶嘴脸。

3、赵婷曾就读于麻省文理学院，主修政治学专业，可见她对于中西方政治情况非常了解。她此次导演的《游牧人生》并非商业大制作，属于投资较小的文艺片。赵婷在这部电影中花费了大量心思，中共假骗偷文化是不可能培养出这样年轻有为的导演。

4、从中共国全网删除有关赵婷帖子这件事就可以看出，赵婷对中共国的要挟没有丝毫妥协，奥斯卡也没有因中共的威胁而妥协。赵婷获得奥斯卡大奖和美国正在全面推动灭共有非常重要的联系，这件事将成为好莱坞影视文化圈转向灭共的标志性事件。美国前国家安全顾问Jamie Metzl在发推力挺赵婷的同时，还提及要彻查中共病毒来源真相。

5、拜登政府上台后就开始推动美国各行业与中共国脱钩，体育行业、影视行业、科学行业、纺织行业等无不如此。如果中共继续封杀赵婷获奥斯卡奖的消息，美国影视圈甚至将转向拍摄有关中共病毒来源真相的纪录片，影视文化圈的话语权正在转向灭共。

6、赵婷的成功经历证明远离中共可以让人生过得更自由，更精彩，闫博士的经历更是如此，只有灭掉中共，墙内人民才有希望，中华民族才有机会重新崛起。
