《福克斯新闻》今早发文，题目“麦卡锡提交议案，因曾经涉嫌与中共国间谍有染，斯瓦韦尔该被踢出情报委员会。”，文章副标题，众议院议员将被迫就“特权”议案进行投票。【1】

全文如下：

众议院少数党领袖凯文·麦卡锡（Kevin McCarthy）周四提交了一份议案，旨在将民主党众议员埃里克·斯瓦韦尔从众议院情报常设特别委员会（House Permanent Select Committee on Intelligence）中除名，原因是他被指控过去与一名中共国间谍方芳（Fang Fang）有染。

《福克斯新闻》获得的这份长达两页的议案中说道，加州的斯瓦韦尔没有否认公开报道，一名疑似中共国情报人员帮助了斯瓦韦尔众议员的政治活动筹款，也没有否认“公开报道的其他麻烦”。

与方芳的关系最早是由《轴线新闻》（Axios）披露的，《轴线新闻》报道说方芳帮助了斯瓦韦尔的竞选筹款活动，并在斯瓦韦尔的办公室安排了一名或多名实习生。共和党人说，一个向外国势力严重妥协的人不应该留在关键情报小组。

加利福尼亚州麦卡锡议员的议案还引用了众议院的规定，“众议院议员……在任何时候都应以对众议院有良好影响的方式行事。”

斯瓦韦尔没有被指控有任何不当行为，当他得知方是间谍时，他与联邦调查局合作。但他一再拒绝回答《福克斯新闻》关于他和方芳的关系是浪漫，还是性，还是自然的问题。

斯瓦韦尔的办公室根据国会议员上周就他在情报小组中的角色发表的声明，回复了《福克斯新闻》，斯瓦韦尔说，“我渴望能继续（保留）这份重要的工作，这份工作能保护美国人和支持我们情报界辛勤工作的英雄，我感谢佩洛西议长和希夫主席的支持和信心。”

据《轴线新闻》报道，方芳曾与多名民选官员发生性关系或浪漫关系，其中至少包括两名中西部市长。

但斯瓦韦尔在民主党人中仍然得到广泛支持，他被任命为参议院第二次审判川普总统的众议院弹劾经理之一，最近又被众议院议长南希·佩洛西（Nancy Pelosi）任命为英特尔小组成员。

麦卡锡议员在众议院提出这项议案后，该机构必须立即或在两个立法日内予以审议，目前还不清楚该议案是否会在周四提出，因为民主党人很可能会把这项决议“摆到桌上”，而这项议案将在没有对议案的实质进行实际表决的情况下被搁置一边。

麦卡锡议员上周暗示，他将提出一项议案，将斯瓦韦尔从情报委员会中除名。

佩洛西刚刚任命埃里克·斯瓦韦尔为情报委员会委员，麦卡锡在推特上说，“根据我和她一起收到的简报，斯瓦韦尔不应该在负责保守我们国家秘密的小组中。” （原文完）
