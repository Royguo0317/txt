## 【香草诗词汇】酒歌（一）
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/958797/)

作者：香草山写作组 木白

![]()![](https://gnews.org/wp-content/uploads/2021/03/MOS1440-【香草诗词汇】酒歌（一）-1-rotated.jpg)

我想朗诵我写的诗，

也想把我喝的酒分享给你。

同样的味道，不同的人生。

同样的醉意，不同的幻觉。

同样的生活长在不同的翅膀上。

用同样的胃正消化着这个虚假的世界。

用押韵的味觉尝一口春天的味道。

用未洗过的嘴吻一吻刚掉落的梅花。

幸运的人，我听见了骨骼破碎的声音——

正酿造着这座小城的孤寂。

2021年3月7日
