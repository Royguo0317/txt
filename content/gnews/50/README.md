## 用灭共经济思维回击任泽平中国经济过热滞胀说

编辑：MG4

摘要：

在研究与反驳任泽平的“中国经济正从复苏转入过热滞胀”说的过程中，发现了中国经济现状中三个违背经济规律的矛盾现象。因此，本文结合货币金融学理论并采用了新的经济思维框架来研究中国经济问题：中共不能代表中国人民，所以中共的经济也不能等于中国人民的经济。中国的经济是中共像病毒与寄生虫一样附着在中国人民之上，不断吸取中国人民养份的畸形经济，所以要把两者解耦并分开来看待。文中提出的三个矛盾关系，便可以分解为中共控制的总供给与中国人民的总需求、中共制造的通胀与中国人民的通缩、中共控制的经济增长与中国人民的潜在经济增长。由此得出的结论为：中共的经济处于虚假过热的阶段，而中国人民的经济却陷入了被寄生的“宿主性萧条”。同时，我们不能把中国的经济简单描述成为滞胀，因为中国的滞胀现象不完全是由理论上的形成滞胀所需的因素所导致，而是中共经济与中国人民的经济两者的现状所叠加出的现象。

关键词：经济过热；滞胀；总需求；总供给；通胀与通缩；经济增长；货币流通速度；

近期中共经济学家任泽平撰文“滞胀来了”，称中国经济周期正从复苏转入过热和滞胀，通胀预期抬升，结构性资产价格泡沫化，我们可能正站在广义流动性的周期性拐点上。任泽平此篇文章是把中国经济所反映出的表象问题用所谓的经济周期与全球化经济的理论来解释，为现状找理由是为了转移国内公众注意力，掩盖造成经济萧条的真实原因。本文结合货币金融学理论，并用一种新的经济思维框架来分析中国的经济问题，揭示中国当前的经济现状并非来源于自由发展的市场经济周期现象，而是中共制造的中国经济“冰河时代”。

研究中国经济是否过热，我们先要理解何谓“经济过热”？依据经济学定义，经济过热是实际经济增长率超过潜在经济增长率。经历了长时间的经济高速增长后，消费者增加的财富所带来的通货膨胀推动了市场总需求以及过度投资的增长。当经济过热发生时，其生产能力无法跟上日益增长的总需求，基本特征表现为经济要素总需求超过总供给，由此引发物价指数全面持续的上涨，这是一个不可持续的高比率的经济增长速度。

从定义中发现了两个关键点，其一是实际经济增长率超过潜在经济增长率，其二是总需求超过总供给，我们先从供给需求切入来分析。众所周知，中国经济由于近年来贸易战、香港台湾新疆问题、生物武器袭击全球、国际反共灭共的制裁压力，以及对内盗取民财、倒行逆施等诸多因素令中国经济连续下滑，即使在疫情后的复苏也是相对于2020年二三月份冰点时期的相对性复苏。这些问题令国内企业经营困难、频现倒闭潮与债务违约潮、金融投资爆雷，随之而来的是百姓失业率增高、收入下降，以及天灾与人祸所造成的死亡，是社会财富骤减，从而使消费支付与投资减少，导致社会总需求下降，从以上情况可初步判断出并未出现总需求超过总供给的现象，物价上涨也并非由高涨的需求所致。从整体来看中国经济何来“复苏”，又何来“过热”呢？下文开始对这个问题进行深入研究。

在以上所描述的现状中，我们需要关注一个重要的现象，老百姓财富的减少、消费的减少会导致社会中货币流通速度下降。与此同时，中共在刻意管控百姓资金流动，包括限制大额资金提现，限制人民币自由兑换外币，郭文贵先生在直播爆料中谈到中共甚至连外币兑换人民币也会管控，中共是不希望大额资金流动，因为它是假的不能碰，一碰整个体系的问题就会暴露。这样做的结果必然再次导致货币流通速度下降。包括中共正在推广的央行数字货币，货币本应具有媒介交换、价值衡量、财富贮藏的基本功能，而央行的数字货币的诞生是为了加强极权统治与管控人民，以及挑战国际现有储备货币地位，并不是为人类提高生产生活效率而创造的财富和价值。如果央行数字货币无法兑换外汇和黄金，就失去了作为货币的基本职能，以统治与管控为目的的伪货币会强化冻结扣减资金及限制自由交易等功能，同样会导致货币流通速度的下降。

通过货币金融学的方程式来分析货币流通速度下降会对中国经济产生的影响，货币流通速度是经济总收入（通常为国内生产总值）与货币供给量的比率，用公式表示为：M货币供应总量 \* V货币流通速度 = P价格水平 \* Y实际经济产出（MV=PY），其中PY可以理解为名义GDP。由此能得到的第一个结论是，在货币流通速度V下降的情况下，中共要维持虚假的经济增长就必须要增加货币的供应M，中共会让央行通过货币政策进行基础货币供应，再通过商业银行进行货币乘数放大（俗称的开动印钞机）。

再来看等式的右边PY，社会需求减少、民企倒闭、外企加速撤离、原材料成本上涨等诸多因素导致实际经济产出Y下降，可以理解为总供给下降，从而又助推了百姓失业率升高、收入减少与需求减少。中共想要维持虚假经济增长，就需要用财政政策与货币政策来增加总需求的方式带动总供给的增长，从而实现经济产出的增长，这就是中共说的经济内循环扩大内需。首先我们需要了解需求包括消费、投资、政府支出、净出口。其中老百姓的消费与投资能力是下降的，因为失业与收入减少、金融骗局与投资违约爆雷都令财富积累缩减，所以中共要提高居民消费来带动经济增长，没钱就负债消费寅吃卯粮用债务拉动经济增长，并进行错误的消费投资观念的洗脑。但是消费金融属于非生产性债务，不能通过创造收益来偿还利息，在萧条的环境中不能持续支撑消费增长。春节前中共央行突然发文提到：“不宜以消费金融来扩大消费”，说明这部分债务风险已经非常严重。关于投资，除了国内投资外，香港问题与贸易战以及制裁令海外投资也受到影响。在谈出口，关税、原材料上涨、外企撤离、墙内企业倒闭减产、物价上涨以及疫情以来利差造成的人民币升值都在冲击着出口。所以中共想要维持总需求的增长，只能通过增加政府支出与减税的财政政策来弥补消费投资出口需求的下降。

近期中共财政部发布《2020年中国财政政策执行情况报告》，其中包括“财政赤字率增加到3.6%以上，赤字规模比2019年增加1万亿元，达到3.76万亿元。发现1万亿元抗疫特别国债，增加1.6万亿元地方政府专项债券。2020年全国一般公共预算支出达到24.56万亿元，同比增长2.8%”，中共的财政报告也验证了上文观点。从理论上讲扩大财政支出的方式拉动市场总需求的增加，从而推动总供给的增加与价格水平的上涨，进而推动社会收入上涨与通货膨胀，维持经济增长。但实际上中国经济并不是按照这样的逻辑来发展，中共的财政支出的资金被中共所控制，绝大多数资金只能流向中共家族、国企、中共控制的企业以及与中共紧密相关的人手中，这部分少数人控制的财富最终被藏匿或转移至海外，而不会流入社会从而推高老百姓的收入与民营企业的利润。比如全国推广5G基础设施建设或事疫苗出口的利润怎能让老百姓享受。但是中共的总需求与相关产业总供给的增长所带动的价格的上涨却是不可逆的，这是属于需求推动型，此外还包括由于原材料上涨与高关税等原因所导致的成本推动型物价上涨。比如出口行业，在高关税、高成本、高汇率的条件下，想要生存只能靠压榨劳动力工资的方式降低成本，进而又降低了老百姓的收入与需求。

所以得出的结论是：中国经济供给与需求的关系是中共为维持虚假经济增长而增加需求所拉动的高价格的、以及国际政治与市场环境所导致的高成本的总供给与中国老百姓被榨取财富后逐渐萧条的总需求之间的不平衡的畸形关系。这是一种寄生性吸血、违背正常经济规律的、矛盾的供需关系，这是本文提出的第一个存在于中国经济体中的矛盾关系。导致中国老百姓面对高物价与低收入的巨大反差是中共体制造成的必然结果，这也是虚假经济的一种体现。

综上所述，中国经济的总需求未超过总供给，低收入无法推高社会需求持续增长，从而无法推动良性的经济增长，所以中国经济并不属于过热。这就是为什么物价持续升高，而6亿人月收入却不到1千元的原因。

此外《财政报告》还提到，中共扩大财政赤字以及增发特别国债与地方政府债券。一方面，在外汇储备吃紧的情况下，对于扩大财政赤字的财政政策，央行会选择应用“无锚印钞”增加货币供给的方式继续放水。另一方面，中共通过扩大债务来维持政府支出与中国经济增长，从而制造政府与全民负债的巨大债务泡沫与危机。近几年各种债务违约的数量与金额逐年升高，其中不乏中共控制的国企债、城投债违约。中共通过负债再违约向银行等金融机构吸血，相当于向金融投资者与银行储户的财富吸血使之造成损失，最终钱又被转移至少数人手中。由于财富缩减与盲目负债又降低了个人和民企的债务偿还能力，助推了债务危机。使银行坏账陡增与资金紧缩，将会导致市场利率升高与资金成本上升，引发金融风险。最近微信群中在流传银行抽贷与收缩贷款业务，说明债务违约与银行资金紧张已经令中共高度警惕，这又呼应了前文提到的中共限制大额提现，降低通货比率维持银行运转。综上所述可以发现，中共增加货币供给的放水货币政策在理论上所造成的通货膨胀与老百姓与民营企业的财富缩减所导致的通货紧缩形成了矛盾的关系，这是本文提出的第二个存在于中国经济体中的矛盾关系。钱都去哪了？

分析了供给与需求之后，再来关注经济过热的另一个现象，实际经济增长率超过潜在经济增长率。所谓潜在的经济增长率是指一国或地区在各种资源得到最优化和充分配置条件下，所能达到的最大经济增长率，这里的资源包括自然资源、人力资源、技术与管理方面。一些观点认为，因为中国存在大量失业人口，人力资源没有得到最优化的充分配置，中国的实际经济增长率并没有超过潜在的经济增长率。我不同意上述的观点，中共以专政集权的方式统治中国并控制了绝大多数的资源与分配方式，包括在商品市场和劳动市场的管制、户籍制度对人口流动的影响、对土地的掠夺占有以及对自然资源与技术的控制，正所谓“一切都是党的，一切都听党的”。在这种情况下，留给中国老百姓和民营企业可掌握的资源与资金已经非常有限，但仍然还要被层层盘剥设置障碍。所以在近年来国进民退的背景下，中国人民在有限资金与资源的条件下所能达到的潜在经济增长率很难超过中共把所有资源资金掠夺于一身在自认为最充分配置下所维持的虚假繁荣的“实际”经济增长。中共对市场与资源资金的控制力越强，“实际”经济增长率与“潜在”经济增长率的差距就越大，这又形成了一个不符合经济学原则的矛盾关系。如果按照我的观点，中国的实际经济增长率超过了潜在的经济增长率，这又符合经济过热的定义，但又与上文分析的经济总需求未超过总供给而得出的未过热的结论出现了矛盾，这是本文提出的第三个存在于中国经济体中的矛盾关系。

至此本文提出了一个问题以及在分析问题过程中发现的三个有违经济学规律的矛盾关系。一个问题是中国经济是否因复苏导致过热与滞胀？三个矛盾关系分别是社会总供给与社会总需求的矛盾、通货膨胀与通货紧缩的矛盾、实际经济增长与潜在经济增长的矛盾。在中国的经济体中为什么会存在着诸多违背经济原则的矛盾关系，因为传统的经济学理论源自于西方，是在拥有法治自由的体制下逐步完善提炼出的理论体系。而中国的经济是中共像病毒与寄生虫一样附着在中国与中国人民之上，不断吸取中国人民的养分使中共变为庞然大物的畸形经济，所以本文采用了一种新的经济思维框架来研究中国经济问题。中共不等于中国、不等于中国人民，我们一直倡导的更多是政治方面，而在研究中国经济问题上，我们同样应该把中共与中国人民分开来看待，中共的经济不等于中国人民的经济，两者属于零和博弈关系。如果把两者看作一个整体，在分析中国经济问题时就会出现有违经济规律的矛盾点。由此，文中谈到的中国经济体中的矛盾关系，可以拆解为中共控制的总供给与中国人民萧条的总需求、中共制造的通胀与中国人民的通缩、中共控制的经济增长与中国人民的潜在经济增长，如果将中国经济置于把中共与中国人民解耦的大框架之下，那么文中提到的存在于一个经济体中的诸多矛盾关系就可以理解了。

回击任泽平“中国经济周期正从复苏转入过热和滞胀”说，根据本篇文章提出的新的思维框架，这个问题我们应该把中共与中国人民分开来分析。根据前文所述可以得出结论：中共的经济处于虚假过热阶段，而中国人民的经济可称之为陷入了被寄生的“宿主性萧条”或者说是“病毒性萧条”，因为中共寄生在中国人民之上，不但不断吸收中国人民创造的财富养分，还阻止中国人民自由获得外界的新的养分，中共本身就是病毒。不仅如此，中共还通过释放超限生物武器把这种经济模式扩张至全世界，如今全球的经济现状是中共试图通过病毒、疫苗、疫苗护照等超限武器绑架全球的最好证明。

关于滞胀，滞胀全称为停止性通货膨胀又称萧条性通货膨胀或萧条衰退，指经济停滞、失业与物价上涨通货膨胀同时持续高涨的经济现象。中国经济当前确实出现了与滞胀定义中相类似的现象，包括失业与物价上涨，通胀与萧条，但我们所看到的所谓中国经济中的滞胀的现象，并非是由复苏与过热所导致。我们不能把中国的经济简单的描述成为滞胀，因为中国经济中的滞胀现象不完全是由理论上形成滞胀所需的因素所导致，而是中共经济与中国人民的经济两者的现状所叠加出的现象而已。所以我们必须从经济领域把中共与中国人民解耦，其中的问题与矛盾才能有和理解释。为什么说中共的经济是虚假经济，除了中共数据造假之外，还因为它不属于中国人民。中国人民的经济必须与中共的经济脱钩，摆脱这种被吸血寄生的关系，中国的经济才有未来，唯有灭中共才能改变现状。
