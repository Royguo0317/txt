在中共国驻法大使馆的声明中说到， “如果说有’战狼’，那是因为有疯狂的鬣狗在攻击我们。”

战狼对疯狂的鬣狗？不，我们不是在’狮子王’的电影里，而是在这场大流行留下的新的（动荡的）地缘政治场景里。因为世界第二超级大国的大使馆和一个单纯的法国学者在推特上的辩证对抗，不是外交上的失误，也不是单纯的轶事，而是新时代来临的标志。

几年来，中国外交一直在境外采取更加咄咄逼人的语气，以便向西方国家表明，他们不再接受规则。这种强硬的态度被称为 “战狼外交”，一些年轻的外交官和中国民族主义者，在日益增长的爱国主义的激励下，正在用这种外交方式来对抗他们认为的’西方偏见’。”

对于习近平来说，中共病毒疫情已经显示出 “东方兴，西方衰”。2020年以来，中国与无数国家在台湾地位、南海控制权、病毒的来源、对维吾尔族的文化种族灭绝等问题上发生了冲突。而为了恐吓对手，压制批评的声音，它对边境上的印度士兵进行了打击，扼杀了整个澳大利亚产业的出口，并在最近的一次事件中，在美国本土对美国外交官进行咆哮。

法国参议员们义愤填膺，要求外交部长让-伊夫-勒德里安作出解释，而勒德里安又召见中国大使进行磋商，给他念了抗议书。勒德里安几天后在接受《世界报》采访时说：”北京在玩分裂欧盟的游戏，”
