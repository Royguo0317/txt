**西班牙2021年2月19日电/西喜社——**一项受到密切监测的调查发现，英国各地新的中共病毒感染率大幅下降，就在首相鲍里斯-约翰逊概述未来几周和几个月内如何放松英格兰的封锁限制的路线图之前几天。

国家统计局在其每周的感染调查中表示，英国四个地区的传播率都有所下降。在英格兰，它估计在截至2月12日的一周内，每115人中就有一人的中共病毒检测呈阳性。在前一周，该比率为80分之一。

类似的情况出现在英国的其他三个地区—苏格兰、威尔士和北爱尔兰。

整个英国是欧洲中共病毒相关死亡人数最多的国家，达到近12万人，处于封锁状态。领导人对解除封锁限制持谨慎态度，但希望在疫苗迅速推广后，从下月起可以放宽许多限制。周一，约翰逊预计将允许部分学生在3月8日返校。
