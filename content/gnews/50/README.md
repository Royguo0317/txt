## 中共可能先攻东沙后打澎湖 将数十万岛民当人质
` 喜联盟编辑部 Himalaya Alliance Editorial Team` [轉載自GNews](https://gnews.org/zh-hans/936153/)

**作者：**军迷Wilson
![]()![](https://gnews.org/wp-content/uploads/2021/02/2021-02-27_coverphoto.png)(图片来自网络)
风传媒2月25日报道，中华民国前国防部长杨念祖在台湾智库举行的应对中共军事威胁座谈会上指出，中共如果对台动武，可能攻打台湾的两个离岛——先攻打东沙，如果台湾不屈服，再进攻澎湖列岛，将岛上数十万居民当作人质，进一步胁迫台湾投降。

杨念祖先生指出，当前中共已经用尽了发动热战前所有的卑劣手段，目的是削弱台湾军民的反抗意志。同时他对美国在台湾协防问题上表达了疑虑: 当前CCP病毒已使美国死亡50万人，远高于该国在一战、二战及韩战中阵亡军人的总人数。那么问题来了，美国还愿意为了协防台湾作出更大的牺性吗？

同时，由于美国的经济遭受CCP病毒重创，债台高筑，战斗力已被大大削弱，而且美国对台湾没有法定的防卫义务，既没有《日美安保条约》这样的防卫协定，又对台一直坚持对中共有利的一中政策，对台防卫一直采取模糊策略，因此杨念祖先生的疑虑不无道理。尤其是现任美国总统认为俄罗斯才是最大威胁，而中共国仅仅是其强大竞争对手，更增加了国际社会对民主台湾前途的担忧。另外，拜登给习近平拜年，双方交谈两小时，给外界传递了拜登对中共绥靖的信号。

中共一旦认为攻打台湾不会受到以美国为首的民主阵营的坚决反击时，定会进攻。而台湾一旦沦陷，美国将失去印太盟友尤其是日、印、韩的信任，其海军力量和影响力将被迫从印太地区撒退到第二岛链。没有日印盟友的支持，在太平洋孤独地对抗中共蜂涌而出的军队，这是美国最不愿意看到被动战略局面，而且中共将扼杀印太地区的民主，推行共产主义极权专制。因此，美国政府在台湾防卫问题上的对中共示弱必将演变为对其自身的伤害。

所幸的是，西方阵营中的加拿大、荷兰等国，先后在最高权力机关国会形成了将中共定性犯下反人类罪、种族灭绝罪的法案，这将起到极强的引导及示范效应，更多国家会跟进。另外，欧美主流政界、媒体界、科学界等对中共的巨大威胁已经彻底觉醒，在CCP病毒来源问题上基本达成共识，中共发动超限生物战的认定和追责将不可阻挡地展开。当全球自动灭共、以毒灭共大潮滾滚向前时，台湾也就安全了。
