最近文贵先生直播中多次提到英国女王和贵族。很多中国同胞受共匪洗脑太深，近乎本能地盲目排斥“王”这个概念，但凡涉及到“王”什么的就马上和共匪定义的“封建专制”等扯上关系，其实这是中共魔鬼给华人种下的又一个思维病毒。今天我们一起来看看圣经中以色列人立王的历史，希望受一些启发，摆脱共匪灌输的思维病毒。

以色列人在走出埃及为奴之地后，又在旷野里经过了神四十年的管教，终于进入了应许之地。但随着摩西、约书亚这样的大先知和老一代在旷野里经历过神管教的人的离去，加上当地外族风俗的不良影响和生活安逸带来的信仰上的懈怠……以色列人很快堕落，进入了肉体淫乱不堪，属灵淫乱不堪，对外胆小如鼠，对内血腥残暴，做事不问神的旨意，行事常常充满血气和非理性的士师时代。神在《士师记》中多次在讲完一个士师时代荒唐而真实故事以后在结尾说到：“那时以色列中没有王，各人任意而行”。正因为以色列人心里没有尊神做王，也没有合神心意的世俗的王规范领导他们，所以他们陷入任意而行的罪恶里。同时，我们还可以看到在士师时代先知、大祭司、政治领袖这三个身份在士师身上很多时候是合一的。那是一个政权教权没有分开，也没有十分清晰的边界的时代，以色列人的政体是一种政教合一，近乎混沌的政体。

到了末代士师撒母耳年间，撒母耳在年老的时候让他的儿子们做士师，但他的儿子们却不效法父亲，贪图不义之财，不行公义。随着属灵领袖的腐败堕落，百姓对神的信心动摇了，以色列人开始反抗神权政体，他们不再相信战争的胜负取决于神，他们认为军事的胜利取决于王这种政治强人的领导力和组织能力。于是他们要求撒母耳给他们立世俗意义上的王，好带领以色列人抵抗外族的军事侵略。其实百姓也通过要求立王在表达对神权政体下祭司滥用职权搞腐败的愤怒和不满，他们潜意识里希望通过立王这种革命来削弱士师的权力，通过把政权教权分开，获得一种权力上的平衡，多一个选择，从而得到军事上的安全感和政治上的公义。然而神通过撒母耳的口告诉以色列人，神自己才是以色列人的王，以色列人要求立王是犯了大罪。以色列人若因为士师一时的有限度的腐败而要求立王，等于是要求脱去木轭换铁轭，因为未来他们立的王必加倍地辖制甚至剥削奴役以色列人，到时候以色列人会因立的王而苦不堪言，绝不像以色列人想象的那样美好。但以色列人不听，非要求立王不可，于是神答应了给以色列人立王。

其实神早在以色列人出埃及的路上就知道以色列人进入迦南地以后会想要求立王，所以在进入迦南地之前，神在《申命记》里就专门给以色列人定下了立王的条例。神要求的立王的原则有以下几点：

1. 必须是神拣选，认可的“以色列人”，这里主要是指是受大祭司，先知认可和膏立的人。按当今的标准来说，必须至少是一位虔诚的基督徒。


2. 王必须废除以色列人内部的奴隶制，不可走埃及人的路。

3. 王不可以为自己加添妃嫔，也不可为自己积蓄极多的金钱。以免他的心因为肉体淫乱而受引诱偏离了神的诫命。

4. 王需要把《圣经》(摩西律法书)常带在身边，一生的年日都要诵读，学习敬畏神，谨守遵行神的话。 这样，他的心就不会高傲，轻看自己的兄弟，也不会偏左偏右，背离神的诫命。

5. 神喜欢王位在王遵守祂的法度的情况下，子孙世袭相传。

这可以说是最早的“君主立宪制”，只是这个宪是由神直接制定的。不像当时绝大多数外族人，王的权力和物质享受不受限制，神要求以色列人立的王的权力，应该受到神的公义的法的精神和慈爱的性情所限制。

立王虽然大大得罪神，亏欠了神的荣耀，但神体谅自己子民当时面临的实际环境和软弱，也默许了。所以在以色列人立王以后，神通过撒母耳告诫以色列人“不要惧怕！虽然你们行了这一切恶事，只是不可偏离不跟随耶和华，总要一心事奉他。 不可偏离耶和华，去追随那些虚空的偶像 (撒12:20 )”。就像摩西岳父给摩西提出管理百姓的建议的时候，只要是对自己的子民有实在的益处，神也默许摩西接受，并没有反对。其实换个角度来看，神本来早就知道自己的子民在特定阶段需要什么了。神对自己子民的忍耐和慈爱在立王这事上再次得到了显明。
