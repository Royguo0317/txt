## 庚子年洪水肆虐大地 辛丑年黄沙漫天飞舞
` 喜韩Gnews中文` [轉載自GNews](https://gnews.org/zh-hans/1130058/)

作者：웬샤밍
![]()![](https://gnews.org/wp-content/uploads/2021/04/D06D6F52-C2A2-4EE4-9A47-837DAEC32E9C-scaled.jpg)
最近很多新闻报道台湾日月潭干枯的事情，讲的是由于里面放了一个“九蟾叠像”的风水摆阵影响的台湾龙脉的事情，且不说这个玄妙的事情造成的玄妙的力量影响，但是台湾真的是面临着56年来最严重的干旱了。很难想象一个处在四面是海的宝岛上竟然水源干涸至此。。。

无独有偶，大陆去年10月以来，南方地区降水偏少明显，福建大部、广东东部、云南西部等地干旱。甚至偏少达五成以上。

预计今年5月份，江南南部、华南中东部、西南地区大部等地干旱持续。其中浙江南部、福建、广东、四川西部、云南西部等地降水与同期偏少两至五成；江南、华南和西南大部气温接近常年同期或偏高，华南中东部、云南西部等地干旱持续。

广东气象部门统计，去年10月下旬以来，全省超过七成气象监测站显示出现中等及以上气象乾旱。全省平均降水量为130.3毫米，较往年同期少58%，全省水库蓄水告急；大中型水库的总蓄水量较往年同期相比少逾20%，比往年减少44.8亿立方米。

统计还显示，流经该省的韩江流域则经历60年一遇的降水新低，去年10月以来平均降水量只有85.7毫米，较往年同期减少近七成。韩江流域涉及江西、福建和广东三省，是8市24县（市）近1,100万人口重要水源。

不仅是广东省，浙闽赣湘桂云六省（区）的降水量是1961年以来同期最少，其中福建降水量为224.8毫米，较常年同期少60%。

南方共有53县市连续无降水，其中云南龙陵、永德和福建福安等11县市破历史极值。

江西鄱阳湖水域面积（1044平方公里）于1月15日比同期偏小四成；浙江、福建大中型水库蓄水总量在2月初较同期偏少，出现供水紧张。

同时，南方气温普遍偏高，其中福建省平均气温是1961年以来历史同期最高，广东为第二高。

去年由于洪水泛滥的原因，南方的几大平原的表层丰富的土壤已经被破坏的体无完肤，本来今年农作物的生长就会更加困难，再加上今年的年初的干旱预计情况来看，农作物的种植会受到很大的影响。
