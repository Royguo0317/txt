**中东有石油 中（共）国有稀土**

中（共）国的稀土储量在70年代占全世界可开采资源的70%左右。这么大的储量，原本对所属地百姓是巨大的一笔财富。但是，由于中共的贪污腐败、权贵勾结，很多地方的稀土资源遭到野蛮式地开采。这种开采方式既破坏了当地的生态自然环境，也对当地居住的百姓造成了巨大的化学污染。很多村庄的癌症率突然巨幅上升。

经过几十年的野蛮开采，截止2014年的公开信息显示，中共国的稀土资源占全世界可开采资源比例下降到28%左右，稀土类初级产品供应约占全球85%左右。因为稀土开采和稀土产品的生产加工，会对当地所处自然环境和人类造成巨大污染影响，所以相关生产和后续治理污染的成本是非常高的。而中共国相关企业大部分忽视对自然环境的防护和开采后的治理，导致其稀土产品价格比国际相同产品低很多。这也曾致使美国最大的稀土生产企业前些年的关停。

**稀土牌已成为中共国对外要挟的工具**

稀土被广泛应用于工业、医疗器械、国防军工、航天科技等领域。小到每个人手里拿着的一部手机，里面的震动器材料；大到美军先进的制空飞机F-35的高性能导航雷达；常用到每家大型综合性医院里的核磁共振仪，这些都用到稀土以及稀土衍生产品。中共国以全球稀土产品的绝对供应优势作为要挟西方国家的工具。比如，《人民日报》旗下的《国际金融报》在2020年9月7日刊文，称“有市场声音建议，将稀土出口产品作为反制的手段”。该文记者采访了中共商务部、厦门大学、中信证券等一些官员和学者。文章在最后表述，虽然中共取消了出口配额，但将在生产领域实行一些限制。

**稀土类股票上扬客观反映中共国与美国关系日益恶化**

2021年以来，中共国稀土类公司股价纷纷走出一波上涨行情。比如，该行业排名靠前的北方稀土、五矿稀土、广晟有色等股票，大都上涨了30%到70%。这波反弹，客观反映了中共国在美国大选后并没有达到缓和紧张局势的预期，拜登总统的上台并不能彻底扭转美国与中共的紧张关系。这在资本市场得到了验证。

**拜习电话引起巨大反弹**

农历春节长达两个小时的拜习电话，圆了习的面子，但激起了美国保守派势力和民间的巨大反击。前国务卿蓬佩奥、纳瓦罗等纷纷站出来，反击当前拜登政府对中共的软弱。因此，可以预见，拜登想缓和的动作将受到美国巨大民意的抵制。前些天，拉莫斯民调显示，69%的美国人认为中共国是美国的一个巨大经济威胁。最后，本文作者提醒大家，中共国的假擀面杖经济和虚假股市不值得任何一位明智的人去投资。
