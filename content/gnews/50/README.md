## 【热点播报】拜登在参议院宣布特朗普无罪后称“民主是脆弱的”

作者：纽约香草山 Wilson 校对/发稿：飞虹

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3-1-2.jpg)



路透社2月14日报道，拜登总统在13日参议院未能获得判定特朗普定罪所需的三分之二多数票几个小时后发表的一份声明中说“我们历史上这一悲惨的篇章提醒我们，民主是脆弱的”。他说，每个美国人都有责任捍卫真相。即使是那些反对定罪的人，如参议院少数党领袖麦康奈尔，也认为特朗普犯有“可耻的渎职行为”罪，并应对挑起国会大厦引发的暴力负“实际和道德责任”。

![]()![](https://gnews.org/wp-content/uploads/2021/02/Jietu20210214-211713.jpg)图片来源：路透社

拜登赞扬那些努力保护美国民主完整性的人的勇气，包括民主党人和共和党人、选举官员和法官、民选代表和投票站工作人员。作为美国人，特别是作为领导人，“我们每个人都有责任捍卫真相和挫败谎言”。

拜登最后强调，当前的任务是结束我们国家的分裂，治愈我们国家的灵魂。

中共是颠倒黑色、贼喊捉贼的高手，因此对拜登的以上说话有似曾相识之感。众所周知，造成美国内部分裂的主要原因之一是刚刚过去的美国大选舞弊，如果舞弊得不到惩罚和纠正，美国的宪政民主将礼崩乐坏，加速分裂。

拜登语焉不详地宣称要治愈国家的灵魂，有史以来对美国民众造成最大创伤的当属中共病毒。截至今天，死于中共病毒的美国人已达49万多人，已经大大超过美国百年以来前三个阵亡人数最多的战争(越战、二战、一战)的阵亡人数之和，而且更多感染病毒的美国人在伴着对中共病毒的恐惧和对美国腐败官僚的失望中不断死去。拜登到底如何治愈在美国内部撕裂以及中共病毒笼罩之下的国家灵魂？
