## **天安门广场手表被拍卖行撤拍**

一块奖给参与1989年天安门屠杀的中共国军队的手表，其拥有者受到来自网上的威胁后，Fellows拍卖行取消了拍卖。

[!\[\]()!\[\](https://spark.adobe.com/page/Za8P9r4CjotTW/images/2f55bcff-3f96-4a38-9f16-7b97c4282d58.png?asset_id=d0648d21-59c4-45da-873c-570cbe2f32e5&amp;img_etag=%2236558639f440e903b3eb9c1f2b23370a%22&amp;size=1024)](https://spark.adobe.com/page/Za8P9r4CjotTW/images/2f55bcff-3f96-4a38-9f16-7b97c4282d58.png?asset_id=d0648d21-59c4-45da-873c-570cbe2f32e5&amp;img_etag=%2236558639f440e903b3eb9c1f2b23370a%22&amp;size=1024)手表上写给解放军士兵的文字是：“89.6 平息叛乱纪念。”照片：拍卖行Fellows

英国一家豪华拍卖行已经撤回了一只中共国士兵的稀有手表的拍卖。这只手表是作为对他们在天安门广场大屠杀中的贡献的奖励，此前该拍卖行称，该卖家已受到在线威胁。

待出售的这只手表，描绘了一个绿色头盔的士兵，并用汉字写着“89.6平息叛乱纪念”，这引发了人们对从与暴行相关的物品中获利是否合乎道德的质疑。

作为计划于4月19日发售的豪华钟表大规模拍卖的一部分，英国拍卖公司Fellows一直在单独推销这只手表。但在周四被媒体报道后，Fellows宣布该表已退出拍卖。

该拍卖行说：“在社交媒体上，对手表所有者的威胁使卖方表达了对其安全性的担忧。我们已决定取消这只手表的拍卖。”

一位发言人告诉《卫报》，所有威胁都是通过数字方式收到的，例如，分享了一条回应了一个新闻报道的推文，询问该表是否“是可以接受的证据，对手表的主人就中共国试图掩盖的大屠杀而受到追查和审判？”

士兵在天安门广场和北京周边街道清除学生抗议者中杀死的人数尚不得而知。估计数以千计。中共国政府正式表示，只有不到24人丧生，但它在中国境内禁止这次事件的全部信息和参考资料。

[!\[\]()!\[\](https://spark.adobe.com/page/Za8P9r4CjotTW/images/7e707e80-cb31-4d91-938a-4b626a9c3e19.png?asset_id=cfb4e14f-3100-4955-9dc1-8f2e41ab8312&amp;img_etag=%229e44e94d51611643808ecd1163d9b040%22&amp;size=1024)](https://spark.adobe.com/page/Za8P9r4CjotTW/images/7e707e80-cb31-4d91-938a-4b626a9c3e19.png?asset_id=cfb4e14f-3100-4955-9dc1-8f2e41ab8312&amp;img_etag=%229e44e94d51611643808ecd1163d9b040%22&amp;size=1024)1989年，成千上万的民主抗议者聚集在北京的天安门广场。照片：Sadayuki Mikami / 美联社

拍卖行Fellows说，这只手表来自一个私人收藏，已经“放在橱柜里很多年了”，最初收到它的士兵的名字是未知的。它对手表的出处进行调查后发现：“据我们所知，卖方与解放军（人民解放军）没有任何关系，也与中共国政府无关。”

林慕莲（Louisa Lim）在她的《重返天安门：在失忆的人民共和国，追寻六四的历史真相》一书中说，这只手表是纪念解放军士兵在镇压学生抗议活动中发挥作用的纪念品之一，还有一本名为《首都捍卫者》的照片集，带有相同铭文的金色奖章。

相信只有几百只这样的手表存在，其中包括在耶鲁大学学习中心展出的一只手表。Fellows在较早的博客文章中强调了该物品，说这只手表的价格在2500英镑到3500英镑之间，“很可能是英国仅有的要拍卖的这种手表之一”。

Fellows在后来的声明中说，这只表“是国际上关注的焦点”，过去它出售过和全球各方面冲突有关的物品。它说：“这里不是我们对过去的事件发表评论的地方。对我们而言，曝光历史事件并以尊重和不带偏见的方式进行报道是很重要的。”
