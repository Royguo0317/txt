“我们已经表现出了巨大的抵制，抵制社会主义，抵制醒目的消灭文化，和一些人谈过这个问题。我们已经抵制我们的民主和自由溜走，” 蓬佩奥表示。“这是一场崇高的、有价值的斗争，我们身在其中。我们进行了艰苦的斗争。我们几乎在每条战线上都为之艰苦奋斗，我们将永远为之奋斗。”

蓬佩奥接着谈到了川普政府期间的就业问题，他指出，在这四年里，我们创造的工作机会比找工作的人多。他继续强调自己和政府的过往，谈及海外成就。

“我和总统一起出访那么多次，他会告诉我他在世界各地的同行，他会说，听着，作为美国总统，我将把美国放在第一位，我希望你能把你的国家和人民放在第一位，我们将共同努力，为我们两个国家完成这个任务”，蓬佩奥解释说，“我们做到了。”他补充说，共和党已经忍受了骂名，但尽管如此，他还是为这场斗争而感到自豪。
