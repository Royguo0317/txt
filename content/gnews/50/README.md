一些公司使用 CCP 病毒作为借口降低了工资，增加了工作时间，但没有改善就业福 利。 在中国，大多数员工必须接受工作场所的歧视和骚扰，而没有其他选择，因为他们必须养家糊口。 中共非法政府并没有帮助遭受 CCP 病毒危机的中国人民，而是推迟社会保障金发放和延迟领取退休金年龄。 Covid-19 病毒爆发后，许多公司停止为工人的保险支付费用。 中共在各地、各省和整个国家的部门对于不发社会保险金这个问题都假装没有看到。
