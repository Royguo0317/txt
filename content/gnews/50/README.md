译文：用使民众强大的方法清除其他不服从法令的强民，政权就会削弱。用削弱民众的方法使得民间不可能产生强民，政权就会强大。**国家对老百姓太好，奸诈的民众就多。**国家富但使百姓穷，对政权来说就是富上加富，政权就强。国家穷但百姓富，对政权来说就是穷上加穷，政权就弱。**做敌人不敢做的，国家就强大。做敌人认为耻辱而羞于做的事，就对政权很有利。**

解读：这段文章体现了商鞅反人道、无底线的治国理念。商鞅认为不能使民众强大，不能在民间产生对政权有威胁的力量，这同现代民主国家保护私产、藏富于民、保护民权是南辕北辙的。他认为国家对民众太好民众就会变得奸猾，这同民主政府争着讨好民众是截然对立的，这是商鞅一贯的弱民、贫民和辱民的治国方针，是典型的反人类的思维。**他要做敌人不敢做、或者羞于做的事情，就是突破底线，超出别人的预期，这就是超限战。**也许很多君王是这么想的，也悄悄地在做，但商鞅不仅公开地说还大张旗鼓地做，也算千古奇观。

**在商鞅之前，国与国之间的战争是有规则和礼仪的，叫做“军礼”，这是“周礼”的一部分**。比如打仗要先下战书或者檄文，昭告对方和天下为什么要打，这叫“师出有名”。然后和对方约个时间和地方，一般约在城外或者某个荒郊，不会在居民区打仗不扰民。有些情况是不能打的，比如“不伐丧”，就是对方国家有重大丧事就不能打；“不因凶”，就是对方国家有灾难如水灾旱灾饥荒等，不能打。“逐奔不过百步，纵绥不过三舍”，就是对方士兵被打败逃跑，如果他跑了100步就算认输，你就不能追杀，跑50步都不算认输，所以有“五十步笑百步”之说。如果对方军队主动撤退了三舍（90里）就算认输，你也不能追杀，所以有“退避三舍”之说。对方逃跑的时候遇到困难比如跑错路了或者车坏了，你还得帮他。“君子不重伤”，如果对方已经受伤你就不能再补刀。“不擒二毛”，就是老兵抓了就要放掉。“不鼓不成列”，就是人家没有摆好阵势你是不能攻打的。春秋五霸之一宋襄公和楚国打仗时，楚军渡河而战，宋襄公的大臣说：楚军强大，我们可以等他们渡河到一半的时候出击。宋襄公说我们要讲仁义，怎么能趁人家过河而击呢？等楚军渡过河的时候，大臣说我们现在可以出击了，宋襄公说等他们列好阵再打。等楚军排好阵势一拥而上，宋军大败，宋襄公也被打伤。这就是被后世列为笑话的“宋襄之仁”，但在当时是符合礼仪受人尊敬的君子之为。古人不会趁人之危落井下石，那时的战争也有美感。
