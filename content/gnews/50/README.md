**简评：**

1911年满清王朝被推翻以后，建立了中华民国，国共内战之后，1949年中共在大陆成立了中华人民共和国。从时间顺序上讲，中华民国先于中共国，从法理上讲，中华民国与中共国也不存在替代关系。所以，对台湾而言，宣布独立本身就是伪命题，更不需要得到中共的承认。

大陆军机扰台，目的有两点：第一个是恐吓台湾，警告台湾当局以及民众，大陆有能力、有意图对台开战，你们要做什么选择？第二是刺探以美、日为首的国际社会的底线。

如果美、日等国坚决支持台湾自卫，同时台湾本土也是同仇敌忾、上下一心，这种情况，中共必将有所顾忌。但是，如果台湾当局以及台湾的资本家们学香港几大家族，迫于压力向中共妥协了，那台湾可能真的避免不了被中共“收复”的命运了。

蔡英文此次当选，受益于中共对香港的镇压，香港反送中事件以及中共政府对香港人的血腥镇压，让很多台湾人特别是台湾年青一代从睡梦中惊醒，看清了中共的邪恶本质。但是2020台湾大选，韩国瑜获得了552万选票，得票率为38.6%，这股亲共势力仍然不可小视。再加上国民党这些年与中共政府勾连太深，台湾的商人与中共也是有千丝万缕的联系，所以面对中共的军事压力，台湾内部是否团结值得忧虑。在自由、民主的社会里待久了，容易疲乏，加上大陆这些年搞改革开放，大陆人所受到的隐形控制和辛苦，在外界看来已经不是那么明显，这也在间接影响着台湾人对中共政权的看法。

几年前有一次我坐高铁出游，途中就遇到一位台湾大姐来大陆旅游，这位大姐在火车上对大陆好一顿夸奖。在她看来，大陆什么都好，消费低、环境好、交通便利。我当时问她，你知道大陆人没有选票么？你知道大陆人不能随便说话么？你们台湾人也这样？她的回答是：“那有什么用，我退休下来，也没有多给我多少钱，还不是要工作才能养活自己？”我对台湾的养老制度不是很了解，但是，从这位大姐的言语中可以看到，会有那么一部分人为了追求物质享受，而忽略了自由与民主的重要性，甚至说这部分人还不在少数。

中共国能在世界上如此横行无忌，一方面依赖于其经济、军事实力的增强，另一方面就是它迎合了当今世界的一股潮流，一切向钱看的价值观取向。民主、自由相对金钱的诱惑，越来越不被人重视，特别是对于那些在自由、民主的环境下生活了很久的人们。

中共能不能尽速灭亡，有人力的推动，也有它的天命，多行不义必自毙，天要灭中共，谁也挡不住。但是，台湾的未来最终是要交给台湾人，大陆的未来，最终要交给大陆人。对于台湾人而言，如果选择向邪恶妥协，那香港的现在将是台湾的未来。对于大陆人而言，中共灭亡之后，如果不能痛定思痛，自我反省，即使一时获得了自由，中国未来的走向仍将不可预知。

## 专家警告说，中共国加快了可能入侵台湾的时间表

#### **一位专家警告说，中共政权正在加速其入侵台湾的计划，因为中共国方面正在加快对台军事演习。**

[!\[\]()!\[\](https://spark.adobe.com/page/9cFVDl9uvRzfc/images/6c2e039f-65b0-42dd-9db8-212c34fdfe53.jpg?asset_id=08bfdaf9-003e-49ea-9002-f81f76052208&amp;img_etag=%22e5a5fc1bd4b4bb816c8479a5cf851ebb%22&amp;size=2560)](https://spark.adobe.com/page/9cFVDl9uvRzfc/images/6c2e039f-65b0-42dd-9db8-212c34fdfe53.jpg?asset_id=08bfdaf9-003e-49ea-9002-f81f76052208&amp;img_etag=%22e5a5fc1bd4b4bb816c8479a5cf851ebb%22&amp;size=1024)

据台湾国防部称，3月26日，中共国20架军用飞机——包括4架可携带核弹头轰-6K轰炸机、10架歼-16战斗机、2架运-8反潜战机以及1架空警-500机载预警和控制飞机——进入台湾防空识别区(ADIZ)。这是该部有史以来报告的最大的入侵事件。

台湾的防空识别区位于该岛的领空附近，是入境飞机必须向岛上空中交通管制员确认自己身份的区域。

[!\[\]()!\[\](https://spark.adobe.com/page/9cFVDl9uvRzfc/images/235fc108-c98a-4d07-a309-e7dfac6084b2.jpg?asset_id=860b2457-9cdb-49fc-be84-1a0dc97b9d6d&amp;img_etag=%22ad26febd7f68903f88ddff1060132888%22&amp;size=2560)](https://spark.adobe.com/page/9cFVDl9uvRzfc/images/235fc108-c98a-4d07-a309-e7dfac6084b2.jpg?asset_id=860b2457-9cdb-49fc-be84-1a0dc97b9d6d&amp;img_etag=%22ad26febd7f68903f88ddff1060132888%22&amp;size=1024)

这次入侵升级是自2020年以来北京对台湾显著增加的敌对状态。去年1月再次当选的台湾总统蔡英文采取了强硬立场应对中国共产党构成的威胁，同时台湾深化了与美国的合作，促使中共政权升级了对台战争意图。

中共将台湾视为其领土的一部分，并威胁要发动战争，以收回台湾。这个自治的岛屿实际上是一个事实上独立的国家，拥有自己的民主选举产生的政府、军事、宪法和货币。

台湾的官方名称是中华民国，是在1911年推翻了中国的清朝皇帝后成立的。在国共内战期间，中华民国被中国共产党击败后撤退到台湾，中国共产党于1949年建立了一个名为中华人民共和国的共产主义国家，而台湾逐渐过渡成为一个民主国家。但直到今天，中共政权仍拒绝承认台湾的主权。

去年，中共国空军向台湾防空识别区飞行约380架次，是自1996年以来一年的最高数量。今年到目前为止，中共国军方几乎每天都在向台湾防空识别区派遣飞机。

4月1日，台湾海岸警卫队宣布，中共国一直在位于南中国海北部的台湾东沙岛附近驾驶无人机。台当局表示，不能排除中共国正在使用无人机进行侦察的可能性。

**除了军事行动，中共政权还加强了对该岛的言辞。今年早些时候，中共国国防部一位发言人威胁说，如果台湾宣布独立，中共国将对台湾开战。**

3月31日，鹰派官方媒体《环球时报》主编胡锡进在社交媒体上写道，他想命令壮丁在战争期间炸毁台湾碉堡。

[!\[\]()!\[\](https://spark.adobe.com/page/9cFVDl9uvRzfc/images/58443181-84fa-4f87-a002-14470fd4a32d.png?asset_id=59ad4479-9ddb-44e9-96c6-b44e77016438&amp;img_etag=%225ca950021998e7026485c69004cb1b92%22&amp;size=2560)](https://spark.adobe.com/page/9cFVDl9uvRzfc/images/58443181-84fa-4f87-a002-14470fd4a32d.png?asset_id=59ad4479-9ddb-44e9-96c6-b44e77016438&amp;img_etag=%225ca950021998e7026485c69004cb1b92%22&amp;size=1024)

3月29日，在被一架台湾拦截飞机的飞行员要求离开该领空后，一名不知姓名的中共国飞行员驾驶一架中共国飞机飞越台湾防空识别区，该（中共国）飞行员说：“这都是我们的。”据（台湾）地方媒体报道，他从脸书页面“台湾西南空域”获得了该飞行员的通话记录。

#### 准备入侵

前国防部长办公室网络安全政策、战略和国际事务主任约翰•米尔斯(John Mills)告诉《大纪元时报》，**中共国此次入侵是准备入侵台湾的一系列演练的一部分。**

米尔斯预计，这些演习在未来两年内可能会以大规模演练达到高潮。米尔斯说，考虑到两栖登陆作战的复杂性，以及中共国军队从未在实战中对敌对力量进行过强制登陆，这些演练是必要的。

米尔斯说，对台湾的任何两栖攻击都可能会有大量的中共国民用商船和渔船的参与。

**他认为，入侵可能会在未来三年内发生，这要比美国印度太平洋司令部(INDOPACOM)司令菲利普•戴维森(Philip Davidson)上将在3月初的国会听证会上给出的六年预计要早得多。**

*迈尔斯说：“如果他们在十年之内没有这么做，我认为(中国领导人)习（近平）可能会被免职。我认为即使是六年的时间也在推动它。”他又补充说，习可能会利用攻击台湾来转移内部问题给他带来的压力，比如经济危机。*

被提名接替戴维森担任美国印度太平洋司令部(INDOPACOM)司令的美国海军上将约翰•阿奎利诺在3月份的提名听证会上拒绝支持戴维森的六年预计，表示中共国入侵台湾的威胁“会比大多数人想象的更接近我们”。

前国家安全顾问赫伯特•雷蒙德•麦马斯特(H.R.McMaster)对此表示赞同，他在三月表示，习相信“他在攻击台湾方面有一个短暂的即将关闭的机会窗口”。麦马斯特说，从2022年开始的这段时间是台湾“危险最大”的时期，并指出这恰恰是在2022年北京冬奥会结束后。

米尔斯说，但是现在，中共国军方还没有准备好攻击台湾。然而，问题是，等待入侵的时间越长，台湾在防御方面就会准备得越充分。

***“我们都需要意识到并为（大陆攻台）时间表的提前做好准备。*****”**米尔斯警告说。

米尔斯说，中共国觊觎台湾的野心主要源于它希望掌握台湾的半导体制造能力。台湾是世界上最大的合同芯片制造商台积电(TSMC)的所在地。

**中共国严重依赖外国半导体-微型芯片，为手机和导弹等各种设备提供动力。**据彭博社报道，中共国在2020年进口了价值3800亿美元的芯片，约占其所有进口产品的18%。

在川普政府对中共国公司实施一系列制裁后，中共政权正在努力确保外国半导体安全（供应）。美国的制裁削弱了中共国科技巨头华为的智能手机业务。中共国芯片制造商中芯国际(SMIC)也被列入了贸易黑名单。
