习近平是个有台湾情结的人。他在主政福建期间，积极解决台湾商人办厂事务，听得懂闽南话，很了解台湾人经商的特点。我推测，当时他的心里已经有了“粤港澳大湾区”的雏形，只是这个想法必须得手握权力才能实施。福建是他政治起家的地方，是他熟悉的地方。因此，他从自己所熟悉的台湾着手寻找脱困之法并建立政绩，也是人之常情。

习近平有好的父母。按照沃伦· 巴菲特的理论，他是中了“排卵彩票”的人——生在中共的权势家庭。但他的父亲在中共夺权后长期不掌控军事、情资、经济、海外藏钱这些系统，导致他与这些系统之间有一层天然隔膜。习近平是外臣之子，长期不在核心圈，即使有封疆大吏的经历，但在任内政绩只是平稳而已。他只求稳。

古代幼主登位时，往往靠外戚、娘家人或身边亲随（太监东厂之流）来掌权，但最后也容易被这些人反噬。习用王歧山来把政敌送进监狱，但始终没有自己的诸葛亮。他在福建和浙江培养的队伍，从能力和素质上没有像年羹尧一样杰出的人。王歧山是“鬼子六”，但人家是江派的人。

一个上位者必须要能听实话，甚至是残酷的实话。习近平最大的问题是，实话到不了他的耳朵里。在中（共）国的官场，基层是有真实数字和摸底的，但这些永远到不了最上面。为什么？因为每一级都会修改真相，都会造假，而到了最上面的，是最漂亮的假话。一个人每天听的都是“这边风景独好”，国外水深火热——不管怎样的人天天浸泡在精神茅台里也会得大头症。

当初习近平上台时，是世界沼泽和中（共）国沼泽联合给他的执政牌照，他才有了执政的合法性。沼泽的底线是，任何人不能碰他们的钱。但习近平错估了自己的实力和对方的能耐：他的步伐太大，走得太远了。香港可以作为一个密不透风的猪圈、与世界沟通的缓冲区，但习近平偏不允许有这个缓冲区。香港的自由色彩可以让中共脸上多点胭脂，但中共不要胭脂要金蛋——我看不出硬把香港搞烂对中共的益处。习一动香港，就得罪了世界沼泽，让沼泽损失了钱。按照秘密社会的规则，这就是挡了别人的财路。

习近平再动台湾，就明显告诉沼泽：日本、澳大利亚是我的，更别提什么亚洲、马六甲海道了。台湾在这期间，会很受惊吓，会很动荡，很多政客会出来演一演，但硬骨头的不多，会有很多比美国军机坠落事故更离奇的事出现。但台湾人只有经历了这些，才会认识谁是小丑。

习近平是个体户，没有自己的人马，钱是抢的，思维还留在猪圈里，他没有走出梁家河。
