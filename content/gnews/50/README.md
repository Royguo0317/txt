## 中共超限战灭港计划（四十八）中英联合声明谈判后期
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1143208/)

搜集/撰文：岁月如歌 / 封面合成：文粤
![]()![](https://gnews.org/wp-content/uploads/2021/04/4276.jpg)
英方在香港主权移交最初的计划是按照他们一贯的做法进行谈判的，但是遭到中共强硬反对。 「英国一贯之政策是协助其殖民地成为主权独立之国家。由于英国向中国租借土地的情况复杂，独立并不适用于香港」，（1972年联合国大会以99票对5票，通过了将香港、澳门从殖民地名单上删除的决议）。双方经过几轮的碰头“斗争”后进入一段时间的冷战。柯利达说服戴卓尔夫人给赵紫阳密函，中共一再表达立场。邓小平关于主权问题，「中国在这个问题上没有回旋余地。坦率地讲，主权问题不是一个可以讨论的问题」，最后「英国理解中国对香港主权归属的立场」做了退让。柯利达在这场谈判采取退让政策，使其未满任期便丢乌纱帽，彭定康更戏称这个为“柯利达症”。随后的谈判英方几乎处于被动状态，不但不能改变中共的立场还被中共利用舆论，民情的方式牵着鼻子走。中共在主权问题上的坚持是对的，但不顾民意、手段不磊落，诚信更是缺失。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4271.jpg)被邓小平评为「孤臣孽子」的钟士元(最前) 和提出「罗保动议」的罗保(最右)
中共一面大打民意牌，一面绝对不可以让香港人参与。在这场谈判中，香港人是多余的，跟他们完全没关系，香港人只不过是刚好生活在这块土地上的动物，是附属品，和大陆的动物一样只管在猪圈，牛圈，羊圈⋯⋯里待着就行。香港前行政咨询机关香港行政局局员钟士元和获英女王封男爵的邓连如，于1984年6月23日去北京见邓小平时，邓一面说欢他们迎来京一面说“欢迎你们以个人身分来北京“，他强调这两个香港官员是以个人身分来京，换句话说他们不能参与官方事宜，只能以个人身分”进谏“。中共可以不必理会采纳他们的建议。邓小平还强调这场谈判是一个”两脚凳“，不是”三脚凳“，即主权回归谈判只能是中英双方的事，香港没资格参与。钟士元心里很不爽，但又不敢反对，他形容香港人在这场谈判中就是被包办婚姻。邓连如提议中共加快民主建设也被邓呛回来，说不是叫她来北京教中共如何管理中国的。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4272.jpg)图片来自世代忏悔录
周南回忆录道出缘由，如果让香港市民参与谈判会出现一面倒的情况。民意调查表现出来是80%以上的市民渴望港英政府继续管理香港，若一面倒向英方谈判团，中共尴尬不说最怕出现「以华制华」的局面，这是他们不愿意看到的。许家屯也曾回北京汇报工作的时候向邓小平透露香港民意问题，邓知道香港人心向谁。如果允许香港人参与进来只会加入英方团谈判团，就显得中共没有体恤民情这样也不好。中共本身就不想香港人参与，不想因此受影响。所以中共无论如何都不能让香港人参与这场谈判。另一方面我们看到香港社会上出现的现象却是香港人在游行示威，各个民间团体在用各种方式表达意见，更甚的是赵紫阳用国家总理的名义给香港人公开回信。中共还暗中利诱有影响力的香港人出来支持回归表达“民意”。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4273.jpg)图片来自世代忏悔录
「尽管香港立法局官方文件写明白“香港本身在谈判中并无地位”，但英方仍希望协议签署前举行没有法律约束力的辩论，由此首席非官守议员罗保在3月14日动议。任何有关香港前途的建议，在未达成最后协议之前，必须在本局辩论」（后称%“罗保动议”）英方这样做当然惹中共不爽，指责英方在背后搞小动作，节外生枝。明眼人可以看出英方这样做都是为了什么。

谈判进入第八轮中英双方都换了团长，中方换周南上英方伊文思上。 1984年4月第12轮会谈双方转入讨论过渡期香港安排和有关政权移交事项。同月英外交大臣杰弗瑞•后艾访华时，邓小平跟他说，在过渡时期会有很多事情要做，没有一个机构怎么行？表示可以考虑这个小组可设在香港、伦敦、北京轮流开会，最后来的三个月谈判中英方仍拒绝这个小组成立。谈判入陷入僵局，英方又再次让步会谈方得进行和取得重大突破。 4月20日杰弗瑞•后艾在香港发表声明宣布1997年7月1日以后英国不再拥有香港主权和治权。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4274.jpg)图片来自苹果日报
邓小平在1984年6月22日、23日会见香港工商界和行政局局员钟士元时讲过，「中国政府会解决香港问题所采取的立场、方针、政策是坚定不移的。我们多次讲过，我国政府在一九九七年恢复行使对香港的主权后，香港现行的社会、经济制度不变，法律基本不变，生活方式不变，香港自由港的地位和国际贸易、金融中心的地位也不变，香港可以继续同其他国家和地区保持和发展经济关系。我们还多次讲过，北京除了派军队以外，不向香港特区政府派出干部，这也是不会改变的。我们派军队是为了维护国家的安全，而不是去干预香港的内部事务。我们对香港的政策五十年不变，我们说这个话是算数的。我们的政策是实行『一个国家，两种制度』，具体说，就是在中华人民共和国内，十亿人口的大陆实行社会主义制度，香港、台湾实行资本主义制度。……所谓香港人没有信心，这不是香港人的真正意见。目前中英谈判的内容还没有公布，很多香港人对中央政府的政策不了解，他们一旦真正了解了，是会完全有信心的。……未来香港特区政府的主要成分是爱国者……爱国者的标准是，尊重自己民族，诚心诚意拥护祖国恢复行使对香港的主权，不损害香港的繁荣和稳定。只要具备这些条件，不管他们相信资本主义，还是相信封建主义，甚至相信奴隶主义，都是爱国者。我们不要求他们都赞成中国的社会主义制度，只要求他们爱祖国，爱香港。……」
![]()![](https://gnews.org/wp-content/uploads/2021/04/4275.jpg)图片来自苹果日报
1984年7月31日邓小平会见杰佛瑞·侯艾时谈话《我们非常关注香港的过渡时期》内容有一句话值得注意：「……根据香港和台湾的历史和实际情况，不保证香港和台湾继续实行资本主义制度，就不能保持它们的繁荣和稳定，也不能和平解决祖国统一问题。……」中共重申了立场和承诺。

邓小平会见香港团体的说话相信香港人都知道并且记得而且印象深刻。 「我们对香港的政策50年不变，我们说这话是算数的」。才22年就等不及了。
