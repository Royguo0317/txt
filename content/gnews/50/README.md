## 中共将会如何同拜登政府在阿拉斯加摊牌
` 喜马拉雅德国慕尼黑感恩农场` [轉載自GNews](https://gnews.org/zh-hans/981741/)

撰文作者：铜豌豆

中共国参考消息网站3月15日援引香港《南华早报》3月14日的一篇刊文，表达了对于3月18日和19日即将在美国阿拉斯加举行的中美双方高级别官员会面的[非积极态度](http://column.cankaoxiaoxi.com/2021/0315/2437415.shtml)，称此次拜登上任以来的中美两国首次高层会晤不太可能带来重大外交突破，认为两国关系不会回到小布什和奥巴马当政时的状态。

此次阿拉斯加会晤，与会的[美方](https://thehill.com/policy/defense/542552-top-biden-officials-to-meet-with-chinese-in-alaska)主要人员为美国拜登政府国务卿安东尼·布林肯（Antony Blinken）以及拜登总统的国家安全顾问杰克·苏利文（Jake Sullivan），而[中方](http://news.cnr.cn/native/gd/20210311/t20210311_525433719.shtml)的主要人员为中共国中央政治局委员、外事工作委员会办公室主任杨洁篪以及中共国国务委员兼外长王毅。杨洁篪属于中共上海帮江家在外交部的大员，而王毅属于中共北京帮习家在外交部的大员。二人各为其主，这次非常罕见的同台出场。

此前郭文贵先生爆料指出，江家在中共内斗中败北，江家势力被排挤至新加坡偏安，然而在与拜登政府阿拉斯加会面这个问题上，习江两家的联手出击，可以从一个侧面说明事态的严重性。同时，从中共媒体喉舌对于此次会晤的冷姿态来看，中共自己对于这次会晤会无功而返、甚至掀桌子都是有心理准备的。中共同时派出上海帮和北京帮两个外交大员，无疑是去给拜登政府施压的。而阿拉斯加会谈地点的选择，则表现出拜登一方既厌恶又不得不面对的矛盾态度。

虽然中共手里拿捏着拜登及其家族成员与中共勾兑的黑材料，然而拜登政府上台后不得不平衡包括共和党建制派、保守势力中坚派、以及亲共卖美沼泽地等等多方利益，尤其是中共病毒对于整个美国经济和社会毁灭性的打击——这个无法避开的话题，使得各方力量逐渐走到了一致灭共的道路上来，现在已经到了拜登政府无法回避问题的时刻了。

白宫发言人珍·萨奇（Jen Psaki）上周二[提到](https://www.reuters.com/article/idUSKBN2B32TC)美国国务卿布林肯和总统国家安全顾问苏利文在台湾问题、香港问题上不会让步，新疆种族灭绝问题也将会成为谈判的议题。而苏利文本人在上周五直接[指出](https://thehill.com/homenews/administration/543046-sullivan-says-tariffs-will-not-take-center-stage-in-talks-with-china)，关税以及出口管控不会成为此次阿拉斯加会晤的中心议题。在中共上周四宣布通过修改香港选举制度的决议后，美国国务院发言人耐德·布莱斯（Ned Price）[发言表示](https://www.reuters.com/article/idUSKBN2B32TC)，中共的这一举动是对香港自治以及自由民主制度的直接攻击，对于阿拉斯加的谈判会增添困难。

一方面，中共会在香港和新疆问题上对于美国以及国际社会不屑一顾，从赵立坚3月12日的外交部例行记者会上的[表态](http://new.fmprc.gov.cn/web/fyrbt_673021/jzhsl_673025/t1860605.shtml)，可以清晰的判断出来。另一方面，分析实际情况，香港业已被中共彻底占领，而在新疆的种族灭绝问题上，中共仍然采取矢口否认的态度，依然操纵联合国[人权理事会](https://www.ohchr.org/EN/HRBodies/HRC/Pages/NewsDetail.aspx?NewsID=26754&amp;LangID=E)，依然旁若无人、明目张胆的[撒谎](http://www.chinanews.com/gn/2021/03-11/9429674.shtml)。中共在阿拉斯加会谈中，在香港和新疆问题上，会采取相同的态度。

在中共病毒问题上，中共会矢口否认制造和释放了中共病毒。中共生化武器专家陈薇少将早在2020年2月就[接管](https://www.rfi.fr/cn/%E4%B8%AD%E5%9B%BD/20200208-%E4%B8%AD%E5%9B%BD%E9%A6%96%E5%B8%AD%E7%94%9F%E5%8C%96%E6%AD%A6%E5%99%A8%E4%B8%93%E5%AE%B6%E9%99%88%E8%96%87%E5%B0%91%E5%B0%86%E6%8E%A5%E7%AE%A1%E6%AD%A6%E6%B1%89p4%E7%97%85%E6%AF%92%E5%AE%9E%E9%AA%8C%E5%AE%A4)了武汉P4实验室，想必早已销毁了关键的证据。在南海岛礁问题上，北约已经给中共划定了[红线](https://www.rfi.fr/cn/%E4%B8%AD%E5%9B%BD/20210313-%E7%BE%8E%E5%86%9B%E9%80%80%E5%BD%B9%E5%B0%86%E9%A2%86-%E4%BB%80%E4%B9%88%E6%98%AF%E7%BE%8E%E5%9B%BD%E7%BB%99%E4%B8%AD%E5%9B%BD%E8%AE%BE%E5%AE%9A%E7%9A%84%E7%BA%A2%E7%BA%BF)，想必中共不会蠢到去自找麻烦，因而南海岛礁触发战事的可能性并不大。然而台湾不一样。在台湾问题上，一方面，习近平收复台湾进而统治全世界的皇帝野心不会有任何的收敛；另一方面，台湾的台积电一旦被中共拿下，中共不仅仅可以解决中共国的芯片短缺问题，而且还可以彻底控制美国乃至世界的芯片生产链供应。无论从哪个方面来讲，中共都不会松手台湾，会想尽办法拿下台湾。

阿拉斯加会晤中，中共会不会一边扮黑脸一边扮红脸，一边是大棒，一边是胡萝卜，一边在台湾海峡进行海军舰艇的军事对峙，一边又以大量的经济利益为诱饵，最终使美国拜登政府屈服。中共会不会出卖中共国老百姓的利益喂养华尔街跨国科技巨头作为交换台湾的筹码？而拜登政府会不会为了华尔街跨国科技巨头而出卖美国的国家安全和美国老百姓的利益？一切将在阿拉斯加的会晤中揭晓。
