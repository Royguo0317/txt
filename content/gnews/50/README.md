## 《路德社》精选新闻简讯：神话数据可信吗？以色列卫生部称辉瑞疫苗施打两剂预防感染95.8%有效

出品：《路德社》编辑：《路德社》精选新闻专栏组

2月20日，以色列卫生部表示，施打两剂美国辉瑞（Pfizer）和德国BioNTech 公司研发的2019冠状病毒疾病（COVID-19）疫苗，预防感染新冠病毒95.8%有效。

相关报导声称：以色列推出的新型冠状病毒疫苗接种行动被视为全球最快速计划。根据以色列卫生部最新数据，全国约900万人口当中，自去年12月以来已有425万人施打第1剂辉瑞&BNT疫苗。约288万人则已经按照建议施打完两剂辉瑞疫苗。

以色列卫生部还声称，根据截至2月13日为止的汇整资料，相较于未接种疫苗者，已接种第2剂疫苗者于两周后，预防感染新冠病毒95.8%有效。

此外，预防重症高达99.2%有效，预防死亡则98.9% 有效。

至于施打第2剂疫苗一周后接受检测者，辉瑞疫苗预防感染91.9%有效，预防重症96.4%有效，预防死亡94.5%有效。
