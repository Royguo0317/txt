伦敦报道——中共国的英语新闻卫星频道已在英国停播，英国媒体监管机构裁定它是由中共所控制的。

CGTN（译注：CGTN为“中国全球电视网”英文开头字母的大写，前身为CCTV-9和CCTV新闻，是一个国际性的英语新闻有线电视服务，提供一个六通道的中国全球电视网络，总部设在北京，是中国共产党宣传部控制下的中国国有媒体组织）将因其对香港抗议活动的（歪曲）报道而面临制裁，包括（强迫）电视认罪。（英国）监管机构表示，它还在对CGTN信号覆盖的范围进行其它三项调查。

监管机构通讯办公室表示，这项决定是基于对CGTN的拥有者进行调查的结果，最终发现该频道是由中国共产党所控制的。根据英国法律，政治机构不得持有广播许可证。通讯办公室得出的结论是，拥有CGTN广播许可证的星华传媒有限公司对该频道的节目并不承担编辑责任，它仅是该中文宣传频道的发行商，而不是内容提供商。此外，（星华传媒有限公司）又拒绝将广播许可证转让给名为“中国全球电视网公司”的实体，因为它最终也是由中共控制的。

通讯办公室的一位发言人说：“我们无法批准将许可证转让给中国环球电视网公司的申请，英国《广播法》不允许这样做，因为它也是由中共控制的”，“我们为CGTN提供了机会，让它遵守规定，但它没有这样做”。发言人说：“现在我们认为，撤销CGTN英国的广播许可证是合适的”。

CGTN在一份声明中称，该一裁决是“极端右翼组织和反华势力”的作为。在该裁决作出超过12个小时后，CGTN网站内都搜索不到这一具有里程碑意义的决定。其工作人员被告知，该裁定不会影响该电视台在英国的运营，（它）将继续制作一些在其它市场上播放的内容。

CGTN在推特上表示，因为英国广播公司（BBC）关于中共病毒大流行的“虚假报道”，中共国已对BBC北京分部提出了“严正交涉”，这表明中共国可能正在对英国采取报复行动。

英国广播公司拒绝了这些要求。英国广播公司表示：“我们坚持对中共国的事件进行准确、公正的报道，（我们）完全拒绝这些关于虚假新闻或意识形态偏见的毫无根据的指控。”

曾任香港驻英国领事馆工作人员的西蒙·郑（Simon Cheng）在逃往英国寻求庇护之前曾被拘留并遭受酷刑，他对此裁决表示欢迎。他说：“在我说出真相的一天后，我看到在电视上被迫做出的认罪，CGTN让我承受了巨大的压力”。

郑说，“（中共）中央电视台选择伦敦作为欧洲的地区总部，（因此）监管机构撤销CGTN许可证的决定具有历史意义，很明显，中共极权政权的喉舌在英国不再有立足之地，这将完成一次正义的反击，为我恢复声誉”，“在遭受酷刑并被迫认罪后，我们的坚持以及勇敢站出来反抗恶势力的行为得到了回报，经过漫长的旅途，正义终于得到了伸张”。

该决定在英国议会受到广泛欢迎。曾担任前首相特雷莎·梅幕僚长，并为IPAC（关于中国问题的全球议会联盟）提供建议的尼克·蒂莫西（Nick Timothy）说，CGTN本来就不应该被允许在英国播出。

他说：“共产主义政权正在进行种族灭绝，我们却允许其喉舌在英国播放节目，这非常荒谬”。蒂莫西告诉《悉尼先驱晨报》和《世纪报》：“防止中共国的宣传和误导，是一个非常受欢迎的决定。”

担任下议院数字、文化、媒体和体育委员会主席的保守党议员朱利安·奈特（Julian Knight）表示，该裁决证实了中共是CGTN广播的最终控制者。“CGTN已经违反了广播法规，它在报道香港抗议活动时出现了逼供和不公正的情况”，奈特说，“应该将其视为强有力的警告，表明广播的力量是带有责任和问责的。不达到此标准是不能被容忍的”。

出席该委员会的资深工党议员凯文·布伦南（Kevin Brennan）也赞成该决定。布伦南（Brennan）告诉此本刊记者：“英国的媒体格局因其内在质量和通讯办公室的高标准监督而受到国际赞誉，所以采取这一行动是正确的”。

CGTN是中共国英语新闻的卫星频道，于2016年底启动，并在160多个国家和地区播出。它的总部设在北京，在伦敦、华盛顿和内罗毕设有“制作中心”。它在伦敦西面的奇斯维克（Chiswick）有专门建造的伦敦办事处，也是其服务欧洲的枢纽中心。
