中央社台北3月6日报道，台北智库专家称，中共国的“疫苗外交”在2021年成为其外交政策的核心，可能会阻碍台湾与发展中国家建立关系的努力。

由政府资助的国防与安全研究所(INDSR)助理研究员龚善善表示，“中共是英国和美国以外少数几个生产COVID-19（中共病毒）疫苗的国家之一，肯定会把它作为外交工具。”北京“疫苗外交”的主要目标是通过人道主义援助提升国家形象，并巩固与“一带一路”倡议国家的关系，这将会对台湾的“新南向政策” （New Southbound Policy）产生负面影响。

“新南向政策”是台湾民主进步党主席蔡英文参选2016年总统大选主要政策之一， 旨在减少台湾对中共的依赖。该政策包括积极扩大与东盟、乃至印度、澳大利亚、新西兰经济市场联系，建立经贸、科技、文化、共享资源、人才与市场等交流政策，进而建立“经济共同体意识”。

菲律宾、缅甸、柬埔寨和老挝等包括在“新南向政策”中的一些国家，现在也是北京疫苗外交的目标。这将直接影响到台湾与这些国家建立合作关系的努力。因为接受中共疫苗的国家在与台湾合作时将感到压力，台湾被中共视为中共国的一个省份，中共的强势外交政策将让这些国家重新审视其与台湾的关系。
