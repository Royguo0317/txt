## 维多利亚州对中共国贸易逆差自一带一路协议以来持续增长

撰稿：Gradient Boost

审稿：Jenny 编辑：MG4
![]()![](https://gnews.org/wp-content/uploads/2021/02/维多利亚州.png)
图片来源：Pexels免费照片素材库；作者：Tom Fisk
据澳大利亚世纪报两日前报道，澳大利亚维多利亚州自2018年与中共国签订“一带一路”谅解备忘录以来，其净出口总量已累计达到负420亿澳元，折合人民币约负2100亿元。净出口实际上是由总出口减去总进口得来，净出口为负数则表示进口总量大于出口，呈贸易逆差态势。

根据世纪报提供的进出口数据我们发现，自2014年现任维多利亚州长上任后，维多利亚州的贸易赤字就在不断扩大（除2017年略有收窄），到2020年总贸易逆差已经达到了2014年水平的两倍左右，为大约20亿澳元。

这似乎与当初增加就业并且提升对外贸易表现的预期相违背，要知道贸易逆差的扩大无非三种原因：进口增加、出口减少或者二者兼备——进口的增加会挤压本地相关产业的发展和就业，而出口的减少则意味着本国外贸企业收入可能降低。

维多利亚州在2020年以前对中共国的贸易不论进口还是出口其实都是大约呈增长态势的，只不过出口到中共国的远不如从中共国进口的多，进出口增长幅度也是从中共国的进口占了上风。到了2020年情况则是进一步恶化，中共病毒疫情在全世界爆发，而维多利亚州又是澳大利亚受病毒疫情影响最为严重的地区，这导致维州的对中共国出口显著下滑，跌幅达17.25%。“一带一路”倡议加上中共新冠病毒的双重组合似乎真的将所谓“双赢”变成了中共“赢两次”。

那么澳大利亚其它州——这些州并没有加入所谓“一带一路”——它们的对中共国贸易情况又如何呢？很可惜，由于笔者检索能力有限，未能找到各州对中共国的净出口数据，不过笔者倒是找到了近年澳大利亚全国对中的贸易数据，发现澳大利亚整体的对中共国贸易事实上一直是出口大于进口，并且即使在近年中澳关系紧张的背景下，净出口额度还在不断上升。

换言之，虽然我们不知道澳大利亚其它州的具体贸易情况，但是既然澳大利亚在维州出现贸易逆差的情况下净出口仍然在扩大，就表示其它一些州对中共国有相当大额的贸易顺差，即出口远大于进口。

如果“一带一路”真的能够促进就业，鼓励出口，实现所谓共同繁荣，那么维州在贸易方面应该成为全澳的领头羊。然而我们观察到的情况却完全与之相反，维多利亚州也许不是情况最遭的，但它在对中共国贸易的净出口方面的确是拖了后腿的。或许澳洲对中共国出口的主要资源在维州并不算丰富，从而导致了维州的贸易逆差。但如果引入“一带一路”的最初目的就是希望通过联动中共国市场，促进经济增长就业，缩小进出口间的差距，那么结果却事与愿违依然是不争的事实。
