牛二心里清楚，光凭一把菜刀不可能治罪李福。他本想利用警察镇住李福，没想到李福根本没有屈服，那句冷冷的“走着瞧”一直回响在牛二的脑海里。牛二不得不重新审视李福，心里继续盘算着如何对付这个“刁民”。

其实，李福还是那个李福，还是那个老实人，之所以变得“不那么老实”，是因为已经完全没有退路了。他把所有的积蓄和希望都压在了鱼塘上，鱼塘就是他的全部，就是他的身家性命，他已经站在了悬崖边，没办法退，也无处可退！

一个人只要到了无路可走的时候，骨子里那份原始的野性就被完全激发出来。李福早已经想好了，你不让我过，就和你同归于尽！这才有了怀揣菜刀怒闯村委之事。也正是因为有了这件事，牛二不得不改变了策略，强攻不行就来智取。

李福有个妹妹，前些年嫁给了省城里的人，这在当时农村也算是个新闻。碰巧的是李福的妹夫和牛二的大舅哥有一个共同的朋友，就这样他们相识了。牛二的大舅哥找到李福的妹夫小安，“你把鱼塘租过来吧，钱我出，我们一起搞，利润将来一人一半。”在利益面前小安动摇了，别人投资、自己分钱，这简直是天上掉馅饼的好事啊，一拍即合。

兴奋之余的小安找到了李福，说道：“大哥，我想在你的鱼塘边上盖个房子，把爸妈接来住，老俩口不想在城里住，想到安静的农村来养老。”李福是个大孝子，见不得父母受半点委屈，听到妹夫的一片孝心感动的不要不要的。李福拍着胸脯说：“没问题的，鱼塘边上有空地，盖几间房子有富裕。”“哥，老俩口来了会影响你，要不这样吧，鱼塘划出一半来，这样有房子有鱼池，既能住人又能休闲，你看呢？”听到小安这样说，李福有些犹豫。小安接着说：“没事的哥，租金我付给你。”李福说：“租金是小事，关键是鱼塘里还有那么多鱼。”小安说：“哥，要不这样吧，鱼也都给我，按照市场价给你钱。”“嗯……好吧。”李福想了想还是答应了妹夫的要求。想当初，兴建鱼塘的时候还从妹妹家拿过不少钱。李福是个懂得感恩的人，别人的好他总是记得。李福并没有多收小安一分钱的租金，鱼塘里的鱼也是按远低于市场的价格，半卖半送，一半的鱼塘就到了小安手里。

手续很快就办好了，小安已经是鱼塘的主人了。然后开始拉砖运瓦，准备盖房的木料。由于小安长期不在现场，盖房子的事由李福一手操持，前前后后忙了几个月，小安的半个鱼塘总算有了眉目。

自从上次警局回来后，牛二一直没有找过李福，这让李福感到有些意外。难道牛二觉得理亏了？难道不再惦记自己的鱼塘了？李福百思不得其解，“管他呢，反正鱼塘坚决不让，跟丫死磕！”但是接下来发生的事让李福彻底明白了。

鱼塘原本就是一个鱼塘，小安分走的一半也只是中间画了条线，自家人嘛，何必分的那么清。忽然一天，来了一伙人，开始紧张地忙碌起来，在分界线上建起了一道墙，随着墙的延伸便和相邻的涂料厂接在了一起，这个涂料厂就是牛二大舅哥的，这是全村人皆知的。

李福找到妹夫小安，问清楚了事情的原委。这时小安再也隐瞒不住了，便把和他们之间的约定，以及利润分成的事情和盘托出。“小安啊小安，你怎么能相信他们，他们一分钱也不会给你的！完全是骗人的鬼话！”此时的小安还信誓旦旦地替对方说话，并发誓将分得的利润交给李福。“别做梦了！醒醒吧！”李福几乎是在吼。牛二在村里巧取豪夺惯了，何时让不相干的人得到过好处？李福太了解牛二了。

原来，这是牛二和大舅哥早就商量好的计策，利用李福的妹夫小安把鱼塘骗过来。小安只是牛二利用的工具，诳小安、骗李福，一箭双雕，达到占取鱼塘的目的。牛二的做法就是中共的看家本领之一：骗。假、骗、丑、恶是中共的标签，自始至终贯穿着这个政党，贯穿着这个政府。大到国家政策，小到村长书记，无一不假、无一不骗，充分体现了中共的丑恶面目。中共国实行反人类的计划生育政策时提出的口号：只生一个好，政府来养老。现在新的说法是：养老不能靠政府。将一个骗字演绎了几十年。中共国无论城乡，满大街贴着标语：“民主、自由、法制”，结果是高筑墙、抓人忙，到处的冤假错案。“民主、自由、法制”可以充斥报纸、电视，可以布满大街小巷，但是，要是有人举着一块“民主、自由”的牌子，分分钟，你的归宿就是警察局。可见中共宣传的“民主、自由、法制”只是遮住他们魔鬼般丑陋面孔的面具。

此时的小安也有些后悔，但是，就像许多中国人一样，心里仍然存在着一丝侥幸，幻想着牛二兑现他们的承诺。相信共产党、走进火葬场，这是至理名言。谁信牛二、谁就范二，这是村里的名言。

“你没有和他们签什么协议吧？”李福带着一丝侥幸问小安，但是小安的回答让李福彻底绝望，“签了，他们出钱，半个鱼塘归他们。”李福知道一切为时已晚，半个鱼塘是要不回来了。万幸的是自己手里还有半个鱼塘，自己和家人的生活还算有个着落，这是李福唯一感到欣慰的。

牛二和大舅哥如愿以偿得到了半个鱼塘，花了很小的代价。虽然只是半个鱼塘，但是扩大生产、款待客户的目的已经达到。更让牛二觉得牛的地方是，鱼塘边的房子居然是李福帮助盖起来的。牛二自己都有点佩服自己，为自己的流氓和无耻感到自豪。此刻，李福对牛二有了更深刻的认识，只有你想不到，没有他不敢做。他承认低估了牛二，低估了牛二的邪恶，牛二的丑恶完全继承了中共的衣钵。自此以后，李福对牛二更加警惕了。
