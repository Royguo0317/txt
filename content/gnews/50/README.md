按照中共党史的记载：1946年5月中国共产党第七次全国代表大会在延安举行，这是中共在民主革命时期最重要的一次代表大会，全党在马克思列宁主义、毛泽东思想的基础上达到了空前的团结，为新民主主义革命在全国的胜利奠定了基础。1946年8月毛泽东发表《对日寇的最后一战》的声明；1948年，淮海战役开始，辽沈战役开始、平津战役开始；1949年辽沈、淮海、平津三大战役，央军委和毛泽东制定的战略方针得到完满实现；1949年9月由中国共产党、各民主党派、各人民团体、各地区、人民解放军、各少数民族、国外华侨及其他爱国分子的代表六百六十二名所组成的中国人民政治协商会议第一届全体会议在北平举行；10月中华人民共和国成立。

1945-1949年的国共内战，中共取得了中国的政权。在中共的党史大事记中，这几年写满了各种“指示”、各种“方针”，也记载了很多“值得炫耀”的辉煌战役。“敌人”也变成了国民党“反动派”。然而抛开中共党史中那些大义凛然（卑鄙无耻）、义正词严（臭不要脸）的记载不谈，这几年的真实历史又是如何呢？

日本遭到两枚原子弹打击后的崩溃要比预期快的多。随着日本的投降，国民政府军与中共军队之间展开了一场激烈的竞赛，双方都力争率先到达敌占区接受日本的投降，以获得更多的武器与军事装备。然而，中共在这方面占有明显的地理优势。就在蒋拼尽全力抗击日本侵略的这几年中，中共在华北、华南、华中建立了十八个“根据地”（解放区），控制了一亿左右的人口，并拥有了一百万人正规军和二百万民兵。北平、天津、上海、南京等大城市在毛“农村包围城市”的策略下，成了农村海洋中的孤岛。1945年8月9日，在日本崩溃在即之时，毛马上向中共军队发布“全面进攻的时机已经来临”的命令，林彪随即率领大军突入东北。

蒋介石当然不想让得之不易的胜利果实被中共窃走，他立即要求中共军队不得擅自行动。朱德随即便谴责蒋为“有利于日本侵略者和汉奸”，并命令日军总司令冈村宁次要向中共军队投降。（无耻至极！！！）由于中共军队占据了地理上的优势，不得已蒋请美国出面调停。在美国的调停下，台湾及北纬十六度线以北的日本军队接受了只向国民政府军队投降的命令（东北除外）。而那时的东北已经由苏联派兵突入，一直深入到热河及察哈尔腹地，并将相当数量的日军武器交给了中共。

1945年8月28日，在美国的调停下（及俄爹的命令下），毛泽东极不情愿的飞赴重庆，与蒋介石进行了一次历史性的会谈——“重庆谈判”，而这次谈判就像是国共双方给国际和国内人演的一场大秀。威信正处于顶点的蒋，刚刚赢得了一场近乎于难以取胜的战争。他除了拥有比中共更加强大的军事优势外，还获得了美国的援助和支持，同期还与斯大林签订了一项友好同盟条约。蒋忍受了八年的抗战，加之对以往的中共的阴险及野心有深刻的认识，他是绝对不想和中共分享他胜利的果实的。而毛的野心及其对于中共的军队及“解放区”（根据地）的控制权，在谈判中也绝对没有妥协的余地。最终，两人在10月10日发表了一份最后协定（史称“双十协定”），这次谈判未能产生任何的具体结果。然而这次谈判却给毛一次站在台前展示自己的机会。在重庆期间，毛会见了国内各方的民主派人士，大谈特谈新联合政府的民主与政治协商（回看历史，事实证明了毛这一切就是在欺骗和演戏）。“重庆谈判”之后，国共双方便开始了全面内战的备战。

随着二战的结束，世界的格局也发生了根本性的变化。战胜国各方都有着自己的利益追求，所以在国共内战中我们不得不谈到美国与苏联所起到的作用。

其实在抗日战争全面爆发后的一段时间内，在经济和军事上对中国援助最多的是苏联。因为前文已经有所叙述，中国的抗战能够大大缓解日本在亚洲扩张后对苏联形成的压力。而1939年7月之前，美国一直是日本石油、废铁及汽车零部件的主要供应者。1939年欧洲战争的爆发及1941年日本偷袭珍珠港，极大的改变了国际对华援助的结构。战时的美国总统罗斯福对中国的态度是友善的，随着美国被卷入二战，他不但对中国在经济和军事上给予了许多援助，废除了过去一个世纪西方对华的一切不平等条约，并且说服了斯大林和丘吉尔，让中国成为“四大国”之一（以丘吉尔为首相的英国对国民政府并不友善）。

1945年4月，罗斯福总统去世，他的继任者杜鲁门基本上延续了罗斯福的对华政策，但是在与国民政府的接触中也产生了一些摩擦。面对一触即发的国共内战，出于对自身利益的考虑，美国和苏联都不想卷入其中。1945年底，驻华特使马歇尔抵达中国，在国共两党之间进行调停。1946年1月，国共两党停火，并组织召开了一次政治协商会议。在马歇尔面前，国共两党装模做样的完成了会议并形成了决议（因为两党都不想得罪美国）。就在马歇尔于3月返回美国之后，国共两党便撕破了脸皮，偶尔的地方性的“擦枪走火”很快变成了大规模的战斗，而这一次会议的决议也成了一纸空文。

蒋对美国大选的误判，使国民党走向崩溃。罗斯福去世后，国民政府对美国杜鲁门（民主党）政府感到失望。蒋预测1948年的美国大选，政府会进行更替，于是便竭力结交共和党人，并在舆论与资金上全力支持共和党候选人杜威，杜威也承诺在其当选后会对中国提供大规模的财政及军事援助。而然杜鲁门却在大选中大获全胜，令蒋不知所措。随之而来的便是是美国两度拒绝了国民政府的援助请求。

关键的东北：苏军进入东北后，并没有遵守《雅尔塔协定》与《中苏友好条约》中三个月内完全撤军的承诺。苏军在东北奸淫掳掠，洗劫了东北全境的工业厂矿，并把日本人建设东北（伪满洲国时）的全部设备运回苏联。而蒋没有听从美军事专家的意见，执意要收复东北，把国民党最精锐的五十万部队投入到东北的大城市中。同一时期，中共的军队在苏联的帮助下，已经完全控制了这些大城市以外的农村地区。蒋的这一决定，为日后内战的全面溃败埋下了祸根。

1946年中期以后，蒋介石决定用军事手段彻底解决中共的问题。他并没有相信美国不会对其内战进行援助的警告，他坚信这一次可以轻而易举的消灭中共。直至1947年6月，国民党军队取得了很大的胜利，收复了大片被中共占领的“解放区”（191,000平方公里面积），并把毛和他的中共中央机关赶离了延安。然而这时的中共已经今非昔比，就在蒋拼尽全力去抗日的那几年中，中共在全国各地共建立起18个“根据地”政权。而1946年7月，中共军队的规模已经达到了195万人。虽然国民政府军那时的规模有373万人，但是越来越多的部队担任起守戍的任务，而作战部队也相应的减少了。加上多年的抗战让国民政府的军人们产生了对战争的极度厌倦感，想要彻底消灭中共似乎已经成为不可能的事情了。

1947年下半年，中共对国民党发起了全面进攻，而对国民政府军最严重的打击便发生在东北。利用“农村包围城市”的策略，1948年中林彪带领大军把东北的几座大城市围得密不透风。最终在1948年底中共占领东北，东北战役让蒋损失了四十七万最精锐的部队。（在东北战役中，鲜为人知的是 “长春围困战”。在久攻长春不下的情况下，毛批准了林彪“围城断粮”的办法，切断一切运输线路，对长春进行了150多天的封锁，饿死无辜百姓几十万人。）

在东北的失败，揭开了国民政府全面崩溃的开端。随着淮海、平津战役的打响，中共多年来的宣传、渗透与策反工作发挥了作用，国民政府军整师编制的部队开始叛变。守卫京津的傅作义将军的城防计划也被中共间谍在司令部内偷走，使之所有抵抗的希望全泡汤了，随后傅作义带领二十万国民政府军“投诚”。从1948年9月到1949年1月，不到半年的时间里，国民政府损失掉150万人。国民政府兵败如山倒……

1949年4月，中共的军队渡过长江，占领南京。国民政府逃亡广州（这时的蒋已经下野，代总统为李宗仁）。10月毛在北京宣部中华人民共和国成立，10月13日国民政府从广州逃亡重庆，12月8日逃亡台湾。至此中共完成了对中国大陆的征服，经过了二十八年的奋斗，毛“窃”得了大陆，达到了权力的巅峰。

（另，这里不得不提一下：1946年5月中国共产党第七次全国代表大会在延安举行，这次大会在刘少奇的主持下把“毛泽东思想”写入了党章。刘把毛捧上了神坛，毛彻底掌握了中共中央的权力。刘少奇的这次“为恶魔捧臭脚”，也为其日后的惨死埋下了祸根。）

文章至此，是记录中共发展历史的一个“节点”。从中共的建党到国共内战的胜利，毛带领着中共“窃”得了中华大陆。读过前面文章的读者一定能了解，从刚刚诞生的中共“寄生”在国民党内以求生存，到夺命狂奔的两万五千里“长征”，再到命悬一线之际阴谋制造了“西安事变”求得生机，又到抗日时期的出卖和背叛使其发展壮大，最终在内战中打败了国民党……一路阴险狡诈，出卖民族、烧杀抢掠，无所不用其极的夺取了政权。
