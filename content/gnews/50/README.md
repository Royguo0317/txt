## 【香港要闻】当局逐步设闭路电视于教室 CCTV 成监视教师政治工具
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/981455/)

搜集\编撰：西西

**建制派积极倡议当局强制在课室安装闭路电视(CCTV)，据悉多间中小学更收到教育局来电询问有关闭路电视的事宜，学界担扰当局是否会全面强制学校在课室安装闭路电视。有校长直言担心当局收集资料的用途，是正在部署闭路电视监察教师。**
![]()![](https://gnews.org/wp-content/uploads/2021/03/image001-32.png)
**个人资料私隐专员公署：学校应考虑「必须、合理及与其目的是否相称」**

据《大纪元香港》3月16日讯，个人资料私隐专员公署指出学校应考虑「必须、合理及与其目的是否相称」等因素，同时咨询师生及家长等受影响人士，另外公署指出闭路电视的保安或监察等目的需要和保障私隐取得一个适当的平衡点，并确保收集的资料需要有实际需要。

教协副会长叶建源认为，目前政治气氛下容易令学校感到担忧，希望当局不要给学校压力，并认为若当局出于希望了解学校的防盗系统，则问题不大，但倘若当局是出于政治目的，则对学校老师和同学是不必要的。
![]()![](https://gnews.org/wp-content/uploads/2021/03/image003-19.png)教协副会长叶建源对课室内安装闭路电视亦表示反对，认为对教师和学生的私隐构成影响外，亦会令教师感到受监控及不被信任，直言当局更应投放资源在教育方面。
**副局长：监察的根本问题是提升教师专业能力**

而对于强制学校在课室安装闭路电视的疑问，教育局副局长蔡若莲曾回应表示监察的根本问题是提升教师专业能力，若然学校要安装闭路电视，必须遵守《个人资料（私隐）条例》及《闭路电视监察及使用航拍机指引》，并表示「如果学校ok，是完全没有问题。」

有市民批评，若然在课室安装闭路电视，难免会令学生感到不安全，并认为应该信任教师的专业，不应过分监视师生。
![]()![](https://gnews.org/wp-content/uploads/2021/03/image005-14.png)
**建制派议员建议监察教师言行操守**

据【香港电台网站】报道，立法会辩论《施政报告》致谢议案，有建制派议员建议当局主动监察教师言行操守，亦有议员建议于课室安装闭路电视。

自由党议员张宇人认为，于课室安装闭路电视时，应只集中摄录教师，相信既有助监察教学情况，亦可以在教师受诬蔑时，闭路电视可以还教师清白。而学生因病休假时，也可以观看录影片段，重温课堂。他举例说，目前不少补习社都有录影课堂，又指他于60年代在美国留学时，亦可翻看课堂片段。

商界议员廖长江说，有教师别有用心将政治主张带入校园，支持《施政报告》提出严肃跟进教师失德问题，希望政府主动监察教师言行操守，揪出害群之马，又指坊间有人将大湾区污名化，令人误解，希望当局加强宣传。

**战友观点：**

无论这些建制派的「忠诚废物」如何合理化在课室安装闭路电视(CCTV)，都无法掩盖一个事实：当局此举是出于政治目的，对老师大范围的监听，在香港校园散播一种白色恐怖，让教师有种寒蝉效应，什么都不敢说。

去年一月份港媒报道， 保良局属下幼稚园暨幼儿园老师揭露校方安装具收音功能的闭路电视后，有保良局属下中学学生及家长投诉校方以防止学生考试作弊为为由，在全校24个课室安装共24部摄录镜头，在考试期间摄录。

该校的闭路电视品牌及型号为「IMOU C26EP」（又称LOOC），拥200万像素镜头，具备缩放、收音、扬声器、被动红外线动态侦测（PIR Detection）等功能，每部市面零售价约450元。该品牌为浙江大华旗下子公司，大华去年因涉协助中共于新疆侵犯穆斯林少数民族人权而被美国列入黑名单。港媒《苹果》早前亦发现驻港解放军营的新闭路电视亦有大华商标。
