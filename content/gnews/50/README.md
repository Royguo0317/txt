## 【香港要闻】汇丰CEO称冻结许智峰户口因为要守法 祈耀年自欺欺人英议员不买账 （追踪报道）

**香港汇丰银行冻结前立法会议员许智峰及家人及另一「好邻舍北区教会」的银行帐户，事件引起英国国会关注。英外交事务委员会于英国时间星期二(26日)下午举行网上会议，就许智峰及家人被冻结银行帐户事件展开听证。出席议会下议院外交事务委员会会议的汇丰银行两名高层多次强调情况「合法」，也维持早前发表声明支持【港版国安法】的决定。**
![]()![](https://gnews.org/wp-content/uploads/2021/01/720-0128-aaaaxu-2.jpg)
汇控集团行政总裁祈耀年(Noel Quinn)，与集团首席法规部主管科林贝尔(Colin Bell) 出席会议接受议员质询。在长达90分钟的会议中，祈耀年与科林贝尔多次被不同的议员质疑，冻结帐户的做法是否恰当与是否有违道德，并怀疑汇丰银行帮助中国当局打压香港的民主自由。

不管国会议员的问题措辞如何，祈耀年不断重覆强调，汇丰银行在这次事件的处理上并没有任何的独特之处。他说，汇丰银行如世界上任何一所国际银行，在任何地方经营都必须要遵守当地的法律。祈耀年说：「站在一个银行家的位置上，我不能评价一个政权或执法机构发出法律行动指示的动机或有效性。我在英国不能这样做，我在任何市场也不能这样做，否则我便会犯上刑事罪行。」

不过，他的回答引起了出席会议的英国国会议员不满。其中来自保守党的博尔斯利议员(Bob Seely)指出，合法不等同正确，在道德层面上更不能说站稳了。斯利说：「你所说的你们所做是合法的，我明白了。你有否想到，你所做的事情尽管合法，(但)在关系到渐渐地侵蚀(香港)法治时，并非那般符合道德标准。 」

多位英国国会议员担忧，汇丰银行冻结香港民主派前议员及其家人银行帐户的先例一开，将会有更多香港人受害。

来自工党的克里斯布莱恩特(Chris Bryant)表示，独裁政权只要胡乱添加指责罪行，便可以要求银行配合，封锁账户并达到政治目的。布莱恩特对汇控集团行政总裁祈耀年提出这样的质问。

当被议员问到去年6月汇丰集团亚太区行政总裁王冬胜签名支持【港版国安法】是否政治表态时，祈耀年没有回答是否自相矛盾，只强调包括他本人在内的汇丰高层亲身经历香港自2019年反修例社会运动以来的各种困难，看到街头暴力事件，认为支持【港版国安法】在香港实施恢复社会秩序，使香港重新上回发展轨道是恰当的做法。他更表明，除了王冬胜外，该项签名联署获得290万香港人的支持。
![]()![](https://gnews.org/wp-content/uploads/2021/01/image002-3.png)
**战友观点：**

汇丰控股行政总裁(CEO)祈耀年，在接受英国国会质询的过程中，值得留意的是，汇丰银行在香港作为其中最大银行，客户的存款到底还安不安全？

祈耀年很直接地说出汇丰银行在事件中是没选择的，如果不冻结许智峰户口就会触犯刑事罪行，甚至波及营运牌照。并称是警方要求，而不是因为许智峰的政治取向。换句话说，是香港警察认为许智峰有问题，要求汇丰银行冻结许智峰户口，基于警方要求要配合调查。如果不配合，可能会触犯法令或被吊销牌照。有议员质疑当中有没有中国介入？祈耀年强调只与警方沟通，并没有与中国官员接触。

祈耀年先生在国会的这番言论简直就是自欺欺人，他身为银行总裁，不可能不明白：没有与中国官员直接接触，只与香港警员接触就等于没有中国官员的介入吗？ 【港版国安法】下，国安署权力无限大，有没有中国官员直接或间接的介入，大家心中有数。

正如许智峰所说：「他们愿意作为(政权)打手」，「汇丰这种取态是应该受到制裁的」。

面对一家随意迎合政权的银行，我们有必要慎重考虑把钱存放在汇丰银行到底还安不安全。
