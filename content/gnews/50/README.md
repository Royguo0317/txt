**《美国政府官员以及科学机构伙同其中国同僚掩盖了大量COVID-19起源的证据》**

**在美国谁害怕病毒真相****?**

到2020年1月底，美国政府和美国科学机构内部的一些人越来越担心美国人民可能会了解有关COVID-19起源的真相。也就是说，COVID-19是在美国科学家和美国政府资金的协助、资助下在中国实验室中制造的一种人工病毒。

除了来自中国共产党的直接压力以外，毫无疑问，国际金融利益集团由于担心病毒真相的揭露会令其在中国的投资受损，也对华盛顿特区那些易受影响的人施加了类似的压力。同样感受到威胁的是那些美国科学机构里的科学家们，他们可能会被认为是同谋或被判有罪，甚至令美国人对科学的权威性失去信任。

就像他们在中国的合作伙伴一样，美国的这些政府官员和科学权威们最担心的就是追责。这就是他们掩盖真相最主要的和自私的动机。

**2020年2月3日的会议**，**以及随后掩盖真相的行动**

2020年2月3日在美国国家科学院举行了一次会议，由美国科技政策办公室主任Kelvin Droegemeier，美国卫生与公共服务部高级科学顾问D. Christian Hassell，美国国立卫生研究院国家过敏和传染病研究所所长Anthony Fauci主持。

与会者包括了科技政策办公室，国立卫生研究院和联邦调查局等其他政府工作人员。大多数与会的科学家们要么与中国有冠状病毒研究的长期合作经历，要么后来得到了中国的有关COVID-19起源于自然的理论的有力支持。

根据在会议之前分发的工作计划，已经确定COVID-19大流行是源于自然的或“进化的”事件，需要反驳相反的论点。 也就是说，美国政府这些官员和科学权威们有责任提供连续的证据来支持预定的结果，从而保护中共，国际金融利益集团以及其自身免受潜在的处罚。

正如他们在自己的电子邮件中所披露的那样，参与会议的科学家们所作的讨论主要集中在如何抑制有关COVID-19病毒可能起源于实验室的任何观点。最后在他们远超会议预期的陈述中编造了COVID-19大流行是一种自然发生的从动物到人类的传播，没有人应该受到责怪的结论。

2020年2月6日，墨迹未干，EcoHealth Alliance的Peter Daszak分别与某些参会的科学家们进行了联系，以期在科学期刊上开展公关活动以支持中国的COVID-19是源于自然的理论，并谴责实验室起源观点是阴谋论。

这些电子邮件中的陈述令人惊叹，可以解释为阴谋论。毫不奇怪，没有任何美国政府官员或美国国家科学院成员出现在电子邮件中。更令人惊讶的是，达萨克在电子邮件中包括了王林发，他曾在武汉病毒学研究所工作，并被视为中共的代理人。达扎克在那封电子邮件中表明，即使不是中国发起的，他的公关活动也得到了支持，目前达扎克和其他一些人与中国的持续研究合作正处于危险之中。

2020年3月7日，英国《柳叶刀》杂志刊登了一篇文章，题为“支持中国的科学家，公共卫生专业人员和医学专家抗击COVID-19的声明”，该文章由达扎克以及上述邮件中出现的那些科学家们共同署名。

随后大量的中国宣传和西方媒体文章支持了COVID-19大流行是自然发生的说法。也是从那时起，有报道称科学和医学杂志正在审查反对COVID-19起源于自然的观点。这是一次非常成功的公关活动，至少他们让美国国家情报总监办公室盲目地接受了中共所宣传的科学“常识”。

**假的真不了**

值得注意的是，与绝大多数表明COVID-19病毒是实验室产生的证据相比，从来没有确凿的科学证据支持它的自然起源。然而，依旧有许多人赞同自然起源的叙述，这主要是出于无知，既得利益，政治压力或仅仅是不愿接受丑陋的事实。
