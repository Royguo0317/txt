据路透社柏林2月23日报道，德国内政部在周二公布的一封致立法者的信中表示，自两年前香港爆发民主抗议活动以来，中（共）国在德国的代理人一直试图恐吓居住在德国的香港居民。

信中列举了2019年8月17日在汉堡举行的支持香港活动人士的抗议活动，期间亲中共政府的反示威者对参与者进行拍摄和拍照，以达到恐吓抗议人士的目的。

内政部在信中还指出：”自香港抗议活动开始以来，已经发现中（共）国（政府）在德国越来越多地试图影响公众舆论以有利于中（共）国政府，以及中共针对抗议活动支持者的行动”。信中交代，约720名香港人在德国拥有居住证。

自由党FDP成员延森告诉路透社：”现在是时候让德国政府意识到 中（共）国政府的行动执行者可能对流亡的香港人构成威胁”。”不幸的是，我怀疑我们的安全机构所使用的机制是否足以有效保护那些受影响的人，”她说。

这封寄给议会人权委员会主席吉德·詹森（Gyde Jensen）的信，是应要求提供有关资料，可能会增加对总理默克尔的压力，要求她在人权问题上对中共国采取更坚定的立场。

中（共）国驻柏林大使馆没有立即回复寻求评论的电子邮件。德国内政部发言人表示，他不知道这封信的内容。
