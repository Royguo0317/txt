2021年3月29日，新浪报导:中共国上海市政府政府前党组成员、副市长、市公安局前党委书记、局长龚道安被河北省唐山市人民检察院以受贿罪起诉。

龚道安被指控，利用担任湖北省荆州市公安局副局长，湖北省公安厅刑事侦查总队重案侦查处处长、经济犯罪侦查总队总队长，湖北省咸宁市委常委、政法委书记、市公安局局长、公安部技术侦察局副局长、局长，上海市政府党组成员、副市长、市公安局局长等职务便利，为他人谋取利益，还通过其他公务员为请托人谋利，并收受财物，数额特别巨大，应以受贿罪追究其刑事责任。

现年56岁的龚道安，曾任湖北省公安县公安局刑侦队副队长、派出所长、刑侦队长；2007年12月升任湖北省咸宁市委常委、政法委书记、市公安局党委书记、局长。2010年11月起，龚道安调至北京，先后出任公安部12局（行动技术局）副局长和局长。2017年6月起任上海市公安局党委书记、局长、督察长，上海公安学院院长。2020年8月18日，龚道安接受中央纪委国家监委审查调查，同年9月23日被免职，2021年2月10日被双开。

龚道安成为了中共十九大之后上海市被查的第一位省部级官员。他也是继孙力军、邓恢林之后2020年第三位被调查的公安系统副部级官员。

据悉，龚道安曾任职的公安部行动技术局（公安部十二局）是专门为其它警种提供技术支持的，包括追踪、监听、定位等。龚道安在担任此职务期间，曾是中央政治局委员兼政法委书记孟建柱的。

根据龚道安火箭般的仕途升迁速度，可以看出他是孟建柱的心腹干将，扼守着江派大本营。目前国际社会追责中共种族大屠杀的高压敏感时期，他被起诉预示着中共内部政治斗争并没有因外部的压力得到丝毫缓和，反而愈发残酷和激烈，绞肉机的体制暴露无遗。

1月份助拜登成功当选美国总统后，习近平自认稳操胜券，于是倾尽全力清洗内部其它派系势力。正如郭先生之前爆料所言，此次大清洗首当其冲的必然是政法公安系统。犹记得2月份中共巡视组刚表态公安部肃清周永康、孟宏伟、孙立军余毒不到位，短短一个月时间就拿龚道安祭旗。由此可见王派势力已经败走政法系统，习派趁机利用中纪委这把利刃对江派的公安系统党羽大开杀戒，进而威慑其它势力，使他们不敢轻举妄动。龚道安此前曾担任公安部十二局局长，所以他知晓很多中共内部重要的机密信息，拿下他也意味着习派彻底掌握这部分情报，将严重威胁江家之前暗中的布局。
