# **英国法律专家总结说，有“非常确实的案例”表明中共国正在对维吾尔族穆斯林实行种族灭绝**

- **英国律师小组在审查“充分的证据”后得出结论**
- **他们还认为中共国对维吾尔族犯下了危害人类罪**
- **一封意见书认为习近平主席对中共国的所作所为负有个人责任**
- **据估计，已经有超过一百万的穆斯林被关进集中营**
- **拜登政府仍将中共国在新疆的政策标记为种族灭绝**


一份正式的法律意见认为，有“非常确实的案例”表明，中共国对维吾尔族穆斯林的迫害犯下了种族灭绝罪。

英国的一群主导律师在审查了一系列中共国针对少数民族的国家授权行为的证据（包括防止维吾尔族人出生的措施）后发表了这一声明。

他们还就拘押维吾尔族人和将维吾尔族儿童强行转移到维族社区外来看护的事实，认定这是对维吾尔族人的严重伤害并犯下了“故意伤害”罪。

该信引述了“充分证据”证明中共国通过对维吾尔族穆斯林奴役、监禁、酷刑、强奸、强迫绝育等手段犯下了该罪行。

评估特别指出，中共国国家主席习近平对这些危害人类罪和种族灭绝罪负有个人责任。
