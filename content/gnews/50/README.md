**3.****英国央行考虑数字货币计划**

[英国央行和财政部宣布，他们将成立一个工作组，探索央行数字货币的可能性。其目的是研究创造一种新型数字货币所涉及的风险和机会。它由银行发行，供家庭和企业使用，将与现金和银行存款并存，而不是取代它们。目前还没有决定是否在英国拥有这样一种货币。然而，政府和世行希望就这样做的好处和实用性“与利益相关者广泛接触”。英国央行此前曾表示，它对央行数字货币(CBDC)感兴趣，因为“这是货币和支付领域发生重大变化的时期”。](https://www.bbc.com/news/business-56799726)
