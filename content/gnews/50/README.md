## 【热点播报】2021年东莞春运数据透露出当前中共国的困局

近日，一份来自东莞市2021年春运小组的内部文件显示，东莞市春运前三周（1月28日至2月19日）的客流量与2020年相比下降25%以上。(感谢香草山教育部Williams提供的资料)

![]()![](https://gnews.org/wp-content/uploads/2021/03/image-8.png)材料提供者：Williams

东莞市春运近三年旅客发送量日趋势图进一步展示了，2021年东莞市春运客流量的断崖式下跌。在2020年的春运初期，新冠疫情还未影响返乡客流，日旅客发送量均在1百万以上。而2021年春运日旅客发送量仅在60万左右，同比下降近一半，甚至比2019年都要低得多。节后的客流量也未出现反弹，依然低于2019年的数据。

![]()![](https://gnews.org/wp-content/uploads/2021/03/image-5.png)材料提供者：Willams

春运客流量下降的主要原因之一，是人们对新冠疫情的担心；二是因为经济的变化。长期以来，东莞市的经济支柱主要是加工生产以及外贸出口，劳动密集型加工企业依赖大量的外来劳动力。2021年春运人口的大幅度下降，在一定程度上，揭示了中共经济的巨大危机。

外贸出口是东莞市的经济支柱。在CCP病毒以及中美经贸摩擦的大背景下，2020年东莞市进出口总额同比下降3.8%，受影响较为严重。2021年2月25日，东莞市政府发布一号文《东莞市人民政府关于加快打造新动能 推动高质量发展的若干意见》，意图改变经济困难的局面。东莞市副市长喻丽君在新闻发布会上称，“面对复杂多变的国内外形势，未来东莞的高质量发展离不开新动能的拉动和支撑。”她说，“在加快构建‘双循环’的大背景下迫切需要开拓内销、提高国内市场份额。” 喻丽君这时候提到用“双循环”来应对国内外复杂的形势，不排除中共不得不为“脱钩”做准备。

中共经济长期以来都是依靠劳动密集型加工产业，靠着剥削穷人的劳动力来攫取财富，老百姓已经苦不堪言。中共政策上采取愚民、弱民等策略，其“假骗偷”的思想充斥着整个社会，社会严重缺乏创新。

多个因素导致2021年中共国的进出口贸易存在巨大的风险。当中，包括CCP病毒的持续肆虐，川普政府推进的贸易关税依然存在，针对华为、中海油、中移动等企业的贸易制裁也仍未放松，以及国外企业的供应链从中共国陆续迁出。再加上最近中共被定义的“种族屠杀”：国际社会很可能对中共采取进一步的制裁。2021年，中共国的经济预计会更加雪上加霜。
