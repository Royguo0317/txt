## 逾二百只中国“渔船”聚集在南海争议海域达数星期，菲律宾派战机促撤离
` 喜马拉雅德国慕尼黑感恩农场` [轉載自GNews](https://gnews.org/zh-hans/1035343/)

收集：JAY；编撰：WLQF

据法新社报道，本月较早前，约220只中国渔船被发现聚集在巴拉望岛（Palawan Island）以西约320公里的牛轭礁（Whitsun Reef）一带，引发外交争议。**菲律宾称该礁位于其专属经济区内，而北京则表示，船队是渔船，为了躲避恶劣天气而聚集在该海域。**

![]()![](https://gnews.org/wp-content/uploads/2021/03/截屏2021-03-30-上午12.24.28.png)MAXAR发布的卫星图片显示，中国渔船聚集在牛轭礁周围

**针对该海域的****200****多艘中国所谓“渔船”的聚集行为，菲律宾表示这侵犯了自己主权，并形容这是****“****群聚而至并具有威胁性的存在****”****。**菲律宾在向中国驻菲律宾大使馆提出抗议后，引发了包括美国和澳大利亚在内的数个国家均对这一事件的持续关注。菲律宾认为，这些渔船是由海上民兵操控的，并配备有武器装备，但是北京坚决否认了这一说法。

中国声称对几乎整个南海海域拥有主权，而菲律宾以及越南、台湾、马来西亚、文莱等均声称在部分海域拥有主权。**就中国渔船牛轭礁事件，菲律宾国防部长表示，菲律宾军方正派遣轻型战机飞越这些中国船只进行监控与巡逻，并强烈要求这些渔船立即撤离海区域。**

菲律宾总统罗德里戈-杜特尔特上周向中国大使黄锡联重申，**菲律宾在****2016****年赢得了一个具有里程碑意义的仲裁案件，在中国的竞争性索赔中明确了其主权权利。**

**中国****–****菲律宾南海仲裁案**

2013年，菲律宾向荷兰海牙常设仲裁法院提出南海仲裁案，控告中华人民共和国违反国际法，将南中国海视为自身领土（领海范围）。

2016年7月12日北京时间下午5时，仲裁庭通过常设仲裁法院公布仲裁结果。仲裁庭认定，中国对九段线范围内的资源拥有“历史性权利”的主张，并无法律基础，违反了1982年的《联合国海洋法公约》。此外，仲裁庭亦裁定中方在礼乐滩采集资源侵犯了菲律宾的主权权利，中方对南沙群岛的珊瑚礁生态系统构成永久而不可挽回的伤害，而中方渔民在南海采取严重破坏珊瑚礁生态的方式，大规模捞捕濒危的海龟、珊瑚等，中方皆明知却没有履行停止这类行为的责任，法庭也判决包括由中华民国（仲裁原文称“中国台湾当局”）实际控制的自然面积最大的太平岛在内的所有南沙群岛的海上地物最多只是“礁”而非“岛”，亦即其自然条件不足以支撑及维持常住人口，最多只能产生领海、不能产生专属经济区；天然状态下的黄岩岛、美济礁、仁爱礁和渚碧礁只是“低潮高地”，不能产生领海、专属经济区或界定大陆架，并认为中国大陆在低潮高地建造人工岛的行为侵犯了菲律宾的主权权利。

**美国国务卿布林肯（****Antony Blinken****）****3****月****28****日在其官方推特上发文说：****“****面对中国海上民兵在牛轭礁集结的情况，美国和我们的盟友菲律宾站在一起。我们将始终站在同盟国一边，维护以规则为基础的国际秩序。****”**
