**西班牙2021年2月18日电/西喜社——**据《零对冲》2月18日报道，周三，美国前总统川普打破一个月的沉默，出现在一些有线电视新闻频道上，暗示2024年竞选，可能推出自己的社交媒体平台，并抨击乔-拜登在推出中共病毒疫苗问题上撒谎。

川普在接受Newsmax采访时，谈到拜登周二晚上公然撒谎说没有可用的疫苗。

“我看到他在上任前说没有疫苗，然而他在上任前就打了一针，”川普说。

“所以，要么他没有说实话，要么他精神不正常。要么是这样，要么是那样。”他补充道。

“他在这个问题上被杀了，”川普继续说，”连仇人都说，’好吧，等一下，这个疫苗很早之前就宣布了。”

“他会不会是在开玩笑？因为，坦率地说，这是一个非常愚蠢的说法。”川普考虑了一下。

拜登周二晚上承认，自从搬进白宫后，他每天醒来都会问自己到底在哪里。

在谈话中的其他地方，川普承认，他很怀念当总统的日子，他说：”一切都曾发生得很好……太糟糕了。”

当被问及是否考虑再次参选时，川普回答说：”我还不会说，但我们有巨大的支持。”

“现在说还为时过早……但我看到外面有很多伟大的民意调查，”他补充道。

提到他在保守派中不断上升的人气，川普宣称 “我是唯一一个被弹劾的人，我的数字会上升。我的意思是弄清楚这个问题。”

在提到社交媒体时，川普宣称 “我们不想回到推特”，并补充道。”我明白它变得非常无聊和数百万人离开。”

川普表示，他的团队正在为回归社交媒体寻找不同的选择，正在 “与一些人谈判”，但也暗示他可以建立自己的平台。

“我们正在研究很多不同的事情，”川普说。
