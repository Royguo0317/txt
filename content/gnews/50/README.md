**1****.****金价录得六个月来最佳单周表现****，因就业数据远逊预期**

[金价周五上涨逾1%，创下去年11月以来的最大单周涨幅，此前美国4月份就业增长意外下滑，加速了美元和美债收益率的回落。现货黄金上涨0.84%，至每盎司1,830.41美元，本周迄今上涨3.5%，为2020年11月以来的最佳周度表现。美国黄金期货收于1,831.30美元，上涨0.9%。芝加哥蓝线期货（Blue Line Futures）首席市场策略师菲利普·斯特里布（Phillip Streible）表示，由于“(就业)数据远逊预期”，收益率将下行，美元也下跌，令金价上扬。但斯特里布补充称，金价涨势可能短暂，因下个月就业数据可能显示“大幅”增长。互联网货币交易公司万达（OANDA Corp.）资深市场分析师爱德华·莫亚（Edward Moya）在一份报告中称，金价可能会升向1,857美元，随后触及1,925美元的阻力位。](https://cn.reuters.com/article/global-precious-metal-drv-0508-idCNKBS2CP003)

**2****.****印度暂停批准进口中****共国****制WIFI设备 **

[两名行业消息人士指出，印度已连续数月暂停批准进口中共国WIFI模组，导致戴尔、HP、小米、Oppo、Vivo及联想等大型厂商推迟在印度推出产品。消息人士指出，从中共国进口任何搭载WIFI模组的电子设备成品，包括蓝牙音箱、无线耳机、智能手机、智能手表及笔记型电脑，都遭到延迟。消息人士指出，印度政府的无线规划与协调部(WPC)至少从去年11月以来便暂停批准从中共国进口电子设备成品WIFI模组的申请。这两名消息人士熟知各家厂商游说争取放行的情况。自那以来，来自美国、中共国、韩国企业的80多件申请案都在等待WPC批准，甚至一些成品来自中共国的印度企业的申请也在等待WPC批准。在印度总理莫迪呼吁加大经济自力更生程度之际，印度对中共国进口商品采取了强硬立场。](https://cn.reuters.com/article/exclusive-china-india-ban-import-wifi-05-idCNKBS2CO0J9)
