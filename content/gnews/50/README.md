## F-35A &#8220;闪电II &#8220;战斗机简介

F-35A “闪电II “战斗机 (F-35A Lightning II) 是澳大利亚国防军的第一种第五代空中作战力。它是一种高度先进的多用途、超音速、隐身战斗机，将满足澳大利亚战胜当前和新兴威胁的要求。
![]()![](https://gnews.org/wp-content/uploads/2021/02/16135398541.png)网络截图
F-35A处于空战技术的最前沿。 其先进的传感器和数据融合将使其能够收集更多的信息，并比以往任何时候都能更快地与其他空军飞机、海军舰艇和陆军部队共享。这将大大增强澳大利亚国防军的了解情况能力和作战效能。此外，除了大大增强解情况能力外，F-35A还为其飞行员提供了显著提高的作战杀伤力和生存能力。

澳大利亚已承诺为皇家空军威廉镇基地和皇家空军廷德尔基地的3个作战中队以及皇家空军威廉镇基地的一个训练中队提供72架F-35A飞机。

首架F-35A飞机于2018年开始在澳大利亚服役，当年12月首架飞机运抵。 第一个F-35A中队3号中队将于2021年投入使用。预计全部72架飞机将在2023年全面投入使用。

飞机技术规格特点包括: 不易被监测到; 内部武器和燃料运输; 高级雷达; 光电和红外传感器; 高级语音和数据链路通信; 以及能够使用多种空对地和空对空武器。它是取代老旧的F/A-18A/B “大黄蜂 “的最合适飞机。

F-35A能够在保留隐身性的同时进行超音速飞行，具有非凡的加速性、敏捷性和9G机动性。对此, 助理国防部长Andrew Hastie在2021年2月12日在对他的选民的通讯中称:“澳大利亚人是一个爱好和平的民族，但我们随时准备保护我们的家园、主权和利益”.为此，我们需要硬实力的军事力量, 即进入F-35A “闪电II “战斗机：一种能够进行超音速飞行的先进隐身战斗机。它是世界上最先进的空战技术之一。

如今，澳大利亚的F-35A机队有33架，随时可以部署作战。到2023年将增加到72架。保证澳大利亚的安全还需要一个有能力的主权国防工业。本周，总理参观了澳大利亚BAESA公司的维修仓库。澳大利亚第一架F-35A就是在这里入库维修的。从这里开始，50多家澳大利亚公司正在分享价值27亿美元的合同，这是F-35计划的一部分。
