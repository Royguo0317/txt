## 香草解评：【世事解评】中共问题专家章家敦呼吁：美国必须切断与中共政权的一切联系

![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-2-1.jpg)



[自然新闻](https://www.naturalnews.com/2021-01-27-china-expert-says-us-cut-ties-regime.html#)（Natural News）1月27日报道了美国中共问题专家保守派评论员章家敦（Gordon Chang），1月17日于在线网络研讨会上对一系列现实问题发表的论述。

![]()![](https://gnews.org/wp-content/uploads/2021/01/image-364.png)

他呼吁：美国必须切断与中共政权的一切联系。他认为，切断与中共的关系，并不仅仅限于切断供应链的联系，而是应该进一步扩大到终止技术伙伴关系和文化交流，取消中共媒体的发行，并将中共外交官和记者驱逐回中共。

章家敦先生表示，中共的共产主义形式是“不可改革的”，其领导人“既傲慢又缺乏安全感”。对于中国的作为，联邦和州一级的执法机构和政府都不堪重负。“这个共产主义国家，利用其与美国所有的联系，试图推翻我们的政府。我们不应该允许敌人利用我们制度的开放性来推翻我们的政府形式。”

章家敦先生说：“我们面对的是邪恶。民主社会的人很难理解邪恶，也没有完美的方法来处理它。我们可能会在处理邪恶时搞砸，但我们最好在我们还有机会的时候现在就处理它。”

章家敦先生揭露了中共在以下领域的恶劣行径：

一、中共允许武汉冠状病毒不受控制地传播：

章家敦先生认为，北京利用武汉冠状病毒大流行来破坏美国和其他国家。他称，这次大流行是“中国自己故意放出的疾病”。他引用了中国解放军国防大学的一份出版物，里面讨论了“特定种族基因攻击”，一种新的生物战形式。他警告说：“中国正在设计一些病原体，这些病原体会让中国人独善其身，但会让世界上所有人都生病。有证据表明，他们实际上正在研究这样的疾病，这不是最后一种从中共土壤中产生的病原体。而且我们有理由怀疑，下一种疾病会更严重。”

二、中共利用网络手段削弱了美国：

他提到，中共利用社交媒体平台攻击美国。早在2020年3月，中共外交部发言人赵立坚利用推特发表言论，造谣疫情始于美国。抨击美国疾病控制和预防中心（CDC）涉嫌掩盖武汉冠状病毒病例的实际数量。中共还利用抖音视频分享应用来破坏美国。 在2020年发生的黑命贵抗议活动中，中共使用了一种复杂的算法来让美国人迷上这个应用。利用大数据和人工智能来识别可能会加入抗议活动的用户。然后，这些用户就收到了关于如何暴动的抖音视频，进一步怂恿他们参加示威活动并制造麻烦。这不仅仅是颠覆。这是一种战争行为。

三、中共加大言论力度，为全面冲突做准备：

早在2019年5月，中共的《人民日报》就刊登了一篇文章，宣称要对美国进行“人民战争”。北京的说法是，美国正试图维护其世界霸权的最高地位，并阻止中共夺取这个位置。实际上，美国是在保护目前的国际体系。但是，中共不想生活在当前的国际体系中，它想完全推翻这个体系。

四、中共政权采取了战狼外交：

这个共产党国家曾威胁其较小的邻居台湾，在其领海挑衅东南亚国家，并通过喜马拉雅山脉边界入侵印度。

章家敦先生提出了中共日益挑衅的两个可能原因。首先，中共似乎正在加强军队作为其主要核心的支持。其次，中共本身也被众多问题所困扰，如民生凋零、经济停滞、缺乏世界范围的支持等。他认为，没有经济的繁荣，中共政权仅存的执政合法性基础就是民族主义。这实际上就意味着发动“海外的军事冲突”。

中共是世界的威胁已经成为了全球的共识，现在需要的是全世界团结起来，推翻中共，恢复世界和平的局面。
