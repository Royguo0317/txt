据美国哥伦比亚广播公司（CBS DETROIT）2月16日报道，凯伦-赫德森-萨缪尔斯（Karen Hudson-Samuels）在底特律的广播新闻业生涯长达四十多年。上周二，这位底特律媒体偶像被丈夫发现死在家中，就在接种中共病毒疫苗后的一天，但她的官方死因仍不明。”我们怀疑可能只是中风，但由于疫苗的正常副作用，可能掩盖了这一点。希望很快就能从尸检报告中了解。”她丈夫说。

萨缪尔斯在WGPR-TV担任主播、制片人和新闻总监而声名鹊起后，这位历史学家和记者用她的眼光创造了她一生中最大的成就之一。为了保护全国第一家黑人拥有和经营的电视台WGPR的历史，萨缪尔斯经过不懈的努力，将威廉-V-班克斯广播博物馆变为现实。

这位68岁的老人突然离世，让业界念叨着她的遗产（博物馆），并质疑她为什么要这么快离开。她的丈夫克里夫-萨缪尔斯说, “当这个博物馆进入国家登记册的时候，她非常非常自豪。” 电视主持人兼制片人格雷格-邓莫尔说, “本着凯伦的精神，我希望你们永远知道，时代可能会改变，但凯伦所理解的这两个字有着巨大的力量，那就是感谢。”WWJ记者维琪·托马斯（Vickie Thomas）说，”她是一个美丽的人。（她的离去）对这个社区来说是一个巨大的损失”。

萨缪尔斯的私人丧礼将于2月19日星期五举行，并于上午10点在网上直播。凯伦的家人要求人们不要献花，而是向WGPR-TV历史协会捐款。

这是又一起中共病毒疫苗致死事件。死者的丈夫在突如其来的丧妻之痛中，仍为疫苗辩护，说可能是中风引起妻子的死亡，疫苗的副作用是正常的。不难理解，他们生活在底特律这个数年极左的、政府腐败、管理混乱的城市。维护政府的名义和尊严是被洗脑多年的公民的责任。一个不容忽视的事实是，底特律在上世纪五十年代有两百万人口，现在只剩下约七十万。有追求有能力的人都陆陆续续地迁移到其他的城市发展和生活。

目前人们仍然在积极地排队注射中共病毒疫苗，疫苗的伤害和死亡事件层出不穷，主流媒体或屏蔽，或不报道。民众如果不去关注主流媒体以外的媒体对疫苗真相的报道，就会让疫苗伤害更多的人。半年或一年以后，那些注射了疫苗的人会怎么样呢？笔者非常担忧。
