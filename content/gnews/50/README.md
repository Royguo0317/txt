即使在CCP病毒爆发之前，世界上大多数国家的负债率已经创下历史新高。CCP病毒肆虐全球一年多以来，各国政府的债务率飙升，据国际金融协会报告，仅截至去年9月，全球债务水平已达到272万亿美元，发达国家的总债务与国内生产总值(GDP)之比在今年第三季度跃升至432%，新兴市场债务与GDP之比在第三季达到近250%，其中中共国为335%，这还不包含各级政府漏报瞒报无法统计部分。

如此高的负债率说明这些国家的财政赤字已经失控或破产。巴西总统博索纳罗在今年1月5日称：“巴西破产了，我无能为力”。如果短期内CCP病毒得不到有效控制，各国的财政收入将继续下降，同时失业等社会救助、医疗支出将继续增加，各国央行被迫印钞以解燃眉之急，必然推高通胀打击实业。纵观历史，财政赤字失控和恶性通胀是孪生兄弟，是所有危机之源，包括引发了多次大型战争。

美联储主席鲍威尔在接受媒体采访时称，美国经济处于复苏的启动阶段，但CCP病毒仍然是主要风险，因而经济复苏存在不确定性。鲍威尔透露，美联储将维持稳定的利率水平及购债规模(目前每个月通过购买债券向市场投放货币1200亿美元，一年就是6万亿，是天量货币投放)，以维持当前美国这部消耗大产出少的机器的勉强运行。鲍威尔还表达了对未来通胀的态度，意思是当通胀率超过通常需要控制的2%水平时，在一段时间内，美联储不会釆用加息或减少当前货币投放量的手段来控制通胀，因为在CCP病毒未得到有效扼制之前，经济系统非常脆弱，一有风吹草动，市场就会风腥血雨。

也就是说，美联储和巴西央行一样，知道继续实行低利率及大量投放货币必定引发恶性通胀也别无他法，否则经济将迅速崩溃，政府将陷入停摆状态，社会在失去国家机器管控管控的情况下，将爆发病毒之外的严重次生灾害——这就是中共发动超限生物战要达到的目的。

在接受采访的最后阶段，鲍威尔意味深长地强调:CCP病毒将继续主宰着通胀及经济前景，不要让疫情再次泛滥。随即，众议院议长南希·佩洛西宣称，疫情的控制和我们的经济恢复息息相关，我们必须粉碎CCP病毒。佩洛西言下之意可能是这样的：你们等着，我这就去和中共算帐！
