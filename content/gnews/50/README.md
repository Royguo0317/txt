## 【墙内新闻浅析】天津一教师教育学生过程中有不当言论，被撤销教师资格
` 喜马拉雅意大利罗马达芬奇农场` [轉載自GNews](https://gnews.org/zh-hans/938822/)

**作者: **RU WEI  **校对/发稿人: **Ting Guo

![]()![](https://gnews.org/wp-content/uploads/2021/02/nimg.ws_.126.net_-1.jpeg)图片来源：网易

中新网2月27日电 据天津市津南区教育局微信公众号消息，津南区教育局27日发布通报称，该局对网上流传肖某某在教育学生过程中存在不当言论事件展开调查。经调查，并与有关人员核实，情况属实。肖某某作为教师，严重违背师德师风，造成极其恶劣影响。通报称，经津南区教育局党委研究决定：给予肖某某党内严重警告处分，降低岗位等级，依据《教师资格条例》撤销其教师资格，调离岗位；对学校主要负责人进行问责，给予党内警告处分；在全区教育系统中深入开展师德师风整顿，深刻反思、汲取教训、举一反三、 引以为戒，坚决杜绝类似事件再次发生。

肖老师教育孩子的语音文字整理：“以往送到我班里的学生，全都是家长当官的，要不就是家里条件特别好，事业单位的你妈妈一个月挣多少钱啊？你爸爸一个月挣多少钱啊？别怪我瞧不起你，我告诉你，XX同学的妈妈一年挣的钱都比你妈妈50年挣的多，你们素质是一样的吗？你们能一样吗？你反思一下你们的家长，有多少素质？”

![]()![](https://gnews.org/wp-content/uploads/2021/02/2fe4027d2da747a7a295a2f5e9cee1ab.jpeg)图片来源：搜狐网

肖老师在班级群里的文字回应整理：“@所有人——各位家长，打扰了，据说今天我的音频火了，这音频对我来说不算什么，我教育学生，我知道这件事，第一想到的是这个孩子，老师向你道歉，我给你提供了素材，你录了，你的无心之举让我火了，你不要有任何心理负担，我不会找你，这个你放心，就当你成长过程和我开了一个玩笑，我依旧把你们当成自己的孩子教育，你们不努力，就会不如你的父母，就会阶层下移，望各位家长把这段话读给自己的孩子，让孩子安心，别怕！”

#### **评论：**

**看完这篇新闻，心情还是比较沉重的，类似事件、比这恶劣万倍都不止的“红黄蓝幼儿园”事件在国内都销声匿迹了。有人会说中共国内的教育出问题了，但是我认为，那些说中共国教育出问题的反而会被中共认定是真出问题的。我在网上整理了该老师的一些语音和文字，还有该老师被处罚后，她的老公和老公的姐姐公然到学校做善后的事情，虽然让人感到很气愤，但更多的是可悲。这体现了中共治下教育的扭曲现状，也是中共故意为之。老师教育孩子的话虽然恶劣，但是她说的是实话；孩子录音放到网上，引起了重大反响后才有了后续的一系列措施，就是为了平民愤；该老师的老公到学校威逼利诱校方和家长为了摆平此事，那都是中共国内的常态，太小儿科了，平头小老百姓只能忍气吞声、息事宁人。**

**“教师”是值得尊重的职业，一句最能反映教师职业重要性的话：百年大计，教育为本。教师的职业，从某种意义上讲在左右着人类社会的发展。如果把人看作一个产品，那么这几十亿的产品都要经过学校教育进行初加工，才能流向社会，才能成为组成人类社会的个体。无论是总统，还是科学家，或是劳动者，谁能否定学校教育是其人生成长的基础。但是中共治下的“教师”变了，变成了中共洗脑的工具、变成了奴化老百姓的帮凶、变成了植入阶级制度的始作俑者，其不知自己也是那个受害者。**
