## 【灭共头条】美国务卿布林肯与杨洁篪通话直面中共的“政治红线” 支持港台疆藏

美国国务卿布林肯2月5日与中共外事委员会主任杨洁篪通话，这是拜登政府就任后首度与中方对话，但双方各自对通话内容的声明却大相迳庭。美驻华使馆官方推特上显示：布林肯提到台湾、新疆、西藏与香港等多项令北京紧张的“红线议题”，并表示会追究中共威胁台海等印太区域稳定的责任。中共党媒则呼吁美方秉持合作共赢，重申中共内政不容外部干涉。

国务院发言人普赖斯（Ned Price）公布的通话内容大意为：布林肯国务卿强调美国将为新疆、西藏和香港的人权和民主挺身而出，持续向中国（中共）施压，并要求中方与国际社会一道谴责缅甸的军事政变。他重申美国将与盟友和伙伴站在一起，捍卫我们共同的价值观和利益，追究（中共）威胁台海、印太地区稳定的责任。

根据中共官媒新华网6日报道，杨洁篪表示，中方会坚持中国特色社会主义道路，“中华民族实现伟大复兴是任何人都阻挡不了的”。“美方应当严格恪守一个中国原则和中美三个联合公报。涉及新疆、香港或西藏议题皆属中国内政，不容外部干涉”。他还强调国际社会应当为缅甸问题营造良好外部环境。”

我们看到，发言人普赖斯的公文中并未提及“一个中国原则和中美三个联合公报”，相反却对所有中共预设的“政治红线”一一表达了明确的态度，并在外交层面上公开双方在缅甸政变的分歧。面对布林肯国务卿的强硬，中共通过官媒发表要实现中华民族伟大复兴的宣言，在缅甸事件上揣着明白装糊涂，假装强调一个中国原则和三个联合公报无疑是在自说自话，为自己捡回颜面。
