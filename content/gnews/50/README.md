美国国务卿布林肯（Antony Blinken）2月8日宣布，拜登（Joe Biden）政府承诺美国致力于以民主、人权和平为中心的外交政策，将重新加入联合国人权理事会。

布林肯称，“联合国人权理事会是有缺陷的，需要改革，但离开并不能解决问题。我们在2018年6月的退出对鼓励有意义的变革毫无作用，而具有独裁议程的国家则利用了这一点。在拜登总统的领导下，我们正在重新参与并准备领导。”

布林肯还表示，“当我们与我们的盟友和朋友密切合作时，我们能够在联合国人权理事会上要求人权记录最差的国家承担责任。我们认为，改进安理会的最佳途径是以有原则的方式与安理会及其成员接触。”

川普（Donald Trump）政府用四年时间，彻底认清被中共严重渗透的全球化组织已沦为中共制约文明国家的工具，享受全球化红利的中共日益暴露出对外扩张的野心，时任驻联合国大使曾指责理事会已成为践踏人权者的保护伞，美国改革该组织的努力被俄罗斯、中（共）国等既不自由也不想被审视的国家阻止。

川普政府退出包括联合国人权理事会、世界卫生组织在内的多个全球化组织，并未妨碍美国在全球人权、民主和自由问题上的担当，其中对香港侵犯人权的14位中共人大常委会副委员长实施强有力的制裁、认定中共在新疆犯下反人类和种族灭共的罪行、以及警示世卫组织应确保彻查病毒来源的独立和公平等等，都是前政府认清中共威胁全球的事实下做出的果断行动。

与之相反的是，拜登上任后，新一届政府加快推翻一系列前任政府的行政举措，例如重返巴黎气候协议、世界卫生组织、联合国人权理事会等多个全球化组织。2月6日，拜登政府撤回前政府对胡塞武装的恐怖组织界定，联合国表示欢迎并给予更多的期望。这是否着预示着拜登政府或将撤销对中共反人类和种族灭绝罪的认定？意味对中政策从敌对退回到绥靖甚至是某种程度上的合作？重回全球化组织与中共接触，美国能否起到领导作用，要求中共在人权问题上负责？我们拭目以待。
