## 中共纸价涨不停，上演“洛阳纸贵”2.0？
` 喜联盟编辑部 Himalaya Alliance Editorial Team` [轉載自GNews](https://gnews.org/zh-hans/1058647/)

#### 作者：香草山农场  鹰(文言)

![]()![](https://gnews.org/wp-content/uploads/2021/04/1-14-5.jpg)图片来源于网络
澎湃网4月6日转载中国新闻出版广电报微信公众号消息，2021年造纸业传统的3月淡季，国内纸价的继续上涨引发热议。

2020年12月初，国内部分纸企发布第一波涨价函，其后陕西、山西、江西、河北、浙江等地企业“引领”第二波涨价潮，太阳、晨鸣、华泰等文化纸企提价200元/吨，广西纸企又带动蔗渣浆生活用纸的涨价。21年2月至3月，太阳、晨鸣、华泰等纸业针对双胶纸（3月份报价达7000-8000元/吨）、铜版纸两次提价；3月份晨鸣、博汇、万国纸业的白卡纸均上涨1000元/吨，白卡纸价格从2020年11月的均价6713元/吨拉高至1月份的7700元/吨，3月初主流品牌白卡纸含税参考价上调至10010元/吨（涨幅接近100%）。近期，维达纸业、恒安集团、中顺洁柔等陆续宣布，因原材料价格上涨等因素，自4月1日起部分产品售价上调（其中恒安集团旗下纸巾产品价格上调约10%-20%）。
![]()![](https://gnews.org/wp-content/uploads/2021/04/2-8.jpg)图片来源于网络
1、纸张生产成本中近60%为纸浆成本，而在2020年10月份以后纸浆期货价格暴涨，升至2021年3月份的7652元/吨（涨幅达76%）。市场上纸浆主要为木浆、非木浆和废纸浆，国内的木浆约60%依赖于进口，针叶浆进口量的下降（20年前1-10月进口量下降1.44%，11月下降9.6%）以及国产阔叶浆、化机浆的供应趋紧都加剧了木浆的需求紧张；非木浆方面，鉴于2020年11月竹浆厂家集中检修，以及原料竹片成本上涨带动非木浆上调200-400元/吨；2020年初CCP病毒大爆发造成废纸收购环节停滞，再有11月中共发改委等联合印发的《关于全面禁止进口固体废物有关事项的公告》明确废纸“零进口”，更直接导致废纸价格的暴涨（根据中共生态环境部公布的数据显示，2020年前三批进口废纸许可数量仅321.46万吨，比2019年同期减少约七成）。

2、2020年1月，中共发改委印发《关于进一步加强塑料污染治理的意见》，规定2020年底全国范围内餐饮行业禁用不可降解一次性塑料吸管。“禁塑令”的全面推行造成白卡纸的供需矛盾日益突出，极大的堆高了白卡纸的需求量；国内白卡纸产能前两位的企业并购，市场占比达50%，紧随其后的两家纸企市场份额约30%，白卡纸的生产主体集中度过高也助长“垄断涨价”的风险。纸浆期货价格的上涨、进口木浆数量下降、成本攀高，传导至原纸企业演变成涨价函的“连发”，受此压力影响生活用纸价格也被迫跟涨。
![]()![](https://gnews.org/wp-content/uploads/2021/04/3-3.jpg)图片来源于网络
3、2020年初因CCP病毒大爆发，国内大范围封城、封区、封村，实体经济行业遭到严重打击，虽此后中共制造疫情转好假象诱使民众外出消费、旅游以带动经济内循环，但经济资本的流通紧张以及假擀面杖子经济的虚浮都加剧了崩溃的危机。一方面是中共控制的国有经济大肆利用疫情发国难财，另一方面则是私营、个体户纷纷破产倒闭，中共大搞“公私合营”侵占社会和个人财产，并对民众加紧盘剥，食品短缺、房价攀高、土地限供等诸多因素造成物价有悖经济窘境的异常上涨，而大型纸企为盈利和生存需要势必“纵容”纸价的飞涨。
