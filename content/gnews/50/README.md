消息传出，中共国驻荷兰使馆发言人于2021年2月26日答记者问，声称：“荷兰议会二院有关涉疆动议罔顾事实与常识，蓄意污蔑抹黑中国，粗暴干涉内政。事实胜于雄辩，新疆‘种族灭绝’就是彻头彻尾的谎言……新疆问题根本不是什么人权、民族或宗教问题，而是反恐怖、反分裂和去极端化问题，更是中国内政，外国无权干涉……”

事实果真如此？“种族灭绝”是谎言吗？笔者就我自己接触到的人和事做一介绍。大概三年前，笔者当时还在墙内时，一天晚上出席一个饭局，参加饭局的其中一人刚巧是从新疆回来，就叫张某吧，他是某省武警部队防暴支队的一个领导。由于新疆形势严峻，当地武警部队严重不足，就从全国武警部队调动防暴支队轮流进疆一年，他们叫“轮战”。对自己内部老百姓用得着叫“轮战”吗？明显就是把维族老百姓也当成了敌对势力进行镇压。

席间，菜过三巡、酒过五味，张某颇有些得意地向大家介绍起了他在新疆所闻所见的一些“光辉事迹”。

从这些简单而直接的事实不难看出，中共国对新疆维族人“种族灭绝”难道是谎言吗？共产党从1949年以来，就一直在逐步推出威胁维族人文化和身份认同的政策。比如有的学校禁止用维吾尔族语言教学，工作招聘有的公示明确不用维族人等等。这些措施强迫维族人必须汉化，才能增加在社会上存活几率。

2015年至2018年期间，中共国政府采取强制绝育和堕胎等恶性，让两个最大的维吾尔县人口自然增长率直线下降84%。且自治区政府制定的2020年“计划生育”目标，要把出生率降到接近零的水平。新疆社科院鼓吹：“2010年至2018年，新疆维吾尔族人口增长25%，明显高于汉族人口2%的增幅。”但是维族人的人口急剧下降是从2017年开始的，为什么不把2018年后的有关数据统计进去？
