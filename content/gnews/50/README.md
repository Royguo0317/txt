## 中共的太空垃圾究竟会落在哪里
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1190599/)

撰稿：脐橙君

根据法新社巴黎8日报道，中国发射的长征五号B遥二火箭在不受控制情况下坠落，部分残骸可能会落在地球未知地点。这一中共国制造的太空垃圾的可能坠落地点，这几天牵动着许多国家和人的心。究竟什么时候坠落，会坠落在哪里，是各方关切的问题。如果最后坠落在有人居住的地点，那么这将是太空飞行器在不受控制情况下重返大气层的最严重事件之一，因为面临着可预见的实际人员伤亡的可能性。

最有可能的情况是，太空飞行器没被大气层高温毁损的任何残骸，会落在海洋或无人居住的地区，但仍会造成人员或财产损失。考虑到考量火箭的高度在距离地表150公里到250公里之间，重返大气层非常难以预测。专家表示，因为低层大气更容易发生密度变化的状况，所以无法确切断定什么时候会发生。

截至欧洲时间7日晚间，有专家预测认为火箭会在格林威治标准时间8日21时至9日7时之间重返大气层，但即使是重返大气层前一个小时，不确定性依然很高。

但可以确定的是，长征火箭主体结构正以大约41度的仰角绕着赤道，也就是说任何残骸都将会落在北纬41度和南纬41度之间。这个区域内就包括希腊、西班牙和北非地区，还有美国和中国的大部分地区。这给人类的安全造成了严重的威胁，因此这几天一直成为各方监测和关注的焦点。

这个问题引起了天文界、航空界等多个领域专家的不满，火箭不受控重返大气层是无法容忍的。因为这给世界安全造成了不可预测的威胁。

也正因为大多数消耗性火箭普遍存在这种情况发生的可能，所以美国的马斯克成功实现了火箭回收才让人如此振奋人心。

我们看到，中共国一味地追求“赶超世界”，背后是不顾人类安全和生命的代价付出，对于中共国来说，他们只对“火箭成功发射”欢呼雀跃，却从不正面面对发射以后的一系列不可控的情况如何解决，这种“好大喜功”扎根在中共这个体制下已经70年，从上至下都是只图其表，不究其里，没有底线，这也就是为什么中共可以快速赶超西方其他发达国家，一跃成为全球第二经济体的原因，因为他们可以为了达到一个“数字”目的，牺牲几代人的未来和无所谓多少人的生命。
