她说“大流行病- **PANDEMIA**， 应该被称为**PLANDEMIA**(这个词是她自己造的，多了一个L, 但PLAN 是计划的意思，意味着这位女演员认为这次流行病是有计划的！)

在接受欧洲新闻的采访时，维多利亚说：“不要再谈什么阴谋论了，我们经历了一年的CCP病毒疫情，我们知道谁是死者。之前他们说解除恐惧的唯一办法就是疫苗。但是疫苗已经推广2个月了，它正在直接把人类当作豚鼠进行测试。”

“如果我说的一切都被认为是阴谋论，那就当它是吧，我们已经经历了一年的行动自由的限制，目前因为疫苗死亡的人比没有接种疫苗死亡的人还多。一方面是电视新闻里怎么说，但是，还有另一方面，是在网络上的一些科学论坛里，那里可以学到很多令人震惊并印象深刻的东西”
