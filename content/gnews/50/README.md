## 上议院就英国法院在种族灭绝裁决中的作用击败政府

**贵族代表第二次投票修改贸易法案，对中共国的人权记录采取更强硬的立场**
[!\[\]()!\[\](https://spark.adobe.com/page/m052Pepygt22l/images/dffa23ba-2eec-4a49-bf33-af5fde1386c8.png?asset_id=d222e15a-f80e-4c97-b0e1-5b6339639abb&amp;img_etag=%225a34efbeea1f2ae5a109ed2d3520a07f%22&amp;size=1024)](https://spark.adobe.com/page/m052Pepygt22l/images/dffa23ba-2eec-4a49-bf33-af5fde1386c8.png?asset_id=d222e15a-f80e-4c97-b0e1-5b6339639abb&amp;img_etag=%225a34efbeea1f2ae5a109ed2d3520a07f%22&amp;size=1024)独立贵族代表，奥尔顿勋爵敦促上议院让高等法院发挥作用，以确定一个国家是否在进行种族灭绝。摄影：张健
贵族代表在对待中共国人权记录的问题上给政府造成了严重的挫败，他们第二次投票修改了一项贸易法案，并让英国法院在确定一个国家是否正在实施种族灭绝方面发挥作用。

任何这样的司法裁决都将要求英国审查与北京以及其他被指控犯有种族灭绝罪的政权的任何双边贸易协定，因为北京虐待新疆的维吾尔族穆斯林。

在跨党派的基础上，贵族代表在周二以359票对188票——171票的多数票——投票，坚持认为英国法院应该被交给这个新角色，现在这个问题将在下周回到下议院。大臣们反对这项措施。

上个月，政府在下议院提案时遭受了重大的后座叛乱，当时政府的多数票从80票减至11票，现在它将面临另一个关键的投票。

贵族代表曾在12月首次以126票的多数投票决定让法院在确定种族灭绝方面发挥作用，因此，上议院中反对政府的趋势正在形成。

这些投票意味着越来越多的保守党议员要求英国对中共国在香港的镇压及其对维吾尔人的对待采取更强硬的立场。这个问题还提出了更广泛的道德问题，即如果部长们尝试脱欧后签署无视人权的自由贸易协定，他们将被如何追究责任。
[!\[\]()!\[\](https://spark.adobe.com/page/m052Pepygt22l/images/2e401dc3-c471-4a5b-84ea-5a950d1fa6c0.jpg?asset_id=4773e984-b8bb-47de-925d-17184113541b&amp;img_etag=%22a0c87f58d36b5026323b751536d9342a%22&amp;size=2560)](https://spark.adobe.com/page/m052Pepygt22l/images/2e401dc3-c471-4a5b-84ea-5a950d1fa6c0.jpg?asset_id=4773e984-b8bb-47de-925d-17184113541b&amp;img_etag=%22a0c87f58d36b5026323b751536d9342a%22&amp;size=1024)
独立贵族奥尔顿勋爵敦促上议院让高等法院发挥作用，他说他具体的措施“既不是徒劳的姿态，也不是美德的信号”。他援引一些资深律师的话说，根据《灭绝种族罪公约》，英国法院有权确定是否正在发生灭绝种族罪。

奥尔顿勋爵补充说，国际刑事法院无法对中共国对待维吾尔人的情况做出这样的认定，因为中共国有能力否决任何此类提议。

他已经调整了他的种族灭绝修正案，以满足反对意见，即原来的提案要求政府对任何高级法院的裁定采取行动。

部长们背水一战，一直试图找到一种让步，使种族灭绝问题远离国家法院，认为法官们不想要这样的角色，它可能会削弱三权分立，其裁决可能会造成外交困难。英国法院已经可以认定个人犯有种族灭绝罪。

贸易部长格里姆斯通勋爵(Lord Grimstone)在周二的最后时刻发给贵族代表的信中建议，外交事务特别委员会可以宣判一个国家是否犯下种族灭绝罪，并可以触发下议院辩论。

外交大臣多米尼克·拉布（Dominic Raab）周一与外交事务特别委员会主席汤姆·图根达特（Tom Tugendhat）讨论了这一提议，周二该提议面临委员会的批评，部分原因是其传唤证人的权力有限，而且这一让步并没有以正式修正贸易法案的形式提出。

前保守党内阁部长布伦卡斯特拉勋爵(Lord Blencathra)说，选择一个特别委员会，每周开几个小时的会议，可能比高等法院日复一日地取证更好，这种想法是愚蠢的。他补充说：“无论专案委员会做出什么决定，政府都可以忽略它”。

他说，外交部对种族灭绝的考量在过去十年陷入了困境，并敦促首相亲自审视，意识到该部门对种族灭绝的旧政策已无法持续。

工党贵族和人权大律师肯尼迪夫人说，全世界都在关注英国是否“能够改变法律的生态，使灭绝种族罪在我们当代世界具有严肃的意义”。
