Newsmax3月1日早发文，公开了国会大厦骚乱的一则内幕，该消息由川普总统提供，全文如下：【1】

川普总统说，他在1月6日国会大厦骚乱前几天对国会大厦附近预期的人群规模感到担忧，并要求部署1万名国民警卫队士兵，但他的要求遭到拒绝，因为众议院议长佩洛西和其他领导人不喜欢这种做法。

“我请求过，我说过这次集会将比任何人想象的都要大。” 川普总统周日晚在《福克斯新闻》（Fox News）史蒂夫·希尔顿（Steve Hilton）的《革命》（The Revolution）节目中公开透露了这则消息，这是他在CPAC演讲后播出的一次采访。“大家都说我们会参加集会，是的，我想，这是我以前曾说过的最大的集会，我说过大的集会，几十万人，不止这些，还有几十万人。”

川普总统说，他向国防部提出了需1万名国民警卫队成员，“据我所知，他们把这个数字给了佩洛西掌管的国会大厦的人，我听说他们拒绝了这个数字，因为他们不认为这个数字好看。”

根据国民警卫队网站，“美国总统可以启动国民警卫队参加联邦任务，联邦任务启动的例子包括部署到科索沃和西奈维稳行动的警卫部队，以及部署到中东和其他地区反恐战争的部队。”

骚乱发生后的第二天，川普总统在一段录音视频中说，他“立即部署了国民警卫队和联邦执法部门保护大楼，驱逐了闯入者。”

前白宫办公厅主任马克·梅多斯周日在福克斯新闻玛丽亚·巴蒂罗莫的《周日早间未来》节目中说，川普总统发出“直接命令”，准备好了多达1万名士兵。

同时，据《政治行动报》报道，美国国会历史学会主席兼首席执行官简·坎贝尔称，国会大厦的安全是一个由四名成员组成的国会警察委员会直接负责，该委员会包括众议院军管负责人和参议院军管负责人，他们向这些议会的领导人报告。

在袭击后时任众议院军管负责人的保罗·欧文、参议院军管负责人迈克尔·斯坦格和国会警察局长史蒂文·桑德相继辞职。根据《华盛顿邮报》和《纽约时报》的报道，欧文告诉《太阳报》，他不想1月6日在国会大厦安排驻军，因为“糟糕的光学系统”。

据《福克斯新闻》报道，佩洛西办公室和国防部尚未就川普总统的说法立即回应。

川普总统还告诉希尔顿，他“讨厌”看到国会大厦发生的事情，但否认他在电视上实时观看着事态发展，并将骚乱与去年夏天波特兰和西雅图等城市发生的骚乱进行着比较。（全文完）
