经济话题对于一般大众来说，有点宏观，有点大，或者说有点虚，往往会让人感到扑朔迷离，结论难以看清楚。而商业其实是经济的表现端，很具体，很现实，很接近大众的生产生活。所以从商业看经济，可以小见大、由近观远、从实看虚。我们试图从商业新闻、事件、人物、活动等来理出这个和经济之间的小大、近远、实虚联系，从而分析出经济的真实、原由和走向等；同时知经济助商业，希望有助于大家管窥真相、寻溯根源、判断趋势，进而更好地开展商业实践。

我们结合最近和商业相关的新闻事件，来看看中国国内现在的经济是好是坏，究竟如何。

先看一则经济新闻。2021年1月26日，《21世纪经济报道》：到目前为止，国内已有至少12个省市在地方人大会议上设定了2021年经济目标，这些目标在约6%至10%以上不等。2020年，中国因为新冠疫情而未提出全年经济增长具体目标，最终年度经济增长达2.3%。2020年的防疫管控措施导致当年第一季度经济出现前所未有的萎缩，这将会使同比数据出现失真，导致2021年经济增长率呈现出非常强劲的表现。

这则新闻里说到的经济目标即经济增长率。经济增长率是个经济学术用语，也是一个经济指标。它是末期国民生产总值与基期国民生产总值的比较，国民生产总值即GNP。以末期现行价格计算末期GNP,得出的增长率是名义经济增长率；以不变价格（即基期价格）计算末期GNP，得出的增长率是实际经济增长率。在量度经济增长时，一般都采用实际经济增长率，经济增长率也称经济增长速度，它是反映一定时期经济发展水平变化程度的动态指标，也是反映一个国家或地区经济是否具有活力的基本指标。根据新闻报道，中国省级各地今年经济增长目标区间多在6%-10%，与去年相比，可谓触底反弹，向好趋势。

再看一则国际经济新闻。国际货币基金组织（IMF）1月26日公布了其对世界经济的预测。在最新版的《世界经济展望》中，IMF表示，世界经济在2020年经历了二战以来最严重的衰退、下降3.5%之后，2021年将迎来强劲复苏，增长5.5%。第二大经济体中国将在2021年实现8.1%的增长，而中国经济在2020年疫情影响下仍然实现了2.3%的增长。
![]()![](https://gnews.org/wp-content/uploads/2021/02/Screen-Shot-2021-02-11-at-2.41.40-AM.png)
结合这两则新闻，似乎国内国际的主流媒体认为，中国经济现在趋势向好。但一些社交媒体上对中国经济有相反的判断，而且观点有理有据，比如全媒体财经、推特和油管上的财经真相等。全媒体认为中国经济灾难将一个接一个。财经真相认为中国经济内部已近崩溃，正尽力试图掩盖转移各种冲突矛盾。我们爆料革命坚持唯真不破，郭先生也警告中国老百姓要进入艰难困苦时期了。那到底中国国内经济是怎样的呢？

主流媒体可能被控制，只允许有导向性的声音发出。社交媒体虽然也会遇到这个问题，因为相对分散，总可以有些不同声音。但媒体宣传手法多变，真中有假，假里有真，我们要获得真正认识，就需要甄别分析。经济商评从商业看经济，给大家提供一个角度来对经济问题和现象进行分析甄别。
