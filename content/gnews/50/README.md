路德社评论：

拜登政府商务部长高度赞赏川普政府对中共国钢铝征收关税，挽救了美国钢铝业的就业，这是一个事实。此前，中共生产的钢铝以极低的价格大量倾销美国，这对美国本土钢铝行业造成毁灭性打击，由于钢铝产业是一个国家的基础产业，决定整个产业链的原料供应，川普政府不得不采取增加关税的方式保住本土钢铝行业。拜登政府上台后完成了对川普政府所有政策的审核，他们也意识到川普政府推动的一系列政策挽救了美国经济，拜登政府接下来还将继续抛出川普总统准备好的灭共板砖，这预示着拜登政府正在愈合两党关系，双方已经达成合作协议。

二、美宣布制裁中共七家超级计算机实体

大纪元4月8日报道[\[3\]](//www.epochtimes.com/gb/21/4/8/n12867255.htm)：美国商务部工业和安全局（BIS）周四发表公告说，已将七个中共国超级计算实体添加到“实体清单”中，这些中企正在进行与美国国家安全或外交政策利益背道而驰的活动。商务部列出的七家中共国公司分别是：天津飞腾信息技术有限公司，上海集成电路技术与产业促进中心、深圳市信维微电子有限公司，国家超级计算济南中心、国家超级计算深圳中心、国家超级计算无锡中心以及国家超级计算郑州中心。商务部表示，这些中共国实体参与建造中共军方使用的超级计算机，以及参与中共破坏军事现代化的努力，和/或参与中共大规模杀伤性武器（WMD）计划。

商务部新规则将立即生效，但不适用于已在途中的美国供应商商品。美国官方文件指出：“美国《出口管制条例》（EAR）对名单上的实体施加额外的许可证要求，并限制了对出口、再出口和转让（国内）的大多数许可证豁免的可用性。”该文件还指出，实体的许可审查政策是“拒绝推定”。
![]()![](https://gnews.org/wp-content/uploads/2021/04/超级计算机.jpg)[超级计算机](https://static.dw.com/image/19342131_303.jpg)
路德社评论：

1、美国商务部制裁这七家超级计算机实体的一个核心理由就是它们参与了中共大规模杀伤性武器（WMD）计划，大规模杀伤性武器包括化学武器、生物武器、放射性武器和核子武器[\[4\]](https://zh.wikipedia.org/wiki/%E5%A4%A7%E8%A7%84%E6%A8%A1%E6%9D%80%E4%BC%A4%E6%80%A7%E6%AD%A6%E5%99%A8)。全世界都知道中共拥有核武器，同时联合国有《核不扩散条约》来约束中共，所以该文件并非针对中共国核武器研发项目。制造化学武器和反射性武器不需要超级计算机进行大规模运算，由此推断，美国商务部此处提及的大规模杀伤性武器指的就是生物武器，而联合国命令禁止所有国家研发制造生物武器。

2、中共在制造生物武器时，需要用超级计算机计算病毒基因序列，预判不同排列组合的基因序列对病毒传染性、致病性和致死性的影响。这也预示着美国军方已经找到制造中共病毒的源头之一，发现这些公司利用超级计算机参与研发改造中共病毒，并且发现这些公司的核心技术都来自美国，所以商务部必须要对这些实体实施制裁。

3、按照美国《出口管制条例》规定，只要美国技术含量占到10%以上的产品都被禁止出口给实体清单中的企业和机构，美国商务部在2019年也是这样制裁华为公司，导致华为手机的全球市场占有率急剧下滑，该禁令涵盖软件、硬件、服务，以及后期维护等所有项目，这对中共国的超级计算机是一个致命打击。

4、超级计算机是一个国家核心竞争力和综合国力的象征，中共经常以此自夸。超级计算机因拥有超级运算能力，它可被用来模拟军事打击，模拟生物攻击，寻找抑制病毒的药物。中共超级计算机所使用的芯片、CPU等核心科技都不是中共自主研发的，全靠进口，这意味着中共国超级计算机接下来的升级换代将遭受致命打击，这些计算机的运算速度将很快落后于发达国家，按照摩尔定律[\[5\]](https://zh.wikipedia.org/wiki/%E6%91%A9%E5%B0%94%E5%AE%9A%E5%BE%8B)，超级计算机运算能力18个月要提升一倍，这是指数级的变化。

5、川普政府在2019年就对中共国五家关键的超级计算机公司和机构实施出口管制，拜登政府这次直接对中共国七家公司和机构实施制裁，这就是川普总统为拜登准备好的板砖，川普政府早已经有确凿的证据证实这七家实体参与中共军民融合项目，现在由拜登政府采取实际行动。

6、川普政府对中芯国际、曙光科技等中共国企业实施制裁时，制裁令只提这些公司参与中共军民融合项目，它们生产出的产品被用于军事领域，并参与侵害人权行动，并未提及大规模杀伤性武器。拜登政府此次发布的制裁令加入了大规模杀伤性武器，这是第一次将该词用于中共，民主党现在的措辞越来越严厉。

7、小布什政府当年因伊拉克从事大规模杀伤性武器研发对该国实施军事打击，最终推翻了萨达姆政权，时任参议院外交委员会主席的拜登就为小布什此次军事打击背书，小布什与拜登背后都有军火商利益集团的支持。

三、Sellin博士首提舟山蝙蝠病毒就是中共病毒骨架

Sellin博士4月8日再发重磅推：中共国武汉病毒学研究所研究员危宏平，他是中共病毒“零号感染者”黄燕玲的指导老师，并与中共军方有联系，其中包括第三军医大学提供的中共病毒骨架：ZC45或ZXC21病毒。

路德社评论：

1、Sellin博士首次在推特中揭露了零号感染者黄燕玲和中共病毒骨架舟山蝙蝠病毒，他此前提及美国成立了类似二战时期的SIXTA组织在全面分析中共病毒的所有情报信息，他陆续发出的重磅推应该就是该组织的分析结果。

2、Sellin博士提及ZC45或ZXC21病毒是中共病毒的骨架，这就进入到中共病毒的本质性问题，这就认定病毒来自中共军方实验室的功能增强型实验，来自军民融合项目。闫博士在去年“119节目”爆料来自中共军方的ZC45或ZXC21病毒就是中共病毒的模板，Sellin承认它就相当于承认中共病毒就是生物武器，承认闫博士的三篇论文。
