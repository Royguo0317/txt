北美Life Site网站在12月8日于华盛顿撰写了有关中共病毒疫苗副作用的文章。

文章报道称，**美国食品药品监督管理局（FDA）在10月份编制的幻灯片演示中包含了中共病毒疫苗可能的副作用的广泛列表。其中包括：死亡，心脏病发作，血液疾病，中风，脑炎，自身免疫性疾病，先天缺陷和川崎病等潜在副作用**。

FDA称将对这些副作用与施用中共病毒疫苗同时进行被动和主动监测。在其原有系统下，FDA将与疾病控制中心（CDC）合作管理疫苗不良事件报告系统（VAERS），这个系统可以让个人向其医疗保健机构报告不良副作用。目前FDA计划与众多合作伙伴一起使用“生物有效性和安全性（BEST）系统”进行疫苗的主动监测。MarketScan是庞大数量的合作伙伴中最大的公司，它拥有超过2.5亿患者。

FDA连同医疗保险和医疗补助服务中心（CMS）一起发表了声明，称其数据可以涵盖“大约5500万年龄在65岁以上的美国老年受益人”。 FDA计划通过使用医疗保险和医疗补助服务中心的数据，基于快速周期分析来监测中共病毒疫苗的副作用。

**FDA承认中共病毒疫苗可能产生的副作用的清单很长。在其列出的清单中有22个单独的条目显示 “可能的不良事件结果”**。

**在FDA名单上排名第一的是“吉兰-巴雷综合症”，被描述为“一种罕见的疾病，这种疾病会让自身的免疫系统攻击你的神经。”该综合征“尚无治愈方法”，死亡率为“4％至7％”。相反，约翰·霍普金斯大学（John Hopkins University）估计，据报道，目前在美国死于中共病毒的病例死亡比例仅为1.9％。**

**在FDA名单上排名第二的是急性弥漫性脑脊髓炎。它是一种 “影响大脑和脊髓的罕见炎症性疾病”。排名第三的是“横断性脊髓炎”，这是一种神经系统疾病，会引起脊髓发炎，引起“疼痛，肌肉无力，瘫痪，感觉障碍或膀胱和肠功能障碍”。**

**“过敏反应”也被列为潜在疫苗可能的结果，这是一种严重的过敏反应，可导致过敏性休克。**

**“中风”和“抽搐/癫痫发作”也是可能的副作用，还有“急性心肌梗塞”或心脏病发作，心脏周围肌肉发炎甚至死亡。**

**英国政府对辉瑞疫苗的警告说，孕妇或哺乳期的母亲和儿童不应使用这种疫苗。其政府文件补充说，目前尚不清楚该疫苗将对生育产生什么影响。应建议育龄妇女在接种第二剂疫苗后至少2个月内避免怀孕。**

**FDA还建议，接种疫苗后可能会出现川崎病。该疾病“主要影响5岁以下的儿童”，并且“必须在医院接受治疗”**。

**儿童也可能受到中共病毒疫苗的影响，出现 “多系统炎症综合征”（MIS-C），美国疾病控制中心（CDC）指出这与中共病毒有关。但是，梅奥诊所（Mayo Clinic）提供了更多细节，并解释说，虽然大多数患有中共病毒的儿童 “只有轻微的疾病”，但那些患有”多系统炎症综合征”的儿童受到的影响则更为严重，器官和组织“例如心脏，肺，血管，肾脏，消化系统，脑，皮肤或眼睛”会严重发炎**。

**目前，FDA对中共病毒疫苗可能产生的副作用的完整清单是：“吉兰-巴雷综合征”；急性播散性脑脊髓炎；横向脊髓炎；脑炎/脊髓炎/脑脊髓炎/脑膜脑炎/脑膜炎/脑病抽搐/癫痫发作；中风; 发作性睡病和瘫痪；过敏反应急性心肌梗塞; 心肌炎/心包炎；自身免疫性疾病；死亡; 怀孕和分娩后遗症；其他急性脱髓鞘疾病；非过敏性过敏反应；血小板减少症弥散性血管内凝血; 静脉血栓栓塞；关节炎和关节痛/关节痛；川崎病儿童多系统炎症综合症；疫苗增强性疾病。”**

面对这些令人惊恐的中共病毒疫苗副作用，人们开始注意到疫苗生产者的法律豁免权问题。**根据1980年代鲜为人知的美国联邦法律，当制药公司的疫苗伤害或杀死某人时，受害方不能在法庭上起诉制药公司。取而代之的是，被疫苗伤害或杀死的人（或其亲属）必须使用1986年制定的国家疫苗伤害赔偿计划。**该计划是在针对疫苗公司和医疗保健提供者的诉讼中提出的，这些诉讼有可能导致疫苗短缺并降低美国疫苗接种率，而该计划旨在保护制药公司免受与疫苗伤害或死亡有关的诉讼。

但儿童健康防卫组织发布的报告说：**“根据受疫苗伤害的人及其亲人的说法，该计划失败了。因为它是一个诉讼性的，破碎的系统。受害人要面对政府疫苗计划，政府拥有的疫苗专利，管理疫苗项目的政府卫生官员以及司法部的律师。没有法官，没有陪审团，也没有诉状。索赔人感到该计划从一开始就为他们的索赔失败做好了准备。”**

**FDA在其文件的最后指出，它尚未完全了解中共病毒疫苗的副作用。案文写道：“疫苗的有效性水平和持续时间的信息在许可时可能有限。**”这可能会促使疫苗制造商 “进行某些中共病毒疫苗有效性的许可后研究”。

事实上，辉瑞的中共病毒疫苗试验实际上产生了“严重”副作用，志愿者报告了头痛，发烧，偏头痛和类似于宿醉的症状。

医生一直都在向疾病预防控制中心警告疫苗的危害。美国医学会的桑德拉·弗雷霍夫博士（Sandra Fryhofer）敦促应该警告人们接种疫苗可能的结果。她说：“我们确实需要让患者意识到，这不是在公园里散步。他们将知道他们将要接种疫苗。他们可能不会感到美妙。但是他们得回来接种第二次。”
