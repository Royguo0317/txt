# **纳瓦罗：民主党通过弹劾“合法当选”的川普来“实施暴力”**

[!\[\]()!\[\](https://spark.adobe.com/page/1pHZHgilP02UB/images/e45a8854-3f6d-47de-b8bb-320cc8c40bb3.png?asset_id=aa29bedd-3169-4b4b-aae4-dcd1666db9ee&amp;img_etag=%229eac2f0d694c08910442abccc4fba0a6%22&amp;size=2560)](https://spark.adobe.com/page/1pHZHgilP02UB/images/e45a8854-3f6d-47de-b8bb-320cc8c40bb3.png?asset_id=aa29bedd-3169-4b4b-aae4-dcd1666db9ee&amp;img_etag=%229eac2f0d694c08910442abccc4fba0a6%22&amp;size=1024)

本周四，白宫贸易顾问彼得·纳瓦罗称民主党在总统任期的最后几天，攻击总统和滑稽地弹劾他，是在对美国国家实施暴力。他同时坚持川普总统在去年11月已经被合法地选举为美国总统。

纳瓦罗对福克斯商业新闻的玛丽亚·巴蒂罗莫（Maria Bartiromo）说：“我要对国会山上的这些人说，住口吧。别说了。让这个人带着尊严和平地离开。他是这个国家历史上最伟大的工作总统，最伟大的贸易谈判代表。他们对他的弹劾是错误的。”

纳瓦罗还坚持认为“如果今天举行选举”，川普将再次获胜，“这就是民主党人所担心的。”

纳瓦罗说：“在这里，我从未像现在这样生气过，我认为投票支持川普总统的7400万美国人的感受和我完全相同。”

纳瓦罗的评论是在讨论社交媒体公司采取行动阻止川普发声，以及采取“钳制行动”（两次侧翼攻击）搞垮社交媒体Parler的讨论期间进行的，Parler被许多保守派用作推特和Facebook的替代品。

纳瓦罗说：“对于谷歌和苹果公司而言，钳制的第一步是不允许下载Parler应用程序。” 他称这些举措“令人不寒而栗”。

他接着说：“然后，随之而来的是亚马逊的残酷绞杀，停止了Parler的云服务，因此它再也无法访问数据。最终亚马逊彻底摧毁了这个公司。”

来自网络，作者Anjali Nair

纳瓦罗表示，他认为Parler会有很多的法律诉讼要进行，而且国会必须认真对待科技巨头的反垄断问题。

他补充说，当推特审查川普而不是伊朗的阿亚图拉或中国共产党等其他国家时，这是“对我们民主的严重威胁”。

同时，国会正在将一个“五五开的国家变成一条鸿沟”，但纳瓦罗说：“这个国家的人民，支持川普的人民，将不会支持这一点。”

纳瓦罗还表示，他想警告美国，到2021年的经济可能会像电影《大空头》（The Big Short）一样，华尔街的上流人士将自己的资金投入10亿美元做空，因为他们都知道次级抵押贷款有违约的危险。

纳瓦罗说：“在这个经济屋子里一直有白蚁在侵蚀，有数千万的美国人失业，这是（众议院议长）南希·佩洛西的错。他们可以在一天内弹劾总统，却无法在大选前，用三个月的时间通过的宏观经济刺激措施。这将会造成2021年成千上万小型企业的死亡。这太少了，也太迟了。”
