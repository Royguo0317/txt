在英国，冠状病毒的印度变种正在日益传播，并令政府担忧。 B.1.617.2的传播应通过大量测试来防止。重点也放在年轻人上。

英国首相鲍里斯·约翰逊（*Boris Johnson*）对印度冠状病毒变种在他国的扩散表示关注。约翰逊（*Johnson*）星期四在对一项研究的回应中说：“这是一个令人担忧的变种。”该研究发现，印度变种在英国的患病率更高-在4月15日至5月3日之间接受测试的127,000人中，有7.7％呈阳性。

根据帝国理工学院的研究，此期间的冠状病毒病例总数降至2020年8月以来的最低水平。这是由于严格的封锁和成功的疫苗接种运动所致。应该仔细观察印度变种，因为它可能比其他变种更具感染力。

该研究报告的发布恰逢计划大规模放宽英国的封锁措施。约翰逊说，现在应该采取所有谨慎和考虑周全的步骤。
