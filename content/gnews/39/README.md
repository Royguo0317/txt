**N****个侧翼——**

家庭：家庭之本质功能在于“生养”和“爱”。前者至少现在以至未来相当时间，都会是一般家庭一项最基本功能（当然也不同于传统农耕中国多子多福之生养概念）。尤其后者“爱”，超越等级尊卑，于夫妻，是风雨阳光、彼此扶持，一生相依的爱；于父母与子女，是父母对孩子护犊情深的爱，孩子对父母天然相依的爱，是少时儿女对父母稚嫩依恋的爱，晚年父母对子女门庭候望的爱，孩子成年父母老壮时彼此念念在心的爱。

教育：人非工具，人自身即是目的，人的最终目的在于作为“全人”（Whole Person）的自我实现，因此教育的本质在于启智，使探索未知，反思自身，尽可能去除内外遮蔽，趋近真知，自我绽放。

信仰：真正宗教，皆通往彼岸的门庭。人者独立自由身，当自主选择信仰之门。信仰之体现，在践行。而修行所依托者，相应核心宗教典籍。僧侣神职人员有引路之功，无托举之力，因其自身尚属沉重凡胎肉身。奉养神职者非得救之途，奉行教典者往永生。

政治：社会乃个人之合和，政府乃人际关系、社会分工与分配复杂化不得已之产物。其直接目标，在于基于平等公平的原则，确保并保护每个社会个体都能各得其分，安享其得；其最高目标在于在负责任的自由基础上，保障每个个体社会自我的最大实现。一人一票，是社会之政治价值得以实现的基本要求；扁平政府架构，是民意得以最大实现、政事服务透明高效的保障。

言论：人皆有言责自负前提下自我表达的权利和自由，媒体所遵循者亦法权之下言论自由的原则。其中，媒体之新闻时政，须信守真相准则，以确保民众知情权，并坚守社会正义，实行社会监督；专业资讯须信守真知准则，以利民用。二者皆须持守唯真不破，禁绝利益偏导。

经济：生存乃人的第一要务，社会一切生产活动和经济行为，其首要目的和终极目的亦在于此。超出此范围者，当以不滥产滥用为限，以节用保护地球资源；防止垄断囤积，以物适其用，财尽其力。

科技：科学乃哲学追问的延展，是人类对真知真理无有止境的探寻。追问无极限，科学无疆界。技术为科学的末枝，本意在于便利优化民生，直接利弊作用于人类生活与生存，当以人文真善美为底色，以人类安全与健康存在为尺度，故须有边界禁区。

饮食：饮食之本质在于为身体供给能量，以维持体能延保生命。时代进步，物质丰富，亦当华美有度，杜绝浪费。尤其奇食异味，当有所禁忌。

医疗：医疗关乎人类生命与健康，乃公共基本核心利益之所在。医疗之本质，即在治病救人。故当“研、产、营”以纳税人政府为主导；私营者限制以医牟利，禁绝唯利是图。

文艺：文艺之本质在于感悟生命，吟哦性情，刻写现实，放飞想象。艺术美为其底蕴，真性情、真感受、真感动为其关键。亦如文贵先生2021年1月1日直播之论音乐本质“感性、感官、感知”三感论。

体育：体育之本质在于强身美体，故须真正实现大众化。进而关乎拼搏挑战、规则公平、和平友谊之体育精神与荣誉问题，再进而才是奖杯奖牌的问题。

传统：所谓传统，无非历史之积淀。以正道主义观之，没有狭隘情绪化的所谓数典忘“祖”，只有是否适应符合人性发展之时代变迁与时代精神的问题。其取舍所依据者，唯联邦中国新时代之全球化格局与自由、平等、法治、民主愿景。
