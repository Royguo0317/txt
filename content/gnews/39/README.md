## 【快讯】河北回应村民被隔离，家畜“专人代养”
` 国内新闻组` [轉載自GNews](https://gnews.org/zh-hans/786864/)

喜马拉雅-国内新闻组 作者/素材：鹰(文言) 校对：加文gavin
![]()![](https://gnews.org/wp-content/uploads/2021/01/image0-2-37.jpg)
新浪网1月21日转载新京报报道，根据中共农村农业部消息，河北为解决中风险地区村民被异地隔离，畜禽无人照料的问题，推出“专人代养”服务。

石家庄藁城区的20个重点疫情村庄中的367个养殖户施行封闭管理；其中增村镇的小果庄、南桥寨、刘家佐村民被要求撤出隔离，而其48个养殖场（户）的3.6头（只）畜禽交由17名基层动物卫生监督人员代养。

1、问题一，农村畜禽主要以是牛、羊、猪、鸡、鸭等为主，以48个养殖户推算，原本每家均是至少一人负责日常喂养和粪便处理；

3.6万头（只）平摊到17个人，平均一人负责750头（只）畜禽的饲养，若750只鸡鸭，每天的饲料、鸡蛋筛检、粪便处理工作就绝非一人可以承担的，而上百头牛羊或猪的饲养难度更大，即3.6万头（只）畜禽饲养工作交由17名专人处理，本身就是虚假宣传，可望不可及。

2、河北省为保证畜禽的养殖不断料，出台政策保证种畜禽、饲料、药品物流畅通，结合上一则疑问，单单现成的外来饲料运进各养殖场再卸货囤放就不是17个人能够完成的工作，更何况还要将饲料分门别类，往返3个村喂养和日常维护，毕竟隔离期至少半个月，而17人应对半个月的“饲养工程”只有“同吃同住”；

而依据中共“党员免费加班”惯例，这一代养工作只会沦为以“饿不死”为底线。

3、石家庄疫情爆发后，区域隔离、镇村封闭管理的同时，河北施行畜禽养殖人员和畜禽实行“双封闭、全消毒、防鼠害、常检测”，对涉疫村庄进行防鼠防虫和消毒。

相比20年初全国封城时期的防疫措施，显著加大了消毒防鼠害的力度和规模，虽有防范动物染毒带毒风险，但不可排除让老鼠为此次疫情“背锅”可能。
![]()![](https://gnews.org/wp-content/uploads/2021/01/image0-3-18.jpg)
河北政府对村民被隔离畜禽处理的回复，反而暴露出其回答“不走心”：一是不了解农村畜禽实际情形，搞出17人负责3.6万头（只），二是暴露出将村庄疫情严重推诿至“鼠虫害”之嫌。
