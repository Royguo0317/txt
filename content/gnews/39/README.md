## 【香港要闻】香港文革2.0 残良而自肥
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1001973/)

搜集\编撰：西西 \ 封面合成：文粤

中共喉舌《文汇报》19日再度出手，狙击科大教授李静君。左媒《星岛日报》亦称接获「图文并茂消息」，指李静君在前年 11 月出席台湾社会学年会，参与《时代革命：香港反送中运动的社会学反思》的主题演讲。又曾以「守护孩子」成员身份到抗争现场，为抗争者争取时间撤退。又指李承认「做了很多时情，在香港法律下定义为非法（a lot of things I’ve done are defined as illegal under HK law）」。
![]()![](https://gnews.org/wp-content/uploads/2021/03/32294-1.jpg)科大学生会临时干事会和人文及社会科学学院学生会发表声明，强烈谴责文汇星岛等狙击李静君教授的行为，表示支持李教授。 （大纪元合成图片）
大纪元香港3月21日讯指，《文汇》引述科大校董、民建联陈恒镔及「市民」意见，认为李静君没有资格在科大执教，要求相关机构尽快严肃处理。前特首梁振英亦表示，「（李静君）尽快自首，否则请警方主动调查」。

第二十九届科大学生会临时干事会亦非常关注今次事件并发出声明，感谢李静君反送中运动中的付出。香港科技大学学生会人文及社会科学学院学生会亦对此发表声明，表示强烈谴责狙击李静君的行为，「并呼吁各科大学子勿受左派媒体煽动及蒙蔽、时刻关注并支持李教授」。声明指，李静君参与「反送中」社运是《基本法》赋予的言论、示威及集会自由，「左派传媒颠倒是非，其指控实属无理」。声明强调，今次因学术交流的演讲强加颠覆国家的罪名，「难免使往后学者人人『自省』，阻碍学术自由传播」。

李静君教授形容反送中示威为「革命性时刻」，曾提及「以武制暴」，并分析「黄色经济圈」、「非官方外交」等例子。总结演讲时，李形容「革命」难以在短期内停止，形容革命是「永久」、「没有尽头」，又指这是建设香港及建立「共同体」的过程。

新闻工作者区家麟专栏在他的专栏里指出，香港有一群开口埋口就「黑暴」、吃「反港独」、吃「国安饭」的人，迅速冒起现形。在教育界、传媒界、公务员系统亦如是，有原则、讲理念、做实事的人正被逆向淘汰，换上识时务者，端起反港独的国安饭碗，大快朵颐。他们掀起香港文革，开始批斗艺发局、批斗M+博物馆、批斗艺术文化界、批斗香港电台、批斗教育界、甚至批斗建制阵营自己人不够忠诚统统废物。香港曾引以为傲的所谓法治，坠落得粉身碎骨，如此心狠手辣。法律武器用得如此丑陋，以莫须有的罪名把有良知的人赶尽杀绝。

李静君教授一向敢言，被坊间称为「女版戴耀廷」。她的言论与行动在港共政府眼中是“累累的罪证”，但在笔者眼中却是辉煌的战绩。
