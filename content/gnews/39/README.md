5月10日，国际歌组发表了由我作词的歌【想跟七哥唠家常】，这首歌是由文贵先生亲自选中，1月份开始配曲创作的，四个月才出来，甚是高兴。急忙打开听听，漂亮！意犹未尽，没了！再看组里，木兰战友发了一段说明：这是一首未完成的歌曲，编曲与主唱战友都已失联。墙内战友心系七哥，谨以此歌代表墙内所有失联战友向七哥送上生日的祝福！我们一定盘古见！顷刻，泪崩・・・・・・

平日里，战友们一起完成作品、一起上街游行、一起说说笑笑，虽说互称战友，但没感觉一声战友有那么重。可突然听到战友被抓了，真的，心痛！再想想文贵先生，那么多亲人、同事被抓了放，放了再抓，他还能在直播中笑面共匪，这得要有多么强大的心脏！更能理解文贵先生了，也更痛恨共匪了。只有早一点灭共，受迫害的战友们才能少受点儿痛苦。祈祷这两位战友，还有千千万万被关在CCP牢狱里的战友能平安无恙！我把完整歌词收好了，等你们平安归来・・・・・・
