```
名言第六句：真正的铜墙铁壁是什么？是群众，是千百万真心实意拥护革命的群众。1934年《关心群众生活，注意工作方法》
```


近90年前的这句话倒是很对应不久前的一个中东对垒——以色列和哈马斯的冲突，法治的以色列和独裁的哈马斯在战争来临时对人民的不同态度。如图所示——哈马斯：人民是我们的铜墙铁壁；以色列：我们是人民的铜墙铁壁。时空相隔，独裁政权的本质都是一样。
铜墙铁壁，是攻不破击不倒的，是硬国力和软实力的综合标志。在自由世界里，没有哪个政府敢把人民推向战争前沿，以人肉来浇筑“铜墙铁壁”。

在中共的历史上，人民的确是它的“铜墙铁壁”。共产党一不种地产粮，二不织布炼钢，三不经商买卖，哪里来的吃穿用呢？都是人民劳动而得，然后他们去偷去抢去骗去占有。在农村，他们煽动不识字的农民斗地主夺土地；在城市，煽动工人罢工对抗资本家；煽动血气方刚的学生罢课对抗教授，学生们吃着白面馒头斥骂抗日的民国政府……这就是共产党嘴里的“铜墙铁壁”！共产党需要这样的“铜墙铁壁”去对抗合法政府，为它篡政积蓄力量。

我们也常常在共产党的影视剧里，看到“鬼子进村”或者“蒋匪军进村”，召集几十上百村民，威胁他们交出窝藏的共产党员。村民宁死不屈，那“共产党员”躲在安全的角落里，咬牙切齿眼喷怒火手攥拳头发誓：“乡亲们，我们一定会回来为你们报仇的！”发誓完毕，溜之大吉，留下村民被枪杀被刀砍！神剧出来之前，一直都是这样的套路！这样的人民，“心甘情愿”做“我党”的人肉炸弹，怎么能不是“我党”的铜墙铁壁呢？

几十年后，当共产党打下江山坐稳龙椅的时候，他们自己就蜕变成“铜墙铁壁”，枪杆子、刀把子、钱袋子、笔杆子、喇叭子“五子登科”都成为了他们的“私奴家丁”，曾经被利用被赞誉为“铜墙铁壁”的人民就变为“泥墙土壁”了，被摧毁被镇压只是看中共的需要罢了……可笑的是，毛是在《关心群众生活，注意工作方法》一文中阐明这个观点的。原来，共产党“关心群众生活”就是让群众肉身去做“铜墙铁壁”，“注意工作方法”就是坑蒙拐骗……想骂人！此处省略三万字。

“真正的铜墙铁壁是什么？是群众，是千百万真心实意拥护革命的群众。”90年前的这个大骗局忽悠了无数的群众，成就了他们的狂想；群众们泪水再多血水变黑也挽回不了过去混沌中助纣为虐的每一个选择。在摇摇欲坠的今天，共产党祭出毛的这句话，想再次欺骗“不明真相的群众”吗？时过境迁，党还是那个百年不变的党，而群众已不是那个愚昧的“群众”了。“群众”们早已走出井冈走出窑洞走出大别山，他们的眼光早已翻阅窑洞的泥墙和山峦的沟沟坎坎，他们看到了蓝天和天上的飞鸟，听到了掠过的哨音和马背英雄……他们意外地看到了东德人涌入西德的场景，体验了台湾厂商给自己带来的现代化管理的优势，尤其是惊喜地发现了“让领导先走”的定点斩首模式……

当今天的哈马斯说出“人民是我们的铜墙铁壁”时，有更多的人终于如梦初醒：共产党和哈马斯都是一路独裁货色，都是踩着人民的血肉躯体享乐无穷，都是把枪架在人民的脖子上，把手无寸铁的人民推到前线为统治者挡子弹！而真正的民选政府恰恰相反，他们把妇孺人民挡在身后，自己扛枪站在敌人面前，用自己的生命来保护人民的安全！这一场“哈以冲突”中对待人民的截然不同的态度再次警醒世界：结束独裁政权，解放奴役中的人民，是21世纪全人类最首要的政治诉求！这也是我们中华民族几千年来最坚定最执着的追求！

我们，再也不要做极权的“铜墙铁壁”！我们要做自己的铜墙铁壁，我们还要一个给人民做“铜墙铁壁”的政府！
