## 爸爸妈妈们想早日体会育儿的自豪和幸福吗？

Youtube上一个叫Tydus的美国小男孩曾经迅速爆红，原因是他3岁时就能给妈妈准备浪漫烛光晚餐，小暖男买菜、做饭、布置餐桌无不有模有样，萌化人心，圈粉无数。当然也让人感慨：自己的3岁孩子稍不如意就撒泼哭闹，这个“别人家的孩子”已经能给父母做饭，到底是怎么炼成的？



美国作家兼教育家尼尔波兹曼在他的《消失的童年》中，曾阐述了100年前的普遍童工现象和今天“娇生惯养”的童年是怎样从一个极端走到另一个极端。

悲惨的童工是令人心痛的一个极端，而“娇生惯养”，避免孩子从事除“学习”外任何劳动的做法则是另一个极端。适当地做些家务劳动，对孩子会有怎样影响？如何能让孩子通过做家务活，分担父母负担的同时，又对孩子自己的人生有积极影响？

领导力Torch的”CEO系列采访”，有一期采访美国游戏直播平台Twitch的创始人Justin Kan。Twitch是美国最有影响力的游戏直播平台之一，2014年被亚马逊以9.7亿美金收购。Justin 的一个弟弟也是创业者，他创立的自动驾驶公司被通用汽车以10亿美金收购；另一个弟弟是一名出色的计算机工程师。是怎样的教育模式，让这一家出了这么多人才？

[!\[\]()!\[\](https://spark.adobe.com/page/j3yLc88on6hkV/images/3dbedfd0-bd77-4513-bc44-933b7567f251.jpg?asset_id=0cabcada-7771-44ef-a2be-f287e1de3f6e&amp;img_etag=%228610c22e75dfbe22b6f929a351487ecd%22&amp;size=2560)](https://spark.adobe.com/page/j3yLc88on6hkV/images/3dbedfd0-bd77-4513-bc44-933b7567f251.jpg?asset_id=0cabcada-7771-44ef-a2be-f287e1de3f6e&amp;img_etag=%228610c22e75dfbe22b6f929a351487ecd%22&amp;size=1024)Justin Kan

Justin Kan认为：他和弟弟的创业成功，要归功于妈妈不仅要求他们做事须尽最大努力做到最好，而且必须做家务。三兄弟每天都在妈妈要求下做家务，做饭、洗碗，甚至刷墙、修理房子，偶尔还要协助妈妈完成工作，比如录入数据。正是这些，培养了他们的协作能力和领导力。

有人翻译了这个采访，把**Justin分享的经验称为“家务教育法”**。

Justin说：他的妈妈分配家务的方式颇有特别之处。一般家长直接告诉孩子做什么，他妈妈是给三个孩子一个清单，告诉他们，这些是你们需要完成的事，没完成之前，谁都不许打游戏。但至于谁做什么，妈妈完全不管。于是，三兄弟要自己商量怎样在最短时间内完成？怎样公平地分配工作？怎样彼此配合？

这锻炼了他们沟通、谈判、分析、制定目标等多种能力。Justin说这种方式让三兄弟把自己看成一个团队，而不是独立个体。他们学会了如何向同一目标努力。这种经验，后来被他复制到了创业公司的管理工作上。

而且，Justin的妈妈在安排孩子做家务这事非常坚持。即便三兄弟强烈抗议，妈妈也从未心软。她会告诉他们：“这很难吗？假如去了我的老家——马来西亚的农场，你们一天都撑不过去。”

这种坚持让三兄弟认识到并培养起一种理念：生活在某种程度上是痛苦的，但也有美好。人必须同时接受“好的”与“坏的”，这才是真正的生活。成为创业者以后，每天面临巨大压力的时候，这个理念成了他们的精神支柱。

三兄弟的成功是不是一种偶然？他们基因好、智商高？未必和做家务有关吧？当然，他们的成功肯定有很多其他因素，比如敏锐的市场判断力，强大的学习能力等。但不可否认，小时候做家务的经历打磨了他们的个人品格。

赵小兰及其家人是被郭先生揭露的中共对外用“蓝金黄”方式拉拢渗透的典型代表之一。赵小兰父亲赵锡城对她们姐妹的一个教育方法值得借鉴，他曾介绍：赵家六姐妹包揽所有家务。即使赵家经济上逐渐好转，有了管家，父母也仍要求孩子自己洗衣服、打扫房间。闲暇时，还要六个孩子分担家里的琐事。每天早晨上学之前，她们要检查自家游泳池的设备，捞出脏东西。周末，则要把院子里的杂草和蒲公英拔掉。家里有客人时，也是几个姐妹全程照顾服务客人，令人难以置信的是，赵小兰家门前长达120英尺车道的柏油路，是几个姐妹自己铺成的。赵家每年安排两次全家旅游，从选择地点、订旅馆房间，乃至吃饭的餐馆，完全由孩子负责。

可以看出，赵锡成的家务教育方法对赵家六姐妹在不同领域取得成绩有很大助益，但是他们一家却因贪婪，没有看清中共的实质，成为自由民主法治社会的破坏者，留下历史污点。

其实，关于家务如何影响儿童的成长，教育专家早已有研究报告。

家庭教育专家Marty Rossman曾做过一个时间跨度长达25年的深度研究。他追踪了84名儿童在家务上的参与程度，分析了不同年龄段的参与度，分别是3～4岁、9～10岁、和15～16岁。采集的数据包括孩子的智商、家长跟孩子的互动方式、完成的家务量、家务类型等等，然后分析这些因素如何影响孩子在25岁时，其学习、事业和人际关系的情况。

结果发现，相比起9～10岁或者是15～16岁开始做家务，从3～4岁就开始参与家务的孩子，成年之后有更强的适应环境能力，他们跟家人朋友的关系更好，事业更成功。这位教授分析，主要原因就是家务劳动，向孩子们灌输了为团体，为家庭做出贡献的重要性，也锻炼了动手能力和解决问题的能力，这让他们在成年后处事更加稳重，更有领导能力，也更有同理心。

**结合相关研究资料，这里总结了几种鼓励孩子做家务的方法：**

1. 以“承担一定家庭责任”为“家务分配原则”

通常，家长给孩子布置的任务都只与孩子自己相关，比如“收拾好自己的玩具，洗自己的袜子”等。但为了使孩子更有同理心，学会不只以自己为中心，不仅仅为自己思考，父母需要将孩子的目标聚集在“照顾家庭”上。比如，需完成的家务清单应包括“给全家做饭”“收拾客厅或厨房”等，而不是“只做与自己相关的某事”。让孩子理解到“做家务”是在承担家庭责任。在劳动中，孩子更容易理解尊重父母或其他人的劳动。所谓的“穷人孩子早当家”，“当家的孩子更懂事”是一个道理。

2. 为孩子树立积极的榜样

父母是孩子最重要的榜样，当家长在做家务的时候，喜欢抱怨，孩子很可能耳濡目染，也容易对做家务形成消极的态度。所以，在日常生活中，要以积极的态度谈论家务活，踊跃地主动分担家务。如果家人都有一种乐在其中的心态，孩子们也会更享受这个过程。

3. 不给物质奖励，多给口头表扬

有的家长会认为，如果孩子做家务，承诺给他们零花钱，他们肯定会更有动力，更勤劳。但事实上并不是这样。心理学家认为，因做家务而给孩子劳务费只会弄巧成拙。假如孩子有“做家务就会给钱”的认知，那么家务活就会从“帮助”变成了“交易”。这不利于帮助孩子培养主动帮助家人的心态。正确的方式是用语言表扬孩子，研究显示，如果对3～6岁的孩子，家长“感谢他们成为一个小帮手”，那么，这会显著提升他们帮忙的意愿。

4. 父母须有这样的认知：要求孩子做家务和学业并不冲突。

即使作业很多的情况下，鼓励孩子在休息时间选一个公共家务清单上他/她能够做的一项家务，完成后给与赞扬，也是一种放松方式。因为“放松”不是什么都不做或者仅仅是娱乐，而是从一种状态转入另外一种不同的状态，尤其是改变久坐状态，之后还有某种成就感，至少不会产生虚无感。家长不仅要“狠下心”要求孩子做家务，也要告诉孩子他们能做到，有必要做。

5. 树立“做家务”是每个家庭成员的义务，也是爱与支持家人的一个重要方式。

因为“爱”，所以愿意付出行动，对于孩子也一样。帮助父母分担家务，为家人做些家务，是“爱父母”“爱家人”的一种很好表达方式。家人之间的凝聚力也会在这个过程中逐渐加强。

事实上，学校教育、社会教育和家庭教育，三者同样重要，教育不能仅仅靠学校，孩子重要的品性和能力通常都是在家里，尤其在父母影响教育下实现的。

中国目前教育严重内卷的情况下，本该通过快乐游戏，适当劳动感知世界、体验世界的学龄前儿童却被要求学习各种知识技能，实际上会扼杀浪费儿童的天赋和未来潜能。

与中国人拥有几乎同样基因的日本人不断获得诺贝尔奖，而1949年中共统治后的中国只出现过很多伤仲永一样昙花一现的“神童”，接受过中共国基础教育的人获得诺贝尔奖的少之又少。可见，好的制度不仅能真正激发人性之善，人的无限潜能，直接影响教育，也关乎我们每个人的未来，因为世界的未来取决于我们的孩子。

孩子的第一任也是人生最重要的老师是“父母”/“养育人”。父母/养育人的认知水平决定了孩子的终身素质。

分担一定家务，增进家人情感，又能帮助孩子获得多种能力，更理解别人，尊重别人劳动，懂得感恩的“家务教育法”，值得家长们早点尝试！相信看到曾经只会玩乐哭闹的孩子也能像模像样地帮助爸爸妈妈做事，面对问题，解决问题时，做父母的自豪和幸福感一定会油然而生。
