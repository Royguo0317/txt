**西班牙20202年12月19日电/西喜社——**

据2020年12月19日路透社消息，中共军方说，在周六时中共军队尾随了一艘正在通过敏感台湾海峡的美国军舰，他们谴责向台湾独立的支持者“暗送秋波”之类的任务。
中共声称民主的台湾为其领土，他们对美国加大对该岛的支持感到愤怒，包括出售武器和通过台湾海峡航行的军舰，这恶化了北京与华盛顿的关系。
美国海军表示，导弹驱逐舰“穆斯丁”号“已按照国际法于12月19日进行了台湾海峡的例行过境”。
它补充说：“该船经过台湾海峡，体现了美国对自由开放的印度太平洋的承诺。”
这是美国海军今年在该海峡的第十二次航行。
中国军方在其东部战区司令部发表的一份声明中说，其空军和海军部队“一直在尾随并监视”这艘船。
它说，这样的任务“有意升温台湾问题，因为他们担心台湾海峡的平静，并向台湾独立势力暗送秋波，严重损害了台湾海峡的和平与稳定”。
声明补充说，美国正在表现武力，并试图利用台湾作为其自私战略目的人质。
台湾国防部在台北说，这艘美国船朝南航行，它也监视了它的动向，“情况是正常的”。
北京认为，台湾总统蔡英文决心正式宣布台湾独立，这是北京的红线。 她说台湾已经是一个独立的国家，名为中华民国。
面对中共在台湾附近反复的军事活动，蔡将加强台湾武装力量作为优先事项，其中包括今年偶尔派出战机穿越非正式的缓冲区台湾海峡中线。
