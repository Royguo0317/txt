谈到世卫专家有没有访问武汉P4实验室权限，这可是一个生物学专业问题，我们需要耐心一点阅读笔者根据闫博士讲话整理的文字。我们来看一下闫博士怎么回答路德提问时说的。

世卫专家真的要查清楚病毒源头，究竟需要多少时间？这又是一个专业问题，冠博士提问闫博士说：“我想问一下闫博士，这些WHO专家如果要真真正正去P3/P4实验室把所有样品都查了，如果认真去查的话，大概需要多久的时间？”

闫博士：“首先我要说，像这样的工作在任何一个实验室都不是一个简单的工作，因为生物工作人员都是有经验的，知道你每一次养一个病毒株，都需要做几十管样品，然后把样品放到冰柜里去。就是这一个实验可能花你几个小时，然后3、4天就能做完的这种实验。然后你有这么多的实验、这么长的实验，就我的工作经历来说吧，我一个人的工作量，我在港大的时候，我做第一个课题的时候，我一年用了800多只老鼠，我亲手做的处理，除了有人帮我饲养老鼠外，每一只老鼠都是我自己做的。”

简单说来，如果你只在一个实验鼠上获取一个病毒株，需要花你几个小时，然后3、4天就能做完的这种实验。还不包括你对一只实验鼠进行全方位利用，闫博士接着说：

“我取的样本，（通常规定只有一个），因为我要充分利用这只老鼠嘛，我最多的时候要在一个老鼠身上取五到六个样本，包括心脏抽血、包括取皮、包括取肺、包括取淋巴结，包括气管内的盥洗物，包括取皮毛，还要给转基因老鼠做ID鉴定。有没有出现肿气的污染，这些都是你要做的，那么你想一想，这么多数据，比如取一管血，我需要做很多个实验。我可以收它的血清、做它的抗体。和这些炎症因子，还有根据这些里面的细胞去做不同的流失分析。那这个又是一串一串的。所有的结果，一只老鼠就有这么多，一年的实验工作量是多大？你不用像我一样在实验室里一呆就是16—18个小时，我就问你，如果你一个星期工作20个小时，这是达不到正常工作量的。一个星期20个小时，一个工作人员有多大工作量？你可以想一下。”
