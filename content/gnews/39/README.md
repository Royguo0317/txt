[法新社](https://www.rfi.fr/cn/%E7%BE%8E%E4%BF%84%E6%96%B0%E6%88%98%E7%95%A5%E6%AD%A6%E5%99%A8%E8%A3%81%E5%87%8F%E6%9D%A1%E7%BA%A6-%E8%92%B2%E4%BA%AD%E7%AD%BE%E7%BD%B2%E7%AB%8B%E6%B3%95%E5%BB%B6%E9%95%BF5%E5%B9%B4)1月29日报道，俄罗斯总统普京已就美俄主要核武条约签署立法延长5年。该项《美俄新战略武器裁减条约》是这两个前冷战敌对国家之间最后仅存的武器裁减条约。

![]()![](https://gnews.org/wp-content/uploads/2021/01/40-2.jpg)

普京26日晚间与新上任的美国总统拜登首度通话后，将这项法案送交国会。克里姆林宫指出，延长这项条约“可保存美俄间战略关系的透明度及可预测性，并支持全球战略稳定”。此前美国和俄罗斯都希望中国加入核军控条约，但均遭中共拒绝。

前苏联加上由其主导的社会主义东欧国家，经济体量和军事实力接近当时的美国，是名符其实的敌手。俄罗斯近几年的GDP从未达到1.7万亿美元/年，而且国民经济严重依赖能源出口，经济体量仅相当于美国的1/12。

与此同时，社会主义极权统治之下的中（共）国，经济体量超过美国的2/3。为了取代西方成为国际秩序的主导者，中共在国际舞台上实行战狼外交：不仅深度蓝金黄美国，还干预美国大选，发动生化战。而拜登在正式就任美国总统之前，一直强调美国最大的威胁来自俄罗斯而不是中国，令人十分费解。

拜登上任总统后，还未发表哪个国家是对美国构成最大威胁的相关言论。
