# 乔治亚州诉讼：证人证明使用了不同的纸来制作“假冒”选票，“水印为纯灰色而不是透明的” 选票100%投给了乔·拜登

律师西德尼-鲍威尔在乔治亚州提起了长达104页的诉讼，指控在2020年的选举中存在大规模的选民欺诈。

西德尼-鲍威尔的诉讼包含了许多违规、异常、老式 “塞票”、违反乔治亚州选举法、非法审计程序和使用投票机操纵选票的指控。

许多目击者作证说，他们看到标明给川普总统的选票被放进乔-拜登的选票堆里，而预先印制的 “崭新 “选票则百分之百投给了乔-拜登。

一位宣誓人作证说，他们看到了带有较深色水印的 “伪造 “选票100%投给乔-拜登。

当选举主管被提醒有假选票时，他们推脱说水印较深是因为使用了不同的打印机。

[!\[\]()!\[\](https://spark.adobe.com/page/R6qSNyO7ISgFm/images/7e874e97-007f-4824-bf75-4d547a8c765d.jpg?asset_id=ba6fb6e3-0bbe-4efb-a8e7-ea105339feac&amp;img_etag=%22358be808f8395df4a0c0d928113879b0%22&amp;size=1024)](https://spark.adobe.com/page/R6qSNyO7ISgFm/images/7e874e97-007f-4824-bf75-4d547a8c765d.jpg?asset_id=ba6fb6e3-0bbe-4efb-a8e7-ea105339feac&amp;img_etag=%22358be808f8395df4a0c0d928113879b0%22&amp;size=1024)Sidney Powell 西德尼-鲍威尔


在西德尼-鲍威尔的文件第35页，宣誓人作证 “使用不同的纸张进行投票，构成欺诈声明如下“：

*我注意到，我审查的选票几乎都是支持拜登的。许多批次的选票都是100%支持拜登。我还注意到，至少有3张选票上的水印是纯灰色的，而不是透明的，这使我认为选票是伪造的。我对此提出质疑，选举主任说这是一张合法选票，是由于使用了不同的打印机。许多选票上只选了拜登，而没有选其他候选人。*

另一位宣誓人【在伍德（Wood）诉Raffensperger案中作过宣誓证词的同一证人】解释说，她观察到一批质地不同的纸质选票，印有机器盖章，98%的选票都是乔-拜登的。

*大多数选票已经被处理过；它们被人填写过，边缘已经磨损。它们有明显的使用痕迹。然而，有一批选票却与众不同。它是原封不动的。纸张的质地有差别–好像它们是用来做缺席投票的，但却没有被用来做缺席投票。手感上也有不同。*

*这些不同的选票都有轻微的预折痕，以便于在扫描机上折叠和展开。这些选票上没有任何标记，以显示它们来自哪里，或在哪里被处理。这些选票很醒目。*

*在我20年处理选票的经验中，我观察到这些选票上候选人的标记异常统一，甚至可能是用选票标记装置。根据我观察和对这些选票的估计，大约98%的选票是乔-拜登的。我只观察到其中两张选票是投给川普总统的”。*
