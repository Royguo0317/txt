# **美国政府官员、美国科学当局和中共国有关部门掩盖了COVID-19大流行起源的大量证据**

[!\[\]()!\[\](https://spark.adobe.com/page/NMGL8Az7cxuEb/images/68f03564-a4d2-444e-a47e-286ba747cd23.png?asset_id=72cf5fd5-2759-48e1-9a04-fdb2fd11d9bf&amp;img_etag=%22d5269bfec2eb3cdec2d73d2a4c12e3a4%22&amp;size=2560)](https://spark.adobe.com/page/NMGL8Az7cxuEb/images/68f03564-a4d2-444e-a47e-286ba747cd23.png?asset_id=72cf5fd5-2759-48e1-9a04-fdb2fd11d9bf&amp;img_etag=%22d5269bfec2eb3cdec2d73d2a4c12e3a4%22&amp;size=1024)

客座文章作者：劳伦斯·塞林(Lawrence Sellin)博士

到2020年1月底，美国政府和科学机构内部的人士越来越担心，美国人民可能会了解新冠病毒起源的真相。

也就是说，这是中共国的一个实验室在美国科学家的协助和美国政府的资助下制造出来的一种人造病毒。

除了来自中国共产党的直接压力外，毫无疑问，国际金融利益集团也对华盛顿特区的易受害和顺从的人民施加了类似的胁迫，如果民众广泛接受中共国制造了新冠病毒，这些利益集团在中共国的大量投资将处于危险之中。

和他们在中共国的合作伙伴一样，美国政府官员和美国科学机构的成员最害怕担责。这是他们似乎已经开始的掩盖行动的主要自私动机。

这一切始于2020年2月3日，当时在美国国家科学院(National Academy of Science)举行了一次会议，由以下人士领导：科学和技术政策办公室主任凯尔文·德罗格迈尔(Kelvin Droegemeier)、美国卫生与公众服务部高级科学顾问克里斯汀·哈塞尔(D. Christian Hassell)、美国国立卫生研究院国家过敏和传染病研究所所长安东尼·福奇(Anthony Fauci)。

以下是与2020年2月3日会议有关的电子邮件：与会者包括来自科学技术政策办公室、国家卫生研究院和联邦调查局的其他美国政府工作人员。

大多数参与的科学家要么有与中共国合作开展冠状病毒研究的悠久历史，要么后来与中共国以及新冠病毒起源于自然疾病爆发理论的有力支持有关。

根据会议前分发的任务和工作计划说明就已经决定，新冠疫请起源于自然发生或“进化”事件，必须反驳与之相反的观点。

也就是说，这一切是美国政府官员和美国科学机构的成员的责任，他们提供源源不断的证据来支持一个预先注定的结果，一个保护中共国、国际金融利益集团和他们自己免受潜在同谋或罪责的结果。

正如他们自己的电子邮件所披露的那样，参与准备对2020年2月3日会议作出回应的科学家们的讨论主要集中在压制关于新冠病毒可能源自实验室的任何讨论。

然而，为了创造新冠疫情是一种动物向人类自然传播、任何人都不应为此负责的说法所做的努力，远远超出了对2020年2月3日会议的回应。

对会议刚刚响应后，EcoHealth联盟的彼得·达扎克(Peter Daszak)于2020年2月6日分别联系了从2020年2月3日会议与会者选中的科学家，以启动一个支持中共国的科学期刊公关活动，支持新冠疫情是一种自然产生的事件的理论，谴责病毒起源于实验室的阴谋论。

这些邮件中的声明令人震惊，可以被理解为阴谋。

达扎克在邮件中自己的话表明，他的公关活动即使不是由中共国发起的，但也得到了中共国的支持，达扎克和其他人继续与中共国的研究合作充满风险：

“这封信措辞谨慎，避免政治言论，我们已经被（中共国？）告知将对支持在此次疫情中继续合作大有帮助。”

2020年3月7日，英国医学杂志《柳叶刀》发表了一篇题为《支持中共国科学家、公共卫生专业人员和医疗专业人员抗击新冠肺炎的声明》，达扎克在2020年2月6日的电子邮件中的一些人是共同作者，声明写道:

“我们站在一起，强烈谴责认为COVID-19并非自然起源的阴谋论。”

这一最初的举动之后，中共国和西方媒体发表了大量文章，支持新冠疫情是自然发生的观点。

自那以后，也有报道称科学和医学期刊开始审查反对自然解释新冠起源的观点。

这是一场非常成功的公关活动。

最夸张的是，美国国家情报总监办公室盲目接受中国共产党宣传的科学“一般常识”，于2020年4月30日发表以下声明：

“情报界也同意广泛的科学共识，即新冠病毒不是人造或基因编辑的。”

值得一提的是，与大量证据表明新冠病毒来自实验室相比，从来没有确凿的科学证据支持其自然起源。
