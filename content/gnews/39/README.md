#### 1. **班农: 新冠疫情法案是40年来最激进的立法**

史蒂夫·班农说，近2万亿美元的新冠疫情救济法案是过去40年中最激进的法案。“从社会角度来说，这是我见过的最激进的法案”，班农周三说：“而且是带着钱来的，必须靠现金来支撑。”

班农说，首先出台的是1.9万亿美元的法案，但到今年下半年，民主党还将添加一个2万亿美元的基础设施法案和另一个2万亿美元的“绿色新政”。

“所以今天（通过）的1.9万亿美元，实在令人匪夷所思”，他说：“当初针对奥巴马的8千亿美元的经济刺激计划的茶党(保守派)集会，已经让我们全面崩溃。”

班农表示尽管这次支出巨大，实际上却只有9%用于的(新冠)病毒的救助，共和党人竟然没有强烈反对该立法。“共和党人的声音像蟋蟀那样微弱不堪”，班农说。

#### 2.      劳伦·博伯特已经受够了“这些伪君子们破坏国家”

共和党众议员劳伦·博伯特(Lauren Boebert)解释了是什么原因激发她拍摄了一段新的视频，要求南希-佩洛西（Nancy Pelosi）拆掉国会大厦周围的围墙。

“（如此看来）墙是起作用的，武装警卫更有用武之地！”博伯特说：“民主党人（自己的做法）恰恰把他们的虚伪表现得淋漓尽致”，“这些伪君子正在摧毁我们的国家，我已经受够了。”

博伯特说，国会大厦周围的墙和金属探测器，只不过是一场“政治戏剧”而已，是佩罗西用来针对像她（博伯特）这样持有隐蔽携带持枪证的人士。

博伯特说她 “不是后座议员”，也不是政客。（备注：后座议员是指一些刚开始议会职业生涯的众议院普通议员）

她说：“我已经厌倦了那些不（为百姓）争取自由的政客。”她把这种失望作为她竞选的动力，她要为勤劳的普通美国民众发声。“若我不再连任，那我就回家做一个母亲。” 博伯特说道。

#### 3.      资助RINO: 川普总统不会让共和党在没有他允许的情况下募集资金

据杰森-米勒（Jason Miller）说，川普总统不会让RNC(共和党全国委员会)以他的名义为那些支持弹劾闹剧的叛徒们筹款，如果说川普总统对此“火冒三丈”还是轻描淡写了。

“未经许可，你不得使用（川普）的模型或肖像”，米勒说。川普总统不会让人们利用他的影响力筹集资金，用来帮助那些共和党的伪类们(RINO)或选择一些不良的候选人。（备注：RINO是Republican In Name Only的节略语，专指那些仅仅是名义上的共和党人，是个贬义词）

“这十个人不会再呆在国会”，班农说：“这些成员其中就包括配合民主党的第二次弹劾的利茲·切尼（Liz Cheney）。”(备注：利兹·切尼是美国共和党成员，前美国副总统迪克·切尼和前美国第二夫人琳恩·切尼的长女）

“（竟然）使用川普总统的模型和肖像（募款），也恰恰说明了他们极度傲慢”班农说。

RNC（共和党全国委员会）还给川普总统的律师发了一封傲慢的回信，宣称继续利用川普来募集资金。NRCC (全国共和党国会委员会)也表示，支持弹劾的共和党伪类（RINO）“将不会被受到惩罚”。

米勒说，目前这个情况已“基本解决了”，那就是，川普总统会支持共和党，但不会让他的形象被那些不是真正代表其选民的共和党人利用。

#### 4.      “被收买”: 拜登利用阿拉斯加会谈向中共屈膝投降

乔·拜登还没会面就已经向中共投诚了。

班农说：“川普是历史上第一次站出来说，嘿，这些都是坏人……他们在各个层面上都对我们构成了威胁，我要和他们对抗。”而现在，拜登将会面地点定在阿拉斯加的安克雷奇，对中国人可是“毕恭毕敬”。

“拜登甚至还没会面，就已经屈服了，说我们要把会面地点离你们近些，以示友好”，米勒说：“他们已经屈服了。”

“在中共还没有开始面对……在我们开始彻底了解发生在维吾尔（种族灭绝）的事情之前，拜登他们竟然就同意了这个会面？”班农说。

“这就是为什么当我们谈论来自境外的金钱和拜登家族怎么被收买的时候，是那么的重要。”米勒说。

当一名高级海军上将警告说，中国正在加强军事力量试图取代美国时，拜登却在中共国问题上表现的软弱无力。

“而你们这些纳税人现在正在全力资助他们（中共国）的军事行动，使用你们支付给他们拥有的美国债券利息。”班农说。
