**西班牙2021年1月3日电/西喜社—**共和党参议院团体称 “大科技公司又来了”

据Just the News周六报道，Facebook声称，在乔治亚州第二轮参议院选举前几天，一个 “自动错误 “意外关闭了共和党候选人的筹款页面。

全国共和党参议员委员会周五在推特上发布消息称，Facebook “关闭了乔治亚州战地基金的广告账户，这是全国共和党参议员委员会、参议员候选人大卫·珀杜（David Perdue）和凯莉·洛夫勒（Kelly Loeffler）的联合筹款委员会。”

“在距离选举日只有4天的情况下，这是不可接受的，”全国共和党参议员委员会在推特上写道。

该公司在关闭该页面时，以涉嫌违反 “欺骗性或误导性 “政策为由。Facebook后来表示，这场争议是由一个未指明的技术故障引起的。

“一个自动错误导致这个广告账户被禁用，”该公司告诉媒体。”此后，该账户已被恢复。”

乔治亚州的参议院第二轮选举将于1月5日星期二举行，该选举是在11月没有候选人赢得多数选票的情况下触发的。
