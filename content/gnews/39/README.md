印度的情况日益严峻，除了第二波流行病继续肆虐外，在患有冠状病毒的患者中还发现了所谓的“黑木耳”，这是由一组霉菌引起的霉菌病，它的特点是死亡率高，并且有可能对存活的患者致残。总理纳伦德拉·莫迪（Narendra Modi）今天说，这是该国面临的另一项挑战。

目前，在印度全国的冠状病毒患者中已有约5,000例确诊病例，但这一数字可能更高，因为只有少数几个州政府向卫生当局报告了病例。马哈拉施特拉邦已记录了1500例霉菌病和90例死亡，而新德里、卡纳塔克邦、哈里亚纳邦以及中央邦和拉贾斯坦邦也报告了相关病例。联邦卫生部随后敦促各州根据法律宣布该病为流行病，使医院必须向联邦当局报告病例。

专家说，这种疾病被称为“黑木耳”，因为它经常伴有鼻子周围区域的变黑，这在糖尿病患者中很常见，并且与类固醇的无节制使用有关。作为冠状病毒的挽救生命的治疗方法。在印度，合法的药物滥用现象非常普遍，大多数药物（包括类固醇）都可以在药店买到，而无需特殊处方。

印度是受大流行影响全球第二大国家，在过去24小时内，印度报告了259,551例新感染和4209例冠状病毒相关的死亡。

简评：在印度的失控疫情全球有目共睹，才有的冠状病毒的印度变异体造成的新的大灾难，现在在印度又出现了这种所谓的“黑木耳”霉菌病更加大了死亡率。印度将何去何从？无人知晓。
