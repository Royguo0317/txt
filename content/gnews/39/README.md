作者：Giselle
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/07183905/image0-40.jpg)
悉尼时间11月8日清早，澳洲总理莫里森在推特上发推文，恭喜乔.拜登和贺锦丽赢得选举，并期待与他们紧密合作。同一时间，澳洲外长玛丽斯.佩恩也转发莫里森的推文称，恭喜乔.拜登当选美国第46任总统，澳大利亚希望和乔.拜登政府合作，建立强大而宝贵的联盟。

澳洲各大主流媒体也纷纷报道了乔.拜登胜选的消息，美国的主流媒体也开始了集体的狂欢，庆祝乔.拜登的胜选，只有川普总统，在推特上发文称，自己赢得了7100万张合法选票，是美国现任总统有史以来最多的。在当天的另一则推文中，川普说，民主党不允许监督员进入计票间，数百万人甚至从未收到过他们的邮件……虽然川普的支持者还在极力抗争，但在全球主流媒体铺天盖地的报道中，川普的声音显得尤其微弱。在舆论上、在数据上，川普的确输了……

此时的川普总统据理力争的样子，令笔者想起了中共政府门口的上访者。是的，我们都知道民主党集体造假，你也有大量的证据可以证明。可是，又能怎么样呢？川普所信赖的民主法治机制，在中共几十年的腐蚀之下，早就溃烂不堪。美国的文明、甚至全人类的文明，已经到了悬崖边缘。此时的川普如果还相信靠这套溃烂的机制反败为胜，将会十分危险！

川普总统商人出身，法律、律师一直是他经商、为人、处事最大的依仗。多年的思维习惯，令川普总统过于迷信美国的法律。但法律是人制定的，也是由人来执行的。是人就有弱点，就可以被收买、被胁迫、被“蓝金黄”，因此，法律也不是百分百的完美，也不能百分百的可靠。

美国2020总统大选自11月3日开始投票以来，备受全球瞩目。只因这次美国大选不只是美国人自己的事情，还关系到全人类文明的走向。人类是选择民主自由，还是选择独裁专制，全依托于这次大选的结果。

由于民主党大规模造假，大选才进入第二天，就开启了法律战。川普总统的法律团队在密歇根州、乔治亚州、宾州、内华达州都提起了诉讼，并要求乔治亚州、威斯康星州重新计票。

截至11月7日，大选计票尚未完全结束，四个关键州包括宾州（20张选举人票）、乔治亚州（16张选举人票）、内华达州（6张选举人票）、北卡罗来纳州（15张选举人票）的最终结果尚未公布。

在这个背景之下，民主党以及全球主流媒体单方面宣布自己胜选，底气何在？

虽然川普总统掌握了民主党大规模造假的证据，可是，正如笔者之前分析，有理就能赢得官司吗？民主党既然敢明目张胆地造假，就一定留有后招，不怕川普总统启动法律战。

笔者曾经跟战友私下探讨过，川普究竟啥时候会动手？对方认为，川普有精神洁癖，他想要青史留名，不留瑕疵，这是他在本次战争当中反复、犹豫、甚至懦弱的主要原因，因为他想做到完美，所以，这场战争没有那么快结束。很可能法院判决出来之后，川普也不会动手。

笔者可以猜到川普究竟在担心什么，毕竟中共对内握着14亿肉票，随时可以准备撕票，对外拥有强大的邪恶的黑暗沼泽。川普处境维艰，难以抉择，可以想象。但是在全人类的生存危机面前，两害相权取其轻，这是川普作为决策者必须承担的责任。所以笔者觉得川普在本届任期未满之际，也就是接下来的两个多月内，可能会动手。川普总统必须意识到，我们正处于战争当中，美国迄今为止已经有1000多万人感染，24.2万人死亡。全球正以一天一个9.11、一天一个珍珠港的速度奔向死亡，无数人失业，经济衰退更胜1930年代，暴乱、谋杀、恐怖袭击此起彼伏。在这种情况之下，川普总统如果还继续守着他的君子底线，那就不是高尚，而是货真价实的愚蠢了。

在美国历史上，共有五次由国会宣战的纪录：1812年的对英国战争、1846年的对墨西哥战争、1898年的对西班牙战争、1917年的第一次世界大战和1941年的第二次世界大战。

目前离川普继任或者卸任还有两个多月，在这两个多月的时间里，川普必须做出选择。是继续相信溃烂的民主机制为他伸张正义，还是启动战时总统机制，一鼓作气把民主党背后的邪恶势力中共灭了？我们都知道擒贼先擒王的道理，只有把中共扳倒了，病毒才会结束，经济才能重启，正义才能真正得到申张，否则就会陷入无休无止的消耗战，而这正是中共早就设置好的陷阱。
