## 伊朗快船侵略性靠近美国船只 海岸警卫队鸣枪警告
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1203373/)

#### 编译：坐看云起时

Newsmax 5月10日报道，美国五角大楼发言人约翰-柯比（John Kirby）说，周一，霍尔木兹海峡附近一艘伴随着导弹潜艇和其他船只的美国海岸警卫队船只向以 “不安全 “方式靠近它们的伊朗快船鸣枪警告。

发言人柯比告诉记者，由伊斯兰革命卫队控制的一大群快船靠近美国船只150码（137米），这时一艘海岸警卫队的快艇用0.50口径的机枪开了大约30枪警告。他说，伊朗船只的行为 “非常具有侵略性”。

这次遭遇是美伊两国在中东地区紧张局势的最新迹象，也是最近几周美国船只第二次向伊朗船只开火。

上个月末，美国公布了美国海军火炮号向进入约68码范围的伊朗快船开火的画面。4月初，一艘伊朗船停在另一艘海岸警卫队船只前面，当时没有开枪。

柯比说： “这是不安全的，是不专业的，这种活动是可能导致有人受伤的活动，可能导致该地区真正的误判。可悲的是，伊斯兰革命卫队海军的骚扰并不是一个新现象。”

柯比说，美国船只在进入波斯湾时遭到了袭击。尽管美国和伊朗的谈判代表在维也纳参加了关于如何恢复2015年核协议的间接会谈，但还是发生了这些问题。
