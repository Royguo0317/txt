# 坦白说：美国和台湾如何从中共国手中拯救芯片产业

[!\[\]()!\[\](https://spark.adobe.com/page/5w1eODsh8l9Kr/images/ea0830a5-592b-4f52-9641-756fd1829c65.jpg?asset_id=51b6f2eb-b97b-4ad9-9456-ed3448931fca&amp;img_etag=%22ef997b6c1dd26a148a256631730d4a93%22&amp;size=1024)](https://spark.adobe.com/page/5w1eODsh8l9Kr/images/ea0830a5-592b-4f52-9641-756fd1829c65.jpg?asset_id=51b6f2eb-b97b-4ad9-9456-ed3448931fca&amp;img_etag=%22ef997b6c1dd26a148a256631730d4a93%22&amp;size=1024)

半导体是美国科技产业的“心脏”。 硅谷（Silicon Valley）的名字就源于这些芯片中的硅。

从手机到烤面包机，半导体的作用相当于任何数字系统的心脏，这使其对任何设备的安全至关重要。 美国和台湾一起主导着半导体工业的各个方面。 现代技术的所有道路，包括无线网络、人工智能和战斗机，都通往硅谷或台北。 现在，中共国叫嚣要在美国已占据的高科技王座上掺一脚从而占领一席之地。

如今，在中共国很少进行尖端的半导体制造。 用于生产芯片的专用工具，用于对芯片进行编程的软件以及用于测试的材料大部分都来自中共国以外。 然而，中共国占世界半导体消费量的60％。 巨大的中共国制造业在很大程度上依赖于采用美国软件设计、然后使用台湾半导体巨头台积电（TSMC）从美国公司采购的设备制造的半导体芯片。 台积电是中共国最大的供应商。中共国制造业及其技术的核心是美国和台湾。 这是美国必不可少的战略优势，而中国共产党（CCP）计划破坏这一优势。

中共于2014年宣布，到2025年，中共国消费的计算机芯片中有75％将在国内生产和设计，从而消除目前对离岸芯片的依赖。 中共创建了一个290亿美元的投资基金，投入到国内半导体行业。 可以肯定地假设，在半导体等重要行业中，中共国的“私人”实体即使不是全部，（也都是）在国家的指导下大规模运作。 为了推动国内产业发展，中共资助了知识产权盗窃、公司收购和人才挖角活动。 但是，中共国最大的武器是市场准入。开发尖端芯片，推动美国和台湾产业的主导地位，需要数十亿美元的研发资金。 公司依靠庞大的中国市场的收入来资助研发。 寻求市场准入的西方半导体公司已经被中共强制通过在岸制造研发和合资企业转让技术知识。

竞争激烈的中共国半导体产业不仅对美国的技术优势构成经济威胁，而且对美国和台湾等伙伴民主国家也构成了巨大的安全威胁。 允许中共国公司制造现代技术的“核心”，会很好地为中共数据盗窃和网络攻击提供数字系统武器。

半导体技术也是美国和台湾重要的杠杆手段：如果这些芯片不可用，依赖它们的中共国在岸产业不仅会退化，更会被摧毁。 这对遏制中共国在南中国海的侵略和维持台湾主权有重要意义。 如果中国大陆制造半导体并使其价格低于台湾和美国，那么全球技术将在经济上更加依赖中国，而关键技术将必定不安全。

[!\[\]()!\[\](https://spark.adobe.com/page/5w1eODsh8l9Kr/images/4a35f18d-658e-456b-be1f-2bed4b23cd4e.jpg?asset_id=e2ccc538-9b96-43af-a104-d43e2f69c5a2&amp;img_etag=%22d43a9d79eeb252203075abd2af29267b%22&amp;size=1024)](https://spark.adobe.com/page/5w1eODsh8l9Kr/images/4a35f18d-658e-456b-be1f-2bed4b23cd4e.jpg?asset_id=e2ccc538-9b96-43af-a104-d43e2f69c5a2&amp;img_etag=%22d43a9d79eeb252203075abd2af29267b%22&amp;size=1024)

中共无疑将使用其屡试不爽的真正的偷窃和强力武装手段继续进行运作。 美国必须与台湾和其他亚太盟国合作，立即采取行动，减缓中共国对半导体技术的收购进程，并实行长期行动，以确保美国继续在技术领域保持领先地位。

短期内，美国应禁止美国公司或使用美国元件的公司出售美国先进的半导体材料，给中共国或与中共有关联的实体，其中包括相关的制造工具和软件。为了弥补失去中共国市场的成本，拥有出口许可证的公司应只能向中共国销售通用芯片，而联邦政府应在数年内弥补其他损失的收入。 美国应通过半导体产业协会提出的建立一波以理工科（STEM）为中心的研发资金（R＆D），基础设施，风险投资来纠正其最近的技术投资下降趋势，其中包括为半导体研发提供500亿美元的补贴。

此外，半导体供应链必须建立在志同道合的民主国家之间牢固的经济和外交关系上。 这些联系越紧密，半导体行业将变得更加坚不可摧和具有创新力，从而保持美国和相关技术的活力，使我们领先于中国十步之遥。 这些行动不仅可以保护我们的数字设备免受恶意攻击，而且可以确保美国半导体行业的持续领先地位。
