根據美聯社2020年12月31日發布調查顯示，中共正在嚴格控制新冠狀病毒起源的所有研究，啟動大外宣並積極擴大病毒可能來自中國以外的理論。外交部也多次發言:“已在世界許多地方發現新冠狀病毒。” ，“科學家應該在全球範圍內發展國際科學研究與合作。”試圖將病毒起源推向國際。習近平2020年2月發布重要指示，要求疾病預防控制中心實驗室等其他工作人員，不得與外部機構或個人分享新冠狀病毒相關的任何數據、標本或研究成果，必須經過習近平審查批准才能公佈。



美聯社獲得內部情報並發現，中共向研究新冠病毒起源的南部解放軍科學家發放數十萬美元的經費。並監視調查結果，要求任何數據或研究成果必須經由習近平直接批准。中共內部洩密罕見的數十頁的內部文件，證實了高層授意嚴控病毒起源調查。並揭示了中共在整個新冠狀病毒大流行，保密是由上而下控制的模式。熟悉中國疾病預防控制中心CDC合作的公共衛生專家說：中共拒絕給國際科學家研究新冠病毒的訪問權限，“他們只選擇可以信任的人，可以控制的人。
