**简评：**

这则新闻报道称，北京的高管对所谓“爱国者治港”说了大实话：爱国者必须爱共产党。近期在炒作的“爱国者治港”更是被视为香港“一国两制”的正式死亡，实为党员治港的法案已在周四通过。

所谓“爱国者治港”是1984年由邓小平首先提出。新闻作者称此时的习近平比彼时的邓小平对此定义更为严苛。长期以来，中共党国一体，捆绑概念是其法宝。此次大言不惭的讲出来，对外实际释放了一个强烈的信号：《中英联合声明》已被中共废了，你们能怎么样？这是一个危险的信号，有如100年前纳粹德国突破莱茵河畔，即中共开始以明显的方式试探。

新闻还提到“改良派邓小平为了平息忧虑而提出”，实则邓当年的做法和如今一样的卑劣。只不过当时共产党要的是收回香港，所以才可以说出支持奴隶制的也算爱国者，只要支持统一。中共不但是说谎高手，还是玩弄文字的高手，它的漂亮话都只为了掩盖其不可告人的目的，按需解读罢了。时到如今，还相信中共一字半句，或者还认为此番解读“爱国者治港”和84年有所不同，那只能说还未看透中共本质。

# **中共国官员说，香港爱国主义包括对党的忠诚**

**中共国外交部驻香港官员说：“爱国者应该尊重中共国共产党。”**

中共国高级官员周二表示，忠于中国共产党是决定香港人是否为“爱国者”的关键。时值北京正在制定新法律对这个金融中心的政客们进行审查。

2019年，在大规模的、有时暴力的民主示威游行席卷了这座城市数月之后，中共国对半自治的香港采取了镇压行动。

[!\[\]()!\[\](https://spark.adobe.com/page/6ySaHS4mL8hjK/images/9e0443a8-d634-4e7b-92e5-50da08e5e593.png?asset_id=c1235a35-47ab-4ef4-ab08-8136aee6d426&amp;img_etag=%224d91d62d031c185b61aaed3b33d40324%22&amp;size=2560)](https://spark.adobe.com/page/6ySaHS4mL8hjK/images/9e0443a8-d634-4e7b-92e5-50da08e5e593.png?asset_id=c1235a35-47ab-4ef4-ab08-8136aee6d426&amp;img_etag=%224d91d62d031c185b61aaed3b33d40324%22&amp;size=1024)图片：Pxhere 摄

中共最新计划是对香港本已限制重重的选举制度进行彻底变革，阻挡任何被认为不忠诚的人，并减少那些由直接选举而产生的小部分席位的影响。

北京称之为“爱国者治港”，并称这些是恢复稳定、修补容纳“反华”势力漏洞的必要改革。

批评人士将其视为香港民主运动棺材板上的另一颗钉子，因为北京继续进行广泛的反对异议运动。

新的提案预计将在周四由中共国橡皮图章代表大会通过第一阶段审议。

一位中共国高级外交官被问及爱国主义是否将被定义为对共产党的忠诚，就像在大陆那样。这也是香港人焦灼关切的细节。

“我们讲爱国，并不是说爱一个文化或历史悠久的中国，而是爱中国共产党领导的、现在的中华人民共和国。”北京外交部驻港公署副特派员宋如安告诉记者。

[!\[\]()!\[\](https://spark.adobe.com/page/6ySaHS4mL8hjK/images/134315ea-8d9a-4a8e-b60c-05475f03cb68.png?asset_id=8f3e4246-32ea-4eb7-b2c7-d10e24ae04a6&amp;img_etag=%2232ee68ec949b9f048d7b9c6841e27cec%22&amp;size=2560)](https://spark.adobe.com/page/6ySaHS4mL8hjK/images/134315ea-8d9a-4a8e-b60c-05475f03cb68.png?asset_id=8f3e4246-32ea-4eb7-b2c7-d10e24ae04a6&amp;img_etag=%2232ee68ec949b9f048d7b9c6841e27cec%22&amp;size=1024)2017年7月1日，高官在中共国国家主席习近平面前宣誓。档案图片：香港政府官网。

他说：“爱国者应该尊重中国共产党。”

“我们将审查候选人是否符合该标准。”

**什么是爱国者？**

1997年香港从英国统治下移交时，中共国承诺其可以在“一国两制”的模式下保持一定的自治和自由。

这座城市拥有部分自己选举产生的立法机关、独立的法院和言论自由保护组织，这是独裁大陆所没有的民主元素。

[!\[\]()!\[\](https://spark.adobe.com/page/6ySaHS4mL8hjK/images/099eddb8-9ddb-41bf-a3e0-8f47de7a3a80.png?asset_id=34c1e621-0f1b-4400-962e-2ac2de694a7b&amp;img_etag=%2280bd11ac91046ee20d702e535020b19f%22&amp;size=2560)](https://spark.adobe.com/page/6ySaHS4mL8hjK/images/099eddb8-9ddb-41bf-a3e0-8f47de7a3a80.png?asset_id=34c1e621-0f1b-4400-962e-2ac2de694a7b&amp;img_etag=%2280bd11ac91046ee20d702e535020b19f%22&amp;size=1024)行政长官林郑月娥（右三）与五位新任命的秘书。图片：香港政府官网。

当他们被允许投票时，香港人倾向于为更进一步推动民主制度或对北京持批评态度的候选人投票。

宋如安的评论反映出中共国对香港政界人士行为的容忍程度发生了转变。

“爱国者治港”一词是由中共国当时的改良派领导人邓小平于1984年提出的，目的是平息人们对移交后香港政治多元化恐将终结的担忧。

邓小平认为，爱国者必须接受香港是中共国的一部分，并支持中共国的未来繁荣，但他们并未被要求成为党的忠诚者。

“只要具备这些条件，不管他们相信资本主义，还是相信封建主义，甚至相信奴隶主义，都是爱国者，”邓小平说，“我们不要求他们都赞成中国的社会主义制度，只要求他们爱祖国，爱香港。”

但是在习近平主席的领导下，中共国明显转向了独裁主义，对“爱国者”一词采用了极为狭隘定义边界。

去年北京当局对香港实行全面的国安法之后，许多香港最杰出的政治异见人士惨遭入狱、被起诉或逃往海外。

宋如安说，中共国并不是试图在香港搞“清一色”。

他说：“香港是一个多元社会，融合了中西文化。”

他补充说：“但是，任何挑战国家基本制度、破坏香港宪政秩序的人都不算是真正的爱国者。”
