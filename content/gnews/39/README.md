据路透社报道，阿富汗洛加省长发言人拉旺表示，1辆装置大型炸弹的汽车4月30日在东部的洛加省（Logar）爆炸，造成至少30人丧生，约60人受伤。阿富汗官方说目前还不清楚谁是幕后策划。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/01024143/%E5%9B%BE%E7%89%8711.png)

据阿富汗卫生部一名发言人说，约40人送医救治，其中一些伤势严重。阿富汗内政部发言人阿瑞安（Tariq Arian）表示，死亡人数恐将持续攀升。

美国总统拜登日前宣布9月11日前美军将全数撤离阿富汗后，最近数周阿富汗频传暴力攻击。

德国国防部长安妮格雷特·克兰普·卡伦鲍尔于4月14日接受采访时表示，德国将配合美国从阿富汗撤军。据《泰晤士报》报道，英国也将跟进从阿撤军，撤出大约750名军人。北约官员4月29日告诉法新社：“北约盟国于4月中旬决定，将于5月1日前展开从阿撤军工作，这项工作已经开始。”

美国和欧洲国家一致决定从阿富汗撤军，集中精力应对中共的世界霸权。这边欧美国家刚刚宣布撤军计划，话音未落，阿富汗马上发生汽车炸弹爆炸。恐怖袭击的发生，恐会引起欧美国家内部对撤军的质疑。恐怖袭击得益者，就是幕后黑手。
