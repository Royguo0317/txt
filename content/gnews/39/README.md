## 白手套在中国集权之下的悲剧
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1292587/)

撰稿人：信心满满

去年10月马云在上海金融高层论坛发表演讲，马云尖锐批评中国的问题不是金融系统性风险，而是缺乏金融生态系统的风险，他还批评“把风险控制为零才是最大的风险”，批评监管当局。之后，马云掌控的蚂蚁金服上市随即被叫停，马云被约谈，随后，阿里巴巴陷入集权发动的“反垄断”风暴，还重罚了等值23亿欧元的罚款。从此，高调马云很少露面和讲话。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/02201550/1622679154.png)图片来自网络
最近，马云一手创立的湖畔大学要办300年被剔除和叫停. 从2015年建立到现在只有5年时间， 没有达到他的初衷和理想。大学设立在杭州，旨在培养商界精英，在当局强大压力下，日前已被迫改名为“湖畔创研中心”，并将调整课程内容。湖畔大学已被迫停止招生，当局对马云的湖畔大学为中国培养顶级企业家的目标充满防备和戒心。

马云一直对教育寄望很高，这位创建阿里巴巴帝国的企业家，在2019年宣布退出。他表示，退出阿里巴巴后将自己的主要精力贡献给慈善特别是教育事业。现在，湖畔大学被迫改名，他被迫辞去了湖畔大学校长。非常明显是对向往办教育的马云一个重大打击，也被认为是中国官方持续打压马云的举动。

文贵先生多次在直播提到中共白手套的可怕结局.例如， 肖建华被抓，叶简明，王健非正常死亡。共产党是一个邪恶组织，中共几个利益集团培养一批又一批它的代理商人或者白手套帮他们获得不正当巨额资产。这些都造成社会不平等和稳定的因子由于几个利益集团控制整个中国以及集团互相争斗。很显然那些白手套就会成为政治牺牲品。只有推翻共产党邪恶体制，中国人才会有平等，社会才会有稳定。白手套才会不存在。
