## 新加坡官宣为全民提供免费新冠疫苗，打还是不打？
` 新加坡狮城农场 Himalaya Singapore` [轉載自GNews](https://gnews.org/zh-hans/655115/)

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/16033403/vaccine-1.jpg)
近日， 新加坡总理李显龙宣布，新加坡将进入疫情解封的第三阶段，其中一项重要措施是：政府拨出10亿元，向疫苗制造商订购疫苗，这些厂家包括莫德纳（Moderna）、辉瑞（Pfizer-BioNTech）以及北京科兴（SinoVac），将是全球最早得到中共病毒疫苗的国家之一，政府订购的首批疫苗预计将在本月底运达。到2021年第三季度，将有足够疫苗为每一个在新加坡的人进行接种。在新加坡长期居留的每一个人都将能免费接种，但不会强制要求，接种疫苗将是自愿的。李总理强调，他本人将带领主要内阁官员率先接受疫苗注射。

消息一出，立刻引起群众热议，尤其中国的在新加坡的同胞中，讨论十分激烈。打还是不打，观点主要有以下三种：

1. 相信新加坡政府，总理都带头打疫苗了，安全性肯定没问题，跟着打。
2. 需要观察一段时间再决定，毕竟一般新药研发都需要七八年时间，这个疫苗怎么这么快？
3. 一定要打中国的疫苗。“一方面是因为中国团队很严谨，国家对人民很负责，肯定不会含糊的。另外一方面，西方媒体一向拿着放大镜找中国问题，基本是坏事大肆报道，好事不提或一带而过。而且在新加坡也没有防火墙，信息渠道很畅通，几乎没有西方媒体关于中国疫苗的负面报道，恰恰说明我们的疫苗很成功。而且中国疫苗最便宜又便于运输，简直就是第三世界的救世主，深深为祖国感到自豪。”


根据闫博士对疫苗的解读，中共国从未研制成功过新疫苗，多年来疫苗案件层出不穷触目惊心，给无数家庭带来苦难。北京科兴生物CEO尹卫东早在2016年就承认行贿负责疫苗审查的官员尹红章夫妇，后尹红章因受贿罪被判10年，而尹卫东却因与所谓与检方合作免于指控，继续执掌科兴生物。如此行为如此背景的生产商，叫人如何信任？关于西方媒体未有负面报道一说，正好说明中共国大外宣、蓝金黄西方主流媒体的手段多么成功，让我们被新闻联播洗脑的同胞即便出了防火墙，也依然遭受另一种形式的欺骗。

在新加坡今年的中共病毒疫情中，遭受伤害最深的就是新加坡的外来劳工群体，这其中有大部分就是我们中国的草根同胞们。在病毒侵袭了客工宿舍之后，他们被隔离在住宿条件很差的客工宿舍，不能工作，没有薪水，不能回家，每天看着身边的工友不断被确诊后送到医院，看着确诊数据每天大幅攀升，随后又有客工自杀的现象出现，那种孤独和绝望可想而知。而此时，中共国高官无一人感染，那个让你自豪的祖国在哪里呢？为什么可以全世界大撒币，却不能派专机接我们的同胞回家？
