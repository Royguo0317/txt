# **针对寻求英国签证的香港人的虚假宣传运动**

[!\[\]()!\[\](https://spark.adobe.com/page/knGXQouep5ah3/images/6d6e1fc1-00ca-4839-b769-ca9b074d75b8.jpg?asset_id=4d29cc70-5b27-47b1-a55d-054814d792d1&amp;img_etag=%2202aff35de79930057331e7f72529e4f8%22&amp;size=2560)](https://spark.adobe.com/page/knGXQouep5ah3/images/6d6e1fc1-00ca-4839-b769-ca9b074d75b8.jpg?asset_id=4d29cc70-5b27-47b1-a55d-054814d792d1&amp;img_etag=%2202aff35de79930057331e7f72529e4f8%22&amp;size=1024)

“离开香港将是一个糟糕举动”的言论，已经升级为“人们有可能变成无国籍”的威胁。

英国天气恶劣。税收很高、工作机会稀缺、人们并不那么友好，移居英国还可能使你的中共国国籍受到威胁。

至少在亲中共国的香港报纸社论和在线聊天室、以及香港高级官员嘴中，我们常常会听到这样的描述。当英国首相鲍里斯·约翰逊(Boris Johnson)去年7月宣布英国将为人们开放成为英国公民的门户之时，有人就开始鼓噪散布为什么迁居将是一个糟糕的决定。

最近几周，随着英国国民（海外）护照持有者获得长期签证的窗口被打开（这是迈向公民身份的第一步），这种言论已变得暗淡起来，甚至还出现了申请签证的香港人可能最终会变为无国籍的威胁之辞。

虽然无法证实这些充斥在留言板上的言论是否经过协调，但中共国历来以开展影响力行动而闻名。无论如何，聊天室和社论都会被官方放大宣传，力图阻止香港人离开这个亚洲金融之都。

这个面向前英国殖民地人群发出邀请之际，北京方面正在侵蚀香港的政治自由，颁布了《国家安全法》，并取消了被认为不够爱国的立法会议员的资格。

英国对冠状病毒的应对混乱。英国死亡人数高达11.8万人，而香港的死亡人数不到200人。并且，英国经济去年遭遇了1709年以来最严重的衰退，这些都是可能让香港人斟酌的潜在原因。当然，英国的税收确实更加高昂。

但是，论坛上也有报道称，移居英国的人在购房时被骗，甚至被中共国帮派追杀。北京和香港的主管部门也都隐晦地发出了有关家庭被拆散风险的警告。香港政府没有立即回应置评请求。

香港的前领导人甚至提议说，那些持有BN(O)护照的人应该失去中共国国籍。梁振英对当地一家媒体说，香港人不应该“脚踏两只船”。行政长官林郑月娥在上个月接受彭博社采访时说，她不明白为什么会有很多人想移居英国。

香港此前也出现过民众外迁潮，包括1989年北京天安门广场大屠杀后以及1997年香港移交给中共国统治之前。

1997年6月，在英国正式将香港移交中共国前不久，香港街头举行了民主示威活动。

但是，对于北京来说，这一波迁离浪潮的出现正值敏感时期，因为北京正试图巩固其对香港的控制，并防止不同政见者在国外继续其政治诉求的努力。当然，也有出于财政的考虑：美国银行(Bank of America Corp.)估计，随着居民前往英国，香港今年可能会有高达360亿美元的资金外流，这相当于2020年香港净流入资金的一半。

英国政府预测，根据该计划，未来五年内可能会有30万人前往英国。英国政府将在5月公布第一批申请者的数据。香港政府的一位高级顾问曾表示：“据现有数字推测，英国政府严重高估了移民人数。”

英国称，国家安全法“明显和严重违反”了1984年保障香港高度自治和自由的条约。中共国表示，这些措施对于防止香港发生进一步的暴力抗议活动是必要的。迄今为止，已有近100人因该法被捕，而在现在，爱国主义课程教育将强制覆盖最小仅六岁的学生群体。

到目前为止，中共国并未如其所威胁的一般行动。北京决定不承认BN(O)护照为旅行证件，这在很大程度上是象征性的，因为人们可以凭借居住证离开香港。同样，香港银行业协会也表示，不会接受这些护照作为身份证明。据报道，北京还在讨论是否禁止BN(O)护照持有者担任公职。

但一些潜在的签证申请者担心，北京当局可能会取消他们的中共国护照，这将使重新入境香港变得更加困难。一位30岁不愿透露姓名的研究人员（化名Adrian）说：“这是最大的顾虑。”Ardian说，他决定先等一两年再申请英国签证。

彭博新闻社采访了十几位寻求利用该计划的居民，他们都重申了自己迁移的决心。不过，根据移民顾问和网站“再见香港，你好英国”的创始人BillyWong称，这些言论还是动摇了一些人的决心。

Wong说：“一些中产阶级和富有的香港人希望通过我的服务获得BN(O)签证，但由于政界人士采取反制措施的威胁，现在他们有了新的想法。”他补充说，在近30名客户中，有少数客户后来暂停了他们的申请。

亲北京的资深政客转而发表评论文章，称居民应在中共国国籍或外国国籍之间做出选择，并警告那些申请签证的人最终可能成为难民。

香港大学专门研究中国宪法的教授陈洪义表示，鉴于中共国法律不承认双重国籍，剥夺BN(O)持有人的中共国国籍在法律上是可能的。“这并不是中共国独有的，很多国家都遵循不允许双重国籍的原则。”

英国外交部发言人说，在香港的英国双重国籍人士已经不再获得领事支持，他们已敦促当局恢复。他们没有评论如果中共国撤销BN(O)持有人的双重国籍，英国将采取什么措施。

加拿大政府得知，一名被囚禁并持有加拿大护照的香港居民被要求宣布他是中国人还是加拿大人。加拿大政府对此表示关切。这说明北京可能已准备好采取类似措施。澳大利亚和英国已经更新了旅行建议，警告拥有双重国籍的公民，中共国可能只会承认他们的中共国国籍。

香港观察者(Hong Kong Watch)组织创始人迪克特·罗杰斯说，还有一事令人担忧，香港当局拟将禁止人们提取国家养老基金。“这些都是英国应该关注的问题。一旦发生，英国应该做好应对的准备。”

一些消息团体呼吁外国停止干涉香港。在LIHKG上，一个名为BNOChallenge的话题标签呼吁人们撕毁他们的BN(O)护照。一位用户写道：“我们不需要外国势力。”BNOChallenge话题在论坛上获得了超过1000赞。不过，另一用户写道：“他们并没有能力知道我是否有BN(O)护照。”

亲北京团体和香港新闻机构在社交媒体上的评论和帖子都聚焦于英国。一位用户问道冠状病毒封锁期间该国如何迎接移民，而其他人想知道，鉴于其脱欧前后流露出的关于移民的言论，新移民可能会受到怎样的待遇。

一个名为“香港沉默的大多数”的亲北京团体创始人RobertChow在该组织的社交媒体平台上写道：“你怎么能肯定去英国会对香港人有好处？”

当地支持香港人到英国定居的团体担心，抵达英国的人会受到既有华人社区的敌意对待。根据1月下旬一份来自三个社区团体的联合报告，一些人报告被跟踪或骚扰。

不过，一位拥有BN(O)身份的前建筑师上月携家人抵达英国。他获得了六个月的签证。他表示，他在英国感觉受到了欢迎。

他说：“当然有困难。工作、住宿、不同的文化、不同的语言，只要做好准备就好。但如果你以积极的态度看待它，就能发现这里的许多机会。”
