# **日本准备在新立法中允许其海岸警卫队向中共国船只开火**

**正在审议的新立法将从根本上改变目前日本海岸警卫队在日本海域与外国船只接触的政策，这可能会在不经意间导致该地区爆发一场涉及中共国的激烈冲突。**

新的法律提案是在中日两国船只在临近台湾且颇富争议的尖阁诸岛（恰好台湾也对这些岛屿宣示过主权）附近发生更频繁、更激烈的冲突事件之际提出的。目前，日本缺乏一套机制可以在与登陆尖阁诸岛的中共国渔民发生任何纠葛时启动自卫队，这可能会升级至涉及中共国的军事巡逻。

但这种情况可能会发生变化，尤其是在北京当局最近允许自己的海岸警卫队军事化之后，这一举动旨在试图阻止其地区竞争对手对东海和南中国海岛屿的主权宣示。东京方面正准备加强自己的即时和快速反应的能力，《日经新闻》（Nikkei）详细介绍了新立法的细节：**“日本政府表示，《警察义务执行法》(Police Duties Execution Act）允许船只向船只开火，以阻止未经授权的登陆。”**该报进一步解释说：“如果警察或海岸警卫队无法做出充足的回应，那么内阁的一个电话和一个快速的决定可以调动自卫队参与警卫行动。”

[!\[\]()!\[\](https://spark.adobe.com/page/bXVNwiYG9kbAT/images/02ca442c-a1fc-47ef-b1e8-7283fd38403b.jpg?asset_id=7926dad6-f84f-4298-9679-1c83dc9eefd2&amp;img_etag=%22dab6b13d9703a2913100c443fad8720c%22&amp;size=2560)](https://spark.adobe.com/page/bXVNwiYG9kbAT/images/02ca442c-a1fc-47ef-b1e8-7283fd38403b.jpg?asset_id=7926dad6-f84f-4298-9679-1c83dc9eefd2&amp;img_etag=%22dab6b13d9703a2913100c443fad8720c%22&amp;size=1024)

而对于为什么日本认为加强其交战守则有所必要，《日经新闻》补充了如下阐述：

*参与了上周防卫政策定案的日本自民党（LDP）议员们颇有不忿，并提出了一份“填补空白”的立法建议。*

*该文件要求修改日本的海岸警卫法。**修改后将允许海岸警卫队船只在国际法范围内，对拒绝遵守驱逐令的外国船只使用武器。***

*这些议员还在推动制定法规，允许日本地面自卫队提前部署到偏远岛屿，以缩短应对敌情的反应时间。*

这将使目前尚被称为“灰色地带”事务——还达不到允许敌对交战的程度——提升至潜在的冲突局势，并有效地**允许海岸警卫队的武器化，以抵御敌方对岛屿（如尖阁诸岛涵盖的岛屿）的突然占领。**

[!\[\]()!\[\](https://spark.adobe.com/page/bXVNwiYG9kbAT/images/74df0a64-6453-4af5-aeec-d4b93e2a4b0b.jpg?asset_id=a98d0017-24c6-4955-abb2-e1a546d439b0&amp;img_etag=%223a36d76dab59e1673c24564909f860cc%22&amp;size=2560)](https://spark.adobe.com/page/bXVNwiYG9kbAT/images/74df0a64-6453-4af5-aeec-d4b93e2a4b0b.jpg?asset_id=a98d0017-24c6-4955-abb2-e1a546d439b0&amp;img_etag=%223a36d76dab59e1673c24564909f860cc%22&amp;size=1024)尖阁诸岛，在中共国被称作钓鱼岛

据《日经新闻》报道的描述，在日本官员内部一种普遍的新兴观点是，中共国确实准备大幅扩大其在尖阁诸岛和其他有争议岛屿的存在感，企图占据控制权。

该报道称，这很可能导致日本海岸警卫队遭遇“噩梦般的情形”，具体内容如下所示：

*一艘中共国渔船在日本尖阁诸岛附近抛锚。中共国在其声称对这些岛屿拥有主权，并称之为“钓鱼岛”的情况下，将指示自己的海岸警卫队保护这艘渔船。渔民们不顾日本的警告，在其中一个岛屿上登陆等待维修零件。在紧张和混乱中，惊慌失措的中共国海岸警卫队队员开始向他们的日本对手开火。*

*在中共国屡次入侵尖阁诸岛附近海域的情况下，这样的场景已不再是考虑范围之外。日本执政党内部再次引发讨论，认为有必要立法明确规定这种情况下的交战守则。*

中共国多年来一直就这些一个多世纪以来争议不断的岛屿向东京发出警告，虽然美国已经正式承认了日本对这些无人居住岛屿的主张，而拜登此前也重申了美国根据《美日安保条约》第五条保护这些岛屿的承诺。

尽管日本自1895年起就对这些岛屿宣示主权，但中共国，特别在上世纪70年代，开始言辞强硬地重提主张，这引发了一场危机。2012年日本政府从私人所有者手中购买了其中三个争议岛屿，危机由此加剧。该地区被认为潜藏有丰富资源，包括可能存在的石油和天然气储备，同时被认为是绝佳的渔场，并靠近关键的航道。
