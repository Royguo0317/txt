郭先生答应一位93岁的战友，今天要换上“红装”，这位大姐刚从中共国被救出来。

首先，郭先生请教科学家。过去郭先生曾经问过闫博士一个深刻的问题，如果让她重新选择，她是否还会选择她的先生作为伴侣。闫博士当时的回答是肯定的。但现在，世界因为病毒，变成了这样，那么多人因此失去了生命，而闫博士因为勇敢地揭露病毒真相，成了全世界的天使和英雄。

闫博士不止一次地说过，她因为被香港运动所感染，然后当得知病毒真相的时候，就勇敢地给路德先生爆料，而路德先生竟然用耐心，在短短几天内，就把病毒的原委搞明白了。路德当时成为了连接科学家和爆料革命的桥梁，而法治基金、法治社会和爆料革命的战友，成了保护科学家的屏障。可以说，科学家的新生命是爆料革命、新中国联邦给的。此时此刻，郭先生问闫博士，有没有想到过今天看到的这个改变世界的局面？

闫博士说，她开始爆料时真的没想过会到看到今天，当她1月16日再次确定了病毒真相后，1月17日，还是去了实验室，再次验证无误，她用了半个小时反复问自己，是不是要做这件事，结果她告诉自己，一定要做，如果不这么做，未来会后悔一辈子。

郭先生问科学家，如果重新回到那天，是否会做出一样的选择。闫博士十分坚定地说，就算回去一千次，她的选择不会变。但是她真的没想过会有今天，因为她知道只要开始爆料了，她随时随地都可能被消失，这是非常现实的问题。因为当时在香港，看到过香港的抗争如此残酷，香港在短短的时间内变得如此不堪，所有人就会明白这点是逃避不掉的。

郭先生接着问，如果重新去选择被爆料人的话，是否依然是路德。闫博士说：是的。

郭先生转头问了路德，今天的闫博士与当时爆料的闫博士有什么变化。

路德先生十分坦诚地说闫博士有了脱胎换骨的变化。路德先生说当时给他爆料的人很多，而且全世界的目光都聚焦在115的贸易协定上，当他爆出了闫博士的信息后，那一次他是真的害怕了。

今天，闫博士已经有了质的变化，因为她不再关注镜头，而是全身心地投入，来表达内心的愤怒和良知。而路德先生的减肥也完全证明了他的了不起。今天，郭先生让路德先生再来评价下如今的郝海东和叶钊颖，有什么感想。

路德先生说，郝海东和叶钊颖夫妇，已经深刻地理解了中共真正害怕他们的原因，就是因为体育是中共最重要的用来洗脑的精神毒药之一，他俩身上凝结了所有中共宣扬的体育精神，当他们站出来，就是对中共最大的伤害。

郭先生还评价了中共为什么喜欢挖人家的丑事，而且爆料说，昨天在澳门有场腥风血雨的较量，可能一个月以后大家会看到一些内幕。共产党最终完全地败下阵去，就证明了我们的实力。之前有内部战友提醒郭先生，共产党准备的关于路德的黑料会十分劲爆。而郭先生的观点是，如果这些丑闻是真的，那么路德现在做的这些，就更能说明爆料革命的伟大，因为把一个过去如此不堪的人，转变为匡扶正义的英雄。同样，对郝海东夫妇、闫博士的策略都是这个，郭先生都一笑置之。真正的原因是，共产党在郭先生这里吸取了一个教训，因为造谣对它们的反作用力要远远大于对于当事人的伤害。

郭先生接着问了闫博士对于她先生的感情是否依然如初。闫博士说如果回到当时相识的时刻，依然会选择他。但是在今天，两个人因为对病毒这件事上价值观的分歧，都无法让感情回到过去。如果有机会，还是想当面问问他，当一个人面临如此大的事件的时候，他为什么会做如此的选择。闫博士说到此处，动容地流泪了。她说，如果不是因为共产党，他们不会经历如此大的被迫的考验。郭先生对镜头前的闫博士的先生说，他知道他正在看着这一幕，而且他曾经的爱人还是如此温柔。郭先生问这些问题的原因，是因为这可能每个家庭都遭受过的重创、威胁还有考验，但不是所有人都能支撑下来，家人背后的支持才是我们走到今天的动力。爱情是有条件的，信仰是改变感情的一切根基。而所有拥有信仰的人，都值得我们称之为英雄。

郭先生评价了今天到场的所有外国友人，他们是坚定的战友，特别是弗林将军的到场，就是让中共军方看到，美国整个权力的变革即将发生。

面对全世界各个农场呈现的视频，闫博士想说，这所有心血和作品，正好说明了中国人的能力。但为什么中共治下的国家，现在却回到了大清时期，为什么中共治下的百姓那么懦弱和畏惧。就是因为制度，我们看着香港从世界顶尖位置一瞬间崩塌，这种冲击对闫博士这样的人，是如此大的震撼。现在当大家要面对病毒真相的时候，她希望我们能将勇气带给更多的中国人，一定要将无辜的中国百姓从诛心的环境中拯救出来，义不容辞。

路德先生作为爆料革命的老战友之一，看着今天这场盛宴，内心十分感慨，一是因为病毒的特殊情况，二是因为组织真的十分有序。这种震撼力，是能直击内心的，要让战友们自觉自发的去传播，这是文贵先生开创的灭共新时代。

接着，郭先生请闫博士谈一下福奇博士的邮件曝光之后，她的心情是怎样的。郭先生爆料说，那个护士的弟弟是黑龙江当地的一个经侦局长，还有个亲戚是绥芬河的官员，当她站出来爆料之后，全家全部消失。武汉也有个爆料人，１２月底就给福奇发了邮件，结果也是全部消失。所以闫博士能离开香港，今天想起来是不可思议的，但背后是无数人的鲜血和生命为代价。

闫博士说，中国从来不缺勇敢和有血性的人，但面对世界第二大经济体的时候，这种英勇的行为是无法发出声音的。活着，但不被重视，人活着，但心死了。这是最可怕的。那些曾经站出来爆料但却被出卖的人，他们的名字一定会被历史记住的。在这场推翻共产主义的革命里，所有新中国联邦人站在战场的最前线，用人心和努力，摧毁他们建筑的高墙，我们的声音已经站在了舞台中央，而且会越来越成为主流，灭共的速度会越来越快。

中共与全世界勾兑的背后，是无数人命的代价，是中共对最关键的人物的控制，当时没人能相信，但经过战友们把这些声音的无限放大，都是巨大的鼓励，没有那么多人的支持，蝴蝶的翅膀不会扇动起来。
