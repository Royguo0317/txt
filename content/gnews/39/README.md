A123リチウムバッテリーシステムーーーーHEVとPHEVの「ウォータールー」

A123システムは2001年に設立された。創設者の1人はMITの蒋業明で、リチウムイオン電池用の新しいカソード材料を提案した。 このシステムは、米国エネルギー省とAmerican Advanced Battery Allianceから15.1百万米ドルの寄付を受けた。

2007年、A123システムは、「ハイブリッド電気自動車」（HEV）および「プラグインハイブリッド電気自動車」（PHEV）用に特別に設計された新しいタイプのリチウム電池を発売した。エネルギー省の「フリーダムカー」プログラムの支援を受けて、開発のために1億200万米ドルの民間投資を受けた。これらの2つのバッテリーの寿命は10年以上で、エンジニアリングカーのバッテリーパックでは150,000マイル（約240,000キロメートル）だ。したがって、A123システムは、ハイブリッドバスおよびトラック用のリチウムバッテリーの世界最大のサプライヤーになった。 当時、欧米の大手自動車メーカーのディレクトリに含まれていたが、A123システムの「ウォータールー」にもなった。

A123システムはバッテリー分野で成功を収めているが、連邦政府、バッテリーアライアンス、および主要な自動車メーカーによって組織および推進されているHEVおよびPHEVプロジェクトは、従来のエネルギー車との競争力がますます低下している。大きな石鹸の泡が破裂したように、A123システムが落ちた。「電気からエネルギーに切り替えるほど、技術的な優位性は小さくなる」と、創設者の1人であるライリーは述べている。

ハイブリッド車の悲惨な見通しにもかかわらず、2009年8月、新たに任命された副大統領のジョー・バイデンは、自動車メーカーおよび関連する電池産業への補助金に24億米ドルを費やすと発表した。 3つの主要な自動車メーカーと3つの主要なバッテリーサプライヤーの中で、A123システムは2億4,910万米ドルを受け取り、2位にランクされた。 それでも、A123システムは11年間苦労した後、2012年10月に破産を申請した。 オバマ政権は、A123システムが破産を申請した日に、依然として100万ドル近くの助成金を提供した。 2012年10月16日の時点で、A123システムは連邦政府から合計2,860万米ドルの寄付、連邦政府の低金利融資2億4,900万米ドル、大手自動車メーカーの資金4,000万米ドル、同社の上場により約3億9,100万米ドル、合計約7億1000万米ドルを受け取った。

フィスカー電気自動車会社

ヘンリック・フィスカー（Henrik Fisker）は、フィスカー電気自動車会社の創設者であり、初期の頃はアストンマーティン用のD89車を設計したことで有名であった。 アストンマーティンを去った後、フィスカーはBMW用に2つのモデルを設計し、13台と1台しか納入しなかった。それから軍隊に目を向け、「ハンビー」の後継者としてハイブリッドジープを使おうとしたが、軍事契約の経験と資金がなかったため、再びハイブリッド高級車の設計に切り替える必要があった。

2008年、フィスカーは拡張範囲プラグインハイブリッド電気自動車（PHEV）の開発を開始した。フィスカーは外観を独自に設計し、「カルマ」を意味する「カルマ」という高級車に変身した。「カルマ」のプロモーションは、ゼネラルモーターズのエンジンを使用し、新世代のA123システムでPHEV用に特別に設計されたリチウム電池を使用することだ。しかし、その後、バッテリーの欠陥のために、フィスカーの会社はカルマを回収した。

コンセプトカーとしての「カルマ」の発売には多額の設備投資が必要となることから、フィスカーは2009年にブッシュ政権が立ち上げた「先端技術車両製造ローンプログラム」（ATVM）に焦点を当て、総額250億ドルの低金利政府融資を行った。 しかし、当時、ブッシュ政権は世界の主要な自動車メーカーを優先していたため、フィスカーは　1 セントの融資も受けていなかった。

フィスク社と元副大統領ジョー・バイデン

オバマが就任した後、フィスク社はすぐに恩恵を受けた。 2009年9月にATVMプログラムから5億2900万米ドルの融資を受けた。カルマの開発を完了することに加えて、米国で生産された電気自動車である新しい「プロジェクトニーナ」もある。 実績のないフィスカー社にとって、こんなに大きなローンを組むにはどんな魔法があるのだろうか。 推測の理由は2つある：

- フィスカー社は200万ドルを費やして、法律事務所にローン申請の審査を依頼した。この法律事務所は民主党に56万ドルのキャンペーン寄付を行い、オバマ政権はフィスカー会社にローンを提供した。
- フィスカー会社の初期投資家には、クリントン政権の副大統領であるアル・ゴア（Ai Gore）とオバマ政権に勤めたジョン・ドアー(John Doerr)が含まれます。 投資収益率を保証するためにフィスク社に融資した。


最も重要なことは、「ニーナプロジェクト」車を製造するために、フィックス会社がデラウェア州ウィルミントンにあるゼネラルモーターズ会社の廃工場を購入し、大規模な改修を行うには多額の費用がかかると主張したことである。 ウィルミントンはたまたま当時の副大統領ジョー・バイデンの故郷だ。

。

「ウィルミントンの工場には大きな利点があり、それはバイデン副大統領の家の裏庭にある。」すべての地元の政治家がこのプロジェクトに働きかけ、勝ちました。 しかし、どのような裏話があるのか​​、人々は知りません。 バイデン副大統領はメディアからの質問に直接答えず、彼の事務所はATVMプロジェクトを支援することを主張しただけで、故郷の工場が連邦政府によってどのように支援されたかについては説明しなかった。この脚光を浴びて、フィスカー会社は連続して少なくとも10億ドルを個人的に調達した。

しかし、2011年3月にATVMプログラムが終了したため、フィスカー会社のローン資金は、パフォーマンス目標を達成できず、1億9,200万米ドルしか引き出しなかったために枯渇した。 実際、フィスカー社は当時1000台のカルマ車しか納品していなかった。

最も目を引く贈り物は、レオナルドディカプリオやジャスティンビーバーなどの有名人にそれらを与えることだ。

2012年10月、ハリケーンサンディに襲われ、港に保管されていた約230台のカルマ車が浸水し、そのうち16台が発火した。 同時に、そのバッテリーサプライヤーA123も破産に直面していた。 2013年4月、フィスクは破産に直面した。
