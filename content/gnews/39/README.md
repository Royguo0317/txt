## 港妈的呼喊

回忆这些心路历程时，就像刚缝上的伤口，再次撕裂般的痛。
作为一个香港人母亲，过去的日子不知道情绪崩溃多少回，不知道流了多少眼泪，不知道感动多少次。我不知道我该如何去用文字表达我所有的情感，我只想对所有香港的孩子说包括我的孩子，你一定要平安回家！希望您们平安和快乐，有一个美好的未来。天佑香港！天佑所有天下善良的人！
夜已经很深、很深。不知道黑夜的无尽，心是如此的荒芜。风一阵阵地吹，片片愁绪随风飘飞。曾经的美好早已化成灰。
我不知道香港人哪里做的不对？也许在这生活的人才能够体会，失去家园民主、自由、法治的滋味。当全世界的人因为疫情都在家躲藏的时候，香港大街上鲜血仍在流淌。
看见他们抗争后只留下疲惫和血泪。孤寂的心谁来安慰？
也许风在伤心，我却内心在流泪。让我痛的如此心碎⋯⋯
对于我这个把所有时间都放在工作和照顾孩子的女人。很少有时间上youtube和社交平台。人生能遇上智者，是一种上天的恩赐。也许是天意，大约2017年4月或5月期间，偶然间看到七哥在YouTube 一段视频推送。之后每一期我都在关注。跟随七哥快四年，让我的人生更加睿智和懂得更多的知识。
原以为，孩子长大了。可以重新找回自己曾经的梦想。2019年4月初辞去多年的工作，计划着旅行休息一年。于是背上背包开始我的背包旅行。
由于旅行上的艰辛和饮食习惯，身体有点累。决定回港休整一段时间再重新出发。5月底回到香港。6月初在休整期间，我报了一个月课程。我想也许它是我未来发展的方向。原以为就这样，等风来，看夕阳。岁月静好，何尝不想呢？是谁打扰了我的灵魂？让我在睡梦中醒来看清这世间。
6月9号这一天，对于我这个政治冷感的人，可我不得不人生第一次走上街头去游行。是为了香港的未来和孩子的未来，我应该发出自已的声音说不⋯⋯

2019年6月9日开始后，冒着烈日继续走上街头，一百万人、二百万人⋯⋯平日里白天继续上课，晚上都一直看网上直播。
有时候也请假去上街抗争。恶梦般的日子慢慢的到来⋯⋯
情绪也越来越糟糕，每天处在忧郁中。7月中课程结束后，身体慢慢垮下来了。在家走路都要扶着走，躺着也天晕地转。只能躺在床上看直播。几乎泪水中度过（不让儿子看见）。整整病了一个月。
当然，在这期间，我非常注意儿子的情绪，在他这么敏感的年龄段。需要跟他交流。希望他告诉我能帮助他什么，有什么想不通或者不明白的问题。可他不想表达太多。只说老师已经在课堂上跟同学们沟通过。
我知道儿子有时候在房间躲着哭。有时候情绪有点激动。
＂妈咪，我出去踢球＂。＂妈咪，我跟同学出去玩＂。＂妈咪，我跟同学去骑单车＂。？？?真的吗？
＂仔，如果你要出去游行，我尊重你，但是你未成年，我是监护人。如果你真要去游行，请告诉我一声，我一定要跟在你后面。＂
＂真的吗？如果我不告诉妳，妳怎么知道我在哪里？＂
＂我希望你能告诉我，反正我有办法知道你在哪里＂
＂妈咪，妳别拖累人，妳跑不快。＂
＂我跑的没你快，我可以保护你呀！＂
儿子微微一笑走进房间⋯⋯

岁月两个字，有些缠绵，有些斑驳。绽放着我所有的悲和喜，那些曾历经的景象，那些拼搏的日子，那些动力十足，谁都无法阻挡的动力⋯⋯生活中的苦，对我来说算什么。曾以为自己是坚强的女人
可是面对着发生在眼前的一切，一天又一天，一周又一周。一月又一月。精神近乎崩溃边缘。
我没有宗教信仰，但我有自己的信念和信仰。
我问神，爱在哪？善在哪？为什么那么多的谎言和凶残充斥着这个世界？为什么人性是那么的无情和残酷？我多想我能爬出去保护那些年轻的孩子啊！有多少回在黑暗崩溃哭泣中呼喊着七哥啊！希望他运用所有的力量去结束这一切的悲惨。救救这些年轻的生命啊！世界上的神都去哪了？？？！！！
我知道必须调整精神状态，因为我还有儿子，不能疯掉。在病情稍为好转之后。在不舍中我再次背上我的背包回到沙漠⋯⋯

再次回到这个沙漠边陲城市，也许能让我得到片刻心灵的平静。可我只是呆呆的坐在房间里窗台上，抱着双腿眺望着。
一天又一天。偶尔的时候也会到阳台上跟来自世界各地的旅友们闲聊。每当别人问我来自哪里时。当我回答说，来自香港。
他们都会问我，现在香港怎样了，发生什么事。我都用我的烂英语尽力去告诉他们我所看到的一切。每当讲到情绪哽咽之时，他们都会过来拥抱我。给我精神上的支持和鼓励⋯⋯
孩子们呀！你们真的让我感动，让我真正的去了解香港的社会价值观。你们的正义，你们的善良，你们的坚持。真的感动上天。可是上天睡着了。还没有来得及去拯救你们的生命。
孩子们呀！你们展现出来的勇气和正义。正是他们邪恶的展现。正是正义和邪恶的彻底斗争。这些能用金钱去收买吗？
我想问问，还为那些恶警黑警吶喊的人，假如你的孩子只是去表达自己的要求时，就被打的头破血流。甚至毁了一生失去生命。你们会痛心吗？你们的良知去哪了？我想问问，这些孩子是罪犯吗？是杀人犯吗？罪犯不是要经过法庭的审判吗？
谁有权利？狗官政府也没有权利这样去做！
每当回忆起那些片段，那些画面。就好像刀刺向我的心脏。我无法想象走在前线那些孩子的心理状态。孩子们，让我拥抱一下你们吧。呜呜⋯⋯

直播上看着香港局势越来越严峻。流血事件越来越多。半个月后，拒绝朋友的劝说。决定回港，我对朋友说，我是香港人，我必须和他们站在一起。如果每一个人出一点点小力量，就会汇成大海般的力量。
回港后，依然面对现实的残酷。几乎毎天大街上战争般硝烟弥漫，让人无法呼吸。年轻人的一次次对峙，一次次抗争。面对恶警黑警的枪林弹雨，孩子们手中只有一把雨伞来保护自己。被抓的人也越来越多。流血事件每天都在发生。让我痛心至极。
我太绝望了，香港人的脸上已经没有了笑容。是谁剥夺了我们香港人表达自由的权利？是谁贱踏我们的自由和法治精神？[港人治港，高度自治！一国两制，五十年不变]去你妈的！谁还相信你？现如今不是赤裸裸的谎言吗！
某一天，我对儿子说。如果我失去了你，我也不可能活下去了。我会扛上石油气罐从楼上扔下去。然后自杀。死，我也要带上几个刽子手一起垫背。
我不支持暴力，也不支持极端手段。可是面对如此的暴政，如果儿子没了，家也就没了。我又能怎样做？有多少家庭因政见的不同而分裂？有多少失去孩子的父母在痛哭？每天，有多少香港人在流泪？
没有暴徒！只有暴政！难道街头上不是每天在上演吗？
看着战友们刚上传的MV，罗大佑「未来的主人翁」这首歌。眼泪情不自禁的流下⋯⋯

记不起多少个无眠的夜晚，流泪到天明。中文大学熊熊烈火的那一夜。我想大部分香港人都是无眠。已经晚上快11点了，中大传来封路。我发现儿子出去还没有回来。我着急的不停打电话，一次、二次、三次⋯⋯依然没人接听。过了半小时，我终于接到儿子的来电。电话里传来警察开催泪弹的声音。
＂妈咪，我求你不要打电话，你会让我分心。＂
＂仔，你在哪里？＂问了几次后，
儿子支支吾吾的说：＂我在送饭，在XX和几个同学一起，我很快回去，我安全，你不要给我打电话。＂
我的天啊！那个地方正在危险之中。但是我只能对儿子说：＂宝贝，一定要小心！＂
这一刻，我能叫儿子回家吗？我知道有多少孩子饿着肚子冲在前线在抗争。
我唯一能做的，祈求上天保佑这些孩子的安全。
一直坐在沙发上，看着纲上直播，一直在流眼泪⋯⋯
凌晨一点多，儿子回来了，满脸疲惫。冲上前拥抱一下儿子，谁也没有说一句话。儿子静静的洗完澡回房间了⋯⋯
我也一直流泪到天明⋯⋯

有时候，我真的不想回忆太多过去。回忆那些惨烈的场面，内心是无比的痛心和心碎。可我能忘记吗？每次听到那些歌，那怕一个音符！看到那些图片。眼泪就会不停的流下，我知道，永远都忘不了⋯⋯
2019年11月12日的晚上，是我这前半生中最最崩溃的日子，也是一个难忘的生日之夜。网上直播着理工大学学生抗争的场景。楼下每天晚上10点呼喊着.＂香港人加油！＂的年轻人也一直在呼喊。我匆忙的换上衣服冲出家门。到了楼下，我哭喊着拦住一个年轻人说：＂快点找领事馆、找议员，找人帮忙，帮助理大的学生，太危险了。现在是人道危机！你们也要小心保护自己！＂
年轻人说：＂妳平静一下，现在已经有议员去帮助他们，妳现在不能过去，全封路了，妳冷静一下，回家吧。＂
徘徊在楼下，真的爱莫难助。叫天天不应，叫地地不灵。蹲在路边嚎嚎大哭。上天呀！为什么这样！为什么！
恨死这些黑警，为什么要这么残酷的对待我们香港的孩子。他们是最有思想、最有素质，最优秀的学生。他们本应坐在课堂上学习，他们原来过着平静的生活。
为什么他们会这样坚持的冒着生命危险走上街头！就是因为他们坚持维护着自由！民主！法治！这是香港人一直拥有的权利！也是人类本应该有的权利！⋯⋯
在最剧烈的日子里，我永远记得一个片段。就在理大大火燃烧抗争的第二天，为了援助理大的学生。在油麻地附近的年轻人对抗着黑警疯狂的镇压。我数了一下当时前线勇武的人数，大约30人左右。头上只有4个人有安全帽！手上仅有的是一把雨伞在顽强抗争。能不心碎吗？能不心疼我们的孩子吗？难道还不能感动上天吗？死的心都有，太绝望了。可是我知道，生活还是要继续。我一定要看到这些刽子手最终的下场。我要活着，好好的活着⋯⋯

抗争持续一年多，没想到今天7月1号又再次站在铜锣湾街头抗争，感慨万千，心中都是悲凉。这个独裁的恶法，让香港人窒息。违背民意、强奸民意、绑架民意。看着一个个年轻的面孔被押上囚车和进监狱，内心真的在流血。 从来不说粗口，但是在这些日子已经用尽所有的粗口诅咒着这些纳粹！你们卑鄙的手段，难道以为香港几百万人是傻瓜吗？你们以为香港人就轻易放弃抗争吗？你们以为香港人跟猪一样让你们宰割？做你们的春秋大梦去吧！！你们根本不了解香港人的价值观。

何时我才能脱离这种痛苦的情绪？也许等待着ccp倒下的那一天吧！我只是一个普通的女人，没有过多绚丽的文笔，没有特别的口才。我只能用粗略的文字去表达自己的感受。我怎么能不心疼同一星空下香港的孩子呢？他们的真善美怎么会不感动上天呢？站在桥上吶喊的小朋友、推着滑板车，瞒着父母亲支持抗争流着泪的小弟弟、每天晚上吶喊的年轻人、满头白发拄着拐杖挡着黑警的老人⋯⋯太多，太多感动的画面⋯⋯

**[世上没有从天而降的英雄，只有挺身而出的凡人]**

香港人”反送中运动”将成为光辉的历史。
非常感谢七哥和所有的战友对香港的帮助。没有你们背后的支持，也许香港会失去更多的孩子。每次在新闻上看到海面又浮起尸体，又有不明原因跳楼，我的心就会痛多一次，恨也增加更多。我没有很大的力量，也不认识什么社团。平日里只能尽我所能去做一些小事情，帮助这些孩子。哪怕送上一瓶水，一个小饭团。多想自己变成魔术师，用我的魔术棒一挥，就能打倒这些万恶残害我们孩子的凶手。把他们都打下十八层地狱。

本来好想好好的写下这一切，可是每次的回忆，都让我无比的痛心和痛苦。我想摆脱这种痛苦，可是依然无法去脱离这种悲伤。没有朋友去诉说，没有可以跟很多人去沟通。只能一个人暗然神伤。有时候不敢去看新闻，也不敢过多的去了解太多。可是，生活在这个环境，又不得不去面对。前方的路该怎样走呢？我真的迷惘。一切已经回不到以前⋯⋯只求上天保佑我们的孩子平安！我也是一位母亲，有哪一位母亲不心疼自己的孩子。可我又能说什么？前程先不说，孩子身上的伤痕和心理上的伤痕何时又能治愈？原本平静生活在校园的孩子，是谁逼着他们走上战争般的战场去抗争？是谁让他们付出青春未来的代价？我只祈盼那些魔鬼快点消失，还孩子们一个平静的生活。

此刻，有多少个远离香港的孩子同样在想家？
有多少个失去孩子的父母亲悲伤欲绝？
所有的言语、所有的伤痛、所有的感受、所有的无奈、所有的无助⋯⋯无法一一去言说⋯⋯

七哥说的对，只有战胜内心的恐惧才能获取更大能量，勇敢的面对—勇气！
我相信，所有受过的伤都会成为最坚硬的地方。
孩子！抹干眼泪！

**香港人擦干眼泪！！加油！！愿荣光归所有勇敢追求正义的人！**
