本周四，瑞士传染病及流行病学专家 迪迪埃·皮特（Didier Pittet）接受采访，当被问及何时人们的生活才能恢复正常时，他回答道：“首先要知道的问题是，什么是 ‘正常’？如果说是重启一万多人的聚会，或者公众音乐会之类的活动，今年夏天是不可能的。” 他还表示，2022年的夏天可能是一个平稳时期，那时一切将会变得更容易。

与此同时，他预警大众，接下来的几周会非常艰难，所有有关部门必须做好准备。
