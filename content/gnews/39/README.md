# 米奇·麦康奈尔：不会将$2000支票、230条款废除、选举安全分开进行投票

[!\[\]()!\[\](https://spark.adobe.com/page/8HlWUdoUOb2lq/images/e16e53ef-125d-4015-9de9-e9ed7c87a1e2.png?asset_id=10488bcf-db94-4802-a2fc-fd5639fc3ef8&amp;img_etag=%22b74436863030afb416cfe40a50d39cec%22&amp;size=2560)](https://spark.adobe.com/page/8HlWUdoUOb2lq/images/e16e53ef-125d-4015-9de9-e9ed7c87a1e2.png?asset_id=10488bcf-db94-4802-a2fc-fd5639fc3ef8&amp;img_etag=%22b74436863030afb416cfe40a50d39cec%22&amp;size=1024)

参议院多数党领袖米奇·麦康奈尔（R-KY）周三表示，他不会将川普总统周末提出的，捆绑在一起的三项要求（即2000美元的刺激性直接现金发放、废除第230条款以及设立选举欺诈委员会）分开表决。

麦康奈尔在参议院发言中表示：“这就是我们的协定：参议院不会将川普总统打包在一起的三个问题分开，因为民主党人害怕解决其中的两个问题。参议院不会因为受到威胁恐吓而匆匆做出决定，将更多借来的钱交到并不需要帮助的民主党富人手中。”

麦康奈尔之所以发表上述言论，是因为众议院通过了立法，将每人直接救助金额从600美元提高到2000美元，这是最近几天通过的2.3万亿美元一揽子计划的一部分。星期二，肯塔基州共和党人阻止了众议院批准的法案进入参议院进行表决，而不寻求连任的参议员帕特·图密（R-PA）于周三投了反对票。

麦康奈尔（McConnell）周二提出了一项法案，其中包括了总统提出的三项要求。

同时，联邦政府已开始向大多数美国人寄出600美元的经济刺激支票。

600美元的金额是9000亿美元的中共病毒救济计划的一部分，该计划在国会进行了数月的谈判，由川普总统周日签署。

财政部长史蒂夫·姆努钦（Steve Mnuchin）周二晚间表示，政府已经开始通过转账将这笔钱存入美国人的账户，现金支票也将在周三开始寄出。

姆努钦说，直接注入的现金通过存入银行账户或邮寄现金支票的方式派送到居民手中，并持续到下周。
