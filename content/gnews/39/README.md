据inews24.it 记者Loris Porciello 2021年2月8日 报道，意大利萨莱尔诺市一名前护士在第二波中共病毒疫情中再次染病因此去世： 他家乡的人们对他的去世深感震惊。

他生前所在的城市的市长公布了死者的姓名： 他叫Matteo Bevilaqua ( 马泰奥•贝维拉瓜)， 64岁。2020年3月曾确诊为无症状感染者。经过一个月的隔离， 患者由阳转阴， 脱离危险。在他自愿隔离期间，将之前所接触的人员信息通知了相关机构。市长也将相关信息通知了当地有关人员， 并进行了安抚。当检测结果呈阴性后， 这位男子退休了，但就在他刚退休几个星期以后，又传来了他检测再次呈阳性的消息。而这次， 病毒比前一次更加凶猛。马泰奥开始感到呼吸困难，他不得不住进医院。住院后，他的病情急剧恶化， 这位前护士最终去世。市长将这一不幸消息告诉了当地百姓。
