**乔叔2020**

**以身作则**

**带回感性**

目前民主党候选人所采取的立场过于激进–充满了空洞的承诺和不切实际的建议，无法吸引人们的注意力。来自保守的亚洲文化的人们开始把我们的民主选举看作是一个笑话或某种真人秀。我们需要有经验的政治家来带回一些好的意识。民主党需要一个睿智的老人来团结和领导。

**最有资格**

你父亲的经验、性格、智慧、声誉和领导力使他成为总统工作的最合格人选。我们很多人都受不了川普。

**川普虽然年纪大了，但还是像个孩子一样，他是那么的**** “****盲目****“****（傲慢****+****无知）。为了他的边境墙而关闭政府，只是他思想落后和不成熟的一个例子**。**修建边境墙比撒钱还糟糕**。除了这个边境墙的提议，如果你把这堵墙算作 “基础设施”，我看不出他如何帮助这个国家更新基础设施的。

是的，非法移民，甚至是合法移民过程中的漏洞，都应该得到解决，但我们需要一个能够聪明而实际地**解决这些问题**的人。**移民改革不是一堵光怪陆离的墙的作秀**，尤其是在当今的数字世界，有了先进的交通和通讯工具，它不需要人们跳过一堵墙去委曲求全，当**骗子可以合法进入这个国家诈骗我们的社会资源时，他们不需要跳过边境的围墙**。中国人花了一百多年时间修筑和升级了**长城**来阻挡 “蛮夷”，但为什么中国当时被 “蛮夷”（蒙古人、满洲人、欧洲人、日本人等）统治和殖民了500多年，而这堵墙并没有阻止中国历史上 “蛮夷 “的各种小规模到大规模的入侵？前苏联修建了**柏林墙**，但柏林墙修建后不到30年的时间里，柏林墙和前苏联发生了什么？

川普没有关注正确的问题，也没有使用正确的方法。他的低效、无效的做事方式会把这个国家引向错误的道路。

**年龄越大，智慧越大**

说实话，以前我觉得年龄是对于乔叔的一大障碍，但现在不一样了。嘿，如果**川普**以他的年龄参选，乔叔为什么不参选？**温斯顿·丘吉尔**（Winston Churchill）在开始他的**第二任首相时已经77岁了**。此外，马哈蒂尔–我和亚洲人都非常尊敬他，但认为他太老了–**在92岁时宣誓就职**（让很多人感到惊讶）！现在我明白了，年龄只是另一个表面因素（如外表、肤色和性别），并不意味着什么。最重要的是你能为你的国家和人民做什么。
