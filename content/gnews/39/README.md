立法院领袖会见川普总统

参议院多数党领袖迈克·谢尔基（Mike Shirkey）和众议院议长李查特菲尔德（Lee Chatfield）今天在白宫与特朗普总统会晤后发表以下声明：

总统星期三晚上向我们发出了邀请，我们都接受了他的邀请，就像我们接受任何在任总统的要求在白宫会面的邀请一样。我们很自豪能有我们的同事加入我们的行列，代表密歇根来到我们国家的首都。

我们利用在白宫的时间向川普总统递交了一封信，明确表示我们支持附加的联邦资金，来帮助密歇根州对抗中共病毒，后来我们将同样的信件送给了国会领导人。

几个月前，密歇根州接受到了来自“关怀法案”的联邦资金，我们利用这些资金迅速支持了一线工人，改进测试，确保足够的个人防护设备，为失业的密歇根人提供额外的支持，并为那些因自身原因而苦苦挣扎的当地企业提供援助。我们再次面临这样一个时刻，额外的支持将大大有助于帮助那些同样需要我们帮助的居民。

我们强调，继续应对中共病毒的影响之际，我们承诺向最需要帮助的密歇根人拨付更多的联邦资金。我们还强调，在我们向前迈进的过程中，我们（也会）致力于在州的预算中承担（应有的）财政责任。

参众两院的监督委员会正在积极地对密歇根州的大选过程进行彻底的审查，我们相信委员会的程序会为我们的公民提供更大的透明度和问责制。我们尚未得到任何改变密歇根州选举结果的信息，作为立法领袖，我们将遵守法律，遵循密歇根州选举人的正常程序，正如我们在这次选举中所说的，“密歇根的认证过程应该是一个没有威胁和恐吓的从容不迫的过程。对欺诈行为的指控应该严肃对待，彻底调查。如果被证实的话，在法律允许的范围内被起诉，赢得最多选票的候选人就赢得密歇根州的选举人票，这些都是简单的道理，应该对我们的选举充满信心。”
