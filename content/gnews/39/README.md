美媒Gateway Pundit 12月18日报道，多米尼投票系统公司通过其雇佣的弗吉尼亚州Clare Locke律所的律师，致信给悉尼‧鲍威尔，要求她收回对多米尼投票系统公司的公开指控。

悉尼‧鲍威尔曾是著名的律师和联邦检察官，曾服务于川普团队，因其嫉恶如仇，勇于揭露腐败的职业生涯而闻名。她曾著书《授权撒谎》，揭露美国司法部的腐败，也曾为被陷害的弗林将军担任辩护律师。如今，鲍威尔正作为独立律师在全美多州起诉大选舞弊，并一直公开强烈谴责多米尼投票机被用做舞弊工具，该公司利用软件算法偷换川普总统的选票转投至拜登。

多米尼的律师函称鲍威尔如果不公开收回对多米尼投票系统公司的指控，她将会面临诸多涉及诽谤的法律风险。 该律师还声称：“因你的不实指控，多米尼公司遭受了巨大的伤害。员工遭受纠缠，骚扰，和死亡威胁。我们要求你立刻公开收回你的不实指控。如果你继续无理诽谤，那么这将被视为实际恶意的补充证据。”

有关多米尼投票机舞弊行为的证据已多次被曝光。在美国大选双方厮杀正酣之际，多米尼公司避免受到法律制裁的对策之一即为吓阻对其大选舞弊的指控，力推拜登上台，通过获得来自“总统”的支持而逃脱法律的制裁。
