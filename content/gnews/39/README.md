欧洲议会官员担心该团体可能与北京走得太近。

一个由欧洲立法者组成的，旨在加强欧盟与中国关系的有争议团体，该团体的主席周一表示，在POLITICO报道其活动后仅几天就被暂停。

11月26日，POLITICO报道了对欧盟-中国友好小组的担忧，此前欧洲议会官员表示担心该小组与中国政府的外交机构关系过于密切。该小组的负责人捷克议员Jan Zahradil周一在议会外国影响力特别委员会的听证会上表示，他决定在11月底关闭该论坛。欧洲议会主席大卫-萨索利的办公室证实，主席已于12月1日接到通知。

Zahradil说，他暂停了该小组的活动，”等待所有这些非正式友好小组制定新的更透明的规则”。

欧洲议会官员表示担心，因为在上个月达成的欧盟-中国全面投资协定的紧要谈判中，扎赫拉迪尔担任贸易委员会的副主席。其他担忧则涉及该集团秘书长盖林，他是中国人，也是扎赫拉迪尔的助手，在过去的15年里，他曾帮助组织了十几次欧盟立法者的中国之行，并与北京广泛的软实力机构网络建立了联系。

Zahradil驳斥了关于他的职位赋予他获取信息的特权的说法，并说他的团体的活动符合议会规则。11月时他还说，对盖林不存在安全问题。

在周一的听证会上，扎赫拉迪尔补充说，他强烈反对 “关于这个团体已经成为中国宣传工具的说法，至少不是在我的领导下。我强烈抗议任何这样的指责。”

他补充说，自2019年10月重新启动以来，该团体”没有从任何地方获得任何资金”。

不过，扎拉迪尔在11月告诉POLITICO，中国驻欧盟代表团当时为其重启活动支付了饮料和零食的费用。

捷克欧洲议会议员没有立即回应评论请求。

外国影响力委员会的立法者正在对外国在欧洲民主国家的影响活动进行为期一年的审查。

与捷克智库Sinopsis和法国智库Institut Montaigne有关的研究人员在周一的听证会上讨论了中国的影响力，提到欧盟-中国友好小组是中国国家试图干预欧洲政治的一个例子。

友谊小组是欧洲议会成员促进与第三国关系的非官方团体。根据该机构的规则，这些团体没有正式的作用，这使得立法者过去很难监测其影响力、资金和活动。

**评论：**近期欧盟议会在涉及中国的问题上表现出越来越谨慎的态度。前几周，他们要求对去年底达成的中欧贸易协议进行重新考虑，而今天这则新闻则表现出了对欧盟内部与中国有关的非官方团体的担心。再结合郭先生一直提及的北约会有所行动，欧盟官方对中共国的态度看来已经转变。
英国脱欧后，欧盟传统意义上的发动机只剩德国和法国。德国联邦国会选举将于2021年10月举行，而法国总统选举则是2022年。再加上意大利总理孔特刚刚辞职，如果提前大选，根据调查显示右翼政党领袖撒尔维尼极有可能当选。结合目前欧盟议会和北约的一系列行动，欧盟这三大国的新任执政集团将如何重新处理与中共国的关系值得期待。
