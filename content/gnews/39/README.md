# **独家新闻：新证据显示美国政府和科研机构参与掩盖新冠病毒的起源**

[!\[\]()!\[\](https://spark.adobe.com/page/asj6M8nk4jhB5/images/10d50cd4-983b-4274-a568-46bca19e4bf0.png?asset_id=459f451b-4862-49e1-a112-7c84dfd8a163&amp;img_etag=%22926876b7259aa29be4ecc06701716b91%22&amp;size=2560)](https://spark.adobe.com/page/asj6M8nk4jhB5/images/10d50cd4-983b-4274-a568-46bca19e4bf0.png?asset_id=459f451b-4862-49e1-a112-7c84dfd8a163&amp;img_etag=%22926876b7259aa29be4ecc06701716b91%22&amp;size=1024)

国际金融利益集团和全球科研机构从未想要揭露COVID-19病毒的真正来源，尽管在大流行开始的早期，病毒被怀疑来自实验室。

现在有证据表明，美国政府内部的一些人与美国科学界的一些特定成员进行了协调，毫无疑问是在国际商界和媒体的支持下，以保护中共国免受责任的影响，同时也保护了他们自己的利益。

2020年1月13日，网上已经有评论说COVID-19病毒在中共国的实验室制造出来，而且美国科学家可能帮助中共国科学家获得了知识做到这一点，特别指出石正丽博士的武汉病毒研究所实验室获得了北卡罗来纳大学的拉尔夫·巴里克博士（Dr. Ralph Baric）的帮助。

2020年1月26日，受人尊敬的调查记者和国防专家比尔·格茨（Bill Gertz）在《华盛顿时报》的文章中写道：“全球传播的致命动物传播的冠状病毒，可能起源于武汉的某个实验室，该实验室与中共国的秘密生物武器项目有关。”

2020年2月3日，共和党建制派的产物、时任白宫科技政策办公室(OSTP)主任的凯尔文·德罗格迈尔（Kelvin Droegemeier）给美国国家科学院院长马西娅·麦克纳特博士（Marcia McNutt）写了一封信，请求：“美国国家科学、工程和医学院(NASEM)快速调查信息并确定有助于确定新型冠状病毒2019-nCoV来源的数据要求，特别是从进化/结构生物学的角度。”

奇怪的是，他的担忧是由一项晦涩的、未经同行评审并且已经撤回的印度研究引起的，该研究名为“2019-nCoV刺突蛋白中独特插入物与HIV-1 gp120和Hag的独特相似性”。

回想起来，该文章似乎是科学的稻草人，暗示其他政府动机也参与其中，但尚不清楚。然而，根据应邀对OSTP的要求做出回应的科学家的意见，德罗格迈尔的调查结果是预定的，即完全免除了中共国的责任。

这些科学家包括，克里斯蒂安·安德森（Kristian G. Andersen，斯克里普斯研究所）、拉尔夫·巴里克（Ralph Baric，北卡罗来纳大学公共卫生学院）、特雷弗·贝德福德（Trevor Bedford，弗雷德·哈金森癌症研究所）、阿拉维达·查克拉瓦蒂（Aravinda Chakravarti，纽约大学医学院）、彼得·达扎克（Peter Daszak，生态健康联盟）、吉吉·格隆瓦尔（Gigi K. Gronvall，约翰霍普金斯彭博公共卫生学院）、汤姆·英格斯比（Tom Inglesby，约翰霍普金斯卫生安全中心）和史丹利·珀尔曼（Stanley Perlman，爱荷华大学）。

引用武汉病毒学研究所的一篇科学文章，说明了疾病的爆发“从当地的海鲜市场开始”，国家科学院的信中写道，OSTP明确支持中国共产党政府的论点，即大流行是一种自然发生的从动物到人类的传播。

此外，国家科学院、工程学和医学院的三位院长建议美国继续与中共国密切合作，特别是中国科学院和武汉病毒学研究所。

为国家科学院咨询做出相应的绝大多数科学家与中共国有长期合作的历史，并且/或者后来与大力支持中共国以及支持COVID-19是自然发生的事件这种理论相关联。

在一些重要的方面，国家科学院2020年2月6日最后的回应与2020年2月4日的原始草案几乎没有相似之处。(见“OSTP NAS电邮2”PDF)

根据美国国家科学院与顾问科学家之间的电子邮件往来，科学家似乎做出了明显的努力支持病毒自然发生的理论，并消除或轻描淡写关于COVID-19病毒是经过工程改造，具有独特的人类结合特性，可能会突变为更高的感染力，并具有在任何相关冠状病毒中均未发现的弗林蛋白酶多价切割位点。

随后，向美国国家科学院提供咨询的科学家响应了他们自己的公关活动，以支持中共国，并提出了COVID-19病毒是从动物自然传播给人类的理论。

2020年3月7日，彼得·达扎克和斯坦利·帕尔曼发表在英国医学杂志《柳叶刀》(Lancet)上的一篇文章指出：“我们站在一起，强烈谴责COVID-19并非自然起源的阴谋论。”

2020年3月17日，克里斯蒂安·安德森作为《SARS-CoV-2的近端起源》一文的高级作者发表文章，声称COVID-19病毒“不是一种故意被操纵的病毒”。

2020年9月21日，吉吉·格隆瓦尔是约翰·霍普金斯大学团队的成员之一，该团队撰写了一篇对中国吹哨人闫丽梦博士的批评，闫博士提供了证据证明COVID-19病毒是在实验室中产生的。

众所周知，拉尔夫·巴里克与武汉病毒学研究所的“蝙蝠女”石正丽长期合作，包括进行非常危险的“功能增强”研究的实验。

正如尼尔·帕特尔（Neil Patel）最近在《每日信号》中所描述的那样，世界卫生组织研究COVID-19病毒起源的唯一美国代表彼得·达扎克也是石正丽的亲密伙伴，并且是用美国纳税人的钱资助武汉病毒研究所的关键人物。

达扎克甚至组织了一场公共关系运动，在进行任何彻底调查之前将实验室泄漏假说描绘为“阴谋”。他的发言人后来表示，此举是为了保护实验室的科学家，但最终，也许是有意的，受益者是中共国、国际金融利益集团和全球科研机构。
