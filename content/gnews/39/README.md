**西班牙2021年1月15日电/西喜社——**联合国卫生机构的紧急情况负责人表示，英国、南非和巴西等地的中共病毒新变种的影响还有待观察，并指责人类行为导致最近一些病例数上升。

“把责任归咎于变异体，并说’是病毒干的’，这太容易了，”世界卫生组织的紧急情况负责人迈克尔·瑞安博士周五说。”嗯，不幸的是，也是我们没有做的事情造成的。”

这是指节日聚会和其他社会接触，以及一些人对公共卫生官员呼吁尊重戴口罩、保持社交距离和洗手的坚持松动。

瑞安还指出，世卫组织紧急委员会提出的新建议，建议各国暂时不应该要求入境旅行者提供疫苗接种证明。

瑞安说：”如果你看看委员会围绕旅行者接种疫苗提出的建议，它说的是’目前’。”瑞安指出，疫苗供应并不完整，科学界仍不清楚现在部署的中共病毒疫苗是否能起到防止接种者传染给他人的作用。
