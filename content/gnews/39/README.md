## 中共国营CGTN因所有权结构问题在英国被监管机构禁播

*Ofcom表示，目前的牌照持有者–星空中国传媒有限公司未能证明其对CGTN有编辑监督权*

英国广播监管机构周四撤销了中国新闻网CGTN的牌照，因为发现其国家支持的所有权结构藐视英国法律，并警告说，当再播出涉嫌刑讯逼供的电视节目，将受到惩罚。

Ofcom说，目前许可证持有人，星空中国传媒有限公司，未能证明它对CGTN有编辑监督权。而且它裁定，一个拟议的替代方案最终与中共联系在一起。

Ofcom在一份声明中说，“我们已经给了CGTN大量时间来遵守法定规则。这些努力现在已经用尽”。

“经过仔细考虑所有的事实以及广播公司和观众的言论自由权利，我们决定撤销CGTN在英国的广播牌照是合适的。”

Ofcom补充说，它将 “很快 ”结束对CGTN涉嫌违反公正、公平和隐私要求的持续调查。

这家英语卫星广播公司因长期在全球广播中模仿共产党的路线而受到批评。

在美国，CGTN是被指定为国家支持的行为者而非独立的七家中国媒体之一。

北京经常威胁要对西方国家针对其媒体业务采取的行动进行报复，这些年来媒体业务越来越复杂，覆盖面越来越广，并扬言警告要对反对中国的BBC等团体进行制裁。

CGTN没有立即发表评论，该公司于2019年在伦敦开设了一个顶级设施设备欧洲总部。

去年7月，Ofcom表示，在支持一名英国国民提出的投诉后，它正在考虑对CGTN进行制裁，该名英国国民称，他被迫做出了该广播公司播出的电视犯罪供词。

**丢失 “关键 ”信息**

2014年，彼得-汉弗莱因涉及制药巨头葛兰素史克的腐败案，被上海法院判处两年多监禁。

出狱后，他向Ofcom投诉当时的中央电视台（后来改名为CGTN）播出的两则关于此案的新闻报道。

央视以公共利益为由向Ofcom证明其报道的合理性，认为央视已经取得了汉弗莱的同意，并不知道有虐待行为。

汉弗莱指控中国当局对他下药，并将他锁在椅子上，关在一个小铁笼子里，进行非法获取和使用个人资料。

节目中包括了他似乎承认刑事犯罪的镜头，并报道了他的定罪和道歉。在这两个节目中，他的身份被确认，但他的脸是模糊看不清的。

通讯办公厅有权对违反媒体公正和公平规则的英国持牌广播机构处以罚款。总的来说，法律规定它们不得受政治机构的控制。

因此，监管机构在发现CGTN的控制实体违反了这一规定后，对其采取了最终的制裁措施。

监管机构说，“我们的调查结论是，星华传媒有限公司是CGTN服务的许可证持有人，对CGTN的输出没有编辑责任。”

“正如SCML不符合对许可服务拥有控制权的法律要求，因此星华传媒有限公司不是合法的广播许可证持有人。”

此外，Ofcom表示，它不能将牌照转让给CGTN提出的替代者，即中国全球电视网络公司（CGTNC）。

“这是因为申请书中缺少关键信息，而且我们认为CGTNC将失去持有牌照的资格，因为它被一个最终受中共控制的机构所控制。”

2012年，Ofcom吊销了伊朗国家广播公司英语频道Press TV的牌照，原因是该公司未能将总的编辑控制权从德黑兰转移到伦敦。

Ofcom称，目前的牌照持有者星空中国传媒有限公司，其未能证明对CGTN有编辑监督权。而且它裁定，一个拟议的替代方案最终与中共挂钩。
