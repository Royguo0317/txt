## 中共超限战灭港计划（三十七）六七暴动之“菠萝”与杨光
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1099742/)

搜集/撰文：岁月如歌 / 封面合成：文粤

2015年前工联会会长阳光过世 ，出殡当天一群红磡人带着横幅标语和菠萝、纸箱到红磡世界殡仪馆送这个曾经的“杀人犯”最后一程。横幅大书「大紫荆赠杨光港共鼓励杀港人」，喊口号「杨光下地狱」，有人把菠萝放在写有「同胞勿近」的纸皮箱上，暗示和讽刺当年六七暴动中共左派在香港到处埋的真假菠萝（炸弹）杀死无辜市民。当年这些一连串的袭击暴动都离不开工联会会长杨光的指使。一个双手沾满香港人沾满鲜血的刽子手，香港回归后第一届特首董建华时代政府给予他大紫荆勋章奖励，死了政府和官员高度赞扬，林郑月娥和工联会隆重举办丧礼。这是「要畀世世代代知道，共产党系会杀人放火」吗？杨光的一生最大的“政绩”是六七暴动，谈他离不开六七暴动，谈六七暴动也少不了他。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4161.jpg)
梁慕娴说杨光获得紫荆奖章，使她的眼前又浮现出林彬容颜，是不是因为「杨光获得大紫荆勋章的同时，商业电台主席何佐芝得到了次一级的金紫荆星章」吗？林彬所在的商业电台和港澳工委在那时侯可是死对头，两边发奖章是什么意思呢？学友社一次都没有提过林彬被烧死这件事，梁慕娴问领导欧阳成潮为何要搞到杀人这地步，领导回答：「这是阶级斗争的需要！」梁慕娴在那样的环境成长当时并不觉得有什么，后来移民加拿大离开组织冷静后思考人生，认清公共产党的本质之后才发现原来共产党可以借阶级斗争之名杀人。就这样商业电台播音员林彬在节目中讽刺评击左派暴行，并多次矛头指向核心人物杨光，惹来杀身之祸。 《明报》当时对暴动进行大量批评，“豺狼镛”（查良镛）也因此继林彬之后成为中共左派暗杀的二号人物。查良镛闻风急忙拖家带口逃到新加坡避难，躲过了一劫。有过这经历的他后来的日子却成为“统战胜利品”，真不敢想像。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4162.jpg)图片来自香港外国记者会
锄奸暴动队司令部公开对林彬的暗杀事实负责并污蔑林彬几条罪状。针对中共地下斗争委员会曾在暴动期间成立过战斗队或除奸战斗队，梁慕娴说这是铁一般的事实，勿用否认。当年她领导的学友社中乐组就有三个成员曾经参加过该组织。杨甘川、阿施和杰仔三人都是一所私立学校的同学，认识了香岛中学毕业生发仔，在他的介绍下又认识了中职新华银行的林某，并在林某的领导下组织学习小组。这个小组讨论组织战斗队的计划，商量如何摆放真假炸弹，响应斗委会的号召实施暴力斗争。当他们把炸弹制造好之后林某突然宣布取消行动，但是这三个人都已经准备好了不听指挥，决心要到街上摆放炸弹，在作案时遇到警察，三人赶紧逃跑，结果杰仔被抓了另外二人逃脱。杰仔被毒打喝头发水坐牢一年，出来后并没有揭发跟他一起去放炸弹的两个同学。这是杨甘川得了肝癌之后亲自打电话告诉了梁慕娴记录备案的。从这个事实可以证明中共地下党利用涉世未深的无知少年和利用暴力斗争损害香港繁荣稳定的证据。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4163.jpg)图片来自香港外国记者会
左派除了从内地偷运武器到香港之外还在香港的左派学校实验室进行制造土炸弹，《消失的档案》采访英国人希路Normal Hill证实那些摆在街边、路上、公交、电车等的炸弹是来自左派学校实验室。希路是六七暴动期间出勤最多，技术最高的拆弹专家。他曾经把炸弹原料送去化验，证实材质是来自化验室中的化学物品。当时他曾经发出过警告但未得到重视。 1967年11月27日左派学校中华中学实验室发生爆炸，警方在6楼的化验室发现一个18岁的学生被炸成重伤，左手折断头和脸血肉模糊送院后昏迷不醒， 11月28日《明报》头版头条报导了这个新闻。第二天港英政府疯了中华中学，这影响很大很多同学因此没有学校可上。当年年底时任助理辅政司黎敦义写了一份厚17页的「机密档案」，送往英国外交及联邦事务部，详细罗列各间左派学校的经营情况，并称二战大战之后香港已经有59间学校被中共控制。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4164.jpg)图片来自消失的档案
可以看出六七暴动整个过程学生是被忽悠被利用最彻底的跑在最前面的炮灰，香港市民工人也是被中共操控的为中共卖命的炮灰。从五月的暴动全面开始这些学生就被安排到假花场支持公有罢工然后又被安排到港都署门前示威，从支持罢工到罢课到上街拜访炸弹都是由这些无知的学生去完成。 50多年过去了杀害林彬的凶手还逍遥法外，而领导这场暴力的人得到了大紫荆勋章，这是个莫大的讽刺。也可以看出当今的政党是不是真正的为人民服务。梁慕娴从过去知道凶手是「××总商会斗委会」属下的 「战斗队」干的，但她没有公开，也许是出于对凶手的安全考虑，毕竟还活在人世。 《消失的档案》在播出时现场都遭到了攻击和人身威胁。
