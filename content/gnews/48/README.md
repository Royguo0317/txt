川普政府从去年开始对中共在新疆侵犯人权和强迫劳动实施一系列制裁后，中共国一家市值达数十亿美元的棉纱生产商——华孚时尚公司（Huafu Fashion Co），在今年初公开承认受到了严重影响，公司已向投资者发出警告，2020年的业绩将首次出现巨大亏损。

据《华盛顿邮报》（the Washington Post）2月22日报道，上月底，[华孚时尚公司](http://quotes.money.163.com/f10/yjyg_002042.html#01c03)（002042）在深圳证券交易所的公开文件显示，“因美国商务部实体清单制裁，多家美国品牌已取消订单和中止合作”，华孚预测去年亏损至少5430万美元，该公司2019年的净利润为6250万美元，面对 2020年预估业绩的急转直下，华孚是为数不多的公开承认受到制裁影响的供应商之一。

实际上，美国对中共在新疆侵犯人权的打击始于2019年10月，国务院宣布开始对参与其中的实体及个人实施制裁。随即海康威视等多家中共公安机构和实体被商务部陆续列入贸易黑名单； 多名参与侵权的中共官员及家属被限制签证；2020年9月，美国开始制裁中共在新疆强迫劳动生产的商品；2021年1月13日，美国海关（CBP）宣布全面禁止所有新疆棉花。1月19日，一个值得永远纪念的日子，美国国务院认定中共在新疆犯下了反人类和种族灭绝罪。

二战时期德国纳粹对犹太人的种族灭绝曾令全世界震惊，而在二十一世纪的今天，中共仍对新疆维吾尔族人施以令常人无法想象的暴行。随着西方媒体对中共罪行的更多曝光，全世界对中共的抵触首先反映在响应美国的制裁令。根据禁令，美国海关将禁止进口一切（怀疑）“全部或部分”用新疆棉制成的产品，无论是直接来自中国还是由中国途经他国周转入美的都将被禁止。

尽管新疆产棉量占世界供应量的五分之一，美国禁令的连锁反应会让全球数以千计的公司受到影响，尽管一年前，总部位于华盛顿的工人权利协会执行董事斯科特·诺瓦（Scott Nova）对此禁令表示不满，“不可能”停止用新疆棉生产纺织品。但抵制中共种族灭绝罪行动的效应似乎已迅速从美国遍及世界各地，各大世界时装品牌，如Gap、H＆M、耐克（Nike），和瑞典家居品牌“宜家”（Ikea）等纷纷响应，英、澳、日等国的企业也陆续加入队伍。中国这家市值数十亿美元的纺织原料供应商因人权问题或将彻底崩溃。
