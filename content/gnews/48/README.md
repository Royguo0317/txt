**前 言**

要诉诉苦、喊喊冤一直是我的愿望。二零一三年秋天，因烦恼事我又萌发想诉苦叫冤的念头，于是开始动笔写了我们家在中共夺权后的苦难遭遇：一九五一年春，毛泽东领导下的共产党建立了新政权，派遣土改工作队组建起来“农会”在农村实施所谓的“土地改革”，从此咱们家几代人成了被为无产阶级政权追剿的“地主”，被瓜分了土地、房屋、财产…… 还被判罚了“剥削债”。

毛泽东逝世后，乡亲们相互观望、模仿、学习……悄悄地把社会主义公社、生产队集体经营管理的土地划分为多个小组承包经营，逐渐地又分包到了个人。之后县公安局来了通知，宣布摘除咱们家在土改中被划定的“地主”成份……日子似乎让咱们家看到希望了，但那段难以抹去的历史和给我们家几代人带来的伤害又如何能够得到治愈呢？我想要将它记录下来，让有缘人也看到在这段日子里我们家是怎样“生存”下来的，让世人知道“马列主义”、“毛泽东思想” 只能够带来痛苦、灾难、掠夺、杀戮和社会动荡，它们是违背人类历史发展的畸形产物，而毛泽东就是祸国殃民的暴君……

同时也算是给自己和子孙留下一个记忆……希望咱的诉苦愿望能够实现、传载。

**(一)剥削债之父亲出逃**

俗话说“凉在风里，穷在债里；冷是风冷，穷是债穷”。我们家也曾有一笔“欠债”，虽说不知 “债主”是谁，却在短短几个小时里，从一个温饱、平静、和谐的农耕家庭落魄成负债累累、穷困潦倒的破落户，从此家里几代人遭尽了惩罚、欺凌、贫穷、剿灭、灾祸……
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/09001332/Screenshot-2021-05-08-230235-6.jpg)
*图片来源：阿波罗新闻网*

一九五一年春节过后，天气渐渐暖和，冬眠的青蛙爬进池塘，粉红的杏花挂满了枝头，小蜜蜂也开始忙碌起来，春耕开始了。一天下午，隔壁的广贤气喘吁吁地跑来我家报信，“叔，快离开，马上有民兵来抓你。”我的父亲听了立即慌张跑出了家，越过庄子河上的小桥，溜到大圩堤埂坡内的下坎里，冲庄子相反的方向的堤埂下坡间逃走了。估计在父亲逃到“大榆树”的地方，约半华里路途的时候，村长率领一队民兵冲进了咱们家，但他们扑了个空……

后来听我父亲说，他逃难到户江县管辖地方窑口庄，他自己舅舅家族的一位兄弟家中。这家男主人已经去世了，是舅奶让我父亲躲进家里的一个储藏山芋越冬、如今空闲的地窖里。舅奶儿子王春生是爸爸儿时的好伙伴，当时是某军区驻军正团级的现役军人，舅奶那是响当当的军属…… 父亲躲藏之处绝对牢靠，当时咱们家人和亲戚中，除了舅奶就唯有三姥知晓了。每天父亲只有在深更半夜的时候，才出来到院子内透透新鲜空气。

父亲躲起来第二天上午，咱家大门前的场子上聚集来了大批人马，土改工作队、村长、农会成员、民兵、贫下中农….. 大约十点左右，土改工作队宣布我们家为–“地主”。从那天起，他们要彻底地清算“地主”往日剥削来的田地、房屋、财产等。土改工作队、村长、农会成员公布了我们家田地、房屋的划分情况，接下来号召民兵和贫下中农们搬出了我们家的大、小农具、拉出了耕牛、弄出了稻囤子里稻谷，全都堆放在大门前的场子上。农会组长从农具中捡起一只粪桶、一条扁担、一只竹丝稻箩、一付稻架子、一把耘耙、—个锄头、一把旧泥锹算是留给咱们家作今后的生产使用，其余的大大小小农具、耕牛全部被瓜分出去了。被弄到场子上的那些作为“种子、吃粮”的稻谷，全部被瓜分给了田户；原来堆集在大门前场子上“稻草垛”作为耕牛饲料，随着分出去的耕牛大部分被带走了。咱们家只落下沾地的、腐烂的脚底货，和上面挡雨雪的旧稻草。家人在人们散去之后，从剩下的稻囤子脚底里挑选出稍微好一点的稻谷、作为咱们家今后种田的“种子”使，余下的用一个大簸箕存放着。

咱们家依照贫农标准重新分得到了田地、房屋：每位家庭成员有“半间房屋，二斗四升农田”。当时家里只居住八口人，奶奶、母亲、三员哥哥、两员妹妹和我。父亲是被村长和农会决定从我们家“除名”了，据说是想镇压父亲的缘故。咱们家乡邻县月巴县是属于合肥市管辖，它比柳平地区早土改一星期。父亲从家住在月巴县的大姥家被划定为地主事情上感觉到情况不妙了！在父亲出逃前三天，家里把姐姐送给了孙家做“童养媳”，所以这次分配土地、房屋时姐姐也被除名了！

在这次土改中，咱们家被分配了两间住房是瓦屋外，有一间做厨房使用的、大约十一、二个平方公尺面积，是半边返水的笔架山稻草厢房。再有一间是以往使用杂树、小元竹、灰巴、稻草搭建的，它只有一边“滴水”，这棚屋最矮处的泥巴墙不足一米高，面积不足十个平方公尺，是咱们家的大牯牛居住的“牛棚”—— 加起来为四间住房。留给咱们家有四块农田，其中只有一块“三升秧田”是强者，其余都是下等货。汪月那块“六斗五低浪田”祖传至父亲手里，已经是第五代了！它是个“五种两不收”，即使遇上好年头也只能夠弄个“跟斗”稻，但却是向来功夫、税、费……一样不少，是白送给人都没人要的沾手货。“汪巴濠”这块田是咱们爷爷使用人工、从浅浅的沟塘里堆积岀来的，它可以种植水生植物，但指望“汪巴濠”来收粮、填肚子那是等公鸡下蛋啊！胡月“一石二”这块田，栽种水稻算得马马虎虎，种小熟就不行了！

咱们家被划定为地主的当天中午，除了箩卧里的“小妹”闲着，大家都是忙碌着搬出房屋，让出给划分的贫农们。此时村长率领一伙民兵又赶来了，催讨在上午时间被土改工作队、村长和农会判罚咱们家的“三十石大米剥削债”。于是乎大簸箕里稻谷被弄去了，米缸内吃米被抵数了，大桌子、小家具、日用品、牲口、吃物….. 能够卖出去的都给贱卖了，折合成大米用来抵这“剥削债”。这些被贱卖的家产与判罚的三十石大米欠债数相比，简直就像是仓库里一个小角落。在催债的那些人撤走后，哥哥们忙碌着把家庭里的站柜、账桌、架子床、接二连三地也抬去三河寄售店去卖钱来抵债……咱家的小狗因为在瓜分财产时吼叫，被民兵们打死拖走了！

虽然地主们的财产、土地被瓜分给了贫农们，却还是令这帮人不满意、不称心，所以还要再找理由来“治罪”地主们，于是就宣传地主们长期集累的财富是从穷人那里 “剥削”来的。有了 “剥削”这个新名词，这帮人就更加隨心所欲地对这些被划成的“地主”进行拴、打、押、刑、狱、诛了！他们使用标语、歌舞、演唱、集会、宣传台喇叭等各种方式阐述、讲演：现在毛主席领导的无产阶级翻身了，穷人要“夺回”属于自己的土地和财产。甚至还把对地主们的高额罚款说成“剥削债”，把给不起的罚款说成了“欠债”不还，冠冕堂皇地把地主们的财产“合法”地掠夺了！这简直比那些土匪强盗还要卑劣无耻！（续）

注：滴水：因屋檐不是三角形，而是一面，所以只有一边可以滴水。

五种两不收：播种五次会有两次没有收成。

小熟：野高粱。
