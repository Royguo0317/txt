- **11.5. 环球时报  为何西方民主满意度更差了**


一场混乱、波折甚至引发暴乱预期的美国大选，再次“刷新”人们对西方民主的认知。这也印证了英美一些研究机构近期有关“民主满意度”的调查报告。那些报告得出大致相同的结论：在全球范围内，人们对民主的不满意程度一直在上升，并且正在达到前所未有的高峰，在发达国家尤甚。

这种结论并不令人吃惊。新世纪以来，已有许多学术成果揭示西式民主在发达国家、发展中国家以及转型国家中遭遇的尴尬局面。而各国政治家在应对接踵而来的各类危机过程中的笨拙甚至拙劣表现，更给西式民主制度抹上了滑稽色彩。进一步说，这种对西式民主的高度不满，更是对西式制度治理绩效的不满，是因这些国家民主选举产生的政府以及既有制度设计无法有效回应和解决各类紧迫问题，例如经济冲击、腐败丑闻、政策危机、跨国合作困境等。

这也揭示出一个更深刻的命题，就是一个国家的合法性不仅来自其所宣扬的价值理念，也来自其制度运行产生的实际效能。由此，我们需要更全面地认识当代世界各国的政治发展。

首先，民主是各国政治发展共同追求的理念和价值，但民主在各国应有不同的制度形态，西方式的只是其中之一，但不是唯一，更不是最终。西方式制度形态在世界范围的流行是多种原因的结果，其中不乏强制施加、简单照搬的方式。当下西式民主制度在各国普遍遭遇的困境，恰恰说明了制度并不总是能够充分体现所宣称的理念，也说明了不能简单地以“民主”之名强加或照搬制度模式，更不能将人们对西式民主制度的不满意等同于对民主价值的怀疑和放弃。

其次，制度绩效是政治发展的标尺和保障。一直以来，制度化被认为是政治发展的核心内容，由此也形成了一种“制度依赖症”乃至制度迷信，认为只要制定了法律规则、建立了组织，制度就能自动运行起来，产生预想的绩效。因此一些发展中国家、转型国家热心于所谓的制度设计、宪法工程，一些发达国家热衷于按照本国的样子改造他国制度，结果则是普遍的治理失效，政治发展遭遇挫折。亨廷顿早就断言，“当今世界各国之间最重要的政治分野,不在于它们的政府形式,而在于它们政府的有效程度。”在上世纪80年代改革开放初期，邓小平曾提出，评价一个国家的政治体制、政治结构和政策是否正确，“关键看三条：第一是看国家的政局是否稳定；第二是看是否增进人们的团结，改善人们的生活；第三是看生产力是否得到持续发展。”

第三，任何一套制度都是有生命周期的。制度是特定社会经济文化环境的产物，处理的是特定的社会关系和社会矛盾。随着环境和对象的变化，制度必须进行相应调整，才能继续发挥效力。如果不能与时俱进进行自我更新，就会出现制度衰退，但制度更新又面临着既得利益的阻挠。欧洲的“福利国家病”、美国的“选举困境”都说明了这点。福山曾多次谈到僵化的观念和既得利益格局造成美国制度功能衰退问题。在这个意义上，全球范围内的民众尤其年轻一代对西式民主的不满，就是因为它无法吸纳和整合他们的诉求。

随着各类风险叠加出现，各国政治发展又开始了新的征途。透过全球对西式民主制度的不满和抗议呼声，我们更应听到的是各国民众对更高水平制度绩效的期待。
