美国大选结局未定，在川普总统竞选团队多个州提起诉讼的情况下，多国领导人却祝贺拜登“当选”。究竟是虚伪政客的外交平衡之术还是利益共同体势力的加持？对此，华文圈网民纷纷谴责这些国家领导人的“操之过急”。

美国东部时间11月7日晚间，民主党候选人拜登及多家左派媒体宣称拜登胜选。

当晚，美国总统川普在一份声明中回应：“我们都知道为什么乔·拜登急于假扮赢家，为什么他的媒体盟友如此努力地帮助他：他们不希望真相被曝光。”**“一个简单的事实是，这次选举远未结束。”**

声明中，川普总统强调：“*拜登尚未被确认是任何一个州的获胜者，更何况在那些最有争议的州，正在被强制性重新计票，我的竞选团队正在提起有效的法律诉讼，确保选举法律得到充分捍卫，真正的胜者获得尊位。”*

*“美国人民理应拥有一场诚实选举：这意味着计算所有合法选票，而不是计算任何非法选票。这是确保公众对我们选举信心的唯一途径。因此拜登在隐藏什么？**我不会罢休，直到美国人民拥有诚实的计票，这是他们值得拥有的，也是民主要求的。*****”**

川普总统及其竞选团队就计票可疑之处在摇摆州启动了一系列法律诉讼。指控宾州，佐治亚州，密歇根州和内华达州的选票欺诈和计票作弊。

为什么绝大多数主流媒体和大科技公司掌控的社交媒体，如此一边倒地打压选举真相而一味支持拜登获胜？

为什么在没有合法程序下，仅由媒体发布拜登“当选”的新闻之后，德、加、英、法、印、日等国领导人，包括台湾总统蔡英文在内却纷纷祝贺拜登“当选”？

据大纪元11月08日报道，有网民指责：“*媒体终于说出了心声，他们要绑架民意，控制国家*。”“*媒体又不代表国家，怎么会有代表国家宣布的道理？”“不应该啊，拜登没有得到任何州的确认啊*？”

“*我真的替那些迫不及待地向拜登表示祝贺的政要们捏一把汗，大选舞弊铁证越来越多，联邦选举委员会主任都已经正式表态大选欺诈问题多多，媒体宣布的胜选无效，这些政要不会不清楚这些问题，如果川普总统最后翻盘，他们的脸往哪儿搁？*”

还有网民指， “*选择川普还是选择拜登，是判断一个人有没有人性的标准。因为你选择了乱伦、诈骗、强奸犯，你就是不可饶恕的从犯。今天居然看到英国法国加拿大欧盟的领导人祝贺拜登，拜登家族的满天飞的乱伦视频和收钱资料，看不见？*”

还有网民有其他方面的看法：“*各国领导人也不是傻子，通过这样方式上台的拜登，以后在世界事务上，不再有原来的号召力，世界再无警察了。”“墨西哥总统说现在不是祝贺的时候，显然他的智商很高。*”

据悉，墨西哥当地时间11月7日，该国总统安德烈斯·曼努埃尔·洛佩兹·奥夫拉多尔（Andres Manuel Lopez Obrador）表示，在法律问题得到解决之前，他不会祝贺拜登在美国总统大选中获胜。“关于美国大选，我们将等到所有法律问题都解决之后。”“*我不能祝贺一位候选人或另一位候选人。我要等到选举过程结束*。”

另外，川普总统夫人梅拉尼娅的故乡斯洛文尼亚，该国总理亚内兹·扬沙（Janez Janša）发推文表示拒绝承认拜登胜选，因为选举诉讼已在美国各州提出，法院还没有开始判决。

巴西总统波索纳洛（Jair Bolsonaro）至今对美国媒体宣称的大选结果保持沉默。

本次美国大选夜，川普总统在87%的计票数基础上胜出40多万张选票的绝对优势下，若干小时后，拜登选票呈现跳帧式的增长，直至三天后在多个摇摆州拜登的选票反超，主流媒体和社交媒体一边倒地宣传拜登“当选”。全美人民目击了整个过程。
