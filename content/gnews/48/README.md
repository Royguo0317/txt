#### **委内瑞拉的****Smartmatic****投票系统**
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/21105343/flags.jpg)图片来源 福布斯
Smartmatic由三名工程师– Antonio Mugica，AlfredoJoséAnzola和RogerPiñate于1997年在委内瑞拉成立，专门研究针对特定应用的技术解决方案的设计和端到端部署。该公司的利基是电子投票系统，智能城市解决方案（包括公共安全和公共交通），用于民事登记的身份管理系统以及用于政府应用的身份验证产品。

该公司的第一家美国实体于2000年4月在特拉华州成立，并于当年11月在佛罗里达州的博卡拉顿(Boca Raton)设立了总部，当时有7名员工。 2000年美国总统大选在佛罗里达州选票上被“不完全打孔票”给破坏了。关于选票计算错误的争议爆发后，Smartmatic开始针对选举投票系统的开发。

到了2003年4月，Smartmatic在委内瑞拉加拉加斯(Caracas)正式宣布了其选举自动化原型。原型的测试涵盖了任何类型的选举所必需的所有过程细节。在测试期间，重点放在系统的加密功能上，这对于机密存储和传输数据以及软件和硬件系统组件的强化性至关重要。该公司发言人说，该系统通过了所有测试，没有任何缺点。

投票系统完全由Smartmatic内部开发。这包括从设计阶段到端到端部署的硬件和软件系统的组成。这种复杂的，专门设计的技术解决方案将需要强大的系统范围的研发能力，而如果没有大规模的技术和资金支持，光在委内瑞拉是无法完成的。尽管Smartmatric于2000年在美国建立了分公司，但几乎所有产品都是在委内瑞拉开发的，委内瑞拉这个国家的资金稀缺，科研和制造水平也不高。

**主要结论/问题1**：融资和研发知识从何而来？

Smartmatic的投票解决方案于2004年8月在委内瑞拉针对总统雨果·查韦斯(Hugo Chávez)的全民公决中首次实施，并成功帮助查韦斯(Chávez)获得59％的选票。结果遭到选举舞弊的指控。当时只提出了有关选举过程和方式的问题。没有人关注Smartmatic投票系统。委内瑞拉的好人不知道那只看不见的手正在操纵其全民投票。

Smartmatic于2000年在美国佛罗里达州的博卡拉顿(Boca Raton)首次成立，然后于2004年将总部迁至荷兰的阿姆斯特丹，并于2012年在伦敦开设了新的总部。

#### **美国红杉投票系统**
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/21110102/linkage-3.jpg)
1984年，红杉太平洋系统公司收购了AVM公司（原自动投票机公司）的投票机业务，并将其重组为红杉投票系统。到红杉收购AVM投票公司时，AVM自动投票计算机（AVC）已经可以投入市场了。在红杉的所有权后，AVC于1986年和1987年在多个州获得了使用认证，并于1990年作为“红杉AVC Advantage DRE投票机”上市。

1997年下半年，得益于美国司法部的反托拉斯诉讼，红杉投票系统获得了Optech系列选票扫描仪的知识产权。它开始制造扫描投票机并开发了触摸屏。但是，经过数年的亏损，产品表现不佳。 2005年3月，该公司被Smartmatic收购，后者开发了一系列先进的选举系统，包括投票机。自那时起，Smartmatic已指派其大多数开发和管理团队致力于改造红杉的一些旧式，老爷投票机，并用专有功能替换其技术，从而开发出新的高科技产品。结果，红杉出售了许多下一代选举产品，并在2006和2007财政年度实现了健康的财务复苏。但是，在2006年总统大选中，红杉的投票系统受到质疑。

1.伊利诺伊州库克县是美国第二大人口县。使用红杉投票系统存在很多问题。怀疑问题与投票系统的软件错误有关。

2. 2000年大选后，佛罗里达州用触摸屏系统取代了打卡投票系统，但从红杉购买的触摸屏系统却存在一些重大问题。

卡罗琳·马洛尼（Carolyn Maloney， D-NY）是负责监督美国外国投资委员会（CFIUS）的小组委员会主席，并共同撰写了《 2007年外国投资与国家安全法》（FINSA） ）。她给时任美国财政部长约翰·斯诺(John Snow)的一封信，询问委内瑞拉政府是否可以利用红杉资本操纵美国大选。在此之后，美国外国投资委员会（CFIUS）才对红杉资本展开调查。马洛尼列举了一个事实，即委内瑞拉政府已向Smartmatic的关联公司投资，该公司目前的所有权被埋在了离岸信托的迷宫中，并揭露了红杉资本已将15名委内瑞拉公民飞往芝加哥，以在当地选举中参与选票计票。马洛尼说：“显然，关于这家公司的事务仍然存在疑问，只要这些疑问持续存在，许多人就会对这些投票机的完整性提出合理的疑问。” “当我首次向美国财政部提起此案时，我认为对CFIUS进行调查的时机已经成熟，因为我们投票机的完整性对国家安全至关重要。当时，Smartmatic断然拒绝接受CFIUS审查。但是现在看来，该公司似乎无法克服其围绕这一疑问的疑云–如果他们能够，我们今天不会谈论红杉出售(的话题)。正如我在5月份所说的那样，似乎CFIUS审查实际上是应该做的。 ”

该公司更换了位于博卡拉顿的总部，转而采用在多个地点设有办事处的复杂结构。美国国务院表示，其委内瑞拉的老板“仍被隐藏在荷兰和巴巴多斯的控股公司网络的背后”；它的组织是“由离岸公司和外国信托组成的复杂网络”。

**关键结论/问题3**：为什么Smartmatic从拥有佛罗里达州总部的透明结构重组为拥有荷兰和巴巴多斯的控股公司和信托的不透明结构？

2007年11月，CFIUS发出命令，命令Smartmatic出售其在红杉投票系统中的所有股份，以换取CFIUS放弃对Smartmatic的调查。但是随之而来的却远非预期。根据CFIUS的裁定后，Smartmatic被勒令出售给红杉的管理团队SVS Holdings Inc（红杉控股）。但是，Smartmatic仍然保留对红杉的多个方面财务控制权，如拥有在美国部署的红杉某些选举产品的知识产权以及对海外业务进行谈判的权利。

直到2008年4月，红杉的竞争对手Hart InterCivic试图对红杉控股进行“敌意收购”时，才揭示了这些安排。通过购买要约发现过程中使其明朗化，很明显，Smartmatic并未完全放弃红杉控股的所有权。采购合同显示了对红杉控股的永久控制权的几个要素。

- Smartmatic仍保留了红杉控股的部分财务控制权。
- Smartmatic还保留了红杉目前在美国部署的某些选举产品的知识产权所有权。
- 红杉控股保留在海外谈判非竞争合同的权利。


*“**关于**CFIUS**调查的保密性具有传奇色彩：以前，**CFIUS**是不允许将其安全审查的结果甚至其存在的事实告知国会的。**” [1]*

但是，据称这些安排是在CFIUS的审批和同意下做出的。红杉控股因侵犯知识产权和投票系统屡次失败而面临重大法律责任

**关键结论/问题4**：为什么Smartmatic保留对财务，知识产权和不竞争条款的控制权，为什么CFIUS批准这种安排？

2008年4月底，发生了令人震惊的事件：Smartmatic的创始人之一（也是美国公司的创始人）与Smartmatic的财务部门和飞机飞行员一起在委内瑞拉的一次飞机失事中丧生。委内瑞拉内政部长（据说是创始人的亲戚）与委内瑞拉选举委员会前主席，委内瑞拉前副总统豪尔赫·罗德里格斯在第一时间一起到达医院。

红杉控股公司在2010年主导了一场反亏为盈，被Dominion Voting System收购了。在2010年6月4日，以前鲜为人知的加拿大公司Dominion Voting Systems收购了电子投票硬件和光学扫描仪的制造商，该公司收购了红衫投票系统公司(Sequoia Voting Systems Inc.)。在2014年2月，Sequoia根据《破产法》第11章提出了破产申请。

沈南鹏（Neil Shen）（生于1967年）是中共国浙江省的企业家和投资者。

1985年，沈南鹏参加了上海交通大学新开设的课程的第一堂课，该课程免除了部分学生的考试要求。 1989年，他毕业于上海交通大学，获得学士学位。 1989年，他前往美国学习-最初在哥伦比亚大学攻读数学专业，后来被耶鲁大学录取。他于1992年获得耶鲁大学硕士学位。

在1992年从耶鲁大学毕业后，沈南鹏接受了美国花旗银行的职位。 1994年，他加入雷曼兄弟(joining Lehman Brothers)公司并返回中共国，在中共国负责投资银行项目。从1996年到1999年，他在香港的德意志银行担任董事，负责中共国债务资本市场的交易。 1999年，沈南鹏创立携程，担任董事长，总裁兼首席财务官。 2003年12月9日，携程旅行网在纳斯达克上市。 2002年，他创立了如家快捷酒店连锁，并担任创始人和联合主席。 2006年10月26日，如家快捷酒店连锁在纳斯达克上市。在2003年和2004年，沈南鹏亲自成为易居中国和中国分众传媒的天使投资人和董事。

在红杉资本的Michael Moritz和Douglas Leone的支持下，沈南鹏于2005年成立了红杉资本中国基金，作为美国公司的附属机构。红杉资本投资于上市公司和私营公司。它专门从事对私营公司的孵化，种子阶段，启动阶段，早期阶段和成长阶段的投资。自1972年以来，红杉资本已投资了1000多家公司，包括苹果，谷歌，甲骨文(Oracle)，英伟达(Nvidia)，GitHub，贝宝(PayPal)，领英(LinkedIn)，Stripe，Bird，YouTube ，Instagram，Yahoo！ ，PicsArt，Klarna和WhatsApp。这些公司目前的公开市场总价值超过1.4万亿美元，相当于纳斯达克的22％。它的投资组合主要集中在金融服务，医疗保健，外包和科技领域。到2017年，红杉资本已进行了68次首次公开募股和203项收购。

#### **加拿大的Dominion投票系统**

Dominion 投票系统，于2002年在多伦多成立，是一家在美国和加拿大销售电子投票硬件和软件的公司。 Dominion投票系统、Election System＆Software（ES＆S）和Hart InterCivic系统，是2020年美国大选使用的三大主要投票软件产品。

Dominion的关键人物与凯雷投资集团（Carlyle Group）相关联，而凯雷投资集团是一家美国跨国私募股权，另类资产管理和金融服务公司，管理资产超过2030亿美元。
