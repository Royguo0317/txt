## 民主党众院选情差！党内掀批斗 波洛西议长宝座动摇
` GM06` [轉載自GNews](https://gnews.org/zh-hans/532140/)

作者：GM40

国会山报（The Hill）披露，民主党中间派正暗地盘算扳倒波洛西，有勉强胜选的温和派向选民承诺不会支持她当议长，估计波洛西新任期续任议长不会一路顺遂。

美国3日举行大选，民主党在联邦众议院选情让人大失所望，至6日仍没能打倒任何一位寻求连任的共和党众议员，还被抢走数个议席，选后党团电话会议也变成批斗大会，多位中间派与温和派众议员发难，砲轰自由派与进步派把同僚害到败选。

多家媒体报导，还不确定能否连任的维吉尼亚州众议员史潘伯格（Abigail Spanberger）痛批选战策略「失败」，自由派高喊「削减警方预算」，结果害她被批为滋养犯罪，「社会主义」的标籤更让众人难以招架。

史潘伯格抨击党内高层屈服于欧加修－寇蒂兹（Alexandria Ocasio-Cortez）与她的「四人帮」（the Squad）等进步派，沮丧地吼道「别再喊社会主义」，「要是我们再用同一招打选战，2022年会他X的再被撕裂」。一位德州众议员也认为进步派高喊削减警察预算和禁止水力压裂法（fracking），让共和党对手平白捡到枪。

但众院议长波洛西不同意史潘伯格的说法，进步派也自认极左基层的热情才成功催动反川普选票。

民主党本来选前看好能斩获额外十席，但目前却落得淨减5席的下场。据指内部正在检讨原因，像民调出错、不敌共和党的「法律与秩序」、隐藏的川普选民、弹劾失败的后续效应与无法敲定新冠刺激方案等。
