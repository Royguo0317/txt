## 美议员呼吁盟国应当联手追责中共国

![]()![](https://gnews.org/wp-content/uploads/2021/02/capture-14.jpg)图片来源：International Relations EDU
据《华盛顿观察家报》2月8日报道，美国共和党议员安东尼·冈萨雷斯（Anthony Gonzalez）近日发表文章，呼吁西方盟国积极合作，采取有效措施对中共国进行追责。

在谈到盟国携手合作的重要性时，冈萨雷斯引用了温斯顿·丘吉尔（Winston Churchill）在二战时的一句名言：“只有一件事比与盟友作战更糟糕的，那就是没有盟友的作战“。 从那之后，美国及其盟国共同努力建立了经济和国际体系，为人类带来了巨大的繁荣和有史以来最长的相对和平的时光。

然而，正如前苏联以其扭曲的意识形态威胁破坏全球秩序一样，如今的中国共产党也在重蹈覆辙，而且规模更是史无前例。中共国对世界的看法与美国及其七国集团（G-7）迥然不同。中共的集权与专制使其寻求对其公民的绝对控制，并希望在世界范围内扩大影响和权力，不论从政治、军事、经济和技术来看，中共国已经成为世界上最成功的民主国家的生存威胁。

冈萨雷斯在文章里指出，在面对像中共国这样大的对手时，任何一个国家独自采取的任何措施都将无法有效地确立适当的问责制。冈萨雷斯批评一些国家领导人对中共国继续犹豫不决甚至轻描淡写的态度。他呼吁现在的美国必须告诉志同道合的盟友们，大家共同面临的巨大威胁是什么，自由世界的人民必须趋同一致地表达对这些威胁的认知以及紧迫性。

冈萨雷斯与欧洲政治领袖、德国联邦议院的诺伯特·洛特根（Norbert Roettgen）多次联合表达了对中共国外交政策的担忧。上周在七国集团举行了数月集会之后，冈萨雷斯和洛特根向各国家元首提交了一封有近70名立法者签名支持的信函，该信函称，盟国之间精准的合作应该成为应对中共国的核心，而当前各国不连贯的做法体现了民主结盟国家的软弱。没有各国之间的协同合作，中共国的势力将继续破坏全球秩序，使各国的生活方式、共同价值以及民主选举体制遭到不可逆转的改变。

同时冈萨雷斯也强调，盟国合作的目的不是为了消灭中共国，而是七国集团联盟需要坚定地维护国际体系的承诺，确保所有参与体系的成员国能够遵守约定的规则及遵循同样的标准，这样做不仅有利于问责制，也能维护近一个世纪以来保持世界相对安全的秩序恢复力。民主国家需要团结起来，捍卫共同价值观，以保护21世纪的和平繁荣。

**简评**：

中共国加入世界贸易组织（WTO）后，利用与世界的经济接轨，通过蓝金黄等手段对西方各国各个击破，导致每个国家为了经济利益不得不屈从于中共。这与二战早期英法等国因对纳粹德国的绥靖政策而导致被德国各个击破的历史惊人的相似。但是中共的邪恶远超德国纳粹，中共采用的是看不见硝烟的超限战，病毒、游客、PPE物资、强国市场、供应链等等一切都是中共国的武器。世界早就处于以超限战方式进行的第三次大战之中，过去4年爆料革命对中共进行了彻底的揭露。西方民主国家逐步觉醒，意识到中共的危害，联合起来共同应对中共是大势所趋。
