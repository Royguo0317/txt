# **11.28. 参考消息 西媒文章：RCEP标志东亚成世界经济新重心**

西班牙中国政策观察网站11月26日发表题为《RCEP：中国搭建东亚经济相互依存的脉络》的文章。全球最大贸易协定《区域全面经济伙伴关系协定》（RCEP）在全球化遭遇最大怀疑的时刻签署，是在推动自由化传统标准的美国和欧盟缺席的情况下于东亚签署的。

该协定的创新点之一是制定了统一的原产地规则，这对于证明货物的原产地和征收相应的关税至关重要。通过降低关税和制定统一的原产地规则，RCEP鼓励在该地区开展业务的跨国公司不转移到其他地区。如果说贸易战背后的逻辑是减少与中国的贸易逆差，那么RCEP就是对美国与东亚贸易平衡的最新打击。RCEP带给北京的最重要胜利是建立了区域联盟，这个联盟制衡了签署国与美国的跨太平洋关系。亚洲向世界展示了区域经济一体化的另一种做法，它将贸易自由化与处于社会经济发展不同阶段的各种意识形态和制度并存相结合。在特朗普退出跨太平洋伙伴关系协定（TPP）之后，美国被排除在地球上经济最活跃区域的主要贸易协定之外。把中国带入世界贸易组织的国家现在已经后撤，而这个亚洲巨人却成为国际贸易的推动力。拜登入主白宫当然将带来转变，但4年来失去的时间和对美国在东亚的经济外交的损害是无法挽回的。

这就是为什么西方媒体将RCEP解读为对美国敲响的警钟。有人认为，中国利用特朗普任职期间美国在国际舞台上的后撤来重新获得在奥巴马政府“重返亚太”策略下失去的空间。亚洲内部贸易日益强大，使得东亚市场对本地和外国跨国公司都更具吸引力。它庞大的消费群和较高的增长潜力与饱和、停滞的西方市场形成鲜明对比。RCEP的签署被视为东亚成为世界经济新重心这一结构性趋势的标志。

***简评：“东亚”的潜台词为“中共国“。中共不甘心即将倒台，不肯放弃中国梦，继续视美国为敌，做最后的垂死挣扎。***

**11.28. 环球时报 特朗普团队花了300万美元，结果便宜了拜登……**

据美国有线电视新闻网（CNN）11月28日报道，威斯康星州密尔沃基县在当地时间星期五公布了总统大选重新计票的结果，拜登在当地的优势小幅扩大。密尔沃基县选举委员会在周五晚上宣布，最新结果显示，在重新计票后，特朗普获得了134482张选票，拜登获得了317527张选票。相比重新计票前，拜登新增257票，特朗新增125票，拜登净增132票。

当地时间11月19日，在收到特朗普竞选团队支付的300万美元后，美国威斯康星州选举委员会下令重新统计该州总统大选中的部分选票（重新计票前，拜登在威斯康星州拥有近20600张选票的优势）。特朗普竞选团队于11月18日提交请愿书，要求威斯康星州的密尔沃基县和戴恩县重新计票。戴恩县的重新计票工作仍在进行中，截至当地时间星期五晚，特朗普净增加了68张选票，预计戴恩县将在当地时间星期日完成重新计票工作。
