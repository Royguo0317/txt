一份爆炸性文件显示，中（共）国科学家在新冠疫情大流行的五年前就开始研究将冠状病毒武器化。

据报道，美国国务院获得的文件显示，中（共）国人民解放军指挥官预测第三次世界大战将使用生物武器。

据报道，中（共）国科学家早在2015年就考虑将冠状病毒武器化。

据《澳大利亚人报》报道，美国官员获得的这份文件是由中（共）国人民解放军科学家和中（共）国高级公共卫生官员在2015年撰写，这是他们自己调查2019冠状病毒疾病（Covid-19）起源的一部分。

中（共）国科学家将SARS冠状病毒描述为 “开启基因武器的新时代”。他们鼓吹这些病毒可以 “被人为操纵成一种新兴的人类疾病病毒，然后制成武器，以前所未有的方式释放出来”。

汤姆·图根哈特（Tom Tugendhat）议员和澳大利亚政界人士詹姆斯·帕特森（James Paterson）认为，这份文件引发外界对中（共）国在新冠（Covid-19）溯源问题上缺乏透明度的重大关切。

图根哈特表示，”中（共）国对生物武器的明显兴趣令人极为担忧，即使在最严格的控制下，这些武器也是危险的。

“这份文件引起人们对那些为政党最高领导层提供建议的人的野心的重大关切。”

长期以来，中（共）国一直被控诉涉嫌掩盖或歪曲其在疫情初期的作用，人们声称中国共产党操纵病例和死亡数字，同时向世界卫生组织（WHO）隐瞒信息。

在今年早些时候，世卫组织进行了一次备受嘲笑的调查之后，该病毒的来源问题仍然存在。该组织下令进行进一步的考虑实验室泄漏可能性的调查。

大多数科学家说没有证据表明新冠（Covid-19）是人造的，但有关病毒是否从武汉的生物实验室逃逸的疑问依然存在。没有证据表明这是中（共）国故意释放的。

这份题为《非典非自然起源和人制人新种病毒基因武器》的中文文件概述了中（共）国军方有关生物武器的计划。

文中写道，”随着其他科学领域的发展，在生物战剂的施放方面也取得了重大进展。例如，冻干微生物的新发现使储存生物制剂和在袭击时将其雾化成为可能。”

众所周知，冠状病毒起源于中国武汉市。

曾为美国、澳大利亚和加拿大政府工作过的数字取证专家罗伯特·波特（Robert Potter）表示，“我们能够核实其真实性，这是一份由解放军特定研究人员和科学家撰写的文件。” “我们能够在中（共）国互联网上找到其来源。”

该文件所列出的18位作者中，有些是中（共）国公共卫生高层人士，其中包括中（共）国防疫局前副主任李峰。
