## 【香港要闻】《有线宽频》新闻部负责专题报道的「新闻刺针」小组全组被解雇，其报道曾多次获奖

编撰：文燕

审核：WLQF
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/01083738/jpg-Untitled-1-15_M9hbF_1200x0.jpg)图片：立场新闻
《有线宽频》新闻部的专题报道「新闻刺针」小组全组被解雇，其余港闻、财经、主播等各组，以及剪接和美术组等，都有员工被裁退。

据【众新闻】报道，有线新闻部出现人事地震，今日（12月1日）突然裁员40人，当中包括专责侦查报道及专题报道《新闻刺针》。公司人事部今早11时宣布消息，被裁的员工需即时离职。有线新闻发出通知后，大批有线新闻员工要求管理层解释今次裁员决定以及标准。有线中国组全体编采人员决定辞职抗议，港闻组、国际组、财经组的组主管亦辞职。

有线宽频发表声明，指今次人事调动涉及约100名员工，包括职务调动及离职。声明又指，在新冠肺炎疫情下，公司以「开源节流」、「觅求新机遇」和「控制成本」的方式维持竞争力，因此需要全面检视及调整各部门架构及人力资源。据了解，被解雇包括《新闻刺针》全体成员、有线中国组助理采访主任黄丽萍、至少两名港闻记者、剪片部主管、多名摄影师共40人。

有线新闻部在今年8月已有人事变动，3位工程部资深主管遭解雇，当时逾220名新闻部员工，包括港闻全体采主、记者以及中国组全体记者，向主席邱达昌发公开信，指3人被解雇对有线新闻是「难以挽回的损失」。

《新闻刺针》专责侦查和专题报道，其报道曾多次获奖。在反修例运动期间，《新闻刺针》曾重组721元朗白衣人袭击事件、612中信大厦外施放催泪弹争议。另外亦曾报道大厦管理围标问题、动物传心师等。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/01083730/jpg-wqeq243image001.jpg)
战友点评：

有媒体翻查资料，有线宽频集团旗下多家公司在两期保就业计划均获批补贴。当中香港有线新闻有限公司获批约2,124万，承诺受薪人数373人；有线财经媒体有限公司获批约207万，承诺受薪人数39人；i-CABLE Network Operations Limited亦获794万，承诺受薪人数152人；香港有线新闻速递有限公司获约111万，承诺受薪人数19人。笔者粗略测算，有线宽频集团属下公司，两期保就业获逾3,000万元补贴。因此，有线新闻部声称公司在新冠肺炎疫情下需要「开源节流」、「控制成本」的说法不成立，就算是也不致于急迫大量裁员。

有线中国组记者曾多次调查追縱内地流弊和维权人士消息，一向被视为有线新闻「生招牌」。 【立场】指，有线新闻是香港广播新闻机构中，少数常设固定人选的专责中国组，连主管共十一人，负责日常中国报道及晚上《有线中国组》节目，并有记者常驻北京及广州。

近年较瞩目的调查报道，包括2012 年访问八九民运人士、被监禁年期最长的李旺阳，访问播出后四天，李旺阳死于他所住的湖南昭阳医院病房，他当时倚在窗边，颈部缠绳，双腿着地，昭阳当局最初宣称李旺阳自杀，后改为「意外死亡」。李旺阳之死在香港引起极大回响，过万名市民游行，认为李旺阳「被自杀」。

2017 年，时任记者林建诚采访异见人士海祭刘晓波，过程在 fb 直播。事后公安追捕刘晓波的朋友，有线新闻所聘请的内地司机一度遭扣押三天，林建诚提早被调回香港，从此不再踏足中国采访，不久后辞职。

去年武汉肺炎爆发初期，有线中国组派出记者深入武汉报道，为少数派记者到武汉现场采访的香港媒体。

有线中国组的制作，曾多次获得人权新闻奖、中大新闻奖等奖项，获奖的报道内容包括〈核试遗害〉、〈乌坎村征地事件〉、〈李旺阳〉、〈公安也维权〉 、〈前途解密三十后〉、〈六四30〉、〈毒奶十年〉等。

因此，今天有线新闻部的人事大地震，难免令人联想到事件与「政治不正确」有关。坊间有句话：如果你当记者被人清算、被整组解雇、被警员拘捕，这证明了你是名好记者。
