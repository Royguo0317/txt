**西班牙2021年3月14日电/西喜社——**

遥望，就在云层上方更高一点的地方，犹如斯皮尔伯格拍摄的天穹的魔幻镜头。午夜时分，这座玻璃立方体的摩天大楼顶部，却丝毫没有“星球大战”里的酒馆那样的嘈杂纷乱。

香港，它的未来是注定没有黑暗的：在那个夜猫子的酒吧里，几何学解决了这个问题。在你的脚下: 云，从它那铂金色的丝网中，折射出：城市。美丽的女郎和隐约可闻的爵士乐，也许是戴安娜-克拉尔，或者是蕾妮-弗莱明正在演唱艾拉-菲茨杰拉德的杰作。城市就是这张蜘蛛网，用蓝线折射出，云朵的铂金棉，在你的脚下。辉映着手中那杯干马提尼，发出冷冷的磷光。

五年前，我就是这样看待香港的。我就会这样记住它。不是她那特点鲜明的建筑奇迹，不是兰桂坊宛如大都市的街道，不是她那犹如迷宫般的小街，那里诞生着所有文学中都可以称为传奇的小摊位。我会记得那块闪着磷光的铂金棉，那里面，有一双神奇的眼睛，像一把万花筒，不断的将各种地图的碎片重新组合，变幻万般。梦幻的眨眼，它遥不可及，却是一切一切的根本，这是一个城市的灵魂。

当时，中共已经开始，收紧绞索。而且真的很难相信，这个世界上最自由的城市，最终会被过去两个世纪以来最封建的独裁政权所吞噬。但是，在其银河酒吧的表面平静中，在其豪华购物商场的原始宁静中，在小餐馆的熙熙攘攘中，这些小餐馆可能在英国人之前就已经存在了，它们的周围生长着玻璃和钢铁的摩天大楼，使这个城市有了今天的钻石般的几何形状，一切的矛盾都能够共生，游客只能觉得几乎不可理解；

在这一切中，人们已经可以察觉到暗暗的危险。中共已经开始收紧绳索。那么今天，这个套索正套在香港人的脖子上，越来越紧。而且，在他们这种被所有西方列强抛弃的、对古老的自由之城负债累累的勇敢抵抗中，人们知道自己正在见证一个银河系最后的闪光，即将熄灭。黑洞中国要吞掉它。没有留下任何记忆。

去年6月4日，在全宇宙都被中国病毒（原文如此称呼）囚禁在家中的情况下，香港年轻人延续了一年一度对天安门英雄的致敬。他们已经这样做了31年。这一次不一样。新的治安法使他们成为罪犯。并允许中国独裁政权将他们驱逐到其境内的任何监狱。自那时起，他们的一些领导人已经逃离。加拿大和英国提供的庇护，被北京的独裁者认为是冒犯。铁钳正在向香港逼近。就像三十年前的天安门一样。我们的懦弱也是如此。

简评：

**亲爱的朋友，与你一样，多少个日日夜夜，含着热泪，回忆着香港的铂金棉，被香港的年轻人感动。但我们不弱懦，香港是我们新中国联邦人的圣城，我深深的同意您说的“香港的未来没有黑暗”。 亲爱的兄弟，让我拥抱你，和我们一起出发吧，我们要去战斗，和香港人一起，没有种族，没有国籍，只有正义，只有热血，和我们一起吧，荣光归于香港！**
