## 【香港要闻】G7敲响了中共的丧钟 外交部反咬“干涉内政”
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/980505/)

搜集/编撰：文法 / 封面：文粤

**据大纪元3月16日报道：中共修改香港制度后，七国集团（G7）和欧盟以及英国均发表声明，批评中共破坏香港自治。**中共对此反应敏感，中共驻英国大使馆再次全盘否定《中英联合声明》，中共外交部发言人赵立坚则强调“干涉内政”。
![]()![](https://gnews.org/wp-content/uploads/2021/03/封面-99.jpg)
赵立坚称，修改香港选举制度“是包括香港同胞在内的全中国人民的共同意愿”，称“香港特区的选举制度是中国的地方选举制度，如何设计、如何发展、如何完善完全是中国内政” 。

“干涉内政”一词来源于中国前总理周恩来1955年在万隆会议提出的和平共处五项原则。互相尊重主权和领土完整、互不侵犯、互不干涉内政、平等互利、和平共处原则。

中共自己的百度百科这样定义”干涉内政“：是指一国或数国为实现自己的意图，使用政治、经济、甚至军事的手段，以直接或间接的、公开或隐蔽的方式干涉另一国的内外事务，使被干涉国按照干涉国的意图行事，以改变被干涉国所执行的某种方针，政策或存在的情势。

按照这个逻辑，似乎中共在讲自己，因为中共在世界各国渗透，在干涉别国内政的例子数不胜数，比如操纵美国2020大选，缅甸军方的屠杀抗议示威群众，最近在巴基斯坦议会参议院发现的至少六个摄像头，中共不仅渗透政治、经济、宗教、科技、甚至在教育领域无所不渗透，数不胜数。而中共背离香港基本法和《中英联合声明》，破坏一国两制，修改香港选举制度，这不仅仅是干涉甚至是破坏香港自治。破坏干涉别国的前提是按照干涉国的意愿行事，七国联盟指责的是要求中共遵守的《中英联合声明》，只是要求中共遵守自己签署的法律，何谈干涉内政？
![]()![](https://gnews.org/wp-content/uploads/2021/03/image003-18.png)
**孟子曾经对“干涉内政”有过阐述**：**行仁义可以干预，为爱人可以干预。孟子的仁义观推到国家间关系上，就必然包含国际干预。仁义是一种普世价值，仁义也是没有国界的。一个国家的行事不合乎这个标准，别的国家就有权进行干涉，这也是人性的道德底线。**

仁义作为一个标准，不仅是用来衡量自己的国家统治的好坏的标准，也是用以衡量他国的一种标准，如果他国的统治者不能行仁义，那么行仁义的大国就有责任将仁义行于他国，甚至不惜用战争手段去‚解放他国民众。这也是一个有着普世价值的大国的道德底线。其关键是看民众从中是否得利。但遗憾的是香港的整个沦陷过程，没有一个大国曾经真正给予了关键性的援助，否则香港不会走到今天。香港改写了历史，也赋予香港人更大的历史使命。

同时我们应该自豪地看到: 因为爆料革命，让世界各国认清楚了中共的欺骗和内在邪恶本质，更加因为中共有意施放出的病毒令全球陷入了灾难，所以”向中共追责”是各国达成的共识。而此时此刻的G7宣言，是全球剿共阵线的宣战檄文和吹响的集结号角。
