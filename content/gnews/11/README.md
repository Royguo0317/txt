李飞飞是**“**红三代**”**吗？

所有宣传媒体、维基和百度都刻意用李飞飞家的“穷”来掩盖其背后的阴谋，并塑造出李飞飞的“不凡”；都在强调李飞飞来美的头两年家庭生活颇为拮据，甚至一度到餐馆打工，做家庭清洁工；后来还跟亲友借钱开了一家干洗店让其父母经营，以维持生计。

她的家庭真的有那么拮据吗？她父母究竟是谁？为何要掩盖其家庭情况？

查遍所有网站几乎找不到任何有关李飞飞父母的介绍，但有消息说张华川或许是李飞飞的父亲。我们可以对比分析一下张华川和李飞飞父亲的信息，看能否找到她的家族关系：

张华川，李井泉的第六个儿子，1952年12月出生于四川成都，出生后即送给中共中央驻西藏自治区代表张经武为养子。1955年张经武被授予中将军衔，调北京任国家主席办公厅主任，张华川随同到京，后就读于北京第十一中学。1971年在狱中的张经武被“四人帮”迫害致死，张经武的妻子杨岗被关进秦城监狱。张华川被抓进北京市公安局“黑帮子女学习班”强迫劳动改造，1971年获释后回到老家江西临川唱凯仓下插队落户。1973年李井泉获得“解放”，张华川被推荐到天津南开大学无线电系学习。1979年张经武被平反，80年代，张华川移民美国。

据维基和百度百科介绍，李飞飞1976年出生于中国大陆北京，在四川成都长大，12岁时，父亲远赴美国。四年后1992年，16岁的李飞飞随母亲一起赴美国新泽西州帕西帕尼-特洛伊山，与父亲团聚并移民定居。父母均为在中国大陆完成教育的知识分子，但因不谙英语，在美国不能从事工程师或科学研究工作，父亲曾做过照相机修理工，母亲曾做过收银员。李飞飞1999年以优异表现在普林斯顿大学取得物理学专业的高等荣誉学士学位，毕业后她婉拒了高盛、麦肯锡公司等投资银行的合约，前往西藏一年研究西藏传统药物。2001年，她在加州理工学院开始研究生的学习和工作，师从计算机视觉业内著名的专家彼得罗·佩罗纳（Pietro Perona）教授和克里斯托弗·科赫 （Christof Koch）教授，主要从事神经科学和计算机科学的交叉学科研究。

通过上面两段对张华川和李飞飞家庭信息的文字描述，我们可以注意到：

1）李飞飞的父亲出国时间大约在1988年左右，比李飞飞早四年左右。虽然所有资料都刻意模糊了张华川移民美国的时间，但应该是在80年代移民美国的，时间与李飞飞父亲的出国时间基本契合。

2）李飞飞出生于1976年，那时张华川的养父张经武已于1971年过世，而其生父李井泉却于1973年“解放”，所以张华川的女儿出生后跟随他生父李井泉的姓是完全合乎情理的。

3）张华川和李飞飞的父母信息都被刻意模糊。

4）张华川出生于四川，在北京上中学；而这些信息也都符合李飞飞从小出生于北京，在四川上学的经历。

5）张经武曾任中共中央驻西藏自治区代表，而李飞飞在1999年大学毕业后不是马上工作，而是去西藏学习藏医药，这个信息也高度契合。当然，学藏医只是借口，这一年，应该是李飞飞开始正式接受任务培训的一年，只是打着学藏医的幌子罢了。

通过对张经武和李飞飞父亲信息的分析比对后，我们可以发现很多信息高度契合，所以，被海内外所有媒体刻意隐藏的李飞飞父亲，极大可能就是张华川。而张华川的生父，正是曾任四川省第一任省长、四川省委第一书记，中国共产党西南局第一书记兼成都军区第一政委，大饥荒时期由于其推行的政策直接导致了几百万以致上千万四川民众死亡的元凶——李井泉。

而张华川的养父，就是曾任中共中央驻西藏自治区代表、中共西藏工作委员会第一书记和唯一一任中华人民共和国主席办公厅主任的张经武。
