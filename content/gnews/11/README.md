1949年10月1日，共匪贼首毛泽东站在北京天安门城楼上，用钢丝刷般的、激动到要哭的嗓音对被组织起来集会的打了鸡血的人群喊：“猪娃淫民拱哈瓜成立啦！猪国淫民从此站起来啦！”

这个世纪谎言到今天，已经骗了全体中国人，包括有良知的共产党员整整71年了。就在此时此刻，CCP的亲爹苏联还占领着旅顺港（大连）和控制着东北铁路；这一年，中国的GDP只有1931年的40%，这一年，全国枪口下的强制性土改才刚刚开始。

1950年6月25号，朝鲜战争爆发，在苏联诱骗下，中共不知死活的于10月贸然出兵朝鲜，三年以卵击石的愚蠢战争，造成中国军人作战部队170万人死伤，民工和后勤人员伤亡远大过这个数字，最后换来全世界发达国家对中共国长达30年的彻底的经济、政治、科技和文化封锁，整整这30年里，中国人就每吃饱过饭，自给在水深火热之中，还他妈四处输出革命，祸害四方。

1950—1978年，连续十几场席卷全国的政治运动，不仅把中国农民和城市居民的财产权利、政治权利抢劫一空，连人身自由和说话的自由也全部剥夺了！文革砸烂公检法，只不过使1949年废除民国《宪法》和民事法典《六法全书》的剧终情节，共产党嘴里的法，实质是赵家的家法和CCP领导人猪头里的想法。

农村、农业、农民是当时中国社会基本面和经济基石，中国经济90％以上是农业，而农民是CCP执政的根基，三年内战导致上千万农民死亡，参加共军的又无不是农民和农村里的游民和流民，包括我的父亲。毛腊肉喊叫的“中国人民从此站起来了”应该主要是指这5亿农民吧？你错了！中国人都错了！连美国人也错了！

1946年以来，在CCP《新华日报》上连篇的赞美颂扬美国式民主、追求宪政和言论自由的文章，不过都是欺骗和愚弄中国人的独裁权谋。

让几百万农民子弟送命的朝鲜战争刚结束，中共就出台政策限制农民自由迁徙，从农民手里抢劫粮食，在全国城市推行粮票，用城市户籍和农村户籍把5.8亿人死死的禁锢在出生地上。

刚刚从200多万地主手里抢来的分给农民的土地（实际上地主拥有的土地仅仅占全部耕地的6%），在1955年开始的社会主义改造和人民公社运动中，连同农民祖祖辈辈辛勤耕耘保留的那一点点土地，都收归了集体所有，中国3000多年的王朝历史上，农民第一次彻底失地了！牛X，CCP.

1950年1月24日，中华人民共和国刚刚成立两个多月，战场上的硝烟尚未散去，中共中央已经指示各地准备实行土地改革。1950年6月30日，中国中央政府公布《中华人民共和国土地改革法》，土改运动在全中国开展起来。与此同时还开展了“镇压反革命运动”。

此后，运动一个接着一个，遍及经济、工业、农业、文化、政治、军事各个领域，无所不在。前中国社科院政治学研究所所长严家其说：“毛泽东的运动一个接一个，不仅有政治运动、经济运动，而且还有思想运动。”

\*毛泽东生命不息运动不已\*

“三反五反”运动、“反右”运动、“大跃进”、“四清”运动、“工业学大庆”、“农业学大寨”、文化大革命、“反资产阶级自由化”、“清除精神污染”、反邪教运动，不一而足。
