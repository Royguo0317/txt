# **冠状病毒病例“ 在第三次封锁隔离的头十天没有下降”：帝国理工学院的研究发现，阳性测试的数量“几乎没有变化”——尽管官方数据显示下降**

- 在1月6日至15日之间随机检测14.2万人，发现“没有证据”表明感染病例减少
- 帝国理工学院的专家们还估计全英新冠病毒感染率R值为1.04，这意味着疫情的规模仍在扩大
- 但是上周官方的初步数据和科学估算表明，在某些情况下，封锁隔离使病例大幅下降
- 继英国疫情死亡人数最多的一天之后，又添1820人死亡


一项重要的政府实施的研究发现，在英国第三次封锁隔离期间的前10天中，冠状病毒病例并未下降——尽管一周后的大量官方数据表明，封锁措施降低了感染率。

[!\[\]()!\[\](https://spark.adobe.com/page/Fo9H1BnHmoE9M/images/d59f14f4-d92d-46a8-847a-4b99288819a2.jpg?asset_id=90d0cde1-4d17-417d-80b2-03d2e196c91f&amp;img_etag=%221e9d98c988b0d1f546d1a9842690fcf4%22&amp;size=2560)](https://spark.adobe.com/page/Fo9H1BnHmoE9M/images/d59f14f4-d92d-46a8-847a-4b99288819a2.jpg?asset_id=90d0cde1-4d17-417d-80b2-03d2e196c91f&amp;img_etag=%221e9d98c988b0d1f546d1a9842690fcf4%22&amp;size=1024)

帝国理工学院在1月6日至15日之间对14.2万人进行了随机检测，发现“没有证据”表明感染数量有所下降，而是表明在这10天的时间里，英格兰有1.58%的人感染了新冠病毒，相当于将近90万人。

专家还估计全国的新冠病毒感染率R值为1.04，这意味着疫情的规模仍在扩大，并且医院的压力将持续增加。

但是，上周的一系列数据和科学估算却与实际情况恰恰相反。剑桥大学的研究人员估计，在伦敦和东南部，R值（每个感染者传播疾病的平均人数）可能已降至0.6。

英格兰公共卫生部门还透露，尽管新冠病毒首次出现在肯特郡，那里的人们担心无法控制这种高度传染性病毒的传播，但除了80岁以上的人群外，每个年龄组每周的新冠病毒感染病例都有所下降。卫生部的数据显示，数十个区域的感染率下降了。

在此之前，继英国新冠病毒大流行最致命的一天之后，又有1820人死亡，死亡人数增至近十万，严峻程度已接近历史里程碑。

按死亡日期细分，数据显示，1月12日有1110人死于新冠病毒，这是自疫情开始以来一天中死亡人数最多的一天。每日的统计是过去24小时内记录的死亡人数，其中包括过去几天死亡而未登记的人数。

尽管英国的死亡人数不断增加，但卫生部的统计数据还显示，由于封锁隔离，英国第二波的感染潮正在继续消退。官方最新记录的新冠病毒病例是38905例，比上周三的47525例下降了18%。

在1月6日至15日之间对14.2万人进行了随机检测，发现“没有证据”表明感染率下降——尽管一系列正面数据显示出相反的结果。该图显示了每天测试的阳性感染率

伦敦帝国理工学院的研究发现，在封锁隔离的前10天，英国有1.58%的人感染了冠状病毒——相当于近90万人。该图显示了研究期间感染人数的总体趋势，其中上升趋势的红线是最近一周的数据

伦敦帝国理工学院的研究小组还揭示了在研究期间检测呈阳性的比率在英国各个地区的不同变化，东西部和西南部的比率下降，伦敦地区比较平稳

由于从感染病毒到病情严重之间需要一些时间，因此死亡人数滞后于感染人数达数周，因此直到下周才可能在死亡数字中感受到1月5日封锁隔离的作用。专家说，英国将遭受更多死亡人数破纪录的日子，数字才会降下来。

追踪爆发规模的其他研究（包括伦敦国王学院的症状追踪）声称，由于封锁，感染案例几乎立即减少。甚至卫生部的数据也表明，圣诞节前病例在伦敦下降，而其他地区则开始了第四次的隔离。

然而，由政府委托帝国理工进行的研究结果却恰恰相反。它表明疫情并没有像希望的那样放慢传播速度，相反住院人数将持续增加数周。

研究发现，1.58%的人（大约每65人就有一人）感染了这种病毒。自从12月初以来，这一数字已经激增，但与圣诞节后的最高估计值（每50人中就有一人感染）相比有所下降。

首席作者史蒂文·莱利教授说：“目前没有证据表明感染率正在下降，即使封锁隔离已有10天。这预示着我们不能期望每日死亡人数下降，也不能期望感染冠状病毒的住院病人的数量会下降。”

“先不提[病例]是否有增加，问题是没有减少新冠病毒的感染人数的事实可能会带来严重的后果。”

卫生部长马特·汉考克（Matt Hancock）说，调查结果表明“为什么我们不能在接下来的几周内放松警惕”。他补充说：“整个英国的感染率很高，这将对我们的国民保健服务体系（NHS）和医院所面临的持续的巨大压力产生连锁反应。”

“绝对重要的是，每个人都应尽自己的义务来减少病毒的感染。”

• 以色列的顶级冠状病毒医务人员之一声称，辉瑞公司病毒疫苗的第一剂效果没有达到他预期效果的一半；

• 帕特里克·瓦伦斯爵士坚持认为，目前的疫苗应能抵抗肯特出现的变异冠状病毒，但他承认，对于来自南非和巴西的变异病毒目前的疫苗是否有效仍存在“问号”；

然而，伦敦帝国理工学院的研究表明，“没有证据”表明自1月6日实行全国封锁以来，感染病例数量有所下降。

研究发现，流动性数据表明，自一月初以来人们的出行次数更多，这意味着可能需要更严格的限制。

但是，赖利教授说，监视测试程序比日常测试数据“领先”，监视测试程序表明疫情范围正在缩小。

他补充说，疫苗接种使感染率下降的效果需要“数周甚至数个月”才能显现。

合著者保罗·埃利奥特教授（Paul Elliot）说：“我们看到R值在高流行时期相当平稳。我们希望在封锁隔离期间能出现下降，我们肯定在（三月份）第一轮的隔离期看到了这个结果，但是在第三轮的隔离期却没有看到。”

该研究的作者说，对于抵抗这种传染性更强的新型变异病毒，最近的隔离似乎不如第一次隔离有效。

当被问及他是否预期两周后住院的病人比今天多时，埃利奥特教授说：“按照目前的患病率，除非感染人数下降，否则我认为答案肯定是比今天多。”在新感染病例数、住院和死亡人数之间有两到三个星期的时间延迟。

首席科学顾问帕特里克·瓦朗斯爵士将医院的当前状况比喻为“战区”状态。

他说：“目前（NHS）的情况非常非常糟糕，承受着巨大的压力，在某些情况下，就人们必须处理的事情而言，看起来像是一个战区。”

帕特里克爵士补充说：“目前的感染数字还远远没有达到必要的水平，需要进一步大幅降低——我们需要确保不懈地坚持下去。”

“如果你去公园或去其他什么地方，生活看起来还很正常。但你要是去医院，如果你在医院工作，你会发现生活完全变了样。我们现在处于一个非常困难和危险的境地，我们需要降低病毒感染的数字。”

在一个月内，冠状病毒住院病人的数量增加了一倍，迫使某些医院将患者转移到其他地方，NHS目前承受着巨大的压力。

昨天，一架海岸警卫队的直升机被用来将患者从怀特岛转移到南安普敦，以减轻当地医院重症监护室的压力。

在其他地方，英格兰南部的主要医院之一朴次茅斯女王亚历山大医院（Queen Alexandra Hospital）的儿童急诊室已经搬迁。该医院表示，这是为成年病毒患者提供更多的移交空间。

首相的官方发言人告诉记者：“我们完全清楚目前NHS所承受的压力。”

“大家已经看到部长们以前在新闻发布会上明确了这一点。这就是为什么我们继续要求公众遵循指导并发挥自己的作用，来保护NHS并挽救生命。”

尽管有三天的疫苗产出下滑，仍希望英国有效的病毒疫苗的推出步入正轨。

昨晚的官方数据显示，英国昨天共推出了近35万剂疫苗。一位政府内部人士告诉“每日邮报在线”，他们希望这意味着计划已“回到正轨”。

为了实现缓解隔离的目的，从现在起至2月15日，政府平均每天必须推出超过35万剂的疫苗，而官员们先前的乐观态度突然变得焦虑。

鲍里斯·约翰逊坚称，英国将实现他的目标，即到2月中旬为1400万脆弱人群接种疫苗。但是，国会议员对首相的行为不屑一顾。鲍里斯·约翰逊警告说，“限制供应”会使情况变得更加艰难。

三天疫苗产出下滑的根源困惑着人们，官员们说，有“太多的不确定因素” 放缓了疫苗的推出。

辉瑞的供应因工厂升级而放缓，工厂升级将持续到下个月。政府消息人士断然否认英国已经有2100万剂疫苗的说法，他们拒绝透露库存量的详细信息，称这将存在安全隐患。

消息人士称，其他因素还包括“断断续续”地物资运送，与剩余的80岁以上老人的接触以及涵盖养老院有困难。

国会议员也对分配物资的方式表示沮丧。伦敦是分配最少剂量的地方，分配措施是基于上个季节的流感疫苗的使用量，而在伦敦流感疫苗相的使用量相对较低。
