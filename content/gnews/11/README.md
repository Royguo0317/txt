## 【热点播报】美国三位联邦参议员拜访台湾并代表美国宣布赠送75万剂疫苗

作者： 纽约香草山医疗部  云豹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/01162447/gnews-logo.jpeg)



美国三位联邦参议员达克沃丝（Tammy Duckworth）、苏利文（Dan Sullivan）及昆斯（Chris Coons）率领的访问团于6月6日早上5点搭乘美军运输机C-17（RCH181）自韩国乌山空军基地起飞，于台北时间早上7时19分降落台北松山军民共用机场。达克沃丝（Tammy Duckworth）参议员在抵达台湾时的简短演讲中宣布，美国将捐赠台湾75万剂疫苗作为援助。达克沃丝说，台湾被列入第一批接种疫苗对美国至关重要，因为美国认识到台湾的迫切需要，“我们重视这种伙伴关系”。这也反映了美国感谢台湾在需要的时刻对美国人民的支持，“我们代表美国人民感谢你们”。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/06053507/45354172_303.jpg)美国参议员昆斯 图片来源：德国之声中文网

随后于8点30分，蔡英文总统在空军松山指挥部接见这三位特地来访的联邦参议员，蔡总统除了表达感谢美国赠送75万剂疫苗给台湾的及时雨行动，并表示此次特别访问团带来了美国国会跨党派支持台湾的讯息，再次展现台美坚若磐石的情谊，她代表台湾人民致上最深谢意。三位联邦参议员及访问团成员停留3小时后，于10时30分再度飞返韩国。

此次来访的民主党籍达克沃丝（Tammy Duckworth）参议员、共和党籍苏利文（Dan Sullivan）参议员是军事委员会成员，曾经多次在美国国会投票力挺支持台湾的议案；民主党籍参议员昆斯（Chris Coons）是外交委员会成员，并是《台湾保证法》及《台湾法案》的共同提案人，这三位参议员都是在美国国会坚定支持台湾的友人。其中， 昆斯（Chris Coons）参议员是美国总统拜登的好朋友，一度传出是国务卿人选。这也是继前联邦参议员陶德4月访问台湾后，第二次有“拜登超级好朋友”来台访问。

在日本于6月4日赠送台湾124万剂疫苗后，美国接着由三位重量级联邦参议员率领访问团于6月6日来台湾，并宣布捐赠台湾75万剂疫苗。这不仅仅是表达日美两国对台湾民众的医疗支持，其背后代表的是日美两国在政治及军事上对台湾的支持，这层意义更显重大。

在三年前，台湾的国际处境非常艰难，如同国际孤儿，处处受到中共的打压掣肘。从川普政府开始转变对中共的政策后，台美的外交、经济、军事关係开始慢慢升温。随着欧美日澳等国家对中共的戒心越来越加深，台湾的战略地位才又重新获得重视并开始获得来自国际社会的支持。

这次美国三位参议员率领的访问团特地抽空从韩国搭乘军机来台湾访问，并由蔡英文总统在空军的松山指挥部接见，除了在疫情上表达对台湾的支援，在政治上代表美日共同支持台湾，也代表了美国与台湾在军事联击上的深层意义。这些都是在三年前不可能发生的事。

文贵先生在2017年发起的爆料革命是促成国际社会开始改变转向的主要原因。台湾本身在过去几十年中处于中共强力打压之下，仍能坚持挺住，维护民主、自由、法治体制的决心，以及台湾人乐于布施助人的善念，都为台湾自己争取到国际友人的认同及支持。

如同郝海东先生所说：“灭共是正义的必须。”在国际灭共浪潮已经慢慢风起云涌即将掀起滔天巨浪之际，不论是台湾的民眾、中国的民众、欧美日澳纽及所有国际上爱好自由、民主、法治制度的民众，都要积极挺身而出，为捍卫自己的生命、财产、自由而战；那些原本与中共合作的各国投机人士，更要识时务地选择站在灭共这一边，这样才不会被灭共大潮所吞噬，也才有机会保住自己及家人的身家性命。正义必定战胜邪恶，这是人类及上天的準则，让我们加紧奋斗，迎接最后的胜利。
