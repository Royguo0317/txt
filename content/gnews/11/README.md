天底下有没有比空气还要重要的东西？有，是信仰、价值观和世界观。为何？连命都没了，还谈何信仰、价值观和世界观？你没听过一句话吗？“有些人活着跟死了没什么区别，而有些已经逝去的人却能永远活在人们心中。”比如，林昭、张志新、叶企孙、胡适、鲁迅……

当你不小心摔倒了，如果周围的人都来扶你一把，是很平常自然的事情；

如果你去医院看病，医生会因让你久等而站起身，向你说久等了对不起；

当你不得不996，公司会精准记录你的工作时长，按照法律规定付工资；

如果你想看国外的网站和听国外的新闻，参与各种话题的讨论都很自由；

… …

很遗憾，中共国没有这些再平常不过的事情！

在中共国，如果你真诚，很多人会欺负欺骗甚至打压侮辱你；如果你刻苦，很多人会努力让你做更多的活儿，把你当成工具看待；如果你值得信赖，很多人会联合起来对付你，直到没有人敢和你交流或为你加油……结果就是，坏人都团结到了一起，好人却团结不起来，坏人越来越坏、越来越强大，而好人先是不为他人做声、然后是没人再为自己做声、直至没有什么声音，只剩下了坏人的声音。

在中共国，想有真正的信仰很难！在没有信仰的地方只剩下了钱、还有就是羡慕嫉妒恨和扭曲的价值观了。在中共国，高楼大厦鳞次栉比，就连市政府、教育局等等部门的建筑物都灰常奢侈豪华，但是，有很多便衣警察挡在外面，不让人进去。就类似于“上海繁不繁华与我何干？”，在中共国光鲜亮丽的外壳背后隐藏着可怕的东西！那是可以毁灭民族甚至全人类的良心和健康的东西！
