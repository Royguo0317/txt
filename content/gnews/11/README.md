俗话说天下乌鸦一般黑，是的，说的就是共产党的村干部。就一个好干部都没有吗？看你怎么定义了。我的定义就是：好一点的村干部贪污少点儿，但是办不了事儿。不好的村干部就是直接卖地卖资源，暴力手段恐吓老百姓。

就拿强拆征地这一件事儿来说吧，村干部跟开发商都是一伙的，政府和地痞流氓也是一伙的。他们组成了征地、强拆、房地产开发的一个黑色产业链。土地就像是他们家的蛋糕一样，想怎么分就怎么分。

他们如果看好了，无论是你家住的地还是你家种的地，想征用做房地产开发的，如果你不同意，他们有各种各样的办法对付你。

要是你家有公职人员，村干部就会报告给市政府，对你家公职人员进行停职和回家反省，到最后还是被逼同意，不同意就不给工作干了，把你直接开除了，让你丢掉饭碗。

如果你是草根儿农民那就更惨了。前期，村干部会上门做所谓的“工作”，劝说你，如果你不同意，他们会找公安局。公安先查查你有没有犯罪记录，传唤你、吓唬你。

你不从，他们就去找地痞流氓，天天去你家滋扰你，让你全家不能安稳生活，手段多的很。

各种压力下一般人没有不就范的。你会问我，不可以去告他们吗？可以，但是常常是官官相互，你告到镇长那儿，镇书记等在你。他们和村干部都是互通信息的，然后唱一出双簧，跑来做你工作。

所谓“做工作”还不就是那些下三滥的威逼利诱手段吗？老百姓拿着时间去告他们的话，最终换来的就是被维稳，甚至家破人亡。

你会说没那么严重吧！有，就有那么严重！老百姓告状是要付出时间和金钱的。他们就把老百姓当作皮球踢来踢去。时间长了，家里的人，气的气死，散尽家财，也什么都换不来。有的有骨气的老百姓，不服，就去镇上告，告不赢再去市里告。

但是，换来的常常还是一样的：好一点的给你两句好听话，事儿不给你办；坏的，直接就跟你的村干部沟通，对你维稳。到了这个地步，如果你还相信共产党，接着去告状，去北京告状，更有办法对付你。

每个省在北京都有驻京办事处，那些人就是在信访局专门等着你，截访。先把你扣押到其他地方。

然后省驻京办就通知市里，市里逐级通知，最后把你接走。接你的就是村干部和村干部的打手（欺负你的那些地痞流氓）。你如果不乖乖跟他们走，换来的会是一顿毒打；你乖乖的走，他们就会给你编织谎言把你哄回去。

如果经历了这些你还是要执意上访告状，好，那村干部就跟你玩阴谋。

你不是嫌征你的地补偿款赔得太少了吗？他们就给你多点钱，然后给你录音，各种办法让你掉进他们设计的法律陷阱里，然后告你敲诈勒索，诈骗。再把你送进监狱，判刑折磨你的精神，摧残你的肉身。

这种办法，他们会循环往复的使用，直到你筋疲力竭，告不了状上不了访。随着时间的流逝，肉体也消亡了。

在乡村这一级，可怜的老百姓遇到不公的时候能使的办法几乎没有，是真真正正的一片黑暗。大多数人能逃离的都拼命逃离，不能逃离的只能默默忍受，非常凄惨。血泪一把。
