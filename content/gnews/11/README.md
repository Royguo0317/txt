作者：纽约香草山写作组   卓别林（文尊）

中共将马克思主义与法家商鞅之策相结合，让阶级斗争不仅有了理论依据，还顺利让独裁统治绑架了传统道德。

八九六四未能让中共乘上东欧共产主义分崩瓦解的趋势，反而因为老布什政府的绥靖政策让中共逃脱一劫，并让他看到要想长存并有一天击溃资本主义自由民主的体制，不但不能再遵循马克思主义的暴力革命，也不能单单实行渗透，瓦解与和平演变，而必须是在渗透的同时拉弄和绑架西方，培养出与自己血脉相连的利益集团。这就是之后中共在全球实施的“蓝金黄”计划和培养的盗国利益集团。
![CCP]()![CCP](https://gnews.org/wp-content/uploads/2021/03/image1-3.png)
1999年在克林顿政府的推动下，中共国加入了WTO，之后通过一路的贸易畅通与从未履行的公平贸易协议，在各国政府睁一只眼闭一只眼的保护下，基于14亿人庞大的人口基数，巨大的购买力，廉价的生产力，和不惜牺牲环境的“大跃进”式发展方式，经过30年的飞速发展成为世界第二大经济体。趁着这两趟东风，中共的全球“蓝金黄”也全面铺开，将利益与华尔街，各国政要，政府，利益集团牢牢捆绑，而它的壮大主要也是得益于绑架后的各国利益集团的背后相助。

共产主义的目标之一：掌握美国的一个或两个政党。早在上个世纪80年代初期，中共间谍就已经开始向民主党主要议员渗透。日本畅销双周刊《SAPIO》曾报导，早在克林顿掌权时期，克林顿夫妇多次收取来自世界各地的华侨（实为中共政权代理人）的政治献金。主要资金来自力宝集团（Lippo Group）。早在1985年克林顿担任阿肯色州州长时，夫妇两就常常往返于香港与中国银行董事长黄建南及力宝集团掌舵人李文正会面。克林顿夫妇在阿肯色州经营的法律事务所正是力宝集团的法律顾问，并获得高额的报酬。据CIA与参议院的调查显示：力宝集团在战后长期协助中共间谍机关事务。力宝与中共军方情报机关分别出资五成在香港设立中国银行。

除此之外，克林顿执政期间，提名美籍华人黄建南（John Huang）为联邦商务部国际经济政策的副助理部长，于1995年调入民主党总部，担任政治资金副议长，一手策划了民主党第二年的总统大选。而之后爆出黄建南为中共在美培养的间谍，也是克林顿政治资金的赠贿者。1996年，经CIA及DIA证实一家名为Poly Technology的中国公司（经营者为前中共国家副主席王震之子）为北韩、巴基斯坦、伊朗输送大量大杀伤力生化武器技术及零件。克林顿政府本应予以制裁，但是商务部却决定进一步向中共输出军事技术。

本应对华强硬的小布什政府却因911事件集中国力对付阿富汗和伊拉克，直至今日川普都认为这是一个非常错误的决定。中共获得一个喘息的机会却变本加厉地祸害世界。2007年的美国次贷危机，中共乘机在美国和全世界铺开他的“蓝金黄”计划，到2008年北京举办奥运会，中共在全球的“蓝金黄”渗透和全球盗国集团的合作已经达到高潮。

据郭文贵先生爆料海航时称：沉船计划也就是盗国计划，怎么让钱财据为己有且不被国家查封？那就是放到基金里。第一，通过海南航空的慈航基金，隔离债权，把钱吸走；王岐山在海南当省委书记的时候，冠军出现的时候，从国有变私有的时候，这个计划就已经开始了，这就是盗国计划。第二，成立家族信托基金（80年代就已经开始了），两手都抓，抓老百姓的命，抓老百姓的钱。慈航不是慈善基金，是一个完全的操作的，盈利的，掩蔽资产的，不可追究的，免法律责任的私人基金，永远不承担国内贷款的任何债务，任何国家法律不可参与。自2017年5.11报料后，7月11号海航向全世界公布29.5%冠军的股份已经捐给了纽约的慈航基金，而该慈航基金的最终受益人是王岐山家族信托基金（其家族包括冠军，刘成杰，孙瑶，姚明山，姚明端等）。郭先生义愤填膺的指出，这不是捐，这是偷，把中国老百姓的6000亿偷到了美国，直接放到了王岐山家族的信托基金。这是明显的偷天换日，祸国殃民。

王岐山家族在美国的财富始于80年代的姚依林家族，从老一辈的盗国家族到小一辈的盗国集团，这背后是中外势力的相互勾结和渗透，连美国政客都认为王岐山是全世界最有权力的人。

福克斯新闻节目主持人马克‧莱文（Mark Levin）曾表示：拜登上台第一步就是出卖美国，他还发推敦促人们进行反击，保护美国《宪法》。然而拜登家族的卖国和盗国早在40年前就开始了。 川普总统曾将在宾夕法尼亚州（拜登家乡）的演讲中批评拜登出卖美国工人，是美国人最糟糕的噩梦。随着硬盘门的曝光，拜登家族的卖国，盗国证据被曝光在光天化日之下：乔·拜登2008年当选副总统后，亨特离开美铁，他成立投资机构罗斯蒙特·塞内卡投资公司（Rosemont Seneca）与中国万向集团有着上亿美元的融资合作；乔·拜登成为背后的大人物“O”；亨特从乌克兰能源公司获利，拜登则利用当时身为美国副总统权势，以10亿美元的补助为要胁，向乌克兰政府施压，要求解雇正在调查该公司的最高检察官肖金（Viktor Shokin），乌克兰检察部门最终放弃对该能源公司的调查；中共用10亿美金通过亨特使得奥巴马政府对南海军事化问题保持沉默，一介贫民总统在两个任期后靠着出卖美国人和中国老百姓的利益发家致富.

类似这样的腐败与勾结举不胜数，这是共产主义至少三十年的精心布局，不仅通过海外间谍网还通过大机构和财团的利益联姻。然而这都只是明线，还有一条更深的暗线，那就是国际共产主义的幽灵所控制的精英集团。（见*[【世事解评】共产主义蚕食世界（四）——精英集团](https://gnews.org/zh-hans/879575/)*）这条暗线的背后孕育着他们要控制世界的惊天大阴谋。
