**西班牙2021年1月8日电/西喜社—**美国加利福尼亚州周三晚些时候发布了新的旅行警告，称 “强烈劝阻 “来自外州的人进入加州。它还说，除了必要的旅行外，加州人应避免离家超过120英里的旅行。

该州之前在11月发布的警告，鼓励人们留在家里或在自己的区域内，但没有给出人们可以旅行的具体距离。它概述了针对州外旅行者的检疫指南，但没有明确阻止旅行。

“州内旅行，同样有可能加剧加州内的社区传播—特别是由于旅行本身（尤其是在航空、公共汽车或铁路旅行中使用共享交通工具）会增加一个人传播和感染中共病毒的机会。”加州公共卫生局发布的新警告说。

从其他地方抵达加州的人，包括州内居民回家，应自我隔离10天。

该建议给予必要的旅行和工作以宽泛的例外。
