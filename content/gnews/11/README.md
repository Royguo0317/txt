今天一打开手机，就发现许多人在朋友圈转发一篇去年的旧闻。在这个信息大爆棚的时间点上，一篇一年前的文章被翻腾出来，若不是作者的笔能生出花，那就是出于某种特定的需要，尤其是这篇文章的内容正处于风口浪尖上。

这篇文章2020年2月17日面世，原标题是：《石正丽，科学狂人与科学伦理》，文章当时只是在网上转了转便消失了。今天又卷土重来，标题改成了：《石正丽，为何要改造病毒》。这是要闹哪样？

昨天，美国国务卿蓬佩奥宣布病毒来自武汉P4实验室，并对中国科学家石正丽提出质疑。就在这个敏感点上，此文章流出来，瞬间阅读突破10万+,其背后的用意是什么？科学家石正丽的小肩膀，能否扛起冠状病毒这口大锅？还真是个问题。

我第一时间在朋友圈转发了这篇文章，并附言：这虽然是去年的旧文，今天读起来触目惊心、毛骨悚然。时间才是试金石，这篇文章存活的越长，石正丽被灭口的风险越大，让我们拭目以待！ 按照中共的审查制度和这篇文章的敏感程度，发朋友圈是一定会被封号的，然而今天没有，此文被转发，却不允许文下评论，看来，中共这是准备把石正丽当成替罪羊了。

在中共病毒大流行一年之际，全球人类健康岌岌可危，迄今已有一亿人感染，200多万人因染疫罹难，中共却想甩锅给一个科研人员，怎么可能让人信服？一个人终其一生，去研究危险病毒，难道石正丽就有这个瘾？十几年如一日翻山越岭进出蝙蝠洞、老鼠洞，怎么看都觉得这不是一般的癖好。

![]()![](https://media.discordapp.net/attachments/755121289911468082/800221518814117958/WhatsApp_Image_2021-01-16_at_19.32.10.jpeg)图片来自网络

我们大致回顾一下这篇文章的内容：首先，详细介绍“蝙蝠女侠”称号的由来。2018年，石正丽在一次演讲中详述自己剑走偏锋、钻洞越岭的奇侠之旅。她自称2011年，在云南某矿洞的菊头蝠身上，发现了SARS病毒的同源病毒，她采集了十几株，并将此发现刊登在《科学》杂志上。

这样的病毒研究“初心”到底是什么？如果对人类的健康有百害而无一利，科学家到底为什么对此孜孜以求？这是一个科学的伦理问题，因为人不能为科学而科学，科学最终的主旨是让人类活得更好而不是更危险。石正丽的研究曾经获得过国际上的资金支援。

到了2014年，美国人发现石正丽研究的病毒可能会成为生化武器，威胁到人类的安全，立即叫停了这个研究，并切断了对相关研究的资金支持。

然而，石正丽并没有停止她的危险行动，继续对这些毒株进行人工干预。如果靠自然进化，这种生活在蝙蝠体内的冠状病毒无法获得人的蛋白质信息，很难使病毒进化成可以传染给人的病毒，专家测算自然传染可能需要上亿年。但在实验室里，冠状病毒可以轻松获取人的蛋白质信息，修改后的病毒可以实现其在人与人之间的传播。 石正丽果然就做了这样恐怖的研究，并且每走一步都留下了清晰的记录。

2015年，她和她的团队在国际著名期刊《自然医学》发表了论文：《一个类似SARS的蝙蝠冠状病毒群显示了在人类出现的可能性》。文章提出：只要把蝙蝠身上的S蛋白里的ACE2（血管紧张素转化酶2）这个受体开关一调，这个病毒就可以传染给人类。利用病毒基因重组技术将蝙蝠的S蛋白和小鼠的SARS病毒组合，得到的新病毒可以和ACE2结合，能感染人类的呼吸道细胞，毒性巨大，且所有疫苗无效。2016年，石正丽已经掌握了相关的基因密码。

![]()![](https://media.discordapp.net/attachments/755121289911468082/800221481303932949/WhatsApp_Image_2021-01-16_at_19.32.01.jpeg?width=804&amp;height=1237)图片来自网络

2017年，她和她的团队在PlosPathgens杂志上发表文章称，她和团队通过五年的样品搜集，发现了蝙蝠中的SARS病毒是中共国此前爆发的SARS更直接的证据。自此，蝙蝠女侠的人生被病毒照亮了。2018年4月，中共喉舌央视将冠状病毒当成巨大的科研成果向全世界显摆。2020年，该病毒全球大爆发。许多研究此病毒的科学家发现病毒正是通过ACE2受体与S蛋白互作的分子机制感染人的呼吸道和上皮细胞。

至此，谁还能相信这是巧合？对于各种质疑的声浪，蝙蝠女侠石正丽歇斯底里地怒斥学界：闭上你的臭嘴！面对学术质疑，凶相毕露，獠牙都露出来了，是长期和蝙蝠相处学来的表情吗？泼妇骂街不可怕，可怕的是许多科学家真的就应声闭嘴了。加拿大、印度、美国等都有与此病毒或疫苗研究的相关科学家相继因“意外”永远闭了嘴。

石正丽为什么要把这个本来对人类没有任何伤害的病毒改造成杀伤性很强的病毒？为什么在美国明令禁止这种研究的时候，还突破科学伦理继续进行这种危险的研究？ 她不过是个御用毒王。科学如果掌握在这种魔鬼的手里，人类会怎样？现在，全世界都看到了答案。

疫情开始后，石正丽的主子，寡廉鲜耻的中共开始大肆甩锅，穿山甲无辜躺枪、美国实验室也未能幸免，这些谣言，无非就是要混淆视听、扰乱真相，让病毒继续虐杀人类，全球经济陷入危机，中共趁机发疫情财，为自己岌岌可危的统治续命。

我们把目光移回2020年1月19日，世卫组织还没有承认病毒人传人，还在帮中共遮掩的时候，爆料革命将闫丽梦博士的信息传送到全世界。闫丽梦博士九死一生，逃出魔窟，向全世界讲述病毒真相，并用两篇证据确凿的论文，证明了病毒就是来自武汉P4实验室。 同样是科学家，闫丽梦博士是拯救世界的天使，而魔鬼一般的石正丽的未来在哪里？

美国基于国家机器的情报和各种专业证据，证实了闫博士的报告，提出病毒来源于武汉实验室，勒令中共彻查病毒来源。至此，武汉P4的这口大破锅，全球转了一圈后，又砸回了它的大本营。

病毒仍在肆虐，正义力量已经觉醒，人类的未来会怎样？一个科学家成为替罪羊，能抹去杀死200万无辜者的罪恶事实？能让世界回到病毒爆发以前吗？这场病毒改变了太多的人和事，相信，罪魁祸首中共付出代价的日子不远了！
