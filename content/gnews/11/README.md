**据《华盛顿时报》作者：比尔·格茨 2021年6月10日报道：**
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/13003721/ppoli20210612.jpg)
**《司法观察》（Judicial Watch）已提起两起诉讼，要求国务院和国家情报总监办公室提供有关COVID -19 病毒起源的文件。**

《司法观察》周四宣布了这些诉讼，称这些诉讼是根据《信息自由法》（the Freedom of Information Act，FOIA）中“文件要求被忽视”提起的。这些诉讼正在寻求自 2017 年以来情报机构持有的所有文件，以及自 2017 年以来国务院关于武汉病毒研究所被越来越多的人怀疑它是病毒起源点相关内容的电报。该组织还要求得到与病毒来源情报相关的文件。

拜登总统最近下令美国情报机构对病毒来源进行为期 90 天的调查，此前一项调查导致几个机构在病毒大流行是自然爆发还是实验室事故的结果上陷入僵局。《司法观察》主席汤姆·菲顿（Tom Fitton）在一份声明中说：“我们不相信拜登政府或深层政府会自愿澄清 COVID 及其与武汉研究所可能的联系。”“《司法观察》的 FOIA 诉讼可能是揭开 COVID 和武汉研究所真相的关键。”

《司法观察》最近根据 FOIA 获取到的电子邮件，将拜登政府的流行病学专家安东尼·福奇（Anthony Fauci）博士与美国为武汉病毒研究所提供的资金联系了起来。4 月，国家情报总监办公室发表声明说，整个美国情报界“一直在为美国决策者和应对（源自中共国的）COVID-19 病毒的人员提供关键支持。”

去年，时任白宫国家安全顾问罗伯特·奥布莱恩 (Robert O’Brien) 在接受采访时表示，美国间谍机构对 COVID 爆发的了解很慢，并且在早期忽视了其传播的危险，将其视为一种类似于流感的疾病。加利福尼亚州众议员德文·努内斯 (Devin Nunes) 是众议院常设情报常设委员会中资深共和党人，他也在寻求情报机构的答复，了解这些机构是否未能对大流行做出适当的反应。

该法律小组上周获得的电子邮件（包括卫生与公共服务部的记录）显示，在 2014 年至 2019 年期间，由福奇博士领导的美国国家过敏和传染病研究所 (NIAID) 向武汉病毒研究所提供了 826,277 美元用于蝙蝠冠状病毒研究。
