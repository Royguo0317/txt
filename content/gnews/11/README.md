# **前美国国务院高级调查人员说，新冠肺炎爆发可能是由于生物武器研究事故所造成的**

**戴维·阿舍(David Asher)认为，这种病毒可能来源于中共国军方在武汉病毒研究所的研究。**

*福克斯新闻，作者詹妮弗·格里芬(Jennifer Griffin)*

[!\[\]()!\[\](https://spark.adobe.com/page/HsLA2ITaRa0iN/images/a112c893-cf4d-4189-be53-da5b65c36d02.png?asset_id=21d37d29-1fe7-466a-af04-3cd27aa39bd7&amp;img_etag=%22a6c45d6281831cbf34543fc892f214fe%22&amp;size=2560)](https://spark.adobe.com/page/HsLA2ITaRa0iN/images/a112c893-cf4d-4189-be53-da5b65c36d02.png?asset_id=21d37d29-1fe7-466a-af04-3cd27aa39bd7&amp;img_etag=%22a6c45d6281831cbf34543fc892f214fe%22&amp;size=1024)白宫就冠状病毒来源向中共国施压

当美国高官准备与中共国同行进行拜登政府时期的首次面对面会谈之际，美国国务院监督新冠肺炎病毒来源专案组的前首席调查员告诉福克斯新闻，他不仅相信病毒从武汉病毒研究所逃逸，而且可能是中共国军方，也就是人民解放军正在进行的生物武器研究的结果。

现为哈德逊研究所(Hudson Institute)高级研究员的戴维·阿舍(David Asher)接受独家采访时告诉福克斯新闻：“武汉病毒研究所不是美国国立卫生研究院(NIH)，它正在运作一个隐蔽的机密项目。在我看来，仅代表我个人见解，我认为它是一个生物武器项目。”

长期以来，阿舍一直是一个“追踪金钱”的人，他曾为民主党和共和党政府的国务院和财政部进行过一些最为机密的情报调查。他领导的团队解开了巴基斯坦核计划之父阿卜杜勒·卡迪尔·汗(AQ Khan)经营的国际核采购网络，并解开了朝鲜秘密铀浓缩的关键部分。他认为，在过去的14个月中，中国共产党一直在进行大规模的掩盖行动。

阿舍在哈德逊研究所主题为《新冠肺炎起源：政策含义和对未来的教训》小组讨论中说：“而且，如果你像我一样相信，这可能是一个出了差错的武器载体，不是故意释放，而是在开发过程中，然后以某种方式泄漏，那这已经变成了历史上威力最大的武器。你（施放者）已经夺走了全球GDP的15%到20%，你杀死了数百万人，而中共国人口几乎没有受到影响。他们的经济迅速回升到整个20国集团(G20)的第一位。”

阿舍说，中共国政府的行径让他想起了他所监督的其他刑事调查。

阿舍说：“动机、掩盖、阴谋，所有罪行的标志都与这些要素有关。而且，最初的受害者群体围绕着从事高度危险、甚至是可疑研究的研究所，这一事实意义重大。”在2003年非典爆发期间，他作为美国国务院的首席代表与中共国政府接触。

起初，中共国表示新冠肺炎病毒起源于武汉海鲜市场。但中共国理论的问题是：第一个病例与该市场无关。去年秋天，美国获得的情报显示，武汉实验室的几名科学家出现了类似流感的症状，他们在2019年11月住院治疗，而该时间点位于中共国报告首例病例之前。阿舍与哈德逊研究所其他小组专家表示，2007年，中共国宣布将开始研究基因生物武器，使用有争议的“功能增强”(gain of function)研究，使病毒更具杀伤力。

2016年，中共国不再公开谈论他们在武汉实验室的研究。阿舍认为，在那个时间点，人民解放军介入了研究，并将研究方向从生物防御变为了生物进攻。同年，中共国顶级国家电视台评论员表示：

“我们已经进入了中国生物战时代，包括使用病毒。”阿舍指出：“我的意思是，中共国政府向人民公开声明，这是习近平国家安全政策下的新重点。”

根据阿舍的说法，中共国在2017年停止公开谈论冠状病毒“可用于武器的疾病载体”研究，与此同时，其军方开始资助武汉病毒研究所的研究。

阿舍说：“对于那是巧合的看法，我保持怀疑态度。”

与此同时，美国的生物武器研究人员仍主要关注炭疽等老式生物武器。在寻找如何防御冠状病毒生物武器的过程中，一个关键的转折点包括有争议的“功能增强”研究，以及荷兰的一项使科学界感到惊讶的突破。

奥巴马总统手下负责核、化学核生物防御计划的前国防部助理部长安迪·韦伯(Andy Weber)回忆说：“我记得正在海牙与荷兰外交部会面，那天新闻爆出荷兰一家由美国国立卫生研究院(NIH)资助的实验室正在对高致病性禽流感进行功能增强研究，该研究专注于增加这种非常危险的流感病毒的传播性。”

奥巴马政府迅速暂停了这种研究，担心它可能成为恐怖分子的教科书。2017年，川普政府取消了暂停令，但在疫情开始后的2020年4月停止了NIH对武汉实验室的资助。

专家们称，长期以来，中共国的生物安全四级实验室的生物安全一直备受关注。

美国国务院官员余茂春(Miles Yu)称：“中共国自2003年非典爆发以来，一直在参与这类病毒的研究。”他最近与前国务卿麦克·彭佩奥(Mike Pompeo)在《华尔街日报》上共同撰写了一篇有关病毒起源的专栏文章。“中共国的生物安全标准非常之低，十分危险。因此事故的发生只是时间问题。”

世卫组织2月派往武汉的团队参观了武汉病毒研究所。他们没有穿上生物安全防护服，并在里面停留了3个小时。但据报道，他们没有接触到科学家，也没有接触到他们需要的数据，无法完全排除病毒从实验室中逃逸的可能性。

中共国外交部发言人汪文斌当时说：“应该指出，病毒溯源是一个复杂的科学问题，我们需要为专家进行科学研究提供足够的空间。”他还说：“中国将继续以公开、透明、负责任的态度与世卫组织合作，为更好地防范未来风险、保护各国人民的生命健康做出自己的贡献。”
