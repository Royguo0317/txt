**麦康纳尔已毫不掩饰 / ****他声称确保拜登的当选结果将是他最重要的成果 FOXNEWS**

麦康纳尔已毫不掩饰。据FOXNEWS2020年12月31日报道，他在周四早上的电话会议中说，认证大选结果将是他最大的成果。

“我在参议院干了36年经历了许多次大选，我认为，仅仅是我认为，这将是我经历的最重要的成果。”他认为“美国已经有了一个当选总统和当选副总统”。

川普总统对于共和党领导人没有拒绝接受大选结果十分懊恼，他正与声称将反对大选结果的莫布鲁克、霍利等组成同盟。他在推特中表示“我们的领导人（当然不是我）是极其可怜的”。

麦康纳尔在周四早上的电话会议上对霍利就他决定反对认证拜登获胜的决定施压，然而霍利并没有上线，麦康纳尔问了几个问题但是无人作答。

**彭斯似乎正在变节 / ****他要求法官拒绝共和党人提起的挑战选举结果的诉讼 USA TODAY**

参议院议长、副总统彭斯似乎正在背叛川普总统。他要求一位法官拒绝由一群要求授权给他推翻选举结果的共和党人提起的诉讼。彭斯在一份周四晚间由司法部律师们起草的十四页的文件中宣称，这项诉讼是在反对国会。他说“这些原告的诉讼并不是提出这些要求的恰当手段，因为他们起诉了错误的对象”。他补充说，讽刺的是那些人想要提升的恰恰是他彭斯本人的权力。

**在乔治亚州的选举中将有八千人自愿充当共和党观察员**

在周二乔治亚州两次参议员竞选中，约八千人自愿充当共和党的投票监督员，这个数字是截至周四为止的过去两周半时间里该州此类工作人数的两倍。这是全国共和党委员会、乔治亚州共和党和现任参议员凯利·洛夫勒和大卫·珀杜的竞选活动共同努力的结果。乔治亚州人民应该获得公平准确的选举结果，我们对成千上万名想要保证选举安全、有保障、每张合法选票都被计入的志愿者表示感谢。乔治亚州共和党人已经动员了该州史上最大规模的地面活动和选举监督，充分做好了让下个星期选举毫无疑议的准备，没人再能质疑选举结果。

**病毒监测系统考验着以色列的民主规范**

在之前的大流行当中，惊慌失措的以色列开始对公民采用大规模监视工具，追踪人们的手机企图以此阻止病毒的扩散。政府鼓吹这项通常用于缉拿巴勒斯坦激进分子的技术是抗击病毒的突破。但是几个月之后，这项技术的效果就受到了质疑，公民们说这对国家的民主原则造成了难以估量的损害。以色列民权协会女发言人玛雅·弗里德（Maya Fried）说：“政府密切监视自己的公民的想法应该引起警觉。”该组织一再向法庭质疑该技术的使用。 “这违背了民主的原则。在危机时期，你不能只会放弃民主。”
