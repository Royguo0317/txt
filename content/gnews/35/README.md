5月25日，[印度主流媒体GoaChronicle](https://goachronicle.in/unraveling-the-corona-virus-origin-mystery-with-dr-lawrence-sellin/amp/?__twitter_impression=true)总编Savio Rodrigues与塞林博士进行了视频交流。塞林博士向印度人民介绍了COVID-19疫情的本质以及中共在其中发挥的作用，其主要观点如下：

塞林博士肯定COVID-19病毒不是来自自然，而是实验室制造出来的，是在中共生物战计划中开发出来的。因为到目前为止，几乎没有证据表明还有其它可能，而我们看到的所有证据都表明，PLA拥有制造这种病毒的所有先决条件并广泛参与了相应工作。并已经确定了与COVID-19病毒研究和制造相关的地点和人员，验证了他们正在做的大部分工作都与PLA关联。现在我们关注的重点是，验证它是意外泄露还是故意释放的。

通过阅读和分析中共国大学教授围绕生物技术或生物战主题而出版的一些期刊或书籍，发现中共军方一直致力于提高生物战作战能力。PLA接管二战时日本关东军在中国东北的生物战计划下的731部队和100部队后，其在生物武器研究方面长时间默默无闻，没有取得什么实质性的进步和突破，但自1990年代，随着基因工程的加速发展而看到了制造他们梦寐以求的基因武器之机会。在其军事文件甚至公开网络上都可以看到他们专门探讨生物武器这个战争选项的两个应用场景，即在战前或战争中使用的场景。

中共生物战计划得到了很多国际帮助，其中大部分来自美国。中共生物战计划的结构有三个层次：第一层次亦即核心层面，就是军科院AMMS及其下属军事机构；第二层次包括中国的民间、研究机构和大学。在中共国，军事和民用研究没有区别，中共在其2016年的五年计划中就指出，他们将融合民用和军事研究。基于第二层面，CCP-PLA与国际社会互动，并已大规模渗入美国病毒研究项目，以摄取和利用美国的知识、技能和资金。

面对这些令人震惊的事实不断浮出水面，美国和世界首先要接受COVID-19是在中共国实验室制造的，甚至可能与中共生物战计划有关。美国的大量医学研究经费没有国家安全责任，包括来自NIH和国防部的经费。我们需要仔细调查这些钱都花到哪里去了，是怎么花出去的，也要调查这些来美的中国科研人员，核查他们是否帮助了中共的病毒研究项目。

WHO的存在是一场灾难，从疫情爆发到最近的调查，它一直说病毒来自大自然，一直帮中共掩盖病毒真相，充当中共的宣传机器。WHO在COVID-19疫情中毫无用处，甚至可能是一个危险组织，需要彻底重组甚至解散！

面对COVID-19疫情灾难深重，世界人民水深火热，中共却加大力度掌控各国，而世界仍然保持沉默的局面，塞林博士坦言自己不能代表世界说话，但可以说一些美国的事情，因美国的情况就是世界的缩影。美国政府不为美国人民服务，只对全球化的金融利益负责，而这些利益正是与中国合作而一起赚的钱。如果美方对中国提出挑战或调查，这些利益之船势必会触礁并损害美方的经济利益，因此很多权贵和富人都面临巨大压力而消极面对这个调查。在科学界，有些人在帮助中共病毒研究计划上是犯罪行为，也许他们并不想这么做，但最终事实上是这么做了！

关于愈演愈烈的印度COVID-19病毒变种问题，塞林博士在专业上一针见血，其突变是非常特殊的，由一段增加了传染性和致病性的福林酶切位点(FCS)的人工插入而产生，这完全不可能来自病毒自然进化途径，与PLA关联的中国科学家知道如何将它插入病毒。FCS片段上的4个氨基酸排列PRRA，分别代表脯氨酸、精氨酸、精氨酸、丙氨酸，FCS的功能是在精氨酸-精氨酸(RR)片段中，而初始的脯氨酸P只是结构元素。但在印度这种变异中发生的是脯氨酸P被另一种精氨酸A所取代，因此不再是两个精氨酸RR，而是三个精氨酸RRR，而随着精氨酸数量的增加，FCS的功能也会增加，这就是其传染性比以前的版本更强的原因。

同在一个地球村，我们需要让中共对制造这种病毒负责，让每个人都清楚并接受这个事实，并深入探究和揭露中共生物战计划的结构和组织，同时着眼于中共如何操纵国际研究网络，促使他们不仅自己得益于正常的病毒研究项目，而且造福中国人！

关于当下推广得如火如荼的疫苗问题，塞林博士压抑住内心的愤怒阐明了自己的观点：人们过分强调疫苗，而忽视了治疗方法，尤其是早期的治疗方法，不仅被无意忽视，而且被刻意轻视。羟氯喹、伊维菌素、维生素D、锌等药物，如果在疾病早期使用，可以阻止许多人病情加重、住院甚至死亡。并不是不相信疫苗，问题是病毒会变异。在疫情期间，疫苗本身也可能引发一些问题，譬如，病毒会试图适应并避开免疫系统，某些疫苗可能会因为持续的变异而变得不那么有效，甚至病毒变异的程度和速度远超疫苗开发的预期。现在全世界都把注意力集中在疫苗上，在疫情流行期间大规模接种疫苗是非常危险和恐怖的，很难相信大量疫苗生产及其接种与某些人的经济利益无关。

从年初开始，塞林博士日拱一卒地保持对COVID-19病毒起源真相的揭露。在Twitter上招贤纳士，新中国联邦战友们应者如云，全球侦探们八仙过海、各显神通，搜集到了无数有关中共病毒和PLA生物战计划的线索和证据。塞林博士还把握一切机会参加各种媒体的节目，向全世界揭示并传播COVID-19病毒的真相。并与The Gateway Pundit的联合创始人Joe Hoft紧密合作，联合发表文章，系统性地报告其COVID-19侦查团队的任何突破性进展，这让世界各地的专业情报机构感到羞愧，催生了拜登总统前天下令美国情报界务必在90天内拿出明确的COVID-19病毒起源报告。这些对世界、对人民都是令人赞叹、不可思议的成就！
