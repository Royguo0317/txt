《圣经》中神审判人类罪恶的时候有一个大杀器，甚至神从来都是十分慎重使用的——那就是瘟疫。当今，世界CCP病毒横行，导致经济崩溃，人类正常工作、生活节奏被完全打乱。而且未来结局如何，至今仍是未知数。我今天从《圣经》出发，结合历史和现实来说说神使用瘟疫的原则和祂通过CCP病毒给人类的启示。

《圣经》中第一次提到瘟疫是在埃及人残酷奴役以色列人的时候。那时，神派摩西、亚伦去对法老说“希伯来人的神遇见了我们，求你让我们走三天的路程到旷野去，献祭给耶和华我们的神，免得他用瘟疫或刀剑击杀我们。”(出埃及记 5:3 )

这里很有意思——摩西没有说神要用瘟疫和刀剑击杀埃及人，反而是说要击杀以色列人。笔者初看时觉得有点难以理解，觉得摩西是不是态度软，对法老示弱。其实不是。这里神其实启示了一个对全人类都适用的普遍原则：“如果人类哪个族群不用心灵和诚实祭拜敬畏真神耶和华，那么罪恶到达一定地步的时候，神必用瘟疫或者战争来审判这族群。”其实，读者如果仔细研究我们中国人的历史，就会发现，我们也受这个看不见的神的原则管制着。

后来，法老硬着心，十次反悔，坚决要奴役以色列人。因此，神终于让神奇的专灭长子的“瘟疫”降临到了埃及人头上——这和如今中共妄图研制的专门定向清除白色人种的病毒武器，似乎有异曲同工之妙。有人会疑问，中共制造病毒攻击白人和神有啥关系呢？其实即使是中共，如果神不默许它做，它也什么都做不了——因为一切主权在于神。

我们知道，让中共得以生存和壮大的核心原因——就如文贵先生说的——是美国贪婪的资本力量对中共的放纵。他们与中共勾结奴役中国人，甚至带领全世界一起这样做。这种行为不知不觉中积蓄着神对他们的烈怒。这次中共病毒爆发的时候，越是和中共勾兑多的国，受害越重，并非巧合。其实，这都是神默许了的。如果美国为首的世界各国再不停止和中共勾兑，再不下定决心灭共，那么，说不定针对白人的病毒就真的在神的默许下被中共获得并且释放出来了。

再后来，在以色列人出埃及的路上，神又总共三次降下瘟疫给以色列人。第一次是由于以色列人到达应许之地边界时，对神缺乏信心，不敢去得神的应许之地。一些人甚至诋毁神，说神是为了消灭他们才带他们出埃及。于是，神降下瘟疫给报恶信之人。第二次是有可拉等首领不顺服神公义的权柄，妄图夺权，并且自以为圣洁和公义，敌对摩西和亚伦。为此，神降下瘟疫给以色列人。之后摩西祈求神，给以色列人赎罪，瘟疫才停住。第三次则是以色列人中了先知巴勒的计谋，和摩押、米甸女子行淫乱，并且去祭拜偶像，假神。这导致神愤怒，从而降下了瘟疫。最终因祭司们止住了这种淫乱行为以后，瘟疫得以停止。

从这三次瘟疫我们看到，当人类整体对神没有信心，抗拒神公义的权柄，肉体行淫乱，灵里行淫乱到了一定地步的时候，神就有可能用瘟疫的形式审判人类。现实中，西方世界基督信仰衰落，大多信仰越来越缺乏真实性。有些人表面上信主，实际上在做事的时候抗拒神公义的要求：为了图廉价和安逸，他们总是选择和拥有14亿奴隶的中共勾兑。又如拜登家族这样的有权势的人，和中共一样，贪图他人妻子，肉体上淫乱无限制，甚至娈童。还有些人信仰上追求新奇的东西，致使各种假神层出不穷，并对基督信仰进行不公正的打压和抑制，厌恶耶稣基督这似乎乏味的无酵饼。所有这些偏离神的行为，都导致了撒旦魔鬼的代言人中共的崛起。撒旦如今甚至都不用找人吃了，它完全是在吃自助餐。如文贵先生说的，全球政客甚至都主动往撒旦“蓝金黄”的网上撞。所以，这一切最终导致CCP病毒在神的默许之下被中共释放出来。

当然，我不是说这只是以美国为首的西方的过错，而是全人类都犯下的过错——只是多给的多收，少给的少收。神给了欧美管理世界的权柄，当然他们要负更多的责任。

趁着还有机会，我们要感谢神，祂使用正直、勇敢并且对病毒来源十分了解的闫博士清楚地告诉全人类：病毒来源于中共实验室，中共和其背后的撒旦是病毒的根本原因，人类若不彻底消灭中共这个撒旦组织，那么等待人类的将会是更多可怕的病毒和瘟疫。

因此，现在人类要做的是，赶快认罪悔改，回归耶稣基督，回归合神心意的生活方式，并且消灭中共这个撒旦组织。在全球推广法治和信仰自由，消除专制集权奴隶制。若人类再沉沦，只怕是更严重的瘟疫和大规模的战争都会被神默许发生。这就是神通过CCP病毒要给人类的最大启示吧。
