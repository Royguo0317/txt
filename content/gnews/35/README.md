# **朱利安·阿桑奇（Julian Assange）：您需要了解的有关维基解密**(**WikiLeaks)创始人的知识**

新闻链接[https://www.aljazeera.com/news/2020/12/20/julian-assange](https://www.aljazeera.com/news/2020/12/20/julian-assange)

维基解密发布者朱利安·阿桑奇正面临着从英国引渡到美国的法庭诉讼。

在2010年至2011年期间，他获得并公布了数十万份机密文件之后，美国指控他黑客攻击政府电脑和从事间谍活动。阿桑奇于2019年4月被英国警方从厄瓜多尔驻伦敦大使馆逮捕，自2012年以来，他一直在那里获得庇护。预计将于明年1月4日宣布其引渡决定。

**维基解密揭示了什么？**

2010年4月，维基解密网站发布了一段39分钟的视频，视频中一架美军的阿帕奇直升机在空中开火，杀死了包括两名路透社记者在内的十几名伊拉克人。维基解密由此在那时而一举成名。曼宁私自泄露的镜头导致了全球的愤怒，重新引发了关于美国占领伊拉克和在中东更加广泛存在的辩论。

同年7月，维基解密与《纽约时报》等多家媒体一起，公布了9万多份与阿富汗战争有关的美国军事文件。

其中包括以前未曾报道过的有关平民死亡、友军伤亡、美国空袭、基地组织在该国的作用，以及向阿富汗领导人和塔利班提供支持的国家等细节。

几个月后，维基解密发布了391,832份与伊拉克战争有关的文件。这些报告也被称为《伊拉克战争日志》，提供了美军于2014年1月至2019年12月报告的地面细节。

这次泄漏是美国军事历史上最大的一次泄漏，暴露了巨大的平民伤亡。

2010年11月，维基解密发布了数十万条美国外交电报，如今又被称为“电缆门”丑闻。从1996年到2010年2月，共发布了约25万份报告。电缆提供了来自全球270多个美国大使馆和领事馆的分析和见解。
