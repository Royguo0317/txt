# **国家安全顾问奥布莱恩的声明**

[!\[\]()!\[\](https://spark.adobe.com/page/g6hJ0P9bgJXZZ/images/8bcff77c-0232-4322-a2a7-a9b506eeb890.png?asset_id=5e8aea86-c910-49a2-b863-ec8bf91ca91b&amp;img_etag=%225ebec9f2a822c62c128159e31b4016fa%22&amp;size=2560)](https://spark.adobe.com/page/g6hJ0P9bgJXZZ/images/8bcff77c-0232-4322-a2a7-a9b506eeb890.png?asset_id=5e8aea86-c910-49a2-b863-ec8bf91ca91b&amp;img_etag=%225ebec9f2a822c62c128159e31b4016fa%22&amp;size=1024)

今天，川普总统要求美国政府部门和机构评估适用法律、规定和政策，并提出规定和政策修改，包括可能施行的行动，以减少联邦政府对中共国商品和服务的采购。

中共国2017年的《国家情报法》要求个人、组织和机构在各种情报工作中有义务协助中共国的安全和情报服务。基于中共国的组织和中共国公民必须保护所有他们所知的国家情报工作秘密。中共政府在中国共产党的领导下，可以肆意压迫和要求中共国的生产商和服务提供商，以配合进行针对美国政府的间谍活动，获取信息优势。其中一些活动是进一步服务于中共国的军民融合战略，该战略要求所有中共国民间机构，包括公共和个人机构，通过购买和应用关键先进技术的方式，协助中共国的军队建设。

正如国家情报总监日前报告的那样，中共国是美国现在最大的国家安全威胁。北京将美国视为他们最大的地区和国际目标的阻拦，因此他们向军民融合战略以及其他相关战略投入了大量资源以获取相对优势。中共国通过网络黑客和其他手段，针对美国政府雇员记录的信息系统为目标，获取军队计划信息系统和其他有价值的数据。为此，美国必须采取应对措施保护美国的利益。我们必须调整规定和政策，并且采取其他必要措施，减少中共国针对联邦政府的科技和间谍人员活动。今天，川普总统针对此事又迈出了一步。
