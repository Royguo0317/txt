1.美国大选：特朗普如何改变全球外交政策

无论11月3日美国大选的结果如何，特朗普领导下的外交政策和外交方式的变化，都已经在全球外交领域产生了深远的影响。

从2016年竞选初期开始，特朗普就用两个英文单词阐明他的外交政策目标：美国优先（America first ）。

![]()如今，四年过去了，这两个简单的单词已经因政策和事件而变得复杂。单边主义和对抗是特朗普外交政策的特点，伴随着人事更替频繁、意外事件不断，也越来越让人困惑不解。

无论11月3日美国大选的结果如何，特朗普领导下的外交政策和外交方式的变化，都已经在全球外交领域产生了深远的影响。

以下梳理了一些主要的变化。

摒弃多边主义

上任伊始，特朗普就开始破坏国际合作。就职后仅仅3天，他就宣布美国退出和亚洲国家签订的贸易协定——跨太平洋伙伴关系协议（TPP）。随后，他又将美国退出了联合国人权理事会和2015年巴黎气候协议。

此外，美国实行单边行动，无视国际共识，比如决定正式承认耶路撒冷为以色列首都，并将美国大使馆迁往此地。

多伦多大学和牛津大学历史教授、美国外交关系协会访问历史学家麦克米伦（Margaret MacMillan）说，美国 “真的破坏了对它有益的同盟国家和国际机构网络。我认为这让美国的全球地位大大削弱”。

事实上，皮尤研究中心9月发布的一项调查显示，许多国家对美国的支持率已降至几十年来的最低水平。

损坏跨大西洋关系

2020年2月，卡内基国际和平基金会在对跨大西洋关系的评估中写道：”特朗普对多边主义的敌意代表了美国和欧洲各国之间的原则性差异。”跨大西洋关系是二战后形成的欧美伙伴关系，代表着双方共同的价值观、目标和全球方针。

特朗普领导下美国与欧盟的裂痕不仅仅体现在意识形态上的分歧，特朗普还频频指责和颠覆跨大西洋关系。他多次质疑北约等同盟组织的价值，宣布从德国惩罚性撤军，颁布对欧盟的贸易关税，并威胁制裁从俄罗斯到德国的天然气管道 “北溪2号 “工程。

麦克米伦认为，这些紧张的关系将带来持久性的改变。

“这就像一段友谊。你得信任你的朋友，这种信任关系一旦被打破，就很难重新建立，”她说，”欧洲原本已经习惯依赖华盛顿大哥。也许现在欧洲人正在说，我们显然不能这样做，我们将不得不——正如我们几十年来一直在谈论的那样——更多地发展我们自己独立的外交政策。”

中国成为焦点

从针锋相对的关税政策引发的贸易战，到美国向其他国家施压阻止和中国公司华为合作建设5G网络，特朗普在中国问题上的对抗路线，使得这个亚洲国家成为国际焦点。

他对中国的尖锐批评受到了许多人的欢迎，他们认为中国长期以来不公平地从全球贸易中获益，同时又从事着侵犯人权的勾当。

“总统就其在贸易领域的行动向中国提出挑战是正确的。”外交关系协会主席哈斯（Richard Haass）在特朗普外交政策中期评估的前言中写道。

麦克米伦也持类似观点。”我不想对特朗普肯定太多，但是我认为他或者说他的政府在知识产权问题上批评中国是正确的，”她说。虽然美中紧张关系在特朗普之前就存在，但特朗普令其”变得更加尖锐和清晰”。

推特外交的危险

在外交政策沟通方面，特朗普和他领导的政府在不同的时间、通过不同的沟通渠道发出好斗的言辞——尤其是特朗普的个人推特账户。

伦敦国王学院科学与安全研究中心研究社交媒体和冲突升级的德鲁（Alexi Drew）指出，美伊关系是一个典型的例子，说明特朗普的推特外交使国际对抗变得多么困难和潜在危险。

“如果你从（伊朗外长）扎里夫和伊朗人的角度看，来自美国政府部门和特朗普推特的说法反复出现矛盾时，你很难确定美国的立场到底是什么。”德鲁说，”国防部说一套，特朗普推特说一套，内容前后不一。”

德鲁并不认为仅靠推特就能引发冲突。然而，”在既有的升级框架、叙事或者历史性冲突之下，使用推特导致意外或冲突升级，超过不使用推特的情况。”

肆无忌惮的独裁者

特朗普四年总统任期的另一个后果是，专制统治者在全球舞台上变得肆无忌惮。虽然其中有些独裁者在特朗普之前就已经在位，但是特朗普与他们的交往，从不加批评到表达钦佩，显示出对他们的执政方式的潜在认可，并透露出对违反人权的行为无意理会。

其中一个例子是记者卡舒吉(Jamal Khashoggi)遇害后特朗普对沙特阿拉伯的立场。在越来越多的证据表明沙特王室高层卷入这桩记者谋杀案的情况下，特朗普对沙特政府表示支持。

麦克米伦说，在特朗普的领导下，”他们（专制领导人）不会从美国受到任何阻击。”

中共第十九届五中全会前夕，中国上下已经在高调显示在科技太空各领域的进步，处处流露出与美国竞争较量的意味。由于美中双方在战略技术项目上的高度重合，美国方面也能洽如其分地表现自己的存在感。 比如美国白宫15日发布的《国家关键与新兴科技战略》，有将近20项技术领域被列入美国关键和新兴科技项目，包括人工智能、量子信息科学、通信和网络、半导体、军事等领域技术。中共中央政治局则在第二天的10月16日举行了以量子科技爲主题的研讨会，将量子科技上升爲国家战略性産业；10月18日，中国国家发改委又赋予深圳在人工智能、无人驾驶、大数据等领域的先行先试权。中美间的竞争也延伸到太空领域，中国航天科工集团正在发展的“空天飞行”计划，发展与美军的X-37B飞行器非常接近的“太空穿梭机”，不用火箭发射，直接在机场跑道起飞前往太空。
