# 鲁迪·朱利安尼：“Dominion不应该出现在任何一个地方的计票系统之中”

[!\[\]()!\[\](https://spark.adobe.com/page/H1Np3oXPcvl3J/images/0911b69f-7062-4850-a31d-2b9789a8185d.jpg?asset_id=647d8204-d68c-41bc-9aff-fa2761c48e70&amp;img_etag=%22c33fccd8e43976b35b1c3c300a08a504%22&amp;size=2560)](https://spark.adobe.com/page/H1Np3oXPcvl3J/images/0911b69f-7062-4850-a31d-2b9789a8185d.jpg?asset_id=647d8204-d68c-41bc-9aff-fa2761c48e70&amp;img_etag=%22c33fccd8e43976b35b1c3c300a08a504%22&amp;size=1024)2020年11月15日（星期日）08:47 AM，总统律师鲁迪·朱利安尼（Rudy Giuliani）在2020年11月7日在宾夕法尼亚州(Pennsylvania)费城(Philadelphia)举行的新闻发布会上对媒体发表讲话。 （Chris McGrath(克里斯·麦格拉思)/ Getty（盖蒂）Images图片社） 埃里克·麦克（Eric Mack）

川普法律团队协调人鲁迪·朱利安尼(Rudy Giuliani)表示，选举违规行为有太多巧合，不可能是偶然的，尤其是多米尼恩投票系统与委内瑞拉和中国相关。

朱利安尼在纽约上午WABC 770空中电台的“猫型圆桌会议”（”The Cats Roundtable”）节目上说:“奇怪的是，在川普以不到1%的劣势输掉的那些非常、非常接近的州，人们都在使用这些机器。”内华达州、密歇根州和乔治亚州使用这些机器。这些机器不应该用于任何美国选举。再说一次，它们是外国机器。

“它看起来像是一家加拿大公司；实际上，这家公司是由两个委内瑞拉人所有的，经营了大约20年，在很多地方被取消了资格，这会让你头晕。”

朱利安尼（Giuliani）告诉主持人约翰·卡西马蒂迪斯(John Catsimatidis)，法律团队将向法庭施压，因为“每台机器都必须在每个州接受审计。”

“多米尼恩正在29个州统计选票，”朱利安尼（Giuliani）说。“多米尼恩不应该在任何地方计算选票。当你稍微调查一下，你会发现多米尼恩用的是一个软件，Smartmatic，这是一个可以追溯到2004年的公司。

[!\[\]()!\[\](https://spark.adobe.com/page/H1Np3oXPcvl3J/images/104b5f67-285b-4827-abd4-ebbccb9f42ab.jpg?asset_id=b04c0e66-d753-4df3-93aa-e84853e5bbb2&amp;img_etag=%2283bac384910453a69cf59f08f363032f%22&amp;size=2991)](https://spark.adobe.com/page/H1Np3oXPcvl3J/images/104b5f67-285b-4827-abd4-ebbccb9f42ab.jpg?asset_id=b04c0e66-d753-4df3-93aa-e84853e5bbb2&amp;img_etag=%2283bac384910453a69cf59f08f363032f%22&amp;size=1024)

它是由两个委内瑞拉人和塞萨尔·查韦斯（Cesar Chavez）创建的。它在阿根廷和委内瑞拉都有既定的糟糕的选举历史。2008年，众议院概述了这一切。它因为严重不符合要求而被逐出德州。它仍然由这两个与马杜罗（[Nicolas] Maduro）关系密切的委内瑞拉人管理，多米尼恩公司并不是一家美国公司。

“这是一家加拿大公司。我们把选票送到美国以外的一家公司，这家公司与委内瑞拉和中共国结盟。这太过分了。”

多米尼恩投票系统公司(Dominion Voting Systems)首席执行官约翰·普洛斯(John Poulos)、曾为众议院议长南希·佩洛西(Nancy Pelosi)担任说客的加州民主党议员于去年1月，在美国爆发全球冠状病毒大瘟疫之前就告诉国会，多米尼恩投票系统与中共国有关联，其中包含有来自中共国制造的零部件。

“民主党人这么做已经有好几年了，”朱利安尼（Giuliani）告诉Catsimatidis。“他们做恶可以不受惩罚，因为他们只在民主党控制的城市这样做。你不会在奥马哈（Omaha），内布拉斯加（Nebraska）看到他们这样做。你不会看到他们在共和党的地方这样做。”

“你会看到他们在民主党控制的城市和地方这样做。”
