# 重要事件

- ** [12.18. Newsmax 最高法院驳回了对特朗普人口普查移民计划提出挑战的诉讼](https://www.newsmax.com/politics/supreme-court-census-trump-challenge/2020/12/18/id/1002203/)**

 美国最高法院上周五驳回了一起诉讼，试图阻止总统唐纳德·特朗普将非法居住在美国的移民排除在用于将国会选区分配给各州的人口统计之外的计划。6-3的意识形态路线裁决j结果是法院的6名保守派占多数，3名自由派持不同意见，使特朗普在总统任期的最后几周推行其针对移民的强硬政策，取得了短期的胜利。然而，在”当选“总统拜登（JoeBiden）于1月20日上任之前，川普政府正与时间赛跑，提出定义模糊的提案。如果特朗普政府完成计划， 法官们也可能重新提起诉讼。
 未签署的决定显示，”司法解决这一争端为时尚早”，部分原因是政府不清楚打算做什么。裁决指出，法院没有权衡特朗普计划的优点。
 以纽约州和美国公民自由联盟为首的挑战者表示，特朗普的提议将削弱拥有更多此类移民的州的政治影响力，包括民主党大的加州，因为该州人口数量不足，剥夺了他们在美国众议院的席位，以利于他的共和党同僚。”如果政府真的试图实施这项政策，我们将起诉。再次。我们会赢，”代表挑战者的美国公民自由联盟律师Dale Ho说。
 政府没有透露它使用什么方法来计算它提议排除的人数，或者哪些移民将作为目标。代理副检察长杰弗里·沃尔在11月30日口头辩论期间告诉法官们，政府可能会错过12月31日法定的最后期限，以敲定人口普查局向特朗普提交的报告，其中载有最终的人口数据，包括被排除在外的移民人数。据估计，有1100万移民非法居住在美国。挑战者认为，特朗普的政策违反了宪法和人口普查法，这是一部概述人口普查方式的联邦法律。
 根据法规，总统必须在1月初向国会提交一份报告，报告各州的人口及其有权的众议院选区数目。

** [12.18. The National Pulse 特朗普内阁将另外60家与中国军事相关的公司列入黑名单](https://thenationalpulse.com/breaking/trump-ccp-blacklist/)**
 美国商务部将60多个中国军事和政府相关公司列入黑名单，使他们无法利用美国技术。
 禁令包括中国最大的芯片制造商半导体制造国际公司（SMIC）。”我们不会允许先进的美国技术帮助建立日益好战的对手的军事。 在中芯国际与军工综合体的关联之间、中国积极实施军民融合任务和国家指示的补贴，中芯国际完美地表明了中国利用美国技术支持其军事现代化的风险，”商务部长威尔伯·罗斯（Wilbur Ross）表示。
 新的规定将限制中芯国际和超过60家与中国共产党有联系的公司”通过要求美国出口商申请向该公司销售许可证来获得某些美国技术”的能力。
 商务部补充称：”在10纳米或以下的先进技术节点上生产半导体所需的项目，将受到拒绝的推定，以防止此类关键支技术支持中国军民融合的努力。
 在另外60家公司中，”这些公司包括在中国允许侵犯人权的实体、支持南中国海军事化和非法海上索赔的实体、为支持中国人民解放军计划而获取美国产地物品的实体，以及从事窃取美国商业机密的实体和人员。

**[12.18. 大纪元新闻网 美核安全局網絡遭APT黑客入侵 影響6機構](https://www.epochtimes.com/b5/20/12/17/n12628408.htm)**

美國能源部週四（12月17日）證實，SolarWinds泄密事件背後的[APT黑客組織](https://www.epochtimes.com/b5/tag/apt%e9%bb%91%e5%ae%a2%e7%b5%84%e7%b9%94.html)還入侵了維護美國核武器儲備的國家核安全管理局（National Nuclear Security Administration，NNSA）系統。
 據「政客」（Politico）引述直接熟悉此事的官員消息說，能源部和維護美國核武器儲備的國家核安全管理局有證據表明，黑客訪問了他們的網絡，這是大規模間諜行動的一部分，至少有六七個聯邦機構受到影響。
 能源部的初步調查發現，聯邦能源監管委員會（FERC）、分別位於新墨西哥州和華盛頓的桑迪亞和洛斯阿拉莫斯國家實驗室、NNSA的安全運輸辦公室以及能源部的里奇蘭外地辦事處的網絡中發現了黑客的可疑活動。
 能源部發言人莎琳·海恩斯（Shaylyn Hynes）表示，調查發現，惡意軟件沒有進入關鍵的包括國家核安全管理局（NNSA）在內的防禦系統，黑客僅侵入了業務網絡。
 「當能源部識別出易受攻擊的軟件後，有立即採取行動降低風險，所有被識別為易受此次攻擊的軟件都已從能源部網絡中斷開。」海恩斯在一份聲明中說。
 黑客最早是通過入侵軟件公司SolarWinds獲得對聯邦機構網絡的訪問權，該公司向數百家政府和私營部門客戶銷售IT管理產品。
 美國國土安全部的網絡安全和基礎設施安全局（CISA）週四稍早發出跟SolarWinds相關的更新警報，指[APT黑客組織](https://www.epochtimes.com/b5/tag/apt%e9%bb%91%e5%ae%a2%e7%b5%84%e7%b9%94.html)對政府和私人網絡構成了「嚴重」風險。
 此次大規模入侵美國網絡的[APT黑客組織](https://www.epochtimes.com/b5/tag/apt%e9%bb%91%e5%ae%a2%e7%b5%84%e7%b9%94.html)至少從2020年3月開始，就在藉機入侵美國政府機構、關鍵基礎設施實體以及私營公司。
 CISA說，黑客展現出耐心、資源豐富且專注於攻擊，他們在受害人網絡上進行了長時間的活動；且在入侵時，黑客使用了複雜的技術手段，這讓網絡威脅難以檢測，也很難被清除。
 SolarWinds是美國聯邦政府部門和前十大電信公司的供應商、美軍的網絡服務供應商，它也是投票機Dominion公司的服務供應商。
