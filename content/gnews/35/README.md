## 十五歲沒有先天性過敏反應的兒童死于輝瑞疫苗（Pfizer Vaccine）造成的心髒病
` 西班牙巴塞罗那喜悦农场` [轉載自GNews](https://gnews.org/zh-hans/1163457/)

- 翻譯：the world


**西班牙2021年5月2日電/西喜社——**

根據美國國家疫苗不良事件報告系統中的數據，科羅拉多州的一名15歲男孩在註射有爭議的輝瑞COVID-19疫苗後僅兩天就死於心髒病，儘管此前沒有過敏反應史。

該案例在數據庫中以VAERS ID 1242573列出，表明該15歲男孩於2021年4月18日“接受了輝瑞/百安泰的疫苗接種”。他於2021年4月19日開始對該疫苗產生不良反應，在接種疫苗2天后於2021／04/20去世。 ” VAERS數據庫還顯示，他沒有其他疾病，沒有既往疾病，沒有已知的過敏，沒有先天缺陷，也沒有永久性殘疾。他只是在接種有爭議的輝瑞疫苗的兩天后死於“心臟衰竭”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/02034506/image-16.png)
該報告還指出，一名婦女在接種了與15歲死亡之前所用的相同的輝瑞疫苗後，經歷了全身癱瘓和劇烈疼痛。田納西州婦女布蘭迪·帕克·麥克法登（Brandy Parker McFadden）在註射疫苗後因灼痛而驚醒，不久就意識到她無法動彈，並被送到醫院。

儘管有與輝瑞疫苗有關的令人恐懼的死亡和致殘報告，但該公司仍堅稱其疫苗是安全的。事故發生後，輝瑞公司發給WKRN新聞2的聲明部分寫道：“迄今為止，全球已有超過2億人接種了我們的疫苗。

輝瑞公司的首席執行官最近還建議，注射有爭議的兩劑疫苗可能還不夠。該首席執行官聲稱，第二次注射後約六個月可能需要加強注射，然後需要每年接種疫苗以保持免疫力。
