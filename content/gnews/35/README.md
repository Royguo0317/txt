举报组织Project Veritas从脸书内部获得的文件中详细证明了脸书为审查COVID-19疫苗的关注而做出的努力。脸书通过评论降级在全球范围内打击 “疫苗犹豫”（Vaccine Hestinacy）计划。一位脸书内部人士告诉Project Verita，脸书试图在这些内容进入你的页面之前，在你看到它之前就加以控制。脸书的目标是减少用户对疫苗的犹豫，同时也减少发相关帖子。

这位脸书的举报人说，脸书根据对COVID-19疫苗接种的质疑或警告程度的评论，使用一个分级系统来排列和决定该评论应如何被审查或被埋没。举个例子，如果以第2级代表对接种疫苗的 “间接劝阻”，此用户评论将被压制。如果描述可能是真实的事件或可能引起安全问题的事实，此评论将被降级。评论是否真实、符合事实或代表现实并不重要。就算事件是真实的或对公共利益有贡献的，但任何引起人们对冠状病毒疫苗接种关注的评论都会被降级和隐藏。

Project Veritas在他们自己的网站上上传了整个脸书疫苗犹豫评论降级文件和整个脸书全球运营入门——健康错误信息文件。记者就这2个文件联系到了脸书的一位高级发言人，这位发言人只给Project Veritas发了一份简短而宽泛的声明作为回复，并未能解决脸书透明度这个最大的问题。Project Veritas创始人詹姆斯.奥基夫在5.24周一的新闻发布会上表示，他想知道马克.扎克伯格将如何为此辩护。他告诉Project Veritas的跟随者，脸书自己的员工也无法忍受他们在工作中亲眼所见的错误行为而来找Project Veritas。

作为2020年6月#Expose脸书系列的一部分，Project Veritas披露了脸书员工如何被命令监控、影子禁止和删除网站上的有关川普内容。今年5月初，脸书监督委员会宣布，脸书裁定维持该平台对前总统唐纳德-川普的禁令。2020年7月泄露的脸书电话会议上，首席执行官马克.扎克伯格违反了这家科技巨头自己的规则，但却说不了解关于COVID-19疫苗修改人DNA和RNA的长期影响。这段录像于今年2月公布，脸书将其删除。前自由民主党党魁尼克-克莱格(Nick Clegg)当时曾在镜头前争辩说，脸书的权力太大了。
