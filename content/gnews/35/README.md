在参议院听证会上，参议员兰德-保罗（Rand Paul）就功能增强研究、武汉病毒研究所的资金以及实验室事故的可能性等问题向安东尼-福奇（Anthony Fauci）博士提出质询后不久，福奇悄悄向Politifact提供了一些惊人的承认。
首先，他现在承认他 “不相信 “COVID-19是自然发展的；其次，也许同样令人震惊的是，他承认美国在研究冠状病毒方面与中共国科学家接触并帮助他们提供资金。
“不，我不相信[COVID-19是自然形成的]，”福奇在一个名为 “美国联合事实 “的虚拟活动中告诉Politifact的凯蒂-桑德斯。”我认为我们应该继续调查在中国发生的事情，直到我们继续尽最大努力找出发生的情况。”当然，调查的人说这很可能是从一个动物库中出现的，然后感染了个人，但也可能是其他原因，我们需要找出这一点。所以，你知道，这就是为什么我说我完全支持任何调查病毒的来源的原因。”
后一点确实呼应了福奇在参议院作证时对保罗的评论，但巧合的是，一个Politifact论坛实际上没有对福奇的其他一些承认进行事实核查，这些承认与他在参议院与保罗的交流中的证词不一致。
保罗就美国是否与中国一起帮助资助和研究冠状病毒向福奇提问，而福奇恰恰承认了这一点，并暗示如果没有这样做就是 “不负责任的”。
“实际上没有，这就是我说的重点，”福奇在谈到需要调查COVID-19的起源时说，”我认为保罗参议员所做的真正不幸的方面是他把与中共国科学家合作的研究混为一谈，你几乎不得不说，如果我们不这样做，我们几乎是不负责任的，因为SARS-CoV-1显然起源于中共国，我们很幸运地逃脱了一场大流行。
“所以我们真的不得不更多地了解那里的病毒。关于人们是否被蝙蝠病毒所感染”。
保罗暗示福奇帮助资助 “功能增强研究”，福奇 “断然 “否认，但福奇将保罗关于功能增强的问题混为一谈，认为美国在与中共国科学家研究冠状病毒感染方面没有作用，也没有资金，而福奇刚刚在上面承认了这一点。
他还向Politifact承认，美国确实帮助资助了武汉冠状病毒研究与 “中共国科学家”–尽管漏说为”中国共产党”。
“因此，在一个非常小的合作中，作为一项拨款的分包合同的一部分，我们与一些中国共产党人进行了合作……，中共国科学家”，福奇告诉Politifact，”他与此混为一谈的是，我们参与了病毒的创造，这是我听说过的最荒谬、最雄伟的跳跃。”
保罗的提问并不是指责美国创造了这种病毒–正如福奇自己所混淆的那样–而是暗示美国国家卫生研究院和福奇的国家过敏和传染病研究所间接帮助资助了可能创造了这种病毒的中国的功能研究。
