## 郝海東：我為什麼站出來！

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/06063554/timg_%E5%89%AF%E6%9C%AC.jpg)
作者：Diago

郝海東先生是足球上曾經最亮的一顆星星，在2020年6月4日新中國聯邦啟航之日，他，勇敢的站出來，在滅共戰場上做了最亮的一顆星星，我們一起來聽聽他在稍後與路德安紅的訪談中都說了什麼？為了更加精簡，對於郝海東先生在訪談中提到的具體人物和事件先略過，郝海東先生用事實說道理，我們看看郝海東大哥為什麼要站出來——

【中共的恐懼，就是我們的力量，我們的真實、我們的勇敢就是對這種體制最大的反抗、反對，對他們說“不”！因為我十歲也當兵，很瞭解這種體制，它是對人性的泯滅、對天賦的摧殘，不讓你有創造力，不讓你有基本的這種思考能力——獨立思考能力，所以，我們倆可能得過無數的榮譽、得過無數個冠軍，拿過什麼最佳射手、最佳運動員，所有的我的愛人葉釗穎也曾經有過世界的冠軍，我得到過無數的這種世界盃出線、萬眾歡呼，但是都不如我今天，我認為我們倆今天就是最高的榮譽！最大的榮耀！可以為新中國聯邦讀這份宣言，就是對這種體制最大的聲音說“不”，因為我們曾經經歷過。我點點滴滴我自己的人生經歷，我認為這種體制，我一直在說，他們應該管辦分離、黨政分開、司法獨立，在我當運動員的時候、在我在國內的時候，我就堅持我的觀點，那麼人類走到今天、人類文明發展到今天，共產主義、共產黨這種體制必須要退出歷史舞臺，這是對老百姓、對未來、對我們的孩子所有的人的一種生活保障、人生保障，而不僅僅在它們的這種統治底下的洗腦教育、所謂的綁架了十四億的中國人，它們把國家、政府、政黨混為一談，尤其是對我們運動員、對足球運動員、對所有的體育人的摧殘、褻瀆，那我們確實深受其害，我們所有的體育，他們只為了一塊金牌、只為了他們的政績，他們綁架了運動員跟這些體育行業的所有的從業人員，他們以國家的名義來去操縱運動員的這種參賽資格、誰得不得冠軍、誰能不能入選國家隊，他們打著國家的名義，包括現在的所謂的足協，它都不是啊，它們是非法組織。我再說一遍，它們是非法組織！中國足協是中國足球運動管理中心，它們不是中國足協的行業管理，一套人馬兩塊牌子，國際足聯明文規定，不允許政府插手足球的事務，但是我們是非法組織。】

【這就是我們最大的惡，對運動、對足球本身本質的這種摧殘、這種褻瀆、這種強姦，他們不尊重行業，不尊重行業規律，不尊重行業的從業人員，我都說郝海東當不當足協主席沒有關係，但是你們要真正的在足球的行業裡應該以行業人來管理，你不能拿當官的，誰官大誰來當這個、來任命，你應該行業管理，行業的人來專業的人幹專業的事兒，但是他們一直沒有來，一直不去啟動管辦分離，他們已經喊了無數、幾十年了，從來我沒有，到今天了已經所謂他們所謂的職業化，他們就是贊助商聯賽，誰有錢，那麼來投資、來養一個球隊，沒錢了，就撤了。包括大連萬達、大連實德、大連阿爾濱，包括無數個這樣的所謂的足球俱樂部，以市場化的名義最後走了這種偽職業。為什麼？因為我們沒有職業聯盟、沒有球員工會、沒有行業管理，他們就是行政命令，都很多的可笑至極的行政命令出臺：頭球一個算兩個，十二分鐘跑要跑到多遠才能上場比賽，這些都是對行業的強姦，但是我們這種在循環往復著。我作為一個十歲就從事這種專業、職業運動的人，不僅僅看到了它對行業、對這項職業的這種褻瀆跟強姦，重要的就是我通過這種現象看到了本質，就是他們對人的禁錮、對開賦人權的踐踏，對尊嚴、對一個人私人財產、私人權利的踐踏跟強姦，這就是為什麼今天我們倆站出來，說出我們內心的這種聲音，告訴中國共產黨，告訴我們的中國人民，告訴我們的老百姓們，你們應該覺醒了，歷史不能再這麼下去了！因為如果這樣的話，我們未來一定會比現在更慘！它們會跟蹤你，它們會蹂躪你，它們讓你沒有一點安全感。可以這麼說，路德，郝海東跟小葉不需要任何的什麼錢、榮譽、所謂的官職，我們不需要！新中國成立的那一天，新中國未來所有的運轉裡邊，郝海東跟葉釗穎不會在裡邊拿一分的利益、一分的權力，這就是我的內心最真實的聲音，是什麼？是未來一這要讓老百姓能有神聖不可侵犯的、天賦人權的這種私有財產受到保護、個人權利受到保護，民主、法治、信仰受到保護，我們就是這麼簡單。對，今天發自內心的心聲，所以我才會站上這個平臺，我也感謝文貴先生、班農先生，以及全世界億萬個對中國老百姓內心的愛的人。】

【中國人怎麼可能不行，我都說嘛，中國足球是中國社會的縮影，不是中國的教練員、運動員不行，是這幫孫子官僚，這幫當官的，他們草菅了這種行業規律，是他們的惡造就了我們最後成績的下滑。】

【就是通過這些事情，你就能看出來，他們對他們的信仰是多麼的輕率，根本沒有真正的、從內心的、他們發自內心的，對這種入黨是個神聖的東西，對他們來講只是一個工具，只是你未來要升到更高的級別的時候。】

【我想和他們勾兌的時候，這些人，我們有大把的機會。但我一直心裡面感受得不好，不是他們對行業的這種踐踏，包括我們的教練員們都已經慢慢的被他們給踐踏成、被洗腦、被他們弄成了很不好的這種行為習慣。】
