## 中共超限战灭港行动（廿九）雨伞运动 之 旺角铜锣湾事件
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1311759/)

#### 搜集/撰文：岁月如歌 / 封面合成：文粤


![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/10015820/6101.jpg)
9月28日凌晨，警方增援和武力升级一共发射了87枚催泪弹驱散示威者群众，最终失败，防暴警察迫于9月29日早上撤退。占领中环行动扩散开来，市民自发占领了旺角和铜锣湾。和平占中运动逐渐演变成为有市民自发上街占领不再是之前由戴耀廷教授所计划的有组织的和平抗议。在旺角弥敦道和亚皆老道交界处全日都有市民在马路上聚集，气氛平和，他们拉起“占领旺角”横额。有不少私家车货车也在亚皆老街路中心停泊，以堵住道路极大程度起到了占领的作用，铜锣湾也全日坐满了示威者。

国庆节前夕香港中文大学4千名学生深夜在百分大道开会决定通过无限期罢课，并提出3点诉求。一，行政长官梁振英及政改三人组(林郑月娥、袁国强、谭智元)下台；二，公民提名；三，撤回人大常委会的决定。 10月1日，学联副秘书长岑敖辉表示，学联已决定大专生将无限期罢课。如果特首梁振英在10月2日前拒绝下台，将发动占领和围堵不同政府部门的行动。他还呼吁示威者国庆节坚守铜锣湾、旺角和金钟三个据点，坚持的战斗到底。占中发起人之一陈健民也表示，占中行动至少持续到国庆节第二日，至于何时结束取决于政府如何回应，香港学联决定大学生将无限期罢课。香港电台则引述政府消息人士报道，特首与学联代表会面时开放态度，特首不会辞职。而占中人士已不稀罕特首对话，与其他官员的对话之门没有关上。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/10021224/6102a.png)图片来自：希望之声
香港政府用催泪弹无法阻止香港人追求民主普选的决心和勇气，三低特首梁振英也被不停呼吁下台，情况一有不妙，中共就会派出它的打手来救局。回看过往的所有动荡，如六七暴动，和2019年的反送中运动，套路如出一辙。此次也不例外，10月3日下午两点左右，在旺角弥敦道和亚皆老街示威者聚集的地方，出现大批反占中的示威者，数量是占中人士的数倍，只要这些人一出场秩序就会大乱。如果大家没记错的话，在占中运动筹备期间，即2014年7月3日，由建制派组成的保普选反占中大联盟。这些所谓的反占中人士一出来就是骂街，辱骂占中人士，掷杂物掷鸡蛋搅局，都是拿钱办事的货色，被电视台捕捉到分钱镜头。这次出现同样是四处挑衅、指骂和推撞，不断拆毁在场帐篷和路障，追打集会人士，还对集会人士和在场记者投掷杂物，警方不但没有维持秩序，反而在暗中支持反诉占中的人士。市民得知警察没有保护学生下班后纷纷赶过来支持学生，反包围了反战中人士。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/10015901/6103.jpg)图片来自：论尽媒体
到傍晚时分反占中人士也不断增多，秩序出现混乱，不时有打斗的事情发生。一名男子头部被打得流血，他称在十字路口被反战中人是袭击，他表示支持学生到现场保护学生，结果被十多人围殴拳打脚踢至受伤，他批评警方完全不执法，漠视市民安全。蓝丝则在一旁大呼清场，打死他，在场女记者也遭蓝丝男子用硬物打手臂。国外多家传媒如华盛顿邮报，CNN，每日电报，加拿大国家邮报均对反战中示威者称「暴徒」。占中人士表示，若警方不阻止反战中人士的暴力行为，就会终止与政府的对话。出于安全考虑和平，战争和学联宣布放弃旺角往金钟集会现场去，但大家都不愿意离开。晚上10点多特首梁振英呼吁在旺角铜锣湾现场的市民尽快和平离去，学生立即离开示威现场，占中示威者对警方的做法很不满，在凌晨2点后包围了警方一辆警车要求警方向市民交代，否则不让警车离开。在旺角多处也发生示威者包围警车，遭警方举红旗警告，并发生在无警告下喷胡椒喷雾。旺角事件中警方不作为，还推脱责任，警察公共关系科高级警司郭柏聪深夜会见记者表示，因为现场面积大，警方执法有难度。到凌晨4:30警方宣布在旺角的反占中冲突中，一共拘捕了20名男子，他们分别涉嫌在公众场合打架、非法集结和殴打。事件中，一共有12人受伤，六名警员受伤。警方同时透露，其中八名非法集结者，疑有黑​​社会背景。

黑社会背景人物出来搅局打斗，这不是什么新鲜的事情，在六七暴动中，黑社会福建帮是整个运动的主力军，而在后来的2019年反送中运动中721的白衣人也是黑社会份子，甚至有公开声称福建帮的黑社会高调到香港支持黑警打压香港人，这都是中共一贯的套路，每到香港人不退让，中共就会放出其圈养的狗咬人。

中共官媒《人民网——人民日报海外版》在事后发表题目，为《香港市民“反占中”力撑警察执法》称赞香港蓝丝发起的两轮反占中运动，“保普选反送中大联盟”搞签名活动大胜利，和反占中运动胜利。大量引用其在插在香港的老吹鼓手、大外宣报纸《大公报》和《文汇报》内容，批评送中运动是早有预谋公开的秘密，组织者早于去年七月份便计划鼓动万人占领街道，更有1000多名占中份子曾接受特训，目的在于挑战特区政府和中央政府。称占中运动是少数人绑架多数人，不是民主，并强调回归法治才能解决问题。这种贼喊捉贼的做法只有中共几十年如一日的使用是谁不尊重法治？是谁违反了声明？是谁说话不算数？ ，全世界的人都知道，中共自己也知道，但它总装得一副无辜的样子，都是香港人的错，是香港人首先违反了法治等等，颠倒是非黑白。
