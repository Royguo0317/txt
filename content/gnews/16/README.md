新闻二

联合早报4月7日报道[2]：中共国将采取一系列措施，推动1亿多新的脱贫人口进入中等收入群体，今年已确定了脱贫人口就业规模不少于3000万的目标，官员称目前进展相当顺利。中共国乡村振兴局的统计显示，2020年底中国建档立卡贫困人口人均纯收入达1万零740元人民币（2198新元），远远超过国际极端贫困标准。

中共国乡村振兴局就是当年的扶贫办，换个名字就代表中共国脱贫成功。“今年推动一亿多脱贫人口进入中等收入群体，今年已经确定脱贫人口的就业规模不少于3000万的目标，目前进展相当顺利。”这句话的意思是说这一亿多脱贫人口要达到3000万就业，言外之意就是这一亿多人口没有工作，那何来脱贫？这就是中共荒滩的逻辑。

二、拜登尚未和习谈论病毒起源问题

大纪元4月7日报道[3]：在白宫举行的1.5亿美国人接种病毒疫苗的新闻会上，福克斯新闻记者彼得·杜西问拜登，你提到554,064名美国人死于COVID-19。许多家庭想知道这是怎么发生的，以及它是如何达到这种程度的。你是否已与习近平谈过病毒起源。拜登回答说：“不，我还没有与习主席对话。”

1、自从1月20日拜登政府上台后，拜习之间经历过彼此试探的过程，双方试探完毕后，拜登政府确定将延续川普总统的灭共政策，拜登为避嫌就主动与中共保持距离，由布林肯国务卿或其他官员来推动针对中共的一系列制裁行动。

2、拜登总统在被问及病毒问题时，他并未绕开，而是直接回答还没有，这恰巧说明拜登政府现在非常重视中共病毒起源问题。中美谈判结束后，前CDC主任在CNN电视节目中说他认为病毒来自中共实验室，其他左媒和前政府官员也开始说病毒来源于实验室，紧接着国务卿布林肯站出来发言称一定要彻查病毒来源，美国将直面挑战。如果拜登正式和习近平谈病毒起源的问题，那就到美国正式向中共国摊牌的时候了。

3、拜登的回答非常决绝，丝毫没有帮中共打圆场的意思，这表示美国已经彻底丧失了与中共就病毒问题兜圈子的兴趣了，再结合Selliin博士这两天发表的推特来看，美国和西方国家的情报系统早已经开始独立调查中共病毒真相。

4、拜登政府在中美谈判时之所以没有谈及中共病毒问题有两方面原因，第一个就是中共私下与拜登政府进行了勾兑，第二个就是美国已经开始就中共病毒问题进行布局，如果美国急于在谈判中定义病毒来自实验室，这必然导致中共内部出现激烈的反应，中共如果狗急跳墙就很可能对台湾动手。所以，美国先与日韩两国举行会谈，着手建立美日印澳印太联盟，以及再次联合北约盟国，美国在与中共商谈病毒来源问题前，先与盟友就中共病毒真相达成一致，做好全面的军事部署，以此堵死中共的后路。

5、美国现在依然在不断印钱，一旦国内经济出现问题，股市出现崩盘，那所有的矛头都将指向中共。在WHO报告发布前，中共如果承认是病毒泄漏导致疫情爆发或许还有与西方国家谈判的余地，但由于中共党内斗争的缘故，习总加速师仍然坚持认为病毒来自自然，这就促使美国加速推进灭共进程。

三、习与默克尔为挽回合作再通电话

美国之音4月8日报道[4]：正在中共国和欧盟关系陷入低谷之际，中共国领导人习近平和德国总理默克尔在周三（4月7日）通了电话。习近平表示，在中欧关系面临“各种挑战”之际，希望欧盟能够“独立”做出正确判断。习近平敦促德国和欧盟与中国共同努力，保持双边合作的健康稳定发展。他还表示，中共国愿与国际社会共同努力，促进冠状病毒疫苗的“公平合理分配”，反对将疫苗“政治化”或搞“疫苗民族主义”。

1、默克尔在今年9月即将卸任，在她任职期间的德国与中共关系非常要好，现在几乎所有欧洲国家都加入制裁中共国的行列，德国目前的表现还较为迟缓，中共想借通话再次拉拢德国，以此重塑中共国与欧盟的关系。但是，德国已经与日本达成军事合作协议，这就是德国的战队表态，北约各国现在在军事上已经开始全面行动。
