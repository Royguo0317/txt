2021年3月5日，中共宣布为保证爱国者治港，将废除香港现有的选举法规，彻底摧毁香港最后一道屏障—司法制度，换言之，正式宣布《香港基本法》作废。这一公然撕毁中英联合声明的，违背“香港现有制度50年不变，高度自治”承诺的行为，受到了国际社会的强烈谴责，同时昭示着中共在本次以生物武器超限战打头阵的世界大战中，再下一城。

2020年4月，班农先生曾在战斗室（war room）节目中，对7000万美国爱国者喊话：中共已对美国开战，已对全世界开战！“中共制造了「珍珠港事件」，美国和世界人民被中共谋杀！现在是美国及全世界与中共的对决。

事隔近一年，中共发动的超限生物武器攻击，并没有像珍珠港事件一样，即刻引发美国参与到第二次世界大战，痛击侵略者。原因就在于，中共筹谋已久，实施完美犯罪的超限生物武器战争，不仅不宣而战，而且做足功课，让世界猝不及防，一败涂地；加上中共早早布局，联合世卫组织，全球左媒，及美国医学高层福奇这样的防疫专家带风向，使得中共发动生物战争的真相迟迟未能定性。敌人都还没能认清，纵有千般武艺美国又如何施展？

不仅如此，美国自开国以来沿用的全民投票选举制度，被以中共为首的邪恶轴心国成员，联合出手破坏，可见中共的渗透之深，布局之久，用心之毒。

郭文贵先生是第一位促使世界关注中共超限战的英雄。超限战是与传统战争不同的新的战争手段，是以包括贸易战、金融战、法律战、科技战、生物战等一切手段，超越传统战争手段范围的新型战争形式。中共出版的超限战一书，直言：“网路战、资源战、媒体战、金融战、文化战，战争已经远远超出穿军装的军人和飞机大炮的范畴，中共已经将所有的领域都军事化看待”。而且“超限战对于中国共产党并不是陌生的新名词，在国共内战当时并没有这名词的时候，就运用了很多超限战概念雏形，去对抗蒋介石的传统军队战争。”

郭文贵先生曾说，如果特朗普政府最后阶段推出的30多条制裁中共措施，早一点实行，中共现在早就嗝屁了。因此，击败中共不是没招，而是出师未捷身先死！如今的中共就是在跟时间赛跑，要在全世界力量聚集之前，将目标一一拿下。香港-就是一个重中之重，接下来的会有台湾，还有新加坡，马来西亚，泰国，缅甸准备顺势收入囊中了，国际社会在缅甸上的束手无策，也进一步增强了中共一统亚洲的野心。
