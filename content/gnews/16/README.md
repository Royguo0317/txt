中共的工信部5月13日发布关于下架侵害用户权益APP名单的通报。

2021年4月23日，工信部向社会通报了93家存在侵害用户权益行为APP企业的名单。截至目前，经第三方检测机构核查复检，尚有39款APP未按要求完成整改。各通信管理局按APP整治行动部署，积极开展手机应用软件监督检查，此次内蒙古、安徽、广东、四川、浙江省（自治区）通信管理局检查发现共有46款APP仍未完成整改。

此外，在近期检测中，我部发现天涯社区、大麦、途牛旅游、VIP陪练、脉脉5家企业在APP不同版本中反复出现同类问题，我部将依法暂停其违规行为，予以直接下架处理。

依据《网络安全法》、《电信和互联网用户个人信息保护规定》（工信部令第24号）、《移动智能终端应用软件预置和分发管理暂行规定》（工信部信管〔2016〕407号）等法律和规范性文件要求，我部组织对上述90款APP进行下架。相关应用商店应在本通报发布后，立即组织对名单中应用软件进行下架处理，并加强举一反三，认真排查、系统排查反复出现问题企业，严格落实企业主体责任，把好上架审核关。

我们从附件可看到，基本大部分的APP遭到审查、监督、整改的原因都是“违规收集个人信息”，“欺骗误导用户下载APP”这几个原因，如果说欺骗诱导用户下载的行为发生在新问世的APP上，那么还符合逻辑，但在本次名单中，类似“天涯社区”、“大麦”等APP都是有很大一批用户的，因此欺骗误导的罪名明显不符合逻辑。

然后说到“违规收集个人信息”问题，我们都知道，在墙国，根本没有“个人隐私”可言，换句话说，任何方式、随时随地，你的个人信息都可能被卖到各种领域中去，根本不需要经过个人的同意。现在居然用“违规收集个人信息”为由，对这些APP进行强制下架，不免找的借口有些牵强了。

以天涯为例，大家知道，天涯社区是一个提供论坛、博客、相册、影音、站内消息、虚拟交易等多种服务的持续很多年的网络社区。自我定位是“全球华人网上家园”，最开始进驻的网友大多是高校教师、中国社科院、新华社等机构的人员，所以天涯社区的定位一开始就是走高端用户路线，其内容也不是没有营养的，相反很多会成为大众讨论的焦点。这次中共的强势下架，是否将所有未来可能成为舆论传播的通道都提前关闭，如果照文贵先生所言，中共的防火墙必倒，那么这些聚集高知和社会精英的论坛，就是一个最好的传播真相，唤醒百姓的思想交流地。也是绝对会加速真相传播的有利渠道。这恰恰是中共最害怕的。因此，在中共真正灾难来临以前，他们将所有能想到的，会对他们产生不利影响的因素统统铲除，以谋求最后的挣扎来续命。

政治高压下，中共内部的舆论管控形势只会越来越收紧，强压会像紧箍咒一样笼罩在14亿百姓的头上，慢慢你会发现，你的嘴自动闭上了，你的耳朵自动关上了，你的眼睛自动失明了，那么这样的你，还是一个正常的活生生的人吗？有的时候，不是向强权低头，就能换来你以为的苟且偷生，越是装聋作哑，可能你连生的机会也没有，我们不得不思考，这样苟且活着的人生，有什么意义？
