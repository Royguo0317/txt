美媒报道，最近参议院两党达成一个协议使“一项旨在对抗中共国的经济和地缘政治野心的庞然大物的两党法案”。参议院多数党领袖查克·舒默的“头等大事”就是推动法案。据报道，“在关键投票保持了三个多小时之后”，这个法案的前景变得“更加光明”。该法案最后“以共和党的决定让该法案前进而告终”。当“一项关于两党贸易提案的协议”被推出后，两党达成了“最终的协议”。可是，共和党参议员仍然“认为这是一个破坏交易的协议”。根据报道，中共国主题可能是参议院今年可以实现“两党最后的重大胜利”。
