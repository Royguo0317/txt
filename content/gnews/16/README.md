立法会公务员及资助机构员工事务委员会昨日开会，讨论公务员宣誓事宜。委员会主席、工联会郭伟强发言时询问，若公务员接受「反华媒体」访问，会否被视为违反誓言。公务员事务局局长聂德权回应称，如果该人士以公务员身份接受访问，需注意言论及表达方式，但如果讲明是「反华媒体」，「这个问题本身都讲得很清楚。」

民建联梁志祥随后追问，香港电台及《大纪元时报》是否属于反华媒体，聂德权未有具体回应。聂之后出席另一个记者会，被追问政府如何定义「反华媒体」，他同样没有正面回应。

港府15日发通告要求18万现职公仆宣誓，并于4周内签署声明。如拒绝宣誓，当局会考虑终止有关公务员的聘用。华员会会长利葵燕在电台节目表示，早前就公务员宣誓效忠问题，向局方提出不少疑问，但至今仍未有答案。利葵燕批评，相关要求透明度不够高，质疑当局有些过于心急。

公务员工会联合会总干事梁筹庭接受电台访问时表示，担心日后就雇员权益上街争取亦有风险。他又表示，近日收到不少会员就宣誓查询，包括即将退休的公务员担心若果不签署声明，会否影响退休福利，亦有一些年轻公务员明确表示，不会签署声明。

**点评：**

聂德权出席记者会时，有记者提问若公务员接受「反华媒体」访问，会否被视为违反誓言？聂德权表示如果以公务员身份表达意见，需注意言论及表达方式，但如果讲明是「反华媒体」，「这个问题本身都讲得很清楚」（意思是不能）。

记者追问如何定义「反华媒体」？聂德权当时没有回答。

民建联 立法会议员梁志祥随后追问，香港电台及《大纪元时报》是否属于反华媒体，聂德权依然未有具体回应。

香港电台，作为香港特区政府商务及经济发展局辖下的部门，是香港唯一的公共广播机构及具公信力的电子传媒。令人费解的是，香港电台本身是公营机构的一部分，也是政府部门，现在却忽然变成「反华媒体」。难道政府部门内已经出现一些被定性为「危害国家安全」的电台吗？香港电台竟然悄悄地被扣上「反华媒体」的帽子？

最令人担忧的是，聂德权既不否认，也不承认，那就是默认。看来聂德权有必要好好澄清一下香港电台为什么是「反华媒体」？香港电台向来作为港府的官方电台，其运营资金来自政府，换句话说，港府是不是变相出钱助养了一个「反华媒体」呢？
