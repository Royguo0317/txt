中共和晋灵公相比如有不同，那就是中共对错误的处置方式更卑劣于晋灵公。可以说当涉及其根基问题时中共从来都是指天发誓拒不认错，因为他们从不懂得“人谁无过，过而能改，善莫大焉”的道理。他们认为认错就意味着投降，低头就意味着失败。所以每当中共犯错或犯罪被追责的时候，他们就会不惜动用全国之力，甚至还会借助早被他们控制了的黑暗势力，在众目睽睽之下用假、骗、瞒等手段来搪塞被他们戕害的人民，来欺骗整个世界。如有可能，他们还会借“灾难”可以“兴邦”之说把自己打扮成一个拯救人民于水火，拯救世界于毁灭的救世主，甚至还会寡无廉耻地自拉自唱，给自己戴上伟大、光荣、正确的花环，也即自诩中共自己是一个永不犯错的政党。

12月18日晚当我在浏览世界各地新闻时， “WHO将于1月至中国武汉调查病毒起源”的一则消息跳入了我的眼帘。看到这一突如其来的消息，我一点儿也不感诧异，因为对我来说，中共的这一招就如盛行于东南亚的西南季风一样，势必要来，差别只是来的是早还是晚的事情。12月18日中共借WHO之口将此消息公布于世，只能说明中共已然做好了完全的准备，也即无论是人证、还是物证，他们已将一切造假工作都准备就绪，只待WHO组织的一帮科学家黑帮兄弟进入造假现场，获取假的人证和物证。我们可以断定“世卫10位来自全球的科学家1月将到武汉调查新冠病毒起源”将是中共企图掩盖其“实验室制毒，投毒世界”这一反人类罪的现场实操，这个由中共和它的走狗WHO一干人马自编自导的障眼法游戏，是企图帮助中共抹掉其实验室制毒、故意投毒和危害人类罪的一场大戏，同样也是一场甩病毒于它国的双簧表演。

中共和WHO的联袂表演的世纪大戏就要开场。开场后会有什么样的事情发生呢？基于中共邪恶、死不悔改的本质及其过往的历史，以及最近几个月以来中共国有意放出的有关新冠病毒的消息，我不禁做出如下这样大胆的猜测：

中共造假历经七十年有余，造假能力已练得炉火纯青。金丝眼镜科学家据虚假证据写出的报告是那样的严丝合缝，是那样的科学，又是那样的理性。讲台下的老外科学家们正经危坐犹如中共国的小学生，被金丝眼镜“科学依据证明新冠病毒来源于美国，而非中国武汉”的演讲给糊弄的一愣一愣的。正当这帮傻白痴犹豫着是否该相信还是不该相信金丝眼镜报告中的结论时，忽然一阵风飘来了10位白衣女实验员。这十位各个长得白皙漂亮，且脖子底下全是腿，她们如火的热情立即融化了老外们那犹豫不决铁一般的10颗心。

于是乎，几天的武汉之行一俟结束，WHO的实地科学调查报告就旋即公布。报告的题目和金丝的惊人的相似，仅有的区别就几个形容词—–“铁的科学依据证明新冠病毒的确源于美国，而绝非源于中国武汉”
