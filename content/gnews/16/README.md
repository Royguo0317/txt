近日，单针新冠疫苗在上海，浙江宁波、温州、绍兴、台州，河南郑州等地开打。这意味着，目前多地已有3类不同技术路线的新冠病毒疫苗可供市民接种。

“单针疫苗”即由军事科学院军事医学研究院陈薇院士团队及康希诺生物联合研发的我国首个腺病毒载体新冠疫苗，全程仅需接种1剂次。

首先，腺病毒载体的新冠疫苗的方式，我们最早知道的是英国牛津大学阿斯利康的疫苗也是这个实现方式进行的，但是在前几个月已经被强制叫停，原因是出现“血凝”的副作用，这样就是说在接种后会出现严重血凝功能障碍，肺栓塞等病因，从目前的数据证明，这个疫苗方案是不成功的，不能用于大批量接种。

其次，为何现在中共国内会强烈推出一针的疫苗呢？第一就是目前在中共强制推广的疫苗接种方案，在全国各地接种范围收到了阻碍，因此推出只需一针即可，让百姓从心里上感觉更加方便，引诱尽快接种疫苗；第二目前国内及全球的疫苗都是背后的资本势力操控的结果，但是中共的疫苗也增加了统治阶层的强制管控百姓的一种手段，必须要推广落实，才能更好的配合他管控统治，另一个原因就是有可能在此过程中进行不为人知的生化实验，来培育杀伤力更加强大的生物武器。邪恶的中共毫无底线，什么都做的出来，只要对他的统治有利。第三就是疫苗经济，这个疫苗经济利益的大蛋糕，各个资本和背后势力都丧心病狂的想在这次屠杀中分一杯羹。
