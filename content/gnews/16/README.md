就3月18日中美高层对话，笔者特别想看看中共怎么说。笔者也知道不管中共怎么说，都是说给其统治下的老百姓听，给老百姓洗脑用的。所以，特别查阅了墙国限定可以浏览的网站。结果不出所料，3月19日央视新闻这张大嘴开了个头、打了个样【中美高层对话：美方开场白严重超时，对中国内外政策无理攻击指责】，紧接着国际商报、购团帮网、教育新闻网、三晋生活网、东方信息网、圈课网、风尚网等群嘴开始争先恐后地照猫画虎的传话，内容如出一辙。

这要是在从前，我也会觉得美国很无理。但现在的我和战友们一样，虽然被长期洗脑带来的疾患尚未痊愈，却疗效显著，凡事习惯去独立思考了。今天，我们就把这件事拿来当练习题，检验一下四年来自己的学习成绩。

单看字面，央视新闻想表达两个意思。

1、说美方的开场白超时了，而且是严重超时。于是，我去查足以证明中共的指责之正确的依据，没找到。但却查到了上图所示美方开场白所用时间，为两分钟。至于为何中共强调其”严重”，就更不得而知了。但对备受文贵先生启发的战友们来说，却可想而知。在中美关系如此紧张的当下，墙国报道中美高层会议，首先拿出个与主题无关的事来说事，这本身就令人质疑，其目的很可能就是转移国民的注意力，以掩盖中共被美国谴责的狼狈相。

2、说美国攻击中共了。查遍墙国可见的网站，没找到。笔者只好拐个弯去弄个究竟。于是，在一个叫做中华网新闻的网站上找到了其转自观察者网的内容。是不是像绕口令？真是费了一番周折，但事实就是如此。中共就像是我们生活中遇到过那些爱撒谎的人，讲话时会躲着对方的眼睛，所讲内容会尽可能不具体，不说实质性的东西。

文中描述杨洁篪的”霸气”有二：

① “我们希望这次对话是真诚的、坦率的。”这句话，作为中国人都替他害羞！地球人都知道啊！中共，你连自己的子民都骗，对自己的老婆都不忠诚，你也配谈真诚？

② “美国没有资格居高临下同中国说话，中国人不吃这一套。最后受损的是自己。”哈哈，连墙内人民听了都会想起外交部发言人那句老生常谈”搬起石头砸自己的脚”。

文中描述王毅的”霸气”如下:

“过去几年，由于中国的正当权益受到无理打压，中美关系遭遇前所未有的严重困难。”

后半句倒是事实，可前半句？？？所谓的正当权益，就是往美国派间谍？就是偷盗美国技术？就是操控美国大选？就是控制美国媒体？就是用孔子学院渗透美国？等等之类吗？

更主要的是，我很怀疑会谈现场这些话到底是如何说的？是如何被翻译传递过去的？因为中共很喜欢跟子民吹牛，很喜欢显示自己的擀面杖有多硬！但不管怎么说，我满网找到的根本不足以说明中共宣传机器的上述报道内容。当好不容易在墙国网站找到了会议视频时我好高兴！本以为总算可以看个究竟了，可是，很遗憾！竟然没有一个霸气回应的画面！

中共知道墙国百姓早已大多不会思考了，谁会怀疑央视新闻，也很少有人会追究其前因后果。于是，撒谎都不肯花成本了。
