大约一个月前，笔者发文报道了亚利桑那州的立法机关下令再次重新计票，这次是手工计票【1】，本月初笔者继续报道，虽然亚利桑那州参议院确定了计票审计小组，但马里科帕郡（Maricopa County）的监事会设置障碍，一会儿说不能在投票中心审计，一会儿利用媒体攻击审计人员【2】，等等。

今天《华盛顿观察》发文【3】，报道了相关的进展情况。

全文如下：

亚利桑那州立法者赢得官司后将对2020年选举结果进行重大审计。

亚利桑那州参议院将开始对该州最大郡的选票进行一次重大的审计，据报道，该郡官员数月来一直阻挠该州的选举审计。

“参议院花了2个半月的时间在法庭上获胜，以维护我们对选举材料发出传票的权利。”州参议院主席凯伦·范恩（Karen Fann）说，“又花了6个星期的时间研究、选出审计小组来进行全面的法证审计。”

亚利桑那州共和党控制的参议院自12月15日以来一直在向马里科帕郡监事会发出传票，并在法庭上争夺获得审计所需信息和（计票、投票）设备的权利。

一名法官支持了议会能够发出传票的权利，为对该郡选票进行“详细”审查铺平了道路，审查内容包括“检测机器、扫描选票、进行全部手工清点以及检查是否有任何的违规行为”。

现在，该州的共和党人正在敲定如何进行审计的最终细节，议员们指责郡政府官员试图“破坏”审计工作。

“马里科帕负责人拒绝让我们利用他们的设施进行审计，”范恩说，“甚至拒绝回答一些简单的问题，比如‘选票是如何封存的？’”

但是，立法者们能够确保在一个“2万多平方英尺的场所，即我们州的体育馆来执行审计”，并将在现场有24小时的物理和实时的安全监视。

“亚利桑那州参议院和审计员对审查结果没有‘预期’。”范恩说，“我们正在进行全面的法庭审计，以消除选民的担忧，或者，如果发现问题，我们必须在下次选举前解决这些问题。”

“我们从未指控任何人欺诈或不当行为，无论是硬件、软件，还是人员行为。”范恩补充道，“我们希望没有故意非法的篡改，但如果发现，我们将把信息移交给州和联邦检察长，以便他们采取进一步的法律行动，我们将继续进行适当的更正。”

范恩说，选举以来亚利桑那州居民签署的宣誓书暗示，“选票中的大量邮寄选票存在一些问题，这些邮寄选票本不应该从已知搬走或去世居民的地址寄出。”

“我们希望这次审计能帮助我们了解如何纠正这些错误。”范恩继续说。

但马里科帕郡发言人菲尔兹·莫斯利（Fields Moseley）说，该郡没有干扰审计，认为该郡官员只是试图“澄清”细节，而亚利桑那州共和党人的“投票决定藐视监事会成员”。

莫斯利指出，该郡发给参议院审计联络处的一封信指出，该郡已发送了数十万页索要的文件，但没有发送“投票、统计表和其他索要材料，因为该郡认为这样做是不合法的”。

马里科帕郡检察官阿利斯特·阿德尔说（Allister Adel），该郡“已经等待了一个半月的指示，等待何时交付剩余的索要材料。”

与此同时，马里科帕县监事会主席杰克·塞勒斯（Jack Sellers）告知范恩，此次审计“并非该郡和参议院共和党核心小组的共同努力”。

塞勒斯说：“马里科帕郡将不会与您的供应商沟通，也不会为他们解释亚利桑那州的法律。”还补充说，不确定的法律条文意味着该郡“不能参与支持您的审计，因为这样做可能会使其承担没有类似法律保护的责任。”

亚利桑那州成为本次选举中最具争议的摇摆州之一，《福克斯新闻网》（Fox News）在（选举日）晚间早些时候称拜登当选，但随后的一系列投票结果显示，拜登的胜算微乎其微。
