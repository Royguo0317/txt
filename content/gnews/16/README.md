- 1956年，2月14日苏联共产党在莫斯科举行第二十次代表大会，中共中央在主要方面采取维护斯大林的立场；
- 1956年，4月28日毛泽东在中共中央政治局扩大会议上说，艺术问题上的“百花齐放”，学术问题上的“百家争鸣”，应该成为我国发展科学、繁荣文学艺术的方针；
- 1956年，9月15日中国共产党第八次全国代表大会在北京召开；
- 1957年4月，《人民日报》发表社论：《继续放手，贯彻“百花齐放，百家争鸣”的方针》，中共中央发出《关于整风运动的指示》，决定在全党进行一次以正确处理人民内部矛盾为主题，以反对官僚主义、宗派主义和主观主义为内容的整风运动；
- 1957年6月，中共中央发出《关于组织力量准备反击右派分子进攻的指示》；9月14日中共中央发出《关于整顿农业生产合作社的指示》；
- 1958年4月中共中央在广州召开有部分中央领导人和中央有关部门负责人参加的会议，讨论十五年赶上英国的问题；
- 1959年4月18日－28日第二届全国人民代表大会第一次会议在北京举行，大会选举刘少奇为中华人民共和国主席；


中共建国后每一年的“大事记”中，基本上记满了各种会议、指示、报告、通知……自中共建党，他们从来都是避开那些真实的事件不提，而去用一些冠冕堂皇的词语描述。混淆视听、转移注意是他们的一贯伎俩。

虽然如此，但在他们“不得不记录”的相关事件中，根据史料“抽丝剥茧”，每个事件都会有千丝万缕的联系，我们只要找到这些承上启下的事件，就不难看出每段时期的真实历史。那么，建国后的“第二个”五年的真实历史又是如何呢？

看过上一篇文章的读者已经知道，建国后的第一个五年，中共在毛的带领下大搞运动、参与战争，国内一片血雨腥风。几个“杀威大棒”打下之后，国内的其他政治团体一片“宁静”。在接下来的这个“五年”中，我们先需要了解一个历史背景：“1953年3月5日，斯大林去世。几经权力斗争后，赫鲁晓夫掌握了莫斯科政权。”

斯大林作为共产国际的“老大哥”，中共一直为其马首是瞻。非“正统”出身的毛其实对斯大林同志一直骨鲠在喉，但是在大林同志的淫威下，毛也只能忍气吞声。斯大林一死可“不打紧”，放眼世界，论“国际共产运动”的资历，毛可是“水涨船高”。而在中共党内，“泥腿子”出身的毛派与“根正苗红”的国际派原本就针锋相对。斯大林的死，让中共党内的“共产国际派”统统靠边站了。然而……

1956年，2月14日苏联共产党在莫斯科举行第二十次代表大会，赫鲁晓夫同志在大会中对斯大林公开“鞭尸”。赫坚决的批判了斯大林的独裁统治，并坚持把其所有政策加以逆转。

“党内民主”的口号，在莫斯科被叫得响天彻地。莫斯科的这一“响”自然震动了北京政坛。自中共“七大”之后，刘把毛捧上了“神坛”，斯大林的死又可以让毛在国际共产运动中“独领风骚”，毛的独裁之路看似一片平坦。

然而，苏共的“二十大”在莫斯科的“这一响”，让中共党内的领导人刘少奇、周恩来、彭德怀等人，流露出了与苏共步调一致，反对独裁，进行党内民主的意愿。天生一副“独裁坯子”的毛，敏锐的嗅到了他独裁统治的危机，于是他“摇身一变”，从反国际派的土共领袖，变成了斯大林幽灵的辩护者。毛挺身而出，力排众议，带领中共中央对外宣部，坚持了维护斯大林的立场。
