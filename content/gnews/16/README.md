**西班牙2021年2月15日电/西喜社——**卫生官员正在敦促更严格的中共病毒限制，并在斯里兰卡部分地区进行部分封锁，因为患者被证实有更具传染性的英国变种。

卫生官员周六表示，该印度洋岛国多个地区的患者已被证实患有一种变种，这种变种首次在英国出现。

“除非立即采取包括封锁在内的强硬和有效措施，否则新变种传播到其他地区的风险更大，因为已经出现的阳性病例数量正在上升，”公共卫生检查员协会主席Upul Rohana周一表示。

斯里兰卡去年3月发现首例阳性病例时，曾进行了为期一个月的封锁。去年10月，在爆发两起新的疫情后，首都及其郊区再次实施封锁。

斯里兰卡目前已确认75653例，死亡397人。
