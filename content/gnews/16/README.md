#### 美中会谈

据3月18日福克斯新闻，中美两国在从人权到国家安全等问题上交锋，中共方无视对维吾尔人、藏人、香港人的虐待，反咬所谓美国的“人权”问题
![]()![](https://gnews.org/wp-content/uploads/2021/03/blinken-with-yang.png)[https://www.foxnews.com/world/china-slams-u-s-human-rights-issues-ignores-abuses-against-uighurs-tibetans](https://www.foxnews.com/world/china-slams-u-s-human-rights-issues-ignores-abuses-against-uighurs-tibetans)
美国和中国的高层外交政策官员周四在阿拉斯加安克雷奇举行了会晤，这是拜登总统上任以来两国高级官员的首次面对面会晤。峰会因中方倒打一耙批评美国“人权”问题而变得尖锐。

杨洁篪说，”我们希望美国在人权方面做得更好。”“事实上，美国国内在人权方面存在很多问题，这是美国自己承认的。在美国在人权方面的问题是根深蒂固的，他们不是过去四年才出现的，比如说黑命贵问题。”

面对中方虚伪的打击，美国国务卿安东尼-布林肯指出中国在香港、台湾以及对新疆维吾尔族穆斯林民众实施的侵犯人权行为。他说，”这些行动都威胁到了维护全球稳定的规则和秩序，””这就是为什么它们不仅仅是内部事务，也是为什么我们觉得有义务今天在这里提出这些问题。”

杨洁篪谴责美国对中共的坚决批评。他说”中国在人权方面取得了稳步的进步，”

虽然预计两国将讨论冠状病毒大流行和考验两国的地缘政治挑战等棘手问题，但争吵的气氛让进一步讨论进展不顺。
