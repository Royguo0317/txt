摘要：中英香港谈判的始末，香港问题的历史渊源。郭文贵先生对香港风云的警告，共产党渗透香港的模式将在美国和全世界复制。

1982年4月，英国在遥远的海外领土、南大西洋福克兰群岛(Falkland Islands)和毗邻的南美最发达国家阿根廷打了一仗，首相玛格丽特·撒切尔夫人领导的英军大获全胜，并导致阿根廷当局的政权更迭。带着大英帝国的无限风光和保守党的强硬姿态，同年9月撒切尔夫人来到北京与邓小平会面，商谈另一个英国海外属地香港的主权交接问题。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/25120554/09bloth-rdv-china-jumbo.jpg)1982年撒切尔夫人和邓小平在北京会谈
从法理上讲，香港岛和九龙半岛分别因为《南京条约》(1842清道光二十二年)和《北京条约》(1860清咸丰十年)割让给英国，是英国领土。而其余的新界地区(1898光绪二十四年)则被租借给英国，租期99年。1997年是新界租界期满之时，然而在中英谈判中，中共政权不承认清朝签署的《南京条约》和《北京条约》，要求收回包括租借的新界和割让的香港岛、九龙半岛的整个香港地区。

共匪僭政大陆后，从上到下都在政治斗争的绞肉机中互相残杀。战后的世界快速发展，相比之下，属于自由世界的香港在西方体制下高度繁荣。到文革结束后CCP才意识到，还有20年时间就要面临香港主权问题了。对比香港当时的繁荣，文革后的大陆一穷二白。香港命运的幕后操盘手CCP从来没有把香港人单纯的理想当真。相反，30年来，CCP酝酿着一个巨大的阴谋。也许这是一种心理的变态。就像一个破败的家族，出走了一位美丽的小妹妹，待到她带着财富回馈家族时，却面临家族的嫉妒。CCP对香港也是如此。

结束文革不久的共产中国弱不经风，共产主义在冷战末期也失去了动力。为了说服英方，邓小平指出：“使香港人放心，更感到我们政策的连续性、可靠性 … … 一个中国，两种制度。我们的社会主义制度永远不会改变。但是，在香港问题上保证其现行的资本主义制度和生活方式50年不变。”

1984年12月19日，中国总理赵紫阳和撒切尔夫人共同签署《中英联合声明》(Sino-British Joint Declaration)。春风得意的赵紫阳想不到五年后他将永远失去自由，撒切尔夫人想不到风光无限的大英帝国签下的文件，成为对方眼中“不再具有任何现实意义”的“历史文件”。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/25120917/96764673_039916053-1.jpg)撒切尔夫人与赵紫阳签署《中英联合声明》 图片来源：AFP
1984年《中英联合声明》，1997年英国移交主权，到2014年的雨伞运动，香港人经历了从期待到失落、抗争的历程。2017年7月1日，香港主权移交20年，习近平赴香港出席香港特别行政区第五届政府就职典礼，新任行政长官就是在多年失败的政治改革下产生的林郑月娥。

时任英国外交大臣鲍里斯·约翰逊(Boris Johnson)称，法治、独立司法体系和自由媒体是香港取得成功的关键，香港未来的成功无疑将取决于《中英联合声明》赋予香港的权利和自由。与此同时，美国国务院发言人也表示，对任何侵犯香港公民自由和新闻自由的行为表示关切。面对西方对香港现状的批评，6月30日，外交部发言人陆慷表示，《中英联合声明》作为一个历史文件，不再具有任何现实意义。希望上述人士认清现实。

20世纪80年代，CCP用正能量的国家大义，贪婪地从香港吸取投资。到最近十年，大陆经济发展后，便开始煽动大陆人对香港的嫉妒和仇恨，同时勾结香港的卖港集团破坏香港民运，最终发展为大陆假扮的香港警察直接在香港街头虐待、强奸、屠杀香港人。

CCP的目的不仅是让香港成为它的奴隶，成为中国的普通城市。由于历史上的落后和嫉妒，心理变态的CCP还要把香港这颗东方之珠在世界面前踩碎，让全世界人摒弃香港、对香港避之不及，让大陆的粉红对香港极尽嘲讽，CCP要用香港的堕落，用香港人的血，去证明只有它CCP的邪恶制度才是“人类命运共同体”的选择。

从1984到2019，35年终于让香港和世界人民看清了CCP的丑恶面目。2019年的香港黑云压城，腥风血雨。CCP的魔性没有止境。在2019年香港大屠杀后期，CCP开始释放生化武器，又在2020年2月29日分水岭之后，把对香港的魔性放大到整个地球。2020年11月3日，美国大选开票，一时间乱象叠出、群魔乱舞，就像香港风云在美国的更大规模上演。

2020年11月7日，沉默了21天的郭文贵先生直播，他说:

“美国的法治在哪儿呢！人类非常脆弱，当你看到香港的孩子流血的时候、香港的孩子消失的时候、香港的孩子哭嚎的时候，被强奸、被轮奸、被自杀的时候，世界是沉寂的。我一再地说、一再地讲，这一天就会在美国发生，今天你们终于让美国看到了。郭文贵推特三年前被封没人管，YOUTUBE被封没人管，把我们满大街赶着，最后逼出了GTV、G-News。香港的孩子流血你也不管，表演性的立了法，没有人去执行。死的人没人管，一万多人被自杀，没人管！今天到达了白宫，让你川普总统的推特被关掉、让你这Lou Dobbs(卢·道布斯，FOX台著名主持人)这所有的人这些都给你关掉。包括不仅让你流血，明确告诉你，你不选我、我就暴乱。你不选我，我就把你给撵出去；你不选我，那就是你是造假；你不选我，大家都没有工作。

“美国西方的法治系统，对宗教的信仰，以及社会安全性，这一次彻底被打破。是如此脆弱，就在资本面前、就在共产主义面前、就在中南坑十几个人面前变得不堪一击。人类非常脆弱。所有这一切的不公在香港去年都演了一遍，这就是美国版的香港运动。灾难还没有到来呢！”

文贵先生在数年的爆料革命中反复强调，香港打开了灭共的大门。恶魔始于香港，必将终于香港。概括而言，1997年前的香港，是普通法系的法治社会，又是缺少普选的非民主社会。1997以后，香港社会的诉求是通过民主改革实现民主社会的治理目标，让香港成为既法治又民主的社会。

然而，残酷的现实告诉香港人，香港的主权从自由世界的英国移交到僭政的CCP后，香港人既得不到理想的民主，还会失去多年来引以为傲的法治，最终还会有一大批建制派香港人失去信仰，沦为出卖香港的CCP的奴才。十几年后，香港人会意识到，大陆水深火热的CCP专制，和香港一枝独秀的自由民主法治是不可能共存的。香港人用热血书写了香港人对自由民主法治的期许，而这一切将会在CCP灭亡之后成真。

我们试图在《香港风云》中，把香港的过去、现在，香港和世界的关系，在千头万绪中稍加梳理。感谢香港人的不屈不挠，感谢郭文贵先生对香港抗争运动的支持。
