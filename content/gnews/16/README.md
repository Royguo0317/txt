**撰稿：脐橙君**
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/28193530/15-14.jpg)
来自新华网5月28日的消息，据中国疾控中心官网最新消息，2020年12月15日至2021年4月30日，31个省、直辖市、自治区以及新疆生产建设兵团报告接种新冠病毒疫苗2.65亿剂次，报告预防接种不良反应31434例，报告发生率为11.86/10万剂次。其中，一般反应占83%，异常反应占17%，严重异常反应发生率为0.07/10万剂次，发生概率在极其罕见（低于万分之一）的范围以内。新冠病毒疫苗的一般反应、异常反应发生率均低于2019年国内其他各类疫苗的平均报告水平。

当全世界对疫苗的推广大行其道之时，对疫苗副作用的关注也伴随着接种人员的数量增多而越来越引发科学界和医学界的热烈讨论。

引述《每日邮报》消息，5月25日，世界上第一位接受辉瑞疫苗注射的老人在英国考文垂大学医院去世了，这则新闻引发了大讨论，将大众的注意力转向至第一批注射疫苗的群体，是否已经开始显现了副作用。这位老人是在去年12月8日接受中共病毒疫苗的注射的，到去世，差不多半年时间，这也是科学界和医学界提出的疫苗副作用出现的时间截点。虽然文中将老人的死亡定义为因中风引起的原因，但中风是否为疫苗所致？疫苗注射半年以后，会引发全身免疫系统的什么病变？都是需要我们持怀疑的眼光做进一步跟踪和分析，毕竟，疫苗的接种面积太大了，如果注射后真的会在身体内埋下一个个隐患，不知何时引发疾病的话，那将是全人类的灾难。

除了这位老人的离世之外，我们也经常看到注射后没几天就死亡的案例。5月6日，一位模特接种了阿斯利康的中共病毒疫苗，于5月19日陷入昏迷，5月22日被宣布死亡，她没有任何潜在的健康问题，而是接种疫苗后引发严重血栓死亡的。同时，BBC女主播在接种阿斯利康疫苗后，也是因为出现血栓，在医院抢救无效后身亡，年仅44岁。这也就是闫博士一再强调的为何SARS1之后没有研发成功有效疫苗的原因，因为无法解决ADE效应对身体带来的严重副作用，而血栓正是ADE效应引发的严重不良反应之一。

上述都是因疫苗死亡的案例，那么由于注射疫苗而导致的面瘫、皮炎、过敏等等副作用的出现就更多了。一位科学家在公开的论坛上指出了MRNA疫苗的副作用，无论是什么年龄，只要注射了MRNA的疫苗，寿命都会大大缩减，除此以外，还会产生过敏反应、神经认知问题和各种炎症，当然不孕不育也是结果之一。

那么此时，墙国的百姓会说，“对啊，就是因为MRNA有严重的副作用，所以中共疫苗才因此采取灭活疫苗的方法啊。”我们再回到中共疾控中心发布的这则消息，文中的寥寥数字就是想传递中共疫苗是如何安全的潜台词，但你们的数据真实吗？你们有跟踪过所有注射疫苗后的被接种人的数据吗？你们提到的“一般反应”、“异常反应”、“严重异常反应”的标准又是什么？

就像闫博士所说，**一个执政70年，至今没有成功研发过一款有效疫苗的国家**，一个执政70年，靠偷抢骗获得科研能力的国家，有什么自信称你的疫苗水平在西方世界之上，而相信这个谬论的人，你们的逻辑又是什么？

在墙国，不是没有接种疫苗后发生事故的情况，我们也曾看到注射后当场倒地的视频流出，但很快就被和谐，注射后严重面瘫的视频流出，但很快也失去踪影，墙国靠信息的封锁，就掩耳盗铃地认为自己的疫苗老子天下最先进，他们认为只要掩盖这些不良反应的报道，就能欺骗所有人，让小粉红们集体高潮，以为中共疫苗天下最安全，从而大肆宣传西方疫苗造成副作用的案例，其目的还是他们的政治操弄，为了从疫苗经济中获得更多的利益。

文贵先生说，**中共疫苗的所有背后股东都是200个红色家族的控制**，从一开始用粮油赠品的小恩小惠吸引民众注射，看效果不好，就下达任务到基层，挨家挨户催促注射，效果还是不理想，就开始利用局部的疫情造成的恐慌，配合注射疫苗的必要性宣传，让民众大规模主动地争抢着注射。看来最后一招的确奏效了，安徽、广东、深圳等地因为疫情爆发原因，导致民众疯狂地注射疫苗，正是CCP最乐意见到的场景。如果不出所料，这个办法会延续到更多的省市，然后完成他们制定的所谓90%以上的疫苗接种率的政治任务。

**文贵先生说，人生在世，最最关键的就是决策能力，**在不知道CCP病毒根本原因的情况下，任何疫苗都是对人类的伤害。这就是闫博士一直挂在嘴边的一句警告：人类的时间真的不多了。面对这次中共制造的超限生物武器对全世界造成的伤害，难道我们不应该首要任务是搞清楚这个生物武器的机理吗？如果连最本质的东西都没有办法知道原因的话，那么还有什么资格让大家信服“盲人摸象”后得出任何结论呢？
