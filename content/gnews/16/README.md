有**570**万人口的新加坡，在几个月前报告染疫确诊人数**6.1**万人，其中大部分为住在外籍工人宿舍的外国劳工，之后几个月疫情控制得非常好，当地每天的感染人数几乎为零或一位数，但几周前疫情开始恶化，感染人数一直在上升，上周四和周五两天确诊了**48**例本土感染，周日则上升到**38**例，新加坡因此关闭了几乎所有的学校。

和其它国家一样，新加坡疫情的再度蔓延，是因为遭遇了毒性和传染性更强的突变病毒的侵袭。新加坡教育部长陈春星说，某些突变病毒的毒性要强得多，它们似乎会攻击年幼的孩子，威胁更多的儿童。陈星春进一步援引新加坡卫生部高级官员的话说，从印度传入的双突变病毒**B1617**，似乎对儿童的影响更大，目前已有**4**名儿童确诊。

今年**4**月中旬曾有报道一名巴西的健康儿童，感染**CCP**病毒出现中风（血栓）症状后死亡，一名几个月大的加拿大婴儿感染了**CCP**病毒，但当时这些婴幼儿感染的还是**CCP**病毒的原始毒株。巴西圣保罗大学流行病专家马里尼奥说，当前发现染疫后发生中风症状的“多系统炎症综合征”的患者正在增加，儿童也不能幸免。马里尼奥称，截止**4**月中上旬有记录在案的，**CCP**病毒已使**2060**名**9**岁以下儿童失去了生命，其中有**1302**名婴儿。可见，中共发动超限生物战，就是要毁灭人类！
