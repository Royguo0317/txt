5月21日，日本雅虎網站登載了【每日新聞】的題為【中國當局からスパイ容疑で116日間拘束された「在日華僑」“恐怖の取調べ”と“真の狙い”を語る】的文章。題目大意是：駐日華僑訴說：以涉嫌間諜罪被中共當局拘留116天的旅行，“恐怖的審訊”和“真實的目的”。控訴者是75歲的老華僑，岡山縣華僑華人總會劉勝德會長。對劉勝德會長進行采訪的是日本記者吉村剛史。通篇不僅報道了劉勝德會長被中共拘留116天的遭遇，字裏行間更是藏盡中共內鬥的醜態。

之六【極刑の示唆も】（中文：還被暗示過死刑）

每日三餐都是普通的中國家庭料理，70歲的劉先生是在日本出生和長大的，對他來說食物太油膩，吃不慣。因此，返回日本時體重下降了很多。但是，與蘇州的收容所相比，天津的收容所房間更大，肥皂和紙巾等日常用品也要好於蘇州。所以，劉先生感覺天津的收容所好像是專門用於收容高級幹部的。而且，食品也好多了。他補充說：還提供熟悉的食物，如天津包子。然而，想去廁所時，必須舉手向官員報告這一非常不自由的要求仍然沒有改變。 他回憶說：”在最初的40天裏，我甚至不敢申請洗澡，因為擔心在陌生的環境中會因感冒或其他原因而生病。
劉先生說：在被拘留期間感到非常不合理，難以理解！他曾強烈抗議：我一直在支持我的同胞，為什麽我就成了間諜呢？
審訊者在暗示劉先生可能會被判死刑或者被長期拘留的同時，還說：負責向日本警視廳提交本次訪中活動日程的岡山縣華僑華人總會的前工作人員，應負監督責任。他們還詢問了劉先生與中國外交官的關系，特別是在王毅擔任駐日大使期間與他的關系。
