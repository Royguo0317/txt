04月05日，俄罗斯总统普京签署了一项关于规范教育活动的法律。相关文件已于4月5日在法制信息官方门户网站发布。

这些是对《俄罗斯联邦教育法》的修正。文件提出了“教育活动”的定义，政府有权决定教育活动的顺序和形式。教育科学部和教育部将不得不就教育机构与国际组织合作的可行性发表意见。

作者在法案的解释性说明中指出，现行法律“为反俄势力打着教育活动的幌子在学校和学生环境中无节制地进行各种宣传创造了前提条件”。

“法律禁止利用教育活动煽动社会、种族、民族或宗教不和；禁止宣传基于社会、种族、民族、宗教或语言归属或其宗教而鼓吹公民的排他性、优越性或劣等性；包括关于各民族历史、民族、宗教或文化传统的错误信息；禁止煽动采取违反俄罗斯联邦宪法的行动。”

该法案引发了学界的不满。接受RBC采访的科学家表示，该法律将使邀请外国专家变得更加困难；对于许多志愿者项目来说，它可能会导致他们的活动受到限制。有人认为，法律“有效地堵住了那些做科普讲座、做教育工作者的嘴”。

该法将于2021年6月1日生效。
