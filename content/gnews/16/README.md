## 【生化危机】5: 闫博士的发现
` 纽约香草山农场` [轉載自GNews](https://gnews.org/zh-hans/618522/)

作者：香草山写作组 Tiffany 的早餐 校对/发稿：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/03014137/logo-3-1.jpg)



摘要：闫丽梦博士奉命调查武汉疫情，发现其中猫腻，向香港方面汇报无果。出于公义她联系了路德，路德公开了闫博士的发现。中共发现无法掩饰，立即承认人传人，并开始大规模封城。

香港大学公共卫生学院的研究人员、临床医学博士(MD)及哲学博士(PhD)、病毒学家闫丽梦博士是全球冠状病毒研究的专业领域小圈子中，最核心的研究人员之一([生化危机 4: 香港P3实验室](https://gnews.org/zh-hans/615765/))。2019年底，CCP病毒刚刚在武汉出现。12月31日，香港大学公共卫生学院教授、WHO顾问潘烈文(Leo Poon)派闫博士去秘密调查中国大陆究竟发生了什么。由于当时中国政府拒绝让海外专家，甚至香港专家去做调查，所以闫博士开始动用她在大陆多年的医学领域的社会关系打听消息。12月31日，她从中国疾病控制中心打听到，已经发现了家庭聚集案例，CCP病毒会人传人。

闫博士在来自不同医院的临床医生群里，注意到已经有人开始讨论这个类似于SARS的病毒的信息，却有人警告“不要问”，说“这个太敏感”，“我们不能谈这个，但我们需要戴口罩”。 她还了解到，武汉的人传人已经非常严重，许多病人不能得到及时治疗和诊断，医生和病人都没有可靠的防护措施。中共政府不允许发布此类信息，医生甚至疾控中心的人都很无助。

作为病毒学家和医生，闫博士非常清楚这个病毒的风险，从科学和人道的角度，她认为应该尽快透露给公众，采取措施阻止大爆发。她把信息第一时间报告给了潘烈文，但潘烈文却要求她保持沉默并要小心，甚至警告她“不要触碰红线”，不要越过中共的程序和规范，否则，“我们会有麻烦”，并且会“被消失”。潘烈文以及裴伟士(Malik Peiris)([生化危机 3: 楞伽山之妖](https://gnews.org/zh-hans/615608/))都没有任何回复。

闫博士对于武汉悄然爆发的病毒有着高度的专业敏感，她开始从专业的角度研究武汉的疫情。随即，她发现这种病毒正是来源于控制在中共军方的舟山蝙蝠。南京军区军事医学科学研究所从舟山的蝙蝠上发现和分离了两种冠状病毒ZC45和ZXC21，之后经过改造形成CCP病毒这种生物武器。此外，闫博士有依据相信病毒是定点投放。武汉华南海鲜市场的冰和水的环境，适合病毒生存，非常适合投放。

闫博士没有想到，一切似乎都是冥冥中自有安排。2014年，刚刚从广州南方医科大学获得博士学位的她，来到香港大学潘烈文的研究团队做博士后研究员。那时正值香港政改图穷匕见，人大831释法，雨伞运动声势最浩大的时期([香港风云 5: 雨伞运动](https://gnews.org/zh-hans/591818/))。闫博士一边做科研，一边不忘关注香港岛上发生的抗争运动。2019年，持续多年的香港抗争升级为反送中和大规模的警暴([香港风云 9：反送中运动](https://gnews.org/zh-hans/593671/))。作为一名实验室里的科研人员，闫博士对于社会问题有了独立的见解。

由于郭文贵先生在海外持续的打假，许多扮演成海外民运的特务纷纷现形，路德访谈节目在众多海外中文自媒体中一枝独秀。闫博士同情香港民主运动，了解共产党和香港当局的无耻和残暴，自然成了海外爆料革命的支持者，是路德访谈的听众。她开始冒着巨大的风险和路德联系，给路德提供重要的信息。闫博士担心自己随时有不测，声明自己不会自杀。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/05030146/p1606049213_36745.png)路德在2020年1月19日美东时间上午节目中展示的来自闫博士的信息
2020年1月19日，路德在访谈节目中公布了闫博士的发现。中共方面终于确定，CCP病毒的事情再也藏不住了。路德的节目播出四小时内，中共迅速反应，把病例数从62改到198，承认病毒人传人。习近平发表了2020年第一个声明，称这是一个新的冠状病毒，是一种普通传染病，与SARS同一级别。然而3天后的1月23日(腊月二十九，除夕前日)，中共封锁了武汉，并开始迅速发展为全国范围的封锁。

中共内部对于消息的走漏也极为震惊。路德访谈作为非营利性的海外中文自媒体，节目的内容却可以打乱中共的步伐，可见对中共内部影响之大。中共一方面调整政策，一方面开始寻找爆料者，一张巨大的网无声无息地向闫博士袭来。

和闫博士联系后，路德和郭文贵先生开始安排顶级的生物科学界人士和闫博士沟通。在一次视频会议中，在国际冠状病毒研究的小圈子中，这个人正好熟知裴伟士(Malik Peiris)([生化危机 3: 楞伽山之妖](https://gnews.org/zh-hans/615608/))，于是向裴伟士询问闫博士的情况。这立即引起了裴伟士的警惕，巨大的危险开始走向闫博士身边。闫博士夫妇和裴伟士在同一个研究团队工作。闫博士的丈夫支持她的研究，可是当她要求两人一起去美国爆料时，他丈夫被激怒了，并责备她，因为他对中共非常恐惧，担心被中共灭口。

闫博士发现能够信任的只有路德。路德了解中共的手段，结合他的情报，通知闫博士很危险而不能再呆在香港，并问她是否想到美国告诉全世界真相。闫博士表示愿意。不幸的是，闫博士和路德的这次联系，被她丈夫发现了，闫博士的生命十分危急。
