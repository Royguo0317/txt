论坛举办期间将发布世界互联网大会蓝皮书《中国互联网发展报告2020》《世界互联网发展报告2020》。蓝皮书就中国与世界互联网发展态势进行全面研究和分析，为全球互联网发展与治理提供思想借鉴和智力支撑。“
