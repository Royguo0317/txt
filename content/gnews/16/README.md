## 广东台山一核电站发生泄漏 迫在眉睫的辐射威胁
` 日本东京方舟农场` [轉載自GNews](https://gnews.org/zh-hans/1321899/)

【日本东京方舟农场】撰稿：青衣        校对：MIMI

据CNN 6月14日报道，广东台山一核电站发生泄漏，援建公司“法马通”将事件描述为“迫在眉睫的辐射威胁”，并请求美国给予技术援助。“法马通”公司于6月8日又再跟进，要求美国能源部加快审核他们的请求。据悉，台山核电站反应堆已经达到了据称修改后上限的90%，“法马通”因此对周围居民的安全表示担忧。

根据美国官员和CNN查阅的文件，美国政府在过去一周一直在评估一份有关中共国台山核电站发生泄露的报告。此前法国“法马通”公司警告称，该核电站存在“迫在眉睫的辐射威胁”。然而台山核电合营有限公司却发布新闻稿称，该核电站的各项运行指标符合要求。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/14081953/89312F8C-1703-47AC-A163-8D9FC73DBBB2.jpeg)

“法马通”公司是法国电力公司的子公司，帮助建造和运营台山核电站。上个月底，“法马通”联系美国能源部，称这座核电站存在一个潜在的问题。随后在6月3日，“法马通”向美国能源部提交了运营安全援助请求，正式请求分享美国的技术援助，以便他们能够解决紧急的安全问题。“法马通”向美国官员发出警告，称核反应堆正在泄漏裂变气体。6月8日，“法马通”公司又再次跟进，要求美国能源部加快审核他们的请求。

在给能源部的信件中，“法马通”公司的专家称，这种情况对场地和公众构成了迫在眉睫的辐射威胁。该公司紧急请求允许转移技术数据和可能需要的援助，以使核电站恢复正常运营。

信件还称，“为了确保场外剂量限制（off-site dose limits）在可接受的范围内，不会对周围人群造成不必要的伤害，台山核电合营有限公司被要求遵守监管限制。如果超过限制，就会关闭反应堆”。这一限制与法国规定的一致，但是“由于故障数量的增加”，中共国国家核安全局后来将这一限制提高到最初的两倍多，这便增加了公众和现场工作人员的场外风险。

信件补充说，截至5月30日，台山核电站反应堆已经达到了据称修改后上限的90%，核电站运营者可能正在紧急请求国家核安全局进一步提高关闭反应堆的上限，以维持运行。“法马通”怀疑，尽管对周围居民的安全存在担忧，但可能还会再次提高限制，以保证反应堆运行。

“法马通”公司在接受CNN采访时承认该公司“正在支持解决中国广东省台山核电站的一个性能问题”。该公司称：“根据现有的数据，核电站在安全参数范围内运行。我们的团队正在与相关专家合作，评估情况，并提出解决方案，以解决任何潜在的问题。”不过“法马通”不愿直接回应写给能源部信件的内容。

美国能源部也没有直接回应关于中共国正提高限制的说法。美国国务院官员表示，在收到信后的48至72小时的时间里，美国政府和法国官员与美国能源部的技术专家保持了多次联系。他们称，法国政府和“法马通”正面临几个紧急问题。CNN已联系法国驻华盛顿使馆请求置评。

美国官员称，美国也一直与中共国政府保持联系，尽管这种联系的程度尚不清楚。美国国家安全委员会上周在监测局势时举行了多次会议，其中两次是副国务卿级别的会议，另一次是周五的助理国务卿级别的会议。

一名消息人士称，目前，美国官员并不认为此次泄漏事件处于“危机级别“，但承认泄漏事件正在增加，需要进行监控。CNN获得的文件显示，美国允许“法马通”提供技术援助或支持，以帮助解决问题，但是否需要完全关闭核电站则取决于中共国政府。

美国政府拒绝解释评估结果，但美国国家安全委员会、国务院和能源部的官员表示，如果中国公众面临任何风险，美国将被要求根据目前与核事故相关的条约，将其公之于众。

CNN联系了中共国政府、广东省政府、中共国驻华盛顿使馆以及合作运营核电站的国有公司，但由于此时正值端午节假期，因此全都无人回应。

不过，台山核电合营有限公司13日晚发布新闻稿称，台山核电站1号机组已进入第二个燃料循环，目前处于满功率运行状态。台山核电站2号机组已按计划完成大修，并于2021年6月10日并网成功。本次大修是台山核电站2号机组投入商运后的首次大修，大修安全、质量、工期等指标均完成了既定目标。自投入商运以来，台山核电站严格按照运行执照文件、技术程序控制机组运行，两台机组各项运行指标均满足核安全法规和电厂技术规范要求。目前，连续监测环境数据显示，台山核电站及周边环境指标正常。

此外，根据国家核安全局公布的中国核电厂总体安全状况，台山核电站的两个机组2021年第一季度所有性能均达到预期水平。

不过，由于中共国政府对于此类事件向来没有公开透明的态度，因此官方公布的数据并不可信。究竟台山核电站是否存在核泄漏，还需继续关注外媒对此的追踪报道以及美国方面的最终结论。
