一、中共判处两名新疆官员投敌叛变死刑

新闻一

新华社4月6日报道[1]：新疆维吾尔自治区高级人民法院首次公开披露新疆在反恐、去极端化斗争中，查处长期隐藏在重点职位上的“两面人”希尔扎提·巴吾东的犯罪事实。希尔扎提·巴吾东曾历任新疆维吾尔自治区司法厅厅长、新疆维吾尔自治区党委政法委副书记，并官至新疆维吾尔自治区政协副秘书长，希尔扎提·巴吾东以党员领导干部身份为掩护，长期策划实施分裂国家活动，投靠“东伊运”组织，出卖国家和人民利益，并向境外人员非法提供情报，被判处死刑。

新闻二

据新华社电 （记者潘莹、路一凡）以沙塔尔·沙吾提为首的分裂国家犯罪集团，将民族分裂、暴力恐怖、宗教极端等内容编入少数民族语言教材，在新疆印发近2500余万册，面向232万名维吾尔族在校学生及数万名教育工作者，使用时间长达13年之久，造成极其严重的危害。法院对被告人沙塔尔·沙吾提以分裂国家罪、受贿罪，数罪并罚，决定执行死刑，缓期二年执行，剥夺政治权利终身，并处没收个人全部财产。

二、中共战狼外交再批美日两国

新闻一

中国日报4月6日发文称[2]：日前，布林肯向美国国会提交报告，继续暂停香港特殊关税地位。在过去一个月里，布林肯四次发推文妄议香港问题。他对香港问题的态度与臭名昭著的前任国务卿蓬佩奥几乎一样。在中美关系问题上，美国新政府正在重复上一届政府的错误，国务卿布林肯的不当言行就是一大证明。美国新政府应当认清形势，改弦更张。

新闻二

中共战狼外交发言人赵立坚4月6日强硬表示[3]：所谓新疆“种族灭绝”是荒谬之极的世纪谎言。美方越是“孜孜不倦”地炮制涉华人权谎言，伪装成人权“卫道士”。美国的人权遮羞布早已遮不住其作过的恶，历史和现实都已记录下它侵犯人权的条条罪状，例如殖民主义、种族主义、输出动乱、干涉主义和双重标准等。
