## 美國第四艘航母來到西太平洋是要幹嘛
`2020-05-27 18:17 GM06` [轉載自GNews](https://gnews.org/zh-hant/214676/)

作者：文茗

近日美國第七艦隊表示，航空母艦“雷根號”（CNV 76）於21日離開日本神奈川行須賀基地，重返西太平洋，以維護該海域的自由、開放，秀出甲板上停滿F/A- 18「大黃蜂」照片。停靠期間也確實實行防疫措施，確保水手健康狀況。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/27175651/1-113-59.png)
美國第七艦隊21日發佈新聞，內容提到，轄下服役5年的航母雷根號及第五艦載機聯隊是美國航艦持續嚇阻、投射力量、決勝海上的基石，也致力維護西太平洋的和平及自由。內容也提到雷根號停靠在行須賀基地時，都有依規定限制官兵行動，以預防群聚感染。

漢尼芬（Pat Hannifin）上校表示，為證明我們的口號，官兵一次又一次證明了我們能解決面臨的所有挑戰，指出這次雷根號水兵們提前完成港口維護工作，替海軍的健康和戰備狀態做出貢獻，並繼續在海軍擔任先鋒。

第五艦載機聯隊少將威考夫（George Wikoff）表示，水手的健康及安全是戰備工作的重點，因此在基地時執行了嚴格的防疫措施，掌握水手的健康狀況。雷根號總共有5000多名人員和60多架飛機，能夠維持全天候的海上行動。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/27175724/2-98-36.jpg)

第七艦隊的新聞稿也附上照片，雷根號甲板上停滿了F/A-18大黃蜂式戰鬥攻擊機，來宣揚戰力。此外，雷根號還載了1000多噸重的軍械，因此在水線上位置下沉了12.7公分。

美國海軍日前宣佈，之前遭疫情衝擊而被迫在關島隔離的羅斯福號核動力航母重返大海，重啟在西太平洋的“自由巡航”行動，雖然還有1900多名水兵在關島隔離，還未返回航母恢復工作，但是美軍依然下令羅斯福號航母強行出海，目前已經進入菲律賓海進行巡航，和之前在岸上部署的第11艦載機航空飛行聯隊大批戰機匯合。

目前還有“雷根”號加快推進部署前的工作，“尼米茲”號航母也接近部署狀態。也就是說美國總共擁有11艘航母，其中四艘航母都到了太平洋，背後隱含了什麼？

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/27175824/3-94-52.jpg)

2020年5月20日，美國防部和白宮網站同時發佈 《美國對中華人民共和國的戰略方針》報告，從問題挑戰、應對策略和貫徹落實三個方面，闡述在2017年《國家安全戰略》指導下美國對華戰略方針。

自1979年中美建交以來，美國一直希望通過與中國的接觸和交流，推動中國實現經濟、政治和社會開放。但40年後的今天，中共國並未被民主、自由與開放的國際秩序所同化，反而正試圖按照自己的意願塑造國際秩序，這不僅傷害了美國的利益，而且損害了世界各國的主權及其人民的尊嚴。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/27175934/4-31-23.jpg)
美國闡述了面臨來自中共國的挑戰，報告認為中共國正在經濟、價值觀和國家安全三個方面對美國發起強烈挑戰。

經濟挑戰

美國認為中共國在經濟改革承諾和國家保護主義政策和實踐方面的不良記錄損害了美國企業和勞動者的利益，擾亂了全球市場，破壞了國際規範和污染了環境。

一是中共國經濟改革不徹底，未履行加入世貿組織時候的承諾。報告指出，2001年加入世界貿易組織後，中共同意接受該組織“開放市場化原則”並將其納入本國貿易體系和制度中，世貿組織成員也希望中共國繼續推進經濟改革和市場化步伐，但這一目標並未實現，中國經濟改革並不徹底。

二是“強制性技術轉移”。（1）要求或強制美國公司向中共國公司轉移技術；（2）通過市場許可的方式限制美國公司技術進入中國市場；（3）直接和不公平地幫助中國公司收購美國公司和資產，以獲得尖端技術；（4）中共國通過網路入侵竊取了大量美國公司敏感資訊和商業秘密。

三是在世界貿易組織享有與自身實力不對等的“身份”。美國認為中共國已經是“成熟經濟體”，但在與世貿組織等國際組織打交道時依舊被視為“發展中國家”，現已成為僅次於美國的世界第二大經濟體，國防支出和對外投資也居世界第二位。這美國和其他國家造成了極大的損害。

四是中共國“一帶一路”。美國認為中共國通過“一帶一路”開展不正當競爭，輸出工業標準、強化本國企業在國際市場的地位，擴大中共國的經濟和政治全球影響力等。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/27180052/5-27-15.png)
價值觀挑戰

美國認為其所提倡的 “普世價值”正日益受到中國的挑戰。

一是中共國國家治理體系。近年來，中共國特色的國家治理體系的優越性開始體現，甚至在很多方面西方發達國家運作更好，特別是新冠疫情爆發以來；

二是中共國尋求在“綜合國力和國際影響力方面成為全球領導者”。中共國通過發展和完善“中國特色社會主義制度”，推動建立“人類命運共同體”，干涉他國內政以及運用媒體宣傳等手段，擴大中共國的國際影響力，對西方“普世價值”造成巨大挑戰。

安全挑戰

報告指出，近年來中共國軍事實力的迅猛發展及其軍事動作嚴重威脅到美國及其盟友和夥伴國的利益，並對全球商業和供應鏈造成挑戰。

一是中共國在黃海、東海、南海、臺灣海峽以及中印邊境地區的軍事和准軍事行動，違背了其和平解決爭端的承諾。

二是中共國的軍民融合戰略。美國認為中共國的軍民融合戰略使解放軍有能力開發和獲取先進技術，由於該戰略的不透明性，美國及其盟友和夥伴國在不知情的情況下使其軍民兩用技術支援解放軍的研究與開發項目。

三是中共國試圖主導全球資訊與通信技術產業。報告認為中國政府通過《國家網路安全法》、與華為和中興等企業合作等方式竊取他國資料。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/27180235/6-21-12.jpg)
由此可見，美國已經徹底認清了共產黨本質，《美國對中華人民共和國的戰略方針》完全就可以看作是美國對中共的宣戰書，這樣也就不難理解四艘航母齊聚太平洋的原因。

共產黨窮途末路之際，唯獨只剩武力來轉移內部矛盾為自己“續命”。但當美國成為CCP病毒最大受害者的那一天開始，美國已經別無選擇，只剩從經濟、政治、軍事等全方位的消滅這個他一手養大的“白眼狼”。
