作者：香草山写作组 Wilson 校对/发稿：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/29231938/logo-3-1-1.jpg)



据大纪元《全球疫情实时大数据》，截至2020年12月27日中午，加拿大感染中共病毒确诊人数达到541,616 人，死亡14,800，死亡率2.73%。
![]()![](https://gnews.org/wp-content/uploads/2020/12/22974225-CP-toronto-600x400-1.jpg)图片来源：加通社
安省过去两天确诊病例新增4301例，死亡81人。单日新增确诊病例已经连续12天超过2000例。

安省卫生官员称，安省已经确诊两起首先在英国发现的强变异中共病毒病例。两人是一对夫妇，近期没有已知的旅行记录，也未与高风险的人有过接触。

安大略省副首席医疗官芭芭拉·亚菲在一份声明中说：“这进一步证明安大略省民众有必要尽可能呆在家中，继续遵守所有公共卫生警告，包括全省从今天开始采取封闭措施。”安大略省26日宣布进入封闭状态。

短评：安省一对夫妇在不明原因的情况下感染强变异病毒，令人不安。

印度是人口稠密、医疗条件相对较差的国家，印度的中共病毒感染死亡率为1%，大大低于加拿大的2.73%，是因为印度在发现病毒时就使用羟氯喹为主的鸡尾酒疗法，让病毒感染者在家服用治疗，他们就是通过这种办法控制住了疫情。
