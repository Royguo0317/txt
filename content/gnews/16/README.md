## 蓬佩奧：中共全世界散佈虛假信息製造混亂，使西方處於危險之中
`2020-06-01 09:27 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/219523/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/01085126/0b793012-97e6-416b-bd9d-454a90569c6d.jpg)
據美國政治新聞網站Politico報導，蓬佩奧於美國東部時間5月31日表示：“如今的中國共產黨與10年前不再相同。”

**中共使美國人處於危險之中**

美國務卿邁克·蓬佩奧（Mike Pompeo）週日表示，從香港到美國，中共在散佈虛假信息和在全世界製造混亂方面已經變得“更加激進”。

蓬佩奧在福克斯新聞的《週日早晨談未來》（Sunday Morning Futures）節目中向主持人瑪麗亞·巴蒂羅莫（Maria Bartiromo）表示：“如今的中國共產黨與10年前不再相同。” “它已經自視為一個意圖破壞西方理想，西方民主和西方價值的中國共產黨。它使美國人處於危險之中。”

蓬佩奧通過多個話題來說明中共構成的威脅，包括對知識財產（譯者註）盜竊的指控，中共政府在中共病毒爆發期間“轉移注意力”的方式以及當前加緊控製香港的措施升級。

**中共違背承諾，美取消香港特殊優惠**

蓬佩奧上周宣布特朗普政府不再將香港視為中共國的自治特區之後，發表了上述評論。唐納德·特朗普總統週五宣布，美國可能對香港實施貿易限制，以懲罰中共對香港施加越來越大的控制。

長期以來，美國一直對香港給予特殊優惠。這使得香港的商業蓬勃繁榮，給予其獨立運作於美國與中共其他地區的關係。

蓬佩奧週日還表示，特朗普政府正計劃審查香港的地位，並撤銷這些規定，並將開始調查在美國的中國學生是否可能與中國軍方有聯繫等行動。

蓬佩奧說：“中國共產黨現在已經違背了諾言，美國將以此為實際問題作出回應。” “我們將研究是否應該準備製裁那些破壞香港內部自由的個人。”

1月初關稅戰後，美中兩國進入第一階段貿易協議時，兩國之間的緊張局勢似乎有所緩和。然而，隨著中共病毒在全美蔓延，這種攻擊慢慢演變成了激烈的口頭攻擊。

譯者註：

“知識財產”（Intellectual Property原文）的範疇不同於且遠遠大於“知識產權”（Intellectual Property Rights），知識產權是對知識財產的權屬上的承認和註冊，受法律保護（見[WWW.WTO.ORG](https://www.wto.org/)），通常不易“盜竊”。容易偷盜並搶注權屬的恰恰是沒有法律授權還在保密的那部分知識財產，它們往往是更加先進和富有價值的。把Intellectual Property 翻譯成“知識產權”算是個約定俗成的謬誤，但這個謬誤在字面意義上大大縮小了受害範圍。
