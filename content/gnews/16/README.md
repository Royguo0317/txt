## 【信仰之声】了解信仰，走出误区
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/924793/)

作者：纽约香草山福音部 德州牛仔 审核：程万里 校对/发稿：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/29231938/logo-3-1-1.jpg)



很多人以为我们可以走进信仰，成为一名基督徒，其实这是我们认识的第一个误区，也是我曾经的误区。

![]()![](https://gnews.org/wp-content/uploads/2021/02/图片-1-17.png)

基督教信仰从来不是我们主动去寻找上帝，然后结局很美好，心诚则灵，我找到了。不是的，人类寻找上帝的结果，就是我们今天所看到的宗教派别林立，你抱着大象腿说大象是一个柱子，他抱着大象耳朵说大象是一面扇子。

上帝知道我们不但不会，而且也没有办法找到他，所以他差派了他的独生爱子——耶穌基督亲自来到人间，这就是我们常说的道成肉身，英文是incarnation，是他来找我们，用我们能看得懂的方式、听得懂的语言，来告诉我们他是谁。

我们人生在世要做什麼呢？我们要来感谢我们的上帝，他愿意屈就他自己，亲自到人间做人。

腓立比书2:5-7说：“你们当以基督耶稣的心为心，他本有上帝的形象， 不以自己与上帝同等为强夺，反倒虚己， 取了奴仆的形象， 成为人的样式＂。

所以我们要感谢上帝，他愿意成为人的样式，让我们看到神在人间。这一点很重要。

约翰福音1:18说：“从来没有人看见上帝，只有在父怀里的独生子将他表明出来。”

**以弗所书**2:8里说，我们每一位基督徒得救都是“本乎恩，因着信”。

这是我们基督徒耳熟能详的一节圣经，很多人在观念中认为，没错，我得救是因为上帝的恩典，再加上我的信，我很荣幸地与上帝同工，做成了上帝救人的计划。

真的是这样吗？“本乎恩，因着信”的下一句是什麼？“你们得救是本乎恩，也因着信，这并不是出于自己，乃是上帝所赐的。”

这里的“本乎恩、因着信”，是作为一个整体，然后以“这”来被引用，“这”并不是出於自己。

那这个因着信到底是谁的信呢？不是你的信，不是我的信，而是因为上帝的信。

上帝的信是什么信？是他的信实，是他的守约，因为他是那位“守约、施慈爱”的上帝。

每一位基督徒得救，都是出于上帝的恩典，因着他的信实，这不是出于我们，而是上帝赐给我们的。

以弗所书1:4说，“就如上帝从创立世界以前，在基督里拣选了我们，使我们在他面前成为圣洁，无有瑕疵。”这是上帝的约，他的信实，他的拣选，他的拯救。上帝他所赐的是什么？

哥林多前书2:9**说：** “上帝为爱他的人所预备的是眼睛未曾看见， 耳朵未曾听见， 人心也未曾想到的。”

他就是马槽里的婴孩，耶穌基督。基督徒的心里有平安，是因为我们所信的这位，因为他是那位赐平安的主，我们在他里面才有那属天的平安。

如此说来，基督徒在世上应该怎么生活？

罗马书12:2说：“不要效法这个世界，只要心意更新而变化，叫你们察验何为上帝的善良、纯全、可喜悦的旨意。”

这句话因着中文翻译的误差，完全把原来的意思给扭曲了。乍一读，有人会以为这句话是说：你们不要学这个世界的坏榜样，你们要更新你们的心思意念让自己能有变化，然后就能明白上帝的旨意了。其实原文的意思根本不是这样的。

那原文应该如何翻译呢？“不要‘被’这个世界同化，而要心意‘被’更新而变化”。主动和被动差很远，两个“被”都没有翻译出来，得出的意思就截然不同了。

这个世界对我们的影响是非常非常大的，以至于我们一不小心就在世界里堕落了。

**彼得前书**5:8给我们有很好的提醒，你们“务要谨守，警醒。 **因为** **你们的仇敌魔鬼，如同吼叫的狮子，遍地游行，寻找可吞吃的人”。**

我们的仇敌是谁？是魔鬼撒旦，而且它吼叫游走，到处找寻能被它吞吃的人。不是我们的能力大到一个程度，我们可以决定自己要不要效法这个世界，而是我们一不小心，这个世界就要反过来影响我们、同化我们。

那我们能怎么办呢？自己让自己的心意更新变化吗？不是的，是要让圣灵来掌权、住在我们心裡，让我们的心意被圣灵的大能来改变。

你看看这句经文的理解多重要，一下子就把我们从人本主义的思想带到了神本主义的思想。很多宗教是要自己立志行善，然而你会发现我们是屡战屡败，斗志强的就会继续，屡败屡战。

我们要放弃依靠自己，因为自己真的靠不住。我们要依靠的只有一位，而且这一位已经胜过了世界。

约翰福音16:33 耶穌说：＂我将这些事告诉你们，是要叫你们在我里面有平安。在世上，你们有苦难，但你们可以放心，我已经胜了世界＂。

不但耶穌已经胜了这个世界，耶穌的门徒也已经胜了这个世界。

约翰一书5:4说，“因为凡从上帝生的，就胜过世界，使我们胜了世界的，就是我们的信心”。

约翰福音3:19-21节又说，“光来到世间，世人因自己的行为是恶的，不爱光，倒爱黑暗，定他们的罪就是在此。 凡作恶的便恨光，并不来就光，恐怕他的行为受责备。 但行真理的必来就光，要显明他所行的是靠上帝而行”**。**

耶穌就是那来到世间的真光，愿我们大家都来就光，靠近光，在这个弯曲悖谬的黑暗世代，显明我们是上帝的儿女。

求主保守看顾每一位，感谢上帝！
