## 《路德社》精选新闻快讯：拜登将发表经济政策讲话，美对中关税维持不变

出品：《路德社》 编辑：《路德社》精选新闻专栏组

![]()![](https://gnews.org/wp-content/uploads/2021/03/032905.png)

据《美国之声》3月29日报道，美国白宫正在寻求与国会合作，推出总值大约三万亿美元的基础建设法案。预计美国拜登总统将于星期三在匹兹堡发表的经济政策讲话上，宣布白宫这一方案。匹兹堡是拜登于2019年宣布参选总统的地方，同时也是美国的制造业重镇，选择在匹兹堡宣布这一方案突显了美国对重建基础设施的愿景，以及为美国人民重建美好生活的计划。

美国贸易代表戴琪在接受华尔街日报专访时表示，近期不会撤销对中共国的关税，不过愿意与中共国继续进行贸易谈判。美国与中共国在2020年1月签订第一阶段美中贸易协议。根据协议，美国与中共国的贸易代表每六个月要会面一次，不过美中两国代表已经八个月尚未举行会晤。戴琪获得任命之后，与14个国家的贸易代表通话，但是并没有中共国代表。

根据美国贸易代表办公室发出的通话纪录，戴琪在与各国贸易代表都谈到中共国不公平的贸易行为，以及将合作改革世界贸易组织。
