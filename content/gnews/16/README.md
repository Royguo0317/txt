这份手写的《成为奋斗者承诺书》主要内容：

［我向我所任职的华为投资控股有限公司或其关联公司（下称公司）承诺：

我深刻理解公司所处行业充满风险和不确定性，竞争激烈而残酷，按部就班地工作只能取得基本的劳动报酬，如果不努力工作不艰苦奋斗，就不能获得更好的发展和机会，为了获得分享公司长期发展收益的机会，我愿意长期艰苦奋斗、努力工作，自愿放弃享受部分福利待遇，我也理解分享了公司创造的价值，就不再是简单的劳动者。

因此，我承诺，自愿成为奋斗者，自愿放弃在公司工作期间的带薪年假及带薪年休假工资，在公司工作期间，不申请带薪年休假，不申领带薪年休假工资，即便从公司离职，我无权也不会要求公司支付未休带薪年休假工资。

签名：曾梦 日期：2017年3月20日］

曾梦的这份成为奋斗者承诺书就是华为公司挖的一个非常低劣的坑，只是这样低劣的坑也能被法院认可，并依次驳回了曾梦要求的［华为支付加班费、年终奖、解除劳动关系赔偿金共计91万元］。

什么是奋斗者？这不是一个法律术语，也不是劳动合同的一个术语，但是这份《成为奋斗者承诺书》却莫名其妙地成为了劳动合同的一个有效附件，并在法院认可之后成为一个“合法”规避法律明文规定劳动者应该享有的带薪年假及相关福利的案例！

七十一年来在共产主义的蛊惑之下，这个国家的公民被剥夺了包括选举权、宗教信仰自由、言论自由等一切政治权利，哪怕这些权利至今依然以白纸黑字详列于其颁布的宪法及各种法律和日常的各种文宣之中，相较于已经被剥夺七十一年的政治权利，这份《成为奋斗者承诺书》又算得了什么呢？它只能算是我们的祖辈、父辈及我们自己服从、配合和妥协的孳息！
