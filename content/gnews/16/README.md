最近班农先生在War Room中不断地提及加密货币、数字货币、数字银行、疫苗护照，让人不由联想到《启示录》中启示的兽印。

![]()![](https://gnews.org/wp-content/uploads/2021/04/image-454.png)图源网络

随着纳米级芯片技术的开发，《圣经》中许多超时代的、在过去让人感觉不可思议、不可能实现的预言，今日已经显明是必定会实现的——只是时间早晚的问题。就好比现在，一个普通手机芯片的计算能力就轻松盖过美国六十年代航空中心的计算能力。《启示录》还中提到，有两位见证人，因为传福音被人杀死，尸体倒在城市的大街上，各民、各族、各方、各国中，有人观看他们的尸首三天半。在过去的人看来，一个城市的大街上发生的事情，跟其他民族和国家的人有什么关系？三天半的时间里，又会有几个人知道这事呢？但是，随着互联网和社交APP的开发，现在任何地方发生的事情，都可以在几分钟之内传遍世界各个地方。现场的图片（例如牵涉George Floyd的执法过程），可以瞬间让不同国家、民族的人看到。由此可见，《圣经》预言的超越性：因为启示《圣经》的是那位超越历史、掌握宇宙万有的神，过去、现在、未来都在他的手中。《启示录》中也提到过兽的印记，在今日也显示出，这是完全有可能出现的事情。就比如在30年前如果有人说，我们将来会无现金扫描完成买卖，人们会认为这只不过是科幻电影的情节罢了，但现在这已成为了事实。

那么什么是兽的印记（六百六十六）？《圣经》关于“兽的印记”的经文主要是《启示录》13:15-18，其它可以参考《启示录》14:9、14:11、15:2、16:2、19:20、20:4。《圣经》告诉我们，兽的印记不是虚幻的、耸人听闻的东西，而是一个有形质的东西，而且有具体安装的位置——就在人的右手或者额头上。没有安装兽的印记会受到不得作买卖、不得从事任何商业活动地限制。这就类似如果你不安装兽的印记，你就不能购买食物，无法旅行，不能进入公司大门——一切存在货币交换的手续，你都不能履行了。甚至你不带上兽印，你将无法证明自己的身份。因为那时，人的身份证明已经注入兽的印记了。准确地说，你不安装兽的印记，就已经不属于地球的公民。而兽的数目是六百六十六。

兽的印记是一种买卖的方式，也是手段，它正在悄然推进中。它开始于商品条形码，条形码是任何商品的标配，含有商品的名称、价格等信息。它看起来是由一些条线和几个黑带组成，但仔细观察这些条形码，它有三对条线比其他条线长，分别在条形码的最前面、中间和最后面——这三条线代表的数字就是666，所有条形码都出来三次代表着6的数字。为此，人们把这条形码叫666码，就是圣经里的六百六十六。为什么开发者把识别条形码的基准数字设为6呢？这是因电脑在读取条形码时，最少出错的理想数字就是6。无论理由是什么，在条形码结构上，6的数字进入三次，难道是偶然的吗？这能说与《圣经》的预言毫不相干吗？随后，条形码升级到QR码，QR码是一个二维的正方形条码，于1994年由Denso Corporation发明，其比传统的条形码更易于被机器识别，并且能够存储大量的信息——即使QR码本身有被涂改也能被读取。因为数字货币的出现，我们正在步入一个无现金社会。这也导致传统的支付方式逐渐被手机APP支付取代。因此，我们的财富不再是肉眼能见的现金，而是一串数字——数字货币时代正悄然来临。CCP国央行正在测试赤币（chibi），英国央行也在考虑发行数字货币。

而今，随着CCP病毒到来的是健康码，也称为疫苗QR码护照，其实质就是社会信用积分系统，只不过换了一个好听的名字而已。这QR码倘若与数字货币、支付系统连在一起，更是威力无穷——无论你到哪里，必须出示疫苗护照，否则寸步难行。因此，如果你新冠病毒检测为阳性，你将被困在家中寸步难移。更不要说，QR码一刷，无论你走到哪里，干了什么、说了什么，与周围的人互动，都会被完全传回数据中心，被纪录在案，交由AI分析处理——你将被全天候、无死角地监控。随之而来的，只要你的谈话触犯了关键字，你的支付通道就会自动关闭；除非你改正，做一个系统认为的没有危险的人，重新挣回你的社会信用，从而恢复通支付通道。

另外，现在已经开发出了可以植入人体的医用芯片——人体芯片。该芯片只要稍微改改，就可以用来储存人的身份信息。据报道，瑞典的试验项目将微芯片植入人体，用以作日常交易支付，如乘搭交通工具和购买日常用品等。那么，兽印有没有可能将数字货币、数字银行与疫苗QR码护照结合，刻入类似于人体芯片一样的东西，再植入人的右手或者大脑前额？很有可能。这样的技术正日趋成熟——人类一定会不断尝试。
