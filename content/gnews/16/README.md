- 其二是為其在美國開設兩個酒莊洗錢，以及提供每年幾百萬的捐助作為藍金黃的運作資本，供其與美國參眾議院大量議員勾結輸送利益及捐贈，並在其住所所在地深度藍金黃當地政府、警察及牧師，編織起屬於他的的黑暗大網。
