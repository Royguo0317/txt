在一个无名的星球上，有个名叫小王子的人画了一幅蛇吃大象的画，觉得非常恐怖，当他把作品拿给人们看时，人们往往会说：“一顶帽子有什么可怕的？”后来他把蟒蛇肚子里的大象画出来，人们却说只有未知的事物才可怕。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/02092023/Screen-Shot-2021-06-02-at-9.20.13-PM.png)
这个小王子觉得很没趣，来到森林，捡起一块石头扔了出去，刚好落在一只猴子的面前。猴子捡起这块石头，心生欢喜，玩起了抛石头的游戏。它抛了一次又一次，石头也一次次地落在地上。小王子无聊地记录着这个无聊的过程，当此之时，一束星光闪过他的眼睛，他忽然发现，猴子抛石头的次数，与正面朝上的次数，呈现出的曲线，与他的作品有些相似。于是他又做了相同次数的记录，所得到的曲线还是很相似。小王子无法回避这个现象，陷入了沉思，他想，是有谁托着猴子的手吗？
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/02092046/Screen-Shot-2021-06-02-at-9.20.36-PM.png)
实际上，猴子并不理解抛石头的行为，但是这个行为却蕴含着规律，就像它并不理解自己的身体，但是它也遵照着规律运行。这些规律都不是人为的，而是自然的。这些自然规律是如此地可靠，以至于我们在有生之年可以高枕无忧地依靠它们，所以它们值得遵守和敬重；它们又是如此地充满未知，以至于我们在有生之年无法大彻大悟地领悟它们，所以它们值得畏惧和探索。然而，它们越可靠、越未知，就越是令我们无法回避这个问题：它们来自哪里，它们还有些什么？苟且偷安者把它们当作理所当然，从而失去了对它们的敬重；自欺欺人者放弃了对它们的探索，把有限的认知当作全部，从而失去了对它们的畏惧。一个对自然规律失去敬畏之心的人，注定碌碌无为、虚度年华，就像小王子遇到的这只猴子，给它一部打字机，用尽一切时间，也无法打出一部《小王子》。因为没有敬重，就不懂得珍惜，就不会积累；没有畏惧，就不会去探索，就会止步不前。假设这个猴子侥幸地打完了一半，然后又失去了侥幸，此一刻，由于它没有积累，所以止步于此，同时，过往所作的努力顷刻间化为乌有，就像什么也没有发生过一样。所以它总是在原地踏步，永远无法打出这部文学著作。有些刁钻之人会狡辩地说，从头到尾地侥幸，就可以一气呵成地打出这部著作。这是痴人说梦，因为这无异于庄稼地里长出满汉全席。

由此可见，自然规律犹如天地，它能孕育，像大地一样承载万物；它充满未知，像天空一样召唤万物。居于其间者，上顶天，下立地，独立自主，是以为人。所以，是谁托着猴子的手？是自然规律；是谁托着小王子的手？是人的自主性。小王子带着问题出发，游历诸多星球，带着答案而归，是谁驱动着这些生命活动？是小王子的敬畏之心。

敬畏之心，是生而为人的根源，是生命力的源泉，是顶天立地的诠释，是人生旅途的明灯。心存敬畏，永远不会迷失；缺乏敬畏，则像小王子所途经的人们，纷纷陷入各自的困境而不能自拔；完全丧失敬畏之心，则像中国共产党，这个地球上有猴子以来最邪恶的存在，不知天高地厚。

中共，这个邪魔恶党，它长期施行与天斗、与地斗、与人斗的黑暗统治；它倒行逆施，长期违背自然规律；它剥夺人权，践踏宗教，长期施行愚民洗脑政策，致使人们浑浑噩噩，缺乏敬畏之心。不仅如此，它还筑起互联网高墙，将中华民族孤立于世界，它操纵舆论，培植民粹主义，绑架中华民族，妄图称霸世界。这种痴心妄想，无异于那些不依不饶的刁钻之人，妄想着用无穷多的猴子，在无穷多的打字机上，打无穷长的时间，打出一部《小王子》。这是不可能做到的，因为这依然是原地踏步的处境。悲哀的是，这正是中华民族当前的真实写照：人们终日忙碌却徒劳无益，虚度年华却无处叹息，沦为亡国奴却不自知。

庄稼地里长不出满汉全席，同样地，天地之间也不会无端地呈现出美好家园。天地为人立法，人应当为自己的家园立法，那么，按照天人合一的思想，人类的家园，当以何为天、以何为地，人居其中，又何以安身立命呢？人类的个体，由自然规律所孕育；人类的家园，则只能在人类的秩序中孕育。如果人们聚在一起，各执一词，各自为政，那么这群人只能是乌合之众；如果受制于皇帝老儿的个人独裁，则只能是个奴隶庄园；如果受制于党魁匪首的一党专政，则只能是个党国。就像此刻的中华大地，被中共侵占，人们名为居民，实为亡国奴。所以秩序必出自普罗大众，又被普遍地遵循，这就是民主与法治，以此为地，可以建国。

物以类聚，人以群分，人们之所以聚在一起建国，是因为在各自的追求中有了一份交集，这份交集既是建国的缘由，也是建国的意义。倘若失去了这份交集，则犹如小鸟虽煽动翅膀却失去了翱翔的天空；种子虽落地生根却没有肥沃的土壤；虽扬帆起航却没了彼岸。也正如此时此刻的中共党国，它的党魁匪首们所追求的假、丑、恶，与中华同胞们所追求的真、善、美，完全水火不容，不得人心到了如此地步，中共党国必将分崩离析、土崩瓦解。所以，以这份共同的追求为天，可以立国。

如果人们聚在一起，依然是自扫门前雪，不管他人瓦上霜，那么，这个人群与禽兽何异，这个国家与丛林何异，这片天地与监牢何异。试想，如果一个人的利益与一个国的利益没有交集，而他却身入其中，那么，这个人明显就是为了揩油；如果一个国的利益与国民的利益没有交集，而它却依然存在，那么，这个国一定是绑架了它的国民。所以，利益中的交集，是人们安身立命的基础。然而，此时此刻的中共党国，党魁匪首们的利益与国家和民族的利益针锋相对，它们将自己的利益凌驾于国家民族之上，是彻头彻尾的寄生虫与盗国贼。

综上所述，在人群聚集之处，凡是能称作家园的，人们的追求必有一份交集，人们的利益也必有一份交集，普遍遵循的秩序也必出自普罗大众。反观中共，它的追求与中华民族水火不容，它的利益与中华民族针锋相对，它所施行的政策是一党专政，在它的黑暗统治下，丝毫没有一个家园应有的模样。由此可见，中华大地，国已不国。

违背自然规律，摒弃敬畏之心，等同于盲人骑瞎马，夜半临深渊，不仅不能建立美好家园，还必定堕入深渊。不同的民族建立不同的家园，无一例外都是敬畏之心的外在表现。中共违背自然规律，抹杀敬畏之心，是民族大敌。消灭中共，是中华民族复国的必须，是重建家园的必须！
