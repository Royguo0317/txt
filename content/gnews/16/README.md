## 【世事解评】中共将中国足球扼杀在摇篮中

作者：香草山农场写作一组 文非

足球作为世界第一大运动，为无数人所追捧，提起中国足球，人们往往想到的是失败，无能，堕落，甚至还会加以嘲讽。虽然只是一项体育运动，但是可以管中窥豹，中国足球的问题又是谁一手造成的呢？国人每每对国足寄予厚望，一次次灾难性后果而让其停滞不前，永远找不到解决问题的渠道。

中国足球的管家–足协是中共治下官僚机构的优秀代表，其一言一行都与自己的主子有着相似之处。足协以前由体育总局分管，自喜欢足球的习大神上台后，中共发扬了“楚王好细腰”的做法，加强了对足球项目的重视，使足球项目脱离了体育总局。习大神在全世界访问时都不断强调自己喜欢足球，之后金元足球便在国内兴盛起来，各个大型企业都加入到足球的项目中来。德罗巴、阿内尔卡、奥斯卡、胡尔克、佩莱等一众巨星的加盟，同时世界最优秀的足球教练也加入其中。例如：巨大金钱刺激下诞生的广州恒大，一度曾两度夺取亚洲冠军联赛冠军，球迷们对中国足球重新给予了期望，青训也开始发展起来；高额的工资和奖金让踢球成为了光荣的职业，中国足球一改之前深陷黑哨风波的败坏形象。

这是真正的中国足球实力吗？在国家队的比赛中便可一看究竟。球员一旦回到国家队，失去了外援的支持，仿佛失去了往日的骄傲。1:5输给了泰国二队和战火中的叙利亚后，打过鸡血的中国足球仿佛变得更加不堪。

近年来的金元足球一度给中国足球带来了些许希望，但是过度依赖外援和金钱的“鸡血“模式终究不可持续。足协发现金元足球不光花费大量外汇，而且取不到实质性的提升；加上足协的一任主席在位时间有限，而每一位上台的新领导都想玩点新花样，足协的政策一改再改。足协的人事变动频繁，中共即使加以重视，也无外乎空降各种外行领导，来展示自己的水平，中国足球成为了一片不断被推倒重建的城墙。解散中国足协甚至成为了球迷心中最大的期望。

奇葩的足球政策更像是中共治国能力的缩影。足协曾出台：头球一个算俩、U23球员必须每场有3个上场、接受特种部队训练、扑克牌比大小决定亚军、猜单双定升级等奇葩政策，足球俱乐部也只能苟且生存，一度逼疯业内人士。因此，可以得出结论，足协的政策，没有最荒谬，只有更荒谬。例如：2019年12月25日，中国足协职业足球政策说明会在北京召开，足协主席陈戌源，秘书长刘奕以及俱乐部投资人等出席会议，确定了新赛季中超外援及球员限薪等多项新规政策。根据微博媒体“西北望看台”透露，会议上有一位足协领导提到了广州恒大俱乐部，他表示，恒大拿再多冠军也没用，就算9连冠10连冠，中国足球也上不去。

中国足协之后开始了一系列大刀阔斧的改革，跟中共整天叫喊的改革如出一辙，限制薪水、规划外援、足球俱乐部名字中性化等强制性的突袭政策打得很多俱乐部措手不及。最后得到的结果是什么呢？2020年中超联赛的冠军江苏苏宁队竟然在联赛开始前宣布彻底解散了，当然这只是其中的代表之一，但已经足够让世界足坛震惊。荒唐、滑稽，充满魔化现实主义的场景在中国足球上一次次重演。

政策的不可持续性和急于求成是中国足球失败的根源性问题之一，上梁不正下梁歪，中共的体制正在彻底扼杀着中国足球，官员的任命制，体制僵化造成的恶性循环正在腐蚀者包括中国足球的生活中各个方面。无法完成完全的市场化，中共害怕自己的足协大权旁落，管天管地管生殖器，但是管理的一塌糊涂。官员们都是为了自己的眼前利益而活。中共体制正如同一个老朽不堪的机器，僵化而不可前进，实力完全经不起真正的考验。中国足球的前途仍然迷茫，中共造的孽仍在戕害着这片土地的人民，唯有彻底消灭中共恶的体制，建立新的可持续的机制，是中国足球唯一的出路。
