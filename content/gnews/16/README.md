意大利首次为孕妇提供抗冠状病毒疫苗。于5月11日下午在撒丁岛的*SantissimaTrinità*医院开始。主任塞尔吉奥•马拉拉基尼（*Sergio Marracini*）表示，现在这是新的一步：该程序将在未来几天全国范围内进行，但是，当然我们现在正在着手遵守所有必要的安全标准。

目前，从第28周开始，优先考虑那些具有发展严重冠状病毒疾病危险因素的女性。在这个困难的时期里，医生团队为新冠病毒阳性妇女提供了帮助，该小组免费为孕妇提供疫苗接种。

**全球各国疫苗接种波折不断，再加上不断出现的病毒变体，疫苗是否真能有效值得严肃对待。刚有新闻报道称巴西出现孕妇接种后死亡，当局暂停了对孕妇接种阿斯利康疫苗。然而，接种疫苗所出现的副作用，似乎并没有丝毫动摇政府接种运动的步伐，这是很可悲的。每个人都成了被实验的对方，谁也不去考虑，自己可能就是被疫苗击毙的那个。**
