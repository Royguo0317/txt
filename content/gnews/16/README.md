最近一个概念在爆料革命中被提及很频繁，就是中共的“军民融合”。

中共从20世纪初成立开始实际上就开始搞军民融合，到抗日战争时期达到高潮。在“不拿群众一针一线”等宣传之下搞出农村包围城市，直至在抗日战争中最后得利。直到今天，CCP领导人还经常表演到群众家里去揭锅盖、掀被子这样虚假的套路。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/20071205/Unknown-2.jpg)
不得不说，CCP在发动群众上很有一套，这也是他们当初上位的关键。这些例子在CCP的各类抗战历史剧里都有极尽夸张的展示。但无论CCP哪个时期的军民融合，都是以更多军民的各种牺牲为代价的。真正可悲的是，几十年以来的军民融合对于这片土地上的人来说，并没有让他们获取应该得到的基本人权，反而一再沦为给CCP续命的工具和现代化奴隶。

几十年过去了，如果说抗日时期的军民融合还有一点抵抗外敌入侵的合理性，今天CCP搞的军民融合就实实在在是绑架人民为它的邪恶体制续命。非常可悲的是，十四亿人里面，绝大部分居然对此不自知甚至十分自愿。这就是CCP集大成的洗脑术的成果。

严格说来，洗脑术在国内外几千年来一直存在。在信息封锁没有通信手段的时代，洗脑是不能避免的。而人类的发展就在于对于人权的争取、对公平的追求、对外部世界的好奇。所以现在的整个世界，除了几个独裁国家，都没有网络防火墙。在全世界几乎所有国家都不存在洗脑的今天，CCP在倒行逆施，把精力都放在如何加固网络封锁上面。为的也是造成如古代一样的闭塞环境，方便洗脑愚民。而几千年以来中国人的奴性也造就了各种洗脑术愚民术的进化。

回到几十年前新中国成立之后，CCP正式掌权，第一板斧就是搞了几十场运动。再加上后来人为的三年灾难和十年文革浩劫。将热血的中华精神冷却下来，把刚站起来的中华膝盖打跪下，把刚挺直的民族脊梁打弯。这是CCP集权统治的暴力时期。那时候的中国人还有血性，不断有抗争，但随着社会的发展这些反抗力量越来越弱。终于到1989年的天安门，一息尚存的的民族抗争精神又一次点燃。而之后的事情我们都知道了。这次不成功的抗争运动造成的是CCP统治阶层里面的邪恶力量前所未有地达成一致：极力加强再加强推行各种愚民洗脑术。党内的温和派、改良派开始彻底被边缘化。 今天郭文贵先生爆出的掌控中国的几大家族的力量壮大都是从那个时期开始的。同时，由那时候开始，中国人感觉一天不如一天自由，一天不如一天开心。

20世纪90年代末，网络的兴起让不少思维相对独立的中国人感到高兴。相比于今天，那时候有的论坛还能公开讨论一些敏感话题。那时候的社会氛围也比后来好得多，有的人说几句政府国家哪里不好，周围人大多是笑笑而已。而今天的氛围就两极化，如果你在一群人中说中共哪里不好，面对的要么是非常佛系的冷漠，要么是神情亢奋的指责，对方活像几十年前的红卫兵。现在的CCP国内环境其实已经是历史最糟糕时期，精神上的堕落和思维上的奴性超过任何战争或困难时期。到处是年轻人的迷茫和无奈，偶尔有被逼愤怒的，出气口也不是对准CCP而是对着无辜的亲人、恋人、朋友。今天有的学生孩子们用着苹果手机，但言谈举止跟几十年前的红小兵相比，有过之而无不及。

这一切，都要归咎于CCP不仅集几千年来的愚民洗脑术之大成，还不断思考利用各种现代化的手段实施进一步管控。电子产业和网络技术的发展，让人类的生活工作前所未有地高效，但落到CCP的手里首先就被用在管控人民上。外来的所有文化都经过CCP在各处的管控审查口子进行过滤和改造，以达到统治人民的目的。CCP几十年来，把偷窃和这种洗脑术的发展认为是自己的能力，甚至开始在全球特别是独裁国家推广监控技术和这种洗脑管控的所谓软实力。

今天的中共国内，已经不是简单的宣传国外不好国内就是好这种洗脑手段。各种大v在各平台上各种伪装、各种利用信息不对称潜移默化地对受众进行洗脑，手段手法非常巧妙，绝大部分人没有足够的分辨能力，只能被这些CCP打手牵着鼻子走。再加上诸如抖音为代表的各种娱乐软件，都在潜移默化地消解CCP统治对人民造成的各种负面情绪和娱乐化的精神洗脑。尤其是抖音国际版在这方面对美国悄无声息地进行着精神侵略。

比如目前国内对于美国大选的看法，几乎所有人都认为拜登会当选，而说到川普就一脸的鄙视。如果你试图跟他们说一些拜登丑闻的真相、选举票作弊的实锤等，对方就连你一起鄙视。在翻墙不是特别困难的今天，居然几乎整个民族如此愚昧不堪，着实让人震惊。

当洗脑到了这样一个层次之后，军民融合就特别自然了。紧急时刻只需要稍微鼓动一下，各种CCP的五毛、自干五炮灰就会蜂拥而上。而国内战友面对的最大问题是，敌人到时候可能不是CCP或者军队这个层面，而就是身边那些愚昧的平民百姓甚至亲朋好友。如郭文贵先生所言，灭掉CCP几个高层甚至整个CCP很容易，但以中国洗脑造成的军民融合程度，到时候群立的军阀地方势力只要举着红旗并且拥有一定的经济、粮食、武器装备等，立刻可以就地称王，如果没有外力干预，会持续相当长的时间甚至一直可持续，因为CCP几十年来植根在几乎所有中国人身体里的那种红色基因，即使到了山穷水尽仍然会存在。那时候只需要一个举着红旗的哪怕明显是土匪的人，他们也会趋之若鹜，就像他们这几十年以来一直在做的那样。

这种基因非常魔性，成因非常复杂。利用的就是几千年以来中国人骨子里形成的那种劣根性，核心之处就是文贵先生讲的贪嗔痴慢疑。具体表现比如各扫门前雪，没有群体意识，抱着别人都遭殃自己会独善其身的侥幸，没有规则意识，别人遵守规则有契约精神认为是傻，自己渴望得到各种特权。……

而跟随爆料革命我们才具体地知道，CCP的这种军民融合的背后是3F美国以达到制霸全球的目的。

首先在美华人都或多或少接触过为CCP工作的人来到你身边。比如搞政庇的、假律师、假牧师、跟你交朋友的……。这些手法不仅用于华人也用于他们要搞定的各种国际人士。在海外的华人最近几年越来越感受到墙外有墙，就是CCP军民融合在海外力量布局几十年的结果。

他们看到世界力量和沼泽地各种无动于衷或者背后各种甜言蜜语欺骗沼泽地主人，就越发肆无忌惮的对人类灯塔进行疯狂攻击。从千人计划各种窃取高级技术，到渗透白宫直至把病毒植入川普团队所有人……，CCP的军民融合把你能接触的所有人类活动都植入它的病毒，让你无处逃避。

因为可悲的是这个世界上真正秉持信仰、坚守正道主义的人并不占多数，特别在中共国内更是凤毛麟角。即使在我们感性认为的正义灯塔——美国——国内，这次拜登能几乎登顶也让我们看到除了支持川普那7000万以外的真正境况。正义力量必须集结并且要史无前例地团结，不仅为了我们在这个地球走一遭不留遗憾甚至在人类历史上留下战斗的荣耀，更为了我们的子孙后代永不为奴。

接下来中共国很可能将面临一场热战或是一场长期消耗战，中共几十年以来煞费苦心的愚民洗脑术造就的军民融合将会把这个国家带入灾难。而不彻底灭掉这种CCP这个毒瘤，它随时将摧毁这个世界。
