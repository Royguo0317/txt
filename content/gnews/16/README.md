## 【战友寄语】中共极权下的香港不再有集会游行自由

搜集：文燕 / 撰文：岁月如歌 / 封面合成：文粤
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/11083857/5116-2.jpg)
据众新闻5月10日讯：《加拿大驻港总领事：六四记忆重要，盼港人恢复可和平集会》：

「六四烛光晚将连续第二年被拒绝在维园举行，首批承认未经批准参与集会者更被判4到10个月不等。加拿大驻香港及澳门总领事南杰瑞在裁决前接受众新闻专访时说的，加拿大当局深信保留64记忆重要，并希望港人可以尽早恢复参与和平集会和示威」。

这是一个美好的景愿，想要实现这个愿望不容易并且是难上加难。现在的香港和回归后10年不一样了，那时候中共虽然完全掌控了香港，但还要做表面工作，要营造一个“港人治港高度自治”的场面，兑现邓小平的“马照跑舞照跳”的承诺，游行示威只要不是推翻中共政权，无伤大雅的都会通过申请。习近平上台后，对香港的政策并没有继续走邓小平路线，而是出尔反尔，撕毁邓小平曾许下的承诺。邓小平承诺香港回归实行一国两制至少50年不变，如果不够可以再加50年，100年，习近平才22年就撕毁了承诺。这是中共早已决定的计划还是习近平个人撕毁承诺的呢？

了解中共政治内部帮派斗争的都知道，习近平上台后邓小平的后代在各个“岗位”就陆陆续续出事了，这个是偶然吗？邓小平的残疾儿子邓朴方为何在2020年5月中共两会前发表《邓朴方给两会代表的公开信》？邓小平的女婿吴小晖以及其掌控的安邦集团也在习近平上台后被控集资诈骗罪、职务侵占罪判处18年有期徒刑，剥夺政治权利四年，并处没收财产人民币105亿元，追缴期违法所得752.4851亿元的密码，安邦也在悄悄易主，这是为什么呢？

大家都知道邓小平是中共国改革开放的总设计师，但是习近平上台后就出现了习仲勋和改革开放的传说，指习仲勋才是改革开放的主要人物，邓小平窃取了他的功劳。为什么不是在习上台前的二三十年出现？习近平对邓小平家族的绞杀是“子代父报仇”。香港回归是邓小平一手主导，习近平撕毁邓小平在香港的承诺，以完善政改等方式变相结束一国两制港人治港虽然是中共邪党的特点也不排除习“公报私仇”。在中共这个绞肉机里无时不刻不无间道。

不管是从这角度还是从中共本身的毫无诚信以骗治国的特点看，邓小平的一国两制承诺已不存在，“马照跑舞照跳”已成空话，取而代之的是红色政权“万万碎”。游行示威自2019年6月反送中爆发后，所有的集会、游行都得不到警方签发“不反对”通知书，取而代之的是反对通知书。这不仅仅是因为中共害怕香港市民游行示威给他们造成国际压力，更是中共统治的本质，在中共独裁统治下的大陆根本不可能存在游行示威，只要出现小规模的群众请愿，中共的第一反应就是大量公安武警“人海战术”强制镇压直到消灭为止。香港现在也是红色政权之下因此游行示威也不可能再像以前一样取得警方同意，警务处也会受到中共的压力不许批准香港人民上街游行。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/11083944/5117-3.jpg)
64是中共30年来最敏感的话题，不管在国内任何一个地方任何一个时间段都不可以提，中共要悄悄地把这段历史抹掉。香港在64发生之后一司徒华倡导的支联会除了出钱出力营救被中共追杀的民运人士外，往后的30年基本每一年都在维园举行悼念活动，这让中共很不爽，一直都要灭掉支联会为代表的香港民主派，现在得逞了，香港民主派被中共几乎收拾干净了。加拿大总领事说「2000年的中国最开放」没错，那时回归10年内，还得做样子。

今年64维园纪念香港人警方不可能发“不反对”通知书，这是意料中的事，但绝对阻止不了香港人几十年积垒下来的对民主自由的渴望和尊重，香港人会以各种形式表达他们的心意，支联会将设街站纪念64事件32周年。
