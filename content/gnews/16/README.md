- 10.1. 山西太原一景區發生火災13人遇難國務院安委會掛牌督辦
10月1日13時許，山西省太原市迎澤區小山岩村臺駘山景區冰燈雪雕館發生火災，當地消防派出30輛消防車、181名指戰員趕赴現場處置。目前，搜救工作已經結束。現場共搜救出28人，其中13人遇難，事故原因正在調查中。接報後，應急管理部立即調度了解救援情況，要求全力搜救被困人員、搶救傷員，科學施救，確保救援安全。派出由部黨委委員、消防救援局局長瓊色帶隊的工作組趕赴現場指導處置。國務院安全生產委員會決定對山西太原遊樂園火災事故查處實行掛牌督辦。
為有效防範化解安全風險，堅決遏制各類事故發生，國務院安委辦、應急管理部立即組織14個工作組，分別對重點地區、薄弱地區分批次開展明查暗訪，特別是對旅遊景區的高風險旅遊項目、大型遊樂設施、大型演出場所等進行全面安全檢查，督促相關地區和部門落細落實各項安全風險防範責任，確保節日期間安全穩定。
