**2020年7月19日**
**班農戰情室284 三峽將引更多災難**

班農說傑克是節目中最早提到疫情會對第三世界的糧食供給造成影響的，現在看到在中國正在發生的人道災難，三峽流域的歷史最大洪水。有人預測中國可能會有1億人會死於洪水或者之後的饑荒。
傑克說：從周一以來三峽大壩一下的水位從最高位有所回退，但只是一點點。大壩上遊的雨水還在增加，而且不知道三峽大壩會如何泄洪。已經預計有四百萬英畝的土地處於危險之中，這都是農耕土地，所以中國的糧食自給會有很大麻煩。還有過去十個月肆虐中國的非洲豬瘟，因為洪水可能要再次感染這些農畜區。大陸的多個媒體報道要求長江流域的居民返回幫助抗洪保堤，但是鄱陽湖已經多處決堤，中國可能會有即將到來的巨大人道災難。中共的媒體已經開始對三峽大壩不那麼樂觀，它們說大壩已經盡其所能了。之前拉希姆也說過，過去十年中共的媒體不斷更改對大壩的描述，從千年一遇洪水到百年一遇，而現在的洪水他們說超過了百年一遇的標準。很多制藥公司和供應鏈，80-90%都在長江流域大壩以下。我想說，如果可怕的事發生了，洪水對中國經濟的長期影響可能會超過疫情。
班農說，我們一直在關註洪水，而且在關註醫療產品的供應鏈。像傑克提到的，普通藥物的原料大概80-90%由這個地區供應，我們過去供應鏈外遷的政策造成了這種局面。同時我們聽到，中國這些老百姓多麼令人尊重，他們是真正的愛國者。抗洪的解放軍和器械不夠，中共就召回老百姓來去前線保護堤壩。這是中共的管理不善造成的問題，他們封鎖消息，在三峽偷偷泄洪，同時媒體又在為三峽決堤做鋪墊。中國之前的豬瘟，病毒疫情，加上現在的自然災害，習主席岌岌可危，就是一個紙老虎。
傑克說，中國統計局剛剛報告食品的年價格增長是11%，過去2周增長2%。隨著洪水繼續，供給緊張，價格可能持續上漲。同時我們密切關註那裏的夏收。而且有些地區已經報告了蝗災，被淹地區的泥土更適合蝗蟲產卵，兩個月後便會成蟲，那將是有一個災難。
