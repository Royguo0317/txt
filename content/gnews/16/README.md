福克斯6月18日新闻报道，一份报告称白宫暂停了对乌克兰的军事援助。Politico援引知情人士的话说，该计划价值高达1亿美元，可能包括反坦克导弹和防空系统等各种武器。

今年早些时候，俄罗斯军队在乌克兰边境集结，引发了人们的担忧。在俄罗斯官员从该地区撤出部分军队后，美国家安全委员会(National Security Council)成员冻结了这批计划中的货物。根据当时的多份报道，在撤军之前，估计有超过10万俄罗斯军队聚集在边境附近。

尽管援助计划被冻结，但有官员透露，如果有必要，武器可以迅速运往乌克兰。五角大楼今年批准了两项价值2.75亿美元的援助计划。

白宫新闻秘书珍·普萨基(Jen Psaki)发表了以下声明：“有关我们推迟向乌克兰提供安全援助的说法是无稽之谈。就在上周——在美俄峰会前夕——我们提供了1.5亿美元的一揽子安全援助，其中包括武器援助。我们现在已经通过乌克兰安全援助计划提供了国会拨款的全部款项……我们还准备了应急资金，以防俄罗斯进一步入侵乌克兰。正如拜登总统直接告诉普京总统的那样，我们将坚定不移地支持乌克兰的主权和领土完整。”

近年来，在乌克兰军方和俄罗斯支持的分裂分子在边境地区发生冲突时，白宫一直向乌克兰提供资源和支持。俄罗斯长期以来一直否认在乌克兰境内有活跃的军事存在。

参议院外交关系委员会(Senate Foreign Relations Committee)资深共和党成员、爱达荷州参议员詹姆斯·里施(James Risch)批评了暂停这项1亿美元援助计划的决定。里施表示：“当普京开始在乌克兰边境集结军队时，已经是时候为乌克兰提供更多的防御了。”“他的撤军不应该导致美国减少对乌克兰的支持。”

在美国总统拜登与俄罗斯总统普京举行峰会之前，俄罗斯对乌克兰的行动是争论的一个关键来源。本星期早些时候，拜登说，乌克兰还没有达到加入北约的标准。他呼吁乌克兰政府打击内部腐败。与此同时，拜登承诺支持乌克兰抵抗俄罗斯在该地区的侵略。

拜登说：“他们有更多的事情要做，但是这并不能证明俄罗斯在顿巴斯、海上或乌克兰任何地方采取侵略性行动是正当的。我们将使乌克兰能够维持他们的实际安全。”
