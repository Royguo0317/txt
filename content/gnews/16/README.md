**西班牙2021年3月5日电/西喜社——**民主党总统拜登近日嘲讽德克萨斯州和密西西比州取消强制佩戴口罩的举动，认为这是野蛮人的思维方式。

拜登说：“我认为这是一个很大的失误，我希望每一个人都认识到口罩的重大意义。我们正在用疫苗来彻底改变颠覆整个疫情。我们正努力在5月底之前准备好疫苗足够给每一个美国成年人使用”。

拜登还说：“我最不希望的就是每当一切看起来已经没事了的时候人们就摘下口罩。别这样，口罩仍然很重要。遵从科学是非常非常非常重要的事（拜登重复了非常3次），经常用热水洗手，戴口罩，和别人保持距离。我知道你们都明白这些道理。我真心希望那些官员也同样明白。”

然而拜登并没有告诉记者，德克萨斯州共和党州长格雷格·阿博特（Greg Abbott）从来没有指示民众不要使用口罩。格雷格·阿博特说，“取消强制戴口罩法案不代表我们停止在乎个人，家属和邻里的健康。”

格雷格·阿博特还说，“从上至下的强制口罩法案已经没有必要了，不过为了控制疫情，德克萨斯人需要持续遵从医学建议。我知道有些地方管员担心完全开放经济会导致疫情日益严重。在我的行政命令里已经包含了对抗这种担忧的实际措施，如果因新冠住院治疗的人连续7天占据15%以上的病床数，那个区域的地方法官可以自行使用进一步的新冠控制政策。但是在任何情况下，没有人可以因不遵守新冠政策而被逮捕，因不带口罩而被罚款。”

格雷格·阿博特最后表示，“这次的公开会不表示我们完全放弃一年以来的安全标准。相反我们重视我们自己和他人的安全。个人安全需要个人自己来负责而不是由政府强制。我们有能力同时拥有安全和自由。”

密西西比州共和党州长泰特·里夫斯（Tate Reeves）说“强制口罩法案被取消的根本原因是现在的状况无法达到当初设定的需要实行强制口罩的标准”。
