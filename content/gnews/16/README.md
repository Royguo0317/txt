## 【信仰之声】西边的云
` 纽约香草山MOS03` [轉載自GNews](https://gnews.org/zh-hans/1085830/)

作者：纽约香草山佛教部    369文驹

![]()![](https://gnews.org/wp-content/uploads/2021/04/logo-5.jpg)



2021年3月15日，是一个值得纪念的日子。当然，笔者所说不是墙国的3.15晚会——那个所谓“打假”其实就是“假打”，笔者所说是另有其事。

3月的春天，阳光炽热，万物生长，生机勃勃。鸟儿经常叽叽喳喳光临寒舍，好不热闹。可是那天中午，笔者却发现有只麻雀粘在了“蚊蝇贴”上，不幸死亡。笔者怀揣伤心和感概，试图将鸟儿取下。当接触到鸟儿时，发现鸟儿的身体居然被太阳烤得有些烫手！这也解释了鸟儿死亡的原因吧。顿感活泼机灵、极富生命活力的小鸟在失去自由后，生命是多么的脆弱。随后笔者小心翼翼将小鸟埋葬在了它们经常停留玩耍的小树下的花台里。

时间来到了当日斜阳西下的黄昏。无意间，笔者瞥见西方落日的地方不知为何有几朵白云在缓缓移动，而天空其他位置是没有云朵的，心里不免犯嘀咕，心想还有这种情况出现？随着云朵的持续移动，神奇的一幕出现了：那几朵云组成了小鸟的形状！笔者赶紧用手机将这一幕拍摄下来，见下图：
![]()![](https://gnews.org/wp-content/uploads/2021/04/图片44.png)
由于云朵在不停移动，前后不过几分钟云朵就分离，飘远，渐渐模糊了。但是构成小鸟的这一幕，深深震撼了我。能形成这样的概率得是多小啊！简单梳理如下：

第一，笔者中午经历了亲手埋葬一只小鸟；
第二，当日傍晚天空出现小鸟形状的云彩；
第三，小鸟云彩出现在西方，而不是其他方位（西方暗寓极乐世界）；
第四，云彩的方位在西方，而天空其他方位没有任何云彩；
第五，时间正好是日落后，“显灵”会不会在日落后的夜晚时段？
第六，构成小鸟形状的时间很短，而偏偏被笔者看见；
第七，“3.15”的日子，一个中共以假治国后还立牌坊“打假”的日子。

在如此“机缘巧合”下，笔者相信这绝不是偶然，冥冥之中自有“天道”。一只小鸟尚且能“显相”，那么在中共残暴统治下致死的人，难道就不能“显相”？就不能做更多点的事儿？被中共迫害的新疆人、被中共镇压的香港人、被中共超限生化武器冠状病毒无声杀害的武汉人、中国人、全世界的人，他们的英灵就这样拉倒不跟中共算账？笔者绝不相信。

亲身经历这件“神奇”的事情，笔者始终无法忘怀，笔者在此记录成文，对生命表达尊重，感悟生命的神迹，感受万佛万神的力量，也与读者分享。
