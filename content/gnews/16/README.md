澎湃新聞援引網信北京微信公號10月10日消息稱：北京市互聯網信息辦公室針對“網易新聞”APP、網易網“網易新聞”頻道、“網易號”跟評環節多次傳播違法違規信息等問題嚴肅約談網易網負責人，責令其立即自查自糾、全面深入整改，嚴肅處理相關責任人。網易網負責人表示，在整改期間自行暫停“網易新聞”APP跟評功能壹周，時間自10月10日12時至10月17日12時。

筆者對此有兩個疑問：1、“違法違規信息等問題”是什麽？難道是有戰友在評論區說出了大家在心裏都知道卻都不敢說的真相？2、網易網可以隨意“自行暫停‘網易新聞’APP跟評功能壹周”嗎？

根據《CCP憲法》第二章《公民的基本權力和義務》第三十五條“中華人民共和國公民有言論、出版、集會、結社、遊行、示威的自由。”北京互聯網信息辦公室，無疑侵犯了憲法賦予網易網的新聞自由的權力。北京互聯網信息辦公室和網易網負責人，無疑侵犯了憲法賦予公民的言論自由的權力。

根據無國界記者組織每年評估的世界新聞自由指數，作為世界第二大經濟體的CCP常年處於倒數位置，與赤貧、獨裁和戰亂國家的新聞自由度難分伯仲。
