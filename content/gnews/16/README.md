据中新网2020年11月12日报道称，中共国移民管理局为防止境外输入性病例，将严格控制不必要人员跨境活动。将从严审批出国旅游等非必要事由的出入境证件申请，劝阻和限制内地居民旅游、探亲、访友等非必要、非急需出入境活动。

以上消息是中共国移民局副局长尹成基，在当日召开的国务院联防联控机制就加强冬季疫情防控，和深入开展爱国卫生运动有关情况的发布会上透露的信息。中共国移民管理局特别强调，随着国际疫情形势发展，将把外防输入作为当前工作的重中之重。结合冬季口岸边境季节性的特点，继续强化移民出入境管理措施。同时提出了“四个严”的指导意见，具体内容如下：一是严格口岸通道管控。二是要严格控制不必要人员跨境活动。三是严密边境地区人员和交通工具通行管理。四是严厉打击非法出入境违法犯罪活动。该消息一经发布，在海内外华人世界瞬间成为焦点话题。“四个严”措施的贯彻实施，将意味着中共国以防疫为借口，断绝内地居民旅游、探亲和访友等正常出入境活动，宣告中国特色的闭关锁国的时代正式到来！
