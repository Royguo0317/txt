根据中共媒体报道，湖北省十堰市星期天（6月13日）发生的燃气爆炸案导致的死亡人数已经从最初的12人增加到星期一（6月14日）的25人，而现场的搜救和清理工作仍在紧张进行当中。

湖北十堰市附近居民称，现场情况十分恶劣，死亡人数可能多达300人以上，受伤的人员更是个未知数。然而该起重大事情并未引起社会的关注，反而是娱乐新闻铺天盖地。由此可见，中共治下的百姓幸福来源于几点：一是对世界的无知，二是对现实的冷漠，剩下的就是对权力的崇拜和对金钱的渴望。

以下是近期见诸于墙内网络上的一些社会恶性事件：

- 5月8日，上海青浦，一男子持刀杀害一男后逃至附近一小超市内持刀劫持一店员。嫌犯被警察击毙，人质获救未到受伤。
- 5月17日在上海，一名女子在分众传媒上海办公楼内持刀划伤5人。肇事者称，因情感纠纷进入办公楼寻人未果，后来情绪激动持水果刀伤人。
- 5月22日，大连劳动公园门前突发严重交通事故，一辆轿车猛烈撞击在过路的行人，没有任何减速的迹象，导致4死3伤。肇事逃逸后被警方抓获的男子31岁。
- 5月23日，江西南昌，一男子持刀伤人，造成1死1伤。犯罪嫌疑人案发后自杀身亡。
- 5月24日，河北任丘，一所中学校门外发生砍人事件，造成3人受伤，包括一名家长和两名学生，其中一名伤者伤势较重。行凶男子30岁，无业。
- 5月28日在湖南郴州，5名小学生在校外被一男子砍伤。当天，在浙江义乌稠城街道化工路，一名男子持刀捅伤一名女子及男子。同一天在湖北武汉，一男子持刀在一小区外伤人，嫌疑人被控制。警方称，有人受伤，无人死亡。
- 5月29日在江苏省南京，41岁男子因感情纠纷行凶，驾车当街辗压前妻、撞车逃窜、持刀捅人，共伤及7人，自杀未遂，共8人送医治疗。同一天，浙江嵊州市发生一起杀人案，导致3死2伤。
- 6月3日，广西柳州，一名42岁男子在社区持刀伤人，现场两人伤势过重抢救无效死亡。
- 6月5日，四川眉山，一男子将理发店内的两人杀害后逃窜，警方发出最高悬赏10万元希望协查通报。


近期另一起令社会震惊的大案是4月28日广西玉林北流市一幼儿园发生的重大持刀闯入伤人事件，造成2人死亡、16人受伤。 18名死伤者中有2名教师，16名幼儿。网传视频显示，幼儿园的游乐区有多名儿童受伤，身上有大量血迹，有孩子坐在地上大哭。

恶性事件如此众多，令人不得不对中共社会的治安状况产生巨大的疑问。而媒体每次只是进行简单的报导，然后这些事情就草草了结。又有谁会去持续关注？媒体轻描淡写的一句话下，却是那些永远失去的生命，以及一辈子受伤的家庭、父母和孩子。

正当中共的宣传机器开足马力为其“百年党庆”歌功颂德之际，这些恶性案件却不断打脸中共，让人们看到中共耗费巨资进行的“维稳”，维护的只是政权的稳定，而不是人心和社会的稳定。执政为民，民心所向；掌权为己，自取灭亡。当欲望失去了枷锁，就没有了前进的道路，只能向左或者向右。左边是地狱，右边也是地狱。而我们身为新中国联邦人，我们的头上，应该只有天空！
