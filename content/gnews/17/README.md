**据《https://dailycaller.com/》通用任务和分析记者，2021年6月15日报道：**
美国和欧盟周二在布鲁塞尔举行的联合峰会上正式成立了贸易和技术委员会（TTC），以协调贸易和技术问题并对抗中共国在这些领域日益增长的影响力。
美国和欧盟都在一份联合声明中承诺重新建立跨大西洋伙伴关系。TTC不仅旨在加强美国和欧盟之间的贸易、投资和技术合作，而且还旨在推进共同的民主价值观和对人权的尊重。
联合声明指出，贸易委员会还将“启动”一项关于新兴技术和全球供应链等事项的合作议程。乔.拜登总统的政府上周二宣布，它正在建立一个“供应链贸易突击队”，专注于巩固半导体和电动汽车电池等关键行业的供应链。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/17082649/10271.jpg)2021年6月15日，欧盟委员会主席乌苏拉.冯德莱恩、美国总统乔.拜登和欧洲理事会主席查尔斯.米歇尔抵达布鲁塞尔的欧盟总部参加欧盟-美国峰会（Kenzo Tribouillard/AFP via Getty Images）。
欧盟委员会在一份声明中指出，TTC将由工作组组成，以解决与贸易和技术有关的具体问题。该理事会将由国务卿安东尼.布林肯、商务部长吉娜.雷蒙多和美国贸易代表凯瑟琳.戴共同主持。
拜登政府的一位高级官员周一在联合峰会前告诉记者，“这里的概念是，美国和欧洲在二战后为世界经济奠定了基础，现在必须共同为下一代写下道路规则，特别是在经济和新兴技术领域，”
这位官员还告诉记者，贸易与技术委员会在应对中共国在贸易和技术事务方面日益增长的影响力这一“重大挑战”方面将非常重要。他们补充说，该委员会将补充总统通过与民主盟友和伙伴协调与中共国竞争的战略。
这位官员说：“处理中共国的非市场行为、经济滥用行为，当然还有它为21世纪的技术制定道路规则的努力，将是这个委员会工作的一个重要部分。”
