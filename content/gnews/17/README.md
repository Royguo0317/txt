## 在欧洲的中共“辛普森夫人”计划
` 喜马拉雅德国慕尼黑感恩农场` [轉載自GNews](https://gnews.org/zh-hans/1050328/)

作者：peacelv
审核：MiYa

![]()![](https://gnews.org/wp-content/uploads/2021/03/photo-1523875194681-bedd468c58bf.jpeg)

二战时期，辛普森夫人是活跃在欧洲政要身边的代表人物。进入二十一世纪的今天，中共以“辛普森夫人 ”命名的全球间谍计划，从欧洲开始陆续被暴露出来。仅以下媒体曝光的一些事件，就能对中共猖獗的间谍活动窥知一二。

**2019年2月12日**，欧盟主要行政机构欧盟执行委员会总部、北大西洋公约组织总部，皆设立在比利时布鲁塞尔。据近日欧盟对外事务部（EEAS）评估，在布鲁塞尔约有两百五十名中共间谍和两百名俄国间谍，他们多数集中在各自国家的大使馆或经贸代表处。此外，EEAS指出以前在布鲁塞尔最为活跃的是俄国间谍，而现在则是中共间谍以及俄国间谍。

**2020年1月 15日**,德国联邦总检察长揭露一起中共间谍案件，此案涉及一家游说公司的三名职员。调查人员搜查了柏林和布鲁塞尔的办公室和公寓。

德国联邦总检察长Peter Frank正在调查一起涉嫌中共间谍的案件，该案件针对来自德国的前欧盟外交官和德国一家知名游说公司的两名职员。按照《明镜》（SPIEGEL）获得的信息，其中两人已将私人和商业情报提供给中共国家安全部，第三名嫌疑人就此进行了解释。

此次调查的重心是一位欧盟委员会的前工作人员，他后来成为欧洲对外事务部的外交官。他曾在布鲁塞尔欧盟内部担任领导位置，之后又成为欧盟驻多个国家的大使。

2017年，他结束了在欧盟的职业生涯，自此进入一家游说公司。调查人员发现，同一年，他开始向他的中共情报官员传递情报。此外，他还招募了另两位被告。其中一位为了与其相关情报官员见面，曾前往中国。另外，这家游说公司过去曾为多家德国大型公司提供咨询服务。

**2020****年5月16日，**法国《世界报》刊登中共在**布鲁塞尔**进行的间谍活动，这篇报导指出，马耳他大使馆是一座非常现代化的九层建筑，它位于欧盟执委会总部大楼前。报道称，马耳他大使馆是一个侦测欧盟情报的绝佳地点，中共于2007年为该大使馆支付了总额为2100万欧元的全部费用，包括所有的装修工程和购买整栋大楼的费用。

这栋大使馆的安全监控中心装有防弹玻璃，和邻近建筑物相接的墙，则是装上厚厚的防弹层。比利时安全机构一直指中共在此进行间谍活动。英国情报机构也发现了这个秘密，并一直协助比利时进行调查。

据比利时国家安全局（VSSE）分析指出，马耳他大使馆内藏有中共情报单位安装的特殊装置，自2010年初以来，中共一直对欧盟机构进行监视，比利时情报部门则认为，监视活动可能至今仍持续进行。

据《马耳他时报》报导，马耳他政府当局强调，所有中共捐赠的木制家具在安装前均已被彻底扫描，一切都按照北约标准进行；而且在过去两年，这些家具约有八成已被更换。

世界报的调查还列举了中共情报机构如何通过孔子学院，当地华人，欧盟公务员等搜集欧盟政治和经济信息。

**2020****年05月16日，**自由亚洲昨日（15日）报导，比利时国家安全局发表声明谴责中共在比利时从事间谍活动。声明指，中（共）国为了实现「2025年中国制造」计划，不择手段地从境外窃取科技成果，他们的间谍试图从**欧盟的生物武器专家、疫苗专家、英国的制药巨头葛兰素史克公司（ GSK）及欧盟的高技术公司**窃取情报。

声明称，比利时必须对类似于中共这种国家保持战略独立性。

欧洲观察网站5月7日披露，比利时国家安全局（VSSE）在2016年的一份机密报告中就写道：「**中（共）国情报机构无论是处于防御还是进攻目的，都对于生物战和疫苗领域的发展非常感兴趣，而且中（共）国一直在将埃博拉病毒作为用于进攻的病毒进行研究**」。

比利时国家安全局还担心，中共正在比利时修建的「中国比利时技术中心」，预计2021年竣工使用，这里有20几家研发生命科学和人工智能的公司，将有800名中共国的专家前来工作。这里即使不是中共情报机构的第一线，也会被中共国安部作为进入欧洲的后门；而葛兰素史克公司则是中共间谍活动的一个主要目标。

**2020****年9月20日，****据英国一名政府官员透露，曾经供职于英国情报机构军情六处（MI6）的英国商人弗雷泽·卡迈伦（Fraser Cameron) 因为向中共卧底间谍出售情报指称被调查。**

弗雷泽·卡迈伦现在经营智库欧亚中心（EU-Asia Centre），被怀疑将有关欧盟的敏感情报交给两名中共间谍。而这两名中共间谍的公开身份是驻布鲁塞尔的记者。

弗雷泽·卡迈伦还被指因这些情报获得报酬数以千计欧元。不过，他向英国《泰晤士报》表示，所有这些指称「荒谬」。

围绕现在弥漫全球的冠状病毒危机，以及中共在欧洲布鲁塞尔等地的间谍活动，这些都是蓄谋已久，相信随着欧洲政要以及人民的逐渐觉醒，中共在欧洲及至美国甚至于全世界的“辛普森夫人计划”将陆续被揭露出来。
