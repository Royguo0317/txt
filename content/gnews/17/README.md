# 对被盗选举之剖析–欺诈不可容忍

在本届总统选举中，有关选举舞弊的证据每天层出不穷。除了Gateway Pundit团队出色的报道外，大多数媒体（包括印刷和电子媒体）都在进行全面的宣传运动，试图说服美国公众相信被揭露的欺诈指控是阴谋论。自从前苏联时期那段糟糕的日子以来，我们还没有目睹过如此可鄙的谎骗模式。如果把巴格达·鲍勃否认美国坦克已经越过伊拉克比做一枚手榴弹爆炸，那么当前的虚假宣传活动则相当于一队B52轰炸机投下了多枚氢弹。

民主党窃取选票的计划完全超出当初虚假选票的原定战略。这次，它包括仅附有老乔•拜登姓名的人造选票和可将这些选票数放大数倍的投票机，同时将唐纳德·川普的选票计数为少于一票。所有这些机器都有一个可供中共、民主党特工及其硅谷助手可以访问的软件后门。

使用这种人造选票和机器修改的组合，导致在六个主要州的结果都是欺诈性的，而且更足以证明欺诈性选票使拜登在以下每个州以所谓的获胜幅度超出：

这些数字说明了一个故事。我们被迫相信乔·拜登赢得了比巴拉克·奥巴马更多的选票。您还记得奥巴马在2008年大选中吸引的人群吗？尽管他们没有达到2020年唐纳德·川普对大片人群的吸引力，但当时他们的盛况是空前的。乔·拜登（Joe Biden）绝不可能获得比巴拉克·奥巴马（Barack Obama）还多1300万张选票。奥巴马有个人魅力和追随者。拜登所依靠的，就好比在7月的烈日下，一堆腐烂的垃圾吸引来的苍蝇，比拜登选举活动的支持者还多。

**考虑到这些事实（感谢帕特里克·巴沙姆的出色分析）：**

• 川普总统获得的选票比任何以前寻求连任的总统和任何以前参与竞选总统的候选人都要多。与2016年相比，他多获得了1200万张选票，增幅超过历史上的任何一位。

• 川普得到的黑人选民支持率比2016年提高了50％。就全国而言，乔·拜登的黑人支持率下降到远低于90％的水平，低于民主党总统候选人通常会输掉的水平。

• 川普将他在西班牙裔美国人全国投票中的份额提高到35％。

• 就像他们的地区对手俄亥俄州和爱荷华州 一样，中西部各州密歇根州、宾夕法尼亚州和威斯康星州总是向同一方向摇摆。而俄亥俄州也同样随佛罗里达州一起摇摆。当前的统计数据显示，除了个别城市，“铁锈地带”多数都偏向川普方向。然而，由于在底特律，费城和密尔沃基出现了明显的似雪崩一样多的黑人选票，使拜登在密歇根州，宾夕法尼亚州和威斯康星州处于领先地位。拜登的“获胜”优势几乎完全来自这些城市的此类选民，而巧合的是，他的黑人选票仅在获得胜利所需的确切地点才达到峰值。

• 拜登以17%的县得票率创历史新低获胜；他只赢得了524个县，而奥巴马在2008年赢得了873个县。

• 在选举日深夜，随着川普稳定的大幅度领先，许多摇摆州停止了计票。在大多数情况下，观察员已被逐出投票点。计票工作普遍在没有观察员的情况下继续进行。

• 恢复计票后，统计上的异常投票数是新常态。它们的数量异常庞大（数十万），并且拜登对川普的投票率异常高（超过90％）。

• 丢失选票 。在宾夕法尼亚州的特拉华县，存在47张USB卡上的50000张选票丢失了。

• 非居民选民。马特·布雷纳德（Matt Braynard）的选民诚信项目估计，有20312名不再符合居住要求的人在乔治亚州进行了投票。拜登胜出了12670票。

• 几项严重的“监管链”问题。无效的居住地址，创纪录的死人投票数字。原始状态下没有折痕的选票，也就是说，它们没有按照法律要求被放到信封中邮寄。

• 统计异常。在乔治亚州，拜登以89％的得票率赶超川普。在接下来的53批选票统计中，拜登在每批中以完全相同的50.05％对49.95％的得票率领先川普。同样的情况也发生在威斯康星州、密歇根州、宾夕法尼亚州和弗吉尼亚州。

2020年大选的真相是，唐纳德·川普赢得了历史性胜利。但是，那些从战争和与中共相勾结的关系以及牺牲美国工人利益的国际订单中获利的美国深层利益集团，渴望淘汰川普。

唐纳德·特朗普是否会在最高法院和国会胜诉还有待观察。美利坚合众国的未来取决于这一时刻。到目前为止，司法部和联邦调查局都没有采取行动。如果他们不做正确的事，维护法律，将犯罪分子绳之以法，那么这个国家的未来岌岌可危。

如果乔·拜登宣誓就职，他的统治将是短暂而灾难性的。投票支持唐纳德·川普的7400万美国人中，大多数人不会承认他是合法领导人。革命的种子将被播下，统治华盛顿的既定秩序将面临该国历史上前所未有的全国反对运动。
