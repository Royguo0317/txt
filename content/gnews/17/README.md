## 今冬的疫苗之爭—妳準備好打疫苗了嗎

***mRNA******疫苗簡介***

mRNA的中文名稱為信使RNA，即攜帶遺傳信息的能指導細胞質中核糖體合成蛋白質的一類單鏈核糖核酸。目前作為疫苗研究的RNA主要有兩種類型：非複製性mRNA和病毒衍生的自擴增RNA。傳統的基於mRNA的疫苗僅對感興趣的抗原進行編碼，而自擴增RNA不僅對抗原進行編碼，而且還對病毒複製機制進行編碼。針對mRNA疫苗易分解的缺點，目前已有大量的體外或體內轉染試劑開發出來，它們可促進細胞對mRNA的攝取，並保護其不被降解。

現在歐美大藥廠研發的疫苗集中在mRNA疫苗上。2018年Nature雜誌曾刊發一篇題為《mRNA，疫苗的新時代》，提到該疫苗具備四個突出的優點：高效、安全、開發時程相對短、生產成本相對低。該疫苗的缺點也很突出，即注射到體內後易分解，不穩定。目前在研mRNA除了針對CCP病毒，其餘的主要集中在針對流感病毒、HIV、癌症等疾病。迄今為止，僅有Moderna研發的針對寨卡病毒的mRNA疫苗獲得FDA快速通道批准。針對CCP病毒，最早進入臨床的疫苗也是由Moderna公司研發，為脂質納米科技遞送的mRNA疫苗。

***疫苗效果如何檢測***

根據新英格蘭醫學雜誌和柳葉刀的文章，目前在研疫苗主要需要進行安全性、免疫應答效果 (免疫原性)檢測。由於疫苗的特殊性，在臨床實驗中也和普通藥物有區別。概述如下：


| 臨床實驗階段 | 主要測試項目 |
| --- | --- |
| 臨床前期 | 在細胞和動物身上實驗，檢測疫苗是否導致免疫反應 |
| I期 | 將疫苗在小範圍健康人群中實驗，以檢測期安全性和探索激發免疫系統的合適劑量 |
| II期 | 將疫苗在100人以上人群中實驗，分年齡段檢測疫苗在不同人群中的反應 |
| III期 | 實驗規模上升至幾千人或更多，如果有安慰劑對照，則觀測注射疫苗的人群中是否有統計學意義上更低的感染率。FDA在今年6月表示冠狀病毒疫苗需保護不低於50%的接種人群才能稱其有效性達標。III期實驗還能有助於發現I期II期中不常見的副作用。 |
| 有限批准 | III期資料如果良好，可獲得有限度的批准，可在規定人群中使用 |
| 正式批准 | 各國醫療監管機構對臨床資料進行審核，如果疫苗防護效果良好，且副作用率相對可控，則可獲得批准上市 |

***CCP******疫苗***

根據CCP網站消息，已經進入臨床階段的８個疫苗中，CCP疫苗就占了４個，包括陳薇院士團隊研發的腺病毒載體疫苗，和3個滅活疫苗。滅活疫苗為第一代疫苗，安全性低且成分不明確。但是CCP為了掩蓋其在疫苗研發上的弱勢和注射CCP疫苗的副作用，不僅在國內宣傳中極力弱化了普遍的副作用，還將自己的II期臨床結果刊登在了最新一期柳葉刀雜誌上。陳薇團隊將注射疫苗後副作用較高歸因為國內外發熱標準不一致，並表示：“如果以國外三十八度為標準的話，我們目標發熱率只有百分之四點八，還有疲乏頭痛肌肉痛惡心等，都比其他已發表資料的疫苗低”。但根據他們在柳葉刀上公佈的資料，有近半數出現副作用，包括46%發燒、44%疲勞、39%頭痛，整體上有9%接種者因而出現活動受礙的現象。對此，閆博士指出，CCP從未成功研發過疫苗，目前疫苗安全性未知，副作用大，能產生ADE讓病毒更容易入侵人體。

***ADE******效應***

前段時間路德節目中揭露的ADE效應，全稱為抗體依賴性增強（antibody-dependent enhancement，ADE），主要是指機體中由病原體感染引起的反應。當機體遭遇病原體感染時，原有的中和抗體不僅不能防止病毒侵入人體細胞，反而可以增強病毒在體內的複製，引起嚴重的機體病理反應。由於CCP病毒是RNA病毒，特性就是容易發生變異。目前全球已報導５例二次感染病例，病毒的基因測序表明兩次感染都是不同的毒株。

該效應帶來最大的潛在危害就是導致疫苗引起的抗體無力對抗新感染的病毒，造成疫苗無效。舉例來說明就是如果感染了A型CCP病毒，體內有了A型抗體，但此抗體對另一個B型CCP病毒無效（來源：[https://www.medsci.cn/article/show\_article.do?id=f17119205820](https://www.medsci.cn/article/show_article.do?id=f17119205820)）。假設病毒庫裡出來了新的亞種，那麼由疫苗產生的抗體有可能充當了“特洛伊木馬”引狼入室，導致病毒瞬間暴增，使得免疫系統崩潰，悲劇再次重演。

***農村包圍城市***

由於CCP病毒疫苗涉及到全人類健康福祉，甚至對全球的政經和平發展起到不小的作用，誰掌握了有效的疫苗，誰就能在經濟復興的跑道上領跑，還能把握其他國家的命脈。但是，目前進入III期臨床階段的歐美藥廠都表示不希望從中賺取大額利潤，阿斯利康的疫苗在歐洲標價僅為2.8美元一劑。面對歐美技術上的優勢，CCP不甘其後，採用了他們一貫的“農村包圍城市”策略，在技術較不發達國家大搞“疫苗外交”，如國藥集團的滅活疫苗在阿聯酋、秘魯、摩洛哥、阿根廷獲得臨床試驗批件，以爭取開展大範圍的III期臨床實驗；還有和巴西、印尼、巴基斯坦、俄國和菲律賓政府進行疫苗合作，將生產出的疫苗分發到這些國家。另外，兩天前國藥集團和華為公司簽署了新冠滅活疫苗資源保障和服務支援戰略合作協定。相關新聞指出20萬華為外派員工工作在一些疫情十分嚴峻的國家和地區，因此“通過緊急接種疫苗、保障復工複產迫在眉睫”。CCP如此急火攻心的研發生產推廣疫苗，並將主要接種人群放在普通老百姓上，對照前文的ADE效應，如果接種後病人仍感染病毒，那麼是否又要甩鍋到ADE效應，或者說是由於歐美疫苗的技術缺陷所導致的呢。這樣一來，不僅全球追責會變成泡影，世界的政治經濟活動都會再受到一次不小的衝擊。

借用魔戒裡的情節來說明現今科學的力量，如果它被邪惡的勢力所掌握，那麼世界都有被摧毀的可能。面對貪欲，面對邪惡力量，我們只有站出來傳播真相，讓科學界的人士儘早意識到CCP病毒的殺傷力和來源，讓各國政府儘早採取直接有效的措施（比如推廣羥氯喹），積極有效的降低感染人數，開發疫苗，同時保護疫苗的智慧財產權，才能在這次鬥爭中大獲全勝。正如閆博士所說：留給人類的時間不多了，讓我們只爭朝夕，為世界和平和我們的後代帶來福祉。
