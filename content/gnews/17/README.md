# 索罗斯提拔Smartmatic董事长为开放社会主席
[!\[\]()!\[\](https://spark.adobe.com/page/KNvIyrbbak9W3/images/5f407cd8-a926-4e47-b988-e9e5c35c83ab.jpg?asset_id=42eea93b-0690-4a41-83bb-63ce8c4e360c&amp;img_etag=%22e2bab8bb6c739822c768a8e888df9848%22&amp;size=1024)](https://spark.adobe.com/page/KNvIyrbbak9W3/images/5f407cd8-a926-4e47-b988-e9e5c35c83ab.jpg?asset_id=42eea93b-0690-4a41-83bb-63ce8c4e360c&amp;img_etag=%22e2bab8bb6c739822c768a8e888df9848%22&amp;size=1024)
**选票公司Smartmatic的主席马克·马洛奇-布朗勋爵（Lord Mark Malloch-Brown）已被提升为乔治·索罗斯的开放社会基金会的主席。**

12月4日宣布的消息使马洛奇·布朗与这个革新的超级富豪之间错综复杂的关系又上了一个层次，其中包括从他那里租了一套纽约的房子。马洛克·布朗将接替帕特里克·加斯帕德(Patrick Gaspard)，很多人指称加斯帕德将加入乔·拜登(Joe Biden)的内阁，可能成为劳工部长。

“接替他担任主席的将是马克·马洛奇·布朗，他是前联合国副秘书长和英国部长，目前在基金会的全球理事会任职。马洛奇·布朗将于1月1日接任。”开放社会基金会的新闻稿中写道。新闻稿中没有提及马洛克·布朗作为Smartmatic公司主席的职位，该投票公司的结果已被篡改，并且和多米尼系统关系密切。

马洛克·布朗本人在2010年代表Smartmatic承认：“我们的部分技术是由多米尼授权的。”

在晋升之前，马洛克-布朗曾担任索罗斯量子基金副总裁、索罗斯基金管理和开放社会研究所副主席。

然而，随着索罗斯为击败川普总统投入了无数资金，美国投票公司与反川普行动者之间的猫腻越来越多，而马洛克·布朗的新职位又增加了这种嫌疑。
