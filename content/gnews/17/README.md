# **中共国在拜登任职后几分钟内宣布制裁数十名川普政府官员**

[!\[\]()!\[\](https://spark.adobe.com/page/yR1x0aswYsKZ0/images/8e50741e-3a7d-44fa-8bf7-d2e9ae4f4b69.jpg?asset_id=ef88ec00-f932-4053-9f59-a3d87a91f982&amp;img_etag=%2237f4a6b3438f5fbbc79d1a1a1b906aa3%22&amp;size=2560)](https://spark.adobe.com/page/yR1x0aswYsKZ0/images/8e50741e-3a7d-44fa-8bf7-d2e9ae4f4b69.jpg?asset_id=ef88ec00-f932-4053-9f59-a3d87a91f982&amp;img_etag=%2237f4a6b3438f5fbbc79d1a1a1b906aa3%22&amp;size=1024)

**中共国外交部周三对28名与川普政府有关的美国人实施制裁，称他们损害了中共国主权。**

“中共国决定制裁28名损害中共国主权和对中美关系问题负责的个人。”中共国外交部1月20日在其官网上一份声明中说。

“这些个人和他们的直系亲属将会被禁止入境中国大陆、香港和澳门。他们及与他们相关的公司和机构也被禁止在中共国做生意。”声明说。

[!\[\]()!\[\](https://spark.adobe.com/page/yR1x0aswYsKZ0/images/4558be9b-9b8d-4405-8b75-07ddb239455f.jpg?asset_id=7c64fc08-717c-4de7-a589-1794a3c64574&amp;img_etag=%22cce78fea2cb873b1d19a3bc74463c740%22&amp;size=2560)](https://spark.adobe.com/page/yR1x0aswYsKZ0/images/4558be9b-9b8d-4405-8b75-07ddb239455f.jpg?asset_id=7c64fc08-717c-4de7-a589-1794a3c64574&amp;img_etag=%22cce78fea2cb873b1d19a3bc74463c740%22&amp;size=1024)

这些被制裁的个人包括川普政府前官员，如国务卿彭佩奥；贸易和生产政策主任彼得•纳瓦罗；国家安全顾问奥布莱恩；亚洲和印台事务助理国务卿大卫•R•史迪威 (David R. Stilwell)；副国家安全顾问马修•波廷格(Matthew Pottinger)；卫生与公众服务部部长阿扎尔(Alex M. Azar II)；经济增长、能源以及环境副国务卿基思•克拉奇 (Keith J. Krach)；美国驻联合国大使凯莉•克拉夫(Kelly D. K. Craft)，国家安全顾问博尔顿；以及白宫首席战略顾问班农。

中共国外交部指控这些被制裁的个人在过去四年川普政府中推行“反华”政策。川普的任期在1月20日中午正式结束，之后几分钟内中共国就宣布了对这些个人的制裁。

伊朗周三与中共国一起宣布制裁川普政府成员，包括川普本人，称他宣扬恐怖主义。

伊朗是世界上支持恐怖主义的首要国家。

“伊朗外交部将一些美国人列入到制裁名单，因为他们犯下了恐怖主义罪行，促进和支持恐怖主义，这些人是对地区和国际和平安全的严重威胁，他们违反了包括人权在内的国际法的基本规则和原则。”伊朗外交部发言人赛义德•哈蒂布扎德(Saeed Khatibzadeh)在声明中说。

德黑兰周三制裁了很多中共制裁的美国官员，包括国务卿彭佩奥和川普总统。

“美国官员已被列入黑名单，因为他们全力支持以色列政权对巴勒斯坦人民的侵略行为，特别是支持以色列恐怖主义攻击伊朗核物理科学家穆赫辛•法赫里扎德 (Mohsen Fakhrizadeh)。”哈蒂布扎德称，指的是伊朗声称去年11月以色列使用远程控制武器暗杀了法赫里扎德的行动。

川普总统的政府曾经多次以支持“恐怖主义实体”为由制裁伊朗政府和军方官员。
