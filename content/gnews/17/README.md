# **关键信息日志从密歇根州投票机中丢失**

更新（1225ET）：根据12月14日发布的一份符合法院令的取证报告，《大纪元》记者伊万·彭措科夫（Ivan Pentchokov）报道了密歇根州安特里姆县的Dominion （多猫腻）投票机缺少重要的安全和裁定日志。

[!\[\]()!\[\](https://spark.adobe.com/page/xYwlcTY97JSlt/images/4404d836-5bdd-41b4-bf7c-81950354f8b9.jpg?asset_id=3d5b043f-c1d1-463f-9efa-848f0b207387&amp;img_etag=%22e34527c200e9c91a2f21a56ff89013cd%22&amp;size=2560)](https://spark.adobe.com/page/xYwlcTY97JSlt/images/4404d836-5bdd-41b4-bf7c-81950354f8b9.jpg?asset_id=3d5b043f-c1d1-463f-9efa-848f0b207387&amp;img_etag=%22e34527c200e9c91a2f21a56ff89013cd%22&amp;size=1024)

“重要的是，这个计算机系统显示了前几年的投票裁定日志，但唯独缺失了2020年选举的所有相关日志。干预计票裁决过程是人工操纵投票的最简单方法。记录的遗失使任何形式的审计都无法进行。这个非常明显的重要日志丢失是非常可疑的，因为在这套软件系统中，前几年的同样信息都是存在的。” Russell Ramsland的报告指出。

**“我们可以得出肯定的结论，2020年选举记录已经被人工删除。”**

系统判定记录的丢失是令人惊讶的，因为取证调查发现投票机曾经拒绝了大量的选票，这是一个人工干预的过程，选举工作人员在此过程中决定了每次投票的最终结果。

密歇根州民主党秘书乔塞琳·本森，多猫腻公司以及安特里姆县（Amrim County）的发言人均未对该质询发表回复。

正如SaraACarter.com的Sara Carter先前介绍的，密歇根州的一名法官下令，星期一公开发布支持川普总统的律师们提交的报告，以及他们提出的选举欺诈指控，这将显示出人们对安特里姆县投票中操纵计算机投票系统行为的严重担忧。据该案的直接知情人士称，取证调查报告中的数据将揭示，该县用于投票的计算机系统并不安全，而且系统中装有外国组件，使其容易受到操纵或欺诈。

[!\[\]()!\[\](https://spark.adobe.com/page/xYwlcTY97JSlt/images/f66ddb2a-8567-4393-81a2-64e4e87f1c0e.jpg?asset_id=7eda2690-9b41-4a1d-a107-bfe044262399&amp;img_etag=%2258a12598b9c8ed8489621427091fa9de%22&amp;size=2560)](https://spark.adobe.com/page/xYwlcTY97JSlt/images/f66ddb2a-8567-4393-81a2-64e4e87f1c0e.jpg?asset_id=7eda2690-9b41-4a1d-a107-bfe044262399&amp;img_etag=%2258a12598b9c8ed8489621427091fa9de%22&amp;size=1024)

密歇根州助理检察长埃里克·格里格（Erik Grill）代表密歇根州务卿乔塞琳·本森（Jocelyn Benson）立即反对发布该报告。川普总统的支持者说，该报告揭露了选举欺诈和影响美国国家安全的重大问题。

据《底特律新闻》报道，Erik Grill认为这个将要发布的报告是“不准确、不完整而且具有误导性”。

*Erik Grill在周一上午的一次虚拟法庭听证会上说：“没有理由隐藏。”*

*“没有什么可隐藏的。”*

但是，致力于揭露认为对美国安全和基础设施构成严重威胁的律师和计算机专家说，该报告将揭示数据的不规范性以及外国势力对软件系统的干预。

**一位知情人士说，法律团队将“根据法官的要求，于周日上午8:30向法官提交取证报告”。**

据消息人士称，本周末的计算机系统取证报告发现有严重影响国家安全的证据，因为“选举系统被归类为关键基础设施，这对国家构成威胁，是一个国家安全的问题。

该诉讼是由安特里姆县居民威廉·贝利发起的。前共和党议员巡回法官凯文·埃尔森海默（Kevin Elsenheimer）允许联合证券运营集团（Allied Securities Operation Group）和贝利（Bailey）给该县的22个投票机进行图像取证，并审查其他与选举有关的材料，以确保选举的完整性。

**该取证分析报告目前处于保护令之下。下周一，Elsenheimer法官会下令作出一些修改，之后才可以公开发布。**

安特里姆县大约有23000名居民，发现多猫腻投票系统把本来投给川普总统的大约6000张选票未经解释就转给了乔·拜登， 从而引发了川普支持者的持续调查。

选举机构给出了不同寻常的的解释是：在这个倾向于共和党的县，由于未能更新投票软件导致乔·拜登得到了数千张本属于川普的选票。大多数川普的支持者和密歇根州共和党的许多人都不接受这个解释。

**如果该取证报告中，对选举的违规行为以及涉及全国多个州的多猫腻投票系统涉嫌欺诈问题是准确的，则该报告可能会产生滚雪球效应，其它使用该系统的州的立法机关会要求对该系统进行类似的司法审核。**
