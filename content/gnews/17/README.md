## 班農戰情室199-1 制止中共否則台灣步香港後塵
`2020-05-29 19:41 GM67` [轉載自GNews](https://gnews.org/zh-hant/217064/)

作者：VOG翻譯組功夫熊貓🐼^文竹^🐼        編輯：VOG翻譯組flasher

香港就像上一次冷戰時期的柏林， 如果香港因美國缺乏對中國的行動而失敗，台灣將緊隨其後，美國在亞洲的所有利益也將如此

蓬佩奧：**CCP**通過在南中國海的島嶼上進行武器化來破壞世界對他們的信任

班農與蘋果日報合伙人馬克西蒙談論中共強行通過涉港國安法

班農：「Mark Simon（馬克·西蒙），蘋果日報黎智英的合伙人，擔任市場總監，Mark是在香港工作的傑出外籍人士之一，他最近才回來幾天，也是陳日君主教非常親密的夥伴，總體而言是一個很棒的人。」

馬克認為（涉港國安法）是中共的持續行動，中共政策基本上摧毀了《基本法》，是對香港進一步的鎮壓，這也是關於價值觀的戰爭。

「香港在很多方面都是一個真正的西方城市。雖然沒有完全的民主，但有民主社會的所有制度，有法治，有自由貿易，有開放的市場，無歧視的法律，各種各樣的好東西。直到最近，共產黨基本上已經確定這不是他們想要的前進方向。中共去年在2019年試圖制定一部我們稱之為引渡法的法律。如果你在中國做了他們覺得你侮辱了中共的事情，他們可以把你從香港引渡出去。」

馬克表示去年因擔心引渡條例實施會面臨（中共）勒索，香港抗議活動變得非常盛大也非常有效，抗議挫敗了引渡法，所以中共氣急敗壞直接跳過香港進行立法。

「中共要接管他們認為無能的香港政府，這實際上是他們對香港行政長官林鄭月娥及其親信的看法，中共人民代表大會上強行通過一項國家安全法，這就是他們在過去幾天里所做的。在通過這些法律時，你實際上看不到什麼法案，他們說的是他們將在35或40天內向你提交一項法案，這就是法律。但我們知道會發生什麼，法案本質上是對香港的破壞，甚至有一天，當我的老闆在推特上發佈一個被國家喉舌《環球時報》說顛覆的言論時，也會被定為顛覆罪，如果《環球時報》說是顛覆，我打賭肯定會被指控為顛覆。」

班農：中共一年前就開始推進，他們目的就是有能力在暗地裡抓任何他們想抓的香港人，隨時把他們扔進中共的監獄。

關於陳日君主教，班農表示主教很值得敬佩，「他們所做的是，他們不想看到這些年輕的孩子，這些去年在香港街頭被橡皮子彈、催淚彈擊中、被高壓水槍擊中的孩子，這些是1775年的愛國者，這些是帕特里克·亨利，這些是保羅·里維爾，這些是不會回到極權獨裁統治的人們，我認為他們向全世界展示了像陳紅衣主教這樣的人，他是世界上最神聖的人，他坐在那裡，阻止這些魔鬼。」

對於中共狂妄的態度以及當下其所做所為，班農、馬克深入討論了如果西方不站出來會發生什麼。

中共媒體宣傳中共公開的表態：「這是我們的領土，我們要制定法律，我們不在乎美國說什麼，我們不在乎英國說什麼，我們不在乎歐盟說什麼，我們尤其不在乎唐納德·特朗普說什麼。我們要啓動這項國家安全法來定義煽動。」

班農：「CCP不喜歡的任何東西都可以被定義為煽動，你不會受到陪審團的審判，不會有任何權利，不會知道人身保護令的事，如果中共不喜歡，他們會帶你去一個黑暗的監獄，《環球時報》說黎智英發推特表達政治觀點的行為是煽動，那就是顛覆。這是1938年的捷克斯洛伐克，中國共產黨在恐嚇西方。」

馬克：「我認為中共在這個時候沒有任何善意來安撫西方，他們正在進行一場全面的攻擊，不論是在南中國海，還是在世界各地的統一戰線行動，只是這些現象沒有得到足夠的關注， 不管你關注中共的哪些方面，他們已經將近一百萬的人關在中國西部。看，這是一個侵略性的極權主義政權，香港真的不是他們的第一步。」

馬克認為中共不會也不允許中國有西方價值觀，中共與西方已經出現分歧，當下的任何協議中共都不可能推進，他認為這時候去擁抱中共的個人或者商業團體就是站在敵人中共一邊，換句話說，這些人宣稱自己是敵人。

馬克：「就香港而言，我認為美國首先要做的是開始考慮一些制裁措施。我不喜歡鼓吹香港特殊經濟地位的重要性，但我認為這對中共來說是可行的，換句話說，中國人民會非常樂意看到這種情況發生，進而支持香港，他們會說，看，誰拋棄了，誰離開了你。換句話說，中共將失去香港作為國際金融中心的地位。明確地說，是美國的認可讓香港成為了國際金融中心，當美國認定香港不再健康的時候，你可以和銀行吻別，可以和每個離開那裡的人吻別了；因為一旦美國說沒有**Swift**代碼，說美國不信任香港的任何支票或類似的東西；至少在官方上，這對他們來說會變得非常困難。我們要考慮的另一件事是，**CCP**官員的家庭，他們為什麼來美國？為什麼他們的孩子在這裡上學？。」

對於如何應對中共對香港的壓迫，班農引用里黎智英的推特：「能施加的最有效的制裁是凍結中共官員在美國的銀行賬戶，揭露他們的行為有多腐敗。這一行為不僅會受到香港人的贊賞，也會受到每個渴望自由的中國人的贊賞」，他還有其他一些語出驚人的地方。他說，香港就像上一次冷戰時期的柏林， 如果香港因美國缺乏對中共的行動而失敗，台灣將緊隨其後，美國在亞洲的所有利益也將如此，在中共面臨最嚴重的經濟衰退之際，現在是對其採取行動的最佳時機了。」

班農指出中共目前可能不在乎被制裁，可以不在乎港幣聯席匯率的取消，不在乎資產的損失，對中共來說，最重要的是不能讓大陸人民看到香港發生的真相，阻止大陸人民對民主自由的認知。

班農還說：「150年到200年來，美國第一次面臨被擠出亞洲的危機。21世紀是亞洲世紀，美國是亞太大國，我們也有盟友，如果我們像1938年那樣被排擠出去，當人們說與我無關，我住在堪薩斯州，我住在愛荷華州，我住在俄亥俄州，我關心奧地利做什麼，我關心捷克斯洛伐克做什麼。那麼三年半之後， 至多四年，在奧地利，在慕尼黑，捷克斯洛伐克和所有這些地方之後，美國人民的孩子們將在北非登陸，然後是西西里島，然後是意大利，然後是在中途的威克島的瓜達爾卡納爾島的諾曼底。現在這種事發生在珊瑚海，而且它就要來了，如果我們現在不阻止他們，所有這些類似的事件都將發生，這是二十世紀未完成的工作。 這是最後一群大流氓，好吧，就像墨索里尼和希特勒一樣，就像蘇聯的最後一組，讓我們拿下我們的另一個敵人——中共。」

蓬佩奧簡短的電話採訪

Jason帶來蓬佩奧的一次簡短的電話採訪，蓬佩奧提到班農團隊是很強大的秘書團。概述了香港正在發生的事情，並且非常具體地談到了取消香港的貿易地位的相關內容，他說這是年度審查的一部分，所以當看到所有這些侵犯人權的行為和中共接管香港的時候就做了這個決定。

蓬佩奧同時明確表示他正在與許多其他世界領導人溝通，提到了英國的多米尼克**·**羅布和世界上其他很多領導人，也提到**CCP**通過在南中國海的島嶼上進行武器化來破壞世界對他們的信任。**Jason**說「他用一種我很久沒有聽到過的方式非常明確地把這些內容表達出來。並且他確實推斷以後會有更多關於香港和中共的額外的行動。所以請繼續關注，這聽起來相當強大。」
