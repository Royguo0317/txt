# **卫星地图显示，中共国正在建设面朝台湾海峡的直升机基地**

**专家称大规模解放军直升机场即将竣工，并将对台湾构成威胁**

[!\[\]()!\[\](https://spark.adobe.com/page/UNDuG4u9v5BJP/images/b2613195-e850-4d05-a1aa-596bf408f574.jpg?asset_id=f0e6ebc9-09d4-42a9-a4d3-049249c9ab1f&amp;img_etag=%22ec8a7caeb4dfc5477b85fcd406212424%22&amp;size=2560)](https://spark.adobe.com/page/UNDuG4u9v5BJP/images/b2613195-e850-4d05-a1aa-596bf408f574.jpg?asset_id=f0e6ebc9-09d4-42a9-a4d3-049249c9ab1f&amp;img_etag=%22ec8a7caeb4dfc5477b85fcd406212424%22&amp;size=1024)从太空中可见中国福建省直升机基地建筑。（推特，@detresfa\_图片）

台北（台湾新闻） — 一位军事专家周五（3月12日）说，新的卫星图像似乎显示，中国福建省正在建设一个可能威胁台湾的直升机基地。

台湾新闻网站pourquoi.tw援引何澄辉（Ho Cheng-hui）的话说，此前并没有媒体报道过该基地。他补充说，这可以被看作是习近平“迫切希望解决台湾问题并提高人民解放军战斗力”的一个计划。

何澄辉表示，该基地是对现有国际秩序的又一挑战，旨在破坏美国的第一岛链防御体系，“甚至直接挑战美军”。他说，该地点可以容纳许多军用飞机。

卫星图像基于一位自称是开源情报分析师在推特上通过账号@detresfa发布的推文。星期四（3月11日），他发布了两张卫星地图，称这些地图显示的是福建省漳浦县的一个“新直升机场”。

在左侧的黑白照片中，可以清楚地看到一个长方形的停机坪和跑道，他估计覆盖面积为1700米，还有一个行政大楼的区域。右侧的彩色特写照片似乎显示出10个直升机停机坪、一条跑道和至少三架已经停在停机坪上的直升机。他估计跑道长600米，有27个飞机机库。

[!\[\]()!\[\](https://spark.adobe.com/page/UNDuG4u9v5BJP/images/4ce141ec-6e16-459b-aa16-f94f5f73d6be.jpg?asset_id=b5b46136-318f-46c3-bbdd-16ce8c53e34a&amp;img_etag=%22ec8a7caeb4dfc5477b85fcd406212424%22&amp;size=2560)](https://spark.adobe.com/page/UNDuG4u9v5BJP/images/4ce141ec-6e16-459b-aa16-f94f5f73d6be.jpg?asset_id=b5b46136-318f-46c3-bbdd-16ce8c53e34a&amp;img_etag=%22ec8a7caeb4dfc5477b85fcd406212424%22&amp;size=1024)地图显示了福建漳浦县的新基地。（推特，@detresfa的图片）

开源情报分析师告诉《台湾新闻》，根据他的观察，该设施的工作于2019年7月开始，包括清理植被和土方搬运。他指出，鉴于其战略位置，它可以增强陆军和海军的作战能力，并使对台湾海峡的监视更加容易。

他补充说，尽管没有观察到无人机，但该基地也可以支持无人飞机（UAV）飞行。从跑道、停机坪和直升机停机坪已经铺好和涂漆的事实来看，他认为基地“接近完工”。

他的卫星图像来源包括欧洲航天局的哨兵2号（Sentinel-2）监测卫星、空客地球观测和谷歌地球（Google Earth）。在谷歌地球上，似乎有一个大型的建筑工地。

[!\[\]()!\[\](https://spark.adobe.com/page/UNDuG4u9v5BJP/images/0e86b7d9-a6c2-4940-a097-d664b75c935d.jpg?asset_id=0cb56402-32b6-4ae4-948c-ce0513ead407&amp;img_etag=%22a3c2dd3109bd8c2ce3bcdc910a5aeb76%22&amp;size=2560)](https://spark.adobe.com/page/UNDuG4u9v5BJP/images/0e86b7d9-a6c2-4940-a097-d664b75c935d.jpg?asset_id=0cb56402-32b6-4ae4-948c-ce0513ead407&amp;img_etag=%22a3c2dd3109bd8c2ce3bcdc910a5aeb76%22&amp;size=1024)土地清理工作已经开始，但飞机跑道尚不可见。（谷歌地球图片）

但是，由于谷歌地球上的卫星图像已有1-3年的历史，因此当前平台上可见的图像不一定反映地面的实际情况。哨兵2号的免费版本显示了飞机跑道、停机坪、直升机停机坪和行政区域的轮廓。

该位置的坐标为24°02’41.5″N 117°50’55.0″E。这里距澎湖约183公里，距台湾东沙群岛（普拉塔斯群岛）约386公里。

[!\[\]()!\[\](https://spark.adobe.com/page/UNDuG4u9v5BJP/images/83a06536-0355-4b90-b7f9-fb9dc0b59dc1.jpg?asset_id=384a3950-6610-4dce-acb0-6719d938f868&amp;img_etag=%226d6c97c04aa501324b776d30027a0fc7%22&amp;size=2560)](https://spark.adobe.com/page/UNDuG4u9v5BJP/images/83a06536-0355-4b90-b7f9-fb9dc0b59dc1.jpg?asset_id=384a3950-6610-4dce-acb0-6719d938f868&amp;img_etag=%226d6c97c04aa501324b776d30027a0fc7%22&amp;size=1024)从太空可见的空军基地轮廓。（哨兵2号图像）

军事专家何澄辉说，除了基地的直升机外，估计的跑道长度将足以使战斗机起飞并在台湾西南部和东沙群岛附近的水域和领空进行巡逻。他说：“所有这些都对台湾构成了威胁和压力。”
