1. 环球时报社论显示中共惧怕制裁将至。自从美国国务卿布林肯第三次确认中共在新疆犯下种族灭绝罪行之后，环球时报社论刊出文章提及美国及西方国家将采取的一系列行动，包括：各国制定政策、进行制裁，显示中共极度恐惧国际社会即将采取的行动。而中共的相应对策便是打舆论仗、打乱仗，并且试图用“强化治理”来混淆“种族灭绝”。在中共“完美犯罪”被揭穿之后，发动舆论战并且启动“科学论证造神”来神化习近平，企图以宣传中国几千年历史进程来洗脑中国人民接受极权统治，同时在国际上宣扬中国人本来就“习惯人治、不需要法治”的印象，希望西方社会不要干涉。除此之外，社论文章中变相承认目前美中对话渠道不通畅，沦落到要在布林肯访问东亚返回美国途中，在阿拉斯加安插会面，才有面谈的机会。
2. 布林肯首次在国会听证会认定中共犯下种族灭绝及反人类罪，在法律层面确认了中共的罪行。布林肯并且提及将有一系列实质、包括针对个人及实体、在外交签证及贸易等的制裁行动。布林肯在听证会上回避了拜登之前提出“Culture Norm 文化常态”的说法，显示中共种族灭绝的罪行已经落实，为布林肯与中共在阿拉斯加的会谈设下底线。
3. 中共宣传口狼狈回应路德社评论阿拉斯加会面地点的政治含义。日前路德社在节目中评论：中美不再有畅通对话渠道而导致中共迫于局势要追至阿拉斯加中途站与布林肯会面。中共宣传口又再一次紧急解画，举出中美“破冰之旅”选择会面地点的各种考量，尝试遮掩中共的困境。而布林肯向美国国会解释回程路过选择见杨洁篪并非战略性对话，只不过想看看是否能解决一些美方所关切的问题。从中方宣传口急切的回应，可以发现中方宣传媒体也是路德社忠实观众，被路德社精准的评论解析牵着鼻子走。
4. 彭佩澳督促布林肯在病毒追责上对中共不能让步。彭佩澳发文督促布林肯，有三个意思：一、提醒布林肯彭佩澳清楚知道内幕情报，二、民主党不可以无视病毒确实来自实验室，三、拜登政府必须向中共追责。川普总统团队继续向拜登政府施压，实际上拜登政府在灭共方向上也已经无法回头，必须继续遵循川普政府设下的方向前进。
5. 美国警告中方不可干涉达赖喇嘛转世。在新疆问题之后，西藏再成为人权焦点。中共试图在西藏利用扶持、控制下一任的喇嘛来控制西藏。达赖喇嘛转世不单只是宗教事件，它更加含有重要的战略意义，因为西藏土地面积广大，并且关系到全球有巨大信徒数目的佛教信仰，其重要性不亚于新疆和香港，并且西藏流亡政府在美国有一定的政治影响力。拜登政府此时提出西藏问题，不只可以避免新疆问题重演并且可以建立新政府政绩，因此美国将不会在西藏问题上妥协。
