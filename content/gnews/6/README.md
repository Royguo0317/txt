## 拍案惊奇短评系列之二十二——北京，是谁的北京

#### 作者：左媛


```
北京市政府近日昭告天下，从6月1日起，恢复《北京市居住证》《北京市居住登记卡》正常签注。
     
     笔者鄙陋之人，不知“签注”为何义。赶紧上网学习，才明白“签注”是印于港澳通行证上的证明通行证持有人可合法前往港澳的注释，有点像出国护照的签证。
     
     大陆人前往港澳溜一圈需要办通行证需要签注，这很好理解，毕竟“一国两制”嘛，不能像在国内走街串巷探亲访友一样随意，说走就走。何况港澳是资本主义的港澳，更不能让社会主义的大陆人受污染，有异议。
     
     但是，北京身处大陆，贵为一国之都，尤其前几年被自誉为“首善之都”，极享国人之歌颂赞美，国人热血沸腾激情澎湃，好像随时都可成为天子脚下一臣民。纯朴又有些愚昧的国人常常有莫名其妙的满足感自豪感，天安门前的欢歌乐舞即使是饿肚子时代也没少过，深山老林的穷人也常常对毛像磕头。
     
     有一段时间，吃饱了饭的人也想看看京城逛逛上海，很正常。倒也方便，火车飞机任你选，长安街上留个影，烽火台上拍个照，也不是难事儿。后来，国家越来越“强盛”，话语权延伸至欧美澳，外交部也越来越有“虎狼之色”，可是，北京却越来越拒绝城门外的中国人进入了，这倒是满腔热情的国人所意想不到的事儿。北京，社会主义祖国的首都，在老爷们的管制之下，国内地位竟等同于资本主义的港澳。外省的父母想去北京看看奋斗的儿女，外省工作的儿女想回北京探望年老的父母，竟然不能随便出入，必须办理居住证或登记卡，还得去签注。要花钱办证了，来去行踪要汇报给居委会了，你能给北京出劳动力至少6个月你就留下，否则哪来哪去，别痴心妄想！
    
      北京，是中国人的首都吗？不是！北京，是“首善之都”吗？不接纳全体中国人的地方，“善”在哪儿呢？奥运会时，《北京欢迎你》人人唱到处唱，现在才明白，它是唱给日本人韩国人的，唱给美国人英国人的，可不是唱给你湖南湖北人云南贵州人的。
     
      你能想象华盛顿DC要给DC以外的美国人办居住证吗？你能想象伦敦要给伦敦以外的英国人办居住卡吗？你能想象台北不允许台南高雄人自由出入台北吗……共产党的北京就是这么牛，外国人可以自由出入，但外省人不能居住，除非你能给它贡献劳动力6个月以上，还得给居住证办签注，二者缺一不可！
      
      北京，是共产党的北京，不是中国人的北京！
      北京，我要自由行！Take Down the CCP!
```
