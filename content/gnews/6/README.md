# **中共国宣传广播电台在英国因偏见和强迫认罪被罚款22.5万英镑**

[!\[\]()!\[\](https://spark.adobe.com/page/IpbLiVANsdH71/images/d967e9d5-4349-45df-95c6-f21db5079d4a.jpg?asset_id=dc9e995b-52e2-49d3-aa8d-e10539869d9d&amp;img_etag=%22dab33fbc0848c61b4ab6b0b213339c4f%22&amp;size=2560)](https://spark.adobe.com/page/IpbLiVANsdH71/images/d967e9d5-4349-45df-95c6-f21db5079d4a.jpg?asset_id=dc9e995b-52e2-49d3-aa8d-e10539869d9d&amp;img_etag=%22dab33fbc0848c61b4ab6b0b213339c4f%22&amp;size=1024)

**中共国国有广播公司中国环球电视网（CGTN）被英国媒体监管机构处以22.5万英镑的罚款，原因是该公司播出香港民主抗议运动时带有明显偏见，并在其频道中播出了涉嫌强迫认罪的节目。**

周一，英国通讯管理局（Ofcom）宣布将对CGTN处以12.5万英镑的罚款，原因是该公司在2019年有关香港自由运动的五次独立直播中，未能遵守英国“应有的公正”规则。

据《每日电讯报》报道，在2013年和2014年的两次报道中，中共国的宣传媒体违反隐私和公正规则，被另处10万英镑罚款，这两次报道有关英国公民彼得•汉弗莱（Peter Humphrey）在上海被捕。

汉弗莱先生向Ofcom投诉，称CGTN播出了他所谓的电视认罪，他声称自己当时是被强迫的。

在获释出狱后，他说：“我在监狱中不断受到骚扰，要求签署一个被他们称为认罪和悔过声明的文件。我从未签署过那些文件，因为我不承认自己犯有被指控的罪行。”

汉弗莱在2013年对制药公司葛兰素史克（GlaxoSmithKline）在中共国的交易进行调查时被判入狱。他声称，他实质上是中共与英国公司之间的“更大范围争端中的附带受害者”。

英国通讯管理局的罚款遭到了中共国外交部发言人赵立坚的批评——尽管色情网站和西方社交媒体网站均在中共国被禁止，但据布赖特巴特伦敦最近透露，赵立坚一直在推特上关注露骨的色情帐户。

这个中共高层“战狼”外交官说，中共国保留对英国通讯管理局的裁决作出“必要反制”的权利。在CGTN的执照被取消后的报复性反击中，北京当局封杀了BBC国际频道节目在国内的播出。

赵立坚继续说：“汉弗莱应该悔改并重新开始，而不是提出毫无根据的指责。这样做不仅不会抹去他的不法行为，还将让他受到道德审判。”

[!\[\]()!\[\](https://spark.adobe.com/page/IpbLiVANsdH71/images/976e24f2-1163-4cdc-8a09-e1b2c0974e36.png?asset_id=62974a5f-204d-4974-8aab-e91f3d0a472a&amp;img_etag=%22fce93f7be5bcc58a521d334e6d8fd659%22&amp;size=2719)](https://spark.adobe.com/page/IpbLiVANsdH71/images/976e24f2-1163-4cdc-8a09-e1b2c0974e36.png?asset_id=62974a5f-204d-4974-8aab-e91f3d0a472a&amp;img_etag=%22fce93f7be5bcc58a521d334e6d8fd659%22&amp;size=1024)

罚款是在英国通讯管理局吊销CGTN在英国运营的许可证后一个月提出的，英国通讯管理局称该网络“最终由中国共产党控制”，违反了英国的广播法。

去年3月，川普政府指定CGTN以及其他四个中共国官方媒体为“外国使团”，并指出宣传媒体不是在做提供真实新闻事实，而是在执行共产党路线 。

英国通讯管理局周一还宣布，它发现星华传媒公司（Star China Media）——CGTN通过该公司获得英国广播许可证——在另外两个据称已播出的强迫认罪的报道中，违反了公平和隐私协议。

前英国驻香港领事馆工作人员郑文杰（Simon Cheng）声称，他在2019年被中共国独裁政府绑架并折磨15天，他向英国通讯管理局投诉，指控共产党的宣传媒体播出了他被强迫认罪的视频。

去年2月，郑在接受伦敦布赖特巴特采访时说：“这是一次非常残酷的经历，我经历了一轮又一轮的审问，询问了我在[香港]抗议活动中的角色以及英国在抗议活动中的角色。”

英国广播监管机构也支持了代表香港书商桂民海提起的一项申诉。他去年因出售批评中共领导人的书籍而被判十年监禁，罪名是“非法向海外提供情报”。这位瑞典公民的女儿以桂先生的名义提起申诉，声称CGTN在2016年和2018年播出的电视认罪是强迫性的。

英国通讯管理局发言人说：“我们发现有关个人（郑先生和桂先生）受到了不公平的对待，并且他们的隐私遭到了无端侵犯。此外，CGTN还未能获得被采访人的知情同意。”

这位发言人补充说：“此外，一些令人严重怀疑他们所谓认罪可靠性的重要事实被排除在节目外。”
