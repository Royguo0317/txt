# **数百万学生没有他们需要的笔记本电脑：报告显示，三分之一的低收入家庭没有在家上网课的设备**

- Pupils have been told to use devices to learn at home while schools are closed
- Has opened ‘digital divide’ between middle classes and low-income pupils
- Sutton Trust found that 35% of families are lacking sufficient access to devices
- 学生被告知在学校停课时在家使用网络设备上学。
- 已经打开了中产阶级和低收入学生之间的“数字鸿沟”。
- 萨顿信托基金会（Sutton Trust）发现，35%的家庭缺乏足够的可用设备。


今天一份令人震惊的报告显示，超过三分之一的低收入家庭没有远程教育所需的笔记本电脑或或平板电脑。

学生们被告知在学校对大多数儿童关闭的情况下，可以使用这些设备在家学习。

这就在中产阶级的孩子和来自低收入家庭的学生之间打开了一条“数字鸿沟”，后者在没有电脑的情况下苦苦挣扎。

社会流动慈善机构萨顿信托基金会(Sutton Trust)的一项调查显示分裂的程度很明显，有35%的家庭仍然缺乏足够的设备，相比之下，高收入家庭只有11%。

**今天一份令人震惊的报告显示超过三分之一的低收入家庭没有远程教育所需的笔记本电脑或平板电脑。**

由于政府在提供笔记本电脑和平板电脑方面进展缓慢，使问题更加复杂。迄今只分发了承诺的130万台电脑中的80万台。

据英国通讯管理局（Ofcom）估计，有多达178万儿童无法使用电脑。

他们的统计没有考虑到更多的与兄弟姐妹和父母共享设备的情况。

更糟糕的是，调查显示，国立学校的老师说，超过五分之一的学生完全没有电脑可用。

这可能是因为因疫情而失业的父母不得不把电脑还给雇主、用电脑来找工作或者他们可能支付互联网费用有困难。

四分之三的中学校长表示，他们在等待政府提供时，不得不尝试采购笔记本电脑。

尽管他们做出了努力，但只有10%的教师报告说，他们所有的学生都能充分获得远程学习的设备。

而只有5%的国立学校教师表示，他们所有的学生都能上网。

对孩子们教育的影响已经很明显了。

根据调查，40%的中产阶级家庭的孩子每天学习时间超过5小时，而工薪阶层家庭的孩子只有26%。

萨顿信托基金会的创始人和主席彼得•兰普尔爵士（Sir Peter Lampl）说，失去学习的影响是“毁灭性的，并将在未来几年内感受到”。

他补充说：“当务之急必须是解决贫富之间的数字产品供应差距。”

该信托基金建议，笔记本电脑和互联网接入“应继续通过政府方案快速推广”。

报告补充说，“每天都有学生缺乏在线学习工具，这都扩大了差距，并损害年轻人的长期前景。”

该慈善机构还呼吁一次性提供7.5亿英镑的学生补助金，让学校为每个符合条件的学生提供400英镑的额外支出，以帮助他们迎头赶上。

** 据英国通讯管理局估计，有178万儿童没有电脑使用。**

在调查中，大多数学校的工作人员认为，更快地推广笔记本电脑是对较贫穷的学生最有帮助的一项干预措施。

调查结果是基于本月对6475名教师和877名家长进行的调查。

教育部（Department for Education）发言人说：“我们意识到弱势儿童在这场危机中面临的额外挑战，这就是为什么我们要提供130万台笔记本电脑和平板电脑，同时提供免费的移动数据使用。”
