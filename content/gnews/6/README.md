## 【健身健心】健身前后的皮肤护理
` 纽约香草山MOS03` [轉載自GNews](https://gnews.org/zh-hans/1172895/)

作者：纽约香草山健身部  周一



![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/03045249/LOGO3_1-1.jpg)



健身可以让皮肤变得紧致有弹性，这是众所周知的。那么，是不是大家只需健身而不用护理皮肤了呢？非也！健身房中有一张烂脸的人也很常见。因此，笔者在这里就分享一下健身前后皮肤护理的方法。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/04230953/image-56.png)图片来源：fourseasons.com

1. 健身前要清洁好皮肤。洗脸后，大家只需要用一些爽肤水和质地清爽的乳液即可。健身的过程中会出很多汗，需要通过皮肤毛孔去散热，此时如果脸上有厚厚的妆，不但影响散热，而且汗水和脸上的化妆品会产生反应，造成皮肤过敏，或者毛孔阻塞。久而久之，脸上会形成粉刺或者黑头、白头。所以，健身前一定清洁好皮肤。
2. 健身过程中及时为你的肌肤补充水分。当你运动时，你的皮肤会失去很多水分，所以在运动中，每次停下来休息时要多喝水，及时补充体内的水分流失。如果皮肤大量的出汗，可用柔软的毛巾轻轻的按压吸走汗水，切忌用力抹搽脸部，那样会导致皮肤敏感。
3. 户外运动一定要做好防晒。紫外线是皮肤衰老的杀手，如果你进行的是户外运动或锻炼，防晒霜是必不可少的。而且，即使是阴天你也需要涂抹防晒霜，最好选择防水防汗、质地较为轻薄、SPF50左右的防晒霜。如果进行长时间的户外运动，注意需每个两个小时左右补充涂抹一次防晒霜。另外，户外运动时带上帽子可以减少紫外线对皮肤的伤害。
4. 健身后及时洗澡，清洁皮肤。由于健身过程中皮肤的毛孔为了散热而张开，所以这时太热太凉的水温都不利于皮肤。因此，用温水冲凉后给身体涂上身体乳，会让肌肤变得更加健康和光滑细腻。此时，大家可在脸部涂抹一层精华或者敷一张补水面膜，你会收到意想不到的效果。


运动本身是一个好习惯，但也要注意运动前后的皮肤护理，才可以越动越美丽！
