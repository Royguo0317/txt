# **报告：英国顶尖科学家帮助中共国发展核武器技术**

[!\[\]()!\[\](https://spark.adobe.com/page/jt6YCXFni8yo2/images/1781162a-0aad-4151-b8c0-e5f8a35d2932.jpg?asset_id=437f7a5f-0905-4d57-88bb-f9a0d1264c68&amp;img_etag=%22d3a55ddba5027e7c402652e2fb2d746f%22&amp;size=2560)](https://spark.adobe.com/page/jt6YCXFni8yo2/images/1781162a-0aad-4151-b8c0-e5f8a35d2932.jpg?asset_id=437f7a5f-0905-4d57-88bb-f9a0d1264c68&amp;img_etag=%22d3a55ddba5027e7c402652e2fb2d746f%22&amp;size=1024)

**一项调查称，至少33所英国大学的科学家与中共国军方研究人员合作开发了核武器。**

据报道，剑桥大学、爱丁堡大学和曼彻斯特大学等大学内的顶尖学者，曾与中共国顶级核武器研究机构中国工程物理研究院（简称中物院或CAEP）的研究人员一起工作。

据《每日电讯报》报道，至少有33所英国大学的学者与中物院或其附属机构的研究人员一起发表论文。据信，一些英国科学家甚至在中共国机构任职。

中物院及下属机构被美国列入了制裁名单。这些机构帮助发展了共产主义政权的核武库和先进武器技术。这些机构也都受到了中共国独裁者习近平主持的中央军事委员会的监督。

报告还称，英国纳税人实际上可能在资助北京的武器计划，因为英国和中共国研究人员之间的许多联合项目都利用了由英国政府的拨款和纳税人资助建造的设施。

其中包括对英国国家超级计算机（名为ARCHER），以及耗资2.6亿英镑的粒子加速器钻“石光源”(Diamond Light Source)的使用。

报告称，爱丁堡大学、伦敦玛丽皇后大学和剑桥大学的学者都与研究“冲击波和爆轰物理”的中物院实验室有所关联。

澳大利亚战略政策研究所称，中物院的四大目标是“核武器、研究用于核聚变点火和定向能武器的微波和激光、研究常规武器相关技术、深化军民融合”。

该国防智库还称，中物院正在寻求扩大“其国际影响力，以吸引领先人才协助中共国发展核武器。”

[!\[\]()!\[\](https://spark.adobe.com/page/jt6YCXFni8yo2/images/48f82872-6d67-4fc5-889d-2f65e988a17d.png?asset_id=aa2513fb-4c51-486f-93c4-cd6f162507ac&amp;img_etag=%22d4b117d9fd82364ba4106190c7537a4e%22&amp;size=2689)](https://spark.adobe.com/page/jt6YCXFni8yo2/images/48f82872-6d67-4fc5-889d-2f65e988a17d.png?asset_id=aa2513fb-4c51-486f-93c4-cd6f162507ac&amp;img_etag=%22d4b117d9fd82364ba4106190c7537a4e%22&amp;size=1024)

所有这些大学均表示，他们的研究符合英国法律，并且这些项目并无军事用途。

但是，前外交部官员查尔斯•帕顿(Charles Parton)表示：“与一家研发核武器的研究所合作的想法糟糕到了极点。”

下议院外交事务委员会主席托马斯•图根达特(Tom Tugendhat)表示：“一些大学对合作伙伴显然缺乏好奇心，这让他们面临与敌对国家勾结、侵犯人权、以及破坏英国安全的指控。”

“显然，他们需要意识到自己的责任，但政府应在确保他们意识到这些合作关系所带来的后果方面发挥必要作用。”

英国大学因与中共的关系而受到越来越多的审查。据报道，军情六处(MI6)正在调查“英国内一些最富盛名的大学”，原因是他们与中共国武器开发商的合作可能违反了国家安全法。

据报道，约200名英国学者也因涉嫌与中共政权分享军事技术而受到调查。

[!\[\]()!\[\](https://spark.adobe.com/page/jt6YCXFni8yo2/images/3fbb555c-f71f-410a-8cc8-c00713922d4e.png?asset_id=9fdd0e97-b2da-403d-af72-cf5b6f752a05&amp;img_etag=%221813f17ba2e83fab039e3c5e82f1b049%22&amp;size=2800)](https://spark.adobe.com/page/jt6YCXFni8yo2/images/3fbb555c-f71f-410a-8cc8-c00713922d4e.png?asset_id=9fdd0e97-b2da-403d-af72-cf5b6f752a05&amp;img_etag=%221813f17ba2e83fab039e3c5e82f1b049%22&amp;size=1024)

在最近的调查中，被指控与中物院有关的教授之一是现居新加坡的剑桥大学地球科学系前系主任西蒙•雷德芬教授(Prof Simon Redfern)。

2016年，雷德芬在被称为“北京高压科学研究中心（HPSTAR）”的中物院子公司担任客座教授，随后与中共国研究人员一起发表了15篇论文。同时，他在两个英国政府科学机构任职，并接受公共资金支持他的工作。

雷德芬教授还获得了中共国“千人计划”的奖励。而该计划与针对全球，特别是美国的敏感技术盗窃相关。

例如，去年，前哈佛大学教授查尔斯•利伯(Charles Lieber) 被指控在“千人计划”中未披露他与武汉理工大学的财务关系。

智囊团Civitas关于英国大学与中共国政府间关系报告的作者拉多米尔•泰勒科特(RadomirTylecote)将最新的调查结果描述为“重大的国家丑闻”，并继续呼吁英国政府建立一项美国式的制裁名单，以禁止这种合作伙伴关系在未来继续存在。
