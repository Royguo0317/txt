# 亚利桑那州成为第18个州，支持德克萨斯州对密歇根州、宾夕法尼亚州、乔治亚州和威斯康星州提起欺诈性选举诉讼

![]()![](https://spark.adobe.com/page/DLIvbTQaJbpo0/images/f75dae3b-d2e3-470b-bfed-4a3912904705.jpg?asset_id=5dbb3b97-e465-463f-a1dc-2f76106012b9&amp;img_etag=%222681b296c78ac06280e91cc05a029656%22&amp;size=2560)

亚利桑那州周三加入了其他18个州，支持德克萨斯州对密歇根州、宾夕法尼亚州、佐治亚州和威斯康星州就欺诈性选举提起的诉讼。

目前，该名单已增至18个州。

密苏里州、阿拉巴马州、阿肯色州、亚利桑那州、佛罗里达州、内布拉斯加州、北达科他州、俄克拉荷马州、印第安纳州、堪萨斯州、路易斯安那州、密西西比州、蒙大拿州、南卡罗莱纳州、南达科他州、田纳西州、犹他州和西弗吉尼亚州。

德州周一晚间向美国最高法院起诉佐治亚州、密歇根州、宾夕法尼亚州和威斯康星州，质疑其选举程序不合法。

德州认为，这四个州违反了美国宪法，因为它们通过法院或通过行政行为对投票规则和程序进行了修改。 但这些州并没有按照美国宪法的规定，通过州立法机构进行修改。

亚利桑那州的欺诈行为，特别是在使用多猫腻（Dominion）投票机的马里科帕郡，是令人震惊的，但司法部长马克-布诺维奇（Mark Brnovich）提交了一份“法庭之友”陈述，支持德克萨斯州。

布诺维奇自豪地说，亚利桑那州没有被列为德州诉讼的被告之一。

“我相信亚利桑那州没有在这场诉讼中成为被告，因为我们的办公室成功地阻止了许多同样令人不安的最后一刻对我们州选举诚信法律的改变，”他说。

“我们的法律文件确保亚利桑那州的利益得到保护，我期待最高法院解决这些国家问题。”

（马克-布诺维奇推文翻译：我们办公室收到很多关于德州向最高法院起诉宾夕法尼亚州的案件，亚利桑那州是否会加入等问题的询问，这是我的声明和我们的法律文件）

亚利桑那州的选举舞弊到底有多严重？

根据Statista的最新汇总结果，亚利桑那州目前的投票率为79.9%，是全美最高的。这也是该州历史上参加投票人数最多的一次。

2020年11月16日，亚利桑那州报告的选民投票率为65.9%–现在2020年11月30日，州务卿推荐的选举结果网站 “亚利桑那投票 “报告的选民投票率为79.9%!

亚利桑那州众议员马克-芬奇姆（Mark Finchem）上周五呼吁他的立法者同僚们一起撤回亚利桑那州的选举认证。

芬奇姆说，现在郡文书们都站出来承认拉票有问题，数字就是对不上。

“我们有匿名证人，”芬奇姆说，并补充说，立法机构将绕过州长杜西。

“我们只需要众议院的31名议员和参议院的16名议员通过一项决议，召回我们的选举人。就这么简单，”他说。”我们的选民为此炸开了锅。”
