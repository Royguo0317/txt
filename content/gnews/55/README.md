04月13日，美国应在乌克兰境内部署“爱国者”防空导弹系统（SAM）。据《时代》杂志报道，乌克兰总统办公室负责人安德里·叶尔马克（Andriy Yermak）这样说。

“乌克兰对俄罗斯的防御不仅是为了自己，也是为了西方。美国将爱国者导弹部署在哪里呢？最近的是在波兰。他们应该在这里。”他说。

4月，华盛顿承诺，如果俄罗斯方面的“军事侵略”升级，将支持基辅。3月，美国国务卿安东尼·布林肯表示，美国将加大对乌克兰的支持力度，包括在安全和经济领域。

美国当局还要求莫斯科就俄罗斯军队集结到乌克兰边境一事作出解释。美国国务院发言人内德·普莱斯称这种行动是对乌克兰的“恐吓”。

3月底，《纽约时报》报道称，4000名俄罗斯军人被部署到乌克兰边境，包括罗斯托夫、布良斯克、沃罗涅日地区和克里米亚。俄罗斯国防部解释说，向克里米亚重新部署部队与南方军区计划的训练活动有关。克里姆林宫表示，其他国家不应该关注俄罗斯军队在该国范围内的行动。
