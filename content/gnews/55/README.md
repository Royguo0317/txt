**简评：**

澳大利亚是较早地公开要求对中共病毒起源进行调查的国家之一。然而，这个合理的要求却招致了中共在政治、经济上的疯狂报复。从去年年底开始，中共不顾自己是WTO成员，以及双方在2015年签订的《中澳自由贸易协定》，单方面、非公开地对澳大利亚的铁矿、煤、红酒和其他农产品进行了全面的制裁。

但是，经过了接近6个月的各种制裁之后，各种贸易数据和澳大利亚的经济指数表明，中共对澳大利亚产品的制裁并未对澳大利亚产生影响。最主要的原因就是澳大利亚的这些产品都是国际市场上的热销产品，并不担心没有其他的国际买家来采购。与此同时，国际市场上铁矿石的价格在过去的12个月中涨幅超过了一倍，完全抵消了中共的制裁和中共的病毒给澳大利亚带来的影响。根据德勤（Deloitte Access Economics）4月的经济报告：“澳大利亚失业率已经降低到5.8%，铁矿石、液化天然气和动力煤价格上涨提振了澳大利亚经济。”

相反，由于中共禁止优质的澳大利亚煤进口，造成了国内因煤价上涨，多个省市在12月供暖高峰时期不得不限制工业用电和民用供暖，造成了巨大的社会影响。

就铁矿石和煤炭这两种商品来说，它们都是中共国必须进口的产品。由于澳大利亚煤炭和铁矿石的高品质以及与中国天然的距离优势，从澳大利亚进口这两种商品对于中共国来说是最佳的采购渠道。中共即使不从澳大利亚进口，也必须从其他的国家进口这两种商品才可以维持国内的生产和生活必须。但这样就会付出更高的运输成本和商品价格，而这些高出的成本只能由国内百姓承担。如本文对中共禁止澳大利亚煤炭的总结：“从其他国家增加进口看上去对降低国内价格没有起什么作用，意味着中国用户仍然支付着比禁止澳大利亚煤炭前高得多的价格。”

所以，中共对澳大利亚的制裁，满足的只是中共领导人的阴暗心理、外交部战狼们的口舌和小粉红们的耳朵。搬起“制裁”这块石头，真正砸到的只是国内的生产企业和普通百姓们的生计。

中共国正为其对澳大利亚煤炭的禁令付出高昂代价

朗斯顿,澳大利亚，4月9日（路透社）——由于非正式禁止澳大利亚煤炭进口，导致动力煤和焦煤在中共国内和替代澳大利亚的外国供应商的价格都在飙升，中共国为此付出了高昂的代价，

中共国是世界上最大的煤炭进口国、生产国和消费国，实际上已经终止了从澳大利亚进口煤炭。作为两国持续不断的政治争端的一部分，澳大利亚是用于炼钢的焦煤的最大出口国，以及用于发电的动力煤的第二大出口国。

根据Refinitiv的船只跟踪和港口数据,中共国对澳大利亚煤的进口限制是从去年下半年开始的，导致进口的澳大利亚煤从2020年6月的946万吨的高点下降到今年头两个月基本为零。

但是，中共国的进口煤消费者却必须承担更高的费用，来自本地和外国替代供应商的价格随着市场的非正式的禁令而调高。

自从实施禁令以来，除了北半球冬季通常的季节性增长外，澳大利亚炼焦煤离岸价格一直在走弱。

新加坡交易所对澳大利亚炼焦煤的合同价格周四以每吨113.71美元的价格收市，比去年10月初，中共国的禁令开始生效时的140美元下降了18.8%。

如果一个中共国进口商从澳大利亚煤转向采购美国煤，那么价格的变动正好相反。

美国东部港口Hampton Roads的焦煤离岸价格从10月初的每吨114美元升到了周四的152.75美元，涨幅34%。

这意味着现在美国的炼焦煤比澳大利亚的每吨贵39美元，这还没有算上由于距离更远，运到中共国要花费更高的运输成本。

自从对澳大利亚实施禁令以来，中共国内炼焦煤的价格也在上涨，大连商品交易所期货价格周四从10月初的1353元/吨上升了16%，达到1573元/吨。

这个价格因为包含了运费和进口关税等因素，无法和澳大利亚、美国的离岸价格直接比较。

但是这确实显示中共国国内的煤价被推高了，也部分反映出来自其他国家的进口煤价格更高了。

邻国蒙古成为了中共国炼焦煤的最大供应地，今年头两个月从蒙古国进口的炼焦煤占总进口的61.7%，而2020年同期仅为17.7%。

从澳大利亚的进口比例从2020年1-2月的68.4%降到了今年的零，而美国的比例从不到2%上升到9.1%，加拿大从6.1%增加到12.1%。

尽管来自蒙古国的焦煤比从海上运来的进口煤便宜，但蒙古煤的价格会跟随国内的价格，意味着它们可能也大幅涨价了，特别是考虑到运输以及洗煤成本后。

国内动力煤之痛

就动力煤而言，禁运澳大利亚货物的主要影响似乎是中共国国内价格走强。据SteelHome评估，秦皇岛基准煤&lt;SH-QHA-TRMCOAL&gt;在周四收于每吨747元，约为114美元。

虽然比冬季最高价1038元/吨有所下降，但仍比10月初的612元/吨高出22％。

据信中共国当局希望动力煤的价格保持在530-580元之间，据说这个价格水平能保证煤矿的盈利并同时让电价具有竞争性。

中共国转向了世界最大的动力煤出口国印尼以填补不能进口澳大利亚煤的空缺，并且从俄国和南非购买更多煤，这两个的国家能提供和澳大利亚煤质量类似的动力煤。

但是从这些国家增加进口看上去对降低国内价格没有起到什么作用，意味着中共国用户仍然支付着比禁止澳大利亚煤炭前高得多的价格。
