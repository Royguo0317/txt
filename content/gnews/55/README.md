# **冠状病毒警告：新毒株“传染性比武汉变种病毒高八倍”**

**一项研究表明，一种新的冠状病毒株的传染性是武汉发现的原始变异病毒株的八倍。**

根据发表在eLife期刊上的一份报告，被称为D614G的变种病毒被认为是世界上目前最主要的变种病毒。这种强大的变种病毒的传播能力比一年前在中国武汉发出警报的病毒的传播能力高八倍。

该研究的作者声称，新变种病毒已经“迅速传播”并导致“死亡人数增加和更高的病毒载量”。

研究员内维尔•桑贾纳（Neville Sanjana）博士说，感染的迅速增加可能与更高的传播率有关。

他说：“自我们最初进行这项研究以来的几个月中，D614G突变的可怕性日益增高：该突变已接近引发大流行，并已成为当前备受关注的变种病毒之一。”

“可以确认这种变种病毒导致更高的传播性，可能在一定程度上解释了为什么病毒在过去的一年中如此迅速地传播。”

研究人员说，D614G变种病毒比原始病毒“更有效”地感染人体细胞，近几个月出现在英国、南非和巴西。

这项研究报告是在物理学家声称病原体可能是从实验室传播给人类之后发表的。

汉堡大学的罗兰•维森丹格博士发表了长达100页的研究报告来证明他的病原体可能是从实验室传播给人类的观点。

世界卫生组织（WHO）的科学家坚持认为该理论“极不可能”。

中共国政府拒绝接受冠状病毒可能源于实验室的类似理论。

然而，维森丹格博士援引实验室周边的安全问题，中共当局未能找到该病毒的动物宿主以及中共在实验室进行的“人体功能增强”研究的事实来佐证他的理论。

维森丹格博士在接受德国ZDF报纸采访时说：“我确信99.9％的可能性冠状病毒是来自实验室。”

维森丹格博士承认他的研究集中在“间接证据”上，他强调尚未发现冠状病毒的天然宿主。

研究人员研究推测，在蝙蝠中发现的名为RaTG13的冠状病毒样本被带到武汉的实验室，其功能被加强后从实验室流出。

他还列举了武汉研究所之前存在的安全隐患以及该病原体专门攻击人类细胞的事实。
