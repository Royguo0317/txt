![]()![](https://gnews.org/wp-content/uploads/2021/02/2-5-5.jpg)
正如前文所言，新加坡守着国际航运重要的战略要冲和节点枢纽——马六甲海峡，想要保持自己的独立地位，不被强国所胁迫，一只强有力的国防力量是必不可少的。

1965年8月9日，新加坡从马来西亚联邦独立，成立新加坡共和国，武装部队也同时从马来西亚军队中分离出来，单独组建新的新加坡军队。
![]()![](https://gnews.org/wp-content/uploads/2021/02/image3-1.png)
1967年，新加坡军队开始施行国民服役政策（英语：National service，简称NS），首批士兵和军官由以色列负责培训。到1968年，新加坡军队的陆海空三军全部形成完毕，1970年原本只有两个营的陆军被迅速扩大到两个旅，国防现代化开始逐渐成型。

为了在政治环境复杂的东南亚求得生存机会，李光耀提出了著名的”毒虾理论”，即将自己比喻为”能产生剧毒的小虾”，这样既可以与”鱼群”一同共舞，又不会被”大鱼”吞食，让国家相对处于安全稳定的境地。作为重中之重的国防安全，就成了新加坡政府最为重视的方面。

有鉴于此，新加坡政府决定从两方面发展自身的国防安全。首先依托比较发达的航运和出口，积累大量国家基金，用作国防的建设；其次发展本国的军工企业，并与欧美各国寻求军事援助和合作，提升自身军事实力的现代化和机械化发展，对周边国家起到一定的震慑作用。在这样的军事扶持之下，新加坡的军事实力有了巨大的提升，其武器现代化程度也逐渐成为东南亚的第一名，各种海外维和任务和军事演习中，新加坡军队也成为了常客。

陆战方面，新加坡向德国购买了”豹2A4″型主战坦克，经过改装后的”豹2A4″在城市作战中，防御力有了明显的加强，车前方和四周都使用了最新的AMAP复合型装甲，使得驾驶员和车内乘员的生存性显著地提高；使用新式火控系统，使得攻击时的火力打击能力有了加强。

而在空军方面，2020年1月，美国国务院初步批准向新加坡出售多达12架具备短程起飞和垂直降落功能的F-35B型战机及相关设备；扩大新加坡的现役军机数量，增强空对空和空对地自卫能力，加强新加坡在保卫边界方面的有效威慑力，以及在联合行动中发挥作用。
![]()![](https://gnews.org/wp-content/uploads/2021/02/image4-7.jpg)
2020年5月，美军吉福兹号濒海战斗舰（US Gabrielle Giffords-LCS 10）和新加坡海军的坚定号RSS Steadfast（FFS 70）在南海进行了为期2天的军事演习，新加坡在大国军事博弈中也逐渐发挥着作用。

通过海外军事装备购买和本国军工的助力，新加坡军队成功地将自己打造成了一支现代化、科技化双重结合的新型武装力量。并成功地将”毒虾理论”进一步地奉行下去，成为东南亚的一个不可忽视的存在。

新加坡在军事、航运等多重领域的重要性，也让CCP觊觎已久，想要拿下新加坡之心路人皆知，通过多方面手段，逐渐渗透新加坡的高层。详细情况，将在下篇文章中与大家进行详细探讨。
