## 杂谈中共独裁下的医疗体系腐败
` 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hans/960370/)

作者：Ray裁决

前几天看到GNEWS一篇文章《【中共天价医疗之恶】70万一针救命药，刷光三代人的银行卡》有感，杂谈在中共独裁体制下的医疗体系从业人员的各种腐败和利益链，来扒一下这个丑恶的行业，是如何在不停的吸着14亿百姓的血。

首先谈下这个行业主要围绕中共体制下的医院这个主体来运转，其中要牵涉到的各个主体用简易图标来表示如下：

![]()![](https://gnews.org/wp-content/uploads/2021/03/030901.png)

最早的药品带金销售是在90年代江苏一家药厂最先开始的。这个先河一开，引来众多药企的纷纷效仿，以至于经过这几十年下来都成为行业的不成文的规定，成为医生收入的一部分，这部分收入比正常的工资都多。以至于现在的医生都有专门的计算公式来计算某种药品的回扣是否达到标准，低了还是高了，高了就多开，低了就少开甚至于不开。时间长了就有了这么一套腐败流程，下面来说说这个大概的流程，我们就可以看到其中的一些腐败利益链。下面给出大致一个流程：

![]()![](https://gnews.org/wp-content/uploads/2021/03/030902.png)

中间的每个环节都存在腐败和利益链的输送，所以看到中共治下的奇葩现象，普通百姓看个病连药都吃不起，更别说那些被垄断的进口药。更有甚者，药代为了药品销量喊医生过度用药，有的医生为了钱拼命开药的奇葩现象。更有一些领导，医院院长的亲戚在里面充当白手套，垄断一家甚至几家医院的经营，从中谋取巨大利益。这还是下面的情况，再往上去，其中的腐败真不是普通老百姓所能想象的。

卫生医疗系统是一个国家的基本保障，被中共制造成了一个巨大的腐败产业，这几十年经济发展药厂还靠仿制药，仿造医疗器械来过日子，处处是贿赂腐败。中共不灭，这个体制不灭，老百姓卖房看病，看不起病的事件不会停止。
