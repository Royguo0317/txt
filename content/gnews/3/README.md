昨夜的月亮超级亮，引来农历四月十六日（公历5月27日）清晨的晴空万里。

旭日阳光普照下的古城，人们又延续了白天的忙忙碌碌。笔者要匆匆赶去参加一场出殡活动。朋友的母亲逝去了，定在上午8：30分去火葬场。行到殡仪服务馆门口，迎面一行人缓缓走出来。人群前面的乐队奏着哀乐，抱着亲人遗像的晚辈紧随其后，帮忙的亲友拿着不同的花篮一步一跟。人群中撕心裂肺的哭声，揪着每一个在世者的心。因为这是人世间的最后一次亲人告别，灵柩中冰冷的逝者即将熔化入熊熊烈火中。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/27123335/2020120215564382.jpg)

进到院内，瞧了一眼墙上的黑色显示屏。不同姓名、不同年龄、不同单位的逝者名单，几乎满屏定格。按照往常情况，秋冬两季逝者才多，现在快到收割麦子的节气了，出现这样的情况有点不寻常！往里走，仿佛进到了一个热闹的集市，每一个牌号门前，都堆积着花篮，绽放的鲜花渲烂夺目，与白色的孝衣形成强烈反差。不同年龄的男女吆喝着，行走着。

来到朋友母亲的灵堂外，遇见提前到来的朋友，聚在一起开始聊天。有一个朋友拿着手机正在观看印度疫情的视频，笔者谈道，印度疫情大爆发的时候，也有很多是接种过疫苗的人群，在这一次疫情中也不能幸免。而且现在媒体舆论都在谈这是一个生物战，这是战争，不同领域的超限战！

你们接种疫苗了吗？很不幸的是，基层体制内朋友在一个月前，就强制接种过两剂了，公共交通领域和国有大型企业的朋友，也被强制完成了接种。还有一个很熟悉的朋友，以前听过笔者劝说，没有打疫苗，但没有劝住自己的孩子，孩子的单位也是强制每个人接种。听了心里真不是滋味，只能告慰他们，既然接种了，就保持一个良好的心态，注意一下身体变化，备用硫酸羟氯喹。文贵先生的最新盖特讲道：“刚刚出来的科学家告诫人们，千万不要打疫苗，领导人没有一个打的。”笔者问朋友，你们领导都打了吗？回答有特殊情况的不多，基本上全打了。笔者告诉了朋友，省以上领导在2019年12月份，内部就配发了使用硫酸羟氯喹。现在的免费接种，告知书有一条是“您所接种的疫苗可能不会产生100%的效果”，相信大多数人不会仔细去看，即使看了也想着不会发生在自己身上。愚昧和无知吗？有的是，但绝大部分是信息来源被控制，看不到听不到中共病毒和疫苗背后的真相。

一个人发出的声音虽然是很微弱的，暂时影响不了大局。但正义的火种在我们手中，让星星之火点亮他们的心灵，划破这疯狂而黑暗的世界吧！
