06月01日，统一俄罗斯主席和安理会副主席德米特里·梅德韦杰夫在接受《生意人报》采访时说，俄罗斯已经厌倦了与西方的对抗。

“不仅仅是个人对对抗感到厌倦。让我再告诉你，我们都对对抗感到厌倦。那么谁喜欢呢？没有人喜欢它。我也不喜欢它。我确信总统不喜欢这样，我的其他同事也不喜欢。”梅德韦杰夫说。

他补充说，发起“螺旋式对抗”的并不是莫斯科。在梅德韦杰夫看来，西方“本可以用更微妙、更巧妙的方式行事”。特别是在克里米亚被俄罗斯吞并的情况下，西方国家本可以在不破坏与莫斯科关系的情况下表达“他们的一些关切”。

“但他们毁了一切，烧成了一片废墟。我不想说每件事都要怪罪一方。但在这种情况下，我们绝对不是这种对抗性螺旋的发起者。”安全理事会副主席说。

据他说，今天俄罗斯和西方的关系比冷战时期还要糟糕。

5月31日星期一，北约秘书长延斯·斯托尔滕贝格表示，北大西洋联盟希望改善与俄罗斯的关系。他说，莫斯科和西方之间的关系是困难的，但双方应该保持对话。

法国和德国也在两国部长理事会的虚拟会议后宣布了继续与俄罗斯对话的计划。
