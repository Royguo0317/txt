## 欲蓋彌彰！武毒所疑自曝洩毒陰招

作者：喜馬拉雅聯盟加拿大農場龍井（文章僅代表作者個人觀點）

北京時間7月6日，中國國際電視台CGTN發表了一則對武漢國家生物安全實驗室的採訪報導，以《沒有許可，一隻蚊子都飛不進去》為題，目的是想給武漢P4實驗室洗地，藉此表明CCP病毒的洩露與該實驗室無關。對此，有網友發帖表示困惑：當初被質疑洩毒時，該實驗室“連屁都不敢放”，為啥在這個時候要藉央視搞個“特別聲明”？更多的網友則認為，實驗室此舉顯然是“此地無銀三百兩”，欲蓋彌彰。

央視報導的導語是這樣寫的：“武漢病毒所在疫情期間一度成為外媒炒作的焦點，總台記者獨家探訪武漢病毒研究所內安全級別最高的P4實驗室，負責人告訴記者：沒有得到許可，一只蚊子都飛不進去；沒有得到許可，裡面的科研人員一滴水，一張紙片都拿不出來。”

央視報導的這個標語和導語立刻引發了眾多網友的熱議。有人直呼“這個標題太讓人無能為力了！”也有網友表示：由於中文的博大精深，我覺得這個導語很容易引起這樣的歧義：一切發生了的，都是經過許可的。

更有網友對這段導語缺乏邏輯性展開了各種姿勢的戲謔：有人問“一滴水都帶不出來”是如何做到的呢？裡面的人出來都要脫水是嗎？ ”“每次出來都要裸檢嗎？ ”“我在實驗室裡喝一口水，出來撒尿；或者我在外面喝水，進去撒尿，就破了負責人的話啦！ ”“請問蚊子的出入證許在哪裡申請呢？ ”“蚊子要愣是飛進去了怎麼辦？ ”“病毒可比蚊子小太多了！而且蚊子如果得到許可了怎麼辦？ ”“負責人可以拿東西出來嗎？ ”“黃婆賣咱瓜信不過，WHO去調查過麼？ ”

有網友半開玩笑地說：“蚊子都進不去，可也攔不住的四婚的關係戶王延軼所長！”有人乾脆說：“蚊子不用你的許可。” “蚊子飛不飛得進去是次要的，關鍵是別飛出來。” “進去的咱不知道，出來的東西公開一下吧！”

有網友則直截了當地質問：“給大家一個真相！這病毒到底哪兒來的？半年了還沒調查出來嗎? ”“這個病毒所在武漢病毒爆發的時候有什麼貢獻？”“剛開始為啥連屁都不敢放一個？”

更有人簡單明了一語中的：“關鍵是’許可’” ，那麼，誰有權力許可呢？ ”

央視訪談中，武漢實驗室某負責人的這段話也讓人遐想。他說“2003年SARS的爆發，暴露出我們國家在公共衛生體系方面的薄弱環節，所以當時我們就啟動建立了高等生物安全實驗室”。仔細琢磨這話，這位負責人是不是一不小心說漏了嘴，透露SARS病毒是因為實驗室的級別不夠高，管理不善而藉助蚊子的翅膀偷偷溜出來的呢？

央視報導全程只有P4實驗室負責人一個人唱獨角戲，央視記者沒有任何發問，與其說是採訪，還不如說是宣傳廣告。而且，整個報導漏洞百出，邏輯不自洽，實在是弄巧成拙，欲蓋彌彰。更蹊蹺的是，病毒所所長王延軼自病毒爆發以來一直在玩“隱身術”，面對鏡頭的是一個連姓名都不敢透露，具體職務都不明確的什麼負責人，足見他們暗地裡藏著多少見不得光的貓膩。
