## 中共闹剧落幕：WHO武汉新闻发布会后再无“进口食品”核酸阳性
` 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hans/931787/)

作者：Stephen文文
![]()![](https://gnews.org/wp-content/uploads/2021/02/2506.jpg)
自从2月9日WHO病毒溯源小组在武汉举行新闻发布会，将进口冷链也列为病毒传染的一种可能性后，中共竟连续两周再无“进口食品”核酸阳性的新闻报道出现。可见中共栽赃嫁祸病毒来源给进口食品的目的已经达到，多一分钟的戏都懒得再演下去了。当然，不排除中共看到我们的文章之后再给自己加戏，再报道几个进口食品阳性的新闻。

WHO在武汉调研的时期，正是中共炒作进口食品核酸阳性的高潮期。仅在2021年1月中下旬，中共报道的进口食品/物品阳性的案例不完全统计就有：美国进口啤酒外包装（1月24日）、阿根廷冻红虾（1月23日）、天津雪糕（1月15日）、山东奶枣（1月23日）、进口车厘子1月21日）、俄罗斯冻狭鳕鱼（1月28日）、印度尼西亚冻马鲛鱼（1月28日）、巴西冻墨鱼（1月28日）、巴西冻鸡腿（1月28日）、秘鲁冻鱿鱼（1月25日）、印度冻杜氏叫姑鱼（1月26日）、进口棉花（1月26日）、俄罗斯冷冻鸡脚（1月27日）。

我们可以看到中共国是世界上唯一一个频繁出现所谓“进口食品/物品病毒核酸阳性”的地方，而且非常频繁。但是，就在WHO调查组开完新闻发布会，离开武汉后，中共就再也未报道过类似事件发生。很显然，在拿到WHO的背书后，中共将病毒来源甩锅境外的目的已经达到，戏就不需要再演了。

中共为什么千方百计甩锅病毒来源进口冷冻食品？因为在武汉华南海鲜市场的环境样本中有33份呈阳性，而其中交易的动物体内并未检出病毒，无法将病毒来源甩锅给海鲜市场中的动物，那么中共要甩锅就只有冷冻食品这一个算是能说得过去的理由了。这就解释了中共为什么从2020年中就开始不断报道进口食品病毒阳性，而这一炒作在1月下旬又达到了高潮，在WHO离开后又偃旗息鼓的原因。可以看出中共的各种食品/物品阳性是完完全全有预谋有目的的闹剧！
