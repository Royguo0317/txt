上文单说迟浩田的“无神论”，揭露其真实目的是为了建立并巩固崇拜中共的“一神论”。今天细说他推出的三个“牢牢抓住”：牢牢抓住民族的生存空间问题，牢牢抓住执政党的领导，牢牢抓住当“地球之王”这个大方向，紧紧拧住这三个问题不放。

先看迟浩田怎么说第一个问题的。“争夺基础性生存资源（土地、海洋）是历史上绝大多数战争的根源……这二十几年发展经济的恶化作用和气候的急速恶化，资源严重耗竭，环境严重恶化，尤其是土地水源和空气问题格外严重。我们的可持续发展甚至是民族的生存，都面临着严重的威胁……我们要解决这个问题，就必须带领人民走出去，向国外发展。”

首先，我们要记住，为了这个报告，中共曾授意新浪网做了民意调查：当战争来临时，若你的上级允许，你会向战俘和妇孺开枪吗？超过83%的被调查者回答“会”，这个结果让中共大喜过望，于是才有迟浩田第二个报告的毫无顾忌。也就是说，迟浩田的报告说明中共一直在做战争的准备，只是战争的形式不一定是兵刃相见。

迟浩田说争夺基础性生存资源（土地海洋）是历史上绝大多数战争的根源。那就用中共自己的案例来反驳他吧。

1949年10月1日，中共宣布篡政成功，当月24日就在厦门向隶属台湾的“金门岛”发起进攻。可惜，天不假功，共军全军覆没，非死即俘，从此，中共对台只敢打口炮，昨天喊“和统”，今天喊“武统”，现在竟然有人喊出“逼统”，在世人面前丑态百出！960万平方公里养不活那时候区区四亿多人，要去争夺那个他们屡屡称之为“弹丸之地”的小岛吗？紧接着所谓“抗美援朝”和资源没有关系吧。60年代初的中印战争不仅与争夺资源没关系，还赔了土地给印度；60年代末的中苏“珍宝岛战争”也和中共巨大胃口的资源无关。70、80年代年代对越战争，更是赔了士兵赔土地，加上50年代割让的“夜莺岛”，中共对越是赔了海岛赔陆地。三四十年大小对外作战六七次，百姓子孙死伤无数，土地像韭菜一样割了又割，中共它又为中华民族争夺了一寸土地吗？都是因为它的政治目的，穷兵黩武，置百姓的生命于不顾。

他们承认了对中国国土的前所未有的“祸害”：连用三个“恶化”和三个“严重”，可知中共对中华民族犯下的历史性罪恶是空前绝后的，而且是很难逆转的。竭泽而渔式的发展，肥的是中共金字塔顶端极少数家族，毁的是国家的整体资源和民族的后代发展。大自然赐予的江河，他们修建数不清的拦截大坝，人为恶化气候和环境，也导致无数人因移居他乡而屡陷困境，比如三门峡移民和三峡移民……一会儿毁林造田（毁草造田），一会儿退耕还林（退耕还草）；一会儿拆毁各地古城，一会儿各地兴建“古城”；一会儿兴建高楼，一会儿炸毁大厦……数不胜数！几十年折腾下来，天上的飞鸟越来越少，雾霾越来越重；江河的游鱼越来越少，流水越来越浊；地上的走兽越来越少，流氓却越来越多。

迟浩田有一句话是对的：我们的可持续发展和民族生存都面临着严重的威胁。这威胁是谁造成的？当然是中共！中共70年的极权统治，剥夺了全民的财产和生命，也剥夺了全民的生存空间。百姓一无所有，只面临满目疮痍的灵和肉与天和地。可怕的是，中共不思悔过，却把目光投向了海外。“西方国家在全世界殖民，从而在生存空间问题上占据了有利地位。我们要解决这个问题，必须带领人民走出去，向国外发展。”看到了吧，这才是中共的终极“中国梦”。它梦想有朝一日“殖民”世界，做“地球之王”，做它的“日不落中共帝国”梦！

之前，笔者用印度和日本相比来说明，中国的土地养活14亿人口绰绰有余，根本不存在“生存空间问题”。从中共惯于欺骗的本质来看，中国现在有没有14亿人口很值得怀疑！但它一定需要“14亿”这个数据，可以帮它解决许多它本身无法解决的问题，更可以让它为自己歌功颂德。1979年中国官方数据是9亿多人口，40年血腥的计划生育政策加上政府对民众的肆意杀戮和各种非正常死亡，笔者严重怀疑“14亿”这个数字。在中共国，猪肉牛肉可以注水，西瓜冬瓜可以注水，GDP数据可以注水，银行存贷款数字可以随意注水缩水，国家统计局各样数字可以注水，人口数量为什么不能注水呢？只要中共需要，给江河“注水”都可以做到。
