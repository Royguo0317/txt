2020年9月29日，中共黨媒人民日報刊發文章《美國大選不應濫用“中國問題”做戲》稱，“美國大選是美國的內政，中方沒有興趣也未進行幹預”。文章認爲“中國問題”是美國總統選舉的固定‘橋段’”，其目的是爲了“轉移視線、外化（美國內部）矛盾”，並暗示美國總統不應以“戰爭”之名競選，否則損害的是自己人民的利益。最後不忘強調“中美關系健康發展，是兩國人民的共同願望”，中美“脫鈎”與此相悖。

那麽，到底是誰在幹預美國大選呢？

中共國一切姓黨無人不知。中共在這片土地上，“上管天，下管地，中間管著生殖器”（郭文貴先生語）。中共國內誰都不敢逆中共說話，人民日報就是中共的傳聲筒。坊間傳聞人民日報某前主編說法“人民日報除了日期是真的，其余都是假的”。

“人民日報”，請你在說話前先把功課做足，別一派胡言。其實幹預美國大選的正是“中共”，爆料革命和美國揭露的事實，上面的微博截圖，都證明了這點。另外，文中所謂的“中國問題”替換爲“中共問題”更符合客觀事實。

爲何美國總統競選要拿“中共問題”說事？

關注爆料革命的人都知道，冠狀病毒從中共的軍方實驗室流出，並被故意散布到了美國和全球，導致美國目前已死亡近20萬人，全球死亡數十萬人，這比珍珠港事件和911事件有過之而無不及。

在美國人民的生命安全受到中共病毒嚴重威脅之際，美國人民不是應該選出那個有決心、有魄力去討伐殺害自己親人、朋友、同事元凶的總統嗎？美國現任總統川普及其團隊多次在公開場合表示，要對中共追責新冠病毒。難道這不符合美國人民的利益嗎？據郭文貴先生稱，中共一直在實施“搞弱、搞亂、搞垮”美國的“3F”計劃。對于美國總統，還有什麽比阻止美國人繼續死亡更重要的事呢？

然而，中共竟然通過“人民日報”，不僅將自己應負的責任推脫爲美國總統競選的“橋段”，甚至偷換概念，將美國即將追責的“中共”偷換爲“中國”，赤裸裸地綁架整個國家來替自己賣命、擋子彈，死要拉民衆墊背，可謂無恥邪惡到極點！
