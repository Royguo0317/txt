**西班牙2021年3月11日电/西喜社——**据美联社3月11日报道，德国疾病控制机构负责人表示，自一年前疫情开始以来，防止中共病毒传播的措施已经挽救了该国超过10万人的生命。

洛萨·威勒（Lothar Wieler）周四在慕尼黑工业大学给学生们做讲座时说，他的机构计算出德国感染中共病毒的致死率约为1.14%，这意味着病毒在全国范围内的传播可能导致80多万人死亡。

“在我们国家，我们通过这些公共卫生措施已经挽救了一万条，甚至十万条生命。”罗伯特-科赫研究所负责人威勒说。

该机构报告说，德国一夜之间出现了14356例确诊病例和321例死亡病例，总数超过250万例，与中共病毒相关的死亡人数达到72810人。

威勒周三在日内瓦的联合国会议上表示，他认为德国已经开始了 “第三波 “感染。
