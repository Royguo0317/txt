据《国家脉动》（National Pulse） 4月22日报道，微软一直面临着来自中共的大规模黑客攻击，如今却与一家最近从与中共军方背景华为公司剥离出来的科技公司Honor合作。

微软在为Honor的笔记本电脑，如MagicBook Pro提供其Windows 10操作系统。2020年12月的合作关系是在Honor公司从华为分离出来仅一个月后宣布的，华为是一家与中共与中共有联系的电信公司，受到川美国府的严厉制裁。

华为被美国国防部称为 “国家安全威胁 “和中共军事合作者，该公司向中共提供信息通过其产品、网络和设备的后门。美国国务院还指出，华为被用作 “不仅是赚钱的工具，也是追求党国议程和实现其战略目标的工具。在国内深深卷入北京的压迫体系，并在全球范围内拥有日益增长的战略野心。”

Honor公司仍然与中国共产党保持着联系，因为深圳市政府是参与以150多亿美元收购Honor公司的几个政府之一。更重要的是，主要购买者神州数码通过与云计算等项目的合作，与华为保持着联系。微软通过与中共政府保持与Honor类似的公司合作，规避了美国科技公司与中共合作的限制。

Honor公司首席执行官赵明对这项交易感到 “高兴”，指出 “将看到我们在全球范围内与微软合作”。信息消费联盟主任项立刚指出，微软决定与Honor公司合作，可能会催化更多美国公司与该公司合作。他强调说：”随着一家美国公司与Honor合作，在不久的将来可能会有更多的美国公司加入进来，这可以更好地解决Honor的供应链问题“。

这次合作是继微软与中共纠缠在一起的一长串事件之后，微软提供操作系统给供中国政府使用，还与中共的军事代理人和中共的科技公司一起出席会议等。拆分和剥离公司是中共躲避限制和制裁的惯用伎俩，微软与损害美国利益的中共科技公司合作到底是喜还是忧？时间会给我们答案。
