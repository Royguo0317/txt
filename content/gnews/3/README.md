中華大地上，東西南北中，有無數個趙家溝村。然而吉林省磐石縣紅旗嶺鎮的趙家溝村，卻與眾不同。

眾所周知，這裡盛產鎳礦，被譽為“中國鎳都”的重要礦區。而早在1960年左右就建成了規模不小的開採區，如今更是催生了總資產32億人民幣的吉恩鎳業股份有限公司。

然而，人傑地靈，真正能夠讓小小趙家村聞名世界，甚至在人類歷史上也留下濃墨重彩一筆的，是這塊地下藏寶的土地上，降生了一位極為特別之人。是誰呢？他就是1970年5月10日，礦區有一郭姓人家裡，誕生的一名男孩。他生著一雙丹鳳眼，星眸靈動，目光灼灼。英唇未啟，一對深圓的酒窩躍然臉頰，讓人過目難忘！

郭家男主人名叫郭金福，女主人名叫吳月香，當時家中已經生養了六個男孩，這是第七個男孩臨世。父母為他取名郭文貴，家人常親熱地喚他“老七”。

郭家生活的趙家溝，顧名思義，是一片聚居的平地，圍繞著起伏的山丘。趙家溝並非普通的農村山溝，那裡是鎳礦開採地，是國家軍事特區。當時已經通達火車，有世界上各種先進採礦設備，包括地下採鎳運鎳的採掘機。聚居此地的人也並非當地土著農民，大多數是外來人口，有東北逃荒來的，有下放的干部，有軍人或情報背景的人，還有國外的專家。因為有很多洋專家的緣故，山溝裡也就能見到全世界最流行的洋裝及其他洋裝備……山青青，水碧碧，群山環繞，一溝人間文明，宛若世外桃園。

然而郭家的日子卻過得異常艱難。郭爸爸和郭媽媽均非等閒之輩，原本是派往當時的蘇聯境內，執行特殊任務的特情人員，半路卻殺出了“文化大革命”被就地安置，於是成為了趙家溝裡的一家。然而他們被打成右派，郭爸爸甚至被游斗毆打，以致被打斷了腿。郭媽媽前後一共生了十個孩子，養活了八個男孩，家里大人孩子一共有十張嘴吃飯。因為他們不是農民，沒有土地，只能到山坡開荒種地，儘管日夜辛勞，家裡依然吃了上頓愁下頓。

吃一頓貼大餅子，或大碴子，已經是家中盛宴，多數時候只能喝稀湯、菜湯。不僅缺糧食，連柴火都不夠，郭媽媽只能拿山上砍來的濕樹枝湊合燃炊，一做飯便濃煙滾滾，熏得人睜不開眼睛……飢餓、貧困、疾病、人禍一直困擾著郭家。

補丁爛衫、稀湯菜湯，這就是70年代中國農村的生活現狀，成千上億的兒童都過著那樣的生活。而且趙家溝遠比一般農村的境況好，其他地區有些孩子下雪天打著赤腳，破洞衣褲灌著風雪，一年或者幾年才吃上一回貼大餅子、大碴子。

但是一個天生樂觀、善良、堅強的孩子，無論環境多麼惡劣，他都能夠在自己的世界裡編寫著美麗的童話。 “老七”就是這樣一個孩子，他在直播中無意提到了兩個片段。

一個是“火內褲秀”。有次，他好不容易盼到一件新衣裝，雖然只是郭媽媽用一塊新紅佈為他縫製的內褲，他也十分喜愛十分興奮。在東北冬天的零下N度裡，他故意起了個大早，不穿外衣，露出紅內褲，外出溜達顯擺。發現別人根本沒注意到，他還故意咳嗽兩聲，好把人家的目光吸引過來，欣賞他的“火內褲”。不難看出，天性裡他是多麼懂得珍惜和享受“擁有”，多麼熱愛生活。

另一個片段，更加唯美。小學他就情竇初開了，跪在窗台上，給班上的女同學寫情書，那時窗外飄著雪花。雖然他在直播中簡短一句帶過，但無不引起聽者的無限遐想。此時此景，他的腦海裡是不是浮現著那女孩兒的音容笑貌，秀發青又青、杏目亮又亮、小嘴笑得彎又彎……小哥的心裡翻江倒海，傾慕喜悅如沐春光，暖暖地汩汩傾瀉筆端。窗外的雪花無聲地飄啊飄，痴痴繞繞，如果還有月輝靜靜相伴，豈不正是“風花雪月”好風景！童話故事的結尾，總是說王子和公主終於結婚了……可是，七哥最後說，寫了很多情書，一隻手都沒牽過。多麼童真的愛戀！

如今的七哥，已經50歲了，他住在美國曼哈頓豪宅里，有私人飛機和遊艇，他發起了爆料革命，他引領著成千上萬的草根滅共，他締造著新中國聯邦。他依然喜歡漂亮的衣服，時不時在直播裡來一下“Gfashion時裝秀”。他依然心繫著成千上萬的女士，包括楊改蘭那樣卑微絕望而死的母親。

或許正是因為七哥有過趙家溝的童年，他才深知天下草根們的苦樂，才能夠心心相通，共同推翻CCP極權；或許正是因為七哥擁有趙家溝的童話，他才深知天下草根們的憧憬，才會帶領草根們建立民主、平等、法治的新中國聯邦，過上體面美好的生活。
