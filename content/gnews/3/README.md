**西班牙2021年5月3日电/西喜社——**

随着新冠疫苗在海外供应的短缺，为了让疫苗能够被更广泛的使用，越来越多的国家被迫延长了第一针和第二针疫苗之间的接种时间。英国的一项关于疫苗接种率的最新研究数据显示出一个令人不安的结果：研究发现，已经接种过第一针疫苗的人群仍然面临着感染变异病毒的风险。

这项研究由伦敦帝国理工学院于周五下午发表在《科学》杂志上，主要研究了医护人员在注射了第一针辉瑞疫苗后的免疫反应。

经研究发现，曾感染过新冠病毒的人,通常对变种病毒的防御能力明显增强。那些只接种过疫苗但未曾被感染的人，表现出的免疫反应则是“在第一针接种以后免疫反应不太强烈，这可能使他们面临感染变异病株的风险。” 研究结果显示，那些接种过辉瑞公司疫苗的人,并不能完全免于变异病株的感染。

与此同时，周五发布的另一项研究也得出了类似的结论。另一项研究也显示出,一些已经接种了第一针疫苗的人,仍然因为出现了新冠症状而进入了医院治疗。

据周五公布的最新数据，65%的英国成年人已经接受了至少一剂疫苗，有3,736,654人在过去7天内接种了疫苗。同时，该国超过了25%的成年人已经接种了两剂疫苗。英国现在已经接种了8,748,962剂COVID-19疫苗，而美国刚刚宣布，1亿美国人现在已经完全接种了疫苗。

但现有的最新研究数据表明，只打过一针疫苗的人仍然很脆弱。尽管研究人员表明，没有疫苗是100%完美的，免疫失败的情况也是可以想象的。但也有人认为，最脆弱的（最可能感染病毒的）人可能会在接种了一次疫苗后就过早的放松了警惕。

这项研究分析了12月初至4月初期间英格兰、苏格兰和威尔士所有医院约总数四分之一的病人，这是最早研究接种疫苗影响随后入院人数的研究之一。

来自利物浦大学的研究负责人卡鲁姆·辛普尔（Calum Semple）教授告诉BBC，这些数据 代表了少数疫苗失败真实的有力证据。
