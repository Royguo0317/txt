## 郭文貴先生的“生意經”（一）
`2020-09-27 04:51 EWENFEI` [轉載自GNews](https://gnews.org/zh-hant/385974/)

整理：文非

郭文貴先生22歲時已經是河南大老闆傢俱廠董事長；27歲時建立了中國最好的5星級酒店——裕達國貿酒店；38歲已經成為全國最頂級酒店——盤古七星酒店的老闆。**郭文貴先生在商業方面無疑是極為成功的**，他曾經在多次直播中談及自己做生意的經驗，筆者來進行了一些整理，以供大家學習。

1.2019年12月7日

***講故事：***

我特想跟大家分享一個小故事。這個小故事是我們每個人每天都在說的，叫**言而有信。**

在大秦朝的時候，有一個非常有名的故事，我記得叫無畏候，可能你也看過。當時是秦初的時候，他想把對面那個軍亭拿掉，但是又不願動用大量的軍力和武力，讓誰去幫助拆呢？怎麼辦呢？誰都不相信政府，政府都是騙人的。最後他出了個招，今天誰把這事幹了，我給你一畝田，發了個告示。很多人不相信，結果有人就冒險去幹了，幹了以後結果就真給了他一畝田。然後，今天誰把這棍子搬到西邊去，搬去之後升什麼官。大家都說這真的假的？結果有人就把棍子搬到西邊去了，給他升了官。然後幾番幾次以後，貼了告示以後，政府都兌現了。突然政府說，誰把對面那亭子移了，我就會給你什麼什麼。嘩~人就過去了，老百姓就把擋在那種田的，對方的亭子給移走了。這是一個真實的大秦朝的故事，後來這就叫政府的信用。今天對比一下我們的政府，大家都知道是什麼結果了。

**第二個就是我們在很小時候，就聽老人講的故事，言而有信**。說個也是秦魏時期的某個故事。一個母親要出去，兒子跟在後面哭，嗷嗷的喊我要這個，我要那個，我要跟你去，大喊大鬧。他爹在旁邊說，你別哭了，叫你媽去吧，你回家，下午我回來給你殺豬吃。當時那孩子就不哭了，看著他爸爸問，真的嗎？真的，回家等著吧，等著爸爸回來殺豬吃。下午他媽媽回來了，看到他爸爸真的在殺豬呢，火了，你怎麼真殺豬呢？咱家就這麼一頭豬，你怎麼殺了呢？

你想想殺豬多大一個事啊。**最後他爸爸說，咱答應了孩子殺豬給孩子吃，咱必須把這豬給殺了，雖然事很大，但他一輩子都會記住，他爹許諾殺豬就給他豬。孩子相信母親，相信爸爸，他會永遠尊重我們**。這豬殺了，這是一個最簡單的故事。

***經驗：***

這個所有的輪回可兌現的金融票據，**信用，言而有信**。更重要的事情，**要對對方絕對的尊重**，還一個更重要的是，當遇到危機的時候，**巨大的危機的時候，你不是把你的壓力變成你的懷疑來挑戰所有你信任的人**。我最大危機的時候，

所有的孩子們，你們要記住，**不要看眼前這個事，不要在一個事情上絆倒，不要把每一天都想成最後一天，不要讓每件事都把你嚇倒**。2014年，2015年我身邊很多人都很我說，文貴啊，快點把這個錢給要回來吧，說啥的都有，萬一要不回來咋辦，**千萬不要讓面前的恐懼使得你失去對未來所有的信任，對所有未來的信心，那個時候你會做出非常不理智的決定，你會慌張，最終讓對方看起來你不信任他的決定，這就會造成對方真的不信任你**。

這就是我認為，**信心，善良，忠誠，守信，言而有信，一定要做到你所說的，做不到的你不要說**，我認為只是我從賀老身上，從當時他認識我到後來對我的鼓勵，這都是我看到最寶貴的東西。

你們現在，大把的賺錢的機會，你們在海外，我要告訴大家，**最危險的事情，最壞的事情，就是你想賺快錢、賺大錢。最好賺錢的是從一點一滴做起**。我郭文貴跟日本人的關係，和這些人的關係，都是從點滴的積累開始的，那不是說一點一滴。千萬別想今天早上發出的願，今天晚上就要收割，或者12點就不算數，那你就完了，一定會出局，一定要有長遠的。

**短視造成了你不可能維持信用，短視讓你不可能有長遠的目標，短視會讓你失去所有賺錢的機會和信用。一定要長遠看，賺錢很重要，但賺錢絕對不能成為唯一目的；要把賺錢的過程當成你做人的過程，你不想要錢，你不想賺錢，錢都砸你的臉，就鑿你，就這麼簡單。**

2.2020年4月23日

請告訴我中國的有一個真正的所謂現在媒體能給你一個安全感的嗎？安全感基於什麼戰友們？三個條件，第一個這家企業的核心，它是首先要賺錢，你別說任何廢話，你是賺錢不是騙錢不是偷錢，不是賣身，**你是真正的靠商業行為賺錢的能力，這個企業如果沒這個能力就要進垃圾堆**。第二個是什麼？你這個企業的所有的存在的**利益和關係**，你還有股東之間他的利益的**安全性**。第三個企業**人才、資源。**

3.2018年8月9日

我賺錢對我來講真的是，就像這一樣（端起水杯），喝杯水。賺錢真不是個多難的事啊。

**當你有信仰有理想，上天想讓你賺錢不賺都不行**。

4. 2017年5月9日

我真想給年輕孩子講講怎麼賺錢，怎麼賺錢我還是很想講講，因為我看到現在的年輕孩子，真需要一個有實際操作價值，馬上就能用的賺錢的招，就像我前天說的“**千招會，不如一招絕**”。我準備了10個有關賺錢的話題。第一個話題，就是“**天道酬勤，地道酬人**”。

我想把我的一些生活中的點點滴滴分享給年輕人，這甚至比爆料還重要。**我認為中國接下來有巨大的機會，但是，機會屬於聰明人，屬於有膽略的人，屬於能把握住機會務實的人**。第一個就是要”天道酬勤“；第二個就是要”地道酬人“。

5.2019年12月8日

戰友們，一定記住，正義，一個非常遠的事業，和千萬別打亂拳，**要千招會不如一招絕，只要你執著下去，你就會賺大錢。還有一個，保持住基本的賺錢的底線，孝敬老人，對朋友對合作者要忠誠，不要看眼前那點事兒。還有一個，這一定要講實話**，沒有一個人，願意和撒謊溜騙兒的。

6. 2020年4月12日

我們所有戰友可以共同發財，不要指望伸手你施捨給我一點，或者我給你錢你幫我幹啥事，這絕對不健康。**共同發財才健康，沒有背判，只有共同賺錢**。這是美國的為什麼聯盟它強大？因為大家共同賺錢。

7.2019年8月28日

幾十年來，都是賺錢，做一個成一個。**我從不炒股票，從不投機。我告訴很多人，你要想在股票上贏就看****3****個地方，看老百姓的糧食和菜籃子價格，看中南海那幾個人的身體健康和鬥爭程度，這就是股票。中國沒有股市，只有賭市，都是賭博的市場**。你要想玩賭就看這兩個地方，低的看老百姓肉蛋、菜籃子，高的看中南坑鬥爭和這些人的健康，誰又換腎了、誰又換肝了？
