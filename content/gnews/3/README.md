03月25日，莫斯科和首尔将在中共病毒形势改善后，制定俄罗斯总统普京访韩的计划。这是俄罗斯外长拉夫罗夫在与韩国外长郑恩宠的联席发布会上公布的。

俄外长说：“我们已经同意，一旦中共冠状病毒流行情况允许，我们将尽快回到这个问题。”部长指出，韩方已确认了俄罗斯领导人的邀请。

拉夫罗夫说，双方还在讨论建立一个最高10亿美元的联合投资基金。

“俄罗斯远东和北极地区投资合作的大型项目正在筹备实施。目前正在进行谈判，以建立一个高达10亿美元的俄韩投资基金。”拉夫罗夫说。

去年9月底，韩国总统文在寅在电话中邀请普京访问首尔。据会谈后的韩国领导人政府部门发表声明说，俄罗斯总统表示，将在中共流行性冠状病毒形势稳定后，在接种俄罗斯疫苗后访问首尔。

3月23日，普京接种了俄罗斯“中共冠状病毒”疫苗。
