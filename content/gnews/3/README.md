闫丽梦博士：WHO的报告中呈现出来的证据是空洞无物的
这份报告完全是WHO用来为中共向全世界声称无辜和撇清罪责的
今年的神圣星期五闫丽梦博士被邀请到史蒂文·班农（Steven Bannon）先生的战斗室参加了节目。她对于世界卫生组织（世卫组织）WHO关于COVID冠状病毒全球大流行的调查报告给出了自己的个人评判。她认为这个报告是不可靠的，无论是从报告中所举出的科学证据看，还是从提供材料和撰写报告的参与者来看都是不可靠的。更重要的是，她揭示出这个报告实际上是中国共产党（中共）跟其沆瀣一气的世卫组织WHO对于全球民众信息超限战的一部分。

世卫组织WHO报告是一份世卫组织WHO与中共国联合产出的报告，更确切一点说，是中共跟世卫组织WHO联合产出的报告。这意味着，撰写这份报告的主要科学家，包括来自中国的也包括来自世界其他国家的，无一例外，都是被中共允许和认可的科学家。世卫组织WHO给了中共[全权](https://thepalmierireport.com/report-who-granted-china-authority-to-veto-scientists-on-wuhan-mission/)处理权，由中共批准哪些科学家可以参与调查，哪些不可以，最终选出了[十位](https://www.who.int/health-topics/coronavirus/origins-of-the-virus)可以参与调查的科学家。然而就在这十位科学家中的两位，已经从自己的国家出发去往中共国的路途中，被中共临时通知取消了参与调查的资格。剩下的调查组八位科学家到达中共国后，在旅馆隔离了14天，之后开始了所谓的调查——他们在武汉病毒研究所（武毒所）只呆了3个小时，就离开了。

世卫组织WHO这个所谓的调查组在武毒所逗留的三个小时中，没有要求中共提交任何的实验室数据、实验室样本、以及工作人员日志，因为中共事先给他们设定了一个[基本规则](https://www.cbsnews.com/news/covid-19-wuhan-origins-60-minutes-2021-03-28/)：世卫组织WHO没有任何发号施令的权限，不可以对中共指手画脚。不幸的是，世卫组织WHO不仅仅同意了中共的这种做法，而且就连这份调查冠状病毒全球大流行起因的报告都全权交给了中共控制的所谓科学家去写，中共写好了报告以后再交给这个所谓的世卫组织WHO调查组的国际专家来用于对外发表，美其名曰：“分享成果”。
