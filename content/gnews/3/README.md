05月28日，美国必须成为发展基础设施和研究方面的领导者。据《华盛顿邮报》报道，美国总统乔·拜登在俄亥俄州的一次演讲中这样说。

根据拜登的说法，该国在发展和研究方面处于第九位，而中共国则处于第八位。然而，拜登并没有说出其他参与排名国家的名字。

30年前，美国在发展和研究方面的支出处于领先地位。总统说，多年来，日本、中共国、意大利、法国和欧盟已宣布对其基础设施进行重大投资。

“我们必须成为世界第一，才能在21世纪处于领先地位。我们不能被抛在后面。”拜登补充说，“现在是建立基础的时候了。”

去年1月，美国研究人员表示，美国在核武器的现代化方面落后于俄罗斯。服务专家艾米·沃尔夫（Amy Wolfe）指出，专家们将两国核武库的现代化计划比作一场军备竞赛，并指责俄罗斯挑起了这场竞赛。美国在这方面是落后的，因为俄罗斯在2000年代初开始部署新的导弹和潜艇，而美国在2020年代末才计划开始使用这种系统。这位专家还指出，俄罗斯正在开发新的“更奇特”的武器系统。
