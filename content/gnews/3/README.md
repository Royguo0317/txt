在第二篇主题文章中，作者将介绍相关财务人员，以及管理人员的行为。并根据行为模式的解读与客观，审慎，勤勉的的态度作出有效评估关于潜在的在的欺诈以及诈骗风险。并有效衡量实际商业运营中可容忍的风险程度。

1. Standard I（C）Misrepresentation
    - 此标准禁止:成员对有波动的投资做出任何具体回报的保证
    - 虚假陈述:任何事实不真实陈述 / 遗漏，或任何虚假或误导的陈述
    - 不是协会成员造成的虚假陈述错误，不违反此条款。
2. Impact on Investment Practice
    - 如果成员无法确定某种程度复杂性，不采取投资行动或投资意见则为最优选择
    - 成员应避免单一投资行动的表现信息代表全部的投资类别，这样会潜在地误导客户。除非单一投资表现可以代表全部投资行动。
3. Performance Reporting
    - 成员所在实体无法控制第三方对于宣传材料的创作，且成员发放含有虚假陈述的第三方材料则违反此条例。因为成员有义务纠正或澄清不明材料
    - 成员即使是在管理第三方资产时，如果意识到第三方任何虚假陈述，也有义务持续遵循政策和程序去解决问题
    - 即使得到外包服务商的许可能够重新包装作业并分发给客户。但也不得使用本公司成员作为外包作业的创作者
    - Guarantee被使用时：应充分解释任何破产可能和任意实体限额内担保之间的差异事实
4. Social Media
    - 任何媒体 / 二手信息来源都应确保安全可靠，最佳方法是直接引用文章作者，并获取信息来源
5. Omissions
    - 分析报告中对于利益冲突的遗漏，以误导投资者认为分析师的工作是客观独立的
    - 对于管理的账户中进行选择性移除，以提升投资组合整体表现。违反条例
6. Plagiarism
    - 直接抄袭或仅有轻微改动
    - 模糊头衔的引用，无具体参考
    - 使用别人准备的进行预测的统计估量已确认来源，但没有二次使用的测陈述或预警
    - 使用图标但，没有标明来源
    - 尤其注意口头音频交流中也会涉及抄袭
7. Work Completed
    - 成员被允许使用其他前任员工的工作成果，但不能仅用成员自己一人名字为代表
8. Standard I（D）Misconduct
    - 会员不得卷入任何不诚实，欺诈，欺骗的行为也不得从事任何对职业声誉或诚信能力造成不利影响的行为
    - 过度酗酒
    - 虚开发票，申请补贴
    - 任何场外的相关行为，同样违反此条款
    - 个人与财务无关的相关行为，不违反此条款（示威游行 / 个人信仰 / 公民抗命)
9. Standard II（A）Material Non-public Information
    - 任何基于重大/非公开材料的交易和信息传播都会违反Standard II（A）
10. Material
    - 特性判断：盈利 / 收购 / 资产重组，质量 / 技术创新 / 执照注册 / 管理层变更 / 券商审计相关
    - 来源判断 领域专家 = 属于重大/非公开消息
    - 其它交易员 = 垃圾杂波
11. Non-public
    - 是否公开传播为标准。未公开时，则禁止基于此消息的交易及传播
12. Mosaic Theory
    - 非重大/非公开信息得出结论，可作为投资建议以及基础
    - 分析师需要越过公司本身去获取信息资源，来自于公司内部人员的口头交流 / 日常内部参观
13. Social Media
14. Industry Expert
    - 领域专家的消息也属于重大/非公开范围，基于此的任何交易和传播都会违反条例
    - 但作为辅助并帮助分析师做出推荐判断决策，不会违反条例
15. Research Report
