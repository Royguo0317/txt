最近，推特上一个美国人告诉我，他刚刚从一家科技公司辞职，原因是该公司被“中国（中共）”资金所控制。公司里面的管理层是一批典型的被中共洗脑的国人，撒谎、欺骗、不守承诺，使这个美国人辞职之前来了一句：“恶心的中（共）国人。”

文贵先生和路德先生都曾多次说过，​灭共不仅要灭掉共产党，而且要灭掉它在全球布下的金融、间谍、实业系统。因为只要共匪在海外的金融资产、固定资产和已经洗白转化为实体企业的资产还存在，它就有钱、有人，随时可以卷土重来。那么，即使我们在大陆灭了它，它依旧可以随时影响和操纵国内的政治和经济局势，甚至依旧可以对全球政治和经济施加巨大的影响，成为一个隐蔽的全球性的“幽灵”党。

笔者曾在越南一家华人开的私企上班——该企业是越南富士康和许多在越南的中（共）国电子厂的供应商。公司有两个老板，他们是姻亲关系，其中二老板是中共军队里团长退役下来的高级中共党员。致使笔者处处感觉得到，实际上他才是公司的真正实权者。也就是说，该“私人”公司本质上是共产党控制的公司。有一次，老板酒后说到他的发迹史，言语当中把自己的英明神武吹得天花乱坠。但让人在意的是，公司的核心在于，二老板利用中共职务的关系，让他能稳定地接到美国贝尔金、谷歌等公司的订单。再加上中共政府、银行和富士康等大企业处处给他开绿灯，他企业因此做起来了。

文贵先生之前爆料过，富士康郭台铭实际上早已被中共控制。而现实的工作经历中，富士康也证明了这一点：单从我的感觉，富士康完全就是一家中共企业。即便到了越南，它的许多供应商都还是中共实际控制的企业。尤其，富士康采购的时候对中共控制的企业表示出特殊的优待，所以有些越南本土企业完全搞不清楚情况：为什么偏偏大订单只给中（共）国公司，而自己只是拿一些备胎一样的小单？他们还以为只是中国人和台湾人同文同种的原因，或者是自己关系拉得不到位。但他们根本不知道，别人原本就是服务于同一个老板：中共。

而且据我所知，富士康前几年收购了​美国通讯器材零售商贝尔金。难怪大老板常说，自己和贝尔金、富士康的关系好得很。现在看来，原因在于他们也是一家嘛。中共控制富士康，富士康收购贝尔金。曾经的一家美国公司，如今，老板实际上变成了中共。之前笔者的亲人在中（共）国做食堂，使用从美国进口的Smithfield牌猪肉。于是，家人托我了解下这家公司——因为猪肉便宜。后来笔者通过仔细观察才了解到，该美国公司实际上也已经被中共控制了。笔者再从生活、工作经历中联想到推特、谷歌、YouTube、微软这些大企业，他们对中共和中共控制的企业也是态度良好……真的难以想象，中共到底渗透、控制或影响着多少美国公司。更可怕的是，收购和控制这些企业的资金大多都已经洗白。

中共控制的这些企业组成了一整个经济系统。他们在越南、美国、印度以及全球各地推行着中共式的价值观和行贿受贿、淫乱无度的生活方式。乃至在越南生活久了的华人自己都知道，“哪里华人多，哪里越南人的民风越败坏淫乱”。这些中共控制的企业里的红色中国人基本上只知道像奴隶一样干活：没有信仰，没有精神生活，有钱了就知道淫乱和吃喝……所以即使没有了防火墙，他们也不觉醒。

文贵先生说过，即使中共灭了，它也还会是一股巨大的力量，这真不是空话。中共这些已经洗白的资金和实际上由它控制的遍布全球的私人企业集团，在共匪灭了以后，若不妥善处理，依旧还会继续散播它的余毒。我认为，未来必须在全球有一个追缴行动：彻底追缴这些中共的黑钱，中共通过种种手段在海外配置的资产和他们为自己在海外搭的窝。但我同时相信，审判的那一天必定会到来。全球必定会以跨国犯罪集团和全球放毒的事实，一起斩断中共经济的魔鬼之爪，阻止其撒旦式的生活方式和价值观的蔓延。在此，也劝告海外企业经营者同胞们，早日站出来，与中共划清界限，与爆料革命联合，方为上策。
