著名的地缘政治学家、前美国安全顾问布热津斯基在1996年就认为，二战后的日本已经失去了对外扩张的野心，日本非常乐意充当美国的小弟兼盟友。的确如此，日本在重大国际事务上一直唯美国马首是瞻，基本上除了不断巩固日美同盟，就是专注于发展经济，闷声发大财。但这种状态因为拜登的上台正在发生巨大的变化。
![]()![](https://gnews.org/wp-content/uploads/2021/03/jpninfo.com-2.jpg)*图片来源：jpninfo.com*
知名记者石坂明夫近日指出，中共的威胁由来已久，但日本政府自拜登上台后动作频频并且反应激烈，是因为之前的川普实行对中共全面对抗的政策，而且川普反共比谁都积极，日本只要在后面跟随就可以了。但拜登上台后实行对中共局部对抗的政策，并且日本不知道拜登心里打什么算盘，从而导致日本的焦虑。

日本的社会结构十分稳定，可以说日本是全世界在国家治理和民主实践中最成功的国家之一，当然其中有很难仿效的民族性等独特因素。日本对中共70多年来致使大几千万无辜平民死亡的残暴统治，以及对本国人民放毒这样的事都能干得出来等等，一直进行深刻的观察，当中共对外进行极权扩张，要把红旗插遍世界时，一贯警醒的日本当然知道在这个生死存亡的时刻无所作为的严重后果。

东盟是中共蓝金黄的重灾区，中共和东盟10国在1997年建立了自贸区，其中的印尼、越南、菲律宾等国和中共在南海存在领海争端。中共正在利用疫苗加紧控制这些国家，在“亚洲北约”中充当主要角色的日本，针锋相对地和中共展开了对东盟的争夺，除了通过印度大量生产疫苗供应东盟，还和东盟的高层进行高层互动。比如本周日印尼和日本两国的防长就中共在南海的军事威胁进行了会谈，紧接着在本周二两国将举行防长和外长的2+2会议。

就美日即将举行的首脑会谈，石坂明夫预测，美日可能对中共在新疆犯下反人类罪等达成制裁的共识，这将极大带动全球对中共的反制。石坂明夫称，日本有几万家企业在中共国尚未撒出，按照中共一贯的作风，一旦日本对中共国实施制裁，中共肯定对这些日企进行报复，但这是日本必须付出的代价。去年3月CCP病毒疫情爆发时，日本政府曾经紧急追加约24亿美元的预算，用于支付日企从中共国撒离的费用。

二战后日本效法英国的光荣孤立主义，不再谋求领土扩张，但对周边大陆国家釆取均势政策，当中共在加入世贸之后不断野蛮崛起时，日本向美国提出了重返亚太的战略建议并被美国采纳。日本还主导了将中共拒之门外的跨太平洋贸易协定(TPP)及围堵中共的“亚洲北约”，和印度等国签订了互用军事基地条约，为在印太的北约海军提供后勤服务等。日本防卫相岸信夫甚至公开宣称，如果台海战争爆发，日本自卫队要支援援台的美军，岸信夫的胞兄安倍晋三昨天也公开表示，日本处于美中对抗的最前线。

经历过二战和原子弹攻击的日本，当然知道现代战争的残酷性。因此有理由相信，当今日本趋于强硬并积极地合纵连横，就是为了团结一切力量对中共形成力量优势和震慑，以期达到不战而屈人之兵的最佳目的。美前国务卿彭佩奥在阿拉斯加会议后曾一针见血地指出:中共吃定了拜登的软弱——软弱激发敌意，软弱引发战争！
