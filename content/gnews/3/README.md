## 新疆维吾尔人权观察报告：在过去的7年中，有630多个伊玛目被捕

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/14122829/iu-1-1.jpeg)[图片来源](https://uighurtimes.com)

在过去的7年中，有630多个伊玛目（代表教长，即人与真主之间的中介）在新疆被捕。英国广播公司今天早上发布的最新《维吾尔人权观察》报告谴责北京（中共）对新疆维吾尔族穆斯林聚居区的紧缩政策。据报道，被中共逮捕的许多维吾尔神职人员被指控为 “极端主义宣传”、“聚众扰乱社会秩序” 和 “煽动分裂主义”。根据该宗教亲戚的证词，他们真正的 “过失” 是宣讲或召集祈祷团体，或者仅仅是 “成为阿玛目“。

根据法院文件、家庭证词和媒体报道，维吾尔人权项目监测了1046名穆斯林神职人员的命运，其中绝大多数是维吾尔族人。所有这些人均被捕，但仅报告了630人。研究指出，其中约有304人被关进监狱，而不是所谓的 “再教育营”。根据该文件，还发现有18名神职人员在拘留期间或之后死亡。

多年来，中共国一直被指控在该国西北部地区侵犯人权和强迫绝育即美国国会声明的 “种族灭绝”。根据各种国际调查，中共已经建造了380个 “再教育营” ，拘捕了至少100万维吾尔人及穆斯林宗教信仰者。

关于维吾尔族，昨天由澳大利亚智库Aspi发表的另一份报告显示该地区的出生率急剧下降：在2017年至2019年之间几乎减半。根据澳大利亚研究人员的研究，该时期将出生率视为率下降了48.7％。报告的作者解释，这些数字显示出前所未有的下降趋势，比联合国收集生育率数据的71年中，包括卢旺达和柬埔寨的种族灭绝中的任何其他地区或任何时期都更为极端。报告说：“中外专家先前的研究发现，自2015年以来，新疆严格了节育政策，自然人口增长相应下降，但2017年之后更为明显。”

简评：中共国对于在新疆犯下的种族灭绝罪，虽然百口狡辩，但是这不争的事实不会被掩盖，越来越多的新闻也开始涉及这个话题。我相信持续的发酵定会加快把中共送往地狱的步伐。
