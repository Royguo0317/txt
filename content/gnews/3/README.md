## 巴里克團隊、港大P3實驗室和中國國家醫學科學院實驗動物研究所的漫漫往事
`2020-09-08 11:00 立武` [轉載自GNews](https://gnews.org/zh-hant/341469/)

> [Ecohealth、拉爾夫·巴里克和武漢病毒研究所的漫漫往事](https://gnews.org/zh-hant/339441/)

這篇文章提到巴里克團隊**利用武漢病毒研究所和** **Ecohealth** **合作提供的病毒**製造嵌合病毒**SHC014-MA15** **，**同時巴里克表示計劃**在非人類靈長類動物上**針對這種病毒進行相關研究，而中共可以**給予巴里克團隊想要的靈長類動物模型研究。**

事實上，SHC014-MA15基因序列被巴里克團隊**[作為實驗室工具來測試抗病毒藥物GS-5734，也就是瑞德西韋。](https://www.sciencemag.org/news/2020/04/nih-s-axing-bat-coronavirus-grant-horrible-precedent-and-might-break-rules-critics-say)**

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/08094630/200514163629-california-covid-19-drug-lottery-00004029-exlarge-169-1.jpg) GS-5734，即瑞德西韋

**瑞德西韋**

在巴里克團隊發表**[關於GS-5734研究的論文](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5567817/#R5)**中引用了巴里克團隊**[研究SHC014-MA15的論文，](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4797993/)**在**關於GS-5734的論文**中提到，一些**類SARS樣冠狀病毒（即SHC014-MA15）**可以有效**利用人類的ACE2受體**在人類氣道細胞中**進行“高滴度”複製，**並且**對現有的單克隆抗體治療方法以及疫苗產生耐受。**

巴里克團隊在這篇論文想要說明的是**GS-5734** **與現有治療方法不同，針對像SHC014-MA15** **這樣感染性強的冠狀病毒具有抗病毒效力。**

這篇論文的作者之一**范德比爾特大學的病毒學家馬克·丹尼森（Mark Danison）**表示， [“沒有比生態健康聯盟正在做的研究更重要的了， **沒有它，我們在瑞德西韋上的工作絕對不會前進。”**](https://www.sciencemag.org/news/2020/04/nih-s-axing-bat-coronavirus-grant-horrible-precedent-and-might-break-rules-critics-say)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/08094848/denison-1.jpeg)馬克·丹尼森（Mark Denison）

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/08094900/Ralph-Baric-UNC-Gillings-School-of-Public-Health-web.jpg)拉爾夫·巴里克（Ralph Baric）

同樣， **皮特·達薩克**在其發表在**《衛報》**裡的一篇文章裡表示**[“我們與突破性藥物瑞德西韋背後的科學家合作證明它對已知的人類冠狀病毒是有效的”。](https://www.theguardian.com/commentisfree/2020/jun/09/conspiracies-covid-19-lab-false-pandemic)**

**靈長類動物模型**

值得注意的是，在這篇**關於GS-5734的論文**中引用了另外一篇**[利用普通狨這樣的非人類靈長類動物模型進行其他藥物相關研究](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7107395/)**的論文，意在**對比GS-5734和其他已被FDA批准的藥物如洛匹那韋、利托那韋等等**之間抗病毒的區別。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/08095036/1000-1.jpg)武漢大學模式動物研究所

後篇論文是**港大新發傳染性疾病國家重點實驗室（** **P3** **實驗室）和中國醫學科學院醫學實驗動物研究所**聯合發布的。

這篇論文的作者包括來自**港大** **P3** **實驗室的袁國勇和張鴻霖等，**以及來自**中國國家醫學科學院實驗動物研究所的所長秦川、黨委書記鄧巍、**感染性疾病平台主任鮑琳琳副研究員等。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/08104729/%E4%B8%8B%E8%BD%BD-1.png) 2019年陳鴻霖到訪中國醫學科學院實驗動物研究所進行學術交流，左一為秦川，左二為鮑琳琳，右為陳鴻霖

這意味著**中共給予巴里克團隊相關的靈長類動物模型上的研究支持。**

**秦川**

2018年9月， **中國國家醫學科學院實驗動物研究所**與港大P3實驗室簽署合作協議， **成立新發再發傳染病聯合實驗室。**

事實上，二者早已經開始在實驗研究方面進行合作， **像上述那篇合作論文就是在** **2015** **年發布的。**

同樣在2016年，MERS爆發次年， **秦川和袁國勇作為共同通訊作者在《自然微生物學》上發布了** [MERS可以誘導腎臟和肺臟細胞凋亡](https://www.nature.com/articles/nmicrobiol20164)的論文， **陳鴻霖、鄧巍亦是作者之一。**

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/08101933/s_0a9b0a28dd61435da730f9693eda94e7-1.jpg)講者為秦川

秦川一直以來**長期從事人類疾病動物模型研製，**創建了中國**最大的人類疾病動物模型資源庫，**既是中國實驗動物學會的理事長，同時也是**亞洲實驗動物科學聯盟（** **AFLAS** **）的副理事長。**

2020年5月7日， **秦川團隊**聯合其他團隊在**《自然》**雜誌上發表研究論文， **號稱[“建立國際首個新冠動物模型”](https://pubmed.ncbi.nlm.nih.gov/32380511/) 。**

不幸的是，這篇論文**被質疑圖片造假，**對此這篇論文的作者回應表示， **僅僅是“一點小失誤”，“不是蓄意造假”。**

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/08101802/716426256a104c90b8eba05655ea825a-1.png)論文作者回應質疑

而在這篇論文發表的前一天， **秦川團隊在《科學》雜誌上**發表了另外一篇“重磅”研究論文，這篇論文**首次“ [公開報導新冠疫苗動物實驗的研究結果，](https://pubmed.ncbi.nlm.nih.gov/32376603/)**研究表明候選疫苗（PiCoVacc）在恒河猴中具有安全性。”

考慮到**秦川團隊的造假史和中共疫苗的偽劣性，**這篇論文同樣值得懷疑。

這篇論文表示， **候選疫苗能夠誘導中和抗體，**並且這些抗體可以**有效中和** **10** **個代表性的中共病毒毒株，**因此論文得出的結論是**候選疫苗可能對全球流行的中共病毒毒株具有更廣泛的中和能力。**

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/08104802/20200821062804221-1.jpg)質量極差的中共疫苗

事實是，通過路德時評的揭露，我們知道誘導產生的**中和抗體不一定百分百安全，不一定不會對人體產生傷害，**而且閆博士再三提醒弄不清楚病毒真相不可能有疫苗， **更不可能有“廣泛中和能力”的疫苗。**

7月3日，秦川團隊又發表了另一篇論文： **[“初次感染SARS-CoV-2後可防止獼猴再次感染”](https://pubmed.ncbi.nlm.nih.gov/32616673/) ，**這篇論文顯然**暗示不用擔心中共病毒二次感染，**事實卻是8月25日**在香港發現了這樣的二次感染的病例。**

由於中共病毒**變異能力強，**初次感染產生的抗體**不一定對二次感染有效。**

由此可見，秦川作為**中共統管的**研究所所長發表的論文**有替中共病毒吹風、洗地的嫌疑，**而和與**中共代表**密切聯繫的**“外部勢力“**自然是**中共病毒的利益共同體，**正因為存在這樣的利益共同體， **揭示中共病毒才如此艱難、不容易。**
