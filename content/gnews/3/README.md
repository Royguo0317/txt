印度海军表示，周一，法国和印美日澳四方联盟在东印度洋地区启动了复杂的海上演习，以增强几方海军的互操作性。与此同时，中共正共同努力加强其在东印度洋地区的影响力。

印度海军发言人周一表示，演习将展示两国海军之间高度的协作、协调和互操作性。印指挥官说:“印度海军参与演习表明了与友好海军的共同价值观，确保海洋自由，致力于建立一个开放、包容的印度太平洋和基于规则的国际秩序。”

印度军舰“萨特普拉”号和 “吉尔坦”号，以及一架波音P-8I“潜猎”飞机首次参加了演习。法国派出了两栖攻击舰“坦纳瑞”号和护卫舰“苏尔库夫”号，美国海军派出了两栖运输舰“萨默塞特”号，澳大利亚派出了护卫舰“澳新军团”号和油轮“天狼星”号，日本派出了一艘驱逐舰“Akebono”参加演习。

印度、美国、日本和澳大利亚组成了“四方安全联盟”。中共一直对2017年末重启的四方会谈持谨慎态度。自2019年四国将论坛升级为部长级会谈以来，北京方面的怀疑有所增加。

去年11月，印度海军参谋长卡拉姆比尔·辛格(Karambir Singh)表示，一场“大国竞争”正在印度洋地区激烈上演，印度海军已加强监视，以遏制中共野心。

此次四方联盟和法国的联合军事演习是对中共在该地区日益增长的霸权扩张发出的强有力的信号,警告中共在南中国海的扩张策略是不可能复制在印度洋的。同时西方对共同抵御中共霸权主义的威胁达成共识。
