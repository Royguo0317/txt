## 丹麥教授:中共病毒對人體產生的後期影響損害巨大
`2020-07-21 01:27 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/271106/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/21012337/Picture-1-132.png)圖片：哥本哈根首都醫院的首席醫生兼傳染病學教授延斯·倫格倫(Jens Lundgren) 
丹麥廣播公司（DR，Danmarks Radio）新聞網站19日報導了哥本哈根首都醫院的首席醫生兼傳染病學教授延斯·倫格倫(Jens Lundgren)對“中共病毒”可能存在後期影響的新聞。他是丹麥該領域的領導者之一，正在進行一項研究，將在未來的整個五年中觀察病毒對人體的後期影響。他說：“當我們在年初開始首次認識冠狀病毒時，有報導說它只是呼吸道中的一種病毒，在最壞的情況下可能發展為肺炎。但是從那以後，全世界的研究人員變得更加明智，事實證明，這種病毒從長遠來看會造成巨大的損害。”

**患者可能遭受該病毒的後期影響**

根據英國倫敦國王學院的一項大規模研究顯示，在8,000多名曾感染中共病毒的陽性人群中，約有十分之一的患者在不同程度上遭受該疾病的晚期影響。在疾病的實際病程結束後的幾個月中，許多人受到了影響。

**病毒可以危害到全身的每個器官**

延斯·倫格倫說：一月份的時候，我們知道病毒在呼吸道中並可能引起肺炎，但是從歐洲和美國的經驗來看，病毒基本上可以危害到全身的每個器官，無論是中樞神經系統，心臟，腎臟還是手臂和腿部神經炎。

**可能有長期受損的風險**

人受到冠狀病毒的影響非常不同，不僅是患有冠狀病毒的重症患者有長期受損的風險，被感染的輕度患者中也可能發生這種情況。延斯·倫格倫說，有些病症可能持續了很長時間，但病人並未察覺。最初，我們對受病毒感染的患者進行了28天的隨訪，我們得到了患者已經康復的印象，而隨後我們對患者進行三個月後的隨訪發現，問題不是病毒本身，而是病毒對身體的損害。因此，我們延長了隨訪時間，以便能夠理解和分析長期後果。

**啟動五年的隨訪研究項目**

由於三個月不足以給出所有答案（譯者註：丹麥是從3月份開始爆發的）。因此，現在正在啟動一項研究，對患者進行為期五年的跟踪調查，他們希望了解長期後果。該研究項目還將調查藥物的副作用。丹麥目前重症治療中用的主要是兩種藥，一種是瑞德西韋，被用作需要吸氧的患者的標準治療方法。第二種療法是地塞米松，與瑞德西韋不同，地塞米鬆在病重的患者中效果最好。

**輕症患者死於後遺症**

該網站還報導了一位輕症患者後遺症的死亡病例：3月18日，羅亞女士的冠狀病毒檢測結果呈陽性，病情輕微，一個月後她重新回到了翻譯的工作崗位上。從那時起，她的健康狀況出現了不同程度的惡化。 4個月後，她帶著幾種不同的新冠后遺症死去。

羅亞曾經說：“你忘了你有多欣賞自己的嗅覺。你以為無所謂，但四個月來，我什麼都聞不到。吃飯的時候，我可以嚐到酸、咸、甜的味道，但卻分不清各種香料的味道。我的嗅覺非常敏銳，我想念香水、食物和鮮花的味道。我因為後遺症而難以勝任原來的翻譯工作，最嚴重的是記憶力減退、嗅覺下降、疲勞、呼吸困難和頭髮大量脫落。我早上可以感覺很好，但一天中有好幾次我累得只想睡覺。我忘記了許多事情。當我要翻譯的時候，我必須把所有的東西都寫下來。精神上我受到脫髮的強烈影響，脫髮越來越嚴重，梳理頭髮時，頭髮一簇簇地掉，這讓我很鬱悶。我每天都要吸塵好幾次。”

她去看醫生，醫生們也不知道怎麼辦，她被建議服用維生素和轉診給皮膚科醫生。然而，這對於羅亞和其他許多同病相憐的丹麥人來說，並不起作用。所以，兩位本身就受到中共病毒晚期後遺症影響的護士建立了Facebook群組”Covid晚期後遺症患者”，患者可以在群組中互相分享經驗。該小組目前有3725名成員。而延斯·倫格倫教授的女兒路易絲·倫格倫（LouiseLundgren）醫生也在工作中感染上病毒，並且也有後遺症狀。

最近幾個月，研究人員對中共病毒有了更多的了解。而事實證明，許多患者儘管起初症狀較輕，但卻遭受了該病的嚴重後期影響。

**譯評** ：截止至7月20號，世界已經有61萬多人死於中共病毒，1476萬人感染，如果按英國倫敦國王學院的研究結論，1/10的患者在不同程度上遭受該疾病的後期影響，可以推測全球有148萬人正在遭受後遺症的折磨，那麼這其中會有多少人如羅亞女士那樣不幸死去呢？這僅僅只過去6個月，但是時間滴答聲不會停止，死亡的人數還在不斷上升，政客們還在扯皮，華爾街眼裡只有股市曲線而不是死亡曲線，科學家依然在意自己的江湖地位，這世界真的是無比荒謬，而留給人類的時間真的不多了。請放下政治金錢名譽吧，全世界必須團結起來滅掉共產黨，進而找到解決方案！
