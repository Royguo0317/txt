9月23日，中共海關通報挪威進口凍尖吻平鲉1個外包裝樣本檢出新冠病毒核酸陽性，根據海關總署公告，即日起暫停接受挪威GADUS NJORD水產品生產企業產品進口申報1周，期滿後自動恢復。

在此之前，以有許多類似報道出現，海關、商檢、疾控部門也把進口冷凍食品作為CCP病毒查驗的重點。但奇怪的是，幾乎從未有其他國家的類似報道，厄瓜多爾白蝦不僅僅只出口中共國，挪威三文魚也是出口全球各地，為什麽總是中共國的進口凍品頻頻檢出病毒陽性。而且幾乎都是外包裝陽性，食品本身並沒有病毒檢出，很顯然應是冷凍食品在運輸、存儲過程中沾染了病毒。

冷藏貨櫃/集裝箱是否有可能是CCP運毒投毒的工具？冷藏貨櫃非常有利於病毒的長時間儲存，而中共國來往於世界各地的冷藏貨物是最容易通過海關檢查，進入到其他國家的。這就為CCP運毒提供了壹個便捷通道。我們僅僅是壹個猜想，沒有任何事實根據與情報，望讀者自行判斷。
