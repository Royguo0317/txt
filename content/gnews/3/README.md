今天，CNN发表了一篇，揭示了武汉内部的一份官方文件的内容表明中共对Covid-19早期阶段处理不当。

今年的2月10日，当习近平在距离武汉1200公里的安全房里对疫情中丧生的人表示哀悼时，中共当局对外公布的新增Covid-19病例是2478起。然而，一份来自中共内部的文件显示，实际数量远远不止这些。

这份文件是湖北地方的卫生部门首次发布的关于Covid-19的报告，上面标有“内部文件，请保密”的字样。这份报告于2月10日发表，列出了总共5918例新发病例，这个数字是中共官方公布的两倍多，大大低估了疫情的严重性。

从湖北省疾病预防控制中心泄露文件共有117页，2月10日的文件只是其中的一份。这些文件的其真实性已经得到CNN的验证。这些文件是自大流行开始以来从中共国内部泄漏的最重大的信息，这为了解中共当局内部到底何时知道疫情真实状况提供了一个清晰的窗口。

中共一直拒绝美国和其他西方政府关于它们故意隐瞒有关病毒的信息的指控，并坚称他们处理得很及时。然而这些文件揭示了中共当局向外界披露的信息与他们实际掌握的信息不一致的地方。

这些文件是在2019年10月至今年4月之间的发布的。文件内容显示出在疫情初期的几个关键时刻，中共政府和医疗系统明显的失误迹象。该系统受制于自上而下的官僚主义和僵化的程序，这些程序不具备应对新兴危机的能力，导致制度失灵。

一份3月初的报告中写道：从症状发作到确证之间的平均时间为23.3天。专家告诉CNN，这会大大阻碍监测和对抗该疾病的步骤。

CNN已与中国外交部，国家卫生委员会以及湖北省卫生委员会联系，该委员会负责监督省疾控中心，要求对文件中披露的发现发表评论，但未得到回应。
