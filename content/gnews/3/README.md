## 川普国家安全顾问罗伯特·奥布莱恩测试中共冠状病毒检测为阳性
`2020-07-27 15:27 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/277717/)

根据美国CNBC在7月27日上午报导，美国总统唐纳德·川普(Donald Trump)的国家安全顾问罗伯特·奥布莱恩(Robert O‘Brien)对中共冠状病毒的检测呈阳性。白宫周一在一份声明中表示，奥布莱恩的症状轻微，目前在自我隔离并在白宫外的一安全地点工作。白宫接着表示，总统或副总统没有受感染风险，国家安全委员会的工作也没有受到影响。

现年54岁的奥布莱恩是川普核心团队中级别最高的成员之一。事实上，之前其他政府官员也被诊断出感染中共冠状病毒。副总统迈克·彭斯（Mike Pence）的新闻秘书凯蒂·米勒（Katie Miller）5月份对中共冠状病毒检测呈阳性。同月，为川普总统服务三餐的一位贴身助理也被测出阳性反应。总统长子的女友金伯利·吉尔福伊尔(Kimberly Guilfoyle)也在7月份的测试中呈阳性。

白宫表示为了确保总统安全，川普总统以及会与总统和副总统有近距离接触的所有官员都定期接受中共冠状病毒的测试。

报导中最後指出，奥布莱恩本月初曾去欧洲与英国、法国、德国和意大利的官员会面。在美国国家安全委员会官方推特帐户的照片显示，在这次旅行中，他与欧洲政要们的距离非常接近，而欧洲官员们在会议期间似乎都没有戴着口罩。
