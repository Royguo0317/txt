## 中共国二波疫情：CCP病毒变异升级
`2020-05-21 09:01 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/208827/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/21085531/1-131-5.jpg)
![]()
最近几周，中共国在东北两个省发现了一系列新的CCP病毒病例，这引发了人们对第二波病毒热潮的担忧。新的病毒爆发表明病原体可能正在发生变异，患者需要更多时间来恢复并且携带病毒的潜伏期会更长。似乎病毒变异得更加狡猾，该如何来帮助受CCP病毒影响的人们呢？

中共国家卫健委的重症医学专家组邱海波说，吉林和黑龙江两省病例的基因序列跟湖北本土的病例不太一样，多数是与输入相关;武汉病例的潜伏期一般只有一周，最长到两周，但新病例感染后出现第一症状的潜伏期时间更长;临床上，武汉病人一般有发烧迹象，多器官受损，有心肌、肾脏、肠道的损害，东北病例则很少出现发烧，往往以肺的损害为主;武汉的很多病例，一般发病有症状后，一周或两周核酸就转阴了，但输入关联性病例核酸的转阴速度比较慢。

邱海波分析说，潜伏期较长会“引起问题”因为人们有更多的时间传播这种病毒。

“当他们与家人聚会时，他们并不在乎这个问题，我们看到了家庭群体感染。”他说。

在美国，研究人员发现多达14个冠状尖峰突变，这些病毒都以冠状病毒命名。洛斯阿拉莫斯国家实验室的科学家说，这些突变之一可能使病毒更具感染力。英国格拉斯哥大学的科学家进行了另一项病毒突变的研，但没有发现它们是不同病毒毒株的迹象。英国弗朗西斯·克里克研究所（Francis Crick Institute）病毒学负责人乔纳森·斯托伊（Jonathan Stoye）警告说，该病毒是“移动目标”。病毒正在发展并且正在改变，他说：“我们还不知道这些变化会带来什么后果。”

中共国周三报告了五例新的冠状病毒病例，低于一天前的六例。卫健委员会在一份声明中说，新病例中有四宗是本地传播，一宗是来自国外的旅行者进口。邱医生说，在吉林和黑龙江发现的病毒可能是从国外进口的，这可以解释不同的症状。他没有说他认为他们可能来自哪里，而是说吉林和黑龙江与俄罗斯接壤。自从武汉第一波感染以来，北京的官方数字仅显示了一小撮新病例，总共只有82,965例感染和4,634例死亡。但是，人们对该政权数字的准确性普遍持怀疑态度，对于中共官员四处游说反对立即调查该病毒的起源，人们不禁要问：他们想要掩盖什么？这不是一个负责任的大国应有的态度。
