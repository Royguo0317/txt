本周郭文贵先生发布GTV公告：原GTV1300把私募椅子或集体升级为GTV的初始股东。那么两者之间的区别在哪里, 原始股东相较于私募股东收益有何不同呢？

初始股东，顾名思义即是公司在初创时开创团队所持有的股权，其股份所占比依是最多，在企业开始融资后获得的收益最高，抗稀释性也最好。但正因为其封闭性和排他性，能获得初始股权的人数也极为有限。私募股权是一种不在股票市场交易的股权资产，天使投资，杠杆收购，风险投资等都可以算的上一种私募股权，其收益相比初始股权较弱。

• **投资周期以及收益方式**
私募股权投资周期较长，其拥有者若要出售该股权，必须在没有交易市场的情况下找到买家，所以退出投资的方式一般是IPO，出售公司或与其他公司合并，或进行资产重组。在这一点上，初始股权相较于私募的优势即是其成本更低，开创团队往往不需追加资金。

• **融资模式**
私募股权虽然是“私募”，但其针对的还是社会上的投资者，融资亦是通过社会上招募完成的。同时，认股条件，投资者的权利义务等通常已经由公司确定，对参与私募，对投资者一致适用，没有另行协商的余地。
而初始股权的投资者是特定的，而非社会上征募。投资者可以与公司和其他创始股东谈判确定其权利义务。同时公司与投资者通常直接谈判，无需承销商参与。

• **监管**
私募尽管不被认为是股权的公开发行，但由于其针对的是社会上的投资人，因此仍由部分监管机构进行有关监管：例如美国尽管发行私募不需要进行事先登记，但在发行后仍需向SEC报备，部分州还要履行信息披露义务。而初始股权的发行或融资扩股不受相关部门的监管。

因此，初始股权相较于私募椅子便有了三大优势：成本低，个性化，重隐私。
