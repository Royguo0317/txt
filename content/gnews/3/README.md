04月28日，美国国务卿安东尼·布林肯在接受美国有限电视新闻网（CNN）采访时说，如果俄罗斯领导人放弃其侵略性政策，美国和俄罗斯可以合作。该新闻在国务院网站上公布。

国务卿指出，华盛顿希望与莫斯科建立更稳定的关系，但后者的命运取决于俄罗斯总统弗拉基米尔·普京。

“如果他拒绝升级，那么我认为会有一些领域，我们可以为了我们的共同利益而合作。例如，在战略稳定领域。”布林肯说。

例如，国务卿引用了今年1月《削减和限制战略进攻性武器条约》的延期。

早些时候，俄罗斯副外长谢尔盖·里亚布科夫（Sergei Ryabkov）在俄联邦委员会发言时说，莫斯科已向美国总统乔·拜登的政府提议，根据俄罗斯领导人在俄联邦议会的讲话，启动关于战略稳定的对话。“我想指出的是，我们已经提议拜登政府按照弗拉基米尔·普京所说的启动关于战略稳定的对话，这意味着要考虑这一领域的关切并寻求解决现有问题的方法。”这位副部长当时说。

3月17日，拜登对他是否认为普京是杀人犯的问题做出了肯定的回答。俄罗斯驻华盛顿大使随后被召回莫斯科进行磋商。克里姆林宫批评了美国领导人的声明，并指责美国不愿意修补关系。3月下旬，俄罗斯总统发言人德米特里·佩斯科夫说，该国当局正在修订与美国的关系。

4月15日，美国进一步实施反俄制裁，指责俄罗斯干涉美国选举，参与对美国基础设施的黑客攻击，侵犯其他国家的领土完整等。新措施包括禁止美国金融机构参与俄罗斯财政部和中央银行债券的首次公开发行。

然而，在围绕乌克兰的局势不断升级的情况下，拜登给普京打电话，建议他们未来几个月在“第三国”会面，讨论两国关系中的所有问题。“拜登总统重申他希望与俄罗斯建立稳定和可预测的关系。”白宫当时说。

在4月22日的气候峰会上，拜登赞扬了普京在峰会上的发言。

“我对普京总统昨天呼吁就消除二氧化碳进行合作感到非常鼓舞。美国期待着与俄罗斯和其他国家在这个问题上进行合作。这是非常有希望的。”美国总统在第二天说。他称这次峰会是“一个伟大旅程的开始”，并指出已经取得的进展。
