## 郭先生8.29寄語洛杉磯總領館:你們一個都跑不了,不要禍延下一代

作者：Diago

郭文貴先生在2020年8月29日與洛杉磯戰友連線時提到——

【連線戰友：謝謝文貴先生，我們還有最後一個問題，現在我們就站在中共國駐洛杉磯總領館的對面，您會看到對面的大的辦公樓，今天雖然是禮拜六，我相信一定有人通過攝像頭來觀看我們的遊行卻不敢出來，來跟我們對質，我想請郭先生對裡面的中共國的走狗們說幾句話。

郭文貴先生：加州啊，無論是洛杉磯還是三藩市，可以說，你們有沒有想過，我在過去直播中說過話，我當年在矽谷住過幾個月，我經常去三藩市去吃東西，然後我在那有個太乙真人的廟，是在一個三藩市的華人區的樓上，我當時到那兒的時候我抽過一個簽，第一個簽就是你千萬不能動，你要到哪就完啦，如何如何；然後第二個簽就是說我要注意身體安全，然後發生什麼事兒我也跟你說過；第三個我抽得叫蘇武放羊，就我可以回家了。然後我就那年就是1999年我就回到了國內，當時是賴昌星、姬勝德那個案子非常非常大，是江澤民親批是讓我回到了國內，我在三藩市那個太乙真人那裡經歷了傳奇，那不是迷信，我現在在我的錢夾裡邊那三個簽兒我還都留著呢，那是個傳奇的地方，那麼當時三藩市領事館，這個地方包括洛杉磯領事館是同一個風水先生看的風水，這位風水先生就是太乙真人廟旁邊給我解簽的那個人，他說你記住，他說郭先生，這個三藩市的這個領事館一定會成為共產黨、成為最大的麻煩之一，我說為什麼，他說三藩市領事館就駐紮在三藩市的一個不穩三邪之地、是一個斷頭之地、是萬箭穿心之地，我今天告訴三藩市領事館所有的人，你們都是我們的同胞，都是文貴的同胞，我們與你無仇無怨，相反你們都是被共產黨植入了人屍丸的人，我不想說那個什麼放下屠刀啊、立地成佛啊，你不用！你聽著我們這些同胞講話的時候，你記住！我們不希望、也不想拯救你們，但我們想拯救你的後代，就象我對面我們站著的這些孩子一樣，我們不希望你的孩子再成為你和成為你這樣的人，因為你們太醜陋了、因為你們已經不值得救了，我們也不想救你們，但希望你們的孩子不要再受你們的禍害了，孩子沒有義務來說服爺爺奶奶、說服爸爸媽媽去知道真相、去尋找正義，但是你們作為父母的、兄弟姐妹的，應該告訴自己的孩子，遠離自己的魔道、遠離自己的邪道，讓自己的孩子遠離現在你們所處的兇險之境，你那不是三邪之地、也不是斷頭之地，就象洛杉磯的一樣，你們這一個風水先生已經早就註定了你們的命運，下一個就會是你三藩市領事館，而且你三藩市領事館你知道發生了什麼事情，頭兩天休斯頓的人就是撤到了洛杉磯和撤到了三藩市，而且你們在幹的事情，美國的FBI和情報部門和國土安全部完全掌握，包括你們在三藩市建立的監聽、偷盜、駭客的基地，我今天公開說出來，你們一個都跑不了，但是我希望你不要再禍延你的下一代了，兄弟姐妹們，我們不用希望對面這幫人他會做什麼好事兒，我們也不在乎他做什麼好事兒，也不要怕他做什麼壞事兒，在爆料革命、新中國聯邦誕生之日起，他們已經什麼都不是了，他們敢站起來面對你們嗎？他敢說一句同胞的話嗎？他敢說一句真話嗎？他們連豬狗都不如！而且在西方這個國家，他們很可憐！很可悲！他們什麼都沒有啦！剩下的一百多斤那種爛肉、而且經常不健身，啥都沒了，可憐可憐他們吧！放過他們吧！忘了他們吧！就當個屁把他們放了得啦！兄弟姐妹們，不要在乎他們、不要浪費你們的美好的未來和青春年華，謝謝！
