据报道，特朗普总统对其亲信圈内的人说，他计划在离开白宫之前赦免他的前国家安全顾问迈克尔.弗林(Michael Flynn)。Axios援引两位消息人士的话报道了这一消息。

弗林在2016年总统竞选期间担任特朗普的高级顾问。

2017年1月22日，弗林宣誓就任国家安全顾问。

2017年2月13日，他在与俄罗斯驻美国大使谢尔盖.基斯利亚克的通信性质和内容上误导副总统迈克.彭斯的信息浮出水面后辞职。弗林作为国家安全顾问的任期是该职位历史上最短的。

2017年12月，弗林与特别检察官罗伯特.穆勒正式达成协议，承认 “故意和明知 “向FBI作虚假陈述的重罪，并同意配合特别检察官的调查。

2020年1月，弗林以政府报复和违反认罪协议为由，提出撤回认罪。美国司法部宣布，打算在2020年5月7日撤销对弗林的所有指控。联邦地区法官Emmet G. Sullivan裁定此事搁置。

AIXO报道进一步指出，弗林的赦免将是四年政治和法律闹剧的高潮，该闹剧始于联邦调查局对特朗普竞选活动与俄罗斯政府在2016年大选中可能合作的调查。这位退休的中将被许多特朗普的支持者视为奥巴马政府政治报复的受害者。而弗林的律师和保守派媒体成员则指责FBI诱捕他，并称他的案件是更广泛的”通俄门“抹黑调查的一部分。

今年早些时候，特朗普对在穆勒调查中被指控的另一名助手罗杰.斯通进行了减刑，总统为其申诉说他在政治迫害中受到不公正对待。 我们知道弗林将军是坚定灭共的美国高层之一，也正是因为如此收到了陷害。现在我们看到川普总统在这个时刻赦免弗林将军，意义可想而知。不管左媒怎么否认，我们都知道川普总统早已赢得大选，现在要做的就是把大选夺回来，而夺回大选就必须灭共。目前调查大选舞弊的民意已逐渐形成，于是就在这个时候就传出了要赦免弗林将军，毫无疑问其实目的就是为了针对中共而来。
