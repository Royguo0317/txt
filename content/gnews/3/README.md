无国界妇女权利组织正在敦促奥委会将2022年冬奥会从中共国移出，否则将面临抵制 “种族灭绝运动会 “的风险。

无国界妇女权利组织准备了一封致美国奥林匹克委员会（USOC）的公开信，并抄送国际奥林匹克委员会（IOC），抗议2022年冬奥会的场地选择。

无国界妇女权利组织主席雷吉-利特约翰对Breitbart新闻说，除了正在进行的对维吾尔人的种族灭绝，中共还犯下了许多 “其他人权暴行，如宗教迫害、强迫堕胎和绝育、强迫摘取器官、强迫劳动和大规模监控”。

小约翰说，中共还参与了 “拘留异议人士和举报人、无所不用其极的环境掠夺，以及投向全世界而发动当冠状病毒大流行”，使其成为世界上最重要的人权侵犯者之一。

在信中，作者提醒美国奥委会和国际奥委会 “章程承诺和与1948年《防止及惩治灭绝种族罪公约》有关的国际义务”。

信中提醒奥林匹克当局注意1936年在德国柏林举行的类似的奥运会，”事实证明，这对阿道夫-希特勒的第三帝国来说是一个巨大的宣传意外之财，使其获得了不正当的国际声望、合法性和可信度。”

“这些好处为纳粹政权后来对600万犹太人的种族灭绝大屠杀和希特勒引发的世界大战过程中所犯下的可怕的反人类罪行提供了烟幕弹。”

信中继续写道：”今天，我们面对的是另一个极权主义政权，在危害人类罪中，积极从事另一种种族灭绝。然而，奥运当局却准备让中共 接受和利用一场宣传盛宴，这将使纳粹所享受到的东西相形见绌”。

作者还提醒奥委会，美国国务院已经宣布，中共正在对新疆的维吾尔族和其他少数民族进行种族灭绝。

信中指出：”这些受害者中至少有一百万人被关押在几十个集中营中，有些集中营中充斥着火葬场，他们在那里被洗脑、被强奸、被强行堕胎和绝育、被折磨、被摘取器官并被迫从事奴役劳动”。

“美国奥委会能接受吗？”作者问道。”你的组织是否愿意与这种野蛮行为有关联，更不用说被视为纵容？”

信中最后呼吁 “国际社会紧急努力，将2022年冬奥会迁移到另一个场地，为种族灭绝奥运会提供一个自由奥运会的替代方案。”

“如果做不到这一点，请你们注意，我们将不惜一切努力抵制奥运会。”信中宣称。

美国务院一再确认中共在新疆犯下的种族灭绝和反人类罪，以及美国左右媒体和民意对中共病毒的定性和追责的推进，中共想在2022年成功举办冬奥会的希望越来越渺茫。可以想象等待中共的将不是万邦来朝的2008年夏季奥运会的盛况再现，而是其很快被永远钉在人类历史的耻辱柱上。
