黑白颠倒、指鹿为马是中共一贯用于舆论洗脑的伎俩，新疆种族灭绝罪这记重锤触痛了中共的神经，新一轮洗脑舆论战近日在中共国主流媒体开启。

新华社北京（国际观察）4月11日发出了题目为：美国侵犯人权五宗罪之一：殖民主义罪孽深重的文章，文章讽刺美国自诩“人权楷模”，扣帽子说美国孜孜不倦炮制针对别国的人权谎言，对本国的人权劣迹却选择性无视。污蔑美国在上百年的时间里，为掠夺土地和资源，屠杀北美大陆原住民——印第安人，文章抽取美国在发展历程中与印第安人在特定历史条件与因素下的冲突事件，断章取义、浓墨重彩，用屠刀霍霍、煽动丑化、血泪之路等等充满杀机的词汇反过来污蔑美国的人权状况，甚至污蔑美国政府推行“美国化”教育，瓦解印第安部落社会和传统文化。

在笔者看来，这根本就是中共新疆种族灭绝罪的经验之谈，中共正是采用文章中历数的一系列手段，对新疆维吾尔族人民进行了无情的掠夺与屠杀，其新疆再教育集中营相关的证据已经越来越多的大白于天下，越来越多的国家与企业加入抵制新疆棉，中共国的这些舆论伎俩只能欺骗大陆被多年洗脑，失去了是非判断能力的民众，中共的一系列欺骗性舆论只是为自己的罪行提供更多更新的证据。

笔者发现，中共国各大网站及媒体关于印第安人的内容均被该类虚假信息覆盖与取代，百度百科中的也换成了该文章提到的内容。
