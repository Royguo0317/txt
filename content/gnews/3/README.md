## 【快評】推倒「防火牆」迫在眉睫！中共網信辦再關8868家「違法」網站！
`2020-10-20 05:45 shangshang` [轉載自GNews](https://gnews.org/zh-hant/435576/)

內新聞/素材：小黑姐（文雅）  校對：雅典娜的聖鬥士(沙加)
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/20053518/FB772DC6-5022-41EB-89AD-6A7494D1D55A.png)
中央廣播電台央視新聞消息：從國家網信辦獲悉，今年第三季度，全國網信系統依法查處網上各類違法違規行為，累計約談網站1211家，警告網站954家，暫停更新網站489家，會同電信主管部門取消違法網站許可或備案、關閉違法網站8868家。

自2002年起，中共國開始使用防火長城對大量網站進行嚴格的網絡審查。當時電腦在大陸還沒有普及，上網的人相對較少，但是網絡審查和控制卻已悄悄開始。

特別是2017年開始，網絡審查和控制越來越嚴苛。下面是從2017到2020年中共網信辦審查和關閉網站的數據：

2017年，全國網信系統約談網站2003家，關閉22587家；

2018年，全國網信系統約談網站1497家，關閉6417家；

2019年，全國網信系統約談網站2767家，關閉11767家；

2020年，全國網信系統約談網站1211家，關閉8868家。

中共在2017年宣佈，所有VPN應用的開發商都需要獲得政府許可證才能在大陸使用，到8月份蘋果公司下架中國區超60個VPN，並嚴格控制虛擬專用網絡VPN市場，進一步縮緊牆內人民獲取自由世界信息的通道。

美國人權組織，自由之家，公佈2020年度網絡自由報告，在全球65個國家的網絡自由排名中，中國連續第六年倒數第一，被列為網絡自由最差的國家。報告指出，新冠病毒疫情期間，中國採取了全球最嚴格的網絡控制手段。今年3月封城期間，武漢和湖北一些地區的網絡出現連接受限。今年6月，中國新《網絡安全審查辦法》推出，嚴格審查通訊、金融、運輸、銀行、郵政等基礎設施網絡的關鍵信息。中共打著國家安全、保護國家利益的幌子，從各方面全面審查互聯網，關閉本地互聯網、限制虛擬專用網絡VPN、刪除網絡敏感信息、過濾敏感詞、關閉賬戶、永久封號、逮捕網絡用戶等，限制國外各大產品和服務進入。至今無法登入使用臉書、推特、youtube、Instagram、Line等全球流行的社交媒體和信息平台。

隨著輿論、網絡愈加嚴格的控制，牆內人們獲得外界信息和發聲的渠道愈加狹窄，牆內人民看不到聽不到自由世界真實的信息，外界也無法正常獲得牆內真實的信息。這種人為製造的無形銅牆鐵壁，把中國人民和世界人民隔絕開來，利用信息的不對稱，欺騙洗腦牆內人民和全世界。只要防火牆還在，真相就不能傳遞，正義就無法伸張，自由就不可能擁有。
