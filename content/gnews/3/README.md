## 爆料革命的戰略轉變和集會遊行的新標竿
`2020-08-30 23:06 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/325802/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/30230020/1-138.png)
作者：半山搖

三年多的時間，由郭文貴先生發起並領導的爆料革命已然成為這個時代的潮流，參加爆料革命成了熱愛民主自由的中國人最大的時尚。

2020年6月4日具有里程碑的意義，《新中國聯邦宣言》向全世界大聲宣告，勤勞善良的中國人已經準備好要建立一個沒有共產黨的新中國。

2020年7月27日，新中國聯邦組織了首次全球聯動的集會遊行並走上街頭，標誌著爆料革命的戰略轉變，勢必掀起新的高潮並影響新一輪的世界範圍的顏色革命。

首次聯動包括美國，英國，法國，義大利，日本，澳大利亞，新西蘭等國家。各國爆料革命的參與者們幾乎同一時間在各地的中共領館前集會，揭露中共隱瞞並惡意傳播病毒的真相，希望“中共撒謊，人們死亡”的現實能夠喚醒更多的民眾。

一個多月後的8月29日，日本大阪，名古屋等地再次把走上街頭的集會遊行提升到新的高度，並得到德國慕尼黑以及台灣等地的響應。特別值得一提的是，郭文貴先生專門錄製了一首名為 “Take Down CCP” 的單曲，節奏強勁，鏗鏘有力，配合遊行活動堪稱完美。

在美國波特蘭等地的遊行示威陷入暴力騷亂甚至失控的當下，新中國聯邦樹立了新的集會遊行的標竿。

1. 專業的組織能力

各地都有專門的團隊負責各項事務，確保配合當地警方的安排，保證參與者的安全，按既定的時間，路線合法有序地進行。2統一的集體形象

世界各地的參與者們都有統一的新中國聯邦的形象和面貌出現，從著裝到口號到傳播資料都體現了專業的形象和精神面貌。

2. 清晰的遊行訴求

揭露病毒真相，捍衛普世價值，反共滅共，聲援香港等是集會遊行的清晰訴求。旨在喚醒民眾，讓自由世界聯合起來，採取立即的行動。

3. 和平有序的遊行

每一次遊行都是和平的，有序的進行，而且遊行結束後從來不會留下任何垃圾或廢棄物，得到當地警察及管理部門的讚許。

4. 深遠的國際影響

每一次遊行都向全世界直播，其影響將超乎想象。爆料革命新一輪週期性的集會遊行正在寫進21世紀民主自由抗爭的歷史。

爆料革命先是以社交媒體為陣地，以曝光事實真相為起點，到如今從雲端降落到地面，並在世界範圍內遍地開花。這勢不可擋的爭取民主自由的巨浪正一波一波地衝擊著中共搖搖欲墜的政權，全世界聯合起來對抗中共已經成了共識，中共非法政權隨時都會崩潰。
