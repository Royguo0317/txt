# **鲍里斯·约翰逊的高级助手陷入利益冲突纠纷：首相策略师“是帮助中共国购买伦敦新使馆的地产公司的顾问，同时也是以2.55亿英镑出售土地的开发商的顾问”**

乌德尼·李斯特勋爵(Lord Udny-Lister)代表外交部与中共国就大使馆问题会谈。他还曾是房地产巨头世邦魏理仕的付费顾问，这家公司出面帮忙购买下了这栋大楼。乌德尼·李斯特勋爵也是拥有该物业的德兰士(Delancey)的付费顾问。两家公司都坚称，他没有代表他们直接参与这笔交易。

鲍里斯·约翰逊的一位高级助手，今天因利益冲突而陷入困境。他声称他代表政府帮助促成了中共国新使馆的交易，但他同时还得到了两家相关公司的报酬。

[!\[\]()!\[\](https://spark.adobe.com/page/cljopiwa6J7Nc/images/de405f53-e7f1-4760-983c-e1363584a287.jpg?asset_id=7fdb1a58-2701-494a-aa09-6bbff3fb265d&amp;img_etag=%2241a9b9f2cbd75147a7e9c0c295895741%22&amp;size=2560)](https://spark.adobe.com/page/cljopiwa6J7Nc/images/de405f53-e7f1-4760-983c-e1363584a287.jpg?asset_id=7fdb1a58-2701-494a-aa09-6bbff3fb265d&amp;img_etag=%2241a9b9f2cbd75147a7e9c0c295895741%22&amp;size=1024)

乌德尼·李斯特勋爵帮助外交部，与中共国就其2.55亿英镑收购伦敦塔附近的皇家铸币厂的交易进行了谈判。

但据《星期日泰晤士报》报道，2017至2018年从事交易谈判期间，曾为爵士的乌德尼·李斯特，还为参与交易的两家公司工作。

[!\[\]()!\[\](https://spark.adobe.com/page/cljopiwa6J7Nc/images/76e32afc-4f22-4aa5-967e-1c92ddacd66e.jpg?asset_id=002eb6f0-868f-4d92-a4c7-441f0098c0e5&amp;img_etag=%22cb027c1fc22feda83f14ff7f011cb4b5%22&amp;size=2560)](https://spark.adobe.com/page/cljopiwa6J7Nc/images/76e32afc-4f22-4aa5-967e-1c92ddacd66e.jpg?asset_id=002eb6f0-868f-4d92-a4c7-441f0098c0e5&amp;img_etag=%22cb027c1fc22feda83f14ff7f011cb4b5%22&amp;size=1024)

现年71岁的李斯特勋爵，是约翰逊首相在唐宁街的首席战略顾问，也曾担任美国商业房地产巨头世邦魏理仕（CBRE）的付费顾问。

该公司受雇于中共国，并为其新的大使馆物色和购买场地。

据《星期日泰晤士报》报道，在谈判进行期间，他还是伦敦房地产公司德兰西（Deancey）的付费顾问，而该公司拥有皇家铸币厂（Royal Mint Court）。

两家公司都告诉该报，乌德尼·李斯特勋爵没有直接参与谈判。

世邦魏理仕的发言人告诉MailOnline：“乌德尼·李斯特爵士在皇家铸币厂的交易中，与世邦魏理仕没有任何关系。”

乌德尼·李斯特勋爵牵头帮助英国外交部，与中共国就2.55亿英镑收购伦敦塔附近的皇家铸币厂的交易进行谈判。图为他与中共国大使刘晓明的合影。

现年71岁的李斯特勋爵是约翰逊先生在唐宁街的首席战略顾问。他曾于美国商业地产巨头世邦魏理仕担任付费顾问。这是约翰逊先生担任伦敦市长时他们的合影。

[!\[\]()!\[\](https://spark.adobe.com/page/cljopiwa6J7Nc/images/80288040-3fd6-45b4-96b2-9414e4573dec.jpg?asset_id=b1b9f682-f1f1-472a-a195-f6131c3fccc6&amp;img_etag=%22a316ba817d53b7a5eecb285735469634%22&amp;size=2560)](https://spark.adobe.com/page/cljopiwa6J7Nc/images/80288040-3fd6-45b4-96b2-9414e4573dec.jpg?asset_id=b1b9f682-f1f1-472a-a195-f6131c3fccc6&amp;img_etag=%22a316ba817d53b7a5eecb285735469634%22&amp;size=1024)

历史悠久的皇家铸币厂大楼位于伦敦塔对面，是前皇家铸币厂的所在地，将改建为新的中共国大使馆。

“他在2016年10月至2017年12月期间，被聘为世邦魏理仕的顾问。”

政府发言人告诉MailOnline：“乌德尼·李斯特爵士应政府要求，作为英国外交和联邦事务办公室(FCDO)的非执行董事承担了这项工作。其中不存在利益冲突。”

2010年，德兰西以5100万英镑的价格从英国女王公共地产的背后公司——皇冠地产(Crown Estate)购买了该地块。

这座历史悠久的建筑位于伦敦塔对面，是前皇家铸币厂的所在地。

该建筑于2018年被中共国政府以2.5亿英镑的价格收购。现在计划成为中共国大使馆的新址，并将成为世界上最大的大使馆之一。

据报道，在提出要约之前，德兰西曾计划将该地块开发成办公室，并且没有公开挂牌出售。

在交易期间，乌德尼·李斯特勋爵是FCDO的非执行董事。他被当时的外交大臣鲍里斯·约翰逊任命主持谈判。

中共国大使刘晓明称赞他为确保达成交易的个人“努力”。而李斯特也在去年获得贵族头衔。

虽然政府坚持认为乌德尼·李斯特勋爵的行为符合规则，但公共生活标准委员会前主席葛兰姆阿利斯泰尔·葛兰姆爵士(Sir Alistair Graham)仍要求进行调查。

他告诉《星期日泰晤士报》：“外交部必须调查他是否满足公共生活的七项原则。”

就在上月，中共国建造其巨大大使馆的计划，引发了人们的对其可能挖掘出数千具黑死病受害者尸体的担忧。

这座建于1809年的建筑，坐落在一个名为St Mary Graces的古老西多会(Cistercian)修道院中。

就在上月，中共国计划建立庞大的使馆（此处图片：有关现有建筑和未来新开发建筑的计划图）由于可能会挖掘出数千具黑死病受害者尸体，人们对该计划表示担忧。这座建于1809年的建筑，坐落在一座名为St Mary Graces的古老西多会修道院遗址之上。

[!\[\]()!\[\](https://spark.adobe.com/page/cljopiwa6J7Nc/images/d7ffabfa-e20a-4963-bd62-c1d406c1e8f0.jpg?asset_id=aa29d5da-786a-4198-a188-bbb2cffb30f6&amp;img_etag=%22029124dbf8c3dcad1f478c1bc190fb03%22&amp;size=2560)](https://spark.adobe.com/page/cljopiwa6J7Nc/images/d7ffabfa-e20a-4963-bd62-c1d406c1e8f0.jpg?asset_id=aa29d5da-786a-4198-a188-bbb2cffb30f6&amp;img_etag=%22029124dbf8c3dcad1f478c1bc190fb03%22&amp;size=1024)

皇家铸币厂从伦敦塔搬到了位于Tower Hill的新址。新址由罗伯特•史密克爵士(Sir Robert Smirke)设计。1810年，这里制造了第一枚硬币。1968年，铸币厂迁至南威尔士(South Wales)。

那里有一个墓地，其中满是14世纪黑死病席卷欧洲大陆时，因疾病逝去的人的坟墓。

1986到1988年间，人们从这个大规模墓地中发现了多达762具尸体。

塔哈姆莱茨区(Tower Hamlets)议员担心，在地下室建造地下房间时，该项目可能会干扰墓地，并破坏历史建筑的地基。

保守党议员彼得·戈尔德(Cllr Peter Golds)呼吁保护历史建筑的政府机构Historic England对该地块进行全面调查和保护。

他说：“这是一处具有重要历史意义的遗址，也是在一场大瘟疫中丧生的数百人甚至数千人的最后安息地。”

大使馆本身需要大量的建筑。如同所有大使馆一样，它拥有很深且安全的地窖。

[!\[\]()!\[\](https://spark.adobe.com/page/cljopiwa6J7Nc/images/5bdbaba2-0d7c-41a5-bf22-d9f2662bd298.jpg?asset_id=80cee2af-7643-47a4-954e-2c526ebbfc1c&amp;img_etag=%22b3723533d278f75474027f26b0106f9d%22&amp;size=2987)](https://spark.adobe.com/page/cljopiwa6J7Nc/images/5bdbaba2-0d7c-41a5-bf22-d9f2662bd298.jpg?asset_id=80cee2af-7643-47a4-954e-2c526ebbfc1c&amp;img_etag=%22b3723533d278f75474027f26b0106f9d%22&amp;size=1024)图：中共国驻英国大使刘晓明。中共国政府于2018年以2.5亿英镑的价格购买了该地块，并希望对其进行翻新，以创造出“欢迎公众的中国形象”。而这一行为因中共国在香港的人权记录一直面临着相当大的批评。

我所担心的是，这个遗址上可能有着英格兰最后建立的西多会修道院的地基和潜在文物。

此外，这里有着黑死病受害者的未受打扰的埋葬地。修道院教堂本身很可能正是跨过墓地而建。

“我希望进行问卷调查和考察，看看是否会出现任何阻止墓地挖掘的意见。如果有这样的提议，我们应采取措施，谨慎且有尊严地移走遗体。”

戈尔德议员补充说，1986至1988年间，对该遗址的挖掘工作，在三条大规模墓地中发现了762具尸体。并且修道院的地基仍然可见。

中共国政府于2018年以2.5亿英镑的价格买下了该地块，并希望对其进行翻修，以打造“欢迎公众的中国形象”。而由于中共国在香港的人权记录，以及对待维吾尔穆斯林的态度，这一行为面领着广泛批评。

计划显示，业主希望建造办公空间和员工宿舍，而这里距离他们目前马里波恩(Marylebone)的家只有4英里路程。

最近该行政区的考古审查将该遗址列入与伦敦塔同样的重要级别，即一级保护建筑。

地面建筑非常宏伟，前皇家铸币厂建筑本身也被列为二级保护建筑。

英格兰历史学家强调，这个巨大遗址对考古学“非常重要”。前皇家铸币厂建筑群下面有着“层层历史”，包括14世纪修道院的遗迹，以及更早时期用作首都黑死病受害者的墓坑。

一位发言人说：“我们正在与潜在申请者联系，并强调了保护完整的修道院遗迹，确定并严格限制对现存中世纪墓葬的任何影响，以及进行彻底考古调查和记录的重要性。”

“对该地块的规划处于预申请阶段。随着新大使馆计划的制定，我们将继续就该地块对地面及地下关键历史元素的影响提出建议，以最大限度地减少伤害，并最大程度地了解该地块的丰富历史。”

皇家铸币厂，又名约翰逊·斯米尔克(Johnson Smirke)大楼，建于1809年，矗立在1350年由国王爱德华三世创建的前西多会修道院的遗址上。
