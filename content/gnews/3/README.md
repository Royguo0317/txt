## 美国企业必须执行禁令与中共国科技公司脱钩

![]()![](https://gnews.org/wp-content/uploads/2021/04/capture-17-5.jpg)图片来源：华尔街邮报
据雅虎新闻4月17日报道，从五月份开始，在美公司须遵守美国政府对中共国和其他敌对国的信息科技产品和服务的使用禁令。据日本经济新闻报道，这条禁令影响到450万家公司。相关规定在三月份已经准备就绪，美国政府对这类售卖和使用科技产品公司进行审查，并阻止任何风险过大的交易，目的在于避免泄露任何敏感信息。

这些规定适用于一系列领域的科技产品，包括用于关键基建和电信网络的软硬件、以及人工智能和量子计算机。这个禁令清单包括个人信息处理服务和监控设备，例如网络监控摄像头、传感器和无人机。这些规定影响所有在美国运行的民营公司。据日本经济新闻报道，商务局估计，美国600多万个公司包括海外的分支公司，里面有四分之三使用外国科技产品。美国商务局正准备给这些公司提供执照或者提前备案从而减少公司损失。

美国政府之前已经制裁了中共国互联网公司，不过这一次力度更大更全面。早在去年，美国政府的投标项目禁止了5家与中共国有关系的公司参与，其中包括华为、中兴和海康威视。但是，新的禁令涉及面更广，不仅仅是针对中共国公司，也针对其他敌对国家的公司，包括俄罗斯、朝鲜、伊朗、委内瑞拉和古巴。但是，中共国的公司将在黑名单里占多数。

如果公司和企业想不受制于禁令，就需要提交所有有问题的互联网设备和服务器的信息，以确保他们不会成为“不当或不可接受的风险”。公司和企业有权反对审查结果，或采取措施降低风险以通过审查。但是一旦不遵守禁令或缓和协议，将会面临民事和刑事处罚。

美国商务部在3月中旬和本周二向中共国公司发出了传票，要求这些公司提供其在美国业务的相关信息，以 “支持对交易的审查”。
