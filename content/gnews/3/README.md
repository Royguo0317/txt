##  **參議院動議禁止政府發放的設備使用抖音（TikTok）** 

[!\[\](https://spark.adobe.com/page/15Gi9bLTGXdxF/images/dee12bd9-02c8-4b03-afe3-28a83c2d7ca9.jpg?asset_id=a5811172-125e-4a23-9084-45a7c7e23cb9&amp;img_etag=%22b6cd181069a25525e4c357cd08b52bc3%22&amp;size=2560)](https://spark.adobe.com/page/15Gi9bLTGXdxF/images/dee12bd9-02c8-4b03-afe3-28a83c2d7ca9.jpg?asset_id=a5811172-125e-4a23-9084-45a7c7e23cb9&amp;img_etag=%22b6cd181069a25525e4c357cd08b52bc3%22&amp;size=1024) Getty Images 蓋帝圖片

一個參議院委員會動議禁止在美國政府設備使用抖音（TikTok）。

美國參議院國土安全和政府事務委員會一致通過了一項由密蘇里州共和黨參議員喬什·霍利(Josh Hawley)提出的法案，該法案將禁止在政府設備上使用這款中共國社交媒體應用。該法案現在將由參議院進行投票表決。

霍利在一份聲明中說：“抖音由一家中共國的公司持有，該公司董事會中有中共黨員，根據中共國法律規定，抖音必須與中共國政府共享用戶數據。正如我們的許多聯邦機構已經認識到的那樣，抖音對美國來說是一個的重大安全風險，不能在政府設備上使用。”

隨著美國和中共國之間的競爭日益激烈，這款應用的潛在國家安全風險一直是人們談論的話題。科羅拉多州共和黨眾議員肯·巴克提議，價值7,410億美元的《國防授權法案》禁止抖音在政府設備上使用。該修正案以336票對71票通過。

本月早些時候，《華盛頓自由燈塔報》有報導稱，該應用程序在華盛頓的民主黨人中間的影響力越來越大。眾議院多數黨黨鞭詹姆斯·克萊伯恩(南卡羅來納州民主黨人)和眾議院議長南希·佩洛西(加利福尼亞州民主黨人)都在遊說反對限制該應用程序。

抖音過去曾面臨聯邦監管。 2019年2月，該應用因非法收集13歲以下用戶的數據而被處以570萬美元的罰款。

抖音的一份官方聲明堅稱，遵守美國國家安全準則是該應用程序的首要任務。該公司的一位女發言人說：“數以百萬計的美國家庭將抖音用於娛樂和創意表達，但我們認識到這不是聯邦政府設備的用途。”
