*文章認為滬深300的市盈率遠低於標普500，中共國對病毒的控制、經濟狀況好於其他大國，因此對股市持續上漲感到樂觀。這些觀點本人完全不能贊同。首先，中共掩蓋了因病毒而死亡的數字真相；其次，中共國的經濟數據完全造假，有人評估GDP早已是負值，今年連GDP增長目標都沒有設定（可能覺得實在編不下去了）；最後，滬深300裡面有多少是黨企、國企、盜國賊家族企業，是靠壟斷14億人的市場發財的，只要打開國門，馬上就被同行企業競爭下去，再加上財務造假等問題，現在的估值已經擠滿了泡沫，隨時可能破滅。*

##  **股市熱潮再次席捲中共國，這次能持續嗎？** 



**我們有理由感到樂觀，但也有非理性繁榮的跡象**
 [!\[\](https://spark.adobe.com/page/jgu69zJvMzuRt/images/2c6f51be-edf8-4e5b-a6d9-b87a6157b3a1.jpg?asset_id=3ca05e22-4b67-4b26-aee4-0c4437c358c8&amp;img_etag=%22399f6289c6afb36183f28613bb967c4e%22&amp;size=1024)](https://spark.adobe.com/page/jgu69zJvMzuRt/images/2c6f51be-edf8-4e5b-a6d9-b87a6157b3a1.jpg?asset_id=3ca05e22-4b67-4b26-aee4-0c4437c358c8&amp;img_etag=%22399f6289c6afb36183f28613bb967c4e%22&amp;size=1024) 
在中共國股市神話中，最罕見的猛獸是慢牛。過去幾十年出現了兩次快速牛市:股價陡峭的飆升，但都沒有持續超過一年。當股市崩盤時，這些現像很快就導致了“快速熊市”的出現，最終，隨著下跌變得更加平緩，市場轉為慢速熊市。大多數時候，隨著市場橫盤整理，會出現所謂的“長蟲”現象，例如滬深300指數，衡量中共國市值最大的股票，過去5年的平均水平與2007年首次達到的水平相同。緩慢的牛市——美國投資者所熟知的一種穩定的、幾乎是可靠的、年復一年的增長——仍然難以捉摸。

中共國不屈不撓的賭客們現在希望，慢牛終於來了。過去一周，股市上漲了15%，較3月份的低點上漲了30%以上。但一些人認為，本輪牛市將比以往更加持久。

首先，中共國的經濟狀況似乎好於其他大型經濟體。由於投資者必須將資金配置到某個地方，市場表現總是存在比較因素。根據IMF 6月底發布的預測，中共國是今年唯一預計會增長的大型經濟體，預計明年仍將創造最強勁的反彈。在今年秋天的流感季節，冠狀病毒可能會在美國造成損失，這引起了人們的高度關注。但中共國已經表現出了遏制疫情再度爆發的各種意圖，讓民眾和企業對未來有更大的把握。
 [!\[\](https://spark.adobe.com/page/jgu69zJvMzuRt/images/0dfe70eb-e13a-4525-903b-f981dff3c228.png?asset_id=341435d7-7ba8-4792-9348-25ae1c7fe000&amp;img_etag=%229ea5d7c0d386493bc7d4136c7649d8b5%22&amp;size=1024)](https://spark.adobe.com/page/jgu69zJvMzuRt/images/0dfe70eb-e13a-4525-903b-f981dff3c228.png?asset_id=341435d7-7ba8-4792-9348-25ae1c7fe000&amp;img_etag=%229ea5d7c0d386493bc7d4136c7649d8b5%22&amp;size=1024) 圖1：標普500和滬深300市盈率
市場動力似乎也有所幫助。即使在股市上漲之後，中共國股市的估值仍是合理的。滬深300指數的股價是公司收益的14倍，遠低於美國最受關注的股票指數，即標準普爾500指數的27倍(見圖1)。外國投資者有更多的途徑進入此前被封鎖的中共國市場;許多公司被迫這麼做，因為該公司的股票如今已被納入機構跟踪的關鍵指數。在7月的前三個交易日，440億元人民幣(60億美元)通過香港的賬戶流入中共國股市，創下了三天以來的最高紀錄。

在最近中共國最狂熱的股市上行期間，儘管更多的投資者已經開始用借來的錢購買股票,融資融券交易的餘額僅僅略高於五年前峰值水平的一半(見圖2)。現在公司更容易在大陸上市,所以新股應該吸收部分湧入市場的資金。 “這為可能持續10年或20年的緩慢牛市奠定了基礎，”康莊資產管理有限公司的常士杉表示。
 [!\[\](https://spark.adobe.com/page/jgu69zJvMzuRt/images/c0a23d7d-52af-4448-a688-24a237313c07.png?asset_id=cdd6723b-ab24-43f2-b9c1-e9cd4a83a87c&amp;img_etag=%22def0cf117161a692aadcf8aacb715cc0%22&amp;size=1024)](https://spark.adobe.com/page/jgu69zJvMzuRt/images/c0a23d7d-52af-4448-a688-24a237313c07.png?asset_id=cdd6723b-ab24-43f2-b9c1-e9cd4a83a87c&amp;img_etag=%22def0cf117161a692aadcf8aacb715cc0%22&amp;size=1024) 圖2：中共國融資交易餘額（單位：萬億人冥幣） 
然而，樂觀主義者可能又一次走在了自己的前面，這種感覺很難動搖。最終應該決定股價的盈利前景仍然嚴峻。在2020年前五個月，工業利潤同比下降19%。

最令人擔憂的是，中國媒體突然轉成啦啦隊模式，這是過去非理性繁榮時期的一個明顯跡象。官方報紙《中共國證券報》7月6日在頭版發表社論稱:“對於我們的後病毒時代來說，牛蹄聲是一種美妙的聲音。”它的姊妹報紙《上海證券》，在7月3日發佈在網上的一篇文章中沒有那麼詩意，但卻更加直接:“哈哈哈哈哈!牛市特徵越來越明顯了!”健康的牛隻需要吃草。給它們注射類固醇會帶來麻煩。
