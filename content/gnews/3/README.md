## 【香港要闻】拒港台染红 19顾问团成员反对RTHK变政府喉舌
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1036702/)

搜集\编撰：西西

**港共政府继续打压新闻自由。香港电台节目顾问团19名成员日前发表声明指，新任广播处长李百全有关做法无视香港电台编辑自主和新闻自由，让港台靠拢政府，失去中立原则，接连对香港电台电视部的节目内容进行整肃和清洗，意图令港台沦为政府的宣传机器。**
![]()![](https://gnews.org/wp-content/uploads/2021/03/3308.jpg)
**声明：有关做法严重违反《香港电台约章》**

据大纪元香港3月29日讯，该声明提到，新任管理层要求事先过目《铿锵集》、《视点31》及《香港故事》等节目，并要求节目在制作之前必须先提交计划书予管理层进行审查；随意抽起或调动节目，拒绝向员工及公众解释原因等。

声明指出，有关做法严重违反《香港电台约章》赋予港台的职责，包括「港台编辑自主」、「提供准确而持平的新闻报道、资讯、观点及分析」、「提供开放平台，让公众畅所欲言，以不畏惧和不偏私的方式交流意见」，以及「不受商业、政治及／或其他方面的影响」。

**港共政府打压香港电台手法不断**

多个享有盛名的节目因为受到亲共阵营投诉被迫停播。通讯局分别在2020年4月及9月向港台发出「严重警告」及「警告」，《左右红蓝绿》遭「严重警告」的节目涉及2019年11月警方围攻大学事件，而收到「警告」的节目也涉及反送中运动的事件，包括8.31太子站防暴警察进入车厢打人，以及警察实弹射击抗议学生等；在2020年5月及2021年1月分别向港台发出「警告」及「强烈劝喻」，涉及讽刺时事节目《头条新闻》。这两个节目已经在去年停播。

多间传媒引述消息说，《铿锵说》主持苏玉华被认为亲民主派，因此主持了两集节目之后就被换掉，将由民建联成员林琳主持该节目。谭家浚质疑有关做法并非政治中立，「太黄的主持人或者偏黄的艺人你抽掉，为何同时间有政党背景的人可以参与节目主持？」

更有消息，如果该集《铿锵集》无法通过管理层审核，节目监制会被追究及调职，编采团队或被追讨制作费用，做法形容非常荒谬可笑。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3309.jpg)《铿锵集》编导蔡玉玲早前因制作《7.21谁主真相》被控违反《道路交通条例》，案件日前（24日）在西九龙裁判法院审讯，法官判决被控虚假陈述罪表证成立。 （宋碧龙／大纪元）
点评：

80年代，港英政府成立的广播事业检讨委员会曾经建议港台脱离政府，以公司化形式运作。当时正值中共与英国谈判香港前途，由于中共方面反对未能成事。英国档案透露，时任国务院港澳办公室二司司长王凤超在1992年到访香港时，表示港台脱离政府剥夺了未来特区政府的宝贵资产，认为港台应成为特区政府的广播机构与中国政府的喉舌。

在主权移交之后，港台虽然不时受到政治压力，仍然维持一定编辑自主。在2019年反送中运动之后，港台没有偏帮政府，客观报道抗议事件，受到亲共阵营的一连串攻击。

一个假、骗、偷的中共政权，媒体就是它的宣传渠道，洗脑工具。阻止「真相爆光」和掐住说真话的喉咙，是中共政权时时刻刻在干的事。随着中共全面控制香港，港台成为起政权的宣传工具只是程度于时间的问题。
