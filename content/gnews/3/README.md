中國和澳洲的關系近來因為壹連串事件變得日趨緊張。同時，中國也在世界各個國家外交關系中陷入困境，遭遇冷眼。接連發生的香港國安法，蒙古課本漢化事件引起的抗議呼聲引發了世界對中國獨裁暴政治國的廣泛關註。肆虐全球的新冠疫情大流行，各個國家已經逝去和正在逝去的萬千個鮮活的生命更加牽動全世界的正義力量壹起挖掘被隱藏的疫情真相。世界面臨著考驗！人類面臨著考驗！這是壹場真相與謊言的較量！正義和邪惡力量的角逐正在繼續！您選擇站在哪壹邊？！

澳大利亞廣播公司周三（9月9日）刊登調查報道說，澳洲當局針對壹名高級官員發起前所未有的外國幹預調查，期間兩名派駐澳洲的中國媒體官員成為調查目標，另有兩名知名中國學者簽證遭到吊銷，“此舉引發了中國政府的憤怒回應”。

成蕾被扣查後，兩名分別派駐北京和上海的澳洲記者先後也被中國國家安全局人員約談。其中，澳洲廣播公司駐北京記者伯特爾（Bill Birtles）透露，他最終在澳洲駐華外交人員的交涉下，同意在北京壹家酒店與中方人員作壹小時的約談，內容與成蕾的案件有關。

他在約談的第二天飛往上海，與另壹名澳洲記者會合後，兩人飛回澳洲。

澳洲外長佩恩（Marise Payne）形容，過去壹星期發生的事情是“壹連串令人失望的事件”。

中國外交部發言人周二就指出，中方對兩名澳洲記者約談是“正常執法行為”，又強調中方依法保障新聞人員的合法權利，但他們也有義務遵守中國的法律法規。

中國新華社和《環球時報》周二（9月8 日）和周三（9月9 日）分別報道聲稱，澳洲情報部門人員6月26日曾經搜查中國駐澳洲記者的住所，又批評澳洲政府的行動對中方機構人員和當地親華派人士制造“白色恐怖”。
中國外交部發言人在周三的例行記者會上稱，澳洲情報人員今年6月下旬以涉嫌違反當地《反外國幹涉法》為由，搜查和盤問了分別屬新華社、中國中央廣播電視總臺和中新社駐當地的記者，總共四人。發言人指，澳洲人員扣起了記者的電腦、手機，甚至記者孩子用的平板電腦等。中國外交部發言人批評澳洲的做法對中國記者和他們家人的身心健康“造成嚴重損害”。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/10141116/080.png)
中國官媒的報道可靠嗎？

中國官方媒體有關中國記者在澳大利亞居所被“搜查”的報道中，《環球時報》引述消息來源指出，澳洲情報部門的人員“長時間盤問記者”，又強行扣押記者的電腦、手機等。

《環球時報》指這次搜查是“近日”發生，但新華社的報道就指出調查是“6月26日拂曉”進行，並對搜查作出更具體的描述，指澳洲人員在沒有正當理由的情況下“‘突襲’搜查中國駐澳大利亞媒體記者居所，對記者進行長達數小時盤問，並帶走記者的工作電腦、手機、移動U盤（電腦記憶棒）、文字手稿等。澳大利亞情報人員並要求中方記者就此次搜查活動對外‘保密’。”

新華社的報道留意到，同壹天澳洲情報部門人員也搜查了新南威爾斯州工黨上議院議員莫斯爾曼（Shaoquett Moselmane） 的辦公室。

《悉尼晨峰報》周三引述消息指出，澳洲安全情報機構（Australian Security Intelligence Organisation，簡稱ASIO）在6月26日的確曾向最少壹名與莫斯爾曼有聯系的中國記者問話，行動也與搜查莫斯爾曼的辦公室有關，但沒有提及行動內容有沒有如中國官媒聲稱，扣押記者的電腦和手機物品等事宜。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/10141153/070.png)
報道說，澳洲聯邦警察的人員當天執行壹個澳洲安全情報機構人員也有參加的調查時，曾搜查莫斯爾以及他壹名兼職職員張智森（John Zhang）的住所和辦公室。聯邦警察當時指控中共當局嘗試透過莫斯爾曼“滲透”新南威爾斯州州議會。

莫斯爾曼今年8月接受澳洲廣播公司訪問時透露，當局調查的重點包括壹個微信群組，這個群組被指用來“支持和鼓勵莫斯爾曼為中國爭取利益”，約有七名成員，包括張智森、壹些外國記者和中國學者。莫斯爾曼說這只是壹個普通的社交群組，他沒有透露當中的“外國記者”是否來自中國。

《悉尼晨峰報》周三指出，被澳洲安全情報機構問話的中國記者是這個群組的其中壹名成員。

澳洲廣播公司（ABC）周三刊登調查報道說，這個微信群組中兩名中國學者的簽證先後也被註銷，兩名在當地的中國媒體高層成為澳洲聯邦警察和澳洲安全情報機構的壹宗涉及外國幹涉案件的調查目標。

該報道說，澳洲聯邦警察和澳洲安全情報機構共同調查在當地的兩名中國媒體高層及兩名中國學者，其中受查的兩名學者，報道引述他們說收到澳洲內政部的通知，以“國家安全”為由註銷他們的澳洲簽證，而澳洲當局正調查中共被指涉嫌利用工黨後座議員莫斯爾曼辦公室的壹名兼職職員、前中國政府官員張智森，“滲透”新南威爾士州議會的案件。

澳洲廣播公司說，澳洲聯邦警察和澳洲安全情報機構共同調查的兩名中國媒體高層人員，分別是中新社澳洲分社社長陶社蘭（Tao Shelan）和中國國際廣播電臺悉尼分社社長李大勇（Li Dayong），兩名中國學者分別是著名華人學者和媒體評論員陳弘（Chen Hong）教授和知名澳大利亞問題研究學者李建軍（Li Jianjun）。當局懷疑他們利用中國大陸社交平臺微信，作為“滲透”工具。

多國媒體6月的時候都有報道澳洲當局對莫斯爾曼的搜查行動，並指出搜查行動與中國被指幹預澳洲內部事務有關。有記者在中國外交部6月29日舉行的例行記者會上提問，發言人回應指不評論澳洲內部事務，但形容壹些澳洲政客“似乎患上了對華恐懼的臆想的偏執癥”。

澳洲總理莫理森當時沒有透露針對莫斯爾曼的調查內容，但指出當局的搜查行動顯示了它阻止外國幹預澳洲事務的決心。
