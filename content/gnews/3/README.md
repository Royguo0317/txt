[#新疆烏魯木齊疫情#](https://s.weibo.com/weibo?q=%23%E6%96%B0%E7%96%86%E4%B9%8C%E9%B2%81%E6%9C%A8%E9%BD%90%E7%96%AB%E6%83%85%23) 【最新！烏魯木齊又增5例確診！地鐵停運、航班大量取消，防控升級】

新疆維吾爾自治區衛生健康委最新通報，7月16日0時至17日12時，新疆維吾爾自治區（含新疆生產建設兵團）報告新增新冠肺炎確診病例5例、無症狀感染者8例，為烏魯木齊市病例。新增確診病例和無症狀感染者均為接受集中醫學觀察人員。

截至7月17日12時，新疆（含兵團）現有確診病例6例、無症狀感染者11例，均在烏魯木齊市，現有135人接受醫學觀察。

昨日，新疆烏魯木齊衛健委通報：7月15日，烏魯木齊市天山區發現新冠肺炎確診病例1例，無症狀感染者3例（均在醫學觀察中）。確診病例系一名24歲的營業廳工作人員，常住地為烏魯木齊市天山區。確診病例感染原因正在調查中。

相關防控措施升級

16日，據烏魯木齊地鐵微信公眾號通報，為了防控疫情傳播，自16日22時起，烏魯木齊軌道交通1號線停止運營，恢復時間將另行通知。

17日上午，新疆昌吉公交集團微信公眾號發佈公告稱，昌吉公交集團所有運營車輛，從即日起暫停營運。已經安排一線全體幹部職工全員做核酸檢測，會在暫時停運期間對所有車輛每日再進行徹底的防疫消毒。

據北京日報消息，記者查詢飛常准業內版資料顯示，截至11時20分，烏魯木齊機場進港航班取消118架次，出港航班取消109架次。部分航司已經向旅客發出近期前往烏魯木齊的出行防疫提示。

17日上午，新疆維吾爾自治區招生委員會發出緊急通知：原定於7月18日舉行的2020年自治區普通高等教育專升本考試延期舉行，具體時間另行通知。
