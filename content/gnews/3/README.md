中国有句老话：“有缘千里来相会，无缘对面不相识” 。在遥远的海外，还能看到自己的同胞，本应是一件很开心的事情。但现在，却出现了一种奇怪的现象。什么现象呢？ 即同胞间互不信任的现象。记得我在出国前，感觉华侨这个群体都是很美好的存在，因为他们总是在国家处于危难的时刻捐款救人，出钱出力，彰显出一种神圣伟大的魅力。以致于出国后，我总希望和华侨同胞们交朋友。但后来发现事实并非如此，很多华侨同胞们都有些封闭自己。在国外的大街上，可能你会遇到素不相识的外国人与你打招呼，而遇到同胞时，反而会冷场。逐渐的，随着认识的同胞越来越多，熟悉之后，就会发现，他们都在强调同一句话：“我从不和中国人打交道”。这是个很奇怪的论调。他们的理由也很多样化，比如国人太复杂啦，太坏了，事太多啦等等。后来，我分析了一下，发现他们中绝大部分都是很好的人，也都是值得交往的朋友，可让人困惑的是，为什么要异口同声的贬低自己的同族？而且话术都差不多？好像商量好了一样。此类话语像病毒一样，极具传染性，并且这类似的话听多了，也深深印在我的脑海里，好像随时可以脱口而出。

我很好奇这是他们被别人传染了，还是发自内心真实的感受？但据我所知，说这话的几个朋友也还是有几个中国朋友的，并非真如他们所说的那样：“从来不与中国人打交道” 但他们也许不了解，从心理视角来看，当某个群体中有很多人都在重复同样一句话时，不久，这句话会变成一种观念，从而被整个群体所接受，慢慢的，此观念也会造成相对应的现实。这也是心想事成的强大副作用。这种心理技术如果被别有用心者所利用，后果会很严重。我感觉这看似简单的一句话，却并不像表面看起来这么简单。极有可能是一场有关意识领域的深层较量。我相信中共深谙此道。如果我们没有觉察到，会输的比较惨。看来中共一直在意识领域里传播意识病毒。比如这些年来，中共的媒体不断发布一些国人如何素质差如何糟糕的文章及视频，让人们在不知不觉中反复确认，以证明“中国人很糟糕”的观点是对的，再互相传播，害人害己。对此，我深有同感，当时看着视频里同胞的所作所为，既生气又无奈，感觉颜面尽失。我很明白视频会带来多大的负面影响。但也无能为力。不知从哪里下手。可以终结这种扭曲。直到加入了爆料革命，我终于明白，原来这是中共在有预谋的带风向。殊不知长期以往，会使得同胞们的意识和认知被扭曲。错误的评价自己的同胞，甚至对自己也产生怀疑。慢慢丧失自信。导致人与人之间也互相猜忌。最终使我们中国人彻底陷入了一盘散沙的状态。当然，这其中唯一的受益者就是中共。这会更有利于他们的统治和胡作非为。难道不是吗？

当然，我们也不排除确实有些素质极低的同胞做了很多出格的事情，也确实伤害到了其他人。但我们要知道，这就像我们很难遇到极好的人一样，生活中，我们并不会经常遇到这样的垃圾人。我们不应该因噎废食，对整个族群一概而论。按概率统计学来说。极好极坏的人都是少数，大部分人还是不好不坏，也好也坏的普通人。如果不相信的话，我们每个人都可以分析一下自己身边的人 ，看看是不是这种情况？至少我身边的大部分人都是普通人，只要以诚相待，他们都会把美好的一面展现出来。其实，每个人都是善恶共存体。这个情况全人类都一样，国人也不例外。但是很重要的一点是，从本质上来说，人心都是向善的。就像所有的生命都需要阳光一样。正常的人性都是向往美好的。而且人性也都是需要被尊重、被认同的。当我们善待、尊重别人时，是会激发出个体善的一面的。在这里，我们可以想象一个实验，当我们在大街上忽然给路人一拳时，回报我们的大概率是五眼青，而当我们为路人送上一朵鲜花时，回报我们的大概率是笑脸喜悦及感恩。所以这充分证明了“种瓜得瓜，种豆得豆”原理。即当我们想要别人如何对待我们时，就需要用同样的方式对待别人。我认为这是适用于大部分人的处事法则。况且人类都是群居动物，谁不愿意有三五好友可以经常探讨人生，八卦聊天或聚餐放松一下呢？这一点在《人类简史》里已经阐述的很清晰了。这是整个人类族群的需要。 所以，并没有哪个国家的人低人一等，人性更糟糕，只有在极权控制下，被操控、洗脑和愚弄的百姓，一旦环境改变，人性必然回归。

但是，中共为了达到自己邪恶的目的，竟然把操控范围延伸到了人们的意识领域，幸亏爆料革命不断的爆料揭露，才让我们意识到其别有用心的宣传战术，哪怕我们不经意间看到的一个个视频，可能都会影响我们的认知和判断力，继而影响到我们这个族群的凝聚力。因此这不能不引起同胞们的重视，以防自己不小心也变成了中共手里的一把枪而不自知。不过话又说回来，既然中共能利用心理战术，从意识层面引导扭曲我们的观念，创造不利于我们的现实，而了解了真相的我们是不是也要学会反利用此战术？那我们就需要经常觉察清理以前的错误观念，并尝试从意识上开始信任自己的同胞，只有更多的人互相信任，团结起来，才能击垮中共的邪恶目的。随着中共的灭亡，社会环境的逐渐宽松，相信大部分国人的良知也会慢慢回归，毕竟，大部分人逃不开环境的影响。既然同样被洗过脑的新中国联邦人能觉醒，其他的同胞们也一定会觉醒，只是时间的问题。当然也需要我们的行动。不是谴责放弃，也不是心灰意冷。而是从改变我们的认知做起。觉察及释放所有对同胞的负面消极评判，相信“大部分中国人都是善良的”这个事实。带着宽容和敬畏之心，迎接他们的回归。也相信在爆料革命的努力下，一旦认清真相，未来的新中国人将成为世界上最友善，最团结的族群。
