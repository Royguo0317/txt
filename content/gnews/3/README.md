- **寻找证据**


在COVID-19全球大爆发的早期，川普政府专注在实施预防措施方面，以确保遏制病毒在美国国境内的传播。国土安全部的官员们，与白宫配合，在实施旅行限制方面做了工作，禁了所有从中共国出发来美国的航班。当时白宫成立了一个工作组，由卫生与公众服务部部长亚历克斯·阿扎尔（Alex Azar）与白宫执行幕僚长米克·马尔瓦尼（Mick Mulvaney）领头，与当时的高级顾问贾里德·库什纳（Jared Kushner）和其他高级官员一起协调配合，为确保拯救生命所需的医用物品和个人保护设备的正常运转，做了各种奠基工作。

“我们的重点是放在拯救美国人的生命上”，一位参与了联邦政府应对COVID-19行动的前高级卫生官员说，“其他的都不重要。我们看的是中共国有关病例和治疗方案的信息，但我们没有看病毒起源的问题，至少没有非常密集的看，这个问题在那个时候还不是重点。”

但对于多名白宫高级官员和国家安全官员来说，病毒来源问题同等重要，特别是因为这个问题可以使美国在世界舞台上比北京中共政权更具优势。在1月和2月间，这些官员开始在行政机构内部提出各种问题：关于该病毒是如何变成这样的，并且，敦促行政机构追查此事，进行深入的调查。

持续施压和要求调查的呼声越来越高，同时国会中对中共国持鹰派立场的人，包括阿肯色州共和党参议员汤姆·科顿（Tom Cotton）在内，开始公开谈论关于中共国对COVID-19的处理方式，以及该病毒来源于武汉病毒研究所的可能性。在这段时间，科顿推测该病毒来自武汉的一个实验室，该实验室“从事世界上最致命的病原体的研究”，“包括，是的，冠状病毒”，科顿在2020年1月30日发推中写道。

科顿对实验室泄漏理论相当着迷，以至于2020年1月31日，在对川普总统的第一次弹劾审判中，他从参议院的座位上走下来，给时任国家安全委员会执行秘书、作为国家安全顾问的马特·波廷格（Matt Pottinger）打了电话，敦促他推动行政机构，更加主动积极的调查这一可能性，一位熟悉该段通话的前白宫官员是这样描述的。

“科顿觉得好像他被情报界搪塞过去了”，这位前官员是这样说的。

其他白宫和国家安全高级官员，包括彼德·纳瓦罗（Peter Navarro）、蓬佩奥（Pompeo）、波廷格（Pottinger）和余茂春——蓬佩奥主要的中共国政策顾问，都指出武汉病毒研究所非常有可能是病毒的起源地。

“当我看到武汉火葬场冒出的烟时，我就马上警觉起来了”，纳瓦罗说。纳瓦罗是一名学院派培养的经济学家，没有直接参与行政部门对病毒起源问题的调查工作。“我预测过中共国会创造一场全球大流行病。我说得完全准确。我之所以能够做到这一点，是因为我的专业知识加上我一直盯着看这个博弈的棋局。当然病毒来自于实验室。”

但是当时，顶级科学家和卫生官员们，比如CDC主任罗伯特·雷德菲尔德（Robert Redfield）和安东尼·福奇（Anthony Fauci），他们两个人都在为白宫冠状病毒工作组提供建议，他们说他们所看到的证据表明，这种病毒起源于自然，而后传染给了人类。至少还有两种其他的冠状病毒——引起SARS和MERS的——在过去20年里，从动物的身上跳到了人类的身上。

“很多很多个星期以来，我们从顶级的博士们那里听到了关于海鲜市场的事情，并且我们有很多关于海鲜市场的讨论，而他们，至少在这个场合里，即使其他人提出来实验室泄露他们也不会接话茬。实验室泄漏说被完全排除在了讨论的范围之外”，一位前川普白宫官员这样说的。

2020年2月，27位科学家在《柳叶刀》杂志上撰写了一封公开信，称“来自多个国家的科学家们已经发表并分析了致病病毒——SARS-CoV-2的基因组，他们压倒性的得出结论：这种冠状病毒起源于野生动物”， “我们站在一起，强烈谴责那些暗示COVID-19属于非自然来源的阴谋论”，那些科学家是这么写的。

“我们都认同（这病毒）同病毒学实验室有牵连。并且，当所谓的专家在2月份的《柳叶刀》文章中断然否定了病毒可能来自实验室的说法时，这让我们感到震惊……”大卫·史迪威（David Stilwell）回忆道，他当时任东亚和太平洋事务的助理国务卿，他接着说，“这太荒谬了，所以从那时候我们开始担心科学界没有光明正大地发挥作用。”

美国国内的回应，（对病毒起源问题的质疑），迅速高涨，行政机构开始对于COVID-19的测试和城市封锁加大了力度，国务院、国家安全委员会和情报机构内部的各级官员开始了病毒起源的调查工作。虽然他们经常开会向高级官员介绍他们的工作进展情况，但他们的调查是相互分离的。

几位前国家安全和白宫官员说，他们认为在2020年春天收集到的间接证据促使他们相信该病毒源于实验室。

“按照常理，中共销毁了病毒样本，他们只让世卫组织的调查人员进入实验室三个半小时，他们对海鲜市场这个现场进行了彻底的消毒，他们没有让台湾进入世界卫生大会，更不用说这个实验室离疫情爆发的中心这么近”，一位前国务院高级官员说，“对我来说，我马上想到病毒来自实验室。”

其他前高级官员告诉POLITICO，他们核实过的那些证据，那些虚假指向并支持实验室泄漏理论的证据，只能表明泄漏看起来像是有过、但不能肯定。另一组官员，包括国家安全和高级卫生官员们，说这些报告过于依赖公开来源的资料。

“我们说，让我们开始考虑一下，这，对于下一次（大流行病）会有怎样的影响吧！我们只是要看看证据把我们导向何方”，安东尼·鲁吉罗（Anthony Ruggiero）这样说，他是前国家安全委员会高级主管，专管反扩散和生物防御的。“我们所面临的部分挑战是，我们一直是（对所有的信息）尽可能多的（——对于不同的消息来源都）保持了开放的态度。我们并没有把某一个信息的重要程度，放于另一信息之上。”在行政机构关于病毒起源的调查中，鲁吉罗的国家安全委员会团队是其中主导调查的团队之一。国务院和情报机构进行了他们自己的单独调查。

“这完全是模棱两可的，可以说病毒来自实验室，也可以说病毒来自自然。作为一名网络情报分析员，你看到的是一系列巧合的集合，你会开始怀疑，它们是否真的是巧合。一个实验室就是专门搞这个病毒的，而恰恰成为了疫情爆发的地点——这种巧合也太玄乎了！” 艾米丽·哈丁（Emily Harding）这样说。她是前中央情报局分析员，去年还曾是参议院情报委员会执行幕僚长。
