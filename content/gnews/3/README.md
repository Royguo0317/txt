**律师会：两层投诉法官制符司法独立**

司法机构拟就投诉法官引入两层、“社会人士”有份参与的处理机制。香港律师会回覆本报查询时指，司法机构提出的改进措施，以《基本法》保障司法独立的条文为出发点，改革后终审法院首席法官仍会就每项投诉作最终决定，符合司法独立原则。

司法机构日前向立法会提交文件，建议将处理法官投诉流程由一层变两层。第一层由一名高院级别法官组成专责的法官小组，处理性质严重、复杂或引起社会广泛关注的可跟进投诉，再交由第二层，由终院首席法官领导、成员包括法官及社会人士组成的谘询委员会审视及给予意见，在考虑谘询委员会意见后，终院首席法官会就每项投诉作最终决定。
