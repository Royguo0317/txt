**2007年，武毒所顶级病毒学家石正丽就在权威学术期刊《自然》杂志发表过论文。2013年，石正丽发现了蝙蝠冠状病毒，进行人工改造后，在人源化小鼠身上做实验。**

**2019年底，武汉的李文亮医生提醒同事有一种新型的不明传染性肺炎，随后他遭到警方逮捕。官方公告禁止谈论此事，宣称新型肺炎不会人传人。**
