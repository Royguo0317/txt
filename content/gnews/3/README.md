## 【热点播报】德国通过法案，为封杀华为网络设备提供法律依据
` 纽约香草山MOS02` [轉載自GNews](https://gnews.org/zh-hans/1137104/)

作者：纽约香草山健身部 Jack Li文阳

![]()![](https://gnews.org/wp-content/uploads/2021/03/GNEW-GTV-MOS-LOGO-2-3.jpg)



据德国《明镜》周刊报道，德国执政党联盟党和社民党不顾所有反对党的反对，于周五在联邦议院投票通过了IT安全法2.0。这部新法案为德国禁止中共国网络设备供货商华为参与5G建设提供了法律依据。

![]()![](https://gnews.org/wp-content/uploads/2021/04/图片5-27.png)

新法规定，允许德国政府以保护重要基础设施为由，将“不值得信赖”的移动网络供货商排除在外。如有供货商是由外国政府控制或参与，被认为具有危险性，并向德国提供涉及安全的关键组件，那么，德国政府就可根据新法规将其排除在外。

新法规定供货商多次违反有关可信度的规定时，有可能彻底被排除在5G网络建设之外。违反可信度指的是企业提供不实信息，不支持安全性检查，在发现IT系统有漏洞时，不立即报告并排除等。除此以外，新法规还规定，德国相关部门除了对其进行技术上的检查外，还会进行政治审查。 新法律是在围绕是否允许华为参与德国5G网络建设而制定的。

自2019年5月，美国商务部把华为及其附属的68家关联公司纳入所谓的“实体清单”。目前，包括英国、捷克、波兰、瑞典、爱沙尼亚、罗马尼亚、丹麦和拉脱维亚8个国家已选择只允许可信的供应商参与5G网络建设，另外，希腊也已同意使用爱立信而不是华为的技术开发5G基础设施。

法国宪法委员会2月5号批准了法国“反华为”法，希望保护移动网络“免受第五代移动通信提供的新功能可能引起的间谍、盗版和破坏的风险”。

2020年7月14日英国政府宣布，本地运营商必须在规定期限内将华为公司设备从本国5G网络中彻底移除。11月30日，英国政府表示，英国通信企业在2021年9月后禁止安装华为5G新设备。

意大利已经放弃与华为的合作，直接拒绝了华为的招标资格。而该国给出的理由和美国如出一辙，“威胁国家安全”。

瑞典于2020年10月20日决定禁止华为(Huawei)和中兴的电信设备进入其5G网络。

众所周知，华为在盗窃数据、危害网络安全方面天下第一。随着美国对华为制裁，越来越多的国家加入脱钩华为的行列。这次，德国这个深受中共“蓝金黄”肆虐的国家也要离开华为了，虽然所有反对党反对，在德国执政党联盟党和社民党努力下促成法案的通过。脱钩中共、脱钩华为成为世界潮流，华为的日子越来越不好过了，中共的日子还长吗？
