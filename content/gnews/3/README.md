眾所周知的中共3F計劃（搞亂美國，搞弱美國，搞死美國），終於在期待已久的2020年9月29日美國東部時間晚9點俄亥俄州克利夫蘭大學舉行的2020總統大選首場辯論活動中淋漓盡致地上演了。

川普總統和民主黨候選人拜登是此次辯論主角，福克斯節目(Fox News)媒體人華萊士（Chris Wallace）作為調和主持人，辯論由6個15分鐘左右的板塊組成，每人對第壹個問題進行2分鐘發言，然後是開放討論，華萊士決定所有問題和話題，據他稱這些提問事先沒有對辯論委員會和辯論雙方公布。

這場為時1.5小時的辯論中，我們看到了中共勢力支持運作下的民主黨候選人拜登作為政界代表如何歪曲、狡辯，顛倒黑白，看到了仿佛寓意和代表美國司法界、媒體界的新聞主持人華萊士如何憑借角色優勢，以時間限制為由，有失公允、幹擾發聲、每次即將涉及問題真相和本質時壓制對方、轉移話題，僭權為辯論方、極力拉偏架的拙劣表演；同時我們也看到了川普總統孤身奮戰、以實力和真相批駁對方、為自己也為國家力挽狂瀾、幫助美國恢復正道的奮力搏擊的身影。這些鬥爭已經超越了為美國國家利益美國公民所做的努力，其背後的強大勢力就是中共的黑手，他們長期從事搞亂美國、搞弱美國的勾當，也終於在今天的辯論舞臺上由其代理和滲透的勢力傾情上演，可以說，如果不是爆料革命揭開了美國正義和保守勢力眼前的迷霧，中共搞死美國的企圖也將勢在必得。

這場辯論中，拜登作為辯論選手並非壹無是處，相反在很多時候、很大程度上確實能夠迷惑選民和不知真相的群眾，仿佛成了溫和、開明和體恤民情的化身，而川普總統卻披上了言辭激進的鬥士的外衣，不了解他的人可能難以接受，這些在以下6個辯論板塊中得以精彩呈現。

第壹、            關於最高法院法官的任命。

川普總統充分肯定了3天前提名的Amy Coney Barrett法官，稱其得到很多高層學院人士的認可和尊重。而拜登認為這是川普和共和黨搞亂法官任命的濫權行為，他很快把議題轉移到總統想取消奧巴馬醫保險，稱女性權利需要得到重視，如果取消這項保險，女性懷孕後需要支付更多費用，而被提名的法官之前認為奧巴馬醫保違憲，所以最高法官要等到明年2月以後再任命。這些論調不可否認能夠拉攏壹些正在享有奧巴馬醫療保險的普通民眾，也體現了美國各派力量極力確保司法界力量的努力，當然“北京拜登”背後則是中共企圖幹涉美國司法的居心。

拜登堅持稱美國要已經有700萬人感染新冠病毒，需要醫保，他批評已經有1000萬人失去醫保，最終會有2000萬人失去該項保障，並抨擊川普總統做事沒有計劃，直接斷言川普是“騙子”。

作為主持人的華萊士則直接反問川普總統自己的醫療報銷計劃。他在壹開始介紹辯論方時把民主黨提名人前副總統拜登直接稱為副總統。

川普總統否認了取消1.8億人醫保，但表示美國正在變為社會主義國家。他強調自己對降低藥品價格所做的努力，能夠切實減輕每個家庭的負擔。

第二、應對疫情的問題。

拜登再次強調美國感染新冠病毒人數，死亡人數已經超過20萬，美國占世界4%的人口，卻有20%死亡，每天都有750-1000人死亡、4萬人感染，但是總統卻沒有計劃，他2月就知道病毒的嚴重性，卻不讓美國人知道，為的是不讓美國人驚慌，可是妳沒有驚慌他卻驚慌了！我們應該去武漢找到病毒有多危險，但是川普卻沒有問問習（近平），反而說習做得非常好，應該感謝習，他為什麽這麽做呢？他不停等、等、等，卻什麽也沒做，他沒有計劃和行動。現在共和黨和民主黨應該壹起，把錢拿出來救命。拜登強調，“川普只關心股市，可是現在很多人死了，很多人死了，很多人死了，妳知道嗎？“這些貌似事實又煽情的言論，不能不說是有策略的，拜登抓住川普執政中的瑕疵痛打，其狡猾和陰險可見壹斑 。

不出意外，口罩又成了攻擊川普的目標，只是這是由主持人華萊士反問川普是否否定戴口罩開始。拜登稱如果我們按照疾病防控中心建議，早點戴口罩，防止病毒傳播，可以少死很多人。對於病毒問題，拜登稱，他相信科學家，但是不相信川普。其言外之意包含了對曾經被川普總統肯定的羥氯喹、這個目前唯壹且有效的預防中共病毒的經濟有效的藥的否定。拜登為了政治選舉做好了置美國民眾生命於不顧末路狂奔的準備。

川普的回擊是他在疫情之前使美國經濟達到前所未有的景況，他稱病毒是中國的問題，並強調在早期做出的旅行禁令，挽救了很多生命。他說很多人在醜化我，並提及豬流感問題。他說疫苗問題將在11月1日得到相關部門的答復。同時諷刺拜登請不要使用“聰明“壹詞，因為他忘記自己的學校名字，上學時期的成績是墊底的。川普隨後掏出口罩示眾，贊同適當時候佩戴口罩，同時稱對於病毒的解讀完全來自疾病防控中心的伏祈博士。毫不誇張地說，川普總統之前對於應對中共病毒疫情所做的正確決策完全是受爆料革命的積極影響，幸虧美國有對爆料革命有積極支持的正義力量，這反過來也拯救了美國。

第二、關於經濟問題

川普總統同意失業率降至8.4%，經濟呈V形上升的恢復趨勢，拜登則認為經濟仍然是K形下滑趨勢。 拜登稱像川普這樣的大富豪可以很好地度過危機，但是卻不顧百姓的家庭。不解決新冠病毒問題，就不能開放經濟，開放學校，我們需要安全。

川普總統稱，我們壹開始及時的旅行禁令防止了可能發生的壹百萬人死亡。現在城市裏像監獄壹樣，存在很多社會問題，必須在遵守疫情防護措施的基礎上重啟經濟，學生們期待回到學校。

主持人華萊士直接提問川普，是否2016-2017年每年只交了750美元聯邦所得稅，當川普回答自己這幾年每年交幾百萬的稅時，主持人再次直接重復之前的質問，仿佛自己成了參加辯論的選手。當川普總統稱自己47個月做的比拜登47年做的都多，“我所做的拜登想都沒想過，包括整頓軍隊”，這位主持人角色轉換回來，立即打斷了川普的發言。特別是當拜登提及川普只顧和中國做貿易交易，川普回擊以拜登的兒子從中國拿到巨額錢財，莫斯科市長的夫人給了他350萬美元，這位主持人多次打斷川普，提高音量試圖壓倒壹切聲音，而拜登只是壹味重復“那不屬實。”很快拜登回過神來攻擊川普的家人，川普回以“我的家庭因我當總統為了幫助政府失去了很多財富。拜登則逃離這個話題，稱“這不是關於我的家庭或他的家庭，這是關乎妳的家庭，每壹位美國人的家庭，川普不關心妳們。這時主持人立即接話，稱結束這壹話題。就這樣拜登兒子和中共之間巨大的腐敗黑幕被草草收場。

接下來的辯論情況有所變化。

第四、關於美國當前社會的暴力、種族問題，川普總統稱黑人社區很糟糕，有很多左派極端分子當領導，族群敏感教育是消除仇恨熱愛美國的教育。奧巴馬、拜登執政時期，美國充滿仇恨和分裂，不談法律和秩序，這是黨派問題。ANTIFA是很危險的組織。而拜登稱自己支持BLM黑人運動是幫助他們能夠找到心理醫生、精神病醫生解決問題，他認為ANTIFA是壹個概念而已，不是壹個組織。川普總統用事實和數據說話，比如奧巴馬、拜登時期有128個法官空位，拜登兒子亨特拜登因服用毒品被踢出軍隊的不光彩的經歷。而拜登只是用判斷性語言給川普下定義，比如不守信用、種族主義者。拜登率先提到普金，稱川普是普金的傀儡，川普則回擊拜登兒子亨特與俄羅斯的腐敗受賄問題。拜登此時在全程辯論中第壹次回頭正面對著川普總統，放棄優雅的姿態，提高音量，指著川普說，“當提到我的兒子，提到軍隊，我想說我兒子在伊拉克呆了很多年，獲得銅牌勛章，他不是失敗的人。”川普問他：您是說亨特嗎？拜登回答，這是保羅。並補充說，亨特和很多美國家庭遇到的問題壹樣，他使用了毒品，但是正在改進。當亨特對俄羅斯受賄的問題再次被提出時，主持人打斷所有人說進行下壹個話題，而他在這個回合裏加入激烈辯論的痕跡更加明顯。

第五、氣候變化的問題。

這個話題占用了比較長時間，對美國加州大火，土地被燒，天氣變化產生辯論。

川普總統之前將美國從國際組織中退出，他說森林大火更多是管理問題，也同意在電動車給予稅收優惠。川普指出，奧巴馬清潔能源計劃，所謂新綠色計劃花銷是200億美元巨額。

拜登則稱不支持電動車，要建高速公路，到2035年創造更多就業，創造綠色基礎設施讓我們更健康，安全，讓颶風洪水消失，我們需要和全世界和解，實現世界和諧。這些論調對於熟悉中共的人而言何其相似，搞基建，花巨資，打著百姓的旗號，搞所謂“和諧”世界共同體。拜登學習中共快，美國人被欺騙得也快。

第六、對選票性質，寄票方法，選舉結束後5周仍然計票。

拜登贊同這個辦法，鼓勵大家這樣投票。

川普稱自己會贏，不會有交接，他說他們想顛覆政權，有的地方30-40%選票消失，有的選票被丟進河裏，有的在垃圾桶裏，這是欺詐行為。人們要監控選票站，做公平的選舉，表示如果有上萬張選票作廢，自己不能接受。這段辯論中，川普總統提到“我們有錄像帶”，這句話給所有爆料革命關註的重心問題予以壹個答復和肯定。

整個辯論中，川普總統精神矍鑠，目光炯炯有神，自信誠懇，應答如流，擺事實講道理，完全是脫稿演講。而拜登候選人則不停低頭看稿，經常使用主觀詞匯，在談到要害問題，試圖掩蓋兒子亨特的巨大問題，甚至在激烈辯論後忘記之前的要點，在主持人的提示下回到話題。然而他使用的辯論技巧則是偽裝以儒雅的氣度，歪曲事實，煽動群眾，把沒有依據變成主觀煽情，抓住對方的弱點狠打，不失在某些時候占據風頭。主持人則作為吹黑哨、拉偏架的選手，時而轉換角色成為辯論方與川普總統發生激烈爭論。川普總統的對手不僅僅是候選人拜登，還加入了調解媒體人。川普舌戰全場，為發出真實的聲音力爭反壓制，顯示出英勇的氣概，
