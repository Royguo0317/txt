網易新聞2020年9月21日報導：廣州市番禺區警方通報稱，當天早上7時許，番禺區鐘村中心小學附近發生一起持刀傷人案件，暫時共發現7名傷者，包括4名小學生和1名幼兒園學生（3人輕傷，2人重傷），現正在搶救中。另有1名成年女性受輕傷，犯罪嫌疑人自殘身亡，警方稱其有“長期抑鬱症”！
