## 楊潔篪的一腳油門
`2020-06-19 10:23 GM30` [轉載自GNews](https://gnews.org/zh-hant/239439/)

**作者：玉米地大姐**

那邊廂大褲衩美女主持人罵彭佩奧罵得高潮叠起，這邊廂派出楊潔篪去夏威夷會見國務卿，爲了借坡下驢說是應邀去的，其實是硬要去的。去之前要談什麽全世界都知道就粉紅們不知道，還以爲川普總統被他們罵服軟了。

新華社是這樣報道的：雙方就中美關系和共同關心的國際與地區問題深入交換了意見。雙方充分闡明了各自的立場，認爲這是一次建設性的對話。雙方同意采取行動，認真落實兩國元首達成的共識。雙方同意將繼續保持接觸和溝通。

![](https://gnews.org/wp-content/uploads/2020/06/1-300.jpg)

“充分闡明了各自立場”說白了就是各說各話、話不投機。看楊潔篪的眼神透著一絲無奈與焦躁。楊不是傻子不會不知道此番行程成必定是豬八戒照鏡子裏外不是人。

去，是與“人類公敵”爲伍，談不成會被主子罵得狗血噴頭。應了那句古話，卿本佳人奈何做賊，估計楊娘娘一肚子草泥馬在奔騰。

毛魔頭時期罵尼克松總統爲“一顆蔥”，以至于用力過猛想往回收的時候大費周章，不得不搞出個“乒乓外交”做鋪墊。當尼克松照片出現在人民日報頭版之時，全國人民還是錯愕了。

![](https://gnews.org/wp-content/uploads/2020/06/2-146.jpg)

于是有民間高人給大家解讀：“毛主席他老人家怎麽可能和美帝國主義建交，讓一顆蔥來是誘敵深入、擒賊先擒王。”

等到尼克松吃完國宴大餐擦擦嘴巴揮一揮衣袖登上飛機樂顛顛回了美國，民間又給驚著了，紛紛向那位高人請教：“毛主席他老人家怎麽讓一棵蔥逃回美國了？”

高人抽了一口“大生産”香煙解讀道：“你們懂什麽，毛主席這叫放長線釣大魚，一根蔥再大再粗能切幾個蔥花，毛主席要的是美帝的一捆蔥，等著瞧好吧！”

“原來是這樣啊”。衆人恍然大悟，豎起大拇指稱贊道：“還是毛主席他老人家英明偉大、站得高、看得遠！。

後來的事情大家都知道了，直到偉大領袖變成臘肉，美帝的一捆蔥也沒等來，那些失望至極的人們將“反美基因”遺傳給了現在的小粉紅，他們接過祖輩的口炮繼續向美帝開火，以至于傷到了黨的人，比如楊潔篪。

懸念大師希區柯克有個著名的電影理論。如果第一幕出現一把槍，那麽第三幕必須讓那把槍開火。

外交部、大褲衩和粉紅們組成的聯合口炮軍團在第一幕窮盡所有下三濫詞彙罵川普、班農和彭佩奧。罵到第三幕楊娘娘意外登場，不顧豬隊友迎面飛來的吐沫星子直奔人類公敵而去。劇情反轉的如此驚悚、詭異，希區柯克地下有靈也會自愧不如。

![](https://gnews.org/wp-content/uploads/2020/06/3-90.jpg)

中共外交至今沒有超過慈禧一朝，老佛爺在危難之際好歹還有個“裱糊匠”李鴻章能和洋人說上話。如今的中共外交除了“堅夫瑩婦爽歪歪”式的罵街撒潑，能拿得出手的只有楊娘娘了。

問題是預設人類公敵在先，楊娘娘此行成與不成都是“令人不齒”。楊潔篪內心該做何感想？還有比這尴尬、窩囊的外交會面嗎？

楊潔篪夏威夷之行，本想在通往地獄的道路上踩刹車，卻誤踩了一腳油門，爲中共盜國賊的滅亡提速加油。
