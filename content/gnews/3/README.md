**本期內容**

本期班農戰情室由史蒂夫•班農（Steve Bannon）與拉希姆•卡薩姆（Raheem Kassam），傑克•馬克西（Jack Maxey）和葛列格•曼茲（Greg Manz）一起討論了關於CCP冠狀病毒大流行的最新消息，期間圍繞著比爾•巴爾（Bill Barr）在具有歷史性意義的演講，對其中揭露了中國共產黨的種種罪行的講話進行了分析。

**具體內容**

節目在開始的時候就引用了美國司法官部長比爾•巴爾先生在昨天的演講一針見血地指出了“CCP統治者的目的就是把美國搞弱並且消滅美國，而不是與美國做貿易”這一點，並且借用美國國務卿龐培奧先生的話告訴全世界人民“美國政府的目的就是要保護每一個的美國人”。

史蒂夫•班農先生（Steve Bannon）指出2020年的7月16日是一個十分值得紀念的日子。

我們可以明顯地看到美國政府的“四個組合拳”包括了1.羅布林•布萊恩先生在鳳凰城的演講；2.美國聯邦調查局FBI局長德威先生的演講；3.上周美國國務卿龐培奧先生的的演講以及4.比爾•巴爾先生的演講。這一切都是為了聯合全球滅共所做出的佈局，而且這一切都才剛剛開始。而且這次美國針對CCP的行動主要是要應對CCP的獨裁統治者。

班農先生在分析美國司法官部長貝爾•巴斯先生在昨天的演講時指出：目前就差美國財政部部長史蒂芬•努欽先生的行動了。 在這次行動中我們應該停止擔心股票市場的起伏，不要理睬來自華爾街資本家的壓力，充分的運用我們經濟制裁手段，例如：停止于CCP進行美元交易，停止於CCP的SWIFT國際結算，停止於CCP的資本和貿易流通。

對於拉希姆•卡薩姆（Raheem Kassam）所指出的近些年在美國的一些大型科技公司比如：雅虎，穀歌，蘋果，以及微軟都表現出了與CCP政府合作的極大熱情這個事實，班農先生明確表示：這些公司多年來與CCP政府的狼狽為奸，甚至幫助造成全球疫情大爆發，很明顯就是CCP的幫兇。蘋果公司在美國拒絕了司法部的調查，但是在中國銷售蘋果手機的時候如果不把私人資訊交給CCP政府就不能進行銷售，而且還在中國的廉價勞動力市場裡奴役勞工——這顯然就是一個雙重的標準——蘋果公司已經純粹為了經濟利益而墮落成為了CCP幫兇，令人作嘔。

傑克•馬克西先生（Jack Maxey）在對於美國目前是否應該放鬆經對於中國的濟政策上請教了班農先生。就此班農先生說：就目前美國的情況而言，基礎建設和經濟刺激是非常重要的。但是目前美國正處於同CCP的戰爭中。所以美國在司法，行政政策上針對CCP做出了巨大的改變。其實CCP就是一個紙老虎，五十二萬億的房地產泡沫就是CCP政府的經濟地雷，並且他們還在不斷增發貨幣。 眼下最重要的事情是和特朗普總統團隊一起打好這一仗，而全球疫情只是貨幣戰爭的另一面。

對於美國司法官部長貝爾•巴斯先生昨天的演，班農先生分析道：對於21世紀來說把CCP與中國人民分開是當務之急。這此的演講是我所聽到過的演講中最強有力的演講了。美國有著非常健全的法制，但是從未針對CCP執行過。只有像特朗普總統與美國司法官部長比爾•巴爾先生這樣的人才能真正的把法制徹底的執行，這也將是蘋果公司等CCP幫兇的末日，這些公司一定會被掃進歷史的垃圾桶。目前美國政府正在把CCP與中國人民分開，讓中國人民認識到CCP才是中國人的敵人，也只有中國人自己才能戰勝CCP。特別是特朗普總統還發出過一個hello kity的推文——預示著正式把CCP政府視為敵人，並將其與中國老百姓分開，以前對於共產黨的天真以及無知已經一去不復返了。

戰情室繼續對巴爾先生的講話進行深度分析，在論及巴爾先生所說的“中共經濟閃電戰”時，班農先生說這是對閃電戰的新定義，也是第一次把中共和納粹聯合在一起，而司法部說的戰勝這個閃電戰的工具就是基於國際規則秩序的自由民主聯盟，中共正在通過21世紀的資訊技術和經濟來對世界發動閃電戰，主要是針對美國，甚至於還有一些美國政客和華爾街鼓吹中共不可戰勝論，有一副刊登在《新聞週刊》封面上的圖，一隻龍與鷹的圖聯合的圖——龍在上，鷹在下，這就是資訊戰中的一部分，大概是宣傳中共所控制的中國崛起壓制了美國，但實際上是美國人的養老金、保險金、銀行存款等作為資本去贊助中共，與中國共產黨進行合作交易。巴爾的講話中提及這樣的資訊是非常有價值和意義的，因為中共正在使用美國人的錢來對美國發動經濟閃電戰，為了引誘美國企業進入中國市場，中共會向投資商提供海量的經濟利益，而之後中共就會借此達到竊取專有技術的目的。

戰情室通過相關的文章指出：美國投資商無視中共的邪惡進而加強投資，這本來是之前美國政府一貫採用的綏靖政策——1989年的天安門事件後，美國不但沒有停止與中共的貿易合作，反而當年就訂下了兩年後的貿易訂單，與中國的貿易訂單從1991年的2.17億美元猛增至明年的20億美元，華爾街和各種投資公司在這些投資中獲取了大量的利益，巴爾先生的講話已經揭露了這些事實——即使是有如此之大的貿易往來於中美之間，中國老百姓仍然過著奴隸般的生活，中共和華爾街就是在榨取這些廉價勞動力的血汗。

戰情室在說到美國各方面的行動時指出，當前已經有了聯邦調查局雷的報告，有了國務院彭培奧的發言，有了司法部巴爾的調查報告，什麼時候能從國庫那邊聽到相關的消息呢？班農先生說如果美國想進行反閃電戰，那麼答案就是香港！中共剝奪了香港人的自由，而香港人是嚮往自由的，停止在香港所有與中共的一切貿易，限制並關閉交易所，截斷中共利用香港獲取美元的管道，制裁中共在香港的幾大金融機構團體，對涉港問題的相關人士進行制裁，不只是制裁林鄭月娥這樣的人，更要制裁中共中央的那些人，比如王岐山和習近平。中共長期以來壓榨利用中國奴工為他們工作，利用美國人的失業金、保險金等獲取經濟利益——但是相比之下中國老百姓卻什麼都沒有——同時中共將獲取的利益用於曼哈頓等地投資美國的房地產，中共核心家族的經濟就是這樣建立起來的。

中共啟示錄的天啟五騎士已經出現了四個，當前有財務部長巴爾，國家安全顧問羅伯特•奧布萊恩，聯邦調查局主任雷，國務卿彭培奧。儅第五位騎士出來做調查報告時，中共就結束了，這是戰勝中共的關鍵！

根據巴爾先生的講話，戰情室分析說當前中共獨裁與西方民主正在進行著資訊戰，華為、抖音、zoom、微信等都是中共資訊戰的一部分，也有像思科這樣的公司，説明中共建造網路防火牆，中共不僅僅監視其他國家，也監視中國老百姓，中共的獨裁統治必將滅亡，而這些幫助中共進行邪惡統治的公司和個人，將會被釘在歷史的恥辱柱上，那些海外註冊的外國代理人以及幫助中共四處遊說的團體，雖然有一些沒有註冊為外國代理人，但是巴爾的講話也透露出一定的信號——他們將被區別開來。在這場與中共的戰鬥中，所有為中共月臺代言的企業和個人，都將受到懲罰，中國人民也跟中國共產黨區別開來了，在這場與中國共產黨的戰爭中，美國在技術戰，資訊戰、經濟戰、熱點戰等各方面都開始猛烈反擊！
