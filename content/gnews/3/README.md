今天和一位在大陆结识的台湾朋友聊天，他从2000年以来一直在大陆工作和生活，和他聊天的结果令我感到震惊。

刚开始我们聊了病毒和香港危机，可是这位台湾朋友对真相却一无所知。我问他假如国民党或者民进党搞公投，宣布台湾并入大陆,你会怎么选择？他说愿意并入。我问他如果中共打台湾，美国人只是做做样子怎么办？他说愿意投降，加入大陆没有什么不好。我很奇怪，大陆的护照哪里都去不了，台湾的护照全世界170个国家免签，你要大陆护照干嘛？他的回答是“赚到钱最重要”。

他认为台湾式的自由是可悲的，今天台湾的民进党比当年的共产党更下流，只是利用民主自由打倒国民党，结果还是一样的贪污腐败，打击不同言论。他甚至认为台湾的所作所为比朝鲜还不如，明知共产党要收回台湾，还故意放任两岸的敌意……这种本末倒置的思维方式令我不禁倒吸一口凉气。

这位台湾朋友在大陆已工作和生活了20余年，是某企业的中层干部，和不少在大陆长期工作和生活的台湾人一样，他的想法具有一定的普遍性，对共产党的凶残一无所知。由于受中共的宣传，他们都和中共党媒的想法一样，成为标准的粉红和自干五。他以前在富士康的手下和朋友过去都很佩服台湾，现在看不起台湾，而且支持中共攻打台湾。这些人满脑子的想法就是那种大国崛起、一统全球的汉武帝秦始皇式的思想，这种抛弃来之不易的民主和自由、甚至罔顾2300万台湾人生命的谬论，让我有一种说不出来的悲哀。

可能是由于大陆沿海经济发展特别好，他已经习惯了上海的那种繁华，甚至认为台湾的发展太慢还不如朝鲜，我一时无语，殊不知上海的高速发展是牺牲了绝大多数中国人的代价换来的。和这位朋友聊完天，我深深觉得台湾的民主估计也保不住了，这位朋友50多岁，本应属于台湾的中流砥柱，却把台湾好不容易换来的民主自由说得一文不值，反而欣赏大陆的极权专制统治。

文贵先生在近日的直播中提到台湾的朋友担心中共会攻打台湾，他认为应该不会太快，中共趁着美国新一届政府立足未稳使用武力威逼恫吓台湾，而和中共勾兑的台湾国民党会站出来呼吁用投票解决，只要一旦开始投票，中共攻打台湾也合法化了，这就是中美之间最大的勾兑，香港和台湾被用来交易。蔡英文可能会是台湾最后一位民选总统，她犯的政治错误和对政治的幼稚，有可能会把台湾葬送。台湾同胞们以为在岛上很安全，殊不知已经命悬一线。

文贵先生曾经最早提到香港会变成臭港，这在过去的一年多以来正慢慢变成一个不幸的现实，今天的台湾又将何去何从？相信只有爆料革命和新中国联邦才能救台湾，也能救香港和亿万个被中共奴役的中国同胞！
