## 职业标准出炉，中共敛财入电竞

**责任编辑：**温哥华圆成农场 比卡丘

**编辑/校对：**首尔喜韩农场 文迹～见证神迹



环球网2月19日转载《中国青年报》消息，近日中共人社部发布《电子竞技员国家职业技能标准》，为电竞行业提供“中共化”的量测标准。

![]()![](https://gnews.org/wp-content/uploads/2021/02/221-1.jpg)

1、《标准》将电子竞技员分为五个等级，并划定申报每个等级最低工作年限：五级要求电竞工作≧1年；四级要求工作≧3年；三级要求取得四级资格后再工作2年以上或电子竞技国家级赛事参赛资格；二级要求取得三级资格再工作2年以上或取得电子竞技国际赛事参赛资格及国家级赛事奖项；一级要求取得二级资格后再工作4年以上或取得电子竞技国际赛事奖项。刨去IG等职业竞技队伍通过参加国家、国际拿奖一举拿到一二级资格外，现行的本科开设电竞相关课程，一个学生循规蹈矩考取一级资格证至少需要11年。

2、《标准》中对资格的考取做了详尽的规定：培训参考从160学时起（五级224、四级192、三级192、二级160、一级160学时）；理论知识和专业能力考核均达到60分以上为合格；两项考试涉及项目操作、战术实施、数据分析、活动表演、项目开发、管理培训等方面内容。即便是电竞世界级冠军都需要从头学起，参加160学时培训，做题、对答案、纠错，才能获得一级资格证，这一冗杂的设定和考取标准在为电竞添堵的同时，却方便了中共各级机构的巧立名目收费，接下来难免出现诸如电竞公司需要配备对应数量一二级资格证，某赛事只能几级以上参加等奇葩规定。

![]()![](https://gnews.org/wp-content/uploads/2021/02/2222.jpg)

在经济惨淡的2020年，中共国失业人数激增，本科毕业或硕士毕业“喜报”快递员、外卖员工作，技能标准出台下，家教、护理、外卖等有了“法定依据”，中共非但没有疫情补贴、减轻人民负担，反而加大了对私企和底层人民财富的掠夺，一方面公私合营加剧；另一方面通过宅基地法、房产税、个人税改等名目加大了对个人财富的剥削；更甚的是利用行政手段，对各行业进行门槛敛财，以虚有的技能资格，带动培训、报考等一系列的产业，但归根结底都是加大了人民的生活开支。电竞标准设立，反映了中共对电竞这一新兴行业的侵占和利益敛夺，酒囊饭袋的专家们，拍脑袋设定的考核内容更是对职业电竞者的侮辱和专业践踏。
