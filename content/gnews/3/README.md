## 中共命令三峽洩洪已淹沒了宜昌，假設潰壩我們應該如何自救
`2020-06-28 16:00 GM08` [轉載自GNews](https://gnews.org/zh-hant/248123/)

網絡上目前有大量的湖北宜昌大洪水的消息被公佈出來，大壩洩洪加上大暴雨，宜昌變成“威尼斯”。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/28071318/%E6%B9%96%E5%8C%97%E5%AE%9C%E6%98%8C.jpg)
據宜昌市氣象台預報：今天凌晨我市自西南向東北出現降水天氣，目前宜昌城區及周邊，長陽東部，遠安城關出現大暴雨天氣。據統計，04-11時大於25毫米86站，大於50毫米33站，大於100毫米8站，最大降雨量141.8毫米，最大小時雨量80.7毫米，均出現在點軍區楠木溪站。預計今天午後到夜間我市仍有大雨到暴雨，局部大暴雨天氣。往年長江流域的主汛期一般是7、8月份，但有香港媒體指出，現在長江上游的葛洲壩和三峽大壩都已經開始全力洩洪，才會導致距離三峽大壩僅40公里的壩下第一城宜昌市全城被淹。

根據中共國應急管理部數據顯示，截至6月26日，遭受洪災人口已有1374萬人，死亡失踪81人，直接經濟損失278億元（約新台幣1012億元）；目前已有15個省共84條河流發生超警戒水位以上洪水，其中，重慶市綦江區更出現近80年來最大洪災。中共中央氣象台晚間持續對南方多各地方發布暴雨黃色預警，時間從今晚8時到28日晚間8時。重慶東南部、貴州東北部、湖南西北部、湖北大部、河南中南部、安徽中北部、江蘇中南部、上海等地有大到暴雨。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/28071350/thread__72738609_20200627194957_11882.jpg)
這只是一場暴雨，長江上水庫一次洩洪，只是剛剛開始就已經將一座城市變成水城。如果持續近兩個月的雨季連續性的暴雨，加之假設三峽大壩潰壩，那麼下游必將生靈塗炭，面對如此可怕的生死門，我們普通百姓該如何自救？我將看到網上一些救生方法分享給可以看到這篇文章尤其是生活在長江流域的同胞們。

如果你就在水庫的正下方不遠處，洪峰到來時基本上就不用逃跑，因為你能夠活下來的概率基本上是零。如果你在洪峰到達之前，跑到遠離河岸幾公里遠的高處，生存的概率就大大增加。若是附近有一個山丘，躲到山丘上，基本上就安全了。如果你住的地方不是洪峰直接衝擊處，距離河岸有一段距離，就可以跑到牢靠的高樓，就會相對安全。距離河道稍遠處，洪水不是一下子高速沖過來，而是慢慢漲高。開始或許只是大街上開始流水，然後水深到大街無法走路，然後一些汽車開始被水浸沒，一些平房也被水淹沒。如果生活在這種地帶，你要熟悉本市的地形和道路，了解不同地點的海拔高度，了解水會從哪個方向流過來，哪條道路會被最先淹沒而無法通行。你還要了解附近每一條道路的窪地。比如在北京，曾有一個人在下雨天困在西便門橋下的窪地，被雨水淹死在車裡。

如果你在家裡，你覺得你家地勢太低，或者房子不夠結實，或者洪水讓你無法在家裡待著，這時候要果斷離開。離開之前，要關好門窗，避免家裡的東西被水沖走。把最重要的東西放在密封袋裡，放在家裡較高的位置。這件事最好平常就做好，不要等到洪水來了才准備。還要帶上你的求生背包。背包裡要有食物，有水，有救生毯，有備用衣服，有抗生素和其他一些藥物。只要有時間，臨時整理一個背包，裝上一下東西背上。或許你在路上走著走著，水就漲高了，你根本來不及走到高處避難。這時候你能做的，只能靠自己的游泳技術和應變能力。你要發現水上漂浮的一些物品，比如大塊的泡沫塑料，一條毛竹，一個密封的塑料桶，這些漂浮物，都可以幫助你活下來。你若是看到一個背包，或者結實的塑料袋，要趕緊抓住。你找到的一些小塊的泡沫塑料、空的飲料瓶，都可以塞到背包裡，這樣就構成一個足夠大的漂浮體。看到一些漂浮的空塑料瓶，無論是可樂瓶還是農夫山泉，可以塞到你的衣服裡面，前面後面各放幾個，衣服下面紮緊，這就是一件簡易的救生衣。在水里要記住避開電線和電線桿。觸電而死是水災中經常發生的事故。還要保護好自己的頭部不要被各種硬物撞到。路上遇到大樹，可以抓住爬上去，讓自己節省體力歇歇腳。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/28072424/unnamed-2.jpg)
大多數情況下，你會找到一個“小島”，就是某塊比較高的地。很多人都會待在一個個小島上。這時候馬上面臨的問題是：如何不凍死、不餓死、不生病。雖然周邊都是水，這些水卻不適合喝，因為洪水可能會把所有的垃圾堆、下水道都衝出來。

大部分人不會隨身攜帶淨水器、消毒片。但是很多人都隨身攜帶打火機，借個打火機可以燒水喝。不能指望看到洪水上飄著一個鋁鍋或者不銹鋼碗。但是你經常能看到一些軟包裝的紙盒子。比如說，光明牛奶或蘋果汁的紙盒子，裡面有一層防水層，不會漏水，你可以用它燒水。只要裡面有水，外面的紙盒不會燒壞。燃料可以用水上漂來的塑料袋、塑料瓶，這是不會打濕的好燃料。若是有木頭，也撈起來晾乾備用。聚酯的衣服也適合當燃料。水不一定需要燒開，只需要燒到65攝氏度以上，就可以殺死大部分有害細菌。所以，即使燃料不足以把水燒開，你也可以湊合著喝。天天喝熱水，你就不容易渴死、失溫凍死，即使完全沒有食物，也是可以活十來天的。偶爾遇到一條魚擱淺，或者漂來一具淹死的狗或豬，這時候可行的辦法就是烤熟吃掉。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/28072447/8073517161_29f221c1c3_b.jpg)
若有可能，不要居住在大水庫下游的沿岸。搬個家，換個房子，你就不必為此焦慮擔憂。如果你必須住在洪災高危地帶，做好一些準備。首先你需要準備幾套救生衣和救生背包。背包裡有換洗的保暖衣服，首選抓絨。衣服要用密封塑料袋封好防止漏水，這樣就可以用救生背包代替救生衣。背包要有結實的腰帶和胸帶，還可以使用前後雙背包，這樣便於前後浮力均衡。救生衣要盡可能選高檔舒服的，有些廉價救生衣在水里長時間穿很不舒服。最好平時練習一下救生衣的使用，在水里體驗一下。背包裡需要有幾張救生毯，在衣服被徹底打濕後，用救生毯鋁箔擋風和把人體熱量輻射回去，可以減少人體熱量損失，避免失溫凍死。還可以放鋁箔救生睡袋和救生帳篷，這些東西折疊起來都很小。背包裡要有體積小而熱量高的食物，比如壓縮餅乾、能量棒、牛肉乾。

要有抗生素和常用藥物，從板橋水庫的統計看，一旦發生水災，泡在水里的人大約一半到三分之二會患各種疾病。要有點火的工具：打火機、防水防風強力火柴、打火石。要有幾塊可以助燃的固體燃料，便於點燃打濕的木頭之類的燃料。要有可以燒煮的容器，比如一個鋁或單層不銹鋼的大飯盆。要有幾瓶純淨水。還要有一小包密封的消毒片以消毒各種不夠乾淨的水。若是有一小塊明礬輔助沉澱以淨化水，就更好。家裡還可以準備幾個頭盔，驢友攀岩、溯溪或漂流用的那些頭盔都可以，甚至一個自行車頭盔也有用，這樣可以保護頭部在水流中不容易受傷。最關鍵的一點：熟悉附近的地勢和道路，這樣水災時知道往哪兒跑。養成關注天氣預報的習慣，一旦看到上游有暴雨，先做好準備。也可以告訴朋友們自己住在危險地帶，若是誰得知堤壩倒塌的消息，即便半夜三更也要馬上給你打電話。現在是互聯網時代，消息傳得很快，你提早二十分鐘知道消息，和晚二十分鐘知道，可能就是不同的命運。

大多數水庫都是在夏天決堤。但是某些大型水庫在冬天也有很多水，這時候可能需要一件防寒救生衣，避免在水里凍死。這些東西，網上都能買到，很便宜。
還需要準備防水手電筒、電池，讓你可以在孤島上度過黑暗之夜。也可以夜間便於被救援隊看到。若有可能，準備一個體積很小的防水衛星電話，那時候，手機是沒用的，因為基站都沖毀了。

這些方法僅供大家參考，每個人都有自己的現實情況酌情考慮，當然我最最期盼的就是不要發生潰壩，我不想任何一位同胞被共產黨製造的“定時炸彈”帶走。
