## 解放軍部分潛艇洞庫位置
`2020-05-07 21:44 GM30` [轉載自GNews](https://gnews.org/zh-hant/196843/)

**編譯：P.W**

中國的9000余英裏（約1.45萬公裏）的大陸海岸線上分布著各種港口。（注：此處爲作者引用CIA中情局數據；中國官方數據爲1.8萬公裏。）

相較其他國家，中國的海軍基地數量大。中國人民解放軍海軍，通過分散部署，防止被突然襲擊。一些海軍基地，更進一步地，建造了地下洞庫爲重要戰艦和潛艇提供防護。

在這個精確打擊巡航導彈和鑽地炸彈的時代，建造洞庫看起來像是過時的主意。但這些洞庫仍可以防護某些空襲，也許更重要的是，洞庫在反軍事衛星偵察上起到的作用。此外只要不是被核彈直接命中，洞庫也能對核攻擊提供防護。

解放軍通常在岩石露頭（露出地面的岩層）處建造洞庫，這樣可以在洞庫上方提供較深的防護。洞庫入口通常朝向內陸（有水體連接），這樣更難被由海洋方向而來的力量攻擊。

最廣爲人知的是兩處洞庫是戰略潛艇部隊基地，其一爲位于山東省青島市崂山國家森林公園南的姜戈莊海軍基地（36° 6’20.76″N, 120°35’2.39″E，圖A）。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/07213318/%E5%9B%BEA.jpg)

圖A. 姜戈莊潛艇基地，092型（北約代號：夏級）及094型（北約代號：晉級）彈道導彈核潛艇都曾駐紮于此，右上方紅框爲可能的地下彈藥貯藏點。

圖片來源：[https://fas.org/blogs/security/2014/04/chinassbnfleet/](https://fas.org/blogs/security/2014/04/chinassbnfleet/)

其二爲位于海南省三亞市榆林港及亞龍灣龍坡的榆林基地（18°12’8.97″N, 109°41’39.34″E，圖B）。2008年已有報道094型彈道導彈核潛艇（北約代號：晉級）進駐了榆林基地。榆林基地爲解放軍海軍的潛艇（以及航空母艦）在南中國海海域的行動提供了便利。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/07213342/%E5%9B%BEB.jpg)

圖B. 榆林基地，核潛艇位于圖中右側龍坡基地，常規潛艇位于左側榆林基地，上方紅框爲可能的地下彈藥貯藏點。圖片來源同上。 其他帶洞庫的海軍基地少爲人知。例如位于廣東省江門市台山市的下川島基地（21°35’45.08″N, 112°33’5.14″E，圖C），紅點處爲一個較小的洞庫入口。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/07213434/%E5%9B%BEC-1.png)

圖C.還有位于浙江省甯波市象山港的戰艦潛艇維修廠（29°31’41.09″N, 121°41’16.98″E，圖D），紅點所示爲洞庫入口。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/07213503/%E5%9B%BED.png)

圖D.還有一些洞庫附近沒有海軍基地，這也許是增加防護的分散部署手段，也可能這些洞庫與解放軍海軍無關。例如位于浙江省甯波市石浦港南的大型洞庫（29°11’2.75″N, 121°56’35.68″E，圖E），其附近有一支導彈艇中隊，但沒有海軍基地。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/07213551/%E5%9B%BEE.png)

圖E.其他較隱蔽的洞庫還有，位于浙江省舟山市岱山島的洞庫（30°15’40.61″N, 122°19’1.43″E，圖F）,以及海南省三亞市東北方向的海棠灣（18°15’42.67″N, 109°43’41.13″E，圖G）。

解放軍海軍的洞庫（建造）與美國海軍條令相比有很有趣的不同點。這些洞庫會提供一些針對突然襲擊的防護；在較長時期的戰爭下，它們也很可能爲解放軍海軍潛艇部隊增加生存能力。

在利用岩石爲自己增加防護上，中國解放軍海軍並非孤例。2019年9月，瑞典海軍宣布重啓其冷戰時期海軍洞庫基地（位于首都圈外的穆斯克島）。穆斯克島洞庫基地可以容納多艘潛艇及戰艦並具有船只維修能力。台灣、北朝鮮、伊朗也擁有潛艇洞庫。
