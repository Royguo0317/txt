## 中共指揮部公告強迫蒙古族接受奴化教育，實則昭示天下，中共就是民族矛盾的罪魁禍首

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/20214008/9F60C36A-9489-4F68-8D3F-E2A46BECACE9.png)
2020年 9月15日，中共錫林浩特市國家通用教材相關工作指揮部發出公告（後附原文），以各種惡毒手段，強迫蒙古族居民送子女去接受奴化教育。

歷史演變到了二十一世紀，中國古代的朝貢體系已經不復存在。當今世界，既有民族自覺、民族自治理念大行其道的歐洲諸國，也有基於公民身份、模糊弱化民族和種族概念從而避免引發歧視的美洲移民國家。唯獨中國大陸，在中共的統治之下，實行的是假民族自治作為外包裝，實際上實行的是十九世紀殖民政策和種族、民族歧視混合的奇特體系。在這個體系下，宗教對抗事件層出不窮，民族矛盾不斷激化升級。可以說，民族矛盾、宗教矛盾的罪魁禍首就是中共。

中共奴役和剝削的是所有民族，不分漢族、維吾爾族、藏族和蒙古族等等，關鍵看是誰擋了中共的奴役之路。宗教擋道，中共就拆廟，那麼就製造出了宗教問題；為了盤剝土地和資源，中共就去搶，那麼就製造出了民族矛盾。毛澤東曾經說「民族、宗教無小事」，他並不是真的重視各民族、各宗教的利益，而是告誡其黨徒，奪取這部分人的利益要小心，否則容易把事情搞大。

以內蒙古為例，早期對內蒙古實行的是安撫政策，搞過幾年的「民族自決」，隨後降格為「民族區域自治」，然後就是一場一場的運動，橫掃蒙古族信奉的藏傳佛教寺廟，僧人強制還俗。八十年代之後，表面上似乎又回到了「民族區域自治」，實質上去過內蒙古的人都知道，那裡與內地的其它省份沒有什麼區別，瓶子里裝的已是中共「黨文化」的酒。同時這幾十年里，實行民族同化，安排大批內地民眾遷移到內蒙古，內蒙古自治區的蒙古族比例由中共竊政之初的50%左右，下降到今天的低於20%以下。

中共還覺得蒙古族同化的速度不夠快，又把矛頭對準了語言問題。語言是民族文化的載體，民族的歷史、宗教信仰、文化藝術、風俗習慣等與其語言唇齒相依。那怎麼下手呢？按照鄧小平的邏輯，「教育要從娃娃抓起」，只要小學生、中學生在幾乎所有課程上充分使用漢語，壓縮蒙語，這批人長大成為社會的中堅後，蒙語的社會地位就會大大下降，甚至成為野蠻落後的象徵，蒙語就會逐漸被拋棄。這就是中共的邪惡計劃！並且，取代蒙語的也不是真正傳統意義上的漢語，而是一種摻入了大量中共黨文化的語言體系，學習這種語言的同時，也就同時完成了中共黨文化基因的注入和洗腦過程。

一個歷史悠久的馬背民族是不會輕易屈服的，不會跪倒在中共民族滅絕政策下。中共自以為是的邪惡政策，已把數十年來沈寂的蒙古同胞喚醒，使其融入到全球滅共的潮流中來。讓我們拭目以待！

附公告原文：

中共锡林浩特市国家通用教材相关工作指挥部公告

一、在规定时间内按时送子女入学的家长，在各类项目和扶持政策上给予倾斜。

二、对9月16日前送子女入学的家长（包括电话报道）按时足额发放草原生态等奖补资金；9月17日及之后送子女入学的，全部暂停发放，并按照牧业年度载畜量标准立即开展载畜量核查，按照超载程度依法从重处罚。

三、对拒不执行国家重大决策部署，在9月16日18:00时前无故未按时报道入学的，高中阶段学生按自动退学处理，取消学籍；小学，初中阶段学生取消享受的各类补贴，资助。

四、对拒不执行国家重大决策部署，在规定时间无故未按时送子女入学的家长，由苏木镇场统计上报至指挥部，取消一切市本级安排的惠农惠牧政策性补贴。

五、对拒不执行国家重大决策部署，在规定时间内无故未按时送子女入学的家长，市农业银行、农村信用联社等金融部门五年内停止办理贷款手续。

六、对不按时送子女上学的家长，根据《中共国义务教育法》等法律法规，依法追究相关法律责任。

2020年 9月15日
