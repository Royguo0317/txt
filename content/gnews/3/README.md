对于美国来说，CCP这些由外而内的渗透力量是如何一步步蚕食正义并且在水下支撑起拜登这座冰山的呢？跟随爆料革命几年，我们从郭先生那里知道，这些渗透力量密如毛细血管，以至于堂而皇之地到达了郭先生的身边。在美国的所有中国人都有可能是渗透力量之一，甚至有的是不自觉地成为其中之一的。而背后更有深层次的deep state在运作。这种CCP军民融合带来的全民皆兵模式造就了今天的局面。

我们今天来试图剖析一下这些既成事实背后的行为动机以及这些动机产生的根本原因。

而出身于这个国土的人从小就被植入了这种基因，后来到了国外，带着这种基因理直气壮地搞各种学术、技术偷窃以及蓝金黄的行动。这表面的没有信仰其实背后是CCP的国内教育和制度决定的。
