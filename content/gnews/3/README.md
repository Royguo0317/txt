7/1/2021日， 美国总统特朗普在国会宣布拜登当选后，发表视频讲话，他表示现在的工作重点是保证政府平稳有序，无缝对接的权力过渡；但是他的团队将继续挑战选举结果，最后特朗普说了一句意味深长的话，一个不可思议的旅程开始了。

特朗普讲话首先谴责了发生在国会大厦的暴力事件，事件发生后，特朗普立刻调动国民警卫队和联邦执法人员，加强国会安全，并驱逐闯入者。他说这些暴徒不代表美国，触犯法律者将付出代价。

对美国大选，特朗普说他一直坚信应该进行选举法改革，验证投票者的合法身份和资格，确保大家对未来所有的选举有信心。但是，国会对选举进行了认证，那么他的工作重点将在平稳有序，无缝对接的权力过渡。

主教所说的时间，是否就是特朗普呼吁的“这个时刻，我们需要康复和和解。”要击败深层政府的病毒攻击，重整美国经济，需要所有的美国人团结起来，“这需要对文明价值观的全新关注，爱国主义，信仰，慈善，社区和家庭。”特朗普 看重的文明价值观，正是天主教徒都秉承的传统价值观。与大主教的主张惊人一致。
