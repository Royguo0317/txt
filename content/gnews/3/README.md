## 中共式創新：「消費扶貧」&#8212;-誰消費，誰買單！扶貧腐敗花樣多
`2020-09-01 21:07 GM01` [轉載自GNews](https://gnews.org/zh-hant/328914/)

國內新聞作者：α-Vega
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/01205941/66A09DE7-229D-41C7-ABA8-F9A3497ECD09.jpeg)
九月臨近，「扶貧」再次登上熱點話題，「消費扶貧」作為中共標榜的創新性扶貧形式，現啓動扶貧月行動。據人報海外版報道，即將到來的九月正是扶貧產品開始大量上市的季節。為進一步推動消費扶貧行動，國務院扶貧辦等11個部門組織開展全國消費扶貧月活動。

在8月28日國新辦舉行的新聞發佈會上，中共扶貧辦王大洋介紹，所謂的扶貧產品就是中西部貧困地區生產出來的具有帶貧、益貧效應的產品。

為實現習近平2020年消除全部貧困人口的目標，消費扶貧即是號召全民一起保政績。中共扶貧辦副主任洪天雲表示，消費扶貧是社會力量參與脫貧攻堅的重要途徑。習近平今年3月6日在決戰決勝脫貧攻堅座談會上明確指示要開展消費扶貧行動。

事到如今，購買扶貧產品成了人人有責。為完成指標，單位強制員工購買扶貧產品，員工叫苦：「按單位人頭數，指定地點去買貧困地區土特產，強制消費數額達到上千元，對普通職工簡直是要命了。單位里幾個家庭困難的，氣的直罵街。」還有人說：「1000指定地點買貴了好幾倍的東西，輔助老闆們脫貧。」 一位大姐說：「單位強制消費1000，而且牛羊肉暫時還沒有，我買1000塊錢玉米不得吃兩年啊。」

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/01210053/966653D4-E6E0-4FD8-A36A-4860DE4764DA.jpeg)
消費扶貧並沒有創造新的市場需求，僅是一種拆東牆補西牆的做法。從供求關係看，本應買郊區農產品，卻捨近求遠買大西北的扶貧產品。物美價廉的產品本身不愁銷路，而這些扶貧商品大多是滯銷產品。因地域偏僻，產品分散，運費高，人工成本高等原因，導致產品價格高，或是產品品質差。這些產品憑政策進駐京東、淘寶等電商平台或線下超市，這沒有問題，但是若無政府補貼可能依然滯銷。政府要求產品入駐各機關單位、學校食堂，將大大增加採購費用，或造成菜品單一，扶貧產品來了，一個月吃不完。

消費扶貧其本質是用行政手段，擾亂市場規律。對於定點消費，指定產品，政府補貼，其中又會滋生多少貪腐。

消費扶貧的壓力不僅轉嫁給普通員工，還讓地方政府變著花樣消費扶貧款，如：開設電商渠道，請主播帶貨等。洪天雲答記者問：「直播帶貨目前也出現了小情況，目前我們發現少數地區請名人大牌，最後算賬下來，直播帶貨請名人花的費用和直播帶貨以後的效應不成正比，成反比，花了好幾十萬、上百萬，最後直播帶貨的銷售才小幾十萬。」從中不難看出，地方政府外行管內行的無能。這樣草率的花費扶貧款，官員吃了多少回扣，我們不得而知。

消費扶貧，百害而無一益。既擾亂市場秩序，又增加財政開支，更增加普通員工的負擔，扶貧款項兜了個大圈卻進了貪官的腰包。如同中共的GDP，拆了蓋，蓋了拆，全是幻象。真正讓農民富起來，還應從本質出發，加速農村城市化、農業現代化才是根本。而中共這些假招子，完全是自欺欺人，麻痹百姓的假象。

根據國家統計局2019年國民經濟和社會發展統計公報，按照每人每年2300元的農村貧困標準計算，年末農村貧困人口551萬人。每年2300元，平均每月192元的貧困標準，徹底撕開了擋中央的遮羞布。試想一下，大概不用交水電煤氣費的地方，每月192元才可以活下去吧。這種一窮二白的生活，何來脫貧？即便這樣，地方政府自上而下，層層剝削，大官大貪，小官小貪，屢屢爆出村幹部私吞扶貧款，騙取房屋改造補貼等情況，貧困村民申請扶貧款，甚至需要給村官上供。為完成扶貧指標，強制脫貧。有村民反映，村幹部每人發2000元錢，強制村民簽字按手印，完成脫貧。更有甚者，強行刷白房屋牆壁就算脫貧。習近平所炫耀的2020年全面消除貧困，只是為了服務中共的面子工程。

今年天災不斷，農產品收成銳減，貧困地區有多少滯銷的扶貧產品不可考證。貧困地區更應存夠餘糧，切莫配合政府表演，導致農產品過度徵收，以防貧上加貧，造成糧食危機、飢荒風險加劇。
