#### 美俄之间的关系十分紧张。现在，两国外交部长都想开会进行个人对话。还应讨论拜登和普京之间举行会议的可能性。

俄罗斯外交大臣拉夫罗夫（*Sergei Lavrov*）和他的美国同事安东尼·布林肯（*Antony Blinken*）计划下周在北极峰会的间隙亲自会面。俄罗斯外交部星期三宣布，已定于5月20日打电话，进行单独对话，处理“双边关系和国际议程的关键问题”。华盛顿的国务院确认了计划的会议。

美国和俄罗斯之间的关系最近收紧了。除其他因素外，他们受到乌克兰冲突中日趋紧张的局势以及俄罗斯与被囚禁的克里姆林宫批评家阿列克谢·纳瓦尔尼（*Alexej Navalny*）的交往的负担。

鉴于两国之间的紧张关系，拜登最近提议在第三国与俄罗斯同行举行首脑会议。根据他的想法，这可能会在夏天在欧洲发生。拜登将于6月中旬前往欧洲参加北约峰会。到目前为止，克里姆林宫尚未公开普京是否会接受邀请。
