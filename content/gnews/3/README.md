我本是一名小镇青年，原本计划毕业后，可以夫妇成偶，堂上有父母，膝下有子女，当有三代，一箪食与一瓢饮，在陋巷过上简朴生活。自三年前接触爆料革命，我的人生轨迹从此彻底改变。无奈负心多是读书人，仗义每多屠狗辈，每每看见身边的读书人自甘堕落，勇气的脊梁被打断，我的信念无时无刻都在拔河。经过激烈的思想斗争，我秉着暂未泯灭的良知选择站出来。生前死后，情状渺茫，不可捉摸，当中的甜酸苦辣不在文中赘述。经过三年多的洗礼，我对人生亦有了重新的认识，如今想要回去以前的生活，现已不太可能。

晨曦时光，突然看到喜马拉雅农场联盟委员会00014号公告，关于凤凰农场已发公告宣布解散一事。联盟委员会作如下声明：凤凰农场是农场联盟委员会管理下的农场，接下来由联盟委员会统一接管。于是今宵提笔写文，有感而发，浅谈一下愚见。期盼各位战友继续团结一致，勿忘来时的初心，以及以后要走的路。

首先我们要清楚集体的定义，我们爆料革命的战友和新中国联邦的公民为何走到了一起？人以群分，物以类聚，因为我们都是有信仰、有使命的人，才会加入这个大家庭。那么何谓集体呢，集体具备五个属性：目的性、自发性、有限性、周期性、平等性。

目的性：毫无疑问，我们大家都是为了灭共建立新中国联邦，目的非常明确。

自发性：战友们都是自发地参加了爆料革命，自愿地给法治基金捐款，自愿地加入新中国联邦。我们都具备自发性，没有任何人强迫我们这样做。但是我们也要明白，集体有自发的开始，同时亦有自发的结束，**不能像共产党一样，只要加入了组织，就宣誓永不叛党。**

有限性：爆料革命组织不能强制要求个体在“非共同目的的领域”之外产生交集。譬如最近郭先生非常懊恼的各农场琐事。相关负责人不能要求战友做一些灭共之外的事情，更不能“论资排辈”和“倚老卖老”。如果缺少有限性，届时会很多出现“夹带私活”的事情。无可否认，在特定的时间，特定的事件，老战友做了很多不可磨灭的伟业，但这不能成为“道德绑架”其他人的理由。爆料革命组织不能拉帮结派，拔除欲箭，这样才可长久安稳。新中国联邦公民以及爆料革命的战友必须遵循喜马拉雅的契约精神。同时我们也要明白，这也是一个优胜劣汰的世界，能成为什么样的人，主要取决于自己的所作所为。

周期性：当集体的目的完成了，又或者发现完全达不到，这个组织的人就会选择离开。直到最后一个人的离开，集体也就自然解散了。之后，又会出现新的目的性，进而会再组织新的集体。譬如刚发生的凤凰农场被宣布解散一事，相信做出这样重要的决定，委员会经过了深思熟虑，绝对不是一时头脑发热。又譬如我们灭共这一个目的完成了，我们又会想如何帮助新中国重建，届时需要的人才与资源亦有所区别。这都是具有周期性的，不会一成不变，更不存在“倚老卖老”。

平等性：在不同规模的集体里面，每个人应该都是平等的，不同的只是每个人在这个集体所扮演的角色和位置。

具备了这五个属性的集体，那么这就是一个非常完美的集体。既能集思广益，做出有利于集体的决策，个人又可以取长补短，自我跃升，这样方可走得更远。反之，倘若加入了像中国共产党这样违背天理人伦的集体，则是灾难。

凡百元首，承天景命，莫不殷忧而道著，功成而德衰，有善始者实繁，能克终者盖寡。古代所有的帝王，承受了上天赋予的重大使命，他们没有一个不为国家深切地忧虑而且治理成效显著的，但大功告成之后国君的品德就开始衰微了。将近四年的爆料革命，这个世界已经发生了翻天覆地的变化，相信很多战友已经看到了胜利的曙光，但是我们要以史为鉴。

我们新中国联邦公民和爆料革命的战友同样肩负上天的使命，但是不是推翻中共自己做中共，那是绝对不可以的。所以郭先生亦多次说到新中国联邦不参与政治，但到时候会起到监督的作用，帮助华族同胞建立法治、民主、自由、信仰的中国。灭共之后，接下来才是真正考验的时刻。

我们不能因为发了几条推文就去“邀功封地”，夸夸其谈。相信很多战友都深受共产党的荼毒，包括笔者或多或少也曾幻想过，灭了中共之后就会有“千亩良田丘丘水，十房妻妾个个美”的幻想。当然，男女饮食，人之大欲存焉，但前提是要树立正确的观念，拿捏有度，切勿异想天开。

简能而任之，择善而从之，则智者尽其谋，勇者竭其力，仁者播其惠，信者效其忠。由于每个人掌握的资源有别，其所处的位置亦不一样，因而扮演的角色不同。深信加入爆料革命的每个战友都有过人之处，各操一艺，但定位要清晰，各司其职。郭先生亦多次鼓舞各位战友，我们要看得起自己，很多外国政要以及重要人物时刻关注我们G|TV、GNEWS的最新动态。所以我们务必要精益求精，团结携手，这是展示新中国联邦公民能力与形象的最好机会。我们要告诉全世界有良知的人，我们和中共不一样，中共不能代表中国人。

经师易得，人师难求。龙行一步，百草沾恩。郭先生是我人生的导师，亦是大部分战友的导师。相信在郭先生的带领下，爆料革命和新中国联邦会为华族同胞开万世太平，用尽毕生精力。匡扶正义，帮助万万华族同胞逐步走上正轨，或许这就是我们参加爆料革命的发愿与初心了。若在厄难，当愿众生：随意自在，所行无碍。
