上次我们说到中共是中国实际的统治者，他们现在正想要控制全世界，包括美国。

打个比方，一个持枪的强盗闯入一户人家劫持了全家老小，在把他们的钱财抢劫一空之后，还拿枪逼着他们为他干活，男人做苦力，女人任意被他蹂躏，甚至杀孩子吃肉，掏出胎儿吃肉，吃胎盘。而当周围的邻居路过或上门来串门的时候，他又威逼着这些受害者配合他演戏，假装他就是这家的主人，一而再再而三地骗过了邻居，甚至是巡警的眼睛，还把被劫持的被害人做苦力生产的东西卖给邻居，拿到的钱全都进了强盗的腰包。

这户人家就是中国，这个强盗就是中共，那些被骗过的巡警和邻居就是美国，欧洲等西方国家。这个强盗一再得手之后，他觉得这些邻居和警察太好骗了，他的胆子越来越大，他现在正打算把这些邻居甚至警察也全都制服，奴役全世界强奸全世界。而他最主要的手段就是，骗！

沙俄的波将金沿途搭起只有一堵墙的“村庄”制造假象取悦叶卡捷琳娜二世；二战时魔术师贾斯帕·马斯基林施展骗术制造假的亚历山大港从而掩护真的亚历山大港，他甚至把苏伊士运河都隐藏起来躲过了德军的轰炸；好莱坞梦工厂在制造逼真场景方面可谓登峰造极，但是所有这些跟中共相比，简直幼稚得不值一提。因为上述的造假都经不起近看、细看、三百六十度全方位看，但是中共的骗术，任凭西方朋友远看近看围着看，甚至就居住在其中看上十年二十年都看不懂啊！中共是真正的骗子王，敢于撒弥天大谎，可悲的是，他们成功了。

中共骗术的关键就是伪造身份。他们太了解西方人的弱点了。西方人最大的特点就是具有科学精神，不管什么都一定要把概念先搞清楚。工程师就是工程师，律师就是律师，教授就是教授，副教授就是副教授，绝不会把头衔、称呼授予不合格的人。举个例子，我的一位同学曾经在饭局上抱怨，在他工作的国外某大学，因为无法满足每年在规定的刊物上发表论文的数量而一直无法升格为教授，只能以副教授的身份受聘。这就是西方人。

中共利用的就是西方人这个弱点，即，他们只要看到对方的头衔，就会认为对方是具有相当资格和水平的人士，即使各个国家的水平略有差异，也不会差到哪里去的。但是他们大错特错了。中共就是敢于给完全没有水平的人发证书，授予头衔，就是敢于给一坨屎贴上黄金的标签拿出来卖！我再举个例子。有一次我和一个中共国的生意人一起吃饭，酒酣之际，这位老兄大言不惭地表示，他最得意的事情就是把5元钱买进的石头，用5万元卖掉。这就是中共。多年前我曾经听中共宣传，要在几年内培养出一百万律师、一百万注册会计师、一百万注册建筑师等等。当时我很不理解，这怎么可能呢？我太天真了，我很快就发现，所谓培养，就只需要发证书而已，至于拿到证书的人，就算是白痴也无所谓。这就是中共。

说到这里，有件事我很不理解。许多中国人到国外之后都因为在中国取得的文凭和职称在国外不被承认而面临就业的困难，不得不在当地重新考文凭评职称，非常辛苦。那西方人为什么又同时会承认中共国官方派出的所谓专家呢？怎么会对石正丽、钟南山这种所谓专家毫不怀疑呢？居然被他们堂而皇之编造的谎言、假论文所蒙蔽，还让他们在《自然》这样的刊物上发表论文，这种自相矛盾的做法是难以理解的。

总之，中共掩盖他们用武力绑架全体中国人的劫匪身份，假冒中国的合法主人奴役中国人为他们做苦力的同时，欺骗西方人和他们做生意，达到把全中国人的劳动所得装进他们的腰包的目的，关键就是用伪造身份的办法。西方人有议会，中共就也对应地伪造一个假的议会；西方人有政府，中共就也假造一个政府；西方有科学院，中共也假造科学院。假造的公司，假造的大学，假造的世博会，假造的奥运会，假造的艺术节，等等等等，这叫作“跟国际接轨”。总而言之，中共的一切都是假造的，都是徒有其名，而没有对应的内容。中共就这样假造出一个所谓国家，和一整套国家机构，而真正拿着枪的劫匪：中共，却躲在这一套假象的背后，吸中国人的血，拿去卖给被假象欺骗的西方人换钱，然后鼓了中共的腰包。

这里面有两个关键步骤，一是拿着枪顶着全体中国人让他们乖乖闭嘴埋头干活，二是笑脸蒙骗西方人出钱买他们手里的中国人当牛做马生产出来的东西，这两个条件缺一不可。除了假造出一整个国家之外，中共还做了两件事情，一是“走出去”，二是“请进来”。

“走出去”，就是把一些出国的华人绑架为间谍，让他们在国外为中共搜集情报，做虚假宣传，以及收买各个领域不同阶层的西方人，把整个西方在不知不觉之间腐蚀得几乎瘫痪了。

“请进来”，就是把西方的专家和政要请到中国来。这方面可以分为两种情况，一是短时间的访问，二是长时间地驻扎一起工作。前者比较简单，中共只要做好布景，安排好群众演员就可以骗过那些西方人了（就像我之前举过的例子，选好看的孩子让并非他们父母的好看的成年人假冒他们的父母扮作家人的样子在打扫干净的街道上闲逛粉饰太平）。而后者也不难对付，只要威逼那些在中国工作的西方人周围的中国人不许乱说乱动，同时又在那些西方人所到之处对他们采取远比中国人优厚的区别对待就行了。这就像在那些驻扎中国工作的西方人周围筑起一道无形的墙，让他们受到区别的对待（条件优厚），从而让他们对中国产生虚假的好感，自发地在国际上为中国做虚假宣传。

我还是举一些具体的事例吧。我接触过很多出国的华人，有我的同学，也有公司同事。他们没有一个人是用完全合法的方式出国的，他们递交的材料都或多或少掺杂了许多虚假证明。有一个同事他根本就不会说英语，只需要按照代理公司的要求提供各种假材料，然后在移民官面试的时候按照他们告诉他的用死记硬背记住的几句话回答就行了，一切早就靠花钱贿赂搞定了。而在他出国后，立即被中共的相关部门约谈，要求他在国外工作的同时，为中共做搜集情报和建立海外华人间谍组织的工作，他们说工作是很轻松而且安全的，并且他们会付给他报酬。由于出国时做了假材料，他的把柄已经被中共抓住，他只能束手就擒，乖乖答应为中共工作。当然，他们之中有些人本身就是红二代，骨子里就有为中共工作的自愿，这些人就会成为中共海外间谍组织里的基层骨干。这样，中共就毫不费劲地往全世界输送了数以十万计的业余间谍。这些人的工作也许是微不足道的，但是由于人数庞大，总体的效果是惊人的。他们会在中共领导人到他们所在的国家访问时被迫出来造势，他们会在推特脸书等社交媒体上为中共传播虚假信息，他们也可能被迫传播中共病毒，他们会在当地随时向当地中国领事馆报告各种情报，包括公司的商业或技术数据，西方同事的隐私信息，公司电脑系统的安全密钥等等。而中共借此就有机会接近并窃取一些核心技术，或要挟以及收买一些重要岗位的西方人，比如律师、证券交易员、银行职员、政府职员、选举观察员、地区议员等等，不知不觉间就腐蚀了西方社会的精英阶层。这是关于“走出去”的部分。

“请进来”呢？我曾经陪一位在华工作的女士一起去一家医院，她偶然发现了“计划生育手术室”，像发现了新大陆一样感到很稀奇，拍了好几张照片。她不知道，这是所有中国女性都知道的常识。在她们结婚的时候，街道的计划生育干部就会找到她们，告诫她们不能随便怀孕，“不要不通知我就怀孕哦，否则我可不给你名额。”这是我所在的街道的计生干部对我老婆说的话。但是在华的这位外国女士是听不到这样的话的，也没人会对她做计划生育的限制。这样的对西方人的区别对待比比皆是，几乎存在于生活的所有方面：在银行邮局车站码头便民中心等等地方，西方人会被优待不排队或少排队而优先得到接待；在办理公司事务或个人事务的场所也是如此；在工作的地方，西方人也会被工作单位优待，提供优厚的住房条件，出行工具，高工资，好的助理，好的实验室等等。而他们的中国同事是绝不可能得到相同待遇的。以我多年的观察可以这样讲，在中国工作的西方科学家得到的金钱、科研条件、荣誉是远比他们在本国所能获得的更多的，所以即使在他们身边有勇敢的人对他们说出一些关于中共的真相，这些已经被中共的好处浸泡得骨头都酥了的西方人也不会相信。中共会充分利用这些西方人在本国的影响力把虚假的信息传向全世界，并通过他们把中共的所谓专家实为间谍介绍给许多大学和科研机构，甚至政府部门等要害岗位。我的同学之中就有至少两人在美国政府部门工作。

虽然很多西方人被中共蒙骗，但并不是所有西方人都对中国不知情的。有时中共也会把一些事情透露给少数西方人，但那是在他们已经完全被中共捏住了把柄动弹不得的情况下，同时中共也对他们许以厚利，甚至给他们换器官之类的特殊待遇。他们和中共已经结成利益共同体，完全成了中共的同谋和帮凶。

中共用这种方式攻无不克，几乎攻陷了整个西方社会的所有要害部位。在前文中我曾说过，我的一个同学曾经一度负责为中共在某国某地区劝说和组织当地华人为中共领事馆工作。有一次我和他一起吃晚饭，我问他为什么现在美国和中国的关系不断恶化，是不是因为美国意识到了中共企图控制美国而展开了反击，再不反击美国就要输了？他轻蔑地笑了，说，美国现在才想到要反击已经晚了，他们已经输了，而不是就要输了。他强调了一遍：美国已经输了，他们现在已经根本没有能力反击了。我当时觉得他太过夸张，但是现在我不得不承认，他的话是有一定道理的。中共试图控制全世界，而且他们正在得手！作为被中共奴役的中国人，我们有义务要告诉身边的西方朋友这些关于中共的真相，全世界爱好和平与正义的人必须联合起来才能打垮中共，拯救世界的同时，拯救我们自己！
