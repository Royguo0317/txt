## 【香港要闻】路透社揭港警擅取民主派银行纪录 国安法下再无私隐
` 喜馬拉雅大使館-粵語組` [轉載自GNews](https://gnews.org/zh-hans/832643/)

搜集：文燕

编撰：文燕

审稿：卡西欧

上传：文粤

**警方月初以【港版国安法】颠覆国家政权罪名，滥捕55****名参加初选人士，旨在灭声。路透社再报道警方未获被捕人士知情及同意下，私下向多间银行要求索取其帐户纪录以作审查，严重侵犯私隐。涉事四间银行，汇丰、恒生、花旗、渣打查询，均拒绝就个别事件作评论。警方亦同样拒绝回应审查该批银行帐户资料的原因，强调不会披露调查行动细节。金融管理局发言人则回应称，财务机构应该协助执法部门的调查与执法工作。**
![]()![](https://gnews.org/wp-content/uploads/2021/01/0128-aaimage001.jpg)邹家成表示，警方擅自取得其银行账户纪录，审问涉及初选时的众筹款项来源及去向。 （大纪元资料图片）
55名香港民主派人士去年起因民主派初选而先后被捕，他们被指控涉违反「颠覆国家政权罪」。路透社1月26日报道，至少6名在月初被捕的人士表示，警方在未得到他们同意下，取得他们的银行帐目纪录，并在他们被捕后查问部份交易详情。初选被捕人士之一邹家成向传媒证实，警方擅自取得其银行账户纪录，审问涉及初选时的众筹款项来源及去向。他批评警方此举严重侵犯市民私隐。

**邹家成：估计警方不仅获银行财务纪录 ****更由银行获取个人资料**

邹家成向港媒表示，警方曾取出包括花旗银行、恒生银行、应用程式 PayMe 的财政纪录，合共10万元的资金来源。事后警方还没收他的花旗银行及恒生银行的提款卡及信用卡。

另一被捕人士刘頴匡指警方曾经向他展示其恒生银行帐户纸本月结单，并要求他解释初选期间的数十笔交易。他指相关帐户用作初选众筹款项。至于另外四名不愿具名被捕人士同样证实，警方曾于录影会面期间出示银行帐户纪录，当中两人属于渣打银行。

路透社又引述一名知情本地银行高级行政人员透露，警方过去半年，亦即【国安法】生效后，对民主运动参与者银行帐户资料索取次数激增一倍。该银行高层无奈称，银行须要遵从警方要求，没有太多选择余地，仅能建议倘帐户交易资料被分享，甚至帐户遭冻结而认为受侵害的人士，可循法律途径追讨。

路透社就事件曾向涉事四间银行，包括汇丰、恒生、花旗、渣打查询，惟全部均拒绝就个别事件作评论。至于警方亦同样拒绝回应审查该批银行帐户资料的原因，强调不会披露调查行动细节。金融管理局发言人则回应称，财务机构应该协助执法部门的调查与执法工作。

**战友点评：**

国安法权大如天，【港版国安法】生效后，警方无须经过法官的批准，便可以要求银行冻结某人户口，或交出某客户的户口，而这里银行必须服从没选择。

**当最高权力失去监察和制衡，绝对的权力必定衍生绝对的腐败。**

从前的香港，执法部分是不敢乱碰的，例如怀疑某部门或某人洗黑钱，执法部门是必须有确凿的证据，在得到法官的批准后，才能向银行索取个资​​料来冻结某人户口。如此谨慎是因为冻结个人户口就等同剥夺私人资产，这与香港的国际中心形象息息相关。因此个人资产和银行私隐必须受到保护。

汇丰行政总裁祈耀年（Noel Quinn）昨天接受英国下议院外交事务委员会质询时表示，当港警要求调阅帐户资料，汇丰银行无权「评断动机」 。今天，国安法权大如天，在绝对权力下，警方向银行索被捕的初选人士的私人资料，银行在没选择下完全配合。可是，有了第一次，就会有第二次，然后无数次，当最高权力失去制衡，香港国际中心的形象将会受到无可逆转的、无可补救的损坏。
