## 【信仰之声】从犹太人的割礼，想到了中国人的面子

作者：纽约香草山福音部 霹雳姐姐



![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-109.jpg)



在中国文化中，面子是首要的。今天听到路德在早间的时评中说：昨晚看了《走向共和》，慈禧太后面对兵临城下的八国联军时说道，“打人不打脸，开战”——她要面子；就这样，慈禧还要让各国使节跪下朝拜她——她仍是要面子。

![]()![](https://gnews.org/wp-content/uploads/2021/03/image-374.png)图源网络

面子，是中国人在处理人际交往时一个不可忽视的人情媒介。“给面子”、“留面子”、“死活要面子活受罪”，说的就是中国人的爱面子。遇到有损名誉的事，国人宁可憋在心里不说出来，也生怕被别人知道、遭到取笑；更有甚者，到了“死活要面子活受罪”的程度。因此有人说，不了解面子，就不能了解中国人。

记得鲁迅说过：面子就是中国人的辫子。深刻呀！

昨晚查经《罗马书》第2章的25到29节：“25你若是行律法的，割礼固然于你有益；若是犯律法的，你的割礼就算不得割礼。26所以那未受割礼的，若遵守律法的条例，他虽然未受割礼，岂不算是有割礼吗？27而且那本来未受割礼的，若能全守律法，岂不是要审判你这有仪文和割礼竟犯律法的人吗？28因为外面做犹太人的，不是真犹太人；外面肉身的割礼，也不是真割礼。29惟有里面作的，才是真犹太人；真割礼也是心里的，在乎灵，不在乎仪文。这人的称赞不是从人来的，乃是从神来的。”

犹太人认为，割礼就是得神喜悦的保证。但是，属于神的真正标记不是在人身体上留于外表的记号，而是圣灵在人里面重生的大能——也就是保罗所谓“心里的”割礼。

人类的始祖亚当和夏娃吃了分辨善恶树上的果子，第一时间知道了羞耻，用树叶遮挡下体。后来，神又与犹太人立割礼的约。这是为什么？圣经是犹太人的历史，同时，它更是影响西方文明的历史。由此，中西方的文明从割礼开始，就产生了界限。神在他的选民犹太人的命根那儿割礼立约——这不是靠人的聪明能想到的，真是因着神的大能和智慧。所以，西方人面对丑事、错事时，敢于面对、承认错误。美国前总统克林顿的蓝裙子事件，最后是通过他的道歉才平息了他之前的谎言和过错。

如果这种事情发生在中（共）国，习进平会道歉吗？想都别想，谎言与面子是中共领导和统治中国人的法宝！
