美国海军“马辛岛”号两栖远征打击集群,于12月6日兵分两路开进南中国海水域进行战备部署.
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/06072358/20201206193224925-scaled.jpg)
“马辛岛”号(LHD-8)两栖攻击舰从北向进入南海海域;“萨摩塞特”号(LPD-25)两栖船坞登陆舰从南向进入南海海域.

此前,该两栖远征打击集群刚刚从美军太平洋夏威夷基地出发,经关岛直奔南中国海海域.

该两栖远征打击群极有可能扮演刚刚重组的美军太平洋第一舰队先锋力量!美海军部长日前表示,未来的太平洋第七舰队将主要负责台海方向的战备巡弋任务.而重新组建的太平洋第一舰队则主要负责太平洋与印度洋间的战备巡弋任务,其主要目标区域便是南中国海水域.
