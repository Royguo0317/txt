##  **暫停或終止與香港的三項雙邊協議** 

中國共產黨已採取了嚴厲的措施以侵蝕北京自己在聯合國登記在案的《中英聯合聲明》中向英國和香港人民承諾的五十年高度自治。川普總統已明確表示，美國因此將以“一國一制”看待香港，並對破壞香港人民自由的個人採取行動。

總統關於《香港正常化》的行政命令（EO 13936）對北京行為做出了一系列的回應，包括暫停和取消對香港的優惠待遇。

作為持續實施中的系列措施的一部分，我們於8月19日通知了香港當局關於暫停或終止三項雙邊協議的決定。這些協議涉及移交逃犯，移交被判刑人員，以及對船舶國際運營所得收入的相互免稅。

這些舉措凸顯了我們對北京決定強行實施《國家安全法》的深切擔憂，該法嚴重破壞了香港人民的自由。
