2021年1月14日，澳交所上市电动摩托车制造商Vmoto宣布与战略合作伙伴ˎB2B大客户Greenmo集团签订总值1300万澳元的供货合同。今年一季度向欧洲市场供应5902台电单车，用于欧洲共享出行业务。

受CCP病毒疫情影响，欧洲各国政府鼓励民众使用电动车出行，减少乘坐公共交通工具。共享出行市场迎来发展机遇。

Vmoto公司在中国南京拥有3万平方米的生产工厂，将低成本的中国制造能力与欧洲设计相结合。旗下品牌包括面向国际市场的E-Max和面向中国国内市场的速珂（Super Soco）。

在Vmoto 这个案例，笔者看到CCP一边制造和散播CCP病毒，一边准备发疫情大爆发后的横财。比如通过游说，让各国政府鼓励民众使用共享电单车出行，然后CCP盗国贼掌控的电动摩托车厂大赚一笔。这是所谓“全球大重启”计划的小小缩影，而中国被奴隶剥削的老百姓是其中的关键一环。
