**宣传前后矛盾，百姓一头雾水**

在“大跃进”式的宣传鼓动下，三至五月间中（共）国迎来了疫苗接种高峰。据中共官媒新华网宣传[1]，“国家卫健委公布最新数据，从历时3个月首破1亿剂次，到仅用16天即完成第三个1亿剂次”中（共）国疫苗接种速度喜人。官媒一方面大力宣传“国产疫苗的安全可靠已得到实践的充分证明”，但另一方面又说“无论是加快疫苗接种还是织密防控网络，都不能有’喘口气’‘歇歇脚’的心态”。为什么老百姓连停下来喘口气也不行呢？官方不是说国产疫苗的防护性得到了充分验证了吗？老百姓也是一头雾水，废了半天劲，排了半天队，生活便捷性却没有任何改善，该有的风险还是存在。那这些疫苗数据是否真实呢，这只有中共心里最清楚。

**疫苗接种，迷雾重重**

我查阅了许多政府部门的新闻[2][3]，从中总结了以下几点，无一例外地都提到了：1）保障规范接种，接种安全；2）保障接种能力，保障群众后续接种的正常衔接；3）保障冷链和物资储备。

可见这几点是目前墙内疫苗接种中反映出的几个问题：1）上级领导宣传上要求老百姓“应接尽接”，下面各级地方政府为了求快、求数据、求政绩，搞快速/加速接种，搞24小时连轴转。但是你们究竟有没有按照疫苗接种指导手册执行呢？接种人员、设施是否专业呢？安全保护措施，防控措施是否齐全呢？这些问题都在社交媒体上被老百姓曝出，广受质疑。2）疫苗接种存在登记快，接种慢，接种混乱的情况。在社交媒体上有许多人反馈，接种区域无序；沟通混乱；排队时间长，拥堵；排了队却等不到疫苗等等问题。3）疫苗的存储与运输是否合规？这方面也受到质疑，随着炎热的夏季到来，这一问题将会更加严重。

另外最近还有新闻显示中共研发的疫苗新包装多为“两人次用”，就是一剂疫苗，先给第一个人接种一半，然后第二个人接种剩下的一半。不知道其他国家的疫苗是否有过这种先例，是否认可这样的操作。以上这些问题都导致了墙内疫苗接种迷雾重重。

**是知情、自愿，还是政治考核?**

同时政府各基层以政治考核来倒逼疫苗接种，考核所在地老百姓的疫苗接种率。一些地方政府完全将疫苗接种作为主要工作考核依据，将其列为重要的政治任务，而将本职工作扔到一边，这实在是荒唐至极；有些地方政府实行疫苗接种对标、评比，对不达标的负责人进行约谈，批评，惩罚；还有些地方政府采取堵门宣传、反复上门骚扰、大喇叭放噪音等恶劣手段，美其名曰是对老百姓宣传、排摸、关爱，实际上却是严重影响了老百姓的日常生活。这些低劣的政府行为都与中共官方所谓的“接种人知情同意、自愿接种”完全背道而驰，是赤裸裸的以假治国。
