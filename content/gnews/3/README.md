班農先生連線Jasmine，讓Jasmine從她個人的故事出發，向大家介紹一下她是如何成為基督徒的，以及在中國共產黨的統治下基督徒的狀態是怎樣的。Jasmine首先就班農先生及其團隊為中國人所付出的一切表示感謝，感謝班農先生將東西方的對比差異展現在了人們的眼前，感謝班農先生團隊將中國共產黨和中國人民區分開來，中國共產黨綁架中國人民已經70年了，中國人民過著可悲的生活，感謝替中國人發聲，並且也非常感謝郭文貴先生領導的爆料革命，是爆料革命為中國人民帶來了希望，現在還有了GTV可以自由發聲，感謝爆料革命所作的一切。

Jasmine說她出生於中國大陸，現在是加拿大籍華人，她在中國的時候就信主。她親眼看著中共多年來對地下教會和城市中的教會進行滲透，中共從最初的起點就對所有的宗教信仰懷有敵意，尤其是基督教新教、天主教以及穆斯林，因為這些宗教都是從國外傳入的，所以共產黨認為這些宗教是國內教徒與網路防火牆以外世界溝通的渠道。

在中共發動的文革時期，人們因為有宗教信仰而遭到批鬥，教堂也被摧毀了，那些不願意拋棄信仰的人不是入獄甚至被處死，就是遭到整個社會的孤立而找不到工作；在80年代的改革開放時期，中共改變了策略，當時教堂還沒有開放，所以人們都是在家庭教會參加聚會，只能在家庭教會中小聲地唱歌並且保持低調。80年代末教堂重新開放了，信徒可以重新回到教會去聚會了，然後中共總是派出一些線人對教會中的信徒進行監視；即使是在今天，中國的信徒也只能在教堂中傳福音，而不能到教堂以外的地方談論宗教信仰，不能對18嵗以下的年輕人談論宗教信仰，甚至給小孩講聖經故事、唱贊美詩都會被認為是洗腦。如果在托兒所被發現給小孩講與上帝有關的故事，立刻就會有人制止，說你破壞了規矩。在中國，中共控制著教會，他們通過宣傳機器告訴大家宗教是迷信。因為中共通過統戰部控制著三自教會，即正式註冊的外面所謂的正式的教堂，中共實際上在三自教會監視著每一個信徒，所以人們只有在家庭教會才有一點自由能真正敬拜神。

班農先生提了一個問題：因為在西方，人們認為毛澤東發起的文化大革命運動是為了破壞中國的家庭關係，打破中國傳統文化的傳承，為什麼他們會迫害宗教呢？Jasmine回答說：中共的一切所作所為最終都是為了摧毀一切，共產黨是要將共產主義作為新的宗教來讓人膜拜。文革之時，人們被要求背誦《毛澤東語錄》，在用餐和學習、工作之前都要背誦，而且被要求看著牆上的毛澤東畫像懺悔，中共就像邪教組織一樣強迫中國人接受他們的共產主義宗教。班農先生又問：因為之前聽說過一些人在文革中被折磨的故事，所以想知道如果人們不願意在中共面前放棄原來的宗教信仰，結局會怎樣？Jasmine回答說：那麼可能會被抓去勞改營，我的一個親人當時就因為不願意放棄信仰而被抓去勞改營關了他十年。班農先生也很想知道，文革的時候，基督教這樣的宗教信仰是如何保存下來的呢？Jasmine說基督是福音，即使是在那個年代，基督徒也是一對一的傳福音，當時冒著極大的風險，但是基督徒仍然小心翼翼地秘密聚會。班農先生想知道當前美國以及全球所有 有宗教信仰的聽眾，能夠為中國有宗教人士做些什麼？Jasmine回答說：首要的就是不要相信中共的宣傳，中共試圖讓全世界相信中國人沒有為宗教自由做好準備，但這都是中共的謊言。其次，西方世界要認識到中國共產黨的邪惡本質，中共就是魔鬼的化身，世界要打破中共建立的網路防火牆 對真相的封鎖控制，把真相帶給中國人。

在中共的殘酷迫害下，有很多人放棄了信仰，但是也有人沒有放棄。班農連線Joyce。Joyce給大家帶來的就是一個不放棄信仰的故事，這是一個非常勇敢的女人，她是一個年輕的牧師，人們叫她楊牧師。楊牧師因為拒絕加入三自愛國教會運動而被關進監獄十二年，她以前是當地非常有名的家庭教會的帶領。她在勞改營的十二年中，她每天晚上需要照看300英畝的果園，她的堅韌感動了勞改營負責人的妻子，兩個女人開始有交流，最終負責人的妻子也成為了基督徒，這件事情被負責人知道了，他差點殺了楊牧師。楊牧師在1979年被釋放了，她從未放棄自己的信仰，出獄後每個星期天她總是打開她的家門，向人們傳福音、作見證。
