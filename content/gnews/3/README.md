## 【香港要聞】【戰友點評】許智峯批港警擅改傳媒代表定義 變相自設發牌制「由行外人管行內人」【跟蹤報導】

香港警方宣布將修訂《警察通例》中「傳媒代表」定義，引發社會噓聲一片。
 民主黨立法會議員許智峯二十二日批評，警方變相自設發牌制度，規管新聞機構工作，斥此舉為「由行外人管行內人」。
 許智峯表示，警方剔除記協及攝記協會員認可身份，已妨礙新聞自由及記者工作，且修訂從未經任何公眾討論，亦並非由政策局牽頭及諮詢任何傳媒機構，只由警察公共關係科單向公佈，警方更可藉此排除不喜歡的傳媒代表，趕絕網媒、公民記者及學生記者，塑造「警察城市」。他強調部份揭露警暴的畫面是來自網媒，例如西灣仔開槍交通警及8.31太子站的警暴事件，指警方隨便修例無理，剝奪了公眾知情權。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/23014411/photo_2020-09-23_01-28-17.jpg)圖片：TG-反惡法公海谷

中共國的特色獨裁高壓政策，已源源不斷，浩浩蕩蕩滲透到香港體制內，港府同流合污，甘做傀儡爪牙，百分百配合的醜態令人齒冷。郭文貴先生已經公開警告香港高官，以美國為首的西方世界，很快會採取針對他們家庭的雷霆制裁行動，希望這些高官能夠認清形勢，及早抽離，免得全家都成為中共的殉葬品。
