2020年09月28日，人民日報海外網刊登來源爲科技日報的“首批疫苗志願者結束半年觀察期：陳薇將軍給我寫下‘感謝勇士’ ”的文章，以武漢志願者梁某結束I期疫苗半年觀察期結束爲背景，講述他兩次接種陳薇團隊研發的疫苗的過程。

第一次是在3月26日接受Ⅰ期“重組新型冠狀病毒(2019-COV)疫苗（腺病毒載體）臨床試驗”，第二次是在9月27日，接受觀察期爲一年的“重組新冠疫苗(腺狀病毒載體)Ⅰ期加強免疫臨床試驗”，以及在Ⅰ期臨床試驗結束後，陳薇給參加Ⅰ期臨床試驗的志願者們“親自挑選禮物”的故事！

中共的疫苗試驗結果數據不透明，傳聞試驗者出現不適反應。這108位志願者，是從5346人精心挑選的，還簽了保密協議。盡管中共國毒疫苗事件層出不窮，108位甚至更多志願者，被中共成功洗腦，甘當小白鼠，可謂真的“勇士”！

闫麗夢博士的科學報告，向世界揭示了新冠病毒是中共軍方實驗室制造出來的生物武器。陳薇院士，作爲軍方病毒專家，外界認爲是中共生物武器第一功臣，後被中共派去掌管武漢P4實驗室，“毀屍滅迹”。

她“輕車熟路”，讓疫苗如此迅速上市。爲了其政績，快速上馬Ⅰ期疫苗臨床試驗。如今假仁假義，給疫苗志願者們“親自挑選禮物”，可謂黃鼠狼給雞拜年故事的現實版上演，非常滑稽！

自從中共病毒疫情開始，中共就不停講故事樹英雄。從高福王廣發石正麗，到鍾南山李蘭娟吳尊友，再到張伯禮張定宇陳薇！一個又一個地“造神”！最倒黴的就是王廣發，牛逼剛吹完自己先倒下了！還有流傳出來的“藝考生”病毒所長王延轶與石正麗的宮鬥劇連甄嬛傳都“相形見绌”！真是看得人眼花缭亂！

從“大國抗疫”、“戰疫情”到“2020開學第一課”，再到“同心戰疫”、“最美逆行者”！終于在9月8日的“全國抗擊新冠肺炎疫情表彰大會”上，上演“封神演義”！然後再讓這些神回到人間“普渡衆生”，“人民英雄”陳薇給志願者發“禮物”，還讓梁某通過媒體曬出他的“幸福”。無辜的志願者們，將來知道了真相，會作何感想？

然而，兔死狗烹、卸磨殺驢的故事緊跟著來了！最近中國多位病毒專家如周育森、趙振東心髒病發，“去見了毛主席”！路德社最近透漏中共尚有一個70人名單。不免擔心，“黃鼠狼”陳薇院士的後面，有沒有一只老虎，正張著血盆大口呢？
