## 官方消息：來自中國的神秘種子包裹被送往三個州的多民眾手裏
`2020-07-26 02:35 GM77` [轉載自GNews](https://gnews.org/zh-hant/276449/)

戰友之家新聞譯制組

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/26022727/image0-25.jpg)

一個來自中國神秘包裹，裏面裝有未經檢驗的種子，被隨機送往多州。

居住在華盛頓 猶他州和佛吉尼亞州的多位居民都收到了郵件，似乎是從中國送來的小包裹裝的種子，官員對記者說道：“今天我們收到報告，有人收到了來自中國的包有種子的郵件，但是他們並沒有下單購買這些東西”華盛頓農業部官員周五告訴我們。“這些種子是以包裹形式發送的，通常標註說裏面是珠寶。這些不請自來的種子極具入侵性，會給當地農作物帶來疾病，或是會傷害當地家禽”

這些種子通常以密封包裝的形式郵寄過來，大家不應該隨意打開這種包裹，官員說道。應當將其以農業走私的理由報告給美國農業部門。

“入侵性物種會嚴重危害當地環境，取代或是毀滅當地植物昆蟲，甚至嚴重毀壞農作物” 佛吉尼亞農業與消費者服務中心的官員在周五的聲明裏如是說到。“采取各種措施阻止有害物種的引進，可以減少入侵物種侵襲和控制成本以及減緩入侵困擾的風險”

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/26022709/image1-15.jpg)

lori culley 住在猶他州的圖埃勒，告訴福克斯新聞，周四那天她在自家郵箱裏發現了兩個小包裹，上面標記有中文。

“我打開看了看，是種子”她告訴新聞站，“很明顯裏邊不是珠寶”

據福克斯新聞報道，在猶他州，至少40人收到了這種神秘的種子。佛吉尼亞和華盛頓州有多少人被郵寄了種子包裹還未知，但是官方人士說“已經有好幾個了”
