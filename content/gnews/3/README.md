中共這個邪惡體制，一方面不擇手段隱瞞真相，一方面千方百計打壓被揭露出來的真相。近幾天，推特48小時內關閉閆麗夢博士的新開賬號，福克斯著名主持人Tucker Carlson臉書上發的閆博士採訪視頻被屏蔽，爆料革命戰友們的無數賬號持續被刪帖、查封等等，這些事件背後的導演都是中共。這些行為實際都違反了美國保護言論自由的憲法權利。

1791年緊隨美國《憲法》頒布實施的第一修正案，明確禁止剝奪言論自由，明確禁止侵犯新聞自由。言論自由，是美國民主自由法治的基石，沒有了言論自由，自由女神高舉的自由燈塔就形同虛設。

美國憲法保護的自由言論涵蓋豐富，包括並不限於政治言論、匿名言論、遊行抗議形式的表達，甚至包括仇恨和冒犯言論。也就是說，無論言論多麼冒犯或多麼令人反感，《第一修正案》都予以保護。

1931年的尼爾訴明尼蘇達案和1971年的紐約時報訴沙利文案，美國聯邦最高法院都判決：任何形式的出版審查行為，都違反了第一修正案，屬於違憲行為。同時，紐約時報訴沙利文案的判例，還將誹謗訴訟中的舉證責任倒置由原告承擔，也就是說，限制言論自由必須由原告出示法庭確認的證據。

中共作為世界公認的專制獨裁者，已將打壓國內言論的流氓手段搬到了美國，多年來，中共利用藍金黃控制了美國主要的主流媒體及社交平台。對不利於中共國的任何形式的言論和真相揭露，這些媒體、平台主動自我審查，動輒就刪帖、屏蔽、封號，下三濫手段中共國內如出一轍。

可悲的是，美國社會並沒有完全認識到這一點，還沒有對這些限制、打擊言論自由的肇事媒體進行有效打擊。好在川普政府的官員和司法機構已經有所反應，對推特擅自查封閆麗夢博士賬號的行為，納瓦羅迅速發推指責，司法部上週起訴了5名中共國的網絡黑客。這些行動明顯還是不夠的，公然踐踏憲法權利的媒體行動還沒有得到控制。

有意思的是，加州一聯邦法官9月19日還以保護言論自由為理由，裁定了暫停美國政府對侵蝕美國的微信、抖音的禁止令。理由冠冕堂皇，但沒有人去問，微信、抖音這些媒體是否保護使用人的言論自由呢？顯然不是。微信、抖音的刪貼、封號等惡劣行徑更是登峰造極。更嚴重的是它們盜取各國使用者手機上的種種個人信息，以中共的邪惡思想洗腦年輕人，危害美國及各國的國家安全。試想一下，如果美國政府不及時出手，推特、油管會以極快的速度，發展成為美國的微信、抖音。到那時候，美國還是美國嗎？微信最終一定會被滅掉。

美國保護言論自由這一憲法權利已經刻不容緩。 “我堅決反對你的觀點，但是我誓死捍衛你說話的權利”，這是對自由社會言論自由的優美表達。美國要保持在自由世界的權威，就應該進一步採取措施，打擊破壞言論自由的相關媒體和社交平台，加大保護言論自由。
