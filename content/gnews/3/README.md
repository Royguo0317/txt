6. 。 美國州長奧薩哈欽森呼籲: 。

要抗擊冠狀病毒,請戴口罩!

由州長奧薩**·** 。 哈欽森。

美國州長Asa Hutchinson發出呼籲:要抗擊冠狀病毒,請戴口罩-最新調查顯示,大多數美國人都同意。 “充滿挑戰的時刻總是喚起人民的偉大。 我們在第二次世界大戰期間的廢舊運動和自由債券運動中看到了它。  9/11之後,當美國人衝向悲劇而不是為了挽救生命而逃離悲劇時,我們看到了這一點。 今天,我們戴著口罩的簡單舉動再次看到了這種精神。  ”

隨著COVID-19病例激增和死亡的增加,醫學專家和我們自己的醫生告訴我們,在公共場所戴著口罩是保護自己和周圍人的有力工具。 如果足夠多的美國人在他們聚集的地方戴足夠長的時間戴口罩,我們可以大大減少傳播並平息冠狀病毒。

這就是為什麼我最近簽署了一項行政命令,要求在阿肯色州覆蓋居民佩戴口罩。 全國許多州長都發佈了類似的命令,還有更多人可能會效仿。  調查發現,有86%的美國人認為幾乎在所有公共場所都戴口罩,這會有效減少冠狀病毒的傳播。 絕大多數美國人(十分之八)贊成州或地方政府頒布此類要求。

美國人擔心,顯然希望領導人採取行動保護他們。 十分之七的美國人認為最嚴重的病毒還沒有到來,三分之二的人報告說該州的感染正在增加,五分之三的人非常擔心家人中有人會感染該病毒。 戴口罩的情況很有說服力。 我們的人民看到,口罩可以用來恢復某種正常感,是一種強大的工具。 戴著口罩表示對每個人的禮節,尊重和關懷,尤其是對那些生病,年長或最易患病的人。

在民族危機之際,美國人一直挺身而出,伸出援手。 我每天都在我的州看到它。 搶救生命第一線的衛生保健工作者;在能源,農業或國防領域工作的基本工人;數以千計的志願者參加了接觸者追蹤,為老年人和弱勢群體提供飯菜,並對青年人進行輔導和指導,以防止學習失誤。  當我們被建議與社會疏遠時,我們就無法使用促進人類互動癒合的傳統同情行為。 但是每個人都可以戴著口罩表達對他人的同情心。 那些戴著口罩保護美國安全與健康的人正在為自由事業服務。 通過共同採取行動,戴上口罩,我們可以更快地恢復正常生活。 我們通過關心國家的福祉來愛我們的國家。 我們戴著口罩向這一承諾表示敬意。

10.廣東:感染個案與香港流行毒株的病毒序列高度同源。

據新華社廣州8月22日電在廣東省人民政府新聞辦公室22日舉行的新聞發佈會上,廣東省衛生健康委員會主任段宇飛通報稱,廣東最近發現的病例,與香港流行毒株的病毒序列高度同源。

廣東汕尾陸豐市8月14日報告發生一起新冠肺炎疫情以來,截至22日8時,本次疫情共發現1例確診病例,汕尾市報告;8例無癥狀感染者,其中汕尾市報告5例,深圳市報告3例。 段宇飛表示,8例無癥狀感染者都是在已納入管理的密切接觸者等重點人群中發現的,均為患者的同事或親屬。

段宇飛說,疫情發生后,廣東迅速開展病毒溯源分析,第一時間開展病毒全基因組測序,採取5例感染個案的鼻咽拭子樣本,對其病毒進行全基因組三代序列測定和分析,發現這5例的病毒基因序列高度同源。 經與全球資料庫中的近8萬條新冠病毒全基因組序列進行比對分析,發現這5例同屬於香港本地流行的HK1分支,與香港流行毒株的病毒序列高度同源,同源百分率達99.99%。
