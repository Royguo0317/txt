家庭教會信徒因踐行信仰被逮捕

**未註冊的新教教會生存越來越困難，因為許多會場被關閉，以此迫使信徒加入官方的三自教會。**

7月28日，四川省地級市南充市南部縣公安局派出30多名警員前往索拉誠意教堂。他們逮捕了會場的牧師和大約50名成員。

[!\[\](https://spark.adobe.com/page/Lm1wGn4r9n0dZ/images/63a8d7d0-b3f3-4c2c-8dbb-961f5ecb276d.jpg?asset_id=34694224-6522-4782-aef2-ee945f5b23a7&amp;img_etag=%221ef0fc557a3c0c0050ad99cd677139e6%22&amp;size=1024)](https://spark.adobe.com/page/Lm1wGn4r9n0dZ/images/63a8d7d0-b3f3-4c2c-8dbb-961f5ecb276d.jpg?asset_id=34694224-6522-4782-aef2-ee945f5b23a7&amp;img_etag=%221ef0fc557a3c0c0050ad99cd677139e6%22&amp;size=1024)  警察把守在會場外

一名會眾告訴《寒冬》說，當天早上7點左右警察趕到時，有70多名信徒正在參加一次禮拜。在一名守衛在會場外的信徒的警示下，20名教會成員設法逃脫了。當被戴上手銬的抗議牧師告訴警官他沒有違反任何法律、人們的宗教自由應該得到尊重時，他們狠狠地踢了他幾次。

[!\[\](https://spark.adobe.com/page/Lm1wGn4r9n0dZ/images/a5112147-6457-4044-b392-f9eafebe5ec3.jpg?asset_id=6a0a464f-22d4-48d1-b572-ffa89ba56f63&amp;img_etag=%2286414f23b007f7341ef1d17800ad45e4%22&amp;size=4195)](https://spark.adobe.com/page/Lm1wGn4r9n0dZ/images/a5112147-6457-4044-b392-f9eafebe5ec3.jpg?asset_id=6a0a464f-22d4-48d1-b572-ffa89ba56f63&amp;img_etag=%2286414f23b007f7341ef1d17800ad45e4%22&amp;size=1024) 警察帶走50多名會眾

警官們沒收了所有的聖經和讚美詩，扯下了十字架和耶穌的畫像。他們還拿走了會場的空調、音響系統和音樂設備，並關閉了會場。

[!\[\](https://spark.adobe.com/page/Lm1wGn4r9n0dZ/images/8aef13a5-113b-44df-a291-ce5b58dabd5b.jpg?asset_id=00661bf1-020a-43ef-9ba9-5b9fcd04a0b1&amp;img_etag=%2201fd8f9392508cff44fe86a108cca6bf%22&amp;size=1024)](https://spark.adobe.com/page/Lm1wGn4r9n0dZ/images/8aef13a5-113b-44df-a291-ce5b58dabd5b.jpg?asset_id=00661bf1-020a-43ef-9ba9-5b9fcd04a0b1&amp;img_etag=%2201fd8f9392508cff44fe86a108cca6bf%22&amp;size=1024) 便衣警察正在將音響系統和其它貴重物品從會場移走

附近的居民回憶說，在突襲前兩天，四名便衣警察在會場外被看到。據該地區的一名年輕人說，在突襲當天凌晨4點警察出現在現場。

名被釋放的會眾透露，所有被捕的信徒都被帶到警察局登記他們的身份證信息、拍照和取指紋。他們已經被警察列入黑名單，可能長期被監視或騷擾。一些成員當天晚些時候獲釋，而牧師和幾名信徒被拘留4到15天。


警察帶走50多名教會成員
6月21日，十幾名四川省遂寧市大英縣民族宗教事務局工作人員和警察突襲了當地一家教堂，五名成員在那裡舉行祈禱會。他們被指“非法集會”，警察記下他們的身份證信息和電話號碼，並給他們照相。在關閉會場之前，警官們將十字架從一面牆上移走，沒收了21本宗教書籍，包括聖經和讚美詩。教會負責人被拘留了三個小時。

![](https://spark.adobe.com/page/Lm1wGn4r9n0dZ/images/e96bd6fb-10e1-4738-8ffc-0be668d1e1e9.jpg?asset_id=cf1b8471-2af3-439d-abbf-13417800963a&amp;img_etag=%22d35fd0131fc78cb12d5f5d738e746203%22&amp;size=1024) 警察移除了會場的十字架

5月19日，警察突襲了東部省份安徽省廬江縣的一間家庭教堂。大多數會眾設法逃脫，五人被抓且被帶到警察局接受審訊直到深夜。

第二天，警察關閉會場後，會眾在附近的樹林裡集會，但被追踪到。警官們向所有年齡的人噴灑胡椒噴霧，試圖驅散他們。一名教會成員回憶說：“噴霧非常辣，刺痛了我們的眼睛；甚至一個4歲的孩子也被噴。“

[!\[\](https://spark.adobe.com/page/Lm1wGn4r9n0dZ/images/503ec9a8-2cfe-44c6-9de1-8aa4c98afeb2.jpg?asset_id=483a044a-50cb-4e46-a0ad-2478c7aadccb&amp;img_etag=%22808857904929691ce603c7dc3843f578%22&amp;size=1024)](https://spark.adobe.com/page/Lm1wGn4r9n0dZ/images/503ec9a8-2cfe-44c6-9de1-8aa4c98afeb2.jpg?asset_id=483a044a-50cb-4e46-a0ad-2478c7aadccb&amp;img_etag=%22808857904929691ce603c7dc3843f578%22&amp;size=1024) 警察關閉了會場，驅散了會眾

會眾在附近找到了另一個臨時聚會場所。然而，警方在6月17日再次突襲了他們，迫使信徒搬到50公里外的另一個地點。

東南省份江西省撫州市東鄉區一座古老的地方教堂因拒絕參加三自愛國運動而不斷受到迫害。不僅會場被清空並關閉，而且會場的成員也不斷受到監視。

7月22日，一名教會成員因在當地火葬場分享福音而被逮捕和拘留8天，她因拒絕加入三自教會而被指派看管她的一名村官向警方舉報。在被拘留期間，她再次受到成為官方教會成員的壓力，但她再次拒絕。 “我不怕被逮捕，”信徒說， “我將繼續分享福音。”

另一名教會成員也在同一天被捕，並被拘留了8天。警方禁止她獲釋後分享福音。

東南部省份福建省省會福州的一名家庭教會信徒因在地鐵站散發福音傳單而被捕。 “他們不允許我們分享福音。這是對我們信仰的迫害，”教堂禮拜者說，“我們將繼續分享福音，而不是退縮。”
