#### 第六章 高考-2

虹看我吃对虾吃得香，就问我：「你想不想喝点酒？」「天啊？喝酒！那可不行！我爸知道会打死我的！」「就喝一点儿，你爸不会知道的，他又看不到。」「我爸能看到的！他说他办公室裡有一个大屏幕，可以看到我的一举一动！」「哈哈哈哈！」听我这麽一说，虹笑得前仰后合。「你爸骗你的，傻瓜！他啥也看不到！」「那⋯ 喝什麽酒？」「茅台，白酒，你敢不敢喝？」其实我喝过白酒，小时候过年时，姥爷曾用筷子沾过一点儿白酒给我喝，可是现在早忘了什麽味道了，就记得特别的辣。我犹豫了一下，下意识地抬头看虹，她鼓励地衝我点点头。「好吧，就一点点儿。」「好勒，你等我一下！」

虹说著就转身出了厨房，不一会儿，她拿著一瓶茅台和两个小白酒盅回来。她熟练地倒了两小杯白酒，一杯递给我，另一杯自己拿著。「来，我们乾杯！」「啊！这麽多，喝了会醉的！我不行！」「没事！不会醉，醉了也没事，我爸妈今天都要值班，家裡就我们俩。」我怔怔地看著虹，我好像不认识她了。她看出我的心思，说：「怎麽？不认识我了？」「你，你跟在学校不一样了！你平时那麽文静，不爱说话的，可是现在⋯」「哈哈哈，就喜欢你这样傻傻的样子，你真是个土包子！来，乾杯！」「那，那我只能喝一点儿！」「行，一点点也行！」我嗫嚅地用嘴唇稍稍抿了一口，辛辣无比，她却豪爽地一饮而尽，我吃惊地看著她。「别担心，就一小杯！我没事！」

一杯白酒下肚，虹的白脸蛋泛起了潮红，她看起来越发的迷人了。「好热啊！」她随手把马尾辫解开，让她的长髮随意地披散下来，我盯著她看，目不转睛。「喂，你发什麽呆呢？吃虾啊！」「我⋯我，你真好看！你长得像张瑜！」「我就是我，我才不要像张瑜呢！」「你怎麽跟在学校不一样了呢？」「在学校，不是真的我，现在才是！当然，只有你知道，你要替我保密！否则，我就不理你了！记住没？」「喔，我记住了，我保证！」

虹的直爽，不由让我想起了鑫，但是她又跟鑫不一样，她更有一种神秘的美。在她文静的样子下，偶尔迸发出潜在的激情，让人惊叹不已。虹拉著我的手，带我去她的房间。我们没有坐在书桌旁，而是直接坐在了她的床边，床上的被子没有叠好，散乱地放著。她躺坐在床上，用肘撑著身体，斜睨著看我问：「我长得好看吗？」「好看！像张瑜！」我脱口而出。「我才不要像张瑜！」「你比张瑜好看！」「胡扯！我问你，你喜不喜欢我？」「喜欢！」「那你亲我一下！」「啊？现在？」「那你还要等到什麽时候？傻瓜！」

虹顺手拉了我一把，我身体失去平衡，一下子压在她的身上。她那天穿了一件白衬衫，也不知什麽时候，她衬衫的扣子解开了好几个，露出她雪白的胸部。因为喝了白酒，那白胸上有一片粉红，像夕阳的红霞。白衬衫裡面，是一件小内衣，从上口望进去，刚好看到她粉红的乳头，乳晕很大，乳房已经发育起来，傲娇地挺立著。这还是我第一次这麽近距离地看到女人的乳房，之前只是摸过，并没有仔细看过。「看什麽看？快亲我啊！傻瓜！」听到她叫，我才醒过神来，再转头望向她的脸。

此时此刻，虹的粉脸已然是全红了，她的嘴唇很薄，上面有细细的绒毛，可爱极了。我轻轻地把嘴凑上去，她搂著我的脖子，迎上来。两唇刚相接，虹就把她的舌头伸进我的嘴裡，碰到她的舌尖，我浑身激灵了一下，像触电一般，这是我人生第一次的法式舌吻。我们俩忘情地激吻，我的左手不自觉地伸进虹的衬衫，去摸她的乳房，刚揉搓了几下，她就死命地把我的手拽出来。「啊！不行！我受不了的！傻瓜！」她抓住我的手，但继续跟我接吻，快活地扭动著身体。过了好一会儿，我们才精疲力尽地分开。我们都瘫倒在床上，我问她：「你最近怎麽改骑自行车上学了，怎麽不坐吉普车了？」她把头靠在我的肩窝里，说：「最近街上太乱，到处是游行示威，我爸爸说骑车更安全些！」「那这样我可以骑车送你回家了！」「嗯，可以的。」

列位看官读到这儿，我猜想你们也知道那是什麽时候了吧。没错，这时候正是1989年的春夏之交。那个热闹非常的夏天，注定给我们每个人留下终生难忘的记忆。

读高中后，我都是骑单车上下学，要跨越三个城区。后来放学要先送虹回家，又增加了我每天的骑行距离。路虽慢长，可一路上并不枯燥乏味，因为每天街上都有游行示威，好不热闹。我们那时候都要自己带饭盒上学，早上把饭菜装在铝的饭盒里，中午在学校的锅炉房热一下，晚上放学车把上挂著空饭盒，叮叮噹噹地一路迴响。

记得那时经常在电线杆上看到各种标语，其中的一个名单，让我记忆犹新。那是一份中共太子党的名单，前两名就是邓小平的两个儿子，邓朴方和邓质方，官倒赚钱，大发特发。后面的名字不记得了，但都是当时中共几大家族的孩子。当然，这些都是盗国贼的亲儿子，红二代，还不是私生子女。再后来，七哥爆料后，才知道中共盗国贼的建筑艺术项目，早在1989年就开始谋划实施了。
