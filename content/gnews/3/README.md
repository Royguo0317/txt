![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/09052518/1-75.jpg)
就在美國當地6月6日，郭文貴先生在GTV平臺進一步透露，在塞爾維亞踢球的郝海東之子在中共的威脅下，已經被所在球隊開除。這一消息在路德訪談中也得到證實。同時，郝葉夫婦的學校以及他們的家人也全部受到中共的威脅。郭文貴先生在為郝葉夫婦勇敢站出來支持爆料革命高興之餘，又為中共對一個孩子下黑手而氣憤和心疼。郭先生一直在努力的控制自己的情感，但在直播中還是一度短暫停滯，幾度拭淚。

從2017年開始爆料革命以來，郭先生自己經歷了無數的困難、痛苦和挫折，但很少見他在鏡頭前落淚。即使在自己的母親去世後，他也沒有在鏡頭前哭。至於給他的哥哥、嫂子、家人和同事帶來的種種壓力，他仍然能坦然處之，並且在視頻裡告訴他們“忘記以前的文貴吧，那個文貴已經死了！”。可這次戰友的孩子所遭遇的不幸卻戳到了他的內心深處，不能自抑，這種戰友情懷感人至深！郭先生評價郝葉夫婦是中國的民意、人心、良心和良知的代表。郝海東和葉釗穎夫婦其實本身在海外過著幸福平靜的生活，他們站出來支持爆料革命完全是出於正義和良知，沒有一分錢的利益。他們希望通過他們的行動能喚醒更多的人，推翻中共在中國大地的殘暴獨裁統治，讓中國百姓過上民主、自由，生命和財產得到保障的未來。相信這星星之火會成燎原之勢，後面將會有更多有影響力、有良知的人站出來支持爆料革命。
