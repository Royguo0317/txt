## 美國之音10/5/2020：美國務院譴責香港“十一”80多人遭任意逮捕
`2020-10-08 09:02 CHANGDAO` [轉載自GNews](https://gnews.org/zh-hant/410541/)

香草山綜合報導





中文稿：
 1.儘管香港警方以新冠肺炎防疫“限聚令”禁止“十一大遊行”，香港人仍以“遍地開花”的方式在“十一”這天組織遊行，要求釋放12名被大陸扣押的港人，重申反送中運動五大訴求。有民眾爬上獅子山頭，揮動“光復香港”旗幟，並在山頂點亮聚光燈，重現去年抗爭景象。
 2.警方部署6,000警力，甚至戒備到獅子山，當天拘捕了至少86人，包括4名區議員。
 3.美國國務院10月3日對此發表聲明，對香港警方的任意逮捕感到憤慨，凸顯了北京完全背離其維護“一國兩制”的承諾，強烈反對當局為了政治目的繼續使用警力，這種做法與基本人權背道而馳。
 4.此外，美國國務院於9月30日公佈了接納香港難民的特殊計劃，而香港政府強調，沒有所謂的受“迫害”的“難民”。
