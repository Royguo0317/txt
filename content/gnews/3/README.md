笔者观点

2020年6月份中印边境冲突发生后，印度军方立刻公布有二十名印度军人死亡，中共方面却迟迟没有公布伤亡人数，明显是因伤亡人数过高而担心舆论压力。中共一贯隐瞒事实、虚假宣传、以“政治正确”为由编造故事引导舆论、塑造自身“伟光正”形象。

但纸包不住火，事发后不久，印度《新德里新闻社》引述消息称中共军方死伤人数为43人，当时网络上也传出疑似中共解放军死亡人员的墓碑，但遭到中共官方否认。俄罗斯国家通讯社塔斯社2月10日发消息称中共军方有45人死于去年的中印边境冲突。

中共官媒统一口径，称当时中共军方为了表现谈判诚意，某边防部队团长祁发宝仅带了少数官兵前去谈判，不料遭到印度军方“有预谋的”暴力袭击。这种说法十分可笑，仿佛双方不是国家军队，而是两个帮派。一个边防部队的团长竟然就边境问题去和对方“谈判”，且看不清局势被对方暴打？最终前来解救的同伴还死了4人？这怎么看都不能算作“胜利”，甚至可以算是指挥不利，而在官媒的宣传中则成了“取得重大胜利”。

中共不公布伤亡人数时，《环球时报》称 “当时前方情况紧急，不对比双方伤亡情况更有利于稳定边境局势”。胡锡进发文称：“显示了中方不进一步刺激印度的善意。这是强者的姿态。”

中共公布4死1伤后，胡锡进称：“如果祖国需要，男人就该像他们这样去活，这样去死。”中共官媒在文章中竟不提“印度军队”，而是用“外军”代替，《环球时报》英文版称这表现出中共的克制。

替中共粉饰其虚假、懦弱、狼狈、外强中干，不顾廉耻不惧骂名，有盘必叼、有地必洗、有屎必擦，《环球屎报》“胡叼盘”的称呼可谓实至名归。

双方都起武力冲突了，竟然还照顾印度方面的感受？不好意思说自己赢了，这是什么脑残逻辑？不公布死伤者是对自己同胞的不尊重与心虚，不敢提“印度军队”是懦弱无耻、毫无国格的表现！至于边界冲突，只要管控言论，捏造故事哄骗国人取得“舆论上的胜利”即可。

不知这些公布和被隐瞒的年轻烈士们，在报名参军时为了走关系交了多少钱。他们当时肯定没想到会成为无能中共的炮灰，甚至死后还要被当作宣传素材继续洗脑百姓。他们都是老百姓家的顶梁柱，本可以有更好的未来、充实的人生，如今却永远长眠，留给家人无尽悲痛。

烈士们的死换来了什么？中印边境问题没有解决，烈士家庭状况也因他们的离去雪上加霜。看看几年前上访老兵的待遇，中共历来是“用时抬、用完踩”。即便中共兑现了宣传中的承诺，那也仅是解决了其家庭最基本的生存问题，远不能与一个鲜活生命的逝去相比。

中共邪恶、官僚的体制令烈士的死毫无意义。中共只是拿他们“卖惨”，用这些本可避免的、无谓的牺牲，给老百姓洗脑，宣扬所谓“伟大的爱国精神”、“牺牲精神”，用“毒鸡汤”和“精神鸦片”代替常识和逻辑，把人培养成没有思考能力、听其指挥、甘愿牺牲的工具，最终获利的是贪腐的中共盗国贼家族。
