据World Tribune12月20日报道，美国人现在才开始清醒地认识到他们的政治领导人被一个共产主义超级大国严重妥协的现实。大家看到，一个美国州长在视频中热切地向一个共产主义外国政权提供自己国家敏感的军事技术和自然资源。亚利桑那州共和党州长杜西（Doug Ducey）的言语令所有美国人感到震惊。

“我们刚才和中国商会开了一个很好的会议。非常令人兴奋，有很多机会，包括公私合作。我已经提到了半导体、电子、航空航天、我们的国防工业、采矿和矿石等。我们希望与中国和中国的商人做更多的特许经营业务到航空航天和国防业务的生意。”

这是杜西在2017年7月直接对中共国国营媒体机关《中国日报》说的。《中国日报》将这段惊人的采访片段发布到YouTube上，标题为 “亚利桑那州州长讨论中国投资”。采访是在罗德岛普罗维登斯举行的2017年全国州长协会会议期间举行的中国总商会欢迎午宴论坛上进行的。

杜西在11月30日认证了该州充满欺诈的选举结果时，引起了美国总统川普的愤怒。这位州长可能不是总统的朋友，但他一定会被中共视为朋友。在中共国智库对所有50位州长对中共国态度的排名中，杜西被标为 “友好”或者“拿下”。

美国国务卿迈克-蓬佩奥在2月8日的一次演讲中警告说，中共正在积极寻求培养美国州长和其他地方官员，为政权服务。他特别提到了全国州长协会的名字。由全国州长协会和一个叫中国人民对外友好协会的东西共同主办活动，其实是中共官方对外影响机构统战部的公开形象。

蓬佩奥12月9日在佐治亚理工学院的演讲中也警告说，中共国 “为了自己的目的，正在毒害我们高等教育机构”。杜西在这个赛场上也为红色中共铺开了欢迎地毯，他告诉《中国日报》：“我们州的公立大学里当然有成千上万的你们的年轻人，我们喜欢拥有他们，希望扩大贸易范围。”

亚利桑那州杜西州长，一个冰淇淋连锁店的前CEO， 正在寻求与中共国进行更多的贸易。他和他的商业伙伴在2007年出售公司时，该品牌已经从一家地方店发展到在全美50个州和包括中共国在内的10个国家拥有1400多家分店。

2016年4月，中共国驻美大使崔天凯访问亚利桑那州。中共国美国大使馆网站写道：“‘冰激凌人 ‘为了吸引中国人对其州的兴趣，正在努力给中国人甜头。州长杜西热烈欢迎崔大使访问亚利桑那州，并表示亚利桑那州始终积极参与推动和发展与中国的积极关系，将加大对科技创新的投入，为即将到来的中共国企业家创造一个有吸引力的、更好的商业环境。杜西州长还表示，希望双方能够加强在教育、旅游和水资源利用等领域的合作。”

杜西极其乐意把美国的宝贵资源卖给中共国这个饥渴的亚洲巨人。2015年，杜西预示着通用电气与中国航空工业集团公司（AVIC）的合资航空公司Aviage系统公司在亚利桑那州开设分公司。这将是Aviage位于美国境内的第一家分公司。

杜西表示：“我很高兴地欢迎AVIAGE SYSTEMS公司来到亚利桑那州，因为它在美国的首次扩张将是支持公司全球发展计划的关键。这是一次有意义的合作。亚利桑那州是美国航空和航天系统制造领域最大的就业地区之一，我们期待着继续发展我们经济的这一关键部门，并帮助中航工业系统公司与我们一起成长。”

中航工业由中共军方控制。据路透社11月报道，川普总统 “即将发布一份89家中国航空航天和其他公司的名单，这些公司将因其军事关系而无法获得美国的技术出口”。中航工业将在这名单中名列前茅。

彭博社报道称中航工业是一家国有企业集团，拥有100多家子公司，超过45万名员工。川普政府在6月将中航工业列入其所说的由中国人民解放军控制或拥有的公司名单。该公司还经营着一家制造客机和私人飞机的民用企业–有些飞机的零部件是由与美国公司合资制造的。

《星条旗》10月的一篇文章令人不寒而栗地详述了中航工业是如何打造中共国的战争机器的。

去年，中共国为庆祝共产党执政70周年举行了一场壮观的阅兵式，习近平和其他领导人在天安门广场的看台上观看时，战斗机、攻击直升机、运兵车和侦察机组成的中队在头顶轰鸣，这场表演是为了给中共国内的电视观众留下深刻印象，并警告国外潜在的侵略者。制造这些飞机的公司。中国航空工业集团公司(AVIC)，这是一家国有企业集团，拥有100多家子公司和45万多名员工，比波音公司和空中客车公司的总和还要多。

这使亚利桑那州处于帮助中共国在军事技术上挑战美国的边缘位置。据中共国国家媒体报道，正在进行的工作包括一架隐形战斗机和一架水上飞机，这将是世界上最大的两栖飞机。加州大学圣地亚哥分校全球政策与战略学院教授张太明对美国军报记者说：”中航工业与美国的差距还是有的，但中航工业已经能够慢慢缩小这个差距”。

现在美国人民知道为什么杜西在他的州认证了充满欺诈的投票结果。大家也明白他为什么要咬紧牙关，非法在中共的 “老朋友 “乔-拜登入主白宫的过程中扮演自己的角色。遗憾的是共和党同僚对他的行为不但不感到恐惧，反而还任命他为共和党州长协会的新主席。
