# **独家：比尔·盖茨为中共影响力小组“开展宣传运动”提供建议**

比尔·盖茨（Bill Gates）曾向“中国共产党统一战线”中的各个团体提供建议和发表讲话，美国政府确定该组织负责中共最顶层的宣传工作，试图影响西方“采取行动或选择支持北京偏爱政策的立场”。

据美中安全与经济审查委员会称，盖茨曾在中美交流基金会（CUSEF）指导委员会任职，该委员会由中共国统一战线内“最高级监督机构”的主席创建。

根据国会的政府报告，这项努力的目的是“选择并消除潜在的反对执政的中国共产党政策和权威的来源”，并影响西方精英“采取行动或选择支持北京偏爱政策的立场”。根据《外国代理人注册法》（FARA）向司法部提交的文件，实际上，该小组已开始“有效地向媒体、主要影响者和舆论领袖以及公众传播有关中国共产党的正面信息”。

盖茨曾在中美交流基金会指导委员会任职，在一份2013年题为《未来十年中美经济关系：走向更深入的合作与互惠互利》的报告中“提供了宝贵的意见和建议”。

[!\[\]()!\[\](https://spark.adobe.com/page/ugbuVZYQ3AN0q/images/59245fcc-64db-4c31-bb28-6bc9a2e000fd.png?asset_id=e321a8a2-c453-4be6-8f8b-43559a8f379a&amp;img_etag=%22f55bbe1942fc0c4170f639e50811eba2%22&amp;size=3293)](https://spark.adobe.com/page/ugbuVZYQ3AN0q/images/59245fcc-64db-4c31-bb28-6bc9a2e000fd.png?asset_id=e321a8a2-c453-4be6-8f8b-43559a8f379a&amp;img_etag=%22f55bbe1942fc0c4170f639e50811eba2%22&amp;size=1024)2013年报告

独立的中共国指导委员会由许多共产党官员组成，包括来自国有银行、大学和党的领导机构的高管。

这份长达400页的报告是由美国学术界、商界领袖以及与中国共产党有联系的同行们撰写，并倡导“在科学技术领域开展更多的企业对企业合作”、“放松美国对高科技产品的出口管制”以及鼓励美中紧密联系的更多政策。

另外，由中美交流基金会赞助的美国大学生代表团成员已与盖茨基金会驻华官员会面。

但是，盖茨与中国共产党统一战线的关系更加深厚——也更加黑暗。

盖茨与英国前首相托尼·布莱尔于2012年在中国国际友好关系协会（CAIFC）主办的论坛上发表讲话并“表达祝贺”：

为庆祝“北京慈善晚会”成立十周年，中国国际友好关系协会（CAIFC）连同《财经》杂志和“北京慈善晚会”委员会于11月30日在北京共同发起了“首届中国慈善论坛”。来自美国、欧洲、澳大利亚和中共国各个领域的300多位热心公益的名人参加了该论坛。论坛组织委员会主席及中国国际友好关系协会主席李肇星致开幕词。英国前首相布莱尔发表了主题演讲，而美国著名慈善家比尔·盖茨通过视频讲话对论坛表示祝贺。

但是，中国国际友好关系协会被美国政府描述为中国共产党“前总政治部的一个前线组织”，扮演着“情报收集和开展宣传以及感知管理运动的双重角色”。

“中国国际友好关系协会与国家安全部、民政部和外交部之间还有更多联系，而且它是部署秘密情报收集者的平台。中美安全与经济审查委员会的报告补充说，除了将情报收集者派往国外，中国国际友好关系协会还赞助外国军事和退伍军人团体、商人以及前政治人物的中国之行，这些人士的行程通常包括与事先选定的人民解放军人员接触。”
