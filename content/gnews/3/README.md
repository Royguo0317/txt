## 孟晚舟在美國大選臨近和政治壓力上升之際將重返法庭
`2020-09-26 16:29 加拿大喜庄园🇨🇦` [轉載自GNews](https://gnews.org/zh-hant/384850/)

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/26162015/mengwanzhou1.png)圖片來源： 網絡
隨著美國歷史性選舉的臨近，以及被關押在中共國的兩名加拿大人的命運懸而未決，孟晚舟下週將出現在卑詩省的法庭上，她的律師將發起最新努力，試圖結束針對這位華為高管的引渡程序。孟的辯護團隊將爭辯說，當美國於2018年12月在溫哥華尋求以欺詐和共謀指控逮捕孟晚舟時，美國向加拿大提供了對她不利的誤導性記錄。這一指控是這位華為首席財務官的律師計劃在未來幾個月發起的三條攻擊之一，目的是讓負責此案的不列顛哥倫比亞省最高法院的法官相信，孟的權利受到了侵犯。

法律觀察人士表示，鑑於加拿大引渡法的性質，此次聽證會不太可能會動搖法官的立場。但他們認為，加拿大司法部長將面臨越來越大的壓力，因為中共指控加拿大進行政治干預，而且中共正在拘留加拿大人，兩個邁克爾Michael Kovrig 和Michael Spavor。

2018年12月1日，孟晚舟在從香港飛往溫哥華前往拉丁美洲參加商務會議的途中被拘留。現年48歲的他被控在2013年8月就華為與一家被控違反美國對伊朗經濟制裁的子公司的關係向匯豐銀行(HSBC)一名高管撒謊。檢方稱，匯豐(HSBC)依據她的說辭繼續為華為融資，使該行面臨被起訴的風險，並因無意中違反了同一套制裁措施而遭受損失。孟晚舟指責美國總統川普在與中共的貿易戰中利用她作為討價還價的籌碼。她還聲稱，加拿大和美國當局對她進行了秘密刑事調查，分享了她電子設備上的信息，並在她正式被捕前對她進行了3個小時的訊問——這與發布引渡令的法官的指示相反。

在一封致加拿大總理特魯多的公開信中，多人表示他們寫這封信是出於加拿大人“對前外交官Kovrig和企業家Spavor的長期非法拘禁深表關切”。自2018年孟被逮捕後不久，他們就一直被中共拘留。今年夏天，中共正式指控這兩個邁克爾從事間諜活動。法律觀察人士認為，這兩人幾乎肯定會在庭審中被判有罪，因為中共法律體系的定罪率為99.9%。

特魯多明確表示，他認為拘留“兩個邁克爾”是武斷的，並公開表示，美國不應該與中共達成不涉及孟晚舟和被關押的加拿大人的貿易協議。

擔任加拿大全球政治和國際法研究主席的英屬哥倫比亞大學教授Michael Byers邁克爾-拜爾斯說，與孟晚舟案有利害關係的每一個人–政治家、律師和被告本人–都在關注即將到來的美國總統大選，看看選舉結果會對未來的道路產生怎樣的影響。拜爾斯說，他懷疑政府會在總統選舉前採取任何行動， 引渡的政治問題意味著之後一切皆有可能。同樣，無論法庭上的情況如何，來自中共的壓力、兩個邁克爾的情況，以及加拿大與美國的關係，都可能導致特魯多和司法部長David Lametti拉梅蒂的政治考量發生轉變。拜爾斯還說這是孟晚舟所採取的策略的一部分，她試圖把這個案子拖得越久越好，因為她知道事情發生了，她的算盤才有機會改變。
