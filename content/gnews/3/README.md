之前说到，虽然康震诗词水平不入流，但解读古诗词却信手拈来——俨如古人肚子里的蛔虫。到其解读完《石壕吏》，其学问简直登封造极。

为此，当代非著名写实派小说家毒夫最近创作了一篇微小说。

《北师吏》——毒夫

东方一文明古国，1949年红魔党篡权，国号墙国，立都北精。北精市有两大名胜，一为沙尘暴，二为中南坑。中南坑为红魔党总部，众老翁老妪坐镇，时有绅士名媛出入，怪异乃半夜之鬼哭狼嚎声。中南坑某同志外号擀面杖，日理万机，唯有好色之雅好，传闻受一全球爆料团体扒皮，压力山大。

北精市有北师乡瘟学村，村支书姓糠名疹，CCAV诗词大会评委，常口吐莲花，因此名声大震。2017年即庆丰六年某日，于CCAV自创《口占七绝》，翌日火遍墙国。擀面杖同志闻之大喜，“好诗，好湿”，急电北精市委，“糠疹有妻否，你懂的。”北师乡吏风驰电掣赶至糠疹住所，见其独自一人，大怒。糠疹一改意气风发之势，老泪纵横，诉苦道：“吾膝下三女，小女陪侍中南坑腹肌同志，杳无音讯；二女应组织要求接待非洲友人，被摧残住院；大女怀上上上上级书记的龙种，嗷嗷待产中。吾妻半老徐娘，风韵犹存，但目前她需照顾住院的女儿，她的事日后再说。我虽年老色衰，但口功了得，拥独门‘舔肛’之术，若不嫌弃，我定让首长体验不一样的感觉……”乡吏半信半疑之间，竟听从其意。糠疹立刻沐浴更衣，专车护送赶往中南坑。

后面的故事外人不知。据说，半年后糠疹升官了，被赐 “舔肛狗”名号，倍受上级器重。还据说，北精市委办事人员、北师乡吏也升了。

（该微小说纯属虚构，切勿对号入座）

这样下里巴人的作品，估计康震“大师”不屑一读——因为古诗词才是阳春白雪，才显出他的博导水准。那笔者按康震的逻辑，用之后的文章来细细解读下该作品。
