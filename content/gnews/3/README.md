## 亚利桑那州州长杜西签署法案，禁止私人资助选举管理机构

[!\[\]()!\[\](https://spark.adobe.com/page/5F30oJFfvOmqU/images/b15afcb9-0364-44dc-8e52-ea6bfd1c18c6.jpg?asset_id=fced3902-cb93-4be1-9a35-5b996d64aac5&amp;img_etag=%22bdf0b3694a79c3d5ac043ad6b96922c8%22&amp;size=1024)](https://spark.adobe.com/page/5F30oJFfvOmqU/images/b15afcb9-0364-44dc-8e52-ea6bfd1c18c6.jpg?asset_id=fced3902-cb93-4be1-9a35-5b996d64aac5&amp;img_etag=%22bdf0b3694a79c3d5ac043ad6b96922c8%22&amp;size=1024)马克•扎克伯格。奇普•索莫德维拉摄 /盖蒂图片社

**亚利桑那州州长道格•杜西（Doug Ducey）周五签署了一项法案，禁止私人资助该州选举管理机构。**

现在亚利桑那州是美国第一个特别禁止在2020年总统大选中引入美国政治进程的有争议的私人资助选举管理机构的州。大选时，脸书首席执行官兼创始人马克•扎克伯格（Mark Zuckerberg）向两个非营利组织捐款4.19亿美元，这两个非营利组织又向亚利桑那州、县和地方选举管理机构提供了资金。批评者说，这种资助不公平地使民主党受益。

美联社报道，杜西在一封信中写道：“在公众对我们充满危机的选举满怀信心的情况下，很明显，我们的选举必须是原始的、无可非议的，而且政府必须是唯一有执行权的。”

HB 2569号法案于3月以31比29在政党路线投票直接投票中获得了亚利桑那州众议院的通过。上周，它又以16对14在政党路线投票直接投票中获得了亚利桑那州参议院的通过。

遗产行动执行董事杰西卡•安德森（Jessica Anderson）在周五晚些时候发表的声明中称赞共和党州长杜西将法案签署为法律：

*州长道格•杜西（Doug Ducey）代表所有亚利桑那州人，采取了迅速行动，签署了这一历史性的选举诚信法。HB 2569法案是亚利桑那州立法者采取的一系列有力措施中的又一个强有力的行动，目的是保护选举的完整性并将私人资金的影响排除在亚利桑那州的选举制度之外。亚利桑那州的选举过程应不受外界的影响和党派资金的支持——通过禁止公司和私人为选举机构提供资金，该法律将保护其选举系统。*

*州众议员杰克•霍夫曼（Jake Hoffman）和亚利桑那州的基层活动人士为推进HB 2569和其他常识性法规而努力工作，以确保亚利桑那州的选举公正透明。在亚利桑那州的立法者继续努力为进一步的选举诚信立法时，他们应该知道自己是在得到了亚利桑那人民支持的情况下这样做的。*

美联社报道，亚利桑那州民主党人将扎克伯格在2020年向该州的选举管理机构提供私人资助的行为归咎于共和党（向该机构）提供的资金不足：

*“亿万富翁大佬很容易翻云覆雨。我也不喜欢他们。但这是我们自己造成这种情况的，”坦佩民主党参议员胡安•门德斯（Juan Mendez）在谈到立法机关的预算决定时说，“我们的选举资金不足，我们县需要钱进行选民宣传。”*

*亚利桑那州州务卿凯蒂•霍布斯（Katie Hobbs）使用了选举创新和研究中心（Center for Election Innovation and Research）的480万美元进行广告宣传，告知选民何时和如何投票，鼓励签署永久早期投票名单，招募民意调查人员以及打击选举前后的虚假信息。*

正如布赖特巴特新闻11月份所报道的那样，亚利桑那州州务卿霍布斯是接受了扎克伯格资助的选举创新与研究中心（CEIR）资金的23个州的州选举官员之一。这些州中的八个州的州务卿是共和党人，包括佐治亚州的州务卿布拉德•拉芬斯珀格（Brad Raffensperger）接受了CEIR的“大笔拨款”。

正如布赖特巴特新闻在12月报道的那样：

*托马斯•莫尔协会的阿米斯塔德项目在周三的新闻发布会上发布的一份报告称，脸书创始人马克•扎克伯格和他的妻子在2020年选举周期中为非营利组织捐款4.195亿美元——其中3.5亿美元捐款给“技术与公民生活中心”（CTCL）的“安全选举”项目，以及向“选举创新与研究中心”提供6950万美元——该捐款“因代表一个特定候选人和政党而不当地影响了2020年总统选举”。*

*阿米斯塔德项目总监菲尔•克莱恩在报告的执行摘要中写道：“ 2020年总统大选见证了前所未有的公私勾结的合作关系，因代表一个特定的候选人和政党而不适当地影响了2020年总统大选。活动组织由脸书创始人马克•扎克伯格和其他高科技利益集团提供数亿美元资金，建立了两级选举制度，根据选民生活在民主党还是共和党据点而对其进行了不同的对待。”*

根据政府问责基金会3月份发布的报告，CTCL于2020年在亚利桑那州的9个县花费了超过500万美元用于选举管理：

*尽管CTCL尚未透露有关2020年选举期间分配给每位受赠人金额的全部详细信息，但据可公开获得的数据显示，CTCL向亚利桑那州的9个司法管辖区提供了超过500万美元的资金。8）根据初步数字，这笔钱的很大一部分花在为拜登总统助选上。9）马里科帕县是美国人口第四大县，拥有亚利桑那州60%以上的选民，不出所料地该县收到了该州分配的所有扎克伯格资助金的一半以上…*

*尽管由于CTCL不愿就捐款分配和支出提供完整报告而使分析数据有限，但初步分析表明，扎克伯格对亚利桑那州的选举结果是有影响的。平均而言，与2016年大选相比，接受扎克伯格捐款的县可以看到民主党总统候选人的选票份额有增加。*

*例如，在马里科帕Maricopa县（该州唯一在2020年大选中翻转的县）中，川普总统选票的总数增加了超过24.8万票，但该县输给了拜登。2020年，拜登将希拉里•克林顿（Hillary Clinton）2016年在该县的选票总数提高了33.37万票。接近9万票的差异无法用注册增加的说法来解释。自2016年大选以来，马里科帕县的民主党选民登记人数增长超过共和党人登记人数，但净增加额不到5万票，登记的共和党人仍比登记的民主党人多10万选民。在2020年支持拜登的县中，扎克伯格似乎帮助提高了民主党的投票率。*

其他几个州的立法机构正在考虑通过立法来禁止私人资助选举管理机构。

肯普州长在3月份签署成为法律的佐治亚州《选举诚信法》中有一项条款，旨在禁止私人资助选举管理机构，但负责阿米斯塔德项目的菲尔•克莱恩认为该州此项法律可能存在漏洞。

克莱恩赞扬了亚利桑那州新法律中简单明了的语句——“在不受限于任何其他法规的情况下，该州以及进行或管理选举的市、镇、县、学区或其他公共机构不可以收取或花费私人捐款来准备、管理或进行选举，包括登记选民”——这是其他州试图禁止私人资助选举管理机构的黄金标准和模式。
