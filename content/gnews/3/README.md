官员们说，印度尼西亚有350多名医生和医务工作者尽管接种了科兴疫苗，但还是感染了COVID-19，有几十人被送进了医院，因为人们对一些疫苗对更具传染性的变体的效力越来越担心。

爪哇中部库都斯区卫生局局长巴代-伊斯莫约（Badai Ismoyo）说，大多数人都没有症状，在家里自我隔离，但有几十个人因发高烧和氧饱和度下降而住院治疗。

库都斯有大约5000名医护人员，正在与一场据信是由传播性更强的三角洲变种驱动的疫情作斗争，该疫情已使其病床占用率超过90%。

医护人员被指定为优先群体，在1月开始接种时，他们是第一批被接种的人。

印度尼西亚医学协会（IDI）说，几乎所有的人都接种了由中国生物制药公司Sinovac开发的COVID-19疫苗。

对于中共国公司的科兴疫苗对较新的病毒变体的功效，科兴公司的一位发言人没有立即发表评论。

世界卫生组织（WHO）本月批准紧急使用科兴疫苗，称结果显示它可以防止51%的接受者出现症状性疾病，并防止严重的COVID-19和住院。

独立的健康关联数据组LaporCOVID-19的Lenny Ekawati说，许多人现在正经历着大流行病的疲劳，在接种疫苗后对健康协议采取了不太警惕的态度。

根据LaporCOVID-19，在整个印度尼西亚，至少有五名医生和一名护士在接种疫苗后死于COVID-19，尽管其中一人只接种了第一针。
