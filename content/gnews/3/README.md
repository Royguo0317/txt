## 脸书说将于5月5日宣布川普总统的账号是否被永久中止
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1168819/)

整理撰稿：蓝精灵

脸书Facebook Inc.独立监督委员会周一在推特上宣布，它将在5月5日宣布是否维持美国前总统唐纳德·川普无限期中止社交媒体平台和Instagram的决定。

在1月6日美国国会大厦事件之后，由于担心进一步的暴力动荡，Facebook阻止了川普的Facebook和Instagram帐户。后来将案件移交给监督委员会。

由Facebook成立的委员会在4月中旬对处理有问题的内容提出批评的回应，该委员会表示已延长了对该案的公众意见征询期。

董事会当时表示，川普的这个案件收到了9,000多条评论，比其他任何案子都多。

编者简评：

前总统川普在社交媒体上非常活跃，而且拥有跟随者，自从这位前总统被迫离开推特和脸书等之后，新闻界，娱乐界一下子安静了下来，之前媒体几乎每天在报道川普说了什么，做了什么，去了哪里。据说川普准备了自己的媒体，但是目前还没有正式的消息。脸书和推特也经常无缘由终止用户的账户，但是又很多新的媒体出现，比方最新的Gnews， GTV，Parler，Gab等等。
