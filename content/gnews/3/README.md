中国航天发射器“ 长征五号火箭（*Long March 5B*）”应该在5月8日（星期六）晚上8:30至5月9日（星期日）上午8:30返回大气层。一些碎片可能会在返回过程中残存下来。

在地球轨道上漂移的中国火箭可能在5月9日星期日凌晨02:24之前或之后的六个小时内落到地球上：在这个时间范围内，有三条轨道可能涉及意大利，这些轨道总共影响到意大利中部和南部九个地区的一部分。

这是民防局的结论，该局还发布了一份警报说明，在意大利中部和南部的9个地区已经设置了兼容过境窗口，希望这些地区的公民在5月8日星期六20:30至5月9日星期日上午8:30期间留在室内。设置过境窗口的9个地区是；翁布里亚，拉齐奥，阿布鲁佐，莫利塞，坎帕尼亚，巴西利卡塔，普利亚大区，卡拉布里亚，西西里岛和撒丁岛。
