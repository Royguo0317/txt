## 中共利用联合国阻止巴以冲突的真相
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1248100/)

- 编译：【默凡】


POLITICO新闻网5月16日报道称：中共外交部长王毅呼吁联合国安理会尽早降低以色列和加沙哈马斯统治者之间的暴力升级。他还指责美国要为安理会至今的行动不力负责。

“遗憾的是，安理会迄今未能达成协议，美国站在国际正义的对立面，”新华社援引王岐山周六与巴基斯坦外长库雷希(Shah Mahmood Qureshi)的电话交谈时说。王毅表示，安理会应再次确认“两国方案”，并敦促巴勒斯坦和以色列尽快在此基础上恢复谈判。

**评论：**

**巴以冲突始于巴勒斯坦哈马斯政权主动向以色列城市与海上钻井平台发射两千多枚火箭弹，以色列铁穹火箭防御系统拦截了超过90%的火箭弹。随后，以色列对哈马斯组织展开定点清除行动，十多位哈马斯高层被以色列导弹定点清除。在这次冲突中，哈马斯组织的上千枚导弹对以色列造成的影响非常有限，哈马斯还损失多名得利干将，以色列还下决心要彻底铲除巴勒斯坦哈马斯组织，由于双方科技实力存在巨大的差距，哈马斯组织损失惨重。**

**哈马斯组织背后的支持者是伊朗，而中共又是伊朗政府的支持者，以色列政府表示要清除哈马斯组织后，伊朗和中共都非常紧张，中共试图通过联合国阻止冲突继续扩大，并要求联合国支持“两国方案”，阻止以色列消灭哈马斯组织。如果哈马斯组织被以色列彻底铲除，伊朗和中共在中东地区的战略威胁将大大减小，进一步降低这两国在中东地区制造混乱的可能性**。
