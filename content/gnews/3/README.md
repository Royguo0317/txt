1. 据《雅虎新闻》报道，自上议院选举以来已经过去了两年，在这次选举中，山本太郎推出了“令和新选组”赢得了两个席位。山本虽然未能成为参议员，但他的“废除消费税”政策得到了很多人的支持。2019年10月，山本和众议院议员、立宪民主党成员马渊澄夫成立了“减少消费税研究会”（以下简称“研究会”）。 研究会提出了一个包括需要“积极的公共财政”和“减少消费税”的建议，并在5月31日举行了一个研讨会。

2. 2021年5月5日，JR东日本公司表示，今年夏天将进行一项调查，询问企业是否赞成“浮动票价制”，该系统将根据一天中的时间和其他因素改变票价。如果导入该制度，一些公司可能不得不为其员工支付更多的交通费用，预计会出现反对的声音。这项调查包括三个经济组织的调查，以帮助其设计该制度。“浮动票价制”的导入，以缓解早高峰期间的拥挤状况。然而，对于无法引入非高峰期通勤的中小型公司来说，提高普通通勤卡的价格可能会增加员工的交通费用负担。另一方面，非高峰期通勤票价的大幅度地降低，将更有效地鼓励公司使用非高峰期通勤。

3. 据《产经新闻》6月5日报道，虽然疫苗接种工作正在取得进展，而且有迹象表明正在为日本的中共病毒爆发做准备，但食品和其他商品价格的上涨有可能阻碍消费者信心的恢复。从美国进口牛肉的采购价格已经跃升至去年秋天的两倍左右。在加工食品方面，J-Oil Mills公司将家庭用食用油的价格从8月份的交货价中每公斤提高了50多日元。除了来自中国的需求增加外，投机资金正流入大豆和油菜籽等原料的国际市场。在中共病毒的影响下，商品和服务的供应无法跟上经济生产活动复苏的需求。据大型航运公司投资的班轮集装箱运输公司Ocean Network Express的一名代表说，由于港口拥堵，进入美国西海岸的主要聚集地洛杉矶的长滩港需要等待一周。今年4月，上海和洛杉矶之间的主要国际航线的集装箱运费比去年同月高出约三倍，这些增长将最终转嫁给消费者。至于食品方面，北美的牛肉和猪肉主要通过海运出口，因此预计直接影响也很大。

4. 据《福岛民报》6月5日报道，被称为“木材冲击”的全球木材短缺正在向福岛县的住房建筑工地蔓延。由于进口木材的减少，福岛县生产的木材价格飙升，而木材的短缺已经推迟了施工期。这是由于中共病毒爆发增加了美国、中国和其他国家对独立屋的兴趣，导致对木材的需求增加。由于建筑商无法将飙升的材料成本转嫁给销售价格，所以他们担心利润的减少，该县居民也对价格上涨感到困惑。

5. 根据厚生劳动省4月4日发布的《2020年人口动态统计》，2020年的总生育率，即妇女一生中会有的孩子数量，为1.34，比前一年下降0.02个百分点，是连续第五年下降。该比率是2007年以来的最低值，并且由于中共病毒的影响，2021年可能会进一步下降。
