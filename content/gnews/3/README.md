## 欧盟决定联手打击中共国

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/04/30030023/1-69-2.png)照片来自法国月刊Le point

据彭博社最新消息，欧盟决定加强对中共国所构成的日益严重的经济威胁的抗衡，做出了具有针对外国国有公司的新政策。

彭博社获得的一份草案显示，欧盟的执行机构欧盟委员会提出了新的规则，以处以罚款和对大宗交易的监督。

虽然提案中没有特别提到中国，但此举是在基于欧洲企业抱怨无法再承担亚洲国家的企业所带来的压力。该举措最终确定之前需要欧盟政府的支持。中共国企业集团已经对这最新举措提出了抱怨。

该议案文档在下周提出之前可能仍会更改。这是欧盟抵制中共国的下一步行动，是成员国推动保护战略公司免受非欧洲国买家的收购的基础上制定的。

在近一个世纪以来最严重的衰退中，欧洲开始显示出保护主义加剧的迹象。大流行使欧洲地区的脆弱性受到破坏之后，欧盟政府一直在争论供应链的“遣返”。法国和德国则表示，该集团应允许建立足以与美国和中国竞争的“欧洲冠军”。

成员国对欧洲公司被信用额度无限制的公司收购或由于竞争对手有能力以低于成本的价格出售而被迫倒闭的前景日益感到震惊。

新规则将与对外国直接投资的监督同时进行。过去几年来，欧洲各国政府一直在加强对外国直接投资的监督，以赋予它们更多权力，或制止他们认为至关重要的行业或部门的交易。甚至对超过10％的少数股权也可以进行严格的审查。

德国否决了可能购买机床制造商Leifeld Metal Spinning AG的提议，这是自2018年首次阻止在中国的竞标。去年，总理安格拉·默克尔（Angela Merkel）的政府同意购买CureVac AG 23％的股份，当时，冠状病毒疫苗竞赛中的关键人物曾是美国买断炒作的对象。

除其他成员国采取类似行动外，德国内阁周二还批准了对外国投资规则的更多修改，以赋予政府增强审查可能影响国家安全的交易的权力。

新法规需要获得议会的批准，针对的是人工智能技术，自动驾驶技术和量子计算等高科技领域。

另外，法国最近以食品主权以及在大流行中确保供应链安全为由，停止了加拿大食品库库塔德公司（Alimentation Couche-Tard Inc.）对食品杂货连锁店家乐福（Carrefour SA）的收购。

该国还以战略利益为由，否决了Teledyne Technologies Inc.对Photonis的收购，Photonify一家为军队生产夜视仪的公司。

总理德拉吉（Mario Draghi）也同意阻止中国深圳Invenland Holdings Co.竞购小型半导体公司LPE SpA。

西班牙政府已暗示可能至少阻止两项交易，一项涉及公用事业，另一项涉及航空零部件制造商。如果欧盟发现一家公司不合法地从外国补贴中受益，那么欧盟也将寻求对其公司处以相当于其年收入10％的罚款，这些补贴包括无限量的国家担保或信贷额度，这些产品削弱了欧洲竞争对手的实力。

根据欧盟规则草案，在欧洲产生的至少5亿欧元（6亿美元）收入，并在过去三年中获得外国政府超过5,000万欧元支持的公司，将需要获得欧盟的批准。草案警告可能会取消授予那些从此类补贴中获得不公平利益的公司的政府合同。

根据这份草案，欧洲官员正在寻求在公司许可和外国知情的情况下检查在欧洲以外的公司办事处的权力。监管机构还提出了一些方法，使公司可以减轻对补贴的担忧，包括向竞争对手授予基础设施访问权，公平条件的许可或发表研究成果。

根据该文件，公司还可以减少产能或扩大市场占有率，剥离资产或不进行投资。中共国驻欧盟代表团未回应置评请求，因为欧盟委员会拒绝置评。尽管采取了强硬立场，欧盟仍继续积极与中国建立业务联系，包括一项投资协议。

欧盟促进了这项协议，该协议将于明年年初生效，以此作为与其第二大贸易伙伴之间重新平衡经济关系的一种方式。欧洲协议使从汽车到电信等行业的投资者能够进入中国市场。

它还试图解决被认为扭曲市场的潜在的中共国政策，例如工业补贴，国家对企业的控制和强制性技术转让。
