## 依法滅共的武器─外國代理人註冊法(FARA)
`2020-08-26 23:39 GM77` [轉載自GNews](https://gnews.org/zh-hant/319386/)

編撰: 臺灣荔枝、文白、文非、Lori文噠

8月17日在夏威夷聯邦地區法院備案,8月21日路德社重磅推出的司法部起訴書中,提到了以下事實:自2017年5月到2018年1月前後,在夏威夷轄區及其他地區,被告NICKIE MALI LUM DAVIS故意並主動幫助和協助Elliott Broidy作為外國領導人的代理人行動,**他們代表劉特佐和中共部長孫立軍以及中共政府**,且未依據法律規定向司法部長註冊,即:他們通過暗箱遊說來解決壹馬基金案(1MDB)調查以偏向劉特佐, 。 並**遊說現任政府及司法部遣返郭文貴先生回中國**。 美國司法部的起訴書中,使用了《外國代理人註冊法》(“FARA”或”《外國代理人登記法》”),本文,著重介紹這個依法滅共的武器。

**FARA****是什麼?**

FARA是1938年《外國代理人註冊法》(修訂版,《美國法典》第22卷第611條及其後各條)的首字母縮寫。 (” FARA”)。 FARA要求在美國境內直接或通過另一人的「外國委託人的代理人」進行註冊和披露。

**(1)****代表外國委託人從事”政治活動”**;

(2)擔任外國校長的公關顧問,宣傳代理,資訊服務人員或政治顧問;

(3)為外國委託人或為外國委託人的利益而徵求,收集,分配或分配捐款,貸款,金錢或其他有價物品;

(4)在美國政府的任何機構或官員面前代表外國委託人的利益。 此外,FARA要求代理商明顯標明在美國傳播的「外國人利益」或「外國人利益」的「資訊材料」。

**FARA****是確定外國在美國的影響力和應對國家安全威脅的重要工具。**

**FARA****的目的是什麼?**

**FARA****的主要目的是通過確保美國政府和公眾知道來自外國代理商的某些資訊源(旨在影響美國的輿論,政策和法律)來提高美國境內外國影響的透明度。**

FARA要求在美國境內從事外國特定活動的人向外國主要註冊人註冊並向司法部披露這些活動,從而提高了透明度。

**“****外國委託人的代理人”是什麼?**

“外國委託人的代理人”是指按照”外國委託人”的命令,要求,指示或控制,以代理人,代表,雇員或僕人的身份行事的任何人。 引入以下:

**1****、在美國從事政治活動,例如打算就美國的國內或外交政策或外國政府或外國政黨的政治或公共利益影響任何美國政府官員或公眾。**

2、在美國境內擔任公共關係顧問,宣傳代理,資訊服務員工或政治顧問。

3、在美國境內徵集,收集,支付或分配捐款,貸款,金錢或其他有價值的東西。

4、在美國政府內代表外國主體在美國政府官員或機構之前的利益。

**為什麼說FARA是依法滅共的武器?**

中共為了實現其獨裁暴政統治全球的野心,把一系列掠奪性且經常是不合法的”戰術”掌握得爐火純青例如:貨幣操縱、關稅、配額、中共主導的戰略投資和收購、盜竊與強制轉讓智慧財產權、補貼、傾銷、網路攻擊、間諜和暗箱操作的遊說。

隨著中共在全球範圍內逐漸失去信譽,越來越多的事實已經表明,中共的官員及其代理人與美國一些人或公司接觸,運用BGY方式誘使擁護中共的違法行為。

他們的目的也許各不相同,但是他們的思路卻驚人相似:商人和中共有利益勾兌,私下向美國公司或美國官員施壓、討好、許以利益,推動有利於中共的政策落地,對美國的司法體制構成了重大威脅。

外國代理人註冊法要求那些充當外國負責人”代理人”的人通過在司法部註冊,公開披露這個關係以及他們的政治或其他類似活動,提高透明度。

17號備案的司法部起訴書中的控罪:**協助和教唆罪以及未註冊的外國領導人的代理人。**這是有實錘證據的違法犯罪行為,在有陪審團的庭審下,越來越多的事實真相會展現出來,而中共幹得最多的就是見不得人的勾當,掩蓋真相、它們害怕曝光,害怕民眾知道真相。

中國人苦共久矣,渴望西方民主法治體系。 美國法律專業的慣例和實例是讓人民接觸[公平正義](https://zh.wikipedia.org/wiki/%E5%85%AC%E5%B9%B3%E6%AD%A3%E7%BE%A9)的重要部分,任何團體和個人只要觸犯法律都會受到應有的懲罰。 依法滅共就是要用法律的手段消滅中共,同時讓中國人更加深刻的學習理解法律的過程、價值和精神,以實現新中國真正的民主、法治和自由。
