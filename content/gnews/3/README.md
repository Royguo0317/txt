[《路德时评》2020年11月30日晚间节目](https://www.youtube.com/watch?v=FnOw6NDm_M4)播报主要内容如下：

**一、宾州共和党州议员要求撤回该州选举结果认证**

《大纪元》报导[1]：周一（11月30日），宾夕法尼亚州共和党州议员正式提出一项决议，对该州2020年大选的结果提出异议，并要求撤回该州对总统选举结果的认证。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/01035025/%E5%9B%BE%E7%89%87.png)图片地址：https://www.epochtimes.com/gb/20/11/30/n12586484.htm



路德社就该条新闻的评论如下：

1. 这是一项经宾州参议院和众议院共同通过的决议，并非议案；
2. 该项决议的诉求完全符合宪法：决议判定州长的认证已经违宪，因为他的诉求是确保本州所有公民都有投票权，也就是只要有一名公民的选举权被剥夺，立法机构就有权收回州长的认证，此前在听证会上已经有证人出席证实选举有舞弊行为，因此该决议完全有效；
3. 只有代表行政机构的州长与州务卿，和该州立法机构同时认证选举结果才最终生效，而美国总统选举结果最终是由美国选举委员会认证；
4. 如果双方都不认证结果，那将进入最高法，如果最高法在12月14日前未裁决，那本州20张选票将无效；
5. 现在认证的是选民投票结果，如果无效还有选举人投票，选举人是议会立法机构说得算，选举人是由联邦众议员和参议员共同选举，因此麦康奈尔将发挥重要的作用；
6. 川普律师团队现在的策略就是否掉拜登胜出的选民投票认证，然后进入选举人投票再一决胜负；
7. 美国有35个州的州务卿经选举产生，其他州的州务卿由州长任命，本州选举最终责任人是州务卿；
8. 州务卿认证本周选举结果后，立法机构可选择是否同意州务卿的认证；
9. 现在州立法机构基本否决了民主党控制郡的选举结果，这样做最终能削弱众议院民主党占多数的优势；
10. 虽然民主党的优势被削弱，但共和党议员也有可能出现反水的可能，因此麦康奈尔非常关键，共和党中的建制派很关键；
11. 如果选举人票的最终投票结果不满270票，那将进入第三个选举投票环节；
12. 第三个环节就是参议院选举副总统，众议院以一州一票的方式选举总统，此时，川普总统稳赢；


二、**亚利桑那州的州长认证结果有效吗？**

11月30日，亚利桑那州州长在认证截止日前签字，承认拜登赢得本州总统选举结果。

该认证结果有效吗？具体分析如下：

1. 州务卿最终负责选举结果认证，州长只有签字权利，否则将会因“不作为”被告；
2. 亚利桑那州的州长是民主党，她即使认证该州选举结果也不能最终算数；
3. 亚利桑那州立法机构已经发出呼吁要求扣留本州投拜登的选票，因为他们有足够证据证明该州投票无效；
4. 亚利桑那州接下来的诉讼流程和宾州基本相同；
5. 截至目前，乔治亚州、亚利桑那州、宾夕法尼亚州都在举行听证会，这三个州总共有47张选票，只要能成功地将拜登的总票数减去这47张，那总数就不到270张，拜登胜选就失败，川普总统的律师团队只要赢下一个诉讼，那他将赢得所有诉讼。


**三、乔治亚州宣布对第三方干扰选举调查**

FOX 11月30日报导[2]：*乔治亚州州长宣布将对外州人员前往本州进行投票的行为展开调查。*

1. 非乔治亚州居民在乔治亚州进行投票是重罪，由于有第三方干涉本州选举投票，该州选举结果暂时还未得到认证；
2. 乔治亚州州长和州务卿都是共和党人，该州选举舞弊调查将迅速展开；
3. 鲍威尔律师此前也在本州发起一场法律诉讼；
4. 未来本州选举结果交给立法机构进行认证，那川普总统将稳赢这16张选票；
