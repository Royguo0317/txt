来自Exbulletin新闻网的社评称，在南中国海霸权主义扩张收到了显著效果，中共政府在喜马拉雅山进行了南中国海蚕食扩张模式的复制，在有争议的边境地区开始积极建设新村庄以扩大或巩固其对印度、不丹和尼泊尔境内具有重要战略性地区的控制。

中共在荒凉、无人居住的边境地区大量积累新的军事设施，这些新设施包括电子战站、防空站和地下军火库。中共在喜马拉雅山新建的边境村庄，相当于其在南中国海人工建造的岛屿。《南华早报》援引中共政府最近的一份文件称，中共计划在有争议的喜马拉雅山地区修建624个边境村庄。以减少贫困的名义，中共驱逐西藏游牧民，强迫他们在偏远的高海拔地区上新建的人工边境村庄定居。中共还派汉族党员到这些村庄担任驻地监督员。

习近平在2017年提出中共边境村庄设计项目，并强调让西藏牧羊人迁徙到边境地区定居成为中共领土的守护者。自2019年以来一直稳步增长的西藏扶贫项目，一直把重点放在把穷人转移到邻国的土地上。

在国际法中，土地主张必须以持续和平地对有关领土行使主权为基础。迄今为止，中共对喜玛拉雅山的主张根植于一种有效的方式，即寻求将其对西藏的吞并扩大到邻国。通过修建新的边境村庄并将人口转移到那里，中共可以援引国际法来支持自己的主张。在国际法上，有效的土地控制是强大主张的必要条件。武装巡逻不能有效控制，但殖民地可以。

中共建造军事化村庄的狂热，引起了该地区对中共扩张战略的关注。中印边境因领土纷争发生冲突。中共政府为在印度边境建设新村庄辩护，称他从未承认印度对该地区的主权。在荒无人烟的边境地区挑起争端是中共土地所有权主张的第一步，然后中共会坚持控制，直到慢慢蚕食试图占领这片区域。

印度陆军参谋长纳拉瓦内(Manoj Naravane)最近声称，中共蚕食侵略策略不会奏效。然而，即使是像印度这样的军事大国，也很难找到有效的方法来对抗中共这种无赖的领土扩张。

中共海上扩张主义的策略复制到了喜马拉雅地区。国际仲裁法庭称，南海西沙岛的建设对珊瑚礁环境造成了严重破坏。同样，中共国在边境地区建设村庄和军事设施，有可能对生态脆弱的喜马拉雅山造成破坏，而喜马拉雅山是亚洲大河的源头。曾经纯净的洞朗高原上的环境已经被明显破坏。不丹声称对洞朗高原拥有主权，自2017年被中共占领以来，洞朗已经变成了一个高度军事化的地区。

中共霸权主义扩张，利用常规和非常规战术的混合推进与渐进的小范围领土蚕食，无需一枪一弹，试图重新规划地缘政治地图。各国需警醒应对中共的新侵略策略。
