## 【英媒观点】中共国制定新规则，全球汽车制造商本地存储汽车数据
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1279993/)

撰稿：喜妈
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/29013052/2021-05-29_Task5_Cover.jpg)（来自网络截图）
路透社报道，很多著名的汽车制造商诸如宝马，戴姆勒和福特都“已经在中共国建立了工厂”，并且“将其汽车产生的数据存储在本地”。这些与中共国有长期合作的汽车制造商“承受着如何处理来自汽车信息的压力”。这里的原因如同文章指出的，“汽车正在安装越来越多的传感器和摄像头，以协助驾驶员”。与此同时，帮助驾驶员的这些信息数据，也可以利于制造商使用汽车上“生成的数据来帮助开发新技术”。文章例举如汽车广泛使用的“自动驾驶系统”等。当然，当这些“信息可能发送到国外时”，就会“引起隐私和安全问题”。

文章以美国电动汽车制造商特斯拉为例，指出其“因在中共国存储和处理客户数据而受到中共国的公开审查”。路透社引用知情人士的消息，由于“车载摄像头的安全问题”，一些中共国政府办公室的工作人员被告知“不要将特斯拉汽车停放在政府大楼内”。而特斯拉则在上周表示，“已在中共国建立了一个站点，以存储其在该国销售的所有车辆产生的数据”。

路透社从其他汽车制造商渠道，得到反馈，他们“也做了同样的事情”。文章称，福特汽车公司表示，“去年上半年在中国建立了一个数据中心，并将所有车辆数据存储在本地”。宝马则表示，它在中共国存在一个为中共国本品牌的车辆所用的“本地数据中心”。戴姆勒公司表示，它在中共国有“专门的汽车后端系统中存储车辆数据”。文章指出通用汽车和丰田汽车公司“拒绝讨论他们如何在中国管理数据”。日产汽车和斯泰兰蒂斯则表示他们“将遵守中共国的规定”，报道称无法得到“更多细节”。

文章指出，大众汽车公司表示，“遵守数据保护规则对于成功进行数字化转型至关重要”，但“由于监管环境仍在快速发展中，现在就具体细节发表意见为时过早”。本田汽车和现代汽车“均未回应”路透社相关话题的置评请求。文章的结论就是，“没有一家汽车制造商表示他们是否会与海外办事处共享来自中共国的数据”。

文章指出，中共国于2017年推出了一项网络安全法，该法“要求所有公司在中共国陆上存储本地生成的关键数据”。当时，这个要求并未得到各个汽车制造商们的重视，此后当“智能”汽车成为关注核心，这个问题才“成为汽车制造商关注”。根据文章，中共国网络空间监管机构本月发布了一项规则草案，“规定汽车制造商必须征求客户的同意才能收集驾驶数据”；仍然处于公众讨论中的草案规则，“还将要求汽车制造商将数据发送到外国实体时，在本地存储数据并获得监管许可”。

文章援引行业内人士分析，各大汽车制造商对于在2017年网络安全法颁布之后，任何有关信息和数据安全的新提议“对任何汽车制造商来说都不应该感到惊讶”。中共国有中共国的一套规则，而欧盟所实行的《通用数据保护条例》从国际标准“对公司如何处理和存储数据制定了严格的规则”。文章指出“目前尚不清楚中共国的要求将对全球汽车制造商研究和开发新模型或技术的方式产生何种影响”。而文章的得到来自多家制造商的高管表示，当前公司“将来自中共国的数据共享给总部以开发模型的情况并不罕见”。行业人士称，“数据最终将成为公司的差异创造者”，因为所有的公司都将使用这些数据“构建产品和服务以帮助其与竞争对手区分开来的工具”。

综合以上的报道内容，笔者认为，中共国对汽车数据的重视，是颇为蹊跷的。众所周知，中共从来不是一个从人民利益出发，国家立场出发的政党和管理者。中共的一切行为出发都是盗国贼家族的利益，都是党派的利益和自己帮派集团的利益。这个对汽车数据的所谓保护和限制，笔者认为是大有隐情的。首先，它不断拿特斯拉说事，找茬，这里也是很大的阴谋。同时，各个汽车制造商自己的数据系统并没有和中共形成共享，这个说明两个问题。一个是中共想要通过汽车大数据的限制和监管得到自己的信息偷取。另外一个就是中共想要过滤和筛查这里面有无涉及它的安全和监控的遗漏。第三，中共国知道这些数据的重要性，希望通过参与数据空间，跟紧国际先进的智能汽车发展。

中共以为他们还可以继续在国际上和本国内玩弄这些冲着中共国大市场的大品牌大厂家们，可惜，它的动向已经引起了这些大公司大集团的警觉。也许暂时他们还是有很多时候不得不低头，可是他们更多地是会小心调整，能适应适应，不能适应就只能退出。特斯拉的例子就是一个很好的说明。如果他们还贪念中共国的市场，至少他们能做的，就是放慢发展的步伐。或者是，他们可以边发展边掂量，边观望边权衡。尤其可以预期的是，当世界知道了一个新中国联邦的存在，很多的事情，他们也会把商业的利益和未来规划联系起来共同考虑。我们可以想象，中共随着国际社会的认知和爆料革命的真相都逐渐展开，就越来越少会有人买它的帐；反过来，它能够去主张任何中国相关发展的事情，也更加欠缺信用和保障。任何游戏规则都是对双方共同起作用的，不能约束双方的规则，只能被对方抛弃。我们从本文中，看到的就是在未来的中共只会更加疯狂的各种作为下，它将成为孤家寡人，被本国人民和国际社会彻底抛弃。等待属于我们的光明吧。
