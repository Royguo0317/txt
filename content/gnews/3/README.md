赛斯心法不反对任何这些东西，但本末不能倒置。前不久前我看到中国的长寿专家叫林海峰的去世，终年51岁，是中国营养健康网董事长、阳光森林生物科技董事、中国保健科技学会医药保健研究会理事等等一堆头衔，中国长寿专家组营养专家。他在去年双十一去世，享年51岁！注重健康没有错，可是有时候，越注重健康及锻炼的人，有的人逼自己每天跑三公里，其实更应该反思你每天在逼自己运动的背后，潜意识你是在担心什么？是对生病的恐惧？有时候越注重健康的人每天过着所谓“最健康的生活”，吃全有机的，全素的，按点吃饭睡觉这些人为什么有时反而更容易生病？因为反向推论，他经常活在担心生病的恐惧当中，虽然他的作息很健康，饮食很健康，各个方面都很健康，可是因为他的出发点是恐惧生病，光是那个恐惧就足以压垮你的身体。

在赛斯哲学里面，我们追逐健康没有错，赛斯心法：1.我们天生相信我们是健康的；2.我们相信我们身体有伟大的自我疗愈力；3.我们相信在宇宙当中健康是身体天然的趋势。人体是全宇宙最高智慧的结晶，身体是有形的灵魂，灵魂是无形的身体，人的意识心要为身体健康负责。这些都是赛斯心灵的守则，大家要记住。身与心同时是精神与物质性的，两者相互交织作用。我们对健康一定要是正面的信念，因为太多的人是出于不信任我们自己的身体，是出于恐惧而生病。如果你是因为恐惧生病而追求健康，这个心态不对。

现在全世界都是因为不信任身体，恐惧生病而追求健康，看到赛斯心法的厉害了吗？光是恐惧生病这四个字，就足以压垮你的健康。你为什么那么恐惧生病？因为你不断的累积你的负面能量，你不断地对身体健康保持的是负面的信念与不信任的信念，你不断地对宇宙、对食物、对身体、对饮水、对空气，你的信念是负面的信念，各位明白吗？那些食物也许是没有那么正向，那些空气也许是糟糕的，可是你的信念才是最重要的，所以战友们有没有发现赛斯心法的不同了？

追求健康没有错，但最主要的出发点绝对不是建立在对健康的恐惧上面，所以我们一定要把自己的信念变成：我信仰健康，所以我追求健康；我相信健康，我没有在信仰疾病。我没有在预防疾病（预防的背后是相信我是会生病的，才会预防疾病哦），了解吗？所以学习赛斯的人建立起了与全世界的人类不一样的核心信念，是一种新的人类，像路德说的，爆料革命可能引发了另一场全世界意识大转变的革命。光是健康这点，赛斯哲学已经和大多数99.9%人不一样了。各位我这样说，你们有没有从内心认为这样才是对的？正面的健康信念就能解脱对健康的恐惧。
