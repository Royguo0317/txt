中共和其他獨裁政府一樣，渴望成為公民的“上帝”，為達目的，可謂無所不用其極。正如上圖顯示，中共制定了一系列所謂的“失信指標”，靠積分扣除的方式來進行信用審查。圖中一共展示了26條，這些條例對大陸人民的生活進行了嚴苛的限制，很多條例並無確定指標，這給執法人員的權力濫用提供了足夠的“空間”。

本文將對上述個別條例進行一一解讀：

3261：強拆、合村並鎮、土地調整的行動正在中國大陸轟轟烈烈進行中，經常有老百姓為了自己的權益，勇敢的對抗中共政府。這個條例顯示只要不配合中共的行動，便會扣除20信用積分，為日後更暴虐的行為提供了“保障”。

3262：亂扔垃圾成為失信參考，亂堆亂放也將納入進去，按照次數還是數量，沒有明確標準。

3267：亂貼或粉刷廣告進行宣傳將被扣10分，中共的治下到處都是其意識形態宣傳標語，此禁令是自己打臉自己。

3275：針對被“喝茶“的人，國內戰友因為投資G系列或者支持爆料革命被喝茶需要注意，這個條例標誌著中共對“喝茶”人的懲罰力度提高。

3276和3277：“重點管控”當事人一般多是各地的上方群眾，他們均是被中共迫害最慘的一批人，以前已經受到了重點照顧，目前更是將其信用扣除10分。

3283：出售封建迷信用品的。中共治下的人民不能有其他信仰，中國數千年的文化被摧毀殆盡，封建迷信的並無準確定義，一些正常的文化傳統活動容易被針對。尤其是南方的“拜老爺”活動。

3284：紅白喜事是老百姓的個人活動，中共蠻橫進行插手，制定自己的指標，通過村規民約的方式，全市建制村、社區對婚事新辦、喪事簡辦等事宜作出規定，特別是對酒席數量、就餐標準、隨禮金額、人員參加範圍、用車數量等作出上限標準。
