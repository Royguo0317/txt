## 【热点播报】印太首脑峰会结束并发表联合声明

作者：纽约香草山美食部 蓝天大海



![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-107.jpg)



3月12日周五，美国、澳大利亚、日本和印度举行了首次四国领导人线上峰会。会后，四国共同发表了联合声明，定义“四方精神”。

![美日印澳四国领导人资料图片](https://s.rfi.fr/media/display/cda97d66-83de-11eb-b69b-005056a98db9/w:980/p:16x9/Capture-406.webp)图源网络

声明中提及：“虽然我们有不同的观点，但我们承诺合作，争取建立一个自由、开放、包容、健康，且以民主价值观为基础，不受胁迫的机制，并以此应对印太及其他地区所面临的威胁。”

声明还提到：“我们支持法治、航行和飞越自由，支持用和平方式解决争端，维护领土完整，推进民主价值观并重申支持东盟的团结与合作。”

声明亦强调，“我们将共同应对新冠病毒对经济和健康造成的影响，将联手扩大疫苗生产，加快经济复苏；要共同应对气候变化，应对包括网路、技术、反恐、金融、救灾等方面的挑战。”

虽然声明中没有直接针对中国（中共）的字眼，但是字里行间仍是透露出对中国（中共）在亚太地区影响力的抗衡，特别是针对中共的疫苗外交和稀土封锁。

最近几年，中共国大幅增加军费开支，不断霸凌威胁周边国家，导致事发频繁，严重威胁到地区安全。今天，四国通过召开首脑峰会，寻求盟友合作，共同应对中共国的扩张，维护地区和平。这个框架将成为一个“印度太平洋政策的基础”。有了这个基础，会吸引更多的印太地区的国家加入，形成事实上的与中共国的脱勾，以此破解中共国“一带一路”的战略目标。不夸张的说，灭共势在必行，谁也阻挡不了。
