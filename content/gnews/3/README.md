巴布亞新幾內亞拒絕接種過中共國冠狀病毒疫苗的國企員工入境，並要求中共國大使館說明疫苗試驗情況，意味著巴方不相信中共所謂的疫苗，對中共持有戒心。中共的“大撒幣”換來的是別國的不信任，一貸一賂的戰略面臨破產。

中共有製造病毒的能力，但沒有研製疫苗的本事。所謂的國產冠狀病毒疫苗是假疫苗，甚至是毒疫苗，接種了以後檢測冠狀病毒呈陽性。

中共讓海外國企員工當小白鼠，接種疫苗，飛往全球各地輸出病毒。可憐這些被壓榨、洗腦的國企員工，相信中共的謊言，認為中共疫苗真能管用，最終可能給自己的家庭帶來滅頂之災。

##  **巴布亞新幾內亞禁止接種過中共國病毒疫苗的人入境** 

由於擔心疫苗試驗，中共國國有的瑞木鎳鈷公司的工人乘坐的航班被流行病管理員取消

[!\[\](https://spark.adobe.com/page/FdHbVdcgPOdiu/images/db6f6d3a-085a-45ef-ae27-f8c8f74fb27a.jpg?asset_id=f44f1ffa-c329-450d-8b7c-d4a8076dc7df&amp;img_etag=%223ce8b28996f607d43cf5628d7810500a%22&amp;size=1024)](https://spark.adobe.com/page/FdHbVdcgPOdiu/images/db6f6d3a-085a-45ef-ae27-f8c8f74fb27a.jpg?asset_id=f44f1ffa-c329-450d-8b7c-d4a8076dc7df&amp;img_etag=%223ce8b28996f607d43cf5628d7810500a%22&amp;size=1024)據報告，礦主拉姆·尼克（Ramu NiCo）說，其48名前往巴布亞新幾內亞的工作人員已於8月10日接種了Sars-Cov-2疫苗。攝影: Amanda Perobelli /路透社

一架滿載中共國礦工的飛機被禁止進入巴布亞新幾內亞，原因是擔心他們在離開前接受了未經批准的冠狀病毒疫苗接種試驗。

由於擔心疫苗接種試驗，一架從中共國起飛的航班被巴布亞新幾內亞警察局長、流行病控制專員大衛·曼寧（David Manning）取消，該航班載有位於馬當省的中共國國有的瑞木鎳鈷公司的工作人員。

[!\[\](https://spark.adobe.com/page/FdHbVdcgPOdiu/images/1a6d3cf2-01d5-4b6e-b2b5-60cbb478562b.jpg?asset_id=5a22dfb4-3c00-4aa4-b313-9d6ec1f32437&amp;img_etag=%22d38e34b75e1c695ca49de38ccc12900b%22&amp;size=2560)](https://spark.adobe.com/page/FdHbVdcgPOdiu/images/1a6d3cf2-01d5-4b6e-b2b5-60cbb478562b.jpg?asset_id=5a22dfb4-3c00-4aa4-b313-9d6ec1f32437&amp;img_etag=%22d38e34b75e1c695ca49de38ccc12900b%22&amp;size=1024)

據報導，由中共國國企中冶集團經營的礦主拉姆·尼克（Ramu NiCo）向巴布亞新幾內亞國家衛生部發表了一份官方聲明，稱其48名工作人員已於8月10日接種了Sars-Cov-2疫苗。

該公司說，由於疫苗產生的抗體，其員工抵達巴布亞新幾內亞後，對冠狀病毒的檢測可能呈陽性。

曼寧（Manning）說，巴方政府沒有得到任何疫苗接種試驗的信息，並禁止這架飛機進入該國。

“鑑於信息的缺乏，這些試驗是什麼，如果他們進入這個國家，可能會給我們的人民帶來什麼風險或威脅，我昨天取消了那次航班，以確保我們繼續按照我國人民和國家的最佳利益行事。”

曼寧（Manning）說：“直到中共國政府通過在莫爾茲比港的中共國大使館提供這些信息，我將聽從我們衛生和醫療專家最好的指導，在考慮接受過疫苗接種試驗的中共國公民的入境申請時，應該採取哪些適當措施。”

[!\[\](https://spark.adobe.com/page/FdHbVdcgPOdiu/images/d89a9416-3569-4131-aae8-3aee0eb78eb1.jpg?asset_id=a726788a-b536-44ea-aea6-3976b0412a37&amp;img_etag=%2276be259cc6f9b495e580ba366601c72c%22&amp;size=2560)](https://spark.adobe.com/page/FdHbVdcgPOdiu/images/d89a9416-3569-4131-aae8-3aee0eb78eb1.jpg?asset_id=a726788a-b536-44ea-aea6-3976b0412a37&amp;img_etag=%2276be259cc6f9b495e580ba366601c72c%22&amp;size=1024)曼寧

曼寧（Manning）已致信中共國駐莫爾茲比港大使和拉姆·尼克（Ramu NiCo），要求說明疫苗接種試驗的情況。

巴布亞新幾內亞還根據其《國家流行病法》實施了一項新措施，禁止在該國境內進行疫苗試驗，並禁止向巴布亞新幾內亞境內任何人提供未經批准的疫苗接種或藥物干預。

任何抵達巴布亞新幾內亞並聲稱在海外接種過疫苗的人都不能免於該國嚴格的隔離和檢測。

冠狀病毒目前還沒有得到認可的疫苗，但一些中共國公司正在參與全球開發疫苗的競賽。

在世衛組織確認的21種目前正在進行臨床試驗的疫苗中，有8種是由中共國企業開發的。

中共國已經批准對在海外的國企員工和軍事人員進行試點。

[!\[\](https://spark.adobe.com/page/FdHbVdcgPOdiu/images/790a962c-103c-4035-8f23-2d2567200acf.png?asset_id=84f07410-0955-47aa-8e06-5669676f6fe1&amp;img_etag=%22b14b781c96c625f4ca207dba5e319077%22&amp;size=2560)](https://spark.adobe.com/page/FdHbVdcgPOdiu/images/790a962c-103c-4035-8f23-2d2567200acf.png?asset_id=84f07410-0955-47aa-8e06-5669676f6fe1&amp;img_etag=%22b14b781c96c625f4ca207dba5e319077%22&amp;size=1024)

巴布亞新幾內亞是美拉尼西亞人口最多的國家，正處於一波潛在的毀滅性感染的臨界點。

雖然以全球標準衡量，關閉邊境和國內停工使官方感染人數非常低——只有361例（感染）和3例死亡——但人們擔心，該國脆弱的衛生系統可能很快被一場不受限制的（疫情）爆發所壓倒。

在首都莫爾斯比港，40%以上的人生活在非正式定居點，通常是家庭外的大群體，這讓（保持）社交距離不可能，而且無法獲得自來水。

也有人擔心，巴布亞新幾內亞冠狀病毒的實際傳播範圍遠比官方統計要大。大流行以來，整個國家只有14400人進行測試。

儘管最近陽性病例激增——從一個月前的不到10起——政府仍面臨著放鬆管制和應對該國嚴峻經濟挑戰的壓力。
