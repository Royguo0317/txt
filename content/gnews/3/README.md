**美国民众对****新冠****病毒的来源和危险认识不足**

福克斯电视台公布的一个民调显示，54%的民众认为新冠病毒来自实验室，17%的民众认为来自自然，29%的民众不确定来源。病毒实验室来源的民意已经起来，接着就要确认病毒来自谁的实验室，然后怎么从实验室里边出来的。我和邻居、朋友讨论病毒，他们最不能理解的是两点，一是中共怎么会把病毒投向自己的人民？二是中共把病毒投向全世界对自己有什么好处？他们受的教育是要合理质疑，相信常识，中共的这些行为直接跳过了人们的常识。我周边的美国人没法相信这是中共制造的生物武器，并且首先投向中国同袍。但是最近有些变化了，有邻居主动提到NIH的福奇资助了中国的实验室做危险的“功能增强”试验，而这些是不允许在美国做的，病毒可能来自危险的试验。

以前的一邻居碰到我说：“新冠疫情Almost over(几乎结束）”，我顿时语塞，真不知道该怎么搭话。我这周的周三去给车子做保养，在装饰精美的经销商的店里边，只有前台和两个员工戴着口罩。我周五去超市，因为是暑假，超市里人气很旺，但多数人都没有戴口罩，感觉生活好像回归正常。美国人对疫情是很乐观的，但数字却不是那么乐观，以佛罗里达州为例，2150万人口，上周日均新冠感染人数是1700人/日。对比人口相近的台湾（2350万), 过去的7天日均感染368人，媒体上宣传的是“台湾防疫出现破口，大爆发，城市自动封城”等。美国不是恢复正常了，而是麻木了。一方面人们盼望生活回归正常，另一方面经济不允许继续封锁限制。

**中共控制世界的野心和行为对美国和世界的安全造成危胁**

5月美国的CPI同比上涨5.3%，是2008年来的新高。我所在的美国南部城市，房子、汽油和食品过去一年普涨15%以上。以前的一位同事研究了美元和一揽子商品的价格后得出结论，美元在疫情发生后贬值了19.5%。中共的全球扩张是美国安全和经济的最大威胁，这在美国已经是共识。美国的议会和政府通过了一堆的法案，《新疆法案》、《香港法案》、总统行政令等，实际对中共造成了什么打击呢？美国政府只有文件，没有行动，中共在供应链中的地位更强了。当印度、越南和台湾都在病毒爆发中挣扎时，6月从中国南方港口到美国西岸的集装箱运价竟然到达了每个集装箱1万美金，这表明美国对中国的制造业不仅没有脱钩，反而是更加依赖了。

拜登总统已经开始了他的G7之旅，女王接见，谈笑风生。他去欧洲前派军机访台，给普京吃下定心丸，计划在瑞士见面拉拢俄罗斯对抗中共国。1972年尼克松访华，目的是拉拢中共国，对抗如日中天的苏联。如今历史是给美国开了一个玩笑，把拉拢和对抗的对象对调了位置。中共夸下了海口，如果美国军机着陆台湾，解放军三天之内收复台湾。在没有互联网的时代，国内的老百姓很难知道外面发生了什么事情。在信息时代，这种消息封不住，全中国人民都知道中共在下一盘大棋，是什么样的大棋呢？

让我们看看最近两年的中共的行为历史：江苏丹阳市长在一个小时前还在安抚学生，一个小时后就武警暴力镇压学生；台湾总统蔡英文说了几句硬话，不接受中共霸凌，台湾的疫情紧接着爆发；日本不接受中共威胁，决心让军队正常化，疫情也爆发；印度在边境上对中共强硬，然后疫情失控，出现人间惨象；香港的年轻人上街抗议回家后，中共陆续抓捕报复。

**中共以冠状病毒超现战偷袭美国，打击美国经济**

上一届美国政府对中共政府也是谈了又谈，在2020年1月15日签订了一份中共不可能履约的贸易协议。协议的墨迹都还没有干透，**中共制造的**新冠病毒在世界大爆发。美国对中共也不设防，中共病毒肆虐美国 。近半数美国人口已经接种了疫苗，如果疫情再爆发，能逃逸免疫系统的新变种病毒，再加上疫苗的ADE效应，后果将是灾难性的。

经济方面，美国的房地产泡沫已是历史最高峰，这是过度印钞引起的虚假繁荣。一旦经济进入下行通道，房价下跌，很多中产阶级会出现资不抵债的情况。美国的泡沫已经起来，既然是泡沫就有刺破的那一天。苍蝇不叮无缝的蛋，如果中共能投入重金突然打击美国的股市，美国的失业会陡增，然后恐慌性抛售房地产，那将是2008年金融危机的重演。
