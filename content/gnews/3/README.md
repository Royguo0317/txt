**西班牙2021年4月18日电/西喜社——**

中共官媒《人民日报》周四建议，欧洲应将中共国制造的冠状病毒疫苗视为其他疫苗的“值得信赖的替代品”，而几天前北京刚刚承认其国产冠状病毒疫苗“没有很高的保护率”。

《人民日报》4月15日写道，中共国疾病预防控制中心主任高福发誓要”解决目前疫苗保护率不高的问题”之后，仅仅5天，中共国疾病预防控制中心就提出了这个建议，“面对因接种缓慢而受到的批评，欧洲国家正在认真努力填补疫苗空白。在这个关键时刻，中共国的疫苗可以成为一个值得信赖的选择，因为它们不仅可以买到，而且其安全性也在几个欧盟成员国得到了认可。”

4月10日，在中国西南地区成都市的一次卫生会议上，高福在介绍国产冠状病毒疫苗和免疫策略时说：”现在正在考虑，在免疫过程中，我们是否应该使用不同技术路线的疫苗。”

4月11日，美联社报道说，当美联社(AP)追问他在会议上的言论时，高福说，他指的是 “世界上疫苗的有效性，而不是特别针对中国”。

“他没有进一步回答他指的是哪些疫苗”。

中共国卫生部迄今已批准5种国产冠状病毒候选疫苗在国内使用：其中3种疫苗是由中共国国营生物制药企业华卫和国药集团生产的；其另一种疫苗由中共国疫苗公司康希诺生产；再另一种疫苗由安徽智飞朗康生物制药公司与中共国科学院和中共国疾控中心高福领导的团队合作开发。

据美联社4月11日报道，”根据各公司的说法，疫苗的有效性从刚过50%到79%不等”。安徽智飞朗康候选疫苗——五种疫苗中最新获批的疫苗——的研发者尚未公布其安全性或疗效率的后期临床数据。

中共国卫生官员在3月初批准了安徽智飞朗康候选疫苗的紧急使用，尽管缺乏”同行评审的科学期刊上公开的有关临床试验数据显示疗效或安全性的信息”。

安徽智飞朗康的发言人在3月15日的一份声明中表示，该候选疫苗的疗效数据”目前还不能公开分享”，并补充说，该公司正在向相关卫生部门提供”可用信息”。

由于后勤问题和供应短缺，欧盟一直在努力向其成员国分发中共国冠状病毒疫苗。许多欧洲国家决定在3月15日停止使用阿斯利康公司的疫苗，因为在欧盟和英国卫生监管机构建立了该疫苗与罕见但致命的血栓之间的可能联系。该疫苗由英国-瑞典制药商阿斯利康与牛津大学合作开发。

据《纽约时报》4月14日报道，”根据YouGov上个月公布的一项调查，61%的法国人、55%的德国人和52%的西班牙人认为阿斯利康疫苗’不安全'”。

欧盟和英国监管机构在4月14日也暂停了美国研发的强生公司冠状病毒疫苗的分销，理由是担心其可能与罕见但致命的血栓有关。
