零对冲5月24日报道，福奇终于承认COVID-19可能来自武汉实验室，他表示“不相信”它的自然来源。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/24175635/34-20.jpg)

文章称，从COVID-19爆发开始，福奇就鹦鹉学舌，坚持认为该病毒不可能来自中（共）国的实验室。但早在2020年4月，川普总统就告诉世界，病毒很可能来自武汉病毒学研究所。然而，据2020年5月5日加拿大CBC News的报道，福奇当时再次否定武汉实验室是冠状病毒的源头。据CNN在同日的报道，福奇粉碎了川普关于冠状病毒起源的理论。

随后，据福克斯新闻社报道，在被兰德-保罗参议员拷问了数小时后，福奇在Poynter媒体研究所下午的“美国联合事实：事实核查节”活动中突然改变了自己的观点。当《政治事实》（Politifact）的凯蒂-桑德斯（Katie Sanders）问他是否仍然相信COVID-19是来源于自然时，福奇回答说：“实际上没有，我对此并不确信，我认为我们应该继续调查在中（共）国发生的事情，我们应继续尽最大努力，直到找出发生了什么。”他又紧张的补充到：“当然，调查人员说，这很可能是从一个动物库中出现的，然后感染了个人，但也可能是其他原因，我们需要找出这一点。所以，你知道，这就是我说我完全赞成任何调查病毒来源的原因”。

事实是，奥巴马政府在2014年暂停了对蝙蝠冠状病毒功能增强研究的联邦资助。在这一决定的四个月前，福奇的美国国立卫生研究院通过向彼得-达斯扎克领导的非营利组织生态健康联盟拨款，有效地将这项研究转移到了武汉病毒研究所，并随后资助了一些涉及武汉病毒实验室科学家的项目，当中包含武汉病毒实验室对蝙蝠冠状病毒的大部分工作。多年来，在石正丽（蝙蝠女）的领导下，武汉病毒实验室与美国大学和机构合作公开参与了功能增强研究。

美国国立卫生研究院在“了解蝙蝠冠状病毒出现的风险”项目下向生态健康联盟发放了370万美元的资助，于2014年6月发放第一笔款项666,442美元，类似的款项一直每年持续发放到2019年5月。
