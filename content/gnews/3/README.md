## 班農戰情室145: 解讀川普玫瑰園講話
`2020-04-29 02:18 GM09` [轉載自GNews](https://gnews.org/zh-hant/188208/)

作者：Kim

今日戰情室進壹步討論美國政府在如何制裁惡意制造並傳播病毒的中共。重磅信息來自昨日川普總統新聞發布會，亦稱玫瑰園講話。班農及其團隊進行逐字逐句解析。來訪嘉賓有資深政策顧問埃利斯(Curtis • Ellis)和當前危險委員會的聯合創始人和副主席加福尼(Frank • Gaffney).

- 班農評價川普總統玫瑰園講話為扔下壹枚炸彈，這是壹次對妳的生命至關重要的講話。班農從川普總統之前提出的“明知故犯”這個概念入手。總統表示現在各式各樣的調查都在同時展開，並且會追查到底，而且人們是知道中國的這些實驗室的，包括p4實驗室。來自布賴特巴特新聞網的馬太•博伊爾昨晚爆出壹則重量級新聞，他拿到美國國立衛生研究院和轉包商的壹封信，證實美國人民的稅款通過福西博士和國國立衛生研究院院長弗朗西•斯柯林斯以經費為由輸入中共研發病毒的實驗室之手。妳會發現這些科學家裏大多數在美國培訓過，而福西博士見過這些中國科學家並說他們都是透明度很高。班農形容福西博士和世界衛生組是壹路貨色。“這本可以很早被制止”，班農這樣解析：這不意味著這是來自壹個實驗室，這句話的意思是不管這個病毒是哪裏而來，在去年十二月份，中共已經知情。南安普頓大學稱如果去年十二月最後壹周提出這個病毒的危害，那麽全世界百分之九十五的死亡和傷害法律層面開始都可避免。班農還指出對付中共的信息戰和經濟戰美國如今逐步出擊，最客氣的是從法律層面出發，有眾議員湯姆•科頓和參議員馬爾科•魯比奧提出去除中共外交豁免權做代表。


- 麥克西提到在尼日利亞，有壹組律師代表國家經濟利益向中共提出賠償。要求中共賠償兩千億美元並列舉中共給尼日利亞國家和人民帶來的打擊，有生命賠償、經濟扼殺、社會導向缺失、精神折磨和尼日利亞人民日常生活被打碎等罪名。尼日利亞兩頭出擊，壹邊直問大使館責大使壹邊遊說尼日利亞政府要盡快立案偵查。


- 班農再次堅定定義中共全球瘋狂囤積個人防護用品為蓄意殺人行為，稱中共卑鄙無恥，是全世界最卑劣的黑幫，並撂話給中共讓他們放馬過來。 並指出中共散步吃蝙蝠視頻為障眼法，是為誣陷病毒源頭是海鮮市場做鋪墊，意在世界範圍內惡魔化無辜中國老百姓，挑撥兩者關系以及增加隔膜，進而稱自己所作所為是保護自己國民，抵擋外國邪惡勢力，這是赤裸裸的欺騙，赤裸裸的信息戰，用在自己國民身上甚是卑劣。


- 川普總統2016競選團隊高級政策顧問埃利斯發起活動，呼籲全部國民聯合抵制中國制造，並舉蘋果手機為例。埃利斯建議民眾在實體店和網店購買壹切商品時查產品制造地，如為中國，切記勿買。


- 麥克西在節目中諷刺了了中共外交部發言人耿爽對川普總統每日疫情更新做出的回復，稱耿爽不論對方是誰，只要是在任何方面對中共做法提出疑問都會得到相似的機械式回復。


- 石油和天然氣產業本應該是排名最靠前的產業，特別是在川普總統時期，而現在卻淪為無人問津的孤兒壹樣。在華爾街被鼓勵的是去投資中國公司，而班農指出中國毫無任何人權和勞工法則、有著盡數的苦役、人們被摘器官、有很多勞改營，完全是厚顏無恥在運作壹個國家。 班農繼而指出美國最強大的武器除了其領先的軍事力量，更重要的是經濟實力，其自由資本市場和為主要儲備貨幣的美元，切掉流向中共的美元才是制敵之法。班農再次提到美國精英問題，毫無遮掩直述如蓋茨、華爾街以及社團主義者都在為中共壹方打工（更多相關信息請看戰情室第142期）。


- 班農就美國人民逐漸發現他們的金錢被用來扶持中共和很多中國公司提出如何制止此類事件再次發生。當前危險委員會的聯合創始人和副主席加福尼舉例並做出詳細回答。首先三年前美國聯邦退休儲蓄投資委員會開始投資其資金到壹些中共體制下的公司裏， 其次美國證券交易委員會竟然允許這些信息不透明、擴散武器、踐踏人權和欺壓中國各少數民族及其他全部人民、制造武器殺害自己國民的公司合法上市，而這些公司在各個方面都不具備美國合法上市資格，但中共公司稱所有信息為國家機密。美國面臨這壹迫在眉睫的重大問題：百分之十壹的節儉儲蓄計劃，五百億美元是否會在接下來幾天流入這些中國在美的上市公司。加福尼呼籲川普總統能把握機會制止此災難發生。
