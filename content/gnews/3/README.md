## 中共國再次出現大規模傳染事件 隱瞞傳染 “不可防不可控難治療”
`2020-09-18 05:25 GM30` [轉載自GNews](https://gnews.org/zh-hant/365572/)

**作者：小妍說**

2020年的前9個多月中共國的百姓一直活在中共制造的病毒恐怖下，然而對于生活在蘭州，尤其是住在城關區鹽場堡一帶的人來說，他們還在同時遭受另一場更可怕的夢魇。這就是2019年底幾乎與中共病毒一起被披露的另一場傳染病，而後者則同樣是一起人爲的放毒事件。9月15日晚，中共國甘肅省蘭州市官方確認蘭州生物藥廠發生泄漏致居民感染布魯氏菌事故造成布菌抗體陽性感染者3245例。官方承諾督促藥廠開展補償賠償工作，補償賠償工作于10月份分批次開展。這是當地政府時隔八個多月後才再次就中共國上市公司中牧股份旗下的蘭州生物藥廠泄漏事故發聲。而現在在網絡上還依然可以找到2019年12月6日法制日報出面辟謠的微博正文，時隔9個多月當時辟謠的新聞卻已成爲事實，到底是誰在隱瞞真相，我們不言而喻。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/18051706/%E5%9B%BE%E7%89%871-18.png)

2019年12月底，甘肅官方通報稱：“2019年7月24日至8月20日，蘭州生物藥廠在獸用布菌疫苗生産過程中使用過期消毒劑，致使生産發酵罐廢氣排放滅菌不徹底，攜帶含菌發酵液的廢氣形成含菌氣溶膠，生産時段該區域主風向爲東南風，導致位于下風向的中共國農科院蘭州獸研所人體吸入或黏膜接觸産生抗體陽性，一共203抗體陽性人員及1名出現臨床症狀的抗體陽性人員。”聲明確認該病毒通過空氣傳播。八個月後，布病陽性確診者達到3000多人，是去年的十多倍。而受影響的人，遠不止3000多人。甘肅官方實際對兩萬多人進行檢測，初步陽性者有4646人。甘肅官方將其定性爲“一次意外的偶發事件”、“短時間內出現的一次暴露”。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/18051813/%E5%9B%BE%E7%89%872-66.jpg)

大多數人對布病可能並不熟悉，一個重要的原因在于，這種病往往在內蒙古、西北等牧區流行。布魯氏杆菌病簡稱布病，是一種人畜共患的慢性傳染性疾病，布病主要損害人、畜的生殖系統和關節。治療若不及時，轉爲慢性，患者會出現精神不振，全身無力，關節肌肉疼痛，也可能長期處于發燒狀態，不能再從事勞動工作，嚴重者會喪失勞動能力，甚至導致殘疾，俗稱”懶漢病“。布病的可怕之處不在于致死，而在于患者的治療經不起延誤。錯過了最佳治療窗口期意味著，他們的急性病症將轉爲慢性，形成隱形病竈，未來將反複發作，遷延數年甚至幾十年。而中共官方只是簡單的提到了賠償問題，而且是事件事發一年後才做出回應，之前還在辟謠隱瞞，真是悲哀，三千多人！這樣惡性的事件，必須有人承擔刑事責任。造成3000多人感染，後續感染會擴大到什麽程度，現在不確定。如此輕描淡寫的處理，不敢徹查是不是背後有什麽不可告人的利益瓜葛，畢竟這是一家中共國的上市公司，其背後如果沒有龐大的勢力，此事怎麽可能被壓制一年之久，可見中共國的老百姓的命在中共眼裏就是如同草芥，與利益相比根本不值一提。而且根據相關人員說：定性爲消毒劑過期引起的事件，是不能成立的。空氣往外排之前，應該是有化驗的，如果有毒株，不能直接排到空氣中，有毒源的空氣長時間的泄漏，不是偶發事件，而是嚴重的生産事故。過期消毒劑和排出的有毒空氣之間沒有必然聯系。這些帶毒株的氣體會儲存在氣罐，排出之前要經過檢測的。空氣顯示合格才能外排，所以這個定性是錯誤的。這就是中共哄騙百姓的一種手段，反正這些百姓什麽都不懂，中共怎麽說就怎麽相信。可憐的是染病的的蘭州人：雖然他們不會死，但很難好好的活了。外交部發言人汪文斌說：中國是最安全的國家，你們覺得呢？
