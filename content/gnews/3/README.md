由于中共病毒在欧洲国家继续扩大流行，有的国家还收紧了限制措施，社区当局警告说，情况仍然非常危险。

**在哪些地方加强检疫措施**

3月2日，欧洲疾病预防控制中心（ECDC）在一份定期报告中说，截至2月底，欧洲联盟和欧洲经济区31个国家中，有13个国家报告中共病毒COVID-19病例通报数量有所增加，4个国家报告老年群体的发病率有所增加，3个国家报告住院人数增加，3个国家报告死亡人数有所增加。虽然大多数国家的总体中共病毒流行情况正在改善，但上述指标的绝对值仍然很高，这表明传播仍然很普遍。专家们不排除未来几周，在报告病例数上升的国家，住院人数和死亡人数会随之增加的可能性。

在29个患病率高的欧洲国家（每10万人中至少有60例登记病例）中，奥地利、保加利亚、捷克共和国、芬兰和匈牙利等11个国家出现了增长。其余18个国家，包括比利时、克罗地亚、塞浦路斯、丹麦、法国、德国、希腊、爱尔兰、意大利等国，比率保持稳定或下降。

这种情况迫使芬兰政府于3月1日宣布全国进入紧急状态。在其他限制措施中，计划关闭餐馆，并正在讨论限制在全国范围内流动的可能性。他们将起草并由议会通过一项关于关闭饮食场所的法案。根据国家卫生和福利研究所（THL）的数据，芬兰的病例总数已经达到58 064例，而6.7%的人口已经接种了疫苗。

据捷克卫生部3月2日报道，本周捷克在整个疫情期间发现了数量最多的中共病毒重症患者。在前一天内，有12150名患者(目前全国有48924人患病)，其中有7717人被送住院，1568人使用重症监护室和呼吸设备。从3月1日起，当局已禁止公民离开其居住地区三周，例外情况则包括上下班、探亲、到政府机关和医生处就诊。幼儿园将停课，已开学的学校的学生将回到网上学习。

意大利政府还将非紧急旅行的禁令延长至3月27日。这一限制是在圣诞节前制定的，截止日期为2月25日。因此，每次最多只能有两个成年人去看望当地人。同时，全国整体发病率也在下降，从去年11月中旬的4万新发病例，下降到2月份的每天1.5万。

在德国，封锁时间也已延长至3月28日。默克尔领导的政府计划在3月3日讨论可能的放松措施。

所有欧盟国家都有不同程度的限制措施。

**疫苗接种方式正在发生变化**

从年初开始，欧盟领导层就计划开展大规模的疫苗接种活动，以逐步恢复正常状态。但是疫苗短缺——由于各种原因，制药公司无法提供所需的剂量，该计划已经停滞。布鲁塞尔的官员批评了制造商，国家当局开始寻找替代供应商。奥地利总理巴斯蒂安·库尔兹（Sebastian Kurz）周二表示，欧洲药品管理局(EMA)在批准疫苗方面行动不够迅速。在他看来，成员国不应该完全依赖欧盟的供给。库尔茨决定在俄罗斯寻找替代方案——在2月26日奥地利总理和普京的电话交谈中讨论了组织供应《人造卫星-V》疫苗的可能性。

在圣马力诺，疫苗接种比计划晚了两个月开始，而且不是用本来的疫苗，改用《人造卫星-V》。第一批疫苗7500剂，于2月26日运抵。根据批准的计划，从3月1日起，这个被意大利包围的国家将开始为医护人员和75岁以上的老人接种疫苗。在第二阶段，60岁以上的公民和在教育部门工作的人将接种疫苗。此后，其他人口(圣马力诺居民总数为3.5万人)将能够接种疫苗。这个小国的当局也准备为意大利人接种疫苗，但仅限于那些在圣马力诺长期工作的人。

在意大利本土，新总理马里奥·德拉吉（Mario Draghi）于3月2日解除了紧急事务专员多梅尼科·阿库里（Domenico Arcuri）的职务，并任命前北约驻科索沃部队指挥官、自2018年以来担任意大利军队后勤部队负责人的弗兰西斯科·保罗·菲洛洛（Francesco Paolo Filolo）将军担任这一职务。新专员的主要任务之一将是确保疫苗接种活动，这也出现了供应延迟的调整。路透社写道，疫苗接种从12月底开始，至今已注射430万支，当局共接种630万支。政府正在考虑改变其方式——不是为全面接种疫苗所需的第二针预留储备，而是讨论为每一个想要接种疫苗的人接种疫苗的方案，针对整个药量。英国采取的就是这种方法，英国的疫苗接种率已经达到30%左右。

**为什么要取消疫苗接种限制**

民众运动不仅受到第二次注射储备金的阻碍，而且还因为一些国家的卫生当局建议不要给年龄最大的群体注射药物。例如，法国曾禁止75岁以上的人接种疫苗。3月2日，据悉，他们将被允许接种疫苗。截至2月27日，法国有455万人至少接种了一次疫苗。卫生部长奥利维尔·维兰（Olivier Veran）告诉法国2号电视台，未来几周还有250万人将接种疫苗。

德国、意大利和其他一些国家也有类似的限制措施，但各地都在讨论脱离强硬路线的可能性。《La Stampa》报纸报道了关于将该方案扩大到意大利老年人的讨论。比利时也有年龄限制，准备重新修改。“我想重设限制，这个过程中的拖延是完全不能接受的。”卫生部长弗兰克·万德布楼肯（Frank Vandenbroucke）说。比利时和法国的权威机构都引用了苏格兰一项对540万人的研究，证明在老年群体中服用辉瑞生物技术公司和牛津-阿斯利康公司的疫苗有助于更容易恢复——接种疫苗的人住院风险降低了81%。
