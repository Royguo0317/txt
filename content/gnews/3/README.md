## 疫情危機凸顯中共制度的邪惡性,到了對其說“不”的時候！
`2020-06-13 11:08 GM30` [轉載自GNews](https://gnews.org/zh-hant/232929/)

**作者：然小哥**

疫情危機就像一面照妖鏡，在其反照下，淋漓盡致地凸顯了中共特色社會主義制度的邪惡性。不管敢不敢說，到了十四億人對其必須說“不”的時候。

“集中力量辦大事”，這是CCP一貫用來鼓吹所謂中共特色社會主義制度的優越性的宣傳口號。而所謂的大事是什麽？民主國家無疑是民生。而在中共國卻是龐大的軍事力量；靠撒幣維系的國際關系；看似雄偉輝煌的但又質量如糟粕的建築；看似四通八達的幹線交通，處處留有被遺忘的大渡橋般的索命交通網絡；無視諸如億萬楊改蘭的民不聊生，大肆發展登月下海技術。

這些所謂大氣磅礴的工程，都是無時不刻在奴役十四億老百姓的基礎上而建立的沙灘上的樓閣，以及對外充當胖子的大腹便便的肚皮而已。

回想貿易戰初期，主張“不怕打，奉陪到底”的中共國家副主席王岐山說：“中共國人民可以吃三年草。”這就是中共獨裁政府當政者的真實邪惡寫照。他們心裏根本沒有把人民當人，而是奴役的牛馬、壓榨的工具、擋箭的盾牌。

可悲的是，被奴役的人民，卻拿著“厲害了，我的國”這一看不見的大餅來充饑，甚至意淫。

但是，大難臨頭，卻露出了廬山真面目。

疫情擴散，對比西方國家首先考慮國民生活的作法，就可見孰優孰劣。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/13110210/image0-103.jpg)

美國最早就對成人每人支付1200美元，兒童500美元生活補貼。經濟對策總金額更是高達2萬億美元。

日本對每個國民包括非日本國籍的居住人口每人（不論成人兒童）支付約1000美元（10萬日元），除此之外，還拿出總金額1萬億美元用于經濟對策。

表：西方各國針對疫情經濟對策預算（截至2020年4月17日）

美國 2萬億美元
英國 0.45萬億美元
意大利 0.03萬億美元
德國 0.9萬億美元
日本 1萬億美元

而曆害國的所作所爲呢？

扣押，轉賣外國捐來的口罩，醫療物資，不發放各地的捐款和生活物資。卻對美國給老百姓發錢，放風嘲笑說“他們窮得沒有存款，不發錢就得等死。”

而據日前李克強總理的爆料，中共國有6億人月收入不到1000，甚至有信息表明有9億人月收入不到2000元。而他們在家隔離期間又活得下去嗎？

美國，日本等國家就是完美的嗎？當然也不是，也有大批不斷批評政府做得不到位的人，但這些呼聲可以直接反映到政府，政府也會及時糾錯。

拿日本來說，原本打算對收入減半的人發放30萬日元（約2億元）補貼。被認爲不公平，遭到國民輿論的反對，改爲現在的補貼制度。

社會制度的優越性不在于哪個領導英明，能否定于一尊，多麽偉光正。而是是否能反映民意，是否能隨時糾錯。重要的是國民不看公仆的臉色，而是公仆看國民的臉色。這樣的主權在民的制度，雖有局限性，但就是偉大的，就是優越的。而集中力量辦一小撮所謂領導集團自己的大事的制度就是落後的，邪惡的。

武漢政府好像也悔改了，邯鄲學步，對1300萬市民發了5億元商品券。折合每人38元，可以購買一斤豬肉。還得說“厲害了我的國，不愧是世界上最有錢的國家”。而武漢市民也只有“要飯不嫌馊”的份，有且只有“感謝偉光正的政府”。

而在隔離方式上，就更無法比擬。在中共國，我們看到了一個個觸目驚心的文革2.0的恐怖場面。而民主國家，在疫情蔓延期間，還是可以放風，還是可以按照自己的意願維持最低限度的具有人性的生活。

哪個邪惡，哪個優越？一目了然。

在中共病毒疫情下，纏在中共身上的最後一塊遮羞布被撕掉了。每個人都知道皇帝的新裝是裸體，這是一個“公有知識”，要把它變成大家都知道彼此知道這個事實的“公共知識”，完成質的轉變，需要有人喊出一嗓子。而6月4日郝海東先生的淩空一腳，把CCP的最後一塊遮羞布踢掉了，這已經成爲“公共知識”了。

十四億人，已經到了對這個邪惡CCP政權抱團說“不”的時候！
