3月23日，据BBC报道，三十年来，欧盟首次因人权问题制裁中共，中共不仅完全否认其新疆“种族灭绝”的罪恶行径，而且立即对欧盟官员实施反制裁。

在新疆，中共以”种族分裂及反恐”为借口，对维吾尔族人进行了残酷的迫害。种种恶行不断地被曝光出来，包括：建立集中营、全面监控、大规模抓捕、强奸、轮奸、性虐、酷刑、洗脑、强迫节育、强迫放弃信仰等。中共意图从肉体和精神上摧毁新疆人的信仰、尊严和意志。

对此，3月22日，欧盟、美国、英国和加拿大相继发表声明，宣布对中共国实施包括旅行禁令和资产冻结等措施在内的制裁，制裁对象是四名中共官员以及一家中共国实体。

不出所料，这一举动立即招致了中方的报复。在同一天，中共宣布制裁包括欧洲议会议员在内的10名个人和4个实体。并称，欧盟是“基于谎言和虚假信息”做出的制裁决定。

显然，为了掩盖真相，中共再次采取了“欺骗、报复、恐吓”等惯用伎俩，而结果却适得其反。被制裁的对中关系代表团团长包瑞翰（Reinhard Bütikofer）发推说：“中共领导人不让我访问中国大陆、香港或澳门，可是我还有台湾（可以去）。”

此外，中共在新疆的“种族灭绝”罪行，已引发了各国一系列反应： 美、英、日企将终止涉及维族被强迫劳动的业务； 多国领导人谴责中共在新疆的“种族灭绝”行径；多国官员呼吁抵制北京2022年冬奥会； 多国呼吁调查中共在新疆的“种族灭绝”行径等。
