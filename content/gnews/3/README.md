## 国内CCP病毒疫苗接种者出现“高烧，小红包，关节痛”
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/944563/)

作者：必有回响

最近微博有位女士爆出她先生于2021年2月17日注射CCP病毒疫苗，起初没有任何不适，但是到了23日她先生开始全身起小红包，24日开始发高烧，38.5度以上，小红包也不断增多，全身关节也痛。28日她先生去看病时也发现有个小伙也是接种CCP病毒疫苗后出现跟她先生一样的情况。直到3月1日她先生身上的小红包一点都没退。
![]()![](https://gnews.org/wp-content/uploads/2021/03/屏幕截图-2021-03-03-193521-2.png)（图片来自微博）
自从疫苗问世以来，副作用不断接种者出现各种不同的症状，也时不时传出疫苗致人死亡的消息，当然，由于全球主流媒体被中共蓝金黄所控制，真实由于疫苗造成的死亡人数被掩盖。

疫苗的造成的问题我们无法具体得知，但是根据郭先生爆料情报显示，疫苗也是“灭白计划“的一环，接种疫苗的行为可能是非常的危险的，因为郭先生多次透露情报说，疫苗本身可能就会是一种”毒药“，会让西方人付出代价！路德也在路德社节目中说疫苗可以被激活从而导致更严重的问题。

根据网上统计CCP病毒疫苗有数十种副作用之多，尤其是中共自产疫苗副作用更达到73种之多，被外界喻为最不安全的疫苗！下表是国外疫苗主要副作用统计。
![]()![](https://gnews.org/wp-content/uploads/2021/03/302-2.jpg)（网上媒体图片）
我们可以从表中可以看到，很多副作用感觉就跟CCP病毒的症状一样，但是目前好像无法检测到有感染迹象。疫苗副作用比较普遍，更严重的是致人死亡。早在1月18日BBC就报道：**“****美国辉瑞疫苗在挪威服用使用后，有****29****人死亡。挪威药品管理局表示，常见的不良反应可能导致虚弱的老年人病情加重****“，**致死案例时不时传出。

最近香港也报道:**” 香港上月26日展开中国「科兴疫苗」植入计画，一名63岁老翁于同日施打首剂，没想到27日突现呼吸急促症状，立即被送往伊利沙伯医院，经过急救仍不幸于28日凌晨离世，成为香港首例接种中国疫苗后死亡的案例。”**但是由于现在媒体严重掩盖真相，我们无法得知真实的数据。

关于疫苗问题老生常谈了，我们爆料革命郭先生，闫博士，路德社一直给世界发出警报，告诉大家CCP病毒疫苗不会起到作用，因为CCP病毒是中共人为制造蓄意投放，为的是摧毁西方文明世界千年来的秩序，甚至消灭白人，然后中共一家独大统治世界。所以为了达到此目的，中共在全世界不断投放变种病毒，致使疫苗不断失效。早在今年1月5日BBC就报道：**“在南非传播的一种新的冠状病毒变种，已经出现在包括英国内部的其他几个国家。”**

紧接着2月17日又网上又报出新闻**:“英国除先前发现的「B.1.1.7」变种病毒之外，英国爱丁堡大学再度发现新型变种病毒株「B.1.525」！而且此次的变种病毒与南非变种病毒相似，能够避免开中和抗体，，影响疫苗的有效性，最初在去年12月15日日出现在英国，如今又扩及11个国家，包含丹麦，奈及利亚等地。”**

不少专家认为这些变种的出现对于现有疫苗是极大的挑战，几乎可以肯定疫苗的防疫肯定会打折，不少专家认为CCP病毒需要像流感一样每年都注射不同种类疫苗才能保证免疫。但是笔者认为流感的后遗症严重性和传染力比CCP病毒要弱几十上百倍，CCP病毒给世界带来的经济损害那是超前的，如果一直靠通过疫苗来防疫，那人类将没有未来。

而根据郭先生爆料情报显示，中共一直持有CCP病毒的解药。笔者认为CCP掌握病毒疫苗的主动权，他们早就有针对他们自己设计变种的解药，从而以有效疫苗来控制世界的政治并从中赚取巨量美元，中共一手放毒，一手解毒，玩弄西方世界于鼓掌之间。中共所谓的“疫苗经济“要从世界弄走上万亿美元的利益，不但毒杀世界，而且要收割世界的钱，还让世界认为是中共救了世界，中共真是好算计，”强奸了你，还让你叫高潮，你还要付钱“！
