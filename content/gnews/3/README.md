## 蒙古族的遭遇讓我明白 義務教育的福利其實是個坑
`2020-09-08 08:22 WENZI` [轉載自GNews](https://gnews.org/zh-hant/341108/)

By 文紫
2020-09-08

義務教育是對學校和社會的強制性規定，意在促進兒童身心健康成長，保障兒童受教育的權利。這本應是令人欣慰的公益政令。但蒙古文化的危在旦夕和蒙古族人民的心路歷劫，讓我明白這是鋪墊已久的一步棋。

根據義務教育要求，若家長不送學生上學，家長要承擔責任；學校不接受適齡兒童、少年上學，學校要承擔責任；政府不提供相應的條件，也要受到法律的規範。如果是給予真知灼見，催人上進的教育，沒有人會反對。但如果學校提供的教育方式和內容不是百姓贊同的，欠妥的，不當的， 心存不善的，毀人不倦的，讓人望而生畏，心生厭倦的，還會有人義無反顧的把這當作福利嗎？但就是這不設前提的硬性規定，日復一日，年復一年的催生了無數的無奈和恐懼，剝奪了無數家長進退選擇的自由，毀掉了無數孩子本應快樂成長的一生。這1982年就已精心埋設的法律漏洞，不知已經消滅了多少民族的語言，摧毀了多少文化的繁榮與多樣性。若不是這一嚴苛的律法，今日的蒙古人民不至於被威逼利誘，以死抗爭。

如今，我們的權利被進一步削減。為孩子擇校本應該像選衣服、看家電一樣稀疏平常，本應建立在平等自願的原則。然而教育市場的壟斷讓我們的訴求變得那麼懦弱。目前社會普遍適用的學校分配制度是依照生源所在戶籍和房產的對應片區來就學。因此，這對於心存美好嚮往卻囊中羞澀的絕大多數老百姓來說，高昂的房價無疑為美好的願望畫上了終點。不管你喜不喜歡，願不願意 ，適不適合，都沒得選擇。不少新建校舍在開學前仍未完工，教室內空氣品質完全達不到環保標準和健康要求；再者就是學校周邊的環境規劃並不合理，不安全的交通環境，影響身心健康的商業環境都是掛在孩子家長心頭的一把刀。我就曾見到不少小學門口就是閃著霓虹燈的洗頭房，要麼就是流動人口聚集的不良治安場所。

現在的孩子有多可憐，一般人並沒有仔細想過。剛滿學齡的他們，就像雞鴨一樣被關進籠舍。教學環境怎麼樣，老師怎麼樣，孩子有受欺負嗎，都不知道。家長不允許陪讀、旁聽，只能旁敲側擊的打聽，即便對教學方法有不滿、有教學水準有質疑，怕孩子被老師穿小鞋的家長仍不得不睜一眼閉一眼，有苦往肚子裡咽。入學前，要冒著新冠的風險，排隊往身體裡注射存在安全隱患的疫苗。不打不行，否則不能入園入學。入學後，五六十歲的教師，知識結構和與對社會的理解早已與大眾脫節，自身深受中共思想的毒害而不知，為了教學目標和績效還要對孩子施加誘導和矇騙。更別提，CCP教材的歪曲和誤導，再加上孩子不能善惡分辨，家長早已望眼欲穿了。

正是因為義務教育的禍根，我們才失去了不上學的人身自由，我們才迫不得已的將生命浪費在強加給自己的學校，不得不忍受自以為是高傲的老師，禽獸般下流的老師，自以為教書育人卻謊話連篇的老師和真假各半的教材書籍。人格的扭曲，智商的拉低，身心的迫害，我們受夠了。滅掉中共，是維護子孫後代幸福的需要。
