**11.14.** **环球网****纳瓦罗拒绝回应****“****中国祝贺****”****，还给了一个惊人的说法**

尽管美国前副总统拜登宣布赢得了今年的美国大选，并得到了多国的祝贺，但美国现任总统特朗普的白宫贸易顾问皮特·纳瓦罗在当地时间周五接受媒体采访时却给出了一个让人震惊的说法。

他不仅宣称特朗普赢得了大选，更表示目前白宫的工作，是围绕着特朗普赢得此次美国大选而开展的。

纳瓦罗是在接受美国福克斯新闻网的采访时说出上述这番话的。

当时，福克斯的一名主播正在询问他关于特朗普当局新签署的一个禁止美国人投资与中国军方有关联的中国企业的行政令，以及特朗普当局打压TikTok的事情。期间，该主持人问纳瓦罗是否担心拜登在明年上台后会扭转这些政策。

对此，纳瓦罗回应说：“我们白宫这边是围绕着特朗普已经赢得连任的情况来推进工作的”。

他还迎合着特朗普的说辞，指控此次美国大选涉嫌欺诈，甚至宣称这次大选中发生的事情，从某种角度来说，是一场“完美的骗局”。

“从数据上来看，很明显特朗普在大选日当天赢得了选举，但在选举日之后，在这些关键州，他们恰好就拿到了足够超过特朗普的选票”，纳瓦罗说。

纳瓦罗的这番表态也引起了不少美国媒体和网民的关注与担忧。

美国CNN的一位白宫跑口记者就表示，不少经济学家都认为纳瓦罗是个疯子，而他此次接受福克斯新闻网采访时说出的内容则再次证明了这一点。最后，一名美国CBS新闻网的记者还透露，当她在周五白天询问纳瓦罗如何看待中国祝贺拜登一事时，纳瓦罗头也不回地就走掉了…。
