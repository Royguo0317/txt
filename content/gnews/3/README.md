据国家脉冲网近日报道，中共外宣网站CGTN.com在周一发布了一篇宣传文章，抨击前白宫首席战略家史蒂夫•班农（史蒂夫班农）。

在文章中写道，自称是伦敦经济学院南亚安全分析师的汉南•猴塞阴（Hannan Hussain）提出：“与班农的陈述相反，没有很多可改变的机会，因为华盛顿对疫情的处理，从压制信息到政治操作，导致美国政府信誉扫地。」

但伦敦政治经济学院在随后发表的声明中称：猴塞阴先生不是LSE南亚中心的「安全分析师」，也不在LSE担任任何其他职务。我们中心什至不存在「安全分析师」的职位。猴塞阴似乎过去曾在伦敦政治经济学院发表过两个博客文，但在大学中没有职位。

猴塞阴为中共大外宣做的采访说班农先生是仇外主义者，种族主义者等。它还指出，美国一些学者对“班农之流”所提出的赔偿论嗤之以鼻。猴的声音并非 案。几十年来，华盛顿的官僚，华尔街的大公司主义者和哈佛等机构大学的知识分子精英们都不断向中共磕头。可惜猴的出版商“外交家”杂志近日也打脸猴，将中央电视台（CCTV）列为「中共镇压武器库中的常规武器」 。

美国的初步调查表明，在得知可能的大流行之后，中共政府的情报隐瞒了六天，并允许武汉市举办群众宴会。中共直到1月20日，第七天，才警告中国人民。

由澳大利亚《每日电讯报》获得的“五眼”联盟（包括美国，加拿大，英国，澳大利亚和纽西兰）的档案发现，中共有意该报告特别突出了中共对疫情爆发的信息的压制，包含但不排除销毁海鲜市场，训诫吹哨人以及拒绝向国际社会提供病毒毒株。美国高级情报官员告诉福克斯新闻社，该档案的报道与美国对爆发源头的调查相吻合。

美国国务卿迈克•庞培奥（Mike Pompeo）表示：「看到美国人和其他国家加入我们，我们感到鼓舞，因为虽然我们知道这始于武汉，但我们还不知道从何开始。」。尽了最大的努力来吸引专家，但他们仍在继续进行掩饰和迷惑。」

4月30日，特朗普总统还证实，他亲眼目睹了武汉病毒研究所与该病毒的起源联系起来的情报。
