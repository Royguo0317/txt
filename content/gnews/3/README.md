中共国“共青团中央”公众号2月21日发表了名为《还好意思污蔑中国“种族灭绝”？！自己啥历史忘了吗？》的文章，妄图继续洗脑国内同胞。文中针对“种族灭绝”的事实进行诡辩，顾左右而言他，总是一副“你们也种族灭绝过，没有资格说我”的丑陋小人嘴脸。

![]()![](https://gnews.org/wp-content/uploads/2021/02/Jietu20210222-101123.jpg)图片来源：美国之音

其中提道：

“关于新疆的真实情况，中方包括新疆自治区方面已通过多种方式进行了详细介绍。但可惜的是，这些人显然不愿意听，听不进去。今天我想再强调四点：

“第一，所谓新疆‘种族灭绝’‘强迫劳动’等说法是彻头彻尾的世纪谎言，是加、美、澳等国个别政客、媒体、学者沆瀣一气、串连炮制的丑恶闹剧。他们这些人从未去过新疆，根本没有亲眼见过新疆的美，没有亲身感受到新疆各族人民群众的和谐幸福。过去40年，新疆维吾尔族人口增长了1倍以上，他们见过这样的‘种族灭绝’吗？”

过去40年才增长一倍，才增长一倍，才增长一倍，重要的事情说三遍。根据共党国自己的数据来看，40年前，1981年，乌鲁木齐已经有94万+的人口了，我们就算94万都是维族人，增长了一倍，也就算整数200万人，到2018年的统计数据表明乌鲁木齐有350万+的人。而根据第六次人口普查2010年的数据，维族只占12.46%的比例，我们用大基数的2018年去粗略地计算一下就能知道，乌鲁木齐只有区区43.61万维族群众。增长了一倍也就是40年前曾经只有不到21.805万人。也就是说40年前21万+的维族群众，在不受计划生育限制的情况下，还没有72万+汉族群众的增长规模大。我们再看被中共找写手用来辩驳的维基百科里的数据，这里提及在现代中共国大环境、医疗卫生物资丰富等一系列条件都优秀的今天，新疆出生率为10.69‰，全中国出生率为10.94‰，维族人口出生率为11.9‰，一个出生率相差不多，维族民众还高一些的情况下，是怎么做到让汉族民众72万+变成300万+，大约4~5倍的增长数据，而维族民众只有一倍的？！其他那些人去哪儿了？确实没见过如此惨绝人寰的种族灭绝。

文章中还提及：他们是不是认为新疆少数民族群众只能像他们自己国家的少数族裔那样，无奈地忍受贫困、失业和歧视？难道新疆少数民族同胞没有自主择业、平等就业，通过自己的劳动去创造更美好生活的权利吗？

退一万步说，中共统计数据还显示从2015年到2018年以维族为主的和田和喀什地区的出生率骤降60%以上，和平年代的人们去哪儿了？为什么骤降了？都去自主择业不生孩子了？还是又发生了例如曾经的伊塔边民外逃事件了？还是被抓进去再教育，“再就业技能培训”？什么技能培训可以绝育啊，摘棉花也没有这样的功效吧，这样更美好的生活怕是没几个人要去争取的吧。
