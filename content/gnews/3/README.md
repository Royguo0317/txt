## 駐港國安蠢蠢欲動 港人不懼砥礪前行
`2020-07-02 23:42 WENZI` [轉載自GNews](https://gnews.org/zh-hant/252475/)

**By 文荷**
**2020-07-03**

昨日，一部從始至終沒徵求過香港人意見的香港國安法在落地前一個小時露出了它的猙獰面容。這個法案比基本法23條更惡劣，在於人大直接繞過香港立法會強行決定條款內容，國安委可直接在香港設立辦案機構隨意執法，允許跨境追捕、跨國追捕任何對中共不滿、甚至一天也沒有去過其境內的的人，四大危害國安罪名的模糊化標準更是為因言獲罪、政治迫害留出無限空間，一旦被定罪刑期最高可達終身監禁。最可怕的是，這部惡法掌握在一個可以任意修改憲法、踐踏人權的暴政手裡。中共面露殺機，最險惡的篇章已經開啟，然而港人卻選擇砥礪前行。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/02233432/1-5.png)

**從現在 血的教訓不能繼續**

在很多國人眼裡，站著的香港人是傻得冒泡，爭取的民主自由，既不能換鈔票，也不能當飯吃，學會閉嘴，跪下，不就可以繼續衣食無憂的享受當下了嗎？可他們不知道，暴政從來和美麗的謊言相伴，對暴政的姑息和讓步，往往就是走向萬劫不復的開始。23年前，中共對世界承諾“一國兩制，港人治港五十年不變的那一刻，香港人信了，換來了一個繁榮不再自由盡失的香港。30年前，因六四屠殺而受到經濟制裁的中共政府向美國伸出橄欖枝，美國人信了，換來了一個被藍金黃滲透被超選限戰暗算得千瘡百孔的美國。70年來，中共對韭菜們說“打土豪分田地”、“讓人民當家作主”，中國人信了，如今地不是自己的，房子只有70年使用權，最後連嘴巴和腦子都不是自己的了，一切跟黨走，一切都是黨的。與中共交易如同與虎謀皮的例子實在太多，一部被篡改的中共黨史就是一部謊言盜國治國史。最可怕的不是被吃掉的結果，而是被吃掉後還被下一個獵物拿來做飯後的笑談，似乎危險永遠與自己無關。這才是悲劇一再上演的本質，也是香港人誓死都要捍衛底線的原因。

**站著才能生 跪著只有死**

中共國家宣傳機器馬力全開為這部被全世界唾棄的惡法粉飾太平，幾百個小粉紅在香港街頭揮舞著五星血旗狂歡慶祝。而勇敢的香港人，在明知可能會以身犯險甚至終身失去自由的情況下依然在走上街頭，向世界展示他們不屈的脊樑，“五大訴求,缺一不可”的呐喊在香港上空迴響，這一幕不知讓多少人淚目。此時此刻，在大陸那一頭，影響大半個中國的滔天洪水中老百姓孤立無援只能自救，三峽大壩等多地水壩無預警泄導致多少人失去生命家園被毀，北京疫情二次爆發也沒能阻止昌平小產權房被強拆的命運，在中國統治的世道下，哪怕天災人禍頻發，官員靜默，媒體靜默，小粉紅們依然覺得歲月靜好。這魔幻的場景讓我想起了電影《駭客帝國》中，Morpheus給了Neo兩粒藥片，紅色的將回答“什麼是The Matrix”的問題，而藍色的藥片僅僅是讓生活像以前一樣繼續。當Neo伸手去拿紅色的藥片時，Morpheus警告說：“記住，我要告訴你的只有一樣東西——真相。” 這也是目前國人和港人面臨的選擇，服下紅藥丸，就要直面殘酷的真相，就要清醒地痛。服下藍色藥丸，自然可以回歸過去的虛擬生活，享受無知的幸福。明知奮起抗爭，就必須接受“攬炒”的現實，為什麼香港人還要選擇吞下“紅色藥片”。因為他們明白自由和真相的價值，不是一點狗糧可以收買的。

**不滅中共 光復香港無從談起**

香港作為世界金融中心的地位，從來不是中共施捨的，而是百年的民主制度發展給香港帶來的自由、法治和安全所掙來的。這是大部分香港人脊樑硬不下跪的底氣所在，也是中共的黑暗，腐敗，集權所最痛恨的地方。一國兩制的失敗，不是溝通和退讓可以解決的，因為這兩種制度從一出生就是你死我活的對抗。中共煽動小粉紅罵香港人忘恩負義，到處宣傳香港的繁榮離不開大陸的支持，卻完全不提香港在回歸前早已是世界金融中心，是香港的自由港背書讓中國獲得世界的信任和源源不斷的外匯投資。為了把自己的意志更便於強加給香港人民，中共給因反送中運動的火焰上潑上“港獨”和“外部勢力干預”的髒水，卻對幾百萬港人合理的遊行訴求視而不見。中共的騙術太高明，自由強大如美國，也是在被一次次欺騙直到滿身傷痕後才醒悟過來。在美國國會審議香港制裁法案的聽證會上，當美國議員問來自香港的抗爭者：“你們有信心有辦法重建香港的繁榮嗎？”得到的回答是：“目前，共產黨統治的中國下，我們沒有辦法，中共是一切問題的根源。”沒錯，有中共在，香港就不再是香港，更談不上重建繁榮，一百個深圳也變不成香港。

讓六億人還在月均1000元的超低溫保線上掙扎的中共，無法給人民帶來廣泛的福祉，卻從不缺少口是心非的說客。“希望(國安法）發揮長遠穩定發展的正面作用”，當帶著全部家當逃離香港、置業英國的香港首富李嘉誠”又準備幫中共給中國和世界人民餵食“藍色藥片”時，我們看到了美國歷史少有的兩黨無一票反對通過了《香港問責法案》，看到了歐洲議會壓倒性票數要求歐盟起訴中共制裁香港政府，看到了被中共病毒肆虐得民生凋敝的世界也正在醒來。正如郭文貴先生在“新中國聯邦”成立大典上說的那樣，“今天，我不想談夢想，因為共產黨讓我們一直在做夢，我們不要再做夢了，我們要行動。”只有行動才能讓更多的人醒來，只有更多的人醒來才能碾碎中共的暴政，減少給世界帶來的苦難。好在那一天已不遠，屆時香港明珠一定會再放光芒，成為中國人心目中的聖城！
