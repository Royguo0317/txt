12月9日上午，美国国务卿蓬佩奥在乔治亚理工学院就“中共对美国国家安全和学术自由的挑战”发表演讲。

我们不能允许中共这个暴政政权来窃取我们的技术，来建立其军事实力，来给我们的人民洗脑，或者收买我们的机构来掩盖他们的恶行，我们不能让中共粉碎我们的学术自由，我们的学术自由让我们的国家获得庇护，让我们拥有就如我今天所在之处这般出色的机构，但是我们需要您的帮助，我们需要学生的帮助，我们需要教师的帮助，我们需要全美各地政府的帮助，我们需要受托人来监管他们的捐赠基金以及大学与中共和由中共支持的团体所达成的相关协议，我们需要相关管理人员关闭孔子学院，并对所谓的学生团体进行调查。这些团体在中共资金支持下到底在校园里做了些什么？ 政府会在此方面提供帮助，但我们也需要人民来协助我们的工作，我们需要研究人员对欺诈和盗窃保持警惕，我们的学术界要拒绝来自中共的金钱诱惑。

百年大计，教育为本。教育是一个国家的立国之本，如果教育也被控制，那么国民的人生观、价值观、世界观都将被改变，这个国家的意识形态也会随之改变，这个国家也将处于危险之中！

彭佩奥国务卿最后说：让我们行动起来，让我们高举自由之旗来捍卫我们的校园，因为自由是这些学校的立校之本，当我们直面中共——我们这个时代的核心威胁的时候，这也会帮助我们来捍卫我们的国家安全！
