中美两国外长将在今天（7日）出席联合国安理会会议。彭博社认为，双方将在会上就如何应对区域和全球危机发表不一样的看法。

彭博社报道，美国国务院高级官员披露，布林肯计划在会上发表讲话时，明确强调美国希望捍卫基于规则的国际体系，而美国认为这种体系正日益受到北京在南中国海、香港和新疆等地区所采取之行动的威胁。

而作为5月联合国安理会轮值主席的中方，其中国驻联合国大使张军早前在吹风会上就曾说：“应对好全球性挑战，唯一正确出路就是维护和践行多边主义。”

何为“多边主义”，即指追求共同目标的多个国家的联盟。其实仅凭这一点，中共与美国拜登政府的观念是相似的。拜登政府上台之后，放弃了川普政府时期的“美国优先”政策，转而在全世界寻求盟友和伙伴，与日本发表联合声明，力求发动拥有相同价值观的各国的力量，来完成阶段性的战略目标。而从拜登政府官员最近的讲话中可以看到，目前美国的聚焦中心，就在于来自中共发起的“中、俄、朝、伊”的威胁，其中，中共国是核心。

美国针对该四国联盟，措施也是十分清晰的。对俄罗斯，始终保持沟通渠道，6月份，拜登与普京总统可能还要会面。只要有沟通的管道，就存在变化的可能，况且俄罗斯只认利益不认朋友，有利可图，“双头鹰”的本质就会显露无疑。针对朝鲜，美国已与韩国建立了频繁的沟通机制。本月，韩国总统文在寅继2+2会议后，首次出访华盛顿，会面的核心内容应该与制衡朝鲜有着密切联系。最后针对伊朗，拜登政府已考虑重回伊朗核武器协定，进而可能松绑对伊朗的制裁。如果四国联盟中有三国的利益被考虑到并摆平，那么以美国为首的西方文明世界的唯一目标就是邪恶轴心国——中共。
