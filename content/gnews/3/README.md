加拿大安省政府2021年1月21日宣布，本周将在奥沙瓦（OSHAWA)和布兰普顿(BRAMPTON)开设中共病毒隔离中心（即隔离营），并正扩大多伦多已有的两个隔离营。除此之外，渥太华、皮尔区（Peel）和滑铁卢（Waterloo）均设有隔离营。

这些是安省“高优先级社区战略”的一部分，投资高达4,200万加元，据政府称该战略旨在“对高风险人群进行隔离”。未来几周内，安省将建1,525张床用于隔离，其中杜林区（Durham）125张，皮尔区840张，多伦多280张，约克区（York）280张。所有新的和扩建的隔离营预计将于2021年1月底投入使用。

迄今为止，已有160多名社区人员参与，还将雇用超过225名，以多种语言向社区成员提供针对性宣传。
