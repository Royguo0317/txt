## 【热点播报】中共试图收购乌克兰发动机公司被拦下

作者：纽约香草山健身部 童谣

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3.jpg)



中共国投资人收购乌克兰关键国防企业“西奇发动机”公司被阻拦下来后，这起事件继续发酵。

![]()![](https://gnews.org/wp-content/uploads/2021/03/Jietu20210317-112210.jpg)图片来源：VOA资料图

乌克兰国家安全和国防委员会3月11日宣布将把西奇公司收归国有。中共国随后在3月12日对此做出反应。中共国外交部发言人赵立坚说，中（共）国要求乌克兰依法维护中（共）国企业和投资人的合法权益，妥善处理相关问题。对于中共国方面的收购，乌克兰官员质疑中共国对乌克兰关键发动机企业投资的合法性。
![]()![](https://gnews.org/wp-content/uploads/2021/03/1018_.jpeg)
乌克兰是中共深度蓝金黄的地区之一，作为“东欧军事情报中心”的乌克兰为中共在“欧洲业务”的拓展起着至关重要的作用。乌克兰公司经常接受到中共的注资和收购并非什么新鲜的事情。之前习的白手套叶简明入股乌克兰能源公司与英德国家利用石油交易换取情报，这种模式直到被乌克兰官员通过拜登硬盘事件揭露出来。此次中共国欲收购被用在猎鹰战机上的西奇发动机公司便即刻被拦截下来，或能看到中共的蓝金黄影响力正在被各个国家内部的正义力量不断瓦解削弱。昔日勾结甚密的“朋友”甚至也不再对中共透露出任何妥协。只有几个乌克兰商人表示乌克兰外贸极度依赖中共投资，担忧乌克兰官方的举动将引来中共的撤资和报复。
