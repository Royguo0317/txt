【24日起强化来自英国的访日人员的隔离措施 除日本籍外暂缓入境】

翻译：待命（文晓）（京都富士会）

校正：待命（文晓）（京都富士会）

排版：待命（文晓）（京都富士会）

内阁官房长官加藤勝信在12月23日上午的记者会上宣布，鉴于英国新型冠状病毒变异、感染率迅速上升的现状，从12月24日起将加强对来自英国人员的隔离措施。

该措施规定，除日本籍外，将暂停来自英国的入境。 对居住在日本，因商务需要短期去英国后再返回日本的人士，不再给予“14天隔离”的免除，仍需要隔离14天。
