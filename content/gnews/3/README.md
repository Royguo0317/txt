## 【独立观点】美国将成立供应链贸易行动小组对抗不公平贸易行为

作者：纽约香草山健身部 Ivy001



![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/01042335/s2-7-1.jpeg)



6月8日（星期二），美国政府高阶官员告诉路透社，美国将成立新的行动小组对抗不公平贸易行为，以锁定中共国为目标。由美国贸易代表率领的供应链贸易行动小组，将找出造成美国供应链空洞化的具体违规行为，并透过贸易补救措施解决，当中包括针对中共国所制定的措施。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/10204329/Untitled-design-2021-06-10T184152.170.png)图片来源：Reuters
美国官员也表示，商务部正考虑是否要依据1962年贸易扩展法第232条款，来调查用于马达和其它工业应用的钕磁铁进口对国家安全的影响。美国的钕磁铁主要从中共国进口。

由于疫情爆发，与中共的竞争愈加激烈，美国政府终于正视由全球化生产带来的问题——美国国内产业链断链、供应链空心化。举个例子：全球抗生素90%由中共国和印度生产，看起来好像没有什么问题，中共国不出口，还有印度呢。但是，印度生产抗生素的原料都是从中共国进口的。这就意味着中共国从原材料一卡，世界就会有大麻烦了。

不仅仅是医疗和医药行业存在这样的断链问题，汽车行业、计算机行业、家电行业……都存在着供应链空心化的问题。而这些问题的根源，就是全球化生产。简单来说，就是中下游的产业链全都移往国外生产，主要的加工业区就在中共国。美国本土只生产中高端以上、高附加值的产品。这必然会导致美国本土的产业链断链，不能形成有效的国内循环。

美国政府现在开始应对这个问题，推动美国本土的产业链发展，修复本土产业链的强韧健康，为将来与中共国的经济脱钩做好准备。

美国政府对中共的态度越来越强势，这个贸易小组已经明确表示：目标锁定中共，美国准备在贸易上进一步反制中共了。这是继华尔街的中概股除牌之后，美国政府为实体经济脱钩做准备，与中共国全面脱钩又推进了一步。
