## 快評：印度戰機全天候巡航  遏制中巴聯手行動

新浪網9月27日轉載，據《印度防務新聞》網站報道，隨著拉達克地區局勢日益緊張，印度空軍表示已做好充分準備，可在中巴前沿進行晝夜間全天候巡航。

印度前線空軍基地距巴基斯坦約50km，戰略要地鬥拉特別奧里地距約80km，而在另一側印度在拉達克地區附近的空軍基地部署了大量軍機，包括蘇-30MKI、米格-29戰鬥機和C-130J、伊爾-76和安-32運輸機。隨著9月中旬5架「陣風」戰機列裝，和第二批的4架陣風即將交付，印軍的空戰能力進一步增強，面對巴基斯坦的F16和中共的2架殲-20，具備完全的制空能力。

為了遏制中巴聯手行動，印軍戰機晝夜不停的巡航，運輸機也載著部隊、口糧和彈藥不斷增援拉達克東部的鬥拉特別奧里地和其他戰略要地。同時為了應對中共的瘋狂基建，大修機場跑道，印空軍將引進國產新型智能反機場武器。

在印軍直接在拉達克地區全天候的戰機巡航之際，中共殲-20卻毫無動靜，而由於西部戰區參演高加索-2020，兵力減弱，中共前線的陸軍部隊行動部署也是雷聲大雨點小。
