## 【新闻连连看】不许走 留下来 打一针
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1287057/)

莫名其妙的中国最大的南方城市广州爆发了疫情，在一系列的排查，管控，封锁之后，现在出台了最新管控措施：离开广州的任何车辆需要持有72小时内的核酸检测证明，失效日另行通知。看上去好像也合理，只要去做个核酸检测，如果是阴性，可以行走自由。但是后来我们发现广州已经上了全国各地的“通缉”名单了，你到达别的地方，需要隔离14天。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/31200754/275.jpg)

很多人想，核酸检测阴性了， 为了能够出行方便，看看报纸，查查新闻，打疫苗是抵抗病毒唯一有效的办法了。政府铺天盖地的都是打疫苗的宣传。就在这个人心惶惶的时刻，好消息来啦：一辆装载着10万剂的一针疫苗抵达广州，还是高级的康希诺生物公司的， 是大名鼎鼎的陈薇将军制造出来的。她还是军事科学院军事研究院的工程研究所的团队首席。

这一套拳行云流水不着痕迹，广州群众这次肯定毫无疑问当了新疫苗的小白鼠。而且引发阿更多的疑问：之前的灭活疫苗是有问题吗？打了一针的现在停打了，接下来怎么办呢？从这个病毒，疫苗的事情，表面上看是一系列良好愿望的措施，但是我们还是不能失去焦点：病毒是从哪里来的，陈微将军为何第一时间去武汉病毒所处理，为何疫苗是他们军事研究院的，为何广州会突发疫情。

这一切事件联系起来，也许有些高人能够给出一个答案，欢迎在评论区给读者和作者解惑。
