- **[04.02 Breitbart： 亲历者：中共国将基督徒囚禁在地下“洗脑”酷刑中心](https://www.breitbart.com/asia/2021/04/02/witnesses-china-imprisoning-christians-underground-brainwashing-torture-centers/)**

在中国共产党政权下，只有由党而非梵蒂冈管理的中国天主教会和同样由共产党管理的 “新教 “替代组织爱国三自教会是合法的。许多中国基督徒拒绝这两种选择，因为党在其合法机构中不允许自由礼拜，并要求神职人员利用他们的平台来推动政权的议程。相反地，许多人选择所谓的 “家庭教会”–在私人住宅中进行非法敬拜，小团体聚集在一起学习圣经和祷告。北京官方估计中国有大约四千万基督徒。然而，算上 “家庭教会 “的基督徒，独立团体认为这个数字接近一亿人，比共产党的党员总数还要多。
在独裁者习近平的领导下，共产党多年来实施了一项被称为 “中国化 “的政策，即把所有宗教 “更中国化”。这已经转化为在中国西部为穆斯林建造了数百个集中营，在那里他们面临极端的灌输，以及大规模逮捕独立的基督徒、藏传佛教徒和法轮功学员。对拒绝共产党控制的基督教牧师的监禁已经被大肆宣传和广泛传播；RFA的报道揭示了似乎是对那些被党努力消失在其严酷的法律体系中的基督徒的另一种惩罚。
一位仅以化名李先生身份识别的证人告诉RFA，他最初是在2018年对其教堂教友的突袭中被捕的，并被保释。检察官显然没有什么针对他的证据，所以他不是落入中国警方的控制之下，而是落入统战部的控制之下，统战部是政权的一个部门，负责培养和保持对共产党的忠诚。
李说，他在统战部管理的一个 “流动 “刑讯所里呆了大约9个月。
“他们使用的方法真的很下作。他们威胁、侮辱和恐吓你。这些人都是统战部的官员，有男有女，有时身份不明，通常穿着便衣。警察对此视而不见。”李先生告诉RFA。”你必须接受他们为你准备的口供。如果你拒绝，就会被视为态度恶劣，他们会把你关在拘留所里，继续打你。
“没有窗户，不通风，也不允许外出时间。每天只给我两顿饭，由指定的人送到房间里。”李先生描述道。”我睡不着，在里面呆了一个星期后，死亡开始比呆在那里更好看。我把自己撞在墙上自残。”
RFA用一位代表受迫害基督徒的律师的证词证实了这个故事，他说基督徒和法轮功学员都受到类似的酷刑和洗脑设施。
“这些洗脑场所与对法轮功的洗脑场所类似。宗教事务官员逮捕了主教和牧师之后，他们并没有追究刑事指控–他们只是让他们失踪，有时一连失踪5年、6年甚至10年，”律师说。”有些人在五六年后就被送回了家，人们就是这样了解到洗脑中心的–从他们的口中得知的。”
广泛的证据显示，中国对基督徒的迫害从较温和的手段，例如向贫穷的基督徒提供食物或其他经济奖励，让他们将家中的十字架换成共产党的标志，到对监狱中的基督徒施加极端的酷刑和人权暴行。人权杂志《苦冬》的一篇报道上周引述了全能神教会（该国一个被禁止的基督教派别）的成员的话，强调了后一种情况。其中一名成员是一名妇女，她匿名告诉该杂志，她忍受了酷刑手段，如被强迫每天站立18小时。
“第四天，当我站立时，我的脚变得麻木，感觉像灌了铅。晚上上厕所时，我不得不用手扶着膝盖慢慢移动。我的脚背和腿都是黑色的。”该女子讲述道。”如果我在厕所里呆的时间超过了他们规定的三分钟，犯人就会大喊大叫催促我，并往我身上浇冷水。由于他们不止一次地往我身上泼水，我的鞋子总是湿漉漉的，使我的脚更加肿胀，以至于我很难把脚穿进鞋子里。”
“这名妇女指出，当她顶住压力签署一份据称是她自己写的放弃基督教的声明时，酷刑加剧了。
另一方面，还没有被政府监禁的主流信仰的基督徒感受到了来自地方官员的压力，特别是在农村地区，因为在那里，政府以前没有采取如此严厉的措施来消除信仰。这些手段尤其用在少数认同名义上合法的三自爱国教会的基督徒身上。
2020年的一份报告显示，三自成员已经开始接受当地共产党官员到家中的探访，要求他们移除所有信仰的标志，并以习近平或毛泽东的形象取而代之。据报道，一些人失去了获得福利项目的机会，或者得到了额外的经济援助，以换取重新装修他们的家。
“这些是最伟大的神。如果你想崇拜某个人，他们就是。”据报道，去年，一名共产党官员对山东的一名基督教村民说，他指的是毛泽东和习近平。


- **[04.02 Breitbart：菲律宾发现中共国在其珊瑚礁上的非法建筑物](https://www.breitbart.com/asia/2021/04/02/philippines-discovers-illegal-chinese-structures-reefs/)**


菲律宾武装部队4月1日说，菲律宾军方发现中国在南海部分岛礁上 “非法 “建造的新建筑。
菲律宾武装部队3月30日在菲律宾帕格卡凯萨银行附近进行了海上巡逻，在一些地区的岛礁上发现了中国新建造的建筑。
“在这次巡逻中，我们能够记录到一些特征上建造的人造结构。这些结构是非法的，”法新社参谋长Cirilito Sobejana中将周四在一份声明中说。
帕格卡凯萨银行是西菲律宾海的一个环礁，包含多个岛屿和暗礁，西菲律宾海是马尼拉对位于菲律宾正西方的南中国海部分的称呼。帕加凯萨银行属于菲律宾巴拉望省卡拉亚安市管辖。
“海洋法 “赋予菲律宾对该地区无可争议的专属权利。这些建筑和其他活动，无论是经济还是其他方面，都有损于我们领海的和平，良好秩序和安全，”Sobejana周四表示。
“我们已经向外交部和国防部等上级民政部门通报了这些侵犯我国主权的行为，”他补充说。
索贝贾纳将军提到了《联合国海洋法公约》（UNCLOS）。在菲律宾提起的一起具有里程碑意义的案件中，根据《联合国海洋法公约》设立的独立仲裁法庭在2016年裁定，北京对几乎所有南海的主张都是非法的。北京拒绝接受这一裁决，并从那时起就一直按兵不动，对大约90%的南海提出非法主张。
菲律宾海岸警卫队3月7日发现220艘据信由中国海上民兵操控的中国渔船 “排队停泊 “在位于帕格卡凯萨银行东北的胡安费利佩礁。胡安费利佩礁位于菲律宾巴拉望省以西约175海里处，位于马尼拉的专属经济区（EEZ）内。
马尼拉曾多次要求中国从胡安费利佩礁召回其渔船队，但北京至今拒绝移动这些船只。作为回应，菲律宾军方已下令在胡安费利佩礁附近加强 “海上主权巡逻”，包括菲律宾战斗机每天飞越。在最近的一次海上巡逻中，发现了中国在帕格卡凯萨银行新建的 “非法 “建筑。

- **[04.02 NY Times: 中共国对外资银行设限令企业忧心忡忡](http://中国对外资银行设限令企业忧心忡忡)**

旨在遏制大额资金流动并可能控制中国货币的新规则，可能会给国内竞争对手带来竞争优势，并使国际企业更加依赖本地贷款人。
上海–为了抵御华盛顿和其他国家对中国在贸易上不公平的指责，北京可以把矛头指向银行。中国领导人一直在稳步降低他们在中国庞大的金融体系周围设置的障碍，让华尔街和欧洲贷款人有更大的机会在这个世界第二大经济体中赢得业务。现在，这堵墙又要建起来了。
据三位知情人士透露，中国的新规定极大地限制了外国银行在中国开展业务的能力，使其在与当地竞争对手的竞争中竞争力下降。12月和1月颁布的一套规则限制了外资银行从海外转入中国的资金数量。其中两位人士表示，周三生效的另一项规定要求许多外资银行减少贷款，并出售债券和其他投资。
有关人士表示，新规则在全球银行高管和在中国的外资企业中引起了轰动，这些企业的资金都依赖于这些贷款人。除其他担忧外，他们还担心，这些规则可能会让外资企业更加依赖中国国有银行系统来获得发展所需的资金。这种依赖性可能给北京带来另一个潜在的压力点，因为它在贸易、人权、地缘政治和其他棘手问题上与华盛顿和其他国家对峙。
银行和贸易团体一直不愿意公开发言，因为担心会引发进一步的监管措施。但在1月份写给中国央行的一封信中，中国欧盟商会对汇款限额提出了担忧，这封信被《纽约时报》评论。
信中说，”在某些情况下，这种重大结构性变化带来的风险可能会从根本上颠覆欧盟银行在中国的战略发展方向”
新规则可能会使已经棘手的政治问题复杂化，比如美中贸易战、中国和欧盟之间悬而未决的投资协议，以及北京如何控制中国货币价值的长期紧张局势。拜登总统对前任唐纳德-J-特朗普（Donald J. Trump）提出的贸易要求几乎没有兴趣松口。双方还在人权问题上发生冲突，特别是北京对中国西北部新疆地区大部分穆斯林少数民族的镇压，以及对香港民主活动人士的镇压。
“我可以理解这是要保护金融稳定，”美国前财政部官员马克-索贝尔说。”我也能理解你如何认为这是对外国银行的歧视。”
中国人民银行是中国的央行，也是规则的发布者，但没有回应置评请求。
中国严格控制资金进出中国，可能担心资金涌入中国可能会导致通货膨胀等恶劣的意外。去年下半年，随着中国经济摆脱流行病的低迷，而世界其他大部分地区的经济活动萎缩，资金涌入中国。
