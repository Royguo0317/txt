接下來仍然是抹黑郭文貴先生和班農先生的陳詞濫調，說郭先生國內被指控洗錢，詐騙，強姦以及其他的罪名，他標榜自己反共被中共迫害，並與班農合夥吸引了大批的追隨者。郭先生說GTV在今年春天已經非公開籌集了3億美金，郭先生的伙伴投入了1億美金，這些錢一部分來自小投資者如一些家庭主婦，藍領工人和學生。文章還扯出班農先生日前被捕接受調查的事件。同時指出班農先生在春天集資期間也在GTV上做事，是不是要以後連班農先生也要牽涉進這個所謂的集資詐騙案裡？
