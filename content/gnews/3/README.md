**西班牙2021年3月1日电/西喜社——**巴西首都已经进入为期两周的封锁状态，与其他几个州一起采取措施减少中共病毒的传播。

由于中共病毒的病例和死亡人数上升，过去一周巴西至少有8个州采取了宵禁措施。周四是巴西自疫情开始以来最致命的一天，有1541人被证实死于该病毒。到目前为止，已经有25.4万人死亡。

巴西利亚州长伊巴尼斯-罗查下令全面关闭酒吧、餐馆、商场和学校，直至3月15日，并禁止人员聚集。晚上8点以后禁止销售酒精饮料。

在联邦区，据当地卫生部统计，周日85%的医院病床被占用。

总统贾伊尔-博尔索纳罗（Jair Bolsonaro）再次批评了这种措施，他在自己的推特账号上说。”人民想要工作” 他周五威胁说，要切断联邦对采取封锁措施的各州的紧急大流行病援助。
