## 香港國安法三十八條，遠比你想的更邪惡
`2020-07-03 06:46 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/252694/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/03064101/Picture-1-22.png)圖片來源：ANTHONY WALLACE/AFP via Getty Images 
近日，《寒冬》雜誌發表了文章，題為《香港國安法三十八條，我們都將會被捕》。文章認為《香港國家安全法》，其實是將中國大陸現有的《反顛覆法》擴展到了香港，並把對中共任何形式的批評都定為犯罪，同時公然撕毀了對言論自由和司法獨立的保證。中共還建立了一個新的機構，即“維護國家安全公署”，用以在當局認為特別嚴重的情況下執行該項法律。該辦公室將在很大程度上屬於保密機構，並有權將香港被告嫌疑人引渡至大陸受審。

儘管很多專業學者和觀察員普遍認為，該法的實施，將會使對香港的管制變得比中共國內地更為嚴厲，但筆者認為有一項非同尋常的條款並未引起足夠重視。這就是該法第38條的規定：“本法適用於非本地區永久居民，從香港以外地區對香港特別行政區所犯的罪行。”對這段拗口文字的簡單解釋就是，中共國正在對這顆星球上的每個人主張管轄權。

筆者進一步闡述， 一名非香港永久居民，如果寫文章來支持香港的抗議活動，或批評中共政府對西藏和新疆人民的宗教迫害，就可以被指控為“分裂主義”而判刑。又或者因批評中共違反《香港基本法》，並向信息管制下的中共大陸民眾傳遞了真實信息，就可以被指控損害了中共國的“國家安全”而被捕入獄。更為糟糕的是，即使你身為外國公民，根據《香港國家安全法》第38條的規定，該法律也適用於“不是該地區永久居民”的所有人，就是說如果你踏上香港的土地，便可遭到逮捕。

對此，一名研究中共國法律的教授做了更詳細的解釋，他指出，該法第38條的規定“使該法的適用範圍比中共國大陸法律更寬泛”。他還進一步舉例說明：首先，比如某人批評中共國占領西藏是非法的，那麼根據該法第38條，應適用於《香港國家安全法》的管轄範圍，如果這名批評者踏足香港就可以被捕。其次，假如上述批評者不是去香港，而是去北京。儘管他根據中共國刑法沒有犯下任何罪行，但根據該法第38條，香港當局依然有權要求中共大陸當局將其移交至香港，這還包括中共國和香港的飛機，以及中共國駐世界各地的使領館。

中共國聲稱擁有全球管轄權顯然是荒謬的，根據國際法是非法的。該法第38條意味著，如果對中共的批評者踏上廣義上的中共國領土，他們將被逮。如果人們天真的認為這在現實中不會發生，那麼可以看看加拿大的“兩個邁克爾”正在發生什麼。雖然是不同的情況，其內在邏輯卻是一樣的。

更為可恥的是，在新近召開的聯合國會議上，只有27個國家（加上美國，由於其與人權理事會的分歧而未簽署），支持英國在聯合國聲明中譴責香港和新疆的侵犯人權行為，而另外53個未公佈名單的國家簽署了古巴支持中共國的聲明，他們是新的恥辱之國。

讓我們來看一下這27個捍衛自由與人權的國家：澳大利亞，奧地利，比利時，伯利茲，加拿大，丹麥，愛沙尼亞，芬蘭，法國，冰島，愛爾蘭，德國，日本，拉脫維亞，列支敦士登，立陶宛，盧森堡，馬紹爾群島共和國，荷蘭王國，新西蘭，挪威，帕勞，斯洛伐克，斯洛文尼亞，瑞典，瑞士和英國。聰明的讀者可以猜到哪些國家不在此列，尤其是所謂民主國家。當他們的公民開始在中共國被捕時，他們也許會醒悟吧。

評：

38條這一毫無法理依據的條款，是中共對全世界赤裸裸的恐嚇，這無疑讓世人又一次看清了中共的流氓本色。我們千萬不能小看了中共的野心和狂妄，即使不進入香港的外國公民，也可能因其所在國與香港有引渡條約而被移交至香港接受審判。聯合國人權理事會的53個支持中共的國家進一步證實了中共國對全世界的侵蝕，也為全世界正義的人們敲響了警鐘。
