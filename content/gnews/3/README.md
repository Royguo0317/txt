5月21日，日本雅虎网站登载了【每日新闻】的题为【中国当局からスパイ容疑で116日間拘束された「在日華僑」　“恐怖の取調べ”と“真の狙い”を語る】的文章。题目大意是：驻日华侨诉说：以涉嫌间谍罪被中共当局拘留116天的旅行，“恐怖的审讯”和“真实的目的”。控诉者是75岁的老华侨，冈山县华侨华人总会刘胜德会长。对刘胜德会长进行采访的是日本记者吉村刚史。通篇不仅报道了刘胜德会长被中共拘留116天的遭遇，字里行间更是藏尽中共内斗的丑态。

之一【精神的な拷問】（中文：精神上的折磨）

“王对你说了什么？”，“你与王交谈了多少次？”…… 我清楚地记得审讯的内容，审讯一直在持续。他们将王毅先生简称为“王……”，并试图彻底调查他在日本的言行。说这番话的是”75岁的冈山县华侨华人总会会长刘胜德先生。

王毅（67岁）。 中华人民共和国国务委员兼外交部长；2004至2007年担任驻日本大使。刘先生是生活在日本的第二代华侨，1946年出生在岛根县出云市。 他的父亲出生于福建省，他没有选择继承服装店这份家业，而是与中国外交使团合作，长年致力于提高在日华侨华人和中国留学生的地位，支持他们的日常生活和学习，并举办促进日中友好的交流活动。三十多岁时，他到冈山任一家中餐馆的经理，后来在中共国的支持下，他开始独立，并一度成为当地一家知名中餐馆“天安门”的经理。然而，2016年11月22日，他和他的中文翻译在江苏省苏州市被拘留，大约一个月后被转移到天津。 直到第2017年3月17日，遭受了116天的严酷审讯。

刘胜德会长说：这些人自称是国家安全部的人，并告诉我，我将因涉嫌从事间谍活动而被拘留600天，之后我被置于他们的24小时监视之下。

刘胜德会长还说：他们威胁我说被枪决，被关押10年都有可能，并对我日夜审讯。被反复问及我与驻日本前大使王毅的关系，以及王毅在冈山县的一次演讲中说的话，甚至被使用了测谎仪。
