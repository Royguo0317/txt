## 誰是我們真正的敵人（三）
`2020-09-27 13:00 加拿大喜庄园🇨🇦` [轉載自GNews](https://gnews.org/zh-hant/384898/)

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/26172231/%E9%AD%94%E6%80%A7.png)
魔性的特徵是啥？他唯一的目的就是為了奴役別人，根本沒有任何的法治觀念。在動物性的時候，人是叫衣冠禽獸，到這裡就是禽獸不如，這就是魔性。

這些詞用在中共身上，特別精準。它講共產黨員就是特殊材料做的，就是有黨性沒人性，它根本就不是人，強調的是魔性。魔性就是鬥爭，消滅別人，就是為了奴役全人類，這就是魔性。所以，從這個角度上來說，我們就看清人類演變的趨勢，根本沒有發展，而是倒退。從一開始向神學習，敬天敬神，倒退到自我膨脹，不用學神，學人就行，學自己就行，那就變成人性，講點仁義禮智信，講點做人的基本準則。再倒退，被達爾文的進化論洗腦，其實根本沒進化，是退化論，然後就退化到人即高級動物，人就是猴子變的，這根本就是謬論。達爾文的進化論毒害全世界、全人類。達爾文自己說，他寫的《進化論》是魔鬼的聖經，其目的就是要跟基督教對立，消滅上帝。 《進化論》被馬克思和恩格斯非常推崇，馬克思和恩格斯就引用達爾文的觀點，兩個人互相論證，他們都是魔鬼。

在我們共產中國，達爾文的進化論是官方的權威學說，我們人就是高級動物。既然我們是動物，當然我們就以動物的要求對待自己。動物的要求就是弱肉強食，爾虞我詐，食色性也。有吃的有住的，然後就去交配，政治權利跟你沒關係，我們就是這樣被毒害的。我們的教育裡面，大量的這種邪惡的教育，這就是在奴化我們，這就是愚民教育。愚民教育的根本，其實是要激發我們的動物性。共產黨邪惡的教育本質，是要激發起我們的鬥爭精神，搞階級鬥爭，人與人互相鬥，不擇任何手段地害人，沒底線地害人，這就是魔性。

人類就是這樣一個不斷倒退的過程，這是一個根本規律。我們用中華傳統智慧的理論來看，我們就會意識到世界上各種各樣的學說、理論，都可以歸納到這四類裡面。譬如說，有些宗教講敬畏上天，向神學習，這就是神性的理論。有的講怎麼做人，講道德，這就是人性理論。有的講，只要享受就好，其他的都不管，物質主義，只要在物質裡面追求快樂，然後就是各種交配、各種性，這就是動物性理論。如果有人講各種商鞅思想，講馭民五術，講馬克思主義，講階級鬥爭，講超限戰，這就是魔性思想。

共產黨在中華大地站穩的原因，就是在國內推行愚民教育，很多人也在點評中國的教育，但都沒有看到中國教育的本質。中國教育的本質，其實就是要扼殺神性和人性，然後激發起動物性和魔性。我們中國凡是真正宗教都被中共消滅了，最後只能信一個，就是共產黨，其本質就是魔性。只要激發起人的獸性、動物性，人就不會造反了。所以，共產主義、共產黨的本質就是邪教，它完全符合邪教的十二個特徵，而不僅是暴政。中國教育的本質是摧毀神性和人性，它鼓勵、鼓吹的是動物性和魔性。在國際上，為什麼中共藍金黃能夠成功，就是因為激發出了他們的動物性和魔性。譬如說，給美國的政客、企業家送錢賄賂、換器官、送明星、送幼女，就是在激發他們的魔性、動物性，然後滿足他們的魔性和動物性，再為中共所用。

現在，整個世界正處在一個歷史的十字路口，七哥提出的正道主義，其實就是要恢復人性、回歸神性。神性就是宗教、信仰，人性就是做人的基本。我們不能去做動物，不能把自己降到一個動物的級別。現在根本性的問題就是中共不讓你信真正的教，只讓你信他的邪教和假的教。這是兩種價值體系，不能共存，所以我們要做一個判斷，做一個選擇。
