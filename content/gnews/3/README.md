据美联社（AP）报道，美国和中共国的高级官员本周四在阿拉斯加州安克雷奇市会晤，对彼此和世界提出了截然不同的看法，这是美国总统拜登（Joe Biden）上任以来双方首次进行面对面的会晤。

在阿拉斯加开始的两天会谈中，美国国务卿安东尼·布林肯（Antony Blinken）和中共外事工作委员会办公室主任杨洁篪在公开场合发表了异常尖锐的言论，双方都将矛头指向对方国家的政策。他们的激烈交锋表明，双方的私下讨论将更加困难重重。在安克雷奇举行的会晤是两国关系日益陷入困境的新考验，两国在从贸易到西藏、香港和新疆西部地区人权等一系列问题上，以及在台湾、中共国在南海的强硬态度和中共病毒大流行等问题上都争执不休。

布林肯说，拜登政府与其盟友团结一致，回击中共国日益增长的威权主义和对国内外愈发强硬的态度。随后，杨洁篪火力全开，指责华盛顿在人权和其他问题上批评中共是虚伪的。

布林肯在谈到中共国在新疆、香港和台湾的行动，以及对美国的网络攻击和对美国盟友的经济胁迫时说， “这每一项行动都威胁到维护全球稳定的基于规则的秩序。 这就是为什么它们不仅仅是内部事务，也是为什么我们觉得有义务今天在这里提出这些问题的原因。”

美国国家安全顾问杰克·沙利文（Jake Sullivan）更是认为中共国进行了一次 “基本价值观的攻击”。他说，”我们不寻求冲突，但我们欢迎激烈的竞争。”

杨洁篪愤怒地回应，他要求美国停止推行自己的民主版本，考虑到美国本国内也有不满情绪。他还指责美国没有处理好自己的人权问题，并表示对布林肯、沙利文和其他美国官员的 “居高临下”不服。“美国现在需要的是改变自己的形象，而不是推广自己的所谓民主，”他说，”而美国自己国内现在很多人对美国的民主并没有信心。”“对中国采取卡脖子的办法，最后受损失的是自己。”王毅附和道，“ 中国的正当权益受到了无理打压，中美关系遭遇到了前所未有的严重困难，这种局面损害了两国人民的利益”。

中共一方持续抨击达15分钟以上，布林肯似乎对这些评论的基调和长度感到恼火：“我（从日本和韩国）听到的全是满意的反馈，他们表示美国回来了，我们重新开始合作，而与此同时，我也听到了对你们政府一些行动的深切关注。”为了进一步表示敌意，美国国务院抨击中共国代表团违反了商定的两分钟的开场白时间限制，并表示 “中国代表团似乎是为了讲大话而来，专注于演戏，而不是讨论实质内容”。“美国对待中共的策略取决于由实力产生的信心，哪怕我们谦虚地知道我们美国在永远致力于成为一个更完美的国家的路上努力着。”

在阿拉斯加会晤的前一天，布林肯宣布了对中共镇压香港亲民主人士的新制裁。作为回应，中共加大了反对美国干预内政的言论，直截了当地开喷。对此，王毅表示：“如果美方试图以此举（制裁）来所谓增强你们对华的优势的话，我想恐怕也是打错了算盘。只能证明内心的虚弱和无力，丝毫不会影响中国的正当的立场，也不会动摇中国人民的已知。”

在川普总统上任初期，他曾为与中国领导人习近平建立牢固的关系而感到自豪。但是，这种关系在中共病毒大流行并引发了一场公共卫生和经济灾难之后就瓦解了。中美关系已经紧张许久，而拜登政府尚未表示是否准备好或愿意摆脱川普政府（对中共）采取的强硬立场。
