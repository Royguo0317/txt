## 【香港要闻】 港人百亿 美元 转移加拿大史上最高水平资金 逃亡浪潮
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1028641/)

搜集\编撰：西西

**【港版国安法】实施在港产生震撼性的资金寒蝉效应。加拿大反洗钱机构「加拿大金融交易与报告分析中心」（FINTRAC），追踪了1万加元以上的转帐记录，共计有约436亿加元（合约348亿美元）的电子资金从香港转移到加拿大，这是FINTRAC有记录以来的最高纪录。**
![]()![](https://gnews.org/wp-content/uploads/2021/03/ddfdfdf-1.png)《港版国安法》实施后，港人已将数百亿美元转移至加拿大，未来可能出现更大规模的移民与资金转移浪潮。


大纪元香港3月26日引述路透社报道指，FINTRAC发言人吉布（Darren Gibb）表示，FINTRAC的数据只涵盖合法流入加拿大经济总量的一小部份，因为许多交易并不包括在内，比如通过加密货币、金融机构之间的转账，或者低于1万加元的交易等。加拿大贷款机构公平银行（Equitable Bank）也表示，在2020年6月《港区国安法》上路后，来自香港的存款激增。

香港移民顾问公司Anlex的行政总裁罗立光（Andrew Lo）表示，过去12个月，他帮助了约36个家庭移民加拿大，平均每个家庭都为加拿大带来150万加元。移民专家透露，即使出境时间因旅行限制和移民办理时间放缓而推迟，一些人已经开始将钱转入加拿大帐户，并开始关注加拿大的房产。

公平银行个人银行业务集团主管莫西马·帕达尔（Mahima Poddar）表示，与香港有关的帐户余额，出现了「显著增长」。这些帐户的平均余额，在去年6月至今年3月中旬期间上升了30%，相比之下，与香港无关的帐户仅上升了4%。专门为香港高资产人士提供移民服务的让·弗朗索瓦·哈维（Jean-Francois Harvey）表示，自2020年年中以来，寻求移居加拿大的客户增加了5倍。他表示，他的客户在过去12个月内至少都转移了100万加元，许多人更转移了500万～1,000万加元。
![]()![](https://gnews.org/wp-content/uploads/2021/03/vhvhhnhnh99990.png)汇丰官网对加拿大居民发出提示，称暂不受理开立新户口申请，引起港人热议，质疑是否该行为防止走资出新招。 （宋碧龙/大纪元）
**点评：**

汇丰银行已经沦为中共在香港的金融打手。有网民最近注意到，汇丰银行网站「开立海外户口」页面显示一条通知，「对加拿大居民的重要提示：由于需求量高，现时我们暂不受理任何开立新户口的申请。」

哪里安全，钱就会往哪里跑。汇丰的这一举动，明显就是呼应港人这种历无前例的数百亿美元转移至加拿大。然而，港共政府无法留住民心，何来留住资金？

中共去年在香港强行实施《港区国安法》，加上去年香港当局肆意冻结了许智峰等的个人帐户，引起了港人对资产安全的担忧，从而使更多人将部份资产移到海外。同时，中共对香港自由的压迫，港人希望移居民主国家的渴望，也进一步推动香港的资金移出。

名移民顾问、律师及房地产经理人向路透社表示，许多香港居民渴望在民主国家开始新生活，一旦中共病毒疫情所引起的旅行限制结束，将会出现更大规模的移民和资金流出。

今天，中共已经全面接管香港，共产党体制下，谁的资产都不安全，从盗国贼们用尽各种方法把他们盗来的财产转移到国外藏起来，我们就不难理解共产党下，私人资产随时会被「共产」。
