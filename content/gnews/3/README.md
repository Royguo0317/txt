## 【香港要闻】港共政府 急不及待 向香港市民 接種中共科興毒疫苗
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/917080/)

编撰 ：恩典
![]()![](https://gnews.org/wp-content/uploads/2021/02/Venom.png)
**当香港市民向港共政府提倡福利，永远拖字诀完事，但今次为了香港市民可以接种到中共疫苗，果然效率升上至高点，根据食卫局公布有关科兴疫苗审核报告，科兴的巴西第三期临床研究，只有5.1%参加者属60岁及以上，数据有限，故没有足够证据证明对长者的有效率。但政府今日先公布5个优先接种群组，包括60岁以上长者，以及10间院舍院友接种科兴。**

**评论：**

**很多人难以置信中共科兴仍未于医学期刊刊登第三期临床研究，外界未能掌握全面数据。食卫局公布科兴疫苗评核报告，指对18至59岁疫苗有效率为50.66%，对60岁或以上长者有效率只有51.11%，因只涉及研究419名长者，数据有限情况之下，港共如此急着安排港人接种中共科兴疫苗。中共一个处心策划的毒针计画，香港人在今天的局势连最基本的健康问题，都沦为中共的政治资本。一个对人类没有尊重的政权，只带来更多的毁灭。**
