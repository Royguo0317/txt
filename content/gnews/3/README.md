![]()![](https://gnews.org/wp-content/uploads/2021/02/博裕资本-1.jpg)图片来源：搜狐网
近期传闻由江泽民的孙子江志成控制的博裕资本正从香港搬迁至新加坡，其原因是中共党内江派在习江斗暂处劣势，权衡之下将业务辗转到香港以外的东南亚地带另辟市场。不过据最新的消息，江家并没有弃战的迹象，似乎正酝酿着新的大动作。

路透社25日报道，博裕资本正为旗下一直专注于中国的新基金募集资金，目标60亿美元。据消息人士表示，这是博裕资本的第五支、也是最大的一支基金，可能会在短期内完成筹资。这一内幕消息被解读为习近平与江泽民恶斗的延续。

习、江两派之间的明争暗斗由来已久，两派的交恶则是出现在美国总统大选前夕。习近平代表着中共的新帮派，其与拜登家族的之间秘密关系掌握在以江家为首的旧帮派手中，旧帮派意识到拜登的上任将威胁到自己的生死存亡。于是，去年11月3日美国总统大选夜的前半个月，旧帮派抛出 “三块硬盘”释放出拜登家族与中共合作的一部分秘密，硬盘门引发的效应，直接导致了美国大选夜当天蚂蚁集团在上海和香港上市的流产，习近平在紧急叫停蚂蚁集团IPO一事上，宣誓了其在中共 “一把手”的绝对权威。

《华尔街日报》不久前对蚂蚁集团背后金主的披露，无异于对外承认习近平是为打击江派等政治对手才整治蚂蚁集团，显示习江斗已经从暗斗摆到了台面上。而《华尔街日报》22日再度发文，引内部消息指博裕资本正从香港搬迁至新加坡，则是江家感受到香港已在习近平权力触角下，不再是江家的天下。可见，江派在习江斗中暂时处于劣势，规避香港监管风险的举动是退而求其次。

不过，江志成在习江鏖战正酣之时高调大手笔募资，可以看作是博裕除了测试投资者的信心之外，还希望以此考验新帮派对江家资本的容忍度。江家对外传递着仍未弃战的信号，习江斗争的不可调和令中共政治局势危机四伏。
