## 【信仰之声】Pietro Parolin——披着主教外衣的谈判者

作者：纽约香草山福音部 小草



![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-109.jpg)



罗马教廷在二战中对希特勒的默许和媾和，在威廉·劳伦斯·夏伊勒的《第三帝国的兴亡》中有详细的描述。罗马教廷对第三帝国若有若无的中立，对纳粹国家社会主义狂热的沉默，对纳粹大屠杀模糊的表态等，让人有无数的疑问。二战后，罗马教廷诚心自省有限，虽然约翰·保罗二世为天主教会的错误道歉[1]，但很不具体，也很勉强。一个对自己错误没有足够反省的人和组织，同样的错误会反复发生，这是规律。

![]()![](https://gnews.org/wp-content/uploads/2021/04/71-1.jpg)

最近，教廷的当红炸子鸡Pietro Parolin，因为伦敦房产丑闻被曝光[2]，其中还牵涉出一个名为Peter’s Pence的组织，类似于教会基金投资机构[3]。而这个机构又牵出一个叫“Athena Capital”的基金管理机构，且其机构成员都有商业背景。这件事暴露出罗马教廷对捐献基金的收集、管理、投资、财务公布及第三方监督上，都有不少问题。

看完Pietro Parolin的履历和表述，我的印象是，他是一个罗马教廷的外交家，是世界舞台的“合纵联横者”，也是与极权社会的沟通者。他最爱用的词汇是，“博爱、多边主义、和平”[4]。好，我们来具体看一下他博爱的对象，多边主义的成员，以及和平实现的效果。

Pietro Parolin与中共最直接的关联是——2018年，罗马教廷与中（共）国达成协议。这个协议牺牲了教职人员信仰的独立性，全盘接受了一个极权体制选拔的主教人员（这些人员宣称忠于共产党）[5]，给中共国穿上了有宗教信仰的“皇帝新衣”——这是中共对罗马教廷的大胜利。然而，中共国随后出现了逮捕地下教会成员、要求神职人员忠于“一尊”、在教会里安装摄像头、拆毁各地教堂等系列事件，全面封杀了信仰自由。中共倒是享受了世界博爱，中共和罗马教廷也多边了，但中国人的信仰和平在哪儿呢？

中国人描述某个人狠辣，往往形容其为“所到之处，寸草不生”。我们来看看这个Pietro Parolin在教廷中“突破性的成果”。

1、教廷与伊朗的《核不扩散条约》。跟一个流氓和无赖谈判，好人要用他们听得懂的语言——拳头。至于外交、协议、世界多边主义这些，都是用来跟君子玩的。给流氓好意，是想让他再多骗你一会儿？再说，这个条约有用吗？如今伊朗随时在中东进行核威胁，对以色列骂骂咧咧，宗教极端化让人不寒而粟。

2、委内瑞拉独裁者的好朋友。委内瑞拉人都吃草了，跟查韦斯、马杜罗之类谈判，能多给人民一块面包吗？还是人民晚上饿着肚子睡觉前，空想明天生活会更好？这些独裁者是群众暴动的根源，而教会的正义是推翻独裁者，是支持人民，而不是政府。一个没有正义的谈判，不会被人真正执行。

3、欧洲难民危机。欧洲各国在解决叙利亚难民问题上是能力不足的，这更需要教会给予支持。而教会要首先解决难民危机的根源——叙利亚内政——而不是强行要求各国一定要承担随之而来的后果。

4、裁军。和平是个好意，但先把中共国和独裁社会的军队减一半，再跟文明国家谈裁军。没有武力装备的和平意愿，永远是独裁者的好朋友。

5、多边主义。一个教会要联合所有人，其目的一定是保护大多数人。倘若想跟独裁者交朋友，一是不要被他们骗了，另外就是不要太天真。世界博爱很好，但一定要有原则。这个原则是确保社会真正的进步和人道的改善——没有成果的博爱都是精神鸦片。

我们不否认罗马教廷对世界人民的善意，但这种善意必须务实，不能理想化。教会自身对资金的管理须公开、透明、可监督——我们对财物地管理，也体现了我们对信仰地持守。罗马教廷的资金亏空，是什么导致的呢？如何减少亏空呢？这些问题，希望今后有较好的解决。
