![]()![](https://gnews.org/wp-content/uploads/2021/02/Senkaku-uotsuri.jpg)图片来源：维基百科

以上报道未透露这位美国防高官是谁。但他(她)和国防部发言人在敏感而且至关重要的主权表述上大相径庭，或将加剧中共国、中华民国和日本在该争议区域的纷争。
