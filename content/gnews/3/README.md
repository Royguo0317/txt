文中提到湖北省會武漢已經恢復了正常。很多人不再佩戴口罩，而在地球的大部分地區，病毒仍在繼續肆虐。文章還稱「武漢自5月18日以來沒有再報告本地感染病例，中國應對疫情的經驗和採取的措施讓武漢成為目前世界上最安全的地方。」
