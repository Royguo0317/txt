## 美国将在G7会议上再谈新疆问题——却未提病毒真相

【日本东京方舟农场】作者：青衣 素材：老温 校对：小油鍋

![]()![](https://gnews.org/wp-content/uploads/2021/04/55AA6841-FC6C-49D0-B2F7-A99B4BCEA172.jpeg)

据《路透社》报道，白宫高级官员周五表示，美国将敦促七国集团盟友对中共国施加压力，迫使其对于在西北部新疆维吾尔族聚居区推行强迫劳动的行为做出回应。

白宫周五表示，拜登将于6月份前往英国和比利时，进行自其上任以来的首次外访，包括参加6月11日至13日在英国康沃尔举行的G7峰会，届时他将以他自己的经验，重点商讨民主国家与专制国家（尤其是中共国）之间战略对抗的议题。

《路透社》援引拜登政府副国家安全顾问兼国家经济委员会副主任达利普·辛格（Dalip Singh）的话称，在康沃尔举行的七国集团（G7）会议将侧重于健康安全；中共病毒大流行对经济的影响；针对气候变化的具体行动；以及“提高七国集团内部的共同民主价值观水平”。辛格在接受《路透社》采访时称：“这些都是志同道合的盟友，我们希望采取具体和切实的行动，表明我们愿意就中（共）国这样的非市场经济国家进行协调。” 他说：“七国集团面临的挑战是（向世界）证明一个开放和民主的社会才有最大的契机，能够解决这个世界上首要的（人权）难题，而自上而下的集权制度绝非可选途径。”

辛格表示，华盛顿已经针对新疆的侵犯人权行为对中共国采取了强有力的措施，但仍将寻求扩大与七国集团盟友的合作。就在上个月，美国、欧盟、英国和加拿大已宣布对被指控在中国新疆地区犯有侵权行为的中共国官员进行联合制裁。辛格称，在会议召开之前，细节仍在制定中，但是这次峰会为美国和盟国提供了表达对这一问题团结一致的机会。

辛格还对《路透社》表示：“我们已经明确表示，当他们进口的商品是由强制劳动制造的时候，我们的消费者应该有权了解（真相）。我们的价值观需要注入我们的贸易关系之中。” 他称，华盛顿将寻求七国集团采取明确步骤，“以推进我们的共同价值观，例如民主制，这些必然有助于当前新疆事件的解决。”

从表面形式上看，美国拜登政府一直在对中共采取进一步措施，就新疆种族灭绝一事督促盟友跟进，加大制裁力度。然而，现在全世界深受中共病毒侵害，新一波大爆发已在全球许多地方出现。在大规模接种疫苗、戴口罩以及实行封锁措施的情况下，印度、日本、俄罗斯、巴西等国每日确诊数和死亡数都在大幅上升，甚至不断创下新高。全球各国首领的当务之急应该是寻找病毒根源，向制造病毒的中共要解药，而不是将重点放在谈什么气候变化和侵犯人权上。

人权破坏乃至气候变化的问题，皆非当务之急，所以应该捋清事务的轻重缓急。即便是全球都认定中共犯下种族灭绝罪，对中共实施制裁，也无法阻止无数生命时刻因中共病毒而死去，以及接踵而来的世界经济萧条。面对中共病毒，拜登政府避重就轻的敷衍态度，这回已是一览无遗。英雄科学家闫丽梦博士一再警告世界：剩给人类的时间已所剩无几！因此，全球的注意力都应聚焦在病毒真相上，否则全人类将陷入穷途末路的境地。
