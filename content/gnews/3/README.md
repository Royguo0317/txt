##  **香港開始大規模病毒檢測，但因民眾對北京的不信任而受挫** 

香港週二啟動了一項大規模的新冠病毒檢測計劃，但由於中共政府對香港的民主運動的鎮壓，致使香港民眾對政府高度不信任，以至於要求數百萬人接受該提議的呼籲未能得到大家的響應。

免費的自願檢測是試圖杜絕從6月下旬開始的第三波感染的一部分，而且這個人口密集的城市還重新實施了對經濟非常不利的“社交距離”措施。

[!\[\](https://spark.adobe.com/page/RMFQo27gw4vmV/images/8a3bd297-fffc-45f3-9087-365b9bd5e9d8.png?asset_id=d0f26b6b-2560-47c2-86c0-56338b185e7b&amp;img_etag=%22a9f256e8ae70f5eac972e26a9cfd5f36%22&amp;size=1024)](https://spark.adobe.com/page/RMFQo27gw4vmV/images/8a3bd297-fffc-45f3-9087-365b9bd5e9d8.png?asset_id=d0f26b6b-2560-47c2-86c0-56338b185e7b&amp;img_etag=%22a9f256e8ae70f5eac972e26a9cfd5f36%22&amp;size=1024)政府人員採集鼻腔和喉嚨樣本。圖片：香港政府一站通視頻截圖。

但是，由於中共國大陸的測試公司和醫生的參與，大規模測試計劃因僅得到公眾有限的回應而受阻– 而且，隨著北京進一步鎮壓香港民主改革的呼聲，公眾對採集個人數據和DNA的擔憂也不斷升溫。

自周六報名開始以來，僅有51萬人註冊，約佔該市750萬人口的7％。

市領導林鄭月娥（Carrie Lam）說，週二上午約有10,000人接受了測試，這其中包括了政府各部門的部長。

她對記者說：“這項計劃有助於香港順利地擺脫大瘟疫，並有利於日常活動的恢復。”

但是為政府提供建議的健康專家表示，可能需要對多達500萬人進行測試，才能全面發現隱性傳播，結束目前的傳染潮。

[!\[\](https://spark.adobe.com/page/RMFQo27gw4vmV/images/65ac8ed6-d4c2-4d79-a478-6d517c8a819e.png?asset_id=fc814dd2-a03d-40e8-8b82-93a4df55b6e5&amp;img_etag=%22ce14f21a04013abaa973b8a85b0adbc5%22&amp;size=1024)](https://spark.adobe.com/page/RMFQo27gw4vmV/images/65ac8ed6-d4c2-4d79-a478-6d517c8a819e.png?asset_id=fc814dd2-a03d-40e8-8b82-93a4df55b6e5&amp;img_etag=%22ce14f21a04013abaa973b8a85b0adbc5%22&amp;size=1024)

自1月下旬該病毒首次侵襲香港以來，香港僅記錄了4800多個感染病例，但其中約有75％的病例是自7月初以來發現的。

三十多歲的母親溫妮·陳（Winnie Chan）在進入檢測中心時告訴法新社記者：“我這樣做是為自己，也是為了別人，我有信心，我支持政府的政策。”

而其他人則說他們不打算去報名檢測。

當地居民艾米麗·李（Emily Li）告訴法新社記者：“我認為這是浪費時間，政府無法用測試計劃的有效性來說服我。”

當局稱該計劃是在中共國的幫助下實現的一項慈善公共衛生舉措。

但是，由於中共國大陸團隊和實驗室的參與致使傳言四起，加劇了人們對北京，這個信奉監控的政權利用生物識別數據來監控公民的擔憂。

[!\[\](https://spark.adobe.com/page/RMFQo27gw4vmV/images/6c304e97-828c-41b4-a121-6b1d075ddcc1.png?asset_id=8517e85b-2924-4fc6-a9f8-c3b0757c4ee0&amp;img_etag=%22b6a1964e63c3d8c007ccd7da81cf30be%22&amp;size=1024)](https://spark.adobe.com/page/RMFQo27gw4vmV/images/6c304e97-828c-41b4-a121-6b1d075ddcc1.png?asset_id=8517e85b-2924-4fc6-a9f8-c3b0757c4ee0&amp;img_etag=%22b6a1964e63c3d8c007ccd7da81cf30be%22&amp;size=1024)政府人員採集樣本進行新冠病毒檢測。圖片：香港政府一站通

一些著名的香港健康專家對大規模檢測計劃的有效性提出了質疑，他們認為對高危人群和弱勢群體進行更有針對性的監測能更好地利用資源。

此外，他們還提出了這樣的擔憂：在一個目前應急規則為禁止兩個以上的人在公共場所聚集的城市裡。這麼多的人接受檢測的行為本身可能會幫助病毒傳播，

一部分支持民主的政界人士和立法會議員，包括著名的活動人士黃之鋒（Joshua Wong），紛紛呼籲公眾抵制這項測試計劃。

他們表達了對大規模DNA採集的擔憂，並擔心香港可能會像內地一樣引入強制性的衛生規範制度。

[!\[\](https://spark.adobe.com/page/RMFQo27gw4vmV/images/1cf2e9b7-56fd-4e65-9f75-c25884c1f13b.png?asset_id=379fcf65-cf27-4d7c-acdb-c0420240ebf6&amp;img_etag=%226d11546e9f37c05ad6c8559ef36a4236%22&amp;size=1024)](https://spark.adobe.com/page/RMFQo27gw4vmV/images/1cf2e9b7-56fd-4e65-9f75-c25884c1f13b.png?asset_id=379fcf65-cf27-4d7c-acdb-c0420240ebf6&amp;img_etag=%226d11546e9f37c05ad6c8559ef36a4236%22&amp;size=1024)社區檢測中心內的等候區。圖片：香港政府一站通視頻截圖。

香港政府一再否定了這些擔憂。政府否認將在檢測計劃中採集個人DNA，並堅持不會將採集的樣本送到大陸實驗室進行檢測。

林此前曾抨擊那些反對檢測計劃的人是“蓄意反北京、反政府成員”，以及“在即使涉及公共衛生問題時，他們也不放過任何製造麻煩、挑起對立的機會”。

而北京的港澳事務辦公室則稱那些反對檢測的人是“反華激進分子”，對公眾健康“不屑一顧”。

上週，林還駁斥了那些對全市範圍檢測的有效性表示擔憂的醫生，稱他們是出於政治動機，這激怒了一些醫學界人士。

[!\[\](https://spark.adobe.com/page/RMFQo27gw4vmV/images/6117bf7a-4904-496b-b902-1013a1be7afe.png?asset_id=c1e6d6c0-9969-4822-8748-9feef673ffca&amp;img_etag=%220f2dc771345450c01b04e95c2067e3f8%22&amp;size=1024)](https://spark.adobe.com/page/RMFQo27gw4vmV/images/6117bf7a-4904-496b-b902-1013a1be7afe.png?asset_id=c1e6d6c0-9969-4822-8748-9feef673ffca&amp;img_etag=%220f2dc771345450c01b04e95c2067e3f8%22&amp;size=1024) Photo: Kaiser/HKFP.

但是，一些表示擔憂的醫生是頂尖的流行病學家，他們一直在幫助政府對抗這種病毒。

在7月下旬第三波疫情的高峰期，香港每天約有150例新增病例。

為此，林鄭以感染率不斷上升為由，取消了每年一次的地方立法會選舉，這激起了親民主陣營的強烈不滿，該陣營曾希望利用公眾從去年大規模抗議活動後的憤怒情緒來贏取大選。

此後，新增感染病例回落到了個位數，星期一隻有9例新增感染病例。同時，政府反對者對政府提出質疑，認為如果政府能夠進行全市範圍的檢測，為什麼還要取消選舉。
