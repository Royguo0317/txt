据《国家脉动》独家揭秘：罗斯蒙特房地产公司是亨特·拜登的罗斯蒙特塞内卡咨询公司的一个分支，被中国共产党支持的公司收购，被美国国会专案组认定为 “中国军方的薄弱环节”。

2015年，中国双子座投资公司收购了罗斯蒙特房地产公司的控股权，该公司是亨特-拜登2009年创立的罗斯蒙特塞内卡公司的姐妹公司。这家中国公司承诺投入30亿美元用于在美国的 “商业办公地产收购”；换句话说，让这家公司控制了美国房地产的大片土地。

双子座投资是远洋集团的子公司，远洋集团是一家国有企业，其总裁在中共党内担任高级领导职务。根据该集团的 “发展历程”，”远洋集团董事长兼总裁李明作为新当选的第十三届全国政协委员出席了开幕大会。”

远洋集团也是中远海运的分公司，这家总部位于上海的物流公司颇具争议，被美国众议院的一个专案组贴上了 “中国军方的单薄武装 “的标签。更重要的是，该公司13名高管中有11人在中远的网站上吹嘘他们与中共的高层关系。

除了与一个充满敌意的外国政府签订交易外，这笔交易还让中共不受限制地进入并控制了该公司的房地产组合，”33个物业，共56栋楼，总面积约800万平方英尺，分布在13个州”。

这种做法是中共关联企业常见的做法，按照美中经济与安全审查委员会的说法，它的功能是中共掠夺性和危害国家安全的美国房产收购的主要途径。根据该委员会2019年的总结：”这些努力往往是在中国政府的指示和协助下进行的，是中国政府为发展国内市场并成为各种技术的全球领导者的更大努力的一部分。这些收购尝试经常针对人工智能、生物技术和虚拟现实等先进技术，这些技术仍处于发展的早期阶段，但未来可以提供军事和民用双重能力。”
