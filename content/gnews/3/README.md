## 从Covid-19大流行到中国维吾尔族的镇压：2020年人权观察(节选）

撰稿人 喜妈 喜安娜
![]()![](https://gnews.org/wp-content/uploads/2021/01/IMG_0232.jpg)图片来源：france24.com
去年，由于冠状病毒大流行，超过一百万人死亡，数十亿人被封锁。这在社会和政治上具有标志性意义。我们将与人权观察执行主任肯尼斯·罗斯（Kenneth Roth）讨论健康危机对全球人权的影响。我们还谈到美国和法国警察暴行的根本原因。最后，我们问如何做才能结束中国少数民族维吾尔族穆斯林遭受的持续镇压。

主持人：

人权观察组织以带着疑惑并发布了对全球人权状况的年度评估报告。 他们对整个一年过去后，冠状病毒这一流行病对健康和社会在各个方面的影响进行了关切考察。人权运动人士关注的一些主要问题，过去一年，包括警察的暴力行为和中国维吾尔族穆斯林人民面临的持续歧视。我今天的客人是肯尼思·罗斯（Kenneth Roth）人权观察组织执行总监。实际上，非常感谢您今天在France24对我们发表的讲话。 我们知道，这次大爆发疫情至今夺去近200万人的生命，我们为还有数十亿人处于世界封城状态而受到影响而感到遗憾。疫情爆发如何在哪些重要方式上影响了全球的人权？

罗斯先生：

我们已经看到许多政府以大流行为借口巩固其权力，并让批评者保持沉默。今天你提到在乌干达举行的大选。这是一个很好的例子。总统穆塞韦尼（Museveni）利用流行病以防止其主要竞选竞争者巴比-旺（Bobi Wine)。如你所知的，我们已经看到在欧洲，匈牙利总理奥尔班博士，他在没有议会的情况下就抓住独裁政权的法律来号令；理由是它有能力为匈牙利人服务。实际上他没有足够理由用欧盟资助基金建立足球场以帮助自己的亲信而与此同时，医院却完全破旧。

从我们所了解的来说，在大爆发疫情中，特别的，信息的自由流通和有能力去批评我们的政府作出回应的能力对于公共卫生至关重要。我们可以追溯到武汉，最初中国政府对吹哨人们进行3周的静音开始。那些医生试图警告我们有关人与人之间传播的信息。这个处理，使该病毒在这三周内传播到全球，成千上万的人每天出行，3500每天旅行海外。这就是你需要知道的，我们需要人权本质。我们需要争论，这才能最好地服务于，特别是像如大爆发疫情这样关于与公共卫生危机有关的事情。

主持人：

当然，大爆发疫情导致了世界范围内的封锁，当然其中一些事情仍在继续。我们知道封锁的意图加剧了性别差距。妇女比例过高地呆在家里，为家庭成员做饭，做清洁，教育孩子，因为学校关闭了。妇女为何被如此高比例地受到疫情大爆发影响？

罗斯先生：

我要向你提到的另外一件事是，妇女遭受家庭暴力的情况大大增加。他们有时会被有虐待倾向的伴侣枪杀。所以我想说的是，从某种意义上来说，这反映了歧视和性别角色以及照顾孩子的负担。我认为妇女在很多方面都面临挑战。妇女不成比例地被要求呆在家里，他们面对如何应付去上学，甚至去杂货店都是一个巨大的挑战。

我认为，疫情大爆发为我们聚焦了指出了社会中的某些差异，不仅是性别差异，还有种族和民族差异，通常是社会中的种族和少数族裔。作为这些人，他们患病的几率更高，死亡的可能性也更大，也更加欠缺可能的渠道得到健康医疗和疫情爆发中的服务。我们希望通过揭示这些不平等之处，帮助他们去找到一线希望，使他们可以获得他们所需的医疗服务并在大流行中得到相关服务。使得每个人都有能力养得活家人哪怕失去工作，哪怕减少收入，孩子都能有权利接受合适教育，即使他们承担不起网络费用或大多数学校所需的手提电脑。而这些不平等的确出现这个疫情大爆发中。
