引用：https://www.foxnews.com/world/china-used-internet-troll-army-to-shape-coronavirus-narrative-report

近日，从中共内部泄露的文件中的更多内容被公布出来，其中包含了3,200多个指令和1,800条备忘录以及中国网络空间管理局（CAC）的其他文件，还包括了中国公司Urun Big Data Services的计算机代码，Urun Big Data Services是地方政府的服务提供商。

ProPublica和《泰晤士报》将黑客组织CCP Unmasked确定为文件来源–并宣布他们已验证了许多材料的真实性。

泄露的文件中描述了对2月7日李文亮博士死亡和其他事件的回应。报告说，中国政府努力的两个主要目标似乎是使病毒影响看起来不那么严重，并使政府官员在控制疫情方面的控制力比实际要高。

而李文亮医生死亡的消息警告了公众关于该病毒的消息，这引发了大规模的宣传和审查工作，以帮助中国政府重新控制国内有关迅速增长的危机的叙述。

该报告还明确提到了中国国家主席习近平于2014年启动了名为中国网络空间管理局（CAC）的行动。

中国的审查员担心有关李文亮医生之死的消息会引起随后有关该病毒的新闻报道的“蝴蝶效应”，这可能会对政府造成伤害，因此他们下令新闻媒体不要宣传有关李文亮的故事，并指示社交媒体网站删除所有有关话题。

随着冠状病毒逐渐成为一个更大的新闻报道，审查员还雇用了网络水军发布“分散注意力的言论”，以帮助影响公众舆论。报告还说，中国政府还下令网管删除那些与政府观点不一致的评论。

报告说，泄露的文件表明，中共花费了大量人力和资金用于言论管控。

ProPublica和《纽约时报》援引加州大学伯克利分校信息学院的研究员肖强的话说，中国的宣传和审查工作是世界上任何其他国家都无法比拟的。他对媒体说：“这是一件大事。” “没有其他国家能做到这一点。”

需要注意的是，这些被公布的文件仅仅只是冰山一角。所有在中国生活过的人都知道中共对言论的监控是无所不在的。发表任何对中共的负面言论都可能受到迫害。但中共在这场席卷全球的大流行病中扮演的绝不仅仅是隐瞒疫情的角色。这是他们13579计划的一部分，即17年开始实施的“1种病毒，3年内释放，保证此种武器5年内有效，彻底瘫痪7个国家，9个月内出疫苗”。这不是阴谋论，而是来自中共内部最核心层的情报。
