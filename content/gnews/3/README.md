## 【国内快讯】南极人“造假”，股价遇闪崩
` 国内新闻组` [轉載自GNews](https://gnews.org/zh-hans/758354/)

喜马拉雅-国内新闻组/素材：鹰(文言) 校对：圣斗士(沙加)

澎湃网1月16日转载中国新闻网报道，自2021年以来南极人深陷“财务造假”风波，股价一路下跌，较2020年7月10日24.41元的最高股价缩水一半，截至1月15日收盘，仅10.48元/股。

南极人成立于1997年，是中国最早的内衣企业之一，采用产销一体化经营模式。但在08年金融危机后，将生产和销售外包，开启品牌授权模式。凭借这一“吊牌”模式，南极人迅速发展壮大，产品涉及内衣、母婴、服装、床品、布艺、厨品、鞋品、服饰配件、户外、家电等9大品类的55大类目产品线。截至19年底，南极人合作供应商1113家，合作经销商4513家，授权店铺达5800家。

2015年南极人上市深交所，更名为南极电商，将贴牌生产和电商服务结合，授权供应商、经销商品牌生产经营权，毛利率急剧扩大，常年毛利达92%以上。2016年南极电商成交总额72.06亿元（营业成本为6655万元），到2019年成交额增至305.59亿元（营业成本达24.03亿元）。2017年11月，收购时间互联，涉足移动端广告营销业务，但受制于时间互联业务模式“增收不增利”（需要大量现金购买流量，且有大量应收账款），南极人现金流开始恶化。

2018年因时间互联导致南极电商预付账款从几近于0变为5.53亿元，2019年应收账款和票据增至8.63亿元，而18、19年公司的经营性现金流净额为5.51亿、12.55亿元，在2020年三季度末应收账款和票据更是高达12.84亿元， 同比增加28%。

由于南极电商品牌授权的经营模式，不涉及生产和采购，所以不可避免的出现质量监管不力问题。2018年南极人品牌的蚕丝被、内衣、童装、卷发器、按摩棒等产品被国家质监部门和地方消协列入不合格黑名单14次；2019年因产品缺陷召回取暖器；2020年7月，彩棉和服三件套、婴幼儿内衣产品被北京市场监督局通报不合格；消费者投诉平台“黑猫投诉”上，南极人的投诉量高达上百条，且南极电商涉及侵害商标权纠纷、著作权权属、侵权纠纷、肖像权纠纷等众多案件，截至21年1月8日，公司开庭公告55起，裁判文书96条。

除了经营模式出现重大产品质量隐患外，南极电商的“吊牌”模式更被业内质疑“财务造假”。

1）净利润过高：南极电商毛利润常年92%以上，2019年营收为39.07亿元，净利润为12.06亿元，毛利率高达93.36%（其中贴牌业务收入13.06亿元，占总收入的33.4%，但毛利润贡献却高达81.15%），而同期茅台毛利91.37%，要知道茅台可是习家的根儿红苗儿正，但净利润上却比不过一个私营的南极人。

2）前五大客户与前五大供应商高度重叠，多家南极人重量级店铺不停更换经营者，或者店铺经营者注册地址和名称不断变更，供货商和经销商之间有一场资金往来和人事变动。这一典型的“体外循环造假”特征却被简单辩解为“前店后厂”和生产经营商均为公司客户。

3）经营规模翻倍，但员工数量反而下降：2019年南极电商在各电商渠道的成交金额增速达49%，而实际各大电商平台增速放缓，阿里巴巴同期增速仅为19%，而且相比于过半的增速，南极电商本部员工数量却从18年的401变为393。

除了饱受市场和舆论质疑外，南极电商更被证监会重点监控，并被要求审计财务报告。

南极人从迅速发家再到短时间“脆断”，除了其经营模式的弊端外，还有因刷单被爆后的信任丧失，一下子从明星品牌堕为“假冒伪劣”，但归结起来源头还在于中共控制下的“计划市场”，完全以国企、央企霸占市场重要资源，并逐步将更具竞争力的私企蚕食殆尽。再者就是中共以假治国，无论各行各业都充斥着假、骗、偷，在中共体制内唯假盛行，最终只能自咽“造假”苦果，难逃被吞厄运。
