## 【香港要闻】港台7投诉其一为转播央视节目 通讯局疑双重标准不接纳《另一个香港》投诉
` 喜馬拉雅大使館-粵語組` [轉載自GNews](https://gnews.org/zh-hans/894429/)

搜集\编撰：爱听歌的猫

**特首林郑月娥上周四（4日）出席立法会答问大会时，批评港台作为公营电台，在不足两年间被通讯事务管理局裁定7宗投诉成立，形容是难以接受。众新闻翻查资料，发现其中一宗投诉，实为港台在前年9月转播中央电视台的《天气预报》，而央视的天气预报节目中含酒精广告，但投诉港台仍被指成立。众新闻再与早前收2000宗投诉的央视节目《另一个香港》比较，发现通讯局同指港台对两个节目没有控制权，但裁定《天气预报》投诉成立，而《另一个香港》不成立，有双重标准之嫌。通讯局回应指两宗投诉个案完全不同，不宜比较。**
![]()![](https://gnews.org/wp-content/uploads/2021/02/jpg-image001-8.jpg)
根据通讯事务管理局的调查报告，涉事的《天气预报》在2019年9月30日晚上在港台电视33及33A台播放，为港台转播的中央电视台节目。通讯局接获一宗有关投诉，指节目在「合家欣赏时间」（每日下午4时至晚上8时30分）内播放某酒精品牌的植入式广告。通讯局裁定投诉成立，港台违反《电视广告守则》，但基于港台转播节目时对节目内容没有决定权，以及属于转播安排下的个别事件，决定投诉成立但不向港台施加惩处。
![]()![](https://gnews.org/wp-content/uploads/2021/02/jpg-image003-6.jpg)
但翻查资料，去年中央视制作的反修例运动纪录片《另一个香港》在港台和多间本地电视台播放，收到近2000宗投诉，指节目立场偏颇、与事实不符、受批评者未获机会回应等。通讯局最终裁定投诉不成立，指「港台作为公营广播机构，有责任完整地即时转播央视综合频道的节目」，又指港台对节目内容没有控制权。通讯局在报告中更指，广播机构要确保非本地制作的节目内容完全符合香港的规管要求，「并不合理亦不切实际」。

两宗投诉同为港台转播央视节目，但裁决结果回异。众新闻比较两宗投诉，发现通讯局在投诉报告中的部分审议内容相近。
![]()![](https://gnews.org/wp-content/uploads/2021/02/image005.jpg)
**战友点评：**

众新闻向通讯局查询，为何在处理两个性质类似的个案时，会出现不同的标准，以及港台到底是否需要为非本地制作的节目负责。通讯局回应指，两宗投诉个案涉及的节目类别、性质、内容、观众对象、投诉人对两个节目的指控，及适用的规定完全不同，不宜作比较。试问，缺乏共正立场的通讯局，如何作出公正的裁判？

有传媒认为，港府先打压报纸老板，再打压「不听话」的电视台，由2003年71大游行至今，中共的手法就是消灭所有香港的异见声音。香港电台早就自我审查，连批判政府也早已不痛不痒，只不过在太过份的事情上「打擦边球」，连新闻报导也往往「跟党走」，大量转载中共的新闻稿，以至几乎没有什么批判就搬字过纸；然而这里是香港，不是中国大陆，如今港共连这样的内容也不能容忍，正说明香港言论自由的倒退，新闻自由的完全沦落，什么「五十年不变」，当然早已抛诸脑后。

林郑出席立法会答问大会时批评港台作为公营电台，表示「以我个人立场而言，港台是极需要改善。」林郑指港台不足2 年间就被通讯局裁定七宗投诉成立，包括1 宗严重警告，3 宗警告，情况「难以接受」，形容要以坚定不移的态度，就问题「拨乱反正」。林郑并喊话说「（港台）约章里面，有一个人的角色叫总编辑，我期望这位总编辑（梁家荣）对住（被投诉）报道事件有一个立场，但我还等紧。」林郑这种带有恐吓性的喊话，政治意味极重，意味着港府即将对港台有大动作。
