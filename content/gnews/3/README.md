```
澳大利亚总理斯科特·莫里森(Scott Morrison)周三表示，澳大利亚将斥资7.47亿澳元（5.8亿美元）升级北部的四个军事基地,并扩大与美国的战争竞技演习。
```


在与中国的外交和贸易争端愈演愈烈之际，莫里森表示，澳大利亚必须扩大其在北部疆域的军事资产，以应对亚太地区未明朗的紧张局势。

莫里森在达尔文对记者说：“我们的目标是建立一个自由开放的印度太平洋地区，以确保此地区和平，同时,澳大利亚能始终保护其自身利益。”

尽管莫里森周三时回避提及中共国，但近年来中澳两国在印太地区的影响力竞争日益激烈，澳大利亚对该地区的军事聚焦随之提升。

在去年堪培拉要求对冠状病毒的起源进行国际调查之后，澳大利亚和中共国之间的关系进一步恶化，这促使北京进行贸易报复。

莫里森说，将延长北部疆域的飞机跑道，以支持更大型飞机运行，对射程进行大修，并为国防人员和美国海军陆战队建立新的训练设施。

军事升级将从今年开始，到2026年完成。

这些资金是澳大利亚国防计划的一部分，它意味着堪培拉在未来十年内将花费2700亿澳元，以提高堪培拉的远程打击能力。

莫里森说，因为亚太地区正经历第二次世界大战以来最大的经济和战略不确定性，去年就有了此项资金需求。

澳大利亚日益自信的做法赢得了华盛顿的青睐。莫里森表示，新设施将加强与美国的联合军事演习。

作为年度联合训练活动的一部分，已有2,000多名美国海军陆战队员抵达澳大利亚北部。

澳大利亚和美国也举行两年一度的战争演习，下一场定于八月开始。

通常，超过30,000名士兵参加澳大利亚东海岸的演习。

美国驻堪培拉大使馆代办迈克尔·高德曼（Michael Goldman）说：“美国和澳大利亚在国防合作方面已深入开展了半个多世纪 “我们将继续寻求更多与盟友澳大利亚合作的方式，以促进美国人，澳大利亚人和印太地区人民的安全与繁荣。”

几天前，澳大利亚最高级的安全官员之一声称，自由民主国家必须为战争做好准备。随后澳大利亚宣布了最新的军事升级计划。

民政事务部部长迈克·佩祖罗（Mike Pezzullo）并未明示他的预警缘由，但此举是在澳大利亚与中共国的关系急剧恶化以及台湾地区紧张局势加剧之后提出的。

澳大利亚国防部长皮特·达顿（Peter Dutton）周日还表示，中共国与台湾之间的冲突“不应轻描淡写”。
