**西班牙2021年1月3日电/西喜社—**

2021年元旦，在GTV 《我是音雄》跨年晚会现场，郭文贵先生与西班牙巴塞罗纳喜悦农场的战友家庭代表亲切连线交流。

郭先生强调了西班牙对G系列发展的重要作用，对于西班牙巴塞罗纳喜悦农场发挥在G系列、G生态的建设，以及凝聚团结西班牙华人，提升华人在西班牙的影响力和社会地位的作用提出了殷切的期望和要求。

郭先生表示和西班牙战友交谈非常开心，他说：“西班牙是我在欧洲最喜欢的国家……饭好吃啊……非常漂亮啊，自然很美啊……有很多朋友，现在郝海东先生，叶钊颖妹妹都在那儿，这两个是我们的大英雄，这两人选择是非常正确的，我也有很好的长期的生意合作伙伴。某个沼泽地的大佬就是长期定居在西班牙的。”

郭先生提到：“西班牙是个美丽的地方，但是西班牙的华人不是这样子的啊。西班牙的华人在那边没有一个相称的地位。西班牙对我们华人形象并没有利好。而且我们给西班牙的贡献巨大……西班牙绝对需要咱们华人的团结和凝聚。而且有巨大的潜力和机会。所有西班牙的社会毛病和缺点，我们华人都能给弥补上。” “就咱们华人在西班牙真没团结起来，没表现出应有的实力。”

郭先生重点谈到了G系列的发展。他说：“未来在欧洲G系列西班牙至关重要。你比如说，现在咱们继续在这个G-Coin、G-Dollar虚拟货币。”

郭先生说，今天下午就和某些国家开了两个会，买这个系统的，未来你会知道吧,其中有一个就是西班牙。“我们G-Coin、G-Dollar他们做过深入的研究。我们这个币可能在西班牙让他们填补在虚拟货币、数字货币上的一个最大的空白，我们G系列有巨大的空间。”

西班牙巴塞罗纳喜悦农场的成员法图麦向郭文贵先生介绍了自己家庭的组成—丈夫神奇四侠比自己小12岁，有个女儿，一家三口都加入了爆料革命。还有个特殊的家庭成员“干儿子”2pac,也是农场的战友。

法图麦说：“孩子一宿没睡觉，旁边还有外国的女朋友在陪伴，我们坚定地加入了爆料革命，我们家是从我做起，七哥的声音就真的是直击我的心灵。我义无反顾的加入爆料革命、新中国联邦。我又感染了我的老公。我的女儿也加入了，现在我又多了儿子。希望七哥有时间能来西班牙。”

2pac说：“我强烈要求直播……因为您每天都是我的精神双修对象。新的一年，希望七哥遇难呈祥，化危机为转机……灭共不管是今年还是明年、后年我们都支持您坚持到最后……我人生最大的财富是跟随七哥三年，七哥教我的很多做人的良知、做人道理、国际政治还有如何对待家庭对待你的同胞，对待地球、对待动物……我们也会去领只宠物，也是跟七哥家的笨笨一样……之前的话，我跟我父母关系不是很好。但是听了七哥的话以后，我要尽我自己的本分，过年送给我妈妈礼物，尽量把关系缓和，我代表在西班牙的所有华人向七哥说声谢谢，虽然有些华人到目前还执迷不悟，还不是很了解，但是我想他们明年会意识到些，希望跟七哥在西班牙早日相见。”

郭先生表示：“西班牙七哥一定会去的,而G系列一定会扎根的。而且西班牙在未来的G系列中，他有着独特的地位，有些话现在时间短，我就不多说了，未来我可以专门给你们连线咱们好好聊聊。”

连线的最后，郭文贵先生说：“G系列在西班牙有生产工厂。西班牙和土耳其现在已经是欧洲大的制造地，我们有很多很多机会，谢谢唐平妹妹、威廉王今天让我们战友们有这机会，和我们的路波切、老班长、长岛哥和我们大家一起在这见面。未来的机会很多，我们的路还很长，一定干倒CCP这是我们最核心的目标。”
