## 【鹰眼墙国】盲人高考单独命题，难掩盲人就业、教育“稀少”现状
` 纽约香草山MOS03` [轉載自GNews](https://gnews.org/zh-hans/1306520/)

香草山写作组 鹰(文言)



![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/01042102/WhatsApp_Image_2020-10-24_at_04.33.05-8-768x103-1.jpeg)



环球网6月7日转载新华社消息，2021年高考，中共为11名盲人考生单独命题，涉及省份是历史上最多，盲人参加高考的人数也是史上最多。

2014年，中共教育部首次印制高考盲文试卷。虽然为盲人高考提供了政策支持和便利，但盲人考生人数依旧极少：2015-2020年盲文试卷数依次为8、7、2、10、5份。今年盲人考生虽最多，达11个，但相比于1731万盲人基数而言，“盲人高考”几乎可忽略不计。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/07223003/Screen-Shot-2021-06-07-at-10.27.16-PM.png)
2019年，国内特殊教育学校数量增至2192所，其中盲人学校仅为27所。虽然2020年特殊教育学校略有增长，但盲人学校数量的巨大缺口仍难以填补，盲人教育依旧停留在口头宣传上。参考《视障者基本信息调查》国内庞大的盲人群体，其中30%基本不外出，而不需要家人朋友陪同可外出的比例仅有24%，约415.4万人。设计缺陷、乱停、乱放、乱占、敷衍了事的“中共式盲道”虽然有全球长度最长、分布最广的“成绩”，但在实际应用上，却深受诟病。各行业无障碍设施普及率仅有40%，高价低产能的导盲犬更是急缺（2020年实际服役导盲犬数量约100只），盲人群体的出行难题依旧无解。

根据信息无障碍联盟发布的2016年《中国互联网视障用户基本情况报告》，63%的视障者从事推拿按摩工作。国内所有的盲校职业高中部几乎只有一个按摩专业，各地残联开办的盲人培训班也几乎只培训按摩。盲人按摩似乎成为盲人群体的“唯一出路”。然而2016至2020年，盲人保健按摩机构数量和医疗按摩机构均逐年下降。2020年全国培养盲人保健按摩员12761名，盲人医疗按摩员7820名，保健按摩机构17313个，医疗按摩机构873个。相比1700万的群体而言，已就业人数和规模仍显“杯水车薪”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/07222947/Screen-Shot-2021-06-07-at-10.27.21-PM.png)
盲人高考从表面上看，似乎中共对盲人重视程度提高，还强调“高校对残障学生投档不可拒收”，但由于国内各大院校缺少相配套的盲文教材和无障碍设施，更缺乏系统性标准，盲人高考依旧只是“过眼烟云”。截至目前，面向全国视障学生开展单考单招的本科院校只有滨州医学院、长春大学特教学院、南京特教学院、北京联合大学特教学院，专业主要为针灸推拿、康复治疗和音乐学。再参照上文提及的约2万人按摩员现状，盲人受教育和就业形势依旧不容乐观。
