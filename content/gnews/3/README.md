## 美國必須和中共國脫鉤
`2020-05-29 14:49 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/216765/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/29144003/%E7%BE%8E%E5%9B%BD%E5%BF%85%E9%A1%BB%E5%92%8C%E4%B8%AD%E5%85%B1%E5%9B%BD%E8%84%B1%E9%92%A9.jpg)
《華盛頓觀察家報》發表參議員約翰·巴拉索的觀點文章，約翰·巴拉索，來自懷俄明州的共和党參議員，整形外科醫師，也是參議院共和黨會議主席。

巴拉索先生認為，當前正處於冠狀病毒的大疫情之中，美國應該從這次球大流行中學到的顯而易見的教訓之一是絕不能依賴中共國，並且美國應該與中共國脫鉤，將包括醫療物資在內的產業鏈移回美國。
中國共產黨一開始就試圖隱瞞疫情、掩蓋真相。並且在冠狀病毒襲擊美國之前，中共國就已經囤積了美國迫切需要的防護物資。中共政府甚至把美國科學家拒之門外，從而沒法幫助世界對疫情做出反應。因而美國在爆發之際，缺乏緊缺的醫療物資。由於製造產業鏈在美國之外，醫療產品供應鏈在應對這次疫情上嚴重不足。
川普政府已經採取一系列措施以確保美國醫護人員獲得足夠的醫療物資，例如使用《國防生產法》來刺激國內製造業和加快生產急缺物資。但這還不夠。比如中共國目前佔據世界一半的口罩生產份額，美國95％的布洛芬、70％的對乙酰氨基酚和40％的盤尼西林都來自中共國。
他認為此時參議院兩黨人士已經達成共識，改變現狀迫不及待。不僅要讓美國公司能在本土生產更多的醫療用品，還要私營企業和政府部門無縫合作以提高疫情應對能力，甚至還要保證充足的物資儲備，以備不時之需。與中共國脫鉤、搬回產業鏈勢在必行。
