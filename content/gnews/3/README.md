1.2020年3月12日

***世界上最快樂的事情是賺錢***

兄弟姐妹們，**你知道世界上最快樂的事是什麼嗎？**最快樂的絕對不是花錢，凡是覺得花錢是最快樂的，他絕對是窩囊廢。**最快樂的是掙錢！**掙錢那個過程！大家知道我從十幾歲，從小，我從六七歲就跟我娘說，我長大了要種川貝母去，川貝母藥材。掙錢給我娘花，讓我娘有錢買醬油買鹽吃。後來大家知道我一系列投機倒把，倒賣摩托車、汽車、牛仔褲、電子錶，倒賣袁大頭、賣糖葫蘆、開包子鋪、倒賣鴿子。我這生意我還沒數呢，一數你看啊…從小第一筆生意，賣糖葫蘆，賣糖葫蘆，販賣糖葫蘆，（鏡頭有點虛幻）糖葫蘆，川貝母沒種成。然後呢是到廣東倒賣電子錶、牛仔褲。莘縣開包子鋪。東北，還有到廣東倒賣摩托車、倒賣汽車，這都幹過。倒賣過火車皮，賣過節油器，當時賣節油器，從看守所出來賣節油器。倒賣過木頭，倒賣過鋼材，拉煤的火車，這都幹過。當然了你們也知道，後來蓋裕達房地產、裕達別墅、裕達花園、裕達國貿。然後參與了N個（項目），廈門的、海南的酒店投資。雖然我不是大股東，但是主要是我來策劃，上海、深圳、鄭州的西區。**然後到北京，最牛的房地產，做倆，證泉、盤古，創建了一個五星級飯店，一個七星級飯店，最大的綜合版塊。中國房地產家裡邊送天然氣的，郭文貴第一人；叫房地產開發****“****綜合版塊****”****，郭文貴第一人；中國個人創建五星級飯店，郭文貴第一人；創建七星級飯店，郭文貴第一人；中國最貴的公寓，是郭文貴創造的；最貴的酒店，郭文貴創造的；最好的中餐廳，郭文貴創造的；最好的日餐廳，郭文貴創造的。“綜合版塊”這個詞兒就是從郭文貴這兒開始的。房地產的****REIT****，叫做房地產信託，第一個創始人，郭文貴。**

***資產“氣化”***

從看守所出來，1991年，負資產，迅速聚集三億五千萬美元，每次在美國開庭都會問：“你從看守所出來，一年你就整了三億多美元投資？”我說：“是啊，哈哈！”那就是真的嘛，是吧！然後大家看到，我把房地產做成了金融，我把金融做成了股票，股票做成了…叫做“嘣”，“氣化”，“氣化”，這就是郭文貴。我當時在1993年，我說誰要把房地產做成房地產，你就是給房地產打工，要不你就是詐騙犯，房地產做成金融你是最高的，把金融你再給它現金化，你最牛的。所有我定的戰略，**第一個，要讓我成長財富的，我必須要有個大盤子，**大盤子你幹啥去呀？我沒講過。共產黨那幾樣，石油不讓你搞、金融不讓你搞、軍火不讓你搞、能源不讓你搞，你還搞啥？特別是那個年代，只能搞房地產了，還得小著搞。所以從房地產起步，大起步，**我搞房地產的時候，把“地”變成“地產”，我把地上蓋上房子叫“房地產”，我要把房地產，我說我不指望賣回來掙多少錢，我要把房地產變成“金融股票”，我變成金融股票我不要兌現，我把金融股票我要把它變成現金化，現金化的錢要變成國際化，而且錢還就在這兒，這就是郭文貴**。這就是把地變成房產，房產變成金融化，金融變成股票化、市場化，股票市場化變成國際化，**而且錢還在這上頭，一分錢不拿走**，哎！這就是文貴做到的。

***基金投資規則***

**基金都要求你一個項目不能投多少、不能超過多少比例，或者某個國家限制你不能超過多少，某個領域不能超過多少。這是基金的規律**。就象戰友們，你們拿錢給我文貴啦，給我文貴時基金有規定的。你給我拿了錢，我收你2.5%—5%的管理費，掙和賠我都收你去2.5%—5%管理費每年。那麼掙了錢我分你20%或者10%，這可以講數的，一般是10%—20%，掙了分你20%，賠了跟我沒關係，都是你的事。

但是這個錢給了我以後呢，不能投你郭文貴家裡多少錢，也不能投某個國家地區超過百分之二十幾，你不能投的什麼領域，比如說酒店領域啊，還有共產黨的喪葬改革啊，火葬場不能投這比例，都可以要求的。

世界權力格局徹底改變，世界財富重新分配，世界各種力量重新崛起。新的力量，各種新的力量將崛起。**戰友們只要你們撐得住、摟得住、幹的好，你們就是新崛起的力量。人類上前所未有的機會，什麼老家族、老錢、老勢力統統嗝！**

這是該草根兒、所有的窮人……戰友們，你們越是沒有錢的人你們越有機會，因為有錢的人這一次全打垮了，有權利的、有名的、有錢的、全部打垮。就那些好萊塢的，還有什麼明星啊，住著豪宅啊、開著名車啊、私人飛機啊，成本啊——養的住嗎？都吃草去吧！都完了！
