美国《国会山报》(The Hill)于2021年3月16日发表一篇由知名美国时政评论员章家敦（Gordon Chang））撰写的文章，直指在即将在阿拉斯加州安克雷奇市（Alaskan Anchorage）召开的中美高级别官员会议上，中共不会有任何有意义的讨论，而是会训诫国务卿安东尼·布林肯（Antony Blinken）和国家安全顾问杰克·沙利文（Jake Sullivan）。

文章报道，美国一位 “前国防官员”说，中共只对美方提出了改变的期望，而他们本身却不提任何改变。在过去，中共官员在类似的场合会向美国做出让步因为他们认为美国很强大。但是那些日子已经一去不复返了。中共党魁习近平认为，现在中美势力平衡很快就会倾向他一方，其宣传机器也直言“东方在崛起，西方在衰落。” 这种宣传口号符合他的政治利益，特别是支持了他领导国家取得历史性成功的说法。共产党的外交政策一直与国内的政治阴谋紧密地联系在一起，尤其是在如今中国稳固派掌权的情况下。对美国作出重大让步将破坏国力，并威胁到习近平的权力。

此外，美国人并不理解共产党对其的固有敌意，这在2019年中共针对美国的“人民战争”宣言中就可以察觉。目前情况变得更加复杂的是，中共政权已经既不尊重也不惧怕美国。

文章批评拜登总统自上台以来，已经颁布了一系列的行政命令和其它行动对中共作出了单方面让步，例如他取消了对中国电气设备的禁令，推迟了对中国军方企业的制裁，重新加入了以中共为主导的世界卫生组织（WTO），并延迟对中共应用程序的禁令。毫不奇怪，中共认为他们不必为拜登做出让步，就可以得到他们想要的东西。

美国不能向一个国家让步，尤其是这个政权犯有种族灭绝和危害人类罪，企图分裂其它国家，运用Dominion系统控制全球。中共政权还在进行冒险的实验，以创建一个新的中国优等种族。无论从哪方面看，中共一直在研究只让外国人生病的病原体。习近平允许冠状病毒越出国境。中共国每年盗窃大约5000亿美元的美国知识产权。

文章指出，在总体问题上美国没有妥协的余地。美国扮演着“现有主权国家竞争的西方国际体系”的捍卫者，而中共正试图用统治世界以取而代之。王毅2017年在中央党校《研究时报》上一篇有颇具影响力的文章中，鼓吹废除国家主权，这是意图使世界人民成为中国臣民的暗示。因此目前中美对话根本无法解决这一核心分歧。

文章分析了在这种中美关系背景下，为什么布林肯和沙利文还要专门去阿拉斯加。其原因是几乎每个人都希望中美两国能够对话，而拜登团队无疑也不想装作固执。正如兰德公司高级政治学家斯科特·哈罗德（Scott Harold）所说的那样，东京，堪培拉，台北和德里都希望美国致力于“在该地区发挥领导作用”，但不希望与中共国竞争。唯一相关性框架是，参与国希望从经济发展到避免武装冲突等问题上，至少战术上地与中共国合作。
