## 【快讯】汇源退市，民族品牌终究不比“家族企业”
` 国内新闻组` [轉載自GNews](https://gnews.org/zh-hans/762333/)

喜马拉雅-国内新闻组：鹰(文言) 校对：加文gavin

澎湃网1月17日报道，联交所宣布于18日上午9点正式取消汇源果汁的上市地位，“国民品牌”黯然收场。

汇源创始人朱新礼于1994年创立北京汇源食品饮料有限公司，在97年斥资7000万拿下中共新闻联播5秒广告时间后，“有汇源才叫过年”，汇源果汁一举成为“国民饮料”。随后汇源乘风直上，在98年全国建厂，到2000年占据市场份额23%，成为行业龙头。2007年2月上市香港后，筹资24亿港元，并继续在山东乐陵、吉林舒兰、辽宁锦州等地布局了9家工厂，大有高飞猛进之势。

然而2008年的一场收购风波却将汇源打落马下：2008年9月3日可口可乐及下属AltanticIndustries公司宣布以179.2亿元港币收购汇源股本和债券，受此消息影响，汇源股票暴涨164.25%。但相比资本市场的活跃，国内7万人参与的民调显示82%网民反对此收购，舆论导向再加上中共商务部以不利于市场竞争为由于09年3月否决该项收购，该项否决直接导致随后3天汇源股价的连续暴跌近58%，市值腰斩。

自收购案“折戟”后，汇源开始江河日下。07年汇源果汁上市后收入达27.07亿元（净利润6.4亿），08年总收入虽增至28.49亿元，净利润却暴跌86%；2011年至16年每年扣非净利润均为负值（依次为-2.3亿、-3.18亿、-4.79亿、-5.75亿、-5.53亿、-2.08亿元）；与业绩下滑相对应的是高管的频繁更换，6年时间5名CEO更迭。

17年8月至18年3月，汇源果汁违规向北京汇源饮料提供42.75亿元短期贷款，不但违背了港交所的对外披露规定，更未获得董事会批准；4月初汇源果汁被停牌，港交所要求其彻查违规贷款并采取补救措施，否则退市。除遭遇停牌危机外，汇源的负债也在逐年高筑，17年底总负债金额达114.03亿元。

2019年4月，汇源果汁宣布与天地壹号成立合资公司以拓展市场，但最终失败收场；5月朱新礼被法院列为失信被执行人；9月汇源果汁被曝出债务危机，旗下4假公司借款达400万元，且均已预期未偿还；12月朱新礼代理的中国德源资本（香港）有限公司被查封，41亿元资产遭冻结。2020年2月创始人朱新礼父女退出上市公司董事会，朱新礼和他的“果汁王国”正式图做他人嫁衣。

收购风波不仅破灭了汇源跻身国际市场的机会，更是直接引爆仓促上市募资带来的“增收不增利”问题；为引领良性竞争的“否决”以及“国民品牌”的民族感终究只是昙花一现，非但没像被制裁的华为那样为汇源赢得更多的支持，反而净利润暴跌，其后更是一蹶不振；民族品牌不仅没有受到海航和华为等家族企业的“另眼相待”，反被停牌“整顿”。即汇源果汁为开拓市场筹集资本选择的上市，终究没有为其赢来更多有效的资本注入，反而债务高筑，最终创始人也被剥离，难逃蒙牛、伊利、京东等被划归“国有”噩运。

汇源的发家史和退市正是中共体制下民企私企的缩影，当“风生水起”之时就会被各种利益捆绑，无奈上市后又被国家机器和可悲的“民族感”所左右。“计划经济”治理模式下的中共，加上各家族势力的分割和掠夺，再具竞争力的民企私企也难逃被抢夺的命运。明眼人如牛根生、刘强东那样的，选择主动交财以保命，而不明就里的叶简明、王健等最终财命两空。中共残暴掠夺的本质决定了任何人与其经济往来都是与虎谋皮，以假治国、以黑治国的手段更是彰显着其流氓嘴脸。中共是篡夺政权的犯罪组织，对待这样的极恶势力，唯有国人全面觉醒起身反抗，才能彻底根除这一共产主义毒瘤。
