## 陳彥霖浮屍案24日開庭, 15歲女童出身破碎家庭身世悲慘
`2020-08-24 06:55 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/314705/)

香港時間8月24日早上9時30分，香港西九龍裁判法院召開陳彥霖浮屍案死因庭。法庭選出2男3女共5名陪審員，首名出庭作證的是陳彥霖的「失踪生母」何姵誼女士。

庭審中，15歲陳彥霖悲慘的身世逐漸曝光於世人眼前：2003年，陳彥霖在屯門醫院出世，其母未婚產下她。彥霖3歲時，其生父吸毒及家暴，導致她們母女搬到彥霖外公家生活。其母后來扔下彥霖，彥霖自幼由外公撫養長大，兩母女大約每週見面一次。

彥霖上初中一年級時，由於長期缺失家庭溫暖，多次離家失踪，有幾次被判入住女童院。去年8月某天，彥霖前往塘福懲教所探望男朋友，與男朋友的父親一同乘的士到東涌，惟她不願意下車，著司機駛回懲教署。由於她沒錢付車資，最終的士司機報警。期間彥霖踢了接報到場的女警一腳，被警員拘捕並押返梅窩警署。

陳母到達警署後，形容彥霖處於極差的精神狀態。她說女兒一反常態，對她異常冷淡。警方要求陳母為女兒保釋，但陳母拒絕，因她完全不能控制彥霖，陳母要求警員送女兒入院。

去年9月12日，彥霖因襲警案上庭，並獲得保釋回家，豈料一周後她變成了「浮屍」。

其母表示，去年9月19日她正在韓國旅遊期間，收到港鐵來電稱撿到彥霖的物品，要求她取回。她稱同日開始已無法聯絡女兒；20日早上，她收到彥霖同學的手機訊息，指彥霖當日缺課，又稱前一天與彥霖乘坐港鐵到美孚站時分開，其母向警方表示，這是最後一次有人目擊彥霖的踪影。

其母稱，她從韓國旅行回港後，在21日報警指女兒失踪，26日警方重案組通知她彥霖已經死亡，9月30日，其母獲認領女兒遺體於10月10日火化。

「天底下哪有這樣的母親？女兒不明不白變成了一具赤祼浮屍，做母親的難到沒有一絲懷疑？竟然勸關心的大眾息事寧人。」部分專程去法庭旁聽的市民不滿彥霖生母在庭上「數臭」女兒。彥霖母親今天在庭上不時哭泣，說女兒性格叛逆，精神狀態不濟，彷彿一切都是女兒的錯。

**然而，很多香港人在網上對陳彥霖的生母身份表示懷疑，陳彥霖的親生母親是否還活著？由於其舅父等相關家人、朋友全部失踪，其生母身份和法庭上所述根本無法得到其他人證確認。**
