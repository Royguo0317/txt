## 失踪儿童规模成谜，“团圆系统”找回率达98.1%？
` 喜联盟编辑部 Himalaya Alliance Editorial Team` [轉載自GNews](https://gnews.org/zh-hans/978423/)

**作者：美国纽约香草山农场  鹰(文言)**

环球网3月15日转载中国新闻网消息，中共国公安部于2016年5月建成儿童失踪信息平台“团圆系统”，截至21年3月，共发布4722条儿童失踪信息，找回率高达98.1%（约4628人）。

根据凤凰网2017年5月的报道《社会能见度：十年寻子路》显示，国内每年有20多万儿童失踪，当前仍未放弃寻找的案件超60万件。《中国每年失踪儿童数据惊人，千万别让孩子找不到回家的路！》报道20万失踪儿童中找回率约0.1%，即200个。澳大利亚媒体报道中共国每年约7万儿童遭拐卖，已成世界最大拐卖儿童市场，参照中共官宣的98.1%，5年时间找回约4628人（每年925名儿童），这一找回的数量远超过前述的国内媒体提及的200人/年。
![pastedGraphic.png]()![pastedGraphic.png](https://gnews.org/wp-content/uploads/2021/03/f38698765f8447999b127640e85f6838-1.jpeg)
在每年失踪20万儿童引发热议后，中共媒体大肆炒作“谣言论”以抵消这一事实真相的残酷程度。2019年5月26日中共前打拐办主任、刑侦局副局长更通过微博称“纯属谣言，造谣者一律追责”，为此作官方定性。然而被称作“中国失踪儿童晴雨表”的“宝贝回家”（创立于07年）截至到2019年5月18日就已有38457条寻找失踪儿童信息，平均每年3204名儿童失踪，这一数字远超公安部提供数据显示的每年失踪925名儿童。而找回率更是完全捏造，参考“宝贝回家”截至18年3月帮助2300多个家庭团聚，以38457为基数粗算，找回率也不过5.9%，远远达不到所谓的九成找回。

仅从“宝贝回家”这一民间网站数据就能揭穿中共的失踪儿童“成果”的虚假和“团圆系统”浮夸。最可恨的是中共对真实现状的隐瞒，如果说寻找失踪儿童需要投入巨大精力人力，政府“难”有作为，而将真实数据“官宣”成谣言这一举动直接暴露中共政府的险恶用心。按照中共的思维逻辑，既然真实现状让人堪忧，易引发社会对政府不作为的质疑和抵制，那就“辟谣”（如2017年9月中共将报道武汉30多名大学生失踪的撰写人以散步谣言扰乱公共秩序罪拘留10日），把失踪儿童数量一降再降，这样造假的话，失踪基数大大减少，找回率创新高。但假的毕竟是假的，造假终究会被揭穿，无论是墙内的与官宣有异的数据（如“宝贝回家”网站信息），还是境外报道“冰山一角”的中共国失踪儿童现状都为世人揭示了中共造假的低劣。

自习上台执政后，不但利用盗取的国外科技，“充分”发挥监控摄像头的作用（2020年10月，国内有超过3.5亿支监控摄像头）实施全民监管，更建立新疆集中营进行种族灭绝。近年来又通过建立各种社保数据共享、银行卡关联，再有2020年推行的集身份、手机等私人信息于一体的健康码，中共利用5G和AI智能对墙内民众进行令人发指的网格化监管。既然有如此密集的监控，再加上如此“完备”的大数据筛查，但为何没能用在寻找失踪儿童上呢？

外国友人掉钱包都可以几小时内找回（据郭先生爆料，外国人入境后12分钟内可实施定点抓捕），上网翻墙发表言论后短时间内（几天到一个月不等）都可被叫去喝茶，那么为何找回父母最牵挂的失踪儿童时却如此效率低下呢？根本原因在于中共的独裁统治和以假治国、以黑治国，国家机器成为几个家族和个人敛财和奴役人民的手段，而非保护民众，所以只要无关其统治地位的，中共又怎会花大力气拨款和出力呢？人力物力财力都用在了维稳和军事扩张以及中饱私囊了，又怎会用来找回韭菜们的走失儿女呢？
