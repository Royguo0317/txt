## 【军事快讯】日本执政党对中（共）国新《海警法》允许在钓鱼岛海域使用武器作出反应

作者：纽约香草山写作组   Wilson



![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-2-1.jpg)



[日本共同社](https://china.kyodonews.net/news/2021/01/c0f3b22864fc.html)1月26日报道，日本自民党（执政党）当天召开国防小组与安全保障调查会的联席会议。围绕中（共）国正式通过允许海警局使用武器的《海警法》，与会人员相继出现担忧和批评意见，并敦促日本政府探讨健全法律，以应对中国海警局船只航行的尖阁诸岛(钓鱼岛）周边的不测事态。

与会议员发表“(中共)海警法条文系针对尖阁”、“无非就是威吓”等意见，还有人呼吁修改法律以便自卫队采取海上警卫行动。出席会议的防卫省干部则回应称“将整理(研究)课题”。

![]()![](https://gnews.org/wp-content/uploads/2021/01/28-20.jpg)

日本自民党是1955年（昭和30年）由自由党与日本民主党缔结成立的保守主义政党，是日本一党独大制度的核心，战后长期支配日本政治。自由民主党总裁长期担任日本内阁总理大臣的职务，因此有总裁即总理的说法。

日本媒体于上周五报道中(共)国通过新的《海警法》，引起日本民众的强烈关注，迅速成为当周最热报道排行第一。今天被公开报道的自民党联席会议，可视为日本执政党对朝野近日关注尖阁群岛(钓鱼岛)事态的一个回应。除了安抚国内情绪，按照日本人少说多做的特性，应对尖阁群岛(钓鱼岛)争议新的方案或法律将很快出台。
