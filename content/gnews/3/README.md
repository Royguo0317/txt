最近几个月以来，阿斯利康新冠疫苗经历了一系列的麻烦。好几个欧洲国家于3月15日星期一暂时停止使用该疫苗。

阿斯利康新冠疫苗很难得到全球的一致认可。它在3月15日星期一已被几个欧洲国家暂停使用。 这些国家担心接受过一剂疫苗注射的患者会出现副作用。 虽然世界卫生组织认为应该继续使用这种疫苗，但这对阿斯利康来说是遭受的又一次打击，它之前已经遇到了其他麻烦。

**交货问题**

在该疫苗的研发期间，阿斯利康制药公司于2020年9月9日经历了首个麻烦。由于一名自愿者出现了无法解释的疾病，它被迫中断了临床试验。 虽然最终重启了试验，但在11月，研究表明，该疫苗的有效性仅为70％，而其竞争对手辉瑞和莫德纳Moderna则为90％。 今年一月份，该公司遇到了交货问题。 在计划给欧盟的1.2亿剂中，阿斯利康仅能提供三分之一的量。
