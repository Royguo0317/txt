**西班牙2021年3月12日电/西喜社——**巴基斯坦规划和发展部长说，第三波中共病毒疫情正在进行中。

阿萨德-奥马尔说，在巴基斯坦的新病例中发现检测到中共病毒英国变种。官员说，最近几个月有数十人从英国回国。

自3月1日巴基斯坦恢复学校正常上课以来，中共病毒的死亡人数和确诊病例稳步上升。巴基斯坦上周五报告了54例死亡和2701例新病例。该国已登记600198例，确诊死亡13430例。
