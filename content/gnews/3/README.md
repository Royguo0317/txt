## 个税减免，东北地区能否迎来“再复兴”？
` 喜联盟编辑部 Himalaya Alliance Editorial Team` [轉載自GNews](https://gnews.org/zh-hans/958805/)

作者：香草山农场  鹰(文言)
![]()![](https://gnews.org/wp-content/uploads/2021/03/GNEWS-2021-03-08-coverphoto.png)
新浪网3月7日报道，辽宁副省长陈向群建议给予东北地区个人所得税减免50%优惠，以促进东北地区的经济复苏。

陈向群建议推动东北地区和京津冀协同发展，建立区域大市场，基建、科研和产业项目上倾向东北地区；在不改变个人所得税税制和税负前提下，给予东北地区50%个税减免政策。2016年中共发改委发布《推进东北地区等老工业基地振兴三年滚动实施方案（2016-2018年）》，3年时间向东北注资1.6万亿元。然而，实际情况是东北每年减少42.73万人口，社会消费品总额显著下滑，几大重要城市都是负增长，揭示着东北地区作为老工业基地由于资源的耗尽，已风光不再。东北地区本身工业占比重，国有经济过于庞大，而国企管理机制是逆淘汰、人员懒惰效率低下。中共国虽给予巨量的资金支持，但大多被各级领导克扣占用，造成“大锅饭”式的公有经济巨大亏空，想靠“拨款补锅”提振经济无济于事。
![]()![](https://gnews.org/wp-content/uploads/2021/03/GNEWS-2021-03-08-coverphoto-fig1.png)
外来资金投资的显著减少，以及人口外流都严重阻碍了东北经济发展。韭菜的减少直接影响当地政府的收益（房地产市场交易量、日常生活开支、税收等），于是就有了前期 “全面放开东北地区生育限制”的建议。但影响东北地区发展的主要制约因素是发展理念（或叫管理模式）。东北“迎来送往”、苛捐杂税、以权谋私等比比皆是，这就是打消外来资本投资热情的重要原因；而人口流失的主因是在于东北国有经济体制，压制了青壮年的工作热情。体制内论资排辈和阿谀奉承才有“钱途”迫使越来越多东北人南下务工，而留下的又因抚养孩子成本急增和未来预期资金压力过大，打消了生育念头，所以放开“生育限制”不能解决生育和抚养压力问题，无法缓解人口外流和促进经济发展。
![]()![](https://gnews.org/wp-content/uploads/2021/03/GNEWS-2021-03-08-coverphoto-fig2.png)
至于“给予东北地区个税减免50%优惠”的建议纯粹视法律为其“家法”。中共个税制度就是其盘剥人民、敛夺财富的“公开”手段，来路不正且负担过重。上述建议提及全国范围内个人所得税税制和税负不变，唯独东北地区减免，这一“见利忘法”的公开表态直接违背了中共倡导和宣称的“法律面前人人平等”。中共的个人所得税多么不合理暂且不说，无论你贫富贵贱、社会地位高低、人际关系多牛、是否拥有红色基因，纳税作为中共宪法规定每个公民应尽的义务，理应适用于每个人。但在中共手中，法律成了奴役人民的工具，各级领导作为特权阶层享有法外豁免权。无论是布衣们去谋求官位，还是已为官者奋力向上“攀爬”，其目的都是为了“树大根深而不动”——享有更多特权，可游离于法律之外。为官者作为体制内人，养老保险自己不用出钱“国家全掏”，反而要求贫民百姓遵纪守法“依法纳税”，何其荒谬可笑。

上述的减免建议更是直接撕下中共法律这一压榨工具的遮羞布。“东北地区和其他地方不一样，应受到优待”的想法成了其要挟的动力来源。如果都可以这样，那还要法律平等做什么。虽然中共是以假治国、以黑治国、以警治国，但毕竟中共作恶前还要套上法律外衣。“区别对待”这一建议直接是扯掉中共的底裤，更打脸了“全面依法治国”的官宣。中共本质就是设计“完美犯罪”去干非法坏事，并逃避追责。如果不灭共，它不仅会用更残暴的手段打击异己，而且会更严酷地劫杀老百姓和其财富。东北想振兴，不灭共不行。
