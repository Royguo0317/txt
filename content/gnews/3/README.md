2021年2月21日，他又说：农民收入增长快于城市居民，目前不宜提高农民养老金最低标准。

根据国家统计局最新发布的数据，中国的农业人口总共有57661万人，占全国总人口41.48%。不计60岁以上的老人和18岁以下的未成年孩子，真正具有劳动能力的农业人口最多3亿人，而严酷的现实却是，农村青壮年大多去到城里打工，用青春换现钞，留守乡土的多是白发黄髫。当年在党的疯狂领导下，全民炼钢，出现了“青壮炼铁去，收禾童与姑”的惨象；今天，还是在党的亲自部署下，农村再次呈现“青壮打工去，收禾童与叟”的荒凉之景！无数农民守着三分薄地，几垄菜畦，再拿着政府恩赐的80元或100来元的人民币，艰难度日。
