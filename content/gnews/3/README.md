## 【遇見爆料革命徵文】遇見文貴，今生有幸
`2020-09-16 07:43 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/360612/)

![遇見爆料革命](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/10071044/Screen-Shot-2020-09-10-at-9.02.30-pm.png)
第一次看到郭文貴這個名字，還是在中共的網絡裡面。記不清是幾年前了，當時保監會主席項俊波出事後，牽連出郭先生的案子，國內媒體稱郭先生是騙子，是強姦犯云云。我當時完全不了解郭先生，在辦公室問：“郭文貴是什麼人呀？”辦公室一個同事說：“鄭州最大酒店裕達國貿就是他建的。”不過我還是不知道。

直到2017年，夫人在澳洲開始接觸爆料革命了，我那時經常還在國內，基本上對爆料革命一無所知。夫人在電話中就經常給我提到文貴先生及爆料革命披露的事。當然，由於是越洋電話，有些話不可能說這麼透，再加上我當時國內還有一些生意需要打理，並且還經常要與體制內的人打交道。我自認為對體制內的人還是比較了解的，也清楚他們的貪腐。但是夫人跟我說到CCP高層黑暗的時候，我還是有些不以為然，認為“天下烏鴉一般黑”，他們貪是貪，只要讓老百姓有點活路，世界還是會正常運行的。所以，我和夫人在電話中也經常發生一些不愉快的爭吵，因為當時彼此心境完全不一樣。夫人叫我早點回澳洲我也沒行動。

到了2018年，我待在澳洲時間有所增多，在澳洲期間也經常與夫人一起看文貴先生“報平安”直播、路德先生的節目。聽到他們爆的料，有些讓我瞠目結舌，尤其是聽到說他們搞“雙修”、活摘器官等惡行，心裡想：CCP真的有這麼邪惡嗎？但始終還是抱著半信半疑的態度。再加上在國內時間還是比較多，受國內媒體及環境的影響還是比較大。夫人叫我趕快處理掉國內生意，我不僅沒有聽，在遲疑中又受到生意夥伴的影響，還追加了投資，房子也未出售。現在想來追悔莫及。收回國內投資款要現在看來遙遙無期，嚴重影響到我們後來想要參與的G系列投資計劃。

2019年，我在澳洲時間更多了，除了日常工作外，還和夫人擠出更多時間看除了文貴、路德的直播之外的面具先生、卡利熙及草根小哥等戰友的節目，並且還惡補了文貴先生之前所有直播視頻，終於徹底醒悟過來。不推翻共產黨，全世界人民都沒好日子過。

本來按計劃，我要在2020年1月份去武漢談一件事情，但夫人聽了路德社的節目後堅決阻止我去。我便沒有去。隨後幾天疫情爆發，我當機立斷，立馬買了到澳洲的機票，趕在澳洲國境關閉前幾天回到了澳洲。現在想想還有些後怕，如果沒有文貴先生他們的提醒，當時很可能就陷在武漢哪裡都去不了了。而且在武漢封城期間會發生什麼，我自己也無法想像。

現在我們一家人都成了爆料革命堅定的戰友，每天擠出大量時間積極參加到爆料革命中來。除了每天在GTV及油管上收看節目外，到周邊社區散發傳單，宣傳病毒真相，並且在GTV上面開設了“西窗對談”夫妻直播節目，也動員孩子開設了自己的頻道，宣傳支持爆料革命。此外，我們全家都參加了“我是音雄”節目，我們夫妻準備參加的是第一季第五集，準備演唱根據我們自己真實的挺郭經歷創作的歌曲《我們在一起》。

感謝文貴先生及沖在前面的戰友們。認識文貴，今生有幸，“保財、保命、保健康”。雖然“保財”沒有全部實現，但是“保命、保健康”我做到了，“留得青山在，不怕沒柴燒”。也非常感謝文貴先生的大愛及睿智，將G系列的巨大價值分享給更多真正的戰友。感謝澳喜農場戰友和VOG的義工隊伍，提供了大量的時間分享資訊，引導我的成長。最後要謝謝夫人，不辭辛苦、煞費苦心地帶我走進爆料革命，從最初的不太相信到半信半疑，再到堅定相信並跟隨文貴先生，讓我成為了堅定的戰士，加入到新中國聯邦的建設大軍中來！
