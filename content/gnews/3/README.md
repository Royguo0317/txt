**1.****日本出口增速创41年之最机械订单亦增加**

[日本5月出口以1980年以来最快的速度增长，衡量资本支出的一个关键指标也在增长，这有助于日本抵消低迷的国内需求，因为中共病毒疫苗接种促进了主要市场的商业活动。出口的跃升在很大程度上反映了出货量从去年大流行导致的暴跌中反弹，这是一个值得欢迎的迹象，因为在长期的防疫紧急状态下，日本经济正在努力从第一季度的低迷中反弹。](https://www.reuters.com/world/asia-pacific/japan-exports-jump-machine-orders-up-sign-recovery-2021-06-16/)

[这一稳健的数据可能会支持日本央行在6月17-18日的政策会议上坚持其超宽松政策的观点，尽管该央行可能会延长大流行病纾困计划以支持脆弱的经济复苏。日本政府最近延长了东京和其他主要地区的中共病毒紧急遏制措施。“我们不能指望私人消费，但出口和资本支出的上升趋势将在第二季度挑起大梁，”农林中金总合研究所首席研究员南武志称。](https://www.reuters.com/world/asia-pacific/japan-exports-jump-machine-orders-up-sign-recovery-2021-06-16/)

**2.日本拟在东京奥运会期间实行准紧急状态**

[据日本时报14日的报道，一名日本政府工作人员周一表示，考虑到一些卫生专家对中共病毒疫情大暴发可能性的担忧，政府拟在东京奥运会期间实行准紧急状态。奥运会将于7月23日开幕，但公众对中共病毒疫情大暴发的恐惧情绪持续不断，他们担心一旦游客涌入首都，或将带来更多变异病毒。自4月下旬以来，东京进入疫情紧急状态，并将延续到本月20日，在此期间第四波疫情有所减弱。政府目前计划使东京转入疫情准紧急状态：违反规定者面临比紧急状态期间较轻的罚款，而餐馆和酒吧仍将被要求进行短时营业，但允许提供酒类。](https://www.japantimes.co.jp/news/2021/06/14/national/tokyo-quasi-emergency-olympics/)

**3.****日本四个在野党提出菅内阁不信任决议案，执政党以多数票予以否决**

[据NHK周二的消息，本次国会即将于16日结束之际，立宪民主党等四个在野党以菅义伟内阁屡次应对中共病毒疫情不力，众多国民生活艰难等理由，15日提出内阁不信任决议案。在野党上一次提出内阁不信任决议案是在前年6月。执政党一方认为没有理由对内阁不信任，在15日下午召开的众议院全体会议中以多数票对内阁不信任决议案予以否决。本次国会面临结束之际，执政党与在野党之间的斗争进入了决胜局。](https://www3.nhk.or.jp/news/html/20210615/k10013085341000.html)

**4.****调查发现，一系列系统故障应归咎于瑞穗的企业文化**

[一项调查发现，瑞穗金融集团(Mizuho Financial Group)的企业文化是其长期科技系统失灵的罪魁祸首，它造成了一种管理者不愿表达意见、无法很好地应对危机的氛围。在今年2、3月间发生了4次系统故障之后，由日本第三大银行委托编写了这份措辞严厉的调查报告。该银行曾在2019年花费了超过36亿美元对其系统进行大修，这是在2002年和2011年两次大规模故障发生之后进行的。第三方报告称瑞穗的技术人员轮换过于频繁，阻碍了系统维护经验的积累，而且管理人员不愿意冒险质疑现状和积极表达自己的意见。瑞穗表示将成立一个内部委员会以防止此类事件再次发生，并计划从科技公司招聘高管以提高专业知识。](https://finance.yahoo.com/news/mizuhos-system-failures-caused-corporate-025711918.html?guccounter=1)

**5.****民众抗议为年少者接种中共病毒疫苗**

[根据时事通信社15日的消息，厚生劳动省从本月起将美国辉瑞疫苗的接种对象由16岁以上扩大到12岁以上。京都府伊根町宣布开始对12岁至15岁的人群接种中共病毒疫苗，随后当地政府接到了大量抗议电话。电话里抗议者称“有10多岁的人接种后死亡的案例”，“请一定要认清疫苗的危险性”。也有抗议者甚至威胁要杀人，地方政府与警方对此事进行了会谈。北海道、爱知県、冈山县等地也相继接到了这样的抗议电话与邮件。抗议正逐渐向全国蔓延。](https://news.yahoo.co.jp/articles/70ed75b010b8463206c168e0a01523ddfd9edbb5)

**6.****日本政府16日起将向越南空运中共病毒疫苗**

[根据共同社的消息，茂木敏充外相15日会见记者时宣布将向越南无偿提供100万剂中共病毒疫苗。16日疫苗将被装上飞机，预计当日抵达目的地。向东南亚联盟（ASEAN）成员国的印度尼西亚、泰国、菲律宾、马来西亚输送疫苗也将在7月以后开始进行。茂木敏充对记者说，向东南亚联盟加盟国提供疫苗是由于他们是实现日本所倡导的自由开放的印太地区极为重要的国家。他还强调该决定除了疫情原因外也是基于外交友好关系上的考虑。](https://news.yahoo.co.jp/articles/2d969371e76485f1d7af05924843bdb85d597638)
