***简评：***

中国共产党（中共）作为共产国际资助的颠覆中国合法政府的恐怖组织，其非法、邪恶本质和欺骗行径，促使它多年来不断的寻找具备高破坏力的攻击手段来威胁甚至统治世界，网络攻击只是众多的手段之一，在计算机网络中，攻击是通过未经授权访问或未经授权使用资产来暴露，更改，禁用，破坏，窃取或获取信息的任何尝试，网络攻击可能会通过侵入易受攻击的系统来窃取，更改或破坏指定目标。中共的攻击行为是网络恐怖主义，使用计算机网络工具来关闭关键的国家基础设施（例如能源，交通，政府运营）或胁迫或恐吓政府或平民。编者认为，中共在印太地区的网络活动更像是一种网络能力的测试和实验，台湾海峡冲突极有可能复制其做法，在拿下香港实施独裁暴政的环境下，中共进犯台湾也只是时间的问题，经济封锁、网络攻击、武力进犯都将成为中共的进攻手段。

***原文翻译：***

中共正在改变东亚的网络游戏–并增加了整个印太地区冲突的可能性。网络安全公司Recorded Future最近的一份报告描述了中共特工针对印度目标开展的一场复杂的网络活动。该报告概述了一个中共支持的组织”红色回声”（Red Echo），如何设法在印度的关键民用基础设施中安装恶意软件，包括电力组织、海港和铁路。虽然对于去年10月的攻击是否导致停电还存在疑惑，但Recorded Future的报告中明确指出，Red Echo的网络入侵与北部山区边境的中印冲突直接相关。当这两个有核国家在零下温度和高海拔地区用中世纪的工具进行战斗的时候，一场更加高科技、21世纪式的战斗发生在网络空间。

利用这场战役，中共在东亚网络领域展开了新的游戏。现在，一个主要的国家行为利用进攻性的网络手段，发出了具有破坏性的政治信号。将网络手段作为国际安全关系工具箱的一部分并不新鲜。中共此前就曾利用网络手段向其他民主国家对手发送政治信息。例如，台湾总统蔡英文当选时，她的社交媒体网络就遭到了中共攻击。然而，这次行动与之前的网络行动不同的是预期效果。之前的网络攻击行动都是数字破坏行为，但现在攻击目的是在物理领域产生破坏性的影响，或者至少是破坏性的影响。这种攻击造成潜在的物理破坏更接近于传统的军事冲突。

这是印太地区网络攻击的重大升级。中共攻击者对目标的选择是升级的另一个迹象。中共攻击者不是针对军事基础设施，而是故意选择攻击民用基础设施。这一系列事件表明，中共现在重新制定印太地区的网络游戏规则。这种重置将增加冲突，从而影响传统的地区安全平衡。通过进行这种破坏性的网络攻击，中共现在已经将冲突升级为更接近于国家层面的军事冲突，而不再是最初狭隘的以边境地区为重点的地域。
