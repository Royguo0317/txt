**西班牙2021年4月6日电/西喜社——**

我的邻居是一位德裔俄罗斯老爷爷，前苏联解体后全家回到德国（这段历史，感兴趣的小伙伴可自行搜索相关新闻）。前段时间问起老人在前苏联的往事，他非常的高兴，还很热心地写了满满十页的回忆录打印出来送给我。他说很开心有人对这段历史感兴趣，他说很遗憾现在的人们早已忘记或者压根不感兴趣。

老人前页写着，回想那段岁月，我时常问自己，为什么我当时对那些谎言那么深信不疑?为什么我完全不知道去反抗?我也是有责任的。

当海市蜃楼坍塌后，内心的荒凉和绝望远比空荡的街道可怕。老人说，前苏联解体后，很多人失业了，昔日繁华的街头，忙碌的工厂和办公楼顿时那么静悄悄，空荡荡。很多人的精神崩溃了，人活一口气大概就是如此，说好的世界第一，全球强国呢？做了大半辈子梦，突然被打醒，失业，货币贬值，物资贫乏。很多前苏联男人们开始酗酒，因为没有钱，只能购买便宜的工业酒精兑水喝，很多男人最后把自己给喝死了，家庭也毁了，孩子们没有吃的，饥饿和没有希望的人生，孩子们流落街头去乞讨，甚至去偷东西、犯罪，孩子们因此被打死是常态。

前苏联解体后，并没有像老百姓期望的那样迎来更好的生活，有能力的人在想办法离开，国有优良资产被上面的人私自贱卖，有的人为此荷包满满，而普通老百姓的生活并没有变好。所以现在有些俄罗斯人反而怀念起前苏联时代，或许如今的吃不好饭还不如当初的精神食粮来得醉人。记得前几年去伦敦玩，当地的朋友告诉我，伦敦很多房子都是俄罗斯富豪买的，而邻居老爷爷告诉我，他在俄罗斯的老友告诉他，他们曾经工作的工业城市，很多人失业甚至吃不上饭。

当社会变革之时，受伤的永远都是普通老百姓。

从老人的字里行间，能感受到他一直为自己的工作而感到骄傲，按照他的描述，他当时所在的国有企业，是全世界该行业屈指可数的制造商，担任工程师的他还曾拥有公司的股份。但是当灾难来临时，一切都是云烟。人们在想办法找吃的填饱肚子，他笑着说家里还有二、三十年前在前苏联抢着兑换的美金。
