## 【香港要闻】钟翰林侮辱国旗非法集结等罪成 判监四个月【即时快讯】

前学生动源召集人钟翰林，被控去年在立法会示威区内毁坏国旗，被裁定侮辱国旗及非法集结两项罪名成立。今日在西九龙裁判法院，裁判官黄雅茵判处钟翰林监禁4个月，即時收监。

钟翰林现年19岁，被控侮辱国旗及非法集结两项罪名。控罪形容：去年5月14日，立法会逃犯条例法案委员会开会的当日，被告在中环立法会综合大楼指定示威区内抗议《逃犯条例》，期间与一亲共团体成员发生争执，并损坏对方手中的一面中共国国旗。

裁判官黄雅茵认为被告多次伸手拉扯国旗、跳高抛开国旗及旗杆的举动构成侮辱或玷污国旗罪；被告在公众地方公开示威，是明显扰乱秩序及破坏社会安宁的行为。据此判定钟翰林两项罪名成立。

除此之外，钟翰林还被控在网上发表有关煽动他人分裂国家的内容，涉嫌违反《港版国安法》第 21 条，被控违反国安法以及洗黑钱等罪名。

香港基本法第三章第二十七条赋予香港居民享有言论、新闻、出版的自由，结社、集会、游行、示威的自由，组织和参加工会、罢工的权利和自由。

一个受到民众拥护的政权，它的旗帜不会因为被抛高而受到玷污，它的稳定不会因为几句口号而被颠覆。相反，一个真正受到民众拥护的政权，它的旗帜即使遭到破坏也会鲜活于人心，它的稳定也一定会因为宽待异己而更加坚固。

一个漠视法律、践踏人权、随意拘禁的政权，一个利用司法、依靠武力的政权，只能换来人心背离，而民众积聚的失望、愤怒和怨恨终有一日会化为最有力的颠覆力量。
