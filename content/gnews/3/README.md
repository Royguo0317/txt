**西班牙2021年5月21日电/西喜社——**

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/20114535/IMG_20210520_133227.png)

5月17日，我去往刘升镇山区兜风透气。返回时，在枣阳市的汉城大道，有幸参与了一场由卡车司机掀起的抗议政府乱作为的维权活动。

这次对抗的导火索是湖北枣阳市颁布限制令，限制自重26吨的卡车只拉5吨的货物，这直接导致重型卡车司机生路受阻，在完全自发的情况下，枣阳市的一些群众在汉阳大道集结。

枣阳市民风彪悍，在人们的记忆里，有多次与中共匪党恶势力的大规模对抗，少有不敢对抗只愿下跪的奴才。这里有独狼精神，有令人倾佩的勇气，涌现过不少惩戒共产党邪恶爪牙的勇者。他们的义举阻止了中共恶党对其他人的侵犯，光荣的事迹在人民之中口口相传。当地人还有悠久的运输史，许多镇子的人靠跑运输维持生计。所以在人们的权益再次被共产党严重侵犯时，队伍很快就自发的集结了。

5月17日，游行示威的第一天，到下午3时这里已经集结了近百辆重卡。队伍的前端和中段汇聚了两群人。下午4时，枣阳市政府派遣公安前来驱离，未遂。6时，路上的车流量增多，市民们发现了这场游行，其中不少人成了自发的参与者。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/20114903/IMG_20210520_133341-1.png)
这天我采访了两位卡车司机，一位是贷款买的车，另一位是全款买的车。他俩向我讲述了5吨货实际上只有多么一丁点，根本没利润可图，连车都养不起。其中贷款买车的那位告诉我，要是当局绝不退让，就只能割肉卖车了。我问他贷了多少款，他说三十万。

5月18日上午，我跑步去集结地。这里多了不少市民，有附近的居民，有商店摊主，连扫街的大爷也参与了进来。我趁机采访了两位群众。那位扫大街的大爷情绪很激动，他说完全是因为痛恨共产党而来的。当听到大家义正辞严痛骂共产党，要打倒共产党时，大爷激动的热泪盈眶，不停擦拭泪水。另一位是附近的居民，他同样痛恨当局。他告诉我当初他孩子没学籍上不了小学，四处求爷爷告奶奶，最后迫不得已去上访。信访办还不让周一去，吃了大亏后才知道必须得周一去大闹，完了后来还被公安警告越级上访下次要行政处罚。

上午8时，车队长度超过了一公里，场面激动人心。8时20分，队伍前端进行了一场会议，参与者主要是卡车司机，我有幸参与其中。会议上通报了最新情况，当局依旧不把大家当回事，连个愿意来对话的负责人都没有。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/20114940/IMG_20210520_133418.png)

这时人群出现了两种念头，多数人主战，他们觉得自己的力量强大，而且还有民意支持。计划用重卡彻底瘫痪当地交通，甚至封死政府和公安大楼，逼迫当局跪下来求饶。少部分人主张走和平道路，和平游行，选出几个代表同政府对话，大家撕扯了好长一阵。我趁机采访了其中一位。我问他支不支持激进行动，他很犹豫。我又问他是不是有顾忌。他坚定的看着我说:“没有!”

最后大体上双方达成共识，搞份书面文件，选俩人给当局送去最后通牒。下午，我在汉阳大道上鸣笛了好长一阵，车队和人群不断增多，群情沸腾。

5月19日上午，当局撤销全部不合理政令，并保证不追究任何人责任，基本上算作一个无条件投降了。
