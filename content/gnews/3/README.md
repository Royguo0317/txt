## 【戰友心聲】 看清中共極權暴政的謊言和欺騙

蒐集/編撰：初心

「以銅為鏡，可以正衣冠，以史為鏡，可以知興替，以人為鏡，可以明得失」，開唐唐太宗的這一句金句卻令我印象深刻，然而至今又有多少人會去了解真正的歷史文化，以史為鑒。有人亦說：「忘記歷史的民族是不會有什麼希望的」，但是篡改歷史消滅歷史文化的政府，肯定不會是什麼好政權。雖然歷史不能重來，但是歷史卻不斷重演，中共極權不但沒有以史為鑒，而是將他們口中的歷史也成為洗腦工具之一。

我們從中共外交部四大不要臉發言時，經常會聽到我們有5千年的歷史文化禮儀之邦，然而我們5千年的歷史文化價值，在中共為了清除異己而發動的文化大革命時期，基本已經被消聲匿跡，剩下的也只不過是經過中共極權改造過的中共特色文化，那還有什麼四維八德之言，而且經過中共各種意識形態的洗腦，大部分的老百姓成為一群被中共控制的「烏合之眾」，正如勒龐在《烏合之眾》書中所言「衝動、易怒、缺乏理性、缺乏批判精神的判斷力、對情感的過度誇張以及他人……」這就是他們的特點一部分，如今在中共謊言欺騙下的中共國老百何嘗不是如此呢？每次都被中共洗腦慫恿，變身小粉紅去反美反日反韓的情緒高漲不能自拔。

而中共極權71年一直以來都是通過謊言欺騙來達到洗腦目的，都是以鞏固自身的利益著想，當有人質疑時就會用各種暴力手法打壓。從中共成立的歷史開始，中共已經知道如何編寫謊言欺騙，連我們最熟悉的所謂的抗美援朝，中共極權政府為了自己的政權利益目的參與韓戰，不但製造許多謠言，卻隱瞞很多事實，為邪惡的政權塑造一個美好形象的謊言。而真實的情況是1950年6月25日凌晨，朝鮮人民軍以”反擊侵略”為由越過三八線大舉進攻南韓引起的戰爭，7月7日，聯合國安理會在蘇聯缺席下通過第84號決議，由聯合國成員國的16個國家的軍隊和5個國家的醫療力量組成的「聯合國軍」介入韓戰。因為真相被刪改，時至今日仍然成為中共欺騙老百姓的謊言例證。

其實中共極權謊言的例證數不勝數，而在今天互聯網絡發達的年代，就算被封鎖在中國強大的防火牆內，想了解真相也是比較容易，用自己的思維思考就會黑白分明，誰是誰非。且拋開中共極權制造的黑歷史不說了，用現今的例證也是舉不勝舉，而我們正在承受著病毒肆虐下的恐懼、痛苦及危機，也是中共謊言造成的，不但制造還隱瞞疫情擴散真相，什麼可防可控可治、不會人傳人，還打壓說真話的人，在家封城期間中共是如何對待生活物資不方便老百姓，暴政行為不是已經一目了然嗎？俗語說「身正不怕影子斜」，而中共千方百計隱瞞疫情的做法，不就是證明中共心中「有鬼」，為了掩蓋中共病毒真相嗎？而我們的英雄科學家閆博士的三份報告中指出，中共病毒中共制造不爭的事實。

而且中共滿嘴滿腦子都是謊言，而身體卻很誠實，眾所週知，中共無論是官員或者所謂的文人「教獸」從上到下都在罵他們的假想敵人西方國家，卻把自己的兒女都送到他們最恨的美國去買房子讀書居住，連中共偽政權一黨之尊的習近平親人，不是加拿大籍就是澳大利亞籍，女兒也是曾就讀於美國哈佛大學，外交部發言人耿爽、華春塋的女兒也不例外，一邊享受西方的自由空間一邊接受西方教育，而楊潔篪也是妻女安置在美國，這些都是有證可尋，而喉舌機器卻要老百姓去仇視被中共假想的敵人，整天在鼓吹美西方帝國主義如何亡我之心不死。而現實中，居然如此恨美國為首的西方國家，為何兒女親人卻向往西方世界？為什麼不送他們的兒女親人及財產去那些中共視為好朋友兄弟的國家？如朝鮮、古巴、巴基斯坦、委內瑞拉、非洲等國。難道真的「反美只是工作，孩子接受美國教育是生活」?而中共極權官員的行為已經告訴我們，这是赤裸裸的欺騙。 而郭文貴先生幾年的爆料中，告訴我們中共就是由一群盜國賊，以邪惡謊言欺騙手法治國，也爆出中共偽政權更多的真實面貌。同胞們該醒了，事實擺在眼前，看清中共邪惡的謊言和欺騙，不要為邪惡作炮灰。正如郝海東先生說的滅共是正義的需要，滅共就差你一個，那怕一點點的力量，能夠堅強勇敢的站起來，與我們一起Take down the ccp，做一個有尊嚴體面活著的人。
