## 中共警方逮捕了新冠疫苗制假团伙的头目和嫌疑人

尽管中共全力以赴，还是在中共国大陆[爆发](https://www.wsj.com/articles/chinas-new-covid-19-outbreaks-trip-up-economic-momentum-11612078814)了最新的新冠疫情，据新华社等中共国媒体的报道 ，**有超过80多名涉嫌制造和销售假冒新冠病毒疫苗犯罪组织成员已经被捕**，其中一些还在寄往其他国家。

据[《美国广播新闻公司》](https://abcnews.go.com/Health/wireStory/china-arrests-suspects-fake-covid-19-vaccine-ring-75629339)（ABC News）引用中共国媒体报道，在北京的警察，连同江苏和山东省， 渗透到他们声称销售疫苗的组织中，这些疫苗由简单的**生理盐水溶液**组成；一些嫌犯在国外被捕。

尽管目前尚不清楚究竟是谁买卖了疫苗， 但据新华社报道，这种疫苗在中共国和其他国家也被发现。中共外交部发言人汪文斌在星期二的每日新闻发布会上说：“中共国已经向相关国家报告了这一情况。”

*“中共政府高度重视疫苗安全性，并将继续努力严格查处任何涉及疫苗的假冒产品，* *虚假销售和非法交易的行**为**。* *同时，中**共**国将加强与有关国家的执法合作，* *认真防止这种违法犯罪行为的蔓延。**”*

发行假疫苗和假药的问题在中共国并不是新问题。 早在2016年，警察就逮捕了两个人，他们经营着一个在美国全国销售数百万种[存放不当的疫苗](https://www.nytimes.com/2016/04/19/world/asia/china-vaccine-scandal.html)的组织， 而在美国街头销售的许多的芬太尼则通过墨西哥和中共国运往美国。

由于假疫苗丑闻的后果，中共方面表示，将改革法规，并加大对假药生产者的刑事处罚。

中共当局警告称对中共国疫苗（其中有7个处于开发的最后阶段）的信心，受到假药丑闻的影响，据报导，这恰恰就在几种疫苗几乎准备好以非紧急的方式销售给各个国家的时候。

这起中共国事件并不是暴露假冒疫苗的第一个事件： 哥伦比亚警方破获了一个提供大约[15美元一针](https://www.zerohedge.com/covid-19/ecuador-police-bust-clinic-giving-fake-covid-vaccines)假疫苗的温泉疗养院。
