中国准军事警察在中国合肥的一个基地为新兵举行的无武装战斗示威中。

- 美国国防部最新的《中国军事力量报告》详细介绍了北京的军事扩张，并描述了其不断扩大的影响力。
- 在下面，您可以看到报告中包括的一些地图，以及它们说明了中国日益强大的军事力量。


五角大楼上周[发布](https://media.defense.gov/2020/Sep/01/2002488689/-1/-1/1/2020-DOD-CHINA-MILITARY-POWER-REPORT-FINAL.PDF)了年度中国军事力量报告，其中详细介绍了中国军队（正式称为人民解放军，PLA）向现代化部队的转型过程。

该报告包含的事实将使美国决策者和国防官员感到不安：中国现在拥有世界上最大的海军，正在扩大其已经庞大且先进的导弹库，并有能力夺取台湾外围岛屿。

报告指出，“过去二十年来，中国已经调集了资源，技术和政治意愿，以在各个方面加强和现代化解放军。”他补充说，“中国在某些领域已经领先于美国。”

这一点在显示解放军部队的位置及其导弹射程的六张地图中都清楚可见。

解放军地面部队

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14184732/15-2-1.png)



[路透社/斯金格](https://pictures.reuters.com/C.aspx?VP3=SearchResult&amp;ALID=2C0BF1FBBH2JR#/SearchResult&amp;ALID=2C0BF1FBBH2JR&amp;VBID=2C0BX4ZBY02O&amp;POPUPPN=10&amp;POPUPIID=2C0BF1FBE32PY)

中国士兵在新疆维吾尔自治区喀什训练。

第一张地图显示了解放军地面部队总部的位置（在报告中标识为解放军或PLAA），以及其集团军的一般位置。

该地图显示了它们在解放军五个战区司令部中的位置：北部，中部，东部，南部和西部。在中国的边远省份新疆和西藏还有另外两个军事指挥部。

该报告称，中国人民解放军是世界上最大的常备地面部队，在13个集团军中拥有915,000名现役人员，其中一个大约相当于美国陆军的规模，大约为20,000至45,000人。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14184755/15-3-1.png)

美国国防部

一个集团军由78个联合旅组成，每个旅可以有多达5,000名士兵。

该旅分为三类：重型（坦克和履带式装甲车），中型（轮式装甲车）和轻型（步兵，空袭，山地和机动）。

有趣的是，地图上包含了解放军的空降兵，实际上是中国空军的一部分。兵团由六个空中联合旅组成，其中至少一个是用可空投的装甲步兵战车ZBD-03机械化的。

解放军海军（**PLAN**）

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14184818/15-4-1.png)

路透社/中国日报

中国海军人员。

下一张地图重点介绍了中国海军PLAN。

它是世界上最大的海军，拥有350艘船，并且“主要由具有先进反舰，反空和反潜武器和传感器的现代多用途平台组成。”

这些船只是解放军北部，东部和南部剧院的三支舰队的一部分。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14184841/15-5-1.png)

美国国防部

该地图显示了所有舰队的组成，其中东部和南部舰队的船只最多。该计划还包括[六个海军陆战队旅](https://www.businessinsider.com.au/how-chinas-marine-corps-compares-to-japans-amphibious-force-2020-8)和一支规模虽小但不断壮大的空军。

尽管该计划显示出雄心勃勃的全球野心迹象，但其主要重点是控制从九州一直延伸到越南周围水域的第一岛链地区。沿这条路线的主要爆发点是台湾，北京将其视为一个独立的省。

人们认为，[如果与台湾发生战争](https://www.businessinsider.com.au/chinas-military-buildup-attitude-toward-taiwan-raises-conflict-fears-2020-9)，则东部和南部战区的战舰将直接介入该岛，而北部战区的战舰将保护通往中国的海上通道。

解放军空军（**PLAAF**）

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14184909/15-6-1.png)

路透社

一架中国H-6轰炸机飞越东海。

根据这份报告，中国空军与空军的机翼相结合，是该地区最大的空军，也是世界第三大空军。

报告指出：“中国人民解放军正在以各种能力和能力迅速赶上西方空军。”

中国的1500架战斗机中有800架被认为是第四代，与西方库存的战斗机相当。中国空军还拥有一架隐形战斗机和一架隐形轰炸机。

该地图显示了PLAAF / PLAN航空战区总部以及多个空军基地的大概位置。重要的是，它还显示了PLAAF轰炸机和运输司总部的基地。
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14184943/15-7-2.png)
美国国防部

去年10月，中国推出了H-6N轰炸机，这是第一架能够进行空中加油的核轰炸机。这标志着中国核三合会空降部队的回归。

随着空中发射核弹道导弹的发展可能正在进行，中国很快将首次拥有完整，可行的核三合会。

该地图还记录了处理电子战和空中加油的特种任务飞机。它们是重要的力量倍增器，对于现代和占主导地位的空军至关重要。

中国的大多数空军基地都位于该国的南部和东部。它在西部的空军基地距离印度相当远，这限制了PLAAF在[紧张地区的](https://www.businessinsider.com.au/amid-border-dispute-tensions-how-china-and-india-militaries-compare-2020-6)作战能力。

中国导弹射程

三张地图显示了解放军最重要武器：弹道导弹和巡航导弹的毁灭性潜力。中国拥有世界上最大，最多样化的[导弹库之一](https://missilethreat.csis.org/country/china/)。

第一张导弹图显示了中国常规导弹的射程，几乎所有导弹都能够到达2000公里以外的第二岛链。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14185004/15-8-1.png)

美国国防部

尤其令人担忧的是DF-26和DF-21。两者都是可移动的地面发射中程弹道导弹，预计将瞄准美国航母及其护卫舰。能够携带常规和核弹头的DF-26被称为“关岛杀手”，因为它能够到达美国在关岛的基地。

中国人民解放军的H- 6J轰炸机能够携带六枚YJ-12反舰导弹，进一步扩大了中国人民解放军的反舰能力。

第二幅导弹图描绘了中国的洲际弹道导弹射程，这更令人生畏。它表明，中国的洲际弹道导弹可以达到1750公里至13000公里。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14185027/15-9-1.png)

美国国防部

尤其担心CSS-4 Mod 2 / Mod 3（也称为DF-5）和DF-41。两者都可以到达几乎整个美国大陆，去年10月首次亮相的DF-41可以搭载多达10辆可独立瞄准的重返载具-10枚弹头，每个弹头可以击中不同的目标，并已释放当导弹到达顶点时。

中国正在增加对导弹技术的投资。该报告说，2019年，解放军火箭部队（PLARF）“发射的弹道导弹用于测试和训练的数量超过世界其他地区的总和。”

最终地图显示了与台湾有关的中国防空，反舰和短程弹道导弹的射程。几乎所有岛屿都被大陆覆盖，而PLAN船能够填补任何空白。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/14185037/15-10.png)

美国国防部

与美国相比，中国军队和武器的质量和能力仍然存在重要差异。

美军[比解放军](https://www.businessinsider.com/chinese-military-is-improving-but-us-has-more-combat-experience-2020-7)（自1979年以来就再没有发动过大战争）以及诸如专用航母任务组之类的强大资产拥有[更多的战斗经验](https://www.businessinsider.com/chinese-military-is-improving-but-us-has-more-combat-experience-2020-7)。美国在与该地区进行了广泛合作的地区也有许多盟友。

解放军还面临着挑战，例如生产高质量发动机的[困难](https://nationalinterest.org/blog/buzz/engine-trouble-why-chinas-home-made-engines-arent-good-enough-its-air-force-133147)以及在中国共产党内部处理[腐败](https://www.scmp.com/news/china/military/article/2187366/chinas-military-needs-checks-power-stamp-out-corruption-top)和[内斗的](https://www.nytimes.com/2020/08/20/world/asia/china-xi-jinping-communist-party.html)[困难](https://nationalinterest.org/blog/buzz/engine-trouble-why-chinas-home-made-engines-arent-good-enough-its-air-force-133147)。

但是，这些地图使报告的重点得到了证实：中国的军事力量是一支强大的现代化力量，只会越来越强大。
