## 中共消滅蒙古文化的黑手伸向法國

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/15023230/Ginghis-Khan-1.png)图片来源 国家宫廷博物馆， dailymail.co.uk, 成吉思汗（c.1155/1162-1227年8月18日）蒙古帝国的创始人和大汗


2020年10月15日據英國每日郵報，中共國要求法國博物館不能提及蒙古族領導人名字“成吉思汗”以及 “蒙古”或“帝國”的名稱，並控制展覽文本、地圖、小冊子和通訊等文刊的內容。法國博物館對此感到震驚，指責中國試圖篡改歷史，並表示將堅持“捍衛人類科學和道德價值觀”。法國戰略研究基金會亞洲專家和研究員對此表示不滿，有人稱中方的要求是“瘋狂的”。

博物館工作人員稱此前與中國呼和浩特的內蒙古博物館壹直存在合作，中方在今年夏天開始要求對蒙古族文化進行管控，原定於近期的展覽被推遲到2021年上半年，後因中共病毒疫情的原因再次被擱置到2024年10月。

中共於今年8月以前開始在內蒙古小學強推漢化教育，遭到內蒙古校方、家長和學生的反對和抗議，海外內蒙古同胞進行了不同方式的聲援活動。此前中共對西藏宗教的迫害，對新疆人的壓迫、文化滅絕政策，對香港人和平抗議的鎮壓和暴行，都是集權政府對有文化、有信仰、有反抗性族群的恐懼而使出的極端控制手段，企圖扼殺所有充滿自由和反抗精神的人。

成吉思汗是中國古代12世紀蒙古帝國的創始人和大汗，他團結了各蒙古部落，當時很快擴展為軍事帝國，並統治了後來的朝鮮，中國，俄羅斯，東歐，東南亞，波斯和印度的大部分地區。蒙古帝國已覆蓋了中亞和中國的大部分地區，可汗征服了相當於四倍之大的亞歷山大大帝的土地，幾乎成了世界壹半的主人，為後代崇拜和敬畏。

蒙古族有如此輝煌的歷史和不願為奴的血統，蒙古人骨子裏追求自由的精神恐怕會被正在如火如荼的爆料革命及其所影響的海內外、國際社會正在開展的反抗共產主義的熱潮催生出新壹波反抗力量。這也就是中共為什麽這麽恐懼蒙古族群，他們在壓迫、同化西藏、新疆、香港人乃至很多國際人士屢試不爽之後，對蒙古族群伸出了黑手。
