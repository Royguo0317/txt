## 上海躍過東京成爲全球第三大國際金融中心，你也信？
`2020-09-30 02:15 Ronald` [轉載自GNews](https://gnews.org/zh-hant/392419/)

內新聞：WENWU      素材：霹雳年      校對:天藍色獨角色

截止至2020年9月25日，全球金融中心指數（GFCI）公布排名報告：上海指數爲748，躍過東京指數747，成爲全球第三大國際金融中心。那麽怎麽理解全球金融中心城市排行“老三”？

據維基百科介紹，以金融業服務業爲中心，是入選全球金融城市的必要條件。這個全球城市必須擁有跨國公司和國際大銀行的總部設立，要有活躍的外彙市場、股票市場、期貨交易、證券市場等金融産品市場。

此外，能依法治國、資金自由進出，並且有著完善的運輸交通、人才教育等硬件設施和體系制度。最後言論自由的規則也是不能缺少的。

2019年6月9日，香港人發起反送中的和平示威，讓中共的反人類暴行在世界面前暴露無遺。我們不得不面對這樣一個事實：在中共威權體系制度下，中共國城市是並不符合“全球”排名的條件，最多只能算“全國金融中心指數”報告。那麽爲什麽中共國和“全球”排名沾邊呢？離譜嗎？

自中共國改革開放，鄧小平等人確立了《八二憲法》，中共停止了對外輸出革命，港澳台和發達國家給予中國的經濟無償援助。就中美“三個聯合公報和台灣關系法”的基礎上，美國國會在1997年生效《美港關系法》。該法是一部美國國內法，容許美國依據《中英聯合聲明》裏中華人民共和國讓香港實行“高度自治”的承諾。在接下來，2001年末，中共國正式加入世貿組織WHO。

以上的事件都發生了，才讓中共國和世界美元主導經濟聯系在了一起。這是不爭的事實，也是中共國可持續發展經濟的基礎。值得關注的是，中共國在存在防火牆的前提下竟能進入全球金融中心指數排名名單！

自習近平上台以後………..一切也改變了

今年7月14日，美國總統川普簽署行政命令，正式中止《美港關系法》第201條(a)區別對待香港，終止香港特殊地位。香港轉爲中共國一般城市，這是曆史性的一刻！

試問：川普總統對貿易協議已經不抱任何希望的情況下，除了和中共獨裁政權穿一條褲子的海外企業，何人還敢把錢投資在中共國？我倒是想看看中共國如何把這個全球第三大國際金融中心的假擀面杖子撐下去。
