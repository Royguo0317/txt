## 又有一名華裔被撤銷美國國籍並遣返，這是間諜投誠的最後機會了！
`2020-10-08 21:35 GM30` [轉載自GNews](https://gnews.org/zh-hant/412254/)

**作者：黃三封**

一名已經加入美國籍的41歲哈爾濱男子余浩洋和他21歲的妻子被撤銷美國國籍並遣返。罪名是［簽證欺詐］和［非法取得公民身份］，還有［竊取商業機密］、［電彙詐騙］、違反［美國出口管制法］的走私罪、等，加起來被控罪名24項。21歲的媳婦陳燕之也被控三項電彙欺詐以及［協助和教唆］罪。

根據這個余浩洋自己的Linkdin主頁記錄，余浩洋出生在中國北方城市哈爾濱，2000年本科畢業于清華大學物理工程專業。2002年通過學生簽證去往美國，2009年獲得在美國的永居權，2017年成爲美國公民。看上去，這是一款經典的美國夢诠釋的版本，值得所有懷揣闖蕩世界夢想的人羨慕。尤其是他的工作履曆，自2014年開始，他作爲首席設計工程師加入到ADI公司，負責開發［單片微波集成電路］。這款科技開發用途及其廣泛，比如廣播通訊、衛星通訊、航空航天、國防等諸多領域。聽上去不明覺厲，讓人鼻血直流！

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/08213021/1-42.jpg)

可是，這個無可挑剔的清華大學物理工程專業的高材生怎麽就折戟沈沙了？2002-2003年期間他在麻州大學Amherst分校進修電子工程。中間有10年的時間，他在美國完成了從留學生到永居再到公民的身份轉圜，其中的狀況艱辛還是順風順水，只有他自己知道了。

從2014年－2017年就職于ADI公司。至于後來爲什麽變成了竊賊，或許一開始就是竊賊，因爲披露出來的情節有限，我們不得而知，然而這些竊賊基本上承接了一套相同的模式，不久前那個麻省理工取得重要成果的複旦大學高材生唐爽，也是這樣的套路，一定要把偷人家的成果帶回她的國家。這些高材生都可謂人中的精英，可是不知道他們的腦子裏爲什麽有那麽多黨毒，當年愛因斯坦如果不是迅速離開納粹，結果是可想而知的，可是這些高材生，離開了邪惡的政權卻又要爲他們所用，可見，讀書或許可以學到技術，卻不見得學到智慧，他們終究還是沒看清事情的本質，終究還是沒有懂得個人與政黨、政黨與母國，母國再到個人，到底都應該是什麽關系。作爲一個人，他應該爲之奮鬥的，到底應該是什麽？是那個綁架了人民無惡不作的政黨，還是被惡黨綁架了的國，還是作爲人該恪守的正義和良知？

不知道美國法律系統是如何運作的，爲什麽美國人在美國犯罪還僅僅是撤銷國籍再遣返？不是應該直接判刑坐牢嗎？

如果他有雙重國籍，那可能就另當別論了，因爲美國是承認雙重國籍的。那麽這種情況下遣返應該是最經濟的選擇，不用美國納稅人花錢養這種吃裏扒外的罪犯了。再說，這樣的［又紅又專］，比較適合待在紅朝。可惜的是，紅朝一貫不會善待這些再無利用價值的灰頭土臉被遣返的落魄專家，他們更歡迎包裝精美、曾爲他們成功竊取過科技情報的最終還能落袋爲安的那些老賊，包裝成［衣錦還鄉］的愛國者，再用苦哈哈的老百姓的血汗錢供養起來，那樣的故事比較勵志，更能顯示他們的偉大光榮和正確，也更能讓後來的傻學究們死心塌地的爲他們去做賊，偷別人的東西。

聯想到前幾日民運大佬楊某自爆入籍被拒，是因爲刻意隱瞞共産黨員的身份。在海外［反共］幾十年，最終竟然是黨員！這種劇情雖然夠狗血的，但也充分驗證了［要想人不知，除非己莫爲］。這個余浩洋的罪名裏也有一條是［簽證欺詐］，或許也有某黨成員的身份。腦洞不妨在開的大一點，21歲的女生愛上41歲的大叔，險些可以公約的年齡差，當然真愛與年齡無關，只與荷爾蒙灌頂有關。可這個叫陳燕之的小女生的罪名中竟然有一條［協助和教唆］，這就有點意思了，21歲的小女生［協助教唆］了41歲的首席工程師，怎麽看都是派來監督執行的，最多是個假戲真做的余則成，和愛情差了一截。夫妻雙雙幹壞事，成雙成對被遣返。顯然政治的戲碼大于狗血愛情劇。

多少中國的優秀人才就被這種［間諜］給毀了，而最可悲的是，甚至直到被滅口的那一刻，還沒醒悟過來，還以爲自己從事的是神聖的事業。豈不知，任何一個正常的社會都會以這種盜竊行爲爲恥，無論多麽冠冕堂皇的說辭，都難以掩蓋做賊的事實。這些頂尖的科技工作者，本應在學術的科技的前沿領域爲人類的進步作出貢獻。父母耗盡家財多年的培養，自己皓首窮經多年的寒窗之苦，都成了泡影。上了賊船，就只剩下不敢再提及的理想和抱負。人生，要是離魔鬼太近，終歸難逃炮灰的宿命，珍重吧，同胞，現在仍然可以選擇主動和魔鬼切割，帶著你所知道的真相，投奔自由社會，等待你的是光明的未來，否則你只能成爲他們的殉葬品。
