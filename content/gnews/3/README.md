## 美称：美若明确介入台海问题，中共国必将四处搅局
` 曹操` [轉載自GNews](https://gnews.org/zh-hans/1156457/)

作者:【默凡】

美国之音4月30日报道称[1]：美国国家情报总监海恩斯说，当台湾看到中共国对香港采取的后，台湾已经越来越“坚定”要独立，如果美国明确表态将介入台海冲突，这可能使台湾更加远离中共国，并将导致中共国在世界各处破坏美国利益。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/04/30055859/55DC9BFD-12B5-48D2-AFFD-A515D6CB5506.jpeg)
国防情报局局长贝利尔则说，“从国防情报的角度来看，我们了解习近平的目标是统一台湾。我们不知道他是否已经决定如何或什么时候要行动。我们看到中国人民解放军过去一年在台湾附近的海域以及空中的行动有所增加。中国现在发生的一切，例如香港，西藏，维吾尔问题，我认为这是我们应对中国的关键问题之一。”

美国情报总监此番发言肯定有足够的情报来源，美国军方很清楚习神铁定要拿下台湾，为自己在2022年能顺利连任赚取政治资本，所以美国一旦公开支持台湾，中共国必定会采取各种手段破坏美国在全球的利益，习神甚至会铤而走险突袭台湾。所以美国提前对日本进行军事松绑，允许日本大力发展军事，同时美国还推动建立印太联盟，希望借助盟国之力共同牵制中共国。美国在没有彻底实现中美脱钩，做好全面的军事部署前，绝对不会仓促激怒中共国。

因此，笔者认为情报总监此番言论一方面是在提醒美国各界人士要警惕中共国威胁，另一方面也是在告诫中共国，胆敢继续危害美国利益，美国必将严惩不贷。
