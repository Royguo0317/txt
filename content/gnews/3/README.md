每日邮报5月29日报道，前内阁大臣大卫·戴维斯(David Davis)28日晚将中共政府阻止全面调查中共病毒疫情是否源于中共实验室泄漏的努力比作切尔诺贝利(Chernobyl)核灾难。

面对国际社会的警醒，越来越多来自世界各地的科学证据指向病毒首次传播的可能性在2019年11月，武汉病毒研究所的研究者感染病毒，引发病毒来源于中共实验室说。然而多个国家追查疫情源头的努力屡遭中方挫败。

引用1986年苏联核泄漏事件，前英国脱欧大臣写道：“苏联最初否认存在问题。面对历史上最严重的核事故的确凿证据，它表明问题并没有那么严重。直到危机最糟糕的时候过去了，它才终于向国际社会求助。和苏联一样，中共国也是一个封闭、偏执、一党专政的国家。它重复了这些同样的错误。我们现在都在为共产主义的保密付出沉重代价。”

戴维斯先生补充说：“我们需要一个全面和坦诚的调查，由采用正确科学方法的科学家领导。”这种病毒是人为的还是自然变异的?它是由动物传播的还是从实验室泄漏的?我们必须得到答案，不是为了攻击中国人，而是为了在未来保护我们自己。

《星期日邮报》(The Mail on Sunday)是首个报道实验室泄露担忧的主流媒体，该报在2020年4月5日的一篇文章中透露，情报专家担心病毒最初是由于中共武汉研究所的一次事故而传播开的。

昨天，一项新的研究称，中共科学家在武汉的一个实验室创造了病毒，然后对该病毒进行逆向工程，使其看起来像是来自蝙蝠的自然病毒。这篇长达22页的英国-挪威疫苗论文的作者，英国教授Angus Dalgleish和挪威科学家Birger Sorensen博士写道，这种病毒没有可信的“自然祖先”，并说这种病毒“毫无疑问”是通过“实验室操作”产生的。

他们还指出，中共实验室存在“蓄意破坏、隐瞒或污染数据”的行为，“希望分享知识的中共国科学家没有这样做，或已经被消失”。

昨日有报道称，英国情报机构已与美国情报机构合作，协助调查。拜登已下令在3个月内汇报调查结果。据了解，英国政府正计划成立一个专职部门以应对来自中共的威胁。消息人士说:“中共国现在是主要的外交政策关注点，而且在未来很长一段时间内都是。”同时英政府正敦促世界卫生组织对中共病毒来源进行“全面和适当的”调查。

以毒灭共进程已经提速，世界日益清醒地认清了中共病毒是源自实验室的生物武器，对中共的围剿追责势不可挡。中共释放超限生物武器，造成上亿人感染，数百万人死亡，全世界人们生活在病毒的威胁下，其邪恶和对世界造成的伤害远大于切尔诺贝利核灾难，中共必须为此付出代价，中共灭亡已进入倒计时。
