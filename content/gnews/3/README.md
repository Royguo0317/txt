4月24日，东京方舟农场在新宿站南口举行了演讲集会，旨在传播病毒真相，揭露CCP的恶行，以唤醒日本社会。集会上，不仅有方舟农场战友，也有东京樱花团、新西兰农场等其他农场战友们不屈的身影，我也有幸参与其中。

难忘的国歌声，2020年的6月4日第一次听。激昂澎湃、那追求民主自由的呼声，旷野中的呐喊，回荡苍穹。那是在电脑前隔着荧屏，感动得热泪滚滚。今天是在现场听，面朝祖国、面朝想回却不敢回去的故乡，难以抑制，泪如泉涌。新中国联邦成立快一年了，每天都和它难舍难分。早起想看有什么好消息，日中留意有什么新任务，晚上等着盼着文贵先生的声音，乐此不比，累也心甘。是什么让我如此兴奋？是什么让我和新中国联邦如此情深？

从出生开始直到邂逅爆料革命，我们都生活在谎言中，生活在被中共的愚弄中。学校学的是假历史，电影看的是抗日神剧，电视播报的是假新闻，人们谈论的是与世界相违的事情。是郭文贵先生发起的爆料革命，让我们知道了自己一直听从的是魔鬼的声音，让我们知道了父母才是最亲近的人。醒了，感觉之前就是一场噩梦，终于迎来了早晨，终于可以说真话了，终于找到了志同道合的人。

今天，这些志同道合的人们聚集在这里，是为了唤醒更多的人，追求自由、弘扬民主、唯真不破、尊严做人。我们的呐喊，令路人驻足，我们的演讲，令路人入神。所以，看到战友们激情演讲、听到战友们高歌新中国联邦、感受口号震天、国旗飞扬。国歌再次响起，怎能不令我为之兴奋，怎能不令我为之欢欣，怎能不令我为之感动，怎能不令我为之泪奔。三生有幸，能参与其中。感谢爆料革命！感谢郭文贵先生！
