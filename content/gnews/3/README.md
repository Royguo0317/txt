根据CNN4月3日报道，佛罗里达州州长瑞安.德桑蒂斯（Ron DeSantis）周五签署了一项行政命令，禁止在该州使用中共病毒护照。该命令禁止任何政府实体发放疫苗护照，并阻止企业要求任何此类文件。

德桑蒂斯以自由和隐私问题作为行动的主要依据，认为疫苗护照的实施和执行会根据疫苗接种情况建立“两类公民”。该命令指出，许多弗罗里达人还没有机会获得疫苗，而一些人可能具有感染获得性免疫力，另一些人可能因为健康、宗教或其他原因无法获得中共病毒疫苗。

德桑蒂斯的行政命令中写道：”个人的Covid-19疫苗接种记录是私人的健康信息，不应该被强制分享”。他周一在新闻发布会上说：”无论是政府还是私营部门，将要求你出示疫苗证明的规定强加给你，仅仅是为了能够参与正常的社会活动，这是完全不能接受的。”

该州长周五在推特上表示，佛罗里达州立法机构正在 “努力使佛罗里达人的这些保护措施永久化”，他期待着尽快将其签署为法律。

关于疫苗护照的隐私问题一直存在重大隐患，特别是如果建立一个集中的、证明人们已经接种了中共病毒疫苗的计算计系统。美国公民自由联盟高级政策分析师杰伊-斯坦利（Jay Stanley）本周在一篇文章中主张建立一个分散的、开源的系统，以确保人们知道他们的数据是如何和在哪里被使用的。

据至少两名政府官员透露，拜登政府一直在与非营利组织和科技公司合作，为人们制定一套标准，以证明他们已经接种了疫苗。不过，白宫新闻秘书普萨基（Jen Psaki）周一表示，不会有联邦授权要求每个美国人都获得疫苗证明。

中共病毒是超限生物武器，没有有效疫苗对应生物武器。知道该病毒真相的人们都不愿意打疫苗，除了有被感染的风险外还有无法预知的后副作用。希望有更多的州和国家能像佛罗里达州一样明智地禁止“疫苗护照”的推行，保护民众的隐私，让那些妄想用“疫苗护照”控制人类的计划落空。
