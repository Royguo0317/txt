## “為什麼G-系列市場潛力不可估量”

[https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/19204412/Miles-20200809\_4\_V2.mp4](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/19204412/Miles-20200809_4_V2.mp4)

郭先生在[8月9日的直播](https://youtu.be/7D_t9a8faA0)中指出G-系列（G-Coin，G-Dollar，G-Fashion等）是史無前例的貨幣系統，具有與黃金和美元掛鉤的能力（加密貨幣以及穩定貨幣的屬性）。矽谷對於確保G-系列技術的領先地位至關重要，因為要絕對保障該系統防止洗錢和防禦網路攻擊的兩大特質。與任何其他知名公司的IPO前的私募慣例截然不同，郭先生旨在為普通戰友能夠在財務上受益，並擁有美好光明的未來。

美國高科技公司擅長市場開發以及軟體技術和應用。但是依靠廣告收入獲利的商業模式阻止其有持續增長的潛質。中共的科技巨頭(BAT，百度/阿裡巴巴/騰訊)在技術發展方面表現平庸，但因為市場”金融化”的特性，以及利用抄襲美國的技術，他們能夠將中國的互聯網領域從3萬億變成12萬億，其中包括互聯網的金融衍生品。

憑藉G-系列在全球範圍內迅速增長的被廣泛認同和支援，G系列旨在將被中共綁架壟斷的的中國市場解放出來，並聯合其他全球市場，加以最尖端的高科技和全球人力資源的整合， G-系列市場潛力將是不可估量的。
