## 配合制裁24家中共南海建設公司 五角大樓還做了些什麼？
`2020-08-27 13:39 WENZI` [轉載自GNews](https://gnews.org/zh-hant/320846/)

By 海奧華
2020-08-28

週三在華盛頓，蓬皮奧表示已經將24家參與南海建設的中國公司列入黑名單，並對高管和其親屬實施簽證限制。這是自7月8日美國向南海派往兩艘航母聯合軍演，警告中共南海不屬於中共管轄範圍後的又一個升級行動。這次聲明中說，中共對南海大部分的近海資源的利用和開發是完全非法的，我們會和盟友一起抵制這種破壞行為。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/27133225/%E6%97%A0%E6%A0%87%E9%A2%98-16.png)

與此同時，美國國防部也在華爾街日報發表聲明指出，中共公開宣佈在2035年完成軍事全面現代化，在2049年成為全世界一流的軍隊。並用這些現代化的高科技來鎮壓自己的人民，尤其是維吾爾族人。相比中共的獨裁，美國是自由開放的全球體系。所有國家都以共同的價值觀，按照規則來長期合作，共同繁榮。美軍是為保障憲法的民主自由的精神。而解放軍只是盲從共產黨，獨裁的工具。

對比上次美國只用航母的軍演來警告，這次除了國防部在華爾街日報發聲明，及宣佈對南海公司和高管的制裁行動以外，其派遣U2楨查機飛入中共自劃的禁飛區。這次的行動不僅僅是在區域邊界處，而是直飛中心區域，拍攝並發佈了中共在南海軍演區發射導彈的情況，並在衛星照片上標注了經緯度。美軍的行動意在告訴中共，你的一舉一動都在我的掌控之中，而南海上的解放軍沒有作出回應。顯然美軍有備而來，中共反應不過來。

讓我們來看看美軍在哪些方面作了戰前準備：在資金上，美國出資3.94億説明印太地區的東南亞國家增強軍事實力。在盟友上，賣給印度，馬來西亞，印尼，菲律賓，臺灣，日本，韓國，越南這些南海地區的利益相關國家先進的武器，如無人楨察機和運輸機，來共同監視中共在南海的非法行為，並可以隨時統一聯合採取行動，對中共圍剿。通過印度和俄羅斯的武器生產合作，間接的拉攏了俄羅斯，減低了中共和俄羅斯的結盟可能性，進一步地孤立了中共。在技術上，五角大樓正投資超音速武器，5G通訊，防空反導一體化系統和人工智慧，並在海陸空，網路上準備好了和中共較量。美國在日本的三澤基地部署了太空軍，如果中國發射導彈的話，太空軍就能通過衛星楨察到並擊落它。

而中共在南海最好的軍備，號稱是它最先進的攻擊艦的引擎，也只是來自目前俄羅斯的技術，落後美國20年。如果南海之戰開打，從目前交戰雙方的盟友數量，軍事技術，資金實力，準備的充分程度上來看，戰友們覺得誰會贏呢？
