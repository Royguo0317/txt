据共同社周三报道，美日两国已敲定4月上旬日本首相菅义伟访美的联合文件主要内容，其中就包括将尖阁诸岛(钓鱼群岛)以文件的形式确定美国的防卫义务，就是要求美国签字画押，将尘阁诸岛纳入《日美安保条约》第5条适用对象之内。
