## 美國要求休士頓領事館72小時內關閉 中國外交官急燒密件
`2020-07-22 05:38 GM15` [轉載自GNews](https://gnews.org/zh-hant/272283/)

![休士頓警方透露，中國總領事館正在銷毀機密文件，因為他們將於星期五（24日）下午4點被驅逐。（圖擷自《Fox26》）](https://img.ltn.com.tw/Upload/news/600/2020/07/22/php3xxyfy.jpg)
中國駐美休士頓總領事館在當地時間21日突然在後院焚燒文件，由於火勢猛烈，驚動當地消防局到場，但不得其門而入，休士頓警方透露，他們正在銷毀機密文件，因為他們將於星期五（24日）下午4點被驅逐。 據《Fox26》報導，中國駐美總領事館在館內後院焚燒裝滿文件的垃圾箱，由於從總領事館內冒出濃煙，附近居民立刻通報消防局，消防人員到場後，由於外交使館等同該國領土，中國使館禁止他們進去，因此消防員只能站在門外待命；消防隊指揮佩納（Sam Pena）表示，焚燒的情況看來在掌控之中，他們沒有獲准進入領事館，只好在外頭監控。
