加拿大公共卫生署（联邦政府应对疫情混乱，出台自相矛盾政策的幕后团队）不会透露邱香果博士和她的丈夫程克定在戒备森严的温尼伯P4国家微生物实验室（National Microbiology Laboratory in Winnipeg）工作时是否是加拿大公民身份 [1]。

公共卫生署可以告诉我们，但以隐私为由拒绝这样做。该机构只是说，当实验室雇用时，法律允许两人在这个国家工作。但透露某人是否是加拿大人怎么可能是一个隐私问题呢？**这似乎是他们在刻意隐瞒本不需要被保密的事情。**此外，说某人“在法律上有权在加拿大工作”，向来意味着其非公民身份。

公共卫生署不愿与加拿大人站在一起，是否真的是为了掩盖在加拿大最秘密和最敏感的实验室中心工作的邱和郑是中国公民的可能性？这对夫妇于2019年被赶出该机构，并于去年1月被解雇，**此前加拿大安全情报局得出结论，这对夫妇有可能与中共国——特别是与中共军方共享机密信息。**

**加拿大安全情报局对邱可能已经转到中国武汉病毒研究所的信息表示特别关注。**

本周《环球邮报》的一份报告透露，自2016年以来，国家微生物实验室特殊病原体部门的七名研究人员共参与了六个关于世界上一些最致命疾病的项目，**并且都与中共国的同行进行了实验并共同撰写学术论文。**唯一一位在六个项目中获得赞誉的中国科学家严飞虎（Yan Fei-Hu）是**中国人民解放军军事医学科学院的研究员**。

一些消息来源还报道称，**中国军事研究人员是温尼伯国家微生物实验室的常客，有时未经安全调查。**如果不事先进行广泛的背景调查，一般人无法进入该机构。

这些联系中的大部分可能只是巧合。但即使国家微生物实验室解雇邱、程的原因仅仅是知识产权的不当分享（而不是泄露机密），**这也是特鲁多政府与中共关系过于紧密的又一个例子**。他们不愿意承认中国政府的百害而无一利。

当疫情刚刚开始蔓延到中国境外时，特鲁多政府向北京承诺，**不会禁止航班从病毒中心起飞。他们****还****邀请了中国高级军官来加拿大****参观****军方冬季训练。**

更经典的是，**特鲁多****总理希望所有****加拿****人获得中国疫苗，而不是西方疫苗**。他如此热衷于，**甚至曾经指示过国家研究委员会（NRC）向中国军事科学家提供加拿大研发的配方，结果中****共****政府在获得了加拿大的机密数据后切断了加方访问****中共****科研的权限。**

特鲁多及其团队必须停止佯装北京值得信任。
