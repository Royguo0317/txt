## 美媒报道：中共国兴起“抵制”外国商品运动令投资者震惊

美媒专文报道，美国商业集团近日警告：由中共发起的全国性“抵制外国鞋类ˎ服装和其他品牌的消费者抵制行动”，令外国投资者十分担忧，“不愿投资”。据文章引用美国商会一份“关于商业状况的年度报告”，中共的做法“加剧了”投资者们对于中共国商业性操作方面“不可靠”的“担忧”。因为，人们发现很多商业性的计划会非常容易被因为有“与中共国利益背道而驰的行为而受到惩罚”。报告的内容反映出来包括美国在内的外国公司对于中共国所存在的“经济和战略紧张局势的影响日益感到不安”。

文章列举之前在中共国非常猛烈的各种“抵制”线上线下的行为，涉及H&M、阿迪达斯、耐克等品牌。“抵制”的背景都来自西方媒体“对中共国西北新疆地区少数民族可能进行强迫劳动”的报道和揭示。

在文章中，引用的美国商会报告称，“接受调查的公司中有78％表示，北京和华盛顿之间的紧张局势正在加剧”。

事实上，川普总统任期内一直在进行贸易磋商，可见中共国与以美国为首的西方国家商业机构的斗争所引发的忧虑从未退去。中共在西方与之积极脱钩的过程中，也有一个和西方的对峙。美国商会报告中提到，很多投资者的担心“因官方组织和中共国媒体的煽动抵制消费”而加剧。调查显示：“五分之一的公司表示担忧”，“有7％的公司表示正在降低其投资（中共国）意愿”。

接受调查的公司中有“38％”的公司表示目前令人忧虑的情况“保持不变”。更加令人关注的是，美国商会指出，有“27％的信息和计算机技术公司”认为，在中共国的“投资条件正在恶化”，是“所有行业中最高的”。据文章分析，这些糟糕的状态出现在“执政的共产党利用补贴ˎ市场壁垒”以及企图保护自己高科技产业公司而对外企进行“非正式压力”的情况下。

从以上的商界动态和市场总体反馈，笔者认为，我们已经完全了解中共国目前的投资环境状况。限制、压迫、危险、忧虑等都是可以用来描述其状况的关键词。虽然我们知道，资本是逐利的，逐利的资本可以承担一定的风险；可是，风险可以有，但风险必须可控。不可控的风险，绝对不是资本会流入的地方。谁都不会做肉包子打狗的事情，谁也不会拿着肉包子当空舞，拿不回来包子是一回事，连手带人给疯狗撕咬粉碎，则是另外一回事。惯于计算和掂量的商业资本，才不会把真金白银白白送给中共。投入与产出都是要衡量的，付出与得到也是要好好盘算的。中共长期不按照国际惯例和市场规范行事，之前还是基本玩转了。但是长期来看，一个没有踏实根基ˎ欠缺安全和规范的中共市场，国际社会将会慢慢与它脱钩。以前中共用得顺畅的招术，世异时移，被戳穿假面具之后，就没人再和它演对手戏了。
