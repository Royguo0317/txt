## 涉千人计划华裔化学家被认定为中共国窃取价值1.2亿美元的商业机密

**作者：**Virgilio Marin 2021-04-27

**翻译：**加拿大温哥华圆成农场 卢卡斯
![]()![](https://gnews.org/wp-content/uploads/2021/04/098.png)图片来源：Virgilio Marin原文
《自然新闻》4月22日，华裔美国移民游晓蓉因从包括美国公司在内的几家公司窃取价值1.2亿美元的商业机密而被定罪。现年59岁的香农•游（Shannon You）正和她的中国同事一起，利用被盗的无双酚A涂层技术在中共国创办一家公司。除了成立这家新公司外，检方还辩称，她偷窃的目的是为中国共产党谋取利益。

经过12天的审判，游被判犯有共谋经济间谍罪、共谋商业秘密盗窃罪、拥有窃取商业机密罪、经济间谍罪和电汇欺诈罪。

游在为两家美国公司工作时窃取了与无双酚A涂层技术相关的商业机密——这两家美国公司分别是佐治亚州亚特兰大的可口可乐公司和田纳西州金斯波特的伊士曼化学公司。2012年12月至2017年8月，她担任可口可乐全球研究首席工程师，2017年9月至2018年6月担任伊斯特曼化学公司的经理。这些职位使她能够接触到几家公司的免 BPA 技术，让她有机会进行盗窃。

被盗商业机密的开发费用近1.2亿美元。它们属于荷兰的阿克苏诺贝尔、德国的巴斯夫、日本的丰田章男以及美国的PPG、舍温·威廉姆斯和伊士曼化学等。

BPA，或双酚A，通常用于在食品容器内部排成一行，以防止风味损失、腐蚀和食品和饮料的危险反应。近年来，由于这种化学品的潜在健康风险，许多公司都转向了不含双酚A的涂料。但化学公司的目击者在庭审中说，开发这些替代品既昂贵又耗时。

移民与中共和中共国企业相互勾结

审判期间提供的证据表明，游窃取了商业机密，在中共国创办了一家新的无双酚A涂料公司。她与来自中国东部山东省的63岁男子刘向辰以及一位未透露姓名刘的亲戚一起工作。

刘在2019年2月与游同时被起诉。他策划了窃取美国技术并用它来建立一家制造不含双酚A包装的公司。检察官发现，作为对被盗技术的回报，他向游承诺在新公司中占有一部分股权。

游在威海金鸿集团找到了一个中共国企业合伙人。根据2018年中共国媒体报道，该公司于2017年同意投资2658万美元给新公司的生产线。此外，他还赞助了游申请千人计划，这是中共支持的招募计划，旨在吸引外国专家到中共国工作。

通过该计划，中共山东省政府和威海市政府将奖励给游443000美元，用于窃取免 BPA 技术。同时该生产线还从这些政府获得了740万美元的资金。

检方称，在审判期间出示的证据，包括游的千人计划申请文件，表明这位华裔美国人不仅希望让威海金鸿集团从中受益，而且造福于中共。

游将于11月1日接受审判。

中共制定的政策鼓励窃取美国技术

此案是一系列涉及中共国赞助的盗窃美国知识产权的起诉中的最新一起。4月19日，一名中共国医院研究员因从俄亥俄州一家儿童医院窃取商业机密到中共国销售而被判处33个月监禁。这名研究人员的同谋是他的中国妻子，她于去年2月被判处30个月监禁。

两人利用窃取的商业机密在中共国创办了一家公司。他们从省政府获得资金，并申请了几个中共支持的项目，包括千人计划。

美国司法部国家安全司副助理检察长亚当•希奇（Adam Hickey）在2019年表示，自2012年以来，该部门提交的经济间谍案件中，有80%以上涉及中共国。他说，近年来案件的频率稳步上升。他告诉CNBC，这种增长可能是由于报道的增加。

然而，律师指出，中共国最近的政策本身也是有问题的。在谈到中共国减少对关键技术依赖的目标时，他说：”问题不在于中共国已经着手这样做。这是他们的产业政策的一部分，他们试图实现这一点的一部分，是国家赞助的盗窃或创造一个环境，奖励或视而不见。
