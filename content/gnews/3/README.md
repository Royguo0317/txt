## 布賴特巴特與路透社10/6/2020：彭培奧在四方會談期間向中共的“惡意行為”喊話並尋求亞洲盟友的支持
`2020-10-12 07:45 CHANGDAO` [轉載自GNews](https://gnews.org/zh-hant/419086/)

香草山綜合報導





中文稿：
 1.10月6日，彭培奧出席在東京舉行的美日澳印四方會談時警告北京“在該地區的惡意行為”，並領導了一場避開中共及其技術的國際運動，呼籲中共為其一貫對鄰國日益好戰的態度負責，尋求最親密的亞洲盟友的支持，以抵禦中共的地區影響力。
 2.蓬佩奧說, 四方合作夥伴前所未有面臨的關鍵問題是，我們聯手以保護我們的人民和盟友免受中共的剝削、侵蝕和脅迫，這些問題在南中國海和東海、湄公河、喜馬拉雅山脈和台灣海峽都能看到。我們可建立一個真正的安全框架，其他國家可以在“適當的時候”加入這個“框架”。
 3.蓬佩奧對中國的批評涉及安全、人權和冠狀病毒大流行等多個問題，並重申了川普政府對中共處理新冠疫情的批評，因這場危機由於中共的掩蓋而變得無限惡化。

評論：
本次四方會談驗證了文貴先生早前提出的全球滅共戰略，隨著日本新首相簡義偉的上台，日本將與澳大利亞和印度一起堅定地站在美國這邊一起反共。繼彭培奧歐洲訪問後緊隨的日本之行，全球滅共的局勢已經形成。
