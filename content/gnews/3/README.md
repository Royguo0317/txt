真相始终在那儿，真相随处可见，从不曾离开，在你手轻松可以触摸到的地方。可我的父母亲！家人们！同胞们！你们为何不曾碰到。就连我自己也迟迟没有伸手去抚摸一下！

当我二十岁的时候还相信成龙能打败泰森；当我三十几岁时还相信高手在民间；当我四十几岁还相信中国共产党解放全中国；当我无比自豪的和人谈论中国的四大发明时；当我觉得爹妈没有共产党亲时；真相对于我已经是遥不可及。

首先对真相有肤浅的了解，是郭先生无数次重复真相的重要。爆料革命坚持真相，捍卫真相时，我才慢慢思考什么是真相？真相为什么重要？才重新审视过往和正在发生的一切事件中真正的真相。当有一次可爱的大脑袋路德先生的一期节目中谈到民主与科学时，我突然增加了一点点对真相的理解。真相的基石是科学和民主。

真相绝不是发生了的结果。就像CCP用谎言，用欺骗，用流氓手段窃取了中国执政权已经七十多年，你能说这是真相?只有共产党才能救中国，你能说这是真相?真相是中国人生活在地狱的深渊。真相是我们几乎唾手可得，但我们却遥不可及。

最近我回想起一段电影画面可以最好证明真相近在咫尺。我们却没有看出共产党拙劣造假的手法。《飞夺泸定桥》片段中红军战士二十二人组成的突击队，攀爬着只剩铁链子的桥面，下面是湍急的大渡河，对面是已建起堡垒架起机枪的国民党守军。红军冒着枪林弹雨通过泸定桥。那时看的我心惊肉跳，对红军战士的无限的敬仰和崇拜。这虚假的精彩蒙骗了我们的长辈，我的同辈，也毒害着我的晚辈们。

战友们你们想没想过。你就是拿着北京板砖也能把红军（土匪）给砸下桥啊！当时的环境下能顺利的爬过去都是专业人员才能完成的，你又往前爬又开枪，共产党真是“神仙”！

文明的社会是科学的，是民主的。为什么现代科技诞生在西方国家。我想是源于在长期的生活中他们追求事物的真相，反复的检验对与错，真与假。

科学技术容不下半点的虚假。这就是 CCP统治下科学技术上一片空白，只能是偷，骗，买或者模仿，生产出很多非驴非马的骡子来。既不中看也不中用。您能找出一个中国的现代科学家不是在国外留学或深造的吗？这也是我佩服郭先生要在国外培养一批年轻的人才，作为建设新中国联邦的储备力量。

绝不能想象着自己伟大，或者高手在民间，中共国还有能人。真相是——中共国大部分精英早就被CCP的历次的运动中赶尽杀绝了。真相是——现在爆料革命新中国联邦做的事情是中国历史上从来没有过的。我们身在其中是最幸运最骄傲的一群人。

“谎言会让我们付出什么样的代价？”是电影“切尔诺贝利”片首语。那场灾难差点让整个欧洲变成荒无人烟，直到现在问题还没有完全解决。而CCP的谎言让中国同胞付出了更加惨痛的代价。大跃进时鼓吹的亩产几万斤，共产党的迷雾，毒蛊驱赶着迷茫的亿万国人，连留过洋能造原子弹的钱学森都附和着说：理论上可以实现亩产万斤 。
