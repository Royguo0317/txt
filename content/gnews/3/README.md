中共全網傳播“五個不答應”，從黨內和民間的反應看，這只是習總加速師代表了少數盜國賊家族，自說自話、自編自演、癡人說夢，企圖用對外威嚇、對內恐嚇加綁架14億老百姓、煽動民族情緒向全球正義力量宣戰。這五個不答應是說給不同的人群聽的。前二個不答應是對國內反對勢力喊話，第三個則是亮明他手裡最後一張牌：綁架14億中國百姓當肉票。後面二個不答應是威嚇國際正義力量。

習總的五個不答應其實是欲蓋彌彰，其實淋漓盡致地透露中國所有的恐懼、所有的害怕、所有的顫抖。這五個不答應暴露了其內心恐懼、困獸猶鬥，垂死掙扎。從習總的身嘶力竭和與會者的五次掌聲中我沒聽出任何自信，更沒有聽出任何改弦易轍的跡象，聽出的只是盜國賊被逼入絕境後的氣急敗壞，臨近滅亡前的末路豪賭與無力回天。這“五個不答應“對應著就是中共盜國賊家族的“五個恐懼”。

**第一個“恐懼“是：新中國聯邦郭先生帶領的國內國外爆料革命戰友向全世界曝光了中共真實面目與邪惡本性。**所謂“絕不答應”無非是表示它不甘心自己的假面具被揭穿，還要繼續編造謊言為自己貼金，以此欺騙和忽悠中國人民與世界人民。

**第二個“恐懼“是：中共的執政合法性將受到蒙蔽已久的中國人民的質疑和反抗。**

中共竊取中國政權70年來，靠著欺騙謊言和血腥暴政，滅絕人性，慘絕人寰，罪惡滔天，天怒人怨。所謂中國特色社會主義道路其實是奴役中國人民的70年，所謂中國人民建設社會主義的偉大成就不過是盜國賊家族瘋狂掠奪中國人民創造的財富。今天，隨著爆料革命三年的傳播，越來越多的人認識到了這一切，認識到了只有結束中共的一黨專政，徹底否定和拋棄中共極權，實行法治自由民主的新中國。可在歷史的大勢面前，習總加速師卻一意孤行[死不悔改](https://www.epochtimes.com/gb/tag/%E6%AD%BB%E4%B8%8D%E6%82%94%E6%94%B9.html) 。所謂“絕不答應”不就是說它要沿著盜國賊家族攫取的中共極權統治條道走到黑嗎？ ！

**第三個“恐懼“是：內部鬥爭黨利益撕裂慘烈內鬥**

這是習再恐懼的一點。沒有了14億擋箭牌，這些魔鬼蛇神暴露於陽光下。 **國內，**網上一片”答應“聲、拒絕”被代表“聲。不僅一般老百姓要求，連既得利益階層也參與進來了，甚至中共內部也不想被習綁架，以共滅共正成形成氣候。最近有曾習鬥導致的上海大抓捕，更早有各地的青煙裊裊、各種逼宮消息滿天飛。 **國外，**更是有我們新中國聯邦的全球大遊行，向全世界宣布中共不能代表中國人民，新中國聯邦公民要滅掉共產黨，整個人類的邪靈惡魔必須在地球上消失。反對習的力量正在大集結，反習勢力的大聯盟正在形成。國內國外中國人一起在努力，中共閉關鎖國成為西朝鮮的夢想都將破滅了，中共能甘心嗎？當然不能。所謂“絕不答應”，就是死活就要把老百姓綁上戰車，威脅國際社會。

**第四個“恐懼“是：全世界認識到了中共不等於中國人民。**

長期以來，中共編造的最大的謊言就是中國共產黨代表了中國人民，這個謊言不但欺騙了中國人民，而且欺騙了世界人民。但時至今日，中共的倒行逆施尤其是其喪性病狂制毒投毒，無恥下作無底線，禍害全世界的惡行，已將自己的魔鬼面目曝光於天下，世界人民和各國政府在郭文貴先生爆料革命和班農先生等國際正義力量的共同努力下，開始將中共與中國人民明確的區分開來，這是對中共的致命一擊。中共若失去了人民，即失去了統治根基，失去了攫取利益的來源，也就失去了對外吹噓、拉攏和脅迫的籌碼，中共將迅速垮台、消失。所以中共才有對美國國務卿蓬佩奧的令人瞠目結舌的瘋狂攻擊，因為蓬佩奧代表西方正義力量揭穿了中共聲稱其代表14億人民這一大謊言，呼籲世界聯合抗共。蓬佩奧清楚聲明：我們還必須與中國人民進行接觸，並賦予他們力量，他們是充滿活力、熱愛自由的人民，與中國共產黨完全不同。

**第五個“恐懼“是：恐懼美國及其盟友為代表的國際社會的覺醒，全面滅共行動就在近期。**

眼下中共已陷入全世界正義力量的合圍之中，內外交困窮途末路。中共胡攪蠻纏，故意把中共遭遇的反制和圍堵歪曲為外國對中國和中國人民的敵意。沒有人阻撓中國人民創造美好的生活，也沒有人破壞中國同他國的交流合作。縱觀時事：華為被排斥，並非”霸凌“，乃是多個國家維護信息安全、防範中共間諜行徑的必要舉措；”一帶一路“遇挫，乃是多國識破中共擴張野心、維護主權和國家安全的動作；國際社會譴責港版國安法，支持香港民眾，加強與台灣的合作，乃是維護國際法的體現。多國追究中共病毒真相，正是維護本國人民健康和生命安全的需要。但狗急跳牆的中共卻顛倒黑白，把國際社會的正義之舉抹黑為“把他們的意志強加給中國、改變中國的前進方向、阻撓中國人民創造自己美好生活的努力”“破壞中國人民的和平生活和發展權利、破壞中國人民同其他國家人民的交流合作、破壞人類和平與發展”。什麼中國人民“絕不答應”分明是中共自己在恐嚇國際正義力量，表示面對他們的合圍，將負隅頑抗到底。

美國司法部文件說明美國兩黨在滅共的問題已經達成一致，這就是龍金石練成的斬中共的刀，親共的力量在司法部文件公佈後，必須要轉向滅共，中共肯定得到了情報，很清楚全球滅共行動的時候就要到來，發表聲色俱厲的“五個不答應“只是走夜路吹口哨。 [習近平](https://www.epochtimes.com/gb/tag/%E4%B9%A0%E8%BF%91%E5%B9%B3.html)赤膊上陣宣讀《五個不答應》就是習特勒的《最後的演講》。看看王毅慘敗的歐洲之行吧，一切都已經開始。
