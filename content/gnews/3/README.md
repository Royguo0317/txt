佛罗里达州参议员杰森·皮佐（D-Miami）周四提交了一项修正案，旨在防止私人企业禁止员工接受实验性COVID-19疫苗。

该修正案是针对迈阿密的一所私立学校提出的，该校共同创始人莱拉·森特纳（Leila Centner）宣布，接种COVID疫苗后的教师不要立即靠近儿童，这使该校在本周成为头条新闻。

“现在，我所在的地区指望我们继续制止这种怪异做法，”皮佐对他的参议员说。

但是，对疫苗的警示远非“怪异做法”，甚至FDA都承认疫苗仅适合目前的“紧急批准”。

然而在州参议院以19票对19票投票后，皮佐的修正案以失败告终。
