## 【世事解评】为什么中共不代表中国人（2）
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/915042/)

作者：纽约香草山写作组 霍比特人



![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-2-1.jpg)



之前我们说过中共国所谓“议会”——人大——选举就是作秀。绝大部分候选人和最终被选上的都是中共党员，其余的也基本上是准党员（即所谓民主党派）。但就是这么个几乎由中共党员组成的“议会”，也根本就不是中（共）国的真正立法机构和最高权利机构。组成这个议会的，不过是中共当中的二流甚至是三流货色。他们必须服务和听命于那几个真正的大佬：江泽民、王岐山、孟建柱等。具体我还是从身边的生活来看吧。

![]()![](https://gnews.org/wp-content/uploads/2021/02/114-1.png)

上次说过，我的一朋友曾经一度当过地方人大委员。有一天，我们几个人在一起吃饭，有人就问他，“你在议会里干点啥呢？”他说，每次开会，所有重要的议题都是预先规定好的，尤其是那些关于人事任免的事情，绝对是事先就定好了的——人大的主任或者代理主任会把所有事情都安排好。而几乎都是党员的那些“议员”——人大代表——必须，也只能遵从“命令”，举手表决通过这些议题。他当时忽然拉下脸来一本正经地说：“这个开不得玩笑的，可不是闹着玩的。”

除此以外，他们也可以提一些鸡毛蒜皮的议题——这个他们是可以做的，而且也应该做，以显示他们确实是人民的代表，能够代表人民进行表决。比如，我这个朋友因为是做教师的，所以他就提出，不应该强制规定学校把牛奶作为学生早餐的标配（大家懂的呀，这些都关系到选哪个商家的哪个牌子，都是产生腐败的问题）。他哪怕提类似这样的议题，都怕会得罪人。

那么，是谁预先规定了那些重要的议题呢？这他就说不上来了。因为没法确定地说到底是哪个人或者哪几个人，但他说，只有三个来源：上级党委、上级政府、同级党委。——“绝对没别的了。”

所以我们就可以看到，中共国的人大只能按照上级党委、上级政府或同级党委的指示来进行表决。因此，人大被戏称为“橡皮图章”。你没有别的选择，如果你不想丢官丢命的话——“可不是闹着玩的。”

到现在为止，我们说到的还只是中共和人大之间的关系。事实上，在中共国，无论你身处何地，无论你是男是女，是老是少，你必定属于整个权利结构的某个级别，处在整个权利巨塔的某个角落，你必定被中共所统治控制。村有村支部，乡镇有乡镇党委，车间有党支部，工厂有党委，中共无所不在。

以中共国的军队为例，连队有指导员，团以上单位都有政委。每个地方都是两个领导，也就是所谓的“双首长制”。工厂有厂长和党委，车间有主任和支部书记。

那么问题来了，在这两个“首长”之中，谁更大呢？谁拥有最终决定权呢？答案是明确的，党的领导权更大。

比如，连队里指导员大于连长，团以上单位里政委大于团长、师长、军长等等。在工厂里，党委书记大于厂长。这就是所谓的“党指挥枪”。

这在中共国是尽人皆知的。之所以还要专门再提一下，是因为外国人，或者港台同胞连这都不知道。他们太天真、太单纯、太善良，从而容易被中共欺骗。我还是举我身边的例子。我曾经在一家原本是国营的公司工作过，当时说要把这个国营公司私有化。但当地政府要约定，这家公司“私营”之后，每年要给当地政府提供一大笔固定的利润分成——不管你经营情况如何，固定给钱。当时的总经理就不同意，说这个付不起。结果，他被公司的总支部书记给批评了，当然也被政府给批评了，最后被撤了职。后来，接替他的新任总经理就说，他是个傻叉，让你交钱你就答应下来嘛，交不出还能枪毙了呢？到时候总有办法对付的。现在好了，被撤职了。

对西方人来说，他们对中共总是没有真切的感受——中共对他们来说很神秘。这是因为，在西方，任何一个政党都只是政党，是社会团体，你参加和退出都是自由的，而且任何政党的费用都是自理：不管你如何募集资金，都是你自己的事情。但中共它可不是一个政党。你入党可以，退党可是不行的。它的费用除了基层党员是缴纳党费之外，其他只要是科级以上干部，都是拿工资的，是国家财政支出。也就是说，它的费用是纳税人承担的。

中共不是政党，中共不是政党，中共不是政党。重要的事情说三遍。

中共就是政府本身，就是议会本身。我叫它“超议会超政府”。中共的干部都有自己的办公室，拿国家的工资。各地方中共政府办公地和中共中央办公地都是由中共国的国家军队武装保护的。中共国所谓的人大，不仅那些人大代表几乎都是党员，而且就这么个人大都是被中共中央以及各级地方党委完全控制的。政府就更不必谈了。中共就是议会本身，就是政府本身，它是“超议会超政府”。

可是它却在名义上顶着个“党”的名称。这样一旦有重大问题发生的时候，它可以不必承担责任，而让并无实权的人大和政府，尤其是政府去承担罪责。我再举个例子。在中共国，房地产项目的验收是由房产商组织监理、施工、勘察和设计等几个方面来进行的，表面上看，没有政府的参与。但实际上，在验收的时候，坐在现场中央检视所有文件的，是政府质监站的公务员。他不点头，验收就无法通过，虽然在书面上没有他的存在。然而，一旦房子产生任何问题，都跟质监站无关，你去找房产商，去找施工总包，去找设计，找监理等等。板子打不到政府的屁股上。有人会觉得奇怪，说那你就不理他不行吗？你要是不想活了，你可以试试。

我举这个例子是说，中共就像例子中的坐镇现场的质监员。他审查一切而不承担名义，总揽一切而不承担责任。

对西方人来说，谈判的一个重要原则是对等原则：总统对总统，市长对市长，议长对议长。但他们不知道的是，当他们来中国谈判的时候，所面对的实际上是个“副手”。中（共）国的总统（主席）只是“副总统”，因为他的权利比中共的总书记小。北京的市长实际上是“副市长”，因为他得听北京市委书记的。西方人来谈判，从一开始就输了一招，因为他们面对的仅是对方的副手而已。

中共国是由“党”，而不是由议会（人大）和政府来统治管理的。中共控制着中国的一切：政治、军事、经济、文化等等。注意，这是白纸黑字、明明白白地写在中共国所谓的“宪法”里的。在我上初中的时候，就在课本里读过所谓“四项基本原则”，即，所谓“坚持社会主义道路，坚持无产阶级专政（1982年《中华人民共和国宪法》将无产阶级专政改为人民民主专政），坚持中国共产党的领导，坚持马列主义、毛泽东思想。”考试也考过好几次。现在的宪法，用词用语上有所变化，但在总纲第一条里，仍明文写着“中国共产党领导是中国特色社会主义最本质的特征。禁止任何组织或者个人破坏社会主义制度。”

在中共国有无数的法律，条目多到让人吐。要说真正被贯彻落实到百分之百的，我想就只有这一条了。

那么，这个事实上统治着中国的中共，又是个什么东西呢？以我个人的观察来说，中共的大部分普通党员都是好人。所谓好人，我的意思是说他们很能干，勤劳，有人情味，耐心细致。但我不认为那些中共的干部是好人，从科级干部开始，官越大人越坏，尽管可能更加聪明。这个话题我们下次再聊吧。
