**西班牙2020年12月11日电/西喜社/**——周四，密歇根州的15位共和党议员提出动议，支持德克萨斯州的选举诉讼。这将对包括密歇根州在内的四个主要摇摆州形成挑战。

立法者们加入了托马斯·莫尔协会的Amistad项目，以支持德克萨斯州的诉讼，该诉讼要求最高法院将确定选举结果的权力还给摇摆州的立法机构，以进行审查和可能的撤销。
