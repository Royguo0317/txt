美国和欧盟能够达成协议这在一年前似乎是无法想象的。但是今年3月份，华盛顿和布鲁塞尔暂停了近几年来针对空中客车和波音公司在关税征收方面的争端。今年4月，美国总统乔·拜登（*Joe Biden*）提议对所有大型高科技跨国公司实行公平征税，昨天，拜登的财政部长珍妮特·耶伦（*Janet Yellen*）将门槛定为15％。这标志着以达成协议的谈判正在进行中，该谈判将于7月在全球前20个经济体之间的威尼斯峰会上举行。

在经历了唐纳德·特朗普（*Donald Trump*）四年的可怕岁月之后，其他迹象证实了欧美关系再次变得放松。周一，布鲁塞尔宣布，它将撤销为报复该前大亨（川普）对欧洲钢铁和铝征收关税的决定。周二，拜登政府取消了对德国人马蒂亚斯·沃尼格的有针对性的制裁（他是弗拉基米尔·普京安全部的前代理人，数十年来领导俄罗斯和德国之间的*Nord Stream* 2天然气管道公司）。在周四，欧洲议会放弃了中共国与欧盟之间的投资条约，虽然默克尔曾强烈希望这一条约。但是这次欧洲议会议员都投票反对与北京达成的协议。

本质上，欧洲人和美国人正在相互作出重大让步以恢复信任的气氛。都认为一切都归因于一个人总是有点短视。马里奥·德拉吉（*Mario Draghi*）在奇吉宫（*Palazzo Chigi*）促进了这种融化。这位意大利总理与耶伦、及拜登的美国气候特使约翰·克里进行会谈，他保证将把意大利以及欧洲尽可能地保留在大西洋路线上。作为交换，拜登首先希望一起在美国与中共国的对抗中形成统一战线，现在，北大西洋再次阳光普照。

**欧盟与美国舍异求同，在咄咄逼人的中共国的威胁之下，双方都意识到了只有站在一起形成统一战线才是对抗中共最好的武器。被世界越来越边缘化的中共国也必将会被慢慢削弱直到最后被消灭。**
