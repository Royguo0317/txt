截至5月6日，加拿大安大略省累计染疫人数逼近50万人，染疫死亡人数8千2百多人，连日以来平均每天感染人数竟然和有1.26亿人口的日本差不多，安省省长道格·福特在4月22日汇报疫情时数度哽咽，为疫情的失控向市民道歉。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/06234134/vansky.com_.jpg)*图片来源：vansky.com*
据 BlogTO 5月6日报道，福特认为加拿大的疫情如此严重，是因为加拿大联邦政府一直没有釆取有力的防范措施来阻断病毒传播源头，因此，福特在刚刚过去的两个礼拜向杜鲁多发送了三封紧急信，要求杜鲁多政府对入境旅客采取更严格的规定，以阻止各种变异病毒进入加拿大。但杜鲁多对福特的紧急呼吁置之不理。

福特在递交给杜鲁多的报告中写道：如果联邦政府早日采取行动，我们本来可以避免今天疫情的泛滥。我们再次恳求联邦政府对边境采取更严格的措施，包括禁止一切不必要的旅行，对国内航班实施出发前病毒测试，并尽快修补陆地过境点的检疫漏洞。福特还要求修补国际旅行者进入加拿大的政策漏洞，并增加和完善强制性的酒店检疫。他还于上周表示，如果可以的话，他将尽快关闭皮尔逊机场，但过境点属于联邦管辖。到今天为止，杜鲁多及联邦政府拒绝作出任何回应。

安省议员韦邱佩珍也向杜鲁多发出了严格管控边境的呼吁，她说：过去数月安省不断呼吁联邦政府加紧控制旅客入境，防止变异病毒流入，但很可惜，联邦政府一直不接受安省的正确建议，其检测也是流于形式，漏洞百出。如果联邦政府再不作为，第四波疫情会来得更加凶猛！

根据安省政府公布的情况，有90% 的确诊病例是来自英国、巴西、南非和印度的变异病毒。安省政府已经尽职尽责进行防疫，比如封闭了本国两个邻省的边界，率先实行入埠强制检测办法。但没有权力封锁美加边境和实行空中封锁，由于杜鲁多政府的不作为，仅过去两个星期，美国就有6.5万人从空中进入安省，有15万人通过陆路进入加拿大，联邦政府仅对空中入境隔离三天等待检测结果，对陆路进入一慨不管，其中的感染者直接进入社区，成为可怕的传染源。

数据显示，过去14天就有多达57个抵多伦多的国内国际航班携带感染个案，病毒在加拿大通行无阻说明杜鲁多政府有大问题！韦邱佩芳最后呼吁：请联邦政府负起责任，堵塞所有病毒传播漏洞以保护民众，规定所有经陆路口岸，或乘坐内陆机到达安省的人，均要接受强制检测及酒店检疫，不要让第四波疫情侵袭安省！

加拿大防疫存在的问题并不复杂，要求杜鲁多纠正错误并有所作为的呼声也很高，但杜鲁多政府却充耳不闻、水泼不进。如果放任杜鲁多继续漠视疫情的蔓延，加拿大将陷入万劫不复的境地，因此，加拿大人应当向杜鲁多政府发出自己的声音，告诉杜鲁多，在疫情面前，他的不作为是严重的犯罪！
