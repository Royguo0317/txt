## 伊朗每年从比特币净赚十亿，中资企业也参与其中

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/24090157/id12970471-GettyImages-1230175365-600x400-1.jpg) (Photo by Ozan KOSE / AFP) (Photo by OZAN KOSE/AFP via Getty Images)

大纪元5月24日报道：根据加密货币分析公司Elliptic发布的报告，伊朗目前占全球比特币挖矿业务总额的4.5%。比特币为其带来了数亿美元的收入，这些资金尤其被用于规避石油禁运。伊朗政府也承认这些尝试是为了减少来自美国对其几乎全面性的经济制裁，制裁涵盖石油、银行、航运业交易。美国从2018年5月退出核协议（JCPOA）后，对伊朗实行制裁，并逐步加强制裁的力度，而这条生财之路也包含了中共国企业的介入。

伊朗国有发电企业1月的一份声明，该声明指矿商消耗逾600兆瓦的电力。根据报告，伊朗矿工使用的电力每年需要用大约1,000万桶原油，占2020年伊朗石油出口总量的4%。

近年来，伊朗正式承认加密货币采矿为一个行业，向其提供廉价电力，并要求矿工将其开采的比特币出售给中央银行。而廉价电力的前景吸引了更多的矿工，特别是来自中共国的矿工和资金。

伊朗挖矿的廉价电力吸引了大量的外来投资，特别是来自中共国的业者。这些中共国企业已经获得了采矿许可证，并在该国建立了业务。这些公司与“伊朗军队”有良好的关系。
