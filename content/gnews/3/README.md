## 北朝鮮，炸毀南北聯絡事務所
`2020-06-17 10:54 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/237063/)

![](https://gnews.org/wp-content/uploads/2020/06/Picture-1-77.png)
據韓國《國民日報》報導，北韓於6月16日當天下午2時50分左右炸毀了開城工業園區內的南北聯合聯絡事務所大樓。聯絡處南方代表兼韓國統一部副部長徐虎，16日強烈譴責道:“北朝鮮炸毀聯絡事務所行為違反了《板門店宣言》”。

徐副部長當晚在政府首爾大樓通過緊急發布會表示:”這是在韓朝關係中史無前例的，違背常識的行為。韓國政府對朝鮮的行為深表遺憾，並表示強烈抗議。朝鮮應該為此行為承擔應有的責任”。

徐副部長還指出:”朝鮮此前的粗暴言辭和單方面切斷通信之後，破壞韓朝共同聯絡事務所的行為不僅令我們國民感到震驚，也令全世界震驚”，”特別是在《6·15共同宣言》20週年的第二天發生了這樣的事情，此行為辜負了所有希望韓半島和平的人民”。

此前，韓國脫北者利用氫氣球飄過世界上戒備最森嚴的邊境，上面裝載的傳單像雪片一樣飄落在朝鮮領土：傳單稱朝鮮領導人金正恩(Kim Jong-un)為「魔鬼」 ，說他將面臨與伊拉克的薩達姆·侯賽因(Saddam Hussein)和利比亞的穆阿邁爾·卡扎菲(Muammar el-Qaddafi)同樣的血腥結局。多年來，數百萬張這樣的塑料傳單衝破朝鮮審查制度的控制，最近已經招致平壤的憤怒回應，威脅要打斷朝韓兩國維持了兩年的相對友好關係。之後北朝鮮單方面表示切斷南北兩邊溝通管道，並於6月16日炸毀南北聯合聯絡事務所大樓。北朝鮮的此次行為是朝鮮半島建立簽訂《板門店宣言》以來的最具挑釁性的。但是，事情僅僅是表面這麼簡單嗎？會不會是中共在背後操縱的利用朝鮮半島局勢牽制美國的一個手段呢？畢竟這一直是中共一貫的做法。
