路透莫斯科6月21日报道，俄罗斯新闻社周一援引俄罗斯外交部的话报导称，俄罗斯和美国已经开始接触，以使各自驻华盛顿和莫斯科大使馆的工作“正常化”。

早在今年的5月，俄罗斯总统普京曾下令，限制“不友好”国家的使馆雇员数量。鉴于此消息，莫斯科的美国大使馆随即便宣布削减人员和领事服务，并建议签证即将到期的美国公民离开俄罗斯。

当时俄美外交气氛，因美国和几个欧洲国家驱逐了俄罗斯外交官，称其参与了针对美国政府机关的网络攻击、试图干预2020年的美国总统大选。此外，美国还对一些俄罗斯实体实施多项制裁。因此，莫斯科则进行反制。

但在上周俄罗斯总统普京和美国总统拜登举行峰会后，俄罗斯驻美大使于周日返回华盛顿，这也标志着俄美两国的外交关系趋于恢复，同时应证此次俄美高层会晤的确达到了预期的积极效果。
