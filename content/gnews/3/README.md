他的兩大護法，班農先生和朱利安尼先生已經公開表態要對中共追究責任，這是壹場中共發動的生化武器戰爭；加之川普總統已經感染病毒，在出院後將對中共發起致命壹擊（參考英國首相約翰遜感染前後的對中共態度）。
