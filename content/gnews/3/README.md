## 中共国将面临第一阶段贸易协议的审查

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/02112722/%E5%93%BC.jpg)图片来源：Twitter
据新加坡《海峡时报》（The Straits Times）近日报道，中共国副总理刘鹤一周内两次与美国政府高层的金融和贸易官员进行了联系。刘鹤刚在5月26日与美国贸易代表凯瑟琳·戴（Katherine Tai，戴琦）就两国共同关切的经贸问题交换了意见，紧接着在6月2日又和财长珍妮特·耶伦（Janet Yellen）就同样议题进行了视频通话。

报道指出，耶伦在简报中说，拜登政府支持与中共国在符合美国利益的方面进行合作，以便使美国经济尽快复苏。同时，两国也需要解决共同关切的问题。而中共国媒体新华社则称，双方认为中美经济关系十分重要。双方本着平等和相互尊重的态度，就宏观经济形势和多双边领域合作进行了广泛交流，对彼此关心的问题坦诚交换了意见，愿意继续保持沟通。但是美中双方均未对所讨论的问题进行详细的说明。

报道认为，自拜登政府上任以来，中共政府似乎一直在寻求与华盛顿的高级经贸官员加强联系，旨在促进双边的经贸合作。但是华盛顿似乎心思并不在与中共国的经贸关系上，而是一直在联合盟友共同指责中共国在新疆犯下的种族灭绝和反人类罪。

在川普政府期间，美中两国进行了为期两年的贸易战，美方对中共国100多个领域的出口商品增加了关税。后来双方签署了第一阶段贸易协议，美方希望通过贸易协议能督促中共遵守国际贸易规则。在此贸易协议中，中共国承诺增加购买美国的农产品、制成品和能源产品，同时扩大美中双方在金融服务领域方面的合作。第一阶段贸易协议的签署暂停了贸易战的升级，但美方并没有撤销对中共国出口商品增加的关税。刘鹤在第一阶段贸易协议谈判过程中，担任中共国的首席谈判代表。

报道指出，半年后，也就是2021年年底，美中第一阶段贸易协议将要到期，拜登政府如今正在审查第一阶段贸易协议的执行情况。上周，美国贸易代表戴琦表示，美方与中方在经贸关系上仍面临巨大的挑战。

评论：

中共国派出刘鹤与美方高级经贸官员接触，实际上代表了三重含义：

1. 中共国的经济形势严峻，急需外部输血，派出刘鹤就是一种死马当活马医的表现，希望通过中美双方经贸方面的接触，能够增加外贸出口，缓解国内经济即将崩溃的压力。
2. 由于第一阶段贸易协议即将到期，中共国知道自己无法完全执行贸易协议中的规定动作，派出刘鹤也是一种向美方的示弱，希望美方在审查第一阶段贸易协议执行情况的时候手下留情。
3. 与美方经贸官员的接触对于刘鹤来说也是一种自我救赎，一尊对于刘鹤作为谈判代表签署的第一阶段贸易协议是绝对不满意的，他在党内也因此面临巨大的压力，继续派出刘鹤与美方高级经贸官员沟通的言外之意即：给你一次将功赎罪的机会。
