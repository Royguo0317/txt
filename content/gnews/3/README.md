据自然新闻（natural news）4月6日报道， 来自南卡罗来纳州的共和党众议员正在呼吁该州州长，禁止在该州实施中共病毒”疫苗护照”。

推动这项提案的是新上任的共和党女议员南希-梅斯（Nancy Mace）。她发表了一封致州长的公开信。梅斯在公开信中写道：”今天，我们要求您根据法律规定，尽一切力量禁止’疫苗护照’进入我们美丽的州。”在这封公开信中，南卡罗来纳州其他5名共和党议员中的4人也签署了这封信。

梅斯和其他代表认为，”疫苗护照”制度不仅将侵犯个人隐私和公民自由，而且可能触犯联邦法。更可能推动和印证疫苗怀疑论，并对本州经济造成负面影响。

州长虽未回应，但在推特中曾写过这样的一段话：”拜登总统和民主党人想强迫美国人在被要求时出示’疫苗护照’，但他们却反对出示身份证来投票”。”疫苗护照在南卡罗来纳州将没有立足之地。这个想法从骨子里就是非美国人的（思维方式）。”

疫苗话题已是老生常谈了，众所周知：疫苗有效只是人们的一厢情愿。疫苗利润才是医药巨头的追求；而疫苗副作用和致死性的现实悲剧却是千真万确。这就是疫苗的现状，难道还有必要谈论”疫苗护照”吗？
