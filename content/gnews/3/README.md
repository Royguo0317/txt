## VOG多語言組採訪日本大阪遊行團隊
`2020-08-30 10:51 GM62` [轉載自GNews](https://gnews.org/zh-hant/324907/)

編輯記者：川渝小哥G    校對編輯：文劍  文宠

8月29號,新中國聯邦在日本開展大遊行活動。 本次遊行由櫻花團、富士會農場和草根小哥團隊,在大阪、名古屋、東京三地聯合行動。其中草根小哥組織的新中國聯邦大阪遊行隊伍,由兩輛改造后的「新中國聯邦戰車」和約150名爆料革命戰友組成。日本方面出動了大量警力維持現場秩序,為本次遊行保駕護航。

問泊僑:請問你從哪裡出發到達大阪的?

問泊僑:是什麼樣的力量讓你花了這麼多時間精力來到大阪參與本次遊行?
