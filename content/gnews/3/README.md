## 【香港要闻】阉割选举制度草案首读 三权制度轰然倒塌
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1096433/)

搜集/编撰：天灭中共 / 封面合成：文粤

**中共「阉割」香港民主选举制度草案，即《2021年完善选举制度（综合修订）条例草案》（14日）正式提交立法会进行首读及二读。**

政务司司长张建宗、政制及内地事务局局长曾国卫、财政司司长陈茂波及劳福局局长罗致光等均有出席立法会的特别会议，曾国卫宣读法案内容时，一再重申中共主导修改选举制度，是「合宪合法、有理有据、出师有名、掷地有声」，是「真心诚意」透过改制确保一国两制的行稳致远。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4-14封面.jpg)（图片素材来源：立法会会议截图）
据特区政府公布的改制后的选委会组成划分，及候选人提名办法来看，地区直选大减至20席，其中10区以「双议席单票制」形式选出，而过往一直是民主派占优势的专业界别亦都改为团体票，为民主派顺利参选设定了重重关卡。因此，多个界别的民主派选委对新制度心灰意冷，表明不会参加。

**资讯科技界：1.2万名选民被褫夺投票权**

代表资讯科技界的前任立法会议员莫乃光一直坚持与民主派阵营共同进退，去年立法会选举之前，他交棒给业界新一代参选人黄浩华，获得界别选民支持，选情占优。

但新制度下，该界别不但被更名为「科技创新界」，而且选举办法改由团体票产生。这意味着，现时，该界别的1.2万名选民将不能在立法会选举或选委会选举中投票，为所属界别选出民意代表。而获投票权的团体名单显示，大部分都与中共企业有关，包括多个「国家重点实验室」、与创科有关的公营机构、以及以「国家」为名的工程技术研究中心等。

对此，黄浩华表示，情况比他早前预计的更差，坦言自己对日后会否参选抱慎重态度：「只能说民主派的参政空间已经变得很小。」

**法律界：辉煌政绩，恐永成历史**

法律界方面，由选举产生的议席减至15席，选民亦不再是律师、大律师，而是由30个与中共关系密切的法律界组织或团体替代。

在这30个选委组织中，除了大律师公会、香港律师会之外，绝大部分是类似中国委托公证人协会有限公司、中国国际经济贸易仲裁委员会香港仲裁中心、中国海事仲裁委员会香港仲裁中心等具亲共建制色彩的公司。

本来将代表民主派出选法律界的前立法会议员郭荣铿，曾经和其他民主派一起巧妙运用议事规则，成功阻碍多个恶法提交立法会，激怒中联办、港澳办多次点名恐吓，要对他严惩不贷，但他始终无畏无惧，获一众市民出钱登报章头版声援。直至其后被中共强行褫夺议员资格之后，他才宣布退出政圈，并回到加拿大生活。

因此，民主派过往曾在法律界选委会全取30席的辉煌历史，恐怕再难看到。
![]()![](https://gnews.org/wp-content/uploads/2021/04/image003-15.jpg) （图片来源：郭荣铿facebook）
**饮食界：黄店食肆选民资格被剔除**

由于黄色经济圈日益成熟，原本饮食界是民主派最有机会翻盘的界别。但是新制度下。饮食界的个人票被全部取消，仅保留业界的团体票。

换句话说，日后要在饮食界投票，除了须是牌照的持有人，更要求必须具备在香港餐务管理协会、现代管理饮食专业协会有限公司的董事会、香港饮食业联会总会董事会中有表决权。门槛大幅增高之下，势令于2019年开始登记成为饮食界的大批黄店食肆，丧失选民资格。

原定参选饮食界立法会选举的香港中小企食店联盟召集人林瑞华指，饮食界对新选举制度气氛冷淡，政府根本不了解饮食界运作，料市民对立法会的认受性会大减。

**新闻界被消失**

最令人惊讶的是，本来一直有票可投的新闻界，在中共「完善」后的选举制度中，被踢出体演文出界（即体育、演艺、文化及出版界）。

对此，记协立即发声明批评，指政府近期修订选举制度，将原属该界别的所有新闻团体剔除，意味着有关团体将失去选委提名权。而且，事先政府未就此作任何咨询，直接将新闻业界排除于选举之外，令新闻业界感到不被尊重，记协亦感莫名其妙。

值得一提的是，历任特首参选时，都会不厌其烦的强调会保护新闻自由、新闻自由是香港的核心价值等等，但原来这个价值如此廉价，可被轻易出卖。

**IT界、社福界心灰意冷**

IT界及社福界以往亦属民主派阵营，但今次两个业界选委都表示对新制度「心灰意冷、无心恋战」。

IT界选委单仲偕认为，「中共剥夺业界的个人票，改成有利于团体选情的游戏规则，令IT界普遍心冷。」

社福界选委赖建国坦言，未来难以预料进入体制内的发挥空间，「进到（选委会）也没有意思，反而若积极参选，好像为新制度站台一样。」
![]()![](https://gnews.org/wp-content/uploads/2021/04/image005-7.png) （图片来源：苹果日报）
**战友观点：**

香港以往的选举制度采取「比例代表制」，制度上虽然算不上真正的民主，但起码允许一定比例的民意代表有机会入到立法会，监督政府施政，为市民发声。当民愤有地方宣泄，民怨就不会积聚，社会张弛有度，香港因此全速发展。

反而来，当市民意愿无处反映，所积聚的愤怒，就会诱发各种不合作运动，这种不合作未必是社会运动，而是当香港人都心灰意冷，对未来感到毫无希望的时候，他们绝不可能好像以前一样，在自己的位置上继续全情投入，努力奋斗。

那么，这样的香港，对中共来说，就不再是一个会「生金蛋的鹅」。诚然，受苦的一定是香港人，不过，中共愚不可及的行为，亦令它自己失去一切从中获得好处的机会。
