## 福克斯著名主持人塔克痛斥“中共式輿論審查”已動搖美國立國之本，美國進入黑暗時刻
`2020-10-16 06:39 GM101` [轉載自GNews](https://gnews.org/zh-hant/427477/)

美東時間10月15號，美國主流媒體福克斯公開了一段視頻講話，由著名主持人塔克針對推特等社交媒體“中共式輿論審查”發表了觀點。此前，推特“審核”了有關總統候選人拜登與中共控制的外國公司發生巨大經濟關係的相關新聞，推特在更早時間段關閉了英雄科學家閆麗夢博士的帳號，封鎖了閆博士的病毒“中共超限生化武器”的報告訊息。

塔克指出一家主流美國報紙報導了關於美國總統候選人（拜登）兒子真實的新聞，然而幾個控制美國媒體的技術壟斷者害怕此新聞傷害到他們所支持的候選人（拜登），因此在大選前三周採用了和中共國政府同樣的管控方式阻止公眾看到相關新聞。

塔克認為這是美國建國245年以來從未經歷過的大規模新聞審查，而自己從未想過“中共式輿論審查”竟會發生在美國，美國的民主只有在公民之間能自由交換資訊時才能真正有效，然而美國已不再擁有民主，進入黑暗時刻。
