**团体批修订借入境条例锁港****大律师：政府借国际公约僭建权力**

《2020入境（修订）条例草案》下星期在立法会恢复二读，多个团体包括职工盟、支联会等傍晚举行记者会，质疑修订后的入境处处长权力过大，并借势锁港，限制港人的出入境自由。大律师邹幸彤批评，政府借《国际民用航空公约》为名，僭建政府权力及控制所有交通工具的流动，形容为不诚实、民间未能参与的条例草案。国际公约原意为要求国家之间互通航班消息，“国际公约不会话授权入境处限制人出境。”她斥当局如要满足国际公约，不应涵盖飞机以外的其他交通工具。

保安局表示，修订是为了履行特区政府在《国际民用航空公约》下的国际责任，推行预报旅客资料系统；又强调已在文件和立法会上清楚解释，该系统对港人的出入境自由没有任何影响，只是要求来港而非离港的航机提供乘客资料。
