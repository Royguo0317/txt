**简评：**

**2021年6月5日匈牙利布达佩斯万民群众上街游行，抗议政府在首都建立中共国复旦大学分校的计划。此前，布达佩斯市长卡拉松尼宣布，把校区周围的街道改名为“自由香港道”、“维吾尔烈士街”和“达赖喇嘛路”，以纪念中共国众多人权暴行的受害者。欧尔班政府不得不尊重民意，决定2023年通过公投，决定匈牙利人是否同意耗资18亿美元建立一所复旦大学的分校。**

**总理维克多·欧尔班年轻时曾是匈牙利的反共先锋，2010第二次当选总理后，奉行实用主义，实施“东方开发政策”，与俄国、土耳其等威权统治国家的关系，似乎胜过欧盟内部的国家，匈牙利也是欧盟内对中共国“一带一路”最为积极的国家，经济利益的捆绑，已经使其成为中共在欧盟的代言人。2021年5月5日，匈牙利在三周内第二度阻挡欧盟通过对中共国的制裁决议以谴责北京当局对香港选举法的改革。**

## 匈牙利数千人抗议中共国大学项目，向政府施压

[!\[\]()!\[\](https://spark.adobe.com/page/6PSvCwTWX7sfK/images/126333ff-3b6b-47d5-8c6b-c1e2aafc22d3.jpg?asset_id=7b00cb53-d27d-4132-bb4a-c8b367331e4e&amp;img_etag=%2213f5fa9fc1ecf65b5b683de3792f149d%22&amp;size=1024)](https://spark.adobe.com/page/6PSvCwTWX7sfK/images/126333ff-3b6b-47d5-8c6b-c1e2aafc22d3.jpg?asset_id=7b00cb53-d27d-4132-bb4a-c8b367331e4e&amp;img_etag=%2213f5fa9fc1ecf65b5b683de3792f149d%22&amp;size=1024)

**本周末，有约一万人走上了匈牙利布达佩斯的街头，抗议政府签署的让中共在首都修建大学校园的协议。周一有报道指出，政府官员发表声明，表示可能不会继续推进该项目。**

匈牙利总理维克多·欧尔班的政府是第一个与中共签署协议加入其“一带一路”倡议的欧洲国家，在该倡议中，中共国向发展中国家掠夺性放贷，承包建造其港口、公路和其他大型基础设施项目。虽然不是正式的“一带一路”倡议的一部分，但欧尔班政府也在1999年（译注：应为2019年）同意让中共国复旦大学落地建造一个海外校区，积怨已久的民众本周末上街强烈抗议，此后官员们声称，校区本身至今鲜有动工，将以公投的形式让匈牙利人民决定是否批准该项目。

报告显示，这所中共国大学将耗资约18亿美元，其中大部分将由匈牙利纳税人或国家接受中共国的贷款来支付。

在这次抗议之前，布达佩斯市长盖尔盖伊·卡拉松尼(Gergely Karacsony)上周宣布，该市将为未来的复旦大学校园周围的街道重新命名，以纪念中共国众多人权暴行的受害者。新路牌包括“自由香港道”、“维吾尔烈士街”和“达赖喇嘛路”。中国共产党当前对中共国西部的维吾尔族正在发起种族灭绝运动，西藏佛教领袖因反对共产主义已经被迫流亡海外几十年，过去一年中他们又监禁、打压了香港各地的和平民主示威。

政治观察家们认为，国内一个主要左翼政党的领导人卡拉松尼是欧尔班担任总理职务的主要挑战者之一。在接受世界上最大的共产党影响其国家之前，欧尔班在国内从事了几十年的反共活动。据报道，本周末约有1万人参加了抗议活动，他们挥舞着匈牙利国旗，举着抗议共产党影响该国的标语。示威者游行的口号包括“支持匈牙利教育”和“我们不会成为中共国的殖民地！”
[!\[\]()!\[\](https://spark.adobe.com/page/6PSvCwTWX7sfK/images/3ed90657-1763-4767-9759-5bf4475bb69d.png?asset_id=27165bee-4335-4fb9-8760-e23934c6b220&amp;img_etag=%2206be9d7269fcbc7c573116dcdfff56e3%22&amp;size=1024)](https://spark.adobe.com/page/6PSvCwTWX7sfK/images/3ed90657-1763-4767-9759-5bf4475bb69d.png?asset_id=27165bee-4335-4fb9-8760-e23934c6b220&amp;img_etag=%2206be9d7269fcbc7c573116dcdfff56e3%22&amp;size=1024)2021年6月5日，在匈牙利布达佩斯，抗议者在反对中共国复旦大学校园计划的示威中游行。




欧尔班政府似乎在周日对抗议作出了回应，提出让匈牙利人民对该项目进行投票。
[!\[\]()!\[\](https://spark.adobe.com/page/6PSvCwTWX7sfK/images/fd804cb7-b0d3-44d9-ba5c-f4a0972195ae.png?asset_id=3b830495-4bb2-4789-a382-7cb32d615b5e&amp;img_etag=%22661b30215ac2b65bb8855bbb5ab13e02%22&amp;size=1024)](https://spark.adobe.com/page/6PSvCwTWX7sfK/images/fd804cb7-b0d3-44d9-ba5c-f4a0972195ae.png?asset_id=3b830495-4bb2-4789-a382-7cb32d615b5e&amp;img_etag=%22661b30215ac2b65bb8855bbb5ab13e02%22&amp;size=1024)一个人举着一个牌子，似乎把欧尔班和中共国的独裁者习近平描绘成了新娘和新郎（欧尔班是新娘）。
根据自由欧洲电台/自由广播电台（RFE/RL）的翻译，“我们支持在布达佩斯举行全民公投，以决定当地人是否希望复旦大学在这里办学，”欧尔班总理府部长盖尔盖伊·古亚什(Gergely Gulyas)周日接受当地一家媒体采访时强调，两国政府没有为建设大学做太多的规划或其他准备，暗示匈牙利政府可以轻易地退出交易。

根据彭博社的翻译，古亚什称：“我们不想做‘好事’，却违背人民的意愿，包括布达佩斯的居民，这就是为什么我们同意，一旦投资条件公布于众，布达佩斯的选民将决定他们是否需要复旦大学。”

总理府部长表示，投票可能会在2023年进行。

中共国外交部对匈牙利的抗议活动反应平淡。

外交部发言人汪文斌周一表示：“跨国合作办学是当今国际教育合作的一种普遍实践，是促进各国民心相通的一个重要平台，符合时代潮流和各方利益，我们希望匈牙利有关人士保持客观、理性、科学的态度，避免将中匈之间正常的人文交流政治化、污名化，维护好两国友好合作的大局。” (译注：外交部原话)

中共国官方《环球时报》周一为该计划和欧尔班政府辩护——而此届政府却以反对共产主义而闻名——将抗议活动斥为反对派“为政治利益而丑化文化交流”的捏造。

《环球时报》预测道：“据报道，暂停或许是维克多·欧尔班政府应对周末最新抗议活动的灵活之举，不会从根本上损害中匈友谊，尽管一些匈牙利反对派政客通过炒作所谓的’中共国侵犯人权’来煽动抗议活动，为与现任匈牙利总理竞争而铺平道路。”

中国共产党的党报《人民日报》周二发表了一篇专题报道，宣传该党与匈牙利的合作协议，认为这是该专制政权能够为发展中国家带来的“广受欢迎的国际公共产品”的代表。

《人民日报》写道：“5月27日，匈牙利为该国最大的太阳能电站举行了落成典礼，该电站由中国机械进出口集团有限公司（CMC中机公司）在西南部城市考波什堡投资兴建，规划装机容量100兆瓦，作为中匈两国‘一带一路’倡议框架下合作的重点项目和众多‘一带一路’项目的代表，考波什堡太阳能发电站充分体现了‘一带一路’倡议所倡导的和平合作与共赢的理念。”

该宣传机构总结道：“由于各方的积极参与和支持，‘一带一路’已经成为世界上规模最大的国际合作平台和广受欢迎的国际公共产品。”
