**西班牙2021年5月24日电/西喜社——**

![]()![](https://gnews.org/wp-content/uploads/2021/03/三年自然灾害-1.jpg)

我是上世纪六十年代出生的。不幸的是恰巧赶上了所谓的“三年自然灾害”的严重饥荒时期——那是巨大的人间惨剧频繁上演的岁月。

中共官方宣传中至今依然保留“三年自然灾害”这个说辞，这是不堪一击的虚假宣传，因为这个所谓的“自然灾害”完全是罔顾历史事实的欺骗宣传。

许多有良知的作家与学者都撰文叙述过那三年中国人的饥饿史。1959年至1961年的水文与气象资料表明，那三年的自然气候是风调雨顺，这是有据可查的史料记载的事实。而中共官宣却欺骗世人，把由于领导人鼓吹“大跃进”带来的人祸而造成三千多万鲜活的中国人的生命被饥饿而夺走了，这是二十世纪全球最重大的人间惨剧，而中共一直谎称为“自然灾害”所致，可见其见不得人的阴险嘴脸。

1960年9月，我出生在中国东部的一个省份。那还是经济相对比较发达的省份，至少也是大陆经济中等发达水平的地区。在我幼年的记忆中，那是坐落于无棣三路的一个小小的四合院，系二层砖混为主加部分木结构的建筑。我家门外就是一支二三十厘米见方的木柱，同样的木柱还有三四只并排位于其它几家的门外，它们支撑起二楼的木质走廊，大院门洞对面是通向二楼的木质楼梯，呈九十度直角。隐约记得大院里有十户人家，每户住房的面积都很小，能住上二十几平方米的房子就会很知足了。我家在一楼，记忆最深的就是潮湿。年幼的记忆中，潮湿二字深深地雕刻一般地存于记忆深处。潮湿到什么程度呢？地下老是黏黏糊糊的，似乎一直都没有风干过。门口处用砖砌了一个台阶，那是为了挡住侵入到室内的雨水。因为这个小院的东面是一面石头墙，墙上是另外一个居民院。这个院子的地面高差比我住的小院高出约五六米的样子，现在想来这也是我家的房子格外潮湿的原因之一。

现在感觉奇怪的是，六七十年代，所在的城市降雨量很大，经常是满院子的积水，幼时人小，显得水很大，浑浊、黑黄色的雨水仿佛小河一样地冲撞、流淌、积淀在小小的院子里，而且常常是数小时不退。雨量大了，院子里的积水常常漫过门槛进入室内。仅仅潮湿也罢了，关键是面积奇小，现在的年轻人好像不会相信，仅仅十一平方米的狭窄房子里面，居然挤得下大小七人居住！好在那座建筑的层高约有三米四五，搭建一个吊铺（吊铺内高一米还多）绰绰有余。我与二个姐姐和哥哥都睡在吊铺上。

饥饿的记忆至今印象很深，三四岁时，常常会饿得大声哭泣。后来听母亲说，她外出前，把留给我的几块饼干放在某个地方，因为哥哥姐姐同样也挨饿，就到处翻看找出来，抢着把饼干分食了，哪里还顾得上照顾我这个小弟弟！我只能饿着，只能继续嚎啕大哭。

我出生最初的几年里，没有吃过几次饱饭，后来母亲曾经数次对我们几个孩子回忆说：有一次，家里什么吃的食物都没有了，四个孩子都在叽叽哇哇地喊着肚子饿，咋办？当娘的不能眼睁睁地看着四个孩子饿昏了吧？于是，她就忍着自己空空的肚子，外出求食物。去哪儿呢？听说三四里路外的本市最大的那家医院，会有菜叶子什么的。母亲跌跌撞撞去到医院，爬进一个侧门，终于找到几个白菜疙瘩（即：大白菜的底部根茎），这可是救命的宝贝食品啊。她很高兴地往家里赶。但是因为腹中空空，她半路饿得晕倒在地。无奈，只好把捡来的白菜疙瘩，拿出一个吃了，也顾不上洗干净啥的，只是用衣服袖子擦擦即吃下去，于是身上有了力气，才慢慢地赶回家，把这几个白菜疙瘩洗洗后，切开分给四个孩子吃了。

饥饿与潮湿，严重的营养不良，我就是这样，在一个普通的、十分贫穷的城市贫民大院中出生，至四岁前一直不能下地步行，基本没有离开过土炕。这座城市的许多户人家，都是五十年代期间，自城市周边的县乡迁入的乡村移民，生活普遍贫困，买不起木质床，只好使用黄泥烧制成土瓦，做成土炕，内留有烟道，做饭时烟火随着烟道绕炕一周，整个炕都很温暖。炕上铺一张席子。长到四岁后，才勉强可以独自摇摇晃晃、步履蹒跚地学习步行走路。

吃糠咽菜的日子，一直伴随我的幼儿至少年时代。记忆犹深的一件事情：小学四五年级时，全校师生到城市的第一体育场参加（观看）运动会，六七十年代每家都有几个铝制品的饭盒。我的饭盒里母亲给放上啥午饭？是几片钙奶饼干，一个煮熟的鸡蛋，一块玉米面饼子和一些咸菜，肩上再斜跨着一把军用水壶。 如果不是开运动会，还敢奢望吃上一个鸡蛋吗？即使家里有几个鸡蛋，也是给父亲吃，在母亲看来，父亲是家里的顶梁柱（多少年都是经济收入的唯一来源者），他在工厂里是重体力劳动，有肉有蛋首先给父亲吃。记得小学时代（六十年代末至七十年代初），放学走回家的路上，那是一条上坡路，几个同学排着队走回家。路上，闻见谁家飘出一股炒菜的香味，或者闻到一股炒熟的虾酱香味，都好似自己也吃到一般，不是那时的嗅觉格外灵敏，而是肚子里早已经咕咕叫得很响了。

现在想想，我幼年患上严重的婴儿瘫，身体基础没有打好。上小学后，大院楼上的邻居，一位王姓的叔叔，他在一家菜店担任经理，因为同情我们家孩子多，经济困难，他经常捎回几斤猪肉骨头给我家，煮骨头汤喝，据说这对人体补钙效果不错，我不但喜欢喝这类汤，而且把煮了多次、已经酥软的一部分骨头再咀嚼着吃了，或许对我婴儿时代的瘫痪康复也发挥了不少作用吧。

因为幼年的饥饿记忆，至今依然十分珍惜粮食。曾经数次看到公司食堂中，年轻人的餐盒中剩下许多食物，甚至把一半馒头都扔到垃圾桶里，十分不解，也会教育他们说：你们知道六十年代有饿死人的事情吗？当他们听说有几千万人饿死的事后，会十分惊异，还有的同事表示从来没有听说饿死人的事情曾经发生于中国的六十年代。但这是历史事实！在我们多灾多难的土地上，不仅有饿死人的残酷事实，而且我前几年阅读过的多篇文章中都记载过，河南、四川的一些乡村还有人吃人的事情发生！那几年中，因为一些村民什么食物都找不到，为了全家几口人的生命，万不得以的情况下，与邻居商量互相换女孩子，你家的女孩给我，我把自己家的女孩给你，互相交换孩子后，就欺骗他们说我给你先洗个澡吧，于是把她放进一口大锅里，放上水，盖上锅盖，再压上一块石头。炉子里填上柴火……活生生的孩子被煮熟了……孩子在热水中的鬼哭狼嚎，也装作听不见。他们是人吗？他们也毫无办法啊，他们的人性让位于苟全自己一家人的生命——不吃孩子，全家几口都得饿死！牺牲一个孩子换来全家四五人的食物，这是确确实实发生过的事情。更为残酷的事实是：据有的学者撰文说，发生大饥荒的那三年，国库里不是没有粮食，而是没有开仓放粮，要把粮食留着援外、留着酿造茅台酒，供中共中央等各级官员们享用。

中国大陆上在共产党领导下，曾经发生过这样的人间惨剧，但是这个党不但不承认、不反思自己因为政策和路线严重失误导致的人间惨剧，反而至今一再坚称什么“四个自信”，你坚持的社会主义公有制的制度，曾经饿死三四千万人，而且是在建政十年后的和平年代发生的，请问：世界上还有第二个这样的政党吗？这样的“道路自信”与“制度自信”还有啥脸面向世人一再吹嘘？
