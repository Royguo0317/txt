《福克斯新聞》在2017年5月13日收到了一封電子郵件，內容包含與中共國能源公司商討6個人的“薪酬待遇”事宜。這封郵件顯示，根據與中（共）國華信能源公司（CEFC）的協議，確認亨特∙拜登為“董事長/副董事長。中（共）國華信能源公司（CEFC）現在已經破產。

有消息人士告訴《福克斯新聞》，“大人物”指的就是前副總統。

《紐約郵報》最早公佈了這些郵件以及其他一些有爭議的信息，相關信息Fox News也已獲得。

喬∙拜登與烏克蘭的關係去年受到特別關注，當時眾議院民主黨人圍繞川普總統7月25日與烏克蘭總統的通話，進行了臭名昭著的彈劾調查。他們誣陷川普總統涉嫌濫用職權向烏克蘭政府施壓，迫使烏政府解僱最高檢察官。

2014年5月的另一封電子郵件顯示，烏克蘭天然氣公司布利斯瑪股份有限公司（Burisma Holdings）董事會的瓦迪姆·波扎爾斯基（Vadim Pozharsky）據稱向亨特∙拜登徵求如何停止“政治動機行動”的建議”。“我們迫切需要你的建議，如何利用你的影響力傳達一個信息/信號等，以阻止我們認為是政治動機的行動。 ”不到一年後，一封據稱來自波扎爾斯基的電子郵件顯示，他感謝亨特拜登邀請他會見了他父親。
