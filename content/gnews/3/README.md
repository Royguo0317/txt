## 微信被禁誰之過？
`2020-09-23 04:49 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/379361/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/23044322/1-128.jpg)https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/23044322/1-128.jpg
今年8月6日，川普總統依據美國憲法及相關法律賦予他的神聖權利，簽發了行政命令，宣布將於45天后，對微信進行禁用，即9月20日後，美國用戶將無法下載、更新及使用相關金融服務功能。總統令裡面指出，禁用微信是基於國家安全的考慮，微信在美國的運行給美國帶來了巨大安全風險，因此依據相關法律對其進行禁用。這同時也說明美國政府已經掌握了大量證據，通過行政命令的方式，快速的解除微信帶來的危險因素。

**那麼微信被禁到底是誰之過呢？**

罪魁禍首就是CCP,它是這一切罪惡的根源，而且解決這些問題的根本就是Take Down The CCP。中美之間的對抗，實際是兩種意識形態的鬥爭，信仰的鬥爭，正義與邪惡的鬥爭，光明與黑暗的鬥爭！這次鬥爭的結果，也將決定著未來人類的走向。

對於微信的禁令，中共外交部還是一貫作風，潑婦罵街加威脅恐嚇，指責美國持強凌弱，不遵守國際貿易準則，打壓中國企業。騰訊網更是發文指出這是川普總統為了選舉而實施的陰謀。美國國內的一些媒體則打起了溫情牌，指出微信是在美華人和國內親朋溝通必不可少的軟件，沒有了微信，生活都無法繼續了。這些聲音背後，我們都能感覺到中共宣傳系統的一貫手法，從不正面回答問題，而是混淆視聽，顛倒黑白。

**只有理虧才會耍“陰招”**

我們從正常的邏輯分析這個問題，騰訊作為一家民營企業，其一個產品“微信”被美國政府禁用，如果騰訊公司對該行政命令有異議，不認同對其的指控，騰訊完全可以起訴美國政府，在美國公證的法律體系下還自己一個清白。然而時至今日，我們並未看到騰訊公司的相關聲明，這符合正常的邏輯麼？另一個有意思的事情是，美國剛剛成立的一個“美國微信用戶聯合會”在加州起訴了聯邦政府，理由是微信禁令違反美國《憲法》及《行政程序法》，大家是否有種似曾相識的感覺。郭先生07年爆料海航巨額貪腐大案時，除了陳峰揚言郭先生說的百分之一千是假的外，並沒有來美國起訴郭先生，而是董克文帶著一群小丑先上了。這就是郭先生說的纏訴，耗費你的金錢、精力，直至把你拖垮，難道要故伎重演了麼？只是他們這次又選錯了對象。希望這幾個華裔律師真是出於對法律尊嚴的維護，對在美華人的熱愛才行此“善舉”。

同時微信之所以成長為一個集即時通訊、社交及金融支付一體的龐然怪物，完全是中共有意為之，通過防火牆屏蔽了大量國際主流的社交軟件，使中國人根本沒有選擇的權利。然後通過微信非法收集用戶信息，打壓言論自由，使其成為重要的維穩工具。微信在美國的用戶達到了1800萬，已經到了美國政府不能坐視不管的地步，禁用微信、抖音也僅僅是一個開始。接下來會有更多的中資企業面臨同樣的命運。只有徹底地把中共剷除，中國的企業才能堂堂正正的和世界做生意。
