《據蘋果日報報導》10.1「沒有國慶，只有港殤」，去年10.1市民號召遊行悼念港殤，結果換來警方6發實彈、逾180人被補。網上有人號召參與今日10.1「月夕行動」，多區警力亦明顯加重，有傳媒指警方今日調派不少於6,000警力在全港各區執勤，包括在高點設立觀測點，假如發現非法集結或違法行為，希望預早介入。
**有一名男子在警方驅散時被撲倒，暈倒在地，需由救護車接走送院。**
     有獲放行的市民指，有其他年輕市民被防暴警要求解鎖手機，檢查Telegram等通訊軟體。大批市民於銅鑼灣百德新街被截查，直至下午6時，約62人被捕，當中包括區議員李予信及梁晃維。
**警：山上疑有非法集結**
     晚上約9時，警方突在慈雲山登山口拉起封鎖線，離開的市民須登記個人資料，部份行經沙田坳道回家的市民亦遭截查，更須展示住戶證。一名竹園北邨居民因為沒有住戶證，與警員理論，警員指稱山上有人疑涉及非法集結，指令他繞路回家，居民大嘆居住在此40多年，從未遇過這種情況。接近10時，再有兩批東九龍衝鋒隊的防暴警員到場支援，並有多批防暴警察到獅子亭一帶佈防，作雙重截查。

**戰友點評：【恩典】**
     原本中秋佳節是人月兩團圓，一家人開開心心享月的日子，但對於香港人而言，並不是令人開心的一天。民陣原定今日發起十．一遊行，惟警方發出反對通知書，有線民發起舉行「和你燈中秋行動」。
     黑警已經嚴陣以待，準備了6000重兵部署，阻撓追求法治民主自由的香港人，目前香港整個社會都在高壓的環境統治下生存，隧道口設路障，商場街道截查市民，驅趕人群，為難記者的下三濫手段通通已經用上。
     登上獅子山賞月為12位手足祈禱，黑警以非法集結理由截查封鎖上獅子山，其唯一目的就是要將香港變成“一國一制”，同時以國安惡法為尊則加速沖向死亡的蔭穀。
     按照基本法和國際法條例，黑警在香港這1年多以來所犯下的反人類罪，最終會受到國際法律的審判。
