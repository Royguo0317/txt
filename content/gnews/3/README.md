## 快讯！香港“初选案”47名被告全部遭即时关押包括15名获保释者！
` JPHIMALAYA` [轉載自GNews](https://gnews.org/zh-hans/947554/)

【日本东京方舟农场】 作者：青衣 校对：文小律

香港“国安法”指定法官苏惠德刚刚宣布：“初选案”15名被告获准保释，其余不准保释。案件押后至5月31日再提讯。

然而律政司当场就保释决定提出上诉。随后法官苏惠德宣布：根据《刑事诉讼程序条例》第211章91条，命令获准保释的15名被告须即时关押，法庭将通知高等法院原讼庭，在48小时内将被告带到高院处理律政司的上诉申请。

这也就是说，全部被告目前仍需即时关押！！有不愿透露姓名的资深大律师及刑事律师表示，律政司提出覆核保释决定，而且还是当庭提出，以往相当罕见。

原获准保释的15人是：

郑达鸿、杨雪盈、彭卓棋、何启明、刘伟聪、黄碧云、谭文豪、施德来、张可森、伍建伟、郭家麒、吕智恒、林景楠、柯耀林、李予信。

不获保释的有：

区诺轩、赵家贤、钟锦麟、吴政亨、袁嘉蔚、梁晃维、徐子见、岑子杰、毛孟静、冯达浚、刘泽锋、黄之锋、李嘉达、谭得志、胡志伟、朱凯迪、黄子悦、尹兆坚、吴敏儿、谭凯邦、何桂蓝、刘颖匡、杨岳桥、陈志全、邹家成、林卓廷、范国威、梁国雄、岑敖晖、王百羽、余慧明、戴耀庭。

其中7名被告放弃8天内上诉的权利，包括：赵家贤、钟锦麟、刘泽锋、李嘉达、谭凯邦、刘颖匡、范国威。 多名被告的民主人士在庭上挥手大喊：“多谢律师！辛苦你们。政治犯无罪！香港人不死！“有被告亲友在获悉裁决后，于庭外失声痛哭。在场市民见状上前安慰：“不要哭，好多人支持你们！”
