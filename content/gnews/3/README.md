## 【热点播报】德州代表对副总统彭斯提起诉讼

作者：香草山写作组 Daxia 校对/发稿：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/29231938/logo-3-1-1.jpg)



据雅虎新闻报道：星期一，得克萨斯州代表路易斯·戈默特（Louis Gohmert）对副总统迈克·彭斯（Mike Pence）提起诉讼，以促使彭斯有权推翻大选结果。
![]()![](https://gnews.org/wp-content/uploads/2020/12/图片6-1-1.png)
雅虎新闻称：特朗普总统拒绝公开承认乔拜登当选，认为民主党人通过广泛的选民欺诈“偷走”了选举。还说尽管与总统结盟的律师多次起诉以推翻摇摆州的选举结果，但这些律师都没有提供足以证明拜登获胜幅度的选民欺诈证据。

国会将在1月6日由副总统主持的会议上公布选举结果。1887年的《选举计票法》要求副总统以仪式性的身份确认选举的获胜者。

路易斯·戈默特另发推文说：“我们仍然希望有一位联邦法官知道，这场大选偷走了这次选举，这将意味着我们共和党的终结。 该诉讼将确保副总统只能接受合法的和合法选出的选民。”
![]()![](https://gnews.org/wp-content/uploads/2020/12/图片7-3.png)
该诉讼将亚利桑那州共和党主席凯莉·沃德列为近十几名原告之一，声称任何宣布拜登为选举获胜者的行动都是欺诈性的。
