## 【医疗前沿世界】疫苗接种异常反应追踪报道
` 纽约香草山MOS03` [轉載自GNews](https://gnews.org/zh-hans/1081372/)

作者署名： 纽约香草山医疗部  圣母院钟声

![]()![](https://gnews.org/wp-content/uploads/2021/04/logo-5.jpg)



据WRIC新闻台3月30日报道，弗吉尼亚居民，74岁的理查德·特雷尔在接受一剂强生疫苗四天后开始出现奇怪的症状。“不适从一侧腋下开始，几天后开始发痒皮疹，红肿，很快皮疹覆盖了全身。一切都发生得太快.。当皮肤剥落时，感觉全身刺痛、燃烧和瘙痒。弯曲手肘和膝盖都艰难。”特雷尔说。

![]()![](https://gnews.org/wp-content/uploads/2021/04/图片11-3.png)理查德·特雷尔在接受一剂强生疫苗后出现的严重皮疹  图片来源：**努坦医生/VCU Health**

他去了医院的急诊室，医生确定他对疫苗有不良反应。“我们排除了所有的病毒感染，包括COVID-19本身,所幸他的肾脏和肝脏还正常，最后我们得出结论，这是他接种的疫苗引起的”，努坦医生告诉WRIC台，“如果不及时治疗，可能会危及生命。因为皮肤是身体中最大的器官，当它发炎时，患者会失去大量的液体和电解质。”

4月7日密西根州的数据显示，246名居民在接种完全COVID-19疫苗后两个多星期就检测病毒核酸呈阳性。这是1月1日至3月31日记录的所谓”疫苗突破病例”。州卫生与公众服务部（MDDHS）发言人通过电子邮件告诉《时代周刊》，其中11名居民住院治疗，3人死亡。死者年龄都在65岁或以上。完全接种疫苗意味着，自一个人接种了两剂辉瑞或莫丹纳mRNA疫苗，或单发强生疫苗后两个星期的血清抗体足以对抗感染病毒。换言之，这246名密西根州居民接种的疫苗并没有起到疫苗应有的保护作用。

4月10日据Epoch Times报道，强生公司位于佐治亚州福赛斯县（Forsyth County）的疫苗接种点在有8人出现不良反应后暂停注射疫苗，这是强生关闭的第四个美国接种点。本周早些时候，科罗拉多、爱荷华及北卡罗莱纳州的3个大规模强生疫苗接种点因疫苗接种后出现大量不良反应被关闭。美国疾病控制和预防中心（CDC）发言人在对新闻媒体发表的声明中说，官员们知道“在这4个州接种强生COVID-19疫苗后，疫苗接受者出现头晕、头痛、感觉昏厥、呼吸急促和出汗（血管或焦虑相关）症状的几起事件。CDC正与各州和地方卫生部门密切合作,已经进行了疫苗批次分析评估这些事件，没有发现任何值得关注的原因。目前，CDC和FDA并不建议卫生部门停止接种任何COVID-19疫苗”。

**强生公司在电子邮件中告訴《時代周刊》**：“我们知道，目前所有COVID-19疫苗都报告了血栓栓塞事件，包括血小板减少症事件。我们对副作用的密切跟踪显示，接种疫苗后发生的一些罕见事件与强生**疫苗接种**之间尚未建立明确的因果关系。”而上周末欧洲对牛津-阿斯利康疫苗接种后数十例血栓形成引起的严重副反应的调查结果显示，血栓形成与疫苗接种直接相关。辉瑞及莫丹纳的mRNA疫苗比强生早获得FDA的“紧急使用授权”（EUA，Emergency-Use Authorization)，早上市使用，已经有大量疫苗接种后的严重副反应报道。强生疫苗于3月初开始被允许使用，不到6周已经开始关闭接种点，关注目前CDC需多久能完成对强生疫苗的副反应调查。而这期间有多少疫苗接种点可能被关闭？又有多少强生疫苗接种者会付出健康，甚至生命的代价？

香港政府在推出对中共病毒的大规模疫苗接种计划的短短一个月内,香港已有13人在接种疫苗后死亡，其中11人被注射了中共国产新科疫苗（Sinovac）。中共在3月28日宣布在中国接种1亿剂疫苗后，没有报告任何严重副作用或死亡病例。目前已在中共国内外使用的疫苗有：北京新科（Sinovac）和中国国药的报道灭活疫苗，康希诺的腺病毒载体DNA重组疫苗刚投入使用。但中国产疫苗有四种奇怪的现象，分析如下。

1. 中共国内无疫苗接种后死亡报道.
2. 选择接种新科疫苗的智利、土耳其和巴基斯坦的确诊病例数量不降反升。智利在2月份接种了近900万剂新科疫苗，接种率47%，成为南美洲疫苗接种率最高的国家。然而，智利记录的阳性检测结果没有下降，3月26日单日记录的病例达到7 626例的新高，导致医院病床供应紧张，首都圣地亚哥最近被封锁。土耳其从1月中旬开始施用新科疫苗，至少有800万人接种了这种疫苗，占人口的10%以上。但他们的病例数在2月底也有所回升，3月30日记录了37303例新发病例，为去年3月11日疫情爆发以来的最高单日数字。此外，自2月初开始使用中国疫苗的巴基斯坦，目前正经历着第三波大流行，全国中共病毒感染阳性检出率升至11%，为疫情爆发以来的最高水平。总统阿里夫·阿尔维3月29日撰文称，他在3月15日接受中国国家医药集团公司生产的第一剂疫苗（国药）后检测呈阳性。此前，巴基斯坦总理伊姆兰·汗在接种国药的第一剂疫苗两天后，于3月18日也对该病毒检测呈阳性。据4月10日《大纪元报》报道，菲律宾总统卫队在接种中共国药疫苗后，有126人先后感染中共病毒，目前尚有45位未康复。中共在2月份声称，中国大陆有2000万人接种了疫苗，但尚未公布任何有关有效率的真实分析。
3. 新科疫苗接种副反应率与对照组相似。中共国疫苗只发布了一期和二期的临床研究数据，新科的相关数据显示不良反应率仅为20%左右，低于其他疫苗。在其他各种中共病毒疫苗中，疫苗接种组的副作用比例通常明显高于安慰剂组。但接种科新疫苗的副作用比例与安慰剂注射相似或更低。且参与I-II期临床研究的受试者人数相对较少，因此很难评估疫苗接种的不良事件。
4. 无第三期临床试验的公开数据。


目前中共国产疫苗已在国内外使用的新科和国药都属于灭活疫苗，其数据混乱的有效性记录让人难以理解，但只要看它们的《接种知情同意书》就清楚了。不同于国外正在广泛使用的8种疫苗，如辉瑞、莫丹纳、强生、阿斯利康等都附有详尽的副反应说明，最典型的发热、乏力、头痛、皮疹等都会一一列出，使接种者有所准备，即知情权。一页纸长的《新型冠状病毒灭活疫苗接种知情同意书》只着重了一点，如图红框，“本疫苗已完成的大动物试验未观察到抗体依赖增强现象，但尚无人群观察数据。”也就是说，这些疫苗是出了实验室就直接用于大范围人群接种，根本没有经过耗时的三期临床试验。中共国疫苗研发机构和制造商，就在那些疫苗使用国接种人群中获得三期试验数据。令人不寒而栗！粗略总结了新科的三期试验数据，有效率，即接种最后一剂后2周的血清中和抗体量：巴西50.3%，土耳其91%，印尼65%；国药疫苗的三期有效率，据秘鲁卫生局官员在媒体采访中透露的数据：北京株为11%，武汉株为13%。还有什么比这些客观数据更能说明问题的！为了快速占领市场，中共国产疫苗在远还未得到人群中的接种试验数据时，就仓促投入使用，其视人命为草芥的恶行，与二战时纳粹在集中营中，用囚犯活体做医学试验有何不同？

![]()![](https://gnews.org/wp-content/uploads/2021/04/图片13-3.png)

《自然新闻》4月1日报道，安东尼·福西博士在接受新闻界采访时承认，他炒作的“COVID疫苗作为拯救生命的一种方式，实际上可能并不安全”。《纽约时报》援引福奇博士的话，“在我们给儿童和孕妇疫苗接种前，国家需要确保疫苗真正安全。儿童和孕妇通常在接种疫苗后容易出问题”。这可能是自去年初中共病毒疫情爆发以来，这位美国疫情防控最高官员说过的最得体的话！

过去的数月，每天打开新闻版，扑面而来的就是似乎看不到终止尽头的疫苗严重副反应报道，对于政府健康监管部门，那只是一个统计数字，对于受害者本人，就是一生的健康，甚至生命。新中国联邦的英雄病毒学家闫丽梦博士早在去年七月就公开警告世界，“新冠病毒在实验室里被恶意增强了功能。所以，依照自然来源理论去寻找解决之道，是得不到终结大流行的有效疫苗或方法的”。中共拥有所有原始病毒株的基因改编信息，但看样子他们只有制毒，没有控毒的能力。
