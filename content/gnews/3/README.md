## 班農先生接受路德社采訪：川普總統競選新方向“TAKE DOWN CCP”

美東時間2020年10月5日，班農先生接受路德社采訪時表示，川普總統壹定會加快滅共步伐。“川普總統是美國有史以來對中國最強硬的總統，他希望和美國人民壹起開始對抗中共。”由於川普總統不幸剛剛染上中共病毒，班農先生希望他的競選團隊將競選方向與“TAKE DOWN CCP”合在壹起。

班農先生對路德先生說：“妳非常了解現在美國和中國境內的政治。川普總統在美國的很多支持者他們都很清楚，特別是由於G-News和G-TV有我們很多戰友進行了翻譯工作，讓他們知道了在川普總統和第壹夫人染病之後，中共在媒體上的壹些宣傳和嘲笑式的攻擊。現在整個美國人都意識到了，中共在川普總統得病後的壹個非常荒唐的態度。”

班農先生指出，在接下來的幾周是總統選舉的沖刺階段，川普總統壹定會主動出擊，在對中共和中共病毒上面提出更多議題。川普總統也會因為自己染病和恢復後，在這件事情上更有發言權，而他的競爭對手拜登壹直都躲在地下室裏。“接下來的幾周都是和打到中共相關的”班農先生說。

班農先生強調“TAKE DOWN CCP”是白宮的壹個整體行動，“我們都非常感謝閆麗夢博士，她上美國的各個媒體為大家解釋中共病毒的來源，包括這個病毒來自武漢，也是中共軍隊的壹個增強性實驗的生物武器。” “**閆麗夢博士的報告得到美國上層資深人士的認同，讓美國整個上層轉變了想法。**”

班農先生表示，非常認同郭文貴先生的觀點，**中共就是紙老虎！“在金融和科技方面，我們要做的事情就是推倒防火墻，同時要想盡辦法讓西方的科技公司不要再給中共提供科技；讓金融業不再給中共提供資金，特別是不要提供美元。這兩點要能做到話，中共可能在90天、最多四到五個月，中共就會投降。**”

班農先生還談到，美國科技和金融圈子裏以前還有不少人想和中共繼續勾兌，但自從川普總統被感染的事情出現後，很多人開始明白：他們不能和中共再在壹起了，中共不僅對普通的中國人是壹個威脅、對美國是壹個威脅，同時對全球整個人類絕對也是壹個很大威脅！

班農先生表示要對這次川普總統和白宮許多人感染病毒事件進行全面認真調查。

在被問到是否會用rico 法案對中共進行制裁時，班農先生回答：“這是非常有可能的，**現在有很多調查已經開始了。現在是對中共的五大家族調查，包括在金融、洗錢、在對國外政治幹涉、對政治家的幹涉、他們擁有的公司、所持有的金融資源等等，都在被調查中。**”“這個調查的靈感來自文貴先生當年對海航集團做的調查，在海航調查中我們就可以看到，中共時如何利用手中的權利進行洗錢，然後控制其他政治家等等壹系列操作。”

班農先生還提到，朱利安尼先生對這件事情是非常上心的。“朱利安尼先生在八十年代、幾十年代當檢察官的時候，成功地利用rico 法案把紐約的五大黑幫家族給端掉，現在他也是真正第壹個做到的人。**現在朱利安尼先生已經明確地說出，中共的五大家族就是黑幫家族！**”

**班農先生最後寄語**：希望所有的聽眾都能知道，在美國有非常多的人，特別是和川普總統站在壹起的人，他們非常感謝、非常欣賞新中國聯邦的人和觀眾為了自由所做的壹切。我希望妳們都能知道，妳們在美國是有真正的盟友的。在美國的盟友也都知道妳們正在經歷著什麽，尊重妳們做的所有工作，特別是包括文貴先生和G-TV所做的這些壹起。所以我們在戰鬥室每天都壹步壹步推進這個節目，為了新中國聯邦和中國人的自由。只有將中共打到，全世界所有的敵人才會消失。
