## 川普與普京發表聯合聲明 重提“易北河精神” 他倆要幹啥？
`2020-04-29 05:17 GM30` [轉載自GNews](https://gnews.org/zh-hant/188297/)

**作者：滅共日記**

4月25日，美國總統川普和俄羅斯總統普京發表一份罕見的聯合聲明，以紀念1945年二戰期間美軍和蘇軍在德國易北河橋上彙合75周年，並以此作爲兩國合作的典範。在易北河的受損橋梁上進行曆史性會面的75周年之際，當時兩國軍隊進行了握手。這一事件預示了納粹政權的決定性失敗。我們各國如何抛棄分歧、建立信任、並進行合作，以追求更偉大事業的一個例子。“易北河精神”是在我們今天努力應對21世紀最重要的挑戰時，我們向所有爲打敗法西斯主義而共同奮鬥的人們所表現出的英勇和勇氣致敬。兩個領導的這一舉措釋放出了要共建信任、共同合作的強烈信號。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/04/29051121/1-266.jpg)

從表面上這個聲明只是對曆史事件的一個紀念日，但是我在網上搜索了一下，發現美俄上一次共同紀念易北河會師，還要追溯至2010年，由時任美國總統奧巴馬和俄羅斯總統梅德韋傑夫發出，當時兩國剛簽署《新削減戰略武器條約》，但是美俄兩國關系在過去10年並未取得進步，尤其是普京在2012接替梅德韋傑夫“回鍋”擔任總統後，兩國關系反而呈現緊張態勢。2014年俄羅斯吞並克裏米亞後，美國空前強化了對俄羅斯的經濟制裁，奧巴馬和普京的私人關系也近乎崩潰。更有意思是2019年是中美建交40周年，但是兩國並未在官方層面進行任何大的紀念活動。而在此前的三十年內，雖然雙方也經常發生衝突，但對于每十年一次的紀念，雙方都非常重視。人們普遍認爲，1979年1月1日中美建交，對于中美關系與世界新格局的形成都具有重要意義。和中美建交40周年的相比，美俄易北河會師的規格顯然不具有任何可比性，而且也並非具有特別意義的紀念年份。但令人意外的是，兩國總統卻就此發布了聯合紀念聲明。這次聯合聲明釋放的信號，不禁讓人産生遐想：這次兩國大軍會在哪裏“會師”? 將聯合抗擊的是誰?

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/04/29051146/2-185.jpg)

由于在美國發現中共疫情，2020年1月底，衛生與公衆服務部部長就宣布美國進入公共衛生緊急狀態。3月13日，特朗普總統進一步宣布全國進入緊急狀態。5天之後，總統援引1950年的《國防生産法》，對急需的醫療物資實施調配。接著，在3月27日特朗普宣布，根據國防生産法，他授權衛生與公衆服務部部長，命令通用汽車公司接受、執行和優先生産抗疫所需的呼吸機。美國衛生總監告誡美國民衆，接下來幾周將會是人們生命中最嚴峻而悲傷的幾周，堪比珍珠港與9.11。事實上，這次中共病毒對美國造成的損害已經遠遠超過珍珠港和911事件。由于中共掩蓋和隱瞞疫情，引發這場全球性災難，已引發全球追責聲浪。除了美國，英國、意大利、德國、埃及、印度等國官方或民間機構都向中共提出訴訟或索賠，總金額高達43萬億美元。畢竟從俄羅斯疫情爆發後，俄羅斯政府就將“中共病毒”認定爲“生物威脅”關閉俄中邊境，暫停俄中交通，禁止中國公民入境等一些措施。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/04/29051214/3-155.jpg)

在全球追討中共的敏感時刻，川普和普京在這個時候發表聯合聲明，這似乎釋放出美俄會聯合對付中共的信號。也意味著共産黨將收到有史以來最大的索賠賬單！
