**西班牙2021年3月2日电/西喜社——**据国会山报3月1日报道，根据哈佛CAPS-Harris周一独家公布的一项最新调查，41%的选民表示不愿意接种中共病毒疫苗。

同一调查发现，59%的受访者表示愿意接种疫苗。

在那些表示不愿意接种疫苗的人中，66%的人表示担心疫苗的副作用，另有33%的人表示不相信疫苗有效。27%的人表示不担心病毒，23%的人表示应该给更多的高危人群接种，17%的人表示对健康有担忧。

“这次调查中最令人关注的一个数字是，在没有接种过疫苗的人中，有十分之四的人不想接种疫苗。这包括60%的黑人选民，”哈佛CAPS-Harris民调主任马克·佩恩(Mark Penn)告诉国会山报。

调查结果显示，强生公司开始在美国各地运送390万剂最近批准的疫苗，加入已经上市的辉瑞和摩德纳疫苗。

虽然美国通过疫苗、口罩和社交距离在抗击疫情的斗争中取得了进展，但医学专家警告说，新的病毒变种有可能使这一进展付诸东流。

“请听清楚我的话。在这种程度的情况下，随着变种的扩散，我们将完全失去我们已经获得的来之不易的成果，”疾病控制和预防中心主任罗谢尔·瓦伦斯基（Rochelle Walensky）周一说。

哈佛CAPS-Harris对2,006名注册选民的民意调查于2月23日至25日进行。它是由哈佛大学美国政治研究中心和哈里斯民调合作进行的。

完整的民调结果将于本周晚些时候在网上公布。该调查是从哈里斯小组中抽取的在线样本，并进行了加权，以反映已知的人口统计数据。作为一个有代表性的在线样本，它不报告概率置信区间。
