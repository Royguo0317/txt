近日由洛杉矶天使农场制作的一档美食灭共节目（[美食灭共节目第一期：家的味道](https://gtv.org/video/id=6029edc5be64150df856fefd)）引起了笔者的兴趣。 该节目通过对家常美食（牛肉面）的烹饪技巧，以及食材价格的介绍，从最根本的民生问题作为切入点，对比中美日常民生物价，深挖中共国粮食肉类进口黑幕。 笔者就此展开了一些相关资料信息的收集和延展。
![]()![](https://gnews.org/wp-content/uploads/2021/02/Screen-Shot-2021-02-21-at-12.53.42-AM.png)中美肉类价格对比
根据中国国官方数据，2018年，中国进口牛肉首次超过100万吨。  2019年，随着中国猪肉产量锐减，肉类进口大幅增加。 海关总署数据显示，2019年1-11月，中国牛肉进口量达到147万吨。 业内预计，全年牛肉进口量将达155万-160万吨。 尽管如此，全国牛肉消费的实际缺口可能在400万吨左右。 未来，这一缺口恐怕还会持续拉大。 据2019年，墙的资料内牛肉批发价格在68-70元/公斤左右，2018年同期在60-62元/公斤，同比上涨15%左右。 而且牛肉进口量也出现大幅增加，进口成本普遍上涨10%左右，部分低价格进口地区，甚至达到20%。 另据中共海关数据，2020年上半年中共国猪肉进口额总计428.95万元，增幅为132.9%。

由以上数据可见，中共所谓养活14亿人口的欺世谎言多么可笑。 让我们来看看本期节目中所提到的，中共如何在进口食品的贸易中获得巨大的黑金利益。

首先，中共国在明知道有很大肉类缺口的前提下，没有通过放开进口去平抑物价，相反，却征收了高额的进口环节关税：进口关税12%，增值税10%，由于关税税额也需要征收增值税，实际税收为23.2%。 比如50元/公斤的到岸价，征税后的成本已经是61.6元了，检验检疫等杂费以及吃拿卡要的成本还不算！ 中共宣称：这么做是为了保护墙内的畜牧业，这完全是谎言！ 墙内的畜牧业肉产品供不应求，何须以这种方式保护？ 真保护的话减少对这些企业的盘剥和刁难才是最有效的不是吗？ 在任何国家，税收都是对老百姓财富的抢劫！ 所以很多国家对于进口，特别是涉及民生的物资，都是免关税的！ 共匪征收的高关税就是通过老百姓不得不消费的肉食品对墙内韭菜们的一种抢劫！ 这种抢劫是导致墙内同胞很多人吃不起肉的原因之一！

由于每公斤高达十多元的税费，每个集装箱20吨计算，每个箱子二十多万元，一批次进口几百吨的贸易商在这个行业里是比较小的玩家！ 那么每批次几百万元的税费导致走私变得有利可图！ 从云南、广西、广东分别形成了肉类走私通道，这些走私管道和墙内任何一个黑恶组织一样都是相关官员参与其中，没有海关、边防、警察官员的参与，走私集团和通道都不可能存活！ 每一条走私渠道的背后都是一个或多个腐败链条！

再有，肉类能否进口也成为了威胁控制相关畜牧业国家的一个手段，中国庞大的市场需求被共匪用作筹码，胁迫这些国家在政治上支持共匪，在人权上不敢批评它，否则就利用停止购买作为惩罚手段！

并且，在全世界肉类进口国家里，中共政府一直是一个奇葩的存在，别的国家进口与否只有两个环节：一是根据肉类出口国是否存在口蹄疫等疫情的情况决定是否允许进口，另一方面是进口环节审核出口方检疫单证并进行进口国检疫，符合进口国标准即可！ 而中共国则将监管延伸到了出口国的企业，每个企业需要单独审批！ 你一定又觉得它们是为了墙内百姓的食物安全吧？ 恭喜你，你是一名合格的韭菜！ 每个企业的审批，没个三五百万人民币的黑钱，你想也不用想！ 那么花出三五百万的黑钱在先，得有多大的利润规模才可能做呢？ 每年至少千万以上吧？ 这么大利润会给不相干的人吗？ 不可能吧？ 只会给检疫高官的亲戚和朋友！ 那么如果主要的经营者是这些高管的亲戚和朋友，假如肉质量不合格的后果是什么？ 只要吃不死人当然是让韭菜吃呀！ 难道让自己受损失？ 你还觉得是为了你的安全吗？

中共之恶，毒害社会各个层面，用商鞅五术奴役中国人，把最基本的民生需求作为敛财和建立腐败链的手段，中国人只能以全世界最高的价格，吃着品质最差的肉。
