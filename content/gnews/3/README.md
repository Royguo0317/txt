## 中共在无症状感染者上做文章漏洞百出
`2020-05-16 19:26 GM06` [轉載自GNews](https://gnews.org/zh-hant/205214/)

作者：立武

5月15日，武汉疫情防控专家咨询会上，疾控中心副主任冯子健表示，武汉的无症状感染者包括隐形感染（14.7%）、既往感染治愈（84.8%）和潜伏期患者，武汉大部分是前两种，传染性很低。**冯子健这个数据是基于追踪****2508****名的密切接触者中仅有2****人转为无症状感染者续发感染率低于千分之一，低于确诊病例的密切接触者的百分之二的续发感染率。**

中共开发出无症状感染者这一名词后，又开发出三大类的无症状感染者，事实上，在15日国新办的一场新闻发布会上，中共卫健委副主任仅仅称无症状感染者包括隐性感染和潜伏期患者。这三大类无症状感染者大概的意思是，既往感染是之前感染治愈后现在复阳呈现无症状的，**其他两类是之前没有查出感染，而其中的隐性感染是感染病毒后全程都没有或轻微有症状，潜伏期患者是后面可能会出来症状的无症状感染者，仅仅为3个人。**

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/16191748/1-106.png)

问题在于

1. **中共曾在月初表示武汉复阳病例在6%左右，但现在通报的既往感染治愈的无症状感染者比例则高达84.8%，明显中共认为这复阳和既往感染治愈后成为无症状感染者不是一回事**
2. 中共并没有明确称如何认为一些无症状感染者后面无症状，而另外一些可能具有潜伏期，后面可能出现症状。**5月15日，中共尚在医学观察的无症状感染者439例，但中共很有把握最多只有3人未来可能出现症状，而其他的均不会。但这种底气来源于哪里，中共并没有透露。**
3. 密切接触者转为无症状感染者仅为2人，那这类分在既往感染显然不是，如果分在隐性感染，那么除了这2人之外的隐性感染都不是来源于密切接触者的转诊，那又来源于哪里，怎么被感染上，这不也应该考虑到传染性的范围内嘛？**仅仅依靠密切接触者的续发感染率来断定传染性，那这些没有源头的隐性感染者难道不也应该考虑传染性吗？**


![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/16191916/3-30.png)

**中共的无症状感染者明显有一部分是突然被查出来的，这种无法追踪的感染者其“上家”也许具有传染性，**但在全面复学复工的情况下，已经很难查的清楚源头在哪，这样的情况下，中共声称现在的无症状感染者感染率不强，却没有保证那些潜在的感染率强的“上家”。

更滑稽的是，中共在4月26日-30日期间以55名无症状感染者的痰液标本、咽拭子标本来进行病毒培养，结果没有培养出病毒，来说明无症状感染者传染性很低，一方面样本数量显然不够，**另一方面在2月5日的时候，中共在新闻发布会上已经明确表示，无症状感染者的咽试纸检查，核酸是阳性的，另外在4月8日中共发布的一篇文章中，明确表示，无症状感染者分为两类，且两类都是核酸阳性，且可信结论是无症状感染者是上呼吸道患者，**因此意味着至少上呼吸道是存在病毒的，至于为什么没有培养出病毒，就得问问中共自己了。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/16192003/5-28.jpg)

无症状感染者有没有传染性或者传染性高不高，凭中共一面之词，而且前后矛盾的话，是没有任何说服力的，而且加上现在许多找不到源头的感染者，**来自哪里的感染，传染性又怎么去了解都是未知数，**可以知道的是，武汉、吉林都存在无症状感染者转成确诊的患者，在这样的情况下，为了维持独裁的需要，极力降低无症状感染者的威胁将是灾难性的。
