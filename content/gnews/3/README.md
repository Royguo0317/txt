## 瘋狂的“自行車”，說你行你就行！
`2020-06-08 21:10 GM30` [轉載自GNews](https://gnews.org/zh-hant/227297/)

**作者：玉米地大姐**

搜狐網近日有一篇吹捧中國自行車的文章——暴漲22倍！歐美瘋搶中國自行車，“土豪車”脫銷，訂單排到一月後。

導讀：誰能想到，自外國人瘋搶衛生紙、口罩、洗手液之後，又一中國産品在歐美被瘋搶！萬萬沒想到，是中國的自行車！

喜提熱搜，網友戲稱，ofo的機會來了，國內共享單車可以回收轉賣。還有網友說，疫情意外促進了全球的節能減排工作，後疫情時代騎行將成爲熱潮。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/08205354/1-72.jpg)

就在股市裏的韭菜沈浸于鳳凰自行車股價暴漲之際，專家們卻毫不客氣地迎頭潑了一盆涼水。

6月8日證券日報》發表一篇采訪報道：上海鳳凰股價迎上漲“三連板” 自行車行業或只是“回光返照”

近日，有媒體稱“歐美瘋搶中國自行車”，受此消息影響，上海鳳凰、中路股份等自行車企業出現了股票上漲的現象，而上海鳳凰又因出産“擺攤神器”迎來了上漲三連板。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/08205435/2-34.png)

香頌資本執行董事沈萌對記者表示，歐美長時間以來一直注重單車出行，所以是不是出現了“瘋搶”、是不是因爲疫情原因導致單車銷量上升還並不能確認，而且目前在單車國際化銷售中，中國台灣的單車企業占主流，其産品從設計到功能等各方面不斷推陳出新，與中國大陸企業低附加值的生産不同。

國內許多自行車企業不注重研發、經營基本靠跟風，沒有下沈打牢基礎，因此發展隨風擺、缺乏自主性，業績也就隨著市場熱點波動。所以，如果不能加大科技附加值含量，只在産業鏈最底端生産制造，這個行業是難以出現轉機的。

透鏡公司研究創始人況玉清對記者表示，因疫情發生的臨時性需求應該很難持續，自行車行業在國內，還得靠共享單車的訂單，個人需求並不大。

上述投行人士則認爲，目前來看，自行車行業整體趨勢不會改變。“共享單車更多的是衝擊自行車低端市場，單車企業做高端産品也好，轉型其他産業也好都是出路，但是要沈下心來做”。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/08205453/3-29.jpg)

這回專家們總算說了些實話，昔日的自行車王國引以爲傲的頭牌産品鳳凰牌自行車，早無當年“三轉一響”的風光，可憐地活在大爺大媽的記憶裏。倒是後起之秀來自台灣的“捷安特”等幾個品牌深受年輕人的青睐。

國産單車最奇葩是傻大黑粗的造型多年不變，侯寶林相聲《夜行記》自嘲他買的那台自行車，除了鈴不響，剩下哪都響。雖是戲虐之詞，卻也諷刺了自行車的質量堪憂。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/08205544/4-26.jpg)

知乎有篇文章：鳳凰自行車爲什麽這麽便宜？

下面跟帖者全是吐槽鳳凰自行車的質量，茲摘錄幾段，與大家分享：

因爲質量很次很次，我是親身體驗，買了3個多月了，剛開始到手就發現刹車片不平，齒輪不平，腳蹬子一騎就有摩擦的感覺。這些問題問客服只給個修理視頻，還要自己修 扳子鉗子買了一大堆最後也沒弄好。

車子是真爛，買了輛山地車2個月，車子放過道，整個過道都是塑料味，把手不能碰，碰了手上的塑料味怎麽都洗不掉

我看舍友買了自行車方便，我也買了個鳳凰牌，代步的小山地，放外面，半個月全是鏽，刹車簡直爆炸。

一輛永久單速車，讓我積累了多少修車經驗出來！這是多麽寶貴的一課啊！然而並沒有什麽卵用，那些落後過時的技術，現在早就不用了，就永久、鳳凰還有，真坑人！

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/08205611/6-7.jpg)

過去說鳳凰自行車如何如何牛，那是因爲中共壟斷市場，指定全國人民消費鳳凰等幾個品牌的自行車，無形中給大衆造成了錯覺，以爲全世界就鳳凰、永久牌自行車最好，天下第一。

文貴先生頭幾天視頻說，他有個做自行車生意的朋友把工廠遷到越南賺了大錢，客戶唯一的要求是，不能是中國生産的自行車。

我們知道，世界十大品牌自行車沒有一個是中國造，至于疫情期間出口到歐美的自行車，筆者猜測大多數是台灣生産。中共輿論混淆概念，統統說成中國自行車，爲拉升股市造勢。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/08205652/7-7.jpg)

中國股市好比一個雞窩，雞和蛋早被盜國賊掏空了，只剩一窩雞毛。爲了穩住韭菜們別跑，官媒意淫出個鳳凰自行車暴漲22倍。意淫得太過玄幻就連專家都看不下去了，用了“回光返照”這樣的狠詞加以嘲諷。

專家說得沒錯，的確是“回光返照”不僅僅是鳳凰自行車，還有中共的整個經濟體系，全靠盜國賊的擀面杖硬撐著，一旦春藥過勁疲軟，必然是轟然倒塌灰飛煙滅。
