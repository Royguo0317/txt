##  **印度-日本-澳大利亞供應鏈進入運行以抵制中共國**

**梗概**

經濟時報獲悉，該倡議最早由日本提出，目前已初具規模。三國商務和貿易部長的首次會議日期正在商定中，將於下周公布。

新德里：由於北京咄咄逼人的政治和軍事行為，印度、日本和澳大利亞已經開始討論發起三邊供應鏈復原倡議（SCRI），以減少對中共國的依賴。

[!\[\](https://spark.adobe.com/page/iPU9Qsz2e526x/images/9e98e389-bab3-4fcf-a645-2d8f03252b67.jpg?asset_id=b5143e47-74b4-4063-a21c-78732526f182&amp;img_etag=%22cf53c11ececc3af55dc6b86b8ffb1996%22&amp;size=2560)](https://spark.adobe.com/page/iPU9Qsz2e526x/images/9e98e389-bab3-4fcf-a645-2d8f03252b67.jpg?asset_id=b5143e47-74b4-4063-a21c-78732526f182&amp;img_etag=%22cf53c11ececc3af55dc6b86b8ffb1996%22&amp;size=1024)

經濟時報獲悉，該倡議最早由日本提出，目前已初具規模。三國商務和貿易部長的首次會議日期正在商定中，將於下周公布。

據知情人士透露，最近，日本通過其經濟產業部與印度進行了接觸，並催促印度抓緊推進這一舉措。消息人士稱，東京希望在11月前啟動三邊供應鏈復原倡議（SCRI）。

印度政府正在相當認真地推進這項建議，特別是考慮到中國在拉達克實際控制線上的侵略性舉動。消息人士說，通常情況下，新德里會謹慎考慮任何此類建議，因為這將被視為一個反中共國的聯盟。

這一次，印度政府似乎已經接受了最高層的號召，成為全球供應鏈的一部分，從而成為中共國的替代者。

這個話題也是莫迪總理週六獨立日演講的重要主題之一，他表示，企業已經開始將印度視為可能的”供應鏈中心”，因此，現在，印度也必須”為世界製造”。

日本提案的雙重目標是吸引外國直接投資，將印太地區變成”經濟動能中心”，並在夥伴國之間建立互補的關係。
[!\[\](https://spark.adobe.com/page/iPU9Qsz2e526x/images/1842ba0a-e210-4a5a-90f8-beb9a076277c.jpg?asset_id=95f9bec6-de8a-47bb-9eba-34c54eb1ded1&amp;img_etag=%22bcb560802739dc91c95a0117d7eaac16%22&amp;size=1024)](https://spark.adobe.com/page/iPU9Qsz2e526x/images/1842ba0a-e210-4a5a-90f8-beb9a076277c.jpg?asset_id=95f9bec6-de8a-47bb-9eba-34c54eb1ded1&amp;img_etag=%22bcb560802739dc91c95a0117d7eaac16%22&amp;size=1024)
消息人士稱，（各國的）想法是，在印度-日本-澳大利亞就此事達成共識後，向東盟（東南亞國家聯盟）拋出此倡議。

首先，目標是以現有的雙邊供應鍊網絡為基礎制定一項計劃，例如，印日建立工業競爭夥伴關係，負責日本公司在印度的設立。

了解初步討論細節的人士告訴經濟時報，最終目的是提高整個印太地區的供應鏈彈性，通過這樣的努力，可以發展一個穩健的、安全的經濟體。

消息人士稱，三邊供應鏈復原倡（SCRI）是針對企業和經濟體對中共國政治行為以及其可能導致供應鏈中斷的的擔憂的一個直接反應。

日本首相安倍晉三在中共病毒Covid-19爆發後，已經設立了一個20億美元的基金來幫助日本企業從中共國轉移回來。澳大利亞和美國，出於對（中共國）日益惡化的安全和透明度的擔憂，已經達成了一項雄心勃勃的協議，以建立一個被稱為”無中共國”的稀土材料供應鏈。
