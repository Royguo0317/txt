2020年1月30日，世界卫生组织（WHO）宣布中共病毒升级为全球公共卫生紧急事件。

WHO主席谭德赛在[当天的发布会](https://www.youtube.com/watch?v=NeDeOxb4ofg)上讲到：“这个声明的主要原因，不是因为正在中国发生（传播），而是因为正在其他国家（传播）。我们最大的担忧，是病毒传播到卫生系统比较薄弱的、准备不充分的国家。

我要明确的是，这个声明不是不信任中国，相反的，WHO依然对中国控制疫情的能力很有信心。

我要再重复一次：我要明确的是，这个声明不是不信任中国，相反的，WHO依然对中国控制疫情的能力很有信心。”

根据[Benzinga新闻报道](https://www.benzinga.com/government/20/01/15222878/who-declares-coronavirus-a-public-health-emergency-but-doesnt-urge-travel-ban)，谭德赛赞扬了中共国的抗疫效果，并反对任何形式的对中共国的旅行禁令。谭德赛还指出，一些国际航空公司减少或者停止了中国的航班，是因为需求减少了，而不是因为病毒本身。

谭德赛在发布会上说，截至当天，全球已经有7834例病例，几乎全部都在中国。目前已经导致170人死亡。

根据[Worldometers.info网站](https://www.worldometers.info/coronavirus/worldwide-graphs/%23total-cases)数据，1月30日全球感染人数突破一万人，为10,263人。
