## 【大象快讯】美团创始人王兴捐赠价值23亿美元股份
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1299025/)

华尔街日报2021-6-5日报道：在中国加强对科技行业监管的背景下，餐饮外卖巨头美团的创始人王兴将价值逾20亿美元的公司股份捐给了自己的慈善基金会。

周四晚间的证券交易所文件显示，身为该公司首席执行官的亿万富翁王兴向其基金会转让了约 5730 万股股票。以当日收盘股价307.40港元，折合约39.62美元计算，捐赠价值约23亿美元。该公司和其他中国科技巨头受到中国监管机构的压力，他们正在推动这些公司公平竞争并保护消费者和其他利益相关者的利益。美团表示，王先生出于慈善原因做出了个人资产分配决定，资金将用于教育和科研。“这一决定并不反映他对美团业务的奉献精神有任何变化，”它在一份声明中说。王先生捐赠的股份相当于他之前持有的美团股份总数的 10% 左右。王先生对公司的控制大于他的经济股份，因为他持有该公司的超级投票权。另一份文件显示，在捐赠的同一天，王先生的基金会出于慈善原因将约 935 万股股份（价值约 3.71 亿美元）转让给了第三方。

其他中国科技企业家也热衷于慈善事业。去年，拼多多创始人黄科林和他的创始团队将公司2.37%的股份捐赠给了一个名为星夜基金会的机构。今年3月，黄先生卸任拼多多董事长一职，以追求个人对生命科学的兴趣。一天后，星夜基金会承诺在三到五年内向中国浙江大学捐赠1 亿美元，黄先生在那里攻读本科学位。这些资金将支持生物医学科学、农业和食品领域的基础研究。
