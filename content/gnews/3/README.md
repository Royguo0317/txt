## 青島公安局長17日稱確診出租司機載客完成追蹤能確保無虞嗎
`2020-10-17 09:26 NewFOC` [轉載自GNews](https://gnews.org/zh-hant/430275/)

作者：Maarago

在前文[青島真的是因CT室引發醫院院內聚集性疫情未發生社區傳播嗎？](https://gnews.org/zh-hans/427480/)引用的[梨視頻](https://weibo.com/pearvideo) [2020年10月16日 11:34](https://weibo.com/6004281123/JpqVlin37?from=page_1002066004281123_profile&amp;wvr=6&amp;mod=weibotime) 來自 [微博 weibo.com](https://app.weibo.com/t/feed/6vtZb0) ，山東省疾控中心馬立新剛剛於2020年10月16日宣佈青島疫情的源闊大為大港兩名感染者，為醫院聚集性疫情，系青島港疫情感染者住院期間與普通病區患者共用CT室引發。這一結論剛剛過了一天，據[青島確診計程車司機載客231人 已全部追蹤到位](https://baijiahao.baidu.com/s?id=1680787215224532889&amp;wfr=spider&amp;for=pc)（央視新聞發佈時間：10-1717:03中央電視臺新聞中心新媒體官方帳號）

［10月17日，山東青島市召開新聞發佈會。會上，青島市人民政府副市長、市公安局局長隋汝文對確診患者計程車司機邵某某的流調情況進行了通報。隋汝文介紹，當前已查明，自9月29日16時至10月10日22時期間，邵某某共載客183次。截至10月16日中午，共追蹤到231名乘客，其中青島本市196人，省內外市11人，省外24人。隋汝文表示，目前已對在青島本市的196人，逐人落實了隔離管控措施。對省內外市的11人和省外的24人也已全部採取了流調、隔離、檢測、管控等措施。］

為什麼追查期限界定於2020年9月29日16時至2020年10月10日22時期間邵某某的載客人次呢？我相信那是因為2020年就是山東省疾控中心馬立新所稱的“住院期間與普通病區患者共用CT室引發”感染的發生日期，也就是說邵某某的妻子江某某應該是2020年9月29日作為護工進入了2020年9月28日被青島大港區兩名新冠病毒隔離患者污染的CT室，從而開始了這一輪在青島的新冠病毒擴散，可是我們非常不理解的是為什麼馬立新和青島市公安局長隋汝文都對具體是哪一天青島胸科醫院的CT室被新冠病毒污染避口不談呢？作為這起重大的新冠病毒擴散事件，界定CT室污染日期是非常重要的，既然馬立新和隋汝文均避口不談具體污染日期，那麼作為公眾的一員，我只能從隋汝文發佈的資訊來推測CT室污染日期。

為什麼這個污染日期非常重要？因為確定了準確的污染日期才能對潛在的新冠病毒感染者和他們的密切接觸者進行跟蹤，那麼目前來看青島市公安採取的措施夠嗎？當然是不夠的，因為［自9月29日16時至10月10日22時期間，邵某某共載客183次。截至10月16日中午，共追蹤到231名乘客，其中青島本市196人，省內外市11人，省外24人。隋汝文表示，目前已對在青島本市的196人，逐人落實了隔離管控措施。對省內外市的11人和省外的24人也已全部採取了流調、隔離、檢測、管控等措施。］，這231名乘客下車後又接觸了誰？在他們接觸了的這些人在隨後又接觸了誰？基於這樣的疑問，青島市公安局長2020年10月17日的通報不但不能安撫恐懼，反而會加重公眾的恐懼。因為目前他們只追蹤了邵某某的載客範圍，並沒有追蹤邵某某和他妻子江某某自2020年9月29日16時至2020年10月10日22時的其他活動範圍及密切接觸者，因為他們夫妻二人不是同時在一輛計程車上，其活動範圍也不僅僅是那輛正在運營的計程車。還有更加令人恐懼的是在筆者推測的2020年9月29日，那些先後進入青島胸科醫院被污染的CT室的人，他們自2020年9月29日至2020年10月10日都去了哪裡、接觸了誰？尤其是那位與他們夫妻二人同日核酸檢測陽性的另一位元：**韓某某，男，****58****歲，待業，居住在市北區洛陽路街道辦事處舞陽路****7****號。****9****月****9****日韓某某因肺結核在市胸科醫院住院治療，****10****月****4****日出院。因韓某某準備****10****月****12****日再次到市胸科醫院住院治療，於****10****月****10****日上午就近到市中心醫院做核酸檢測，結果為陽性。****10****月****11****日市疾控中心核酸複檢陽性，結合實驗室檢測結果和臨床表現，專家組確定為無症狀感染者，轉至定點醫院隔離觀察。**他是10月4日出院，按馬立新的說法，他也是由於進入了前一天被污染的青島市胸科醫院CT室被感染的，考慮到他和江某某都是在普通病房而不是隔離病房，那麼在他2020年10月4日出院前他接觸了多少人，在他2020年10月4日出院至2020年10月10日核酸複檢陽性期間，他一共接觸了多少人，他接觸的這些人又接觸了多少人？
