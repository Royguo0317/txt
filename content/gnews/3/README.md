## 百姓的储蓄如何安全 由中共国小银行的倒闭说起

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/04231319/pexels-pixabay-210574-scaled.jpg)图片来源：Pexels免费照片素材库
今天文贵先生直播中提到中共国国内4个小银行破产的事情确实非常重要，需要引起我们更多的注意。

4家先后破产的银行分别是海南发展银行、汕头商业银行、河北肃宁上村农村信用社、包头商业银行。如果说前三家主要是因为高息揽存而引起的资不抵债，而最近倒闭的包商银行则主要是被大股东大量挪用资金而引起的资金链断裂。

14年间，包商银行的大股东“明天系”通过注册209家空壳公司，以347笔借款的方式套取信贷资金，对包商银行形成的占款高达1560亿，最终都成为不良贷款。

在目前中共国遭受外资企业大量撤退，国外采购对象大量转移到其它国家，中共国外贸大趋势的衰退已经不可避免。另外占用银行大量资金的房地产市场也在快速降温，在外贸和房地产两大引擎衰退的情况下，已发贷款无法收回的情况将大量发生，不仅仅在于小银行，大银行的情况也非常危险，文贵先生提到的民生银行、招商银行、其它国有银行及国有不良资产处置公司都在此列，如果情况继续恶化，几个国有大银行也将难逃此劫。

除了上面这些情况外，更为深层的问题是权贵与银行资金剪不断理还乱的混乱局面，这是在专制政权中无法摆脱的怪圈。高官及其子女通过隐秘的手法攫取银行资金。比如刘呈杰、贯军、姚庆等就获得了大量的银行资金，而且很多资金甚至是银行主动送上门的。这种贷款投放毫无监督，也无从监督，最终大概率成为收不回的坏账，由全体老百姓买单。

可以说，在当今的中共国，中国人民银行、证监会、保监会、外汇管理局等，哪个常委的子女来调动资金它们能挡住？它们谁也挡不住，而且不但挡不住，它们还主动卖身求荣！中国人民银行就是最大的抢劫老百姓资金的银行，而且它的抢劫还是“依法抢劫”。

目前，中共宣布的几家银行的倒闭，是为未来更多更大的银行倒闭做准备。它们想让老百姓习惯这样的倒闭，如同温水煮青蛙，并且让老百姓把损失责任怪到自己身上，为中共的掠夺和腐败脱身。

在前三十年中共国经济高速发展的情况下，银行坏账的问题还不是十分突出，可以用新账还旧账的方式来掩盖。但是，现在整体经济下滑，潮水退去，裸泳者就再也无法遮掩。可以预料银行倒闭的事情将愈演愈烈。而目前出台的政策是50万以下由国家兜底，50万以上则只能自求多福了。而在未来，估计更严格的赔付政策也会出台。

鉴于此，如果手头有资金的老板们，奉劝你们慎重考虑资金的存放。存在中共国的银行已经不是保险箱，甚至将是个陷阱，到时候可能血本无归。因此，尽量把资金换成外币，而且不能存在中共国的银行，或者转移出境，可能才是安全的途径。
