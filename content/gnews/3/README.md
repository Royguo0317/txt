3月12日塔斯社莫斯科报道，美国及其盟友正计划发起一场大规模的宣传活动，以诋毁俄罗斯中共病毒的疫苗发展，因为对俄罗斯《人造卫星五号（Sputnik V）》疫苗的需求越来越大。克里姆林宫的一位高级消息人士周五告诉塔斯社。

“在全球包括欧洲国家对俄罗斯的《人造卫星五号》疫苗的需求日益增长的背景下，根据我国情报部门的说法，美国及其盟友正计划开展大规模的宣传活动，旨在对国内在抗击中共病毒感染传播领域的科学发展形成偏见。”他指出。

他还强调，与此同时，美国及其盟友还计划爆料有关俄罗斯专家在疫苗接种和免疫学领域无能的“揭发材料”。

消息人士还指出，反俄宣传活动的目标主要是已注册《人造卫星五号》用于应急的欧洲国家。这些国家是匈牙利、斯洛伐克、塞尔维亚、黑山、圣马力诺和北马其顿。“在西方，这些国家对我国疫苗的认可，已经引起了反俄宣传的高潮。特别是荷兰媒体把我们的《人造卫星五号》不是作为治疗中共病毒的良药，而是作为‘克里姆林宫分裂欧盟队伍的杠杆’和‘莫斯科的宣传工具’。匈牙利被列举为例子，它‘屈服于俄罗斯的伎俩’，在没有与布鲁塞尔协调的情况下就开始给公民接种这疫苗。”塔斯社的对话者说。2020年11月，匈牙利成为第一个接收《人造卫星五号》样品进行研究的欧盟国家。今年2月，俄罗斯疫苗终于获得匈牙利专业医疗机构的批准，之后开始在该国使用。
