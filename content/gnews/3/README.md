**萬物均有定時**

“凡事都有定期，天下萬物都有定時。生有時，死有時；殺戮有時，醫治有時；拆毀有時，建造有時；哭有時，笑有時；哀慟有時，跳舞有時；爭戰有時…我見神叫世人勞苦，使他們在其中受經練。神造萬物，各按其時成為美好，又將永生安置在世人心裡。然而神從始至終的作為，人不能參透。”【傳道書3:1- 11】的經文把人生滄桑，世事變更的規律以及背後的法則做了隱喻性總結。其中神的作為人不能參透這句話，隱性地表達了宇宙萬物的主宰和進程高於人的認知界限和範圍。主宰天地的萬佛萬神會按照預定的時間和節點來成就萬事，並非按照人的意志而行。故此，中共邪魔的消亡和新中國聯邦亮相世界舞台皆有來自上天的回應和暗示。

**仰望星空獲天啟**

縱觀世界五千年的歷史，偉大人物的誕生，重大事件發生以先，神透過宇宙中天體運行位置顯露出種種的跡象。聖經新約的四大福音書從不同角度記錄了耶穌的生平事蹟和偉大思想。四福音裡有關耶穌誕生的記載，都不約而同地談到了東方的博士們朝拜聖嬰的事蹟。根據《馬太福音》2章1至12節的記載，耶穌降生後兩年左右，有幾個東方的（可能是古波斯，現伊朗地區）博士夜觀天象，看到了一顆奇異的星星，他們就背井離鄉，在那顆星的引領下，前往異國他鄉的伯利恆去尋找新生王。中文和合譯本的【聖經】翻譯的“博士”一般解釋為研究星象的學者、智者，包括預言家、星相家、解夢的人。雖然那幾個東方博士並不是上帝的選民，但這些博士們卻通過仰望星空，觀看天象領悟到了上帝的啟示。

而在耶穌降生前後，漢孝哀帝建平二年（公元前5年），中國天文學家也觀察並記錄了幾個不同尋常的天文現象。漢哀帝建平二年二月。彗星出牽牛七十餘日。傳曰：牽牛日月五星所從起。歷數之遠。三正之始。彗而出之。改更之像也。其出久者。為其事大也。 （《漢書·天文志》卷26頁1312）。翻譯過來就是：建平第二年的第二個月，彗星出牽牛70餘日。

彗星的出現意味著新的要代替舊的。牽牛、太陽、月亮和其它五星在運動，意味著人類新紀元的開始。一顆彗星出現70多天的時間，這是很罕見的現象。這顆彗星和稍縱即逝的普通彗星迥然不同，它明亮長達70餘日。所以被評為“慧而出之，改更之像也。其出久者, 為其事大也”。

**火星逆行白羊座**

在2020年的下半年，特別是9月9日至11月13日，火星進入了逆行期。火星大約每兩年逆行一次，但鮮少在白羊座逆行。火星上一次在白羊座逆行是1988年，下一次要等2067年。今年的火星將比1988年更強，它在1988年的行進中沒有與任何外行星形成困難相位。而上一次火星逆行白羊座的1988年，發生了柏林牆的坍塌，次年1989年，龐然大物的蘇聯解體。
