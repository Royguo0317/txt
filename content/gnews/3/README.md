## 親共的沉重代價：俄羅斯病毒肆虐
`2020-06-17 20:22 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/237183/)

![](https://gnews.org/wp-content/uploads/2020/06/Picture-1-78.png)
大紀元時報(The Epoch Times)16日發表評論文章，分析了為何俄羅斯在中共病毒疫情控制初見成效之後，隨後發生大逆轉，情況直下。到上週末為止，俄羅斯居然有超過53萬人感染中共冠狀病毒，僅次於美國和巴西，成為世界感染率第三高的國家。

文章首先介紹了在病毒大流行的初期，俄羅斯採取了各種迅速而果斷的措施來有效控制病毒在俄羅斯境內的傳播。早在1月31日，俄羅斯就關閉了與中共國的16個過境點，並停止向中共國簽發電子簽證，同時也暫停了俄羅斯往返中共國以及朝鮮之間的火車。中俄之間的航班​​也幾乎全面停止飛行。美國之音曾經稱俄羅斯對病毒大流行所採取的控制措施是最為嚴格的。因此截至3月16日，整個俄羅斯只有不到100例病例。

但是，好景不長。隨著俄羅斯開始恢復與中共的關係，並倡導兩國合作。從3月底之後，確診病例的數量迅速增加。到5月初，連續十天每天新增感染病例10,000多例。評論員指出俄羅斯的親共立場直接導致該國對病毒防禦的全面崩潰。

**普京與中共之間的三次通話**

中共病毒開始在中共國以外的地區廣泛傳播之後，普京總統與中共高層進行了三次電話交談。俄羅斯總統府和新華社均報導了這三場對話。大紀元評論員通過這三次對話，分析了俄羅斯與中共之間微妙的關係變化，並且結合這一變化解釋了俄羅斯國內病情的怪異走勢。

第一次通話發生在3月19日，這可以稱之為一場遲到的對話。中共高層在與其他十幾位國家領導人交談了之後才與普京通電話。這場遲來的對話發生的背景是：俄羅斯在沒有提前知會中共的情形之下，迅速地採取單邊行動，關閉了與中共國相鄰的邊界口岸；官方與民間同時表現出對中共國冷淡的態度。新華社與俄羅斯總統府官方網站對這第一次通話的報導為—俄羅斯高度評價中共在遏制中共冠狀病毒蔓延所取得的成就。新華社另外報導說俄羅斯希望繼續與中共互相支持與合作，深化中俄全面戰略合作。

第二次通話發生在4月16日。新華社報導說，俄羅斯認為有人試圖在病毒起源上另做文章，同時表示俄羅斯願意在中共冠狀病毒大流行控制等多個領域與中共加強交流與合作。俄羅斯總統府官方網站則對電話交談內容進行了更為詳細的報導，普京說“與中共相互支持，共同來抵抗這一全球威脅，進一步證明了俄中全面戰略夥伴關係的特殊性。俄羅斯承諾將進一步加強在這一領域的合作。中共國在抵抗病毒流行中所採取的有效行動應得到讚揚。國際社會對中共沒有及時發布關於病毒真實信息的指控只會起到適得其反的效果。

第三次通話是在5月8日，中共向普京祝賀衛國戰爭勝利75週年。新華社報導說，普京提到：“有幾股勢力想把病毒大流行歸咎於中共國。俄羅斯反對他們的這一行為，並將與中共國站在一起。”然而，俄羅斯總統府沒有這一聲明的報導記錄。

儘管中俄雙方對三輪電話交談的內容與共識有不同的看法，但普京親共的立場已經非常鮮明。在對世界格局有影響力的國家中，目前他是唯一的公開反對美國、 反對將病毒大流行歸咎於中共的國家領導人。當中共面臨全世界的聲討與譴責時，俄羅斯的支持對中共國政權具有重要意義。但同時，自3月下旬以來，俄羅斯的疫情也急劇惡化。可以看出，這絕非偶然。

**俄羅斯外長的聲明**

俄羅斯的親共立場不僅體現在普京與中共近期的三次對話上，也體現在其外交部對世界衛生組織的態度上。 4月28日舉行的金磚國家（巴西，俄羅斯，印度，中國，南非）外交部長電視電話會議上，俄羅斯外交大臣談到世界衛生組織時說：世界衛生組織的作用是至關重要的，它提供了一個收集來自各個地區疫情信息無與倫比的平台。不論其他國家是否退出世衛組織，俄羅斯都將繼續支持世界衛生組織。

俄羅斯外交大臣還高度讚揚了俄羅斯與中共之間共同對抗中共冠狀病毒的合作，並說：俄羅斯在爆發初期就向武漢提供了援助，而現在，中共正在幫助包括俄羅斯遏制病毒的蔓延。

世界衛生組織在此次病毒大流行中幫助中共向國際社會隱瞞疫情真相，已經成為眾所周知的事情。在全球100多個國家聯名要求深度調查世衛組織如何與中共勾兌之時，俄羅斯卻高調稱讚世衛組織，稱讚中俄合作，公然與中共站在一起。

**俄羅斯與中共的歷史**

俄羅斯在前蘇聯解體之前經歷了73年的共產黨統治。前蘇聯在中共早期對其提供了許多支持。儘管俄羅斯在前蘇聯解體之後，放棄了共產主義制度，並在解體之後一度積極向西方民主國家靠攏，將中共暫時擱置在一邊。然而不幸的是，俄羅斯始終未能擺脫共產主義病毒的影響。到1992年底，俄羅斯與中共恢復了外交和貿易互動。

普京在2001年上台後與中共政府簽署了《 2001年中俄友好條約》，開始了戰略夥伴關係，並在2010年升級為全面戰略合作夥伴關係。近幾年來，中共與莫斯科的交往更加頻繁。 2017年7月，中俄海軍在波羅的海舉行了首次聯合軍事演習。 2018年9月，中共國參加了俄羅斯年度軍事演習。這些行動均引起了外界的極大關注。

具有諷刺意味的是普京總統在2017年10月30日親自揭幕了名為“悲傷之牆” 的紀念碑，用以紀念前蘇聯時期遭受政治迫害的受害者。他發表講話說，我們的責任是不要讓那段歷史被遺忘。聽起來普京似乎很清楚共產黨的邪惡本質。然而在行動上普京卻選擇與中共保持著密切的伙伴關係。中共已經取代了蘇共成為這個地球上共產黨的老大，繼續以謊言和暴力迫害人民。普京事實上正在帶領俄羅斯靠攏那個已經被俄羅斯人民摒棄了的共產主義政權。

中共經常吹噓中俄之間的“互信程度”，“合作水平”和“戰略價值”。實際上，這種黑暗而邪惡的關係是由不道德的利益來驅動，偏離了普遍的價值觀。對於任何政府而言，與中共建立良好關係只會帶來無盡的麻煩。俄羅斯大流行爆發的戲劇性逆轉就是一個深刻的教訓。

評：

大紀元評論員從分析俄羅斯自疫情爆發以來對中共態度的轉變的角度，完美地解釋了為何在此次疫情中俄羅斯從最初的歐洲最為安全的國家，三個月之後發展為歐洲疫情噩夢的國度。可悲的是俄羅斯在擺脫了前蘇聯的極權統治之後，卻無法擺脫共產主義餘毒對人民思想的桎梏。所以文貴先生說，中共政權可能會在瞬間脆斷，然而除去中共餘毒留在人們心中的心霾卻會曠日持久、任重道遠。
