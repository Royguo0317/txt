- **10.30．新华网 新华国际时评：美国推行文化霸权，妨害多元文明共存**


“强化美国文化作为世界各国文化‘榜样’的地位，是美国维持霸权所必需实施的战略”——这是美国前国家安全事务助理兹比格纽·布热津斯基对美文化扩张战略的高度概括；“成功的帝国主义政策，不是征服和掌握经济，而是控制人的头脑”——美国政治学家汉斯·摩根索如此阐释美热衷于文化扩张的底层逻辑。

自二战以来，特别是冷战结束之后，美国以称霸世界的经济、军事实力为基础，通过文化渗透向全球输出以美国价值观为基础的意识形态，力图让服务于美国利益的所谓“民主”“人权”等观念“占领”其他国家民众的大脑，以实现美国文化在全球的“一家独大”。这种文化霸权行为，损害其他国家的文化独立性，阻碍世界多元文明交流，危害全球文化安全，对世界和平发展形成重大威胁。

美国对外通过影视产品、图书、各种媒体以及资助非营利性文化机构等方式“嵌入”“配售”美国价值观和生活方式，以此打造出一个以美式文化为主导的文化和舆论空间，推动文化霸权。美国学者约翰·耶马在其《世界的美国化》一文中指出，关于文化扩张，“美国真正的‘武器’是好莱坞的电影业、麦迪逊人街的形象设计厂和马特尔公司、可口可乐公司的生产线”。

美国大力推行其价值观、维护其文化霸权的同时，还对很多其他非西方国家的历史文化和制度等进行“妖魔化”，力图使这些国家的民众丧失对自身文化的自信。多年来全球多地频发的“颜色革命”中，美就充分利用操控当地媒体等方式，抓住当事国经济社会困局大做文章，制造关于执政当局的各种负面消息，让民众尤其是年轻人对自己的国家失去信心，将美式价值观奉为圭臬。美国未来学家阿尔文·托夫勒就认为，向对方国家的年轻人灌输美国的价值观，将达到“暴力和金钱无法征服的目的”。

美国通过向全球各地推行其价值观，竭力在各国培养美国价值观的“同路人”，以实现更好维护其在全球的政治、经济、军事等领域霸权的目的。正如美国著名学者约瑟夫·奈所说：“美国面临的关键问题不在于是否拥有21世纪最充足的资源供应，而在于它在多大程度上可以控制政治环境，使其他国家按照美国的意愿行事。”如今，美国的文化霸权战略随着互联网的发展不断升级，其攻势显得更隐蔽，攻击范围更广，因而更为值得警惕。

许多年来，美国在欧亚等不少国家推行所谓“自由、民主”，助推“颜色革命”，多国由此陷入经年不息的政治和社会动荡。美国强行输出其价值观，破坏他国固有文化和政治生态，许多国家文化安全受到严重威胁。美方一些政治势力还刻意鼓吹所谓“文明冲突”，对“挑战”美国文化影响力的其他文化进行遏制打压。

美国推行文化霸权，对世界和平和安全以及世界文明健康发展形成巨大威胁。对此，世界各国需保持清醒认知，认清美式“民主、自由”等“温情”面纱背后的霸权本质，不断增强自身文化自信，大力加强国际间的文明交流，让推行文化霸权的图谋破产，让互尊互鉴成为世界文明版图上的主流风景。
