## 【呐喊】教廷国务卿帕罗林与中梵协议
` 纽约香草山MOS03` [轉載自GNews](https://gnews.org/zh-hans/1098617/)

作者：纽约香草山佛教部 369文驹



![]()![](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-109.jpg)



据BBC报道，中（共）国与梵蒂冈在2018年9月22日签署有关主教任命的临时协议，协议自10月22日起生效，效期两年。

2020年协议到期后，中梵再度续签了该协议。但是，在中共打压、迫害宗教人士，拆毁、烧毁天主教堂，践踏宗教信仰活动愈演愈烈的背景下，梵蒂冈续签该协议引发各界的关注与不满。

![]()![](https://gnews.org/wp-content/uploads/2021/04/image-246.png)图片来源：维基百科

该报道援引美国前国务卿蓬佩奥于9月18日在美国宗教杂志《第一要务》（First Things）上的发文称，中共对包括天主教徒在内的各种宗教信徒进行了“可怕的迫害”，自从梵蒂冈与北京（中共）达成协议以来，中国天主教徒的处境恶化了。现在，中国人民比以往任何时候都更需要梵蒂冈的道德见证和权威，来支持中国的宗教信徒，但是该协议的条款从未被公开披露。

值得关注的是，在中共正式宣布之前，教廷国务卿帕罗林（Pietro Parolin）枢机主教已经向媒体透露，证实中梵将续签主教任命临时协议。

另外，据法广消息，帕罗林在去年10月纪念意大利传教士抵达中国传教150周年的演讲中，针对中梵协议的批评声音作出了教廷的权威回应。他表示：

第一，梵蒂冈对中（共）国的对话策略是一脉相承的。

第二，中梵协议只是关于主教任命，完全没有触及中（共）国教会生活的其他层面，也没有涉及政治议题。

第三，这项协议值得续签，因为为期两年的协议太短了，很难评估其价值。

![]()![](https://gnews.org/wp-content/uploads/2021/04/image-247.png)图片来源：RFI

报道引用了香港荣休枢机主教陈日君9日博客发布的题为“因为爱真理，我无法保持沉默”一文。该文揭露，帕罗林是瞪眼撒谎，文章称：

1、帕罗林知道自己在说谎，也知道我知晓他是个骗子，会将真相公诸于世。但帕罗林还是无耻又大胆地说谎，如今他还有什么不敢做？他恐怕连违背自己的良心也不怕。担忧帕罗林连信仰都没有。

2、帕罗林在一次纪念前任卡萨罗尼枢机主教（Agostino Casaroli)）时提出，“选择主教不是选斗兽场的角斗士”，不期盼主教挑战政府威权或走上政治舞台。卡萨罗尼被公认于冷战时期，成功地在东欧共产主义国家中建立了忠于教廷的天主教体系。

3、帕罗林只字未提中共把传教士贴上“帝国主义压迫者”的标签送去审判，把教廷代表驱逐出境，多位神父因此长年被监禁等。天主教中（共）国教会人士半数曾被当局送进劳改营，另外半数被关进监狱，在文革时期被折磨了十年。

4、有人说中梵协议的成果是使教宗成为中（共）国天主教领袖，教宗对中（共）国主教有最后任命权，但为何不公布白纸黑字来让人信服。极权主义政权的话能否得到信任？你不记得与拿破仑达成的条约和与纳粹政府达成的协定了吗？

据资料显示，帕罗林是教廷现任国务枢机卿，也称教廷国务卿，是圣座国务院的最高首长，执行教廷和梵蒂冈城所有的内部政治和外交功能。作为掌管教廷的实权派人物，帕罗林不顾外界、甚至是教廷内部有良知主教和信众的质疑和反对，一意孤行为中共站台和洗地，背后是与中共勾兑合作、不为人知的利益交换。他被中共“蓝金黄”，或者说跟中共沆瀣一气的勾当，不仅损害的是教廷的信誉，更是助纣为虐、残害门徒生命、破坏信众信仰，以此血淋淋的筹码与魔鬼中共交易。
