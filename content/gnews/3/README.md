## 中共国再现复阳病例
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1177905/)

作者:【默凡】

5月5日，山东青岛市委建委发布公共称，山东平度市发现1例入境确诊后治愈的复阳人员。该名3岁的刘姓男子，在阿联酋迪拜工作，3月6日自阿联酋迪拜出发抵达广州并就地集中隔离，期间被诊断确诊，治愈返回平度市健康监测期间，核酸检测均为阴性。5月5日进行复查时，核酸检测阳性。目前已对该患者实施隔离医学观察治疗，居住场所环境核酸检测均为阴性。青岛市卫健委称，国内尚未出现确诊病例治愈后复阳再传染人现象，让广大市民不必恐慌。

评论：

去年一季度至今，中共多地陆续报道发现复阳患者，中共媒体刚开始对这些新闻进行封锁，随着越来越多复阳患者被曝光，中共媒体不得不公开报道，国际社会至今也鲜少报道复阳患者，这应该就是媒体有意掩盖的结果。笔者认为复阳病例的出现应该属于次生灾难，这将任何国家都是严重的打击，如果复阳病人具备再次传播病毒的可能，那中共病毒问题可能永远也无法解决，好在至今还未出现复阳病例传播的案例。

一旦查出复阳病例，政府不得不调用大量医疗资源对这些复阳患者进行多次检测和隔离观察，复阳患者的密切接触人员也会成为被重点关注和隔离的对象，这会给社会造成巨大的资源浪费和恐慌。其实，要解决中共病毒的最好办法就是消灭中共，灭共不但能防止中共再次投放病毒，还能逼他们交出病毒关键信息和治疗解药。
