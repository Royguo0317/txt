2. 公投绑大选案

**本文针对「莱猪」议题撰写，其他公投议题将在其他文章中讨论**

**何谓莱猪？**

由国民党提起，目的为反对民进党政府同意进口美国施用莱克多巴安的猪肉进口台湾市场，并表示台湾人民健康将会因为美莱猪进口受到伤害，强调民进党利用独裁手段出卖台湾消费者的健康。

对此议题，先不论民进党为何要通过莱猪进口，在人民对安全疑虑发出意见后，执政政府立马推行应对措施。未来台湾商店使用美国莱猪必须在菜单上表明，让消费者在选择前有管道可以知悉商品的来源，并且做出合理的选择。

但国民党仍不满意，并且许多国民党籍政治人物，呼吁地方政府禁卖美国莱猪，意图形成中央、地方对于同一政策不同调的情形出现。

**台湾政体运作，就全国性事务不允许出现中央、地方不同调**

美国采联邦制度众所皆知，而新中国联邦未来也欲采联邦制度。联邦制度与邦联制度，具有本质相异，不可混同。

又美国联邦制度，将政府区分为联邦、地方州政府，且联邦政府、地方州政府都具有完整独立性，造成联邦、地方政府会有争权情形出现。而联邦制度，也因为只要求特定领域的地方政府权力上缴，好比「特定重大犯罪的审理权」。除此之外，州政府享有拟定法律、决定制度的权力。也造就了国中国的概念，但反面也带来国家法治的混乱，延伸出许多社会问题。

本文并非要表示联邦制度不好，只是希望让部分读者了解到赞美联邦制度之前，需要对体制的优缺有相当程度了解才恰当。美国治安议题、财富分配极度不均以及其他议题在亚洲看似中央集权的体制下，几乎得到了很好的解决，人民也多受到政府完善的照护。根本原因在于，联邦制度，地方、中央各自有很大权限，联邦就无法透过施压、执行全国统一性的惠民政策。

举例来说，国家推行的健康保险，使人民可以透过合理价格获得好的医疗照护，日本、韩国、台湾都做得很好，因此政府愿意亏大钱推行，不计成本。而在联邦制度下，基本上地方政府有完整的政策决定权，联邦政府是否能够要求各地方政府都推行统一的健保政策就产生疑问。

**台湾采原则给予高度地方自治，但涉及全国统一性事务就由中央统一办理**

所谓原则高度地方自治，即是一个事务在划分由中央或地方管辖时，原则划分给地方办理，此称为高度地方自治。

而地方政府(台北市政府、新北市政府等)，其地方自治权受宪法制度性保障。

所谓制度性保障，是指中央法规范只能对地方自治事项为上限、下限、上下限规范，但不可实质妨害地方政府就自治事项的自主决定权利。

所谓上限规范，指中央制定全国统一的侵害上限。如：各地方政府针对随地便溺的罚锾上限为10万，若台北市今天立法规定随地便溺处以1000万新台币，该法规牴触中央上限规定，无效。

所谓下限规范，指中央制定全国统一的「最低照料标准」，落实民主宪政精神中的国家照护义务。如：各地方政府必须对低收入致无法维持生活的家庭经济扶助等，台北市政府就不可认为花钱而不予推行。

回归正题，台湾制度强调高度地方自治，但同时对于全国统一性事务，仍由中央办理。诸如：**防疫、莱猪进口等**。因此，国民党鼓吹中央地方不同调，可以认为目的就是为捣乱而已。

**民进党政府突准许莱猪进口，可能为促进「台湾FTA」签订**

所谓**台湾FTA**，即是「台美自由贸易协定」。美国是世界消费大国，欧盟成立也是为建立像美国一样的消费市场，透过欧盟各国拟定统一的进口标准，可以达到目的。

又台湾过去经济持续依赖大陆市场，造成台湾在经济方面被架空。因此，台美自由贸易协定的签署，将使台湾经济重心可以透过协定的签署成功移转一部分到美国市场。

然而，台湾本身缺乏自然资源，仅可依靠国际贸易维持经济的特性。为扶植自身企业的成长，长期采用贸易保护主义，对进口商品课征高关税，以确保财政收入，同时造就台湾厂商商品可与进口商品竞争。讲白了就是，有钱买进口，没钱本土厂商，谢谢。

因此，美国贸易代表主张「台湾针对限制美国牛肉、猪肉进入市场」的贸易保护措施，已经实质违背了自由贸易的理念，因此否决了台美自由贸易协定的签署。

**美牛、美猪为台执政政府敏感议题，反覆后民进党突通过美猪进口**

美牛、美猪，因养育过程施用特定激素，被认为对人体有害，而受到不少国家的限制进口。诸如欧盟过去限制美肉品的理由即是「养育过程施用的激素经研究有害人体」。

对此，美国曾主张自身对于激素残留的法规标准严格于国际，在符合严格于国际的标准下，国际应无理由限制美国肉品的输入。

总之，台湾政府针对美牛、美猪也是经历过多次冲突，不论是国民党、民进党执政，唯一不同的指示赞成、反对方的互换。

而民进党政府在川普政府任内，突然通过美猪进口，呈上所述应该可以很明白的认为是政治交易。但台湾交易到的利益为何，无法确定。

**但若允许美猪进口可以确保台美自由贸易协定的签署，是否就成功地将台湾经济一部分与美国建立关系，至少确保台湾贸易可以不如过去一般如此依赖大陆市场。**

**国民党操弄反莱猪公投，目的恐为实质阻挡台美FTA、台进入国际市场**

呈上，台美自由贸易协定，对于现在台湾经济发展十分重要。若执政政府真以进口莱猪辅以配套措施，让台湾消费者可以知悉购买商品是否为莱猪，换得台美FTA，并非坏事。

但国民党大力炒作，并且提起反莱猪公投，恐实质的妨碍利益交换结果发生。

不过退一步来说，通过美猪进口是否真为台美自由贸易协定，也仅为个人猜测。但从国民党如此积极的反对，可能真有相当的可能性为事实。
