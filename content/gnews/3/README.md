辛辛那提報導—— 新冠病毒以前所未有的方式挑戰著我們的國家。為了克服這一危機，我們正在尋求世界一流的研究機構來幫助開發針對該病毒的治療方法和疫苗。但是這些開發研究和創新的地方也成為小偷們的主要目標。

今年5月，我們震驚地得知，據稱在俄亥俄州東北部，曾隸屬於克利夫蘭診所（Cleveland Clinic）的一名研究人員竊取了該診所實驗室的研究成果並準備帶到中共國。根據美國司法部的資料，該研究人員及其研究團隊從美國國立衛生研究院（NIH）獲得了由美國納稅人資助的360多萬美元的撥款，但是他隱瞞了他是中共國大學院長的身份。

根據刑事訴狀，該研究員同時也是凱斯西儲大學（Case Western Reserve University）的一名教授，並從中共那裡獲得了300萬美元的資助，用於復制他在克利夫蘭診所（Cleveland Clinic）的研究成果。在俄亥俄州全職工作期間，他還必須每年在中共國工作10個月，他甚至承認自己親自把生物樣品和克利夫蘭診所的科研項目直接帶回中共國。此外，該研究員還試圖為中國國大學招募其他美國研究人員。

如果一切屬實，這位研究人員不僅欺騙了美國政府，而且還欺騙了美國納稅人以及克利夫蘭診所（Cleveland Clinic）的同事。值得注意的是，克利夫蘭診所（Cleveland Clinic）已與執法人員和美國國立衛生研究院（NIH）密切合作調查此案。這是一個令人震驚的故事，但它不是一個孤立的事件。

作為參議院常設調查小組委員會主席，我去年領導了一次為期一年的兩黨共同參與的調查，調查中共是如何利用所謂的人才招聘計劃（最著名的是其千人計劃）來竊取美國納稅人資助的研究項目，來幫助它在過去二十年中軍事和經濟方面的崛起。
[!\[\](https://spark.adobe.com/page/VLff4x5jE5vJU/images/d1355a5f-2901-43a7-b90e-3965a7948bdd.png?asset_id=8d569a11-9f52-478e-8003-6873c8b153d4&amp;img_etag=%2253303f813598116ca5f0b2eaa52b6123%22&amp;size=1024)](https://spark.adobe.com/page/VLff4x5jE5vJU/images/d1355a5f-2901-43a7-b90e-3965a7948bdd.png?asset_id=8d569a11-9f52-478e-8003-6873c8b153d4&amp;img_etag=%2253303f813598116ca5f0b2eaa52b6123%22&amp;size=1024)美國參議員羅伯·波特曼（Rob Portman）是俄亥俄州的初級參議員。 
每年，聯邦撥款機構都會從美國納稅人基金中分配1500多億美元用於研究。然後，這些資金以撥款的形式分配給我們的研究機構。在大多數情況下，該系統運行良好。但是我們發現了一個令人不安的模式，像中共國這樣的外國政府系統性地瞄準了最有前途的美國研究人員，將他們招募到如千人計劃這樣的人才計劃中，並收買他們將美國納稅人資助的研究帶到中共國的大學。雖然竊取研究成果非常不好，但更糟糕的是，這種偷竊並非出於學術目的。據美國國務院證人在去年11月舉行的參議院聽證會上說，從美國實驗室竊取的研究成果與中共國軍事和經濟方面的最新進展有著明顯的聯繫。

這種行為必須制止。因為包括俄羅斯，伊朗和北朝鮮這樣的敵對國家在內的其他國家和地區也都在執行此類人才招聘計劃。據報導，就在最近，作為這一問題的部分調查結果，全國有54名美國國立衛生研究院（NIH）資助的研究人員因隱瞞與外國研究機構的關係而辭職或被解僱。在美國國立衛生研究院（NIH）的案例中，有70％的研究人員沒有上報外國的贈款資金，而被調查的的研究人員中有90%與中共國有關。

我最近提出了一項名為“保護美國創新法”（Safeguarding American Innovation Act）的兩黨共同立法來結束這種做法，並確保追究聯邦撥款的接受者在申請聯邦撥款時隱瞞與外國的關係的責任。該法還將對外國研究人員簽證的申請程序進行改革，要求對敏感研究採取更多的保護措施，並幫助我們更好地跟踪撥款的去向。該法案已獲得參議院國土安全和政府事務委員會的一致通過。

通過這項立法，我們可以追究中共的責任，並徹底制止他們偷盜由我們納稅人資助的研究。
