**简评：**

虽然主流媒体仍然装聋作哑，但越来越多关于2020美国总统大选舞弊的证据潮水般涌现，势不可挡。有志愿者根据公开使用的数据做了一个分析，列出了完整的名单，里面包括经多米诺（Dominion）（美国许多州使用的投票机）系统篡改和失踪的选票，发现有超过50万张选票从川普总统篡改成乔-拜登。此外，还有2,865,757张票丢失，作者本人帖子里提到——全部从川普名下走到了拜登那里。

此次大选，作假的一方几乎是无所不用其极：从死人投票到狗狗投票、操控投票机、甚至最简单粗暴的直接扔掉对手选票。看来美国的《宪法》是应该改了，但绝不是极左派主张的要把《宪法》推倒重来，而是面对中共国的无处不在的超限战，对与他们沆瀣一气的美国内部掌握权力的邪恶势力做出更与时俱进、更有效的约束，特别是被标榜为“第四权力”的主流媒体，由于其背后的资本的垄断性，已形成“话语权垄断联盟”，《宪法》中的新闻自由对他们早已不在适用。

**来自所有州的选举之夜数据分析显示，数以百万计的投票要么是从川普总统转变到拜登，要么是迷路了–使用多米诺（Dominion）投票机及其它系统**

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/13214858/image-206.png)

我们曾报道了2020年大选的诸多事件，而这些事件被媒体轻描淡写为系统”小故障”。

昨晚我们提出了另一个”小故障”，这次是在威斯康星州，可能逆转拜登在该州报告的票数领先。

今晚，我们获得了一份未经审计的分析数据，这份数据可供所有州查找与总统选举有关的系统故障和其他奇怪的票数。

昨晚我们报道了另一个所谓的系统”故障”，证据显示，威斯康星州的罗克县的选票总数似乎在川普总统和乔-拜登之间被调换–9,516张票从川普总统转移到乔-拜登身上，这19,032张票的差异经修正后将逆转拜登在威斯康星州的领先优势。

在昨晚的帖子中，我们还确定了我们在密歇根州、宾夕法尼亚州和佐治亚州报道过的问题，这些地方也发生了类似的”故障”。

接下来，我们要尝试获得更多相关数据，以便确定这个问题可能的广泛性，和/或是否有合理的解释。

然后今晚我们被引到了互联网上的一个网站,在这里有人看到了我们的帖子，决定自己做一个分析。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/13222256/1-111.png)

作者在文章中称，他列出了一个完整的名单，里面包括被多米诺（Dominion）（美国许多州使用的投票机）系统把川普篡改成拜登的选票以及被消失的选票。

(作者声明他的工作已被核实，但我们还没有核实结果，所以我们目前将他的结果标注为”未经审计”)。

作者决定自己获取数据，对从川普篡改为拜登的全国选票和被消失的选票进行分析（整个计票过程中，计票总数减少了这个数量）。

作者获得了被广泛使用的数据，据报道是《[纽约时报》](https://static01.nyt.com/elections-assets/2020/data/api/2020-11-03/race-page/pennsylvania/president.json)在选举当晚[拍到的](https://static01.nyt.com/elections-assets/2020/data/api/2020-11-03/race-page/pennsylvania/president.json)。

通过检查数据，作者能够找到一个将选票从川普掉包成拜登的实例，因此他决定对整个数据集进行分析，从使用多米诺（Dominion）投票机的州开始。

作者还声明，这些数据来自爱迪生研究公司（Edison Research），并且ABC新闻、CBS新闻、CNN和NBC新闻都在用这些数据进行选举报道。

《纽约时报》的网站也使用了，可能还有其它网站也是如此。

根据这一初步分析，我们发现有超过50万张选票从川普总统篡改成乔-拜登。

除此之外，还有2,865,757张票被删除。

请注意，这些数字还没有经过我们的审核。

我们还没有在这些州按县进行这项工作，这这些州可能会对全国选票总数产生更大影响。虽然更多工作有待完成，但显而易见的是，根据报告，投票机和应用程序中明显出现了一些非常可疑的操作，需进一步审查。

我们知道在2020年大选中存在着大量的舞弊行为。

没有人关注乔-拜登，他的支持率为零，但他的得票数却超过了川普总统领导的美国历史上最伟大的竞选活动。

我们难道应该相信乔-拜登击败了川普总统，同时失去了美国众议院和州议会的席位？这是很不可能的。

看起来，民主党为了窃取这次选举，做了一切可以想到的事情。

问题是他们从来没有想到川普总统会领导一场破纪录的竞选，他们被抓包了。
