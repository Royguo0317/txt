**簡評：**

在以假治國思想的指導下，中共國的數據造假已不是什麼新鮮事。在中共病毒、旱災洪災、工廠倒閉、失業暴增的情況下，中共國工業利潤竟然持續上漲。國家統計局表示，產量和需求的持續復甦，以及企業成本和費用的下降，是工業利潤增長的原因。

本文指出中共國工業利潤增長的幾個矛盾之處：第一，在中共國工業收入增長停滯不前的情況下，工業利潤大幅增長是非常可疑的；第二，正值中共國通貨膨脹之際，工業利潤的大幅增長是如何做到的？第三，近期工業活動大幅下滑，為何工業利潤大幅增加？ 。

在鐵證面前，中共國所謂的工業利潤增長假得有些離譜，統計局的解釋顯得蒼白無力。中共國的經濟到底糟糕到了什麼程度，相信離真相大白那天越來越近了。

**中共國工業利潤持續上漲，但這只是更多的“虛假數據”嗎?**

昨天，中共國國家統計局公佈，8月份中共國工業企業利潤保持了令人驚訝的韌性，較上年同期增長19.1%，與7月份19.6%的年增長率相比略有放緩。北京方面表示，中共國工廠在大瘟疫停產後仍保持增長勢頭。與去年同期相比，8月份工業企業的利潤率進一步上升。按行業分類，數據顯示，8月份上游行業的利潤增長優於下游行業。

**以下是主要亮點:**

8月份中共國工業利潤同比增長19.1%，略低於7月份的19.6%。 8月份非年化利潤環比下降1.6%。

與去年同期相比，8月份上游行業的利潤增長超過了下游行業。 8月份下游行業利潤同比增長放緩至+16.9%(7月份同比增長28.7%)，而上游行業利潤同比增長增加至+31.0%(7月份同比增長4.6%)。從主要行業來看，通用設備製造業利潤同比增長37%(7月份同比增長26%)，電機和設備利潤同比增長13.3%(7月份同比增長8.1%)。汽車製造業的利潤仍相對穩健，8月份同比增長55.9%，不過較7月份的102.5%有所放緩。在大宗商品價格上漲和需求復甦的推動下，8月份礦業企業利潤同比下降11.9%，較7月份的利潤同比下降40.6%明顯收窄。

在和數據一起發布的一份聲明中，中共國國家統計局表示，產量和需求的持續復甦，以及企業成本和費用的下降，是造成這次（利潤）增長的原因。國家統計局還指出，礦業盈利能力正在復蘇，設備製造商的利潤也在快速增長。

然而，和往常一樣，當談到中共國的“數據”時，報告中有幾個關鍵的問號——這些對在與川普正在進行的冷戰中的習近平非常有利——但這讓數據的有效性受到了質疑:

首先，工業收入增長仍然停滯不前，儘管工業總收入同比增長從7月份的3.3%增加至4.9%，但與利潤的飆升相去甚遠。在沒有收入增長的情況下，利潤的持續飆升完全取決於利潤率(總利潤除以收入)的上升，8月份利潤率升至6.9%(去年同期為6.1%)，為總利潤的穩定增長做出了貢獻。在供應鏈轉型本應擠壓利潤率的情況下，中共國如何在不大規模裁員的情況下做到如此急劇的利潤率擴張，仍是一個巨大的未知數。

其次，讓這些數據的可信度受到進一步質疑的是，工業利潤的大幅增長正值中共國所謂的通貨膨脹之際。也就是說，作為決定當地企業定價權的關鍵因素，並因此成為工業利潤的關鍵驅動力的生產者價格，仍然處於強勁的負值。事實上，在過去三年PPI（生產者價格指數）持續走低之後，工業利潤從4月份開始神奇地飆升，而同期生產者價格卻繼續下跌。因此，當前工業利潤和PPI之間的差距是有史以來最大的。

![]()![](https://spark.adobe.com/page/Pm4kxUVHhPoC3/images/262077cd-c56e-4acb-b4a5-b21bdaf44cbc.jpg?asset_id=af906782-1f41-4623-8251-451523d94c20&amp;img_etag=%22f0f63241c22409ee6bdad2b93494f7f2%22&amp;size=1024)            中共國的工業利潤（藍線）和生產者價格指數（紅線）的同比增長率對比

這兩個傳統上緊密相關的數據集之間的脫鉤錶明了三件事:

中共國PPI（生產者價格指數）在未來幾個月將大幅上漲，因為不斷上漲的價格是驗證工業利潤率和利潤躍升的一個關鍵因素，將導致更廣泛的通貨再膨脹浪潮從中共國湧出，波及世界其它地區。

工業利潤的飆升只是一個一次性的例外，其原因是最近本地信貸創造的激增，自冠狀病毒大瘟疫爆發以來，新增信貸已達數万億美元，而一旦北京當局放鬆控制，利潤將大幅下降。
