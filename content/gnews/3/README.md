2021/1/12早-路德时评 路安墨

**习近平至今依然没有露脸**

15-18年的今天习近平都在忙大事，但2021年的今天习还是和前些天一样依然没有露脸。在上次的PS照片被路德社发现后，中共还进行几次修改，这么看来中共很擅于听取人民群众的意见。为了避免上次被别人发现的尴尬，今日的新闻就不敢再用习的照片了。

**兴师动众的调查**

据传闻此次因照片事故问题，习秘密调查摸排了千余人，只因一张照片便如此兴师动众，各个宛如惊弓之鸟，让人觉得十分好笑。其中着重旁敲侧击打听处理有关照片的人员近况。之前因习的手术已有一批含冤入狱之人，如今恐怕也少不了在大牢中多添几缕冤魂了。但中共人人生性惜命，相互推诿责任，最后无奈将所有的事情都推到了美国的技术和美国的黑客身上，借此以苟全自己性命。

**杀人诛心**

此次不是简单的中共马脚被抓住而已，这段时间路德社也不是在简单的爆料习近况而已。这一切从头至尾都是在验证一件重要的事情，并且这也是对中共的杀人诛心。

照片的诸多漏洞怎么泄露的？这些相关的工作人员为什么偏偏在此时出了问题？这些问题就留给中共自己细细思索吧，中共会慢慢体会到什么是杀人诛心的滋味。

**政法委工作会议在说哪般**

近日进行的政法委工作会议上阐明了内外的形势分析了各种风险。其中重点点名了互联网的风险，这就是在暗指爆料革命。中共一直恐惧爆料革命所掀起舆论传到中共国内进而引起社会各方面的种种问题。殊不知此次会议中有多少爆料革命的战友。现在的中共，由于习的长期不现身。党内人员已经开始纷纷站队了。加之报道中习的人被重点凸显，导致下面的官场互相猜忌，中共的官场已经乱成了一锅粥。

**王春宁的出席是何用意**

刚刚上任的武警部队司令员王春宁也出席了此次政法委的会议。新闻报道中重点突出王春宁参加该会议，其是为了表明习不仅掌控卫戍，还掌控了武警，并对与会的政法委官员进行恐吓和警告，进而稳住局面。
