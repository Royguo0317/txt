## 滅共先鋒蓬佩奧,開始為2024年總統競選鋪路
`2020-07-21 05:47 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/271296/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/21054110/Picture-1-136.png)圖片來源：Olivier Douliery-Pool/Getty Images 
據《華盛頓郵報》最新報導，美國國務卿邁克·蓬佩奧（Mike Pompeo）週五帶著一份新的人權報告返回愛荷華州，該報告讓他朝著2024年總統候選人的方向更進一步。他在得梅因舉行的家庭領導峰會上說：“我們管理政府的方法借鑒了《創世紀》第一章的美好教導，即所有人類都是按照上帝的形象造的。目前，全世界依然有五分之四的人們不享有宗教自由，我們此時正在竭盡全力改變這個狀態。”

蓬佩奧（Pompeo）否認了他此行是為總統選舉推波助瀾的猜測。但這次訪問標誌著他自擔任美國最高外交官以來第二次前往愛荷華州，他解釋稱這趟行程是一項具法律效應的公共外交行為，這一行程是共和黨總統候選人尋求該州基督教保守派祝福的傳統活動，蓬佩奧認為這一次的訪問為國務工作提供了很好的機會，有助於獲得支持者，同時為美國外交政策優先事項提供了支持。近年來，他在堪薩斯州，德克薩斯和佛羅里達州都舉辦過公開活動。週五他在家庭領袖峰會上說：“我的任務之一就是走遍整個美國，以強調突出我們正在進行的偉大工作。我們執行的外交政策是得梅因，迪比克和達文波特的美國家庭可以信賴的。這是一項以美國為中心的親國家外交安全政策，這也是一項支持宗教自由的外交政策。地方領導人和國家觀察人士基本不太記得前任國務卿們在任職期間是否到過如此重要的總統競選地，但蓬佩奧知道，任何訪問愛荷華州的國家政治人物都會帶來人們對總統競選的猜測。

家庭研究委員會的托尼·帕金斯（Tony Perkins）說：“不僅聽眾會聽到蓬佩奧的聲音，人們還將廣泛聽到福音派人士的聲音。”蓬佩奧可能會在未來川普總統離任之後繼續擔任共和黨的領導權。在他擔任中央情報局局長期間，他曾向總統提供每日情報簡報，以及在不與總統相悖的情況下提供必要的外交策略訣竅。帕金斯說， 蓬佩奧對總統的支持和忠誠以及他為捍衛國際宗教自由，面對中共採取的勇氣，都能引起人們極大的共鳴。在周五的演講中蓬佩奧觸及了這些觀點，宣稱：“上帝是真理”，並使用該公理作為譴責中國共產黨的依據。 “我們每天都在講CCP病毒的來源和共產黨掩蓋該病毒真相的掩蓋。”

根據一些觀察人士的說法，蓬佩奧公開參與同塔利班簽署和平協議，也許會使他受到潛在的共和黨總統競爭對手的攻擊。一位前白宮官員說，他與塔利班領導人的合影，日後也許很容易成為被對手進行攻擊的例子。另一方面，蓬佩奧也許已經找到瞭如何將其外交政策工作轉化成一個能引起福音派選民的國內政治平台，他有最大的機會將福音派選民鞏固成為2024年競選時的基礎。

評：國務卿蓬佩奧作為滅共先鋒成員之一，讓美國重新偉大，制止中共對美國的滲透和傷害也是他與團隊成員當下的責任和使命。川普總統以及他的團隊肩負著的使命是前所未有的，即：滅共。無論中共的宣傳戰對他以及他的團隊向世人做出多少虛假的報導與不實信息，這一天終將會結束。正如郭先生直播中所預測：2024年的美國總統一定出自蓬佩奧，班農和科頓中的一位！祝福並感謝他們！
