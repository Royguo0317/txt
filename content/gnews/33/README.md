继英国通讯管理局(Ofcom)宣布撤销中共国官媒的中国环球电视网(CGTN)的播放执照之后，据路透社上周五报道，德国沃达丰不得不停止在其有线电视上播放CGTN 的节目。沃达丰表示希望能继续转播CGTN的电视节目，但是没有合法的执照。他们目前正在与地方媒体主管部门和广播公司代表就执照被撤销事宜进行讨论，以厘清当前的法律状况。截止到目前，CGTN 并未对此作出任何回应。

# **德国沃达丰(Vodafone)暂停中共国有线电视**

[!\[\]()!\[\](https://spark.adobe.com/page/PlOglCOZCrIIH/images/86f29045-7edf-469b-82ab-52229f751a5c.jpg?asset_id=980f4692-d876-46e8-b9eb-b3c20333cf6b&amp;img_etag=%2279bfa9a657d361ff2d42f812a3859c1e%22&amp;size=2560)](https://spark.adobe.com/page/PlOglCOZCrIIH/images/86f29045-7edf-469b-82ab-52229f751a5c.jpg?asset_id=980f4692-d876-46e8-b9eb-b3c20333cf6b&amp;img_etag=%2279bfa9a657d361ff2d42f812a3859c1e%22&amp;size=1024)资料图片：2013年9月12日，德国沃达丰公司总部在杜塞尔多夫拍摄。路透社/伊娜·法斯宾德/资料图片

柏林(路透社)——由于中英两国的媒体纠纷, 德国沃达丰不得不在其有线电视服务上停止播放中共国国有的中国环球电视网(CGTN)电视节目。

英国电信集团沃达丰旗下的子公司上周五表示，希望恢复CGTN的服务，但目前没有有效的许可证。

英国上周吊销了允许CGTN在英国播放的许可证，这引起了中共国的抗议，周五中共国禁止BBC进入其电视网络，并且限制了BBC在香港的转播。

[!\[\]()!\[\](https://spark.adobe.com/page/PlOglCOZCrIIH/images/d1bb4d87-b8c0-4a82-adad-91b445424291.jpg?asset_id=dea24ae3-a5a9-4792-a272-519855d0a0a5&amp;img_etag=%22361cdc2275718f00af7697f13e254001%22&amp;size=2560)](https://spark.adobe.com/page/PlOglCOZCrIIH/images/d1bb4d87-b8c0-4a82-adad-91b445424291.jpg?asset_id=dea24ae3-a5a9-4792-a272-519855d0a0a5&amp;img_etag=%22361cdc2275718f00af7697f13e254001%22&amp;size=1024)德国沃达丰

德国北莱茵-威斯特法伦州(North Rhine-Westphalia)是迄今为止可以使用CGTN的地区之一，该州监管机构的发言人表示，CGTN是根据英国的许可证在德国播放的。

根据1989年由欧洲委员会（英国仍是其成员国）主持下达成的一项“跨境电视 ”协议的条款，一个欧洲国家的播放许可证适用于整个欧洲大陆的大部分地区，这意味着CGTN现在可能不得不在整个地区停播。

所有欧盟国家以及许多非欧盟国家，包括大多数巴尔干国家和乌克兰，都签署了这项协议。原则上，任何一个国家的许可证都可以允许恢复播放。

沃达丰表示：“我们目前正在与地区媒体主管部门和广播公司代表关于许可证被撤销的事宜进行讨论，以澄清法律状况。”

周五是中共国的公共假日，CGTN在正常工作时间之外并未立即回复电子邮件的评论请求。
