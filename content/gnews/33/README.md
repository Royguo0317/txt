路透社报道，脸书公司周三表示，它已经阻止了中共黑客利用脸书平台监控生活在国外的维族人的行动。中共黑客将恶意软件链接到维族人的设备上，从而实现监控。

脸书证实，被攻击的目标将近500人，他们大多来自新疆地区，但主要生活在土耳其、哈萨克斯坦、美国、叙利亚、澳大利亚和加拿大等国家。该公司表示，大多数黑客活动都发生在脸书平台之外，他们利用脸书分享恶意网站的链接，而不是直接在该平台上分享恶意软件。

脸书公司表示，这些黑客在安全行业被称为“地球恩浦萨”(Earth Empusa)或“邪恶之眼”(Evil Eye)，攻击的目标是民主活动人士、记者和持不同政见者。此次攻击的主要目标是维吾尔人，这是一个在中共国面临迫害和种族灭绝的少数民族。

脸书网络安全调查人员在一篇博客文章中表示，“这一活动具有资源充足、持续不断的特点，同时也混淆了幕后黑手的身份。”黑客组织使用虚假的脸书账户伪装成虚构的记者、学生、人权倡导者或维吾尔族社区成员，以建立与目标的信任，并诱骗他们点击恶意链接，在他们的设备上安装间谍软件。黑客利用与维吾尔族和土耳其流行新闻网站域名相似的域名建立了恶意网站，并破坏了目标访问的合法网站。

同时，黑客组织还模仿第三方安卓应用商店网站创建含有维吾尔族主题的应用，比如祈祷应用和词典应用，其中含有恶意软件。调查发现两家中共国公司，北京百思特联合科技有限公司和大连九速科技有限公司开发了该黑客组织部署的安卓工具。

中共驻华盛顿大使馆没有立即回复对此报道的置评，北京则否认网络间谍的指控。
