## 披露：欧盟-美国峰会闭幕声明草案内容

#### 校对：文枉
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/11094401/1.jpg.png.jpg)图片来源：brunswickgroup.com
据比利时首都布鲁塞尔的《欧洲政治周报》(Politico Europe)网站（politico.eu）6月9日报道，在本周末的七国集团首脑会议和下周一的布鲁塞尔北约峰会后，欧盟和美国的高峰会议也即将到来，两位欧盟官员和《欧洲政治周报》的记者看到了欧盟-美国峰会闭幕时将发表的声明草案，对此作了相关信息透露。

报道称，欧盟和美国将宣布一项有关技术和贸易的广泛伙伴关系，旨在反击中共国并促进民主价值观。去年12月，欧盟提出了这一倡议，但文章认为在川普（Trump）担任总统期间，跨大西洋关系出现了坎坷，目前华盛顿愿意与其盟友合作。因此，虽然草案透露关于这个新机构的谈判仍在进行，但美国总统乔·拜登（Joe Biden）和欧盟委员会主席乌苏拉·冯德莱恩（Ursula von der Leyen）计划6月15日在布鲁塞尔会晤时宣布该项目的细节。

报道分析了欧美峰会的现实需要，指出，中共国作为一个科技超级大国的崛起，解释了为什么美国和欧盟有动力在科技和数字贸易方面进行合作。因为中共已经把目光投向了人工智能、机器人和半导体等下一代产业，企图成为全球领导者和标准制定者，并计划通过积极的投资、收购和有利于阿里巴巴和华为等国内公司的规定来实现这一目标。

双方官员们希望新的跨大西洋机构将为欧盟和美国提供一个更强大的立足点，以保持一致行动。作为欧盟-美国贸易和技术委员会（TTC）的一部分，华盛顿和布鲁塞尔预计将详细说明他们会如何共同促进新兴技术和制定共同标准，并做出承诺，以更好地监督现在由少数硅谷科技巨头主导的数字世界。

报道认为，从错综复杂的技术供应链（目前是基本通过中共国）到对数字项目进行集体投资，欧美联盟可以解决一切问题。并且该设计也适合于与其他志同道合的国家，包括澳大利亚和日本，以及世界贸易组织等国际机构合作。负责数字政策的欧盟委员会执行副主席玛格丽特·维斯塔格(Margrethe Vestager)上周对欧盟政界人士说：“重点当然是推动真正的全球标准，并且是建立在隐私、数字完整性和个人尊严基础上的标准，以此作为起点。”

《欧洲政治周报》的记者本周也得到了白宫国家安全顾问杰克·沙利文(Jake Sullivan)的采访回应，这位安全顾问说：“拜登和他的欧盟同行们将专注于调整我们对贸易和技术的态度，从而由民主国家而不是其他任何人，不是中国或其他专制国家，能够书写21世纪的贸易和技术规则。”

报道分析认为，布鲁塞尔和华盛顿之间的重新合作也绝不会一蹴而就。美国官员经常批评欧洲国家在一系列反垄断调查和监管建议中不公平地对待美国科技公司。而欧盟的许多人认为，这些政策是这个由27个国家组成的集团，为在顶级数字地位上重新获得与美国和中共国并驾齐驱所做努力的核心。并且，一些欧盟领导人，如德国总理安格拉·默克尔（Angela Merkel），也不太愿意把中共国单挑出来，因为他们相互间有着重要的经济联系，不想在可能发生的贸易战中受到损害

报道认为，新的欧盟-美国机构还必须避免以前努力的命运，如跨大西洋经济委员会，这是小布什时代为重振美国和欧洲之间的经济和贸易关系而发起的一项倡议，但最终却无疾而终。

因未获授权公开谈论正在进行的草案讨论，两位匿名发言的欧盟官员称，为了避免这些陷阱，华盛顿和布鲁塞尔计划在下周的声明中重点关注三个核心领域。

第一个领域（主要由华盛顿倡导）将集中于通过围绕国际贸易和技术标准的跨大西洋合作来反击中共国，以及共同解决在中共病毒疫情危机中出现的关键供应链问题。目前，全球微芯片短缺，美国和欧盟都已宣布计划向公司提供财政激励，以使目前集中在环太平洋地区的半导体制造业回到各自的管辖地区范围。贸易和技术委员会将围绕人工智能、网络安全、竞争力、数据治理和技术平台设立欧盟-美国小组，以创建跨大西洋标准。

该草案文件称：“我们承诺在重新平衡全球半导体供应链方面建立欧盟和美国的伙伴关系，以加强欧盟和美国各自的供应安全，以及设计、生产最强大且资源高效的半导体的能力。”

第二个合作领域的重点将在数字领域促进民主价值，包括回击专制政府对部分互联网的钳制，并在专制国家内推广民主原则。欧美有着共同的利益基础，如开放社会和公平市场，因而最终有可能会形成监管协调机制，因为布鲁塞尔和华盛顿都寻求通过新的数字立法。

第三个领域是双方将关注改善科技领域竞争政策和执法方面的协调，包括成立一个专门的政策工作组来研究数字反垄断问题。七国集团的监管机构将在今年晚些时候举行会议，协调解决各国对数字市场日益被少数科技公司所主导的担忧。

报道指出，提案最终将促进欧盟和美国之间更大的创新和投资，在这个领域，双方都自上而下的在努力与中共国的经济模式竞争，包括旨在利用美国和欧洲的经济实力，设立合资企业，以促进围绕量子计算等新兴技术的研究和开发。

但是，报道也指出，欧盟方面担心华盛顿的官员渴望利用新的倡议来反击中共国，而布鲁塞尔的同行们则步履缓慢。欧盟官员告诫说，这不会减少欧盟和美国之间的其他潜在爆发点，比如正在进行的贸易争端。一位不愿透露姓名欧盟贸易外交官说：“建立贸易和技术委员会当然很重要，但我们必须看到它在建立后的实际运作情况。这对跨大西洋关系来说绝对是一个好兆头，但对贸易来说，关税讨论才是最重要的。”
