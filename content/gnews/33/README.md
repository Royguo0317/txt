## 美国国家情报总监：冠状病毒来自感染的动物或中共国的实验室

[!\[\]()!\[\](https://spark.adobe.com/page/6gIxpUSOY5y75/images/97461e6b-313c-44b6-97fd-1ffeefe5105e.png?asset_id=d94a41ed-32d4-45db-84dc-92ff9d06109a&amp;img_etag=%229b98b06000734520391623c9c42d2693%22&amp;size=1024)](https://spark.adobe.com/page/6gIxpUSOY5y75/images/97461e6b-313c-44b6-97fd-1ffeefe5105e.png?asset_id=d94a41ed-32d4-45db-84dc-92ff9d06109a&amp;img_etag=%229b98b06000734520391623c9c42d2693%22&amp;size=1024)国家情报总监艾薇儿·海恩斯（Avril Haines）2021年4月14日星期三在华盛顿国会山参议院情报委员会上有关全球威胁的听证会上作证（Saul Loeb / Pool通过美联社）

国家情报总监艾薇儿·海恩斯周三对国会表示，新冠肺炎疫情或是从中共国实验室发生的泄露事故，或是通过感染的动物传染给人类的。

海恩斯女士首次以国家情报总监身份出席参议院情报委员会议时说：“基本上，[情报]要素的汇聚指向了两种可能：要么是人与受感染动物的接触自然产生，要么是实验室泄露事故。”

海恩斯女士的评论与世界卫生组织-中共国政府联合报告的最新结论相矛盾，该报告认为实验室事故“极不可能”。

仅仅几个月前，美国科学家将这种病毒从中共国实验室逃脱的理论称为“阴谋论”。此次海恩斯女士对实验室泄露理论的评论，是高级情报官员首次就该病毒是从中共国实验室中开始传播的可能性发表的正式官方评论。

美国情报机构正在努力寻找致命疾病爆发的线索，该疾病已经夺走了全球200万人的生命，而武汉病毒研究所和中共国的其他实验室一直是被重点关注的对象。国务院一月发布了一份关于该所的情况说明书，称有大量间接证据表明该病毒始于此地。

证据包括该实验室的工作人员在2019年秋季患上了类似COVID症状的疾病；该实验室一直在研究蝙蝠冠状病毒，他们研究的冠状病毒与COVID的病毒（称为SARS-CoV-2）具有96％的相似性；这个实验室为中国人民解放军进行秘密工作，在那里进行了动物实验。
![]()![](https://www.gnews.org/wp-content/uploads/2021/04/Covid-19_.jpg)
海恩斯女士的证词与中共国政府有关该病毒不可能来自他们的实验室的主张相矛盾。

世卫组织-中共国联合研究小组的中共国团队负责人梁万年表示，专家“一致同意”，“这种病毒极不可能是从实验室泄漏出去的。”梁万年说，因此，除非有新的证据，否则将来的病毒起源追踪任务将不再专注于实验室泄漏理论。

海恩斯女士没有提供这两种理论的详细信息，但强调情报界并不确切知道该病毒在何处、何时、或如何出现和传播。她说，情报机构正在继续收集有关病毒来源的信息。

（情报）委员会共和党副主席佛罗里达州参议员马可·卢比奥说，发生实验室事故是合理的，因为“迄今为止，尚未明确确定人畜共患病的这种传播途径。”

“武汉病毒研究所的研究人员从他们的出版物中证明，他们熟练地对蝙蝠冠状病毒进行了基因改造，从而创造出了新的人造病毒，这些新病毒具有很强的能够在人类中造成疾病的能力。”佛罗里达共和党人卢比奥说。

他说，中共国已经记录了多次实验室病毒泄漏事件，包括原始的SARS病毒。

卢比奥还指出，曾经访问过有高级别安保的武汉病毒研究所的美国外交官在2018年警告说，安全性很差。

他说：“我们无法得出结论，导致新冠肺炎的病毒自然发生，直到有一条传播链确定了该病毒如何在物种之间进化和传播。”

中情局局长威廉·伯恩斯（William Burns）也在星期三向参议院小组作证时说，他同意海恩斯女士的看法。

伯恩斯作证说：“我们和我们的分析师很清楚的一件事是：中共国领导层与世卫组织的合作中，既没有全面配合、开诚布公，也没有提供原始的、完整的数据来帮助我们找到答案。”
