# 字节跳动（ByteDance）挑战川普的TikTok撤资令

[![](https://spark.adobe.com/page/whb124D3AhcKH/images/21d4f18c-5f94-4ab6-9d39-9b377b009dbf.jpg?asset_id=145a3916-24ee-4892-b20f-d1fb989b8718&amp;img_etag=%22964a7dac5c6f45cb208dfa78917c8742%22&amp;size=2560)](https://spark.adobe.com/page/whb124D3AhcKH/images/21d4f18c-5f94-4ab6-9d39-9b377b009dbf.jpg?asset_id=145a3916-24ee-4892-b20f-d1fb989b8718&amp;img_etag=%22964a7dac5c6f45cb208dfa78917c8742%22&amp;size=1024)法新社图片

视频分享应用程序抖音（TikTok）的中国母公司——字节跳动（ByteDance）周二晚些时候向美国上诉法院递交一份诉状，挑战川普政府将于周四生效的行政命令，该行政令要求其从TikTok撤资。

美国总统唐纳德•川普（Donald Trump）在8月14日的命令中，要求字节跳动（ByteDance）在90天内从该应用程序撤资，该命令于周四生效。

TikTok在一份声明中说：“面对不断提出的新要求，也不清楚我们提议的解决方案是否会被接受，我们要求延长30天，这是在8月14日的命令中明确允许的。”

该公司表示：“如果没有延期，我们别无选择，只能向法院递交诉状，以捍卫我们的权利。”

白宫和财政部拒绝置评。司法部也没有立即发表评论。

川普政府认为，由于中国政府可以获取美国用户的个人数据，TikTok构成了国家安全隐患。在美国拥有超过1亿用户的TikTok否认这一指控。

该诉状书将川普、司法部长威廉•巴尔、财政部长史蒂文•姆努钦和美国外国投资委员会（CFIUS）一起命名为“跨机构小组”，负责审查涉及国家安全问题的某些外国投资交易。

该诉状表示，美国外国投资委员会（CFIUS）的行动和川普的命令试图强迫TikTok大规模撤资，TikTok是一家建立在“字节跳动（ByteDance）”开发的技术基础上的价值数十亿美元的企业，该公司是根据政府据称对一项为期三年、涉及不同业务的交易而进行国家安全审查而成立的。

该命令基于政府对字节跳动在2017年收购美国社交媒体应用程序：Musical.ly 的审查，因为字节跳动把应用程序—Musical.ly 合并到TikTok。

美国商务部对TikTok的单独限制已被联邦法院阻止，其中包括对原定于周四生效的交易的限制，TikTok警告称，这实际上可能会在美国禁止该应用程序的使用。

谈判一直在进行中，目的是敲定沃尔玛（Walmart Inc.）和甲骨文（Oracle Corp.）入股一家新公司以监管美国业务的初步协议。川普曾表示，这笔交易得到了他的“祝福”。
