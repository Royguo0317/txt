鳳凰農場－舊金山金喜站

任何一個農場都是一個小的經濟實體，同時也是一個小的政治實體。農場要想管理好，避免混亂，避免被五毛滲透，必須要有一套合情，合理，合法的規章制度，從中華傳統智慧的角度來講，就是要上合天道，下合人心

說到政治制度，西方的民主法治其核心本質與法家思想不謀而合，即相信人性本惡，所以需要用法律約束，所以需要把權利關在籠子裡，具體的方法西方社會採取了六個。治理一個國家和治理一個小區有很多類似之處，下面以小區為類比解釋這六根拴住權利的鐵鍊。

第一個就是**立法權**，立法權必須由**議會**行使，議會就是**小區業主委員會**。議會就是人民選出來的代表，比較有智慧的，比較有德行的，然後共同決定是否需要立法，如何立法，**立什麼規則**，這就是議會。議會可以審核、質詢政府，不滿意政府有權炒掉政府。議會不可以執法，不可以既當裁判又當球員。必須把權力分散，分成六夥人互相牽制，絕對不能獨裁；

第二個就是**行政權**，行政權就是由**政府**行使，政府就是**小區的物業**，**政黨**就是不同的**物業公司**；

第三個就是**司法權**，小區的任何糾紛不可能由物業公司（政府）裁判，必須由一個獨立第三方的法院來作出**公正裁判**，這就是司法權，這就是法治。這個法律是議會（小區業主委員會）制定的，絕對不可以由政府（物業公司）制定。政府隨意制定的是惡法，是暴政惡法，絕對要推翻它，這就叫**司法獨立**。西方傳統的三權分立：議會行使立法權，負責立法和監督政府；政府行使行政權；法院行使司法權；

第四個就是**軍隊國家化**，假如分立的三權違背憲法勾結在一起禍害人民，軍隊（小區保安隊）就有權消滅他們。小區保安隊保護的應該是小區居民，而不是物業公司。共產黨的軍隊則是屬於黨，黨指揮槍，是黨衛軍，不是用來保衛人民的；

第五個就是**媒體監督權**，假如立法權、行政權、司法權和軍隊都被獨裁者壟斷，就需要揭發真相的媒體監督（**小區公告板**），所以媒體必須獨立。媒體不可以屬於黨，不能成為黨的喉舌，媒體告訴人們真相，媒體必須是獨立的監督。共產黨把所有的媒體都藍金黃，包括海外媒體，國內媒體都是黨的喉舌，開始欺騙全世界；

第六個權力就是**擁槍權**，人民必須有槍，如果獨裁者把上述五個權力全部壟斷，**人民必須自己站起來反抗（小區業主聯名換掉物業公司**）。所以美國有四億支槍在民間，政府絕對不敢到美國老百姓家強拆，人民享有擁槍權至關重要。但共產黨禁止人民擁槍，不僅槍不可以有，連菜刀也實名制。

現代社會的大眾普遍認為，西方的民主法治就是人類政治治理的最高典範，卻很少有人真正了解中華政治治理的高深智慧。當我們真正了解了中華上下五千年的完整歷史，就會發現其內在的規律：中華政治治理其實是一個不斷倒退的演變過程。

中華的政治治理一共經歷了**五個演變過程**，從一萬年前，三皇時期上合天道，下合人心的帝道政治開始，中華政治治理是一步步倒退到今天中共暴政加邪教的邪惡統治的，在了解完這段歷史以後，相信我們新中國聯邦和各個喜馬拉雅農場應該採取什麼樣的政治治理理念就顯而易見了。

中華治理的最高典範：**帝道政治**

中華上古的三皇時期，以**伏羲**為代表的帝道政治的治理方式是**道治**，道治的核心是**以道治民**，倡導**上合天道，下合人心，無為而治**。這個時期人們的智慧和德行都達到了非常高的境界，大多數人都達到天人合一的境界，人民之間和諧共處，根本無需治理，也就是所謂的無為而治，這個美好的時期持續了約五千年。

接下來從**軒轅黃帝**開始的五帝時期，人民的德行和智慧都有所下降，政治的治理也從無為而治的道治，漸漸退化為**有為而治**的德治。**德治**的核心是**以德化民**，倡導以**德教結合法治，則天之明，因地之利**，也就是通過德行的教化來提升人民的德行和智慧，同時發展經濟使人民百姓生活富足，並以法律來懲戒壞人使人民有所畏懼，堯舜就是以這樣的方法來構建和諧社會。

三皇時期的道治，和五帝時期的德治，統稱為**帝道政治**，這是中華政治治理的**至德要道**，也是後世所有帝王努力追求的最高境界。這就是《孝經－開宗明義章》講的“**先王有至德要道，以順天下，民用和睦，上下無怨**。”

中華治理的最低層次：**魔道政治**

近百年來，在蘇聯的支持下，中共勾結日寇，不斷坐大，最終篡奪了政權，70年來，殘害了數億中華兒女，中共的殘酷邪惡統治是集古今中外邪惡思想之大成，可謂東邪西毒集於一身，首先中共繼承了中華歷史上的糟粕——秦皇漢武的**商鞅思想和偽儒的奴化教育**，其次還嫁接了西方的**馬克思主義邪教理論**，並結合了蘇聯**列寧和斯大林的邪惡實踐經驗和組織體系**，還結合了**希特勒的納粹德國宣傳洗腦術**，並兼容了**墨索里尼的法西斯主義思想**，其核心理念就是要激發出人們內心的魔性，實施魔道政治，對內實施暴政極權，建立高科技下的現代奴隸制，對外發展共產主義國際邪惡勢力，對民主國家進行滲透和超限戰，妄圖奴役全人類。

而與政治治理方式逐步退步相對應的是，普通老百姓的平均思想水平也在不斷地倒退，人心在不斷敗壞。人心也是從上古時期先天的**神性**，敬天敬神，無私無我，墮落到夏商周三代時期強調的**人性**，就是講仁義禮智信，孝悌為本。自從周文王篡改了先天八卦為後天八卦，先天易道逐漸失傳，大道不顯，直到周朝末年積重難返，諸子百家不明易理，各執一詞，人心產生了巨大的退化，墮落到了**動物性**，道德退化，智慧退化，不敬上天，不孝父母，只追求食色，衣冠楚楚的高級禽獸。再往後歷代帝王不斷篡改易理，愚民洗腦，最後墮落到大部分人崇尚**魔性**思想，強調鬥爭哲學，成王敗寇，利益至上，叢林法則，以奴役他人為樂。

**而一切的問題都是人的問題，人的問題就是人心的問題，所以要從根本上解決中華大地上的問題，乃至全世界的積弊，一定要從人心動手。**

喜馬拉雅農場的管理，最理想狀態當然是恢復到先王的**至德要道**，帝道治理，**西方法治＋中式德教**，那麼首先就要讓戰友們在思想上**覺知**到和**主動拒絕**被共匪種下的魔性和動物性思想，然後讓戰友們學會互相**愛敬**，恢復人性和神性思想。正如王陽明所講“**聖人之道，吾性自足”**，陽明心學其實是儒家＋佛法。佛陀自菩提樹下悟道之後也說過“**奇哉！一切眾生，皆具如來智慧德相。只為妄想執著，不能證入**”。所以，每一位戰友都是可以恢復人性神性思想的，但是要先得遇明師指點，學會覺知、分別、放下內心的妄想和執著。

根據我在新中國聯邦建國後，在德教雲書院六個月的Discord群管理實踐經驗，對於Discord群的管理問題我們只有一條規則，就是**群內戰友們互相尊重，互相關愛，互相學習，互相幫助，不得相互攻擊挑釁**。若有違反，請戰友們互相勸告，如果多次違反請與管理員聯繫。管理員會酌情禁言，取消違反者部分discord功能，情節嚴重者或永久性踢出群。

擁有禁言和踢人權限的只有幾個管理員，人不能多，當發現有違反規則的人存在的時候，第一時間派一位管理員和違反規則的人私聊勸告，絕不被對方激怒生氣，也不攻擊挑釁對方，時刻牢記不給對方留下口實。

這個制度如果要落實的好，就需要管理員很有德行和智慧，所以所有的管理員包括見習管理員，首先應該努力學習提升自己的德行，提升自己的智慧，一個基本要點就是管理員首先要做到**理直氣和**，對所有戰友做到**愛與敬**，同時受所有戰友監督。

下面具體談一談**愛與敬**。
![This image has an empty alt attribute; its file name is %E7%88%B1.png]()![This image has an empty alt attribute; its file name is %E7%88%B1.png](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/22172754/%E7%88%B1.png)
所謂**愛**，用繁體字更能體現他的意思，**愛**。中間有一個心，外面是一個感受的受，所以合起來就是**用心去感受對方的感受**，而不是自己的感受，不去做傷害對方的事情。所以管理員首先要做到用心感受戰友的感受。所謂**敬**就是**尊重**。 《孝經-天子章》談如何做天子時是這麼說的，“**愛親者不敢惡於人，敬親者不敢慢於人。愛敬盡於事親，而德教加於百姓，刑於四海**。”　爆料革命郭先生**愛敬盡於事親**做的非常好。眾所周知，郭先生是個大孝子，對待自己的父母首先做到愛敬。郭先生在直播裡面也反复叮囑戰友們，一定要愛自己的父母，這是每一個人都可以做到的。愛自己的父母，所以不敢討厭別人，尊重自己的父母，所以不敢不尊重其他人。然後郭先生把這份感情延展到其他人身上，對待戰友們就像對待親人一樣，郭先生用自己的言行給戰友們做好了榜樣，告訴戰友們什麼叫作**愛敬盡於事親**。下一步就是**德教加於百姓**，所有的戰友們都要主動地去學習與思考如何愛敬，如何提高自己的德行和智慧，努力地去踐行爆料革命的正道主義。

其次談一下管理員的**管理**。其實所謂的管理員也不應該稱為管理員，管理是西方的概念。**西方的管理**是**放羊模式**的，有一個牧羊人，一群羊亂跑，然後是胡蘿蔔和大棒，前面有個胡蘿蔔就是利益，後面有個大棒，不聽話就要受到懲罰，合起來就是西方管理的精髓，**激勵和懲罰**。

而**中華傳統智慧**則不說管理，而是只談**領導**。郭先生經常叫Sara姐是大領導，這是對Sara姐的讚賞，同時也是鼓勵和期許。領導的導，繁體字這麼寫，**導**，上面是**道**，下面是**寸**。也就是說作為一個領導，做事要合道，合天道地道人道的規律，則**天之明，因地之利**，一分一毫一寸都要合道，不能有偏差。正所謂差之毫釐，謬以千里。

**导**，是要**导正行为**，**首先要导正自己的行为**，**以身作则**。后面的人才能跟着学习，人是人，不是羊，得心甘情愿地跟着学习才行。一个基本的道理就是要把人当人看，而不是羊，所有的人就像他的孩子一样，领导像父亲一样在前面走，**引领正确的方向**。这才是真正的领导力。所以中华传统智慧讲，人要先修身，自己获得快乐幸福，才能再给别人带来快乐，给家人带来幸福快乐就是齐家，然后才有可能利益天下，给全天下人带来幸福快乐。

綜上所述，對**鳳凰農場－舊金山金喜站**的規章提出三點提案，和戰友們**約法三章**：

**１．標明了不可以外傳的消息絕對不可外傳。**

**２．群內戰友們互敬，即互相尊重。**

３**．群內戰友們互愛，即互相關愛，不得相互攻擊挑釁。**

若有違反，請戰友們互相勸告，如果多次違反請與管理員聯繫。管理員會酌情禁言，取消違反者部分discord功能，情節嚴重者或永久性踢出群。
