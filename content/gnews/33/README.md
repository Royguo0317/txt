伊维菌素是一种可大规模获得的低成本药物，它具有抗病毒抗炎症作用，且没有免疫抑制副作用，是一款超过40年应用的口服抗寄生虫药物，其发现获得了2015年诺贝尔医学奖。

该药物对Covid-19强大的抗病毒和抗炎症作用，在实验室和临床试验研究中均得到了证明。



在世界各地的超过40项临床研究表明，依维菌素不仅在感染初期减少病毒血症方面极为有效，而且在继发阶段依旧效果良好。我必须说伊维菌素的治疗效果让我感到惊讶。我没想到35年前批准的用于其他适应症的药物对Covid患者如此有效且耐受良好。

2020年12月参议院国家安全委员会举行了一场关于伊维菌素的听证会。当时，皮埃尔·科里（Pierre Kory）博士引用了各种证明其功效和安全性的[研究数据](https://covid19criticalcare.com/wp-content/uploads/2020/11/FLCCC-Ivermectin-in-the-prophylaxis-and-treatment-of-COVID-19.pdf)来解释伊维菌素的工作原理，以至于说服了美国卫生部门（NIH / FDA / CDC）修改对该处方药用于Covid-19的禁令。尽管美国国立卫生研究院（NIH）尚未宣布完全支持其使用，但对伊维菌素的立场已从“反对”更改为实质上“中立，有待进一步的测试数据”。一月中旬以来，已经可以在美国针对Covid-19开具依维菌素处方。

最初我也很怀疑，但是当我在Covid患者中开始使用它后，我坚信伊维菌素确实可以成为对抗Covid19的游戏规则改变者。在阿根廷的一项研究中，788名每周服用一次12毫克伊维菌素的医疗专业人员中竟然没有一个受到感染，而在感染Covid的患者中通过使用这种药物，防止了患者的住院或进一步加重病情。

特别是当我亲眼目睹了已经有数月呼吸困难的患者，以及急性重症需要吸氧或插管的患者，在服用了伊维菌素后，非常迅速的好转。这些证据说服了我迅速改变看法。

世界各地的文献证明了该药物在感染的各个阶段的有效性和安全性，[详细数据](https://ivmmeta.com/)。

**这种药物如何起作用？**

伊维菌素具有不止一种可能对Covid-19有效的作用机制，主要机制之一是它能够与冠状病毒Spike1蛋白结合，从而防止病毒附着在ACE2上，ACE2是病毒进入我们细胞的主要通道，进而降低病毒进入人体的能力。如果把Covid留在细胞之外，那么它就很容易被免疫系统消灭。因此，与单克隆抗体不同，伊维菌素可以对抗所有变体。

病毒的蛋白质对接区-尖峰-不仅与ACE2受体结合，而且还与其它受体结合，这些受体依赖于唾液酸CD147受体并被称为a7nAChr的胆碱能受体。我不想讲技术，但是这个前提可以帮助我们了解Covid如何攻击我们的身体。

一旦病毒进入我们的肺部并造成损害，病毒就会进入血液，并附着在红细胞和血小板上，因为它们都被唾液酸和CD147受体包围，因此，它会侵入所有血管，并“释放”到内壁（我们血管的内部），然后导致全身性血管炎，这就是为什么我们大多数人最终无法克服感染的原因。

当病毒在血管内皮细胞内复制时，会使其发炎并形成红细胞和血小板的粘滞聚集体，最初阻碍血管的小凝块引起毛细血管的局部循环，随后逐渐肿胀并阻塞越来越大的血管。当这种情况在肺部发生时，会导致氧气不足，从而损害我们所有的器官，例如大脑，肝脏，肾脏和心脏，尤其是对老年人，糖尿病患者和患有先前疾病的人是致命的。

病毒在血液中移动的机制也解释了为什么Covid在高血糖患者（如糖尿病患者和糖尿病前期患者）中更具致死性。因为由唾液酸和CD147组成的受体基本上是糖。在糖尿病患者中，这些唾液酸含有更丰富的糖（葡萄糖），因而令病毒的黏附性更好，其全身传播甚至更容易。

病毒对目标的选择性也发生在血型上，例如那些具有A或B型血的人，其红细胞表面具有更多唾液酸/糖存在。而0型血对病毒的敏感性较低，因为它的红细胞内唾液酸受体较少，因此病毒的滞留率较低。 糖尿病患者或具有A和B血型的患者在统计学上会具有更严重的病症。

伊维菌素可以在患者体内（红细胞，血小板和血管）与病毒之间形成一道屏障，因此促进了在血管中携带氧气的红细胞的流动，因此患者可以奇迹般地开始正常呼吸并再次正常地充氧24-48小时，并导致各个器官功能的迅速改善。

**“游戏规则改变者”**

药物实际功效的评估必须通过对同行评议的科学出版物的筛选来进行。在这方面，秘鲁已在全体人口（超过3300万人）中大量使用了伊维菌素，结果令人印象深刻。这项研究表明，与延误分发伊维菌素的地区（例如首都利马）相比，死亡率下降了（峰值为90％）。此外在法国和美国完成的研究也取得了良好的效果。

伊维菌素是一种非常安全的药物，自从最初被批准用于人类以来已使用了40亿剂，任何制药公司在任何地方都可以容易地以药丸的形式大量生产伊维菌素，因为它是免专利产品，因此成本极低。

总之，伊维菌素在Covid的不同阶段均有效的原因是，它不仅具有抗病毒作用，而且如上所述也具有抗血小板作用（与肝素和阿司匹林作用相似但更具针对性）。最后，它具有类似于可的松的强大抗炎作用，但不像可的松那样具有免疫抑制作用，因此，在使用过程中，T细胞和B细胞（产生抗体）的免疫反应继续发展，作用于转录因子NF-KB（当被Covid过度激活时会引起细胞因子风暴）。
