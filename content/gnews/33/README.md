## 欧洲8国暂停牛津阿斯康力疫苗疑似会引发血栓

引自

《7个欧洲国家因 “严重” 的血栓报告而停止阿斯利康新冠疫苗》

更多的国家已效仿丹麦，暂停了对阿斯利康-牛津 (AstraZeneca-Oxford) COVID疫苗的使用批准。挪威、丹麦、爱沙尼亚、立陶宛、卢森堡、拉脱维亚和意大利现已选择停止使用该疫苗。这为欧洲迟缓的疫苗推广带来更多问题。随后，冰岛也成为最新一个暂停使用该疫苗的欧洲国家。

EMA (相当于欧洲的FDA) 证实，已经统计了不少于30起接种疫苗的患者出现危险血栓的事件，包括丹麦至少一例患者死亡的事件。

在停止使用前，最近接种该疫苗的患者发生了血栓事件。奥地利报告了两起，但维也纳允许继续使用该疫苗，至少现在仍是如此。在丹麦和其他国家也有严重的血栓病例报告。丹麦卫生部长在确认停药的推特中表示，目前无法确定严重血栓的案例是否与疫苗有关，但这种情况肯定值得深入调查。他说：”我们很早就行动了，需要彻底调查。”

就在几天前，意大利总理马里奥-德拉吉 (Mario Draghi) 停止了一批运往澳大利亚的阿斯利康疫苗，这标志着欧盟领导人首次援引规则–优先考虑国内疫苗的使用。现在，意大利当局在发生两起可疑死亡事件后，暂停了当前一批阿斯利康疫苗接种。

与此同时，布鲁塞尔和伦敦的卫生当局驳回了这些担忧，坚称该疫苗是安全的，同时继续推进批准强生公司单剂疫苗的计划。欧盟药品监管机构表示，在对强生公司的数据进行 “彻底评估” 后发现该疫苗符合疗效、安全性和质量标准，建议授权所有18岁以上的成年人使用该疫苗。强生疫苗是第四种获得欧盟内部使用许可的疫苗。

欧洲备受诟病的疫苗接种推广刚刚遇到了另一个障碍，丹麦卫生当局越来越担心有害的副作用与阿斯利康-牛津疫苗有关。这种廉价的COVID补救措施本打算帮助欧洲赶上美国，英国和以色列。

丹麦当局周四暂时停用阿斯利康的COVID-19疫苗，此前有报道称，有患者体内形成危险血栓的案例，其中包括一起发生在丹麦的此类事件。丹麦卫生局局长Soren Brostrom在一份声明中说：”我们和丹麦药品管理局都必须对来自丹麦和其他欧洲国家可能出现严重副作用的报告作出反应。” 丹麦药品管理局表示，随着当局在欧盟其他成员国的协助下对血栓事件展开调查，暂停使用该药的时间将持续14天。

尽管没有说有多少关于血栓的报告，但奥地利也已经停止使用一批阿斯利康疫苗，同时调查一起因凝血障碍而死亡的事件和一起因肺栓塞而患病的事件。

阿斯利康公司声称，其疫苗受到严格的质量控制，”没有任何证实与疫苗相关的严重不良事件”。该公司正与奥地利当局联系，并将全力支持他们的调查。该公司还表示，接受过阿斯利康疫苗的人，发生（血栓栓塞事件）的数量不高于普通人群。欧洲药品管理局周三声称，没有证据表明阿斯利康疫苗与奥地利的两起血栓病例有关。

而至少有一位投资者称，丹麦以血栓问题为由暂停阿斯利康的COVID疫苗，说明 “寻找潜在安全问题的检测系统正在发挥作用”，而这些 “安全事件” 最终大多会与自然过程有关，而不是注射疫苗。

“很高兴看到安全信号检测系统起作用，重要的是，任何安全信号都要使用正确的方案进行跟进，” Shore分析师Adam Barker告诉彭博社。他说，疫苗三期试验的数据表明，”你会预料到大多数安全信号最终不会与疫苗相关联。”

然而，”很难对股东价值的影响做出判断，” 他补充说，因为有 “很多随时变化的部分。” 但归根结底，任何疗法都要进行风险/回报的权衡；”只有当所有的数据都最终确定且清晰的时候，你才能有信心对这个决定做出判断。”

尽管如此，阿斯利康公司的股价周四还是暴跌，因为有迹象表明欧洲疫苗的推广面临新的怀疑和障碍。股价在3月11日伦敦午盘交易中下跌超过2%。
