据法新社报道，在北京强行全面改变香港的选举制度后，七国集团要求中国履行在香港的承诺，结束对民主人士的压迫。

![]()![](https://gnews.org/wp-content/uploads/2021/03/56834738_303.jpg)图片来源：德国之声中文网

中共两会3月11日通过了一项改革方案，称只有“爱国者”才能管理这座城市，而在英国1997年移交控制权之前，中共曾向国际保证香港会拥有独立的政府系统。

英国、加拿大、法国、德国、意大利、日本和美国以及欧盟在一份联合声明中表示，这样的决定强烈表明中共国大陆当局决心消除来自香港的反对声音。

七国集团高级外交官表示，他们“呼吁中共国和香港当局恢复香港政治机构的信誉，结束对那些促进民主价值观的人的无理压迫。”

“应该相信香港人民为了香港的最佳利益投票。允许讨论不同观点，而不是压制它们，是确保香港稳定和繁荣的正确途径。”他们说。

北京无视早些时候国际关于香港的警告，并于去年通过了一项严厉的安全法案，用于拘留异议活动人士。

这些行动发生在2019年香港爆发的反对北京侵犯香港独立民主自由的抗议活动受到中共大规模的、甚至是暴力的镇压之后。
