2013年5月2日，nature刊登了一篇题为[Scientists create hybrid flu that can go airborne-H5N1 virus with genes from H1N1 can spread through the air between mammals](https://www.nature.com/news/scientists-create-hybrid-flu-that-can-go-airborne-1.12925)的文章。文章指出，中共国农业科学院哈尔滨兽医研究所与甘肃农业大学动物医学院合作，陈化兰，于康震，邓国华，周继勇，李泽军等人，利用具有高度致命能力、但从未在人类传播的H5N1禽流感病毒，与极易感染人类H1N1流感病毒通过基因重配，制作成了5种超级病毒，这些超级病毒具有通过空气迅速在哺乳动物间传播的能力。他们的研究成果发表于Science杂志[https://science.sciencemag.org/content/340/6139/1459](https://science.sciencemag.org/content/340/6139/1459)。

2013年5月6日，《中国科学报》、《中国科学院》也刊登了题为《H5N1与甲流病毒结合或可人传人》对陈化兰团队这一成果进行了后续报导。

福克斯新闻第二天就进行了报道，警告“新的病毒株有可能逃脱实验室，这可能导致全球大流行并导致数百万人死亡。” [Chinese scientists slammed for creating mutant influenza virus in laboratory](https://www.foxnews.com/health/chinese-scientists-slammed-for-creating-mutant-influenza-virus-in-laboratory)

WIRED当天就撰文指出，“尝试在实验室中制造人工毒株，如果这种人工毒株被意外释放或用于邪恶目的，则可能构成潜在的全球健康威胁”，“公共卫生官员最担心的是将一种病毒的致死性与另一种病毒的可传播性结合在一起的混合体，造成致命的全球性大流行… …一些科学家认为风险不会超过收益，即使有制度保障措施也无法充分减少发生事故的机会。公众对此类实验感到不安……”[Chinese Scientists Create New Mutant Bird-Flu Virus](https://www.wired.com/2013/05/h5n1-h1n1-reassortment/)

英国皇家学会前会长Baron May说，“他们声称是为了帮助研发疫苗，但实际上他们是受到了没有任何常识的野心的驱使。他们正在创造能在人类之间传播的危险病毒，这是非常不负责任的。”

法国病毒学家Simon Wain-Hobson说“中共研究人员创造的新毒株中，可能有部分容易在人类之间传播，具有高致命性H5N1禽流感的部分特征”。

就连中共国自已的媒体科学网都在2013年5月6日以《中国科学家创造新禽流感毒株被批极不负责任》来报道陈化兰等人这一不负责任的行为。称 “国外科学家批评了中共国研究人员的做法极不负责任”。事实上，在这之前，全世界病毒界对有关H5N1高致命病毒的研究非常谨慎，一直禁止H5N1病毒相关研究。是陈化兰等人于2013年1月联名公开信，才得以重新启动相关研究工作。

然而，尽管这种威胁全人类安全的行为遭到了国内外媒体的批评，但中共国政府随后对陈化兰进行了一系列褒奖，这就更让人怀疑究竟谁是这些研究的幕后推手。2013年，中共国政府授予陈化兰等人国家自然科学奖二等奖，2016年授于其全国杰出科技人才奖，2017年让她成为中共国科学院院士，2018年给予其全国妇联副主席，2019年更是给予中共国医学科学院学部委员。

根据中共国的公开资料：陈化兰的研究方向为禽流感病毒跨种属感染及其致病性的分子遗传与分子致病机制。怎么和现在的中共CCP病毒看起来这么像呢？
