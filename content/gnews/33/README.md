04月22日，美国务院发言人内德·普莱斯表示，与伊朗就恢复核协议的谈判取得了一些进展，但大部分工作仍在进行中。

普莱斯说，“已经取得了一些进展。这个方向的大部分工作可能还在进行中。” 他谈到，谈判成功的道路上面临着许多挑战，特别是对各方的不信任和通过调解人进行谈判。尽管如此，与伊朗的对话是建设性的，并表示希望在这个方向上取得进一步进展。然而，在涉及制裁时，美国将只在与伊朗核计划直接相关的领域做出让步。

普莱斯说：“伊朗不应该有任何幻想我们不再对侵犯人权、部署弹道导弹计划、支持该地区的恐怖主义负责。”与此同时，美国国务院发言人说，美国准备取消的制裁清单已经传给了伊朗。

伊朗在2018年与俄罗斯、美国、英国、德国和中共国达成了一项核协议。根据该协议，这些国家承诺放弃对伊朗的制裁，而后者则有义务削减其核武器计划。2018年，美国总统唐纳德·川普的政府退出了该协议。川普指责伊朗违反了协议，并对伊朗进行制裁。德黑兰的回应是拒绝遵守协议的某些条款。

今年4月初，美国宣布打算重返该协议。伊朗拒绝参加直接谈判，并要求首先解除制裁。该国当局表示，只有在这之后，他们才会准备好履行协议的条款。

美国国务院随后表示，不可能取消自2017年以来实施的所有制裁。同时，美国表示准备取消那些与该协议直接相关的限制。
