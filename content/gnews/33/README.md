【2020年香港施政报告专题】（七）《施政报告》写法用词如内地干部汇报文章

搜集： KEN老广、Peacelv

编撰：文燕
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/27004207/jpg-image001.jpg) 图片：立场新闻
据【明报】报道，早前被DQ的4名民主派议员而提出请辞，今日（25日）在立法会外回应《施政报告》，民主党主席胡志伟形容，今年的《施政报告》不属港人，称林郑月娥的用字似内地干部，又说她并没为所做的恶行感悔疚，只「照本宣科」读出中央未来管治香港的内容。

另有港媒指行政长官林郑月娥今日公布施政报告，大篇幅感谢中央支持，又有多项与大湾区有关措施；民主党主席胡志伟批评，报告并非为香港人而订，认为林郑月娥一心一意做内地的干部、随从，「报告内容的写法，其实好似内地高干、省长、市长写着一份报告给中央政府。」公民党议员谭文豪则形容是「大湾区报告」，相信许多香港人阅后会反问「关我什么事？」
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/27005013/jpg-thestandnews_127000412_389519878863640_583124652803919032_n.jpg) 图片：立场新闻
战友点评：

笔者看完整篇施政报告，发现林郑除通篇赞扬中央，感谢中央，就是报告中的遣词用字的确内地化，语句不是港式中文。

**1）报告第5项： 香港享有「近水楼台先得月」的优势，而中央在「一国两制」下对香港一贯支持，可以为香港经济发展注入新动力，而香港也可以继续发挥「国家所需、香港所长」的作用，积极参与国内国际双循环，更好融入国家发展大局。**

第5项显然有张晓明手印，有媒体翻查资料，发现港澳办常任副主任张晓明分别在2015-2016年多次提过「国内国际双循环」，估计这一段应该是张晓明添加的。 2016年张晓明接受媒体采访时指出，国家「十三五」规划的实施、「一带一路」建设，是香港的机遇。近水楼台先得月，香港不应错失这一大机遇。

「近水楼台先得月 」一句就漏了底牌，证明当初这份施政报告上递中央后，经过中央照肺后，还要修改内容，还要添加许多内容，一直到合中央心意为止。

**2）报告第9项：去年中共十九届四中全会通过把「坚持『一国两制』，保持香港、澳门长期繁荣稳定，促进祖国和平统一」作为我国国家制度和治理体系所具有的显著优势之一，明确中央对「一国两制」方针是坚定不移的。**

「明确……不移的」这种表达方式，语句绝对不是港式句子，这一句佐证了施政报告有国内人参与书写和添加。

报告第19项：「全国人大常委会在本月11日通过有关立法会议员资格问题的决定，明确议员因宣扬或者支持港独主张」，这明显是内地式中文，绝非港式中文。

**3）报告第39项：当中表示将以改革创新为根本动力，加快形成以国内大循环为主体、国内国际双循环相互促进的新发展格局。 「双循环」策略的要点是扩大内需……**

其他还有「国内国际双循环」、「 中共第59届五中全会，最近通过关于制定国民经济和发社会展的第14个五年归划」、「更好利用国内循环来促进国外循环。 」这些都不是港式用词和术语。例子很多，笔者不胜枚举，可见林郑该份报告都是党内人士代笔。

由此可见，今年的施政报告需要经过内地高官的批改，那么报告押后也是「情理之中」。除了部分内容中央不同意外，也有中央政府要添加部分内容。因此我们可以毫不夸张地说，整份施政报告都不是香港的施政报告。

的确，2020年的香港施政报告，内地官员笔迹处处，这代表了从今年开始，往后香港施政报告内容都会由京官执笔，未来香港长官只是施政报告的宣读者，也进一步证明中央全面接管香港。
