**西班牙2021年2月16日电/西喜社——**
![]()![](https://gnews.org/wp-content/uploads/2021/02/852798.jpg)
澳大利亚战略政策研究所2月15日在其网站《战略家》上发表文章称，习近平对中共国海岸警卫队的控制，以及授权海岸警卫队在中共定义为本国的地方对外国船只使用武力的新法律，是一个重大的变化，但迄今为止引起的关注远远低于它应有的程度。

也许这是因为在川普(Donald Trump)担任美国总统的最后几周和拜登(Joe Biden)上任初期，习近平已经在多个方面采取行动，以维护中共国的权力并承担风险。一些举动—如对川普高级官员、其家人和雇佣他们的公司实施制裁—理所当然地被视为报复性措施而受到关注。另一些，比如人民解放军侵入台湾领空，则是为了进一步推动北京孤立和恐吓台湾的运动，并考验美国和国际社会的决心。

综合这些举动来看，习近平对海岸警卫队的大胆行动表明，他正在加大愿意承担的风险，与其他国家对抗，并利用他所拥有的杠杆来投射中共国力量。而海岸警卫队的举动是给他提供了非常实用的新工具，以造成破坏和不安全，并以其他国家—特别是军队—无法、可能也不应该匹配的方式采取行动。

中共国国家媒体对这一法律轻描淡写，称其与其他国家的做法类似—但事实并非如此，因为这标志着中共国在使用武力时将无视国际海洋法和国际法庭的裁决，并通过中共国对其海洋边界的单方面定性来界定海岸警卫队使用武力的管辖权。而中共国海岸警卫队在实践中可能的运作方式也将大不相同。

我们已经习惯了中共国渔船队和中共国民兵舰艇恐吓其他国家的船只，甚至撞上他们以达到目的的故事，特别是在越南和菲律宾声称主权的南海海域，但也到了印尼专属经济区的纳土纳群岛。

中共国船只在去年击沉了菲律宾和越南的渔船，事后似乎也不太愿意履行对需要帮助的水手提供援助的义务。我们也已经习惯了中共国海岸警卫队对中共国渔船队的跟踪，如果他们接触到其他国家的船只，随时可以进行干预。

但现在不同的是，习近平通过这项新法律，告诉他的海岸警卫队要成为海上的“战狼”—并使用武力，包括致命的武力，以维护中共国的利益。

中共国海岸警卫队一直在建造一些新颖的船只，让它不仅可以用船上的武器，还可以用船只本身来运用武力。像万吨级的 “海迅 “号这样的海岸警卫舰，不仅比许多在南海和东海活动的海军舰艇大，而且还加强了船体的强度，专门用于故意撞击其他船只—”肩扛 “是海军的术语。

想象一下，像 “海迅 “号这样专门设计的大型舰艇’肩扛’越南、菲律宾、印尼甚至美国的海军舰艇，在习近平的法律和他的指挥下，中共国的机构和官员能够与世界进行艰难的’斗争’。

这些海军（以及澳大利亚皇家海军）运营的舰艇没有这样的强化船体。它们的设计是为了抵御一些损害，主要是来自武器的损害—而最主要的方法是防止导弹命中。

为了了解与大型船只相撞对这类舰艇造成的损害，我们以挪威的英斯塔德号护卫舰为例，该舰在2018年与一艘油轮相撞，然后被故意搁浅并沉没。最后以护卫舰被拆解而告终，因为损伤（来自碰撞和在水下4个月）太大，无法修复。

所以，我们可能需要思考的不是中共国海岸警卫队向他国船只开火的问题，而是如何处理那些满脑子狼性战士精神、被习近平特许惹事的海岸警卫队指挥官，以及如何对付那些不使用武器就能伤害他人的船只。

不使用武器就能造成伤害的能力，让中共国海岸警卫队在遭遇战中变得轻松。一艘军舰如果不能在不损坏自己的情况下撞回去，就只能选择后退，把遭遇战交给中共国，或者使用武器，率先开火。这两种情况都不是很好的选择。

中共国海岸警卫队这样使用这一新法律和它的船只，也许会在北京得到欢呼，让那里的激进民族主义者高兴。但如果有中共国领导人认为这种 “非致命性 “的武力使用是一种低成本、无政治成本的好事，那就大错特错了。

海岸警卫队的船只不仅撞上、撞坏甚至可能撞沉渔船，还撞沉其他国家的军舰，无论北京如何定性，这都是一系列极具升级性和侵略性的行为，尤其是在有争议的水域。也许习近平需要听听其他国家领导人的意见，然后我们才会开始看到这样的水上动作。

拜登给习近平的电话，也许德国总理默克尔等领导人的电话，可以在庆祝投资协议的同时提及这个问题。在与拜登的第一次谈话中，习近平表示，美国需要表现出谨慎—好吧，这是他自己可能会接受的信息。

在基本的战术层面上，用智能手机捕捉中共国海岸警卫队行动的视频，并制定一个通信计划，在北京编织虚假信息的故事之前—’这不是我们。不是我们，没有发生。他们先做的’—也是有道理的。
