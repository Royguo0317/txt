## 帕勒（Parler）起诉亚马逊 要求恢复其使用该平台

据美保守派网络媒体DJHJ（DJHJ MEDIA）1月11日报道，周一，社交媒体公司帕勒（Parler.com）因为托管它的社交媒体平台亚马逊（Amazon）暂停了其应用程序，而起诉了该公司。称该服务的暂停违反了反垄断法，也违反了公司合约。

毫无疑问，帕勒所说属实。左派类型的公司也无疑已经在以对他们有利的方式操纵游戏。亚马逊的争议是，尽管他们尊重帕勒允许人们在其平台上发表自由言论的权利，但他们在服务条款上有一项政策，不允许发表煽动暴力的言论。我看到亚马逊将面临的一些问题，其中之一是：它们不会以相同的方式对待其为之服务的其他社交媒体平台。另一个是：平台受《通讯规范法》第230条保护，不对其网站内容负责。发布者应承担责任，因为他们控制所有出现在其网络空间中的内容，而平台仅是人们发布自己内容的一个媒体来源。在230条款的保护下，平台被允许以他们认为违反其服务条款（如种族主义、暴力威胁等）为由管控内容。

但在过去几年中，脸书（Facebook）、推特（Twitter），还有其他公​​司改变了他们的服务条款，使其能针对他们不喜欢的用户，而允许他们喜欢的用户有同样的违规行为却不受约束。这并不是230条款所要表达的主旨。

首先，苹果和谷歌都已将帕勒的应用程序从其应用程序商店中移除，基于与目前亚马逊几乎相同的理由，给予人们自由言论的权力对我们共和国来说是一种威胁，我们的建国先父们都会因此不安。

在其诉讼中，帕勒这个平台在很大程度上以言论自由的态度来放松平台上的内容，因此受很多被脸书、推特和其他左翼运营的社交媒体平台骚扰的保守派所喜爱，他们请求法官下令复原该平台在线状态。

尽管亚马逊是一家公司而非政府机构，第一修正案也适用于此。帕勒公司陈述了理由：

1.这是一项针对禁制令救济的民事诉讼，包括临时限制令和初步禁制令救济以及损害赔偿。上个月，被告亚马逊公司和普遍使用的社交媒体平台推特签署了一项为期数年的协议，因此亚马逊可以支持每天发送数百万条推文。亚马逊当下为帕勒提供了同样的服务，帕勒是推特的保守派微博用户的替代品和竞争对手。

2.当推特两个晚上前宣布将在其平台永久禁止川普总统，保守派用户开始大量逃离推特，转向帕勒。倾巢而出者规模之大，以至于第二天，也就是昨天，帕勒成为了苹果应用程序商店下载第一的免费应用程序。

3.然而昨晚，亚马逊宣布将从1月10日（周日）太平洋标准时间下午11: 59起暂停帕勒的帐户。它声明暂停的原因是：亚马逊不确信帕勒可以就那些在其平台，鼓励或煽动针对他人的暴力行为的内容适当的进行监管。 然而周五晚，推特上最受欢迎的推文之一是“ 悬吊麦克•彭斯（ Mike Pence）”。但是，亚马逊没有计划，也没有发出威胁要暂停推特的帐户。

4. 亚马逊决定有效终止帕勒账户显然是出于政治动机。它也显然有意减少微博服务市场的竞争，从而使推特受益。

5.因此，亚马逊与被告推特一起违反了《谢尔曼反垄断法》第1节。亚马逊还违反了与帕勒的合同，该合同要求亚马逊在终止服务前，提前30天通知帕勒，而不是亚马逊在实际只有不到30小时才通知。最后，考虑到在预期不久的将来，将有数百万用户注册，亚马逊是在对预期的经济优势故意进行干扰。

6.该紧急诉讼寻求一份针对被告亚马逊的临时限制令，以阻止其在今天结束时关闭帕勒的帐户。这么做等同于拔掉用于维系医院病人生命的设备的电源插头。这将扼杀掉帕勒的事业，在它即将飞黄腾达之际。
