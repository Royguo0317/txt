继上周宾夕法尼亚州议会派代表团考察、取经了亚利桑那州的“国家审计”后，佐治亚州也将派团队明天（6月8日）考察佐治亚州的国家审计。

《门道专家》（The Gateway Pundit）发表文章，引用了《美国新闻网》（ONA）记者克里斯蒂娜·博布（Christina Bobb）的话，可能会有更多的州来参观学习。

文章说，周一（6月7日）上午，记者克里斯蒂娜·博布告诉《美国新闻网》，有几个州可能会奔赴亚利桑那州，加入（参观）历史上第一次全面的法庭审计的旅行。

《门道专家》的权威人士上周报道说，由州参议员道格·马斯特里亚诺（Doug Mastriano）、州参议员克里斯·杜什（Cris Dush）和州众议员罗伯·考夫曼（Rob Kauffman）组成的宾夕法尼亚州代表团进入退伍军人纪念体育馆（the Veterans Memorial Coliseum）进行了现场参观。

参议员道格马斯特里亚诺分享了他对这一非凡过程的看法。

克里斯蒂娜·博布补充说，一支来自佐治亚州的队伍星期二上午也将进入退伍军人纪念体育馆参观这个过程。

她没有具体说明谁将被派遣，但《门道专家》的乔丹康拉德森（Jordan Conradson）明天也将在现场。

克里斯蒂娜还说，“我希望本周在这里看到的不仅仅只有佐治亚州。”

随着法庭检验工作台开始（搬进）地面，取代了清点工作台，审计工作也即将完成，各州必须急于观察到这一过程，以便在全国复制这一过程。

未触及托盘的数量仍然是7个，但他们将开始（清点其中）一个，甚至今天可能开始清点两个托盘。

我们期待本周或下周会看到手工清点结果，我们也期待本月底会看到法庭审计结果。

这史诗般的，历史性的亚利桑那州审计很快就要结束了，整个联邦的各州都希望复制这一过程，因为这是选举诚信的黄金标准。
