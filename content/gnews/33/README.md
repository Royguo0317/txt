## 中共国特使威胁采取‘坚决的行动’回应‘美国干涉’中共国内部事务

据《thegatewaypundit》 作者：克里斯蒂娜-莱拉 ([Cristina Laila](https://www.thegatewaypundit.com/author/cristina/))，2021年3月18日报道：

![]()![](https://gnews.org/wp-content/uploads/2021/03/1-175.jpg)

美国国务卿布林肯（Antony Blinken）本周早些时候访问了日本和韩国，并表示美国将继续致力于无核化，并反击中共国在亚洲日益增长的“胁迫和侵略”

当布林肯本周早些时候访问该地区时，朝鲜领导人金正恩的妹妹金与正警告拜登，如果美国想要“睡的安稳”，就不要“制造麻烦”。

周四，美国国务卿布林肯和国家安全顾问杰克 沙利文在阿拉斯加安克雷奇会见了中国最高外交官杨洁篪和国务委员王毅。

布林肯与中共国最高外交官的会晤在周四晚上迅速恶化。“我们对中共国行动的深切关注，包括在新疆、香港、台湾的行动，对美国网络的攻击，对我们盟友的经济胁迫”布林肯在公开谴责中说，据路透社报道。

“其中的每一项行动破坏了以规则为基础的秩序，该秩序维持全球稳定”布林肯说。

在与民主党对应的谈话要点中，这位中共国外交官抨击美国“虐待少数民族”，并告诉布林肯，美国应该处理自己事务，中共国也会处理自己事务。

据路透社报道：

冗长的回应美国的开场陈述，杨予以回击，指责美国利用其军事和金融霸权施压其它国家，滥用国家安全威胁国际贸易的未来。他说，新疆、香港和台湾都是中共国领土不可分割的部分，中共国坚决反对美国干涉中共国内政。杨说，由于黑人被“屠杀”，美国人权处于低点。并补充说美国应该处理自己的事务，中共国也是。

杨说有必要放弃“冷战思维”和对抗，并补充说 我们看待和美国的关系的方式就像习总加速师说过的，我们希望看见不对抗、不冲突，相互尊重和与美国合作双赢

法新社报道中共国特使随后威胁用“坚决的行动”回应“美国干涉”它的内部事务。
