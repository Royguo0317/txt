# **独家：拜登选择中共国军事同盟者为美国负责政策事务的国防部副部长**

**拜登总统任命的美国负责政策事务的国防部副部长科林•卡尔（Colin Kahl）曾经在斯坦福大学主导了一个与中国共产党和人民解放军有着“多年”关系的中心，还曾参加过中共国政府“赞助”的会议。**

卡尔于2018年1月以斯坦福大学弗里曼•斯波格利国际问题研究所高级研究员的身份加入该中心，并担任“国际安全与合作中心”（CISAC）的联合主任。

在此任命之前，他曾担任时任副总统拜登的国家安全顾问，也为他赢得了宾夕法尼亚大学“拜登中心”战略顾问职位的机会。

除了斯坦福大学“大量”漏报的“收到的数以百万美元计的中共国匿名捐款”之外，卡尔所在的中心还与中国共产党有着极为密切的关系。

**与中共国的军事联系。**

根据美中关系全国委员会（NCUSCR）的报告，国际安全与合作中心CISAC接待了许多中共国军事和政府相关实体的研究员，其中包括被联邦调查局（FBI）标记为有间谍活动并被美国列入黑名单的实体：

*国际安全与合作中心接待了来自许多外国的访问学者，其中包括许多披着“和平与合作项目”外衣的中共国学者。 […]这些学者来自上海社会科学院、中国工程物理研究院（第九研究院）、中央党校、国防大学以及外交部。*

[!\[\]()!\[\](https://spark.adobe.com/page/ZN6CkgcMBp1z1/images/0087d09a-32df-4642-911d-a7f29e757c4e.png?asset_id=42457638-6a63-4245-a1d4-70d9a9c34ca8&amp;img_etag=%22fe018d4b668eebc0e2e85425552196a2%22&amp;size=2560)](https://spark.adobe.com/page/ZN6CkgcMBp1z1/images/0087d09a-32df-4642-911d-a7f29e757c4e.png?asset_id=42457638-6a63-4245-a1d4-70d9a9c34ca8&amp;img_etag=%22fe018d4b668eebc0e2e85425552196a2%22&amp;size=1024)

联邦调查局表示，其中一个实体——上海社会科学院——是被用作“中共国情报搜集和海外间谍招募的前线组织”。

联邦调查局描述说，中国共产党依靠上海社会科学院的工作人员来“发现和评估”潜在的可培养的西方间谍。联邦调查局还描述说，中共国国家安全部官员也“利用上海社会科学院隶属关系作掩护身份”来严重“影响其他国家的外交政策”。

[!\[\]()!\[\](https://spark.adobe.com/page/ZN6CkgcMBp1z1/images/0537140d-887e-444b-826a-805404000e95.jpg?asset_id=ef25a262-6038-4021-b41a-925545cc5a51&amp;img_etag=%22ef584391a10d46ae2cacfcdd3e68fc2b%22&amp;size=2560)](https://spark.adobe.com/page/ZN6CkgcMBp1z1/images/0537140d-887e-444b-826a-805404000e95.jpg?asset_id=ef25a262-6038-4021-b41a-925545cc5a51&amp;img_etag=%22ef584391a10d46ae2cacfcdd3e68fc2b%22&amp;size=1024)

中国工程物理研究院因与中共国人民解放军的紧密联系而被美国商务部列入黑名单——参与研究的中共国人民解放军国防大学也面临着这种威胁。

中央党校——一所“培养党的中高层领导干部和马克思主义理论干部的高等院校”——以及外交部，也完全由中共国政府机构组成。

该中心还接待了数家（中共国）公立大学的演讲者。

**与中共国政府的关系。**

[!\[\]()!\[\](https://spark.adobe.com/page/ZN6CkgcMBp1z1/images/0823dcf2-9532-474e-be6c-70037281f219.png?asset_id=916033df-7848-4dc2-9cab-d326e9da2eaa&amp;img_etag=%22966a1ee816b06545308ef8be739209c7%22&amp;size=2560)](https://spark.adobe.com/page/ZN6CkgcMBp1z1/images/0823dcf2-9532-474e-be6c-70037281f219.png?asset_id=916033df-7848-4dc2-9cab-d326e9da2eaa&amp;img_etag=%22966a1ee816b06545308ef8be739209c7%22&amp;size=1024)

斯坦福的国际安全与合作中心还参加了在北京由中共国外交部“主办和共同赞助”的会议。

斯坦福大学的一位研究员指出了研究小组之间的“多年交流”，并补充说，他们将在诸如太空合作等问题上“制定联合蓝图”：

*“这些交流是坦率和建设性的，因为它们建立在国际安全与合作中心和中共国国际问题研究所（CIIS）之间经过多年交流之后建立的理解和信任的基础上。在下一阶段，美国和中共国专家组成的小型小组将制定联合蓝图，以增进对核稳定和空间合作至关重要的问题的理解。”*

这位研究员补充说：“尽管该会议是在美中关系的未来充满不确定性的时刻举行的，这次会议还是包括了关于战略稳定、太空合作的障碍以及其他敏感主题的建设性交流。”

会议上中共国代表 “来自各种类型机构——中国国际问题研究所、中国工程物理研究院、解放军火箭部队、火箭军大学、中国国防科技信息中心、解放军海军军事学院、解放军南方司令部、中国社会科学院、中国人民大学、国防大学、清华大学等。”
