## 【国际金融快讯】富时罗素宣布将中国联通、中芯国际等三家中共军企移出全球指数
` wenwu` [轉載自GNews](https://gnews.org/zh-hans/717047/)

喜马拉雅-国内新闻组/素材/封面：Meilen美伦  校对：文迹~见证神迹

据英国富时罗素公司官网2021年1月4日发布的一份声明所示，自1月7日开盘起，将从富时全球股票指数系列（FTSE Global Equity Index Series）和富时中国A股指数（FTSE Global China A Inclusion Indexes） 中，删除中国联合网络通信集团有限公司（中国联通）、南京熊猫电子股份有限公司和中芯国际集成电路制造有限公司（中芯国际）三家公司的股票。
![]()![](https://gnews.org/wp-content/uploads/2021/01/image-36.png)
声明中，富时罗素表示，公司已参照特朗普政府禁止投资被指“有军方背景中资企业”的行政令，经审核后把一些中资企业移出全球指数。

最后富时罗素表示，将继续关注国防部和外国资产控制办公室(OFAC)的任何进一步的更新，国防部和OFAC公布的任何受制裁的额外名单，将在2021年1月11日之后进行评估。
