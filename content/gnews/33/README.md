国民党从2021年起，不间断对民进党即其他政党人员提出罢免案，从最初的桃园王浩宇罢免成功之后，又提起对高雄黄捷罢免案。而黄捷罢免案最后是未达到罢免门槛，故2/7号罢免案结果为不通过。

接着在2/8号，随即向中央选举委员会(简称中选会)提出针对台中选区的台湾基进党立委陈柏惟的罢免案。并且罢免提案人杨文元表示，罢免案提出要陈柏惟正面各界质疑，罢免提出「无关蓝绿问题」等语。

杨文元，为无党籍人士。但在国民党台中市议员罗廷玮及大里区立仁里长郑伯其陪同下召开记者会，宣布成立「删Q总部」。虽然罢免发起人为无政党，但从陪同成立罢免总部的国民党籍议员来看，此罢免与国民党的确有相当程度关联。但上文里长郑伯其同样为无党籍人士。

罢免理由有4，如下：

1. 陈柏惟的政见承诺一再跳票，包括台中火力发电停机一天都做不到
2. 陈柏惟身为台中立委，却不关心自己选区，还从桃园、新竹到台南广开服务处，跟**桃园前议员王浩宇**一样是「全国不分区」
3. 陈柏惟在立法院及社群媒体的发言和行为让人无言
4. 人民就是不想吃莱猪，陈柏惟表决上赞成莱猪进口，把人民的食安放在哪里？


根据新闻报导指出，针对罢免陈柏惟的总部在02/01成立，短短五天就达到提出罢免案的门槛。

从上可知，台湾内部仍然有许多国民党的死忠支持者，自以为怀抱爱国热情，殊不知是抱着蓝皮红肉的共产党恶魔之腿。主张着所有支持国民党的行为，是基于促进台湾政治改革、社会进步而努力着，殊不知实际上是将帮助中共将台湾一步步陷入泥淖中。将自己还有家人的安全暴露于风险中。
