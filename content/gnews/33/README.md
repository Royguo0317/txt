**西班牙2020年12月30日电/西喜社—**川普总统周二指责参议院多数党领袖米奇·麦康奈尔阻止众议院批准的一项两党法案，该法案将为数百万美国人提供2000美元的救济支票。

“除非共和党人有死志，而且这也是正确的事情，他们必须尽快批准2000美元的付款。600美元是不够的！”总统在推特上警告说。

在众议院的立法中，周一以275-134票通过，44名共和党人投了赞成票，收入达到7.5万美元的美国人，将获得2000美元的直接付款—比国会两院批准的中共病毒救济方案中的600美元支票有所增加，并由总统周日晚上签署成为法律。

至少有四名共和党参议员表示支持众议院的法案。

麦康奈尔周二在没有解释的情况下阻止了该法案。但在周二早些时候，参议院领袖表示，他可能会将2000美元的支票与废除科技责任屏蔽和选举相关调查打包。川普在周末表示，参议院将 “启动针对这三个问题的投票程序”。

“总统强调了另外三个具有国家意义的问题，他希望看到国会一起解决，”麦康奈尔说。”这就是总统连在一起的三个重要议题。本周，参议院将开始一个进程，将这三个优先事项作为重点，”他补充道。

周一，参议员马尔科·卢比奥(Marco Rubio)表示，他支持2000美元的经济刺激支票。

“我同意总统的观点，即数百万工人阶级家庭迫切需要额外的救济，这就是为什么我支持向因流行病而挣扎的美国人直接支付2000美元，”卢比奥在周一的一份声明中说。

“几个月来，共和党人试图通过对工人、家庭和小企业的额外救济—但每次都遭到民主党人的拒绝，”卢比奥讲述道。”记住，几个月前，议长南希·佩洛西和民主党人拒绝了政府之前提出的每个成年人1200美元和每个儿童1000美元的建议。值得庆幸的是，她终于不再以工薪家庭作为要挟了。”

“我和许多同事一样，对额外支出的长期影响感到担忧，但我们不能忽视这样一个事实，即全美数以百万计的工薪阶层家庭仍亟需救济。”卢比奥补充说，他的信息传递最近出现了民粹主义的转变。”国会应该迅速通过立法，将直接支付给美国人的金额提高到2000美元。”

乔治亚州的另外两名参议院共和党人大卫·珀杜（David Perdue）和凯利·洛夫勒（Kelly Loeffler）表示支持2000美元的支票。

“川普总统是对的—我支持这次推动2000美元直接救济美国人民，”珀杜在推特上写道。

而参议员林赛·格雷厄姆(Lindsey Graham)上周末也表示支持。他在推特上写道：”国会将对额外的刺激措施支票进行投票”。”干得好，总统先生！”
