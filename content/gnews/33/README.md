## 中共国航空公司暂停了向印度运送医疗用品的货运航班

编辑:康州盘古农场 – 心照

据《wionews》作者：PTI，2021年4月27日报道
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/04/29194026/opakfl-3.png)中共国国家主席习近平 照片: ( 路透社 )
中共国国营的四川航空公司已将其所有飞往印度的货运航班暂停15天。这对私营商人从中共国采购急需的氧气浓缩器和其他医疗用品的努力造成了重大干扰，尽管北京向该国提供了所谓“支持和援助”，以应对印度最近激增的COVID-19病例。

在双方私人贸易商为从中共国采购制氧机而忙碌的情况下，四川川航物流有限公司（该公司是中共国国营公司四川航空的一部分）在周一给销售代理的一封信中说，该公司已经暂停了六条航线的货运航班，包括西安至德里。

在PTI看到的信中，该公司表示，面对（印度）疫情的突然变化，为了减少进口病例，决定在未来15天内暂停航班。

印度航线一直是川航的核心战略航线。此次停航也给我（川航物流）公司带来了巨大的损失。信中说，我们对这种不变的情况感到非常抱歉，并寻求“其销售代理人的理解”。信中还说，公司将在15天后对这一情况进行重新评估。

暂停货运航班让正在迫切地试图从中共国采购氧气浓缩器的代理商和货运代理大吃一惊。

还有人抱怨说，中共国制造商把价格抬高了35%至40%。总部设在上海的货运代理公司Sino Global Logistics的辛哈（Siddharth Sinha）说，运费已经增加到20%以上。

他在这里告诉PTI，在目前严峻的形势下，四川航空公司取消航班的决定，已经严重干扰了两国的私人商人为确保快速供应氧气浓缩器赶往印度的尝试。

他说，现在要赶制这些物资变得非常具有挑战性，因为它们必须通过不同的航空公司转道新加坡和其他国家。这就延误了急需物资的供应。

他说，因为印度的冠状病毒情况而暂停航班是令人惊讶的。因为飞机在印度没有更换机组人员，而是由同一个机组人员驾驶飞机返回。

印度正在与大流行的第二波爆发作斗争。在过去的几天里，每天有超过30万新冠状病毒病例被报告，有好几个邦的医院因为医疗氧气和床位短缺而陷入困境。

除了航运服务外，中共国的货运航班在整个大流行期间一直在运作，向印度紧急供应利润丰厚的移动电话设备以及大量其他中共国出口产品。

辛哈还说，中共国供应商也采取了不人道的做法，严重抬高价格，在大流行病的紧急情况下赚黑心钱。

暂停航班也让印度官员感到惊讶，因为它是在中共官方声称向印度提供支持和援助的背景下发生的。

中共国外交部发言人赵立坚在4月23日的媒体发布会上说，中共国正在与印度沟通，以提供必要的支持和援助。他说：“中共国对印度最近不断恶化的局势向印度表示真诚的同情。”

中共国政府和人民坚决支持印度政府和人民抗击冠状病毒。他说，中共国愿意根据印度的需要提供支持和帮助，并就此与印方进行沟通。

他说，我们相信，印度人民早日战胜病毒。

印度的COVID-19病例总数已攀升至1,73,13,163，而活动病例已超过2800万大关。根据联邦卫生部周一更新的数据，死亡人数增加到1,95123人，新增死亡人数为2812人。
