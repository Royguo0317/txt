中共疫苗到底有没有效？看看深圳的感染病例就知道了。中共为逼民众打疫苗，近期操弄出一波本土疫情的“小高峰”：辽宁、安徽、广州、深圳纷纷曝出所谓不知真假的本土病例，从而逼出一波民众抢打疫苗的小高峰。

假如深圳本次疫情是真的，那么就可以发现事实上中共的疫苗起不到防护作用。此前，我们已经分析报道过5名注射过疫苗的中国专家到越南以后全部感染印度变异毒株事件，这足以说明中共疫苗不具备对病毒的防护力（[https://gnews.org/zh-hans/1261644/）。](https://gnews.org/zh-hans/1261644/%EF%BC%89%E3%80%82)

５月23日，中共深圳市政府举行疫情防控新闻发布会，通报盐田区排查出４名无症状感染者。根据流行病学调查及病毒基因检测结果，中共初步推断与国际货轮作业有关。但是，中共国已经为环境口岸工作人员都注射过疫苗，作为第一批重点人群，口岸工作人员早在2020年底～2021年初全部接种疫苗完毕。若本次4名盐田港工作人员的感染是真的，那么就又一次说明中共疫苗不具备防护能力，一次性出现4名打过疫苗的人员都被感染，已经足以说明问题。

另一方面，中共这波疫情的操弄也有可能是假的，目的在于逼迫民众抢打疫苗。反正所有核酸检测数据都是中共掌握的，中共非常容易造假（一切都听党的嘛！）。随便搞几个假的阳性结果，就可以制造出来一波“可防可控”的小疫情，让中共国内民众紧张一阵。

如果疫情是真，那么疫苗无用，墙内民众就不必去打疫苗。

如果疫情是假，那么墙内民众就不需要过度紧张，依然做好应有的防疫措施就可以了。中共疫苗的长期副作用还没有被揭露出来，一旦具有严重长期副作用，那么后悔也来不及。
