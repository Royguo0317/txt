- 本日，台灣新增一例本土案例、兩例境外移入。


今天跟大家宣布有三例新增案例，两例境外移入，一例本土案例。境外移入案例是从英国跟美国入境编为883跟884，883为英国籍10多岁男性12月28日曾于当日确诊并自我隔离十天，今年1月20日来台持有搭机前三日检验阴性报告，入境后到集中检疫所检疫，现在没有症状，今年1月21日接受英国专案采检，于今日确诊。同班机接触者调查匡列中，不过在本土因为他都在集中检疫所内所以并没有其他匡列，在搭飞的我们要匡列。案884为美国籍20多岁女性学生，去年12月31日搭机来台进行语言文化交流。1月1日入境台湾，持有12月24日检验阴性报告，入境后至防疫旅馆检疫，现在没有症状。居家检疫期满后，在1月20日自费采检，今日确诊。掌握接触个案有9人，其中同行友人2人列为居家隔离，其余7人皆有适当防护列为自主健康管理。

所以近来我们检验只有新检验出一例就是案885，其他已采检的目前跟昨天维持一样，在医院采检者498名阴性、6名阳性，社区接触者127名阴性、6名阳性。检查结果这样，这两天比较辛苦的疫调人员相关的疫调部分，是我们把居家隔离的范围把它变大所以目前隔离者总共有967人主要增加的是医院增加了一部份，昨天好像是319人，那医院部分也在就是在没有必要来上班的我们尽量都让他居家隔离起来，另外增加主要是在回溯，回溯大概昨天我们有红区的191人已经都列为居家隔离的对象，这工作已经完成。然后他的陪病者一百多人也把他列到相关居家隔离的对象。那陪病者的数目还会再增加。因为在出院者本身相关的资讯是比较完整陪病者可能还是要从医院的相关的资料去整理另外还要去问相关的当事人。所以这个速度稍微会慢一点。不过基本上最主要的191人已经把他都隔离起来。
