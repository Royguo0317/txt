《经济学人》资讯社（Economist Intelligence Unit）发表了2020年民主指数（Democracy Index 2020）报告。针对全球167个国家和地区政权的民主程度的指数，该指数是根据选举过程及多元化、政府功能、政治参与、民主政治文化和公民权利等项目进行调查和评比，政权按得分分为“完全民主”（8至10分）、“部分民主”（6至7.9分）、“混合政权”（4至5.9分）和“专制政权”（低于4分）四类。台湾是进步最多的国家，从2019年的第31名跃升为第11名，在东亚的排名最高，达到「完全民主」水准。

**从选举罢免的公民权反思民主**

选举是选贤与能，将清廉有理想、有抱负、肯负责的人，选出来为民众服务。罢免是将不符合选民期待，违背自己政见，贪赃枉法的当选官员淘汰的机制。

### 但看看在野国民党，自从高雄市市长韩国瑜被罢免成功后，不晓得痛定思痛，了解人民的心声、需求，反而为罢免结果扬言报复，短短几个月，从北到南，报复性罢免直辖市议员[王浩宇](https://zh.wikipedia.org/wiki/%E7%8E%8B%E6%B5%A9%E5%AE%87)（罢免成功）、[陈致中](https://zh.wikipedia.org/wiki/%E9%99%B3%E8%87%B4%E4%B8%AD)（罢免立案未成功）、[黄捷](https://zh.wikipedia.org/wiki/%E9%BB%83%E6%8D%B7_%28%E6%94%BF%E6%B2%BB%E4%BA%BA%E7%89%A9%29)（罢免未成功）、[高闵琳](https://zh.wikipedia.org/wiki/%E9%AB%98%E9%96%94%E7%90%B3)（罢免立案中），四处扇风点火，浪费公帑，政府运作停顿。

民进党副秘书长林飞帆表示，罢免不该作为党派间对抗、报复式的操作工作，台湾主流民意、大多数公民也不希望罢免作为政治动员工具，盼此作为应该停止，让优秀的国会议员、公职继续在体制内服务。无论党派，民进党都会全力协助，让好的人留在国会与职务上。

民进党吴怡农表示，政党重要任务是为市民、社会留住最好的人，罢免制度原意为品德、能力让社会大众失望者的汰除机制，照理说，其制度原意并非排除对手阵营优秀政治工作者。

还说，对于民主无帮助、破坏此本质的罢免活动应该适可而止，社会大众会支持把好的、认真做事的人，留在体制内继续为社会服务。

### 选民经历了一次又一次的选举罢免，对自己的权利认知更深，更有智慧。看看香港的民主如今荡然无存，就因为红色的中共政权介入，所以这次的评比虽说进入了民主的资优班，但是台湾面对中共长久以来不遗余力的全面渗透，对中共立场应该更明确，立法的界定应该更清楚，对于张安乐们、红粉们和红色媒体，政府也应该更大刀阔斧，为国安把关，不让他们有站在民主的土地宣扬共产邪恶、似是而非的言论。

**摇着民主的旗帜反民主**

台湾位于反共第一岛链的枢纽，中共一直以来虎视眈眈，对台湾的蓝金黄的渗透，可说无所不用其极，无所不在，从政府到民间，在民间的宫庙更可看到斧凿斑斑，为了眼前的利益拼经济，台湾会不会成为香港第二？中美贸易，台湾获利，是否又能疗愈台湾人的斯德哥尔摩症候群，不畏中共，的文攻武吓，经济利诱呢？

国民党的政客为了得到黄复兴党部等人的支持，赢得党内初选及提名，做出了令人匪夷所思的事。

[2019年韩国瑜当选高雄市市长，对性别及国籍歧视等发言，损害城市及台湾形象，对市政荒腔走板，未经政府及议会授权进入在香港的中联办，4个月后抛下选民的期待，红媒介入加持，韩粉偏激言行时有所闻，激发出高雄人的愤怒，提出罢免案](https://zh.wikipedia.org/wiki/2020%E5%B9%B4%E9%AB%98%E9%9B%84%E5%B8%82%E5%B8%82%E9%95%B7%E9%9F%93%E5%9C%8B%E7%91%9C%E7%BD%B7%E5%85%8D%E6%A1%88)，经过严谨的联署、立案，最终中选会决定于2020年6月6日举办高雄市市长韩国瑜罢免案投票，同意罢免市长韩国瑜的票数达93万9090票，韩国瑜成为中华民国及台湾地方史上首位被罢免的直辖市首长。

# 随后扬言报复，短短几个月，从北到南遍地开花，报复性罢免直辖市议员[王浩宇](https://zh.wikipedia.org/wiki/%E7%8E%8B%E6%B5%A9%E5%AE%87)（罢免成功）、[陈致中](https://zh.wikipedia.org/wiki/%E9%99%B3%E8%87%B4%E4%B8%AD)（罢免立案未成功）、[黄捷](https://zh.wikipedia.org/wiki/%E9%BB%83%E6%8D%B7_%28%E6%94%BF%E6%B2%BB%E4%BA%BA%E7%89%A9%29)（罢免未成功）、[高闵琳](https://zh.wikipedia.org/wiki/%E9%AB%98%E9%96%94%E7%90%B3)（罢免立案中）和酝酿罢免吴思瑶，四处扇风点火，浪费公帑，政府运作也受影响。

现为中国广播公司董事长兼总经理赵少康，看到如此景况，经历三中案疑云，中广又被认定国民党「随附组织」追缴77亿， 于是2021年2月1日赵少康公开承认正在申请恢复国民党党籍，并在随后接受媒体访问时表示不排除参选国民党主席一职并争取代表中国国民党参加2024年中华民国总统选举。这一声明，搅乱了各政党的布局，也考验着选民的智慧。
