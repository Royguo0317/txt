致各位同路人：

身陷囹圄，我每天经常祈祷，与自己对话，多鼓励自己，要坚持撑着，但有时真的提不上气，心知自己已尽能力，人是的确要有适当的渠道抒发感受，所以每次跟外间书函联络，便是我最好的渠道。

到了写信的时刻，心中仍有重重无影感压着，我多番思量亦未能找到适当原因，以往偶发的迹象，亦肯定抑郁症状有些微影响着自己，但我不感奇怪，毕竟每次也能自我辅导，调整，深信一时的低谷，并不代表永远。而抑郁于这世界上，可悲地普遍化。

于这段期间，看了两本书籍，〈被讨厌的勇气〉以及〈写给心中尚未崩坏的你〉，这两本分别叙述从基督徒角度，以及用苦学加心理的新派角度，去剖析世界及人性，人其生来，不需要过于苛求别人认同，别人是必然会因为意见/立场/角度不同而否定自己，更甚当我们为了坚持自己所守护及捍卫的价值，否定我们的可能更会相当讨厌我们，因我们在处的，并不是他们「所认同的」价值及思想。

这个问题，我便从小已经疑惑，我亦感觉自己从小时就非常害怕别人会讨厌自己，亦于多种因素影响下，生出自卑，本来我也来学懂爱人，自卑感变成已注定及抉择自己变成认定自己成为什么人，但书中以阿德勒这位伟大及具前瞻性的新派哲理学者，与弗洛伊德及荣格同称心理学三大巨头之一的这人，提出人生是以「目的论」为主要思想，而大部份人类均会以「决定论」作为单一思想。 （固有想法）

换句话说，如你认为「人生是不能改变」，那你就注定一生来只会停在原地，因为我们需要了解的不是过去的「原因」，而是现在的「目的」，因为人必定为了某种目的而生存，何之所以无法改变，是因为自己下定决心「不要改变」。

所以「宁愿甘于现状」？如现在的自己并未适合改变，先与自己好好相处，了解自己后，透过人生给你的经验，将它赋予新的意义后，人才能前进。而不是人生定义赋予给你的经验来作动力前进，错误地定义人生，只会让自己倒退。

这封信，我也避不了分开两次拼写，而这刻是星期六的一个早上，心情已经渐渐变好，除了天气有关系以外，自己亦放开了一些无谓的想法及挣扎，愿主继续让自己坚强起来，然而什么难题也不再是问题。
