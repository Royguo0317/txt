亲爱的战友们，请你问自己几个问题：比如，你累死累活赚的是“钱”吗？你省吃俭用省的是“钱”吗？当今世界各国流通使用的纸钞和硬币是“钱”吗？银行账户里那一串数字是“钱”吗？如果你觉得似是而非，请接着往下读。

**一：钱是什么？**

先来说说交换，交换就是用你有的换你没有的，交换最重要的两个因素就是平等和有价值。我用一个苹果换你的一台苹果笔记本，你肯定不干；但是如果我用一卡车苹果换你的一台苹果笔记本，你可能就接受了。因为它们所包含的价值大概差不多，这就是平等。烂苹果或者坏掉的笔记本也完成不了交换，因为其中一方的价值已经没有了。

举个具体例子，你汗珠子掉地下摔八瓣地为老板工作，月工资一万。你的时间和劳动创造出价值，老板应该付出相等价值来交换你的劳动。如果老板分别用一台苹果笔记本电脑， 一万块纸币，一枚一盎司熊猫金币这些大约都值一万块的商品来支付你的工资，并且可以任选，你选哪个？
![]()![](https://gnews.org/wp-content/uploads/2021/03/Screen-Shot-2021-03-17-at-8.14.33-AM.png)图片来源：网络
分析：

1. 笔记本再好再先进，也逃脱不了被更高的科技所取代的命运，肯定价值缩水。

（比较电脑286…486…95…98…xp…10…性能的不断提升和价格的持续下降）

2. 一万块纸币当下可以买好多商品，但随着印钞厂的不停印刷，肯定价值缩水。

（上世纪80年代万元户都是稀缺，现在2021年百万“富翁”都是贫农）

3.金币恰恰和前两样不同，在时间的长河中，黄金的本质不会发生变化。几千年前，现在，几千年后，它都是客观世界存在的一种元素，性质极为稳定；而且地球表面的易开采的金矿愈来愈少，也就是说人们要付出更多劳动价值才能获得黄金；更重要的是印钞机不能印制黄金。它的这些主要属性使得黄金可以很长久的保有价值。（1971年以前20或35美元/盎司，价格固定。现在2021年大约1750美元/盎司，价格浮动）

小结“钱”：“钱”是可以用来交换并且在很长（长到几千年）的时间里可以保有价值的商品。最好的代表是黄金，当然也还有白银及其他贵金属。

**二：币是什么？**

简要谈了“钱”之后我们再来说说“币”。现如今，人们早已经不再用两条鱼去换一只鸡了，取而代之的是各国政府发行的纸币和硬币，也称法币，比如美元，英镑，日元，欧元，还有人民币等等。普通百姓用自己可以提供的劳动创造价值换取法币，再用这个法币去换取日常生活中需要的其他商品或者服务。法币是政府利用法律规定大家都认可的一个中介，一个连接买卖/供需双方的桥梁。这些印钞机，铸币厂流水线下生产出的花花绿绿的纸钞或硬币本身几乎没有价值。否则，100美元的那张纸的面积应该是1美元的100倍才对。
![]()![](https://gnews.org/wp-content/uploads/2021/03/Screen-Shot-2021-03-17-at-8.20.18-AM.png)图片来源：网络
小结“币”：币是由政府发行基于信用的一种交换介质，本身没有价值。

**三：“钱”和“币”的异同**

钱和币有很多相同的地方，但是最重要的不同之处在于：钱是罐子，在长时间内保有价值，积累财富；而币是漏斗，本身没有任何价值，会流失财富。

下表总结了钱和币的异同：

几乎没有老板用实物或贵金属来支付员工的工资，而普遍用纸币或纸币的数字形式–银行账户中的那串数字来代替。这又隐藏了怎样的秘密呢？

**四：政府如何逐渐远离“钱”，利用“币”来掠夺普通百姓的财富**

这是一个在前苏联真实框架下的虚构故事，悲惨，滑稽，令人深思！

假设一个小孩1961年1月1日出生，父母为他存储了1000卢布，2021年这个小孩60岁退休。
![]()![](https://gnews.org/wp-content/uploads/2021/03/Screen-Shot-2021-03-17-at-8.23.14-AM.png)
情景一：按照当时黄金的价格，这1000卢布可以等值大约3.17盎司黄金，相当于2021年大约5500美元。

情景二：在1961—2021 这六十年中，伴随着前苏联政局的巨变，苏共解体，法币卢布发生了4次重大重组。从最初的1000卢布缩水成2021年的0.1卢布，大约只值0.0013美元。

5500美元对比0.0013美元是多么大的差异呀！

这个小故事很好的说明了“钱”—黄金可以存储价值，是罐子；而政府发行的“币”是漏斗，会流失财富。
