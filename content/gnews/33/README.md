2030年6月4日，天气闷热。

很久没有写日记了，今天，是父亲葬礼的日子。

父亲出生于1949年，名字叫“建国”，和大多数出生于上世纪50年代左右的人一样，建国同志“长在红旗下，育在党怀里”，属于被中共严重洗脑的一代人。中共建政之前，我的祖父母都是当地的农民，两人怀着救国之心追求“进步”加入了共产党，是村子里最早的一批党员。父亲成年之后到城里参加工作，由于“根正苗红”得到组织的认可，也很早就加入了共产党。

父亲喜欢舞文弄墨，字又写得隽秀，所以单位里的黑板报成了他施展才华的天地，也因为这个原因，一位喜欢文艺的女同事后来成了我的母亲，再后来就陆续有了我和弟弟妹妹三人。小时候总觉得父亲很严肃，头发剪得很短，有时候他会留一点胡须，给人一种不怒自威的感觉。父亲对我们兄弟妹三人的要求非常严格，尤其是对我的学习，打小就被逼着做数学写日记，所以到现在还有时不时写点东西的习惯。也许正是因为父亲的严格要求，我的学习成绩一直不错，高考后进入了北京一所名牌大学，那是1988年，我刚满17周岁。

上大学是我人生第一次独自离家远行，刚进大学时觉得一切都很新鲜，和父亲也经常有书信往来，不时汇报自己的学习和生活情况。80年代末大学校园思想活跃，接触到很多自由的思潮，于是对事物就有了越来越多自己的见解，渐渐地和父亲的通信就变成了一种敷衍。到了第二年开学返校后，同学们每天都聚集在一起针砭时政，对社会上的各种现状积极发表自己的看法，4月份校园里陆续开始出现纪念胡耀邦的宣传海报，到了4月24号学校开始罢课，同学们纷纷开始集会演讲走上街头，我也扎着头巾成为了广场游行静坐的一员，那是一段青春飞扬的日子。

1989年的天安门广场聚集了世界的眼光，时局每天都有不同的变化，同学们对中国的未来都充满了责任感，但是我却当了逃兵。5月底的某一天，父亲意外地出现在我的面前，他板着脸要求我立刻跟他走，那时候父亲在我的心里依然是威严的家长，我没有多争执就跟着他走了，甚至都没有回校收拾一下换洗的衣服。到家后母亲告诉我，是父亲在电视里看到了我，当天晚上就买了火车票到北京，这也成了后来若干年我一直不能原谅父亲的心结，而父亲则认为是他救了我的命，让我没有成为六四的牺牲品，我和父亲的争吵一直持续了很多年，其实现在想来我只是不能原谅自己的懦弱。

大学毕业后，由于参加过学生运动组织，档案里有“污点”，留京成了奢望，在当时的环境下也没有什么好的工作单位愿意接收我，最后父亲找关系让我回到老家原来我毕业的中学教物理，以前的老师们一下子成了我的同事。学校并没有给单身教师的宿舍，于是我平时就住在家里，和父亲的争吵和摩擦与日俱增，最后到了水火不相容的地步，那是一段对双方来说都痛苦的经历。最后为了逃避这苦闷的生活，我没有和父母商量，辞去学校的工作离开家乡去了南方，一路走来，工作、成家、生子、创业……曾经很多次，我以为随着时间的推移和时代的进步，中国的民主进程也会与时俱进，然而却一次次的失望，六四屠杀仿佛成了一场虚渺的梦，在中国从来没有发生过。2012年，对中共彻底绝望的我举家移民，来到了澳大利亚。

我在国内的很多年，和父亲的交流几乎是零，除了逢年过节回老家，平时都是通过母亲或者妹妹传话。到了异国他乡，遥远的距离反而拉近了我们的关系，那时父亲早已过了耳顺之年，而我也已是不惑，视频电话中双方总是很客气，岁月已经将彼此的棱角慢慢磨去，最后只剩下亲情。2015年父亲和母亲一起来澳洲探望我们，那是他们第一次出国，出生于农村的父亲对澳洲的自然环境非常感慨，人与自然间的那种高度和谐让他对资本主义的抵触有所缓和。有一次我去华人区拿了一些大纪元之类的报纸，对于父亲来说法轮功就是邪教，其它“反动报纸”也都是“帝国主义亡我心不死”，但也许是为了打发时间，他从开始的不屑一顾骂骂咧咧，到后来逐行逐字阅读，并且开始和我分享关于国家民族的一些看法，我给他在网上找了一些关于中共高层腐败的资料，以及天安门六四的真相，这些对父亲都有一些触动。那次澳洲之行后，父亲的思想显然发生了某些转变，其实他对共产党的本质也心知肚明，所以在一旦明白了政党和国家的区别后，我们两人之间的很多分歧就烟消云散了，但是作为一位几十年的老党员，他依然无法摆脱对那个邪恶组织的感情。

在2017年以前，中共国力与日俱增，共产党极权俨然成了西方热捧的一种体制，以至于我曾经认为此生都看不到中共倒台的那一天，直到一次偶然的机会看到《美国之音》断播的新闻，于是开始关注爆料革命，随着时间的流逝，越来越相信郭文贵先生可以灭共，从此一发不可收拾……夜深人静的时候，二十多年前的往事像潮水一般涌来，历历在目令我心潮澎湃，当年那个弱冠青年心中的激情仿佛又回到了我的身体。终于，我也像1949年之前我的祖辈那样怀着救国之心加入了爆料革命，冥冥中，这仿佛是我们祖孙三代的命运轮回，作为共产党员的后代，我觉得自己有一种使命，将中共终结在我们这一代，不让它再继续祸害我们的下一代。

2018年我将父母再次接到澳洲，茶余饭后时不时和父亲说一些文贵先生的爆料内容，一开始他半信半疑，当时正好发生了海航王健先生离奇的死亡案，慢慢的他和我一样也开始“追剧”，关注着文贵先生的一举一动。当时母亲很是担忧，而父亲却出人意料地没有表达反对的意见，只是他一直认为指望文贵先生灭共是不可能的，因为爆料革命既没枪也没炮，而共产党却拥有强大的力量和资源，并告诫我一定要注意安全，我明白这不是一位共产党员的忠告，而是一位父亲的忠告。

2019年中共在重重压力下释放生物病毒，中共病毒造成的伤害一直持续了四年之久，在这场人类最大规模的生化超限战中，全球约有10亿人被感染，近千万人丧生，并造成了全球严重的经济危机，2022年各国股市崩盘，全球进入大萧条时代。在全世界一片愤怒的追责声中，中共内外交困，2022年4月底以美国为首的西方国家对中共进行了最后的通牒，中共内部土崩瓦解，5月10日李克强通告全国，中国共产党即日起正式解散，中共终于走到了历史的尽头，而这一天正好是文贵先生的生日，父亲第一时间给我发来信息：“事实证明，你是对的”。

没有了共产党的中国，天空的雾霾渐渐散去，人们终于见到了久违的蓝天。但是中国的经济在受到西方资本抽离、金融崩盘、企业大量倒闭的重创之后，很多人失去了工作，百姓们流离失所，社会秩序动荡。2023年，中国人终于有了自己真正的民选政府，但是中共七十余年的黑暗统治给中国人造成的伤害还在持续，人心的雾霾并没有散去，政治的博弈还在各地上演，也许还需要两代人才能完全肃清人们心中的余毒。不过，没有了邪灵恶党的中国，一切都在向好的方面发展，近年来，在喜联储强大的支撑下，中国的经济慢慢恢复了生机，越来越多的投资回流，人们对未来又充满了信心。

中共垮台之后，父亲和九千万普通党员一样没有受到任何冲击，但不幸的是去年确诊了肺癌晚期。今年的清明节，父亲坚持要坐着轮椅去老家扫墓，在祖父的坟前，父亲一言不发，也许他只是想跟地下的那位老共产党员约一约见面的时间，很快，他也走到了生命的终点。和千千万万个共产党员一样，父亲只是一个普通人，有着每一个中国人都有的家国情怀，只是他们是不幸的一代人，因为他们没有选择，而我是幸运的，我选择了爆料革命，选择了正义。

外面突然传来一阵阵礼炮的声音，天空中出现一道道美丽的烟火，因为今天是新中国联邦成立十周年，也是一个曾经的共产党员的葬礼。
