# **香港：47名主要活动分子被控颠覆罪，如果罪名成立，将面临终身监禁**

**亲民主组织被控于去年7月组织非正式选举，以最大程度地镇压竞选者**

[!\[\]()!\[\](https://spark.adobe.com/page/d3oKAzeU8F20b/images/62e0c5aa-09f8-4d87-81d9-507444ed036c.jpg?asset_id=e61c447c-cde7-4890-a39b-41d613fd651b&amp;img_etag=%220e277c48b2732a7e603e41b3687eef1d%22&amp;size=2560)](https://spark.adobe.com/page/d3oKAzeU8F20b/images/62e0c5aa-09f8-4d87-81d9-507444ed036c.jpg?asset_id=e61c447c-cde7-4890-a39b-41d613fd651b&amp;img_etag=%220e277c48b2732a7e603e41b3687eef1d%22&amp;size=1024)香港法律学者戴耀廷在被控颠覆国家政权罪之前接受了媒体采访。摄影:杰罗姆Favre / EPA

在香港警察指控47名亲民主运动人士和政治家阴谋实施颠覆后，现在香港几乎所有发声的异见者都已入狱或流亡。如果定罪，所有人都将面临终身监禁。

[!\[\]()!\[\](https://spark.adobe.com/page/d3oKAzeU8F20b/images/998d5943-9bb7-4bb4-840d-f209bca17740.jpg?asset_id=65b43af4-8fb2-4c8d-83df-08ca929748a7&amp;img_etag=%22508056402b96729b4cfad91ac5e44803%22&amp;size=2560)](https://spark.adobe.com/page/d3oKAzeU8F20b/images/998d5943-9bb7-4bb4-840d-f209bca17740.jpg?asset_id=65b43af4-8fb2-4c8d-83df-08ca929748a7&amp;img_etag=%22508056402b96729b4cfad91ac5e44803%22&amp;size=1024)

上个月，在去年举行的初选中，有55人被捕，其中大部分人来自这群人。这次黎明时的突袭行动，是根据有争议且严厉的国家安全法进行的规模最大的一次行动。

周日，警察部队说，除8人外，其余所有人都被控以一项罪名，并将在星期一早晨法庭提审前被拘留。

欧盟驻香港办事处表示，这些指控明确指出“香港将不再容忍合法的政治多元化”，并呼吁立即释放被拘留者。人权观察和大赦国际组织都指责当局利用国家安全法使批评者保持沉默，并呼吁撤销所有指控。

被捕者包括年轻的竞选者、活动人士、地方议员以及建制派政治家，例如毛孟静（Claudia Mo）、朱凯迪（Eddie Chu Hoi-dick）和陈展程（Ray Chan）。活动家和前政治家黄之锋（Joshua Wong）已经入狱，因与抗议有关的指控服刑13个月。

自从周五人们被告知要比以前指示的时间提前一个月向警方报到以来，人们一直担心会发生大规模指控。许多人预计他们会被起诉并被拒绝保释，开始为此做了一些准备工作，包括与家人共度时光、安排照顾宠物以及为入狱购买舒适的衣服。

当地媒体报道，民主党议员涂谨申（James To）、邝俊宇（Roy Kwong）和美国律师约翰•克兰西（John Clancey）都是周日未被起诉的八人之一。克兰西告诉记者，他的保释期延长至5月4日，他同时表示，香港人越来越像生活在拘留中心，“人们的自由和权利越来越受到限制”。

[!\[\]()!\[\](https://spark.adobe.com/page/d3oKAzeU8F20b/images/d7665bbf-437f-41d3-a5ad-b3726f3cdb80.png?asset_id=06f17c37-1634-480a-9d73-4c58e11534f3&amp;img_etag=%225a49b6a331ba948dd6d53ee82a42192b%22&amp;size=2560)](https://spark.adobe.com/page/d3oKAzeU8F20b/images/d7665bbf-437f-41d3-a5ad-b3726f3cdb80.png?asset_id=06f17c37-1634-480a-9d73-4c58e11534f3&amp;img_etag=%225a49b6a331ba948dd6d53ee82a42192b%22&amp;size=1024)岑子杰 （Jimmy Sham）

2019年抗议活动的主要组织者岑子杰（Jimmy Sham）在进入警察局前在警察局外发言时表示，他们将保持坚强并继续战斗。他说：“民主绝不是天赐的礼物。必须靠许多人以坚强的意志来赢得它。我们可以告诉全世界，在最痛苦的制度下，香港人是城市之光。”

许多被指控者在社交媒体上向支持者留下了信息。

[!\[\]()!\[\](https://spark.adobe.com/page/d3oKAzeU8F20b/images/814d59bb-3e53-4e29-90ec-e5a85a00b1e3.jpg?asset_id=545fb63c-a773-4f14-badd-3e5b591d1e8b&amp;img_etag=%22bac904eb14dc2b5401e17efe161d61be%22&amp;size=2560)](https://spark.adobe.com/page/d3oKAzeU8F20b/images/814d59bb-3e53-4e29-90ec-e5a85a00b1e3.jpg?asset_id=545fb63c-a773-4f14-badd-3e5b591d1e8b&amp;img_etag=%22bac904eb14dc2b5401e17efe161d61be%22&amp;size=1024)

这是对现年64岁的毛孟静的首次指控，毛孟静是一名前新闻记者，率直的亲民主派议员，去年与同事辞职以示抗议。她在周日在脸书上写道：“我的身体可能有些虚弱，但我的精神坚定。”

“不用担心。我们都爱香港啊。”

[!\[\]()!\[\](https://spark.adobe.com/page/d3oKAzeU8F20b/images/072ebee8-a0ef-4917-b5fc-74df00107fac.jpg?asset_id=866d835a-4224-4f77-8c9f-306d7ad471ef&amp;img_etag=%225d3c59ac43ca17a6672fdae00ae69d4a%22&amp;size=2560)](https://spark.adobe.com/page/d3oKAzeU8F20b/images/072ebee8-a0ef-4917-b5fc-74df00107fac.jpg?asset_id=866d835a-4224-4f77-8c9f-306d7ad471ef&amp;img_etag=%225d3c59ac43ca17a6672fdae00ae69d4a%22&amp;size=1024)朱凯迪 （Eddie Chu Hoi-dick）


朱凯迪说，他很感谢香港人民给了他贡献的机会，并对为了他们的共同理想而遭受指控“深感荣幸”。

[!\[\]()!\[\](https://spark.adobe.com/page/d3oKAzeU8F20b/images/68a2ccb8-4276-457b-9dce-ca2ad374573b.jpg?asset_id=60246f07-4f90-40dc-98d1-f4cc37d4d0ad&amp;img_etag=%22ef07665224359bc80be2e6bdaa6ea8cc%22&amp;size=2560)](https://spark.adobe.com/page/d3oKAzeU8F20b/images/68a2ccb8-4276-457b-9dce-ca2ad374573b.jpg?asset_id=60246f07-4f90-40dc-98d1-f4cc37d4d0ad&amp;img_etag=%22ef07665224359bc80be2e6bdaa6ea8cc%22&amp;size=1024)郭家麒

前立法会议员郭家麒说：“监狱可以隔离我们，但不能阻止我们彼此联系和相互照顾；锁链可以锁住我们的身体，但不能锁住我们的思想和灵魂！”

“保持冷静并继续前进。这终将过去！记住：不坚持就没有希望，坚持不懈才有希望！”

这些指控源于去年由泛民主阵营举行的非正式初选，目的是寻找最有实力的候选人参加香港大选，并在立法会中赢得多数席位。2019年的大规模抗议运动和当局的残酷镇压为亲民主政治派提供了更大的支持，在2019年底的区议会选举中，他们赢得了绝大多数席位。超过60万香港人参加了投票。

但在抓捕时，香港保安局局长李家超告诉当地媒体，被捕者旨在通过他们赢得大选并阻挠立法，计划“瘫痪”香港政府。他提到了初选组织者，法学学者戴耀廷（Benny Tai）早些时候发表的社论，以此作为一个有预谋的“邪恶”计划“使香港陷入深渊”的证据。

[!\[\]()!\[\](https://spark.adobe.com/page/d3oKAzeU8F20b/images/12b49450-0517-46e8-b37b-0b7dcae7af7d.jpg?asset_id=af858000-bff2-4caf-88d2-abafa866d422&amp;img_etag=%2221edb0713744e73575f28ee72b31982c%22&amp;size=2560)](https://spark.adobe.com/page/d3oKAzeU8F20b/images/12b49450-0517-46e8-b37b-0b7dcae7af7d.jpg?asset_id=af858000-bff2-4caf-88d2-abafa866d422&amp;img_etag=%2221edb0713744e73575f28ee72b31982c%22&amp;size=1024)戴耀廷（Benny Tai）

在周日早些时候的社交媒体帖子中，戴耀廷写道：“我获得保释的机会不会太大。”

[![](https://spark.adobe.com/page/d3oKAzeU8F20b/images/78cb0284-2329-4a4a-946a-6b5c661a4305?asset_id=c6e112e4-ee08-465b-b54f-83812e283d47&amp;img_etag=%220b43633f17b83d136cbf6c4a61571f81%22&amp;size=2560)](https://spark.adobe.com/page/d3oKAzeU8F20b/images/78cb0284-2329-4a4a-946a-6b5c661a4305?asset_id=c6e112e4-ee08-465b-b54f-83812e283d47&amp;img_etag=%220b43633f17b83d136cbf6c4a61571f81%22&amp;size=1024)

选举最终推迟了一年，表面上是由于疫情。自那时以来，北京和香港政府对阻止反对派候选人获胜甚至参与竞选都设置了许多新的障碍。上周，他们宣布了一些规则，要求所有政客和候选人宣誓效忠中国共产党的统治，并誓言不进行反政府行动，否则将面临被取消资格。

官员们说，新法律将确保只有“爱国者”才能治理香港，用一句话总结就是，爱国主义意味着对共产党的忠诚。

在香港政府的支持下，中共国政府于去年6月实施了全面的国家安全法，这些法律是有关部门为消除香港的异议而做出的最新努力。根据这项法律，迄今为止至少有99人被逮捕。这项法律的广泛定义是，颠覆、分裂国家、与外国势力勾结和恐怖主义行为被视为非法。
