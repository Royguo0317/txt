12月21日，国务院联防联控机制召开国务院联防联控机制新闻发布会 [1]。

中国疾控中心免疫规划首席专家王华庆表示，根据前期研究，要阻断新冠肺炎的流行，人群中要有70%左右具有免疫力。

当疫苗的保护效力是80%，疫苗的接种率要达到75%，才能使人群免疫力达到70%。接种疫苗是每个人的权利，也是履行控制传染病流行个人应尽的义务[2]。

多家官方媒体都转发了此消息，内容几乎完全相同，看得出是新华社要求的通稿。

此外，张文宏21日晚接受央视新闻采访时说，如果中国国内疫苗接种不够普及，向外开放将可能受阻。张文宏还称，疫苗是“义务”和“权利”的综合体。接种疫苗“义务”与“权利”的比例将随新的防疫节点的出现而有所进展[3]。

这两个人强调接种疫苗的义务，应该是代表中共在提前放风，试探民意。中共的很多政策拿不准要不要推行时，都会提前放风，如果没有遭到强烈的反对，就会正式下命令执行。

中共一向把医疗当作赚钱敛财的工具来用，把没什么效果的莲花清瘟、板蓝根都吹成了抗疫神药。这一次恐怕又会借疫苗之名大肆敛财。但与敛财相比，更令人担心的是，中共的国产疫苗会不会给接种者带来长期的副作用和ADE风险。

1．中共国产疫苗非常糟糕，起不到防护作用

自由亚洲报道，包括乌干达、安哥拉和塞尔维亚等多地都有中国派遣的工人在接种国药集团的疫苗后，仍然大量感染新冠病毒。其中规模最大的发生在塞尔维亚潘切沃，天津电力建设公司的一个项目部里，该处的400多名员工，已确诊了约300人。

当中包括大批来自国内的员工，他们事先也已经接种了国药的新冠疫苗，但依然被感染。中国政府迅速封锁消息，并努力将这个严肃的科学问题变为政治话题[4]。

考虑到强制接种政策可能也允许你接种国外疫苗，让我们看下国外疫苗的情况。

2．美国疫苗也有明显副作用

比如知名度很高的辉瑞-BioNTech联合研发的COVID-19疫苗，也有明显的副作用。

辉瑞疫苗的接种者中有84％报告了注射部位反应。

注射部位反应是指注射药物的周围组织发生炎症或损伤。有两种类型的注射部位反应：一种是局部过敏反应，称为耀斑反应，另一种是更严重的反应，其特点是由于血管外渗造成的组织损伤。血管外渗是指少量药物从注射的血管中渗出。

这两种反应的症状通常包括红肿、触痛、发热和瘙痒，但外渗的后果更严重，可能包括疼痛、起泡和严重的皮肤损伤。由于外渗会造成严重的组织损伤，所以要非常小心地预防外渗的发生。

一种常见的方法是将可能造成组织损伤的药物注射到躯干中较大的血管中，而不是手臂中较小的血管中。根据不同的原因，注射部位的反应可以用热敷、冰敷、可能是某些药物外渗的解毒剂来治疗，在极端的情况下，可以进行整形手术[5]。

辉瑞-BioNTech的COVID-19疫苗还有其他常见副作用，包括疲劳、头痛和肌肉疼痛。大约63％的人报告有疲劳感。55%的参与者报告有头痛，38%的参与者报告有肌肉疼痛。在大多数情况下，这些症状在一天左右的时间内即可缓解并缓解。

少数参与者报告接种疫苗后出现寒战、关节疼痛或发烧的不良反应。

参与者在接种第二剂疫苗后更有可能报告此类症状。“对第二剂的反应往往是更激烈反应，因为第一剂疫苗打入后，你的免疫系统已经被它唤醒了，你产生了相应的抗体，所以第二剂打入后，副作用会来得更快更激烈。”海因茨说[6]。

CNBC报导，参与疫苗临床试验的志愿者，24岁的亚西尔·巴塔尔维（Yasir Batalvi）说：“注射后，我的副作用和第一次一样：局部疼痛和僵硬，但更严重一些。我的手臂酸痛得更快，到我回家的时候，我开始感到疲劳，就像任何人都会感到如果他们得了流感一样，当天晚上，更显著的症状呈现了。我发了低烧，寒战，那个晚上很难熬[7]。”

一个24岁的年轻人都感觉很难熬，更不用说老年人和孩子。这就是辉瑞的疫苗，它已经是几个疫苗中相对比较好的。

除了注射部位反应，更可怕的是，在接受疫苗的参与者中报告了4例贝尔氏麻痹症（Bell’s palsy），而在得到安慰剂的参与者中一个都没有。

贝尔氏麻痹症是一种导致面部肌肉暂时性无力或瘫痪的疾病。当控制面部肌肉的神经发炎、肿胀或受压时，就会出现这种情况。这种情况会导致一侧脸部下垂或变得僵硬。您可能在受影响的一侧难以微笑或闭上眼睛。

在大多数情况下，贝尔氏麻痹是暂时的，症状通常在几周后消失。虽然贝尔氏麻痹可以发生在任何年龄，但这种病症在16至60岁的人群中更为常见。贝尔氏麻痹症是以苏格兰解剖学家查尔斯-贝尔的名字命名的，他是第一个描述这种情况的人[8]。

3.信使RNA技术从未在人类身上使用过，有很大的潜在风险

辉瑞/BioNTech和Moderna的疫苗使用的都是信使RNA技术，这种技术从来没有在人类身上使用过，所以根本无从得知会有什么样的后遗症。

信使RNA，缩写为mRNA，是一种将DNA指令付诸行动的分子。在细胞内部，mRNA被用作模板来构建蛋白质。贝克雷吉安-丁（Bekeredjian-Ding）教授说：“mRNA基本上就像一个蛋白质的前期形态，它的（序列编码）就是以后蛋白质的基本构成。”

为了生产mRNA疫苗，科学家们模仿病毒用mRNA来构建其感染性蛋白质的过程，生产出一种人工合成版本的mRNA。这种mRNA被输送到人体内，人体细胞将其读作构建该病毒蛋白的指令，因此自己会制造一些病毒分子。

这些自己制造的蛋白质是孤立的，缺乏其它组件配合，它们不会组装成病毒。免疫系统会检测到这些病毒蛋白，并开始对它们产生防御性反应。[12]

3.1 缺乏mRNA用于激活B细胞的研究

迄今为止，大多数利用mRNA引发免疫反应的研究都集中在癌症研究上，肿瘤mRNA被用来帮助人们的免疫系统识别和应对特定肿瘤产生的蛋白质。贝克雷吉安-丁教授说：“这项技术对于肿瘤领域来说非常好，因为你可以开发出针对患者的疫苗，因为每个肿瘤都是不同的”。

以这种方式使用肿瘤mRNA，可以激活人体的T细胞，杀死肿瘤细胞。这个经验也可以用于冠状病毒。贝克雷吉安-丁教授说：“在病毒感染中，我们常常知道需要强大的T细胞反应，因为病毒喜欢躲在细胞中。”

但要对抗SARS-CoV-2这样的病毒，很可能还需要激活后天免疫系统的另一部分–B细胞，B细胞会产生抗体，将病毒标记出来，以便被人体消灭。而这方面的经验很少（除了动物感染模型），因为对于肿瘤模型来说，这并没有那么大的意义。[12]

3.2 人体实验数据严重不足，有很多悬而未决的问题

因为mRNA疫苗现在才开始在人体中进行测试，所以相比其他技术手段的疫苗，使用这种技术的疫苗有很多相当基本的数据都是没有的，只有通过人体试验才能得到答案。

“我认为，目前真正的挑战是要了解这些疫苗是否真的能够在人体内启动足够的保护性免疫反应，并了解例如需要多少数量的mRNA才能做到这一点。”贝克雷坚-丁教授说。

其他悬而未决的问题包括：疫苗所选择的蛋白质是否是防止冠状病毒在体内感染的正确蛋白质。免疫反应对这种特殊的冠状病毒有多大的针对性。各种免疫力能持续多久。以及它是否会引起副作用，如发红和肿胀等炎症反应。在最坏的情况下，是否加重疾病。[12]

3.3 以前的mRNA药物试验都在试验早期阶段以失败告终

Moderna和辉瑞/BioNTech公司的疫苗，目前的数据显示出90%至95%的高有效性。目前还不清楚为什么有效性这么高，而与此形成鲜明对比的是，此前针对COVID-19以外的病原体的mRNA药物试验并不乐观，不得不在试验早期阶段放弃。

医师科学家玛格丽特-刘（Margaret Liu）表示，这可能是由于 “投入研发的资源量太大”，也可能是疫苗 “引发了对mRNA的非特异性炎症反应，尽管改良的核苷技术减少了炎症，但还没有完全消除炎症”；

“这也可以解释了某些接受mRNA SARS-CoV-2疫苗的人报告的疼痛和发烧等强烈反应”。这些反应虽然严重，但都是短暂的，另一种观点认为是对脂质药物输送分子的反应。[13]

3.4 mRNA分子非常脆弱，可能在没进入细胞前就分解，针可能白打

与DNA分子不同，mRNA分子是非常脆弱的分子，在暴露的环境中几分钟内就会降解，因此mRNA疫苗需要在非常低的温度下运输和储存。 在细胞外，或其药物输送系统中，mRNA分子也会很快被宿主分解。

mRNA分子的这种脆弱性是任何mRNA疫苗疗效的一个障碍，因为在进入细胞之前，它就会被大量分解，这可能会导致一种错觉，人们以为打了疫苗后就有免疫力，但其实他们没有得到疫苗带来的免疫力。[13]

3.5 仓促上马这种技术的原因很可能是容易大规模生产

通过让人体自己产生病毒蛋白，mRNA疫苗省去了一些制造过程，比传统疫苗更容易大规模生产。mRNA疫苗的生产过程只需几个月，而传统疫苗需要1-2年[12]。考虑到全球很多国家需要大规模接种疫苗，采用这种技术的生产优势太大了。

疫苗的研发时间一般长达10年，典型的成功率大约为6%。大量的时间用于评估疫苗的短期和长期安全性。数据收集问题是疫苗开发的典型问题，比如免疫系统较弱的老年（和婴儿）人群对疫苗的反应可能不同于年轻至成熟的成人人群，需要时间来得出这些结论。

信使RNA技术从未在人类身上使用过，它的安全性问题更加严重，需要更长的时间来排除安全隐患。而在这次病毒全球大爆发的情况，可谓病急乱投医，各种不合规的研发结果都被特别允许投入市场。

信使RNA技术就是其中一个例子，打这种疫苗其实就相当于是在其他新药研发阶段报名做志愿者，要冒很大的风险。

此外，基于抗体的感染增强效应（ADE）可能导致更为危险的情况。同样需要时间和进一步的研究来评估这些问题。

4.ADE效应可能导致感染急剧发作，甚至猝死

ADE是基于抗体的感染增强效应(Antibody-Dependent Enhancement)。

当患者感染过某种病毒并康复后，患者不会再感染同类型病毒，因为患者体内的针对该病毒的抗体能迅速识别病毒并消灭，而ADE效应是指，针对该病毒的抗体可以帮助其他类型的病毒增强感染，使患者对其他病毒变得更加脆弱。

最常见的有ADE效应的例子是登革热病毒。登革热是在热带环境中非常常见的病毒，可以通过蚊子传播，每年有超过1亿人感染，4万人死亡。

登革热病毒有四种血清型，都支持保护性免疫。同型保护已显示出持久的保护作用，但针对不同血清型的交叉中和抗体的持续时间相对较短（最多持续2年）。当保护性抗体滴度下降时，再感染不同血清型就会变得危险。

非中和抗体取代中和抗体，与登革热病毒结合形成复合物，这时本来应该被消灭的病毒，不仅没有被消灭，反而被免疫系统带入细胞内部，利用免疫系统大量复制，导致病毒复制速度远远超过通常情况下的病毒感染速度，病人病情急剧恶化，甚至死亡，这种感染增强现象，就是ADE。

研究人员证明，如果患者以前感染过登革热，当患者体内的抗体滴度低到某个程度时，则发生登革热严重病程的风险增加。在登革热的疫苗研发过程中，研究人员也观察到了这一现象。2015年，该疫苗被批准进行疗效试验。

经过对疫苗试验的跟踪数据进行评估，证实打疫苗3年后的儿童的住院率高于不打疫苗的对照组。ADE效应被认为是导致这个结果的原因。在经过足够长的时间后，患者的免疫力已经下降到足够低的程度时，一旦再次接触感染源，就容易发生ADE[9]。

2016年在菲律宾接种登革热疫苗的83万孩子中有14个死亡，虽然比例并不高，但是后果很严重，其中6个是在打了疫苗的30天内突然死亡的。

菲律宾因此叫停了该疫苗[10]。还向赛诺菲索赔7000万美元疫苗费，并准备追查其中的腐败、渎职，追责赛诺菲和为其背书的WHO。

2003年的SARS和后来的MERS在体外动物模型中都观察到了ADE现象，与它们非常相似的SARS-COV-2会有ADE效应的可能性非常大。连中共的专家高福都承认新冠疫苗面临的最大挑战是ADE[11]。

今年急速批准上市的疫苗，和其他的新药一样，都是没有经过大量人群的长期数据跟踪的，尤其是不能证明是否可以避免ADE效应。如果不能避免ADE，那么打疫苗反而是增加风险。

5.应该尽可能的发声反对强制接种疫苗

综上所述，目前的新冠疫苗仍然很不成熟，即使是国外的疫苗也存在很大的问题，更不用提中共的国产疫苗。

中共向来不把人命当回事，把医疗当作赚钱敛财的工具来用。这一次恐怕又会借疫苗之名大肆敛财。但与敛财相比，更令人担心的是，不可靠的疫苗会不会给接种者带来长期的副作用和ADE风险。

中共在此时放风说打疫苗是义务，可能是为之后的强制接种疫苗做准备。希望在国内外有能力发生的人一定要对此高度重视，说出自己的反对强制接种疫苗的意见，或许还有机会迫使中共放弃强制推行疫苗的政策。
