## 【香港要闻】港共政府 为巩固 极权统治拟修真枪配件规例
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/916877/)

搜集： PO / 编撰: 心听见

**【据苹果报讯】港府近年屡以公共安全为借口，限制市民人权自由。继早前建议购买电话卡须实名制后，政府拟收紧规管真枪配件。保安局昨指，为打击不法之徒不当使用枪械，建议修订《火器及弹药（枪械宣布）规例》，将指定的真枪配件纳入枪械定义，公众咨询至3月16日，为期四周。民主党林卓廷担心修订或酿成滥告，「我想要好小心了，随时不是用来发射的配件，都可以拘捕」。**
![]()![](https://gnews.org/wp-content/uploads/2021/02/NEWNEWNENWNENWNNENENNENEWNWENEWNWENEWNWEN.png)
政府昨日仅透过新闻稿交代咨询文件。当局指近年本港涉及真枪的罪案增加，其中有不法之徒利用现时《火器及弹药条例》枪械定义的漏洞，将真枪配件付运到港，再在本地装嵌为功能齐全的真枪。海关于2016至2019年间，共侦破24宗涉及怀疑真枪及配件的走私案，2020年的案件数目更增至35宗，涉及超过550件真枪配件。

有熟悉枪械人士称，现时要将真枪配件合法从外地进口，须先向政府申请许可证，并由海关执法。市面部份叫价甚高的气枪或会使用真枪部件，如肩托、前握把等，修例对持有或售卖此类模型枪的商店或有影响。

战友点评：

中共在香港借修改律法来达到目的，已经不是什么秘密。为了全面控制香港，修例已经修上瘾，一部邪恶的「国安恶法」已经满足不了中共的邪恶欲望，任何被视为不利中共继续作恶的香港律法都被一一整改，用极权流氓手段来继续巩固他们在香港独栽的统治。

而港共政府当局继续为修例解释，指根据现行规例，控方须证明配件是「用作或拟用作发射投射物」，增加检控的难度。为堵塞漏洞，当局建议修订《火器及弹药（枪械宣布）规例》附表，将部分真枪配件纳入枪械定义，包括枪管、枪膛和枪闩等，强调修订是为了打击不法之徒不当使用枪械，威胁市民生命及财产安全。
