**摘要：**民弱国强；民强国弱。故有道之国务在弱民。民胜其政，国弱；政胜其民，兵强。

**原文：**以刑去刑，国治。以刑致刑，国乱。故曰：行刑重轻，刑去事成，国强。重重而轻轻，刑致事生，国削。（选自《商君书·去强第四》）

**译文：**用重刑杜绝犯罪，国家就能治理好。刑罚太轻导致犯罪很多，国家就混乱。所以轻罪重治，（因为威慑力量）导致即使不用刑罚也能杜绝犯罪，国家就强大。重罪重罚、轻罪轻罚，虽然有刑罚但还是会经常发生犯罪，国家就会削弱。

**原文：**民弱国强；民强国弱。故有道之国务在弱民。（选自《商君书·弱民第二十》）

**译文**：民众弱，政权就强；民众强，政权就弱。所以要想政权有作为，就必须削弱民众。

**原文**：辩慧，乱之赞也；礼乐，淫佚之征也；慈仁，过之母也；任举，奸之鼠也。民胜其政，国弱；政胜其民，兵强。（选自《商君书·说民第五》）

**译文**：民众善辩有智慧，就会生乱；礼与乐，是淫秽的征兆；仁慈，是纵容犯罪的根源；任用举荐，奸邪就有藏身之地（结党营私）。民众的力量胜过政权，政权就弱；政权压制住民众，国家的兵力就强大。

**解读：**这几段话充分体现了商鞅的弱民思想。为什么民弱国才强、民强国就弱？为什么不能民强国也强呢？商鞅的逻辑就是所有极权专制政权的逻辑，将政权和民众对立起来，视民众如猪狗，民众就是政权的奴隶。

弱民的方法有很多，前面说到的愚民和壹民，不能让民众有智慧，不能让民众从种田和战争之外获得财富、利益和荣耀，用连坐的方法让民众互不信任，用奸民治善民，都是在弱民强国。商鞅认为，对民众要用重刑，要形成威慑，即使很轻的罪也用重刑，实行恐怖统治，这样民众就会安分守己不敢轻举妄动，不敢有异想，便于统治。商鞅之前的周礼上说治乱世用重典，但商鞅认为用重典防乱世，从思辨角度来说是有道理的，但是滥用重刑对民众却是苦难，商鞅防备的不是民间之乱，是防止民众对政权之乱，字里行间透着极权专制统治的思想。

商鞅之后历代专制统治者都在想着怎么弱民，在经济上对民众苛以重税，打击地方豪强，防止民间和地方势力对政权形成威胁。除了极少数的如文景之治、贞观之治、宋朝盛世、康乾盛世等专制制度下极少的盛世，皇朝政权实行轻徭薄赋、与民生息的政策，其他90%的时间民众都处于贫穷和战乱之中。在刑罚方面，专制政权制定了一系列惨无人道的严刑峻法，如腰斩、活剥皮、凌迟、灭族等，骇人听闻。
