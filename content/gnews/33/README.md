这一篇记一下英国的外交。

从标题也许你已猜到，“骑墙”可以说是英国外交策略的一大特色，英国的外交在全世界出了名的老练，在国际舞台上特别擅长平衡外交和大国博弈，长袖善舞。也有人说英国这个国家骨子里就是现实、投机、骑墙、见风使舵、两头下注、没有底线。说白了，就是利益，争取英国利益最大化而已。

一国以自己的国家利益(National interest)优先，如川普奉行“美国优先”（American First）政策一样，其实无可指摘，甚至值得提倡，美国总统不以美国国家利益为最优先，难道还要他国利益为先？处处损害自己国家的形象，这样的蠢货领导人也不在少数。

美国外交以强大的国力为基础，强大的军事，经济和货币构成的硬实力；加上好莱坞，美国梦等美国文化组成的软实力，难有匹敌。只是过于“单边”的外交策略，树敌也多。反观英国，作为与美国关系最为紧密的特殊伙伴之一，在很多国际事务上的立场基本和美国一致，例如反恐，人权，伊战，遏制和制裁俄罗斯，维持国际规则等。但在不少外交事务上，英国有不同于美国的主张。而且从全世界范围放眼来看，过去一两百年历史中，英国外交上基本没有“站错队”，英国在全世界的朋友很多，敌人较少。这一切大抵归功于“骑墙”术的高明，不信？且听俺道来。

英国位于欧洲，然而与欧洲大陆相隔一条英吉利海峡的地理位置，决定了英国的外交“生存法则”：主张欧洲大陆均势，反对一国在欧洲鹤立鸡群，不能允许欧洲大陆产生单极力量。通过保持欧洲大陆若干大国之间的基本均势，避免出现实力对比的失衡，既可以避免对英国的威胁，也可以以保证英国对欧洲大陆事务干涉的合理性和有效性，以便有效控制欧洲，维护欧洲和世界霸权。

英国发起或参与的很多场战争，体现了务实外交和利益平衡之高超技术。

特拉法加海战，1805年

西班牙是16世纪海上霸主，也是第一个“日不落帝国”，西班牙无敌舰队在1588年遭受英国海军重创（也有天气和运气因素），失去海上霸权，但令英国正式夺取海上殖民霸权的，是在1805年英国海军大破西法海军的特拉法加海战。这是英国海军史上的一次最大胜利，几乎歼灭了整个法西联合舰队，摧毁了拿破仑进攻英国本土的计划。英军主帅霍雷肖·纳尔逊（Horatio Nelson）海军在以前的征战中失去过右眼和右臂，不幸的是他在这次战斗中阵亡。英国人在伦敦市中心修建了特拉法加广场，广场中央树立纳尔逊将军全身立像以纪念这位国家英雄。

滑铁卢战役，1815年6月18日

英国主导的参与的“反法同盟”在1793年到1815年发动过六次对抗新兴的资产阶级法国的战争，全部失败。决定性的滑铁卢战役是第七次战争中的终极会战，以反法联军的胜利而告终并结束了拿破仑王朝，改变了欧洲历史。这场战争也为伦敦成为欧洲金融中心打下了基础，因为滑铁卢战役的胜利让英国公券（国债）大涨，罗斯柴尔德家族凭借情报网和决断发了大财。

滑铁卢大捷是英国历史和军事史上最为光荣伟大的时刻之一，是英国“大陆均势”政策的胜利和最好体现。英军主帅惠灵顿公爵一战封神，从伦敦最大的地铁站之一滑铁卢站的名字，到惠灵顿公学，惠灵顿花园或广场，无数个惠灵顿酒吧，滑铁卢战役成为不列颠辉煌史中的浓重一笔。经此一役，法国失去了在欧洲大陆的统治地位，而英国得以维持欧陆均势，顺利进入维多利亚女王时代的黄金发展时期，也是大英帝国最为辉煌的时期。

克里米亚战争，1853-1856年

克里米亚战争是第一场现代战争，火炮枪械和水雷等兵器得到大发展。无论军事发展角度和欧洲地缘政治与外交的变迁方面，影响深远。英国担心俄罗斯在欧洲大陆过于强大影响欧洲大陆的均势，防止欧陆出现一个独占的霸主，但英国却要保持海上的绝对霸主优势，绝不允许其他国家挑战自己的海上霸权，“保持英国作为欧洲仲裁者的地位，而后是全世界的仲裁者，这些由英国的经济和军事实力决定”——这就是英国高超骑墙术的基础。

此外，英国女护士南丁格尔在克里米亚战争中赴前线护理伤员，使伤病员死亡率下降，由此改善了战场医疗并诞生了南丁格尔护理制度——即后来的护士这一职业。

一战，1914-1918

为了“欧洲均势”战略目标而从被动参战到成为战争主力。英军一战四年下来平均每年阵亡近30万人，其中索姆河一天战死6万英国人，更不要说数十倍多的伤残、神经失常者。几乎每个英国家庭都感受到一战的残酷性。虽然英国在一战后的巴黎和会上保持了国际事务领导者的地位，但经济、军事实力严重倒退，政治、军事霸主地位开始动摇。一战让英国保持了日不落帝国最后的尊严和辉煌。

二战，1939-1945

熟悉历史的人知道，丘吉尔的前任首相张伯伦的绥靖政策鼓励了纳粹希特勒，这可能是英国外交史上“骑墙”最为失败教训最为惨痛的一次。

好在有邱吉尔，唤醒国民认识到绥靖政策的祸害，并为国家指出正确的方向，对抗纳粹德国的威胁，永不投降的精神让丘吉尔成为最伟大的英国人之一。英国虽然不是美苏那样的主角，但作为欧洲抗击纳粹德国的中坚力量和同盟国一员赢得了反法西斯战争胜利。邱吉尔也是意识到斯大林领导的布尔什维克主义对西方世界威胁的第一人，发表铁幕演讲开启了东西方冷战。二战之后，英国曾经的世界霸主地位彻底让位于美国，但英国仍然作为国际新秩序缔造者之一，成为联合国五常国之一，在欧洲仍然具有重要影响力。

伊拉克战争，1998

引用英国前外交大臣库克的话，“事实很清楚：英国参加了一个没有任何国际组织同意授权的战争，北约也好、欧盟也好、联合国安理会也好，都未同意发动这场战争。美国财大气粗，有能力独自负担这场战争，但英国并非超级大国，单边行动不会使英国利益得到最佳保护，英国需要多边协议和一个按规则行事的世界秩序。”

然而，当年的英国仍然追随美国参战伊拉克。究其原因，或者利益驱动，不外乎几个考量：第一无疑是石油，第二打击欧洲兴起的超国家联盟运动，第三是在伊拉克战后重建中的大笔定单。同时英国为参战也付出伤亡、国内人民反对以及国际口碑下降的代价。

脱欧，2016-2020

脱欧这件事从“黑天鹅”到落地为安，在英国本土、在欧洲乃至全世界都争议未休，但有一点不得不承认：脱欧意味英国摆脱了欧盟束缚。至于结果到底对不对，好不好，一定是有得有失，得失哪一个大，尚无定论。英国脱欧不应该被称作“战争” ，而更像是一场马拉松式的离婚。两方为了多分家产，撕破脸皮，过程虽然痛苦，但终于在2020年1月31日，英国在法律意义上脱离欧盟，正式进入后脱欧时代。

英国《金融时报》专栏作家马丁·沃尔夫一语道破英国的处境：“英国因体量太小而无法与欧盟平起平坐，同时因实力太强而必定被欧盟高度重视。”英国必须有效抗衡以法德为首的欧盟，才能发挥在欧盟事务上的政治影响。

英国脱欧的谈判过程，一方面展示了平衡、腾挪、驾驭、博弈和坚持核心利益的技巧，一如它当初加入欧洲议会，融入欧洲大家庭（单一市场）一样的战略和技巧，另外一方面将国家重新定位于“全球英国”——在世界舞台上将有更大的发展空间，这至少是英国人的想法，而且从英国过去的外交实绩和历史来看，这并非一厢情愿，有可能念想成真。

英国和美国是拥有共同的价值观、历史和语言的长期亲密盟友，而且经历过数次战争洗礼和历史考验，在触及国家安全、核心利益的事情上，两国基本都在同一阵线。虽然英国很多时候被诟病为美国的小弟，跟屁虫，但并非所有事情都跟随美国，尤其脱欧之后，对欧盟，对日本、中共国或其他经济体，英国有更加独立的话语权和自主权。

毫无疑问，中共国的崛起不可忽视，而且“中共国作为一个非民主大国”，拥有真正挑战西方霸权的经济、国防和外交实力。”中国巨大的人口、市场，投资或进口所需，都是白花花的银子，很难让重视务实外交的英国或其他国家拒绝。英国曾是第一个承认中华人民共和国的西方国家，也曾是第一个加入亚投行并积极拥抱“一带一路”的西方国家。数年前中英关系一度火热，曾有“黄金时代”定调。然而世易时移，英国重视钱但并非只认钱，在西藏、香港、华为、核电、新疆、疫情等重大问题上，英国都坚持维护立场表现出一定的国际担当，呼吁中共国改善法治、人权状况，倡导促进民主、市场透明和监管、尊重国际贸易公平和规则、气候变化承诺等。

中、美之间选择哪一个？

擅长“骑墙”的英国恰恰永远都不会做这种选边站队，非一即二的选择题，英国秉持所谓“避免选择”战略，即使不得已选择也留有余地。既有独立于欧盟的政策，也能游刃于中美俄欧盟等大集团之间，骑墙周旋发挥其独有的作用。

最近的G7外长会议，除了七国集团的成员国，英国作为主办国还邀请了澳大利亚、印度、南非以及东南亚国家联盟（ASEAN）的主席国文莱外交部长参与峰会，这几个国家全部是英联邦成员国，在经济实力和地区事务上有一定影响力。此外韩国外交部长也受邀参加峰会，朝鲜半岛的安全与稳定无疑是关注点。看看，英国的外交手腕还算纯熟？

此外还有“五眼联盟”——同属“盎格鲁圈”的美、英、加、澳、新，不仅都以英语为母语，还拥有共同的价值观、政治传统、军事结构。除美国外，另外四个国家同为英联邦成员，有着共同的国家元首伊丽莎白二世女王，英国的“小圈子”对于美国老大哥也玩砝码和制衡？情报共享是五眼联盟的宗旨，个人主义、自由和法治是“盎格鲁圈”这一愿景的规范基石，这样的基石积累起来的“墙”，你觉得英国又骑得如何呢？

与其说“骑墙”外交不如说是务实外交曾经让英国强盛不衰、矗立不倒而且长袖善舞，获得最大化的利益。但不得不承认，英国的实力今非昔比，后脱欧时代的英国向何处去，如何在不断变化的世界格局中掌握主动和平衡，将取决于英国精英政治家的智慧和选择。

也许很难再出现一个丘吉尔，但不要忘记，取得滑铁卢战役胜利的英军主帅惠灵顿公爵有一句著名的话：滑铁卢战役的胜利早就在伊顿公学的操场上决定了——这句话反映的，是英国顶尖贵族学校培养出来的精英的一种自信和骄傲：作为普通官兵一员——可以坚守阵地，严守纪律，勇敢牺牲，战胜敌人，而作为社会精英和治国领袖——有理性和智慧，务实进取，有负责担当，百折不挠精神，可以领导一个国家战胜挑战。

伊顿公学是英国三百年来政治家的摇篮，英国77位首相中有20位就读伊顿，这种精英教育的传统，不断与时俱进，发扬光大，是不是更加有力量呢？这样的墙，你难道不希望自己的儿子有机会骑上去吗？
