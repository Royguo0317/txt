据《每日信号》（The Daily Signal）文章评论，不管是上个月的阿拉斯（Alaska）加会面，还是其他地方的会面，中共领导人以居高临下的口气与拜登政府官员对话没什么好奇怪的，他们为了把美国构陷成一个充满不公的种族主义贼窝堪称费尽心机。毕竟，这也是精英和教育者们一贯向学生们描述的美国。

但美国人逐渐看清了美国极左翼和中国共产党之间的利益趋同和辞令游戏。因此，中共外长杨洁篪在阿拉斯加安克雷奇（Anchorage）对美国国务卿安东尼·布林肯（Antony Blinken）和美国国家安全顾问杰克·苏利文（Jake Sullivan）的训话不过是拾人牙慧，诸如“美国要改变自己的形象”，“很多美国人对自己国家的民主完全没有信心”之类的说辞，早就被MSNBC（美国广播公司新闻频道）或者美国高校说烂了。

无独有偶，外交部一号战狼赵立坚在新闻发布会也是大嚼别人剩下的馍——把华裔美国人遭遇袭击说成是特朗普政府的言论和对中共国糟糕人权记录的批评的结果——民主党众议员杰里·纳德勒（Jerry Nadler），甚至拜登政府官员早就说过这样的话。中共最高领导人也是换汤不换药，他们像杨洁篪一样拿“黑命贵”（Black Lives Matter）说事，责难美国在人权方面面临的挑战是根深蒂固的，“黑命贵”之类的人权问题不是过去四年的新产物，而是历史问题。

布林肯和沙利文应该作何反应？像前届政府高官一样回怼杨洁篪，厉声为美国正名——“美国实际上是地球上最自由、最繁荣的乐土，种族主义难望中共国项背”？但是他们不能这样说。如果他们说实话，他们自己所属的党派将会把他们生吞活剥。

美国左翼越来越受到诸如中共高官之类的极端因素的影响，这意味着左翼希望美国经历深度变革。这里面有几层含义：

1. 中共和极左派都喜欢“黑命贵”组织及其领导人从内部搞垮美国——中共想把美国这个对手变弱，而激进的左翼分子醉心于摧毁美国的体制和经济系统，最终把美国社会主义化。
2. 虽然各怀鬼胎，中共和极左派却一致想让美国停止批评中共国：无论是其糟糕的人权纪录——在新疆，大多数穆斯林维吾尔人被关押在集中营中进行汉化；在香港，中共毁约一国两制；在西藏，中共已经霸占了60年；在台湾，中共正对着这个自由的资本主义宝岛流口水。还是 COVID-19 起源于武汉这个铁一般的事实——中共和极左派置中国共产党的谎言于不顾，“齐心协力”、不遗余力地指责前总统川普防疫不力，恨不得把川普刻在脑子里。对于他们而言，美国无论作为一个社会，还是一个国家，都没有资格站在道德层面批评中共国。
3. 中共和极左派也将利用目前因仇视亚裔美国人而导致的犯罪率飙升为自己捞取政治利益——中共不遗余力向世界，尤其对国内人民，力证美国模式不如中共国模式，美国左翼则把亚裔美国人的成功视为“洪水猛兽”，是对其意识形态的冲击。在针对攻击亚裔美国人犯罪率上升的小组委员会听证会上，纳德勒说：“重要的是要认识到，这波（仇视）不仅仅出于对中共病毒大流行的恐慌而自发产生。其中部分责任完全归咎于因病毒和眼下的地缘政治争端妖魔化中国的（美国）政要，反过来，亚裔美国人就成了替罪羊。”纽约时报（The New York Times）的彼得·贝纳特（Peter Beinart）最近写道，若要批评中国，美国还必须进行自我检讨。“如果美国政客只谈中共国的军力和好战，而又没有提醒美国人中共国的军事预算只是美国的一小部分，中国与主要的民主国家的关系很好，而且最近几十年来中共国发动的战争比美国少得多。这将加剧某种歇斯底里的恐惧感，在过去的一百年里，脆弱的美国人一次次受害。” 这些美国人，特别是华裔美国人，努力工作、学习，然后享受回报。他们脚踏实地、遵守法律，了解新国家的人文环境，并成功组织起来阻止种族歧视。这就是为什么左翼必须抓住机会不动声色地积怨，并设法使他们自己看起来是受害者。


极左不希望普通人成功，因为那意味着会有新人进入精英体系、分走他们的蛋糕。他们希望粉碎美国良性的社会系统，中共的想法也是如此。这就是美国极左和中共有很多共同利益和共同立场的原因了。
