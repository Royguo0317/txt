## WeChat 是中共打造數字帝國的監控利器
`2020-08-12 10:16 GM30` [轉載自GNews](https://gnews.org/zh-hant/294190/)

**作者：喜馬之光**

關鍵點：微信通過對民衆生活的融入，悄無聲息地把監控的觸角伸進了我們的生活。在中共國大數據庫中，你就是一個透明人。無論是一言一行、或是旅遊消費、還是活動半徑，都在數字帝國的監控之下。

作爲一款用戶數量達12億的APP，幾乎在華人世界裏都非常熟悉。然而，在我們普遍使用的同時，我們的隱私和數據的安全是否被保護呢？這款APP又是如何監控你的方方面面呢？

微信會監控你什麽？

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/12100806/1-45.jpg)

微信的生態鏈覆蓋了普通民衆生活的方方面面。從功能劃分：社交類、日常生活服務（第三方平台）、金融服務、廣告等等。

a 社交類

主要通過文字、照片、地理位置、語音、視頻來實現交流。APP會獲取你的通訊錄信息、手機照片、以及你的朋友圈信息。你經年累月的社交圈內交流信息（你的文字、聲音、圖像、照片、活動地理信息、喜歡的公衆號等）將被保留在數據庫中，並處理分析。通過收集這些海量的信息，微信就會慢慢掌握你的職業和社交圈子，你的興趣愛好、政治經濟觀點（朋友圈信息和公衆號以及點贊）。描繪出你職業、性格、興趣愛好、政治社會觀點、社交活動及活動範圍（即使你不發信息，也可以通過你的朋友圈的信息來定義你）。比如，你參加了哪些微信群等等。

b 金融服務

主要提供各種支付服務（收付款服務、信用卡還款、生活繳費、零錢通或基金理財（貨幣基金、債券基金、股票基金、混合基金等），小額貸款服務、保險服務等等。支付服務一般必須綁定你的銀行卡。這樣在你消費支付和理財的同時，微信悄悄的掌握了你的消費偏好、消費地點、部分金融資産的信息。比如，生活繳費功能類裏有：水電氣繳費、固話費、寬帶費、油卡充值等等。別小看了這些生活信息。這些點點滴滴的生活信息，微信通過大數據都可以描繪你平時的生活場景、生活方式、以及你的生活消費能力。

C 第三方平台

第三方平台基本涵蓋了吃穿住行娛樂等方面。比如，滴滴打車服務、車票和機票訂購、美團外賣、京東和拼多多購物、電影購票、酒店預定、甚至貝殼找房。通過這些細致入微的信息采集，你的生活出行圈已經記錄在大數據庫裏了。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/12100855/2-21.jpg)

微信會保護數據安全嗎？

微信的母公司騰訊，總部在深圳。中共國直接在騰訊公司設立了公安局調證中心。微信已經赤裸裸的淪爲中共統治的信息采集中心、維穩中心。

母公司騰訊與FACEBOOK 對比

a 科技研發

騰訊的起家就是一部抄襲的曆史。QQ1999年出産，抄襲了1996年出世的ICQ。WeChat2011年出産，抄襲2009年問世的Whatsapp。國際上抄襲完，又抄襲國內。財付通抄襲支付寶，QQ網遊抄襲盛大遊戲、QQ醫生抄襲奇虎360等等。

b 營收對比

母公司騰訊在2019財報營收3772億元（約538億美元）。這是基于其國內市場霸主地位和集各種金融保險牌照于一身的壟斷資源所決定的。分析騰訊的營收，主要是三個板塊。增值服務、金融支付、網絡廣告

增值服務主要營收構成是網絡遊戲、會員服務、雲服務等占公司營收53%。金融支付主要爲支付平台和理財平台，營收占公司營收26.8%。網絡廣告，營收占比僅18.12%。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/12101200/3-24.jpg)

對比FACEBOOK 活躍用戶數量16.6億，2019財報營收707億美元。其中廣告收入爲696億美元占公司營收98.5%。公司擁有800萬個廣告主。但針對公司營收單一問題，facebook已經嘗試在旗下另一産品Instergram 開發電商平台。

c 數據安全對比

Facebook 曾發生過5千萬用戶數據泄密事件。數據泄密使得對其股價産生巨大影響。在這種情況下，Facebook 只能加大安全防範，才能保住其社交類市場霸主地位。因此美國資本市場的正常反應促使Facebook做出正確措施。對比微信的母公司騰訊，公然將民衆所有信息交由公安部門管理，嚴重侵犯用戶隱私。

總結：微信依靠其壟斷地位，通過對用戶社交、日常生活等海量信息的采集。全方位的監控民衆的生活，已經徹底淪爲中共的數字監控工具。
