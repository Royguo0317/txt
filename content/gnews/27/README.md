## 五角大楼采购部门负责人的被提名者认为与中共国在科技方面的竞争非常关键
[!\[\]()!\[\](https://spark.adobe.com/page/VlUBDBwQOYGDz/images/d566f963-1e94-4906-be24-4b6966a43bc7.png?asset_id=dc593064-521f-429c-8225-8db3fe009efc&amp;img_etag=%227fb4445a30edee1bf16fb317b2db4f24%22&amp;size=1024)](https://spark.adobe.com/page/VlUBDBwQOYGDz/images/d566f963-1e94-4906-be24-4b6966a43bc7.png?asset_id=dc593064-521f-429c-8225-8db3fe009efc&amp;img_etag=%227fb4445a30edee1bf16fb317b2db4f24%22&amp;size=1024)国防创新部主任迈克尔·布朗（右）于2019年9月12日在内华达州克里奇空军基地会见了国防数字服务部成员。来源：美国空军
华盛顿 — 迈克尔·布朗是一位资深的科技产业高级管理人，自2018年以来一直担任五角大楼商业拓展办公室负责人。白宫4月2日宣布，布朗被拜登政府提名为国防部采购负责人。

布朗曾被提名为国防部负责采购和维持的副部长。他目前是位于硅谷的国防创新部门（DIU）的主任。奥巴马政府于2015年成立了该办公室，目的是与初创企业和风险投资者建立联系，并帮助把私人企业的技术应用到军事项目中。

布朗在DIU工作期间，与中共国的技术竞争一直是他最关注的问题。在2018年，他与人合作撰写了一篇论文，该论文论述了中共国如何利用风险投资来获得“美国创新皇冠上的宝石”。

该论文称，近年来，为早期创始阶段技术公司提供资金的中共国风险投资交易迅速增长。“中共国投资的技术领域与美国公司投资的领域基本相同，这些领域将成为未来创新的基础：人工智能、自动驾驶汽车、增强虚拟现实、机器人技术和区块链技术。”

在布朗的领导下，DIU成为了诸如火箭实验室、维珍轨道和相对论空间等新兴太空发射公司的早期支持者。DIU选择了这些公司来为国防部发射实验性的火箭，主要目的是用来解决国防部如何影响商业发射服务。

[!\[\]()!\[\](https://spark.adobe.com/page/VlUBDBwQOYGDz/images/a67e9553-24f0-48e4-81ae-c83ed577e5cc.jpg?asset_id=9fdd58cc-0ffa-4418-92e0-0a6b6e79d8f7&amp;img_etag=%2215cb02041e36b7f4e786d9920fee9d6b%22&amp;size=1024)](https://spark.adobe.com/page/VlUBDBwQOYGDz/images/a67e9553-24f0-48e4-81ae-c83ed577e5cc.jpg?asset_id=9fdd58cc-0ffa-4418-92e0-0a6b6e79d8f7&amp;img_etag=%2215cb02041e36b7f4e786d9920fee9d6b%22&amp;size=1024)

DIU还投资了太空监测初创公司LeoLabs，该公司使用相控阵雷达网络跟踪低地球轨道上的卫星活动。

作为负责采购和维持的副部长，布朗将监督每年超过2400亿美元的军事产品和服务的预算。

在加入美国政府之前，布朗曾担任两家硅谷上市技术公司赛门铁克（Symantec）和昆腾公司（Quantum）的首席执行官。

在最近接受政府事务主持人弗朗西斯·罗斯（Francis Rose）的采访中，布朗论证了使用商业解决方案可以节省纳税人的资金。

布朗说：“在接下来的几年中，国防部的预算将越来越紧张，这对于我们利用商业技术来说确实非常重要。我认为这很关键，以便我们可以节省下来资金开发那些需要国防部自行开发的技术（如高超音速或定向能量）。”

布朗主张五角大楼更快地采用商业领域的创新。DIU的目标之一就是不让相关的技术失去活力而陷入所谓的“死亡谷”，因为五角大楼可能需要花费数年的时间来资助一项计划和签订合同。

空军前高级收购执行官威尔·罗珀（Will Roper）对选择布朗作为国防部采购部门的负责人表示赞赏。

罗珀在社交媒体上写道：“对于所有热衷于帮助我们的军队在全球技术新时代竞争的人来说，这是个极好的消息。”

拜登周五还提名了前国家安全局官员罗纳德·莫尔特里（Ronald Moultrie）为负责情报和安全的国防部副部长。莫尔特里目前是Oceanus Security Strategies的总裁兼首席执行官。

在奥巴马政府任职的前五角大楼审计长麦考德（Michael McCord）被再次任命担任国防部的审计长，即五角大楼的首席财务官。

国防部长劳埃德·奥斯丁（Lloyd Austin）认可了提名。他在一份声明中说：“这些人中的每一个都有才干、经验丰富，并且具备重要的国家安全职责，如果得到确认，他们将代表国防部承担这些重要职责。我期待着与参议院一起加速对这些人选的确定。”
