04月11日，韩联社援引韩国和美国情报机构的话写道，朝鲜准备在近期部署一艘3000吨级的新型核潜艇。

“韩国和美国情报机构都认为，朝鲜已经完成了2019年7月亮相的3000吨级潜艇的建造。”机构消息人士说。

该机构称，朝鲜当局正在等待合适的时机，以最大限度地揭开潜艇的面纱，包括向美国施压。

韩联社指出，该潜艇将能携带“普光-3”等3枚潜射弹道导弹。

4月早些时候，美国门户网站“北方38号”报道称，在朝鲜新浦造船厂发现了一些动静，这可能表明朝鲜的新型潜艇可能会在近期完工并交付使用。

今年1月，平壤举行了朝鲜劳动党第八次代表大会结束的阅兵式，并在阅兵式上公布了潜射弹道导弹。朝鲜中央通讯社称其为“世界上最强大的武器”。路透社援引朝鲜国家媒体公布的照片指出，这可能是2020年10月展示的“普光-4”的升级版“普光-5”导弹。
