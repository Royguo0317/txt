据National File 2月16日报道，美国总统乔·拜登在CNN举办的总统市政报告活动中公开表示，他拒绝谴责对香港人民的压迫、强制结束中台“一个中国政策”以及对少数民族维吾尔穆斯林的种族灭绝，声称“在文化上，每个国家及其领导人都遵循不同的规范。”

拜登还觉得有必要阐述中国共产党主席习近平的“中心原则”。拜登说：“习近平的核心原则是，必须有一个统一、严格控制的中国，这为他所做的事情提供了基础。”“我向他指出，如果美国总统不反映美国的价值观，他就无法维持总统身份。”

许多美国人对拜登这番言论感到震惊，但National File认为事态发展不足为奇，因为特朗普总统在 2020 年总统竞选期间揭露了拜登与中共共产主义政权的关系。

10月，National File报道称，亨特·拜登的前商业伙伴托尼·博布林斯基在记录中将乔·拜登确定为“大人物”，该电子邮件记录了亨特中国能源公司CEFC的可疑交易。

11月，正如National File报道的那样，在全美对2020年美国大选中普遍存在的选民欺诈感到沮丧的时候，中共国大外宣《环球时报》在一篇专栏文章中向“即将上任的拜登政府”发布了命令。

拜登对中共国的做法与特朗普总统截然不同，特朗普总统将对抗中国共产党的影响确定为政府的最高优先事项之一，在特朗普政府在任的最后一天，美国国务院正式宣布中共对维吾尔人实施种族灭绝。

拜登政府自上任以后一直在找各种机会帮助中共推翻特朗普政府指定的一系列不利于中共的政策，其中也包括对中共种族灭绝的认定。
