## 【大象快讯】特斯拉首席执行官在公开评论中拒绝了监管机构要求克制的要求
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1292716/)

多年来，美国证券交易委员会一直试图让埃隆·马斯克表现得更像一个典型的企业领导者，但是没有奏效。美国证券交易委员会官员在 2019 年和 2020 年写信给特斯拉，称首席执行官埃隆马斯克的一些推文应该经过 2018 年和解协议所需的监督。

美国证券交易委员会在2018年起诉马斯克先生和特斯拉公司 TSLA，监管机构在推特上认定其存在欺诈行为，以总计 4000 万美元的价格和解，并改变了公司治理制度。作为交易的一部分，它还制定了一种独特的方式来限制马斯克先生的社交媒体习惯：迫使公司预先批准其首席执行官可能推动市场的任何声明。

《华尔街日报》周二报道称，美国证券交易委员会官员在 2019 年和 2020 年两次给特斯拉写信，称马斯克的一些推文应该经过必要的监督。特斯拉不同意，告诉美国证券交易委员会，他的信息不在协议范围内。

金融执法方面的沟通监督政策尚无先例，尽管监管机构经常敦促公司在丑闻或失误之后提高合规性。限制一个傲慢的 CEO 与公众和投资者沟通的方式只会导致与马斯克和特斯拉的更多争执，他们似乎了解他们拥有的影响力。

SEC还开始监管别人在社交媒体上的言论了，投资人的钱被无故冻结，你不去调查去监管，那可是SEC的本职工作范围。那么多财务造假在美国上市的中概股不去查，天天管人家一个火星人要做这做那的，还管人家说什么，滑稽不滑稽，可笑不可笑。
