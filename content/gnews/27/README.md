## 從香港銅鑼灣書店到紅色基因版圖的淫亂穢史
`2020-08-25 07:14 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/315984/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/25070920/Screen-Shot-2020-08-25-at-9.09.01-pm.png)圖片來源：Lam Yik Fei/Getty Images 
2015年之前，銅鑼灣書店是一間其貌不揚的破落書店，位於香港駱克道531號2樓，面積約30平方米。這家書店由香港人林榮基在1994年創辦，除了普通文藝歷史書籍之外，還出售不少在內地無法出版發售的政治禁書，很多到香港的大陸游客都會專程到此購買政治類書籍。

2014年，林榮基將書店賣給巨流傳媒，他擔任店長。公司由桂民海、李波、蔡嘉蘋三人持股。 2015年10月起，書店的股東及店長、業務經理等4人，先後在外地及內地失踪，包括東主桂民海。第5名失踪的書店經營者之一李波，同年12月30日於香港境內失踪。

銅鑼灣書店事件可謂是最早的「送中事件」，種種跡象顯示，來自中共的國安人員直接在香港及海外抓人，暢通無阻。

2016年1月2日，當時的政務司司長林鄭月娥表示會展開調查，但現在看來就是一個笑話。

據路德社透露，鬧得沸沸揚揚的那本書《習近平和他的6個女人》，坊間傳聞是銅鑼灣書店出版此書遭到拘捕，原來習近平在2015年10月才知道自己的風流韻事全部曝光，甚至將出版成書，這才龍顏大怒，將香港銅鑼灣書店5人全部緝拿歸案，結果是捕風捉影的大冤案。 
這件事要從中南海大總管令計劃說起。

2012年3月18日凌晨，令計劃的兒子令谷攜兩名女伴駕駛名貴法拉利跑車，在北京保福寺橋東輔道時發生嚴重車禍，令谷當場死亡，兩名女子身受重傷被送入醫院搶救。

車禍的內情至今未解密，外界看到中南海發生巨變，令計劃迅即落馬，團派遭到滅頂之災。

2015年3月，其弟令完成出逃美國。據悉，令完成出逃時帶走大量中共高度機密文件，其中包括被命名為“建築藝術項目”的基因版圖高度機密名冊。

路德社表示，令計劃當年暗中動用人力物力，將習近平的成長生活軌跡，從梁家河公社，到福建省到主政浙滬，查得一清二楚，鉅細無遺，包括每晚和哪個女人睡覺，是否懷孕生子，全部掌握登記造冊。

這本名冊除了令完成擁有，郭文貴先生亦有，這才令盜國集團慌了神，不惜動用一國之力通緝郭文貴先生。沒想到三年過去了，中共不但一無所獲，反而被郭先生將計就計，以千億美元財富與情報為餌，令中共的假、醜、惡在全世界面前暴露無遺。

很快，世人將會徹底看清中共高層的淫亂穢史：共產共妻，盜取中國14億老百姓的血汗錢，用來穩固他們的千秋萬代，不但奴役14億中國老百姓，還將邪惡的紅色基因散佈在世界每個角落，企圖操控全世界。

爆料革命正在揭開中共盜國賊家族的遮羞紅布，扒出他們隱藏在世界各地的百子千孫，索回他們盜走的幾十萬億美元資產，終結邪惡的中共政權，讓新中國聯邦的旗幟在中華大地飄揚，讓中國老百姓過上有民主、法治、自由，受人尊敬、有信仰的新生活。
