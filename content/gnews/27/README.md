**1、纳瓦罗：“小人”福奇在白宫椭圆形办公室面对川普总统满口谎言**

彼得·纳瓦罗（Peter Navarro）说，在周六的《战斗室》节目，川普总统应该揭露“小人”福奇在白宫是如何对他撒谎的。

谈到川普即将在北卡罗来纳州的演讲时，纳瓦罗说“老板要追究福奇”。

“我认为老板需要说一下，那个小人在疫情危机期间进出白宫100次，他都没有一次正视着过我的眼睛说，‘我认为它（病毒）来自武汉实验室，它是基因改造产品’。”

纳瓦罗说：“如果政府当时知道该病毒的真正来源，它将改变我的一切。因为我们会有一个完全不同的（应对）方式。”

纳瓦罗还抨击了克里斯·华莱士（Chris Wallace），称华莱士是“邪恶的”，因为华莱士声称福奇是政治攻击的受害者，并虚假声称福奇的电子邮件中没有确凿证据。

【备注：克里斯托弗·华莱士（生于1947年10月12日）是美国记者，也是福克斯新闻节目福克斯周日新闻的电视新闻主播，华莱士在福克斯新闻为福奇博士辩护。】

纳瓦罗说，早在2020年1月31日政府资助的科学家克里斯蒂安·安德森（Kristian Andersen）在与福奇的一封电子邮件中，就证明他们知道中共病毒有可能是“被设计的”。

“这是前所未闻的重大事件”,纳瓦罗说：“这个家伙[福奇]使用美国纳税人的钱（资助武毒所病毒功能增强型研究）……结果造成了一场大流行，然后他向世界隐瞒了这场大流行。他不配做一个公务员”。

# **2、揭秘：中共军队如何利用美国的“有用白痴们”开发冠状病毒生物武器**

中共病毒主题专家、美国陆军预备役上校劳伦斯·塞林（Lawrence Sellin）博士说，在中共病毒的研发和掩盖其来源方面，美国的科学精英们“是有罪的”。

塞林周六告诉《战斗室》，看到这种病毒，“立刻就知道它不是来自大自然”，而是由中共军方设计的。

塞林说，像福奇（Fauci）博士和弗朗西斯·柯林斯（Francis Collins）这样的人，要为帮助中国军队在全世界部署生物武器承担责任。

塞林博士说：“他们都是有罪的，从一开始他们就在掩盖病毒的起源。”

塞林博士说：“他们押上了工作、金钱和学术声望……他们故意压制关于新冠病毒实验室来源的任何讨论。”

塞林博士补充说，他们掩盖和资助中共病毒将面临可能的刑事指控，柯林斯就是中共的“有用白痴”。

【备注：有用的白痴（useful idiot）是一个[政治](https://zh.wikipedia.org/wiki/%E6%94%BF%E6%B2%BB)术语，是贬义词，指的是为某一事业进行宣传而不完全清楚这一事业的目标的人。】

塞林在谈到中共时说：“他们创造了一种生物武器，被传播到世界各地。他们已经知道我们对此事的反应，对于他们来说不是一件好事。”
