**西班牙2021年6月10日电/西喜社——**
![]()![](https://gnews.org/wp-content/uploads/2021/05/image0-2-28.jpg)
根据一组前川普政府官员的报告，中共国政府正继续通过隐瞒与冠状病毒大流行有关的细节和信息，来阻碍国际社会。

作为回应，这些前官员敦促对中共国实施制裁，并表示美国国会应通过立法，惩罚北京对**COVID-19**疫情的不当处理。该疫情已造成**300**多万人死亡，并使全球经济遭受破坏。

本周，这位前官员在智库哈德逊研究所发表的一份报告中说：**“**无论人们相信**COVID-19**源自人畜共患的宿主、蝙蝠洞、冷冻食品链，还是武汉实验室未申报的生物武器项目的危险**’**双重用途**’**研究，世界都需要中国共产党出来回答。**”**

这份题为《对北京滥用**COVID-19**的公正回应》的报告，敦促拜登政府停止对危险研究的所有资助，如使病毒对人类更具传染性的**“**功能增强**”**工作。

前川普政府的官员们还建议拜登政府，解决北京违反国际条约的问题，包括要求与条约成员国协商和共享数据的《国际卫生条例》。

报告说：**“**北京的军事项目需要被核实为符合违反《生物武器公约》的** ‘**和平目的**’**条款。

此外，该报告呼吁美国国务卿和财政部长对武汉病毒研究所实施制裁，许多人认为该研究所是大流行病的潜在源头。报告呼吁加大对未来大流行病的准备工作，并停止对生态健康联盟的所有资助，该非政府组织获得了超过**1**亿美元的联邦资金，但批评者称其未能完成预测大流行病的任务。

报告说：**“**政府应该采取更好的战略，例如建立公私合作关系，以开发和部署一个生物威胁探测传感器的全球网络。**”**

报告说：**“**中共国将有望成为这个国际生物传感网络的创始样本国，因为这个倡议的目的是共享数据，类似于天气预报和地震学的联合探测、预测和协调。**”“**至少我们会从这个流行病学的**’**开放天空**’**倡议中，得到自然或人造疾病爆发的更早预警的福利。**”**

这份报告是由前国务院官员余茂春**(Miles Yu)**、戴维**·**费斯（**David Feith)**和托马斯**·**迪南诺**(****Thomas DiNanno****)**，以及戴维**·**阿舍尔（**David Asher**）共同编写。戴维**·**阿舍尔是一名武器专家，在川普政府的最后几个月参与了由国务院领导的对冠状病毒大流行的起源调查。马修**·**茨威格**(Matthew Zweig)**是一名负责叙利亚问题的国家官员，现在是保护民主基金会的高级研究员，他也是这项研究的参与者。

在川普执政期间，余茂春先生是国务院负责中国事务的政策规划官员。迪南诺先生负责的是军备控制合规局，他在寻求确定大流行病的来源方面最为积极。费斯先生曾在国务院的东亚和太平洋事务局，担任中国问题专家。拜登政府在**1**月上任后结束了这项调查。

前川普政府的官员们还建议国会成立一个独立的专家委员会，来研究病毒的起源。

报告说：**“**作为对**[**世卫组织**]**的制衡力量，该委员会可以对**COVID-19**的起源进行客观分析，包括可能的自然人畜共患病和实验室相关的起源**”**。**“**为了强调（以避免）美国未来的漏洞，它还可以包括对中共国与病毒有关的功能增强研究活动，还有生物武器项目的全面来源分析，包括机密研发、军事战略和政策，以及针对合成生物学和病毒**[**生物武器**]**的能力。**”**

国务院最新的年度武器合规报告指出，中共国从事的活动，引起了人们对可能违反了进攻性生物武器全球禁令的担忧。

报告中建议的其他国会行动，包括加强对** “**军民两用的滥用** ”**的限制**—**利用民用研究作为军事武器开发的掩护。

中共国正在进行一项被美国官员称为，军民**“**融合**”**的重大技术采购计划，旨在帮助中国人民解放军大规模发展核武器和常规武器，以及中共国的经济。

报告补充说，国会应该审查美国的刑法，包括《战争罪法》，以寻求加强对**“**灰色地带生物战**”**禁令的方法。

报告说，立法还应该包括更强有力的制裁。例如，修订《化学和生物武器控制和战争消除法》，以制裁那些**“**在维护其生物设施方面有严重或重大过失，或在应对重大事件时候故意隐瞒信息**”**的人、公司和政府。

该报告的作者表示，第一个被制裁的机构，应该是与中共国军方有联系的武汉病毒研究所，还有中国科学院。

此外，还敦促拜登政府支持对中共国提起民事诉讼。如果中共国政府未能在规定的时间内提供与该大流行病有关的充分合作，包括在特定时间内直接获得实物证据。

报告说：**“**国会可以授权民事诉讼当事人，对那些在研制和传播**COVID-19**病毒方面，有严重过失或故意传播的中共国个人、机构或部门，寻求赔偿。**”**

为了进行这样的行动，国会必须扩大法定的例外情况，以允许联邦法院执行任何针对被发现的**——**对**COVID-19**大流行病负有直接或间接责任的中共国国民或机构的判决。

报告总结说：**“**现在有机会通过两党、两院的倡议，建立一个**21**世纪的框架，以保护美国和国际伙伴免受另一个毁灭性大流行病的威胁。**”“**当我们进入一个可能是人工合成生物冒险主义和潜在生物战的世纪时，我们不能再承受中共政府的逍遥法外和美国政府的被动。**”**
