## 共和党试图阻止经济刺激支票流向囚犯和非法移民，遭民主党否决

据《每日电讯》瑞安·萨维德拉（Ryan Saavedra）报道：

![]()![](https://gnews.org/wp-content/uploads/2021/03/Screen-Shot-2021-03-11-at-15.57.20.png)*拜登 – 图片来源：Shaw Thew/EPA/Bloomberg via Getty Images*

共和党参议员汤姆·科顿（AR），特德·克鲁兹（TX）和比尔·卡西迪（LA）在参议院会议上提出了一项修正案，阻止囚犯能够获得刺激经济支票，该修正案以49-50的投票结果被阻止。

福克斯新闻报道，”该法案包括为收入低于7.5万美元的个人提供1400美元的经济刺激支票”，”囚犯也包括在领取刺激性支票的人群中，就像之前科维德的两个救济法案中提供1200美元和600美元支票一样。”

科顿在回应时强调了一些令人发指的被关在美国司法系统中的罪犯，他们将在拜登的经济刺激协议中获得资金。

“迪伦・鲁夫谋杀了九个人，他被关在联邦死囚牢房。作为民主党’COVID救济’法案的一部分，他将获得1400美元的经济刺激支票”。科顿在推特上写道：”Dzhokhar Tsarnaev，波士顿炸弹客，谋杀了三个人并恐吓了一个城市。作为民主党人’COVID救济’法案的一部分，他将获得1400美元的经济刺激支票。”

科顿补充道，”亚伦-沙莫因向毫无戒心的买家出售’100万颗掺有芬太尼的假催情药’而被判处终身监禁”。”他在狱中会从民主党人的’COVID救济’法案中获得1400美元的刺激经济支票。”

三位参议员就民主党人阻挠他们的修正案发表了以下声明。

- 卡西迪：“囚犯不交税。纳税人为他们的一切需求付费。囚犯不能刺激经济。但是，根据这个法案，他们可以收到刺激经济的支票。这是一个非针对性的、不恰当的、完全浪费开支的完美例子。荒谬的是，这是法案中包含的。”
- 科顿：”COVID救济资金应该给受到创伤、 守法的工人，而不是囚犯服务于社会的债务。不幸的是，民主党人宁可把20亿美元花在联邦和州的囚犯身上，而不是花在美国家庭上面。”
- 克鲁兹：”民主党的法案有多么不负责任，最明显的一个例子就是，这个法案优先向被监禁的重刑犯发放1400美元的救济金–这些人并没有因为COVID而失去收入。这个法案中不合理的倾斜对美国是有害的，我很自豪地和我的同事们一起努力弥补这个令人震惊的漏洞。”


据《国会山报》报道，参议院还以49-50票反对克鲁兹提出的一项修正案，该修正案将阻止”1400美元的经济刺激支票发放给无证移民”。

“八名民主党人在上个月初关于参议院预算决议的辩论中，投票通过了由参议员托德·扬（Todd Young）（R.Ind。）和汤姆·科顿（Tom Cotton）（R-Ark。）提出的类似修正案。”报告补充说“但是，民主党核心小组统一的在周六击败共和党修正案。”

参议员迪克-德宾(Dick Durbin)声称，非法移民不会收到刺激经济的支票，因为他们”没有社会安全号码”，因此，”他们没有资格获得刺激经济的救济支票”。

克鲁兹和德宾在参议院会场上发生了争执，德宾指责克鲁兹撒谎。克鲁兹问杜宾是否会让步，杜宾拒绝接受。

一位共和党参议院高层人士告诉《每日电讯》，杜宾错了，因为据该人士透露，民主党的法案将给那些持批准的工作签证合法来美的移民提供经济刺激支票，然后他们的签证超期，这样他们就成了非法移民。然而，当他们在美国合法工作时，他们可能会收到一个社会安全号码。社会保障局在他们的网站上指出，非公民是可以申请社会安全号的。
