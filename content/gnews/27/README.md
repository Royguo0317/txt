# 敦促美国对中共电信巨头中兴的腐败采取行动的报告

[!\[\]()!\[\](https://spark.adobe.com/page/fZit29pN9woJb/images/5969fe9f-e0cb-47ed-8a11-f9cc088474e6.jpg?asset_id=baf0c40f-8436-4dff-8f24-293529a641ed&amp;img_etag=%2207f9ba7e84318f975490f29d4071cc0f%22&amp;size=2560)](https://spark.adobe.com/page/fZit29pN9woJb/images/5969fe9f-e0cb-47ed-8a11-f9cc088474e6.jpg?asset_id=baf0c40f-8436-4dff-8f24-293529a641ed&amp;img_etag=%2207f9ba7e84318f975490f29d4071cc0f%22&amp;size=1024)

档案照片摄于2018年9月26日，北京国家会议参观人员在中共科技公司中兴的logo下走动（美联社Mark Schiefelbein摄）。

中共电信公司中兴在全球范围涉及国际贿赂案件，但是根据对冲基金经理卡尔·巴斯(J. Kyle Bass)被授权进行的报告，目前它逃脱了司法部的起诉。

这份基于法庭文件，检察官的采访以及新闻报告的调查，在2019年9月完成并提交司法部调查官员，该调查指出中兴与许多国家进行腐败勾当。卡尔·巴斯先生的调查員指出中共电信巨头中兴违反了1977年的外国腐败法案(Foreign Corrupt Practices Act ,FCPA)。

该报告是由对中国共产党统治明确谴责的卡尔·巴斯先生领导的团队所做出的。报告称，“中兴在国际商务中进行了系统性地，由中央控制的腐败行为，其规模之大是国际贸易间罕见的。”

该法原本是用于限制美国人的腐败，但是在1998年被修正以覆盖外国在美国股市上市交易的公司，例如总部在中国深圳的中兴，它有一个在德克萨斯的分支。华盛顿时报获取了该26页报告的一份拷贝。

六月，中兴和另一个中共主要科技公司的华为被宣布为美国国家安全威胁，这是出于对他们设备可能用于帮助中共情报部门监视美国的恐惧。

中兴发言人对要求针对这份报告的评论没有邮件或电话回复。

但是，在回应三月份的一次NBC关于美国检察官在调查中兴腐败案件的新闻时，中兴否定了从事非法活动。

该公司说，“中兴全心全意遵守法律规定，并履行义务。” “本公司领导层的最优先任务是确保中兴在全球市场上是可信和可靠的生意伙伴，本公司对目前的成就感到自豪”。

一名司法部发言人拒绝评论。

卡尔·巴斯的报告提供了一个国别详细的犯罪案件列表和涉嫌犯罪活动，牵扯中兴官员在18个国家的活动，主要是非洲国家，以及一些亚洲国家和委内瑞拉。

“中兴利用贿赂和回扣以获得政府合同的行动模式直接伤害了美国利益，破坏了新兴市场的法治和民主，帮助了从最穷的人民手中盗窃（通过大量虚高的合同价格）大量的资源。

[!\[\]()!\[\](https://spark.adobe.com/page/fZit29pN9woJb/images/2a46153e-7eb2-4ee4-b94b-9bc423fad0b9.jpg?asset_id=a11add69-50bc-4cfe-87a1-0c8708431532&amp;img_etag=%22a0ca532e40366907b2705230677a62da%22&amp;size=2560)](https://spark.adobe.com/page/fZit29pN9woJb/images/2a46153e-7eb2-4ee4-b94b-9bc423fad0b9.jpg?asset_id=a11add69-50bc-4cfe-87a1-0c8708431532&amp;img_etag=%22a0ca532e40366907b2705230677a62da%22&amp;size=1024)

接受中兴贿赂的阿尔及利亚官员被定罪

例如，8年前一名中兴行政人员在阿尔及利亚向一名在阿尔及利亚国家通信部门的高级官员支付100万美元贿款而被定罪。

该报告指称，在2014年禁止中兴在阿尔及利活动之后，中兴收购了一个叫“Netras” 的土耳其电信公司，然后通过该公司在北非国家进行活动。

在2018年六月，之前被称为中兴电信设备集团的中兴因违反协议将美国的电信设备售往伊朗，同意支付14亿美元的罚款。作为罚款协议的一部分，中兴被商务部出口管制实体。

该惩罚是在中兴保证惩罚涉及伊朗交易的腐败官员之后，但不但沒逞罰，還是又给奖励。报告指出，该协议要求中兴将4亿美元交给第三方管理，这些钱在中兴不配合腐败调查时可以被没收。

司法部的中共国行动是一项由川普政府发起的计划旨在对抗中共国的非法技术盗窃和间谍活动，这次行动的其中一项宗旨就是由FCPA犯罪的案件中查出與美国竞争的中共公司。

卡尔·巴斯先生说该行动已经起诉了多起案件，但是没有一件违反FCPA案件的中共公司被公诸于世。

他称中兴是中共出售低价制造通信网络以“允许中共政府监听所有在该网络上操作的设备” 的 “招牌”。

卡尔·巴斯先生说川普总统在该公司販賣伊朗裝备交易后，赦免其经济制裁，等于“饶了中兴一命”。

卡尔·巴斯先生说，“川普这么做是为了换取他和中共国的贸易协议，这是一项重大战略失误。” “现在是美国律师把中兴的案子轻松解决的时候了，结果就是把他们一次性全部清除出美国”。
