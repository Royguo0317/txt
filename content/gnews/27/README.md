一、川普总统说《国防授权法案》是给中共和俄罗斯的礼物

12月23日，川普总统周三否决了一项本月通过国会的7400亿美元（6060亿欧元）《国防授权法案》。川普总统反对限制从阿富汗和欧洲撤军，并从军事基地撤除同盟领导人名字的规定。他还希望取消『230条款』[1]。

路德社评论：

1、该法案中有许多细节暗藏各利益集团的利益，川普总统废除该法案一方面得罪了各大利益集团，另一方面他最重要的目的就是希望废除『230条款』；

2、如果不废除『230条款』，中共就能通过操控美国主流媒体限制美国人民言论自由，这严重危险美国国家安全；

3、《国防授权法案》中有一条款『允许乌克兰从美国进口军事武器』，这就是中共埋的雷，路德社在此前节目中爆料乌克兰是东欧重要情报中心，中共以乌克兰为跳板，利用乌克兰虫从欧美进口的武器设备盗取它们的先进技术；

4、《国防授权法案》已连续数十年都顺利通过实施，该法案也获得三分之二国会议员投票通过，但川普总统依然勇敢废除该法案，国会议员收到该法案后只有二十个小时的时间阅读，根本无法详细阅读每项条款，而川普总统有十天时间认真阅读该法案，然后再决定是否签署通过；

5、下周，国会将再次投票否决川普总统的“否决”，只有获得国会三分之二议员的投票支持，这项否决才生效，现在是考验川普总统在国会支持率的时候；

6、班农先生此前也说到，川普总统如果成功否决该法案将有重大意义；

（1）川普总统此前已经获得七千多万美国民众的投票支持，他想借此投票机会测试国会到底支持谁当总统，；

（2）该法案通过需要至少51%支持率，而只要超过三分之一国会议员支持川普总统对该法案的“否决”，那该法案就无法通过，所以川普总统才在此时表态；

（3）如果川普总统在此次国会否决投票中胜利，那他基本上就能夺回佩洛西议长手中的权力，这就迫使民主党坐下与川普总统进行谈判，这就能破除中共借用美国两党制特点进行挑拨离间的诡计，可以看出川普总统团队具有极高的政治智慧。
