## 杨洁篪访问巴尔干两国不受待见 中欧关系已经如履薄冰

#### 翻译：Lynn | 校对：人间四月
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/30101834/%E6%9D%A8.jpg)图片来源：DWNEWS
据《外交家》（The Diplomat）报道，中共最高外交官员杨洁篪在本周早些时候与俄罗斯同行进行高调会晤之后，继续访问了斯洛文尼亚和克罗地亚两国。 尽管杨洁篪对莫斯科的访问引起了人们广泛关注，但外界忽略了他在西巴尔干的停留。 现任中共中央外交事务委员会主任，政治局委员的杨洁篪此前曾担任外交部长，更早前是中国驻美大使。 他被认为是当代中共外交政策的缔造者之一，因而即使他在巴尔干的访问中没有什么里程碑意义的消息被发出来，他的海外访问仍然有一定的象征意义。

但是，到底是什么让杨洁篪去访问斯洛文尼亚和克罗地亚呢？ 这两个国家都是欧盟成员国（分别于2004年和2013年加入欧盟）。 此外，斯洛文尼亚和克罗地亚不仅是中共领导的“ 17 + 1”集团的成员（“ 17 + 1”集团旨在促进中共国与中东欧国家之间的接触），而且该两国在“一带一路”倡议（BRI）的支持下与中共建立了双边机会 。

在过去十年里，北京与卢布尔雅那（斯洛文尼亚首都）和萨格勒布（克罗地亚首都）之间的经济关系愈加紧密且多样化。 而且这两个国家在亚得里亚海沿岸有着得天独厚的战略位置。 他们的港口不但有利于中共商品向欧洲市场的分销，还能吸引大量从中共国国内来的基础设施投资。

克罗地亚的小港口Zadar已经被中共国投资者以多数股权持有，投资者们也有兴趣对将港口与运输网络连接的铁路线进行翻新和现代化改造。 2018年7月，中共国和斯洛文尼亚当局签署了一项协议，以促进中国宁波港与斯洛文尼亚唯一的商业港口Koper之间的合作，这也是“一带一路”倡议的一部分。

中共官员此前的访问曾引起巴尔干同行的热切捧场。 例如，在外交部长王毅于2019年12月访问时，斯洛文尼亚总统博鲁特·帕霍尔就曾表示中共国不仅是重要的经济伙伴，而且是值得信赖的政治伙伴。 那年早些时候，李克强成为首位访问克罗地亚的中共国总理，他形容对那次旅行“充满期待”。 李克强在访问期间，克罗地亚总理安德烈·普伦科维奇宣布了一项重大基础设施项目，作为“克罗地亚与中共国之间友好合作的象征”，为今后合作提供了保证。

然而最近，这些双边合作关系开始模糊。 例如在3月份时，斯洛文尼亚联合其他13个国家和地区（澳大利亚，加拿大，捷克共和国，丹麦，爱沙尼亚，以色列，日本，拉脱维亚，立陶宛，挪威，韩国，英国和美国）共同批评了世界卫生组织对于中共病毒报告的透明度，可读性和及时性。 另外，在2020年8月，斯洛文尼亚签署了一项由美国主导的“ 5G清洁网络安全”非约束性协议，旨在将不受信任的中共电信供应商拒之门外。 这一举动对中共和华为来说无疑是一个微妙的信号。

中共国努力想要将这两个巴尔干国家发展成为中国游客的潜在旅游目的地，但商品贸易和投资仍是双边关系的主要支柱。 例如在2018年，中共国有企业海信集团以3.39亿美元的价格收购了斯洛文尼亚家用电器公司Gorenje 95％的股份，该公司是欧洲该行业最大的制造商之一。 中共国在克罗地亚的业务涉及：著名的电动汽车生产商Rimac Automobili，酒店和风力发电行业，以及一个重要的桥梁建设项目。佩列萨克大桥的项目也很著名，因为它是中共在欧盟成功拿下的第一个中资建设项目，这座桥将会把克罗地亚被波斯尼亚-黑塞哥维分开的两部分土地连接起来。

尽管取得了一些进展，但中共国为“ 17 + 1”区域所勾画的未来并不稳健。在今年二月，一场拖延已久的峰会在中共领导下最终以网路会议形式召开，但这次会议不但没有发出任何统一的消息，反而显示中欧和东欧国家之间的裂痕日益加剧。例如，六个中欧和东欧国家（保加利亚，爱沙尼亚，拉脱维亚，立陶宛，罗马尼亚和斯洛文尼亚）降低了他们参会代表的级别，任命了部长而不是国家元首或政府首脑去参加。 5月下旬，立陶宛宣布将完全退出该组织，并且鼓励其他欧盟成员国效仿。正如立陶宛外交大臣加布里埃留斯·兰德斯贝吉斯（Gabrielius Landsbergis）采访中说：“已经没有什么17 + 1的组织了，出于现实原因，立陶宛已经退出来。”他补充说：“在我们看来，现在是欧盟从分裂的16 + 1格式转变为更加同统一且效率更高的27 + 1的时候了。”

此外，由于本月早些时候欧洲议会决定冻结对中欧投资协议的谈判，中共国可能在寻求在其他方面达成双边合作。 克罗地亚在2020年前六个月担任欧盟理事会轮值主席，而斯洛文尼亚将在2021年下半年轮值。鉴于轮值主席的职能是监督理事会会议，制定议程，开展工作以及促进理事会和其他欧盟机构之间的交流，中共可能希望找一个关系好的欧盟成员国帮忙稳定布鲁塞尔和北京之间的关系。

评论：从欧盟议会冻结中欧投资协定到立陶宛宣布退出“ 17 + 1”，再加上国际上对中共病毒溯源呼声的高涨，五月份的中共已经感受到了多方面实质性的打击。 立陶宛外交大臣加布里埃留斯·兰德斯贝吉斯的话很有意味，他说现在是时候让欧盟成为更加同统一且效率更高的27 + 1的时候了，并且鼓励其他欧盟国效仿。这意味着欧盟将要开始团结起来，而中共会逐渐被抛弃。毕竟与英国签署的中英联合声明，中共可以单方面说撕毁就撕毁，那么中欧投资协定以及“ 17 + 1”所勾画的未来其实都是虚无缥缈的，这些欧洲国家不可能不知道。
