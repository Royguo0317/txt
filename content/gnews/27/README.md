## 我们要建设怎样的社会——理解爱系列之五

**耶稣对众人说，你们要努力进窄门，许多人，我告诉你们，将试图进去，却不能。**

一．信仰

在新时代运动的信仰中，每个生命都是“原一”在探索自己之内无限可能，每个存在都是“神”透过记忆面纱忘记自己真正之所是，投入于此生的游戏中探索宇宙的无限可能。在探索的各个阶段，都在学习和领悟不同层次的“爱”，因为神即是爱。

在二元世界中，爱表现为自爱和互爱，越是偏向自爱一头，就表现得越自私，极度自私的人一般会不择手段追求更大的权柄，这类生命被称为“服务自我者”；越是靠近自爱和互爱的平衡点，越是偏向互爱一头，就越表现出互助互爱，大师都会展现出不一般的大爱。

二．窄门

一个能够尽可能支持所有生命去探索自己的无限可能、展现自己的爱的社会，才是那道地球智慧经历了五个世代依然没有进入的“窄门”。

美国作为地球上最民主的国家，已经很靠近那道“窄门”了，但是由于人性的弱点和整个社会对洞察力的缺失，整个社会仍可能被服务自我者通过各种手段侵蚀和欺骗，从而像亚特兰蒂斯一样沉沦并最终毁灭。46届总统大选的过程就展示了这个可能，带着对那深深恶意的了解的中国人，在川普失势后的失望，是从心底对这个世代失去了希望。幸运地是，仍然抱着希望的中国人们，坚持不懈地一次又一次地提醒着世界人民、美国人民，以事实给全世界补充了缺失的洞察力，让世界得以在最危急的时刻整体转弯。

地球智慧经历了五个世代，依然没有能够整体进入“窄门”，那是因为社会并没有真正理解生命的本质，社会制度并没有真正支持生命去尝试、去探索无限可能，去学习爱。

要知道，每个人、每个生命都是神探索自己无限可能的一个分身，任何忽视了这个本质的社会都进入不了“窄门”。

三、通往“窄门”的路

要想支持生命的无限探索，必不能限制和打击探索的积极性，相反，要尽最大的努力保障每个社会个体探索无限可能的积极性。

当下主流的意识中，存在物质平均主义和机会平均主义两种形态。

物质平均主义很容易辨认，经常提出的口号是均贫富、打土豪分田地、黑命贵等，以劫多济少为主要手段。生命带着记忆面纱进入探索，做着各种选择，必须对自己的选择负起责任，才能够更容易地实现投生时所接受的挑战。然而并不是每个生命都能够坦然面对责任，对自己的选择负责的。外推责任、转嫁责任非常简单，有理无理，把责任推给别人，把别人的成果据为己有就行了。物质平均主义会吸引那些不敢或不想面对责任的人，而必然被服务自我者加以利用，被垄断发展机会，开启养猪模式。劫多济少本身从两个方面限制和打击了生命无限探索的积极性，一是打击了积极探索的人，二是滋养了不劳而获的人，它一开始就是一个内卷的系统，内卷的系统最终必然消亡。

机会均等主义强调给予机会去争取，鼓励成功，它的出现必然经过生命长久的摸索，怀着慈悲的爱心，带着对生命无限可能的理解，以支持生命无限探索为出发点，赋予个体全面的支持，鼓励个体的不断探索无限。它源自于对自己真正所是的明悟，源自于对一切都是“神”探索自己之内无限可能的深刻理解。它契合生命的本质。

四、进入“窄门”

要支持生命无限探索，首先要确保生命在挑战失败之时，仍然有重新站起来的机会；同时让每个成员知道，自己即使失败也仍然能够再次接受挑战。所以一个完善的社会保障体系必须建立。然而，这个保障体系只能保障到支持每个生命有尊严的生存，更高的追求要自己争取。

其次，为了限制服务自我者和防止滋生服务自我者，令机会不被他们窃取和垄断，限制特权的产生，一个全体共同参与、共同监督的民主体系必然要建立。而法权民主才是真正的民主，英美民主的成功的根本即是从《大宪章》开始的法权民主的建立，所以，体系必须以立法、执法等法权民主为基础。

当完成了上面最主要的部分，私产保护是支持生命无限探索的保证，也是探索动力的主要源泉，私产保护必须作为最主要的条文被执行。

围绕美国46届总统大选出现的种种问题，关键在于对“无限探索也包括探索黑暗面”的洞察力的缺失，在于自诩为有爱心的群体无视其他族群的苦难，甚至利用其他族群的苦难而产生的连带后果。地球作为一个整体，必然要推进爱心水平的共同提升。否则，丰裕的生活只会消磨掉整个族群的洞察力，像亚特兰蒂斯一样，最终被受到服务自我者统治的势力侵袭，并最终一同毁灭。

地球智慧经历了五个世代探寻的“窄门”，会因为我们的努力而在这一世代打开吗？
