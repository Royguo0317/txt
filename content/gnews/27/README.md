#### 川普撤销对中共国华为1200亿美元的销售许可

**川普政府已经撤销向中共国电信公司华为出口产品的科技公司的许可证——估计价值千亿美元。**

“川普政府通知包括芯片制造商英特尔在内的华为供应商，（川普政府）正在撤销向华为销售的某些许可证，并打算拒绝其他数十份给华为的供货申请。”路透社首先报道。

此举预计将是针对与中共军方合作、并为中共政权提供后门以访问其数据和设备的中共国电信公司的最后一击。

“在路透社看到的一封记录这些行动的电子邮件中，半导体行业协会（Semiconductor Industry Association）上周五表示，商务部已经发布‘意图拒绝大量对华为出口的许可证请求，并撤销至少一个之前颁发的许可证’。”路透社继续报道。

此举可能危及与华为价值3000亿美元的“货物和技术”贸易：

*一位知情人士表示，在最近的行动之前，约150项价值1200亿美元的货物和技术许可证正等待审批，由于许多美国代理商无法就是否应该授予这些许可证达成一致，这些许可证一直被搁置。该消息人士称，华为还有2800亿美元的货物和技术许可证申请仍未得到处理，但现在更有可能被拒绝。*
