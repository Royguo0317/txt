2月21日，加拿大CBC新闻记者Hannah Thibedeau撰写新闻报道称：随着加拿大加速分发中共疫苗，各省正争相加速实施国民接种中共疫苗的工作；同时加国民众出于对中共病毒及其中共病毒变种的恐惧，期待通过接种疫苗来加强自己的免疫系统对抗中共病毒，焦急问询政府医疗部门“何时轮到自己接种中共病毒疫苗”[1]。

为什么加拿大政府加速推进在全国范围内接种中共病毒疫苗？原因是：第一，加拿大公共卫生部的中共病毒传播模型警、警示，尽管如果不采取有效的公共健康措施，已经传入加国的中共病毒新变种（英国变种B117，南非变种B1351和巴西变种P1）有可能会导致加拿大全国范围内的再次大爆发[2]；第二，加国总人口有3千800万，疫苗接种人数仅仅达到140万，已接种疫苗的人数中也仅有50万人得到辉瑞或莫德纳疫苗二次注射，这种缓慢的进展受到了来自在野保守党的指责和压力，杜鲁多政府声称加速疫苗接种是他当前最优先的事务[3]。与此同时，中共国外交部长王毅于2月17日在北京出席联合国安理会新冠疫苗问题部长级视频公开会表示：疫苗是战胜疫情的有力武器，各国政府、企业和科学家正在共同努力，以创纪录的速度研发和生产新冠疫苗，现在存在“产能赤字”和“分配赤字”，需要积极开展委托生产、联合生产等国际合作，疫苗已经迅速、大量进入高收入国家，现在需要加大加快进入贫穷国家，避免导致“免疫鸿沟”，促进疫苗公平合理分配，不让落下有需要的国家，也不要遗忘任何等待疫苗的人[4]。中共的企图就是模糊化中共病毒的起源，把各国的注意力引导到开发疫苗，经济重启上。在中共疫苗经济的推动下和各国政府不顾中共疫苗存在致命性副作用而盲目推进疫苗的接种以便重振经济，这将导致巨大的人道灾难。中共病毒专家闫丽梦去年以来一再警告说，在中共没有公开中共病毒的基因序列和研究明白中共病毒的机理机制之前，不可能有真正的疫苗，解决中共病毒世界大流行的根本办法就是先要找出中共病毒的来源。最近的研究表明：目前被广泛使用的中共疫苗存在着巨大的风险，一旦被接种，就会成为潜伏在人体内的不定时生物炸弹。

英国政府最近收集到接种辉瑞疫苗后安全数据表明：在英国，每333个接种者中至少有1个发生严重的疫苗引起的副作用，由于人们常常犹豫或羞于报告自己所受到的损伤，这个受害比率可能更大；人们常把疫苗接种当作是100%安全有效的抵抗病毒的办法，不敢对疫苗提出质疑，并且许多医生拒绝接受疫苗伤害是真实存在的；然而疫苗的副作用不是一天内就显现出来，其副作用可能造成一生伤害的，比如心梗，心肌炎，肌肉麻痹，致盲和过敏性休克[6]。疫苗公司跟世界上大多数政府签订了颠覆法律的免责协议，可以免去其疫苗对接种者造成伤害的任何责任，其实他们已经知道这种伤害但是美其名曰“它是不可预见的”;政府管理部门继续低估这些疫苗对人的损害，把这些伤害归于病人的老年化或本身免疫力存在问题，以此来保护那些疫苗公司。以上这种社会病态的医疗欺诈从未像现在这样令人震惊，可怕和简直是种族灭绝[6]。可怕的是，如今疫苗公司成为一个超越法律的行业，它可以不负法律责任，在人们的生活中拥有绝对的权力，并且能够将其疫苗产品的缺陷归咎于他们所伤害的受害者；现在媒体上充斥着冠状病毒疫苗伤害的报道，但是这些伤害被归咎于接受者的免疫系统，而不是疫苗存在的科学缺陷[7]。

以上这些报道表明，目前广泛推广使用的辉瑞和莫德纳等中共疫苗缺乏临床数据来证明其安全性，而大量接种者所受到的疫苗接种后的损害案例警示人们需要认真考虑接种疫苗对个人可能造成伤害甚或失去生命的严重后果。由于疫苗制造公司和各国政府的利益勾结，压制真实的疫苗副作用的报道，不顾人们的生命健康，一味地推行全民接种中共疫苗，掩饰隐瞒疫苗本身存在科学缺陷，草菅人命。另外，中共病毒仍在持续进化之中，不断产生新的变种，而且新变种之间可以互相结合成为更加具有传染性和致命性的新病毒变种，已接种的疫苗是否对这些新变种有效都是未知数。因此，拒绝接种中共疫苗，避免与他人接触，致力于提高自身的免疫能力和服用硫酸羟氯喹预防中共病毒，仍然是一个明智的选择。真正的疫苗还需要等到中共交出中共病毒基因序列之后才会有可能研制出来，因此中共病毒的溯源依然是最重要的课题。
