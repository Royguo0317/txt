- **[05.08 Breitbart：巴西总统博尔索纳罗提出冠状病毒是 “化学战 “的理论](https://www.breitbart.com/latin-america/2021/05/07/brazil-bolsonaro-floats-theory-coronavirus-chemical-warfare/)**


中共国外交部周四谴责了巴西总统博尔索纳罗的言论，他认为 “化学、细菌和放射性战争 “可以解释中国的冠状病毒大流行，似乎在指责北京发动了一场攻击。
博尔索纳罗在周三的讲话中没有提到中共国的名字-当被要求澄清他是否有意指责中国时，他强调，”我没有说’中国’这个词”–但他问道，自这场大流行开始以来，”哪个国家的GDP增长最高”，这个称号以很大的优势属于中国。
博尔索纳罗在2018年的总统竞选中是作为一个坚定的反共主义者，担心中国共产主义在该国日益增长的影响。作为总统，他领导的政府对中共国的友好程度大大超过了他的承诺，为中共国和巴西公司之间有利可图的企业交易铺平了道路，并在最初声称要保护巴西人不成为共产党的 “小白鼠 “之后，允许中共国对疫苗候选人进行临床研究。博尔索纳罗还允许在巴西广泛使用中国制造的候选疫苗，并感谢中共国向该国输送疫苗成分。
这位巴西总统还经常因其对该大流行病的立场而引起争议。博尔索纳罗热情地反对使用国内旅行限制和商业停业来防止病毒的传播，在某些情况下，他鼓励巴西人组织大规模活动来光顾企业。由于巴西在联邦制下运作，博尔索纳罗的反对并没有阻止一些州长在其境内实施封锁限制。
博尔索纳罗也是使用抗疟疾药物羟氯喹治疗冠状病毒感染的积极支持者，在他个人被诊断出冠状病毒后，他经常发布自己服用该药物的图片。在一次离奇的遭遇中，一名记者拍下了博尔索纳罗在一只红嘴鸥面前挥舞着一瓶羟氯喹的照片，红嘴鸥是一种原产于巴西的大型不飞鸟。博尔索纳罗因推广羟氯喹而受到国际谴责，因为他没有医学背景，也没有研究表明该产品可以帮助病人从冠状病毒感染中更快地恢复。
“这是一种新的病毒，没有人知道它是在实验室里诞生的，还是因为人类吃了不合适的动物而诞生的，”博尔索纳罗周三在总统府Planalto的一个活动上对记者评论道。”但它就在这里，军方知道什么是化学、细菌和放射性战争。会不会是我们正面临一场新的战争？”
他又补充了一个问题。”哪个国家的GDP增幅最大？”他补充说，”我不会替你说。” 后来面对澄清他是否指的是中共国，他坚持说：”我没有说’中共国’这个词。”
根据中国共产党的说法，中共国的GDP在2020年增长了2.3%。中国经济是2020年唯一一个记录正增长的国家，这一年的特点是经济急剧下滑，这是因为各国政府为防止中共国冠状病毒的传播而将本国的正常商业活动定为犯罪。
中共国外交部周四对博尔索纳罗的评论作出反应，指责他试图 “污名化 “病毒。
“病毒是人类的共同敌人。当务之急是所有国家携手合作抗击疫情，争取早日彻底战胜疫情，”外交部发言人王文斌在例行新闻发布会上被问及博尔索纳罗的言论时说。”我们坚决反对任何将病毒政治化和污名化的企图。”

- **[05.08 News Max：中共国在非洲的扩张可能会让美国面临来自大西洋的威胁](https://www.newsmax.com/newsfront/united-states-china-africa/2021/05/07/id/1020493/)**


美国负责非洲事务的高级将领警告说，来自中共国的日益增长的威胁可能不仅来自太平洋水域，也来自大西洋。
美国将军斯蒂芬-汤森在接受美联社采访时说，北京正在寻求在非洲西海岸建立一个能够容纳潜艇或航空母舰的大型海军港口。汤森说，中共国已经与从毛里塔尼亚到纳米比亚南部的国家接触，打算建立一个海军设施。如果实现，这一前景将使中共国能够在大西洋和太平洋上为其不断扩大的海军建立军舰基地。
“他们正在寻找一个可以重新武装和修理军舰的地方。负责美国非洲司令部的汤森说：”这在冲突中变得有军事意义。”他们在吉布提建立这一点已经走了很远。现在他们把目光投向大西洋沿岸，想在那里建立这样一个基地。”
汤森的警告是在五角大楼将其重点从过去20年的反恐战争转移到印度-太平洋地区和来自中共国和俄罗斯等大国对手的威胁时发出的。拜登政府认为，中共国迅速扩大的经济影响力和军事力量是美国主要的长期安全挑战。
全球各地的美国军事指挥官，包括一些可能失去部队和资源以支持太平洋地区发展的指挥官，警告说中共国日益增强的自信并不仅仅发生在亚洲。他们认为，北京正在积极地对非洲、南美和中东国家施加经济影响，并在那里寻求基地和立足点。
汤森说：”中共在非洲的一些国家正在超越美国，”汤森说。”港口项目、经济努力、基础设施以及他们的协议和合同将导致未来更大的准入。他们正在对冲他们的赌注，在非洲下大赌注。”
中共国的第一个海外海军基地是多年前在非洲之角的吉布提建立的，它正在稳步增加其能力。汤森说，有多达2000名军事人员在该基地，包括数百名在那里处理安全问题的海军陆战队员。
“他们肯定有武器和弹药。他们有装甲战车。我们认为他们很快就会把直升机设在那里，可能包括攻击直升机，”汤森说。
一段时间以来，许多人认为中共国正在努力在坦桑尼亚建立一个海军基地，坦桑尼亚是非洲东海岸的一个国家，与北京有着牢固、长期的军事关系。但汤森说，现在似乎还没有这方面的决定。
他说，虽然中共国一直在努力争取在坦桑尼亚建立基地，但这不是他最关心的地点。
“这是在印度洋那边，”他说。”我希望它在坦桑尼亚而不是在大西洋沿岸。大西洋海岸让我非常担心，”他说，指出从非洲西海岸到美国的距离相对较短。以海里为单位，非洲北部大西洋海岸的基地可能比中共国的军事设施离美国西海岸更近。
更具体地说，中共国一直在关注几内亚湾的港口位置。
美国国防部关于中共国军事力量的2020年报告称，中共国很可能考虑在安哥拉等地增加军事设施，以支持其海军、空军和地面部队。报告还指出，从非洲和中东地区进口的大量石油和液化天然气，使这些地区成为中国未来15年的高度优先事项。
美国和平研究所的高级政策分析师亨利-图根哈特说，中共国在非洲西海岸有很多经济利益，包括渔业和石油。中共国还帮助资助并在喀麦隆建造了一个大型商业港口。
他说，北京在大西洋沿岸获得一个海军港口的任何努力都将是中国军事存在的扩张。但他说，对海洋通道的渴望可能主要是为了经济利益，而不是军事能力。
汤森和其他地区军事指挥官在最近的国会听证会上阐述了他们对中国的担忧。他与美国南方司令部负责人克雷格-法勒上将和美国中央司令部负责人弗兰克-麦肯锡将军一起，在五角大楼继续审查向大国竞争的转变时，正在为保留其军事力量、飞机和监视资产而斗争。
国防部长劳埃德-奥斯汀正在进行一项全球态势审查，以确定美国的军事力量是否被安置在它需要的地方，并以正确的数量在世界各地，以最好地保持全球主导地位。这项审查预计将在夏末完成。

- **[05.07 NY Times：布林肯在联合国会议上与中共国和俄罗斯较量](https://www.nytimes.com/2021/05/07/world/americas/blinken-china-russia-in-united-nations.html)**

拜登总统的高级外交官说，美国将维护国际规则，并对那些不遵守国际规则的人进行 “强力反击”，这与特朗普时代形成鲜明对比。
美国国务卿安东尼-布林肯(Antony J. Blinken)周五在会见中国和俄罗斯的同行时说，美国将对违反国际规则的人进行 “有力的反击”，即使他承认自己的国家在特朗普政府的领导下违反了规则。
布林肯先生的同行、中共国外交部长王毅和俄罗斯外交部长谢尔盖-拉夫罗夫对美国进行了自己的外交抨击，指责其虚伪，并以旨在维护西方世界主导地位的术语来定义国际规则。
这些交流是在联合国安理会的一次会议上进行的，该会议由中共国召集并通过视频会议链接虚拟举行，主题是针对大流行病、全球变暖和其他共同威胁的多边合作。
这在某种程度上是布林肯先生和王先生之间的一次重演，王先生是两个月前在阿拉斯加会议上粗暴地教训美国的中共国高级代表团的成员。那场没有剧本的对抗在中国被视为英雄，中共国政府在那里煽动了不断上升的反美主义和民族主义。
尽管周五的会议所使用的术语和语气更为外交化，但布林肯先生和他的同行们所支持的世界观存在着明显的差异。这些分歧表明，安全理事会大国之间的僵局不会很快缓解。
会议召开的同一周，布林肯先生在英国会见了七国集团外长，强调了他所说的 “捍卫民主价值和开放社会 “的重要性–这是拜登政府打算在人权、虚假信息和其他问题上挑战中国和俄罗斯的一个信号，而唐纳德-J-特朗普总统的政府已经不再强调或忽略了这些问题。
在拜登政府发出的另一个明确信号中，布林肯先生还访问了乌克兰，他承诺支持乌克兰打击俄罗斯支持的叛乱活动，该活动自2014年以来已造成13000人死亡。
布林肯先生在安理会的发言中声称，联合国仍然是世界上一支至关重要的正义力量，自其在二战结束时成立以来，对现代历史上最和平和最繁荣的时代负有责任，但现在正受到严重威胁。
“布林肯先生说：”民族主义卷土重来，镇压不断加剧，国家间的竞争日益加深–对基于规则的秩序的攻击也在加剧。”有些人质疑多边合作是否仍有可能。美国认为它不仅是可能的，而且是必须的。”
布林肯先生说，美国将与任何国家合作应对冠状病毒和气候变化带来的全球威胁，”包括与我们有严重分歧的国家”。
同时，他说，作为对中共国和俄罗斯的明确警告，美国将 “在我们看到一些国家破坏国际秩序，假装我们都同意的规则不存在，或者干脆随意违反这些规则的时候，进行有力的反击。”
他没有提出任何新的立场，但显然试图强调拜登政府致力于扭转唐纳德-J-特朗普总统的外交政策遗产，后者经常贬低联合国，并带领美国走上批评者所称的破坏性的单边道路。
“我知道，我们近年来的一些行动破坏了基于规则的秩序，并导致其他人质疑我们是否仍然致力于此，”布林肯先生说。”我们要求世界通过我们的行动来判断我们的承诺，而不是听信我们的话”。
他列举了拜登政府如何重新加入巴黎气候协定，阻止特朗普先生退出世界卫生组织，并正在寻求重新加入联合国人权理事会。
“我们也在采取步骤，非常谦虚地解决我们自己民主中的不公平和不公正，”他说。”我们公开和透明地这样做，让全世界的人看到。即使它是丑陋的。即使它是痛苦的。”
王先生的国家在5月份担任安理会轮值主席国，他试图将中国描述为一个负责任的全球公民，遵守国际法。他没有提到美国的名字，但责备那些他认为将国际规则定义为 “少数人的专利或特权 “的国家。
他还宣称，”任何国家都不应该期望其他国家输掉比赛”，这反映了中国对美国试图压制中国崛起的指责–布林肯先生和其他人否认了这一指责。
拉夫罗夫先生对美国及其盟友的批评更为直接，他将布林肯先生提到的 “基于规则的秩序 “描述为西方镇压其他国家的幌子。
他特别批评了美国和欧盟对俄罗斯和其他他们不同意的国家实施的经济制裁，拉夫罗夫先生说，这些制裁的目的是 “把对手赶出游戏”。
- **[05.07 Fox News: 武汉 “实验室泄漏 “冠状病毒理论成为焦点，众议院共和党人要求答案](https://www.foxnews.com/politics/wuhan-lab-leak-coronavirus-theory-in-focus-as-house-republicans-demand-answers)**


国会共和党人正在领导对COVID-19大流行病的起源进行 “全面 “调查，包括武汉一家实验室 “意外 “泄漏的可能性。
众议院能源和商业委员会的最高共和党人，华盛顿州众议员凯西-麦克莫里斯-罗杰斯，本周写信给国务卿安东尼-布林肯，要求提供文件以 “协助 “他们调查COVID-19的起源。
霍利-布伦法案将迫使拜登政府取消对武汉冠状病毒泄漏情报的保密。
“作为对公共卫生有管辖权的国会委员会的领导人，我们强烈支持对COVID-19大流行病的起源进行全面调查，包括实验室意外泄漏的可能性。麦克莫里斯与众议员布雷特-格思里和H-摩根-格里菲斯一起写道。
共和党人要求布林肯发布 “与该部2021年1月15日的概况介绍中断言中共国武汉的武汉病毒研究所（WIV）与中共国军方合作进行机密研究，包括实验室动物实验有关的非机密文件，并酌情解密其他文件供公开发布。”
这份在特朗普政府期间发布的概况介绍声称，WIV与中共国军方合作开展 “秘密项目”，并声称 “尽管WIV以民间机构自居，但美国已经确定，WIV自 “至少在2017年以来 “在中共国军方的出版物和秘密项目上进行了合作。”
共和党人指出，WIV “一直是 “美国政府和世界卫生组织在审查COVID-19的起源时的主要焦点。
“我认为中国知道，在COVID的早期阶段，它没有做它需要做的事情，那就是，实时地让国际专家进入，实时地分享信息，实时地提供真正的透明度，”布林肯说。
同时，在另一项调查中，威斯康星州众议员迈克-加拉格尔（Mike Gallagher）给国家过敏和传染病研究所（NIAID）所长福奇博士写了一封信，要求提供有关该大流行病 “原因 “的进一步信息。
“加拉格尔在周三给福奇的信中写道：”了解这次大流行的原因–并确保类似的事情不再发生–是当今世界面临的最重要问题。”鉴于事关重大，我们不能满足于对这一可怕疾病的起源的有限、盲目或政治化的理解。
而在国会的另一边，共和党的参议员乔什-霍利和迈克-布劳恩也发表了自己的看法。乔什-霍利和迈克-布劳恩提出一项法案，将迫使拜登政府解密与COVID-19起源有关的情报。
“一年多来，任何询问武汉病毒研究所的人都被打成阴谋论者，”霍利在上个月的一份声明中说。”世界需要知道这次大流行是否是武汉实验室疏忽的产物，但中共已尽其所能阻止可信的调查。”
他补充说，”拜登政府必须解密它所知道的关于武汉实验室和北京试图掩盖该大流行病起源的情况。”
白宫新闻秘书Jen Psaki在上个月呼吁中国和世卫组织允许国际专家 “不受限制地获取 “数据，并允许他们向疫情爆发时在当地的人提问。普萨基说，美国医学专家仍在审查这份报告，但白宫认为它 “不符合时机”。
