**简评：**

这是一篇来自中共国独角兽公司字节跳动内部员工的独白，让外人看到中共国的社交平台是如何完美地完成政府监控任务的。可是员工也是平台用户，当监控帮凶自己变成了监控对象时，可以想象当事人的愤懑、内疚、压抑、无奈。

社交媒体是以原创内容为生的开放平台，人人都可以参与，人人都可以互粉，但也要求人人都认同既定的规则。违规内容不能大行其道，如同在现实世界，也是有规矩的。内容审核是社交媒体的自我管理，如同行业规范，防止自由权利被滥用，这本是理所当然的事。

但是在如今的中共国，监控无所不在，开放的万维网变成了封闭的区域网，欢快的社交平台变成了压抑的告密平台。中共啊中共，你在国内愚弄百姓封网禁言，在国外输出监控横行霸道，一个偌大的国家只允许一个声音。你可知道，没有一个个鲜活的个体，又何来人类命运共同体？

# **我帮着制造了字节跳动的审查机器**

我不以此为荣，同事们也不以此为荣，无奈生活在当今中共国。

[!\[\]()!\[\](https://spark.adobe.com/page/s2xn5IoteVhzJ/images/a7c39611-e98f-4b3b-9373-5a0df326b5f2.jpg?asset_id=fbf2068b-1a6f-4bed-9ade-e0700112c82d&amp;img_etag=%228bceefb6cb88b9a6831bf7ac376a1519%22&amp;size=2560)](https://spark.adobe.com/page/s2xn5IoteVhzJ/images/a7c39611-e98f-4b3b-9373-5a0df326b5f2.jpg?asset_id=fbf2068b-1a6f-4bed-9ade-e0700112c82d&amp;img_etag=%228bceefb6cb88b9a6831bf7ac376a1519%22&amp;size=1024)位于北京的字节跳动总部，图片 Emmanuel Wong/Getty Images

这是字节跳动化名李安的前员工向本报记者沈璐讲述的故事。

那个夜晚，李文亮医生在武汉市中心医院急诊室与死神拼搏，我和很多中国网友一样，一直没睡，不断刷新微博，了解他的最新病情。李医生是一位眼科医生，在冠状病毒爆发初期就敲响了警钟，但是很快就受到政府的恐吓，之后又染上了病毒，当他在2020年2月7日星期五凌晨去世时，我和许许多多的中国网友一样，在微博上表达了对事件的悲痛和愤怒，没料到我的账号竟然被删除。

可我感到的内疚多于愤怒，当时我是字节跳动的一名技术员工，开发审核内容的工具和平台。换句话说，是我帮着建的系统审查了我这一类人的账号，自己把自己埋进了中共国不断扩大的网络坟墓里。

[!\[\]()!\[\](https://spark.adobe.com/page/s2xn5IoteVhzJ/images/0062d476-8ce1-48b4-9c1f-c7722beca0c8.png?asset_id=a38b1da7-b5a4-4817-92b6-2839c006934e&amp;img_etag=%22e202448d0622f2bd7a55b83af2b8e137%22&amp;size=2560)](https://spark.adobe.com/page/s2xn5IoteVhzJ/images/0062d476-8ce1-48b4-9c1f-c7722beca0c8.png?asset_id=a38b1da7-b5a4-4817-92b6-2839c006934e&amp;img_etag=%22e202448d0622f2bd7a55b83af2b8e137%22&amp;size=1024)

我没有收到屏蔽李文亮的明确指令，但微博肯定不是当晚无情删除帖子和账号的唯一一家中共国科技公司。我知道字节跳动的内容审核大军正在使用我帮着开发的工具和算法来删除内容，重编故事，篡改冠状病毒大爆发给人们留下的惨痛记忆。每天我都感觉自己是一个小齿轮，运转在一台巨型、邪恶的机器中。

字节跳动是中共国最大的独角兽公司之一，创造了短视频分享应用平台TikTok即“抖音”以及新闻聚合器“今日头条”。字节跳动去年成为美国数据是否与北京共享的争议焦点之后，其国内的工程师不再有权获取包括TikTok在内的海外产品数据，TikTok还计划在洛杉矶和华盛顿建立两个实体“透明中心”展示内容审核的全过程。但在中共国，内容审核大都是暗箱操作。

我在中心技术部工作，支持字节跳动数据部的核心——“信任和安全”部，而数据部主要是为短视频平台开发技术。截至2020年初，我们开发的产品适用于字节跳动海内外所有产品的内容审核系统，包括国内的“抖音”和国际上的同类产品TikTok。产品部有大约50名员工，技术部有100到150名软件工程师。此外，字节跳动还雇佣了约2万名内容审核员，负责监控国内的内容。他们在天津、四川成都、山东济南和其他城市的“基地”工作，有些是字节跳动的正式工，有些是外包工。
[!\[\]()!\[\](https://spark.adobe.com/page/s2xn5IoteVhzJ/images/1ddb0369-24b3-47d7-bac8-b36d792cb464.jpg?asset_id=301d38eb-f15d-4bc0-9fc1-6cfe0adac716&amp;img_etag=%222a57764e423a2fe210bf278c5a2a2893%22&amp;size=2560)](https://spark.adobe.com/page/s2xn5IoteVhzJ/images/1ddb0369-24b3-47d7-bac8-b36d792cb464.jpg?asset_id=301d38eb-f15d-4bc0-9fc1-6cfe0adac716&amp;img_etag=%222a57764e423a2fe210bf278c5a2a2893%22&amp;size=1024)
我的工作是利用技术提高一线内容审核员的工作效率。比如，我们创建了一个工具，可以将一个视频片段直接扔进我们的数据库，搜索类似的内容。

在字节跳动，我们曾收到了多个基地的请求，要求我们开发一种算法：一旦自动检测到“抖音”用户说维吾尔语就切断直播。审核员之所以提出这个要求，是因为他们不会维语。用少数民族语言和方言做直播的人都会收到警告，要求他们改说普通话，因为审核员只会普通话。如果他们不遵守，审核员会以人工切断直播的方式回应，不管实际内容如何。但是当涉及到维吾尔语时，如果有一个算法可以自动辨别并切断，审核员就不必为遗漏当局可能认为是煽动“分裂主义”或“恐怖主义”的内容而负责。我们最终决定不做开发，一是我们的系统没有足够大的维吾尔语数据库，二是最受欢迎的直播间已经被严密监控。

事实上，政治言论只占被删除内容的一小部分，中国网民精通自我审查，知道什么不该说。字节跳动的产品如抖音、今日头条、西瓜视频和火山小视频大多是娱乐应用，我们主要审查中共国政府认为有道德危害的内容，如色情、淫秽、裸体、图片和脏话，以及未经授权的直播销售和侵犯版权的内容。
[!\[\]()!\[\](https://spark.adobe.com/page/s2xn5IoteVhzJ/images/7f1af8e7-98c7-445f-8776-60d02315fb26.jpg?asset_id=83dd8835-07f1-47fe-98b4-0dde1609c79d&amp;img_etag=%22be065df52822549411db27f2a3c64bc4%22&amp;size=2560)](https://spark.adobe.com/page/s2xn5IoteVhzJ/images/7f1af8e7-98c7-445f-8776-60d02315fb26.jpg?asset_id=83dd8835-07f1-47fe-98b4-0dde1609c79d&amp;img_etag=%22be065df52822549411db27f2a3c64bc4%22&amp;size=1024)
但政治言论审查仍然是重头戏，中共国的用户原创内容平台最怕的是未删除政治敏感内容，让公司遭受政府的严查，这可是一个生死攸关的问题。字节跳动的内容审核系统偶尔会瘫痪几分钟，搞得人人紧张，因为我们不知道在那个窗口中会发生什么样的政治灾难。作为一家年轻的独角兽公司，字节跳动并不像其他科技巨头那样拥有强大的政府关系，所以它每一秒钟都在走钢丝。

一个内容审核定制定政策团队，配上2万名内容审核大军，让字节跳动免受重大政治影响，取得了商业上的成功。字节跳动的强大算法可以对用户进行精准的预测并推荐相应的内容，这已是世人皆知。不为人所知的是，此算法还能帮助审核员迅速审查内容。在中共国没有多少科技公司花费巨资专注于内容审查，其他用户原创内容平台没法与字节跳动相提并论。

我的许多同事对我们的工作感到不安，有些人在大学里学过新闻，有些人是顶尖大学的毕业生，都是受过良好教育的自由派。我们时常公开谈论我们的工作是在助纣为虐，可都觉得无能为力。

当然，理想主义的微光还未破灭，也许是我太天真，可我想如果我再努力一点，“把枪口抬高一寸”，就可以让更多的言论偷偷溜走，不过我也知道我的影响力其实是有限的。

在日常审查方面，字节跳动公司的“内容质量中心”负责监督国内内容审查，国家互联网信息办公室（网信办）向该中心频繁颁发指令，有时一天100条之多。之后，中心会责成不同的部门将具体监控指令贯彻到实时言论平台，同时也在以往数据平台上搜索内容并删帖。

在直播节目中，为了便于审查算法运行，每个音频片段会自动转录成文本，与一长串不断更新的敏感词、日期和姓名以及自然语言处理模型比较，再用算法分析内容是否有足够的风险需要单独监控。

如果用户提到了一个敏感词，内容审核员会收到原始视频片段和显示该词汇出现位置的文字记录。如果审核员认为该言论敏感或不恰当，他们会关闭正在进行的直播，甚至暂停或删除该账号。在政治敏感的节日前后，如10月1日（中共国国庆节）、7月1日（中国共产党生日）或重大政治纪念日（如1989年天安门广场抗议和镇压纪念日），内容质量中心会生成特殊的敏感词汇清单，供内容审核员使用。网络大咖享有特殊待遇，有内容审核员专责监控其频道，以防他们的内容或账号被误删。字节跳动另有一份白名单：极受欢迎的网络大咖、国家媒体和政府机构名列其中，他们是属于合规免审查的。

我部门的同事并没有直接接触到内容审核员或者互联网监管员。内容质量中心拿着审核规定，直接与基地经理们对接执行任务。在重大事件或敏感的周年纪念日之后，运营方面的同事会向大家汇报哪些地方行之有效，哪些地方是还需改进。我们会参加此类会议，便于更好地支持审查业务。

我们的作用是确保一线内容审核员能像大海捞针一样，尽快发现“有害和危险的内容”。字节跳动制定完“社群规范”，我们的任务就是提高审查效率，用最少的人力找出最多的违规内容。我不记得我在字节跳动任职期间，受到中共国政府监管部门的政治打击，我们可谓顺利完成了任务。

这当然不是一份我可以向朋友和家人炫耀的工作。当他们问我在字节跳动做什么时，我通常会告诉他们我负责删帖，一些朋友会说：“现在我知道是谁挖走了我的账号了，”我帮着设计的算法也是可以用来打击假新闻，但在中共国，这些技术主要用于审查言论，时不时还会被用来消除网络对重大事件的集体记忆。

李文亮医生警告他的同事和朋友，一种未知的病毒正在侵袭武汉的医院。他因此受到了惩罚。而当局掩盖了危机的严重性，过去几个星期了我们都不知道到底发生了什么。去年这个时候，许多中共国科技公司都在主动删除帖子、视频、日记和图片，因为它们不属于中共国政府批准的“正确集体记忆”。试想一下：如果有任何一个社交媒体平台能够拒绝政府的审查指令，保留李医生和其他举报人的警告，也许今天会有数百万人的生命得到拯救。
