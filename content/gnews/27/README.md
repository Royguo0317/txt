## 【香港要闻】港共政府发薪 林郑领153万现钞回家放 中资银行『爱莫能助』

因为香港实施《国安法》，今年8月,美国财政部对林郑月娥等香港和中国大陆负责香港事务的高官实施了制裁。据【禁闻】11月29日讯，林郑月娥最近在接受香港国际财经台（HKIBC）采访时表示，由于美国对她的制裁让她无法拥有银行账户，因此不得不把工资以现金形式保存在家中，结果导致家中「现金成堆」。

林郑月娥对HKIBC表示，坐在你面前的是一位没有银行账户的香港特别行政区行政长官。林郑月娥还表示，由于没有银行账户，政府只能用现​​金支付她的工资。同时，她还补充说，虽然受到美国政府「不公正的制裁」，但她感到 「非常荣幸」。

战友点评：

林郑这种向媒体诉苦，实在是对中央喊话，言其为党服务，证明自己因忠心而受罚，不知道能否为她的连任加分。

有人调侃，林郑对中央忠心耿耿，为什么中央不出手相助，随便造个假名字，在中资银行给她开个假户口存钱？原因很简单，所有有国际业务的银行都要考虑违反美国金融制裁令可能给自己的国际业务造成的影响。中资银行也不例外。中央是不会为了林郑「铤而走险」的。

《路德时评》2020年11月29日早间节目对此进行了剖析，指出林郑连领取现金薪水也违法。 FDIC(美国联邦银行金融监管机构)规定：全球的金融机构，凡要用美金，就必须遵守美国联邦金融监管机构相应的规定：只要与美金有兑换关系，必须遵守FDIC 规则。

由于港币本身就是一个金融产品和货币产品，而且港币与美金挂钩互换，因此香港必须遵守FDIC 规定。港币是金融产品，而美国的制裁令规定：禁止所有金融机构为林正月娥提供金融服务，那也就意味着港币给林郑月娥提供了服务。

以提款为例子，港府向林郑支付港币，就相当于港币给林郑月娥提供了服务。这个制裁规定所有的金融机构都不能为林郑提供金融服务。因此，港府用现金支付林郑工资就已经是违反美国制裁。

《路德时评》还提出，香港政府支付给林郑的港币必定是从某银行支取，这也违反美国制裁令，也将被美国政府制裁；未来只要美国展开调查，香港金融机构就必须配合，比须交代清楚谁让林正月娥提走工资、谁前往提款，这些人或机构都有可能被制裁。要是林郑不能证明这些钱是她的，那么钱是不是第三方？这些都是在美国制裁下所产生的连锁反应。

在美国制裁下，林郑应该一分钱也领不到，她应该没有现金流才对，现在却『金银满屋』，这证明她违规。林郑可谓求仁得仁，有权也有钱，家里现金之多堪比中共贪官。但这大堆钱就是给美国做抓手。我们且看美国会否就这事追击林郑。
