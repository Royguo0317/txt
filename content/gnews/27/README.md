##  **微軟與被制裁的中共國軍方代理合作，為中共打造操作系統** 

[!\[\](https://spark.adobe.com/page/v9fe8jjE1rxvH/images/be6a038f-52f5-4219-ba50-7afd48bc9673.jpg?asset_id=efcc5e75-9774-4ec1-b207-5179376d7e92&amp;img_etag=%22fd4026bce8c752ba9b4bee5c51fa36c9%22&amp;size=2560)](https://spark.adobe.com/page/v9fe8jjE1rxvH/images/be6a038f-52f5-4219-ba50-7afd48bc9673.jpg?asset_id=efcc5e75-9774-4ec1-b207-5179376d7e92&amp;img_etag=%22fd4026bce8c752ba9b4bee5c51fa36c9%22&amp;size=1024)

**微軟曾作為抖音的潛在購買者，降低人們對該應用程序與中國共產黨關聯的擔憂。微軟還與中共國軍方代理合作，專門為中共國政府創建了一個操作系統。**

微軟收購TikTok並不能消除人們對該應用屈從於中國共產黨的擔憂，還有一長串理由可以說明這一點。 2015年，這家美國科技巨頭為中共國政府開發了一款頗受歡迎的“Windows”操作系統。

**根據該公司的新聞稿：**

**“中國電子科技集團（CETC）與微軟公司宣布了一項合作夥伴關係，旨在為政府機構和關鍵基礎設施國有企業專業領域的中國用戶提供世界一流的操作系統技術和服務。”**

然而，聲明沒有提及，至少從1999年起，美國國防部就認定國有中國電子科技集團（CETC）與中國人民解放軍（PLA）合作。

CETC最近還因在南中國海建造軍事島嶼而被川普政府盯上，美國商務部將該公司列入了“實體名單”。該名稱最初是為了阻止美國產品協助大規模殺傷性武器計劃於1997年設計的，該名稱使公司必須遵守更嚴格的許可要求才能在美國開展業務。

Windows執行副總裁泰利·梅爾森（Terry Myerson）詳細介紹了該公司如何“在Windows 10的安全審查方面與中共國政府認真合作”。

[!\[\](https://spark.adobe.com/page/v9fe8jjE1rxvH/images/93db9d0e-cc11-4afc-830b-a9c495d6e409.jpg?asset_id=7e1d7c01-0da9-4293-9382-e53c8799cb11&amp;img_etag=%22d4b3f7c77423036853d40455a368c90f%22&amp;size=2560)](https://spark.adobe.com/page/v9fe8jjE1rxvH/images/93db9d0e-cc11-4afc-830b-a9c495d6e409.jpg?asset_id=7e1d7c01-0da9-4293-9382-e53c8799cb11&amp;img_etag=%22d4b3f7c77423036853d40455a368c90f%22&amp;size=1024)中國電子科技集團、神州網信、聯想、微軟企業之間的合作

同樣令人擔憂的是，梅爾森概述了世界各國政府如何使用類似的Windows操作系統。換句話說，微軟授予了以黑客、網絡攻擊和間諜活動而臭名昭著的中共國不受限制的訪問權，該平台可與美國政府使用的平台相媲美：

**“ Windows 10是有史以來最安全的Windows操作系統版本，我們很感恩，美國國防部，澳大利亞衛生部和意大利國防部等世界各地的政府都由於其安全性選擇Windows 10，現在中國政府有專門為其創建的Windows操作系統版本。”**

儘管中國電子科技集團保留了合資企業51％的股份，而微軟保留了其餘49％的股份，但（人們）對於微軟和Windows對操作系統的影響知之甚少。

此次合作只是該公司與中國共產黨廣泛關聯的一個例子–考慮到TikTok的商業利益和市場准入，這一令人擔憂的前景可能成為微軟收購TikTok的籌碼。
