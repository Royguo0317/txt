- 推廣羥氯喹是人道需要


傑克連線弗拉基米爾·澤連科博士(Dr.Vladimir Zelenko)，澤連科博士一直處在與CCP病毒作戰的最前線，他也是最早宣導使用羥基氯喹、阿奇黴素和硫酸鋅組合治療CCP病毒的專家，博士曾多次提到這樣的治療方式非常有效。現在有所謂的疫苗出現，但是這樣的疫苗是否真能有效非常存疑，所有證據研究都確定羥氯喹對新冠病毒治療非常有效，這樣好的組合治療方式卻不被FDA（美國食品藥品管理局）推廣，主流媒體也對羥氯喹的療效撒謊——這完全是反人類的犯罪行為——受害的是普通無辜的老百姓，特別是年長者。現在正是睜開雙眼，看清真相，阻止人道危機的時候。

弗拉基米爾·澤連科博士強烈指責世界上這些骯髒的力量，他們為了獲得權力，滿足欲望，選擇犧牲別人生命去換取他們想要的利益，他們已經準備好做任何事，包括大量謀殺無辜的人和摧毀我們的經濟

Vladimir Zelenko 博士說很多醫生現在吃羥氯喹預防感染病毒，但他們在自己所在的醫院都不敢提及羥氯喹，因為他們會因此成為被抵制的對象。我們需要一些富有的資助商和公關團隊來對羥氯喹加強宣傳，我個人知道幾百個醫生都在給他們自己，家人和病人服用羥氯喹；但是他們不能讓外界知道，不然的話他們很快會失去工作，特別是那些在大醫院工作的醫生。Vladimir Zelenko 建議醫生們勇敢的站出來，尊重生命的價值，捍衛天賦人權，而不是因恐懼而躲在背後。

班農先生問Vladimir Zelenko 博士如果您今天可以向川普總統提幾點建議，你最想說什麼？

Vladimir Zelenko 博士說：第一建議川普總統取消對羥氯喹的限制；第二要有足夠的羥氯喹分發到各個藥店和醫生們的手中；第三，應該引導美國FDA全面注重門診病人。

班農再問如果您所提的建議今天開始實施，會對現在死亡人數有什麼影響？

Vladimir Zelenko 博士說如果病人在感染病毒５天之內，但最好是在染病後兩天服用羥氯喹，保守估計會減少８４％的住院率，死亡率也會減少80%——我的資料告訴我只有０.７１％的病死率，然後經濟運作可以在一周內開放。

Vladimir Zelenko 博士說這些人出於直覺自我保護，手上沾滿鮮血，他們終將會被曝光。中共是哈佛大學和耶魯大學的最大的資金來源。中共也控制著美國的媒體，中共是馬克思意識形態的保守派，他們不相信上帝的存在，他們只相信自己擁有絕對的權力。現在就是相信上帝的人與對權力極度渴望人的鬥爭。我們正處在第三次世界大戰中。現在是需要準備犧牲的時候，即使是犧牲你的生命，不然將會有百萬人的死亡。

二， 美國暴雨加暴亂
