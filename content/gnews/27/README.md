翻译：香草山写作组  billwilliam  校对/发稿：飞虹

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3-1-2.jpg)



以下是拉希姆·卡萨姆12月30日在《国家脉动》的节目《1月6日会怎么进行》“How January 6th plays out”的摘要，主要谈听证会公布乔治亚州大选舞弊，霍利参议员遭沃尔玛在推特上攻击，从宪法角度解释川普如何获胜，以及中共对新闻界的渗透。
![]()![](https://gnews.org/wp-content/uploads/2021/01/Jietu20210101-090752-1.jpg)
1. 在乔治亚州，有人做证发现投票机有wifi功能，能同时收发数据，很容易被骇客远程操控。不管是在移动投票站还是政府大楼，这都不应该发生。骇客只需要攻入一个点就可以，因为一台平板电脑就可以影响一系列附近的投票机和设备。理论上，投票机应该是不受无线电控制的。

2. 霍利参议员和30名其他共和党议员打算对选举结果提出异议。拉希姆号召老百姓给彭斯副总统打电话或发电邮，要求他对大选舞弊开始行动。

3. 嘉宾Ted Noel（前医生及自由作家）说，有争议州的选举结果应该作废，因为没有进行真正意义上公平有效的选举。乔治亚州规定必须由选民提出申请才可以进行缺席邮寄选票，但实际上所有人都获得邮寄选票，而且没有检查邮寄选票的签字是否一致。宪法第二章第一节明确说选举归州议会管。州长、州务卿和州最高法院无权介入。

4. Noel说，根据宪法第12修正案，由参议院议长也就是副总统打开有认证的信封，开始统计选举人票。因为7个有争议州的选举违规无效，选举结果作废，在这几个州民主党和共和党都不得票，不存在认证。那么统计剩下的州，川普得232票，拜登得222票，川普以多数优势获胜。想达到这个结果需要彭斯副总统站出来。

5. 因为霍利参议员对大选结果表示异议，沃尔玛的官方推特号攻击他说：去呀，去进行2小时的辩论，#酸酸的失败者。之后，沃尔玛澄清说这个推是由他们社交媒体组 “错误“发出的，说他们不打算谈选举人团认证的问题。拉希姆说大公司这样做不负责任。

6. 推特和脸书封锁了拉希姆和国家脉动的账号，而且不给任何理由。拉希姆有投钱来推广他的账号，所以脸书对他封号是违反合同。这对他的新闻生意有巨大影响，他的44. 6万粉丝收不到国家脉动的新闻提示。这个造成经济损失并伤害了他的名誉。

7. 娜塔莉·温特斯重新谈了中共的统战组织中美交流基金会（CUSEF）渗透美国媒体的情况，几乎每个主流媒体都被拿下。这个前一天她讲过。
