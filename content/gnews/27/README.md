## 中共超限战灭港计划（二十三）双十暴动(下)
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1039951/)

搜集/撰文：岁月如歌 / 封面合成：文粤

“双十暴动”发展到不可收拾的地步是必然的，表面看似因港英徙置事务署撕去贴在墙上的青天白日国旗而引起的暴乱，但是我们分析当时的国共两党势力在港的明争暗斗，就很清楚为何会这样。与其说国共双方势力在香港是抢夺主导权不如说是国共内战的延续，只是从大陆打到了香港。没有蒋政府的支持，没有中共掺傻子掀不起这么大的风浪。当年的双十暴动级别一点也不比2019年的反送中温和，两者都造成社会很大的撕裂，只是性质不一样罢了。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3314.jpg)图片来自风传媒
没有深仇大恨，哪来的“死博”国共两党的深仇大恨比杀父之仇还严重，对国民党而言那是亡国恨。只可惜随着时间的推移，今天的国民党不仅因为当年的双十暴动大势已去，还被中共几十年如一日的渗透腐蚀了原本的根基和曾经的骨气，由一个正统的老大败落成土匪流氓中共的小弟。回看历史不胜唏嘘，世事沧海桑田，谁主沉浮。

话说当年双十暴动到后期演变成国共两派势力的对决。三合会原本是支持国民党势力的黑社会组织，但是三合会的所作所为真不能让人苟同，和反送中出现的白衣人、美国的antifa没什么两样，都是对社会公共秩序很大破坏，并被中共利用。为此蒋政府不得不利用《香港日报》淡化国民党的责任同时与其割席。中共就是最喜欢这类人，并善于利用这些人的破坏力对付对手，我们看白衣人和antifa是如何为中共所用。或许也由于割席之故，三合会由支持国民党势力转变成中共的主力，让中共趁机占领香港，上演1941年日本战胜英政府——港九沦陷的悲剧。中共居高临下之势，对港英政府不断施压，嚣张的气焰仿佛要港英政府吃了。我们不要忘了前面文章提到的中共与港英政府的所谓的外交谈判勾兑。看到这情况很多心里明白的香港人、国民党势力认为若中共此时收复管治香港，比大英管治还好惨，都纷纷退出暴动。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3315.jpg)图片来自香港记忆
综观香港历史上的几次大运动，“双十暴动”，“六七暴动”，“雨伞运动”，“反送中运动”都离不开中共的影子。自1921年起就像瘟神一样，在香港阴魂不散，直到今天已是香港的魔王。不同的是，前期的运动“双十暴动”，“六七暴动”中共躲在背后搞事，后两个运动则走到前面来了，充当真正的魔鬼。前面它是参与者，后期它是施暴者，身分发生变化，使得性质发生变化。共同点是中共和黑社会都出现在这些运动里。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3317.jpg)图片来自搜狐手机网
在反送中运动没出现前，很多研究香港历史的专家学者们都认为雨伞运动是双十暴动的延续，有很多相同之处，比如黑社会出现，雨伞运动已开始不久，黑社会就出来了，暴打游行人士。管理者的言词也类似，最后清场也一样。反送中运动出现，个人认为更是”双十暴动的“2.0版，一切都是重复升级当年发生的事情，并为之过而不及。我们也清晰地看见中共在香港地从无到有，从小到大，从弱到强到超级魔鬼。香港不仅是香港人文明发展到历史纪录本，更是中共发展的的罪恶印记。历史会记住。
