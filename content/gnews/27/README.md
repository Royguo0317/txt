路德在3月8日节目中曾谈到担心两会期间有人投毒，故两会代表有可能都被“捅菊花”，接受肛拭子检查。虽然路德是以半开玩笑方式说出，但是采用肠道带毒的方法是具有可行性的。

1. SARS-Cov-2可长时间在肠道及粪便内生存。因肠上皮细胞存在较高水平的ACE2和丝氨酸蛋白酶表达，所以SARS-Cov-2可侵染肠道上皮细胞。对阳性病人肠道活检、尸检发现病毒颗粒和肠道炎症反应，证明了SARS-Cov-2在肠道内的生存能力。

2. SARS-Cov-2病毒可在粪便内存活5天时间，并且具备传染能力。

3. 人可保持长时间“无症状”状态，即使肠道内已感染SARS-Cov-2。研究表明一半左右的患者呼吸道样本病毒核酸转阴后粪便样本核酸阳性平均持续11.2±9.6天。也就是说肠道感染比呼吸道感染具有更强的隐蔽性！

4. 粪便+气溶胶传播性可以很强。2003年香港淘大花园事件是典型的SARS“粪便+气溶胶传播性”案例，造成了同一小区331人确诊、42人死亡！

所以通过肠道带毒、放毒是可实现的。特别是带毒者通过服用“病毒胶囊”类似的东西，将病毒直接送入肠道感染，而不经过呼吸道，则可保持带毒者长期无症状，具备极强的隐蔽性。排便到哪里，就带毒到哪里，甚至放个屁都是在排毒！
