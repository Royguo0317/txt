英国首席科学顾问帕特里克·瓦伦斯（Patrick Vallance）周五说，有证据表明，英国发现的中共病毒变种测试呈阳性的人死亡风险增加。

瓦朗斯在新闻发布会上说：“有证据表明，与旧病毒相比，感染新变种病毒的人的死亡风险有所增加。”

“假设六十多岁的男性，平均每1,000名感染中共病毒的人中，预计将有10人死于该病毒。而平均每1000名感染病毒变种的人中，大约有13或14人死亡。”
