**到底谁“不吃这一套”？**

自2021年1月19日蓬佩奥离任前一天宣布中共及习近平犯有种族灭绝罪行后，中共媒体和政府的反映就是骂字当头，认为蓬佩奥是给新疆扣大帽子，骂蓬佩奥是“上演最后的疯狂”。对于种族灭绝只是说些不痛不痒的话，希望拜登总统上台后给予平反。超出他们想象是，短短两个月，美国、欧盟、澳大利亚、英国等都列出了制裁清单。3月22日，欧盟宣布制裁新疆生产建设兵团党委书记王军正和新疆公安局长陈明国及其各自前任。制裁宣布后，3月24日，新疆维吾尔自治区人民政府发布《新疆维吾尔自治区人民政府关于对欧盟无理制裁的严正声明》，开头表示“欧盟基于谎言和虚假信息，以所谓‘新疆人权’问题为借口，对中国新疆有关个人和实体实施单边制裁。欧盟此举罔顾事实、颠倒黑白，严重违反国际法和国际关系基本准则，严重干涉中国内政，严重伤害新疆各族人民的感情。对此，新疆维吾尔自治区表示坚决反对和强烈谴责！”这些话已经听腻了，类似老牛反刍咀嚼，毫无新意。

![]()![](https://gnews.org/wp-content/uploads/2021/03/image-415.png)

新疆政府在自己家门口坚决反对、强烈谴责的论据还是老一套：人口增长说明种族灭绝是谎言；清真寺里有水电路气通信，有10所宗教院校，有四种文字出版的《古兰经》等说明新疆没有限制宗教信仰自由；认为新疆设立的“教培中心”与英国的DDP（转化和脱离项目）、法国设立的去极端化中心本质上没有区别，是为了预防反恐和去极端化而采取的有益措施和积极探索；新疆使用少数民族文字和颁布的各种条例、非物质文化遗产说明不存在“文化灭绝”；新疆劳动者按照劳动法有各种权利，不存在强迫劳动。这些事实的列举，空洞、没有常识，偷换概念，用人权问题代替种族灭绝。针对中共的这些假话套话，骗人的鬼话，其他国家已经不吃“这一套”了。

如何使种族灭绝舆论转向？借助H&M声明恢复之机，3月24日11:48分，中共共青团在新浪微博上以“一边造谣抵制新疆棉花，一边又想在中国赚钱？”为题，@了H&M中国，同时标签“H&M碰瓷新疆棉花”。“超级碰瓷”作为热词，让“新疆棉花”成为头条，凡是与新疆棉花沾边的明星和产品，要么抵制，要么使用，“新疆棉花”取代“种族灭绝”占领了舆论高地。中共外交部、商务部等政府部门也纷纷表态，商务部发言人高峰表情凝重、一句话一低头地念着稿子：“纯白无瑕的新疆棉花，不允许任何势力玷污、抹黑，对于个别企业基于虚假信息作出的所谓商业决策，中国消费者已经用实际行动作出了回应。”外交部华春莹拿着一个美国60年代监狱犯人从事劳动改造摘棉花的照片说是美国黑奴被迫摘棉花，用新疆机械化采棉照片说明新疆不存在强迫劳动，并强调“中国老百姓不允许外国人一边吃着中国的饭，一边砸着中国的碗”。《环球时报》也以《是时候给这些赚中国人钱的外国品牌立个规矩了》高调呼应共青团倡议。

尽管中共政府和媒体强硬反对，但阻止不了制裁法案的施行。亚马逊电商平台将逐渐下架所有来自中国的棉制品，耐克、阿迪达斯等也发声明抵制新疆棉。而在中共国，周冬雨、陈伟霆、倪妮、王源、杨幂、黄轩、香港歌手陈奕迅等艺人宣布与支持种族灭绝不使用新疆棉的品牌取消广告代言合约。一时间，新疆棉花成了官媒和社交媒体的热点。

**谁是“新疆棉花运动”的幕后推手？**

共青团新浪微博起底了抵制新疆棉花的“幕后推手”——BCI。BCI全称是Better Cotton Initiative，中文翻译为良好棉花发展协会，于2009年注册，总部位于瑞士日内瓦，是一家非营利的国际性会员组织机构。BCI会员体系共分为四类：零售商和品牌、供应商和制造商、生产者组织以及公民社会。截至2019年底，正式会员超过1800个 ，零售商和品牌包括耐克（Nike）、阿迪达斯（Adidas）、彪马（Puma）、迪卡侬（Decathlo）、沃尔玛（WalMart）、优衣库。在2020年9月22日，美国国会通过了《防止强迫维吾尔族劳动法》后，该协会随后倡议“不与位于新疆的任何服装制造工厂合作，也不从该地区采购产品或原材料”。新疆主要服装的原材料就是棉花，2020年的棉花产量已经占到全国产量的87%。新疆棉花棉绒长，质量好，占世界棉花产量的20%。

《环球时报》在3月27日撰文，认为BCI所说的“强迫劳动”是新疆扶贫，直接骂BCI无耻。作为国家官媒，《环球时报》将强破劳动与新疆扶贫挂钩，无逻辑裸奔到“突破天际”了。

3月26日，美国白宫和国务院发言人都发声指责中国政府主导了抵制外国服装品牌的社交媒体运动，呼吁国际会社反对中共以私人企业依赖其市场作为武器，扼杀言论自由。

《环球时报》认为，这次新疆棉花运动完全是媒体自发的，中共官方没有威逼过任何人和企业。虽然网上声讨激烈，但中共全境并没有发生针对相关外国品牌门店的现场破坏事件，是美国的反华政策制造了极其严重的市场对立。
