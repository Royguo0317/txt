中国知名的民营企业家最近的是一个接一个的出事。如蚂蚁金服IPO，在上市之前突然被叫停！之后，马云被官媒狂批几天。紧接着11号河北农民企业家孙大午被抓，大午农牧集团被查。17号南京首富民营企业家杨宗义被刑事拘留，公司被政府部门控制。20号重庆民营企业家李怀庆被重判了20年，中共割民营企业的韭菜已经开始了！中共国民营企业现在是风声鹤唳，人人自危了！

这次跟过去的一些民营企业家被抓的情况还很不一样，这次的是很多连公司的高管都一起抓。公司被中共直接控制，而且了抓捕在多省同时发生，都是在中共五中全会之后。

11月17号被抓捕的南京前首富杨宗义是江苏扬州富兴财富资产管理公司的实际控制人，他是在习近平考察南通，扬州和南京离开江苏三天以后被抓的，理由是非法吸收公众存款。

就在习近平江苏行之前，大午集团创始人孙大武和他的十多名集团高管被警方跨市抓捕，11号凌晨一点钟左右，六辆大巴载着手持冲锋枪，牵着警犬的特警闯入了大午新民居，大午员工和20多名公司高管被捕，罪名是寻衅滋事和破坏生产经营。大午集团28家子公司都也被官方接管，随后，有人组织员工，村民到公安局请愿，结果39人被抓。

11月20号已经被抓捕三年的民营企业家李怀庆，以欺骗和煽动颠覆国家政权罪等四项罪名，重判了20年，宁夏的资产全部冻结，他的8名员工也被判刑，多数的刑期超过十年，李怀庆是重庆富华典当公司的创办人，2018年1月31号，李怀庆被重庆警方逮捕，被指控造谣，诽谤等方式煽动颠覆国家政权，今年六月才开庭审理。
