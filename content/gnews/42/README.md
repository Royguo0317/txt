毁人不倦的中共国教育部12月23日制定颁布了《中小学教育惩戒规则（试行）》，《规则》将于2021年3月1日起实施，其中首次对教育惩戒进行了定义。
![]()![](https://gnews.org/wp-content/uploads/2020/12/974135cc97933f1c046d2080512f1070.png)
其中第二条称：本规则所称教育惩戒，是指学校、教师基于教育目的，对违规违纪学生进行管理、训导或者以规定方式予以矫治，促使学生引以为戒、认识和改正错误的教育行为。

是基于什么教育目的呢？《规则》里不说。看着教育部血色背景中脑袋有病的圣上，不言自明。

![]()![](https://gnews.org/wp-content/uploads/2020/12/3a92030ad9eecd2134ff52ec54f602ec.jpg)
