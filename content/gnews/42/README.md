从2016年底到现在，在全球华人世界里，“郭文贵”和“爆料革命”这几个字，可以说是无人不知无人不晓。在北美，亚洲，欧洲，人们在许多大城市都看见过衣着整齐的爆料革命战友的身影，有声势浩大的游行，也有在中共国大使馆前面的抗议。可以说，全球有中国人的地方就有爆料革命的战友，包括在美国中西部华人比较少的城镇，人们都可以在当地的mall (商场）、居民区看到头戴七芒星蓝色帽子的战友们派发CCP病毒传单。 还有社交媒体上不分昼夜时差的推特大军，除了华语世界，#Miles Guo 和#Whistleblower Movement也成为美国主流媒体和英文社交媒体的热搜，如班农战斗室，华尔街时报，纽约时报，福布斯，外交政策杂志，全国公共广播电台等。

从419 美国之音采访被断播，哈德逊演讲被取消，到美国司法部公布的重大司法文件，郭文贵先生和中共的角力牵出了一系列重量级人物的司法案件，包括美国共和党的金主 – 共和党全国委员会前财务副主席Elliott Broidy，美国智库Center For Security Policy的副总裁和美国前参议员大佬的遗孀French Wallop。无论是在华语世界，还是西方社会，无论是毁谤，还是赞誉，郭文贵先生都成了一个极具争议的人物，他就像一只走进瓷器柜的公牛 (A bull in a china closet)， 彻底打碎了人们赖以生存的那个被精致的谎言所编织起来的世界。在他面前，伪君子的内裤被扒掉，顶着各种光环、身价千亿的明星变成了公用卫生纸，西方政客大佬口中满嘴的自由民主变成了满地掉落的痔疮。从2017年开始，四年多的时间，他以一己之力抗衡一个巨大的国家机器，面对各种围追堵截和遍布世界各地的天罗地网，他不但杀出了一条生路，而且创造了一个非凡的传奇，他唤醒了数以亿计的中国人的良知和勇气。经历70多年中共的统治，中国人的脊梁早已被打断，尊严早已被血泪淹没，这个高贵的民族很久已经不知道站起来的感觉是什么了，是郭先生和他领导的爆料革命让中国人站起来，找回了久别的尊严。好奇的人们经常问 “郭文贵到底个是什么样的人？”

跟随郭先生、爆料革命的人们，数以亿计，他们来自世界的各个角落，来自社会不同的行业。他们中，有来自中共国的党政军高官，有体制内的成功人士，工农兵商，有学生，也有陪读妈妈，有老人也有孩子。

本文介绍的就是这样一个普普通通的爆料革命战友，赵向往，来自辽宁沈阳，来美国20多年，今年67岁，住在美国西部落基山下的犹他州。走在街上看到他，你会觉得他是个朴实无华的邻家大爷。在华盛顿DC抗议SEC时，我看到他在休息的时候跟一个东北老乡唠嗑才知道，他来华盛顿DC参加抗议活动一周，他的女朋友每天都打电话关心他，叫他注意安全，我很好奇，快70岁的中国大爷也有这么浪漫贴心的女朋友啊，我就问他看女朋友的相片，赵大哥从手机里滑出一张他跟女朋友的相片，让我惊呼不已，这是一个地道的美国大美妞呀，蓝色的大眼睛忽闪忽闪的，长长的睫毛，身材丰满健美，突兀有致， “这是我女朋友，比我小20多岁，犹他州三届选美冠军”， 赵大哥淡淡的说，“她跟了我7年了，打都打不走”  。我更好奇了，让赵大哥给我们介绍介绍他撩妹的秘诀。

“我有两个顶天立地的肩膀，不管狂风暴雨，女人都可以把头靠在我的肩膀上停止哭泣，我有一个soft heart (温柔的心），女人是水做的，你得对她好，她怎么做错事，我都能原谅“ 。

赵大哥除了心好，身材也很棒，看不出是快70岁的人，每天从早上9点到下午5点，在SEC抗议现场，腰杆儿挺得笔直，精神抖擞，休息的时候还给我们演示他平时练瑜伽的动作。

“郭先生让咱们健身，我每天都健身，还练瑜伽”，难怪赵大哥身材保持得那么好，软件（心灵）、硬件（身体）全面发展，吸引了比他年轻20多岁的犹他州选美冠军。

我问他，怎么还不结婚？ 赵大哥说， “我快70岁了，她还年轻，我叫她找男朋友结婚，她不走，一直跟着我”。

“那您的媳妇呢？” 我问赵大哥。他抽一口烟，低下头，神情黯然，“我媳妇命苦，40多岁就得肺癌死了，她在沈阳化工厂工作，你知道，化工厂都是污染，她从不抽烟，但是得了肺癌，污染的，留下了一个女儿给我带大，我带女儿来美国，女儿今年30多岁了，不想结婚也不想生孩子，我这辈子是当不上姥爷了。”

**“为什么离开中共国？”**

“我在国营建筑公司工作，搞机械制造，我不会溜须拍马，也不服管，也没有关系，永远不可能有升迁机会。我爸是林彪四野部队的，当兵8年没提干，别人当团长，司长，他不会溜须拍马，不会舔，不会送礼，永远是大头兵。”

“他外号叫 ‘长命星’，因为他参加这么多战役，没打死过一个人。 我爸参加过抗美援朝，辽沈战役，平型关战役，海南岛战役，枪口总是抬高三厘米，不忍心打死任何一个生命，谁不是爹妈养大的。 我爸活了80多岁。我跟我爸一样，草根，没有升迁机会。”

“96年，我找一个算命的老太太给我看看，这老太太会摸骨算命，她说 ‘你得去西方，不然在中国你会家破人亡’我听了她的话，1996年趁着一次公派出国的机会就不回去了。”

果然3年后的1999年，江泽民开始迫害法轮功，铺天盖地残酷打压，我身边认识的几十个人、朋友被抓，被失踪。沈阳有个马三家，那是折磨人的地狱，我要是留在国内，肯定会去反抗，我媳妇说我不服管，路见不平，我肯定上去就打，肯定会被抓被打被弄死。”

“算命的老太太有先见之明，96年就叫我快跑。我在美国，在科罗拉多州呆了10年，然后在犹他州呆了10多年，以炒锅维生（中餐馆厨房炒菜）。2016年底在网上看到‘中国流亡富豪郭文贵爆料’，当时就想，‘这个人厉害’，很佩服，后来又看了陈小平采访，何平采访。一直听到419断播就不可收拾了，每天必听，一天不听就感觉缺了什么。”

“打工的时候，也忍不住抱着手机听爆料革命，老板不高兴了。后来我去新的餐馆找工作，我跟老板说 ‘工资多少无所谓，我只有一个条件，活儿我不给你耽误，但是我干完了活，我要自带电脑看爆料革命’，这是我找工作的唯一条件。”

**“为什么相信郭先生，为什么跟随爆料革命？”**

“咱都想灭共，但除了远走高飞保住一条性命，还能干啥呢？ 我相信老郭，因为他知道怎么灭共，以美灭共，以钱灭共，以共灭共，以毒灭共，我从没有怀疑过。我快70岁的人了，在有生之年加入了爆料革命，在这个天翻地覆的时候，跟着爆料革命把共产党灭了，咱也没白活这三万多天，我不怕死，死就是长眠。这一生能和爆料革命在一起，值了。”

疫情期间，赵大哥参加过去年的817大游行，从没有停止传播中共病毒的真相。“我自己印了5000份闫丽梦博士介绍病毒真相材料，在州政府、商场Mall派发，走街串巷往邮箱里塞，有一回在一个商场，一个开奶茶店的中国人把我发的真相材料给扔了，我急了，这传单上有闫博士美丽的相片，他给扔了，这不等于跟侮辱我一样吗？ 我叫他捡起来，他不捡。我告诉他，知道你为啥戴口罩吗？ 这是中共CCP病毒，他说你有啥证据？ 我告诉他，这传单里就有证据，你捡起来自己看。 他说我不会看英文，然后他骂我是美国狗，我说你这个屁民，看我不揍你！ 别看我60多少岁，我打你也一样！后来我忍不住揪住他的脖子，结果他喊警察救命。”

**“为什么来华盛顿DC抗议SEC？”**

“GTV敢为天下先，让中国人挺起了脊梁。一个由中国人创立的社交平台，第一个告诉全世界的人们中共病毒的真相，第一个曝光亨特·拜登的硬盘门，第一个深刻全面独家爆料中共蓝金黄、全世界的真相。来华盛顿DC抗议SEC，是咱们爆料革命战友的责任，我也是GTV的投资人，我戒掉了赌博，把省下的钱都投资了GTV，我来这里也是为自己站台，为自己发声，为真相发声。”

**“投资赚了钱会做些什么？”**

“我今年67，再过几年70多岁了，如果赚了很多的钱，我能吃多少，能喝多少？ 我这辈子也当不了姥爷了，如果我投资有了很多的钱，我要回馈生我养我的那片土地。

“中国有很多贫困地区，无家可归的孩子，没人赡养的老人。我听我同学说，我们当初上山下乡的地方还那么贫穷，我在内蒙古昭乌达盟阿鲁科尔沁旗的天山口红旗大队插队两年。将来我如果有了钱，我想给那里的孩子们建一个好学校。”

**喜马拉雅之最： 贵族精神**

“郭先生为战友们谋福利，造福大众，这是贵族精神” ，赵大哥说。这跟中共国和美国的富豪们不一样，那些人代表的是既得利益集团，是靠舔共和奴役草根老百姓而致富的。中共国的富豪马云，潘石屹，张欣身价亿万，他们为草根老百姓做了什么？  美国的富豪小渣（脸书CEO）身价亿万，打压老百姓的言论自由，世界首富亚马逊公司(Amazon) 的创始人贝索斯(Jeff Bezos)，身价1770亿美元，他创建的亚马逊帝国，也是全美领取政府救济金人数最多的公司。在亚历桑州，三分之一的亚马逊公司的全职员工靠工资无法养家糊口，付不起医疗，必须依靠联邦政府的救济金和医保维持生活。除了亚马逊，还有沃尔玛公司(Walmart) 公司, 沃尔玛的货源来自中共国被奴役的廉价劳工，在美国给工人最低薪水，挤垮了数以百万的小商家，变成了全世界最大的私营公司和家族企业 （其控股人为沃尔顿家族）

小渣是贵族吗？贝索斯是贵族吗？沃尔顿是贵族吗？马云是贵族吗？潘石屹是贵族吗？ 张欣是贵族吗？ 他们不是，他们是金钱和权利的奴隶。

贵族精神，不是指金钱上的富有，而是真正独立自由的意志，社会的当担，维护人性的尊严，扶助弱势群体，不是恃强凌弱，他们的存在不只是自己发光，而是帮助更多的人找回生命的高贵和尊严。

贵族精神，为了追求真相和正义，敢于挑战强权，敢于超越自我，不会被政治强权与多数人的意见所奴役，也不会在金钱面前失去正义和信仰。

在赵大哥的眼里，郭先生是真正的贵族，在郭先生身上，他看到了什么是真正的贵族精神。年过花甲的他，在爆料革命中也找到了自己人性的尊严和高贵的贵族精神。

这种精神在每个爆料革命战友的身上都有，不论男女老少，无论他们过去的人生经历，他们不约而同地走进了爆料革命，因为他们都在郭先生身上看到了人性的尊严和高贵，追求自由和真相的勇气，跟随爆料革命，也让他们自己找回生命深处的贵族精神。

“我们骨子里不是草根，不是贱民，我们是真正最高标准的贵族，有尊严，追求真相的” ，郭文贵先生对战友们说。

战友们在“[喜马拉雅之恋](https://www.youtube.com/watch?v=ed41ohKdiWg)”的歌里回应郭先生, “在时空的一瞬间，我们彼此相遇； 从陌生到生死相依，只为那远方的喜马拉雅；她圣洁的光芒，召唤着你我的魂”。
