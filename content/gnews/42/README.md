据《酷冬》（Bitter Winter）网2月12日报道，上周，英国广播公司（BBC）突然间破坏了中共国从世卫组织（WHO）调查员那里取得的优势，这些所谓的“调查员”表扬了中共武汉病毒实验室对调查的合作以及展示出的透明性。BBC指出，没有任何侦探可以在罪犯用超过一年的充足时间销毁证据后能找到线索，而中共国与世卫组织的“朋友”关系已是多年来的公开秘密。

报道指出，但是，BBC讲述了一个更可怕的故事，即中共是如何通过新疆集中营系统地管控妇女并对她们性侵，因此，中共自认的与国际媒体同欢的假象消失了。BBC讲述的故事并不是偶尔发生的虐待，勇敢站出来作证的维吾尔妇女谴责了集中营内普遍存在的“强奸文化”，在这种文化中，性虐待是破坏女囚犯的穆斯林信仰和她们自尊心的方式。不仅如此，整个性侵过程还被中共高层官员秘密监控。在集中营内，女囚犯是中共官员及其狱警满足自身欲望的工具。

据该报道，中共国外交部狡辩说，作证的妇女是BBC招募的“女演员”，但实际上她们全都以其真实姓名出现的，而且她们与难民营的联系也得到了证实。美国，英国和澳大利亚政府援引BBC的该报道，谴责了中共对新疆妇女的暴行和虐待，全世界其他国家的政府，国际机构和领导人也同样对中共的暴行进行了谴责。

《酷冬》网站通过在集中营里拍摄的独家录像带，为BBC的报道提供了证据，并在其节目播出后，又用了几个小时对其内容进行审查。该网站说，“我们通过一些常规的渠道，收到了来自中国‘教育’营地和监狱中的前囚犯对我们的文章以及BBC报道的称赞。一些从来没有勇气以真实姓名作证的妇女告诉说，她们正在考虑这样做。”

报道指出，这些受害者坚持认为，BBC节目中描述的可怕的强奸和酷刑场面，不仅仅只发生在新疆，穆斯林妇女也不是唯一的受害者。来自被禁止的宗教运动，例如全能神教和法轮功的女囚犯，也遭到了强奸。他们的故事与维吾尔族和哈萨克族妇女在新疆集中营中遭到强奸的经历非常相似。中共使用强迫灌输和性虐待的组合拳来贬低他们的信仰，但这些全能神教会和法轮功的女性信奉者遍布全中共国，她们拥有中共不能摧毁的强大的意志力。

报道进一步指出，现在，许多居住在国外的哈萨克族与《酷冬》一起坚称，中共应披露有关男狱警如何在新疆集中营强奸男性囚犯的故事。他们说，在难民营中对男人的性虐待也是一种经常性事件，而不是偶尔发生，特别是对于年幼的男孩而言。对于为什么没有男性敢站出来为这些少数勇敢妇女的指控提供证词这一问题，《酷冬》称，他们曾询问过一些男性受害者，其中有哈萨克族人，甚至包括为BBC的报道做过证词的人。得到的答案是，在哈萨克社会和侨民中，承认这种强奸是件非常困难的事，因为这会使前囚犯的某些亲戚朋友误信他们是同性恋者，但事实上他们并不是同性恋者。对《酷冬》反对哈萨克斯坦将同性性行为定为犯罪的态度，被询问者回答说，尽管法律规定同性性行为并不违法，但任何人只要被怀疑是同性恋，就会受到多种形式的社会排斥。这一点似乎曾得到证实，在2015年的一项国际调查中关于同性恋幸福度问题上，哈萨克斯坦的世界排名是第118位。

报道认为，在新疆集中营里被性侵的一些人，确实向哈萨克斯坦和其他人权运动组织讲述过他们的故事，但并没有授权可以公开这些故事，因为此举可能会暴露他们的个人信息。在某些被强奸的男子提供其姓名，姓氏和拘留细节作证之前，中共肯定将否认在集中营的所谓“改造”中发生了对男囚犯的性侵事件，就像他们曾经被实锤证据指控后，依然否认对女囚犯做出过相同的事。但《酷冬》指出，那些讲诉这些故事的人都在讲真话，这令人感到满意。

据该报道，福克斯新闻社（Fox News）后来采访了一位曾上过BBC节目的名叫图尔森奈·齐奥杜顿（Tursenay Ziawudun）的妇女，她说道，“我们是人类，但是他们折磨这些女孩甚至男孩的方式就像对待我们是动物一样。”说完后她又强调，“甚至是男孩。” 这可怕真相的另一部分正在逐渐浮现出来。
