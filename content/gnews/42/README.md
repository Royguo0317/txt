**世卫组织负责人表示，“所有假说都摆在桌面上”以解释大流行的根**

![]()![](https://gnews.org/wp-content/uploads/2021/02/310x190_devant-hopital-wuhan-chine-6-fevrier-2021.jpg)图为2021年2月6日，中国武汉一家医院外 图片来源: 来自原报道图片

他于2019年12月在冠状病毒大流行的发源地武汉度过了一个月。但是，世卫组织派遣彼得·本·恩布雷克（Peter Ben Embarek）和其他调查人员试图了解这一流行病的根源，他们感到非常的困难。

“我们需要更多数据。代表团团长在接受法新社采访时说，我们要求提供更多数据。他补充说：“对于在给定的时间范围内可行的做法，有一系列的挫折感，但也有现实的期望。”他希望中共能提供他们要求的数据。

这些专家在一周之初完成了任务，但没有得出任何最终结论。起源日期和地点仍然未知。

正如特朗普政府声称的那样，迄今为止，他们似乎已经排除了该病毒从武汉病毒研究所意外泄漏的可能性。对于他们来说，这是一个“极不可能的”假设。

但是，世卫组织的负责人潭德赛（Tedros Adhanom Ghebreyesus）随后似乎在纠正这种情况，星期五他说，“所有假说都保留”来解释大流行的根源。

世卫组织专家认为，中共病毒起源于蝙蝠，可能已通过另一种哺乳动物传播给人类。然而，即使在2019年12月之前在武汉或其他地方没有大流行的报道，他们也不知道病毒大流行的真正爆发时间和地点。

**美国重返世界卫生组织**

为了给中国施加更多压力，白宫也加快步伐。

国家安全顾问杰克·沙利文（Jake Sullivan）说：“我们对如何传达中共病毒调查的最初结果以及有关实现该过程所用程序的疑问深表忧虑。” 他补充说：“为了更好地了解这一流行病并为下一次流行做好准备，中国必须在流行的头几天提供其数据。”

在唐纳德·特朗普的管理下，美国离开了世界卫生组织。前共和党主席指责世卫组织与中共沆瀣一气。

但在1月20日上任后，他的民主党继任者乔·拜登宣布华盛顿重返世界卫生组织。

“夺回我们的位置也意味着要求最高标准。杰克·沙利文（Jake Sullivan）表示，在这个关键时刻，保护世卫组织的信誉至关重要。
