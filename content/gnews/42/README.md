中共最激进的党报《环球时报》于6月2日发文宣称，美国最新军费预算中的“弹药采购清单”显示，美国空军正在大幅度削减专门用于对付伊斯兰恐怖组织的导弹等武器装备，却同时大量囤积“与中国开战”的新型先进导弹。

《环球时报》透露，美国空军为了确保远程打击太平洋地区军事目标的新型导弹采购数量，将减少针对“伊斯兰国”等恐怖组织的导弹采购。从川普政府开始，美军已经实施从阿富汗、中东撒离军事力量，将战略重心转移到印太，目标就是围堵中共，拜登政府继承了这个战略。

据美国《军事》网站报道，美军还增加了高超音速武器的研发投入并要求尽快交货，同时希望釆购敌方难以发现和拦截的隐形导弹，即联合空对地防区外增程导弹（JASSM-ER），这种导弹可以在中共的防御范围之外对太平洋广大地区进行精确打击。为了尽快得到这种导弹，美军向导弹生产商支付2.11亿美元用于“提高生产线能力”。同时，美军将针对恐怖组织的各种导弹采购量，大幅削减了3～9倍。

另据媒体报道，美军紧急采购了478架F-35战机，为了尽快生产出这批战机，美国上万工人在24小时全面赶工。这还不包括美国盟友采购F35的订单数量，这些国家拥有F35的主要目的无疑是用于对付中共。F35有海陆空多种机型，已知其中有三款机型具有隐身功能，可以突入内陆而且来去自由，这是啪啪打脸中共的神器。F35部分机型可以垂直或短跑道起降，售价约1亿美元左右。

美国及其盟国积极备战，目标是不战争，因为现代战争过于残酷。但必须给中共足够的震慑，避免中共产生战略误判，以为可以打胜仗，以为西方畏惧战争而蠢蠢欲动，最终将人类拖入万劫不复的深渊。如果不出意外，三个月后，西方将认定中共为恐怖犯罪组织、中共国为敌国，将启动全面制裁和封锁，现在西方积极备战，旨在表明中共狗急跳墙时将予以坚决回击。
