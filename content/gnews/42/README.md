当人们群情激愤地涌向H&M大呼抵制西方品牌；当“中国人不吃这一套”的文创（暂且叫文创吧，虽然很亵渎文创这个词）产品在网上热卖；当打疫苗送鸡蛋的福利吸引无数人排队前往；当国内的亲朋好友不停地说“国外经济都不行了，只有中国行”……扑面而来的，是那么多的“正能量”，全民全社会的“正能量”涌向你的时候，你不与他们一起振臂高呼就显得与他们格格不入，似乎会被社会所抛弃。

当海内外自媒体铺天盖地为国外华人受到歧视而鸣不平，多少国内同胞投来同情而又焦虑的目光，善意地以“还是国内安全”为理由劝说着国外的朋友回去；又有多少海外的华人放不下满屏“安全”、“强大”、“岁月静好”的诱惑，使出浑身解数，多方努力，终于拿到回国的健康证明和机票……再不走，似乎就要与无限的机遇失之交臂；再不走，似乎你就要堕入地狱。

被社会所抛弃、堕入地狱，这是多令人担惊受怕啊！于是，上文所描述的那一幕幕就在不停地上演，这就是中共病毒肆虐一年多来，国内外的华人所感受的，一波又一波不断升级的某种“力量”。人们被这种“力量”所左右，在这种“力量”的推动下做出各种判断和决策，但却对它毫无察觉。每个人都以为自己的决定是个人的，是有利于自己和家庭的，但是，当所有人的决定汇聚在一起的时候，你会看到什么？

1944年，著名的经济学家哈耶克因出版了《通往奴役之路》而受到广泛的关注。但这并不是一本纯粹的经济学著作，他将经济学、政治学、社会学结合在一起，在半个多世纪之前就描述了一派今天我们所见的景象以及它必将导致的结果。其中的许多文字在今天读来格外掷地有声： “像学会如何驾驭自然力量那样学会如何驾驭社会力量，……这不仅是一条通向极权主义的道路，而且是一条通向我们文明的毁灭的道路，一条必然阻碍未来进步的道路。”这句话已经被文革所验证。而现在，你能感觉到吗，那股推动着所有人的“力量”，正是有人在试图驾驭社会力量，习神，他几乎已经做到了。

“如果一个人不需要服从任何人，只服从法律，那么，他就是自由的。” 中共国有法律吗？没有，只有党的意志。 “自由——服从共同的抽象规则；奴役——服从共同的具体目标。” 小粉红们觉得现在生活富有了，世界没有资格居高临下跟他们说话，这就是自由了，但哈耶克告诉你，这不是自由，这是奴役。 仔细读这几句话，那股“力量”清晰可见，现在的中国社会正是被驾驭着通向一个共同的具体目标，走向奴役，走向极权，走向文明的毁灭。并且，习神还在试图将这股力量推向世界，企图驾驭全世界的社会力量。事实上，这就是哈耶克在1944年指出的“通往奴役之路”。愿意放弃自由来换取保障的人，其最终既得不到自由，也得不到保障。
