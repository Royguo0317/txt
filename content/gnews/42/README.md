# 南中国海：美国宣布巨型致命导弹驱逐舰将返回消灭“侵略者”

[!\[\]()!\[\](https://spark.adobe.com/page/D77L4xwyEdE9o/images/4b030d5e-c5b2-4a66-a8fc-4cc511699f4c.jpg?asset_id=52d37aa4-901d-4276-bf45-86e2b971c690&amp;img_etag=%2218542c259bd01031fc92eff422ee16f8%22&amp;size=1024)](https://spark.adobe.com/page/D77L4xwyEdE9o/images/4b030d5e-c5b2-4a66-a8fc-4cc511699f4c.jpg?asset_id=52d37aa4-901d-4276-bf45-86e2b971c690&amp;img_etag=%2218542c259bd01031fc92eff422ee16f8%22&amp;size=1024)

美国宣布，美国海军导弹驱逐舰巴里号返回南中国海执行安全任务，引发了人们对有争议水域公开冲突的担忧。

在上周六，美国太平洋舰队的官方网站发布了再次使用阿利·伯克（Arleigh Burke）级巡洋舰的声明。 声明说，巴里号战舰将帮助“促进该地区的和平与稳定”。

巴里号指挥官克里斯·盖尔（Chris Gahl）表示：“在南中国海的持续巡航对维护自由开放的印度太平洋至关重要。

“所有国家在国际水域中航行的自由至关重要。

“巴里号昨天从台湾海峡过境，确保了这项权利，并给所有在南中国海进行贸易和交流的国家灌输了信心。”

关于过往船只的目的，巴里号船员之一的海军中尉佐敦·布鲁克斯（Jordan Brooks）说，：”每天有大量的渔船和商人通过这些水域开展业务，这真是令人难以置信。

![South China Sea: USS Barry]()![South China Sea: USS Barry](https://cdn.images.express.co.uk/img/dynamic/78/590x/south-china-sea-1363329.jpg?r=1606135225403)南中国海：美国宣布大型致命导弹驱逐舰返航消灭“侵略者”（图片来源：盖蒂）

“为了安全，有效，专业地完成我们的任务，巴里号船员不断地进行团队合作，时刻保持警惕和沟通。”

美国最大的同类驱逐舰15中队的美国海军制导导弹驱逐舰一直停留在该地区。

本月是2020年巴里号第五次被部署在该地区执行例行任务。

巴里的计划和战术官，海军少校蒂莫西·贝克（Timothy Baker）说：“今年四月，巴里号在西沙群岛（Paracel Islands）周围进行了自由航行（FON）行动，然后与美国海军远征军打击小组会合，在南中国海进行行动。

“无论是个体还是作为更大的集群的一部分，巴里号都是美国可以用来打败侵略的压倒性力量的鲜明标志。”

北京在环礁上建立军事基地之后，美国呼吁其他国家反对中国在有争议水域中的统治地位之际，这一消息爆了出来。

美国东亚及太平洋事务局负责区域与安全政策及多边事务的副部长助理大卫·费思（David Feith）表示，华盛顿将增加“随船观察员”的数量以应对中共国的“侵略行为”。

费思先生说：“在某些地区，例如北太平洋，无国籍渔船具有中文注册的特点。

“此外，据估计包括3000多艘船只的中国海事民兵，在公海和其他国家的主权海域积极进行侵略行为，还胁迫和恐吓合法渔民，以支持中国共产党的长期海事战略目标。”

根据随船观察员协议，允许一国当局人员登上另一国巡逻的执法船只或飞机。

对非法捕捞进行管制的举动引起了美国东南亚国家联盟（ASEAN）的几个盟国的关注。

印尼战略与国际研究中心研究员吉朗·肯巴拉（Gilang Kembara）表示，雅加达不赞成美国采取军事行动。

他说：“我认为，如果美国提出其海岸警卫队和印尼合作是一件好事，因为非法，不报告和不管制捕鱼（IUU）是一种犯罪活动，因此我们需要执法部门予以打击。

“但是，如果他们提出的是与美国海军的合作，这将成为[军事]问题……这种方法被夸大了，因为我不认为非法、不报告和不管制捕鱼对一个国家是一个生存威胁。”

菲律宾大学海事与海洋法研究所所长杰伊·巴通巴卡尔（Jay L Batongbacal）警告说，菲律宾也可能提出反对。

他补充说：“但是，[马尼拉]可能会对海上活动的信息共享感到满意，至少在过去的两到三年中，政府，特别是渔业局，实际上已经利用了美国提供的有关外国在菲律宾专属经济区（EEZ）进行钓鱼活动的信息。”
