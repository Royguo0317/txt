有史以来最大的战争灾难无疑是二战，在这场近代战争中死亡的军人达1800万，被刻意杀戮的平民和战俘大约也是1800万(维基)。两项指标均大大超过一战说明，武器的杀伤力提高和人类道德的提升对比产生了严重的失衡。由此可以断定，如果大规模杀伤性武器被邪恶的中共掌握，肯定对人类产生迫切而巨大的生存威胁。

简单对比一下二战前的德国和当今的中共不难发现，两者都穷兵赎武，但有很多不同点。1933年上台的希特勒追求国家社会主义，他的口号是“向失业宣战”，除了大搞基建(当时修的高速公路现在还在用)，还让更多的人当公务员、当兵以解决失业问题；希特勒反马克思主义、反犹太(部分原因是谋犹太人的财)，为了报一战战败之仇而穷兵赎武——希特勒的极端思想得到了绝大多数德国民众的支持。请注意，当时的希特勒并未被神化，希特勒及其內阁成员、军方要员之中没有任何人同中共的众多盗国贼一样富可敌国。

希特勒纳粹残暴凶恶，但不屠杀本国人民。反观中共，执政70多年来，已直接或间接迫害致死约7000万中国无辜百姓(笔者参考了不少中外史料，这个数字还是保守的)，证明中共比德国纳粹更邪恶、道德更沦丧，因此中国人民对中共更仇恨，这是中共要还的旧债。当前中共面临更大的危机是——国内经济崩溃已无法避免，国际上因故意放毒被围剿同样不能幸免。

自知已处于穷途末路的中共，为了维护统治及避免被清算，在无计可施的情况下，极有可能效法伊朗，建立一个由宗教领袖控制的政权体系，通过彻底愚化奴化中国人民，以达到苟且残喘的目的，下面分析其可能性。

据美国防部透露，中共的海军总吨位及岸基导弹弹头数量已经超过美国，而且掌握了可跨洲际投送核弹的超高音速技术。[【1】](https://nationalinterest.org/blog/buzz/what-makes-chinas-military-so-deadly-one-word-missiles-180862)中共近年来凶相毕露，胆敢在香港、新疆犯下反人类罪，发动超限生化战，就是因为加入世贸财力增强后，不断扩张军备并掌握了大量的大规模杀伤性武器。美国海军上将戴维森在近日的国会军委听证会上指出，中共已经“胆大妄为”，如果缺乏令人信服的威慑，中共将大胆采取行动，破坏基于规则的国际秩序和自由开放的印太愿景中所代表的普世价值观。这里有一个基本判断，“道不同不相与谋”，由于中共的邪恶被爆料革命暴露无遗，其共产专制和他国的价值观格格不入，因此中共不可能同以前一样依靠金钱、武力收买和臣服他国，取代美国独霸世界是中共不可能实现的妄想。

所以，当前最大的概率是，中共龟缩本土，并集中常规军事力量在台海、南海制造事端，在几个大洋布置战略核潜艇进行核恫吓。此前习近平公开宣称中(共)国的安全环境不稳定，要作好以战逼和的准备，就是为了对抗世界各国对中共发动超限生化战的集体讨伐。中共的好战派一贯认为，即使中(共)国本土遭到猛烈攻击甚至核打击，由于人口众多，地形复杂，有足够的战略纵深，中共仍然可以打持久战，西方不可能把军队开进中(共)国本土。有的人甚至认为，美国有十个原子弹打到中(共)国，“我们”有一个打到美国就赢了。和李毅的死几千人不算死是一个恶毒逻辑。

简而言之，接下来中共极有可能实行彻底的信息封锁与闭关锁国，重回高压的毛式文革时代。当年的毛泽东仅称“万岁”但并未称神，习近平及7常委极有可能封神的目的，是用神来掩盖自己极端的邪恶、无能和彻底的失败，用神强化对中国人民的愚弄和奴役，用神的名义镇压不满人群，用神的名义把一切责任嫁祸西方。可以预见，即使中共发动超限生化战在全世界范围内被裁定为铁案，但西方的讨伐大概率止步于中(共)国的国门之外，如果中国人民不奋起灭共，中共就绑架十四亿中国人民继续其残暴统治。

郭文贵先生之前曾经有预见性地指出，不要幻想美国帮我们灭共，最终灭共得靠我们中国人。是的，中共能存活多久无疑是由中国人民来决定的。纵观世界各国历史，民主和自由从来不是君主的恩赐，而是英雄的战利品，历来如此！
