一、新中国联邦新闻

1. “路德时评”本日摘要（[早间](https://www.youtube.com/watch?v=wBz14JJNyis)、[晚间](https://www.youtube.com/watch?v=zW90SWCc9xw)）:

Ⅰ. 在乔·拜登宣誓就任美国新总统后，中共“崴脚部”发表声明，制裁包括蓬佩奥在内的前川普政府28名官员。蓬佩奥在离任前夕发表声明，正式认定中共对新疆维吾尔族等少数民族”犯有种族灭绝罪行”。中共的制裁实属形式，只会警醒美国政界，加速灭共进程。被定为反人类罪在中共党内将引发巨大矛盾。

Ⅱ. 乔·拜登正式宣誓就任美国第46任总统。就在上任的第一天，他一口气签署了17项行政令，将川普时期的多个重要问题彻底逆转。针对中共制裁的行政令无一取消。爆料革命经几年不懈努力在灭共的进程上取得了巨大胜利，灭共生态圈已经形成。

Ⅲ. 乔·拜登四大高官今天集中表态“战胜中国”。1月19日，美国参议院分别就布林肯、国家情报总监人选海恩斯、财政部长人选耶伦和国防部长人选奥斯汀的提名举行了听证会，“中共国挑战”和“战胜中共国”成为这一天国会里最响亮的词汇。这些最新迹象表明，“新政府将延续川普政府对华的强硬政策”。

Ⅳ. 1月18日的中共“崴脚部”例行记者会上，华春莹对病毒起源妄称：“如果美方真的尊重事实，就请开放德特里克堡基地，并就美海外200多个生物实验室等问题公开更多事实，请世卫组织专家去美国开展溯源调查，回应国际社会关切，用实际行动给国际社会一个交代。”中共“崴脚部”的回应，说明病毒起源真正成为各国关切的重要问题。

2. 中共纪委国家监委网站1月20日发布题为《贵州省委原常委、原副省长王晓光案以案促改工作启示：斩断茅台非法利益输送链 推进名贵特产特殊资源专项治理》的文章，谈及该案时指出，名贵特产是地方的特殊资源和金字招牌，决不能成为敛财的“摇钱树”、送礼的“敲门砖”、变现的“硬通货”。

3. 中国联通、中国移动、中国电信分别在港公告，将于1月20日，向纽约证交所提出书面要求，要求纽约证交所董事会的一个委员会覆议下市决定。根据纽约证交所上市公司手册，复议将获安排在提出复议要求之日起至少25个工作日后举行。外界认为，中共正在揣摩拜登政府上台后，美国政府灭共的决心。

4. 1月21日，推特表示，他们锁住了中共国驻美大使馆的账号，原因是该账号先前发布的一则关于新疆的推文违反了该平台的准则。华春莹则表达对此事的关切与不解。

5. 中共国务院应对肺炎疫情联防联控机制办近日发布了关于进一步做好当前疫情防控工作的通知。对不同人口规模城市的核酸检测能力做出明确，常住人口500万以上的城市，应具备在3-5天内完成全员检测的能力。此前多个大中型城市在疫情应对中确实发现不具备短期实施全员检测的能力和储备。

6. 中共全国人大常委会本周三至周五开会审议《海警法》草案，授权中共海警在处理外国船只在其理解的 海域进行“违法”活动时，可以使用武器。有分析认为，《海警法》通过后，战时中共海警在武装部队体系中的定位将更明确，甚至会成为“第二海军”。

二、世界新闻

1. 美国第45任总统唐纳德·特朗普1月20日登上空军一号飞往佛罗里达州。他的四年总统任期当天结束。他离开白宫时的讲话说，担任美国总统是“一生的荣誉”，祝下一届政府取得成功。川普总统不参加拜登就职典礼，再次打破了数十年的传统。

2. 1月20日，乔·拜登在华盛顿宣誓就职，成为美国第46任总统。随后在演说中他多次强调”全美团结”，表示深知美国目前面临”现实而深刻的分裂”。

3. 乔·拜登上任的第一天，发布了17项行政命令重塑权柄。其签署的第一条行政令废除了川普政府组建的“1776委员会”， 该委员会旨在重新审视美国建国历史，并努力让美国人民重新团结在建国价值观和原则周围。

4. 台湾驻美代表萧美琴获正式邀请，出席乔·拜登的第46任美国总统就职典礼。据悉这是自1979年以来，台湾第一次获得美国就职典礼筹备委员会的正式邀请。此前新政府国务卿布林肯在其提名会上已表示，美国对台湾和《台湾关系法》有一个长期的、两党的承诺，并需要确保台湾有办法阻止侵略，保护自己。

5. SpaceX首席制造工程师杰西·安德森在周三（1月20日）的发射网络直播中表示：“本月早些时候，我们扩大了其早期‘Better than Nothing’服务计划，将英国的客户也包括进来。” 截至目前， SpaceX已经发射了1000多颗星链高速互联网卫星，并积极寻求其他国家的监管批准。

6. 乔·拜登已签署行政令，让美国重返《巴黎气候协定》，这标志着拜登政府将完全抛弃川普政府应对气候危机的方式，回归多边主义。
