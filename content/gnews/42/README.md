据CCN报道，近日一个在网络广泛流传的视频中，中共国一所高中校外，一群高呼“真相，真相，真相”的人们手拿白花，要求还原一个17岁高中生在校园内突然坠楼身亡的真相。

这就是最近几天受到中共国网民巨大关注的成都49中高二学生林唯麟坠楼身亡事件。5月9号当天，母亲节，林的母亲送他到学校，时间是17点40分左右，一个小时后坠楼身亡，这是学校报告给出的死亡时间 ，但是林的父母得到消息晚了2个小时，已接近晚上9点钟。并且此时，他们的儿子已经躺在了火葬场，父母来到学校，却被禁止与任何老师学生接触了解原因，还有，学校也不允许死者家长查看监控录像。

如此诡异的剧情，家长走投无路，在和校方交涉一夜无果后，也是在几乎所有媒体不给其发声的情况下，林的母亲在微博上发出了她的求救声音。

她的这个微博消息得到了几十万的转发，她写道：“我曾经在新闻上无数次看到这样的事情，但从未想到有一天会 发生在我头上”。关于她儿子莫名其妙死亡的故事引发了数千万网民的同情，并要求校方和地方当局曝光监控录像。

这就是为什么视频中出现那群人在学校门口高喊真相的原因。

人们对校方事发后诡异的行为提出质疑：学校在隐瞒什么？？？

1. 发生事故后驱散了老师学生以避免和家长接触 ；
2. 没有第一时间通知家属，整个推迟2个小时才通知；
3. 叫了救护车120，医生确认死亡迹象后直接送了火葬场（为什么不给家属见死者）；
4. 不给家长看监控录像；
5. 媒体在哪呢？


林的死亡触动了其它几个引起中共国人广泛共鸣的话题：中共当局缺乏透明度；打压媒体自由：校园自杀案频发。

CNN又举例了一些校园自杀死亡案件，同样的剧情，同样的处理方法，家长们都无法从校方和当局那里要到真相，只能救助于网络。比如，4月份，在重庆一所职业学校跳楼的15岁女孩，还有发生在山东的16岁女孩死在学校寝室里等等，父母们都无法看到监控录像，地方警察给出的说法几乎都是一致的，都排除他杀，不是说死前有自杀倾向，就是和林唯麟一样草率把尸体送到火葬场了事。

中共国的校园自杀、学生升学压力巨大已经臭名昭著了 ，然而即使很多专家呼吁了要关心青少年的心理问题，也无济于事。

在网民的压力下，林的母亲最终看到了视频，但是最关键的自杀时的录像消失了，理由是那里录不到。学校最后对此事做了说明，说调查已经排除那些不当说法，林的自杀倾向是个人行为。并且有发现林之前的短信有感情问题想不开，有试图割腕自杀倾向，警方最后也给出一样的说法。

林母亲的微博已经消声，警方说林家对调查没有异议。

在所有媒体都不敢报道的情况下，网络上有很多说法，一种说法是林的化学老师把他推下了大楼，因为他占用了学校的出国名额，这个目前无法证实。学校的学生没人敢正面发声，有的在网络上传出一些边边角角的讯息，不过这种很快就会被中共官媒给压下去。

匿名网民对中共监控设备的这种操作，讽刺的指出，抓人罚款的时候可好用了，一旦有事故发生，需要寻找真相就不好使了，那真是一个神奇的存在。

最新动态显示，林的母亲11号就不再更新微博，在林母消声之际，很多阴谋论出来带风向，挑动舆论。什么“敌对势力”论，有人上传视频称那些在学校抗议的人为CIA工作，还有的说他们是和成都美领事管有关的 ，甚至还有和颜色革命扯上关系的。

由于那个抗议视频已经被审查，所以各种谣言四起，把民众去抗议歪曲成了以上各种论调。

中共这种流氓，总能把自己干的缺德事最后说成是别人干的，欺骗、忽悠、恐吓齐上阵，无耻至极！

学校对于公众质疑，也给出了一些回应，比如，通知家属晚了是因为无法辨认死者身份。但是仍然不能消除人们心中的不满。况且，死者母亲至今没有发声，没有任何第三者出来讲述和被害人最后接触的情境，对于一个充满疑问的事件，始终雾里看花一样。最让人匪夷所思的是，为何历来的校园死亡案件学校、警方给的理由都像是一个模子刻出来似的——从来没有明确答案。
