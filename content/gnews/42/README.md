在视频里，老师带着懵懂的孩子们在国旗下宣誓效忠中共及国家领导人。

老师们也无奈的说，以往她们只教孩子们一些传统的儿歌，例如洗手歌、吃饭歌，但现在要响应党中央号召唱红歌，这是国家的政策。

视频还显示在10月底，江西一所幼儿园的孩子们还被要求穿着红军、八路军的服装，甚至家长也必须穿军装、工农服装进行耕地、插秧、施肥，以重现中共在延安时期的“大生产运动**”**。而据《大纪元》曾经的报道，《南泥湾》大生产所种植的其实是鸦片，并不是粮食。不知家长们如果得知真相，看到幼小的孩子被如此洗脑，愚弄。会是什么心情？

特别一提的是，在洗脑的内容里，有关宗教问题还被重点关注。例如3月7日，江西省上饶市皂头镇当局要求该镇所有小学及幼儿园开展 “相信科学，远离邪教**”** 主题班会，为了保证效果，还要拍照、录像并交给上级部门。另外，山西某小学竟然不许学生穿戴有宗教标志的服饰，更邪门的是如发现周围有人信教，也必须举报。

而在北京也在上演着类似的剧情，今年3月底，20多名便衣在北京房山区六年级教室内，逐一盘问学生家人是否有信全能神的，并利诱道：**”**谁说出真相，就奖励500元人民币。**”**之后，警察还逼迫学生填写反对宗教及全能神的调查问卷和答题，最后竟然要求学生及家长在一张写有 “永久不信全能神，踏踏实实跟党走**”** 的单子上签字。

这就是文革再现吗？还是升级版的文革2.0。如此倒行逆施，胡作非为，真是离灭亡不远了。

连老师也看不过眼，感叹道：小孩子最相信老师的话，政府让我们对幼童灌输这些，这明显就是让小孩从小就排斥信仰；孩子还小，什么都不懂，我不愿这样做，可没办法，只能走形式。**”**
