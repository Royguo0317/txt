# 彭佩奥指责中共国 “毒害” 美国大学

[!\[\]()!\[\](https://spark.adobe.com/page/9dkdtGEycYVOn/images/b459e358-690e-4e0d-a548-44ea1a24bad6.jpg?asset_id=052f23f8-010d-40b3-98a2-ae1f002321e0&amp;img_etag=%22c98d5714f99301f632457fca3a91922c%22&amp;size=2560)](https://spark.adobe.com/page/9dkdtGEycYVOn/images/b459e358-690e-4e0d-a548-44ea1a24bad6.jpg?asset_id=052f23f8-010d-40b3-98a2-ae1f002321e0&amp;img_etag=%22c98d5714f99301f632457fca3a91922c%22&amp;size=1024)

2020年12月9日星期三，亚特兰大，国务卿迈克-蓬佩奥在佐治亚理工学院发表讲话。(美联社图片/John Bazemore)

美国国务卿迈克-彭佩奥（Mike Pompeo）周三表示，中共国正在渗透美国的大学和研究中心，以图增强他们的实力。

彭佩奥先生在佐治亚理工学院的一次演讲中说：”美国人必须要明白（中共）为了自己的目的，是如何毒害我们的高等教育之资源，他们的这些行为是如何弱化我们的自由和国家安全的。”

彭佩奥先生在讲话中回顾了几年前佐治亚理工学院教授王飞凌（Wang Feiling）在访问中共国时被中共当局绑架的情况。

他说：“他们想要了解他关于中共国的研究，以及他在我的母校西点军校任教的讯息。他们认为他们可以恐吓他，或者招募他，因为他是华裔。”

在大学（向中共国）施加压力后，这位教授才被释放。

但彭佩奥先生说，“从这一事件中得到的教训很明显：中共想要我们所拥有的东西，它会不惜一切代价地去得到它，从偷我们的东西到向批评（中共国）的人施压，让他们保持沉默。”

彭佩奥先生说，北京的目标是要主宰美国和自由世界，这是许多西方人士直到最近才意识到的。

他说：“没有人可以责怪，长期以来，共和党人和民主党人都认为，通过与中共国的贸易和接触，中共会实行改革、会有所松动。事实上正相反，中共利用了这些财富，对中国人民实行更进一步的控制，并建立了一个世界上从未见过的运用高科技来实施高压的国家。”

中共国家主席习近平一直在寻求 “全面控制”，力图通过建立起一支拥有常规武器和核武器的高科技军队，使中共国成为世界上最有势力的国家。

彭佩奥先生争辩说，中共在美国的大学里招募了研究人员，而这些研究人员中的一些人没有申报他们受雇于中共国，以及他们与人民解放军的关系。

他说：“中共国在美国撒钱，有预谋、有目的、包藏祸心，就是为了获得宝贵的知识”。中共国还在美国校园里宣传洗脑美国人，恐吓中国公民。

马里兰大学学生杨淑萍（Yang Shuping）因在美国发表开幕演讲时，歌颂 “言论自由的新鲜空气 ”，而遭到中共宣传机构的骚扰。

[!\[\]()!\[\](https://spark.adobe.com/page/9dkdtGEycYVOn/images/2621cb0a-a951-43e7-8a74-b8c45b086600.jpg?asset_id=6c09db00-b676-428d-ba85-486b60665e53&amp;img_etag=%225a4909ed89540c112d57bee43d3f5ef6%22&amp;size=2560)](https://spark.adobe.com/page/9dkdtGEycYVOn/images/2621cb0a-a951-43e7-8a74-b8c45b086600.jpg?asset_id=6c09db00-b676-428d-ba85-486b60665e53&amp;img_etag=%225a4909ed89540c112d57bee43d3f5ef6%22&amp;size=1024)

在普林斯顿大学的《中国政治》的课上，学生被迫使用代号，以避免他们因为批评在香港实施严厉的新国安法而受到中共当局的起诉。

彭佩奥先生说：“我认为，这让人们对‘安全空间‘的概念有了新的认识，美国学生希望用‘安全空间’来屏蔽他们不喜欢的想法。中国学生则需要’安全空间‘来学习他们喜欢的思想。”

彭佩奥先生补充说，中共国也在利用左倾大学进行 “反美信息传递”，在校园里建立了一系列的以推广中国文化和语言为表象的孔子学院。

国务卿说：“这就是为什么他们在我们的校园里安插了孔子学院，这也是为什么这里有一些叫做中国学生学者协会的团体。他们在中共国的大使馆或当地的中共国领事馆指导和长期资助下，追踪掌握学生的情况，推动支持北京的事业。”

彭佩奥先生称，美国大学已经因为“害怕得罪中共国”而进行自我审查。

彭佩奥先生透露，麻省理工学院拒绝接受他在那里发表演讲的提议，因为他的论点 “可能会让他们的华裔学生和教授感到曲辱。但这绝不是我的本意。”

他说：“由于怕‘伤害感情’而反对（演讲），就是被中共玩弄于股掌。这是中共在回应世界各国的合理批评时一贯说的话。”

他补充说：“你看，我们不能让中共把政治正确当作武器来干涉美国的自由，欺骗性的种族主义和恐中症的叫嚣永远不应该淹没对中共的坦诚揭露。”

彭佩奥先生说，美国需要采取行动来应对中共国所构成的威胁。

他说：“我们绝不允许一个暴虐的政权窃取我们的东西来建立他们的军事力量，并对我们的人民进行洗脑，或者收买我们的机构来帮助它掩盖真相。”

他呼吁大学的领导们回击中共的压力，关闭孔子学院，调查中共资助的学生组织。

彭佩奥先生说：“让我们高举自由的旗帜，保卫我们的学校和我们的安全，抵制我们当代的核心威胁——中共”。
