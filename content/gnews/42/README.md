**西班牙2021年1月12日电/西喜社——**康涅狄格州将在未来10天内在康涅狄格大学的足球场开设第一个大规模中共病毒疫苗接种点，因为该州准备对75岁及以上的居民进行注射，州长内德·拉蒙特周一表示。

东哈特福德的Rentschler Field将在未来两周内，和其他约6个大型地点一起提供大规模的疫苗接种。

疫苗接种也将在私人供应商的办公室和卫生诊所提供。康涅狄格州还计划通过流动疫苗接种单位为贫困和其他服务不足的社区提供注射。

医护人员一直是康涅狄格州首批接种疫苗的人员之一，该州最近完成了对养老院居民的疫苗接种。

75岁及以上人群的疫苗接种提前登记计划于周四开始，该群体从下周一（1月18日）开始接受注射。
