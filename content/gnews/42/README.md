辛亥革命本质上是满清朝廷因为面对千年未有之变局，科举这种选拔官僚的方式不再适宜，于是将其废除，探索新型选拔方式。思路很对，但是有个问题就是读书人怎么办？

朝廷想了个招，扩招新军，顺便把这些读书人安排到新军做小军官。认为既解决读书人问题，又可以扩充朝廷兵马，一举两得。但是他们没想到这些原先是秀才功名的读书人给自己的目标是县太爷起步，现在却安排和低贱的丘八（对兵士的贬称）在一起，心里没有怨气？

读书人读了多年书或许有理想主义种子，但大多数始终以考功名为重，其实并不在意朝廷是谁做皇帝。平时可能做做“小粉清”，开口太后万岁，闭口朝廷英明，但是真到了紧要关头，那是谁在中央支持谁。比如1900年联军破京，太后西狩，读书人想到的不是进京勤王或者是关心老佛爷的安危，而是翘首盼望瓦德西元帅开科取士，好让大伙有做官的机会。否则也不会流出赛金花吹枕边风给瓦德西重开科举的段子了。

对朝廷有怨气，加上本身血气方刚，那潜伏在新军的革命党就可以把这些读书人发展成为自己的基本盘，随后再放出清军入关后扬州十日嘉定三屠文字狱的故事，激发愤怒之心。时间久了，读书人的心自然和朝廷南辕北辙，关系愈发紧张，到这地步就差一根导火索的事。其实武昌起义初期，清军战斗力不强，但新军战斗力也不强或者战斗意志也不强，因为连武昌城内的清军消防队都打不过，双方可谓菜鸡互啄。这也证明大多数读书人还是很鸡贼，爱惜自己的一百多斤肉，若不是湖广总督瑞澂决策连连失误，居然在两军僵持之时，弃城而逃，让武昌彻底被革命党翻盘。一点爆，全面崩，武昌的翻盘，让各地区的新军纷纷响应，最后发展到连铁杆盟友蒙古诸部王爷都背叛了，这时清朝灭亡是肯定的。

而这次“本降专”引发的学潮与此相似度很高，1999年共产党出于敛财因素，推动高校扩招，中国人读书考功名是刻在骨子里的，为此悬梁刺股者不知凡几，自然现有高校可容纳的人数和考上分数线的人数不匹配，这也顺势允许民办学院并承认证书资格，以容纳很多考上本科分数线的读书人，这套政策得益于共产党加入了世贸组织，让经济发展迅速，就业岗位增加，所以运转得还是很顺畅。

但是到了川普上台后的2017年，WTO要到期时，墙内也开始出现“本降专”的事情，比如本科不用功，毕业成专科的华中科技大学。到了现在，外资大规模撤离，民营企业也被共产党搞得哀鸿遍野，就业岗位稀少，而且习王这一票人都是衙内女婿出身，又经历过文革，自然喜欢又红又专，对于集权那是非常感兴趣。党政军民学，对前面四项下完手，自然该对学下手。资源的减少，必然引起内卷，共产党现在本身侧重讲政治，985，211是共产党自己树立起来的牌子，暂时还不能动，那就对挂“民”字的民办本科下手，先把它降为专科，下一步就把证书取缔都有可能。

其一，就是树立阶层意识，给还没下手的高校提升含金量，增加虚荣心，让他们更好的忠于党忠于总书记。

其二，就是给社会立威，断人前途如杀人父母，若是这都没反抗或者被共产党压下去了，共产党会愈发得寸进尺。共产党现在外部困顿，必然要整顿内部，思路是把内部压紧压实，加强社会控制，打成铁板一块，以应对越来越临近的外部危险。

回到学生视角，要是在以前，自己经过十几年的苦读，好不容易快熬到本科毕业，虽然是民办，但也比其他专科强太多。自然是风花雪月，不问国事，更过分的是不分是非，日常兼职小粉红为党鼓吹。但是读书人很现实，当自己日常歌功颂德的党要断绝自己的前途了，想想自己辛苦寒窗十几年，倾家耗血，而且自己在千军万马中考了本科的分数，要是没考上没话说，但是自己是考上了，而且大学也读了四年，结果现在却变成专科！！！

虽然现在专科和本科学历想找满意的工作都难，但因为还没出社会，心里自然给自己的日后规划是坐办公室吹空调，谈个漂亮女朋友，自己的工作收入加上父母贴补在城里付个房子首付。而现在学历变专科了，办公室吹空调的工作肯定颇多波折，而且谈女朋友在这本科遍地走，专科多如狗的时代，一说学历专科，大概率被PASS掉，爱情事业都受到严重影响，心理落差太大，心情和被安排到新军的有秀才功名的读书人自无二致，自此抗议发生。

共产党镇压还是老套路，口蜜腹剑，打“暂停”牌，威逼利诱，发动学生会班干部导员之类的有望能保障自身利益的人群对学生威逼分化，各个击破。而现在的大学生日常打游戏，刷微博追星磕CP，学校旁边的很多宾馆酒店也适合他们开房纵欲等等，这些都是消耗意志的东西，可能有相当一部分人会“翻墙”，也看过法轮功制作的共产党罪行信息。但是意志消磨了，血气自然见底，而且得益于共产党媒体常年累月的宣传，脑子里也有共产党的警察是为人民服务的，共产党的军队是人民子弟兵的观念，平常也是即使对上访维权，法轮功，香港的遭遇可能有所耳闻，但脑子里还是自动屏蔽这些信息。

他们认为自己不反共产党，只是维权。但是读书没读通，忘了书本里明明白白写着警察和军队是人民民主专政的暴力机器，所以遇到警察打人居然大惊小怪。其次对于政治课本没有理解通透的地方就是无论国家机器叫什么名号，民主专政也好其他的也罢，都是少数人统治大多数人，以少御众的，更何况没有民主选举的共产党？这其中最重要就是需要平稳，政治书课本上描写的农民运动，工人运动，学生运动之类的基本是朝统治阶层要权，冲击统治的稳固性。要是民主国家出来的遇到这大规模抗议，双方还有坐下来谈的可能性。而这种独裁政权，要是一抗议就退让的口子一开，那其他人也有样学样，它以后怎么统治？政治课本上相信描写这类例子也不少，但书就是没读透。这种独裁政权面对这种情况往往是权衡利弊后实行恩出于上，收取人心。就是我给你你必须要，我不给你你不能抢，常规套路就是新人扮演包青天给你平反。例如：远的如文革后的各种平反，近的则是十八大后习王孟废除劳教制度等等。

当然这群学生能在受了几十年共产党垃圾思想的毒害后，在自己利益受到威胁时，没有束手待毙，仍然敢出来抗争，而且此起彼伏，从南到北，是很多人没想到的，虽然其中期望青天大老爷的心理还是很天真，但也说明中国人不是某些高贵人口中的懦夫。

当然由于初期只为维权，并无其他诉求，意愿不强烈，对于共产党来说并无难度，毕竟从建政初期到现在处理的群体事件早就无数，称得上轻车熟路，对于这类群体事件不过小菜一碟，所以虽然套路老，却有效。但如今不同往日，外有国际围追堵截，内部也是人心涣散，而且这个政策也有反对派拾缀出来的可能，给共产党造出对手盘添堵（民办本科高校430所，一所学校最少5000人，至少有215万学生要“本降专”，也就是215万个家庭受影响，都是对手盘）。

以往共产党对这类群体事件能解决的顺利，比如98大下岗也是几千万工人没有生计和次年的法轮功，被共产党镇压得悄无声息，最主要原因是外部无事，内部还算团结，但现在这种情况类似于学生和共产党对线较量，学生虽然水平比共产党以往对手水平还菜，但是架不住共产党快下机了，网管（国际）一直嚷嚷续费，但共产党现在没钱续费，只能花时间去筹钱续费上机，而且还有人扯住它袖子不让它去筹钱，自然减少时间精力对线，这相当于学生得到了场外援助，给了学生的时间。

而且共产党现在不可能给学生让步，学生前途断绝，也不可能就此罢休，这只能激发学生的愤怒之心，增加学生的怨气，在怨气增加的情况下，自然对于共产党所造的罪孽真相能解除屏蔽，积极接收，心态发生逆转，而且每个群体都有一定的杰出人物在，这类人聪明开朗人缘好有激情讲义气，这类人在获得群体内部认同后，完成整合净化，踢出学生会班干部学霸凤凰男这些两面三刀的人，再改变策略，调整心态，积极和境外互动，造出风暴眼，成了足以影响人心的势，由普通学潮往革命进化，配合国际时势，成为共产党灭亡的其中一个因素是有很大可能的。当然第一桶金都很难的，无论是做生意还是干事业都是如此，共产党对中国人的监控达到了巅峰造极的效果，若无其他因素或者有聪明绝顶之人想出躲避共产党监控的方法，完成群体整合净化，那至少要多花很长时间，这需要能力，更需要运气。不过现在这个形势下一切皆有可能，中国有勇气有智慧的人多得是。
