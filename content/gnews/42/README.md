**西班牙2021年1月28日电/西喜社——**西班牙官方统计数据显示，中共病毒大流行在2020年摧毁了62.26万个工作岗位，使失业率达到劳动人口的16.1%。

这比前一年增加了2.3%，是2012年以来的最高值，当时西班牙遭受了金融危机的最严重影响。

周四的INE数据显示，尽管2020年最后三个月的表现好于预期，但今年年底仍有370万失业人口和1930万就业人口，其中包括约60万被辞退的工人，他们的工资由国家支付，而他们的企业仍处于关闭状态或受到疫情影响。

西班牙左翼执政联盟本周将就业休假制度延长至5月底。

全员失业的家庭数量从18.39万户飙升至2020年的近120万户，不过超过三分之一的家庭是只有一名成员的家庭。
