据《纽约邮报》12月22日报道，一份新的报道称，检察官指控谷歌和脸书达成协议，帮助对方抵御反垄断投诉。

据《华尔街日报》报道，这一指控是在上周德克萨斯州和其他9个州的总检察长对谷歌提起的反垄断诉讼的未删节版本中披露的。

共和党的检察长们指责这两家科技巨头达成了一项 “非法协议”，谷歌在其在线广告拍卖中给予脸书一定的优势，以换取该社交网络对其竞争威胁的退让。

根据这份未经编辑的诉状，这份协议被赋予了 “星球大战 “主题，代号 “绝地蓝”，在其中的一项条款中，两家公司承诺 “合作并协助对方应对任何反垄断行动”，并 “及时、全面地通知对方任何与协议相关的与政府之间的通信”。

诉状还称，2018年的合同是由脸书首席运营官谢丽尔-桑德伯格（Sheryl Sandberg）签署的，她给CEO马克-扎克伯格（Mark Zuckerberg）和其他老板写了一封电子邮件，称 “从战略上讲，这是一笔大买卖”。公开的、经过删节的诉讼版本没有提到桑德伯格的名字。

德克萨斯州主导的诉讼是谷歌面临的三起反垄断案件之一，另外还有美国司法部10月份提起的诉讼，以及上周38个州和地区检察院提起的另一起诉讼。脸书同样也遭到了联邦贸易委员会和46个州的反垄断投诉。

谷歌和脸书都对德克萨斯州诉讼中的指控提出异议。谷歌称德克萨斯州总检察长肯-帕克斯顿关于其广告技术（秘密交易）的说法 “不准确”，并否认其操纵广告拍卖。

共和党检察长们的诉状和证据清清楚楚地证实谷歌和脸书之间有协议，谷歌给脸书广告，脸书在社交网络竞争上向谷歌退让，大家和平共处，相互提携，将谷歌的广告和社交网络这两块牢牢地掌握在他们两家手中。这无疑是垄断，谷歌和脸书输定了。

谷歌多年来，在强大的资金支持下，买断所有中小公司，做大做强，以其超强的搜索引擎形成了垄断地位。

脸书最初做出了一个受年轻人欢迎的软件，而后在强大的政治背景下，上市并成为热捧股票，公司与参与其中的各方势力利益趋同，公司越做越大。

谷歌和脸书在做大做强后，开始垄断并控制信息的发布，他们反川普，屏蔽一切对极左翼和中共不利的消息，网页上充斥著污蔑川普总统和支持川普总统的，在政府、军方和经济等方面重要人士的文章和假消息，成为实实在在的假媒体。
