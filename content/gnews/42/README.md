美中角力下，最后的9局下半场，美国务卿庞皮欧，仍马不停蹄的公布多项的重要外交决定，如：

1. 对台的政策与台美关系的落实，取消美国和台湾官员往来限制。等同认同台湾是一个正常的国家，在亚洲自由民主阵营的一个重要伙伴。
2. 重新将古巴列为支持恐怖主义国家，将叶门反政府武装组织胡塞（Houthi）列为外国恐怖组织。
3. 承认摩洛哥拥有长期存在争议的西撒哈拉的主权。
4. 对伊朗实施新一轮制裁。


笔者认为：这些政策都是在巩固川普以往视为优先的事务，同时为拜登政府未来制定新方向时设下障碍。

1. 取消美国和台湾官员往来限制，这项决策等同认同台湾是一个正常的国家，在亚洲自由民主阵营的一个重要伙伴，是反共第一岛链和印太战列的重要成员。
2. 此外，美国疫情延烧至今，因中共病毒所造成的死亡和生病的人数仍居世界之冠，所以国内反中情绪高涨，加上还有极右翼分子对与中国实现外交正常化的反对，所以也是因应民意，未来只要有人想修改或触犯该政策，就会被贴上“亲中国”标签。
3. 美国佛州南部的古巴裔选民，在传统上是倾向共和党，倾向对古巴采取强硬态度，但在欧巴马政府时期，却建立了美国古巴关系正常化。所以重新将古巴列为支持恐怖主义国家，也算是对反共产主义的一种宣誓，但也有舆论认为庞皮欧此举是为未来共和党总统候选人铺路。
4. 西撒哈拉的争议涉及不同文化、宗教、种族和国家的地缘政治博弈，对于这个争议国际社会长期以来有着不同的分裂态度。**美国即将卸任的总统川普撮合摩洛哥与以色列建交，也就是**承认摩洛哥在争议的西撒哈拉的主权。
5. 将胡塞列为恐怖组织及被指一直在背后支援胡塞的伊朗实施新一轮制裁，借此定调他们为中东问题根源，为拜登未来外交政策设下障碍，让他没有机会重新加入川普在2018年退出的伊朗核协定。
