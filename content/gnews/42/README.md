## 【信仰之声】信仰和科学分得开吗？——从一个初信者的角度谈《圣经与科学》观后感

作者：香草山农场福音部 一棵草

![]()![](https://gnews.org/wp-content/uploads/2021/03/WhatsApp_Image_2020-10-24_at_04.33.05-9.jpeg)



这一期的《福音漫谈》节目谈了圣经与科学的关系，生命美歌弟兄和爱的灯火姐妹的解答，一定会引发大家去深入思考。

生命美歌弟兄说：“科学是客观的，很好；但是说科学是衡量一切的唯一标准，那这个就是错误的。科学不断地推翻之前的科学是错的，所以科学不能证明是对的。只能讲，到现在我还不能推翻这件事情。”

关于这一点我找到了一张图表：《圣经的话，之后科学和现在科学的对比图》。我把这个英文图表翻译补充了，和大家分享。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/05021659/%E6%9C%AA%E5%91%BD%E5%90%8D%E7%9A%84%E8%AE%BE%E8%AE%A1-71.png)图来源：https://biologos.org/articles/no-modern-science-is-not-catching-up-to-the-bible

由此可见，几千年前的《圣经》中记录的物质世界的真相，被现在的科学验证了。人类在《圣经》之后的时间，不断修改自己的认知，所以科学只是人类认知世界的方法，不是世界的真相。

生命美歌弟兄说，科学不是万能的，实验科学方法论是有局限性。如何用数据、可量度性、可控制性、可重复性，衡量爱和包容、谦卑和执着、勇气和感动、多重、压力多大、多长、体积、面积？”

联系现实，怎么衡量计算共产党邪恶，怎么衡量计算七哥带领的爆料革命灭共的决心和勇气？这些都是无法测量的，超越科学的范畴，但是正是这些无法衡量的属灵的正义力量在改变和拯救全世界。

“《圣经》讲的内容不只是物质的世界，还有属灵的世界。物质的世界从神而来，科学家和神学家应该相辅相成，科学和信仰彼此相辅相成。谦卑的科学家用一颗虔诚的心来研究上帝创造的物质世界。”

美歌弟兄的这段话，让我们不禁要问：人类世界是需要顺服神的谦卑科学家，还是需要没有信仰的科学狂人？

如果没有正义的闫博士，冒着生命危险，爆出病毒真相，这个世界会如何？

如果科学家都像福奇，和中共国那些没有信仰的制造病毒科学家一样，这个世界又如何？

我们心中已经有答案了，科学离开了信仰，就是灾难，就是人类的自我毁灭。

路德有一期说俄罗斯有一吨病毒，是生物武器的老大。众所周知的核武器，可以把世界毁灭多少次？人类一直都在自我毁灭的程序里，但是没有信仰，不顺服神的人，就会启动那个毁灭世界的按钮。

爱的灯火姐妹的分享：

“科学与《圣经》不冲突，超越物质的世界。神给我们理性智慧，双手去劳动，认识这个物质世界，有创造力去发明一些东西。造福我们，并管理这个世界。但是人活着不单单是仅此而已，而是人要超越这个物质，超越这个自然界。

“我们的灵与神连接上，有源源不断的智慧，源源不断的真善美的创造力在我们里面。有公义之心，是非之心，在我们里面。这样我们生命被造的初衷的尊贵的意义才能够活出来。

“所以耶稣说人活着不是单靠事物，是神说的一切话。人的心灵可以认识世界，认识自我，认识上帝的。需要通过圣经认识神的伟大。认识我们的有限，来认识生命的意义和价值。并且我们一生有盼望，死亡不是终点，是一扇门，打开永生的门。”

爱的灯火姐妹的分享让我知道，是神给了我们智慧，我们才学会了用科学这个方法去了解物质世界，但是属灵的世界，我需要用心灵去认识，去顺服神，感恩神对我们的爱，感恩神给我们的一切赐予。阿门！
