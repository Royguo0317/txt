## 【简讯】振动的赛格大厦将封楼检测
` 喜联盟编辑部 Himalaya Alliance Editorial Team` [轉載自GNews](https://gnews.org/zh-hans/1261936/)

#### 作者：日本东京方舟农场 雪国望乡
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/21075648/133-2.jpg)中国深圳79层的赛格大厦，连续3日摇晃，引发民众恐慌逃离。（STR/AFP via Getty Images)
深圳市赛格集团5月20日发出通知，将于5月21日零时起对赛格大厦进行封闭管理，以便对该楼栋进行检测。所有无关人员即时起不得进出大厦，直至检测完成后再视情况而定。

5月18日，网络流出视频显示，赛格大厦发生不明原因的振动，室内物品有晃动，人们吓得纷纷跑出大厦。

据《每日经济新闻》报道，赛格大厦一度是深圳第三高楼，也是世界最高的钢管混凝土架构大厦之一，也曾创造2.7天一层楼的“深圳速度”。在赛格广场晃动引发舆论关注后，一份多年前的关于赛格广场建造研究的论文也被翻了出来，称赛格大厦是“边设计边施工的”，论文题目为《深圳赛格广场建设项目评析》，作者是华中科技大学金典琦。论文中提到，在1999年4月8日钢结构封顶之后，地面行人发现赛格广场的天线在剧烈地摇摆，当时深圳只有微风习习，原来天线的设计计算错误造成共振。当时建设方和设计方立即召集全部人马，在来不及搭设安全设施的情况下，日夜奋战割除顶部26米，解除险情。重新计算后安装上去13米，并做了局部修改。

尽管中共国的“豆腐渣”工程人尽皆知，但据中共官方通报，5月20日专业机构对大厦的振动、倾斜、沉降等情况进行实时监测，未发现有超出相应标准要求的数据。目前，产生振动的原因还有待进一步核查。
