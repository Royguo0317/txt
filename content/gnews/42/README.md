二、路德社深入解读中共军事教材
![]()![](https://gnews.org/wp-content/uploads/2021/02/f5619e0ae4f56194f04ad54e5fab3327.jpg)
路德社深入解读中共军事教材《非典非自然起源和人制人新种病毒基因武器》，作者在本书中最新提出人制人新种病毒基因武器的概念，中共病毒就是这一概念的最完美诠释。

本书主编是徐德忠[1]（第四军医大学流行病学教研室，教授；INCLEN，CEU，主任），李锋（总后勤部卫生部防疫局，副局长；中国健康促进与教育协会，副会长）。这两位主编是中共军方高层专家，这本书就属于中共军方认定的教材。

**· 书本新术语**
![]()![](https://gnews.org/wp-content/uploads/2021/02/术语1.png)
书中前言部分提及：“本书提出的新术语，指目前国内外尚未见的流行病学和基因武器术语，是由我们首次提出，用汉语表述。”整本书就以本书提出的新术语为核心展开，新术语中提及“人制人病原体动物群体性适应试验”，本书定义为将动物病原体（目前已病毒为主）以各种方式和途径，攻击与人类细胞受体十分相似的或与人类近亲的动物，并做多种多样的许多次传代，最终使之适应在这种动物群体内传播，再通过类似方式部分适应于人群。中共病毒是中共人制人病原体动物群体性适应试验的产物，完全符合上述定义。路德社在去年节目中揭露中共病毒就是经过数次动物传代，经过筛选后让病毒慢慢适应人。所以中共就需要建立大量的P3实验室来筛选适合的病毒。
![]()![](https://gnews.org/wp-content/uploads/2021/02/第七.png)
第七个新术语“生态型基因武器型人类新发传染病”，中共病毒的骨架舟山蝙蝠病毒就符合该属于定义。从这本书新术语部分可以非常清楚地看到中共就是利用无数次的动物传代发展出最适合当作生物武器的病毒，虽然从表面上看该病毒是经过传代进来而来，但无数次传代却是在人工干预情况下进行的，最后打造出生物武器。
![]()![](https://gnews.org/wp-content/uploads/2021/02/第12.png)
第十二个新术语“生态型战剂贮存宿主动物”，选择某种宿主动物来贮存研制出的病毒。中共病毒可以寄生在雪貂和仓鼠中，这两种动物感染后仅出现轻微的症状却不致死，这样就可以保存病毒。

这本书已经将生物战提升到非常高的高度，中共病毒的所有特点完全验证这本书对新型生物武器的定义。这本书证实中共对于生物武器的研究已经处于世界领先水平，主编徐德忠是中共军方顶级的传染病学家，是2003年SARS的防控专家，但是他不是病毒学家，书中关于病毒的数据和介绍是来自中共军方大量的病毒学家和病毒研究实验室，这证实中共军方肯定有一个部门专门从事病毒生物武器研究。
