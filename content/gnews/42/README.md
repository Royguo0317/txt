为什么共产党要那么着急地批判一个外国人对外国的评价呢？说明中共和俄罗斯的某些政治家在心底里并不愿意彻底放弃共产党思维，如果给了他们机会，他们或许还会试图复活苏共乃至改革开放前的中共。并且，这可能代表着中共内部和外部的压力过大，他们必须要引导舆论去控制民众的思想，如果不这么做，那么民众们很快就会发现共产党在实施欺骗和愚弄，后果对共产党将是不可挽回的下台乃至灭亡。
