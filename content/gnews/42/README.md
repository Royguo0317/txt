近代中国出了个晋商，晋商不是一个人，晋就是山西，山西晋商。在明清两代商业非常发达，其实在宋朝就已经不得了了，明清两代就更加发达。全国形成十大商帮，最牛的就是山西的晋商。晋商商业做得非常成功，他们纵横整个欧亚大陆9000里，远到莫斯科，东北一带，生意做的到处都是。那个时代还是以马车为主要交通工具，他们就靠两条腿和马车，走遍整个欧亚大陆，汇通天下。当时号称有麻雀的地方，就有山西商人。在整整500年时间里， 山西商人，明清两代，都是稳居老大，称雄整个商场。 明清占据中国商业龙头，清朝统计，全国排前16位的大的企业集团，全部都是晋商，无一例外。

晋商有一整套培养人才的，非常严密完整的体系，还有运营体系，管理体系。后来统计，晋商手里的现银，超过当时清朝国库，这是什么概念！ 晋商的成就，所有成功的精髓，一副对联就把它总结了，上联是诚信为本，纵横欧亚九千里；下联是以义致（制）利，称雄商场五百年，致和制两个词都对。 一个是他地理上的范围9000里，在这个范围内他都是老大； 时间上称雄了500年。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/15135046/%E5%95%86%E9%81%93%E5%85%B8%E8%8C%83CHART1.png)
他的核心精髓其实用三个词就可以概括，他有商道的核心价值观义、信、利。以义制利，以诚信为本，这是他们做任何事情，判断的基础，也就是他们的核心价值观。他们第一优先就是看这件事情是否符合道义，如果不符合道义，坚决不做，不管这件事挣多少钱。第二点就是讲诚信，当利益和诚信发生冲突的时候，一定取诚信，不能不讲信用，不能见利忘义。 第三条，最后才取利益。在这个基础上才取利，挣应该挣的钱，这是利。 先义，再信，再利。晋商就已经懂得不义之财，坚决不要。

一定要先取义，再取信，这就是晋商最重要的核心理念。 单凭这一条， 我们看到现在那些著名企业就不能比，多少著名企业都是见利忘义的， 看到利益就不讲道义。晋商还有很多好的传统，包括晋商非常团结。晋商还在全国各地都建了会馆，只要是晋商，在各个地方都有一个办事处。我们七哥的喜马拉雅农场的概念就是取自晋商会馆，就是大家合作，互相帮助。比如说有人缺资金了，他们有银行，内部可以互相融资。晋商也很讲道义的，比如说徒弟在师父这儿学做生意，他绝对不可能说学到本事了，就跳槽在师父对面去开一个店铺去竞争，他绝对不做这种事，就算要做生意，也要远走他乡，去很远很远的地方，不跟自己的师父，自己的东家去竞争，甚至有的人做的更极端，我就不做这个行业。这就是晋商的德行，他非常注重道义和人才的培养。他们在那个时候都是有顶身股，就是全员持股。每个员工都有股份的，包括现代化的职业经理人，他们叫掌柜，把所有权和经营权进行分离。如果不是被共产党这些流氓篡夺国家政权，晋商的发展将不可限量。

晋商的很多东西被德国人学了去了，德国企业很多都是晋商的做法，下一期我们重点介绍德国的中小企业。
