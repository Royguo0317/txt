相信每位爆料革命的战友都对特德·克鲁兹（Ted Cruz），这位资深的美国德州参议员耳熟能详。最近，以克鲁兹为首的共和党11位参议员已经表示准备拒绝在1月6日举行的国会投票中认证美国2020大选结果，提议选举委员会对“有争议州”的选举结果进行10天紧急审核。可以说，克鲁兹一直走在维护美国选举制度和宪法的前沿。同时，他也一直是一位反共先锋，因为支持香港民主运动而在2020年8月被中共列入制裁名单。

然而，他做过的一件事情曾经令我一直心怀芥蒂。在灭贼运动中，他公开在2020年10月9日发推支持中共在美国的大特务-假牧师付希秋（Bob Fu），该推文中文翻译如下：

羞耻：中共共产党针对美国公民付希秋在其家进行羞辱骚扰。付为中国人权无畏发声，令习近平及同党闻风丧胆。中共是新的邪恶帝国，我们必须团结起来反抗他们。
![]()![](https://gnews.org/wp-content/uploads/2021/01/702.jpg)该图片为特德·克鲁兹推文截图
这篇推文颠倒黑白，把前线灭贼的战友们说成是中共，而付希秋俨然是民主斗士。作为爆料革命一员，我们当然反对克鲁兹的这种做法，于是我也如其他战友一样，到他的推下留言，取关他的推特号。而且推测他应该也被中共蓝金黄计划俘获，看似反共，其实是小骂大帮忙，充当的也是中共帮凶。

最近，随着美国大选的滑稽进程，我们可以看到副总统彭斯和参议员汤姆·科顿等人的立场和行为的不断变化，这些都印证了郭先生在2021年首次直播中谈到的“实力”之说：在实力面前实力可以改变忠诚。中国人认为忠诚大于天。在美国不是这样，实力大于信用，大于忠诚。忠诚和信用是服务于实力的。搞明白这些就没有所谓的背叛，川普不会有背叛的感觉。

我想郭先生的解释会让很多战友豁然开朗，在灭共过程中，每位跟随的战友都在不断提高认知维度，不断修炼自己的小心脏。对于克鲁兹这样的政客，他们受一批人和利益集团支持，也要代表和维护其利益。即使昨天还反对我们，只要现在支持灭共和新中国联邦，都是我们的战友。
