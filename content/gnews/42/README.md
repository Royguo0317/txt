**西班牙2021年3月25日电/西喜社——**据美联社3月25日报道，丹麦政府决定继续暂停使用阿斯利康疫苗三周。

丹麦于3月11日暂停使用阿斯利康疫苗作为预防措施，此前有报道称，一名60岁的妇女在接种疫苗一周后因身体多个部位出现血栓而死亡。丹麦也有报道称，丹麦有第二个人在接种阿斯利康疫苗后死亡。

丹麦卫生当局表示，他们没有证据表明疫苗对这两起死亡事件负责。挪威和瑞典也已暂停使用阿斯利康疫苗。

丹麦约有15万人注射了阿斯利康疫苗。
