**西班牙2021年3月26日电/西喜社——**据美联社3月26日报道，西班牙官方统计机构称，该国去年GDP收缩了近11%，创下至少50年来最差表现。

统计机构周五表示，西班牙第四季度录得零增长，因为经济受到流行病限制的影响。

这是自2013年以来首次出现全年萎缩。

去年唯一出现增长的经济部门是农业，该部门扩张了5.3%，而建筑业则感受到了最剧烈的收缩，下降了14.5%。该机构从1970年开始收集统计数据。
