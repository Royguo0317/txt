●赵高在望夷宫逼死秦二世胡亥后，沙丘政变3人组只剩下赵高一人。赵高手捧玉玺，认为自己称帝时机已到。他命令手下召集群臣于咸阳宫大殿等候，自己稍后准备登上大位，彻底接管秦帝国的最高权力。群臣被迫聚集在朝政大殿之上，人心惶惶，各种消息满天飞舞，一会儿是赵高起兵造反了，一会儿是胡亥派兵捉拿赵高了，一会儿是刘邦率军打进咸阳城了……，所有人都在不安中等待。

●此时，赵高正站在咸阳宫脚下，仰望着数公里远、百米高台上的朝政大殿，心潮澎湃。他迈开脚一步一步攀登这座宫殿的台阶，就如同他获取权力的过程，谄媚、隐忍、背叛、密谋、陷害、杀戮充斥着他罪恶的人生，现在他终于踏着累累白骨登上了人生巅峰。赵高没有罪恶感，因为他伺候的秦始皇嬴政就是这样一个屠夫、刽子手，他亲眼看到暴君嬴政是怎么靠杀戮得到天下土地，又是怎么靠暴政奴役天下人的！

●赵高走进朝政大殿，看到大殿上群臣熟悉的面孔，在这些或老、或少的面孔上，赵高看到了震惊、愤怒、胆怯、窃喜、冷漠、哭泣、讨好……各式各样的表情，全部表情都蕴含着对自己的畏惧，赵高满意了、满足了，还有什么比靠自己获得这一切更美好的？李斯、章邯、赵佗、刘邦……通通都靠不住，到头来还得靠自己。赵高不再犹豫，快步走到最高的龙椅处，他转过身来面对群臣座了下去。大殿上的时间仿佛被定格了一般，一片死寂，顷刻间所有人都明白发生了什么，沉默是所有人一致的态度。龙椅上的赵高突然觉得自己好像忘记了什么，他的视线越过群臣，越过大殿，飞向南方的山川。此时，天边一片云朵飘来，遮住了下午的阳光，宫殿逐渐进入阴凉之中。

●突然间，赵高感到头晕目眩，回过神来发现自己一人坐在阴沉沉的大殿上，群臣、侍卫已不见了踪影，远处的天空也不见了太阳，宫殿四周充满了凄厉的惨叫声；放眼望去，高台四周是已经死去的扶苏、蒙恬、李斯、冯去疾、胡亥……还保持着临死前的样子正不断向自己索命，更远处无数死去的士兵和百姓的亡魂不断化作厉鬼向秦王宫奔来，凄厉的鬼声惨叫充斥着天空和大地。赵高心胆俱裂，只想逃离此处，还未起身，脚下就传来巨大轰鸣将自己震倒，把赵高从眩晕中惊醒，他发现自己正歪坐在龙椅上，汉水已经湿透了背脊，看到殿上群臣、侍卫皆惊慌不定的神情，赵高强装镇定，让侍卫搀扶自己离开咸阳宫。

●赵高改变了称帝想法，他现在要保命，由于秦始皇的后裔已经死伤殆尽，于是赵高选择王室宗亲、秦始皇嬴政的弟弟、胡亥的叔叔、大秦的王爷——子婴，来当这个替死鬼。赵高逼迫子婴继秦王位，不称皇帝。王爷子婴名声较好，他清楚赵高想法，设计谋托病不去，赵高心急如焚，上门来请，宦官韩谈趁机刺死赵高，子婴立即带兵诛灭赵高三族，并在咸阳城公示。

●公元前207年十月，刘邦率兵入关，在位仅46天的秦王子婴投降刘邦，秦朝灭亡。
