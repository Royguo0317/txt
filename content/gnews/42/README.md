## 【信仰之声】如何用《圣经》看社会问题

我们基督徒在这个世界上是寄居的，是客旅。本不管世界上的事，但我们主耶稣基督吩咐我们不要作宗教徒，要做世界上的光和盐，要努力救人。哪怕多救一个人，也好在将来天父面前有一个交代，即我们不白占地土。我总是在说，属灵人参透万事，《圣经》就是我们的尺子，遇到困惑看不明白的事，我们要成为事件真相的解读专家，我们总能给人以答案，叫听见的人疑惑消失，心结解开。我们遇到极恶无比的事件和人，我们遇见前所未有的困难和苦难，我们不但自己能释怀，还能安慰同样需要安慰的人。我们在这个大变革时代，正是这样的人，总能给人积极、乐观、向上的信心和盼望。这不是我说的，乃是我主耶稣基督的话语告诉我们的。在我们来看，耶稣基督都说了些什么？这些话都是我们如何看待社会、看待世界问题的尺子。

经上说（和合本）：

**世界的败坏**

论到世界的败坏，经上记着说，“世界在神面前败，坏地上满了强暴”(创世记 6:11 )。对比当下的处境，何尝不是呢？究竟为何如此败坏呢？施洗约翰明确指出，世界败坏的缘由是这个世界上的人不承认耶稣基督是神。他说：**“**他在世界，世界也是藉着他造的，世界却不认识他。**”**(约翰福音 1:10 )这个“他” 指的就是基督。因为人们总是效法所谓成功者，和大有权势之人。使徒保罗说：“不要效法这个世界，只要心意更新而变化，叫你们察验何为神的善良、纯全、可喜悦的旨意。”（罗马书 12:2）主耶稣警告说：“这世界有祸了，因为将人绊倒；绊倒人的事是免不了的，但那绊倒人的有祸了！” (马太福音 18:7 ) “绊倒人”的事有大有小，从小的事情上看，我们会想到身边朋友的背叛和暗中欺骗以及一些小聪明使自己吃亏。大的绊倒人的事呢？政府对人民的背叛和欺骗，比如合法的股市操控等诸如此类。看哪！“神观看世界，见是败坏了；凡有血气的人在地上都败坏了行为。” (创世记 6:12 )这需要我们扪心自问，我们在这个世界能独善其身吗？就算你行，你圣洁，你也是带着原罪来的。你能保证自己没有跌倒、软弱、犯罪吗？这都是世界的败坏造成的恶果。

**神的公义**

但神是公义的，先知以赛亚在传达神的话时说：“我必因邪恶刑罚世界，因罪孽刑罚恶人，使骄傲人的狂妄止息，制伏强暴人的狂傲。”(以赛亚书 13:11 )因为神对世界和世界的人的刑罚是为了保护他所拣选的人。因为“为审判，是因这世界的王受了审判”(约翰福音 16:11 )。这世界的王是谁呢？就是那空中掌权者的灵撒旦。但撒旦不亲自攻击人，牠会在这世界上寻找牠的代理人，撒旦的代理人是谁呢？就是现今的“悖逆之子”——习近平。试想当下，中共成了这个世界最大的敌基督力量，他攻击了人类，我们知道，二战，希特勒只攻击了西半球，日本军国主义者只攻击了东半球，而中共习近平则攻击了整个人类。我们能想到的是，诺亚的时代，神用洪水洁净了世界，用方舟救赎了义人。“故此，当时的世界被水淹没就消灭了。” (彼得后书 3:6 )如今，神要用一种新型战法消灭、拔除、毁坏中共这个政权——撒旦的代言者。神的公义要再次拯救义人，之所以这场生物战一直在胶着状态，神的意思原是要让多人起来悔改，让恶人全都暴露出来。我们这样看待神的公义，你会发现神的大能超出我们所思所想。

**神的大能**

当我们论到神的大能时，耶利米先知说：“耶和华用能力创造大地，用智慧建立世界，用聪明铺张穹苍。”(耶利米书 51:15 )，想想我们居住的世界，神的能力、智慧、聪明在哪里呢？想想我们的星球在浩瀚的宇宙中如此渺小得几乎看不见，却又如此有规律地运行着；想想我们的智慧生活和我们人的聪明才智。你能说，这些都是人类与生俱来的吗？世界也绝非大爆炸出来的。

**生命意义**

想想我们生命的意义吧！耶稣基督说：“人若赚得全世界，赔上自己的生命，有什么益处呢？人还能拿什么换生命呢？ ”(马太福音 16:26、路加福音 9:25 ) 想想习近平吧，他就是蓝金黄了全世界、统治了人类，也总有一死，且死后下地狱，受那永火的煎熬。那时候，他还能去蓝金黄撒旦不成？所以，主耶稣说：“他们不属世界，正如我不属世界一样。” (约翰福音 17:16 )“他们”指的就是我们。耶稣继续说道：“我的国不属这世界；我的国若属这世界，我的臣仆必要争战，使我不至于被交给犹太人。只是我的国不属这世界。” (约翰福音 18:36 )主耶稣的意思乃是说，我们不要指望耶稣在世界上做我们的王，主乃是天国的万王之王。使徒保罗劝勉我们说：“不要效法这个世界，只要心意更新而变化，叫你们察验何为神的善良、纯全、可喜悦的旨意。”(罗马书 12:2 )

**苦难神学**

论到苦难，神说：“我不求你叫他们离开世界，只求你保守他们脱离那恶者。” (约翰福音 17:15 )尽管如此，耶稣基督继续强调：“因为那时必有大灾难，从世界的起头直到如今，没有这样的灾难，后来也必没有。” (马太福音 24:21 )弟兄姊妹们，我想说的是，眼下的全球大瘟疫仍然不是末日灾难，将会有更大的独裁者出现，再次制造人间大灾难。但我们依然选择与恶魔战斗到底。我们虽然受尽苦难，但我们不属于这个世界，因为肉体的消亡可能是一瞬间的事，但永生的生命绝非只是灵魂的延续。

**属灵战争**

当我们在属灵层面认识到苦难，一场持续的属灵争战将不可避免，“因我们并不是与属血气的争战，乃是与那些执政的、掌权的、管辖这幽暗世界的，以及天空属灵气的恶魔争战”（两“争战”原文都作“摔跤”）（以弗所书 6:12）。记住，这里提到的执政的、掌权的、属灵恶魔指的都是撒旦和撒旦的使者，地上的邪恶政府代理人无非都是撒旦的党羽及其替身。正如我们说，某人被恶魔附体了，而不是某人一样。因为那个人很大程度上就是我们自己，比如我们每天面临情欲的试探、权势和各样诱惑一样。我们甚至因为麻烦了一位弟兄、一位姐妹而深感内疚一样，我们无时无刻不在做属灵争战。

**信心依靠**

但我们完全依靠神，过信心生活。信心里包括我们的刚强，凡事上做一个有信、有智慧的人，保罗说：“人不可自欺。你们中间若有人在这世界自以为有智慧，倒不如变作愚拙，好成为有智慧的。” (哥林多前书 3:18 )这里的不要自欺，多么像我们爆料革命。七哥经常说，我们要唯真不破，但要强调有智慧，唯真不破不包括你把万事看做死的定律。比如耶和华神说亚当夏娃，不要吃那树上的果子，吃的日子必定死，结果没死。其实是死了，只是我们太把问题看成死的了，就像七哥说，川普一定会赢，赢了吗？煮熟的鸭子飞了。其实美国拜登政府仍在执行川普的政策，中共垮塌后，川普的政治遗产会无限放大，从这个角度看唯真不破，川普赢了。因此，“我们因着信，就知道诸世界是藉神的话造成的；这样，所看见的，并不是从显然之物造出来的。”(希伯来书 11:3 )

**圣洁生活**

神教导我们要圣洁，圣洁生活的根本问题是“不要爱世界和世界上的事。人若爱世界，爱父的心就不在他里面了” (约翰一书 2:15 )。这是肯定的，“因为，凡世界上的事，就像肉体的情欲、眼目的情欲，并今生的骄傲，都不是从父来的，乃是从世界来的” (约翰一书 2:16 )。问题是，我们几乎一睁眼就看见情欲和各样的诱惑并今生的骄傲。要如何保持圣洁，真的不是靠十条诫命，乃是靠我们里面的“心意更新而变化”（罗马书 12:2）。怎样的心意更新呢？不仅仅是我们受洗的那一刻，乃是我们要常常靠着祷告如此更新我们的心意，要想到我们的主已经胜了世界。不仅如此，老约翰亲切地说：“小子们哪，你们是属神的，并且胜了他们；因为那在你们里面的，比那在世界上的更大。” (约翰一书 4:4 )弟兄姊妹们，我们真的有多少时间想过这问题？难道遇到问题，我们认为我们的主不够强大吗？当我们利益受损的时候，在主动去寻求解决之前有没有在神面前安静下来想想自己是否圣洁？

**神的救赎**

耶稣基督告诉我们：“若有人听见我的话不遵守，我不审判他。我来本不是要审判世界，乃是要拯救世界。” (约翰福音 12:47 )这话要怎么理解呢？耶稣又对众人说：“我是世界的光。跟从我的，就不在黑暗里走，必要得着生命的光。”（约翰福音 8:12）主的意思很明显，新约时代主乃强调恩典，而不是戒律，就像未成年孩子，需要戒律约束他，成人后，父母就放手他去做事。因此，主说：“若有人听见我的话不遵守，我不审判他”，但强调说，“跟从我的，就不在黑暗里走。”也就是说，有末日审判在那里为多行不义而不悔改之人预备了地狱的大黑暗和火的熬煎。

**神的审判**

论到神的审判，不是末世的审判，乃是末日审判，但耶稣多次警告我们说：“民要攻打民，国要攻打国，多处必有饥荒、地震。”（马太福音 24:7）不过这些事是必须有的。神必不坐视不管，诗篇94篇2节说：“审判世界的主啊，求你挺身而立，使骄傲人受应得的报应。”谁是骄傲的人呢？当下人类面临的主要威胁就是中共政权，就是那“骄傲的人”。这里我们注意到，诗人向我们发出信息，即我们可以用祷告来恳求神实行公义的审判，铲除恶人。目前，我们爆料革命和整个西方文明阵营面对中共的超限战由防守正在准备采取进攻了。这种局面的反转，也离不开基督徒的祷告。我认为这也是客观事实。神不仅要审判这些敌基督者，并且到了末日审判时，要将基督里的假信徒和假师傅如同薅稗子一样，“将稗子薅出来用火焚烧，世界的末了也要如此”（马太福音 13:40）。因“他的判断是真实公义的；因他判断了那用淫行败坏世界的大淫妇，并且向淫妇讨流仆人血的罪，给他们伸冤”(启示录 19:2 )。而中共就是那大淫妇，在审判中共这件事上，主的审判必速速临到，要“向淫妇讨流仆人血的罪”，给我们伸冤，“因为主要在世上施行他的话，叫他的话都成全，速速地完结。”（罗马书 9:28）

这就是我们看世界和看人生的标杆，只要我们对所针对的问题去思考主的话，主的话一打开，就立刻在你里面说话，你就能看到神要告诉你的话，你就会释然，而不是像不信的人一样愁眉苦脸。我再说，神的话就是我们看待世界的尺子，神的灵就是我们打开神的话的钥匙。只要我们肯来到神面前，愿意寻求他的话，神必叫我们得着恩光。

（2021年5月18日写于东亚）
