# 重要事件

- [**05.13. BBC：IFJ国际记者联盟报告称北京捆绑“疫苗外交”重塑世界媒体格局**](https://www.bbc.com/zhongwen/simp/chinese-news-57067692)


过去一年多，美国等西方国家批评中共国在2019冠状病毒（COVID-19）疫情下试图操控国际舆论。总部设在比利时布鲁塞尔的国际记者联盟（IFJ）一份最新报告尝试证明中共国已经建立一套行之有效的国际媒体宣传体系，并试图揭露其如何运作。

IFJ 委托三位澳大利亚媒体学者对全球50个国家和地区的新闻工作者作调查采访，其证言显示中共国花大气力建立了一整套国际媒体基础布局，并在全球抗击疫情期间投入使用。

报告还指出，中共国把这些对外宣传工作和“疫苗外交”捆绑到“一带一路”倡议的投资项目中，以确保受惠国在联合国会议等场合支持中共国、赞扬中共国。

中共国外交部官员回应称， 此报告所述内容都是“美国一直以来的做法”，中共国旨在回应美国等“赤裸裸的舆论或者话语霸权”。



- [**05.13. timesofindia:关于冠状病毒起源的报告:中共国生物武器的 “实锤 “证据Reports on the origin of Coronavirus: ‘Smoking Gun’ proof of the Chinese biological weapon**](https://timesofindia.indiatimes.com/blogs/ChanakyaCode/reports-on-the-origin-of-coronavirus-smoking-gun-proof-of-the-chinese-biological-weapon/)


S-D-普拉丹曾担任过印度联合情报委员会主席、副国家安全顾问、特别工作组的主席。

最近有一些报告显示，冠状病毒是在武汉病毒研究所作为生物武器制造的。巴西总统Jair Bolsonaro对中国处理Covid-19的方式提出了严重质疑，S-D-普拉丹认为中共国可能在实验室里开发了这种病原体，并将其作为一种 “生物战 “传播，以获取经济利益。

但他并不是唯一这样做的人。美国最近也提出了这个理论。据英国《太阳报》援引《澳大利亚人报》首先发布的报道，美国国务院获得的 “重磅炸弹 “文件据说显示，中国人民解放军（PLA）的指挥官做出了险恶的预测，即下一次世界大战将用生物武器来进行。据报道，美国已经获得了这些文件，这些文件是由军事科学家和中共国高级公共卫生官员在2015年撰写的，是美国对COVID-19的起源进行调查的一部分。这份题为 “SARS的非自然起源和作为基因生物武器的人造病毒新物种 “的军事文件清楚地揭示了中共国军队在生物武器方面的计划。
该文件是解放军徐德忠将军的生物武器教科书，其中指出了生物战争的两个重要方面。首先，冻干微生物的能力使储存生物制剂并在攻击时将其气溶胶化成为可能。第二，生物武器攻击可以使 “敌人的医疗系统崩溃”。

这些文件的真实性已经得到了专家的验证。曾为美国、澳大利亚和加拿大政府工作的数字取证专家罗伯特-波特表示，该文件已在中共国互联网上被定位。另一位确认其真实性的人是闫丽梦博士，她曾在香港工作，并正在研究冠状病毒的起源。她曾表示冠状病毒是在武汉实验室制造的，当她开始指出这一事实时，她被警告不要透露她的研究结果。她后来逃离了中共国，在美国避难。她证实中共长期以来一直在研究这种生物武器。

最近，闫博士在接受Gaurav Sawant的采访时说，”这份文件是’实锤证据’，可以证明中共国有一个非传统生物武器的长期计划，中共国计划用它来征服整个世界。重要的是，她指出，这份文件是故意发布的，中共国政府知道这一点，这就是为什么他们在等待了几个小时后立即有了回应。她还说，她已经提供了 “足够坚实的科学证据和情报证据”，这是中共国无法否认的。自从她到达美国以来，她一直在不断地声明冠状病毒是在武汉实验室制造的。她曾说，COVID-19的基因组序列看起来像人类的指纹。

据报道，澳大利亚下议院外交事务特别委员会主席图根哈特说，这份文件引起了人们对一些为党的最高领导层提供建议的人的野心的重大关切。

此前，1989年起草《生物武器法》的美国弗朗西斯-博伊尔博士曾发表详细声明，承认2019年的武汉冠状病毒是中共国的进攻性生物战武器。博伊尔博士对生物武器非常熟悉。他曾起草了美国国内执行《生物武器公约》的立法，即1989年的《生物武器反恐怖主义法》，该法得到了美国国会两院的一致批准，并由当时的美国总统乔治-H-W-布什签署成为法律。博伊尔博士指出，在中共国武汉爆发的冠状病毒是从生物安全四级实验室（BSL-4）释放出来的。他认为该病毒具有潜在的致命性，是一种进攻性的生物战武器或经过基因改造的具有增益功能的两用生物战武器制剂，这就是中共国政府最初试图掩盖它的原因。

有可靠的报告称，那些来自武汉实验室的科学家最初提出这一事实时，被告知要保持沉默。在这些告密的医生中，有一位眼科医生李文亮。这八人因 “散布谣言 “而被警方逮捕，并被迫签署声明，撤回他们的主张。李医生后来在2月死于Covid-19。媒体报道还显示，2019年12月底和2020年1月初，研究新型冠状病毒的中国实验室收到了销毁其样本的命令。中共国一直反对独立调查。

虽然上述声明和报道表明，该病毒是在武汉病毒研究所产生的，但《环球时报》的一篇文章指责说，这些都是企图玷污中国的形象。

自从大流行病出现后，北京更加依赖虚假信息和海外影响行动。北京利用大量的虚假社交媒体账户来推送其信息。ProPublica曾追踪了超过1万个疑似虚假推特账户，这些账户参与了与中共国政府/中国共产党有联系的协调影响力活动。其中有来自世界各地的用户的黑客账户，他们发布有关冠状病毒爆发的中共国宣传和虚假信息。中共国的影响行动是为了捕捉思想，推动个人投射中共国的宣传。为此收集了大量数据，并精心策划了各种方案。个人被大量的宣传轰炸，以至于他们的独立思考受到影响，他们无法进行逻辑思考。

可以补充的是，很难区分实验室制造的病毒和天然病毒，因为两者的过程是一样的。在实验室中，科学家可以通过调整来加快传播速度，但这也可能发生在非实验室病毒中。使用人工智能是可能的。

印度和其他一些国家一样，已经意识到我们周边的国家正在努力拥有这种能力。在《核理论》中，报复的触发点被扩大了，包括 “用生物或化学武器对印度或印度在任何地方的部队进行重大攻击”。这被认为是必要的，因为在印度周边地区正在对士兵进行化学和生物战的培训。纳入核理论的目的是为了震慑对手。这似乎并不奏效。在目前的条件下，印度必须采取必要的措施，为面对这样的挑战做好准备。这就要求建立一个强大的医疗保健基础设施，并拥有BSL-4实验室，以获得必要的疫苗和危险病毒的药物。
