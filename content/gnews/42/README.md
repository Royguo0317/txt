印度快报1月27日报道，来自伊朗国家电视台的消息，伊朗和塔利班官员周三在德黑兰会面，指责美国推动阿富汗战争。

伊朗最高国家安全委员会(Supreme National Security Council)秘书阿里·沙姆卡尼(Ali Shamkhani)告诉到访的塔利班政治领袖阿卜杜勒·加尼·巴拉达(Abdul Ghani Baradar)，美国继续在挑动阿富汗的战争。

沙姆卡尼指出美国试图将这个国家的不安全和不稳定归咎于阿富汗的个别组织。

他说，“美国的战略支持阿富汗不同政治派别之间战争和流血的继续”。

本月早些时候，塔利班代表和阿富汗政府在卡塔尔恢复了和平谈判，旨在结束数十年的冲突。去年2月，美国与塔利班签署了一项和平协议，并在本月实现了将驻阿富汗部队人数减少至约2500人的目标。

但是由于最近暴力事件的激增，双方相互指责。塔利班政治领袖巴拉达批评美国违反了其对去年2月协议的承诺。他表示，“我们不相信美国，将打击任何为美国服务的雇佣军组织。”

伊朗和阿富汗有大约945公里的共同边界，伊朗将美军在邻国阿富汗和伊拉克的存在视为家门口的威胁，并经常要求美军撤离。伊朗和塔利班官员偶尔会举行会晤，德黑兰说会谈的目的是帮助促进阿富汗内部的对话。

对于伊朗和塔利班的指责，美国没有立即对此发表评论。

伊朗，作为中共的“小弟”，继中共邀请塔利班访问北京，德黑兰也定期与塔利班会谈，伊朗、中共公然与恐怖组织为伍破坏世界和平。伊朗一直试图发展核武器成为世界的不安定分子，而中共，对新疆维吾尔族人犯下了种族灭绝和反人类罪行，更是世界上最大的犯罪集团。

在美国新政府刚上任期间，伊朗频频挑战美国，中共对台湾军事入侵，中共携手“伊朗小弟”给拜登政府施压，想在世界格局变化之际谋取新的生存和发展空间，这是共产主义集权与自由民主的战争，期待拜登政府的回应。
