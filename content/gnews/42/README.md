## 中共外交部回应美国突击检查入境共产党员身份

整撰：文錦；审核：WLQF

据中共媒体《凤凰网》报道，截至11月11日，中方共有21艘轮船在美港口停靠时遭美突击检查。外交部发言人华春莹在11月 30日的例行记者会上证实此事。

该报道称，“近期美国有关部门执法人员多次突击检查中方航运企业人员的共产党员身份。截至11月11日，中方共有21艘轮船在美港口停靠时遭美突击检查；9月以来中方航空公司共有16架次赴美航班机组人员遭问询、盘查。据悉，美方盘查有时长达数个小时，反复纠缠共产党员身份，连入党原因也要盘问。”
