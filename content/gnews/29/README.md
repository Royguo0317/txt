自去年**反修例**風波後，政府打壓香港傳媒的「無形之手」越見明顯，傳媒業界近日頻頻「地震」，各大電視台均有高層人事變動。
**有線新聞大地震！ 前亞視副總裁陳興昌「空降」與李臻齊任總監**
     有線(I-CABLE)新聞部有人事變動。去年「回巢」有線新聞部出任執行董事的馮德雄，8月10日將轉任顧問一職，並改由國際財經台主管謝燕娜（Edna）掌管新聞部。消息更指，新聞總監亦會一開二，改由兩人擔任，分別是前亞視高層陳興昌及現任有線寬頻內容業務總經理李臻。有學者指，有線新聞撤走資深傳媒人，質疑換人是以政治忠誠作考慮。
     近月屢傳出要「換人」的有線新聞部，今日管理層終向員工交代。有線寬頻財務總裁郭子健今向新聞部員工發內部通告，指執行董事馮德雄下周一將調任顧問，但未有交代原因。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/29102437/image003-6.png)圖片：蘋果日報
**新聞部高層換班不足兩周 有線炒工程部3名元老級主管**
     今年八月有線新聞部高層大換班後，即向資深員工「開刀」。消息指，一直協調新聞部工作的3名工程部資深主管今日被解僱，即時生效。據悉，三人在有線工作逾20年，幾乎是開台的元老級員工，主要負責網絡接駁及直播訊號等，亦會協助新聞部工作，隨記者一同外出採訪。有記者知悉三人被炒，忍不住落淚。據悉，管理層以節省開支為由解僱。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/29102454/image005-2.png)圖片：立場新聞
**無線前主管陳鐵彪將空降掌 NOW 新聞　新聞部召員工大會　高層稱未來難做但會堅守宗旨**
     NOW 新聞部主管、高級副總裁張志剛在6 月 23 日離任，轉為高級顧問，準備 8 月退休；該台日前宣佈將由無綫新聞前中國組主管陳鐵彪接替張志剛。另據了解，現時於無綫新聞任職採訪主任的吳宜婷，及高級監製李偉亮亦會跟隨「過檔」。
     NOW 新聞部舉行員工大會，會上消息指新聞部現有管理層不會離開，新聞部亦會按現時編採原則採訪及製作，但有主管級人員直言未來「會難做」。有與會的編採人員表示，會上最多提及是「見步行步」，一切有待陳鐵彪下周二上任後再觀察。張志剛向全體新聞部員工發電郵，表示今天要離開難免不捨，亦直指在現時香港時局和傳媒經營環境下，「大家今後工作不會輕鬆」。
      員工大會由原本被認為會接替張志剛的 NOW 新聞副總裁鄭麗矜主持，參與者大部份係採訪港聞員工。據知在會上有主管級人員指「未來條路好難行」，但會堅守新聞宗旨、報道事實。主管人員又多次表示「 見步行步」，又指新聞部高層會「頂住」，會堅持原則底線，秉持專業精神、不偏不倚。
      早前已傳出會接掌《Now 新聞》編輯組的《無綫新聞》前總編輯主任葉惠民，9月28日履新，而在葉惠民空降後「坐歪」的現任編輯組總編輯陳智燊，今日因身體不適無上班。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/29052423/thestandnews_120204893_1775174292638549_8865332390694280780_n.jpg)圖片：立場新聞
**商經局曾要求港台懲處《頭條新聞》製作人員**
     港台節目《頭條新聞》及《左右紅藍綠》分別被指「侮警」及批評警隊等，被通訊局發出警告，節目隨後被停播。《頭條新聞》被發警告後，商經局即發聲明，指港台作為公共廣播機構，接連違反多項守則條文，情況非常嚴重，下令港台道歉，並懲處涉及疏忽和失誤的製作人員。
     至今年5月底，商經局宣布成立專責小組檢討港台的管治及管理。在港台任職超過30年的署理副廣播處長（節目）陳敏娟，在小組成立3天後以健康理由遞信請辭。港台早前有消息指，邱騰華堅持要港台「交人」。電視部製作班底都是陳敏娟屬下的人，承受壓力最大。但港台發言人否認「交人」一說。
     商經局年中更成立專責小組「檢討」港台，民間亦有建制組織屢次要求「整頓港台」，及解僱部份記者或主持。
     有港台中人質疑，今次向利君雅「開刀」是商經局暗中出手施壓。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/29102416/photo_2020-09-29_08-10-23.jpg)圖片：TG反送中文宣谷
**港台中人：商經局一直向港台管理層施壓**
     多個港台節目包括《左右紅藍綠》、《頭條新聞》以至《鏗鏘集》，近年都屢被建制派及政府發炮批評內容偏頗。港台中人指，政府可借今次對利君雅的行動，威嚇其他港台員工。
     面對中共的種種打壓，香港採訪前線仍沒布放棄，繼續堅持「與真相同行」。而從各大傳媒以報導香港抗爭的主題屢屢在國際上獲得大獎，就可見他們的專業得到了國際同行的認可。《鏗鏘集》等節目亦未有因壓力，詳細報道7.21及反送中示威等敏感議題，獲得多個新聞獎項。
