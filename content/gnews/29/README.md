11.疫苗開發「不會草率申請批准」9  9家大型製藥企業聯合聲明。
![]()

進行新型冠狀病毒疫苗開發的9家世界生物· 。 製藥企業於8日發表了不同尋常的聯合聲明,稱不會草率申請批准疫苗。 並認為申請需要安全性及效果證明。 雖然疫苗作為平息感染擴大的王牌而備受期待,但是此舉意在打消人們對於因政治意圖導致批准程式歪曲的擔憂。

在疫苗開發中已開始最終大規模臨床試驗的英國阿斯利康製藥與美國輝瑞等9家公司聯名發表了聲明。 強調稱,需要基於美國食品藥品監督管理局(FDA)等專業管制當局的指導,證明安全性及效果。 並發表宣言「臨床試驗以及製造過程將遵循科學、倫理的高標準」。

世界衛生組織(WHO)表示,開發中的新型冠狀病毒的候補疫苗約有180種,其中34種處於臨床試驗階段。 在美國,面臨11月總統選舉的總統· 。 特朗普,屢次暗示將在選舉前使疫苗投入實際使用,有人擔心有可能會因政治意圖導致批准程式歪曲。

與此問題相關,阿斯利康製藥於8日表明,將暫時中斷已進入最終階段的臨床試驗。 其新聞負責人向媒體發表聲明稱「有可能會引起原因不明的疾病」。 據《紐約時報》報導,根據相關人士談話,英國臨床試驗的參與者被診斷出會引起疼痛及麻痹的橫貫性脊髓炎。 據悉,尚不清楚是否與疫苗接種有關。 在國內外加緊進行開發的情況下,要求謹慎的呼聲不斷增強。

以下是最近幾個小時內最新的冠狀病毒發展概況:

- 伍德沃德的書說,特朗普知道科維奇是致命的,但想**“**淡化它**“**。 [根據美國最受尊敬的調查記者之一的採訪,唐納德](https://www.theguardian.com/us-news/2020/sep/09/trump-bob-woodward-book-rage-coronavirus)**·** 。 特朗普(**Donald Trump**)知道**2**月份致命的冠狀病毒威脅的嚴重性,但有意通過**「**淡化**」**來誤導公眾。
- 隨著病例激增,法國的合租住院人數達到一個月高位。 法國週三每天發生的新的**Covid-19**感染每天在六天內連續第三次上升,超過了**8,500**,該疾病的傳播速度是自該國出現以來最快的。 攜帶病毒被送往醫院的人數也增加了**43**人**–**連續第**11**天增加,達到一個月的新高**5,003**。
- 鮑裡斯**·** 。 詹森(**Boris Johnson**)寄希望於**1000**億英鎊的**「**月球計劃**」**以避免第二次鎖定。 根據洩露的官方檔([**「**月球行動**」**計劃),英國首相 鮑裡斯](https://www.theguardian.com/world/2020/sep/09/boris-johnson-pinning-hopes-on-covid-testing-moonshot-leaked-papers-show)**·** 。 詹森(**Boris Johnson**)相信,大規模測試計劃是**「**我們避免在接種疫苗之前進行第二次國家封鎖的唯一希望[**」**](https://www.theguardian.com/world/2020/sep/09/boris-johnson-pinning-hopes-on-covid-testing-moonshot-leaked-papers-show)。


- 葡萄牙報告自**4**月**20**日以來每日最高病例。 葡萄牙報告了**646**例新的冠狀病毒病例。 該國衛生部長賈米拉**·** 。 馬德拉**(Jamila Madeira**)表示,傳播主要發生在家庭家庭中。
- 德國建議不要前往歐洲目的地的遊客。 德國外交部已建議遊客,由於冠狀病毒感染率高,因此請勿前往包括布拉格,日內瓦,杜布羅夫尼克和科西嘉島等歐洲目的地。
- 馬克**·** 。 紮克伯格(**Mark Zuckerberg**)說,儘管**Covid**擔心,**Facebook**不會刪除反疫苗帖子。 這位**Facebook**負責人[表示](https://www.theguardian.com/technology/2020/sep/09/mark-zuckerberg-facebook-not-rightwing-echo-chamber),即使領先的病毒專家對今年下半年或明年年初可以進行**Covid-19**疫苗接種表示謹慎樂觀,他也不會刪除抗[**Vaxxer**帖子](https://www.theguardian.com/technology/2020/sep/09/mark-zuckerberg-facebook-not-rightwing-echo-chamber)。
- 荷蘭每日病例自**4**月以來最高。 在過去的**24**小時內,荷蘭新登記的冠狀病毒病例數上升至**1,140**,是自**4**月以來的最高每日總數。
- 希臘呼籲英國審查**「**不公平**」**的隔離決定。 希臘旅遊部長表示,英國政府應審查其隔離從該國七個最受歡迎的島嶼出入境的人隔離的決定,並將此舉標記為[**“**不幸和不公平**“**](https://www.theguardian.com/world/2020/sep/09/greece-calls-on-uk-to-review-unfair-quarantine-decision)。


- 專家說,我們不知道**Covid-19**疫苗的來源。 英國領先的醫學專家之一警告說,沒人能知道第一種針對**Covid-19**的安全有效疫苗將來自哪裡,因為牛津大學的領先者試驗被擱置了。
- 西爾維奧**·** 。 貝盧斯科尼(**Silvio Berlusconi**)說,他正在與冠狀病毒的**「**致命**」**案件作鬥爭。 這位前義大利總理曾表示,在醫院接受**Covid-19**病毒治療的醫生告訴他,他的病毒載量嚴重程度為**「**第一**」**,並且他正在努力擺脫[**「**重症**」**疾病。](https://www.theguardian.com/world/2020/sep/09/silvio-berlusconi-fighting-hellish-case-coronavirus-italy)
