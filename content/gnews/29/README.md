# **面对风暴：武汉居民回想从疫情最初封城一年来所付出的代价**

2020年1月23日，冠状病毒大流行起源地的武汉进入了长达数月的全面封城。一年过去了，当地居民反思了这座城市的变化。

[!\[\]()!\[\](https://spark.adobe.com/page/I9YHF0QoqXOXZ/images/53a4b2f9-4bec-4cae-b2b2-73c851ae36aa.jpg?asset_id=90c155f0-73ca-4ad9-a6ab-7cdd61b77a22&amp;img_etag=%22f7c3b7c895a05d3c03f691605306d0ac%22&amp;size=2560)](https://spark.adobe.com/page/I9YHF0QoqXOXZ/images/53a4b2f9-4bec-4cae-b2b2-73c851ae36aa.jpg?asset_id=90c155f0-73ca-4ad9-a6ab-7cdd61b77a22&amp;img_etag=%22f7c3b7c895a05d3c03f691605306d0ac%22&amp;size=1024)

*武汉居民们参加了一个关于武汉抗击冠状病毒的展览*

在武汉一个平常的早晨，地铁挤满了赶着去工作的人们。除了他们脸上的口罩，似乎很明显，一年前进入长达数月封城的这座城市现在已经从那种超现实的经历中恢复了过来。

“人们现在正在通过手机播放音乐或在地铁上大声说话，这与一年前的武汉完全不同，”现年31岁的武汉市英语老师梁(Liang)说。“我仍然记得，一年前武汉全面封城的前一天，地铁一片死寂。没有人在地铁上发出任何声音，空中有一种强烈的恐惧感。”

对一些人而言，1月23日将永远是记起来令人痛苦的一天。武汉居民张海(Zhang Hai)回忆说，那天是他的父亲在接受整形外科手术后在当地医院感染冠状病毒的日子。一周后，他父亲去世了。

张(Zhang)对《独立报》（The Independent）说： “我仍然相信武汉市政府在宣布全面封锁这座城市的计划之前几周就知道了这种病毒的危险。他们在计划生效前几天宣布该计划，允许超过50万人离开武汉。我一直在想，他们中有多少人已经感染了冠状病毒。”

封城生效后，武汉的公共交通全部暂停，市民被禁止离开家。当地医院对出现感染病毒症状的病人不堪重负，许多人被要求回家，因为医院没有足够的空间使他们在医院接受治疗。

经过数月的全面封城，武汉的疫情得到了控制，政府于2020年4月结束了封城。但是，从最初的(疫情)爆发开始延伸出来的恐惧，对武汉许多人来说，仍然挥之不去。

“在过去的一年中，戴口罩已成为日常要务，我们必须在去的几乎任何地方都扫描健康码，”梁(Liang)告诉《独立报》，“尽管一些购物中心在夏季稍微放宽了扫描健康码的规定，但由于过去几周中共国本地病例再次增加，他们开始再次执行这个规定。”

对武汉的许多居民来说，长达数月的与世隔绝，使他们能够为在中共国发生的任何冠状病毒复发做好准备。“我们知道当疫情再次回到武汉时会发生什么，我们也知道政府会期望我们做什么，” 32岁的武汉小贩陈(Chen)说，“我认为去年封城的最大影响是使我们为将来的疫情或其它流行病做好准备。”

为了“庆祝”其成功遏制冠状病毒的爆发，中共国政府在武汉举办了一次大型展览。展厅里摆满了身穿危险品防护服的医务人员模型和医院病床的装置。展览的另一个明显特征是习近平主席的大幅肖像，并附有赞扬他“果断决策”的文字，使中共国成功地遏制了疫情。

但是，武汉的一些人发现很难接受政府试图在中共国传播的庆祝口吻。梁(Liang)说：“如今，在武汉每个街区的入口处，政府都将进行宣传，突显中共国政府如何克服这一疫情。我发现很难简单地接受政府试图放大的所有这些信息，因为我们都知道中共国公民在整个过程中不得不付出的代价。”

张海仍然认为，武汉市政府官员因在疫情爆发初期隐瞒有关疫情的信息而应受到起诉，因为如果他们在封城前几周披露了疫情的整个规模，他认为可能挽救武汉许多人的生命。

*戴口罩的居民排队购买奶茶*

张说：“中共国政府官员需要知道在疫情期间应优先考虑公民的生命。如果没有人因隐瞒有关疫情的信息而受到惩罚，我认为当中共国遭受到另一场疫情时，类似的事情仍然会发生。”

随着中共国河北省、黑龙江省和吉林省的本地病例数量开始增加，武汉市的人们也正在加强预防疫情的措施。陈说：“现在要求公民几乎在每个地方都扫描他们的健康码，许多社区现在已经进入半封闭模式。我们都准备好了再次封城。”

在武汉封城一周年之际，一些人认为武汉人已经选择搁置因封城而产生的负面记忆。“我们会被中共国其它地方发生的类似事情唤起记忆，但我们中的许多人都尽量不把它放在心上。”

但是，对于那些在疫情中丧失亲人的人来说，还有一个未完成的使命：要求政府正式道歉。“无论政府多少次试图阻止我要求他们道歉的努力，我都会不断寻找方法为已故的父亲以及因政府在疫情爆发初期的掩盖而丧生的许多其他无辜者大声疾呼，”张(Zhang)告诉《独立报》。

“他们可以试图指控我犯罪或试图将我关起来，但这不会阻止我为父亲寻求正义。我已经为最坏的情况做好了准备，因此他们可以随时来找我麻烦。”
