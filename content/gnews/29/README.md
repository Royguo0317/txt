## 【热点播报】中共宣布无限期暂停中澳战略经济对话机制，中澳对峙升级
` 纽约香草山MOS02` [轉載自GNews](https://gnews.org/zh-hans/1179313/)

作者：纽约香草山农场 乱谈秦

![This image has an empty alt attribute; its file name is GNEW-GTV-MOS-LOGO-2-109.jpg]()![This image has an empty alt attribute; its file name is GNEW-GTV-MOS-LOGO-2-109.jpg](https://gnews.org/wp-content/uploads/2021/01/GNEW-GTV-MOS-LOGO-2-109.jpg)



2021年5月6日，中共国国家发展和改革委员会（下称发改委）在其官网发布中英文声明，“自即日起，无限期暂停国家发展改革委与澳联邦政府相关部门共同牵头的中澳战略经济对话机制下一切活动”。声明称，做出此决定是由于“澳大利亚联邦政府某些人士基于冷战思维和意识形态偏见，推出系列干扰破坏两国正常交流合作的举措”。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/06092131/MOS2642-%E3%80%90%E7%83%AD%E7%82%B9%E6%92%AD%E6%8A%A5%E3%80%91%E4%B8%AD%E5%85%B1%E5%AE%A3%E5%B8%83%E6%97%A0%E9%99%90%E6%9C%9F%E6%9A%82%E5%81%9C%E4%B8%AD%E6%BE%B3%E6%88%98%E7%95%A5%E7%BB%8F%E6%B5%8E%E5%AF%B9%E8%AF%9D%E6%9C%BA%E5%88%B6%EF%BC%8C%E4%B8%AD%E6%BE%B3%E5%AF%B9%E5%B3%99%E5%8D%87%E7%BA%A7-1.jpg)图源：中共国发改委官网截图



引发中共此举的导火索是2021年4月21日，澳大利亚外交部长佩恩（Payne）在一份声明中宣布，中国与澳大利亚维多利亚州此前签署的“一带一路”备忘录和框架协议已被该国联邦政府取消。佩恩声称，“这一协议不符合澳大利亚的外交政策”。

当天，中共国驻澳大利亚使馆在官网发文回应，对这一行为“表示强烈不满和坚决反对”，指出**“澳方此举是针对中方的又一无理挑衅行径”，**澳方“必将搬起石头砸自己的脚”。

另一个导火索则是，今年3月底出任澳大利亚国防部长的对华鹰派彼得·达顿，他于4月明确表示，正在慎重考虑中共国国有企业岚桥集团2015年中标的对澳大利亚达尔文港的99年租约。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/06092144/MOS2642-%E3%80%90%E7%83%AD%E7%82%B9%E6%92%AD%E6%8A%A5%E3%80%91%E4%B8%AD%E5%85%B1%E5%AE%A3%E5%B8%83%E6%97%A0%E9%99%90%E6%9C%9F%E6%9A%82%E5%81%9C%E4%B8%AD%E6%BE%B3%E6%88%98%E7%95%A5%E7%BB%8F%E6%B5%8E%E5%AF%B9%E8%AF%9D%E6%9C%BA%E5%88%B6%EF%BC%8C%E4%B8%AD%E6%BE%B3%E5%AF%B9%E5%B3%99%E5%8D%87%E7%BA%A7-2.jpg)“迈特兰”号驶入达尔文港。图源：ABC转自澳国防部

中共国声明中所说的“中澳战略经济对话”，始于2014年6月，时任澳国库部长霍基、贸易和投资部长罗布在北京与时任中共国发改委主任徐绍史举行了首次对话。同年11月，中共国国家主席习近平访澳，与时任澳大利亚总理阿博特共同宣布实质性结束中澳自贸协定谈判。2015年6月，中共国商务部长高虎城与澳贸易和投资部长罗布在堪培拉分别代表两国政府正式签署中澳自贸协定。8月，第二次中澳战略经济对话在澳大利亚堪培拉举行。12月，中澳自贸协定正式生效。最近一次中澳战略经济对话是2017年9月，时任澳国库部长莫里森、贸易和投资部长乔博前往北京出席“一带一路”国际合作高峰论坛，并与中共国发改委主任何立峰举行了第三次对话。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/06092154/MOS2642-%E3%80%90%E7%83%AD%E7%82%B9%E6%92%AD%E6%8A%A5%E3%80%91%E4%B8%AD%E5%85%B1%E5%AE%A3%E5%B8%83%E6%97%A0%E9%99%90%E6%9C%9F%E6%9A%82%E5%81%9C%E4%B8%AD%E6%BE%B3%E6%88%98%E7%95%A5%E7%BB%8F%E6%B5%8E%E5%AF%B9%E8%AF%9D%E6%9C%BA%E5%88%B6%EF%BC%8C%E4%B8%AD%E6%BE%B3%E5%AF%B9%E5%B3%99%E5%8D%87%E7%BA%A7-3.jpg)图源：中共国发改委官网

此后，中澳关系不断恶化。2018年8月，澳洲成为第一个公开禁止在5G网络中使用华为设备的国家。2020年初，中共病毒疫情在全球爆发，澳大利亚外交部长佩恩于2020年4月呼吁对新冠疫情展开国际独立溯源调查，两国关系更是急转直下。此举招致中共国的贸易反击，波及范围包括大麦、原木、煤炭、龙虾、葡萄酒等。

中共国对澳大利亚贸易制裁，柿子专拣软的捏，意欲“杀鸡儆猴”，看看效果如何。据媒体报道，2021年前三个月，澳洲对华出口创下新纪录，铁矿石的需求强劲。根据中国海关公布的数据，今年第一季度澳大利亚对华出口总值增长20.7%，达到337亿美元（443.6亿澳元），其中3月份的出口额为132亿美元，创下历史最高月度纪录。一季度中国对澳出口额为141亿美元，增长50.5%。

目前最新消息，针对中共国“无限期暂停”的声明，中共外交部在今天例行记者会上表示，“澳方应当对此承担全部责任”。而澳大利亚贸易部长丹·特汉回应称，中方的决定“令人失望”，澳大利亚仍“对与中国对话持开放态度”。
