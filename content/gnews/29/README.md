**西班牙2021年4月3日电/西喜社——**

他接种了疫苗，但还是被感染了。
阿根廷总统阿尔贝托-费尔南德斯（Alberto Fernández）证实，在他62岁生日当天，病毒检测结果是阳性。这个消息对民众和政府来说是一盆冷水。
费尔南德斯在感觉身体不适，服用止痛药后，接受了新的测试。”我想告诉你，今天晚上，在呈现了37.3的发烧和轻微的头痛后，我进行了抗原检测，其结果是阳性。”他在自己的推特账号上写道。
接着，阿根廷总统又发了一条报平安的消息：”让所有人知道，我的身体很好，虽然生日这天我不希望有这样的消息，但我的精神很好。我从心底里感谢大家对我的生日祝福”。
阿尔贝托-费尔南德斯和他的副总统克里斯蒂娜-费尔南德斯（Cristina Fernández）一起，是第一批接受俄罗斯产的斯普特尼克五号疫苗的人。1月21日接种了第一剂，一周后接种第二剂。这除了保护之外，还向社会传递了一个隐性信息：俄罗斯疫苗没有风险。在当时的民众中，存在着对俄罗斯疫苗的不信任，以他为榜样，是克服不信任的一种方式。
总统府联系了近几日所有与总统关系密切的人，开始进行检测。根据Infobae的说法，其中包括外交部长，马尔维纳斯区的总书记，总统发言人和总统的合作伙伴，他们的测试结果为阴性。
阿尔贝托-费尔南德斯今天将与布宜诺斯艾利斯市政府首脑奥拉西奥-罗德里格斯-拉雷塔以远程方式举行会议。
在阿根廷已经接种第一剂疫苗的有3,440,284人，相当于人口的7.6%。第二剂，682,868人，仅占1.5%。据官方统计，已有56,023人死于中共病毒，如西班牙的实际数字会更高。疫情发生以来，感染人数为2,373,153（2百37万多）人，康复人数为2,121,954人。
