# **美国国务院警告说，中共国正在进行冷战以来“最大规模的核武器库扩张”**

## **总统军备控制问题特使说，产量增加三倍可能违反军备条约**

[!\[\]()!\[\](https://spark.adobe.com/page/G7fodvuHBTi48/images/c6311220-c5bd-4522-81b2-cf08842545f6.jpg?asset_id=dfc2ee95-57b9-4c93-a220-e87ef48ffc41&amp;img_etag=%2295c06e4041a185a6800ad959bc4e622d%22&amp;size=2560)](https://spark.adobe.com/page/G7fodvuHBTi48/images/c6311220-c5bd-4522-81b2-cf08842545f6.jpg?asset_id=dfc2ee95-57b9-4c93-a220-e87ef48ffc41&amp;img_etag=%2295c06e4041a185a6800ad959bc4e622d%22&amp;size=1024)

中共国的DF-26弹道导弹正在使美国感到担忧，因其可以在很远的距离外发射，并且其精准度足以命中移动的船只（图：美联社，拍摄：Andy Wong）

根据美国国务院发布的一项情报评估，过去十年来，中共国快速扩大了其核导弹和常规导弹的军事力量，将其弹道导弹生产能力几乎扩大了三倍，并部署了多种核导弹及常规导弹系统。

美国国务院还于周四通知国会，他们根据北京当局拒绝加入美国和俄罗斯进行的核武器削减谈判，认为其已接近违反《不扩张核武器条约》(NPT)。

负责军备控制的总统特使马歇尔·比林斯利(Marshall Billingslea)说，“正如这封信所写，中共国似乎没有遵守《不扩张核武器条约》第六条规定的义务。下一届政府的当务之急是继续采取各种外交、经济和防御措施，将中共国带回谈判桌前。”

这个警告经由一信函传递于参议院外交关系委员会，并指出了新一轮与俄罗斯的《削减战略武器条约》(START)军备控制谈判的现状。

比林斯利先生在1月14日的信中说，“中共国的例子是我们自冷战以来见证到最大扩张的单一核武器库。”

一份由情报数据构成的图表在周四由国务院武器控制、核查与合规局公开。其结论是，中共国的导弹生产能力自2010增加了180%。这份图表最早出现于三周前美国向北约盟国做出的简报之中。

此外，情报显示，2010年，中共国80%的导弹是常规武器，而只有20%的导弹装备了核弹头。相比之下，现在中共国有40%的核导弹，剩余的60%为常规武器。习近平主席2012年上台执政后，导弹力量的扩张加速了。

从2010年到2015年，导弹产量增加了30%，从2015年到2020年则呈指数级增长。2020年的成长率比十年前高出180%。

负责军备控制和国际安全事务的比林斯利先生说：“更加了解中共国导弹力量的规模和射程范围，和其迅速扩张，以及采取减轻其对地区和全球稳定危险的行动必要性，是美国人民和国际社会的当务之急。”

中共国官员长期抗拒着美国要求其加入军备控制谈判的压力。他们争辩说，北京的核武器库是为了自卫，并在华盛顿和莫斯科当局的导弹库存面前相形见绌。

中共国常驻联合国副代表耿爽于十月告诉联合国大会，“中共国的核力量与美俄完全不在一个量级，要求中共国参加三边军控谈判不公平、不合理、不可行……。” “美国意在为其拒绝履行核裁军特殊优先责任制造借口，为其谋求自我松绑、建立绝对军事优势寻找理由。”

## **导弹疑云**

[!\[\]()!\[\](https://spark.adobe.com/page/G7fodvuHBTi48/images/17852f95-ebd2-42b7-9457-abb8591a457e.jpg?asset_id=7bf0561f-be08-464b-b76e-2ccd2c2f282f&amp;img_etag=%22fd4488b03d80838265da29c3a497d38e%22&amp;size=2560)](https://spark.adobe.com/page/G7fodvuHBTi48/images/17852f95-ebd2-42b7-9457-abb8591a457e.jpg?asset_id=7bf0561f-be08-464b-b76e-2ccd2c2f282f&amp;img_etag=%22fd4488b03d80838265da29c3a497d38e%22&amp;size=1024)

五角大楼警告中国正在接近核武器集结的里程碑 Copyright © BloombergQuint.

美国最关切的导弹问题为中共国的“航母杀手”DF-21和DF-26导弹。这种独特的导弹可以远距离发射，并有对海上移动的船只有足够的攻击精度。

由于DF-26的射程能够触及美国在关岛这个南太平洋岛上部署的主要军事枢纽，中共国将其称为“关岛杀手”。解放军八月在南海举行的大规模军事演习中试射了四枚DF-21和至少一枚DF-26。

五角大楼官员在那时说：“导弹打到了海南岛与西沙群岛间的南海地区。”

美国军事计划者还关切着中共国的新型超音速DF-17导弹。这是一种可操控的滑翔机，能以极高的速度在高层大气和外太空之间飞行。

DF-17针对穿透日益复杂的美国导弹防御系统而设计，例如终端高空区域防御系统(THAAD)和保护美国免受远距离攻击的地面中程防御系统(Ground-Based Midcourse Defense)。

五角大楼在其关于中共国军方的年度报告中表示，正在开发的远程导弹“将大大改善其可负载核武器的导弹军力，并由此暗示核弹头产量的增加”。现在部署于公路和铁路移动发射器中包括DF-41这样的多弹头导弹。

除了产量的提高，中共国去年还测试了250枚导弹，多于2019年的测试量。而在2019年北京当局进行了比其他所有国家总和还多的导弹试验。

尽管因中共病毒大瘟疫的限制，中共飞行测试仍达到数百人次，相比起来，却限制了美军导弹测试的能力。

此外，中共国还增加了弹道导弹潜艇的生产。五角大楼表示，这明显为北京当局提供了用核导弹打击美国本土的能力。

“我们看到，中共国正在迅速增加其承载核弹头的导弹力量，包括远程、潜射和中短程弹道导弹和巡航导弹。”比林斯利告诉《华盛顿时报》，“在此之前，我们已经强调了中共国大量增加的核武器生产基地。”

中共国导弹建造正被美国所重视。美国正努力追赶中共国对于中程导弹的大规模部署。几十年来，中共国制造了18种中程弹道导弹，射程从1864到3418英里不等。美国和俄罗斯因条约而被禁止制造类似的导弹。

据估计，中共国目前的导弹火力包括约2000枚弹道导弹和数百枚地面巡航导弹，其中包括约1200枚中程导弹和数百枚空对地巡航导弹。

川普政府在宣布俄罗斯部署了违反1987年协议的陆基巡航导弹后，于2019年宣布退出《中程导弹条约》。

比林斯利先生告诉《华盛顿时报》，川普政府已采取行动，以支援陆军和海军陆战队部署地面发射能力，并与亚洲和北约盟国商讨如何最有效阻止中共国和俄罗斯的侵略。

## **破坏计划**

[!\[\]()!\[\](https://spark.adobe.com/page/G7fodvuHBTi48/images/0da7a4da-b8a4-498b-9341-2c4fe4e263c1.jpg?asset_id=8f470d20-7a32-4170-87c9-9c6f483acafa&amp;img_etag=%226d22f0c6df290690fe615e35fb34d6a3%22&amp;size=2560)](https://spark.adobe.com/page/G7fodvuHBTi48/images/0da7a4da-b8a4-498b-9341-2c4fe4e263c1.jpg?asset_id=8f470d20-7a32-4170-87c9-9c6f483acafa&amp;img_etag=%226d22f0c6df290690fe615e35fb34d6a3%22&amp;size=1024)

中共秘密核武器建设区被曝光

十一月，美国国务院发布了四个曾为机密的简报展稿，揭示了中共国钚和铀生产厂的快速扩张，佐证了由国务院称为＂秘密破坏计划＂的核武库建立项目。

有关中共国于其西部的罗布泊核试验基地的细节已于先前发布。该基地最近已开始全天候运作，表明核试验工作已经加速。

对于中共国不遵守《不扩张核武器条约》问题，比林斯利先生说，此次递交参议院的通知信函“与中共国拒绝真诚地与美国进行谈判有关。”中共国于1992年签署了该条约。

该条约第6条规定，签署国必须“真诚地、就早日停止核军备竞赛及核裁军的有效措施进行谈判，达成在严格有效的国际管制下全面彻底的裁军条约。”

川普总统指出，与俄罗斯新的《削减战略武器条约》的任何扩展都应包含中共国，中共国一直在不受军备控制条约约束的情况下参与大规模建设核军力和导弹运载系统。新的《削减战略武器条约》将于2月5日过期，并将成为拜登政府将面临的首批国家安全决策之一。

比林斯利先生在致参议院的信中表示，使协议期限小幅延长，已算取得进展。根据提议的合约，俄罗斯同意对其核弹头库数量设置上限，以换取《削减战略武器条约》的一年延期。

比林斯利先生在信中说：“不幸的是，俄罗斯似乎已得出结论，可以从下届政府处获得更有利的协议。”

五角大楼有关中共国军力的报告称，中共国新增了两艘导弹潜艇部队，入列了其包含四艘晋级（09IV，北约代号Jin-Class）导弹潜艇舰队。并在組额外的攻击潜艇舰队。

海军周一在推特上透露，核武器弹道导弹潜艇俄亥俄号最近驶入了关岛阿普拉港。

由于导弹核潜艇的部署通常情况下为保密信息，因此俄亥俄号部署情况的披露很不寻常。

关岛附近战略性的潜艇活动，发生于中共国在台湾问题上的紧张局势，以及下周的总统府过度期间。
