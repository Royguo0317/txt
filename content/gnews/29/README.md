##  **報告：日本希望加入“五眼”聯盟** 



**日本防衛大臣河野太郎(Taro Kono)在周五發表的一次採訪中對日經新聞社(Nikkei)表示，日本希望幫助建立更多(成員的)“五眼”情報共享聯盟，並將其添加到澳大利亞，加拿大，新西蘭，美國和英國(聯盟中)。**
[!\[\](https://spark.adobe.com/page/Lp8qcMPolIfNb/images/8662d52d-2f41-4776-b038-d183832fa4e7.jpg?asset_id=db4f9fef-b63e-457c-849a-41c5c199dca1&amp;img_etag=%221ca3be62fc0aaab9741df75a4a8bfac9%22&amp;size=3181)](https://spark.adobe.com/page/Lp8qcMPolIfNb/images/8662d52d-2f41-4776-b038-d183832fa4e7.jpg?asset_id=db4f9fef-b63e-457c-849a-41c5c199dca1&amp;img_etag=%221ca3be62fc0aaab9741df75a4a8bfac9%22&amp;size=1024)該機制旨在打擊網絡空間的惡意舉動，”五眼”國家–美國、英國、澳大利亞、加拿大和新西蘭–已經要求其他三國加強反制措施。 
如果允許的話，該框架將使日本能夠在早期階段獲得關鍵的機密信息，因為它可以(幫助)抵禦南中國海及其他地區日益嚴重的安全威脅。

“這些國家有著相同的價值觀，” 河野(Kono)在接受采訪時說。 “日本可以更靠近[同盟] ，即使它被稱為“六眼” ( 聯盟)。”

“五眼”集團是一個情報共享聯盟，由具有深厚歷史和文化聯繫的國家組成，這些國家具有共同的盎格魯-撒克遜文化遺產和英語使用基礎。

通過多邊“UKUSA協定”，所有人都可以團結在一起。它們的共同目的包括在信號情報方面的合作，分析和共享通過竊聽網絡獲得的信息，以追求自由世界安全的目的。

正如布賴特巴特-倫敦報導的那樣，這不是第一次討論增加現有“五眼” ( 聯盟)網絡的(成員國)，而是日本第一次公開自己(申請加入) 的願望。

正如《日經新聞》(Nikkei)採訪所概述的那樣，儘管沒有正式成員資格，日本仍在與該組織共享信息。

河野(Kono)強調，擴大日本與五眼聯盟的聯繫將使該國能夠在早期階段共享信息，並獲得高度機密的情報。

“五眼” 聯盟的特點是在情報共享的基礎上建立牢固的外交關係，並就共同關注的問題發表聯合聲明。河野(Kono)說：“ [成员国]在外交和經濟上保持步調非常重要。”

為了回應中共國不斷增長的軍事擴張，“五眼”( 聯盟)的一些成員還尋求與日本進行更緊密的合作，以共享機密信息。例如，在應對香港和冠狀病毒大瘟疫的外交緊張局勢加劇的情況下，英國對中共國保持警惕，尋求利用日本擁有的信息。

河野(Kono)透露，(“五眼”聯盟)已經與日本“在各種場合”共享信息。他說：“如果採用固定的方法，那麼它可能被稱為“六眼” ( 聯盟)。

但是河野(Kono)說，由於該組織不是一個國際組織，他認為日本不需要經過某些程序即可成為正式成員。 “我們只是將椅子帶到他們的桌子旁，並告訴他們把我們計算在內。”
