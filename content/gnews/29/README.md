# **加拿大投票承认中共国对待维吾尔族的做法是种族灭绝**

议会此举没有得到特鲁多的支持，很可能会加剧两国外交的紧张关系

[!\[\]()!\[\](https://spark.adobe.com/page/NFKPXDFGwmouh/images/77093893-f4a6-4d81-bd11-dc41f4f744d2.png?asset_id=884ad41a-d7df-4b7b-ab6a-ed4c158b75c5&amp;img_etag=%22020a36642e88a595edef34336d87807d%22&amp;size=2560)](https://spark.adobe.com/page/NFKPXDFGwmouh/images/77093893-f4a6-4d81-bd11-dc41f4f744d2.png?asset_id=884ad41a-d7df-4b7b-ab6a-ed4c158b75c5&amp;img_etag=%22020a36642e88a595edef34336d87807d%22&amp;size=1024)2月19日，乌梅尔·杨出席加拿大大使馆外的集会，鼓励加拿大将中共国对待维吾尔族民众的行为定性为种族灭绝。照片：莉亚·米利斯/路透社。

加拿大已成为世界上第二个将中共国对其维吾尔少数民族的待遇描述为种族灭绝的国家，此前议会进行了有争议的投票，这可能进一步加剧两国外交的紧张局势。

议员们批准了由反对派保守党提出的不具约束力的动议，认定中共国在西北新疆省的行为是对维吾尔人穆斯林的种族灭绝。

总理贾斯汀·特鲁多和高级内阁成员没有参加周一的投票。除了外交部长马克·加诺（Marc Garneau）代表政府投了弃权票外，在场的其他自由党成员都投了赞成票。

通过将中共国的行动称为种族灭绝，加拿大加入了美国的行列，美国在唐纳德·川普离任前不久做出了这一决定。本月早些时候，英国的一次类似的投票尝试失败了。

但加拿大议会的这一举动很可能给这位总理带来一系列新的政治挑战，因为他近年来一直试图在回击中共国的敌对行动和与北京当局保持友好关系之间找平衡。

特鲁多所在的自由党在议会中没有获得多数席位，他此前曾公开反对将该动议付诸表决，他对记者说，种族灭绝是一个“极其复杂”的词汇，需要对这个问题进行更多的研究才能做出决定。

周末，中共国驻加拿大大使批评了这次投票，他对加拿大媒体说，渥太华的官员不应该涉足中国的内部政治。

“我们坚决反对，因为这与事实背道而驰。而且这就像，在干涉我们的内政，”丛培武说，”新疆根本就没有发生类似种族灭绝的事情。”

超过100万维吾尔族人被关押在新疆省的集中营中，有报告称存在系统性的强奸和性暴力。中共国否认了这些指控，并将这些设施描述为职业和教育培训中心，并表示正在努力消除极端主义。

但去年10月，加拿大议会的一个小组委员会认定中共国对待维吾尔人的方式构成了种族灭绝。

这项议案很可能使两国之间的紧张关系升级，自2018年12月华为的孟晚舟因美国逮捕令被逮捕以来，两国的关系已一直处于极其紧张的状态。几天之内，两名加拿大公民——迈克尔·科夫里格（Michael Kovrig）和迈克尔·斯帕弗（Michael Spavor）——在中共国被拘留。

加拿大将这两人的拘留称为“人质外交”。

在投票前，保守党领袖艾琳·奥图尔(Erin O’Toole)曾表示，这项动议是必要的，以发出一个“清晰明确的信号，即我们将站在人权和尊严的一边，即使这意味着牺牲一些经济机会”。

保守党、新民主党、魁北克党和绿党都要求对中共国官员进行制裁，并呼吁将即将到来的冬奥会从北京移走。
