# **突发新闻：川普竞选团队就威斯康辛州22.1万张选票提起诉讼**

#### **川普竞选团队今天上午已在威斯康辛州提起诉讼，指控影响多达22.1万张选票的多项选举犯罪。**

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/04083344/2-7.jpg)2020年1月28日，在新泽西怀尔德伍德的怀尔德伍德会议中心，美国总统唐纳德·川普（DONALD TRUMP）在“使美国伟大”的竞选集会中注视。（图片：索尔·罗布/AFP盖帝图像）

**今天上午竞选团队报道：**

唐纳德·J·川普（Donald J. Trump）总统的连任竞选团队在其要求重新计票以揭露无可辩驳地改变此次选举结果的舞弊和滥用行为后，向威斯康辛州最高法院提起诉讼。今天的诉讼包括有明确非法证据的四起案件，诸如，非法修改缺席选票信封、计算缺少必需申请的选票、忽视无限期受限的非法诉求、以及举办名为“公园里的民主”（Democracy in the Park）的非法投票活动。在威斯康辛州投出的300多万张选票中，不少于大约22.1万张选票受到这些非法行为影响。

威斯康辛州选举委员会指使市政办事员违反威斯康辛州的法律，非法更改不完整的缺席投票信封。办事员被指示，他们可以凭借自己的“个人知识”，或者未指定的“他或她所支配的名单或数据库”，在退回的缺席选票上添加缺失的信息。根据威斯康辛州的法律，不完整的缺席选票必须由选民更正，并且只能由选民更正，否则选票不予计算。

在另一个例子中，市政办事员向选民发放缺席选票，却未要求必须提出申请，这与威斯康辛州的缺席投票保护措施直接冲突。威斯康辛州的法律明确要求，在未收到要求选票的书面申请的情况下，不得发放缺席选票。尽管存在明确的法规，麦迪逊和密尔沃基的办事员却在没有收到书面申请的情况下，在2020年10月20日至2020年11月1日的两周本人缺席投票期间，发出数千张缺席选票。

在威斯康辛州，选民身份证明是一项基本要求，以确保只有合格选民才能投票。在麦迪逊和密尔沃基，选举官员欺诈性地允许选民规避选民身份法，并申请缺席投票身份，而这种身份其实只适用于身体有病、虚弱、年迈、或者残疾的情况下被无限期受限制的选民。2万多名选民声称有这种身份，在未提供合适的身份证明并且不符合这种身份要求的情况下收到并寄回选票。根据威斯康辛州的法规，这些选票不应计算在内。

最后，麦迪逊市通过“公园里的民主”（Democracy in the Park）投票活动，在200多个公园和城市地点设立非法投票点。这些投票活动在郡批准的投票地点之外举行，并且没有遵守该州严格的缺席投票要求。他们不仅不遵守法律，乔·拜登（Joe Biden）的竞选团队还鼓励这种非法投票，把这些活动宣传为投票机会，并告诉选民带上填好的选票去提交，或者带上尚未填好的选票让所谓的“投票工作人员”作为见证人。选民不得在指定投票地点之外的任何地方提交缺席选票，在那里非法投出的任何选票不应计算在内。

“威斯康辛州的人民应得到统一执法的选举程序，简单明了。在戴恩郡和密尔沃基郡的重新计票过程中，我们绝对肯定地知道，非法选票已经不适当地影响了该州的选举结果。威斯康辛州不允许300多万张选票被哪怕一张非法选票所侵蚀。”竞选团队律师吉姆·楚佩斯（Jim Troupis）说：“我们将继续代表美国人民战斗，通过帮助恢复我们选举的完整性和透明性，以捍卫他们重获自由和公平的选举权。”

“正如我们从这个过程一开始所说，我们想要所有合法选票，并且只有合法选票才能被计算在内。”前纽约市市长、川普总统私人律师鲁迪·朱利安尼（Rudy Giuliani）说：“美国人民必须能够信任我们的选举结果，并且我们不会停止，直到我们能确保选民再次对我们的选举程序有信心。”
