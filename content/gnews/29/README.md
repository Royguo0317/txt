《门道专家》（the gateway pundit）11月6日发表文章称，沉寂了两天的参议院多数党领袖米奇·麦康奈尔（Mitch McConnell）实在看不下去了，终于发声谴责民主党企图窃取选举。

参议院多数党领袖米奇·麦康奈尔在11月6日周五早上打破沉默，谴责腐败的民主党在选举日后为乔·拜登提供了数百万张非法选票。

麦康奈尔刚刚在肯塔基州战胜一位资金雄厚的对手赢得连任，毫无疑问，川普总统帮助推动了他的竞选活动，不少媒体指责他不应该等那么久才开口说话。

“在我们这个伟大的国家，这必须是每一张合法的选票都应该被计算在内，任何非法提交的选票不得计入。各方都必须观察这个过程，而法院在这里是为了应用法律和解决争端。” 麦康奈尔说，“这就是美国人的选票决定结果的方式。”

此前参议员霍利（Josh Hawley）、汤姆·科顿（Tom Cotton）、特德·克鲁兹（Ted Cruz）和林赛·格雷厄姆（Lindsey Graham）公开为川普总统辩护，反对在宾夕法尼亚州、佐治亚州、威斯康星州以及密歇根州和内华达州发生的明目张胆的盗窃和欺诈行为。

林赛·格雷厄姆（Lindsey Graham）昨晚确实加大了力度，并向川普总统的法律辩护基金捐赠了50万美元。

“我今晚在这里与川普总统站在一起。他和我站在一起……每一位参议院共和党人和众议院共和党人都需要上电视，讲述这个故事。” 林赛·格雷厄姆周四晚在福克斯新闻台（Fox News）与肖恩·汉尼蒂（Sean Hannity）一起亮相时说。
