# 彭斯“欢迎”共和党议院反对拜登选举人票的努力
[!\[\]()!\[\](https://spark.adobe.com/page/V6UlHt7fHCqbA/images/a4567722-d210-4e50-8ece-7f588e668182.png?asset_id=9a0e4878-30f5-4ff0-9773-a0e874ef51f0&amp;img_etag=%22d696ec3d809126503436e185dbdd4041%22&amp;size=1024)](https://spark.adobe.com/page/V6UlHt7fHCqbA/images/a4567722-d210-4e50-8ece-7f588e668182.png?asset_id=9a0e4878-30f5-4ff0-9773-a0e874ef51f0&amp;img_etag=%22d696ec3d809126503436e185dbdd4041%22&amp;size=1024)副总统彭斯2020年12月9日周三在佛罗里达州卡纳维拉尔角肯尼迪航空中心国家航天委员会的会议上讲话。（美联社照/John Raoux）
副总统麦克·彭斯周六晚表示，他欢迎共和党议员挑战选举人团在1月6日联席会议的结果，联席会议上选举人团将正式记录选举人票。

“副总统彭斯传达了数百万美国人对上一次选举时的欺诈和选举违规行为的担忧”，副总统的幕僚长马克·肖特 (Marc Short)在一次声明中说。“副总统欢迎参众两院议员使用法律赋予他们的权力反对投票结果，并且在1月6日把证据带到国会，让美国人民看到。”

该声明公布的数小时前，科鲁兹参议员（德州共和党）、其他10名共和党参议员和预计当选参议员共同宣布在周三国会召集会议记录选举人票时挑战乔·拜登的选举团投票。他们要求国会指定临时选举委员会，在1月20日就职典礼前进行为期10天的投票结果审计。

彭斯先生以参议院主席的身份将于周三主持联席会议，并受到越来越大的压力，要求他代表特朗普总统采取单方面行动，拒绝有争议的战场州对候任总统拜登的投票总数。副总统一周前和共和党立法人员在白宫一同出席了战略会议。

拜登发言人麦克·格纹(Mike Gwin)驳斥了共和党的努力，认为这是一个不会改变选举结果的 “噱头”。拜登赢得了306张选举人票，而川普只有232张。竞选人需要270张票取胜。

国会的民主党人表示共和党拒绝承认并不会帮助川普逆转选举结果。

“许多法庭和州立法者已经发现没有证据显示2020选举存在大量欺诈现象”，民主党弗吉尼亚参议员马克·沃纳 (Mark Warner)表示。

“许多共和党参议员努力帮助一位被击败却耍脾气的总统，这样的行为只会损害民众对大选的安全和神圣不可侵犯的信心，这也是我们的先贤所反对的。我敦促我的同事严肃考虑他们行动的后果，我担心他们会损害我们的民主体系和我们的国际信誉，这是无法轻易挽回的。”
