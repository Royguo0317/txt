# 汤姆·科顿：拜登提名的国土安全部长马约卡斯因向中国公民出售绿卡而不具备领导资格

周三，参议员汤姆·科顿（Tom Cotton）在福克斯新闻频道的“Fox & Friends”上，针对拜登提名的潜在内阁成员进行了探讨。

这位来自阿肯色州的共和党议员特别提到了两个人，分别是拜登提名的国务卿安东尼•布林肯(Anthony Blinken)和国土安全部部长亚历杭德罗•马约卡斯(Alejandro Mayorkas)。

科顿参议员对这两人都持批评态度，但称马约卡斯不合格，因为奥巴马政府的监察长发现了他代表民主党捐赠者向中国公民出售绿卡。

![]()![](https://spark.adobe.com/page/soZ1zDRJNMCir/images/5289c7da-8c83-4ba6-ba67-ee5f883e03d9.jpg?asset_id=cd01644a-5356-4768-93ad-dc10982c1bd3&amp;img_etag=%222ecdc5b9aaf15a6ce9869e57cccb9bc3%22&amp;size=2560)拜登提名的国务卿安东尼•布林肯(Anthony Blinken)

他说:“我不愿在拜登提名人选尘埃落定之前作进一步揣测，因为拜登这个星期早些时候宣布的提名人选中有太多的事情需要考虑。我的意思是，以移民问题为例，乔·拜登昨晚说，他将在头100天内提交一项特赦1500万非法移民的法案。这可能会导致大量非法移民试图在该法案通过之前涌入我们的边境。”

科顿补充道:“看看托尼·布林肯说的话，国务院要对难民负责。他希望将难民接收人数增加7倍，而不是仅仅把它们翻了7倍。一般来说，如果难民来自像叙利亚或索马里这样的国家，我们是无法彻底调查清楚他们的身份或背景的。甚至以国土安全部候选人亚历杭德罗·马约卡斯为例，奥巴马政府的监察长发现，他曾代表富有的民主党捐赠者向中国公民出售绿卡。试想，代表民主党捐款人向有关系的中国公民出售公民权。这就失去了领导国土安全部的资格”
