## 美国正式批准瑞德西韦，世界卫生组织（WHO）否认其疗效
` GM42` [轉載自GNews](https://gnews.org/zh-hans/445795/)

编撰：文燕、文锦

审核：WLQF

美国食品药品监督管理局（FDA）10月22日批准美国制药巨头吉利德科学公司（Gilead Sciences）的「瑞德西韦（Remdesivir）」用作新冠治疗药。今后将能以入院患者为对象广泛使用。这是美国首次正式批准新冠治疗药。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/24055247/image001-6.jpg)抗病毒药物瑞德西韦（图片来源：BBC中文）
据【BBC NEWS中文】10月23日报道，美国正式批准瑞德西韦用作新冠治疗药。

美国食品和药物管理局（FDA）表示，瑞德西韦在临床试验期间将新冠患者的恢复时间缩短五天，是首个获批的治疗新冠药物。

瑞德西韦最初是作为埃博拉的治疗药物而开发的。作为一种抗病毒药物，它通过攻击病毒在细胞内复制所需的酵素而起作用。瑞德西韦在美国国家过敏与传染病研究所（NIAID）主导的临床试验之中确认了有效性。结果显示入院患者的康复平均提前5天。

美国食品和药物管理局在声明中说，该药已于周四（10月22日）获批，「可用于12岁及以上，体重至少40千克（约88磅）的成年和儿童住院患者」 。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/24055252/image004-1.jpg)瑞德西韦是最早用于治疗新冠肺炎的几种药物之一。 （图片来源：BBC中文）
美國總統川普確診新冠病毒感染後，瑞德西韋是其服用的藥物之一。

吉利德的首席執行官（CEO）丹尼爾·奧黛表示，「從新冠最初的病例報告算起，在不到1年裡獲得批准，有需要的患者可以利用」。

戰友點評：

世界衛生組織（WHO）10月15日發布調查結果稱，瑞德西韋對於降低新冠患者的死亡率「基本或完全沒有效果」。對此，吉利德反駁稱調查結果「未經嚴格審查」，並拒絶承認世衛的試驗結果。

從世界衛生組織（WHO）否認瑞德西韋的藥效，恰恰印證了WHO在國際上充當中共喉舌，堅持與中共勾結到底的醜惡面目。 CCP病毒肆虐全球，至今已有 4,156 萬人確診，累計 113.5 萬人死亡，急需有效療法治療病人。而世界衛生組織此時此刻還繼續罔顧病毒的真相，依然選擇與中共魔鬼一條路走到黑。

年初疫情爆發，福奇博士代表美國醫學界的權威就出現在美國人們的視野，當川普總統宣布頒布緊急旅行禁令和羥氯喹對病毒有一定的預防效果的時候他堅持反對意見。而羥氯喹這種價格低廉的藥物，經過很多臨床使用被全世界很多臨床醫生認為是對預防和輕症的病毒感染者有顯著效果，而且是已經過美國FDA批准的藥物，令人不解的是羥氯喹始終得不到美國相關部門的解除處方管制，反而在此時讓相對價格昂貴的瑞德西韋通過FDA的准入。從WHO否認瑞德西韋療效，到美國FDA准入瑞德西韋、不放開羥氯喹處方藥，還有中共在國內吹噓蓮花清瘟，如今又積極地推廣病毒疫苗接種，我們看到全世界科學界的亂相和深層的逐利本質。
