## 歷史的糾纏之中東篇___以色列與阿聯酋的和平協議
`2020-10-19 11:33 CHANGDAO` [轉載自GNews](https://gnews.org/zh-hant/434242/)

作者：香草山教育組 坐看雲起時

2020年8月13日在美國總統特朗普的協調下，以色列和阿拉伯聯合酋長國達成了一項歷史性的和平協議。這是以色列25年來首次與一個主要的阿拉伯國家達成邦交正常化的協議。

協議名稱：“亞伯拉罕協議”，因為聖經人物亞伯拉罕被視為該地區三大宗教之父：猶太教，基督教和伊斯蘭教。 “亞伯拉罕”這個名字希望這三個偉大信仰實現團

根據協議內容：以色列暫停吞併約旦河西岸地區的計劃，阿以兩國關係全面正常化，將互設使館和大使，並開始在的廣泛領域進行全面合作。

談到以色列和阿拉伯人之間的和平，必須得從猶太人和阿拉伯人之間的恩怨，近百年來猶太人遭受的迫害，猶太復國主義，歷次中東戰爭，土地換和平，波斯人和阿拉伯人之間的歷史仇恨，CCP的一帶一路，方可以了解這次和平的艱難，以及其重要意義。

本是同根生

猶太人和阿拉伯人都是中東閃米特人的後裔，同宗同源，根據《聖經·舊約》上記載，他們的遠祖是亞伯拉罕，而亞伯拉罕的大兒子以實瑪利是公認的阿拉伯人祖先，同父異母的小兒子以撒則是猶太人的祖先。

猶太人的悲催遭遇

只從猶太人被趕出自己的家園，開始流浪後，他們就不斷遭受歧視與迫害。 1世紀開始，猶太人就開始遭受來自羅馬天主教會不同程度的歧視和迫害，原因是當時天主教認為猶太人對耶穌受難有不可推卸的責任。歐洲一直存在歧視猶太人的規定，比如猶太人不能做騎士，不能擁有土地，不能開辦工廠，不能幹這，不能干那，只容許他們做一些低賤的工作，比如從事放貸收貸的事情。在二戰的時候歐洲的法國，英國，俄羅斯都存在排猶的現象，有些國家也建立了集中營，其中最臭名昭著的是納粹德國對猶太人系統和工業化的屠殺中摧毀。這些徹底喚醒了國際社會和猶太人本身。

猶太復國主義

因為猶太人在歐洲不斷受到迫害，不斷流浪，從而萌生建國的想法。 1897年8月，在瑞士巴塞爾，舉行了第一次世界猶太復國主義者（錫安主義）大會，大會通過了《世界猶太復國主義綱領》，規定：“猶太復國主義的目標，是在巴勒斯坦為猶太民族建立一個由公共法律所保障的猶太人之家。”。第一次世界大戰後，奧斯曼帝國瓦解，其在中東的統治權易手。 1917-1948年，英國統治巴勒斯坦。 1917年11月2日，英國外交大臣阿瑟-詹姆斯·貝爾福寫信給英國猶太復國主義聯盟副主席羅斯柴爾德，表示贊成猶太人在巴勒斯坦“建立一個民族之家”，這就是著名的《貝爾福宣言》。在經過納粹德國對猶太人大屠殺中摧毀了歐洲的猶太族群後，錫安主義的民粹運動於1948年達到了高潮。猶太人全國委員會於1948年5月14日發布了《以色列獨立宣言》，現代以色列建國。

歷次中東戰爭

伴隨著以色列1948年5月14日宣佈建國，阿以中東戰爭並開始了：

1. 第一次中東戰爭：於1948年5月15日開戰，以色列人稱之為「獨立戰爭」，阿拉伯人稱之為「大災難」。

2. 第二次中東戰爭：於1956年10月29日開戰，又別名為「蘇伊士運河危機」和「西奈戰役」。

3. 第三次中東戰爭：於1967年6月5日開戰，以色列稱之為「六日戰爭」，阿拉伯國家稱之為「六月戰爭」。

4. 第四次中東戰爭：於1973年10月6日開戰，又別名為「贖罪日戰爭」、「齋月戰爭」和「十月戰爭」。

5. 第五次中東戰爭：於1982年6月6日開戰

土地換和平

雖然以色列建國時間短暫，但是自從他建立的第一天起，他就在跟阿拉伯世界在戰鬥，但是幾乎所有的戰鬥中以色列都勝出，隨著以色列不斷取勝，它實際控制的領土比當初聯合國承諾的大很多。但是隨著時間的流逝，阿拉伯國家也明白戰爭無法消滅以色列，同時各阿拉伯國家也有自己內部問題，再無法組成阿拉伯聯軍，大家陸續開始談判。談判重要內容並是以色列把自己這些年佔領的土地讓出，周圍的阿拉伯國家不再攻擊以色列，即是“土地換和平”。 1993年9月，《奧斯陸協議》。這個協議導致了以色列和阿拉法特的巴解組織相互承認。以色列在承諾巴勒斯坦自治的同時，相繼交還了1967年戰爭取得的包括希伯倫，加沙走廊，約旦河西岸和傑里科在內的大量領土。但是以色列用戰爭打出來的土地哪有那麼容易讓出，而巴勒斯坦內部也無力確保以色列執行協議，並發動戰鬥以色列的襲擊……

和平協議

隨著打打停停，巴勒斯坦各種反抗組織的衰弱，以色列的強大，和平總算是真的來臨

以色列曾經與埃及（1979年）和約旦（1994年）簽訂過和平協議，但是那都是快30年前的事了。

而今年（2020年8月13日）在美國總統特朗普的協調下，以色列和阿拉伯聯合酋長國達成了一項歷史性的和平協議。這是以色列25年來首次與一個主要的阿拉伯國家達成邦交正常化的協議。

協議的意義

阿聯酋在中東處於一個金融中心的地位，七哥介紹過阿聯酋是阿拉伯國家中現金最多最富有的國家：迪拜奢華的酒店，奢侈品店，豪華的跑車，世界各地聚集於此的商賈。那當然就是最讓CCP垂涎的肉，CCP把阿聯酋作為一帶一路的中東橋頭堡，用它輻射整個阿拉伯世界，但是如今如意算盤落空了，阿聯酋和以色列的這個協議表明：阿聯酋願意和西方主流文明國家為伍！這種情況可以從周邊國家對該協議的反應得知，據網上資料，最反對的國家有：伊朗，敘利亞，土耳其。這三個國家都是CCP的好哥們。

阿聯酋因為佔據波斯灣最重要的位置，而且國土面積中等規模，人口也還可以。而且該國家的伊斯蘭教也基本上世俗化了。所以阿以和解具有非常重要的意義，但是它只是中東阿拉伯人和以色列和解序幕的開始，它開啟了中東真正的和平進程，使以色列和阿拉伯世界一起對付伊朗（這個CCP在中東的主要盟友）成為可能。在歷史上波斯人基本上一直統治著阿拉伯人，直到穆罕默德創立伊斯蘭教。雖然伊朗和阿拉伯人都是伊斯蘭教，但是他們其實是差別的，阿拉伯人是遜尼派，而伊朗人是什葉派。也就是說阿拉伯人和伊朗人是世仇，這從兩伊戰爭可以看出，伊拉克那邊是有大量的阿拉伯國家支持。

而中東阿以和平協議，基本上可以說是阿拉伯何猶太人這兩個曾經的兄弟總算聯合起來了，同樣也讓美國能夠從中東的亂局中抽身出來，集中力量對付CCP。
