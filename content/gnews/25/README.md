據福克斯新聞報導：《華盛頓郵報》專欄作家喬希·羅金（Josh Rogin）週一發推表示：NBC（美國全國廣播公司）對關於武漢病毒研究所（WIV）的一篇新報導前後矛盾，極具誤導性。他說:“NBC新聞在武漢病毒研究所（可能是冠狀病毒發源地）的實地考察有幾處錯誤，最明顯的是它歪曲了美國官員在2018年外交電報中所寫的內容。”

NBC報告稱，美國的官員在實驗室中發現了幾處安全隱患。但羅金表示，武毒所的科學家早在2018年1月的電報中向官員們透露了這些問題。中共所稱“高級科學家”的WIV所長王延軼是一個藝術畢業生，她告訴NBC記者美國官員到訪該所的時間是2018年3月，但是“這已是第一封電報寫好後的兩個月。真相是，在2018年1月的電報前後他們訪問過病毒所三次。美國官員徹查過病毒所嗎？不太可能。”羅金嚴重質疑王延軼說法的可信度，關注點聚焦在她聲稱生物安全問題並未與美國官員討論的說法，並指出她實際上在指稱美國為“(病毒)製造者”。羅金批評NBC公司的報導對王的說辭竟然“沒有任何回擊” 。他還質疑NBC的記者為什麼要去實驗室參觀，他指出：“這些記者能找到什麼？一張被中共忘了丟掉的一張寫著’CCP病毒起源證據’ 的紙嗎？”羅金認為NBC記者能證明任何事是及其荒謬的，還有NBC報告說安東尼·福奇（Anthony Fauci）博士不相信CCP病毒是意外地從實驗室中洩露的，但不排除實驗室事故的可能性（這是非常錯誤的觀點）。

羅金還談到：WIV副主任袁志明博士重申，沒有證據表明CCP病毒是在武漢華南海鮮市場從動物傳播給人的。總的來說，人們不能責怪遵守共產黨路線的那些中共科學家們，他們與被中共判死刑的距離很近。但是，美國的新聞機構必須恪守職業標準，爭取做到最好，而不僅僅只是去實驗室裡隨意走走，對事實真相進行虛假報導，來誤導世界關於CCP病毒的任何信息。
