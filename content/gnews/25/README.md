**“如果妳現在在壹個高樓陽臺上，會想什麽？”**

“**我想，我還能怎麽辦？”**

**——胡波《大象席地而坐》**

這是電影中於城與韋布的問答，而到了現實生活又發生了什麽呢？

10月10日，南京審計大學審計學專業2017級學生李某某從宿舍墜樓身亡。

10月13日，大連理工大學化工學院壹名研究生被發現在實驗室身亡（GNEWS文章參考：《國內時評|中共畸形體制逼死大連理工在校研究生》）。

10月14日，興寧壹中壹教學樓有人墜樓，經120醫務人員現場搶救無效死亡。死者陳某現為廣州機電職業技術學院2020級學生，原興寧壹中學生，據調查，陳某於當天17時許著校服進入學校，18時許從該教學樓6樓自行跳樓。

10月15日，江蘇大學食品與生物工程學院壹名學生從主A樓6樓衛生間墜樓身亡，警方調查稱“排除他殺”，然而學生父親發文提出五大疑點，並以自己身份信息作保願意承擔壹切法律責任。

**像我們這樣的人 是不是應該在上世紀就死去？**

**——陳升《像我們這樣的人》**

如果在GNEWS上搜索“墜樓”的話，已經有多達6頁的結果了。為什麽這麽多人都走到了死亡這個結局？並且這壹段時間路德時評在持續揭露地獄硬盤中的信息，很多人應該能明白節目中的這壹句話“當妳看完了全部這些資料後，絕對會懷疑人生。”從CCP的91號與94號文件到這地獄硬盤，很多人的心情都非常抑郁。而戰友們抑郁的心情與墻內這些死亡的生命們共通的壹點就是，大家在各自的世界中抽絲剝繭，但每壹層剝開後發現其中的山盟海誓都是謊言，換成誰都接受不了。

**謊言中埋藏著許多的謊言 緩慢地真假早已不分**

**每壹天都想要檢討我自己 老覺得作為人類真可憐**

**——陳升《像我們這樣的人》**

在這個充滿謊言的庚子年，通俄門是偽造的，CCP給病毒披上了千萬層虛假的外衣，新聞上的大好河山遮掩了南方嚴重的水災、三峽的安全性以及必然會發生的糧食危機。等到今年的末尾，妳又看到了CCP的假疫苗、數字貨幣還有地獄硬盤中特別是美國的高層們對於本國核心價值的背叛。2020除了蔓延著死亡，還有就是謊言。

在這個最沒有悲憫的庚子年，這些已經逝去的生命，他們面臨了什麽。有把學生當成技術型勞工壹樣的處境，有導師對學生任意的挾制，甚至還有語言文化的滅絕危機，以及對父權的怨恨。不管原因如何，他們在死亡這壹道大門前可以確定的是比任何人都還要清醒，那些活著的學會了妥協的人，沒資格對絕不妥協的人說妳可以委曲求全。

**我壹天上了多少網 我壹天用了幾個塑膠袋**

**我壹天走了多少的路 我壹天毀了多少個夢**

**我壹天說了多少慌 我壹天瞎掰了多少的諾言**

**——陳升《像我們這樣的人》**

以上種種讓筆者最能夠感到絕望的就是興寧壹中這位高三畢業生的墜亡了。在他跳樓自殺這件事發生過後，網上也找到了他的遺書。看完了這封遺書後，壹是因為字裏行間與我所想不謀而合，另外就是感覺到他作為壹名高三畢業生，在有防火墻並且不知道爆料革命信息的前提下思想高度完全到了盡可能達到的地方，但是世界上真正的邪惡在離他萬尺高的地方就開始籠罩天空了，知道真相之後會怎麽想，至少我不敢想。遺書當中寫道“十八歲的我會永遠嘲笑他們和變成三十歲的我。也許領導也有領導，這上面永遠沒有盡頭，也許人類造不出巴別塔，或許可以把自己疊上去。”還有他作為父權的天然受益者卻深刻意識到其中的不公平，以及遺書中對母親的關心，希望父親不要再辜負母親還有對姐姐的祝福。“妳說這世界繽紛多彩，可天空不也還是壹無所有。”今年看到的最棒的壹篇文字居然是這封遺書，這可是黑得不能再黑的黑色幽默了。

不過也正是這樣壹篇遺書，天空不會壹直是壹無所有的。像我們這些活著的人，盡管也已疲憊不堪，但壹定會幫妳見證這個世界到底是多麽荒謬。臺灣因為它的民主化歷程。所以太陽花學運取得了到現在都有影響的成果，而香港因為英式基本法作為基礎，盡管遭到了CCP殘酷的鎮壓，結果目前未蔔，但抗爭的過程讓所有人尊敬。不過墻內之前的狀況都是還未開始便以結束，連抗爭的機會都沒有，這對於每壹個有良知的中國人來說都是壹種恥辱，妳擁有都是僥幸，失去的都是人生。終於現在我們有了壹個扭轉的機會，既能改變過去的這個恥辱，也可以改變未來讓我們繼續自己的人生，這也是我們面對死亡與謊言仍然保持信心的理由。所以我們不得不帶著自己的專業，離開原來的軌道，但請相信，最終壹定會回來相見的。

像我們這樣的人，就是社會學中的薛定諤的貓，同時活著也同時死去，因為不容於這個社會所以感到痛苦，只有時間的盡頭才能成為安定的居所。從高樓陽臺上縱身壹躍，我覺得不僅能到時間的盡頭並且過程中也終於能得到片刻的自由吧。
