## 習近平西藏十個必須宗教中國化與邊境安全引關注
`2020-09-06 00:05 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hant/336533/)

編輯:玫瑰天空

2015年8月24日至25日,中央第六次西藏工作座談會在北京召開。 在這次會議上,習近平以”六個必須」明確了黨的治藏方略。 時隔5年後,2020年8月28日至29日,中央第七次西藏工作座談會在北京召開,習近平以”十個必須”進一步豐富和發展了新時代黨的治藏方略。 十個必須:「宗教中國化」引人關注! ![](https://gnews.org/b46d3ee0-5a24-41a0-9d76-41a61f18ff8d)

習近平在會上提出的「十個必須」: 。

1.必須堅持中國共產黨領導、中國特色社會主義制度、民族區域自治制度; 。

2.必須堅持治國必治邊、治邊先穩藏的戰略思想;3. 。 必須把維護祖國統一、加強民族團結作為西藏工作的著眼點和著力點; 。

4.必須堅持依法治藏、富民興藏、長期建藏、凝聚人心、夯實基礎的重要原則;

5.必須統籌國內國際兩個大局; 。

6.必須把改善民生、凝聚人心作為經濟社會發展的出發點和落腳點; 。

7.必須促進各民族交往交流交融; 。

8.必須堅持我國宗教中國化方向、依法管理宗教事務; 。

9.必須堅持生態保護第一; 。

10.必須加強黨的建設特別是政治建設。

這與5年前相較,對一些表述進行了補充或完善。 如:明確提出「必須把維護祖國統一、加強民族團結作為西藏工作的著眼點和著力點」;將”必須統籌國內國際兩個大局」這一條單獨列出;進一步提出”必須堅持我國宗教中國化方向、依法管理宗教事務”;同時,首次提出”必須堅持生態保護第一”。

從1980年召開的中央第一次西藏工作座談會,到2020年中央第七次西藏工作座談會;從百萬農奴翻身解放,到經濟社會跨越式發展, 。

再到決勝全面小康的大飛躍。

在中印邊境衝突升級,新疆、內蒙等少數民族及人權問題持續為中國帶來國際輿論壓力的時刻,中國領導人習近平通過一次關於西藏的會議向外界顯示,他將在這些問題上繼續保持強硬,不斷全面鞏固自己的權力。

十個必須:**「**宗教中國化**」**引人關注。

有分析認為,新方略體現出習近平在民族和邊境事務上鞏固與強化權力的意向。 習近平對邊境安全及少數民族宗教問題做出特別指示,提出**「**要確保邊防鞏固和邊境安全,並推進藏傳佛教中國化**」**。

西藏問題是長期關注中國人權及宗教問題的西方國家關切的重點問題。 雖然中國正面臨愈發棘手的外部環境,而習近平在此次會議中表現出,民族與宗教問題不會成為他的弱點,他對這些問題擁有絕對控制。
