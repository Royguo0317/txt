据路透社4月7日来自华府的突发新闻，世界经济合作与发展组织（OECD）在努力动员其37个成员国，统一设定一个公司所得税最低税率。该组织认为，应该支持美国这一方面的建议，“不能眼睁睁地看着美国单独行动”，这句话含意深刻。

冷战之后的全球化环境之下，各国为了争夺资本投资和商品市场，普遍实行了竞争性降税，使资本向更低税收及更低人力成本的地区流动，这种恶性竞争使各国的税收普遍下降，而心怀不轨的中共通过提高国内进口关税，对外倾销廉价商品，破坏了基于诚信和规则的全球贸易秩序，造成发达国家资本严重外流及贸易赤字居高不下，发达国家产业空心化致使失业率上升及国家财政困难，这是美、日、欧盟在CCP病毒爆发之前就已经债台高筑的根本原因。

近日美国财长珍妮特·耶伦在20国集团中呼吁共同设定公司所得税最低税率，建议把最低税率提高至28%。虽然这项提议理论上完全符合这些大国的利益，但是要形成协议难度极大，即使完成协议，最后还是因为各谋私利等原因使协议变成一纸空文。因此，耶伦在G20推动共同增加公司税，仅可能和六眼联盟(包括日本)等少数国家达成有效共识，结果是美国对海洋系等发达经济体以外的国家大幅提高关税。世界经合组织非常清楚，耶伦推动统一提高公司税，是为了美国即将启动基础设施建设及设置关壁垒做准备。

通过基建投资启动一轮超级经济增长周期，是美国川普政府首先策划的。19世纪后期美国的基建热潮，推动了美国经济的快速发展，使美国进入经济的黄金年代。罗斯福在大萧条后期启动基建，也推动美国走出了大萧条的泥潭。当时的罗斯福首先大幅度提高进口关税，用高关税壁垒来保障大基建使美国的产业链受益，使美国的财政受益。

只有保护自己的市场即保证美国人买美国货时，投资基础设施建设，重建以制造业为核心的工业体系才有意义。听起来很老套，但美国下一步还得必须这么走下去，这意味着美国将不可避免地重回孤立主义或半孤立主义状态。

川普吸收二战后美国开始放弃孤立主义走向世界的失败教训，除了把战略重心放在印太扼制中共以维持海洋霸权之外，从中东、阿富汗等大量撒兵，要求欧洲、日韩为美国驻军支付更高费用等，就是为了退出吃力不讨好的欧亚事务，让财政空虚的美国回归孤立，休养生息。但中共干预美国大选，发动超限生物战，至少打乱了爱国者川普、班农的美国复兴计划。

被CCP病毒笼罩的世界更加需要一个伟大的美国，拜登政府领导的美国将向何处去？
