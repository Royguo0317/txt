## 六四港共通過國歌法意味著什麼
`2020-06-05 22:05 GM06` [轉載自GNews](https://gnews.org/zh-hant/223912/)

作者：立武

6月4日，被中共控制的香港立法會三讀以41票贊成、1票反對（鄭松泰）通過中共偽政權的《國歌法草案條例》，香港政制及內地事務局局長曾國衛隨後宣佈，**《國歌條例》將於6月12日刊憲，正式實施，標誌著中共侵略香港的又一大勝利。**

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/05215248/121.jpg)

曾國衛於2016年4月獲時任行政長官梁振英提名和建議任命後，被中共委任入境事務處處長。隨後，他在今年4月又晉升政制及內地事務局局長，當時他辦公室顯眼處放置習近平的瓷照引起熱議**。****5****月****31****日，曾國衛曾對香港國安立法發表親共言論，稱“別有用心之人想汙名化、妖魔化相關立法則是賊喊捉賊”。**

中共利用《國歌法》滲透香港要早到2017年，當時中共人大常委會將《國歌法》納入香港基本法附件三，中共認為國歌法屬於全國性法律，適用於香港。**11****月，時任行政長官林鄭月娥表示，希望儘快完成以本地立法的形式制訂相關法律。**隨後香港政制及內地事務局展開本地立法工作。

**然而，國歌法的內容和圍繞其立法的相關宣傳都是中共妄想抹去香港人民主與法治的記憶，將集權和烏托邦法西斯主義植入香港人的生活。**
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/05215355/1211.jpg)
2018年，在香港政制及內地事務局在向立法會提交的立法檔中，建議內容就明確提出了**包括規定中小學生瞭解國歌歷史（也就是中共篡改的洗腦史）以及電視臺播放國歌宣傳片和奏唱國歌須肅立莊重（實質上樹立党的權威）等等。**

2019年1月行政會議通過了《國歌條例草案》，隨後提交立法會進行首讀和二讀，**但在香港人抗爭中共對香港自治進行侵蝕的情況下，二讀程式一直延遲到2020年的5月27日，並在次日通過該草案。**

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/05215528/12113.png)

在國歌條例生效後，中共將可以根據條例中對侮辱國歌的規定判監三年，**一方面何為侮辱國歌中共可以隨意定義，另一方面，在中共的黑監獄裡什麼事情都可能發生，意味著中共對愛好民主法治的香港人的亂抓亂捕將上升到更高規模。**

**6月5日，香港足球總會主席貝鈞奇表示，足總有可能設立黑名單，禁止“滋事”球迷入場。**在去年11月的兩場世界盃預選賽中，香港人都以噓中共“國歌”的方式表達了中共對香港侵蝕的不滿。

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/05215610/1214.jpg)

在6月4日三讀期間，民主派議員一共提出了21項修訂案，其中**包括降低國歌法違法的罰則、加入國歌作曲作詞人的生平（作詞人田漢在文革期間被中共迫害致死）、反對宗教場合需奏唱國歌等等，**這些訴求放在任何一個民主法治的國家都合情合理，然而都被中共的傀儡立法會議員全盤反對。**曾國衛反對“先審議後訂立”，認為先訂立合適，昭示了中共強權滲透的決心。**

在三讀期間，民主派議員在表達對六四紀念的同時也多次抗議和抵制國歌法。在6月4日臺北自由廣場紀念六四燭光晚會上，香港著名填詞人林夕也表達了對香港通過國歌法的質疑，**他批評國歌法是“隨便抓誰就抓誰”，是“人治”。如果中共真的對它的國歌有一點敬畏之心的話，就不會擔心在有人在公共場合高唱“起來不願做奴隸的人們”。諷刺的是，中共正是想把香港人做為奴隸才強行通過國歌法，而香港人用最文明的抗爭行動證明了什麼才是真正“不願做奴隸的人們”。**
