《华盛顿观察》2月19日发表了一篇看似偏向中共女婿麦康奈尔的文章，品味一下，能感觉到共和党建制派对川普总统难以割舍的心态，因为共和党很清楚，如果川普总统自立门户的话，共和党不会有未来。

文章说，川普总统和参议院少数党领袖麦康奈尔曾经合作，共同努力重塑了联邦司法体系，建立了保守派最高法院多数派（有何用？关键时刻出卖了川普总统！笔者注），现在却在争夺共和党的控制权。

共和党内部人士不确定哪一方会获胜。一些人认为，川普当选总统是一个标志，表明在可预见的未来，普通选民已经转向反对共和党建制派政治。另一些人则怀疑这位前总统是否有足够的注意力或组织能力与这位来自肯塔基州的狡猾的参议院战术家竞争。

但许多人担心，美国两位最重要的共和党领导人之间旷日持久的公开争吵所产生的后果，“我们都很庆幸DJT（此处应该指川普总统）没有推特账户了。”一位共和党战略家、前川普2016年竞选官员表示。

早期迹象显示，草根阶层仍坚持支持川普总统，在昆尼皮亚克大学本周公布的一项民调中，只有11%的共和党人认为川普总统应对1月6日他的支持者袭击国会负责。四分之三的人希望他继续在党的未来发挥“突出作用”。

相比之下，麦康奈尔自斥责川普以来，他在本州共和党选民中的支持率骤降29个百分点至41%。麦康奈尔在肯塔基州11月刚刚以近58%的得票率连任，而川普总统以62.1%的得票率赢得该州，并将拜登的得票控制在100万票以下。

当标志性的保守派电台脱口秀主持人拉什·林堡（Rush Limbaugh）周三去世时，川普总统接受了一轮媒体采访，他讨论了主持人的遗产——而不是麦康奈尔。麦康奈尔在参议院的服务可以追溯到里根总统的政府时期，他对共和党政治的参与更为深远。

两人在川普总统任期初期发生冲突，此前参议院未能废除和取代奥巴马医改法案，而该法案在众议院获得了微弱通过，当时两院都在共和党的控制之下。

但川普总统很快开始欣赏麦康奈尔的立法智慧，此前他曾引导三名最高法院提名人获得确认，并帮助击退了民主党的首次弹劾行动。就在一年前，川普总统在白宫与麦康奈尔和其他国会共和党人庆祝了他最初的弹劾闹剧。共和党议员鼓掌时，川普总统说，“米奇·麦康奈尔，我想告诉你，你做得非常出色，这家伙很棒，我很感激，米奇。”

麦康奈尔再次投票宣布川普总统无辜，但这一次他自己并不是前总统的盟友。“川普在实际和道义上对挑起当今事件负有责任，毫无疑问。”他在参议院发言时说，“冲进这栋大楼的人们相信，他们正在按照总统的意愿和指示行事，并秉持了这样的理念，（可以通过）虚假陈述、阴谋论和无顾忌夸张的愈演愈烈（获得）一个可预期的结果，失败的总统一直对着地球上最大的扩音器大喊大叫。”

川普总统对此发表了一份措辞激烈的声明，谴责麦康奈尔是“冷酷、阴险、不苟言笑的政治黑客”，同时强调是他为共和党赢得了荣誉，以事实为例，（在他的努力下）使得共和党离参议院多数席位只有一席之遥，包括麦康奈尔自己的连任，并指责肯塔基人反对该党的2000美元的个人刺激金，导致了佐治亚州的失败。

（此处略去质疑川普总统的评论）。
