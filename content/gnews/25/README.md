## 最新澳洲民調：中共國徹底失信 威脅澳洲國家安全
`2020-06-24 09:52 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/244290/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/06/10065415/bigstock-Flags-Of-Australia-And-China-P-122780135-150x150-1.jpg)圖片來源：[migrationtranslators.com.au](https://insidestory.org.au/china-and-australias-fifth-icy-age/)  ( *Daniren/Alamy* )
據澳洲《新日報》和美國CNBC《消費者新聞與商業頻道》6月24日報導，澳洲智囊洛伊研究機構(Lowy Institute)當日發布了最新澳洲民調顯示，澳洲民眾當下的世界形勢和現狀感到悲觀，安全指數下降，對中共國信任也急劇下滑，以及對經濟衰退的擔憂上升。洛伊研究機構於3月16日到29日發起此次民意調查，總共2448名澳洲人參與。

**民調數據：中共國**

- 只有**23%**受訪者認為中共在疫情以及其他國際事物中的行為是負責任的，同比2018年52%減半有多;
- **94%**受訪者呼籲澳洲政府需積極發展其他貿易夥伴，減少對中共國的經濟依賴;
- **41%**的受訪者認為中共國已經對澳洲構成了威脅;
- **80%**的受訪者還認為，澳洲政府應對侵犯人權的中國官員進行嚴厲制裁;
- 只有**31%**認同中共國的疫情處理方式，而93%的受訪者對澳洲政府應對疫情採取的措施表示滿意。


洛伊機構一名負責人納塔莎▪卡薩姆(Natasha Kassam) 接受澳洲廣播電台採訪時說，澳洲民眾直至兩年前都一直把中共國視為一個巨大的經濟機遇。但是近期中共國被屢屢曝光的行為(注：比如對他國滲透及乾預，華為盜取知識產權及危害他國國家安全，中共國對香港出台的《國安法》以及對香港自由民主遊行的鎮壓等） ，讓更多澳洲民眾清楚的意識到，中共國對澳洲乃至世界安全的威脅。以上數據充分地說明了這點。

**民調數據：美國**

- 儘管不到三分之一的受訪者認為美國總統唐納德▪川普在國際大事件中有能力做出正確的判斷和決定，但超過四分之三的受訪者表示澳洲與美國的關係對澳洲的國家安全至關重要。


**其他民調數據：**

- 安全感用來衡量澳洲人對國際事物態度。此次民調顯示澳人的安全感降至16年以來的最低點；
- 只有一半的受訪者認為澳洲是安全的國家；
- 幾乎四分之三的受訪者認為，全球經濟的嚴重下滑已經構成了嚴峻威脅，只有52%的民眾對澳洲的經濟前景表示樂觀，而一年前這一比例為65%；
- 近四分之三的受訪者表示，乾旱和水資源短缺以及COVID-19對澳大利亞的切身利益構成威脅，是2020年的三大最嚴重的威脅；
- 澳洲人最信任的政府領袖是新西蘭總理雅辛達▪阿登(Jacinda Arden)，其次是日本首相安倍晉三(Shinzo Abe)，然後是澳洲總理斯科特▪莫里森(Scott Morrison)
- 受訪者對印度和印尼等鄰國的信任度幾乎不受影響，分別是45% 和36%.


評：

中共國目前是澳洲第一貿易夥伴，範圍包括農產品，原材料（例如鐵礦石）和酒。貿易額佔澳洲總出口量的三分之一。自從澳洲政府要求對病毒來源進行獨立調查後澳洲關係日趨緊張，中共禁止進口澳洲牛肉並對澳洲的大麥徵收高達80%的關稅，藉此施壓澳洲政府。隨後還對中共國民發布澳洲旅行預警，以及以澳洲歧視亞裔的名義警告中國學生慎重選擇赴澳留學。此次民調是在這一系列澳中關係升級事件前進行的，因此可以想像現階段對中共國的民意會比民調數據更糟糕。

澳中關係隨著中共種種妄自尊大的行為備受考驗。中共秘密增加其核武器種類及數量，拒接參加軍備削減條例的談判，以及最近[對澳洲進行的惡意網絡攻擊](https://gnews.org/zh-hans/238826/)都已經嚴重影響了澳洲的國家安全。澳洲的偉大盟友美國這次疫情受到很嚴重的打擊與損失，儘管澳洲民眾對美國的信任度有所停滯，但是大部分澳洲人依然相信澳洲和美國的堅定的盟友關係會維護澳洲的國家安全。

2020年中共病毒造成的世界大流行以及其帶來世界政治經濟及人文格局的動盪，由於疫情澳洲又出現了29年首次的經濟衰退，因此導致了最新民調的悲觀情緒。此次澳洲民調其實可以代表了大多數西方國家民眾的態度，甚至可以代表很多在中共國內國民的態度。自中共1949年建立政權以來，一直採取欺騙恐嚇以及紅色民族主義宣傳，洗腦了幾代中國國民，也洗腦了眾多西方國家。中共的妄自尊大終將物極必反，西方法制國家及民眾已經慢慢醒來，中共的犯罪行為不久也會被昭告天下，終將受到法律的製裁。
