## 【香港风云】13: 以警治港、以黑治港
` 纽约香草山农场` [轉載自GNews](https://gnews.org/zh-hans/639314/)

作者：香草山写作组 Tiffany的早餐 校对/发稿：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/07015922/logo-3-1-1.jpg)



摘要：721元朗事件、831太子站事件，是标志性的暴力流血事件。港共在香港警队多年布局，从勾结放任黑社会破坏和平抗争，到黑警直接袭击平民。愈演愈烈的警暴把反送中运动推向深渊。

从1989年夏到2019年夏，风云变换，共产党说服了美国留下一条生路，开放市场，加入WTO，加强海外渗透，操纵国际舆论。三十年来世人似乎忘记了共产党过去的魔性，以为富裕可以让共产党变得文明。2019反送中，在世界上最文明、治安最好的香港，共产党三十年前的魔性再次释放出来。

1989六四惨案([香港风云 11: 六四屠城](https://gnews.org/zh-hans/632273/) )三十年后，2019年6月15日下午，反送中运动中的35岁男子梁凌杰坠楼身亡，成为第一位为了香港民主事业公开牺牲的烈士。随之激起的民愤导致香港历史上最大规模的200万人示威游行。之后，香港为民抗争的自杀殉港事件越来越多，越来越频繁，越来越可疑。香港的非自然死亡事件开始激增，从疑似的谋杀很快变为警方（真警察和披上警服的大陆解放军）公开的、大规模的拘捕、刑讯逼供、虐待、性骚扰、猥亵、强奸、轮奸、谋杀、抛尸等血腥的罪行。香港的抗争演化为人类现代文明中难以想象的屠城（[香港风云 9：反送中运动](https://gnews.org/zh-hans/593671/) ）。

共产党从没打算适应香港人的诉求，作为历史上最大的黑帮集团，任何光鲜的外表都掩饰不了共产党杀戮的本性。7月11日，中联办新界工作部部长李蓟贻在元朗十八乡乡事委员会就职典礼上发言，中途他放下讲稿开始讲反送中条例，希望“爱国爱港”的村民，不让“暴徒”进来元朗生事，进来就一定要赶他们走。李蓟贻的发言标志着，中联办官方开始动员暴力破坏香港人的和平抗争。

不久之后，元朗的连侬墙、传单派发、抗争纪录片放映等活动不断遭到地痞流氓的破坏。以效率、专业、廉洁、政治中立等特征在世界上鼎鼎大名的香港警察，对元朗黑帮的横行熟视无睹。元朗地处新界，接近深圳边界，便于与广东联络。元朗当地黑社会的后台是时任警务处行动副处长邓炳强。

香港警务系统是四级管理体系，即郭文贵先生总结的以警治港、以黑治港的祸港四人帮：行政长官林郑月娥、律政司长郑若骅、保安局长李家超、警务处长卢伟聪([香港风云 8：黑云压城](https://gnews.org/zh-hans/593631/) )。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/11111256/2019102807165217565.jpg)卢伟聪和邓炳强
警务处是执行机构，设置两个副处长主管行动和管理。共产党镇压反送中运动后，设置了一个副处长主管国家安全。

邓炳强于2012年至2014年担任元朗警区指挥官，之后升任港岛总区副指挥官、指挥官、警务处助理处长、行动处长、警务处主管行动的副处长。他在元朗任职期间和黑社会关系密切，长期培养可以按照北京意志行动的基层暴力团体。2014年他在港岛总区任职，负责处理雨伞运动（[香港风云 5: 雨伞运动](https://gnews.org/zh-hans/591818/) ）。在警务处他成为反送中运动中实施警暴的骨干，并于11月19日接任退休的卢伟聪出任警务处长，成为祸港四人帮的新成员。

香港的以警治国、以黑治国绝非简单的警匪勾结。邓炳强早在1987年从香港中文大学毕业后即加入警队。共产党很早就培养他，由警方高层安排他到中国人民公安大学、中国浦东干部学院、中共中央党校及国家行政学院受训。值得一提的是，2006～2008年，他还被派往法国里昂的国际刑警组织(Interpol, ICPO)秘书处，出任刑事专向主任，并升任有组织犯罪及暴力罪案组主管。可笑的是，这个职位是打击跨国有组织犯罪的，而邓炳强就是香港黑社会的管理者。更可笑的是，邓炳强的老板CCP，最后将被打成跨国犯罪组织。

他的前任卢伟聪也有国际刑警组织的履历。而国际刑警组织是中共在国际上最重要的渗透对象。2018年10月5日，时任国际刑警组织主席、公安部副部长孟宏伟失踪。两天后，绞肉机CCP通告孟宏伟在北京落马，世人才惊醒共产党早已把国际刑警组织变成了公安部的分部([世纪大案 5：红色通缉令](https://gnews.org/zh-hans/351469/) )。

反送中活动持续多月，在多个周日发起大规模游行。7月21日周日游行的同时，数百名身穿白衣的黑社会分子开始在元朗地区集结。他们以白衣区分反送中活动的主要黑衣着装。之前，元朗区和警方有关系的居民得到警告，勿穿黑衣出门。当日入夜后，参加反送中活动的人士陆续开始返回元朗的住所。

18时许，白衣黑帮开始有组织地袭击黑衣人士。抗议人士回家人流增加，而白衣的袭击者则聚集到近千名。事态越来越严重，而警方接警后拒不出警。晚上22时，大批白衣人在元朗地铁站，持木棍无差别殴打市民。之后，暴徒攻击临时停车的港铁车辆，殴打车内的大量乘客。警方直到白衣暴徒撤退后才到达，引发站内市民愤怒，并围堵警员。黑社会、市民、警察的冲突持续到深夜。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/11111833/Yuen_Long_Station_White_Tee_people_attack_citizen_in_platform_20190721.png)着白衣的黑社会暴徒攻击列车上的乘客
2014年8月31日的CCP人大释法把香港人对政改、对民主社会的梦想撕裂，催生了雨伞运动([香港风云 4: 香港政改](https://gnews.org/zh-hans/591795/))([香港风云 5: 雨伞运动](https://gnews.org/zh-hans/591818/))。五年后的反送中运动正值人大831释法五周年，香港人积极上街示威。而CCP为了五年来的反抗谋划了一场血案。

8月31日晚上，九龙旺角附近的港铁太子站里，突然冲进大批防暴警察，即香港的速龙警队。这些警察冲进停靠的地铁车厢见人就打，场景极其混乱和残暴。施暴方为全副武装的纪律部队，被施暴方则是无区别对待的平民。故事后称为831太子站恐怖袭击事件。

事件中的目击者称有人被当场打死。然而，港铁和港共当局以各种理由拒不提供完整的站内监控资料，试图把事情拖过去。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/11112143/0e095032c10df4edd6945e5d2aea7621.jpg)警察冲入车厢攻击乘客 图片来源：《明报》
香港市民不会忘记831，每个月都会在站前自发纪念。而港共对831惨案的纪念活动也开始干涉。之后通过了港版国安法，港共更加肆无忌惮。到了2020年，虽然香港市民一点都没有忘却831的意思，但在所谓的国家安全的名义下，市民在太子站给死者鲜花，就可以直接被抓捕获罪。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/11112133/160770363979768_P6834856.png)香港市民纪念831太子站袭击事件 图片来源：AFP
港共在香港警队多年布局，从勾结放任黑社会破坏和平抗争，到黑警直接袭击平民。愈演愈烈的警暴把反送中运动推向深渊。在CCP的武装侵略下，港府成为了伪政府，香港成为了沦陷区，香港市民成为了军事占领区的俘虏。香港人自始至终的反抗，就如同对侵略者的反抗一样永不言弃。
