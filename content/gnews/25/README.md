## 武漢CCP病毒死者親屬起訴地方政府隱瞞疫情真相被法院駁回，說明了什麼？
`2020-09-17 11:54 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/364128/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/17114537/1600357505602.jpg)圖片來源：https://commons.wikimedia.org/ 
《法新社》2020年9月17日武漢消息，武漢CCP病毒死者親屬起訴武漢地方政府隱瞞疫情真相瀆職，目前至少已經向武漢市中級人民法院提起了五起訴訟，但是這些案件被法院突然駁回，理由是不符合程序，目前起訴人和律師均面臨巨大壓力。

武漢退休人員鍾漢能（Zhong Hanneng）的兒子在今年二月份死於CCP病毒，承受了作為一個母親最悲傷的噩夢。她與其他喪命於CCP病毒的死者親屬一起，發起法律訴訟起訴對她兒子死亡負有責任的地方政府。死者親屬指責武漢市和湖北省政府在去年底首次爆發CCP病毒時隱瞞了疫情真相，沒有及時向公眾發出警報並採取措施控制傳播，致使CCP病毒失控大爆發。迄今為止，根據官方公佈的數據，CCP病毒已經造成將近3,900武漢人死亡，而全球已經有90多萬人喪命。原告要求約人民幣200萬元（合29.5萬美元）的賠償和公開道歉。

但是，據訴訟人士稱，他們的訴訟被法院突然駁回，數十名起訴人面臨著當局不許起訴的壓力，當局警告律師不要幫助他們。武漢法院駁回訴訟的決定是通過簡短的電話通知的，而不是通過法律要求的官方書面解釋，這顯然是在避免留下書面文字記錄。武漢法院的工作人員拒絕了法新社的置評請求。

CCP病毒去年12月在武漢出現，但武漢市政府當局最初隱瞞疫情，並迫使舉報醫生封口噤聲。共產黨繼續推卸責任，甚至質疑病毒是否起源於中國，同時大肆宣傳其後來在抑製家庭感染傳播方面的成功。 67歲的鍾漢能說：“他們說這種病毒來源於自然。但是，現在這樣嚴重的後果是人為造成的，您需要找出誰應該負責”。到今年1月下旬，CCP病毒在武漢迅速蔓延，但官方仍未在全市範圍內發出警報。隨著中國農曆新年長假節日的臨近，數以百萬計的武漢人到外地度假，將CCP病毒傳播到了全球。鍾漢能告訴法新社：“我們不知道公共汽車上充滿了病毒。所以我們每天都出去。我們甚至都不知道口罩”。鍾漢能和她的兒子彭怡（Peng Yi，39歲的小學老師）在琳瑯滿目的商店裡開心地購物。 1月24日，武漢終於開始封城，而她和兒子卻生病了。雖然她很快康復，但她兒子的病情惡化了。在接下來的兩個痛苦的星期中，他們在不堪重負的醫院裡度過了漫長的時光。先是乞求讓彭怡入院治療，但沒有取得結果。由於缺少測試劑，彭怡一再被醫院拒之門外。彭先生終於在2月6日住院，但兩週後，他死在呼吸機上，死亡時沒有家人陪伴。

作為武漢病毒受害者家屬的代言人和發言人，張海（Zhang Hai）的父親在治療與病毒無關的疾病時在武漢一家醫院被病毒感染，然後死於CCP病毒。張海指責武漢政府當局正在對他進行污名化，封閉他的社交媒體帳戶，並散佈謠言稱法律訴訟是對參與起訴家庭的一個騙局。與此同時，其他人也收到了官方的恐嚇，許多死者親屬加入了一個聊天群，但是死者親屬聊天群已經被警察滲透，聊天群裡的大多數人都不敢輕舉妄動。張海說：“他們知道，如果我能成功提起訴訟，其他許多家庭也會跟著提起訴訟”。武漢市政府對次未回應法新社。

由於在武漢的初審被法院駁回，張海最近向湖北省級法院提起了上訴，鍾漢能也有同樣的計劃。張先生打算不顧個人風險，一路向中共國最高法院上訴，追討公道。他說：“父親是我的動力”。

評：中共國的法院就是中共專制統治的工具，武漢的法院就是受到武漢政府的直接領導和控制的，只會維護CCP和武漢政府的統治，根本不會為老百姓主持公道。武漢CCP病毒死者親屬起訴武漢地方政府的訴訟案件被法院駁回，表明中共國根本沒有公正的法制體系，所謂的法律是騙人的，本質上是人治而不是法治。法院和政府都是和CCP穿同一條褲子的，對中共的法院和法律都不能抱有任何幻想。
