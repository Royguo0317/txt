# 欧盟有沦为中共国奴仆的风险

欧盟正在走上加强与中共国关系的道路，希望带来经济复苏。新冠病毒爆发之前，欧元区已经陷入严重麻烦。读者可能会争辩，但始于2017年的经济疲软期从未停止。布鲁塞尔制定的最新计划似乎政策是扩大并假装一切都很好。2014年，当欧洲央行开始实施量化宽松计划（QE）并将资产负债表扩大至创纪录水平时，欧盟放弃了所有结构性改革。**困扰欧洲的问题是，在2019年，欧元区GDP总增加值的近22%来自旅游业和休闲产业，而该行业不太可能很快恢复。**

[!\[\]()!\[\](https://spark.adobe.com/page/Y9hUwwN0IbTkd/images/1bb03713-95c4-401e-b8b0-4350a2709219.png?asset_id=ea2065b2-2fc0-496c-8568-f505e3cfa4d1&amp;img_etag=%222b3cc74a4f303af293592f226174e52f%22&amp;size=2560)](https://spark.adobe.com/page/Y9hUwwN0IbTkd/images/1bb03713-95c4-401e-b8b0-4350a2709219.png?asset_id=ea2065b2-2fc0-496c-8568-f505e3cfa4d1&amp;img_etag=%222b3cc74a4f303af293592f226174e52f%22&amp;size=1024)

**为了避免联盟破裂，欧盟委员会（EuropeanCommission）去年公布了一项空前的7500亿欧元的新冠病毒恢复计划，**其中包括向成员国提供5000亿欧元赠款和2500亿欧元贷款。这意味着布鲁塞尔正在寻求将他们的权力范围扩展到他们可以借钱的地方，前提是这将有助于结束欧洲历史上最严重的衰退，同时支撑意大利。这将导致欧盟在布鲁塞尔的理事机构发生变化，允许其在资本市场上筹集前所未有的资金，以支持受沉重压迫的欧盟国家。

欧元区实体经济的大约80%由银行业提供融资，银行业的不良贷款超过6000亿欧元。**失业也是一个问题，预计有近30%的欧元区劳动力将在未来几年处于某种形式的失业。**法国、西班牙和意大利对创造就业机会有重要的规定和税收负担，可能遭受甚至更长时间的大量失业。截至2017年，没有一家欧洲公司跻身全球15大技术公司之列，全球50强技术公司中只有4家是欧洲公司。这就是为什么持怀疑态度的人担心的是，如果以政治为导向的“绿色新政”议程不能促进经济增长或减少债务，欧元区将在经济上停滞不前。

**欧元区根本没有竞争力是显而易见且一直回避的问题（The elephant in the room）。**欧盟缺乏技术和知识产权，正进一步落后于美国和中共国。该地区的制造业大国德国继续吃力地试图避开经济衰退，而法国、西班牙和意大利面临多年的高失业率。显然，欧盟在2020年春季陷入困境，当时欧盟委员会因新冠病毒大幅下调了对该地区的经济增长预测。到目前为止，欧盟委员会对经济将迅速反弹的预期已被第二波病毒大流行所破坏。

**为了给人们留下希望的印象，欧盟在布鲁塞尔的领导人正试图通过加强与中共国的联系来出其不意地摆脱困境。**《卫报》最近报道说，中共国和欧盟现在似乎已经解决了他们在保护中共国劳工权利方面的分歧，并准备签署一份拖延已久的投资协议。这将加强它们之间的联系，并使两个集团的经济更加相互依存。此次投资会谈针对的是对欧洲的投资开放中共国市场，并讨论欧盟反对的中共国有关工业补贴、国家对企业的控制和强制性技术转让的做法。

**2013年启动的会谈的症结，在于维吾尔族穆斯林的待遇以及在香港有组织地压制言论自由。**这些谈判的核心一直是欧盟对这些问题的关注以及如何执行和仲裁协议其它部分。必须指出的是，欧洲议会必须批准该协议，同一成员最近通过了谴责在中共国使用强迫劳动力的决议。同样，美国和即将上任的拜登政府对中欧全面投资协议并不满意，这标志着欧盟对亚洲政策的重大转变。

拟议中的交易与北京的“一带一路”倡议（OBOR）相吻合，并且是在与意大利签署一项协议后签署的，意大利被许多人视为破产。去年，意大利总理在罗马与中共国国家主席习近平签署了具有历史意义的谅解备忘录，这可算是一个大胆的举动。该协议使意大利成为正式签署“一带一路”协议的第一个创始欧盟成员国和第一个七国集团国家，希望它能够支撑其疲弱的前景。最终，意大利与中共国达成的协议可能会证明是与恶魔的协议。

**中共国努力与贫穷、软弱但讨人喜欢的意大利达成协议的主要动机是，它渴望剥削意大利并将其用作扩大欧元区市场的后门。**中共国和意大利签署的协议包含了发展协议，涉及港口管理、科技、电子商务甚至足球。现实情况是，中共国渴望控制可以合法扩大的欧盟入境点。这对于该地区来说不是一个好兆头。

**过去，欧洲年复一年与美国贸易顺差，而对中共国而言却并非如此。**根据欧盟统计局（Eurostat ）的数据，2019年欧盟对美国的顺差为1530亿欧元（1803亿美元），这意味着对美国的出口额大于进口额。欧盟是中共国的第二大贸易伙伴，但从中共国的进口额远高于中共国从欧盟的进口额。这些数字不是异常值，而是对多年来一直在增长的趋势的证明。简而言之，美国一直背负着欧洲，从美国流向欧洲的金钱和财富却很快流向了亚洲。

可以说布鲁塞尔正在领导欧盟进入中共国的伏击圈，欧洲无法对中共国维持自己的立场。过去，美国和欧盟都抱怨中共国希望自由贸易而不公平竞争，认为中共国是一只老虎，突然濒临精神错乱改变了毛色。与中共国的条约不会纠正市场失衡，也不会给欧洲提供相同水平的市场准入或投资者寻求的非歧视性环境。他们会发现，这不是中共国第一次签署它不尊重的协定。**欧洲已经看到它的制造业因中共国和其它低工资国家的廉价仿制品而贬值，因此将更多此类商品引入自己的市场只会有小小不舒服。**

**可以说中共国的系统是为剥削而设计的。**中共国的国有经济基于掠夺性商业模式，旨在通过压制竞争来扩大规模。中共国决心进军高科技产品，其计划的重点是国有和私营公司投资和收购外国公司以窃取其技术创新。用许多方式补贴那些在其系统内工作的公司可以帮助中共国实现这一目标。中共国以略低于成本的价格出口商品，以便从其它国家获得制造业就业机会，这不会改变。**我们这些对中共国抱有这种看法的人认为，与中共国建立更紧密联系的举动可能会加速欧洲的灭亡。**
