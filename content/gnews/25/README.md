红领巾用三个容易打动人心的名义来入侵孩子们的心——荣誉感、爱国热情，以及对“革命先烈”（这里指打倒“国民党反动派”的、“牺牲”了的中共军人）的敬重。

首先说荣誉感。少先队员是“少年儿童学习中国特色社会主义和共产主义的学校”、“建设社会主义和共产主义的预备队”，给人感觉十分高大上，似乎加入了就能有学习“先进”的“共产主义”和“中国特色社会主义”的好机会，能够“建设”被中共吹得天花乱坠的“社会主义社会”，因此孩子们通过对红领巾的喜爱，加深了对中共的热爱。

第二，爱国热情。基于上一段提到的荣誉感，加上中共将爱党与爱国混淆，能够建设“社会主义和共产主义”自然会激起孩子们的爱国爱党热情，进而对共产党顶礼膜拜——谁敢批评党、批评政府，谁就是“反动分子”、“公知”、“美分”。

最后讲中共国民众对“革命先烈”的敬重之情。因为中共自始至终强调，“人民当家作主”的政权是“革命先烈”的鲜血换来的，因此应该“珍惜现在的美好生活”——这样给民众灌输对中共的感激和敬仰之情。一旦有人提出改革社会制度，提出对中共的质疑，马上就会有民众去指责他不珍惜现在“稳定”、“和平”的生活，如果这个人被惩罚了，民众会拍手叫好。
