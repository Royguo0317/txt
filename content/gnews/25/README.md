**我要做什麼**

作為牆國居民，翻牆是一個獨特且有趣的經歷。能在Gnews上看到這篇文章的牆民，一定對翻牆已經非常熟悉。然而，熟悉不等於已經做好準備。作為爆料革命的牆內戰友，翻牆只是尋求正義和真相的第一步。在爆料革命的大潮中，翻牆戰友有以下幾個層次的遞進目標：

1. 獲取爆料革命的消息，保護自己和家人；

2. 註冊社交平台賬號，發出自己的聲音；

3. 參加爆料革命農場，尋求投資和滅共；

4. 加入爆料革命活動，直接貢獻力量。

以上四個目標層層遞進，隨著目標以及行動的深入，被CCP發現的後果就越來越嚴重。對於一個以遊客身份觀看路德社節目的戰友，被發現後可能也就是“批評教育”一下，而如果加入農場義工被CCP發現，那也許就真的被消失了。

因此，我們做一些事情之前，一定要意識到自己行動的危險性，時常評估自己的軟件、設備、留下的信息會不會為自己帶來不必要的危險並迅速作出調整。誰都不願在眼看就要天亮的時候倒下去，您說是不是？

目前，爆料革命已經到了一個非常關鍵節點，邪共在滅亡的時候一定會想方設法拉著全體中國人一起下地獄，為了全球華人的子孫和未來，我們戰友需要多多發聲，讓全世界看清楚，我們是反共的，是勇敢站出來滅共的，中國人是善良、勤勞、誠實的，是值得交朋友、做生意的。在這個關鍵的節點上，能否在保護自己不受CCP迫害的同時勇敢的發出聲音就顯得非常重要了。

**變身無面人**

不管使用哪種翻牆技術，其本質都是把數據加密，讓防火牆不知道你在看它不想讓你看的東西從而繞過審查。然而在牆內，使用VPN、上網消費、甚至買菜刀都要實名化的今天，數據加密絕不等同於身份匿名。計劃深入參與爆料革命的牆內戰友一定要對此有所準備。在翻牆之前，就要有意識的學習、收集甚至去創造“去實名化”的方法。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/26144335/12-5.png)
以梯子的使用為例，我們都知道牆內能夠下載的VPN翻牆軟件，CCP一定會進行監控。但我們可以使用這類VPN翻出去搭建自己的梯子，然後通過自己的梯子翻牆來實現“去實名化”，讓CCP找不到我們。關於去實名化的具體方法以及注意事項，未來小編將會結合自己的經歷，在本系列文章中與大家逐步分享，幫助牆內戰友們以安全的滅共。

**規劃虛擬身份**

翻牆出來，我們都是以暱稱相見。在使用這類虛擬身份發聲前，有一件事請很容易被忽略，而這件事情卻是保護自己的關鍵：關於我的真實身份信息，哪些可以說，哪些打死也不能說？這一點看似簡單，實則在被套話、被激怒，被忽悠的情況下很難克制。我的建議是，有關自己的真實身份信息，把能說的寫下來，反复看，形成條件反射。

比如，小明作為戰友翻牆出來，給自己立下7條鐵律：

1. 性別男，愛好女，可以說。

2. 年齡設定是45歲，可以說。

3. Gmail郵箱地址，可以說。

4. 家住上海，可以說，再詳細堅決不透露。

5. 家庭成員工作狀況，堅決不提，也不透露。

6. 戰友限discord或whatsapp交流，拒絕其他聯繫方式。

7. 不透露自己的照片，不露臉的照片也不行。

如果能夠像小明一樣，制定幾條鐵律並嚴格遵守，可以避免很多被CCP喝茶甚至更嚴重的傷害事件。特別是經歷過最近借款或投資項目臨時突破了這些鐵律之後還能夠回來繼續堅持是非常不容易的。

相信大部分牆內戰友在面對家人、朋友的時候，都會感到孤獨，甚至失落和些許的失望。我們保護好自己，也要觀察身邊的人文環境是否足夠安全，確認可以在牆內分享的信息。比如我們的挺郭經歷，我們的推文，海外賬號信息是否允許家人或朋友知曉？如果決定保密，那麼我們翻牆專用的手機平時應該保存在哪裡？身邊的自乾五同事打嘴炮的時候，哪些事情堅決不能說？

**其他的牆內準備**

保持穩定的翻牆能力，家裡面使用的設備和軟件肯定需要升級。也許您現在已經順利的在翻牆，但還是推薦將以下措施補齊，提高安全防護能力：

1. 所有國產路由器均留有後門，採購牆外品牌的路由器（如台灣某廠）

2. 購買爆料革命專用翻牆手機，不能是二手、破解版本，不插SIM卡。

3. 購買國外電話卡或者虛擬手機號碼，不在日常手機上使用，非必要不聯網。

4. 由於微軟與360進行了深度合作，避免使用牆內版本的windows，正版也不行。

5. 使用膠帶封住手機和電腦攝像頭，關閉所有定位服務。
