## 新西蘭總理Jacinda Ardern 和副總理及外交大臣Winston Peters反對共產黨的香港國安法
`2020-07-02 01:14 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hant/251637/)

翻譯：紅鬍子

資源:newshub

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/02005314/220px-Ardern_Cropped.png)

總理賈辛達·阿登（Jacinda Ardern）和外交大臣溫斯頓·彼得斯（Winston Peters）加入全球聯盟，譴責中國對香港有爭議的新國家安全法北京周二繞過香港的地方議會通過了這項立法。正如新西蘭政客所訴，該立法打破了一直在香港運作的“一國兩制”框架彼得斯星期三發表的聲明說，新西蘭政府對中國全國人大常委會通過的香港國家安全法表示“深深的失望”。

這項新法律定義了四種罪行：分裂活動，顛覆國家，恐怖主義活動，以及與國外勢力勾結，這些罪行最高可判處終身監禁彼得斯說：“新西蘭一向強調其對香港國安法的嚴重關切，該法律的通過沒有進行包容性磋商或所有香港機構的適當參與。”“我們與國際社會一樣，期望香港及其人民在「一國兩制」的架構下維持高度自治和自由”“正是這種自1997年以來的自治和自由，配合開放的治理，獨立的司法，以及對法治的共識，在香港成為全球金融和經濟樞紐的發展中起到至關重要的作用”.

總理阿爾登（Ardern）週二被問及對有關中國已通過新安全法時，回應說，政府已明確表態。 “我們已經清楚地認識到這項立法將會終結「一國兩制」，因此我們對此有明確的看法，並對此作出了表述，當然，我們看到了這項立法已經造成的破壞”“有時您會看到集體陳述，有時會看到單個陳述……您會看到相同的消息傳達，五眼聯盟夥伴與我們在這個問題上所採取的立場之間幾乎沒有什麼區別。 ”

綠黨司法發言人戈拉茲·格哈拉曼（Golriz Ghahraman）在推特上將香港國安法描述為“嚴重違反香港的民主和人權的行為”中國駐新西蘭大使館已經對此事回應英國駐聯合國大使朱利安·布雷思韋特（Julian Braithwaite）代表包括澳大利亞和新西蘭在內的27個國家發表聯合聲明，對新法律表示關注美國政界人士一直在嚴厲批評，其中包括眾議院議長南希·佩洛西（Nancy Pelosi），她表示香港國安法為“殘酷的”法律，其目的是“嚇,、恐嚇”和“壓製香港人民的言行”美國國務卿邁克·龐培（Mike Pompeo）將其描述為“殘忍的”法律.

香港，作為中華人民共和國香港特別行政區，在1997年以“一國兩制”的方式從英國回歸北京彼得斯說，中國需要保護人權。他說，新西蘭政府將研究這項立法，以了解它將如何影響香港人民“對於保護香港人世世代代的基本人權和自由，現在是關鍵時刻。新西蘭將認真研究這項立法，並密切監測其實施情況和對香港人民的影響，我們與香港人民有著密切的關聯。”

北京方面說，自去年6月以來有時發生暴力反政府和反北京抗議活動之後，該立法變得必要。新西蘭在4月與美國、英國曾一起譴責北京逮捕反北京的香港民主運動分子。當時，香港警方逮捕了15名民主運動人士，其中包括民主黨創始人馬丁·李和工會領袖李卓陽。他們被指控犯有組織和參加“非法集會”等罪行。在香港國安法中，損壞某些運輸工具和設備將被視為恐怖主義行為，任何被判為違反新的安全法的人將被禁止參加任何香港選舉。
