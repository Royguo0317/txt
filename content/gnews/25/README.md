## 楊潔篪出訪新加坡、韓國 “討論”疫情應對及朝韓“安全”
`2020-08-20 01:27 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/303632/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/20012026/65769_61306_800_auto_jpg.jpg)圖片來源：China Daily 
據南華早報2020年8月19日報導中共的最高外交官楊潔篪將在本週四周五分別訪問新加坡和韓國。

楊潔篪是中共權力中心—政治局委員的24人之一，中共外交政策的規劃師。外交部宣布楊潔篪定於週四訪問新加坡，屆時將會見新加坡國務資政兼國家安全統籌部長張志賢和外交部長維文，並將與李顯龍總理和副總理財政部長王瑞傑進行電話會議。據報導楊潔篪將於週五訪問釜山，並將於週六與韓國國家情報院長徐薰舉行會談。

此次出訪被認為是在地緣政治競爭加劇的環境下，中共尋求加強與亞洲鄰國的聯繫以緩解壓力的表現。

這將是自2018年7月以來楊潔篪首次訪問韓國。雙方“計劃在周六上午進行會談，並舉行午餐會，並就韓中等共同關心的問題交換意見。 中共病毒應對合作，包括高層往來在內的雙邊關係以及朝鮮半島和國際安全局勢”，預計兩人還將討論習近平出訪韓國的可能。韓國總統文在寅在2017年的北京之行中邀請習近平於2020年初訪問韓國，但因中共病毒大流行而推遲。習近平上一次訪問首爾是在2014年。

現在中共與美國正處在非常緊張的局勢，包括貿易，技術，香港，台灣，南海等諸多問題。美國上個月再次指責中共對南海有爭議水域的廣泛主張是非法的。

評論：根據郭文貴先生早前透漏，中共主席習近平將於近期訪問韓國，那麼這次楊潔篪的訪問是不是給習近平訪問韓國進行鋪路呢？自從今年6月18日楊潔篪和彭培奧國務卿在夏威夷會面之後，中共與美國基本處於斷交狀態。現在川普總統執政的美國政府已經完全不再信任中共，並即將在世界上刮起滅掉中國共產黨的狂風暴雨。中共已經失去很多盟友，中共政府已經搖搖欲墜，現在拉攏新加坡、韓國兩個鄰國以挽救獨裁政府，最終估計會無功而返。楊潔篪所代表的獨裁政府繼續欺騙周邊國家已經完全不可能。世界各國都看清共匪欺騙世界的真面目，不會再信任共匪的任何話。
