**中共的目標——白宮幕僚長**

白宮幕僚長或白宮辦公廳主任，是**美國總統辦事機構的最高級別官員，**通常**負責監督白宮幕僚的行為、管理總統的日程安排、並決定允許誰與總統會面，**是一個**擁有很大權力的職位，**常被成為**“華盛頓第二最具權力的人”。**

白宮幕僚長的職責**既是管理上的又是諮詢上的，**它包括：

1. **選擇白宮主要工作人員並進行監督**
2. **構建白宮工作人員制度**
3. **控制進入橢圓形辦公室的人員流動**
4. **管理信息流**
5. **保護總統的利益**
6. **與國會、行政部門的其他成員以及政府外政治團體進行談判，以執行總統的議程**
7. **就各種問題向總統提供建議**
