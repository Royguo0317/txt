## 港幣脫鈎引爆中美深度金融決戰
`2020-07-11 18:26 喜馬拉雅戰鷹團 Himalaya Hawk Squad` [轉載自GNews](https://gnews.org/zh-hant/260523/)

香港與中共國的經濟關係：

1. 中共國通過香港購買科學技術和工業材料、醫療器材、軍事敏感技術和敏感地區的能源。
2. 進口和出口各類生產資料、以香港為中介背書認證（被英美法律認同的條文和制度），借過香港之橋連結中共國與歐美日包括邪惡軸心國之間的中轉貿易。
3. 中共國從香港輸入世界專業人才並同時輸出自己的特殊科技人才，滲透和偷騙外國企業和大學里高科技國防5G芯片等敏感行業的技術與情報。
4. 中共國國企、軍企各種海外的非法收入，經香港聯繫匯率金融系統洗成合法收益，再隱藏在世界各地的離岸公司、賭場、家族信託中由各國銀行、證券公司、基金等受託和代持。
5. 中共國利用香港偷竊騙取收買台灣、歐美日、東南亞等世界各國的軍事、政府、航天科技和經濟金融的各種情報。
6. 80%的中共國國有企業的融資，充分利用亞洲最大的香港證券交易所和最完善的世界級金融服務，騙取香港股民和全世界的金融投資者或機構的美元和信任。

香港是內地第三大的貿易夥伴，僅次於美國和日本。2019年香港政府統計顯示，香港的主要貿易夥伴是中共國，佔香港整個貿易總額的51%，貿易總值達4.2萬億港幣，第二是美國佔6.2%達0.5萬億。中共國經香港轉口貿易額達2.2萬億。5月香港財政司司長陳茂波表示「生產總值GDP按年跌8.9%已進入深度衰退，其中三架馬車出口、消費、投資已經死咗火.」同時中共國經濟在天災人害下更糟糕，百姓在水深火熱中。2020年中共國六大基礎產業全部嚴重衰退：包括服裝業、電器業、汽車業、化工業、有色金屬和黑色金屬加工業。而極其荒謬的卻是中共國的上證指數從6月29日2900點在10天里如火箭般急升到了3400點。這讓我想起全球股票指數瘋牛最高的委內瑞拉和第二辛巴威股票指數，而緊跟其後的狂熊股市，導致委內瑞拉貨幣巨貶到今天的一美元兌換21萬波利瓦爾。當年法國被德軍兵臨城下時，股市還能暴漲到最後一分鐘閉市關門，法國百姓如韭菜般最後徹底被刨根。中共國在自己七十年假騙偷養大了的骯臟軀殼最終倒下時，也要邪惡凶殘的榨乾中共國人民的最後一滴血嗎？

今天美國已經發行了26兆美元的國債，世界各大央行史無前例的壯大自己的資產負債表，其中美聯儲剛滿7萬億，已證明世界正處在嚴重的金融危機衰退中。根據爆料革命提到的，美國兩院都已通過的「香港自治法案」，一旦川普總統簽署，則可以勿庸置疑的預期30-60天內，中共國和香港的假股市，以及中南坑常委們串謀英美華爾街金融大鰐們，在專制獨權下最後一同合演的嗜血成性、「龍膽虎威」的熱錢騙局被撕下偽裝後，將風捲殘雲般徹底崩潰！

中共國和香港股市即使不關門，數字貨幣、一帶一掠的人民幣結算的內循環也是其切斷命門後的垂死掙扎。中共國輸了金融輸血的香港，這是多米諾骨牌連鎖效應的第一張牌，隨後中共國股市會倒下，嚴重影響東南亞股市、歐洲和美國股市。如CCP病毒肆虐世界一樣，不聯合世界金融體系打爆香港和CCP假股市，世界股市都會有危險；不消滅共產CCP制度的病毒，世界民主國家不得安寧和平，世界人民不會安全健康！滅掉中國共產黨，中國人民將迎來自由、民主、法治的新中國聯邦！華夏大地將會將權力關在籠子里、還權於民、保護私有產權的新時代！世界人民將迎來與普世價值觀相融的、與世界千年和平的新中國聯邦！
