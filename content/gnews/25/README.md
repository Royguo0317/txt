作为其现代化工作的一部分，美国陆军正在研发中型和远程导弹，该部队将其视为与太平洋中的中共潜在冲突的重要武器。

陆军传统上是陆基兵种，但它正在发展快速移动的跳岛部队，以打击西太平洋的敌舰，类似于海军陆战队传统上所做的那样。

周二，美国陆军领导人讨论了该举措在印度洋-太平洋的未来，并表示，由于军事领导人担心资源有限，美国陆军不会与海军陆战队或其他分支机构竞争资金。

“我们正在开发的系统更多是按照战役的路线而不是快速的远征型系统。因此，我认为我们不会与海军陆战队竞争。” 陆军参谋长麦康维尔（James McConville）告诉战略与国际研究中心。

美国陆军太平洋地区负责人保罗·拉卡梅拉（Paul LaCamera）将军说，陆军可以充实该地区的海军陆战队。他说：“因此，我可以先认定他们（海军陆战队）是否已进入该地区，我们可以布局在他们的身后，从而保障他们没有后顾之忧继续前进。”

美国海军陆战队也正在改组军队，为太平洋地区未来的冲突做准备。美国印度太平洋司令部向国会提供了增加$ 27+亿资金预算计划，该司令部希望部署包括在中共国大陆沿海海域附近的远程导弹网链，尽管目前尚不清楚哪些国家愿意实际部署这些导弹。

星期二，美国陆军军官被问及美国如何让该地区的盟国接受部署此类导弹。麦康维尔说：“这是一个政治决定。” “我相信，[拜登]政府将出台一项政策，这将取决于我们（与盟国和合作伙伴）两国之间的协商解决。
