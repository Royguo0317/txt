作者：左媛

上文谈到迟浩田的“民族优越感”，并意淫“世界文明中心转移论”，转移到中国，准确地说是转移到“中共”。那么，中共会怎么做呢？本文继续揭露共产党的“灭白计划”。

迟说，上世纪末，中共在北京建了一个“中华世纪坛”，“新世纪来到的那一刻，中央领导集体在那里高举周口店火炬，举行了迎接中华世纪的誓师大会，就是要按照历史规律，把实现中华世纪做为我们党的奋斗目标。”“十六大报告中写进了“民族复兴”的伟大目标，在新党章中规定，我们党是中华民族的先锋队……体现了我们党的胆略。”

这两段话可以说是共产党要独霸世界21世纪的纲领性讲话，几个关键词都出来了：中央领导集体，誓师大会，奋斗目标，十六大报告，民族复兴，新党章，先锋队，我们党……

非常清晰，所谓“中华民族的伟大复兴”和中华民族没有任何关系，和中国人民没有任何关系。这个“党”是“我们”的，和其他人也没有关系，和普通老百姓更是八杆子打不到一处。但是它要把普通老百姓绑架在一起，有灾的时候老百姓要承担，有难的时候要老百姓冲在第一线，缺钱的时候薅老百姓的羊毛，战争的时候驱使老百姓当炮灰……唯独有利益的时候老百姓靠边站。中共深知自己的历史和现实每一页都是罪恶累累的，为了自己的生存和发展，必须绑架更多的人要么同归于尽，要么为它的存活而挣扎！

读了迟浩田的这个军中高层会议报告，我们就能很好的理解，中共为什么要在美国和全球行驶“3F”和“蓝金黄”，为什么要制造并释放病毒。只是我们知道的时候，中共已经作恶又有20年了，以至于它和世界的黑暗势力纠缠在一起，很难分离。这就是一种利益的捆绑邪恶的绑架。

它能做到吗？它能做到，并且在很大程度上已经成功。它是怎样做到的？还是看迟浩田说吧。

“我们要借鉴人类历史的宝贵经验，掌握人类文明一切优秀成果，吸取其他民族的教训。”众所周知，几百年里世界的文明在西方。现代社会几乎所有的民主法治自由和教育科学技术成果等等都在西方，而中华民族的优秀成果被历朝历代帝王尤其是中共的统治所屏蔽，统治阶级实施的是“驭民五术”，民间留传的是“三十六计”……精华淹没在历史烟云，糟粕却盛行于人们的言行，这不能说是我们这个民族的悲剧。中共的所谓“掌握人类优秀文明成果”不过就是对西方多年的先进科学技术做下三滥的“偷骗抢”手段，为了达到目的无所不用其极，而真正本质上的人类文明成果是“法治民主自由”，是三权分立是对人权的尊重，这些都是被中共强力屏蔽的。

它所说的“教训”又是什么呢？是德国日本的失败和苏联东欧的亡党。迟浩田说，“我们在探求大国兴衰定律的政治局学习会上……在确定走德国道路复兴中华时，绝不能重犯德国日本犯过的错误。”
