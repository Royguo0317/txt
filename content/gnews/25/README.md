## 【快讯】债务即将“爆雷”，全球五大锂矿供应商之一天齐锂业面临违约风险

内新闻/素材：鹰(文言) 校对：文迹~见证神迹
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/14074508/PSX_20201114_161529.jpg)
澎湃网11月13日报道，由于现金流吃紧，天齐锂业股份有限公司将无法偿还于11月底到期的18.84亿美元债务，又一大型资源性企业面临违约风险。

天齐锂业此前与中信银行牵头的并购贷款银团签署了相关协议，并购贷款金额中的18.84亿美元于本月底到期，而天齐锂业最近的审计净资产约为10亿美元，仅占该债务的55.8%，加上流动性紧张，无力偿还。

天齐锂业于2004年10月成立，注册资金达14.7亿元，主要经营碳酸锂、锂矿及锂化工产品的制造和销售，是国内最大的锂电新能源核心材料供应商，也是全球五大锂矿供应商之一，在2019年全球新能源企业500强榜单中排名第149位。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/14074729/image1-25.jpg)
天齐锂业自成立以来极速并购扩张，2010年在深交所上市，营收3亿、净利润3000万；

2014年5月收购澳大利亚泰利森母公司文菲尔德51%权益，间接控股泰利森锂业；

2015年4月收购银河锂业国际100%的股权，将江苏张家港1.7万吨碳酸锂加工基地收入囊中；

2016年10月，“年产2.4万吨电池级单水氢氧化锂项目”动工，在2019年第三季度竣工并试生产，将锂精矿产能增至134万吨/年；

2018年以40.66亿美元拿下智利SQM公司（智利第一大、全球第二大锂生产商）23.77%的股权，成为SQM第二大股东。

凭借Greenbuhes的资源优势（泰利森拥有的格林布什锂矿，世界上储量最大、品质最好锂辉石矿藏，碳酸锂储量424万吨、品位高达2.8%）和两大加工基地的规模，天齐锂业成为国际锂业巨头。

2014-2019年，天齐锂业营业收入由14.22亿元增长至48.41亿元，复合增长率27.76%；归母净利润由14年的1.31亿增长至18年的22.00亿元，复合增长率达102.44%。

为并购SQM，天齐锂业筹备52.79亿元更新增35亿美元并购贷款（境内25亿美元，境外10亿美元），财务费用大增，仅2019年并购贷款利息合计就达16.50亿元，一方面由于债务累积，流动资金趋紧；

另一方面又由于锂精矿销售数量和锂化工产品价格下降，营收和利润严重下滑。2019年归母净利润降至-59.83亿元，2020年第三季度营收为24.27亿元，净利润为-11.03亿元，同比下降890.95%，经营活动产生的现金流为6.65亿元，同比下降51.94%。

截至三季末，天齐锂业总资产为426.7亿元，总负债为346.77亿元，负债率高达81.27%（流动负债185.30亿元，一年内到期的非流动负债为133.05亿元）。

除天齐锂业面临资不抵债、违约风险外，其控股股东天齐集团未来一年到期的质押股份达3.55亿股（占持股比例75.82%，公司总股本24.03%），若业绩持续下滑，股价继续下跌，也将面临偿还质押融资或补仓。

作为四川省重点扶植的企业，天齐锂业从成立之日起就直接是由当地政府控制和政策资金扶持，再加上其历次的大型收购，背后都有中共政府对外扩张侵占全球锂矿市场的意图。

通过吞并泰利森，持股SQM，中共利用天齐锂业短期内就控制了全球最大的锂辉石资源，占据全球五大锂矿供应商之一的席位，可以说天齐锂业是中共对全球锂矿资源瓜分的重要工具。

背景雄厚、坐拥“金山”的大型企业却资不抵债，面临违约风险，从中也能看出其背后金主中共的资金流吃紧、经济不堪的现状，尤其是中美贸易战以来，西方技术的“断供”限制了国内锂化工产品的开发和升级，加上原有技术不成熟、产业链断裂，国际市场更缺乏竞争力，加剧了营收亏损。

在2020年CCP病毒爆发后，全球经济萎靡，而中共前期将大量资产泡沫化、房市化，导致整个社会通胀通缩并存，经济日渐崩溃，在此大背景下，天齐锂业即便有“国资”注入，也只是苟延残喘，终究会破产倒闭。

再者，中共近年来加大对锂电池的市场开发，2019年以来更就在国家政策上导向电动汽车市场，国内电动汽车企业井喷式发展，电动车市场也颇具规模，但作为重要原材料供应商的天齐锂业却业绩下滑、入不敷出，这其中又该有多少中饱私囊、盗为私用？
