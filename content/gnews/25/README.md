## 你只想賺點利息，盜國賊們要的卻是你的本金
`2020-10-21 09:03 澳洲雅典娜农场` [轉載自GNews](https://gnews.org/zh-hant/438143/)

作者：翼族編輯/審核：Giselle
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/21084630/Picture8-4.png)圖片來源： [https://inewsdb.com](https://inewsdb.com/%E8%B2%A1%E7%B6%93/%E9%B5%AC%E8%8F%AF%E8%B3%87%E7%AE%A140%E5%84%84%E7%94%A2%E5%93%81%E7%88%86%E9%9B%B7%EF%BC%8C%E6%98%AF%E4%BD%95%E5%8E%9F%E5%9B%A0%E6%89%80%E8%87%B4%E5%BC%95%E9%97%9C%E6%B3%A8/)
在G系列受到全球投資者熱捧的同時，中共國內的投資圈卻是一片哀鴻。前些天傳出的鵬華基金聚鑫系列產品違約超40億，雖然是最近才出來的新聞（ [https://dy.163.com/article/FPDMPUTV0519H1CT.html](https://dy.163.com/article/FPDMPUTV0519H1CT.html) ），其實鵬華基金從今年七月份開始就已經無法兌付了。

自打汎亞e租寶洗劫數十萬投資人以來，中共國的金融圈爆雷事件已經屢見不鮮了。但是作為一款預期收益率4.1%的固定收益類產品，其發行人“鵬華資產”又是中共國頂級公募“鵬華基金”的子公司，而且還是通過銀行業市值第一的工商銀行代銷的，連如此令人放心的理財產品都不能按時兌付，不得不讓所有人都問到底是怎麼回事？

據媒體曝光，此次爆雷的“鵬華聚鑫系列”100萬起投，期限為6個月，滾動發售。其中“滾動發售”說穿了就是藉新錢還舊債，從這些富裕人士的左口袋裡掏出錢來，再還到他們的右口袋。加上鵬華聚鑫系列總共有28款產品，所以按照其原計劃，也許是可以在這個資金池中四處輾轉騰挪的，只是由於其大量購買的海航債券無法按時兌付，即使延期了兩次兌付，最後鵬華資產還是不得不站出來公告其無法完全兌付。

按照新聞中分析的原因是什麼所謂的“期限錯配”問題，說白了就是藉進來的新錢付不了舊債了。但是依我看，或許這本來就是一個陷阱，是海航的這些盜國賊們從高淨值中產人士的口袋偷錢，並最後由國家和人民來為他們的偷竊行為買單的卑鄙行徑。

先來看看這個基金產品牽涉的兩大主體之一，鵬華資產。鵬華資產成立於2013年，控股公司為鵬華基金，而鵬華基金的最大股東為國信證券，佔股50%，所以鵬華的實際控制人應該是國信證券。 2015年原國信證券的總裁陳鴻橋自殺，是國信證券在全國引起很大關注的一個事件。陳鴻橋，曾經是原證監會主席助理張育軍的副手。 2015年9月張育軍受到中紀委調查後被免職，陳鴻橋10月23日在家中自縊身亡，留下遺書，上面只有五個字“請勿擾妻兒”。眾所周知中共十八大以後，王岐山出任中紀委書記，以反腐名義對證券行業進行了大清洗，相信最後國信證券也被其拿下。陳鴻橋作為一個馬仔，被拋出來抵罪，也許他最後由於家人被威脅而不得不自行了斷，做了王岐山的刀下鬼。

在這樣一個背景下，也就不難理解為什麼鵬華聚鑫系列理財產品不是配置不同主體的資產以分散風險，而是大量買入海航債券這單一的高風險資產。王岐山和陳峰波切們只不過是把海航的垃圾重新包裝了一下，硬塞給鵬華資產而已。

而另外牽涉到的一個主體，是工商銀行。由於鵬華資產聚鑫系列產品的爆雷，目前相關的處置兌付將由工商銀行來進行。工商銀行將於2020年10月20日兌付該產品的50%，其餘50%轉成工商銀行的其它理財產品延後一年兌付。一個擊鼓傳花的遊戲，但是相信絕大多數的投資人在這樣一個前後交困的情況下，最後都會不得不選擇這個解決方案。

然而,工商銀行既然只是一個代銷機構，為什麼要背上這個黑鍋？一方面也許是法律上的問題，另一方面雖然工商銀行不像建設銀行和中國銀行屬於王岐山的嫡系，但應該也在其控制之下。十九大以後，王岐山表面上退出了權力核心，但是他最鐵桿的馬仔之一周亮卻到了銀監會出任副主席，所以他對中共國金融行業的控制力並沒有削減。這也正是為什麼2017年郭先生爆料海航，導致海航在海外圈錢的能力被徹底腰斬後，中共國的銀行繼續給海航貸款供血的原因。 2019年由國開行牽頭的40億貸款裡面，就有工商銀行的身影。

通過以上所描述的一系列暗箱操作，海航的垃圾資產被堂而皇之地賣給了鵬華資產，再包裝成固定收益類的銀行理財產品，無法兌付後再由工商銀行來買單，而其實真正買單的卻是千千萬萬個普通的中國百姓。這就是盜國賊們的醜惡嘴臉。

在中共國的假擀麵杖經濟裡，從房市到股市，從P2P到數字貨幣，中共盜國賊們對韭菜們收割了一茬又一茬。現在，隨著中共國一家家地方銀行的破產倒閉，他們開始連銀行固收類理財產品也不放過了。相信在接下來的日子裡，這樣的金融爆雷事件會越來越多。我只想提醒中國同胞們，捂好你的錢袋子吧，你只是想賺一點利息，但是盜國賊們要的卻是你的本金！
