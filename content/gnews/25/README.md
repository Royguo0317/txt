## 【戰友點評】12名抗爭者逃亡失敗被捕送中，內地律師受家人委託會面被拒

整理：文燕 文锦

```
    上月23日以“香港故事“成員李宇軒為首的12名港人，在潛逃台灣途中被廣東海警截獲，他們被指違反《國安法》，至今仍扣押在深圳市鹽田看守所。
     內地律師盧思位向《立場新聞》透露，他受其中一位被捕喬姓女子的家人委託，到鹽田看守所試圖會見當事人，但遭拒絕；當局拒絕理由是：當事人有可能涉嫌“組織他人偷越國邊境罪”，俗稱”蛇頭”，比起拘留通知書寫的“偷越國邊境罪” 更重，一旦定罪，最高可判囚終生。
     盧思位亦向《蘋果》透露，他當天到達鹽田看守所門口時，負責公安不讓他進去，而是請他到鹽田公安分局去。經過一翻波折，當局最終未有批准他會見當事人，理由是他沒有“公證”，無法證明是其委託人身份。所謂“公證”， 盧指出一般是香港於內地的民事訴訟，法庭要採信證據的一個程序。他目前已取得公證，會盡快再到深圳爭取於當事人見面。
然而，盧律師在鹽田看守所吃閉門羹並不是唯一的一位。另一受委託的內地律師範標文即使出示了家屬的委託書到看守所，當局以同樣理由把他拒之門外。
```



戰友點評:
中共國有自己的一套法律系統，可是這套法律體系往往虛設，統治者與執法者從來都是根據自己的利益執法，甚至於未審先判，私判定罪，強加罪名，失去了法律應有的公平公正和透明。該次12人被捕送中，中共諸多刁難，禁止家人所委託的律師與當事人會面，又隨意調整控罪定罪，毫無法律觀念，這是人治而非法治。
從這件事我們可見中共的恐懼與瘋狂，當局以『法律名義』殺雞儆猴，透過此案來阻嚇未來準備逃離香港的抗爭者。目前這12人在看守所內生死未卜，唯恐他們在裡面受了多少苦難和折磨，我們希望以美國為主的西方社會加以關注，向中共施加壓力，阻止中共傷害甚至殺害這12位抗爭港青。
