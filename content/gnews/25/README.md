**西班牙2020年12月25日电/西喜社——**

Amistad项目于12月22日向美国最高法院提出诉讼，旨在迫使参议院议长–副总统迈克·彭斯（Mike Pence）拒绝包括五个州的选举人票，直到这些州的立法机关可以证明他们的选票为止。

该诉讼将副总统迈克·彭斯（Mike Pence），国会两院以及诉讼中指定的五个州的各个行政部门和立法官员列为被告。

Amistad项目隶属于托马斯·莫尔协会（Thomas More Society），他们声称几项以中共病毒防护为幌子制定的联邦和州法律，将各州立法机关认证选举人票的权力，违宪和非法地授权给州行政部门。

Amistad项目正在争论，由于这种违宪的权力转移，宾夕法尼亚州，密歇根州，威斯康星州，乔治亚州和亚利桑那州的立法机关应被禁止认证12月14日的各州总统选举人票。

根据《美国宪法》第二条，总统选举人必须由各州以州立法机关规定的方式任命：

“每个州应以其立法机关指示的方式任命选举人，其数目等于该州在国会中有权获得的参议员和众议员的总数……”

该诉讼指控，宪法第二条禁止州议会以任何身份将其权力下放给州行政部门官员，因为宪法没有授权他们这么做。

Amistad项目向美国最高法院寻求的救济是要求高等法院宣布，部分联邦和州赋予行政人员认证选举权力的法律违宪。

该诉讼还试图阻止副总统迈克·彭斯（Mike Pence）主持选举委员会的投票，并阻止国会接受亚利桑那州，乔治亚州，密歇根州，宾夕法尼亚州和威斯康星州的选票，直到他们各自的州议会能够独立于行政部门开会和认证选票。

迄今为止，彭斯办公室尚未就他是否将履行其宪法义务以保护选举人团投票的合法性发表声明。
