据卫报报道，美国国务卿安东尼·布林肯和英国外交大臣多米尼克·兰尼·拉布呼吁重建和扩大G7联盟坚决捍卫民主社会和基于规则的全球秩序，远离中共国和俄罗斯的独裁威胁。

在周一的伦敦会议后，拉布在讲话中说，他认为各同盟国越来越需要分享其共同的价值观并保护联盟的多边体系。事实上英国邀请了G7以外的澳大利亚,韩国，印度和南非的官员前往伦敦七国外交大臣会议，这标志着加速了相同意识形态的国家一起协作的脚步。拉布说，西方外交的大门始终对独裁国家敞开着，但是警告俄国总统佛拉迪米尔·普京，要求他结束与乌克兰的边境冲突，停止网络攻击，停止传播虚假信息以及造成普京反对者阿列克谢·阿纳托利耶维奇·纳瓦利内中毒，不仅侵犯人权还使用了化学武器。

布林肯说遏制或者按住中共国不是美国的目的，美国真正目的是维护以美国规则为基础的全球秩序，美国过去几十年所维护的秩序使美国公民以及全世界人民包括中国人民共同受益。

无论什么国家，如果中共国想方设法挑战国际秩序甚至破坏腐蚀世界秩序，中共国应履行的承若不兑现，出尔反尔的话，，美国将站起来对付中共国，捍卫国际秩序。在布林肯前往英国的旅途之前，他指控中共国对内高压统治，对外霸权扩张。但是他也认为世界上两个最大的经济体之间发生军事冲突都不符合双方的利益。

布林肯还称赞英美两国的特殊关系，美国是英国亲密无间的盟友也是亲密无间的伙伴。
