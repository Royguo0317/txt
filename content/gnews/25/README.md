据福克斯新闻 (Fox News) 1月27日报道，前国务卿蓬佩奥 (Pompeo) 于周二接受了福克斯新闻黄金时段主持人玛丽亚·巴蒂罗莫 (Maria Bartiromo) 的专访，他指出中共正在寻找一种方法来压制新政府，正在寻找拜登政府的 “软肋”，希望实现川普总统所阻碍的一系列努力。

主持人玛丽亚问：“你预期北京试图以它进入香港的方式接管台湾，以它取得西藏的方式接管台湾吗？”

蓬佩奥回答：“我对习近平总书记的最终野心没有任何疑问，我完全相信他们会挑战这届政府，看看他们是否能找到软肋，看看他们是否能找到一条继续向台湾人民施压的道路。”

关于新疆种族灭绝问题，前国务卿蓬佩奥在专访中谈到“这些都是真实的人，受到我们自20世纪30年代以来从未见过的事情的真正影响”，“在新疆发生的暴行，绝育、强迫堕胎的行动……世界，不仅仅是美国，而是整个世界都需要团结起来，向中共施压，使他们改变并停止在那里发生的这些暴行。”对他的继任国务卿安东尼·布林肯 (Antony Blinken) 批评中共国对待新疆西北部穆斯林少数民族维吾尔人的言论蓬培奥感到高兴。

他补充说：“我希望这届拜登政府能做到过去几届政府都做的事情。我认为，我们的政府以一种独特的方式做到了这一点。要说明的是，我们期望中共会以同样的方式履行对台湾的承诺。这对美国人民来说是正确的，这是你把美国放在第一位，也是中共最会理解的方式。”

蓬佩奥还表示，看到拜登政府注重“醒目”的语义，禁止在联邦文件中使用 “中共国病毒 ”一词他感到失望。

他说：“它始于武汉。事实上，它是一种来自那个地方的病毒。我们知道，中共掩盖了这一点。我们知道，他们让医生和记者失踪了，因为他们想写这件事，却被告知不能写。我认为到现在为止，世界卫生组织仍然没有能够看到这个病毒可能来源哪里这最重要的一点。这些都是我担任国务卿时国务院在谈论的事情……这些都是重要的事情。美国人民需要知道它们，因为它们很重要。它们关系到我们的健康、经济繁荣和我们的安全，我希望也指望下一届政府能按照美国人民的要求去做，继续与中共对抗。”

前国务卿蓬佩奥在任期结束前一天将中共定义为种族灭绝和反人类罪，并称必须追究中共的责任。他在任期内一直就新疆问题、西藏问题、香港问题、台湾问题及中共病毒真相向中共国施压。正是由于这些卓越政绩，中共诋毁蓬佩奥先生为“人类公敌”，并在他任期刚结束就宣布制裁他。中共的这些邪恶、龌龊的做法足以说明蓬佩奥先生做的事情是正义的，他是中国人民真正的朋友，他是把美国放在第一位的爱国者。

虽然蓬佩奥先生结束了国务卿任期，他仍致力于传播并揭露中共病毒真相，追讨中共对美国、对全人类造成的危害，继续和新中国联邦及爆料革命一起推进灭共事业。他的行动力和他的政治智慧都将成为他竞选2024年美国总统的绝对资本。
