## 【香港要聞】【12港人送中】 AG遠洋船 或睹關鍵過程 【跟踪報導】

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/02041309/image001-2.png)圖片：蘋果日報
12港人被內地拘留至今逾40天，他們當天如何乘快艇出海、如何被捕，一直以來都只有廣東海警局的單方面說辭，外間無從探究。 《蘋果》近期連日出海，模擬和重組12港人出航路線，又視察香港東部水域，整理疑點，發現原來曾有一艘遠洋貨輪與12港人當日曾在毗連區水域「擦身而過」，兩者僅相距約2公里，可能目擊12人被粵警追截的關鍵過程。一直協助12名港人的前立法會議員朱凱廸向《蘋果》透露，獲悉一艘名為「Angelic Glory（下稱AG）」的遠洋貨輪，或目擊12港人在公海被捕的情況。該貨輪正停泊香港水域內。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/02041325/image003-1.png)圖片：蘋果日報
**貨輪控制室居高臨下 或睹追截過程**
     該艘貨輪現停泊香港境內，《蘋果》利用航海網站追蹤其位置，發現該艘希臘註冊貨輪9月13日駛進香港，現泊在長洲及南丫島之間，根據特區政府的檢疫令，該船早前接受14日隔離。換句話說就是該貨輪於8月23日、即12港人出逃當天早上，一直在香港東南方中國水域外毗連區徘徊。根據中國海警公佈，12人在8月23日早上9時許於「N 21°54′00″、E 114°53′00″」位置被捕。翻查「AG」的航行紀錄，「AG」當時距事發地點只有約2公里。
     一名不時穿梭中港水域的船家指，2公里是近距離，如視野清晰，很大機會看得清楚，又指中國海警如發現非法活動船隻，一般先以廣播警告，若對方高速逃走，就會上前攔截。貨輪的控制室居高臨下，中國海警當日的關鍵追截過程，或能引起「AG」船員注意。
   《蘋果》記者本週一(9月28日)，即「AG」隔離令完成當天，乘船靠近「AG」，查問船員會否登岸，有南亞籍船員以有限英語說「No」；至於原因，船員稱整艘船「detained by police」(被警方拘留)，故不能離開，再三表明無法上岸；由於語言不通，未能查證是否目擊事件。記者向警方查詢該船是否被扣留，以及警方有否上船接觸船員？發言人拒以書面回覆，僅稱「沒聽水警說過」。
     朱凱廸回應指，12人被捕過程一直只有內地單方面說法，希臘貨船或可提供有用證據解釋：「希臘貨船上雷達紀錄，如果還有記錄保存的話，當時海警和這12人駕駛的船隻到底是怎樣的關係？是兩隻船一齊駛去指定水域？還是有追逐的過程？這對於官司是非常重要的證據，如果不是追逐，那就可能是他們被控制了然則被帶去事發地點，那整個故事就是另一版本；船員在甲板上可能看到了整個過程發生，這一點都不出奇，對了解事情始末有很大作用。」至於香港警察會否調查希臘貨船船員，朱坦言不寄予厚望，更不排除中國海警也知悉貨輪存在：「從壞方向去想，船公司或者船長水手，可能已收到了禁言令。」
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/02041254/image007-2.png)圖片：蘋果日報
**12名港人是被追截還是一早已被控制？**
     外界一直猜測中國海警是否早知悉12人的行程。曾有多次「越境」經驗的船家向記者解釋並加以分析，「如果快艇一直全速車出公海，當然會引起警方注意」。平常越界的釣魚及潛水客亦有機會被中國海警截查，視乎他們（中國海警）「搞不搞你而已」。
     不過船家指，經驗告訴他很難相信中國海警不知12人的出海計劃，「大海茫茫，哪有那麼容易抓到你？」，加上快艇有一定速度，除非一早部署，否則未必成功追截。船家又爆料，走私集團多買通官方，能在某段時間在某區域獲「放生」，但只要官方「反咬」一口，便會被緝獲。中國海警捕獲12人的位置在擔桿島以東，該處範圍甚大，亦是他們不時採取行動的區域，內地傳媒曾多次報道廣東省海警在該區執法成功。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/02041347/image005.png)圖片：蘋果日報
**村民目擊：中國海警曾越境 脅至邊界拉人**
     另一疑點是中國海警有否越境擄人。布袋澳村民對他們駛進香港水域不陌生，以前就有媒體踢爆在果洲群島見過他們的船，亦目擊他們在香港水域邊界抓人，村民指，中國海警會「命令你跟他們進入中國水域，或者叫你上他們船，去到中國水域才抓人」。村民指，因12人「出海不犯法」，香港水警未必敢拘捕他們，且從最近的水警基地出發，亦未必可趕及在香港水域內追截。
**6月至8月警棄巡布袋澳 製造「無警時分」　村民質疑「放人出海」**
     由於布袋澳向來是走私活動熱門地點之一，甚至曾有偷渡來港人蛇在該處上岸，故村民多抱「見慣不怪」的心態應對。有不願具名的布袋澳村民透露，在12人出發的8月23日之前，曾有村民目擊他們準備出逃過程，看見他們準備燃料和物資。惟區內不時有走私活動，見怪不怪，但村民指他們對事件曝光後，警方未有特別增加巡邏感到意外，稱警未有派員入村調查或向村民查問。
     另外警方在村內天后廟旁設置了巡邏簽更簿，但翻查發現整個6月份及8月份，均沒有任何警員的巡邏簽到紀錄，儼如令有關月份成為「無警時分」。事有湊巧，《文匯報》及《大公報》在6月時曾刊出「調查報道」，聲稱知悉「亂港分子紛紛四出打探逃走路線」，甚至指有人嘗試多次逃亡但失敗。這不免令人猜測，警方有意放棄巡查製造「空窗」然後通知內地海警越境抓人。
