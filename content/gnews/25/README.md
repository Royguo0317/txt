千重雲山，萬里荒煙，立盡斜陽。
眺汪洋無岸，撕心兩半；
乾坤有誤，分骨多方。
故國春秋，他鄉歲月，
舊夢新愁肚裡藏。
身為客，借景留孤影，依舊皮黃。

殘山剩水人亡。
失憑據，高寒歸路長。
嘆飄零落葉，何曾作主？
旅途倦鳥，豈止迷茫！
託付蒼穹，天涯芳草，
四海吟風曲未央。
還尋找，問鄉關何處，全在心房。
