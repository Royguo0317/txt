## 欧洲议会以压倒性多数支持欧盟拟定企业伦理道德法
` JPHIMALAYA` [轉載自GNews](https://gnews.org/zh-hans/969566/)

【日本东京方舟农场】作者：青衣 素材采编：武汉老温 校对：文小律

欧洲议会周三以504票支持，79票反对的投票结果通过了一项立法提案，敦促欧盟委员会尽快就跨国集团的企业伦理道德问题制定法案。该法案旨在迫使跨国集团必须对其货物供应链进行彻底的监督，监督商品从原材料到最终推向市场的整个生产链中是否存在违背人权，破坏环境等现象。

根据欧洲议会递交的法案草案，欧盟企业必须对其供应商的行为做出彻底的评估，并且对其产品生产链的整个过程对外公布，做到完全透明。这些措施不仅涉及跨国集团，也同样适用于一些中小型企业，他们必须做到对侵犯人权，破坏环境等现象提出警告，并且进行密切的跟踪。一旦出现违背谨慎规则的行为，企业将被追究责任，除非企业能够证明已经采取了一切可能的措施。

不过该草案并未提及在涉及重大事件时追究企业的刑事责任，“大赦国际”、“地球之友”以及“标签上的道德”等国际非政府组织对此表示遗憾。上述非政府组织呼吁欧盟委员会在立法时制定一系列明确的规章，加强对触犯规则企业的制裁。本周四，他们发表公告指出，只有制定严厉的制裁措施，才能够阻止跨国集团利用新疆维族人的强迫劳动换取利益。他们呼吁法国政府协助欧盟推动上述议案。同时他们还呼吁欧盟各国政府在联合国制定有关跨国集团与人权公约的谈判问题上起到积极的作用。

众所周知，中共多年来在新疆地区对维吾尔族及其他穆斯林少数民族施加各种暴行，其中就包括强制劳动。无论是在他们称为“再教育营“或”职业培训学校“，实则为”集中营“里的维吾尔族人，还是在军警特宪高压管控下”散养“的维吾尔族人，都被强制无薪或低薪地从事生产劳动，近年来已引起全球广泛关注和谴责。

为了制裁中共在新疆地区侵犯人权和强迫劳力的恶行，美国前任总统川普签署了一项于上个月开始生效的行政命令，该行政命令禁止美国从新疆进口棉花等产品。此禁令对世界服装产业产生了很大影响，尤其是对中共的纺织品进出口贸易打击巨大。根据禁令，美国海关官员可以禁止进口他们怀疑用新疆原材料制成的产品，无论是直接从中国还是途径其它国家进口。

美国国土安全部（DHS）代理副部长库奇内利（Kenneth Cuccinelli）表示，强制劳动属于现代奴隶制度，而“中国制造”并不只是反应源头产地，而是一个警告标签。他指出当局不会容许任何类型的强制劳工，同时呼吁其他企业应将新疆的产品排除在供应链名单之外。根据美国海关及边境保卫局（CBP）发出的暂扣令（Withhold Release Order），受影响的产品涵盖多项日常用品，包括服装、纺织品、番茄籽、番茄罐头和番茄酱等。而在其它国家制造但选用新疆棉花或番茄的产品也在禁止名单上。

新疆的棉花产量占全球两成，美国2020年从中国分别进口总值90亿美元的棉花及1000万美元的番茄。华府曾在去年下令禁止从新疆建设兵团进口棉花，并于此后在美国港口查扣了43批棉花制品。

川普政府时期有关新疆的这项禁令政策在贸易上给了中共重重一击，如今的拜登政府也以“种族灭绝”来定性中共大规模关押新疆维吾尔人以及强制劳动等恶行，并呼吁国际社会联手发起相关制裁措施。现在欧洲议会也通过了支持欧盟拟定企业伦理道德法议案，全世界对于犯下“种族灭绝罪“的中共所展开的围剿行动，已从政治和经济等各方面全面展开。
