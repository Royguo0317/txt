《纽约邮报》3月23日报道，中共军队将在今年党的100岁生日上展开大规模的效忠共产党及其领导人习近平的运动，庆祝香港驻军的同时，不会举行群众游行。中共不放过任何一个纪念重要周年的机会，来提醒人民中共的历史和党在国家生活中的核心作用，要求人民心存感激。

自2012年中共十八届一中全会上，习近平被选为中共中央总书记、中共中央军委主席，成为中共国第五代最高领导人。习上任后，显示出强烈的政治军事野心，为加强对军队这一中共政权基石的控制，习借助反腐运动来清洗不忠诚的将领和政敌，坚决把“党领导一切”的观念延伸到企业和政府，及社会各行各业。

中共国解放军一直处于这些努力的中心。在对手势力被清除后，军队最高指挥权被习近平个人掌握。习近平把解放军变成了他的政治权力基础，习向军队提供了诸如隐形战机等新式装备，同时试图消除对其忠诚的任何疑虑。

早在2018年习近平对军队领导人讲话，便提出军队要”绝对忠诚”。少将李军在周二北京举行的新闻发布会上再次强调，军队必须“坚决听从习主席的指挥，对习主席负责，让习主席放心。” 作为军队的指导思想，每个士兵需要了解党的历史，使他们能够“忠于核心、坚持核心、遵循核心”，此处的党领导核心指的是习近平掌舵中共。他补充说，这种教育的目的是让军人“理解习近平中共国特色社会主义思想的光荣真理”，“牢固树立习近平强军思想的指导力量”。

作为送给中共的生日礼物，中共国军方将开放一个新的海军博物馆和一个关于中共解放军驻港部队的展览中心。中共将对香港人民惨无人道的残酷镇压作为炫耀的成果展示。而在7月1日中共党生日当天，不再举行面向群众的阅兵游行。军队已经成为中共专制的手段、镇压人民的工具，不再是人民解放军，与人民不相关。
