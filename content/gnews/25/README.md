## 華大基因瑞典政府指控其病毒檢測試劑高假陽性率
`2020-08-27 09:26 GM64` [轉載自GNews](https://gnews.org/zh-hant/320422/)

8月25日，瑞典公共衛生局聲明宣稱華大基因生産的新冠病毒檢測試劑盒存在缺陷，有過高的假陽性率，並導致3700人的檢測結果出現假陽性。瑞典公共衛生局還向歐洲有關部門和世界衛生組織通報了這一情況。

這則新聞一出，華大基因馬上就進行了所謂澄清：公司關注到某歐洲媒體報道公司新型冠狀病毒檢測試劑盒存在“高陽性檢出率”問題。

華大基因的所謂澄清完全是爲了忽悠牆內群衆，玩弄文字遊戲。

1. 瑞典公共衛生局的公告中說的是“假陽性”，而華大給自己美化成“高陽性檢出率”。這完全是兩個不同的概念，“假陽性”是指樣品本身爲陰性（不含病毒），而檢測結果是陽性，即檢測結果是錯誤的。“高陽性檢出率”是指試劑盒能檢測到病毒含量很低的樣品，能更有效檢測出弱陽性樣品，前提是樣品本身就是陽性的。
2. 華大基因把人家一個負面的詞硬生生說成了正面表揚的詞，不愧是CCP的企業，跟CCP一樣的不要臉。
3. 試劑盒的靈敏度高低是由標准病毒RNA參考品來衡量的，不是華大基因自己吹牛吹出來的，你自己的産品什麽質量自己心裏應該有數吧。
4. CCP審批上市的新冠病毒RT-PCR試劑盒全部都是國産貨，進口試劑盒産品一個都不能走綠色通道獲得快速審批，導致了明顯的劣幣驅逐良幣現象。
5. 國産新冠疫苗試劑盒的靈敏度低，陽性檢出率只有30%～50%，已經被很多媒體報道過。華大基因就不要再爲自己塗脂抹粉了
