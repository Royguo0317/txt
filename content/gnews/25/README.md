#### **荷兰国防部确认皇家海军”埃弗森”号护卫舰将加入即将部署到亚太地区的英国航母打击群**。



据荷兰国防部的Instagram官方动态显示: “埃弗森”号在与英国航母打击群出发前，在码头进行维护。和这支国际舰队的联合，”埃弗森”号将很快驶向亚洲。”

“埃弗森”号也在其官方推特发布了相同信息:
![]()![](https://gnews.org/wp-content/uploads/2021/03/IMG_3619.jpg)


今年早些时候，由于英国国防部从一篇发布讨论部署的文章中删除了提及荷兰舰船加入该航母打击群的消息，因此引起了一些困惑。

国防部写的帖子中详细介绍了英国皇家海军伊丽莎白女王号及其打击群即将部署的情况，其中提到了一艘荷兰军舰，该陈述现已被删除。

该帖其中两部分标题为”英国航母打击群解读”的内容被删除。

第一部分写到: “该航母打击群包括北约最先进的驱逐舰，它们来自英国皇家海军、美国海军和荷兰海军”。

第二部分是:  “该打击群包括北约最先进的驱逐舰- 英国皇家海军的45型 HMS “钻石 “号和HMS “防御者 “号、美国海军阿利·伯克级的 “苏利文斯 “号、英国护卫舰 “诺森伯兰”号和 “肯特”号以及荷兰海军的 “埃弗森 “号。

该帖子现在显示的是:  “该航母打击群将得到美国海军陆战队和美国海军人员以及装备的补充。其中包括一支由美国海军陆战队F-35B “闪电II” 战机以及美国海军的 “苏利文斯”号驱逐舰组成的分队。

现在的内容没提到荷兰驱逐舰，奇怪的是，原帖中的图文现已被删除。该图如下所示，最初显示的是为了详细说明该航母打击群的运作方式。







这张图被删除是否是因为它展示了那艘荷兰驱逐舰？

“埃弗森”号去年曾在(英国)航母打击群进行演练准备工作时亮相，训练与英、美舰船并肩作战。
