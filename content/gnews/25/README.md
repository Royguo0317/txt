## 报告：在COVID-19挑战中，美国在拉丁美洲的“毒品战争”需要全面改革

美国国会委员会将在本周发表的两党报告中陈述，如果华盛顿要有效应对因COVID-19大流行而恶化的毒品问题，美国需要改变在拉丁美洲的反毒品政策。

这份长达117页的报告敦促以美国国务院为主导的“更智能”的跨部门政策，以减少危险药物的供应。它还呼吁当局阻止通过使用加密货币和复杂的跨境金融交易非法资金流动来打击洗钱活动。

这是对“禁毒战争”进行的为期18个月研究的结果，这场战争花费了数十亿美元纳税人的钱，却没有结束西半球大部分地区的高暴力和腐败现象。

西半球毒品政策委员会在报告中说：“日益复杂的威胁需要采取更加灵活，适应性强的长期战略。”路透社在该报告发布之前看到了这份报告。

冠状病毒爆发相关的健康问题和经济压力加大了该地区消灭毒品贩运的难度。由前民主党和共和党政府官员和众议院议员组成的小组发表的报告说：“这种流行病加剧了不断恶化的阿片类药物危机，例如缺乏适当的治疗，经济困境和社会孤立。” 。

该报告赞扬了一些政策，包括在哥伦比亚实施的古柯种植（可卡因来源）替代方案，以及在墨西哥加强刑事司法改革的努力。 它援引了警察改革计划难以在因陷入困境而被称为“北方三角”的国家推进–萨尔瓦多，危地马拉和洪都拉斯。

但是，报告还发现，禁毒政策也已经造成了相当大的危害。例如，消除古柯种植的措施，导致其生产和与此相关的暴力活动转移到新社区。

针对毒品帮派头目或主要人物的执法工作已摧毁了部分贩毒集团，这导致更多的由帮派成员争取控制权而造成的暴力事件。

乔·拜登（Joe Biden）政府有望将美国政府做法转向该地区，与川普总统侧重制裁委内瑞拉和古巴以及阻止美国边境的移民相比，将更重视对外援助、外交和人权。

译者评：拜登的绥靖政策只会给美国带来更大的灾难。郭先生在直播中屡次警告美国政府，用毒品搞弱美国是中共对美国的3F（搞弱，搞乱，搞死美国）的计划之一。中共通过边境贩毒集团走私毒品到美国，每年毒死高达10万的美国年青人。委内瑞拉，危地马拉，洪都拉斯等地的人在中共煽动下大规模试图非法进入美国，这些人中难免混有恐怖分子，犯罪分子，贩毒分子。大量非法移民如果进入美国，不仅会对美国国家安全造成巨大潜在危害，还会增加纳税人的负担以及造成更为严重的贩毒活动。川普总统建边境墙的措施阻止了很大一部分非法移民，从而也阻止了恐怖分子，贩毒分子混入难民队伍。无论是冠状病毒还是毒品走私猖獗的背后都有中共的黑手在作祟，只有消灭中共才能从根本上打击中共支持的邪恶势力。
