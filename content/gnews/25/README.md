28日获悉，日本政府去年以不能透露信息来源为条件，向美国和英国政府提供了中共强制收容穆斯林少数民族维吾尔族人的信息。一直在人权问题上对中国政府施加压力的美国，根据这些信息指责中共打压维吾尔人。（日美关系相关人士透露）

因为来自中国和朝鲜的威胁越来越大，日本政府内部开始呼吁加入 “五眼联盟”。”五眼联盟 “是包括美国、英国和加拿大在内的五个英语国家共享机密的联盟机构。很显然，信息一体化正在悄然推进。  (联合)
