#### 作者：斩草要除根

人民日报北京4月13日发文称，民政部、中国残疾人联合会今天在京召开残疾人两项补贴资格认定申请“跨省通办”电视电话会议。会议明确，两项补贴（即困难残疾人生活补贴和重度残疾人护理补贴）“跨省通办”将于4月15日试运行，4月22日正式运行。

#### 简评：

我们来简单的了解一下目前中共国对于残疾人两项补贴的条件及金额（提高后）：属于低保家庭中的残疾人可以享受困难残疾人生活补贴（每月110元），残疾等级为一级、二级且需要长期照护持续时间达6个月以上的重度残疾人（不包括重度智力、精神残疾人或其他残疾人）可以享受重度残疾人护理补贴（每月110元），目前大部分地区只有低保家庭，或者重度残疾人才能享受到两项补贴。如果三、四级残疾人是低保家庭，那他只可以享受困难残疾人生活补贴（由各地自行确定），如果三、四级残疾人不是低保家庭（不包括低收入残疾人及其他困难残疾人），不能享受两项补贴。

我们再来看一下提高以前的大体情况：根据对全国2017年第一季度残疾人两项补贴发放情况的调查显示，从生活补贴标准来看，22个省份处于每月50-80元之间，10个省份达到或超过每月100元，最高的北京市每月900元；最低的安徽省每月33元。从护理补贴标准来看，18个省份处于每月50-80元之间，14个省份达到或超过每月100元。最高的浙江省每月500元，最低的云南省每月40元。

2020年中共国第三次人口普查后的残疾人数量由于残疾人联合会的网站不能打开而无法知晓，我能仅通过维基百科查到数据是： 根据2006年第二次全国残疾人抽样调查数据显示，中国现有残疾人8300多万，其中重度残疾人（一、二级）2457万人，其余残疾人5839万**人**，涉及7000多万个家庭、2.6亿人口。2020年12月05日“经济日报”报道称：民政部针对困难残疾人生活补贴和重度残疾人护理补贴两项补贴年发放金额约270亿元，残疾人两项补贴制度已惠及困难残疾人1153万人、重度残疾人1433万人。庞大的人数对应着杯水车薪，更让人痛心的是有1024万重度残疾人和4686万其他残疾人是没有得到任何补助！

在劳动年龄段生活能够自理的未就业城镇残疾人占比高达60%以上，未就业的农村残疾人占比为50%，也就是说能够自理的残疾人有一半以上是没有任何收入来源，在就业领域残疾人就业多从事技术要求低、薪资水平差的行业且职业渠道也是非常单一，我们所能了解到的似乎只有按摩行业。

那些重残无业、一户多残、老残一体、依老养残的一个个家庭所面临的生活困境又是怎样的凄惨。国家没有任何社会福利保障措施提供给这些群体在没有亲人照顾的状况下有生存条件，一个残疾人拖垮一个家庭，特别是双亲年老还需要照顾生活无法自理的重度残疾人和重度智力残疾人。

有新闻报道，广州市越秀区一位83岁老母亲黄某细心照料智障的儿子40余年，近年来因身体每况愈下，担心自己去世后儿子得不到好的照顾， 选择“先送走”智障儿子，后自首，试想一下是怎样一种被逼无奈才让一位母亲能亲手杀死自己的骨肉？还有老残一体，城乡老年残疾人的生活照料仍主要由残疾人家庭提供，其负担沉重且支撑力脆弱，特别是无儿无女的农村老年残疾人，缺乏自理能力同时缺乏医疗和救助，他们的晚年是毫无尊严的苟且的活着。面对这八千多万残疾人群，中共喊出的口号是“残有所养，残有所依”。而现实却是冰冷的，在强大的世界第二大经济体，国力强盛的中共国，外表似乎永远都是那么光鲜亮丽，可这一片昏暗的角落里却看不到一缕阳光。
