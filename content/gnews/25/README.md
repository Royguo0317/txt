# **在北京批准“爱国”计划控制香港选举后，外交大臣多米尼克•拉布指责中共国进一步“掏空”香港民主**

在北京批准一项有争议的“爱国”计划以控制香港选举之后，英国外交大臣多米尼克•拉布指责中共国进一步“掏空”香港民主。

中共国的人民代表大会今天通过了一项修改香港选举制度的法案决议。

此举将会进一步削弱香港政治体系中的民主代表，并引入使政客效忠北京当局的机制。

拉布先生说这项决议是“北京掏空香港民主的最新动作，这与之前中共国做出承诺完全相反。”

外交大臣说北京“已经进一步削弱了西方对中共国履行国际职责的信心和信任。”

英国政府正在评估这项决议的细节，以了解中共国是否违反了《中英联合声明》。

中共国的这些举措是其持续加强对香港管控工作的一部分。

在此之前去年6月份，中共国强制实施了有争议的国家安全法，有批评人士将其称为继2019年发生的一系列民主抗议运动之后的一种打压反对人士的工具。

自从抗议运动开始以来，大多数备受瞩目的民主政治家和活动人士已被送入监狱或流亡国外。

在对修改香港选举制度回应时，拉布先生在一份声明中说：“这是北京掏空香港民主的最新动作，这与之前中共做出的承诺完全相反。”

他说：“这只会进一步削弱西方对中共国作为一个国际社会主要成员履行国际责任和法律义务的信心和信任。”

中共国的人民代表大会最终以2895票赞成、0票反对、1票弃权通过。

北京驻香港代表机构联络处说“中央政府有良好意图”。

该联络处在一份声明中说，我们期望社会各界和市民对这项工作负责，修正法案并提出建议，以便在爱国主义和爱香港的旗帜下，聚集强大的正能量。

在另外一份声明中，香港特首林郑月娥表示将“坚定支持”并表示“真诚的感谢”。

这项修改实际上消除了能够对香港选举结果产生反对影响的任何可能性。香港是前英国殖民地，于1997年重返中共国，当时中共国承诺香港实施高度自治。

分析人士说，对“爱国主义”的全面要求增加了香港政客将开始争夺谁更忠于北京，而不是谁对如何管理这座城市有更好的意见。

香港政制及内地事务局局长曾国卫将爱国主义定义为对中共国的“全面的热爱”，其中包括中国共产党的领导。

这些措施将改变香港立法机关和一个选举委员会的规模和组成，该选举委员会会选举出亲北京当局的行政长官。

该委员会还将会被授权选出很多城市议员。一个新的机制将会被建立，该机制用来审查候选人并评估选举获胜者的行为，以确保只有那些被视为爱国者的人才能治理香港。

北京将会把选举委员会的人数从1200人增加到1500人，立法机关的席位将从70个增加到90个。

目前，立法会的70个席位中有一半是直接选举产生的，随着选举委员会选出的额外席位，这一比例将会减少。另一半席位是工会和各行各业的代表，基本上都是亲北京当局的。
