# **英国通讯管理局撤销中共国环球电视网在英国的广播许可证**

**2021年2月4日，伦敦，格林威治时间（GMT）10:15：英国通讯管理局(Ofcom)于今天撤销中共国环球电视网（CGTN）在英国的广播许可证，因为其调查结论显示，该许可证被星华传媒有限公司（Star China Media Limited）错误地持有。**

中共国环球电视网（CGTN）是一个国际英语卫星新闻频道。

在英国，由议会制定的广播法规定，广播许可证持有人必须对获得许可的服务有控制权——包括对其播放的节目进行编辑监督。此外，根据这些法律，许可证持有人不能受政治机构控制[1]。

我们的调查结论是，星华传媒有限公司(SCML)是中共国环球电视网（CGTN）服务的许可证持有人，但是对CGTN的输出信息没有编辑责任。因此，星华传媒有限公司不符合对许可服务拥有控制权的法律要求，其不是合法的广播许可证持有人。

此外，我们无法批准将许可证转让给一家名为中共国环球电视网络公司（CGTNC）实体的申请。这是因为该申请缺少关键信息，而且我们认为CGTN将被取消持有许可证的资格，因为它被一家最终由中共控制的机构所控制。

我们已经给予CGTN大量时间去遵守法律规定。这些努力现在已耗尽。

经仔细考虑，并顾及到所有事实及广播机构和观众的表达自由权，我们认为撤销CGTN在英国的广播许可证是适当的。

我们预计将很快完成对CGTN的单独制裁程序，基于应有的公正和公平以及侵犯隐私。

**英国通讯管理局(Ofcom)的调查**

我们的调查发现，星华传媒有限公司并未对CGTN的节目表负有选择和编排的编辑责任。它是CGTN在英国的分销商，而不是服务的“提供者”[2]。

此外，参与CGTN的决策或频道日常运作的雇员似乎都不是受雇于星华传媒有限公司。中共国环球电视网确认，其全球编辑委员会是CGTN服务的节目选择和组织的最终决策者，并行使编辑控制权。

**CGTN的许可证转让申请**

在回复我们的询问时，CGTN承认星华传媒有限公司并未控制该频道，因此不应再持有该许可证。但它确认有意重组，将CGTN部门从中央电视台（CCTV）分离出来——中央电视台最终由中共控制，因此失去了持有英国广播许可证的资格[3] — 并申请将许可证转让给控制该频道的实体。

鉴于撤销广播许可证是对广播机构表达自由权利的重大干预，我们认为应给予CGTN一段合理时间，让其遵从规定。

将许可证转让给中共国环球电视网络公司CGTNC的申请于2020年9月提交。然而，我们无法对其进行适当评估。申请书中缺少关键信息，并且CGTN表示的重组尚未发生，且至今仍未发生。

CGTN多次没有回复我们评估其转让许可证申请所需的重要问题，也没有提供任何关于其重组进展的最新情况。

**中共国环球电视网络公司（CGTNC）与中央电视台（CCTV）之间的联系**

经考虑现有证据后，我们决定不能批准星华传媒有限公司将许可证转让给中共国环球电视网络公司的申请。

在我们的调查过程中，中共国环球电视网提交的信函清楚表明，中共国环球电视网络公司由中央电视台控制，并且中央电视台也是中共国环球电视网络公司的唯一股东。

鉴于中共国环球电视网络公司由中央电视台控制——中央电视台作为中共国传媒集团的一部分，由中共控制，因此根据英国广播法规定，中共国环球电视网络公司失去了持有广播许可证的资格——我们认为中共国环球电视网络公司将失去持有许可证的资格。

英国通讯管理局发言人表示：“我们的调查显示，中共国环球电视网的许可证由一个没有对其节目享有编辑控制权的实体所持有。我们无法批准将许可证转让给中共国环球电视网络公司的申请，因为该公司最终由中共控制，而英国广播法不允许这种情况。”

“我们曾多次向中共国环球电视网提供让其遵守规定的机会，但它没有这样做。我们现在认为应该撤销中共国环球电视网在英国的广播许可证。”

**其他CGTN案例**

2020年，我们发现中共国环球电视网CGTN在报道香港抗议活动时未能保持应有的公正性，违反了英国通讯管理局的广播规则，同时也发现它严重违反了我们的公平和隐私规则[4]。

鉴于这些违规行为的严重性，我们通知CGTN，我们将考虑对其实施制裁。

今天的决定不会影响对CGTN的这些制裁程序，并且我们预计很快就会做出决定[5]。

我们还有三项关于CGTN服务内容的公平和隐私调查，这些调查也在进行中，有待进一步审议。
