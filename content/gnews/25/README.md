12.澳大利亞研究人員發現，新型冠狀病毒可以在玻璃和貨幣上存活28天

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/12084513/12-12-1.png)

在中國發生新的冠狀病毒病（COVID-19）病例後，身穿防護服的醫務人員在城市範圍內的測試中收集拭子進行核酸測試。路透社照片。

10月12日，墨爾本

澳大利亞研究人員周一說，引起COVID-19的病毒可以在鈔票，玻璃和不銹鋼上存活長達28天，比流感病毒要長得多。他強調需要清洗和洗手才能抵抗這種病毒。

澳大利亞國家科學機構CSIRO的研究人員發現，SARS-COV-2病毒在攝氏20度（華氏68度）的情況下，在光滑的表面（例如在手機屏幕上發現的塑料鈔票和玻璃）上仍具有28天的感染性。該研究發表在《病毒學雜誌》上。

相比之下，已發現甲型流感病毒在表面存活了17天 。

CSIRO的研究涉及將人造粘液中的病毒在一系列表面上乾燥，其濃度與COVID-19患者的樣本相似，然後在一個月後提取病毒。

在20、30和40攝氏度的受控實驗室環境中進行的實驗表明，隨著溫度的升高，生存時間下降。

CSIRO首席執行官拉里·馬歇爾（Larry Marshall）在一份聲明中說：“確定病毒在表面上真正能夠存活的時間可以使我們更準確地預測和減輕其傳播，並更好地保護人們。”

體液中的蛋白質和脂肪也會大大增加病毒的生存時間。

“該研究還可能有助於解釋SARS-CoV-2在脂類或蛋白質污染較高的涼爽環境（例如肉類加工設施）中的持續存在和擴散，以及我們如何更好地應對這種風險， ”主管Trevor Drew說CSIRO的澳大利亞疾病預防中心。

澳大利亞在抗擊COVID-19方面的表現比大多數其他富裕國家都要好得多，在2500萬人口中，共有27,000例感染和898例死亡。

該國第二波感染的震中州維多利亞州週一報告了15例新病例，遠低於政府設定的緩解目標，即不超過5例的目標是放鬆對州府墨爾本的嚴格封鎖。

人口最多的州新南威爾士州週一報告了6例新病例，其中5例是返回隔離區旅行的旅客。

全國冠狀病毒病例上升至712萬

Covid感染導致的死亡人數增加816，至1,09,150

衛生部的數據顯示，過去24小時內，印度的冠狀病毒病例總數增加了66,732例，至週一上午達到了7120萬例。
