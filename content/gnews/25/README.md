## 美國必須更大範圍停止投資和審計管控中共概股
`2020-05-27 11:33 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/214460/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/27111748/%E7%BE%8E%E5%9B%BD%E5%BF%85%E9%A1%BB%E6%9B%B4%E5%A4%A7%E8%8C%83%E5%9B%B4%E5%81%9C%E6%AD%A2%E6%8A%95%E8%B5%84%E5%92%8C%E5%AE%A1%E8%AE%A1%E7%AE%A1%E6%8E%A7%E4%B8%AD%E5%85%B1%E6%A6%82%E8%82%A1.jpg)
美國安全政策中心(Center for Security Policy)主席弗蘭克·加夫尼(Frank Gaffney)近日在其官方網站上發聲，強烈支持特朗普總統(President Trump)關於阻止美軍人員和其他聯邦僱員的退休儲蓄投資中共在美上市公司(中概股公司)。

弗蘭克·加夫尼表示特朗普總統此舉明智且理由充分：首先，中概股公司在美沒有遵守美國旨在保護投資者免受欺詐和重大風險的規則；其次，有些中概股公司直接威脅美國的國家安全；最後，一些中概股公司與中共同謀勾結侵犯了中國人民乃至全世界人民的人權。以上所列理由可從眾多事實中體現，例如前段時間瑞幸咖啡被爆做假賬，還有中概股網絡公司(如新浪、網易等)長期在中國境內乃至全世界對自由言論的打壓。

弗蘭克·加夫尼還指出，儘管特朗普總統叫停了美軍及其他聯邦僱員的退休儲蓄投資中概股，但是以華爾街為首的很多基金組織正在利用美國軍人和平民的其他儲蓄(非退休儲蓄)來投資中概股。更糟糕的是，華爾街以及證券交易委員會(Securities and Exchange Commission)對這些中概股公司毫無盡職的調查或可靠的審計。基於此，弗蘭克·加夫尼呼籲美國應更大範圍停止投資和審計管控中概股。

附註：弗蘭克·加夫尼是華盛頓特區安全政策中心的創始人兼執行主席。在加夫尼先生的領導下，該中心已被國內外公認為是對外交和國防政策問題進行及時、知情和深入分析的資源。加夫尼先生在里根政府任職期間曾擔任國防部國際安全政策助理部長，其後四年內擔任國防部負責核力量和軍備控制政策的副部長助理。
