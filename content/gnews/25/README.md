如果从单纯个体来说，生命的逝去无疑是令人痛惋的。但因为被杀的对象是“党委书记”，社交媒体推特上基本上是一边倒地支持复旦教师。有人说，“不管什么原因，杀书记，基本不会错的”；还有人说，“党委书记不会有好人，杀得好”。……且不论这些说法的对错，这洪水般的民意也是恶贯满盈、公信力尽失的中共最惧怕的。

如果稍微了解一点中共体制的人，就会明白老百姓对党委书记为什么如此痛恨！而一个姜文华是杀不完割不掉中共国的“党委书记”——中国人身上这颗毒瘤！

为了让大家更了解中共国“党委书记”是怎样毒瘤的存在，我们先来了解关于“党委书记”两套班子这庞大吓人的数字！

中共国机构共分中央、省、市、县、乡和村（城市里相对应的是社区居委会）六级。任何一级都是党政两套班子匹配。根据行政区划统计，全国有省委书记34个，市委书记333个，县委书记2861个，乡党委书记44067个，村委书记（村支书）663000个，城市社区居委会书记77000个。这一组数字加在一起，一共是787295个。这近79万个书记只是浮在机构表面的书记数。

中央机构分中央和国务院两套班子，部级或副部级机构就有七十三个同级别的书记。全国人大、全国政协、最高人民法院、最高人民检察院、团中央、全国妇联和全国总工会，这七个机构，各有其书记，共七个书记。与上述七十三个书记加在一起，一共是八十个书记。这七个机构，每一个机构的建制规模（不是指行政级别）都相当于一个部。

平均按每部下设 二十个局级机构计算，八十个部共有一千六百个局级机构。一个局有一个局级书记，于是中央八十个部级机关里也就有了1600个局级书记。局下面设处，一千六百个局，平均每个局设八个处，就共有12800个处级书记。至于有些大处下可能还设科，这里就忽略不计了。80个部级书记，加上 1600个局级书记，加上12600个处级书记，一共是14480个书记。这即是说，仅首都北京中央机关里就有14480个党委书记。

依据2003年的资料，全国国有企业有十五万家，那就是15万个书记。按每家国有企业下设二十个分支机构计算，十五万家国有企业里分支机构的书记数就是300万个。合计国有企业书记总数是350万个。全国城镇集体企业是六百万家，就是600万个书记。有些大的城镇企业下面分支机构的书记数不计。

《南方周末》的文章说，全国国有企业、集体企业和私营企业里共有党委书记960万个，如果按一正二副计算，全国共有正副书记4890万个。按每人每年年薪五万，即24000亿，据相关部门估算全国实现免费医疗仅需1600亿。

全国这些书记，就有四千多万人，相当于一个欧洲大国了，全靠老百姓养着。这是一组吓人的数字，两套班子，说穿了就是要两倍的税收。

有很多人质疑这些数字的真实性，全国9000万党员400多万基层干部，4890万个党委书记是不是有点多了？要知道共产党的数字永远都是造假的，就像14亿人口国家还要你生三胎是一个道理！相信共产党跑步走进火葬场！至于4890万个党委书记的真实性就见仁见智了！
