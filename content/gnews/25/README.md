## 中共放行WHO专家组进入武汉调查 准备对抗川普政府解密病毒来源真相的可能性
` 经济金融组` [轉載自GNews](https://gnews.org/zh-hans/753057/)

整撰：MIYA
审核：WJMG

今天（14日），世卫组织（WHO）专家组乘坐的航班到达湖北武汉，在机场接受检测，并隔离14天，之后将和中共专家一同开展对中共病毒的溯源研究工作。WHO表示，专家组中2人因无法通过病毒抗体检测，因而未能成行。

![]()![](https://gnews.org/wp-content/uploads/2021/01/3-27.jpg)2020年1月28日世卫组织总干事谭德赛在北京受到习近平的接见

1月11日，中共国家卫健委发公告称，经双方协商，WHO中共病毒溯源国际专家组将于1月14日来华考察，同中方科学家就病毒溯源进行联合科研合作。但简单一句话并未提及考察时间的长短，也没有说明详细的行程。

WHO今日（14日）发推文称，专家组的15名成员从本国出发前已经接受聚合酶连锁反应（PCR）检测和抗体测试，全部显示无感染。当各人抵达新加坡后再接受测试，其中两人免疫球蛋白M（IgM）测试结果显示受感染，未能登机仍然留在新加坡。其余13名专家已于14日上午11时许抵达湖北武汉天河机场T3航站楼。

此次赴华考察的专家组成员之一、越南生物学家阮洪（Hung Nguyen）13日对外媒表示，专家组14日清晨从新加坡飞往武汉。他们在武汉一家宾馆隔离两周后，计划花两周时间采访当地的研究机构、医院和海鲜市场的人员。希望今次能够揭示病毒起源的新细节，但不会期望一切明朗。

从2019年底武汉疫情爆发，WHO就开始了与中共的“密切配合”。2020年1月14日，WHO采用中共官方的说法，称“未有证据显示病毒会人传人”。1月22日、23日的WHO会议上，中共代表向委员会和总干事施加了压力，称“宣布全球范围的突发公共卫生事件是不可能的”。直至1月23日武汉封城后，世卫组织仍然拒绝宣布此疫情为“国际公共卫生紧急事件”。

2020年2月3日，谭德塞还在告诉全世界对中国旅游和贸易不需采取措施，不要产生“不必要的干扰”。当中共宣称中国已经控制疫情传播后，谭德塞称，“中国的努力应该得到认可”，公开称赞中共所谓的“牺牲自己，拯救全世界”。

过去一年中，世卫组织曾先后去过中国两次。去年的2月16日，由世卫组织12人组成的医疗专家组在北京、广东和四川三个地方进行调查，专家组公开的行程里并没有列出武汉，却在22日抵汉至24日又匆匆离开；去年7月，世卫组织在向中国派出了尚未公布身份的一名动物卫生专家和一名流行病学家，并未能前往武汉。

WHO虽两次赴华，明着是调查中共病毒的起源，实际行动上从未展开过公开、透明、独立的调查，相信这一次也不会例外。

英国周日邮报（Mail on Sunday）1月9日披露了，中共官方机构“国家自然科学基金委员会”（NSFC）此前在网上发布的超过300项的武汉P4病毒研究所（WIV ）的研究数据全部遭到删除，其中包含关于病毒动物传人的调查研究，也包括该研究所“新发传染病研究中心”主任石正丽的诸多重要研究数据，包括从蝙蝠身上类似SARS 的冠状病毒展开的跨物种感染风险研究，及另一项蝙蝠携带人类病原体的研究，亦已无法在该网站上查到。

结合最近有媒体报道的川普总统极有可能在任期内解密病毒与武汉P4实验室相关绝密情报的消息。编者大胆地推断，中共这次已经“做足了准备工作”，对信得过的WHO专家组13人给与放行，在武汉再次导演一幕“最彻底的病毒来源调查”，所有这一切都是为了对抗川普政府未来可能对病毒来源真相的解密。
