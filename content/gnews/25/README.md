# 北约报告称中共国是欧洲 — 大西洋安全的威胁; 北京称此为“被害妄想症”

北约在最新报告中呼吁，投入更多时间、政治资源和行动以应对“崛起的中共国”带来的“安全威胁”。

*由格洛丽亚·梅斯里撰写*

北大西洋公约组织（NATO）呼吁投入更多时间、政治资源和行动来应对“崛起的中共国”带来的“安全威胁”。该联盟在其《北约2030年报告》中建议，在联合情报与安全部（JISD）内设立一个特别部门，以监控影响欧洲— 大西洋安全的俄中合作。

报告指出，北约盟国之间的内部政治分歧是危险的，因为它们有助于“俄罗斯和中共国等外部行为体利用欧盟内部分歧，并以危害其集体利益和安全的方式利用个别盟友。”

报告进一步说明：“联盟应考虑建立一个咨询机构，讨论其与中共国安全利益的各个方面。它必须加大努力评估中共国技术发展的影响，监测和防御任何可能影响到欧洲最高统帅司令部（SACEUR）责任区的集体防御、军事准备或应变能力的中共国活动。”

# “中共国是北约盟国的一个严重威胁”

北约的报告建议，它必须做出改变以适应“由持续挑衅的俄罗斯和中共国的崛起带来的，以系统性竞争回归为标志，日益严峻的战略环境挑战。”

报告说：“由于近几十年来中共国在遥远的印太区域不断意图对邻国使用武力、经济胁迫和威吓外交等手段，因此在未来几十年中，中共国很可能会挑战北约在建立集体应对、保护重要基础设施、解决诸如5G等新兴技术以及保护包括供应链在内的经济敏感部门的能力。”

报告补充说，从长远来看，中共国越来越有可能在全球范围内，包括在欧洲—大西洋地区投入潜在的军事力量。

报告还说：“中共国在包括核子、海军和导弹能力在内的所有领域的军事现代化，都给北约联盟和战略稳定带来了新的风险和潜在威胁。”

# 中共国称：“北约患了被害妄想症”

北约将中共国视作对其成员国的威胁的报告引起了北京的强烈反响，北京嘲笑说，北约已经过时了，而且“如果没有设敌并大力宣传来自敌人的威胁，北约就没有存在的理由。”

北京通过喉舌《环球时报》说：“这些威胁是北约的被害妄想症假想出来的。中国一向坚持和平发展，对北约成员完全没有军事上的威胁，而污蔑中国的技术发展只是为了将中国描绘成敌人。”

中共国进一步表示，除了进行一些联合演习外，中共国军方几乎从未到过北约报告中提到的为了证明中国不断扩大的影响力而触及的地区。“报告提到诸如大西洋、地中海和北极地区，实际上是北约在故意炒作中国威胁论。”

中共国外交部呼吁北约应以理性的方式看待中国的发展，而不是看成对任何人的威胁。
