据联合早报报道，美国前总统川普周二（4日）在他的网站开设了一个空间供他发布短信，并允许其他人把他的信息在推特和脸书上进行分享。

早前路透社报道，美国国会1月6日遭川普的支持者暴力冲击后，川普目前仍被推特和脸书“封杀”。而脸书内部的的监督委员会将在周三审议是否解除永久封禁川普总统账号的决定。

川普总统的高级顾问米勒在推特上说，该短信空间并不是川普计划要推出的社交媒体平台，并表示很快会公布更多消息。
