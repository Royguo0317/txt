# **亨特・拜登、布里斯马及腐败：对美国政府政策及相关质疑的影响（节选）**

[!\[\]()!\[\](https://spark.adobe.com/page/bPhuVbtJgxPdu/images/ba8c6736-59c2-4ed0-b145-16e171a1ef8f.jpg?asset_id=072f735e-c02d-478e-8ea5-e33e50f2dbac&amp;img_etag=%22c4667a84aa5f1d129e97fb423e71533a%22&amp;size=2560)](https://spark.adobe.com/page/bPhuVbtJgxPdu/images/ba8c6736-59c2-4ed0-b145-16e171a1ef8f.jpg?asset_id=072f735e-c02d-478e-8ea5-e33e50f2dbac&amp;img_etag=%22c4667a84aa5f1d129e97fb423e71533a%22&amp;size=1024)

**行政摘要**

2013年底至2014年，乌克兰基辅爆发大规模抗议活动，要求融入西方经济，结束困扰该国的系统性腐败。2月21日，乌克兰总统维克托-亚努科维奇放弃总统的权力并逃离了乌克兰，抗议活动达到高潮，至少有82人在抗议中被杀。不到两个月后，在短短28天的时间里，涉及拜登家族的一系列重大事件被揭露。

2014年4月16日，拜登副总统在白宫会见了儿子的商业伙伴德文-阿彻。5天后，拜登副总统访问了乌克兰，他很快就被媒体称为 “政府处理乌克兰问题的公众代言人”。访问的第二天，4月22日，阿彻加入了布利斯玛（Burisma）的董事会。6天后，4月28日，英国官员从布里斯马公司老板米科拉-兹洛切夫斯基（Mykola Zlochevsky）的伦敦银行账户中扣押了2300万美元。14天后，5月12日，亨特・拜登加入了布里斯马公司的董事会，在接下来的几年里，亨特・拜登和德文-阿彻因参加董事会而从一位腐败的乌克兰寡头那里得到了数百万美元的报酬。

2014年基辅的抗议活动被称为 “尊严革命”–乌克兰的反腐革命。在那场革命之后，乌克兰政治人物迫切希望得到美国的支持。 兹洛切夫斯基（Zlochevsky）确保了乌克兰相关官员很清楚亨特被任命为布利斯玛董事会成员的事情，并以此为筹码。亨特・拜登在董事会中的地位立即产生了潜在的利益冲突，给美国和乌克兰双方官员带来麻烦，并将影响乌克兰政策的实施。

主席们对潜在利益冲突的调查始于2019年8月，格拉斯利主席（Grassley）就奥巴马政府与亨尼格斯（Henniges）交易有关的潜在利益冲突致函财政部.1在奥巴马政府期间，美国外国投资委员会（CFIUS）批准了一项交易，将亨尼格斯（Henniges）的控制权交给了一家中国政府拥有的航空公司和一家与中国政府有固定关系的中国投资公司，Henniges是一家掌握军事应用技术的美国抗震技术制造商。参与亨尼格斯（Henniges）交易的公司之一是一家名为渤海华美（Bohai Harvest RST)的10亿美元私人投资基金。渤海华美（BHR）于2013年11月由中国政府的关联公司渤海资本和一家名为罗斯蒙特‧塞内卡（Rosemont Seneca Partners）的公司合并而成。罗斯蒙特・塞内卡（Rosemont Seneca）由时任副总统乔-拜登的儿子亨特・拜登（Hunter Biden）、前国务卿约翰・克里（John Kerry）的继子克里斯・海因茨（Chris Heinz）和其他2人于2009年成立。

由于刑事调查、弹劾程序、COVID-19的爆发和好几次阻挠行为，相关文件和证词的获取一直受到阻碍。因此,这项调查耗费的时间长于它本应消耗的时间。主席的努力一直是源于我们的信念，即公众有权了解政府内部发生的错误行为和利益冲突，特别是政府官员的行为所带来的冲突。这是一项廉政监督调查，它依靠的是美国机构和官员的文件和证词，而不是像我们的民主党同事所谎称的一场俄罗斯造谣宣传。

主席们在调查过程中发现，奥巴马政府知道亨特・拜登在布里斯马（Burisma）董事会的职位有问题，而且确实干预了对乌克兰政策的有效执行。此外，这次调查还显示了奥巴马政府内部的官员在其副总统的儿子加入乌克兰腐败寡头所拥有的公司董事会时，在多大程度上忽视了明显的警告信号。而且，正如后面的章节将讨论的那样，亨特・拜登并不是唯一一个利用乔-拜登作为副总统期间进行牟利的拜登家族成员。

本报告不仅详述了涉及Bidens家族的大量且复杂的金融交易的例子，还描述了其他美国政府官员在试图指导和支持乌克兰的反腐败努力时面临的困境。委员会将继续评估获得的信息和证据。

**主要调查结果**

▶️ 2015年初，美国驻乌克兰基辅大使馆前代理副大使乔治-肯特（George Kent）向副总统乔-拜登办公室的官员提出了对亨特・拜登在布里斯马董事会中的角色被认为存在利益冲突的质疑。肯特的质疑没有得到解决，2016年9月，他在给同事的电子邮件中强调：”此外，亨特・拜登在布里斯马董事会中的存在，对于所有在乌克兰推动反腐议程的美国官员来说，是非常尴尬的。”

▶️ 2015年10月，国务院高级官员阿莫斯-霍赫斯坦（Amos Hochstein）向副总统拜登以及亨特・拜登提出了质疑，认为亨特・拜登在布利斯玛董事会中的地位使俄罗斯的虚假信息工作得以进行，有可能破坏美国在乌克兰的政策。

▶️ 尽管肯特认为亨特・拜登在布利斯玛董事会中的角色对于所有在乌克兰推动反腐败议程的美国官员来说都是尴尬的，但委员会只知道有两个人–肯特和前美国特使兼国际能源事务协调员阿莫斯-霍赫斯坦–向副总统乔-拜登（霍赫斯坦）或其工作人员（肯特）提出了质疑。

▶️ 奥巴马政府官员的尴尬一直持续到他卸任总统之后。前国务卿约翰-克里对亨特・拜登在布里斯马（Burisma）董事会中的角色有所了解，但当2019年12月8日在新罕布什尔州纳舒亚市举行的市政厅活动中被问及此事时，克里撒谎道：”我对这些都不了解。完全不知道。不”。相反的证据详见第五节。

▶️ 前负责欧洲和欧亚事务的助理国务卿维多利亚-努兰德（ Victoria Nuland）作证说，对抗寡头将向乌克兰发出反腐信息。肯特告诉委员会，兹洛切夫斯基是一个 “可恶的寡头”。然而，2015年12月，副总统拜登的幕僚没有遵循美国对抗寡头的目标，而是建议他避免评论兹洛切夫斯基，并建议他说：”我不会去点名或指责个人。”

▶️ 亨特・拜登在布里斯马（Burisma）公司董事会任职时（据说是为公司治理和透明度提供咨询），据称兹洛切夫斯基（Zlochevsky）向乌克兰总检察长维塔利-亚雷马（Vitaly Yarema）手下的官员行贿700万美元，以 “了结针对兹洛切夫斯基的案子”。肯特作证说，这笔贿赂发生在2014年12月（亨特加入伯里斯马董事会7个月后），在得知此事后，他和驻地法律顾问向联邦调查局报告了这一指控。

▶️ 亨特・拜登从2009年1月29日至2014年7月8日是美国特勤局的保护对象。在他最后一次以保护者身份出访的前一天，《时代》周刊发表了一篇文章，描述了布利斯玛（Burisma）加大对美国官员的游说力度，以及亨特涉足布利斯玛（Burisma）董事会的情况。在结束保护细节之前，亨特・拜登在多个国外地点的旅行中接受了特勤局的保护，包括莫斯科、北京、多哈、巴黎、首尔、马尼拉、东京、墨西哥城、米兰、佛罗伦萨、上海、日内瓦、伦敦、都柏林、慕尼黑、柏林、波哥大、阿布扎比、内罗毕、香港、台北、布宜诺斯艾利斯、哥本哈根、约翰内斯堡、布鲁塞尔、马德里、孟买和科莫湖。

▶️ 民主党的俄罗斯虚假信息的代言人安德里-特利岑科（Andrii Telizhenko）与奥巴马政府官员，包括奥巴马国家安全委员会成员伊丽莎白-赞托斯（Elisabeth Zentos）至少会面10次。民主党的一家游说公司 “蓝星战略”（Blue Star Strategies）在2016年至2017年期间与特利岑科签订了合同，并在最近的2019年夏天继续要求他提供协助。最近的一篇新闻文章详细介绍了其他Telizhenko与奥巴马政府官员之间的其他广泛接触。

▶️ 除了布里斯马（Burisma）为亨特・拜登和阿彻的董事会成员身份支付了400多万美元外，亨特・拜登及其家人和阿彻还从背景可疑的外国人那里获得了数百万美元。

▶️ 阿彻从哈萨克斯坦的肯吉斯・拉基舍夫（Kenges Rakishev）那里收到了142,300美元，据称是为了购买一辆汽车，同一天，副总统拜登与乌克兰总理阿尔谢尼・亚采尼克（Arseniy Yasenyuk）一起出现在基辅，并就俄罗斯在克里米亚的行动向乌克兰议员发表讲话。

▶️ 亨特・拜登收到莫斯科前市长夫人埃琳娜-巴图琳娜（Elena Baturina）350万美元的电汇。

▶️ 亨特・拜登与董功文开设银行账户，资助了詹姆斯-拜登和萨拉-拜登10万美元用于在全世界的消费。

▶️ 亨特・拜登与叶简明、董克文，其他与中共政府和人民解放军有关的中国公民有商业联系。这些关联带来了数百万美元的现金流。

▶️ 亨特・拜登付钱给非本国妇女，这些妇女是俄罗斯或其他东欧国家的国民，似乎与 “东欧卖淫或人口贩运集团 “有关。
