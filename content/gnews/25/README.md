- **12.20. ZeroHedge** [数据分析师: 乔-拜登在使用Dominion或HART投票机的县的得票率看起来更好Joe Biden Appears To Outperform In Counties Using Dominion Or HART Voting Machines: Data Analyst](https://www.zerohedge.com/political/joe-biden-appears-outperform-counties-using-dominion-or-hart-voting-machines-data-analyst)


“分析证据表明，使用Dominion Voting Systems公司生产的Dominion X/ICX BMD（选票标记设备）机器和HART InterCivic公司的机器似乎对选举结果产生了异常影响。”数据分析师在周四公开的一份报告中写道。

据《大纪元时报》查阅的资料显示，这名数据分析师出于安全考虑选择匿名，他有30年的数据分析经验

在研究中，他利用投票数据和2017年的人口普查数据建立了一个预测选举结果的基线情境。然后，他用基线模型来比较11月选举的结果。

结果显示，在78%使用这两家公司投票机的县，拜登落在预测线以上。

他的分析还表明，拜登在这些县的得票率始终比他应该获得的多5.6%。

“这不应该发生……使用Dominion机器的县显然太高了，”他在介绍自己研究成果的视频中说。

“在我看来，这是一个戏剧性的危险信号……这里发生了一些事情。”

他说，”加5.6%”的格局并不是因为上述投票机在民主党强势县广泛使用造成的，因为在总统唐纳德-特朗普支持率较高的县也能发现这种情况。

该研究尚未经过同行评议。

Hart InterCivic公司没有立即回应《大纪元时报》的评论请求。多米尼投票系统公司拒绝对分析结果发表评论。该公司表示，”美国各地的法院已经多次确认，没有可信的证据表明使用多米尼克投票系统存在任何换票或舞弊行为。”

分析称，他没有对谁造成可能的违规行为作出任何结论。

不过，他认为，如果他分析中的潜在违规行为是由故意造成的，那一定是一个大阴谋。

“我所看到的模式是横跨许多县的–所以我的发现涉及一个具有大规模、全国性影响的组织。也就是说，我看到的不是类似于一个地方的光学扫描操作员将一箱选票在系统中运行三次这样小规模的情况。”

他没有指出任何公司或相关方的错误行为。
