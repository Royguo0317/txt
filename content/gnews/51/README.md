先认识一下疫苗。所谓疫苗就是将少量具有病毒特征的物质注入人体，人体的免疫系统对病毒识别，然后产生抗体，会对以后进入身体的病毒予以攻击、消灭。当然，疫苗中的病毒是经过了灭活和减毒。就像毒蛇，两颗毒牙被拔掉了。但是，如果毒牙没有拔掉，或者拔掉了一颗，或者没拔干净，那么后果就严重了。相当于病毒人为地注射身体中。当然了，会有三种结果。第一，直接感染，表现出中招CCP病毒的各种症状。巴基斯坦总理伊姆兰·汗就属于第一种，在接种中共疫苗的两天后确诊。第二，没有症状，但是已经被感染。俗称的无症状感染者，这样的人数恐怕是众多的。身体的免疫系统和病毒达成一种可怕的平衡，没有发热、咳嗽等症状，但是体内已经有病毒存在。无症状感染者的最危险之处在于具有传染性、隐蔽性。因为无症状，患者自己都不知道被感染，具有相当的欺骗性。第三，抵抗住了CCP病毒的攻击，产生了抗体。第三种人属于免疫力强大的那种，强大的病毒在他的面前被驯服，就像当时欧洲流行的“黑死病”，总有一些幸存者。这样的人属于可遇不可求。
