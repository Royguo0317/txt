作者：香草山福音部  山城小哥

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3.jpg)



《士师记》中有一段描述：“每逢以色列人撒种之后，米甸人、亚玛力人，和东方人都上来攻打他们，对着他们安营，毁坏土产，直到迦萨，没有给以色列人留下食物，牛、羊、驴也没有留下；因为那些人带着牲畜帐棚来，像蝗虫那样多，人和骆驼无数，都进入国内，毁坏全地。以色列人因米甸人的缘故，极其穷乏，就呼求耶和华。”(士6:3-6 )

![]()![](https://gnews.org/wp-content/uploads/2021/03/Bible1.jpg)

从这里，我们可以看见仇敌深知征战中粮食的重要性。所以每当神家播种以后，仇敌就从四面八方全部一起上来，从根上全面破坏粮源，一点不给神家儿女们留下。因此，我们基督徒不仅要多多播种，而且要学会持守，不能让飞鸟来吃尽种子，不能让仇敌来破坏。很多时候，再好的道，我们是今天听了，明天就忘了，并没有吸收进去。所以我们要学会反刍，慢慢消化，反复吸收，持守住起初得到的爱心，持守神的话语。像大卫说的那样“昼夜思想”，便为有福。这样，我们在属灵的征战中就不会饥饿，充满力量。

实际上更高的境界和神常常要求我们的是“以战养战”，让仇敌成为我们的粮食。大卫在诗歌里说“你在我仇敌面前，为我摆设了筵席”，其实仇敌就是我们的粮食。这怎么理解呢？在生活中，侍奉中，我们常常遇到许多困难，许多艰难的环境。像约伯一样，无故受到许多苦难。但其实就是在这样的环境之中，我们靠主得胜以后，反而会更深刻地认识神，炼净自己，真正活出耶稣基督的样式。如参孙一样，狮子不但没有能吃掉他，反而他靠着神的灵杀死了狮子，还从狮子那里得到了蜂蜜，不仅自己吃，还可以分享给家人吃。所以他给非利士人出谜语的时候，发自内心感慨地说“吃的从吃者出来，甜的从强者出来”，因为这是神给他的独特的经历，所以没有人能猜出来。我们每个人其实都有这样的独特得胜经历，它们就是神给我们独有的灵粮。虽然可能只是五饼二鱼，但分享出去，却能使众人得饱足。

在爆料革命里，我们看到，每当文贵先生要给战友们谋一些福利，希望战友们，特别是贫寒的战友们能通过G系列获得一点利益的时候，中共就上来百般阻扰，无非就是想“饿死”绝大部分贫寒的战友们。但是每次文贵先生都能靠着神给的力量胜过这些仇敌制造的陷阱和环境。并且和川普总统提高对中共的关税来补贴美国农民一样，文贵先生不但断共匪粮道，而且用G系列和一系列司法调查、资产查封等手段隔空取钱，让中共盗窃的资产成为我们爆料革命战友们的粮食。真可谓是以战养战的高人！

如摩西和亚玛力人征战得胜的秘诀是祷告一样，文贵先生也是一个常常祷告的人。愿我们每个弟兄姊妹都常常祷告，征战得胜，让“吃的从吃者出来，甜的从强者出来”。把荣耀归给基督。   阿门。
