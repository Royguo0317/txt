2021年中共两会还在召开中，讽刺的是由于CCP病毒原因预计到3月16号左右才完毕的会议缩短到3月10日左右。都会提出很多与我们社会各阶层广大百姓利益没有多少相关的提案，就算有也是一些缺乏建树，没有专业性，站在统治者一边的提案，并没表现出解决百姓问题，社会问题的诚意和意愿。这反应了这些所谓的“人民代表“他们根本不关心百姓，因为他们本来就没有经过我们百姓的选举，”这些人民代表“都是中共非法政权选出来的非法代表。

那么这些非法代表都有哪些不专业，没有水平，跟法治社会格格不入，也完全跟我们老百姓利益没有丝毫关系的提案和言论呢，下面笔者就汇总一些最具代表性中共两会代表提案和言论。

连续当选13届中共“人民代表”的申纪兰是非常典型的中共的好奴才，每届她都是为中共的切身利益着想，为中共的独裁统治着想，凡是中共想的，她必然举双手赞同，甚至于一副比中共还急切的样子。外界对于申纪兰送与其“雅号”： “举手机器”、“恐龙”、“僵尸”等。”
申纪兰最出名的名言：“我非常拥护共产党。当代表就是要听党的话，我从来没有投过反对票”。申纪兰在2012年中共两会上提案说：“这个网，你不能想上就上，我有个想法，网也应该有人管，不是谁想弄就能弄，就跟人民日报一样，外国那些人那是瞎弄的，咱不能这样，要按照原则去弄，不要好的弄成坏的了，想说什么就说什么，咱是共产党领导下的社会主义国家。”

此话确实让人惊掉下巴，申纪兰所说的就是中共最想的，中共专制不但在政治上而且在思想和言行上想要控制每一个中国人，所以网络对于中共来说就是双刃剑，因为网络一方面可以加强中共的统治洗脑，但是另一方面网格也打开了西方自由世界的天窗，让没有出过国的中国人能够看到自由民主的世界是什么样子。

不过这个丑陋的人终于在2020年6月28日去世了，俗话说：“好人命不长，祸害活千年”，享年90岁。当然她去世的之前还不忘最后祸害一把好人，申纪兰一生最后投票日为2020年5月28日，投票支持“港版国安法”。

当然申纪兰虽然是“举手党“里面最典型最具代表性的中共人大代表，但要论知名度和网络爆光度那是远远不如中国电力国际有限公司前董事长，也是李鹏之女李小琳，李小琳在十一届五次人大代表会上提出，“我觉得应该给每个公民建立一份道德档案，以此约束大家，每个人要知耻。”

这位红二代中共人大代表，也是非常典型的代表中共红二代想法的人，中共的红二代都以贵族自居，因为在他们的心里面认为江山是他们打下来的，他们就应该封候拜将，世袭罔替，他们决不会正眼看老百姓一眼，他们对百姓的看法正如中共元老姚依林女婿王岐山所说：“中国百姓就要管，不管要乱“。红二代从来都以统治者自居，认为百姓就是奴才必须要管，李小琳所说的当然是给老百姓建立所谓的道德档案，而中共自己肯定不会受这个约束！

由于李小琳言论惊人，网上送称号“李电表（婊）“，网友还戏称：“煤她家挖的，电她家发的，表她家做的，线她家拉的，价她家定的，钱她家拿的…他们家建立了最成功的中国模式 — — 权贵资本主义” 当然这样说是有原因的，众所周知的三峡工程20多年来一直收取中国人每度电1分多钱所谓的“三峡建设基金”，这些钱当然全部进了李鹏家族的口袋。李小琳日子过得是相当滋润，多年以来李小琳一直就是两会“超模”，每次会议都身着不同的奢侈时装品派，这些时装售价高达惊人的数万人民币，这对于一般百姓来说简直咂舌，这些奢侈时装品牌的名字一般百姓闻无未闻。

其它奇葩雷人言论和提案：

2013年：两个热点分别是“乳”和“房”的问题。众所周知，中国老百姓对国产奶粉普遍缺乏信心。可是，全国政协委员、国家质检总局副局长杨刚声称：“我亲友家小孩90%都吃国产奶粉。”一位网民在跟帖时意味深长地说：“有种国产叫特供…”

2013年：刚刚参与了温家宝总理最后一次政府工作报告起草工作的全国人大代表、中科院某研究所所长王毅：“企业也可以通过提高价格将一部分税负转移给消费者…老百姓不能总想着呼吸新鲜空气却不付出代价。”

全国人大代表、中华慈善总会荣誉副会长周森提出的建言是慈善立法。他说：“每个人的工资，必须要有一笔钱进行慈善公益，就像现在纳税一样。”

2013年：说到慈善事业，被称为“中国首善”、以当代活雷锋自居的陈光标在毛泽东“向雷锋同志学习”题词50周年纪念日身着雷锋范儿军装接受搜狐访谈时称：“我和雷锋的最大差距是缺少毛主席题词。”

2017：全国政协委员、人民出版社社长黄书元在两会期间提交提案，建议相关部门制定《国家英烈名誉保护法》等专门性法律，通过立法明确革命英烈的法律地位，在民法、刑法以及社会管理类法律中，增加规定对革命英烈名誉的保护和相关法律责任，对侵害革命英烈名誉案件的诉讼管辖做出司法解释。

评：此举是维护中共统治，进一步扼杀言论，消声批评中共编造假英雄的声音，比如课本上杜撰的英雄”雷锋，董存瑞，黄继光等”。

2017:中国八大“花瓶”党派之一民进中央在相关提案中提出了一些建议，包括提升网速以及设置境外网站访问权限的负面清单。对于所谓”负面清单”，具体建议是，对于与相关法律法规相抵触的网站内容，列入负面清单予以严格监管并屏蔽，禁止访问；对于非政治类网站，特别是科研人员访问最多的境外大学和科研院所等网站，予以全面放开；对于中立性网站，比如搜索引擎、信息类、技术类等，对其中的部分敏感内容进行筛选。

2019年：人大代表罗杰则将他的目光放到了“全民减重”上。他认为，中国应加大对健康管理的投入，设立专门的机构来做关于减重方面的工作。
据中国媒体《文汇报》报道，罗杰建议当局从行政、事业单位的公职人员入手，出台公职人员体重及健康管理规定：男性腰围超过33.5英寸（85厘米）、女性超过35.4英寸（90厘米），且血糖、血压、血脂出现任何一项不合格，必须在3个月内自行减肥。

2021年: 全国政协委员白岩松在接受媒体采访时提出了一个建议：60-70岁退休人员对西北、西南和东北地区进行三年为周期的准志愿服务。

评：中共的养老保险亏空严重，加上现在中共打鸡血的经济再过不多久可能出现严重的问题，所以中共意图全面推进延迟退休，此举笔者认为中共放白岩松出来对社会舆论的试探。

2021年：政协委员建议鼓励大中小学定期开设学农类实践活动。
评：这是学毛的上山下乡运动，意在拍全面毛化的习的马屁。

2021年：中共全国政协委员、九三学社中央委员许进称，“在义务教育阶段，英语等外语课程不再设为与语文和数学同等的主课，增加素质教育课程占比；不再将英语（或外语）设为高考必考的科目；禁止义务教育阶段学生参加非官方的各种外语考试。”

评：英语是通向自由世界的桥梁，此举是想进一步扼杀百姓对于西方世界的认知。
