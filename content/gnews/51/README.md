据美国有线新闻（CNN）报道，两名政府官员透露，气候特使约翰·克里（John Kerry）本周将前往中共国，讨论美中在气候变化问题上可能采取的联合行动。近一个月前，拜登政府官员与中共官员在阿拉斯加举行了紧张的会谈。

报道称，克里此次访华将气候危机列为拜登政府外交政策的首要任务。拜登政府还承诺与中共国竞争，并在人权问题上对中共进行谴责。中共国是全球第一大碳排放国，气候专家表示，解决环境危机的唯一途径是与中共合作。这将是拜登政府首次对中共国进行正式访问。

此次访问将在4月22日地球日线上全球领导人气候峰会之前举行。美国总统拜登（Joe Biden）上月对记者表示，中共国国家主席习近平和俄罗斯总统弗拉基米尔•普京（Vladimir Putin）“知道他们受到了邀请”。预计拜登将在峰会上概述美国在2030年前减少碳排放的目标——即巴黎气候协议下的既定贡献。

美国国务院发言人告诉CNN，克里和俄罗斯外长拉夫罗夫上周在新德里“就气候问题进行了简短的非正式对话”。这位发言人说，两人“都有不同的行程，碰巧住在同一家酒店”，他们在那里进行了简短的交谈。

报道指出，美中关系的变化，让克里对中共国的访问尤其引人注目，不仅是因为其地缘政治影响，还因为上月美中官员在阿拉斯加州安克雷奇(Anchorage)举行的一次面对面会晤，引发了双方在外交上的公开交锋。美国国务卿安东尼·布林肯（Antony Blinken）表示，中共在新疆，香港和台湾等地的行为，以及对美国的网络攻击和对美国盟友的经济胁迫，威胁着维护全球稳定的规则和秩序。
