**西班牙2021年3月11日电/西喜社——**

伊斯坦布尔（路透社）3月8日消息 — 与伊斯坦布尔数百名妇女抗议中共国对维吾尔族人的迫害，努西曼-阿卜杜拉西特（Nursiman Abdurasit）泪流满面地想起她在新疆被囚禁的母亲，并担心像她这种在土耳其的维吾尔人有一天也会因为引渡协议被迫送中。

中共国于去年12月批准了中国和土耳其两国之间的引渡条约，该协议目前还在等待安卡拉议会的最后批复。生活在土耳其的约4万名维吾尔族人中的活跃分子加大力度宣传他们的困境与恐惧。

阿卜杜拉西特（Abdurasit）2015年来到土耳其学习，四年前与家人失去联系。去年夏天，她发现自己的父母和两个兄弟因涉嫌从事未指明的 “恐怖主义相关活动 “而被判处长期监禁。

她在中国驻伊斯坦布尔领事馆附近的伊斯坦布尔海滨纪念妇女节的活动中，愤慨发言：“我知道我的母亲过去四年中一直在集中营中受到压制。”

在她周围，1000多名群众举着失踪亲人的照片，挥舞着代表新疆的东突厥斯坦独立运动的蓝白旗帜。

联合国专家估计，至少有100万维吾尔人和其他穆斯林被关押在中国西北新疆地区的集中营，美国1月表示，中共国镇压维吾尔人犯下了 “种族灭绝和反人类罪”。

中共却矢口否认对新疆侵权行为的指控，并称其在该地区设立的集中营是提供职业教育的培训中心，以帮助铲除伊斯兰极端主义和分裂主义。

中共国大使馆发言人上个月说，最近几个月经常在中国驻土耳其外交机构附近举行抗议活动的维吾尔人，试图欺骗土耳其人民，破坏双边关系。”这些人编造谎言的目的是利用新疆的相关问题来攻击中国政府对新疆政策，黑化中国的形象”他说。
