情报官员星期二警告说，中共国是一个“近距离”的竞争对手，正在以各种方式挑战美国。

国家情报总监办公室的一份报告指出，中共国对南海的侵略、与俄罗斯的关系以及一带一路的国际倡议，都是外交政策关注的问题，官员们认为这是最大的安全隐患。

俄罗斯、伊朗和朝鲜的安全挑战，使其成为美国下一个最大的国际威胁，其次是中共病毒大流行性病、气候变化和新兴技术等全球风险。

星期二发布的报告说：“北京、莫斯科、德黑兰和平壤已经显示出了以牺牲美国及其盟国为代价来增进自己的利益，尽管疫情已经蔓延。”中共国越来越成为一个近距离的竞争对手，在多个领域——特别是在经济、军事和技术方面——挑战美国，并正在推动改变全球准则。”

报告阐述了美国对中共国发展大规模杀伤性武器、太空和反太空能力的担忧，以及“至少”可能导致美国“关键基础设施中断”的网络攻击。

俄罗斯的地区活动也与美国高级安全官员有关，尽管这不是国际主导地位，而是一种挑衅影响。报告指出，俄罗斯试图破坏美国与西方国家的联盟，并“把全球事件塑造成一个主要角色”

该报告不仅可能会影响拜登政府如何应对俄罗斯的威胁，而且还将影响与伊朗的谈判。

有关伊朗的部分暗示，安全官员不认为这个中东国家目前正在推进“关键的核武器开发活动”，但指出，在取消制裁之前，伊朗领导人很可能不愿意在核不扩散问题上进行合作。

朝鲜对美国的主要挑战包括继续发展大规模杀伤性武器以及它对韩国、日本和美国构成的威胁。

国家情报局局长艾薇儿·海恩斯（Avril Haines）和中情局局长威廉·伯恩斯（William Burns）将于周三在国会山参议院和众议院情报委员会上作证。
