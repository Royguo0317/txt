## 【信仰之声】《圣经》与中国传统文化之——麒麟

《圣经》里曾提到过一种被称为“河马”（behemoth，希伯来文编号930）的动物，但恐怕它并不是我们所熟悉的河马。

![]()![](https://gnews.org/wp-content/uploads/2021/02/图片7-20.png)

《圣经》对它的记载只有下面这一处：

- 15“你且观看河马。我造你也造它，它吃草与牛一样。
- 16它的气力在腰间，能力在肚腹的筋上。
- 17它摇动尾巴如香柏树，它大腿的筋互相联络。
- 18它的骨头好像铜管，它的肢体仿佛铁棍。
- 19“它在神所造的物中为首，创造它的给它刀剑。
- 20诸山给它出食物，也是百兽游玩之处。
- 21它伏在莲叶之下，卧在芦苇隐密处和水洼子里。
- 22莲叶的阴凉遮蔽它，溪旁的柳树环绕它。
- 23河水泛滥，它不发战，就是约旦河的水涨到它口边，也是安然。
- 24**在它防备的时候，谁能捉拿它？谁能牢笼它穿它的鼻子呢？”（伯**40:15-24**）**


**总结一下**，这种动物有以下特征：

1. 生活在陆地。
2. 像牛一样吃草。
3. 大有能力，身上有类似刀剑的武器，身体仿佛铜管铁棍，摇动尾巴像香柏树。
4. 在神所造的物中为首。
5. 它觅食的地方，百兽在那里游玩，即它虽然厉害，却很和平。圣经中提到的四种厉害动物大鱼、河马、鳄鱼和拉哈伯中，只有它没有被神攻击的记录。


**然后我们看一下中国古籍对麒麟的记载：**

1. 麒：仁兽也。麋身牛尾，一角。从鹿其声。麐：牝麒也。从鹿吝聲。（《说文解字》）
2. 设武备而不为害，所以为仁也。（段玉裁《说文解字注》）
3. 麒麟者，仁兽也。牡曰麒，牝曰麟。不刳胎剖卵则至。麕身而牛尾，狼项而一角，黄色而马足。含仁而戴义，音中钟吕，步中规矩，不践生虫，不折生草，不食不义，不饮洿池，不入坑阱，不行罗网。（《宋书-**志**–**卷二十八》**）
4. 有毛之虫三百六十，而麒麟为之长。（《大戴礼记-易本命第八十一》）


将其与“河马”逐点对照一下：

1. 都生活在陆地。
2. “河马”吃草，麒麟走路都不踩别的动物，大概也不会吃肉。
3. 能力方面，上面说了，麒麟有一角，又说它有武备，也就是武器。从古文描述上没找到太多其他描述，但麒麟的图片基本还是比较符合上面对“河马”的身体和尾巴的描述。
4. 有毛之虫三百六十，而麒麟为之长，就是说长毛的动物有很多，而以麒麟为首。这与“河马”的描述符合。
5. 古籍称麒麟为仁兽，宋书里边的描述就更说明这一点了，也**很符合。**


综上，我猜测《圣经》中的“河马”就是中国传说的麒麟。
