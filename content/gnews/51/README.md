美国国务院发言人办公室
2021年3月12日

以下声明由加拿大、法国、德国、意大利、日本、联合王国和美利坚合众国组成的七国集团(G7)外交部长以及欧洲联盟高级代表发表。

*开始正文:*

我们，加拿大、法国、德国、意大利、日本、英国和美利坚合众国的七国集团外长以及欧盟高级代表，一致对中共国当局从根本上侵蚀香港选举制度的民主成分的决定表示严重关切。这一决定强烈表明，中共国大陆当局决心消除香港的不同声音和反对意见。

全国人民代表大会通过的一揽子改革方案，加上大规模逮捕民主运动人士和政治人物，破坏了香港在“一国两制”原则下的高度自治。该方案还将扼杀政治多元化，这与《基本法》所规定的争取普选的目标背道而驰。此外，这些变化还将削弱言论自由，而这是《中英联合声明》中所保障的权利。

我们应该信任香港人民会为香港的最大利益而投票。对不同的观点进行讨论，而不是压制，才是确保香港稳定繁荣的途径。

我们呼吁中共国按照《中英联合声明》及其他法律义务行事，并尊重《基本法》所规定的香港基本权利和自由。我们还呼吁中共国和香港当局恢复对香港政治机构的信心，并结束对那些倡导民主价值观、捍卫权利与自由的人的无端打压。

*全文结束*
