**中共战机大举入侵台湾防空识别区**

台湾国防部称中共1架次Y8反潜巡逻机、8架次轰6K、4架次歼16接近台湾西南防空识别区。此时美军罗斯福号航母打击群正进入南中国海，这次中共可能是模拟对其进行攻击。这是中共使局势升级的做法，美国政府发表声明敦促中共停止对台湾施压，并再次宣称对台湾的承诺坚定不移。这是美国的一个表态，警告中共如果升级美国会有对应的措施，措辞非常强烈，如果中共轻举妄动美国一定会出手。这是中共不断试探拜登政府的又一手段，同时也是对拜登政府实行的施压政策。蓬佩奥先生多次说过中共违反三个联合公报，这次对台声明拜登政府搞了平衡，不往前推进也不颠覆上任政府的做法，但保卫台湾的决心没有任何改变。

**习近平打算进攻台湾的理由**

党内斗争升级，给习压力最大的就是种族灭绝和反人类罪，习对应的回答就是香港赢了，120终极之战又大获全胜。中共病毒和119的定义已经把中共重回国际的路基本堵封死。这促使习为转移国内矛盾和清理军队异己很可能对台湾动武，完成他所谓的统一大业。军队一旦动起来，习可以把其它派系军内的势力作为炮灰消灭掉。用占领台湾来缓解经济上和其它方面的危机，同时可以提高党内的威望，近似于比肩毛泽东，这是一个巨大的政治资本，这样他就彻底安全了。这份功绩是独裁者想要掌握权力和统治人民最好的理由。习认为他与美国现在的关系使得美国不会插手对台湾的进攻，当然这只是习自己的想法，如果走向这个方向的话，只能是继续加速，结果就是内斗加剧以共灭共，外部美国介入军事打击，中共走向灭亡。

**美国新政府在台湾问题的态度**

中共在大选和病毒等事情上本来就已经给美国造成了巨大的压力，如果说这个时候中共对台湾动手，那美国不出手也得出手，这是大的形势决定了的。如果美国不和台湾站在一起，美国将失去在亚太地区的主导权，这也涉及到美国在亚太同盟一系列的问题。习如果对台采取军事行动，美国肯定会冻结中共在美国的资产，其中大部分是胡江王三大势力拥有的，为了打击对手，习对台湾动手的机会很大。所以习如果攻打台湾，内部三大势力的活路就越来越小，这几股势力绝对要做出动作来反击，这也是可以给拜登政府一个新的抓手解决中共问题。所以从内外两个角度来说，美国肯定会派兵阻止中共对台湾的占领。

**拜登新政府对华态度对中共的影响**

今日拜登表态完全赞同对中共种族灭绝和反人类罪的定义，表示将对北京侵犯人权的行为采取强硬态度，他将不得不对中共国的种族灭绝行为采取行动。同时美政府顾问表态病毒可能来源于实验室，今天的华尔街日报也说来自于实验室，支持拜登加入WTO并控制对病毒的调查，让谭德赛下台。如果WTO不能组织力量去调查真正的病毒来源真相，美国届时再退出WTO。同时今天对中共军机逼近台湾，美国也发表措辞强硬的声明.这一系列的事情等于说让习近平看到，拜登虽然上来但对中共没有任何松绑，习觉得赌错人被耍了。同时党内斗争三大势力追问到底120赢到哪里了？各方面逼宫，习为了证明自己的政治高度，很大可能不得不打台湾牌。
