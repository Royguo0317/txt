#### 译评：

此篇报道主要围绕着一名悉尼突发的本地新增病例，文中没有展开背景介绍。这位不幸感染中共病毒的安保人员因为在隔离酒店工作，属于高风险行业，所以接种的疫苗是号称有效率比较高的辉瑞疫苗。他接受了第一次疫苗注射十余天后病毒检测为阳性。澳大利亚因为一直实行比较严格的限制边境措施，总体疫情控制较好。虽然此文分析可能是受海外旅客传染，而这位保安接触感染者的概率不算太高，最后依然感染了中共病毒，可见疫苗的预防效果并不理想。

西方科学家和政治家对这些不足一年历史的疫苗的潜在风险所知甚少，却信誓旦旦地声称其安全。实际上已经有学者发表论文指出中共病毒疫苗可能导致心血管疾病，老年痴呆，震颤麻痹，甚至有学者推测其可能诱发胰腺癌等恶性肿瘤。也有很多案例报告疫苗注射后流产，有些人出现类似中共病毒感染后的缠绵不愈的症状（**long-haul symptoms**）。最近德国，法国，意大利，西班牙都因为副作用问题暂停了阿斯利康（**AstraZenec****a**）的应用。而这个疫苗正是普通澳大利亚人所将接受的疫苗。由此可见，如果不去朔源中共病毒源头，恶魔中共不灭，全世界开发的任何疫苗都无法保证有效性和安全性。

#### 译文：

澳大利亚悉尼3月15日消息，在保持了将近两个月的无新增病例后，有一名接种了第一剂辉瑞疫苗的隔离酒店安保人员的中共病毒检测呈阳性。这位47岁的男子在两家检疫酒店从事安保工作。新南威尔士州卫生部长布拉德·哈扎德（Brad Hazzard）表示，这位保安是在工作期间被感染的。首席卫生官凯瑞·尚特（Kerry Chant）博士说，上周在索菲特酒店，他可能被返回的旅行者感染，但当局（对传播链调查）一直保持“开放态度”。尚特博士说，他于3月2日接受了两剂辉瑞疫苗的第一剂，但可能需要长达14天的时间才能影响他的免疫系统。

新州卫生部长哈扎德表示，已在周六晚间将当地新增病例告知当局，并且正在追踪密切接触者和进行基因组测序。 尚特（Chant）博士说：“我们已经联系了于3月12日（星期五）晚上7点到3月13日（星期六）早上7点时间段内工作的大约130位（检疫人员）。我们正在要求这些人立即自我隔离并进行病毒测试……这使我们有时间进行研究并确定与该保安密切接触的检疫人员的情况。” 新州警方在一份声明中表示，它将继续审查和监控隔离过程，并提供广泛支持，以确保所有感染控制措施均已到位。

任何在3月13日上午10:45至中午12点到访了在比弗利山庄（Beverly Hills）的岩石区（the Rocks）的餐厅Pancakes 的人都被确认为密切接触者。在此时间段，这名被感染的保安参与了那里的几个活动。无论测试结果如何，当时到访该场所的人都必须进行自我隔离14天。

该患者于3月12日星期五和3月13日星期六曾乘坐好市围（Hurstville）T4线到达中央换乘线（Central line）。目前，这两条公共交通路线也已处于戒备状态。

该患者乘坐从好市围（Hurstville）到市区的火车，于3月12日星期五下午6:30抵达；并于3月13日（星期六）上午9点至9点30分访问了贝克斯利水上运动中心（Bexley Aquatics Centre）。

上周3月8日至3月12日，他还多次访问了好市围私人医院（the Hurstville Private Hospital）的一家咖啡店。这位保安的全部家庭成员的COVID-19测试结果呈阴性。

尚特（Chant）博士说：“我们已经联系了大约130人，他们于3月12日（星期五）晚上7点到3月13日（星期六）早上7点工作”，“我们正在要求这些人立即自我隔离并进行病毒测试……这使我们有时间进行研究并确定与该安保人员密切接触的检疫人员的情况。”

新州警方在一份声明中表示，它将继续审查和监控隔离过程，并提供广泛支持，以确保所有感染控制措施均已到位。“作为这些询问的一部分，新州警察将通过审查闭路电视，进行面部识别和分析名册来协助健康调查员。”

新的感染打破了新州55天没有本地新增案例的连续零纪录。

总理斯科特·莫里森（Scott Morrison）淡化了有关此案例的任何警报，声称他没有受到打扰，疫苗接种并没有立即生效。“我对此并不感到惊讶，因为您担心这些事情会在奇怪的情况下发生。”
