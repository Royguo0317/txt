**推荐者语：**

针对中共国在新疆实施种族灭绝及反人类罪，美国多次发出声明及作出对应制裁。与此同时，欧盟、英国与加拿大近期联手美国，相继对四名参与新疆种族灭绝罪的中共官员以及一个实体作出相应制裁。但中共为了掩盖其犯罪事实，并试图通过其一贯的“战狼”外交施压反制英、美、欧、加的联手制裁，竟然随后对欧盟及英国的多名人员及多家机构进行反制裁。而最近，中共再一次发起“反击”，继续反制裁美加两国的人员及机构。但殊不知，中共和习总加速师的这一系列的疯狂行为，也为灭共大大加速。

中共国外交部3⽉27⽇宣布对美国、加拿⼤的个⼈与实体进⾏制裁，称这是针对两国此前“基于谎⾔和虚假信息”的对华制裁。对于北京（中共）此举，美国和加拿⼤进⾏谴责，表示不会停⽌捍卫⼈权的⾏为。

美国国务院3⽉27⽇发表了布林肯的声明，声明写道，“美国谴责中华⼈⺠共和国（中共国）对两名美国国际宗教⾃由委员会（USCIRF）委员进⾏毫⽆根据的制裁，这显然是对美国制裁与新疆严重侵犯⼈权有关的中（共）国官员的报复。此前，中共国在1⽉制裁了28名美国官员，并在2020 年7⽉制裁了美国官员和在世界各地促进⺠主和⼈权的组织。”

布林肯称，“北京（中共）试图恐吓和压制那些为⼈权和基本⾃由发声的⼈，这只会促使国际社会对新疆正在发⽣的种族灭绝和危害⼈类罪进⾏越来越多的监督。我们声援加拿⼤、英国、欧盟以及世界各地的其他伙伴和盟友，呼吁中（共）国停⽌对新疆以穆斯林为主的维吾尔族和其他少数⺠族和宗教群体成员的⼈权侵犯和虐待，并释放那些被任意拘留的⼈。”

在布林肯发表上述声明前，加拿⼤总理特鲁多已经对北京（中共）的制裁表示谴责，誓⾔捍卫⼈权。特鲁多在推特上写道：“中共国的制裁是对⾔论⾃由和透明度的攻击，⽽这些价值观是我们⺠主制度的核⼼。我们与议员们站在⼀起，反对这些令⼈不可接受的⾏动。我们将继续与我们的国际伙伴⼀起捍卫世界各地的⼈权。”
