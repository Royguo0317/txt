04月01日，美国国务院发言人内德·普莱斯（Ned Price）在通报情况时评论了俄罗斯外长拉夫罗夫与华盛顿的关系“已经达到谷底”的说法，并列举了美方对莫斯科的不满。

其中，他点出了“2016年和2020年对民主的攻击”其中有黑客攻击的指控，俄罗斯与塔利班勾结的报道，以及阿列克谢·纳瓦尔尼的审判。他说：“正因为如此，莫斯科和华盛顿的关系就处于这样的状态。”

同时，普莱斯指出，华盛顿寻求稳定与莫斯科的关系。

“确实，我们继续寻求与俄罗斯建立稳定和可预测的关系。仅仅因为我们之间存在如此深刻的分歧，而且关系处于目前的状态，并不意味着不会有任何（利益）趋同的领域。”国务院发言人说。

拉夫罗夫表示，俄罗斯并不希望与美国发生对抗，但不会允许冒犯。俄外交部负责人将美国的行为比作“院子里的首席恶霸”的行为。拉夫罗夫称，虽然“（与美国）的对抗已经到了谷底”，但“在内心深处”仍然希望“所有的人都是成年人，都明白与紧张局势上升有关的风险”。

去年12月，俄罗斯总统普京在年度新闻发布会上谈到与西方国家关系恶化时，他说，他觉得自己要为“俄罗斯及其人民的遭遇负责”。“而我将为俄罗斯联邦的利益做一切事情。”
