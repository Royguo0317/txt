## 中共超限战灭港计划（十五）占领舆论阵地之私营报纸的存亡
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1001701/)

搜集/撰文：岁月如歌 / 封面合成：文粤

港英密电中所报告的秘密日记里，记录了中共如何对待民主人士，还有一条决定中国乃至香港新闻界命运的会议讨论记录，即如何处理私营报纸问题。这个讨论记录显示当时旅港的民主人士也有份参与。

中共窃得政权已成定局，在这个趋势下，文人墨客，知识份子的取舍对自己下半生的命运很重要。中共也在有意统战这些人，于是才有了大规模的费尽周折的旅港民主人士北上运输的故事。中共也以此为豪大吹特吹，但是他们心里想的和大吹特吹是两码事。在改朝换代时刻，中共不忘管制它最有力的武器之一——舆论。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3221-4.jpg)图片来自搜狐
田牧著书《走出中国新闻审查禁地》云，中共非常注重舆论的作用，毛泽东很早就意识到“笔杆子”的重要性。他对历代王朝更替的研究意识到，天下争夺归根到底是人心的争夺，「得人心者得天下，失人心者失天下」。毛泽东说，搞革命靠“枪杆子”也要靠“笔杆子”，「笔走龙蛇经风雨，白纸黑字写人生」。在井冈山时期开始，毛就开始抓笔杆子了。 1939年10月毛泽东在《共产党人》刊词总结中共18年来的革命经验有三大法宝：统一战线，武装斗争，党的建设。统战不是武装占领不是武力征服，是理念的宣传。凭笔杆子宣传取悦人心，夺人心智，赢得社会同情和支持，枪杆子决战场输赢笔杆子影响人的心智。 “凡是要推翻一个政权总要先造成舆论，总要先做意识形态方面的工作”。综观中共的发展史也确实如此，中共至今还牢牢紧记毛泽东的笔杆子战术。郭文贵先生在谈论中共的特点时也经常说到，中共的舆论战是最主要的，凡是有什么事，中共首先就是舆论影响，舆论导向。为此它一直控制言论自由，控制舆论宣传机器。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3222-3.jpg)图片来自环球实报
1948年前后香港是远东唯一一个自由的港口，中共利用香港相对自由的环境作为解放中国的旗棋子，部署内地各游击队根据地与香港紧密联系，香港分局成为统领。 12月，管控宣传口一刻也不松懈，香港分局针对这个问题展开多次讨论，在中共统战报《华商报》上公开讨论，私人报纸的去留问题。各知识份子纷纷表态踊跃书见解，基本以新中国成立后来讨论。中共国代表着人民，代表着无产阶级，新中国是一切公有化，不允许有任何自由经济存在的，私营报纸当然也不能存在，即使民主人士有意见，但是他们最终还得接受现实。这是他们讨论的结果，就是把生米煮成熟饭。报章上也有不同观点的，认为不管是国营公营还是私营，只要符合新民主主义，新文化原则的应予以尊重和保护。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3223.jpg)图片来自搜狐
1949年4月27日，旅港民主人士还在北上过程中，《华商报》就抛出论题，《文汇报》的徐铸成，王芸生，赵超构，刘尊棋都参与讨论，关心新中国是否允许私人办报纸。这些人在中共过渡期的新闻政策中都精神都受到了各种打击。共产党的笔杆子革命，掀起一股风，国民党的后院天津和上海成为左翼文化运动的摇篮，涌现一批左翼文化文人、团体和报章杂志，现在新闻界改革这两个地方也成了首先收拾的地方。知识份子的踊跃表态和香港的“五区公投”类似，都是一番热情到最后遭无请打击。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3224-1.jpg)圖片來自360百科
1948年下半年中共中央，中宣部，新华总社总结在城市报纸宣传工作经验教训，并发出一系列指示。 8月15日，中宣部发布《关于城市党报方针的指示》，强调报纸为谁服务，10月2日，刘少奇提出办好新​​闻工作的四个条件。中共最重视的宣传之一就是新闻内容的统一性。 1949年1月6日，中共中央又发布《宣传约法三章，不要另提口号》、《勿擅自向外表示态度》两项指示。 10月10日，华北《人民日报》发表一篇报导当地灾情文章《全区人民团结斗争战胜灾害》，按照正常的思路应该是大赞领导抗灾情况，取得什么样的胜利成果才对。但是该文没有过多赞扬领导救灾工作，大篇幅报导实际灾情惨状，结果受到了中央强烈批评，实在顶不住压力了，该报不得不在11月8日登报检讨承认客观主义。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3225-1.jpg)圖片來自Gnews
中共在与国民党内战夺城过程中加强占领舆论阵地，打者「保护人民的言论出版自由和剥夺人民的言论出版自由》幌子，迅速接管原有新闻媒体，对国民党政府及其下属体系所有新闻机关一律接收，反对国民党的保护，中性的不禁止，自己继续办报，新解放区的新闻单位一律向当地政府登记。香港在英殖民地下，报业相对自由一些，这与中共的策略有关，在《中共权贵在香港的扩张》里已提过，毛泽东回答记者时说暂不急着收回香港的言论。中共要利用香港的自由为其服务。
