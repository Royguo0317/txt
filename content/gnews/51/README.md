04月12日，俄外交部长谢尔盖·拉夫罗夫在尤里·加加林太空飞行60周年纪念日的视频讲话中，呼吁就禁止在太空部署任何武器和使用武力进行谈判。该文本可在外交部网站上查阅。

将太空“用于创造性目的，造福全人类”的条件，他称之为“保证防止军备竞赛”。

“我们赞成就制定一项具有法律约束力的国际文书开始谈判，禁止在那里部署任何类型的武器，以及禁止使用武力或以武力相威胁。”拉夫罗夫说。

外长补充说，莫斯科相信，太空领域的建设性合作仍应是国际议程的关键领域之一。

去年夏天，俄罗斯外交部在评论美国的太空防御战略时针对俄罗斯和中共国的“威胁”，指责华盛顿的侵略行为旨在“确保美国在太空的军事优势”。

外交官称，太空对美国来说是一个军事行动的舞台，不仅是防御性的，也是“通过敌意利用外太空击败敌人”的军事舞台，而华盛顿仅对伙伴和盟友赋予支持作用。

五角大楼则指责俄罗斯和中共国将外太空“变成了一个战争区域”。
