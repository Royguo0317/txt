在周五，一个非政府组织联盟呼吁法国对四家跨国服装制造商进行调查，指控它们从中共国维吾尔族的强迫劳动中获利。投诉的对象是服装巨头Inditex，优衣库和SMCP以及鞋类制造商Skechers，后者指控他们在生产商品时隐瞒了强迫劳动和危害人类罪。

该申诉是由反腐败组织夏尔巴，道德运动非政府组织（清洁成衣运动的法国分支机构和欧洲维吾尔学院）以及一个在中共国新疆的一个难民营中关押的维吾尔族妇女提起的。

非政府组织在一份声明中说，该申诉旨在向法国当局提供证据，证明这些公司与强迫劳动有所牵连，他们说维吾尔人在中共国难民营中受到了迫害。

他们说，“面对经济全球化背景下的侵权行为，该申诉谴责了这些行为者有罪不受惩罚。”

他们指责这四家公司以及其他实体继续使用新疆地区生产的棉花来销售商品，因此是 “在该地区犯下严重罪行的同谋。”

声明说，这是在今后几个月将在欧洲国家提出的此类请愿书中的第一起。

人权组织认为，至少有100万维吾尔族人和其他大多数为穆斯林的少数民族被关押在新疆地区的难民营中。在该地区，中共国还被指控对妇女进行被迫绝育和强迫劳动。

美国说，对该地区的维吾尔人和其他穆斯林少数群体实施了“种族灭绝”，而中共否认了所有关于虐待行为的指控，并坚持其在新疆的政策对于打击暴力极端主义是必要的。
