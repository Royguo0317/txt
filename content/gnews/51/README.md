面对美国人民，拜登如此解释：

“习近平的核心原则是，中国必须统一和集权，这是他的政治思想。我跟他指出，任何一个美国总统如果不体现美国的价值观，他这个总统就不能持续下去。所以他在香港、在中国西部山区对维吾尔人做的事情，还有在台湾做的事情，试图通过强硬的手段结束一个中国的政策。我说的这些，他明白了。从文化上讲，每个国家都有不同的（行为）准则，他们的领导人也要遵守。”

拜登在这里没有明确表达他对习近平的说法（谓语动词丢失），但结合前后语句能够得出这样的结论：他认为习近平在对香港、新疆、台湾的做法符合中共国的文化和价值观，他本人对此表示充分的理解，正如他自己所认为“如果他不能遵守美国的价值观，就做不了美国总统。”
