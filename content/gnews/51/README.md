**纳瓦罗说川普只要听班农的就会赢**

川普2020年的选举不是民意的失败，是策略的失败，在以退为进的策略上被人出卖。纳瓦罗说不管川普是否参加2024年总统竞选，只要听班农的就会赢得选举。而班农背后就是爆料革命，郭先生和闫博士。川普曾专门安排纳瓦罗与闫博士会面，与会还有美国第一生物武器专家。但是由于川普内部信息不通透，听信了库什那谗言走到现在的局面。

班农2020年就在战斗室说，如果川普不能赢得选举，那一定是因为没有重视闫博士报告，只要宣布中共病毒是生物武器就必赢。纳瓦罗当前在福克斯讲出来令人震撼，验证了爆料革命的政治影响力。同时纳瓦罗在川普成立前总统办公室里，该办公室包含多数前白宫原班人马，说明纳瓦罗的讲话已经得到川普认可。同时纳瓦罗刻意提到班农的名字，就是释放一个强烈的信息，听取班农先生的就是听爆料革命的。

**中共最高院裁决农民工不属于法律意义上的实际施工人**

中共的流氓政策意味着农民工利益受到损失后没有权利上诉，断绝了农民工合法讨薪的一条路。可以看到中共已经黔驴技穷、彻底没有钱了，因为解决不了问题，只能解决提出问题的人，巨大的社会灾难正在酿成。

由于美中对抗，中共经济急速下滑，大量人口失业，劳动力富余，就通过压榨农民工阶级保资本家利益，与中共宣传的“社会主义”完全不符合。

其次如果中共推出数字货币，就要打压现金流动，而农民工是现金最大交易群体。所以中共可能埋了一个雷在这里，如果出现农民工薪资欠缺问题就为其发放一些数字货币，一来推行自己的数字货币政策，二来又能占据道德至高点。
