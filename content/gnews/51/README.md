英国已经开始接受其计划向符合条件的香港人发放英国海外签证的申请，随着可能成为数百万申请者中的第一批人的申请，北京方面感到不满，并再次大肆抨击。中共对英国挑战中共国的地缘政治力量感到非常愤怒，上周禁止持有英国护照的游客入境，周日他们抨击伦敦的决定是**“厚颜无耻的强盗逻辑”**。

中共指责伦敦违背了在香港移交过程中所作出的承诺，**具有讽刺意味的是，考虑到中共国本应该允许香港在“一国两制”原则下享受其民主自由，从法律上讲，这实际上是直到2047年。**

在英国开放申请后不久发表的一份声明中，一个内阁级的港澳事务办公室对此举提出了“强烈谴责”，指责伦敦试图将数百万香港人变成“二等公民”（有点像中共国对待维吾尔族人一样？）

本质上讲，北京重复了北京自己的指责和侮辱（辱骂），直接回击。

*该办公室表示，**这项政策相当于违反了两国在1984年签署的《中英联合声明》**，该声明为香港在1997年回归中国后的发展制定了基本规则。它声称：“英方没有信守诺言……甚至公然宣称这是出于对与香港历史关系和友谊的尊重，来美化其侵略和殖民的历史。” **“这是一个厚颜无耻强盗的逻辑。这是对中国主权的公然侮辱。我们坚决反对。”** 北京驻香港联络处也指责英国侵犯了中共国主权和国际法。*

英国早在（去年）7月份就首次宣布了这项计划。当时，许多受过良好教育和较富裕的工人表示，他们完全打算离开香港，可能永远不会回来。到目前为止，还不清楚有多少人报名了。许多香港人告诉《南华早报》，他们会等到2月23日，英国政府表示届时将开发一款智能手机应用程序。
