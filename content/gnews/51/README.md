继“中国人民不吃这一套”爆红中共国网络之后，又出现了“新疆棉花不吃这一套”的口号。

美东时间2021年3月23日，中共“共青团中央”在微博上曝出H&M集团一则声明，称该公司深切关注对于新疆强迫劳动和歧视少数民族宗教的指控，并表示不与位于新疆维吾尔自治区的任何服装制造厂合作，也不从该地区采购。北京时间当晚，中共官媒央视和新华社也纷纷发表评论文章，批评该品牌“实则吃中国的饭、砸中国的锅”，“一边造谣抵制新疆棉花，一边又想在中国赚钱，痴心妄想”，还发图文强调：“新疆棉花不吃这一套”。此举引发了世界各知名品牌禁用新疆“血棉花”的骨牌效应。

其实H&M的这份声明早在2020年10月就已刊登在公司网站上。中共喉舌现在将其翻出来大肆宣传，与日前34国对4名中共官员和1家实体进行制裁、跟进谴责有关。随后，中共对欧盟进行了所谓的“反制裁”，点名10名人员和4个实体。而欧盟及多个成员国也立即做出回应，欧洲议会取消了一个对《中欧全面投资协定》（CAI）的审议会，通常能够进入审议程序的议题基本上都是已经私下沟通完毕，只待议会审议通过。该协议曾由习近平亲自出马，被认为是习的“外交胜利”。此外，欧盟8国怒召本国的中共派驻大使表达抗议。

本次是中共主动挑起事端，其背后的原因值得玩味，结合当前的大环境，我们分析可能存在以下几方面原因：

首先，面对全球近50个主要发达国家对中共种族灭绝的制裁，中共不想惹怒美国和更多的大国，否则会给自己带来更大的麻烦。于是妄图利用反制裁H&M这一欧洲小国瑞典的一个不大不小的品牌，展示中共的强硬，并借此转移各国视线。

其次，中共一直奉行利益至上，赤裸裸地用利益交换道义。利用“14亿人”的巨大市场准入作为经济利益引诱西方国家，在政治上对中共妥协，这一招可以说被中共玩“神”了，从建政以来无往不利，这次中共还想祭出这一招：想发大财，就闭上你们的嘴，无论我们做什么你们都必须视而不见；想指责我们迫害人权，那就要准备牺牲大把的银子，承担被选民轰下台的风险。同时，也是借此手段分化欧美同盟，各个击破，最终瓦解西方国家同盟本次对中共的“围剿”。

再次，煽动中国民众的民族主义情绪，激化民族矛盾，将中共党内矛盾转嫁到国外，达到进一步稳固盗国贼家族专制集权的目的。经过众多中共官媒的一系列操作，引得大量小粉红义愤填膺，叫嚣要“坚决抵制”、“让H&M滚出中国”等等，并且网络上传出小粉红到耐克抗议，当众烧毁耐克鞋的视频。H&M代言人黄轩的工作室第一时间发表声明，与H&M“撇清”关系，强调“反对任何形式企图对国家及人权进行抹黑造谣的行为”。到笔者撰稿时为止，已有近百位艺人或被CCP胁迫、或出于自愿与各大品牌解约。

最后，自去年底美国大选开始，中共就用尽各种手段，妄图改变灭亡的命运。但是经过3个多月的运作，结果让中共掌权者感到了绝望：拜登政府不听吆喝了，之前被深度蓝金黄的欧洲诸国也渐行渐远，“市场准入”牌也不管用了。这次西方崇尚自由、民主的各发达国家对于和中共脱钩是“王八吃秤砣——铁了心”。中共正是看到了一点，感到回天乏术，那么与其“被脱钩”不如我主动脱钩，又可以借此对内洗脑。

与H&M同处BCI（良好棉花倡议协会）标准之下的企业多达2000多家，包括我们耳熟能详的优衣库、阿迪达斯、耐克、无印良品、Zara、宜家、匡威、迪卡侬、新百伦等等这些更大的品牌。中共战狼外交表面煽动民众抵制国外品牌企业，其实只想在互联网上演演戏，如果无脑粉红民众真的举牌现场抵制，反而会遭到当地政府和公安的阻止和逮捕。因为官员们心里清楚，这样一股脑的抵制国外企业，对中共经济和庞大的维稳支出统治将是灭顶之灾。下面的Twitter视频很好地说明了中共既想玩火演戏又害怕真着火的矛盾心态。
