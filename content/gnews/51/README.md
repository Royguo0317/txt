美国国务卿安东尼-布林肯周三表示，他认为中共国官员在中共国新疆省对维吾尔人进行种族灭绝的观点没有改变。

布林肯周三在新闻发布会上说：“我的判断仍然是中共对维吾尔人进行了种族灭绝，这一点没有改变。”

关于目前的美中双边关系，他说：”美中关系可以说是未来世界上最重要的关系，这不是什么秘密。它将塑造我们所有人生活的未来。而且这个关系越来越具有对抗性，它既有竞争的方面，也有合作的方面。”

布林肯补充说， 与中共国在气候变化方面进行合作符合美国的利益，尽管这符合两国之间关注领域的更大范围。

周三早些时候，美国总统乔-拜登提名的联合国大使琳达-托马斯-格林菲尔德表示，国务院正在审查将中共国对待新疆维吾尔族和其他少数民族的行为定为 “种族灭绝”的这项认定。

据斯普特尼克报道，在此之前，在确认听证会上，布林肯表示认同前国务卿迈克-蓬佩奥对中共参与新疆维吾尔人种族灭绝的认定。
