这与上海警方，在新闻发布会中所说没有殴打杨佳的说法，截然相反。事实上，匪共警方的话，一向是与事实相反的，很多案例已经证明了这个事实。

事后，杨佳多次投诉，并亲自找上海警方讨要说法，上海警方就是不肯承认辱骂殴打了杨佳。
