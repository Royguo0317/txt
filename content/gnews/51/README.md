3月19日(周五)，中共驻法大使卢沙野在推特上发布了一条消息，称战略研究基金会(FRS)成员安托万·邦达兹(Antoine Bondaz)是“小流氓”，中共驻法大使公开辱骂法国学者令法国举国震惊。

据法国广播公司报道，中共驻法大使卢沙野2月份致函法国参议院，要求法国取消访台代表团行程遭到拒绝，法国学者、中国问题专家邦达兹指责卢沙野这封信是对法国民主制度的干预。中共驻法使馆公开在推特上辱骂邦达兹是“小流氓”。中共如此不顾外交尊严毫无礼节的做法引起各界讨伐。欧洲议员一致支持邦达兹，批评卢沙野作为外交官有损国家形象。

马克龙的前部长、欧洲议会议员Nathalie Loiseau在推文中写道，“很少见到有外交官对自己国家的形象造成如此大的伤害。粗暴、粗鲁，这就是您所展示的中（共）国。”巴黎政治学院的中国问题专家白夏教授分析指出，卢沙野大使攻击邦达兹的原因可能与他新开设的台湾问题研究项目有关。

今年1月，FRS推出了一个名为“台湾安全与外交”的新项目。3月21日下午，中共大使馆发表了一份题为“关于言论自由的民主讨论”的长篇公报，为中共大使侮辱性的言辞辩护。文章以一种诽谤性的方式将邦达兹描述为“意识形态巨魔”，指责邦达兹“与台湾当局的可疑关系”。中共官媒《环球时报》随后发声强调，“如果法国议会代表团访问台湾，肯定会损害中法关系。”

自从法国对新疆维吾尔族的大规模拘禁问题表达了坚定的立场以来，中法关系已经紧张起来。
