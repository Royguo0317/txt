## 百姓的噩梦—中共针对宅基地的一系列把戏

![]()![](https://gnews.org/wp-content/uploads/2021/04/f968c7f1ee15a4eb.jpg)**图片: 昆仑竹**



从去年年底开始，中共在河南省进行宅基地改革试点，宅基地开始迈向有偿使用。中共规定一户农民只允许有一个宅基地，多余的会被拆掉，超过167平米以上的要缴费，而且是每年都缴费。缴费标准如下：

(1)超出面积1-50平方米部分，按每年2.5元/平方米计费；

(2)超出面积51-100平方米部分，按每年5元/平方米计费；

(3)超出面积101-200平方米部分，按每年10元/平方米计费；

(4)超出面积201-300平方米部分，按每年20元/平方米计费；

对大于300平方米以上的宅基地，不鼓励执行有偿使用政策，建议村集体经济组织或村民委员会组织收回。说白了就是，平米数越多缴费越高，300平米以上的直接收归国有。

对于中国北方来说，一年从一户农民身上轻松随意就能收割2000元，因为北方都有很大的院落。一家如果盖3间房，按这个标准，很容易超标，如果每平米按照5元算，中共每年从一户农民身上就增加了近500元财政收入。而南方虽然院落小，如果中共想多榨些油水，只要提高单价，每户也能收一千元，甚至更高。

在中共国，宅基地就是农民们全部的希望。以前他们一直认为宅基地是自己享受的一种福利，可以免费搞农业生产。曾有一度，很多城市人都羡慕农民的宅基地，因为既可以居住，又可以自给自足。感觉就像传说中神仙般的生活。谁也没想到，忽然刮来一阵邪风，打碎了所有人的美梦。

其实，一直以来中共都是牺牲农民兄弟的利益来扶植工业的，因此造成了农民的日益贫困。谁曾想到，即使这样也没逃过再次被割韭菜的命运。现在不但触角再次伸向5亿农民，而且还是每年都要缴费。试问，中共为农民的切身利益想过吗？这会不会激起更大的民怨？农民们本来的收入就极低，一人每年不到一千元，如果再徒增这些费用，搞不好会使他们的生活陷入困境。那些本身已在困境中的贫困户又该怎么办？真是不敢想呀。这是要上演现实版的民不聊生啊！为了稳固他们的政权，他们脑子里只有如何增加财政收入，哪管百姓的死活。只是可怜了这些农民兄弟们。

中共的这些政策，断了很多农民兄弟们搞副业的路，而且即使搞了副业，也不知收入能否抵得上国家所征收的费用。以前引以为傲的大宅基地现在变成了负资产。如果按规定上缴“国家”，不但彻底断了副业的路，以前置办的农机器械也无处安放。如果不上缴，又怎么交得起每年中共征收的费用。真是两难呀。更恶劣的是，自家在宅基地上盖起的房子，很多已经被强拆了，有些没被拆的也是有今天没明天，过得提心吊胆的。要知道那都是他们多年没日没夜地打工生涯换来的血汗钱，或者东拼西凑一砖一瓦盖起来的小窝，就这么说拆就拆了。这其中的心酸、绝望只有当事人最清楚。可是，身在在中共铁拳的统治下，又能怎样？他们干得过中共政权吗？别说他们了，城市又能好到哪里去？一套城市商品房挖空几代人也就只能住70年，有些地方才只能住50年。这样还嫌不够，中共这几年一直在很多城市搞宅基地出让。目前已有9个城市集中出让宅基地，而且有愈演愈烈之势。这所有的一切，真是让人痛心！

中共如此原形毕露，而且不管不顾、明目张胆地抢劫百姓。看来真是财政出大问题了。可以预见，如此的恶性循环下去，中共离真正的崩溃也不远了。

文贵先曾在一次直播中说：“共产党灭亡后，要把所有土地、房产无偿地就地还给人民。所有老百姓欠银行的钱都不能还！不信让我们干几年试试！” 为了这一美好的目标，我们会继续追随爆料革命，努力行动着，期待着这一天的早日到来！我也想对那些挣扎在贫困线上的兄弟姐妹们说：现在是黎明前的黑暗，你们的苦日子快到头了。一定要互帮互助，坚持下去。等待天亮，天很快会亮的。
