## 【军事快讯】英国会议员呼吁禁止中俄投资英国国防供应链
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/908674/)

作者：纽约香草山  人间世 校对/发稿：飞虹

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3.jpg)



英国国会的国防专责委员会（Defence Select Committee）2月14日表示，应该禁止中俄等敌对国家对英国国防供应链的投资。

![]()![](https://gnews.org/wp-content/uploads/2021/02/图片17-1.png)图片来源：《金融时报》

针对国防供应链在COVID-19（中共病毒）大流行期间的脆弱性，英国会的国防专责委员会做出了专项调查，并在周日发布了一份报告（Foreign Involvement in the Defence Supply Chain），报告显示来自中俄等国家的投资对英国的国家安全构成潜在威胁。英国的国防和外交政策审查将于3月份进行，国会此时发布这份报告，也是向政府施压，要求政府对中俄等国采取更强硬的立场。

调查结果表明，Covid-19大流行使许多中小型企业面临很大困难，如果政府不能给予相应帮助，他们可能不得不接受被外国收购的命运。 据《金融时报》报道，保守派议员、下议院国防小组委员会主席理查德·德拉克斯说：“必须禁止包括俄罗斯和中（共）国在内的所有不在批准名单之内的国家对国防供应链的投资，任何英国公司都不应该面临要么破产、要么接受敌对国投资的两难选择。”

议会调查显示，英国的国防供应链对于中共等敌对势力几乎处于不设防的状态。过去10年，有6家将国防列为关键业务的英国公司被中国投资者收购。其中一家公司eXception PCB生产用于F-35战斗机电路板的塑料部件。英国国防部还从中（共）国购买二手设备，比如购买二手737飞机，然后将其改装成E-7楔尾喷气式飞机，用于英国的机载预警系统。这些做法着实让外界大跌眼镜。中共对于西方社会的渗透之深，于此可见一斑。但据《金融时报》透露出的消息，英国政府官员比如国防大臣本·华莱士（Ben Wallace）仍然对明确指出中共对英国国家安全构成威胁持谨慎态度。

中共在香港、新疆问题上的做法正日益引起英国社会的强烈反感和高度警惕，双方近期摩擦不断。英国国会的这一报告想必又将令中共大动肝火。我们期待越来越多的西方国家站出来，直面中共威胁，并做出强有力的反击。
