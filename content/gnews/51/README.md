#### 欧洲联盟委员会周一（4月12日）表示，欧盟无法帮助希望成为欧盟成员国的黑山偿还其积欠中共国的债务，但可以帮助它完成一项开始时用北京的巨额贷款建造的高速公路的计划。

这个巴尔干小国今年开始,将艰难地偿还中共国提供的10亿欧元贷款。2014年，黑山从中共国获得这笔贷款，来建造巴尔-博尔加里高速公路的一期工程。此高速公路工程完成后将把亚得里亚海沿岸的巴尔港和内陆邻国塞尔维亚连接起来。
