## 独家—玛莎·布莱克本：中国的“谎言”意在转移外界对其在疫情角色上的注意力

[!\[\]()!\[\](https://spark.adobe.com/page/anPTseabmV8sF/images/220b947d-f0a6-4060-b72b-45bf8a25eace.png?asset_id=275f948e-2be4-412f-b26d-079317dd3f72&amp;img_etag=%227a70ecc5b3381ba7cfeb5d3c642cfb48%22&amp;size=2560)](https://spark.adobe.com/page/anPTseabmV8sF/images/220b947d-f0a6-4060-b72b-45bf8a25eace.png?asset_id=275f948e-2be4-412f-b26d-079317dd3f72&amp;img_etag=%227a70ecc5b3381ba7cfeb5d3c642cfb48%22&amp;size=1024)

参议员（共和党-田纳西州）玛莎·布莱克本（Marsha Blackburn）周二对布莱特巴特新闻说，她认为中共正试图通过散布“谣言”并对她发起骚扰攻击，转移其在中共冠状病毒大流行中的作用。

“中共已经开始散布谣言，企图掩盖他们在这场大流行中的致命角色。我将永远不会停止与中共所犯下的罪恶作斗争。”这位参议员说。

布莱克本的这番话旨在回应陈卫华对她的攻击，陈卫华是中共国宣传机构《中国日报》（欧盟）分社社长，陈卫华周二称她为“虚伪的政客”。

这不是陈卫华的推特账户第一次对布莱克本进行人身攻击。2020年12月3日，布莱克本评论了她认为中共国有“五千年的作弊和偷窃的历史”，引发了陈的回应，称她为“婊子”。

“这只是偏执的种族主义仇外心理。没有其它方法来描述定性一个国家几千年来都有这样的负面品质，尤其它还代表了全世界五分之一的人口[原文如此]。”乔治城的教授唐·莫伊尼汉（Don Moynihan）发声道。

在这位参议员带头质疑中共与美国国家篮球协会（NBA）之间的关系后，中共国支持下的对布莱克本的攻击升级了。

“我曾写信给[NBA总裁]亚当·西尔弗，因为NBA与阿里巴巴关系密切，在中共国制造的产品使他们在经济上受益数千万或数亿美元，”布莱克本去年告诉布莱特巴特新闻，“通过这种商业关系以及他们的转播权和授权商品的销售。他们应该，如果他们要如此关注美国发生了什么，为什么不关注在中共国发生了什么？”

许多NBA明星都对美国所谓的歧视行为进行了公开表态，但面对继川普政府之后的拜登政府认定中共国在新疆西部针对穆斯林少数民族的“种族灭绝”行为时，他们一直保持沉默。在遭到人权活动人士数月的谴责后，NBA最近关闭了在新疆首府乌鲁木齐的“训练营”。

上周，布莱克本再次向NBA施压，因为NBA试图与中共国国家电视台达成利润丰厚的电视转播协议，而拒绝“对中国奴隶劳工营的问题发表看法，因为害怕会对与中共国政府价值数十亿美元的交易带来压力”。

《中国日报》的编辑陈卫华也曾攻击过其他挑战共产党的参议员，不过通常只是在推特上以取“绰号”的形式出现。在2020年，陈卫华在推特上对参议员乔希·霍利（Josh Hawley）（共和党-密苏里州）就进行了攻击，称他是“无知的政客”。霍利的回应是宣称他喜欢“被#中共国官方媒体侮辱”。

陈卫华还通过转推希拉里·克林顿（Hillary Clinton）、参议员查克·舒默（Chuck Schumer）（民主党-纽约州）、《华盛顿邮报》专栏作家詹妮弗·鲁宾（Jennifer Rubin），表达了对前总统唐纳德·川普（Donald Trump）的许多政治对手的支持。
