## 美国与其他13个国家一起，批评世卫组织在中共国新冠病毒报告中缺乏透明度

**要点**

- 美国和其他13个国家表示，世卫组织及其成员国必须再次对“开放、透明和及时性”作出承诺。
- 白宫新闻秘书珍·普萨基（Jen Psaki）告诉记者，拜登政府仍在审查世卫组织的报告。
- 普萨基说，这些调查结果给出了一份“部分且不完整的结论”。

[!\[\]()!\[\](https://spark.adobe.com/page/k8Gz9Jan84z1X/images/2d094bd2-ecaa-4217-a781-6682fbfd234f.png?asset_id=fd3688da-ab0a-452f-a8c3-e6fd11897cb8&amp;img_etag=%22c2b20fdeb3924807ec9717d4ef01a7f8%22&amp;size=2560)](https://spark.adobe.com/page/k8Gz9Jan84z1X/images/2d094bd2-ecaa-4217-a781-6682fbfd234f.png?asset_id=fd3688da-ab0a-452f-a8c3-e6fd11897cb8&amp;img_etag=%22c2b20fdeb3924807ec9717d4ef01a7f8%22&amp;size=1024)这张图片拍摄于2020年2月17日，显示了在中国中部湖北省武汉市一个改建成医院的展览中心工作的医务人员。 STR |法新社通过盖蒂图片社
华盛顿——美国周二与其他13个国家签署了一份联合声明，批评世卫组织这份期待已久的有关新冠病毒起源的报告。

澳大利亚、加拿大、捷克共和国、丹麦、爱沙尼亚、以色列、日本、拉脱维亚、立陶宛、挪威、韩国、斯洛文尼亚、英国和美国政府在一份联合声明中写道：该报告“严重拖延，并且无法获得完整的原始数据和样本”。

根据联合声明表示：“在一种具有潜在大流行的未知病原体严重爆发的情况下，对其起源进行快速、独立、专家主导和不受阻碍的评估是至关重要的，它可以使我们的人民、我们的公共卫生机构、我们的企业以及我们的政府更好地做好准备，以成功应对这种大爆发并预防未来的大流行。”

“展望未来，世卫组织及其成员国现在必须对开放、透明和及时性作出新的承诺。”该小组补充道。

于周二发表的这份120页的世卫组织报告由国际科学家团队撰写，尽管它有助于促进科学界对席卷全球的致命病毒的理解，但仍缺乏全面评估。

世卫组织总干事谭德塞周二在一次新闻发布会上说：“我们还没有找到病毒的来源，我们必须继续遵循科学，不遗余力地去做。”

“找到病毒的起源需要时间，我们有责任为世界找到病毒的来源，这样我们才能共同采取措施，减少再次发生这种情况的风险。任何一次研究之行都不可能提供所有的答案。”他补充道。

在白宫，新闻秘书珍·普萨基告诉记者，拜登政府仍在审查世卫组织的报告，并补充说，这些调查结果给出了一份“部分且不完整的结论”。

“报告缺乏关键的数据、信息和获取途径，这代表了一个局部和不完整的画面。”普萨基说。“我们认为，这一过程的第二个阶段应该由国际独立专家来领导，他们应该可以不受限制地获取数据。”她补充道。

当被问及中共国参与世卫组织的报告时，普萨基抨击北京缺乏透明度，该报告至少包括17名专家。

“嗯，他们一直不透明，没有提供基础数据，这当然不能算是合作。”她说。
