法新社3月25日报道：中共病毒疫苗临床事件评估专家委员会今晚（24日）见记者，共同召集人孔繁毅表示，委员会在今日会议讨论9宗接种中共病毒疫苗后出现“贝尔面瘫”的个案，9名事主均是男性，年龄介乎39至86岁，其中8人接种科兴疫苗、一人接种复必泰疫苗。专家委员会继续留意相关个案，若发现不寻常信号，会建议当局采取适当措施，例如要求药厂适当更改产品说明书。

评论：

闫博士早在去年就多次警告世界，如果不搞清楚中共病毒溯源及其基因序列，是绝对不可能生产出有效的疫苗，甚至还会生产出有严重副作用的疫苗。香港在大面积接种中共病毒疫苗后，曝光多例因疫苗出现严重不良反应，甚至是死亡的案例。这再次验证爆料革命闫博士对中共病毒真相揭露的准确性，当全世界都认识到中共病毒疫苗无效时，他们才会采取实际行动要求中共政府开放实验室接受调查，公开病毒真相。
