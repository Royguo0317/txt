**西班牙2021年4月21日电/西喜社——**
![]()![](https://gnews.org/wp-content/uploads/2021/04/international-5319462_1920.jpg)
本周三，第二十七届伊比利亚-美洲国家元首和政府首脑会议在安道尔召开，会议的主题是 “创新促进可持续发展–2030年目标：伊比利亚-美洲面临冠状病毒的挑战”，大部分美洲国家的领导人将以远程方式参与。

在那些已经确认参加虚拟会议的人中，有两个人引起了特别注意：古巴总统米格尔-迪亚斯-卡内尔（Miguel Díaz-Canel），他刚刚被任命为古巴共产党的第一书记；以及查维斯主义的委内瑞拉领导人尼古拉斯-马杜罗。

令人震惊的是，古巴总统已经20年没有参加这一活动了。直到今年，古巴官方媒体证实–三天前–迪亚斯-卡内尔将会出席峰会。确切的说，古巴一直没有以高级别的代表参加会议，尽管低级别的代表并没有缺席过。

一些分析家看来，促使古巴政府恢复其在最高层的存在的原因是 “古巴经济的情况，它处于非常复杂的状态，现在需要其他盟友”。他补充说，“这显然解释了迪亚斯-卡内尔的立场，以及在古巴共产党第八次代表大会之后试图传达一种正常和开放的感觉”。

峰会的另一个意外是尼古拉斯-马杜罗的实际参与，这引起了委内瑞拉反对派的极大不安，他们期待临时总统胡安-瓜伊多代替他出席。据埃菲社报道，巴西总统贾伊尔-博尔索纳罗非常不高兴，正是因为组织国安道尔决定邀请查韦斯塔领导人，他才会拒绝参加这次峰会。

**简评：**

**消灭中共是全球灭共大势不容置疑的主线，而支线上“小中共”们的一举一动也同样值得关注。古巴也算是老牌“社会主义国家了”，与美国的迈阿密隔海相望。因为地缘关系，古巴也是社会主义阵营安插在美国后院的一颗钉子。“古巴导弹危机”差点引起世界核大战。在中南美洲，古巴和委内瑞拉互相支撑，虽然不是灭共的主线，但也对中共在美洲的布局有着重要的支撑作用。**

**近年来，古巴发生着一些改变，自菲德尔-卡斯特罗逝世后，其弟劳尔-卡斯特罗接位。2年前，指定米格尔-迪亚斯-卡内尔为接班人。前不久，劳尔-卡斯特罗以健康原因辞去了古巴党中央总书记的职务，这点比较耐人寻味，劳尔据称患有癌症，但在接受采访时，身体状况看上去尚可。**

**米格尔-迪亚斯-卡内尔被称为卡斯特罗的“马屁精”， 上台后放出了一些改革开放的信号，并出人意料的宣布参加伊比利亚-美洲国家峰会。国际舆论普遍认为，因为古巴国内糟糕的经济状况，古巴需要重返国际社会，以获得更多援助，也有可能在后卡斯特罗时代，古巴会慢慢改变体制。**

**“小中共国”的政治体制是否会改变，还需要继续观察。但在这全球联合灭共的关键时刻，每一份支持中共的力量的削弱，都是值得高兴的。况且，古巴如果能够转向，将改变中南美洲的地缘状态，改善美国的地缘环境。这一消一涨，能对中共造成不小的打击，还是值得关注的。**

**我们也同样可以看出，搞共产主义，搞极权主义，都会给经济带来灾难，因为它是反自然规律的。最终也会因为“钱”的问题，而难以为继。中共国的社会主义其实只是一件外衣，以公有制来剥夺大部分人拥有财产的权力，而将资源集中到少数家族手里，它的脆弱性在于一旦这些家族的资产倒了，整个国家的经济就倒了。从这里可以看出“以钱灭共”的重要性和前瞻性。**
