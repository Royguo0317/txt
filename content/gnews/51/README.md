即使在那些受冠状病毒轻度影响的人中，许多人抱怨疲劳或“脑雾”，这种混乱的感觉使他们难以专心或工作。然后是消失或改变的味觉和气味、听觉异常、消化不良、失眠或呼吸困难。广泛且令人不安的症状表明，*Sars-CoV 2*感染不仅影响呼吸道，还影响神经系统。

博洛尼亚大学神经病学教授彼得罗·科特利（*Pietro Cortelli*）解释说，关于细胞因子风暴的讨论太多了，不要忘记炎症是由中枢神经系统控制的，对神经系统的损害也会干扰身体控制疾病的机制，我们对神经病学和免疫学之间的关系知之甚少，但是正是这些过程解释了为什么不同的生物体对感染的反应如此不同。

提示神经系统受累的最初症状是气味和味觉的改变，但最常见的症状可能是头痛，有时甚至比发烧更严重，科特利 （*Cortelli*）解释说:所谓的“脑雾”也非常普遍，一种或多或少的严重困惑，形式从持续的疲劳，记忆力和注意力不集中到真正的认知迷失，甚至是严重的昏迷,这可能表现为脑病的症状，可能还具有不同严重程度的脑功能受损或脑炎（即大脑发炎）。

实际上，我们甚至不知道该病毒在什么情况下到达大脑，因为疾病以非常可变的形式发生，并且还因为我们不确定病毒是否位于这些细胞中，因为它可能通过其他方式（例如通过血液）到达中枢神经系统。

此外，要确定这些疾病在多大程度上取决于病毒的直接作用，以及在多大程度上取决于该疾病的临床表现（例如呼吸衰竭或凝血问题），并不容易。佩齐尼（*Pezzini*）强调说，试图了解这种疾病是否会对神经系统产生影响，例如是否可能由于感染而导致痴呆或吉兰·巴雷综合症（一种表现为急性肌肉无力的神经疾病）的病例？病毒感染引起神经系统疾病并不罕见。

意大利的第一批数据似乎表明，已经患有神经系统疾病（包括中风）的患者比那么不体弱的受试者具有更高的死亡率和致残率。另一方面，要研究中风与感染之间的关系更加困难：在伦巴第的多中心研究中显示新冠病毒患者中风的严重程度会更高，这可能与病毒触发的凝血改变或异常激活有关。

#### 简评：

**之前有新英格兰研究表明，神经损伤不是归因于*Sars-Cov-2*的直接作用，而是归因于血管损伤引起的感染和炎症；丹麦发表在《循环》上的一项研究证实，新冠病患者中风和心脏病发作的发病率分别高10到5倍。全世界的专家和学者从他们的学术领域都在证明着这个病毒是极不寻常的，点点滴滴的线索总有一天会汇集到最终的那个点去，而中共国想掩盖和抹杀新冠病毒的真相又怎么会让它得逞。**
