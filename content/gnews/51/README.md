**德国总理默克尔执政4界将近16年。 2021年大选后她将离任，在德国的历史上能够执政16年的只有Helmut Kohl（赫尔穆特 科尔）。 默克尔能够4界当选， 说明德国的大部分民众对她的执政能力和效果是赞赏并支持的。 在她任期的最后一届德国经历了各种危机， 难民危机， 英国脱欧等等。 但是对她来说的滑铁卢可以说是病毒大流行的抗疫政策， 封城解封再封城再解封。 民众已经失去了耐心， 到了2021年3月3日， 联邦，州的大会最后决定的阶段性解封政策更加引起了各方的强烈争议和质疑。 这时候不光是政党政客们就议题的争议， 也包括了普通民众， 联邦州和州之间的也产生了不和谐。**
