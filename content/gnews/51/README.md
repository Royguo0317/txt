04月12日，中右翼“CREO运动”的前银行家吉勒莫·拉索（Guillermo Lasso）在厄瓜多尔总统选举中获胜。这是《El Universo报》的报道。

据该报报道，在处理了超过93%的选票后，拉索赢得了52.51%的选票，而他的对手来自中左翼联盟“希望联盟”的安德烈·阿劳斯（Andres Araus）获得了47.49%选民的支持。

“民主已经胜利了。你用你的选择权，你用它选择了一个新世界，不同于厄瓜多尔过去14年的世界。”拉索说。

阿劳斯承认失败，并宣布将祝贺对手获胜。

在厄瓜多尔，进行了第二轮投票。第一轮由安德烈斯·阿劳斯获胜。他当时赢得了近33%的选票。吉列尔莫·拉索在第一轮竞选中赢得了约20%，领先多民族团结运动帕查库蒂克的雅库·佩雷斯（Yaku Perez）0.5个百分点。

自2017年以来，厄瓜多尔一直由左派列宁·莫雷诺（Lenin Moreno）领导。
