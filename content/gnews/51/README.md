二、美参议院外交委通过灭共战略竞争法案

法新社4月21日报道[\[1\]](https://www.rfi.fr/cn/%E4%B8%93%E6%A0%8F%E6%A3%80%E7%B4%A2/%E8%A6%81%E9%97%BB%E5%88%86%E6%9E%90/20210421-%E7%BE%8E%E5%9B%BD%E5%8F%82%E9%99%A2%E5%A4%96%E5%A7%94%E4%BC%9A%E9%AB%98%E7%A5%A8%E9%80%9A%E8%BF%87%E4%B8%BA%E4%B8%AD%E5%9B%BD%E9%87%8F%E8%BA%AB%E5%AE%9A%E5%88%B6%E7%9A%84-%E6%88%98%E7%95%A5%E7%AB%9E%E4%BA%89%E6%B3%95%E6%A1%88)：美国参议院外交委员会4月22日高票通过跨党派议案“战略竞争法”，据国会15日公布的草案内容，该法案长达281页，共有5大章节，针对掠夺性经济作为、恶意影响力攻击、数字威权、军事扩张、对台野心、压迫香港与新疆等多项北京的问题行径提出了应对措施。

在美国参议院外委会22日会议上，民主党籍参院外委会主席梅南德兹（Robert Menendez）表示：来自中国的挑战在规模、范围与急迫性上皆是史无前例的，需要美国具备全面竞争性的政策和战略来因对。该法案集结了空前的跨党派努力，力图动用美国的所有经济、外交工具，来制定新的印太战略，让美国能真正面对中共国带来的国安、经济挑战。
![]()![](https://gnews.org/wp-content/uploads/2021/04/美国民主党籍参院外委会主席梅南德兹.jpg)[美国民主党籍参院外委会主席梅南德兹](https://s.rfi.fr/media/display/8f1c03d2-1677-11ea-8eea-005056a99247/w:980/p:16x9/468fc00866f34d6652139171940bf58aebb374ed.webp)
路德社评论：

1、《战略竞争法案》以高票获得美国参议院外交委员会通过，之后参众两院将对该法案进行投票表决，当拜登总统签字后就正式生效。拜登总统在小布什政府时期曾任外交委员会主席，他对美国的流程制度非常熟悉，拜登政府正在为全面推动灭共补流程。

2、《战略竞争法案》针对中共国多项问题提出了应对措施，其中明确规定法案生效后的180天内给出中共病毒最有可能的来源，必须制定有关中共病毒来源、武毒所等所有问题报告的时间表，该法案要求调查有关武汉病毒研究所的所有信息，包括其他相关实验室的研究信息，该法案已经罗列出病毒调查报告的大纲，这才是该法案推动灭共的最重要部分，**路德社爆料美国参议院将成立病毒调查委员会，Sellin博士将被选入该委员会中。**

3、该法案计划每年拨款3亿美元针对中共的网络水军，每年拨款1亿美元用于应对中共媒体战，美国已经开始针对中共发起的媒体超限战进行反击。

4、该法案是两党达成共识的法案，这也反应出美国的转向，以及对中共进行全面定义，此后就是颁布各项灭共政策。中美建交时，美国帮助中共国加入WTO，让中共国获得资金援助，促进与中共国的研究合作，开放留学教育，允许转让尖端科技，美国这么做的目的就希望中共国经济发展起来后可以推动内部民主和法治，让中共国参与到美国建立的国际秩序中。习神近几年通过军事扩张、经济贿赂等蓝金黄模式试图统治全球，计划彻底推翻美国建立起的国际秩序，这让中共国成为美国的敌对国。

5、该法案在定义中共国时，就将中共病毒纳入其中，说中共利用病毒破坏香港的自治，持续对东海和南海进行侵略，加剧与中共之间的紧张关系，隐瞒病毒真相，制造混乱，所以这份灭共法案反映了两党的意志，在接下来的投票审议中不会遇到任何阻碍。

6、共和党建制派参议员米特‧罗姆尼就该法案提出了一项修正案，将对北京奥运会实施美国外交抵制。罗姆尼说，一个国家在实施种族灭绝的同时被允许举办奥运会，这“令人震惊和愤怒”。[\[2\]](https://www.epochtimes.com/gb/21/4/21/n12896380.htm/amp)这就是美国进行法案审议的规则，美国所推动的两党政治虽然有钟摆效应，但并不是你死我活的结果，两党只是表面上用你死我活的态度去压迫对方妥协，从对方身上争取利益，一旦获得利益就会出现妥协，所以川普总统和拜登总统虽然在大选前互相批评职责，当大选结束后，双方又重新回归到各自岗位做事。

三、白宫证实中美领导人将不会进行双边会谈

大纪元4月22日报道[\[3\]](https://www.epochtimes.com/gb/21/4/21/n12896403.htm)：白宫周三证实，中共国家主席习近平将参加周四与周五（22日至23日）美国主办的领导人气候峰会，但总统拜登不会与习近平进行双边会谈。
![]()![](https://gnews.org/wp-content/uploads/2021/04/拜登总统.jpg)[拜登总统](https://i.epochtimes.com/assets/uploads/2021/03/GettyImages-1306585835-600x400.jpg)
路德社评论：

1、此次气候峰会将详细商讨具体行动方案，美国在此次气候峰会上将拉着全世界再次向中共施压，若中共仍然不遵守承诺，那留给中共的最后一扇门将被关闭。

2、文贵先生曾说，气候问题就像跟临死前的死刑犯谈论理想和哲学问题。拜登政府总是用软硬兼施的策略对付中共国，美国一边继续往灭共的方向推进，一边又要做到斗而不破，防止中共狗急跳墙，等带时机成熟时，美国再对中共发起毙命一击。

3、拜登和习近平在此次峰会上将不会举行双边会谈，拜登总统一方面想与习保持距离，另一方面，双方其实也没有什么好谈的，因为双方在关键问题上已经不可能达成共识。中共让王岐山出席博鳌论坛，换来的却是象征性的表态，可见中共手里可出的牌已经不多了。
