**1.美国石油因极寒天气减产三分之一**

寒流之前美国日产约1100万桶石油，目前全国原油日产量下降了至少350万桶，仅德克萨斯州二叠纪盆地（美国最大的油田）的产量就暴跌了65％。由于极低的温度导致井口和地面管道中的油和气液的冻结，德克萨斯州许多大型工厂已经不得不关闭，油田生产陷入困境。油价已升至今年的最高水平。

**2.2020年中共国取代美国成为欧盟第一大贸易伙伴**

欧盟统计局（Eurostat）周一发布的数据显示，中共国在2020年超过美国，成为欧盟第一大贸易伙伴。报告称，2020年欧盟对中共国的出口增长了2.2％，从中共国的进口增长了5.6％。相比之下，欧盟对美国的出口下降了8.2％，从美国的进口下降了13.2％。英国在2020年成为欧盟第三大贸易伙伴。欧盟在12月达成了一项投资协议，旨在有利于欧洲公司在中共国的经营，但是该协议尚未得到欧洲立法者的批准。

**3.2020年美国航空客运量下降60.1％**

华盛顿（路透社）- 美国运输部周二表示，受COVID-19的影响，2020年航空旅客总数为3.68亿，比2019年的9.226亿下降60.1％，是自1984年以来的最低水平。2020年美国国内航空旅行减少了58.7％，国际旅行减少了70.4％。美国9家最大的航空公司去年的税前亏损高达460亿美元，而且预计旅客数量在2023年或2024年之前都不太可能回到以往的水平。

**4.美国一天然气中心价格飙升至999美元，比一周前上涨24,000％**

(彭博社）- 周二俄克拉荷马州Oneok天然气运输中心两份第二天交货合约的价格为999美元/百万英热，而一周前的价格仅为4.19美元。卡尔加里王朝电力公司的交易员戴维·霍伊（David Hoy）表示，作为纽约期货基准的路易斯安那州Henry Hub天然气期货今天的交易价格约为19美元。一周前现货价格仅为3.24美元。

由于严寒天气引发停电，并导致液体在管道内结冰，迫使油井和加工厂停产。美国本土48州的天然气产量已暴跌至三年来的最低点。随着供暖需求飙升，冬季结束前库存可能会一直趋紧。

[https://www.bloomberg.com/news/articles/2021-02-16/gas-at-one-u-s-hub-is-trading-at-999-last-week-it-was-at-4](https://www.bloomberg.com/news/articles/2021-02-16/gas-at-one-u-s-hub-is-trading-at-999-last-week-it-was-at-4)

**5.法国2020第四季度失业率降至COVID-19前的水平**

巴黎（路透社）- INSEE统计办公室的周二官方数据显示，法国失业率在2020年第四季度下降1.1个百分点至8％。相比之下，第三季度修正后的失业率为9.1％，2019年第四季度修正后的失业率为8.1％。在15至24岁的人群中失业率下降最快，女性的失业率略高于男性。 长期合同工人数从上一季度的49.2％略升至49.8％。

[https://www.reuters.com/article/us-france-economy-unemployment-idUSKBN2AG0GF](https://www.reuters.com/article/us-france-economy-unemployment-idUSKBN2AG0GF)

**6.抖音因涉嫌违反消费者法律在欧洲受到投诉**

布鲁塞尔（路透社）- 周二，抖音因涉嫌违反欧盟的消费者法律以及未能使儿童免受隐藏广告和不当内容的侵害而遭到欧洲消费者组织的投诉。欧洲消费者组织（BEUC）在投诉中列举了几个问题，包括其服务条款“不清楚，模棱两可，并偏袒抖音而不利于其用户。它的版权条款同样不公平，因为它们赋予抖音使用，分发和复制用户发布的视频的不可撤销的权利，而且没有任何报酬。” 除了BEUC的投诉之外，15个国家的消费者组织也呼吁当地监管机关采取行动。

**7.京东物流在港交所提交招股书**

2月16日晚间，京东物流向港交所提交了招股书，成为京东旗下继京东数科和京东健康后第三家IPO的京东系子公司。招股书数据显示，2018年、2019年、2020年前9个月京东物流的收入分别为379亿元人民币、498亿元和495亿元。2018年、2019年及2020年前9个月产生的净亏损为人民币28亿元、人民币22亿元及人民币1170万元。招股书显示京东物流的经营成本占比不断下降，销售成本占比不断增加，研发成本占比不断下降的趋势。
