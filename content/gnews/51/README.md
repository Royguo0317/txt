在中国和美国之间正在展开的技术“冷战”中，欧洲已经成为一个主要战场。“(４月１５号)罗马尼亚政府刚刚批准了这项对罗马尼亚至关重要的法案，从而敲定了2019年在华盛顿签署的一份备忘录。路透社援引罗马尼亚众议员帕维尔·波佩斯库(Pavel Popescu)的话说”。批准了这项法案，实际上是禁止中国和华为参与该国5G网络的开发。罗马尼亚是巴尔干半岛面积最大的国家，他的经济在欧盟国家几乎垫底，政府依然残留着齐奥塞斯库时期的腐败，官僚。但罗马尼亚这些年有了很多的举动，为了经济有活力和政治更稳定，积极的加入欧盟，加入北约。这就是抛弃共产主义以谎言治国的表现，直到今天明确的表态，排除应用中共国华为5G网络。这在全球剿灭CCP的天平上又添加了一个砝码。

罗马尼亚行动缓慢，反应迟钝是源于负重前行。但是，罗马尼亚终于进入了全球（主要是西方发达国家）灭共的行列。

最重要也是最基本的缘由——消灭CCP是每一个想有美好未来，想有新希望国家的唯一选项。

提及罗马尼亚首先让我想起，同时也是战友们熟悉的一个大名顶顶的人物——“齐奥塞斯库”。之所以有名是因为齐奥塞斯库的人生结局，是所有独裁，专制，残暴的统治者们的一个标准的模板。也是共产主义国家独裁者的一个梦魇。更或许是上帝对他们的惩戒，是他们终其一生对本国人民欺骗，愚弄和奴役的最好的奖赏。紧随其后的是在地洞被俘，不久上了绞刑架的萨达姆。然后是躲在下水管道被人发现时还恬不知耻说：我是你们的父亲卡扎菲。然后就没有了然后。下一个会是谁呢？相信战友们心里都有了答案。

历史回放的画面。伟大的共产主义战士齐奥塞斯库同志的宿命，在一次几万人大会上，一个普通人发出一句简短却实用的声音“打倒齐奥塞斯库”，随即旧的虚假的一切就荡然无存。罗马尼亚迎来新的真实的今天。亲爱的战友们，我的同胞们，当我们回顾罗马尼亚当年发生的那一切，使我们懂得：只要我们追求真相时，当我们捍卫真相时，那些奴役我们，欺骗我们的盗国贼是多么的不堪一击，只是一个小小的声音，他们就灰飞烟灭。一切是那么的自然和那么的轻松。

历史不会给我们准确的答案，只能给我们重要的线索。罗马尼亚之所以今天才明确的表态，我个人的观点:共产主义国家给社会造成的创伤是需要很久很久才会愈合，需要几代人的不懈努力。需要大胆，需要勇敢，需要智慧，必须积极主动了解及加入西方国家。

这就是郭先生最为让我敬佩不已的。郭先生在西方的经营，耕耘，以及在许久以前就撒下的种子，现如今已经开花结果，新中国联邦及爆料革命在世界上有了一席之地，得到了西方国家的认可，获得尊重。就连他的死对头，魔鬼中的魔鬼王岐山也佩服得五体投地。郭先生还经常提及的我们（新中国联邦）一定要和西方以美国为首的国家，世代友好。我们才会有未来，有稳定和美好的明天。假如像今天的CCP的战狼外交，天天喊打喊杀的，哪会有未来及子孙后代的福祉。

真相——是最好的武器，当郭先生为真不破成为我们爆料革命的信条，我们的信仰，我们最好的利器。试想当所有的中国人都知道真相时，CCP能存活一天吗？同样安红女士也几次提及胡耀邦说过“要是让人民知道了我们共产党的历史，人民就要起来推翻我们了”。
