美国国务院在世卫组织公布了“中共病毒报告”不到24小时，发布了这份联合声明，现将这份联合声明翻译出来，供战友参考。【1】

原文如下：

关于世卫组织召集的中共病毒起源研究的联合声明

媒体说明

发言人办公室

2021年3月30日

美利坚合众国、澳大利亚、加拿大、捷克、丹麦、爱沙尼亚、以色列、日本、拉脱维亚、立陶宛、挪威、大韩民国、斯洛文尼亚和联合王国（英国）政府发表了以下声明文字版。

文本开始：

澳大利亚、加拿大、捷克、丹麦、爱沙尼亚、以色列、日本、拉脱维亚、立陶宛、挪威、大韩民国、斯洛文尼亚、联合王国（英国）和美利坚合众国政府继续坚定地致力于与世界卫生组织（WHO）以及肩负重要使命的国际专家合作，以及全球社会了解这一流行病的起源，以便改善我们的全球集体卫生安全和应对措施。我们共同支持在不受干扰和不受不当影响的情况下，对中共病毒疫情的起源进行透明和独立的分析和评估，在这方面，我们共同对世卫组织最近在中共国召开的研究会议表示关切，同时强调必须共同努力，发展和利用一个迅速、有效、透明、科学和独立的进程，以便在未来对这类来历不明的疫情进行国际评估。

世卫组织的使命对促进全球卫生和卫生安全至关重要，我们充分支持世卫组织的专家和工作人员，并承认他们为结束中共病毒疫情所做的不懈努力，包括了解该流行病是如何开始和蔓延的。有了这样一项重要的任务，同样重要的是，我们必须表达我们共同的关切，即关于SARS-CoV-2病毒来源的国际专家研究严重拖延，无法获得完整的原始数据和样本，像这样的科学任务应该能够在产生独立和客观的建议和发现的条件下开展工作。我们共同关注这些问题，不仅是为了尽我们所能了解这一流行病的起源，也是为了让这项研究的下一阶段以及为下一次健康危机奠定一条及时、透明、循证进程的途径。

我们注意到这些发现和建议，包括需要对动物进行进一步研究，以找到引入人类的方法，并敦促专家推动的第二阶段研究。展望未来，世卫组织和所有会员国现在必须对准入、透明度和及时性作出新的承诺。在一种具有大流行潜力的未知病原体的严重爆发中，快速地、独立地、专家主导地和不受阻碍地评估其来源，这对于我们的人民、公共卫生机构、我们的行业和我们的政府更好地做好准备，成功应对这种爆发并预防未来的疫情至关重要。对于独立专家来说，充分利用所有相关的人类、动物和环境数据、研究以及参与疫情早期阶段的人员，以确定这一流行病是如何出现的，这一点至关重要。有了所有的数据，国际社会就可以独立地评估中共病毒的起源，从这次大流行中汲取宝贵的教训，并防止疾病爆发带给未来的毁灭性后果。

我们强调需要一个强有力的、全面的、由专家领导的机制，以便迅速调查来历不明的疫情，这一机制是在所有利益攸关方充分和公开的合作下，按照透明、不被公众误导和科学研究完整性的原则进行的。我们将与世卫组织合作，加强能力，改善全球卫生安全，激发公众对世界检测、预防和应对未来疫情能力的信心和信任。

文本结束

笔者的关注点：

1、“我们共同支持在不受干扰和不受不当影响的情况下，对中共病毒疫情的起源进行透明和独立的分析和评估……发展和利用一个迅速、有效、透明、科学和独立的进程，以便在未来对这类来历不明的疫情进行国际评估”；

2、“我们必须表达我们共同的关切，即关于SARS-CoV-2病毒来源的国际专家研究严重拖延，无法获得完整的原始数据和样本”；

3、“……而且也是为了让为这项研究的下一阶段以及为下一次健康危机奠定一条及时、透明、循证进程的途径”；

4、“我们注意到这些发现和建议，包括需要对动物进行进一步研究，以找到引入人类的方法，并敦促专家推动的第二阶段研究。展望未来，世卫组织和所有会员国现在必须对准入、透明度和及时性作出新的承诺。”

对于联合声明如此之快出炉，昨天在白宫记者招待会有记者进行了点评，“……坦率地說，这份声明相当官僚主义，就死亡人数而言，或许不符合这个国家危机的严重性。”普萨基辩解，“有时看起来是官僚主义的，但却是善意的。”【2】

“联合声明”除了冠冕堂皇地指责了中共的不透明等恶举外，提出了“下一阶段”的概念，昨天白宫发言人也提到了这个词，与华大妈相吻合的地方是强调了专家的作用。
