**西班牙2021年1月28日电/西喜社——**一家日本制药公司将通过许可协议在日本生产超过9000万剂阿斯利康中共病毒疫苗。

官员周四表示，这将涵盖这家英国-瑞典制药公司承诺向该国提供的供应量的四分之三。许多人认为，疫苗接种将是日本今年夏天举行推迟的奥运会的关键，但因为该国已知的谨慎决策过程和公众对疫苗的警惕，前景仍然不确定。

内阁官房长官加藤胜信表示，该计划是阿斯利康公司根据双方已经达成的协议承诺向日本提供的1.2亿剂疫苗的一部分，并补充道。”在日本建立疫苗生产系统是非常重要的”

公司称，阿斯利康已与总部位于日本西部兵库县的JCR制药公司签署了一项许可协议。

宣布这一消息时，官员们正在为2月下旬的接种做准备，预计日本卫生部将批准辉瑞的疫苗。日本还从Moderna公司获得疫苗供应。日本表示，已经从这三家供应商那里获得了超过1.27亿人口的疫苗。

官员们说，在阿斯利康将提供的1.2亿剂疫苗中，剩余的3000万剂预计将在3月前进口。

日本在2月7日前处于部分紧急状态，已报告375607例确诊病例，5361例确诊死亡。
