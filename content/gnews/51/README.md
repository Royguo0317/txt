近日，中共中央政治局委员、全国人大副委员长王晨表示，中共国绝不能走西方的“宪政、司法独立、三权分立”的路子，而是要走适合中共国的法治道路。王晨还撰文提及习神的思想表示，在世界百年未有的大变局下，必须推进国内法治和涉外法治。即使涉外法治对中共来说，推行起来有很大弱势，中共也要结合立法、执法、司法等手段开展对外斗争。并且，中共更要推动所谓全球治理体系变革，积极参与国际规则的指定，要在国际规则中，加入更多中共的影响力。

中共曾在2017年、2019年多次表示过，要坚决抵制西方“司法独立”、“宪政民主”、“三权分立”等文明的法治制度。现在，这位去年12月因侵害人权及破坏香港自治而受到美国制裁的中共高官之一王晨，再度出来表态中共国应该与文明法治彻底背道而驰，他的这些话透露出多少中共的狂妄与卑劣？

首先，中共所谓的“从中共国国情和实际出发，走自己适合的法治道路”，实际上还是为了巩固自己的强权高压统治，而放弃对全社会、以及中共国民众更有益处的“司法独立”、“三权分立”、“民主宪政”的制度。在中共眼里，自己的统治地位和权力威严不容任何挑战。无论现在的中共国的法治黑暗到什么地步，老百姓无处伸冤严重到什么程度，司法体系以权力和金钱为指标来维系中共的法治建设都不会改变。而郭先生四年爆料一直提及的，中共国是以黑治国、以警治国、以贪反贪，也将持续存在。中共统治下的司法体系不仅毫无进步，反而还在持续开历史倒车，彻底与西方文明社会分道扬镳之心日益显著。

其次，中共眼中所谓的“百年未有大变局”是习加速师所提出的，通过其“亲自指挥、亲自部署”的肆虐全球的病毒超限战，来打垮全世界的经济，并通过疫苗外交来威胁全世界，从而达到进一步称霸世界、统治世界的目的。时至今日，中共认为病毒超限战和疫苗外交已经进行到可以开始完成其重构世界秩序、实现“东升西降”的狼子野心。通过干预香港的自治制度、法治体系，利用国安法在全世界范围内打击、陷害公开挑战中共邪恶力量的正义人士。结合中共最擅长的蓝金黄手段，来完成他们梦寐以求的涉外斗争，进一步将全世界的法治文明摧毁。

最后，中共还梦想着能按照自己的标准来制定国际规则，即全世界都要按照中共的黑暗、贪婪、无底线的规则来运行。全世界都要跪舔中共，一切以中共的利益为根本，没有任何国家、任何团体、任何个人可以挑战其野蛮落后的规则。如果谁胆敢反抗，那么病毒、疫苗、以及被中共阉割后的法律体系等等，总有一款超限武器可以用来威胁、消灭反对者。

时至今日，全世界已在慢慢觉醒，看清中共的本来丑恶面目。然而清醒的速度还不够，中共还在肆无忌惮的加强对全世界的打击和破坏。唯有尽快消灭这个共产主义的毒瘤，才能还世界以清净和平。
