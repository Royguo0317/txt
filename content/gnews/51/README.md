03月22日RBC报道，美国乔·拜登（Joe Biden）政府在国际舞台上延续了其前任的路线，以制裁为主要政治工具。俄罗斯外长拉夫罗夫在接受中共国媒体采访时如是说，全文见外交部网站。

拉夫罗夫认为，制裁不仅成为美国的主要工具，也成为整个西方国家的主要工具。这位部长说，这些国家“已经忘记了如何使用经典外交”，即建立关系，倾听对方的意见，并找到利益的平衡点。

拉夫罗夫说，西方国家在国际舞台上使用最后通牒，一旦“遇到礼貌的回绝”，就会立即采取行动进行制裁。

“不幸的是，这些礼仪在美国已经根深蒂固。早在奥巴马执政时就开始了，并在唐纳德·川普总统执掌白宫时持续了四年。现在，我们看到美国新政府表现出同样的‘本能’。”拉夫罗夫说。

3月18日，华盛顿扩大了对俄罗斯的出口限制，禁止武器进出口和提供国防服务。在9月1日之前，在国家间空间领域合作框架内的业务以及商业性空间发射均属例外。

扩大制裁范围并对俄罗斯官员实施限制的原因是，俄罗斯当局被指控对被判犯有叛国罪的前GRU上校谢尔盖·斯克里帕尔（Sergei Skripal）和反对派活动家阿列克谢·纳瓦尔尼（Alexei Navalny）使用了禁用的有毒物质。俄罗斯否认参与这两起案件。
