**西班牙2021年4月10日电/西喜社——**

当中共政权竞相通过强制性政策和宣传攻势为数千万人接种时，一些地方官员却自己悄悄地躲避镜头。

《大纪元时报》获得的日期为3月底的内部文件显示，在北京积极推行疫苗行动的过程中，辽宁部分地区的数十名官员以各种健康理由不用中共国COVID-19疫苗。

在该省中部的新泰镇，66名官员中有3人接种了疫苗，另有2名在登记册上，显示出不到10%的惨淡意愿率。

根据3月29日一张总结全区疫苗接种工作的图表，50名政府官员和当地司法所、执法部门的官员给出的理由是潜在疾病、过敏、怀孕、近期手术或感冒等。

过敏是最主要的原因，被提到了22次。 还有20个人提到了高血压，5个人提到了糖尿病——尽管这两种疾病都是慢性病，都将使他们非常容易受到中共病毒的感染。

在拥有该镇管辖权的泰安县，住房和城乡建设局由56名成员组成，只有4人同意接种疫苗，其中3人以“健康不佳”为由推脱。在县政法委员会中，这个中共资金雄厚、直接监督法院系统的机关里，有一个人提到了牙疼。在泰安的财政服务中心，有些人说他们正在吃中药或患有不明疾病。

该县的主要技术学院泰安职业教育中心，疫苗接种验收情况也不乐观，约有三分之一的人声称自己有过敏体质，很多人说自己正从季节性流感中恢复。

在中共国最新的疫苗接种行动中，这些肩负着 “以身作则” 任务的官员的参与率很低，这似乎意味着中共国国内更缺乏热情。

该政权一直在国内外推广其自产疫苗，向69个国家/地区免费运送疫苗，并在3月底前创下了在中共国管理1亿支疫苗的记录。中共国疾病预防控制中心主任高福最近对官方媒体表示，他们的目标是为70％至80％的中国人口接种疫苗，以实现群体免疫。

![]()![](https://gnews.org/wp-content/uploads/2021/04/image0-2-1.png)

然而，达到疫苗接种记录需要在全国范围内动员共产党干部。有的地方，过分热心的官员挨家挨户为居民接种疫苗，甚至威胁将不合作的居民列入黑名单。

与此同时，这场运动也充满了对疫苗质量的质疑。

对于该政权急于分发的中共国疫苗，人们仍然知之甚少。主要的疫苗制造商——国有的国药集团3月28日宣布计划在海外进行第三阶段试验，以观察对病毒的保护能持续多久。在香港，至少有12名当地人在北京的 “华生 “疫苗注射后死亡，尽管当局说疫苗不是导致他们死亡的直接原因。

两家公司都没有公开其临床数据，尽管世界卫生组织3月31日保证其功效水平高于50％（疫苗公开使用的最低门槛）。

“我们对这种疫苗一无所知，”一位姓李的大陆人告诉《大纪元》的中文版新唐人。 “根本没有第三阶段的试验数据，也没有分析它是否可以产生抗体，我们是否仍然可以被感染或感染他人。我们什么都不知道。

“我们最好简单地在家里喝几碗鸡汤，提高免疫力，你说是不是？”

在公众的质疑声中，该政权一直在加大压力。官方媒体的主播们赞同疫苗接种的口号，敦促人们“伸出援手”。

在海南省南部，一个名叫万城的当地小镇告诉居民，如果没有疫苗接种证明，就不能乘坐公交车，也不能去餐馆和超市。
