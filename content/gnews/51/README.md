## 中共超限战灭港计划（二十四）暗杀周恩来
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1046701/)

搜集/撰文：岁月如歌 / 封面合成：文粤

雙十暴動的最終結果是中共左派勢力佔領了香港，並以此威脅港英政府，港英政府當然不會答應他的要求，並且巧妙地脫身於國共兩黨爭端的漩渦中。國民黨政府右派在蔣政府的杯葛下逐漸走下坡路，從此幾乎沒有什麼大的動作，大勢基本過去。

雙十暴動是國民黨右派組織復仇發起的騷亂，但是最終沒能在這次暴動中奪回丟失的政權反而讓自己幾乎一蹶不振。在共產黨奪取政權之前，就一直在香港做了很多鋪墊做了很多準備工作，並且和港英政府也有很大的勾兌，可以這麼說中共港英政府的半公開默許下，在香港早就奠定了基礎。中共最善於用掺沙子的方式來戰勝對方，在雙十騷亂後期，中共掺沙子並且利用了三合會成功佔領香港。中共逐漸佔了上風，而國民黨政府一開始支持香港右派，但由於事態發展越來越嚴重，加上在此事件中，瑞士駐港領事館副領事兼參贊恩斯特及其夫人的座駕，被示威者推翻並焚燒，造成恩斯特重傷、其夫人死亡。這件事對國民政府來說是非常不利的，而且是很難收拾的局面。因此國民黨政府為了自身利益，不得不杯葛這次暴動，並協助維持香港治安，逃避了責任。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4032.jpg)图片来自维基百科
国民政府选择杯葛这一次暴动，也是迫不得已。因为国民政府害怕这次暴动闹大了对他过去做的事情败露，并被追究责任。大概于双十暴动发生前的一年多，国民政府在密谋暗杀周恩来。 195年亚非会议（万隆会议）准备召开，国民党情治人员获悉周恩来将率中共代表团前去参会，中共代表承包一架印度航空公司客机——克什米尔公主号，国民党情治人员认为这是一次千载难逢的机会于是他们密谋趁机干掉周恩来。经过周密的商量决定由戴笠的得意门生之一赵斌丞，他当时是保密局敌后部署组组长。此人平时沉默寡言但非常有心计，另外一个人叫陈鸿举，两人都是台湾派到香港从事颠覆活动的情治人员。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4033.jpg)图片来自今日头条
赵陈二人到机场考察了一番认为可行，他们商量要如何完成这件暗杀的任务。要暗杀周恩来就必须要在周恩来乘坐的飞机上安装炸药，这件事由谁来完成呢？人选非常重要，既能胜任，又能保密，又不能让人认得出人来。赵陈人经过一番寻找最后目标落在在启德机场的做清洁工小郑的身上。小郑原名周驹，也是情治人员，是新面口没几个人认得，又刚好在启德机场工作，家庭人员简单只有一个父亲，没什么后顾之忧能放手去干这事。小郑被50万（也有的说是60万）港币吸引，答应做此事。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4034.jpg)图片来自《随意写》
只要把炸药安装在飞机上，时间一到炸药爆炸所有的人包括飞机、证据都一切烟消云散。证据被炸毁了无法追查，也就不用承担任何责任，事情商榷的差不多了，赵陈二人回台湾亲口向当时的保密局局长毛人凤报告，毛听后大赞，认为可行。但是50万酬劳太高，赵陈二人不敢跟毛人凤讲，叫有“地下局长”之称的谷正文帮忙搞定。谷正文听完赵陈二人的详细计划后非常赞同，认为这是一箭双雕而且天衣无缝。一，暗杀中共领导人本来就是特务做的事，二，可以籍这机会对港英当局施压，他们对国民党情治人员太苛刻，抓到就重判。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4035.jpg)图片来自AM730
赵陈二人拿着50万港币来到香港，按照谷正文的建议，把小郑和他的父亲接待旅馆一起住，以防小郑因为害怕临时退缩，并在旅馆里教会他如何使用炸药和熟悉计画步骤。这个炸药是美国中情局提供的高科技TNT，可暗藏在牙膏里蒙骗过关带入机场，工作人员是可以携带牙膏之类的东西进去的，小郑顺利通过。小郑其实也是个有计谋的人，在研究成功实施后如何撤退上他的建议很不错，并在事后他成功逃脱。小郑在启德机场工作非常熟悉这里的环境，他建议在成功实施后，躲到飞虎将军陈纳德的客机机轮间，因为这架飞机经常来往台湾和香港之间最方便不过了。小郑顺利地躲在陈纳德的飞机逃到了台湾，但是当他从机上跳下来的时候引起了一阵骚动，大家都以为他是个偷渡客并要把它抓起来。谷正文吓得赶紧前去接人，遭阻挠。小郑在事情未结束前绝不能爆光，否则事情败露了，谷正文为了不让事情功亏一篑，吓唬阻挠的赵姓上校，顺利接走小郑，据说该上校因此被免职。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4036.jpg)图片来自bbs
与此同时台湾获得消息，喀什米尔公主号并没有按照计划从香港起飞，周很可能没有上飞机。据后来得知周临时改变了主意应约去了缅甸，在仰光约会再转机去万隆，因此逃过一劫。据维基百科记载，与此同时，中共也接到秘密情报，并由新华社香港分社将此情况通知了印度公司，也告知了港英政府希望能加强安保。周一行主要人物先去缅甸再包一架印航飞机C-54“空中霸王”前往印尼。登机的只是一些工作人员和外国记者等。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4037.jpg)图片来自新浪军事
印度航空公司当时认为飞机在香港只停留1个多小时问题不大，决定按计划行事，抵达启德机场后全部人员都不得带行李，下飞机去机场餐厅用餐，留机械师一人看守。 1955年4月11日中午11时20分，克什米尔公主号从北京飞抵香港启德机场，港英政府虽已知情报，也在机场内加强保安，但也认为问题不大，仅派出一辆警车监视。解密档案显示，包括机组人员在内，共有16名中外人士遇难，其中包括8名中外记者。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4038.jpg)图片来自搜狐历史，亲历者回忆克什米尔公主号事件
公主号被炸了，人也死了不少，中共一口咬定是台湾特务干的，但是国民政府打死也不承认，说没有的事。中共得到的匿名情报也无法顺藤摸瓜展开调查。究竟是不是台湾情治人员干的呢？几十年过去了，当时任保密局祯防组组长谷正文打破沉默，说出了克什米尔公主号暗杀周恩来始末。
