**西班牙2021年4月21日电/西喜社——**
![]()![](https://gnews.org/wp-content/uploads/2021/04/0001.jpg)
就在“七千人大会”结束的6个月后，1962年9月24至27日，毛便组织召开了“八届十中全会”。在会上毛声色俱厉的把6个月前“七千人大会”的纠左政策，全部翻了案。毛坚持全党全国要继续进行“阶级斗争”，并坚持“阶级斗争”要“年年讲、月月讲、天天讲”。

并同时提出要在全国各阶层推行一项以阶级斗争为主旨的“社会主义教育运动”。这一运动反映在农村工作上，便是所谓的“四清运动”：清公分、清账目、清财产、清仓库。

反映在城市工作中，则是所谓的“五反运动”：反贪污盗窃、反投机倒把、反铺张浪费、反分散主义、反官僚主义。（后来扩大为“大四清”，即“清政治，清经济，清组织，清思想”。）这样一来，从国家政策上来说，毛、刘两派就变成水火不容，“不是你死，就是我活”的两个敌对“政党”了。

为了扩大“四清运动”，毛在1963年5月的一次工作会议中，制定了一项指导四清的工作纲领，史称“前十条”。此时的刘与毛还未正式翻脸，毛所制定的政策，刘还得加以推行。刘认为这个“前十条”是个极左文件，真正执行起来打击面太大，为了保护党内的正常干部，及可能遭殃的无辜百姓，刘对这“十条”加以修改，并在1964年9月的一次工作会议上修正通过，这便是所谓的“后十条”了。

刘的这一修正，立刻激怒了毛。几乎就在“后十条”颁布的同时，毛便又一次召开了中央工作会议，立即废除“后十条”，同时颁布了更左的“二十三条”（《农村社会主义教育运动中目前提出的一些问题》）取而代之。这一来，刘、毛二人短兵相接，就非要斗个你死我活了。（这“后十条”也正是文革时期红卫兵小将用来批斗刘少奇的，所谓“妄图反抗毛主席所炮制的‘黑文件’”。）
![]()![](https://gnews.org/wp-content/uploads/2021/04/0002.jpg)
这一拖三年的，可怕的“四清五反运动”又整了几百万人，几万人被逼死。毛派算是大获全胜，但是其对手刘派并未败下阵去。刘之所以仍然在“狂风”中“稳坐钓鱼台”，其数十年的革命历史及建国功勋，是主要原因。

再者，毛大搞“公社化”的胡作非为和极左路线，在党内的“大小山头”那些比较务实的干部中是没有广泛的群众基础的。刘的纠左刹车政策，是和这些“稳健山头”的方向一致的。

如坐镇组织部二十余年的安子文；把北京市委弄成个“针扎不透，水渗不进”局面的彭真；以及坐镇南中国的广东省委书记陶铸，他们对毛的服从都是有条件的，正所谓“上有政策，下有对策”。

而刘的“七千人大会”确定了一致的方向性，更是让全国那些务实的“大小山头”干部们以刘少奇马首是瞻。故此，毛大力推动的，搞了三年的“四清五反”的极左政策，实际上对刘“毫发未伤”。并为清醒者对其又一次的胡乱瞎搞所耻笑。

然而，这个以“斗争”为快乐之本的“斗争专家”毛，岂非常人？非常之人，行非常之事，建非常之功。为了重建那个他一人独裁的“毛家天下”，彻底搞掉那个犯上作乱的“刘家乱党”，全国的党政系统也要来个大换血。毛只有另行“非常手段”了！于是，那个以“社会主义教育运动”为前奏的“文化大革命”，便在毛的策划下，慢慢开始了。

笔者简评：

这五年间的中共，依旧是各种“斗争”。“治国无方，弄权有术”的毛，满脑子的传统帝王思想。并且一直以来毛所迷信的过激的共产主义，让他日日夜夜沉迷在“共产主义”乌托邦的幻想中，也使得他带领全党、全国走向了极左的路线。

在人类的历史上，中国共产党实在是组织最严密的一个大“帮会”。中共的这部统治大机器，就像一个针插不进、水渗不透的大“金字塔”。从乡村到城镇，从城镇到省市，从省市到中央，这个硕大的“金字塔”，又由无数个地方的小“金字塔”堆砌而成。

而组成这些小“金字塔”的，正是由塔顶端的中央，指派下去的各地方的党员干部。在塔的内部层层节制，上级服从下级，全党服从中央，任何人不得违背。他们所推动的政策也是层层下达，看政策性质，规定到下达的层面。

而站在“塔尖”的毛，为了弄权，亲手打破了这个体制。在全国推行“公社化”之时，“公社”这一共产主义建制便代替了原有的地方建制，毛便可以越级指挥。而这种弄权，对全国人民、乃至全党带来的伤害是显而易见的。

面对毛破坏体制的这一行为，周恩来装聋作哑，视而不见；刘少奇忍气吞声，敢怒不敢言；彭德怀痛心疾首，点名叫骂……而中共这部“绞肉机”般的体制，却只能以大独裁者毛为主轴，全党上下都围绕着他一人在转。任凭他在光天化日之下，红口白牙的指鹿为马、颠倒黑白。偶有敢于仗义执言者，也都被他以“莫须有”的罪名，以“斗争”的方式踢进监狱，最后落得个惨死的下场。

不可否认的是，在中共党内以刘少奇为首的一些领导，也想要想让人民安居乐业，想要把国家扶向正轨。但是他们低估了毛对权力的贪婪，低估了毛的弄权之术。

1946年的中共“七大”，刘少奇亲手把毛泽东捧上了“神坛”。22年之后，毛泽东送给刘少奇的是“隐姓埋名的惨死”。毛刘之斗，也把中华民族带入了万劫不复的黑暗深渊——“文化大革命”。
