**西班牙2021年3月2日电/西喜社——**

据infowars3月1日报道，日本政府已要求中共停止对日本公民进行中共病毒的肛门拭子检查，理由是该手术引起“心理痛苦”。

日本内阁官房长官加藤胜伸说：“一些日本公民向我们在中共的大使馆报告说，他们接受了肛门拭子检测，这造成了极大的心理痛苦。”

中共在1月份在几个城市推出了肛门拭子测试，声称它们是检测中共病毒的更有效，更准确的方法。

上周，中共否认强迫美国外交官接受肛门拭子手术，中共国外交部发言人赵立坚在一次新闻发布会上说：“中共从未要求驻华的美国外交人员进行肛门拭子测试。”

但是，工作人员告诉美国国务院，他们已经接受了这项检查，其中包括将3-5cm的棉签插入肛门，然后轻轻旋转以收集样本。
