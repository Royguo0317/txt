# 重要事件

- **[05.28 Newsmax：武汉实验室的科学家们声称他们被蝙蝠咬过](https://www.newsmax.com/newsfront/wuhanlab-origin-covid19-donaldtrump/2021/05/28/id/1023156/)**


武汉实验室的科学家们被视为大流行病的潜在来源，他们声称被蝙蝠咬了，并被溅上了血。
据中国国家电视台报道，录像显示武汉病毒研究所的研究人员在处理蝙蝠时无视个人防护设备的使用。2017年12月29日在中国首次播出的这段录像，切入了一个人的肢体严重肿胀的片段。
据台湾新闻网报道，病毒专家崔杰一度说到，被蝙蝠咬到就像 “被针刺 “一样。
据《纽约邮报》报道，在一篇现已被删除的文章中，武汉实验室的研究人员石正立说：”这项工作并不像大家想象的那样危险”。
在大流行病爆发之前，石正立写道：”直接感染人类的几率非常小”。目前还不清楚她指的是在实验室里处理蝙蝠还是病毒。”在大多数情况下，只会利用普通的保护措施，”她补充说。
该视频在中国国家电视台播出，以展示该实验室并庆祝其 “蝙蝠女 “主任石正立。
在COVID-19大流行病出现后，该实验室成为焦点，成为一个可能的起源。然而，该病毒的起源仍然是个谜。据《太阳报》报道，一位被称为 “零号病人 “的科学家在疫情开始时从实验室消失了。据怀疑，病毒专家黄燕玲在2019年秋季首次感染了该病毒。

- **[05.28 Breibart：报告显示中](https://www.breitbart.com/tech/2021/05/28/report-china-could-use-huawei-cloud-to-blackmail-global-economies/)**共**[国可能利用华为的云服务勒索全球经济](https://www.breitbart.com/tech/2021/05/28/report-china-could-use-huawei-cloud-to-blackmail-global-economies/)**


总部设在华盛顿的战略与国际研究中心（CSIS）发表的一份报告警告说，由于电信巨头华为以折扣价向几十个国家的政府提供大量互联网存储的战略，中国可能获得对亚洲、非洲和拉丁美洲国家的 “胁迫性杠杆”–实质上是讹诈材料。
这份题为《华为的全球云战略–经济和战略影响》的报告描绘了华为向41个国家出售与行政服务和计算机硬件捆绑的云存储的努力。
报告分析的70项不同交易的大多数客户国被自由之家认为是 “不自由 “或 “部分自由”–或如CSIS所说的 “非自由主义”–使它们至少在意识形态上倾向中国的专制政府。华为的大部分客户位于亚洲或撒哈拉以南非洲，被归类为 “中等收入 “国家。
CSIS观察到，这些也是购买了华为 “安全城市 “计划的新兴经济体的主要特征–换句话说，大规模的视频和电子监控系统。
简而言之，这些国家正在变得更加繁荣，但也更加独裁，更加依赖中国提供维持其政府系统所需的工具。它们也恰好代表了未来十年世界人口增长的大部分。
SIS报告的作者担心华为在新兴经济体的电子基础设施中成为如此大的参与者的 “商业和战略影响”，尽管它在整个国际云存储市场上仍然是一个相对较小的力量。
华为的主导地位可以帮助中国共产党（CCP）塑造这些成长中经济体的政治文化，提前阻止任何可能由日益繁荣激发的自由化趋势。
CSIS指出，这些 “不自由 “的国家也为华为及其共产党主子提供了理想的试验场，以发展在西方世界会被视为不道德的监视和控制系统。这可能转化为华为未来对美国和欧洲电子政务供应商的竞争优势。
“报告指出：”在这场竞赛中，发展中经济体本身就是宝贵的试验场和奖品，与发达经济体相比，它们的需求强劲，进入壁垒较少，审查也较少。
华为正在抢占较贫穷国家的市场份额，其成功的说法无法证实，一系列硬件和软件捆绑在一起，吸引了渴望电子基础设施的发展中国家，并通过中国的国有银行提供积极的融资。
“在几乎所有可以确定的融资案例中，中国的融资实体都是来源。毫不奇怪，最常见的融资来源是中国进出口银行和国家开发银行，这是北京最大的两家政策性银行。CSIS报告说：”在某些情况下，这些数据中心基本上是来自中国的礼物。
中国通过华为的硬件、存储和服务捆绑对其客户国发展 “胁迫性杠杆 “的危险来自中国系统臭名昭著的安全漏洞。CSIS列举了几个例子：华为建造的设施安全性差，其系统存在严重的性能问题，采购过程中的腐败指控，以及中国黑客在华为出售给贫穷国家客户的系统中疯狂作案的记录。
报告设想中国情报人员从华为的系统中采购有价值的信息，这些信息可以用来影响发展中经济体或威胁其政府遵守中国的政策目标。
美国之音新闻(VOA)周五评论该报告时指出，中国法律规定每家公司都要完全听从人民解放军及其情报部门的安排，反过来说，任何公司如果与 “国家情报工作 “合作而背叛其客户，就会受到中国政府的保护。
CSIS最后建议美国和盟国的政策制定者和企业更努力地竞争发展中国家的电子政务和云存储合同，消除使西方云服务提供商处于不利地位的监管障碍，并集中资源 “扩大数字基础设施、技术援助和培训的融资和资金”。
“需要采取行动，因为在未来十年，发展中国家将在全球网络中发挥更大的作用，”CSIS提醒说。”美国在云计算方面有优势，可以使更多的发展中国家受益，提高他们的竞争力，并在这个过程中支持美国经济。一个更有效的推销将需要对低收入市场的决策者所面临的权衡问题给予同情，因为在这些市场中，价格往往高于安全问题，并制定激励措施以鼓励采用替代方案。”
“美国手上有强大的替代品。现在它必须进行竞争，”报告总结道，并暗示如果美国在发展中国家认真竞争电子基础设施方面等待的时间太长，这些竞争优势可能很快就会丧失。

**[05.28 Daily Caller：据报道福奇不顾病毒大流行的威胁冒险支持病毒研究](https://dailycaller.com/2021/05/28/fauci-covid-research-outweighs-the-risks-pandemic/)**

据《澳大利亚周末报》报道，尽管存在大流行的风险，安东尼-福奇博士还是支持武汉病毒研究所（WIV）的冠状病毒实验。
国家过敏和传染病研究所所长福奇支持武汉病毒研究所的功能增益研究–将冠状病毒重组为具有高度传染性的疾病–他知道该研究可能导致大流行。据《澳大利亚周末报》报道，2012年10月，福奇在为美国微生物学会撰写的一篇论文中认为，实验室研究的好处 “超过 “了大流行的风险。
“在一个不太可能但可以想象的转折中，如果这位科学家感染了病毒，导致爆发并最终引发大流行怎么办？Fauci写道：”在这个领域工作的科学家可能会说–事实上我已经说过，这种实验和由此产生的知识的好处超过了风险。
Fauci认为，既然大流行病起源于自然界更为合理，那么科学家们应该继续进行涉及功能增益研究的实验。他指出，担心科学家 “马虎 “地复制研究是一个合理的担忧。
“Fauci写道：”在自然界中更有可能发生大流行病，而需要在这种威胁面前保持领先是进行可能看起来有风险的实验的主要原因。”在研究界，许多人表示担心，重要的研究进展可能会停止，只是因为担心有人在某个地方试图马虎地复制这些实验。这是一个合理的担忧”。
尽管他支持这项研究，但Fauci理解这项研究的风险，反对者有合理的担忧。
“他写道：”我们不能指望那些有这些担忧的人简单地相信我们这些科学界人士的话，认为这项工作的好处大于风险，我们也不能忽视他们要求提高透明度的呼吁，他们对利益冲突的担忧，以及他们就这些实验首先应该进行的努力进行对话。
据《周末澳大利亚人》报道，2014年，由于高度传染性的流感病毒的传播，奥巴马政府禁止为这项研究提供资金。据报道，同年，剑桥工作组与其他科学家一起写了一封信，警告该研究可能导致高度传染性的爆发。
“在这样的环境下，意外的感染可能引发难以或无法控制的爆发。信中说：”从历史上看，新的流感菌株一旦在人类群体中建立起传播，就会在两年内感染世界上四分之一或更多的人口。
据《周末澳大利亚人》报道，当美国国家卫生研究院（NIH）宣布将于2017年12月开始重启功能增益研究时，福奇并没有告知特朗普时代的白宫官员有关大流行的风险。
“我认为，（国家安全委员会）工作人员、总统、白宫办公厅主任，这些人都被蒙在鼓里，不知道他正在调回研究，”一位官员告诉《周末澳大利亚人》，这种说法是真的。
福奇说他怀疑冠状病毒是自然产生的，并在5月11日的参议院听证会上否认资助功能增益研究。

**[05.28 Washington Examiner：美国共和党国会议员提议成立一个小组调查COVID-19的起源](https://www.washingtonexaminer.com/news/republican-pushes-new-committee-to-investigate-covid-origin)**

一名众议院共和党人重新提出立法，以建立一个两党和两院的小组，调查COVID-19的起源，反映了最近几周引发的新关注。
由肯塔基州众议员安迪-巴尔提出的这个专门委员会将包括众议院的10名成员和参议院的10名成员，并由国会两院的两名共同主席领导。
“巴尔在一份声明中说：”一年多以后，60万美国人悲惨地死去，我们仍然不知道COVID-19病毒的起源。”这就是为什么国会需要建立一个两院的、两党的调查，以获得美国人民的信任和信心，提供关于COVID-19的真相。”
在众议院外交事务委员会任职的巴尔此前在2020年4月提出了一项决议，提议成立一个类似的小组，调查围绕中国处理COVID-19疫情的活动。
据约翰-霍普金斯大学(Johns Hopkins University)称，2019年底在中国武汉报告了首批COVID-19病例，自那时以来，全世界有超过350万人死于该病毒。
关于COVID-19的起源，一直有很多争论。
特朗普和拜登政府都对中国和世界卫生组织在2021年初进行的研究方式表示怀疑。尽管世界卫生组织-中国的报告说，从动物跳到人类是最有可能的，但特朗普官员、共和党领导人和一些科学家指出，从武汉病毒研究所意外逃脱是大流行病的一个非常合理的起源。
美国国家过敏和传染病研究所所长安东尼-福奇博士本月早些时候表示，他不再相信COVID-19是自然出现的，因为他呼吁对其来源进行进一步调查。
周三，美国总统乔-拜登呼吁美国情报界成员 “加倍努力”，为他带来可能有助于在90天内结束这一谜团的信息。

**[05.28 Daily Mail：为追究北京的责任两党立两项法案支持美国新冠病毒的受害者起诉中共国，并成立9/11式的委员会来调查病毒的来源](http://为追究北京的责任两党立两项法案支持美国新冠病毒的受害者起诉中共国，并成立9/11式的委员会来调查病毒的来源)**

众议院议员预计将提出两项与COVID有关的新法案，一项是允许受害者家属起诉中国，另一项是成立一个类似于9/11事件的委员会来调查该大流行病的起源。
福克斯新闻周五报道说，宾夕法尼亚州的民主党众议员康纳-兰姆和共和党众议员布莱恩-菲茨帕特里克将发布 “永不再发生的国际疫情预防法案”，该法案将剥夺中国对诉讼的主权豁免权。
该法案还针对任何其他’在疫情上故意误导国际社会’的国家。
此前有报道称，拜登总统呼吁美国情报机构在90天内报告病毒的来源，这是因为他们声称拥有大量 “未经审查的证据”。
据《纽约时报》报道，这些证据需要额外的计算机分析，并可能导致更多关于实验室泄漏理论的信息浮出水面，在大流行病蔓延到世界各地之前被感染的实验室工作人员，以及COVID如何在武汉周围传播.
菲茨帕特里克在上届国会也提出了第二个法案，即 “美国制造应急准备法案”。
在本届国会提出的版本中，又有四名共和党人和五名民主党人加入了他。
福克斯说，该法案将组建一个两党共同参与的、类似于9/11事件的委员会，调查COVID-19的起源和美国的反应。
该委员会将负责向白宫和国会提出建议，说明哪些物资需要由美国制造，以应对未来的物理或医疗国家灾难。
去年大流行病开始时，个人防护设备的短缺影响了医务工作者安全护理受感染美国人的能力。
应对当前这场危机，我们决不能再发现自己措手不及，无法保护我们的社区，”宣布该法案的新闻稿说。’我们再也不应该看到近60万美国人的生命受到威胁，每天的生活都被颠覆。 菲茨帕特里克周一发表了一份措辞强烈的声明–甚至在白宫分享解密情报说美国情报界无法就病毒的起源达成一致之前–敦促 “我们现在需要答案。
情报界对COVID-19是自然起源，从动物到人类，还是从武汉的一个实验室逃出来的想法存在分歧。
有充分的理由相信，关于冠状病毒的真正起源，已经发生了错误的报告和掩盖。我们应该得到答案。宾夕法尼亚州共和党人说：”不管是地狱还是高山，我都不会停止，直到我们得到答案。
