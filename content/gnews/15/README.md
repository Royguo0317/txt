# 美国视频会议公司中方高管为中共监视用户活动
[!\[\]()!\[\](https://spark.adobe.com/page/DFqZ4kfneUgj6/images/79c33f2e-7e70-43c0-9975-322f44e71254.jpg?asset_id=5f0f83ba-4ac2-4972-b753-b9ddcaa2883b&amp;img_etag=%22dd9cb6aeba65d6cff25190d50a828ecf%22&amp;size=1024)](https://spark.adobe.com/page/DFqZ4kfneUgj6/images/79c33f2e-7e70-43c0-9975-322f44e71254.jpg?asset_id=5f0f83ba-4ac2-4972-b753-b9ddcaa2883b&amp;img_etag=%22dd9cb6aeba65d6cff25190d50a828ecf%22&amp;size=1024)*中共国外交部发言人汪文彬2020年12月14日在北京举行的新闻发布会上发表讲话。（托马斯•彼得/路透社*
美国司法部周五宣布，一名中共国公民曾在一家未具名的美国视频会议公司担任高管，并在该公司提供的服务平台上为中共情报部门监视该公司的用户活动。

上周五一份诉状说，金新疆（Julien Jin, 音译）在美国他所工作的视频会议公司监视使用该公司平台的用户活动，并关闭了至少四场纪念天安门大屠杀周年的会议。

金目前身在中国，FBI于今年11月把他列入“头号通缉”名单，他被指控串谋实施州际间的骚动，以及串谋转换身份识别的非法行为。

该文件表明，该公司是总部位于圣荷西的Zoom 视频通信公司。今年6月份，Zoom曾应中共国政府的要求而道歉，并关闭了纪念天安门事件的视频会议。

该公司在一份声明中表示：“今后Zoom不会应许中共国政府的要求来影响中共国大陆以外的任何人。”

投诉称，中共指示金向中共政府报告视频会议平台某些用户的姓名、电子邮件地址和IP地址。金与同伙一起，于今年5月和6月渗透进被锁定的会议，对参与会议的用户捏造假证，谎称这些用户违反了会议平台的服务条款，并说服该公司关闭了视频会议并暂停了用户的帐户。

司法部说：“捏造的证据谎称，会议内容有虐童或童工、恐怖主义、种族主义或煽动暴力的讨论。”

为了能在中共国开展业务，该美国视频会议公司接受了中共国的强制交换条件，即雇用金来执行“整顿”计划。投诉称，该计划意味着该公司已同意“主动监控用户通讯内容，包括那些表达中共国政府不可接受的政治观点”。

根据金向中共国提供的信息，中共国政府拘留了至少一名准备在天安门事件纪念会议上讲话的中共国持不同政见者。中共当局还拜访了参加电视会议的另一名持不同政见者的家人，并命令该持不同政见者的家人告诉该人停止反对中共国政府的言论。
