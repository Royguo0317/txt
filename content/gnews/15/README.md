## 【信仰之声】谈基督徒的政治与道德责任
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/935351/)

作者： 香草山福音组  山城小哥 编辑：光明 校对/发稿：飞虹

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3.jpg)



论到我们基督徒，保罗说：“我们从他 (耶稣) 受了恩惠并使徒的职分，在万国之中叫人为他的名信而顺服真道”(道即“话语”)。 这里意思是说我们基督徒就是相信耶稣是道成肉身而来到世间的神的独生子，且是自己已经死而复活的救赎主，从而愿意顺服耶稣，按耶稣的话去行的人。

![]()![](https://gnews.org/wp-content/uploads/2021/02/v2-b006cc1b01be57c4e9d82dddbe19c092_1440w.jpg)

使徒雅各也说到：“我的弟兄们，若有人说自己有信心，却没有行为，有什么益处呢？这信心能救他吗？ ……这样，信心若没有行为就是死的。”（《雅各书》 2:14,17）由此看来，我们若没有和信心相称的行为，我们死的信心是救不了我们的。

耶稣基督也亲自说过：“凡称呼我 ‘主啊，主啊’ 的人不能都进天国；惟独遵行我天父旨意的人才能进去。” (《马太福音》 7:21 )主耶稣说得再清楚不过了：“唯独遵行我天父旨意的人才能进去。”

保罗、雅各，甚至耶稣基督都强调了，信心必须有相称的行为，才是活的信心，才能让我们真正得到神的救恩。 虽然人不可能行为完全，但是基督徒必须在行为上不停地改善，向着耶稣这个标杆直跑，努力活出耶稣基督的样式来。

耶稣基督要我们遵行天父的旨意，因此我们一定要从神的话语里去弄清楚神的旨意到底是什么，以便我们能够遵行，方可得天国永生之乐。

论到神的话语，这里有必要啰嗦几句。约翰曾受启示说：“太初有道，道与神同在，道就是神 ……万物是藉着他造的；凡被造的，没有一样不是藉着他造的。 生命在他里头，这生命就是人的光……道成了肉身，住在我们中间，充充满满地有恩典有真理。我们也见过他的荣光，正是父独生子的荣光。” (《约翰福音》 1：1,3-4：14 )。道啊什么之类的，让人有点不解，这段经文直白地说也就是 : 神的话语与神一直同在，神的话语就是神，神通过祂的话语创造了世界，神的话语就是生命，神的话语就是人类的光，耶稣就是神的话语成了肉身来到人世间……这段话太奇妙了，值得每个人好好思索和感受。

耶稣基督立了领圣餐这个仪式，基督徒也常说要吃主喝主。其实就是要把神、把神的话语“吃”到我们心里，通过这个“吃”而与神合一。而当我们按在我们里面的神的话语去生活去行的时候，我们就可以说“现在活着的不再是我，乃是基督在我里面活着” (《加拉太书》 2:20 )。既然神永活的话语在我们里面，而且活了出来，我们也就因此能得到和耶稣基督一样的复活和永生。因为耶稣说过：“天地都会逝去，我的话却永不逝去。” 感谢神，这是基督徒独有的福分。

回到正题。神的话语在圣经里启示出来的就已浩如烟海。要弄明白祂的旨意，让许多基督徒望而生懒。但感谢神，神自己对此已有总结和提炼。神藉着先知的口说：“世人哪，耶和华已指示你何为善。他向你所要的是什么呢？只要你行公义，好怜悯，存谦卑的心，与你的神同行。”(《弥迦书》 6:8)这里神通过先知明明白白地强调了，我们一定要去行神要的公义。

论到公义，神在圣经里说：“耶和华的道理洁净，存到永远；耶和华的典章真实，全然公义。”(《诗篇》 19:9 )这里的典章是指神藉着摩西颁布的摩西律法说的。真正重要的概念常常无法用简单的语言把它定义得清楚完备，而需要人用心灵去感知。

为了让读者感知一下神的公义，下面我摘录几条摩西律法供读者阅读和感受:

因为耶和华你们的神，他是万神之神、万主之主、伟大有力和可畏的神；他不徇情面，也不受贿赂。 他为孤儿寡妇伸张正义，又把衣食给寄居的人。 (申命记 10:17-18 )
不可传播谣言，也不可与恶人携手作假见证。
不可随众行恶，不可在诉讼的事上随众说歪曲正义的话 。
不可在穷人诉讼的事上歪曲正义。 要远离虚假的事。
不可杀无辜和正义的人，因为我决不以恶人为义人。
不可受贿赂，因为贿赂能使明眼人变瞎，又能歪曲义人的话。 
不可压迫寄居的，因为你们在埃及地也作过寄居的，知道寄居者的心情……
但是第七年，你要让地歇息，不耕不种，使你民间的穷人可以有吃的；他们吃剩的，田间的野兽可以吃；你的葡萄园和橄榄园，也要这样作。 (《出埃及记》 23:1-2,6-9,11 )
第七日是属于耶和华你　神的安息日；这一日，你和你的儿女、仆婢、牛驴和一切牲畜，以及你城里的寄居者，不可作任何的工，好使你的仆婢可以和你一样享受安息。 
不可贪爱你邻舍的妻子；不可贪图你邻舍的房屋、田地、仆婢、牛驴和你邻舍的任何东西。(《申命记》 5:14,21 )
既然在地上必有穷人存在，所以我吩咐你说：‘你总要向你地上的困苦和贫穷的兄弟大伸援助之手。(《申命记》 15:11 )

即使从这吉光片羽之中，我相信大多数人都能感受到从摩西律法中彰显出来的神的公义。

难怪有人说读完圣洁的摩西五经，灵魂就感觉如同洗了个澡。其实任何文字性的律法，都不可能穷尽人世间万花筒般复杂的情况。摩西律法也不能。

然而法律的核心，不在其条文，在于律法所彰显出来的立法者的性情。摩西律法清晰地彰显了神公义的性情里的一些基本要素：如真实，公正，自制，怜悯，不偏私……

通过摩西律法，我们也可以清晰地看到神其实一开始就要求基督徒积极地参与公众事务，并且要在公众事务中彰显神的公义，荣耀和大能，赏善罚恶，救济不幸的人……而所谓公众事务就是政治。所以我们基督徒是有义务积极参与政治的。

在此，我认为有必要简单地说一下神和人的关系和救恩的历史，从而更好地理解神的公义。

神创造和统管万有，且所有人都是神按照自己的样式所造。所以我们称神为天父（神是公义，慈爱，全知全能，自有永有……的完美之父）。

神一开始就是一家之长，是我们的主，我们的王，我们的立法者，我们的审判者，我们的医治者，我们的救赎者……神是我们的一切。

然而由于人自由意志的属性和肉体的腐败，人常常不把神放在第一位，甚至故意不认他。人人喜欢偏行己意，为了自己的欲望和骄傲，伤害自己的弟兄姐妹，甚至故意抢夺弟兄姐妹之物，更有甚者甚至直接杀人越货……

因此神在摩西时代，通过摩西颁布了公义的律法，向世人彰显神的性情，让人好效法他，让自己的子女在世界上彼此之间以公义，怜悯，慈爱相待，好活出神的荣耀，活在神的祝福和喜悦中。且神明明白白地告诉世人：敬畏神，遵行神公义的律例典章就可得福。反之，就必在自己的罪恶中灭亡。然而，最初接受神律法的以色列人，他们接下来的历史却向世人如实地展现了人类到底是如何的败坏堕落。他们进入迦南地以后的历史基本上充满了不忠，强暴，欺骗，奴役，淫乱，背叛……一直到灭国。

神作为万灵之父，不愿意看到世人在罪恶中走向毁灭（其实他早就知道一切，也早就预备好了耶稣）。于是神把独生爱子耶稣给了世人，因为“神爱世人，甚至把他的独生子赐给他们，叫一切信他的，不至灭亡，反得永生。“ (《约翰福音》 3:16 )。神的爱远超越我们人世间父母子女之间、妻子丈夫之间的所有人类之间的那些有条件的、有瑕疵的、对彼此有许多要求的爱。因着这样的爱，神把永生的应许通过耶稣带给了世人。

耶稣来了以后，他明明白白告诉世人“你要尽心、尽性、尽意爱主——你的神。 这是诫命中的第一，且是最大的。 其次也相仿，就是要爱人如己。 这两条诫命是律法和先知一切道理的总纲。”圣经也强调过“律法的总归就是爱” 。由此我们可以明白，神赐世人公义的摩西律法，也是因为神爱世人。也就是说爱是公义的源头。

通过神与人的关系和救恩的历史以及神的话语，我们可以看到神的旨意，就是要我们去践行爱和从爱而来的公义。

如圣经里所说，这爱和公义的律法，其实是早就刻在人心板上的。虽然说到爱和公义，我们很多人都说不出个所以然来。但是当我们在现实中遇到具体事情的时候，我们其实心理大概都有点数，知道如何去做才符合神爱与公义的原则，即使他从未听闻福音，只是每个人的选择不同，结局也不同。

圣经中有这样一句话简单地说了一下什么是爱，但我觉得值得每个人仔细读一读：“爱是恒久忍耐，又有恩慈。爱是不嫉妒，不自夸，不张狂；不作失礼的事，不求自己的益处，不轻易动怒，不计较人的过犯；不喜欢不义，只喜欢真理。爱是凡事包容，凡事相信，凡事盼望，凡事忍耐。爱永存不息。”

整个《旧约》，神一直就在对以色列人反复强调两件事：以色列人应该全心全意爱神，在公共事务上要行神所要的公义。

圣经中，以色列人进入迦南地立王以后的历史和现实经验告诉我们：当掌握政治权力的寡头或组织为了一己之利，有意识或者无意识地系统性地行奴役和剥削之实而不顾公义、任意妄为时，常常带来的就是国民普遍的上行下效的道德和信仰的败坏。在这样的普遍的败坏中，神的信徒或基督徒即使满有爱心地去帮助那些在这种普遍性的败坏中的不幸的个人，终究会发现是治标不治本，自己的微薄之力只是杯水车薪，因为无力与政治权力机器抗衡，甚至自己很多时候也只能成为暴政之下苟延残喘的可怜虫。当然，我们不能因为自己能做的有限而不做。在这种情况下，神即使赐下先知也作用有限，能得救赎的人也是极少数，反而先知常常成为掌权者和已败坏了的人民厌恶的对象，甚至被囚禁杀害。这样的状态一直持续到神大洪水式的审判来临。

而当政治权柄掌握在约瑟、大卫这样的虔诚，愿意行公义的人手里的时候，与之相伴的却是信仰的建立，道德的复兴，邦国的兴盛。

由此可见，政治上的权柄掌握在什么样的人手里，对基督徒是多么重要！有人说，对这种时代和环境，我们要交托祷告神。对，是的，我们要交托祷告。但是我们也要行动，努力去扭转局面，因为神常常是在我们行动中加力量给我们。我们的神是积极的神。我们祷告常常说：“……愿人都尊你的名为圣……愿国度权柄荣耀全归给您，直到永远。” 但是如果基督徒不去传福音，人怎么尊神的名为圣？如果不去为国度和权柄征战，权柄又如何归给神，为神的教会和儿女服务呢？有人会说，国度权柄荣耀本来就是神的，也对，但我们这里不是从这个角度去谈。既然政治权柄总有人会执掌，为什么不努力让它掌握在愿意行公义，愿意自我约束的基督徒手里，让这权柄和荣耀归给神，行神的公义呢？或者建立一个系统，确保权力受控，即使是流氓骗子掌权都不能偏离公义太远呢？难道神有说过不允许我们执掌属世的权柄吗？绝不是这样。

耶稣基督绝对没有教我们做懦弱的好好先生，没有原则地追求虚假的和睦，甚至对恶徒唯唯诺诺俯首帖耳，那不是神要的谦卑，自制……耶稣基督说他来是给世上带来刀兵的，这既有属世的刀兵也有属灵的刀兵。耶稣基督也让门徒在他走后要买刀。试想如果当年如果没有十字军和基督徒勇敢的征战，可能就不会有现今的基督文明了。当然，我不是鼓吹暴力，因为征战有很多形式。我是说，我们基督徒要随时有和受撒旦操控的邪恶势力进行不管是属灵还是属世的征战的准备和心态。当我们忘记这些的时候，共产党这样的撒旦组织可没有忘记，它不停发展自己的“统一战线”。

基督徒要有责备那些掌握权柄而行不义的人或组织，并督促其改正，必要时也与他们征战的勇气和智慧。因为在征战中，公义若后退一寸，邪恶必定谋求前进一尺。没有征战，神的国就白白降临吗？不可能。当然，我们也要有智慧，不做无意义的牺牲。不到上十字架得胜的时候，耶稣也会躲避那些要用石头打他的人。遇到圈套，耶稣也会避免正面回答法利赛人故意刁难的问题。

因此在这政治的事情上，我们基督徒都要立场坚定，态度温和，勇于征战，灵巧如蛇，不怕得胜，不怕胜而又胜……我们应尽我们所能，让神的国的荣耀在世上的国上彰显出来。政治权柄是个放大器，若能通过它行出神的公义来，那将是覆盖范围最广，而且泽被当下和将来世代的大爱。因此我们基督徒一定要积极，且有组织地参与政治。

方今中共掌控中国政治的权柄，放病毒于世界，输出腐败于列国，强灌无神论给国民，逼迫扰乱圣教会，奴役淫乱我同胞，歪曲公义于天下……种种恶行，罄竹难书。

且中共本身并非一个政党，乃是一个敌基督的，无神论的邪教。它集魔鬼奴役和毁灭人类的旧计划《商君书》和新计划《共产党宣言》为一体，抄袭且又修改基督徒传播福音组建教会的方法，其邪恶理念流毒于全球。其所做所传，所言，所为……无一不是基督之敌。中共已然是我们基督徒在传福音，建立基督教会之最大的实体化组织化的敌人，它在全方位，系统性地侵蚀着全人类的基督信仰。我们基督徒要看到运行在中共这个组织背后的魔鬼撒旦的本体。这场征战，是全人类追求神公义的人特别是基督徒无法回避的，我们本就不应该回避。正是美国基督徒弟兄姊妹对邪恶的回避，对邪恶妥协，对金钱的贪婪，盛行的享乐之风，让他们丢失了他们先辈十字军时代的卫道精神，才导致中共的做大。

而如今应时而出的爆料革命，不畏中共之残暴，坚持以真灭假，以善灭恶，毫不妥协。这场革命集结了前所未有的正义力量。四年以来披露中共过往之劣迹，聚焦中共当下之恶行，昭示其奴役毁灭世界之野心。它追求法治，公义，致力于救华人于水火，救世界脱魔爪，让华人得享敬拜主耶稣的信仰自由。其实这些事本来应该是我们基督徒发起和带头去做的。

基督徒如果这样公义的运动和事业都不参与，那么还有什么更公义的事情可做呢？ 我们要记住耶稣基督说的：你们不要以为我来是要废除律法和先知；我来不是要废除，而是要成全（《马太福音》 5:17 ）。神对政治上公义的要求从来没有变。爱是公义的源泉，公义是爱的底线。我们基督徒一定要以实际的行动守住这爱的底线，荣耀基督。阿门。
