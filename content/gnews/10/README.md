# 选举人的 “安全港 “期限 “不符合宪法”，”不适用于有争议的州
[!\[\]()!\[\](https://spark.adobe.com/page/KoSjISBgvvovV/images/401a1c1b-53fd-41fe-9579-51687c4455d4.jpg?asset_id=4c030e4a-c2d2-4e07-ad56-630d6d9aa260&amp;img_etag=%229b5ad8a3fa8bb1117be78248b0138647%22&amp;size=1024)](https://spark.adobe.com/page/KoSjISBgvvovV/images/401a1c1b-53fd-41fe-9579-51687c4455d4.jpg?asset_id=4c030e4a-c2d2-4e07-ad56-630d6d9aa260&amp;img_etag=%229b5ad8a3fa8bb1117be78248b0138647%22&amp;size=1024)
监督选举舞弊的阿米斯塔德项目机构（Amistad Project）发布了一份白皮书，概述了12月8日选举人团选举的最后期限如何不适用于选举结果有争议的州。

选举结果有争议的州包括亚利桑那州、佐治亚州、威斯康星州、密歇根州、亚利桑那州和内华达州：总共70张选举人选票。

阿米斯塔德项目指出，总统任期结束的唯一时间是“宪法规定的最后期限，1月20日中午”。

所有其他最后期限，如12月8日的“安全港”最后期限，12月14日的选举人团投票，以及1月6日的国会计票，都由联邦法律确定。如果“联邦法律对忠实遵守宪法的要求构成障碍，那就有必要忽略联邦法条来彰显宪法的明文含义”。

换句话说，《宪法》要求的决定选举结果的唯一最后期限，是2021年1月20日或之前。

“此外，基于对过去的选举经历的考量，这些日期是经过裁决的。”该组织补充说。

为了强调选举人团最后期限的主观性，白皮书列举了与选举有关日期的变化历史：

*在1789年1月的第一个星期三任命了选举人，选举人在2月的第一个星期三开会，3月的第一个星期三，国会联席会议对选举投票进行了统计。半个多世纪以来，从1792年到1844年，选举人必须在12月的第一个星期三前的34天内被任命，选举人在12月的第一个星期三举行会议和投票。139年来，除了1876年的选举外，在1792年至1932年的每一次选举中，国会都是在2月的第二个星期三登记选举人投票。在1888年至1932年的44年中，选举人是在1月的第二个星期一开会并投票。*
[!\[\]()!\[\](https://spark.adobe.com/page/KoSjISBgvvovV/images/404272aa-fd68-4a20-8335-9b4508901bd5.jpg?asset_id=b7c4ce7c-716b-47c2-b339-5bb01e43b7f0&amp;img_etag=%22dbb4f2cfc95cfa4e4cf908a1d5c5ec20%22&amp;size=1024)](https://spark.adobe.com/page/KoSjISBgvvovV/images/404272aa-fd68-4a20-8335-9b4508901bd5.jpg?asset_id=b7c4ce7c-716b-47c2-b339-5bb01e43b7f0&amp;img_etag=%22dbb4f2cfc95cfa4e4cf908a1d5c5ec20%22&amp;size=1024)
直到1848年才设立一个全国选举人日。

还有一个历史和法律先例，“如果结果不明确或可疑，或其他特殊情况，选举截止日期就会被推迟或更改”。

在1789年，3月4日是选举人投票可以正式统计的最早日期。

在重建时期1876年的选举中，选举人投票非常混乱。

*选举人投票不仅接近（从全国报纸已经刊登的各州民众选举结果的报道中可以很容易看出），而且南方至少有三个州（如内战后重建时代）发送了两套选举人票–分别有利于各自政党的候选人。更糟糕的是，其中一个主要政党控制了国会两院之一，而第二党控制了国会另外一院，至使国会无法进行有效投票。*

国会通过立法绕过通常的选举计票程序，要求国会举行投票注册联席会议，以尽早判别哪些州的投票存在争议。国会后来通过“一个由参议员、国会议员和由国会任命的美国最高法院大法官组成的所谓‘选举委员会来’来处理这些争端。”
