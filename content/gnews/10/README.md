13.融化的南極冰層將升高海平面,並可能導致人類「放棄…… 。 紐約” 。

研究指出,如果溫度上升攝氏2度,海平面將上升8英尺。

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/25054637/25-11-3.png)

一項新的研究表明,如果不實現2015年《巴黎氣候協定》的目標,南極冰蓋將融化,導致全球海平面上升到人類必須「放棄…… 。 紐約”的地步。

這項發表[摄氏度，海平面将升高在《](https://www.nature.com/articles/s41586-020-2727-5.epdf?sharing_token=IgybQa4pNDQrJM4avhupVdRgN0jAjWel9jnR3ZoTv0OuZB9Rvamxk4HmXZx1OT90oi5_ByA7GJo49TBx_C-1rhkTnSu0U34nWwMyTEqKGiDCuNXaUeh03RaM93xso5_USoygNZ732yfKc6YmPBEwAQEaDB-AfcBA1JW7O6q9iLcZayHOG9Px_l87vImY81XRYv2S9KyKVeqewXVMXSgtxZw5RRAVS9bmjxhd1VNnXRx9JzG8ZCDpDo_n4Hgwy9_uj1GTn7G46dFFUKeyYXk2IV9Wb7RPjInZfhDS0wxilY2_CyTjxVTmmnQME0QOiHE7gRSD02iwEDUmMC7e0HQP5bZUBw5Azq0g_Z78bNKVnS-LeElTbX6ZsktsAOz_7zNkXPFFLnK7mNw4lD3-vRsayA%3D%3D&amp;tracking_referrer=www.theguardian.com)自然》上的研究指出,如果溫度升高攝氏2.5公尺(88英尺),這是《巴黎協定》規定的溫度極限。 如果溫度升高4或6攝氏度,海平面最終將分別升高6.5米(21英尺)和近12米(39英尺)。

研究合著者里卡達· 。 溫克爾曼(Ricarda Winkelmann)在一份[聲明中](https://www.pik-potsdam.de/en/news/latest-news/stability-check-on-antarctica-reveals-high-risk-for-long-term-sea-level-rise)說:「南極洲擁有地球一半以上的淡水,凍結在厚約5公里的巨大冰原中。  ” 由於人類溫室氣體的排放,周圍的海洋水和大氣變暖,南極的白帽失去了質量,最終變得不穩定。  “

![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/09/25054657/25-13-2.png)

南極洲的冰架(例如此處看到的蓋茨冰架)對海洋溫度升高很敏感。 海洋和大氣條件是冰蓋損失的一些驅動力,科學家在一項新研究中估計了2100年全球海平面還會進一步上升。 圖片來源:Jeremy  Harbeck / NASA 。

溫克爾曼繼續說:「由於其巨大的規模,南極洲對海平面的貢獻是巨大的:我們發現,在2度的變暖,融化和加速的冰流流入海洋之後,最終將導致全球海平面上升2.5米如果將這些溫度水平維持足夠長的時間,則僅從南極洲就上升了。 在4度時將是6.5米,在6度時將接近12米。

奧巴馬政府於2015年達成了具有里程碑意義的《巴黎氣候協定》,其長期目標是將加溫幅度限制在1.5攝氏度以內。 包括中國在內,近200個國家簽署了具有里程碑意義的協定。

在2019年11月上旬,川普政府[開始](https://www.foxnews.com/politics/united-states-begins-process-of-withdrawing-from-paris-climate-agreement)正式退出該協定。

2月[發表](https://www.foxnews.com/science/climate-change-destroy-half-earths-animal-plant-species-in-50-years)的[另一項研究](https://www.foxnews.com/science/climate-change-destroy-half-earths-animal-plant-species-in-50-years)表明,如果未來50年全球溫度升高0.5攝氏度,則大約一半的世界物種將在當地滅絕。 如果溫度升高到2.9攝氏度,則95%的物種將在當地絕種。

3月,另一項研究[表明](https://www.foxnews.com/science/half-worlds-beaches-disappear-by-2100-climate-change) ,如果[氣候](https://www.foxnews.com/category/science/planet-earth/climate)變化繼續下去,到2100年,世界幾乎一半的沙灘都將消失 。

8月,[研究人員](https://www.foxnews.com/science/28t-tons-ice-lost-23-years-study-direct-consequence-climate-warming)發現,在過去23年中,主要來自北極海,南極冰架和高山冰川的冰已流失了28萬億噸,這是「氣候變暖的直接後果」。

2019年5月,另一項研究[表明,](https://www.foxnews.com/science/climate-change-sea-levels-rise) 到2100年, 氣候變化可能會使海平面上升7英尺。
