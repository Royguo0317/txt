近來，[中共國多地洪水](https://gnews.org/zh-hans/234706/)，牆內媒體不單充耳不聞，更是「喪事當喜事辨」可怕的語徑統一和網絡封殺。由於多省、多城洪水，大家都非常關注惡共標榜的“偉大工程”三峽大壩，這裡引用戰友[500Miles的推文](https://twitter.com/Safeguard008/status/1266351908467249153?s=20)，以及6月15日文貴先生的直播中也提到，中共知道三峽問題，也有可能會利用其問題禍害華夏、禍亂世界。接下來就拋磚引玉，希望三峽下游的百姓可以及時準備，做好預防，早早搬離，沒事買安心，有事避災禍。**「相信共產黨，走進火葬場」–文貴先生說。**

**暴雨成災**

6月開始進入雨季、颱風天氣，最近中共國南方已經暴雨成災，江西、湖南、廣西等22個省（區、市）都受到影響。在這暴雨時刻，長江三峽也提前9日完成洩洪，水位由175米下降到145米，30米約221.5億立方米被排出，對應未來日子暴雨來臨時，水庫還有空間儲備。

三峽大壩每年都會被關注，尤其今年-庚子年，也是天災人禍年。其實原因很簡單，中共國最大產出就是「豆腐渣工程」，而且強行而建，人才敗才，家族利益，三峽大壩的崩潰只是時間問題。中共國現在經濟崩潰、人禍、天災，黨內鬥爭壓力日益嚴重，新聞媒體連洪水也不見報道，更何況是岌岌可危的三峽大壩這類超級工程！

結合以上，會造成三峽大壩崩潰的可能性：

1. 大壩內部結構崩潰，無法承擔大量水庫的水。

2. 連續多天暴雨，超過最大壩高181米。

3. 地震引起大壩壩體結構的崩潰。

4. [共振湧浪](https://twitter.com/Safeguard008/status/1266373597011996684)：如果大壩上游發生強烈地震或者大暴雨，會導致長江兩岸的山體大幅滑坡，會形成共振湧浪。大壩泡水一側和裸漏在空氣中一側的熱膨脹係數不一致會導致大壩承受衝擊的能力變弱（變脆）。

而最可怕的就是三峽大壩壩體內部結構崩潰，再加上連續大雨，超過其內部承受力，崩潰後，洪水就會直奔而下，淹沒下游。

**三峽問題**

1. 庫區污染

2. 泥沙沉積

3. 增加上游流域洪水威脅

4. 水庫引起的地震活動

5. 下游地區泥沙減少，引起城市沉降

**結論**

三峽大壩崩潰為什麼在今年風險非常高，天時、地利、人為這三要素都集齊在庚子年。

人為：中共的豆腐渣工程。由於中共一貫貪腐成風，由上而下非貪即腐，相互隱瞞、欺騙。雖然超級工程未必大範圍造假，但偷工減料未必不會。時間一長，維修失利等都使得「炸鍋」更加可能。而且自上而下欺瞞、報喜不報憂的風氣、人才的流失，就算報告真相，更大可能被打成「造謠者」。其次，偽政權中共，70年的管制卻沒有改善城市的排水系統，暴雨來臨卻沒有一個有效的排水系統，這正正是城市淹沒的其中因素。

天時：因為連日暴雨(降雨量超500MM加連續不停的日子)，超過紅色警戒線。加上沒有良好排洪，儲存水量突如其來的排走，大量河水突然而來，下游就會來不及分散到其他河川，最終就是降雨量和河水一起把城市淹沒。其次，地震和共振湧浪，對三峽大壩結構的破壞，多方效應結合，最終會造成可怕的大壩崩潰。

地利：長江一帶，極為重要，做好水利工程有利於民生和低窪地區被洪水淹沒。可惜，中共並沒有民心，視人民為奴隸、韭菜。而在長江一帶建立起水壩，本來要做良好的排洪，好好的規劃，避免洪水泛濫。但中共卻霸道的限制水量，讓下游乾旱，結果是最近排洪，大量的河水突然相湧而下，結果可想而知。補：下游地區包括武漢，又武漢！

**最後**

做好必要準備，能先離開到別地區那是最好不過，不能離開也時刻留意天氣，尤其暴雨天。見到動物有異，或是有震動時，必要往高海拔或是不斷往高處走，準備好可以攜帶的食物、藥品，不要停留。(突然想起，上帝要諾亞建方舟，周邊都笑諾亞是傻子，結果……。如今又會如何，拭目以待吧，沒事是最好不過。但真的就是中共爛賬，百姓承擔，生靈塗炭。而且，偽政權、邪魔中共肯定會說：這是美帝攻擊，反人類……不可想象。)
