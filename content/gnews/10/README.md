```
什么是数字货币？
```


数字货币是仅以数字或电子形式可用，而不能以物理形式可用的一种货币形式。也称为数字货币，电子货币，电子货币或网络现金。

**数字货币和真实货币有什么区别?**

如果由某个国家的中央银行以受管制的形式发行，则称为“中央银行数字货币（CBDC）”。除了受监管的CBDC，数字货币还可以不受监管的形式存在。在后一种情况下，它有资格被称为虚拟货币，并且可以在货币开发商，创始组织或定义的网络协议的控制下，而不是由集中监管者控制。这种虚拟货币的示例包括加密货币以及与优惠券或奖励挂钩的货币系统

**数字货币如何使用？**

数字货币是只能通过计算机或手机访问的货币，因为它们仅以电子形式存在。由于数字货币不需要中介，因此它们通常是最便宜的货币交易方法。所有加密货币都是数字货币，但并非所有数字货币都是加密货币。数字货币稳定并且可以与市场进行交易，而加密货币则通过消费者情绪和价格波动中的心理触发来进行交易。

**数字货币对中国百姓危害**：

央行称数字货币对百姓有好处： 减少纸币流通， 减少纸币硬刷成本。 但是习近平统治下的老百姓会没有好处在使用数字货币，弊大于利：

1. 老百姓的资产个人信息完全被控制，老百姓的资产和个人信息被强迫登记，政府可以直接清楚了解每个人的财产情况。
2. 共产党执政了70 年， 一贯做法是共老百姓的财产.总是接着贪污腐败去不正当去掠夺资产。他们和他们的子女或者私生子女在海外的资产是惊人， 根据郭文贵先生的爆料。
3. 共产党用另一种手段继续奴役和压榨人民。


他们用无信用方式随意发货币， 就像60-70年代使用的粮票和布票来控制百姓来达到掠夺更多百姓财产。实施数字货币和其他外国货币不流通和兑换。纸币被替代后，任意超发数字货币， 引起通货膨胀，老百姓生活水平下降。
