# 揭露：参加中国共产党宣传活动的西方媒体机构的完整列表
[!\[\]()!\[\](https://spark.adobe.com/page/KGj0uTxd1z32s/images/c38a0330-8234-4c61-b82d-8194d34e2700.png?asset_id=7d9fb56b-cd11-4d06-89fe-8d30bfb89997&amp;img_etag=%22457ed3e74b1737ef737b43f67f542f02%22&amp;size=1024)](https://spark.adobe.com/page/KGj0uTxd1z32s/images/c38a0330-8234-4c61-b82d-8194d34e2700.png?asset_id=7d9fb56b-cd11-4d06-89fe-8d30bfb89997&amp;img_etag=%22457ed3e74b1737ef737b43f67f542f02%22&amp;size=1024)
本日，《国家脉动》发布了一份完整的新闻媒体清单。根据对于中美交流基金会(China-United States Exchange Foundation)和注册为中共国代理人的美国游说公司“BLJ Worldwide”（以下简称BLJ）的调查，这些媒体给予了中国共产党以及其外宣机构“有利报道”或“积极信息”。

（调查链接：[https://thenationalpulse.com/exclusive/media-private-ccp-dinners-trips/](https://thenationalpulse.com/exclusive/media-private-ccp-dinners-trips/)）

发布这份名单的同时，《国家脉动》还曝光了中美交流基金会(CUSEF)。该基金会由中共国政协(CPPCC)副主席成立，并已被美国国会美中经济与安全审查委员会(U.S.-China Economic and Security Review Commission简称USCC)确定为中共统一战线工作的重要组成部分。

（基金会曝光新闻链接：[https://thenationalpulse.com/exclusive/media-private-ccp-dinners-trips/](https://thenationalpulse.com/exclusive/media-private-ccp-dinners-trips/)）

根据美国政府的报告，该基金会旨在“协调并消除潜在的对于中国共产党执政政策和权威性的反对意见”以及“影响海外华人群体、外国政府、及其他力量，使其采取行动或站队支持北京当局倾向的政策立场”。

（报告链接：[https://www.uscc.gov/sites/default/files/Research/China’s%20Overseas%20United%20Front%20Work%20-%20Background%20and%20Implications%20for%20US\_final\_0.pdf](https://www.uscc.gov/sites/default/files/Research/China's%20Overseas%20United%20Front%20Work%20-%20Background%20and%20Implications%20for%20US_final_0.pdf)）

司法部《外国代理人注册法》(Foreign Agent Registration Act简称FARA)相关文件证明，这些媒体机构和中美交流基金会的关系已存在十年以上。

[!\[\]()!\[\](https://spark.adobe.com/page/KGj0uTxd1z32s/images/4935a25c-320d-4fe4-8c21-6c9b9310a243.png?asset_id=4455b6b5-ee97-4be6-84a3-46c387a288c4&amp;img_etag=%22b59f67c77b938d4aba6d09a0887ce331%22&amp;size=1024)](https://spark.adobe.com/page/KGj0uTxd1z32s/images/4935a25c-320d-4fe4-8c21-6c9b9310a243.png?asset_id=4455b6b5-ee97-4be6-84a3-46c387a288c4&amp;img_etag=%22b59f67c77b938d4aba6d09a0887ce331%22&amp;size=1024)（现任中美交流基金会主席：董建华）

#### 赞助旅行

根据外国代理人注册法提交的文件，在BLJ游说机构的帮助下，中美交流基金会赞助了去往中共国的旅行活动。这些旅游活动有着明确的目的，即“有效地向媒体、大V、意见领袖和公众传播正面的消息”

这趟旅程被称为“熟悉之旅”，参加者的选拔标准包括“有利报道的时机及有效性”，这些标准将确保中共得到的报道是“有利”的。

“为了在关键国家媒体上获得有利报道，BLJ将继续组织和安排到中共国的‘熟悉之旅’。这项活动将招募顶尖的记者前往中共国，而招募条件为有利报道的时机和有效性。”
