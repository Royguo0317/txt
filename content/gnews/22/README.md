# SnapChat宣布永久封禁川普

**头条：**据Axios首次报道，Snapchat周三成为最新一家对美国总统川普采取行动的科技公司，永久禁止了他的账户。

*** 关键事实***

该公司周三宣布，Snapchat将于1月20日川普离任时永久禁止其账号。

上周国会大厦围攻事件发生后，Snapchat已经锁定了川普的账户，今年夏天，该社交网络限制了他的频道覆盖范围，以回应总统对“黑命贵”运动的评论。

***关键引用***

“为了公共安全，基于他试图传播错误信息、仇恨言论和煽动暴力的行为，这明显违反了我们的准则，我们决定永久终止他的账户，”Snapchat发言人说。

***关键背景***

Snapchat只是骚乱发生后对川普采取行动的几家社交媒体公司之一。在过去的两个月里，川普利用他的扩音器宣传毫无根据的选举舞弊阴谋论，这促成了他的支持者上周冲进国会大厦。推特永久禁止了他，脸书无限期地暂停了他的账户，油管将他踢出去7天，就连Shopify、Twitch和Stripe等平台也在最近几天限制了总统。
