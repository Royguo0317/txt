环球网5月31日转载央视财经消息，自2021年3月起，国际黄金价格一路上涨。截至5月28日，黄金收盘价达1903美元/盎司，涨幅逼近15%。黄金价格上涨的同时，黄金开采企业产量翻倍，业绩暴增300%。

根据中共黄金协会的数据，2019年国内原料黄金产量为380.23吨（同比下降5.21%），进口原料黄金120.19吨，全年黄金平均价格1395.60美元/盎司（同比增长9.84%）；2020年国内原料黄金产量365.34吨，进口原料黄金114.16吨，人民币黄金价格同比增长25.73%；2021一季度原料黄金产量74.44吨，同比减少9.92%。根据以上数据，近年来国内的黄金产量和进口量都有小幅缩水。2020年，全国黄金实际消费量820.98吨（同比下降18.13%）。在供需都疲软的前提下，黄金价格仍有巨大涨幅，市场操纵痕迹明显。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/31213734/Screen-Shot-2021-05-31-at-9.35.50-PM.png)
2021年一季度，作为国内黄金主要产区的山东省有两家金矿企业发生安全事故。受此影响，山东省二月份停产并进行大检查，一季度黄金产量也因此明显下降。作为国内第一产金企业的山东黄金矿业公司由于停产，一季度产能同比下降40%，营收减半。然而央视财经报道中提及的内蒙古某矿业公司，不但2020年业绩比19年翻3番，2021年一季度更比同期暴增1800%。即便国内数一数二的产金企业也没能收获如此“暴利”，数据的真实性令人质疑。

随着CCP病毒真相不断浮出水面，全世界联合灭共进程加快，人们对爆发金融危机和通货膨胀的担忧与日俱增……这些都是黄金价格上涨的动力。然而，货币日渐贬值是人民币黄金价格大涨的幕后推手。2020年实体经济凋零，而抗疫经济利益却有67万亿规模（来自李玲，《疫情引发对中国未来发展模式的重新思考》论坛）。由此可以看出，国内资产泡沫膨胀程度已经十分严峻。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/31213705/Screen-Shot-2021-05-31-at-9.35.55-PM.png)
郭先生近日爆料平安、民生、邮储、招行即将迎来破产倒闭潮。由此可窥见国内经济数据的虚假以及中共政府的财政资金紧张程度。所以，此番炒作黄金价格、吸引民资或散户“入股”，目的是实现权贵阶层的套现并将黄金资产泡沫转嫁给散户。类比2020年贵州茅台价格的一路上涨，中共官媒不但毫无质疑茅台价格的虚高，反而鼓吹价格高昂的合理性。这些的实质都是为了收割散户的民脂民膏。黄金价格热炒的背后也同样是劫“富”敛财。
