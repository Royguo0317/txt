出品：《路德社》 编辑：《路德社》精选新闻专栏组

北京、辽宁等多个社区已进入疫情战时状态，其中辽宁省大连、沈阳接连出现超级传播者。1月4日，辽宁省大连市疫情防控新闻发布会称，本次疫情传播速度快、涉及场景多、传播力强，还具有潜伏期长、情况复杂等特点。

有网传视频披露，沈阳市医生在微信警告，该市疫情异常严重，传播速度非常快，传染性非常强，怀疑已经出现变种病毒。

官方还提到，有的病例经过11次核酸检测才显出阳性，有的地区进行了6轮核酸检测仍然有阳性病例出现。

而且，辽宁省沈阳市自2020年12月20日启动应急响应，全面进入“战时状态”。目前，沈阳市已有16个疫情中风险区。

根据市民拍摄视频显示，中共军方的防疫机构已进驻沈阳市。

有评论指，疫情爆发后，卫健委统一口径玩数字游戏，无论疫情多么严重，通报确诊病例永远是个位数字，搞全民核酸检测仅仅是走形式，确诊病例被转入医疗系统，而官方通报却声称“核酸检测结果均为阴性”。

大纪元记者注意到，尽管疫情严峻，但有责任向社会提供疫情信息的沈阳市疾病预防控制中心，其官网最近一条消息却停留在2020年12月16日。而大连市疾病预防控制中心网站最“新” 一条消息则停留在2020年10月30日，显示的是关于接受社会捐赠的信息。
