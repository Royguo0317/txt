#### 文稿作者: 铜豌豆


如果可以说WHO的报告不是给中共洗地的，那么就可以说苏伊士运河根本就没有被集装箱货船堵塞
马特·里德利是一位英国记者，同时也是一位生物学家和作家，他的作品广泛的涉猎了科学、环境以及经济领域的知识。最近他在自己的推特上，上传了一段受访的片段。这段采访是里德利子爵在看过世界卫生组织WHO对于武汉病毒研究所调查报告之后，对于该报告的评价。世卫组织WHO在报告中得出结论：认为武汉病毒研究所对于SARS-CoV-2的全球大流行完全没有责任。里德利子爵对此的评价是：这是彻头彻尾的洗地行为。

经过了持续了数月的所谓的同中共的协商，WHO的调查团队的主要成员同意将WHO的调查依附于中共国科学家的工作成果，那些被中国共产党政审通过的根红苗正的中共国科学家写的报告怎么可能承认武汉病毒研究所就是此次冠状病毒全球大流行的始作俑者？WHO没做独立调差，对中共更是言听计从。

在这份300页的报告中，只有一段匆忙抛出SARS-CoV-2病毒不太可能是来自武汉病毒研究所实验室的结论外，其余20到30页的篇幅都在大讲特讲中共是如何对4万5千只野生动物进行了彻底全面的测试，都得出了冠状病毒检测呈阴性的结果。然而，报告突兀的最终得出了结论说很有可能正是野生动物中的一只，不知怎么了把病毒带到了武汉。

世界卫生组织WHO这份让人啼笑皆非的报告，毫无疑问是对于世界上每个人的侮辱，尤其对于全世界遭受了或者正在遭受中共病毒残害的家庭来说，等同于是二次戕害。
