# **美国查获14500张中共国制造的假身份证**

[!\[\]()!\[\](https://spark.adobe.com/page/p42nid0GvOxb8/images/62494ced-e1c5-4103-9f81-1406fc3d5791.png?asset_id=80a22663-8c88-464c-a4b0-345a47510ed7&amp;img_etag=%224f8389da698a5f76ed19ac568514a95f%22&amp;size=2560)](https://spark.adobe.com/page/p42nid0GvOxb8/images/62494ced-e1c5-4103-9f81-1406fc3d5791.png?asset_id=80a22663-8c88-464c-a4b0-345a47510ed7&amp;img_etag=%224f8389da698a5f76ed19ac568514a95f%22&amp;size=1024)

2020年，仅辛辛那提的海关和边境保护官员就查获了14504个虚假身份证，其中97%来自中共国和香港。

被发现的身份证件类型包括驾照、镭射贴纸或印章、层压板、社会保障卡、护照和签证。

“官员们发现，镭射贴纸、层压板和塑封往往与空白卡片一起运送，显示出制造假信用卡和身份证在黑市上出售的意图。这些货物被运往美国各地的地址，”海关和边境保护局的一份摘要指出。

[!\[\]()!\[\](https://spark.adobe.com/page/p42nid0GvOxb8/images/e470a280-fa8f-4605-a9b1-25c60412f749.png?asset_id=edea4bef-5efe-41e9-8243-a46c0a02b1ff&amp;img_etag=%2218f8101dbf76ae828262522b3d39371c%22&amp;size=2573)](https://spark.adobe.com/page/p42nid0GvOxb8/images/e470a280-fa8f-4605-a9b1-25c60412f749.png?asset_id=edea4bef-5efe-41e9-8243-a46c0a02b1ff&amp;img_etag=%2218f8101dbf76ae828262522b3d39371c%22&amp;size=1024)

被海关和边境保护局查封的康涅狄格州的身份证。

官员警告说，这些假身份证可能会被用于一系列邪恶的行为，包括 “公共利益欺诈和人口贩运”。

“假身份证被用来方便未成年人喝酒，这本身就很危险。然而，故事可能变得更加黑暗。假证件与身份盗窃、公共利益欺诈和人口贩运有关，恐怖分子利用假证件逃避旅行检查措施”。
