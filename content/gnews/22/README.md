据CNBC 1月6日的报道，在南非出现了一种新冠病毒变体，跟在英国发现的变体类似，这种病毒变体也具有更大的传染性，这引起了人们的广泛关注。

随着新冠病毒变体在更多地方出现，人们现在开始怀疑，去年以惊人速度研发出来的新冠病毒疫苗对现在出现的病毒变体比如非洲新发现的变体是否有效。虽然科学家认为英国的病毒变种不太可能会影响西方目前已经推出的疫苗的功效，但南非的病毒变种是否也是如此还有更多的不确定性。

目前科学家了解到的情况是，南非的病毒变种在刺突蛋白中携带另外两个突变（E484K和K417N），这是英国病毒变种中没有的，专家说，这可能会对新冠疫苗的效果产生影响。

WHO周二指出，目前越来越多的国家发现了新冠病毒的变体，但各国出现的病例数还是很少的。目前多个国家已经停飞了发往南非的航班以阻止病毒的传播。

本周早些时候，英国卫生部长马特·汉考克（Matt Hancock）表示，在南非发现的这种变种尤其令人担忧。前FDA负责人Scott Gottlieb博士周二对CNBC说：“南非的这种变体现在非常令人担忧，它似乎能抑制抗体药物，并且正在向其他地方扩散。”
![]()![](https://gnews.org/wp-content/uploads/2021/01/图片23-1.png)
美国疾病控制与预防中心（CDC）周日表示，科学家正在研究这些变体，以更好地理解它们如何更容易地传播，以及目前是否获准使用疫苗来保护人们免受其侵害。没有证据表明这些病毒变种会导致更严重的疾病或增加死亡风险。

随着诸如英美这些国家争先恐后地启动疫苗接种计划，专家们指出，新冠病毒变异的最大潜在后果之一就是它们“逃避自然免疫或疫苗产生的免疫的能力”。CDC强调，“没有证据表明这种情况正在发生，而且大多数专家认为，由于病毒的性质，逃逸突变体不太可能出现。”
