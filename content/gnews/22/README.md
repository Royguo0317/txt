环球网5月21日转载央视网报道，20日第五届世界智能大会在天津开幕。

2019年第三届世界智能大会有40个国家和地区参加，总计签约126个项目，投资额达1082.2亿元；2020年第四届世界智能大会因CCP病毒肆虐而采用“云签约”的形式，总计签署项目148个（其中外资项目17个，约16亿美元），投资额超900亿元。而2021年，第五届世界智能大会不但没有习近平“致信祝贺”环节，国际参与企业更是不见踪影，虽签约项目数增长为210个（其中外资项目21个），但投资总额1057亿元，不及19年水平。这说明大项目的规模和数量均有缩水。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/21204612/16x9-feature-photos-MOS-logos-83.png)
2019年第三届世界智能大会上，5G网传速率、设备连接技术新发展成为讨论的焦点，百度、阿里巴巴、腾讯三家龙头企业开启了大数据时代；2020年第四届物联网技术、光伏发电、人工智能学习分析技术成为主要展品；到了2021年，5G基站铺设依旧不及需求的二成，网速也难达预期反而深受手机用户诟病，航天领域的展品成了“独家宴”：长征五、嫦娥五、空间站模型占了大部分。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/21203722/Screen-Shot-2021-05-21-at-8.34.11-PM.png)
国际上越来越多的国家与中共削减经济科技合作，美国更开启了科技圈学术圈的脱钩（无尽边疆法案），再加上西方各国对高端芯片技术和机床工艺的严加限制，工艺滞后问题的“国芯”成了笑话。最新22nm批量生产的国芯制造水平与世界主流7nm（精尖2nm）差距悬殊。严重依赖芯片工艺的人工智能发展也成为“泡影”。中共的“闭门造车”再加上思想钳制下的“针对创新”，都决定了国内人工智能已经走上绝路，再无反转。
