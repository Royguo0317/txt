**西班牙2021年1月16日电/西喜社——**中国周六报告了130例新确诊的中共病毒感染病例，没有死亡病例。

中共国家卫生委员会说，这些确诊病例中有90例发生在河北省，该省毗邻北京，是该国最近最大的疫情发生地。

该委员会说，另外23例是在东北的黑龙江省。

有15个感染病例被确定是在国外感染的。

委员会说，中国的死亡人数为4,635人，共有88,118个确诊病例。
