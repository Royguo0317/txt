据《人类自由》4月10日的报道，mRNA疫苗可能会通过“逆向整合”来编辑你的基因，而损伤的DNA可能会传给后代。
![]()![](https://gnews.org/wp-content/uploads/2021/04/Picture1-18.png)
报道称，随着时间的推移，越来越明显的是，mRNA疫苗是对人类的致命武器，mRNA疫苗可能会对人类的生存构成威胁，这要归功于一种机制的存在，即mRNA序列可以转化为DNA，并插入到你的人类基因组中。

这意味着，mRNA疫苗有可能将你变成一个“基因工程”生物体，这会对你自身的健康以及生育能力和子孙后代造成潜在的灾难性后果。

从技术上讲，mRNA疫苗有可能通过改变人类的DNA而导致广泛的不孕症、出生缺陷或可怕的死亡。问题是，由于没有进行过长期的安全试验，没有人知道这种方式可能会影响多少人。

全世界有成千上万的医生和科学家公开反对这些实验性疫苗注射，有些人甚至称它们为“大规模杀伤性生物武器”。许多医生和科学家声称它们甚至不符合法律上对“疫苗”的定义。而制造商自己则将其描述为“生命软件”的“操作系统”。在COVID之前，它们从未被批准用于人类群体。

这些医生和科学家的声音在制药公司控制的企业媒体和大科技公司中被审查，而那些因注射而死亡和受伤的人都是支持疫苗的人，他们只能从这些被审查的消息来源获得信息，而这些信息来源是由比尔-盖茨等华尔街企业亿万富翁资助的。

当未经FDA批准的新实验性“疫苗”获得紧急使用授权以对抗现已超过一年的“大流行病”时，据报道，与2020年第一季度末FDA批准的疫苗接种导致的死亡记录人数相比，到2021年第一季度末，报告死亡的美国人数量飙升到6000%。

死亡报告的增加肯定与新的实验性COVID注射有关，但CDC和FDA的立场是，这些死亡没有一个与COVID注射有关。

中共病毒肆虐一年多的时间，太多无辜的生命逝去。中共制造病毒毒害全世界，疫苗的次生灾难频现。现在，在这个星球上的人都不得为是否接种疫苗做出艰难的选择。但是正像闫丽梦博士所说，只有确定病毒来源和原始毒株，才能研制出真正的疫苗。只有灭共，才能结束这场人类浩劫。
