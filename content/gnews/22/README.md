![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/01162447/gnews-logo.jpeg)



新浪网6月3日转载央视报道，2日中午陕西蒲城一小学46名学生出现呕吐腹泻症状。

《反食品浪费法》从2020年12月22日草案首次提出，此后又经历了4个月的修订和铺垫，终于21年4月29日正式通过。法案中不但对节约粮食等做出明确规定，更对食品安全、防腐防变提出了更高要求。然而，进入2021年以来，食品安全问题依旧多发：1月份江西萍乡156名学生出现诺如病毒感染症状，以致该校停课居家隔离3天；2月份济南历城一居民区生活用水中检出诺如病毒；4月9日吉林工商学院部分学生诺如病毒检测呈阳性……虽然相比2020年的诺如病毒感染人数有所下降，但食品安全问题依旧呈多发态势。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/03220413/Screen-Shot-2021-06-03-at-10.00.46-PM.png)
2020年可谓是诺如病毒的“爆发元年”：先是1月份广东省诺如病毒聚集性感染391例，之后因CCP病毒全国范围内爆发有所缓解，而后中小学“形式防疫”开学后全国多地诺如病毒感染事件频发；5月底吉林榆树第一中学41名学生呈现诺如病毒感染症状；9月份国内报告超过30起诺如病毒感染事件，病例近1500例；10月份，仅南宁兴宁区就有十馀所幼儿园出现诺如病毒感染病例；11月份哈尔滨继红小学116人感染，广东东莞一幼儿园63人感染，四川自贡某幼儿园50馀名学生感染……除了诺如病毒的大爆发，广西哄抢榴莲致500多人食物中毒，云南5到7月份食用野生菌事件273起，致死12人，兰州更出现了布鲁氏菌病群体事件（截至2020年12月，检出的布鲁氏菌病抗体阳性人员达1.1万人）。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/03220429/Screen-Shot-2021-06-03-at-10.00.42-PM.png)
虽然2020年在诺如病毒爆发密集期，国内的各大院校和中小学均有相应的预防和处理方案，同时8月以后的“勤俭节约”行动也使得食材处理呈细致化趋势，然而2020年下半年以来，诺如病毒感染，其他类型的食品安全事故仍多发、频发。这暴露出监管机构和学校在环境卫生、食品安全上的监管力度和预防整治措施都略显无效。新推出的《反食品浪费法》虽依旧强调食品安全性，但其着重点却在严打浪费层面。

暂且不论粮食短缺问题是否可以“以法”、“以罚”缓解，仅食品安全的行政监管和整体意识上的“淡化”状态就十分背离高效利用的初衷，更别提与食品安全息息相关的青少年生命健康了。《反食品浪费法》的“低效”利用，以罚止“费”的监管模式都暴露出中共对民众食品和健康安全的轻视甚至无视，也从侧面反映出中共逐利执法现状和剥削民众的麻木不仁。
