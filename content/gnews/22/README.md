# **Parler 首席执行官说，在其应用程序被删除后，他正面临死亡威胁**

[!\[\]()!\[\](https://spark.adobe.com/page/f1lQKGgozc04B/images/c16a2e3a-c1af-4724-81f5-757e068c761a.jpg?asset_id=d488a195-c604-4d67-bac7-5a5413608082&amp;img_etag=%22f28ff5df865912a8bda64c6c9721b0e1%22&amp;size=2560)](https://spark.adobe.com/page/f1lQKGgozc04B/images/c16a2e3a-c1af-4724-81f5-757e068c761a.jpg?asset_id=d488a195-c604-4d67-bac7-5a5413608082&amp;img_etag=%22f28ff5df865912a8bda64c6c9721b0e1%22&amp;size=1024)

“有人在威胁我的生命。”Parler首席执行官约翰·马特泽（John Matze）说。

Parler首席执行官约翰·马特泽表示，在App Store、Google Play和亚马逊网络服务(AWS)有效关闭他的社交媒体应用程序之后，他正受到死亡威胁。

“有人[正在]威胁我的生命，”马特泽周二晚上在福克斯新闻的《卡尔森今夜秀》(Tucker Carlson Tonight )节目中说，“这不仅仅是我们的公民自由。他们可以在一夜之间关闭一家价值5亿美元的公司。”

Parler是Twitter的另一种选择，在Twitter开始审查保守派人士的一些账户之后，甚至在这个社交媒体巨头上周删除川普总统的账户之前, 他们就转移到了Parler平台。

随后, 苹果和谷歌周六禁止Parler进入他们的应用商店。而据《每日传讯》报道，亚马逊宣布周日午夜后将不再托管该应用程序。

苹果公司表示，之所以将Parler从应用商店（App Store）中删除，是因为其政策不允许“暴力威胁和非法活动”，并且“Parler没有采取足够的措施来处理（解决）这些威胁人们安全的扩散问题”。

马特泽质疑，为何在没有任何警告的情况下，在同一天作出决定并强制执行。

他说：“我们是先在网上看到新闻的，这太令人震惊了。然后，在他们树立了那个榜样之后，你知道，我们收到了一封又一封的电子邮件。这几乎就像你只是在等待，谁会是下一个丢弃（抛弃）我们的人？”

马特泽表示，他将继续努力以恢复其应用程序，但他在这样做的过程中面临着各种挑战，并指出，其他服务器不愿意托管Parler。

周一，Parler起诉了亚马逊。

他说：“我们最终会回来的，因为我们不会放弃。”
