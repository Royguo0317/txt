5月24日，马胡塔在接受《卫报》（The Guardian）专访时表示，在国际舆论压力之下，新西兰政府将对中共国侵犯人权、迫害香港自由等行为采取更强硬的态度。再加上澳大利亚与中共的贸易战正越演越烈，马胡塔相信新西兰受到影响只是时间问题。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/25225559/44-6.jpg)图片来源：路透社

由于经济上依赖与中共国的贸易，与新西兰的盟友们（五眼联盟的美国、英国、澳大利亚、加拿大）相比，新西兰一直是比较亲共的国家。对中共国在新疆的种族灭绝、在香港的迫害自由，新西兰的态度一直都是“房间里的大象”（隐喻某件事虽然明显，但是集体视而不见，不做讨论）。在过去一年以来，随着中共国在新疆对维吾尔人种族灭绝的罪行被曝光，加之其持续镇压香港的民主自治，国际以及新西兰国内希望政府对中共国更强硬的呼声越来越强烈。

目前，虽然新西兰的出口贸易仍然是依赖中共国，但又是什么促使新西兰做出这样的改变呢？显然，5月13日布林肯与澳大利亚外长佩恩的会谈，以及美国承诺“不会让澳大利亚独自面对中共国的经济胁迫”让新西兰看到了机会——脱离被中共国经济控制的机会。新西兰想要不被中共国经济胁迫，必须尽快加入到以美国为首的灭共同盟。

马胡塔指出，新西兰不能漠视澳大利亚和中共国日趋激烈的对抗关系。她认为，新西兰与中共国的关系恐恶化，因此需出口多元化。这是新西兰选边的信号，显而易见，新西兰选的是盟友的一方，是加入到灭共的一方。

如今，加入到灭共同盟的民主国家越来越多了，中共国的末日也越来越近了。一切已经开始。
