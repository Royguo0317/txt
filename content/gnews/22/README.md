**简评：**

中共最近几天再次发起了对川普总统的攻击。《华盛顿邮报》最近收到中共数百万美元的贿赂，首当其冲卖力抹黑川普总统。他们故意不公布通话细节，歪曲事实，把一个以透明的方式试图了解选票欺诈真相的通话说成是川普总统电话施压欲获得更多选票，试图更改选举结果，涉嫌违宪，进而要再次弹劾川普总统。这一颠倒是非、转移焦点的方式得到了中共真传。而这几天中共国内的媒体也同步大篇幅报道这一消息，配合得天衣无缝。

因为知道华盛顿在接下来的24到48小时之间会逐步公布细节，这些肮脏内幕会很快曝光，川普一定会连任，到时所有关联此案的人和组织一定会遭到法律制裁。中共知道来日不多，一定会遭到清算，所以发动媒体和在美代理人，同时跳出来孤注一掷地想把川普拿下。而这又何尝不是川普团队的一次钓鱼机会呢？说不定美国正等着中共出手。从《国家脉动》的这篇文章中可看出，川普总统和他的团队已经掌握了大部分有关选举欺诈的核心消息，包括深陷其中的两党政客以及媒体。而这次事件，使各阵营站队暴露得更明显了，中共真的要搬起石头砸自己的脚了。

**原文翻译：**

**川普的电话实际上揭示了一位总统想要以透明的方式深入了解细节，而建制派共和党人轻蔑、不情愿、拒绝透明**

**“我不知道那个……我面前没有……我们正在调查……”**

[!\[\]()!\[\](https://spark.adobe.com/page/gECzlFY5V4VCY/images/fe258a9b-2857-477a-b51a-417037d4532d.jpg?asset_id=e90b1142-0e03-4e86-b964-38bdcc5aed1c&amp;img_etag=%22f4b65736bf5407fcae5e19b51e435f88%22&amp;size=1024)](https://spark.adobe.com/page/gECzlFY5V4VCY/images/fe258a9b-2857-477a-b51a-417037d4532d.jpg?asset_id=e90b1142-0e03-4e86-b964-38bdcc5aed1c&amp;img_etag=%22f4b65736bf5407fcae5e19b51e435f88%22&amp;size=1024)川普：我需要找到12000张选票，我说过很多次了

这些《华盛顿邮报》周日泄露的含糊不清、不置可否的电话通话内容不是川普总统说的 — 《华盛顿邮报》最近从中国共产党手中拿到了数百万美元 — 它们是建制派的共和党人州务卿布拉德·拉芬斯珀格（Brad Raffensperger）、瑞安·杰米尼（Ryan Germany）和副州务卿约旦·福克斯（Jordan Fuchs）在1月2日的电话会议上的发言。

尽管来自《华盛顿邮报》的党派人士表示 — 这通电话以某种方式反映了川普要求其共和党同事投票的意图 — 但总统实际上并没有这样做。在整个电话会议中，总统明确表示，他的呼吁是选举透明度，全面和透明的审计以及公众参与。总统一点都没有像传统媒体所描绘的那样，暗示过他想要谋求或更改选票。他甚至主动提出回避谈话的部分内容，最后要求“真相……就这么简单”。

您可以阅读完整的笔录，并收听本文下方的整个电话通话。

实际上，总统是通过详述细节开始通话的，而电话的另一方（即建制派共和党人）从未谈过这些细节。川普指出：

![]()![](https://spark.adobe.com/page/gECzlFY5V4VCY/images/9f4b4b73-2f6e-438c-84c8-c1fa15b50db6.jpg?asset_id=e9c405e0-b732-4ea3-aa35-6223593bce11&amp;img_etag=%224f056b7192b767c9068b0c9940d4598a%22&amp;size=1024)布拉德·拉芬斯珀格：总统先生，您遇到的问题是您得到的数据是错误的

- “我们至少有2到3个地方 — 任何一处有250张到300000张的选票被神秘地投了进来。其中大部分与富通县有关，该县尚未经过检查。我们认为，如果检查签名 — 对富通县的签名进行真实检查 — 你至少会发现数十万冒充选民的伪造签名。”
- 他继续说：“据我们掌握，我相信，大约有4502位投票者参加了投票，但他们不在选民登记名单上，所以有4502位投票者，不在他们必须注册的选民登记册上。你们有18325个空缺地址的投票者。地址栏是空的，因此不予计数。那是18325票。”
- 总统接着说：“你们有外州投票者。他们在佐治亚州投票，但来自州外，有4925票。你们已将缺席选票寄出到空白地址，他们是已发送到空白地址的缺席选票。他们上面没有任何地址信息，那是2326票。”


细节水平实际上是惊人的，特别是媒体继续指责总统而不是关心细节。

而拉芬斯伯格团队的反击甚至没有特别猛烈，这大概就是为什么他们泄漏了电话而不是继续公布讨论和要求的细节。通话中有几个要点反映出佐治亚州建制派共和党领导人的恶意：

**1.富通县**

总统和他的团队反复提出在富通县的涉嫌违法行为，特别指控拉芬斯伯格和团队转向了科布县，而不是处理人口稠密的富通县的问题。

川普总统在这里说：“你把我们送去我们不想进入的科布县。你说它将向公众开放。所以我们让我们的专家在那儿，可你们不允许他们进入房间。 但是我们不想去科布县，我们想去富通县，而你们不允许我们去。现在，我们为什么不查签名呢？为什么不能向公众开放？”

对此，律师瑞安·杰米尼简单地回答：“我们之所以选择科布县，是因为这是唯一有证据表明签名验证未正确完成的县。”

不仅承认该州有不当行为，而且还承认有选择地采用严格的标准，并拒绝让总统团队进入该州人口最多的县。

**2.“同意令”**

这与大选前共和党和民主党签署的协议有关，以及川普总统为何说极左翼的佐治亚州激进主义者史黛西·艾布拉姆斯（Stacey Abrams）通过让他们签署违宪协议来“击败”共和党。该协议本身就是一个胡乱解释的文字沙拉（请自己阅读），以说明选票上的签名匹配。如果没有一方打算进行选举欺诈，人们可能会辩称不需要这种“协议”。

拉芬斯伯格和杰米尼没有直接解答这个问题，而是开始挑剔“同意令”一词，该词已在国家、州和地方媒体中多次用于描述该协议，甚至是反对川普的律师也使用了这个词。

拉芬斯伯格在整个交流过程中只是简单地说：“这是一个和解协议。” 他的同事瑞安·杰米尼（Ryan Germany）也避开了问题的症结：“我不信。我不信。我不相信[这是一个同意令]，而我手上也没有这个。”

**3. 佐治亚律师拒绝进一步公开。**

通话快结束时，有人听到参谋长马克·梅多斯（Mark Meadows）说：“听起来我们得到两个不同方面承认，我们可以研究这些方面，我认为我们可以在接下来的24至48个小时中做这个，继续调和这一点，以便我们可以查看两方面的声明，并确保可以访问州务卿的数据，以验证他们的声明是有效还是无效。对吗？”

杰米尼立即打趣道：“不，那不是我说的，”并试图打消梅多斯进一步调查佐治亚的总投票数的念头。

当总统的律师库尔特·希尔伯特（Kurt Hilbert）提议“州务卿可以代理总统的律师，这样我们就可以访问这些信息和私人信息，而你不会有任何违规？”时，总统正确地推辞以避免知道更多，他说：“我不想知道是谁。你们可以非常机密地做到这一点。你们可以签署保密协议。没关系。我不需要知道名字。”

**4.鲁比·弗里曼**

也许这不是拉芬斯伯格团队的错，但是《华盛顿邮报》的音频本来是指佐治亚州选民柜台和左翼活动家鲁比·弗里曼和她的女儿莎耶·莫斯（Shaye Moss）。尽管《国家脉动》没有指出就是这两个人，但令人困惑的是，《华盛顿邮报》为何愿意隐去两人的名字，尤其是它们在互联网上的知名度已经变得很高的情况下。

这些名字被《华盛顿邮报》隐去了18次。

![]()![](https://spark.adobe.com/page/gECzlFY5V4VCY/images/669a1d86-08c5-42c8-bf0f-d6adafd6ee20.jpg?asset_id=07440a88-5167-4cfa-b32f-72dab99ccd24&amp;img_etag=%22ea3676cd30f5c7d121922e75a43b376c%22&amp;size=1024)川普：我们的回击才刚刚开始

**5.“我们只想知道真相。”**

通话结束时，川普总统反复说：“我们只想要知道真相。”

这次通话并没有像《华盛顿邮报》所报道的那样充满“压力”，而是总统特意在62分钟的时间里审核了细节、障碍，并不断确保他在寻找准确性、透明性和公开性。

律师克莱塔·米歇尔（Cleta Mitchell）可能在电话中说得最好，她说：“……你可以继续告诉我们并公开声明对此事进行了调查，而且看不到任何异常。但是我们对此一无所知。我们所知道的就是你告诉我们的。我不明白的是，为什么探索真相，比较数字不符合每个人的最大利益？如果你说的话，因为……试图了解真相，因为我们没有任何方法可以确认你告诉我们的内容。你告诉我们你们在州立农场竞技场进行了调查。我没有看到任何报告。我从未见过调查报告。我不知道调查了什么。我一直参与其中却什么也不知道。那只是25个类别之一。”

对于像亚历山德里亚·奥卡西奥·科尔特斯（Alexandria Ocasio-Cortez）这样的民主党人来说，因为这通电话呼吁再次弹劾的努力既可以预见又令人作呕。在回应媒体的置评请求之前，“亚历山德里亚·奥卡西奥·科尔特斯”很可能甚至没有听过这个电话。

但这就是今天的政治和媒体 — 不断狂热地消弱宪法规范，同时将矛头指向另一个方向。

真正听到此次通话的任何人都将听到川普总统讲得详尽而直率。可以理解的是，川普当然感到沮丧，他发出了一些他典型的纽约商人式的咆哮。但更重要的是：他是以开放和透明的方式对话，而另一边根本没有。

这告诉了我们需要知道的一切。
