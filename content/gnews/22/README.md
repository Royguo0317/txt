最近，墙内有人因为抗议耐克不用新疆棉花，把自己的耐克鞋给烧了。

不得不说，这很蠢。鞋子是你自己的钱买的，烧了它对别人没有任何影响，而你自己却损失了一双鞋。

还有人跑到耐克店的门口用中文大骂耐克“傻X”。要骂，就应该用英语骂。但他们跑到外国品牌门口用中文骂人，是欺负老板不懂中文吗。

其实，烧鞋和骂街是一种作秀，不管这背后有没有中共的操纵，其首要目的是煽动社会上的狂热民族主义。如果一个人真的爱国，就应该把不需要的鞋捐给贫困人群，中共国有六亿人的月收入在一千人民币以下，他们穿不起耐克鞋。

这些愚蠢而粗鲁的举动，和1900年的义和团风格很像。

当年的慈禧太后不自量力地对十一国宣战，对义和团采取“抚而用之”的政策，利用民间组织广杀洋人，下诏褒奖义和团为“义民”，并拿出10万两银子、2万担米作为义和团的经费，要义和团听命于朝廷，从而进一步欺骗和控制义和团，作为朝廷的“武器”。

现在，中共政权同样不自量力，用CCP病毒作为超限生物武器对全世界宣战，在新疆犯下的种族灭绝罪行暴露之后，利用不明真相的“小粉红”与世界正义力量对抗，企图绑架十四亿中国人作为他们的“炮灰”、“肉盾”。

历史总是在轮回中。

义和团运动之后，清朝统治随之动摇，大清国陷入一片混乱。同时列强趁着大清国混乱之际扩张了势力，北京沦陷了……后期，慈禧将怒火全部洒向义和团，绞杀义和拳民。

如果此次大规模的抵制运动没有成功，中共是否也会像慈禧一样，将气全部撒在“小粉红”身上呢？

解放前，中共曾经认为义和团“排外”、“灭洋”是错误的：“义和团的思想由最原始最落后的反动的宗教思想和进步的反对异民族压迫的思想两种成份组成的，反映农民知识的狭隘性。”

但文革突然掀起了颂扬义和团运动的高潮，称其为“近代中国历史上的一次伟大的、反封建的革命群众运动”。

出现这种历史的反复，是因为中共发现，义和团式的民族主义有利于巩固政权、永保江山。所以，共产党一直将其运用到了今天。
