# 重要事件

- **04.21 路透社 [美国会议员加大了两党对抗中国的努力](https://www.reuters.com/world/asia-pacific/us-lawmakers-look-advance-sweeping-bid-counter-china-2021-04-21/)**


美国参议院两党反制中国的动议在周三得到了推动，参议院委员会以压倒性多数支持了一项法案，敦促北京就人权和经济竞争问题展开辩论，其他议员也提出了一项寻求数十亿美元用于技术研究的措施。

参议院外交关系委员会在21-1之前支持《 2021年战略竞争法》，将该法案发送给拥有100名成员的参议院审议，尽管委员会成员表示需要采取更多措施来反击北京。

委员会对该法案进行了数十项修正。美国宗教自由委员会也建议由美国官员而非运动员抵制2022年北京奥运会。

“通过两党压倒性的投票，《战略竞争法》成为我们希望成为国家一系列立法活动的第一步，以最终迎接中国在权力，政治，外交，经济，创新，军事和军事等各个方面的挑战。甚至是文化”，参议院专家组民主党主席鲍勃·梅嫩德斯（Bob Menendez）说。

参议院长达280页的法案通过诸如增加国际发展资金以及与盟国和国际组织合作等努力来应对与中国的竞争。它推动了人道主义和民主价值观，例如对对待少数穆斯林维吾尔人的制裁和支持香港的民主。

该法案强调需要“优先考虑为实现美国在印太地区的政治目标所必需的军事投资”。它支持该地区与安全相关的资金急剧增加以及与台湾的紧密联系。

它将扩大美国外国投资委员会（CFIUS）的范围，该委员会将审查潜在的国家安全风险的金融交易。美国大学担心该法案的条款要求CFIUS审查一些中国的助学金和合同。

埃里克·塞耶斯（Eric Sayers）在倾向保守的美国企业研究所（American Enterprise Institute）研究亚太地区的技术政策和美国国防战略，他称该立法是应对中国所面临挑战的重要组成部分。

塞耶斯说，拟议的支出以早先的立法为基础，“通过投资于我们的国内竞争力并在我们的供应链中建立新的适应力，在与中国的技术竞争中迈出了大胆的一步。”

- **04.21 GatewayPundit  [中共国成为美国另一次网络攻击的幕后黑手](https://www.thegatewaypundit.com/2021/04/china-behind-another-cyber-attack-us-hacked-us-computers-networks-accessing-systems-remotely/)**

这种黑客的工作原理是闯入Pulse Secure，该程序经常被企业用来使员工远程连接到他们的办公室。该公司周二宣布，用户将如何检查自己是否受到了影响，但表示，要防止用户遭受风险的软件更新要等到五月份才能发布。

该运动是近几个月来针对美国的第三次针对美国的网络间谍活动，其活动严峻，强调已经紧张的网络安全员工队伍。美国政府在一月份指控俄罗斯通过德克萨斯州软件公司SolarWinds入侵了九家政府机构，该公司被美国企业和政府机构广泛使用。今年三月，微软指责中国启动了“免费服务”，通过微软Exchange电子邮件程序，数十名不同的黑客闯入了世界各地的组织。

- **04.21 WashingtonExaminer [司法部起诉伊利诺伊州教授在获得美国政府资助的同时在中国秘密工作](https://www.washingtonexaminer.com/news/doj-indicts-illinois-professor-secretly-working-china-while-getting-us-government-grants)**


司法部以两项电汇欺诈罪和一项虚假陈述罪名起诉了一名中国出生的伊利诺伊州大学教授，罪名是为在中国政府所属大学中秘密工作，并在申请并获得美国政府“国家科学”计划的151,099美元资助时隐瞒了这些联系。

现年59岁的肖晓波周三被指控“隐瞒”获得他的联邦拨款，“隐瞒了他从中国政府和中国大学获得的支持”，当时他在南伊利诺伊大学卡本代尔分校担任数学教授。自2000年以来一直受雇。

起诉书说，2019年3月，在他的美国资助申请待决期间，肖向中国自然科学基金会提交了另一项建议，以深圳大学的雇员身份申请资金，并且“没有向NSF透露新的中国建议。”美国政府机构要求申请人披露所有“组织关系”和“当前和即将得到的支持”，包括“外国政府机构”。

美国已指控许多研究人员隐瞒与中国的关系，司法部在2020年11月表示，在过去一年中，美国对10名在职学者进行了欺诈，虚假陈述，税收，走私及其他与中国有关的指控。

法院文件称，肖于2018年4月与深圳大学签署了一份合同，“取代了先前的合同。”这所中文学校“将肖的月薪存入了平安银行的一个账户。”当美国政府代表于2019年4月就他的美国赠款提案与他联系以确保他透露任何“全球消息来源”时，肖晓波错误地回答说：“我没有其他赠款或未决的提案。”

如果发现某人犯有电汇欺诈罪，肖可能会面临最高20年的监禁，如果他犯有虚假陈述，他可能会面临长达五年的监禁。美国司法部说，这是其“中国倡议”的一部分，“这是一项广泛，多方面的努力，旨在应对中国的国家安全威胁并维护美国的知识产权。”
