**西班牙2021年5月13日电/西喜社——**

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/13084952/angry-face-700x420-1.jpg)Artur Pawlowski牧师在2021年4月3日将警察赶赴加拿大卡尔加里的教堂，以COVID-19检出警察后的片刻

加拿大牧师亚瑟·帕洛夫斯基（Arthur Pawlowski）周六被捕，因举行教堂礼拜，被涉嫌在大流行期间反公共卫生命令。周一晚上从监狱获释后，他说他觉得自己像是住在香港。

“我几天前刚从香港事件中醒来，”帕沃夫斯基在获释后数小时接受《新闻快报》（Newsmax）采访时说。 “我的意思是，我以为我移民到了我们心爱的加拿大，但我却像还是在香港，全力的打压中。”

过去几年，香港人一直在争取自由，但遭到中共支持的地方政府的残酷镇压，许多激进分子最近被捕。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/13090022/%E5%9B%BE%E7%89%8724-4.png)

“这简直是疯狂——逮捕牧师，关闭教堂。 疯狂至极！” 帕沃夫斯基补充道。

帕沃夫斯基和他兄弟星期六在高速公路上被特警队逮捕。 他和他的兄弟都在周一晚上被释放。

卡尔加里警察局星期六在一份声明中说：“今天早些时候，警方逮捕了两名教堂组织者，他们违反了阿尔伯塔省卫生服务局（AHS）的一项新的涉及强制性遵守聚会的公共卫生法院命令。”

数周前，帕沃夫斯基将警察轰出教堂，并将他们比作是纳粹盖世太保，引起了媒体的关注。

“我和我兄弟已经成为政治犯， 我们被拘留，像一块肉一样扔在警车上，我们被拒绝与律师联系长达24小时。” 帕沃夫斯基继续说道。

“这太恐怖了。 这是历史的重演。” 帕沃夫斯基说。 “我在铁幕下长大。 我见过警察滥用职权抓人，你可能无缘无故会在凌晨五点被破门逮捕。他们会因为你听了欧洲电台的广播就有理由折磨你，逮捕你，并将你关进监狱五年。”

帕沃夫斯基在1990年代从波兰移民到加拿大。 他和他的兄弟举行的聚会，拒绝官员进入位于卡尔加里多佛的教堂。 他还因举行教堂礼拜，这就违反了公共卫生命令而屡次被罚款。

“我逃脱了共产主义，我逃离了波兰，因为我想来到一个自由的国家。在这里，我们再次重复着相同的错误，相同的历史。 我必须站起来，为自己的权利而奋斗，为自由想来敬拜上帝的人们开放教堂，而不是为了作恶。”

艾伯塔省于5月4日宣布了新的强制性健康限制措施，以“帮助阻止COVID-19的传播并保护卫生系统。”

这些措施包括新的限制，禁止所有公共或私人室内聚会；户外聚会的人数上限为5人，在感染人数少的地区上限为10人； 对于礼拜场所，在情况较轻的地方，最多15个人参加，或区域消防法规占用率的15％。

卡尔加里市宣布，当务之急将继续是执行公共卫生命令。
