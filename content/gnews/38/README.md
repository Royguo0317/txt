05月15日，伊朗外交部长穆罕默德·贾瓦德·扎里夫（Mohammad Javad Zarif）拒绝访问维也纳，因为奥地利外交部大楼上悬挂着以色列国旗。奥地利广播电台《ORF.at》援引奥地利外交部的话报道了这一消息。

据该部称，扎里夫原计划于5月15日星期六抵达维也纳，与奥地利外交部长亚历山大·沙伦堡（Alexander Schallenberg）进行会谈。

奥地利外交部说“我们对此表示遗憾并注意到了这一点。”

前一天，奥地利外交部在推特上发布了一张照片，显示该部大楼有一面以色列国旗。这张照片附有一条推特，表达了对以色列的声援。

“我们声援我们的以色列伙伴。这就是为什么今天我们在维也纳的外交部大楼上升起了国旗。我们对以色列安全的支持是坚定不移的。”消息说。

作为回应，伊朗副外长阿巴斯·阿拉格奇（Abbas Araghchi）对维也纳的立场表示不满，并表示支持巴勒斯坦。

“看到一个占领政权的旗帜是令人震惊和痛苦的，该政权在几天之内在维也纳的政府办公室上空残暴地杀害了数十名无辜平民，包括许多儿童。我们与巴勒斯坦站在一起。”他写道。

5月10日，在以色列法院决定驱逐东耶路撒冷的几个阿拉伯家庭后，巴以冲突升级。阿拉伯民众因此爆发了抗议活动。随后，哈马斯从加沙地带向以色列发射了火箭弹。

以色列的回应是发起了“围城行动”，这是一次开始轰炸加沙地带的军事行动。据以色列军队称，两名哈马斯高级军事情报官员在这次行动中被消灭。
