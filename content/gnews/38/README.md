## 【聚焦】左媒收视率骤降，川普总统人气依旧

#### 作者/编辑：翼族
![]()![](https://gnews.org/wp-content/uploads/2021/03/image0-100.jpg)
图片来源：墨尔本雅典娜农场设计组

根据“尼尔森媒体调查”（Nielsen Media Research）近日公布的数据，MSNBC电视3月份15日至19日的这个工作周的收视率骤降，是自2019年12月30日以来的最低收视率。【注：尼尔森是全球市场调查领域的领导企业，特别是在收视率的调查领域，有约4万名员工。】

作为MSNBC有线电视较受欢迎的一档政治采访节目，乔伊·瑞德（Joy Reid）的“The ReidOut”节目经历了有史以来最糟糕的一周，创下了其每周观众总数的最低记录，比2月份的8日至12日的这一工作周减少了61％的观众，而当时观看该电视节目的观众曾达到一个高峰，原因是川普总统的第二次被弹劾。而自从川普总统离开白宫之后，自由派主持人失去了攻击的目标，包括MSNBC在内的左媒们的收视率就开始一路下跌。

回顾MSNBC有线电视的发展历史，可以很明显地看到，它的黄金时段节目在川普担任总统期间得到蓬勃的发展，尤其是在2016年总统大选期间，MSNBC重点关注对川普总统“通俄门”的指控，并借此机会收获颇丰。但是随着川普总统的离任以及第二次被弹劾的结束，MSNBC电视节目人气不再，观众们开始迅速远离。与川普总统被弹劾的一周相比，上周MSNBC黄金时段节目的收视率整体下降了49％，而由于其网络观看主要面向年轻群体，其收视率下降更多，达到了57％。

同时MSNBC的两档著名新闻节目——“瑞秋·梅道儿秀（The Rachel Maddow Show）”和“与克里斯·海耶斯一起（All In with Chris Hayes）”——的收视率也骤降至2019年9月16日以来的最低。另外查克·托德（Chuck Todd）的“每日媒体见（MTP Daily）”节目在预演中跌至五年来的最低，其表现是自2016年4月11日以来最不受广告商青睐的一档电视节目。

MSNBC电视的旗舰级王牌节目——早间新闻“早上好乔（Morning Joe）”——也经历了自2020年2月24日以来最糟的一周，其收视率连续第四周被福克斯有线电视的同类型节目“福克斯和朋友们（FOX＆Friends）”超越。截至上周结束，福克斯新闻在全天观看数和黄金时段观看数上，均比MSNBC高出了两位数的百分比。从3月15日到21日，福克斯新闻的平均黄金时段观看者为230万，而MSNBC的平均观看数为170万；福克斯电视的全天观看平均数为120万，而MSNBC只有99.7万。

与此同时，离开白宫的川普总统却依然保持着强大的人气。同样根据“尼尔森媒体调查”的数据，2月22日的这一周川普总统参加了美国保守政治行动会议（CPAC），福克斯转播的川普总统的演讲排在了所有有线电视节目收视率的首位。
![]()![](https://gnews.org/wp-content/uploads/2021/03/117-13.jpg)
图片来源：[Nielsen Media Research](https://www.nielsen.com/us/en/top-ten/)
