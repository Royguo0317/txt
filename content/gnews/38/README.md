#### **《七芒星下斩红魔》**

作者:花岗岩

不经意间

多看了您一眼,

人生从此

彻底改变,

已经麻木的生命

被注入了

信仰的灵魂，

生命从此

变得有意义，

因为

要解救更多的生命。

一三五七九，

3F和蓝金⻩，

红魔嚣张，

向世界开出毒方，

欲将世界拿下，

甚是狂癫。

六四闪电

绽放新中国联邦。

蓝色海洋

驱赶邪灵恶党。

正道之光

唤醒华夏沧桑，

七芒星旗

于我神州⻜扬。

上天的安排，

中华儿女的心愿，

拔剑斩红魔，

我们就是

四十九颗七芒。

拯救世界

唯我新中国联邦！

- 为纪念六四新中国联邦成立一周年而作
