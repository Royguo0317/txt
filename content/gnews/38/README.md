你的笑容宛若秋天的红苹果林
清香，层层叠叠
羞涩。浑圆而饱满
钟情于土地，垂首致意

采摘苹果的女子
那儿，飘来歌吟
孩子们在树下嘻笑、奔跑

红苹果，绿树林
紧紧拥抱着

天空阴云翻滚，风暴在酝酿
