**川普总统任期内发起美史上第一个与中共全面竞争的政策**

如小标题所示，同时也应证郭先生直播内容，川普总统的确是美国政治发展上，自中共崛起后，不论中共内部利用如何手段逃避人权议题的国际制裁，第一位明面的与中共在军事、经济、科技、外交上采取实质制裁制策的总统。

**军事**

川普总统任期内，大力提升美军在东亚的军事力量，对于台湾军事抗共有很大帮助。但截至目前，东亚美军基地集中在日本两处。距离台湾约200公里，面对台湾抵御中共第一波突击军事攻击，恐无实质帮助。

又若中共利用远端打击武器，摧毁两大驻日美军基地，那美军欲援驰台湾，必须从距离台湾五百公里远的「关岛基地」派出战机。惟美军现主力战机航程也差不多为五百公里，飞到之后必须即时补充油料才可成为即战力。

即便有空中加油机、空中加油道，但在开战之时，中共军机势必会对已知的空中走廊进行封锁。综上，川普政府任期内加强东亚军事力量，对维持区域和平确实有大助益，但仍须更多配套措施，才会趋于完善。

**经济**

自二战以来，川普总统对中共课予最严格的关税政策，对于利用便宜人力发展经济的中共，即便现在已经从农业社会发展到工业社会，关税始终是限制经济发展最好的武器。因此，高关税、反托拉斯法的制裁，都实质的重击了中共经济。

此外，还有众所周知的美元脱勾的政策，使中共无法轻易取得美元。美元是现今国际贸易主要使用的货币，约占90%。中共无法换汇，即代表无法轻易取得美元，直接影响到中共的国际贸易。

对此，近日中共突击宣布封锁台湾凤梨。以个人观点，中共封锁、制裁台湾商品，目的通常是为实质的造成台湾经济损失。然而，台湾凤梨出口仅占约一成，即便中共突击制裁，对台湾凤梨经济影响也不会过大，可以轻松的以加工品，诸如凤梨酥、水果制品等，予以有效解决。

因此，对于中共突击的封锁台湾凤梨，着实让人怀疑中共经济已经衰败到一个无法想像地步？ （仅个人猜想）

**川普政府、拜登政府，皆强调强硬反共，但方式略有不同**

拜登政府，先前经郭先生爆料，即便是与中共勾兑的拜登，在上任之后也将会持续川普政府的反共政策，且会更为强硬。

本日，从拜登政府发布的首份国安战略指南来看，拜登政府的确承袭川普政府的反共态度。

然而相对于川普政府的双边外交主义，拜登政府转回原本的多边外交主义。

拜登政府尝试复制二战后，美国利用强大军事力量、金融力量，在世界建立的民主联盟。利用集合的力量打击中共。

相反的，川普政府再执政后，非常致力于退出美国已签署的多边国际协定，希望重新签署双边协定，将国与国之间的交流，可以再分别的协定当中得到利益最大化。

（比如：原美日澳签署三边国际协定，寻求三国特定领域的协力发展。川普政府则希望转采美日、美澳两个协定方式达成同样目的。）
