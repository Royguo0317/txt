**西班牙2021年1月2日电/西喜社—**法律要求 “对联邦调查局的数据进行详细分析，这些数据来自于对入侵美国限制领空的不明空中现象数据的调查”。

川普总统上周日签署的2.3万亿美元的中共病毒救济和政府资金法案中，塞进了其他一些奇怪的东西，比如五角大楼和间谍机构倒计时180天公布他们所知道的关于UFO的一切。

福克斯新闻报道说：”这项规定很少受到关注，部分原因是它并没有被列入长达5593页的立法文本中，而是作为年度情报授权法案所附的 “委员会意见”，被卷进了这项庞大的法案中”。

该意见来自参议院情报委员会—参议员马尔科·卢比奥（Marco Rubio）是该委员会的代理主席—指示国家情报局局长、国防部长 “以及其他这些机构的负责人””在该法案颁布之日起180天内，向国会情报委员会和武装服务委员会提交一份关于不明空中现象的报告”。

意见还说，相关机构需要报告 “观察到的未被识别的空中物体”，包括 “详细分析通过以下方式收集到的未识别现象数据：a.地理空间情报；b.信号情报；c.人类情报；d.测量和信号情报”。

报告还必须包含 “对联邦调查局的数据进行详细分析，这些数据来自于对入侵美国限制领空的不明空中现象数据的调查……并评估这种不明空中现象活动是否可能归咎于一个或多个外国对手。”

“我们知道，参议院情报特别委员会关于2021财年《情报授权法》的报告中包括一项要求，即要求国家情报总监与国防部长协商，在颁布后180天内提交一份关于不明空中现象（UAPs）的报告。”国防部发言人苏·高夫告诉《纽约邮报》。

4月，国防部解密了三段著名的UFO视频，承认它们是由美国海军飞行员拍摄的。

“国防部已授权发布三段未解密的海军视频，其中一段拍摄于2004年11月，另外两段拍摄于2015年1月，这些视频在2007年和2017年未经授权发布后，一直在公共领域流传。”国防部在一份声明中说。

“美国海军此前承认，这些在公共领域流传的视频确实是海军视频。经过彻底审查，该部门确定，授权发布这些非机密视频并没有揭示任何敏感的能力或系统，也不会影响后续对不明空中现象入侵军事空域的任何调查。国防部公布这些视频，是为了澄清公众对流传的镜头是否真实，或者视频是否有更多内容的误解。视频中观察到的空中现象仍被定性为’未识别’。

“发布的视频可以在海军航空系统司令部FOIA阅览室找到。”
