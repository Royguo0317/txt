据《华盛顿邮报》02月07日报道，白宫新闻发言人詹·普萨基宣布，向美国前总统川普提供机密数据的问题将由情报官员决定。

这是白宫对乔·拜登提出的剥夺前任情报数据访问权的建议做出了回应。2月6日拜登总统表示，不应该允许川普获得机密简报（前总统通常有权获得这些信息）。

“他们对他有什么价值？除了他可能会吹毛求疵，说些什么之外，他根本没有什么影响力。” 拜登解释了他的立场，称川普的行为不可预测。

“总统对前总统川普获得机密情报信息表示担忧。但他对自己的情报团队也有很深的信任，如果前总统川普提出要求，他可以决定具体提供何种形式的情报信息。”普萨基说。

传统上，美国前总统会收到情报官员的简明报告。同时，现任总统得到了更详细的信息。前任总统如果在海外代表华盛顿，就会得到更全面的机密信息，这也是他们准备工作的需要。

早些时候，普萨基在简报中表示，向川普提供机密数据的问题“正在考虑中”。前总统本人在2016年上任之初曾表示，他不太信任情报。同时，美国情报官员在川普上任后承认，他们对将国家最严密的机密交付给新任总统感到忧虑。
