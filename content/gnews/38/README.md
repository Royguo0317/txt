## 中共式割韭菜就是这么霸道无理心脏支架从万元降至798又回万元
` GM08` [轉載自GNews](https://gnews.org/zh-hans/557070/)

作者：文茗

11月5日，在天津举办了高值医用耗材全国集采采购大会。此次全国集采的产品为冠状动脉药物洗脱支架系统（下称“冠脉支架”）。天津市医药采购中心官网当日公示的拟中选结果显示，旗下吉威医疗、易生科技、金瑞凯利等在内的10家企业产品拟中标。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/13231412/unnamed2.png)
据此前公示的全国带量采购文件，此次冠脉支架集采首年意向采购总量为107.47万个，由联盟地区各医疗机构报送采购总需求的80%累加得出，采购周期为2年。本次集采共有12家企业参与投标，涉及27个冠脉支架产品。据悉，27款冠脉支架中，共26款产品报价，Micell Technologies Ireland Limited未报价。其中，12款产品的报价低于1000元，仅4款产品报价高于文件中参考价2850元。

经本次采集，冠脉支架市场中，国产支架价格为0.9万元至1.1万元，进口支架为1.5万元至1.9万元。本次带量采购拟中选产品价格为469元至798元不等，场上报出最低价的是山东吉威，原价1.3万多元的支架，山东吉威开出了最低价只要469元。（这也印证了支架的生产成本很低的事实。正如上市公司赛诺医疗曾披露过国内主要支架企业的毛利率水平：乐普医疗79.28%、柏盛国际83.46%、微创医疗70.21%。一款出厂价3000元左右的支架，实际生产成本也就不到600元。）与2019年相比，价格平均降幅为90%以上。

作为高值医用耗材的首个全国集采品种，冠脉支架集采正式拉开了高值医用耗材中共国家垄断集采的序幕。A股市场也对此作出了反应。或许受此次集采结果影响，相关部分医疗器械公司股价有所下滑，截至5日收盘，下跌3.72%，乐普医疗下跌2.75%，微创医疗下跌2.3%。

而据媒体报道，即冠脉支架之后，经会有更多的高值医用耗材纳入带量采购。不完全统计，全国范围内，目前已有江苏、安徽、山东、重庆、山西、辽宁、甘肃、湖南、云南、海南等省份或省级联盟开展。随着区域的不断扩大，带量采购品种不断增加。其中，江苏、福建等省份已逐步形成制度化的招采。安徽省第二批高值医用耗材骨科关节类、心脏起搏器类集中带量采购已完成，其中骨科关节类240个产品平均降幅81.97%，心脏起搏器类82个产品平均降幅46.75%。很明显中共对于这类医疗产品正在形成国家级垄断。

冠脉支架从材质上可分为钴铬合金或铂铬合金，载药种类为雷帕霉素及其衍生物。在国内的价格动则以万元起步，而由于技术的进步，实际国际价格换算人民币也不过区区几百元而已。即使加上所有灰色和实际成本，一个出厂价3000元的心脏支架，卖到医院的价格为1.15万元，医院售给患者翻倍至2.7万元；而同类的进口支架，到岸价6000元，到医院就成了2.3万元，医院售出则在3.8万元上下。冠脉支架等进口医药产品的这种暴利已是公开的秘密。这种暴利往往却被中共那些权贵家族垄断，百姓也只能恨的牙痒痒，却很无奈。正如网上的一个段子说的是“昨天一个支架可以买6瓶茅台，今天一瓶茅台可以买6个支架”。看似一段笑话，却道出了老百姓的无奈。

国家医疗保障局医药价格和招标采购司司长钟东波表示：“不下调支付总额，剩下的给医疗机构。现在把成本给你降下来了，空间全给你，医疗机构的话，通过合理考核，给咱们这些心脏科的大夫会划点绩效的这个鼓励。”
