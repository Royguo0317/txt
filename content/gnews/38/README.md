由于中共对台海、南海等西太平洋地区的威胁日益严重，美国在其围堵中共的印太战略中更加倚重日本。日本是商品出口大国，因而也是油气、原材料进口大国，从中东、印度洋、马六甲海峡到南海、台海、东海的海洋航线，是岛国日本的运输生命线，而这条航线正在遭受中共不断加剧的威胁，日本的小学生都知道，一旦中共封锁这条航线，日中就必有一战。
![]()![](https://www.zegrahm.com/sites/zeg/files/styles/d2_scale_width_2400/public/japh1701_print_1.jpg?t=1CE93u&amp;itok=80pg2yM2)*图片来源：zegrahm.com*
在“中共威胁”日增的背景下，日本首相安倍晋三在2015年推动以解禁集体自卫权为核心内容的新安保法并在参众两院通过，成为正式法律。这部新安保法摆脱了二战后美国给日本加上的紧箍咒，允许日本在受到确定的威胁时先发制人，行使此前被禁止的集体自卫权，对敌国行使军事武力。时至今日，中共的军力大大扩张，美国更加需要日本的力量牵制中共，日本势必要求与亚洲老大相称的军事力量和国际政治权利，将引发印太新一轮的军备竞赛。

现代战争非常残酷，只有象毛泽东和习近平这样的邪恶疯子才想打仗。知名记者石坂明夫认为，中共只要认为能够在西太平洋的冲突或战争中有取胜的把握，它就会毫无犹豫发动战争。日本政府现在要做的事情，就是联合所有的力量，对中共形成战略优势和力量震慑，才能避免战争的爆发。日本最近和北约不同成员国、美国、印度等国频频举行联合军演，极力将台湾拉入国际大家庭，加强对累受中共欺负的东盟各国的援助和合作，都是为了寻求和平进程。

因日本和菲律宾的疫情都在加重，两国上个月的首脑会议被取消后，日本首相菅义伟本周四和菲律宾总统杜特尔特举行了电话会谈，双方就共同对抗中共试图改变西太平洋现状达成共识，特别就如何利用《联合国海洋法公约》对抗中共国的《海警法》进行了磋商，双方还敲定了日方援助菲方200亿日元的新冠疫情对策，双方的互访还将继续协调。

杜特尔特被认为是迷失了方向，一直向中共靠拢，是亲中共派。美菲之间有与美日《安保条约》相似的巜联防条约》，该条约于1952年正式生效，有效期无限。估计邪恶的中共和老杜都认为美国不会轻易为中菲爆发南海争端而战，因而近年来和中共走得很近以捞取好处。当菲律宾的近海专属经济区遭受中共的侵扰，国内群情激愤时，老杜不得不和美国修复关系以制衡中共，日本的出现，就是来拉迷糊的老杜一把的。

在菲律宾国内的反共情绪日益高涨的情况下，杜特尔特于近日的全国电视说话中禁止其内阁和政府成员公开谈论南海争端问题。菲律宾前国防部长、前参议长恩里莱向杜特尔特提议：“我们承担不起与中国对抗的后果，除了与中国谈判，我们别无选择。如果不能与中国达成协议，我们的经济和安全将受到影响”，据悉杜特尔特采纳了恩里莱的建议，并对投降派恩里莱说：“对我来说，你是唯一重要的人”。这不仅仅是菲律宾弱国无外交的问题，在亚太地区也不单是菲律宾有这样或那样的问题，因此日本的合纵连横之路将代价不菲和充满艰辛。
