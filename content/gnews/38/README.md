图片来源于：网络

2月11日一名知情人士对《华尔街日报》说中国科技业巨头腾讯公司的副总裁张峰被当局拘留，张峰涉嫌在未经授权的情况下，分享腾讯旗下通信软件微信的数据。张峰从2020年初以来，就开始受到中共反腐机构的调查。张峰涉嫌将微信收集的用户个人资料交给中共公安部前副部长孙力军，而这个孙力军去年4月时已经落马，调查人员正在调查张峰和孙力军分享了哪些信息，以及孙力军可能利用这些信息做了什么。张峰是在2018年的时候加入腾讯的，作为腾讯负责政府事务的高管，他主要负责和中共政府官员会面并处理公司与北京各部委的关系。腾讯公司也在2月11日向路透社证实，该公司一名管理层人员因“个人腐败案”正遭到有关当局的调查，但没有说这个管理层人员具体是谁。

去年4月19日，孙力军在没有任何先兆的情况下，突然落马了。而就在他落马的两个月前，孙力军还作为中共防疫大员，去了被封城的武汉。在孙力军落马后曾有中共国保的消息人士透露说，孙的落马，对中共体制、公安系统，尤其是秘密警察，都造成了巨大动荡，在中共官场产生的影响，可能不亚于2012年的王立军事件。关于孙的落马原因，当时有传闻说，可能是因为孙力军向美国泄露中共肺炎疫情机密而引发，当时多数分析人士认为，孙的落马很可能是起因于中南海内的派系权斗。

虽然中共官方一直没有披露孙力军落马详情，但随后落马的重庆公安局长邓恢林、上海公安局长龚道安都被通报“参与在党内搞团团伙伙”，被暗示涉及反习政变活动。孙力军与邓恢林、龚道安，都是江泽民集团上海帮大佬、前政法委书记孟建柱的心腹旧部属。而孙力军很可能是江泽民集团政法系统反习政变活动的关键人物。而腾迅的副总裁张峰，涉嫌将微信收集的用户个资交给孙力军，是否也涉入江集团的反习政变活动，目前还难知更多细节。如果说张峰与孙力军真有牵连，那这个事儿可能麻烦就大了，马化腾能否脱了关系目前也很难说清。

随着腾讯高管张峰受调查的消息传出，外界对于最近一直没有公开露面的马化腾的动向也感到好奇。从目前得到的信息看，马化腾应该没有被指控有任何不法行为。

马化腾涉个人私隐言论引强烈质疑。

2020年4月，《福布斯》公布的全球富豪榜上马化腾排名第20。在中国，“二马”称得上是有绝对影响力的科技巨头。

2月8日，网络上热传了一段影片，马化腾在影片中笑着说，每一天他们有超过十亿张的照片上传，绝大部分都是人脸。马化腾还特地强调，尤其是中国人的脸特别多，他们有几乎每个中国人过去十几年来脸的变化，所以他甚至可以预测每个中国人老的时候是什么样子。马化腾的这番自感得意的言论惹怒了不少中国网民，有人直接质问说：“营运商有什么权利拿我们的肖像？把肖像权的法律践踏得面目全非。”还有网民要求官方介入。但是中国是官商一家亲，怎么介入呢？

和马化腾相关的事发生了好几件。新华社主管主办的《上海证券报》，在头版发表了一篇名为“高质量发展岂能缺少企业家精神”的文章。文章说，中国的企业家从改革开放初期的“草莽英雄、百无禁忌”的形象，经过“大浪淘沙”，发展成今天的“遵守市场规则”的现代商界领袖。文章还说，若将商海比作江湖，企业家便是当之无愧的“掌门人”。“江湖之中，有血雨腥风、刀光剑影，也有同气连枝、相倚为强。”并说，中国自古以来的“商训”，皆讲究义与利的辩证统一。

文章还列举了一长串中国当代企业家的名字，特别“点赞”腾讯的马化腾“改写移动互联网时代”，可文章中偏偏少了马云的名字。马云，这个在国际上都有影响力的科技巨头，他的名字怎么可能被遗漏，合理的解释就是刻意作为。在讲“商训”，但文中绵里藏针，看上去是一方面再度暗示马云问题没完，另一方面警告被点赞的企业家们也要乖乖听话，站好队，不要步马云的后尘。

就在马化腾被官媒捧杀的同时，北京字节跳动公司旗下抖音向北京知识产权法院正式起诉腾讯，认为腾讯通过微信和QQ限制用户分享来自抖音的内容，是垄断行为，要求判令腾讯立即停止这一行为，并赔偿9,000万元人民币。腾讯方面回应称，字节跳动的相关指控纯属失实，是恶意诬陷。并指，字节跳动所属多款产品，包括抖音通过各种不正当竞争方式违规获取微信用户个人信息，破坏平台规则。

当然，在没有市场规则可言的中国企业界，这就成了“公说公有理，婆说婆有理。”腾讯公司的反舞弊调查部发布公告称，2019年第四季度至今，腾讯共发现查出违反“腾讯高压线”案件60多起，100多员工因违反“腾讯高压线”被辞退，40多人因涉嫌犯罪被移送公安机关处理。

把这些事连起来看，抖音起诉腾讯违反反垄断法；腾讯对外公告称，40多名员工因涉嫌犯罪被移送公安机关处理。这无疑是对官媒报导中“遵守市场规则”说法的反讽。

明年2022年，又到了中共“二十大”换届的时候了，对习来说，反习势力搞政变的阴影一直是习的心病，接下来习要考虑的是明年是否能顺利连任，而今年扫清障碍，排除隐忧就成了关键。如此看来，马云、马化腾的处境确实微妙，这两位科技巨头是否真的纠缠到中共顶层的权斗中，是否可以安全落地，而习的盘算又是否能够梦想成真？这些，可能都会在这一年多里逐渐揭晓。
