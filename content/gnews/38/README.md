2021年1月16日，习近平报道专集发表了《这篇重要文章，习近平这样看待时与势 》文章。题目看上去与习近平小学文化水准相配，狗屁不通。就这狗屁不通的文章纷纷占据头版头条，各大媒体如中国经济网、中国军网等均有转载，给人感觉习总加速师在山洞里正在挖掘时光隧道一样，又底气十足。
这篇报道其重要的信息是什么呢？我们看几个关键词就知道，文章强调“主动求变，勇于开顶风船，善于转为危机”。那么怎么叫“主动求变”？是不是感觉不行了，要主动出击？这就是进一步踩油门，进一步暴露邪恶计划的举措，究竟在临死前会有什么反击？我们只能拭目以待。至于“勇于开顶风船，善于转为危机”这句话就进一步透漏了自己的败局已定。中共南湖那条破船在邓小平手上休养生息，平稳开到了今天，轮到习总加速师，他竟然愚蠢地拿他当泰坦尼克号豪华邮轮开，全力加速，突然发现冰山了，不得已，为了续命，竭尽所能掩盖要撞冰山的事实。命令船长全速前进，遇山填山，遇海填海。这就是习共的加速思维——愚蠢。
该报道提到六点内容，主要是以总体归纳为瞅准危机中的机遇，看似否可以求生、开启国内内循环经济、以科技作为内循环的驱动器（数字经济加速剥削百姓）、深化改革（基本是句废话）、以该水平的对外开放打造估计合作（进一步蓝金黄，加大一切可能收买欧美政客，对可能的拜登政府仍然有一丝期待）、共治共享拓展社会发展（这里实际上是围绕内循环经济的共享、共产共妻模式）。这样一个思路来诠释共产党如何求生。
笔者特别指出第六点：共治共享，原标题，“以共建共治共享拓展社会发展新格局发展”。在这一章节中提到的一句话是非常邪恶，“强化公共卫生和疾控体系，促进人口长期均衡发展，加强社会治理、化解社会矛盾、维护社会稳定”。这几乎不用思考即可明白，根据综上所述，他们最后的疯狂一定是借助生物科技和网络化的监管模式加大力度奴役百姓，继续使用生化武器消灭弱民，减少人口，降低难以为继的国民开支。用网络舆论监管消除反对和真实黑暗中的声音，以此达到14亿人大国中的监狱管理模式，习共自有这样做，才能达到他们所谓的维稳。
