## 北极上不凋落的七芒星 (一)
` 温哥华圆成农场🇨🇦` [轉載自GNews](https://gnews.org/zh-hans/559279/)

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/14180407/beiji2.png)https://www.youtube.com/watch?v=hktE02RIkAI
驶过群山，花飞满天。

重峦叠嶂。天高云淡，

最美风景，都在路上。

只能感受，难以记录。

如暗夜流光里的一场梦境。

十三天的行程，来回七千多公里，从温哥华出发，一路向北。日落星起夜起风，秋意已到销魂处，路上美景鸿城，风光旖旎。只有你的脚步到达了真正的远方，才能看到平时看不到的风景。

那里天地寂寥，峰峦高耸，冰原伟岸。

你和远方在一起，和苍天大地在一起，

和几百万年冰川河谷在一起

凭空荡胸生层云时，物随心转，境由心造，感慨世间烦恼多心生。

我们一行先是到BC最北部的大城市Price George，接着北上。

第一晚在一个叫meziadin lake park的湖边宿营地camping.

那是一个很静谧的湖，

湖水神秘诱惑，湖中小岛乱木狰狞。

我们的camping 营地周围不少房车和各类帐篷，有那种超大的中军大帐，也有小不点的徒步时背的单人小帐篷。在湖边星星点点，在夕阳余晖下相映成趣。

看着湖水连天天连水，秋来分外澄清。

我忍不住换了泳衣下水。天气已经是八月底，九月初，人进去水中之后，寒冷冰凉

游了不一会儿，骨头都冻透了，在水里失温超快。而且可能由于营养丰富，湖水发黏，人游起来很吃力。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/14180605/beiji3.png)https://youtu.be/LKi7MYWxv5I
边上营地的是两个白人女孩，年轻健康而富有朝气。

她们哼着小曲，弹着不知名的乐器。

我到帐篷里换衣服。

看她们开始点篝火。

小雨窸窸窣窣下了下来。

她们半天点不着，

我于是借给她们fire start。帮她们把柴火点燃。

闲谈中知道她们是一对couple.

现在算是度蜜月性质

她们本来计划是去南美的

现在没有了航班，也担心各种路上的麻烦

只能作加拿大境内游。

我有点小吃惊，虽然温哥华司空见惯

但看着她们神态自若，很滋润的样子

也只能说 congratulation，一边问她们对病毒怎么看

然后给她们闫博士的宣传单

她们看了看，充满疑惑，有些奇怪的看了我一眼，

然后继续准备晚餐。

我那边朋友们也把篝火点燃。

熊熊烈火在暗夜中熊熊燃烧。

烤上各种培根，香肠，土豆，玉米，

清香四溢。馋虫打动，劳累了一天，大家都是又累又饿，伴着雨水，烟雾缭绕中大快朵颐。

半夜大雨倾盆。

我的帐篷雨嘀嗒嘀嗒下来，内帐和外帐贴到一起，很容易把被子大湿。

没办法半夜摸着黑起来换备用帐篷

点着头灯摸索着，穿帐篷支架。

那两个女生悉悉索索不知道在做什么，不时发出短促的声响。

被雨声压住，倒也对我影响不大。
