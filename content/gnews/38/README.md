美国疾病控制与预防中心主任罗谢尔·瓦伦斯基本周早些时候表示：“接种疫苗的人不携带病毒，不会生病。”（见上图）

但卫生机构4月1日澄清了称瓦伦斯基的说法，认为其缺乏明确证据，不够严谨。

CDC发言人告诉《纽约时报》：“一些完成疫苗接种的人也可能会感染新冠病毒。目前尚不清楚他们是否可以将病毒传播给其他人。我们正在继续评估证据。”

纽约纪念斯隆·凯特林癌症中心卫生政策和结果中心主任彼得·巴赫博士表示，对疫苗效果的类似宣传可能导致美国人在接受注射疫苗后拒绝戴口罩。

![]()![](https://gnews.org/wp-content/uploads/2021/04/图片16.png)2021年4月1日，居民和工作人员在洛杉矶米申希尔斯的阿拉拉特护理中心
为接种疫苗的居民举办的复活节音乐会上跳舞。

周一，瓦伦斯基告诉微软全国广播公司的雷切尔·马多：“今天来自疾控中心的数据表明，接种疫苗的人不会携带病毒，也不会生病。”

她补充说：“临床试验数据和现实生活的数据都说明了这一点。”

瓦伦斯基的评论没并有得到科学研究的支持，即疫苗可以完全阻止病毒的传播。迄今为止，还没有疫苗被证明对所有人100%有效。

本周早些时候，疾控中心发布了一项对4000人的研究结果，数据显示辉瑞和摩登纳的疫苗在第二剂后对新冠病毒感染的预防效果高达90%。

闫丽梦博士一再强调新冠病毒是中共人工制造的超限生化武器，而且他们从未公布过这种病毒的任何原始数据，人类对这种病毒的认知还非常有限，加之疫苗研发流程不完善，在这种情况下轻言疫苗有效是极不负责任极其危险的，不但会误导大众，让人们疏于防范，还会让中共有机可乘，继续利用病毒去控制全世界。

要想研发出真正有效的新冠疫苗，首先要全面了解病毒，掌握全部的相关数据，对制造病毒的武汉P4实验室进行彻查，找到新冠病毒的原始毒株……，而要做到这一切的前提是推翻中共政权，彻底降服这个为害人间的恶魔。
