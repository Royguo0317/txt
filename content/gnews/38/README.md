## 【墙内美文】 清明说风水，阴宅新玩法&#8211;“阴阳混居”
` HoustonLitstar` [轉載自GNews](https://gnews.org/zh-hans/1053989/)

作者：休斯顿星光农场 写作组 文初

- ![]()![](https://gnews.org/wp-content/uploads/2021/03/Screenshot-2021-03-25-230524.jpg)


国人的忌讳从来多不胜数，婚丧嫁娶特别讲究。亲人过世一定要选风水宝地，才能保佑子孙福寿延绵，兴旺发达。几千年来阴阳两界一直相安勿扰，清清静静，没想到脑洞过大的墙国人竟做了一件连阎王爷都意想不到的事情，硬生生建成“阴阳混居”的城市新景观。您说瘆不瘆人！瘆不瘆人！没办法啊，阴宅比阳宅贵，活人不能让死人给穷死啊！

据了解，在中共国的上海、天津、阜新等一、二、三线城市都存在着这样小区。整栋楼不是用来住活人，而是专门用来安放亲人的骨灰。天津的静安陵园，更是懂得钻法律的空子，偌大的小区全部都是灵堂，称之为“楼房式灵堂”。楼中每个房间的门窗皆为黑色，并署名祠堂，有的门上还别着大红花，走廊里处处透露着阴森感。众所周知，现在墙内一线城市的房价高的吓人，但为什么要给死人住这么贵的阳宅呢？无非是因为钱呗！难道还是因为孝心？！

- ![]()![](https://gnews.org/wp-content/uploads/2021/04/1-14.jpg)


当时这个阳间阴宅小区刚推出时曾经遇冷。每平方米售价仅为3000-4000人冥币（在辽宁阜新市的价格300-400人冥币/平方米），但仅仅6年时间，价格火速攀升至7000元人冥币/平方米，快赶上墙国三线城市阳宅的平均价格了。 如今该小区已经入住了3000多个家族，近十万个骨灰盒。活人的空间开始被死人侵占，魔幻啊我的墙国！连阎王爷都要给你点赞了！

在中共的县城，一块经济型墓地已可以售价高达20-30万人冥币，而且只有20年的租赁使用权，到期要续交管理租金，并非归个人产权。而一套小户型的房子也就售价几十万人冥币，还拥有70年产权，并且在极端情况下，还能当作正常房屋投资出售。 既然买房子更划算，干嘛非得要买墓地？

回想当年给父母买墓地的时候，一个墓穴6000元人冥币，现在的管理费都已经涨到12000人冥币，是当年售价的两倍了。到了孙子一代负担时，是不是要乘以5以上呢？那孙子的孙子呢……不敢想象！如果子孙逾期不交，身后埋骨之所，将按无主的墓穴处理，至于骨灰，自然也一样被“无主”处理了。怎么个处理法，一切皆有可能，刨出来，扔出去，那时候真的是尘归尘，土归土喽！为此“热心”的他们早已“想你之所想”： 该陵园市场部负责人称，因行业特殊，可以办理贷款而不需要抵押，“贷款不会贷给本人。如果客户50岁、60岁，他的子女就可以作为贷款人。如果客户70岁、80岁，他的孙辈就可以作为贷款人。” 想想墙国人，可不可以再惨一点？活着的时候，还房贷；死了，子孙后代还墓贷！

- ![]()![](https://gnews.org/wp-content/uploads/2021/04/2-6.jpg)


墙国还有一则更让祖坟气冒烟的消息。广东湛江一家陈姓家族存在了300年的祖坟，居然收到相关部门的通知书，认定此祖坟为违章建筑，被开发商要求限期迁走，陈氏后人无法接受！据陈氏后人出示的族谱和相关文书记载显示，这块地是陈氏祖上买来用于安置族人的坟地，从乾隆年间算起，大约已有300年的历史。 当地管理部门称，这块坟地的土地所有权早在1977年就被划给中共林业研究所了，2004年被中共政府拿出来挂牌交易。坟地没有什么手续，也就没有办法证明是他们（陈氏族人）的土地有合法来源。估计陈家祖先泉下有知，听到这个狗屁强盗土匪逻辑都会气冒烟了。咋叫没办法证明？刻着年代的300年石碑是假的吗？在这里住了300年难道需要找阎王爷开证明吗？300前你们共匪在哪里转筋还不知道呢？就算强盗土匪都不敢这么玩吧！

当然也有民告官胜利的案例。2004年3月25日，中共武汉江夏区的一位七旬老汉倪灯财三告湖涠镇政府。他不请律师，自己脚穿雨鞋站在法庭上慷慨陈词：“这块地在清朝时是我家的，民国时也是我家的，日本小鬼子来了仍是我家的，你们来了就变成你们的了？我就不信邪！” 2004年6月，武汉中院做出终审判决，倪老汉胜诉。法院判令湖涠镇政府无偿退还倪老汉的土地。

当下墙内百姓只能 “好死不如赖活着”下去，因为你真的不敢死，死不起啊！
