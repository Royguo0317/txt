**中共为造假推迟公布人口普查报告**

中共国每十年进行一次人口普查，最近一次应该是在2020年11月1日至12月10日展开，却因故延迟。中共国家统计局曾表示将在2021年4月初发布数据，但随后又表示需要更多时间。4月份，英国《金融时报》引述知情人士报导，中共国面临数十年来首次的人口下降。中共国家统计局在4月29日仅用一句话反驳说，中共国在2020年人口是增加的，而原预定发表的人口普查因为需要更多细节而延迟。

据《华尔街日报》报导，在推迟人口普查报告的几周之后，中共国终于发布“灌水”结果，报告中显示小幅上升的数据。中共国的人口在2020年达到14.1亿，比2019年的14亿官方数据稍有增加。与2010年的人口普查相比，中共国的人口仅增加了7200万。中共国家统计局局长宁吉喆说，2020年中（共）国的生育率（妇女一生中的平均生育人数）下降至1.3，他承认这是一个低水平。相比之下，在同样面临生育率下滑的美国，2020年的总生育率为1.64。这个结果表明，这个号称世界上人口最多的中共国，将比预期更早地面临人口下降的威胁。

政府智囊团——中共国人口与发展研究中心——2020年底发布的一份研究报告预测，中共国人口将在2027年达到峰值，达到14.17亿。这比中共2017年的原本预测的2030年提早了三年。目前尚不清楚在此次人口普查结果发布后是否需要进一步修订。

**中共国出生率急遽下滑**

中共国家统计局局长宁吉喆在稍后的新闻发布会上说，2020年有1200万新生婴儿，比起2019年的1465万下降了18％，出生人口下降趋势使得中共当局必须停止计划生育的一胎政策。2015年，中共结束了长达三十年的“独孩”政策，因此，2016年的生育率得以上升，之后一直呈下降态势。

长期批评中共国人口政策的驻美国研究员易富贤说，由于中共国的独生子女政策，导致育龄妇女人数下降，未来几年的出生率可能仍会进一步降低。

中共病毒疫情对中共国人口数量的影响尚不清楚。尽管中共国宣传已迅速控制其境内感染，但人口统计学家表示，中共病毒疫情的影响可能导致出生率下降。

**中共国令人难以接受的生育政策**

中共国实施独生子女政策之后，使得父母将所有的资源都集中在一个孩子身上，许多家庭认为他们根本负担不起第二个孩子。2016年，中共全面放开“二孩”，但他们希望出现的婴儿潮并未到来。

然而，中共国的生育限制政策一直都在，想要两个以上的孩子的少数夫妻仍将面临罚款风险。杭州一名33岁李姓前地方政府工作人员起诉她的雇主，原因是她去年在生下第三个孩子的四个月之后就被解雇(中共国的法律禁止雇主在孩子出生后的几个月内解僱员工)。今年年初，李女士诉诸杭州的一个劳资纠纷仲裁委员会，该委员会裁定她违反了政府的计划生育规定，因此无法享受生育假的保护。李女士在微博上说，她觉得在中共国，生育是一种罪过。

也有一些父母及人口统计学家说，近年来，一些地方政府已经悄悄地允许家庭生第三个孩子，而不像往常一样采取惩罚措施。即使是长期支持中共国计划一胎政策的官员和研究人员，现在也开始改变措辞，强调需要增加生育率。

**中共进入老龄化社会**

中共国的老年人口仍在增长之中。数据显示，中共国60岁及60岁以上的人口比例从2010年的13.3％急剧上升至18.7％。2020年，年龄在15至59岁之间的中共国人口比例为63.35％，低于2010年的70.1％。

驻美国研究员易富贤认为：“数据显示的人口老化是对中共国经济灾难性的影响。”人口老化预计会是消耗中共国储蓄的主要因素。虽然中共国目前鼓励大众消费，然而由于担心国家可能无法支付养老金，只生一个孩子的退休老人很可能不愿意花钱消费。

在很短的时间之内，中共国的人口问题已经转变成为中共当局头痛的经济难题。多年来，中共国的人口结构已改变。劳动人口逐渐减少而退休老人快速增加的态势已经很明显。由于美中贸易战与限制自由经济的私人企业，导致中共庞大的债务危机。中共领导人蜡烛两头烧，无法及时有效解决中共国的人口隐忧。

如今，中共再也无法忽视人口问题带给中共国长期的影响。由于中共国东北部许多经济萧条的城市养老金不够发放，迫使中共中央政府要求国有企业以及南部较富裕和劳动人口较多的省份补贴养老金。

**中共国劳动人口不足**

根据中共国官方统计，自2012年以来，中共国的劳动人口一直在萎缩。中共领导人总认为”自动化”就可以消除劳动人口减少所带来的威胁。然而经济学家对此政策的可行性表示怀疑。今年三月，中共国中央银行的研究人员发表了一篇论文，呼吁对中共国可怕的人口下滑做出更强有力的回应。论文中强调：“中共国必须意识到，教育和技术的进步几乎无法弥补劳动人口的下降。”

虽然，一般预计中共国仍将超过美国成为全球最大的经济体，但一些经济学家警告说，如果工人人数持续减少，中共国可能无法保持此一优势。与美国不同的是，中共国无法依靠年轻的移民来补充国内的劳动力。

**从自我催眠到认清现实，中共国为时已晚**

2015年，在中共国取消独生子女政策后，时任中共国计划生育委员会副主任的王培安声称，“计划生育”是中共国应长期坚持的“基本国策”。2017年，王培安对中共国可能面临人口短缺的风险提出了质疑。他在新闻发布会上说：“不光是现在不缺劳动人口，未来一百年都不会缺。”他在新闻发布会上夸下海口，认为从2017年到2020年之间，每年将有1700万至1900万的婴儿出生。然而现实的状况是，除了在2016年新增人口增长到1786万，之后的每一年的出生人数都下降了。

今年三月，中共中央银行发表论文说，应全面放开和鼓励生育。中央银行研究人员写道：“若稍有迟疑，将错失以生育政策应对人口转型的宝贵视窗期。”

今年四月，根据党营谘询机构经营的一家报纸称，现任中共国共产党政治谘询机构成员的王培安呼吁“对人口政策进行重大调整”，以采取鼓励生育的措施。
