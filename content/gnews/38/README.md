《外交政策》（Foreign Policy）10月28日发表了一篇充满偏见和敌意的文章，对傅希秋和吴建民受到华人抗议事件鸣不平。

文章开始就描述：一位著名的中国持不同政见者把孩子从学校拉出来，逃离德克萨斯州的家。另一位加州的一名民主活动人士，几天来面临着聚集在他家车道上的抗议者的威胁和骚扰。这两人都是中国流亡商人郭文贵上个月公布的一份持不同政见者名单上的目标。郭文贵在没有证据的情况下声称他们是中共的间谍。

作者还特别提到，”让我们消灭世界上的叛徒吧。”上个月，手持雪茄的郭文贵在网上发布的一段视频中对他的追随者说。”让我们开始吧，先把这些汉奸干掉再说。”

文章继续写到，在随后的几天里，数十名抗议者每天在牧师傅希秋(Bob Fu)的家乡得克萨斯州米德兰(Midland)外举行示威活动。傅希秋是福音派基督徒中的知名人物，而在美国的中国异见人士中，福音派基督徒占很大一部分。他们拒绝透露他们是谁，也拒绝透露他们从哪里来。傅家门外的一名男子向当地媒体自称是 “德州牛仔”，但他们明确表示，他们正在听从郭文贵的号召，抗议非营利组织 “中国援助”（ChinaAid）的创始人，该组织为在中国面临迫害的基督徒提供法律援助。

郭文贵及其追随者的行为已经引起了联邦调查局的注意，遭到了参议员马可-卢比奥（Marco Rubio）和泰德-克鲁兹（Ted Cruz）的谴责，对这位亿万富翁房地产大亨到底在搞什么，引起人们的猜疑。

文章还引用傅希秋无耻的话，在他和家人被保护性拘留后他告诉记者，中共本身就是这些攻击的幕后黑手。“我认为这是一场资金充足、有导演、有策划、复杂的抹黑运动，郭文贵认为自己是罗宾汉，组织私人民兵来伸张正义”他说。

文章故意将傅希秋等人与郭文贵混为一谈，说道在充满争议的中国持不同政见者的世界里，指责其他人为中国共产党工作并不少见。这并不奇怪，因为中国共产党确实花了大量的时间、金钱和精力来对付这些组织，他们认为这些组织是对其统治的严重威胁。但这也可能意味着个人恩怨变成了政治偏执。

文章还污蔑说郭先生的背景和行为引发了令人不安的问题。在中共错综复杂的商界和政界游走的郭文贵曾经是一位杰出的玩家。但2015年，在他的支持者、国家安全部副部长马建倒台后，他逃离了中国，处境危险。从那以后，他一直试图把自己塑造成一个有原则的中共批评者。在抵达美国后，他与川普的前策略师、极右翼者史蒂夫·班农(Steve Bannon)结成了联盟。两人发起了一个流亡政府–新中国联邦–以及媒体公司GTV媒体集团（GTV media Group）。据报道，GTV媒体集团正受到美国联邦调查局(FBI)和美国证券交易委员会(sec)的调查。

文章继而更加肆无忌惮的诽谤GTV是郭文贵和班农的喉舌，散布有关冠状病毒来源的虚假信息，并发表有关总统候选人乔-拜登的儿子亨特-拜登的阴谋论并称这些阴谋论已渗透到特朗普的核心圈子和《纽约邮报》的编辑部。说郭先生利用这些渠道来诽谤像傅希秋这样的异见人士。今年1月，郭文贵的GNews网站发表了一篇名为《揭露傅希秋 – 一个披着牧师外衣的中共间谍》的文章，文章从严厉的指控–没有事实根据的人口贩卖和性骚扰指控（傅否认这些指控）–到据称是傅亲属拥有的德州房产的截图，以及谷歌对傅妻子拥有的一家酒庄的批评性评论，林林总总。

文章说傅希秋觉得搞笑，但他已经认真对待了。“从那时起，我就开始觉得，他真的在威胁我们的家庭，”他说。

文章结尾说道，除了一起非法入侵事件外，中部地区的抗议活动一直保持和平和守法，郭文贵也表示他不容忍暴力。但这些言论引发了人们对郭文贵是什么样的人的好奇。
