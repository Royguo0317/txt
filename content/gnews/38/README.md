- **美国还希望在北领地的军事基地储存军火弹药**
- **美国大使迈克****–****戈德曼说该地区面临“新的地缘战略背景”**
- **发生于中共国和西方之间的政治紧张局势继续升级之际 **


由于中共国和西方国家的政治紧张局势继续升级，澳大利亚正在与华盛顿谈判，以便在本国领土上制造美国设计的弹道导弹。

美国军方还希望在北领地的军事基地储存一批弹药和国防设备，美国大使馆临时代办迈克-戈德曼（Mike Goldman）向《澳大利亚人报》描述此举为**“前沿部署”。**

戈德曼先生说，鉴于该地区面临的“新地缘战略背景”，（美澳）更紧密的合作“是有意义的”。

虽然还没有最终确定，但总理斯科特-莫里森在3月份宣布，联邦政府计划出资10亿美元，开始在澳大利亚生产高科技制导导弹。

“这些事情都不是一蹴而就的，但我们的军队和战略思想家正在讨论（美澳）如何最好地共同面对这些新挑战，”戈德曼先生说。

“尤其是当我们谈论如何在任何类型的紧急情况下投射武力时，其中的很多内容都具有完美的直观意义。”

“我们现在处于一个新的地缘战略背景下，需要一套不同的平台和不同的部队态势。”

在堪培拉这位美国人还抨击了中共国和习近平主席对澳大利亚的“经济讹诈”运动，并说美国永远不会让澳大利亚人“独自在球场上”（独自面对中共国）。

在过去的一年里，中共国对澳大利亚的出口施加了价值超过200亿澳元的任意贸易禁令和关税，这显然是对澳洲呼吁对中共病毒大流行病的起源进行独立调查的惩罚，2019年该流行病首次出现在武汉。

上个月，包括国防部长彼得.达顿（Peter Dutton）、前国防部长克里斯托弗.派恩（Christopher Pyne）和内政部长迈克尔.佩祖洛（Michael Pezzullo）在内的各物级人都表示该地区的“战鼓”越来越响，（亚太地区）紧张局势进一步升温。

人们非常担心中共国会利用其不断增长的军事力量将台湾与中共国大陆（武力）“统一”。

这个专制国家在整个地区和其有争议的领土上变得越来越有侵略性，并根据其“一中”政策镇压香港的民主团体和新疆的穆斯林少数民族。

中共国的观察家们说，在美国和日本的支持下吞并该岛是习主席终身的战略目标。

随着紧张局势的升级，像澳大利亚这样的民主国家面临的压力也越来越大，（西方）要求其遏制中共国的势力。

戈德曼先生重申，“牢不可破”的澳新联盟——即将在今年9月迎来其70周年纪念日，澳新联盟将在面对不断上升的敌对行动时继续保持紧密合作。

他说，这一战略联盟将超越当前的紧张局势，并补充说，这一伙伴关系比任何一个突发事件都要强大和全面。

什么是安第斯条约？

即澳大利亚、新西兰、美国安全条约，（安第斯条约）是一项不具约束力的战略军事协议，于1951年第二次世界大战后首次签署。

根据该安排，三国将在太平洋地区的军事事务上进行合作。今天，据了解，ANZUS条约也适用于世界其他地区。

对于澳大利亚和新西兰来说，该条约是一个重要的安全防护措施，可以防止一个更大和更强大的国家的攻击或入侵。

第二次世界大战后，安第斯条约将前苏联帝国视为国家安全的一个主要潜在威胁。如今，这根接力棒已经传给了中共国这个独裁国家。
