据《华盛顿观察家报》（Washington Examiner）1月13日报道，民意测验专家弗兰克·伦茨（Frank Luntz）称，据一项民意调查显示，尽管国会山发生骚乱，但压倒性的绝大多数川普选民仍坚持他们的立场，会再次投票支持总统。

弗兰克.伦茨在周一他的推文中写到，尽管自11月3日到上周，对川普的批评声一直不断，但91％的支持川普的选民表示，如果今天举行另一场总统选举，他们仍然会投票支持川普。接受民意调查的川普选民中，只有25％表示总统对国会山骚乱负有责任。 此外，在接受调查的800位川普选民中，78％的人表示同意总统“大选被窃走”的说法。67％的人认为，如果选票能准确公平地计算，川普会是赢家。当被问及总统是否应该继续为留在白宫而斗争时，选民中有50％的人表示愿意，而50％的人表示应该离开。

**评论**：首先这民意調查是有误差的，尤其在左媒当道，暗黑政府猖狂的情況下，民调结果的准确性和代表性大大打折扣，这在大选前就窥见一斑，当时BBC的选前民调为拜登52%，川普为48%，但大选日的真实投票结果是川普的得票率倍数于拜登，当然这最后结果被恶劣的舞弊所扭转。就是在这么严酷的政治舆论环境下，该民调还是显示出川普在他的选民中的极高的信任度和忠诚度。川普总统和他的支持者都是宪法的守护者，相信很快“窃走”的大选会被拨乱反正，川普仍会是下届人民真正选出的总统。
