虽然人民日报在早期2015年已经呼吁[取消出租车公司](https://c.m.163.com/news/a/G6MF4ED50547B3C1.html?spss=newsapp)的风声，中共国如今再次呼吁取消出租车，也证明了出租车从辉煌时代步入了一个终结时代！大多数出租车公司有国企、准国企、即使是私企也要得到当地政府部门的许可方可运营。

####  简评：

**关于出租车公司的取消最主要的原因还是利益不均所导致！这也是必然的结果，新创新网约车平台到来加速出租车公司的淘汰。经济问题矛盾冲突加大，再加上中共病毒疫情持续不断、上访、社会治安问题频频发生！挂靠经营与承包经营模式已经不具备再现如今的市场有竞争力和强大的盈利能力。**

**对于市场盈利不佳的情况下，还要向具备有经营权的出租车公司交纳各种费用，并附带各种连带责任问题。 现在取消出租车公司符合时宜！在应对失业、社会经济等严重问题，以便于应对可能出现的更多上访行为和社会治安问题，现在所有权和经营权合二为一给予个人表面上是惠于大众，实际却想推卸责任到个人。 在大背景下也同样符合趋势！网约车的趋势智能、智行大数据可追踪查询，行驶路线、出行人的个人信息、人脸长相……这些功能和生态是更符合中共国目前的利益和发展形势！对于监听、监视国内民众是一大利好！**
