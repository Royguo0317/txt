2月8日新中国联邦奠基人郭文贵先生爆料，中共国主席习近平下了命令，必须牢牢把握春节契机，让拜登总统打电话恭喜春节，恢复中美关系。果然，2月11日中国农历除夕当天，完全按照习近平的剧本美国总统乔·拜登（Joe Biden）开启与中共国家主席习近平上台后的首次通话。

根据白宫发布的最新通讯稿，“拜登总统强调，他对北京的胁迫性和不公平的经济行为、镇压香港、在新疆的侵犯人权行为，以及包括对台湾在内的印太地区日益强硬的行动表示根本上的关切。并表示将致力于在符合美国人民和盟友利益的情况下，与中方进行务实的、注重结果的接触。最后，两国领导人就应对COVID-19疫情，以及全球卫生安全、气候变化和防止武器扩散等共同挑战上交换了意见。”

对于中美两国领导人的首次对话，中共官媒表现出我行我素，只字不提COVID-19疫情。新华网报道称，中共主席习近平指出，中美合作是双方唯一正确选择，中美对抗对两国和世界将是一场灾难。双方应该重新建立对话机制，准确了解彼此的意图避免误解误判。习近平表示，希望美方谨慎处理台湾、香港、新疆等事关中国主权和领土完整的事务。

中共在香港、新疆的暴行以及对台湾明目张胆的军事侵扰，拜登仅仅只表示了“关切”，暴露出其对中共习近平的示弱和服从。他在前政府明确指出有病毒来自中共军方实验室的前提下，仍选择与中共“交换意见”，体现了拜登政府对习近平的姑息养奸。拜登政府只为利益而合作的想法会让美国落入中共设下的一轮又一轮的谈判陷阱和中共病毒灾难惨局。

拜登上任不到一个月，就快速让美国重回巴黎气候协议、世界卫生组织、联合国人权理事会等全球性组织，并撤销教育机构公开与孔子学院的合作信息强制令。拜登政府单方面寄希望通过重回国际组织发挥美国的领导作用，重新与中共接触甚至合作从而获取更大利益的政策倾向极其危险。与反人类种族灭绝的罪犯、与制造生物武器的最大恐怖分子做交易将给拜登政府埋下致命的自杀式炸弹。
