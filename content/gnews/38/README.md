《福克斯新闻》2月24日发表文章，题目是“预计川普总统会在保守派政治行动年会上（CPAC）针对移民和中共问题敲打拜登”，副标题是“川普总统将在佛罗里达州奥兰多的会议上发表讲话。”

全文如下：

知情人士对《福克斯新闻》说，川普总统周日在保守派政治行动年会（CPAC）发表讲话时，将对拜登从移民到中共的所有问题进行抨击。

与此同时，人们期待川普总统能更早一点宣布2024年的总统竞选。消息人士说，他将在“为2024年竞选的想法热身，还是直接进入宣布另一场竞选的路线”之间进行选择——不过，预计川普总统不会做出实际性的宣布。

川普总统将成为周四在佛罗里达州奥兰多会议的头条关注人，他将在会议的最后一天发表讲话，（年会）聚集了来自全国各地的众多保守派和共和党人士。

这将是川普总统卸任后的首次公开露面，正值拜登采取多项行动撤销前任政策之际。

拜登在移民问题上的动作或许最快，他已经采取了一系列措施来铲除川普总统时代的政策，他已停止在南部边境修建隔离墙，结束旅行禁令，并采取措施放宽移民保护协议（MPP）等庇护政策。共和党人警告说，边境危机正在形成之中，这是由于美国在执法和庇护政策上的自由化导致了移民激增。

目前尚不清楚川普总统将在多大程度上抨击2020的大选舞弊，川普总统实际上是赢得了大选（此句话为笔者作了稍微的改动）。

川普总统的讲话将受到密切关注，既有2024年可能参选的迹象——早期民调显示，如果川普总统选择再次参选，他将是共和党初选中轻松的领先者——也有他是否打算谴责共和党内部的政治对手。

一位熟悉川普总统讲话的消息人士上周对《福克斯新闻》说，川普总统将谈论共和党和保守派运动的未来，以及拜登在赦免和边境问题上的政策。

《轴线》（Axios）本周报道称，川普总统将发起对那些反对他的议员进行挑战，并为“拥川的候选人”提供资金和支持。

川普总统最近发表了一些声明，包括关于他的弹劾闹剧和拉什·林堡的逝世，但这将是他卸任后的第一次重要讲话。

密苏里州参议员乔希·霍利（Josh Hawley）、德克萨斯州参议员特德·克鲁兹（Ted Cruz）、南达科他州州长克里斯蒂·诺姆（Kristi Noem）和前国务卿迈克·蓬佩奥（Mike Pompeo）等保守派大腕将出席，但共和党的叛徒（叛徒为笔者加注），如犹他州参议员罗姆尼（Mitt Romney）和怀俄明州众议员切尼（Liz Cheney）不会出席。
