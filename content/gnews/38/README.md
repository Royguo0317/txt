2020年底开播的美国大选这部连续剧仍在全球热播着，谁也说不好大结局将在哪一天上演。

现任总统本色出演了一生中最倾情的男一号，其剧情之惊险、编剧之滑稽使得他之前演绎的影视作品完全无法与之相比。

现任副总统虽初涉演艺圈，却也尽其所能的呈现了鬼使神差终露狰容的角色。

原副总统则出演一个名副其实的大丑角，但却无怨无悔真心卖力，合盘豁出了家族丑闻。

就连大名鼎鼎的好莱坞名角都甘为这部大戏出演配角，甚至还是丑角。

各大媒体充其量也就混了个群众演员，却也在享用了迟迟不肯露面的大导演赏赐的劣质盒饭后，异口同声步调整齐，仿佛走火入魔，忘记了自己是啥工种。

剧中看点颇多，精彩纷呈。

比如令观众们非常震惊的一幕是，美国大选竟然可以夜半掺假。

比如令观众们空前惊恐的一幕是，鬼魂们也受邀纷纷赶来参演。

比如令观众们难以置信的一幕是，媒体的权利可以高至禁止现任总统发声。

然而，这些令人瞠目结舌的演绎，还够不上是该剧的最高看点。

剧中另有一举足轻重的角色，那便是新中国联邦。

该角色高举蓝色七芒星旗，成员来自世界各地，一改演员围着导演指挥棒转的俗套，勇敢智慧的牵着导演的鼻子走。团队决战15天，撕下了第一大丑角的底裤，令该剧导演尴尬无比，不得不频频改戏。

铺天盖地的网络宣传，唤醒了亿万民众，令该剧导演防不胜防漏洞百出、傲慢殆尽。尽管投资无度，却不敢站出来承认自己是导演。

不知该剧将如何收宫，但有一点可以确信，那就是将由新中国联邦来告诉世界：谁，才是美国大选的幕后导演？
