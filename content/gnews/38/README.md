本周早些时候，在欧盟、美国、英国和加拿大星期一宣布对中共国官员实施制裁，中共国否认了有关中共国官员在新疆侵犯人权的指控。北京方面回击称将对欧洲议员、学者和研究机构实施报复性制裁。

2020年，中共官方媒体曾报道了H＆M的一份声明，该声明中称H&M对新疆人被强迫从事劳动的报道深表关切，而且强调从未从中共国地区采购棉花等原材料。目前还不清楚为什么H&M的声明会再次出现在公众视野中。

政府呼吁阻止外国品牌玷污中共国的声誉，这在社交媒体上引发了一场狂潮，网民开始寻找外国零售商此前发表的有关新疆的其他声明。

耐克公司（Nike Inc.）在一次未注明日期的声明中表示，它对强迫劳动的报道感到“担忧”。德国运动服公司阿迪达斯（Adidas）也是如此。许多中共国网民表示，他们将停止购买耐克，并将支持李宁和安踏等本地品牌，甚至有人直截了当地要求阿迪达斯离开中共国。

国产品牌安踏和李宁在陆续发表声明称将继续使用新疆棉花后，安踏体育用品有限公司股价周四在香港上涨了6％以上。李宁公司的股价飙升了7％以上。

狂热的中共国网民还瞄准了促进棉花可持续生产的全球组织BCI，该组织在10月曾表示，出于对人权的担忧，将暂停批准在2020～2021年度从新疆采购的棉花。一位网民写道：“如果你抵制新疆棉花，我们将抵制你。要么阿迪达斯退出BCI，要么退出中（共）国。”BCI的成员包括人们熟知的耐克、阿迪达斯和日本迅销等品牌。

耐克、阿迪达斯和BCI没有立即回应置评请求。

H&M为了平息这场突如其来的骚动,在周三表示，它尊重中共国消费者，并承诺在中共国进行长期投资和发展。但是到周四早上，在百度地图上搜索H＆M商店已经没有结果，而它在天猫上的官方旗舰店也已经无法访问。

新疆乌鲁首府木齐市的一家百货商店在其网站上发文称已关闭H＆M部门，并要求该公司为“散布谣言”并损害新疆地区和中国的利益道歉。

中共官媒《人民日报》发起了一场社交媒体运动，鼓动国人支持来自新疆的棉花。它在官方微博上发布的“我支持新疆棉花”至今已获得220万个点赞。

习总加速师顶着种族灭绝的大帽子，开着马列毛的破车，猛踩着战狼外交的油门，把中共政府赶到了孤家寡人的境地。世界各国纷纷开始或准备与中共国在各方面进行切割，而且这种切割已经从国家和政府层面延伸到民用的商务贸易等各个领域，这次的H&M事件标志着全球灭共的大潮已然涌起，即将把中共冲刷的尸骨无存。
