宋朝人均GDP达2280美元，整个宋朝占世界的GDP的65%，北宋80%，南宋50%，是中国历史最富有的朝代。

明朝的万历时期占世界GDP的80%，整个明朝的GDP是45%，可是明朝末期发生灾荒和战争使其从80%迅速下滑至一半，在明朝中后期部分江南地区已经恢复至宋朝水平。清朝，即使是在没落的清朝，也占世界GDP在35%-10%之间，这个在全球的排行也是位于世界前列。这个也是慈禧向全球宣战的底气，当时的势力确实有这个势力。

但是自从邪恶的中共政权进入到中华大地后，老百姓就一直处于水深火热之中。大饥荒饿死人，文化大革命被整死的人，法轮功被迫害的人，新疆集中营被随意屠杀的兄弟姐们，东方的耶路撒冷香港孩子们为了自由付出的鲜活生命，这些惨不忍睹的场景都是邪恶中共政权造成的。另外去年李克强还在说，目前中共国1000块收入的百姓还有6亿人。但是习包子为首的邪恶中共，天天在墙内给百姓洗脑，伟大的国要给世界指明方向，要引领世界潮流，要给大家把脉。但是百姓创造的财富被邪恶中共的几大盗国贼全部掏空。留给百姓和后代的是一个千疮百孔的烂摊子。在中共释放病毒后，对病毒无法控制的时候，居然想出来，弃鄂保京海的邪恶想法，这个就是把百姓的性命视为草芥，根本都不管子民的死活。跟着共产党走进火葬场，现在又在大力宣传要脱贫攻坚战，就是为即将到来的粮食短缺，资源短缺造势，同时也会把这个锅在着手准备全部甩出去，准备打台湾，转移矛盾，顺理成章的把墙内的矛盾进行转移。但是邪恶中共这个计划1000%不会达成，因为有爆料革命引领的世界力量在唤醒大家，已经有大批量的体制内战友和有正义感的战友都已经清醒，这些以文贵为代表的爆料革命的新中国联邦人是绝对不会让邪恶中共的计划达成的。全球的战友们一起加油灭掉中共。
