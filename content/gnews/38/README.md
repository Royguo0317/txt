**12/25/2020 ****路德时评（安墨谈）**

**阿里巴巴为重组一天被瘦身近千亿美金**

马云想方设法瘦身配合中共完成苦肉计，旨在未来摆脱美国对中概股制裁。其很有可能是剥离坏账，过滤优良资产，化整为零，择优上市，而最后赔钱损失的则是不明就里的散户。此外对互联网公司的反垄断调查工作量巨大，中共只不过是走个过场，用一纸调查令换取阿里巴巴面对美国审查的一枚护身符。若阿里巴巴此次成功躲过审查，未来其他中共公司照本唱戏即可。

**中共替文革知青下乡平反是故技重施**

中共开始舆论铺垫大力弘扬知青文化，肯定上山下乡的伟大成就。然而事实是当年知青被虐待侵犯案例比比皆是，大量学子下乡无异于焚书坑儒，对学生洗脑又可控制无数家庭。此次鼓吹上山下乡，说明其很可能做好了准备把未来中国五十年要求进步民主的思想根基扼杀断绝。

**中共开始主推党领导经济**

为了适应下一阶段的艰难国际形势，中共决定加强党对经济的领导，加强政治意识。说明习要从王的手中把经济拿过来，加强自己对经济的影响。未来国内的经济重点放到讲政治上，一切以政治为首位，客观规律是次要。政治立场是决定性的，经济发展好不好不重要，政治站队正确才是关键，百姓死活更是无关痛痒。计划经济，公私合营等老戏码若隐若现。

**粮食浪费法即将正式出台**

此反粮食浪费法案推进如此之快，是因为中共外部环境太差，内外交困之势已经形成。不能对外开源只有对内节流，节约粮食迫在眉睫。但就食品浪费标准化，顾客点餐浪费收费规则等皆是一笔糊涂账。

**中共将以疫为由镇压百姓**

墙内危难时刻有钱不一定能买到粮食。如果老百姓因为没有粮食发起抗议，中共就会可以利用疫情爆发为借口封闭整个小区或村落，这样老百姓就会顺理成章受到镇压，而且不会有任何人同情和帮助。
