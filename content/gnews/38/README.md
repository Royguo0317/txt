最后一点，我想和各位聊一聊这名女士提到的刹车失灵，根据中共官媒《人民日报》子频道对这个部分的报道，我们可以简单的得到几个信息。根据这名张女士的说辞，她父亲驾驶她的model 3 汽车在离路口200m左右时依然保持着60km/h的高速，并开始刹车，但刹车出了故障，车辆以50km/h继续向前行驶。虽然特斯拉对其车速和车辆性能问题发出了声明，然而，这名张女士却拒绝任何第三方机构进行检验，但笔者依然从仅有的这些信息中找到了线索。线索依然是来自于model 3车型，model 3由于采用电能驱动，动能回收系统作为保持长续航的关键科技在这部车上也有搭载。当电门踏板松开的时候，动能回收系统将会自动介入，回收动能的同时大幅降低车速。这就意味着传统刹车介入的机会将会更低，不仅出现故障的概率更低，即使刹车无法工作，车辆也应能够完成减速动作。而如果说这名女士所驾驶的model 3汽车在动能回收系统正常工作的时候依然能够保持50km/h的速度行驶，我想这辆车的问题将远不止刹车失灵这么简单。但她依然将问题指责为特斯拉刹车问题，足见其对车辆技术特征并不熟悉，相信她也应该不可能将车辆的动能回收系统的设定调整为低或取消，毕竟特斯拉在2021年1月18日model Y 交付之际，将国产特斯拉的动能回收设置为仅有标准一种选项。基于以上的内容笔者更倾向于相信特斯拉所公布的声明，并且注意到118.5km/h的速度在国道上超速超过限速20%不足50%应罚款150元。
