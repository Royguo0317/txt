在有关人类事务的发展过程中，当一个民族必须解除其和另一个民族之间的政治联系，并在世界各国之间依照自然法则和自然神明，取得独立和平等的地位时，出于对人类公义的尊重，必须宣布他们不得不独立的原因。

我们认为下面这些真理是不证自明的：人人生而平等，造物主赋予他们若干不可剥夺的权利，其中包括生命权、自由权和追求幸福的权利。为了保障这些权利，人们才在他们之间建立政府，而政府之正当权力，则来自被统治者同意。任何形式的政府，只要破坏上述目的，人民就有权利改变或者废除他并建立新政府；新政府赖以奠基的原则，得以组织权力的方式，都要最大可能地增进民众的安全和幸福。的确，从慎重考虑，不应当由于轻微和短暂的原因而改变成立多年的政府。过去的一切经验也都说明，任何苦难，只要尚能忍受，人类都宁愿容忍，而无意废除他们久已习惯了的政府来恢复自身的权益。但是，当政府一贯滥用职权、强取豪夺，一成不变地追逐这一目标，足以证明它旨在把人民置于绝对专制统治之下时，那么，人民就有权利，也有义务推翻这个政府并为他们未来的安全建立新的保障。

在美国国父们以此豪迈庄严的宣言，向世界发出了独立的声音后，给予美国人之所以是美国公民的保证，美利坚合众国孕育而生，它奠基了美国法律的一切基础，开创了美国伟大且又辉煌的历史，影响着延续至今的人类文明。

在这个最初的契约之下，马丁.路德.金恩曾经决不相信正义的银行会破产。决不相信这个国家巨大的机会宝库会存款不足。因此,在1963年以实现美国立国信条的真谛：“我们认为这些真理不证自明：人人生而平等。”为名，来到华盛顿，兑现这张支票——自由和正义的保障。

他以《我有一个梦想》的演讲，激发了人们对这个庄严承诺的信心，推动了美国在1964年通过《民权法案》。

随后在1965年3月7日，因赛尔玛投票权引发的“血腥星期天”事件，金恩以《还要多久？快了》的演讲，再次向这个神圣的契约要求兑现，因为坚信“因为道德光芒万丈，而正义更加耀眼辉煌”。

而当今的美国社会，正再次陷入如此的僵局，左派的社会主义者，共产主义者，肆无忌惮的打压传统的美国人。在1月6日的国会山事件中，重演“血腥的星期天”。

犹如专制政权的社交平台，审查人们的言论，打压使得他们不悦的新闻，试图将美国的社会，转变成共产主义的制度，以公有制之名，合法的抢夺美国公民的财产，试图让美国人民活在马克思列宁主义的恐怖之中，不惜出卖因那个不证自明的真理而伟大的美国人的权利与利益，谋求当权者的利益最大化。

所有的行为，都是对《独立宣言》的讽刺，令人蒙羞的可耻。

面对这样，传统派的美国公民们已经忍耐许久，是否应该对宣言中说到的，任何形式的政府，只要破坏上述目的，人民就有权利改变或者废除他并建立新政府。而开始思考。
