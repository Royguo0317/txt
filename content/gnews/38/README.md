在回顾非洲国家的经济状况时，人们不能不对这些国家目前正在发生的经济危机进行思考。大多数非洲国家似乎都欠了一个国家的债。中共国。似乎有一个精心策划的计划，来陷害每一个希望其基础设施得到发展或踏上进步之路的非洲国家。有时，那些在腐败中根深蒂固的政党希望这种“发展”能给他们带来杠杆作用，使他们能在有利的位置上与公民接触。但是，非洲公众似乎正在慢慢觉醒，意识到他们所处的现实。由于具有战略性的地理位置，非洲国家，特别是那些拥有港口的国家已经开始意识到，中共国一直在用影响其主权的条款来对待他们，这些条款有可能剥夺他们进入自己港口的权利，他们的基础设施是通过几十年来对丰富的自然资源的开采而发展起来的，等等。肯尼亚也不例外，也落入了中共国的这种政策陷阱。

情况非常严峻，目前，肯尼亚迫切需要偿还价值超过64亿美元的对华未偿贷款。这些贷款大部分是由中共国进出口银行提供的。正如斯里兰卡岛国的情况一样，据预测，肯尼亚政府也将面临即将到来的偿还贷款的危机，并最终开始交出/移交关键的基础设施，而这些基础设施正是用从中共国银行获得的资金来支付当初建造这些项目的中共国建筑商。这种债务循环并没有因为接管项目而结束。为了换取贷款偿还方面的放松，中共国向肯尼亚当局勒索了即将到来的项目。这些活动有助于维持债务陷阱，而中共国则象征性地给予货币豁免或允许推迟偿还贷款的时间。

最近，肯尼亚也一直在系统地忽视其活动家和法律社会团体提出的请愿，抗议中共国路桥公司（CRBC）通过藐视合法的采购程序获得一个基础设施项目的投标。中共国路桥公司早前获得了价值近47亿美元的标准轨距铁路项目。即使在华为事件中，虽然美国电信专家对该公司可疑的间谍活动以及与中共国政治和军事领导人的关系发出了警告，但肯尼亚没有吸取任何教训，其最大的电信运营商Safaricom已经在全国范围内部署了由华为驱动的第五代（5G）移动网络。对于一个没有完全熟悉2G和3G就直接跳到5G网络的国家来说，这是一种讽刺。这种模式是可以看出的，因为5G网络的所有技术进步将导致向中共国的蓝眼电信间谍网络华为泄露国家机密。

在另一个重要的发展中，肯尼亚石油和天然气民间社会平台显然要求透明，以揭开肯尼亚当局与一家中共国石油化工公司—中共国化工集团在2019年达成交易的条款和条件。在双方达成协议后，中共国公司赢得了购买石油的竞标，但没有透露给公司国库的增加额或公司从销售肯尼亚原油中获得的净收入。中标后，正如人们所期待的那样，中共国从肯尼亚进口原油运输。

不仅仅是在开采自然资源方面，中共国还公然侵犯人权，特别是侵犯少数民族和不同族裔的人权。肯尼亚人亲眼目睹了这一点，2018年，一家中共国摩托车公司的一些中共国官员虐待肯尼亚工人，称他们为“猴子”。尽管肯尼亚人普遍意识到中共国人往往具有种族主义和仇外态度，但由于各种经济和地缘政治因素，他们几乎无法代表自己的关切或为自己的权利而斗争，而不得不向中共国的力量投降。然而，这种涉及身份以及民族自豪感和尊严的问题不可能被长期掩盖，今天的肯尼亚人越来越关注自己国家的种族主义和歧视行为，而这些行为是以中共国人为代价的。

除了口头上的言论和贬低，中共国人的种族主义歧视态度甚至在他们的基本人际交往中都可以看到，尽管可以忽略不计。中共国企业建造的工厂为中共国官员/劳动力提供单独的厕所；而肯尼亚工人则使用不同的厕所，这是按种族区分的明显标志。此外，如果一个肯尼亚雇员上班迟到15分钟，他/她可能会失去当天6到7个小时的工资，而管理中共国人的经济参数却明显不同。

除了种族歧视，我们还可以说，中共国人和肯尼亚人之间几乎没有任何社会交往。中共国人显然看不起肯尼亚人，认为他们不值得平等交往。但具有讽刺意味的是，肯尼亚有大量的中共国人口，大约有4万多人。中共国人住在内罗毕的大型住宅区里，通常都很隐蔽，只与其他中国人为伴。中共国人和采取单独的交通设施进行通勤，这无形中留下了很少的社会互动空间。

考虑到人与人之间的接触或缺乏接触，可以得出结论，中共国人在肯尼亚的活动似乎带有明显的种族偏见。对于一个依靠另一个国家的资源来实现自身“发展”的国家来说，这显然是对当地居民和他们的生存方式的蔑视。粗略地看一下中共国人正在处理的项目，就会发现他们专注于促进债务陷阱，开采珍贵的资源和侵占具有战略意义的基础设施。承担和偿还债务的整个过程是一个巧妙设计的自我维持活动，它螺旋式上升，成为一个巨大的问题，各国几乎看不到。当他们最终看到时，已经太晚了。希望肯尼亚能够及时走出这个泥潭。
