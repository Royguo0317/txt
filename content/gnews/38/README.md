笔者5月7日发文，报道了亚利桑那州选举腐败和可疑的问题，其中第二个问题，“马里科帕（Maricopa）郡选举管理团队没有进入投票系统的“管理”权限【1】，这个荒唐的问题，得到了证人的验证。

《门道专家》（the gateway pundit）发表文章，“马里科帕郡选举见证人证明，多米尼（Dominion）主持了整个选举—郡官员和观察员从来没有访问系统或获得过密码！”

全文如下：

早在2020年11月30日，马里科帕郡选举见证人简·布莱恩特（Jan Bryant）在亚利桑那州议会作证。

简有很强的项目管理背景，她无法相信自己在亚利桑那州马里科帕郡2020年选举期间目睹的一切。

简在2020年11月30日说，马里科帕郡的官员没有参加选举的管理工作！多米尼的员工约翰（John）和布鲁斯（Bruce）做了这些工作。

简的证词或许可以解释为什么马里科帕郡的官员没有管理员密码，也没有进入多米尼投票机的权限。

简还宣誓作证说，郡的员工，甚至不是IT人员，被允许进入投票机，她看到多米尼的员工在计票室用笔记本电脑。

简在马里科帕郡选举部门（MCTEC：Maricopa County Elections Department）工作了6天，她有MBA学位，拥有技术方面的项目管理背景。

以下是简作证时的证词：

……回到我的开场白，我想告诉你，我确实有很强的项目管理技术背景，我想说，这不是每个人都在夸耀我们将如何运作，以及我们工作的房间有多紧。我在马里科帕郡选举部门（MCTEC）的制表中心，你知道房间是玻璃隔断，里面有服务器，所有的电脑都在一根你能看到的电缆上，这些都不重要，因为有两件事，第一，多米尼的员工是唯一运行这些设备的人。

我在制表中心待了六天，日班和夜班，没有郡里的员工，没有IT人员，没有其他人碰过任何软件。他们（多米尼）为计票员做了所有的培训，他们管理编制了所有的报告。所以我在房间里的第一天就提到了这个，我作为一个有背景的人，我说这似乎不对。过一百万年我也不会把我的公司最重要的事情交给别人，却只有两个外来的人（多米尼恩的布鲁斯和约翰），他们完全控制了一切。

我还参加了（2%）随机抽签的手工审核，挑选选票，你知道，决定我们要选哪个箱子。尽管如此，多米尼还是为它写了一份报告（要拉哪个箱子）。所以没有人，我的意思是，他们（多米尼）确切地知道什么是在手工审计箱子里。这也让我很紧张，他们也知道我们什么时候拉最后一个箱子，大概是在他们停止计票前8天，作为一个管理过大型项目的高级经理，就是过一百万年我也不会允许我所看到的这些事情在那里发生。

回顾：被抽检的、2%强制人工审核密封（选票）箱是由多米尼员工决定，通过他们的EMS软件，而不是随机决定的。

我想确定的另一件事，最后一周他们在计票，我想是星期二，当我在星期五离开之前，我没有注意到（玻璃）墙后面的房间里有一台笔记本电脑，多米尼的员工约翰正在处理。当我星期二进来，在他们手工录入选票的地方走来走去时，我注意到笔记本电脑放在那里，约翰正在处理。如果你把笔记本电脑带进带出房间，这就不是一个安全的房间，谁知道那里发生了什么？在里面的50个台式机中，每一台都有一排USB接口。所以当我听到这里的一些领导说“哦，这里很安全，没有东西可以进出”时，我非常生气，那不是真的，我会告诉你这可能是我遇到的最大问题。

（全文完）
