## 【世事解评】形式防沉迷，难解“手机儿童”现状

作者：纽约香草山写作组  鹰(文言)

环球网5月2日转载半月谈网文章，农村孩子沉迷手机游戏、小视频，过度使用手机现象堪忧。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/02042954/%E5%9B%BE%E7%89%8733.png)图片来源：环球网（新华社/王鹏）

虽然今年中共两会期间，不少人大代表倡议未成年人游戏防护，通过立法、立规等方式防沉迷，但实操上却依旧困难重重。截至2020年12月，国内应用市场上游戏类APP达88.7万款，占APP数量总量的25.7%，虽然2019年就出台《关于防止未成年人沉迷网络游戏的通知》，但一方面是行业数据不互通，另一方面则由于一些大型游戏企业以防沉迷之名行充会员之实，防护系统仍形同虚设。

除网络游戏的泛滥外，国内对竞技游戏的分级化监管、考证式的行动也刺激了社会上对网游的热衷，高校开设相关专业，家长们对孩子接触网游排斥减弱，这些“南辕北辙”的现象反而更增大了玩手机、玩游戏对儿童的吸引力；在手机游戏之外，影响更为巨大的是小视频APP的普及。为配合中共的数据渗透，抖音作为“先驱”输送国外进行大数据监测和思想行为引导，而在国内，抖音、西瓜、火山等软件更是在青中老年中被广泛使用，家庭、社会乃至媒体对小视频的推崇，都加深了儿童对短视频软件的沉迷。

由于2020年CCP病毒大爆发导致线上教育在国内风行，线下培训机构的接连爆雷变相助涨了网络授课的热度，于是各大网课纷纷上线，无论是红色类别的党史教育，还是中小学日常授课，以及网络会议等都通过线上平台开展。这一现象增加了儿童对手机的使用时间，另一方面由于网络信息繁杂、网游短视频等诱惑遍布，线上教育沦为儿童沉迷手机的助力。

近年来外出打工的农民比例攀升，农村留守儿童数量和规模都十分庞大，而“全民脱贫”更使基本的生活保障和教育都得不到满足，在这种情形下谈农村儿童的防沉迷更显得十分可笑。大型游戏公司一方面推出形式防沉迷系统，另一方面游戏产品“推陈出新”，校外兴趣班依旧普及、线上教育又增大儿童对手机的使用时长。再有以抖音为首的短视频平台继续做大做强，中共奴化弱化大众思想的行径没有丝毫改变，而娱乐化、红色洗脑化青少年就更加顺理成章了。所以在现行的社会风气和中共驭民五术下，儿童防沉迷只是一个不可能实现的空想。
