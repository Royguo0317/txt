## 【世事解评】美国加大科研投入应对中共的“斯普特尼克时刻”
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/1119933/)

作者：纽约香草山健身部  童谣

根据美国之音报道，美国参议院就提高科技领域竞争力举行听证会，以应对当下的“斯普特尼克时刻”。专家表示美国应整合全面的创新生态系统，从联邦政府、高校、公司机构、社会组织各个方面，建立一套涵盖从基础研究到应用开发的组织结构，以抗衡中共国“自上而下的国家主导创新”。

![]()![](https://gnews.org/wp-content/uploads/2021/04/图片18-3.png)图片来源：VOA

冷战初期，苏联在1957年向太空发射“斯普特尼克”号人造卫星，当时被形容为“斯普特尼克时刻”，因为随即引发美国总统艾森豪威尔将科技上升为国家任务，成立了美国航空航天局（NASA），并大幅增加对研发的支持。

当前在中共所引发的超限战时刻，世界格局发生着分裂般的变化。曾经在美国全力支持下，中共才得以加入WTO进入国际贸易舞台，开始了之后一系列如鱼得水的蓝金黄行动，进行不平等交易并偷窃西方国家先进技术，游离在国际法之外屡试不爽。

当前中共这只被美国养大的狗开始冲着世界狂吠，疯狂发展5G等尖端技术，渗透美国各个信息基础设备，并不断发展现代化军事，制造东海、南海、以及台海危机。在病毒生物武器入侵一年之久后，美国才终于意识到了中共的“斯普特尼克”其实早已发射，潘多拉的盒子已被打开，美国将不会甘心继续被蒙在鼓中。

共和党议员维克尔在美国科技话题的听证会上说，中共国迅速成为威胁美国的科技强国，美国应该推进科学生态系统以保持领导力。

而有外星人技术相辅助的美国在科技领域本身就落下中共几条街，在决战时刻美洲狮一旦醒来，中共的虚假“墙国大厦”必是摧枯拉朽般倒塌。
