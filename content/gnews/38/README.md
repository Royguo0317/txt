【备注：布赖恩·肯普（Brian Kemp），是一位美国商人和政治家，

自2019年1月起担任佐治亚州第83位州长。他是共和党人，曾于2010年至2018年担任佐治亚州第二十七任国务卿，并于2003年成为佐治亚州参议院议员直到2007年。】

【备注：杰夫·邓肯（Jeffrey Darren Duncan；1966年1月7日－），自2011年开始，他是南卡罗来纳州第3选举区选出的美国众议院议员。党籍共和党。在当选美国众议院议员之前，邓肯曾担任南卡罗来纳州众议院的议员。】

【备注：布拉德福德•杰伊•拉芬斯伯格（Bradford Jay Raffensperger，1955年5月18日出生），美国佐治亚州的政治家、商人和土木工程师。共和党人，担任佐治亚州州务卿，自2019年1月起任职。曾在佐治亚州众议院任职，代表第50区。】

#### 2、班农对精英们怒吼：你们是懦夫，而你们的孩子们正在成为白痴

在爱国者日，班农说，列克星敦和康科德的人（发起的北美独立战争）建立了人类已知的任何国家中最富有和最繁荣的国家。

他说：“ [列克星敦和康科德的英雄们]对你们还不至于感到厌恶，他们只是不理解你们。你们龟缩在家里，没有惯有的风度，没有共同的目的，没有人来到这座桥上，举行一个仪式来回顾和思考过去和现在？”

‘这帮精英的孩子会成为白痴’。

#### 3、重磅：新增新冠病毒患者中60%为疫苗注射者

一位耶鲁大学的医学博士说，在新确诊的冠状病毒患者中，超过一半的人已经接种了疫苗。

哈维•里施博士（Dr. Harvey Risch）加入了《战斗室》，解释了公共卫生官员的（中共病毒）疫苗犹如“诱饵和开关”（一种商业欺诈销售策略），他们使用的治疗标准与他们公布的完全不同。

【**备注：**哈维•里施博士（Dr. Harvey Risch）是耶鲁大学公共卫生学院和耶鲁医学院流行病学和公共卫生系的流行病学教授。里施博士在加州大学圣地亚哥分校获得医学博士学位，在芝加哥大学获得博士学位。】

“临床医生告诉我的是，他们看到的新冠病毒病例中，超过一半的治疗对象是已经接种过疫苗的人”，里施博士说：“他们估计在他们所治疗的新患者中，有60%的人是接种过疫苗的人。”

里施博士说，医疗机构对疫苗的有效性撒了谎，因为他们正在引导每个人进行疫苗接种。

“他们担心如果你说这个疫苗只有50%或60%的有效性……那么人们就不会去接种疫苗，”里施博士说。

里施博士说，很明显，福奇（Fauci）博士的计划与公共卫生议程不同，因为他不会诚实地解答群体免疫力问题。

里施说，去年秋天，在没有疫苗的情况下，美国有一半的州已经超过了群体免疫力所需的标准。

“群体免疫力只会从感染曲线达到峰值开始”，里施解释说：“在这峰值点之超过了50%或60%的人群将获得群体免疫力。”

“要让感染迅速下降，我们需要的不仅仅是群体免疫力”，他说：“这就是我们在南达科他州、北达科他州、田纳西州、罗德岛州、犹他州去年秋天10月和11月看到的情况。”

“他们在任何疫苗接种开始之前就超过了群体免疫力所需感染人群”， 里施博士说：“他们已经积累了足够的病例。”

“美国有一半的州现在都已经超过了群体免疫力所需人数”，他说：“病例正在急剧下降。”

里施博士也说，封锁是“适得其反”的，只会耽误他们的居民达到群体免疫力的进度。

福奇博士不愿意回答基本的问题，比如为什么已经接种过疫苗或已经接触过病毒的人也要戴口罩，这表明他有另一个目的计划。

“[他]其实上不愿意说出群体免疫这个词”，里施博士说：“我不明白为什么会这样，除非有其他计划的动机，而这些计划与治疗新冠病毒大流行病是不一样的。也没有别的什么目的可解释了。”

“还有另一个不明显的目的计划”，里施博士说：“看看[世界卫生组织]WHO，世卫组织悄悄地从群体免疫中删除了自然感染引起免疫力贡献的资料。这在科学界引起一片哗然，这违背了至少一百年以来验证过的科学结论。”

里施博士说，对“政治和金融”的担心已经影响到了医疗机构，并且正在影响福奇博士和其他人的行为。

里施说，中共病毒的治疗方法有很多，这在科学界是一个“公开的秘密”。

“早期的门诊治疗是有效的，而且非常有效”，他说。这包括羟氯喹、伊维菌素、阿司匹林、抗生素和其他廉价的治疗方法。

“我们没有必要化三千美元一粒药的治疗费用，”里施博士说。

“存在有两类人”，他说：“一类人是真正经历和看过这些治疗方法的人。而另一类人嘲笑者、学者，而他们从来没有治疗过病人，他们对这些治疗的实际效果懵懂无知。”
