# **中共是麦康奈尔的老板**

[!\[\]()!\[\](https://spark.adobe.com/page/uNPRcKC9DnHAZ/images/f06e65fc-1b39-492b-971d-8e52eab52425.jpg?asset_id=b77afe47-cf0b-4be3-b514-bc67eba29a0c&amp;img_etag=%221e3c3f88f0e90ad2b47376f2e0fd725a%22&amp;size=2560)](https://spark.adobe.com/page/uNPRcKC9DnHAZ/images/f06e65fc-1b39-492b-971d-8e52eab52425.jpg?asset_id=b77afe47-cf0b-4be3-b514-bc67eba29a0c&amp;img_etag=%221e3c3f88f0e90ad2b47376f2e0fd725a%22&amp;size=1024)

本周二，共和党参议员领袖麦康奈尔公开承认民主党总统候选人拜登赢得了2020年的大选，并呼吁其他共和党成员也支持拜登，着实让人惊掉下巴。

本次大选涉及严重的舞弊，有利于川普总统的相关法律诉讼也还在各个关键州紧锣密鼓地进行中，就现已公开的大量的人证物证显示，大选在程序、邮寄选票、监票和投票机等各个环节皆有问题，而作为共和党参议员的领袖即便是为了维护本党的利益，也应该支持川普总统的法律诉讼，最起码在所有法律程序结束之后再表态也为时不晚。但是，麦康奈尔却偏偏无视民意和证人的证词，在选举结果尚不明朗的情况下，选择和民主党站在一起，令人匪夷所思。俗话说事出反常必有妖，不错，只要对麦康奈尔的华裔妻子及其家族有所了解，就不难发现其中的奥秘。

本周一，美国政府责任研究所（Government Accountability Institute）主席施韦泽（Peter Schweizer）参加了的SiriusXM电台《Breitbart News Tonight》节目，并在节目中与主持人丽贝卡-曼苏尔（Rebecca Mansour）和乔尔-波拉克（Joel Pollak）讨论了麦康奈尔与中共国之间的关系，并以麦康奈尔为例，详细阐述了美国的政治精英们如何暗中腐败，避开财务公开的相关法规，将政治影响力转化为财富。

[!\[\]()!\[\](https://spark.adobe.com/page/uNPRcKC9DnHAZ/images/e1b292e2-9103-40ef-8353-070378c1fad1.jpg?asset_id=e631a90c-8a29-4df2-b78f-6bd9c9088a92&amp;img_etag=%22c37622a45faf6e8794f68939871e6c8f%22&amp;size=2560)](https://spark.adobe.com/page/uNPRcKC9DnHAZ/images/e1b292e2-9103-40ef-8353-070378c1fad1.jpg?asset_id=e631a90c-8a29-4df2-b78f-6bd9c9088a92&amp;img_etag=%22c37622a45faf6e8794f68939871e6c8f%22&amp;size=1024)

康奈尔(Mitch McConnell) 和妻子赵小兰（Elaine Chao）

施韦泽（Schweizer）说，康奈尔(Mitch McConnell) 的妻子赵小兰（Elaine Chao）是川普政府的交通部长，她的家族与中共政府有着很深的经济往来和金钱关系。赵家是做航运生意的，赵父赵锡成 （James Chao）与中共80年代领导人江泽民是旧时同学，借助江泽民的关系，赵父和赵小兰的妹妹赵安吉（Angela Chao）在2007年和2008年加入了中国国家造船公司的董事会 —— CSSC控股。事实上，在1993年，参议员麦康奈尔和他的岳父赵锡成作为中国国家造船公司的客人，一起访问了中国。中国国家造船公司是中共国最大的国防承包商，它所支持的各大项目将使中共的海军实力在2030年超过美国海军。他们不仅为赵家建造船只，还为赵家提供极其低息的贷款，帮助赵家购买部分船只，并为赵家的船提供船员，以及中共国有企业在太平洋航线上的大宗航运合同。这似乎与渤海金控借资给亨特拜登认购渤海华美的股份如出一辙，说白了，就是送钱。

事实上，不光赵小兰家族闷声大发财，这些财富也以巧妙的方式流进了麦康奈尔个人的口袋。几年前，赵小兰给了麦康奈尔一份500万到2500万美元的礼物，这使他的净资产一夜之间翻了四倍多。天晓得才貌双全的航运大亨爱女、华府第一位华裔女部长赵小兰在运作夫婿竞选参议院领袖时，会砸下多少金钱。而这些钱又源于她娘家的航运事业，源于中国国家造船公司。

施韦泽（Peter Schweizer）还提到麦康奈尔在关于中共国问题上的表现得“越来越软 ”，他将此归结为麦康奈尔家族与中共的关系和他们与中共国的经济往来。他说：“如果你把过去二十年来麦康奈尔参议员在中国问题上的立场一一列出，你猜怎么着？你会非常地震惊，他在与中共国有关的问题上变得越来越软弱，我认为他非常清楚自己家族未来的财富与中共政府对他的表现是否满意和他对中共政府的诚意息息相关。”
