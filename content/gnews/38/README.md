据《 布雷特巴特新闻》3月18日报道，众议院司法委员会主席杰里·纳德勒(纽约州民主党人)在听证会上指责在中共病毒大流行问题上“妖魔化中共国”的立法者是导致如今亚裔美国人受攻击增加的主要原因。

他说：“重要的是要认识到，这并不是仅仅因为人们对中共病毒大流行的恐惧而自发产生的。部分责任完全归咎于妖魔化中共国的政治领导人–既是因为病毒，也是因为持续的地缘政治紧张局势–反过来，亚裔美国人也受到了伤害，“纳德勒在一场关于”针对亚裔美国人的歧视和暴力“。“语言是有力量的。我们说的话很重要。我们如何对待彼此很重要，“ 他补充道，“我们今天进行的对话早就应该进行了，国会对这一问题的关注非常重要。”

纳德勒没有点名批评任何针对中共国的议员。根据仇恨与极端主义研究中心的一份报告，从2019年到2020年，针对亚裔美国人的仇恨犯罪上升了149%。在纳德勒发表评论之前，佐治亚州伍德斯托克的罗伯特·亚伦·朗(Robert Aaron Long)承认周二晚上在亚特兰大大都市区的按摩院杀害了8人，其中6人有亚洲血统。郎（Long）是白人，在杀戮事件发生后不久被捕，并被指控犯有谋杀罪和袭击罪。官员们在周三的新闻发布会上说，郎告诉警方，枪击事件不是出于种族动机，他可能患有“性瘾”。
