# **中共国：计划生育政策保护维吾尔女性免于成为“生育机器”**

[!\[\]()!\[\](https://spark.adobe.com/page/972vEowlTwe5B/images/f4ede103-968c-410b-a17a-85302416c55f.jpg?asset_id=eb4dd157-dbbf-4acd-ad96-3e9cacaac1c1&amp;img_etag=%22d8cba05a6db75bd5e033a0f44bb38d35%22&amp;size=2560)](https://spark.adobe.com/page/972vEowlTwe5B/images/f4ede103-968c-410b-a17a-85302416c55f.jpg?asset_id=eb4dd157-dbbf-4acd-ad96-3e9cacaac1c1&amp;img_etag=%22d8cba05a6db75bd5e033a0f44bb38d35%22&amp;size=1024)

2018年12月3日，人们在中共国新疆西部城市阿图什市的昆山工业园职业技能教育培训服务中心排成一队。

本周中共国驻华盛顿大使称，针对中共国西部少数民族维吾尔人的压制政策包括旨在阻止维吾尔女性成为“生育机器”的计划生育政策。

“研究显示，在根除极端主义思想的过程中，新疆维族妇女的思想得到了解放，性别平等和生育健康得到了提升，这使她们不再是生育机器。她们现在更自信更独立。”大使馆官方推特账号在周四的一条推文中说到。

推文包括来自隶属于中共的《环球时报》的一条报道链接。该报道称，在新疆出生率和自然人口增长率下降“导致了宗教极端主义的消灭”。

该报道基于在新疆所做的一项人口调查。美国官方称，在新疆有100多万维吾尔族穆斯林被投进了集中营，遭受镇压。这是中共政府反恐怖主义运动的一部分。

对于用“生育机器”来描述维吾尔族女性，一位中共大使馆发言人没有给出评论。

但该发言人否认了西方研究者关于中共国在新疆执行强制绝育和其它可疑人口控制措施的报道。

德国学者和中国问题专家阿德里安·曾兹说过，中共政府参与了针对维吾尔人的人口控制。

“中共当局在新疆建立的人口控制制度，其目的在于抑制少数民族人口的增长，同时增加汉族人口。所采取的手段有：增加汉族人口的出生和迁入；采取手术节育方法的严厉措施，使国家能够随意增加或减少少数族裔的人口增长。这就像打开或关上一个水龙头一样。”曾兹先生在6月份发表的一篇报告中写到。

由一家政府机构“新疆发展研究中心”所做的一项中方调查指出，“极端主义”导致当地人口对“计划生育”的抵制。根除极端主义给了维吾尔族女性“在决定是否要孩子时更多的自主权。”

报道称，这些变化不是像一些西方学者所声称的那样，不是强制节育的结果。

根据《环球时报》，未明确的计划生育政策已在该地区按照中共国法律“全面执行”。

2017年在中共国西部出台的管理规定，各个民族都要执行一项计划生育政策，该政策允许城市夫妇生育2个孩子，农村地区的夫妇可以生育3个孩子。

通过使用避孕措施、输卵管结扎和宫内节育器，结果导致出生率从2017年的1.6%下降到2018年的1%，人口增长率也从1.1%下降到0.6%。总体来看，维吾尔族人的数量由2010年的1020万增长至2018年的1270万。维吾尔族是一个主要由穆斯林构成的族群，其支持者们表示他们正在寻求在当地恢复独立的东突厥土耳其斯坦。

《环球时报》说，出生下降“更多是出于个人选择而非政府政策。”

收到对该推文的到投诉后，推特公司回复称说，中共国大使馆的推文没有违反推特的规定准则。
