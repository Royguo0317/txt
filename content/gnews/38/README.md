［墨尔本华人杨怡生被指控违反《反外国干预法》：

65岁的杨怡生（Duong Di Sanh，又名Sunny Duong）为控违反《反外国干预法》，并于今天下午（注：指2020年11月5日）在墨尔本地方法庭出庭。杨怡生在墨尔本东南区的亚裔华人社区赫赫有名，他所在的团体与中国在海外施加影响力的工作有关。杨怡生被控准备实施外国干涉罪（preparing for a foreign interference offence），最高可判10年监禁。但是目前尚不清楚杨怡生被指控罪行的确切性质。据了解，杨怡生已获准保释，并将于2021年3月再次出庭。杨怡生是墨尔本澳华历史博物馆的副主席，也是大洋洲越南柬埔寨老挝华人团体联合会的主席。杨怡生与中国和平统一促进委员会有联系，和统会隶属于中国政府的海外影响力机构——统一战线工作部。］
