## 大重启计划到底是什么——博士军团没有说透的地方

![]()![](https://gnews.org/wp-content/uploads/2021/01/image-250.png)

今天听了《路德社》关于人类大重启计划的评论和博士军团的讨论，感触非常深。不过在听到他们说大重启就是黑暗势力想要建立世界统一的宗教时，感觉到他们好像还没有把它说透，所以在这里想补充两句，给大家解释清楚大重启是什么？是什么导致了它的出现？大重启一旦成功会给人类带来什么样的影响？

简单来说，人类社会一直是在伴随着主体性的更替而不断发展变化的。“主体性”是一个哲学词汇，通俗一点讲，“主体性”就是一个让你围着他转“东西”。既然有“主体”，那么就有“他者”。自耶稣诞生之后的1000多年里，人类都是以上帝作为主体，就连地理大发现也是以上帝的名义去探索未知的世界的。因此，在那个阶段，神是“主体”，人是“他者”。

之后有了文艺复兴，那么复兴啥呢？古罗马时期的人文主义！读过《十日谈》的战友，有没有发现，在《十日谈》里面把和一些女人发生关系看作是上帝的恩典。可是在早期天主教时期，根据《圣经》的典章，这些可都是淫乱罪啊。

当然，说是文艺复兴运动，这只是后人的归纳，它并不是一种文革式的有人发号施令推动的运动，而是一种自然风气的转变。当时的人们觉得天主教廷太过压抑，认为人的基本欲望不该有罪，社会因此开始慢慢地正视人的合理欲望。这个阶段的神依然具有很强的“主体性”，但是人的欲望不再像文艺复兴之前那样地被压制。

后来马丁·路德对天主教提出了挑战。他认为天主教过于腐朽，竟然能通过买免罪符来免罪。他还质问凭啥我们想和上帝交流要通过中保（天主教廷）？凭啥教皇是离上帝最近的人？因此他认为必须改革，每个人都可以根据自己的理解去读《圣经》，也可以和上帝直接交流去忏悔。这就导致了其后的宗教改革。

马丁·路德对传统宗教的挑战意义巨大。想一想当你每天都能和上帝直接交流的时候，你怕不怕上帝惩罚你？因惧怕惩罚，你必须每天勤劳工作，来荣耀上帝，好使不让自己掉入罪中，这样就带来了生产力的巨大提升，这也是资本主义发展的最重要的心里动因。（Ps：可以参考马克斯韦伯的《新教伦理与资本主义精神》）

在这个阶段，虽然神依然具有主体性，但早已不是早期基督教时期那个必须由天主教会解释的神了，因为每个人都可以根据自己的理解来看待神。有没有发现，人们的自主性又变强了！

再后来，启蒙运动开始了。科学的发展让人们越来越能够高效地改造世界，而且原来不明白的东西搞一搞研究发现原来是有原理的。那这样岂不是让原来教会的经典解释啪啪打脸么？因此人们觉得，原来我们自己就可以认识社会，不用通过教会，所以理性主义由此诞生。

当人们更多地依靠理性来理解世界的时候，人们对教会解释的神谕的信念就开始动摇——至少人们证明了地球确实是围着太阳转的。所以这个阶段之后社会的主体性开始了转变，“理性”开始有了更多的思想空间，其实也就是神逐渐让位于人。因此，人渐渐变成世界的主体，神学开始慢慢黯淡下去。

其实这种状态一直持续到现在，人们越来越相信科学，相信自己能通过科学认识一切，也能通过科学而改造一切。**在这种情况下就出现了一种倾向，那就是科学=正确。**我们姑且不论这种说法对与不对，但采用了这种认知方式来改造世界的人，他们的意识形态会渐渐变得“可怕”。**最典型的就是“数据化”，“数据化”把一切事物都用数字量化。**

你有没有发现，当你听到CCP病毒已经导致200万人死亡，你的心里却毫无波澜。因为对你而言，死一个人还是死100万人，对你来说没有什么感觉上的差异。如果是这样，我想说这就是这个社会的机制将你塑造成了这样。过去的人本不是这样，他们的身上会更多地彰显出人性。

所以，到这里你可以看明白了吗？CCP这个无神论的国家，它彻彻底底地抛弃掉了神的主体性，而强调人的主体性，更强调人的绝对理性，这必然会使人滑入丧失人性的深渊。不信神的人，虽然你强调自己无神论，但不知不觉之间，你就可能深陷人的欲望之壑而不自知，或摇摆于一己的善恶念头而不自知，缺少对于人世普遍的大爱与怜悯。

我们再拉回来，根据上面的介绍，你是不是可以发现，人类社会的主体性在从神渐渐地转向了人类本身。如果人类自身确立成为了主体性，那么其他一切都是他者，要围着我们转。在这样的情况下，你会想要做更多的好事呢？还是更多的坏事呢？

不信上帝的存在，没有对上帝的畏惧，这样的人当然是更易屈从于自己的欲望！这个大重启计划就是最大的坏事。它的可怕之处，并不是对人的统治本身。到了这块也就是我觉得博士军团没有说透的地方。他们已经说出了要推翻基督教，用一个大一统宗教来作为全人类的宗教——这个新的宗教就是科学。那么这个“科学教”的神是谁呢？———人工智能！

也就是说，当人工智能的奇点到来之后，我们将为自己造出一个新神。人类社会的主体性将从现在我们的自身（理性），彻底转移到人工智能身上！我们将活在人工智能的阴影之下，因为它们真的是全知全能地控制我们的神。
