作者：文秋   审稿：DongDong    文宣：DongDong

大概5-6年前，义乌几乎无人不晓此案件。

一位广州商人在非洲某国做小商品生意，每年会从义乌进口大量的集装箱货柜。偶尔在货柜里会偷偷的装一些性药，药品在每个国家管控都很严，需要各种临床数据及认证才可以进口，非洲也不例外。众所周知，这些东西在国内的土化学家手里，根本不是问题。他进口的性药工厂设在福建，该工厂同时向世界各国出口产品。整柜整柜的输出欧洲，具体欧方如何清关不明。这是背景资料。

有天，此商人从非洲回国，在上海机场被公安带走，理由是涉嫌跨国走私违禁药品。稍后，被羁押于福建某看守所。家人得知情况，火速回国找关系。此后，将他转押在某宾馆。最终，家人辗转国内，找到福建省一别级的关系。但被告知，此命令由中央直达，任何人无权过问！事实真相究竟如何呢？

原来是，福建工厂出口到欧洲某国的货柜被所在国海关查获，涉及性药总量居然高达几十条货柜！于是，欧方直接跟中方申请协助调查。因此，厂长被捕，随之招供说，所有货物全部属于非洲经商的广州人，出口到欧洲的货物也是属于他。

广州商人被捕后，招供在该厂购买的几批货，分别装在哪几批货柜里，并提供了各种收据作为证明。他根本不认识什么欧洲的商人，也不知道工厂出口给了谁，在欧洲也没有任何生意，银行账户也没有长期付款给此工厂的流水，所以怎么也没办法对上口供。

很快广州商人又被转回看守所里关押，来来回回的提审，周而复始，最终也无法证明出口到欧洲的那批货是他的。就他本人购买的那些性药，按照中国的法律顶格是罚款而已，但由于最高权力机构直接干预，即便无罪，看守所也不敢放他。

非法羁押了他18个月后，终被定罪。居然是，妨碍司法公正！获刑一年零八个月，生生的将其从看守所转到监狱又羁押两个月才释放！
