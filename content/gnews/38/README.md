**西班牙2021年1月13日电/西喜社——**外交部长王毅周三承诺，中国将通过分享经验、提供疫苗和支持这个东南亚国家开始在国内生产疫苗来帮助印尼对抗中共病毒。

“尽管疫苗需求急剧增加，但我们仍然克服困难，毫不犹豫地响应印尼的需求，向印尼朋友提供疫苗，”王毅在雅加达会见外交部长雷特诺·马苏迪后告诉记者。

印尼已收到300万剂中国产科兴疫苗和1500万剂疫苗的原料，这些疫苗将在印尼生产。

总统佐科·维多多周三在政府批准科兴生物的疫苗紧急使用后接受了第一针疫苗，并开始努力为世界第四人口大国的数百万人接种。

印尼已录得超过84.6万例中共病毒病例，为东南亚之最，死亡人数超过2.46万人。

印尼还与其他疫苗公司签订了协议，包括诺瓦克斯和阿斯利康，不过他们的产品都还没有到达印尼。

马苏迪表示，印尼正在寻求医药行业的自力更生。

“关于这个计划，印尼对与中国和其他国家的合作持开放态度，”他说。
