2020年总统大选出现各种选票造假奇葩：点票机软件作假、提前打印填好的選票、销毁川普選票、选民人数超出100%……

据布赖特巴特新闻网（Breitbart）11月5日的一篇文章报道了“公共利益法律基金会”(PILF) 本周四提起的诉讼称， 宾夕法尼亚州的选民名单上至少有2.1万名是已故死者。宾夕法尼亚州未能根据联邦和州法律及时维护和更新他们的选民登记记录，以应对2020年总统大选。

诉讼称，”截至2020年10月7日，至少有9212名登记人已经死亡至少5年，至少有1990名登记人已经死亡至少10年，至少有197名登记人已经死亡至少20年”。”本案是关于那些已故的人不能收到选票，这个案子并不复杂”。PILF主席J. Christian Adams在一份声明中说，他们一直在向宾夕法尼亚州官员提供 已故者登记的具体数据，以确保在选举日紧张的结果出来之前，做好数据处理。民主党推銷郵寄选票，多年前积累的選民名單数据錯誤，将来会以不必要的收件人或選票未处理的形式显现出来。

5月，宾夕法尼亚州选举官員承認，有向已登記的選民重复寄发的選票的情况。同样，司法观察组织（Judicial Watch）也起诉了宾夕法尼亚州，指称该州选民名册上有80万非活跃选民。

该案在美国宾夕法尼亚州中区地方法院立案。案件编号为1:20-cv-01905。
