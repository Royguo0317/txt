和打了疫苗或者感染了病毒的经常在一起吃饭是否会影响到自己的健康有待进一步研究。

ADE中文意思是抗体依赖病毒和免疫增强效应。

如果感染国中共病毒，体内会有抗体发生ADE效应。第一波感染的人也许是隐性的，第二波感染会有更严重的变异病毒。

对于无症状感染者，我们人体没有针对病毒的天然抗体。只要我们的免疫系统是正常的，我们有天然的防御机制。这种防御机制叫做先天性的防御系统，人一生下就有这个机制。但是专门针对新冠病毒的抗体我们是没有的。
