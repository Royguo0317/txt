林伍德是美国顶尖的维权律师，这位受人尊敬的律师接过许多社会影响巨大的案子，曾为17岁学生尼克-桑德曼（Nick Sandmann)诉讼华盛顿邮报诽谤，最终获得高额庭外和解。伍徳郑重指出是11月3日的选举是非法的。

伍德现在聚焦佐治亚州的民选代表，包括州务卿和州长布莱恩-坎普，因为他们都否认佐治亚州系统性舞弊重新计票会导致错误结果。”佐治亚州州务卿是个骗子，”伍德说。”我不会轻易提出这类指控，我们的州务卿还有我们的州长布莱恩-坎普都很脏，他们被中共的钱收买和通腐蚀了。”

伍德还提出需要召开佐治亚州立法机构的特别会议，这样可以选出选举人投票给川普。他表明其实川普已经在佐治亚州赢得了压倒性的胜利，跟全国其它各地一样。

随后，伍德又令人震惊地宣称，佐治亚州多米尼公司（Dominion Voting Systems）的电脑投票系统含有被操纵的软件，有人在背后控制投票，这家公司抹去了川普总统的压倒性胜利，给乔`拜登制造了一场虚假的胜利。
