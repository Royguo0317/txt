## 脸书“事实核查”那些从科学角度讨论冠状病毒起源于中共国实验室的言论

[!\[\]()!\[\](https://spark.adobe.com/page/wxJ26i2tmURtW/images/66815241-7433-4988-a941-81085a0c7b10.png?asset_id=e68dfa56-e13b-4bb1-9371-a4cffc7b92d0&amp;img_etag=%226360e7c2881d2257a13da9bd8cb1d152%22&amp;size=2560)](https://spark.adobe.com/page/wxJ26i2tmURtW/images/66815241-7433-4988-a941-81085a0c7b10.png?asset_id=e68dfa56-e13b-4bb1-9371-a4cffc7b92d0&amp;img_etag=%226360e7c2881d2257a13da9bd8cb1d152%22&amp;size=1024)盖蒂图片社

脸书正在以核实事实的名义检查那些支持冠状病毒起源于中共国实验室论断的帖子。这一论断越来越引发科学界的关注，特别是上周科学记者尼古拉斯·韦德（Nicholas Wade）在一家名为《Medium》的博客平台上发表了一篇相关文章之后。韦德在文章里说明了他为什么认为实验室泄漏理论是最有可能的解释。

脸书警告用户，那些例证冠状病毒起源于中共国实验室的文章是“缺少上下文”的，并将这些文章链接到一则2020年事实核查结果，称没有任何证据表明该病毒不是起源于自然。 韦德的文章同时发表到脸书上，并被链接到脸书于2020年4月所做的一则事实核查结果。 而这个所谓的事实核查结果充斥着各类语法错误，仅引用了两篇学术文章和五位科学家在推特上反对病毒非自然起源说的帖子。 记者联系到一位发言人请其置评，他向《华盛顿自由灯塔》介绍了脸书的新冠肺炎政策。这些政策禁止声称新冠肺炎是制造或生物工程的内容。。

这是脸书一直激进地进行言论管制的最新例子。当川普政府的高级官员和共和党议员对位于武汉的病毒实验室表示担忧时，实验室泄漏的假设就成为了一个政治热点。 尽管主流媒体将这一假设说成是阴谋论，但随着对病毒自然起源说始作俑者的追查，实验室泄漏说越来越受到关注。

上周，韦德在《Medium》上发表的文章里概述了实验室泄漏假说的间接证据，从而引起了广泛关注。然而，韦德的这篇文章在星期二下午暂时从该平台上消失，尽管截至本文发稿时已恢复，但该博客平台未回复进一步置评请求。《Medium》也认为有关新冠病毒被实验室人工修改过的言论是“阴谋论”，应明确禁止。他们还禁止将新冠病毒成为“影响深远的阴谋或掩盖”。

脸书承认他们会对其认定的有问题的内容进行访问限制，包括附上“事实错误”或“部分事实错误”的标签，以此限制其传播范围。脸书没有回应是否会人工对加上“缺少上下文”标签的内容进行访问限制。一个对已发布该文章的账户的调查结果显示，与同一时间段内发布的其他文章相比，被加有标签的文章受到的关注度明显降低。

在最近对前总统唐纳德·川普禁止使用该平台的裁决中，脸书的监督委员会认可了对言论进行降级评级的做法，以避免不符合其标准的“放大言论”。

科学家们和各个情报机构仍在对病毒的起源进行调查。 到目前为止，还没有人能有力反驳病毒的实验室起源理论，但是像脸书和《Medium》这样的公司却企图限制与之有关的自由讨论。 脸书先前也限制了《纽约邮报》关于亨特·拜登（Hunter Biden）的报道，错误地将其标为“不实信息”，而限制了其传播范围。
