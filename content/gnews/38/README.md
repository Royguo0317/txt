**西班牙2021年3月5日电/西喜社——**一则报告指出最近陷入困境的美国纽约州州长安德鲁·古莫被指控掩盖在纽约因中共病毒而死亡的老年人的人数。安德鲁·古莫最近同样被两名前工作人员指控性骚扰。以前的研究者一般认为安德鲁·古莫篡改数据是在他开始起草个人“抗疫英雄传记”的时候，然而最新报告表明安德鲁·古莫篡改数据的时间远远早于他写书的时间。

根据纽约时报报道：
安德鲁·古莫对卫生局提供疗养院具体死亡人数数据十分震惊。数据表明去年6月份超过9000人死在疗养院于新冠病毒。安德鲁·古莫和他的核心团队希望这个数据永远不要被公开。他们重新篡改了这份报告而后将报告发表。

这件事就发生在安德鲁·古莫刚刚开始为他自己书写那份“抗疫英雄传记”的时候。6月份超过9000人在疗养院死亡的事件还仅仅是冰山一角，因为最新的报告指出在此之前还有类似隐瞒疗养院死亡的事例。

根据华尔街邮报的报道:
去年2月份安德鲁·古莫当局故意隐瞒掩盖中共病毒的严重性，以免川普总统从此获得政治上的优势。
联邦调查员从8月开始试图调查各州疗养院的真实死亡人数。调查发现安德鲁·古莫的确曾试图掩盖疗养院死亡数据。安德鲁·古莫当局现已面临司法部门和FBI 的调查。
