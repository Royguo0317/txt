仅在2019年的一年时间里，中共海军建造的舰艇下水总吨位就达到24万吨，等于完成了24艘大型护卫舰或3艘大型航母的建造。海军传统强国日本的总吨位(排名世界第四)不过是50万吨，中共海军一年新增吨位几乎是日本海军总吨位的一半——在过去二十年，中共海军的规模增加了3倍，而且增长的速度有增无减。

中共国20年前加入世贸协定后不仅经济规模大幅增长，造船工业产能也迅速膨胀。美国次贷危机的爆发是美国为首的西方经济进入衰退的显著标志，也是国际商品购买力下降的转折点。

高度依赖出口拉动经济，并以欧美为主要出口对象的制造业产能出现了严重的过剩，于是中共加快海军舰艇的制造，除了消化当前国营造船业的过剩产能，主要是为了攻打台湾、突破第一岛链进行海军扩张——其经济困局及需要转移国内矛盾、输出落后产能、对外军事扩张的野心和二战之前的法西斯德国非常相似。

据巜国家利益》3月10日报道，中共正在加速建造航母、核潜艇、驱逐舰、两栖舰和快速攻击舰等几乎所有现代海军军舰，前期目标是取得印太海上主导权并统治亚洲，然后进一步取代美国充当世界霸主——这个消息来源自美国海军、海军陆战队和海岸警卫队联署的一份名为《海上优势》的新战略文件。

文件指出，中共海军规模迅速超过美国本身就说明其扩张野心由来已久，并已经咄咄逼人地实施针对美国的行动——通过遍布全球的一带一路，把庞大的海军部署到极地、印太及大西洋的各个角落。中共在非洲正在建立由其控制的金融、商业系统，在吉布提建设军港，试图控制海洋运输咽喉，破坏航行自由及海洋公约，达到最终称霸世界的目的。

从最近英德法甚至西班牙的军舰都集结西太平洋，以及号称亚洲北约的美日印澳共同防卫框架快速推进(继外长会后又马上展开四国首脑会谈)等态势来看，中共称霸世界的险恶野心已经引起西方的高度戒备——最后担当抗共主导力量的无疑还是海洋系国家。

回溯一百多年以来的历史，发现几次全球性的大型战争包括冷战，都是由具备极权专制特色的大陆国家发起挑战，而最终获胜的全部是海洋系国家:从强大的拿破仑在滑铁卢被英军打败，到一战二战的结局，再到冷战搞跨苏联，都是海洋系国家最终战胜大陆国家，这一次海洋系国家和中共的对抗将毫无例外。

当今的海洋系国家主要以五眼联盟(美英加澳新)为主，他们以契约文化和基督教信仰作为联结纽带，因而在危机关头最终都团结在一起。反观中共，就是凶残的皇权专制思想和虚伪邪恶的共产主义杂交而成的恶魔，除了愚昧和凶恶，再无任何特征。

4年以来，由郭文贵先生引领的爆料革命，已经把这个伪善恶魔的画皮全数揭下，把它造毒放毒的罪行公诸于众，中共将不可避免地走向灭亡。
