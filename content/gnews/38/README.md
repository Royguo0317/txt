美东时间2021年4月7日，联合新闻国际（UPI）、国防视觉信息分配服务（DVIDS）和多伦多之星（TheStar）等几家媒体均发布一则讯息：美国海军与马来西亚空军举行南海双边演习。中共媒体则对此保持缄默。

根据美国国防部视觉信息分配服务网站报道，自4月6日（星期二）开始，马来西亚皇家空军（RMAF）与美国海军的西奥多·罗斯福航空母舰打击群一起在南中国海进行了联合军演，这是2021年马来西亚和美国之间的第一次双边演习。演习集中于联合空中作战和单位整合。

对于听惯了“美XXX联合军演”的战友来说，“美马军演”似乎无足轻重，但是其传递出的信号却意义重大。

首先，马来西亚皇家空军的装备组成以俄系战机为主，本次与美国的联合军演，目的不言自明——统一作战体系，为即将到来的战争扫平障碍。继美日联合军演，美国“手把手”指导日本之后，美国再次充当“教练员”角色。

其次，虽然马来西亚的军事力量在世界排名靠后，但是联合军演传递的是一个政治信号。郭文贵先生曾爆料，马来西亚是被中共深度“蓝金黄”的国家，但是，在马外长刚刚“拜见大哥”之后，转头就投入美国怀抱，把“大哥”的脸打得“啪啪”响。可见，中共的邪恶手段并不能阻止正义力量的集结。
![]()![](https://gnews.org/wp-content/uploads/2021/04/联合军演2.jpg)图片来源：百度百科
最后，结合近期美国在印太地区的一系列行动，我们可以看出，以美国为首的正义力量对中共的海上包围圈已经完全形成，彻底掐死了中共的海上进出“门户”。加上中共西部、北部的陆上封锁，一个针对中共的闭合封锁圈已悄然形成。

在东部（西太平洋）地区，日本、韩国一直是美国的传统盟友。近期，日、韩与美国的几次联合军演、美日韩2+2会谈、日本首相即将访美，这些都旗帜鲜明地表明了日本和韩国的立场；中共的“眼中钉”台湾毋庸赘述——中共必须转移内部政治、经济矛盾，双方最终必有一战。由此可见，第一岛链固若金汤，牢牢掐住了中共的脖子。

在南部，菲律宾、越南与中共在海疆问题上争端不断。刚刚因为海上武装民兵船事件，在美国的给力支持下，菲律宾一改之前懦弱形象，强硬回呛中共；扼守马六甲黄金水道的马来西亚，完全切断了中共由南中国海进入印度洋的路径。

在西南部，与中共在边境发生冲突，并有多人伤亡的印度，已经摩拳擦掌急于报一箭之仇。刚刚结束的法美日印澳5国印度洋联合军演，让中共妄图通过缅甸、孟加拉和巴基斯坦进入印度洋的美梦彻底破灭。

至此，联军已经对中共完成海上全面包围，再看陆上对中共的封锁。

在西部，因其在新疆实施种族灭绝，作为与维族人“同根同源”的几个西亚“斯坦”，肯定不会站队中共一方，加之因“一带一路”吃尽苦头，一旦世界围剿中共，他们必将加入。

最后再看中共的“俄爹”和蒙古国。老牌政客“普京大帝”也不过是惦记着中共兜里最后的几两银子，与中共虚与委蛇，等把中共榨干之后，就会弃之如敝履，新仇（病毒）旧恨（苏共解体）一起算，背后捅刀子；而处于“夹缝”中的蒙古国，与俄罗斯的关系就像两国人民形容的一样，是小牛和大奶牛的关系，加上之前中共在内蒙古实行的“文化灭绝”，就曾引起世界各地蒙古族人抗议示威，所以，蒙古国必将紧随俄罗斯。

鉴于中共制毒、投毒，对全世界犯下累累罪行，世界正义力量正在集结，据郭文贵先生透露，截止发稿为止，已经超过100个国家形成灭共联盟。中共百年运作，从当年对抗八国联军，发展到如今被百国联军报复、群殴，如此看来中共确如它自己吹嘘的“强大”了！
