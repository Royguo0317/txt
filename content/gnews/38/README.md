《路德时评》2021年4月24日晚间节目主要内容：

一、习神前往三亚交接三艘舰艇

环球网最新报道[\[1\]](https://news.sina.com.cn/c/2021-04-24/doc-ikmxzfmk8720670.shtml)：23日，海军三型主战舰艇——长征18号艇、大连舰、海南舰在海南三亚某军港集中交接入列习近平出席交接入列活动并登上舰艇视察。中共军事专家评论此次集中入列创下三个“首次”：国家最高统帅首次同时为三艘大型主战舰艇入列授旗；海军首次同一天一次性入列三艘主战舰艇；一支舰队同天接收三艘水面、水下大型主战装备也是首次。《环球时报》记者了解到，长征18号艇是最新型的战略核潜艇，海南舰是首艘075型两栖攻击舰，大连舰是055大型驱逐舰。
![]()![](https://gnews.org/wp-content/uploads/2021/04/三艘主战舰.jpg)[三艘主战舰](https://nimg.ws.126.net/?url=http%3A%2F%2Fcms-bucket.ws.126.net%2F2021%2F0424%2F3d2962afp00qs2kkf00asc000u000g2c.png&amp;thumbnail=660x2147483647&amp;quality=80&amp;type=jpg)
路德社评论：

1、这次是交接仪式，说明这三艘舰艇经过试验试航确认正常后交付中共海军，文中说：“这也表明海军官兵已经完成了对新式舰艇装备的学习、培训、试验等环节，已初步具备操作使用能力。”中共国海军只是初步掌握操作技能就敢直接进入列装，实在让人感到无语。日本即使在二战时期，他们生产的大和号战列舰进入列装前都需要试航一年多，如此大的舰艇只试航几个月就交付，其质量堪忧。

2、这就是中共特色，和当年的共产主义苏联一模一样，此次交付仪式只是中共为海军节准备的一项献礼工程，不在乎是否已经完工，所有共产主义国家都喜欢干这种假大空的事，好大喜功，中共军队的仪式性远超过中共实战能力。

3、文中写道：“这三艘舰艇都是大国重器，同一天部署南海的意义不同凡响，此次入列凸显了南海的重要性，也表明国家经略南海的决心和意志。”南海的国际战略意义毋庸置疑，关键是这三艘舰艇对部署南海毫无意义，南海与中共国大陆相去甚远，中共国战斗机作战半径有限，此时最需要的是在南海部署航母打击群来缩短战斗机作战半径，而不是靠这三艘舰艇。

4、“长征18号艇入列专家：解放军二次核反击能力大大增强”这恰恰反映出中共的核反击能力非常薄弱，就是因为基数小，所以新增一艘核潜艇和十多枚导弹就能大大提升中共国的核反击能力。

5、“055型大型驱逐舰的入列，不但标志着中国海军驱逐舰实现由第三代向第四代的跨越，也意味着中国现役装备主力战舰已经处于世界先进行列。”路德社在此前节目中爆料中共产055型大型驱逐舰只是吨位和体积大于驱逐舰，能多携带弹药，但是吨位又小于巡洋舰，这就是中共宣传的世界先进行列。

6、中共国选择在4月23日举行交付仪式，这并非重大节日，与中共国一贯的作风相违背，其中必有蹊跷。中共媒体报道王岐山在20日前往海南参加博鳌论坛，三天论坛结束后，习神就亲自前往南海参加舰队交接仪式，辽宁号航母因遭受美国电子战攻击而瘫痪，习神也没有亲自飞往海南，从这篇漏洞百出的宣传稿和仪式安排就能看出习前往海南完全是临时安排，可见习神极不放心王岐山。

7、中共国海陆空三军中要数陆军人数、装备、资产和拨款最高，此前中共国海军在军队中毫无地位，中共国空军地位也比海军高，但是远不及陆军地位高。习神安排空军出身的许其亮担任军委副主席，他在陆军中的影响力非常有限，陆军将领肯定不服。习神上台后非常重视空军发展，一方面他想占领南海，许其亮又是他可以信赖的将领，另一方面他认为空军就可以防止党内敌对势力出逃。

8、2019年“423海军建军节”是在青岛举行，2020年没有庆祝该节日，今年只是72周年（非整数），更何况南海舰队总部在广东湛江，并不在海南三亚，由此可以判断习已经意识到南海舰队对他皇位极其重要。他此次前往南海舰队就是想将南海舰队将领换成自己的马仔，南海舰队在海南拥有洞库，党内高层若想叛逃，可以直接从山洞中乘坐潜艇逃跑，躲过空军的监控。

9、中共094A型战略核潜艇直达到前苏联上世纪70年代的战斗水平，094A型战略核潜艇脊背上有一个类似乌龟壳的突起，这是用来存放导弹的弹药库。因为中共国技术有限，潜艇外壳无法加大，导弹又无法做小，所以就只能在潜艇脊背上增加一个弹药库，这是前苏联上世纪70年代的潜艇设计。094A型战略核潜艇的噪音水平也只相当于上世纪七十年代前苏联的水平，核潜艇以核为燃料，所以在行使过程中始终不能关闭发动机，当进入近海地区时，降噪水平就至关重要，美国和俄罗斯在降噪技术远远领先中共国，所以常规动力潜艇在近海地区的优势就比核潜艇大。

10、075型两栖攻击舰使用的是活塞柴油机，不是用燃气轮机，而辽宁舰使用的是烧重油的蒸汽轮机，柴油机在西方的商船或者两栖运输舰比较多见，这是法国的技术，航速只有20节左右，快不起来，中共舰艇的动力就和美国相差一大截。由于中共没有垂直起降的战斗机，所以075型两栖攻击舰只能搭载直升飞机，这使得075型两栖攻击舰的战斗力大打折扣。由于中共基础工业和基础研究水平非常落后，军舰又是一个大型体系化工程，这就导致中共舰队有无数短板。
![]()![](https://gnews.org/wp-content/uploads/2021/04/075型两栖攻击舰.jpg)[中共075型两栖攻击舰](https://5b0988e595225.cdn.sohucs.com/q_70,c_zoom,w_640/images/20180710/86598b3c24484a64a85670c44514cc46.jpeg)


11、印尼潜艇出现故障后下沉到海下800米处，过大的压力毁坏了整艘潜艇。潜艇之所以下沉到800米，就是因为潜艇控制系统出现问题才导致此次事故。墨博士在上期节目中说道，通过给弱电系统加屏蔽就可以防止强电系统干扰，这是军工设备的基本需求，印尼潜艇肯定也具备这项功能，甚至还有一套备用的机械操作系统来应对电子操作系统失灵。中共军舰肯定也具备上述基本实力，但是无论中共如何提升装备，美国依然有办法黑掉中共军舰的操作系统。

12、路德社之所以要揭露中共军舰的真实实力，就是希望中共高层认清现实，别以为拥有这些装备就自我感觉良好，不论中共配备电子操作系统还是机械操作系统，只要用电，美国就有能力破坏中共舰艇的电力系统。

13、小布什政府对伊拉克萨达姆政府发起军事打击时，美国先用石墨炸弹摧毁伊拉克的供电设施，然后再用电子脉冲炸弹摧毁当地所有通讯设备，这还是二十年前的事，美军电子战水平在这二十年肯定已经取得巨大飞跃。中共国军事设备只要用电就需要使用芯片，只要使用电载荷，就有遭受电磁干扰的可能，美军未来发动战争的战场铺设是以强电磁环境为基础，中共还不具备应战能力，所以中共开战必输。

AC-130J“死亡骑士”，又称“空中炮艇”，美军特战部队专用空中支援，配备105MM主炮和其他各种枪炮炸弹。火力之强令人发指，AC-130J“死亡骑士”已经飞往南海。

路德社评论：

1、AC-130J“死亡骑士”是以C-130大力神运输机为平台的空中炮艇，这已经是第五代战机，最早在越战中使用，当时为支援地面特种部队开发的空中炮艇。C-130大力神运输机配备四个涡轮螺旋桨发动机，可以在未准备跑道上起降[\[2\]](https://zh.wikipedia.org/wiki/C-130%E9%81%8B%E8%BC%B8%E6%A9%9F)。美军在这架大力神运输上加装了各式武器，包括105MM加农炮，还有各种规格的加农炮、导弹、机枪和地狱火导弹，造价不菲。

2、AC-130J“死亡骑士”主要是配合地面部队定点清除地方军事目标，可以用大口径加农炮对地面进行大范围攻击，自身又具备反红外和抗干扰反击能力，也是第一款加装激光武器的特战部队战机，这架飞机从来都是和特战部队配合使用，它此次飞往冲绳和南海适应环境，就是为应对中共对台湾发起登岛攻击。

3、AC-130J“死亡骑士”是战时使用的强打击武器，由此可以看出美国特战部队已经进入角色，将在南海问题和台湾问题上发挥重要作用，这架飞机从冲绳飞往南海的过程中还需要进行几次空中加油，美国F-15战斗机从日本北部三泽基地飞往南海要进行四次空中加油，这表示美军已经做好战机空中加油的所有准备。

4、以下是美国军事新闻发布的AC-130空中炮艇战斗实力的视频，AC-130空中炮艇对地打击实力让人震惊，此次飞往印太地区就是提前踩点，并作实战演练，为将来军事打击中共做全面准备。



5、美国海、陆、空和太空军正在为灭共做全面准备，中共现在还在搞舰艇交付仪式，真让人觉得荒滩，在绝对实力面前，中共使用任何忽悠、技巧和战术都不足挂齿。
