## 【香港要闻】港版『一带一路』『明日大屿』大白象工程将掏空香港（二）
` GM42` [轉載自GNews](https://gnews.org/zh-hans/577571/)

整撰：文粤、文燕

审核：MY
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/21014545/image001-40.png)「明日大屿」工程
最近几天，香港新闻媒体、金融界不乏有人在鼓吹市民把强积金投资「明日大屿」计划。投资达人庞宝林在他的【基金人语】专栏撰文，提倡把港人强积金（MPF）投资「明日大屿」。

庞声称自己一直认为若香港退休规划能更妥善处理，相信回报稳定及良好，也协助纾缓未来香港面对人口老化等问题，随着这些问题日趋严重，香港在2000年成立强积金已属起步稍迟，倘若港人能把香港数项政府投资全部注入强积金投资里，或许回报更理想和息率更佳，记得当年领展IPO价只是10元，现已上升7倍多，每年派息4%以上。

现时各界纷纷谈论「明日大屿」发展计划，故香港市民必须尽量把握机会，最近一众有识之士提倡就「明日大屿」众筹1.1万亿元，此意念及想法不俗，希望港人能透过此计划多方面受惠。

1. 过去及未来均从房地产投资获得理想的回报，若能给予所有感兴趣的港人以资产增值形式投资，并成立「明日大屿」投资公司，管理及直接投资于旗下地产项目，将来更可安排上市，中长线复息回报应不俗，风险也较低。须知大部分港人一直「钟情」砖头，即使属旧经济的房地产公司股价升幅或许比不上新经济股，却可能获得不俗的派息率。
2. 此投资公司亦可发债，集资购买土地及发展所需约万亿港元，并直接出售予港 人，息率约5%，较iBond及银色债券利息更吸引，认购次序应以MPF投资者为首，继而港人投资者，再其次为国内或外国机构投资者。
3. 此项目对日后香港发展非常重要，特区政府宜考虑利用「同股不同权」形式控股，并主导及统筹土地规划等事宜，再与私募基金、国际投资基金等合共占25%投资公司股权，余下75%先行让MPF账户持有人先行投资、接着为港人；真的未能全数认购，才分配予机构及海外等投资者，预计发债额达1万亿，应可满足大部分MPF投资者对债券之需要。
4. 预料上市集资应可为MPF投资者、港人或其他投资者提供退出（套现）机制，却可能必须考虑是否需要限制政府持有此投资公司的最低控股权比例、港人在这75%持有者中之最低持股比例等。

![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/21014450/43562254_1961649327230873_3814685107860537344_o.jpg)图片：公民党 谭文豪
战友点评：

笔者拜读完庞先生的文章后有以下疑问：

1. 动用港人的强积金入股，「同股不同权」形式控股，政府占25%作主导；港人75%，但香港人没有投票权，由政府主导。那么万一公司倒闭，投资者向谁控告？就算告上法庭，估计法庭一定偏帮政府。万一出现状况，投资者将血本无归，到时能向谁追讨？万一到时是烂尾楼，怎办？谁也无法预料未来2025到2032这八年将发生什么事情。
2. 「明日大屿」工程原先造价是6,240亿港元，但最新造价时1.1万亿元，因此港府需要众筹1.1万亿元，说白了工程造价最少是1.1万亿元。
3. 如果「明日大屿」工程最少造价是1.1万亿元，那就相当于用了政府库房一半的钱。于是林郑政府开始打港人强积金主意。用这种主动强行替港人把强积金投进「明日大屿」计划，那么港人的强积金也就送羊入虎口。
4. 「明日大屿」70%起公屋能有盈利吗？我们从没听过政府建公屋能盈利，根据房屋委员会（负责港人伸请公屋事务）的报告显示每年都在亏损。 「明日大屿」工程到时真的会建70%公屋吗？要是真的建70%公屋，那一定亏本。
5. 按照计划，2025年开始填海，2032年入伙。既然5年后才开始填海，中间这5年的债息从哪里来？
6. 港府为何不动用储备？是否香港的储备已经被挪用得差不多了。除了不能动用的美金，香港还有2万多亿储备，为什么港府不动用这笔储备？而要用市民的强积金，还怂恿市民投资买债券。


基于以上几点，笔者有理据相信林郑早前上北京开会的真正原因：DQ 4位立法议员，为了到时在立法会提出强积金方案可以立刻通过做铺垫。

「明日大屿」融资计划既然退而求其次，在不能动用财政储备的情况下，一定会向港人强积金开刀，因为估计香港的储备已经被挪用得所剩不多，现在需要动用香港人的强积金，强行把港人的强积金投入「明日大屿」计划，还放风怂恿市民投资买债券，令港人资金被套牢。

引用香港某网媒观点，『上市公司需要有盈利才能发放债息给投资者。要是公司没盈利，那么唯一的办法就是把投资者的钱变回利息给投资者。如真是这样，那就是庞氏骗局』。换句话说，「明日大屿」这大白象计划就是硬生生地把港人的强积金转到中国的建筑公司，而中国的建筑公司背后就是中共利益集团。

以上观点仅代表笔者本人
