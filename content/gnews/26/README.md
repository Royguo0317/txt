# **国人口说爱国 身体最诚实**

搜集\编撰：西西

**棉花事件持续发酵，多间国际时装品牌关注新疆强制劳动问题，并停用新疆棉，包括运动品牌 Nike 及 adidas，内地官媒抨击、网民声言抵制。不过消费行为很诚实，在内地网购平台「得物」，Nike 及 adidas 波鞋的销情仍然良好，平均每分钟都有人落单购买。**
![]()![](https://gnews.org/wp-content/uploads/2021/03/3273.jpg)
据【立场新闻】报道，H&M 和 Nike 等跨国品牌表明拒用「新疆棉」，除了被中国官方批评，民间亦声言抵制。据内地网上消息，浙江宁波有青年纷纷到「李宁牌」运动用品店买鞋；另有中超球队过往一直穿着Nike 赞助球衣，近日在微博发帖时疑似改图，「执走」品牌标志。
![]()![](https://gnews.org/wp-content/uploads/2021/03/3274.jpg)上海申花足球俱樂部微博（资料图片）
中共官媒发表评论，形容西方针对中国的行动「才刚刚开始」，料陆续有来，但是中方会团结反击，称中共国不可以任由西方强加「西式人权价值观」。

网军组织「帝吧」深夜在官方微博表示，有网友投稿指，很多大学生都在浙江宁波的李宁运动用品店排队买鞋，「隔壁的 AJ、H&M 都没人」。帖文形容，国产服装品牌都要迎来「机会」。大批内地网民都对此赞好，「这些年轻人越来越棒了」、「我今天也去线下买李宁了」、「中国后生有望」，有留言指「李宁品牌的鞋子也很好穿」、「李宁人超多，我买的也不少！」

《环时》评论认为，民众要有「自信和底气」，让外界理解到新疆棉花好，「你们扺制，我们内销，损失的是你们」，又指西方打压中国会是一场「持久战」 ，「这需要战士的英勇，也需要参谋的智慧和眼光。」

点评：

新疆棉花占了中国绵花产量 90% ，绝对关乎中共外汇来源，每年大概二千亿美金收入。根据自由亚洲所收集的中国数据，中国纺织业统计数据显示，中国的棉花和纱线等材料主要被用于三大领域：1）服装领域；2）家用纺织品领域；3）产业用纺织品领域。这三个纺织品领域总出口规模接近1900亿美金，出口占中国纺织业总产量超过一半。

新疆绵花占全国棉花出口是中共赚取外汇主要来源之一，一旦外资企业抵制的话，中国将会损失接近2千亿美金收入，这将重创中国高度依赖出口的纺织业，进一步恶化目前中国的失业问题。

民族感情就是面子，民族感情的另一面就是金钱。如果是公平竞争的，为何这些年来国内品牌占不了国内市场？而是美国、德国、瑞典等产品一直占据中国大部分市场。为何国产品牌迟迟不能崛起？需要用民族主义去抵制和赶走国际产品才得以崛起？在和平竞争情况下，为何国产李宁运动用品赢不了国际产品？

这证明了中共国产品的竞争优势欠缺、质量不行，需要用政治手段消灭对方，用民族主义的手段消灭国际产品，抵制国际产品，而不是靠品质，靠成功的市场营销占据市场。对于崇洋媚外的一族，口说爱国，身体最诚实，我们且看接下来有多少人拒用苹果手机而 换成小米手机。
