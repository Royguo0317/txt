## 海地瓦吉亲历的新疆集中营恐怖遭遇

***翻译/简评：草原三省战友之家 文莱***
![]()![](https://gnews.org/wp-content/uploads/2021/03/Equateurs2012_RescapeeDuGou.jpg)图片来源：The Siasat Daily
根据美国广播新闻公司（ABC News）3月7日报道，古尔巴哈尔-海地瓦吉（Gulbahar Haitiwaji）作为新疆省维吾尔族的一员，她在10年前就已经离开中共国并寄居法国，但她仍持有中共国护照。她在2016年11月被中共国传唤回国并要求签署有关她在石油公司工程师的退休文件，当她无法授权亲属处理此事后，她于月底前往中共国。她说，在接下来的三年里，她被禁止出国，也禁止见自己的丈夫，而一位逃离新疆的维吾尔族同胞则陪海地瓦吉的两个孩子在法国生活了三年。

海地瓦吉表示，她是被骗回中共国并被指控为恐怖分子，因此被判处7年监禁并被关押在新疆臭名昭著的“再教育营”中。现年54岁的海地瓦吉至今仍想知道自己究竟为什么会成为目标。她表示，在警方的审讯中，有警员给她看了一张她女儿在巴黎参与流亡维吾尔人组织抗议活动的照片，因此她被不断地问及女儿的活动、她自己去法国的理由、以及她被要求必须毁掉女儿和丈夫的户口。

海地瓦吉告诉美国广播新闻公司：“我们身边没有律师，只有中共的警察，他们呆在我们身边不说话。这持续了不到10分钟，他们就列出了我的‘罪行’并指控我为恐怖分子。”

海地瓦吉提供的第一手资料证实了人权组织认定的新疆100多万维吾尔人以及哈萨克族和其他穆斯林少数民族近年来正经历“永久洗脑”和“强制文化同化”的计划。现在，海地瓦吉获得了自由，并回到了法国的家中，她与法国记者罗岑-莫尔加特（Rozenn Morgat）一起写了一本关于她亲身经历的书，名为《中国古拉格的幸存者》（Rescapée du Goulag Chinois），该书的英文译本将于今年年底在美国出版。

海地瓦吉在书中描述了在石油资源丰富的克拉玛依市附近的白剑潭的一个“再教育营”中的遭遇，讲述了在没有窗户的教室里接受共产党的洗脑宣传，并要忍受经常性的虐待。她写道，所有人被教导要每天重复对习近平的美好祝愿，并学习美化版的中国历史，以证明自己接受了“再教育”，并不再是“恐怖分子“，这意味着维吾尔人需要在集中营中度过更长的时间。

海地瓦吉还讲述了集中营中类似于军事训练的体能锻炼，强度可令人筋疲力尽。她在书中写道：“有些人有时会昏迷，如果有犯人不回应看守的呼喊，仍然昏迷不醒，那么就会有警官毫不客气地一巴掌把人打起来。”

海地瓦吉说，为了能早日获释，她选择了当“勤奋的学生“和”模范犯人“。同时，她默默地转向了阿拉，并开始整天瞒着监控摄像头祈祷、做瑜伽。

她的口径与美国广播新闻公司此前采访的十多名维吾尔族人一致，但中共政府在其驻巴黎大使馆的一份声明中抨击她的指控是捏造的，就像他们对所有异见者一样。不过，中共当局确实承认他们曾逮捕并关押海地瓦吉。

虽然海地瓦吉所记录的情况与近年来国际媒体广泛报道的情况类似，但她的第一手证词罕见地详细描述了集中营中的生活。在这本书中，海地瓦吉描述了她被锁在床上20天中的心理变化，这种痛不欲生的过程让她觉得最后“什么都没有了“。而12月的一个晚上，在她两年多的拘留期即将结束时，她描述了自己被放进一辆卡车，但没有被告知要去哪里，她担心会被枪决，非常消极。她说：”那时（在卡车中），我已经死在里面了。“

海地瓦吉的女儿曾说服法国外交部出面要求中共当局释放母亲。谈判初始，外交会议和媒体经常提到海地瓦吉的案件。2019年3月，海地瓦吉表示在她又经历了6天的审讯后，她再次被迫供认参与了一个由维吾尔族恐怖分子在法国组织的非法活动。她最终被释放并飞回法国。

中共国驻法国大使馆发表声明说：“海地瓦吉长期从事分裂主义和恐怖活动，完全是分裂分子和恐怖分子并存，她于2016年自愿返回新疆以寻求‘正常的生活’。”

但是共产主义受害者基金会的中共研究高级研究员阿德里安·曾兹（Adrian Zenz）说，如今公开的关于新疆集中营的目击者描述有几十个、甚至上百个，关于这个国家内部秘密信息现在已经相当全面了。

他说：“大量的证据可以验证新疆集中营正在大规模地拘留维吾尔人。我们有卫星证据、有泄露的文件，比如中共的电缆信息。它们为我们提供了很多关于这些集中营如何运作的详细信息，证据表明中共当局特别针对男性和户主。”

而中共外交部最近几个月采取了越来越主动的方式来回击这些证人，试图混淆受害者的证词。中共外交部发言人汪文斌2月23日指责一名证人是不法分子，诽谤和抹黑中共的“演员”。中共方面对曾兹的“特殊待遇”也是显示出了中共愤怒的窘态，中共指责这位学者“滥用数据、歪曲材料、编造案件”，甚至希望“阿德里安·曾兹和其他公然诽谤者可以受到司法惩罚“。

法国外交部长说，法国“收到的证词和确凿的文件显示了中共对维吾尔人采取的无理做法和大规模的制度化监视以及镇压系统”。

**中共对新疆残暴历史与未来展望**

新疆西部地区拥有众多的民族，最突出的是维吾尔族，大约有1300万维吾尔族穆斯林生活在该省，他们在语言和文化上与汉族截然不同，维吾尔人的语言与土耳其语是近亲，不是普通话；而他们的主要宗教是伊斯兰教，不是无神论。

自邻国苏联垮台以来，一些维吾尔人要求更多的自治权以及更公平的待遇，但是他们与在该省涌入的汉族人摩擦不断，最终导致了叛乱和民族暴力事件的爆发。2009年7月，维吾尔族的愤怒点燃了该地区首府乌鲁木齐的骚乱，197人死亡（大部分是汉族人），1600多人在连续几天的反抗和袭击中受伤，随后中共当局掀起了逮捕浪潮。

在接下来的几年里，中共国遭到了一系列的恐怖袭击，包括在乌鲁木齐和北京天安门广场毛泽东画像前的自杀式汽车炸弹袭击，以及在昆明火车站发生的直接导致31人死亡的致命恐怖刀袭，但中共当局将这些袭击事件全部归咎于被“外国极端思想“灌输的维吾尔人。在2014年乌鲁木齐发生了一起袭击事件后不久，习近平就在访问乌鲁木齐期间发动了”反恐人民战争“。

专家称，新疆是中共当局的战略要地，对中共的“一带一路“项目至关重要。只不过中共对新疆的控制多年来不断升级，最终导致了2017年以来有关于镇压浪潮的报道铺天盖地，据报道，这一年出现了第一个”再教育营“。

研究共产主义国家犯罪的非营利组织 “共产主义受害者基金会 “的中共研究高级研究员阿德里安-赞兹曾在海地瓦吉被关押期间与她的丈夫接触，他估计有多达180万人经历了某种形式的法外拘禁。

澳大利亚战略研究所称，卫星图像显示新疆至少有385个这样规模的集中营，不过这些集中营中的惩罚严重程度上是有所不同的，有一些是“再教育中心“，被拘留者在那里接受文化洗脑，其他集中营则是强迫劳动。

中共官员一再坚称，这些“再教育中心“中的人们是自愿而来，并否认该地区侵犯人权的指控。更令人作呕的是，中共极力地吹嘘该运动取得了成功，因自2015年以来，国内没发生过一起恐怖袭击事件。

本周，赞兹撰写的报告披露出新疆的一些劳工被转移到“其他地区和其他省份是国家管理计划的一部分，以强行将他们连根拔起、同化他们并降低他们的人口密度“。报道称这种制度直接构成危害人类罪。而在集中营之外，有报道称，维吾尔人和其他少数民族现在生活在监视状态下，在对他们的生活进行更广泛地控制时，未经官方许可，他们不能进行文化活动。更有报告披露中中共为了控制人口，强迫维吾尔族妇女绝育。

川普在执政的最后一天曾表示，中共对其穆斯林少数民族的虐待构成了“种族灭绝“和”反人类罪“。所以美国已经禁止进口所有来自新疆的棉花和西红柿产品，因为担心它们是由奴隶劳工生产的。但海地瓦吉说，世界上许多国家因担心经济报复而迟迟不敢批评中共，包括以穆斯林族裔占多数的国家。

海地瓦吉说：“不幸的是，中共的经济实力在人权问题上占了上风。尽管国际社会对维吾尔人的关注越来越高，但国际社会仍然需要采取行动并制造压力以关闭这些集中营“。对于海地瓦吉来说，她现在一直担心还住在新疆的亲戚，因为”他们根本不知道这本书的存在，但是我还是要把真相说出来，因为如果我的佐证可以帮助关闭这些集中营的话，那么许多同胞将会获救“。

赞兹悲观地表示：“中共目前认定的方向已经很难被国际社会的压力所左右，这是种族灭绝的形式—为了磨灭一个族群的身份同化并强行消除其宗教，将父母与子女强行分离，在寄宿学校实行洗脑以消除语言和文化等。这是一种长期的分裂社区、分裂家庭的形式，并且中共对集中营的内部与外部的整体控制导致当地人民的生活也遭到了绝对监视。”
