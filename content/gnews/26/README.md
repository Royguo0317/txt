## “百年屈辱史” 已成為中共洗腦而採用的“選擇性創傷”，煽動民族主義的工具
`2020-07-05 08:53 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/254681/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/05062213/Picture-1-36.png)圖片來源：The Economist (製作*Claudio Munoz* ) 
新聞門戶網站OpIndia7月4日發表了一篇關於中共如何煽動民族主義和在世界舞台上扮演著一支好戰力量的政治評論文章。

文章指出，中共除了對本國領土進行極權主義統治外，它還回溯歷史事件，對當前他國領土提出主權要求，這些歷史事件有時可追溯到14世紀。最近，在與俄羅斯和日本發生隱秘爭端之後，中共還對俄羅斯的海參崴市（Vladivostok）提出了領土要求。

**何謂“百年屈辱史**”

中共對海參崴的主權要求清楚地提醒人們，“百年屈辱史”在塑造中共的外交政策及其在國際舞台上的行動中發揮著非常關鍵的作用。它同時也在中國人的民族印像中佔有非常重要的地位。這一時期是指從1939年第一次鴉片戰爭到1949年中共在國共內戰中取得勝利的一個世紀。其特點是中國在與西方國家、俄國和日本發生的交戰中吃了多次敗仗。據估計，這一時中國對其領土的控制縮減了三分之一，其數千年曆史的帝制也隨之崩潰。此外，這一時期的中國還存在內部叛亂、起義、內戰不斷。可以說，沒有人希望成為那個年代的中國人。海參崴是沙皇俄國在1860年通過《北京條約》吞併的。在此之前它一直由清政府統治。除了海參崴，當時的清政府還把九龍半島割讓給了英國。

中國問題專家大衛•斯科特（David Scott）說：“在那個一百年間，中國在國際體系裡步履蹣跚，處在一種獨特的尷尬階段。它一方面是全球人口最多的國家，佔當時世界人口的四分之一，另一方面它不得不把領土和主權獻給了多個國家，甚至包括比利時和葡萄牙這樣的歐洲小國。”

一位美國評論家指出，在中國，對於這段歷史，一般用’鴉片戰爭前’和’鴉片戰爭後’來非常寬泛的概括思考，旨在不想多提及那個世紀的屈辱和軟弱。 另外，還有人指出，中國的民族主義不僅僅頌揚中華文明的輝煌，它還不要人民忘記中國疲弱時的屈辱。CNA公司亞洲政策高級研究員艾莉森•考夫曼（Alison Kaufman）在她的一篇論文中指出，“百年屈辱史”所呈現的不僅僅是一個關於過去經歷的警示故事，更是中共國關於世界如何運作的信念來源。無論是明示的還是隱含的，那個時期形成的詞彙和問題還依然被用來解釋當前的國際關係動態。

**“百年屈辱史”已成為煽動民族主義的工具**

中共利用“百年屈辱史”擴大其統治中國民眾對其的支持力度。這一黑暗世紀的結束恰逢中共的內戰勝利，因此，中國人民傾向於從好的方面看待中共。如今的中共有著更大的野心，希望給全世界把脈。中共正力圖糾正它認為過去對它的不公正待遇，也力圖確保它永遠不會再遭受過去那種屈辱。

有人認為，“百年屈辱史”其實是一個歷史謎團，它已經成為中共對大眾進行洗腦而採用“選擇性創傷”。中共為了喚起民眾的民族主義情緒，鞏固對自己政權的支持，使自己的統治合法化而採取的宣傳手段。不管它究竟是什麼，都不能否認“百年屈辱史”在中共國民眾意識概念中佔據著主導地位，以至於至今甚至台灣的教科書都將其歷史劃分為鴉片戰爭前和鴉片戰爭後。
