2018年10月4日，美國副總統[邁克·彭斯](https://zh.wikipedia.org/wiki/%E8%BF%88%E5%85%8B%C2%B7%E5%BD%AD%E6%96%AF)在華盛頓智庫[哈德遜研究所](https://zh.wikipedia.org/wiki/%E5%93%88%E5%BE%B7%E9%81%9C%E7%A0%94%E7%A9%B6%E6%89%80)就美國政府的中國政策發表長篇演說，這標誌著中美之間的冷戰鐵幕已經拉下!

美國副總統彭斯：「從本屆政府的第一天，這就是唐納德·川普總統一直在做的：保護、捍衛，以及慶祝每個美國人的自由，無論種族、信仰以及膚色。」彭斯副總統抨擊中國試圖破壞美國的民主制度，並談及美中貿易爭端、中國在南中國海的軍事化、對台灣的打壓，還有中共對國內民眾的監控和壓制以及在美國和海外施加影響力等等課題。並反思美國對華政策，表示**「曾期待中國成為自由國家，希望中國尊重傳統的自由主義原則、尊重私人財產、個人自由和宗教自由，尊重人權。」「但是希望落空。」**「[鄧小平](https://zh.wikipedia.org/wiki/%E9%82%93%E5%B0%8F%E5%B9%B3)的[改革開放](https://zh.wikipedia.org/wiki/%E6%94%B9%E9%9D%A9%E5%BC%80%E6%94%BE)政策已經變得空洞。」「如今，**中國已經建立了無以倫比的監控國家，範圍越來越廣，越來越具侵入性，而且經常是在美國技術的幫助之下。**」「中國正在與美國的盟友和敵人打造自己的關係，與北京的任何和平或積極的意圖背道而馳。」「前幾屆政府忽視了中國的行動。在很多情況下，他們還助長了他們。**但是，這樣的日子結束了。」**

翌日，[中國外交部](https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%9B%BD%E5%A4%96%E4%BA%A4%E9%83%A8)通過發言人[華春瑩](https://zh.wikipedia.org/wiki/%E5%8D%8E%E6%98%A5%E8%8E%B9)以答記者問的方式，對美國副總統的發言予以回應，強調[一個中國原則](https://zh.wikipedia.org/wiki/%E4%B8%80%E5%80%8B%E4%B8%AD%E5%9C%8B_%28%E4%B8%AD%E8%8F%AF%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9C%8B%29)和中國在南海的主權，表示中國尊重人權，抗議美國干涉，並要求美方自我反思。

**中美兩國的地緣政治野心沖突正在醞釀壹場可能比最初的冷戰更危險、後果更嚴重的競爭。**

在過去的30多年間，美國和世界列強為了利益，對中共[侵犯人權](https://tw.aboluowang.com/tag/%E4%BE%B5%E7%8A%AF%E4%BA%BA%E6%AC%8A_1.html)裝聾作啞，在全球化的旗幟下，幫助中共發展壯大，不僅成長為最大的[世界工廠](https://tw.aboluowang.com/tag/%E4%B8%96%E7%95%8C%E5%B7%A5%E5%BB%A0_1.html)，成為世界第二大經濟體，而且全面滲透美歐等強國政治經濟文化商業各個領域，滲透和影響了聯合國和[世衛組織](https://tw.aboluowang.com/tag/%E4%B8%96%E8%A1%9B%E7%B5%84%E7%B9%94_1.html)等世界組織，對外擴張的野心也日益膨脹。

兩國關系的裂痕有可能已發展成為壹道鴻溝。幾乎沒有壹天不發生壹些針鋒相對的交鋒；抑或指責對方，抑或出臺壹些旨在使對方國家生活困難的動作，或鼓吹各自政治制度優越性的行動。

中共最終在無視[國際法](https://tw.aboluowang.com/tag/%E5%9C%8B%E9%9A%9B%E6%B3%95_1.html)和國際價值觀的情況下，掩蓋病毒真相，把魔鬼釋放出籠，成為今天威脅全球人生命和經濟安全的頭號敵人。

美國指責中國強迫維吾爾族婦女絕育；遊說歐洲取締中國安檢公司威視公司；對香港新國家安全法中被追究責任的中國官員實行簽證限制；對中國記者的工作簽證實行90天限制。

作為回應，中國外交部將美國對其維吾爾族政策的批評標榜為 “毫無根據”，同時直言不諱地告訴華盛頓，不要再插手香港事務。北京早些時候撤銷了三家美國主要報紙記者的記者證，並威脅要將美國公司列入被禁外國實體名單。

這種迅速陷入沖突的情況讓很多人感到意外。在本世紀的大部分時間裏，中美之間的競爭因需要在壹系列必須合作的全球經濟、金融和地緣政治問題上共同努力而得到緩和。但這些合作沖動幾乎完全消失了，而冠狀病毒大流行的責任問題上的指責更加劇了這種沖動，這暴露了雙方互不信任的深度。

北京認為華盛頓壹意孤行地遏制中國，以延長美國的衰落力量，同時剝奪中國在陽光下應有的地位。美國人越來越認為，北京正在威脅美國的安全利益，破壞美國的繁榮，幹涉美國的民主，並挑戰美國的價值觀。反華情緒將原本分裂和黨派的華盛頓團結起來。

**中共和西方精英同床共枕「特朗普比里根更艱難」**

**雙方的戰略野心沖突和政治制度的對立。**

[美國](https://zh.wikipedia.org/wiki/%E7%BE%8E%E5%9C%8B)媒體高管、政治人物、戰略家、前投資銀行家、Breitbart News的前任執行主席、「[爆料革命](https://zh.wikipedia.org/wiki/%E7%88%86%E6%96%99%E9%9D%A9%E5%91%BD)」支持者——班農在採訪期間拿出一本書——《超限戰》，裏面夾滿了筆記便條，「這是我讀得最多的一本書。」**「（美國製造業流失）正是中共發起的經濟戰略所致。」**《超限戰》是1999年出版的中共軍方兩名大校寫的書，書中談及多方位攻擊美國，包括經濟戰、政治戰、網絡戰、恐怖襲擊等。

白宮貿易顧問納瓦羅重申美國戰略背景與緣由，從中可見川普施政脈絡,他是抗中鷹派知名學者，擔任白宮國家貿易委員會主任。**他認為****,** **中國持續對美國所犯下的「**[**七宗罪**](https://opinion.udn.com/opinion/story/120611/3976989)**」：七大致命結構性犯罪。包括：國家組織網路駭客入侵、剽竊智慧產權、強制技術轉讓、國家補貼造成不公平競爭、惡意傾銷、貨幣操縱、大量非法輸入芬太尼（****Fentanyl****）。**正如納瓦羅所認為，美國對中國「脫鉤」已經是進行式。「無論如何，川普總統已經啟動從中國撤離供應鏈的程序。」

對美國總統唐納德·特朗普來說，與中國的較量是為了通過公平的貿易和科技競爭環境來糾正被認為的不公平，並鞏固美國作為全球頭號大國的地位。中國的習近平也希望糾正過去的不公正，並抓住他的時機，讓中國恢復其作為亞洲並最終成為世界主導國家的 “合法 “地位。對習近平來說，實現這壹雄心壯誌、擺脫中等收入陷阱的時間已經不多了，因為人口下降、生產力下降，以及全球對他強硬的外交政策的反擊越來越多。

從合作到戰略競爭的轉變，這引發了壹場關於世界是否處於新冷戰邊緣的激烈辯論。懷疑論者對此與冷戰有六個明顯的相似之處。**首先，美中競爭是世界上最強大的兩個國家之間的競爭，壹個是自由民主國家，另壹個是公然的共產主義國家。第二，這是壹場全系統的爭霸。第三，它既是價值觀的問題，也是權力的問題。第四，這將是壹場多世紀的全球崛起之爭。第五，世界可能出現第二次地緣政治分叉。第六，雙方都不希望發生全面的軍事對抗。**總之，這不是壹般的大國沖突

當然，也有重大的差異。中國已經取代俄羅斯成為主要威脅。美國和蘇聯之間的戰略競爭主要在政治和軍事領域進行；兩個競爭集團之間幾乎沒有貿易。但中美之間的主要競爭是經濟上的競爭，也就是說，貿易、投資、技術和戰略產業是今天競爭的核心。

在最高峰時，蘇聯的GDP只有美國的40%。但中國的已經達到了65%，而且還在快速增長。兩者之間，美國和中國的GDP占全球GDP的40%左右。如果這兩個巨人中的任何壹個打了個噴嚏，世界其他國家就會感冒——**就中國而言，是真的感冒，因為冠狀病毒的影響繼續給數百萬人的健康和繁榮造成破壞。**

雖然新的冷戰正在世界各地上演，但其地理重心是印度洋-太平洋，而不是歐洲，因為全球商業和貿易的中心已經從大西洋轉移到太平洋，反映了亞洲的崛起和歐洲的衰落。美國和中國都是太平洋大國，所以他們的競爭將在印太地區感受得最為強烈，尤其是在海上，他們的利益在這裏碰撞，有幾個潛在的軍事對抗的誘因。

北朝鮮、東海和南海是最有可能發生沖突的地區。**但臺灣和香港也是潛在的沖突舞臺，這不僅僅是因為它們的政治意義。臺灣是美國和中國的重要技術生產地。香港是中國通往世界的金融門戶，美元仍是國際貿易的主導貨幣。**

雖然冷戰低於大 “熱 “戰的門檻，但如果不小心管理，很容易導致冷戰。崛起的大國和在位大國之間的緊張關系往往是在軍事沖突或長期對抗和不穩定之前發生的。**如果不采取果斷措施，中美緊張局勢的加劇可能會使正在形成的冷戰更加惡化，預示著壹個戰略競爭加劇的時代將對國際貿易和世界秩序產生巨大的破壞。**

**歷史記錄表明，雖然熱戰並非不可避免，但卻非常可能。**然而，更有可能的是，中美之間的競爭在可控範圍內不斷發酵，但仍未達到大規模戰爭的門檻，但卻經常發生代理沖突，尤其是在網絡空間。雖然腐蝕性大於爆炸性，但這將迎來壹個漫長的大國競爭時期，可能會使70多年來貿易自由化的成果倒退，破壞全球供應鏈，使互聯網局域隔離化，並將世界劃分為兩個互不相容的政治體系。

**中美關系的核心問題是它們截然相反的政治制度和相關價值觀，再加上它們的特殊主義意識。**自2008-2009年金融危機以來，**中國領導人對民主國家的弱點的看法變得更加批判，並堅信他們自己的專制模式的優越性，這種模式將政治穩定和社會秩序置於個人權利和言論自由之上。**

當雙方都懷疑對手想把自己制度中的不良因素強加給對方或在國際上宣傳這些因素時，問題就變得更加嚴重。這些看法正在加劇美中緊張關系，使其更難解決。長期以來，中國領導人壹直對他們認為是對其內政的無端幹涉以及美國人對其行為和政治制度進行說教的傾向感到不滿。現在，特朗普政府痛斥中國幹涉美國國內政治、進行政治戰爭，並試圖向其他國家輸出其獨裁模式。

前總統奧巴馬習慣性地低估了美國的巨大余威，反常地助長了中國崛起是必然的神話。然而，特朗普卻搶占了心理制高點，霸占了推特等網路媒體，迫使中國防不勝防，並表明沒有任何其他國家能與他所掌握的經濟、金融、軍事等顛覆性力量相提並論。這種做法的弱點在於，美國的實力展示是和以懲罰為威脅與鄰國討價還價的政策聯系在壹起，這種政策把朋友和對手都達到了對立面，並造成了系統性的不穩定。

現在仍有時間來平抑不斷攀升的敵意曲線，扭轉沖突的勢頭。那些認為民主國家和專制國家永遠無法找到必要的調和的人，忽視了歷史的教訓。盡管存在分歧和幾次險情，但美國和蘇聯還是找到了合作的方法，並在其數十年的對抗中避免了壹場重大戰爭。

**中共是對世界的最大威脅，這不是美中之間的戰爭，而是善與惡，真與假，自由與奴役之間的戰爭。美國需要與盟國合作。不久的將來，地球將有壹個向上的變化。**
