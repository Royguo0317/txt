## 马克·扎克伯格是否可以为美国各州的政府竞选出资？

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/09101004/Picture7.png)图片来自：cnbc.com
网络杂志（The Federalist）12月7日刊登了华盛顿特区资本研究中心的主席斯科特·沃尔特(Scott Walter)的文章。文章关于亿万富翁私人是否可以投资美国大选的问题做出了质疑。

作者首先让大家想象一下，如果一个自由主义倾向的亿万富翁（例如查尔斯·科赫Charles Koch）将3.5亿美元捐给了一个由共和党成员经营的非营利组织，这个共和党成员曾在一个与卡尔·罗夫（Karl Rove）捆绑在一起的“黑钱”组织工作，他们在那里培训数字活动家和竞选工作人员。然后，再想象一下，这个非营利组织将数百万美元重新发给地方选举官员，以“帮助”他们进行2020年大选，购买投递箱，聘用临时员工，进行“选民教育”等等。最后，想像一下，在2020年，一个在总统选举中通常是红州的地区，以微弱的优势变成了捐赠者喜欢的蓝色，而接受 “帮助 “的县，原本绝大多数是帮助共和党赢得该州的县，如今很多县都从历史模式中大幅由红转蓝。

即使假设这一切都有完美的道德和法律原因，然而富有的私人党派若对选举造成了影响，也会成为头版新闻。不被看好的政党的民选官员会大声反对，威胁要提起诉讼，要求对接受资金的选举官员进行调查，并坚持要修改选举法，以防止今后再发生类似的事情。

作者要告诉大家一个真实的故事是，一项由左倾的州外捐赠者马克·扎克伯格（Mark Zuckerberg）和妻子普莉希拉·陈(Priscilla Chan)发起的计划，他们向据称是“无党派”非营利组织“科技与公民生活中心”(Center for Tech and Civic Life, CTCL)捐赠了3.5亿美元，而CTCL又将这笔资金重新赠送给全国各地成千上万的政府选举官员。作者所在公司，资本研究中心已经开始逐州研究这些资金是如何使用的，从乔治亚州开始(接下来是宾夕法尼亚)。资本研究中心是一个监督非营利组织使用和滥用的机构。

作者无法详细说明CTCL给乔治亚州各县的所有资金使用情况，因为作者所在的资本研究中心不会向美联社或其他任何人披露这些信息，尽管他们被要求在下一次的国税局申报文件中披露这些信息。但是，通过当地政府的记录和新闻报道，作者发现了许多大额拨款，大多数都是发给了最大的城市地区，CTCL也披露了哪些县收到了拨款，但没有公布具体数额。值得注意的是，CTCL资助的县不到美国的三分之一。

作者将拨款与相关县的官方投票总数联系起来，发现了一些值得注意的现象：

在CTCL已知的的10个最大的拨款中，有9个是给了乔·拜登(Joe Biden)获胜的县。

在向民主党总统候选人转移最多的10个县(与2016年和2020年的选票相比)中，有9个县获得了CTCL的拨款。

这9个受资助的县平均有13.7%的人倾向于民主党。在为拜登赢得最多选票的四个县中，有两个是科布(Cobb)县和格温内特(Gwinnet)县。

虽然从原始数据来看，CTCL资助的县中，唐纳德-川普赢得的县比拜登赢得的县更多，但它只资助了所有川普县的21%，而拜登县却有55%。因此，拜登县获得资助可能是川普县的2.5倍以上。

CTCL资助了为拜登提供10万张或更多选票的所有四个县。而且CTCL对这四个县的资助非常丰厚。按人均计算，他们每个男人、女人和孩子都能得到4.38美元到10.47美元的资助。

相比之下，就作者目前所知，川普获胜的县中，只有切罗基(Cherokee)县获得的资金超过了人均1美元，这个县的民主党支持率跃升了70%，而共和党的支持率则上升了24%。卡罗尔(Carroll)、卡姆登(Camden)和朗普金(Lumpkin)等川普县只拿到了大约半美元或更少。

拜登在33个县赢得了1万张或更多的选票。CTCL资助了其中的70%。相比之下，川普赢得了46个给他1万张或更多选票的县，但CTCL只资助了其中的20%。因此，支持拜登选票最多的县获得资助的可能性是支持川普选票最多的县的3.5倍。

将CTCL资助的所有44个县的选票加起来，共和党总统选票比上次选举增加了20.7万张。民主党总统选票猛增53万张，是共和党涨幅的2.5倍多。

以上这些数字听起来是否与党派无关？如果不是，那么乔治亚州和其他战场州的当局以及华盛顿的当局是否应该开始调查发生了什么。 CTCL这个被法律禁止在选举中代表党派出现的机构，是否越界了? 为了使公众对选举有信心，这样的机构需要避免出现任何利益冲突。

作者认为，乔治亚州立法机构应该问的最大问题是，是否应该允许加州的亿万富翁大摇大摆地进入桃子州，并资助他们选举的各个方面？他们是否希望亿万富翁在未来如此不公平、不公正地使用选举资源？对于乔治亚州和她的选民来说，这是一个活生生的问题，因为CTCL已经在为1月份的美国参议院第二轮选举提供更多的拨款给县级办公室。

评论：

经历过2020年大选出现的严重舞弊现象，让人们开始反思资本在这次大选中产生了什么影响。活生生的事实就是扎克伯格向全美各县捐赠了3.5亿美元，以控制和引导选民给民主党投票。媒体和社交媒体的一面倒反川也是被左派的资本大佬控制造成的。为了说服一些地方政府使用Dominion投票机，扎克伯格免费赠送。用上Dominion，就不知不觉的进入民主党设计的圈套。这场大选还没结束，尽管我坚信川普必胜，但是舞弊现象给美国社会造成了较大的伤害，人们对公平公正的选举恐怕需要时间来重新树立信心。
