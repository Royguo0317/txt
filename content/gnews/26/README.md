## 美国科学技术界遭遇中共技术间谍超限渗透
` 日本东京方舟农场` [轉載自GNews](https://gnews.org/zh-hans/1338995/)

#### 作者：画画

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/21092105/052ED3BF-070B-4BA4-A621-23968B9D101B.jpeg)

作者看到在2021年4月22号左右在中共国内全网统一口径宣传的一条新闻《李飞飞、颜宁等8位华人学者入选美国艺术与科学院院士》。其中李飞飞这个耳熟能详的华裔人工智能的科学家的名字，也是中共在美国网络安插的特洛伊木马。但是，排名第二名的颜宁何许人也倒是很少听闻，这也引起作者对此人探究的兴趣。

同时，作者也认为：中共的技术间谍向美国发起了超限战的渗透典型手段方式，毫无疑问的是来美国偷盗科学和技术。她会不会有蓝金黄美国科学界任务？ 还有可能会是再次配合释放放毒吗？这些都是作者大大的疑问。

首先，来认识一下颜宁女士吧！

颜宁（图1），女，1977年11月出生于山东章丘，结构生物学家，美国霍华德休斯医学研究所首届“国际青年科学家”，美国普林斯顿大学教授、博士生导师、清华大学生命科学学院兼职教授。其主要从事与疾病相关的重要膜转运蛋白、电压门控离子通道的结构与工作机理及膜蛋白调控胆固醇代谢通路的分子机制方面的研究。


```
1996年－2000年，在清华大学获学士学位；
  2000年－2004年，在美国普林斯顿大学分子生物学系学习，获博士学位；
  2005年－2007年，在美国普林斯顿大学分子生物学系从事博士后研究；
  2007年－2017年，任清华大学教授、博士生导师；
  2017年5月7日，已接受美国普林斯顿大学邀请，受聘普林斯顿大学分子生物学系雪莉·蒂尔曼终身讲席教授的职位；
  2019年4月30日，当选美国国家科学院外籍院士 ；
  2021年4月22日，颜宁当选美国艺术与科学院院士。
```


![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/21092116/62F470D3-5AD4-4B9E-B014-E0EDB088B378.jpeg)图1

从上述简介可以看出，颜宁也是一位中共国精心培养的生物工程领域的顶尖专家和技术间谍，甚至是从事蓝金黄美国科学界的中共顶级特务。其中，2000年-2004年在美国普林斯顿大学期间，特别标注她师从施一公。

大部分中国人并不陌生施一公，上至中共国家级宣传机构，下至知乎、抖音、微信朋友圈等中共媒体，几乎把生物技术偷盗的施一公像当年钱学森把原子弹技术偷回中共国一样的吹捧。

2008年，施一公回中共国加入清华大学并出任清华大学生命科学与医学研究院副院长。

2009年，决定成立清华大学生命科学学院，同时撤销清华大学生物科学与技术系建制。原生物科学与技术系主任、医学院常务副院长、生命科学与医学研究院副院长施一公教授任生命科学学院院长。

通过清华大学生命科学学院这个学术平台笼络了海内外顶级的生物基因工程、蛋白质等领域的专家，也是中共病毒制造重要参与者之一。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/21092123/6C75791E-6218-481C-90FC-AB1B2F5E19B5.jpeg)图2

通过检索颜宁的专利申请和发表的论文情况，发现颜宁与施一公同时在列的专利文献十几件之多，也在Science等科技杂志共同发布论文，具体参考图3和图4案例。上述师徒合作也可能为师徒恋埋下伏笔。当然中共国体制下，墙内的百姓听到大学里的师徒恋绯闻已经麻木不仁了，尤其施一公神一样人物，即使真有其事，有也无伤大雅，司空见惯。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/21092131/81787739-D2A3-4E6C-975A-041775C17554.jpeg)图3

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/21092139/EE6A4D52-C23E-43B3-806E-BEF35034611A.jpeg)图4

颜宁也是在中共国的科学界明星一样人物存在，几乎获得了中共科学界所有大奖项。


```
2012年，获得霍华德·休斯医学研究所国际青年科学家奖、谈家桢生命科学创新奖、美国HHMI首届国际青年科学家奖、中国青年女科学家奖；
  2013年，获CCTV2012年度科技创新人物称号；
  2014年，获第十八届“中国青年五四奖章”、何梁何利基金科学与技术进步奖；
  2015年，获高等学校科学研究优秀成果奖（科学技术）青年科学奖、国际蛋白质学会“青年科学家奖”、赛克勒国际生物物理奖；
  2016年，入选美国《自然》杂志评选为十位“中国科学之星”之一；
  2017年，获2016-2017年度“影响世界华人大奖”提名、第十八届吴杨奖、“影响中国”2017年度科技人物；
```


然后在2017年，在中共科学界鲜艳的光环笼罩下，颜宁从清华大学辞职，选择到普林斯顿大学任教。而颜宁对中共媒体称：只是想在相对更好的条件下进行她的科学研究，且自己永远是一名中国人。颜宁这番话进一步地洗脑中国科学界的知识分子，激发极端的民族主义和狭隘的爱国主义。

九层妖塔中共媒体通过类似《颜宁给国家未来敲响重重的警钟！》的宣传，通过小骂大帮忙策略维护洗脑墙内中国人和部分海外华人。例如：中共宣传：一个院士都要给管科研项目的行政处长点头哈腰的环境，一个外行官员可以管住内行大咖的体制，一个被权力笼罩的学术氛围，是需要再来一次改革开放了！作者认为，中共自从邓小平主政已经改革三四十年了，没有人再相信中共改革的谎话了，这次彻底消灭中共邪恶的体制！

同时，中共故意策划释放出颜宁的负面新闻，颜宁与恩师施一公有染等（如图5），致使颜宁离开清华大学。然后，有些打配合的媒体出来为中共国惋惜，放走了颜宁这样顶级科学家人才。作者不禁想问，若颜宁是真正的顶尖科技人才，中共会让她离开中共国吗？

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/21092149/9092B804-8697-489D-9D00-E4ED2D06DC58.jpeg)图5

总之，中共媒体的九层妖塔编出各种故事来哄骗各个不同阶层，不同版本故事洗脑愚弄中国人。

所以作者认为中共在国内编造出各种舆论和八卦新闻，让中国人和美国都相信对中共科研体制的不满，或因情感等个人的原因离开中共国去到美国。而实质上，都是中共背后策划，将其安排在美国科学界的特洛伊木马。然后通过在美国成功运作，把她包装成为美国国家顶级院士，并进行偷盗美国科学和技术，以及蓝金黄美国科学界。
