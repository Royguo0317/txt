**二、中国教会正在儒学攻势下堕落**

这是灾难的起头，这是末后的启示，儒教与基督教抢夺神的圣所一事就这样上演了。中国基督教两会关于《推进我国基督教中国化五年工作规划纲要》（以下简称《中国化纲要》）一文中提到这样一种论点，《中国化纲要》认为当下中国基督教神学建设存在一些问题，需要改进，认为有些地方教会“存在这是政府要求的一场‘运动’的错误认识，对推进工作敷衍推诿；有的地方在推进过程中简单复古、生搬硬套或牵强附会，走向庸俗化、低俗化”。甚至错误地将他们自己倡导的中国化早期方向——“三化异象”——认为是“海外反华反三自势力提出‘三化异象’蛊惑人心，造成对基督教中国化的误解”，因此应加强这方面的控制，“推进基督教中国化，不断增强与当代中国政治上自觉认同、文化上自觉融合、社会上自觉适应，建立与中华民族血脉相连、与祖国荣辱与共、与人民相濡以沫的感情”（见《中国化纲要》）。好一个建立感情的号召，这还是基督的教会吗？如果是，你们能把孔子拉出来再钉他在十字架上冒充“神的儿子”吗？

值得一提的是赵天恩早年提出的“三化异象（中国福音化、教会国度化、文化基督化）”（见 赵天恩），就是“中国化”中共早期方向，居然在《中国化纲要》文件的结尾出现自家人批自家人。颇有“鬼王赶鬼”的意味（见 马太福音 9:34）。想必，时下中国基督教神学思想建设真正进入了混乱化、淫乱化、小学化这样一个三化境地里。早在2018年3月27号就出台了该文件，如今全国三自教会基本上都在响应这一“运动”。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/17074701/Unknown-1-6.jpg)

运动往往有着它的迷惑性和煽动性。当你身处教会历史发展的当下，可能只是“不识庐山真面目，只缘生在此山中”，觉的十分有趣，甚至惊讶地令人目瞪口呆，似乎这样一个大巴比伦异像就展现在你面前。就在美国驻以色列大使馆迁址到耶路撒冷之后不久，我们国家就紧锣密鼓，又犹抱琵琶琵琶半遮面地进行着“基督教中国化”运动。悲哀的是，笔者得知以色列正在发动全国性“同性恋大游行”运动。时间是自2018年6月8号日至8月2日，为期两个月。不同的国家发生不同的事件似乎不能连在一起，但是都和教会有关，在这末后的时代，你不得不思考，这是怎样的一个异像呢？

经上记着说：“以色列的女子中不可有妓女；以色列的男子中不可有娈童（申23:17）。你们岂不知不义的人不能承受神的国吗？不要自欺！无论是淫乱的、拜偶像的、奸淫的、作娈童的、亲男色的（林前6:9）。又拆毁耶和华殿里娈童的屋子，就是妇女为亚舍拉织帐子的屋子”（王下23:7）。

上述经句足以说明同性滥交和淫乱已经进入圣殿，这实在是耶和华神所不能容忍的污秽，但基督教兼容儒学岂不正是宗教淫乱吗？岂不是将这种淫乱引进教会吗？难道我们文化里一直存有这样盲目自信到可以与神同等吗？回想改革以来，我们国家福音的传播可谓遍及大江南北，这其间，少不了义人为国家复兴代求的祷告。才使我们华夏文化的根脉和人类始祖联系上了。《DNA研究显示全球近30亿人共同拥有一位男性祖先》一文中验证了人类共同的祖先就是圣经宣称的“夏娃”（见 生物通）。因为，你只要信永生神上帝，就必信上帝所造的亚当夏娃，只有你相信神造世人，你就信亚伯拉罕是我们因信称义的信心之父了。如果你进一步认识亚伯拉罕，你就知道万国必因他得福，万国也因他而出。正是上帝看出他的义而这样赐福给他。耶和华神说：“亚伯拉罕必要成为强大的国，地上的万国都必因他得福”（创18:18）。

如今，上帝藉着赐福给亚伯拉罕，因着神的应许和不变的恩约，使这救恩的福音延伸到全地，即救恩临到万国。这是以色列的神和全人类的关系。然而，这是怎样的一种奥秘呢？因为“这奥秘如今显明出来，而且按着永生神的命，藉众先知的书指示万国的民，使他们信服真道”（罗马书 16:26），就是耶稣基督在十字架上为全人类的罪做成了救赎之功。

虽然有许多中国人还不知道国家的福祉和敬畏耶和华神以及耶稣基督的十字架救恩有关，但这是千真万确的。中国福音的种子如今四处发芽，大有福音化的趋向，有一首歌这样唱道，“我们带上中国的丝绸复兴，得胜的号角已在全地吹响，我们要迎来大地丰收”。这首脍炙人口福音歌曲似乎是词作者小敏的一个异像，我们似乎都沉浸在这种福音化到来的喜悦的盼望中，心就被感化，为这“大异象”不住地感谢神，然而，上帝似乎给我们一个更大的试验，就是“福音中国化”。

要知道国家兴衰如何，先看看古罗马帝国是怎样由一个敌基督者转变为敬畏神的上帝的国度的。从凯撒迫害基督教到君士但丁承认基督教（曾在313年颁布米兰诏书，承认基督教为合法且自由的宗教），并建立基督教政教合一的国家后期间经历了死亡与复活。可以说，自古罗马帝国君士但丁大帝以来，福音开始从罗马影响这个欧洲，乃至世界。其实，凯撒大帝时期的罗马并非迫害基督教最凶猛的时期，我们从圣经中的凯撒大帝时期犹大分封的王彼拉多看出，杀害基督并非他的本意，“彼拉多见说也无济于事，反要生乱，就拿水在众人面前洗手，说：‘流这义人的血，罪不在我，你们承当吧。’”(马太福音 27:24 )然而，昔日巴比伦帝国和亚述帝国都曾经强盛一时，结果衰退消失了。我们先看看圣经是怎么说的：“所以耶和华如此说：‘我必为你伸冤，为你报仇；我必使巴比伦的海枯竭，使她的泉源干涸。巴比伦必成为乱堆，为野狗的住处，令人惊骇、嗤笑，并且无人居住。’”（耶51:36-37）又有第二位天使接着说：“叫万民喝邪淫、大怒之酒的巴比伦大城倾倒了，倾倒了！”（启14:8）“耶和华必伸手攻击北方，毁灭亚述，使尼尼微荒凉，又干旱如旷野”（番2:13）。“尼尼微人哪，设何谋攻击耶和华呢？他必将你们灭绝净尽，灾难不再兴起”（鸿1:9）。经文中提到的尼尼微就是古亚述帝国都城，今天的伊拉克。耶稣基督说，“当审判的时候，尼尼微人要起来定这世代的罪，因为尼尼微人听了约拿所传的，就悔改了”（太12:41）。如今那些被毁灭的帝国只剩下他们文明的影子存在于欧洲和两河流域、中东地区。上帝拣选的以色列也不例外。比如还有波斯帝国、阿拉伯帝国和奥斯曼帝国等。纵观大国命运，无一例外。由于篇幅有限，我们仅此举例，表明大国兴亡与敬畏上帝有着直接关系。换言之，就是上帝始终主导世界局势发展，而今天的罗马——意大利，福音仍旧兴旺，欧洲仍然复兴。

照《圣经》所说，华夏文明何如？自称是炎黄子孙，拒不承认是亚当的后裔。无论中国如何回应上帝，但，耶和华神始终坐着为王，我们知道战争一直都在改写华夏文明，如我们知道的大战役有涿鹿之战、淝水之战、汉匈百年战争、崖山之战、松锦之战。作为封建帝国的终结，刚好六大战役（由于篇幅有限，不展开详述），满了“人的数目”。直到现代战争中的甲午海战和二次世界大战以来，战争的洗礼虽然没有摧毁华夏文明，但每一次战争都是一次新的文化融合与重组，或者叫宗教的混乱，但汉文化始终占据主导地位。这也许是有上帝永恒计划中的美意，但这个美意究竟要使用我们华夏文明干什么？也无从知晓。有西方学者说，上帝造十三亿中国人不会没有用处，神必定要大大地使用中国。中国的未来究竟如何，这是一个奥秘。但笔者肯定地告诉你，中国绝不会在上帝管教之外，神要我们在这末后的世代看见那更大的神迹。

诸位看官也许不同意笔者观点，坚持认为中华文明是礼仪之邦，甚至还有人坚持认为是万世师表。因为国人说我们敬天亲民，我们还出台了“社会主义核心价值观”，我们富强民主。真的如此这般倒也蒙福，问题是，我们所敬的“天”是自然神论和不可知论的“天”，我们古书中的“上帝”也是不可知的神明也。既然我们亲民（民主），我们的传统文化里的道德伦理为何如此破碎不堪？为何有法不依？男盗女娼呢？我们试问一下自己，我们多少时候都在面对“说一套，做一套”的虚假现象，为何这种现象如此司空见惯呢？我们的敬天在哪里？一个泛神论文化里的人自然避免不了宗教敬拜上的混乱，既承认有神，又相信“人定胜天”，相信以人为主导的神论观。在如此文化背景下，中华文明在女蜗补天、开天辟地的神话故事中出现了孔子，是我国自封建社会以来，影响最深，也最有争议，且一直影响着中国人的价值观的建立，被奉若神明的偶像人物。

中国知识分子拒不接受上帝，自认为最有力的证据就是《圣经》成书时间晚于《论语》。当然，基督徒不能认同这样的说法，理由并不是捍卫自我立场，乃是《圣经》乃上帝的默示，虽藉着使徒的手完成，但上帝是创造宇宙万物的神，孔子算什么？老子孟子算什么？岂不都是被造物吗？如此一来，谁先谁晚，一目了然。举例说明一下，假如我造了一个智能机器人，可它却自己编写了一本书。后来的人类视这本书为最高精神法典和自然法则。于是，造它的“我”就被历史所否定和遗忘。这其实就是中国人的思维方式，在一个向来没有“是就说‘是’”的文化语境下，国人普遍缺乏客观看待世界的眼光，却搞出了一个“实事求是”的口号。这种空空洞口号几乎未在任何领域实事求是过，竟然成了统治阶级治理天下的招牌。

一种高度自我崇拜的现象自儒家思想体系形成以来就一直被统治阶级拿来使用，利用其凝聚力逐渐演变成为强加于中国人头上的“紧箍咒”，成为统一思想的有力工具，这就是为什么中国人完全没有出现过真正敬畏真神的历史人物——像亚伯拉罕那样蒙上帝喜悦。因为中国文人实在是不能脱离这样的文化魔咒，因为国人坚持认为孔子是无罪的圣人。以至于要用孔儒来诠释《圣经》。神的话却成了次等真理，竟然把孔子学说纳入到“普遍启示”体系中来，可谓导致国家陷入大罪也。

从诸多可以查考的史料记载中，有关于孔子等思想家的记载都认为他们是高大上的“义人”“圣人”，完美得像神一样的人物，从未说孔子是罪人。不像《圣经》记载，不仅真实地记载了门徒们的软弱，还记述了耶稣的软弱，主在客西马尼园的祷告，求天父挪去那苦杯（路22:42、太26:42）。而我们国家人物传记都是只写好的一面，负面的绝对不写，这种完全脱离事实的史记文化导致国人养成了虚伪好面子的为人处事的习惯，这种中国式的虚伪就这样被写进中国人的基因里。一个大国如果缺乏纠错机制和认罪文化，那是很难改变的一种罪性文化的传承，更不可能诞生什么“圣典”或“法典”之类的精神产物，可以指导人向真理迈进。

不难发现，中国文化的糟粕就在于缺乏认罪文化。民国时期，国学功底深厚的章力生出生在中国尚在探索君主立宪制的混乱时期，认识上帝后毅然决然地抛弃了儒学枷锁，在皈依基督后著书立说十分勤奋，他在最具影响的神学著作《人文主义批判》一书中，有关儒家思想有这样一段表述，“孺教的本质──孔子——一生饥渴慕道，对于人生的意义，有深切的体认，其‘朝闻道，夕死可矣’， 至今仍被尊为‘万世师表’。但是他一生追求渴慕、率性修道的结果，仅仅止于‘属人的’，‘属地的’境界(参 约三12～21,27,31)；所谓‘性与天道’，却是‘不可得闻’，完全茫然。即是‘止于属人的道理’就是罪人思考的罪的结果，必然越修越茫然”。章力生进一步批判说：“他的理想的天国，乃是建立在人上面。他认为祗要明德修身，便可谋天下之治平，止人类于至善。这乃是一种彻头彻尾的‘以人为本的’(Anthropocentric)人文主义。所以中西学者，每称儒教是一种‘人文教’。这种彻头彻尾的人本位思想实际上就是人要做评审”。孔子只是提出“善治”的概念，却未提出“善治”的方法。而我们的文化里一直都充满了虚伪的口号和理念，使行和说截然分开，其原因就是儒学的根本是止于人，非真神上帝也。在没有敬畏神的文化里谈齐家治国平天下，岂不荒唐。

若有国学迷仍不肯降服真理，首先，你要认识到一点，不畏惧神，诚信体系无法建立，没有诚信，或不健全的诚信就注定不能建立诚信体系，没有这种诚信文化，人的一切行为则无从约束。这就是一系列社会问题的根源。
