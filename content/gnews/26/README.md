## CCP滑向深淵——英國首相有望數月內將華為踢出英國5G市場

編撰：夢中情人、映蘭軒、文寶、臺灣荔枝、文秀、Lori文噠
![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/07012948/7.7-1-3.jpg)
據英國《每日電訊報》周六報導，英國首相鮑裏斯•約翰遜（Boris Johnson）將逐步淘汰中國科技巨頭華為在英國5G網絡中的應用，這一過程最快計劃六個月內完成。

官員們正在起草提案，停止英國5G網絡中安裝新的華為技術設備，並加快淘汰現有華為技術。英國政府此舉是基於該國情報機構GCHQ對華為技術提出了新的安全隱憂。

一份GCHQ國家網絡安全中心在報告中得出結論：美國對華為施加新的制裁，該制裁旨在阻斷華為向美國制造商購買芯片，對華為產生了“嚴重”影響，將迫使該公司使用不受信任的技術，並大大改變了GCHQ的風險計算，由此可能導致風險無法控制。

該報告預計將在本周提交給約翰遜。約翰遜發言人本周早些時候表示，官員們將很快完成審閱美國對華為最新制裁所帶來的影響。

**編者觀點**

引領爆料革命的郭先生是揭露華為並不單純是一家私人企業第一人，而是服務於中南坑盜國賊利益集團的軍方企業，他進一步預測華為雖垂死掙紮但一定和海航的結局一樣。

華為這家最初不為人知的企業，利用國家情報力量在全球範圍內盜取行業最先進技術，在國家開發銀行天量資本的支持下，不計成本地掠奪市場。

自2015年以來，華為企業級路由器產品在中國市場保持第一；以太網交換機在中國市場居首位，全球市場位列第二；光網絡在亞太地區市場占有率位居第一。

中共利用各種力量擴大華為在全球的5G市場份額，包括正在提供超過1000億美元的激勵措施，以資助客戶購買其設備，使其在短短的一二十年，成為除北美以外各大洲的領先供應商，占領了全球基礎設施市場40％的份額，其主要對手是芬蘭諾基亞（17％）和瑞典愛立信（14％）。

在不久的將來，若任由其發展下去，華為無疑會以更大優勢占領全球市場，屆時將難有競爭對手將其撼動。如此不計代價的以國家資本彎道超車的做法，是否能夠讓華為如魚得水，稱心如意？

但是劇情反轉直下,以郭文貴先生帶領的爆料革命，無疑給華為施予重擊。郭先生的真實、準確、超前的情報信息，讓西方世界刷新了對華為是中共軍方、間諜企業的本質，也將CCP欲利用華為監控和奴役全世界的野心昭示天下。

全世界以美國為首的西方同盟開始明白，華為其軍方背景，為中共政權攫取各種信息和情報，因涉及國家安全，監控個人隱私，已經對西方民主國家造成致命威脅。西方社會必須要堅決對CCP說NO！

未來五年，5G版圖將重新劃定，因時間緊迫，美及其盟友必須迅速采取行動。自美國對華為開始制裁以來，華為5G已被美、澳、日、印、韓、加、新等國家聯手禁用，加入此隊伍的國家也在不斷擴大。

雖然英國起初並沒有和美國保持一致，但因COVID-19疫情爆發帶來的重創，包括對其皇室成員以及首相約翰遜生命的威脅，英國當局也發生態度驚天逆轉，英情報機構GCHQ對華為技術風險重新評估，旨在為下一步剔除華為做鋪墊。正如周四（7月2日）首相約翰遜說，他要在華為的決定上謹慎行事，因為政府不希望重要的基礎受制於“潛在敵對國家的供應商”。

未來數月，對美國來說，可能會收到英國盟友傳來的好消息。
