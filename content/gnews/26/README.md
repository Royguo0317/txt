5月9日福克斯玛丽亚·巴蒂罗莫（Maria Bartiromo）的《周日早间未来》（Sunday Morning Futures）除了采访了前众议院议长纽特·金里奇（Newt Gingrich）外【1】，也与罗恩·约翰逊（Ron Johnson）参议员谈论了中共病毒、拜登政府不作为等问题，以下摘译了玛丽亚与约翰逊参议员的对话：【2】

玛丽亚说：接下来，随着中共（恶行的）进一步曝光，美国国务院获得了新的爆料文件，披露中共军事科学家在这次疫情爆发前五年就讨论过将中共病毒武器化。

我们会和参议员罗恩·约翰逊谈这一个问题。

（视频开场）

兰德·保罗（Rand Paul）：关于病毒的起源有一个巨大的伦理问题，伦理问题是，我们应该做功能增强实验吗？我们是不是应该在实验室里对动物病毒进行升级，让它们变得更容易感染人类？

我不能肯定地说这是实验室泄露，我认为这是一个合理的质询，但是你应该问的是福奇，为什么他在监督这些实验室时，允许他们做功能增强？为什么他允许实验室获得政府资金来升级动物病毒，让它们能感染人类？

（结束视频剪辑）

玛丽亚：哇，这是参议员兰德·保罗上个月在这个节目中提出的这些问题，他想知道中共病毒的起源。拜登总统当时尚未承认这场疫情是从中共国武汉开始的，更不用说向习近平提起了，这个问题他从来没有提过。

现在加入我的是威斯康星州的参议员罗恩·约翰逊，他是参议院国土安全和外交关系委员会的成员。

参议员，真高兴今天早上见到您，非常感谢您在这里。

今天早上我收到了一篇报道，来自澳大利亚的沙里·马克森（Sharri Markson），她写了这篇报道，她还说，中共国军事科学家在中共病毒疫情开始的五年前就讨论了SARS冠状病毒的武器化问题，并在一份文件中概述了他们的想法，该文件预测第三次世界大战将以生物武器方式进行。参议员，您怎么看？

约翰逊：早上好，玛丽亚。首先，让我说，这会震惊吗？这肯定不会令我震惊。

玛丽亚：不，不会吧？

约翰逊：当您看到中共国在南海诸岛军事化及其在全球范围内的恶劣行径时，我们应该感到关切。

安东尼·福奇显然继续资助功能增强研究，并与武汉实验室合作，这确实让您难以置信，因为他知道——他必须非常清楚武汉实验室——几乎中共国的任何组织都与中国共产党有联系，也与中共国军队有联系。

那么，我们在和中共国合作做什么？我同意保罗参议员的观点，这是一些严肃的问题，我们需要一些答案。但是，多年来，我一直试图从联邦机构那里得到答案，我不想再等待下去。

玛丽亚：好吧，我们继续看到有关亨特·拜登（Hunter Biden）以及他在投资方面与中共国的关系的消息，他仍然在那里拥有一家中共国公司的10％股份。

您对本届政府有什么看法，以及他们对中共国强硬的能力？我们刚刚收到纽特·金里奇议长的来信。他说，许多政府对中共国情有独钟。我的意思是，中共希望超越美国成为第一大国，他们是不遵守承诺的共产主义者。看看香港，（拜登政府）一点也不问责，我们看到中共国警察现在大摇大摆地走在香港。是因为本届政府存在利益冲突吗？

约翰逊：嗯，我非常担心。您知道，这就是问题所在，这是我和格拉斯利（Grassley）参议员在大选前发表报告的原因之一，这样美国人民就可以理解亨特·拜登和拜登一家的这些外国财政纠葛，以及这可能会如何影响美国的外交政策。

我是说，反情报威胁，勒索威胁，我们不知道，什么都不知道，我们又写了一封督促信，我们正试图从这些政府机构得到信息，但媒体对我们的报道完全不屑一顾，说里面什么都没有。但里面确实有各种各样的信息。
