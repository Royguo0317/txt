#### 中美因世卫组织在武汉的调查任务问题发生冲突

日内瓦（路透社） — 美国周一呼吁中共国允许世界卫生组织（WHO）的一个专家小组采访武汉市中心的“护理人员、前病患和实验室工作人员”，引来了北京的指责。

[!\[\]()!\[\](https://spark.adobe.com/page/7C6wj9RKVRuux/images/775b99d7-b4a1-4a52-9047-75c0e222e30a.jpg?asset_id=7e305cbf-c9d2-4f31-ae17-e762a94d2220&amp;img_etag=%22170f74abe7254582ce18d8d8b42014bd%22&amp;size=2560)](https://spark.adobe.com/page/7C6wj9RKVRuux/images/775b99d7-b4a1-4a52-9047-75c0e222e30a.jpg?asset_id=7e305cbf-c9d2-4f31-ae17-e762a94d2220&amp;img_etag=%22170f74abe7254582ce18d8d8b42014bd%22&amp;size=1024)

由世卫组织领导的独立专家小组于1月14日抵达武汉，以尝试确定新冠状病毒起源。他们在为期两周的隔离期间与中共国同行进行电话会议，然后开始实地工作。

美国指责中共国掩盖了本国最初爆发的程度，美国呼吁进行一次由世卫组织领导的“透明”调查，并批评访问条款，根据访问条款，中共国专家已进行了第一阶段的研究。

[!\[\]()!\[\](https://spark.adobe.com/page/7C6wj9RKVRuux/images/498d95c4-dacf-491a-99c9-0250d1b97b8c.jpg?asset_id=655bd341-5311-4e9d-90b1-3be6e08c599c&amp;img_etag=%22606fce82a4e8b17562dff8f5b612f837%22&amp;size=2560)](https://spark.adobe.com/page/7C6wj9RKVRuux/images/498d95c4-dacf-491a-99c9-0250d1b97b8c.jpg?asset_id=655bd341-5311-4e9d-90b1-3be6e08c599c&amp;img_etag=%22606fce82a4e8b17562dff8f5b612f837%22&amp;size=1024)

美国代表团团长、卫生与公共服务部部长加勒特•格里格斯比（Garrett Grigsby）表示，中共国应分享所有采自武汉海鲜市场的动物、人类、和环境样本的科学研究，人们认为SARS-CoV-2病毒2019年末出现在那里。

他告诉世卫组织执行委员会，对此类遗传数据进行比较分析将有助于“寻找引起COVID-19大流行爆发的重叠和潜在来源”。

格里格斯比说：“我们负有神圣的职责，以确保这项重要的调查可信且客观透明地进行。”他还提到了在英国、南非和巴西发现的病毒变种。

[!\[\]()!\[\](https://spark.adobe.com/page/7C6wj9RKVRuux/images/4a46ef49-69ec-4ec7-9258-6254b74984da.jpg?asset_id=354efbfa-cf2b-408f-afcf-1736e06e153c&amp;img_etag=%22c44d28898e67325ff1dde1a00096f0a8%22&amp;size=2560)](https://spark.adobe.com/page/7C6wj9RKVRuux/images/4a46ef49-69ec-4ec7-9258-6254b74984da.jpg?asset_id=354efbfa-cf2b-408f-afcf-1736e06e153c&amp;img_etag=%22c44d28898e67325ff1dde1a00096f0a8%22&amp;size=1024)

中国国家卫健委卫生应急办公室主任孙阳对世卫组织执委会说：“病毒起源研究具有科学性。它需要协调、合作。我们必须停止任何政治压力。”

澳大利亚代表团还呼吁世卫组织团队调查“有关数据、信息和关键地点”。

世卫组织紧急事务负责人迈克•瑞安（Mike Ryan）上周五对记者说：“无法保证能找到答案。要完全确定起源是一项艰巨的任务，有时可能需要两次、三次或四次的尝试才能在不同的环境中做到这一点。”
