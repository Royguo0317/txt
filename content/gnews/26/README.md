1. 1.m-RNA疫苗是一种新的、实验性的疫苗设计，以前从未在人类身上使用过。

2. 2.m-RNA疫苗还没有得到加拿大卫生部或美国CDC或大多数国家的全面授权。

在美国或世界上大多数国家，m-RNA疫苗还没有得到加拿大卫生部或美国CDC的全面授权。在加拿大，它们处于 “临时授权 “状态。在美国，CDC将其置于 “紧急使用授权 “之下。完整的疫苗批准需要几年时间和多种安全考虑–这在mRNA疫苗上还没有发生。

3. 为了符合 “紧急使用授权 “的条件，必须有一个紧急情况。

对于老年人口、弱势人群、医护人员来说，当然是有紧急情况的。

加拿大有几千人死于covid-19，但死亡者的平均年龄是83岁。Covid-19并不对我们的孩子构成威胁。他们死于Covid的风险低于0.003%–这甚至低于他们死于流感的风险。

4.  儿童不容易将(CCP)病毒传染给成人。在许多国家，教师的covid-19比率明显低于普通人群。

5.  m-RNA疫苗已经与数千人的死亡有关。

由美国政府出台的疫苗不良反应报告显示，  截至2021年6月4日，在短短几个月的时间里，有5,888例死亡与m-RNA疫苗有关。

6.  m-RNA疫苗已经给世界各地的孩子带来了严重的医疗问题。患心肌炎的风险确实明显增加 。

7.  在小规模的试验中，疫苗对预防儿童的covid-19的好处却微不足道。

辉瑞公司的小型试验，虽不能发现真正的风险，但也解决不了中长期的问题和长期存在的副作用。
