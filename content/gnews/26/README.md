台湾在我的心里始终像个美丽的女孩，成熟已久却待阁未嫁，显然台湾早已春心荡漾急盼着坐在花轿之上，去往自已的应有的新生活，今天终于等来迎亲的锣鼓声。台湾的新生活马上就要开始了！这就是台湾的宿命！

我没有去过台湾，但我内心是无比的喜欢他，至与为什么，我一时真说不出来，抑或是那种自然的发自内心的喜欢吧！

机会往往是对手给予的，今天台湾一连串的好机会都是对岸的流氓CCP亲手送的。我们几乎可以看到台湾马上就要以一个国家或者完全独立的政府的身份出现在国际的舞台上。尽管以前也是有自己的邦交国，在国际上以国家的身份参加一些活动，但是总是躲躲闪闪遮遮掩掩的，有时会被无厘头的加上“中国台湾”名字出现在各种的国际活动场合。还有台湾驻其他国家的机构只能称为台湾办事处而不能称台湾大使馆。

战友们！我们可以清晰的看到前面有去年的特朗普政府的卫生部长阿札尔 (Alex Azar) 与国务院次卿克拉奇 (Keith Krach) 访台，今年有在4月9日美国国务院公布（新版对台交往准则）后第四天的13日，台湾迎来拜登派出的代表团前美国参议员多德（Chris Dodd）、前副国务卿阿米蒂奇（Richard Armitage）和斯坦伯格（James Steinberg）。还有美国的参众两院极力提议台湾成为WHO（世界卫生组织）中的一员。

这一切都是CCP亲手为台湾缝制的美丽嫁衣。我们今天不讲述台湾的历史，单讲最近几年中共这个低级无赖怎么弄巧成拙的。中共是做梦都想得到台湾。

第一，CCP觊觎台湾的战略位置。拥有台湾，中共不但可以直通蓝海海洋，直接跳出第一岛链，还可以扼守台湾海峡，威胁日本和韩国能源运输线，以获取利益和其他不可告人的目的。

第二，台湾的高科技制造技术（芯片）和大量科技人才也是CCP急于得到的。

第三，抢钱，抢人，抢地盘，是共产党的核心的核心。很简单，抢钱直接归自己所有，抢人是为了可以当自己的奴隶和人肉盾牌，抢地盘可以像以前的皇帝一样想给谁就给谁（毛泽东不是就送给过朝鲜，越南，俄罗斯，印度大片土地）。
