**原文翻译：**

# **报告称，“多猫腻”(Dominion)错误率为68%，是“为制造系统性舞弊而故意设计”**

**来自联合安全行动小组对密歇根州安特里姆县选举结果的一份法证报告认为，“多猫腻”投票系统是“故意而有目的设计的，存在固有错误，为的是制造系统性舞弊和影响选举结果”。**

该报告的理由是，虽然“联邦选举委员会准则规定的选举容错率为25万分之一（.0008%）”，但是他们观察到“多猫腻”选举系统的错误率为68.05%。其调查结果集中在安特里姆县，该县选举结果出现故障，将数千张川普总统的选票转投给了乔·拜登。

12月13日的报告摘要指出：“这表明在安全和选举诚信方面出现了重大致命的错误。”

该报告的关键在于让人知道，这些错误是“机器和/或软件错误造成的，而不是人为错误。”
