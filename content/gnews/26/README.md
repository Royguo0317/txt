以色列的一项研究表明，病毒疫苗已无法再针对南非变种B.1.351正常有效。

特拉维夫-现有疫苗还能预防病毒突变体的感染吗？英国变种B.1.1.7似乎就是这种情况，它在许多国家和德国几乎完全取代了Sars-CoV-2的“野生型”；最多减少一些效果。但是对于变种B.1.351完全不同，后者首次出现在南非。

以色列还有一个坏消息：根据特拉维夫大学和以色列最大的健康保险公司Clalit Healthcare Organization的最新研究，接种疫苗的人被感染南非变种的频率是未接种疫苗的人的八倍。但是，总数总体上很小，因为这种病毒株在以色列几乎不流行，其中90％的感染归因于英国变种。迄今为止，该研究仅在medRxiV上预印，尚未进行审查。

mRNA疫苗对南非病毒变体的的几乎没有保护作用。

在以色列，已有490万人（占人口的53％）已经接受了两种疫苗的接种。 那里只注射了mRNA疫苗，其中大部分来自Biontech / Pfizer。 在其研究中，研究小组检查了来自150人的样本，这些人的冠状病毒检测呈阳性，部分无症状，但部分也患病，尽管他们已经接受了第一次疫苗接种或什至都接种了疫苗。
