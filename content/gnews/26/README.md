《动物庄园》是由英国作家乔治·奥威尔 (George Owell) 创作的托寓小说，是非常著名的反乌托邦的作品之一，1945年8月正式于英国出版。该小说曾于1954年和1999年分别被英国和美国改编成电影，但是这些电影部分弱化了小说的情节。据说网飞 (Netflix) 已经购得了《动物庄园》的电影改编权，期待更精彩的电影版本出现。

奥威尔称，此一寓言反映了1917年俄国革命至踏入斯大林时期的历史事件，用动物对人类抗争后自建家园来再现前苏联的整个历史情形，是一则讽刺斯大林的寓言故事。他认为，苏联已是惨无人道的独裁统治，而这种统治手段建立在对斯大林的个人崇拜和大清洗上。另外，他在乌克兰译本的序中说：“对我来说，极其重要的是，西欧的人们应该看清苏联政权的真实面目。极权主义的宣传能多么轻易地控制民主国家开明人民的舆论。”如果奥威尔还活着，他一定会说，全世界都应该看清楚共产主义的本质和真实面貌并打破其魔咒！

很多人说历史总是惊人的相似，实际上不是历史总是往复，而是极权专制导致社会发展会走向一个必然的模式。无论是奴隶制社会，封建皇权社会，还是社会主义/共产主义社会无一不是重复着一样的统治模式和悲剧。马克思编造了共产主义的谎言，他为共产主义穿上了乌托邦、理想国的外衣，事实上共产主义是极权专制的集大成者。

我高中政治课上关于共产主义的说法是唯物主义、无神论、不存在私有财产、生产资料公有制、按需分配、阶级将消亡、人人平等、货币将消失、国家和政府被废除，当时虽然没有很深的见解，但是认为共产主义绝对不可能实现。

我认为理想国/乌托邦只存在于上帝刚刚造好人类时的伊甸园之中，后来人被撒旦的化身蛇引诱吃了善恶树上的果子后，被赶出了伊甸园。这之后就再也不存在理想国了，因为人已经被撒旦败坏了，开始有了恶念。共产主义实际上就是利用了人对乌托邦般的美好生活的向往，利用人性弱点、人性之恶而掌权并维护其统治的。

政治学家总是说极左会形成共产主义，极右会形成纳粹。在我看来，就结果来说二者并无区别，无论极左产生的共产主义还是极右产生的纳粹都是极权专制，都是共产主义，换汤不换药而已。所以我认为极左和极右只是混淆视听的词语，都是邪恶的共产主义极权专制借着其所创造的伪科学，制造矛盾，挑起争端及仇恨，加上冠冕堂皇的外包装，来实现在全球范围内的渗透和统治。如今典型的如安提法运动，打着反纳粹的幌子，思维与做法却与共产主义一脉相承。在我看来这就是共产主义裹上了糖衣而已，像里根总统所言，法西斯主义必会改头换面以自由主义的面目回到美国。

奥威尔在此书中成功预测了前苏联最后的结局——倒台。我想说的是奥威尔在这本书中不仅讽喻了前苏联，它描述的对所有历史与现存的极权专制都是契合的。奥威尔1950年1月21日就去世了。纵观中共执政后中共国社会的发展轨迹，像是按照《动物庄园》这个剧本在表演一样，完全继承了前苏联的极权专制统治，且有过之而无不及。

让我们通过动物庄园来揭露共产主义的本质。

夜里，在曼纳庄园的主人琼斯先生睡着后，一头受动物们尊敬即将离世的公猪老少校召集动物们开会，并发表了一番演讲。它痛斥了动物们的统治者人类对它们劳动成果即私有财产的剥削和压榨以及最后动物失去利用价值后的将来的结局。尽管老少校所过的12年生命是猪合乎自然的生活，它还说：“动物们的苦难完全是由人类造成的，只有推翻人类的统治动物们才会富足和自由。所以一定要造反，要牢记并一代一代传下去，同志们（动物们）必须团结下去与所有人类为敌，对待人类及他们的行为我们必须抱着不共戴天，他们必亡的信念。”

动物主义在这次会议上初具雏形。老少校把《英格兰之兽》之歌教给了动物同志们，这首歌描述了如天堂般美好的梦想未来，动物们学着并唱着这首歌，进入空前激昂和亢奋的状态，无疑这首歌是一剂“鸡血”。那晚之后没几天老少校就离世了，秘密会议仍然继续进行，由被认为是最聪明的猪领导，主要学习动物主义。

小说开头的部分，主要是对动物主义（共产主义）前期的铺垫和宣扬，令我想起了列宁曾经说过的话“我们必须仇恨——仇恨是共产主义的基础”，前苏联教育专员说的话“我们要的是恨，只有那样我们才能征服宇宙”。对照小说里的情节是不是惊人的相似？曼纳庄园里的动物统治者只是琼斯先生这一个人类，并不是所有人类都对庄园里的动物做过什么。在公猪老少校的倡导下，他们似乎代表了所有的动物，就像马克思主义一样要代表所有的农民和工人。我们可以换一句话说“共产主义要统治全世界，正是通过控制工会、控制农民和工人的思想意识并以仇恨为手段来迅速扩展和传播的”，一切资本家、地主和富农都是仇敌，资本主义是共产主义的仇敌。正如前苏联在欧洲和亚洲扩张势力的手段，亦如中共近百年的发展史以及在前苏联解体之后，中共在整个世界范围内渗透和发展的策略。

小说中的《英格兰之兽》所唱的美梦也正像中共一直给老百姓描绘的美梦，中国梦，并且一个阶段一个梦，很多梦，一个也不能实现。中共宣布2020年实现消除贫困的梦，却还有6亿人每个月收入1000元人民币或不足1000元；更讽刺的是习近平在2020年5月还宣布基本实现了小康社会。中共创建之初提出的目标是“解放全人类”，看看100年后的今天，中共国人被解救到何种水深火热的境地？世界被中共渗透、祸害到如此地步，中共实现的是以一个个大梦来迷惑人，从而颠覆全人类的目标吧？
