- **[06.18 Daily Caller：专家称他曾警告苹果公司在中国开展业务，并表示你们明白习近平是谁吗](https://dailycaller.com/2021/06/18/xi-jinping-apple-business-china-guthrie/)**


据报道，苹果公司聘请了一位中国问题专家帮助他们驾驭中国市场，该专家警告公司高管，在一个对西方寻求权力的专制国家开展业务有风险。

道格-格思里在周四发表的一篇报道中向《纽约时报》讲述了他为苹果公司工作的时间。他说，他对中国在习近平主席的领导下越来越多的威权主义转向以及他们希望获得对美国公司的影响敲响了警钟。

据报道，当苹果公司在2004年决定将其制造基地扩展到中国时，其制造伙伴指着一座小山，告诉在场的两位苹果公司高管，工厂将建在那里。《泰晤士报》报道说，一年后，这座山不见了，工厂已全面投入运营。

据报道，随着苹果公司在中国的影响力开始增长，长期研究中国问题的学者格思里于2014年被该公司聘用，就如何驾驭中国市场向他们提供建议。他告诉《泰晤士报》，他警告说，苹果公司越来越依赖中国的制造业，这正符合习近平对美国公司施加影响的目标。

“我正在与商界领袖接触，我说。’你们了解习近平是谁吗？你们是否在听这里发生了什么？这是我的重要名片，”2019年离开苹果的格思里说。

“我们错了，”格思里补充说。”通病是习近平。”

据报道，苹果公司还不得不努力应对中共强加的苛刻要求和法律，这些要求和法律使其中国客户的数据面临风险，并帮助协助中国政府的审查行动，《泰晤士报》上个月也报道了这一点。

“我们从未在中国或我们运营的任何地方损害过我们的用户或他们的数据安全，”苹果公司发言人告诉《纽约时报》，并补充说，格思里是 “中层员工，没有在苹果公司制定过政策”。

除苹果公司外，耐克和可口可乐等公司也因反对美国政府阻止在中国使用强迫劳动的努力而受到批评。

苹果公司也对美国的一些政治问题直言不讳，如乔治亚州的投票法，尽管他们与中国政府有合作关系。

- **[06.19 Washington Examiner：美国联邦通信委员会加大对 “国家安全威胁 “的压力，华为在法院上诉中败诉](https://www.washingtonexaminer.com/news/huawei-loses-appeal-fcc-increases-pressure-national-security-threat)**


华为对联邦通信委员会宣布其为国家安全威胁并拒绝其获得联邦资金的命令提出的法庭挑战被驳回，因为联邦通信委员会加大了对该公司的压力，并寻求拒绝对其以及与中共有联系的类似企业的所有未来授权。

美国联邦通信委员会（FCC）最初在去年夏天将华为和中兴定为国家安全威胁，禁止这两家公司获得美国政府的补贴，以建设通信基础设施。该委员会表示，来自其83亿美元年度普遍服务基金的政府补贴 “不得再用于购买、获得、维护、改进、修改或以其他方式支持由华为生产或提供的任何设备或服务”。公共安全和国土安全局认为，华为是一个国家安全威胁。在华为寻求对该指定进行审查后，联邦通信委员会于12月再次确认该公司 “对我们国家通信网络的安全和完整性构成威胁”。

2月，华为向美国第五巡回上诉法院提出上诉，声称FCC的裁决 “超出了FCC的法定权限，违反了联邦法律和宪法，是任意的、任性的和滥用自由裁量权的，而且没有实质性证据支持”。三位巡回法官（小布什任命的Jennifer Walker Elrod法官和特朗普任命的Cory Wilson法官和Stuart Kyle Duncan法官）于周五驳回了华为的上诉。

“FCC处理的是国家通信，而不是外交关系。它不是国防部，或国家安全局，或总统。如果我们确信FCC在这里充当了’一种初级的[国务院]’……我们会把这个规则放在一边。但是，没有这样的诡计，”冗长的意见中写道。”评估电信网络的安全风险属于联邦通信委员会的工作范围。而且，该机构对国家安全的判断得到了其他专家机构和官员的有力支持。因此，我们相信，在制定该规则时，该机构在国会赋予它监管通信的广泛权力范围内合理地行事。此外，在仔细考虑了这些公司根据《行政程序法》和《宪法》提出的其他挑战后，我们发现这些挑战也是不成立的。”

前共和党FCC主席Ajit Pai周五庆祝了这一裁决，他说：”2019年，FCC采取了果断行动，以确保通信供应链，禁止公司使用FCC补贴从被视为国家安全威胁的实体购买设备。很高兴第五巡回法院今天驳回了#华为对我们决定的上诉”。

华为在上诉法院败诉的前一天，美国联邦通信委员会提出了一项新的规则，将进一步打击中国电信，该委员会表示，该提案将 “禁止今后对被认为对国家安全构成不可接受的风险的通信设备进行授权”。

中国外交部发言人赵立坚周四对联邦通信委员会的新提案表示遗憾。

“直到今天，美国的一些人仍然在延伸国家安全的概念，滥用国家权力，在没有提供任何证据的情况下打击某些国家和公司，”赵立坚说，”美国应该立即停止滥用国家安全，对某些中国公司进行无理的压制。”

参议院两党在5月提出的一项法案将强制要求联邦通信委员会不再审查或批准联邦通信委员会指定名单上的实体的业务请求，参议院办公室表示，这将 “防止华为和其他中国国家支持的公司在美国境内进一步整合和销售”，”无论是否涉及联邦资金”。

3月，联邦通信委员会发布了一份 “被认为对国家安全构成威胁 “的中国通信公司的最新名单，其中包括华为和其他企业，它说所有 “生产的电信设备和服务已被发现对美国国家安全或美国人的安全和保障构成不可接受的风险”。该黑名单还包括中国公司的子公司和关联公司，是根据去年通过的《安全网络法》建立的。

司法部和美国情报机构认为，华为、中兴和其他中国公司与执政的中国共产党携手合作，有可能让中国的监控国家接触到世界各地的硬件和网络。美国一直试图限制华为的全球影响力，特别是在5G领域，推动其 “五眼 “伙伴澳大利亚、加拿大、新西兰和英国拒绝在其通信网络中使用华为的技术。

2020年2月，司法部公布了2019年对华为指控的补充起诉书，指控华为犯有敲诈勒索和阴谋窃取商业机密罪。

商务部在2020年12月解释说，华为于2019年5月被列入实体名单，因为该公司及其关联公司 “从事了违反美国国家安全或外交政策利益的活动”。工业和安全局在2020年5月修订了其外国生产的直接产品规则，以 “针对华为收购作为某些美国软件和技术的直接产品的半导体”，并在2020年8月宣布了更广泛的限制。

- **[06.19 Zero Hedge：以下是阻碍拜登的全球企业税收协议的所有 “技术障碍”](https://www.zerohedge.com/economics/here-are-all-technical-obstacles-standing-way-bidens-global-corporate-tax-deal)**
正如我们已经说过的那样，尽管西方媒体有很多乐观的报道，拜登政府推动的新的最低公司税可能永远不会成功–这一现实将最终限制美国公司税率的提高程度，以资助拜登的 “伟大社会 “野心。


即使七国集团在最近的会议上达成了初步协议，对经合组织的国际税收框架进行全面修订–这将构成一个世纪以来国际税收方面最大的动荡–将需要几十个国家的同意，包括像爱尔兰、印度尼西亚和新加坡这样成功利用其低税率推动经济发展的国家。其中任何一个国家都可以通过拒绝降低税率来破坏该协议。

为了试图弥补这一点，拜登政府向外国政府承诺，他们将有权从美国跨国公司创造的利润中获得更大的份额。七国集团的协议将对 “最大和最赚钱的跨国企业的利润超过10%的部分 “适用这一 “胡萝卜”。甚至还进行了谈判，专门将亚马逊的低利润率电子商务业务排除在外，允许根据其更有利可图的部门，如AWS的利润来征税。

在未来几周内，外交官们将在7月的20国集团会议前就新的公司税框架举行会谈，涉及100多个国家的政府，华盛顿希望在这次会议上能够达成协议的大纲。该计划要想成功，最终需要100多个国家同意。

彭博社报道说，鉴于所涉及的利益竞争范围惊人，企业争相将其排除在税收之外，而各国则争夺各种特殊利益的豁免权，这一过程最终可能需要数年时间–即使政府推动在夏季结束前取得重大突破–并涉及一个复杂的立法网络以弥补无数的 “技术 “复杂问题。这些问题包括。

商定哪些公司将被包括在内，以及决定政府如何在最低税率的情况下仍能使用税收激励措施来鼓励良性的经济活动，这些都是给经济合作与发展组织主持的多年会谈蒙上阴影的其他几个挑战。

包括巴西、印度和南非在内的24个发展中国家的政府间集团，根据其上个月发给其他国家政府的一份政策说明，希望逐步扩大范围，以包括100多家公司。

在七国集团同意重新分配超过10%利润的 “至少20%”之后，各国还必须决定分享多少税收。发展中经济体希望从在其境内运营的跨国公司那里获得尽可能大的税收收入楔子。

如果金融服务如预期的那样被排除在协议之外，这就构成了另一个挑战，因为在金融服务和科技公司之间划出一条明确的界限越来越难。

爱尔兰仍然是一个 “难点”。

此外，包括中国在内的一些国家希望在允许它们以税收优惠吸引高科技投资的规则中排除在外。”最低税率正在下放税收主权的一部分，以及你如何保持对某类外国投资的激励，”毕马威会计师事务所全球税务与法律主管大卫-林克说。”这是一个困难的问题。”

可以肯定的是，拜登有一个大胡萝卜可以提供。经合组织估计，美国对外国收入的更严格规定和15%的全球最低税率，每年可以产生1500亿美元的额外收入。

一项协议可能涉及到放弃对主要是美国科技公司的一系列征税，这些征税是各国近年来单方面颁布的，引起了美国的报复威胁。谈判者必须就哪些措施将被取消以及何时取消达成一致，以便恢复信任。

实施20国集团在7月或10月会议上达成的新规则将需要对条约和国内法律进行许多修改。这对欧盟来说尤其成问题，因为整个集团的税收变化指令需要一致同意，而几个国家可能会反对这种立法来执行经合组织的协议。除了爱尔兰对15%的最低税率持保留意见外，匈牙利总理维克多-欧尔班称该计划是 “荒谬的”。

美国可能会看到来自共和党人的挑战，因为一项协议可能需要在国会立法，并在参议院修改条约，需要三分之二的多数票。

虽然民主党人在很大程度上支持耶伦的努力，但由于有如此多的障碍，拜登不可能在准备加税时将此作为筹码。但是，由于他已经承诺了支出，政府很可能无论如何都会继续下去，放弃用税收抵消新支出的承诺，并最终允许美联储将大部分支出货币化，这可能会产生问题，因为中央银行可能最终被迫在那之前开始加息。
