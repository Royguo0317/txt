## “我们有更多的稳定性”：阿里巴巴联合创始人称香港在鸦片战争中失守，需要制定安全法

**“在 2019 年，当人们在街上抗议时，我实际上害怕走上街，”布鲁克林篮网队老板说。**

阿里巴巴联合创始人蔡崇信曾表示，香港需要一部《国安法》，因为它曾经是一个殖民地，并在 1800 年代输给了英国。这家中共国科技巨头的执行副主席周二还告诉 CNBC （美国全国广播公司商业频道）的 Squawk Box，在 2019 年的抗议活动中他害怕走在街上。

[!\[\]()!\[\](https://spark.adobe.com/page/ulf4EnEa2lUA2/images/a83c55f7-b9b8-4ea2-8f10-476f2448919a.jpg?asset_id=141d26a1-2a1b-4577-bc7e-0b6b1ebb66c6&amp;img_etag=%226229c8703c1f50439f30a7b0912ed95b%22&amp;size=1024)](https://spark.adobe.com/page/ulf4EnEa2lUA2/images/a83c55f7-b9b8-4ea2-8f10-476f2448919a.jpg?asset_id=141d26a1-2a1b-4577-bc7e-0b6b1ebb66c6&amp;img_etag=%226229c8703c1f50439f30a7b0912ed95b%22&amp;size=1024)阿里巴巴董事局副主席蔡崇信， Cody Glenn摄

蔡说，这座城市是少数没有安全法的地方之一，北京在历史上因外国势力瓜分这个国家而“伤痕累累”：“这是反对煽动的……这些都是不允许的。你知道为什么？因为香港曾经是一个殖民地，你知道，几百年前，中国因为鸦片战争把香港输给了英国人。英国人想向中国出售鸦片，经过了一些战争，中国不得不放弃香港。”

鸦片战争是 1839 年至 1842 年间英国和中国清朝之间因在广州查获非法鸦片货物而引发的一系列军事对抗。由此产生的《南京条约》将香港岛割让给大英帝国。

**“人身受到威胁”**

这位布鲁克林篮网队的老板表示，尽管他在台湾长大，但在 2019 年民主抗议和动乱期间，他因为讲普通话成为目标：“我实际上害怕走上街。你知道为什么？因为这些抗议者……所以，我实际上感到人身受到威胁，是这样的。所以，我认为现在我们有了更多的稳定性。香港会好起来的。你知道为什么，因为这是一个自由市场经济，”蔡说。

他补充说，这座城市已经“稳定”，税率低，“非常适合资本的形成”。

在经历了一年的民主抗议和动荡之后，北京于2020年6月绕过了地方立法机构，直接将《国家安全法》纳入了香港的小型宪法。它将颠覆、分裂国家、外国干涉和恐怖主义行为定为刑事犯罪，这些行为的广义定义包括破坏公共交通和其他基础设施。此举赋予警方广泛的新权力，令民主人士、民间社会团体和贸易伙伴感到震惊，因为此类法律已被广泛用于压制和惩罚中共国的持不同政见者。

**马云“很好”**

当被问及联合创始人亿万富翁马云的下落时，蔡说他（马云）已经开始学习画画了：“就杰克马而言，他很好。他现在很低调。我每天都和他通话。我实际上刚刚给他发了消息，我们有自己的消息传递平台，他实际上做得非常非常好。他把绘画当作一种爱好。其实还不错，等会儿我可以给你看几张他的照片。”
