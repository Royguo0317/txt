# **为什么新冠肺炎疫情爆发后，中共国国家福利领取人突然减少了15万？北京因“掩盖病毒死亡人数”引起新的争议**

中共国政府的数据令人不安。其表明，在疫情爆发后不久，湖北省（包含武汉市）领取国家补助的老年人数大幅下降。

官方数据显示，去年第一季度领取补助金的人数下降了超过15万人。这加剧了人们对中共国新冠肺炎造成的死亡人数远高于政府承认人数的担忧。

根据中共国当局的说法，在这个有14亿人口的国家里，只有4636人死于新冠肺炎，除了湖北这个位于中部的省份外，仅有124例死亡。这些数字引起了广泛的怀疑。

死亡数字与对武汉火葬数据的研究和来自武汉的新闻报道相冲突。其中一则现已无法查看的新闻称，两天内有一万个骨灰盒被送到了一家殡仪馆。

湖北省政府网站披露，“享受高龄津贴的老人数量”从2019年第四季度的1149153人下降到2020年第一季度的997332人，减少151821人。

这发生于疾病爆发之后，大多数科学家认为，这种疾病已于在2019年10月或11月的某个时候在武汉扩散而未被发现。国家补助金发放对象是80岁以上老人。

著名民运人士、前政府官员韩连潮说：“湖北这种高龄津贴申请者急剧减少的现象非常可疑。我担心那些失踪的老人可能都是新冠肺炎疫情的受害者。共产主义中国习惯于撒谎，甚至在自然灾害方面也是如此，所以政权面对这种流行疾病一直撒谎也是不足为奇的……执政党不尊重人的生命。”

零星的湖北数据显示，2018年和2019年底，领取养老金的人数处于类似水平，然后在疫情高峰时期显著下降。在4月至6月略微上升4141人，然后在第三季度回升至与之前几乎相同水平。

自2019年以来，武汉市的数字尚未更新。在政府承认新病毒人传人三天后，这个拥有1100万人口的城市在去年1月23日实行了封城措施。其他津贴（如伤残津贴）也没有类似的下降，因此高龄津贴下降的可能性不大可能是由于限制措施将人们留在家中所导致的。

中共国的人口占世界的六分之一。尽管中共国是疫情发源地，而且为了压制疫情爆发的消息，中共国当局犯下了骇人听闻的隐瞒罪行，当局声称疫情死亡人数比英国少25倍。

在武汉封城后的几天里，中情局警告说，中共国隐瞒了真实的感染和死亡人数，而几项研究表明，死亡人数至少比承认的高十倍。武汉殡仪馆的焚尸炉昼夜不停地工作，焚烧尸体。据估计，在最初的几个月内，每月有近3万次火化。

圣路易斯华盛顿大学医学院病理和免疫学副教授HeMai说：“这些研究都非常一致，其均表明死亡人数比官方承认的要高得多。”

这位华裔医生发表了一篇关于火葬场数据的论文，发现死亡率至少是北京宣称的十倍。

另一项对3月份撤离香港的452名湖北居民的研究表明，该地区的感染水平比官方数字高30倍。

而我们所了解的最新情况是，有越来越多的证据表明，该病毒在武汉开始传播的时间比共产独裁政权承认的时间要更早，之后于武汉市多个地点爆发，并非像最初声称那样在海鲜市场爆发。

中共国当局告诉世界卫生组织，它已得知2019年12月武汉有174例确诊的新冠肺炎病例，以及在发现该病的两个月中，有90人因类似症状住院。

这些数字比之前承认的更高，意味着那时该市还有数千例轻度或未确诊的病例在流传，然而世卫组织团队获取有关这些病例以及其他数千例可能的早期病例的数据的请求被当局拒绝。

1月，美国政府提出，武汉病毒研究所的几名研究人员在2019年秋季生病，并出现了新冠肺炎类似症状。

被北京审查的世卫组织团队遭受了指责。团队以“极不可能”为由拒绝了实验室泄露的建议，并推动中共国的说法，即病毒可能在境外出现，并通过冷冻食品进口到中共国。

西方卫生机构纷纷无视这一观点。世卫组织一位发言人说，对病毒来源的调查预计将在本周发布临时报告。
