参议院周三晚间一致通过了一项法案，要求拜登政府和国家情报总监解密有关COVID-19起源的情报。

“美国人民有权知道COVID-19的起源，”密苏里州共和党参议员乔希·霍利(Josh Hawley)在该法案通过前在参议院说。“他们应该知道这场可怕的大流行病是如何肆虐全球和我们国家的，它是如何开始的，中共国在其中扮演了什么角色。”

他说，选民应该看到证据并自己做出判断，而不是政府官员对病毒可能来自哪里的不同猜测。

印第安纳州共和党参议员迈克·布劳恩(Mike Braun)问道:“无论是在医疗保健、国家安全、COVID-19等问题上，谁不同意透明度?”“阳光下揭示了一切。”

布劳恩指出，就连在两届总统任期内领导美国应对冠状病毒的安东尼·福奇(Anthony Fauci)博士也呼吁对病毒的起源保持透明度。

在投票前的一份声明中，拜登说，情报官员正在权衡两种可能的情况，一种是病毒通过接触受感染的动物传染给人，另一种是病毒意外从中共国武汉的一个实验室泄漏。

所谓的“实验室泄露论”最初被川普政府的批评者嘲笑为阴谋论，并在今年2月至周三期间在Facebook上受到审查。但在缺乏证据表明新冠病毒有其他来源的情况下，这种理论在最近几天获得了支持。
