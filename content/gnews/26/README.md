加拿大反对党本周要求加拿大公共卫生局披露解雇温尼伯传染病实验室两名科学家的细节，以及该实验室与中共国武汉实验室的联系。而这一要求早就应该得到满足。根据《布莱克洛克记者》（Blacklock’s Reporter）对国会委员会的报道，加拿大人还应该得到自由党政府在中共病毒大流行期间采取的其他有问题的行动的答案。

5月21日，《布莱克洛克记者》报道了对公共卫生局和温尼伯的国家微生物实验室的监督异常有限的问题。2014-2019年的评估报告没有提到加拿大皇家骑警将两名科学家从该国家实验室设施移除。5月10日，保守党议员迈克尔·庄（Michael Chong）在下议院加中关系特别委员会的听证会上指出了这一点。他说，他统计了其中一位科学家在2017年至2018年期间五次访问中共国。他说，“其中一次是去中共国武汉新建立的四级病毒学实验室培训科学家和技术人员。人们正在提出问题。这不是网络上所谓的阴谋论推动的某些灰暗故事。这些都是一些有威望的人提出的非常真实的问题。而作为加拿大议员，我们的问题是：温尼伯的国家微生物实验室在与武汉病毒学研究所的合作中扮演了什么角色？”

公共服务和采购部副部长比尔·马修斯（Bill Matthews）6月2日告诉下议院政府运营委员会另外一个与中共病毒疫情有关的问题。他的部门，而不是公共卫生局，首先要求“向SNC-Lavalin集团公司订购了1.5亿美元的野外医院”。他说，该部门在2020年4月大流行的最初几周自行决定将合同交给SNC-Lavalin集团公司，而不是通过公开竞标。

保守党议员凯莉·麦考利（Kelly McCauley）问道，“这是谁的决定？我们被告知，因为紧急的缘故，这个项目必须由SNC-Lavalin集团公司独家承包。在确定是紧急项目之前，没有任何一个省向他们提出要求。那么，是谁做出的决定下了这个订单呢？”

根据内部备忘录显示，该部门说，因为没有时间通知其他联邦承包商的缘故，它向SNC-Lavalin集团公司发出了紧急订单。该部门在10月12日的备忘录中说，“由于中共疫情大流行造成的需求的紧迫性，没有发出公开招标。”而实际上，这些野外医院并没有被使用，并且目前已经不需要了。

保守党对自由党政府在中共疫情大流行期间支出一直保持关注，并于2020年10月通过一项动议，强制要求公开所有关于中共疫情大流行管理的“备忘录、电子邮件、文件、笔记和其他记录”。到2021年5月25日为止，下议院卫生委员会被告知，枢密院办公室只公布了8000份左右的记录，仍有数十万份没有公布。新民主党议员唐·戴维斯（Don Davies）说，“枢密院书记以书面形式向本委员会承认，总计共有超过一百万份文件。而到目前为止，已经过去七个月了，本委员会只收到了大约8000份。”

公布的8,000份文件显示了一些令人瞠目结舌的细节，披露了为获得大额合同而进行的游说工作。根据《布莱克洛克记者》的说法，这些文件包括了一个前自由党议员的公司获得了一份价值2.37亿美元的合同（在蒙特利尔建造一个疫苗工厂）；蒙特利尔的一个承包商获得了一份价值2.825亿美元的检测不合格的设备的合同；指示各机构隐瞒供应短缺，以及内阁故意将过期的医疗用品运往各省。

下议院政府运营委员会命令枢密院书记在6月7日出面解释延误情况。戴维斯，“政府是在故意拖延提供文件。任何有理智的人都不可能得出其他结论”。
