## 【香港风云】6: 铜锣湾书店
` 纽约香草山农场` [轉載自GNews](https://gnews.org/zh-hans/592032/)

作者：香草山文艺组 Tiffany的早餐 校对/发稿：飞虹

![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/20165916/logo-3.jpg)



摘要：铜锣湾书店因为出版《习近平的六个女人》，遭遇了多名股东和经营者相继失踪。中共向澳大利亚投诚的间谍王立强透露，他亲自参与了铜锣湾书店经营人员的绑架活动。

2013年10月底，因流亡海外的作家余杰筹划在姚文田的晨钟书局出版新书《中国教父习近平》，CCP的国安部门把姚文田诱骗到深圳拘捕，2014年5月宣判走私工业原料罪名成立，判处时年73岁的姚文田监禁10年及罚款25万元人民币。姚文田案让香港文化界感受到了CCP的手段。而之后发生的铜锣湾书店案件则更加令港人震惊。言论自由在香港彻底成为历史。

香港港岛铜锣湾骆克道，崇光百货边的临街，蓝底白字的招牌写着“銅鑼灣書店”，又有指向标牌提示书店在楼上。铜锣湾书店营业面积大约30平方米，小店也承受不起地面的高昂租金。如此不起眼的一个书店，过去曾经是大陆游客常来的一家网红店。书店的特色是出售不少大陆的政治禁书，对大陆好奇却缺乏信息渠道的游客，铜锣湾书店绝对是他们睁眼看世界的一扇窗口。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/26012840/Causeway_Bay_Bookstore_exterior-scaled.jpg)铜锣湾书店 图片来源：维基百科
维基上列出了书店销售的一些禁书：

《习近平与他的情人们》
《总书记的8段情缘》
《彭麻麻暗斗宋贵妃》
《老江气杀习大大》
《软禁江泽民》
… …

仅凭书名不难猜测这些书的品位和可信度参差不齐，也不太会对CCP的统治高层产生多少实质的影响。然而，也许正是铜锣湾书店计划出版的一本新书《习近平的六个女人》或《习近平的情人》，触发了之后的一系列事件，甚至从某种程度上，导致了之后的送中条例政策的出台，以及悲壮的反送中运动，促使国际社会反思对华关系，并最终经过一系列事件葬送了CCP本身。

1994年，香港人林荣基创办铜锣湾书店。2012年，桂民海创办巨流传媒有限公司，股东为桂民海、吕波、《大公报》专栏作家蔡嘉苹(笔名舒非，书店经营者李波的妻子)。桂民海，宁波人，1985年毕业于北京大学历史系，任人民教育出版社编辑。2014年，巨流传媒收购铜锣湾书店，林荣基留任店长。

2019年11月23日，CCP驻港台的特务王立强向澳大利亚政府的澳大利亚安全情报组织(ASIO, Australian Security Intelligence Organization)投诚，惊动全球舆论。在王立强的供述中，包括承认他自己亲自参与策划绑架李波，并负责谈判和后继任务。而绑架李波的原因就是因为令中共习近平不快的书《习近平和他的六个女人》。

2015年10月开始，书店的经营人员陆续走失。总经理吕波到大陆“去东北治疗腰痛”失联，桂民海在泰国芭提雅被人带走，业务经理张志平在深圳被人带走，店长林荣基在深圳过关时被便衣带走。四人失踪后，书店经营者李波接受BBC匿名采访，称自己因经营大陆政治题材出版，近年已不敢返回大陆。李波，为持英国护照的香港公民，妹妹是当年佳艺电视当家花旦“大眼妹”李通明，和米雪、文雪儿合称佳视三宝。

12月30日上午，六名特工将李波从铜锣湾书店的柴湾书库直接带往中国大陆。李波和夫人素来用粤语交流，而当日李波夫人几次接到李波用普通话报平安的电话，说他要配合调查。她注意到号码显示为深圳，敏感地感到他可能已被威胁，问他是否在深圳，是否可以和身旁的人通话，随即通信断掉。李波夫人在家找到了李波的回乡证，而香港警方也表示并没有李波的离港记录。显然，李波并非走正常的通道，而是被绑架离港入境大陆，动用了特殊的通道。

2016年2月29日，李波在媒体被认罪、被辟谣。他宣称自己去大陆是协助调查，并非被绑架。3月24日，李波被大陆移交香港。

铜锣湾书店事件震惊香港和国际社会。香港人终于意识到，大陆执法的黑手已经可以越过香港的司法体系，直接从香港抓人。香港人不仅没有得到民主的进步，连过去的法治基础也被严重地侵袭了。铜锣湾书店事件让同业经营者噤若寒蝉。类似在香港经营此类出版物的同行们纷纷停止或者减少此类业务。铜锣湾书店事件实际上起到了杀一儆百、压制香港出版业的效果。

CCP早已不满足于铜锣湾书店复杂的抓人方式。为了更方便地在光天化日之下，在香港大规模地抓人，CCP在香港大兴土木，并引入《逃犯条例》，引发了香港历史上最大规模的抗议活动——2019年反送中运动，导致CCP终于对香港大开杀戒，让每一个香港人都深切感知到生命的威胁。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/26015433/Causeway_Bay_Books_at_Zhongshan_Dist._Taipei_City_Taiwan_-_Entrance_1-scaled.jpg)铜锣湾书店台北新址 图片来源：wiki
2020年4月25日，林荣基重新筹资数百万新台币，在台北重开铜锣湾书店。2020年1月11日，借助香港运动给台湾带来的巨大震撼，蔡英文在选战中击败CCP的傀儡韩国瑜，并于5月20日宣誓就职。旋即在几天后的5月29日造访书店，发表讲话，感谢林荣基对香港民主人权的支持和台湾的欢迎。她还写便笺纸条：自由的台湾撑住香港的自由。
