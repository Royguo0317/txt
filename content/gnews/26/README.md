如果具体探究Bespoke和Made-to-Measure，会发现有最主要的5个不同点：

1. 西装之本：版型


Bespoke：依照每位客人自己身形定制西装版型，独一无二 。

MTM：以一个通用的西装版型为模版，在它的基础上根据客人的尺寸修改调整。

2. 顾客与裁缝的沟通接触

Bespoke：通常会由裁缝亲自丈量尺寸和制作模版。像郭先生那样级别的客人，裁缝可能会上门服务。

MTM：会在客人到店后，由经过培训的高级销售人员或者经理进行接待和丈量尺寸，裁缝有时候也会一起服务。

3. 挑选面料与Mills的种类

Mills是指纺织作坊，每个纺织作坊经常制作的三大面料有羊毛（包括羊毛、羊羔毛、美丽诺、羊驼和羊绒等），棉质/麻织品和高级纤维类（如桑蚕丝）等。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/26025913/1-233.png)
羊毛主要是按照纱支数来分级的，常见的有60、80、100、120，代表的是羊毛纱线的粗细，支数越高，说明毛纱越精细，对工艺和设备的要求就越高，价格越高，100支的比80支的纱支要细。郭先生选择的面料一般都是200支纱的。目前最精细的纱支代表面料是英国顶级品牌Scabal的Super 250，也就是郭先生视频中放的图片的品牌。优质材料可达几千上万美金一码，再有金线的加入和其他的细节处理，一套用掉4/5码面料的西装，标价可达16万美金，就是这样计算出来的。

Bespoke 选择性很多。一般为客人准备来自8～10个精选出的世界上最顶级作坊的材质。这些作坊多见于英国和意大利。一般每个作坊都会生产刚才列举的三大面料。有不同的颜色、式样（如花纹、条纹、格纹），以及每一季新增加的材质，单单一个作坊的面料簿就可以有上百种选择，比如郭先生视频中我们看到的那一本本面料样本。8～10个纺织作坊提供的选择有时更会达到千种。另外，定制不仅局限于西装，这些品牌也会提供夹克、裤装、外套的定制，面料簿也是单独成册供客人选择。

MTM 一般提供1～2个作坊的材质，这些作坊可能是该品牌自己拥有的或者一直合作的，会有几十上百种的选择。

4. 设计和个性化服务

除了面料、颜色、式样的选择，西装也有很多细节需要注意，包括内衬、纽扣、锁眼、口袋、褶、下摆开叉的选择式样等

Bespoke：任君选择，凭己喜好。

MTM：在提前准备好的多种“套餐”中挑选。

5. 成衣制作、调整和完成

Bespoke：客人一般要和专属裁缝见面3～5次，进行试穿和尺寸再测量，确保精准和满意度。平均来说，一套西装需要前后1～3个月至少70个小时的专属定制才能完成。

MTM：会在完成前试穿修改一次，中间穿插一次对半成品的试穿。

通过以上的介绍和分析，我们可以看出Brioni的品牌魅力以及定制西装上极为考究的特点。这些同时也反映了G-Fashion在质量上追求完美的宗旨：唯真不破。
