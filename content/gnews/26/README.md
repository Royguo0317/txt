## 与狼共舞 中共国生产的病毒抗原自测试剂盒竟热销德国
` JPHIMALAYA` [轉載自GNews](https://gnews.org/zh-hans/961032/)

【日本东京方舟农场】作者：青衣 素材采编：武汉老温 校对：文小律

德国连锁超市巨头奥乐齐（Aldi）和历德（Lidl）星期六（3月6日）正式开始销售自助式快速冠状病毒检测试剂盒。德国官方称希望通过加快疫苗接种和扩大快速检测范围，帮助德国社会逐步走出封锁状态。

据中新社报道，奥乐齐选择在实体门店出售日前已获得德国官方批准的快速检测试剂盒，而历德则在其网站上在线销售。两家出售的试剂盒都是基于抗原检测技术，价格在21.99欧元至24.99欧元不等。消费者购买后可自行在家采集鼻拭子，并于15到20分钟后获得检测结果。德国最大的药妆连锁超市DM和各大药房也将在数日内上架自助快检试剂盒。

此前，德国总理默克尔与16个州的州长开会决定，自3月开始加强民众自测。德国政府认为，只有快速检测更普及并提高检测量，国民才能更早地恢复正常生活。德国政府预估每月进行3000万至4500万次检测。为此，德国卫生部也决定，民众每周可以在卫生机构进行一次快速检测，据称每次仅需支付1欧元的费用。

根据媒体报道，德国政府部门目前批准了多个制造商的自检产品，其中中国占多数，还有个别欧美企业的产品也是由中国企业代工的。据称这些产品的检测准确率在95%以上。

中共冠状病毒自测盒在德国面市后销售火爆，很多商家几分钟就卖断货。目前自测盒缺货问题在德国引发关注，各州正在与包括中国在内的制造商进行紧急沟通。有媒体预测，在“口罩潮”后，中共的冠状病毒自检盒也将“如潮水般涌入德国”。

另据欧联网援引欧联通讯社报道，当地时间2月18日，为了解决欧盟27个成员国抗原检测混乱问题，欧盟列出一份各成员国均认可的快速抗原测试产品清单，其中包括几款中国快速抗原检测试剂，如乐普(北京)医疗器械股份有限公司、浙江东方基因生物制品股份有限公司、厦门市波生生物技术有限公司、中翰盛泰生物技术股份有限公司、杭州隆基生物技术有限公司等。

中共国媒体也发出报道称“8300万德国民众将全民进行中共冠状病毒抗原检测，IVD（体外诊断医疗器材）企业迎来绝好‘出海’机会！“ 首都医科大学附属北京天坛医院实验诊断中心康熙雄教授表示，国际上不少指南都认为中共冠状病毒抗原检测是核酸检测的一个重要补充，虽然它的敏感性理论上可能比核酸检测要低，但它的优势就是快，15-20分钟就能拿到结果。

目前中共国国内有多家企业还开发了国产中共冠状病毒抗原检测试剂盒。比如万孚生物新型冠状病毒（2019-nCoV）抗原检测试剂盒（胶体金法）、北京金沃夫生物工程科技有限公司的新型冠状病毒（2019-nCoV）抗原检测试剂盒（乳胶法）、深圳华大因源医药科技有限公司的新型冠状病毒(2019-nCoV) 抗原检测试剂盒（荧光免疫层析法）、深圳市中科先见医疗科技有限公司研制的新冠抗原（ SARS-CoV-2）检测试剂盒（胶体金法）等等。

中共病毒爆发以来，不仅造成了全世界过亿人染病，死亡人数已超过260万，全球经济也因此遭受了重大打击。而中共国官方却向国内民众宣传“全球一片糟，风景这边独好“，一方面用假的疫情数据和经济数据欺骗民众，另一方面借疫情大肆发展其所谓的疫苗经济，其邪恶本质昭然若揭。之前不断有媒体爆出，中共的核酸试剂准确率不足3成，出现大量假阳性病例。而如今以德国为首的欧盟国家竟然还要选择购买中共国生产的病毒快速自测盒，这就如同向纵火犯购买劣质灭火器一般，实在让人匪夷所思，以前在中共国有个常见的事情，就是一些不法汽修店在附近路上撒上铁钉之类的利器，车一经过就爆胎，然后被迫去汽修店修车。像这样的不法商仅仅只是为图财，而中共利用病毒荼毒全球不仅图财还害命，世界何时才能清醒地认识到中共的本质，不要再与狼共舞。
