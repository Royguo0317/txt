# **时隔一年，中共一边庆祝武汉封城，一边与病毒大爆发战斗**

周六，中共在“英雄”城市武汉——中共病毒的发源地，开始提前庆祝所谓的对几百万人的成功封锁。与此同时，中共病毒在一些大城市再度猖獗，其中包括北京、上海和香港。

在东北至少就有11个地区实施了事实上的封锁，这还不包括上海和香港在内——距中共在武汉组织了一个大型的，有成千上万人参加的庆祝新年集会，大概两周后，大爆发开始。从照片中可以看到密密麻麻挤在一起的人群陶醉狂欢的场面。公共卫生专家建议，在病毒仍然流行的国家，人群不要大量聚集。然而中共还是举行了这场聚会，并且自鸣得意地把它拿来跟纽约等城市取消类似活动相对比。

[!\[\]()!\[\](https://spark.adobe.com/page/MXTpaA8RvVcwM/images/bd2f5379-af23-418a-8f09-3295ca389a7d.jpg?asset_id=82dc688a-355b-45ba-b40d-a7a4faa344e3&amp;img_etag=%22985eacdb6108d2101241cec8c3afcea8%22&amp;size=2560)](https://spark.adobe.com/page/MXTpaA8RvVcwM/images/bd2f5379-af23-418a-8f09-3295ca389a7d.jpg?asset_id=82dc688a-355b-45ba-b40d-a7a4faa344e3&amp;img_etag=%22985eacdb6108d2101241cec8c3afcea8%22&amp;size=1024)

2020年1月，中共允许几百万人离开武汉，到世界各地旅行。——后来的证据揭示，他们已经掌握了信息，表明武汉有传染性疾病传播。尽管有明显的病毒社区传播迹象，中共政府在封城之前还是举办了世界上最大的，有13万人参加的家庭宴会。人们在宴会上共享盘子和餐具。武汉市长周先旺当时承认有500万人——几乎是武汉市人口的一半——在实施封城前离开。

从武汉泄露出的视频片段清楚地显示了封城的残酷。人们看到官方把门焊死，把居民困在家里。多份报告显示，对于有特殊需求的人——有些是孩子，他们死在了家里，因为除非是感染了新冠病毒，他们不被允许(离家去接受)其他类疾病的医疗救护。有很多人自杀。

[!\[\]()!\[\](https://spark.adobe.com/page/MXTpaA8RvVcwM/images/3a314551-b0a6-4293-af56-a58f3d096fc5.jpg?asset_id=dbbf50d0-398f-442f-b964-d4ce5d67770a&amp;img_etag=%22a3d21263ef959317f2e41ea6c8188aad%22&amp;size=2560)](https://spark.adobe.com/page/MXTpaA8RvVcwM/images/3a314551-b0a6-4293-af56-a58f3d096fc5.jpg?asset_id=dbbf50d0-398f-442f-b964-d4ce5d67770a&amp;img_etag=%22a3d21263ef959317f2e41ea6c8188aad%22&amp;size=1024)

中共政府这周末在武汉的电影院上映了一个宣传影片，目前还在公演，来庆祝所谓的武汉封城胜利。据《南华早报》报道，“由30个电影制作者打造的影片《武汉日夜》，描绘了该市1100万居民，医务人员和一线工作者的苦难和牺牲。” 该报还说，中共正在制作至少3个纪录片，来宣传武汉应对新冠病毒爆发的措施是有效的和成功的这样的观点。

(英国)南安普敦大学的WorldPop人口分布图小组在3月发表的一项研究发现，如果中共，尤其是武汉官员处理新冠病毒爆发得当的话，他们或许能在国内阻止多达95%的新冠病毒病例。直到2020年1月末，中共官方还在坚称，没有证据表明病毒会人传人；尽管台湾的官方医疗机构在2019年12月末就已经开始对患者使用隔离的办法。

“我们想用动态的画面来记录跟新冠病毒（中共病毒）疫情战斗的历程。一些细节像重症监护、焦急的等待、心碎的离别和满怀希望的再生，它们可能会触动观众的心弦。”导演曹金玲对中共国媒体说到。

中共的国家宣传喉舌《环球时报》对这部纪录片的首映做出了呼应。该报对几个武汉“英雄”做了广泛报道，这些“英雄”充满感激地赞扬了政府的封锁措施。《环球时报》声称，武汉市民一开始对新冠病毒爆发很“困惑”，但亲眼目睹在世界其它地方尤其是美国的所谓的失败后，他们最终狂热地支持政府。

“当武汉在4月回到忙忙碌碌的日常生活的时候，西方国家一个接一个地去面对更严重的新冠病毒攻击。很多在一开始对政府的封城决定疑惑不解的武汉人，这时变成了封城的坚定支持者，” 《环球时报》宣称。“在现代社会，没有什么比封城决定更糟的了，很多当地人对此不理解。但在看到武汉从新冠病毒的打击中迅速反弹，而此时一些西方国家正一个个陷入新冠病毒的泥潭，这些曾经感到困惑的市民，很快变成了坚定支持严格管控手段的人。”

《环球时报》声称，和其他国家相比，中共国是“从新冠病毒疫情中反弹最快，最有力的。” “比如美国，它已经经历了几轮的封锁，” 但中共病毒还在持续发生。

“那些西方国家没有权利抨击武汉，他们自己被每天冲击新高的新冠病毒纪录所拖累。什么是他们一直谈论的‘人权’？让人活着就是人权最重要的部分。”引述一位“陈”姓武汉居民的话这样说到。

《环球时报》的这篇报道没有提到，疫情当下正在全国范围内爆发、肆虐。在2021年1月中旬，《环球时报》自己报道说，在中共的东北，至少有11个地区实施了“事实上的”封锁。环绕北京城的河北省与东北的黑龙江和吉林一起，正遭受着全国最严重的病毒爆发潮。北京本身正在对居民的活动实施极度的限制。在其他地方，如北京西边的山西省，官方已经开始为另一轮的新冠病毒激增做准备。

这周，上海开始对人口密集的社区实施封锁，以阻止病毒的进一步传播。之前作为一个自治城市的香港，看起来也在开始准备在几个社区采取重大封锁行动，这将围困几万居民，也是自新冠病毒爆发以来在香港实施的最大封锁。
