每四年的大选就是一场大戏，只是大多数的年份平淡无奇，只要几个关键的摇摆州出了大致的结果，基本上胜选者就会在出来演讲，败选者也会有风度的承认败选，并且打电话祝贺新总统当选从而体面的收场。美国和西方文化中非常鄙视一种人——Sore-loser，就是输了还不认，拼死抵赖，撒泼打滚的失败者。好多美国人可能以为一场大选最多就是出一个sore-loser，可没成想等来的居然是这么荒诞的一个结果。民主党的大规模造假已经让大选乃至美国陷入了混乱，那么川普总统手里能有什么牌可以打呢？

首先就是基于《美国宪法第三条》设立的联邦大法院。二十年前第一次看美国大选，当时的小布什和戈尔在关键摇摆州佛罗里达厮杀的难舍难分，最终一路奔向了高院。高院不得不顶着多方的抨击裁定了2000年的大选。如果说那场大选争议很大的话是因为两党候选人的差距太小，在关键的佛州仅仅2000票之差，在重新计票的时候两党产生了分歧，从而引入了高法的裁决。可以说当时高法的任何裁决都会引来争议，因为任何一个党的支持者都很难接受由于高院的判决从而以微弱的劣势输掉大选，高法无形中成了改变天平的那一粒稻谷。

在本次美国大选之前，当舆论、媒体、甚至包括两党候选人都准确预言了会有大规模造假的时候，高院的作用在提名和任命巴雷特大法官时被再一次推上了前台。鉴于美国国会、总统、高法三权分立体系分权和制衡的特点，高法介入总统选举是历来为人诟病的，且高法本身也十分克制。在9月26日川普提名巴雷特大法官的时候遭到了民主党人的强烈抵制，因为他们知道最终很可能走向高法。美国民主的基石就是天赋人权和平权，最具体的体现就是《第一宪法修正案》里规定的各种自由，以及一人一票的选举权利。所以民主党此次大选的舞弊无形中正在撼动这块基石，懂美国法律的人知道，此举比叛国罪还重。在川普总统大幅领先的六个摇摆州，先后在大选当天关停了计票，这一行为本身就是违宪。为了配合民主党舞弊，社交媒体上大规模的封杀支持川普爆料真相的言论，甚至连川普总统本人的若干条推特都被审核，这已经是严重的破坏了《第一修正案》。这一桩桩罪行，哪一个没有证据？哪一个没有证人？哪一个没有受害者？如此明显的案子，我相信对于美国高院不难做出相应的违宪判决。

**下面开开脑洞，想想川普总统手里到底有什么样的证据和证人。**

从Hunter Biden的硬盘一段语音上能听到Hunter明知道何志平是中共间谍还去去纽约南区检察院作伪证，那这段语音会不会是用FISA法案获取的情报。如果是的话，那从至少从2017年11月何志平被捕时就已经开始了监听。从Hunter这条线很容易就引到Joe Biden和他的同伙，而且监听的时间也不会短。这期间中共如果跟Biden的团队勾兑干扰美国大选，应该都会被记录下来。路德节目在9月24日报出来硬盘事件，可是距离10月14日《纽约邮报》的报道整整三个星期，这是为什么？如果是给听众科普的话，不需要这么久，一个星期应该差不多了。我猜想路德是报给中共听的，好让中共跟Joe Biden窜通来重新布局大选造假。因为随着硬盘事件的爆发，一定大幅影响Biden在关键摇摆州的选情以及最后两周的竞选活动，那这样造假需要的工作量就会激增，所以三周时间是让他们充分准备，也好通过FISA取得更多的直接证据。从大选日《战斗室》节目可以感受到班农先生对于若干个摇摆州相继停止计票并不惊讶，而且还有点期待。用班农先生的话说就是 “他们开始了真正的游戏” 。相信班农先生早就有关于民主党人何时、何地、如何造假，以及参与人员的情报，等的就是这张底牌。

另外会不会参与造假的人站出来作证？我觉得可能性很大。第一，这么大规模的造假不可能都是民主党的头头脑脑亲力亲为的完成。这个过程需要打印、填写、运输、邮寄、统计选票等等诸多环节。民主党就能保证这诸多环节都是自己最可靠的人来完成的么？我相信不会。有正义的人站出来作证就可能会成为第二个Tony Bobulinski. 第二，会不会提前安插线人做卧底。这么巨大的造假工程也需要跨市、跨州协同作业，在两党势力犬牙交错的地界，共和党安插线人在其中也是可能的。第三，已经在社交媒体上有大量的选民反应了选举舞弊的情况，而且有好多人都已经收集好了现场的证据且愿意作证，相信在这一群体和证据数量将是庞大的。

到写稿时为止，在宾州、乔治亚、密歇根和内华达四周，川普总统的竞选团队已经提起了诉讼。在最为诡异的威斯康辛州提出了重新计票的要求。这么快速的反应且在不同的州措施不同，可不可以理解为川普总统早就已经了解民主党在各州作弊的细节，所以才有了不一样的反击手段？在推特上看到两党都在号召统计每一张合法选票，也就是说Biden并不怕重新计票。那会不会有一种方法来快速区分真假选票呢？负责大选的机构是美国国土安全部（DHS），昨天看到一种说法是国土安全部在选票上放有非放射性同位素水印，如果CCP和Biden不知道这一机密的话，那他们打印出来的选票就很好区分了。而且即使Biden反诉川普总统造假的话也没有证据了，毕竟真假选票是有了一个极容易区分的标准。

总而言之，最高法院一定会成为本次大选的关键，也将是捍卫美式民主的最后一道铁闸。但是不排除川普总统手中已经掌握了大量的语音、人证和物证可以在去高法的路上直接KO掉Biden。同时更重要的是可以趁机清理掉若干个州的沼泽势力，因为中共的渗透不单是在联邦层面，同时也到达了各个州和地方政府。如果不清理掉这些人，即使川普当选，他们也会对川普总统下一步揭开疫情真相、控制疫情和灭共上带来很大阻力。如果我的猜测是真的，那这是一盘妙棋。先用三块硬盘定点爆破众议院的佩洛西、FBI的Wray还有司法部，再用硬盘内容引爆Biden叛国窝案，最后用大选将CCP、Biden和州一级的沼泽势力连根拔起，从而启动Rico法案。
