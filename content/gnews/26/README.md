**西班牙2021年2月27日电/西喜社——**据thejournal.ie周六报道，爱尔兰已经成为第二十个签署国际跨党派立法者联盟的国家，这些立法者正试图在全球贸易、安全和人权问题上对中共国做出协调的反应。

三名参议员和一名道明会议员加入了各国议会中共国问题联盟（IPAC），以争取爱尔兰政府对中共国采取更强硬的立场。

各国议会中共国问题联盟（IPAC）致力于改革民主国家的对华态度，拥有来自北美、欧洲和澳大利亚各政党的200多名议员。

该联盟成立于2020年6月，其使命是站出来反对 “中华人民共和国目前的行为和未来的野心所带来的挑战”。

“作为民选议员，我们有责任不仅在国内，而且在国外维护民主事业。中共国政府对香港民主的压制是对国际法不可接受的违反，是对基于规则的国际秩序的明显挑战。”参议员拜恩说。

“爱尔兰必须发挥自己的作用，追究中共国政府对这些侵权行为的责任。我们只有站在一起，认识到这个挑战太重要了，不能因为国家或党派的分歧而分裂。”
