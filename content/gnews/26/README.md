今天是1月18日，又是一个值得记录的日子。

2020年的今天，中共病毒源头湖北省武汉市，政府倡导、社区牵头、全民总动员，搞了一场举国奔小康的大型庆祝活动——万家宴。这场数万人参加的集体聚餐由一个主会场、九个分会场组成，为配合奔小康宣传的政治秀，武汉可以说是做到了极致。今天回看去年的视频，难免让人唏嘘。

![]()![](https://gnews.org/wp-content/uploads/2021/01/WhatsApp_Image_2021-01-18_at_17.59.58.jpeg)主持人小月，图片来自网络

视频中，主持人小月隆重登场，口若悬河、眉飞色舞地用修饰过头的播音腔盛赞百步亭社区“人人都是金凤凰”，更称颂这次活动是“展邻里亲情、看传世佳风”，“奋斗新时代、圆梦小康年”。

只可惜，仅仅三天之后，武汉全境封禁，百步亭里的金凤凰们无法飞出社区展示他们美丽的羽毛，邻里亲情迅速被突如其来的疫情砸得粉碎，传世佳风在人人自危的现实面前成了一地鸡毛。

![]()![](https://gnews.org/wp-content/uploads/2021/01/unnamed-14.jpg)图片来源：中新网

1月21日，武汉封城，武汉市百步亭社区，从张灯结彩歌颂盛世到沦为地狱般的封城封门，仅仅时隔三天，完成了从可歌到可泣的乐极生悲切换模式。

今天回看，无论是1月18日还是1月21日，都让人无限感慨。因为12月30日，李文亮医生等8名吹哨人被约谈、被训诫的事情已经发生两个多星期，而且就发生在他们眼皮底下，而各大医院不明肺炎感染者正在迅速飙升、人满为患。什么样的盲目自信能让执政者完全枉顾专业人士的告诫，还搞这种假大空的政治秀？为了迎合上意不惜拿全城人民的健康去冒险？他们真的是麻木迟钝？还是为了向上爬不惜一切代价？亦或是压根他们就知道自己在干什么，他们之所以倡导这样几万人聚集的万家宴，就是为了让病毒传播？如果真是这样，多么让人毛骨悚然啊！

如果不是这样，那么他们的执政能力简直是猪都不如！如果不是这样，为什么他们封城之后，国际航班仍在运行，满城的可能携带了病毒的百姓迅速飞往世界各地！

万家宴果然不负众领导的愿望，两周之后，武汉各大医院医疗资源告急，一床难求，开始建方舱医院。百步亭社区50幢居民楼出现发热病人，这个拥有13万人口的大型社区，率先沦陷于疫情之下，宴会惊魂就此愈发不可收拾，百步亭社区彻底封禁，不出不进，人民开启自生自灭模式。自此“人人都是金凤凰”的百步亭社区居民，彻底明白了什么叫“拔毛的凤凰不如鸡”。早知道“圆梦小康年”圆的是噩梦，还不如当初不做白日梦了。

今天回看，若不是这些中共的御用官僚们有意让病毒传播，就是他们的管理能力还处在原始阶段。事实上，就连三天后的封城举措，也是出于1月19日（中共国时间20日）路德社在节目中报道了病毒来自实验室、人传人、大爆发等重要细节之后，才不得不做出的反应。

而接下来武汉的抗疫过程更像一出狗血剧。除了封门就是堵嘴，造假数据隐瞒真实疫情、半夜偷偷焚烧尸体、提供假病毒序列号让专业人士无法破解、伙同WHO向全世界散布假消息等等，导致了一年后的今天，上亿人感染、200多万人死亡的悲惨事实。

这场疫情，赤裸裸地揭示了中共的本质，这就是一群篡夺了人民权利的流氓。他们毫无底限、贼喊捉贼，完全不讲逻辑、枉顾事实。武汉的吹哨人、万家宴、载着病毒携带者的国际航班、P4实验室、华南海鲜市场等等一系列的关于病毒的事件，中共似乎完全看不见，一味到处甩锅、推脱责任。面对铁的证据，美国人指责中共是病毒源头，中共却说这是将病毒政治化，而反咬一口，说病毒来自美国，这时候他们就不说是政治化了，而是以科学之名；当闫丽梦博士将颠扑不破的科学证据亮出来之后，中共又无耻地混淆视听，胡搅蛮缠之后又装聋作哑，集体失声了。

一年之后的今天，疫情仍在继续，一年前路德社的爆料一一应验，全世界越来越多的正义之士纷纷站出来敦促中共披露病毒真相。中共国的“圆梦小康年”更是民不聊生，噩梦连连了。

万家宴两个多月后的百步亭社区，幸存者们在细雨蒙蒙的清明节里，排队领取亲人的骨灰。然而，那据说是免费派发的廉价骨灰盒里，盛放的不只是他们的亲人，而是所有遇难者的混合体——万家灰。生在中共统治下的人们，命贱如蝼蚁，生无尊严可谈，就连死了，化成的那把灰都不是自己的。

去年今日万家宴，今年荒冢连成片，全球染疫无幸免，中共依然在狂欢。就在全世界都成了疫区的时候，中共却在大会堂开起了抗疫大捷的庆功会，“带货大王”钟南山等人披红挂彩成了共和国功臣，“庆丰帝”亲授勋章，把无耻演绎到了极致。
