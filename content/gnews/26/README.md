## 韓國對抖音罰款1億8千萬韓元
`2020-07-16 07:49 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/266639/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/16074421/Picture-1-101.png)圖片來源：騰牛網
據韓國媒體報導，韓國政府以違反個人信息相關規定為由，對中共國企業字節跳動（ByteDance）旗下的抖音處以上億韓元罰款。 

在韓國國內月活躍用戶規模達到300萬名的中共國視頻平台”抖音”，因擅自收集、轉移個人信息的行為受到韓國政府的製裁。抖音在美國等國外也曾因非法收集兒童個人信息而受到罰款或調查。 

廣播通信委員會（簡稱廣通委）15日召開全體會議，就抖音違反個人信息保護法規（信息通信法）的問題進行討論後，決定對其罰款1億8千萬韓元。在去年10月國會國政的監查中，未來統合黨議員宋熙京指出，”受到青少年追捧的抖音未經同意，直接獲取用戶的SIM卡、IP地址等位置信息”。抖音從2017年開始在韓國提供服務，截至去年年底，在韓國國內收集並保管了1057多萬的個人信息。

從調查結果來看，抖音沒有將針對未滿14周歲的兒童提供服務內容包含在個人信息處理聲明中告知消費者，用戶在註冊階段只有直接輸入生日信息或只確認是否年滿14周歲以上的項目，並沒有確認用戶實際年齡的程序。另外，也沒有得到兒童法定代理人的同意。廣播通信委員會表示:”非法收集的信息超過6000件。 我們在調查過程中終止了相關賬戶。” 

此外，”抖音”還違反了將國內用戶的個人信息轉移到國外的相關規定。根據《信息通信網法》規定，信息通訊服務提供者（抖音）將國內用戶的個人信息轉移到國外時，應告知用戶，並得到用戶的同意或通過個人信息處理聲明予以公開。但是，抖音將韓國國內用戶的所有個人信息保管在美國、新加坡等地的雲端服務器，並沒有將這一事實告知國內用戶。據悉，抖音向廣通委方面的解釋竟然是”對韓國法規並沒有完全理解”。 

抖音非法收集兒童個人信息在國外也是備受關注的焦點。去年2月，美國聯邦交易委員會（FTC）以”抖音非法收集兒童個人信息”為由，對字節跳動公司處以570萬美元（約66億韓元）罰款。英國政府也於去年7月表示，認為抖音違反了歐盟一般數據保護法規(GDPR)的規定，非法收集有關兒童的個人信息，目前正在對其進行調查。

評：繼華為之後，抖音等科技網絡公司在全世界收集個人情報的工具被法治國家發現並製裁，甚至抵制。連韓國這樣的近鄰國家也開始覺醒，這讓中共伸向世界的魔爪得到遏制，說明中共的末日已經臨近。希望一直和中共曖昧不清的韓國能採取更多反制措施，看清局勢，不要被中共拉下深淵。
