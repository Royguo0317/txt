# 川普团队就选举问题在亚利桑那州、密歇根州和宾夕法尼亚州的立法机构赢得听证会

[![](https://spark.adobe.com/page/cH7Z3YlxQqf1h/images/8020d294-cf9d-4f87-8035-4da29a598ca6.jpg?asset_id=a3cd6118-562b-49b7-86e1-fd0b3b62561a&amp;img_etag=%221e7c2dae19109e8537bfae16e669411d%22&amp;size=1024)](https://spark.adobe.com/page/cH7Z3YlxQqf1h/images/8020d294-cf9d-4f87-8035-4da29a598ca6.jpg?asset_id=a3cd6118-562b-49b7-86e1-fd0b3b62561a&amp;img_etag=%221e7c2dae19109e8537bfae16e669411d%22&amp;size=1024)

听证会将 “努力给人们提供信心，确保在11月3日的选举中，所有的合法选票都会被计算，而非法选票不会被计算”。

[![](https://spark.adobe.com/page/cH7Z3YlxQqf1h/images/eb5472e5-67aa-4663-906e-9b97b1475dcb.jpg?asset_id=f7cc4fb2-e996-4054-9bc3-83dbd7566df4&amp;img_etag=%22209766efba80093633a4150c32ac0124%22&amp;size=1024)](https://spark.adobe.com/page/cH7Z3YlxQqf1h/images/eb5472e5-67aa-4663-906e-9b97b1475dcb.jpg?asset_id=f7cc4fb2-e996-4054-9bc3-83dbd7566df4&amp;img_etag=%22209766efba80093633a4150c32ac0124%22&amp;size=1024)

川普总统的竞选法律团队周二宣布，亚利桑那州、密歇根州和宾夕法尼亚州的立法机构将很快举行与选举有关的听证会。

根据一份新闻稿，这些听证会将 “努力给人们提供信心，确保在11月3日的选举中，所有的合法选票都会被计算，而非法选票不会被计算”。

该团队表示，宾夕法尼亚州参议院将于周三举行听证会，鲁迪-朱利安尼将在会上发言。参与听证会的参议员将发表5分钟的开场白，提交宣誓书指控选举舞弊的人将提供证词。

下周，亚利桑那州立法机构的听证会将在周一举行，密歇根州立法机构的听证会将在周二举行。

新闻稿说，“州议会具有独特的资格和地位，在选举人被确定之前就选举中的违规和舞弊行为举行听证会。根据美国宪法第2条第1.2款的规定，州立法机构拥有挑选选举人代表的唯一权力，这为防止选民舞弊和选举操纵提供了重要保障。”
