## 香港：英国指责中共国违反联合声明

![]()![](https://gnews.org/wp-content/uploads/2021/03/1-66.png)多米尼克·拉布（Dominic Raab）的干预标志着伦敦和北京之间的外交紧张局势升级。照片：Andy Rain / EPA

外交大臣多米尼克·拉布（Dominic Raab）说，北京有“持续不遵守1984年协议”的罪责。

在对北京试图加强对香港的控制的批评中，多米尼克·拉布（Dominic Raab）指责中共国违反了有关香港治理的法律协议。

外交大臣在外交紧张局势的严重升级时说，英国认为中共国处于中英联合声明的“持续不遵守状态”。他谴责北京决定减少公众对于选择香港领导人的作用。中共国将权力交给了亲北京的一个委员会，该委员会将任命更多的立法会成员。

拉布说，此举是在不到九个月的时间内第三次违反具有法律约束力的中英联合声明，这是“旨在骚扰和扼杀所有批评中共国政策的声音的模式的一部分”。

他说：“北京作出重大改变以限制人们参与香港选举制度的决定，明显进一步违反了具有法律约束力的中英联合声明。”

“中共国当局继续采取的行动意味着我现在必须报告，英国认为北京正处于不遵守联合声明的状态，这表明北京的承诺与其行动之间的差距越来越大。

“英国将继续支持香港人民。中共国必须按照其法律义务行事，并尊重香港的基本权利和自由。”

自从中共国在香港实施新的国家安全法以来，紧张局势一直在加剧，这个法律使得镇压抗议者变得更加容易。

中英联合声明是由中英两国于1984年签署的。伦敦坚持相信这个联合声明能够保护香港的重大自治和自由。

多米尼克·拉布得到了七国集团（G7）国家外交部长的支持。他们对香港选举制度中民主元素受到明显的侵蚀表示关注。

七国集团外长的声明说：“我们还呼吁中共国和香港当局恢复对香港政治体制的信心，并制止对那些倡导民主价值观和捍卫权利与自由的人的无端压迫。”

欧盟在周五也表达了关切。欧盟高级代表兼副主席约瑟夫•博雷尔（Josep Borrell）表示，中共国“通过违反国际承诺和香港基本法，主动摧毁了其’一国两制’的原则。”澳大利亚和新西兰也分别表达了忧虑。

但香港政府表示强烈反对质疑中共国是否愿意维持“一国两制”的言论，称其为“毫无根据的指控”。
