我们知道科沙瓦尔兹·尼亚博士的重要性是因为就在两个半月前，《纽约时报》刊登了一篇关于一名天才骗子对美国情报和军事团体大规模数百万美元欺诈的周日长篇。科沙瓦尔兹·尼亚博士是少数看起来表现不错的人之一：

与他一起工作的人说：“纳维德·科沙瓦尔兹·尼亚总是房间里最聪明的人” 。在为中央情报局（C.I.A.）、国家安全局（NSA ）和联邦调查局（F.B.I.）做网络安全和技术反情报工作时，他花了数十年时间研究多项绝密信息点。与考特尼先生（Mr. Courtney）合作几个月后，他也开始将这些点联系起来。他不喜欢这些点的导向。

科沙瓦尔兹·尼亚博士不仅具有天生的才智，而且在网络欺诈侦破和分析方面也拥有非凡的学术和实践技能。 我们知道他的资质的原因是，在他签署的支持佐治亚州申诉的声明中用了七段话列出了他的这些资历。

他的资历包括在电气和计算机工程两个不同领域的学士、硕士和博士学位。此外，他还“接受过国防情报局（DIA）、中央情报局（CIA）、国家安全局（NSA）、国土安全部情报与分析（I＆A）和麻省理工学院（MIT）的高级培训。”

专业上，科沙瓦尔兹·尼亚博士曾担任网络安全工程师。

他声明说，“我有跨度35年从事技术评估、数学建模、网络攻击模式分析和安全情报等方面的经验 ”。

*锤子和记分卡是真实的，不是骗局（正如民主党人所声称的那样），两者均用于操纵选举结果。*

*多米尼系统（Dominion）、ES＆S、Scytl和Smartmatic系统都容易受到欺诈和投票操纵的影响 — 主流媒体过去曾报道过这些漏洞。*

*在其他国家，多米尼系统已被用来“伪造选举结果”。*

*多米尼的公司结构故意被混淆来隐藏与委内瑞拉、中国和古巴的关系。*

*多米尼机器很容易被黑客入侵。*

*具有对系统的加密密钥访问权限的Dominion存储卡于2019年被盗。*

尽管科沙瓦尔博士无法访问这些机器，但是他研究了有关选举和投票结果的可用数据。根据这些信息，他得出结论。

*在宾夕法尼亚州，威斯康星州，亚利桑那州，内华达州和佐治亚州同时作出假装停止点票的决定，这是前所未有的，并显示出协调一致的努力串通来达到想要的结果。*
