# **北京对大科技公司的持续打击——腾讯被指责，蚂蚁集团负责人被迫出局**

上周五，由政府发起的对中共国科技巨头的镇压行动重新起火，阿里巴巴蚂蚁集团（Ant Group）负责人胡晓明（Simon Hu）突然辞职，（去年支付集团从阿里巴巴分离出来其上市计划被习近平和中共毫不客气地取消了）。据报道，中共国科技巨头神殿的另一成员腾讯被中共警告。

胡晓明——由阿里巴巴于2019年末聘请，在高管改组期间经营蚂蚁集团——本来带领金融集团在分拆取得成功之后，雄心勃勃地要进军西方市场，并与美国科技和金融业巨头在西方的自家后院展开竞争。对于蚂蚁集团的美国竞争对手而言，幸运的是中共已决定宁愿让充满活力的中共技术巨头牢牢掌握在自己的手中，也不愿冒险让它们积累更多的经济实力。

[!\[\]()!\[\](https://spark.adobe.com/page/1YXrE0Ri0SLAC/images/fab62f70-a5d2-4f3d-be38-e070bad15488.png?asset_id=58217a9b-7005-4a15-b379-14d2d633f339&amp;img_etag=%22da89259c4d34116f6144cce3b24d59ae%22&amp;size=2560)](https://spark.adobe.com/page/1YXrE0Ri0SLAC/images/fab62f70-a5d2-4f3d-be38-e070bad15488.png?asset_id=58217a9b-7005-4a15-b379-14d2d633f339&amp;img_etag=%22da89259c4d34116f6144cce3b24d59ae%22&amp;size=1024)

早些时候，马化腾的腾讯受到中共国反托拉斯监督机构的正式指责（中共国反托拉斯监督机构在大规模地打击高科技公司的行动中发挥了重要作用），并处以象征性罚款。据彭博社报道，现在可能会要求腾讯成立一家金融控股公司，以剥离其银行、保险和支付服务业务。

李克强总理在全国人民代表大会上承诺要加大对金融技术的监督，消除垄断，防止资本“无节制”地扩张，几天后就发生了上述的事情，这一举动将标志着中共国对遏制其科技巨头运动进行的一次重大升级。

过去六个月公布的一系列规则旨在瞄准中共国最成功的网络创业者所建立的统治地位。马云和阿里巴巴首当其冲，其子公司蚂蚁集团350亿美元的首次公开募股在最后一刻被挫败，随后阿里巴巴遭到了反托拉斯调查。

在过去六个月中，腾讯的股票跑赢了阿里巴巴，这大约是在中共取消蚂蚁集团的首次公开募股之后的这段时间——那将是中共国历史上最大的国内发行——之后阿里巴巴的创始人和董事会主席马云批评中共国的国内法规，指责它“扼杀了创新”。腾讯股价周五在香港下跌逾4％，因为很明显，中共国反托拉斯监管机构起草的新规则也将损害腾讯。拟议中的打破数字支付市场集中度并控制在线消费者贷款的规则将损害腾讯的微信支付及其更广泛的金融科技业务的前景，后者主要致力于为中共国的14亿人提供零售银行业务和消费者借贷服务。

[!\[\]()!\[\](https://spark.adobe.com/page/1YXrE0Ri0SLAC/images/2a7def98-feef-4e6b-9551-b12b038010e4.png?asset_id=37052121-3c62-45b5-abdf-1f2973a8e96b&amp;img_etag=%22b8e2b34ce9c8dc577a825f7d7df182b5%22&amp;size=2560)](https://spark.adobe.com/page/1YXrE0Ri0SLAC/images/2a7def98-feef-4e6b-9551-b12b038010e4.png?asset_id=37052121-3c62-45b5-abdf-1f2973a8e96b&amp;img_etag=%22b8e2b34ce9c8dc577a825f7d7df182b5%22&amp;size=1024)

将这些业务合并为一家新的金融控股公司，将使中共能够以监管国内其他银行（主要由国家拥有和控制）的相同方式来监管腾讯。在无限期终止蚂蚁金服的首次公开募股之后，北京迫使这家总部位于杭州的公司转变为一家金融控股公司，并接受新的资本限制，申请新执照和接受所有权的审查。彭博分析师表示，尽管新的限制可能不会立即对腾讯的盈利产生影响，但对腾讯增长前景的影响是显而易见的。

*金融科技既不是腾讯增长最快的业务，也不是最赚钱的业务，它最大程度地减少了对财务的直接影响，但事件的转变可能预示着更加严格的监管时代开始了，这与阿里巴巴命运的转变产生了强烈的呼应。*

腾讯在一份声明中表示，公司将继续调整以适应监管的变化。

腾讯在反托拉斯监管机构罚款后的一封电子邮件声明中说：“我们将继续适应监管环境的变化，我们认为这对行业有利，并将寻求确保完全合规。”该公司拒绝就金融监管事宜发表评论。

北京对国内股市的收紧显然仍在起作用，交易员们开始查寻企业债务数据，以了解市场在北京监管的重击之下如何反应。压力导致腾讯在2030年到期的2.39％美元债券的利差扩大了约9个基点。

中共国一直努力将对中共国科技企业家的支持纳入官方的共产主义思想之中，这些企业家由于外国投资的涌入而变得异常富有。权力仅次于习近平的总理李克强，来平衡中共越来越强硬的立场，他保证北京将继续支持“平台公司的创新与发展” ——只要它们遵守中共国的法律。
