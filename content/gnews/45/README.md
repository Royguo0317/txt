## 【热点播报】欧盟多国对默克尔急于和中共达成投资协议表达不满

作者：香草山写作组 人间世

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/02104531/logo.jpg)



据Politico网站1月1日报道，欧盟多国对默克尔在其任期即将结束前强势推动欧盟达成和中共的投资协议表示不满，指责默克尔和马克龙没有公正地对待欧盟其他国家。

![]()![](https://gnews.org/wp-content/uploads/2021/01/MOS0911-【热点播报】欧盟多国对默克尔急于和中共达成投资协议表达不满.png)Tobias Schwarz/AFP via Getty Images（图片来自Politico）

报道指出，德国总理默克尔在2020年最后几天推动欧盟签署中欧投资协定之后，来自意大利、波兰、比利时和西班牙的官员对此提出了尖锐批评。这些官员说，**他们感到被默克尔和欧盟委员会内部的****“德国引擎”，即委员会主席乌苏拉·冯德莱恩（Ursula von der Leyen****）和贸易部主任萨宾·韦恩德（Sabine Weyand****）（他们都是德国人），粗鲁地忽略了**。值得注意的是，协定的签署，恰好赶在德国作为欧盟理事会主席国的任期结束前最后一刻。一位欧盟外交官对德国人这种做法十分不满，他说，“**这就是欧盟在英国脱欧后的工作方式吗？英国人刚刚出局，我们已经开始怀念他们开放的、以市场为导向的工作方式了”，他担心如果德国权重过大，欧盟小国就没机会说话了**。

另一个让批评者不满的原因是，这一协议是“中共国国家主席习近平的政治胜利”。因为正值中共政府残酷镇压香港民主、新疆少数民族和报道冠状病毒大流行起源的记者时，“我们向他们发出了积极的信号”。

意大利的斯卡法罗托（Scalfarotto）指出，欧盟不应该在美国新政府上任前匆忙缔结协议。他说，“即将上任的（美国）政府没有机会表态，但它已经明确表示，更希望欧洲等待。”

意大利还对法国总统马克龙和代表欧盟轮值主席国的默克尔一起参加与习近平的视频会议，而其他欧盟领导人却没有受邀参加表示不满。“我不认为让马克龙，也就是让一个国家而不是其他26个国家的人一起参加是合理的。这是一种不寻常的形式，这也标志着我们意大利人的失败。”斯卡尔法罗托透露，意大利曾要求只让欧盟领导人参加通话。

两名外交官表示，马克龙和默克尔私下有约定。默克尔在德国担任轮值主席国期间获得缔结协议的机会，而批准和签署协议将在2022年上半年由法国担任主席国时完成。
