**港共林郑集团今（1 日）公布新一轮防疫措施，大厦若录一宗源头不明个案、或污水样本阳性，便会纳入强制检测。工作场所若出现两宗或以上确诊，亦会纳入强制检测。变相全港大厦都被纳入「指定核心区」。港共奴才张建宗又预告，未来十天内，将每日「封区」强制检测，并可能一日封多区。**
![]()![](https://gnews.org/wp-content/uploads/2021/02/jpg-藉口1.jpg)图片：立场新闻
大家记得去年9月份港共林郑集团曾经进行过一次＂全民检测＂的大阴谋吗？但是香港人明白其目的何为，最后只有区区137万人参加(当中有不少是重复检测,占总人口不到20%)，港共集团全民检测失败告终。

之后又一个令香港人不安心的＂安心出行＂二维条码，以追踪CCP病毒名意，实际达到其监控香港人言行举止最终目标。

**对于港共集团达到的目标还差很远，主要因素香港人的不愿做中共的奴隶顺民，导致港共集团颇为失利。**

中共奴才张健宗（1日）又指为了逐步恢复社会正常运作营业及学校开课，所以有需要加强检测的台词假大骗计量忽悠香港人，全球被中共病毒肆虐下，以现在的数据显示是越来越严重，而且全球多国都陆续出现新变种病例，死亡人数不断攀升之际。现在一改以往守株待兔方式，以突撃式封锁区域，逼所有香港人强制检测，结果4个突击检测区确诊数为0，当然港共林郑集团＂项庄舞剑、意在沛公＂，其目的昭然若揭，就是假借抗疫之名施新强式管制为实。
![]()![](https://gnews.org/wp-content/uploads/2021/02/jpg-藉口2-2.jpg)图片：立场新闻
港共林郑上星期三通过视频与中共＂魔王＂述职之后，果然不出所料，中共最为担忧之事，如何全方位无死角控制香港人迟迟未了。港共林郑述职后的重中之重中共下达的命令任务，就是香港必须是爱中共的奴才们才能控港，说白一点就是如何将香港人变成奴才顺民，理所当然港共林郑集团迅速执行中共命令突撃封区域行动。

就着早前已四次封区检测，港共另一个奴才徐英伟说，过去在4 日在已解封的「受限区域」，进行5 次突发执法行动，合共发出约30 张定额罚款告票。以发告票为名，为求香港人乖乖自投落网，为求得到中共主子的满意度，花费了多少人力物力逼使香港人就范。当然中共最为重要的事，就是港共终于拿到香港人的DNA了。

这次更明显看到中共为求达到＂全民听话＂目的，已不惜一切成本代价，香港已如同新疆全民被控制模式看齐，中共真不愧为反人类、种族灭绝罪的集团。
