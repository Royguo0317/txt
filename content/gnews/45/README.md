首先给大家普及一个概念，什么叫封建？千万别说什么反封建，封建就是不好的，这是一大误解。欧洲中世纪，比如说英国国王，他手下有很多的贵族，世袭贵族，这些贵族每个人都有一块领地。有公爵、伯爵、侯爵，每个人都有一块自己的领地，他们只要向英国国王称臣，在他自己的领地他就是王。他有自己的军队，有自己的货币，有自己的所有财政，有自己的人民，有自己的土地，这个叫封建。封建是分封建国，分块土地给你，让你建国。

在中国什么时候是封建呢？从夏朝开始，夏商周三代都是搞的封建，那个时候的统治者叫天子。天子一个人管不了这么大的国土，因为那个时候交通不方便，他就搞分封制。他的一些重要的贵族，每人都分一块土地，让他们去管理，也有自己的军队但必须听中央的号令。周武王当时是三千诸侯联合去推翻商纣王，那个时候叫诸侯。一方诸侯就是他自己有土地、有军队、有钱币，就是一个小国，这叫封建。封建比较鼎盛的时期就是在西周，周武王打下了天下，他就把这个天下封给了有功之臣，包括他的家人姬姓的这些兄弟们，每人分块国土，划分了70多个诸侯。最大的两块，第一块就是功劳第一大的姜子牙，在今天泰山以北就叫齐国；功劳第二大的就是他的弟弟周公旦，他也分了一块很大的土地，和姜子牙做邻居，在泰山以南就叫鲁国，所以齐鲁大地那是圣贤的故乡。到了春秋时代，周天子虽然还在，但只是名义上的，大家都不听他的。到战国时期周天子基本就不存在了，后来周天子被秦国给灭了。最后秦朝统一天下，把所有的六国的贵族都杀了个一干二净。后来他就搞郡县制，就不搞分封建国这一套了。他搞郡县制，搞大一统，郡县制就是国家直接管理地方官吏。后来汉朝初年，也是不得已，封了几个有功之臣，还有刘邦的子孙封了几个王，但是后来到文景之治的时候，又开始叛乱，史称七国之乱。平定这场叛乱之后，分封建国就没有了，汉朝后面全是郡县制。还有一次一个小阶段搞过分封建国，就是西晋的司马炎，封了几个王，后来就搞了个八王之乱，把西晋给整了个半死，从此以后，中国历史上再也没出现过有实际意义上的诸侯，贵族都已经不存在了。战国时期的七国的这些贵族，他们以前都是轩辕黄帝的后人，那是真正的贵族，上古贵族。后来，六国全被秦国灭了。秦国的祖先也是上古贵族，是很有来历的。他的祖先可以追溯到跟大禹一起治水的，是大禹的一个得力助手。大禹治水的时候的，那是真正的上古贵族。秦国后来被刘邦给灭了，后面就没有贵族了，被杀光了。所以，封建早就不存在了，就夏商周三代有封建。所以你说反封建，你反的哪门子的封建啊？你反的不是封建，是中央集权，暴政集权，你反的是邪恶的帝制。凡是说反封建的那都是错误，根本不存在的，2500年前就已经没有封建了。

历朝历代的统治阶级怎么统治人民的，就两招叫外儒内法。外在用儒家的奴化教育，当然这个儒家是假的，叫伪儒。内在就搞暴政，一手是洗脑，一手是暴政，暴政就是商鞅的那一套。把这两个解释清楚了，我们就看清了共产党的那个套路，一模一样的，没有变的，他没啥创新。但是共产党是集中人类历史上古今中外所有的邪恶于一身。就是古代帝王的那套他也会；马恩列斯毛那套邪恶的他也会，希特勒、墨索里尼邪恶的那套他也学了。共产党的本质就是东邪加西毒。
