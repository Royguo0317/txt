【看中国2020年11月11日讯】伦敦市长选举将于明年5月举行，保守党候选人贝利（Shaun Bailey）表示如果当选就会结束伦敦与北京姊妹城市的关系，更欢迎港人来伦敦定居，届时将安排一系列广东话服务。贝利表示提出的一系列措施并非为了拉选票，是想真心帮助港人。

49岁的贝利在9月写了一封“致香港人的公开信”，提到其祖父母代表英国参军后，由牙买加移民至英国生活。身为移民后裔的他明白离开自己的家乡绝非易事，故提出政策帮助港人，希望身处异乡的香港人不感孤单。他以自己的亲身经历证明，在英国外来移民也可独当一面，比如移民可以成为民选议员，伦敦拥有多个移民社区，也非常欢迎移民人士。他勉励港人，身为移民后裔，伦敦黑人也曾面对许多挑战，但仍可成功，香港移民可以借鉴。

根据英国国家统计局2011年的人口普查，英国约有98,700名香港出生的居民，当中大多数居住在伦敦，占27%，而中国出生的居民比香港的多1.5倍，贝利提倡解除伦敦与北京的姊妹关系，并向中共发出强烈反对声音。他表示这关乎香港人的权益，要帮助港人抵抗中共，此举不会有助或损害他的选举，但这样做让他每晚都睡得安稳，因为知道自己在做正确的事。

英国内政部10月底发布经济影响评估，预计最极端情况会有100万港人透过BNO签证赴英定居，不少人更已提前置业。英国统计局公布截至8月，英国平均楼价为£239,196（约合241万港元），比去年升2.5%，中部以东一带（East Midland）升幅最多，达3.6%。伦敦楼价始终最贵，平均楼价达£489,159 (约494万港元）。被问及“会否担心港人涌至而推高英国楼价？”贝利称，伦敦在过去二、三百年间，一直有处理国际货币影响的经验，不担心香港移民会对楼市造成影响，英国鼓励融合，反对将一部分地区打造成“小香港”。

流亡英国的前香港众志主席罗冠聪此前也与贝利短暂会面，讨论“国际线”工作。他表示，初次与贝利面谈，觉得他态度诚恳，有青年工作的背景，十分关怀人民。罗又指，贝利针对香港人拟定的配套政策，都经深思熟虑，也认同其取消与北京姊妹城市的建议，“取消这衔头证明伦敦或英国政界，均对中共扩张感持有警惕，对中共释放较强硬态度。”他也期望，下一任当选市长可以帮助港人适应英国，推动更多课程让港人进修英文、学习英国文化。

自从中共在港强制实施港版国安法，英国政府十分关注香港议题。伦敦政治经济学院国际事务、外交和重大策略研究中心（LSE IDEAS）顾问研究员Luke Cooper表示，英国跨党派政客非常关注北京对香港态度，认为英国政府需要对中共在侵犯人权，和在香港国安​​法方面采取更强硬态度。他分析，五年前开始的中英关系“黄金时代”步入尾声，预计未来双方在人权方面的矛盾会令关系渐趋紧张，加上英国计画在基建设施等方面拥有更大的经济独立性；但最终中英关系如何发展，还须留意英国政治和经济精英在发展与人权问题上存在多少分歧。
