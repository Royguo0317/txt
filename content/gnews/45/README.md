# **中共国已用收购国防公司的方式“正在与英国冷战”**

**昨晚有人警告称，中共国通过收购国防公司的方式“已经对英国发动了冷战”。在此之前，国会议员呼吁禁止中共国和俄罗斯参与向英国武装部队提供设备的活动。**

[!\[\]()!\[\](https://spark.adobe.com/page/hIMxzfO5GAuAO/images/bd465ca9-9bec-4d47-963e-2d39a91cf307.png?asset_id=cedcfa0c-3850-41dd-a0ef-349756f44a72&amp;img_etag=%227012b6b1f4b6ce0d1c89ca3e1c752886%22&amp;size=2560)](https://spark.adobe.com/page/hIMxzfO5GAuAO/images/bd465ca9-9bec-4d47-963e-2d39a91cf307.png?asset_id=cedcfa0c-3850-41dd-a0ef-349756f44a72&amp;img_etag=%227012b6b1f4b6ce0d1c89ca3e1c752886%22&amp;size=1024)

下议院国防小组委员会说，海外公司在英国国防供应链中的广泛使用使英国国防对“潜在的敌对外国公司介入”保持了开放。报告说，特别是中共国有大规模盗窃知识产权的记录，这引起了人们的担忧，即与中共国供应商合作的英国公司可能会被盗取技术。该报告列出了六家尽管属于中共国所有，但仍构成国防供应链一部分的公司。

一家为F-35打击战斗机提供零件，另一家为阿帕奇（Apache）攻击直升机提供零件。

大多数公司都是自2017年以来介入的，此前四年，中共国主席习近平实施了所谓的军民融合战略，这一战略结束了中共国民营企业与国家之间的分隔，以推进军事、经济和技术实力。

其中三家被与中共国航空航天和采矿业有关的陕西省矿业资源公司（Shaanxi Ligeance Mineral Resources）收购，该公司一直在政府的监视名单上，出于安全原因，前商务部长格雷格•汉德斯（Greg Hands）阻止了该公司于2018年试图收购北方航空有限公司的计划。

[!\[\]()!\[\](https://spark.adobe.com/page/hIMxzfO5GAuAO/images/99c51d16-db12-4ec5-8e98-661c689cd94b.jpg?asset_id=b294f1bb-a282-4b40-a850-c1d76b31bf19&amp;img_etag=%2210b85bd9418e3b7b914eb7c7bcec6929%22&amp;size=2560)](https://spark.adobe.com/page/hIMxzfO5GAuAO/images/99c51d16-db12-4ec5-8e98-661c689cd94b.jpg?asset_id=b294f1bb-a282-4b40-a850-c1d76b31bf19&amp;img_etag=%2210b85bd9418e3b7b914eb7c7bcec6929%22&amp;size=1024)习近平启动了所谓的军民融合战略 （图片来源：盖蒂）

尽管商业性收购是合法的，但安全专家昨晚警告说，由北京控制潜在的重要国防资产可能对未来产生影响。

该委员会主席托比亚斯•埃伍德（Tobias Ellwood）议员说：“这是冷战行为的一个例子。”

“这不仅与建立武器系统有关，还关乎中共国通过收购能够限制我们能力的公司，来宣传自己的意识形态和挑战我们的意识形态。”

该委员会敦促部长们草拟一份“友好”国家清单，很高兴看到他们继续投资于国防供应链，而其余国家则应予以禁止。
