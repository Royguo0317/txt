## 【香港要闻】警察福利基金收到捐款暴增 香港过去一年警民矛盾是异常尖锐

搜集：Ken- 老广、 Po

编撰：文燕

审核：MY
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/15011350/image001-10.png)图片：香港媒体
据【立场新闻】报道，警察福利基金向立法会提交2019 至2020 年度年报（截至2020 年3月底止全年），基金上年度总收入高达1.88 亿元，比前一年的1949 万元激增8.6倍，主要是因为收到捐款暴增，由上年度约630 万元，增逾26 倍到上年度近1.8 亿元，而过去基金都不需要公布捐款来源。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/15011409/image003-6.png)图片来源：立场新闻
除了收入大增，基金的开支亦大幅增加，其中增幅最多的是「担任特殊职务警务人员茶点费用」，由上年度仅120 万，急增52 倍到近6500 万，发放予「陷入困荏难的警务/文职人员补助金」开支增加一半，至约320 万元，「员工关系开支」亦加了34% 到约313 万元，总结上年度警察福利基金盈余约7300 万。

警察福利基金是根据《警队条例》设立，主要宗旨是维持设施活动，向警员提供贷款、资助金和津贴等；据《星岛日报》报道，早前警察福利基金会首次向正规、辅警及各职级的文职人员派月饼，每人一个，月饼有四款设计，分别写了「忠」、「诚」、「勇」、「毅」四字，合共订制了三万九千个月饼。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/15011323/image005-4.png)图片：香港媒体
左媒【香港01】指，茶点费用上升，主要因去年因应连场反修例示威，为3万名执勤警员提供食物和饮料，每人每日平均支出少于7元。在决定接受捐款与否之前，基金管理委员会会按严谨的程序进行审核，并遵守政府内部及警队的相关规定，为尊重捐款人的个人意愿，因此不会公开捐款人数据。

据了解，担任特殊职务警务人员茶点费用上升，主要因去年因应连场反修例示威，警方推出「踏浪者」行动期间，每日有过万名前线警员当值，他们往往长时间外出工作，未能按时返回警署用膳，有时甚至未能获安排膳食，因此人员会获分配干粮、小食、能量棒/啫喱或饮品等，让他们于有需要时食用；行动期间警察总部及各区警署内会安排杯面、干粮、罐头食物及饮品，以作应急之用。另外福利物资开支方面，包括个人卫生用品如牙膏、牙刷、女性用品和即弃内裤等；退热贴 / 保暖贴；药物；维他命补充品。

**战友观点：**

众所周知，中共的「维稳费」惊人，而作为香港的维稳部队香港警员也得「肥养」。

【自由亚洲电台】去年有关中共「维稳费」的报道。中共官方公布的2019年中央级别的公共安全支出将近一千八百亿元。但也有国内媒体报道说，全国的公共安全支出， 也就是所谓的「维稳费」，将近一万四千亿元。与五年前相比，这是个什么概念呢？而巨大的维稳开支又折射出哪些社会问题呢？

香港《苹果日报》2019年3月11日发表的署名文章，引述广东《21世纪经济报道》稍早前引述官方发行的俗称「图解国家账本」的数据，2019年公共安全支出预算占全国一般公共预算支出的5.9%。如果按这个比例以及235244亿元的基数计算，中共的公共安全支出的预算高达13879亿元，比军费还高，2019年中国军费开支是11900亿元。

长期受到维稳人员限制自由的南京异议人士邵明亮在接受自由亚洲电台采访时表示，高额的维稳表明中国官民冲突事件不断，社会矛盾日趋激烈：「维稳费1.3万亿高于军费，而且限制逐年增加，他们恐惧，说明国内的矛盾将异常的尖锐，异常之激烈。中国人民反抗，各类群体事件，现在是风起云涌，暗潮涌动。」

早前立法会不足半小时便通过626亿元追加拨款条例草案。

当中包括警务处申请逾30亿元拨款，在没有民主派提出质询、只有两名议员发言下，草案不足半小时极速以35票赞成大比数通过。立法会前议员、公民党谭文豪批评议会犹如「善堂」。

今年警察福利基金收到捐款暴增，捐款者是谁？相信大家心中有数。而从捐款暴增也说明了香港过去一年警民的矛盾是异常的尖锐。
