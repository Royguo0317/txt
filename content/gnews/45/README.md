![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/21081137/image0-6-4.jpg)
COVID-19病毒是在中共国的一个实验室制造的，是中国人民解放军（PLA）生物战计划的一部分。

以上是已经清楚的。尚待解决的问题是其释放的方式，是意外还是故意的。

有越来越多的证据表明，2019年11月发生了一起实验室意外事故。

尽管缺乏证实，但关于解放军故意投毒的谣言仍然存在，其中一种情况是在2019年10月18日至27日，武汉举行的第七届CISM军事世界运动会上。

在2021年5月20日的《网关专家》（the Gateway Pundit）[文章中](https://www.thegatewaypundit.com/2021/05/know-covid-19-created-laboratory-new-revelations-inside-china-indicate-location-military-origin/)，引用了中共国内部的一个消息来源的说法：

1.在2019年的最初几个月，一种完全成型或接近完全成型的COVID-19病毒，从中国人民解放军东部战区司令部抵达武汉病毒研究所。

2.与流行的说法相反，COVID-19病毒被送到武汉病毒研究所，主要是为了进行测试。

3.测试工作被分配给一组年轻的病毒学家。该团队领导人的名字是已知的，该领导人的科学背景也与该角色一致。

4.病毒测试是在非人灵长类动物身上进行的。

关于以上信息，有两个要点需要记住：

首先，总部设在南京的解放军东部战区司令部，就是分离了（舟山蝙蝠）ZC45和ZXC21的实体设施，又和武汉病毒研究所有联系。即爆料革命的战友闫丽梦博士所说的，该实验室提供了制造COVID-19病毒的蝙蝠冠状病毒的骨架。 其次，由武汉病毒研究所科学家单超领导的非人灵长类动物病毒测试。此次病毒测试，可能不是在其位于江夏区的生物安全四级实验室进行的，而是在武汉大学位于武昌区的（武汉大学动物实验中心）ABSL-3动物实验室进行的。该实验室是早期爆发的震中。（链接：[twitter.com](https://twitter.com/LawrenceSellin/status/1396754639924695041?s=20)）

即使是武汉病毒研究所的 “蝙蝠女”石正丽也说，江夏区生物安全四级设施在疫情爆发前，没有资格使用非人灵长类动物。另外，疫情爆发后该实验室才申请。（链接：[twitter.com](https://twitter.com/LawrenceSellin/status/1396750355418030090?s=20)）

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/26044732/unknown-36.png)

此外，疑似推定COVID-19非人灵长类动物实验的负责人单超，列出了他的专业地址，不是在江夏的生物安全四级实验室，而是在武汉病毒研究所位于科济路的生物安全三级设施。该设施是靠近武汉大学的ABSL-3实验室。（链接：[twitter.com](https://twitter.com/LawrenceSellin/status/1396682901220102145?s=20)）

现在出现了更多的证据，其中一些可能被认为是“铁证”已经出现。ABSL-3实验室支持COVID-19病毒可能在进行致病性和传播性研究时，或许过程中感染了该设施的工作人员，才释放出了测试病毒的说法。（链接：[twitter.com](https://twitter.com/DrLiMengYAN1/status/1396911086092357632?s=20)）

尽管此前在2016年被评为国家“万人计划”领军人才，但在2020年5月6日，李红良突然被免去武汉大学动物实验中心主任和武汉大学ABSL-3实验室主任的职务。根据以上报道：

“经过调查，没有发现李红良造假行为，但发现很多论文有滥用图片的现象。这反映出该实验室对实验数据的处理不够严谨”。

之后，对武汉大学ABSL-3实验室进行了一系列广泛的安全检查，并于2020年8月24日[宣布了](http://kfy.whu.edu.cn/info/1005/2161.htm)新的安全协议，专门针对冠状病毒研究。

还有其他间接[证据表明](https://twitter.com/TorontoFarmEN/status/1396961056212365316)：2019年11月在武汉大学，发生了一些不寻常的事情。 大量证据表明，COVID-19大流行病可能起源于武汉大学的ABSL-3动物实验室。 但该实验室是新冠病毒大流行的肇事者，还是解放军更邪恶行动的替罪羊呢？（链接：[twitter.com](https://twitter.com/DrLiMengYAN1/status/1396855117568172033?s=20)）
