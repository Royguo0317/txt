- **中共病毒暴发、扩散速度非常惊人**，不到一年，已经扩散到了世界各个角落。但有证据显示，**未来中共病毒很可能继续进化，产生更可怕的传染率和致死率。**必须提高警惕，为未来更凶险的情况做好准备。
- **中共病毒的威胁将会持续下去，**不过我们从疫情中也学习到了很多。大家已经变得更具有组织性、更为警醒以及在特殊情况下如何生活更有经验。
- **中共病毒很可能只是预警，未来会有更为致命的中共变异病毒出现****，**这就像整个人类社会通过对抗中共病毒进行了一次演习。演习之后，致命病毒很可能会降临，致死率将会高很多。


本月底，WHO正式通报了中共新冠病毒自出现以来的主要变异情况：包括四种变体，其中三种传播力增强。
