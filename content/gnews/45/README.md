今天早上一家名为Assoeventi的专门从事承办私人聚会和仪式的组织，其总裁米歇尔·波卡迪（Michele Boccardi）发起了抗议活动，他接受采访表示，这些活动自去年3月以来已停止，我们是唯一不被重新开放的人，有永远不会重新开放并失去重要份额的外国旅游的风险，涉及到30个相关行业近5000家公司，其中850,000名员工为固定雇员，我们这个行业的营业额为300亿美元，占意大利GDP的2.5％。

米歇尔·波卡迪（Michele Boccardi）是Assoeventi的总裁表示：我们有两家婚礼组织公司，去年我已经安排了246个活动，只有11个活动举行。十月份由于大流行已经关闭了所有活动，我们已经取消了约85％的订单，因此营业额损失了90％。在我们的行业中，有46670家公司雇用了850,000多名意大利公民。我们有失去一切的危险，已经关闭了14个月” 。

许多夫妇对推迟结婚感到厌倦，婚礼仪式已经成为一个重要传统行业的终结。然后这却是意大利旅游业重要的一部分：最近五年尤其是普利亚，托斯卡纳，坎帕尼亚，西西里和撒丁岛，这些是来自世界各地最受欢迎的举办婚礼目的地，但现在该行业被取消，因此，这种类型的旅游业有可能会转移到其他目的地，例如希腊或斯洛文尼亚。

#### 简评：

中共病毒疫情使意大利经济遭受重创，依赖外国游客的旅游业及依赖出口市场的制造业受到了严重的打击，2020年意大利国内生产总值大幅下滑。从病毒爆发到现在，似乎只有超市和药店是正常状态。这篇新闻稿中的行业，只不过是众多受疫情严重影响的一个缩影而已，而意大利也不过是整个世界经济的一个缩影。疫情的持续存在，让未来的经济很不明朗，政府总是试图推出一些刺激政策来提振经济，而真相是摆脱不了中共病毒的缠扰，意大利的经济复苏之路注定会非常渺茫，意大利本来在欧盟中属于拖后腿的角色，现在真是雪上加霜了。
