- **11.3. 环球网 中方侵占尼泊尔150公顷土地？汪文斌与英记者交锋**


在3日举行的外交部例行记者会上，有来自英国《每日电讯报》的记者提问称，尼泊尔官员称，中国已经在5个地区侵占了尼泊尔150多公顷的土地，同时中国还改变了作为边界的河流的流向，此前在水下被淹没的尼泊尔的土地也被中方吞并，中方对此有何回应？

对此，汪文斌在回应中首先询问该记者道：“你说的是《每日电讯报》的报道吗？”在得到该记者肯定的回复后，汪文斌指出：“我要说的是，有关报道完全是子虚乌有，纯属谣言。”

该记者随即追问称：“中方能否提供有关的证据，来证明你刚才所说的这只是毫无根据的传闻？”汪文斌回应称，“既然这是一个谣言，恐怕有关报道的起草者应该首先证明，他们做出这样的报道有什么样的依据”。

该记者继续追问：“我们是有证据的，我们是从尼泊尔官员那得知的这些‘中方侵占领土’的消息，所以我才来问中方有何评论。”

汪文斌表示：“我想建议你们做出这样的报道之前，应该事先要做负责任的核实，之后再做相关的报道。我可以再告诉你，这样的报道完全是谣言。”

该记者最后一次追问称：“我们已经向中国驻尼泊尔大使馆提交了问题单，希望得到他们的回复，但是他们没有回复我们。”

汪文斌回应称：“我在这里可以给你一个准确的回复，那就是这样的报道没有任何事实依据，纯属谣言。”

- **11.3. 环球时报 港媒：中国雄心之巨大 令首尔东京华盛顿都吃惊不已**


日本的发展模式曾是世界用来衡量亚洲成功的尺度。到20世纪80年代，两种“亚洲虎”出现。前日本殖民地韩国和中国台湾地区成为制造业超级明星。他们或多或少采用了东京的战略，通过工业增长来磨炼竞争力，提高生活水平。现在，千禧年的中国正颠覆这种屡试不爽的发展模式——中国领导人上周详细阐述了多管齐下的高速发展计划，清晰地表现出这点。中国雄心之巨大，可能令首尔、东京和华盛顿都吃惊不已。

长远看，北京的十四五规划是掌握未来的路线图。中国推出从人工智能、金融、医疗、可再生能源到半导体等行业领先的战略。中国在走“超级虎”的路线。尽管借鉴了日韩做法，但北京显然在另辟蹊径——且以非常宏大的方式。

“中国制造”的宏伟计划为特朗普的对华攻击建立了某种预期性防御堡垒。这意味着，在关税和对中国科技公司的攻击冲击中国经济之际，该国各种结构性改革已在进行，以推动从烟囱产业向服务业转型。中国的新五年规划大大升级，在某些方面甚至可以给日本启示。

中国追求的技术自立自强，是发达得多的日本从未实现过的。日本的标志性企业曾引领全球创新竞赛。但如今，它们只满足于作为全球供应链中的环节而存在。而中国正在打造自己的企业和创新世界，它注定是东京甚至鼎盛时期的日本不能相比的。中国现在已清楚地认识到刺激内需和本土技术的需求。

真正有意思的问题是，特朗普的关税和攻击对于推动中国向价值链上游攀升有多大帮助。正如中国科技部部长王志刚所言，科技自立自强首次摆在各项规划任务的首位。

如果不是特朗普要在中国与最先进计算机芯片以及完善这些芯片的机器之间筑起一堵墙，中国推动半导体自主（的需要）可能不会那么紧迫。特朗普试图扼杀中国5G产业的种种举动，正在产生惊人的相反效果。

新冠疫情实际上展示了中国在人工智能算法方面的进展。一批人工智能初创企业崛起，许多在公共卫生领域。这可能会让硅谷、东京、首尔等地更多地向北京而非华盛顿寻求飞速解决方案。不仅仅是速度，中国展现出一种发展模式，其规模是世界上前所未见的。。
