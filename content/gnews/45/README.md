据郭文贵先生说，在2021年1月初GTV就会上线卫星频道，为了让战友们方便，容易的收到GTV卫星信号，了解第一手爆料革命信息，上次发文解释了卫星电视的基本原理，本文再进一步讲解卫星锅的硬件准备需求，目前GTV卫星参数未发布，暂以韩星5号为范例。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161621/Picture1-2-23.png)
一、**准备**

**A.接受条件确认**

1.查您的所在地有没有卫星信号（场强）

每颗卫星都其覆盖的信号范围，在购买设备前要在网上查相关数据，看看信号是多少。

信号关注两个方面，1.有没有信号。 2.根据信号计算买多大的锅。请参考下图。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161723/2-34.png)
2.有没有摆放锅，调星的地理条件

要确定摆放卫星天线的空间。您的阳台，窗边，有没有摆放锅的空间。阳台、窗的方向能不能对着卫星。如果有楼顶或房顶可以对着360度的天空是最好的。

有信号+有空间+有方向=可以收卫星，否者请不要随便浪费钱。请参考下图。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161637/3-18.png)
**B.**需要准备四个设备与备用接头，共五种。

1. 卫星机顶盒


作用：把卫星信号解码为电视号，用来输出到电视，例机型号：NEXTWAVE HD2020

**GTV参数未公布，机顶盒请不要提前购买**
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161702/4-8.png)
2. 高频头

作用：用来接收卫星信号，用同轴线连接到上面的机顶盒，例机型号：KU 9750MHz 10600MHz双频头，噪声系数0.5Bb(此值越低越好)，选购什么品牌都行，价格不贵。

**GTV参数未公布，高频头请不要提前购买**
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161714/5-8.png)
3. 接收天线（卫星锅）

作用：用来把卫星波反射到高频头上，可以调节仰角。例机型号：45KU，即45CM尺寸的收KU信号的锅，有条件的战友可以使用尺寸大的锅，尺寸越大信号越强。

**GTV参数未公布，卫星锅请不要提前购买**
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18162333/6-5.png)
4. 同轴线

作用：连接高频头与机顶盒，需要多少米就买多少米，如图。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161739/7-3.png)
5. 接头

作用：F接头，同轴线接头坏了时替换用，准备1-2个即可。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161747/8-3.png)
**二、硬件安装**

1.  安装接收天线

按照说明书，拼装天线（锅盖）
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161827/9-2.png)
2. 安装高频头

注意：圆头向锅，接头向外。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161849/10-3.png)
3. 接同轴线同轴线一头接高频头，一头接机顶盒。注意同轴线的金属针插入接头，然后拧紧。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161829/11-9.png)


4.机顶盒接线

机顶盒接电源，HDMI视频线，卫星天线的同轴线三个地方。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/18161905/12-6.png)
