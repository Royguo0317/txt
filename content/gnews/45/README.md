**一、我们每时每刻身体的细胞都在新陈代谢，我们的意识每时每刻都活在很多小死亡中。但意识没有丝毫受损。在每个人的一生中我们每七到八年身体每个细胞完全被更换过，但我们的意识（灵魂）并不知觉而继续顺滑地驾驭者我们的新身体。也就是说在我们认为我们“活着”的时候，我们就每时每刻都跨越了无数死亡，而我们丝毫未损。**

《赛斯书》，赛斯（是个不在我们物质世界层面的，在更高维度层面的高灵）现在我们来看赛斯书《灵魂永生》中对死亡的诠释，生與死是人生大事，可以说没有人不恐惧死亡的，大部分的人对死亡容易呈现负面的害怕恐惧心，最主要原因是对于死亡未知带來的恐惧。当肉体不再是活着的时候，会发生什么？我的感觉会如何？我还是我自己吗？有没有天堂或地狱？有没有神或魔、敌人或我所爱的人来迎接我？当我死了，我还记得那些我现在所爱的人吗？死后的世界到底是怎么样的？是不是阴森恐怖，是不是有六道轮回，会不会下油锅？下辈子变鸡变狗？对于这些，想起来就让人觉得恐慌。

赛斯告诉我们：我们现在活着，我们是一个知道它自己的意识，而且是在你知道你的意识的过程中你活着，与此同时我们身体的细胞原子和分子却发生着大量的死而复生得变化。就是现在活着的我们身体每时每刻发生身体细胞的死亡和新细胞的产生。因此，我们是活在许多“小死亡“的中间，而我们的意识却意识不到这件事。

人体一共有40万亿—60万亿个细胞，肉体需要通过细胞的新陈代谢活动提供能量，在这个过程中，不断有旧细胞衰亡，也不断有新的细胞出现。人体细胞新陈代谢，每三个月会替换一次，随着旧细胞的死去，新细胞则诞生。由于不同细胞代谢时间和间隔的不同，将一身细胞全部换掉，女的大概是需要七年，男的是八年。也就是说，在生理上，肉体上，我们每七年八年就完全换了另外一个人。即没有一个细胞是七年前的。我们现在的身体没有一粒包含“它”七、八年以前所有的物质。也就是我们的身体现在与七、八年前是完全不同了，我们七、八年前拥有的身体已经不存在了死了。而我们的意识却显然不觉得我们完全换了个身体，维持着我们原来的意识，，我们能够用我们全新的、七、八年前根本不存在的新嘴巴说话，用七、八年前根本不存在的新眼睛来读书。我们现在在用的整个身体七、八年前并不存在，就是说我们的意识已经是跨越了很多次身体的更新而到今天的！而且这过程继续得如此平滑，以致你根本不会觉察到。

文贵先生说他现在的身体与原来是完全不一样的身体，各个方面，连他的太太都说他根本就不是同一个人！我自己本人的感觉也是一样的。我前面讲的新陈代谢，因为时间是那么的短，所以我们的意识愉快地跳跃了过去。 因此我们的意识（精神）是在我们的永远在变化的肉体形象周围闪烁。

赛斯说：你可以把你现在所知的意识比为一只萤火虫，它闪烁不已时明时暗的，永不会完全的熄灭。因为虽然在你看来你的意识好象是连续的，其实不是。所以我们换了几个身体，我们意识都感觉不到。现在，量子力学也证明电子闪烁，原子核电子具有波粒二重性：你观察它时在这里，不观察它时不知道在哪里，我们的意识意识不到这一点。赛斯说：我在用你们自己的说法。因此，说到“死亡”这个词，其实是指意识完全不贯注于物质实相内，当你的意识活在（聚焦）在肉体里时，它其实也不是所有的时间都活在肉体里，并且定位于肉体里的。也就是说，我们的的意识有一瞬是“活着”，贯注于物质实相里，在下一瞬它完全贯注于别处，在一个不同的实相系统里（其他空间里面），但是我们感知不到那个切换的瞬间。晃神，就是一个很好的例子。

赛斯用“一瞬”来比喻，意识在物质的系统内是“死了”或不活动，然后又活了或又活动，但我们不能感知它们不存在的那一瞬。因为速度太快。就像我们在看电影，每一张底片都是静止的，但如果我们以每秒24张底片的速度播放，荧幕上的人物故事就活龙活现，而意识的脉动比放电影的速度要快太多。

赛斯说：整个的宇宙结构也以同样方式存在。它闪烁生灭，并且以某种节奏，如呼吸的节奏，一进一出，在生灭。在它们内有无穷的个人性变化—-几乎象是宇宙的新陈代谢。

**二、每天我们的睡觉就是意识离开肉体，是意识另一种小死亡的体验，我们的意识（灵魂）不但没有被损伤而且睡后起来我们更加精神饱满电力充足。**

赛斯说：其实，人的意识已很习惯这些离开肉体的短暂一段空隙了，而较长的空隙则会令意识或多或少迷失方向，当我们入睡的时候，当肉体睡着了，意识常常离开物质系统一段的时间，但因为我们的意识不是在正常的肉体醒时状态，它不察觉到这种空隙，我们做梦时在不同时空来回穿梭几个小时，我们的意识并不担心是不是死了。

如果意识在正常的肉体醒时状态中撤出身体比较长的时间，它就会认为它自己死了，因为它不能给这个维度和经历的空隙（间隔）合理的解释。在睡眠状态下，我们每个人都在某个程度上经历了意识离开了物质实相的情形，那其实与在死亡时的经验是同一类的。意思是我们睡眠中做梦的状态和我们认为的“死亡”的状态没有什么差别。

意识在我们醒来后又回到了身体，但是，我们其实已经踏进过了其他时空的门槛许多许多次，所以赛斯说：你们认为的“死亡”这种情形并不是像你们认为的那么陌生和恐怖。

死亡只不过是你一觉醒来发现自己是个婴儿！但一旦你穿透死亡的迷雾，你就豁然开朗，不再害怕与恐惧不再受到死亡的限制制约。因为我们的意识是被保护的，认为死亡恐怖的是我们外在的自我意识。当你明白真相，就会达到如《心经》里说的：心无挂碍，无挂碍故，无有恐怖，远离颠倒梦想！究竟涅槃！

《灵魂永生》是赛斯的第一本书，被印刷成世界多种语言文字，是所有赛斯思想的总论，灵魂永生论能吸引很多人是因为它讲到很多生死的问题，它不止详细纪录人是怎么轮回转世，还包含了人在断气的那个瞬间到底意识发生了什么，甚至人在断气之前产生的所谓的瞻望状态，以及断气之后人的意识到底是什么样的状况，要经过哪一些的阶段、经过那些阶段之后又如何回到人世间来投胎。关于重新投胎，其实要简单的多，就是看时代、背景和重新跟周遭的人建立起一个新的缘分。那么，已经完成了轮回经验的人，又有哪一些不一样的选择呢？简单来讲，在灵魂永生论里面，它讲了从古到今从来没有人讲得清楚的东西，因为能讲清楚这些的都“死掉”了，活着的都讲不太清楚。

虽然我们生活在科学发达的二十一世纪，然而，现代科学并没有弄清楚到底有没有死后的世界？死后要去哪里？多久之后再回来？已经往生的人用什么样的形式存在？他们过着怎么样的生活？他们那里有没有吃的有没有喝？有没有朋友？他们能不能看到还活着的家人？他知不知道你在做什么？知不知道你发生了什么事？而赛斯对这些都讲得一清二楚。

赛斯的书非常有意思，你一般看2-3页就会睡着，醒来再看，再看再睡着，因为你需要花时间消化吸收。即使你看完后，回头再看，又像新书一样，每一遍都会有很多新的体会，非常奇妙，它能让你这辈子反复去思考、反复咀嚼。

赛斯借用罗伯兹的身体说话

赛斯说空间跟时间都是幻想，他从五维空间看我们地球，仿佛好像在看一副画，这幅画代表我们整个宇宙，当你走进画里面，那幅画可以不断延伸出去，在画的外面你以为是一个平面，当你进入那幅画，那幅画就变成立体的。你可以走进100公里、100万公里，像我们打网络游戏，可以在游戏里走很久很远，但对虚拟空间外面的人来说，他根本没走嘛，没有距离也没有空间，但游戏里的人却走了很远很久。高山、大海、河川对三维里面的人来说是真的，但对外面的人（赛斯）来说是假的，所有的空间与距离根本就是幻相，我们的宇宙就是一副画而已。所以，赛斯说我们的科学家不管制造出多快的太空梭，只是在一副画里面进行旅行，从地球飞到火星、飞过太阳系，这些我们现在认为的太空旅行其实是在一幅画里面旅行。赛斯说真正的太空旅行不是坐太空梭，而是精神体（意识体）的穿梭。因为我们的精神（意识）是在多维空间里的，只有用它才能离开那幅画，人类现在太聚焦在物质层面了，如白天与黑夜，我们太聚焦醒时的意识状态了，而对睡眠中意识状态不胜了解。

**三、我们认为的死亡其实就是当我们醒来发现我们是个婴儿是个孩子开始了新的一生。就是这个过程。死亡是安全的平安的。**

我们所谓的往生就是把肉体抛下，从一副画到另外一副画里面去。赛斯说：人类所说的死亡只不过是插入了较长一段那不为你知觉的脉动，可说是意识在另一个实相（维度空间）里作较长的停留。

因为肉体不能带离开这幅画，它是这幅画里面的原素。所以肉体是不能真正进行太空旅行的。真正的太空旅行是意识旅行到另外一个地方。在另外一个地方重新组一个肉体。那才是真正的太空旅行。我们如何进到其他的空间？很简单，做心里时间练习（冥想）或灵魂出体或抛下肉体“死掉”，你就到另外一个空间去了，唯有这样你才能够离开伪装实相（三维物质），所以，我们以为我们的宇宙很大，其实，我们就在一副画里面，飞来飞去。每次我想到赛斯这个比喻我就觉得感叹！三千大千世界原来全部都是幻相，三千大千世界原来就在一副画里，我们再把这个比喻再延伸出去，你的前世在哪里？你的前世在另外一副画里；前前世呢？又在另外另外一副画里，每一世的自己，都在一个独立的时间与空间里面。时间空间是幻相，我们广阔的宇宙也是一个虚拟实相，我们每一个人的肉身是来地球出差旅游、学习考察兼玩耍的，但是，灵魂永生的！
