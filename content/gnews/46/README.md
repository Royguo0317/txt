一、彭佩奥演讲传递重要灭共时间线

彭佩奥国务卿在乔治亚理工大学的演讲透露[1]：北约已经从传统上的对抗苏联俄罗斯，转为对抗中共，上星期的北约会议花了近一半时间讨论如何对付中共。他还说，中共和习近平想把他们的专制模式强加于美国和世界，中共的这个模式虽然自己会最终崩溃，但美国也不能仅仅等待，他们有义务保护自己。

路德社评论如下：

1. 彭佩奥全段演讲内容主要针对中共，他上周参加北约会议，北约已明确将目标从俄罗斯转向中共，起初北约的建立是为应对苏联挑战，**路德社爆料俄罗斯此次将与北约联盟共同对抗中共**；
2. 彭佩奥在演讲中暗示他们为了保卫美国的繁荣，将主动出击对抗中共，他经过长期的努力说服北约成员国共同参与灭共；
3. 《北大西洋公约》第五条规定[2]：一旦确认成员国受到攻击，其他成员国将做出及时反应，该条款被理解为各国防部自动参战，不需要各国政府的参战授权；
4. 北约已有备战经费和作战部队，完全拥有随时参战的实力，公约规定三种情况下，北约可以参战：成员国受到攻击，欧洲出现战事，联合国授予北约参战；
5. 川普总统如果要独自采取军事行动对付中共，一个月内可能无法结束战斗，此时就需要国会投票决议是否对中共采取军事行动，一旦北约成员国参战，即可将战争时间缩短到一个月内，川普总统就能跳过国会直接采取行动；
6. 美国驻华大使馆也直接发推提及彭佩奥的演讲内容[3]：“在校园里，中国共产党最大的受害者有些其实是无辜的中国公民自己，这是一个悲剧。”

二、中共国防部长发言称要坚决维护国家主权

国防部发布微信公号12月10日消息[4]，第七届东盟防长扩大会视频会议当日举行。中国国务委员兼国防部长魏凤和出席并发言，说到：“中国军队坚决维护国家主权、安全和领土完整。”

1. 这是国防部长魏凤和在东盟防长扩大会视频会议的发言，现在的东盟类似于当年苏联的《华沙条约》联盟，当初由中共以反恐的名义牵头组织；
2. 国防部长不拥有军队，中共军队由军委掌握，中共常利用安全部（政法委下属）的名义在海外开展“蓝金黄”行动；
3. 魏凤和此方发言证明中共已经意识到北约即将采取行动，中共希望联合东盟力量帮忙与北约抗衡。

2. 此前中共已切断与川普总统对话，现在他们已意识到川普总统即将连任，并且要消灭中共，中共希望通过所谓的“对话”赢得“蓝金黄”美国政府官员的时间；
3. 此次灭共行动的参与国不仅只有美国和北约成员国，还有印度、日本、韩国、台湾等国家也会参与；
4. 美国只需要根据军事情报就可以确认遭受中共攻击，北约就必须参与灭共战争。
