#### [外商投资范围再扩大！新增127条 制造业仍是重点 涉及自动驾驶硬件、ECMO制造](http://finance.eastmoney.com/a/202012291753818873.html)

12月28日，国家发改委、商务部发布《鼓励外商投资产业目录(2020年版)》，自2021年1月27日起施行。新版目录总条目1235条，与2019年版相比增加127条、修改88条，重点增加制造业、生产性服务业、中西部地区条目，引导外资投向，提振外资信心，促进外贸外资基本盘稳定和[产业](http://baike.eastmoney.com/item/%E4%BA%A7%E4%B8%9A%E9%93%BE)链供应链稳定。

本次修订主要含括了以下重点,
1. 扩大鼓励范围，条目只增不减; 为保持政策连续性、稳定性，同时体现产业优化升级和区域协调发展需要，使符合支持方向的新投资享受优惠政策，在条目上只增不减。

2.全国目录继续将制造业作为重点鼓励方向. 根据“引资补链”“引资强链”“引资扩链”导向，全国目录增加了相关内容。涉及原材料领域，零部件领域，终端[产品](http://baike.eastmoney.com/item/%E4%BA%A7%E5%93%81)领域等. 

3.全国目录还鼓励外资投向生产性服务业, 顺应国内制造业服务化发展和市场需求升级需要. 
促进服务业和制造业融合发展也是本次修订的重点之一。全国目录中，新增高端装备维修、数字化生产线改造与集成等条目。现代物流领域，新增或修改跨境电子商务零售、大宗商品进出口分拨中心、社区连锁配送等条目。信息服务领域，新增在线教育、在线医疗、在线办公等条目。

4. 中西部目录助推区域开放型经济发展和外向型产业集群. 
具体来看，在黑龙江、云南等省份新增农产品加工、旅游开发等条目。在河南、陕西、广西等省份新增医疗器械、防疫防护用品、原料药生产等条目。在湖北、四川、重庆等省份新增半导体材料、石墨烯、工业陶瓷等条目。在辽宁、安徽、湖南等省份新增职业院校条目。在海南新增商贸、航运、金融、旅游等相关条目，支持海南自由贸易港建设。
