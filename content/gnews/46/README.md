**缅甸断网部署军队，缅甸战友发布一手信息**

缅甸人民示威游行要求昂山素季回归，媒体报道缅甸军方再次断网，并在各地部署军队。据爆料革命战友说，由于中共和缅甸军方的勾兑，民众一致示威如果断网就炸掉皎漂油气管道，这条中缅油气管道是连接中国西南地区和缅甸的原油及天然气输送管道。而缅甸军方释放了约2万名囚犯扰乱社会、祸害民众，并且安排警察对民众进行非法打击，这些手段都同中共镇压香港的卑劣方式如出一辙。

当前因为这位缅甸战友的反共、宣传真相，被缅甸华侨包括合作已久的生意伙伴纷纷屏蔽。而中共曾经刻意捆绑既得利益的华人华侨，将他们安插、渗透在世界各地传播中共的意识形态。

**格雷厄姆与川普通话谈论重建共和党**

格雷厄姆作为共和党建制派给川普打电话试探口风，并将到弗罗里达与川普会面。在电话之后，格雷厄姆放出川普对重建共和党似乎表示兴趣的消息，他称赞川普总统和MAGA支持者是共和党不可缺少的力量，就是为了削弱川普创建MAGA党的言论风头，不希望共和党势力被分离。但是共和党建制派是否可以让川普放手去重建？冠博士称这可能又是一个利用川普再将其抛弃的陷阱。

美国大选的结果就是川普被共和党建制派出卖，无论是麦康奈尔还是彭斯，而格雷厄姆在大选之前也是见了川普总统表面上要支持，但在1月6日还是反川普的。包括到120那天有很多策划也被出卖了，所以最后一些事情并没有发生。

此外建制派知道若让麦康奈尔去通话，川普百分百不会答应，所以就让格雷厄姆去谈。如果谈判后共和党能让川普说了算，那可以直接进行改革重组，但是如果MAGA只是被共和党拿来用，最后还是建制派说了算，那么川普总统绝对不会答应。

**沙特将不顾拜登政府对胡塞武装认定**

根据沙特在联合国常驻代表说，尽管美国拜登政府对胡塞武装认定的解除，沙特将继续把胡塞武装列为恐怖组织。这是首次同盟国产生的巨大分歧。拜登也正在将沙特置于了危险境地。

影响中东地区和平关键的在于耶路撒冷问题，而以色列同阿拉伯、阿联酋等国的建交，使得中东地区获得稳定，美国得以腾出手来集中力量对付中共，此举是川普在国际地缘政治上最重要的布局。中共发展的黄金时期就来自中东乱局，由于被美国拉拢打恐怖分子从而获得了大量关系和资源。

同时如果以色列能够不受沙特、埃及等周边穆斯林国家牵制，对付伊朗绰绰有余。所以川普灭共关键一步棋就是把中东地区稳定。而拜登将也门胡塞武装恐怖组织定义取消后，伊朗就可以合法支持胡塞武装，使其破坏性增强，以色列就没法腾出手来到印太地区应对中共，而拜登就是在帮中共、伊朗点燃中东的火药桶。
