## 【战友心声】为匪作伥的白虎——共产党的洗脑术
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/1043143/)

作者：纽约香草山写作2组 苟日新

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3.jpg)



通常印象中的印度电影，除了遥远追忆的《大篷车》之外，就是近些年的宝莱坞出品，阿米尔汗的一些商业片，如《三傻大闹宝莱坞》，《摔跤吧！爸爸》，讲述一段平易近人且积极向上的故事道理，片中穿插印度独有的歌舞。

在文贵先生推荐《白虎》之后，一直在想一部印度片会怎么呈现目前这个乱字当道的世界。看完电影，再听文贵先生的分析讲解，才反应过来，片头挑衅般地、“臭味相投”地写信给时任总理温家宝的情节贯穿始终，大有中共国红歌时代的站队既视感。
![]()![](https://gnews.org/wp-content/uploads/2021/03/Jietu20210303-093443.jpg)
看似轻描淡写，却将其膨胀无比的野心展露无遗——灭白，将现如今全人类所有的问题归咎于白人。

倒叙是电影的经典拍摄手法。与《美国往事》《教父》等以倒叙的方式回味过往，品味其中蕴含的行为道理的手法不同的是，《白虎》的倒叙手法更像是为酝酿一个大事件做铺垫，这位刚刚从最底层爬上来白虎，找到了一个和自己成长道路相似的、可共事的、强大的“伙伴”。

片中主人公巴拉姆出生在一个低种姓家族，却因为自己的天资聪慧被老师标注上百年难得一见的白虎标签。不甘心屈服于社会和来自家族的压迫，巴拉姆找寻着他力所能及的逃脱方式——当高种姓人的仆人。

仆人？是不是有很多人冒出了一些问号，现代社会还有主仆关系？是现代奴隶制吗？

由于被奶奶中断了读书，巴拉姆能汲取到的知识和信息，只能来自周遭的环境。而在偏远而落后的村落里，最不缺的就是陈旧的信息和被人诟病的糟粕，比如——种姓制度。
![]()![](https://gnews.org/wp-content/uploads/2021/03/Jietu20210303-093923.jpg)
种姓制度来源于彼时的“身毒”（古代对今印度国名的音译）被雅利安人入侵之后。为了以少量人口掌控庞大的印度，雅利安人发明了婆罗门教，婆罗门教用一套严密的等级体系来控制印度人。

前面说了那么多就为了解释一个巴拉姆所处的环境，但这是由一个白人、一个时期的白人导致的吗？我们可以实事求是地说，雅利安人入侵后导致了非常多社会的变动史料可查，但是古婆罗门时期之前身毒土著时期呢？还有废除种姓制度、摆脱殖民地之后呢？原因是综合性的。如果将其简单地归罪于某一方，这无异于共匪1949年之后一直给孩子们洗脑的抗日战争和国共内战的二元对立，让小孩子认定日本人和国民党就是敌人，却对战争的实情避讳不谈，因为中共知道，一说实情就会露馅。正是这样的奸邪之言，歪理邪说，洗出来一批又一批的共匪所谓“暴民”。

杀掉阿肖克之前，巴拉姆对着神像上香。杀阿肖克的时候，他也带着神像。想想这是何其讽刺。但是，这样的事情却每时每刻都在被共匪治下的中国进行着。拜佛是为了求个心安理得，事成之后就来还愿。试问如果真是如此，神佛岂不是跟这些腌臜之流一样是作奸犯科之辈？

谁都希望自己正直、上进、有责任感，美好、纯净、有好品德。但俗世浑浊，社会浮躁，有多少人被逼无奈，就此沉沦，却反过头来怪罪被它们搞坏的民风秩序。

共匪盘踞中国日久，四处破坏各种正道和信仰不说，还侵蚀了让国人挺起脊梁的知识学问，只顾瞒骗、洗脑、政治倾轧，教育分数论，学校少有育人，多是生意，没有教会独立、自立和正直的观念，却歌颂着不存在的功绩。只把华夏大地当做一个徒有其表的世界工厂，出卖资源换取短期的所谓利益。

在被搅动的乱世之下成为一个值得别人和自己信赖的人，变真、变正、变强而不变正道的信仰，纵使金钱色欲乱拨是非，邪恶肮脏伪装得再好总是会露出插入后颈的锋口，只要行得端正，取财有正道，面对任何谎言欺骗，也只当风拂面，何足挂齿。
