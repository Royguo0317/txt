## 纳瓦罗的报告指控美国大选的六项违规行为

据NewsMax12月17日报道，白宫贸易顾问彼得·纳瓦罗（Peter Navarro）周四发表了一份长达36页的抨击本次大选舞弊的报告。纳瓦罗在报告中明确指出，被揭露的选民违规和选举舞弊行为数量之多足以使川普总统大选翻盘。

纳瓦罗的这篇名为《完美的欺骗：选举违规的六大模式》的报告通过研究六个关键选举州的六种违规舞弊行为，对2020年总统选举的公平性和完整性进行了评估。报告指出，六个关键选举州的选举违规模式是如此一致，以至于它们表现出了一种协调战略，即便不是完全窃取选举，也是以“塞满选票箱”的方式对选举过程进行战略操控，使选票倾向于有利于拜登-哈里斯一方。

报告中提到的与六种投票违规行为相关的六个关键州分别是：亚利桑那州、佐治亚州、密歇根州、内华达州、宾夕法尼亚州和威斯康星州。报告的结论支持川普总统和他的法律团队的指控，即11月3日的选举结果被广泛的舞弊所破坏，川普一直拒绝承认竞选失败。尽管选举人团确认乔·拜登（Joe Biden）获胜，但还将于1月6日由国会再次认证。

被指控的六项违规行为是:

1. 彻头彻尾的选民违规
2. 选票处理不当
3. 可疑的程序违规
4. 违反平等保护条款
5. 不合规定的投票机
6. 重大的统计异常


纳瓦罗在报告里指责民主党政府官员在所有六个州的选举过程中参与了各种舞弊，而从州长、州议员到法官共和党官员却对没有做任何事情阻止他们， 媒体界也助纣为虐，企图淡化、嘲弄或忽视广泛存在的选民和选举舞弊的各种证据。反川普的主流媒体和社交媒体串通一气，拒绝调查越来越多的合法申诉，让美国公众了解不了真相，媒体的可信度、美国的政体和共和国的稳定都遭到破坏。

纳瓦罗还斥责“粉饰拜登”的记者、专家和政治领导人，要求他们必须承认已发现的六种选举违规行为，并在总统宣誓就职日前展开调查，还以民众2020年选举真相，否则在总统宣誓日时，不被大部分美国人民支持的非法总统上台有可能会上台。如果不能对选举违规行为进行积极全面调查，它将不仅反映出反川普的主流媒体和社交媒体的失败，也标志着美国立法和司法部门的失败。
