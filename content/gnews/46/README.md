# **中共国要求申请入境签证的人接种中共国产疫苗**

[!\[\]()!\[\](https://spark.adobe.com/page/8eh0nsh95suxH/images/b0c167bc-1cb4-466b-911f-e009bd3f75fc.png?asset_id=c705c1e8-2db7-491d-8536-7fa01f942cd2&amp;img_etag=%22108b5d74870be27d2d786aaba73914b4%22&amp;size=2560)](https://spark.adobe.com/page/8eh0nsh95suxH/images/b0c167bc-1cb4-466b-911f-e009bd3f75fc.png?asset_id=c705c1e8-2db7-491d-8536-7fa01f942cd2&amp;img_etag=%22108b5d74870be27d2d786aaba73914b4%22&amp;size=1024)上个月，人们在香港一个社区疫苗接种点排队接种中共国科兴疫苗

北京——从周六开始，中共国增加了在国际疫苗竞争方面的赌注，声称那些希望从香港进入中国大陆的外国人如果接种中共国产新冠疫苗，将被要求提交较少的文件。

美国、印度、日本和澳大利亚宣布计划向其他国家更广泛地提供疫苗，这四强国承诺到明年年底将资助印度生产至少10亿剂新冠病毒疫苗。在此声明发布后一天，中共国就宣布了这项针对所有申请前往中共国签证的外国人的政策。

尽管科学家们和外国政府一直在敦促中共国的疫苗制造商要在临床试验数据方面更加透明，但中共国仍试图增加其疫苗的国际吸引力。中共国政府发言人郭卫民（Guo Weimin）说，截至2月底，中国已经向69个国家发送了疫苗，并开始向28个国家进行商业出口。

中共国官方媒体机构也已经开始了一场误导的宣传活动，质疑莫德纳（Moderna）和辉瑞-生物技术（Pfizer-BioNTech）所产疫苗的安全性，同时，宣称中共国疫苗是更好的替代品。

尽管匈牙利已同意购买500万剂中共国产疫苗，但是，中共国制造的疫苗目前尚未得到西方大多数监管机构的批准。中共国也尚未批准在中共境内生产或销售外国疫苗。

本周，中共国向其国民推出了一种国际性的电子护照，可以显示旅行者是否接种过新冠病毒疫苗。目前还不清楚中共国外交部在周六宣布的政策对居住在香港的外国人有多大影响，因为中共国最近几乎没有对这些人发放过签证。

此外，香港的边境已经对非本地居民关闭了近一年。所以新政策对很多在其他国家因工作或家庭原因想回中国大陆的外国人非常不利。

香港政府允许居民在中共国大陆产的科兴（Sinovac）疫苗和德国进口的辉瑞-生物技术（Pfizer-BioNTech）疫苗中进行选择。上周六的公告中没有说明已经接种过辉瑞-生物技术疫苗的香港人是否需要再次接种科兴疫苗。

中国美国商会主席艾伦·比比（Alan Beebe）表示，边境限制已成为在中共国境内做生意的跨国公司最关心的问题，他质疑根据旅行者选择哪种疫苗来限制入境的行为是否有必要。

他说：“我们并不清楚进口的疫苗和中共国生产的疫苗有什么区别。”
