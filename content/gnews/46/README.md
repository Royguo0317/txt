**西班牙2021年2月9日电/西喜社——**挪威已将全球旅行限制延长至4月15日，建议不要进行非必要的旅行，并敦促人们复活节假期留在家里。

外交部说：“全球的感染状况意味着许多国家对入境采取了非常严格的限制，并采取了广泛的感染控制措施。”

外交大臣伊内·埃里克森·苏雷德（Ine Eriksen Soereide）告诉挪威新闻社，“我们离正常情况还很遥远，世界许多地方的感染情况正在迅速改变。”

爱沙尼亚周二取消了一些旅行限制，称来自保加利亚，芬兰，希腊，冰岛和挪威的旅行者将不受2月15日开始的10天检疫的限制。
