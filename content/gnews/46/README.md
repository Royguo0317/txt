#### **译评：**

澳大利亚曾是中国留学生的热土，据澳大利亚教育部统计数据显示，从2016年到2019年，中国内地留学生在澳大利亚注册数逐年增长，2016年为19.5万多人，2019年达到25.9万人。不过，这一趋势自2020年开始有所下降，2020年注册数为22.9万人。

由于澳大利亚牵头表示要对中共病毒进行独立的国际调查，中澳关系迅速变冷，中共不但在煤炭、红酒、龙虾等进出口贸易上打击澳大利亚的经济，还不遗余力地试图对澳大利亚的留学教育产业进行打压。时隔半年，中共教育部2月5日再次发出对澳大利亚的留学预警，这是4年内教育部第3次发出对澳大利亚的留学预警，而最近两次预警时隔仅半年。

同时，由于中共病毒大流行的影响，澳大利亚限制国际旅行和以学生签证进入澳大利亚，来自中国的留澳学生人数骤降。另一方面，澳大利亚政府放宽了在澳的学生签证持有者的工作时限，以便这些留学生可以更好地维持生活和学习。

这次中共教育部以种族主义为名再次发布虚假信息，夸大澳大利亚的种族主义，误导希望留学澳大利亚的中国学生家庭，这只是中共一贯的做法，因为中共一向擅长以种族主义煽动各个国家和各个种族之间的对立情绪。我们应该警惕中共的这种卑鄙行径，反对中共试图挑起澳大利亚人和华人之间种族对立的做法，只有全世界人民站起来消灭中共，全人类才能够真正的和谐相处。

#### **译文：**

（澳大利亚）联邦教育部长阿兰·塔吉（Alan Tudge）指控北京在警告学生不要去澳大利亚留学之后又散布了“虚假信息”。

据报道，中共教育部周五告诉（中国）学生们要根据种族主义、反华袭击和新冠病毒大流行的报道，对赴澳大利亚留学进行“全面风险评估”。

但是塔奇先生反驳了（中共）这一说法，称澳大利亚是国际学生最安全和最热情的目的地之一。他说：“我拒绝中共武断的言论，在澳大利亚访问或学习不安全就是虚假信息。”

“澳大利亚比其它大多数国家都拥有更高的标准，我们不容忍种族主义，当然我们也不容忍暴力。”

这是八个月来（中共）当局第二次告诉（中国）学生，由于种族主义他们要认真考虑去澳大利亚学习。此举可能使本已紧张的贸易关系更加恶化。

中共国有媒体报道说，一月份墨尔本、悉尼和堪培拉连续三天发生了“一系列针对中国学生的恶性袭击”。《环球时报》写到，这些事件对中国学生的人身安全构成了“严重威胁”，而冠状病毒大流行也使国际旅行面临风险。

澳洲八校联盟（译者注：又称澳大利亚八大名校，是由澳大利亚历史悠久、享誉国际的八所顶尖研究型综合大学组成的组织）的首席执行官维姬·汤姆森（Vicki Thomson）表示，关于澳大利亚是一个不安全的学习目的地的报道是令人失望的。汤姆森女士说：“我们肯定会像过去一样，为学生已遭受到的种族攻击向中共教育部和大使馆寻求建议。”

“我们任何一个学生都不应该容忍这些。我们的大学为所有学生都提供了安全的地方。”她说，包括墨尔本大学在内的八所大学对他们的中国学生都做出了承诺，并且八校联盟认识到他们保持离岸在线学习“有多困难”。

澳大利亚的大学严重依赖中国籍学生，他们是国际学生中人数最多的群体。然而，澳大利亚公民希望从海外返回的数量一直在增加，以及有限的检疫能力，这也给将国际学生用一架架的飞机运回到全国校园的工作带来了麻烦。

塔奇先生说，由于澳大利亚成功地应对了病毒大流行，他对澳大利亚继续作为所有国家国际学生一个有吸引力的目的地仍然抱有信心。

但是，英国和加拿大作为中国学生希望出国留学的替代性已经增加了。

《环球时报》报道说，教育部门的预警就是澳大利亚“毒害”了与中共国关系的证据。

驻上海的澳大利亚学者陈宏（音译）说——此人已经在经过情报调查后被澳大利亚官员取消了签证——中国学生在澳大利亚面临的“令人担忧的歧视问题”已经达到“惊人的高度”。他说：“澳大利亚政府对中共的持续攻击被媒体回响放大，尤其是在COVID-19大流行之后，已经误导澳大利亚当地人对中国人产生敌意。”

斯坎伦基金会本周发布的社会凝聚力研究发现，澳大利亚2020年对亚裔人士的负面看法“相对较高”。五分之三的澳大利亚华人回复认为，在COVID-19大流行期间澳大利亚的种族主义是一个“很大”或“相当大的问题”。
