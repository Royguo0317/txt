澎湃网11月3日报道，上海票据交易所正式上线跨境人民币贸易融资转让服务平台，标志中共官方机构公开参与货币战争。

该平台首批参与机构64家（其中境内24家），上线首日的交易金额达10.44亿元。自中共掀起货币战争，挑战美元国际货币市场地位以来，一方面是利用大量印刷“假钞”等方式，造成美元市场的通胀，降低美元信誉；另一方面则是通过与邪恶轴心国和蓝金黄势力勾兑，扩大人民币的交易份额，2019年人民币跨境收付金额达19.67万亿。

跨境人民币交易平台的上线，是中共货币扩张锋利的爪牙的公开亮相，也是货币战争更加激烈的标志。中共官方机构的出场一则说明随着美国大选临近，中共只能提前亮出底牌，“人民币国际化”只能仓促公开亮相，即便跨境人民币交易金额可观，且有增长趋势，但其在国际货币支付体系中的劣势地位（约占国际货币市场的1%）仍决定了，人民币主导国际交易的幻想仍是“幼蛇吞象”、痴心妄想。二则中共盗国贼通过此举，可大规模地“合法”转移其资产为美元资产，有了“树倒猴孙散”的即视感。

近年来中共利用明面上的国与国交易人民币和控制暗网以推行虚拟货币，但由于其体量十分有限（交易份额和市场信誉），且中共对法律肆意修改及其落后的技术水平，仍严重限制了人民币的“普及”和区块链的推行。所以中共在向外拓展的同时，更将重心放在国内搜刮民财上，利用数字货币、强征暴敛等手段强制开发区块链市场，更要以此裹挟全世界。

虽然中共布局长久，但其本身的缺陷导致根基不稳、扩张步伐虚大，限制着其渗透国际货币市场谋算的施行。随着爆料革命对中共邪恶的本质、企图霸凌全球的行径的揭露，促使美国和越来越多的西方国家的醒悟，不仅“一带一路”胎死腹中，区块链更遭到联合抵制，再加上美国对中共等邪恶轴心国的军事、经济、科技等立体围剿，中共企图将人民币国际化的动作只是垂死挣扎。
