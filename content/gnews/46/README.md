# 
在大瘟疫开始在全世界迅速蔓延的时候，中共国隐瞒了每天数千例新冠病例确诊，检测结果三周后才出来

- 湖北省疾控中心泄露的文件显示，中共国的系统需要平均23天才能返回测试结果。
- 2月10日，中共国官员报告了2478例中共病毒病例，但是在湖北—武汉所在地，当地官员有当天秘密的5918例病例的名单。
- 与内部文件相比较，中共政府也瞒报了死亡数据。
- 最新曝光的文件显示，去年12月，湖北省爆发了一场此前未披露的流感疫情。
- 中共国务院在六月声称，中共政府一直以“及时、公开、透明”的方式披露冠状病毒信息。


新曝光的文件显示，没准备好的中共国淡化了中共病毒在2月的传播，在疫情迅速传播到全世界之时，对每天确诊的数千例病例保密。

据泄露给CNN的湖北省疾控中心的机密文件显示，冠状病毒检测结果平均需要23天以上的时间，而此前未公开的流感爆发在12月份，重创了未来的疫情中心。

中共国政府一直坚称，关于新冠病毒的公开声明是透明的。去年年底，新冠病毒首次在湖北省武汉市被发现。

中共官员在2月的一天内报告了2478例中共病毒病例，但是武汉所在地湖北省的官员有当天5918例保密的感染人员名单。图中医疗人员走出武汉红十字会医院，摄于1月25日。

中共总书记习近平在2020年早期中共疫情危机上面临合法性危机。分析人士指出，虽然他很强硬地封锁了整个国家，但是中共政府瞒报坏消息本性导致了非常严重的数据低估。

中共国务院在六月声称，政府一直以“及时、公开、透明”的方式披露冠状病毒信息。

但是在2月10日一天之内 ，中共政府报告了2478例新中共病毒感染病例，而湖北省的官员有当天保密的5918例感染名单。

从该文件中看出，很明显中共政府瞒报了中共病毒的死亡数据。

在2月17日的中共病毒死亡病例是176例。但是湖北省公开的是93例。

虽然中共政府在3月7日总共公开了2986例自爆发以来的死亡病例，但内部文件显示死亡人数为3456人，其中包含了647例“临床确诊”死亡和126例“疑似”死亡。

CNN报道，中共政府使用“疑似病例”的名称来混淆真实感染病毒的数据。

分析人士告诉CNN，向公众报告的数字与湖北省官员能够接触到的数字之间的差异，可能是由于政府倾向于压制坏消息，以及报告系统存在缺陷。

美国外交关系协会全球卫生高级研究员黄延忠（音译）告诉CNN ：“很明显，他们确实犯了错误——不仅仅是在对付一种新型病毒时才会犯的错误——他们在处理问题时也犯了官僚主义和政治动机方面的错误。”

所有这些瞒报都是在中共领导人习近平采取强硬措施封锁整个国家之后。超过7亿人因为政府使用复杂的监控技术实行封锁而被关在家中。

在一个人们用个人自由换取稳定和日益繁荣的国家，共产党的合法性岌岌可危。冠状病毒威胁到了这一切。

尽管湖北官员自述他们对中共病毒采取了有效的应对措施，但文件显示，在3月测试结果超过三周才能出来，导致人们几乎无法确定中共病毒是如何传播的。

测试结果出来时，只有30%至50%确诊病例被测出阳性—表明绝大多数测试存在“假阴性。”

根据《柳叶刀》报道，医生早在去年12月1日就发现了首例中共病毒病患。据CNN报道，同样那周，湖北省发现传统流感比例相比去年同期增长了2000%。

咸宁和宜昌是两个离武汉很近的城市，被流感冲击最严重。

目前尚不清楚中共病毒和流感的联系。

今年自从病毒被发现以来，中共病毒疫情在全球感染了超过6300万人，接近150万人丧生。

该爆料人的行为让人想起李文亮医生警告民众关于新型冠状病毒在武汉爆发的事情。

12月30日，作为一名眼科医生，李文亮因在社交媒体上发布“武汉海鲜市场非典”的警告而被警方训诫，并被指控传播“假新闻”，这引起了公众的关注。
