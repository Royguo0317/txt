**简评：**

在之前川普政府时期，三大中共国龙头电信集团：中国移动、中国电信和中国联通因与中共军方有关联，已被两次退市。而在拜登总统上任后的第二天，这三家公司就向纽交所和新政府提出申请，要求重新上市。而“应对当前中共国危险委员会”坚信不应这样做。鉴于纽约证交所可能正在考虑将这些公司重新上市，“当委会”主席肯尼迪根据一家独立研究公司——RWR咨询集团的一项新的分析结果，写信给纽约证券交易所总裁和董事会，同时将信件副本发送给拜登总统和他的相关高级下属，力图阻止中国共产党利用数以百万计几乎不知情的美国投资者的投资资源，投资到中共对美国日益增长的军事威胁上。

中国移动、中国电信和中国联通与华为、中兴等企业一样，本质上都是参与中共国国家战略计划的军民融合型企业。RWR咨询集团报告显示，这三家集团都为中国共产党武装部队提供支持，从事军事和网络间谍活动、南中国海建设活动以及在受美国制裁的国家的不法活动。这不仅不符合美国国家安全利益，也为全世界民主国家所不容。美国已意识到这些公司在纽交所上市给美国国家安全带来的巨大威胁，但从之前川普政府努力迫使这三家公司退市的过程中，所受到来自美国财政部和纽交所的多重阻挠来看，在中共“蓝金黄”政策的收买下，华盛顿的亲共派和反共派之间的斗争非常激烈。

这次中共要求新政府批准其公司重新上市，仍然面对华盛顿各方势力的较量。鉴于“当委会”的权威性，希望它在这一关键时刻发出的这封信能有效地提醒拜登政府、纽交所以及华盛顿各方势力，在做出决定前需要三思而后行，避免因出卖美国利益而付出无法承受的代价。现在中共已被美国前政府宣布犯下了“种族灭绝罪”和“反人类罪”，事实上已经等同于一个“恐怖组织”，目前全球灭共呼声越来越高，美国国内及全世界都在观察和等待美国政府的进一步行动。拜登政府如果想与中共勾兑，恐怕也有心无力。极有可能招来国内的反对甚至弹劾，也会面对国际社会的谴责。

# **发布：委员会敦促纽约证券交易所不要帮助中共军方公司承销证券**

[!\[\]()!\[\](https://spark.adobe.com/page/MybvfatsgChP2/images/87be7bee-a582-4407-ac4d-d9e1478cfcb6.jpg?asset_id=ab5799d2-6cb5-4f10-9a5b-ae6625f2bff2&amp;img_etag=%22b6c6694ffcc340210b025f88aaf1eb2f%22&amp;size=2560)](https://spark.adobe.com/page/MybvfatsgChP2/images/87be7bee-a582-4407-ac4d-d9e1478cfcb6.jpg?asset_id=ab5799d2-6cb5-4f10-9a5b-ae6625f2bff2&amp;img_etag=%22b6c6694ffcc340210b025f88aaf1eb2f%22&amp;size=1024)

**呼吁关注表明中共的电信公司与中共国人民解放军的威胁有危险联系的研究**

华盛顿特区 — “应对当前中共国危险委员会”今天加倍了长期以来的努力，以阻止中国共产党（CCP）利用数以百万计几乎不知情的美国投资者的投资资源，投资到中共对美国日益增长的军事威胁上。**委员会主席布莱恩•特拉维斯•肯尼迪（Brian Travis Kennedy）**就独立研究公司**RWR咨询集团**进行的一项新的分析结果，写信给纽约证券交易所总裁和董事会，该分析记录了三大中共国电信集团：**中国移动、中国电信和中国联通**向中共武装部队提供了物质帮助。纽约证交所（NYSE）可能正在考虑将这些公司重新上市，而“应对当前中共国危险委员会”（CPDC）坚信它不应该这样做。

肯尼迪先生给**交易所主席史黛西•坎宁安（Stacey Cunningham）**的信——以及发送给拜登总统和他的相关高级下属的副本——指出纽约证交所三家中共电信公司近来被制裁的历史：

*2020年11月11日，川普总统发布了13959号行政命令，该命令禁止全球任何美国人和公司实体在一年内持有这些在五角大楼名单上，被认定为与中共国人民解放军（PLA）有关的中共国公司证券。*

*在此类隶属于中共国人民解放军的公司中，最重要的是三家大型电信公司：中国移动、中国电信和中国联通。由于这些公司被列入五角大楼的中共军工企业（CCMC）名单，并发布了13959号行政命令，您的交易所决定于上个月将其退市，这是值得称赞的。*

*不幸的是，美国财政部的矛盾指导促使纽交所撤回了决定，并重新在上市名单中列出了这些电信公司，但在联邦政府进一步澄清了川普总统于2021年1月13日修订的该命令的意图和影响后，纽交所又再次将其退市。*

*如您所知，拜登总统上任后的第二天，中国移动、中国电信和中国联通就向您的交易所和新政府提出请愿，要求重新上市。在您评估此请求时，我们希望确保您了解由独立研究公司RWR咨询集团发布的一项重要新报告，题为“与军方有关的中共国主要国有电信公司：中国移动、中国联通、中国电信”。该报告提炼了有关这些公司与中国共产党武装部队有令人担忧的关系并支持它们的性质的开源信息。*

肯尼迪先生接着提供了RWR咨询集团分析的重点，包括：

*这三家公司都有记录显示，参与了中共国国家战略计划（军民融合）和战略基础设施项目（南中国海网络建设），以及能够审查（在选定区域中止服务）和涉嫌参与恶意网络活动（例如互联网流量的重定向）。*

*这种不符合美国国家安全利益的具体例子包括：军事和网络间谍活动、南中国海建设活动以及在受美国制裁的国家（特别是朝鲜）中的活动。*

肯尼迪主席最后呼吁美国首要资本市场领导层：

*这些信息和经修订的第13959号行政命令，“应对当前中共国危险委员会”敦促纽约证券交易所以最强硬的方式继续阻止属于中共军工联合体的三大龙头电信公司：中国移动、中国电信和中国联通上市。*
