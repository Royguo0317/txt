## 【快讯】广东红海大桥建成3年未通车，“面子工程”谁来背锅？

新浪网12月15日转载央视报道，对广东省红海大桥在建成后被闲置3年情况进行梳理。

红海大桥地处东莞、惠州、广州三市交汇区域，是连通东江南北岸的公路桥梁，主要连接东莞石龙镇、惠州石湾镇、广州增城石滩镇三地，总长为814米（其中桥梁长为684米），总投资约1.2亿元，项目于2013年开工，2017年建成并完成验收交工。但却闲置至今，大桥东侧往惠州石湾方向的桥头双向路段被围墙封堵；大桥北侧左连石滩镇，仍是农田没有路，而右侧连接石湾镇，硬底化道路和泥路，都设有路障，是“断头路”。

石湾镇政府称，为配合贯通红海大桥，该侧的红海路（全长约1008米）于2015年修建，并于2016年初建成通车。而根据《博罗县石湾镇与东莞市石龙镇交通道路网规划对接图》显示，未通车的石龙段（已规划未实施，长约约1120米）在东莞石龙镇辖区内，该段被囊括在石龙铁路集装箱办理站项目中，所以一直未能动工。

红海大桥北侧的377亩地，连同周边共计2913亩地被划归为广东石龙铁路国际物流基地产业园区，是“一带一路”重要项目。2015年9月东莞石龙镇政府、中外运和俄罗斯阿尔法运输公司三方签署协议，推进上述中俄贸易产业园建设。

根据《广东石龙铁路国际物流基地产业园区总体规划2017-2035》，红海大桥及北端区域整体规划物流产业园，在红海大桥完工验收期间，石龙铁路集装箱办理站项目刚刚完成临时水电接入，而红海大桥和配套道路建设（隶属产业园建设项目）分属不同项目，产业园的路网未能具体完善，导致大桥完工3年来却迟迟未能通车。

红海大桥在建成以来，就由于石龙镇配套连接道路未建成而成“断头桥”，但地方政府却罔顾这一事实。在2016年6月大桥合龙后，东莞石龙镇政府某负责人曾表示，红海大桥将实现“一桥连三地”格局；2018年东莞市政府工作报告中也提及红海大桥项目完成通车。事实上的“断头桥”却在宣传中变成了“已通车”。

2015年9月，在习包子普京见证下，由广东省政府、中外运、俄开行、俄罗斯出口中心共同推动位于石龙的中俄贸易产业园合作项目，后续由于项目的“重要性”和层级颇高，地方上无任何管辖权，最终导致红海大桥北侧路段规划和实施被搁置。

闲置3年后，媒体又重提旧事。在舆论压力下，对大桥未通车事由，三地政府仍在扯皮甩锅。包括石湾镇政府口中的“2018年通车的红海路”，却被爆刚刚完成主路面硬底化铺设；石龙镇政府将北侧连接道路的未动工的原因为石龙铁路集装箱办理站项目路网规划未完善；而相关记者也好巧不巧的不再深挖这一所谓“一带一路”项目的进度。

后续：目前，惠州、东莞、广州三地协调后，在红海大桥北侧东西各修建一条临时道路连接大桥，惠州博罗县交通运输局和石湾镇正在修建一条临时道路连接石龙，确保在2020年12月30日前完工通车，并在2021年5月1日前连通红海大桥与石湾镇红海路。

由于东莞市签署的“一带一路”项目路网未规划实施，造成大桥不能通车，而现在却由石湾镇和石滩镇修建临时道路做弥补，“面子工程”的黑锅最终还是落到了级别更低的组织层次上，而“始作俑者”的物流园得以刨除在外。
