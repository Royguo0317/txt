# **中共国为力争保护北京，在香港实施史无前例的CCP病毒封锁**

有迹象表明，目前在中共国东北地区再次爆发的新冠病毒疫情可能比中共国（及其附属的党媒）所宣称的更为广泛，香港上周五刚刚宣布了有史以来的首次封锁，这一措施让人想起了近20年前打击非典（SARS）疫情所采取的更为严厉的措施，非典（SARS）疫情曾重创了这个小小的（正式的）自治城市。

[!\[\]()!\[\](https://spark.adobe.com/page/X18OKft52c6yr/images/3c94d329-2d8c-4479-bc1a-b00a0961aaf0.png?asset_id=b280ffa6-b697-4f8f-8402-0255287fe098&amp;img_etag=%22cec2b2b9d25bf063367af4e1274673f2%22&amp;size=2560)](https://spark.adobe.com/page/X18OKft52c6yr/images/3c94d329-2d8c-4479-bc1a-b00a0961aaf0.png?asset_id=b280ffa6-b697-4f8f-8402-0255287fe098&amp;img_etag=%22cec2b2b9d25bf063367af4e1274673f2%22&amp;size=1024)

据《南华早报》报道，香港将封锁受冠状病毒影响的油尖旺区约150栋住宅楼。为了执行这项措施，市府将出动超过1700名 “纪律服务人员”—-500名来自警方，其余来自海关、入境处、消防部门和惩教部门–以 “史无前例 “的方式保护这个塞满了老化的分隔屋和公寓的社区。与中共国其他地方一样，最近几周香港的新病例一直在上升，周五该城市新发现了61例冠状病毒病例，其中55例为本地传播病例。26例被认为 “无法追踪”。另有50人检测初步阳性，正在等待确认。

预计封锁将于午夜时分在该区指定的强制检测区域开始。早前有消息澄清，深水埗区原本被纳入计划，但目前还未受影响。

[!\[\]()!\[\](https://spark.adobe.com/page/X18OKft52c6yr/images/edbb365c-4310-4657-89ff-809f5154dce4.png?asset_id=dcdb11a3-9542-48de-93d3-1d4d3ba531a6&amp;img_etag=%22eb8d833916b9c0b73d384c02823ffb5e%22&amp;size=2560)](https://spark.adobe.com/page/X18OKft52c6yr/images/edbb365c-4310-4657-89ff-809f5154dce4.png?asset_id=dcdb11a3-9542-48de-93d3-1d4d3ba531a6&amp;img_etag=%22eb8d833916b9c0b73d384c02823ffb5e%22&amp;size=1024)

**在中共国其他地方，北京和湖北当局正在组织更多轮的强制性CCP病毒测试，以覆盖数百万人，同时他们继续力争把病毒阻挡在北京城外。**

以下是南华早报对于新的封锁措施将带来什么影响的快速分析。看起来，这次封锁甚至比英国的紧急等级制度还要严格，而英国的措施已经引起了英国人民的极大不安：

**封锁在哪里？**

上周划定的强制检测区，以弥敦道、佐敦道、锦绣街和渡轮街为界，预计在周五午夜前实行封锁，等待官方公布消息。从周六早上开始，住在封锁线内约150栋楼宇的居民将不得离开该区域，直到限制解除为止，最快也要等到周日。

在全市750万人口中，约有4%的人口居住在整个油尖旺区。根据官方人口普查数据，该区涵盖油麻地、尖沙咀、旺角和佐敦等地区，是全市18区中人口第二多的地区。被封锁的区域位于该区中部，沿弥敦道大道。

[!\[\]()!\[\](https://spark.adobe.com/page/X18OKft52c6yr/images/d5b51489-8efd-4f59-8751-08e015983710.jpg?asset_id=6e1780f1-3030-4fe5-8e5c-259198ce151a&amp;img_etag=%2237f1a3d802756862b4709dfc5d5e8243%22&amp;size=2560)](https://spark.adobe.com/page/X18OKft52c6yr/images/d5b51489-8efd-4f59-8751-08e015983710.jpg?asset_id=6e1780f1-3030-4fe5-8e5c-259198ce151a&amp;img_etag=%2237f1a3d802756862b4709dfc5d5e8243%22&amp;size=1024)

该地区以著名的旅游景点而闻名，比如庙街。庙街有一个繁华的夜市，其中一部分是受影响最严重的区域，被禁止外出。不过，区内其他景点，如女人街的纪念品摊位、旺角的花市，以及被称为 “果栏”的水果批发市场等，都没有被封锁。

虽然被封锁的区域涉及九龙的旧住宅区，但由于该市的高密度情况，因此封锁区域靠近国际公司聚集的华丽购物中心和摩天大楼。

**谁会受到影响？又会有哪些限制？**

[!\[\]()!\[\](https://spark.adobe.com/page/X18OKft52c6yr/images/dc6fbd27-d1be-4c34-ac1a-ac1344409098.jpg?asset_id=15fef0f7-af5a-4aa2-bcb2-224608cf303e&amp;img_etag=%22fa7fd784a0270184f7fe4c08460326a5%22&amp;size=2560)](https://spark.adobe.com/page/X18OKft52c6yr/images/dc6fbd27-d1be-4c34-ac1a-ac1344409098.jpg?asset_id=15fef0f7-af5a-4aa2-bcb2-224608cf303e&amp;img_etag=%22fa7fd784a0270184f7fe4c08460326a5%22&amp;size=1024)

只有三类人可以进入该区域：居民、与需要照顾的人住在同一单位的亲属，以及急需部门的工作人员，如老人护理。

对于需要看病或面临身体伤害的居民，将获得豁免。

一名政府消息人士表示，即使是那些曾经在强制检查中被通知冠状病毒检测呈阴性的居民，也会被要求尽可能地留在家中，而不能在封锁区内走动。

[!\[\]()!\[\](https://spark.adobe.com/page/X18OKft52c6yr/images/34582a80-e84c-4a8f-a2ac-af196dfc9dd8.jpg?asset_id=8dfba09b-9961-43c8-a820-4cc432f33eac&amp;img_etag=%229531fa3533b65ebb2b951c0a76f6a2b6%22&amp;size=2560)](https://spark.adobe.com/page/X18OKft52c6yr/images/34582a80-e84c-4a8f-a2ac-af196dfc9dd8.jpg?asset_id=8dfba09b-9961-43c8-a820-4cc432f33eac&amp;img_etag=%229531fa3533b65ebb2b951c0a76f6a2b6%22&amp;size=1024)

如果店主不住在指定区域内，指定区域内的商店将继续关闭，政府雇员将为有需要的居民提供食物和日用品。

**禁闭令有必要吗？**

呼吸道疾病专家梁志超博士建议，在最初发布强制命令的同时，就应该实施封锁令，限制无症状携带者在特定区域内的流动。

“当出现非常局部的疫情时，在病例刚出现的初期就实施封锁，可以防止感染蔓延到其他地区。”梁志超说。

他认为现在发出命令为时已晚，因为居住在冠状病毒热点地区的病毒携带者可能在日常生活中把病毒传播到其他地方。

佐敦北区议员何富荣（Frank）表示，在最后一刻发布命令，让该地区的居民毫无准备。

我们也将密切关注任何香港的疫情发展，随着中共国各地疫情的恶化—-这将有助于支持北京的 “说法”，即最新一波病毒从中共国境外传播回境内–无论这是否准确。
