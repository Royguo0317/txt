**01/05/2021路德时评（路安墨谈）**

**1、    习近平依然无法露脸只能签发文件**

1月5日习近平签署任免驻外大使指令依然没有露脸。据情报透露这次介入手术后习至少休息13天以上，11日都可能无法出席与马克龙的视频会议。这次路德社披露习手术的准确消息，主要是告诉习：很多事情是爆料革命掌握的，包括习对生物武器的一系列指示、签名和语音都是有情报证实的。

**2、    纽交所突然宣布取消中共三大电信公司退市**

纽交所原计划在1月11日采取对中共国三大电信营运商采取强制退市行动，4日突然宣布撤回退市决定。主要原因是拜登称一旦正式就职总统，三周内取消川普总统的禁令。纽交所是否已经被中共控制的话题，引起美国民众的热议。这件事情暴露出中共在美国各个层面的影响力如此强大，当然中共的这种行为越多越有利于川普总统接下来有关大选的谈判，因为中共邪恶势力对美国的威胁以及与中共勾结的美国利益集团的暴露反而让美国人警醒。

**3、    中共为什么要渗透梵蒂冈教廷**

天主教有近两千多年的历史，其有资金、组织和情报。几千年的积累让梵蒂冈拥有了巨大的资金；天主教在全球有严密的组织和秘密力量，这个组织保证了梵蒂冈拥有全球统治的软实力。分布全球的组织也让教会拥有了强大的情报能力。中共只有七十年，虽然继承了共产国际隐形势力，但要想控制全球就必须控制梵蒂冈，中共一直以来都在进行颠覆梵蒂冈的行动。教宗被共产党所控制和勾结，成为邪恶势力的一部分，这不是宗教的错，而是人的错，宗教不能完全代替信仰。

**4、    中共已经在准备贺锦丽访华**

习近平的团队正在准备贺锦丽1月28日对中国的访问。1月20日的选举结果还没有出来，贺锦丽也没有辞掉参议员的职位。届时是以副总统身份还是参议员身份访华？如果是副总统身份到了北京，就可以宣布对中共的制裁全部取消，又重新回到全球化时代，然后和中共继续共同治理全球。中共如此迫不及待的提前准备贺锦丽访华，更进一步揭示出拜登整个利益集团与中共盗国贼的紧密关系。

**5、    华尔街与梵蒂冈的利益**

华尔街的资金来源除了私募、各类基金外最大的来源其实是宗教资金。梵蒂冈通过千年的财富积累，大量的资金需要投资。深层教会和华尔街金融大鳄的合作促成了梵蒂冈与深层政府联合，这一勾结共同推出了全球化主义这个项目。川普总统的排干沼泽其实是打击了这一利益集团，同时也打击了他们的“大重启”计划；梵蒂冈、华尔街加上有共同利益的中共一致认为川普总统是主要敌人，是需要清除的障碍。
