**原文：**民，辱则贵爵，弱则尊官，贫则重赏。（选自《商君书·弱民第二十》）

**译文**：对于民众，羞辱他才会看中社会地位，削弱他才会尊重官员，贫穷了才会看重(君王的)赏赐。

**原文：**民有私荣，则贱列卑官；富则轻赏。治民羞辱以刑，战则战。（选自《商君书·弱民第二十》）

**译文：**民众有自己的价值尺度，就会轻视官位鄙视官员；民众富裕了，就不看重(君王的)恩赏。用严刑酷法羞辱的方式治理民众，战争时他们才会出力死战。

**解读**：除了愚民、壹民、弱民外，商鞅还要羞辱民众。商鞅是一个对人性有着深刻理解和把握的人，当代西方社会研究的**斯德哥尔摩综合征**，商鞅早在2300年前就已经洞悉。他认为对民众就是要打压、羞辱，然后给他们一点好处，他们就会感恩不尽。如果这个好处是有限的，民众就会为了挣得这点有限的好处而争相讨好甚至赞美打压羞辱他们的人，而全然忘了他们本不该被打压和羞辱。民众被羞辱而没有尊严，不仅不会有威胁政权的想法和行为，专制统治者用一点小恩小惠就可以轻易对民众实施有效管制。

**专制社会用划分等级来体现人的高低贵贱，以此来羞辱民众**。商鞅时代，社会分级主要有贵族、平民和贱民（奴隶）之分，平民中又有士农工商的等级划分，之后又分为**三教九流**，所谓九流就是尊卑不同的九种身份，到了专制社会后期，等级划分越来越细，有上中下三个九流之分，**上九流**依次为：帝王，圣贤、隐士、童仙、文人、武士、农、工、商，**中九流**依次为：举子、医生、相命、画匠、书生、琴棋、僧、道、尼，**下九流**依次为：师爷、衙役、秤手、媒婆、走卒、巫、盗、窃、娼。在每个等级内也有细分，比如官场中分品级，每个品级中又有正和从之分，每个品级的服饰不一样。清朝皇帝的后宫也分皇后、皇贵妃、贵妃、妃、嫔、贵人、常在、答应等八级。就连娼妓也分为书寓、长三、幺二、野鸡四个等级。

将民众划分等级后，就有了尊卑贵贱之分，等级高的就有更多的尊严和财富，可以羞辱等级低的。比如官员见到帝王要跪拜，民见官要跪，官场中下级见到上级要主动行礼，要绝对服从，所谓官大一级压死人。平民要想成为官员就要参加科举，所谓“朝为田舍郎暮登天子堂”。士兵要想升级就得立战功。由于越高的等级人数越少，所以要想升级除了自身努力外，就得奉迎上级，就得忍受各种羞辱，就得耍诡计排斥同侪，一旦升级了，又对下级颐指气使，以求得心理的平衡。

中共篡政后立即开始搞“三反五反”、“镇压反革命”、反右等一系列运动，将上千万民众打成**“地富反坏右”**，也就是**“黑五类”**，将知识分子打成**“臭老九”**。文革期间将“黑五类”和“臭老九”公开批斗、戴上高帽游街、各种公开体罚和羞辱，很多知识分子如老舍、傅雷夫妇，演员如上官云珠，地下党如陈琏，等等，不堪羞辱而自杀。不仅他们本人，他们的家属上亿人受到波及，婚姻、工作、子女求学等受到极大影响，他们成了社会底层的贱民。文革后上千万知识青年学生被“上山下乡”，从城市到了贫穷的乡村，一代青年人的青春被毁。吊诡的是，文革后这些人被平反后，很多被迫害多年的人却为中共唱赞歌，很多当初上山下乡的青年还很怀念在农村的苦难岁月，号称“青春无悔”，上演了一场现代版的斯德哥尔摩综合征。

笔者幼时的邻居就是这样的人。他出身于书香门第，从小受到良好教育，会英俄日三门外语，五十年代是一个中学的高级教师，反右时30岁的他被打成右派，文革时继续被批斗，他太太被迫带着3个孩子和他分居。据他的孩子说文革时他经常被戴着高帽游街，他们几个孩子觉得没脸见人，到处被歧视，初中毕业就辍学，后来也没有机会接受高等教育，在工厂里做最艰苦的工作，婚姻也不顺利。我年幼时有印象，他在学校打扫厕所，每月只有8元工资，没有钱抽纸烟就买便宜的烟丝用纸裹着抽，如果有人给他一支纸烟，他会夹在耳朵上几天不舍得抽，即使抽到最后剩下的烟头里面的烟丝他也留着。住处不远有家医院的煎中药房，煎药用的燃煤灰中有没有烧尽的煤炭，他就将其捡回做燃料，药渣里面有谷物，他也拿回来喂鸡。1979年他被平反，补发工资重新安排工作和住房，后来还被选为当地的政协委员。此后他逢人就说中共伟大，能纠正错误，完全无视了他22年右派期间被羞辱的悲惨生活、被耽误的青春和子女。我长大后和他有过对话，让我感到悲哀的是，他对中共的感激是发自内心的，不是为了政治投机。他去世前我去看望过他，告诉他我送孩子到美国读中学，他还认真叮嘱说学成了一定要回国报效党和国家。

**中共还将全民分为农村户口和城市户口，城市户口中还分工人和干部**。我读初中时，很多农村来的同学学习很刻苦成绩很好，但他们初中毕业就考中专，放弃读高中上大学的机会，为的只是读中专就能有城市户口和干部身份。即使现在户籍制度还造成很大的不平等，很多农民工进城打工，做最艰苦的工作却没有城里人的医疗和养老保险，子女也不能在城里就读，造成骨肉分离，农村留守儿童现象严重。即使是在大城市工作的白领阶层，如果户口不在大城市，子女高考时也必须回原籍参加高考。

中共的**官本位**现象严重，再富有的商人，再有学问的学者，见到小官也得毕恭毕敬，酒桌上得让官员上座，敬酒时得先敬官员喝酒。官场上更是等级森严，中共的官员都有级别，同一个级别又有不同的权重，比如同为副厅级干部，市委常委的地位就比不是常委的副市长高，在中共国，领导人的排位顺序、出场的先后、主席台的座次、酒桌上的排座，是一个很大的学问。级别的高低还涉及车辆的安排、医疗待遇、阅读文件的待遇等等。改革开放后一切向权看、一切向钱看，商人为了从官员那里获得项目、低级官员为了讨好上级官员，除了金钱利益上的行贿，还奉献女色，**女人把自己奉献了，男人把老婆奉献了，有人甚至把女儿也奉献了！**被判死刑的原杭州市副市长许迈永供述有一半的女下属与其有染，被判死缓的原江苏盐城市市长徐其耀与医院护士长有染，护士长为了让市长给女儿安排一个好工作，居然主动把女儿也奉献给市长。山东青岛三个女律师为了从法院获得业务，居然同时陪法院副院长淫乱。

文人也不甘落后，主动向中共献媚。汶川地震后，以《文化苦旅》一书而出名的作家余秋雨公开发文，含泪劝告地震中失去子女亲人的民众不要向政府问责，不要上境外敌对势力的当。山东作协副主席王兆山还写了歪诗，面对6万多死难者，居然说出“十三亿人共一哭，纵做鬼也幸福“的句子。于是出现了两个新成语：**秋雨含泪，兆山羡鬼**。大学里也早就不是净土，很多女研究生为了学位论文获得通过拿到学位，不惜与年长的导师上床。

**中共治下的中国，普遍道德沦丧世风日下，在古代娼妓乃下九流之末，但在中共红朝却普遍存在“笑贫不笑娼“。中共还能做到让民众不以羞辱为辱，自辱，甚至认为如果没有自辱的机会就失去了升官发财的机会。中共不除，中国人民将永远生活在耻辱之中。**
