## 路德社重磅解读：鲍威尔律师大量转推方正信息

文章撰写：【重生】

《路德时评》2020年11月20日晚间节目深度解读鲍威尔大量转推方正信息及其重磅发言；美国务院正式发布《应对中共挑战蓝皮书》，详细内容如下：

一、鲍威尔重磅发言

近日，参与调查美国大选舞弊案的鲍威尔律师接受一家私人媒体采访，采访中放出众多重磅信息[1]：
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/21115544/image-29.jpeg)
1. 鲍威尔律师最新采访为回应极左媒体关于美军获取大选服务器的虚假报道；
2. 鲍威尔手上的有关选票造假证据像海啸般袭来；
3. 多米尼在多伦多和丹佛的办公室突然关闭并搬离，一百多名员工在领英上删除与多米尼的关系；
4. 美军已收缴德国的服务器，该服务器信息都被传至四个美国利益极端敌对国家；
5. 从本周末开始大量证据将如海啸般涌现出来。


从该新闻可以看到：

1. 鲍威尔也是当事人，曾是弗林将军的辩护律师，现在是川普总统个人律师，她被授权在媒体上发声；
2. 鲍威尔律师不仅拥有丰富的经验，还掌握大量大选舞弊案铁证，这给她极大的信心；
3. 采访中提及多米尼在多伦多的办公室与索罗斯有密切关系，他们同时撤离多个办公地点也说明这就是大规模的集体犯罪行为；
4. 多米尼公司的员工非常清楚该公司的行为，员工集体在领英上删除与多尼米公司的关系就是为担心他们未来的职业生涯，他们也非常清楚未来将发生何事，他们此举也是为了不让司法部寻找到证人；
5. 涉及美国大选的投票软件服务器本不应该将信息传送至四个与美国有严重利益冲突的国家，这些彻底威胁到美国国家安全；
6. 公司中存储核心信息的电脑最具有取证价值，投票软件程序代码中肯定有许多非法信息指令，他们就是  想在司法部调查举证前转移或销毁证据；
7. 加拿大和美国有引渡条款，文贵先生此前也说多伦多是中共的间谍中心，美国完全有能力展开调查；
8. 只要有外国势力涉案  危害美国国家安全，美国就可以用FISA法案对其进行监控来收集证据，即使他们销毁证据也无济于事；
9. 多米尼背后有方正集团参与，鲍威尔近期也转发多条涉及方正集团的推特，这条线索未来帮助调查中共如何影响美国总统大选；


二、美国务院正式发布《应对中共挑战蓝皮书》

美国务院正式发布《应对中共挑战蓝皮书》[2]，该份文件中总共74页，在此关键节点发布该份文件值得深思。其中内容包括如何应对中共威胁美国，渗透美国，干扰美国大选等信息。这是美国政策局经过长时间研究编写的文件，这份文件将奠定未来中美数十年的政策走向。

此文件可以看到：

1. 川普总统过往四年一直以贸易、经济、知识产权偷窃为主线让美国人民意识到中共的威胁，但中共对美国的威胁远不止在此层面，这份文件就清楚阐述中共如何在价值观层面对美国构成威胁；
2. 文中告诉我们中共通过国家资本的方式输入共产主义价值观，和全球左派一起打造一个全球中共模式；
3. 此前基辛格主义一直   主导中美的政策走向，此前中美建交是为   通共对抗苏联共产主义，现在中共变得比苏联还强大，比苏联还邪恶，他们直接向资本主义外壳中输入共产主义，这就是价值观之争，现在仍有许多美国人还未苏醒；
4. 该文件清楚阐述中共如何输出中共价值观，中共本质就是   国家资本主义，对内宣传民族主义，对外输出共产主义，这三种形式的结合，中共将国家资本的扩张和霸道说成是中国人的优势，将民族主义和共产主义融合说由中国人以共产国际领导世界，再通过建立“一带一路”，打着为   世界人民谋福利的口号将国家资本主义和共产主义融合；
5. 川普总统未来四年任期必定将清剿中共势力摆上台面，让所有美国人清醒地认识到中共就是敌人，所有政策都将瞄准中共


三、鲍威尔律师大量转推方正集团意味着什么？

鲍威尔近期大量转推涉及方正集团的信息，分析如下：

1. 多米尼系统只是指针，谁编写该软件，谁将其提供给委内瑞拉，又是谁允许该软件出口到其他国家，再结合郭文贵先生直播中重磅爆料方正在多米尼系统中的核心作用；
2. 鲍威尔律师大量转推方正集团的信息，就是指向方正集团就是多尼米软件的幕后操守，最终将指向中共；
3. 鲍威尔律师是主导此次美国大选舞弊案调查的最重要成员，她知道众多内幕，爆料革命第一时间告诉她方正科技掌握汉字源代码，随后她就立即转推，意义深远；
4. 中共第一届“国家最高科学技术奖”获得者就是北大方正的创始人王选[3]，他所领导的科研集体研制出的汉字激光照排系统为新闻、出版全过程的计算机化奠定了基础，始终享受常委级别的待遇；
5. 汉字语言代码对中共有非常重要的作用，只要网络上或系统使用到汉字，都得对它进行权限开放，这条重要情报内容只有爆料革命最早提出；
6. 文贵先生此前是方正集团的大股东，李友当时不惜一切代价争夺方正，就是因为方正集团拥有中共最核心的机密，中文编码是网络一大核心，只要使用中文字体，中文输入法，中共就可以对其进行监控。
7. 中文汉字处理系统当时由王选编写，当时就进入DOS系统中，只要想拥有此项功能就必须要这部分代码，但是这部分代码对微软系统不开源，是黑箱子，不论系统如何更新，该部分功能模块始终都存在于系统中。编译是单向，指令编译变成“0”和“1”二进制后开始工作，要再进行反编译非常困难，这次多米尼系统使用的是Windows XP系统，微软早已停止更新该系统，这些漏洞可供中共进行操控；


海外势力对美国的渗透、影响和操控已达到令人发指的地步，随着更多信息被公诸于众，将会有更多人看到中共的邪恶。
