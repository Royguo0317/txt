# **美国在2020年对中共国援助减少一半**

[!\[\]()!\[\](https://spark.adobe.com/page/BJITQ06Jxizss/images/9355b3ad-8209-429f-8513-40a430334cb7.jpg?asset_id=4362e53b-5c7e-49c7-9168-79f0cd52a32d&amp;img_etag=%22c0207b206ae61f3738214306876d32e3%22&amp;size=2560)](https://spark.adobe.com/page/BJITQ06Jxizss/images/9355b3ad-8209-429f-8513-40a430334cb7.jpg?asset_id=4362e53b-5c7e-49c7-9168-79f0cd52a32d&amp;img_etag=%22c0207b206ae61f3738214306876d32e3%22&amp;size=1024)

根据美国管理和预算办公室（Office of Management and Budget）的报告，美国在2020财年削减了对华援助3200万美元，从2019年的6200万美元降至3000万美元。

在川普进入总统任期的最后几天之际，发布了第一份涉及全政府部门的中共国支出报告。川普政府对华实施了激进的经济政策，以制止中国共产党在美国和全球市场上日益增长的影响力。

川普在2016年以打击中共国的经济政策参加竞选，他说中共国的经济政策通过贸易不平衡和操纵货币价值来“利用”美国公民。

去年，在新冠病毒大流行开始后，川普总统为遏制中共国在全球政治和市场中的影响而进行的努力升温：7月，川普因世界卫生组织未能令中共国对致命的新冠病毒疫情爆发承担责任而退出世界卫生组织。他对支持中共国军队并在联合国争取中共国影响力的中共国公司施加了更多制裁。此外，美国在2020财年对中共国进口商品征收600亿美元的关税。
