**新中国联邦和班农、蓬佩奥等美国正义力量联合揭露中共病毒生化武器真相**

2020年12月18号，据新中国联邦创始人郭文贵先生透露：美国和大多数西方相关机构已经认识到SARS-CoV-2病毒是一种人工制造的生物武器的本质。各国相关机构高度肯定了爆料革命在界定SARS-CoV-2病毒为生物武器本质方面的关键作用。

闫博士2020年1月19日通过路德社披露了SARS-CoV-2病毒的五大特点：传染性强、人与人之间传播、变异率高、从中共军事实验室爆发、中共故意放任病毒传播。之后闫丽梦博士进一步提供了两篇无可争议的科学论文，直接参与生物武器项目的一线顶尖科学家也提供了SARS-CoV-2病毒的科学证据。美国及其盟国相关机构在此事实、科学证据和情报的基础上，以极其谨慎、小心和确证的态度将SARS-CoV-2病毒定义为生物武器，是中共通过在解放军实验室制造并蓄意向全世界释放的。这个没有公开的声明，意味着以美国为首的西方盟国必将实施联合军事行动，铲除中共政权。

2021年1月14号郭文贵先生盖特视频又透露，SARS-CoV-2病毒来源已经推进到了美国官方：“美国官方已基本说明病毒来源于中共武汉实验室，来自武汉实验室就是武器，不存在故意或非故意释放。”

2021年1月初到20号之间，前川普政府内阁官员频繁在公开场合的说法也证实了郭先生所透露的信息：肯定了病毒来源于中共实验室。对此，纳瓦罗先生更是走在前面，已经公开使用“中共病毒（CCP Virus）”这个称呼，并将之与中共生物武器实验室联系在一起。

**拜登政府“战略忍耐” 中共趁机大肆搅局**

自1月20号拜登总统宣誓就职以来，拜登政府的外交政策，尚未定型，在这段空档时期，中共软硬兼施，试图通过无赖撒泼影响白宫对中共的政策。

军事上中共对台湾的军事施压活动大幅升级。上周中共军机连续三天挑衅台湾，27号开始又在南海军事演习；与此同时，习近平又在25号达沃斯世界经济论坛上对白宫伸出橄榄枝，呼吁走和平共处，互利共赢之路。

面对中共的挑衅，白宫发言人詹·莎琪（Jen Psaki） 25日依然在例行记者会上首提“战略忍耐（strategic patience）”以处理和北京的关系，这是拜登团队第一次明确提出对中共的战略，虽然比较模糊笼统，但对于中共外交人士来说，这相当于一个信号弹，预示了一个可预测的外交政策走向。可能在中共的字典里，忍耐等于放纵。这让中共理解为拜登政府将一改川普政府后期的“极限施压”政策，而拜登总统的首席疫情顾问福奇则宣布，美国将继续为世卫组织出资，川普政府退出该机构的计划被撤销。

这个“战略忍耐”政策来自于奥巴马政府。奥巴马时期，对朝鲜的核武危机一直采取的就是“战略忍耐”；结果朝鲜在奥巴马任内进行了4次核试验，占核试验总数的三分之二。而川普上任后，明确表示，对朝鲜的“战略忍耐”时代已经结束。川普转而对朝鲜采取“极限施压”政策，对朝鲜展开全方位的国际制裁和单边制裁，一直持续到今天，成功迫使朝鲜自2017年9月至今再无核试验。
