从天空俯视纽约曼哈顿岛，犹如一枚中国古代刀型钱币。当初印第安土著祖先们倒大方，仅以相当于24美元的小物件，就把曼哈顿岛的拥有权交换给了第一批荷兰探险者。这样的狗血事件在现今的财富神话里再次出现，第一批比特币拥有者里，有人用一万枚比特币换过几块比萨饼。看来，巨额财富都是起源于及不起眼的物品，无论是有形的曼哈顿岛，还是无形的比特币。

4月初的曼哈顿，天气如同股市般变幻无常。本已入春，一场阴冷的飓风暴雨，在哈德逊河交相汇盈，寒彻心骨。疫情后的街道冷清可鉴，在200 Vesey street, Amerrican Express Tower前，却迎来了一道亮丽火热的风景。那是一群GTV的投资者，秩序井然地站成一排，手持美国国旗，拉起标语横幅，喊着史上最强音：“SEC + CCP = Corruption”。

如果把华盛顿DC比作SEC的头脑，曼哈顿下城就是其心脏。这里精英荟萃，政治金融宗教民族交融，有纽约市政厅（想结婚的来此申请），百老汇，三一教堂，华尔街，原世贸大厦911原址，中国城，小意大利，纽约地铁交通枢纽的终点，也可说是起点。华尔街旁还有一所专门供华尔街精英们进修的Pace Univercity，笔者有幸从此校获得MBA。这块风水宝地就是美国金融与财富之巅。

GTV是美国合法注册的网站，投资者们遍布世界各地。其中有逃过CCP的各种封锁拦截，甚至生命威胁，冲破艰难险阻，把钱从中共国汇出，就是为了找到一个安全之地安放自己一生的心血，GTV给了他们希望与寄托，美国给了他们安全感。但一直以来与CCP利益勾兑狼狈为奸的SEC，偏信CCP，为了几个别有用心的GTV投资者的所谓告发谗言，从去年七月以来，以调查GTV为名一直压着GTV投资者的资金，使得GTV私募难以按期往前推进，投资者蒙受巨大损失。这是这群抗议者站出来的起因，也是他们喊出SEC与CCP沆瀣一气腐败勾兑的底气。

30几人冒着瓢泼大雨毫不退缩，五辆大型宣传车带着揭露真相的巨幅电视画面，以及六辆小车组成的车队在曼哈顿游行。此镜头计入了谷歌地图，可望不久会成为曼哈顿一景也未可知。来往车队向我们鸣笛，行人纷纷拍照或注视打听，分发传单几千份。笔者询问过来巡视的警察，大楼的管理人员，里面的员工，行人，几乎所有人都赞同GTV投资者的口号–SEC允许垃圾中概股在美横征暴敛美国人的财富，为了私利出卖美国人的利益，与CCP勾兑。这样的真相第一次由一群华裔为主的GTV投资者大声揭露出来，如果说911以恐怖战震惊美国人民，GTV投资者在SEC旧址（据说SEC已搬迁至华尔街）的抗议，就是震惊美国金融界的一枚炸弹。CCP，伪类们也在关注，派来的一个个乔装打扮的人员，要么与现场抗议者辩论不要丢华裔的脸，要么凑近车窗拍照；伪类们在媒体上大肆造谣GTV投资者们围攻SEC，等等。真相只能有一个，那就是SEC与CCP在勾兑。
