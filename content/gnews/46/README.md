11月5日媒体报道，一名美国邮政工作人员在加拿大边境被捕，其汽车后备箱藏有800多封未被投递的邮件和选票。

涉事人叫布兰盾∙威尔逊，今年27岁，来自纽约州的水牛城。其被捕和被起诉的罪名是“拖延和销毁邮件罪”。 如果罪名成立，他将面临最高五年刑期和25万美金罚款。

美国联邦检察官詹姆斯∙肯尼迪在起诉书中称：“邮政部门不仅有义务保证邮件的完整性而且必须将每个人的选票及时送达以保证大选公平公正性。而威尔逊的行为损害了公民的以上两项权益。“

报道称，2020年11月3号大选日晚上大约7：34分，在和平桥美加过境处，美国海关及边境保护局工作人员在对威尔逊的车辆进行例行检查时发现了一个邮政专用箱，上面有邮政专用标记并标有威尔逊本人名字， 内有大量邮件。 威尔逊称“这些邮件是寄给他和他母亲的，但却无法解释印在邮件上的寄信人名字。之后他又辩称他本来是”打算把这些邮件投递出去，但忘了把它们送到邮局。”

美国邮政局总检查官办公室被要求追查这批被截获的邮件。调查显示其中包括3张邮寄选票是发自纽约伊利县寄往水牛城，106件竞选宣传品，220封快递信件和484封普通信件，总共约813封邮件。快递信件寄往邮政编码14227、14211、14214，其它大部分邮寄目的地为14215。取消日期显示为2020年9月16日至2020年10月26日期间的七个日期。

负责这起案件调查的是美国邮政总局监察长办公室由下辖的宾夕法尼亚州匹兹堡东区分处的特别主管肯尼思·克莱维利负责，美国海关与边境保护局方面由现场运营总监罗丝·布萝菲负责。

最近有多起大选邮寄选票舞弊案被爆出，这是涉及邮政部门诸多案件中的一起。
