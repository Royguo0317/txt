## 王毅出访日韩与中共政治局会议可能与台海走势有关

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/02125506/Capture-13.jpg)图片来源：今周刊
最近，两件中国大陆发生的事值得引起我们的注意，一件是王毅访问日本和韩国，另一件是习近平主持的中共中央政治局会议审议《军队政治工作条例》、《中国共产党统一战线工作条例》和《中国共产党党员权利保障条例》。

**王毅访日韩为台海危机探口风**

在王毅访问日本期间，中日双方并未提及习近平访日的问题，仅与日方在加强经济往来、社会交往等方面达成一些共识，提及了钓鱼岛（Senkaku Islands）问题。王毅建议，除公务船以外，中日双方不应允许其他船只进入钓鱼岛周边海域，也就是说渔船或军舰不得进入该地区。但双方依然存在分歧，日方不同意王毅的建议。从王毅访日来看，中方想通过不让中方渔船进入钓鱼岛附近来换取日本不让军舰通过该地区，而日方没有答应。记得日本海上自卫队一位不愿透露姓名的军官曾对记者表示，一旦中共对台动用武力，自卫队将按照西南地区“有事”的防卫方针“迅速应对”。这也为日本出兵支援台湾埋下伏笔。

在韩国访问时，王毅在首尔会见了韩国总统文在寅，讨论了习近平访韩、朝鲜半岛无核化及疫情合作等多个议题。反观文在寅，他是亲中派，回顾其在今年大选中的大获全胜，结合目前美国大选暴露的资料所显示，中共国通过红杉资本和凯雷资本控制 Smartmatic 和 Dominion 投票系统，有多个国家的控股公司涉及，其中就有韩国前总理朴泰俊， 我们不禁要问，文在寅是不是通过这个选举系统靠舞弊获得胜选的呢？答案不言而喻。这也就能解释文在寅为什么对中共亦步亦趋了。

**中共中央政治局会议再次强调表忠心**

现在来看看习近平主持的中共中央政治局会议。会议审议了《军队政治工作条例》《中国共产党统一战线工作条例》和《中国共产党党员权利保障条例》，其中《军队政治工作条例》表现的还是一如既往要确保军队始终成为党绝对领导下的军队， 也就是要保证习近平对军队的绝对领导；关于《中国共产党统一战线工作条例》，本次会议强调，要加强党对统战工作的集中统一领导，也是统战工作必须在习的领导之下，提出要团结一切可以团结的力量，这也体现了王毅去日韩的目的之一，即从政府层面企图“团结”周边力量。拉拢韩国也许是成功的，但日本之行好像并不那么顺利，但我们也要密切观察中共在各国渗透的暗势力是否在行动。

从上述两项条例的审议反映出，中共要保证军队忠于共产党，即忠于习。随着中共在美国大选中的操控行为不断的曝露，川普的连任已是不争的事实。可以预见，不远的未来，中美将可能会有军事上的冲突，台海将是几个热点之一，中共企图为预谋攻打台湾做好铺垫，而目前看来想让日本合作的目的没能实现，但是否能指使朝鲜和韩国在日本附近挑事端也未为可知，同时也要注意，在日本内部的中共的渗透力量能否顺利牵制住日本介入可能的台海危机。

第三个审议的是《中国共产党党员权利保障条例》。会议强调发挥党员主体作用，把全体党员更加紧密地团结在党中央周围，要求党员开展监督，也就是鼓励党员告密，通过告密排除异己，以此来实现中共的恐怖统治。这种靠告密来维护政权的做法必然导致人心向背，政权愈加不稳。这不禁让人想起明朝，当时的皇帝就有鼓励告密的传统，当外族入侵时，无人站出来帮助崇祯皇帝，以致外族长驱直入，明朝迅速灭亡，中共的结局也必然如此。
