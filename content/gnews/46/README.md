Ⅰ. 美国国务院发布文件说，国务卿布林肯与中共杨洁篪进行了交谈。布林肯强调，美国将继续支持人权和民主价值观，包括在新疆、西藏和香港，并敦促中共国与国际社会一道谴责缅甸的军事政变。国务卿重申，美国将与盟友和伙伴共同努力，捍卫我们的共同价值观和利益，让中共国对其威胁包括台湾海峡两岸在内的印太地区稳定的努力以及其破坏基于规则的国际体系的行为负责。杨的态度非常强硬，并且提醒美国政府要维护以联合国为核心的国际体系，中共极有可能通过与联合国程序平息种族灭绝罪和反人类罪带来的国际影响。布林肯强调模糊化的民主和人权，没有明确向中共表明美国在台湾、香港、新疆问题和病毒真相上的态度。拜登政府如果不能迅速认清中共的邪恶，将付出代价。

Ⅱ. 中共媒体报道，世卫专家组详查6天6夜，最终极力赞扬中共在疫情防控上的突出贡献，证实中共病毒与武汉病毒研究所毫无关系，并将与中共病毒专家共同撰写中共病毒调查报告。以Peter Daszak为主导的世卫专家组成员被中共收买，WHO出具的病毒疫情调查报告禁不起时间的考验，这些病毒专家终将成为历史的罪人。

III. 中共国教育部2月3日报道，针对有全国政协委员提出《关于防止男性青少年女性化的提案》，教育部官网日前回应，将更多注意学生“阳刚之气”培养。中共国呼吁培养阳刚之气就是要让更多老百姓做炮灰，在为攻打台湾做准备，中共寄希望通过攻打台湾来改善国内萧条的经济环境，转移国内和党内被严重激化的矛盾。

2. 北京时间2月7日是李文亮医生去世一周年的纪念日，有人在周六到达他去世的医院用鲜花纪念。他是中共病毒最早八名爆料人之一。中共正想办法让民众忘记他，包括他的家人也因为各种原因，拒绝接受采访。

3. 卢比奥参议员2月5日针对香港政府根据国安法宣布的教育指令发表声明，对中共在香港推行教育改革表示谴责，批评中共现在通过实行全面的教育改革来继续镇压香港的自治。

4. 因中共阻挠，圭亚那2月4日宣布终止与台湾设立办公室的协议后，美国在台协会（AIT）2月6日表示，美方鼓励更多国家深化与台湾的非官方关系，也期待与台湾在区域紧密合作。

5. 中共国人民银行上海总部7日召开视频会议，要求地方法人银行要回归服务当地的本源，不得以各种方式开办异地存款。此举将降低储户的收益率，并诱使储户存款转向中共所谓的投资产品。
