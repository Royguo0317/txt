## 从美俄制裁大战看中美下一阶段发展

作者： 小强

刚刚结束的阿拉斯加会面没有实质性的结果，那么美中关系下一阶段如何发展，我们尝试从2014年美国制裁俄罗斯的历史中寻找答案。

2014年3月16日，乌克兰克里米亚民众以公投的形式决定脱乌入俄，但以美国为首的其他西方国家并不买账。随后以美国和欧洲开始对俄罗斯进行了一系列的“制裁”。俄罗斯方面，也毫不示弱进行了一系列的强硬回应。截至今日制裁仍在进行中，回顾一下整个过程，看看中共国有可能面临的哪些制裁，以及何种结果。

2014年3月到12月美国共发布了7条制裁措施。篇幅原因不详细陈述，主要分为几类：

1. 针对俄罗斯领导人及重要人士的个人制裁，包括拒发签证，冻结个人海外资产。
2. 针对俄罗斯公司的制裁，包括冻结公司在美资产，吊销公司出口许可证，在美销售执照，禁止美国公司与俄罗斯公司技术合作。
3. 针对俄罗斯银行的制裁，包括禁止美国销售俄罗斯银行发行的债券。
4. 联合盟友，扩大制裁范围，美国通过游说先后得到了欧洲和亚洲多数国家的支持。


面对制裁，俄罗斯进行了反击，主要分为几类：

1. 针对美国领导人的对等制裁。
2. 部分科技领域进行自研，拜托对外依赖，稳定国内民生。
3. 处分麦当劳，置其关闭部分店铺。
4. 号召俄罗斯财阀回国，将海外资产带回俄罗斯，有部分财阀响应。
5. 印钞支撑俄罗斯国内经济。


显然俄罗斯的反击没有实质效果，2014年12月16日，俄罗斯遭遇“黑色星期二”：周一，卢布大幅贬值11%引发俄罗斯央行火线加息至17%；外汇市场对于俄央行的加息根本不买账，周二，卢布崩盘贬值幅度一度超19%；俄罗斯RTS股指盘中亦大跌19%，这是该市场1995年以来最大跌幅。俄罗斯的GDP2013年是2.3万亿，2014年已经下降到2万亿，全面措施开始后2015年下降到1.36万亿，2016年1.28万亿，世界排名2013年第8名，2015年跌至12名并保持至今，后面几年GDP略有回升，但GDP的增长几乎与美元的通胀保持一致，被制裁后俄罗斯在世界经济中的地位和角色发生了根本性的下降。面对俄罗斯困境，美国白宫2014年12月16日警告称，在多轮制裁、油价下跌以及卢布贬值等因素的共同作用下，俄经济已处于“危机的边缘“。言外之意，美国认为制裁已经达到目的，没有用出更进一步的手段，是俄罗斯在2015年后得以维持在“危机的边缘“的原因。

可以看出，美中关系的走向与2014年美俄关系如出一辙。美国目前仅仅对中共国用出了1和2部分的招数。中共国就已经进行了对美领导人制裁，大炼芯片，强占外资企业股份，在美企业退市返回A股上市等与俄反制手段相似，而印钞这种手段多年来一直未停。那么中共经济目前还得以维持的原因就是，美国还没有用出全部的措施2和措施3&4。按俄罗斯的例子，当美国把措施2全部用出的时候，就是国内受制裁的行业破产重组并被CCP割韭菜的时候。当美国用出措施3的时候，就是人民币汇率的大幅贬值的时候。当美国用措施4的时候，就是总加速师闭关锁国的时候。

再推测一下中共国被制裁后的结果，俄罗斯被制裁后GDP的大约下降40%，中共国GDP按2020年15.6万亿打六折之后，大概是9.3万亿。仍然远高于第三名日本5.3万亿。虽然其他国家会瓜分中共国失去的那部分GDP，但是短期内不足以撼动中共国GDP第二的排名。

因此可以看出，如果美国仅仅用出与制裁俄罗斯相同的手段来对付中共，虽然能很大程度上弱共，但不足以让中共束手就擒，也不足以说服盟友对GDP仍排第二的中共发生热战，这也就是为什么总加速师敢猛踩油门的原因。拜登政府如果真心灭共，就必须要用出比制裁俄罗斯更厉害的手段，把中共的GDP至少挤出世界前5，让美国及盟国分食中共国的GDP蛋糕，最大程度上弱共。而唯一能够保证这点的就只有病毒真相，逼迫拜登政府无法勾兑，不得不用出杀手锏，迅速将中共国的经济打入谷底，逼总加速师踏过红线，主动向世界宣战，结束中共最后的生涯。

以毒灭共，就是这个过程。我们每天传播的病毒真相，整个世界都在等待。2022年中期选举前应该会得到答案。
