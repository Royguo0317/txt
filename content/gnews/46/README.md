不想神化文贵先生，宁可称他为七哥。

相信七哥和川普都是神选之子！是神为了预备新世界的到来，赐给人类的两个先知，一中一西！相信他们将成为圣人，但不是神。圣经上记载的先知约翰在耶稣之前与约旦河为人们施洗，为主铺平道路预备人心。

神展开了东西方两条主线。在中国神兴起了郭文贵，在美国神拣选了川普。

七哥在华人世界发起的爆料革命始于三年前，但七哥真正领受神的旨意应该是三十一年前，在清风看守所的时候，用他自己的话，他在那里重生（Reborn）了。神在那里拣选了他，把他从死亡的十字架上救了下来，因为怜悯公义的神知道他是无辜的，知道他是善良的，知道他嫉恶如仇，知道他忠诚坚韧，知道他聪明好学，知道他讲义气，知道他苦大仇深，知道他为失去八弟而痛不欲生，知道他是唯一一个可以担起重建中华文明的人。奇迹出现了，犹如七哥在他的直播中与我们分享过的那些故事，神在监狱里给他安排了老师，当走出看守所的那一刻，聪明的七哥也大学毕业了。

之后，我最关注的是七哥两个至今都引以为傲的建筑：裕达国贸和盘古大观。神奇！说实话我是先认识盘古大观才认识的七哥，也是盘古大观让我坚信他不可能是“红色通缉令”所描述的那个人。我仍然记得我站在盘古大观和北京鸟巢奥运场馆之间的感受，感觉盘古大观可以和紫禁城一样傲然鹤立在北京几百年，但鸟巢？能不能挺过十级大风是个问题。

过去三年，文贵先生对国际经济、国际政治、国际金融、民生、社会的判断，无一不应验。可是七哥自己谦虚的说我们不是预测，是情报。在我看来，神把合适的人在合适的时间放在了文贵先生的生命里，来辅佐他，来帮助他，化险为夷，遇难成祥的奇事只有七哥自己最有体会，这也是为什么七哥如此敬畏神明，因为他知道自己是有使命的人。我跟很多战友聊过挺郭经历，惊讶的发现战友中有信仰的比例非常高，或是佛教徒或是基督教徒，但是一股神奇的力量把我们都吸引到了郭先生身边。这是无法解释的奇妙！

这股力量或者说这个理念就是郭先生倡导的正道主义！

爆料革命的正道主义：“真、善、狠”，“郭七条”，“唯真不破”，“对战友们不放弃、不抛弃、不忘记”，”我们一定要坚持住。一定要把华人形象要维持住，坚决不能放弃。“ 七哥定义中共盗国贼以警治国，以贪治国，以生殖器治国的说法更是让我们不能同意的再多。每一次看七哥的直播，都有一种灵魂升华的感觉，有的时候是一句话，有的时候是一大段话，改变着我们的认知，也统一着我们的认知。我们这些最先觉醒的人的和七哥一样是神为新世界预备的战士！虽然我们大多数人只是在网上交流，甚至使用的都不是真名，但我们亲密如一家人，特别是当我们的父母兄弟姐妹，发小闺蜜都无法理解我们的时候，我们和战友却知无不言，言无不尽。谁能解释这一切？！
