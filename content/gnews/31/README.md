## 美军将向印度出售P-8反潜巡逻机
` 曹操` [轉載自GNews](https://gnews.org/zh-hans/1163460/)

#### 作者:【默凡】
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/02052646/A16495DC-ADF2-480B-AD98-767742325845.png)
据路透社报道称[1]：美国五角大楼本周五（4月30日）表示，美国务院已经批准向印度出售6架P-8“波塞冬”反潜巡逻机和相关设备，预计价格为24.2亿美元，将有助于印度扩大其海上监视能力，该一揽子计划将包括战术无线电、导弹预警传感器、GPS系统、支持设备、备件和技术支持。

五角大楼国防安全合作局(Defense Security Cooperation Agency)周五向国会通报了这项计划。尽管得到了美国国务院的批准，但这并不意味着双方已经签署合同或谈判已经结束，波音公司是这些武器的主承包商。

**P-8“波塞冬”是美国波音公司所设计生产的一种海上巡逻机，主要用途为海上巡逻、侦察和反潜作战，机上配置有5个内置与6个外置武器挂载点。P-8的主力侦搜装备为AN/APY-10雷达，除了具备传统反潜作战需要的水面搜索能力，后续亦开发了沿岸监视模式，可以测绘出高精确度的陆地雷达图像，其精确度可辨识出陆地载具[2]。若这笔交易达成，印度反潜侦察能力将大大提升，中共国潜艇在印度洋将遇到大麻烦。**
