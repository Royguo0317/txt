## 印度疫情肆虐，邪共趁机抬高原料药价格

(ABID BHAT/AFP via Getty Images)

大纪元报道：近一段时间，印度受到中共病毒瘟疫重创，15日死亡人数依然在4000人左右，随着瘟疫在该国扩散，印度对治疗瘟疫相关的药物和原料药的需求激增。

据印度《经济时报》5月16日引述消息人士的消息说，一种证明有助于降低瘟疫患者死亡率的药物，也是用于制造抗寄生虫药伊维菌素的原料API（用于生产各类制剂的原料药物，是制剂中的有效成分，由化学合成、植物提取或者生物技术所制备的各种用来作为药用的粉末、结晶、浸膏等，但病人无法直接服用的物质），其价格最近上涨了300％，而印度制药公司从中国采购了几乎70％的API。印度政府已敦促中共当局稳定API的价格并允许货运飞机不受干扰地飞行。

评论：

这就是中共所谓的病毒外交，由于印度不得不从中共国进口治疗药物的原材料，中共就趁机从印度牟取大量利益，同时还能用原材料作为牵制印度的工具。中共既在印度投放中共病毒，然后又抬高药品价格，再以此作为与印度谈判的筹码，可见中共是一个多么邪恶的政权，只有彻底消灭中共才能阻止疫情的再次爆发。
