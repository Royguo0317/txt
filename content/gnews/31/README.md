1月11日，加拿大CP24媒体报道，安大略省护士学院(CNO)正对两名参加了1月6日在美国华盛顿特区一场反封锁集会的护士克里斯汀·纳格尔（Kristen Nagle）和莎拉·乔乔尼安（Sarah Choujounian）进行调查。
![]()![](https://gnews.org/wp-content/uploads/2021/01/image-2.jpg)
图片来源：CTV

从”护士反对封锁（Nurses against lock down） “发布在Instagram的视频中，可以看到勇敢的护士在华盛顿特区举行的 “全球一线护士 “峰会发表讲话，这次活动在美国最高法院前举行。在”全球一线护士 “网站上可以看到当日与会的特别嘉宾有一直支持羟氯喹防治中共病毒的泽连科医生（Dr. Vladimir Zelenko）。
![]()![](https://gnews.org/wp-content/uploads/2021/01/屏幕截图391.png)图片来源：globalfrontlinenurses.com
莎拉·乔乔尼安是一名来自加拿大多伦多的护士，她16年的职业生涯大部分时间都是在养老院工作。她表示担心疫苗会伤害老年人，并称养老院对访客的限制构成 “反人类罪”。她说自己在社交媒体上发表反对封锁的言论后被解雇。

另一位克里斯汀·纳格尔是安省西安大略大学儿童医院新生儿重症监护室护士。据纳格尔工作的上级伦敦健康科学中心称，她在参加 “当地的反口罩集会 “后，自11月起就一直处于停薪留职状态。伦敦健康科学中心表示，他们 “立即采取行动，以确保医院对每个人来说仍是安全的环境”，他们将等待对纳格尔的行为的调查结束后对她的就业状况作出最终决定。

安大略省护士学院指出，其网站上有一份支持现行卫生措施的声明，敦促护士对自己在网上发布的内容负责。安大略省护士学院表示，正在对这两位护士进行调查。

“安省护士学院正在调查克里斯汀·纳格尔和莎拉·乔乔尼安，”该学院在一份声明中说。”法律不允许我们提供有关进展或正在进行的状况的细节”。

由此可见，不只是美国的挺川者受打压，加拿大拥有一线抗击中共病毒疫情经验、揭露真相的护士也被强权打压，受到失去工作的威胁。终极之战中站在正义这一方注定会经历无数风雨和坎坷，向这两位敢于发声的加拿大护士致敬。
