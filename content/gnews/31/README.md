# **有报告称，由中共国国家资助的黑客大规模攻击全球各地的公司**

**袭击事件与一个代号为蝉的组织有关，据信该组织由中共国政府资助。**

研究人员发现了一个大规模的黑客活动组织，该组织正在使用复杂的工具和技术来入侵全世界公司的网络。

这些黑客很可能来自中共国政府资助的知名组织，他们配备了现成的和定制的工具。 这种工具利用了Zerologon（Windows服务器漏洞的名称），可为攻击者提供即时的管理员特权来攻击有漏洞的系统。该漏洞已在8月被修复。

赛门铁克公司（Symantec）使用代号”蝉”（Cicada）为该组织命名，该组织被普遍认为是由中共国政府资助的而且该组织还使用APT10、石熊猫（Stone Panda）和其他研究组织发现的云端跳跃（Cloud Hopper）等绰号。 该组织与任何名为”蝉”(Cicada)的公司都没有关系或没有隶属关系，该组织至少从2009年开始就活跃于间谍式黑客活动，几乎专门针对与日本有联系的公司。 尽管在最近行动中，目标公司位于美国和其他国家/地区，但这些公司都与日本或日本公司有联系。

#### **监控显示**

安全公司赛门铁克（Symantec）的研究人员在一份报告中写道：“与日本方面有关的组织必须保持警惕，因为很明显，他们是这一复杂且资源丰富的团队的主要目标，而汽车业似乎是此次攻击活动的主要目标。但是，由于攻击针对的行业广泛，各行各业的日本组织都必须意识到自己有遭受此类活动的风险。”

这种攻击广泛使用了DLL侧载，这种技术是在攻击者用恶意文件替换合法的Windows动态链接库文件时发生的。 攻击者使用DLL侧载技术将恶意软件植入到合法程序中，从而可以防止安全软件检测到该黑客。

该黑客组织还使用了一种能够利用Zerologon的工具。 通过在一系列使用Netlogon协议的消息中发送零字符串来利用漏洞，而Windows服务器操作系统使用Netlogon协议让用户登录网络。 未经身份验证的人员可以利用Zerologon通过一个组织的皇冠上的明珠—为所有连接到网络的计算机充当全能网络卫士的活动目录（Active Directory）域控制器。

微软（Microsoft）在8月修补了关键的特权升级漏洞，但是自那时以来，攻击者一直在使用它来危及尚未安装此更新的组织。 FBI和国土安全部都敦促(微软)立即修补这些系统。

在攻击过程中，赛门铁克(Symantec)公司发现计算机的域控制器和文件服务器受到损坏。 公司研究人员还发现了从某些受感染机器中窃取文件的证据。
