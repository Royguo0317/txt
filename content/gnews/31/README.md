# **独家：拜登内阁人选受邀参加中共会议，集体吹嘘此次会议为与美国官员的“私人吹风会 ”**

**百人委员会 — 和中共共同主持召开一系列会议并得到习近平的赞扬。— 它吹嘘是与美国政府官员和有关美国政界人物（包括拜登政府提名的内阁秘书和顾问）的“私人吹风会”。**

该组织是对中国共产党绥靖主义的亨利·基辛格（Henry Kissinger）的创意，它定期举行会议以增进中美之间的政治和经济联系。

然而，百人委员会似乎利用其成员的身份（包括杰出的中、美两国公民）来将美国人与中国共产党联系起来。

在这个组织的“任务和历史”页面上，突出显示的是该小组成员与包括习近平在内的中共现任和前任领导人的合影照片：

[!\[\]()!\[\](https://spark.adobe.com/page/g3J55k6iWwY3J/images/dbf8dbdc-9d63-4580-bd00-de4fe2da4935.png?asset_id=4d6de7da-3518-4297-bd76-8a6a96a41dcd&amp;img_etag=%22c571472ec429e59921f5ca73aece26a9%22&amp;size=2604)](https://spark.adobe.com/page/g3J55k6iWwY3J/images/dbf8dbdc-9d63-4580-bd00-de4fe2da4935.png?asset_id=4d6de7da-3518-4297-bd76-8a6a96a41dcd&amp;img_etag=%22c571472ec429e59921f5ca73aece26a9%22&amp;size=1024)来自百人会网站

中国共产党现任领导人在2015年曾赞扬该组织所做的努力，称赞该委员会以及亲中共国游说团体的其他成员对中国共产党的“友好”，应得到他的“衷心感谢”：

*中国有句俗话说，众人拾柴火焰高。是两国政府，地方当局，友好组织和各界人士的关爱和辛勤工作使中美两国关系得到了蓬勃发展。 特别是，美中关系全国委员会，美中贸易理事会，美中政策基金会，美国商会，美国中国总商会，百人委员会，布鲁金斯学会和 其他友好团体和个人 多年来，为促进两国之间的友好关系与合作做出了不懈的努力，并将这一关系推向了新的高度。 让我向所有致力于中美友好事业的地方政府，社会组织，大学，智囊团和社会各界人士表示由衷的感谢。*

除了通常由中共国政府高级官员参加的会议之外，该组织还举行“国会山方面关于中美重要议题的私人吹风会”：

*自2015年4月起，百人会（C100）已举办了6次国会山方面关于中美重要议题的私人教育简报会，吸引了该国首都的近300名国会议员参加，包括现任国会议员、前国会议员、外交团体议员和在参、众两院的高级工作人员等等。 会议的长期目标是建立一致且系统性地存在于国会山的同盟，并增强C100的长期能力，以便在与我们双重使命至关重要的问题上与领先的决策者互动。*

百人会还送其成员访问中共国，参加“与几位中共主要领导人的私人会议”，例如中共国文化部长蔡武。

该委员会还组织 “美国年度代表与北京，香港和台北的高层政治和商业领袖会面”，并与中共高层领导人举行会议。

该组织最近的一次会议有中共国前外交部长李肇星和驻美国大使周文忠参加，可看到发言人呼吁两国之间建立更紧密的联系。 发言人们强调了“脱钩不是長长久之計”，而且美中两国需要“相互合作”。

同年，为回应美国司法部（DOJ）针对与中共有关的研究人员掠夺知识产权并未能披露其外国关系的大量指控，该委员会举行了一次峰会，讨论了“华裔美国研究人员面临的挑战” ，试图破坏司法部的调查。

总结说道：“随着越来越多的人关注中共国人在美国从事间谍活动，以及中美之间的紧张局势升级，尤其是从事科学和研究的华裔美国人越来越多地受到雇主、同事和检察官的审查。这些问题不仅对单个科学家和研究人员，而且对大学，研究机构和企业都产生了令人不寒而栗的影响。”

与一名最近被曝光的中国共产党情报人员多次合影的朱迪·朱（Judy Chu）与亚当·希夫（Adam Schiff）在活动中发表了讲话。

2019年的另一次大会活动将乔·拜登的顾问劳伦斯·萨默斯（Lawrence Summers）与马克斯·鲍克斯（Max Baucus）一起推为开幕式主讲人，马科斯·鲍克斯是奥巴马时代的驻中国大使以及被《国家脉动》曝光的拜登亲密朋友，他利用他的身份保证做中国共产党的顾问工作。 中国共产党控股的中国银行也是该活动的赞助商。
