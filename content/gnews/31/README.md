新法律的最终版本给了内政部更多权力来阻止华为的5G合约（VCG via Getty Images）

德国立法机构周五通过了更严格的5G安全立法，明确了2年以来的疑问：这个欧洲最大市场是否会强硬对待中共国电信巨头华为？

给了内政部更多权力来阻止华为的5G合约

德国联邦议院批准的新的《信息技术安全法2.0》（IT Security Law 2.0）限制了“不可信任”的5G技术供应商的角色，并要求电信运营商在签订关键5G组件合同时必须通知政府。它还赋予政府阻止它们的权力。

经过多年的争论，这些规则使柏林与巴黎、伦敦和其他主要首都在规范敏感网络技术方面更加接近。即便如此，实施这些规则仍需要政治意愿。鉴于德国将在9月份进行联邦选举，所以，决定如何实施这些标准的工作将很大程度上落在德国下一届政府身上。

内政部长霍斯特·西霍弗（Horst Seehofer）告诉联邦议院：“如果供应商不值得信赖，则可以拒绝使用关键部件。”

对更严格的5G法规的投票，是在中共国外交部长王毅与德国总理海科·马斯（Heiko Maas）关于5G通话的几天之后进行的。一位知情人士说：“双方应共同反对所谓的‘脱钩’，共同维护全球产业链和供应链的畅通与稳定。”

必须符合北约的目标

在国会议员对总理安格拉·默克尔（Angela Merkel）对华为处理表示沮丧之后，最新的文本比政府去年12月提出的提议更为强硬。去年，由于政府部门和联合政党未能就关键的市场准入问题下定决心，有关德国对华为立场的谈判基本上陷于停滞。

与此同时，在美国对华为强硬施压的情况下，默克尔推动了与北京签署了一项投资协议，意图维护与中共国的贸易关系。

因为华为是德国4G网络基础设施的三大运营商中的优势供应商，执政联盟在如何处理华为问题上存在分歧并一再拖延。

该法律的最终版本赋予内政部更多的权力来阻止5G合同，同时赋予其他部委—（例如过去非常支持华为的经济部）更少的权力。

在运营商将购买新的关键5G组件的计划通知政府之后，内政部将有2到4个月的时间，根据国家安全标准来检查这些交易。

议员们还补充说，使用新的关键5G组件必须符合德国、欧盟和北约的“安全政策目标”。后两者已采取立场，警告不要在新的电信网络的关键和敏感部分使用中共国的设备。

制造商必须提供“可信度声明”，国会议员则提高了供应商达到“可信度”这一标准的门槛。
