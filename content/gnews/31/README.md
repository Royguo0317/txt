港府正积极考虑　强制食客用「安心出行」

政府早前推出「安心出行」应用程式，但反应平平，创新及科技局局长薛永恒今日（8日）在立法会资讯科技及广播事务委员会会议上表示， 正积极考虑强制要求市民进出食肆等高风险处所前，必须扫描「安心出行」二维码。

逾 50 跨国议员及政界人士去信汇丰　要求解冻许智峰及家人户口

英国《泰晤士报》报道，逾50 名来自英国、德国、欧盟等13 个地方的议员及政界人士，以对华政策跨国议会联盟名义，向汇控主席杜嘉祺发信，批评香港警方以政治动机指控许智峰洗黑钱，而汇丰在没有提供任何法院命令下，冻结许智峰及其家人的帐户，其后又冻结好邻舍北区教会等的账户。信中提出3 项呼吁，要求汇丰立即解冻结许智峰及家人所有汇丰银行账户，并解释冻结帐户决定，又要求汇丰公开承诺，在中共国和香港当局的出于政治动机的指控下，仍会会保护个人及家庭资金。汇丰银行回覆只能遵守警方要求。

屡参与反送中游行　中大教授麦高登延迟退休　盼续留港教书捍言论自由

中文大学人类学系主任、国际知名学者麦高登表示，未来 3 年将继续于人类学系任教。 1994 年麦高登获中文大学人类学系聘请，在 2014 年雨伞运动期间，他多次现身占领区陪伴学生、确保他们人身安全。他指雨伞运动令他对香港产生了一种前所未见的情感」，亦让他清楚看到这个都市美好的一面。他慨叹香港的言论自由正在收窄，而「以尽责、负责任的方式教授学生、捍卫言论自由，变得特别重要，是他希望做到的事。

港警互抄口供 黄大仙非法集结案八人无罪释放

前年10月1日示威者「遍地开花」，多区有示威活动，其中9名男女被指当日在黄大仙非法集结，案件早前开审，揭露督察作供与现场片段不符，连日审讯中多名警务人员，更被辩方质疑互抄口供。裁判官黄士翔今裁定案中八人，表面证供不成立，毋须答辩当庭释放；唯独当日被搜出身上有溪钱的女护士表证成立，须继续受审。

亲共英籍大律师伦为中共传声筒 打压本港民主运动竞逐和平奖

美国9名民主共和两党国会议员，早前去信诺贝尔委员会主席，提名「香港民主运动」竞逐今年诺贝尔和平奖，香港前刑事检控专员英籍大律师江乐士，致函诺贝尔委员会主席，批评提名「香港民主运动」的美国国会议员反华，更形容香港警队是真正的英雄。据悉，江乐士与港共关系十分密切，九七年就获港府委任做香港刑事检控专员。
