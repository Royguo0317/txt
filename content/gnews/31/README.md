**保皇党区选败部复活占选委逾一成**

中共决定将特首选举委员会人数虽扩大至1,500人，但选民基础却大幅倒退。人大常委会的决定废掉现时117个区议员产生的议席外，除了大举引入分区委员会、灭罪委员会、防火委员会等外，更加内地港人团体代表和全国性团体香港成员代表，而获任为分区委员会的委员，不少是在2019年区选中落败的建制派，在中共颠倒香港选举制度下，咸鱼翻身变赢家。

**老妇打科兴倒毙家中 卫生署否认疫苗责任**

港府以恢复正常生活为由，利诱逾45万人接种武汉肺炎疫苗；惟至少13人针后亡，至少11均打中共科兴，引爆信心危机。卫生署虽设有事故通报机制，但被发现，1名老妇打科兴翌日暴毙家中，经法医及医护判断与疫苗无直接关系后，署方却没有呈报专家委员会及公开事件就了事，与之前13宗死亡个案的处理明显有异，过去13位死者同样未被证实死因与疫苗有关，但需要呈报及公开事件。病人组织代表批评署方隐瞒，估计疫苗开打至今逾月，有同类个案被湮没。
![]()![](https://gnews.org/wp-content/uploads/2021/03/image006-18.jpg)
**陈肇始称打疫苗才是恢复正常生活的希望**

食物及卫生局局长陈肇始昨宣布放宽部份限聚措施，包括重开泳池泳滩，戏院、剧场及主题乐园放宽入座率等。陈肇始表示，本港目前没有条件大规模放宽社交距离措施，而当局在考虑放宽昨日公布的措施时，主要考虑到相关活动风险较低，例如需全程佩戴口罩；虽然疫情缓和，甚至出现零本地确诊，但陈肇始说不等于当局完全有信心社区上无个案，亦未达致「清零」，重申接种疫苗是市民恢复正常生活的希望。

**中大封杀学生会**

中大校友捍卫院校自主关注组（关注组）早前发起校友联署，要求中大校友评议会常务委员会（常委会）召开特别会员大会，讨论校方停向中大学生会提供场地、行政支援及代收会费等议题。惟常委会昨（29日）以讨论内容或违《港区国安法》为由，拒绝召开特别会员大会。关注组指常委以国安为由拒绝开会是「狐假虎威」，做法亦违章。关注组呼吁校友踊跃参加4月的周年大会，向常委会追究。
![]()![](https://gnews.org/wp-content/uploads/2021/03/image010-4.jpg)
**47民主派被控︱黄碧云保释之身探战友 曾健成引智叔对白吁香港人「不要习惯（这种状态）」**

47多名民主派人士因参与民主派初选，被控「串谋颠覆国家政权」罪，大部份人不获保释，至今已还柙一个月。今早（30日）继续有亲友到荔枝角收柙所探监，初选47子之一、早前获保释的黄碧云，亦到场探望民主党战友尹兆坚，她表示，以保释之身探望同案其他在囚被告，感觉并不好受，「他们的自由都被褫夺了」，故盼望他们亦可尽快获保释。黄又寄语港人不要放弃香港，「坚持我哋所信守慨民主理念」。

囚权组织「石墙花」创立人邵家臻探望范国威，并指范虽然很惦记其他被告民主派，以及新民主同盟的战友，但他状态很精神，有恒常做运动。范在收柙所内认识了很多被囚抗争者，并表示希望外界不要只关心47子，千叮万嘱要留意其他手足。

社民连曾健成探望「长毛」梁国雄，他指长毛情况不错，并没有因保释申请失败而失望。曾又叹，对现在的香港感到很陌生，为香港发声的人，现在都在各大监狱中，但他呼吁香港人「不要习惯（这种状态）」。

**林郑拒纳传媒可公司查册**

财经事务及库务局封杀公司查册，削弱市民知情权、公众利益，记协斥会影响调查报道，林郑月娥称要平衡个人私隐，而公共名册存在本身已是一种平衡，让有需要人士检阅，但同时要保护私隐，如非必要性的资料可作保护不予公开。她又引述自己2月4日在立法会答问大会提及会就私隐立法研究，防止个人资料「武器化」及禁止网上仇恨言论等，而公共查册是其中一环，亦毋须经立法，行政上已可处理，她拒绝传媒要求可查阅公司人员资料「指明的人士」。
![]()![](https://gnews.org/wp-content/uploads/2021/03/image013-11.png)
**李百全上任一个月抽3节目, 疯狂批评《议事论事》《铿锵集》6宗罪**

港台新任广播处长李百全，多次出手「整顿港台」，多次抽起不同节目，一直未有交代抽起节目原因。港台今早再抽起台亦承认，共抽起了3个节目，分别《香港故事：疫行者》、《议事论事》，《铿锵集》其中一集，并停播《五夜讲场》制作，至晚上首次发长稿回应事件，并力数涉事节目的「六宗罪」，披露近日多次抽起节目原因，当中包括对《港区国安法》有不准确描述。

港台强调港台编辑自主，不受商业、政治或其他方面影响，吁各方「尊重港台编辑决定」。
![]()![](https://gnews.org/wp-content/uploads/2021/03/image015-9.jpg)
**林郑盛赞李百全清算港台**

广播处长李百全上任不足1 个月，已经多次被指抽起香港电台节目，昨日（29 日）亦有消息指港台电视部清谈节目「五夜讲场2021」制作将暂停，要等待管理层进一步审批。特首林郑月娥今日（30 日）出席行会前见记者，被问到港人收看相关节目的权利有否被剥夺，她表示港台既是公共广播机构，亦是政府部门，而且港台须遵守《香港电台约章》，广播处长作为「总编辑」有把关角色，行事要相当「小心」。林郑评价李百全上任以来一直表现「尽责」，并且做到她预期港台「总编辑」应该做到的工作。
![]()![](https://gnews.org/wp-content/uploads/2021/03/image017-21.png)
**《不割席》入围, TVB停播奥斯卡**

全球影坛盛事、第93届奥斯卡颁奖礼将于香港时间下月26日上午8时于洛杉矶举行，无线电视广播有限公司（TVB）发言人表示，无线由于今年没有购买亦不会继续商讨转播权，届时明珠台不会播放奥斯卡颁奖礼，更称「纯粹商业决定」。

自1969年52年来无线首次在港停播这项电影界盛事。无线忽然表明今年不播放奥斯卡，不其然令人联想到是否因今届由挪威导演Anders Hammer执导、以反送中为题材的电影《Do Not Split 不割席》有份入围最佳纪录短片。

**《风再起时》讲警贪, 电影节撤播**

由郭富城及梁朝伟主演、以警贪为题材的电影《风再起时》，被选为第45届香港国际电影节开幕电影之一，原定在香港举行世界首映，昨日电影节在官方网站突然公布取消《风再起时》两场原定4月1日及4月4日在香港文化中心的场次，指是应片方要求，因技术问题而取消放映。

《风再起时》故事讲述上世纪七十年代港英政府成立廉政公署后追捕一班贪污探长，电影改编自 已故探长吕乐的贪污事迹。

**世界新闻摄影展移师金钟首日预约爆满 参观市民：珍惜红线下仅余自由**

由世界新闻摄影基金会举办，获荷兰驻港总领事馆及香港摄影记者协会支持的「世界新闻摄影展2020」，原定由本月1日起在浸大举行摄影展，但校方早前突然拒绝提供场地，因今年其中一辑得奖作品叙述2019年反送中运动, 最终摄影展要移师金钟举行。吸引不少市民特意前来欣赏，由于及反送中议题的活动未来举办的空间或收窄，称要趁有展览时好好珍惜仅有的观赏自由。

**疫苗中心聘旅游员工, 耗资1.5亿**

旅游业界受疫情重击，失业率达11%。政府将5月1日起为旅游业从业员提供约2,000个社区疫苗接种中心短期职位空缺，专责管理及行政支援工作，整个计划预算开支达1.5亿元。

**登机限制取消, 百滞英者或下月准返港**

政府自去年12月21日宣布英国航班禁飞香港后，至今超过三个月，大批滞留当地的港人终于归期有望，食物及卫生局局长陈肇始昨日称，考虑英国疫情及接种疫苗进度，将取消英国登机限制，港府正与航空公司商讨于4月下旬，安排身处英国的港人乘搭指定航班返港及检疫21日。政府并放宽低风险地区的检疫期，由现时21日缩减至14日。
