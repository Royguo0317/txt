据《国家评论》吉姆·格拉吉蒂（JIM GERAGHTY）报道：

在今天的《华尔街日报》专栏中，前国务卿迈克·庞培奥（Mike Pompeo）与其中共国最高政策顾问余茂春（ Miles Yu）写道，“中共国实验室的肆意妄为使世界处于危险之中”，并且直截了当地宣布冠状病毒大流行是武汉病毒学研究所发生事故的结果。

庞培奥和余并没有指出任何新的或惊人的证据，但他们确实指出了一些没有引起足够重视的间接证据。也许最值得注意的是，专栏声明：“ 在2021年1月，美国国务院证实，在2019年秋天在武毒所人们已经严重的病倒了，并且武毒所与中共军方进行了秘密生物武器研究。”

这个得到了国务院发布的情况说明书的反应，从警告性告示开始，美国政府不知道该病毒是如何首先跳入人类的，但随后继续“美国政府有理由相信武毒所内部的几名研究人员于2019年秋季生病，在第一个确定爆发病例之前，其症状与COVID-19和常见的季节性疾病一致。”

在上周日，川普总统的前国家安全副顾问马特·波廷格（Matt Pottinger）在哥伦比亚广播公司新闻的《面对国家》中说：

我们有充分的理由相信中共军方正在同一实验室进行秘密的动物实验，至少要追溯到2017年。我们有充分的理由相信，2019年秋季在武汉病毒学研究所工作的研究人员中爆发了类似流感症状的疾病，但这没有立即成为第一个有案可查的案件被曝光。

中共国于1984年签署了《生物武器公约》，该法令禁止生物武器的开发、生产、获取、转让、储存和使用生化武器。

美国政府不认为中共国声称它完全遵守《生物武器公约》。在2019年，美国国务院发布了有关遵守军备控制协定的最新报告：

在报告期内，中华人民共和国从事了具有潜在双重用途的生物活动，这引起人们对其遵守《生物武器公约》的关注。此外，根据《公约》第二条的要求，美国没有足够的信息来确定中共国是否取消了已评估的生物战计划。

报告补充道，“来自中共军事医学机构研究人员有关可用的研究信息，通常会识别出可能具有异常的生物活动，因为演讲讨论了识别、鉴定和测试具有潜在双重用途的多种毒素。”

这不足为奇，不寻常或必然违反条约的中共国政府对研究潜在危险的病毒特别感兴趣。

2003年广东省爆发首例非典(SARS)病毒 — 距武汉大约500英里。 即使在最初的非典(SARS)疫情消失后，非典(SARS)对中共国公民的威胁仍持续存在。2004年，非典(SARS)逃离了中国疾病预防控制中心的北京研究实验室。这是两次事件。

2017年，研究人员在云南省“确定了单一的马蹄蝙蝠种群，其中藏有带有非典(SARS)的所有遗传构件的病毒株” — 距武汉大约740英里。为了确定非典(SARS)爆发的根源, 数以千计的马蹄蝙蝠及其血液和病毒样本采集由 … 武汉病毒学研究所的石正丽和崔杰。

武汉病毒学研究所保存了“几百”种在蝙蝠中发现的冠状病毒样本，但是武毒所的工作人员坚持认为SARS-CoV-2与他们的任何样品都不匹配。武汉市疾病预防控制中心单独地收集了他们自己的蝙蝠中的冠状病毒样本；如华盛顿邮报所述，“一个在2019年12月发布的视频显示田俊华（武汉疾病预防控制中心杰出的研究员）在没有适当防护设备的情况下对蝙蝠进行野外研究” 。 但是，许多人继续坚持认为该病毒是从第三种非实验室自然来源中产生的，这种来源涉及与武汉市相距数千英里的一种蝙蝠。

武汉病毒学研究所事实可能一直在与中共人民解放军合作进行“双重用途”研究，但这并不自然而然的意味着该病毒是一种生物武器，或者是有意的被用作为一种生物武器。许多病毒研究都适合双重用图，甚至功能增强研究，“增强潜在的大流行病原体的致病性或传播性” — 换一种说法，使其更加危险。一些病毒学家争辩说，如果你想知道如何阻止致命且具有传染性的病毒，你需要在实验室中研究致命和具有传染性的病毒。其他的争论在于意外暴露或释放的风险太高，无法进行此类研究。 如果中国政府正在武汉病毒学研究所研究病毒以作为将来的潜在生物武器使用，那么就违反《生物武器公约》，这就可能解释了中共政府为了压制有关疫情爆发的新闻而做的努力，和自大流行开始以来，普遍的、令人生疑的秘密活动。
