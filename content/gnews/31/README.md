Veritas计划是一个专门为爆料者提供平台，挖掘真相的一个组织。今年5月份Veritas计划联合Facebook的一位内部人士公开了Facebook在用户不知情的情况下，暗自删除用户评论控制用户流量的行为。很显然，Facebook禁言和封锁的标准除了在用户守则那里明确列出来的之外还有一个更重要的标准，那就是该用户是否发表了新冠病毒疫苗的负面言论，不管那个言论是不是真实的，不管那个言论是不是得到Facebook官方认证的。

这位爆料者告诉我们被封锁的人包括医疗专家、科学家、世界领导甚至还包括澳大利亚的国会议员。Facebook使用评级系统来给用户的言论打标签，以识别该言论是否构成了疫苗怀疑现象，这一切都是在用户不知道的情况下完成的。

疫苗怀疑现象是Facebook最新发明的一个词汇。疫苗怀疑所指的是你对疫苗的质疑程度。质疑程度的涵盖范围很广我在这里举几个被Facebook认定为疫苗怀疑的例子：

1. 如果疫苗是那么的有效，我们为什么还要坚持戴口罩和其他人保持距离
2. 我知道有个打疫苗的人死了
3. 如果你曾经感染过新冠病毒，则你不需要再去打疫苗
4. 这是一个实验性疫苗它有很多副作用
5. 疫苗不能保护你感染新冠病毒


Facebook还会给用户的留言评分，他们将这个分数称为VH分数，VH分数（vaccine hesitancy score）就是疫苗怀疑分数。Facebook再根据这个分数来决定是否将降低这个留言的曝光率，还是放着不管，这一切都取决于留言的内容。所以当参与这个工程的软件工程师编写算法的时候，他们会让Facebook检查这些留言寻找其中和疫苗有关的关键字。然后再给这个留言打分。爆料人表示现在这个程序还在测试中。

这位爆料人还指出Facebook想要在全世界范围开展这个程序，他们会为他们能找到的所有的语言开设这个程序，就像是给全人类开展的程序一样。

所有参加这个项目的人不光具备信息行业的技术，他们所有人同时也在Facebook健康部门工作。他们编写的程序把用户分成了5个等级。这个评分系统是他们如何审查留言的标准，他们会根据留言的内容来决定怀疑疫苗的程度有多少。他们还建立了一个应急系统，这个应急系统直接和Facebook的人工审查员挂钩，他们的算法无法决定某一个留言的时候，他们就把这个留言交给人工审查员，他们会下最终评判。
