# **又变朋友了？中美将在阿拉斯加举行最高级别“重启”会议**

即使他没有像前任那样咄咄逼人，但美国总统拜登已经向美国人民承诺，他将保留川普总统对北京当局的怀疑态度，而北京当局被普遍认为是美国最大的地缘政治敌人。

自一月份来到华盛顿以来，新政府决定维持川普对华贸易战中的关税（尽管各翼全球主义“专家”们宣称贸易战将抑制经济增长并最终损害美国工业），这让领导层十分懊恼。

[!\[\]()!\[\](https://spark.adobe.com/page/GGaS3eHv3MXBQ/images/82555071-a18c-4e69-b91f-ec4c09968608.jpg?asset_id=1de07f0a-ecc9-4fa9-ae18-56e8cb94c4af&amp;img_etag=%2271ccb705d299bb69c7881d40511d94d5%22&amp;size=2560)](https://spark.adobe.com/page/GGaS3eHv3MXBQ/images/82555071-a18c-4e69-b91f-ec4c09968608.jpg?asset_id=1de07f0a-ecc9-4fa9-ae18-56e8cb94c4af&amp;img_etag=%2271ccb705d299bb69c7881d40511d94d5%22&amp;size=1024)

与此同时，习近平主席正紧锣密鼓地推进香港去民主化的计划，而习近平和其他中共高官则在中共将其目标定为夺回台湾的同时，增加了鼓吹战争的言论。

然而，考虑到拜登及其家人与中共国的长期关系，北京当局寻求与华盛顿关系的“重启”也就不足为奇了。

拜登家族与中共国的关系一直有据可查。拜登的儿子亨特•拜登(Hunter Biden)不仅参与了与中共关系密切的个人组织的商业活动，还参与了被称为兜售影响力的活动。而且由于拜登在参议院和副总统的经历，这使他与习近平主席有着长达一年的关系。一些人甚至认为，两位领导人关系友好。

北京想利用这一点也不足为奇。在中共国寻求在拜登领导下“重启”关系之际，《南华早报》报道称，两国正准备进行第一轮高层面对面会谈。

这些会谈将有双方重量级人物参加。美国代表团将由国务卿安东尼•布林肯(Antony Blinken)率领，中共国代表团将由中共国最高外交官杨洁篪主持，他经常担任习近平主席的特使。最近几周曾多次对美国发表激烈言论的中共国外交部长王毅也可能会露面。

虽然据报道，拜登和习近平上个月通过电话交谈了几个小时，但“重启”会议将是新政府抵达华盛顿后的首次面对面会谈。布林肯最近声称，华盛顿与北京的关系“可能是世界上最重要的”。

不出所料，中共国政府及其喉舌对会谈的前景感到兴奋。《环球时报》主编胡锡进在推特上表示，他希望这一消息是准确的。显然，这条消息是中共国官员向《南华早报》泄露的，或许是为了向拜登团队施压，迫使其同意会谈。

[!\[\]()!\[\](https://spark.adobe.com/page/GGaS3eHv3MXBQ/images/f25aa6a3-b22f-4368-adb2-fa56de0e41d2.png?asset_id=9b9e1e7b-7376-4e70-a90b-93c7340bd5c5&amp;img_etag=%22d0fc5f15174303ba769c089f03d12493%22&amp;size=2560)](https://spark.adobe.com/page/GGaS3eHv3MXBQ/images/f25aa6a3-b22f-4368-adb2-fa56de0e41d2.png?asset_id=9b9e1e7b-7376-4e70-a90b-93c7340bd5c5&amp;img_etag=%22d0fc5f15174303ba769c089f03d12493%22&amp;size=1024)

即使拜登政府对北京对待新疆维吾尔族的问题大加抨击，但这次双边会晤仍可能标志着双方向减少对立关系迈出了第一步。

*复旦大学美国研究中心教授韦宗友也认为，在拜登政府对川普的中共国政策进行重新评估的情况下，如此高层次的会晤可以为两国关系定下基调和方向。*

*韦宗友说：“双方很可能就如何协调中美之间的竞争以及加强务实合作进行磋商和讨论。”*

*韦宗友说：“高层会晤也将表 明，两国领导人不希望双边关系继续走川普时代的对抗之路。”他认为此次会议还有可能为习近平和拜登之间的未来会议奠定基础。*

值得注意的是，《南华早报》称，此次会谈很可能在阿拉斯加安克雷奇(Anchorage)举行。为什么是这里？好吧，显然，中共国热衷于阿拉斯加，因为它离华盛顿和北京的距离相等，同时也在美国大陆之外。理论上，这将使政府及其宣传机构更容易将会谈描述为在中立领土上进行，同时还会使外交官避免受到媒体的严格审查。

外交官之间的合影几乎可以肯定会放出。但即使会谈进展顺利，在此时此刻，会谈对于阻止太平洋地区的新“冷战”也无济于事。彭博社周二早些时候报道称，解放军正在测试可以瞄准美国航母的新型导弹，而这些航母执行川普领导下持续进行的南中国海“航行自由”行动，这极大地激怒了北京的政治机构。

但拜登政府仍需做出选择：它可以坚持川普政府确立的强硬路线，也可以恢复到奥巴马时期的绥靖状态。
