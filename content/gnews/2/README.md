8.中国贸易顺差扩大，人民币为何没有大幅升值？

中国8月份出口同比增长9.5%，好于预期，进口则有所下降。这在外汇市场上引发了一个挥之不去的问题：贸易顺差不断扩大之际，人民币为何没有更多地升值？

全球死亡人数超过890,000

全球冠状病毒死亡人数已经超过了另一个严峻的里程碑，超过了890,000，而[约翰·霍普金斯大学的](https://coronavirus.jhu.edu/map.html)研究人员确认了890,260 。

由于某些国家的测试率和定义不同，时间滞后以及可疑的漏报率，真正的死亡人数可能更高。

就死亡人数而言，以下是全球十个受影响最严重的国家：

- 美国：189,166
- 巴西：126,960
- 印度：71,642
- 墨西哥：67,558
- 英国：41,643
- 意大利：35,553
- 法国：30,732
- 西班牙：29,516
- 秘鲁：29,838
- 伊朗：22,410
