**西班牙2020年12月9日电/西喜社——**共和党27名国会议员在周三致川普的一封信中敦促川普总统让总检察长威廉·巴尔任命一名特别顾问来调查选举违规行为。 德克萨斯州众议员兰斯·古登（Lance Gooden）上周给总统写了一封信，信中只有他的签名，后来又从共和党议员那里收集了更多签名以支持这项工作。

古登在巴尔表示司法部尚未在11月3日的总统选举中发现选民欺诈行为足以改变投票结果的事实后发出此信。

信中写道：“美国人民应该就悬而未决的不确定性做出最终解决，但选民欺诈的合法问题仍然没有得到答案。”

得克萨斯州的路易·高莫特（Louie Gohmert）和阿拉巴马州的莫·布鲁克斯(Mo Brooks)也在签署这封信的其他共和党成员中。

“司法部已要求在多个场合被要求调查此事，司法部的不作为以及司法部长发表的公开声明表明，缺乏意愿调查您的竞选团队和其他全国的胜选官员指控的违规行为。信中还写道。  “任命特别顾问将建立一个调查人员团队，其唯一职责是揭露真相并提供美国所需的确定性。”
