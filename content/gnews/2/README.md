## 一个孩子都养不起，你还要我生三个？
` 文名` [轉載自GNews](https://gnews.org/zh-hans/1288189/)

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/01024153/Magnus-Liam-Karlsson-imagebank.sweden.se-magnus_liam_karlsson-fathers_and_parental_leave-6575-edited.jpg)图片：Magnus Liam Karlsson/ imagebank.sweden.se

作者：英国伦敦喜庄园 文名

笔者在瑞典，传说中已经进入社会主义的国家。在这里生娃的福利不要太好，简直是孩子的天堂。

1. 生孩子免费（收约500元的助产费）
2. 生完孩子有每月1000+元的补助，直到孩子18岁。
3. 父母共有480天带薪产假，专心在家带娃。
4. 上学完全免费，托班每月最高500元左右。
5. 一直上到研究生都是免费，博士还给发工资。
6. 孩子生病医疗完全免费，成人每年医疗上限两三千块，超出部分政府买单。
7. 而且孩子权益被保护到变态，不准打不准骂，孩子不开心可能就会招来社工剥夺父母抚养权。


这些东西说出来真是太拉仇恨了，毕竟是从出生到坟墓的福利。听说某些移民家庭就专心生娃，靠孩子的补助也能维持全家的生活。而且我还听说独立自主的北欧女性，有些会出国寻找艳遇，怀孕后回国自己一个人生下来抚养成人。
这些匪夷所思的事情，先决条件就是，国家解决了他们的所有后顾之忧。

换在中共国呢？
我的孩子在上海出生和长大，对国内的教育也算深有体会。咱就别想上面瑞典人这些变态的福利了，在中共国几乎没有不让人担心的事情，各方面能让人稍微省点心就是烧高香了。

1. 担心安全。从来不敢让孩子自己一个人出门，生怕遇上人贩子或报复社会的或变态。
2. 担心学校。老师要小心伺候，害怕给孩子穿小鞋。上班伺候好老板，下班还要伺候好微信群的老师。
3. 担心升学。要报这个班那个班学十八般武艺琴棋书画，上海有些小学要面试才艺，还要审查父母甚至爷奶背景才能排上队。学区房这种重氪选项老百姓就不用想了。
4. 担心洗脑。孩子在学校被抹掉天性，最可怕的是变成小粉红。我有一个朋友说每天孩子回家都要对其进行反洗脑，把老师教的东西洗掉。
5. 担心健康。吃的喝的都是不知道咋做出来的化工产品，呼吸的空气都是不知多少倍的PM2.5。这种环境猜猜看多少岁开始患癌症？
6. 担心养不起。学费、补习班、吃穿用度。学校组织个演出还要买专门的演出服。从孩子出生到大学毕业，得花多少钱？不敢算。政府管你吗？


一个字，累！
这一切都压在两个夫妻身上，谁愿意生呀！
一边是瑞典的无忧无虑毫无负担，一边是中共国的N座大山压得人喘不过气。
瑞典是生得越多，福利越多。中共国是生得越多，压力越多。

人家瑞典是取之于民，用之于民。高达57%的税率，但是实实在在每一分钱都用在老百姓身上了。
中共国呢？就差点连根拔了，民脂民膏都被老杂毛家族搜刮干净洗到国外了，剩下的钱都拿来维稳了。平时对韭菜不管不顾，连根割完还嫌韭菜长得慢要拔苗助长了。韭菜唯一的选择就是含泪躺平了。
