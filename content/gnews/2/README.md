## 【世事解评】中共的区块链服务网络BSN

香草山金融部 文唐

2021年3月，美国外交关系委员会CFR 发布一份重磅报告，全面系统地揭示了中共“一带一路”战略在经济、政治、气候变化、安全和健康等各方面对美国利益的威胁(1)(2) 。在金融科技(fintech)板块，报告着重分析了 CCP 全球扩张的两大核心武器，一个是数字人民币 DCEP，另一个是区块链服务网络 BSN。网络上关于 DCEP 的解读很多，BSN 却很少被提及。

BSN，全称 Blockchain-based Service Network 区块链服务网络，由国家信息中心、中国移动通信集团公司、中国银联股份有限公司、北京红枣科技有限公司(5)共同发起，其目标是成为部署和运行区块链应用的全球性公共基础设施网络(3)。
![]()![](https://gnews.org/wp-content/uploads/2021/04/Picture1-35.png)
区块链（Blockchain）技术伴随着比特币诞生于2008年。比特币是电子货币，区块链就是电子账本。通过一个公开的账本，让全民参与记账，解决电子支付记账的难题，从而彻底实现了去中心化的电子货币：所有节点平等共享所有信息，完全不依赖银行等中心机构。

随着比特币的流行和火热，越来越多的人才加入到区块链的研究和改善中。后起之秀以太坊 ETH， 更是在比特币区块链的基础上，提出了智能合约（Smart Contract）的概念，大大拓展了区块链技术的应用场景。智能合约不仅仅是简单的记账，更可以做数字身份认证、法律合同签署、租赁融资交易等等。由区块链保证合约的强制执行，去掉了律师、平台等第三方媒介，提高可信度的同时降低交易成本。

我们平时谈论的各种币：比特币、以太坊、狗狗币，每个币背后都有其相应的独特的区块链网络。而在这些公开的区块链网络之上，还构建有许许多多的应用和服务。可以说，币也是链上的服务之一，币不能脱离链而存在；链却可以没有币，而专注于其他的服务。当前基于区块链的服务还在试水探索阶段，但已经展现出蓬勃的发展潜力，不远的将来可能我们交水电费、存款借款、收入纳税都会在不同的区块链上记录、公证、溯源。
![]()![](https://gnews.org/wp-content/uploads/2021/04/Picture2-11.png)
然而，构建和维护一个公共的区块链门槛是很高的。这涉及到核心算法、IT架构、网络通信、安全攻防等等方面。如果有一家公司可以把这些基础做好，开发者只需要关注顶层的区块链服务，那自然是极好的。BSN 正是要做区块链服务的基础设施，从而大大降低开发区块链服务的成本，促进区块链生态的繁荣。就像在互联网行业的初期，搭建一个网站门槛很高，需要懂网站设计、服务器运维、网络内容分发等等。但随后亚马逊做了云计算基础设施AWS，将这些底层的技术统统打包，使得搭建一个网站变得非常简单。

但是，把自己的服务构建在他人的基础设施之上，意味着自己的数据和命脉也被他人掌控。在 2020年美国大选中，众多 Trump 支持者聚集在社交媒体 Parler 上积极发声，而在大选的关键时刻，Parler 直接被 AWS 关停了服务器。CCP 强推 BSN，做区块链服务的基础设施，很明显是看中肆意左右他人命运的极权。

如果 BSN 全球推行结果会是怎样？BSN 就像华为的5G网络，不直接面向公众，却可以监视公众的一举一动。更为严重的是，BSN 监视的是每个人金钱相关的服务，每一笔交易在什么时间、什么地点、什么人干了什么，透过 BSN 都可以看的一清二楚。通过大数据挖掘和人工智能技术，可以了解一个人的行为方式、生活习性，甚至预测未来的行为走向。

同样对于使用 BSN 服务的企业，所有的核心数据都会被CCP掌握。而且，CCP 通过控制 BSN 拥有最终的裁决权：反共的个人，如香港的勇士，不配合中共的企业，如拒绝新疆棉花的 Nike 、H&M，都会被 BSN 封禁，无法享受正常的服务、链上的财产都会被剥夺；甚至一些国家的区块链基础设施依赖 BSN，中共就有能力在 BSN 上封杀这个国家。而一带一路上的国家，大多数没有能力独自构建区块链基础设施，面临着中共的哄骗、威胁和控制，中共也得以全球推行其老大哥式的极权监管。
![]()![](https://gnews.org/wp-content/uploads/2021/04/Picture3-12.png)
区块链技术是伟大的发明，CCP 发掘和利用区块链技术，来实现其全球称霸的野心，注定失败。灭共之后，更多积极的、受监管的基于区块链技术的创新会迸发出来，继续影响和深入到我们未来的日常中，成为数字金融、数字生活的一部分。
