## 澳洲护士起诉强制接种中共病毒疫苗的雇主
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1036855/)

翻译与评论：Shirley
![]()![](https://gnews.org/wp-content/uploads/2021/03/vaccine1.jpg)网络图片
《澳大利亚人报》报道，针对澳洲昆士兰首府布里斯班发生的雇员因拒绝接种中共病毒疫苗而遭解雇一事，各组织与部门正在检视其合法性。

一名已工作了十年的前“澳洲护理提供商”（Ozcare）护士，因拒绝接受中共病毒疫苗注射被解雇后，已将该护理提供商告上“公平工作委员会”（ Fair Work Commission）。

64岁的护士格洛弗（Maria Glover）说，她小时候在菲律宾时，对流感疫苗有过敏反应，此后再没有接种疫苗。前几次她的拒绝都被Ozcare接受了，但该公司去年收紧了政策。

格洛弗女士和Ozcare之间的案件正受到工会、行业团体和政府的密切关注。昆士兰州州长帕拉斯祖克（Annastacia Palaszczuk）此前曾表示，疫苗可能会对养老护理等高风险行业进行强制接种，但大多数政府已经排除了强制接种疫苗的可能性。

周一在布里斯班举行的听证会上，代表Ozcare的大律师霍华德（Leigh Howard）表示，强制接种使老年护理行业的流感死亡人数稳步下降。这场疫情大流行使该公司的一些客户，拒绝让护理人员进入他们的家，Ozcare有义务优先考虑客户的安全。格洛弗女士则表示， “我明白Ozcare对他们的客户有责任，但对员工也应该有责任。”

格洛弗女士拒绝了Ozcare要求医生评估她所谓的过敏症的请求。共有35名Ozcare员工拒绝接种疫苗，其中15名员工屈服并接受了注射，6名辞职，10名被解雇。包括格洛弗女士在内的4名员工对接种疫苗提出了医学上的反对意见。

“公平工作委员会”的裁判专员亨特（Jennifer Hunt）援引此案中 “真正的公众利益”，表示她将优先考虑对此案的决定。预计她将在4月得出调查结果。
