作者:举头三尺外星人

ccp病毒全球肆虐到目前为止已经一年多了，最近通化突然爆发的疫情不过是中共控制住疫情谎言的又一次纸包不住火式的败露，通化的疫情爆发绝不是中共所谓的18日，如此大规模的爆发按照新冠的传播的方式和时间，疫情至少在1，2个月之前就开始大规模传播，我们比较武汉疫情，武汉疫情是在10月份就开始传播了，武汉的官员忠实的执行封锁信息，警告医疗单位，控制网络媒体，网络五毛引导舆情，甚至直接恐吓或者抓捕控制传播的真相的人等手段成功的让武汉人民成了瞎子聋子哑巴并把ccp病毒传播至全世界。

这次通化的灾难我们看到通化绝对是武汉疫情的重演，不但从灾难发生的方式到中共政府整个舆论控制的的手段，完全就是复刻了武汉的控制方式都不带一点点修改的。
![]()![](https://gnews.org/wp-content/uploads/2021/01/1文章配图1-2.png)（图片来自百度截图）
不过这次通化疫情由于大量民众在网上传播真相，中共没有第一时间控制住，结果一时间网络舆论震荡，中共有点措手不及，马上于封城第二日19号开始了对传播真相的民众进行了大抓捕，19日就有3至4人被以散布谣言的名头被公安局抓捕，跟武汉的处理方式如出一辙，武汉警方在1月初以散布谣言的名义，传唤了8名在网络上传播有关疫情信息的人员。并在后多日宣称该疫情“可防可控”，没有新增病例。这是中共解决问题一惯常的手段：“解决不了问题就解决提出问题的人”。

根据网上报道：“2021年1月19日，通化市公安局东昌区分局龙泉派出所接到线索，辖区居民王某在疫情期间通过微信群发布不实信息。为防止谣言扩散，龙泉派出所立即展开调查，并对王某进行询问，王某对其虚构事实扰乱公共秩序的违法事实予以承认。根据《中华人民共和国治安管理处罚法》第二十五条第一款之规定，东昌公安分局决定给予王某行政拘留10日，并处罚款500元的行政处罚。”
