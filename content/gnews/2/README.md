# 重要事件

- **美国之音：**[**美参议院通过抗衡中**](https://www.voachinese.com/a/us-senate-passage-innovation-competition-act-20210608/5921902.html)共[**国的一揽子法案**](https://www.voachinese.com/a/us-senate-passage-innovation-competition-act-20210608/5921902.html)


美参议院通过了耗资大约2500亿美元的《美国创新与竞争法》(American Innovation and Competitiveness Act)，通过加大对科技领域的投资来赢得与中国的竞争。参议院多数党领袖舒默说，这项立法将成为美国参议院近期历史上最重要的跨党派成就之一，为美国在今后一个世纪的领导地位奠定基础。美国总统拜登对该法案在参议院获得通过表示赞赏，称美国必须保持其作为地球上最具创新力和生产力的国家的地位。

中国全国人大外事委员会星期三发表声明，指责美国国会参议院这项法案“打着创新和竞争旗号，干涉中国内政，妄图遏制中国发展”。声明说，“处处把中国当作假想敌的做法，不得人心，注定失败。”

在经历了长时间的幕后谈判、一系列最后的修改以及一整夜的谈判等戏剧性发展后，美国参议院星期二以68票对32票通过了这部一揽子法案。

这部得到两党支持的法案将加大对科学研究和高科技制造的资金投入，增强美国的竞争力并推动经济增长。

法案是由多数党领袖、来自纽约州的民主党人查克·舒默(Chuck Schumer)和来自印第安纳州的共和党参议员托德·扬( Todd Young)联袂发起的，得到包括全体民主党参议员在内的两党参议员的联署。

舒默在参议院就这项法案进行表决之前发表讲话说，现在的世界比二战结束以来的任何时候都更具竞争力。如果美国什么都不做，美国作为占主导地位的超级大国的日子可能就要结束了。

舒默说，在世界各地，威权政府认为，像美国这样争吵不休的民主国家无法在国家优先事项上团结一致，而他认为，他们是错误的。

“我相信这项立法将使美国在未来的产业中在创新、生产和竞争方面超越世界。我相信，在这个议事厅里，两党在这项法案上的努力已经表明，我们都相信，另一个美国世纪即将到来，”他说。

法案共同发起人托德·扬对参议院通过这个他所说的“具有里程碑意义的法案”感到自豪。

这位共和党参议员还表示，“让历史记录，这一刻，我们团结在一起。通过面对今天的挑战，我们为美国人建立了更光明的明天。”

美国总统乔·拜登(Joe Biden)也发表声明，对参议院团结在一起通过这项法案表示赞赏。

“我们处在一场赢得21世纪的竞赛中，而且发令枪已经响了。随着其他国家继续投资于自己的研发，我们不能冒落后的风险。美国必须保持其作为地球上最具创新力和生产力的国家的地位，”拜登总统接着说。

拜登总统说，在众议院通过这项立法后，他会尽快将它签署为法律。

- **路透社：[拜登撤销WeChat、TikTok以及支付宝和微信支付等应用的相关禁令](https://cn.reuters.com/article/biden-wechat-tiktok-ban-0609-wedn-idCNKCS2DM01W)**


美国总统拜登周三撤销了特朗普时代试图禁止新用户下载WeChat和TikTok的一系列行政令，并下令商务部审查这些应用和其他应用带来的安全担忧。

前总统特朗普的政府曾试图阻止新用户下载这两项应用，并禁止TikTok和WeChat的所有其他技术交易。两家公司均表示，这些行政令将实际上阻止这两项应用在美国的使用。

法院阻止了这些从未生效的行政令的实施。TikTok拒绝置评。WeChat没有立即置评。

WeChat用户联盟(WeChat Users Alliance)的首席律师Michael Bien称赞拜登政府撤销了“对WeChat的错误禁令……指禁令会导致美国数百万人依赖的一个主要通讯平台关闭，这是从来未曾发生过的事。”

拜登的命令称，收集美国民众数据“可能会让外国对手获得这些信息。”

该命令指示商务部“持续评估”任何“对美国关键基础设施或数字经济的安全性或韧性造成灾难性影响的不当风险”的交易。

拜登的行政令要求美国情报和国土安全机构在商务部进行审查时，在60天内向商务部提供对外国对手控制的美国数据的脆弱性和威胁评估。

共和党参议员Josh Hawley在推特上表示，撤销特朗普的命令是“一个重大错误－显示了在中国获取美国人个人信息以及中国日益增长的企业影响力方面令人担忧的自满情绪。”

- **美国之音：[美国防部长下达内部指令 展开多项行动抗衡中共国](https://www.voachinese.com/a/dod-issues-internal-directive-on-counter-china-20210609/5922867.html)**


美国国防部长劳埃德·奥斯汀(Lloyd Austin)下达了一道内部指令，要求展开几项抗衡中国的行动计划。至于这些新倡议与已经展开的行动有多大不同，美国官员没有披露多少细节。

美国几年来一直把抗衡中共国作为国家安全政策的核心。两国在从科技、人权到中共国在南中国海的军事行动等各个方面都在角力。

美国总统乔·拜登(Joe Biden)今年2月下令五角大楼成立一个特别工作组，奥斯汀的这项指令是这个特别工作组提出的建议的结果。

一位要求不透露姓名的高级国防官员在对媒体做吹风介绍时说：“来自部长的这项指令最终是关于整理好本部内务，并确保它能够应对已阐明的有关中国的优先事项。”

五角大楼的声明说，这些行动将改进军方的能力，以“重振我们的盟友与伙伴网络，增强威慑、并加速发展新的作战概念”。

五角大楼没有披露这些行动计划究竟是什么的细节。

上个月，拜登政府的国防预算要求把大批用于老旧系统的开支转用于军队现代化，以遏阻中国。

50多亿美元将用于旨在抗衡中国的“太平洋威慑计划”( Pacific Deterrence Initiative)并聚焦在印-太地区的竞争，目的在于通过为雷达、卫星和导弹系统拨款而提高美国在印-太地区的战备状态。

但是一些议员和专家批评说，这部预算案对来自中国的近期威胁聚焦力度不够。

悉尼大学美国研究中心的汤曾德(Ashley Townshend)对路透社说：“拜登政府对研发大量投资，以确保军队在未来高端冲突中的竞争优势，这一做法虽然是正确的，但是这不应当以损害增强针对中国的短期常规威慑力为代价。”
