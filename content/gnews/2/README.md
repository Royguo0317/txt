**【****10.1 ****集结案判刑】量刑起点高达两年　历来最高　张达明：判刑准则或违背终院判例**

壹传媒创办人黎智英、李卓人及何俊仁等10 名民主派人士，被指前年10 月1 日组织及明知而参与未经批准集结等罪，各被告早前认罪，今( 28 日) 被判囚14 至18 个月，当中单仲偕、蔡耀昌获判缓刑。其中就煽惑及组织未经批准集结的罪名，法官胡雅文指考虑到案情严重性，均以２年为量刑起点。法官又指，涉案游行路线曾出现堵路、纵火等暴力事件，虽然并非于被告身处的地方发生，但他们有份煽动及组织游行，须负上刑责。

港大法律学院首席讲师张达明接受《立场》访问质疑，法官在今次案件采用的量刑准则，或违背终审法院的判例，当中指明在和平集结，个别人士使用暴力，不会引致其他人的集会自由被剥夺。他相信，若有被告就刑期提出上诉，有关争议会是主要争拗之一。

大律师苏俊文形容， 本案的量刑起点定得非常高，“令人感到惊讶”。他又指，留意到近日几乎任何罪行的判决，法官都因为考虑 2019 年的社会状况，“一刀切”越判越重，担心会令外界忧虑《基本法》下的游行示威权利不再受保障。

张达明：法官判刑或违背终院案例

港大法律学院首席讲师张达明就质疑，虽然法官指今次案件的游行路线沿途出现暴力，但在控方未能证明被告有份参与有关暴力的情况下，法官完全不考虑公民抗命元素，采用较重的判刑原则，是否恰当。张达明指，控方必须证明被告明知而有份参与案中所指的暴力行为，才可使用较重的判刑原则。加上未经批准集结罪，罪行元素本身不包含暴力，“如果证明被告有参与暴力，就应该告另一条罪”。

张达明认为，法官今次判决或违背终审法院的判例，相信若有被告考虑就刑期提出上诉，有关争议或成主要法律争拗之一。

欧盟司法专员 Didier Reynders 表示，新冠肺炎疫情引来进一步数码化，带来新风险，特别是对于脆弱的消费者群。他表示，欧盟严禁针对儿童和未成年人的隐藏广告，例如在影片中的横幅。

欧洲消费者组织(European Consumer Organization，BEUC) 于今年2 月，曾向欧盟委员会提出投诉指，TikTok 部份服务条款不公平，又指TikTok 未能保护儿童和青少年免受隐藏广告或潜在的有害内容影响。

**掌掴余波｜中共国妻子掌掴服装店员惹外交风波****比利时驻韩大使今夏离职**

比利时驻南韩大使馆周五表示，比利时外交部决定今年夏天解除驻韩大使莱斯寇河（Peter Lescouhier）的职务。莱斯寇河的中共国妻子相雪秋（Xiang Xueqiu），今年4月逛首尔一间服装店时，因不满女售货员误会她偷窃，竟怒掴售货员，令其半张脸被打致红肿，事件在韩国掀起轩然大波，莱斯寇河事后曾代妻道歉。

首尔警方事后就事件立案调查，但相雪秋声称身体不适住院，因而延后接受调查。比利时驻韩大使馆周五透过Facebook透露，比利时外交部确认相雪秋在出院后，已亲自向两名受害店员私下会面并道歉，而在警方要求下，相雪秋放弃外交豁免权，将在健康状况许可下全力配合警方调查。

不过，韩联社向韩国外交部官员确认有关言论时，外交部指比利时政府所称“放弃外交豁免权”，仅限于接受警方调查的部分，并不代表会接受法院审判等司法程序。

此外，比利时驻韩大使馆声明中亦提到，莱斯寇河赴任至今3年，而在这次风波之下，难以继续顺利执行大使职务，比利时副首相兼外相维尔姆斯（Sophie Wilmes）决定结束莱斯寇河的任期。考虑到相雪秋后续接受调查等因素，莱斯寇河夫妇预计今年夏天返回比利时。
