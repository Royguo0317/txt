## 【军事快讯】美军航母上的F-35足以震慑俄罗斯和中共

![]()![](https://gnews.org/wp-content/uploads/2021/04/图片88.png)

2020年12月，美国海军陆战队首个具有舰载能力的F-35C Lightning II战斗机中队具备了初始作战能力，这是其在航空母舰上首次部署之前的一项关键认证。海军陆战队攻击中队（VMFA）314在今年早些时候退役了原有的F / A-18A / C黄蜂之后，成为第一个过渡到洛克希德·马丁联合攻击战斗机的F-35C变型的海军陆战队中队。

该中队隶属海军陆战队11机队，在2020年的大部分时间里将飞行员和维修人员转移到了新的单座第五代战斗机上，该战斗机配备了比F/A-18s更为强大的导航、航空电子、武器系统和传感器。作为海军陆战队2019年航空计划的一部分，该服务宣布将为四个舰载中队采购67架F-35C。但是，在2020年，海军陆战队还宣布，正在考虑将每个中队的喷气机数量从16减少到10。

美国海军陆战队还使用了F-35B，这是“联合打击战斗机”的短距起飞和垂直着陆（STOVL）变型战机，可以在短跑道以及平底两栖攻击舰（LHA）上操作。

F-35C被开发为美国海军的第一个低可见舰载舰载航空平台，它正在取代海军的F / A-18C / D黄蜂，成为舰载打击组织（CSG）的主要进攻战斗机。多用途F-35C还可以用于CSG的近距空中支援、地面攻击任务和空中防御。

美国海军是F-35C的最大客户。这种联合攻击战斗机的变型机拥有该平台中最大的机翼，并具有更坚固的起落架–这些特点使其非常适合弹射发射以及在航母上进行飞行逮捕。F-35C的翼尖也可以折叠，以便在部署时在航母的甲板上有更多的空间。

F-35C还拥有联合攻击战斗机中最大的内部燃料容量，每架飞机可以携带近2万磅的内部燃料——这比目前服役的任何一种战斗构型的战斗机都能提供更长的航程和更强的持久性。与美国海军陆战队所采用的F-35B一样，F-35C可以利用探针和垂尾加油，这使海军可以在距离威胁的安全距离内操作其航母，但仍然可以使战斗机到达远程目标。

除了能够在航母甲板上作战外，F-35C仍然具备隐身、融合传感器和可靠性等第五代能力。它极低的可观察隐身性能使该机能够躲避敌方的探测，并在反潜和竞赛环境下作战，同时提高了杀伤力和生存能力。飞机的有源电子扫描阵列(AESA)雷达，以及分布式孔径系统(DAS)、电光瞄准系统(EOTS)和头盔安装的显示系统也使飞行员对战区的情况有了前所未有的认识。

F-35C可作为情报、监视、侦察资产和战斗管理者，并可与战区所有联网的地、海、空资产共享信息。它可以携带超过5000磅的内部武器或超过18000磅的内外结合武器。这使得F-35C在必要时可以隐身作战，或根据任务参数增加武器装备来提高其杀伤力。

目前F-35C仅由美国海军部门使用。
