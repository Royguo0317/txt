# **美国、英国国会议员说，器官摘取真实存在，而且仍在发生**

**美国推出了一项两党法案，而英国上议院也通过了一项修正案，以打击中共最令人发指的一项犯罪行为。**

[!\[\]()!\[\](https://spark.adobe.com/page/rzICOgw5zXppV/images/764bb10f-ec35-4ca0-b879-bb37c4c50ecc.png?asset_id=806f903a-b3bb-44f5-ae90-16e9f746cc0f&amp;img_etag=%224c1b93144231c0fca5a5ef2dbfa6ff7e%22&amp;size=2560)](https://spark.adobe.com/page/rzICOgw5zXppV/images/764bb10f-ec35-4ca0-b879-bb37c4c50ecc.png?asset_id=806f903a-b3bb-44f5-ae90-16e9f746cc0f&amp;img_etag=%224c1b93144231c0fca5a5ef2dbfa6ff7e%22&amp;size=1024)

美国国会议员克里斯·史密斯（个人简介）

器官摘取是由中国共产党领导的中共集权政府暗中主导的一场悲剧，中共应被追责，但似乎它总能在最后一刻设法逃脱。这就是为什么政府和议会所采取的任何步骤都是宝贵的，都要朝着让公众知道这一可怕现实的方向发展。

2020年12月15日，美国两党通过了一项名为《2020年停止强迫器官摘取法》的法案，该法案是根据参议院参议员托马斯·B·科顿（共和党-阿肯色州）提出的S.5016号法案，以及众议院国会议员克里斯托弗·H·史密斯（共和党-新泽西州）和托马斯·R·索兹（Thomas R. Suozzi）（民主党-纽约州）提出的HR 8972号法案形成的，以扩大美国政府直接打击器官摘取的权力。

这是一项史无前例的立法，为寻求解决此问题的全新方法开辟了道路。这样，器官摘取的恐怖行为就从被专门小组和非政府组织的认可转移到了世界上最强大、最有影响力的国家的更广泛的公共政策领域的认可。该法案正在等待美国国会批复，随着即将于1月20日就职的新美国政府上台，该法案肯定会获得更多动力。在美国立法机构的两个分支部分中，“人类器官摘取法案”的三个签名者实际上已经在11月3日全部再次当选，并渴望就这个问题继续两党的政治斗争。他们知道与中共国的关系如何对美国具有战略意义，甚至更加相信人权问题应该在与北京打交道中发挥至关重要的作用。

同样非常重要的是，在英国，上议院于1月12日在来自工党的金希斯地区亨特勋爵的倡议下，通过了《药品和医疗器械法案第13条修正案》。根据通过的修正案，英国法案现在禁止“与人类医疗有关的组织或细胞[…]的使用。”正如亨特勋爵明确指出的那样，这是对人体组织非法交易和器官摘取的直接打击，也是对这种“现代奴隶制”的打击，“它已经进入英国供应链，毫无疑问，我们目前是同谋。”

金希斯亨特勋爵（个人简介）

正如亨特勋爵所说：“在国内，该法案提供了一个可以防止英国同谋参与此类罪行的机会，同时也向其他国家发出重要信息。我的修正案旨在解决当前英国人体组织立法中的空白。目前，《人体组织法》不要求对进口的人类组织有适当的许可。另外，对用于医学研究的进口人体组织不要求可追溯性。当前，《人体组织（人体使用的质量和安全）条例》和《人体组织法》都没有要求用于医疗的进口人体组织需要适当的许可。我的修正案授权部长们行使这项权利。我应该解释一下，‘组织’和‘细胞’这两个词是术语，涵盖了用于医疗目的的所有人体材料。这包括器官。”

亨特勋爵说：“现在，“世界越来越意识到中共国从政治犯身上摘取器官。” 他补充说：“这种可怕的罪行是从活着的受害者身上强行摘除器官 — 这一过程将不可避免地导致谋杀 — 中共强摘器官独立调查法庭最近发现这种罪行在中共国正在大量发生，” 而且“目前有[数百万]中共国公民被拘留在劳改营中。据联合国专家估计，特别在新疆地区的难民营中至少有100万维吾尔人被关押。在中共国其他地区，藏族佛教徒，法轮功修炼者和基督教徒等其他少数民族和少数宗教群体也被关押在劳改营中。”

亨特勋爵（Lord Hunt）广泛地借鉴了去年“中共强摘器官独立调查法庭”的结论，并承认终止中共国移植滥用国际联盟提供了无价的支持，然后，亨特勋爵直接呼吁英国政府尽可能“寻求向世界卫生组织施加压力，使他们严肃对待此事。”

亨特勋爵总结说，“我的修正案的通过将是一项重大行动”，因为它赋予“部长制定规章的权力”。亨特勋爵补充说：“这是英国针对中共国可憎做法的一项特定行动”。当然，他还指出，“我们需要看到那些法规已提请并得到议会通过。但是，在国际上，英国的行动将被视为一个标志，也是向其他国家发出的一个真正信号。”
