**一、川普总统正式赦免班农先生**

美国白宫19日川普总统发布赦免令，对143人进行赦免和减刑，这其中班农先生尤为引起媒体的关注，他曾担任过美国白宫首席策略长和顾问，而且是坚定的灭共者，爆料革命的支持者。2020年8月20日，班农因在“We Build the Wall”项目（一个为修建美墨边境隔离墙筹款的民间项目）中涉嫌挪用款项被捕，后被保释。川普总统对班农的预赦意味着班农不需要再接受检察官的调查和问询。本次赦免不包括川普总统本人、家人和朱利安尼先生。

**二、美国政府形成的灭共趋势不可逆转**

在20日美国政权交接之前，川普总统政府通过了一系列的法案和启动对CCP的定性。其中最重要的是中共的“反人类罪”和“种族灭绝罪”，这两个是对中共最有意义的定性和灭共趋势的必然，不管谁是总统、谁是当权政党，包括欧洲等地。全球化的灭共趋势已经不可逆转，真正的进入自动灭共，依法灭共的时期。

**三、接下来中共将会如何被灭**

蓬佩奥国务卿在最近在推特上对中共的批判和直接赤裸裸的揭露CCP武汉病毒研究所的态度，说明美国已经完全掌握了中共在病毒和CCP是全世界最大威胁的绝对证据，并形成了一个坚定的灭共生态圈，这也是郭文贵先生之前预测的一样，全世界都会追究病毒来源的真相，美国总统选举舞弊真相以及各大公司华尔街利益集团舔共的追责。全世界谁想在未来的政治上，经济商业上有所收获，因此灭共会是最重要的条件。

**四、爆料革命的实力和在全世界的信用**

最近美国政府一系列的政策和对CCP释放出来的罪状，明眼人一看就知道这里面很多的情报信息都是来自于爆料革命的建议和资料来源。特别是蓬佩奥国务卿的推特可以看出，爆料革命的实力已经得到了美国政府完全的信任与支持。据郭文贵先生介绍，全世界都在想和爆料革命新中国联邦拉关系，希望可以进行绝对的全面合作，并有着一致的未来发展目标。美国政府马上就要所谓的换届，不管是谁当总统，谁不别想和中共继续勾兑。爆料革命新中国联邦的态度和情报的共享都是大家想拉拢和合作的对象。
