特朗普政府已控制篡改大选结果的服务器，掌握了民主党选举舞弊的实锤证据。根据特朗普总统2018年9月12日签署的一项行政命令，特朗普政府有权查封涉嫌干预大选的所有个人和组织的资产，并采取法律行动。因此，特朗普总统不仅能够成功连任，还将彻底摧毁所有涉嫌操纵大选的阴谋集团。

据以“深入分析和研究时事”闻名的Natural News网站11月14日报道，2018年9月12日特朗普总统曾经签署了一项鲜为人知的行政命令（可在白宫网站Whitehouse.gov找到），标题为 “关于在外国干涉美国大选的情况下实施某些制裁的行政命令”。重要的是，在这份行政命令中，特朗普总统宣布全国进入紧急状态，而且一直持续至今，2020年的大选就是“紧急状态”下进行的。凭借这一行政命令，特朗普总统不仅能够一举翻盘、成功连任，还能彻底摧毁所有涉嫌操纵大选的阴谋集团以及所谓的“深层政府”（deep state）。

特朗普总统在该行政命令中指出，已知部分位于美国境外的人和组织能够 “干扰或破坏公众对美国选举的信心，包括未经授权访问选举和竞选基础设施或秘密传播政治宣传和散播虚假信息”。这种外国干涉美国选举的行为 “对美国的国家安全和外交政策构成了不同寻常的威胁” 。

为什么说这一部分内容，和民主党以及深层政府共同搞出的大规模选举舞弊有关呢？因为此次美国大选使用的两套选举软件，Dominion和Scytl分别来自加拿大和西班牙，这使此次选举舞弊成为 “外国干涉问题” 。据传，Scytl与乔治-索罗斯和民主党有关系，根据The Gateway Pundit的报道，“比尔-盖茨也拥有Scytl的股票”。

文章引用印度媒体Great Game India报道称：“在2020年美国大选被加拿大Dominion Voting Systems公司通过所谓的 ‘小故障 ’操纵后几天，现在Great Game India发现另一家可疑的外国公司也参与干预了美国大选。美国人所投的票是由西班牙一家破产的公司Scytl在西班牙统计的。和Dominion Voting Systems公司一样，Scytl公司在各国的选举舞弊历史悠久，包括在其选举软件中注入后门。这一问题引发专家质疑，为什么将计票这一敏感工作外包给外国公司？一家破产的西班牙公司怎么能在西班牙统计美国人的选票？由于这种大范围的舞弊，美国联邦选举委员会主席特雷-柴纳（Trey Trainor）认为，2020年美国总统选举是不合法的。”

被普遍忽略的是，特朗普总统2018年的行政命令赋予了司法部扣押协助或掩盖这次外国干涉美国选举的共犯——无论个人还是公司——所有资产的权力。国家紧急状态令特别要求扣押 “直接或间接参与、赞助、隐瞒或以其他方式参与外国干涉美国选举的实体的所有资产”。这当然要包括几乎每一个CNN的假新闻主播，每一个大科技公司的CEO，每一个NBC News、《纽约时报》和《华盛顿邮报》捏造假新闻的记者-恐怖分子。他们都是毒害美国、掩盖刚刚发生的外国大规模干涉美国选举行为的同谋。

特朗普的行政命令进一步解释说，资产扣押应针对有以下行为的个人和实体：“为本节(a)(i)小节所述的任何活动或根据本命令被冻结财产和财产权益的任何人提供实质性协助、赞助或资金、物质或技术支持，或为其提供货物或服务；或(iii)为根据本命令被冻结财产或财产权益的任何人所拥有或控制，或直接或间接为其或代表其行事或意图行事。”换句话说，该命令涵盖了每一个被中共腐蚀或买通的人。而根据我们的估计，这至少涵盖包括范斯坦（Feinstein）和希夫（Schiff）在内的80%的民主党议员、参议员。

但这还不是此行政命令所要求的全部内容。在第8条中，该命令解释说，“人 ”还指 “合伙、协会、信托、合资企业、公司、集团、子集团或其他组织”。换句话说，任何协助掩盖或协助外国干涉美国大选的媒体组织都是该行政令的具体目标。在定义部分，行政命令进一步解释道，“选举基础设施 ”是指联邦政府或州或地方政府在管理选举过程中使用的或代表联邦政府或州或地方政府使用的信息和通信技术和系统，包括选民登记数据库、投票机器、投票表设备以及安全传输选举结果的设备。这将包括Dominion投票系统以及最近选举中使用的所有其他投票系统。最后，紧急命令描述了 “外国干涉 ”一词的含义，其意思是：“外国政府或作为外国政府代理人或代表外国政府行事的任何人的任何秘密、欺诈、欺骗或非法行动或企图行动，其目的或效果是影响、破坏对选举的信心，或改变选举结果或被报告的选举结果，或破坏公众对选举过程或机构的信心。”请注意 “改变选举结果或被报告的选举结果 ”等重要字眼。这正是几大科技巨头和假新闻媒体已经和正在做的，不仅仅是犯罪，更是叛国罪，而且正好落入了特朗普在2018年公开宣布的国家紧急状态陷阱，现在所有这些公司和组织的资产可以在一夜之间被查封。

根据今天早些时候播出的众议员路易斯·戈默特（Louis Gohmert）的采访视频，美国军方刚刚在西班牙巴塞罗突袭了现已破产的Scytl公司的服务器集群。

针对11月3日发生的大选操纵阴谋的许多爆炸性报道，都来自The Gateway Pundit。现在，它联系到消息来源，证实了美国军方在欧洲的军事行动与收集窃选证据有关。The Gateway Pundit在报道中称，当美国政府确定Dominion服务器涉及换票时，情报部门马上开始搜索该服务器，并确认其在德国。随后在司法部和国务院的配合下，他们成功取得德国政府的合作，扣押了相关服务器。这次行动获得美国军方的支持，但军方并不是主导者。但这有助于解释为什么国防部长埃斯珀会被解雇，并换上了米勒和帕特尔，因为只有这样才能保证军方积极配合这项行动。通过掌握服务器，特朗普政府将能够掌握直接证据，以证明投票系统是何时被指示停止计数的，是谁下达了停止计票的指令，又是谁启动了开始换票的算法。值得注意的是，中情局完全被排除在这次行动之外。
