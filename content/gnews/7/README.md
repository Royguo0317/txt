**当官媒批评香港情况或者人士，并加以狙击的情况更频繁出现, 文革2.0也被视为常态。针对性狙击打压每一件事或者一个曾经为捍卫香港核心价值的人士已经不是偶然。**

**（据立场新闻报讯）由中联办控制的《文汇报》、建制报章《星岛日报》在（19 日）分别刊出报道，再狙击科大教授、社科学者李静君。而《星岛》称接获「图文并茂消息」，指李前年11 月出席台湾社会学年会，演讲中表示曾以「守护孩子」成员身份到现场，为示威者争取时间撤退，又指李承认「做了很多时情，在香港法律下定义为非法（a lot of things I’ve done are defined as illegal under HK law）。」在打击任何有不同意见的人士方面事情，中共极权真的是咬着不放，不把一个人打倒誓不罢休。陈年往事都要挖出来，歪曲言论再加油添醋来混淆大众视听，再扣以罪名，用如此恶劣的手段，真不知道怎样来形容邪恶中共的卑鄙无耻。只因参与「守护孩子」也成为批斗的理由。**

去年因为反对修订逃犯条例，中共违反《中共联合声明》诚诺，干预了香港立法，引起民众的不满，香港市民发起的和平集会活动此起彼落。在港共政府配合中共暴政暴力镇压引起整个社会氛围紧张，甚至不同政治立场人士的人身安全受到严重威胁，因恶法横行无理，无奈之下只能在，曾经历过的悲痛恐惧中沉默。但是也难逃秋后算账政治报复迫害遍地开花，曾经参与活动的学者、政客、个人都一一被滥捕滥判。可以想像得到，官方发起批斗活动由党媒体发动，有如文化大革命式大批斗。

而讲到文革，那段曾经疯靡全中共国黑暗历史，对于那些还存在老一辈人，经历过那被摧残痛苦的岁月里记忆应该还在，因为中共不愿意人们知道他们做过的恶事，这段黑暗历史在中共统治范围是禁止，而年轻一代人可能比较陌生，甚至不知道什么是文革。
![]()![](https://gnews.org/wp-content/uploads/2021/03/989ilgvfdssoe.png)图片来源：大纪元
文革是一场由时任中共共产党中央委员会主席的恶魔头子毛泽东与中央文化革命小组，自上而下动员成千上万红卫兵在中共国进行的全方位的阶级斗争。一般认为，文革正式开始的标志是于1966年5月16日出台的《五一六通知》；此前，三面红旗的挫败造成了三年大饥荒中数千万人的死亡，毛泽东被迫退居二线，文革初期毛以「反苏修、反美帝」口号为籍口，以革命名义攻击「走资派」，尝试重回党核心。 《通知》标志着文化大革命正式爆发，展开持续十年惨无人道的「无产阶级文化大革命」。

虽然时隔境迁，其实文革式的批斗一直仍然存在，这个也是中共极权对不同政见者的打击手段之一，自从「国安法」实施后，「爱共爱党分子」除了用恶法来打压威胁市民的言行举止，也不忘于刮起香港文革式批斗风气，这种变态的阶级斗争陋习，不会因为世纪的变迁而消失，反而又在曾经的国际大都会香港开始，日益增长，当然也少不了港共党媒喉舌的推波助澜。通过批斗的形式是希望可以打压，一切个人或政治、言论自由不同的声音，以前的批斗模式想透过群众暴力令被批斗者改变思想。而如今的科技发达时代已经有所不同，取而代之的是先用党媒体攻击，而没有改变的是，舔共的奴才们随时变身为「红卫兵」为党媒添加助力。

而中共的文革批斗手法层出不穷，先由官方发起，用假新闻唱衰你，起底被斗者亲人，以及多年的言论亦可以成为批斗的理由，被狙击批斗的李静君教授不正是如此吗？被用歪曲以前的言论来作为批斗的理据。而陪斗，除主要的批斗者外，支持者、家属、朋友、同事、社交媒体追踪者亦会成为被批对象，记得曾经被批斗的新任食卫局常秘刘利群女士，连她的丈夫也不能幸免。勾结外国势力更加是欲加之罪，将批斗者扯上关系，以合理批斗滥捕，已经被判刑的民主派人士、学者、政见不同人士不正是如此被扣以罪名吗？

经济后果，被批斗者要要承受失去工作，甚至抄家劳改，如许智峰被政治迫害逃离，经济封锁还要被封锁其银行帐户，连家人也受到牵连，武斗也有出现的时候，2019年元朗袭击事件不就是吗？外国人配合中共与论，这个经常发生，蓝金黄下产生不少为中共背书的人士。看到中共病毒期间，除了那些虚伪的科学家，而联合国就是其中之一。如此多的手法真是令人咋舌。

然而李静君教授事件，共奴前特首中共国政协副主席梁振英转载《星岛》报道，要求李静君「尽快自首，否则请警方主动调查」。真是中共的好奴才，任何时候都会为主子鞍前马后，虽然李静君教授其后发声明反击，指相关报道错误扭曲她以英语发言的意思，并基于错误讯息作出评论。但是对于邪恶的流氓中共政权，为了巩固自己的独裁极权利益不受影响，一直审查民众思想、言行举止，无论如何去发表声明、解释都起不到什么作用，因为在魔鬼的国度里是没道理律法可言。
