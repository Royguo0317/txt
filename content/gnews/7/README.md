因国际新闻紧密聚焦以色列及巴勒斯坦两国情势，这些新闻也莫名地触动笔者回忆起10多年前曾经看过的一部美国喜剧电影“特务沙龙” (You Don’t Mess with the Zohan) 中的故事情节一一浮现笔者的脑海中。

这是一部2008年上映的喜剧电影，当时的评价，褒贬不一，在此笔者将不多作赘述。电影的主轴是穿插在三位主角之间所发展出来的故事，因不同国籍背景，从以色列、巴勒斯坦及美国在这三个国家，巧妙地交织出对自己祖国的情怀，省思自己血统所背负的原罪，以及各自在美国这块自由国土实现自己梦想的故事。

亚当山德勒(Adam Richard Sandler) 饰演电影中的主角柔汉（Zohan），是一位以色列国内最英勇的反恐英雄特务，在沙漠中出身入死地保卫自己的祖国，但逐渐对以色列政府派予的任务感到厌倦，于是假装战死沙场并化名潜逃至美国纽约追求自己的梦想，终于成为一位发型设计师。

还有另外两位重要的角色，一位是故事中与Zohan坠入爱河的女主角Dalia，另一位则是主角Zohan的死对头，名叫Phantom的恐怖组织分子，他被误认为成功击杀以色列特务Zohan的任务后，阴错阳差地成为巴勒斯坦的民族英雄，因此声名大噪并在巴勒斯坦创业卖披萨，摇身一变成为一位事业有成的企业家，角色设定此两人为兄妹关系并来自于巴勒斯坦。

这一部喜剧电影，描述三位主人翁，不约而同地在美国相遇，并用幽默的拍摄手法，让观众可以在短短的2小时内，感受到以色列与巴勒斯坦两国之间长期的紧张情势，及普通百姓其实内心非常渴望和平，但却因背负历史的原罪与宗教冲突的因素，演变成以巴两国长期以来对于国际、政治、军事上的对立。然而两国人民之间无形中筑起的那道鸿沟，反而在远离自己家乡，来到自由的国度美国后，才有机会释放及忘却。

笔者10多年前在观看这部电影后，只是单纯的感觉，亚当山德勒这位好莱坞(Hollywood)知名的喜剧演员应该只是想透过这部电影，用比较诙谐幽默的方式来传达，凸显有关犹太人与阿拉伯人两个民族之间长期以来所存在的种种矛盾与冲突问题。

多年后，笔者经历爆料革命的洗礼之下，对于此剧已有更深一层的认知! 原来这位拥有犹太人血统的好莱坞的大明星，除了主演本片之外，同时也兼具电影监制及编剧的角色，透过这部电影，恐怕更想要传达给观众的讯息，一定不是只有剧中所呈现的许多娱乐效果，反而更想要传达的是揶揄种族主义或暗示民族仇恨思想所铺排的剧情效果吧?

因为透过电影的呈现，美其名说是艺术的力量，实际上也就是郭先生所述的，美国另一个沼泽地的力量: “好莱坞”，它对美国社会也正在无时无刻地释放与投射沼泽地大老们所要带给民众的讯息及风向，进而影响操弄民意的工具罢了!
