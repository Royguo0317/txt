大约一周前，美国邮政总局的一名吹哨人Richard Hopkins（他是一名退伍海军）向Project Veritas举报该局存在的选票欺诈。他说：邮政局长罗布·魏森巴赫（Rob Weisenbach）要求职员将11月3日以后寄出的选票的邮政日期篡改到11月3日，以便可以符合宾夕法尼亚州的选票日期的要求。

Project Veritas向公众发布了这个举报信息之后，引起了广泛关注。吹哨人Richard Hopkins也被人盯上了。

今天，Project Veritas发布了一则录音视频：特工拉塞尔·斯特拉瑟试图胁迫和恐吓霍普金斯撤回指控，拉塞尔·斯特拉瑟是登记在册的联邦特工。

录音中，特工拉塞尔·斯特拉瑟说：“我正试图扭曲你一点。”

“我们有参议员参与。我们有司法部参与。我们有……”斯特拉瑟说。

Richard Hopkins插话说特朗普的律师也一直在与他接触。

“我不是-我实际上是。我想稍微扭曲一下你，因为无论信不信由你，你的思想就会开始跳动，”斯特拉瑟说。 “我们喜欢控制自己的思想。当我们这样做时，我们可以说服你的记忆。但是，当您承受一点压力时（这是我有意为您做的事情），您的想法可能会更加清晰。我们也将做另一种练习，以使您的头脑更加清晰。所以，这都是故意的。”斯特拉瑟说。

Veritas项目负责人曾经提到过，他们也曾经遭受过记录在案的联邦特工试的恐吓和威胁，试图强迫他们退缩，但他们会坚持到底并且很快Veritas项目会把这些录音公之于众。

Veritas项目还发布了一段视频。视频中Richard Hopkins澄清《华盛顿邮报》关于他否认指控选举舞弊的说法是个谎言。

Veritas项目创始人James O’Keefe发推文说，Hopkins他试图和《华盛顿邮报》的记者联系，告诉他们这篇文章的内容是虚假的，但《华盛顿邮报》甚至拒绝与他交谈，他完全被忽视了。O’Keefe说：“所以，你知道退伍海军Richard Hopkins今天下午在接受采访时哭了，这些联邦特工这样对待他，这比他在阿富汗所经历的还要难。”
