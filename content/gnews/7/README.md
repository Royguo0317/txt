**簡評：**

美國在受理了幾千個中共間諜危害國家安全的案例之後，終於決定禁止中共黨員移民了。這對於習慣了從美國偷盜各種技術的中共來說，肯定會帶來諸多不便。但’環球時報’主編胡錫進卻發推說，這樣可以為中共國留住人才，大有吃不到葡萄就說葡萄酸的味道。

胡把中共的地痞嘴臉暴露無疑。一個國家的人才，要靠他國禁止進入才能留住，這是一個怎樣的國家？這是一個但凡有辦法你就想離開的國家。不是那裡的山不好，也不是那裡的水不好，更不是那裡的人不好！是中共的極權統治讓人們傷透了心。

當然，那些中共政權的既得利益者們和那些被洗了腦的人，即使到了美國，也會為中共工作。既然這樣，那美國取消他們的簽證或禁止他們移民或入境，用胡錫進的話說，不錯嘛，正好把他們留在中共國吧。

**HKFP:美國禁止中共黨員移民美國**

根據美國公民及移民局（USCIS）上週五發布的最新政策指南，華盛頓已決定禁止共產黨員移民美國。

美國公民及移民局（USCIS）宣布，任何曾經是共產黨員，以及有共黨或獨裁政黨背景的人都不能移民美國。該機構說，這一政策是國會通過的一系列旨在解決對美國安全和保障造成威脅的法律的一部分。

[!\[\]()!\[\](https://spark.adobe.com/page/5hlKJNA7wio2Z/images/d4a7fc0f-41b3-41f5-9f48-f5521416d29a.jpg?asset_id=f648ac9b-012a-4765-8262-a7155a8a1a9c&amp;img_etag=%22f63405424faecb3bc571374d2b74a8a4%22&amp;size=1024)](https://spark.adobe.com/page/5hlKJNA7wio2Z/images/d4a7fc0f-41b3-41f5-9f48-f5521416d29a.jpg?asset_id=f648ac9b-012a-4765-8262-a7155a8a1a9c&amp;img_etag=%22f63405424faecb3bc571374d2b74a8a4%22&amp;size=1024) Photo: Jnn1776, via Flickr.

美國公民及移民服務局（USCIS）在一份聲明中寫道，“共產黨員或其它任何極權政黨的隸屬成員，與美利堅合眾國的移民誓言是不一致，不相容的，其中包括’支持和捍衛美國的憲法和法律’的誓言’，” 該聲明並未明確點名中國共產黨（CCP）。

此舉發生在華盛頓和北京之間的緊張局勢加劇之際，因為美國總統唐納德·川普（Donald Trump）出於安全考慮，試圖阻止包括抖音TikTok和微信在內的中共國應用軟件。

中共國還就美國在香港頒布有爭議的國安法後對香港官員實施“野蠻無禮”的製裁進行了譴責。

根據德國數據庫公司Statista的數據，中共在2019年擁有約9191萬黨員。其中，近200萬是在校大學生。

美國公民及移民服務局的這份指南中，包括了對非自願入黨黨員的豁免，例如，16歲以下入黨黨員，或因為獲得就業機會，限量的食物或其它生活必需品而加入該黨的人。

豁免情況還包括在移民申請日期前至少兩年退黨或退出從屬關係的人。極權獨裁政黨黨員或與之有聯繫的非美國公民，必須在移民申請提交日前至少五年與該組織斷絕關係，才能享受豁免。

在這兩種豁免情況下，領事官員必須認為申請人對美國安全不造成“威脅”。

如果尋求移民的人是美國公民的近親或出於人道主義理由而遷入，司法部長可酌情免除該規則。

該政策指出：“為了家庭團聚，或為了公共利益，如果移民申請人對美國安全沒有威脅，該規則也可以免除。”

作為回應，中共國資訊《環球時報》（Global Times）主編胡錫進週日表示，許多中共國優秀人才都是中共黨員。他說，美國新的移民指南將幫助中共國將人才保留在其境內。

“美國的決定消除了中共國人才移民的幻想，這有助於將他們留在中共國內。 這不錯嘛。 而且，非黨員（CPC）現在對移民美國的興趣也大大降低了，”胡在推特（Twitter）上寫道。
