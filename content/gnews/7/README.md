# 观察人士发现，情报分析人士淡化了中共国对选举的影响以避免支持川普的政策

根据一名情报部门观察人士，美国情报机构对2020年美国大选外国影响的评估存在政治化问题，其中包括分析人士明显地隐瞒中共国干预选举的信息，因为他们不同意川普政府的政策。

分析监察人士、长期担任情报部门官员的巴里·祖拉夫(Barry Zulauf)周四向参议院情报委员会提交了一份14页的报告，《华盛顿观察者》获得了该报告，报告称他的调查“是作为情报部门投诉选举威胁问题的行动”。另外，他悲叹道：“两极分化的氛围已经威胁到破坏我们共和国的基础，甚至渗透到情报界。”

[!\[\]()!\[\](https://spark.adobe.com/page/BjpcFjq4pOcBp/images/9ad28aa3-3c04-4971-9fc4-d672a16cad38.jpg?asset_id=524e7dca-917d-4b0b-b018-49cc64fc2712&amp;img_etag=%22a20c44ef2fb8707c6fd67e51548f6a90%22&amp;size=1024)](https://spark.adobe.com/page/BjpcFjq4pOcBp/images/9ad28aa3-3c04-4971-9fc4-d672a16cad38.jpg?asset_id=524e7dca-917d-4b0b-b018-49cc64fc2712&amp;img_etag=%22a20c44ef2fb8707c6fd67e51548f6a90%22&amp;size=1024)美情报系统巡查员巴里·祖拉夫(Barry Zulauf)

周四，情报界关于外国势力对2020年大选影响的机密评估也提交给了国会。这份评估不会关注邮寄欺诈的指控，也不会关注毫无根据的投票机倒转数百万张选票的指控。由于高级情报官员在中共国扮演的角色上发生冲突，本该在12月发布的评估被推迟了，而作为国家情报总监，约翰·拉特克利夫(John Ratcliffe)寻求在最终分析中纳入更多观点。

鉴于俄罗斯和中共国的分析人士在分析他们的目标时存在差异，中共国的分析人士似乎不太愿意将中共国的行动视为不当的影响或干涉。这些分析人士似乎不愿让他们对中共国的分析被提出，因为他们往往不同意政府的政策。实际上，我不希望我们的情报被用来支持这些政策。” 祖拉夫总结说，这种行为违反了独立于政治考虑的分析标准。

该监察员分享了一系列被拉特克里夫接受的建议，包括“通过从情报总监办公室到全体员工，以及从各部门主管到各情报部门直接领导的沟通，加强保护分析完整性和重新承诺分析客观性的重要性，避免在政策和实施中的政治化”。
[!\[\]()!\[\](https://spark.adobe.com/page/BjpcFjq4pOcBp/images/67f15864-d050-4f84-8659-e1031e93e35f.jpg?asset_id=6103480f-f303-4c7f-bbf2-292a1bb6b233&amp;img_etag=%22cbf615c43282f686e11a06cf8315ec1e%22&amp;size=1024)](https://spark.adobe.com/page/BjpcFjq4pOcBp/images/67f15864-d050-4f84-8659-e1031e93e35f.jpg?asset_id=6103480f-f303-4c7f-bbf2-292a1bb6b233&amp;img_etag=%22cbf615c43282f686e11a06cf8315ec1e%22&amp;size=1024)约翰·拉特克利夫(John Ratcliffe)（左一）宣誓就任国家情报局局长
周四，来自德克萨斯州的前共和党众议员拉特克里夫签署了一份三页长的非机密信，《华盛顿观察家报》也获得了这封信。他在信中称：“作为一个掌握了美国政府所有有关中共国的最敏感情报的人，从我独特的角度来看，我不认为情报界分析人士的多数观点完全准确地反映了中共国政府努力影响2020年美国联邦选举的范围。”拉特克里夫补充说，监察员的报告“包括有关中共国选举影响力报道政治化的披露，以及对基于情报提出不同观点的分析人士施加不当压力的披露。”

一名高级情报官员告诉《华盛顿观察者》：“在情报部门内，我们必须解决这份报告中列出的问题，我们内部的评判基本上认为拉特克里夫是对的，我们的一些职业人士，甚至CIA的管理人员都在把中共国的情报政治化。”

四年前，一份关于俄罗斯干预2016年大选的评估出炉，一些人仍在争论这一问题，争论的背景是对中共国对美国议员的影响的日益担忧。据估计，太阳风公司(SolarWinds)进行的大规模黑客攻击很可能是由俄罗斯实施的。周三，川普总统的支持者冲进国会，试图阻止国会统计当选总统乔·拜登获胜的选举人票，引发了混乱。

国会一直工作到深夜和凌晨，在周四凌晨4点前确认了拜登的胜利。在这个决定之后，川普说他承诺在就职日有序交接权力。

一个熟悉编写“2020评估”的人士告诉《华盛顿观察者》，除了内部的辩论正在进行之外，推迟向国会提交报告的另一个原因是，他们希望通过周三的审议，以确保在有关国会统计的选举人票的辩论中，这份报告不会被政治原因利用。该消息人士还提到了对政客的担忧，比如众议院情报委员会主席亚当·希夫(Adam Schiff)等直言不讳地批评川普的人，可能会利用这份报告，但也试图避免让与川普结盟的律师西德尼·鲍威尔(Sidney Powell)和林·伍德(Lin Wood)等阴谋论者对这份报告做出误导性的声明。在川普拒绝让步的情况下，这将让他抓住这个机会。。

该监察员的报告补充道：“俄罗斯分析人士评估表示，存在明显的可靠证据表明，俄罗斯影响了选举活动。他们称情报部门管理人员减缓了进度，或者不想让他们的分析报告给公众看到，声称报告不受欢迎，这让他们感到沮丧。分析人士认为，这是对情报的压制，近乎于将来自上级的情报政治化。”

祖拉夫说：“由于对敌对国家领导意图的行为对国内影响力的收集和了解不同，中共国和俄罗斯分析团体对‘影响力’和‘干涉’以及相关信任水平的定义使用是不同的。”他指出，俄罗斯分析人士可以依靠正式的定义性文件，但是“中共国没有相应的文件”，因此，“这些术语在情报机构的使用并不统一”。

在信中，拉特克里夫说“对我来说很明显，聚焦于分析来自不同国家对大选造成威胁的不同分析团体在同样的恶意行为上使用不同的名词”，并且“俄罗斯和中共国的类似行动被评估，并以不同的方式传达给政策制定者，这可能导致错误的印象，即俄罗斯试图影响选举，但中共国却没有”。

该监察员说，试图政治化情报“最过分的例子”是“国家反情报和安全中心主任比尔·伊瓦尼纳在2019年3月发表的介绍性声明中提供的谈话要点，但他并没有撰写这种声明”。祖拉夫还提到伊瓦尼纳七月和八月的陈述，称伊瓦尼纳“说他认为他自己代表了整个情报界的统一观点”，但是该监察员总结道：“其实，他们并没有完全代表整个情报界的统一观点。”

在时任国家情报总监理查德·格雷内尔 (Richard Grenell)之下工作的现任情报总监拉特克里夫在3月向国会提供了一份解密的证据列表简报，称“情报部门没有下结论称克里姆林宫直接帮助任何候选人连任…这不仅仅是俄罗斯的问题”。7月，伊瓦尼纳发表声明承认“我们主要关心的是中共国、俄罗斯和伊朗”。

八月的声明警告称，俄罗斯正在“利用一系列手段，主要是污蔑”拜登，指出乌克兰立法人安德里·德卡赫 (Andrii Derkach)正在努力损害前副总统拜登，伊瓦尼纳同时称伊朗试图“损害”川普总统，并且在2020年前尝试分裂美国。另外，他称“我们评估显示中共国希望川普不会赢得选举”，而且中共国“已经在2020年11月之前就已经扩展自己的影响力”，并且“认识到这些行动”可能影响选举。

祖拉夫说，“分析人士称国家情报部门的领导层一直在评估过程中淡化结论，鼓吹中共国的威胁，让俄罗斯的威胁听起来没有争议”，同时称，“国家情报官员指出，国家情报办公室高级官员在影响这些改动”。但是监察员称拉特克里夫“只是和当前分析人士认定的中共国情报意见不一致”，并且引用了拉特克里夫的话，“我们忽视了中共国对美国的影响，中共国正在影响选举”。监察员称，“最终国家情报总监坚持把中共国的材料放到报告中，他也知道他的意见与分析人士相左”，并且“作为结果，最终发表的情报报告被许多分析人士很愤怒地认为是对他们分析的曲解”。祖拉夫和拉特克里夫承认“许多分析人士认为我们不靠谱”，但是“他们没有意识到我们是基于情报得出的结论”。

检察员揭露了两个情报官员在一份10月“国家情报部门另一种分析解读备忘录”中写道的，该备忘录“表达了对中共潜在影响选举活动的不同观点”，但是强调“这些不同观点遇到了组织上巨大的反对压力”。

“国家情报总监办公室必须保证另一种观点得到表达，即使他们和大部分人的观点不同。一个健康的挑战文化在情报部门可以促使不同的分析观点形成，并且保证这些是被情报部门共享的资产，”祖拉夫总结道，“在我和他的讨论中，国家情报总监拉特克里夫对于另一种分析观点备忘录中表达的担忧表示同意”。监察员强调了分析的标准，评判需要基于所有情报来源。

“分析情报检察员称，拉特克里夫不是政治化——他只是诚实，中共的情报确实是因政治原因被打压下来了。”高级情报官员向《华盛顿观察者》承认。

“很明显，对国家情报总监办公室发表的评估的错误描述，升级成为工作人员对整个情报界在俄罗斯和中共国对选举的影响和干预问题上的政治化和客观分析丧失的普遍看法，”检察员评价道，“没有国家情报办公室的员工说过对选举威胁情报的审查或编辑有明显的政治性质。相反，从国家情报总监办公室领导层的观点，官员们是在寻求一种方式，以川普政府将接收的方式传递情报。”

拉特克里夫承认，“这份报告中关于中共国影响选举的行动所表达的多数观点是不正确的”，并且“关于中共国干预选举的另一种观点没有得到适当的容忍”。他说目前没有被公布的情报报告“给人们错误的印象”：国家网络情报官员是“唯一的分析人士且支持少数人对中共国的观点”，并且“把国家网络情报官员放在一个比喻上的孤岛，通过把他的名字和少数人观点贴到一起，这是对他勇气和效率的证明，这显示了整个机构对他和其他同意他的人士身上带来的压力”。

“在1962年，国家情报评估(National Intelligence Estimate)称苏联不大可能在古巴部署导弹。当时CIA总监约翰·麦科恩 (John A. McCone)极力反对该评估结果，并且之后部署了U-2侦察航班，发现了其实古巴已经有部署导弹，”拉特克里夫总结道，“本着同样的精神，我加入我的声音来支持少数人的观点，即中共国试图影响2020年美国联邦选举，这是基于所有可用的情报来源，定义一致适用，独立于政治考虑或不适当的压力。”

川普情报总监在12月的一次《华盛顿观察者》的采访中提供了情报部门内部辩论的一点信息。

“你有从冷战时期就在这里工作的分析人士，他们已经习惯了这里是俄罗斯，或者在过去20年里，他们习惯了这里是反恐主义的地方——再次强调，我不是在贬低这些——但我们面临的最大威胁，我们需要更多地关注中共国。”拉特克里夫说。
