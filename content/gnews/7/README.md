外星人，是人類對地球以外的智慧生命的統稱。從古至今，人類對其他行星是否存在外星人，以及外星人及其文明是什麼樣的，一直存在猜測、想像、研究與探索。那麼，真的有外星人存在嗎？郭文貴就這個問題詢問了班農先生，得到了肯定答案，地球上真的存在外星人。小編整理了郭文貴先生在視頻中4次談及外星人的片段，以供參考。

1.2019年10月5日

外星人和美國，和這個地球，所有的存在，幾萬年來，她是有邏輯的。上天有上天的安排，就像我今天相信我們所有的行為都有上天的安排一樣。**外星人它不可能來了就接管地球，或者說什麼都可以分享，甚至事無鉅細的，管我們的一言一行，我覺得這不可能。**就像我說過的一樣，地球在宇宙中，就像地球上的一粒沙子一樣。你想他再厲害的人，今天川普再厲害，你能管那一粒沙子去嗎？管不了。**但這個地球要被毀掉，那沙子肯定沒了，它管的是大事。中共不管它多麼邪惡，在這一粒沙子和地球這種等級的差距上，它還是很渺小的。**

這就共產黨人，他們沒有意識到，你多牛，你也是太微小的。跟人類，跟時間比，跟神比，跟主比，跟默罕默德、釋迦摩尼、基督比，跟天主比，你太渺小了，所有他無知嘛。那麼我們不能無知的看待這個問題，就是外星人它並不是神，就是說它並不一定是宇宙的主宰者。我們在別的星球面前，我們也是外星人，我們能去主宰人家嗎？我們還在探索，主宰不了。這完全是不同的問題。外星人不是神，外星人也不一定能統治地球。還有，我們地球不要把自己看太重要，就是一粒沙子跟地球比較。

還有一個，共產黨這些狗屁嘮叨的事，在這些面前，什麼都不是。所以說，我們的要求，我們的幻想，永遠希望著另外一個力量在幫我們。這也是我們人類內心最大的痴，貪嗔癡慢疑的痴。就是無知嘛，你覺得外星人會來救我們，它也可能是來殺你，不是來救你。你為啥不想想，外星人沒有把我們毀掉可能是因為美國呢？也可能啊。**外星人真來了，也不一定有這本事管我們的事啊，對不對？它也許管不了。**

2. 2019年9月2日

大家可以上網查一查亞利桑那州。**來亞利桑那州的路上就途徑美國的沙漠叫51區，就是研究外星人的軍事基地。同時在這個地方有很多美國秘密地點，軍事基地，美國很多的飛機飛行基地在這兒。據說美國離天最近的地方就是亞利桑那州。同時，藏著美國軍事秘密的地方也在亞利桑那州。同時美國軍機的訓練最多的也在亞利桑那州。**我們來的時候能看到很多戰機，各種F15，F16，還有各種美國先進的戰機。

3. 2019年9月29日

我本身就是個外星人的迷，我一遇到這個就感興趣的不得了，所以那天我去亞利桑那州，去圖桑，在飛機上，班農先生就坐在我旁邊，我就問他。他跟我說，這是51區，如何如何，他本身也是外星人迷，然後他當上國家安全顧問，第一個就是了解這個是真是假，更確切的證據。具體的我不能告訴你，但是我可以告訴你這是真的。

我說真的有外星人，他說是的，**美國的很多技術確實是外星來的技術。**我當時我真的是，我從飛機上我都懵了。因為這東西太大了，這事兒。我也認識很多美國人，包括太空總署的很多人，很多人都是不敢說，因為這是絕密。然後給我講了一些細節，我在這不能講，這是肯定有的。

接下來我又見了一個另外一個在51區工作的人，這個人我很感興趣。我說你現在是研究太空，他在我對面，那個牆上展示出來，就是整個地球，最大的危害是隕石。砸過來呀，錄的視頻啊，當年在亞利桑那州，圖桑。看著隕石落下來，落到圖桑那個地方，有三個印第安人自治的州。落到那個州里去了，印第安人認為外星來的都不吉利，他不動。這些人衝過去就把那些隕石給找回來，做研究。那個視頻是存在的，他答應給我了，等哪天我放到郭媒體上。

另外一個他說，他研究了這麼多年，前邊老師的老師的老師，一個結果。他說我們人類，你去看一看，當你把地球看小的時候，外星的垃圾，外星的隕石，星河系，哇塞，那就想我們對面大海一樣，無數個點點組合而成。就是幾萬年來，地球，他說你只要一次，他說恐龍被滅就是一個13米的隕石砸到地球上，恐龍沒了。地球就變成了所謂今天的這個樣子，水呀海洋啊，大變化。

大家都知道各種時代，完全變了。那麼為什麼那麼多年沒砸下來呢，人類還在呀。他說早晚一天，一定會被砸下來的，現在美國在致力研究，在砸下來之前把它（隕石）給毀了。那麼反過來說了，人類在科學研究中發現，這個地球，最簡單的例子，比例跟人類探索的太空比，就所有的沙子，大家知道有多大，你只要拿起一粒來，地球就是那一粒沙子。我們在沙子裡邊就這麼大的玩意兒，還在這想什麼太空呢。

所以說，想來想去，為什麼所有的科學家還有歷史哲學家，都成了最後的宗教人士。沒法研究了，最後的問題很簡單，**就是人類是外邊有主人的。他是受外力控制無數的。**從這個角度來講，你說什麼51區啊，還有什麼迷信呀，宗教都已經完全沒有存在的意義了。因為你來解釋這麼一粒沙子裡邊地球上所有的沙子，你怎麼解釋的了。你解釋不了。

但反過來你要認識到，**地球有太多物種了，是我們不知道的。有太多力量是你不知道的，人類是極為脆弱和渺小和無知的，然後我就問他，到底人類有沒有靈魂，到底世界上有沒有鬼魂，他說百分之百有，為什麼。**他們發現了一個秘密，就是人類上所有的物質變化，最後你知道變到哪去了嘛，它都在某個空域還存在。但是發現在研究人的身體裡邊，就咱說叫磁場。人死之後磁場的去向和反應完全不同。很快就會公佈，現在陸續的在公佈出來了。

就人死了，靈魂呀，身體啊，就人死之前，身體有靈魂眼睛肉體，和你實際上的重量和你實際色彩的比例，和死後最大的比例，沒有了那口氣，沒有了血色，還有一個，他說你的質量就是物理上的質量巨大的變化。質量去哪了，他們發現是在不遠的地方，甚至不出星河系，這些東西是並列存在的。也就近代科學家說的，人的靈魂並不死亡。

**他還發現了一個，就是很多人類的信息有重複性，這種重複性說明是有靈魂還會回來的。但是靈魂就像發射的衛星回到地球的時候，就被那個熱度，大氣層的那個熱度，燒焦的感覺，它要進行失憶甚至被毀掉。他說這是一次新生。然後他就說了，有些靈魂是走不了的，就在我們身邊。**

然後他給我講了幾個特別的例子，他親身經歷的。哎呀媽呀，當時講的我呀，我真的是渾身難受，又害怕又興奮。哎呀，要停，別別別，你繼續講。講的我真的是全身出汗，太興奮了。第二宗教只是解釋我們人類存在方式的一種方式，太渺小了。不要迷信，**還有人類絕對是有靈魂的，是不死亡的。**我先回答你到這。更具體的我不能回答你，因為說了好多秘密，我確實不能說。
