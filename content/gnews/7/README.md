## CCP匯率操縱史
`2020-07-07 03:10 GM77` [轉載自GNews](https://gnews.org/zh-hant/256325/)

作者：簡體中文
整理：映蘭軒、文寶Porshce



2019年8月6日美國財政部宣布中共國為匯率操縱國以來，世界才開始嚴肅對待人民幣匯率操縱問題。但你知道嗎？CCP從竊國後就一直操縱匯率，已經持續了70年。匯率操縱是CCP盜國賊剝奪中國人民和洗劫全世界的超級武器。這是為何人民幣在世界各國外匯占比達到2%後仍然無法成為主流貨幣的原因。

跟著短視頻，我們一起看看CCP的匯率操縱史：

（1）第一階段1950-2005

•     1950年至文革結束，中共實行經濟管制，匯率控制在2左右（美元兌人民幣，下同）；

•     文革結束CCP放開經濟管制後，從1981年開始爆發通脹、人民幣持續貶值；

•     1993年底，CCP實施匯率並軌，人民幣匯率一天之內、從5.8貶值到8.7，暴跌47%；

•     1997年香港回歸後，CCP開始通過香港這個通道、悄無聲息地套取大量美元，從而控制了人民幣匯率貶值趨勢；

•     2001年入WTO後，中共國通過出口獲得了大量外匯，人民幣開始升值，但CCP為了保持出口競爭力，一直壓制人民幣匯率；

•     2003年，美國政府第一次抗議CCP操縱匯率，CCP被迫在2005年7月21日重啟匯率改革，開始了人民幣的三年升值期；

（2）第二階段2006-2020

•     2006年1月中共引入詢價和做市商制度；4月人民幣匯率升破8關口；

•     2008 年4月取消強制結匯制度， 5月9日人民幣匯率升破7；

•     2008年次貸危機爆發後，中共再次出手壓制人民幣升值保出口，人民幣匯率在6.83附近橫盤達兩年之久；

•     2010年，美國會議員聯合呼籲政府將中國列入匯率操縱國，迫使CCP放松控制，人民幣進入4年升值期；

•     2014年1月1日，在人民幣即將升破6關口一步之遙時，CCP再次出手幹預，兩天內引導人民幣匯率下跌一千個基點；

•     2015年8月11日，CCP要求做市商改變外匯報價制度，人民幣連續兩日暴跌；

•     2016年12月31日，外管局宣布改進個人外匯信息管理，控制民間購匯；

•     2017年5月26日，央行引入“逆周期因子”主導人民幣升值；

•     2018年3月，中美爆發貿易戰，人民幣再度被迫大跌；

•     2019年夏，中美上海貿易談判破裂，人民幣跌破7關口；

•     2020年1月15日，中美簽署第一階段貿易協議，人民幣匯率在7附近企穩；

•     2020年6月30日，中共強行通過香港國安法，7月初美國通過對中共金融制裁法案，中美經濟即將全面“脫鉤”，人民幣隨時面臨崩盤。至於是復制前蘇聯的盧布，還是復制伊朗和委內瑞拉的貨幣走勢，讓我們拭目以待。
