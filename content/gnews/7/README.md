**上周发布的西藏白皮书是中国共产党试图通过宣传粉饰现实的一种尝试。**

自中国共产党诞生以来，宣传一直是最有力的工具之一。从毛泽东到习近平，宣传一直是他们控制舆论的一个组成部分，通过宣传，他们的地位被提升为中国老百姓心中的邪教教主。

简而言之，宣传是中国政府体系运作的核心。随着习近平的到来以及他对尖端技术的使用，宣传现在已经成为牛津大学帕特里夏·桑顿( Patricia Thornton)恰当地称之为“舆论的技术官僚工程”。同时，中共也通过宣传为其所占西藏、东突厥斯坦（新疆）和南蒙古等领土的强行同化和镇压行为，以及实施香港国家安全法作辩护或否认。

**西藏是中共宣传的实验场**

1951年中共控制的中国入侵西藏时，当时被称为红军的人将入侵宣传为解放西藏。从此，中国共产党掌握了表面温和的宣传艺术。中共通过甜言蜜语安抚了西藏当局所谓的“西藏现代化使命”，直到西藏人民能够自主自立。已故藏族学者达瓦诺布写道：“1953 年，中共向萨迦喇嘛和地方当局解释他们闯入的原因。这是中国未来几年在西藏的官方政策的实质内容。”

中国和西藏之间的解决看起来像是两国政府之间的相互协议。委婉地称它为十七点协议，实际上它是在没有西藏政府的自愿同意或西藏人民的支持下签署的。后来由于国际上对西藏侵犯人权的批评日益增多，从1990年代开始，中国共产党就发动了宣传攻势，从那时起就开始大量出版有关西藏的白皮书。

**中共在Covid-19期间的宣传**

去年是中国外交部、《环球时报》等中共党属媒体和中共支持的网络战士反击违背党利益的重大言论的繁忙时期。在 Covid-19 大流行期间，Twitter 成为其防御性和进攻性宣传的平台之一。外交部官员使用的语言和他们的网络行为本质上是如此具有攻击性，以至于被称为战狼外交。由于中国政府官员的一些帖子的强度和不敏感，推特决定给他们的账户贴上标签。此后不久，推特决定给联合国安理会五个常任理事国管理的大部分账户贴上标签。

近日，外交部发言人华春莹决定在她的推特账号上分享加拿大约克大学社会学教授吴凯瑞在《华盛顿邮报》上发表的一篇文章。华春莹之所以感同身受，原因很简单，就是这篇文章通过对约2万名对党的表现竖起大拇指的受访者的调查，对中共在控制疫情方面取得的成就进行了赞扬。有人可能会好奇，中国的2万名受访者如何能决定一个14亿人口的国家的判决。请记住，即使是像阿里巴巴的马云和美团的王兴这样的亿万富翁，也不能幸免于对任何批评中共的言论直言不讳。

**未来场景：一场邪教崇拜的马拉松**

在即将到来的中国共产党成立100周年庆典之际，人们可以想象它将是对中共国整体发展和对世界的贡献的盛大展示。中国共产党已经开始在全国影院放映宣传片。

在兜售党的周年纪念日，可能伴随着将习近平描绘成救世主的形象，因为 Covid-19 大流行严重损害了习主席领导下的中共国形象。

人们可能会期待习近平就他重振“中国梦”的计划发表讲话，从而巩固他自己的终身执政的梦想。也许还会发行一部完整的中英文纪录片或电影，颂扬中共国在习近平领导下抗击 Covid-19 大流行的成功故事。未来几年，中共中央宣传部（也称中共中央宣传部）部长黄坤明和中共外事工作委员会主任杨洁篪将都会很忙。他们可能需要额外预算来宣传中国共产党和习近平的国内和国际运动。

这与毛主席在大跃进失败后决定重新振作时面临的情况类似。因此，习近平很有可能将自己重新定位或重塑自己成为中共国人民和世界的救世主。也有可能制作一部与好莱坞标准相当的关于西藏和东突厥斯坦的全面宣传影片。

总之，在未来的日子里，根据国内外的宣传力度，我们可以得出一个结论，即中共和习近平的不安全感正在加剧。
