主流媒体最近都在欢呼疫苗的快速推出，全球范围内的大规模疫苗接种正在展开，各界大咖也竞相在镜头前直播自己的疫苗接种。然而，对于疫苗的副作用和潜在的安全问题却鲜有报道。

据Activist Post报道，在Moderna公司提交给美国食品药品管理局（FDA）的报告中列出了在新冠疫苗试验中的13例死亡病例，其中疫苗对照组6例，安慰剂对照组7例。虽然在报告中，Moderna公司声称这些死亡病例和疫苗的注射没有关系，但是专家表示这一论断还为时尚早。

辉瑞（Pfizer）公司的新冠疫苗已经在英国对公众进行接种，在美国，第一批疫苗的接种工作也已经展开。民众必须认识到，辉瑞公司的疫苗还没有获得食品药品管理局的批准，它只是获得了紧急使用授权（Emergency Use Authorization，EUA），这意味着该疫苗没有经过标准的流程获得监管机构的正式批准。现在，Moderna公司的实验性新冠疫苗也将获得同样的紧急使用授权，与辉瑞公司的疫苗一样，该疫苗将会被分发给数百万人。

随着辉瑞疫苗大规模接种的展开，人们已经看到疫苗的短期不良影响（副作用）。虽然疫苗公司一直在鼓吹疫苗的有效性高达95%，事实上，没有人知道疫苗的长期影响是什么，现在的疫苗接种似乎是一个全球性的疫苗测试实验，而公众已经成为了这个实验的小白鼠。

Moderna的这份报告被一个独立的医疗咨询专家小组审议后，以20-0的投票结果通过了紧急使用授权。然而，第三方的专家在仔细分析这个报告后，发现了一些猫腻。Moderna选择性地挑选了疫苗实验的参与者，以达到预期的实验结果，从而获得FDA的紧急使用授权。同时，主流媒体也配合掩盖疫苗实验中的13个死亡病例。
