## 蹊跷的柬埔寨2.20社区传播事件，与中共疫苗经济有关？
` 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hans/955206/)

作者：Stephen文文
![]()![](https://gnews.org/wp-content/uploads/2021/03/030701.jpg)
从全球范围内来看，柬埔寨是疫情防控最好的国家之一，到3月6日，其总确诊病例仅为900多例。然而其中将近一半（446例）都是和2月20日的社区传播事件相关，且其中多数均为中国人。在2.20社区传播事件之前，柬埔寨的确诊病例非常少，基本上以输入病例为主，本土病例少发。

我们先来看2.20社区传播事件的蹊跷之处： 
1. 疫情的发现竟然是因为几名中国人回国的核酸检查。最早是一名中国女子计划2月18日回国，在做核酸检测后确诊，随后追踪到她的一批接触者（23名中国人）被确诊。 
2. 另有2名中国女子和1名中国男子也于19日在回国检测时被确诊。 
3. “N8夜店”本认为是此次疫情的爆发点，此前有4名中国女子在隔离期间从酒店偷跑到N8夜店，其中的2人随后确诊。 
4. 这4名女子是在酒店隔离期间是买通了保安后，偷跑到N8夜店。 看起来本次柬埔寨2.20社区传播疫情源头和几名中国女子的关联性很高。疫情是否由中共安排？

柬埔寨方面对中共疫苗的态度并不积极，中共借2.20疫情向柬放施压，促使其使用中共疫苗的可能性还是有的，从动机方面中共作案具备一定的可能性。柬方疫苗相关信息： 
1. 2020年12月16日，柬埔寨卫生部表示其订购的100万剂新冠疫苗并非中共疫苗。 2. 2月7日，中共捐赠柬埔寨60万剂疫苗。 
3. 2月10日，柬埔寨政军界高官接种中共赠送疫苗。 
4. 2月18日，柬埔寨表示要向印度订购500万剂新冠疫苗。 
5. 2月20日，柬埔寨社区疫情爆发。 
6. 3月1日，中共再次援助柬埔寨40万剂疫苗。

可见在疫苗购买角度，柬埔寨方面并不信任中共疫苗，去年12月和今年2月的购买都来自于其他国家，而中共是采用无偿援助的方式向柬放供应少量疫苗。中共目前国内外强推疫苗经济，但效果并不太好，国内外使用中共疫苗的积极性并不高，不排除中共使用小规模放毒的行为刺激其疫苗经济。

以上结论是笔者根据事件之间的关联推测得到。
