![]()![](https://gnews.org/wp-content/uploads/2021/01/image0-2-11.jpg)
截至到今日为止，全世界感染中共病毒的人数，根据统计表明大概已经超过一亿多人，死亡人数大概已超过两百多万人。可病毒依然在急速传播，能够预防的有效疫苗和解决办法遥遥无期。中共国的病毒感染情况，显然已经是纸包不住火了，各地相继封城，人道灾难上演。对于西方欧美国家来说，虽然不至于出现像中共国那样饿死人的惨状，但由于各国的管控和封城措施，也是怨声载道。对全球经济有着冲击性的负面影响，对世界人民的身体健康和心理健康造成严重的威胁和影响。

由于中共病毒的存在，给世界各国造成了医疗资源的匮乏，医护人员的紧缺。因此世界人民的就医受到了很大的限制和影响，比如癌症筛查，常规检查，正常程序，等等……更不用说牙科了。

![]()![](https://gnews.org/wp-content/uploads/2021/01/38481890-9187261-image-a-33_1611647430666-1.jpg)图为: 周一晚上，一名男子在荷兰鹿特丹的冲突中被警方逮捕。图片来源: EPA

据英国《每日邮报》报道，荷兰周一遭受第三次骚乱袭击，在鹿特丹，阿姆斯特丹和海牙发生冲突。警方说，到周二早上已有184人被捕，并承诺说将进行更多逮捕。数以百计的暴徒洗劫商店，放火并与警察发生冲突，警察用水枪反击。在政府实行夜间宵禁后，周一荷兰数百人与警察发生冲突，有放火，抢劫商店，多个城市的汽车被推翻的情况发生。

![]()![](https://gnews.org/wp-content/uploads/2021/01/38481882-9187261-image-a-7_1611647237216-5.jpg)图为: 骚乱者砸碎窗户，鹿特丹的一家超市工人站在一家超市的门前。 图片来源: EPA

港口城市鹿特丹遭受的暴力首当其冲，警察说多达200人向警官投掷石块和烟花，然后试图放火烧警察局，警局部署了水炮来制止他们。一群警察被包围后，一名军官还用他的执勤武器开了警告枪。指挥官说至少有50人被捕。

一家针对经济研究的美国网站《AIER》预测，由于药物过量，自杀，酗酒，凶杀和未得到治疗的抑郁症，将会有十万多人死亡。这些死亡并非是由病毒引起的，而是由于强制性的人与人隔离，经济低迷，商业和学校关闭，封闭的医疗服务，以及因失去自由和选择而产生的普遍抑郁。两位经济学家证明，破坏正常社会功能的政策将像病毒一样严重，将造成大规模，完全不必要的痛苦和死亡。这篇文章已经被引用很多。

在上述经济学家所发表的文章中，结合中共病毒发生一年后的今天来看，最糟糕的情况已成事实。这两位经济学家应该被视为先知。美国疾病控制中心已经记录了，令人震惊的非中共病毒引发的死亡人数。

#### 斯科特·阿特拉斯总结：


```
西班牙裔：40％的额外死亡与病毒不相关
黑人：46％与病毒不相关
白人：38％与病毒不相关
25-44岁之间的人：77％的额外死亡与病毒不相关
65岁以上：39％与病毒不相关
```


最令人震惊的数据涉及25-44岁年龄段。这是一个与病毒相关的感染致死率为0.0092％的人群，按说这组人群几乎每个人都没有疾病。然而，他们的死亡速度远高于预期，而且大部分死于与病毒不相关的问题，他们不应有过多的死亡率。相反，他们发现这些人死亡情况让人吃惊。

这些过多的死亡，对美国少数族裔社区产生了深远和不成比例的影响。与过去几年相比，2020年循环系统疾病，阿尔茨海默氏病和痴呆症以及呼吸系统疾病的死亡人数有所增加。目前尚不清楚，这些疾病在多大程度上代表是中共病毒死亡造成，还是由于大流行病间接相关的死亡（例如，由于中断医疗服务或利用）。

这是一种听起来很残酷的说法，即由于中共病毒，而实施的封锁措施，无法完全保护最脆弱的人群，反而增加了老人群体的死亡人数。此外，新的研究表明，养老院中过多的死亡人数大量增加，这很可能是由于防止探亲带来的绝望和孤独感。
