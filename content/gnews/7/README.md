# 福林将军敦促川普支持者在国会宣布川普败选时到州首府抗议

[!\[\]()!\[\](https://spark.adobe.com/page/abheFW8cGiOB0/images/de5c930a-1c40-4de5-82d5-9630b71ddab3.jpg?asset_id=7d460bc6-c7a7-46ed-bd20-43ec2ab5fb4f&amp;img_etag=%2295912d1a8fc8a4f10ba2866457b6be0f%22&amp;size=2560)](https://spark.adobe.com/page/abheFW8cGiOB0/images/de5c930a-1c40-4de5-82d5-9630b71ddab3.jpg?asset_id=7d460bc6-c7a7-46ed-bd20-43ec2ab5fb4f&amp;img_etag=%2295912d1a8fc8a4f10ba2866457b6be0f%22&amp;size=1024)图片摄于2017年2月1日，国家安全顾问麦克·福林在每日白宫新闻发布会上讲话，华盛顿。

福林将军周六呼吁:一旦国会议员(1月6日)开会并且官方宣布川普总统败选给预计当选总统乔·拜登，全国各地的川普总统支持者到各州首府抗议游行。

福林将军是川普总统的前国家安全顾问，他在社交媒体上表示:川普支持者们应该在国会清点选举人票时到各州首府对最近的总统竞选进行抗议。

“明年1月6日，全国各地不能到首都的爱国者应该饱含敬意并且和平地到各州首府抗议这次虚假的选举”，福林将军在推特上说道。

国会预定在1月6日举行联席会议，届时，选举人代表团投票确认拜登赢得总统选举的结果将被统计并正式公布。

川普总统和福林将军都是共和党人中显赫人士，他们声称总统选举被操控并且拜登没有胜选，但是目前没有可信的证据证实他们的指控。

[!\[\]()!\[\](https://spark.adobe.com/page/abheFW8cGiOB0/images/c2d70490-8017-4b41-b495-424bf23c5075.jpg?asset_id=3c7bf08c-7781-4a79-92f1-6a6b3196b432&amp;img_etag=%22eb75b18a7401ae2e6782ba52a5b78433%22&amp;size=1024)](https://spark.adobe.com/page/abheFW8cGiOB0/images/c2d70490-8017-4b41-b495-424bf23c5075.jpg?asset_id=3c7bf08c-7781-4a79-92f1-6a6b3196b432&amp;img_etag=%22eb75b18a7401ae2e6782ba52a5b78433%22&amp;size=1024)

福林将军是前陆军中将，他在推特上关于抗议游行的发文是回应川普总统在周六早间暗示支持者在华盛顿举行抗议游行的推文。

“‘正义的’司法部和联邦调查局(FBI)在2020选举欺诈事件上没有任何行动，这是美国历史上最大的欺诈，都已经有如此多的证据。他们应该感到羞耻。历史会记住这一切。永不放弃。希望1月6日在华盛顿见到大家”，川普总统在推特上说。

“我对美国‘只顾自己’的司法部略知一二”，福林将军回复道，福林将军是川普总统几个被2016年选举特别检察官穆勒调查的盟友之一。

福林将军今年62岁，他在2017年早期因为有爆料称他对联邦调查局撒谎而离开白宫。他之后对于这个说法认罪，但是上月被川普总统赦免。他从此后就不断地建议川普总统使用“戒严法”并且让军队监督一次新的选举。
