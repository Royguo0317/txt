# **秘鲁暂停中共新冠疫苗的临床试验**

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/12/14064716/Picture1-2-16.png)

版权：Pixabay/CC0，公共领域

秘鲁在发现一位测试者神经问题之后，暂停了由中共医药巨头国药控股(Sinopharm)生产的新冠病毒疫苗临床试验。

根据地方媒体报道，秘鲁国家卫生院在周五表示，他们在一名志愿者发现难以移动手臂之后决定终止疫苗测试。

“几天前我们按照要求向管理部门发出警示，我们的一位受试者出现了神经系统症状，这可能与格林巴利综合征(Guillain-Barre syndrome)有关”，首席研究员戈尔曼·马拉加(German Malaga)对媒体说道。

格林巴利综合征是一种罕见且无传染性的疾病，这种疾病影响手脚运动。在发生多起病例之后，秘鲁去年六月在五个地区宣布了临时卫生紧急状态。

在1970年代，一场向美国人接种针对据说具有毁灭性的猪流感病毒（疫苗）的运动，在450位被接种者出现该综合症之后被停止，该综合症也可能导致瘫痪。

秘鲁的国药控股疫苗临床试验在测试约1.2万人之后，本应在本周结束。

如果他们成功的话—成功与否在2021年中才可以得知—预计秘鲁政府会购买多达2000万剂疫苗来为他们三分之二的人口接种疫苗。

全球已经有6万人接种了国药控股的疫苗，包括在阿根廷、俄罗斯和沙特阿拉伯的志愿者。

秘鲁是世界上（新冠）病毒人均死亡率最高的国家之一，截至周五，已有36499人死亡以及979111例感染。

这次疫情让南美国家经济严重受挫，在今年第二季度GDP下降了30%以上。
