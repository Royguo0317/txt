《The True Defender》刊登了一篇快讯，题为川普将在2月28日正式复出，并称他已经给乔拜登准备了一个大惊喜。

作者表示我们已经看到了，拜登作为一个老男人是多么的软弱。他呈现给我们的是一个崇尚自由主义和全球化的总统，也让我们看清了他绝对是中共的傀儡。

同时他表示据一位消息人士称，川普作为一名重要发言人，将于2月28日星期天在保守派政治行动大会(CPAC)的闭幕仪式上公开复出。

他将关注共和党的未来，以及现任总统拜登正在推动的灾难性大赦和他的边境政策。川普将发表演讲，并将声称他“仍在掌控”共和党，以此宣示自己在共和党的影响力。

今年的CPAC将在佛罗里达州奥兰多举行。

一位前总统的长期顾问透露，这次演讲将是一场“权力的展示”，川普会在演讲中宣布，“我没有Twitter，也没有椭圆形办公室，但一切都是我在管。”
