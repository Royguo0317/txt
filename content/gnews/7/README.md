## 獨家：中共間諜對澳大利亞海軍造船計劃構成“極端威脅”
`2020-08-16 22:39 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/299949/)

![IMG_256](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/08/16223045/1-73.png)圖片來源: [http://www.bw40.net/11785.html](http://www.bw40.net/11785.html) 
據澳大利亞廣播公司（ABC）國防記者安德魯·格林（Andrew Greene）獨家報導，澳大利亞國防部警告說，外國特工目前正密謀竊取澳大利亞的軍事機密，這對澳大利亞斥資900億美元的海軍造船計劃構成“極端威脅”。

信息自由（FOI）應用評估外國情報機構對主權能力和英聯邦戰略利益構成了極端威脅，國防部用該評估阻止莫里森政府公開發布有關當前和未來海上項目的機密簡報。

國防部在本月發表的一份正式文件中指出：“這些對手非常積極地尋求與澳大利亞當前和未來海上項目有關的信息，以提升自身利益並削弱澳大利亞的能力。發布這些信息可能會直接或間接損害澳大利亞利益。”

近年來，澳大利亞安全官員公開提到了外國特工對澳大利亞大規模軍事項目的威脅。澳洲安全情報機構（ASIO）總幹事邁克·伯吉斯（Mike Burgess）於二月警告說，“敵對的外國情報機構正針對我們”，部分原因是“我們正在全面加強國防軍和國防工業基地”。

中立參議員雷克斯·帕特里克（Rex Patrick）表示，國防部最新聲明中的措辭是史無前例的。 “這項評估超出了所有我聽到過的來自澳洲安全情報機構，國防部或《外交》（Foreign Affairs）雜誌的評估。這是一個嚴重的國家安全問題，政府必須採取一切必要措施來減少和消除這一威脅。”

在接下來的幾十年中，澳大利亞將斥資900億美元進行大規模的海軍造船計劃，將設計和建造數十艘尖端軍艦和潛艇，其中大部分工作將在南澳大利亞完成。

**中共** **被私下認定為罪魁禍首**

格林最後提到，國防部拒絕公開承認外國情報機構正在針對澳大利亞海軍造船業，但私下里，國家安全人士認為中共是罪魁禍首。對中共間諜的擔憂，澳大利亞不同政黨的政治家統一呼籲關閉駐南澳大利亞的中共領事館，或減少駐南澳大利亞的中共外交官人數。

2016年，北京在南澳大利亞州阿德萊德的西郊芬登開設了新的總領事館，可容納約10名外交人員。帕特里克參議員說：“南澳的中共領事館在澳洲聯合政府宣佈如此重要的海軍造船計劃的同一年建立了起來，我不認為這是一個巧合。”

美國聯邦政府和反對黨成員都對中共在一個擁有大量國防工業和太空研究的國家中擁有龐大的外交機構表示擔憂。

南澳州工黨議員尼克·錢皮恩(Nick Champion)說：“阿德萊德領事館的人數明顯超員，應該通過談判來減員。”

自由參議員康塞塔‧費拉萬蒂‧威爾斯（Concetta Fierravanti-Wells）稱，根據澳洲安全情報機構的記錄顯示，冷戰期間蘇聯間諜曾採取相似的策略針對澳大利亞的軍事研究。如果澳洲安全情報機構和執法機構認為中共領事館違反澳洲法律，就應將’外交官’驅逐。 ”

7月，美國認為中共駐休斯敦市領事館被用來經營間諜網，被美國政府下令關閉。同時，來自中共國家安全部的兩名特工還被美國當局起訴，涉嫌入侵美國及世界各地的政府和商業計算機系統，其中包括一家澳大利亞國防公司。

美國助理司法部長約翰·C·德默斯（John C Demers）於7月21日宣布：“如起訴書所示，中共黑客的目標鎖定在包括澳大利亞在內擁有高科技行業的科技公司。”

評：領事館的主旨是促進兩國之間的政治關係，促進經濟，文化交流。然而格林先生在報導中指出的中共駐休斯敦領事館現已被美國查實被作為中共間諜的“聚集地”，中共駐南澳領事館也疑似對澳大利亞海上軍事計劃收集情報。格林先生在文章中呼籲澳洲政府應對國防部的警告予以重視，向美國學習，盡快採取行動對中共“外交官”進行驅逐。
