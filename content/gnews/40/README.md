## 盘古相聚 7： 新一代盗国集团
` 纽约香草山农场` [轉載自GNews](https://gnews.org/zh-hans/536527/)

**作者：香草山文艺组 Tiffany ** 校对/发稿：飞虹

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/05173212/logo-2.jpg)



摘要：江胡斗的最后阶段，南普陀计划出炉，从十八大前的薄熙来家开始，一路灭掉了周永康、令计划、徐才厚等集团。十八大习王渔翁得利，新的盗国集团成形。习王以反腐之名大开杀戒，开始了“以警治国、以黑治国、以贪反贪”的时期。

1992年以邓小平为代表的老人集团为了平衡江泽民、李鹏，隔空指定了接班人胡锦涛。2008奥运会后胡锦涛声望极高，有了心腹李源潮控制中组部，令计划控制办公厅，可以确保他在退休前进一步完善他的人事布局。2011年底江泽民病重。眼看着胡锦涛的集团将要主导中国未来的人事之际，江泽民的病奇迹般地好了，疯狂的报复计划——南普陀计划——开始。

20年的江胡斗，在2012年的十八大达到了最为惨烈的地步。从十八大前后这把火一直烧到现在，烧出了海伍德案、谷开来案、王立军案、薄熙来案、令谷法拉利车祸案、令计划案、周永康案、徐才厚案、郭伯雄案 … … ([世纪大案 9：南普陀计划](https://gnews.org/zh-hans/377802/) )。不仅如此，十八大后上位的小媳妇习近平在内斗中坐大，和美国的奥巴马-拜登美奸集团合作，媳妇熬成了婆。

奥运时期的王岐山，尚在政治局夹着尾巴做人，头顶上胡锦涛、江泽民、温家宝都是他的老板。南普陀计划铲除了周永康，政法委书记的位置给了监国孟建柱，孟建柱又把心腹孙力军安排到了公安部一局管国保([世纪大案 7：中美执法合作](https://gnews.org/zh-hans/364009/))，再加上卖主求荣的傅政华，郭文贵先生爆料的王岐山、孟建柱、傅政华、孙力军盗国集团浮出水面。到了南普陀会议后的2012年11月8日，CCP贼窝开了十八大，王岐山终于在一人之下万人之上，CCP历史上最凶残的、也是危害世界最深远的习王盗国集团成型。

另一方面，王岐山得手后，盗国集团对刘志华案的疯狂报复和掩盖真相也开始了。中纪委、监察部、国安部等部门当年参加盘古刘志华案的人被大清洗。2015年3月底，王岐山的老相好胡舒立控制的《财新》发布一万多字的特别报道《权力猎手郭文贵》，报道刘志华、马建、民族证券诸案，一边攻击郭文贵先生，一边在舆论上支持打击王岐山的政治对手。经过盘古案、民族证券案、郑介甫案、谢建升案、曲龙案、李友案等，盗国集团的眼中钉肉中刺们纷纷落马。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/08004742/bkncn-20160513162354107-0513_05011_001_01p.jpg)
另一边，CCP的魔性围绕着十八大的斗争展露无遗，薄熙来、周永康、令计划、徐才厚集团也在这一进程中被团灭：

- 尼尔•海伍德(Neil Heywood)案——2011年11月13日，海伍德在薄家勤务人员张晓军的陪同下被紧急召集前往重庆。11月14日，重庆南山丽景度假酒店一房间，在谷开来在场的情况下，海伍德被人按倒强行灌入氰化物死亡。11月18日，在未进行尸检的情况下海伍德的尸体被火化，骨灰随后被空运回伦敦。重庆公安局长王立军调查此案期间被薄熙来调任，后前往美国驻成都总领事馆避难，引发王立军事件。
- 王立军案——海伍德后，薄家掩盖事实，并和王立军发生矛盾。薄熙来对王立军不满，并将其调离关键的重庆市公安局长岗位。王立军感到威胁，并于2012年2月6日携带着指控薄熙来的资料和CCP高层的秘密，化妆前往美国驻成都总领事馆并设法进入滞留。听闻消息的薄熙来安排市长黄奇帆调动大批警力到成都跨境执法，包围领事馆，希望把王立军抓到自己手上。时任驻华大使骆家辉(Gary Faye Locke)、国务卿希拉里、奥巴马-拜登政府商议此事，最终决定不收留王立军。在北京的干涉下，最终薄熙来没有捞到王立军，不得不把王立军交给国安部副部长邱进([世纪大案 12：邮件门](https://gnews.org/zh-hans/415418/) )。
- 薄熙来案—王立军案惊动中美高层，薄熙来自身不保。随后的两会薄熙来进京，高调面对媒体谈“也有不少人给重庆泼脏水，给我本人给家庭泼脏水，甚至说我儿子在外边学习，开红色法拉利，一派胡言，我感到非常气愤”。“敢同恶鬼争高下，不向霸王让寸分”。2012年3月14日，两会温家宝答记者问，谈王立军薄熙来事件，次日薄熙来被免职接受调查。
- 令谷法拉利车祸案——薄熙来被免后，北京各路武装力量发生异动，坊间多有军事政变传言。两天后的3月18日，令计划的独生子令谷和两名裸体美女驾驶法拉利跑车在北京北四环路保福寺桥发生车祸。令谷当场死亡。车上两女子分别是青海省公安厅副厅长女儿扎西卓玛，不久死亡；另一位是活佛的女儿杨吉，曾被救活，但几个月后死亡，似为被灭口。令计划私自调动中共警卫局封锁车祸现场，和有道路交通执法权的北京市公安抢夺事故现场。因中央警卫局和公安两边交恶，令计划出面不惜和昔日对手周永康结盟，合力摆平此事。之后周永康石油系的手下、中石油董事长蒋洁敏负责出钱让相关方面闭嘴。
- 319政变——薄熙来案发当日3月15日，胡锦涛因对安全局势的担忧，免去北京军区驻保定的38集团军军长王西欣，调派他出任国防大学副校长，接替他的是胡锦涛的亲信许林平少将。令谷案次日，周永康大规模调动京畿地区由政法委管理的武警力量，抢夺薄熙來案的重要证人徐明。胡锦涛调动38军发大批士兵和装甲运兵车入京，围攻北京市东城区灯市口西街十四号中共中央政法委总部。虽没抓到周永康，并有江泽民求情，但周永康大势已去，实权移交给了日后的监国孟建柱。


这一系列扑朔迷离的案件之后，江胡两边人马互有损失，习近平保住了王位，解决了最大的挑战者薄熙来，胡锦涛裸退。江家铲除了背叛的马仔，新的马仔出镜。2012年11月15日，十八大召开。十八大确立了新一代盗国集团的核心——习王体制。以反腐之名打击政治对手的大规模清洗开始。从那时开始到郭文贵先生爆料，就是所谓“以警治国、以黑治国、以贪反贪”的时期。

江家的南普陀计划、王岐山灭口奥运前的专案组知情人、习近平的宫斗和打击异己，这几股不同的动机交织在一起，中国的官僚阶级、红顶商人阶级，遭受了一次史无前例的大清洗。南普陀圈定的江家叛将，胡锦涛的团派，军队中的异己，王岐山视为眼中钉的中纪委，公安部等刀把子系统中的老人，纷纷以反腐的名义被拉下马，初略的看，大案便有：

- 孙政才——重庆市委书记
- 郭伯雄——军委副主席
- 蒋洁敏——国资委主任
- 李东升——公安部副部长
- 王珉——辽宁省委书记
- 项俊波——保监会主席
- 张阳——军委委员，政治部主任
- 房峰辉——军委委员，总参谋长
- 王建平——武警司令，副总参谋长


单单是公安部，从李东升、杨焕宁、孟宏伟，最后天道轮回，也少不得孙力军，乃至孟建柱本人。除了明面上用腐败来内部倾轧，还发生了2014年3月8日马来西亚航空MH370事件和2015年8月12日天津大爆炸事件。这些事件和CCP的内斗有着复杂的联系，最后却以无辜的平民生命为代价。

从薄熙来家的海伍德案发，到免薄之后的令谷车祸、319政变，直到十八大后习王以反腐之名祭起的滚滚人头。CCP这台绞肉机高速运转，从北京到地方，甚至到这个庞大机器的底层，都有大批官员落马，连带着家人百万人的命运从此改变。

在这一系列眼花缭乱的政治斗争中，世人无不服膺于习王下手之利落，甚至许多不明就里的人认为一个公正不阿、为国为民的包青天时代到来了。殊不知，这一切只是新一代盗国集团铲除异己，以便自己胡作非为的宫廷内斗。更大的秘密则在于习近平能赢的关键，却是美国掌权的人物——奥巴马-拜登集团。这条中美关系的暗线，以及这条暗线背后CCP对世界蓝金黄的数十年布局，还将在习近平当局的十年中继续变本加厉地作恶，并且危害人类的存亡。
