**原题目：川普的高级官员们推动了COVID-19实验室泄漏理论，调查人员曾有疑问**2

**原作者：Erin Banco（艾琳·班寇） 和Daniel Lippma（丹尼尔·力普玛）**

**以下是译文的第一部分**

新的文件和采访显示，总统和他的高级助手如何根据自己的喜好只把有利于自己的证据挑选了出来，并将政府自己的病毒调查员搁置一边。

川普政府高级官员在2020年春天决定强烈暗示COVID-19来自中共国实验室，尽管调查该大流行病起源的情报官员们没有对支持这一假设提供确凿证据。

这一信息传播源于为了反击中共国所做出的联合努力，因为中共国试图将病毒的传染归咎于美国。在POLITICO最新获得的纸质文件和在网路电文中，官员们分享了谈话要点，强调甚至北京政权自己的通信记录也承认疫情始于中共国的武汉省。

但白宫、国家安全委员会和国务院的一些官员曾经敦促美国进一步行动。他们想指责中共国掩盖大流行病的起源，并在无确切证据的情况下坚称病毒来自于武汉的一个专门研究危险蝙蝠病原体的研究机构。他们将此举描述为是一种外交上的冒犯行为。官员们说，其目的，部分是为了向中共国施压，让美国和国际社会进入武汉进行调查。

在几次场合下，前总统唐纳德·川普和国务卿迈克·蓬佩奥都公开发表观点，拥护有争议的实验室泄漏理论，远远超出了各机构之间商定的官方谈话要点。

在2020年4月30日的新闻发布会上，川普说政府有证据表明COVID-19来自武汉病毒研究所的一个实验室，尽管他拒绝提供具体细节。“我不能告诉你这个，我不被允许告诉你”，川普说。

几天后，5月3日，在美国广播公司的《一周总览》节目中蓬佩奥告说“大量的证据”表明COVID-19来自该实验室。

五名前川普政府官员，包括三名直接参与政府行政部门关于病毒起源调查的官员表示，政府收集到的证据并不能完全下结论的支持这些说法。在那个时候，这些官员已经审阅过了获得的文件资料、报告及其他情报。这些材料显示，该病毒有可能是源自武汉实验室，但他们无法证明。这些官员还说，在川普和蓬佩奥发表声明宣布实验室泄漏时，他们仍在考虑支持自然来源的证据，而且对于到底病毒是实验室来源还是自然来源，他们没有发现“冒着烟的枪”，意思是说，他们没有发现确凿的证据。

“我真的不是很确定他们到底在谈什么（证据）”，一位前国家安全高级官员在谈到川普和蓬佩奥时这么说的。“我们只是努力做到埋头工作，并在证据出现的时候，看这些证据，干自己的活儿。”

在这些网路电文和纸质文件中川普政府2020年春天关于病毒起源的内部谈话的这些描述，包括当时国家安全委员会（National Security Council，简称NSC）牵头对于信息的推进，体现了事态的严重度。材料显示，尽管并没有获得那个负责调查事务的小圈子里的人的内部集体共识，那些高级官员推进了实验室泄漏理论。高级官员们，特别是在实验室泄漏理论的间接证据方面，对他们所审阅的证据，在证据的份量和意义上，存在着分歧。

而白宫和国家安全委员会以及国务院内部的官员小组当时做出了决定，美国已经收集了足够多的间接证据，可以在公开声明中指出武汉实验室是病毒的来源，没有考虑调查当时所处的未最终下结论的状态。

“被普遍传播的假设是：（病毒）意外的逃出了武汉一个实验室”，一位前川普政府官员，一位知晓这次信息传播过程的人士这么说，“一部分的目的是测试你的假设，尤其是测试你对自己的假设是否有信心……看看中共国对这个假设的反应。”

两名曾在川普政府工作的国务院高级官员说，蓬佩奥的讲话超出了行政机构常规要求的谈话要点，并且，引发了盟国的兴趣：是什么导致了国务卿得出了病毒源自实验室的结论。

蓬佩奥的评论激起了人们的不知所措，在其他行政部门，在一些正在调查COVID-19来源的官员中，引起了混乱。多名国家安全委员会和国务院官员表示，他们是在看到了蓬佩奥在美国广播公司节目中的评论之后，才去阅读和核实这些有关实验室泄漏的报告和证据的，他们所看到的非常令人震撼，尽管不是结论性质的。

关于这段故事，POLITICO对蓬佩奥发出了询问，他在回复声明中是这样说的，“我们并没有根据自己的喜好只把有利于自己的证据挑选了出来。我们当时阅读了所有的情报。而且，现在全世界都知道了，我们的判断得到了巨量证据的支持，这些证据还在继续增加。”

（译文第一部分结束）
