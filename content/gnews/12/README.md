# **麻省理工学院教授被捕并被指控犯有科研拨款欺诈罪**

## **据称被告未能向美国能源部披露他为中华人民共和国所做的工作**

[!\[\]()!\[\](https://spark.adobe.com/page/j6Ou8B4kaJr5M/images/8a68c266-9df3-444c-9280-7116b348d3d3.jpg?asset_id=cac82d4a-1516-4846-96a3-17b895f2d457&amp;img_etag=%22dccb4358f5c34aae8034ef1c0c6c2813%22&amp;size=2560)](https://spark.adobe.com/page/j6Ou8B4kaJr5M/images/8a68c266-9df3-444c-9280-7116b348d3d3.jpg?asset_id=cac82d4a-1516-4846-96a3-17b895f2d457&amp;img_etag=%22dccb4358f5c34aae8034ef1c0c6c2813%22&amp;size=1024)

现年56岁的陈刚（Gang Chen）因涉嫌电汇欺诈，未提交外国银行账户报告（FBAR）和在纳税申报单中虚假陈述而被刑事指控。

波士顿 — 今天麻省理工学院（MIT）的一名教授兼研究员因未向美国能源部披露来自中华人民共和国（PRC）不同实体的合同、任命和奖励而被起诉和逮捕。

现年56岁的陈刚（Gang Chen）因涉嫌电汇欺诈，未提交外国银行账户报告（FBAR）和在纳税申报单中虚假陈述而被刑事指控。今天，陈刚将首次出庭面对地方法官唐纳德·卡贝尔（Donald L. Cabell）。

根据指控文件，陈刚是在中国出生的、已入籍美国的公民。他是麻省理工学院的教授兼研究员，担任麻省理工学院帕帕拉多微/纳米工程实验室主任和固态太阳能热能转换中心（S3TEC）主任。大约自2013年以来，陈刚在麻省理工学院的研究得到了美国多个联邦机构提供的超过1900万美元的资助。

据称，自2012年以来，陈刚担任多个中共国安排的职务，旨在通过提供建议和专业知识来促进中共国的科技发展——有时直接向中共国政府官员汇报——并通常以经济补偿作为交换。这包括应中共国驻纽约总领事馆的要求，担任中共国政府的“海外专家”，并至少担任两个中共国人才计划的成员。自2013年以来，据称陈刚接受了大约2900万美元的外国资金，其中包括来自中国南方科技大学（SUSTech）的1900万美元。

更进一步指控，陈刚在2016年2月使用自己在麻省理工学院的电子邮件帐户发送的一封电子邮件中，部分详述了陈刚为促进中共国科学和经济发展的努力。电子邮件内容如下：

陈刚在MIT实验室

1. 促进中共国合作
2. 中共国把创新（科学）作为关键和核心，而不是追随时尚流行（原文如此），而且因为无论是从历史趋势还是从我们现阶段来说，我们必须做到这一点
3. 我们的经济是世界第二位，但从技术（经济结构）和人力资源上，我们远远没有达到第二
4. 除了劳动成本上，我们也付出了巨大的环境代价，是不可持续的
5. 在同一个地方的环境保护与发展相比，环境保护要高一些，是否花费更多在清洁能源上，而减少钢材、水泥。我们必须依靠技术，不能像过去一样成长
6. 共产党第十八次代表大会，科学创新为核心。我们不仅要实现自主创新，同时也要实现国际化以规划和促进创新。闭门创新是行不通的；创新作为驱动力


至少从2017年到2019年间，当陈刚在中共国和中共国实体中担任多个咨询职务时，陈刚申请并获得了美国能源部（DOE）赠款，以资助他在麻省理工学院的部分研究。据称，陈在这样做时，没有按照美国能源部的要求披露有关其与中共国长期联系的信息。

据称，陈刚还没有在其2018年纳税申报表中向国税局透露，在2018年他在中共国拥有一个超过1万美元的银行账户。

电汇欺诈罪的最高刑罚为监禁20年、监督释放3年和最高25万美元的罚款。作出虚假陈述的指控可判处最高5年监禁、3年监督释放和罚款25万美元。未能提交外国银行账户报告（FBAR）的指控将被判处最高5年的监禁、3年监督释放和罚款25万美元。联邦地方法院法官根据美国《判刑指南》和其他法定因素判刑。

美国检察官安德鲁·莱林（Andrew E. Lelling）；波士顿外勤部联邦调查局特工约瑟夫·博纳沃隆塔（Joseph R. Bonavolonta）； 美国东北部防务刑事调查局美国国防部特别负责人帕特里克·J·哈加蒂（Patrick J. Hegarty）；波士顿负责国土安全调查的代理特工威廉·沃克（William S. Walker）；负责美国国税局在波士顿进行刑事调查的代理特工乔恩·辛普森（Joleen Simpson）； 监察长办公室能源部特工吉姆·布雷肯里奇（Jim Breckenridge）今天宣布了这一消息。莱林的国家安全部门负责人，助理美国检察官斯蒂芬妮·西格曼（B. Stephanie Siegmann）以及同为莱林国家安全部门负责人的杰森·凯西（Jason Casey）和蒂莫西·基斯纳（Timothy Kistner）正在国家安全部反情报和出口管制科的审判律师大卫·亚伦（David Aaron）的协助下起诉此案。

起诉文件中包含的细节均为指控。除非直到在法庭上被告经证明毫无疑问有罪，否则被告被推定为无罪。
