# **美国官员说：“越来越多的证据”表明COVID-19从中共国实验室泄漏**

[!\[\]()!\[\](https://spark.adobe.com/page/vtNCNvtQyTCHF/images/76cc80ba-e602-4b60-b6ec-8913a4eecec3.jpg?asset_id=8985cc10-fcf3-4777-8f60-4f9fb8dbe6ad&amp;img_etag=%22ec74584bf740e9aba3a370ba3afc0abe%22&amp;size=2560)](https://spark.adobe.com/page/vtNCNvtQyTCHF/images/76cc80ba-e602-4b60-b6ec-8913a4eecec3.jpg?asset_id=8985cc10-fcf3-4777-8f60-4f9fb8dbe6ad&amp;img_etag=%22ec74584bf740e9aba3a370ba3afc0abe%22&amp;size=1024)

位于湖北省中部的武汉病毒学研究所园区

美国国家安全顾问博明（Matthew Pottinger）对新冠病毒从中共国武汉的一个政府运营实验室泄露的理论进行了再加码。

据称，博明是坚定的对北京批评者，他是在最近与英国官员的Zoom视频会议上提出这一说法的。

据《每日邮报》报道，“有越来越多的证据表明，该病毒最可能来自实验室。”博明说。

这位川普任命的官员推动了这一理论。上周，欧盟不顾博明的抗议和即将上任的拜登政府的犹豫，与中共国达成了一项新的投资协议。

博明是早在2020年1月就在白宫对病毒来源发出警报的首批美国官员之一，据报道，从疫情爆发初期他就怀疑新冠状病毒源于中共国实验室。

据《纽约时报》4月报道，他命令美国情报机构搜索其拥有的证据。

一位中共国病毒学家说她对新冠病毒做了一些最早期的研究，并公开声称新冠病毒是人造的，中共国政府掩盖了它的危险性。西方医学专家已经否定了这一理论。

自2020年1月首例病例公开以来，世界卫生组织一直在调查病毒的来源。零号病人尚未找到。

博明在最近与英国官员的通话中表示，世卫组织的调查是个诡计。

博明对议员们说：“全世界的议员应该发挥道德作用，以揭露世卫组织的这场‘波将金村庄’式的调查。”他引用了18世纪，为使来访的俄罗斯女皇凯瑟琳大帝相信该地区非常好，而在克里米亚制造的假村庄。

据称，博明说：“连北京的大人物都公开否定了海鲜市场的说法。”他指的是另一种理论，即病毒是在武汉的一个野生动物市场内由动物传染给人的，武汉的第一个病例群就出现在那里。
