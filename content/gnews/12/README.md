# 科尔特斯：共和国……还是寡头政治？

**据报道，1787年在费城举行的制宪会议闭幕时，当被问及制宪者为我们的新国家成立什么样的政府时，本杰明•富兰克林（Benjamin Franklin）回答：“一个共和国，如果你能守住它。”**

像其他开国元勋一样，富兰克林深刻地理解像罗马共和国这样的代议制政府如何演变为寡头政治和最终的专政。因此，《美国宪法》努力从古典世界引导最佳自治做法，同时避免了类似的后续命运。

[!\[\]()!\[\](https://spark.adobe.com/page/fiLpRuaGHkVlC/images/b7f37992-af6e-4159-8128-937a634813fc.jpg?asset_id=856b0478-5b8e-4260-9299-fb395856c963&amp;img_etag=%22853fe5630c8f91b0df46a30aa30393e2%22&amp;size=1024)](https://spark.adobe.com/page/fiLpRuaGHkVlC/images/b7f37992-af6e-4159-8128-937a634813fc.jpg?asset_id=856b0478-5b8e-4260-9299-fb395856c963&amp;img_etag=%22853fe5630c8f91b0df46a30aa30393e2%22&amp;size=1024)*(本杰明•富兰克林签署美国宪法)*

未来一周，在美国很可能会决定“是否可以保留”这一部分。

在美国，最强大的利益集团以惊人的同步性，试图完成一场盛大的选举盗窃，以永久分裂我们国家的方式真正颠覆我们共和国的基础，并鼓舞那些试图将美国转变为一个有效寡头政治的非民选权力代理人。

对于2021年美国相关统治阶级的名人来说，这种观点似乎是极端的。在纽约的新闻编辑室、华盛顿的游说办公室、公司会议室和大学教师休息室中，对操纵11月3日总统选举持怀疑态度的人的确是精神病。但实际上，数千万美国人非常明智地对我们选举的完整性抱有严重怀疑。

尽管企业和媒体竭尽全力压制相关信息，但怀疑者的数量却迅速增加。拉斯穆森（Rasmussen）最近的一项调查显示，有47%的美国选民相信民主党人为拜登窃取选票和/或毁灭了川普的选票以确保所谓的胜利。在支持川普的选民中，这种愤世嫉俗几乎达到了空前的一致，根据美国消费者新闻与商业频道（CNBC）的民意测验，只有不到3%的支持者支持总统让步给拜登。

这些普遍的疑问代表了基于证据、基于统计数据和我们的宪法的深思熟虑的怀疑。

[!\[\]()!\[\](https://spark.adobe.com/page/fiLpRuaGHkVlC/images/769fa40c-64ba-4edb-8c40-eece2e2b119a.png?asset_id=5266ec68-a47e-49fe-9406-8305204f1991&amp;img_etag=%227574d00e65f29a6a1e4e5e4498ccdfcd%22&amp;size=1024)](https://spark.adobe.com/page/fiLpRuaGHkVlC/images/769fa40c-64ba-4edb-8c40-eece2e2b119a.png?asset_id=5266ec68-a47e-49fe-9406-8305204f1991&amp;img_etag=%227574d00e65f29a6a1e4e5e4498ccdfcd%22&amp;size=1024) (皮特•纳瓦罗指控在2020年总统选举中如何发生选民欺诈)

在数据方面，有很多证据表明，拜登的胜利需要统计异常的融合，这是不可能的，实际上是不可能达到的。认真的思考者继续扩展这种推理。我提供了自己的初步总结，这源于我为全球最大的对冲基金25年概率分析的工作经验。

然后，具有一流学术资历的世界级人才通过更深入的白皮书来补充这一案例，这些白皮书涉及拜登“干净”获胜的可能性。其中包括哈佛大学博士、白宫经济顾问皮特•纳瓦罗以及加州大学洛杉矶分校和前常春藤盟校的约翰•洛特（JohnLott）教授。

除了这些令人信服的数字论证之外，由于主要战场州完全篡夺了有效选举的法律先决条件，因此违宪的意见增加了。具体来说，在佐治亚州、宾夕法尼亚州和威斯康星州等州普遍存在违反《宪法》第14项修正案的事实，这实际上创建了两类选民，对川普的选民进行了严格的核实，而对所谓的拜登投票的验证几乎完全无效。

但是，尽管有这些重大和有效的反对意见，但迄今为止，强大的利益集团还是匆忙完成了一个充满污点的选举过程，并将一个违法的，甚至可能是不合法的新总统强加给美国人民。

正如我所详述的，乔•拜登将出任美国历史上最令人怀疑的新任总司令。

他不是被授权的，而是从跳跃中蹒跚着进入椭圆形办公室，因为选举中的渎职行为和该死的中共贿赂指控像两朵乌云一样笼罩着他的新总统任期。除非美国人民在任何新总统宣誓就职之前对这些黑暗情况进行全面和透明的审查，否则拜登的政府将陷入瘫痪。

美国社会有抱负的权贵们没有表现出这种透明度，反而给我们国家造成了惩罚性的不透明，嘲笑并惩罚那些勇于挑战官方裁决的勇者。谁构成了这种新兴寡头集团？有形成了富豪们欺骗性的三连胜的三组人：公司媒体、大型科技公司和建制派政治家。

多年来，那些试图将可笑的、毫无根据的俄罗斯阴谋推向美国的媒体专家，现在都故意否认了有关11月3日全民投票的大规模欺诈的真实证据。偏颇的媒体甚至不假装做一个调查新闻，直接无视这些明显的违规行为，例如视频记录的非法计票程序，以及在关键州无法原谅的午夜暂停计票，以及随后在相同的州，又出现了大量统计上无法解释的拜登投票激增，一个制表就重新开始了。

大型科技公司很乐意与公司媒体合作，干预我们的选举过程 — 实际的选举勾结。例如，在竞选期间，诸如Facebook、Google和Twitter之类的社交媒体公司很容易对关键的中共国-拜登贿赂丑闻的言论实施审查，甚至在川普竞选活动中禁言了我们的官方账户。自大选以来，过滤和审查制度仍在继续，甚至川普总统本人在分享对选举欺诈和违宪行为的分析时，也受到近乎常态的编辑和压制。

这些巨型科技公司明知接受拜登任职总统将增加其经济主导地位。例如，他们明智地预测，拜登的同谋将撤回对中共国的经济制裁，这是大多数大型跨国公司的主要政策目标。此外，拜登承诺的“黑暗的冬天”，即目前主导民主党司法管辖区的严格中共病毒限制措施将遍布全国，这将极大地增加受益于此类封锁的大型企业的市场地位，同时消灭较小的竞争对手。

各大媒体和大型科技公司在全美两党的民选公职人员中找到许多愿意推动者。即使在白宫呆了四年之后，川普总统仍然在很多方面是一个“局外人”和可悲的人，他所坚持的“美国优先”运动一直是寡头计划的严重障碍。

我们的主权、经济民族主义和权力扩散的运动目标都可挫败勾连的统治阶级的邪恶目标，该统治阶级视乔•拜登为一个有用的支持者。不幸的是，许多共和党人都同意，并且很乐意退回到符合杰布•布什（Jeb Bush）和保罗•瑞安和（Paul Ryan）的共和党立场。尽管川普是“伟大的揭露者”，迫使其中许多人陷入严酷的政治气氛，但大多数这些虚假无瑕的“保守主义者”都试图掩饰自己的真实忠诚。例如，参议员马可•卢比奥（Marco Rubio）和约翰•科宁（John Cornyn）等全球主义社团主义的共和党人最近表示，他们渴望加入拜登政府，实行大规模的大赦，这将使我们的边界形同虚设并残害美国工人。

2021年1月很好地为我们这一代人提供了一个“与命运的约定”。美国会屈从于短期的舒适并默许吗？我们会允许新生的寡头统治颠覆我们的共和国吗？或者，我们的人民是否会以值得我们作为历史上最伟大国家的公民的身分崛起，要求透明并遵守我们神圣宪法的戒律？

这是我们的共和国……如果我们能保住它的尊严。
