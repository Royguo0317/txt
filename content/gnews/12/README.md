# 美国科技巨头要千方百计保障选举安全

[!\[\]()!\[\](https://spark.adobe.com/page/IOUjJOydA3C9p/images/888255ce-0c1c-43f2-9c6b-4fc8497545df.png?asset_id=ad9fcac7-b217-4555-8aa8-4c794d7e6c68&amp;img_etag=%220c21b0b01652eb1490a73f3053c3035c%22&amp;size=1024)](https://spark.adobe.com/page/IOUjJOydA3C9p/images/888255ce-0c1c-43f2-9c6b-4fc8497545df.png?asset_id=ad9fcac7-b217-4555-8aa8-4c794d7e6c68&amp;img_etag=%220c21b0b01652eb1490a73f3053c3035c%22&amp;size=1024)

在总统大选前夕，拜登和川普在网上交锋，而科技巨头们正竭尽全力防止选举干扰自己的平台。

为了防止假新闻、错误信息的传播和外国(势力)干扰，美国出台了一系列新措施，目的是阻止社交媒体平台被用来干扰选举的合法性—- 就如在2016年大选中看到的那样，美国的社交媒体公司被外国(势力)当作煽风点火的武器。

围绕着大科技公司对经济和社会日益增长的影响，在国会听证会和更广泛的反垄断关注之后，这些科技巨头已经成为焦点，他们正在尽一切努力确保这种潜在的影响不会以有害的方式蔓延到选举中。

在过去的4年里，在政府、智库和学术界的参与下，旨在保障选举的安全，科技公司已经花费了数十亿的资金准备来防范这种情况的发生。

在最基本的层面上，互联网公司承担了为选民提供准确、最新的选举信息的责任。

脸书决定运行一个投票信息中心，以打击虚假信息，鼓励投票。这个功能早在6月就宣布了，出现在用户反馈的顶部，并给出了投票时间和如何投票的信息。

(科技)公司们正在采取一些措施来应对这种虚假信息。抖音(TikTok)在周三的一篇博客中宣布，它将更新其关于 “误导性内容 “的政策(规定)，并(与其他公司)扩展事实核查(业务) 的合作伙伴关系，以 “帮助核实与选举有关的错误信息”。抖音用户将能够通过在大选前新增的应用内功能举报虚假信息。

[!\[\]()!\[\](https://spark.adobe.com/page/IOUjJOydA3C9p/images/bec58783-7100-4c32-86fe-f21f06755bd3.jpg?asset_id=638deafd-c0f3-45aa-a940-d6bfcaed4a62&amp;img_etag=%2239d5fc887c1b99eddb13a522d90c770f%22&amp;size=1024)](https://spark.adobe.com/page/IOUjJOydA3C9p/images/bec58783-7100-4c32-86fe-f21f06755bd3.jpg?asset_id=638deafd-c0f3-45aa-a940-d6bfcaed4a62&amp;img_etag=%2239d5fc887c1b99eddb13a522d90c770f%22&amp;size=1024)

由于专家预测选举(结束)当晚可能不会公布正式结果，脸书正提前为万一发生的情况做准备，以防止人们因得不到最终结果而被误导，甚至看到虚假的结果信息。

据《纽约时报》报道，脸书的员工正在 “制定应急计划，并完成大选后的情景，其中包括川普先生或其竞选团队试图利用该平台使选举结果非法化”。
