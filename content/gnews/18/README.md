**北约正处在十字路口，必须在政治和战略上重新定位自己。该联盟的国家元首和政府首脑希望在6月14日于布鲁塞尔举行的峰会上对此进行讨论。**

北约军事委员会密集地准备了这次会议，以证明该联盟的可行性。提醒一下：法国总统埃曼纽尔·马克龙（Emmanuel Macron）将北约描述为“脑死了”。

意大利军事代表罗伯托·诺迪奥说，除了传统的威胁和恐怖主义之外，还有俄罗斯这样的区域侵略和网络攻击带来的新风险，必须对此加以防范。此外，在大流行紧急情况下，部队承担了民用任务。由于军方无法独自应对挑战，因此需要欧盟等国际合作伙伴。

北约中将再次表示，北约与欧盟之间的合作并不容易，但要奏效。从长远来看，有必要向欧洲纳税人解释为什么他们应该支付两次费用，一次支付给欧盟，一次支付给北约。

今年，北约将结束其在阿富汗的20年任务。即使在撤军之后，北约仍将与阿富汗保持军事伙伴关系，充分尊重阿富汗主权。

北约和欧盟将继续捍卫西方民主价值观。

简评： 很多人把欧盟和北约都混肴， 欧盟有27个成员国，全部在欧洲。 而北约是包括美国在内的军事互助联盟。 而欧盟的27个成员国中有22个是北约的成员。 北约将在6月14日召开峰会， 美国总统拜登上任后的第一次出访就是这里。 我们都知道，  全世界都面临着中共的威胁， 美国总统首次出访不进行外交互访， 而是参加北约峰会， 意义非同一般。而北约成员国对各国承担的北约费用是有分歧的， 希望这次峰会能够在这方面取得进展。 北约峰会的议题上肯定是少不来对付中共的议题。 希望会议之后会有北约的集体行动。
