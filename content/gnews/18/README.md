**西班牙2021年6月12日电/西喜社——**

根据国际特赦组织周四发表的一份新报告，中国政府对该国穆斯林少数民族群体的行为构成了反人类罪。该报告详细介绍了中共的国家组织的对新疆人民，包括维吾尔族和哈萨克族的系统性大规模监禁、酷刑和迫害。报告还详细介绍了中共政府对其种族灭绝行为的掩盖。50多名被关押在集中营的人为国际特赦组织的报告提供了证词，他们中的每一个人都说他们受到了中共的酷刑或其他虐待。

联合国说，中共国内有多达150万维吾尔人被关在拘留在集中营中。共产主义受害者纪念基金会中国研究高级研究员阿德里安·赞兹（Adrian Zenz）去年在接受NPR周末版采访时称，这可能是自大屠杀以来对一个民族宗教少数群体的最大监禁行为，并表示这一切都符合联合国对种族灭绝的定义。

今年早些时候，美国与欧盟、英国和加拿大一起对中共国进行制裁，抗议其 “侵犯人权”。

NPR（全国公共广播电台） 的All Things Considered（万事皆晓栏目）采访了国际特赦组织的高级危机顾问和该报告的主要作者乔纳森·勒布，关于与前拘留营的被拘留者们进行的访谈，这份报告证明酷刑在这些拘留营中是普遍存在的，以及关于在中共国根除伊斯兰宗教活动的情况。

艾尔莎·张：“如果我没弄错的话，这份报告是对在新疆被拘留的人员的第一手资料的最大汇编。是这样吗？”

乔纳森·勒布：“是的。尽管在过去四年中，至少有数十万人被送进了拘留营，新疆的数百万穆斯林受到了那里局势的影响，但很少有人能够走出国门，公开谈论这一问题。在过去的18个月里，国际特赦组织试图做的是确定并联系其他能够离开新疆的人，但出于安全原因，他们之前一直不愿意公开发表言论。因此，我们投入了大量的时间和精力来追踪55名以前没有进入公众视眼的难民营被拘留者们。我们确保我们可以通过安全的渠道进行这些采访，并尽可能为他们提供安全保障。”

艾尔莎·张：“现在，关于这些集中营和对这些集中营以外的居住在新疆的维吾尔族人进行大规模监视的报告大约在四年前开始出现。那么，请告诉我，这份报告为我们全面了解在新疆发生的事情增加了哪些最重要的新细节？”

乔纳森·勒布：“我们的报告是对现有证据的补充，而不是重复。因此，这些是新的证词，它们提供了大量新的细节，不幸的是，关于集中营中正在发生的可怕事情。我们得出的结论是，每一个被送进拘留营的人都会经历酷刑或其他虐待，这既是集中营日常生活的写照，也是他们中的许多人在集中营期间的审讯和惩罚中经历身体折磨的记录。”

艾尔莎·张：“请允许我问一下，有没有什么特别的细节让你印象最深刻？”

乔纳森·勒布：“是的，不幸的是，国际特赦组织采访过的前被拘留者中，大约有17或18人在被固定在老虎椅上时受到审讯和身体的折磨，老虎椅基本上是钢制的，你的手和脚被固定在椅子上，你完全无法动弹。”

中共国政府早就说过，它之所以特别针对这一人群，是因为这一地区的人们给政府带来的 “恐怖主义威胁”。我们应该注意到新闻报道，成千上万的维吾尔人去为叙利亚的ISIS作战。在新疆发生的种族灭绝事情绝对是令人遗憾的，但是否还有一些理由值得关注？”

每个政府都有权根据国际法对任何合法的恐怖主义威胁作出反应。但是，我们在这里看到的是一场完全基于其宗教和文化而针对整个穆斯林民族的行为。”

艾尔莎·张：“正如你所说，最持久的影响之一可能是导致新疆本土文化的丧失。人们因为说自己的母语而不是汉语普通话而受到惩罚；人们因为携带甚至只是宗教主题的图像而受到折磨；妇女被绝育。你认为中共国的这些少数民族穆斯林人口的未来会是什么样子？”

乔纳森·勒布：“不仅是未来格外暗淡，现在也是如此。我们在这里谈论的很多事情都已经发生了。许多对遵循伊斯兰教至关重要的传统——无论是祈祷、去清真寺、教授宗教知识、穿宗教服装、给孩子取伊斯兰教的名字——现在实际上都被中共禁止。而结果是，为了生存，新疆的穆斯林被迫改变了他们的生活和行为方式，中共不允许他们再从事本民族的宗教活动。”
