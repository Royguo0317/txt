## **前国民党主席洪秀柱：国民党针对此议题应回归价值立场思考**

今日，洪秀柱出席「护宪保台论坛」研讨会，针对记者追问新疆棉花争议，洪透过脸书回答「站稳价值，才能够坚定我们两岸交流的信心」。

所谓护宪保台论坛，是国民党在202年推动的议题。从护宪保台决议文中可得共有六大主张，其中四大点对和平协议灭台有极重大影响：

1. 中华民国领土主权范围以宪法之固有疆域为准。 （指包含大陆）
2. 为达成「三民主义统一中国」，国民党反对任何破坏以三民主义之势力，坚决反对台独、共产极权主义
3. 以两岸和平为目标，强化两岸交流，透过两岸间的制度竞争来验证三民主义的时代价值。 （[和平方式：和平协议公投](https://gnews.org/zh-hant/964277/)）
4. 国民党宣示捍卫以三民主义、五权分立为本之中华民国宪法。 （可认为是要捍卫中共犯台的合法理由）


所谓应该要回归价值立场思考，国民党现任党主席江启程表示：煽动言语、诉诸民族主义、相互的指责或抵制，并无助于了解真相，也无助于促进新疆当地人权与生活、劳动等条件的再提升。

## **台湾新闻媒体连自己人都不看，挺川、灭共早就是人民的立场**

台湾新闻媒体素质低落，报导虚伪不实资讯为常态，煽动、曲解更是日常手段。

但台湾新闻媒体长年的炒作，以及低水准表现，早已经使台湾人民不以之作为吸收资讯的管道。台湾人民也早已经站在灭共的立场上，但并没有实际上支持作为。

两岸关系诡谲，经郭先生直播表示，台湾政府在政治决策上过于保守，为求短期平安而拒绝美国伸出的援手，才导致了现在身处战争爆发边缘的处境。

又台湾也存在许多内患尚未铲除。从近日的议题来看，中共代理人在台湾利用法治、公投等，炒作环境议题、能源议题、主权议题、和平经济议题，意图混淆、煽动民意，利用无知的人民去支持没有效益的环保、经济、外交策略，达到真正出卖台湾的政治价值。

## **对台湾伸出援手的新中国联邦，是否有机会在台湾传播声名？**

新中国联邦成功让世界区分中共、中国人民，并且一系列护台、港、世界的动作，揭露病毒真相、中共邪恶意图等，早已经成为世界舞台的中心。

又新中国联邦是否可以顺利在台湾打开知名度，个人认为有些关键点需要攻克。台湾长久受到中共打压，配合媒体炒作，对于「中国」一词的使用，稍微敏感。

可以认为这是一种厌恶到极致后，产生的效应。但并非代表讨厌大陆人民，而是在明确知道中共、中国人为不同的前提下，厌恶自身被冠上中国名号的情形。 （非常复杂，但人心正是如此）

因此台湾农场有别于世界其他农场，在传播真相同时，需要特别避免勾起民众反抗的心态，专注利用「病毒真相」以及其他人权议题，达到传播效果，方有可能打开台湾民众对于新中国联邦的接受度。

新中国联邦对于台湾的帮助以及包容，绝对是直得台湾人民信任。即便后续加入的支持者，或多或少有各自的政治立场。但总体而言，以新中国联邦具有代表性地位的人物来说，新中国联邦对于台湾的付出绝对是建立在保护人民、文化以及民主发展上，并且一系列的作为也相当的尊重台湾使用者。只要台湾农场可以避免勾起台湾人民的反抗心态，的确可以使新中国联邦在台湾民间有影响力。
