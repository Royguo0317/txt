中共病毒产生新变种，可能出现二到三种基因变异，并发现有与之前不一样的主要表现症状 ，印度的Souradipta Chandra医师在4月22日报告了此种新发现。

新的变种病毒较常见到的症状包括：腹泻、腹痛、红疹、结膜炎、意识不清、脑雾、手脚趾发青、鼻子与喉咙出血等。此不同于目前所知道的常见症状，例如：喉咙痛、身体痠痛、发烧、嗅觉味觉丧失等。
