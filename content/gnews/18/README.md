《门道专家》（TGP）的乔丹·康拉德（Jordan Conradson）采访了亚利桑那州的审计总监肯·贝内特（Ken Bennett），称“取得了巨大的进展，包括增加3倍清点量的计划，以及舞弊选票数量的议题。”

《门道专家》发表文章【1】，周五晚上，网站的权威人士乔丹·康拉德森采访了亚利桑那州的审计总监肯·贝内特，谈了历史性的马里科帕（Maricopa）郡法庭审计的最新进展。

肯·贝内特就这次“历史性审计”给了《门道专家》一次独家专访的机会，“历史性审计”周五仍然在凤凰城的退伍军人纪念体育继续进行。

《门道专家》的乔丹·康拉德森：到目前为止进展如何？

肯·贝内特：一夜之间，（体育馆内）我们的计票点从20个变成了46个！分包商正在与一家临时服务公司合作，以获得所需的额外人员。希望在接下来的一两天里，他们能做好充分的准备，如果我们现在讨论的是三班制，我们的点票能力几乎会增加三倍。

乔丹：你们有没有发现任何你们认为有欺诈嫌疑的选票？

肯·贝内特：不，我们在审计结束前不会公布任何消息，所以现在他们只是在清点选票，他们正在查看评估文件，看看邮寄回来的选票是否有褶皱，（确定）椭圆标记是由手工填画的，而不是用复印机

复制的一堆选票。所有审计结果和调查结果都将作为最终报告的一部分，我们不打算每天透露一点，因为某一天我们发现了好的事情，也许第二天就有不好的事。

乔丹：所以我想问你为什么不每天公布数量？这对公众来说不是更有用吗？

肯贝内特：很显然，承包商，参议院与网络忍者的合同要求不定期发布调查结果。所以我认为他们强烈地感觉到所有的东西都应该在最终报告中，所以所有的东西都作为一个整体来看待。我们能想出一个办法来公布我们每天的投票数吗？这不是一个调查结果，那只是一份进度报告，我们正在努力。

乔丹：最近有三个国家左派团体向拜登司法部请愿，试图关闭这一进程，对此你有什么看法？

肯·贝内特：我所看到的只是布伦南中心（the Brennan Center）的信，但我认为它是由其他一些团体共同签署的，所以如果你指的是这封信的话，我在他们的上诉书中读到的、要求的伸张正义，是完全没有根据的，他们所称的可能发生的危险，选票丢失、被偷或被毁等，是完全荒谬的，我们有很强的安全保障，我们每天收到的最大的投诉是来自媒体或观察员，或是试图进入大楼，但名单上没有他们名字的人……我们的情况非常安全，我正设法抽出时间把我的信寄给司法部，以便回复布伦南中心（的怀疑）。

乔丹：我也听说过紫外线光扫描，你能告诉我们这些是干什么用的吗？

肯·贝内特：紫外线光扫描是他们对纸质（选票）评估过程的一部分，他们用来寻找选票上的水印，郡选举（办公室）向我们保证选票上没有水印……这是他们用来评估纸质文件和我前面提到的东西的一部分。

乔丹：那么你能在截止日期前清点完所有选票吗？

肯·贝内特：没有最后期限，我们知道，我们在5月15日前可以一直使用这个设施，之后的5天，凤凰城联合高中区的18所中学要在体育馆举行毕业典礼，所以我们知道在这段时间里体育馆是不能用的。但是我们可以把选票移到安全的地方，也许我们甚至可以让人们休息四五天，毕业典礼一结束，我们就可以马上回来，回到我们离开的地方……他们告诉我们，在这五天结束后，我们可以拥有这个设施，只要我们需要它。

亚利桑那州参议院和亚利桑那州共和党正在认真对待这一进程。

历史正在被创造。
