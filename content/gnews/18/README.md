## 【香港要聞】 中共魔爪伸向香港教育界 響起警號  打壓言論自由
`2020-10-07 00:48 GM42` [轉載自GNews](https://gnews.org/zh-hant/407598/)

蒐集：文燕 KEN老廣
編撰：文燕
覆核：文白
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/07003454/20201006-05_l6fPW_1200x0.png)圖片：立場新聞
據**【立場新聞】**10月6日訊，日前教育局取消九龍塘宣道小學一名教師的註冊資格，指其在教材中散播“港獨”訊息。有資深校長向【立場新聞】表示，今次教師被釘牌事件“屬政治，無庸置疑”。就算當局認定教材有“傳達意識”，但並沒有明確、實際資料，顯示老師在“散播”港獨或任何政治理念，形容當局為釘牌解畫砌詞 “好牽強，絕對不尋常”。

**【立場新聞】**翻查教育局回覆財務委員會的書面資料，從 2010 年至 2019 年，教育局一共取消或拒絕 98 名教師註冊。教育局表示，有關教師主要觸犯性罪行、與性有關的罪行或欺詐罪行，以及重複違犯一些輕微的罪行，或有嚴重的誠信問題等。對此，資深校長坦言，取消教師註冊本為嚴重懲處，以往多數是涉及嚴重刑事罪行，例如非禮學生、在學校偷竊等，尚且要待法庭作審訊及判刑，確保涉事老師有充分抗辯機會，才會考慮取消其註冊。

資深校長認為，涉事的宣道老師充其量“價值觀”不符當局標準，並沒有向學生灌輸”港獨是對的、港獨是出路”，當局形容老師有計畫地”散播”港獨訊息，指控相當牽強。
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/07003511/20201006-1220copy_6Owhg_1200x0.png)圖片：立場新聞
**國際特赦組織中國組**主管羅助華（Joshua Rosenzweig）指，政府對履行職責的教師施以“嚴厲和毫無根據”的處罰，令教育工作者憂慮在課堂討論時事、政治及人權會有後果， “對教育界來說是一個警號。”
![]()![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/10/07003437/photo_2020-10-06_23-42-35.jpg)圖片：立場新聞
**戰友點評：【文燕】**

**1.事件的發生明顯是中共要“殺一儆百”，港共找人祭旗，找替死鬼。**
     經過反送中運動後，年輕就是罪，尤其是穿上黑衣的年輕人，走在街上，都會無端端地被黑警截停查身。而教育界也被視為重點的政治對象。事實上，北京早就磨刀霍霍，要“殺一儆百”，港共找人祭旗，找替死鬼。根據以往個案，以往多數是涉及嚴重刑事罪行，例如非禮學生。該事件的嚴重程度完全不到被“釘牌”的嚴重程度。但當教育局受中共和【大公報】、689梁振英等越來越大的壓力時，教育局局長楊潤雄只好找舊個案跟進交差，而這名宣道小學的老師就這樣不幸被選中了。

**2.林鄭月娥重申“教育不可以是一個無掩雞籠”言論，言論打壓未來陸續有來。**
     整件事的發生是去年3月份反送中之前，香港的社會氛圍很不同，當時香港還是有言論自由，在課堂上沒有言論方面的壓力。據宣道小學前校長指出，是學生在課堂上提出相關問題，老師做了該份工作紙與學生在課堂上進行討論。可見，不是老師主動，而是學生提問題，老師解惑。這是合情合理的，並沒有所謂“刻意討論港獨 ”。

現在受害的不單是宣道小學老師，不單是教育界，今天港共可以註銷一個教師的牌，明天他可以註銷一個律師的牌，後天他可以註銷一個醫生的牌……所有的專業人士，隨時可能因為一句話而掉了工作。
     林鄭月娥重申“教育不可以是一個無掩雞籠”言論，需要有人把關，支持教育局 “嚴肅處理” 。這種言論打壓，是香港前所未有的。這一刀下去後，相信未來陸續有來。
