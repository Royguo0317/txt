- 10.1. 第一財經 2.7萬億結構性存款搬家銀行負債端壓力或影響流動性


近日，央行公佈了中資大、中小型銀行的人民幣信貸收支表，根據統計，截至8月末，我國商業銀行結構性存款規模為9.42萬億元左右，為今年以來首次降至10萬億元以下。此前，有報導稱，6月以來，部分銀行收到了來自監管的窗口指導，要求壓降結構性存款規模、規範結構性存款管理，在2020年9月30日前壓降至年初規模，並在年底前逐步壓降至年初規模的三分之二。

今年1月至4月，結構性存款規模實現連續四個月增長，並在4月首次突破12萬億元，其中，中小銀行是增長“主力軍”。數據顯示，截至4月末，中小銀行結構性規模為7.91萬億元，相比2019年年末增長約1.72萬億元，增幅是大型銀行的兩倍還多。

然而，結構性存款迅猛增長的勢頭未能持續，監管於6月份下發通知稱要嚴控規模增長。比如，北京銀保監局於6月12日下發《關於結構性存款業務風險提示的通知》，要求轄內銀行切實採取有力措施，逐月壓降結構性存款規模。

還需注意的是，伴隨著部分資金流出銀行體系，各大銀行也在尋求新的負債來源，其中，同業存單較受銀行青睞，這也使得同業存單市場近期出現“量價齊漲”，發行規模已經連續3個月回升。再從淨融資額來看，6月以後同業存單基本維持淨融資額，7月下旬至8月上旬每週都維持在高位，同業存單的存量更是達到新高。

業內的共識在於，四季度銀行仍將面臨較大的負債壓力，尤其是對於中小行而言。方正證券首席經濟學家顏色對第一財經表示，未來，部分股份行和城商行將進一步壓縮結構性存款規模，這就使得銀行負債端壓力加大，對流動性產生影響。 “這個影響可能還不小。”
