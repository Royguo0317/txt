***原文翻译：***

本周，拜登政府宣布了一项由七国集团支持的在全球发展中国家建设可持续基础设施项目的计划。在美国和中共国之间竞争日益激烈的背景下，该计划被广泛地认为是对中国“一带一路”(BRI)大型项目的直接挑战，该项目也旨在为发展中国家建设公路、港口和铁路。但是，被称为“B3W”（Build Back Better for World）的七国集团新计划中到底有什么内容，它的目的是什么？这里有一些需要思考的问题。 B3W与BRI有什么不同？在相当多的方面，B3W被誉为BRI的对立面，BRI（一带一路）为世界上一些长期被传统投资人忽视的高风险地区的基础设施项目筹集了资金。然而，它也因为资助有污染的煤厂，无视公平的招标程序，使弱势人群流离失所，并使项目国背负着对北京的巨额债务而受到批评。相比之下，B3W的推动者声称，它将为“价值驱动”的基础设施项目提供资金，这些项目符合《巴黎气候协定》，以透明方式授予，为社区提供长期利益，并提供发展中国家能够承受的条款。

抛开雄心勃勃的目标，筹资模式也非常不同。与直接由中共国或中共支持的公司提供资金的BRI不同，B3W将其希望寄托在美国资本市场和发展金融机构，如进出口银行。七国集团不会直接出资，但会为私营部门投资者的交易提供便利和担保，最终由他们来买单。 我们在这里讨论的是多少钱？这一点并不清楚。B3W官员说，就目前而言，他们打算筹集“数千亿美元”。但这仅仅是沧海一粟，考虑到七国集团本身，估计到2035年，发展中国家需要投资40万亿美元用于基础设施。这个数字也远远落后于2013年启动的“BRI”到2020年中期为其在100多个国家的近2600个项目争取到的3.7万亿美元。是否会有附带的政治条件？从技术上讲不会，但是，该计划是作为七国集团推动团结“志同道合”的民主国家以抗衡中共独裁的更广泛的一部分。因此可以认为，至少在最初，B3W将优先考虑那些民主国家，包括那些因为难以获得国际货币基金组织和其他国际金融机构贷款而转向北京的弱国。这可能会吸引像赞比亚这样的国家，他们知道美国债权人会希望看到他们的钱回来，但不会像中共那样要求获得铜矿。

偏袒民主国家的意外后果是，专制政权可能开始更加依赖中共来建设其发电厂、港口和道路。这方面的例子有政变后的缅甸和一些现金匮乏的中美洲国家，它们不希望自己的统治受到质疑。

美国国内政治是否会受到干扰？有可能。共同对抗中共国是民主党人和共和党人如今唯一能达成共识的事情。但拜登政府尚未澄清它将如何帮助B3W在美国资本市场筹集资金，以及将为境外投资提供哪些税收优惠（如果有的话）。共和党立法者则宁愿给美国公司（一些）钱来修复他们自己国家的破烂基础设施，而不是在亚洲建造一个新港口或在非洲建造一条铁路。尽管乔·拜登一直告诉他的七国集团朋友“美国回来了”，但他们还不相信。他们知道，B3W的命运在很大程度上取决于未来两个美国选举周期。如果共和党人在明年的中期选举中夺回众议院和参议院，该项目可能失去动力。如果是一个坚持唐纳德·川“美国优先”的共和党候选人，或是川普本人在2024年赢得总统职位，B3W可能注定要失败。
