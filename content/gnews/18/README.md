# **坚持了一段时间之后，拜登政府承认中共国正在进行的维吾尔族种族灭绝是“不可忽视的”**

白宫承认中国共产党正在对维吾尔族穆斯林进行种族灭绝，结束了数日来国务院不愿承认这种暴行正在进行的顽固抵抗。

[!\[\]()!\[\](https://spark.adobe.com/page/kva9rLaEnzwcg/images/9c456840-05d3-44aa-b67b-9d120e4da757.jpg?asset_id=7b9473f9-2188-436a-b515-dbc8829d71f8&amp;img_etag=%227ba51d2252265d5f5139337b24dc38e7%22&amp;size=2560)](https://spark.adobe.com/page/kva9rLaEnzwcg/images/9c456840-05d3-44aa-b67b-9d120e4da757.jpg?asset_id=7b9473f9-2188-436a-b515-dbc8829d71f8&amp;img_etag=%227ba51d2252265d5f5139337b24dc38e7%22&amp;size=1024)

拜登总统团队的让步肯定会产生重大的地缘政治影响，它重新调整了自己的观点，以保持与川普政府执政最后一天的观点相符，当时的国务卿迈克·彭佩奥在一份公报中说，他相信在中国西部的新疆，中共正在对大约100万至200万维吾尔人和其他宗教少数群体进行“种族灭绝”，他们被关押在该地区的拘留中心。

这份声明也是在“华盛顿智库创新战略与政策研究所”和“拉乌尔瓦伦贝格人权中心”发布了第一份此类法律分析报告的一天后给出的，该报告明确了中共国政府对待维吾尔人和其他穆斯林少数群体的行为，违反了联合国《防止及惩治灭绝种族罪公约》所禁止的每一项行为。

“拜登总统于2月10日和习主席进行了交谈，谈话的一部分内容明确表明了我们对新疆侵犯人权行为的强烈担忧，”白宫发言人周三晚间对《华盛顿观察家报》说，“曾经施加在维吾尔族人身上，并且一直在进行的这种反人类罪和种族灭绝罪不可忽视，（中共）必须要为此付出严重的代价。”

“我们的中共国政策取决于我们的核心力量来源，包括我们的价值观和与志同道合的伙伴和盟友协调工作的能力。我们与这些合作伙伴在双边和多边议题上合作，以决定我们如何共同向中共国施压，确保可以停止这些暴行。”白宫发言人补充说。

在提名为首席外交官的宣誓仪式上，国务卿安东尼·布林肯（Antony Blinken）承认，中共国犯下了种族灭绝罪，但是，就拜登政府本身是否对新疆当前的局势也持同样的立场，他们却表现得神神秘秘。每当记者要求布林肯团队确认他们是否承认“中共国正在进行种族灭绝”时，他们就刻意回避，说中共国对新疆维吾尔族人“实施过”种族灭绝。

也许措辞上的差异很小，但共和党人却认为，拜登政府对中共国太软弱。

“中国共产党正在对新疆的维吾尔族和其他少数民族进行种族灭绝，”德克萨斯州的参议员泰德·克鲁兹星期二对《华盛顿观察家报》说，“拜登政府急于与中共国勾兑，我深感忧虑的是，他们拒绝明确承认和谴责正在进行的种族灭绝是勾兑的一部分。”

[!\[\]()!\[\](https://spark.adobe.com/page/kva9rLaEnzwcg/images/4d34a515-ae5a-4f92-9731-17b78942c2c3.png?asset_id=99510816-7782-4db7-b89e-70d6090bed4f&amp;img_etag=%2216f426406d67e0bfb9ce4b12622fd417%22&amp;size=2560)](https://spark.adobe.com/page/kva9rLaEnzwcg/images/4d34a515-ae5a-4f92-9731-17b78942c2c3.png?asset_id=99510816-7782-4db7-b89e-70d6090bed4f&amp;img_etag=%2216f426406d67e0bfb9ce4b12622fd417%22&amp;size=1024)

尽管中共国官员否认了种族灭绝的指控，并为他们的“职业教育和培训中心”辩护，说这是为了打击恐怖主义所做的努力，但维吾尔族幸存者则称，他们因宗教信仰而受到看守们的侮辱、殴打、虐待和强奸。

当布林肯和拜登的国家安全顾问杰克·沙利文(Jake Sullivan)下周飞往阿拉斯加州安克雷奇(Anchorage)与他们的中共国同行会面时，这个事情将会被提及。

布林肯在周三的众议院外交事务委员会听证会上说：“重要的是要广泛传播，确保其他国家也在做同样的事情。如果中共国听到的不仅仅是我们的指责，更多的是来自世界各国的指责，我们就越有可能获得一些机会（阻止中共继续进行种族灭绝）。”

布林肯在随后的声明中说：“我们的观点很明确，我的观点也很明确，我认为这是种族灭绝，并存在其他严重侵犯人权的行为。”他补充说，如果中共国没有什么可以隐瞒的，那么就应该允许国际社会进入新疆。
