美日印澳四方安全对话Quad在3月12日举办首次峰会，强调合作对抗中共军事霸权后，法国驻印度大使馆31日发表声明，法国与印度海军长期军演背景下，法国海军直升机航舰托纳尔号与苏尔库夫号巡防舰组成的圣女贞德Jeanne d’Arc战斗群，3月30日到4月1日在印度克勒拉省柯枝港停泊，之后将航向孟加拉国湾，参加由法国领导的拉佩鲁兹La Perouse联合军演。
![]()![](https://gnews.org/wp-content/uploads/2021/04/quadnato-400x204.jpg)圖片來源美聯社
中共解放军长期扩张南海礁岛领土，2012年中共成功从菲律宾取得黄岩岛的案例，近日西菲律宾海国家特遣部队声明，在南沙群岛九章群礁的牛轭礁周边，集结220艘大型渔船。早在2012 年在西沙、中沙、南沙等，由 280 多个岛礁、沙洲组成的礁岛海域，中共就自行宣布该区隶属于海南省成立的三沙市管辖范围，尽管越南政府强烈抗议中共严重侵犯越南主权，却无法阻止中共藉由入侵宣示动作来逐步扩张在南海的侵略，根据2019年世界日报报导，在 2014年到 2019年的 5 年间，中共在南沙群岛 7 个礁岛兴建造超过 1千万平方公尺的军事用地，其中三座最大的永暑岛、美济岛、渚碧岛，在岛上已建造可以让轰炸机、战斗机起降的跑道，包括各种雷达侦测等军事设施，不但侵犯邻国经济海域，更直接占领实质领土、渔业资源等，完全不在乎邻国强烈反弹，更加速扩张进程造成区域冲突，导致南海礁岛区域局势失去平衡。

近日中共享海警法来对钓鱼台列屿(尖阁群岛)、印太地区邻国海域的侵犯冲突升高，此次特别声明指出，拉佩鲁兹军演将从4月7日起将加入美国和印度、日本、澳洲一起的5国海军联合演习，形成亚洲北约盟友的合作关系，并藉由联合作战演练来确保印太地区安定，促进海上国家伙伴关系广泛的行动，未来英国和德国也将参加南海军演。本文仅代表个人立场
