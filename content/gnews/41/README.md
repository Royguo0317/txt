2020年12月30日，习近平和德国总理默克尔、法国总统马克龙、欧盟委员会主席和理事会主席参与视频会议。这个视频会议中说到达成这个欧中投资协定，这个过程耗时七年。中欧双方对于中欧投资协议达成共识。根据欧盟的公告，中共承诺给与欧盟投资者“前所未有”的市场准入待遇，同时欧盟在中国的经贸活动的安全性以及可预见性将得到改善。接下来协议将进入签署和审批程序；协议签署两年内双方还要完成投资保护问题的谈判。
![]()![](https://gnews.org/wp-content/uploads/2021/01/Unknown-3.jpeg)中国国家主席习近平、德国总理默克尔和法国总统马克龙视频连线，图源：路透社


欧盟在中国的投资企业很多，所以德国总理默克尔和法国总统马克龙在内的欧盟领导人都十分重视中欧投资协议。当然为了促使与中共达成投资协议， 欧盟的领导人刻意地忽略了其在香港以及新疆侵犯人权的行为。对于中共病毒起源问题更是选择保持沉默。

**冠博士评论：**这个协定中中共对欧盟做了很多承诺，但是这个协定被美国白宫副国家安全顾问博明先生狠批说：“欧盟委员会这件事是不顾中共侵犯人权的行为，急于与北京合作是扯下了一块”遮羞布“。而另外一个信息是这个协议虽然说所谓的达成了，但是仍然需要欧盟成员国的政府正式批准，以及欧洲议会的投票表决，所以说距离协议正式生效还是有很长的路的。所以说这个时候习近平高调来说这个协议更像是一种政治宣誓，来表明自己中共对于欧洲还是有一定的影响力。
