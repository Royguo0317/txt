**1.军心不稳，习近平急于给军队大幅度加薪意味着什么？**

习近平面对逼宫局势、军心不稳，重点对年轻军官和在边疆、东海、南海服役的军人大幅加薪40%。

路德解读在中共军队体系中，只有少将级别不会退役，大校（两杠四星）政师级别，退役后可做地方军区司令和政委等职。而上校（两杠三星）参谋长级别基本是炮灰，退役到地方将直接降到副处级别。

所以上校到大校晋升非常难必须得有背景。此次加薪就是买通为晋升还在努力做事的上校，以此稳定军心。

调整后上校可以月薪2万，将打破现有公务员薪资体系。毛时期文革和三年灾害期间只以军队为重。邓时期大幅裁军、军队薪酬待遇一般，江时代彻底结束军队参与地方建设，将军队与地方油水彻底分割。习此次加薪，意味着先军政策回归。增强社会对军队地位的认可度后，刺激年轻人踊跃去做炮灰。
