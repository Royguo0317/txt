## 中共病毒导致感染者神经系统损伤

翻译：武松Sam
![]()![](https://gnews.org/wp-content/uploads/2021/04/076-e1619638616939.png)图片来源：网络
截至目前为止，中共病毒造成美国人死亡的人数已经超过了第一次世界大战、越南战争和朝鲜战争之和。死亡原因大多数是由于中共病毒引起的肺部并发症造成的，但是，人们也越来越认识到该病毒还会攻击神经系统。芝加哥一家大型医学中心的医生发现，超过40％的中共病毒患者一开始就表现出神经系统受损，而超过30％的患者认知能力受损。在某些情况下神经系统受损是毁灭性的，甚至可能导致死亡。同时根据最新的研究表明，全球2700万以上感染中共病毒的幸存者会有神经系统受损的严重后果，尤其令人不安的是，越来越多的证据表明，许多幸存者可能会出现脑部损伤，从而引起认知、行为和心理问题。

研究发现中共病毒会引起脑炎从而直接对大脑造成损害，在一项针对十二名脑炎患者的研究中发现，只有一名患者完全康复，十名部分恢复，一名死亡，许多中共病毒患者会患有中风等症状。加拿大医生发现，70岁以上的人罹患与中共病毒感染相关的中风的风险特别高，但即使是年轻的人，与典型的流感病毒相比，患中共病毒的中风的可能性也要高7倍。芬兰中共病毒患者的尸检数据表明，造成脑损伤的另一个主要原因是氧气不足，几名接受尸检的患者在其感染中共病毒之前并未显示出任何脑部损伤迹象，但感染该病毒之后均患有脑部损伤，其中一名患者失去了味觉，另外两名患者出现呼吸困难的症状。

约翰·霍普金斯大学和哈佛医学院的医生进行的一项新研究发现，死于中共病毒感染的患者的脑部毛细血管中发现称为巨核细胞的大细胞，巨核细胞会产生血小板(血小板是构成人体凝血系统的组成部分)，这些神经病理学家以前从未见巨核细胞存在于大脑中，医学文献也从未报道过这种情况，这些巨核细胞在脑部出现很可能就是中共病毒患者中风的原因。研究还发现感染中共病毒中的三分之一的人表现出极大的认知障碍，对这些人进行的神经心理学测试的表现可与中度外伤性脑损伤的表现几乎一致。在日常生活中，这种对记忆力、注意力和执行力的认知影响可能导致与别人日常交流的障碍。

这些研究得出一个可怕的结论：中共病毒会导致感染者脑部损伤，尤其是70岁以上的人。尽管许多感染者恢复以后可以像往常一样的正常生活，但仍有许多人现在或以后可能会遇到日常生活中的难题。由德国和美国医生最近发表的一篇论文得出的结论是，病毒直接导致全身性炎症、中风以及对身体器官（如肺和肝）损害的叠加影响，甚至可能使中共病毒幸存者将来处于罹患阿尔茨海默氏病的高风险中。

**评：**中共病毒是在中共最高层领导下，由中共军方、病毒实验室以及邪恶科学家一起经过多年的实验，人为造出来的超限战生物武器，该生物武器对于人体造成不可预估、不可逆的损伤，甚至导致死亡。现今已经导致了数千万人感染，数百万人死亡的严重后果。要想尽快了解该病毒原理、尽快拿到解药，只有尽快消灭中共一条路可以走。留给人类的时间不多了。
