![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/09070335/portugal-4675077_640-4.jpg)

唐宁街将西班牙排除在12个目的地的绿色名单之外，从5月17日起，英国人将可以前往这些目的地休闲旅游，回国后无需接受检疫。目前，在被选为安全国家的短名单中，有葡萄牙、以色列和澳大利亚。这对西班牙旅游业来说是个坏消息，至少让其希望拯救加那利群岛旅游业的计划落空。通常，英国当局将每三周审查一次信息。评估目标国家的关键因素是接种疫苗的人口比例、感染率、令人担忧的变异体的流行率，以及该国获得可靠科学数据和基因组测序的情况。根据这些数据，已经建立了一个 “交通灯系统”。根据该系统，前往 “绿名单 “目的地的乘客在出发和返回时都必须接受检测，但返回英国时不必接受隔离。

列入 “红名单 “的旅行者在回国后，将不得不返回酒店进行隔离（费用约为2000欧元），并接受至少三次诊断性测试。英国游客一直是西班牙的主要客户之一。2019年，来自英国的1800万游客占到西班牙接待游客总数的23%。他们花了大约18000万欧元。

面对中共病毒病例的增加，英国政府在今年早些时候暂停了海外旅行。在加强边境管制同时，还发现了冠状病毒的新变种，这给科学界敲响了警钟，目前的疫苗可能对它们不太有效。因此，鲍里斯-约翰逊这么晚才确认何时恢复国际旅行，因为他面临巨大的压力。

西班牙政府不同意联合王国的决定。欧洲新闻社咨询的外交人士认为，英国的评估对 “真实 “的流行病情况 “没有提供足够准确的描述”，因为它是 “根据国家平均数 “做出的。西班牙希望评估可以具体到某个地区，比如加那利群岛，那至少可以挽救那里的旅游业。

**马上就要到6月了，英国的这个决定对西班牙就像一次地震，打破了西班牙希望能在今年夏天，依靠旅游业作为恢复经济的重要组成部分的想法。不仅仅是经济，对一些抗疫的政策也可能产生重要影响。**

**5月9日，西班牙结束了已经实行了1年的紧急状态法，改为各自治区根据实际情况自行制定限制措施。一些旅游业占据比重比较大的自治区，已经准备放开餐饮业和酒店业了。比如马德里，餐馆将可以营业到00：00。**

**紧急状态法本身是桑切斯政府的一把武器，在此法律框架下，他可以实现限制，可以关闭某个行业，可以集中国家资源，可以调用军队等等，因此，也饱受其他政党的攻击和批评。其中一条，就是紧急状态法，会影响游客的到来。**

**另外一条评估政策是接种率，这又是一把疫情武器，接种率有没有达到预期，也是各个政党之间攻击最常用的武器。**

**而当地的旅游业从业者，也无法承受如此巨大的压力，准备采取一些极端的行动。例如，[加那利群岛的酒店业者发动了游行，声称“不接受限制措施，周一我们将正常营业”](https://www.abc.es/sociedad/abci-sublevacion-hosteleros-baleares-contra-restricciones-abriremos-lunes-normalidad-202105080152_noticia.html)。**
