# **世卫组织文件称：中共国在疫情最初几个月“几乎没有”对疫情来源进行追查**

[!\[\]()!\[\](https://spark.adobe.com/page/LsApRFNmAHaX7/images/a06eea54-2aec-4599-be6f-56c1b0a5c472.png?asset_id=5c5d734a-c764-4e54-8a40-e980d7b59d48&amp;img_etag=%22e80bdba12f5e848cfc4b096d54c225c5%22&amp;size=2560)](https://spark.adobe.com/page/LsApRFNmAHaX7/images/a06eea54-2aec-4599-be6f-56c1b0a5c472.png?asset_id=5c5d734a-c764-4e54-8a40-e980d7b59d48&amp;img_etag=%22e80bdba12f5e848cfc4b096d54c225c5%22&amp;size=1024)一名警务人员在华南海鲜市场外站岗。这里是武汉市海鲜中心，病毒首先在此处被发现。图片来源：赫克托·雷塔马尔(Héctor Retamal)，发表于法新社/盖帝图片社

《卫报》看到的世卫组织内部文件显示，新冠肺炎于武汉爆发后的最初8个月内，中共国官员在流行病学起源方面的调查“做得很少”。

这份日期为2020年8月10日的世卫组织内部旅行报告摘要还说，该团队与中共国同行的会面是寻找病毒起源任务的一部分。当时，该团队获得的新消息极少，并且在与中共国官员的广泛讨论中，并未获得任何文件或书面数据。

这份去年夏天的报告撰写之时，新冠肺炎已在全球感染了2000万人。这份报告提供了新的见解，表明世卫组织的科学家研究中共国疫情的最初努力似乎受到了阻碍。

而在此之前，拜登政府刚刚发表了明确的声明，表示了对与中共国合作研究疾病的担忧，并指出世卫组织必须保持高标准，且必须维护其信誉。

美国国家安全顾问杰克·苏利文(Jake Sullivan)也呼吁中共国公开疫情最初爆发以来的所有数据，并要求包括中共国在内的所有国家，采取“透明和稳健”的流程预防和应对突发卫生事件。

《卫报》看到的2020年8月世卫组织内部文件提供了一些线索，揭示了可能导致美国失望的原因。

[!\[\]()!\[\](https://spark.adobe.com/page/LsApRFNmAHaX7/images/92b7b2cf-8298-4b92-984c-926107c8169b.png?asset_id=c54ee1ba-c487-4ead-8a6d-b4c595ebaf79&amp;img_etag=%22d4fbfbbdd478371e05f445129fab2068%22&amp;size=2560)](https://spark.adobe.com/page/LsApRFNmAHaX7/images/92b7b2cf-8298-4b92-984c-926107c8169b.png?asset_id=c54ee1ba-c487-4ead-8a6d-b4c595ebaf79&amp;img_etag=%22d4fbfbbdd478371e05f445129fab2068%22&amp;size=1024)

这份长达两页的内部旅行报告，是世卫组织项目经理兼任务负责人彼得·本·恩巴雷克(Peter Ben Embarek)对2020年7月10日至8月3日期间中共国之行的总结。这次旅行被称为世卫组织任务先遣队，目标是研究新冠肺炎病毒，以及“审查迄今为止有关病毒来源的工作”。

旅行报告在摘要中说，代表团首先进行了为期两周的隔离，随后和相关部门举行了为期10天的面对面会谈。这些部门包括中共国国家卫健委、国家市场监管总局、农业农村部、以及包括武汉病毒研究所在内的其他机构。

报告称：“经过中共国同行的介绍和与他们的广泛讨论，自2020年1月以来，围绕武汉市展开的流行病学调查工作似乎很少。与2020年1月紧急委员会会议相比，此次旅行仅口头上获得了一点点更多的细节数据。这次会谈没有展示相关的显示文稿，也没有分享任何文件。”

世卫组织发言人拒绝就该“内部文件”发表评论。中共国驻华盛顿大使馆未回应置评请求。

当《科学》杂志记者在8月21日新闻发布会上询问世卫组织官员有关2020年7月至8月的任务时，传染病专家、新冠肺炎技术负责人玛丽亚·范·科霍夫博士(Dr Maria Van Kerkhove)表示，该团队最近已从中共国返回。在那里，他们向中共国同行“学习”“一直在进行”的工作。

世卫组织突发公共卫生事件规划执行主任迈克尔·瑞安博士(Dr Michael Ryan)随后补充说：“任务的推进十分重要，但一些初步研究也需要进行。我们在中共国的同事与先遣队深入讨论了这些问题。我们希望这些研究能够尽快开始。”

最近（上月）前往中共国研究病毒起源的旅程之后，关于中共国研究病毒起源的合作又出现了进一步的问题。调查小组成员、澳大利亚传染病专家多米尼克·德威尔(Dominic Dwyer)最近告诉记者，世卫组织在2021年1月的访问中，曾要求中共国同行提供原始患者数据，但只得到了一份摘要。

德威尔告诉路透社，分享匿名化的原始数据是疫情调查的标准做法。他说，原始数据对于理解新冠肺炎的努力尤为重要。因为174例初始病例中，仅一半的病例曾暴露于首次发现病毒的、已被关闭的市场。

德威尔说：“这就是为什么我们坚持要求原始数据的原因。至于为什么没有拿到原始数据，我无法评论。”

世卫组织在向《卫报》发表的声明中说，世卫组织“从最初开始”一直在强调必须了解病毒来源，并在2020年全年与中共国讨论研究和分享信息的必要性。

声明写道：“2020年7月，世卫组织应中共国政府邀请，派出一个先遣工作组前往中共国，为国际科学家团队的工作做准备。2020年秋季开始，中共国与国际科学家组成的联合团队举行了若干次网络会议。”

世卫组织发言人补充说：“他们于2021年1月至2月访问了武汉。在疫情爆发早期，挽救生命、了解疫情和抑制传播是绝对优先事项。但我们也认为，了解任何疫情来源的工作也应尽早开始，因为一些特定的线索在早期更易被发现。”

发言人说，今后，对疾病起源的研究应与抑制病毒和挽救生命的紧急措施同步进行。
