新的基民盟领导人将于1月22日公布

现在终于确定了：基民盟要在1月22日之前选出新主席。因为新冠大流行，代表们要进行电脑投票。

新一届基民盟联邦主席将于1月22日最终确定。这是秘书长保罗-齐米亚克（Paul Ziemiak）在柏林宣布的。在1月16日的联邦党代会上进行数字预选后，将进行邮寄投票。

“1月22日将进行公开计票，结果将由选举委员会公布。”

——秘书长保罗-齐米亚克（Paul Ziemiak）

齐米亚克强调，这一程序符合目前的法律状况。

纯数字化党代会

根据基民盟联邦执行委员会周一的决定，因为新冠大流行，联邦党代会将以纯数字化方式举行。齐米亚克说，这是 “德国政党的伟大创举”。没有其他政党召开过纯数字化的党代会选举。基民盟表现出了责任感，因为它放弃了要在新冠大流行中举行的党员大会。同时，它也在展示自己的行动能力。

关于推迟或出席的讨论

此前，由于党内主席团成员的感染数字持续走高，曾讨论过再次延期。然而，在执行委员会的决定投票中，数字化党代会获得了绝大多数的支持。根据几位与会者提供的信息，石勒苏益格-荷尔斯泰因州总理丹尼尔-贡特和北莱茵-威斯特法伦州内政部长赫伯特-鲁尔对该提案投了反对票。

巴登-符腾堡州文化部长Monika Grütters和Monica Wüllner投了弃权票。

1月16日选举党主席

根据来自党内的消息，在线党代会有1001名代表参加，最早将于1月15日开始，进行初步发言、规范，并可能进行第一次选举。1月16日，除党主席外，还要选举其副手及其他领导机构成员。

除了前联盟派别领导人弗里德里希-梅尔茨（Friedrich Merz）外，北威州州长阿明-拉舍（Armin Laschet）和外交事务专家诺伯特-罗特根（Norbert Röttgen）也希望成为基民盟主席。

据与会者介绍，拉舍特（Laschet）没有作为当事人参加联邦执行委员会的投票。因此，他没有在讨论中发言。

在主席团会议上，他曾明确表示，他将接受这一决定。由于新冠的原因，党魁安娜格雷特-克朗普-卡伦鲍尔（Annegret Kramp-Karrenbauer）在2月宣布辞职后，人事问题就从春天拖到现在。

与一周前的星期二相比，新感染的数值增加了​​近400例。

柏林： 感染冠状病毒的死亡人数和新感染人数仍处于高位。德国卫生当局周二上午宣布，在一天之内向罗伯特-科赫研究所（RKI）报告了14,432例新病例和500例死亡病例。在前一周的星期二，报告了14, 054例新感染病例和423例死亡病例。上周五已达到报告病例29875例、死亡598例的高位。周二的500人死亡人数是自疫情开始以来第三高的水平。

最近，每日死亡人数呈上升趋势，这是在新感染病例急剧上升后的预期。死于或确诊感染Sars-CoV-2的总人数上升到22,475人。

7天的发病率——7天内报告的每10万居民新感染人数——对评估形势至关重要，略有下降。周一最高为176.4，这一数值在周二降至173.7。

自疫情开始以来，RKI已统计出德国有1,351,510例已证实的Sars-CoV-2感染病例（截至12月15日00时）。据估计，此后约有1,003,300人康复。

RKI简介：罗伯特·科赫研究所（德语：Robert Koch Institut 简称RKI）是德国联邦政府机构和研究机构，负责疾病控制和预防，位于柏林和韦尼格罗德。为联邦高级机构，隶属于联邦卫生部。该研究所成立于1891年，以创始人、现代细菌学的创始人和诺贝尔奖获得者罗伯特·科赫的名字命名。
