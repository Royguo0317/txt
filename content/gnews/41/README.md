**加国驻港领事：国安法损一国两制框架****不少在港国际企业检视应急计划**

将于本月中卸任离港的加拿大驻香港及澳门总领事南杰瑞（Jeff Nankivell）称，“港版国安法”损害“一国两制”框架，并透露不少在港的国际企业，正因应《国安法》实施检视营运，包括研究转移部份客户资料，并检视一旦需要离港的应急计划。南杰瑞又指，非常关注港府修订本港选举制度，认为做法削弱了港人在制度内的声音，亦影响市民参与政治的权利；对于立法会的直选议席比例减少，他更形容是“不幸”。

去年《国安法》生效后，南杰瑞坦言在港的外交工作出现变化，部份政党与非政府组织拒绝再与外国领事对话，比例不高但显著，他形容非常可惜。他形容，自去年夏季开始，领事馆的工作受到香港部份政客和中共中央控制的传媒发动的“恐吓运动”所影响。

加拿大早前为香港年轻人提供特别移民计划，其中工作签证计划（open work permit）2月起接受申请后，同月即接获逾500份申请。南杰瑞称，该计划的申请人数正上升，形容计划颇受欢迎，而加拿大正准备推出另外两条移民路线，分别供正在加国工作或读书的港人，让他们直接成为永久居民，稍后会公布详情。他预计，随着疫情的旅游限制放缓，将有新一批今夏毕业的大学生申请加拿大早前为香港年轻人提供的特别移民计划。

朱敏健表示，外佣或因不同理由而无法打针，例如健康问题，或签约前未能预约打针时间。倘若政府一刀切、强制外佣续约或来港时打针，相关措施或易受法律挑战。他建议，政府可提供打针以外的替代方案，例如要求外佣接受频率更密的检测，或做好个人防疫措施。

东涌映湾园、东区康怡花园，以及荃湾荃威花园部分居民突然要离开家园，被安排入住隔离营。原本要被隔离21天，及后政府突然改变隔离政策，他们又可提早“重获自由”。康怡和荃威花园的居民，可以在今日（9日）母亲节提早“出营”。不过，回想竹篙湾隔离营的生活，却令居民叫苦连天；居住环境狭小、要求物资回覆迟缓、甚至日前爆出怀疑食物中毒事件。有居民向记者透露，有被隔离的外籍男子，怀疑受不了叫天不应叫地不闻的待遇，违规走出门口，高叫“Is this the way you want to treat us？”有康怡居民从竹篙湾离开，收到取消检疫令的证明时发现，不只身份证号码再次出错，连其英文名字都串错。
