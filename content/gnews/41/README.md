梁颂恒今天在其Facebook个人专页上帖文：

港共今天又立苛法，于选举中提倡投白票废票唔投票者皆犯法，最好刑罚判监三年。

我个人就最憎人犯法，垃圾话留着我来说。

假如政权再搞一次选举的话，我提议所有香港人投废票。

![]()![](https://gnews.org/wp-content/uploads/2021/04/WhatsApp-Image-2021-04-14-at-17.15.27.jpeg)

作用有二：

（一）珍惜每一次跟同路人展示大家的心还在的机会

今日的香港抗争成本正以几何级数增加，在未有新的策略出现前，组织一个“示威”是困难的。在这段黑暗时期中我们都不能避免地会感到无力气馁。

我自己最迷茫，同时也是零生产力的时候，最能将我从无力感的漩涡拉回现实的，往往是其他同伴正在做的事——不论是为香港人承受囹圄之苦、流亡，或是继续做在他们身位上能做的事情。

人的习性正是这样，每当看见有同伴前行一步，自然也会想自己一并向前走一步。

难得有机会跟战友们展示“我们还在”，能力范围许可下又何乐而不为？

（二）让世界看见香港人

我们在一九年就已经明白，香港不可能永远是世界聚焦的中心。

世界每一天都有事情发生，比香港惨十倍百倍皆有之，他们皆需要世界的关注。然而人能关注的事也是有限的，在短时间内香港不太可能发起一波大型抗争的这段时间，怎样才可以令自由世界国家不会忘记支援香港呢？

想通过大型抗争引起关注是好的，但低成本又可以让人见到，其实也不坏呀？

梁颂恒简介：

2016年，梁颂恒代表青年新政参选立法会选举，结果，梁颂恒在新界东地区直选顺利当选。

2016年11月15号，高等法院法官区庆祥裁定，梁颂恒同游蕙禎祯两人的宣誓无效，议员资格于2016年10月12日起丧失。

2018年9月，梁颂恒加入香港民族阵线并担任发言人。

2020年11月，梁颂恒抵达美国，流亡海外。
