# 推特将迈克尔·弗林将军和西德尼·鲍威尔账号永久拉黑

[!\[\]()!\[\](https://spark.adobe.com/page/gyTOy5XhWhALI/images/e5b8efb7-b065-4545-9f93-7385449aa6bc.png?asset_id=1618a0f2-6396-4a2d-90bc-6cc91e4369b1&amp;img_etag=%2261b11c28377bee0cde9ec86a91f2e4d3%22&amp;size=2560)](https://spark.adobe.com/page/gyTOy5XhWhALI/images/e5b8efb7-b065-4545-9f93-7385449aa6bc.png?asset_id=1618a0f2-6396-4a2d-90bc-6cc91e4369b1&amp;img_etag=%2261b11c28377bee0cde9ec86a91f2e4d3%22&amp;size=1024)美联社照片/ 曼努埃尔·巴尔斯·塞内塔 / 阿勒姆·博卡里 2021年1月8日

退休的迈克尔·弗林中将和他的律师西德尼·鲍威尔的推特帐户今天都被列入黑名单。

据《国会山报》称，这些禁令是永久性的。

推特发言人告诉《国会山报》，由于与QAnon运动有关，因此这些帐户被禁止了。

发言人说：“我们一直有明确的态度，我们将对可能导致线下伤害的行为采取强有力的行动，鉴于在未来几天内，围绕此类行为再次出现暴力的可能性，我们将永久禁止专门致力于分享匿名者Q（QAnon）内容的账户。”

这件事情发生在针对川普总统和政府其他知名盟友进行了一轮前所未有的审查之际。

Facebook和Instagram无限期地封禁了川普总统的账号，引来了甚至是非政治和自由主义人士的广泛关注，例如格伦·格林瓦尔德（Glenn Greenwald）和名人艾米丽·拉塔科夫斯基（Emily Ratajkowski）。

包括前第一夫人米歇尔·奥巴马（Michelle Obama）在内的民主党人一直在敦促大型科技公司将禁令永久化。

弗林（Flynn）之前曾在川普政府成立之初短暂担任过国家安全顾问，但在被指控向联邦调查局（FBI）做了虚假陈述后，为了自证清白，他经历了多年的法律斗争。

司法部提出动议，驳回所有对这位获得过勋章的前中将的联邦指控后，弗林在2020年11月被川普总统赦免。

过去几个月，弗林和鲍威尔都与川普总统见过面，讨论选举结果。

作为Twitter打击QAnon的一部分，互联网留言板8Kun的前管理员罗恩·沃特金斯（Ron Watkins）也被封禁。
