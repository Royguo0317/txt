**中共早在新冠疫情爆发前在全球购买大量医疗级口罩、防护衣和相关物资**

中共早在2020年Covid-19爆发前几个月就在全球市场上购买大量医疗级口罩、防护衣、和各种相关物资，无论是否在美国制造的用品，都被买到断货，在第一时间掩盖病毒散播，让大量病毒透过旅客飞往世界各国机场，并和WHO合作隐瞒病毒人传人的事实，刻意造成全球染疫病使毒快速扩散，同时石正丽造假的RaTG13与云南矿洞虚构病毒和中共启动大外宣使病毒起源于自然界。

**习近平2020年2月严格控管病毒起源**

同时习近平2020年2月发布重要指示，要求疾病预防控制中心实验室等其他工作人员，不得与外部机构或个人分享新冠状病毒相关的任何数据、标本或研究成果，必须经过习近平审查批准才能公布。根据美联社获得内部情报并在2020年12月31日发布调查显示，中共向研究新冠病毒起源的南部解放军科学家发放数十万美元的经费。并监视调查结果，中共内部泄密罕见的数十页的内部文件，证实了高层授意严控病毒起源调查。
