## 为防中共美国号召网络清洁活动
` Himalaya Wenya 喜馬拉雅的文雅` [轉載自GNews](https://gnews.org/zh-hans/662561/)

美国政府网于12月18日清晨发推，以传播官网发布的“我们呼吁所有热爱自由的国家和企业加入清洁网络”的号召。

推文称, @SecPompeo “清洁网络计划是川普政府保护国家资产，包括公民的隐私和公司最敏感的信息，以免落入中国共产党之手。”



> .[@SecPompeo](https://twitter.com/SecPompeo?ref_src=twsrc%5Etfw): "We’ve urged countries to become CLEAN COUNTRIES so that their citizens’ private information doesn’t end up in the hand of the Chinese Communist Party.” Learn more about the Clean Network at [https://t.co/SXzHsiErNi](https://t.co/SXzHsiErNi). [pic.twitter.com/Vr9fXWvYbY](https://t.co/Vr9fXWvYbY)
> — Department of State (@StateDept) [December 18, 2020](https://twitter.com/StateDept/status/1339886025661960193?ref_src=twsrc%5Etfw)



该号召主要内容如下。

清白网络解决了专制恶意行为者对数据隐私、安全、人权和原则性合作对自由世界的长期威胁。洁净网络植根于国际公认的数字信任标准。它代表了一个多年的、全政府的、持久的战略的执行，它建立在一个值得信赖的合作伙伴联盟的基础上，并基于快速变化的技术和全球市场的经济。包括以下几个方面。

数字信任标准

2020年5月，应国务院的要求，战略与国际研究中心（CSIS）召集了来自亚洲、欧洲和美国公司及研究中心的25名专家，制定评估电信设备供应商可信度的标准。这些《电信网络和服务的安全和信任标准》是对《布拉格提案》和欧盟5G工具箱工作的补充。它们为政府和网络所有者或运营商提供了一个额外的工具，用于确定可信度和安全性。

布拉格建议(Prague Proposals)

2019年5月，来自全球30多个国家的政府官员与来自欧盟、北大西洋公约组织和工业界的代表一起参加了关于重要的国家安全、经济和商业考虑因素的讨论，这些因素必须成为每个国家评估5G供应商的一部分。捷克会议主席发布的5G安全布拉格提案，为各国在设计、建设和管理5G基础设施时提供了一套建议和原则。

清洁网络工作路线

2020年8月5日，国务卿蓬佩奥宣布扩大清洁网络计划。

清洁载体

确保中华人民共和国的运营商不与美国的电信网络连接。这些公司对美国国家安全构成威胁，不应提供往返美国的国际电信服务。

洁净店

从美国移动应用商店中删除不受信任的应用程序。中国的应用程序威胁我们的隐私，扩散病毒，审查内容，并传播宣传和虚假信息。美国人最敏感的个人和商业信息必须在手机上得到保护，以免被中共利用和窃取。

清洁应用程序

为了防止不受信任的中国智能手机制造商在其应用商店中预装或以其他方式提供下载受信任的应用。华为作为中国监控国家的一个分支，正在利用领先的美国和外国公司的创新和声誉进行交易。这些公司应该从华为的应用商店中删除他们的应用程序，以确保他们没有与人权侵犯者合作。

清洁云

为了防止美国公民最敏感的个人信息和我们企业最宝贵的知识产权，包括COVID-19疫苗研究，被存储和处理在基于云的系统上，通过阿里巴巴、百度、中国移动、中国电信和腾讯等公司被我们的外国对手访问。

清洁电缆

确保连接我国与全球互联网的海底电缆不被中国超大规模地颠覆，用于收集情报。我们也会与外国伙伴合作，确保世界各地的海底电缆不会同样受到破坏。

洁净之路

2020年4月29日，美国国务卿蓬佩奥宣布，美国国务院将开始要求所有进出美国外交设施的5G网络流量必须使用清洁路径。

5G清洁路径是指不使用华为、中兴等不受信任的IT厂商的任何传输、控制、计算、存储设备的端到端通信路径，要求符合中共的指示。

5G清洁路径体现了最高的安全标准，防止不受信任的高风险厂商破坏、操纵或拒绝向私人公民、金融机构或关键基础设施提供服务。

此外，不受信任的IT供应商将无法访问美国国务院系统。我们将遵循法律条文，确保我们有一个干净的路径，让所有5G网络流量进入我们所有的设施。期。我们将继续尽一切努力，保证我们的关键数据和网络安全，不受中共的侵害。

5G清洁国家和清洁电信公司

许多国家都选择在其5G网络中只允许可信的供应商。例如，英国、捷克共和国、波兰、瑞典、爱沙尼亚、罗马尼亚、丹麦和拉脱维亚。希腊已经同意使用爱立信而不是华为来开发其5G基础设施。

全球一些最大的电信公司也正在成为 “清洁电信公司”。法国的Orange、印度的Jio、澳大利亚的Telstra、韩国的SK和KT、日本的NTT、英国的O2都拒绝与华为等中共监控国家的工具做生意。加拿大三大电信公司决定与爱立信、诺基亚、三星合作，因为舆论绝大多数都反对让华为建设加拿大的5G网络。

当前中共日益猖獗的网络攻击美国人为大选正义之战的硝烟中，变得更加肆无忌惮。澳洲是中共毒手深入伸向的重灾区之一。我们尚未看到澳洲加入这个网络清洁计划，但是可喜的是Telstra 和 Optus 两大电信运营商已经榜上有名。与此同时，美国作为中共网络骇客的最主要和最严重的受害方，现在已经脱下含蓄的外衣，直接赤膊上阵，点名中共，是侵害公民的隐私和公司最敏感的信息、进行恶意入侵的盗贼，没有之一。美国反共的号角声声，灭共的行动将在多方位多维度展开。与此同时，中共国的电信运营商、应用程序、手机开发商、网络云公司等均被屏蔽在美国合作商之外，也将被排除至参与美国网络清洁计划的各国合作运营商业务之外。中共对中国尤以民营企业和人民智力财富的伤害是把网络骇客、偷盗知识产权等恶行与中国企业直接挂钩，将贻害无穷，所以为了不为中共背锅，中国同胞必须与中共划清界限，推翻中共，在新中国联邦的协助下，我们才能被世界重新接受。
