大地复苏，满地青草，又到六四了。我在海外这么多年，认识了民主自由，看清了中共的面目，心态的转变从以前的困惑到如今的坚定。今天回想过去，有感而发。

1989 年6月，当时是青少年的我目睹了本地的大学生游行声援活动，他们受到了民众的大力支持（鼓掌助威，送饮料，送面包），游行学生高喊口号，气势高涨，使我想起了银幕上看过的五四青年游行示威抗议暴政的场景。本人也和小伙伴们进入了游行队伍，感受到了前所未有的兴奋。当时我们看到了贴在电灯柱子上的标语以及死伤学生的传真照片，很气愤和悲伤。我大概游行了几个小时就溜回家了。我父母很紧张，教训我不能再去游行。现在看来，他们当时是被中共在几十年来的各种政治运动的秋后算账吓怕了。我们家是地富反坏右之一，属于被专政的对象，爸爸连少先队员也不是，我妈妈战战兢兢地给我说了家里以前的事。从那次游行以后，我就对中共反感了，我对中共所说的各种官方数字化的口号，心里是抵触的，什么 “四有新人” 、“红花少年” 、“五讲四美”、 “六不六要”、 “三八红旗手” 、“新长征突击手” 等等。

事后，我们小伙伴之间都避免谈到六四活动，记得当时老师也没有要求同学们互相揭发和自我检讨。其实成年人中也有不少人能分清是非的。我记得语文老师在班上说，你们的作文里，需要“站稳立场来说谎话”，这样才能得到及格以上的分数。我也算佩服他肯说出心里话。所以背熟了张海迪，雷锋，焦裕录，中国女排等时尚的句子作为写作文的万无一失的空话，起码可以拼凑成文。那时我觉得不真诚是可以接受的，抄袭也可以，那时候很迷茫。

我不知道全国其他城市的人民知道多少六四真相。但是在沿海地区，《争鸣》等海外杂志能偷运入境，自制天线的信号虽然被官方干扰得很微弱，但大众还是可以知道真相的。当人们觉得无法改变体制时，他们的想法是“也只能是现在这样了”，“逃吧”。在七八十年代，年轻人逃亡去香港在南部沿海地区是一种常态，多少人失去了生命啊! 我认识的就有三位，都没有到20岁。为了争取自由和美好的生活，代价是沉重的。现在国内的年青人，认为体制就应该是这样的，这样也挺好的。新闻的封锁，大量的洗脑以及恐吓，使人变得麻木，敏感，自私，自大，拜金，拜权，假，骗，偷是当然，只要你够胆。

走出国门的动力，是因为我目睹了亲友的入党升官的人生路，觉得是自己不会走的路；也见到了医学老教授在路边摊和小贩讨价还价的可怜情景，那可能是我的未来。当我拿到签证后，父母催我两周内赶紧走，怕夜长梦多，那是多么希望离开故乡的一种复杂心态啊! 我没有和亲朋告别就走了。路过香港，第一次见识了童子军，竟然冒出想法，他们会不会偷取捐款箱内的钱，我是把国内的坏习惯想法带出来了，以坏的心态来判断别人的真诚。

我在美国求学、工作，再成为美国公民，一晃已经多年了，当年的民运人士们没有溅起多大的水花，反而柴女士和阮先生的新闻使我知道他们的存在。回想起香港人民每年六四烛光晚会的纪念活动，以及在国内遭受自然灾害时候他们的捐款行动，我感觉到民主自由的理念是从小培养的，是需要在民主土壤的国度才有的。有新闻自由，宗教自由的国度，你才有选择的能力，不容易被洗脑。新移民很难接受西方宗教，因为国内是无神论当道，从小被洗脑，对新的观念有很强的抗拒，没有了解世界来源的好奇心，而且每做一件事都考虑性价比，马上的好处。在美国，我有投票权，可以通过媒体看到不同的信息，我可以组织家庭教会，可以上街游行，不用考虑秋后算账。

当海外华人拥有了车子，房子，孩子以后，我们的同情心和同理心会不会被重新触动呢？什么是普世价值？我们活着还有什么责任？对于邪恶的中共利用病毒扼杀人类的罪行，我们应该做的就是必须推翻它。感谢上天给予现在灭共的机会，我们需要所有觉醒的人们来参与。有一首被中共禁止的歌曲《在人间》，歌词写道：“也许争不过天与地， 也许低下头会哭泣，也许六月雪要飞进心里， 会有柏林墙出不去，一生与苦难做邻居， 伟大时光已夺走你什么，谁能证明你在人间来过” 。我们都知道上天有眼，世人在推翻罪恶中共的运动中的作为，以后在天上是会被评审的。如果你认定了中共的罪恶，请加入爆料革命，我们一起消灭它！
