4月16日，日本驻韩国大使馆墙壁上贴出日本副首相麻生太郎的巨幅画像，韩国民众抗议日本向海洋排泄核污水的决定。“战狼”外交官赵立坚14日就这一事件口出狂言：“海洋不是日本的垃圾桶，太平洋不是日本的下水道!”，“至于日本个别官员称这些水喝了没事，请他喝了再说”，如此充满挑衅的一席话一出口，立即让一些不满日本的亲共网民兴奋，上了热搜！

日本副首相兼财政大臣麻生太郎此前曾解释日本政府排污水时说，日本核废水的氚浓度比中国与韩国核设施排放的废水要低，喝了也不会有事。有分析认为，麻生是泛指，赵立坚的“下水道”“请你先喝”等用语就有强烈的人身攻击意味，于一国发言人身份不配。

记者16日问麻生对中共外交部发言人下水道一句话有何观感，麻生借力打力：“中共说‘太平洋不是日本的下水道’，难道是中国的下水道吗？”麻生说，太平洋“这是大家的海洋”，麻生还解释，“我们可以稀释到世界卫生组织标准的七分之一，这是可以饮用的”。麻生可能没想到，他回呛中国外交部发言人的一席话在网络也有很多人追捧。根据香港『众新闻』报道，中国现存的49座核电机组，同样都有将核废水倾倒进大海。根据中国生态环境部2017年的一份政府批文，距离香港只有50公里的拥有六台机组的大亚湾核电站，每年放射性物质液态氚排放量平均为上限的20%，即大约49.5兆贝克，比日本福岛核电站制定的排放标准高出十倍。中国许多网民可能不知道？中国向海洋排放核污水是事实，争议可以继续进行下去。

现在需要了解一下，为什么日本要向太平洋排放核污水？原来，福岛核电站储存的核废水2022年下半年就要饱和了，日方一个由政府委托成立的独立专家小组指出，此类做法早已存在，而中国就是向海洋排放核污水国家之一。日本向海洋排核污水是否损害健康？日本大分护理与健康科学大学的辐射风险评估专家Michiaki Kai专家对法新社表示，关键在于严格控制好稀释和剂量以及核废水的氚浓度，这位专家解释说，科学界的共识是严格处理过的数值达标的核污水排放后引发的健康问题极其微小，当然不能说零危险，这就是为什么引起争议的原因。伦敦帝国学院分子病理学专家汤玛斯（Geraldine Thomas）认为，氚本身不会对健康带来风险，考虑到废水将融入大海，氚因素引起的风险就更小。她还认为，福岛核电站的碳14同样不存在危险，消费者更应该关心大洋中的化学污染，如水银等造成的污染。国际原子能机构也认为日本决定排放核污水的做法符合国际惯例，“是适当的做法”。
