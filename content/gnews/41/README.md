## 香港的今天是中共称霸世界的预演

《达拉斯晨报》（Dallas Morning News）近日发表了哈德逊研究所兼职研究员克劳迪娅·罗赛特（Claudia Rosette）写的专栏文章，认为在中共国违反有关香港的国际条约之后，世界不应该相信中共会遵守任何其他条约。

**中共强施港版《国家安全法》暴露了其不守承诺的真实面目**

文章开头写道，2020年是人类历史上的重要一年，除了CCP冠状病毒大流行和美国大选外，对香港人来说，这一年最刻骨铭心的事请是6月30日晚上11点，也就是这个前英国殖民地回归中共国23周年前夕，中共对香港实施新的《国家安全法》(National Security law)。中共用这个共产主义的工具，令人震惊地剥夺了中共政府承诺的回归后的50年里给香港的权利和自由。这部法律由中共中央政府制定和通过，在不允许香港人有任何发言权的情况下强行实施，有效地消除了将香港相对自由开放的制度与中共日益专制的统治分隔开来的制度性障碍。根据这项法律，香港的任何选择行为几乎都不是权利问题，而取决于中共的授意或不授意。

文章认为，这部法律的通过，标志着共产主义暴政第一次全面摧毁了一个繁荣成熟的自由社会。对香港来说，这是一场巨大的悲剧。对我们所有人来说，这是21世纪世界秩序的预兆，中共国的最高统治者已经在他的“人类命运共同体”的愿景下对世界产生了重大影响，并提出要主导世界秩序。

由于香港处于中共国主权之下，中共称其对香港的镇压纯粹是内部事务， 这是错误的。 中共国与英国签署了一项有约束力的国际条约, 即1984年的《中英联合声明》, 承诺从1997年香港回归之日起至2047年，香港在国内事务上实行自治，在中共称之为 “一国两制 “的安排下享有50年的 “高度自治”。

今天，令人瞠目的不仅仅是中共的不诚信，它违反了一个还有27年有约束力的条约，发布了如“2060年将碳排放减少到零”这些毫无价值的声明，这都应该让任何还倾向于相信中共许诺的人重新思考。除此之外，更让人震惊的是，中共迅速并粗暴地将香港这个充满活力的开放城市变成了一个镇压、审查和政治犯的囚笼。

在确定中共对下一个目标台湾的威胁以及中共国最高领导人希望在全球范围内取得统治地位的时间表时，首先要搞明白中共在决定行动后是如何迅速控制香港使其沦陷的，要知道两年前的香港还是自由世界的伟大城市之一，一个享受着自由贸易、自由言论、英国人的法治传统的750万人口的沿海国际大都市，被自己的政府骄傲的称为 “亚洲的世界城市 “。但是，越来越多的迹象表明大麻烦即将来临。中共背弃了民主的承诺，留给香港的是一个由中共选出的特首，以及长期以来多数席位亲中的立法会。尽管如此，在动态开放的香港与专制共产主义监视下的中国大陆之间，还是存在着巨大的差异。

**中共利用恶法在香港疯狂实施红色恐怖打压民主**

文章用很长的篇幅细数了中共和港府如何沆瀣一气在香港打压民主的恶行。2019年，香港行政长官林郑月娥(Carrie Lam)试图强行实施一项允许向中共国引渡的《逃犯条例》，从而爆发了大规模的街头示威活动，林郑月娥被迫撤回《条例》，但香港抗议活动迅速转变为对民主的诉求求和防范中共对香港日益严苛的管控。抗议活动持续了几个月，抗议者和警察之间的街头冲突不断升级，中共方面拒绝做出任何妥协，并发出了可怕的威胁。香港人仍坚守着阵地。2019年11月底，他们抓住了一个象征性的机会，让民主派候选人在区议会选举中获得压倒性胜利，从而表达了他们的民主愿望。虽然这些职位在很大程度上无实权，但这至少是一个投票的机会。这一胜利让人们燃起了希望，即在2020年9月举行的更重要的立法院选举中，可能会有足够多的亲民主派的选票，最终推翻旨在确保亲中共的多数席位的选举制度。

2020年初，CCP新冠病毒从中共国武汉传播。从那以后，相关的限制措施就成了中共肮脏勾当的面纱。今年3月，香港政府对几乎所有非居民实施了入境禁令，现在延长至2021年3月，这大大限制了香港的国际交通，并将许多国际记者隔离在外。今年6月，当全世界都在关注疫情时，中共通过了《中华人民共和国香港特别行政区维护国家安全法》，给香港的自由造成了致命打击。这部法律共有66条，但归根结底是中共赋予自己干预香港事务的权力，只要是中共政府认为与国家安全有关的事务——在中国共产党的统治下，这个概念由党的领导界定—— 中共都有权介入。中共在法律上将中共当局认为危险的批评或与 “外国分子 “的 “勾结 “等活动定为刑事犯罪，用词模糊，即使以前认为再正常不过的讨论都可能将置香港人于危险之中，被视为严重违法者可被送往中共国起诉和处罚。该《安全法》条文使中共的定罪权不仅仅局限于香港，而且延伸到全球任何地方的任何人。据美中经济与安全审查委员会(U.S.-China Economic and Security Review Commission)去年12月提交给国会的一份报告称：“如果不加以约束，这部法律将赋予中国政府审查全球言论的广泛权力。”

根据这项法律，今年夏天，内地公安部门首次公开抵达香港，并被授权在他们认为合适的情况下进行监视、无证搜查和逮捕。香港当局也开始清除公共书架上有关民主的书籍。他们将著名的抗议口号 “光复香港！时代革命！”定为非法，禁止学生歌唱或演奏抗议者去年推出的那首令人难忘的圣歌“荣耀归香港”， 逮捕那些高举空白标语示威的抗议者。香港政府正在修改教科书，删除三权分立等民主思想，用中共国版本的“爱国教育”取而代之。有报道称，香港家长不敢在自己的孩子面前讨论政治，唯恐学校当局发现持不同政见的迹象。

对于香港的民主倡导者来说，定于9月举行的立法会选举提供了一丝希望。为了赢得多数选票，他们组织了7月中旬的初选，结果产生了创纪录的投票率，但是选举从未举行过。7月31日，林郑月娥宣布，由于对CCP冠状病毒的担忧，选举将推迟一整年。这样一来，旧的立法机构依然存在，民主派议员依然占少数。但即便如此，对中共来说还是异议太多。11月份，中共全国人民代表大会常务委员会以 “国家安全 “为由，取消了四名香港民主派立法者的资格，剩下的十五位民主派议员随后辞职以抗议毫无希望的局面。

与此同时，逮捕和指控行动进展迅速，香港的监狱里挤满了持不同政见者。自去年抗议活动开始以来，已有超过1万人被逮捕，其中许多人是在《国家安全法》颁布之前被捕的，但现在所有这些人（或与此有关的香港人）都将面对这样一个体系，即中共严苛的“国家安全”定义正在重塑规则。本月早些时候，杰出的年轻民主倡导者黄之锋(Joshua Wong)和周庭(Agnes Chow)被判入狱，73岁的香港最著名的民主异见人士黎智英(Jimmy Lai)目前正因多项指控等待审判。黎是一位富有的商人和出版商，也是广受欢迎的民主派报纸《苹果日报》的创始人。香港当局今年已多次逮捕他，并让他戴着手铐脚镣示众，搜查其办公室，最后根据《国家安全法》指控他勾结外国势力，有可能被判终身监禁，并在中共国服刑。香港当局还向多位已逃离香港的不同政见者发出逮捕令，其中包括一位已获的美国公民身份的人士，罪名是他游说美国政府支持香港民主诉求。八月，12名香港异议人士在试图经水路逃往台湾的途中被中共海警抓捕，被囚禁在大陆而不是香港，其中10名已被指控。

**在美国的带领下，世界已经认清中共的野心**

文章最后赞扬称，在总统川普的领导下，美国率先在全球范围内对这一中共造就的噩梦作出了回应。美国撤销香港的特殊贸易地位，取消引渡条款，并将包括行政长官林郑月娥在内的部分香港高官列入制裁黑名单。最重要的是，川普政府在唤醒世界对中共国野心的警惕同时，也已经开始放弃了几十年来帮助中共强大的开放的贸易往来政策。文章也提醒，即使拜登上台，其政府中任何可能会被诱惑而重走老路的人都应该仔细看看中共最高领导人今年在香港这个曾经的自由之城所造成的悲催景象以及其毁坏之速度，不要重蹈覆辙。
