据2021年6月8日发布的中国光大银行5月31日出具的公告显示，三亚凤凰国际机场有限责任公司、海口美兰国际机场有限责任公司分别有15亿元、28亿元中期票据违约，不能偿还到期债务。公告明细如下：

1. [中国光大银行股份有限公司关于三亚凤凰国际机场有限责任公司2017年度第一期中期票据违约后续进展情况的公告(2021年5月)](http://www.chinamoney.com.cn/chinese/zqzdsx/20210608/1980497.html?cp=zdsx)，涉及10亿元违约。
2. [中国光大银行股份有限公司关于三亚凤凰国际机场有限责任公司2017年度第二期中期票据违约后续进展情况的公告(2021年5月)](http://www.chinamoney.com.cn/chinese/zqzdsx/20210608/1980500.html?cp=zdsx)，涉及5亿元违约。
3. [中国光大银行股份有限公司、国家开发银行关于海口美兰国际机场有限责任公司2017年度第一期中期票据违约后续进展情况的公告(2021年5月)](http://www.chinamoney.com.cn/chinese/zqzdsx/20210608/1980496.html?cp=zdsx)，涉及15亿元违约。
4. [中国光大银行股份有限公司、国家开发银行关于海口美兰国际机场有限责任公司2016年度第一期中期票据违约后续进展情况的公告(2021年5月)](http://www.chinamoney.com.cn/chinese/zqzdsx/20210608/1980498.html?cp=zdsx)，涉及13亿元违约。


根据公告, 两家机场均被中共法院出具了破产公告和重整裁定书，[同属于海航破产重整案件附带影响的一部分](http://www.dagongcredit.com/uploadfile/2021/0301/20210301042315761.pdf)。

2021年6月7日渤海租赁公告，其[发行的短期融资券展期270天](http://www.chinamoney.com.cn/dqs/cm-s-notice-query/fileDownLoad.do?mode=open&amp;contentId=1979664&amp;priority=0)，本质上是违约的后续处理办法，渤海租赁隶属于海航集团。

其中三亚凤凰国际机场的破产重整申请人是安徽省安源消防工程有限公司（以下简称安源），说明安源存在很多工程款未能得到付款，安源是是一家从事消防系统工程、机电工程、电子与智能化工程、建筑装饰工程与设计及材料销售等的公司。而工程款违约通常涉及到工程企业的债务违约，工程企业的负债项目通常包括应付账款（核算应付未付的分包劳务款、材料款等）、应付职工薪酬、应交税费，包括了对农民工的工资债务。

同时农民在海航集团的银行债权人的存款安全性也会受到海航破产重整的损害。银行通过发放贷款给海航集团、购买海航集团成员企业发行的中期票据成为债权人，银行发行的理财产品，也叫资产管理计划，也会购入海航集团的债券成为债权人，部分农民可能用自己省吃俭用积累下来的工资买了理财计划，成为受害者。

工程企业的工程结算款=营业收入+增值税=增值税+净利润+税金及附加+所得税费用+营业成本+销售、管理、财务费用+减值损失-投资收益-其他综合收益+其他项目等，营业成本=材料费用+人工费用+机械施工费用+其他成本费用等。

有位农民工[在网上表示](http://www.ma173.com/remen/remen/thread-88505.html)“建筑工地月结的基本属凤毛麟角，大多是每月付生活费，遇上学生开学或农忙季节，另有一笔借支。一桩工程做完，耿直的结清帐，否则拖到年底，年底还常常拖欠。于是便有了大年三十还在要帐的情况。我干过许多工地，无论是中建、中铁、中冶这些大公司，还是一些小建筑单位，对于工资拖欠的现象都比较严重”。

笔者去过中铁、中交等企业的项目部，中午吃饭的时候，施工单位项目部领导在小包间吃饭，农民工在外面大厅吃饭，荤菜比例非常少，大锅菜，素菜以油麦菜、海带、大白菜等为主，有的是小空调，有的是用风扇。

联想到最近中共政权规定农民工讨要工资违法，[要判刑](https://www.163.com/dy/article/G1UBEADS0545HLK1.html)，[把讨薪工人当恐怖分子](https://www.ntdtv.com/gb/2021/05/22/a103125686.html)，可以看出中国共产党通过盗国行为侵犯了工人利益，再次侵犯农民利益，有的地方还要求不打中共的毒疫苗，不能上班，中共已经彻底成为中国农民和工人的敌人。

要知道为何农民出来打工，996加班挣钱。是因为中国共产党对农村和农民的掠夺性政策，中国共产党从未赔偿其农业税、计划生育政策对农民的严重伤害。首先虚报产量导致的非法征收农业税饿死了多少农民，生命可以计价吗？其次农业税未设定免征额，实施的是比率税制（意味着无论产量多少，都得按比率交税，不考虑农民维持生命所需要的最低食品数量），再加上打压言论自由，不尊重私有财产权，阻碍农业科技发展，抑制了农业产量，导致即使不虚报产量，留给农民自己消费的也不多，导致农村普遍存在营养不良问题。

虽然农业税已经停止征收了，但是农业税非法征收对农民的身体伤害、死亡损害赔偿了吗？农业生产资料里就不包括非法征收的税了吗？随意没收农民安装的用于收看卫星电视的锅盖有道理吗？给农民孩子发放助学贷款、奖学金、助学金。让农民跑断腿有道理吗？给农民工的工资价格公允吗？加班对农民工身体的伤害不用赔了吗？仅仅发放种粮补贴、搞村村通公路形象工程就够了吗？农民不需要这些补贴和形象工程，先追究中国共产党的法律责任，赔钱。

计划生育也一样，不再只限一胎，可以三胎，但是损害赔偿付了吗？计生补贴蝇头小利能够弥补人口损失、生育能力损失、人权侵害的巨大伤害吗？受够了，先追究中国共产党的法律责任，赔钱。
