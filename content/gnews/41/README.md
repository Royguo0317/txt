据《SBS新闻》在本周六的**[报道](https://www.sbs.com.au/news/dalai-lama-gets-coronavirus-vaccine-in-india)**，85岁西藏精神领袖**达赖喇嘛（Dalai Lama)** 在印度北部山区小镇达木萨拉的一家医院接种了第一针中共病毒疫苗注射后，他敦促人们站出来，勇敢地接种疫苗。佐尔医院的G.D.古普塔博士告诉记者：**住在达赖喇嘛住所的另外十人也接种了疫苗。**所有11人都接受了科维希尔德疫苗，该疫苗由牛津大学和英国制药商阿斯利康开发，由印度血清研究所制造（价格便宜）。以下是中共媒体对该疫苗的相关报道：

- 据北京时间2021年2月8日《新华网》援引《金融时报》的**`报道`**，阿斯利康发言人说：“牛津大学和阿斯利康已经开始改进疫苗，从而让它能够应对南非B.1.351变异毒株。临床试验将加速开展，以期秋季可以供货。”另外一名研究员萨拉·吉尔伯特说：“我们一直预测，随着疫情持续，变异毒株将成为‘主流’，而最终可能需要新版本的疫苗。”
