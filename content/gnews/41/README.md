周育森就职于军事科学院研究室并担任该实验室主任一职，曹务春（军事医学科学院微生物流行病研究所所长）是周育森的直接领导，周本人长年从事病毒致病性研究，具有丰富经验。周育森本人，其妻杜兰英与李放等多位病毒专家曾共同参与多项冠状病毒研究。

去年9月，路德先生在节目中爆料周育森于2020年5月离奇死亡，其死亡时间点和原因十分蹊跷，周死亡前并出现任何异常症状，中共媒体直到7月份才公开其死亡信息。更让人惊讶的是，周育森的妻子杜兰英对丈夫的去世表现得异常平静，始终沉默不语。杜兰英从2007到2008年在香港大学P3实验室进行博士后病毒研究工作，Malik和袁国勇教授当年就是杜兰英的指导老师。由此判断，中共因害怕计划败露就预谋杀死周育森，并要挟杜兰英，要起闭嘴。
