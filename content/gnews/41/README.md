## 【正道观察】中共军事扩张不休, 美纵横各国戒备升级

校对：萌萌的朋克
![]()![](https://gnews.org/wp-content/uploads/2020/12/Picture16-3.png)图片来源：德国之声 www.dw.com
今年，美国海军战争学院（NWC）资深学者安德鲁·爱立信(Dr. Andrew S. Erickson)分析了美国五角大楼发布的《2020年中共国军力报告》，他在分析报告中着重强调了中共国近年来在某些军事领域的进步实际上已经领先美国。尤其，今年中共国海军舰队总数已经超过了美国。另外，安德鲁的分析报告还特意指出：中共国的核弹头储备在2020年度已经增加了一倍。

笔者并非军事专家，也无相应资质，但经过一些新闻的分析、比对之后，还是发现美国国防部在《2020年中国军力报告》问世后，便迅速对其印太战略进行了大幅度的调整。其中非常明显的，从媒体报道中可以一窥究竟的有：

一，美国海军迅速调整了对南中国海的战略部署。未来的海战，美军与其采用单刀赴会作战方案，不如考虑连横多国部队介入南中国海的军事冲突。尤其值得关注的是： 在《2020年中国军力报告》公布后的三个月时间里，美国及其周边战略联盟国家在印太地区进行了如下大型军事演练，意在对付中共海洋扩张战略：

七月，台湾海军陆战队实施“联合登陆作战”演练；

七月，美日澳三国海军于菲律宾海举行三国军演，与此同期，美印海军还在安达曼海举行了联合军演；

八月，两年一度的环太平洋军演已经没有了中共国的影子，这次“2020环太平洋军演”（RIMPAC 2020）只有美、澳、日、韩四国参加，但受邀国增加到了十个，其中包括越南；

十月，日本自卫队（JSDF）和美国海军在台湾海域和日本冲绳海域附近进行“利剑-21”（Keen Sword 21）大型军事演习；

十一月，印、美、澳、日四国海军再次集结进行了“马拉巴尔-2020”（Malabar 2020）联合海上军演，地点在阿拉伯海，其中包括美、印“双航母”演习和跨甲板飞行；

十一月，美国海军陆战队赴台湾进行了相应的军事演练。

日、法、美三国共同决定于2021年五月进行联合军演，届时会邀请英国航母和德国战舰参加演练。

二，美国海军加速调整了其在南中国海的先进武器部署方案。为了避免与中共国海军过多的正面冲突，美海军开始加紧研发新一代的无人舰船和无人空中打击系统。其中包括：大型无人水面舰艇和无人舰队，新型无人机系统和无人机战斗群。另外，美军继在日韩部署了萨德系统之后，还将在澳洲部署该系统，用以对付中共海军 （其中部署韩国的萨德系统可以拦截中共火箭军部署在东北、山东和新疆的东风-21D和东风-26系列导弹）。

三，澳洲政府也迅速调整了《2020国防战略更新计划》，其中包含了重塑澳大利亚战略环境和遏制中共破坏澳大利亚国家利益的目标，将其重心放在了环印太地区。德国政府更推出了《印太政策指导方针》，用以加强与印太地区国家的合作，保障本国在该地区的利益。

综上所述，中共在环印太地区加速扩张军力的野心和企图已经引起了全世界的广泛关注和高度警觉，英、法、美、德、印、澳、日、韩等国家已经组成了强大的战略联盟来共同抑制这种“新中共海洋霸权主义”。 2020年是各种国际联盟重新调整对环印太战略的关键一年，最终成效必定会在2021年得到着实的显现。
