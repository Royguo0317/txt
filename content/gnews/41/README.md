## 德国对华为的态度与欧盟一致

作者：文虓Bobby

![]()![](https://gnews.org/wp-content/uploads/2021/04/042502.png)

德国国会议员周五通过了更严格的5G安全立法，这为两年来人们对欧洲最大市场是否会对中国电信巨头华为采取强硬态度的疑虑扫清了障碍。

德国联邦议院(Bundestag)批准的新《信息技术安全法2.0》(IT Security Law 2.0)限制了“不可靠的”5G技术供应商的角色，并要求电信运营商在签署关键5G组件合同时通知政府。它还赋予政府权力来阻止它们。

经过多年的争论，这些规定使柏林在监管敏感问题上更接近巴黎、伦敦和其他主要国家的标准。

即便如此，实施这些规则仍需要政治意愿。德国将于9月举行联邦选举，决定如何应用这些标准的工作将在很大程度上落在下届政府的肩上。

德国内政部长霍斯特·泽霍费尔(Horst Seehofer)对德国联邦议院(Bundestag)表示:“如果供应商不值得信赖，就不能使用关键部件。”

就在对更严格的5G规则进行投票的几天前，中国外交部长王毅在与德国外长海科·马斯(Heiko Maas)通话时提到了5G。

“双方应共同反对所谓‘脱钩’，共同维护全球产业链和供应链的畅通和稳定，”中国商务部部长表示。

由上可知，德国针对华为在5G上的态度坚决地与英国、法国站在一起，严格限制华为的5G通信相关项目，说明欧洲在逐步觉醒，针对中共国的超限科技信息战，德国、英国等西方国家与美国站在一起。而中共国外交部长王毅所谓的双方应共同反对所谓“脱钩”，只是他的一厢情愿，完全是在自说自话。实际上，这是欧洲等西方国家在事实上与中共国的经济正在“脱钩”！

**“必须符合北约的目标”**

在议员们对德国总理安格拉·默克尔(Angela Merkel)处理这一问题的方式表示失望之后，最新的提案比去年12月政府提出的提案立场更加强硬。

有关德国对华为立场的谈判去年基本上陷入停滞，原因是政府部门和联合政党未能在关键的市场准入问题上下定决心。

与此同时，在美国要求默克尔采取更强硬立场的压力下，默克尔推动与北京签署一项投资协议，并维护与中国的贸易关系。

华为是德国三大4G基础设施运营商的特权供应商，执政联盟在如何处理华为问题上存在分歧。

该法律的最终版本赋予了内政部更大的权力来阻止5G合同，而经济部门等其他部门的权力则减少了，经济部门过去对华为更有利。

在运营商通知政府计划购买新的关键5G组件后，内政部将有2到4个月的时间根据国家安全标准检查该交易。

议员们还补充说，新的关键5G组件的使用必须符合德国、欧盟和北约的“安全政策目标”。后两者已采取立场，告诫不要在新电信网络的关键和敏感部分使用中共国设备。

制造商还必须提供“可信度声明”，而议会成员也提高了供应商达到“可信度”标准的门槛。

众所周知，德国默克尔总理针对中共国和华为的暧昧态度让德国议员和美国感到不满。但是，在全球灭共的大趋势下，必须符合北约的目标，北约的利益是第一位的。因此，德国政府不得不逐步转变立场，或者说必须清晰地选择灭共的立场，站在美国和北约一边。
