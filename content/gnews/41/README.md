# **香港迅速陷落于中共的压迫为我们所有人拉响了警报**

**香港曾经是世界上最自由和最开放的地方之一，如今却沦落成为独裁政府。它的陷落应使西方感到恐惧。**

[!\[\]()!\[\](https://spark.adobe.com/page/L0IlqH3MvdQsA/images/6e2128c9-ba9c-4dca-a78f-957b24a3b98f.png?asset_id=6d87856f-2675-4792-8279-de5e06ab0e3d&amp;img_etag=%22c64bdd1ec4eb47cb347bec1f469782bb%22&amp;size=2560)](https://spark.adobe.com/page/L0IlqH3MvdQsA/images/6e2128c9-ba9c-4dca-a78f-957b24a3b98f.png?asset_id=6d87856f-2675-4792-8279-de5e06ab0e3d&amp;img_etag=%22c64bdd1ec4eb47cb347bec1f469782bb%22&amp;size=1024)

香港的坠落始于1997年，当时中国共产党从英国政府手中接管了香港。从那时起，二十多年来，香港人目睹了中国共产党慢慢地侵蚀着他们的自由。这场全球流行的新冠疫情为中共提供了一个加速控制香港及750万居民的绝佳机会。

去年夏天，中共国政府绕过了香港的本地立法机构，对香港实施了严苛的港版《国安法》。香港民主派立法委员郭荣铿说，北京的行动“基本上意味着‘一国两制’的终结”。直到2020年6月30日午夜，香港政府在其网站上发布了该法律，香港和世界其他地区的人甚至都不知道这项法律意味着什么，此时，该法已经生效。

《国安法》将任何“分裂国家、颠覆国家政权、恐怖活动和勾结外国或者境外势力”的行为均定为刑事犯罪，最高可判处无期徒刑。对每一种所谓的“犯罪”，甚至是构成“国家安全”的定义都是如此模糊，然而，仅仅是一条支持香港民主运动的推特就可能把一个人关进监狱。

为了执行《国安法》，北京在香港建立了一个新的拥有广泛权力的安全机构（国家安全公署），（职能）包括接管香港的某些警察的案件。这个机构不受《香港基本法》（事实上的地区宪法）的约束。所有被逮捕的人都将在中国大陆接受审判，这意味着被告人将不会有法定诉讼程序、充分的法律代理或一个公正的审判。

《国安法》还授予香港警方前所未有的权力，包括“进行无令搜查、扣押财产、调查嫌疑人、截取通讯、冻结资产和限制受查人员离境的权力。”

据《华尔街日报》报道，自《国安法》生效八个月以来，香港“所有人都受到了控制”，北京在多个方面都实施了这项法律以打击持不同政见者，并解散了为保护香港人自由和维护法治而建立的相关机构。

**监禁香港泛民主派领袖**

去年8月，香港警方逮捕了著名社运活动人士黄之锋、周庭、黎智英和其他参与2019年反送中条例抗议活动的人。黄之锋、周庭和林朗彦已于去年12月被判入狱，而黎智英则仍在为对他的指控而抗争。

同月，中国大陆的海岸警卫队逮捕了12名社运活动人士，号称“12港人”（HK12），他们试图通过快艇逃离香港。他们的年龄从16岁到33岁不等。根据《国安法》，“12港人”被关押在内地的一个监狱，并被内地法院判处长达三年的监禁，尽管他们的家人一再要求在香港法院审判他们。

香港警方逮捕了另外11名香港人，指控他们协助“12港人”逃离香港。另有9名社运活动人士，包括黎智英和82岁的泛民主派资深人士李柱铭，目前正在香港受审，可能会被判处长达5年的监禁。

**清除香港立法会中的反对派**

去年夏天，香港当局利用《国安法》取消了十几名泛民主派候选人的资格，包括参加即将到来的香港立法会选举的现任反对派议员和社运活动人士。北京随后颁布了一项法令，要求香港政府绕过地方法院，驱逐那些“支持香港独立、危及国家安全、拒绝承认中共国对香港的主权”的立法会议员。

15名反对派议员为表示抗议而集体退出立法会。北京任命的香港行政长官林郑月娥以冠状病毒疫情为借口，将香港的立法会选举推迟了一年。然后，在今年的1月6日，香港当局部署了1000多名香港警察，逮捕了53人，包括在去年举行的非官方民主派立法会初选中的6名组织者和47名参与者，该初选是为确定下一次参与立法会选举的民主派候选人而举行的。

就在上周日，香港当局根据《港版国安法》正式指控47名前议员和社运活动人士阴谋颠覆国家政权，其中一些人可能被判处终身监禁。显然，对中共和香港傀儡政府来说，推行民主化进程是对国家安全的威胁。

中共对监禁的候选人不满意。在3月的两会上，中国的橡皮图章立法机构将通过一项新的法律，永久改变香港立法会的组成。这将取消“分配给香港区议会的117个席位，区议会是一个至今仍由反对派主导的团体”，并将这些席位分配给“真正的爱国者”，这些人将从北京接受命令。

**攻击香港司法独立**

[!\[\]()!\[\](https://spark.adobe.com/page/L0IlqH3MvdQsA/images/94fbca18-e12a-464f-88d0-7fe81636fa95.jpg?asset_id=eac52bcc-9f4d-4251-bafb-11664721fea4&amp;img_etag=%22abbdbacfbca23889afb5bda4cbab6007%22&amp;size=2560)](https://spark.adobe.com/page/L0IlqH3MvdQsA/images/94fbca18-e12a-464f-88d0-7fe81636fa95.jpg?asset_id=eac52bcc-9f4d-4251-bafb-11664721fea4&amp;img_etag=%22abbdbacfbca23889afb5bda4cbab6007%22&amp;size=1024)

香港的行政部门牢牢控制着北京的效忠者，北京成功地消除了香港立法中的任何反对派，香港的司法制度是维护香港人自由的最后一道防线。

然而，多年来，北京一直在干涉香港的司法系统，包括绕过香港司法系统，派遣中国大陆警察到香港逮捕香港书商（桂明海）和一名中共国大亨（肖建华）。由亲北京议员提出并支持林郑月娥的2019年香港送中条例，是又一次削弱香港司法制度的尝试，该条例的目的是试图让香港交出北京想要遣返回大陆的任何人。林郑月娥在面临数个月的大规模抗议后才撤回了该条例。

内地官方媒体和官员经常公开批评香港法官，这些批评变得如此恶毒，致使香港（终审法院）首席法官马道立采取了一种非寻常的举动发布声明，警告那些对香港的司法制度的攻击。自从港版《国安法》生效以来，只有那些经北京批准的法官们才被允许受理与《国安法》有关的案件。

虽然一些有独立思想的法官选择退休而不是屈从于北京的压力，但《华尔街日报》报道，北京是决定让香港的司法制度更像大陆的司法制度，因为“一个独立的司法体系与大陆的司法体系是不相容的，因为大陆的法官被认为是实施党的统治的政治机构的一部分。”

**洗脑香港青年**

今年2月，香港当局宣布强制性的“爱国”教育，从幼儿园开始，将教育小学的孩子遵守《国安法》。据《华尔街日报》报道，香港当局正在改写教科书，让六岁以下的香港年轻人牢记，事关国家安全“没有辩论或妥协的空间”，教师应该“培养学生维护国家安全的责任感。”

香港第二任行政长官曾荫权(Donald Tsang)在2010年试图引入类似的教育，试图将香港学童塑造成北京想要的忠诚公民。虽然他也在十多年前的抗议中最终撤回了他的计划，但这次没有人敢抗议，因为害怕在《国安法》下面临严厉的惩罚。

此外，香港当局还加强了对当地媒体和互联网的控制，包括阻止香港人访问特定的网站，这是香港过去从未发生过的事情。难怪香港人抱怨说，《国安法》统治下的镇压“和1989年北京的枪支和坦克一样有效，当时天安门广场周围的抗议学生遭到射击和坦克的镇压。”
