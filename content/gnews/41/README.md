白宫易主后的中美首次高层对话是德国媒体关注的焦点。

德国电视一台报道说，美中关系已长时间紧张。美国新政府上台后的两强第一次会晤上，双方甚至很罕见地公开争吵。美国批评中国的人权，但中共也针尖对麦芒，批评美国的人权和民主。双方分歧之大，难以靠近。本来，会晤后通常会有共同记者会，但这一点根本就不在计划中。

德国《商报》报道说，即便在两国外长会晤的第二天，双方还是语言强硬，几乎没有靠近。美中峰会使两大强国之间的严重分歧变得明显可见。峰会周六结束时，连原本很低的期望也没有得到满足。人们看不到两国关系将缓和的迹象。华盛顿和北京的关系迈入了一个新的冰河时代。

这不仅是冰河时代，这是美国和中共最终火战的第一次外交对话，以中共的种种表现来看这一战随着病毒真相的曝光必将到来，全球联合灭共的时代真的来了。
