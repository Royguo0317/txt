# **川普：摇摆州认证拜登胜选是“应受严惩的罪行”**

[!\[\]()!\[\](https://spark.adobe.com/page/CQUh9ly5YAZ5n/images/0184f12d-c801-4612-8d61-5e5af87e0047.jpg?asset_id=9bdc1ca4-5457-436b-bee4-4ffb7b53510c&amp;img_etag=%225e8e6744ce052042f3f5257130794304%22&amp;size=2560)](https://spark.adobe.com/page/CQUh9ly5YAZ5n/images/0184f12d-c801-4612-8d61-5e5af87e0047.jpg?asset_id=9bdc1ca4-5457-436b-bee4-4ffb7b53510c&amp;img_etag=%225e8e6744ce052042f3f5257130794304%22&amp;size=1024)

美国总统唐纳德·川普登上海军陆战队一号之前，走在华盛顿白宫南草坪上。（AP图片/帕特里克·塞曼斯基（Patrick Semansky），文档）

川普总统在选举团投票核实乔·拜登当选前夕表示，摇摆州认证舞弊选票是在实施“应予严惩的罪行”。

“发现大规模选民舞弊的摇摆州，也就是所有摇摆州，不能合法认证这些选票为完整且正确的，否则就犯下应受严惩的罪行，”川普先生发推说。“这次选举正受到抗议！”

选举团成员将于周一在各州首府会晤，为候选人正式投票。拜登先生赢得306张选举人票，川普先生获得232票。候选人需要270票才能获胜。

上周五晚在最高法院的选举挑战中失败的总统重申其主张，即拜登的胜利是通过民主党官员在战场州的大规模舞弊获得的。

“大家都知道，死人、低于投票年龄的人、非法移民、伪造签名、囚犯以及其他许多非法投票的人，”总统说。“此外，机器’故障’（舞弊的另一个词）、收割选票、非居民选民、伪造选票、’塞满票箱’、有偿投票、粗暴对待共和党的投票站观察员，有时甚至选票多于投票的人，发生在底特律、费城、密尔沃基、亚特兰大、匹兹堡和其他地方。”

他说：“在所有摇摆州出现的情况中，选票远远多于赢得该州和选举本身所需的票数。因此，选票不能被认证。”

总统的法律团队在六个州指控选举舞弊或其他违规行为的法院诉讼中遭到数十次拒绝。
