据Mayo诊所介绍，淋巴结存在于身体的各个区域，包括颈部、腹股沟和腋窝，并在细菌、病毒等感染时或有癌症的情况下肿胀。

根据Susan G. Komen基金会的说法，通常，腋窝区域的淋巴结肿大需要对乳房组织进行活检，因为它可能是乳腺癌开始传播的标志。但是放射科医生表示，这种情况在接受新冠病毒疫苗注射后是可以预料的，通常不会引起关注。

马萨诸塞州总医院在最近的一份新闻稿中表示，在大多数患者中，接种新冠病毒疫苗后淋巴结肿胀的情况通常不需要采取进一步行动，除非肿胀持续存在或引发其他健康问题。

疾病控制和预防中心在关于Moderna疫苗和辉瑞疫苗的报告中指出，淋巴结肿胀可能与疫苗有关，最常见的是在接种疫苗后两到四天在手臂和颈部区域出现。Moderna的肿胀平均持续一到两天，辉瑞的肿胀平均持续10天之久。

马萨诸塞州总医院警告说，随着全国疫苗接种力度的加强，放射科医生们可能会发现乳房X光检查期间出现淋巴结肿大的病例数量有所上升。

马萨诸塞州总医院乳房成像主任Constance Lehman告诉《纽约时报》，她很高兴听到这一事件的原因正在传播，这样可以减轻人们的焦虑。她说，“我特别希望向所有在癌症成功治疗后仍在接受观察的患者们传达这个消息。”
