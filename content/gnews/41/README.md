美中关系有那么一段时间所发生的事实是任何人都不能否定的，即在拜登上任之前，川普政府对中共态度异常强硬，中共高层陷入了前所未有的恐惧，熟悉中共内幕的著名记者石坂明夫称，习近平因为害怕被斩首甚至躲了躲起来了，和路德社披露习大神隐身二十多天变成天线宝宝是相互印证的。

拜登上台后中共象打了鸡血一样重新亢奋，立即出台在争议海域动用武力的《海警法》，并对台湾、日本、南海频频骚扰，加剧了西太平洋局势的动荡，而美军也毫不示弱，在印太加强了军事部署。拜登政府为了避免双方的误判和冲突，表示“由于印太地区的紧张局势不断加剧，奥斯汀与许其亮的对话非常重要”。但中共对拜登政府的呼吁不予理睬，一副典型的“一掐脖子就翻白眼、一松手就吹牛逼”流氓无赖嘴脸。

美防长奥斯汀向中共军委副主席许其亮三次发出会谈要求，希望恢复川普执政最后一个月业已中断的美中军方高层对话，但许其亮采取躲避的办法三次拒绝，加上美国在3月份阿拉斯加会谈后主动中断了与中共的高层对话，当双方的军队脱离了安全管控机制时，极容易产生误判而引发冲突。

美国还是那个美国，中共如此前恭后倨是有原因的，习近平在川普和拜登不同时期的两付嘴脸，和前国务卿逢佩奥所指“中共吃定了拜登的软弱”有关。许其亮对奥斯汀的谈判要求置之不理，说明习包子不急于和美国管控军事风险，这和他多次强调的“以战逼和”是一脉相承的。中共出台《海警法》，在台海、南海及钓鱼岛实施高强度的军事骚扰，实际上就是在进行战争恫吓，如果美国表现出丝毫的恐惧，中共就会得寸进尺。

还有另一种可能，在中共的极权专制之下，许其亮是谁根本不重要，重要的是谁是一尊。由于习一尊已被普遍认为是个草包，中共党内的反对派会认为，如果任由习包子折腾下去，所有人都是习包子的陪葬品，在取代习的内部权力争斗异常激烈的当下，对当前美中关系的政策调整无法统一意见，导致指令迟迟不能下达也是有可能的。
