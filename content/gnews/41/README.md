受中共病毒疫情影响，中共国经济形势严峻，通货膨胀的势头已经越来越猛。据[中共国家统计局](http://www.stats.gov.cn/tjsj/zxfb/202104/t20210430_1816900.html)数据显示，4月份，中国制造业采购经理指数（PMI）回落。同时，[中共国年轻人失业率](https://www.yicai.com/news/101023752.html)居高不下。种种迹象表明，中共国经济可能出现滞胀。

滞胀是指停滞性通货膨胀。一般在正常情况下，经济衰退时，失业增加，收入减少，消费受到抑制，物价也会持续下降。但发生滞胀时却恰恰相反，在经济衰退的同时，物价反而持续上涨，形成高通胀、高失业和低经济增长的独特经济现象。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/08083605/%E3%80%90%E7%83%AD%E7%82%B9%E6%92%AD%E6%8A%A5%E3%80%91%E9%9B%AA%E4%B8%8A%E5%8A%A0%E9%9C%9C-%E4%B8%AD%E5%85%B1%E7%BB%8F%E6%B5%8E%E5%8F%AF%E8%83%BD%E5%87%BA%E7%8E%B0%E4%BA%86%E6%BB%9E%E8%83%80.png)图片来源：中共国家统计局网站

中国制造业采购经理指数（PMI），是由5个分类指数加权计算而成的，分别是：新订单指数、生产指数、从业人员指数、供应商配送时间指数和原材料库存指数。从中共官方公布数据来看，相较于3月份，4月PMI较上月回落 0.8个百分点，中共制造业供给和需求双双走弱。制造业生产扩张力度有所减弱，市场需求扩张有所放缓，主要原材料库存量较上月下降，企业用工景气度有所回落，原材料供应商交货时间延长。制造业和非制造业PMI双双回落。

年轻人就业困难，据报道，3月份，中共国年轻人失业率高达13.6%。中共号称经济已恢复到疫情前，但如此严重的失业率却对内需的回升带来重大的影响。

中共国通胀加剧。根据[中共国家统计局](http://www.gov.cn/xinwen/2021-04/09/content_5598594.htm)公布的数字，今年3月，中国居民消费价格指数（CPI）同比上涨0.4%；工业生产者出厂价格指数（PPI）环比上涨1.6%，同比上涨4.4%。从4月1日起，中国大陆很多行业都出现了涨价潮，包括金属铜、铝、不锈钢、线路板、运输费用等等，这些都出现了三成以上的涨幅。

回顾文贵先生的告诫：独裁国家经济出现问题，民生银行、包头银行、哈尔滨银行、盛京银行、许家印倒了也不会倒，一个个往上积累，是三角形的，所有这些问题导致银行烂账增加，高度政治的绝对集中，市场高度绝对垄断，掩盖所有市场的危机性，问题问题问题，把问题往上推，都聚集到中央，这是集权经济导致的问题。现在看来，独裁中共的经济问题已经出现，银行烂账、坏账率已经达到40-50% ，这个时候，中小企业没有现金流通，没有贷款，无钱可贷，社会虽然存在大量房地产、矿产、世界富豪，但这全是死钱。这个国家一下子成为集合性灾难经济体，当集中到最上面的时候，就将是经济崩塌式的脆断。
