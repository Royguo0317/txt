![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/25225844/WhatsApp-Image-2020-10-24-at-08.09.25-2.jpeg)



摘要：波布林斯基是郭文贵先生2017年爆料革命三年以来，第一位在美国公开站出来呼应的爆料者。在2020大选最后一场辩论会前召开新闻发布会，配合执法部门后，他旋即接受了FOX台的专访。采访他的就是两次连线闫丽梦博士的卡尔森。

2020大选最后一场辩论前日，奥巴马在费城发表挺拜登演讲，诬陷川普串通中国云云。几天后路德节目惊爆亨特·拜登在白宫和奥巴马密谈，奥巴马勾结CCP卖美求财的内幕。奥巴马也许知道自己贼喊捉贼，可在拜登家族的压力下，硬着头皮也得上，还得如他出道以来的一贯作风装作义正言辞。奥巴马编的故事，譬如川普和中国私通，在中国开有银行账号，给中国交税这些鸡毛蒜皮之事，就算自身已被深度蓝金黄的主流媒体，非常配合地纷纷启动了所谓通俄门2.0的宣传，也仍然提不起选民的兴趣([世纪大案 17：贼喊捉贼](https://gnews.org/zh-hans/441412/) )。大料在川普手中。

10月22日，田纳西纳什维尔的贝尔蒙特大学(Belmont University, Nashville, TN)，最后一场辩论开始。辩论固然万众瞩目，而川普带来的一位嘉宾则更有意义。他就是美国海军退役中尉、亨特拜登在SinoHawk Holdings的合伙人、公司CEO波布林斯基(Tony Bobulinski)。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/28045655/imrs.gif)波布林斯基在大选辩论现场；来源：The Washington Post
严格地说，波布林斯基是郭文贵先生2017年爆料革命三年以来，第一位在美国公开站出来呼应的爆料者。在辩论会开始前，波布林斯基就召开新闻发布会，通过声明的方式把自己的荣誉和拜登家族的腐败进行了切割，指出了一些已经掌握的拜登家族的犯罪事实，并宣布将把证据交给执法部门。

次日，波布林斯基就此事与参议院委员会成员会面，并和FBI约谈，而FBI实质上已经架空了变节的雷，相关案件的指挥权已经在朱利安尼手上。

在奥巴马-拜登美奸集团还没反应过来的时候，FOX台名嘴卡尔森(Tucker Carlson)预告，10月27日晚他收视率极高的栏目今晚卡尔森(Tucker Carlson Tonight)将专访波布林斯基。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/28045746/tucker-carlson-facebook-instagram-misinformation.png)卡尔森采访爆料人闫丽梦；来源：FOX
生于加州，从CNN到MSNBC，现在跳到FOX台，卡尔森的立场越来越鲜明地呈现出保守派特征。2016年底开始，卡尔森主持FOX台热门栏目今晚卡尔森。2020年9月15日和10月6日，他在今晚卡尔森节目中采访了从香港大学P3实验室出逃美国的冠状病毒爆料人闫丽梦博士。第一次采访揭露了冠状病毒并非来自天然，而第二次采访适逢川普总统感染冠状病毒，该节目成为美国三大新闻网全天收视率最高的节目，其收视率超过同时段CNN和MSNBC的总和。

10月27日晚，今晚卡尔森节目如期开播。综合波布林斯基近日的公开言论，他谈到2017年5月2日，在亨特(Hunter Biden)及其叔叔吉姆(Jim Biden)的要求下，他和老拜登(Joe Biden)在洛杉矶的见面。拜登家族请他吃饭，展示出他们的强势作风，希望波布林斯基出任合资公司的CEO。这家公司由中国能源企业CEFC和特拉华州的Oneida Holdings合资。

“他们全家卷入此事，他们非常清楚他们在干什么。和一家中国公司做交易，从CCP那边得到了强力的财务和政治支持。”

波布林斯基还谈到拜登家族内部的互相不信任。在他和老拜登在一个停车场的第二次会面中，老拜登直接告诉他“盯着他儿子(Hunter)和兄弟(Jim)”。

波布林斯基谈到“现在有前海豹突击队队员保护他的家庭”，“他感到不安全”，“他收到了死亡威胁”，“如果必要我不得不在接下来的四年中躲避追杀。”
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/10/28031203/Unknown-10.png)FOX台今晚卡尔森专访波布林斯基；来源：FOX台截屏
波布林斯基的采访引爆了公众舆论。OAN记者、班农战斗室的嘉宾波索比克(Jack Posobiec)发推透露：波布林斯基的采访刚刚结束，参议院多数党领袖麦康奈尔立即给川普总统打电话。显然这件事引起了共和党高层的重视，当晚川普在内布拉斯加的奥马哈(Omaha, Nebraska)的竞选集会延迟。之后在竞选集会上，川普总统直接谈到波布林斯基在今晚卡尔森节目的采访，“他用第一手的资料说明副总统拜登直接地、亲自地卷入了和中国的腐败交易，并从中获利。”

至于拜登阵营，官方则毫无反映。有趣的是，一位自称来自拜登竞选团队的内部成员随即爆料：

“没有人谈论这个事情，但从他们的脸上可以看到答案。拜登的竞选结束了。… … 拜登不会赢得选举。我们打了很多电话，也登门拜访，很多人都让我们马上就滚。”

拜登取消了大选前的所有活动深藏起来。他看上去已经不需要美国选民的支持，他也无法面对美国选民的质疑，他放弃了美国选民，美国选民还有必要不放弃他吗？他是在等待CCP发招，用选举舞弊、刺杀等阴谋术帮他胜选，还是在打点行装准备跑路？我们静待11月3日大选。
