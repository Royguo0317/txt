超限战是一个普通人难以理解的概念。

一说起战争，大家脑海中显现的可能都是战争片中的炮弹轰轰轰，子弹啾啾啾，士兵啊啊啊。

但是随着人类迈入现代社会，人文、经济和科技的飞速发展也让战争发生了根本性的改变。

战争的核心已经从消灭转变成了消耗。简单点说，就是打死，不如打伤。

打死不如打伤这个概念，可以从现代地雷的设计上初见端倪。

早期的地雷设计都是越大杀伤力越好，恨不能一炮炸死一个连。现代的反步兵地雷，反而刻意降低了致死杀伤力，能炸断一条腿，就不炸死一个人。

为啥呢？

因为一个伤员会给敌方造成巨大的压力。你又不能放着他不管，得有同伴救助下前线吧？得有医疗兵抢救吧？得占用后勤资源吧？得占用医疗资源吧？

这些伤兵痊愈后的赡养也是巨大的负担。伤病员回国后可能会激发更多社会问题和民间的反战情绪，逼迫政府停战。

这可比一个死人要耗费的资源多太多了，比一个死人能造成的影响多太多了。

有了这样一个简单的理解，我们再看看现在中共发起的这场新冠病毒超限生化战。纯粹从战争角度去看，它简直妙不可言。
1. 隐蔽性高
2. 传播性好
3. 有效期长
4. 摧毁经济
5. 耗费国力
6. PPE与疫苗要挟
7. 可定点刺杀

简直是完美犯罪，打了你，还要你说爽；打了你，还要你感恩；打了你，让你有口说不出。

兵法有云，上兵伐谋，其次伐交，再次伐兵，其下攻城。

战争的目的是让敌人屈服。如果有办法不战而屈人之兵，不是上上策吗？

只是这样的超限生化武器太阴毒，只有毫无人性可言的中共魔鬼可以想到、可以做到。

要破解这个超限生化战，唯有爆料革命的唯真不破。

超限战所仗者，唯秘密二字。只要揭露中共的阴谋，用事实说话，用科学说话，用情报说话，真相会警醒世人，世界的正义力量绝不会放过中共。

如果没有文贵先生和爆料革命，这世界会是什么样子啊！

武汉封城的惨状将在全世界上演，感染人数、死亡人数会比现在高N个数量级。各国经济崩溃，甚至政权崩溃。
