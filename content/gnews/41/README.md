#### **川普即将把包括中芯国际在内的更多中共公司列入贸易制裁实体清单意味着什么？**

川普决定要对于中芯国际等公司列入制裁名单，上次是国防部的，这次是贸易部的制裁。12月5日，中芯国际，已经被国防部列入了制裁清单，中国军方控制的企业。现在这个是直接列入实体清单，除了中芯国际还有另外80个企业，还有和中国军方有关系的，还有侵犯人权的企业。这样就是对生态链进行全部的禁运。原来是军事的高端技术不能给中芯国际，现在是所有的美国技术都要进入这个管理规定。原来是高技术，现在可能普通的和民用级别都会列入黑名单，也许几十纳米技术的都不能出售。这个制裁就是低技术的都不能出口给这些中资企业。

这次已经列入了实体清单，意思就是和这些企业有相关联的企业和行业都会收到制裁。这是非常切实的措施。这是美国对军民融合项目的全面打击，中国的企业就是以民为皮，以军队为核心，如果只打击军用是不能达到目的的。只有打了民用就是抽他们的血。但是这就意味着巨大的潜在失业率，中国老百姓给军民融合当了炮灰。

12月3日制裁了中芯国际， 12月10日就有人购买了中芯国际股票的人起诉了中芯国际，寻求经济补偿。墨博士说: “只要危害国家安全的财产都是可以没收的，很多的中共的钱都回不到中共的手里。他们在美国勾兑的很多钱拿不回来，他们非常着急。中共的上层人物是最着急。”

这些制裁和限电都有关系。国内大面积停电。全频都在说这个事情，江西，湖南，都停电。中共的理由是因为现在天气比较冷，南方都是用电取暖，所有就限电保护老百姓的取暖。主要原因可能是因为不能进口澳大利亚的媒，因为澳大利亚的媒的燃烧率比较高，而国内的媒的燃烧率比不了澳大利亚的媒。中共说有序用电，这就是他们的计划经济的表现，先官再民。中共的百分之70的点是来自煤电，减排就可能导致老百姓挨冻。限电是中国经济崩溃的开始。计划经济以后会是各种有序，有序用粮，有序用电，各种有序。有序出行，有序出行。
