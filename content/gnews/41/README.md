05月04日，俄罗斯卫生部长米哈伊尔·穆拉什科（Mikhail Murashko）在与各地区举行的打击中共病毒会议上说，中共病毒疫情仍然很紧张，因此地区当局应根据流行病学情况自行决定是否放宽限制。

穆拉什科强调，重要的是继续形成集体免疫力，为中共病毒高风险群体的公民接种疫苗。

他指出：“老年公民和患有心血管疾病和糖尿病的病人一直并仍然是我们的首要任务。”对于这样的患者，穆拉什科建议在住院治疗的患者出院前接种疫苗。

1月18日开始对俄罗斯人进行中共病毒的大规模疫苗接种。副总理塔季扬娜·戈利科娃（Tatyana Golikova）宣布，截至4月23日，超过1100万俄罗斯人已经接种了至少一剂疫苗，而680万俄罗斯人已经接种了两剂疫苗。

4月30日，俄联邦消费者权益保护和福利监督局（Rospotrebnadzor）负责人安娜·波波娃（Anna Popova）称俄罗斯的中共病毒形势稳定。然而，据她说，如果俄罗斯人在5月假期期间停止采取预防措施，情况可能会变得更糟。该机构建议在乡下或在乡间别墅与家人一起过节。

据该部门称，一天之内在俄罗斯发现了7.7万个中共病毒的新病例，另有337人死亡。自疫情开始以来，受感染的病例总人数达到480万，其中11.1万例死亡。
