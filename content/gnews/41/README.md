加拿大总理特鲁多(Justin Trudeau)政府工业部长表示，加拿大需要在中共问题上与五眼联盟建立统一战线。

加拿大国会下议院议员，特鲁多内阁中先后出任联邦国际贸易部长、基础建设及社区部长、外交部长、和创新﹑科学及工业部长商鹏飞 (François-Philippe Champagne)刚刚修改了加拿大对外国投资国家安全审查的规定。他强调与五眼联盟中的美国、英国、澳大利亚和新西兰合作的重要性，以协调五眼联盟对中共威胁的应对方式。合作的重要领域包括关键矿产和敏感技术的政策，比如是否禁止华为技术有限公司(Huawei Technologies Co.)参与下一代无线基础设施建设等。

本月，加拿大与美国和英国一起，就虐待维吾尔族穆斯林问题加入了欧盟(EU)的行列，对中共实施了制裁。

“这不仅仅是加拿大的问题” ，商鹏飞强调，“这是真正的西方民主正在经历的时刻。这就是为什么我觉得拥有相同价值观和原则的国家渴望合作。”

加工业部长表示，新的外国收购审查规则关注的是技术和关键矿产方面的“国家安全问题”，并不是针对某一个特定的国家。但他表示，对美国和其他西方盟国不需要那么担心，并补充说，加拿大“张开怀抱”欢迎投资很重要。

到目前为止，加拿大是五眼联盟中唯一没有禁止华为设备进入5G网络的国家。这也是特鲁多政府在政治上最敏感的问题之一。商鹏飞表示，“当涉及到国家安全时，你不能走捷径。”
