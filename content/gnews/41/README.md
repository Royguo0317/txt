## 以色列似乎证实对伊朗核设施进行了网络攻击

据《卫报》 作者：Martin Chulov，2021年4月11日报道：

以色列似乎证实了对伊朗核设施进行了网络攻击。
![]()![](https://gnews.org/wp-content/uploads/2021/04/37E302D8-506B-4F63-99EE-FDE66220C46A.jpeg)*在纳坦兹反应堆的新离心机启动数小时后，就发生了宕机事件。纳坦兹核设施的卫星图像。照片：Maxar Tech/AFP/Getty.Maxar Tech/AFP/Getty*
以色列似乎证实了周日对伊朗主要核设施进行网络攻击是幕后黑手的说法，德黑兰的核能负责人称这是一种恐怖主义行为，应该对肇事者作出回应。这次明显的攻击发生在纳坦兹反应堆的官员重新启动可加快浓缩铀生产的先进离心机数小时之后，这被称为伊朗核计划的关键时刻。

在伊朗当局争分夺秒地处理纳坦兹的大规模停电时，以色列国防部长阿维夫-科查维说，该国“在中东的行动并没有瞒过敌人的眼睛”。以色列没有像以往类似事件发生后常做的那样对报道进行审查限制，以色列媒体广泛报道了这起明显的袭击事件。公共广播电台不走寻常路，声称摩萨德情报机构对该事件发挥了核心作用。以色列总理本雅明-内塔尼亚胡周日晚些时候表示，“努力打击伊朗及其代理人和伊朗军备是一项巨大的任务”。他补充说：“今天的局势不一定将是明天的局势”。但他没有详细说明。

这场不明原因的宕机事件被认为是这两个宿敌之间一系列交锋中的最新一次，十多年来，他们在整个中东地区进行了一场广泛而不断升级的影子战争，其核心是伊朗的核计划及其对境外事务的参与。

最近的冲突更多的是在公开场合进行的，有对航运的打击，有对伊朗首席核科学家的处决，有对叙利亚境内的伊朗代理人的数百次空袭，甚至还有当地官员声称是环境破坏的以色列北部神秘漏油事件。

纳坦兹一直是以色列担心的焦点，去年7月发生的爆炸破坏了一个离心机组装厂，2010年中央情报局和摩萨德联合使用名为Stuxnet的计算机病毒进行网络攻击，造成大范围的混乱，并将伊朗的核计划推迟了数年。

伊朗核主管阿里-阿克巴尔-萨利希敦促国际社会和国际原子能机构（IAEA）对袭击者采取行动。他证实，一次“恐怖袭击”破坏了纳坦兹核电站的电网。原子能机构表示知道这些报道，但拒绝进一步的评论。萨利希说，这次袭击是由“反对国家工业和政治进步的人实施的，他们的目的是阻止核工业的蓬勃发展”。

伊朗议会能源委员会发言人Malek Chariati称这是破坏行为。Chariati说，“在伊朗努力向西方施压要求解除制裁之际，在国家核技术日到来之际发生的这一事件，强烈怀疑这是破坏或渗透行动。”

这些事态的发展正值美国总统拜登准备重启一项争论激烈的协议，以提供制裁减免，来换取德黑兰限制其核计划和不寻求发展核武器。2015年的协议，是巴拉克-奥巴马政府的外交政策核心，但很快被他的继任者唐纳德-川普取消，他转而采取了一种强势的姿态，以扼杀伊朗的经济，同时支持中东地区的其他美国盟友。

美国国防部长劳埃德-奥斯汀（Lloyd Austin）周日抵达特拉维夫，部分原因是向持怀疑态度的以色列官员重申华盛顿的新立场，他们担心即使是缩减伊朗计划，也会为制造能够到达地中海东部的核武器提供掩护。

在会见奥斯汀后，以色列国防部长本尼-甘茨说，“我们将与我们的美国盟友紧密合作，确保与伊朗的任何新协议会符合世界的重要利益，美国的利益，防止中东地区危险的军备竞赛，并保护以色列国家安全。”

对纳坦兹的袭击是在以色列在红海对一艘伊朗货船进行明显的水雷袭击5天后发生的，西方情报官员一直声称这艘货船是用来支持也门战争中德黑兰支持的胡塞武装的指挥和控制船。这艘名为“萨维兹”号的货船被至少一枚水雷严重炸毁，水雷在水线下引爆。附近的沙特阿拉伯海岸警卫队都收到了该船多次发出求救信号。这次袭击是几年来各国在区域水域对航运进行的一系列报复性攻击中的最新一次，其中大部分攻击都没有得到承认。此后，以色列在叙利亚进行了一系列空袭，破坏了大马士革附近的一个军事基地，据称该基地被忠于伊朗的代理人使用，为黎巴嫩民兵和政治强国真主党提供支持，真主党仍然是伊朗外交政策的重要分支。

以色列去年打破了对叙利亚8年空袭的沉默，承认对约1000次袭击负有责任，以色列称这些袭击主要是为了防止真主党在黎巴嫩境内的初级火箭弹上安装先进的制导系统。

以色列对叙利亚的打击对该国已经被十年的起义，和战争蹂躏的军事基础设施造成了广泛的破坏，并推动了以阿拉伯联合酋长国为首的外交努力，向叙利亚总统巴沙尔-阿萨德施压，以切断其与伊朗的联盟，这有助于他继续担任领导人。尽管有几位值得信赖的安全官员的敦促，也有俄罗斯的支持，俄罗斯也在确保其政权安全方面发挥了作用，但阿萨德拒绝了这些示好。代表伊朗提供军事力量的真主党仍然强烈反对这样的建议，高级官员担心这样的重新定位可能是为了最终迫使其与宿敌进行和平谈判。

西方官员认为，以色列在试图破坏伊朗计划方面变得越来越明显，并指出去年11月该国主要核科学家穆赫森-法赫里扎德（Mohsen Fakhrizadeh）被杀，他与其保镖一起在一条乡村公路上被枪杀。伊朗声称，人工智能被用来识别法赫里扎德的身份，他是被遥控自动武器枪杀的。随后，载有该武器的小卡车发生爆炸。
