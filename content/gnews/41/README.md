**西班牙2021年1月7日电/西喜社—**

按照中共党史的记载：

1926年1月国民党在广州举行第二次全国代表大会后，蒋介石夺取了国民党中央的领导权。随后蒋制造了“中山舰事件”。1927年，共产党员与国民党左派，挫败了蒋介石在南昌另立中央的企图，蒋在上海发动了反革命政变，汪精卫也背叛了“国共合作政策”。1928年，朱毛在井冈山会师，全国各地的革命群众，为了反抗国民党反动派的白色恐怖和残酷的屠杀政策，先后发动了一百余次武装起义。1929年、1930年，中共组织了若干次武装起义，建立了若干个“根据地”，建立了若干个“苏维埃政权”。

在这五年里发生了很多事儿，被中共称为“民贼独夫”的蒋介石先生“粉墨”登场。然而和中共党史中那大义凛然（卑鄙无耻）、义正言辞（臭不要脸）的记载完全不同的真实历史又是如何呢？

上篇曾经简述过，1923年孙中山先生去世后，汪精卫成为了广州国民政府的主席。（背景介绍：孙去世后政治重任便落在了汪精卫与胡汉民身上，而两人分别为国民党的左派和右派领导人。当时的蒋介石任黄埔军校校长，掌握着军权。孙的亲信廖仲恺为党代表，一切军事命令与规章都要经过党代表的连署才可以生效。）

同年8月廖仲恺遇刺，大约15名国民党执行委员会和监察委员会右翼成员在北京城外西山于11月23日举行中央执行委员会第四次会议，号召从国民党中清除共产党员。而广州的国民党左派以不足法定人数为由，召开了自己的中央执行委员会第四次会议。而此时的共产党已逐步完成了国民党内部权力的渗透，共产党员不但在检查委员会中赢得了新的席位，在执行委员会的九人常务委员中，三人是共产党员，三人是共产党的支持者。国民党中央党部九个部中有六个部受共产党控制：组织部、宣传部、工人部、农民部、海外部和青年部。（各位读者在此有没有“一样的配方，一样的味道”的感觉？）随即，西山会议派（国民党右派）在上海建立自己的党部，标志着与被中共渗透的广州党部分裂。

1926年，被中共宣传的由蒋介石制造的“中山舰事件”，是蒋与国民党内的这些“共产主义者们”分裂的转折点。几十年来，国共两党对“中山舰事件”始终各执一词。中共坚称此事件为蒋介石制造的“反革命事件”。而国民党记录此事件为中共的“阴谋叛乱”，称之为“蒋中正救党”。本文不对“中山舰事件”做过多评述，只提供两点供读者参考：1）中山舰舰长李之龙为共产党党员；2）中山舰事件发生后，当时的中共党魁陈独秀、张国焘等人一再道歉赔礼。

“中山舰事件”发生以后，蒋介石采取了一系列针对共产党员的限制性措施。而那时寄生在国民党内部的这些共产党员们，一切的行动都要听从共产国际，即斯大林同志的指示。在蒋忙于“北伐”的同时，斯大林指示中国共产党员要在国民党内部强化政治工作，改善军事知识，以待军中要职。（在政治权力渗透的同时，共产党已然盯住了国民党的军事权力。）这时的武汉政府（1927年1月随着北伐战争胜利，国民政府迁至武汉）已在国产国际代表鲍罗廷与国民党左翼领导人汪精卫统治之下。

在“北伐”取得不断胜利时，蒋似乎看破了共产党在国民党内部不断渗透夺权的狼子野心。1927年4月开始了“清党”运动，与此同时他公开了与莫斯科的友好关系，显示其只是局限于清除本土的共产党员。

被共产党完全渗透的武汉国民政府，于1927年4月17日解除了蒋介石国民革命军总司令的职务，随即蒋在胡汉民的帮助下组建了南京国民政府。从此，武汉的共产主义政权和南京的国民政府两个权力中心的裂痕成为了不可逾越的鸿沟。

1927年6月，斯大林的一封电报，要求在武汉国民政府的共产党员组建武装力量，重组武汉政府。这一消息被汪精卫得知后，汪终于意识到共产国际的真实意图是要摧毁国民政府，把武汉政权变成共产主义傀儡。之后汪便在党内和政府各职位内驱逐共产党员。1927年8月1日，共产党员假借国民党左派名义，在南昌发动起义。（ “八一建军节”无耻的由来。）汪精卫最终决定全面清洗共产党员。

在共产党在国民党内部的渗透夺权阴谋被识破后，武汉与南京之间的所有分歧都解决了，蒋被重新任命为国民革命军总司令，汪精卫也制定了出国计划。1928年武汉政府解散。1929年，蒋介石结束了十三年的内乱，统一了中国。

这五年中，中共寄生在国民党之中，费尽心机渗透夺权，充分利用国民党内部的“左右之争”分化、瓦解。在狼子野心败露之后， 还发动了武装起义（南昌起义）。这便是你在国内学不到看不到的“五年”历史，这便是中共百余年来不断粉饰，不愿告知国人的真实历史。
