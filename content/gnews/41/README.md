**西班牙2021年1月22日电/西喜社——**美国加利福尼亚州报告的中共病毒单日死亡人数创下764人的纪录，但新感染率正在下降。

加州公共卫生局周五报告的死亡人数超过了1月8日创下的708人的上一个记录。在过去的两天里，加州已经记录了1335例死亡病例。

不过，住院人数和新确诊病例一直在下降，卫生官员们越来越乐观，认为最新激增的最坏情况已经过去。

周五报告的23,024个新病例不到12月中旬高峰期近54,000个病例的一半。住院人数已降至2万以下，两周内降幅超过10%。
