## 美国国会议员抨击WHO对中共病毒源头的洗白报告
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/1042791/)

纽约香草山写作组   四季

据《零对冲》Zero Hedge 3月29日的报道，共和党代表李·泽尔丁（Lee Zeldin）抨击世界卫生组织是中国的“有用的白痴”，因为该组织对冠状病毒爆发的起源的调查草案被披露为“极不可能”地否定了实验室泄漏理论。
![]()![](https://gnews.org/wp-content/uploads/2021/03/图片31-1.png)
泽尔丁抨击中国“向世界掩盖疫情的起源”，并呼吁进行真正的独立调查。他在推特上说，自COVID-19疫情爆发以来，中共一直向世界撒谎，掩盖疫情的来源。世界卫生组织一次又一次地扮演着中共有用的白痴的角色。一场彻底的、真正独立的调查早就应该进行了。

上周，特朗普总统领导下的美国疾病控制和预防中心前主任罗伯特-雷德菲尔德博士宣称，他认为该病毒源于中国武汉的一个实验室。这个理论得到了美国和英国的国家安全高层官员以及数十位病毒学家和其他科学家的支持。

德国汉堡大学的一位著名科学家在今年2月份公布了一项长达一年的研究结果证实，冠状病毒大流行最可能的原因是武汉病毒学研究所的泄漏。

冠状病毒是利用一种叫做“弗林蛋白酶切割位点”（furin cleavage site）的特殊工具嵌入人体细胞的。一份新的科学报告显示，在自然界的1000种冠状病毒中，与导致COVID-19新型冠状病毒最相似的，没有一种冠状病毒拥有类似的“弗林蛋白酶切割位点”。这表明，该病毒不可能来自于自然。

中共病毒已经肆虐全球一年多了，我们感受到了无尽的封锁，病毒对人类的摧毁，以及疫苗的伤害。世卫组织的报告重复了中共一贯的宣传，即病毒很可能是由动物传染给人类的，而不是来自实验室。希望更多的媒体和有良知的科学家，病毒学家和政治家能够不断的揭露真相，追责中共，尽早还世界安宁。
