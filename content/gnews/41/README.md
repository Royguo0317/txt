最后，闫博士继续警告说，“如果这次放过中共政权，他们会继续在动物身上寻找新的病毒，希望搜寻损害你们大脑的病毒。”现在事实证明，目前感染者中有三分之一的人大脑出现了问题。最后，闫博士呼吁世界上所有医务工作者和政客们明白发生了什么。并强调，“我定义超限战是因为这是没有限制的。最终的目的就是摧毁你们的经济，摧毁你们的社会秩序、摧毁你们的文明。你们都将受控于中国共产党……那时候，我们将没有未来，没有安全，没有子孙，什么都没有。”正如闫博士所发出的一连串的警告！这些实实在在的威胁已经发生了，这些是发自一位良知科学家内心肺腑之言。“世界都希望这是中共最后的时刻。”（见《郭文贵》）相信没有人愿意一生中在口罩的约束下生活。

**四、媒体新星的现场演讲均高喊灭共**

**结论：新中国联邦成为扭转乾坤不可忽视的力量**

新中国国联邦周年庆典——一场自由、团结、灭共的盛会就要闭幕了，最后。郭文贵先生在家用视频连线方式向庆典纽约主会场发表庆典闭幕演讲：“今天站在前台的所有人是踩在那些默默无闻的战友们的肩膀上，更重要的是，中共国14亿同胞，看到我们今天的行动，他们该如何面对？”我们是人，不是生来就是奴隶，“我们要反抗，我们要斗争，但我们要不流血的斗争，我们要文明的斗争。我们要没有暴力的抗争。”社交媒体就是夺取自由的武器，郭文贵先生向所有支持者很好地诠释了“爆料革命”即非暴力、非流血的革命。这完全颠覆了古今中外对“革命就要流血、就要暴力抗争”的认知，特别是中国人，也向戊戌变法的六君子谭嗣同他们诠释了新型革命方式，即爆料革命人士以揭露统治阶级黑暗、联合世界上一切可以联合的力量来抗衡，以此最终实现推倒极权的目标。

从受邀的美国贵宾摩根夫妇及其子女、弗林将军、比尔格茨军情作家、战略家班农先生、军情届大佬劳伦斯·塞林先生等人以实际行动支持爆料革命来看，实现新中国联邦终极目标志在必得。特别是劳伦斯·塞林先生，在推特上，以病毒溯源的的方式展开战斗，并且在塞林先生推下，聚集了无数无名中国英雄们与其并肩作战，将中共生物武器研发人员名单及其网络关联图都已了如指掌。在真正收网之时，它将起到一网打尽的作用。

可以说，新中国联邦周年庆典又可以看作是爆料革命人联合美国力量向中共发起第一轮反击的战斗庆功大会。这个庆功大会对中共打击力度毋庸置疑。事实证明了这一点，这是一次历史性的局面扭转，因为美国共和民主两党在讨伐中共问题上达到空前一致。正如灭共领袖郭文贵说，“我们没有战友们的努力。不可能有法治基金，不可能GTV、 Gnews和喜币，不可能有今天的局面等。”

在历数中国共产党的邪恶程度，郭文贵先生以震惊的爆料口吻说，“共产党最爱吃的就是人肉，所谓的胎盘。他们吃胎盘的时候，从未想象这是一个母亲创造的生命，是人类的根本和未来。共产党远的不说，从计划生育杀掉的人，六四杀掉的英雄们，和香港杀掉的英雄们，以及病毒在全球杀掉的人，数以亿计无辜的人被他们杀害。” （见《闭幕讲话》）这些控诉绝非危言耸听，甚至每一个中国人都知道，那是公开的秘密，首先，人们必须承认食用产妇胎盘算不算“吃人肉”？如果你否否定这是事实，他肯定也吃过。计划生育杀人，那是实实在在的，婴儿三个月就有人的形象。有的计生干部将即将分娩的孩子强行堕胎，活生生的现场，公开杀人。而运动式杀人更是屡见不鲜，是中国人可以否认的吗？最显著的例子，在香港被中共杀掉的一万七千多人，有谁可以否认吗？

郭文贵继续警告说：“这绝不是最黑暗的时候，黑暗绝没有过去，人们要在黑暗中走很长很长时间，如果我们迷失方向、如果我们信念不坚定、如果我们迷失了，明年六四的时候，我们中这些人还会不会在这里？明年的六四，我们还有谁会站在这里？”这是一个令人不安的警告，根据郭文贵先生的警告，中共制造的人间炼狱似乎刚刚开始，这将考验每一位参与爆料革命的战友，因为，这取决于“我们将以什么样的成绩面对32年前天安门六四中被残酷杀害的同胞？”正如郭文贵先生说的那样，“这32年（中共对同胞的杀害）从来就没有停止过，它同样发生在西藏，发生在新疆，发生在内蒙古，发生在香港，生在全球各个国家（病毒的猎杀）。还发生在中共国各种大大小小的运动中。”面对中共的杀戮，郭文贵先生鼓励地说，“战友们，我们不要骄傲，我们要看着明天升起的太阳，想想我们该干什么？”（见《闭幕讲话》）这是一个切实的忠告，中共在窃取政权，从八年抗战到三年内战，他们以不断杀戮异己而存活、壮大，而新中国联邦人以自由之精神，做好了内部以任何形式的背叛的心理准备，按郭文贵先生所言，这些都不见怪。问题是，任何一个组织和团队，或集团公司，管人性如何面对人性的弱点，除了需要管理智慧外，还在乎培养每一位参与者如何在逆境中坚持吗？如何在背叛中成长？能理解吗如何及时识别敌我关系？管毕竟会智者千虑必有一失，这就需要每一位参与战斗者同样拥有果敢处理情感的智慧，因为中共杀人诛心的各种招数恐怕是越来越隐蔽，甚至挑起夫妻之间、父子之间、母女之间的仇恨，叫你在做抉择时生不如死。恐怕这些因素才是郭先生需要爆料革命人引起警惕的要点。

任何一支优秀团队的建设，都必须要基于一种发自内心的彼此赞扬、彼此包容、彼此接纳和彼此举荐，总要看别人比自己好才是一个不愧“新中国联邦第一代公民”身份，好成为我们新中国人效法的榜样。为此，郭文贵语重心长地说：“我们心中必须有战友，必须有包容。我们明天或许出现无数个九指妖，出现背叛我们的人，这一点都不稀奇，只要我们够强大，今天哭的最难受的人，就是反对爆料革命的人和共产党。但我们最终是实现我们新中国联邦这唯一的目标。就是干倒共产党。”最后，郭先生提议，全体起立，向新中国联邦周年庆典主会场，向911死去的灵魂们默哀30秒钟。愿3000多位灵魂安息，因他们证明了自由是伟大的。为此，郭文贵先生向全球75亿人发出祝福。祈愿人类和平安宁。（见《闭幕讲话》）这就是郭文贵先生心中有同胞，有爱。郭文贵从2017爆料以来，追随他的人如雨后春笋般地、根连着根四海蔓延，不断增长，他的追随者们用实际行动证明了郭文贵正是符合中国文化里所言的那样，可谓达者兼济天下也！
