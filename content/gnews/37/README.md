在1月世界纪念国际大屠杀纪念日上，联合国敦促每个国家通过制定预防未来种族灭绝的方案，向600万犹太纳粹种族灭绝的犹太受害者致敬。联合国仪式上，美国和欧洲领导人与大屠杀幸存者一道，传达了捍卫真相记住大屠杀的紧迫责任，并承诺“绝不会”允许种族灭绝在我们的手上发生。

1月份美国国务院认定种族灭绝是中国共产党（CCP）所为：“有计划地摧毁新疆的维吾尔族和其他少数民族。” 暴行包括：“通过拘留，拘留营，软禁和奴隶制系统监禁超过一百万平民；强制性的人口控制措施，包括强迫绝育，强迫流产，强迫节育；强迫维吾尔妇女与非维吾尔人结婚，以及对礼拜，集会，言论和迁徙自由的严格人权限制。” 国务院以这种令人难以忘怀的比较得出结论：“第二次世界大战纽伦堡法庭以危害人类罪起诉肇事者，而在新疆也犯有同样的罪行。” 结束新疆种族灭绝的道德要求是我们时代最统一的两党问题之一。由参议员鲍勃·梅嫩德斯和约翰·科宁提出的一项决议，指定了中共在维吾尔族侵犯人权是种族灭绝的行为，得到了压倒性的支持。

这场针对中国穆斯林的种族屠杀令人震惊地联想起纳粹大屠杀。在其灭绝的前奏中，纳粹将犹太人非人道化。他们称犹太人为老鼠，剥夺了他们的人权和合法权利，并将它们像牛一样放进集中营。今天，中共官方的录音记录显示，党派人士将维吾尔族描述为“恶性肿瘤”，将其信仰与“传染性瘟疫”进行了比较，并说：“您不能一一将在田间作物中的所有杂草除根，您需要喷洒化学药品以杀死所有这些物质。” 最近，令人不安的图像显示成排的剃秃头，手铐的维吾尔族人跪在地上，等着被困在火车上，然后运往中国380个集中营之一。在纳粹大屠杀时，我们已经看到了这一点。怀疑中共针对维吾尔人的行为是否符合种族灭绝的定义？众议院议长南希·佩洛西召集了我们将问题定格，她说：“如果美国出于某种商业利益而不反对在中国侵犯人权，那么我们将失去所有道德权威。” 国会议员迈克尔·麦考尔表达了摆在我们面前的紧迫性：“我们不能袖手旁观，让国家支持的种族灭绝文化彻底消灭整个文化，这一做法如果继续下去，我们的沉默将是同谋。”
