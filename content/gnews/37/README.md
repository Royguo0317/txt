## 上合天道下合人心的中华传统教育体系（四）

我们现在中共国都整成宫斗了，都是宫廷剧， 整一堆故事，听故事去了， 对我们没啥意义。古人很重要的一个观点，古代没有历史学家一说，他们都叫经史学家。 古人认为学史之前必先学经，否则不能学历史。 因为学经有两个目的，第1个就是开大智慧，否则你想那12岁小朋友他怎么能够去当大国的宰相呢？第2个培养贤良的人格，培养一个好的人格， 经就两个作用，一个就是提升德行，一个就提升智慧，让我们成为有大德行，大智慧的人， 但是可能不会灵活应用。所以接下来再去学历史， 就会灵活应用了。 学了经再学史，就看得懂历史了，就不是只是看故事，而是看到王朝兴替背后他的规律，能透过现象看本质。

如果不学经，只学史，像王岐山这种，他也是历史系毕业的， 他会学历史里面那些坏家伙怎么样的坑蒙拐骗的， 别人学好的，他专门学坏的。 在中国历史上很多的大奸臣，他们都是很聪明的，很精通历史的，从历史中借鉴坏人的经验，王岐山就这样子的，他就是没学经。毛魔头更坏，他专看历史里面的垃圾，什么《金瓶梅》、什么《三国演义》，净看那些怎么奸淫掳掠，怎么坑蒙拐骗，怎么阴谋诡计，他就学这些玩意， 这家伙是人类有史以来最坏的一个。再接下来学子，子就是各种技术，诸子百家里面什么技术都有，兵家就是怎么打仗的； 阴阳家是看风水的； 纵横家是谈判的，搞公共关系的； 农家就是怎么种地的； 经济家就搞经济的； 诸子百家都在子的范畴里面，他只是工具而已， 各种实用技术。最后再去学集，集就是文学艺术修养， 包括诗词歌赋，琴棋书画都在集的范畴。 诗词歌赋无一不精，琴棋书画无一不通，厉害吧，这才是我们中华民族讲得培养的人才，我们就培养这种精英。

我们今天在学校里设置的所有课程，都可以归入这四个类别。 语文就在子；数学也是在子；物理化学也是子；体育还是在子；英语也是子；我们虽然也有历史课，但是不讨论案例，学历史，只是讲故事，最多也就是子，还有很多家长让孩子去学画画、音乐、弹钢琴，这是集，我们今天现代教育体系的所有的内容不是在子，就是在集， 经史完全不懂。 西方可能有历史探讨，最多也就到史， 但他没有经的这个智慧。

中国为什么培养不出人才， 就是由于失去了根本， 抓的都是细枝末节，子和集都是细枝末节。经必须要有高手指点，否则都不知道。
