2020年8月，习近平公开发表了“对舌尖上的浪费公开说不“的指示。2021年4月29日，十三届全国人大常委会第二十八次会议表决通过反食品浪费法，自公布之日起施行。从制定到修改再到公布，历时不超过十个月，习大神的一句话，就要用法律使其上升到一个高度，背后反应的又是什么内情？

为何要突然谈粮食浪费？最直接的理由，和粮荒有不可切割的关系。首先，2020年，全球经历了病毒肆虐，其次，下半年对于中国来说，又是几十年难遇的水灾和蝗灾，第三，中共与他国外交关系僵化，贸易遭受重创，可用于进口粮食的外汇捉襟见肘。第四，中共国的“种子“也严重依赖海外，一旦断供，就是卡脖子的问题。这些都是造成未来可能引发粮食危机的直接导火索。因此，官方为了撇清责任，转移矛盾，就火急火燎地制定出了这部法律。那这部法律说了什么？

首先，“餐饮服务经营者可以对造成明显浪费的消费者收取处理厨余垃圾的相应费用。收费标准应当明示。“这条表明，店家可以以浪费为由，变相涨价，而且涨多少，没有唯一标准，由店家自己决定。

但既然没有标准，那么如何收费，收费合理与否谁又能说了算呢？厨余垃圾收费的具体标准，连政府都没有给出过所谓的指导服务价格，那么就这样转嫁给消费者，是否公平合理？

法律还规定：“诱导误导消费者超量点餐最高罚1万元。“那么诱导和误导的证据如何提供？而超量到底以什么量作为参照数量？这两点也成为了执行之中的难题。

网络上对于该部法律的正式内容，并没有完全公开，可查询的资料也是在今年一月用以征求意见的草案，通读整篇内容，最直观的感受是文字的表述太过模糊，对具体执行层面，存在难度。何为浪费？每个人对于食物的需求都是因人而异的，不能单纯用人数或者年龄来界定所需食物的数量，那既然无法用统一的用餐标准，如何定义统一的浪费标准？这个前提就没有答案。

所以通篇的法律草案只有对于情节的描述，但缺乏惩罚的依据。如果从积极面来考虑，说明国家意识到粮食的危机在日后可能爆发，所以用法律来提前给人以警示。但中共做事一向不会没有目的，更可能的是，中共希望用一部模棱两可的法律为以后的自由裁量打下伏笔。也就是说，我要说你浪费，你就浪费，因为我有法可依，而制定的标准，看当时情况，自由变动，也无需通知你。

这才是这部法律真实的意图。我们可以预期，粮食危机应该在前方不远处等待，最后几个字最重要：囤粮、囤粮、囤粮。
