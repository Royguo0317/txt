据美国Axios新闻网近日报道，美国要求各国披露其用于影响美国的活动经费的法案开始收到成效。根据美国“应对政治中心”（Center for Responsive Politics）的数据，中共支付其外国代理人的费用已从2016年的略高于1000万美元，增加至去年的近6400万美元，增幅高达6倍。

报道指出，国营的中共新闻机构《新华社》是最新披露其在美国境内业务的机构。《外国游说观察》（Foreign Lobby Watch）最先报道称，尽管美国司法部三年前就通知《新华社》要按《外国代理人注册法》(the Foreign Agents Registration Act)做登记，但《新华社》在美国的分支机构于上周才正式注册为外国代理人。该法案原本是为了揭露纳粹组织在美国的宣传而制定的。在该法案之前，中共的媒体机构是不受法律限制的，更没有被要求披露其结构和财务状况。

《新华社》最初只把自己登记为仅受中共政府监督的一个独立的合法实体。但是实际上，《新华社》是中共所有，由中共的高级官员直接管理，更是中共对外宣传的主要喉舌。美国“应对政治中心”的数据显示，到2020年，中共实体在外国代理商活动上的支出，比任何其他国家都多。新的外国代理人注册记录显示，自2020年3月以来，中共《新华社》的母公司向其美国分公司支付了860万美元，包括直接向华盛顿，洛杉矶，休斯敦，旧金山和芝加哥等分支机构付款。中共的支出主要是给了《中国日报》和《中国环球电视网》等国有媒体，他们合计占2019年中共《外国代理人注册法》登记支出总额的三分之二以上，到2020年达到五分之四以上，超过了电信巨头华为和监控技术公司海康威视。

报道指出，大外宣是中共实现其国际地缘政治野心至关重要的手段。中共通过在美国传播社会动乱和虚假信息的行动，对美国的商业，政治和社会环境产生了非常直接的影响力。
