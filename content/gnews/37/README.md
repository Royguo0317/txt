## 【香港风云】8: 黑云压城

**作者：纽约香草山文艺组 Tiffany的早餐 校对/发稿：飞虹**

![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/20165916/logo-3.jpg)



摘要：林郑月娥继梁振英任行政长官，上任后立即实施广深跨境高铁西九龙站的一地两检方案。郭文贵先生最早爆出以警治港的祸港四人帮，以及以一地两检为代表的，共产党和香港建制派在大湾区的一系列准备工作，全部的目的都是为了日后大规模镇压香港民主运动。

2016年12月9日，梁振英宣布放弃竞选连任，之后CCP给了他政协副主席的虚职。面对他任期中香港政改的汹涌民意，他退到了后台。梁振英任内对北京卑躬屈膝而不得民意的政务司司长林郑月娥参加了2017年选举。香港民主运动人士之前多年围绕2017年选举所做的种种努力，得到了林郑月娥这样的结果，香港的民主化进程陷入低潮。

曾荫权的入狱意味着香港政治中立公务员系统的崩溃([香港风云 3：曾荫权入狱](https://gnews.org/zh-hans/590953/))。在CCP的步步进逼中，香港的公务员系统全面开始姓党。梁振英当然是实质上的党家的奴才，可表面上还是以港人治港的形象出镜。到了林政月娥，从还没有做行政长官之时，已经完全是奴才丫鬟样，在她的任内CCP抓紧控制香港直至血洗香港。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/26115500/phpXxTr9v.jpg)习近平、梁振英、林郑月娥三家
图右林郑月娥配偶林兆波是唯一公开抵制习近平和香港建制派的人物；
SARS之后CCP向香港的渗透速度逐渐加快，其中一种重要的渗透方式就是单程证，即前往港澳通行证。单程证由中国公安部门派发，每日限额150人。持有者将获准在港合法居住七年，到期后可申请永久性居民身份证资格。香港每年的人口增长在数万人，其中大多数是持单程证定居香港的新移民。

单程证的制度是畸形的。单程证的审批权不在香港而在大陆公安，大陆让谁进香港，香港就得放行。在这样的政策下，大量中共高官亲属、富商巨贾，特别是带着特殊任务的情报人员渗透到香港。年复一年，十几年来有数十万人被塞进了香港，深刻改变了香港居民的结构。CCP对香港社会方方面面的控制，一点一滴地开始收紧。

2017年7月25日，郑月娥新政府上任不足一个月，即在行政会议上通过和宣布了争议多时的广深港高速铁路“一地两检”方案。在次年第三季通车、全球造价最贵的广深高铁路上西九龙总站共10.5万平方米的站内设大陆口岸区，占西九龙总站面积的四分之一，以租赁方式租予大陆。大陆口岸区连同营运中的高铁列车车厢范围内，实行大陆法律，由大陆法院行使司法管辖权，而中国大陆的执法人员在区内享有全面执法权。香港港铁主席马时亨辩解说，“没有一地两检，高铁只会被视为速度快一点的直通火车而已，效益将大打节扣”。

随即社会上引起广泛争议，不少市民忧虑大陆人员会跨境执法；担心实行一地两检会损害《基本法 》一国两制原则。经过了2015年的铜锣湾书店事件，香港人已经对CCP对香港司法系统的渗透有了清醒的认识(香港风云 6：铜锣湾书店)。
![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/26115820/bbc25dc8d8508bb936fc2564507f8776.jpg)一地两检图示 图片来源：《明报》
按照香港的政治制度和刑事司法制度，香港政府三权分立，行政分支分为三司(政务司、财政司、律政司)、决策局、部门三层。三司之一的律政司是检控机关；保安局为决策局之一，由政务司领导；警务处是保安局下设的部门。郭文贵先生多次指出以警治港的祸港四人帮，这四人给日后香港大规模镇压民主运动做好了准备工作：

- 行政长官林郑月娥：“林郑月娥上来以后，废除香港当时学习的英语，英国人在的时候的教科书，然后全面开始来学习大陆共产党的这个恶教育。然后改变了出入境管理条例。林郑月娥几十次进大陆所谓的各种理由访问北京，然后到广州参与各种事情都是勾兑去了。”
- 律政司司长郑若骅：“郑若骅是律师界出来的。几年前，僭建的风波是把唐英年干掉了。但是郑若骅上来，僭建风波没事。”
- 保安局局长李家超：“出现在香港最多的是国保头子孙力军。在香港和孙力军接轨的是保安局长李家超。”([世纪大案 7：中美执法合作](https://gnews.org/zh-hans/364009/) )
- 警务处处长卢伟聪(2019年11月19日起由邓炳强接任)。

![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/26120258/hBsu1V9QgnX17RbnRO81xBYxcTDnMIyz2Q2IgdkNiIE.jpg)祸港四人帮，从左至右卢伟聪、郑若骅、林郑月娥、李家超 图片来源：HK01
林郑月娥任内的一系列政策，大多是北京的主子和香港的奴才勾结，出卖香港框架的具体执行。其中最为黑暗的是为大规模武力镇压香港民主运动所做的各项准备。通过条例和基础设施方面的准备，CCP已经可以通过交通设施，通过公路、铁路、水路大批快速地运送武装部队入港，也可以方便地在香港绑架逮捕居民，直接转移到大陆，而无须铜锣湾书店那样复杂的情报机构的秘密行动，这样可以实现大规模的逮捕 。除了公开的项目，郭文贵先生还透露了许多秘密工作，涉及CCP所谓大湾区的港珠澳之间的配套项目包括：

- 高铁进九龙：往香港运送军人和后勤物资，从香港抓人去大陆
- 高铁大陆口岸区，九龙站一地两检：CCP在香港可以随便抓人，送到九龙站下便归大陆管
- 港珠澳大桥通车
- 驻港部队在赤柱大量扩建
- 在珠海准备了大量的水船
- 在深圳有上万辆克隆的香港警车、准备几十万套香港警服
- 在深圳宝安，有个一比一的镇压民运的演练场
- 在深圳珠海开了十几条生产线，供应维稳武器
- 防暴警察装备：纳米声弹、布袋弹、催泪瓦斯
- 驻港解放军总部作为指挥中心，华润大厦作为备用指挥中心，在中联办有情报中心。
- 修订《逃犯条例》
- 修改出入境条例和相关软件：大陆情报人员不用登记。
- 港版国安法
- 废除英国教科书


除此之外，郭文贵先生还爆料，京港的卖港贼已经在深圳和珠海的30万等待的大军和警察，一定会下狠手的。香港的地下道，不但能走车，还能走装甲车。香港的共产党控制的各大楼里边，藏着千军万马，他们会毫不犹豫地动武，而且会使用最先进的现代武器，来对付老百姓。

2014年9月28日，十年来第一次，香港警察对抗议者发射数十枚催泪弹，引发了120万香港人投入的雨伞运动。雨伞运动的口号是“一个蛋也不能碎”，呼吁抗议者和平抗议，展现法治社会的公民素质（[香港风云 5：雨伞运动](https://gnews.org/zh-hans/591818/)）。香港人太善良。在香港人还在寄希望于和平抗争推动民主时，共产党已经开始磨刀霍霍，准备一场史无前例的屠城。
