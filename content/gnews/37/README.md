# **美国高级海军上将说，中共国有一百万人在服务于宣传机器以破坏美国**

[!\[\]()!\[\](https://spark.adobe.com/page/mbWKGtJmt3cmf/images/37ce4539-ad0c-4bcc-901b-ddc8e0eb40db.png?asset_id=8255ed88-a109-42f1-b5c0-13d7a6d87e9d&amp;img_etag=%22f8fa0230d745d57df8e2f4cefce008b2%22&amp;size=2560)](https://spark.adobe.com/page/mbWKGtJmt3cmf/images/37ce4539-ad0c-4bcc-901b-ddc8e0eb40db.png?asset_id=8255ed88-a109-42f1-b5c0-13d7a6d87e9d&amp;img_etag=%22f8fa0230d745d57df8e2f4cefce008b2%22&amp;size=1024)2020年8月，美国印太司令部司令菲利普·戴维森上将在国防部长会议（CHODS）上。(美国海军照片：“大众传播专家”二等兵Anthony J. Rivera)

在周二参议院军事委员会的听证会上，美国印太司令部(INDOPACOM)司令菲利普·戴维森上将(Adm. Phillip Davidson)表示，“中共国有一个庞大的虚假宣传机器”，背后有一百万人，目的是破坏美国。

在回答参议员珍妮·沙欣（Jeanne Shaheen）关于中共国虚假信息对印太地区构成威胁的问题时，戴维森说：“他们利用常规媒体和社交媒体，在他们的宣传机器里有近百万人。”

戴维森说，中共国造谣努力的目标是“破坏美国的利益，为自己的利益制造故事，并以令我们的盟友和伙伴对美国可靠性产生怀疑的方式破坏环境。”

戴维森在评估中说，中共国的虚假信息宣传与俄罗斯在印太地区的虚假信息不同。中共国的行为是为了播撒怀疑，俄罗斯反而“表现得更像一个破坏者”。戴维森说，俄罗斯没有提出自己的愿景或宣传叙事，而是寻求“否认该地区其他国家的愿景”。

他说，美国印太司令部正在与美国国务院“深度接触”，以改善美国在该地区的信息传递，巩固美国的利益，“不是提供虚假信息，而是将突出其他国家在该地区的弊端”。

“虚假信息是新的地缘战略结构的一部分，使竞争变得如此困难。”戴维森说。

戴维森说，美国应该在“全球范围内”加大信息传递力度，并指出国务院在冷战时期在传播“真相”和对抗虚假信息方面拥有“巨大力量”。

在去年新冠病毒大流行的情况下，人们对中共国虚假信息工作的关注有所增加。欧盟外交和国防综合部门、欧洲对外事务部(EEAS)的信息分析部门2020年4月的分析发现，俄罗斯和中共国推动了关于新冠病毒起源的相似虚假信息叙述，并试图“利用公共卫生危机，常通过直接挑战欧盟及其合作伙伴的信誉以增进地缘政治利益”。

中共国的造谣活动试图否认新冠状病毒源自中国，甚至声称美国军方将病毒带到了武汉，而武汉是第一批病毒病例出现的地方。

中共国最近还流传了一张经过篡改的图片，描绘了一名澳大利亚士兵割开一名阿富汗儿童的喉咙，进一步指控澳大利亚士兵在阿富汗犯下的战争罪。其中一项战争罪指控是，澳大利亚士兵确实割断了两名14岁阿富汗男孩的喉咙，但内部调查并没有证实这些说法。另外，澳大利亚调查人员确实发现了“可靠的信息”支持关于2009年至2013年期间25名澳大利亚士兵参与谋杀39名阿富汗平民和囚犯的指控。
