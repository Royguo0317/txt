I. 闫丽梦博士接受路德社采访透露中共使用舟山蝙蝠病毒ZC45和ZXC21作为SARS-Cov-2骨架的原因。闫博士根据已知的探索发现，舟山蝙蝠病毒骨架的E蛋白为中共病毒提供了嗜神经毒性（neurotropism），这正是有些新冠患者出现脑部神经病变的原因。中共病毒是一种“超限生物武器”。

II. 在3月1日举行的中共国外交部例行记者会上，发言人汪文彬强硬回应美国国务卿布林肯2月26日反对任意拘押加拿大公民一事。布林肯26日的评论代表此前美中双方磋商失败，而中共明确司法主权不容干涉，更是代表无法与美方拥有谈判基准点。习和拜登彼此互不信任，双方矛盾已经激化。

III. Scott Perry 等多名众议员联名上书美国国立卫生研究院(NIH)，要求彻查NIH对武汉病毒所的资金支持，并且点名达扎克的健康联盟组织可能作为中间人。信件揭露NIH通过健康联盟组织秘密向武毒所提供资金支持。

2. 美国贸易代表办公室周一表示，拜登政府将使用“所有可用工具”来打击中共国的不公平贸易做法，并将优先处理针对维吾尔族和其他少数民族和宗教少数群体被中共当局强迫劳动事项。

4. 路透社周一报道了一份独立民意调查机构的报告，内容显示，近三分之二的俄罗斯人不相信本国的人造卫星V号疫苗，大约有相同数字的人认为这种新的病毒是人为制造的生物武器。
