![]()![](https://gnews.org/wp-content/uploads/2021/03/1-75.png)2021年1月20日，即将离任的美国总统唐纳德-特朗普在佛罗里达州西棕榈滩的棕榈滩国际机场降落后挥手。(Alex Edelman/AFP via Getty Images)
 

前总统唐纳德-川普周五在离任后的首次公开演讲中表示，他将以某种方式复出，但没有详细说明。

《观察者》文章称，川普在佛罗里达州西棕榈滩的川普国际高尔夫俱乐部就餐时发表了上述言论。

周三上午，在前往佛罗里达州之前，川普发表了类似的言论，称自己“以某种形式回来了”。

川普在乔-拜登总统就职前几个小时对支持者说，“我要告诉你们，这个国家的未来从未如此美好。我祝愿新政府好运，取得巨大成功。我认为他们会取得巨大成功。他们有基础做一些真正壮观的事情”。

川普告诉支持者，“你会看到令人难以置信的数字出现”–指的是美国经济–如果拜登政府不对他的政策加以约束。

然而，在执政第一天，拜登就采取行动撤销了川普关于移民问题的一系列行政命令，暂停了Keystone XL输油管，取消了川普的1776委员会等。

“我想让你们知道，我们发起的运动才刚刚开始。从来没有过这样的事情。”川普周二也在网上发布的一段事先录制好的视频中说。

同样在周五，川普前顾问斯蒂芬-米勒对《华盛顿观察家报》表示，“佛罗里达州确实是成为MAGA运动新总部的最佳地点，也是总统下一步努力的发射台。”米勒说，川普可能在政治上扮演一个角色。
