2021年3月31日，闫丽梦博士的第三篇科学报告闪亮登场，将中共制造并释放新冠病毒这一超限生化武器的事情更近一步揭露和证实，在世界各国引起强烈反响。

一石激起千层浪，曾为中共站台的所谓“专家”的日子变得不好过了。2021年4月1日，美国国会议员马乔丽·泰勒·格林向国会提交了一个新的法案，题为 “解雇福奇法案”。该法案意在寻求剥夺这个薪酬最高的美国联邦雇员和COVID-19 “之父 ”福奇作为国家过敏和传染病研究所（NIAID）主任的位置。

福奇从2020年疫情发生开始，所作所为可谓臭名昭著。2020年3月，他告诉美国人民 “不应戴口罩”，并补充说 “没有理由戴口罩”，而我们都知道戴口罩对预防新冠病毒传染的必要性。2020年5月，他帮助中共洗地，强烈坚持说病毒是来自于自然，并将实验室泄露说解释成“即便是病毒从实验室泄露，那也是先来自于自然，再进入实验室，然后从实验室泄露”，简直荒唐可笑至极。

福奇除了对新冠病毒的拙劣反应外，他还是功能增强型研究的长期倡导者和资助者。这种研究意在增加像新冠病毒这样的疾病的毒性和传播性。更重要的是，福奇与世界卫生组织检查员彼得-达扎克博士关系密切，而达扎克博士最近承认与中共有联系。福奇曾向达扎克的生态健康联盟组织输送了数百万美国纳税人的资金。生态健康联盟组织与武汉病毒学研究所合作进行各种形式的研究，福奇还多次在该组织的活动中发言。

福奇从1984年开始就担任NIAID主任一职。“解雇福奇法案”将有效地把福奇的工资从美国最高削减到0美元，而接替他的人则将由参议院确定并投票确认。该法案还将引发政府问责福奇，对其在2019年10月1日至2021年12月31日期间的数字通信和金融交易进行审计。
