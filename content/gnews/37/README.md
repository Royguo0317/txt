中共在国内外的恶性活动损害了美国利​​益，破坏了我们盟国和伙伴的主权。美国将采取所有可用的对策来，包括防止中华人民共和国（PRC）的实体机构利用美国商品和技术在内的一切恶意行为。今天的举动又一次表明了我们决绝的决心。

美国对一些破坏我们国家安全和外交政策利益的实体实施新的限制。具体来说，商务部将59个中国实体添加到其出口控制实体列表中。

**大规模监视，军事现代化和侵犯人权**

美国正在将四个实体添加到制裁实体列表中，因其通过向中共国政府提供DNA检测样本或高科技监视设备而在中國境內侵犯人权。我们敦促中国共产党尊重包括藏传佛教徒，基督徒，法轮功成员，维吾尔族穆斯林以及其他种族和宗教少数群体成员在内的中国人民的人权。

此外，商务部正在将另外19个实体添加到“制裁实体列表”中，因这些公司或机构涉嫌系统地协调和实施在十多个美国公司盗窃商业机密的事件，以促进中国国防工业园区的发展；或從事破坏美国打击核材料和其它放射性物质非法贩运的活动；或利用美国的出口来支持中国人民解放军和中共国的国防工业基地，其最终目标是超越他们认为是竞争对手的其它国家（特别是美国）。

**南中国海**

这些新制裁也打击了北京在南中国海的非法胁迫行为。美国商务部将“中国船舶工业集团公司”（China State Shipbuilding Corporation）下属的25家造船研究机构以及其它六个“为中国人民解放军海军提供研究，开发和制造支持或试图获取美国原产产品用来支持中国人民解放军项目”的实体机构加到制裁实体名单中。商务部还增加了五家中国国有企业，包括中国交通建设公司（China Communications Construction Company），因其在南中国海压迫索赔国的行为。
