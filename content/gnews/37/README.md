## 辉瑞急为中共病毒疫苗安慰剂组受试者注射疫苗动机可疑

据每日邮报（Daily online）2021年1月1 日报道，辉瑞在其官方网站的新年说明中称，将会告知所有16岁或以上的中共病毒疫苗临床受试者他们是被给予安慰剂还是疫苗，对于得知自己接受了安慰剂的受试者来说，他们可以选择在留在研究中的同时接受研究性疫苗。受试者现在将能够在3月1日之前获得他们的第一剂疫苗，这比最初预期的时间提前了几个月。

上述决定是在与美国食品和药物管理局（FDA）成员和一个外部顾问小组进行了数月的辩论之后做出的。两个小组都对辉瑞公司的 “揭盲 “计划表示担忧，称这可能会使继续收集赢得FDA对疫苗的全面批准所需的安全性和有效性数据变得更加困难。然而，安慰剂组受试者表示，如果不准许他们像普通公众一样接受疫苗注射，会将他们自己的健康置于危险之中。就连联邦政府 “战速行动 “负责人蒙塞夫·斯劳斯（Moncef Slaoui）博士也认为志愿者应该立即接受疫苗注射。

他在2019年12月23日对记者说：”我认为非常重要的是我们应该立即揭盲，并为安慰剂组提供接受疫苗的机会，而不管他们的风险程度如何。这些志愿者在帮助我们开发这些疫苗方面发挥了重要和关键的作用，他们应该得到奖励”。

根据辉瑞公司科学家的推文，已经有超过2500名接受安慰剂的受试者接受了第一剂辉瑞疫苗。

辉瑞公司和BioNTech公司研发的中共疫苗是使用病原体的部分遗传密码，称为信使RNA或mRNA，mRN编码的是病毒外部发现的所有棘蛋白蛋白，病毒通过这些棘突蛋白进入和感染细胞，从而使人患病或者死亡。

根据辉瑞公司的报告，在44,000名中共病毒疫苗临床受试者中，有一半的人被注射了疫苗和另一半是组被注射的是安慰剂。在临床试验中，如果受感染的人数明显低于安慰剂组，那么就认为疫苗是有效的。在此次临床试验数据中，该疫苗的有效率为95%。

评论：
辉瑞公司在疫苗三期临床结束后非常短的时间就计划对安慰剂组受试者注射疫苗，这确实比较蹊跷，也令人生疑。

众所周知，在疫苗研发过程中必须遵循安全、有效和保护持久的原则。而安全性在这其中又是重中之重，

在短期安全性考察中，临床科研人员可以对接种疫苗后会出现轻微副作用，如轻度发热和注射部位疼痛或短期的肿胀等进行观察记录。在此次临床试验中，已经有媒体曝出了疫苗接种者出现面部瘫痪、昏厥甚至死亡的案例。而长期安全性试验中，还需要对疫苗是否对生殖细胞有影响，是否会导致胎儿畸形和染色体突变等因素进行考察，而这需要几个月到几年时间的跟踪随访，目前这些可能还没有开展。如果此时对安慰剂受试者进行疫苗接种，因为缺乏对照组的数据进行比较，疫苗安全性将无从考察。

其次，对于疫苗的有效性考察，在临床上疫苗有效率是采用如下方法进行计算的：如果安慰剂组的中共病毒发病率是10%，而疫苗组的新冠患者发病率是1%。疫苗的免疫保护效果的计算式为（10%-1%）➗10%=90%。辉瑞的临床试验数据如下：
![]()![](https://gnews.org/wp-content/uploads/2021/01/4-13.jpg)图片来源：NYTIMES
通过公式：0.74% – 0.04% = 0.7%，

辉瑞公司计算出疫苗有效率 0.7%➗0.74%=95%。

但是要注意到，如果对比美国全国感染率，会发现参加试验的受试者的群体感染率显著低于全美平均感染率，可能由于各种原因，比如测试时间不足，参与者普遍卫生习惯较好，采样在地域上分布不均匀等，这都会导致测试对象感染率和整体感染情况还有较大差异。所以仅从目前透露数据来看，在样本上偏少，并且对于暴露在高浓度病毒环境，疫苗持续的有效时间等尚未做出进一步说明。

辉瑞公司在临床试验没有取得完整数据时候就匆匆揭盲结束临床试验，只能说明他们对自己研发的疫苗没有信心，并不是他们所声称的为受试者健康考虑。因为在三期临床早期揭盲，将会导致前面所有的研究功亏一篑，正常情况下没有从来没有一家药物公司会采取如此策略。

众所周知，此次的中共病毒为中共军方实验室研制的生化武器，在舟山蝙蝠冠状病毒骨架上通过基因工程技术加入各种不为人知的功能片段，以达到病毒潜伏期长、传染性强和致病性的特点，在没有得到完整的病毒研发数据的情况下，是不可能研发出真正安全有效的疫苗的。

世界上所有的大药厂为了获得高昂的利润和配合大重置计划，压制唯一对中共病毒预防和治疗安全有效的羟氯喹的使用，从而达到不可告人的目的，最终结果是他们多年积攒的声誉和利润付之东流，而他们的所作所为也必将接受正义的审判。
