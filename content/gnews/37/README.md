据2020年11月19日米德兰市（Midland, Texas）当地媒体MRT（Midland Reporter Teleglam）报道：日前，居住在米德兰市的牧师傅希秋（Bob Fu），以诽谤、跟踪、攻击以及其它缘由，起诉了郭文贵先生。报道中指出据信郭文贵策划了针对傅的抗议活动。傅因被怀疑是中共政府的间谍，而成为抗议的目标。

上周提起的联邦诉讼中，傅将郭文贵先生（Miles Guo）及其媒体公司（GTV Media Group）和另两名抗议人士列为被告。报道同时指出傅指控他们会为“伤害”或甚至“杀死”傅希秋而提供金钱奖励。

该诉状称，郭文贵先生愿意向个人支付高达10,000美元的费用，以支持其对傅的“袭击”。据述，有联邦政府官员9月28日陈述傅受到威胁。

据诉状称，从10月4日开始，郭文贵先生的追随者开始在傅的家前抗议，并向傅的邻居散发传单，指责傅是中国共产党（CCP）的特工。

该诉状称：除了煽动针对傅的抗议外，郭文贵先生还在GTV网站上发布了虚假文章，指傅是“假牧师”、中共间谍、贿赂米德兰市官员并犯下其它罪行。

报道中提到，这些指控是源于傅所声称的郭文贵先生在其社交媒体账户所发出的威胁。诉状称，郭文贵先生告诉他的追随者傅和其他人是“环球大骗（global kill cheater）” 运动的一部分，并指示他的追随者“灭（get rid of）”傅。

报道指出，10月21日，米德兰警察局的炸弹小队紧急待命声称郭文贵先生告诉他的追随者去炸毁傅的家。

自10月初以来，傅和家人一直在米德兰郊外的安全屋中受到执法人员的保护。诉状称，在执法部门的建议下，傅以及他的妻子和两个孩子住在四个不同的地方。

傅的律师休斯顿Lanier律师事务所的劳伦斯·威尔逊（Lawrence Wilson）称：“傅博士是基督教事工和宗教自由的国际公认发言人，这种对其声誉的毁坏和鼓励对他施加暴力的行为必须停止”； “他数十年来在他的祖国和全世界致力于人权和民主的奉献，绝不能遭受这些攻击”；现在，这场诉讼将把郭文贵的行为和动机公之于众”。

据称，那些抗议者要么是郭文贵先生的跟随者、要么是郭的雇佣者；他们在经过近一个月的抗议后，于10月底离开了傅的家。据称，傅之前告诉记者说，他之前收到的信息表明抗议者将于10月底前返回。

诉状称，郭文贵先生在社交媒体上说：“一旦我们的同志们来到你的房前，我们将永不撤退。在把你“搞定”之前，我们将不会开启另一场抗议。”
