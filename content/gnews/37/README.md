## 美国的对中共国新战略：打击核心利益？

原文：韩国某人士

2021年是中国共产党建党一百年，党员总数达919万4000人（按去年底统计），目前世界最大的政党。考虑到中国总人口为14亿3932万4000名，相当于每16名中就有1名共产党员。 1921年7月1日建党时党员只有53名，但到1949年建立共产政权时党员增加到了449万名，2018年末已突破了9000万名。 82%的党员是在中国开始改革开放后的1978年入党的。 另外，20世纪80~90年代以后的出生者也超过了三分之一。 拥有大专以上学历的党员占一半以上，达到4661万5000名。 中国是本着”以党治国”原则的党国体制（Party-State System，中国共产党领导国家）的国家。 共产党掌握着国家的一切权力，他不仅掌握了三权（立法、司法、行政），还掌握了媒体。 这是根据共产党的意见，国家有条不紊的构造。 将中国称为一党独裁国家也是因为这个原因。

甚至在民间企业也有共产党组织。 中国共产党从2015年开始，包括民间企业在内的所有企业都有义务设立党组织。 党组织的作用是”监视”企业是否忠实地遵循党的路线，而且中国大企业的总裁和创始人也大部分都是共产党员。 世界最大的通信设备企业华为的任正非、中国三大信息技术企业百度的李彦宏、阿里巴巴的创始人马云、腾讯的马化腾、房地产开发及娱乐企业万达集团的王建林、中国最大的电动汽车公司比亚迪的王传福、小米的雷军等都是共产党员。

中国共产党根据金字塔式的领导体制行动。 共产党员中选出2287名党代表组成的全国代表大会（党大会）是党的最高领导机关。 通过每5年召开的党大会选出200多名中央委员，他们将成为领导党所有事业的权力核心。 中国共产党经常使用的”党中央”就是指这个中央委员会。 中央委员会由最高权力机关25人规模的中央政治局组成，从中选出7名政治局常务委员。 政治局常务委员兼首长的总书记总管国家所有业务。 另外，他还兼任形式上的国家元首国家主席和军队统帅中央军事委员会主席。 共产党总书记拥有如此强大的权力。

2012年11月18届党代会上选出习近平为中共中央总书记以来,他一直进行权利强化,‘皇帝’一词为”铁拳统治”。 习总书记在2017年10月19次党大会上无视中国改革开放的设计师的邓小平定下的‘(现任领导人、隔代指定一个时代之后的那下一个时代的领导人提名的方式)隔代指定’的原则，但仍对此不以为然。 邓退位时决定让继任领导人只能连任一次总书记和国家主席，执政10年，并制定了共产党的内部章程。 习总书记在2018年3月举行人大代表者大会(全国人大)上通过修宪禁止连任3任国家主席的条款删除，并把’习近平思想‘写入宪法等打造’1人中心的领导体制’的法律基础。 习进平将于2022年10月第20次党代会上修改党的章程删除禁止连任3任总书记的条款。顾名思义，他成了无所不为的独裁者。

习近平之所以能够登上所谓的”红色皇帝”的宝座，可以看作是实现中华民族伟大复兴的中国梦的野心。 习近平提出到2035年使中国进入发达国家行列，到2050年超越美国成为世界最强国的蓝图。习近平为了最大限度地延长执政期限到81岁的2035年，他只能维持绝对权力。 中国共产党人支持习近平实现《中国梦》，为了实现这个梦赋予习近平在毛泽东之后的绝对权力。 连中国最大的富人马云也强调因“1949年以来共产党一党专政,使社会稳定,这是中国发展的原动力”。甚至党员们也彻底支持习近平的领导，包括习进平在内的中国共产党人明确了西方自由民主主义制度在中国应该不是标准答案。再加上他们绝对信奉写入习近平思想的‘中国特色社会主义制度’。 他们还以中国国内生产总值（GDP）占世界15%的经济快速增长为基础，自信外交、军事等各领域超越美国。

**美国正面瞄准习近平**

与中国正式展开霸权之争的美国已瞄准已成‘恐龙’的中国共产党和‘独裁者’的习近平。 迈克蓬佩奥去年7月23日在尼克松总统纪念图书馆进行以‘共产主义中国和自由世界的未来’标题的演讲,演讲中宣布打到习近平和中国共产党是美国政府的新中国战略的指导。 在演讲中认定习近平为’破产的极权主义意识形态的信徒’、中国是‘掌握世界霸权的新极权主义的独裁国家’。蓬佩奥指出:”必须在与这一新独裁中国的斗争中获胜”，”如果我们不改变中国，中国就会改变我们”。 特别值得关注的是蓬佩奥将中国共产党与中国人分开，并呼吁中国国内反体制人士，香港和台湾的民主化势力与民主国家组成新的同盟为替换中共政权应积极站出来。这样来看,美国的‘打击中国’与11月大选结果无关,展望未来中国的经济也衰落,共产党即将崩溃。

**打着人权旗号谋求少数民族分离**

那么，美国推进的新中国战略的核心是什么呢？ 中国由汉族和55个少数民族组成。 除汉族以外，少数民族的人口只占总人口的8.5%，但他们的居住面积却占中国领土总面积的60%。 少数民族居住的地区不仅天然资源丰富，而且与邻国接壤的地方很多，因此对中国来说，在战略上非常重要。 因此，中国台湾、香港、新疆维吾尔自治区、西藏等地都被称为”核心利益”。 所谓核心利益指的是中国绝不妥协，不惜使用武力也要遵守的地方。 中国制定《香港国家安全法》强力镇压香港反中民主化势力，也是出于维护核心利益的考虑。 中共认为，如果香港维持民主主义体制并最终独立，那么新疆维吾尔自治区和西藏也将实现分离和独立，中共政权最担心的就是巨大的领土被分割出去。 可以说，中国提出”一个中国”原则，把与台湾的统一作为最大课题也是出于同样的原因。 美国的新中国战略是，提出少数民族的人权问题，最终图谋分裂中国。 如果美国政府在西藏和新疆维吾尔自治区等少数民族聚居的地区煽动与中国的分离独立运动，中共政权将陷入相当尴尬的立场。

实际上，2020年以来，美国为保护居住在新疆维吾尔自治区的维吾尔族的人权，制定了各种法律等，展开了针对中国的”要害”行动。 新疆维吾尔自治区人口2100万左右，但面积却占中国总面积的六分之一。 这里埋藏着中国全部石油和天然气储量的30%、煤炭储量的40%的能源宝库。 该地区自1755年被清朝吸收后一直受到中国的统治。 清朝灭亡后的 1933年、1944年在该地区建立两次东突厥斯坦共和国，但1949年被中国强制合并。 该地区的维吾尔族此前一直展开武装斗争要求从中国分离、独立。 中国为了防止这种情况发生，还建立了强制收容所，监禁了数百万维吾尔族人，还实施了洗脑教育。 美国政府2020年6月17日启动了由特朗普总统签署生效的《2020维吾尔人权政策法》(维吾尔人权法)。 美国国务院今年7月9日根据该法律，对新疆维吾尔自治区党委书记陈全国等共产党干部采取了制裁措施。 当时蓬佩奥发声明谴责“中国共产党的维吾尔族人权镇压传闻是世纪的污点”。

美国卫生部长是台湾问题特使

美国无视”一个中国”原则，试图承认台湾为”国家”，也是企图分裂中国的意图。 美国卫生部部长艾利克斯·艾扎在2020年8月9日至13日访问台湾并亲自会见蔡英文总统，事实上展开了特使活动。艾扎是美国政府与台湾自1979年断交后派遣的最高级官员。 美国政府在与中国建交后的41年里，一直回避与台湾进行正式交流。 艾扎部长访台以新型冠状病毒防疫合作的名义。其实美国警告中共政权美国随时会废除”一个中国”的原则。 蔡英文对艾扎表示:”台湾今后将在印度和太平洋地区发挥自由民主主义的坚固堡垒作用”，”进一步加强与美国的交流与合作。

在这种情况下，特朗普总统的亲信、前白宫首席战略家史蒂夫·班农和在美国、揭露中国共产党领导层腐败的房地产大亨郭文贵先生于2020年6月4日建立”新中国联邦(Federal State of New China)”。中国成功的房地产企业家出身的郭文贵先生，被邪恶的大陆公安定罪于很多假嫌疑的郭文贵先生于2015年去美国。 从2017年开始一直通过在线视频和社交媒体揭露中国核心政界和企业界精英之间的腐败勾结。 班农2017年8月离开白宫后与郭文贵先生展开习近平和中国共产党滥用权力等反中共调查活动。2020年6月4日，为迎接天安门民主化运动31周年，两人和在全世界各地的爆料革命战友们在纽约前海以自由女神像为背景发表了”新中国联邦”宣言。 他们的主张是将中国变成像美国一样建立联邦制。

新中国联邦建国运动

中国足球明星出身的郝海东积极参与这项运动。 在YouTube等社交媒体公开的25分钟视频中郝海东主张:”中国共产党的消失是出于正义的需要”，”应该诞生三权分立的新联邦政府，保障人权、自由、言论自由，加强与国际社会的合作。” “中国共产党的现有统治方式彻底藐视人权、违背 民主与法治”,强调“中国人民再也不能当共产党的奴隶”，一直停留在西班牙的郝海东还指责中国向全世界传播新型冠状病毒。 上世纪90~2000年代被称为中国最佳前锋的郝海东，在国家队A级比赛中创下最高得分纪录（39分），堪称传奇。 他还曾效力于英格兰职业足球冠军赛（乙级联赛）谢菲尔德联队。 他的夫人是20世纪90年代中国羽毛球明星叶钊颖。 足球英雄对体制进行批判后，中国共产党大吃一惊。 中国共产党随即关闭了郝海东的微博账号并把他的记录从正式文件中全部删除。

美国的交替中共政权和剔除习进平的新战略退出等成功与否是个未知数,但分离共产党和人民的 中、长期计划在某种程度上可以发挥作用 。美国克莱蒙•麦肯纳大学教授闵新培2020年5，6月《外交事务》上刊登的标题为《即将到来的中国巨变》的文章中主张:”中国共产党正在重复前苏联犯下的错误”，”美国在与苏联的冷战中获胜的因素是苏联共产党体制及其领导人的僵硬性。” 教授指出,“习近平独占权限和责任的情况下维持美中竞争的轨道，那么由此带来的矛盾和分裂在共产党内部将会显现。”。教授还说“习近平为了对抗美国继续挑起民族主义，但越挑起民族主义,共产党的选择越少，灵活的战略是更难”。” 随着共产党一党独裁体制的崩溃苏联分裂成15个国家。 中国共产党政权是重蹈苏联覆辙，还是今后继续维持一党专政，备受关注。
