I. 前美国国务卿迈克‧蓬佩奥周六出席全美保守政治行动大会（CPAC）表示，实力才能抵挡坏人，软弱什么都做不了。蓬佩奥在演讲中历数担任国务卿时，川普政府打破常规取得的成功案例。发言明确表示拜登政府已经不可能重回原先的怯懦政策，也在为保守派未来四年的行动方向定调。

II. 2月25日中午，一位投资人向21世纪经济报道记者透露，华西村遭“挤兑”。 中共媒体爆出华西村挤兑危机，习是想借华西村事件打击邓和江派势力，借此将中共国经济的危机转嫁给前任领导，缓解党内紧张的斗争气氛。

2. 周六，美国前贸易代表罗伯特·莱特希泽（Robert Lighthizer）参加了保守党大会有关反对中共国的小组讨论。他回顾了过去美国与中共国贸易关系史，表示美国需要在政策上更加反省。发言者还表示，中共政府实施了许多贸易操纵行为，其经济行为是掠夺性的。这位前官员表示，川普先生仍然是党内最受欢迎的人，也是不言而喻的领袖。他还赞扬川普政府”对付”中共国的做法。

3. 英国脱欧领导人奈杰尔·法拉格（Nigel Farage）指责英国政府将资金放在优先地位之上，而不是保护该国免受中共“新殖民主义”的侵害。

4. 有消息显示，中共半导体制造商在中美贸易紧张局势下急于生产本土产品，正在抢购二手芯片制造机，诸如光刻系统之类的核心设备的价格已经上涨了三倍。

5. 美国司法部2月26日发表声明，现年64岁居住在香港的中共国商人吴志龙于2月25日遭到起诉。他被指涉嫌在2017至2018年期间，串谋窃取通用电气（General Electric）的半导体技术及商业机密。
