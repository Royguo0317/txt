众议院议员报告发现，COVID-19起源异议被情报界 “压制”，众议院共和党情报监督小组最近的一份报告认为，与COVID-19全球大流行病有关的美国情报失误包括压制关于该病毒来源的不同意见。”我们认为情报界未能以及时的信息和分析来支持政策制定者。此外，有证据表明，其失败在大流行病的后续发展中继续存在，”少数人的报告在一项研究的附件中指出，该研究将中国的武汉病毒研究所（WIV）与大流行病的爆发联系起来。

报告说：”具体而言，共和党成员知道有指控称，[美国情报机构]压制了与大流行病起源有关的不同意见，而且该机构依赖’外部’专家，这些专家的纠葛和背景尚未被披露。该报告没有具体说明可能有冲突的外部专家。然而，报告特别提到了生态健康联盟，这是一个位于纽约的病毒研究组织，其主席彼得-达扎克是病毒非自然来源理论最直言不讳的批评者之一，该理论认为病毒不是来自动物宿主，而是武汉实验室的研究人员意外或故意泄露的。

生态健康联盟在冠状病毒监测和研究方面获得了超过1亿美元的美国政府资助。达扎克先生还在2020年英国医学杂志《柳叶刀》上组织了一封由一群科学家在大流行初期签署的信，批评实验室泄露理论是一个阴谋。该小组的共和党人还表示，人们担心 “美国政府直接或间接地与中国进行危险的科学研究，尽管知道中国军方的参与”。

报告说：”委员会必须确定[情报机构]在监测或评估与这些工作有关的美国政策风险方面发挥了什么作用，如果有的话。上周公开的这份20页的报告是基于该委员会获得的机密信息，作为其对17个美国间谍机构进行监督的一部分。报告认为，”重要的间接证据 “显示，该病毒很可能是从一直从事与COVID-19类似的病毒研究的WIV中逃脱出来的。

“为什么这很重要？这是关于生物武器。这是关于生物战的。这是非常令人担忧的，”加州众议员德文-努内斯（Devin Nunes），众议院情报问题常设委员会的共和党人，在上周发布报告时告诉福克斯新闻。报告说，一种相反的理论认为，病毒从蝙蝠跳到宿主动物，然后再跳到人类，或者直接从蝙蝠传播到人类，这种理论缺乏证据。

病毒来自实验室的证据包括中国从实验室泄露致命病毒的记录，包括与COVID-19相似的SARS或严重急性呼吸系统综合症的逃逸。

病毒逃逸理论还包括美国外交官在2017年发出的警告，即WIV在没有使用适当安全程序的情况下从事危险的病毒研究。

中国在WIV的军事研究也让人怀疑这次大流行是否可能与美国政府报告所说的中国 “有记录的生物武器计划 “有关。

人们已经知道，WIV的几位研究人员在2019年秋天出现了类似冠状病毒的症状。华尔街日报周末报道说，WIV的三名工人需要住院治疗，中国官员没有披露这些病症的细节。中国政府否认病毒是从WIV或任何其他实验室泄露的。相反，中国政府断言该病毒是由美国军方引入中国的，或者是通过冷冻食品包装进入中国的，大多数病毒专家认为这是不可能的。

美国国家过敏和传染病研究所所长、拜登总统的首席医学顾问安东尼-福奇博士最初批评了实验室泄密理论。但在本月早些时候刚刚得到关注的评论中，福奇博士说，现在他 “不相信 “病毒是自然产生的，并呼吁继续调查其来源。”我认为我们应该继续调查在中国发生的事情，直到我们继续尽最大努力找出发生了什么，”福奇博士在5月11日由PolitiFact组织的活动中说。”当然，调查的人说，这很可能是从一个动物库中出现的，然后感染了个人，但也可能是其他原因，我们需要找出答案。”报告说，助长对该病毒始于实验室的怀疑的另一个因素是，北京多次努力掩盖2019年12月在武汉开始的疫情的真实情况。

努内斯先生援引了情报界的一项指令，即分析应旨在防止报告中的 “偏见、政治化或其他问题”。

报告表明，情报机构严重依赖外部专家，这些专家误导了情报界，使其认为该病毒很可能是以自然发生的疾病爆发开始的。例如，国家情报总监办公室在2020年4月30日的一份声明中说，美国情报分析人员 “也同意广泛的科学共识，即COVID-19病毒不是人造的或转基因的。” 一年后的2021年4月，国家情报局局长艾薇尔-海因斯告诉国会，实验室泄漏理论和动物传播理论是美国政府的两个主要假说。在作证之前，许多科学家和他们在主流新闻媒体中的支持者都将病毒实验室泄密事件视为右翼鼓吹的阴谋论。

国家情报局的一位女发言人没有回复寻求评论的电子邮件。特朗普政府白宫国家安全顾问罗伯特-奥布莱恩(Robert C. O’Brien)去年夏天说，情报机构在病毒爆发初期未能认识到病毒的严重性。他在接受采访时说，2020年1月的情报分析人员 “仍在说这是类似流感的东西，没有SARS那么严重”。

报告指出，动物来源理论的证据很少，而实验室逃逸方案的证据更多。报告说：”尽管进行了密集的搜索并测试了超过8万只动物，中国的研究人员未能找到[COVID-19]可能跳出的原始蝙蝠种群或物种。共和党的报告说，最近的一篇科学论文 “表明该病毒有几个特点，如果放在一起，不容易用自然人畜共患的假说来解释”。

努内斯先生要求国家情报局的海因斯女士提供所有关于冠状病毒的情报，包括承包商和外部专家制作的报告，以及2021年1月国务院概况介绍背后的所有情报–这是第一份公开报告，揭示了将大流行病与WIV联系起来的证据，包括该研究所与中国军队的关系。

共和党人还在寻求对中国实验室正在进行的所谓 “功能增强 “研究的危险性的任何情报评估。功能增强工作涉及修改病毒，使其对人类更具感染性，这可能导致像COVID所产生的大流行病。据估计，该病毒已经感染了1.63亿人，在全球范围内造成330多万人死亡。

“[情报界]是否了解在WIV进行的危险研究的全部范围？有哪些……知识上的差距？” 努内斯先生在4月16日给海因斯女士的信中问道。

共和党人寻求的其他信息包括外国科学家与世界知识产权组织之间的合作细节，以及情报机构是否就这种合作产生了任何威胁报告。

无独有偶，报告发现中共干预2020总统大选同样被情报界 “压制”

在特朗普政府的最后几天，间谍机构分析师之间的情报压制被披露。时任国防部部长约翰-拉特克利夫在1月份指责美国情报分析人员淡化中国在干预11月3日总统选举中的作用，是政治化的表现。

“拉特克利夫先生在1月7日的一份报告中说：”从我作为可以获取美国政府关于中华人民共和国的所有最敏感情报的个人的独特优势来看，我不相信[情报界]分析师所表达的大多数观点充分和准确地反映了中国政府影响2020年美国联邦选举的努力范围。

拉特克利夫先生说，”中国选举影响报告的政治化 “是由于 “对那些根据情报提供另一种观点的分析人员施加了不适当的压力”。

情报分析家们反而强调了俄罗斯对选举的干预。

海因斯女士在1月份的提名听证会上发誓要防止情报工作的政治化。作为国家情报局局长，她承诺 “向情报界发出明确的信息，希望我们能够提供非政治性的、不加修饰的情报”。
