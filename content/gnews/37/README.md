**（1）中共操控电台发牌 王维基非爱国者**

三个免费电视牌照申请的审议由开始至2013年8月已超过1300日，惟政府、有关当局及苏锦梁一直只重复表示「会按《广播条例》及既定程式，完成审核3份申请，并向行政长官会同行政会议提交建议。有关申请现正由行政长官会同行政会议审议，当作出决定后，会尽快公布结果」、「行政长官梁振英会同行政会议，正在处理三个免费电视牌照申请，工作是有进展」、「问题复杂，需小心处理」等字眼，又不愿就发免费电视牌照定下死线，令公众怀疑政府是否不想发牌。

审批结果行政会议决定只发两个新免费电视牌照，并否决王维基的「香港电视」牌照申请，引起社会反弹和争议。

早在2008年，王维基刚入主亚洲电视，计划将亚视大改革成新媒体。他曾说：「如果亚视要变成大陆台，我没有兴趣。亚视是一个香港人的电视台。」、「即使有北面支持，我不会接受。」表明亚视不会成为「中央台」，更坦言「亲共的电视没有人看，只有反共的电视才有人看」，这些言论都被认为得罪了中共，亦引起了红色资本家的不满，因此大纪元时报认定不发牌给香港电视网络完全是政治审查和政治决定。

香港市民起初对增发免费电视牌照没有太大反应。但直至2011年中，城市电讯主席王维基大举向无线及亚视挖角，准备组将建香港电视网络电台，港人才开始关注中共背后操控发牌事件。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4098.jpg)图片：2013年时任特首梁振英在不交代理据的情况下不发牌予香港电视网络，封杀王维基电视王国。
**前官员及政界人士质疑不发牌有政治动机**

在政府否决香港电视网络的免费牌照申请后，不少前官员及政界人士均批评政府的决定。

前公务员事务局局长王永平指出，发牌是特首所作的决定，行政长官拥有最终决定权，因此行政长官梁振英有责任向市民解释政府的决策，他批评政府以保密制为由不解释是混淆视听。他质疑梁的决定有不可告人的秘密，又指事件亦已打击行政会议的公信力，令政府陷入严重管治危机，更会使大众担心将来的政改咨询会否与事件同出一辙。

前立法局及行政局首席非官守议员兼前全国人大代表李鹏飞批评，政府决策背后一定有政治因素，政府应详细解释发牌的理据。
![]()![](https://gnews.org/wp-content/uploads/2021/04/4099.jpg)图片：港共政府否决王维基的「香港电视」牌照申请，引起社会反弹和争议，当时有部份示威人士
诉求： 「拒绝黑箱政治」。
**( 2 ) 修改《警察通例》下「传媒代表」定义 变相引入「官方发牌制度」**

去年9月，港警加码打压新闻自由，当局修改《警察通例》下「传媒代表」的定义，变相引入「官方发牌制度」，赶绝公民记者及网媒。香港媒体环境更进一步迈向「大陆化」。

去年9月22日，港警向4个新闻工作者协会发信，表明将修改《警察通例》下「传媒代表」定义，不再承认只持香港记者协会或香港摄影记者协会会员证的传媒，变相限制公民记者丶网媒及学生媒体在现场采访，被指进一步侵害新闻自由。
![]()![](https://gnews.org/wp-content/uploads/2021/04/40994.jpg)图片来源：立场新闻
从2019年以后，港警不仅不知节制地在抗争地点肆意阻挠或禁止拍摄、公然羞辱、暴力对待乃至拘捕记者，要求他们包容谅解警队，甚或反斥大量「假记者」干扰执法或美化违法行为。

2019年10月，有传言指警方建议由政府新闻处统一发出记者证照，且构想早见于2014年雨伞革命。稍后特首林郑月娥虽矢口否认相关计画，但亲中媒体声讨「黑记」之声不绝于耳。

直至去年5月监警会公布反修例运动之审视报告，林郑回应称可拟定记者及警员共同遵守的工作守则，「希望为记者提供协助且不会对警方工作造成障碍」，较早前又谓学生、义务记者未受记者专业训练，质疑他们不足以从事现场采访工作；立法会议员梁美芬更打蛇随棍上，重提政府认证记者制度。

民权观察关注警方单方面就《警察通例》第39章「警察与市民和传播媒介的关系」的条文作出重大修订，包括（一）收紧传媒代表的定义，及（二）指明警员在不影响行动效率的情况下，才需要配合传媒和不妨碍摄录工作。

两项修订严重侵犯《基本法》及《公民权利和政治权利国际公约》所保障的新闻自由及采访空间，与筛选传媒无异，将对前线记者造成不合理的阻碍，并增加他们因为履行职责采访而要承担的个人安全及法律风险。
![]()![](https://gnews.org/wp-content/uploads/2021/04/40995.jpg)警务处处长邓炳强
据媒体披露，《警察通例》是警务处处长邓炳强根据《警队条例》第46条而制定，属邓炳强向警员下达的一般命令。民权观察批评邓炳强口称尊重传媒和新闻自由，但事实是「讲一套、做一套」，下令收窄采访空间。

有句老话：「平生不做亏心事，半夜敲门也不惊」。中共以假、骗治国，隐瞒着无数个不可告人的秘密，用更大的谎言去盖过之前的谎言。那些有良知、有理智的良心媒体和记者来对他们有如天敌，老共怎能轻易放过？
