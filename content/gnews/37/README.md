# 重要事件

- [** 04.17 The National Pulse: 国会要求福奇和中共傀儡彼特达扎克提供文件。**](https://thenationalpulse.com/breaking/congress-demands-ecohealth-docs-and-briefing/)


来自能源和商业委员会的众议院共和党人要求生态健康联盟提供文件和财务记录，该联盟是一个与武汉病毒学研究所合作的美国团体，并从福齐博士的国家过敏和传染病研究所接受美国税收。

4月16日的信是写给该组织的主席彼特达扎克的，信中提出了34个单独的问题，包括询问有关生态健康联盟（EHA）与武汉病毒学研究所（WIV）合作的文件。

“EHA在中国的蝙蝠冠状病毒研究方面有着广泛的历史，其中一些病毒被推测为SARS CoV-2的祖先。2 此外，EHA与WIV在这一研究领域建立了合作关系，WIV将EHA列为其八个国际合作伙伴之一，也是美国唯一的合作伙伴，”信中声称。”我们相信，通过其研究活动、合作以及EHA与WIV作为联邦奖励次接受者的关系，EHA拥有的信息和文件将为WIV的蝙蝠冠状病毒信息和该领域的进一步研究提供洞察力，”它补充道。

信中要求在2021年5月17日之前，从美国国立卫生研究院（NIH）获得所有与该集团联邦研究拨款相关的 “文件和信息”，财务信息，以及WIV研究设施的安全性。该请求是在几篇《国家脉搏》报道之后提出的，强调了EHA如何从Fauci的国家过敏和传染病研究所(NIAID)获得百万美元的拨款来研究蝙蝠冠状病毒，并多次接待他作为演讲者。

通过达斯扎克，EHA还率先推翻了前疾病控制和预防中心主任罗伯特-雷德菲尔德博士认为新冠病毒是从中国实验室 “逃出来 “的理论。《国家脉搏》还挖掘出了达扎克与中共广泛的经济关系。

“由于EHA有信心实验室泄密不是原因，我们希望你能欢迎有机会分享你所掌握的任何和所有与WIV的蝙蝠冠状病毒研究有关的信息、文件和专业知识，”信中补充说，然后要求 “向少数派委员会的工作人员介绍情况”。

- [** 04.18 Reuters: 蚂蚁集团为马云探索退出之道**](https://www.reuters.com/technology/exclusive-chinas-ant-explores-ways-jack-ma-exit-beijing-piles-pressure-sources-2021-04-17/)


据一位熟悉监管机构思路的消息人士和两位与蚂蚁集团关系密切的人士透露，蚂蚁集团正在探索创始人马云剥离其在这家金融科技巨头中的股份并放弃控制权的方案，因为与中国监管机构的会议向该公司发出信号，此举可能有助于在中国政府对其业务的审查下划清界限。

路透社首次报道了最新一轮会议的细节，以及关于马云对蚂蚁未来控制权的讨论，马云对蚂蚁的控制权是通过复杂的投资工具结构行使的。《华尔街日报》此前报道称，马云在11月与监管机构的一次会议上提出，将蚂蚁的部分股权移交给中国政府。

据熟悉监管机构思路的消息人士和其中一位与蚂蚁关系密切的消息人士提供的说法，央行、中国人民银行(PBOC)和金融监管机构中国银行保险监督管理委员会(CBIRC)的官员在1月至3月间分别与马云和蚂蚁进行了会谈，会上讨论了这位富豪退出公司的可能性。

蚂蚁否认曾经考虑过剥离马云的股权。”剥离马先生在蚂蚁集团的股权从未与任何人讨论过，”蚂蚁发言人在一份声明中表示。

- [**04.17 The Hill: 美国和中共国表示，他们 “致力于 “在气候变化问题上进行合作。**](https://thehill.com/policy/energy-environment/548861-us-china-say-they-are-committed-to-cooperating-on-climate-change)


美国和中国周六晚间发表联合声明，确认两国承诺在气候变化问题上进行合作，并 “加强”《巴黎气候协定》的实施。

根据美国国务院发表的声明，美国气候特使约翰-克里和中国气候变化特使解振华于周四和周五在上海举行了会晤。

经过在中国的讨论，两国同意就包括《巴黎气候协定》在内的多边进程进行合作。

声明中写道：”美国和中国致力于相互合作，并与其他国家合作，以应对气候危机，必须以气候危机所要求的严肃性和紧迫性来解决气候危机”。

据美联社报道，这两个国家是世界上最大的两个碳污染国。

两国同意根据协议，”作出这样的努力”，”将全球平均气温上升幅度控制在2摄氏度以下，并努力将其限制在1.5摄氏度以内”。

“两国回顾其通过领导和协作为《巴黎协定》的制定、通过、签署和生效做出的历史性贡献，”声明写道。

声明还指出，美中两国将在短期内采取行动，帮助应对气候变化，包括 “采取适当行动，最大限度地增加国际投资和资金，支持发展中国家从基于碳密集型化石燃料的能源向绿色、低碳和可再生能源过渡”。

“他们还各自承诺逐步减少氢氟碳化合物的生产。

各国的通知是在克里本月早些时候表示，他对中国愿意与其他国家合作实现碳排放目标 “没有信心，但充满希望 “之后发出的。

“……中国是其中非常重要的角色，”克里当时在接受《今日印度》采访时说。”我们希望中国能走到谈判桌前，发挥领导作用。习近平主席已经谈到了领导力，谈到了中国在其中的作用。”

克里表示，拜登总统承认中美两国会有分歧，但这 “并不意味着我们应该忽视摆在我们所有人面前的危机，这需要我们所有人共同应对，这就是气候危机。”

未来，两国同意继续围绕美中两国在2020年代可以做出的具体减排行动进行讨论。

这些行动包括 使工业和电力脱碳的政策、措施和技术；增加可再生能源的部署；绿色和具有气候适应性的农业和节能建筑等。

克里与谢的会面也是在下周白宫的气候峰会之前，包括40位世界领导人将虚拟地肉讨论气候变化。

白宫的峰会定在苏格兰格拉斯哥的联合国气候变化大会之前。

- [**04.18 Fox News：中共国指责美日 “拉帮结派”**](https://www.foxnews.com/world/china-accuses-us-japan-of-ganging-up-on-them)


中共国4月17日指责美日 “拉帮结派”，此前两国讨论了一系列地缘政治问题，包括中国的全球扩张。

日本首相菅义伟表示，在拜登总统上任后首次在白宫举行的当面会谈中，两国领导人讨论了 “严峻的安全环境 “和中华人民共和国（People’s Republic of China）在印太地区的行动。

两国的不仅谈到了中国在南中国海的军事介入增加和中国为向台湾施压而采取的安全措施，还谈到了新疆和香港的人权侵犯问题。

中共国外交部4月17日在一份声明中说。 “美国和日本实际上是拉帮结派，形成小集团，煽动集团对抗，”；”这种不合时宜的举动与该地区内外绝大多数国家对和平、发展与合作的共同愿望背道而驰。”

在冠状病毒大流行爆发后，中共国和美国之间的紧张关系一直在上升，但日本对他们的太平洋邻国也有自己的顾虑。

在双边会谈后的声明中，日美两国对中国在东海尖阁列岛周边的军事介入表示担忧。

这些岛屿又称钓鱼岛，无人居住，由日本管理，但由中国主张。

“台湾和钓鱼岛都是中国领土。有关香港和新疆的问题纯粹是中国的内政。”外交部说。”中国对南海诸岛及其周边海域拥有无可争议的主权。”

美日会谈是在22架中国飞机飞入台湾防空识别区后几天举行的，其中包括战斗机和具有核能力的轰炸机。

美国海军则增加了在台湾海峡的存在，拜登说，美国与日本一起，”致力于共同应对来自中国的挑战”。
