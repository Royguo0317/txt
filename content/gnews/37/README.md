# 内华达州爆料人在签字证词中称拜登有在货车里的选票制造工厂

[!\[\]()!\[\](https://spark.adobe.com/page/A52BV7n8CmLu9/images/60e66611-dfdc-4029-bfa3-7ffac66ac5ee.jpg?asset_id=c46613b7-6081-4d16-b309-ca6a3c8f6df0&amp;img_etag=%22c700980155efb7ba8a46201198177752%22&amp;size=1024)](https://spark.adobe.com/page/A52BV7n8CmLu9/images/60e66611-dfdc-4029-bfa3-7ffac66ac5ee.jpg?asset_id=c46613b7-6081-4d16-b309-ca6a3c8f6df0&amp;img_etag=%22c700980155efb7ba8a46201198177752%22&amp;size=1024)

拉斯维加斯，内华达报道—一位曾在克拉克郡（Clark County）担任选举工作的爆料人第二份宣誓担保证词显示: 邮寄选票是在投票站外的拜登-哈里斯竞选团队车辆内被不正当填写的。

华盛顿观察报（Washington Examiner）在11月8日获得了一份经过删节后的签字证词，显示先前由美国保守联盟 (American Conservative Union)主席马特·施拉普 (Matt Schlapp)所描述的并非空穴来风，他先前带头帮川普总统在内华达州提起法律诉讼。在一个周日的记者招待会上，施拉普说到有一名选举工作人员看到一辆“拜登-哈里斯”标识的箱车内有打开的选票。

该爆料人在10月17日到30日之间在选举站工作，他投诉拜登-哈里斯箱车经常停在选举站外面，并且经常有“演讲、跳舞、音乐和其他节庆活动”，尽管活动组织者被“数次”告知“保持在选举地点100英尺以外。”他也说到没有正规证件的投票人被允许进行临时投票。

但声明中最令人震惊的是，该声明证实了非法填写的投票属于可能当选的拜登总统支持者，(这是)爆料人在10月28日或29日一次午餐休息散步时看到的。

“我亲眼看到两个人交了许多没开封的邮寄选票信封给另外两个人，另外两个人把信封打开并且靠着拜登-哈里斯面包车的侧面就把选票填写了，”该证词说到。“那两个填写选票的人于是将被填写好的选票放入粉白色的官方信封。那些人不是投票站工作人员。”

证词继续说道：“在我散步的最后一圈，另外有约5至6个人组成了人墙，我走动他们也移动，很明显他们在遮挡我看那四个人打开选票信封，填选票和把选票放到粉白官方回邮信封。”

[!\[\]()!\[\](https://spark.adobe.com/page/A52BV7n8CmLu9/images/ee1f5549-6ad0-44c2-89a8-a322c9e1979f.jpg?asset_id=27dd3659-84da-4cc7-bf04-d709e540c5cf&amp;img_etag=%2280ddb5855118bbd5079ade3c70b7e8d2%22&amp;size=1024)](https://spark.adobe.com/page/A52BV7n8CmLu9/images/ee1f5549-6ad0-44c2-89a8-a322c9e1979f.jpg?asset_id=27dd3659-84da-4cc7-bf04-d709e540c5cf&amp;img_etag=%2280ddb5855118bbd5079ade3c70b7e8d2%22&amp;size=1024)

该爆料人向上层领导报告了该事件，领导随后打电话给克拉克郡选举办公室，但是没有回复。

“那边有人告诉他拜登-哈里斯车在那边搞的选票和信封，他们无能为力，”爆料人在证词中说。根据该证词，选举办公室补充道：“收集选票在内华达合法，” ，但是这个行为并不是爆料人所详述的。

“选票收集”指一个人收集然后提交已经填写好了的选票。在法律上，不在政府职能部门工作的人不允许给其他人分发邮寄选票或者填写好了的选票。

内华达州的克拉克郡，在一个计票中心工作的另一个工作人员签署了证词，爆料称一个主管人员指使他处理邮寄选票，同时无视选票签名是否和信封签名吻合。

川普总统在内华达落后拜登36，200票。
