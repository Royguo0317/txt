据《每日新闻》凯蒂·杰科维奇（Katie Jerkovich）3月1日报道，洛杉矶联合学区(简称LAUSD)公校机构与微软合作开发了”每日通”应用程序，以监控学生的中共病毒疫苗接种状态、症状和体温。

据福克斯新闻报道，洛杉矶联合学区总监奥斯汀·贝特纳（Austin Beutner）在周日的每周更新中说，”这个应用程序有点像’威利·旺卡’（Willy Wonka，译注：威利·旺卡是著名电影《巧克力工厂》中邀请小朋友参观其巧克力工厂的主人公）中的金票，每个人都可以凭借这张通行证轻松进入学校建筑。”

洛杉矶联合学区网站上的一则声明显示，该学区推出了一款专门为公校学区打造的一站式服务网络应用程序，以确保尽可能安全地将学生、教师和管理人员送回学校和办公室。

这则声明描述了如何使用该应用——“学生、家长和访客可以在任何个人电脑、平板电脑或移动设备上访问网址为dailypass.lausd.net的‘每日通’应用。回答每天的健康检查问题（只需不超过两分钟！）。如果一个人近期的中共病毒检测结果为阴性，并完成在线健康检查，那么该应用将生成一个二维码以供当天和特定位置使用。学校会有专门的’迎宾员’在每个位置扫描二维码并在学校入口处测量体温。”

洛杉矶联合学区网站还以动画片段展示了该应用程序。在视频中，一名学生因使用该应用程序获得”每日通”而可以返校。该通行证包括每日健康检查、每周中共病毒测试细节、以及为学生安排中共病毒疫苗接种。

尽管采用了新技术，洛杉矶联合学区承认该应用“不会检测到那些无症状的中共病毒携带者”。但学校官员表示，他们将通过对教师和学生的每周中共病毒测试来对付这个问题。

目前，洛杉矶联合学区开放学校目标日期定在4月9日。而自2020年3月起，南加州大部分学校就已经停课。
