中共的极权统治，向来都披着着各式各样的民主，法制外衣，许下的承诺无限美好，实行起来却是打翻在地，还要踏上一只脚。历史从来不会欺骗人们，但是悲剧却还是一次又一次的上演。这边厢，香港警方拘捕起诉47名民主派人士，罪名：“串谋颠覆国家政权罪”；那边厢，200多名香港政协委员和代表赴京共商选举制度改革。剥夺香港人民的民主权力，大开历史倒车的卑鄙行径，硬是被中共官媒活生生吹捧成同心筑梦共襄复兴伟业。

法国国际广播电台3月4日发表“中国政协会议是这样协商的”文章，指出“坐在主席团前排的不是会议主体的各民主党派，而是中国执政党的党首们，即中共中央政治局常委习近平、李克强、栗战书、王沪宁等人，而且会场内只有他们这些中共的政治局常委和政治局委员可以不用戴口罩”。这个中共标榜的，中国特色协商民主的主要载体和实践形式，人民政协被揶揄成中共自家人，是可有可无，招之即来，挥之即去的“假民主招牌”。

建国初期的政协，确有有识之士向中共要过民主的。据维基百科的资料，1957年5月，响应中共发起的整风运动，“大鸣大放”，许多知识分子和民主党派人士向中共提了很多意见。甚至提出要共产党退出机关、学校，公方代表退出合营企业，要求“共产党与民主党派轮流坐庄”；“根本的办法是改变社会主义制度”，“请共产党下台”，“一党执政有害处”，要开放“海德公园”（注：英国伦敦的一个公园，人们可在此自由发表言论），各个民主党派，都有首脑人物向中共提出，既然是共同执政，中共有自己的武装部队，那么民主党派也要有自己独立的武装部队等等.

但是一个月不到，1957年6月初，毛泽东就发出《组织力量反击右派分子的猖狂进攻的指示》，7月1日直指民主人士是有组织、有计划、有纲领、有路线，都是自外于人民的，是反共反社会主义的。这种人不但有言论，而且有行动。他们是有罪的，‘言者无罪’对他们不适用。”至此，暴风雨般的反右运动，使得40万到70万知识分子失去职位、并下放到农村或工厂中劳动改造，总计约140万人被归为右派或者中右派遭到迫害。

对中共出尔反尔，阴险恶毒的小人行径，《人民日报》还自我吹嘘“让大家鸣放，有人说是阴谋，我们说，这是“阳谋”。因为事先告诉了敌人：牛鬼蛇神只有让它们出笼，才好歼灭他们，毒草只有让它们出土，才便于锄掉。”所以跟中共谈民主，就是与虎谋皮，就是送命的下场。

与中共建国初期，被中共欺骗，背信弃义所愚弄的民主派人士不同不同，今日的香港政协委员们，完全是心知肚明，心甘情愿地向独裁中共投怀送抱。他们拼命自我改造，向中共看齐，只要能分得独裁者些许残羹冷炙，人大或是政协，都无所谓，就是要他们成为中共党员，也是在所不惜的。可惜中共需要在会议议程名单里有香港区政协委员装点门面，新闻里要借港区代表的口，说香港特区选举制度需要与时俱进；“人大决定“有利香港长治久安……

香港民主派有近30年的历史，支持香港全面推行普遍选举，并且对于中国大陆人权状况常有所批判。有别于其他国家中反对党或在野阵营民意不及执政党的情况，民主派在立法局引入直选以来直至主权移交后的立法会换届选举中，一直以约五成五至六成普选票数力压建制派。这样真正代表民意的民主政党，在独裁中共的眼中，成了中共取消一国两制，香港大陆化进程的绊脚石。把民主写入核心价值观的中共，在遇到真正的香港民主派时，狰狞的面目便显露无疑：违反基本法，引渡条例修订，到今天的“颠覆国家罪”，修改选举法。无一不在暴露中共假民主，真独裁的嘴脸。

美国传统基金会（The Heritage Foundation）最新公布的2021年《经济自由度指数》报告，不再将香港及澳门特别行政区进行独立评分，剔出指数之列。标志着世界对香港陷落，民主自由沦丧的一致认同。香港47名民主派被控「颠覆国家政权」案件，备受国际社会关注，美国、欧盟、英国等多国外交官排队听审。多国发声明表达关注，其中美国国务卿布林肯（Antony Blinken）、国家安全顾问苏利文（Jake Sullivan），以及国务院发言人，分别在案件开审前作出回应，促请当局立即释放被捕的泛民人士。

看看历史，世界就不该再对中共抱任何希望，当年反右运动前，毛泽东面对越来越多批评的言论，认为这些言论将危及中共在中国政治中的的领导地位，他说 “共产党执政还不到八年，就有三十多万条意见、错误、罪状，那共产党是不是该下台？那我姓毛的不是要重返井冈山！”连提意见和指正错误都要被打成右派的中共黑帮，要它乖乖地将权力与人民大众分享，真正给人民当家做主，那他就会将枪口对准人民。反右，89六四，镇压香港民主派都是血淋淋的事实。政治协商会议已成挂羊头卖狗肉的假招牌，不消灭中共，人民就永无出头之日，民主也绝不会将临到中国这片土地上。
