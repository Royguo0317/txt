我們在一開始做了正確的事情。我們拯救了數百萬條生命，這是我們做的。我們做對了，但是永久關閉根本不是答案，對特定區域進行小規模關閉（但我們不希望這樣做），小規模關閉可能會非常有幫助，但不是一個長期方案。

所以，不，我想得到，我希望我可以站著，高舉我的手，（宣佈）大獲全勝，因為我們正在與我們的國家一起做我認為沒有人能做的事情。儘管發生了這種病毒疫情，它正在毀滅其餘的世界。順帶一提，毀滅性的。

（近期）發表的文章之一是“全球新冠病毒復發”。這是《華爾街日報》的社論，是昨天《華爾街日報》的主要社論。我並不總是同意他們的看法，但是他們有“全球新冠病毒復發”（這篇社論）。 “被譽為模範的各國看到了……”然後他們繼續說，這個病毒回到了前所未有的、他們都沒有看到過的水準。

我們一直在讚揚某些國家，現在這種病毒像第一次（爆發）一樣感染了它們。但這是一個非常有趣的話題，它談到了許多國家，每個人都在擁護他們，並說他們做了多麼出色的工作。好吧，這只是這些事情的其中一個。（全球抗擊疫情的）效果不是很好。

所以我們想進行選舉。我希望看到選民的身分證明，但這（郵寄選票）與選民的身分證明相反。民主黨人喜歡它，但共和黨人討厭它。我們都同意缺席投票是好的，郵寄選票將導致最大的欺詐。

你知道，我們談論“俄羅斯、俄羅斯、俄羅斯”已有兩年半了，然後他們什麼也沒發現，其實確實沒有任何東西，但是他們說“俄羅斯、俄羅斯、俄羅斯。”他們談論中國，他們談論所有這些國家。他們說他們參與了我們的選舉，這很簡單，你可以偽造選票。對於外國而言，這要容易得多。

繼續，史蒂夫。

問：但是，但是推遲選舉可能沒有成功的希望。我的意思是，您不同意這點嗎？

總統：我只是覺得，我不想耽擱，我想進行選舉，但我也不想等待三個月，然後才發現選票全都丟失了，然後選舉也沒有任何意義。這就是將會發生的事情，史蒂夫。這是常識，每個人都知道。聰明的人都知道，愚蠢的人可能不知道，雖然有些人不想談論它，但是他們知道。

不，我們希望進行一次人們真正參加的選舉，並且-“您叫什麼名字？” “我的名字是張三或者李四。” 好！那請您簽署這本書，就像我多年以來所做的一樣。

這對我們的國家非常、非常不公平。如果他們這樣做，我們的國家將成為全世界的笑柄，因為每個人都知道它行不通。

例如，他在加利福尼亞發送多少張選票？ 2800萬個或者某個巨大的數字？其他州正在發出數百萬張選票。好吧，他們做到了。他們做了實驗。他們讓新聞機構進行試驗。

而且，請閱讀《華盛頓郵報》中有關郵寄投票的故事；這是一場災難。坦白地說，我很驚訝看到這個故事是他們發出的，這個故事是一場災難。因此，我們是在自己招惹很多麻煩。

而且，不，我是否希望看到日期更改？不，但是我不想看到一個不正當的選舉。這次選舉將是歷史上最嚴重的舞弊的選舉。
約翰（請說）。

問：總統先生，您說過，您不想看到選舉推遲，但是看來這些郵寄選票的過程將持續到11月3日。

總統：嗯，約翰，我們有很多案件。例如，我們有一個在賓夕法尼亞州西部已經簽署了一段時間的（案件），是關於郵寄選票的。
問：我只是想知道，是⋯⋯

總統：順便說一句，約翰，我們舉了很多例子，很多關於所有欺詐行為和與郵寄投票有關的所有事件。

問：我只是想知道，您今天上午發佈的推文和您現在正在談論的內容的最終結果，是否會令人懷疑11月3日的選舉結果？

總統：嗯，這已經產生了一個有趣的影響。我不知道它會產生什麼影響。人們現在正在看的是：我說的對嗎？但這不是（關於）我。所有這些故事是不是正確的闡述了一個事實，就是這些選舉將是欺詐性的、被預定的、被操縱的？每個人都在觀察，並且很多人都在說：“你知道。那可能會發生。”

請（說），珍妮佛。

問：總統先生，為了打破國會的僵局並防止失業保險倒閉，您今晚打算採取什麼行動？您正在⋯⋯

總統：嗯，這是一個很好的問題。不過，我無法告訴您，因為從談判的角度來看那不是很明智。但是，我們將把某些事情放在桌子上（進行談判）。

問：您有一個擺在桌上的計畫嗎？

總統：我們想為人們搞到錢。這不是他們的錯。我們想為人們搞到錢，這必須做實。所發生的事不是他們的錯。

事實是，人們不喜歡說這個事，但是他們知道這是真的：這是中國的錯。好吧？這不是他們自己的錯，這不是失業工人的錯，而是中國的錯。事情就是這樣。

OAN，請（說）。

問：謝謝總統先生。昨天，國土安全部與俄勒岡州州長達成協議，撤走了聯邦警員、轉由俄勒岡州的部隊接任。市長特德·惠勒顯然沒有參加該協議。先生，您是否有信心俄勒岡州能夠平息波特蘭的抗議活動？如果暴力事件仍在繼續，您會考慮重新部署聯邦部隊嗎？

總統：我們的人民，國土安全部，做得很棒。他們一個多星期前去了俄勒岡。這個地方很亂。波特蘭市就是一場災難。您看到了，很多人都沒有正確地報告（當地的情況）。他們試圖假裝這是一場抗議活動，而不是一群無政府主義者和煽動者。你明白我的意思。這是一團糟。

他們不久前去了那裡，並拯救了一座耗資數億美元的聯邦法院。他們在法院大樓周圍設置了環形圍欄，拯救了法院大樓，但是那裡的小組基本上是要拯救建築物的，並且它們非常有力、非常強大。為了保護這些非常昂貴、價值連城和具有心理重要性的建築物，他們並不需要經常從他們所造的繭中出來，對吧，就像（他們保護）法院一樣。

我們一直在與州長和市長打交道，我們認為他們不知道他們在做什麼，因為這種情況本不應持續60天之久，除非發生緊急情況(我現在認為是緊急情況)，否則這不是我們的工作。進入並清理城市不是我們的工作，這應該由當地執法部門完成。

昨天，州長達成了一項協議，並且他們將會執行這項協議。我們將會旁觀，然後他們將會執行。那很好，那很好，但是她沒有那樣報告，她的報告的內容完全不同。她說：“我認為川普想接管這個國家。”這太瘋狂了。

因此，我們的人民就一直呆在那裡，看看他們今天和明天是否能夠做到這一點。如果他們不這樣做，我們將派遣國民警衛隊，我們來保護這個城市。並且我們現在正告這些抗議者，他們當中許多人應該被捕，因為他們是專業的煽動者，是專業的無政府主義者。他們是討厭我們國家的人。我們現在告訴他們，我們，國民警衛隊，即將到來。（這將會是）很多人，很多非常堅強的人，這些人不是那些僅僅守衛和拯救法院的人。這些人被允許前進並做他們該做的事。我認為這使州長和市長的工作變得容易得多。

因此，他們今天（可能明天）正在努力清理這些恐怖分子的巢穴。如果他們這麼做，我會很高興的。然後，慢慢地，我們可以開始離開這座城市。如果他們不這樣做，我們將派遣國民警衛隊。
請（說）。

問：川普先生，鑒於美國職棒大聯盟正在發生的事情、以及今天羅格斯足球隊已被隔離，您如何確保人們可以安全地重新開放學校？

總統：那麼，你能向任何人保證任何事情嗎？我再說一遍：年輕人幾乎對這個疾病免疫，我想年齡越小越好，他們更強壯，他們更強壯，他們具有更強的免疫系統。這是不可思議的，以前沒人見過，各種類型的普通流感對年輕人的傷害要大於老年人，但是年輕人幾乎將近免疫。如果你們查看百分比，那是1％的很小的百分比，這只占1％的一小部分，因此，我們必須開放學校。我們必須保護我們的老師，我們必須保護我們的老人，但是我們必須要開放學校。
