# 重要事件

- **05.19. 美国之音 [美国会智库报告：中](https://www.voachinese.com/a/china-nuclear-arm-russia-20210519/5896916.html?utm_source=twitter&amp;utm_medium=social&amp;utm_campaign=dlvr.it)**共**[国未遵守防核扩散承诺](https://www.voachinese.com/a/china-nuclear-arm-russia-20210519/5896916.html?utm_source=twitter&amp;utm_medium=social&amp;utm_campaign=dlvr.it)**


美国国会智库国会研究所表示，美国政府继续对中共国向其他国家扩散核和导弹相关技术的情况表示关切。美国官员称，中国军队的核能力正在迅速增强。

这份周二公布的报告称，虽然中共国政府没有直接参与转让核和导弹相关项目，但在华实体仍继续出口与这些项目相关的货物，特别是向伊朗和朝鲜出口。

报告指出，在华实体还为扩散这些敏感技术提供其他形式的支持，例如非法金融服务和洗钱。

美国财政部2018年的一份报告显示，中共国实体为伊朗和朝鲜的大规模杀伤性武器计划提供了支持。具体而言，朝鲜在中国经营了一个金融代表网络，他们作为朝鲜金融机构的代理人，设立空壳公司，管理秘密的银行账户，来转移和掩盖非法资金，逃避制裁。

时任国务院东亚和太平洋事务副助理国务卿黄之瀚（Alex Wong）在去年11月的一次演讲中指出：“中共国接待了不少于二十名朝鲜大规模杀伤性武器和弹道导弹采购代表和银行代表。”

他还表示，中共国无视联合国安理会决议关于驱逐这些代表的要求，并称美国已经 “向中共国提供了关于在其境内发生的联合国禁止的活动的大量可操作信息”，但北京 “选择不采取行动”。

中共国在巴基斯坦建造民用核反应堆一直是美国国会关注的另一个重点。美国认为这些项目违反了北京作为核供应国集团（NSG）成员的承诺。中国已经在巴基斯坦建造了四个反应堆，并且正在建造另外两个这样的反应堆。

另有媒体报道说，沙特阿拉伯在中国帮助下建造了一个从铀矿中提取黄饼铀的设施，这将推进沙特刚刚起步的核计划。

尽管美国寻求在全球推动减少核武器，但中共国军队正在不断壮大其核能力。

美国裁军大使伍德（Robert Wood）周二在联合国举行的一个会议上说：“尽管中华人民共和国持有的核武器大幅增加，但不幸的是，他们仍然抵制与美国就降低核武风险进行双边会谈。”

伍德并说：“迄今为止，北京方面一直不愿意进行有意义的参与，或举行类似于我们同俄罗斯那样的专家讨论。我们真诚期望这种情况会有所改变。”

俄罗斯和美国今年早些时候同意将《新削减战略武器条约》延长五年，维持限制世界上两个最大战略核武库部署的最后一个条约。

俄罗斯总统普京和美国总统拜登预计在下个月举行会晤，讨论武器控制和安全问题，战略核稳定将会在议题之中。

然而，就在美国和俄罗斯外长周三在北极理事会见面前夕，中共国国家主席习近平和俄罗斯总统普京早些时候通过视频连接观看了中俄核能合作项目的启动仪式。

据中国媒体报道，这将标志着在中共国辽宁省和江苏省的两个核电站建造四个新的反应堆，这两个核电站都使用俄罗斯技术。

- **05.19 纽约时报 [以色列谴责中国官媒散布“种族主义谎言”](https://www.nytimes.com/zh-hans/2021/05/19/world/asia/israel-criticized-chinese-state-media.html?utm_source=tw-nytimeschinese&amp;utm_medium=social&amp;utm_campaign=cur)**


CGTN日前在Twitter发布视频，指成功的犹太商人在金融、媒体和互联网行业拥有重要影响力，可能有着“强大的游说力量”，对美国外交政策影响深远。以色列驻华大使馆周三回应称，对此类信息“感到失望”，希望CGTN删除这条“散播谎言与种族主义的侮辱性视频”。

中共国已有多家官媒谴责美国对以色列的支持，称其虚伪地追究新疆事务，却对巴勒斯坦人的人权被践踏熟视无睹。目前该视频已被移除。

- **05.19 BBC [随着中共国严厉打击加密货币，比特币进一步下跌](https://www.bbc.com/news/business-57169726)**


在中共国对加密货币施加新的限制之后，周三，比特币的价格三个月来首次跌破34,000美元（24,030英镑）。

北京禁止银行和支付公司提供与加密货币交易有关的服务。它还在周二警告投资者不要进行投机性加密交易。周三下午，比特币有所回升，尽管仍下跌-10.4％至38,131美元。同时，其他以太坊区块链网络为燃料的数字货币（如以太币）和Dogecoin分别下跌了22％和24％。

为了遏制洗钱活动，自2019年以来，加密货币交易在中国一直是非法的。但是人们仍然可以在线交易诸如比特币之类的货币，这引起了北京的关注。周二，包括中国国家互联网金融协会，中共国银行业协会和中共国支付与清算协会在内的三个国家支持的组织在社交媒体上发布了警告。他们说，如果消费者要承受加密货币投资交易的任何损失，他们将得不到任何保护。他们补充说，最近加密货币价格的剧烈波动“严重违反了人们的资产安全”，并且正在扰乱“正常的经济和金融秩序”。

Markets.com的尼尔·威尔逊（Neil Wilson）表示：“一段时间以来，中共国一直在向加密货币领域施加压力，但这标志着一种加剧的趋势-随着各国央行向自己的数字货币迈进，其他国家现在可能会紧随其后。“直到现在，西方监管者对比特币还是很放松的，但这可能很快就会改变。”

- **05.19 NationalPulse [英国开始从5G网络中淘汰与中国共产党联系紧密的华为](https://thenationalpulse.com/breaking/uk-rips-out-huawei-technology/)**


英国已开始从其5G网络中淘汰与中国共产党有联系的电信公司华为的技术。从赫尔市开始，政府开始实施一项为期7年，耗资7亿美元的任务，以拆除该公司的产品。该国最大的网络BT Group Plc将用诺基亚Oyj的组件取代该技术。

BT首席技术官霍华德·沃森（Howard Watson）指出：“我们非常热衷于选择一个城市区域并完成全部工作，并确保我们能真正检查出我们对客户服务没有负面影响。” “到目前为止，这些迹象真的很好。”英国电信的18,000个移动桅杆，屋顶和其他场所中的12,000个将不得不重复这一过程。

彭博社将此举归因于特朗普政府强力执行的“美国领导的针对中国公司的运动”。

华为与中国共产党和中国人民解放军有着广泛的联系，促使美国国务院将其确定为“国家军械，或更确切地说，是中国共产党”。华为被特朗普政府标记为“国家安全威胁”，并被美国国防部长达数十年的中共国军事合作者标记为华为，通常向中共提供后门访问其产品，网络和设备的权限。
