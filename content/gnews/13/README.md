# 墙内咨讯

**11.21.参考消息“外媒关注：习近平指明亚太合作未来之路”**

中国国家主席习近平11月20日在亚太经合组织（APEC）第二十七次领导人非正式会议上发表重要讲话，引发外媒高度关注。

据俄罗斯卫星社北京11月20日报道，中国国家主席习近平11月20日在亚太经合组织（APEC）第二十七次领导人非正式会议上发表讲话时表示，当前，世界和亚太正在经历深刻变革。亚太合作未来的路怎么走，关乎地区发展，关乎人民福祉，关乎世界未来。

习近平指出，实现自由开放的贸易和投资绝非可以一蹴而就。亚太地区要继续领风气之先，坚决维护和平稳定，坚定捍卫多边主义，坚持构建开放型世界经济，毫不动摇支持以世界贸易组织为核心的多边贸易体制，促进自由开放的贸易和投资，引导经济全球化朝着更加开放、包容、普惠、平衡、共赢的方向发展。

报道称，他还表示，明年中方将举办数字减贫研讨会，发挥数字技术优势，助力亚太地区消除贫困事业。

习近平指出：“数字经济是全球未来的发展方向，创新是亚太经济腾飞的翅膀。我们应该主动把握时代机遇，充分发挥本地区人力资源广、技术底子好、市场潜力大的特点，打造竞争新优势，为各国人民过上更好日子开辟新可能。”

报道还称，他表示：“我们要全面落实亚太经合组织互联网和数字经济路线图，促进新技术传播和运用，加强数字基础设施建设，消除数字鸿沟……中方提出倡议，推动各方分享数字技术抗疫和恢复经济的经验，倡导优化数字营商环境，激发市场主体活力，释放数字经济潜力，为亚太经济复苏注入新动力。”

另据塔斯社北京11月20日报道，中国国家主席习近平11月20日在APEC领导人非正式会议上表示，中方将考虑加入全面与进步跨太平洋伙伴关系协定（CPTPP）。

习近平在讲话中说：“我们要继续推进区域经济一体化，早日建成亚太自由贸易区。中方欢迎区域全面经济伙伴关系协定完成签署，也将积极考虑加入全面与进步跨太平洋伙伴关系协定。”

此外据德新社曼谷11月20日报道，中国国家主席习近平20日对包括美国总统特朗普在内的亚太领导人表示，北京将积极考虑加入CPTPP。

CPTPP是跨太平洋伙伴关系协定（TPP）改名后的版本，TPP是奥巴马政府推动的一项地区贸易协议。特朗普在2017年初上任后不久就退出了TPP。

又据共同社北京11月20日报道，中国国家主席习近平11月20日在APEC领导人非正式会议上呼吁各成员经济体努力营造开放、公平、公正、非歧视的营商环境。

他还表示，中国愿“积极考虑”加入有11个成员的CPTPP。APEC成员认为，CPTPP和几天前包括中国在内的15国签署的区域全面经济伙伴关系协定都是实现更广泛的亚太自由贸易区的途径。

报道称，21个环太平洋经济体的领导人11月20日召开视频峰会，以讨论应对新冠疫情的举措，并将通过一项后疫情时代地区合作的新愿景。

报道援引马来西亚总理穆希丁·亚辛在会议开幕致辞时的话说：“新冠疫情对我们的贸易和经济造成了持久影响。我们应当依靠贸易与投资走出现阶段的经济下行，我们必须团结起来，作出建设性努力，引领本地区走上一条强劲、包容并且可持续的经济复苏和增长之路。”

美国总统特朗普出席了此次APEC峰会，这是他自本月初总统选举以来首次在全球舞台露面，也是他自2017年以来首次出席APEC会议。

另据法新社吉隆坡11月20日报道，在美国于特朗普任内开始退出多边机制后，北京已经成为APEC这个经济合作机制背后的主要推动力。

11月20日晚，在马来西亚总理的主持下，特朗普与其他领导人一同通过视频连线出席了此次峰会。

报道称，所有领导人的屏幕都使用了APEC的一个官方背景，但特朗普除外，他使用的是米色背景。一名参与会议安排的消息人士说，美国总统拒绝使用官方背景。

近年来的APEC会议因为美中贸易关系紧张而蒙上阴影。在2018年的峰会上，领导人们未能就联合声明达成一致。但在拜登任内，美国有望采取更具建设性的态度。

**11.21. 环球网 “美调查：美企对在华业务更乐观了”**

美国CNBC网站11月19日文章，原题：调查显示，在拜登当选后，大多数美企对在华经商持乐观态度

上海美国商会周五发布的调查显示，无论从政治还是收入来看，美企在华前景都在改善。拜登胜选后，124名（美国）企业负责人于11月11日至15日接受调查，其中仅有2人表示对在华经商更悲观。

调查发现，鉴于（拜登政府）料将不同于特朗普政府，半数以上美企（54.8%）表示（对在华经商）“更加乐观”，8.1%表示“远比以前更乐观”。“我们的大部分受访者都将此视为一个有利因素”，上将中国通信行业巨头华为和其他公司列入黑名单，禁止它们购买来自美国关键供应商的部件。

在接受上海美国商会调查的受访者中，仅有5.6%认为在拜登政府治下将加征更多关税。但70.2%预计美国新领导层将与其他国家合作，向对华贸易关系施压。该调查研究由上海美国商会与普华永道联合开展，涵盖50家全球收入超过10亿美元的会员企业。

中国从新冠疫情中实现经济复苏——而美国仍难以控制疫情，也在帮助（在华美国）企业。随着2020年仅剩一个多月，近半数（47.6%）受访企业预计2020年收入将超过去年。这明显高于今年7月时的比例（不到1/3）。

该调查发现，大多数在华开展制造业务的企业都计划未来三年继续在中国生产，仅有3家企业打算将至少30%的制造业务迁往海外。

不过，对在华美企的情绪来说，这种得到提升的乐观并非一种“警报解除”信号。调查还发现仅有13.7%的受访者计划增加在华投资，多数都对在华发展计划持观望态度或仍拿不定主意。
