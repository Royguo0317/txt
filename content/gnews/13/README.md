台湾媒体报道，一位美国外交政策分析人士周一表示，北京方面对台湾采取的非军事胁迫行动，如外交孤立和政治干预，意在击垮台湾人的信心，可能比军事威胁更令人担忧。

布鲁金斯学会(Brookings Institution)高级研究员瑞安·哈斯(Ryan Hass)在接受总部位于纽约的美中关系全国委员会(National Committee on US-China Relations)采访时表示，美国和中共在台湾问题上正进入一种“行动-反应动态”，双方都认为自己是在对对方的行动做出反应。

虽然这导致华盛顿越来越多地谈论冲突风险，但哈斯表示，他更担心台湾面临来自中共的“非军事胁迫挑战”。

这种胁迫采取外交孤立、网络行动和政治干预的形式，目的是消磨台湾人的心理信心。哈斯说，该战略的最终目标是让台湾“在与大陆打交道时依靠自己的力量，它通往和平或繁荣的唯一途径是欢迎北京的拥抱”。

哈斯曾在2013年至2017年担任美国国家安全委员会(National Security Council)中国、台湾和蒙古政策主任，他表示，为了应对这些努力，美国可以对台湾提供“稳定、明确、坚定的方针”，这仍在其“一个中国”政策的范围内。

哈斯说，让台湾人对自己的未来感到安全和自信，在世界舞台上享有尊严和尊重，扩大与世界其他国家的贸易和投资联系，符合美国的利益。

哈斯表示，与此同时，美国和中共需要就各自行动的意图以及它们认为自己在回应什么进行“明确而直接的对话”，而不是依赖直觉。他补充说，“现实是，华盛顿和北京都不会从局势恶化中受益，台湾人民也不会。”

本月早些时候，美国国务院发布了新的指导方针，允许美国官员与台湾官员更自由地互动，而拜登派出的一个非官方代表团上周访问了台湾。

在美国采取行动的同时，据报道，越来越多的中共战机在台湾附近飞行，而中共辽宁号航空母舰在台湾附近海域进行了军事演习。

尽管中共最初预计，随着美国总统乔·拜登(Joe Biden)今年1月就任美国总统，中美关系将得到改善，但两国关系持续恶化中。在中共病毒是实验室制造的超限生物武器这一事实面前，中共疲于应对国际社会的压力已将攻打台湾作为维系其岌岌可危专制政权的手段。美国在印太联盟和日本结盟战略，标志着世界格局的重新划分。有共同民主信仰的国家将结盟应对中共霸权对世界的入侵。台湾处于风暴的核心，牵动着世界的形势。
