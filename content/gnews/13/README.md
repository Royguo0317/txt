**今天继续和大家浅谈河北卡车司机”金某“北斗掉线事件。**

关于中共党媒为北斗洗地，除了我昨天分享的媒体舆论导向以外，这两天更是出现了一些不敢透露姓名的专家和业内人士出面为北斗掉线洗地。

无姓名专家一表示：在卫星导航系统的设计中，已经通过星座和卫星可靠性等设计保证定位授时服务的连续不间断。对于此次大货车卫星定位掉线事件，具体原因不好评价。从北斗系统提供服务的角度来看，北斗能保证可见卫星数和信号质量，从而保证定位。

无姓名专家二表示：如果是北斗导航系统的信号不好，那肯定会是很大一片区域的信号都不好，绝不会只有个别设备受影响，“可能是几十万平方公里都收不到，甚至一个省几个省都收不到”，“所谓‘断网’就更不可能了，北斗导航系统断网是不可能的事情。”并且强调，“通信不是北斗导航系统的事情，北斗只管定位”，该卡车定位系统所谓“掉线”，有可能是设备自身对外通信的问题，不是导航卫星的问题。“车载设备是不可能和北斗卫星连接的，因为北斗卫星不管这个。可能是和某个交通方面的网络连接，和北斗没有关系。”

从事相关行业的业内人士赵翔表示：北斗卫星没发射信号这种说法几乎是不存在的，只可能终端接收不到，不存在北斗不发信号。更大的可能性是汽车接收的记录仪、接收设备出现问题。更有北斗终端的服务商出面解释了北斗掉线的原因，服务商称：北斗系统掉线，要么就是设备的问题，第二个原因就是说他主机是网络方面欠费，一些东西导致的他就掉线了，建议司机经常检查设备。
![]()![](https://gnews.org/wp-content/uploads/2021/04/北斗图片1.jpg)图片来源：网络图片
这些所谓的专家，业内人士包括北斗终端的服务商，说了这么多的废话，目的只有一个就是告诉老百姓，北斗掉线，断网那是不可能的，要么是设备出故障、线头松了、原部件烧了，要么就是货车司机自己未及时缴费，和北斗卫星导航完全没有关系。

事情还没查清楚，就开始疯狂的洗地撇清关系，没办法谁让北斗卫星导航是中共军方的呢。如果不是因为涉及到北斗，我想金师傅的自杀也如同其他普通百姓一样，早就消声觅迹了。

北斗卫星导航系统到底是什么能让中共利用举国力量为它洗地呢？北斗卫星导航系统是中共国自行研制的全球卫星导航系统。从2000年开始，该系统主要在中共国境内提供导航服务。2012年11月，第二代北斗系统开始在亚太地区为用户提供区域定位服务。2020年6月23日，北斗三号最后一颗全球组网卫星在西昌卫星发射中心点火升空，成功发射北斗系统第五十五颗导航卫星。可对全球任何一个点进行定位、导航、测速、授时和短语通信。

在北斗卫星全球组网之前，早在2012年，中共国交通运输部就做出了规定，12吨以上固定参数和用途的车辆，例如货运车辆和半挂牵引车辆等，必须安装北斗定位设备。该办法还规定：道路运输管理机构对未按照要求安装卫星定位装置，或者已安装卫星定位装置但未能在联网联控系统正常显示的车辆，不予发放或者审验《道路运输证》。

简单来说，就是货车想要顺利年审，必须要安装卫星定位系统，而且需进入联网联控系统。此外，还明确了道路运输经营者使用卫星定位装置出现故障不能保持在线的运输车辆从事经营活动的，由县级以上道路运输管理机构责令改正。拒不改正的，处800元罚款。有下列情形之一的，由县级以上道路运输管理机构责令改正，处2000元以上5000元以下罚款：（一）破坏卫星定位装置以及恶意人为干扰、屏蔽卫星定位装置信号的；（二）伪造、篡改、删除车辆动态监控数据的。这不是地方行政要求，而是国家法规。
![]()![](https://gnews.org/wp-content/uploads/2021/04/北斗图片2.jpg)图片来源：Youtube网络截图
但安装北斗的费用却要由司机个人承担。那么什么是北斗记录仪？行车记录仪是类似于盒子的一个设备，相当于疲劳超速的“监控器”，可以接收北斗或GPS信号，当货车速度超过规定速度，或者连续行驶4小时，该设备就会发出警报。也能显示大车的实时位置，路线，还有车速等等，也会在出事故的时候进行记录。此设备不但安装费用昂贵而且全国各地完全不统一，国内很多卡友都在网络上表示过对收费的不满，青岛卡友表示：18年的国五车，当时安装北斗交了3200元，而且必须安装指定的北斗，否则无法拿到营运证，每年的服务费还需要另交300元；山东卡友表示：每年需要交680元的服务费；漯河卡友表示：车上自带的北斗坏了，安装新的北斗需要420元，加上服务费720元，一共花了1000多，而且每年的服务费都要720元；黑龙江卡友也表示，北斗的安装被强制和营运证挂上关系，如果不按照公司的相关要求安装北斗，就拿不到营运证。而河南的卡友表示，需要参加线上培训，交140元的培训费，才能拿到营运证。为了正常运营，很多卡友也不得不低头。
![]()![](https://gnews.org/wp-content/uploads/2021/04/北斗图片3.jpg)图片来源：网络截图
然而并不是交钱之后就“万事大吉”了，这样高昂的成本完全无法保障设备的稳定性和实用性，在使用过程中，类似金德强北斗掉线的问题并不是孤立，随便在网上查询一下都可以搜索出很多，江苏盐城卡友爆料，车停着，北斗却显示在移动；另有一位卡车司机也悲愤交加地控诉北斗定位系统的掉线猫腻；就在金德强北斗掉线事件的隔天又有司机因北斗掉线被罚2000元。更讽刺是有卡友指出淘宝上那些便宜实用的汽车定位设备，经过几年的测试，竟然比那些符合交通部标准的北斗设备还稳定！这些只是目前在网络上可以看到仅存的能发出的声音，还有多少被噤声的我们无从得知。不仅这样，北斗设备也完全不具备人性化，监控，说起来，是为了车主好，不要疲劳驾驶。可是从来没有从实际问题上考虑过这些卡友的实际情况，如果警报响的时候货车就在高速上怎么办？难道直接原地停车休息吗？毕竟没有人是可以预测交通状况的，如果赶上节假日人多或是服务区关闭，那就更不可能停车休息了，这时候卡友所面临的就是停在路边的应急车道上也会被罚款，开车也会被罚款，那只能利益权衡下选择一个罚最少的款的选项。
![]()![](https://gnews.org/wp-content/uploads/2021/04/北斗图片5.png)图片来源：网络截图
真的为卡友好，为什么用北斗？既然利民为什么不降价安装？为什么还要收服务费？

根据《中国卡车司机调查报告》中的数据显示，中共国有将近3000万卡车司机，算算总共多少钱，背后的利益让人不寒而栗。我觉得真有用的话应该是车企统一安装，而不是国家强制安装，这就是中共政策性的垄断，所谓利民的政策不过是某些人敛财的借口。所以不管说得如何漂亮，事实上已经造成了卡友的更重的负担。

**归根结底中共国对底层百姓的政策就一个词“剥削”。**

货车司机是底层百姓中相当特殊的一个群体，他们的工作涉及了多个政府部门，如，车辆管理，交通管理，货运管理等，**每一个相关政府部门和货车生产企业都把货运司机当成韭菜一样的收割，巧立名目收取费用，各种不合理的处罚。**本可以如GPS一样廉价可靠的系统，却被中共政府做成牟取暴利的工具。可以把系统维护的更合理，价格更容易让人接受。却偏偏不选择这么做，**把弊端丢给本就生活不易的卡车司机群体艰难承受。**

**就是这样的邪恶流氓政府，对各个稍有收益的行业层层盘剥压榨。让已经辛苦劳作，维持生计的穷苦老百姓挣扎在贫困线上。**
