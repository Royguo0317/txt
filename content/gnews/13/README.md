两周前国会大厦抗议事件和新冠病毒大流行将给当天的就职典礼蒙上阴影——国家广场没有拥挤的人群，仪式后将没有大型的庆祝活动，卸任总统将不会出席会议，这将成为整个美国历史上政府权力和平移交的标志性事件。

以下是当天将会发生的一些事情。

—=何时宣誓？=—

拜登和卡马拉·哈里斯大约将在中午宣誓就任新总统和副总统。

哈里斯——首位黑人和印第安裔美国女性副总统，将由最高法院大法官索尼亚·索托马约尔宣誓就职 。哈里斯宣誓将阅读两本圣经：一本属于第一任非裔美国最高法院大法官瑟古德·马歇尔（Thurgood Marshall），另一本属于里贾纳·谢尔顿（Regina Shelton），后者是哈里斯的母亲。

最高法院首席法官约翰·罗伯茨（John Roberts）随后将安排拜登（Biden）宣誓就职，拜登将把他的手放在家庭圣经上。

—=谁会出席，谁不会出席？=—

唐纳德·特朗普：他将是美国自1869年以来第一位跳过继任者就职典礼的现任总统。拜登对特朗普的缺席表示知情，称这是“一件好事”，又说“这是他和我仅有的几件达成一致的事情之一”。

共和党参议院议长米奇·麦康奈尔（肯塔基州）和众议院少数党领袖 凯文·麦卡锡 （加利福尼亚州）从此与特朗普保持距离，并有望参加就职典礼。前总统奥巴马，乔治·W·布什和克林顿也将出席。现任副总统彭斯也计划出席来支持这次总统权力的过渡。

就职典礼的策划团队考虑到疫情计划缩小活动规模，今年在国家广场和看台上不会挤满成千上万的人。加之1月6日发生的国会大厦事件，国会的某些议员也可能最终选择远程观看这次活动。

—=就职典礼会在户外举行吗？=—

拜登坚持认为，仪式仍在国会大厦西面的户外举行，并在1月6日之后坚持宣称：“我不怕在外面宣誓。”

流通的空气可能有助于限制COVID-19的传播，因此组织者已经开始筹备在户外进行活动，而且，如果美国最高政府官员不在同一地点，安全担忧就会有所缓解。

“我对我们在安全方面的状况感到很好，但是，正如我所说，四年前担任主席时，有人问我：‘就职典礼的最佳时机是什么？’ 我说，‘当每个人回到里面时’。”森说。罗伊·布朗特一直负责领导联合国会就职委员会，他说：“我的意思是，很显然，这是我们政府最脆弱的时刻，也是重要的时刻，我们要展现出作为民主国家的实力。”

—=安全性如何？=—

鉴于在同一地点出席的知名官员人数众多，就职典礼始终保持高度戒备，尤其在国会大厦事件发生后，安全措施已经接近极限。国会大厦现在挤满了成千上万的国民警卫队员，还有多层高高的安全围栏。 通往国会大厦的大部分街道以及该市大部分市中心的道路都被关闭，连接维吉尼亚州和美国首都的许多地铁站甚至多座桥梁都将关闭。

—=最新的威胁是什么？=—

就职典礼前仍存在安全隐患。 最近几天，至少有三人因试图未经授权进入安全范围或在检查站附近携带武器而被捕。负责安保的陆军国民警卫队也剔除了12名成员，原因是他们在安全性审核过程中发现了“不适当的评论或文本”或其他可疑行为。

据报道在周一，就职典礼彩排正在进行时，附近一座桥下几个街区外发生火灾，国会大厦被暂时封锁。

这些特别的安全措施似乎已经阻止了一些潜在的威胁。执法部门一直在为应对极端分子做准备，其中包括一些参加了国会大厦事件的团体，这些极端分子星期天来到华盛顿进行武装游行，但最终没有人聚集。

—=总统弹劾也会在星期三进行吗？=—

众议院尚未将指控特朗普煽动叛乱弹劾条款的议案提交参议院开始审判， 民主党人也担心会分散人们对周三就职典礼的注意力，这意味着他们很可能至少要再过几天才能启动参议院审判。

—=拜登第一天会做什么？=—

拜登在上任的第一天有一份很长的计划要做， 预计他将采取许多行政措施，包括要求教育部延长学生贷款的暂停时间；重新加入《巴黎气候协定》；撤销特朗普对多个穆斯林国家的旅行禁令；并发布有关联邦财产和州际旅行的法令。拜登还有望公布将为非法移民美国的数百万移民提供合法公民身份途径的立法。

—=在宾夕法尼亚大道和就职舞会上会不会游行？=—

今年是社交疏远时代，但是将会有一个虚拟的“美洲巡游”，大约从下午3:15开始，来自所有50个州和几个地区的表演。

游行将由演员托尼·戈德温（Tony Goldwyn）主持，并有乔恩·斯图尔特（Jon Stewart）和奥运会运动员内森·陈（Nathan Chen），阿里森·费利克斯（Allyson Felix）和凯蒂·莱德基（Katie Ledecky）等表演者和运动员登场。

参加开幕典礼并参观了阿灵顿国家公墓后，拜登和哈里斯将由军车护送从第15街到白宫。护送人员将包括美国陆军乐队，联合服务荣誉警卫，以及来自美国第三步兵“老警卫队”的总司令卫队和法夫鼓乐队。
