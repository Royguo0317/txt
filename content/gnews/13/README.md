## 中共軍方是亞太各國政府網絡間諜活動的幕後黑手
`2020-05-08 07:42 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/197290/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/05/08074009/%E4%B8%AD%E5%85%B1%E5%86%9B%E6%96%B9%E6%98%AF%E4%BA%9A%E5%A4%AA%E5%90%84%E5%9B%BD%E6%94%BF%E5%BA%9C%E7%BD%91%E7%BB%9C%E9%97%B4%E8%B0%8D%E6%B4%BB%E5%8A%A8%E7%9A%84%E5%B9%95%E5%90%8E%E9%BB%91%E6%89%8B-1.jpg)
據美國CNBC新聞網7日報道，以色列網絡安全公司檢查點軟件技術有限公司(Check Point Software Technologies)周四發布了一份報告，稱來自中共的黑客團隊長期在亞太地區各國政府部門間從事網絡間諜活動。
文章中提到，一個名為“Naikon”的黑客小組使用“Aria-body”這種隱形網絡攻擊工具遠程控制目標計算機，獲取情報信息。尤其是針對包括澳大利亞、印度尼西亞、菲律賓、越南、泰國、緬甸和文萊等國的外交部門、科技部門及政府控制的企業等多維度信息情報的竊取。
據了解，“Aria-body”可以侵襲任何用來打開嵌入它的文件的計算機，並馬上讓機器服從黑客的指令。這可能包括建立一條秘密的、難以查出的通訊線路，讓目標計算機上的數據通過這條線路流向攻擊者使用的服務器。通常是發送一封嵌入該惡意程序的郵件給對方，並把該文件偽裝成正常文件，一旦目標打開該文件，它能迅速下載啟動其他服務進而控制該計算機。
文章稱，這個名為“Naikon”的黑客小組於2015年首次被發現，同年美國網絡安全公司Threat Connect發布了一份報告公布了其黑客手段及該小組與中共軍方的關係。據Threat Connect的報告，這個黑客小組似乎隸屬於中國人民解放軍第二技術偵察局78020部隊，主要駐紮在中國南方城市昆明，負責中國在東南亞和南中國海的網絡行動和技術間諜活動。俄羅斯網絡安全公司卡巴斯基實驗室(Kaspersky Lab)的一份報告也提到，該黑客小組是亞洲最活躍的“高級持續性威脅”之一。安全專家經常用此說法來描述長期從事入侵活動、有政府背景的黑客。
檢查點安全公司人員表示，2015年Threat Connect發布的報告將“Naikon”的主要網絡武器公之於眾後，這個黑客團體似乎消失了。但實際上它一直在默默的活躍中，尤其是在2019年和2020年的第一季度，加快了網絡間諜活動。
CNBC的記者聯繫了中共外交部，但他們並沒有立即對此事發表評論。在此類批評面前，中共近年來一直都聲稱，反對任何形式的網絡攻擊，並且中共政府和軍隊不參與任何形式的間諜活動。
