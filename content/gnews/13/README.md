任何談論盧布歷史，只對匯率大驚小怪，不談論將舊貨幣換成新貨幣的人都是無賴。從1970年代到2020年，盧布兌美元匯率下跌了12萬倍；從1991年蘇聯解體開始，美元對美元貶值了2,563次。路德說，人民幣將貶值25,000倍，這一點也不為過。

關於盧布的英文文章有兩種類型：

第一類是美國人寫的。要么只談論1992年之前的盧布歷史，要么只談論1998年之後的盧布對美元的匯率。總而言之，盧布對美元的匯率一直在下跌，並且存在黑市交易。

2.第一類是“從中世紀到今天的魯布的時光之旅”一文，其中記載了1992年及以後盧布的歷史。這是一篇“ 50美分”的文章，是蘇聯和俄羅斯貨幣政策的洗地。他們純粹從匯率的角度談論了盧布的歷史，最後得出的結論是，從1924年到2020年，盧布兌美元貶值，但只有31倍（45 / 1.4 = 32）。他們有意識地忽略了一個重要信息。俄羅斯發行了新硬幣，並以1：1000的比例將新硬幣換成舊硬幣。他們用新硬幣在人民手中收穫了許多舊硬幣。

關於盧布的文章，在中國世界上有一篇不錯的文章：“我親身經歷的盧布的興衰”。在本文中，提出了一個關鍵問題。 1998年，俄羅斯政府將1：1000新貨幣兌換為俄羅斯人民持有的舊貨幣。據此，我們可以計算出從1970年代開始（盧布：美元= 1：1.6），到2020年，盧布兌美元匯率下跌了約12萬倍：1.6 / 0.013 \* 1000 = 123,076（0.013是2020年9月）盧布對美元的匯率在7日）。如果根據1991年蘇聯解體（匯率為1:30）計算，則美元兌美元貶值了2563次：1/30 / 0.013 \* 1000 = 2,564（0.013是盧布對美元的匯率） 2020年9月7日的美元）

在這段時間裡，人民幣對美元的匯率已經從7點改變為6.9點，並且已經上升，這是荒謬的。但另一方面，外匯管制越來越嚴格。與前蘇聯完全一樣，大量資金很難流出，匯率極高，但同時要控制外彙和要控製商品流通。路德曾經說過，中國共產黨的“人民幣”匯率將下跌2萬倍。與盧布的歷史相比，它只能超過盧布，而沒有其他。

自延安時期以來，中共一直使用發行貨幣的方式來收穫財富並將其藏在海外。 1949年成立後，它用新幣兌換舊幣，並貶值了10,000次（1955年為10000：1）。中國人民（隱藏在海外）的絕大多數財富，以一種糟糕的方式造成了“建設新中國”的幻想。 1989年和奧運會之前，財富大量轉移。現在他們覺得發行額外的貨幣還不夠，他們徒勞地試圖使用數字貨幣和西王幣（西王去世）來繼續收穫中國人的救命錢。
