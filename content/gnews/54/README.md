据Great Game India 3月8日报道，根据美国食品药品监督管理局（FDA）和疾病预防与控制中心(CDC)收集的疫苗不良事件报告系统（VAERS）报告，约有1000人在接种中共病毒疫苗后死亡。

辉瑞公司全球媒体关系总监斯蒂夫-丹尼（Steven Danehy）在给《大纪元时报》关于VAERS死亡报告的回复中写道, “到目前为止，已经有数百万人接种了我们的疫苗。不幸的是，严重的不良事件，包括与疫苗无关的死亡事件，很可能以类似于普通人群的死亡速度发生。”

在12月至2月期间，有超过19000份关于疫苗不良事件的报告。这些报告是在接种中共病毒疫苗后向VAERS提出的，包括辉瑞mRNA疫苗或莫德纳。这些信息是根据VAERS关于中共病mRNA疫苗的报告得出的。这些疫苗是应急使用的，它们没有得到FDA的批准。目前2月中旬以后的数据无法获得。

截至2月19日，在总不良事件报告中，有5%也就是966人被报告注射后死亡。8.9%的人在接种疫苗当天死亡，13.4%的人在一天后死亡，97人在2天后死亡，61人在3天后死亡，53.2%的人在7天后死亡，106人在20天后死亡，173人在1周至13天内死亡。接种疫苗后死亡的85%的人属于60岁以上的人。5人死于20至29岁，20人死于40至49岁，8人死于30至39岁。

法国加奇（Garches）医院克里斯蒂安·佩龙（Christian Perronne）博士在欧洲提交的一份投诉中声明：”他们提供给我们的第一种疫苗不是疫苗. 这些是基因治疗产品。他们……注入核酸，会使我们自己的细胞制造部分病毒。”

中共病毒疫苗的死亡率高于流感疫苗。根据CDC的报告，只有粗略估计死亡率。根据2019-20报告，只有1.7亿人接受了流感疫苗接种，约占美国总人口的51.8%。根据VAERS报告，2018年有46人死亡，2017年有20人死亡，2019年有45人死亡。

疫苗不良事件报告系统(VAERS)

疫苗不良事件报告系统成立于1990年，目的是为了发现美国许可的疫苗的意外反应或安全问题。该系统由FDA和CDC共同管理。疫苗不良事件报告系统在人们接种疫苗后发现、分析和接受副作用报告。任何人都可以向VAERS报告副作用。疫苗制造商和医疗专业人员必须报告他们知道或发现的所有副作用或不良事件。

VAERS的设计不是为了确定疫苗是否引起了健康问题，而是用于（接受）报告不寻常或意外的不良事件，以警示疫苗可能存在安全问题。VAERS为CDC和FDA提供有价值的信息，相应地，CDC和FDA展开调查，以进一步评估可能的安全问题。

除接种疫苗后的死亡事件，疫苗不良事件报告系统（VAERS）还报告称，有34例孕妇在接种中共病毒疫苗后发生自然流产或死胎。

评：

近1000人死于中共病毒疫苗，只限于美国VAERS统计的数据，并不包括其他国家的数据。除了死亡，还有很多严重伤害，如致癌、免疫力受损和罹患艾滋病等后果。

英雄科学家闫丽梦已就中共病毒起源和其实验室改造的分析发表了两篇报告，最近即将完成并发表第三篇关于中共病毒是中共为征服世界而发动超限生物战的一部分的论文。按照闫博士在推特中所说，没有阅读闫博士的论文而研发中共病毒疫苗，等于浪费时间。

闫博士是中共病毒爆发之初唯一一位负责联络中方和世卫组织的专家，她掌握了大量一手来自中共国武汉等地的病毒感染病例详情。同时，她与世界第一冠状病毒专家共同在一个实验室工作，研究冠状病毒（包括中共病毒）。

研制中共病毒疫苗的专家，只有认真研究闫博士的报告，尊重科学、唯真不破，才有可能研发出真正安全有效的疫苗。
