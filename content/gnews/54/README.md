## 【医疗前沿世界】中共病毒疫苗护照是与非
` 纽约香草山MOS01` [轉載自GNews](https://gnews.org/zh-hans/1046254/)

作者: 纽约香草山医疗部  圣母院钟声

![]()![](https://gnews.org/wp-content/uploads/2020/12/logo-3-1-2.jpg)



3月26日，纽约州民主党州长库默（Andrew Cuomo）宣布该州将实施“疫苗通行证”，即进入体育赛事、婚礼集会、饭店酒吧等民众日常活动场所的人，需出示疫苗接种证明或及时核酸阴性检测报告。这使该州成为美国实行“疫苗护照”的第一州， 也使“疫苗护照”成为主流媒体的热炒话题。拜登政府发言人也公开回应说：白宫正在考虑向各州的“疫苗通行证计划”提出指导性建议。

![疫苗]()![疫苗](https://gnews.org/wp-content/uploads/2021/03/vaccine-2.jpg)

而3月29日，佛州共和党州长德桑迪斯（Ron DeSandis）签署行政令，禁止佛州社交场所索要客人疫苗接种证明；俄亥俄州共和党州长德万恩（Mike Dewine），也向其本州居民保证不会实施“疫苗护照”，更不会强制接种疫苗。同时，明尼苏达州的共和党参议员本森（Michelle Benson）和乔治亚州的共和党众议员格林（Marjorie Greene）也提出了抵制“疫苗护照”的议案，他们认为“疫苗接种应该是个人在其医生指导下的私人选择，任何人不该被强制接种疫苗或出示接种证明”，“人们已经因病毒疫情失去很多人身自由，强制性政策绝不是解决问题的好办法”。

这些共和党州纷纷做出选择是基于渐渐明朗的实情, 即接种中共疫苗并没有控制住感染及死亡率，而疫苗接种后的严重副反应却带来了更大困扰。据美国疾控中心CDC疫苗接种副反应报告系统VAERS 3月31日的统计：过去3个月死于中共疫苗接种的人数超过过去十余年免疫接种人身意外的2倍以上，为1750例，已多过9/11遇难人数的一半。这些死亡案例绝大多数是接种了辉瑞和莫丹纳的mRNA疫苗所致。而英国则报道了孕妇接种牛津-阿斯利康中共病毒疫苗后的流产率飙升366%，该疫苗世界范围接种后，大量血栓形成而引起的并发症案例也络绎不绝地被爆出，以致欧洲多国不得不停止使用。

全球医疗界已经注意到：中共病毒感染人数在二三月份有所降低后,上周又开始快速攀升，这是在疫苗广泛施打之后发生的！3月31日CDC负责人瓦兰斯基博士（Dr.Rochelle Walensky）在接受MSNBC采访时说：“对目前的疫情感到担忧，恐怕疫情会恶化。” 传染病学家出身的瓦兰斯基博士的极度担忧是完全可以理解的。作为CDC负责人，她深知世界似乎已经用尽人为手段：关闭经济、隔离、口罩、社交距离及目前的广泛施打免疫疫苗，政府机构也积极配合，加快通过疫苗的“紧急使用授权”(Emergency-Use Authorization,EDA)，疫苗研发药企也以十二分的努力加大加快出产。所有社会资源都已调动，可人们曾给予厚望的病毒疫苗似乎并没能使上劲，帮上忙。过去一周，每日中共病毒感染人数都有明显增加。最令专家担心的是：感染病毒变异株的病例上升幅度很大.而英国病毒变异株的传播速度比原先的病毒株要快5～10倍，已占美国检测到的病毒变异株的26%，且具免疫逃逸的特性。更可怕的是其在免疫逃逸过程中，可继续变异。

这一现象验证了荷兰著名病毒疫苗专家吉尔特·凡登·博歇博士(Dr.Geert Vanden Bossche)早先的分析:

1. 接种疫苗并没有预防病毒的感染或传播，反而创造了更多无症状带病毒者。
2. 接种疫苗仅仅是人工刺激了抗原特异性抗体的产生，而非有效的、由自然杀伤细胞参与的、针对中共冠状病毒的中和反应。
3. 疫苗接种后产生的抗原特异性抗体不仅对变异病毒无效，其永久残留还有与人体免疫自然杀伤细胞竞争的潜在可能，使后者失去对肌体原有的天然保护力。
4. “教会”病毒逃避人体免疫识别系统，间接创造更多变异毒株。


也就是说，目前的全球中共病毒疫苗广泛接种非但没有缓解疫情，反而使疫情更难控制，因为出现了更多有免疫逃避功能的”狡猾”变异病毒株。

根据英国牛津大学最近的调查分析，全球已完成疫苗接种的人口占总体人口的1.6%，巧合的是中共病毒全球感染人数比例也约为1.6%。所以，不能用尚不够广泛的疫苗接种来解释今年二三月份全球中共病毒感染率的下降。有病毒学家认为更可能是这波变异病毒传播过程中暂时性减弱的内在规律所致。

世界各国都在纷纷加快推进中共病毒疫苗接种计划，其中以色列名列第一，全国接种辉瑞mRNA疫苗率为97%，但已有12000多例完成疫苗接种后病毒核酸检测为阳性的”突破性病例”。无独有偶，3月31日华盛顿州也公布了102例同样完成疫苗接种后病毒核酸检测为阳性的”突破性病例”，这102人均有不同程度的症状，其中8人入院治疗，2位80岁以上患者去世。

回顾免疫疫苗研发史，多数疫苗在给孕妇、婴幼儿和青少年施打之前，需经历大约10年的安全期测试。而mRNA疫苗在去年夏秋季才完成实验室研究，临床试验时间稍稍大于零，且无第三方验证，然后就是媒体上铺天盖地的“对中共病毒感染具95%的预防有效率及接种后轻度副反应”的结论，且对堆积如山的严重副反应案例有意回避报道。如果接种后死亡、流产、癫痫属于轻度副反应，那么什么才属于严重副反应? 政府及疫苗商们正不遗余力地灌输给民众: 只要能换来全民免疫，出现疫苗接种后的头痛、面瘫等不适是完全值得的，是必要之举。那么,为什么对感染此病毒后症状轻，且死亡率极低的18岁以下孩童也计划施打疫苗? 至2020年12月3日的统计，美国境内感染中共病毒的140万18岁以下人群的死亡案例是154例，15个州未出现一例该年龄组死亡病例。想过会有多少孩子丧生于或残疾于他们也许从不会感染或感染后很快能痊愈的中共病毒疫苗吗?

记得小时候父亲告诉过我，他曾被带到有麻疹患儿的邻居家中,长辈就为了让他对麻疹病毒有免疫力(即产生抗体)，期望以后不会得病。这是对”群体免疫”的形象解释，是我爷爷辈就有的医学常识。为什么疾控中心（CDC）在有足够统计数据支持下，还建议强制关闭所有托儿中心和学校，不主张在病毒威胁性极低的孩童中建立起群体免疫? 现在太多不可思议的现象无法用常识去解释。就连民主党左派代表人物——前克林顿政府白宫顾问沃尔夫博士（Dr.Naomi Wolf）——也认为“疫苗护照”太离谱。她于3月28日发推：“疫苗护照”将是对西方民主自由的完全终结……。接种疫苗只是个借口，等于每人体内侵入一匹特洛伊木马，将被360度监控。以后能否得到社会资源、食物、信息，可否迁徙等，都取决于个人行为是否被(政府)接受。沃尔夫博士的一针见血能令习惯于自由选择生活方式的美国人不寒而栗，人们不禁会联想，美国离目前中共国、北朝鲜现状还有多远?

辉瑞前副总裁、主任科学家伊顿博士（Dr.Micheal Yeadon）告诉“美国前线医生联盟”：“这个最终连婴幼儿也不会放过的全民中共病毒疫苗接种计划是邪恶的”，“给60岁以下健康人接种仅仅试验了几个月的疫苗是违反医学伦理学的，因为没人知道其对人体的长期影响。”

这种世界范围的病毒及疫苗的雾霾正迅速笼罩地球。中共用病毒攻击全世界，随后中共及邪恶利益集团把病毒疫苗当作”特洛伊木马”,通过注射把”木马”注入到我们的身体，使人类被监控，甚至被操控，并美其名曰”疫苗护照”。我们目前力所能及的是考虑如何避免被操控及绑架。因为所谓“疫苗护照”的限制，人们的活动范围可能在短期内会受影响，但随着病毒真相被抽丝剥茧地揭露,越来越多普通民众会站出来发声，相信这种不合理政策一定会有所改变。

英雄病毒学家闫丽梦博士一再强调，没有中共设计这款病毒生物武器的具体基因修改信息，人类将难以在短期内研制出有效疫苗。这也是天然病毒与生物武器的区别。以新中国联邦为后盾，闫博士领导的博士军团正为揭露中共病毒真相全力以赴。他们相继发表的3篇核弹级报告，将这个病毒的前世今生，从基因学到动物试验，从具体原版舟山蝙蝠病毒信息到揭露那些为中共洗地的“世界知名科学家”，层层递进，步步为营，使其谎言狡辩暴晒于天下，为以美国为首的病毒讨伐联军提供了依法惩治恶魔中共的最有力抓手。世界正翘首以待……
