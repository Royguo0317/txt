2021年2月缅甸军方发动政变并扣留翁山苏姬( Aung San Suu Kyi)所属的「全民联」领导人等而点燃全国民众的怒火，抗议者多为Z世代的年轻人以「和平」且「公民不服从运动」表达对军方的抗议。自缅甸政变以来军政以血腥镇压示威者，这些Z世代的年轻人以手机拍摄镇压过程并即时透过网路社交媒体播放至全世界，引起观注希望联合国制裁军政府。

## **「你们惹错一整个世代的人」**

发起缅甸反军事政变示威者核心成员是几乎没有政治经验年轻的「Z世代」。他们平均年龄在25岁左右，也是生长在网路世界里。这一代人从未经历1988年缅甸民主化的血腥屠杀。由于2月被军方软禁的翁山苏姬领导「全民联」等人是在缅甸2015年以民主选举取得压倒性胜利，结束了将近50年的军事统治。面对现在国家再度陷入军政统治，他们无法接受也不愿回到过去由军政统治的时代。这次镇压激起Z世代向军方怒吼「你们惹错一整个世代的人!」(You messed up with the wrong generation)。这些年轻的新世代坚定举着非暴力标语”WAP” (We Are Protesting Peacefully)，他们勇敢站在争取民主的第一线不惧水柱，催泪瓦斯，警棍和子弹而走上街头和平抗议就如同香港的青年们的反送中运动及去年泰国学运的反抗军事统治一样。

## **年轻示威者利用社交媒体记录一切,且即时现场真播**

缅甸的Z世代透过网路了解真相，由于自军事政变以来，军政府的网路控制，各种威吓抗争者等行为，跟他们在网路上看到中共如何以武力镇压香港抗争者的情况一样。在抗议过程中，他们处在极度危险的状况，为了自我保护，做好防护措施如头载头盔，自造盾牌，机车安全帽，面罩和护目镜等。他们也利用手机拍摄抗争及镇压过程并即时透过网路社交媒体让世界看到他们如何被镇压。为了传播真相，他们靠自己记录一切并及时传播讯息并呼吁世界付出行动帮助他们对抗恐布份子，他们也透过现场直播至世界媒体，让联合国制裁军政府。

## **缅甸何时走向民主化**

缅甸未来充满未知。原生于网路Z世代的年轻人从上一代得知上一代人如何被镇压导致无法正常受教育，而军政的子女却可以独享各种特权。在军政统治的时代缅甸人几乎没有人希望回到那个国家陷入军事独裁，经济制裁和大规模贫困的年代。

新世代年轻人经过这次不合作运动后，让世界看到缅甸人坚决追逐民主化国家，这场战斗让缅甸的这一代正以他们无畏的精神及过人的勇气，正在慢慢改变这个国家，而缅甸人民展现出的坚强意志，也是渴望换取更多国际支援及关注。 Z世代的崛起,是国家的希望。他们的梦想才刚起步，国家正在重建，但这需要民主自由的环境才能实现，所以他们必须为自己及国家而战斗。

## **不再沉默接受国家极权统治的新世代**

亚洲的年轻新世代的民主运动对抗极权政府正在风涌云起，他们没有政治经验，却以和平，理性，幽默有序以民主方式游行抗议，而相对于极权政府却以武力血腥镇压人民。 2019年香港反送中和平运动的年轻新世代以和平游行抗争行为树立了和平示威活动风潮。 2020年泰国学运反军政，抗议者以幽默风趣字眼吸引全球观注。 2021年缅甸Z代积极用手机拍摄抗争及军政镇压的影片即时传上网路社交媒体以突破军政控制媒体,让全球看到真实状况。

新世代年轻人透过网路学习到许多资讯，要实现自身价值及理想是需要以民主自由体制下才可以实现，他们不再沉默接受由少数几个人用军事极权统治方式对他们不平等的待遇。若国家发生政变而犠牲多数人民的利益，他们持续保持沉默接受这样的体制，他们必会过着跟上一代人相同贫困的命运，为了自己的将来,必须立刻做出回应。虽然他们能力有限，但新世代年轻人生长在网路世界，也学习到许多资讯知识超越上一代人，他们必将以创新手法运用网路来改变国家的体制，所以「不要惹错这一整个世代的人!」。
