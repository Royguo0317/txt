“美国就业计划 “包括修复老化的道路和桥梁，启动交通更新项目，重建校舍和医院。它还将扩大电动汽车的规模，更换所有铅管，并彻底检修国家供水系统。

但该计划远不止于基础设施。

它同样是一个就业计划，该计划旨在培养美国的清洁能源劳动力，扩大制造业，并促进护理工作作为一种服务于老年人和残疾人的职业。

**拜登的计划中有什么内容？**

该计划是拜登经济议程的核心，接下来需要美国国会通过。

据白宫介绍，该计划主要分为四个方面。以下是这笔钱将如何使用：

1. 交通基础设施：6210亿美元
2. 家庭生活质量提升：6500亿美元
3. 老年人和残疾人的护理：4000亿美元
4. 科学研究、技术开发和制造：3000亿美元


**如何筹集基金？**

拜登希望将企业税率提高到28%来支付该计划，将美国跨国公司的最低税率提高到21%。

根据白宫的说法，这项被称为 “美国制造税收计划 “的税收改革旨在激励在美国创造就业机会和增加投资，结束向避税天堂转移利润的行为，并确保大企业支付 “公平的份额”。

该计划将取消一项规则，即允许当美国公司将投资地点设在其他国家时，前10%的利润无需缴税。

根据加税和其他改革措施，例如，消除知识产权的税收漏洞，拒绝离岸工作税务扣除。白宫预计，这些支出将在15年内全部付清，并在随后几年减少赤字。
