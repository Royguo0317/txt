德国外长马斯正在向美国传递一个信息：对抗潜在的竞争对手，欧洲是一个可靠的合作伙伴，双方必须更加紧密地团结在一起。马斯认为，欧洲人正在承担更多的外交政策责任，并在跨大西洋关系中肩负起自己的责任。

第一次驶入印太地区

马斯认为，与这些大国打交道，对跨大西洋的未来也将是决定性的。最重要的是：”我们必须承诺采取共同立场”。需要说明的是：”我们如何与一个日益挑战我们、与我们对抗的中共国创造一个公平的竞争环境？”以及该如何应对”越来越具有侵略性和压迫性的俄罗斯”？马斯给出的答案是：”对我来说，这意味着每当俄罗斯、中共国或其他国家威胁到我们的安全、繁荣、民主、人权和国际法时，我们都会予以反击。”

他说：自2014年以来，国防开支增加了50%；这将继续下去。在扩大数字基础设施方面，特别是保护新5G网络的立法已经”加强了对外国影响的保护”。作为一个贸易国，德国致力于维护海上自由航线，这也是为什么柏林最近决定”首次向印度洋-太平洋地区部署海军”。

他还希望扩大跨大西洋的合作，旨在加强和更好地保护大西洋两岸民主制度的国内联合，并努力纳入其中。马斯周二表示，仇恨言论、操纵和虚假信息正在破坏”我们对彼此的信任”。他说：现在需要的是”更好的监管，包括在国际一级的监管”。这是一项”显然不能让大型科技公司单独承担的任务”。

**评论：**德国外长马斯表达了两个大观点：一.要正视中共国和俄罗斯给西方带来的威胁；二.希望跨大西洋各国以及拥有相同价值观的西方社会统一战线。
而且他通过德国增加国防支出、对本国5G网络的立法保护以及首次向印太地区部署海军，这几个具体措施来表明本国的态度和决心。也向美国及西方其他国家发出了团结抗敌的呼吁。
马斯虽然把俄罗斯也列为需要防范的国家，但我们可以从以上三点行动中看到其主要针对的目标就是中共国。
在报道结尾处，马斯还直接表达了对大型科技公司的不信任，而且认为西方各国之间的很多误会都是大型科技公司所引发的。
