## 管窥CCP治下的腐败乱象——基层科研工作者的生态环境
`2020-07-02 19:49 秘密翻译组` [轉載自GNews](https://gnews.org/zh-hant/252264/)

![](https://s3.amazonaws.com/gnews-media-offload/wp-content/uploads/2020/07/02194749/1-4.png)
随着美中之间的较量全方位的展开。无论是军事、情报、金融还是贸易领域的战斗都牵动着世界的神经。我这里想谈谈墙内的科研，毕竟当今的较量，无论是军事战争、情报获取、金融领域还是正在进行的如火如荼的“生化战”（冠状病毒）都或多或少的与科研有关。

墙内的科研究竟是什么人去做呢？不了解的可能会说，应该是大学的教授、老师，或者科研院所的人去做吧。其实这里我要说的是，如果您了解墙内科研，您就知道，墙内科研多数情况下都是学生去做的。很多科研院所的项目也会给学校去做，这是为什么呢？因为学生的报酬少，比较“省钱”。很多学生在专家教授的眼里就是廉价劳动力。再加上墙内的所谓“奉献精神”，还会让不少学生都觉得免费或者在只拿少量劳务费的情况下干活是应该的。

墙内的学生，尤其是研究生和博士能否毕业，几乎是你的指导教师一句话的事情，因为最后需要他签字认可。在一个缺少监管的体制内，可想而知这种模式会导致什么事情发生。一些博士，不仅会成为老师的实验员，还会成为老师的秘书，老师的助理。帮老师取快递，给老师带饭，帮老师搬家，帮老师布置会场，帮老师打扫办公室卫生，这些对于很多博士来说都是家常便饭。有时，只要你有一点事情办错了，如果遇到很差的导师，导师心情不好，你可能就会遭到劈头盖脸的谩骂。带家长的谩骂也会经常挂在一些老师的嘴边。

仅仅是这些吗？如果你遇到那些禽兽教授，如果你是一个有姿色的女孩子，会发生什么事呢？只要你束缚于那个毕业证，只要想毕业，或者你想出国，你想在老师那里拿到任何的利益，你就不得不屈服于老师的淫威之下而无法自拔。有时也会有忍无可忍的学生鼓起勇气向学校反映一下情况，但是一般情况下，都会因为所谓的怕造成不良“影响”再次妥协。而且事后你几乎无法和老师再好好相处，你面对的可能只有退学。

上述情况导致每年都会有一些学生有轻生情节。而目前国内博士年限已经调整至8年，也就是说，如果你对你的老师有意见，且还达不到毕业标准，要么你忍8年，要么你就退学。此外也可以申请博士转硕士，但是很多高校都会收取你博士期间的补助费，补助费都退还后，你才能转成硕士。

再来说说待遇，作为科研的主力群体，博士，本应该是一个国家待遇不错的群体。即便他们还是学生，他们也是一个学校或者一个科研院所的科研主力。但是他们的待遇究竟如何呢？前几年，在清华某牛人的建议下，博士每月的补助终于涨了，每个月涨了250元。也就是说，一名博士每月的国家补助为2000元上下（硕士更少，每月几百块）。在这个物价飞涨的年代，这些钱如何让一个博士能够真正致力于科研？即便只有每月2000，你如果觉得你无法读下去，想转成硕士，学校也会把你博士期间所得的这些补助收回去，才给你转硕。如此对待低收入群体，实在是不够仁道。

如果你遇到一个项目充足的老师，你可能会从他那里每个月多拿几百块或者1、2千块的补助，但是这毕竟是少数学生，多数学生还是只有2000多块的补助。试想，在这种情况下，有哪个博士愿意留在学校没日没夜地给学校、给老师写文章、做实验？即便是那些有所谓“奉献精神”的人，也基本都以迅速毕业，拿到文凭为目标。

再来看看工作时长，基本上是早8晚10。没错，那些项目充沛的课题组里，每天的工作时间基本上是早8晚10，中午可以稍事休息，吃过饭后回到实验室继续肝。有的老师甚至希望你住在实验室里。每周可能只有周日休息一天，如果这一天不走运，有些老师要开组会，也就是课题组里的会议，那周日也无法休息。

如此拼命，假如做的事情有意义也还值得，但我们来看看大多数博士和硕士究竟在做什么吧。

1、项目

多数课题组的博士、硕士会比较擅长撰写项目书，也就是帮助老师申请项目。项目申请下来后，比如你最后要生产出一种结构，达到某种抗压标准。如果你的老师关系够硬，那么你做的20个里面，有一个达到标准就算你项目通过。而这个由你们做出的20个表面上看起来一模一样的结构，由于加工工艺的差别造成内部细节有不同，这个符合规定的结构究竟为什么符合规定，是否是因为工艺水平不足、材料不均匀而偶然达到标准，能不能量产，无人知晓。

多数课题组在课题结束迎接专家组审查的时候，会请负责审核的专家旅游、吃饭、赠送纪念品。至于项目要求的目标，差不多就行了，甚至有的可能离项目要求差很远。此外，各种项目还有一个奇葩规定，就是第一年或前两年，必须花光项目经费的70%或以上（项目不同可能略有出入）。如果钱花不出去，到期之后这些钱就被会冻结，就再也用不了了。这导致项目审批下来后，各课题组不是去研究怎么做，而是先准备突击花钱，很多博士、硕士都会帮老师想办法花钱。但是很多项目，起步时可能花不了那么多钱，这就逼着很多人去开“空头发票”。只开发票，不买东西，您懂的。比如说，有的老师为了完成“任务”，让学生帮助自己去买一个往返车票，然后说去某地考察，但是实际上可能根本没有去，几天的差旅费就这样出来了。这些其实也不怪教授们，这种奇葩的花钱规定基本就是逼迫你这么做。把你拉进坑里，你以后就不好再对上面有什么意见了。博士和硕士们每到这个时候就会在财务和老师这里两面跑，有时一整天、一个礼拜都在跑这些所谓“报销”的问题。

2、组会

再来看看组会都做什么吧。所谓组会就是每周一次的学术会议，如果遇到有点学术水平的老师还可以，但是如果你们组的老师是靠行政水平上来的，那这个组会就很搞笑了。这些搞行政的老师，他们可能连最基本的公式都不懂，还自己装成什么都懂的样子，每次组会指导你的课题题目该怎么写，逻辑语法有没有问题，纠结你的参考文献、标点符号，PPT是否美观等问题，而一到真正的公式推导、实验分析等“干货”上就迅速掠过。如果遇到一个对于你的研究几乎完全不懂，还要每天指手画脚的装成什么都懂的教授，还有几个博士、硕士愿意多待一天？

3、论文（SCI）

再来看看发论文（墙内一般叫发文章）。每个博士需要在读博期间发表SCI收录的论文。先来说说什么是SCI。所谓SCI是指Science Citation Index，是美国科学信息研究所（Institute for Scientific Information，简称ISI）出版的一部世界著名的期刊文献检索工具，其出版形式包括印刷版期刊和光盘版及联机数据库，还发行了互联网上Web版数据库。每天宣扬大国崛起，但墙内博士毕业却要用美国科学信息研究所所列的杂志为标准，请问这是什么操作？

回归正题，在墙内，发文章基本上是每个墙内博士必须完成的任务，如果你四年内发不出一片SCI文章，那么你就要被延期毕业。延期毕业还不是最恐怖的，恐怖的是博士四年以后，国家不再给博士补助，每月2000元都没有了，你还要交学费和住宿费。按照规定，老师也可以不用再给你每个月几百的零花钱了。也就是说你真正成了一个零收入甚至负资产的群体。在这种情况下，很多博士的论文水平可想而知，即便是发布在顶级期刊上的杂志都有数据造假，甚至花钱雇人写论文的现象。为什么会这样？很简单，为了发文章而发文章，为了毕业而发文章，在一个没有信仰，监督机制缺乏或形同虚设的国度里，为达到目的就可以不择手段。我们知道，墙内一篇文章要有实验和软件模拟。为了让模拟数据和实验数据匹配，在模拟的时候，很多人都会通过修改各种参数来达到目的。但是这些参数修改之后，与真实情况究竟有多大差别，无人知晓，只要模拟软件的数据和实验数据吻合就行了。再加上很多老师每天除了催你干活，什么作用都不起，每天只会让你加快速度，文章的质量水平就更不用说了。而国外杂志的审稿人多数也不会去重新做你的实验，更不会去重新做你的模拟。如果你的通讯作者够牛，或者你写的文章词藻够吸引人，还是会给你通过发表。想想墙内的如此糟糕的科研氛围，再加上国外杂志主编和审稿人被蓝金黄的情况，墙内怎么可能有什么影响人类的发明创造产生？

再来谈谈各课题组是如何拉项目资金的。多数课题组为了项目资金，可能一整年都在“拉项目”（获得项目支持）。这里就涉及到，你们组里的大老板（学科带头人），人脉如何，在基金委里有没有熟人等问题，每年所谓学术“大牛”举办的学术会议你是否都参加，你的项目是否是“上面”喜欢的，是否能紧跟他们的步伐。如果你有别出心裁的项目，并且可能挑战了这一群学术领域的既得利益者，那么你的项目很难通过。如果你在申请项目之前，没有事先和这些大牛打好“招呼”，你的项目几乎很难通过。所以，很多人，即便是“大牛”，每年的精力也都放在了开会、拉人脉、聚餐、找项目上。这种情况下，老师们基本没时间，也没精力去自己搞科研，而他们的学生们，在老师们参加会议期间只需做好秘书角色，替导师订机票，订酒店，做个PPT。如果你参加的学术会议多了，你就会发现，其实有的所谓“大牛”或者特邀专家，他们每次学术会议的演讲都是一样的内容。甚至有的人的PPT都不会有改动。这些学术会议并非主要讨论学术问题，亲近关系、保持联系、在牛人面前混个脸熟、拉项目、聚餐、旅游等恐怕才是比较重要的。

科研本身是为了造福于人类，从第一次工业革命蒸汽机的出现，到今天生物克隆、航天科技、通讯技术的迅猛发展，人类的生活越来越得益于科技的进步。但是像国内这种浮夸风气、人脉社会、视基层科研工作者为廉价劳动力的氛围，想要和老美PK，是不是有点某生物想吃天鹅肉的感觉？
