#### 搜集/撰文：岁月如歌 / 封面合成：文粤
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/20015617/5201.jpg)
经过快20年（1988～2007）的拉锯，香港人追求民主普选的呼声越来越高，中共一拖再拖，最终在2007年12月中共人大常委会否决了2012年双普选方案时，提出了2017年香港推行普选的时间表。表示2017年第五任行政长官可以实行有普选产生，行政长官普选产生后立法会可以实行全部议员由普选产生，这次决定是北京当局首次承诺，首次为香港普选行政长官定下具体时间。但是2012年双普选被否定已经不可抵挡地激起民愤，香港人要求双普选被一次又一次的玩弄和推托最终愤怒至极，发起了大游行，港共矛盾在不断地加深。五区公投变质失败和刘晓波因言获罪在社会上掀起的巨浪对香港人更是恐惧，香港人明白只有争取才有机会得到自己的权利。

香港的政制改革在提出2017年行政长官可以普选之后主要议题就是围绕这个进行讨论和准备。
行政长官产生方法的重点：

1. 提名委员会的选民基础；
提名委员会的产生办法；
3. 政长官候选人的程序；
4. 普选 的投票安排；
5. 任命行政长 与本地立法的衔接。

而立法会产生的办法的重点：

1. 立法会的一席数目和组成；
2. 功能界别的组成及选民基础；
3. 分区直选的区数目和每个选区的一席数目。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/20015634/5202.jpg)
香港特区政府为了表示落实普选的诚意，充分考虑市民的意见由政务司师长林郑月娥领导政改咨询专责小组于2013年12月4日宣布进行政改方案前的公众咨询。律政司司长袁国强和政制及内地事务局局长谈谭志源一起组合成”政改三人组“处理政改事宜。首次咨询主题是”有商有量，实现普选“。 2014年3月3日政务司师长林政月娥发表文章《让普选行政长官讨论回归〈基本法〉》，表示如果各方不回到《基本法》的法律规定及不接受政治现实， 2017年普选行政长官这个目标恐怕会打水漂。林政月娥发表这篇文章的时间这是首轮公众咨询期间，说明特区政府收集到市民对政改的意见并不符合中共理想，只好搬出《基本法》来牵住香港市民的双普选要求。大家都知道否定200708、2012年双普选都是根据《基本法》45条和68条以及附件，如果还按照《基本法》，2017年双普选也只是一句空话。港府此时发表文章要市民必须按照《基本法》的法律基础和接受政治现实，否则2017年双普选会打水漂。这么说按不按照《基本法》都是一个结果——镜中花水中月。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/20015647/5203.jpg)图片来自中央广播电台
2014年4月还在首轮咨询期间立法会安排全体议员访问上海，并拜见中共主理香港政改官员， 有43名建制派和14位泛民主派议员参加。长毛梁国雄在访沪之前就宣扬他将带有关”64“的东西回大陆，结果在边境被要求交出相关物品，梁国雄没有入境返回香港。这个小插曲也可以看出中共对八九六四有多紧张。 2014年6月10日中共国务院新闻办公室发布《一国两制在香港特别行政区的实践》的白皮书，单方面解释「一国两制、高数字高度自治」是什么。说香港社会有人对「一国两制」方针政策和《基本法》认识模糊，理解片面，有必要解释一番。 「一国两制」方针中所指的香港高度自治限度在中央授予多少权力香港就有多少权利，并指出在一国两制中，两制只能从属于一国，特首人选必须爱国爱党，特首与立法会普选制度都必须符合国家主权、安全和发展利益，符合香港实际，兼顾社会各阶层利益，体现均衡参与的原则，有利于资本主义发展，特别是符合香港特别行政区作为直辖于中央人民政府的地方行政区域的法律地位，符合香港基本法和全国人大常委会有关决定的规定。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/20015656/5204.jpg)图片来自自由时报
中共主动单方面对「一国两制、港人治港、高度自治」做出解释，也就说明中共意识到香港人对双普选的决心。走下去必定不是中共想要的和能掌控的，必须搬出看家法宝来阻止。告诉香港人所谓的高度自治就是中共给你多大的权利你就有多大的权利，中共不施舍港人，港人什么都得不到。旨在警告港人别痴心妄想。这《白皮书》一出来，引起香港各界纷纷议论，《明报》表示中共处理香港事务由过去的宽松变为收紧，中共全面掌控香港事务；港区政协委员何柱国的《星岛日报》社论表示，中共拥有对香港的全面管制权并没有什么新意，只是中共首次由此详细的阐释对香港的权利而已，是为了打破一些人的幻想。 《苹果日报》时事评论员林和立表示，中共中央总书记习近平对国际认可的法律条文《中英联合声明》及《基本法》的修订版，而新的演绎违背了邓小平的意愿，习近平这样做也违背了国际法精神，将会使国际投资者对香港失去信心，破坏香港作为国际金融中心的地位。通过这些报导可以看出香港人对于白皮书完全不看好。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/20015707/5205.jpg)图片来自风传媒
2014年8月27日全国人大常委会副秘书长李飞就政改决定草案所做的说明再次强调行政长官由「爱国爱党」人士担任的原则。 2014年8月31日全国人大常委会正式通过决议，为2017年特首普选方法舍下框架，提名委员会要按照第四任行政长官选举委员会的规定组成，维持1200人，特首候选人规定在两道三人，每个候选人需获得提名委员会半数支持才能成为正式候选人，是原先八分之一选委门槛的4倍。提委会各界别的划分以及每个界别有哪些组织可以产生提委会委员可在本地立法成面处理。中共接二连三的发表白皮书讲话报告，得有多恐惧啊！它非常恐惧香港人双普选，恐惧被香港人选出不是中共控制的、喜欢的狗腿，中共多紧张啊，就像要倒台了的恐惧，赶紧阻止再阻止。为什么这么多年都在拖在敷衍香港人民主选举，一下子都暴露出来了，真是太可笑又可悲的强盗政党！
