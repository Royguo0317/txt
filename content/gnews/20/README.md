自2020年9月中旬印度新冠确诊人数平均每天超过93,000例的峰值以来，感染率开始稳步下降。截至今年2月中旬，印度平均每天统计11,000例病例。该疾病每日死亡人数的七天滚动平均数已降至100人以下。

就在一个月前，当欧洲疲于应对病例激增和施行严格的封锁时，印度各地普遍认为，经过2020年11月左右第一波疫情，印度人获得了群体免疫和自然抵抗力，可以避免疫情第二波的爆发。自去年底以来，战胜病毒的喜悦情绪一直在增长。政客、决策者和部分媒体认为，印度真的走出了疫情困境。

然而这种情况很快就被第二波疫情大爆发彻底粉碎，就在本周，印度确诊人数再次超过巴西，成为全球受影响第二多的国家，病例总数超过1368万。每天都有新感染病例的新纪录；周二，这个数字是161,736人，而死亡人数也继续攀升，到周日，情况进一步恶化，有超过270,000确诊病例和1,600多例死亡。根据《柳叶刀》杂志新冠病毒委员会的报告，即使不考虑恒河边的传染情况，到6月的第一周，印度每天的病毒死亡人数也可能会突破2300人。

印度医学研究理事会（ICMR）在周四召开的新冠病毒工作组会议上指出，在第二波冠状病毒疫情中，0～19岁年龄段的人群受到影响的比例比上一次更高，儿童现在也有被送入医院重症监护室的病例。

科学家和当局继续关注世界各地正在出现的病毒新变种。据印度卫生部称，截至3月底，印度已确认770多起由新冠病毒新变种引起的病例 。

疫情已经使印度的医疗系统濒临崩溃，医院内住满了新冠病毒病人，由于没有病床，实际上许多患者仍在家庭隔离中，当患者测试呈阳性时，他们才会被送往非专科的医院。

随着死于病毒的人越来越多，火葬场已经被尸体占满，有些家庭甚至开始在空地上焚烧尸体。

![]()![](https://gnews.org/wp-content/uploads/2021/04/图片16-3.png)



到目前为止，印度已注射了近7600万剂疫苗，完全疫苗接种率还不到该国13亿人口的1%。

专家们说，印度政府宣称印度在战胜新冠病毒方面具备独特的条件–人口更年轻，具有本地免疫力，大部分是农村人口，为时太早。是官方的狂妄自大，超民族主义，民粹主义和大量无能官僚共同造成了这次危机的爆发。

盲目乐观战胜不了中共病毒，甚至人类现有的医疗技术也未必能彻底预防它，闫丽梦博士早在一年前就告诉世界：人们对这种新型病毒所知甚少，靠有限的数据研发出来的疫苗无法有效抵御病毒；加上病毒强变异的特点，人们也不要指望所谓的群体免疫能对这种病毒有效。在这种空前的超限生化武器面前，人类的时间已经不多，唯一能有效解决病毒危机的办法就是推翻病毒制造者中共政权，彻查武汉病毒实验室，拿到原始病毒的相关数据，及时、有针对性、有依据地研发相关药物。
