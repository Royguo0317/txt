德国之声中文网：6月6日（周日）美国参议员达克沃丝（Ladda Tammy Duckworth）、苏利文（Daniel Scott Sullivan）和昆斯（Christopher Andrew Coons）乘坐C-17运输机抵达松山机场。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/06215249/Untitled-design-2021-06-06T194711.804.png)
三位参议员是从韩国赶来，他们目前正在韩国进行访问。在此次访问中，他们还与台湾总统蔡英文会面。民主党参议员达克沃丝演讲中表示“美国将捐赠给台湾75万剂疫苗。”这位参议员还说，“我们来到这里是为了告诉你们，美国不会让你们独自面对。我们会与你们站在一起，确保台湾人民得到其所需，来战胜疫情……因为我们认识到你们的迫切需要，重视我们之间的伙伴关系。”

共和党参议员苏利文还特意带着特制的口罩，上面写着“Love FromTaiwan”，他指着口罩说：“这也是来自美国的爱”。

这是在台海局势紧张的情况下，美国跨党派议员又一次合作。此前，中共国多次表示愿意提供中共国的疫苗给台湾，不过台湾以中共国疫苗不安全为由拒绝了。现在美国愿意捐赠75万剂疫苗，并且是在第一批捐赠的名单上，台湾欣然接受。可以视为这场疫苗较量，美国打脸中共国。

值得注意的是，他们这次乘坐的飞机是美国空军的C-17运输机，有别于以往美国高级别访客一般乘坐的行政专机。

由于中共国军方一直对美国放言：“美军胆敢登陆台湾之日，就是我军收复台湾之时。”美国一直以来都是奉行“模糊战略”，避免直接触怒中共国敏感的神经。现在，美国参议员直接乘坐军机访问台湾，表明美国国会山与五角大楼在台海问题上达成共识。

这是国会与国防部的一次重要合作，是针对中共国的“反碰瓷”行动。一是为了表明美国的台海战略转向：由更为强势的“竞争模式”，取代以前的“模糊战略”。二是对中共国的试探，对中共国军方底线的试探。三是表明美国强烈支持台湾，不怕触怒北京的态度。

在D Day纪念日，（1944年6月6日，法国诺曼底登陆），美国军机高调登陆台湾，相信这具有重大的象征意义。
