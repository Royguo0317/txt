路德先生十分坦诚地说闫博士“脱胎换骨”，因为她不再关注镜头，而是全身心地投入，来表达内心的愤怒和良知。

郭先生问这些问题的原因，是因为每个家庭都可能遭受过的重创，威胁还有考验，但不是所有人都能支撑下来，家人背后的支持才是我们走到今天的动力。爱情是有条件的，信仰是改变感情的一切根基。而所有拥有信仰的人，都值得我们称之为英雄。

面对全世界各个农场呈现的视频，闫博士说，这都是心血之作，说明了中国人的能力。但为什么中共治下的百姓那么懦弱和畏惧？是因为制度，我们看着香港从世界顶尖位置瞬间崩塌，给闫博士带来极大的冲击和震撼。当大家面对病毒真相时，她希望我们能将勇气带给更多的中国人，一定要将无辜的中国百姓从诛心的环境中拯救出来，义不容辞。
