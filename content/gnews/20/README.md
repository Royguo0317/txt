货币的发展从金本位到信用货币体系再到即将到来的数字货币，这一切的发展变化都与社会生产力的发展技术的变革以及各方利益的斗争分配有关。本文所谈到的G货币（HCoin、GDollar）产生于加密币技术日益成熟，购买日益网络化，主权信用货币竞相超发，以及中共利用一带一路推进人民币国际化挑战美元霸权绑架全球金融及其即将覆灭的时代和事件背景之下，可谓是生逢其时！下面按照货币的发展顺序进行叙说探讨。

**一、货币的发展历程**

货币产生的目的便是商品的交换，由于物物交换很难找到价值相等的双方，因此必然导致“货币”的诞生。那么至于“货币”是由一般等价物演变而来，还是天生就是“账本”，我们现在还无从考证，但是很明确的是，货币的前身并不具有主权特性，因此选择具有稀缺性的贵金属作为货币成为必然。

**1、金本位**

“金本位“下大英帝国横扫全球具有一定的必然性。在地球的稀有金属中，唯有黄金的储量具有稀缺和稳定的特性，按郭先生所说，全人类公开的黄金储量也就2.86亿盎司，它的诞生来源于超大恒星死亡时的超新星暴发，在整个宇宙都具有稀缺性，在可见的未来储量不会发生大的变化。因此，黄金的稀缺性和储量稳定性充当了国家的信用，即不需要任何人背书就能保证收款方的利益，因为充当货币的每一盎司黄金都实际具有价值。那么”金本位“的问题在哪里呢？我们都知道货币的总量最好一直等于流通中商品的总价值，这样的货币价值是恒定的，即没有通胀与通缩。但是随着社会生产力的大发展，商品的总流通价值会迅速增长以至于远远超过黄金的开采速度，那么我们现在看到的新款iPhone年年涨价的情况在”金本位“下会变得很诡异，iPhone年年都会降价！于是人们会倾向于把货币储存起来，而不是进行消费，因为货币的购买力年年涨，这就会打击消费，最终影响生产力发展，把社会经济规模拉下来最终达到平衡，这可能也是”金本位“体制下的全球经济没法超越”信用“体制下的全球经济的原因。

**2、信用货币**

”金本位“体制必然导致钱不够用的诡异情况，为了解决这种情况那就只有造假，比如布雷顿森林体系规定35美元兑换1盎司的黄金，可是最后印制的美元远远超过美国的黄金储量（香港联系汇率规定7.75至7.85兑1美元，可是港币的数量究竟几倍于香港银行美元的存量呢？(¬‿¬)），这也最终导致了“金本位”退出了历史舞台，取而代之的是以国家信用背书的信用货币体系。那么现在我们所使用的信用货币究竟是如何发行的呢？

以美国为例：

首先是美国财政部以税收为担保发行国债，美联储发行美元买入国债，这样财政部就有了美元可通过投资、补贴等政策将美元投放到市场中；

其次商业银行通过向美联储抵押国债、票据等抵押物，借到了美元（贴现），然后再把美元借给市场主体，这时，有一些很奇妙的事情发生了，借给商业银行的美元，会魔术般地变为实际数量的好多倍（货币乘数）。

假设：

A拿到了银行货款——存了10%，付款90%给B；

B把钱存进了银行——银行又有了相当于A货款金额90%的存款；

银行又可以把这90%的钱借出来，然后可以观察到这钱打个折扣又回来了银行，周而复始，这样一笔钱进了商业银行，大概可以变也10到15倍的钱来，每个人看到自己账户里都有这笔钱，但如果大家都去取钱，会发现这笔钱取不出来，而美联储（中央银行）可以通过要求商业银行上缴一定比例的存款准备金，来控制货币乘数，也就是控制流通货币的数量。除此之外，美联储也可以控制商业银行的借款利率，也就是通过控制成本抑制借款人的借贷金额；更可以直接在国债市场通过购买来投放美元，通过卖出来回收美元（公开市场操作），这就有了主权国家对经济的控制能力。

中共国呢？在朱镕基时代，奠定了以外汇（主要是美元）为锚印刷人民币的制度，这就让人民币相当于美元的代金券，使人民币在其后相当长的一段时间里成为中国经济坚实的基础。但是，随着新时代的到来，这基础早已打破，拿永不偿还的地方债来印钞已经见怪不怪（相当于无锚印钞）。

黄金美元结束了，信用美元需要找到一种基础商品进行定价锚定，才能透过这种商品与其它商品的比价成为世界货币（度量衡），这种商品很自然就是大宗商品之王——石油。美元很自然地搞定了OPEC国家，于是石油美元的时代开启了，信用货币时代的美元再次成了世界货币。

一切看上去都那么美好，对吧？美联储有了完全控制货币的能力，其它主权国家的央行也一样，一切尽在掌握！可惜，人性的贪婪终将毁掉这一切。每一次的经济危机，我们都有机会让货币回归正常，多余的货币该回收就回收，保证货币的价值，但是，有人哭了，大而不能倒，我们是吃饭了，但是他们应该买单啊，不然受伤的还是他们……于是，吃饭的永远是权贵，买单的永远是平民（老百姓）。

**二、数字货币时代到来**

信用货币带来了很严重的“信用”问题，因为掌握货币发行的其实是一小撮“权贵”，很容易就通过财政政策、货币政策等各种充满伪装的“工具”转移属于百姓的财富，如：温和通胀有利于经济发展，结果谁存钱谁倒霉；大而不能倒，要救，要QE，结果该淘汰的没淘汰，狗犯错让猪买单……因此，数字货币、区块链、公共账本的诞生正是信用货币这套不公平体制走到历史尽头的必然结果。

**1、比特币**

比特币就是在此背景诞生。比特币作为延生的第一款实验性“货币”，成功验证了公共账本数字货币的可能性，当然，它并不具备成为货币的可能性，这在后面再说。

比特币的天才设计中：

（1）由于诞生于思想实验，没有能力使用任何有价值的“等价物”，于是使用了“成本”为锚，人为让其有了“价值”（挖矿成本），这就是所谓的POW机制——工作量证明；

（2）安全性。比特币用的公钥私钥（非对称加密算法）体系早就在计算机系统里应用多年，公钥加的密只能用私钥解开，私钥加的密只能用公钥解开，所以，能用私钥解开的比特币，就是属于你自己的比特币，花钱（转账）也只不过用别人的公钥（所谓比特币的地址）进行加个密而已，这样这笔钱就只有私钥持有者才可解开使用，而以目前的通用计算机的能力，宇宙湮灭也无法通过公钥算出私钥，这就保障了资金的安全性；比特币的区块链实际就是个按块组织的账本，个人资金能够保证安全，那如何保证账本不被篡改呢？比特币的每一个区块（一页账本）都通过“Hash函数“算出一个特征值，这个值的特点是只要改动区块内任意一个数字，值都将发生改变，而每个区块在计算特征值时都会带上上一个区块的特征值进行运算，也就是说，想要改动某一个区块数据，就得把这个区块后面的所有特征值都进行改动，而且这个账本不是保存在某部机器中，在比特币网络的所有挖矿机器都有一份相同的账本，这几乎没有可能；

（3）唯一性。去中心化系统存在时间不同步的问题，如果全部机器都同时进行记账，那么很快就会因网络延时及广播的先后顺序造成各个区块的账本不同步。为了解决这个问题，就不能让所有机器都进行记账。于是，比特币网络在计算每个区块的特征值的时候，又带入了一个Nonce变量进行计算，Nonce值可以由每部希望记账的机器随意改动，但只有试出某个Nonce值使得计算出来的特征值符合比如：开头为N个0的16位数字，才有记账的资格。这就大大减少了有资格记账的机器数量，偶然遇到同时计算出结果的几台机器的情况，比特币网络使用长链原则，即其它机器只会选择在最长的区块链上进行下一个区块的记账，这就是比特币网络转账需要等待几个网络确认的原因。一般来说，两个区块的网络确认就可以了，六个区块的网络确认几乎没有风险；

（4）经济性。比特币诞生的目的是创造一种去中心化、没有通胀、自我运行的数字货币系统。这个系统中，计算出正确Nonce值的进行记账的机器将得到比特币奖励，这个奖励从第一区块的50BTC开始，每210000区块减半，而比特币网络通过控制特征值开头0的数量便可以控制计算难度，控制的结果是无论参与计算的机器算力如何增长，平均需要四年才能开采210000区块。也就是说，每个比特币的开采成本是逐年上升的，而以”成本“为锚定物的比特币理应波动上涨。矿工的酬劳还包括转账的费用，矿工会优先对愿意支付更高手续费的转账进行打包。

比特币存在的问题：

（1）区块容量有限，即每一区块能够记录的转账信息有限，无法支撑大量交易；

（2）交易速度有限，每确认一次区块，交易才被确定下来；

（3）发行总量有限，即2100万个，虽然在设计上比特币的最小单位是1聪，1BTC=1亿聪，不存在支付上的问题，但如果成为真正的货币，将存在巨大的通缩，即大家都知道明天的钱更值钱，都不愿意消费；

（4）锚定物为”成本“而非”商品“。与金银货币不同，比特币的这个锚定物对别人没有任何价值，一个人的成本对另一个人没有意义，这可以从某些垃圾股失去内在价值后下跌一波，换手一批人成本归一后又下跌一波（筹码分布、行为金融学原理），直到归0的结局可以预示这个品种的最终结局；

（5）容易操纵。因为锚定物非商品，不存在商品与比特币之间的套利操作，于是价格操纵只取决于韭菜的高度（据文贵先生爆料：比特币主要是中共、华尔街操纵价格进行洗钱）。

比特币的意义：

比特币是这个世界对信用货币体系不信任的第一次挑战，它为去中心化的非主权货币指明了方向，验证了很多技术理念，它的任务其实已经结束，为后面将会出现的全球通用数字货币铺平了道路。货币不应是属于某个国家、某个组织、某几个人的权力工具，它应该公平地属于地球上每一个辛勤劳动的人！
