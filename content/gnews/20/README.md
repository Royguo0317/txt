## 布莱克本议员称拜登对中共的立场非常有问题

《布莱特巴特新闻》(Breitbart) 3月1日报道了对田纳西州参议员玛莎·布莱克本（Marsha Blackburn）的独家专访。在佛罗里达州奥兰多市，她对《布莱特巴特新闻》表示，美国总统拜登对中共的处理方式代表了一种 “非常有问题” 和软弱的方式，错误地将中共国视为 “盟友”，而不是这个太平洋势力上的“对手”。

当被问及拜登处理与中共国独裁者习近平的首次通话是否有成效时，布莱克本说，“这是非常有问题的，因为我们必须认识到中共国不是我们的盟友，他们是我们的对手。他们意图统治全球。他们对香港自由斗士、西藏人和台湾人实行侵犯人权的行为，他们对维吾尔族穆斯林少数民族人口进行种族灭绝，他们把这些人关进奴隶劳动拘留营，他们想消灭这些人口。谈到经济，看看他们在做什么？他们实行的是所谓的抢劫、复制和替换， 这就是偷你在他们那边制造时的知识产权， 他们以一个较低的价格点复制并制造它，然后他们在市场上取代你来摧毁美国公司和破坏美国的就业机会。我们已经谈到了他们的软宣传，有孔子学院和孔子课堂，还有姐妹城市项目，然后你再看看他们对大科技公司的做法。他们和大科技公司的关系是如此的亲密。大科技公司需要他们的超算芯片和微处理器，所以他们从来不会对中共进行封杀、节流、影子禁令、停用、去势，中共会把说反对中共话的人关进监狱。”

拜登与习近平通话中提到了与维吾尔族穆斯林有关的人权问题，但拜登后来在CNN承认他只是在通话中敷衍了事。布莱克本说，拜登对中共的行为缺乏认真态度这是危险的。

布莱克本在谈到中共国时说，“他们想要控制，中共正通过一些幌子试图以邪恶的方式影响美国文化。”

她还说，“有了孔子学院在我们的校园和课堂，有了对好莱坞电影公司的投资，是的，中共国深度参与投资美国，但他们所遵守的规则与美国公司所遵守的规则不同。我们需要拜登政府说‘你知道，唐纳德·川普在某些方面是对的’。如果他们不打算这么做，那就要由这个国家的人民说，我们必须继续川普的政策。”

布莱克本是在CPAC上接受《布莱特巴特新闻》专访时发表上述言论的，CPAC是美国保守派联盟（ACU）每年举办的大型保守派聚会。布莱克本还认为，拜登在总统任期初期的一些行为相当于给中共送了一份大礼，从能源到经济，甚至拜登的人事决策都是如此。

布莱克本提到，“这就是人们应该关注的地方——拜登的行政命令，并真正关注他在那里做了什么，他的大部分行政命令都是对谁有利的？中共国。你看看KeyStone管道。谁从加拿大获得这些原油？中共国，你看看《巴黎气候协议》，这不是要帮助美国工人，它将掏空140万美国人的工作岗位。当你谈论制造太阳能电池板时，这些工作不在美国。地球上有10家公司在生产太阳能电池板，其中8家在中共国。当你谈到风力涡轮机，是在哪里制造的？它们是在中共国制造的。当你说要取消制裁的时候，你说的是让中共国自由化。当你说我们要改变一些贸易政策的时候，你是在说对中共国有利。人们需要注意这一点——然后看看他提名的在他的内阁和他的政府中担任领导职务的人，他们是激进的。”

布莱克本特别指出，拜登的美国驻联合国大使琳达·托马斯·格林菲尔德 (Linda Thomas Greenfield) 在孔子学院发表了一次有偿演讲，她在演讲中赞扬了中共在非洲的破坏性的一带一路倡议活动。

布莱克本说到托马斯·格林菲尔德，“她是由孔子学院付费发表演讲的，尽管她声称自己是去上大学的，她也没有意识到这是孔子学院，但她赞扬了中共国和一带一路倡议，以及中共国进入非洲提供援助，并说与中共国合作是赢、赢、赢，我说这是输、输、输。”

对我们来说，这是国家的一个损失。这是我们盟友的损失，也是美国工人的损失。这是我们孩子的损失，因为中共国正试图通过这种软性宣传窃取他们的思想。
