2020年1月1日，日本共同社报道指出，一项对日本96家在外资投资监管名单中的上市企业的问卷调查显示，其中近四成企业（42家）正在调整供应链，要把生产基地从中共国分散至东南亚、印度等地，以此来规避因美中经济“脱钩”和供应链过于集中在中共国所带来的风险。另因澳洲公布的海外共产党名单大部分被证实已渗透在各国家核心通信、科技、军事等领域，针对美国近期对中共盗取信息及科技成果的指控，调查中近六成企业表示为防止敏感资讯泄露，会进行员工内部培训及重要技术锁定。

自日本前首相安倍晋三于2020年8月因健康原因意外提前辞职，9月16日菅义伟接任日本首相公开表示，“日本外交的基础是日美同盟，应在此基础上与亚洲各国进行紧密交往。日中之间存在许多难题，应当继续灵活运用高层对话之机会来逐一解决难题”。随着美中关系的持续恶化、日中钓鱼岛冲突升级及中共病毒疫情影响，不同与安倍时期的亲中政策，新首相继任后日中关系面临转型。

在安倍执政期间，日中关系出现了缓慢但显著的改善。安倍根据美中关系的调整，同时平衡国内利益，通过高层互访引领带动日中两国关系的改进。然而自中共病毒疫情之后，两国之间再度出现有关领土和价值观等问题的争端，迫使日本重新澄清其对华政策。安倍的辞职，更可以解读为日本政府对中共政策调整的需要。

2020年10月中共国船只在尖阁诸岛（中共国称之为钓鱼岛）周围的领海逗留和与日本船只进行了长时间对峙，打破了之前在该海域的停留记录，且对日本海岸警卫队要求离开该海域的要求置之不理。中共此举被看作是对日本新政府的一次试探。菅义伟首相对此试探做出了强硬的回应，在与习近平的第一次对话中直接谈及了钓鱼岛的紧张局势及对《港版国安法》的关注。

2020年10月底日本和美国展开名为”锐剑-21″的陆海空大型军演，以”展现日美同盟的力量”。11月初日本又与美国、印度和澳大利亚在孟加拉湾展开“马拉巴尔2020”联合演习。自此，“四方安全对话”(Quadrilateral Security Dialogue，简称Quad)四个成员全数到齐，共同抵抗中共的军事扩张。

日本与美国、印度和澳大利亚举办了 “四方安全对话”，以推进 “自由与开放的印太 “为议题，讨论了包括东海、南中国海在内的形势，被认为是对中共国在亚太海域谋求主导权的反击。日本首相菅义伟上任后出访了越南和印度尼西亚。菅义伟表示，日本将帮助东南亚国家联盟（东盟）的成员，包括印度尼西亚和越南，打击非法捕鱼导致的海上冲突，意指中共。此外，菅义伟和澳大利亚总理莫里森对《互惠准入协定》达成一致意见。这是两国在印太战略框架下达成的具有里程碑式的防务协定。

因军事冲突和疫情影响，日本民众对中共国的观感极具恶化。根据日本11月17日公布的调查数据，近90%的日本民众对中共国的印象表示“不好”，中日之间的友好关系越行越远。

援引自台湾《旺报》的社论：“(日本防相)河野以‘扩张主义国’指称大陆，期待‘五眼联盟’纳入日本成为‘六眼联盟’，此无疑向呼吁‘建构对中包围多国联盟’的美国国务卿蓬佩奥立下‘投名状’”。

随着美中角力白热化加剧，日中关系也面临拐点，等待中共病毒为生物武器真相大白天下之时，各国对中共的围剿之势将不可逆转。中共末路狂奔，时日无多了。
