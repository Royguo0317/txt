# 内华达州共和党传讯车管局（DMV），得到数据显示有数千名非公民选民

**内华达州共和党从该州车管局获得的数据显示，数千名非公民选民被注册参加投票，并最终在2020年选举中投出选票。**

[!\[\]()!\[\](https://spark.adobe.com/page/hydV5gclgPtto/images/24670f92-ad0a-4291-98b6-04096bc0b70f.jpg?asset_id=9fa287ab-f9de-4f78-aba0-b21b97858f78&amp;img_etag=%22f2ce5577e919cd565b0ae491a70e04c0%22&amp;size=1024)](https://spark.adobe.com/page/hydV5gclgPtto/images/24670f92-ad0a-4291-98b6-04096bc0b70f.jpg?asset_id=9fa287ab-f9de-4f78-aba0-b21b97858f78&amp;img_etag=%22f2ce5577e919cd565b0ae491a70e04c0%22&amp;size=1024)

因为内华达州车管局（DMV）将所有申请驾驶执照的人自动注册为选民的真相爆出，使这个关键的摇摆州一长串选举欺诈活动的记录又多了一项。内华达州共和党解释说，由于合法和非法的非公民都可以持有驾驶执照和身份证，如果DMV无法做到“尽职调查”，非公民可以轻松地被注册进行投票。

根据一次传讯，该州的共和党得到了显示绿卡持有者和获得驾驶执照的非公民的数据。根据这组官方数据，内华达州共和党“将这一详细信息与内华达州的县选民记录进行了比较”，结果“发现有6260名非公民被注册投票，而其中3987名非公民已经投票。”

该小组的数据分析师杰西·卡姆佐尔在其签署的一份“愿接受伪证处罚”的声明中，概述了他的发现，并强调“值得进一步调查。”

**具体地，卡姆佐尔的发现：**

*把DMV内部文件同选民文件进行比较，有6,360人次相匹配*

*这些匹配的注册选民记录中，有6,136（96.5％）人的选民登记记录处于“活跃”状态*

*在匹配的注册选民记录中，有3,987（62.7％）个人已投票，并且投票显示已被计数— 也就是说，根据从电脑系统（SoS）和/或县选举部门收集的每一份数据，这些选票未被显示为临时或被拒绝的邮寄选票*

*那些匹配的注册选民记录中有4,546个人（71.5％）显示已尝试投票，包括从SoS和/或县选举部门收集的每份数据中的临时投票和拒绝的邮寄投票*
