公民投票，是人民行使创制、复决的重要性权利，系直接民主的行使。但公民投票针对重要性事务，人民应该享有充足时间真正了解公投议题，并且真正了解社会各界对于公投议题的主张，理性衡量后作出决定，才是公投真正的目的。

**何谓“捍卫主权法案”？**

捍卫主权法案，其核心目的就是要恢复已经终止适用的“国家统一纲领”。

已經終止適用的[國家統一綱領](https://zh.wikipedia.org/wiki/國家統一綱領)，應遵守的四大原則，分列如下：

1. 以全民的福祉为依归，而不是党派之争。
2. 保障基本人权，实践民主法治为宗旨。
3. 大陆与台湾均是中华民国的领土。
4. 首应尊重台湾地区人民的权益并 维护其安全与福祉。

彭迦智所提出的公投主文，虽说目的是要恢复“国家统一纲领”，但主文提及应遵守的四大原则却有删减。如下图。

若“捍卫主权法案”公投结果是希望国家统一纲领恢复适用，那可就着中共魔道了。

呈前述，两岸法律上国名皆为中国，若台湾公投结果恢复国家统一纲领适用，便是向国际宣布，台湾有意继续推进实现统一。

有意实现统一不是重点，且每个人都有自己的政治立场，都应备受尊重。

若世界大多数国家都承认中共才是合法代表政府，反面推之，中华民国政府是“流亡、叛变政府”。

配合“恢复国家统一纲领适用”，正式对外承认自己为流亡、叛变政府。给予中共一个武力征服叛变政府的理由，且可能导致国际在第一时间无法介入、提供援助。

虽说国家统一纲领内容，大多是希望透过和平方式完成国家统一。但希望以和平方式推动统一只有单方，中共以武力征服的意图日渐明显。

若恢复国家统一纲领适用公投通过，就是给予中共合理侵犯机会。



**当初为何要制定国家统一纲领？为切割统独政治议题，发展民主法律议题**

国家统一纲领制定，由时任总统李登辉(国民党籍，蒋经国总统逝世由副总统李登辉继任)推动制定。

李登辉前总统，对于台湾民主法治发展有特别重大且无人能及的贡献，无庸置疑。

之所以制定国家统一纲领，是当时两岸面临开战情形下，不得以而为之。

经李登辉总统于公开演讲的答覆、前幕僚张荣丰的采访内容，是为**“阻绝统独争议，全力推行民主改革”**。

（[風傳媒關於「李登輝為何制定國家統一綱領」相關新聞報導鏈接](https://www.storm.mg/article/2901292)）

李登辉自蒋经国于1988年死亡，以时任副总统资格继任总统之位，直到1996年才举行第一次正副总统人民直选。在1988~1996年期间，李登辉前总统为何不举行人民直选，直接运作民主制度？

根据作者之前的文章，强调中国合法执政政府为何，是因为两岸发展都受到了“合法地位争夺”的拘束。

中华民国政府1949年退守台湾，主张自身为合法代表中国的南京政府，疆域为大陆、台澎金马及其他区域。又依照中华民国宪法，当初以有推选出各省国民代表，但随后便退守台湾。

在中华民国政府失去对大陆疆域的实质控制权后，要依照原先针对“全部疆域”所制定的中华民国宪法，为不可能。

根本原因在于，丧失控制权的大陆疆域，无法选出新一届的国民代表。当初退守台湾的国民代表，组成国会后，不断延任，任期长达“43年9个月”。（俗称：万年国会。）

因此，在蒋经国前总统逝世后直到1996年实施第一次人民直选，期间李登辉前总统便致力制定“中华民国宪法增修条文”，区分大陆、台澎金马地区。

而依照宪法增修条文，中华民国得在实质控制领域内，继续施行民主制度。

然而，当时两岸充斥着统独假议题，深深影响政府推动民主化作业。李登辉前总统颁布“国家统一纲领”，目的就是要将“统独的政治问题”予以切割。在切割政治问题、法律问题后，便专注在民主制度塑造的法律部分。

从这一历史事实便可知道，中共定义李登辉为大台独为错误。当时，李登辉政府颁布国家统一纲领，目的全力形塑现行台湾民主法治，根本不处理统独问题。

而两岸发展，也因为李登辉政府的作为，有了本质上的差异。台湾朝向民主发展，且民主法治发展融合了中华文化，取得了巨大、无法抹灭的成功。

以个人理解，台湾民主法治发展，虽不如美国般光鲜亮丽或纸本上先进。但是在实质运作下，台湾民主法治实在有自己傲人之处。

一双鞋再怎么漂亮，不合脚总是没法穿是吧？牺牲一点光鲜亮丽外表，制定合乎自己民族特色的法治、民主制度，穿得舒服是重点。

中华文化，儒家思想，讲究“人伦”，以人为本。制度设计，是为最大程度排除人治的因素，但制度却要规范由人结合而成的社会运作。

重“人”的中华文化，在台湾，李登辉政府花10年时间，便将两相异的制度予以融合。你说值得不值得骄傲？

即便现行台湾看似混乱，但制度形塑确实的融合了中华文化、西方民主法治先进观念于其中。
