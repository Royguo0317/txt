## 马克龙：在需要的情况下我会接种疫苗
` 喜马拉雅巴黎七星农场` [轉載自GNews](https://gnews.org/zh-hans/657406/)

编译：Lina 校对：Ikonic

12月4号法国总统马克龙就疫苗接种问题在法国民众中所产生的分歧接受布鲁特 ( Brut ) 的视频采访，当被问及是否会像奥巴马，布什和克林顿承诺的那样“在摄像机前进行疫苗接种”时，马克龙说：“在需要的情况下我会接种疫苗。”

总统补充说：目前，我们并没有要求要对所有人实行疫苗接种，他不认为法国会强制接种。首先接种疫苗的是最脆弱的老人。

根据法国民调机构IFOP在12月3日进行的民意测验中显示，有61％的法国人不打算注射疫苗。因此总统强调，这次法国对疫苗的选用将不仅仅取决于实验室的数据，而是基于专家的独立而又透明的报告。尤其是要听取卫生总局（HAS）的建议。

另外，法国总理让·卡斯特克斯(Jean Castex ) 12月3号称，疫苗接种将在1月初从养老院的老人和工作人员开始，2月份，将依次邀请老年人和长期病患者以及照顾他们人接种疫苗。要在入夏前逐步完成对治安人员，教育界和成年人的疫苗接种。
