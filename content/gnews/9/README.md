投票安全功能的秘密

没有多少人充分意识到美国人使用的每张投票纸票都有多种类型的安全功能。

以下信息是从美国专利局获得的，并摘录了（以下）其中的一些安全功能。

“…在印刷正式选票的过程中，嵌入打印机纸浆和/或印在选票上的多种安全特征。

在安全特征包括明文或加密数据的情况下，光学扫描仪能够解释数据并验证数据。

此外，如果印在选票上的安全特征包括一个预先分配的选票序列号，光学扫描器将能够核实存在的序列号是为该次选举授权的，并且尚未被处理。

上述具体特征可以包括但不限于：

**UltraViolet ( UV ) Security Features ****–**

紫外线**(UV)**安全功能

这些特征在”普通白光”下看是”看不见”的，但在”紫外线”光源的”照射”下就会变得可见……[也可以拥有]”吸收紫外线”的特征；通常，这些特征是[“油墨内的特殊配方化学基质特性”。

红外线（**IR**）安全功能

这些特征在正常的白光下是看不见的，但在红外光源的照射下就会显现出来。它们也可以是吸收红外光的特征。典型的是”油墨”。

磁性安全功能

这些都是具有特殊磁性的特征；通常，”磁性材料嵌入纸中”的条状物，然而，也有”磁性油墨”可供选择。

磁性可以是“简单的—比如均匀的磁性—或者是”复杂的”，比如”沿着它有不同磁性强度的材料条”，它可以代表一种”模式”或”数据”。

荧光安全功能

这些特征在“普通白光”和“荧光灯”下观察时，可能是可见的或不可见的，具有“预期的强度范围”—当被某些”光频率”[光波频率]照亮时；通常，这些是”油墨”。

油墨**(**可见**)**安全功能 **–**

这些特殊的防伪特征由”非常独特的可见标记”组成，在”正常光线”条件下很容易检测到。

油墨（隐形）安全功能

这些特殊的防伪特征包括”看不见的化学配方矩阵标记”，只有使用”高度特殊的实验室方法设备”才能检测到，以确定投票纸墨水（本身）是”官方”还是”欺诈”。

它们通常在正常的白光下可见，但”无法通过印刷技术复制”。

水印也可以，拥有。

– 紫外线灯功能。

– 红外线灯功能。

– 荧光灯功能；或。

纸张安全功能

静态[安全]功能组指的是以下情况下

– 安全]功能被放置在’纸质存货’中[内]，并由投票单位寻找[光学扫描]。

这些静态[安全]特征”不包含数据”，因此安全特征仅由”存在[安全]特征”或”不存在安全特征”组成。

通常情况下，静态[安全]特征可以（例如）由选票上”设定的位置”或”设定的区域”中的”标记”组成，这样静态[安全]特征的”定位”就不会改变。

通常情况下，[ 安全性 ] 功能是”静态”的，因为它们”改变起来很昂贵”。

例如，将磁条[嵌入]”纸浆 “是一个 “相对昂贵的工艺”。

这样的应用相对难以改变，所以，同样会应用于大批量。

水印是另一个安全功能的例子，因为它通常是静态的。

美国选举投票过程基础知识

根据我23岁时在大学政府课上所经历的上述粗暴觉醒，美国总统选举并不像几十年前我的教授所介绍的那样，完全由人们向其所在的州区选区提交选票决定。

总统选举基本上是通过”选举团”程序完成的，其”选举成员”(Electoral Member)(选民)根据”他们的特定政党”而”数量不同”，再加上每个州的”地区”投票措施，考虑到”选民人口统计”，提供”预选数量”的”选民”，将他们的选票”提交”选举团”的计票程序，最终决定”谁”实际当选为”美利坚合众国总统”。

重新划分选区的投票措施，直接关系到”有多少政党选举人能够在选举人团集体投票中提交他们的个人选票”，并决定谁将成为美国总统。
