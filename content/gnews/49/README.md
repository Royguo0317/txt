**CCP**病毒在全球肆虐一年多以来，出现了世界性的产业链、供应链断裂，大多数企业受到冲击而停工减产，各国的失业率直线上升，国家财税收入严重下滑，在收入下降的情况下，抗疫及社会救助支出却不断攀升，从而造成各国财政赤字连创新高。不可扼制的财政赤字必然推高物价，通胀又破坏生产，可以预料，在疫情未得到控制之前，各国经济将陷入恶性循环，社会矛盾及国际矛盾将双双激化。

在绝大多数中小企业陷入绝境的情况下，脸书、推特这样的社交巨头，沃尔玛、亚马逊这样的商业物流企业，依靠垄断地位或政府批准继续营业的特权，在市场哀嚎一片的情况下，赚得盆满钵满，这是全球性现象，系由**CCP**病毒摧生出来新的不平等及社会问题。如果疫情继续恶化，不仅贫富分化继续扩大，还会产生严重的社会矛盾。

据布雷巴特新闻网**4**月**13**日报道，联合国秘书长安东尼奥·古特雷斯**4**月**12**日呼吁，在全球范围内对富翁征收财富税，为帮助全球从严重的疫情中恢经济提供资金。此前古雷特斯敦促各国对在疫情期间获利的富人征收团结税或财富税，以减少极端不平等；他在2021年经济及社会理事会发展筹资论坛上声称，去年世界上最富有的人的财富增长了5万亿美元，税收将是重新分配这些财富的一种方式；古斯特雷认为，仅10个发达国家就获得了世界上约75%的可用疫苗，这是极大的不公平，因而需要进行财富、权力再分配，来帮助贫困国家人口公平获得疫苗，以及推动全球经济的复苏。

表面上看古特雷斯的建议很合理，但实际上几乎不存在可操作性，因为联合国是由主权国家组成的非主权的国际组织，而征税权是主权国家的法权，联合国无权干预。即使是发达国家，在疫情大爆发的当下财政已经处于崩溃边缘，这个时候古斯特雷打人家富翁的主意，无异于与虎谋皮。另外，积弊过多的联合国，被中共、俄罗斯这两个常任理事国长期操弄和利用，已形同摆设，毫无公信力。古斯特雷对病毒溯源这种能够从根本上解决疫情大流行的关键问题只字不提，完全本末倒置。作为联合国秘书长，他有责任在全世界面前振臂高呼，敦促所有相关国际组织及各国政府追究病毒来源和责任，但他没有这么做。

联合国下属机构世界粮食计划署主任大卫·比斯利则更为直接，近日他提议，要求世界上2000多名亿万富翁（总净值为8万亿美元）提交他们的银行账户，将他们的资金提供给联合国这样的国际组织，准备用于解决即将到来的全球性大饥荒。他在去年**11**月就提出警告**:**为了应对即将到来的大饥荒，我们需要准备数十亿美元的粮食救助资金。比斯利对大饥荒的忧虑值得肯定，但要警惕他提议向富人要钱的方法，如果我们对随意侵犯私有财产的行为不能进行有效制止，人类将陷入更大的灾难。**70**多年来中共在华夏大地制造的空前浩劫，也是在各种正义、公平的幌下肆意侵独私有财产及践踏人权的。
