《又见清明》

词：李文星

清明时节雨纷纷，

不见牧童短笛和牵牛。

尸骨润了草上坟头，

亿万冤魂斥王侯。

马列奔踏，

登基于四九。

炎黄彻底，

蒙上了红绸。

之前！

自从五星，

乱了四季。

山呼啸，

风萧萧，

江河泛滥，

天空成了灰色调。

一人！

半遮琵琶，

拔剑斩红妖。

那一日！

蓝色将士染中土，

雾霾散尽现本色。

这一年！

又见清明雨纷纷，

东京素怒北京荤。

联邦战友执天道，

都是神启菩提根。

《醉雨》

词：李文星

外面下着雨，

思绪在心底。

浊酒泛红晕，

挥泪洒衣襟。

七尺男儿志，

奈何女人紧。

只因时节到，

不得不上坟。

烧纸念先人，

立誓除红魂。

畅快把醉浇，

力拨去龙鳞。

拳打峨眉峰，

脚踢武少林。

试问谁英雄，

战友抗星旗。

上街战病毒，

亡命又何惜。

墙内锁明智，

海外破真身。

待到共妖灭，

深山定神筋。

只管江湖笑，

他人江山新。

《批妖》

作者：李文星

古老的东方，

有一所破墙。

那里的人民，

遭七十年殃。

奴役和捆绑，

毛泽东思想。

世界的邪灵，

中国共产党。

习王是流氓，

粉饰太平妆。

暗地放病毒，

祸害众四方。

号声为中华，

实则乱纲常。

砸碎炎帝庙，

天安门前躺。

作恶灭伦理，

早晚见阎王。

又逢清明节，

万鬼来告状。

挥手一支笔，

扫射红妖党。
