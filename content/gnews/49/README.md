![]()![](https://gnews.org/wp-content/uploads/2021/03/capture-14-18.jpg)图片来源：Getty Image
据《华盛顿观察家报》近期发表的一篇文章，提到了爱尔兰裔美国人的英勇故事与教会及军队这两个令人可敬的制度是密不可分的。教会是爱尔兰人从家乡带来他们所崇拜的信仰，而军队则是爱尔兰子孙们勇敢服务的战斗所在。耶稣会的爱尔兰裔乔瑟夫·奥卡拉汉（Joseph T. O’Callahan）神父正是其中的表率人物。

在四分之三个世纪之前的1945年3月19日，当他所属的富兰克林号军舰在日本凶险的外海遭遇日本的猛烈攻击，美国海军预备队奥卡拉汉牧师（后来成为上尉）坚持抵抗，丝毫不放弃已倾斜的军舰及舰上所有船员的性命。他因为不顾自身安危、冒着生命危险的卓越英勇行为之后获得了荣誉勳章。

第二次世界大战曾经产生了许多伟大的文学作品，然而《我是富兰克林号舰上的牧师》却是被大众忽略的闪耀宝石。书中的主人翁是一位来自圣十字学院而被分配到一艘航空母舰的书呆子数学教授“神父乔”，该航空母舰正为了太平洋战役中最血腥的战斗”入侵冲绳”做准备而驶入了危险航道。书中以富兰克林号的1200名天主教水手和每位海军陆战队员参加的3月17日圣帕特里克节的弥撒开始。就像古代的仪式一样，弥撒是用拉丁语：“我来到上帝的神坛上，上帝为我的青春带来愉悦。……并且会从不义和邪恶的人手中拯救我的灵魂。”牧师告诉他在富兰克林号的信众们，无论他们发生了什么，他们都不是为了要在战斗中得到生存而祈祷，而是祈祷在即将到来的战斗中获得恩典和力量，并准备为上帝和国家提供最好的服务。他提醒他们，他们正行驶在离日本的本国岛屿50英里的危险海域，准备对日本发动战争，为珍珠港被日本偷袭而报仇并击败日本法西斯主义。

牧师将宽恕给予了那些还没有接受悔圣礼的人，因为很快就有太多人需要它。两天之后的3月19日，日本用炸弹炸死了他的924名船友。正如他的荣誉勳章所引用的那样，“奥卡拉汉（O’Callahan）勇敢且镇定地穿越熊熊的烈火及扭曲变形的金属障碍来营救船上的士兵，他匍匐前进，穿过烟雾弥漫的走道，来到了开放的飞行甲板，在炮火猛烈攻击下，炸弹碎片、砲弹残骸如雨点般倾巢而下，大火肆虐，怒火不断，不断爆炸的火箭和各种武器此起彼落。他无畏的照顾伤者及垂死的士兵，安慰并鼓舞有着各种不同信仰的人们；他组织并带领消防人员抵达炽热如地狱般的飞行甲板，他夥同水手将即将爆炸的实弹弹药及炮弹带到甲板边缘上丢弃，阻止了进一步可能的大爆炸。他带领他们进入主要的弹药库，弄湿弹药库以防止大火点燃那里的炸弹和武器，并亲自用消防水管冷却在倾斜甲板上滚动的危险炽热炸弹。尽管周围笼罩着让人窒息的炽热烟火，他仍然不顾一切用往直前。”

多亏既是牧师又是海军军官的奥卡拉汉（O’Callahan）的英勇壮举，富兰克林号舰得以幸存，成功的安全返回布鲁克林海军基地；善良的奥卡拉汉神父也悄悄地回到马萨诸塞州伍斯特市从事大学数学的教学工作。从1945年那个在富兰克林号舰属于爱尔兰裔美国人漫长又艰难的3月17日圣派翠克节日(为了纪念到爱尔兰勇敢传教的爱尔兰圣派翠克Saint Patrick主教)中，我们应该汲取什么教训？

爱尔兰裔美国人总是无怨无悔地全力为他人提供服务，不只是像奥卡拉汉在战斗中为美国和他那艘遭受重创的富兰克林号所做的那样出色，而是在平常的任何时间，他们都是如此谦卑与无私的对人们在实质上或者精神层面的需求表现出怜悯和关怀。

美国或许即将在太平洋的远方被迫与中国共产党进行战斗，虽然美国人并不希望此事发生。然而，如果真的必须发生战争的话，爱尔兰裔美国人也会勇敢迎战。而无论是在战争或和平的世代中，爱尔兰裔的奥卡拉汉（O’Callahan）英勇无惧都是所有美国人必须效法的榜样。
