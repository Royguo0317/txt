发生在35年前的切尔诺贝利事件是前苏联人民心中永远抹不去的伤痕。它给民众带来的无尽痛苦和伤害，淤积成了他们心中的怨恨。5年后，这个看似强大无比的红色帝国，在人民怒火爆发的顷刻间分崩离析。

1986年4月26日凌晨，在乌克兰一个4万多人的小镇，随着切尔诺贝利核电站的一声爆炸，一条30多米高的火柱直冲云天，将核反应堆的外壳爆开，大量的高放射性物质被释放出来，人类的噩梦从此开始。

这次爆炸释放的放射性物质相当于400颗美国投放在日本的原子弹。为了灭火，没有任何防护措施的消防队员被派了上去，陆续赶来的还有军人、直升机飞行员。没有人告诉他们核辐射的严重程度，他们中的许多人用自己的生命避免了反应堆的二次爆炸。

这是人类历史上最大的一次核灾难。然而，官方行动迟缓，在事故34 小时后，才开始进行疏散工作。几天后，在乌克兰的首都基辅，仍蒙在鼓里的民众还在举行五一大游行，而这里距切尔诺贝利还不到100公里，核辐射正在无情地侵蚀着他们的身体。非但如此，电台也在一刻不停地做着“局势很稳定”的虚假的报道，全然不顾此时带有辐射的尘埃已经随着大气在国内，并向欧洲甚至北美四处扩散。

“历史不会简单的重复，但总是惊人的相似”。33年后，“中共版切尔诺贝利”在武汉上演，只是这次核辐射换成了中共病毒。同样，为了掩盖真相，中共采用了“监控、禁声、隐瞒、打压、欺骗、恐吓、甩锅、信息封锁、网络控制”等一系列手段，具体包括虚报疫情数据，抓捕“吹哨人”，隐瞒病毒人传人的事实，有意将病毒在世界扩散和甩锅给其他国家和动物等。

不仅如此，为了免于追责，免于“灭白计划”被曝光，中共还“收买学术界、收买媒体界、收买政客、收买WHO”，试图将谎言推销成真相。而对于病毒真相揭露者，他们则采用”歪曲、丑化、造谣、抹黑”等手法，“打压对手、引导舆论、转移视线”。

为何谎言总是如此相似？因为共产邪灵的本性从未改变。中共不仅与苏联老大哥“一脉相承”，而且“更加丧心病狂，穷凶极恶”。如果说切尔诺贝利核事故暴露出一个充满“私欲、集权、腐败”的病态体制，那么中共病毒疫情则暴露出其制造”生化武器”、意图消灭“白人种族”，并称霸世界的邪恶野心。而且, 中共病毒造成的全球性伤害，对世界经济、金融的重创都是前者所无法比拟的。

然而，两场灾难又有其共性，那就是共产暴政为了苟延残喘，可以无情地“践踏、剥夺”人民的生命。看看那些因不知情而遭受核辐射的人们，和无辜被病毒感染的人们，那些没有良好防护措施就被送往一线的消防员和医护人员，那些被隔离等死的病患……每个人的生命在统治者眼里如同草芥，只是死亡名单里的一个数字，甚至很多逝者连成为数字的资格都没有。

戈尔巴乔夫曾说：“切尔诺贝利核事故可能成为5年之后苏联解体的真正原因，其重要程度甚至要超过我所开启的改革事业。” 同样，中共病毒必将成为中共灭亡的最主要导火索，但绝不会再给它5年时间，正义力量不会，1亿多被感染者不会，250多万死去的冤魂更不允许！切尔诺贝利事件距今已30多年了，它不仅是一次世纪灾难，一场恐怖的噩梦，更是一面镜子，让我们提前看到了邪恶中共的最终结局。
