中共没有丝毫的廉耻感，中共私自制造并释放了CCP病毒一直肆虐全中国和全世界，中共却说他们为CCP病毒患者报销了费用28.4亿，这简直就像路德社所说的：“（中共）砍你一刀，然后把叫救护车救你，所以你要感谢我，因为是我救了你的命。”实际上根据郭先生内部情报，单单武汉被掩盖的CCP病毒死亡人数至少是几十万人。另外实际感染数量比中共自己公布的数据要高十倍不止！2020年武汉全市十来天里领取骨灰盒那段时间，每天各殡仪馆至少出500个骨灰盒，而且武汉在疫情期间全家被灭门的情形比比皆是，这些被灭门的人大部分甚至没有人会去接收他们的骨灰，这些事实证明了郭先生的情报是准确的。

中共通过控制数据来掩盖真实情况，实际上那些没有被计入感染的人根本得不到报销！如果把真实感染数量和死亡数量算上，中共这28.4亿元平均下来也就根本没有多少。当然CCP病毒是中共一手制造，亲自投放，蓄意造成世界巨大的灾难，中共意图用自己制造的疫苗来换取世界各国经济与政治上的臣服。根据郭先生的情报显示，中共甚至想更进一步达到消灭西方信仰，消灭白人的恶毒用心！我们中国人在不知情的情况下就首当其冲成了CCP病毒灾难的冤魂，也就是中共“13579灭白计划”的牺牲品。中共却要我们中国人感谢他们，甚至要世界感谢他们生产出了所谓的“有效疫苗”，天理何在！

CCP毒杀了不计其数的中国人和世界各国的人，这还不够，它还想从中国和世界各国人民身上捞钱。CCP病毒新型疫苗在中共国内并不是免费的。根据中共2020年7月公布的《基本医疗保险用药管理暂行办法》中指出：预防性疫苗不纳入医保支付范围，也就是说你用医保卡报销不了。然而中共却不承认不是免费，反而含糊其辞的说由地方筹备费用解决。实际上各地出台的疫苗接种根本不免费，个人支付费用达到每剂200元左右。

按照CCP病毒的实际变种速度来预测，每年老的疫苗有效性肯定降低，为了保证所谓的“有效”，每年要接种至少两次新疫苗。毫无疑问就算CCP病毒不变种，中共都会想办法让CCP病毒变种。因为CCP病毒是由中共完全控制的，中共想你一年变种几次都没有问题。于是百姓只能不停的接种疫苗来所谓的防疫！那么我们来算算这个经济帐，按照中共的要求“95%的接种率”，中共国内14亿几千万人，至少有14亿人要求接种，那么就是14亿x200元x2/每年，那就是每年至少5600亿人民币的规模。当然这只是一年中共只让CCP病毒变种一次的钱，如果中共每年让CCP病毒变种2次呢，就是万亿！
