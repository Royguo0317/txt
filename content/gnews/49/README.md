日本海上保安厅长官奥岛高弘17日在记者会中宣布，因应对中共的《海警法》，日本也准许在日本领海巡逻的警备队，在符合国际法的前提下，必要时“可以使用武器”。
日本《海上保安厅法》第20条就海上保安人员的武器使用要件规定，该条例适用于《警察官职务执行法》，若有逮捕犯人和防止其逃跑等恰当理由，就可以在合理且必要的限度下使用武器等。
16日上午一度有4艘中国海警船同时停留在尖阁诸岛周遭海域，日方视为“侵入领海”，其中1艘中国海警船疑似配备类似大炮的武器。针对中共海警船在《海警法》施行后，不断进入尖阁诸岛周边的日本领海，奥岛高弘当天强调，迄今未见中方海警船动向有大的变化，但日方将保持警惕，做到万无一失。
