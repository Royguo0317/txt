据美国《每日电讯报》（Daily Caller）记者迈克尔·金斯伯格（Michael Ginsberg）2021年2月28日的撰文，前总统川普（Donald Trump）在上周日保守党政治行动会议（CPAC）的演讲中再次宣称，2020年总统大选是被操纵的，并且他才是真正的胜选人。

文章称，川普在演讲中说道，“虽然有些人在新罕布什尔州举行的党内预选上名列第二，或在爱荷华州排名第一，这对他们来说也许大局已定，并享誉终身，但我们赢得了两次（总统）选举。” 据悉，川普在2016年爱荷华州共和党总统候选人预选中获得第二名（仅次于德克萨斯州参议员泰德·克鲁兹‘Ted Cruz’），并在当年新罕布什尔州的初选中名列第一。

川普在演讲中进一步说道，拜登（Joe Biden）未能解决移民危机，将使民主党在中期选举遭受沉重损失，并接下来在2024失去白宫。他甚至提到他自己有可能决定第三次参加总统大选击败民主党。他的演讲获得了在场参会保守派人士的不绝掌声。

川普随后重申了关于2020年总统大选被操纵的观点，认为选举计票时全国各地都在接受大量涌入且无加甄别的选票。除了有非法移民和死人的投票，还有其它很多可怕的事情发生。川普表示，这次选举舞弊的调查结果揭示出的选举舞弊程度令人难以置信，选举是被操纵的。随后他还抨击联邦最高法院没有胆量和勇气做出正确的决定，对选举舞弊行为毫无作为。

**评论：**

该文章明显带有“左媒”色彩，几乎全盘否定了川普的演讲内容，但又没有列出有实质性的反驳内容。2021年CPAC演讲是川普总统自卸任以来第一次亮相，战斗状态依然如初，并提出了2024年参加第三次总统大选的可能性。川普的底气无疑来自于他在美国人民和保守派团体中的超高人气。笔者相信，川普定能从2020年大选失败教训中走出来，有能力带领共和党，尤其是川普阵营2024年 “王者归来” ，但前方最大的障碍就是如何避免2020年大选中出现的不诚实行为，真正让选举代表民意，这才是川普总统 “让美国再次伟大”（Make American Great Again） 的关键。
