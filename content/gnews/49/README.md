# **美国司法部：中共国军人为获得J-1签证在斯坦福大学从事研究而撒谎**

[!\[\]()!\[\](https://spark.adobe.com/page/AdCcRC5WLSrXP/images/76b0544f-9afb-4553-826e-5706a7fb5e54.png?asset_id=9706ec56-112b-4df2-af24-55ad4a788e71&amp;img_etag=%228a0e760c46f0330a94a9a5d2e43a49cb%22&amp;size=2560)](https://spark.adobe.com/page/AdCcRC5WLSrXP/images/76b0544f-9afb-4553-826e-5706a7fb5e54.png?asset_id=9706ec56-112b-4df2-af24-55ad4a788e71&amp;img_etag=%228a0e760c46f0330a94a9a5d2e43a49cb%22&amp;size=1024)司法部

美国联邦起诉书称，一名中共国军人为了获得赴美签证，向联邦移民官员撒谎，以便能在加州斯坦福大学从事研究。

美国司法部宣布，一个联邦大陪审团对39岁的中共国公民宋琛（音译）发出了一份取代性起诉书，她涉嫌通过谎报自己在中共国军队（即中共国人民解放军）的入伍情况，骗取了赴美J-1签证。

宋琛被控签证欺诈、妨碍官方程序、销毁记录、向政府机构作虚假陈述等罪名，她在前一个月提交申请后，于2018年12月持J-1签证进入美国，在斯坦福大学从事研究。

宋琛在申请书中说，她是一名神经科医生，想来美国斯坦福大学从事脑部疾病的研究。宋琛告诉联邦移民官员，她曾在2000年9月1日至2011年6月30日期间在中共国军队服役，并声称她目前的雇主是中国北京的一家医院，她的最高头衔是“学生”。

根据起诉书，宋琛在J-1签证申请中撒谎。实际上，起诉书指控宋某在申请签证和进入美国时仍是一名中共国军人。

同样，起诉书指控宋琛实际在北京的一家军方医院工作。

[!\[\]()!\[\](https://spark.adobe.com/page/AdCcRC5WLSrXP/images/f9f58151-25de-4ee0-b41e-00260d7e30b9.jpg?asset_id=a84099cf-5dee-414e-bd97-e2939eed37eb&amp;img_etag=%222402f94b0468da2b4399546a6297dfe2%22&amp;size=2560)](https://spark.adobe.com/page/AdCcRC5WLSrXP/images/f9f58151-25de-4ee0-b41e-00260d7e30b9.jpg?asset_id=a84099cf-5dee-414e-bd97-e2939eed37eb&amp;img_etag=%222402f94b0468da2b4399546a6297dfe2%22&amp;size=1024)

美国联邦调查局(FBI)反情报分部的小艾伦·科勒(Alan Kohler Jr)在一份声明中说：“中共国人民解放军军人不能为了来美国学习而在签证申请上撒谎，又指望联邦调查局和我们的合作伙伴不会抓住他们。”

科勒说：“中共国政府一次又一次地将窃取美国的研究成果并利用我们的大学作为优先事项，而不是遵守国际规范。”

2020年6月，起诉书称，当宋琛听说美国司法部起诉另一名中共国军人骗取赴美签证到一所大学从事研究的案件时，她曾试图销毁透露她真实身份的记录。

该案指控中共国公民王欣犯下类似罪行，此前他涉嫌骗取J-1签证进入美国，在加州大学旧金山分校（UCSF）从事研究。联邦检察官称，王欣和宋琛在北京的同一家军方医院工作，能够在美国加州大学旧金山分校的从事一年多的研究工作。

在接受美国联邦调查局问询时，宋琛否认了她在2011年后与中共国军方和北京军方医院的关系。根据联邦检察官的说法，随后，将宋琛与中共国军方联系在一起的公开信息从互联网上被清除干净。

目前，宋琛因签证欺诈、妨碍公务罪、向联邦官员作虚假陈述等罪名，面临最高35年的监禁和75万美元的罚款。对宋琛的审判定于4月12日开始。

J-1签证项目和其他学生签证项目，一直被中共国轻易地用来为其公民在美国大学和公共资助的研究项目中争取名额。中共国仍然是国际学生赴美留学的主要国家。

例如，2018年的最新数据披露，中共国有近48万公民在美国大学就读，这些大学由于收取学费，为这些机构提供的资金超过100亿美元。2019财年的数据显示，中共国仍然是J、M、F这三种签证的最大受益者。
