## 《路德社》精选新闻简讯：纽约最高法院法官裁定《纽约时报》使用 &#8220;欺骗性虚假信息 &#8220;抹黑Project Veritas项目。
` 路德社精选` [轉載自GNews](https://gnews.org/zh-hans/1007833/)

出品：《路德社》 编辑：《路德社》精选新闻专栏组
![]()![](https://www.gnews.org/wp-content/uploads/2021/03/捕获-29.png)
据零对冲的文章报导，上周，纽约最高法院的一名法官对《纽约时报》进行了严厉谴责，裁定他们在几篇诋毁举报者组织Project Veritas的文章中，将他们的两名记者欺骗性地将观点当作事实。

法官说，他审查了《纽约时报》这些报导的语境和语气，得出的结论是，这些文章可以让读者相信这些插入的文章的句子是在传达有关Project Veritas项目的真实事实而非意见描述。

Project Veritas团体在一份声明中说。”这项裁决意味着Project Veritas现在可以让《纽约时报》有关记者和《纽约时报》执行编辑，被迫回答我们的问题。我们将记录这些证词，并将其曝光，让全世界都能看到。”

长期以来，《纽约时报》的所作所为与无数针对零对冲和其他组织的打击性文章类似；使用宽泛的、没有依据的笔触来诽谤他们的意识形态对手。

解释：

Project Veritas：James O’Keefe于2011年成立，作为一家非营利性新闻企业,Project Veritas调查并揭露公共和私营机构的腐败、不诚实、自我交易、浪费、欺诈和其他不当行为，目标是以实现一个更加道德和透明的社会。
