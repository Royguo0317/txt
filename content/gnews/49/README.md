## 《路德社》精选主题专栏：中国共产党是中国人民和美国人民的共同敌人

《路德社》精选主题专栏组出品

![]()![](https://gnews.org/wp-content/uploads/2021/04/042304.png)

2021年4月22日，美国首府华盛顿， [汤姆.科顿](https://zh.wikipedia.org/wiki/%E6%B1%A4%E5%A7%86%C2%B7%E7%A7%91%E9%A1%BF)，[玛莎·布莱克本](https://zh.wikipedia.org/wiki/%E7%8E%9B%E8%8E%8E%C2%B7%E5%B8%83%E8%8E%B1%E5%85%8B%E6%9C%AC)，和[汤米·图伯维尔](https://zh.wikipedia.org/wiki/%E6%B1%A4%E7%B1%B3%C2%B7%E5%9B%BE%E4%BC%AF%E7%BB%B4%E5%B0%94)三位共和党参议员共同发起一份，限制中国研究生申请美国大学理工科学生签证的立法议案 。该议案确立一项旨在保护美国的研究成果不受中国共产党的间谍和政治影响的《[校园安全法](https://www.cotton.senate.gov/imo/media/doc/secure_campus.pdf)》。该法禁止向中国籍公民签发所有理工科的本科和硕士生签证，禁止所有中共千人计划参与者和所有中国籍公民参与有联邦政府资助的任何理工领域项目，法案还包含其它急需的改革措施。

在提议讲话中，科顿议员说到：“让中共随意进入美国研究机构，就像在冷战期间让苏联科学家进入我们的重要实验室一样。 我们不应允许中国共产党继续利用美国研究机构的开放性作为漏洞。 《安全校园法》将有助于阻止中共窃取美国技术，并用于对付美国自己的部队和企业。”布莱克本议员表示，中国共产党一直都在利用美国的教育机构来对付美国。图伯维尔议员表示，中共偷取美国的知识产权，寻找机会扰乱美国经济，威胁美国国家安全。

![]()![](https://gnews.org/wp-content/uploads/2021/04/042305.png)

《安全校园法》并不是中美政治较量的一时之作。早在2019年7月28日，当美国与中共国还保持良好关系时，美国全国公共广播电台就曾报导，[美国FBI因察觉到中共对美国学术机构的渗透，开始敦促大学监视某些来自中共国的学生和访问学者](https://www.npr.org/2019/06/28/728659124/fbi-urges-universities-to-monitor-some-chinese-students-and-scholars-in-the-u-s)。2020年7月24日，[中共国现役军官唐娟，在美国因签证欺诈被捕](https://www.washingtonpost.com/health/chinese-researcher-charged-with-visa-fraud-now-in-custody/2020/07/24/dd78e5ec-cdf4-11ea-99b0-8426e26d203b_story.html?outputType=amp)，唐娟未通报自己是现役人民解放军军官身份的情况下骗取签证，并在美国加州大学工作。同年8月26日，[因涉及共产党的军民融合战略，美国北得克萨斯州大学突然决定中止中共国公派学生学者在该校学习的资格](https://thecollegepost.com/unt-expels-chinese-scholars/)。

《安全校园法》的主旨在于防范中国共产党的威胁，然而普通中国老百姓也被列在了限制名单里，以作者对当前美国的了解，这是议案的无奈之果，因为中国共产党就隐匿在普通中国人中。中国共产党一直以中国人的面目和身份在全世界偷窃技术，干丑恶污秽之事，他们一旦被抓，就用中国人的信誉为他们背负罪行。中国人如果再不与中国共产党划清界限，中国共产党将把中华民族拉入世界黑名单。

如果一个国家是光明正大地崛起，是不会去偷窃别国技术的，就算别人送技术也仅作为本国研发的参考，因为偷来的东西永远是无源之水，不能长久。中共偷窃美国技术，凸显中共治理下的中共国完全没有自我研发能力。以中共国海光集团旗下的X86 CPU 芯片为例，芯片从里到外，就连核心安全与加密模块全都不加修改地抄袭美国AMD公司的知识产权，抄完即结束，因为海光没有升级换代CPU的研发能力。
