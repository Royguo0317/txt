美国国务院该项声明重点在于放宽美国对台来往的限制，单从表象上看似乎并不新颖出奇，也不重磅轰动，但放诸当下的国际局势，再结合声明中提及的几个细节，便会发现大有文章。

眼下正是中共对台湾蠢蠢欲动之际，先有川普政府的经济制裁，后有混乱治理下的疫情危机，巨大的社会矛盾正在墙内累日积增，而党内斗争愈演愈烈，战狼外交也从未停止过将矛盾转移到外部的努力。习总加速师为平众议，眼前似乎只剩，甚至说不定已经选择了武力犯台这一条路。而大洋彼岸的美国不可能看不到这一点，在此刻选择发正式声明以推进非正式关系，足以证明它的立场恰恰已经站在中共的对面。声明所提深化关系看似可大可小，而就近日拜登政府与印太诸盟友日渐热络，且双航母编队仍在南中国海游弋这两点来看，这一次的“深化”似乎将要远胜过此前的任何一次。

而细观声明本身，首先，美方对台湾的定义，无论民主体制、伙伴关系以及国际地位，任何一点都是中共目前所匮乏的。因为分享同一套价值观，西方先进国家可以认同台湾，但永远不会认同中共。其次，声明此时提及“一个中国”则更耐人寻味，尤其是在重申以此前对台湾的承诺为前提的情况下。这“一个中国”指的是哪一个中国，暧昧不清之下似乎又已经对某一方有所倾斜。

# **美国政府与台湾当局互动的新准则**

美国国务院发布了美国政府与台湾当局互动的新准则，以鼓励美国政府与台湾接触，反映我们不断深化的非官方关系。该准则强调台湾有着充满活力的民主体制，是一个重要的安全和经济伙伴，也是国际社会中的一股正义力量。这些新准则放宽了美国与台湾接触的限制，符合我们的非官方关系，并为整个行政部门开宗明义，以有效执行“一个中国”政策，该政策以《台湾关系法》（Taiwan Relations Act）、三个联合公报（three Joint Communiques）和六项保证（Six Assurances）为指导。新的指导方针是根据《台湾保证法》的规定进行审查后发布的。
