## 在这里，中共国在创纪录的时间内为埃及建造新首都

图片：LiveEO / Skywatch： 图片解释，预计今年夏天将有第一批人搬到埃及新首都。

中共国公司正在筹资并建立新埃及首都巨大的商业中心。 随着沙漠中的高楼大厦，中共国正在获得政治影响力-但是北京和开罗几乎总是很近。 “自上而上的经济”是与LiveEO的合作。

几年前，在LiveEO的卫星图像上只能看到几条街道和很多沙子。 但是从那以后，巨大的摩天大楼如雨后春笋般冒出来。

- ![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/08081143/Cairo-000823_USGS_LiveEO.jpg)


图片解释： 2000年这个区域只有几条街道

目前，埃及正在以通常只有中共国建筑工地才能知道的速度建造新首都。 最早的建筑工作于2015年开始，第一批人将于今年夏天搬迁。 高速并非巧合，因为在最重要的埃及建筑项目中，中华人民共和国的业务量很大。

这座新大都市到现在为止仅被称为“新行政首都”，位于当前首都开罗以东45公里处，建成后将成为650万人的新家。 长期以来有超过2000万人居住的开罗大都市将得到释放。开罗正在建设新的（副）城市来解决这个问题，这并不是一个全新的发展。 郊区叫新开罗（New Cairo），建于2000年，预计未来几年人口将从几十万增加到五百万。

新首都目前正在沙漠中建造，面积714平方公里，估计费用在45至580亿美元之间。 除新议会和总统府外，埃及的新首都也将是该国最大的机场的所在地。 还有一个娱乐区和一个城市公园，面积是纽约中央公园的三倍。
