**GNEWS记者：华盛顿DC农场 DC教练**
![]()![](https://gnews.org/wp-content/uploads/2021/04/图片1-37.png)
**摘要：**GTV投资者们在华盛顿SEC门前的抗议活动进入了第七天，很多投资者从外地赶来，抱着强烈的决心要在这里等到SEC认输，得到最后的胜利。

美东时间4月13日，GTV投资者们在华盛顿SEC门前的抗议活动进入了第七天。今天的天气有点阴霾，可能会下点小雨。记者来到现场的时候，看到发生了一些变化：

第一，投资者们买了一排椅子，整整齐齐的放在SEC的门口，打算打持久战，要坚持站在那边、坐在那边，要一直等到SEC认输的时候为止；

第二，特别有意思的是，SEC门口的旗杆都是降半旗的，虽然不知道什么原因，但是这是个非常奇怪的现象；

第三，投资者们买了一批喇叭，抗议的呼喊更加响亮，可以听出来他们的斗志昂扬，甚至在两三条街以外都能听到他们的声音；第四，有几个投资者在附近的街边发传单，向路人宣传SEC迫害我们华人投资GTV的恶行。
![]()![](https://gnews.org/wp-content/uploads/2021/04/图片2-28.png)![]()![](https://gnews.org/wp-content/uploads/2021/04/图片3-17.png)![]()![](https://gnews.org/wp-content/uploads/2021/04/图片4-16.png)
整个行动非常有序，参与抗议的投资者们都斗志昂扬、精神抖擞，有一股很强的气概，势必要把SEC压下去。记者首先采访了一位投资者参与抗议的感受和诉求。

“我们又有新的投资者从外地赶来了，今天已经是第七天了，但是我们大家的韧性和坚持一点也没有减少。长期以来，SEC和华尔街为CCP的国有企业上市大开方便之门，所以从事实上来讲，SEC和华尔街对共产党政权的延续，提供了资金方面的支持。如果没有华尔街，没有SEC给他们开方便之门的话，那共产党肯定是大失血，它根本就没有任何本钱去外面兴风作浪，包括搞一带一路，那都是瞎扯的事。那么SEC对我们这些小的投资人呢？特别是我们GTV，我们GTV肩负着灭共的使命，但是SEC对我们这样的投资却是百般的阻挠，穿小鞋。SEC就是美国证券业的城管，就是欺软怕硬，今天我们就是要来SEC门口跟他们抗争一下，他们这样对待GTV的投资者，我们是不会答应的，我们一定要抗争到底，分出个是非、输赢、高低！”投资者坚决的说到。

“对SEC具体的要求，我想我们的小王子已经讲得非常清楚，有这么几点。SEC是怎么跟中共勾兑的，这个信息应该披露出来；还有就是SEC泄露投资者的信息，甚至造成GTV投资者在国内受迫害，付出了生命的代价，他们要对此付责任的，就像我们诉求的一样，SEC Must Pay！他们必须要付代价；还有一个，我们的GTV投资者在财产上面、经济上面遭受了非常大的损失，SEC也应该赔偿我们个人的财产方面的损失，包括我们投资方面的损失；另外，SEC必须是更弦易张，不能再与共产党沆瀣一气，中共企业在华尔街运行根本就不需要提供跟美国企业同等的会计准则，SEC和华尔街与共产党共谋欺骗美国投资者，大开方便之门，这种错误必须得到纠正。”这位投资者补充道。



记者看到现场的人数跟上次差不多，来抗议的投资者一点没少，在附近问了几人后，发现他们好多都是从外地来的，有的甚至买的是单程票，抱着强烈的决心要在这里等到SEC认输，得到最后的胜利。记者专门采访了几位从外地专程赶来的投资者。

记者：“这位投资者是从弗洛里达来的，那您家里也有很多事情，在华盛顿DC已经是第二个礼拜了，您花了这么多时间，对家庭生活造成的影响，您是怎么处理的？”

弗洛里达投资者：“我觉得人人都有很多事情，但是哪件事情是最重要的我们就先做哪件事，灭共是我们的头等大事，所以我必须来抗议灭共。”

记者：“请问您是从哪里来的？能不能讲一下为什么大老远来到SEC门口进行抗议？您打算在这里待多长时间？您具体有什么诉求？”

达拉斯投资者：“我们是从达拉斯来的，我们投资就是金融灭共，以钱灭共，我们的权利要我们自己来维护。完成星期五的这个活动就先回去修整，如果有时间就再赶来。我的要求就是尽快公正的调查出结果，给投资者一个公平公正的回答，让我们的投资在法律保证下得到保障。”
![]()![](https://gnews.org/wp-content/uploads/2021/04/图片5-15.png)
记者：“这里有位投资者是从洛杉矶来的，让他讲两句他的感受。”

洛杉矶投资者：“这是第二周了，我是坐半夜的飞机，在飞机上睡了6个多小时，然后第二天就直接来现场了。我想在这里坚持到最后，需要到什么时候就到什么时候，只要站出来就感觉越勇敢，越能看到希望，越有信心。我站出来是为了让站在我前面的投资者不孤单，而且这本来就是为了争取我们自己的利益。如果我们只是想着赚钱，但是我们利益受损的时候不站出来，那我们赚了这个钱也心有不甘，所以说我觉得我应该站出来。虽然说有这个病毒，但是我觉得可以克服，我相信胜利会属于我们。”
![]()![](https://gnews.org/wp-content/uploads/2021/04/图片6-14.png)
记者：“这位投资者是从康州来参加抗议的，我们现在请他讲两句。”

康州投资者：“我这次是想来维护我们GTV投资者的个人利益，因为我们受到了拖延，受到了不公平的对待，我们希望得到公平的对待。这是我第一次参加这种活动，我感觉投资者们都非常给力团结，大家伙相互帮助，有问题大家伙解决，我非常开心和投资者们在一起。”

记者：“您对最终能让SEC承认我们的合法权益有没有信心？您觉得在SEC做这样的抗议对将来华人在美国的地位，无论是经济上还是政治上，有什么样的影响和帮助？”

康州投资者：“我是非常有信心的，因为我们是代表维护合法的权益，而且不带有任何其他的色彩，就为了公平，所以说相信我们的正义能够得到维护。我感觉我们华人应该维护自己的权益，应该勇敢的站出来，表达自己的意愿，积极参与到美国各方面的工作活动，这不光代表我们自己，也代表我们所有要维护自己权益的人。如果我们让出了自己的权益，不去维护，一定就被别人利用、被打压、被消失，所以我们一定要勇敢的站出来，站在第一线，维护自己的权益。”
