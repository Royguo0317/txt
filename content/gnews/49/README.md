**译评：**

觉悟养老基金（Aware Super）是澳大利亚工业退休金基金，是澳洲最顶尖的养老基金之一，该基金决定中止其在中共国扩张的战略，体现了目前西方社会考虑退出中共国市场的一种风潮。虽然中共国一再以市场为馅饼诱惑西方资本，然而西方资本对中共政策一直都有担忧，尤其是CCP病毒全球肆孽之后，澳洲政府牵头追溯病毒真相，中共借机对澳洲发起报复性贸易制裁。如此卑劣行径让西方社会更加看清楚了中共的本质，中共对其承诺的投资商机和贸易大单随时可以无缘由取消，甚至连资金撤回都困难，因此与缺乏信用的中共国做生意绝对不是一桩好买卖。最终全球化盛宴将曲终人散，中共也将内外交困、危机重重，并走向灭亡。

**译文：**

觉悟养老基金（Aware Super）首席投资官达米安•格雷厄姆（Damian Graham）透露，这家澳大利亚第二大的养老基金在其一头扎进亚洲国家（市场）几年之后，现在已经放弃了在中共国的扩张战略，因为其政府没有采用西方的商业规范。

格雷厄姆先生表示，虽然在COVID-19危机之后，中共国的经济形势比其它新兴市场恢复得更快，但（管理着）1350亿澳元的觉悟养老基金现在不会按2018年中旬制定的寻求扩大对中共国投资的计划。“三四年前，我认为中共国的思维会变得更西方化一些，”格雷厄姆先生周一在彭博社的一次活动中说，“这可能并不是他们正在进行的道路。（中共）政府已经相当公开地表示了他们的做事方式。”

虽然澳大利亚觉悟养老基金在北京设有办事处，但格雷厄姆先生表示，如果基金去北京开设离岸办事处，那其实在欧洲或美国开设要比在中共国更好。“我们一直在考虑我们（全球）地域分布的安排，”格雷厄姆先生说，“中共国肯定不会是我想要开始的地方。”

作为觉悟养老基金前身的第一州养老基金（First State Super）的首席投资官，格雷厄姆先生在2018年中旬就公布了要通过持有中共国股票以及在中共国非上市私募股权的投资，将该基金在中共国10亿澳元的投入增加一倍以上的计划。一年前，第一州养老基金获得了（中共国）合格外国机构投资者牌照，这是第一个获得该牌照的澳大利亚养老基金。

**对少数（股权）投资者来说很棘手**

第一州养老基金最初通过上海一家基金管理公司将价值约5亿澳元的会员储蓄投资到了中共国的股票中，并通过跨国公司华平投资（Warburg Pincus）和中共国今翊资本（Nexus Point）向中共国私募股权投资了约3.5亿澳元。

格雷厄姆先生说，这些投资已经达到了预期的效果，尽管是以“合理的低效率 ”的方式。然而，他表示，中共国的市场过于复杂，“甚至在如何拿回你的钱方面也是如此”，而本地企业和离岸企业之间棘手的交易活动，也使得成为公司的少数（股权）投资者变得困难。

觉悟养老基金还没有撤出任何资金，但现在没有计划增加在中共国的投资。格雷厄姆先生说：“现在需要暂停一下。我们正在观望等待。”他表示，印度等类似的新兴市场也不利于少数（股权）投资者。

觉悟养老基金公司对中共国“完全改变观点”，是中澳商业关系恶化的最新迹象。中共国已对澳大利亚大麦、部分牛羊肉生产商、葡萄酒、海产品和木材实施贸易制裁。
