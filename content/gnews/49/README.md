《国家利益》2月21日报道，由于复杂的地质条件和投资不足，中共国投入巨资开发的页岩气开釆项目刚刚开始，可能在几年内就以失败收场。

根据美国最近公布的数据，中共国能源信息管理局掌握的已探明页岩气资源超过美国的页岩气储量。中共国能源对外依赖度高达80%，为了加强能源安全，中共一直在高调宣传要大力开发页岩气资源。中共自2016年以来向页岩钻采部门发放了超过20亿美元的补贴，但目前全国仅仅钻了不到2000口页岩气井。而美国单个气田项目就钻了数千口，此外，中共国页岩气总产量仅占美国去年总产量的3%。

中石油和中石化曾经夸下海口，称在未来五年内将页岩气产量提高75%，达到250亿立方米。然而专家们认为，除非取得进一步的技术突破，否则即使到遥远的2035年，总产量最多达到55亿立方米。路透社最近的一项分析称“复杂的地质条件和未能吸引更多投资者”是制约中共国页岩气产业发展的主要障碍，这将对中共国减少天然气进口依赖的努力形成毁灭性挫折”。

据伍德·麦肯齐的天然气研究员张贤辉说，与美国常见的更平坦地区相比，在四川山区使用水平钻孔和液压水力压裂等技术要困难得多。2019年，英国石油公司在发布令人失望的财务业绩后，成为最后一个离开中共国页岩行业的国际石油巨头。而这些石油巨头留下的烂摊子都由中共国国有大公司接管。

中共仗着手中掌握核武、可携带核弹头的超高音速导弹(只有几个大国拥有该技术，目前很难拦截，尤其从太空中发射)技术等威慑性武器，认为美国、北约等都已经难以对其构成有效制衡；另外中共认为中共国幅员广阔，有多层次战略纵深，除了高端芯片之外产业链齐全，而且有14亿人可以绑架，可以闭关锁国，也不惧怕对外战争。因此邪恶的习近平及中共才敢于对内实行残暴高压的专制统治，对外实施战狼外交，不仅操纵美国大选，还丧心病狂地发动超限生物战。

但中共这个恶魔也有死穴，那就是能源严重依赖外部供给。一旦中共的能源运输线被切断，中共国的经济将立即瘫痪。另外，如果中共的外储耗尽，中共的能源、粮食进口都将受到打击，从而动摇中共的统治根基，中共当然深谙其中的利害关系。

几个月以来中共一直在国内作战备动员。对外方面，拜登上台后中共立即改变川普执政时期的雌伏熊样，重新活跃并表现得咄咄逼人，除了在台海等制造事端，还支持缅甸军政府发动政变，非常明显，这是中共对美国及西方明确发出“以战止战”的恫吓，同时为了应对战争一旦爆发而提前做出保护能源运输线的布局。

中共主要从中东和非洲进口能源，运输线全部经过印度洋，每天有约50万桶的原油及相当数量的天然气，在印度洋孟加拉湾的缅甸皎漂港转口，进入横穿缅甸全境的输油管道及输气管道直达昆明，其余则需要经由美国控制的马六甲海峡进入南海→台海→中共国沿海城市。如果中共控制了缅甸，就可以快速修建油气管道复线并控制缅甸的自产油气，尽量绕开由美国镇守的马六甲海峡，这更加突显了缅甸的重要性。缅甸有位居全球储量前10名的石油及天然气资源，以上就是中共意欲控制缅甸的重要原因。

外交是内政的延续，显然，观察美国等西方国家是否有灭共决心，要看美日印及北约是否和中共展开对缅甸的争夺；军事是外交的延续，当外交手段对行将毁灭人类文明的中共无效时，军事手段理应成为必须选项。但估计要等到全世界完成对CCP病毒溯源，对中共发动超限生物战定性的时候，西方及其印太盟友才会真正出手。
