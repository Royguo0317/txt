03月04日，据俄新社报道，普京总统在与全俄《我们在一起》互助运动的参与者会面时说，如果互联网不受“道德”法律的约束，就会从内部摧毁社会。

在互联网上的负面现象中，普京点出了传播儿童色情、诱导自杀、未成年人参与非法抗议等问题。

普京总统提到，“不幸的是，我们遇到抗议，作为未经批准的活动的一部分，所有成年人都知道这一点以及隐藏的东西。我们还遇到网络儿童色情、儿童卖淫、毒品传销，目标受众都是少年儿童，有把他们拉到大街上欺负，和警察打架，然后躲在孩子们的身后，把他们往前推。”

普京在前一天的内政部委员会扩大会议上已经宣布了来自互联网的威胁越来越大。他要求在互联网上更积极地发现那些让未成年人参与非法行动，包括未经批准的行动。普京说：“但不管是谁，也不管以什么借口……试图冷血地利用儿童来实现自己自私的目标，我们永远不应该忘记，这些都是我们的孩子，我们需要以不给他们的生命和健康造成额外威胁的方式开展工作。”

3月2日，调查委员会主席亚历山大·巴斯特金指示对社交网络平台抖音（TikTok）中针对儿童的自杀电话分布情况展开调查。TikTok中出现诱导未成年人自杀的材料，是在莫斯科学校向家长发出警告后才知道的。他们在家长聊天群里发信息，要求特别关注孩子。TikTok在评论当局的说法时强调，该社交网络正在删除违反其规则的内容，包括描述、宣传、正常化或美化自杀。
