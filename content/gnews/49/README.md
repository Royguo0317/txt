1、特斯拉公司公布了这位女士事故发生前1分钟的行车数据，最后发现根本不是特斯拉汽车刹车失灵，而是这位女士在差点撞到人时，特斯拉的自动避让系统及时调整了行车方向和速度，才避免了一场交通事故。

二、FCC颁布新规开始针对中共大外宣

美国之音4月23日报道[\[3\]](https://www.voachinese.com/a/fcc-broadcasters-sponsorship-identification-requirement-20210422/5864018.html)：美国联邦通讯委员会（FCC）星期四（4月22日）通过了一项新的有关赞助身份确认要求的规定，该规定要求广播电台或电视台披露外国政府或其代表在它们的播出频道租用时间的情况。这个新的规定被认为是针对越来越多地试图在美国宣传某种叙述方式的外国政府，尤其是中国和俄罗斯。

1、拜登政府上台后，FCC规定已经细化到对电台/电视台中的记者、主持人和频道进行身份确认，这背后肯定有非常了解中共的专业人士为拜登政府出谋划策，如果电台/电视台出现隐瞒行为，就触犯《外国代理人登记法》，如果上报登记，这些国外代理人就要受到美国政府的监控。

2、美国每个州有不同的法律，中共可以在不同州的不同地区和不同的网络公司签合同，中共可采取租用频道、节目或者广告的方式与这些美国电台公司合作，这大大增加了FCC的调查难度，这项行政明令要求所有公司主动上报外国代理人信息，这就减少了FCC调查的负担。

4、在美国，除了电台和电视台，像Youtube这样的公司也归FCC管辖，其次，在美国要获得电台或电视台牌照非常艰难，持有者需要接受非常详细的调查才能获得执照，所以中共只能通过租用的方式使用电视台，该命令颁布后，电台执照持有者绝不可能冒吊销执照的风险帮中共隐瞒真实情况。

6、从电视台执照就可以看出美国管理体系非常先进，刚开始降低门槛，允许资金参与投资，解决就业，几年后再收紧管理，加强监督。美国的电影行业、电视电台、赌场等行业都需要执照，如果你的资金不在美国沉淀两代到三代，压根就不可能进入这些行业。西方国家虽然曾今通过殖民的方式获得了大量财富，但是他们获得财富后就开始着手建立体系，将财富藏富于民，让全民共同保护建立起来的体系，这是西方政府非常高明的管理方式。
