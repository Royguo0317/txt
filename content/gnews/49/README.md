# **香港：英国指责中共国违反联合声明**

**外交大臣多米尼克·拉布(Dominic Raab)声讨北京“持续不遵守”1984年协议的罪责**

[!\[\]()!\[\](https://spark.adobe.com/page/AZWN9McX4m7OA/images/3904d1cf-ae2c-4984-bb4b-b6966ca41088.png?asset_id=1859251e-2f70-457c-b54f-a9304b5e161e&amp;img_etag=%22deb79ba4d3e6418ebaf6c5577841b66b%22&amp;size=2560)](https://spark.adobe.com/page/AZWN9McX4m7OA/images/3904d1cf-ae2c-4984-bb4b-b6966ca41088.png?asset_id=1859251e-2f70-457c-b54f-a9304b5e161e&amp;img_etag=%22deb79ba4d3e6418ebaf6c5577841b66b%22&amp;size=1024)多米尼克·拉布的介入标志着伦敦和北京之间的外交紧张局势升级。照片：Andy Rain/欧新社(EPA)

伴随着对北京加强对香港控制企图的批评声浪，多米尼克·拉布指责中共国违反了有关香港治理的法律协议。

在外交紧张局势的重大升级中，这位外交大臣说，英国认为中共国处于“持续不遵守”《中英联合声明》的状态。他谴责北京决定减少公众在挑选香港领导人方面的作用。中共国已将权力交给一个亲北京的委员会，该委员会将任命更多的议会成员。

拉布说，此举是在不到九个月的时间内中共国第三次违反这份具有法律约束力的联合声明，这是“旨在骚扰和扼杀所有批评中共国政策声音的策略”的一部分。

他说：“北京决定进行激进改变，对香港选举制度的参与进行限制，这明显进一步违反了具有法律约束力的《中英联合声明》。”

“中共国当局的持续行动意味着现在我必须报告，英国认为北京正处于不遵守联合声明的状态，这表明北京的承诺与其行动之间的差距日益扩大。”

“英国将继续支持香港人民。中共国必须按照其法律义务行事，尊重香港的基本权利和自由。”

中共国在香港实施的新的国家安全法，使其更容易钳制异议人士，此举也使紧张局势不断加剧。

《中英联合声明》由两国于1984年签署。伦敦一直坚持要求保护香港重要的自治和自由。

拉布得到了G7国家集团外长们的支持。这些国家对香港选举制度中民主元素受到明显侵蚀表示担忧。

G7外长们声明说：“我们也呼吁中共国和香港当局恢复G7国家对香港政治机构的信心，结束对那些促进民主价值、捍卫权利和自由的人的无端压迫。”

欧盟周五也表达了关切。欧盟高级代表兼副主席约瑟夫·博雷尔(Josep Borrell)表示，中共国“蓄意瓦解‘一国两制’原则，违反了其国际承诺和香港基本法”。澳大利亚和新西兰也分别提出了关切。

香港政府表示，强烈反对质疑中共国是否愿意维护“一国两制”的言论，称这是“毫无根据的指责”。
