即将到来的下周四中美最高外交官首次会面备受关注。在近日举行的国会众议院外交事务委员会听证会上，众多国会议员对此次中美会面、新疆和香港人权问题等，集体向国务卿布林肯（Antony Blinken）施加压力。

布林肯国务卿和国家安全顾问杰克·沙利文（Jake Sullivan）将于3月18日，在阿拉斯加会晤中共中央外事委员会办公室主任杨洁篪和中共外交部长王毅。中美双方对这场首次会面提出了不同期望。布林肯国务卿定调为非战略对话的会谈，被中方誉为“应美方邀请”的一场“高级别战略对话”。拜登政府面临病毒真相、经济等方面越来越沉重的压力，与中共重启关系的行动显得迟缓，而中共明显已经按捺不住。

听证会上，布林肯国务卿被包括共和党和部分民主党人在内的国会议员们连连追问：中美会面会否因气候问题对中共让步？会否提出中共在新疆严重侵犯人权的问题？会否因中共侵犯人权、让他们付出“严重的代价”？布林肯称，中共在军事、外交、经济上对国际规则构成了挑战，美国不会对其轻易退让，并将重建以人权和民主为核心的外交政策，运用诸多如《全球马格尼茨基问责法》的机制，制裁那些直接责任人。期望借此次会面，能促使中方对美国提出的反对意见予以解决。

听证会上还出现一大亮点，布林肯国务卿承诺，将邀请台湾参加由美国筹办的民主峰会。他称赞台湾在全球疫情中，表现出一个有担当的民主国家形象。民主峰会是美国总统拜登在去年竞选时期提出的倡议，旨在汇集全世界民主国家对抗民主倒退的国家。据《纽约时报》近期一篇报道称，首场峰会预计在年底举行。
