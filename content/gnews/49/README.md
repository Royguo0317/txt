当笔者第一次看到这个新闻的时候，再次刷新了我对中共科技实力的理解。自特斯拉2013年进入中共以来，至今已迈入第八个年头，距离2017年签订独资国产化协议也有了四个年头。而中共在这个领先全世界的社会主义制度下，终于在2021年发现了特斯拉车上装配的摄像头等设备有安全隐患。我们先不谈为什么这些军官家属不买大国崛起的红旗、北汽、一汽等国产汽车，也不谈家属院里面究竟能有什么军事秘密。笔者只想和大家分享为何中共要在这个时候再三对特斯拉进行特别关照。

顺带一提，如果中共国软件经营研究特斯拉需要耗费这样长久的时间，所谓的国产系统，国产芯片的真实度又有几何？

2021年，中共开始行动，是真实关系百姓国家安危？还是别有用心？

2021年2月9日，特斯拉因所谓质量安全问题被中共五部门问话，例如电池起火，异常加速等问题。这让笔者想起自2019年以来持续真实发生多起安全事故的蔚来汽车，与其低销售量相比大量事故被报道的同时，却没能找到任何中共部门约谈的情况。甚至在已经有实际证据证明蔚来汽车用户隐私数据存在泄露的时候，中共依然没有任何行动来确保军区大院的安全。是否可以侧面的理解为蔚来汽车背后的实际控制人就是中共呢？我相信各位读者只需查询蔚来汽车股东即可一目了然。

中共为了控制国内市场，设置极其高昂的进口关税和车牌政策限制普通人的选择。中共政府仅允许国外汽车制造商以合资身份在国内建厂，方便偷窃各大车企制造技术，还通过大肆宣传还未成熟的新能源汽车扶植傀儡车企，骗取来自于老百姓纳税钱的新能源补贴。

在这样畸形的车市当中，为什么中共愿意允许以进口车身份缴纳大量进口车关税且可以享受新能源补贴的特斯拉以独资身份进入中共市场？难道是中共终于想开了决定痛改前非，开放市场？

特斯拉为什么这么特别？

没有传统车企长期的制造经验，特斯拉制造的汽车一向都不以机械质量闻名世界。移动的大ipad，手机智能控制，无人AI 驾驶这类更加数字化科技化的词汇显然更符合这家来自硅谷的电动车企。

而笔者认为特斯拉最特别的是其Autopilot软件技术。特斯拉使用摄像头作为主要硬件辅以其软件成就Autopilot技术。特斯拉在技术介绍中也将宣传页面留给了八个覆盖全车四周的摄像头。这些摄像头的识别距离最远可到250米，识别角度达120度，而这些参数还是受限与车载平台的低下的性能所限。

说了这么多，究竟中共想要什么？依笔者拙见，中共自2017年筹划和特斯拉签订国产协议开始，便早早布局，择机偷窃特斯拉无人驾驶及摄像头识别技术，用于现有的天网系统，更全面的监控中国人。

而在2021年的一步步行动，也只是刚刚开始。笔者猜测，在接下来的时间里，中共会逐步施压，直到以国安理由检查并获得特斯拉无人驾驶系统源代码为止。非常巧合的是，在特斯拉被中共问话的10天后消息人士透露同为蔚来汽车股东的小米也确定要造车。

如果中共监控真的用上了特斯拉技术？中共国会变成什么样？

根据新华网的新闻分析，到2025年中共国的互联网汽车保有量将达到2800万部。这些国产汽车的配置虽不及特斯拉，但其配备的前后摄像头、车内麦克风、5G网络和运算性能，不仅可以在车上运行抖音大屏洗脑老百姓，用于全方位实时监控老百姓也是刚刚好。
