#### 原文翻译如下：

拜登政府任命的商务部长吉娜·雷蒙多（Gina Raimondo）赞扬前总统唐纳德·川普，因其在与中国的贸易战中使用关税。

雷蒙多在宣誓就职之后，周四美国电视（MSNBC）的史蒂芬·鲁尔（Stephanie Ruhle）对她进行了采访。在采访中，她被问到将如何谈论中共国。雷蒙多对此回应了中共国的行为，并概述了拜登政府与他们接触的方式。

在将中国电信公司标记为“对美国经济和国家安全的威胁”之后，雷蒙多将“关税和反补贴税”列为美国与中共国打交道的部分选择。鲁尔指出，川普在担任总统期间对从中共国进口的钢铁和铝征收关税，因此她询问拜登政府是否打算将这些政策保持不变。

雷蒙多说：“数据表明这些关税已经生效。” “而且我认为拜登总统所说的是，我们将对所有这些政策进行政府整体审查，并决定维持那些有意义的政策。”

鲁尔进一步询问了雷蒙多是否打算比特朗普更远地瞄准像华为这样的中共国企业。雷蒙多回答 “它们应该成为目标” “如果有必要，那就是我们要做的。”
