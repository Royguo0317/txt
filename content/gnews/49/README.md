作者：喜马拉雅东京樱花团 / 捆绑CCP一千年

前言：对于中共国外交政策的变化，堪称70年完美外交骗局。对于习近平新时代的“战狼外交”更是有目共睹，可谓丢尽了国人的颜面。不过，值得庆幸的是郭文贵先生领导的爆料革命挽回了中国人的面子，在美国特朗普政府时期成功地使美国做到了将“中共”与“中国人”的分别对待。然而，这也不得不归功于习近平的外交做法，即习失去人民的支持，却重回毛泽东时代的“零和”外交局面。笔者仅以此文就中共与美国的外交演变做一个缩影式的阐述，希望能够帮助读者认清中共对美外交严重欺骗和不对等的事实。因为中共对外交往的本质就是“小人欺负君子，小人一直赢”的过程。

如何看待正在进行的中共年度两会现状？备受讥讽的就是中共的外交官王毅。他在回答被安排的香港记者提问时说：“香港在殖民统治时期，没有任何民主可言。回归24年来，没有谁比中央政府更关心香港民主的发展，更希望香港保持繁荣稳定。”这种将杀戮作为邀功的外交答辩实在超出人类底线思维，但这底线思维一定不是习近平的底线。对此，美国时间3月7日，路德时评（路博艾冠谈）解读了王毅这一反人类的外交用语。博博士说：“所谓人民当家作主，就是毛泽东1949年在天安门城楼上振臂一呼，中国就有民主了。共产党夺了权，就代表了人民夺了权，那就是人民当家作主了。以此类推，共产党夺取了香港政权，代表香港人，就意味着香港人有民主了。”

**毛泽东的外交欺骗**

毛泽东外交思想的三个方面：积极谋求主权独立、坚持新中国外交自主、强调以国家安全为主体。其中所谓**积极谋求主权独立**：即“鸦片战争以后，西方列强对中国的侵略日益加重，救亡图存成为中华民族面临的首要任务……新中国必须采取独立自主的外交政策”（见《中共党史（毛）》）可以说，中共自毛泽东以来，就已经在潜意识里对中国人实施了完美犯罪计划。而鸦片战争、主权独立等反帝口号就是其中完美谎言之一。在中国人完全无法获取外界信息的情形下，中国人对于毛泽东玩弄的“主权之争”这些概念一概不知，因此，用所谓“鸦片战争”和“主权独立”这样极具欺骗的谎言完美地笼络了人心。中共国50年代的人仍旧对“西方列强”耿耿于怀，却丝毫不去思考为什么中共高官的子女都纷纷移民到西方列强国家中去了？

中共一手炮制了“鸦片战争时期西方传教士充当间谍”的说法深入中国知识分子心里。笔者曾经和许多知识分子进行过辩论，他们这些所谓大学生的思想认知有一个共性，西方传教士对华援助不安好心，把正常的国际人道主义精神国别化、政治化。

有目共睹的是，西方传教士为中国所做的贡献数不胜数，这一事实完全被视为有目的的侵略。如教会医院、教会大学等，目前中共国最好的大学和三甲级医院都是传教士援助的遗产。比如中国医科大学原为盛京医科大学。盛京医科大学是1882年英国苏格兰教会创办的。比如北京大学，即燕京大学也是教会大学，由出生于中国杭州的美国传教士司徒雷登创办的。但毛泽东的《别了，司徒雷登》的文章就是对西方民主价值观完全的践踏。

备受讽刺的是，鸦片战争这种完美谎言就是在被揭露的前提下，中共仍然玩的风生水起，问题在哪里？在于鸦片战争谎言本身就是鸦片，中共还要中国人吃这种精神鸦片要到几时？

疯狂捞取鸦片战争谎言红利的毛泽东，又以“红色基因”的方式借尸还魂地附着在现代毛主义的习近平独裁者体内，可谓中国人的悲哀。在毛泽东眼里，毫无人权与民主法治的影儿。

如何理解所谓**坚持新中国外交自主**？（见《中共党史（毛）》）实际上是被迫闭关锁国的思维。而“外交自主”这个名词就是一个杜撰的前后矛盾的谎言，而中国人就活在谎言里已经好几代人了，至死也不明白什么叫外交自主。有点思想的人，都不会相信一个国家的外交可以自己说了算，包括中共国。首先，这是对“外交”的歪曲和侮辱，因为外交就是对外交往，尤其是一个国家的对外交往，他必须有谈判，谈判本身就有让步的意思。而“自主”就是字面意思，你自己说了算。既然毛的外交自主，司徒雷登怎么没有向毛泽东屈服？笔者举个不太恰当的例子，如果你非要坚持“外交自主”说法是正确的，就等于你去违背一个女性的性意识，强行与之发生性关系，因为你想和她发生性关系，你说了算。这不就成了性交自主演变成外交自主吗？中共搞“双休”大概就这样一个逻辑发展而来的。

毛泽东究竟是怎样一个外交自主？中共党史认为，“抗日战争是毛泽东自主外交思想的实践前提。”这可以理解为毛在抗日战争时期意识到自己不可能争取到西方世界的认可，就萌生了建立政权后只能搞没有外交的外交游戏。因为“毛泽东在思考中华民族与外国列强之间的关系时，形成强烈的民族自主崛起意志。”（见《中共党史（毛）》）笔者认为，毛在西方强国军力对比下思考自己的出路时意识到只能兄弟无靠，自己干自己的。所谓“自主崛起”就是这帮文笔吏欺骗人的说辞，倒不如说没有一个女人能看上你，你只能自淫高潮而已。这就是毛的自主崛起的外交思想。

“冷战”的爆发后，毛泽东的“一个中间地带”和“两个中间地带”思想实际是毛泽东看到苏联的结局后，想要玩中共与西方民主圈分庭抗礼的局面，即默认美国为首的西方国家领导西方，毛可以玩弄亚非拉外交。“朝鲜战争进一步坚定了毛泽东独立自主外交的战略思想和政治意志。”这好比是在说，毛泽东以牺牲90万大军的生命为代价，硬逼着西方绅士坐下来和谈，也可以说，毛的流氓态势逼出了美国对中共绥靖政策的萌芽。因为中共党史认为，“毛的抗美援朝已经获得了正义性的支持，这坚定了他关于新中国必须要独立自主地参与战争的决心和意志”。

毛强奸西方民意后，自己认为你不能拿我怎么样？你不敢越过三八线、不敢跨过鸭绿江。典型的把绅士当做软弱可欺的对象。中共党史认为，毛对苏联能否坚持共产主义表示怀疑，看准自己夺取国际共产主义阵营老大的机会，进一步坚定了毛玩弄西方民主的决心。把“朝鲜战争定位为帝国主义霸权势力”，认为此时在朝鲜战场“参战是走向独立外交的政治保证”。即看准夺取国际话语权的机会，不惜牺牲一切士兵的鲜血为代价，也要改变零外交为有外交的局面。在毛的“枪杆子出政权”的思维下逼着西方世界与之交往。

**强调以国家安全为主体**。（见《中共党史（毛）》）这只是又一个美丽谎言的伪命题。因为“国家安全”可以被非常模糊广泛地定义。但这究竟是什么概念？当今社会，民主化程度越来越高，国与国之间对于文化、科技、军事和农业等领域合作、分享程度越来越高。甚至有些欧洲国家都没有国防力量，你要如何界定国家安全？因为，国与国之间各行业的合作交流难免受到控制和反控制，如何界定一个国家受到控制如今也越来越难以界定。比如，中共超限生物基因战偷袭了世界各国，至今完美地躲避了法律的追责，实现了完美犯罪。因此，国家安全这个概念仍然是模棱两可的，它是从较为简单的定义演变而来的，尽管这些定义强调不受军事威胁和政治胁迫的自由，但中共还是政治胁迫了美国民主选举。

中共党史将毛泽东的外交观点定义为**“民族主义外交**”是对的，（见《中共党史（毛）》）而毛的“保卫领土主权完整”和“争取世界的持久和平”严格意义上讲不是外交思想，而毛的对外扩张战略以及因此形成的扩张主义才是毛的世界观。我们通过郭文贵先生透露的“13579”计划和“2035”、“2049”等计划足以证明中共统治全球计划自毛泽东时代就已经决定了，而中共历代领导人都是秉持独霸天下的战略思维的“零和”游戏，但盛唐万国来朝的盛况一直是中共梦寐以求的政治野心。

**邓小平的外交伪装**

进入邓小平时代的中共国又是怎样的一个改革开放的外交政策改变呢？黑猫白猫论，“让一部分人先富裕起来”，“社会主义不是贫穷”，1988年，邓小平指出：“社会主义的特点不是穷，而是富，但这种富是人民共同富裕。”。（见《人民网》）这些观点鼓舞了人们向往美好生活的斗志，很快，90年代社会上出现万元户、暴发户和今天的土豪，中共国在邓小平的带领下，时至今日，人们没有和西方比技术、比文化，比的是钱财。中共高官可以买大都会博物馆名画，农村娃只能破屋面壁。

在土豪价值观兴盛的今天，人们冷眼看贫穷，却高调谈发展。国内所谓良知政治家均认为这是中国最好的时代。邓小平的外交战略首先放弃了战争思维，使老百姓吃饱肚子，摆脱了一穷二白的局面，开启了中美外交和与世界各国和平相处。或者说，邓小平的一些外交上的重大改变是民主改革的信号？70年代成长起来的中国人都知道这段历史，不可否认，中国人是要吃饭，但不能因实现这一作为人最基本的生存权为由，要求人民跪地感激他们。相反，中国人，尤其是知识分子，应该思考更深层次的问题。为什么中共政党承诺的人民民主后面要加一个“专政”两字？八九64究竟是怎么回事？（笔者这里不探讨这个人尽皆知的问题）毋庸置疑，这个血债自然算在邓小平身上。

对于邓小平的改革开放，与其说是邓的外交改变，不如说是中共的斗争思维的改变。据中国领导干部资料库记载，“邓小平肯定地说，‘和平力量发展起来了，战争是可以避免的’”。（见《中共新闻》）的确，自越南自卫反击战后，邓小平结束了战争，却把枪杆子对准“自己的父母”——邓小平自称是中国人民的儿子，从此之后，中共镇压人民的国家犯罪从未停止过。64大屠杀以后，中国社会“集体失语”，从此，一个新兴的极权纳粹化政党就在邓小平韬光养晦的完美欺骗下发展强大起来。

正如中共党史所称那样，“在世界社会主义运动处于低潮的时候，邓小平旗帜鲜明地指出：社会主义前途依然一片光明。”（见《中共新闻》）这就是邓的完美欺骗。《圣经》说：“这也不足为怪，因为连撒但也装作光明的天使。”（《哥林多后书》11:14）不过，的确如邓小平所言，这个时期的社会主义运动自苏联解体后确实处于低潮。这给邓小平看到了机会，即在共产主义国际阵营中如果能胜出，历史机遇时不再来，在邓小平眼里，以‘一国两制’成功外交行动和“发展中国共产党同他国政党党际关系的指导原则都具有深远历史意义。”（见《中共新闻》）这个“深远历史意义”可谓回味无穷，令笔者想到了中共“蓝金黄”外交创举。

笔者从这之后的40年格局看，中共的确成功游说了世界主要大国。悲哀的是，中共执政者都是一群流氓，如郭文贵先生所言，“大英帝国完蛋了，撒切尔夫人在和我们中国人民的好儿子邓小平同志对谈中被小平同志一口痰痰倒了。”（见郭文贵链接）结合当下中共发动的超限生物基因战，不但英国跪下了，西方大国都被打趴下了。

**“江湖”时代的外交买办**

或许你说的有理，江湖时代（江泽民、胡锦涛）虽然腐败，人民生活质量还是大大改善了。中共国人总是善于自欺欺人，笔者知道，（包括我在内）这不是中国人的问题，试想，任何一个国家人民被极权政府捂住眼睛和嘴巴，对你专政长达70年，你还有能力反抗吗？

那么，江湖时代的外交政策有哪些变化？或者说，胡锦涛时代真的更好？笔者将以对比的方式进行一些重要观点的解读。

中共国以非市场经济主体加入西方自由市场主导的WTO世界贸易组织可谓江泽民外交的重大成果，这在1998年前，几乎是不可能的事情。这虽然进一步改善了人们的生活水准，中国人就因此有理由感恩中共政党？这却是中共乐于见到的事实，因为这进一步巩固了中共政权对人民的压迫。

南方新闻网是这样评价江泽民的外交努力，“加入世界贸易组织显然是中国外交的一项光辉成就”。（见《南方新闻》）笔者认为光辉成就的背后是蓝金黄外交、偷盗西方技术外交（学术交流）、3F美国的外交策略全面开启。的确，对于国际贸易来说，“世贸组织没有中国的参加是不完整的”，但江泽民只说了半句真话，而“中国要作为发展中国家加入世贸组织”这句话却是半真半假，事实是这样的，WTO组织明确规定，只有自由市场经济体才能加入。江泽民应该说，中国要作为专制的非市场经济体发展中国家加入世贸组织。这显然是不可以被授权的。当然，中共代表团为此吃尽了苦头，仍不被允许，南方新闻网这样记载的，“1999年11月，在中美世贸谈判遭遇挫折的关键时刻，是江泽民和克林顿通的电话最终促成了双边协议。对此，克林顿总统委托美国贸易代表巴尔舍夫斯基当面表示‘正是由于江泽民主席从战略的高度来看待和处理两国关于中国加入世贸组织的谈判问题，才使得双方终于达成了协议。’”（见《南方新闻》）

有关江泽民与克林顿的神秘电话内容不得而知，除了向克林顿开出天价贿赂的承诺外还有一个公开的秘密，即“江泽民与时任美国总统比尔·克林顿（Bill Clinton）就中美关系和一些世界与地区问题达成共识，双方发表《中美联合声明》，确立了中美关系未来发展目标。”（见《BBC中文网》）熟悉中共外交骗局的人都知道，杨洁篪一直是中共外交灵魂人物，没有他，蓝金黄美国也是非常困难的。在江泽民时代，外交部经常会出现这样的官话，“中国外长杨洁篪宣布，中方希望美方恪守中美三个联合公报和中美联合声明。”说明美国对于中共渔利心里一直很不爽，但半推半就的对华绥靖政策就这样被延续到奥巴马政府。

奉行和谐社会观的胡锦涛外表看上去是儒雅的，过于拘谨的。胡锦涛在一次访美与小布什会谈的时候，被小布什总统抓住胳膊肘子的一幕传递了“胡布外交”不和谐之声。BBC中文这样报道的，“在华盛顿的欢迎仪式上致辞时，胡锦涛承诺在汇率问题上推进改革，并加强与美国在贸易和投资方面的合作。”“在胡锦涛到访期间，西藏和法轮功的抗议人士也在华盛顿等地进行示威；而据美联社报道，在白宫的欢迎仪式上，美国方面还将胡锦涛误称为中华民国领导人。”显然，人民币汇率操纵国一直在胡锦涛时期备受美国指责。而一个重要人权问题，就是法论功问题却在胡锦涛访美时期备受干扰，但中共对法论功信仰者的镇压和活摘器官反人类罪是江泽民做的。不幸的是，胡锦涛还被美国外交官口误称为中华民国领导人。

不过，美国奥巴马时期的胡奥外交有着很大的突破，2011年，拘谨的胡锦涛与时任国务卿希拉里·克林顿（Hillary Clinton）举杯笑饮，当时，《卫报》、《华尔街日报》等西方媒体发表文章称，“成为世界第二大经济体的中国在很多方面已经堪与美国平起平坐，而美国也不需要惧怕中国的崛起。”（见《BBC中文网》）显然，这里有个问题，胡锦涛时代的中国在经济体量上位居美国之下，堪称平起平坐，竟然被媒体忽悠说美国不惧中国崛起。

当然，这个时候西方国家对中共和中国是不分的。尽管奥巴马指责了中国人权问题，而胡锦涛回应人权问题时含糊其辞地说，“中国在人权问题上‘有很多事情要做’。”笔者宁愿相信胡锦涛在试图改善人权问题上有心而无力，因为胡锦涛的外交努力的背后都是江泽民幕后掌控所致。

**结论：中共外交的自我毁灭**

在蔑视人权问题上，中共国究竟还有哪些事情要做？习近平的所作所为回答了美国和西方政府关切的问题，即加大对新疆人和香港人的杀戮的同时，加大毒疫苗外交，在这个盈利和进攻策略上“双赢”的新型外交领域，绝对是高举阻击病毒大流行这个高大上的旗帜，加速向欧美亚非世界发起中共政权危机最后的攻势，可以说，中共外交在走向死亡的路上不可能再有回头路。

一个威权主义狂热症的习近平政权从来不会像邓小平、江泽民、胡锦涛他们那样有妥协的智慧。而王毅作为中共现任政权外交官，在回答美国广播电台记者有关新疆、西藏、香港和台湾问题是如何处理的回应中强调“不干涉别国内政，是《联合国宪章》的明确规定，……你刚才提到的很多问题，都是中国的内部事务。中国做得好不好，中国人民最有发言权；中国应该怎么做，中国人民才是主人翁。”（见《新华社》链接）这种陈词滥调现在看来更加显得苍白无力，最不可理解的是，美国政府都已经区分中共和中国人的关系，而王毅继续要绑架人民，说中共做得好不好，中国人民有发言权。你打开防火墙了吗？你放弃言论审查了吗？你们要放开言论自由，听听被你奴役的中国人的声音吗？中国人的口水都会淹死你们这帮刽子手。最后，王毅不忘记抓住美国拜登政府软肋说：“中美可以合作、需要合作的清单就放在我们面前，包括抗击疫情、经济复苏、气候变化等等，”因为所谓抗疫合作的疫苗经济仍然有福奇博士替中共把持着话语权，而气候变化又是中共牺牲中国人的利益换来的国际话语权，笔者就此问题，在《巴黎气候协定》一文中讲过：一则《巴黎气候协定》的五个神话一文中提到“中国正在采取比美国更有力的行动”并预感到“这一事态意味着中（共）国有望成为全球领导者。”该文作者援引美国詹姆斯因霍夫的话说，“《巴黎气候协定》只不过是空洞的承诺。而这就是中共完美犯罪的外交思维，空洞无物中的有物正是中国哲学的核心，这个核心文化里隐藏着最大的阴谋就是以共产主义威权模式统治世界，并成为“全球领导者”。

试想，一旦西方科技和美元为他所用，西方的航母舰船和太空军还有能力反击吗？因为一旦成为全球领导者后，那些都将是中共说了算的领域。值得庆幸的是，西方世界正在苏醒，我们不会错失拯救人类文明的最后机会。首先向中共说不的还是中国人——新中国联邦。
