4. 美国国务卿布林肯周五表示，美国支持加拿大坚持要求立即无条件释放两名在中共国被拘留的加拿大公民。

5. 斯里兰卡外交部长迪内什·古纳瓦德纳2月20日通过媒体透露，在该拖欠中共国开发汉班托塔港而提供的一带一路贷款后，港口的近200年实际控制权已于2017年让渡给中共。观察人士认为，斯里兰卡将一个印度洋战略港口交给中共国，是对这个岛国主权的严重威胁。在2月24日的例行记者会上，中共外交部否认此事。
