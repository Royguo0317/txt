# 中共国没收了采自新冠病毒（COVID-19）起源地的研究样本

*2020年12月30日；娜塔莉•温特斯*

[!\[\]()!\[\](https://spark.adobe.com/page/E2tRJdyjHjPHO/images/c7de0f3d-b504-486b-a11b-d37744cb03cc.jpg?asset_id=820b560f-b820-4d21-b0b0-90a278645b23&amp;img_etag=%220266a4d90a41118ee42b3026e9e06a60%22&amp;size=2560)](https://spark.adobe.com/page/E2tRJdyjHjPHO/images/c7de0f3d-b504-486b-a11b-d37744cb03cc.jpg?asset_id=820b560f-b820-4d21-b0b0-90a278645b23&amp;img_etag=%220266a4d90a41118ee42b3026e9e06a60%22&amp;size=1024)

在中共国南部郁郁葱葱的山谷深处有一个矿井的入口，曾经在矿井里栖息的蝙蝠携带有目前已知最接近COVID-19的病毒。

这是一个具有极大科学价值的地方，因为在这里可能找到冠状病毒起源的线索，这种冠状病毒已在全球范围内杀死了170万人。然而，由于政治敏感性和保密性，对于科学家和新闻工作者来说，这里已经成为一个没有信息的黑洞。

两名知情人士说，一个蝙蝠研究小组最近到过这里并设法采集了样本，但样本已被没收。冠状病毒专家已被严令不得对媒体说出此事。而在11月下旬，美联社的一组记者被便衣警察开车尾随而来，便衣警察封锁了进山的道路和站点。
