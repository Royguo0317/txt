近期，印度的新一波病毒感染浪潮对该国造成了毁灭性的打击。新德里各地医院已经不堪重负，很多地方两人一起使用一张病床，医疗资源极度匮乏，氧气危机，很多患者被拒之门外，社交媒体上充斥着心急如焚的家庭成员，他们的亲人无法获得他们所需的医疗保健因为得不到及时治疗在很短的时间内病逝。连殡仪馆火化都要排队抽号，排不到号的家属不得不在大街上架起火堆火葬至亲，现场满目疮痍，惨不忍睹。

鉴于印度的疫情，世界各国如：新西兰、香港、英国、美国、加拿大，纷纷禁止往返印度的直航，随着疫情进一步加剧断航国家的列表可能会更长。

2020年年底印度疫情一度控制很好，每日感染病例和死亡数都控制在相当低的状态，当时看似乎前景相当乐观。可从2021年3月中旬开始，感染病例急剧上升一路飙升，从每天死亡不到100人增加到现在的单日最高峰3780人，而且居高不下。这一切到底是怎样造成的呢？

科学家们列举了印度病例激增的各种原因，印度媒体也在纷纷寻找原因总结教训，似乎以下两点是造成目前印度人民生灵涂炭的主要原因。

第一是政府和民众的松懈。2020年底印度官员宣布该国已经扭转了疫情曲线。形势似乎一片大好，单日新增病例数一度降至1万例以下。相比2020年9月中旬平均每天超过9万的高峰期，感染率稳步下降。截至2月中旬，印度平均单日新增11000例，七日平均死亡不足100例。印度政府在全国范围内解除封锁，民众欢呼自由。开始各种旅游、聚会，举办盛大的婚礼、宗教仪式、大规模政治活动。大部分人群不带口罩，也没有很好的保持社交距离。

第二个原因是病毒的变异。和世界上其它一些地方一样，印度去年就发现中共病毒变种。科学家新近将其命名B.1.617，并称之为超级变种。这一变种的棘突蛋白发生双重突变，比原始菌株更具感染力。据印度方面的数据，在疫情严重的邦，超过60%的患者感染的都是B.1.617病毒，超过了英国中共病毒变种的覆盖面。世卫组织发言人也指出，变种的含义是指更容易传播、导致更严重的病症，且逃避免疫系统、改变临床表现或降低已知工具的有效性。

表层蛋白上，印度冠状病毒变体B.1.617呈现出两种基因突变：E484Q和L452R，这两个神秘数字揭示了其各自基因组改变的确切位置。E484Q与E484K相似，后者此前已在英国、南非和巴西的变异病毒中出现；L452R则已在加利福尼亚变体CAL.20C中出现。然而，现在的情况是，在印度，这两种突变首次同时出现，也因此常被称为印度 “双重变异”变种的两个突变，这种突变会导致抗体或T细胞的中和能力降低，其程度尚不清楚。

**E484Q****突变**，类似于在巴西和南非变种中鉴定出的E484K突变，最近几个月也有报道。这种突变会改变冠状病毒刺突蛋白的一部分。刺突蛋白形成冠状病毒外层的一部分，是病毒用来与人细胞接触的蛋白。接触后，冠状病毒随后使用刺突蛋白与人细胞结合，进入并感染人细胞。疫苗刺激的免疫反应产生特异性靶向病毒刺突蛋白的抗体。因此，如果突变显著改变了刺突蛋白的形状，那么即使在已经接种疫苗的抗体中，抗体也可能无法有效识别与中和病毒。科学家也正在研究E484Q突变是否也可能如此。

**L452R****突变**也发现于一个变异体中，该变异体被认为与加利福尼亚的爆发有关。人们认为，这种变体增强了刺突蛋白与人类宿主细胞结合的能力，从而增强了其感染性。对该突变的研究还表明，它可能有助于病毒逃避疫苗和先前感染均可产生的中和抗体，这一切仍在研究中，其程度有待于将来的结果。

美国最高公共卫生专家安东尼·福西（Anthony Fauci）呼吁，除了全国封锁“数周”，以打破“感染的连续性和传播”。还建议进行大规模疫苗接种，他说道：“目前，他们应该开始尽可能多地给人们接种疫苗，包括他们在印度自行开发的疫苗以及可以从其他供应商（例如美国）购买的疫苗供给”所有的科学家和专家认为疫苗是阻止中共病毒的最有效的办法，而且所有国家都愿意死死抓住疫苗这唯一一根救命稻草。

图一是2020年3月14日到2021年4月18日，中共病毒爆发以来印度死亡数量曲线图。图二是印度从2021年1月23日到5月3日疫苗的接种量。

从图表中看到，疫苗是2021年1月份开始打的，三月中旬数量上升，到目前疫苗注射数量仍在大幅的提升。在死亡数据图一中，死亡数量也是几乎从一月份开始慢慢上升，3月中旬开始一路飙升，曲线弧度和图二中同期几乎一致。让人怀疑疫情造成的大规模的死亡数量是否和疫苗的注射数量有直接的关系？但目前我们还不能确定。

巴赫尔博士是印度一家制药公司的领先医学研究人员，同时也是一位治疗中共病毒患者的医生，他在接受有关印度近期疫情危机问题的采访中透露：2020年三月份开始的第一波疫情中绝大多数感染的病例是生活在贫民窟里社会底层的人们，而贫民窟的居民在政府和市政药房可以接受羟氯喹和锌的治疗方案。这有助于保持极低的死亡率。而第二次疫情中社会中高阶层的病患也被拖入疫情中。印度大多数家庭医生不再推广使用价格便宜的包含羟氯喹、伊维菌素和吸入布地奈德等药物做为治疗中共病毒的药品，而是使用价格更昂贵的药物如Favipiravir（法匹拉韦）和Remdesivir（瑞得西韦）等作为治疗药品。这导致了瑞得西韦的巨大供需缺口。同样值得注意的是，当使用瑞得西韦时，即使由家庭医生给予，羟氯喹HCQ也会被停止使用，而且这些人比其社会经济地位较低的同龄人更早被接纳为住院病人，更有机会使用医疗资源，这使得更多的贫民窟里的穷人们失去治疗的机会。
![]()![](https://gnews.org/wp-content/uploads/2021/05/Screen-Shot-2021-05-11-at-11.39.54-AM.png)图三  来源：COVEXIT.COM
巴赫尔博士提到印度有7个州占所有活跃病例的67％。该图显示，最右边的乌塔邦（Uttar Pradesh）做了正确的事情，而玛哈斯特拉（Mahashtra）、查提撒贺（Chattisgarh）、卡纳塔卡（Karnataka）却没在正确的轨道上。从上面的例子来看，乌塔邦（Uttar Pradesh）的人口是玛哈斯特拉（Mahashtra）的两倍，但活动病例的数量是却不到玛哈斯特拉（Mahashtra）的1/4。这仅仅是因为每个人都被提供包含伊维菌素的家庭治疗药盒产生的积极效果。果阿是另一个州，尽管比上面列出的7个州要小得多，但它提供了依维菌素家庭治疗套件。这使两个州都可以避免封锁，而玛哈斯特（Mahashtra）则处于完全锁定状态。即使与孟买相比，它的孪生大都会塔那（Thane）也具有更广泛的伊维菌素用途，并且死亡率更低。

第二波的另一个特点是，与第一波相比，受影响地区的数量更多地集中在城市地区。占病例前50％的地区的数量已从上次高峰时的40多个地区减少到目前的不到20个地区，表明大流行病更加集中。与上一波浪潮相比，一线医生人数大量减少，而且医生对治疗这种疾病的方法也发生了变化，不再像以前那样使用了羟氯喹和伊维菌素等药物。首先，在小型私立医院和疗养院中不会像配Favipiravir（法匹拉韦）和Remdesivir（瑞得西韦）那样备有羟氯喹和伊维菌素。一些具有影响力的意见中越来越多的趋势是只促进血液稀释剂和类固醇用于家庭治疗， 其他人则在媒体上宣传，扑热息痛和免疫力足以在家中对抗病毒。一些州政府机构通过声明不使用HCQ的方式取消了HCQ的使用。

由于整个国家都遵循所有标准的预防方法，例如口罩、卫生措施和社会隔离，因此差异可能归因于这些州的患者所接受的差异化治疗方案。

另据印度QUARTZINDIA NEWS（QZ）记者莎玛(Niharika Sharma)2021年2月3日报道：在2020年疫情的初期羟氯喹曾经在印度大量使用，6月中旬WHO声明，没有数据显示硫酸羟氯喹对COVID-19有明显作用，使得美国、法国和澳大利亚的科学家停止对硫酸羟氯喹的研究。就此使得在医疗界使用硫酸羟氯喹对抗COVID-19的治疗方案没有了下文。停用硫酸羟氯喹是不是造成印度中共病毒大规模爆发的另一个主要原因？还有待于科学家们给出正确的答案。

2021年5月4日和5日赛林博士连续2次发表公开声明指出：新冠病毒是中共制造的生物武器！ 病毒并非源于自然！他将这一信息传递给所有政府官员和媒体，尤其是那些科学界人士。 COVID-19是中共病毒功能增强研究的产物，来自于中共科学家的军民融合项目，是实验室制造的。 中共军方生物实验室套上民用生物研究所和大学的画皮，以知识、技能和资金的方式骗取美国纳税人的数十亿美元，其中大部分的资金，来自福奇医生领导的美国国家过敏和传染病研究所。虽然武毒所是焦点，但病毒的原材料和技术（包括病毒骨架）是多个军方控制的实验室提供的，包括病原体培育、疫苗生产项目。 在解放军北京军事医学科学院的监督下，那些最令人感兴趣的病毒被收集和发送到多个实验室，包括武汉病毒研究所、武汉大学实验室和武汉军医院。

赛林博士曾任美国司令部中央顾问团司令，目前任职太平洋特种部队司令部参谋长。 选择此时发表声明，意味着美国军情届对中共病毒的定义已完成。塞林博士向全体美国人民发出号召：是时候与中国人民站在一起，彻底消灭中国共产党！
