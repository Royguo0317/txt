据法国广播网2020年12月12日报道，瑞士联邦委员会12月11日宣布对包括白俄罗斯总统卢卡申科（Alexander Lukashenko）及其子兼国家安全顾问维克多·卢卡申科（Viktor Lukashenko）在内的15人实施金融制裁和旅行禁令。有关个人被禁止入境瑞士，其在瑞士的资产将被冻结。

上个月，欧盟已经宣布对包括卢卡申科在内的15人实施制裁。

白俄罗斯于今年8月9日举行新一届总统选举，时任总统卢卡申科以80.1%的得票率胜出，第六次当选总统。明斯克等城市随即爆发大规模抗议活动，指责选举不公正。西方民主国家大多不承认白俄罗斯选举结果，并相继推出制裁措施。

据BBC中文网上的消息，亚历山大·卢卡申科被称为“欧洲最后一个独裁者”，铁腕掌控白俄罗斯已经20多年。卢卡申科的领导风格带有苏维埃时代遗留下来的专制色彩，比如，掌控主要媒体，骚扰、打压政治对手。2002年，卢卡申科曾经公开讲，“专制的领导风格是我的特点，我从来没否认过。需要控制住国家，主要任务是不要毁了人民的生活。”白俄罗斯强大的秘密警察部门密切监视异见人士，导致他们大多或流亡海外，或被关入监狱。

据《维基解密》几年前披露，中共历任独裁者家族及高级官员在瑞士银行拥有约5千个账户，总资产达数万亿美元。随着以美国为首的西方对中共一党专政的邪恶本质看得越来越清楚，中共非法掠夺的中国人民的财产终将重新回到人民手中。
