《国防新闻》2021年1月29日报道，通用动力公司负责人菲贝·诺瓦科维奇1月27日告诉股东，通用动力公司的潜艇制造厂预计在未来几年内的收入将显著增长。

![]()![](https://gnews.org/wp-content/uploads/2021/01/B4A64BDWOBBM5P4DS54S7OUA7E-3.jpg)建造中的“伊利诺伊”号攻击潜艇 图片来源：美联社

诺瓦科维奇称，由通用动力制造的新一代弹道导弹潜艇哥伦比亚号将成为收入增长的主要原因。随着中国成为世界上最大的海军力量，美国可能会将潜艇视为打击日益复杂的竞争对手的王牌。

诺瓦科维奇说，装备哥伦比亚号这样的先进潜艇是美国海军的头等大事，这是因为潜艇仍然具有独特的竞争优势。通用动力公司去年和美国海军签订了建造两艘超过100亿美元的哥伦比亚级潜艇的合同。

美国由于西临太平洋，东靠大西洋，有漫长的海岸线，因此在国防战略上对潜艇规模及反潜技术一直要求很高，从而成就了美国第一潜艇大国的地位。

核潜艇具备装载和发射核武能力，其隐蔽性为近距离袭击对手提供了潜在的条件，因此有极高的战略威慑力。哥伦比亚潜艇又称俄亥俄级核潜艇，是目前世界上最先进的核潜艇之一。
