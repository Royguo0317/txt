## 日本外相致电王毅罕见强烈批评中共

编撰：叶落知秋（文秋）, PEACEMAN
审核：MiYa

![]()![](https://gnews.org/wp-content/uploads/2021/04/1-14-3.jpg)图为日本外相茂木敏充。(KAZUHIRO NOGI/AFP via Getty Images)

4月5日，日本外相茂木敏充给王毅打电话90分钟，表达了罕见的“强烈担忧”，他要求中共停止入侵争议海域，呼吁中共改善维吾尔人的人权条件，并停止对香港的镇压。

4月16日日本首相菅义伟将亲赴美国，与美国总统拜登进行首次首脑会谈。日本在这次会谈前释放出罕见的强硬信号，引发国际关注。

更加引发外界联想的是，日本官方的通话记录中没有提到王毅的回答，而直到4月5日当天晚上，中共官媒才对这次两国外长之间的对话进行简短报导，但是对日本提到的领海、民主和人权等问题只字不提。

《华日》报导，根据日本官方的通话记录，在一个半小时的通话中，日本外相茂木敏充对王毅提出，中共的海岸警卫队船只继续出现在争议海域。

日本外交部也在一份声明中说，茂木敏充还就中共入侵、香港局势和有关维吾尔族的人权问题等，对王毅表示了“强烈担忧”。声明还说：“他还强烈呼吁采取具体行动。”但未透露更多细节。

日本防卫省周日（4月4日）表示，前一天早上，中共海军航母辽宁舰等共6艘解放军舰艇，通过冲绳本岛与宫古岛之间海域，南下进入太平洋。这是自去年四月以来中共船只首次穿越该地区。

3月16日，美日举行“2＋2”会谈。会后美日发表联合声明，称两国确认“中国（中共）的行为不符合现有的国际秩序，给联盟和国际社会带来了政治、经济、军事和技术挑战”。这是日本罕见的直接点名中共，这让中共气得跳脚，外交部发言人赵立坚抨击日本“仰人鼻息”，“狼狈为奸”。

编者观点：
一直以来，日本都尽量避免激怒其最大的贸易伙伴——北京，东京一直试图与北京维持一种平衡的关系，但是最近开始越来越直言批评中共。根据日本共同社报导，菅义伟此行还会跟拜登谈到台湾问题，强调“台湾海峡和平与稳定的重要性”，他表示期望日美合作，创造两岸和平解决的环境。但外界预计这会刺激中共跳得更高。

中共一直视亚洲为其势力范围，并通过无底线渗透和产业分工的经济钳制手段，试图控制亚洲各国。随着外界越来越清醒地认识到中共控制亚洲甚至称霸全世界的野心，中共同美国及西方的对抗必将走向不可调和的地步，届时美国的盟友日本包括印度韩国等亚洲国家一定同西方走向合作并围剿中共。
