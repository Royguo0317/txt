**香港人写给西贡区议员钟锦麟的信**

Ben,

记不记得我和你及区诺轩，同期出道，同期当选，甚至是同年龄当选？

记得有一天，你笑着跟我说：「我们三个还未曾拍过合照！」我笑了一笑，然后就走了，不是因为我「寸」或者「高傲」，而是本身我就有性格障碍，我害怕与其他人混熟、永远会对「同行」充满戒心，假面具总是挂在脸上。但阿Ben你永远对我伸出善意的手，约我喝下午茶；搬去马鞍山后又探访我的办事处，想多个朋友，我是知道的。

近日看到47人案的新闻，我和办事处同事好多时侯都会忍唔住哭泣，里面有好多我们的朋友；Ben，我们之后一定要好好坐下来，还要拍一张这么多年都拍不成的合影，除此之外，我承诺终生请你们喝下午茶，因为身为香港人的我，亏欠你们。

沙田区议员麦仔
![]()![](https://gnews.org/wp-content/uploads/2021/04/40396.jpg)
人物介绍：

钟锦麟（英文：Ben Chung，1988年11月19号—），岭南大学文化研究文学士，现任香港西贡区议会主席兼欣英选区区议员。新民主同盟前成员，亦为新民主同盟创党成员、民主党前成员，现为独立民主派人士。 2021年1月6日因早前参与立会初选涉嫌违反港区国安法而被捕，目前正在羁押中。
