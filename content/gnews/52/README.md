## 中共超限战灭港计划（四十三）六七暴动之中共欲武力收复香港

搜集/撰文：岁月如歌 / 封面合成：文粤

六七暴动发展到最后演变成中共的红色恐怖活动。香港遍地的炸弹处处伤人，左派激进份子时时暗杀警察，外交上也到处犯口犯手。大家有没有发觉这一场所谓的反英抗暴反殖民地暴动从头到尾都是中共左派拿自己人来开刀、自己人斗自己人制造矛盾，制造动乱，逼迫港英政府出手维护治安拯救人民，利用这种方式逼迫港英政府在维护社会治安的时候触犯法律犯错等来诬蔑或者控告港英政府的法西斯统治法西斯暴行。到最后受害的都是香港无辜的人民和在港英警察部门工作的警察。
![]()![](https://gnews.org/wp-content/uploads/2021/04/42210.jpg)图片来自pinerest
从1967年5月6日到12月最初的工人薪资问题制造矛盾，引领工人罢工、游行、抗议、集会到放火、街头打架、街头枪击、到炸弹袭击和滥杀平民和军警。矛盾加剧。港英政府努力维护社会治安成功地平息了左派的阴谋暴乱从某个意义上来讲这是积极的。但是港英警察在执行任务的过程中所犯的错也是不能忽视的我们必须要记住，功是功过是过。长达八个月的六七暴动期间给香港带来前所未有的创伤，也给香港人民带来从未有过的恐怖。箇中的点点滴滴无法用文字表达完整。港英统治下的一代又一代的香港人在这片土地上遭受的痛苦、挫折和磨难都比不上香港来了中国共产党这个魔鬼到来之后所遭受的痛苦和磨难多。
![]()![](https://gnews.org/wp-content/uploads/2021/04/42211.jpg)图片来自medium
五十多年过去了这些罪犯依然逍遥法外，并在偷偷地抹去曾经犯下的罪恶。如六七暴动档案正本英国全部运回去放在英国国家档案馆保存。而香港历史档案的馆的影印文件几乎全部消失了，只有21秒的影像。英国能好好的保存在国家档案馆，香港的却不翼而飞这说明什么呢？ 2015年9月香港警察官方网站介绍警队历史部分内容中关于六期暴动的内容遭篡改，并且警队历史博物馆并没有展出在六期暴动中搅和的左派的攻击性武器。警方为什么不展示？难道怕世人知道土制炸弹和燃烧弹的威力吗？还是怕香港人知道当年六七暴动在北角的华丰国货公司里边竟然隐藏有一个「战地医院」吗？这场惨重的暴动也没有在教科书中出现，教育局负责人说这场暴动只是鸡毛蒜皮的小事，并没有具备重要的历史意义。罪犯不但逍遥法外而且还受到嘉奖，例如前工联会会长杨光在香港回归后第一时间得到香港特区政府颁发的大紫荆勋章，得到勋章的还有庄世平、吴康民、李泽添。得到政府的嘉奖的人绝对是对香港有突出贡献的人，这无疑是对这些人在六七暴动中有突出贡献，承认六期暴动是政治正确。
![]()![](https://gnews.org/wp-content/uploads/2021/04/42212.jpg)图片来自维基百科
2015年，杨光死后，吴康民解释六七暴动的本质，「在内地的“文化大革命”运动弥漫全国的时候，香港的北京派出机构新华社领导人，既为表功响应中央号召，又为自保，居然利用一场劳资纠纷，加上不少劳工阶层对港应的统治手法不满，煽动起这一场所谓的“反英抗暴”大风暴。但实际上“反英抗暴”根本违反了中央自解放以后对香港的政策，而是香港新华社一手包办的“斗委会”」。吴康民不但帮中共擦屁股洗地还为杨光脱罪称他只是个傀儡，都是“斗委会”的责任，斗委会只是个临时组织，这样一来中共的主要参与人都轻易地脱罪了。事实上是不是这样呢？中央真的不打算收回香港吗？要向人不知除非己莫为。
![]()![](https://gnews.org/wp-content/uploads/2021/04/42213.jpg)图片来自pinterest
1997年《党史纵横》第八期刊登了一篇题为《高瞻远瞩果断英明——“文革”中周恩来阻止进军香港》，1967年文革如火如荼，毛泽东在北戴河跟中央文革小组成员、中央军委文革小组成员和当时没有被点名靠边站的政治局委员共32人召开中央政治局扩大会议。并一致通过了中央文革小组起草的《关于立即收回香港，结束应帝国主义殖民统治》的决议。这个议案在67年4月由林彪、江青、陈伯达、康生、谢富治、张春桥等人首次提出。谢志富和张春桥负责起草。

周恩来知道后感到事情严重了，赶紧找朱德、董必武、陈云、李先念、陈毅、李富春、刘伯承、叶剑英等老头子怜悯给毛泽东写信要求暂停收回香港。周前后七次跟毛说，最后毛听取了周的建议，「并做出决定：一、暂停进行收回香港的各项准备工作；二、建议五年至十年不作新的决定；三、向英国表明，不准香港变成反中的地区，不准台湾势力渗入香港，如果发生剧变，中国随时收回香港」。有了毛的点头，周恩来叫停了香港的文革。
![]()![](https://gnews.org/wp-content/uploads/2021/04/42214.jpg)图片来自自由时报
笔者在前面就说过中共非常想收回香港但是由于自己的经济问题需要利用香港挣钱，创汇等，不得不让香港继续由英殖民地管制。这篇文章就披露出中共想利用六七暴动收回香港阴谋，也证明了中共一直想收回香港。九七回归之前邓小平也担心不能顺利收回香港，曾表示用武力收回。 《周南口述：遥想当年羽扇纶巾》一书披露「小平同志语气很严厉，他说香港的情况我很清楚，香港问题就是一句话，对英国人一点也不软的⋯⋯对他们背信弃义的做法必须坚决顶住，绝不能让步。要质问他们，中英协议还算不算数？如果他们一意孤行，我们就要另起炉灶。另起炉灶这是小萍讲的⋯⋯我在1982年见撒切尔夫人时提到必要时在另外的时间、采取另外的方式收回香港的讲话，今天仍然有效。⋯⋯另外的方式就不是和平的方式了⋯⋯要做好各方面的准备工作，包括军事方面的准备工作」 。
![]()![](https://gnews.org/wp-content/uploads/2021/04/42215.jpg)图片来自夜猫
2019年反送中运动又再一次证明中共一直都想收回香港的阴谋。虽然九七回归已经让香港回到“祖国”的怀抱，但是由于提出“一国两制，港人治港，高度自治”这个多少也让中共不能公开插手香港事务，恨得痒痒的。为了牢牢的把香港捏在手里随意操控，中共在回归后依然在不断地渗透香港自由和蚕食香港民主，熬了22年终于忍不住了，撕毁了《中英联合声明》公开掠夺香港，把它真正握在手里。周恩来虽然操控了六七暴动，但是他毕竟对香港了解，知道收回香港的后果，所以不敢轻易的出手把香港拿下来。在知道利用暴动无法拿回的情况下，口风一致说成是左派的错，中共无意收回香港。而习近平就是一个侵略者暴君，不管三七二十一就武力拿下香港，并在香港实施红色统治。他以为拿回香港就可以为所欲为，他以为中共统治下的香港依然还能够保持原来的状态为中共赚大钱创更多的外汇等等，但是人算不如天算，丧尽天良做的事就要遭到天谴。中共习近平毁了香港。中共的作恶让全世界看清了他在香港的真面目，美国取消了香港的最优惠国待遇，让香港一夜之间成为中共国的一个省，失去了真正的意义。现在中共握着香港还在继续做恶，相信报应会很快到来。
