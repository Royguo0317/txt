- 欧盟针对香港选举改革向北京发出警告，称已准备好采取额外措施以应对任何人权恶化的情况。


- 中共国驻欧盟代表团表示，从香港近年来的情况看，该地区的制度有待完善。


据中共国国家媒体报道，欧盟警告北京反对香港的选举改革，并称准备采取额外措施以应对香港政治自由和人权的任何进一步恶化。

在欧盟发出这一呼吁之际，中共国领导人周五在全国人民代表大会(NPC) 的第一天–中共国政治历史上最重要的年度活动，发誓要改革选举制度，以“阻止外部势力对香港和澳门事务的干涉”。

中共国对欧盟的言论提出异议。

据《环球时报》报道，中共国驻欧盟使团敦促该集团摒弃在香港问题上的错误立场，停止讲无益的话、做无益的事，停止干涉香港和其他中共国内政。

作为香港选举改革的一部分，采取的措施细节尚未公布。不过，据《南华早报》报道，立法会将在该市行政长官的选举委员会中挑选出20名议员增加到立法会名额中。该委员会本身可能会再增加300名选民，成为一个1500人的外围机构。

欧盟的声明写道，“如果颁布，这种改革将对香港的民主原则和民选代表产生潜在而深远的负面影响”。

欧盟声明中还写道，“这项改革也会与香港以往的选举改革背道而驰，并违背了《基本法》第四十五条和第六十八条关于在行政长官和立法会选举中实行普选的承诺。”

中共国驻欧盟代表团回应称，香港近年来的情况表明，该地区的制度需要完善，为落实“一国两制”和“爱国者治港”提供系统支持。
