#### 原文翻译如下:

**社交媒体平台Parler 周一宣布在离线一个多月后已重新启动。**

1月9日，在国会大厦暴力骚乱发生三天后，Parler网站被亚马逊的云託管服务（Amazon Web Services）暂停，并于第二天下线。该公司因未能规范要求暴力而受到批评，据报导该公司被一些特朗普支持者用来协调“制止偷窃”运动。

尽管苹果公司在当天早些时候，将Parler应用程序升至免费应用程序的榜首，但苹果公司将Parler从其应用商店中暂停后，亚马逊也暂停其服务。谷歌还暂停了Parler在其谷歌应用商店中的销售。亚马逊声称，Parler在未能删除构成暴力威胁的帐户后被暂停使用，但Parler坚持保留对尽快删除该帐户看法。新贵的社交媒体平台因其言论自由的承诺吸引了用户。

Parler宣布不再与科技巨头合作，而是 “以可持续的独立技术为基础”。该公司还宣布了新的临时首席执行官马克·梅克勒（Mark Meckler），他是右翼组织茶党爱国者（Tea Party Patriots）的共同创始人。Parler本月早些时候解雇了其前首席执行官约翰·马特（John Matze）。

梅克勒周一在一份声明中说：“Parler旨在提供一个社交媒体平台，以保护言论自由，并重视隐私和公民话语。当Parler在1月份被渴望让数千万美国人保持沉默的人们下线时，我们的团队团结了起来，决心兑现我们对高度参与的社区的承诺，即我们将比以往任何时候更加强大。” “我们很高兴欢迎大家回来。” 梅克勒补充说，Parler正在“由一支经验丰富的团队经营，并且会留下来。” Parler于2018年8月首次推出，现在拥有超过2000万用户。

**评论:**
可见Parler两千万的用户力量不容小觑，也代表着一种民众看清主流媒体平台对于言论自由的打压，特别是在这次美国大选之后。当人民呼喊的声音大到一定程度的时候，当局就不得不重视了。而且主流媒体在中共病毒真相和美国大选报导面前可以说是真相已死，在越来越清醒的美国人面前，科技巨头想要完全压制拥有2000万账户的社交媒体平台是不可能的。所以我认为Parler恢复是必然的，至于换掉CEO是不是妥协的结果，我们还需要再观察。 （**评论只代表译者本人的观点**）
