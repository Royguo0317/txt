## 独家：彼得•施韦泽：赵小兰“家族与中共国政府关系紧密”

[!\[\]()!\[\](https://spark.adobe.com/page/9ZumgL4xpL60t/images/782bb3bd-d196-4e58-a486-a8fd47fb0741.jpg?asset_id=3e6231d4-6f7f-4223-9bd0-589a90baa298&amp;img_etag=%2256d94d6c30d8384c975ffd0bb976d985%22&amp;size=2560)](https://spark.adobe.com/page/9ZumgL4xpL60t/images/782bb3bd-d196-4e58-a486-a8fd47fb0741.jpg?asset_id=3e6231d4-6f7f-4223-9bd0-589a90baa298&amp;img_etag=%2256d94d6c30d8384c975ffd0bb976d985%22&amp;size=1024)华盛顿特区——1月11日，参议院商务、科学和运输委员会在国会山Dirksen参议院办公大楼举行的下一任美国运输部长确认听证会上，赵小兰（中）听取了她的丈夫、参议院多数党领袖米奇•麦康奈尔（Mitch McConnell，肯塔基州共和党议员）（右）讲话……（图片：Chip Somodevilla，盖蒂图片）

**《秘密帝国：美国政治阶级如何隐藏腐败并使亲友自肥》的作者彼得•施韦泽(Peter Schweizer)在SiriusXM 广播频道周五的布赖特巴特每日新闻节目中与主持人Alex Marlow指出，曾任运输部长、参议院少数党领袖米奇•麦康奈尔（肯塔基州共和党议员）的妻子赵小兰(Elaine Chao)的家族与中共国政府有着广泛的经济联系。**

周二，运输部监察长致信，概述了对有关赵小兰利用其办公室和工作人员帮助其家人从事国际航运业务指控的调查。

ForemostGroup（福茂集团）是赵小兰父亲赵锡成(James Chao)创立的航运企业。这个家族的生意主要围绕中共国展开，因此这需要赵小兰的家族与中共国政治精英之间的联系。中共国最大的国防承包商中国国家造船公司(CSSC)的子公司，为福茂集团制造船舶。

施韦泽谈到福茂集团时说：“赵小兰的父亲赵锡成和她的妹妹赵安吉(Angela Chao)一起经营一家航运公司，这项业务与中共国政府有着非常紧密的联系，这意味着中共国政府出资建造他们使用的船只，提供船员，而他们负责建造船只并提供合同。因此，人们非常担心中共国可能对麦康奈尔-赵氏家族施以影响。”

施韦泽描述了被外界诟病的赵小兰之前的运输部长一职与其家族的造船和航运业务之间的利益冲突。

施韦泽说：“监查长(IG)报告中的指控是，赵小兰在2017年1月作为运输部长组织了一次前往中共国的正式旅行。她的计划是带着她的父亲和妹妹一起参加这次正式旅行，而他们要拜访的地方包括与该家族有业务往来的机构。该计划遭遇了来自运输部内部人士的阻力。”

赵小兰还被指利用她以前的运输部长职位来帮助宣传她父亲写的回忆录。

[!\[\]()!\[\](https://spark.adobe.com/page/9ZumgL4xpL60t/images/fc52e5a1-8c31-4389-96d2-52f6fdedbc82.jpg?asset_id=ad46ee4e-2df6-413b-be48-b38c30d60c43&amp;img_etag=%22cad37d66c447a8c0ebc09fc1d53955c0%22&amp;size=2560)](https://spark.adobe.com/page/9ZumgL4xpL60t/images/fc52e5a1-8c31-4389-96d2-52f6fdedbc82.jpg?asset_id=ad46ee4e-2df6-413b-be48-b38c30d60c43&amp;img_etag=%22cad37d66c447a8c0ebc09fc1d53955c0%22&amp;size=1024)

施韦泽评论道：“赵小兰正在使用由纳税人资助的运输部来推广她的书。这是一本在中共国发行的书，而主要受众也在中共国。”

施韦泽解释说：“她的父亲在大陆发行了一本书籍，算是他的人生故事。赵锡成常被认定为来自台湾，其实他来自中共国大陆，并和一个叫江泽民的人是同学。江泽民最终在20多年前成为中共国的领导人，这也是他为什么会（对中共国政府）如此津津乐道的真正原因。”

施韦泽说：“这个家族与中共国政府有着紧密的联系。事实上，她的妹妹赵安吉是少数几个在中国银行董事会中任职的外国公民之一，而中国银行对于中共国经济国策而言，是最大的银行，也是最重要的银行。”
