# **英国更新香港商业指南，删除了“国际金融中心”和“相互合作”的内容**

**指南还提到，根据北京去年夏天通过的国家安全法，这个城市的自由正在减少。**

英国外交部更新了香港的海外商业风险指南。这次更新删除了伦敦与香港享有“积极的、前瞻性的”关系以及与香港“互利合作”的短语，这意味着伦敦与香港未来的关系将更加冷淡。

更新后的指南于周二发布，为希望在香港市场开展业务的英国企业提供了经济和地缘政治风

[!\[\]()!\[\](https://spark.adobe.com/page/yChM9eVGMcfeB/images/d94afc3f-beda-421f-a07b-9f6d937e07c3.png?asset_id=d5cca79b-95c5-4a05-8685-5760a1a5e935&amp;img_etag=%22b8f0ead4c4c5fc2ee818c502d790204f%22&amp;size=2560)](https://spark.adobe.com/page/yChM9eVGMcfeB/images/d94afc3f-beda-421f-a07b-9f6d937e07c3.png?asset_id=d5cca79b-95c5-4a05-8685-5760a1a5e935&amp;img_etag=%22b8f0ead4c4c5fc2ee818c502d790204f%22&amp;size=1024)2019年在英国举行的抗议活动。图片：香港自由新闻。

指南还提到，根据北京去年夏天通过的《国家安全法》，这座城市的自由日益减少。指南写道：“[国家安全法]中的罪行定义广泛，但它已经在减少香港人能够行使其基本权利和自由的程度。”

英国外交部也对言论自由表示了关切：“据报道，由于法律中的规定在今后如何被使用的不确定性，这鼓励了自我审查，对行使自由产生了更广泛的寒蝉效应。”

该指南最近一次更新是在2019年8月，之前的内容是“香港普遍尊重公民的权利和自由。”

现在，指南警告说，根据国家安全法，“法人团体或非法人团体可被处以罚款，并可以暂停其业务或吊销执照/许可证”。

[!\[\]()!\[\](https://spark.adobe.com/page/yChM9eVGMcfeB/images/9635485f-29f5-40c9-a3f7-f5cebfd03739.png?asset_id=0c8a1609-cbd7-4df7-88c9-b07247533797&amp;img_etag=%227c877cae99069b184dde7890d1a885b8%22&amp;size=2560)](https://spark.adobe.com/page/yChM9eVGMcfeB/images/9635485f-29f5-40c9-a3f7-f5cebfd03739.png?asset_id=0c8a1609-cbd7-4df7-88c9-b07247533797&amp;img_etag=%227c877cae99069b184dde7890d1a885b8%22&amp;size=1024)图片：来自英国政府网站的屏幕截图

周二的更新还删除了“经济概览”部分下，将香港描述为“国际金融中心”的措辞。取而代之的是，新的指南将香港称为“区域中心”。

海外商业风险服务为英国企业提供信息，以保护他们免受国外的安全和政治风险，包括与人权、腐败、恐怖主义、犯罪活动和知识产权相关的潜在风险。

## **紧张局势上升**

**该指南的更新正值伦敦和北京有关这个前英国殖民地问题的紧张局势日益上升之际。**

英国外交大臣多米尼克·拉布(Dominic Raab)多次批评香港当局利用国安法镇压该市的政治异议，指责北京违背了《中英联合声明》中关于香港享有高度自治50年不变的承诺。香港方面则表示，法律只会影响少数人，权利和自由仍然会受到保护。

[!\[\]()!\[\](https://spark.adobe.com/page/yChM9eVGMcfeB/images/b55c4b9f-2946-4188-967f-7b5cfa034ddd.png?asset_id=853f7ae6-df83-4644-a826-6aac80432d6a&amp;img_etag=%229cb8ca34fb9067734546d8c011f05bf2%22&amp;size=2560)](https://spark.adobe.com/page/yChM9eVGMcfeB/images/b55c4b9f-2946-4188-967f-7b5cfa034ddd.png?asset_id=853f7ae6-df83-4644-a826-6aac80432d6a&amp;img_etag=%229cb8ca34fb9067734546d8c011f05bf2%22&amp;size=1024)档案照片：JimmyLam/联合社会出版社。

上月，英国开始了一项延长英国国民（海外）签证计划，允许高达70%的香港人在英国生活和工作，并可在6年后获得正式的公民身份。

作为报复，北京宣布不再承认BN(O)为有效证件，而行政长官林郑月娥则确认，香港的双重国籍持有人将不会获得领事支持。

2020年6月，在经历了一年的民主抗议和动荡之后，北京绕过香港立法机构，直接将国家安全法规插入香港的小宪法中。它把颠覆、分裂国家、外国干涉和恐怖主义行为定为刑事犯罪。这些行为描述广泛，包括了对公共交通和其他基础设施的破坏。此举使警察获得了新的权力，并令民主人士、民间社会团体和贸易伙伴感到震惊，因为此类法律已被广泛用于压制和惩罚中共国的异见人士。
