在扫黑除恶运动还没结束， 2月27日，中共政法队伍教育整顿动员部署会议就已经开始。会议提出，要发扬彻底的革命“自宫”精神，再掀起政法系统内的“扫黑除恶”活动。会议要求，坚持全链条发力、全系统整治，刀刃向内、刮骨疗毒，坚决清除政法系统害群之马、彻底整治顽瘴痼疾。通过整治打造一支习总信得过、靠得住、能放心的政法铁军。

这是继去年7月份，中共政法队伍教育整顿试点工作的延续。试点期间成果显著，全国政法系统至少已经有10余名高官落马，其中包括两名省部级官员。试点地区共处分处理政法干警2247人，占干警总数的14%；立案审查448人；移送司法机关39人。可以说，处理的人数比例相当大，两千多个家庭又完了。全国政法系统人数大约300万人，如果按照14%的处理比例，今年将有40万人受到处理。

这次整肃运动是十分严厉的：公安机关将对2018年以来的警情、案件进行抽查检查；监狱对“减、假、暂”案件倒查20年。20年前是江泽民执政，周永康把持政法系统的年代。

以黑治国、以警治国，孟建柱、孙立军等等这些人和事，对爆料革命的战友们而言是太熟悉不过的。中共的政法系统对爆料革命的战友和中国人民犯下的也罪恶罄竹难书。

政法系统作为中共合法的杀人工具，是每个执政者坐稳江山必须控制的刀把子。去年扫黑除恶还没结束，习近平就要赶在二十大之前对“打黑者”——政法系统开始打黑。习近平就是要通过这次整肃运动换成自己的嫡系部队，绞肉机模式已经加速运转。
