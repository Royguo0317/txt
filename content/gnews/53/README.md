03月09日，克里姆林宫曾表示，俄罗斯存在的本身就会引起美国的担忧。俄罗斯总统发言人德米特里·佩斯科夫（Dmitry Peskov）现在用这句话来回应评价美国太空司令部负责人关于俄罗斯和中共国太空军事力量的言论。

“感觉就像俄罗斯的存在引起一些美国将军感到担忧。”佩斯科夫评论道。

克里姆林宫发言人还补充说，这些担忧是没有根据的。佩斯科夫指出，“俄罗斯是一个和平的国家，赞赏俄美在太空领域的合作。”

此前，美国太空部队指挥官约翰·雷蒙德将军在接受《纽约时报》采访时，对俄罗斯拥有可能破坏美国卫星的武器表示担忧。将军还谈到了俄罗斯的一颗卫星，他认为俄罗斯的卫星是为了 “摧毁地球轨道上的美国卫星”。

2020年12月，美国国防部太空司令部报告称，俄罗斯测试了直接拦截反卫星导弹（DA-ASAT系统）。五角大楼没有具体透露试验的日期和地点，但指出莫斯科正在研发和试验几种类型的空间武器，包括激光武器。另外，美国太空司令部还报道了俄罗斯反卫星导弹在2020年4月的测试情况。然后，五角大楼明确指出，这种武器可以摧毁处于低地球轨道（高于地球表面的高度–160-2000公里）的卫星。7月份再次谈到俄罗斯在测试美国太空反卫星武器的问题。

在俄罗斯，美国的声明被称为宣传信息倾销。
