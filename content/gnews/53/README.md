1933年1月30日，在经历了多年的动荡，经济破产和共产主义运动后，纳粹党接管了德国的政权。希特勒正式就任德国总理，当天夜里纳粹党组织了两万五千人的火炬游行。他们绕着勃兰登堡门沿线的主干道转圈子，对外宣传称游行队伍有10万人。

支持希特勒上台的幕后势力和妥协者说：是我们雇佣了希特勒，兴登堡就是其中的一个，众所周知他很轻视希特勒。他们不知道希特勒有自己的计划。希特勒从政的最初动机是复仇，慢慢地形成了自己的政治理想。简而言之：

1.劣等种族应该被消灭，优秀的雅利安人种有权得到更多的生存空间。东方未实现工业化的土地是适合的目标。

2.一战后，德国经济的崩溃和人民的苦难是因为凡尔赛和约带来的，欧洲很多国家抢掠了德国。德国需要把它们重新抢回来，并向这些国家复仇，恢复德意志的荣耀。

1923年，希特勒在啤酒馆暴动失败后，被以叛国罪起诉，他在法庭上辩护称：如果有一个盗贼偷走你的东西，你把它再抢回来，难道你就也成为盗贼了吗？这就是希特勒的逻辑。

凡尔赛合约压迫着德国的经济，削弱了德国的主权。阿里萨斯洛林，萨尔地区被割让，莱茵兰被定为非军事化区，但泽被国际托管，大量的赔款，鲁尔区被抢走的工业设备，对所有爱国者来说，这都是伤痕。希特勒成功抓住了边界问题，就此扮演着救世主的角色。客观地说，凡尔赛和约多项条款的确是多国的趁火打劫，对德国的民生造成巨大的伤害，没有一个德国人会忘记内心的仇恨。他们在等待着报复的时刻。

**民族主义需要两个基石：自私与仇恨。这就是中共为什么要编各种假故事，煽动着中国人心中对过去历史的仇恨。**

希特勒和亲信幕僚制定了一个8年计划，在这8年里使德国全方位地做好战争准备。在1940年，最晚不迟于1941年发动战争，占领德意志民族所需要的生存空间，惩罚德国的宿敌。时间不能拖得更久，希特勒说：否则纳粹党的第一代领导人将会老去，精力能力将不足以领导这场伟大的斗争。

刚刚上任，希特勒就开始推行他的反犹运动。冲锋队组织了各种抵制犹太人商业活动的活动，其用意是引起德国人的注意，怂恿他们开始行动，并认为这是理所当然的。外面的世界震惊了，但经历过这些的人们，包括德国籍犹太人都认为这是党内极端分子所为，当局势安定些了，自然会停止。邪恶不在初始时被制止，代价必是千万倍。

新德国要逮捕所有的敌人：社会稳定的破坏者、政治异议者、不屈的记者和劣等人种。负责干这件事的就是冲锋队。在1934年，冲锋队扩充到了450万人，是原来人数的10倍。现在纳粹党合法地掌握了政权，从现在开始，冲锋队对法律不再有兴趣了。

他们建立了集中营，使用非常残酷的手段。他们给被关押者准备第一顿欢迎餐，在吃的东西里加入化学用的含有大量杂质的盐，如果不吃，就会挨一顿狠揍。如果吃下去，这种盐对胃有刺激，会反胃吐出来，这时候囚犯就会被逼着把所有吐出来的东西再吃进去。

这些只是开始，这些集中营后来就演变成了进行种族灭绝的死亡营，由党卫军接手。他们剪掉死者的头发，去编地毯、 脂肪用来做肥皂、人皮做灯罩、 尸体和骨架被火化做肥料。苏联军队解放某个集中营时，没有运走的人的头发7吨，鞋子两百万双…… 这是人类的行为吗？这种罪行就叫反人类罪。

**现在多国政府认定中国共产党在新疆犯下了反人类罪，他们一定看到了更多切实的证据，只是还未公开，几百万人被集体关押，杀人，器官买卖…… 真不知道还有多少令人发指的事情。**

纳粹还开展了一场文化革命运动，焚毁不符合纳粹意识形态的书籍，统一思想。书籍被扔进了火堆中，知识分子，作家，科学家纷纷逃亡，把自己的智慧献给了其他欧美国家。

希特勒的支持者主要是德国的中产阶级、破产的店主、失业的职员、工人和被工业化所淘汰的手工业者。希特勒把从犹太人那里抢来的资产和物品分给他们，并加大国家采购，采购用的钱自然也是抢来的。他还通过了保护农民土地权的法律，希特勒给了所有人工作和面包，纳粹党推行集体主义，社会主义，但资产家依然富有，他们一察觉有风吹草动时就开始资助纳粹党。看来，社会主义这个把戏，从纳粹党到现在的中国共产党，都没有什么变化，永远都是个骗人的幌子，借国家的名义把私人财产国有化，然后再把它放入握有权势者的腰包。

纳粹奉行的是操纵，而不是统治。1933年德国人民迎来了戈培尔铺天盖地的宣传，他们突然觉得自己象神一样高大，爱国热情，自私狭隘的民族自豪感充斥着德国的大街小巷。纳粹还在海外发动宣传和思想渗透，奥地利自不必说，他们在美国也建立了纳粹党，我们可以从历史镜头看见当时纳粹势力在美国有多么嚣张。

引导舆论当然也是必须的。在美国做生意的德国商人对媒体说：“在美国，有很多反对希特勒政府的报道，他们的大多数观点，无论从哪一方面看，都是匆忙的，有待论证的。我能在这里让今日的美国人意识到，这些是不真实的，没有根据的。我发现新德国在阿道夫.希特勒带领下，拥有了新的活力，他本人也赢得了民众的敬仰。新德国能够生存，因为我们有当今世界上最好的中央集权政府。“来自社会主义中国的人，是不是对这样的话很耳熟啊？这不就是现在的马云，吴征之流的论调吗？

希特勒在德国的支持率达到了90%，无论他到哪里，都可以自发地随随便便地聚集起十几万人夹道欢迎。但他依然感到威胁，他害怕国防军那些老派的保守的将领们，害怕党内的持不同信念的人，尤其是冲锋队全国领袖恩斯特.罗姆。终于在1934年6月30日长剑之夜，清洗掉了罗姆等一批纳粹党重要成员。

在希特勒的8年计划中，青少年要被重新组织起来加入希特勒青年团，灌输纪律和集体主义，野营训练，增加军事知识。等他们大了就会加入冲锋队，冲锋队则以准军事组织的形式进行训练，熟悉各种武器。在国家需要时，他们就会被征召入伍，这时军队生活对他们来说就会毫不陌生了。

新德国需要重新武装起来，海军得到了新式战舰，空军也在以航空运动联盟的名义孵化，陆军秘密扩充了两年，早就已经突破了凡尔赛和约规定的10万人。任一国家的军方都可以发现德国正在发生的一切，但他们什么也没有做。只有当希特勒在1935年发表了一个申明：和平时期国防军人数为50万时，各国也开始了军备竞赛。希特勒也在伪装自己，他保证德国无力进行军备竞赛，主动提出军备协议。英国第一个签署了海军力量备忘录。希特勒很肯定，这有利于他下一步的领土要求。“凡尔赛联盟“已经开始分解了。

1934年萨尔的一个煤矿区通过全民公投回归德国。紧临萨尔区的莱茵兰按照凡尔赛条约规定为非军事化区，法国是把这个地区作为它自己的战争缓冲地带来对待的。收回莱茵兰，就是对同盟国的直接挑衅。正因为莱茵兰重要的经济和战略位置，希特勒无论如何也得冒这个险。

1936年3月5日，希特勒确认了进军莱茵兰的命令，三支部队进入亚琛，特里尔和萨尔布吕肯的前沿基地。3月7日，德国军队跨过大桥开进莱茵兰地区，这是德国“英雄纪念日“的前一天，希特勒专门挑选了这一天。这是一个极大的冒险，军队也得到命令，如果英法有任何反应，军队立即渡河返回。当军队进行大冒险的时候，一波波的舆论宣传也开始了，通过宣传和外交途径传达到西欧。希特勒亲自在国会发表了和平演讲，表达自己的和平意愿。西欧被这种伪装的和平主义所蒙蔽。英国外交大臣艾登(Eden)也向内阁谏言，英国和法国最好与德国进行谈判。至少仍然有机会从德国得到回报。当时英国大约有50个反战组织。没有一个针对莱茵兰再军事化进行游说和示威，而是几次所谓的“和平”集会，旨在阻止政府采取反制措施。

关于如何以及在什么时间停止纳粹德国战争机器的问题，1936年对莱茵兰的军事化一再被视为重要的历史事件。 1947年法国对此做了专项调查，在其最终报告中指出，1936年，由于当时法国巨大的军事优势，无疑可以立即进行反占领。德国的抵抗能力是不足为虑的，法国军队完全有能力推进到德国的其他核心地区。如果真的是这样，那也就没有二战了。捷克总统米洛斯•泽曼（Milos Zeman）在2015年1月在布拉格举行的大屠杀纪念仪式上说：如果法国和英国在1936年进行军事干预以反对对莱茵兰的占领，那么就不会发生大屠杀。

历史总在重演，中共对香港的所作所为像极了莱茵兰事件。同样的大环境，欧美完全有能力直接切断中共的经济血脉，但却像当时一样沉浸在利益和私欲中，为了被共产党骗进去的几个钱，为了中共许诺的一点小利，除了喊几声外，什么也没有做。难道我们要让二战重演？超限战病毒不是已经放出来了吗？中国共产党已经开始了，难道还要再等着中共进行下一步吗？中国共产党就是当代的纳粹!

希特勒的下一步也开始了。两年后，纳粹在奥地利的布局已经完成，英法对奥地利的支持也被成功分化。很多奥地利人在纳粹“民族与统一“的宣传下，迷失了自我。“一个民族，一个国家，一个元首“就是当时的宣传口号。

希特勒通过各种军事威胁和心理攻势逼迫奥地利许士尼格政府把权力移交给自己在奥地利培植起来的塞斯.英夸特内阁。这场大戏堪称心理战的典范。许士尼格则以举行奥地利全民公投作为抵抗，希特勒不能允许这样的事发生。1938年3月11日，在奥地利的纳粹党员冲进奥地利总理府，占领楼梯，走廊和办公室。在各种威胁下，许士尼格在当天下午同意取消公投，并通过电台宣布辞职。3月12日，早已安排好的纳粹官员接管了各级政权。德军顺利占领奥地利。3月13日纳粹党通过了德国-奥地利统一法案。

中共的下一步是谁呢？2020年12月17日中共国防时报写到:“ 今年以来，台美勾连加深，售武、派高官访台，严重影响了两岸的和谐稳定……大陆民间要求增订《国家统一法》的呼声也越来越高, 国家统一法将规定统一的过程和方式“。

集权政府只要尝到一次甜头，欲望就会无休止的膨胀。1938年4月21日，希特勒要求国防军最高统帅部研究入侵捷克的可能性，代号绿色方案。此时的德国军队已经今非昔比，由原来的10个师扩展到了45个师。德国空军也装备了1000多架最新式的战斗机。从1938年5月开始，纳粹党在捷克住有德国移民的苏台德区故意制造事端，要求得到苏台德的领土。

最终，英法为了自己的私利出卖了捷克斯洛伐克，这私利里面也包括：希望通过雇佣希特勒来对付东方的共产主义。他们天真地相信能与纳粹达成一个“和平与光榮的解決方案”，那就是《慕尼黑协议》，还相信希特勒会遵守它。那当然是不可能的。德国军队在拿下苏台德后并没有停下脚步，他们直接向捷克斯洛伐克的内陆进军，各路部队于同一天抵达了布拉格，没有任何抵抗，英法也没有任何举措。

至此，大战前的准备工作完成了。

战争往往是因为经济压力和资源冲突造成的。二战前，纳粹靠的是掠夺财富和大规模国家投资发展经济。这样是不能持续的，但这不重要，因为它的目的是战争，抢回“应该属于德国的财富“。换句话说，接下去的发展之路依然是掠夺，剥削殖民地和大规模国家投资。
