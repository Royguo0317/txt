**西班牙2021年3月3日电/西喜社——**

据ABC新闻3月2日报道，2月糟糕的就业数据让ERTE（国家负责的工作保障计划，即轮岗，减少工作时间，拿70%工资）的就业人数出现新的增长。具体来说，约90万人，短短28天增加了16万多人。

旅游业是这次危机的重灾区。在受ERTE影响的总人数中，三分之二是与旅游业有关的雇员。共计594,317人。

酒店业占了ERTE的大部分。餐饮服务和住宿服务活动影响了近45万人。此外，在工作岗位方面，根据社会保障部的数据，与2020年2月相比减少了近30万个。

另一个在ERTE中积累了大量人员的行业是零售业。有98433名工人。即使是1、2月份的销售旺季，也没有丝毫好转。

航空业是另一个重灾区，流动性的限制给航空公司带来了巨大的损失，有超过40%的相关航空公司受到影响。旅行社的从业人员也很突出，2月ERTE上共有21611人。
