2021年4月14日，拜登政府驻联合国大使琳达·托马斯-格林菲尔德在[全国行动网络](https://usun.usmission.gov/remarks-by-ambassador-linda-thomas-greenfield-at-the-30th-annual-summit-of-the-national-action-network/)（National Action Network）第30届年度峰会上致辞时表示，“**美国在人权理事会上教训其他国家之前，必须首先反思自己猖獗的白人至上主义**”。还称种族歧视在建国之初就已渗入美国的根基之中。
![]()![](https://gnews.org/wp-content/uploads/2021/04/MOS2054-【热点播报】拜登政府的驻联合国大使直言美国是种族主义国家-1.png)美国驻联合国大使琳达·托马斯-格林菲尔德（图片来自美国之音中文网）
正因种族灭绝问题而被全世界围剿的中共和正在监禁反对派领袖阿列克谢·纳瓦尼（Alexei Navalny）的俄罗斯，听到这话恐怕会禁不住笑出声来。他们在自己的媒体上骂美国，肯定没有美国驻联合国大使亲自出来骂美国更有“说服力”。 琳达·托马斯-格林菲尔德还强调，“**当我们在全球范围内提出公平和正义的问题时，我们必须谦虚**”，“我们必须承认，我们是一个不完美的联盟，从一开始就是这样。”“**我亲眼目睹了奴隶制的原罪如何将白人至上主义编织到我们的建国文献和原则之中**。”这位美国驻联合国大使的话，就像是从中共嘴里说出来的一样，不免让人惊讶。

全国行动网络由黑人民权运动领袖阿尔·夏普顿（Alfred Charles Sharpton, Jr.）牧师创立于1991年。批评者认为，夏普顿是煽动黑人种族主义的政治激进派，美国近年来种族纷争不断恶化，有他一份“功劳”。哈佛大学社会学家奥兰多·帕特森（Orlando Patterson）更称他为“种族纵火犯”（racial arsonist）。值得注意的是，夏普顿是美国前总统奥巴马(Barack Obama)的重要支持者，奥巴马曾在2014年全国行动网络(National Action Network)年度大会上发表讲话。

![]()![](https://gnews.org/wp-content/uploads/2021/04/MOS2054-【热点播报】拜登政府的驻联合国大使直言美国是种族主义国家-2.png)阿尔·夏普顿（网络图片）

种族主义问题是拜登政府的重要施政议题之一。拜登总统和哈里斯副总统都誓言和美国系统存在的种族主义做斗争。

然而，不幸的是，借种族主义之名攻击西方是中共惯用的伎俩，也是中共能够一举多得的法宝。在美国国内煽动种族仇恨，制造社会分裂，可以达到搞乱美国的目的。在国际上打出种族牌，可以拉拢亚非拉地区的有色人种国家，建立反对“白人”的统一战线，推行其邪恶的“灭白计划”。同时还可以将西方对中共的谴责和制裁说成是出于种族主义偏见，并给西方民主国家戴上“虚伪”、“双标”的帽子。反过来还能借机将中共和中国人捆绑在一起，煽动国内的民族主义和仇美情绪。拜登政府在种族问题上的偏执和激进，无异于给中共递刀子，把美国最大的弱点暴露给了中共。假如美国无法在病毒问题上迅速达到击垮甚至消灭中共的目的，中共一定还会借种族问题为美国制造更多麻烦。
