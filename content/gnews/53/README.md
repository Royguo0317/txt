# 重要事件

- **[04.08 The National Pulse：《纽约时报》的几名员工承认，曾为中共媒体《中国日报》工作，包括该刊现任摄影总监](https://thenationalpulse.com/news/nyt-writes-former-china-daily-director/)**


此消息的出现，是正因《纽约时报》屏气支持大企业反对乔治亚州的投票法。不过，《纽约时报》似乎不太在意雇佣种族灭绝的中共附庸。

现任《纽约时报》摄影总监乔纳-凯塞尔（Jonah Kessel），在2009年7月至2010年11月担任《中国日报》创意总监，之后他以中国摄影师和摄影家的身份离职，其客户包括中华人民共和国新闻部。

乔纳-凯塞尔称自己是在 “重新设计”《中国日报》，他对这一工作感到 “兴奋”，并夸耀《经济学人》等出版物如何炒作他的重新设计。

在为《中国日报》工作期间，乔纳-凯塞尔多次在推特上说自己是在为中国共产党 “工作 “并 “拿钱”。

2010年7月，他在推特上写道：”有时候为中国共产党工作也有好处：）”。

而在2009年11月，他在推特上写道：”当有人问起你的工作场所时，你脑海中浮现的第一个词是’和谐’时，你就知道你在为中共国工作。

乔纳-凯塞尔还称赞中共国的 “国庆节 “假期是为了纪念中国共产党接管中国，”非常酷”。

而即使在离开《中国日报》后，乔纳-凯塞尔也承认自己在担任个人电影摄影业务创意总监的同时，仍在 “为中国共产党工作”。

现任《纽约时报》参谋编辑和设计师的迪亚穆德-麦克德莫特（Diarmuid McDermott），此前也在《中国日报》担任同样的职务。

从2012年到2020年，他在这家国营媒体的 “亚洲周刊 “刊物上工作，他在简历中描述的角色涉及 “文字编辑；改写原始稿件；设计版面和图形；采购新闻报道和图片；开发和维护多个平台的网络存在；为整个地区的印刷商输出页面”。他的个人网站也在宣传他为《中国日报》所做的工作。

另一位《纽约时报》记者亚历克斯-马歇尔(Alex Marshall)根据他的领英（LinkedIn）资料，在2003年至2004年期间担任《中国日报》编辑。报道文化的历克斯-马歇尔称自己是 “中共国的辩护人”，并在推特（Twitter）上称赞习近平的讲话。

- [**04.10 Breit Bart: 亚利桑那州州长杜西签署法案，禁止私人资助选举管理。**](https://www.breitbart.com/politics/2021/04/10/az-gov-ducey-signs-bill-into-law-that-prohibits-private-funding-of-election-administration/)


亚利桑那州州长杜西（Doug Ducey）周五签署了一项法案，禁止在该州进行私人资助选举管理。

亚利桑那州现在是全国第一个明确禁止私人资助选举管理的州，在2020年总统大选中，Facebook首席执行官兼创始人马克-扎克伯格向两个非营利组织捐款4.19亿美元，这两个非营利组织又向州、县和地方选举管理提供资金，批评者称这对民主党人不公平地有利。

美联社报道称，杜西在一封信中写道：”随着公众对我们选举的信心岌岌可危，很明显，我们的选举必须是纯净的、无可指责的–也是政府的唯一权限。”他在信中表达了签署该法案成为法律的理由。正如Breitbart新闻在12月报道的那样。

托马斯-莫尔协会的阿米斯塔德项目在周三的新闻发布会上发布的一份报告称，Facebook创始人马克-扎克伯格和他的妻子在2020年大选期间向非营利组织捐款4.195亿美元–其中3.5亿美元捐给了技术与公民生活中心（CTCL）的 “安全选举 “项目，另外6950万美元捐给了选举创新与研究中心–“代表某位候选人和政党对2020年总统大选进行了不当影响”。

例如，在2020年大选中该州唯一翻盘的马里科帕县，特朗普总统的总票数增加了24.8万多票，却在该县输给了拜登。2020年，拜登比希拉里-克林顿2016年在该县的总得票数提高了33.7万多票。这近9万票的差距不能用登记增加来解释。虽然自2016年大选以来，马里科帕县民主党选民登记的增长幅度超过了共和党登记的增长幅度，但净增加的票数不到5万张，登记的共和党人仍然比登记的民主党人多出10多万票。在2020年支持拜登的县里，扎克巴克似乎帮助提高了民主党的投票率。其他几个州的立法机构正在考虑禁止私人资助选举管理的法案。

佐治亚州州长坎普3月签署的《选举诚信法》包括一项旨在禁止私人资助选举管理的条款，但阿米斯塔德项目的菲利普-克莱恩认为，该法律可能存在漏洞。

- [**04.09 OANN：美国参议院出现全面抗衡中共国的细节。**](https://www.oann.com/u-s-senate-moves-ahead-with-sweeping-effort-to-counter-china/)


美国参议院外交关系委员会的领导人周四提出立法，通过促进人权、提供安全援助和投资打击虚假信息来增强国家反击中国不断扩大的全球影响力的能力。

这项名为 “2021年战略竞争法案 “的措施草案规定了反制北京的外交和战略举措，反映了国会民主党和共和党人在与中国打交道方面的强硬情绪。

这份长达280页的法案不仅涉及与中共国的经济竞争，还涉及人道主义和民主价值，如对少数民族穆斯林维吾尔人的待遇实施制裁，支持香港的民主。

它强调需要 “优先考虑实现美国在印太地区政治目标所需的军事投资”。它呼吁为此支出，称国会必须确保联邦预算与与中国竞争的战略要务 “适当一致”。

该法案建议在2022至2026财年为该地区提供总计6.55亿美元的外国军事资助，同期为 “印太海上安全倡议 “及相关项目提供总计4.5亿美元的资助。

该法案将扩大美国外国投资委员会（CFIUS）的范围，该委员会负责审查金融交易是否存在潜在的国家安全风险。然而，与该法案的许多条款一样，这一条款可能会在委员会和参议院全体会议上进行修改。

立法草案呼吁加强与台湾的伙伴关系，称这个自治岛屿是 “美国印太战略的重要组成部分”，并称不应该限制美国官员与台湾同行的互动。中共国认为台湾是一个分离的省份。

### 04.10 环球时报：制裁中国超算者终将失算

美国商务部8日发布公告，将包括天津飞腾、上海高性能集成电路设计中心在内的7家超算机构列入“实体清单”，美方的公开借口是这些实体涉嫌“破坏军事现代化的稳定”。

由于西方在超算技术方面一直保持制裁，在超算的核心技术上，中国具有完全自主研发能力。超算硬件主要由运算系统、互联通信系统、存储系统、维护监控系统、电源系统、冷却系统等组成，其中最关键的是运算系统、互联通信系统，这两个系统早已实现了国产化，“天河2号”的互联通信网络，“神威太湖之光”的芯片和互联都是由国内单位自主研发。就性能来看，纯国产的超算也达到世界顶尖水平，“神威太湖之光”超算不仅多次位居TOP500榜首，还多次斩获戈登贝尔奖，证明国产超算不仅能够实现性能顶尖，还能在应用上达到国际先进水平。

以史为鉴，美国制裁只能延缓中国超算技术的进步速度，并不能彻底扼杀中国超算，反而会使中国超算研发机构放弃幻想，彻底走上独立自主道路。2015年美国禁止英特尔向四家中国超算中心销售Xeon Phi计算卡，只是对“天河2号”升级进度造成了影响，“天河2号”使用矩阵2000计算卡替换掉了Xeon Phi，性能反而获得了提升。

面对美国的严厉制裁，最好的应对办法就是加快构建国内大循环。中国市场非常大，客户对超算的需求是现实且迫切的，中国超算研发能力不亚于美国，完全可以通过内循环应对制裁。虽然制裁短期会延缓中国超算进步速度，对中国超算研发单位造成一些影响，但从长期看，美国此举纯属搬起石头砸自己的脚，将刺激中国进一步加强自主研发力度，推进科技自立自强。

***简评：**感觉是一篇给粉红和上级领导的定心丸，意思就是让他们认为区区制裁不会受多大影响，反而能让中共国因此自强不息自主研发，但是，也就真的只能忽悠一下脑残和小学博士水平以下的人群。本应，超级计算机可以帮助人类解决很多科学上的问题，但中共通过超级计算研制生化武器，殃及全人类，现在被美国制裁，到底是谁把自己脚用石头砸了。*
