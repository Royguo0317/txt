#### 作者：Ru Wei
![]()![](https://gnews.org/wp-content/uploads/2021/04/英国国际学校.png)封面来源：网络图片
英国教育专家及国会负责教育事业的议员表示，鉴于英国各级教育机构在中共国开办的50所国际学校，在教学上面临官方“全国统一必修课程”的要求和限制，这些国际学校正打算迁往越南、埃及等国另行开办。

英国《星期日泰晤士报》（The Sunday Times）4月17日针对国际学校的迁移计划做出了专门的报导。中共国近年来开始对教育严加管控，中共国教育部今年3月间向包括双语私立学校在内的所有学校发出新指示，要求各校只要有中共国学生，就须安排“全国统一的必修课程”。
