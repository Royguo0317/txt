![]()![](https://gnews.org/wp-content/uploads/2021/04/image-109.png)图片来源：推特
在经历了一年多的中共病毒大流行后，每一个人都希望恢复正常的生活，于是很多人错误认为接种疫苗是一种可行的预防方案，而各国政府出于各种目的也在大力推广和宣传疫苗接种。截止到目前为止，接种中共疫苗的人次已突破一个亿，相当于每13个人里面就有1个打了疫苗。美国接种疫苗的人数也超过3000万了，相当于每10个人里面就有1个人接种，然而即使如此，疫情依然还在蔓延。

有很多国家的总统和政要官员都声称接种了疫苗，比如美国总统拜登、俄罗斯总统普京（普京并没有允许发布自己接种的视频和照片）、印度尼西亚总统佐科、塞舌尔总统拉姆卡拉旺、土耳其总统埃尔多安、法国卫生部长韦兰……只是不知这些政客是否只是为了作秀假装接种来推广疫苗。

而真正接种了疫苗的一位知名政要，是85岁的达赖喇嘛尊者。2021年3月6日，这位西藏人民至高无上的精神领袖，在印度达兰萨拉佐纳尔公立医院首次接种了Covishield疫苗。Covishield疫是由牛津大学（Oxford）与阿斯利康制药公司(AstraZeneca)共同研发的。尊者打完疫苗后说：“根据我可信的朋友们的说法，其中包含医生，他们一致建议要去接种新冠疫苗，因为新冠疫苗对预防严重恶化的帮助极大，这点很好。”只是无人知道尊者这些“可信的朋友”到底是谁，他们到底是想帮他还是在害他。
![]()![](https://gnews.org/wp-content/uploads/2021/04/image-110.png)图片来源：达赖喇嘛办公室


受中共病毒大流行的影响，很多明星无法拍戏和演出，过去一年多娱乐圈也沉寂了不少。随着时间的推移，各国开始大力推广接种疫苗，很多明星也在社交媒体上晒出自己接种疫苗的照片，本篇文章将对这些明星进行盘点：

一、林青霞接种中共国的科兴疫苗

2021年3月5日凌晨，林青霞通过微博透露自己已经接种了国产的科兴疫苗，并称：“感觉很好，没什么反应，心情愉快。”以林大美女的身份，即使接种疫苗也不应该选择不靠谱的科兴疫苗，是否受到了中共的威胁，不得而知。
![]()![](https://gnews.org/wp-content/uploads/2021/04/image-111.png)图片来源：微博


**二、欧阳震华接种了中共国的科兴疫苗**

2021年3月18日，TVB的收视福星欧阳震华（Bobby）在微博说他已经打了科兴疫苗。想当年笔者经常在TVB电视剧里面看到他，没想到他也迫不及待地接种了疫苗。
![]()![](https://gnews.org/wp-content/uploads/2021/04/image-112.png)图片来源：微博


**三、施瓦辛格接种了新冠疫苗**

2021年1月21日，饰演《终结者》的现年73岁的阿诺·施瓦辛格在推特上发布了在车上接种新冠疫苗的视频并配文：“今天是很棒的一天，从未对排队感到如此开心。如果你符合资格，请和我一起去申请疫苗。”最后他还说了一句终结者的经典台词——come with me if you want to live（想活命就跟我一起）。

**五、休·杰克曼接种新冠疫苗**

此外还有很多明星，比如[**小甜甜**](https://stars.udn.com/search/tagging/1022/%E5%B0%8F%E7%94%9C%E7%94%9C)布兰妮带着男友山姆一起去打了疫苗，还特意拍了短片；还有塞缪尔·杰克逊，安东尼·丹尼尔斯等等，在此不一一列举。

这些明星纷纷接种疫苗，并在社交媒体上传播，给很多普通民众造成了疫苗接种对预防中共病毒有效的错觉。119路德社节目爆出中共病毒真相的病毒学博士闫丽梦，一直在苦口婆心的劝说人们不要接种疫苗，因为疫苗既不会保护自己免于再次感染，也不能预防变异的新冠病毒，更不能达到群体免疫效果。

已经陆续有报道，一些接种了疫苗的病人全身红肿，甚至有人当场去世，以及有人接种了疫苗后继续感染中共病毒的各种新闻，接种疫苗有百害而无一益。可是全世界的民众，甚至各大明星，有越来越多的人在政府的误导宣传下接种了疫苗，这说明还有大量的人并不知道中共病毒的真相，不知道中共的邪恶。中共控制了媒体，媒体控制了百姓。爆料革命坚持“以毒灭共”，只有彻底揭开病毒的真相，才能唤醒更多的人。新中国联邦，就像茫茫戈壁沙漠中的一小片绿洲，只有我们才能给世界带来希望，只有我们才能拯救世界。
