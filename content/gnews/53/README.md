**西班牙2021年3月11日电/西喜社——**

据ABC新闻3月11日报道，丹麦卫生官员周四宣布，由于担心接种疫苗的人出现血栓，丹麦已暂停使用阿斯利康公司的中共病毒疫苗作为预防措施，直到进一步通知。包括丹麦，目前已有八个欧洲国家做出相同决定

国家卫生局表示，此次停用疫苗是因为 “几份关于接种了阿斯利康中共病毒疫苗的人出现严重血栓病例的报告 “。但他们澄清，”目前还不能断定疫苗与血栓之间有联系”。

此前，奥地利已于周三撤销了一批阿斯利康公司的疫苗使用权，原因是一名奥地利人在接种疫苗10天后被诊断为多发性血栓形成（血管内形成血栓）并死亡。另一名奥地利人在接种疫苗后因肺栓塞（肺部动脉堵塞）住院，但目前已康复。此外，本批次疫苗还收到另外两例血栓事件的报告。

欧洲药品管理局(EMA)正在研究奥地利在注射阿斯利康公司的中共病毒疫苗后报告的血栓事件，但到目前为止表示：“没有迹象表明接种疫苗导致了这些情况，目前这些情况没有被列为疫苗的副作用。”

欧洲药品管理局(EMA)辩解说，根据 目前掌握的信息，接种疫苗的人发生血栓栓塞事件的数量 不高于普通人群中观察到的数量。截至3月9日，欧洲经济区300万接种阿斯利康疫苗的人群中，已有22例血栓栓塞事件的报告。

西班牙卫生部长卡罗莱纳-达里亚斯(Carolina Darias)说，欧洲监管机构 现在正在开会 解决这个问题。据我所知，西班牙还没有发生过这类血栓事件。到目前为止，疫苗与血栓之间的因果关系尚未得到证实。他在接受媒体拉塞克斯塔（La Sexta）采访时说：“我们的情况很好，请大家放心。”

这些事件发生后，丹麦卫生当局周四宣布，他们将暂停使用阿斯利康的疫苗，作为奥地利病例发生后的预防措施，尽管丹麦境内还没有发生任何此类案例。

继丹麦之后，挪威、冰岛、奥地利、爱沙尼亚、立陶宛、拉脱维亚、卢森堡等国加入暂停接种阿斯利康疫苗的行列。同时，意大利在西西里岛发生两人死亡事件，一名43岁的士兵和一名50岁的警察，都是在西西里岛死亡，意大利已经阻止使用一批疫苗。
