## 日本前首相安倍晋三：日本已成为美中两国对立的最前线
` 喜马拉雅德国慕尼黑感恩农场` [轉載自GNews](https://gnews.org/zh-hans/1029472/)

收集：文W；编撰：WLQF

据共同社3月27日报道，日本前首相安倍晋三27日在新潟市举行的自民党新潟县支部联合会会议上发表了演讲。**在谈及中美对立时，安倍晋三表示，包括日本在内的亚洲地区已成为美中两国对立的最前线。**

![]()![](https://gnews.org/wp-content/uploads/2021/03/截屏2021-03-28-下午8.44.45.png)日本前首相安倍晋三，图片来自网络

安倍晋三在演讲中谈到关于日本对华政策时表示：“日本需要抱着印度太平洋地区已成为前线的认识和心理准备，致力于外交与安全保障政策。”**此外，安倍晋三还谈及了中国军事力量的崛起、以及在东海和南海单方面改变现状的尝试。**他说：“在这种情况下，可以说最前线已经转移到印度洋太平洋地区，就其本人认识而言，太平洋地区，包括日本在内已成为中美对立的最前线，因此我们有必要在这种认识的基础上，抱有充分的心理准备，坚实地提炼我们的外交安全保障政策。

安倍晋三谈到美国总统拜登在4月上旬拟与日本首相菅义伟会谈时指出：“这将是拜登首次与日本政府之间面对面的首脑磋商。**安倍表示：****“****美国外交和安保战略上的重要地区已经转移到了印度太平洋。这就表明日美安保条约变得真正重要了起来。****”**

![]()![](https://gnews.org/wp-content/uploads/2021/03/截屏2021-03-28-下午8.47.40.png)美利坚合众国与日本国之间互相合作与安全保障条约正本，图片来自网络

在谈到日本自卫队时，安倍晋三说，非常遗憾在任职期间没有完成《日本宪法第9条》的宪法修改事宜，目前自卫队还处于违反宪法的状态。**但安倍晋三在谈及****9****月份将要举行的下届众院选举时安倍强调：****“****菅义伟政权正在竭尽全力努力工作，将与菅首相一起，无论如何都要获胜，必须建立更稳定的政权并强有力地推进政策，全面推进国家的发展。**

**日本自卫队知识拓展：**

![]()![](https://gnews.org/wp-content/uploads/2021/03/截屏2021-03-28-下午8.43.50.png)日本自卫队，图片来自网络

自卫队是日本在第二次世界大战后至今的国家防务部队，成立于1954年7月1日。由陆上自卫队、海上自卫队、航空自卫队等三个部门所组成，全体武职人员称“自卫官”，共25万人，全由防卫省统筹管辖，内阁总理大臣统帅。**根据《日本宪法第****9****条》，日本放弃与他国以军事手段解决争端的权力，再加上二战之后日本禁止征召军队，因此自卫队在名义上不是军事组织，但在实际上的功能等同其他国家的军队，只是在法理与预算上受限。**

自卫队的主要功能是要维持日本的自我防卫能力，而踏进21世纪后，自卫队对于海外维和任务的参与愈来愈积极，除了参与了2002年阿富汗战争后的维和任务外，亦在2004年派遣“自卫队伊拉克复兴支援群”协助美伊战争后的重建与安全维护。**因应朝鲜核武问题、中国军事威胁等周边局势，日本国内存有修改日本宪法第****9****条、并将自卫队升格为****“****国防军****”****的声浪；日本政府也透过制定****2016****年施行和平安全法制，成功为自卫队施行集体自卫权解套。**
