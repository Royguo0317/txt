大纪元报道，澳大利亚政府承诺在太平洋地区的基础设施项目上投入3亿澳元(2.32亿美元)，以此打击北京在该地区掠夺性贷款的行为。此举是澳大利亚加强太平洋努力的一部分，包括为升级电信、交通、医疗设施和电力项目提供大量贷款和赠款。

澳启动该项目的目的是抵抗中共的“一带一路”倡议(BRI)和亚洲基础设施投资银行(AIIB，简称亚投行)提供的掠夺性贷款，为太平洋地区提供替代北京资金的其他选择。

过去一年，亚投行对太平洋地区的贷款有所增加，因为各国都在寻求摆脱中共病毒疫情造成的经济低迷的方法。新西兰惠灵顿梅西大学(Massey University)国防与安全研究中心(Centre for Defence and Security Studies)高级讲师安娜·鲍尔斯(Anna Powles)称，来自亚投行的贷款可能会给参与国制造债务陷阱，就像“一带一路”倡议的一样。她指出，“如果亚投行成为太平洋地区的主要贷款机构，而该地区的经济复苏是由中共的贷款推动的，那么有理由担心经济依赖可能被利用。”

亚投行是由中共倡议成立的，中共是亚投行第一大股东且总部设在北京。亚投行倡议的主要目的是为了增强中共在亚洲的霸权地位，实现人民币国际化 、推广“一带一路”计划及中共国内过剩产能输出。因此中共对亚投行有着绝对的影响力。

中共政府通过“一带一路”贷款，越来越多地利用债务陷阱外交，在世界各地获得立足点。这些贷款通常还伴随着高利率和短期还款，让一些国家背上了无法偿还的贷款的负担，当一些国家无力偿还债务时，北京要求它们做出重大让步。

例如，斯里兰卡因无力偿还贷款，不得不将其具有战略意义的汉班托塔港70%的控制权移交给一家中共国有企业，租期为99年。与此同时，吉布提最终将其主要港口移交给了中共，中共人民解放军在那里建立了永久性的军事基地。

这是澳大利亚的担忧和做出抵抗中共贷款陷阱努力的原因。因为这意味着中共通过贷款陷阱可能在太平洋地区建立军事设施。若是如此，太平洋基地可以为中共“胁迫澳大利亚”和反击美国的行动提供立足点，中共将有能力威胁到周边岛屿国家,包括澳大利亚,在军事上断绝美国和日本的重要的补给线，这将导致地区安全危机。
