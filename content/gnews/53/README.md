**西班牙2021年3月16日电/西喜社——**

据ABC新闻3月16日报道，英国首相鲍里斯-约翰逊（Boris Johnson）于周二宣布，英国将把获准储存的核弹头数量增加 “40%以上”，根据报告，核弹头的数量将增加到260枚，尽管英国曾承诺最多能达到180枚。

这将是冷战以来，该国首次增加核武库。首相认为，这一决定是由于该国面临的 “技术和理论威胁 “越来越大，这份约百页的文件列举了分别代表俄罗斯和中共国的 “主动威胁 “和 “系统性挑战 “。此外，它还警告说，一些恐怖团体 “到2030年将成功地发动化学、生物、辐射或核攻击 “的 “真实可能性”。这项有争议的决定将耗资约100亿英镑，结束了自苏联解体以来30年的渐进式裁军。

该报告是约翰逊关于英国在离开欧洲联盟后在全球局势中作用的战略的一部分。并指出 “北约防务分配的最低限度的、可信的和独立的核威慑力量，对于确保我们和我们盟友的安全仍然至关重要”。

据斯德哥尔摩国际和平研究所估计，截至2020年1月，联合王国拥有215枚核弹头，这些核弹头可以从专门为携带核弹头而设计的潜艇上发射。适合于此的四艘潜艇在不进行巡逻时，通常都停泊在苏格兰沿海。

国际废除核武器运动（ICAN）主任比阿特丽斯-菲恩（Beatrice Fihn）在一份声明中批评了这一决定，并称其为 “世界舞台上的巨大挑衅”。他说：”英国决定在大流行病中增加大规模杀伤性武器库是不负责任的、危险的，违反了国际法。他还说：“当英国人民努力应对大流行病、经济危机、暴力侵害妇女和种族主义时，政府却选择增加世界上的不安全感和威胁。”

国际废除核武器运动认为，”核武器在英国不受欢迎，大多数公众、许多城市和议员都在呼吁政府加入《禁止核武器条约》”，所以英国显然是 “走错了方向”。此外，该组织还透露，”英国目前正在建造新的核能力潜艇，以取代现有的舰队”，”成本可能高达410亿英镑，不过包括所有相关费用在内，”核裁军运动 “认为核升级费用为2050亿英镑”。
