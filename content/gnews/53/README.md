最近美国主流左派媒体在失掉川普总统这个“靶子”之后变得集体“失常”了。比如原先非常亲共的媒体，突然良心发现，开始揭露中共的邪恶，追查病毒的真相了；原来为民主党保驾护航的媒体，开始抱怨拜登未兑现继续与中共勾兑的承诺。
![]()![](https://gnews.org/wp-content/uploads/2021/03/Screen-Shot-2021-03-26-at-3.38.38-PM-1024x682-1.png)
一例是左媒《CNN》首席医学记者Sanjay Gupta对前CDC主任罗伯特-雷德菲尔德博士的采访。在这个重磅采访中，前CDC主任罗伯特-雷德菲尔德博士说中共病毒“最有可能来自实验室”。并且他人为，病毒自然起源说“在生物学上是说不通的，” 他相信这次疫情比人们知道的要早几个月开始，而且它不是起源于海鲜市场，而是起源于实验室里面。

《路德时评》节目对这个重磅采访做了[详细解读](https://www.youtube.com/watch?v=rcEzcnvouFY)。

另外一例是《纽约时报中文网》发表的一篇文章，题目为《[拜登先生，别再对中国“放狠话”了](https://cn.nytimes.com/opinion/20210322/us-china-biden/?utm_source=tw-nytimeschinese&amp;utm_medium=social&amp;utm_campaign=cur)》。
![]()![](https://gnews.org/wp-content/uploads/2021/03/unnamed-2-1.jpg)
文章作者是[张彦（Ian Johnson）](http://www.ian-johnson.com/uploads/pdf/Ian%20Johnson%20Resume.pdf)，美国记者，过去30年他有近一半时间是呆在中国。现在仍然是《纽约时报》驻中国记者。还兼职在北京中心某大学教课。从他的经历来看，我们不难猜出他与中共的关系，和作为左派媒体一贯的为中共说话的立场。

文章一开始就既捧拜登，又黑川普总统，同时也流露出了对拜登对华政策的不满：

***自将近两个月前就职以来，拜登政府马不停蹄地改革和重新审视几乎所有关键问题领域，只有一个除外：从特朗普政府继承的混乱且不一致的对华政策。***

对于阿拉斯加双方外交会谈，他刻意淡化中共战狼外交的失态，对于双方的衝突各大五十大板：

***这些刺耳的对话只会加剧世界上两个最强大国家之间的关係的危险恶化。双方似乎都陷入了必须看起来、听起来都强硬的陷阱。***

接着他开始抱怨拜登，说拜登在上任之初说中美关係是“极端竞争”而不是冲突。而布林肯在东京也说，与中国的关系是非常复杂的：有敌对的方面，有竞争的方面，有合作的方面。

令他失望的是，“***迄今为止，政府的行动很大程度上还是遵循着川普的做法。***”

他甚至反对因参与镇压香港的政治自由而对24名中共官员实施制裁，和布林肯将台湾称为“国家”-他认为这违背了美国历届政府都遵循的“一个中国的”政策-只是在川普总统离开白宫前才将这个称谓改了，而拜登政府却照单全收的继承了川普总统的政策。

而他下面为拜登政府的对华政策建议，可以称得上直接替中共喊话了：

第一，拜登政府应重启奖学金计划，川普总统取消改计划，他认为损害了美国培养新一代学者和分析师的能力。
第二，美国政府应停止诋毁孔子学院，称其为邪恶的宣传机器。
第三，拜登政府应允许被川普总统驱逐的一些中国记者回到美国——前提是北京也要同意再次让美国新闻机构认可的新闻工作者回到中国，并承诺不骚扰他们。
第四，美国政府应取消对前往美国的中共党员签证限制。
第五，应邀请中国重新开放休斯敦领事馆，作为交换，中国政府会允许美国重新开放北京出于报复而关闭的成都领事馆。

这五条，如果出自中共喉舌《环球时报》主编胡锡进之口，也不会觉得意外！

正如《路德时评》所分析的那样，中共苦心孤诣地把川普总统搞下去，换上民主党上台，以为就可以把美国搞定了，其实更是打错如意算盘。从目前来看，在灭共这件事上，拜登政府是完全继承了川普总统既定的政策，甚至还走的更远！而从这片报导里， 我们也看到某些亲共的左派媒体的“失常”，他们尽管和中共沆瀣一气，但在面对拜登政府灭共举措和决心时，却显得无可奈何，无病呻吟。中共必灭，是大趋势，也是历史的必然。
