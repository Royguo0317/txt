# **美日称中共国为国际秩序的威胁**

*在新政府的第一次2+2会谈中，双方承诺加强联盟合作，以应对双方共同关注的中共国“侵略行动”。*

[!\[\]()!\[\](https://spark.adobe.com/page/qi1mmtiRoGf6f/images/9ae582e0-7b3c-4a49-9688-c881f5459701.jpg?asset_id=40a850a4-661b-4349-ad8e-183962f48bb4&amp;img_etag=%2229e15c8f6381914196d137b3e66f4454%22&amp;size=2560)](https://spark.adobe.com/page/qi1mmtiRoGf6f/images/9ae582e0-7b3c-4a49-9688-c881f5459701.jpg?asset_id=40a850a4-661b-4349-ad8e-183962f48bb4&amp;img_etag=%2229e15c8f6381914196d137b3e66f4454%22&amp;size=1024)2021年3月16日，国务卿安东尼·布林肯（Antony J. Blinken）和美国国防部长劳埃德·奥斯丁（Lloyd Austin）在日本东京会见了日本首相菅义伟。 Credit: State Department photo by Ron Przysucha 图片来源：国务院摄影：罗恩·普齐苏卡（Ron Przysucha）

日本和美国的外交和国防部长在东京结束了备受瞩目的“2+2”对话，主要侧重于通过“坚定不移”的美日联盟遏制中共国的崛起。两国政府做出了一个不同寻常的举动，将矛头直指中共国，并表达了对中共国的海上自主权以及香港和新疆维吾尔自治区人权状况的“严重关切”。

两国政府重申了对自由开放的印度-太平洋的承诺。他们在随后的联合声明中一致同意“反对破坏基于规则的国际秩序的行动，以及任何企图改变现状的单方面行动。”日本和美国称，中共国的行为与现行国际秩序“不一致”，并表示这构成了对美日联盟的政治、经济、军事和技术挑战。

这次会议召开之际，日本对东海有争议的尖阁诸岛（Senkaku Islands）的管理面临着来自中共国在日本海域日益增长的海军活动的压力。日本表示对中共国定期巡逻这些岛屿有一种危机感，这些岛屿在中共国被称为钓鱼岛（Diaoyu）。中共国新的《海警法》允许海岸警卫队在中共国宣称拥有主权的水域使用武力，这引发了人们对中日之间冲突升级的担忧。日本发言人强调：“毫无疑问，中共国即将夺取尖阁诸岛，仅日本独自应对将会输掉。”随着中共国船只成为有争议岛屿周围的常客，日本仍准备与美国合作遏制中共国的海上扩张。

在讨论结束后的新闻发布会上，美国国务卿安东尼·布林肯强调了“和平解决问题”的重要性，但警告说“当中共国采取强硬措施时，我们将作出回应。”同样，最新的联合文件不仅提到新的《海警法》“引起混乱”，而且强调美国和日本“将反对任何破坏日本对尖阁诸岛管理的单方面行动。”在之前2019年的2+2联合声明中，并未明确将中共国列为威胁；但在3月初，布林肯将中共国称为“ 21世纪最大的地缘政治考验”。

美国强调了保卫日本的坚定承诺，以及《日美安全条约》第5条的适用。该条约规定，美国将在发生袭击时协助日本。美国官员明确表示，这项保证也适用于尖阁诸岛。日本还保证加强国防，提高其支持同盟的能力。部长们还表示，当前的地区安全形势再次表明，有必要进行双边和多边训练演习，以增强联盟的反应能力和威慑力，保持对中共国的遏制。在目前的安排于2022年到期后，覆盖驻日美军的费用预计将在夏天商定，华盛顿很可能会要求东京分担更多的费用。

这是菅义伟和拜登政府领导下的第一次2+2会议，这也是自两个月前乔·拜登总统就职以来的首次海外部长级访问。鉴于川普政府威胁要从日本撤军，并对日本钢铁和铝征收关税，安全谈判是拜登政府东亚外交政策战略的一个令人欢迎的信号。布林肯强调：“拜登政府选择日本作为其第一个海外目的地绝非偶然。”

两国政府都认为这次会议是合作和领导地区事务的好机会，包括应对新冠病毒的蔓延和气候变化。会议还讨论了中共国最近对香港选举制度的改变，缅甸民主制度的恢复以及朝鲜的无核化。双方部长同意在今年晚些时候再次举行2+2会谈。
