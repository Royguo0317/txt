## 加拿大智库：认定中共种族灭绝后，将继续推动政府实质行动

翻译/简评：加拿大草原三省战友之家 文莱
![]()![](https://gnews.org/wp-content/uploads/2021/03/Macdonald–Laurier-Institute.jpg)*图片来源：Macdonald–Laurier Institute*
加拿大渥太华民间智库麦克唐纳-劳里埃研究所（Macdonald-Laurier Institute，简称MLI）网站2月23日刊文，加拿大议会一致投票通过认定中共国对新疆维吾尔族穆斯林的行为构成种族灭绝。这是该智库在外交政策、中共和人权方面的问题的又一重大胜利。

正如该智库专家们长期以来所认定的，中共实施大规模的国家监视、强迫劳动、强迫同化和放弃宗教信仰，在集中营中进行大规模关押、强迫绝育、强制节育、系统性的性虐待，还有更多的行为明显达到种族灭绝的程度。

最近，该智库高级研究员萨拉·泰希（Sarah Teich）在《多伦多太阳报》（Toronto Sun）上写了一篇专栏文章，揭露出2020年10月加拿大国际人权小组委员会就新疆问题举行多次听证会后，“确信中共的行为构成《灭绝种族罪公约》规定的种族灭绝。”小组委员会随后明确呼吁加拿大政府“承认在新疆对维吾尔族人构成种族灭绝的行为”。但是，加拿大总理贾斯汀·特鲁多（Justin Trudeau）却拒绝承认中共在新疆犯下的种族灭绝罪。他明确表示，只有在“事实和证据”确凿的情况下才能判定中共的违法行为，而且必须要有“适当的理由和证明”。
![]()![](https://gnews.org/wp-content/uploads/2021/03/Macdonald–Laurier-Institute2.jpg)*麦克唐纳-劳里亚研究所2021年1月14日举办“捍卫中国人权”在线研讨会 图片来源：Macdonald–Laurier Institute*
麦克唐纳-劳里埃研究所高级研究员查尔斯·伯顿（Charles Burton）说：“议会已开始认识到长期以来明显的事实，这是积极的进步。然而，由于内阁投了弃权票，所以我们还未能看到加拿大政府会改变其对待中共的消极态度。”

该智库的专家已对这种“令人痛心的种族灭绝”发出过警告，并敦促加拿大政府采取行动已有相当一段时间了，如果国际社会不谴责中共国在新疆迫害穆斯林少数民族的运动，那么就会开创先例，其他政权也会实施这种暴行。

认定这一种族灭绝问题是马克-加诺（Marc Garneau）就任外交部长时的首要任务之一，据统计，1949年毛泽东军事占领独立的东突厥斯坦后，中共对突厥人的迫害就开始了。此后，该地区一致被用作1964年至1996年47起核爆炸的不人道实验室，许多居民因辐射致癌。自2016年以来，情况急剧恶化，到了种族大屠杀的程度。

针对于中共种族灭绝新疆维吾尔族的问题，麦克唐纳-劳里埃研究所日前举办了网络研讨会，并邀请拉希玛·马赫穆特（Rahima Mahmut）等维吾尔族维权活动家参与到其中。会议中，一些世界人权和国际事务的最高思想家与该智库一起讨论：

**新疆拘留营侵犯人权的性质、香港民主的节制，以及中共国从西藏到蒙古的虐待和恐吓的历史；**

**自由世界的各国政府、联合国及其附属机构是如何自满的，在某些情况下，它们是如何共谋并让这些行为不受控制；**

**建议采取步调一致的全球对策。**

此外，鉴于中共国存在明显的侵犯人权模式，因此该智库成员警告绝不能允许中共政府操纵联合国塑造人权的未来。

该智库的专家们已经多次出现在加国媒体上或文章被引用，包括CBC、CTV、全球新闻、《多伦多星报》、《多伦多太阳报》、《国家邮报》、《环球邮报》、《麦克林报》等。麦克唐纳-劳里埃研究所目前是加拿大唯一在认定中共种族灭绝问题上有如此影响力的智库。

麦克唐纳-劳里埃研究所进行的民意测验清楚地表明，加拿大人强烈支持联邦政府既定的外交政策目标，该智库的努力显然有助于形成公众舆论。80%的人支持恢复以规则为基础的国际秩序，91%的人同意加强加拿大在北美的地位，86%的人支持优先消除贫困，56%的人认为加拿大应该再次尝试在联合国安理会获得席位，89%的加拿大人支持包容性贸易。而结果显示，迄今为止加拿大人对中共国持最尖锐的负面看法。只有7%的加拿大人对中共国持中等积极的态度，而73%的加拿大人持否定态度。总体而言，民调显示，加拿大人更喜欢民主政体而不是专制政权，衡量的所有民主国家的平均净得分达到19%，而专制政权的平均净得分为-66%。

具体来说，公众普遍认为，加拿大和其他民主国家应该更加公开地反对中共国侵犯人权的行为。但是，52%的受访者对加拿大政治家的表现非常消极。只有24%的加拿大人对政府的做法持积极态度，另有24%的加拿大人持中立态度。卡尔顿大学教授、COMPAS研究公司创始人康拉德·温（Conrad Winn）在回顾民调时指出，投票结果“令人吃惊”，加拿大政治史上，公众很少对中共侵犯人权事件“如此团结地关注”，所以这种趋势应该给联邦政府敲响警钟。

但还有很长的路要走。虽然议会的投票朝正确的方向迈出了重要一步，但加拿大应为解决中共国日常侵犯人权问题的全球行动做出有意义的贡献，这包括：

**1）对责任人实行马格尼茨基制裁，《马格尼茨基法案》在处理侵犯人权问题方面相当有力，它将限制所有这些不同国家侵犯人权的人不得出国旅游、购买房产以及做各种事；**

**2）将从新疆进口的产品定为强迫劳动产品；**

**3）禁止使中共国在新疆进行镇压的华为公司参与加国的5G建设；**

**4）反思加拿大对中共政府的整体态度。**

麦克唐纳-劳里埃研究所常务董事布莱恩·李·克劳利（Brian Lee Crowley）认为：**“麦克唐纳-劳里埃研究所在针对于中共和许多其他重要问题独树一帜，成为加拿大的主要智库，摆脱了党派的束缚，有些议员已表明，他们基本上同意该智库对中共的评估，他们还希望加拿大坚持领导地位并结束其对中共的绥靖政策。麦克唐纳-劳里埃研究所将继续引导政策制定者朝着正确的方向前进，无论是宣传中共对维吾尔人的种族灭绝还是其他无数加拿大人最关系的问题。”**

评：
成立于2010年的麦克唐纳-劳里埃研究所是独立的、无党派智库，以加拿大两位最早、最知名的总理名字命名：约翰·A·麦克唐纳爵士是加拿大第一任总理，威尔弗里德·劳里尔爵士是加国第一位法裔加拿大总理。该智库在加拿大国家安全战略的制定领域开​进行了一系列研究。

该智库深刻认识到了中共对加拿大的威胁，密切关注着中共国人权问题，并不断呼吁。2019年，该智库建议，加国政府应驱逐中共国大使，并加强与台湾的关系。同年，该智库选出的“年度决策者”不是加拿大政客，而是习近平，因为他领导的中共政府现出侵略性，处处制约加拿大。近年来，虽然有加拿大亲共组织不断试图渗透、影响该智库，但该智库高级研究员查尔斯·伯顿（Charles Burton）明确表示“加国人士维护国外利益的行为违背了加国公民的职责”。

被中共超限生物武器袭击至今，民生、经济的动荡促使着西方民主国家正逐步觉醒。麦克唐纳-劳里埃研究所用实际行动坚决捍卫民主自由，给被中共蓝金黄严重的加拿大带来了希望。
