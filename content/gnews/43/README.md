巴黎（美联社）4月18日-法国总统马克龙表示，与俄罗斯保持对话至关重要，但在乌克兰问题上，一定要划定“明确的红线”，以便对俄实施可能的制裁。

马克龙接受了美国电视广播网哥伦比亚广播公司新闻部（CBS News）采访，在谈到俄军最近在俄乌边境增兵的情况时说：“我们永远不会接受在乌克兰领土上发起新的军事行动。” 他说，关于这一点，西方国家必须在外交和承诺上做出表率。

当被问及对俄制裁的可能性时，马克龙说：“我认为如果（俄方）做出了不可接受的行动，我们必须对其制裁。” “而且我认为我们必须与俄罗斯划定明确的红线。 这是唯一有效的方法。”

但是他也补充说：“我认为光制裁本身是不够的，但制裁至少是（对俄）所有措施的一部分。”

周五马克龙在巴黎与乌克兰总统沃洛德米尔·泽伦斯基（Volodymyr Zelenskyy）会面，两人与德国总理安格拉·默克尔（Angela Merkel）举行了视频会议，会后马克龙接受了录播采访，该采访于星期天在“面对国家”栏目上播出。介于俄乌相邻两国加剧的紧张局势，马克龙和默克尔要求俄罗斯撤出俄方在俄罗斯西部集结的军队。

美国与北约认为该军事集结是自2014年以来最大的一次，当时俄罗斯吞并了乌克兰的克里米亚半岛，俄方支持的分裂分子与乌克兰军队在乌克兰东部爆发了战斗。 在乌克兰东部长达7年的武装冲突中，至少已有14,000人死亡。

德国和法国于2015年在白俄罗斯首都明斯克协调了乌克兰东部的和平谈判。

马克龙自2017年上任以来，一直致力于与俄罗斯就一系列问题进行建设性对话，在此次“面对国家”栏目的采访中，他继续强调与（俄方）保持对话的必要性。

法国总统马克龙说：“我确信普京总统已经准备好重启对话。” 从军备控制到世界各种危机，“我们需要与俄罗斯进行坦诚的对话”。
