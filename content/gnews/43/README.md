# **诉讼：苹果员工因批准上架反共应用程序而受苹果公司惩罚**

**前苹果公司雇员在诉讼中称，公司因为他批准上架批评中共政府的应用程序而惩罚了他。**

[!\[\]()!\[\](https://spark.adobe.com/page/wcW4t7AqXzAn0/images/bea73552-f987-400f-9b21-26055c98d2f5.jpg?asset_id=a229ae89-b75e-4e90-9ba3-4917f61d9db1&amp;img_etag=%22fff966b1f932d3642f9432a79abcf91c%22&amp;size=2560)](https://spark.adobe.com/page/wcW4t7AqXzAn0/images/bea73552-f987-400f-9b21-26055c98d2f5.jpg?asset_id=a229ae89-b75e-4e90-9ba3-4917f61d9db1&amp;img_etag=%22fff966b1f932d3642f9432a79abcf91c%22&amp;size=1024)

日前一份报道指出，科技巨头苹果公司一名前雇员在近期一份诉状中指控苹果对他个人进行惩罚，因为他批准上架了批评中共政府的应用程序，苹果出于迎合北京共产政权而做出此举。特里乌•潘 (Trieu Pham) 曾经在苹果公司任职应用程序审查员，于2019年12月向加州圣克拉拉高等法院提起诉讼，指控苹果公司歧视和不当解雇。

在诉状中，潘先生指出，2018年他被苹果公司管理人员批评，因为他授权上架了一款“批评中共国政府”的应用——郭媒体。郭媒体是一个由郭文贵先生主持运作的网站，郭文贵先生是一名中国商人，流亡在美国，目前被北京政权通缉，北京政权指控他涉嫌经济犯罪。文贵先生此前指控中共政府高层大面积腐败。

潘先生称在他2017年9月提交了一份内部歧视投诉之后，苹果管理层对潘先生的行为进行了审查，并且给了他一份“记录在案的督导计划”，该计划包含那些没有被正确完成的审查项目。该计划称潘的最严重错误是授权郭媒体上架，郭媒体在苹果商店中国区被禁止。

潘先生称郭媒体被允许上架，但是之后中共政府联系苹果公司并要求移除该应用程序。苹果公司称他们进行了内部调查，发现潘先生是授权郭媒体上架的审查员。
