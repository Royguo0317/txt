2月1日，在一份福奇发送给美国国家过敏症和传染病研究所（NIAID）的副主任 休·奥金克洛斯（Hugh Auchinclos）的邮件中说：

我和你在今天上午要进行很重要的通话。保持你的手机开机。我在上午7:45与Azar有电话会议，大概会在上午8:45结束。请阅读这篇论文和我转发给你的邮件。你今天必须要完成这些任务。
谢谢，
托尼”
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/06104425/861-pag1.jpg)
休·奥金克洛斯回复福奇的邮件：你发送给我的论文中说，这些实验是在功能增强被中止之前进行的，但是是由NIH审查和批准的。我不确定这是什么意思，因为艾米莉（Emily）确定，没有在P3框架下进行过冠状病毒的研究工作。她会去确定我们是否与国外的这些工作有任何疏远的关系。

附在电子邮件中的学术论文的标题是：一种类似SARS的蝙蝠冠状病毒群显示出可能出现在人类身上的潜力。

它讨论了美国和中（共）国的研究人员如何操纵病毒，以更好地了解它们如何将自己依附在人类身上。

**摘要说：**

使用 SARS-CoV 反向遗传学系统，我们生成并表征了一种在小鼠适应的SARS-CoV骨架上表达蝙蝠冠状病毒SHC014刺突蛋白的嵌合病毒。结果表明，在野生型骨架中编码 SHC014 刺突蛋白的 2b 组病毒可以有效地使用 SARS 受体人血管紧张素转化酶 II (ACE2) 的多个直向同源物，在原代人气道细胞中有效复制并在体外达到相当于SARS-CoV流行毒株的滴度。

根据美国国立卫生研究院（NIH）的说法，“功能增强性 (GOF) 研究涉及旨在或预期（和/或实际上确实）增加病原体的传染性和/或毒性的实验。此类研究由负责任的科学家进行，通常旨在提高对致病因子、它们与人类宿主的相互作用和/或它们导致大流行的潜力的了解。此类研究的最终目标是更好地为公共卫生和准备工作和/或医疗对策的发展提供信息。尽管有这些重要的潜在好处，GOF 研究 (GOFR) 可能会带来生物安全方面的风险。”
