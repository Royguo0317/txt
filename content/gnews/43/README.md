**西班牙2021年3月26日电/西喜社——**据美联社3月26日报道，德国空军正在向巴西空运数十台呼吸机，巴西受到中共病毒爆发的严重打击。

周五上午，士兵们在科隆-波恩机场军事区将80台呼吸机装上空客A310 MRTT。

这些呼吸机正飞往亚马逊大都市马瑙斯，那里的医院因大量中共病毒患者需要氧气和呼吸机而不堪重负。
