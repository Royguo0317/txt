巴西总统贾尔·博尔索纳罗（Jair Bolsonaro）表示，冠状病毒可能是在实验室中有目的开发的，这次瘟疫大流行是一种新的战争形式，WHO称这是一种阴谋论，是右翼领导人最新提出的误导大众的信息。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/07084219/image-97.png)
据法国新闻社法新社报道，博尔索纳罗周三说：“这是一种新病毒。没人知道它是在实验室中出生的，还是因为人类吃了他们不应该吃的一些动物。”

世界卫生组织上个月表示，实验室泄漏理论“极不可能”，而且这种病毒最有可能是通过蝙蝠传播给人类的，尽管有关官员指出，中共国当局在调查期间还不够透明。

博尔索纳罗说：“这到底是化学战、生物战还是核战？军方应该最清楚。我怀疑我们会不会是在打一场新型战争？哪个国家的GDP增长最快？”

他的评论被广泛理解为是对中共国的抨击，这个最先暴发疫情的亚洲国家是20国集团中唯一一个在2020年实现经济增长的国家。

这并不是博尔索纳罗第一次对冠状病毒大流行发表有争议的评论，参议院正在对他的政府应对新冠病毒大流行采取的措施进行调查。

总统淡化了病毒的严重性，称其为 “小感冒”，并称社会疏远措施比大流行病本身更危险，他推动羟氯喹和其他药物作为冠状病毒治疗方法，尽管缺乏研究表明它们是安全和有效的。

博尔索纳罗不是第一个暗示中共国实验室与冠状病毒大流行有关的国家领导人。博尔索纳罗的盟友美国前总统唐纳德·特朗普在大流行初期很早就提出了类似的说法。四月份特朗普就说过，在没有证据的情况下，他就确信这种病毒来自武汉的一个实验室，而武汉被认为是新冠病毒最先爆发的地方。

新冠疫情已经造成巴西411,000多人死亡。博尔索纳罗本人也曾于去年七月确诊染疫。

在巴西总统发布上述言论后第二天，中共国外交部就出来搅混水称，病毒是人类共同的敌人，当务之急是各国联合抗疫，而不是把病毒政治化、污名化。今天中共真是黔驴技穷，居然用这种此地无银的招数来给自己辩白，看来它也自知来日无多。

G7峰会后，全球对病毒追责的大势已经形成，现在巴西总统打响了第一枪而且正中目标，其他国家为了争夺政治和舆论制高点一定会争相恐后的向中共发难，这场以毒灭共的“小风暴”刮得有点猛啊。
