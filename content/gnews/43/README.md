三、世界新闻

1.11月6日，川普总统的竞选总顾问Matt Morgan发表题为“This Election is Not Over”的声明，指出“这次选举还没有结束”， 总统的团队在佐治亚州、宾夕法尼亚州、内华达州、亚利桑那州都在采取相应措施，查处选票违规行为。拜登离胜选还很遥远，川普总统将取得连任。

2.美国当地时间11月5 日，参议院官方推特发布消息，宣布共和党议员集体声明支持川普总统。声明中表示，“川普总统在过去四年里维护了美国利益。 我们（参院共和党议员）将与他肩并肩，行进在未来道路上的每一步”。

3.本周在呼吁对联邦调查局局长克里斯托弗·雷和流行病专家安东尼·福奇博士进行制裁之后，班农先生的一些节目已被 Twitter “永久中止”，而且在油管上的WarRoom相关视频已经被删除。

4.11月5日媒体报道，一名美国邮政工作人员在加拿大边境被捕，其汽车后备箱藏有800多封未被投递的邮件和选票。其被捕和被起诉的罪名是“拖延和销毁邮件罪”。如果罪名成立，他将面临最高五年刑期和25万美金罚款。

5.斯洛伐克议会11月4日通过了一项法律修正案，宣布从1948年到1990期间执政的捷克斯洛伐克共产党以及在斯洛伐克的共产党分部是犯罪组织，因为共产党执政期间曾镇压公民社会和限制民众自由。这项法律还禁止在纪念碑和纪念牌匾等地方出现象征共产主义、纳粹法西斯等专制政治体制的标志，并禁止用共产党和纳粹法西斯人士的名字命名街道、广场以及其他公共场所。

6.欧盟周五（11月6日）公布，将卢卡申科等共15名白俄官员列入制裁名单。欧洲指他们在白俄今年8月的总统选举后，涉及暴力镇压和恐吓和平示威者以及反对派成员和记者。欧盟将冻结他们在欧盟司法管辖范围内的财产，并禁止他们入境欧盟，禁止欧盟成员国公民和实体向他们提供资金。
