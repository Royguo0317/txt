印度媒体报道，英国首相办公室周二宣布，英国首相鲍里斯·约翰逊将于4月访问印度，这是其政府向印度洋-太平洋地区倾斜的一部分，以在该地区释放新的机遇。这将是约翰逊在英国脱欧后的首次重大国际访问。

伦敦将于周二公布备受期待的安全、国防、发展和外交政策综合评估报告，该报告长达100页，名为《竞争时代的全球英国》(Global Britain in a Competitive Age)，被称作是自冷战结束以来对英国在世界上的地位进行的最激进的评估。

伦敦首相办公室发表的一份声明称，综合评估将制定一些外交政策的转变，包括“向印度洋-太平洋地区倾斜”，约翰逊将于4月底访问印度，“释放该地区的机会”。 英政府声明称印度洋-太平洋“日益成为世界地缘政治中心”，并指出伊丽莎白女王号航空母舰今年将首次在该地区进行军事部署。作为这一外交政策转变的一部分，英国还在申请东南亚国家联盟(Asean)的伙伴地位。

英国最大军舰“伊丽莎白女王”号及其打击特遣部队在印度洋的部署，被官员们形容为英国“20年来最雄心勃勃的部署”。英国希望扩大其在印度洋-太平洋地区民主国家中的影响力，保持与美国的紧密联系，抗衡中共威胁。

英评估报告称，英国将“毫不犹豫”捍卫其价值观，并将加强对关键基础设施的保护。英将在安全与威慑问题上采取更强硬的立场，这可能会让军队更定期地、更长时间地派驻海外。报告强调中共对繁荣构成了“系统性挑战”。

约翰逊表示，英国需要使用“我们所掌握的一切工具”，“在一些国家试图破坏开放和自由的国际秩序时，确保一个民主仍能蓬勃发展的世界。”《卫报》称，英国向印太地区倾斜的国际优先事项重置，也是为了建立一个与中共抗衡的民主力量。
