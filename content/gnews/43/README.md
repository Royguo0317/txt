主要内容

1. 美国参议院外交关系委员会提出联合法案《2021年战略竞争法案》。该法案也称为“两党对华全面法案”，该法案要求美国政府制定政策与中共在经济和地缘政治领域全面对抗，保护和促进美国的利益和价值观，法案涵盖内容非常广泛，媒体报导法案部分内容包括1、重申美国在全球和区域性国际组织的领导地位，应对中共在全球范围内对美国的挑战；2、继续倡导人权和普世价值，在新疆、香港、台湾问题上对中共采取强硬态度；3、打击中共的技术偷窃、保护知识产权，对抗中共掠夺性的国际经济行为；4、积极应对中共的军事威胁，加强盟友合作和情报分享等；预计该法案将走快速通道通过，路德社认为该法案就是两党的灭共法案，路德透露该法案的推动得到了国会建制派的全面支持，法案将会很快在国会参众两院过关，加上美国制裁中共国7个超级计算实体直指新冠病毒威胁等一系列事情，中共知道美国真的要和中共拜拜了，美国无论是谁、无论什么党都不可能在病毒的事情上跟中共勾兑。
2. 中共新闻联播指美国是影响人类和平和全球稳定的病毒。4.9日的新闻联播中引述中国人权研究会的文章称美国对外侵略战争造成严重人道主义灾难，将朝鲜战争、越南战争、海湾战争定义为美国的侵略战争，把责任全部推给美国，路德表示这是中共在新闻联播上鲜有的动作，中共和美国已经彻底撕破脸了，现在就开始仇美的战争动员，中共学者称美国若抵制冬奥会将意味着中美关系“全面破裂”和“彻底摊牌”，拜登政府说不抵制冬奥会，不是不抵制北京冬奥会，冬奥会会另择场地举行，中共宣传部依然玩弄文字游戏，自我意淫。
3. 《2021年战略竞争法案》立法要求美政府提交新冠病毒起源报告。法案第214项包含几项立法1、要求国家情报总监和美国政府其他部门180天内提交一份新冠病毒起源报告；2、该报告需要包括新冠并对最可能的来源，美国政府掌握的信息和武汉病毒实验室和其他机构的信息；3、记录中共如何阻挠新冠病毒来源调查，包括实验室泄漏的调查，发动大规模虚假信息误导世界等内容；4、详细介绍美国掌握的武汉病毒实验室进行冠状病毒功能增强实验的情况；5、评估美国相关机构与武汉病毒实验室的合作情况，包括资金支持情况。这几项要求就是一项新冠病毒的爆料提纲，字里行间基本暗示病毒不是来自于自然，病毒跟武汉病毒实验室关系密切，中共在实验室进行了冠状病毒功能增强实验，并且背后有美国的技术和资金支持。路德社认为法案的要求跟闫博士的报告高度契合，拜登政府中最核心的人都在研究闫博士的报告，法案通过以后对于病毒来源的调查就是依法进行，包括对福奇博士、皮特.打杂客这些与中共合作的人、包括他们之间来往的邮件都将被依法取证调查，中共不配合的话，习就是萨达姆的下场。
4. 中俄合作指美国在国内和他国建生物实验室意图甩锅。针对美国和全世界正义力量要求调查病毒真相、追究中共制造和释放超限生物武器的要求和努力，中共心虚无法逃过这一关，外交部发言人赵立坚配合俄罗斯加紧宣传，倒打一耙，指美国建大量实验室进行生物武器研究、阻挡《禁止生物武器公约》设立核查机制，意图把新冠病毒责任推给美国，把十四亿老百姓绑在中共的战车上。
5. ZeroHedge报导也证实美国政府制裁的7家超算实体被指控涉嫌参与大规模杀伤性武器。所谓的大规模杀伤性武器就是与新冠病毒有关，这些实体运用超级计算机进行病毒和药物的研究，路德表示这次制裁并没涉及全部的超算实验室，尽尽只是针对参与了新冠病毒研究的实验室，让这些超算中心无法继续运作，只是一个防御性的作法。
