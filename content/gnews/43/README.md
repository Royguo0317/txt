**评论背景**

香港在被英国超过一个多世纪的殖民统治下，虽然最终没有实现民主政治制度，但在港英政府的治理下，香港社会各方面的发展整体是趋向完善与稳定的，并成功的从一个普通的港口蜕变成为国际金融中心。当地人民也得益于香港拥有中西文化思想交汇的熏陶及优势，特别在1984年中国及英国签署《中英联合声明》之前，香港当地华人参与代议政制的比例可见循序渐进的发展，整个社会渐趋民主社会的雏形，逐步具备进行全民普选的条件。因此在香港回归之前，经过中英双方的长期磋商，最终于1990年公布并在1997年7月1日实施的《基本法》，也有为香港订明实现全民普选香港行政长官及立法会议席的愿景及路线图。

当然，自香港回归中国之后，在中国共产党的统治下，与委任制下的香港特别行政区的首长和各司长的配合下，在香港落实全民普选将注定成为泡影。中共岂能容许一个享有自由民主法治的制度存在于它的管治范围之内？归根究柢，中共惧怕的是一个拥有自由民主法治的普世价值的群体，那么为了驾驭香港及统一大思想，其中可实现又甚具效果的手段就是透过移民政策。

移民政策对于一个国家或地区来说，长远来讲是一个优化本地人口、应对现代人口老化、补充本地劳动人口和增强本地整体竞争力的措施。而发生在这座国际大都市香港的移民政策则恰恰相反，中共反其道而行，自1997年香港回归中国之后，香港持续放寛中国大陆居民入境限制，以新出台的愈见宽松的移民政策，增加大量大陆居民前往香港定居，美其名曰是促进中港家庭团聚、资本投资和优才计划等有利香港的借口，这都是中共向香港掺砂子的移民政策。中共长期利用大量大陆人民冲淡香港本地人口结构，导致香港人民生活愈见困迫，令大量大陆新移民成为香港人口比例中具代表性的群体，使两个具有不同文化思想背景的群体长期处于对立面等等，这看起来是否极像中共惯用的商鞅驭民五术？

笔者需强调，此文章的简单论述绝无歧视中国大陆同胞之含意。并且需一再反复强调的是，香港在中共统治下只是一只被任意摆布的和用以挑战国际的棋子。如今的香港已非昔日我们认识的香港，可以简单作对比的是97回归前后，香港本地居民的生活质量是否变好、社会是否趋向和谐、香港是否按基本法的指引平稳过渡至民主社会等等的大方向去思考。

**中共商鞅五术共冶一炉：愚民贫民疲民弱民辱民**

香港是一个富裕的城巿，其劳动巿场的平均收入水平对于中国大陆的整体同胞来说是相对较高的，因此持续吸引大陆居民移民香港，中共持续利用香港向大陆开放的宽松的移民政策，不断冲淡香港本地人口结构，以类似汉化的手段改变香港原有的独具特色的中西合璧的人文基础。同时，大量的大陆新移民也对香港社会的医疗及住房等各方面造成负担，容易造成大陆新移民抢占社会资源的负面印象，此结果往往导致香港本地居民对大陆新移民的歧视及反感。

另外，早年持续向大陆开放的诸如优才移民、投资移民等政策，为大陆特权阶级及富裕人士移民香港打开窗口。首当其冲的是楼巿成为被炒作的盈利工具，源源不绝的大陆资本不断推高香港楼巿价格水平，香港如今成为全世界楼价最难负担的城市，香港的生活成本之高亦是冠绝全球，香港的贫富差距悬殊程度的排名在全球亦是居高不下，今天的香港巿民已普遍彻底沦为了楼奴。生活的压力迫使人民为了生计疲于奔波，无暇理会社会议题，很显著的例子是2014年雨伞革命占领中环及2019年反送中的社会运动期间，社会上不缺乏那些为了生计需要工作，从而反对社会运动的人士（无论身份是当地居民或新移民），因为经济活动大受影响，最终社会运动不但未能成功，反而导致社会内部的严重分裂。

另一方面，教育是中共必须统战的必要环节。由于香港近年出生率持续下降，那些伴随父母从国内移民而来的适龄儿童，可为香港学龄人口下降填补缺口。但是，出于生活不能适应又或其他原因，为数不少的学童被其父母安排每天跨境上学（在邻近香港的深圳出入境口岸附近定居），但生活仍主要是集中在深圳。可想而知，他们长期在大陆生活以及被灌输的是中共价值观，试问如何能融入一个截然不同的社会呢？同时，在不久的将来他们也会定居香港生活。透过不同年龄层的大陆人口移民香港，从根本上改变香港的本地人口结构，把香港赤化。

目前，中共对香港掺砂子移民策略是很成功的。回归至今只不过20多年，中共透过向香港掺砂子的移民政策，长期下来可以令争取民主的香港人民反而变为少数群体，因为移居香港并将会自动拥有选民投票权的是那些容易倾向认同中共对港政策的支持者。长此以往，这些群体也会成为香港社会中的主要群体之一，甚至是占有举足轻重的数量及地位。大量长期接受了中共国文化教育洗脑的人民已经完全缺乏对法治民主自由的认知与追求，试问移居香港生活以后的他们，对于诸如追求民主法制自由之类的社会议题的参与和热衷程度会有多强烈呢？

中共不断使用那些所谓的正义凛然的“勾结外国势力”、“反中乱港”和“爱国者治港”等口号来煽动国民的盲目爱国情绪。在这种被中共制造的政治气候下，中共人大会议及港共政府最终会利用各种借口完善香港选举法案及打击“反对派”，进而达到其完全控制香港的邪恶目的。
