![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2020/11/28234404/Screen-Shot-2020-11-28-at-8.43.35-PM.png)2020年11月25日，宾夕法尼亚州参议院网站截图
由26名宾夕法尼亚州众议院议员组成的小组正在发布一份备忘录，寻求支持一项拟议的决议-宣布该州对总统选举人和其他全州选举结果的认证存在争议。此前两天，美国总统唐纳德·川普的私人律师鲁迪·朱利安尼在宾夕法尼亚州参众两院共和党议员小组面前就选民舞弊和选举操纵的指控进行听证，因为这些指控此前被法院认为毫无根据。

宣布2020年大选的全州竞选结果存在争议，众议院共和党议员小组宣告他们有意提出这样一项决议，要求州长汤姆·沃尔夫 （Tom Wolf）和州务卿凯西·博克瓦尔（Kathy Boockvar）撤回在11月3日对总统和其他全州竞选结果的认证。

26名议员在周五晚间发布的新闻稿中援引他们所说的选举法遭到侵蚀，与邮寄投票，预拉票和拉票有关的违规和不当行为，指出在选举中出现的问题“破坏了我们的选举程序， 因此，我们无法接受对全州竞选结果的认证。”

决议案如果被提出来，其寿命也很短，因为立法会议将于11月30日星期一结束。所有悬而未决的法案和决议都将在会议结束时失效。

众议院议长布莱恩·卡特勒（Bryan Cutler）（兰卡斯特县（Lancaster）县共和党人）的发言人表示，卡特勒没有参与撰写该决议案。卡特勒也没有在会期结束前安排任何会期来审议该决议。

众议院多数党领袖凯里·本宁霍夫（Kerry Benninghoff，Centre 县共和党人）的发言人周五重复了他的核心小组领导人早些时候所说的话: 他们秉持传统，即总统选举中的普选获胜者获得选举人。

周二，州长汤姆·沃尔夫认证的选举结果显示，当选总统乔·拜登以超过8.1万票的优势赢得了宾夕法尼亚州。其他州内选举结果，共和党审计长候选人蒂莫西·德福尔（Timothy DeFoor）和共和党州财务长候选人斯泰西·加里蒂（Stacy Garrity）在他们的竞选中胜出，而民主党现任检察长乔希·夏皮罗（Josh Shapiro）则赢得了连任检察长的竞选。

尽管全州范围内的三名共和党候选人中有两名被认证为获胜者，但议员们表示，他们和拜登的总统选举人的认证（一样）应该被撤销。决议案还敦促国会 “宣布联邦总统选举人的遴选存在争议”。

在21条条款的决议中，议员们表达了有关选举公正性和完整性的担忧，这与唐纳德·川普总统的法律团队在周三参议院多数党政策委员会会议上提出的论点相似。

“如果我们要确保公众对我们的选举制度保持信任，我们绝对有必要采取这些措施。对政府的信任首先来自于选择政府的选举制度和过程的信任，”众议院共和党议员的新闻稿称。”正如宾夕法尼亚州（当年）领导了我们国家的初建一样，宾夕法尼亚州也应该带头确保我们的联邦继续作为我们国家的基石，因为无论那些选举的最终结果如何，自由和公平的选举都是最重要的。”

众议院民主党发言人比尔·帕顿对此提案作出了表态。

帕顿说：“鲁迪·朱利安尼（Rudy Giuliani）等人给宾夕法尼亚州带来了一个马戏团，这就像表演结束时拥挤的小丑车转来转去。”巴顿说。”只是有点可笑的闹剧。简单的事实是，大选结束了，乔-拜登是当选总统。”

签署了该决议案的共同提案人是：共和党人拉斯·戴蒙德（Russ Diamond），埃里克·尼尔森（Eric Nelson），保罗·斯普拉德尔（Paul Schemel），格雷格·罗斯曼（Greg Rothman），弗兰克·赖安（Dawn Keefer），迈克·琼斯（Mike Jones），大卫·罗维（David Rowe），迈克尔·普斯卡里奇（Michael Puskaric），芭芭拉·格莱姆（Barbara Gleim），巴德·库克（Bud Cook），克里斯·杜什（Cris Dush），斯蒂芬妮·布罗维奇（Stephanie Borowicz），大卫·齐默曼（David Zimmerman），达里尔·梅特卡夫（Daryl Metcalfe），大卫·马洛尼（David Maloney），丹·摩尔（Dan Moul），布拉德·罗伊（Brad Roae），凯西·拉普（Kathy Rapp），吉姆·考克斯（Jim Cox），罗伯·考夫曼（Rob Kauffman），马修·道林（Matthew Dowling），埃里克·达万佐（Eric Davanzo），里奇·欧文（Rich Irvin），亚伦·伯恩斯汀（Aaron Bernstine）以及安德鲁·刘易斯（Andrew Lewis.）。

州和地方官员一直坚称没有证据表明与选举有关的广泛欺诈。

川普在宾夕法尼亚州输掉了对选举结果进行的一系列的法律挑战。最近的一次发生在周五，联邦上诉法院小组拒绝阻止对宾夕法尼亚州投票的认证。法官斯蒂芬诺斯·巴斯（Stephanos Bibas）写道，没有证据支持他关于2020年总统选举不公平的说法。

“对不公平的指控是严诉的。但称选举不公平并不意味着选举（真的）不公平，”巴斯在美国第三巡回上诉法院的意见中写道。”指控需要具体的指控，然后证明。我们这里两者都没有。”
