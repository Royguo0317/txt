联邦卫生部长詹斯·斯潘（Jens Spahn）警告过冠状病毒继续感染的危险。 CDU政客周一在柏林参观Messe疫苗接种中心时说：“疫苗接种不能阻止第三波流行，第三波流行正在增加。”疫苗接种率较高的国家，例如智利，英国或美国等表明限制人员接触仍然是必需的。

Spahn认为，由于入住率的上升，重症监护病房和诊所的情况令人担忧。他说：“我们必须战胜第三波流行，减少接触。” “特别是在私营部门，学校，在工作中，只要有可能。”

部长呼吁联邦各州“有100多个州持续实施商定的紧急封锁措施，以减少紧急事故的发生”。

关于罗伯特科赫学院对全国统一规则的要求，斯潘说，目标必须是战胜这一次大流行。  “理想情况下，有统一的国家程序。”紧急封锁的机制是共同商定的。据此，如果感染率超过100，各县，地区，州将不得不制定更多限制措施，并撤回开放。

Spahn再次为已经接种疫苗的人提供救济。这是关于能够像接受阴性测试的人一样对完全接种疫苗的人进行治疗。对于阴性测试的人和完全接种疫苗的人，仍需要距离和接触规则，卫生或医疗防护口罩。 “只要能够通过测试或接种疫苗降低感染的风险，任何人不可能比接种疫苗的人，更有自由。 ”
