# 重要事件

- ** 02.10 NewsMax [拜登在五角大楼访问时宣布检讨对华防务立场](https://www.newsmax.com/politics/china-biden/2021/02/10/id/1009536/)**


美国总统拜登宣布检讨美国对中国的军事立场，称中国需要一个 “强硬 “的方法，同时承诺他的政府只会在万不得已的情况下使用武力。

拜登以总统身份首次访问五角大楼时，还试图打破其前任特朗普的政策，承诺美国国防力量不会将国内政治排除在国家安全之外。他说：”我永远不会不尊重你们。”副总统卡马拉-哈里斯和国防部长劳埃德-奥斯汀站在他身边。

在与五角大楼的高级将领和文职领导人会面后的简短讲话中，拜登重点谈到中国是美国的首要战略威胁。他宣布成立国防部中国问题特别工作组，并下令立即审查军方应对来自北京挑战的战略方针。他说，此举是必要的，”这样我们才能在与中国有关的事务上规划出一条强有力的前进道路，这将需要整个政府的努力”。

“我们需要应对中国带来的日益增长的挑战，以维护和平，捍卫我们在印太地区和全球的利益，”他补充说。

新的特别工作组有4个月的时间就美国的对华军事战略，技术、部队结构和部队态势，以及与北京的联盟和双边防务关系提出评估和建议。

拜登说，五角大楼的审查结果需要得到两党和国会的支持，也需要汇集盟友和其他合作伙伴。”这就是我们如何应对中国的挑战，确保美国人民在未来的竞争中获胜，”他说。

- ** 02.10 TheHill [世卫组织发现COVID-19病例的可能证据比查明的时间早两个月](https://thehill.com/policy/healthcare/538342-who-finds-possible-evidence-of-covid-19-cases-two-months-earlier-than)**


据《华尔街日报》报道，为期四周的调查显示，在2019年12月8日（中国称发现首例冠状病毒患者的日子）之前的两个月，中国中部有超过90人因冠状病毒症状而住院治疗。

世卫组织的调查人员告诉《华尔街日报》，世卫组织希望中国从2019年秋季开始对更多人群的血液样本进行检测，以证实他们的理论，但中国当局表示还不允许他们进行检测。

中国官员确实对去年秋天发现有冠状病毒症状的90名患者中的一部分做了抗体检测，但在他们身上没有发现抗体。不过，WSJ称，WHO调查人员认为他们只是等了太长时间才对他们进行检测，抗体就消退了。

中国国家卫生委员会冠状病毒小组负责人梁万年周二表示，已经检查了血液样本和医疗记录，但没有证据表明该病毒在2019年12月之前出现。为了确定冠状病毒是否在2019年12月之前存在，还需要更多的研究和血液样本。

- **02.10 WashingtonPost [疾病预防控制中心表示，口罩应更贴合或加倍保护](https://www.washingtonpost.com/health/2021/02/10/cdc-double-masks-covid/)**


联邦卫生官员周三敦促美国人考虑戴两个口罩作为几个策略之一，以更好地保护自己免受冠状病毒更多传染性变种的威胁。

两种方法大幅提升适合和保护，根据CDC报告和其网站上的更新指南。一个是戴在一次性手术面具上的布面具。二是通过打结耳环和塞进靠近脸部的两侧，防止空气从边缘漏出，形成更紧密的贴合，来改善单一手术面具的贴合度。

报道称，在使用假人的实验室实验中，这两种方法都能将潜在传染性气溶胶的暴露量减少95%以上。”我们知道通用掩蔽是有效的，”疾病控制和预防中心的covid-19响应的医疗官员John T. Brooks说。”而现在这些变种正在流传……无论我们能做什么来改善口罩的贴合度，使其更好地工作，我们就能更快地结束这种流行病。”

在冠状病毒抵达美国一年后–死亡人数接近50万–更新后的指南强调了戴口罩的重要性，将其作为抵御更多传播性变种的最佳防御手段之一，同时强调了社交疏远、避免人群和通风不良的室内空间，并实行经常洗手。

- **02.10 BreitBart [中国：声称被强制绝育的集中营受害者是 “天生不孕”](https://www.breitbart.com/asia/2021/02/10/china-concentration-camp-victim-claiming-forced-sterilization-was-inherently-infertile/)**

周三的新闻发布会上，其中一位官员徐贵祥声称，齐亚乌墩在被迫进入集中营之前是出了名的不孕不育，她的所有指控都是谎言。”图尔苏奈从未接受过’强制绝育’–她所有的家人都知道她天生不孕。”徐贵祥称。对于这一指控，徐某没有提供任何证据，比如显示她不孕的病历或所谓前夫的证词。

在这次袭击事件之前，中国政府曾公开吹嘘新疆的限生措施，称这是女权主义的胜利，把维吾尔族妇女从 “制造婴儿的机器 “中解放出来。

兹亚乌敦因作证她所目睹的恐怖事件而受到攻击，这是针对指责中共种族灭绝者的此类攻击浪潮中的最新一次。2019年，该党对一位名叫米赫里古尔-吐尔逊的妇女进行了抨击，她和齐亚乌敦一样，作证说自己在集中营系统中被电击，并声称中国政府在集中营中杀死了她的一个三胞胎婴儿。一名中共官员指责图尔荪携带 “梅毒和其他传染病”，以此来诋毁她的证词，而不涉及她的指控。

周三，徐建国随后试图诋毁英国广播公司，称该通讯社未经中共允许就采访了集中营受害者。”这严重背离了新闻职业道德，充分暴露了BBC出于政治目的抹黑新疆的恶意，也破坏了BBC作为国际知名媒体的声誉。”徐称。

### 2021.02.10 参考消息 北京测试数字人民币

据美国消费者新闻与商业频道网站2月8日报道，北京市政府最近宣布，这座城市将在中央银行的小规模数字货币试验中发放约150万美元（1美元约合6.43元人民币——本网注）。深圳和苏州过去几个月进行过类似的试验。

报道称，在此次试验中，北京将发放5万份数字人民币红包，每份金额为200元人民币。中签者可于春节期间在指定线下商户或电商平台京东的相应专区进行消费。

红包有效期为2月10日至17日。此次数字人民币试验只向持有中国第二代居民身份证的人开放。

报道称，中国人民银行一直在开发数字货币，预计其运行方式类似通过现有支付软件进行的交易。移动支付在过去几年已成为中国消费支付的重要形式。

报道表示，与比特币等众所周知的数字货币不同，中国央行开发的这种货币由一个强大的主体控制，而不是由全球用户管理的去中心化系统。

***简评：***总加速时快马加鞭，让数字货币一统天下。
