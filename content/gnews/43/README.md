亚洲新闻1月22日报道称欧盟与北京之间的双边投资协议的签定或将因香港人权问题推迟。欧盟议会21日投票通过了涉及香港的决议，谴责中共对香港民主运动的镇压，呼吁维护香港自治，同时捍卫香港、新疆与西藏人权。

欧盟委员会称与北京达成的协议尚未得到批准，预计在两年之内不会全面执行。此时提出对香港人权问题的关注，让协议增加了变数。同时，据推测，美国将中共对新疆维吾尔族人的镇压定义为“种族灭绝与反人类罪”，也严重破坏了该协议的达成。

欧洲议会成员认为与中共签署协议使欧洲丧失了捍卫人权的信誉。目前已有多个欧盟成员国中止与香港的引渡协议，并考虑对镇压民主抗议者的香港和北京官员进行制裁。

欧盟负责平等事务专员达利（Helena Dalli）公开表示， “香港局势急速恶化，当局将合法的政治活动定为刑事犯罪，目的在于扼杀多元化、民主的声音，以及香港的政治与基本自由。”[1] 欧盟会持续关注并且支持维护香港自治。

对欧盟的决议，中共国外交部发言人华春莹表示反对称，香港事务纯属中（共）国内政，根本不是什么民主人权问题，任何外国政府组织和个人都无权干预。[1]

中共对香港自由民主运动人士的镇压和拘禁逮捕，已经引发了世界各国的关注。美国、澳大利亚、加拿大和英国政府1月9日就香港民主人士被大批拘捕发表联合声明， 声明中指出“国安法明显违背《中英联合声明》，破坏‘一国两制’框架。它限制了香港人民的权利和自由。很明显，国安法正被用来铲除异议和敌对政治观点。我们呼吁香港和中共国中央政府有关部门尊重香港人民不受逮捕和拘禁恐惧的法律保障的权利和自由。重要的是，要公平举行包括代表不同政治意见的候选人的九月份被推迟的立法会选举。”[2]

对香港自由民主的镇压、对新疆和西藏的迫害，对新疆维吾尔族及其他少数民族的“种族灭绝和反人类罪”的认定，已经彻底撕下中共虚伪的面具。迫使各个国家重新审视与中共的关系，因为继续与中共合作勾兑将被视为中共“同盟”，一起背负践踏人权利用强迫劳动获利的反人类罪名。
