布赖特巴特新闻(Breitbart News)报道，中共官方媒体《环球时报》文章称缅甸作为中共主要稀土供应国，其稀土供应出现了问题。自上个月军事政变击垮了民选政府引发全民抗议以来，其对缅甸国内经济和贸易的负面后果正显现出来。

中共第一次公开表示政变将可能损害他们的商业利益。

缅甸民主活动人士指责中共积极支持军政府，一些抗议者因此破坏了在缅甸的中资企业。缅甸愤怒的抗议者甚至高呼要烧毁中共的天然气管道，更在中共大使馆外举行抗议，质疑中共政府在“一带一路”基础设施计划中为缅甸提供数十亿美元资金的贷款陷阱。

《环球时报》周日承认，“缅甸对中共国的稀土出口在一定程度上减少了，这导致市场上稀土氧化物价格飙升。”据报道，自今年年初以来，稀土价格已经上涨了30%到40%。缅甸的矿山表面上仍在按计划运行，但“物流问题”阻碍了向中共国的发货。缅甸的“政治动荡”正蔓延到有稀土矿的北部地区，在不久的将来可能危及矿山的运营。

消息人士表示，“最大的问题是缅甸的稀土无法运往中共国，目前还不清楚这一障碍何时会被拆除，因为它高度依赖缅甸的政治局势。”

今年2月，商业分析师曾怀疑中共是否低估了政变可能带来的供应中断的影响。稀土精矿是电子、机器人、智能手机、强力磁铁、先进武器系统和其他高科技产业的重要资源。2018年缅甸稀土短缺，当时由于与中共的争端，缅甸政府短暂禁止出口，导致稀土价格飙升，中共国高科技产品出口偶尔中断。缅甸政治不稳造成的新短缺正在打击中共国科技制造业。

《环球时报》坚称，中共国的稀土储量“可以轻松满足不断增长的国内需求”，中共国的矿山可以提高产量来弥补来自缅甸的短缺。这掩耳盗铃的论调让人想到去年中共单方面禁止澳洲煤炭进口造成中共煤炭短缺，各地出现拉闸限电的窘迫境况。
