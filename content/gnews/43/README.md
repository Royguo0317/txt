## 现实中的第八号当铺
` 喜马拉雅意大利罗马达芬奇农场` [轉載自GNews](https://gnews.org/zh-hans/1050028/)

#### 作者：Tito

《第8号当铺》，是改编自香港作家深雪的同名作品的电视剧，共分三部份，116集。记得大学时期看这部作品，被作者的才华深深吸引。故事的剧情先不必说，仅仅是世界观的设定，就发人深省。一个什么都可以典当的当铺，第八号当铺，可以典当眼睛，手脚，视觉，味觉，灵魂，爱情，亲情，等等等等，换取你想要的一切。只不过需求的越多，典当的价格就越高。大学时我深深的好奇，突发奇想，如果世间真的有这么一间当铺，我一定要找到它。而这部电视剧的主题曲，我也格外喜欢，叫做[《黑色契约》](https://mojim.com/cny100044x31x1.htm)。


```
黯淡的招牌　低头　迎面而来
街上的灵魂　全都　可以买卖
愚蠢的以为　什么　自己最爱
那些欲望　我静静的主宰
＊对魔鬼乞讨　虚荣的骄傲
黑色的契约　像月光下的暗潮
Woh~我冰冷地抚摸着你的悲哀
Woh~只是交换没有爱
 
漆黑的窗外　说不出的阴霾
墙上的影子　单纯已经不在
脸上的表情　残酷没有期待
狂风掠过　我跟你谁才腐败
```


每当听这首曲子，就不断在自己脑海中重复着当铺老板和典当者签约的场景，当铺老板那一句“契约成立”，真的非常个性。

步入社会，如果没有深厚的背景，每个人都要从最底层做起，我也不例外。刚开始的时候总想一步登天，遇到挫折，就很容易打退堂鼓，琢磨着这世间是否真的存在第八号当铺，试图进去典当些什么，可以让自己的生活平顺些。现在想想，那个时候真的是太天真，这种可怕的想法都有。可更怕的是，这世间真的存在第八号当铺，而且是真的什么都可以典当，现实中的第八号当铺和电视剧情中的一样，当一个人有需要时，八号当铺的名片才会出现在那个人的手上，然后这个人才会找到第八号当铺。

在中国，八号当铺的名片，便是[入党誓词](http://cpc.people.com.cn/GB/64156/65682/4440819.html)。

**我志愿加入中国共产党，拥护党的纲领，遵守党的章程，履行党员义务，执行党的决定，严守党的纪律，保守党的秘密，对党忠诚，积极工作，为共产主义奋斗终身，随时准备为党和人民牺牲一切，永不叛党。**

这就是八号当铺的名片。有了这张名片，走入了党内，就可以开始典当了。我们会发现，有人典当了自己的妻子，有人典当了自己的儿女，有人典当了自己的全家女性，有人典当了自己的身体器官，有人典当了灵魂，有人典当了爱情。八号当铺，一直存在，而且即使你要典当，也要看当铺要不要你的典当物，不是你想典当就典当了。

事实上，中国人是最有契约精神的，他们一生只遵守和党的黑色契约。每个中国人降生那一刻，面前就准备好了一张契约，只不过要待到成年时，方才可以签署，因为成年人才有典当的资本。只不过这份契约只有没了良心的人才可以签署。中国人，最厉害了，可以没有血性，可以没有亲情，可以没有爱，但是必须要向魔鬼乞讨，换来虚荣的骄傲，而后心满意足了。

中国人一生只签订一张契约，就是和共产党的黑色契约。除此之外其它一切的契约都不予理会，签契约的必要条件就是没人性。而中国人恰恰就是什么都不能没有，就是可以没人性。

当我听到有中国人可以专门把自己一家女性典当给党的时候，我心里一个激灵。什么都可以典当的含义，残酷没有期待，原来是如此模样。

在中国，我们可以看见，老人不能扶，但你必须爱党。父母可以不孝顺，但你必须爱党。可以忽视周围的生命，但你必须爱党。赚钱的营生都是断子孙根的行当，但你必须爱党。可以杀人放火草菅人命，但你必须爱党。但是当你爱了党，为它赴汤蹈火后，党给我们的，就是典当我们自己一切的机会。党的招牌，迎面而来，街上的灵魂，全都可以买卖，现实中的八号当铺，恐怖如斯。

我忽然庆幸了，在我最懵懂容易做错事的时候，我离开了中国。免于我接收八号当铺的名片，免于和魔鬼签订契约，保留了自己的人性。

现在再看八号当铺，不仅仅是佩服作者的才华了，更佩服作者的洞悉能力。原来，什么都可以典当，是如此灭绝人性的一件事情。原来无止的欲望，可以吞噬人间的一切。原来与魔鬼交易，踏上的，是一条不归之路。

那路的终点是，地狱。
