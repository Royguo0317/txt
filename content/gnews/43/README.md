据路透社报道，前美国联邦通信委员会(FCC)主席阿吉特·帕伊（Ajit Pai）在日前专访中发出警告，中共潜在的间谍活动以及对美国电信网络和互联网自由的威胁，将是美国监管机构未来四年面临的最大国家安全问题。

他指出，中共的“广泛”活动令人关切，包括监视、经济间谍活动以及潜在将恶意软件植入美国或世界各地的网络等行动，导致使用不安全设备处理敏感信息时，可能发生严重后果。

帕伊于2017年1月被前总统川普任命为FCC主席，并于周三（1月20日）辞职。在他任职期间，FCC严厉打击了中共国网络设备制造商，例如华为和中兴通讯。

帕伊特别强调，中共具有非常坚定的世界观，他们想要统治互联网空间并施加自己的意志甚至超越边界。这不仅对互联网自由构成严重威胁，而且对美国及许多盟友的国家安全也构成威胁。

据悉，出于对中共利用网络设备从事间谍活动的担忧，截至目前，美国联邦通信委员会已经启动撤销中共最大电信公司“中国电信”在美国的运营授权；包括华为、中兴通讯等制造的网络设备被禁止纳入联邦电信设备采购。美国国会还在去年底批准特别拨款，用以支付在美国通信网络中撤换中方设备的费用。
