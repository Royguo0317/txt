主要内容

1、北约加强军事部署在南中国海附近活动频密。

继美英两国之后，北约其他成员国法国、义大利同德国，相继派遣军事力量，部署在南海。法国同德国的军舰长期驻守大西洋，这次转战太平洋地区，驻守在南中国海，明显是在为可能发生的南海战争，作准备。

这意味着，掌握全球四分之三主要军事力量的北约，已经将目标对准中共，将兵力囤积在中共的门口：南中国海。

不同于中共，凡事无规无矩的做法，北约在部署军事力量之前，需经过严谨的作战决议，制定完整的作战计划，才可以获得各个国家的资金援助。因此，一旦北约制定作战计划，部署兵力，绝对不会因中共“蓝金黄”而改变。

而且，根据国际惯例，北约30国组织的军事行为，向来不受制于任何一个国家的政治倾向。

不仅如此，北约30国组织有个规定：如果任何一个盟国确认遭受攻击，都会被视为整个北约受到攻击。早前捷克大主教已经公开直言“病毒是生物武器”，路德社亦曾经在节目中透露过，北约各军情部门，一早知悉病毒是中共生物武器一事，只是心照不宣，政府可以继续玩弄政治游戏，但北约的军事行动只会跟住情报，不会跟着政治风向，随意转态。

所以，北约最近在南中国海频密的军事部署，很有可能是在作“战前准备”。

2、中共痴梦：全球无法排除中共供应链。

中共喉舌发表社评：“建立排除中共的供应链，是痴心妄想”。坚持对墙内民众灌输夜郎自大的腐朽思想，14亿廉价劳动力的产能冠绝全球，因此，全球都不可能同中共脱钩，排除中共供应链。

足见中共仍然维持在中世纪的落后思维，罔顾现实，绑架14亿人，沉浸在自己的痴梦之中，相信总有一日，一定会为之付出代价。

欧美之所以发展迅速，仰赖于其无大台，反垄断的自由体系，因此能够培育具备自由意志的创新型人才，共同建构覆盖全部产业链的生态圈；而中共奉行集中制，垄断制的专制体系，阻碍了人才的横向发展，最终迫使各行各业，全数走向复制、抄袭的死路上。

中共几十年来致力于实现“完美犯罪”，靠“假骗偷”欺骗全世界。而新中国联邦的中国梦是依靠法治，以公平的方式，建立中国文明，用实力获得世界尊重。

3、习近平：美国是阻碍中共崛起的最大威胁。

左媒《纽约时报》报道：中共党魁习近平警告说“东方正在崛起”，美国是最大威胁。

习近平表示，希望在后共和党时代的世界中确保中国的繁荣和实力，他指出，中国正在进入一个“东方在崛起而西方在衰落”的机会时期。他还告诫官员：“当今世界最大的混乱来源就是美国”、“美国是我们的最大威胁。”

二战时期，美苏原本是盟友，后来，美国一篇报导揭露前苏联将美国当作最大的威胁，双方随即进入冷战时期。

左媒刊登这样的文章，将习近平定义为反美分子，以习近平为首的共产党与美国是敌人。释放出几个非常重要的信号：

第一：中共没有将美国视为敌人，是习近平仇视美国。

第二：该篇报导的撰文记者，克里斯·巴克利，是长期驻守中国的外国记者。他之所以能够获得习近平的秘密言论，并将其报导出来，很有可能是习近平在党内的敌对势力刻意提供给他，目的是想将中美矛盾全数推给习近平。

第三：中国共产党不希望就此灭亡，惟有推习近平去承担所有罪责，想借着这篇文章，暗示拜登政府，罪魁祸首就是习近平，只要消灭习近平，中美就可以重修旧好。

不过，习近平看到这篇文章之后，亦都不会坐以待毙，他会思考谁会是背后推手，并作出反击，所以，相信这篇文章一定会进一步发酵。

4、美国务卿布林肯将对抗中共列为优先工作重点。

美国国务卿布林肯发表对外政策讲话，将对抗中共列为优先工作重点。他强调，当新疆人权被侵犯、香港民主遭践踏，这个时候，美国应挺身捍卫自身价值，联合盟友力量，共同对抗中共。

布林肯在发言中特别指出，美中关系是“应该竞争时竞争，能够合作时合作，必须敌对时敌对”。

同时，他还暗示，中共妄图重新建立规则，挑战现行这个由美国建立起来的，稳定开放的国际秩序，那么，不但中美将会脱钩，美国还会联合盟友一起与中共脱钩。

布林肯的发言，是美方正式回应王毅所提出的，“四三三”要求，表示美国政府已将“敌对”列入应对中共的选项。

纳瓦罗早前指出，川普总统在卸任前，已将所有对抗中共的政策制定完毕，拜登政府执政后，经过思考和评估，发现只能按照川普的抗共政策继续推动。

实际上，爆料革命路德社早在2018年，就已经提出“中美脱钩”的概念，只不过，无论当时川普政府的灭共计划有多完美，都必须获得两党共识，获得两党的民意支持，这样才可以顺利实施。

布林肯的这次公开表态，说明美国的左派势力，已经加入到对抗中共的行列，因此，接下来，新疆种族灭绝问题，香港问题，中美脱钩都将成为中美对抗的重点。

5、中共对拜登出手：微信散播亨特硬盘黑材料。

有网友爆料，近期中共利用微信公众号，开始散布亨特·拜登的硬碟黑材料。显示“习拜电话”之后，拜登未有跟随习近平的要求，中共恼羞成怒，开始对拜登出手。

未来中共很有可能，会公布关于拜登大选舞弊的证据，势必会对拜登造成巨大的威胁。

路德社早前在节目中曾经揭露，中共手握的黑材料，任何一个都足以弹劾拜登总统，如今，中共放任黑材料流传，是公开对拜登进行恐吓，而拜登政府惟有加快灭共步伐，才能避免继续受制于中共的要挟。
