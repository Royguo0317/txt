## 千人计划的待遇

为什么千人计划在中国如此有影响力？因为获得千人计划头衔的人获得了极其优越的政治待遇及经济待遇。

千人计划从2008年开始，到2020年经过了12个年头。成为中国仅次于院士头衔的一个在学术和技术等领域第二重要的头衔。首先获得这个头衔后可以得到1000万以上的各项现金补贴。包括中组部100万元人民币的首补，然后省级的不少于两倍中组部补贴的现金补贴，至少200万，市级的不少于以上两级补贴的1-2倍现金补贴，一般在500万左右，然后区里还会有一定的补贴。这样综合不少于1000万元。一般还会有另外计算的住房补贴，在100-200万元。

另外，获得千人计划头衔后，就有了政治待遇，表面你是被组织“被认可的人”。到其他省市进行科研、创业等活动，还可以获得各种补贴，一般都在300-500万人民币之间。一般都会到多个城市去走穴捞金，年景好时外省市凭这个头衔就可以获得1000万以上的补贴。

当然地方的补贴是分阶段拨付的，一般三年。补贴有一部分是直接给个人，有一部分是给他所在单位或创立的企业。因为国内大部分科研基金的大部分都用于改善科研人员的生活，所以用于真正科研的资金占少部分。

科研人员坐冷板凳，是讲给老板姓听的。

千人计划中，真正带来技术的人较少，大部人是忽悠及勾兑名利。但人才计划确实有较明显的技术提升效果，因为很多参评人员有意或无意中带来了很多的技术信息，或者叫情报。

由于中美关系的矛盾，千人计划中少数真正执行“计划”的大咖被逮，千人计划及各地人才计划的网页于2019年消失。并加强了“政审”，以前欢迎八方来客的场面已经改为地下模式。并且因为进入了不公示的模式，各地方政府的组织部人才办对自己不信任或不喜欢的人，随意剥夺入选人员的千人计划头衔，不予通知及兑现以上所提待遇。

另外，由于国内国际疫情、政治、社会等问题严重，各级官员无暇顾及人才项目。所以实际上包括千人计划等各地人才计划已经慢慢萎缩，也不能真正起到吸引人才及获得技术的目的，只成为了相关项目负责人及其亲朋好友兑现利益的工具。
