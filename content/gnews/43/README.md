公元前三世纪的中共国诗人屈原在他深爱着的南方家乡，在血腥的内战中用凄美的颂歌和的热情来表达他的爱国主义：当局为他奉献了到达火星的使命，以此为天文施洗。 从他的诗经到天堂。习近平主席天堂提出了很明确的问题：中共国打算在太空，和美国在民用和军事上展开竞争。中共国的火星漫游车“祝融号”，它以神话中的“火与天”之神命名，从而完成了该行动的政治和技术象征意义。北京打算在这个十年内与宇航员一道，在宇宙太空方面超越。

这一挑战给美国总统乔·拜登，俄罗斯领导人普京和欧盟带来了战略困境。2018年，中共国发射的卫星数量超过任何其他国家，2019年它们到达了月球的另一端，而长征导弹也具有良好的抵抗力。习近平没有掩饰自己的野心，称赞聚集在距北京一个小时车程的雄安市的科学家们。雄安市从一个昏昏欲睡的省变成了航空航天枢纽。他延用了毛泽东的口号并继承了绝对的权力。当时，中共国人是莫斯科的附庸，“两枚炸弹，一颗卫星”，而习下一步雄心勃勃的步骤是，在2024年之前，与美国国际空间站（*ISS*）的落成相吻合，启动中共国空间站。

为了不重燃战争之风，北京承诺国际在进入太空基地之前，除与普京达成协议外，还与欧洲航天局，法国，奥地利，阿根廷建立合作。但是，说服性的外交还不足以说服白宫和五角大楼的军事指挥。在1月25日发布的文件中，美国空军在“反太空”中定义了空军学说，这是一场太空战，可能对天空造成严重破坏。每种航空航天活动都有双重用途，很容易在战争中倾覆，击中敌方卫星，或者在干扰行动中使用激光遮蔽监管GPS导航，电话，电信，网络的中心，使敌对国家瘫痪或通过欺骗使其迷失方向。，它将更改的命令发送到地面服务器。因此，习近平于2020年6月下令部署北斗卫星系统，这是GPS的中共国替代方案。

到目前为止，拜登还没有动用由唐纳德·特朗普（*Donald Trump*）创建的新的太空部队军事指挥部，以动员它作为威慑力量。霸权仍在华盛顿一边，美国宇航局的预算为190亿欧元，而中共国为8.5欧元，但北京在私营部门方面绝对占主导地位，协调该中心的战争和经济努力。自2011年以来，在美国副总统弗兰克·沃尔夫（*Frank Wolf*）的修正下，美国国会禁止在太空领域与中共国合作，禁止*NASA*接收私人实验室的科学家，在这种不信任的气氛下，如今的任何解冻似乎都不成熟。美国人一直在与私人公司合作，从埃隆·马斯克（*Elon Musk*）的导弹到*Axiom*航天集团的空间站，这些公司可能会取代国际空间站。

因此，天上的军备竞赛将继续下去，整个欧洲和意大利航空航天业都在进行科学合作，就目前而言，在共产党成立一百周年的七月庆祝活动前夕，北京正在庆祝其成功。引以为豪的口号没有提到诗人屈原，是因为这位诗人是在残酷的内战中感到不安而选择淹死自杀。

#### 简评：

太空已成为大国竞争的舞台，而中共国是美国未来面临的最大威胁，北京在加速它的威胁性手段建设，中共国的太空野心也昭然若现。也许有一天太空冲突真的在所难免。
