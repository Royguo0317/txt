一、新中国联邦新闻

1.“路德时评”本日摘要（[早间](https://www.youtube.com/watch?v=soRGpqnslz8)、[晚间](https://www.youtube.com/watch?v=72B-caXcVYI)）:

Ⅰ. 在中国的农历新年即将到来之际，美国总统拜登在美国时间星期三(2月10日)晚上与习近平通了电话。白宫表示，拜登向习近平阐明了美国对中共国一些行为的“根本性的关注”，包括经济、香港、新疆和台湾等问题。拜登还与习近平讨论了在符合美国利益时与中共国合作的领域。白宫一位高级官员表示，拜登希望与习近平建立开放的沟通渠道。

Ⅱ. 拜登总统选择的通话时间显示出非常大的外交弱势，这是川普政府时期很难见到的现象。习现在最关心三个问题，分别是新疆种族灭绝罪、中共病毒疫情真相和如何消灭爆料革命核心成员问题。如果拜登政府选择和习展开合作，这又重新定义国际秩序，对海外灭共势力也会产生重大影响。

III. 拜登在推特中声明，只要对美国有利，他就会选择和中共国合作。拜登可以将江、曾、孟家族在美国的资产和企业冻结，为美国输送利益，洗清他与中共勾兑的嫌疑；另一方面还帮助习清除敌对势力、巩固党内权力。中共党内的平衡将被彻底打破，必将掀起一波残酷的厮杀。

2. 美国参议员马可·鲁比奥和鲍勃·梅嫩德斯要求亚马逊首席执行官杰夫·贝佐斯解释其与中共面部识别提供商大华生物的关系。大华生物最近开发了专门为中共当局识别维吾尔族人的生物识别技术。

3. 美国国务院发言人普莱斯周四表示，美国对中共国在技术上的“掠夺性”行为深表关切，并补充说，拜登政府将在符合美国利益时与中共国接触。华盛顿对中国共产党的活动，包括通过孔子学院所进行的活动表示担忧。

4. 美国一位高级政府官员星期三表示，拜登政府将考虑与盟国合作，对向中共国出口某些敏感技术增加“新的、有针对性的限制”。

5. 前美国国家情报局局长拉特克利夫周三在福克斯新闻采访中说，中共军方“命令武汉病毒学研究所的科学家进行冠状病毒实验”，其中一些冠状病毒”与目前的中共病毒有96.2%的基因相似度。而且，研究这些类似冠状病毒的科学家中，有一些人在2019年秋天出现了类似的中共病毒感染症状”。

6. 拜登政府2月11日要求一家联邦上诉法院暂停围绕川普政府试图禁止微信海外版WeChat的诉讼。一天前，拜登政府要求另一家法院暂停审理有关抖音海外版TikTok的案件，因为拜登政府需要时间来审议特朗普政府禁止这两个应用程序的提议。

7. 中共官员在周三的新闻发布会上再次宣称，所有在中共国集中营系统中对种族灭绝举证的幸存者都是被雇佣的演员，特别是毫无证据地指责被强制绝育的一名妇女，声称她一直没有生育能力。

8. 美国总统拜登启动五角大楼对中共国战略的审查，这可能会为部署更多的部队和武器开辟道路，这是对付中共的“连贯”政策的一部分。拜登同时宣布，计划成立一个新的工作组来研究美国对华政策。

9. 2月11日，中共国家广播电视总局宣布，英国广播公司（BBC）世界新闻台不符合境外频道落地的条件，决定不允许其世界新闻台继续在中共国境内落地。英国外交大臣多米尼克·拉布表示，“中共国在大陆禁止英国广播公司世界新闻台的决策，是在削减媒体自由，这不可接受。中共国对媒体和网络自由的限制，在全球属于最严厉的，这最新的举措只会损害中共国在世界眼中的形象”。

10. 《国家脉动》周三曝光了，世界卫生组织病毒调查组成员皮特·达扎克接受了中共的研究经费，并在美国经营着一个与武汉病毒学研究所合作的争议性组织。达扎克所在的调查小组刚刚宣称，病毒”极不可能 “来自中共的实验室。他们的说法与中共的说法相呼应。

11. 印度和中共国周四开始从喜马拉雅山脉西部饱受争议的湖泊地区撤出部队和作战坦克，这是进行了长达几个月的对峙后取得了突破。

13. 知情人士称，自去年年初以来，腾讯副总裁张峰一直在接受中共国反腐机构的调查，他涉嫌未经授权将腾讯社交媒体应用微信收集的个人信息提供给前公安部副部长孙力军。同时消息透露，该公司董事长兼联合创始人马化腾暂时没有受到调查。中共政权早就完全掌握腾讯的数据信息，此次被查疑与党内斗争有关。

14. 据中共官媒报道，中共国财政部已正式要求德勤公司进行内部调查，此前德勤的一名北京雇员匿名举报揭发了对该纽约上市公司的审计违规行为。外界评价，中共可能会主动清理掉一些“高风险”的在美上市公司，避免美方介入调查后揭发出更大的丑闻。

15.中共国工信部本周表示，已要求与汽车和芯片制造商“高度重视”和“增加产能配置”，来帮助解决汽车行业的全球半导体短缺的“危机”。 芯片的主要生产地为台湾，中共可能进行全球采购囤货居奇，谋求暴利。

16. 媒体周三报道说，巴基斯坦正在要求中共减免中共国在“一带一路”倡议下兴建约十二座发电厂的债务。这是中共加强对巴基斯坦控制的一部分。

18. 习近平为首的中共对新闻发布加强了管理，为其造假统一口径提供方便。2月9日，中共国务院新闻办公室发布2021年中央国家机关和地方新闻发言人名录，不仅发言人总人数增加，且发言人级别和参与部门也大幅提升。
