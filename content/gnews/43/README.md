众所周知，比特币监管难的原因在于它的隐匿性，它不需要你证明你是你，只要密码没错就可以完全控制自己的账号，开通新账号（比特币地址）不需要做任何实名认证，即使开通100个账号都不用任何证明，保管好相应的密码（密钥）就行了。

随着以美元为主的主权货币贬值预期的上升，加密货币受到追捧，比特币总市值从12年前的1900万美元，在今年2月份首次突破1万亿美元，加上各种加密货币近来价格的成倍上涨，对传统信用货币的虹吸引起了各国的警惕。同时比特币也成了众多犯罪分子用于邪恶的计划、资助恐怖主义和洗黑钱、藏匿贿款、逃税等。美国新上任的证管会（SEC）主席加里·金斯勒因此发出了明确的监管信号，可以预见，全球更多国家的监管机构将快步跟进。

据路透社5月15日报道，为美国东海岸每天输送250万桶燃油的Colonial Pipeline （美国殖民管道）公司，在遭遇网络犯罪团伙攻击后其网络运营中断，约9天前，该公司按照网络犯罪团伙的要求，用比特币支付了500万美元的赎金，收到付款后的黑客才提供了解密工具。

最大的勒索软件公司之一的WannaCry，曾经要求20多万名受害者以比特币支付赎金。据专门从事网络安全的PurpleSec公司称，一半以上的网络犯罪分子要求以比特币支付赎金。去中心化、匿名的比特币，一定程度上已经成为犯罪分子的帮凶，如果比特币不改变其隐匿的设计，面临的挑战和法律风险将越来越大，但失去隐匿性的比特币还会有吸引力吗？

距离特斯拉官网接受比特币支付还不到两个月，马斯克就在 5月13表示，因对比特币挖矿使化石燃料使用量增加的担心，特斯拉已暂停接受比特币购买其车辆，随后加密货币大跌。难道马斯克此前不知道比特币挖矿机耗电巨大吗？他的变脸应该和比特币无法监管并逐渐沦为犯罪工具，已经被监管部门“拉清单”有关。

话说马斯克如此着急去拥抱加密货币，无疑是对美元的信心在下降，对通胀失控的担忧在加大，美元如果发生巨幅贬值，一切都乱套了，对马斯克的商业王国将造成毁灭性打击，因此他急于寻找一个能够分散风险的加密货币，比特币被否定后，马斯克于本周三透露，他正在追求一种能耗较小的绿色加密货币。

马斯克话音刚落，就立即引来了不少加密货币专家的各种猜测，有的认为马斯克可能出资改造比特币的耗电环节，但这种改变需要几年时间才能完成，马斯克的商业王国等不了；有的专家称，马斯克更倾向于抛弃比特币转向更环保的数字货币一一这位专家说到点子上了。除了犯罪团伙，所有的人都和马斯克一样，需要合法、安全的加密货币作为支付工具，需要被广泛接受和使用的加密货币作为交易媒介和价值尺度，需要将自己的合法财富进行价值储藏，而企业需要更广领域的市场。

新中国联邦喜币由于锚定黄金，有双币设计，已获准进入全球加密货币支付系统，有自身的金融系统和联储货币发行机构，还有自成生态的G系列经济王国，具备了一个现代国家的雏形，喜币实际上就是第一个主权国家真正意义上的加密货币。亚马逊、特斯拉、苹果等这些大公司如果不捷足先登率光使用喜币，将是它们的损失，聪明透顶的马斯克不会不知道。七哥即将宣布的喜事，是否有特斯拉宣布使用喜币这样的重磅消息？
