# 英国政府“低估了香港移居计划的接受程度”

**调查发现有超过60万人可能想移居英国，在一月（新移民计划）实施后许多人在两年内要移居英国**

[!\[\]()!\[\](https://spark.adobe.com/page/qEcVCGW4ShMkY/images/560ded58-3f0b-4b14-8902-6fab01ec1604.jpg?asset_id=3d5d20a0-8574-431c-8fcd-f2d3ce7632bf&amp;img_etag=%224aa10c2c08c7dab2901994ffa2f559f8%22&amp;size=2560)](https://spark.adobe.com/page/qEcVCGW4ShMkY/images/560ded58-3f0b-4b14-8902-6fab01ec1604.jpg?asset_id=3d5d20a0-8574-431c-8fcd-f2d3ce7632bf&amp;img_etag=%224aa10c2c08c7dab2901994ffa2f559f8%22&amp;size=1024)英国政府向持有英国国民（海外）护照的香港人发出邀请，以回应中共国对异议人士的镇压。

一个新的倡导组织说，香港居民搬到英国的速度可能会比英国政府预期的要快，应该为他们的到来做更多的准备工作。

英国港侨协会（HKB）对希望在英国政府1月开始实施的新计划下移民的香港居民进行了调查，该计划允许具有殖民地时代的英国国民海外（BNO）身份的人获得签证并寻求获得“公民身份之路”。

英国内政部已经表示，预计在头三年中将有近50万人接受这项提议，但英国港侨协会表示，这一数字可能会超过60万。

计划移居的人中，约有四分之三持有大学学位，薪水收入远高于该城市的平均水平，因此将会为英国经济做出贡献。但是很少人在英国有亲人，只有一半人有在英国的朋友，因此他们可能需要帮助安顿和融入。该团体说，四分之三的人计划带孩子出行，所以英国学校需要为学生的涌入做好准备。

五分之四的受访者希望在未来两年内到达英国，这比英国政府预期的要快。

英国港侨协会的Rikkie Yeung表示：“他们想到达英国或离开香港的速度是相当惊人的。许多人很快就会到来。大多数人，80％的人计划在两年内离开香港”。
[!\[\]()!\[\](https://spark.adobe.com/page/qEcVCGW4ShMkY/images/2ee3fae3-2509-480c-92f0-a594110cd53d.jpg?asset_id=9cba8aa0-b47a-451d-a10a-69ddceccca68&amp;img_etag=%22ac605ce49317193740bec45bf11cc8f6%22&amp;size=1024)](https://spark.adobe.com/page/qEcVCGW4ShMkY/images/2ee3fae3-2509-480c-92f0-a594110cd53d.jpg?asset_id=9cba8aa0-b47a-451d-a10a-69ddceccca68&amp;img_etag=%22ac605ce49317193740bec45bf11cc8f6%22&amp;size=1024)令人向往的英国护照
英国针对取得公民身份的计划是回应北京今年夏天通过的一项国家安全法而制定的，该法已被用来压制香港政治界、学术界和媒体界的异议人士。

因为移民到英国在政治上是敏感的，所以对希望将利用计划移民的人的调查并不是对香港居民的随机抽样。中共国谴责该政策，并威胁要停止承认英国国民海外护照，并采取其它“对策”。

取而代之的是，英国港侨协会在其拥有大量关注者的社交媒体渠道上寻找了调查对象。超过300人以匿名方式参与了调查。

因为香港许多人考虑英国的签证提议或移民到包括加拿大和澳大利亚在内的其它国家，预测会出现人才流失。在香港，约有300万人，即香港人口的近一半，有资格获得英国国民（海外）护照，并且能够与家属一起出行。

那些想来英国的绝大多数人说，他们背井离乡、辗转半个地球的主要动机是政治压力。

英国港侨协会的报告说“百分之九十六的人认为，在国家安全法通过后，香港不再是他们习惯于居住的安全和自由的住所”。

几乎所有人都将他们的移居视为迈向公民身份的一步，其中93％的人希望在居住5年后有资格申请。

2019年以来，一小部分希望移居的人因其在支持民主抗议中的角色而被捕。尽管有犯罪记录通常禁止办签证，但英国港侨协会呼吁英国当局在被控犯有政治罪行的申请人的案件中运用酌处权。

接受调查的几名受访者表示，他们担心中共国安全部队的监控，甚至到了英国也是如此。该团体呼吁英国当局考虑将可能“损害国家安全”的组织（例如香港警察和官员）排除在签证计划之外。

该团体还呼吁扩大覆盖范围，以涵盖那些没有英国国民（海外）护照身份但需要避风港的人，包括在1997年从英国殖民统治下移交后出生的年轻抗议者，或者其父母没有英国国民（海外）护照身份的人。
