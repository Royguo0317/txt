《经济学人》（The Economist）是一份英国的英文新闻周报，是一本涉及全球政治、经济、文化、科技等多方面事务的综合性新闻评论刊物，着重于对这些议题提供深入的分析和评论。 （摘自[天下杂志](https://www.cw.com.tw/author/57)）

**经济学人：台湾是世界最危险地方**

《经济学人》29日提及美中必须避免台湾岛屿发生战争，台湾是地球上最危险的地方。

《经济学人》指称，北京政府一直以来都宣称只有一个中国，台湾政府是叛乱政府；美国虽然同意一个中国的说法，但长期却努力维持2个中国的存在。**台湾这座岛屿的主权争议已长达几百年**了，如今这种诡异的现状正在瓦解，紧张情势可能很快就要来到紧要关头了。

内容提及「台湾岛主权争议」与04/19美议员提出「台湾国际团结法案」指称「联合国大会2758决议」不涉及台湾、台湾人民，似乎有相呼应。

**台湾团结法案**

由美国跨党派众议员0419提出「台湾国际团结法案」，阻止北京扭曲国际组织攸关台湾的决议文字。特别提到「联合国大会2758决议」，虽然承认中华人民共和国政府代表，是中国在联合国组织的唯一合法代表，但并未处理台湾与台湾人民在联合国或周边组织代表权的问题，亦没有在中华人民共和国与台湾关系上采取立场，或**包含任何关于台湾主权的声明**。 （[中央社](https://www.cna.com.tw/news/firstnews/202104200033.aspx)报导）

**台湾主权未定论**

二战时，台湾为日本殖民地，为日本领土。二战日本战败，日本签订了「旧金山和约」，宣布放弃对台湾、澎湖主权。但并未明示台湾、澎湖主权归属中华民国。 （抛弃对台主权，但没有说给中华民国政府）

中华民国政府在盟军的同意下「占领台湾」，并宣称作为反攻大陆的基地，不断主张台湾主权已经回归中华民国。并在教育、政策上渐进式的塑化、洗脑，将台湾主权未定论变为台湾主权已定论。

中共所谓大台独已故前总统李登辉，耗时长达10年的台湾民主改革成功，对共政策上则推出「中华民国在台湾」，其实就是基于台湾主权未定论。所谓中华民国在台湾，即是中华民国虽然实质占领，但并未取得主权。随后陈水扁政府时期，改为「中华民国是台湾」；蔡英文政府则主张「台湾是中华民国」。

这可能就是主权未定论渐变主权已定论的表现之一。 （仅个人分析）

美国的一中政策，虽承认中共为合法代表中国的政府，但始终中国不承认拥有台湾主权，只有建立在台湾主权未定论才可能成立。

**取得领土主权的方法之一：转让**

由领土所有国基于自己的意思签订「处分条约（国际条约）」将其领土主权移交给他国，移交方式包含强制性、非强制性「割让，买卖，交换与赠与」。但单方声明放弃领土，不产生转让领土主权的法律笑过，仅丧失领土。

日本战败，签订旧金山和约，只表示放弃对台湾、澎湖主权，并未表示转让主权。因此，台湾、澎湖的主权依据国际条约，中华民国政府并未合法取得主权。 （台湾实质管领台湾、澎湖、金门、马祖，金马为中华民国领土）
