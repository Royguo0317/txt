# **中共国推动控制美国人医疗保健的未来**

**美国官员表示，中共国政府正试图收集美国人的DNA，他们认为，一家中共国公司最近提供的协助检测新冠的提议值得怀疑。乔恩•韦特海姆报告。**

尽管华盛顿面临两极分化，但这是一个罕见的共识来源：由中共国推动获取我们的医疗保健数据（包括美国公民的DNA）的新威胁。美国官员告诉我们，共产主义政权积极收集我们大多数个人信息对国家安全和经济均构成威胁。随着各机构、政党和总统行政机构的警钟响起，过去一年中，政府的不同部门采取了行动，以阻止我们的医疗数据流向中共国的浪潮。寻求控制我们的生物数据，进而控制医疗保健的未来——这种威胁已超过了民族自豪感，已成为新的太空竞赛。我们的调查始于在新冠危机爆发时，来自海外主动提供的一项惊人提议。

去年三月初，华盛顿州是美国首轮主要的冠状病毒爆发地点。由于新冠患者比率和检测需求激增，总部位于中共国的全球巨头——全球最大的生物技术公司——华大基因集团（BGI Group），以一份诱人的提议与华盛顿州接洽。

在给州长的一封惊人的私人信件中，华大基因集团提议建立并帮助运行最先进的新冠检测实验室。华大基因集团将会“提供专业技术”，提供“高通量测序仪”，甚至“额外捐款”进行支持。

这似乎是该州无法拒绝的议案，尤其是在迫切需要的情况下。但是官员们对华大基因及其与中共国政府的关系表示怀疑。

比尔•伊万尼纳（Bill Evanina）：他们是与共产主义国家以及军事机构双方都有联系的终极公司。

比尔•伊万尼纳最近卸任了美国最高反情报官员一职，是联邦调查局（FBI）和中央情报局（CIA）的资深人士。他对华大基因的新冠检测建议非常担心，最终谁会得到这些数据？因此他授权发布了一个罕见的公开警告：“外国势力可以从新冠检测中收集、存储和利用生物识别信息。”

比尔•伊万尼纳：我们不仅向每个美国人，而且向医院、协会和诊所发出了警告。我们知道华大基因是一家中共国公司，但我们是否了解该数据的去向？

乔恩•沃特海姆（Jon Wertheim）：今年有成千上万的美国人进行了新冠检测，你并不会认为其中有很多人在思考：“哇，这些数据要去哪里，有什么样的第三方参与其中？”

比尔•伊万尼纳：我不建议任何人那样想。但这显示了中国共产党利用诸如新冠之类的全球危机的邪恶思想。

伊万尼纳怀疑这些实验室提供的是现代特洛伊木马。华大基因带着礼物来到美国，但怀有其他动机。他说，目前尚不清楚华大基因或任何新冠检测仪是否会从鼻拭子中获取DNA，但实验室是建立立足点的一种方式，将其设备带到这里，开始挖掘数据，并在你的附近开店。

爱德华•尤：你必须退后一步，问问自己，谁有权访问该数据。

主管特工爱德华•尤（Edward You）是一个前生物化学家，后来成为联邦调查局的调查员。

爱德华•尤：随之而来的是一个非常令人不安的事实，在过去的十年左右的时间里，你会看到中共国通过大量投资收买或收购实际公司，以访问我们的数据。

中共国的专制政府及其领导人习近平一直大胆地公开其欲打败西方、从DNA科技进步中受益的雄心壮志。这个共产主义政权甚至发表了一个有琅琅上口名字的宣言。

爱德华•尤：他们有一个叫“中国制造2025”的东西。在这些国家战略中，他们明确喊出要成为这个生物时代的主导者。因此想成为领导者并能够开发疫苗，发展精密医学。

比尔•伊万尼纳：这将是我和父亲的争论。设想今天开始十年之后的某一天——我父亲收到了一个电话，并被告知：“嘿，顺便说一句，我们了解您可能有高血压。而且已处于帕金森氏症临界点。这是您应该服用的三种药物 ，今后可帮助减轻某些症状。”我父亲会说：“嗯，他们怎么知道的？”而公司来自中共国。因为他们已经根据他的DNA对我父亲进行了微定位。我父亲说：“好吧，（拍手）我会照做的。”

乔恩•沃特海姆：恶魔的拥护者论点会说：“听着，如果您能够在我的DNA中指出某些东西，我会报名购买。”

比尔•伊万尼纳：那正是我父亲所说的。（笑）所以对他来说，我的论点是从国家的长期存在价值考虑的，我们愿意这样做吗？我们是否想让另一个国家系统地消除我们的医疗保健服务？作为一个国家，我们可以接受吗？如果我们是这样一个国家，那就这样吧。但这就是正在发生的事情。

据美国联邦调查局的爱德华•尤（Edward You）说，我们在新冠病毒期间对中共国的依赖——个人防护设备、口罩——与我们未来潜在的医疗上的依赖相比，将显得微不足道。

爱德华•尤：如果我们意识到我们未来所有的药物、疫苗，医疗保健……都完全依赖一个外国渠道，会发生什么？如果我们不醒悟过来，有一天我们会意识到我们已成为医疗保健的瘾君子，而像中共国这样的国家成为了我们的药贩子。

爱德华•尤：这些公司被认定为在为收集维吾尔族人的基因信息提供便利。如果说这对我们所有人来说都应该是一个警告信号的话，那就是如果我们的数据被别人掌控，会被如何利用，这有可能会发生什么。

鉴于华大基因与中共政权的密切关系，其卷入维吾尔族危机并非巧合。2010年，在获得中共国政府15亿美元的资助后，华大基因得以急剧扩张。

比尔•埃瓦尼纳：他们像怪物一样。他们与全球60多个国家签订了合同，不仅提供基因组测序服务，还提供分析服务。

乔恩•沃特海姆：他们说，“我们是一家私人公司”。他们是吗？

比尔•埃瓦尼纳：在中国共产党治下就没有私人公司这回事。

根据一系列在西方民主国家难以想象的法律，像华大基因这样的中国公司有义务与中共国政权分享数据。这就好比，谷歌、亚马逊和脸书必须按要求把数据交给中情局一样。

乔恩•沃特海姆：所以你是想告诉我，中共国政府，无论是对生物技术公司还是（其他公司），他们可以说：“嘿，我们想要你的信息，请提供。”

比尔•埃瓦尼纳：绝对的。你必须提供中国共产党要求的任何和所有数据。其中可怕的是，有时并不都是属于他们的数据。如果你是合资企业、合营企业，他们的数据现在很容易被中共得到。

就像华大基因在自己的网站上宣传的那样，该公司一直在稳步发展与美国国内的医院和生物技术公司的合作关系，使华大基因——进而使中共国政府——有可能获得我们的DNA数据、测序技术和分析结果。

乔恩•沃特海姆：华大基因如何与美国公司合作？

比尔•埃瓦尼纳：他们的做法首先是用钱。就是投资。我想在你的公司投资1000万美元、2000万美元、8000万美元。每个公司都说好，进来吧。同时，他们会有一个不成文的规定，就是他们可以利用这些数据和你的测序能力。这些公司不知道的是，中共国保留了这些数据，并给你一份副本。

华大基因拒绝了我们的采访请求，并在一份声明中说：“美国公民的基因组数据以任何方式通过华大基因在美国的活动而受到损害的说法是毫无根据的。”他们说，他们是“一个私人组织”，成立的目的是“有利于人类健康和福祉”。

还记得华大基因提议为华盛顿州建设COVID实验室吗？《60分钟》了解到，该公司还向其他五个以上的州提出了类似的建议，包括纽约和加州….。而在联邦官员警告不要与华大基因合作后，每个州都对华大基因的实验室说了不。

不仅仅是中共国认识到你的DNA是多么有价值的商品。正如你将听到的那样：一些发展最快的美国科技公司也在这个领域。事实上，你可能已经通过在管子里吐口水交出了你的DNA。

有关美国公司和外国实体之间关系的问题可以直接向国家反间谍和安全中心（NCSC）和国家情报总监办公室（ODNI）提出。
