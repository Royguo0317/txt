## 【时事播报】美对中涉嫌强迫劳动的渔业产品发出暂扣令
` HoustonLitstar` [轉載自GNews](https://gnews.org/zh-hans/1280110/)

作者：休斯顿星光农场 写作部 火星
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/24064634/Screenshot_2021-03-25_230524-16.jpg)
5月28日，美国海关与边境保护局(CBP)针对中共国大连海洋渔业集团(Dalian Ocean Fishing Co., Ltd.)的产品发布了禁止进口的暂扣令(Withhold Release Orders,WRO)，该局称，有信息表明该公司在捕鱼作业中存在强迫劳动。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/29033532/photo_2021-05-29_04-10-51.jpg)
据路透社报道，调查显示，许多被雇用到大连远洋渔船上的印尼工人遭受了身体暴力、扣薪和债务束缚以及虐待性的工作和生活条件。

CBP表示，将立即在美国入境口岸扣留大连远洋渔业有限公司的金枪鱼、箭鱼和其他产品。CBP一位官员说，禁止进口的”禁售令”也适用于其他含有该公司海鲜的最终用途产品，如金枪鱼罐头和宠物食品。

国土安全部部长亚历杭德罗·马约卡斯说，这次行动标志着CBP首次禁止从整个捕鱼船队进口，而不是过去针对的个别船只。

马约卡斯在新闻发布会上说：“国土安全部将继续积极调查远洋渔船和其他各种行业使用强迫劳动的情况。”
