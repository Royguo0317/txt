图片来源：CNN

2021年6月19日，[澳大利亚广播公司报道，](https://www.abc.net.au/news/2021-06-19/australia-to-refer-china-to-wto-over-wine-tariffs/100228388)澳洲将就中共国对澳洲葡萄酒征收高达220%的贸易终端关税的决定向国际贸易仲裁提出上诉。这是在过去的六个月内，澳洲第二次向世界贸易组织（WTO）提出申诉，此前,前贸易部长西蒙·伯明翰曾于去年12月就中共国对大麦贸易征收的破坏性关税向世贸组织提出过申诉。

自从2020年11月，在澳洲提出对中共病毒起源进行独立调查后，中澳两国的较量持续不断。[中共国商务部](https://www.bbc.com/zhongwen/simp/world-55104613)表示，从2020年11月28日开始，采用保证金形式对澳洲相关葡萄酒实施临时反倾销措施，各公司的保证金比率为107.1%—212.1%。[澳洲政府则严正驳斥其倾销的指控](https://www.dw.com/zh/%E4%B8%AD%E5%9B%BD%E5%B0%86%E5%AF%B9%E6%BE%B3%E6%B4%B2%E8%91%A1%E8%90%84%E9%85%92%E5%BE%81%E6%94%B6212%E5%85%B3%E7%A8%8E/a-55750989)，并指责中共的做法“不公且毫无根据”。

澳大利亚葡萄酒一向以高品质享誉中国市场，在市场上的销售价格也高于同类产品，因此中共对澳大利亚葡萄酒倾销的指责纯属无稽之谈，中共的无理举措让中澳关系恶化到几十年来的最低点。某些澳洲官方人士认为，这些行动只是中共对澳洲的报复，原因是澳洲去年修改外商投资规定，盯紧中共在敏感领域的投资；以及澳洲总理莫里森呼吁国际社会对新冠病毒源头展开独立的国际调查。

自中共首次推出关税后，澳洲对中国的葡萄酒出口从11亿美元急速下降到2000万美元。 2021年6月19日，澳洲贸易部长丹·特汉（Dan Tehan）表示，“中共政府采取的行动对澳大利亚葡萄酒业造成了严重损害”，但澳洲政府将大力维护被中共国指控为反竞争行为的葡萄酒行业的利益。

同一时间，中共媒体《[参考消息](https://www.163.com/dy/article/GCRP10350514BQ68.html)》使出惯常做法开始扭曲事实，暗示澳洲软化表态愿意与中共坐下来谈判，称加征关税问题是为了中国消费者。文章指出，澳洲政府声明，澳洲愿意与中共国直接对话以解决葡萄酒争端。中方主管部门依法依规对外国输华产品采取相关措施，符合中共国法律法规和国际惯例，也是对中共国内产业和消费者负责任的行为。

[百度甚至有文章](https://baijiahao.baidu.com/s?id=1702702040260207941&amp;wfr=spider&amp;for=pc)，嘲笑英国为支持澳洲让自己变成“接盘侠”。英国和澳洲于2021年6月15日新达成了英澳自由贸易协议，该文章嘲讽到：在美国等西方朋友见死不救的情况下，澳洲红酒等商品终于找到了销路，英国愿意充当“接盘侠”。

事实上，此次澳中之间的葡萄酒争端，再一次体现出澳洲政府不畏中共的经济威胁，不为一时的经济利益而牺牲长期的安全利益。虽然澳大利亚就葡萄酒关税争端提交世贸组织裁决不会在短时间内得到问题的解决，但是澳大利亚政府的态度非常明确，澳洲政府不会在中共的经济施压下屈膝下跪。

随着全球“以毒灭共”步伐的推进，当前国际形势已经形成了坚不可摧的灭共统一战线。而中共高层还在中南坑洋洋自喜，感觉良好，作死地在国际舞台上亮出“独裁大哥”的流氓手腕，殊不知，中共越是使用流氓手段施压澳洲，越是加速把自己推向火葬场。
