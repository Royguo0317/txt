3月18日，西安市第八医院已完成新冠疫苗2针接种的检验师刘某确诊为新冠肺炎病例。官方解释是，“高危暴露”是疫苗保护失败的重要原因，与普通社区人群相比，新冠疫苗在新冠肺炎病人救治的医护人员保护率较低。

在5月30日举行的中国科学院学部第七届学术年会上，高福认为，当建立起群体免疫时，疫苗最主要的功能“不是防感染，而是防重症和防死亡。”

6月3日，广东省卫健委通报疫苗，有4人还没来得及打第二针，就被感染了。

6月11日，在联防联控机制发布会上，针对记者关于广东市民接种了疫苗还感染，是否说明疫苗的保护效力不够的提问时，王庆华回答道，“疫苗的保护率都不是100%”。他还说，免疫屏障是“达到过去一直常提到的降低感染率、降低重症率、降低病死率”。

既然保护率不是100%，那是1%还是99%？他的意思全民免疫效果只能减低死亡率。

中共对接种疫苗感染的解释由“高度暴露说”演变为“减低死亡率说”。如果全民防疫后，再发生大爆发，不知中共如何自圆其说。

众所周知，接种疫苗主要是预防和控制传染病，现在中共专家对接种疫苗的效果竟解释为减低死亡率，看来他们越来越不自信了。

接种中共疫苗带来的不仅是后遗症，更严重的将导致死亡，在疫情爆发之初，郭先生反复强调不要接种中共疫苗！

中共释放病毒的目的是要“毁掉世界经济，让世界经济转向中共，把从中国的疫苗经济，从星星之火的疫苗经济，成为全世界的中共经济为核心，然后铲除美元，让中共成为世界之国”，郭先生说。

截止6月12日，中共已经向43个国家出口疫苗，供应了3.5亿剂疫苗，平均每剂的价格在110元左右，出口额达385亿元，可谓挣得盆满钵满。
